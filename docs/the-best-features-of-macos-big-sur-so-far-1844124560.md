### 我的朋友到底在哪里？

macOS 上的地图应用终于获得了与其 iOS/iPadOS 版本相同的 TLC。除了苹果宣布的对 iOS/iPadOS 14 的所有改进——包括为骑自行车和电动汽车的司机提供新的路线选择——地图用户还可以创建他们想去的最喜欢的地方的列表。如果这太麻烦，你可以查看你所在地方(或者你想去的任何地方)的旅游指南。

macOS Big Sur 用户还可以访问“地图”的室内地图，以及 iOS/iPadOS 的“环视”功能，这可以用三个词来概括:谷歌街景。

我特别喜欢的一个功能是，如果你的朋友外出时与你分享他们的预计到达时间，你现在可以看到他们的位置。如果，比方说，你的配偶正在从商店回家的路上，而你正在整理工作，这是非常有用的。你将能够看到它们在你的台式机或笔记本电脑上的位置，而不是必须拿出另一个设备来签到。简单。

### Safari 扩展变得更加注重隐私

Safari 中的扩展？那是 [没什么新鲜的](https://support.apple.com/en-us/HT203051) 。然而，你现在可以在苹果应用商店的一个专门的类别中查找你的浏览器的扩展——你*确实*使用苹果应用商店，对吗？(我开玩笑，只因为 Mac App Store 一直感觉像是苹果被遗忘的战场。)

更重要的是，你现在将被要求在每个站点的基础上启用扩展，这是苹果方面的一个可怕的决定，以确保更粗略的扩展不会处理更多的数据。你还可以在访问网站时将扩展限制为“一日通”——因此，实际上，你可以控制*如何在 Safari 中激活*和*何时激活*扩展。

除此之外，Safari 正在进行一些急需的调整，使其更接近竞争对手:更好的标签管理(包括 favicons，以直观的方式显示您打开的网站)，当您悬停在标签上时的页面预览，以及——Edge Chromium 被诅咒——一个可定制的起始页，除了内容部分(如阅读列表)，您还可以配备任何您想要的背景图像。

这还不是全部。Safari 也有了自己的翻译功能，当网站的内容不是你所说的语言时，它会自动记录下来(在推出时，它支持七种语言中的一种)。点击一个按钮，你正在浏览的网站将立即转换成你喜欢的语言。

### 更深入地了解应用想从你身上得到什么

与 iOS 14 非常相似，Mac App Store 现在将在每个应用的产品页面上提供小功能框，通知你他们想从你那里收集什么数据，以及其他哪些第三方服务可能会访问这些数据。虽然这可能不会阻止你安装上述应用程序，但它至少会让你思考你正在看的应用程序是否真的是你需要的东西，或者可能是隐藏在一般产品背后的数据刮刀。

就是这样！老实说，macOS Big Sur 感觉更像是一个图形的重新设想，这很好。显然，苹果正试图将你在 iPhone 或 iPad 上喜欢的功能(和 UI 元素)直接集成到你的 Mac 体验中，因为有一天，一个应用程序将成为一个应用程序，你将能够在你拥有的任何苹果设备上使用它，并享受类似的外观、感觉和体验。(当然，您将从一个应用商店购买该应用，您将使用它来管理您的所有购买、订阅、加载项等。)
