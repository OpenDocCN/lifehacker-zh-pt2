# 把服装袋拿到干洗店，减少塑料垃圾

> 原文：<https://lifehacker.com/reduce-plastic-waste-by-bringing-a-garment-bag-to-the-d-1834424692>

每年春天，我都会干洗十几件冬季大衣。等我把它们放回壁橱时，我已经有足够的几乎没用过的塑料材料来包装我的整个公寓了。

Watch

如果你每个月甚至一周干洗一次衣服，这些塑料垃圾会很快堆积起来。

你典型的干式清洁塑料通常是由 [低密度聚乙烯](https://recyclenation.com/2014/11/recycle-dry-cleaning-bags/) 或#4 塑料制成的，就是你可能在购物袋、面包袋或收缩包装中找到的那种，通常被称为“薄膜”而不是袋子。由于其柔韧性，这种塑料会堵塞回收机械，需要工厂工人用手清除。出于这个原因，你们的路边回收项目一般不会接受这种塑料(尽管你可以在网上搜索一下当地项目的规则以确定)。

但是如果你想减少塑料垃圾，有一个简单的解决办法:下次去干洗店时，带上一个装着需要清洗的衣服的袋子，并要求放在原来的袋子里。一家干洗店通常会接受这个来代替塑料薄膜。

如果你正在寻找一个便于运输的服装袋，绿色的可以兼做环保服装和行李袋——，或者带上你衣柜里已经有的任何一个袋子。你也应该带上自己的衣架，这样可以避免使用不必要的钢丝衣架。(许多干洗店会很乐意接受这些重复使用。)

在网上搜索你附近的绿色干洗店；T2:他们通常会选择把衣服放在可重复使用的袋子或服装袋里。

如果你还不能放弃干洗塑料的习惯，使用 [地球 911 的搜索定位器](https://search.earth911.com/) 找到附近接受 4 号塑料的商店或设施。像家得宝(Home Depot)和西夫韦(Safeway)这样的商店经常有店面收集箱来收集这种类型的塑料，所以你可以对你的洗衣废物感到不那么内疚。
