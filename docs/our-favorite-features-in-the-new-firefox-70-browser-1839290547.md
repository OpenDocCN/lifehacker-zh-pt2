# 新 Firefox 70 浏览器中我们最喜欢的功能

是时候更新到全新版本的 Firefox Quantum 了。虽然你的浏览器最终会帮你解决这个问题，但我建议你点击汉堡图标，点击“帮助”，然后点击“关于火狐”。当火狐 70 下载到你的台式机或笔记本电脑时，这里有一个新的快速浏览。

### 节省更多的 Mac 电池电量

由于 Firefox 的合成管道的一些改进——如果你想了解令人讨厌的细节，你可以在这里阅读——浏览器现在比以往任何时候都更省电，在某些情况下提高了 3 倍。我认为 Mozilla 的这个例子说明了一切:

如果你想知道火狐的改进——切换到部分合成——是否能在其他平台上运行，Mozilla 是这样说的:

> “Firefox 在一些平台和 GPU 组合上使用部分合成，但不是所有的平台和 GPU 组合。值得注意的是，在最新版本的 Windows 上，以及所有关闭了硬件加速的系统上，Firefox 在 Windows 上为非 WebRender、非 Nvidia 系统启用了部分合成。Firefox 目前在 Linux 或 Android 上不使用部分合成。”

### 为你阅读一份关于 Firefox 拦截的所有追踪器的报告

Firefox 现在不仅屏蔽了 LinkedIn、脸书和 Twitter 等社交网站上的跨站点跟踪 cookies 这是一个全新的功能——而且你现在可以查看自己的隐私保护报告，向你显示 Firefox 在后台屏蔽的所有内容。要查看报告，只需点击地址栏旁边的盾形图标，然后选择“显示报告”

你不仅会得到过去一周 Firefox 屏蔽了多少追踪器的详细数据，还会快速了解与你的 Firefox 帐户相关的电子邮件地址最近涉及了多少数据泄露——如果你也注册了这项免费功能的话。

如果你想成为一名真正的玩家，我建议你在 Firefox 的设置中(在主页部分)设置`about:protections`为你的浏览器主页。这样，无论何时你点击主页图标，或者点击键盘上的 ALT+Home，你都可以直接跳转到你的隐私报告。我觉得这比一张白纸有用多了，对吧？

### 为网站设置更安全的密码

Firefox 通过浏览器的 Lockwise 功能，使得为新账户生成安全密码变得更加容易(或者在你登录时输入你保存的旧密码)。正如 Mozilla [中写的](https://blog.mozilla.org/blog/2019/10/22/latest-firefox-brings-privacy-protections-front-and-center-letting-you-track-the-trackers/) :

> 现在，当你创建一个账户时，系统会自动提示你让 Lockwise 生成一个安全的密码，你可以直接保存在 Firefox 浏览器中。对于当前帐户，您可以右键单击密码字段，通过填充选项访问安全生成的密码。所有安全生成的密码都会自动保存到您的 Firefox Lockwise 帐户中。

对于任何类型的密码管理器来说，这都是一个非常标准的内容。不过，我还喜欢的是重新设计的 Lockwise 仪表盘，你可以通过点击汉堡图标和点击“登录和密码”来访问它

该屏幕不仅使删除、创建和更新您的密码变得容易，而且如果当前存储在 Lockwise 中的任何密码涉及任何类型的数据泄露，您还会收到通知。这类似于 1Password 的 [Watchtowe](https://lifehacker.com/1passwords-updated-password-manager-is-a-must-have-for-1826379117) r 功能，我很喜欢。

### 更多的 Windows 用户使用 webrender

您可能不知道什么是 webrender，但是如果您可以使用它，您会感受到它的效果— 性能和稳定性的提高是由于您的 GPU 渲染 web 内容而不是您的 CPU。在 Firefox 70 中，更多的 Windows 用户将有机会使用 webrender，因为 Mozilla 已经向那些运行英特尔集成显卡和低于 1920 x 1200 像素的显示分辨率的用户开放了该功能。
