# 为什么你的苹果提醒在 iOS 13 中消失了

> 原文：<https://lifehacker.com/why-your-apple-reminders-disappeared-in-ios-13-1838702354>

苹果提醒今年正在进行一次大检修。新版本现在可以在 iOS 13 和 iPadOS 13 上使用，并将在本月 Catalina 更新发布时在 macOS 上发布。它看起来更好，并增加了新的功能，增加了有用的深度，如创建与行动项目相关的列表的能力。



不幸的是，它也可能吃掉你的待办事项清单。

由于一个 [兼容性问题](https://support.apple.com/en-us/HT210220) ，新提醒 app 中的笔记与旧版本不兼容，反之亦然。这意味着当你在运行 iOS 13 的 iPhone 或运行 iPadOS 的 iPad 上的提醒中添加任务时，这些提醒不会在 Mac 上显示，除非你使用的是 MacOS Catalina beta。同样，您在 Mojave Mac 上制作的提醒不会出现在您更新的手机和/或平板电脑上。

更糟糕的是，当你将手机、平板电脑或电脑更新到新版提醒时，你使用该设备 [创建的所有更新将在所有平台上删除](https://www.theverge.com/2019/9/30/20892460/apple-ios-13-reminders-broken-icloud-sync-macos-catalina-delay-upgrade) ，无论新旧。

即使你已经升级了，你仍然有可能丢失你的信息。新的提醒应用程序会在你第一次打开它时要求你确认更新。如果在打开应用程序之前，您已经在 iOS 13 或 iPad OS 设备上使用辅助应用程序或进程制作了提醒事项，当您打开应用程序并按下“立即升级”时，这些提醒事项将被删除

Windows 用户也受到了这一变化的沉重打击。新版 [iCloud web app](https://www.icloud.com/reminders) 支持新版提醒，但是 [苹果称](https://support.apple.com/en-us/HT210220#shared)iCloud for Windows app 更新后“看不到 I cloud 提醒”。

并且 [多个](https://www.zdnet.com/article/apple-creates-ios-13ipadosmacos-10-15-catalina-upgrade-mess-and-potential-data-loss/) [网点](https://www.theverge.com/2019/9/30/20892460/apple-ios-13-reminders-broken-icloud-sync-macos-catalina-delay-upgrade) 均报告用户存在全面同步问题。所以最重要的是，你的提醒可能不会像预期的那样起作用。普遍的共识似乎是，一旦 macOS Catalina 推出，一些(如果不是全部)同步问题将得到解决。(据 [ZDNet](https://www.zdnet.com/article/apple-creates-ios-13ipadosmacos-10-15-catalina-upgrade-mess-and-potential-data-loss/) 报道，那可能发生在本周。)不过，这只是一个有根据的猜测。

那么我们该怎么办呢？遗憾的是，没有什么奇特的方法可以解决这些成长的烦恼。在我看来，最好的方法是认识到这样一个事实:当你升级设备并做相应的计划时，你可能会丢失一些提醒。在将您的任何设备更新到 iOS 13/iPadOS/MacOS Catalina 之前，请截图或记下您的提醒事项列表。

如果你已经更新了其中一个设备，但没有更新其他设备，你可以使用新版本的应用程序重新制作设备上的所有提醒事项，这样当你更新其他设备时就不会丢失任何内容。一旦你更新了操作系统，确保进入每台设备上的提醒应用程序并启动升级，即使你通常使用 Siri 来进行提醒。
