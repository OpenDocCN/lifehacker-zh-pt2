# 像伊尼戈·蒙托亚一样介绍自己

> 原文：<https://lifehacker.com/introduce-yourself-like-inigo-montoya-1840314386>

你好，我叫伊尼戈·蒙托亚。我在楼下的业务发展部工作。我能在午餐时给你一些建议吗？



导师制项目 [、工作午餐](http://www.workinglunch.online/) 的首席执行官艾丽·麦克林(Ally McLean)发明了一个巧妙的助记法，用来恰当地介绍自己，这个助记法受到了《T4:公主新娘中可敬的剑客的启发。

为了更自信、更有意义地向别人介绍自己，你应该模仿伊尼戈·蒙托亚在《公主新娘》中的著名台词:“你好，我叫伊尼戈·蒙托亚。你杀了我父亲。准备受死吧。”有了这三句话，伊尼戈:

*   问候他的新联系人
*   给自己起名字
*   解释了他和他的联系人之间的联系
*   设定会面的基调和目的

任何时候你亲自介绍自己，通过电子邮件或电话，你都应该这样做。如果你讨厌打电话或见人，准备一句涵盖这四点的开场白会非常有帮助。

它在专业环境下工作:

> 嗨，我是尼克！我在入门网站 Lifehacker 工作。我在写如何成为一名竞技小丑。我在沃斯堡股票展和牛仔竞技会上看到了你的工作视频，我想从你那里得到一些建议。

它工作起来更随意:

> 嘿，我是诺姆，我是这里的常客。我以前没在酒吧见过你，我能请你喝杯啤酒吗？

通常这种事情是显而易见的。当你心慌的时候，当你处在一个新的或令人害怕的环境中，当你忘记了基本的事情时，它就会派上用场。它甚至适用于一屋子的陌生人，假设你有一些理由和他们在一起。

工作午餐是对澳大利亚和新西兰游戏和科技行业中代表性不足的人的指导。麦克莱恩在一次工作午餐介绍中加入了伊尼戈·蒙托亚的建议，并在推特上发布了她现在的私人账户 中的相关幻灯片。它的 [在线弹出](https://twitter.com/VikiLovesFACS/status/1202212904071811082) 每隔 [几个月](https://laughingsquid.com/inigo-montoya-guide-to-networking-success/) ，经常不编辑。所以下一次，你会知道是谁发起的:一个真正的专业社交者。
