# 将珍珠洋葱变成美味的烧焦的球体

> 原文：<https://lifehacker.com/transform-pearl-onions-into-savory-charred-orbs-1845056629>

你可能知道，也可能不知道，我居住的俄勒冈州波特兰市目前情况很糟糕。我不会在这里列出我们所有的疾病，但此时此刻最紧迫的问题是空气，这是不好的。我一直试图呆在我的公寓里以避免它，这意味着我已经不能去杂货店那么多。(我已经不怎么去了，因为疫情，它仍然非常真实！)



这很糟糕，但一个小小的亮点是，我被迫在我的储藏室和冰箱里寻找被遗忘的食物。就像*砍了*，但是反乌托邦！最令人兴奋的发现是一块猪排和一袋安森米尔斯粗燕麦粉，但最令人昏昏欲睡的是一袋吃了一半的珍珠洋葱，我在绝望(和无聊)中摇来摇去。

“这太愚蠢了，”我喃喃自语道，同时在我的华夫饼烤盘之间压了一些稍微抹了点油和盐的冷冻洋葱。“这一切有什么意义？”

重点是，至少就华夫饼而言，将光滑冰冷的小葱球转变成烧焦的甜味球。华夫饼干洋葱介于烧焦和焦糖化之间，中心柔软，美味可口。他们也煮得很快，这是我一直在寻找的准备方法。(然而，它确实会制造混乱，但是我们有 [策略来对付那个](https://skillet.lifehacker.com/how-to-clean-your-electric-waffle-maker-1844234369) 。)

我让它们冷却一会儿，然后加入你在上面看到的藜麦，它们给这道菜带来了风格和美味。我认为它们会成为素食玉米卷或潜艇三明治的可爱馅料，但没有必要变得超级有创意——它们是烤鸡和炖肉的简单配菜。

冷冻珍珠洋葱需要少量的准备:在冷水下冲洗它们以去除任何霜(并稍微软化)，然后在纸巾上晾干几分钟。淋上足够的油(我用葡萄籽油是因为它的烟点高)，然后撒上大量的盐，用中高火烘烤，直到它们变成金黄色并烧焦。这取决于你的洋葱解冻的程度，以及你一次解冻的数量，这可能会在一分钟内发生——所以要经常检查。如果你打算放在沙拉中食用，放凉，或者与红烧或烤肉一起享用。
