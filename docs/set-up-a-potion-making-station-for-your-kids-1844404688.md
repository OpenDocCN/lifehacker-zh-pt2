# 为你的孩子建立一个魔药制造站

> 原文：<https://lifehacker.com/set-up-a-potion-making-station-for-your-kids-1844404688>

现在是夏天。有一种疯狂的病毒在传播。孩子们对常见的谜题、棋盘游戏、书籍和玩具感到厌倦。是时候让他们酿造一些“药水”了。



你小时候可能也做过类似的事情。将不同的液体和其他材料混合在一起，调制出某种奇怪的、泡沫状的、渗出的药剂。(如果你没有，那就太遗憾了。你错过了。)没错，这很乱，但这也是一种经典的感官活动，会让小孩子一次忙碌好几分钟。

[的 Susie Allison 在 Instagram 上给了我们一些关于如何建立这样一个站的指导:](https://www.instagram.com/p/CCmHb9Plva-/)
