# 如何像曼达洛人一样为人父母

> 原文：<https://lifehacker.com/how-to-parent-like-the-mandalorian-1841360861>

曼达洛人 以其新鲜的故事情节、朗朗上口的配乐、一些机器人和一些时髦的武器取悦了观众。但所有这些都比不上那个孩子，我们都叫他尤达宝宝，尽管我们知道他不是真正的*尤达。事实证明，曼达洛人不仅仅是一长串传奇中的又一部《星球大战》传奇。《曼达洛人》是一个关于单亲父母的故事。*

Watch

这个曼达洛人并不是第一个被意外抛入养育角色的人。但他这样做是出于优雅、耐心和一种与生俱来的意识，即在照顾一个小家伙时，什么才是真正重要的。以下是曼多教给我们的育儿之道。

### 拥抱刚为人父母的短暂的全神贯注的天性

初为人父母的人可能会感到不知所措，因为他们的生活从事业、家庭、朋友和浪漫的美好小组合，一直到所有的婴儿。我们谈论“失去”自己 就好像对一个小孩的关心和幸福完全盖过其他一切不是完全自然的。

这个曼达洛人从一个勤奋、备受尊敬的赏金猎人(在他的领域中名列前茅！)到主要关心让这个小家伙活着。我们能理解！但我们这些经历过的人也知道，最终，事情会变得更容易，他很快就会回来，再次给所有的客户留下深刻印象。

### 偏爱“自然结果”胜过纪律

孩子们——他们挑战极限。你对他们不跳到家具上有多认真？他们会多跳几次只是为了看看。扔玩具绝对*永远*可以吗？让我们把这个扔到房间的另一边看看。

你可以大喊，你可以让他们暂停，*或*你可以让他们知道他们行为不端的“自然后果”。和他们的朋友在床上跳？你们需要找另一个地方玩。向你妹妹扔一只塑料恐龙，那只塑料恐龙会消失一段时间。

曼多提醒我们，体验生活中的自然结果是最有效的老师——这就是为什么 [尤达宝宝不允许坐](https://youtu.be/_5v9ef5TDOc) 靠近他船上的按钮:
