# 数字秤比模拟秤更精确吗？

> 原文：<https://lifehacker.com/is-a-digital-scale-more-accurate-than-analog-1835377277>

我们都踩过老式的模拟秤，并坚持认为数字是错的——一定是秤的问题！但是，购买一台昂贵的新电子秤及其所有额外功能会更准确吗？

在大多数情况下，你可能会发现数字秤是一个更好的选择；首先，它们通常需要较少的校准，并且更加精确。不过，在购买之前，你应该考虑一些事情，特别是为什么一些额外的功能可能会被证明是无用的，以及为什么在任何秤(无论是不是数字秤)上称重时一致性都是必要的。

### 模拟秤可能容易磨损

作为背景，模拟秤使用弹簧称重，而数字秤通常使用传感器操作。具体来说，数字秤依赖于连接在称重传感器上的应变仪(在您的体重下会弯曲)。当这些仪表被拉伸时，它会向转换器发送信号，转换器将信号数字化，在您的天平上产生可读的数字。

西奈山医院的临床营养师朱莉·德文斯基 说，数字体重秤*趋向于*更准确，原因有很多。首先，模拟秤的弹簧可能会逐渐磨损，并且随着时间的推移可能会提供不准确的读数。

也没有真正的方法来看待十进制测量，这意味着你要么盯着 164 磅或 165 磅，要么盯着中间的某个地方，没有真正的方法来测量这两者之间的精确数字(如果你*认为*关心的是具体性)。

同样，你可以很容易地读出数字刻度上的。如果你曾经站在模拟体重秤上，眯着眼睛想弄清楚这条线到底指向哪里，那么在自我报告体重时会有一点误差，不像数字体重秤上的数字。

### 数字秤有额外的功能，但您可能不需要

与此同时，电子秤可能还带有许多其他有用的功能，比如记录你的称重历史。不过，其他功能也很成问题。(有人认为体重秤可以准确检测你的身体脂肪、、或骨密度，正如亚马逊上的这款 [产品](https://www.amazon.com/Digital-GreaterGoods-Accurate-Composition-Measurements/dp/B019928F74?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/is-a-digital-scale-more-accurate-than-analog-1835377277&asc_source=&tag=kinjalifehackerlink-20) 所吹嘘的那样，这种想法是值得怀疑的——不过，如果你选择购买一台，你可以忽略这些额外的功能。)数字秤也很贵，需要定期更换电池，不像标准的模拟秤。

至于证实一种尺度是否比另一种更准确的研究，并不多；2013 年的一项研究对 61 种不同的家用体重秤进行了测试，发现研究中的模拟表盘秤往往不太精确，但没有明确的原因。

研究人员推测，数字体重秤可能有“更少的移动部件不对齐或损坏”，也可能在参与者的自我报告中出现错误。模拟秤的使用寿命几乎是数字秤的两倍，这表明磨损可能是原因之一。

### 踩秤时，一致性是关键

在这两种规模中，有*几件事是你应该做的，以保持整体的准确性。*

首先，你应该总是在坚硬平坦的表面上称体重；正如 [Wirecutter](https://thewirecutter.com/blog/can-i-trust-my-bathroom-scale/) 所写，在地毯或不平坦的表面上放置任何秤都有可能导致读数不准确。您还应该尽可能经常地校准秤。在模拟天平上，使用天平侧面的旋钮很容易做到这一点。尽管这也是一个相当简单的过程，但您应该参考您的数字秤的校准说明。

如果减肥是你的目标，Devinsky 还提供了另一个关键提示:不要粘在秤上。“对我的许多客户来说，一直称体重的做法可能会变得强迫症，”她在电子邮件中说。“此外，你的体重可能会自然波动 3-5 磅，所以每天称体重可能是一种低效的跟踪进展的方式。”

取而代之的是，每周称一次体重(甚至更少),每次选择踩在同一时间(例如早上)用同一称保持一致。

正如 [断线钳](https://thewirecutter.com/blog/can-i-trust-my-bathroom-scale/) 所言，即使不精确的秤也可能仍然有价值；如果你看到一个数字在几周或几个月内上升或下降，即使这个数字本身是错误的，它仍然可以作为对你整体进步的一个有用的评估。
