# 安排一天的医生预约

> 原文：<https://lifehacker.com/schedule-a-day-of-doctor-appointments-1830382060>

典型的美国人很不善于请假，也不善于像他们应该做的那样经常去看医生。当然，对于许多人来说，这有结构性的原因，但随着一年接近尾声，考虑将这两者结合起来，假设你有稳定性和方法:请一天假，把你所有的医疗预约都解决掉。



Lifehacker 建议 [休假一天](https://lifehacker.com/take-a-micro-vacation-to-tackle-your-personal-to-do-lis-1826329141) 去检查你待办事项清单上的项目——类似于的迷你生产力假期。这是类似的，但你不是清理你的衣柜和更新你的驾照，而是照顾你的健康，让自己在 2019 年处于最佳状态。

下面是怎么做的。参考你的保险政策，确保你已经利用了你这一年的“免费”预防性保健服务，包括身体检查、牙齿清洁和你一直推迟的任何事情。美国退休人员协会 报道说，“健康护理专家说一天预约两次是有意义的，也许最多三次，所以你需要挑选安排什么，”该协会称，体检、洗牙、乳房 x 光检查、皮肤科预约或妇科检查等都是不错的选择。(另一方面，结肠镜检查需要麻醉— 所以最好单独预留一天。)你可以使用美国卫生部的 [My Health Finder](https://healthfinder.gov/myhealthfinder/) 来查找预防服务。

如果你 [有一个 FSA](https://twocents.lifehacker.com/how-to-spend-your-extra-fsa-money-1821055768) ，在年底前安排约会会更有好处，这样你就可以把钱用在有价值的服务上，而不是囤积瓶防晒霜。如果你还有剩余资金，记得你可以买一大堆有用的非处方药，比如阿司匹林和创可贴。

给你的医生办公室打电话，看看他们什么时候有空。当天安排好预约，为看医生/牙医不可避免的延误留出足够的空间。美国退休人员协会建议，打电话到办公室询问他们什么时候最忙，避免在那个时间预约，因为等待时间可能会更长。

如果你还没有去看医生，你可以使用像 ZocDoc 这样的服务，让用户在网络中搜索医生和牙医。它还可以通过应用程序轻松查看预约时间、阅读评论和安排日程。

记住，当你去赴约时，带上你的保险卡、身份证和金融服务管理局/HSA 卡。美国退休人员协会建议带上一份你服用的所有药物的清单以及剂量(纸质副本或登录你的手机)，以及一份你有任何问题或你想谈论的问题的清单，以便你在和医生在一起时可以参考。

当你在那里的时候，安排一个后续的预约或者下一年的拜访。人们很容易忘记去看医生做预防措施，或者只是一直推迟，但这样做可能会损害你的健康——并且会让你在未来付出比休假多得多的代价。
