# 用“坏消息”网页游戏识别假新闻

> 原文:[https://life hacker . com/identify-fake-news-with-the-bad-news-web-game-1835916627](https://lifehacker.com/identify-fake-news-with-the-bad-news-web-game-1835916627)

去年 [我们告诉你坏消息](https://lifehacker.com/learn-about-fake-news-with-this-web-game-1823229554) ，这是一个网页游戏，通过让你扮演一个拖拖拉拉的媒体大亨来教你如何辨别错误信息。现在，游戏的创作者有证据表明游戏 [可以作为对抗错误信息](https://www.nature.com/articles/s41599-019-0279-9) 的“疫苗”——他们为孩子们创造了一个简化版本。

Watch

我今天第一次经历了坏消息。这很简单——你只需在每一步的几个选项中做出选择，只需几分钟就能完成整个游戏——但这是基于一个非常聪明的前提。你不是被告知如何发现错误信息，而是实施策略来创造错误信息。

我开始为我的媒体选择一个名字，诚实的真相在线。然后，这个游戏引导我通过人造推特诉诸情感，可信的阴谋论，以及对试图揭穿我的人的攻击。在游戏开始和结束时，你可以选择做一个小测验，对游戏中的一些推文的可靠性进行评级。

一开始，我持怀疑态度:许多这样的推文和标题*可能是真的；我怎么知道？到最后，我对他们中的大多数人都点击了“不可靠”，因为他们显然采用了我在游戏中使用的一些策略:模仿、诉诸情感、将问题两极分化等等。所以，我想你可以说它起作用了。*

根据游戏创作者 本周发表的一项研究，这是大多数人的反应。无论玩家在游戏前还是游戏后看到的游戏中模拟真实新闻的推文都有相似的可信度评级，但那些旨在操纵的推文在人们玩后得到的评级较低。换句话说，这个游戏似乎是有效的。

这款游戏现在有更多的语言版本，还有一个儿童版，适合八岁到十岁的。我让我九岁的孩子玩游戏，游戏让他负责一个虚构的学校网站。在游戏的敦促下，他散布了一个谎言，说学校要带小狗来上课，他还煽动了一个关于电子游戏禁令的谣言。他后来告诉我，人们可能会撒谎，他并不感到惊讶，但他从来没有真正考虑过为了乐趣和利益操纵人群的问题。希望他把这些知识用在好的方面，而不是坏的方面。
