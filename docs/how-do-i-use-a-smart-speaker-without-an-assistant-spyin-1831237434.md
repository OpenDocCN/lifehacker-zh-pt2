# 我如何使用智能音箱而没有助手监视我？

> 原文：<https://lifehacker.com/how-do-i-use-a-smart-speaker-without-an-assistant-spyin-1831237434>

由于这是下周大送礼日之前的最后一个生活黑客 [技术建议专栏](https://lifehacker.com/c/tech-911) ，所以本周的问题来自一位需要帮助为自己挑选礼物的读者是再合适不过了。实际上是多份礼物——他不想让任何公司在他安排好之后监视他。



### 这个问题

生活黑客读者罗伯特写道:

> 我想要不听的“智能”扬声器。我真的不喜欢让 Alexa 或 Siri 或其他什么东西听所有正在发生的事情。我目前在 Airport Expresses 上插入了扬声器，这样我就可以从我的设备上分享音乐了。在某个时候，我的小“网状”网络将不得不升级(希望它能坚持到 wifi 6 出来)，我将失去这个伟大的功能。你知道有其他选择吗？我想在各种地方(商店/车库/室外)安装扬声器...)让我可以分享音乐。我不在乎他们是否可以运行应用程序，我可以通过流媒体服务或设备上的音乐分享我的设备上的音乐，我不想要语音控制。

### 医生的建议

这是一个有趣的例子——没有语音接入的连接扬声器。数据隐私对我来说是近在咫尺的，我完全理解为什么你不会想要家里的东西 [听你的每一句话](https://gizmodo.com/the-amazon-alexa-eavesdropping-nightmare-came-true-1831231490) (即使只是为了确定你是否希望设备激活并响应一个命令)。

我最初的想法是，你可以通过一个应用程序或使用扬声器本身的硬件开关，从众多带有静音功能的智能扬声器中选择一个。我自己喜欢后一种选择，因为麦克风永远保证关着；谁知道某些软件更新或应用程序重新安装是否会意外地启用您想要静音的功能。(至于静音的具体细节，谷歌在这里 列出了过程 [，亚马逊在这里——这是两个更受欢迎的智能扬声器公司。)](https://support.google.com/googlehome/answer/7072889?hl=en)

关闭智能扬声器上的麦克风，它将听不到你说的任何东西——也不会意外地或以其他方式将你说的话传到云端。你仍然可以使用任何你喜欢的兼容应用程序将视频流传输到设备上，但与它们交谈绝对没有任何作用。

同样，你可以寻找类似亚马逊的 [Echo Link](https://www.amazon.com/Echo-Link-Stream-stereo-system/dp/B0798DVZCY/ref=sr_1_3?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-do-i-use-a-smart-speaker-without-an-assistant-spyin-1831237434&asc_source=&ie=UTF8&keywords=echo link amp&qid=1545335756&s=amazon-devices&sr=1-3&tag=kinjalifehackerlink-20) 或 [Echo Link Amp](https://www.amazon.com/Echo-Link-Amp-amplify-speakers/dp/B0798GJVTR/ref=sr_1_cc_2?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-do-i-use-a-smart-speaker-without-an-assistant-spyin-1831237434&asc_source=&ie=UTF8&keywords=echo link amp&qid=1545335756&s=aps&sr=1-2-catcorr&tag=kinjalifehackerlink-20) 的设备。两者都没有内置麦克风，但都允许你连接自己的扬声器，并通过 Alexa Cast 播放音乐。谷歌的 [Chromecast 音响](https://store.google.com/us/product/chromecast_audio?hl=en-US) 也是如此。您不能对着它说话，但可以对着它(和任何连接的扬声器)流式播放。

远离亚马逊和谷歌设备，你也可以选择一个具有流媒体功能的智能扬声器——像[Sonos One](https://www.sonos.com/en-us/shop/one.html)——而不用设置 Alexa 组件。(点击扬声器顶部的按钮[将禁用其麦克风](https://en.community.sonos.com/announcements-228985/introducing-the-new-sonos-one-6791304) 。)如果你没有将扬声器链接到 Alexa，你的扬声器就不会费心监听触发命令，你可以通过检查确保麦克风正在使用的 LED 灯不亮来确认这一点。

我认为这三个选项中的任何一个都是好的，这取决于你的预算以及你愿意在多大程度上容忍一个开关而不是没有集成任何数字助理。我甚至还没有接触过真正的多房间设置，这需要一个接收器或一个放大器，将电线连接到你房子周围的专用扬声器，以及大量的耐心和周末时间来设置，但这也是一种选择。你也可以 [买一个便携式蓝牙音箱](https://www.sony.com/electronics/wireless-speakers/srs-xb41) 带着它去任何地方——或者 [一些更永久的](https://www.marshallheadphones.com/mh_us_en/stanmore-bluetooth) (和复古)你可以把它插到墙上任何你想摇滚的地方。
