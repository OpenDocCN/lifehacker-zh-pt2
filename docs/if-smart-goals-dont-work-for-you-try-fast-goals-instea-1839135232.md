# 如果聪明的目标对你不起作用，试试快速的目标

> 原文:[https://life hacker . com/if-smart-goals-don-t-work-for-you-try-fast-goals-instea-1839135232](https://lifehacker.com/if-smart-goals-dont-work-for-you-try-fast-goals-instea-1839135232)

你可能熟悉 SMART 目标的概念:具体的、可衡量的、可实现的、现实的、及时的。这种目标可以得到结果，但它可能不是让你达到目的地的最佳方式。

在这种情况下，你可能要开始设定快速目标。

管理专家 Donald Sull 和 Charles Sull 在麻省理工学院斯隆管理评论 中定义并解释了快速目标:

> 我们发现四个核心原则支撑着有效的目标系统，我们用缩写词 FAST 来总结这些要素。[...]目标应该嵌入到**频繁的**讨论中；**雄心勃勃**在范围上；用**具体的**指标和里程碑来衡量；并且**透明**给组织里的每个人看。

眼尖的读者会注意到，聪明和快速的目标有一个共同的属性:*特异性*。在你开始朝着一个目标努力之前，明确你想要实现什么，以及你如何知道你已经实现了。

以下是这两种目标的不同之处。如果你设定了一个快速的目标，那就让它雄心勃勃而不是现实。正如苏尔和苏尔提醒我们的:

> 设定目标时谨慎行事的诱惑是可以理解的，但往往会被误导。回想一下，追求远大目标的员工明显比目标不那么具有挑战性的同事表现更好。

换句话说——我不敢相信我在打这个字——向月亮射击，即使你错过了，你也会在星星中着陆。

一旦你设定了雄心勃勃的目标，不管是为你自己，你的工作场所，还是你的家庭，经常回顾它。这是苏尔和苏尔。

> 如果一年 363 天都被归档，那么即使是最精细的目标也不会有什么影响。为了推动战略执行，目标应该作为一个框架，指导全年的关键决策和活动。

这与我最近在 Nir Eyal 的新书 [*中读到的一些东西有关:如何控制你的注意力并选择你的生活*](https://www.nirandfar.com/indistractable/) *。Eyal 建议，在你设定一个目标后，你应该评估随后的每一个决定，看它是让你更接近还是更远离你的目标。*

当然，这并不总是 100%现实的，但是既然我们已经在向月球进发，为什么不试一试呢？

那就是照顾*频繁的*、*雄心勃勃的*，以及*具体的*。首字母缩略词中的最后一个词*透明的*，应该是不言自明的。让每个与帮助你实现目标有利害关系的人都清楚你的目标，以及你努力实现这些目标的原因。

你们中的一些人可能会对透明的方面吹毛求疵。我并不是说你需要分享你的全新创意项目、日常锻炼或诸如此类的每一个细节。事实上，有研究表明，如果你不告诉别人，你更有可能实现个人目标。

但是，如果你的目标涉及到其他人的参与，比如减少你的家庭订购外卖的次数，而不是在家准备食物，那么对目标本身透明是一个好主意(“每周只订购一次外卖”)，你为什么要设定目标(“为了省钱”)，以及你如何知道你是否已经实现了目标(“我们是否将外卖限制为每周一次”)。

公开相关人员如何帮助你实现目标，或者帮助整个团队实现共同目标，这也是一个好主意。这样，他们会知道他们的行动和决定是否帮助每个人更接近目标。

例如，如果你想每隔一天早上有一个小时的空闲时间去跑步，让你家里的其他人知道他们如何支持你。如果你想设定一个每周只点一次外卖的家庭目标，就要明确每个人在膳食计划、食物准备和清洁方面需要承担和/或分担新的责任。

记住，快速目标不同于聪明目标，是雄心勃勃的。如果有几个星期你点了两次外卖，那也比每天都点要好。同样的道理也适用于你设定的其他快速目标——所以你最好尽快开始设定它们，看看它们能让你走多远。
