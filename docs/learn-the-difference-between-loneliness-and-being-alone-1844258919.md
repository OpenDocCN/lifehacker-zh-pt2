# 了解“孤独”和“独处”的区别

> 原文：<https://lifehacker.com/learn-the-difference-between-loneliness-and-being-alone-1844258919>

在过去的几年里，我们被告知孤独是一种公共健康危机，其危害性不亚于每天吸 15 支烟。传达的信息是，我们是人，是需要人的人，花太多时间独处是一个大问题。哦，但同时，显然 [我们现在都是内向的人](https://thoughtcatalog.com/heidi-priebe/2015/09/why-everyone-and-their-mother-is-now-identifying-as-an-introvert/) ，和别人 [社交是让人疲惫的](https://psychcentral.com/blog/social-exhaustion-avoiding-introvert-burnout/) 。那到底是哪个？原来，两者都可以，因为“孤独”和“独处”是两个完全不同的概念。以下是它们的区别，以及为什么更好地理解每个术语的含义会有所帮助。

### 孤独对孤独

鉴于过去几个月来我们一直在进行的隔离、物理距离和自我隔离，我们的假设是，那些独自生活的人一定会感到孤独。虽然对一些人来说可能是这样，但对另一些人来说却不是，他们觉得自己一直被朋友、家人和同事包围着，即使是虚拟的。这两个概念并不相互排斥。让我们来看看它们各自的含义。

在 MindBodyGreen 的一篇文章中，[Margaret Paul 博士提供了这样的解释](https://www.mindbodygreen.com/articles/difference-between-loneliness-and-being-alone) :

> *孤独感*是当你想与某人联系，比如你的伴侣，而要么没有人可以联系，要么你的伴侣无法联系时，你产生的感觉。

Eglantine Julle-Daniere 博士在为 [Psychology Today](https://www.psychologytoday.com/us/blog/talking-emotion/202004/being-alone-vs-being-lonely) 撰写文章时指出，孤独是“不与另一个人(可能是人或动物)在一起的身体状态”，而孤独是“一种心理状态，其特征是当一个人(自我)认为自己的社会关系在数量和质量上低于期望时出现的痛苦经历。”换句话说，就是当你在某个特定时间的社会接触对你来说不满足的时候。

如果你现在独自一人，Julle-Daniere 建议利用这段时间重新关注你自己、你的需求和让你感觉良好的事情。“这是一段时间，用来确定你想与哪些人联系，以及你想培养什么样的爱好，” [她写道](https://www.psychologytoday.com/us/blog/talking-emotion/202004/being-alone-vs-being-lonely) 。

这里的要点是，你可以独自度过大部分(或所有)时间，但不会感到孤独——或者，你可能一直被人包围，并一直体验孤独。了解两者之间的区别可能会帮助你更好地应对你目前的情况。
