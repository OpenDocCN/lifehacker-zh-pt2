# 你可能不会从一个包裹里得到新冠肺炎

> 原文：<https://lifehacker.com/you-probably-arent-going-to-get-covid-19-from-a-package-1842397431>

社交距离意味着很多人会避开商店。取决于你住在哪里，这些商店甚至可能会暂时关闭。所有这一切都意味着，我们在网上购物的次数会比平时多一些。那些递送的包裹也伴随着一个相当大的问题:它们安全吗？



在包裹到达你家门口之前，很可能有相当多的人接触过它:打包它的人，拿起它去送货的人，沿途的卡车司机，最后是把它送到你家门口的人。

美国疾病预防控制中心研究了以前的冠状病毒 ，发现它们总体上“由于这些冠状病毒在表面的生存能力很差，在环境温度下经过几天或几周时间运输的产品或包装传播的风险很低。”

疾控中心不建议人们对包装进行消毒。

世界卫生组织(世卫组织)提供类似的建议 。当病例集中在中国时，它在疫情早些时候发表了这一声明:“从中国收到包裹的人没有感染新型冠状病毒的风险。”

我们对通过当地邮递员或隔夜邮件递送的包裹有疑问。 [据世卫组织网站](https://www.who.int/news-room/q-a-detail/q-a-coronaviruses) 报道，“感染者污染商品的可能性很低，从被移动、旅行、暴露在不同条件和温度下的包裹中感染导致新冠肺炎病毒的风险也很低。”

《纽约时报》， [报道了 2020 年 3 月 17 日星期二在](https://www.nytimes.com/2020/03/17/health/coronavirus-surfaces-aerosols.html) [【新英格兰医学杂志】](https://www.nejm.org/doi/full/10.1056/NEJMc2004973) 上发表的一项研究 ，称的研究“表明病毒在纸板上一天的时间内分解，减轻了消费者对快递会在在家呆着和工作期间传播病毒的担忧。”

UPS，FedEx 和 USPS 都发表声明，表示他们在所有交付中遵循 CDC 的指导方针，并将任何可能表现出任何流感样症状的司机或处理人员送回家。

在 [接受*商业内幕*](https://www.businessinsider.com/how-long-can-coronavirus-live-on-packages-2020-3) 采访时，宾夕法尼亚州立大学传染病动力学中心主任 Elizabeth McGraw 说:“如果我们通过包装传播，我们会在疫情爆发早期看到中国的即时全球传播……我们没有看到这一点，因此我认为风险非常低。”

她说，“我们知道，在我们暴露包装的条件下，包括温度和湿度的变化，病毒可能只能存活几个小时到几天，”这一立场得到了 CDC 的回应。

如果你真的很关心一个包裹(再说一次，你不应该)，那么你最好的做法是像平常一样打开它，靠近你的前门，或者甚至在你的门阶上。然后把包装拿到你的回收箱里，然后马上洗手。整个过程中最重要的部分是最后的洗手。

根据世卫组织的说法，“保护自己免受新冠肺炎病毒侵害的最好方法是经常洗手。这样你就消除了手上的病毒，避免了接触眼睛、嘴巴和鼻子时可能发生的感染。”
