# 通过这个 Z 世代词汇表学习新的青少年俚语

> 原文：<https://lifehacker.com/learn-new-teen-slang-through-this-gen-z-glossary-1834496709>

我不是一个试图了解俚语的人。我在杀人吗？

哦，你好，你好。我正在温习我的 Z 世代词汇，感谢一位名叫詹姆斯·卡拉汉的高中老师在推特上分享的[。显然，他一直在用电子表格记录他将从学生那里学到的新俚语。本周，他班上的一个孩子在推特上发布了它的截图，然后这条推特就火了，然后卡拉汉通过公开整个文件给了世界一份礼物。](https://twitter.com/lhs_sociology/status/1123689146382528512)

Watch

该列表按字母顺序排列，包括从“A force”(不必要地过度使用武力)到“yurrr/yerrr”(肯定地呼吁注意)的术语。有“mad mad/big mad”(非常疯狂)、“slaps”(高质量)和“sus”(可疑；黑幕)。共识似乎是这些定义令人印象深刻地准确(呃，*有效？一个年轻人在推特上写道:“每次我说一些&无法解释其实际含义的话，请参考这条推特。”* 

作为一个住在郊区的妈妈，我承认我从未感到如此不酷。我没有那么老，对吗？哦，上帝，我已经那么老了。我的一部分对此很满意——我很高兴坐在我的摇椅上(我真的有一把，而且是最好的)，幸福地忘记了“假发被抢走”的定义无论如何，保持青少年俚语的领先地位将永远是一场艰苦的战斗...我很累了。
