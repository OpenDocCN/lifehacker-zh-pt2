# 向这些来自国会图书馆的老式公共领域猫图片抛媚眼

> 原文：<https://lifehacker.com/ogle-these-vintage-public-domain-cat-pictures-from-the-1834779933>

国会图书馆拥有一个巨大的公共领域图像库，可以用于任何你想要的东西，包括商业用途。他们偶尔会从他们的 [数据库中挑选出超过 80 万张照片、印刷品和图画](https://www.loc.gov/photos/) 作为小型收藏。图书馆已经建立了 [28 个主题的图片收藏](https://www.loc.gov/free-to-use/) ，包括 [一个迷人的老式猫照片和图纸收藏](https://www.loc.gov/free-to-use/cats) 。这里有一些我们最喜欢的老猫。



“粗暴对待他们！“这张美军海报上说，不幸的是实际上并不是在招募猫。

在 [这张军事照片](https://www.loc.gov/resource/fsa.8c01765/) 中，一只爱达荷农场的猫喝着起泡的鲜奶:

[布伦希尔德](https://www.loc.gov/resource/ppmsca.51533/) 是一只打扮成维京人的 20 世纪 30 年代的猫，令人失望的是*不是*一张军事照片:

柯立芝家的猫 Tige， [曾经逃出白宫](https://www.loc.gov/resource/npcc.10856/) 在华盛顿特区游荡，直到被一个叫 N avy 警卫 Benjamin Fink 的人发现。

其他图书馆影像收藏还有 [视觉震撼海报](https://www.loc.gov/free-to-use/poster-parade/)[童书](https://www.loc.gov/free-to-use/classic-childrens-books)[《非洲裔女性变革者》](https://www.loc.gov/free-to-use/african-american-women-changemakers/) 和【狗狗】。你可以通过 [搜索图书馆的目录](https://www.loc.gov/photos/?q=cat) 找到上百张猫的图片。
