# 这周你听说了冠状病毒的什么？

> 原文：<https://lifehacker.com/what-have-you-heard-about-the-coronavirus-this-week-1842567263>

新冠肺炎的科学进展比以往任何一次爆发都要快:科学界在短短三个月内从发现病毒到开始疫苗试验。但是尽管有这样的速度，我们仍然不知道我们想知道的一切。无论科学家们在实验室里多么努力地研究一种病毒，我们都不可能预测它在世界范围内传播时会发生的一切。从来没有过这样的病毒，只有一种方法可以了解新疫情是什么样的。



但是我们都想知道更多。我们想知道它到底有多致命，它到底是从哪里来的，它到底是从哪里来的，我们到底需要和 T4 保持多大的距离，以及它感染我们或我们所爱的人的可能性有多大。

这意味着在我们想知道的和实际知道的之间有巨大的差距。各种各样的东西填补了这个空白。有一些真正的新闻和初步的科学发现，但它们只是我们好奇心的空桶中的一小滴。填补这一空白的其余部分是更不确定的有用性:观点、猜测、数学建模。医学专家的直觉，政治家的怀疑和希望。阴谋论。而且，很有可能，一些我们还没有意识到的精彩观察。

很明显，在新冠肺炎时代，从错误信息中整理信息比以往任何时候都更加棘手，但也比以往任何时候都更加重要。在 Lifehacker，我们通常避免报道初步的或看起来虚假或不可靠的信息——除非它非常受欢迎，然后我们会跳出来揭穿它。但是由于所有的不确定性，我们现在要采取不同的方法。

所以这就是为什么我们开始了一系列的周一公开主题，在这里你可以分享你最近遇到的事情。我们可以在这里讨论一点——如果你愿意，我会给你我自己的直觉——然后 Lifehacker 团队将利用这一讨论为我们本周的报道提供信息。

为了让你开始，这里有一些我最近遇到的事情:

*   几周前，有警告称布洛芬治疗新冠肺炎患者的发烧可能不安全。但是那个警告 [结果并不是基于临床证据](https://www.wired.com/story/the-ibuprofen-debate-reveals-the-danger-of-covid-19-rumors/) ，也没有被任何主要组织作为官方推荐。尽管如此，还是有很多人把它发给了我。我呼吁不要在 Lifehacker 上报道这一点，但当时我真的感到犹豫不决:证据不在那里，但我们应该讨论它在多大程度上是无根据的，以及它看起来似乎合理的原因吗？我希望人们不要在家里的药品(据我们所知)完全没问题的情况下涌入药店购买泰诺。
*   最近有很多关于公众建议不戴口罩是否是一个错误的讨论。《华盛顿邮报》最近的一篇观点文章 认为我们都应该用旧 t 恤给自己做面具。事实上，我现在正在写一篇关于织物口罩利弊的文章，但我认为这是一个微妙的话题。 [医用口罩供不应求](https://vitals.lifehacker.com/quit-buying-coronavirus-masks-you-dont-need-1841521105) ，需要留作医护人员和有医疗需求的人使用。织物口罩*不*被视为医院工作人员的 PPE(个人防护装备)。关于织物面具在现实生活中是否有用，科学上存在争议。(大量的实验室测试，很少对现实生活中的人群进行研究。)就我个人而言，我不打算戴面具，但我理解为什么要讨论这个问题。
*   在一些地方，医院的停车场和候诊室是空的。显然有一场运动(显然是在政治光谱的一边) [要求人们记录空虚](https://www.mediamatters.org/coronavirus-covid-19/fox-news-contributor-i-am-seeing-videos-twitter-empty-hospital-parking-lots) 以表明不堪重负的卫生系统的整个想法是一个骗局。但是医院的这些可见区域是空的，因为一个更加现实和明显的原因:非紧急程序被取消，因为 [疾病预防控制中心要求医疗机构让病人远离，除非绝对必要](https://www.cdc.gov/coronavirus/2019-ncov/healthcare-facilities/index.html) 。许多医院也取消或减少了探视政策。这在一定程度上是一种社会距离措施，在一定程度上是一种补偿医疗需求激增的方式，这种需求已经在一些地方出现，预计将在其他地方很快出现。

这些只是我认为值得讨论的许多故事中的三个，但最终比快速的“我们应该”或“我们不应该”建议更复杂。

所以现在我们想听听你的意见。你听说过哪些关于冠状病毒(或任何相关问题)的事情让你感到好奇？你对什么感到兴奋，害怕，渴望分享，或者厌倦地重复看到？如果你读过的一些东西让你想到“嗯，既然我知道了这些，也许我应该在我的生活中做一些不同的事情”，我们很想听听。
