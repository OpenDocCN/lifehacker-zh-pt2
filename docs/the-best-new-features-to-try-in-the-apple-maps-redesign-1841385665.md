# 苹果地图重新设计的最佳新功能

> 原文：<https://lifehacker.com/the-best-new-features-to-try-in-the-apple-maps-redesign-1841385665>

苹果地图带着一个全新的版本回来了，该公司希望这个版本比谷歌地图和应用程序世界的 Wazes 更有吸引力(和竞争力)。新的苹果地图不仅看起来更好，这要归功于苹果已经打包的关于建筑物、地标和地形的所有细节，而且苹果还在隐私方面加倍努力为你的地图体验。

如今苹果地图有很多让人喜欢的地方。先说它的准确性。阿图尔·格拉博斯基在 2018 年进行了一次非常令人讨厌的测试 ，让苹果的服务与 Waze 和谷歌地图竞争。在比较了 120 次旅行中每种地图服务提供的估计时间后，格拉博斯基发现苹果地图是估计时间最准确的——尽管谷歌地图在测试时，实际上比苹果地图和 Waze 略快一些。

苹果现在不再依赖第三方为苹果地图提供数据，而是创建自己的数据(这要归功于一直在全国稳定行驶的配备激光雷达的车队，以及卫星图像和其他来源)。据推测，这应该会提高苹果地图方向的准确性。不敢保证，但我至少会试一试——尤其是在任何大都市地区。

至于你能在苹果地图上找到的所有其他古怪的功能，这里是我们最喜欢的一些功能的快速汇总——适用于新手和回归的方向障碍用户。

### **设置您最喜爱的地点(和收藏)**

将你的家和工作添加到苹果地图是一件很容易的事情——这使得通过应用程序或对 Siri 快速大喊来获取方向变得很容易。但是你也可以加上你喜欢去的其他地方。点击苹果地图搜索栏右下方收藏夹下的“添加”,你会得到一个你经常去的建议地点列表。选择一个，或手动输入地址，然后您就可以自定义其标签，并选择一个联系人，以便在您开始导航到该地址时通知您。

一旦你有了几个最爱，最好的整理方法就是把它们都放进一个收藏里。这甚至可以是一些令人向往的事情:“例如，在 酒吧，我*想要* [喝酒”。在苹果地图中创建新收藏很容易——在“收藏”部分下查找“新收藏”链接。点击它，给你的收藏命名，点击“创建”，然后点击收藏开始添加位置。](https://skillet.lifehacker.com/how-to-get-a-bartenders-attention-without-being-a-jerk-1838220196) 

要和其他人分享你的收藏——可以说是酒吧常客——只需向上滑动并点击分享按钮。(你也可以在苹果地图的主屏幕上向左滑动特定的收藏来分享它。)在选择 iPhone 或 iPad 之前，一定要留意苹果的警告——使用联系人分享你的列表:

### **召唤~~街景~~远眺**

与谷歌的街景视图不同，没有简单的方法来查看苹果支持的所有区域——它的“我们的汽车驶过并拍下了这个特定位置的照片”功能。如果这是可用的，你可能需要放大一个区域才能实现，你会看到一个双筒望远镜的小图标出现在右上角。点击它，你会得到一张特定地点的照片，然后你可以像一个典型的 360 度图像一样平移。轻按或在地图周围移动手指以扭曲到新的位置——但是，只有那些以蓝色突出显示的位置，如果你缩小得太远，可能很难看到。

### **了解您的每日天气预报**

这是一个小技巧，但是很有用。当你在苹果地图中查看一个地点时，注意小的天气图标(和空气质量指数！)在右下角。长按该按钮可以调出该地区未来几个小时的天气预报，以及当前天气、预计最高和最低温度以及降雨几率的概览。

### **弄清楚如何使用公共交通**

如果你对一个地方不熟悉，或者只是想要一个关于你可以选择的所有公共交通工具的提醒，你所要做的就是点击苹果地图右上角的“I”图标。将你的视角从地图切换到公交，你会看到所有可供你选择的公共交通工具的小亮点。(不好意思，公交车。仅限地铁和铁路！)

### 去室内(虚拟)散步

当您在苹果地图上缩放时，您可能会注意到一些位置在地图上的名称下方有一个蓝色的“向内看”链接。点击它，你就可以“打开”它们，看里面是什么。例如，如果你在看一个购物中心，你可以看到它的布局——包括里面所有的商店，不管大楼有多高。虽然如果你能使用苹果地图的“环视”功能在*和*商场中漫步就更好了，但至少你下次迷路时不用去找那些奇怪的商场信息亭地图了。

### 让 Siri 静音(或召唤它)来帮助指引方向

一旦你选择了要去的地方，并在你的 iPhone 上加载了路线，从屏幕底部向上滑动你的手指——巨型地图下方的小条。你现在会看到一些额外的选项。您可以为您的旅行添加停靠点(加油站、吃饭的地方或咖啡店)，这也是您可以设定您希望 Siri 在旅行中帮助您多少的地方。

点击 Audio，你会看到四种不同的音量选项，Siri 计划在你的旅程中对你大喊:静音、低音量、正常音量和大音量。如果您不想让您的体验听起来很奇怪，您也可以使用此屏幕让 Siri 在她为您指路时自动暂停您的播客或有声读物。

要为 Siri 设置默认音量，你需要跳出地图应用，打开**设置**应用。点击**地图**，然后选择**驾驶&导航。**查找“导航音量”，并选择您喜欢的选项。

当你在苹果地图的设置中时，你也可以选择你希望苹果地图默认使用的交通类型:驾车、步行或公共交通。您也可以通过点击“交通”选项来关闭您不想让“地图”在行程规划过程中使用的公共交通选项。

此外，您可以使用设置在“苹果地图”中的英里和公里之间切换。你还可以关闭它的空气质量指数和天气图标，甚至可以通过记下你的车停在哪里来决定是否需要地图的帮助。(作为一个经常发现自己在停车场迷路的人，我不明白为什么你不想要那个功能。)

### 与朋友分享您的预计到达时间

苹果在 iOS 13 中添加了一个有趣的功能，即向联系人发送关于你旅行进度的自动更新——当然，这对约会之夜很有用。一旦你收到了某个地点的路线，并盯着你的旅行地图，从屏幕的“到达/结束”部分向上滑动。在那里，点击“分享 ETA”，并选择一个联系人。

当你分享某个地点的预计到达时间时，你的(iOS)联系人会收到一条短信，告诉他们你的预计到达时间，以及你要去的地点的名称。然后，他们可以点击位置来设置他们自己的方向或查看你在哪里——如果你们俩在某个地方见面，这很有用。这项功能对 Android 用户来说也是一样的，他们只是不能通过点击来调出地图。

### 进行虚拟飞行

苹果地图中的一些地点——主要是大城市——带有一个特殊的“天桥”功能，当你搜索它时可能会看到。例如，查找伊利诺伊州的芝加哥，点击“天桥”按钮，就可以看到这个城市美丽的 3D 视图。你可以通过移动手机或用手指做出收缩/张开的手势来操作你正在看的东西。我发现直接点击“开始城市之旅”按钮会更容易些，这样就不会那么容易晕车了。这几乎让我想起了微软的飞行模拟器。*差不多。*
