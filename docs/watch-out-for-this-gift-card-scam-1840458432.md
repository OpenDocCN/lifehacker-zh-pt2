# 小心这个礼品卡骗局

> 原文:[https://life hacker . com/watch-out-for-this-gift-card-scam-1840458432](https://lifehacker.com/watch-out-for-this-gift-card-scam-1840458432)

如果你还在寻找礼物，这就是为什么你可能要避免那些杂货店礼品卡，因为它们在你绝望的时候很诱人。科罗拉多泉报 的 [结束时，几名消费者报告说他们从沃尔玛和当地杂货店购买了礼品卡——却发现在离开商店后不久余额就被清空了。](https://gazette.com/news/voice-of-the-consumer-gift-cards-can-be-drained-in/article_08a241f2-1dfe-11ea-b593-e3ae61f6c0d2.html)

“我回到家，刮掉这些代码，在我的亚马逊账户上查看，他们说已经有人认领了，”一名送礼者告诉《公报》。"他们身上没有钱。"在这个特定客户的案例中，这意味着失去了价值 850 美元的亚马逊购买。

事实证明，礼品卡并非不会受到黑客攻击。骗局很简单:骗子从未激活的礼品卡上窃取信息(包括剥离或刮擦标签以显示其 PIN 并将其掩盖)，并使用零售商的网站监控卡的活动。一旦激活，骗子就会突袭，花光剩余的余额。

早在 2017 年，一名安全研究人员向 [Wired](https://www.wired.com/story/gift-card-hacks/) 详细介绍了一些精通技术的骗子如何甚至可以使用暴力破解技术，或试用- 和- 错误技术，来预测现有的没有 pin 的礼品卡的数量，因为一些零售商的卡可能只相差几个数字。使用这种技术，他们就可以在零售商的网站上核对每个号码的价值(如果卡号存在的话),然后在没有亲眼看到卡或卡号的情况下就能获得成功。

总之，根据美国联邦贸易委员会的说法，涉及购物卡的诈骗已经让受害者损失了 7400 万美元，随着圣诞节的临近，骗子们没有更好的时机下手了，因为他们知道那些杂货店的购物卡通道将被清理干净。如果你想通过购买礼品卡来避免被骗，有几个简单的事情要记住。首先，在离开商店之前，一定要检查卡是否有被篡改的迹象(例如，如果覆盖 PIN 的标签丢失或者看起来与商店中的其他卡不同)。把收据也留着，以防万一。

正如《洛杉矶时报》推荐的 ，你也应该尽可能坚持在网上购买礼品卡，而不是在商店购买，这可能会给你多一层安全保障。如果必须的话，一定要把卡放在架子后面或者柜台后面，以避免卡被损坏。请记住，有些礼品卡可能比其他礼品卡更容易受到黑客攻击。(我想你买 5 美元的星巴克卡可能比买 250 美元的百思买卡更安全。)

如果你最终被一个聪明的骗子盯上了，你应该立即联系那个零售商； [FTC](https://www.consumer.ftc.gov/articles/paying-scammers-gift-cards) 有几家零售商客户支持热线的号码，以防欺诈。在某些情况下，如果你行动迅速，你甚至可以拿回你的钱。
