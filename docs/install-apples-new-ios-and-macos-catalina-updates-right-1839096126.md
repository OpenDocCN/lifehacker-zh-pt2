# 马上安装苹果新的 iOS 13 和 Catalina 更新

> 原文:[https://life hacker . com/install-apple-new-IOs-and-MAC OS-catalina-updates-right-1839096126](https://lifehacker.com/install-apples-new-ios-and-macos-catalina-updates-right-1839096126)

苹果今天发布了 iOS 13 和 macOS Catalina 的新更新。如果你不喜欢那些阻止你做你想做的事情的错误或其他问题，比如完成 macOS Catalina 安装而不出错，你会想尽快更新你的 iPhone 和/或 Mac。

Watch

据苹果公司称，以下是该公司在最新版本的 **iOS** (现为 13.1.3)中修复的内容:

*   *解决了一个可能会阻止设备因来电而响铃或振动的问题*
*   *修复了可能无法在邮件*中打开会议邀请的问题
*   *解决了夏令时调整后健康应用中的数据可能无法正确显示的问题*
*   *修复了从 iCloud 备份恢复后语音备忘录录音可能无法下载的问题*
*   *解决了从 iCloud 备份恢复时应用程序可能无法下载的问题*
*   *修复了一个可能导致 Apple Watch 无法成功配对的问题*
*   *解决了 Apple Watch 上可能收不到通知的问题*
*   *修复了某些车辆上蓝牙可能断开的问题*
*   *提高蓝牙助听器和耳机的连接可靠性*
*   *解决使用 Game Center 的应用程序的启动性能问题*

换句话说，这是一个严格意义上的“修复错误”补丁；不要期望在运行更新时看到任何新功能(通过**设置>通用>软件更新**)。

*   *提高了 macOS Catalina 在低磁盘空间 MAC 上的安装可靠性*
*   *修复了在某些安装过程中设置助理无法完成的问题*
*   *解决了当多个 iCloud 帐户登录时无法接受 iCloud 条款和条件的问题*
*   *提高离线玩苹果街机游戏时保存游戏中心数据的可靠性*

当然，如果你已经成功安装了 macOS Catalina 更新，你可能不会太关心苹果正在修复的与安装过程相关的问题—[](https://lifehacker.com/if-your-macos-catalina-installation-gets-stuck-try-thi-1838980635)*。不过，如果你发现自己因为任何原因不得不重新安装操作系统，至少你不会遇到任何问题。* 

*要安装 macOS Catalina 更新，只需点击左上角的苹果标志，打开系统偏好设置，然后点击“软件更新”*
