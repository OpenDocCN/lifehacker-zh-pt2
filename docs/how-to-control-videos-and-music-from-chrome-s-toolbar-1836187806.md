# 如何从 Chrome 的工具栏控制视频和音乐

> 原文：<https://lifehacker.com/how-to-control-videos-and-music-from-chrome-s-toolbar-1836187806>

虽然 Chrome 已经让用户在个别标签 中的 [静音一段时间了，但谷歌目前正在浏览器的最新版本](https://lifehacker.com/how-to-mute-tabs-in-google-chrome-1798506886) [中测试新的媒体控制功能](https://www.zdnet.com/article/google-chrome-to-get-a-video-playpause-button-on-the-toolbar/) 。启用后，用户可以直接从浏览器的工具栏播放、暂停和跳过/回退音频和视频内容——例如，在您工作时静音(或启动) Spotify 非常方便，即使您没有看 Spotify 选项卡。

### 如何在 Chrome Canary 中打开全局媒体控件

如果不想处理金丝雀，可以 [用各种](https://chrome.google.com/webstore/detail/music-plus-for-google-pla/ipfnecmlncaiipncipkgijboddcdmego?hl=en-US) [Chrome 扩展](https://chrome.google.com/webstore/detail/amazon-music-hotkeys-cont/lnkimoaahjmlmbiafbfjdjdnmmoeecoo?hl=en-US) 获得类似的控件 ，但是它们的功能仅限于特定的 [应用和媒体](https://chrome.google.com/webstore/detail/universal-video-player-co/ppdnkejgcieghdpjnjokjeefbojbjdaa?hl=en-US) 。相比之下，Chrome 的新控件是通用的。它们对所有支持的媒体都是一样的，它们允许你暂停和播放来自许多网站和服务的媒体。

以下是如何试用 Chrome 的新媒体控件。

1.  从谷歌 下载最新的 [Chrome Canary build](https://www.google.com/chrome/canary/thank-you.html?statcb=1&installdataindex=empty)
2.  在你的地址栏中输入“chrome://flags/”并点击回车打开标志菜单
3.  在搜索栏中，搜索“全球媒体控制”标志
4.  将选项的下拉菜单设置为“启用”
5.  重新启动 Chrome Canary 以应用更改
6.  重新启动后，媒体控件会出现在 Chrome 的地址栏旁边。

所有关于未完成、测试版功能和软件的正常警告都适用于此:Chrome 可能会崩溃，播放控制可能不会对你使用的每个服务或网络应用都有效，但它仍然是一个超级有用的功能。我们很想知道它在最终的公开版本中会是什么样子。
