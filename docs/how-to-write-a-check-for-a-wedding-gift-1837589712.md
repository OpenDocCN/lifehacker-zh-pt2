# 如何为结婚礼物写一张支票

> 原文：<https://lifehacker.com/how-to-write-a-check-for-a-wedding-gift-1837589712>

在庆祝一对幸福夫妇婚姻的众多快乐中，隐藏着无数的失礼，正等着被你发现。其中之一是:写一张对新婚夫妇来说基本上无法使用的支票，这对新婚夫妇真的很想在他们的蜜月计划中增加几次短途旅行。虽然给双方都写一张支票感觉很特别，在中间夹上“和”，但实际上你可能会让他们更难存下这份慷慨的礼物。

Watch

Reddit 用户 [jabronie69](https://www.reddit.com/r/LifeProTips/comments/cu7uo9/lpt_when_gifting_a_check_to_a_newly_married/) 提供了这个提示:

> 当赠送支票给新婚夫妇时，在他们的名字之间用“或”代替“和”。他们可能还没有联名银行账户，或者还没有完成更名，所以这会让他们在银行感到头疼，你也不必重写。

事实上，这种写支票的方法让你避免了不止一次，而是两次的不幸。

首先，它避免了夫妇之间的麻烦，如果他们还没有这样做的话，他们不得不匆忙以他们两个人的名字开一个银行账户。虽然共用一个银行账户可能是常态，但这不是婚姻的必要条件！使用“或”,他们可以把现金放在任何他们想放的地方。

然后，还有“假设一方改变了他们的名字”的陷阱，这给任何没有这样做的人，或者没有在仪式后暂停四分钟排队领取新照片的人带来了麻烦。改名需要时间！他们头痛！真的，你不应该对是否有人用了别人的名字做任何假设。现在是 2019 年。

使用婚礼请柬上的相同名字，你会让新婚夫妇更容易享受你的礼物。
