# 用这个免费工具包了解黑人历史

> 原文:[https://life hacker . com/learn-about-black-history-with this-free-toolkit-1844118935](https://lifehacker.com/learn-about-black-history-with-this-free-toolkit-1844118935)

如果说过去的一个月教会了我们什么，那就是许多美国人在了解这个国家的黑人历史和系统性种族主义方面还有很长的路要走。这可能不是我们的错，我们在学校被教导了一个扯淡的美国历史版本，在这个版本中，我们赞美种族主义的白人历史人物，并假装当马丁·路德·金告诉我们他的梦想时种族主义消失了。但是作为成年人，如果我们现在不教育自己，那绝对是我们的错。

幸运的是，有很多好的、准确的、重要的内容供我们阅读、观看和学习。甚至*更幸运的*是，有像 [泰瑞·博伊德-佩特斯](http://www.tyreebp.com/) 这样的人愿意策划它，并把它直接交给我们。

公共历史学家兼博物馆馆长博伊德-佩特斯(Boyd-Pates)最近告诉《洛杉矶时报》 ，他创建了一个反种族主义工具包，他将其命名为“ [自由文件](https://drive.google.com/file/d/1Sr_0JLez1dVmAjhxvyGU_l9eiGjbjnG4/view?fbclid=IwAR0POhQ0DJ6QC-JA_xHEt6eTuOTAekxxhrDg2ztrnQub_8Zb7AsoIoEDB24) ”，发送到他的社交媒体网络和他们的盟友，以便他们能够更好地了解黑人政治教育。

> “从代际角度来说，这一代人的工具箱里有一套工具——上一代人没有的技术，”他说。“挑战白人霸权的方法是我们这一代人正在积极重新想象的东西，设想它可能超越体面的政治，以便被认为更有人情味。”

在大约 20 页的篇幅里，Boyd-Pates 为我们策划了一份基本信息清单，包括:

*   黑人历史里程碑的时间表
*   有影响力的演讲
*   书籍和文章
*   音乐播放列表
*   电影
*   成为盟友的准则

你可以在这里下载博伊德-佩特斯的 [自由论文工具包](https://drive.google.com/file/d/1Sr_0JLez1dVmAjhxvyGU_l9eiGjbjnG4/view?fbclid=IwAR0POhQ0DJ6QC-JA_xHEt6eTuOTAekxxhrDg2ztrnQub_8Zb7AsoIoEDB24) 和 [访问他的网站](http://www.tyreebp.com/) 订阅他的时事通讯了解更多他的工作。
