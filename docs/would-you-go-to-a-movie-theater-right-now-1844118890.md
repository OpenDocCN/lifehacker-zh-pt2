# 你现在会去电影院吗？

> 原文：<https://lifehacker.com/would-you-go-to-a-movie-theater-right-now-1844118890>

时间，空间，人，地点。随着企业开业，事件开始再次发生，我一直在思考这些因素。在一个封闭的空间里和很多人呆很长时间不是我想呆的地方【新冠肺炎风险谱】。对我来说，这意味着很长一段时间内没有电影院。

Watch

但是一些剧院正在开业。在风险谱上排名相当高的许多其他事情也是如此:一些教堂开始再次提供面对面的服务。总统举行了一次集会。许多地区的餐厅都开放室内用餐。

作为顾客，我个人对此不感兴趣。但我也担心这些地方的工人，因为他们容易感染我们所有的细菌。作为一名顾客，你可能会决定偶尔出去吃饭，作为一种享受。作为一个服务器，你不能决定你偶尔会工作。

因此，当我们看到更多走出家门的机会时，我们必须仔细权衡这些风险，因为其他人——公司、政府——不一定会以我们的最佳利益为重。 [开放的东西不代表安全](https://lifehacker.com/just-because-you-can-go-out-doesnt-mean-you-should-go-1843523899) 。

据《华盛顿邮报》报道，电影院正试图保持他们的安全措施微妙 ，因为他们担心如果人们不断被提醒电影院有多危险，他们会感到不舒服。嗯，也许就应该这样。

就我个人而言，我尽最大努力避免以下情况:

*   人群
*   无面具的人
*   唱歌或喊叫
*   室内聚会
*   长时间靠近他人

我要去购物。有一天，我可能会考虑在非高峰时间去健身房做一些远离社交的蹲起运动。我不会马上上飞机，也绝对不会在电影院坐两个小时。你呢？
