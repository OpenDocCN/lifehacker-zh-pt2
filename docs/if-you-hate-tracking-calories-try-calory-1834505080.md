# 如果你讨厌记录卡路里，试试“卡路里”

> 原文：<https://lifehacker.com/if-you-hate-tracking-calories-try-calory-1834505080>

我最讨厌任何一种减肥计划的部分是“我必须拿出手机，输入我吃的所有东西，以保持诚实”这一整方面。因为尽管我假装我能心算出我吃的食物，但我往往会忘记这里的一把葡萄干，那里的一根蛋白质棒，一罐微型可乐就这样神奇地出现在我的桌子上——诸如此类的事情。

Watch

虽然一些应用程序，如[**my fitnesspal**](https://itunes.apple.com/us/app/myfitnesspal/id341232718?mt=8)，可以非常容易地跟踪你正在消费的每样东西里面有什么，但他们起初可能会感到畏惧。如果你和我一样，需要慢慢适应计算卡路里(或碳水化合物)的新习惯，我推荐你看看 [**卡路里**](https://itunes.apple.com/us/app/calory/id1441252752?mt=8) 。这不是免费的，但这本身就是一个目标:与其在汽车餐厅买一瓶汽水，或者在电影院买半瓶汽水，你是在为你的健康投资 3 美元。

我最喜欢卡路里是它的友好。一旦你通过了应用程序的初始屏幕，这些屏幕会询问你的尺寸和体重目标(并要求你将应用程序与 HealthKit 等相关联)，你就会进入一个简单、易于处理的布局。中间的大圆圈代表你每天摄入的热量，当你在清单上填满食物时，这个区域就会被小点填满。就这么简单。

当你把食物输入你的肚子时，把它输入你的*应用*只需要你点击底部的大加号按钮。然后，如果你知道它们的热量，你可以快速计算(我想是通过阅读盒子)，或者你可以点击通过应用程序的设置菜单设置的预定义收藏夹列表。

如果你正在吃一些新的东西，你也可以使用卡路里的内置数据库(任何使用过卡路里跟踪应用程序的人都应该很熟悉这个数据库)来搜索常见的食物，如蔬菜、水果、肉类、饮料等。这款应用的一个缺点是，你不能扫描你正在吃的任何东西的条形码，然后直接从盒子里输入每一个营养成分，这一点将在更新中体现出来。(此外，当它到来时，这可能是一个“高级附加组件”。)

我喜欢 Calory 的另一点是，该应用程序与 Siri 快捷方式配合得非常好——通过对着手机大喊，在咀嚼中，而不是试图用两个没有沾满酱汁的笨拙手指拉起来，可以很容易地将你刚刚吃的披萨添加到 Calory 中。你在应用程序中定制的任何喜爱的“盘子”都会立即成为 Siri 快捷方式，基于它们的名称——例如，添加圣代-那对我不好，”如果你在应用程序中是这样称呼它的话。

你也可以使用 Siri 快捷方式来快速查看你还剩多少卡路里(或者看一眼你的 Apple Watch，如果你也启用了该应用的话)。

有了大量的颜色定制选项，黑暗模式，甚至是一个可调的应用程序图标，Calory 感觉像是一种更友好的方式，让你自己享受跟踪你吃的一切的快乐。如果你所关心的只是那个大的圆形“卡路里”数字，那么你很难忽视一个巨大的可视化表示，即你每天可以吃多少。(对于你们当中的数据爱好者来说，卡路里甚至可以让你按日、周、月、年查看你跟踪的卡路里——与你的体重相互参照，这样你就可以看到你的饮食是否真的在起作用。)
