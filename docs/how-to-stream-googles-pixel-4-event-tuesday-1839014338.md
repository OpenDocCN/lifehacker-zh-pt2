# 如何在周二直播谷歌的 Pixel 4 活动

> 原文:[https://life hacker . com/how-to-stream-Google-pixel-4-event-Tuesday-1839014338](https://lifehacker.com/how-to-stream-googles-pixel-4-event-tuesday-1839014338)

周二，谷歌将发布 Pixel 4。也许是历史上泄露最多的手机之一，如果你对这款设备感兴趣，你可能已经对它有所了解。

Watch

也就是说，谷歌将于周二上午在纽约市的一场活动中证实所有这些传言(或者抛出一个曲线球，证明它们都是错的)。公告预计在美国东部时间上午 10 点开始，如果你在英国，就是太平洋时间上午 7 点和下午 3 点。

如果你想现场观看公告，但没有获得活动邀请，谷歌将在 YouTube 上播放 [整个公告。](https://www.youtube.com/watch?v=XKmsYB54zBk&feature=youtu.be) 

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-XKmsYB54zBk&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-XKmsYB54zBk&start=0) 

至于期待什么，预计 Pixel 4 将拥有全新的面部解锁功能，以及运动感应雷达和第二个后置摄像头。这款手机预计将有两种不同的尺寸，有一个可以自动转录录音的录音机和一个当它检测到你发生车祸时会自动呼叫紧急服务的新功能，以及其他功能。

除了 Pixel 4，我们还可能会看到下一代谷歌助手、新的 Pixel Buds 和新版谷歌 PixelBook 笔记本电脑的发布。我们还可能会看到谷歌网状路由器系统 Nest Wifi 的新版本，或许还有谷歌 Home Mini 的新版本。

一如既往，请务必在公告发布后查看这里以及我们的好朋友 Gizmodo 以获取更多关于所有已发布消息的信息。
