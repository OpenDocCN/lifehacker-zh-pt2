# 结束争论的最快方法

> 原文:[https://life hacker . com/the-fast-way-to-end-a-argument-1839719076](https://lifehacker.com/the-fastest-way-to-end-an-argument-1839719076)

有时候，当有人就是不肯闭嘴时，直截了当地打断他们是让他们插话的唯一方法——当这不起作用时，诚实也起作用。( [)下面是如何做到这一点。](https://lifehacker.com/how-to-get-someone-to-shut-up-1838906639) )

但是当你和某人 争论，而对方甚至拒绝承认你的想法时，你该怎么办呢？嗯，即使面对冷酷无情的事实，有些人还是不愿意听真话。当你和这种人打交道时，完全避免谈话往往比全面争吵更容易。

我们最近写了一些避免不愉快对话的方法，特别是当它 [涉及政治](https://lifehacker.com/how-to-avoid-awkward-political-conversations-1839032773) 时，以及与 a 洞 争论 [的正确方法，但这里有一个小小的黑客，由 Reddit 用户](https://lifehacker.com/the-right-way-to-argue-with-an-asshole-1838104824) [u/nfhii](https://www.reddit.com/r/LifeProTips/comments/dt9ve0/lpt_saying_fair_enough_is_the_quickest_way_to_end/) 提供。如果你陷入了一场激烈的争论，而你想退出，这里有一个你听过无数次的短语，你可以用它来回应他们提出的任何观点:“很公平。”

它之所以有效，是因为它简单；你是在承认他们，而不是暗示他们更多的信息。“‘很公平’表明你理解了他们的观点，争论得到了解决，”[u/toesandmoretoes](https://www.reddit.com/r/LifeProTips/comments/dt9ve0/lpt_saying_fair_enough_is_the_quickest_way_to_end/f6vse2w?utm_source=share&utm_medium=web2x)写道。“‘求同存异’表明你根本不在乎寻找/教授真相，只想让争论停止。”换句话说，任何其他措辞可能只会火上浇油。

但是如果他们完全错了，他们的观点一点也不公平，那该怎么办呢？好吧，信息是一样的。承认你至少已经*听过*了，如果你没心情吵架的话，就迅速转移到手头的话题之外的话题。(或者把这个建议作为一个 [邪恶黑客](https://lifehacker.com/c/evil-week) 让自己远离任何谈话，句号。)

当所有其他方法都失败了，或者他们的观点与你的不一致，没有办法调和两者时，那么就诚实一点，当你感到不舒服的时候就离开。(或者迅速顺从 [一个](https://offspring.lifehacker.com/use-your-kid-to-get-you-out-of-stuff-you-dont-want-to-d-1839412476)[我们的](https://lifehacker.com/excuses-to-get-out-of-plans-last-minute-ranked-1839502154) [许多](https://lifehacker.com/how-to-cancel-plans-without-pissing-someone-off-1826194573) [借口](https://lifehacker.com/how-to-politely-exit-a-conversation-1839592512) 来迅速退出任何尴尬的社交设定。)
