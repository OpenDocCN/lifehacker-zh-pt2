# 没有人需要热胶鞋

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-dvdx4fHkbWI&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-dvdx4fHkbWI&start=0) 

上面的黑客，在很大程度上，是可怕的。你知道它们很可怕，每个人都知道它们很可怕，可能制作它们的人也知道它们很可怕。但它们太受欢迎了，以至于这个 20 分钟的汇编，收集了来自同一个 YouTube 频道的以前的视频，在四天内获得了超过 300 万次观看。所以去吧，放纵自己。

Watch

有很多 Instagram、脸书和 YouTube 频道充斥着这样的糟糕黑客(偶尔也有好的黑客)。我报道过其他的，像 [一系列糟糕的可口可乐黑客](https://lifehacker.com/these-coca-cola-hacks-are-as-pointless-as-they-are-ente-1827480241) ，用明亮的颜色拍摄，用看起来不可能的缩略图营销，并得出结论，整个“黑客”或“DIY”或“工艺”方面只是一种营销工具，这些视频的创作者不可能想象他们有有用的活动。

这些视频是预算更高的版本【the chum box】，这是一组令人震惊的“网络”链接，像丁果果一样挂在新闻文章的底部，承诺向你展示哪些名人衰老不佳，或者一位母亲治愈皱纹的奇迹。我也写过关于 [那些链接](https://kernelmag.dailydot.com/issue-sections/staff-editorials/10329/around-the-web-content-recommendation/) 的恐怖之处，以及如何屏蔽它们。但你不需要屏蔽糟糕的黑客视频。

相反，你，我，和每个人，我们需要看着他们。我们需要狼吞虎咽。我们需要制作模仿和回应视频，就像詹娜·马博讽刺地制作热熔胶人字拖，穿着它们，观察“感觉就像站在柔软的乐高积木上。”我们需要从这些黑客中获取#内容。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-k4VzWKMc38k&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-k4VzWKMc38k&start=0) 

因为那样的话，糟糕的黑客们就不再是这个安静的行业，他们吐出垃圾，吸走广告收入。那样的话，糟糕的黑客技术就会成为一种趋势，一种时尚。每个人都承认他们，赞美他们，在嘲笑他们中建立一个副业。因为听起来很有趣。所有有趣的事情都被毁了。

每一个好的迷因最终都会隐退到脸书，在那里老年人可以错误地使用它。每一个好的 GIF 都死了，被埋在一个特朗普推特回复线程的墓地里。庆祝一件好事会毁了它。所以让我们把糟糕的黑客变成好事。这是他们唯一的死法。
