# 为什么一些 USB-C 耳机连接器不工作

> 原文：<https://lifehacker.com/why-some-usb-c-headphone-dongles-dont-work-1836316524>

自从苹果抛弃了 iPhone 的耳机插孔，让你的旧耳机与新智能手机兼容就成了一个棘手的问题。许多 Android 机型仍然有 3.5 毫米音频插孔，但一些机型，如三星即将推出的 Galaxy Note 10，已经跟随苹果的脚步，转向一个单一的连接选项:USB-C

你仍然可以将耳机插入这些设备，但这样做需要一个适配器——一个*转换器*——将你的充电端口变成 3.5 毫米耳机插孔。然而，症结就在这里。并非所有的加密狗都是一样的，你刚刚从哪里买的廉价适配器可能根本不允许你从你的设备上获取音频。

### USB-C 耳机连接器并不完全相同

USB-C 加密狗的一个大问题是，在智能手机开始放弃耳机插孔之前，没有人制定新的硬件标准。

通过耳机播放数字媒体需要一个数模转换器(DAC)和一个放大器，DAC 将数字音频数据转换为耳机可以播放的模拟信号。分歧在于，这种转换应该发生在智能手机本身，还是发生在你所连接的适配器内。

当一些制造商将 DAC 从他们的适配器中取出，或者放入一堆不能很好地与所有设备配合的专有电路中时，你将在你的耳机中得到一大块空白。

目前，只有一小部分 USB-C-to-3.5mm 适配器包含 DAC，包含 DAC 的适配器可能比不包含 DAC 的适配器更贵。

### 如何找到合适的适配器

所有可以通过 USB-C 传输音频的智能手机都可以使用所谓的有源电缆，即内置 DAC 和放大器的电缆。智能手机*也可以*用无源线缆做这个同样的过程，但是 T3 这个是可选的。当有疑问时，你最好为你的设备选择一个活动的适配器。

如果你知道要找什么，找到合适的适配器会很容易。购买 USB-C 耳机适配器时， [查看 DAC](https://www.amazon.com/HIDIZS-Portable-Headphone-Amplifier-Smartphone/dp/B07F2DTTGT?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/why-some-usb-c-headphone-dongles-dont-work-1836316524&asc_source=&tag=kinjalifehackerlink-20) 或 USB-IF 认证(仅授予内置 DAC 的适配器)的产品说明。如果您在适配器的手册或描述中找不到提到的任何一个，您至少要做一点功课，看看其他人是否成功地将适配器连接到您拥有的同一设备。

如果你甚至不想处理这个，你可以选择一副像样的蓝牙耳机。当然，你必须解决电池寿命和充电问题，但至少你可以毫无问题地工作。
