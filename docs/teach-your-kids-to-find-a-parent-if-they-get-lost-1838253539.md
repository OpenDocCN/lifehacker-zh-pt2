# 如果你的孩子迷路了，教他们“找父母”

> 原文：<https://lifehacker.com/teach-your-kids-to-find-a-parent-if-they-get-lost-1838253539>

我儿子四岁时，我和丈夫带着他和他当时两岁的妹妹去佛罗里达海滩玩了一天。场景完美无缺:阳光明媚，我们设法记住了防晒霜和沙滩玩具，两个孩子都满足于在我们的沙滩毯前建造沙堡。当我丈夫提出带孩子们去海滩散步时，我很高兴能有一点时间来赶上我的海滩阅读。

Watch

当我丈夫回来时，我正站在水边。两个孩子朝我走来，我用眼角的余光看到我儿子从我身边走过。我以为他去了我们的海滩。

但他没有。

我们不知道的是，当我们聊着去哪里吃午饭时，他一直沿着水边朝另一个方向走去，没有意识到他的父亲已经停下来和我说话。

一瞬间，我们的生活改变了。我们在日光浴和沙滩玩耍的海洋中找不到我们的儿子。随着恐惧在我胸中升起，我开始为他尖叫，谢天谢地，其他去海滩的人很快跳进来帮助我们寻找。

我们最终在海边找到了他，和一对慈祥的老夫妇坐在一起，手里拿着一个巨大的冰淇淋甜筒，这是他们意识到他找不到父母而惊慌失措时给他买的。我们感激地(含着眼泪)感谢他们，当我问他当他意识到我们不在时做了什么时，他说，“我不知道该找谁帮忙，妈妈。”

我意识到我需要给我的儿子一个具体的方法来寻求帮助，以防他再次与我们分离。

虽然我们经常告诉我们的小孩，如果他们迷路了，就去找警察，但事实是，在杂货店的过道里或拥挤的学校运动会上，警察并不容易找到。对于一些孩子来说，军官制服可能看起来令人害怕或令人不知所措。

我需要给我儿子一个他能理解的参考点，即使在他很小的时候。所以我告诉他，如果他再次迷路或与我走散，他应该找一个推着婴儿车的妈妈或爸爸(或另一个推着购物车带着小孩的父母)。

小孩子都认识到，婴儿车通常由带着小孩的妈妈或爸爸陪伴。婴儿车很容易被发现，对于小孩子来说，它通常是一个舒适熟悉的景象。

作为一个在游泳池、杂货店和社区活动中帮助过受惊孩子的母亲，我也知道，我的婴儿车总是装有擦手纸、额外的瓶装水、安慰玩具、书籍和其他可以帮助受惊孩子的物品。

当然，没有哪个父母希望自己的孩子在百货商店里走失，但事实是孩子很快——在一瞬间，他们可能会走失，真正迷路。

告诉你的孩子，如果他们害怕或独自一人，就找一辆婴儿车，这将有助于他们在可怕的几分钟内集中注意力。万一你不能很快找到你的孩子，推着婴儿车的父母总是知道该怎么做来帮助你的孩子找到一个安全的地方等你。

这种策略也适用于成年人。

最近，我参加了一个商务会议，我发现很难与与会者沟通。在女洗手间的时候，我注意到，你猜对了，一个推着婴儿车的妈妈正在努力给她的小宝宝换尿布。我微笑着，主动提出在她——一位与会者——洗手和清理更衣室时逗逗这个婴儿。

走出浴室后，我们意识到我们都要去酒店餐厅，她笑着说，“你愿意和我一起吃午饭吗？”

结果是我们都感到有点失落，看到一位推着婴儿车的妈妈，我从未如此感激。
