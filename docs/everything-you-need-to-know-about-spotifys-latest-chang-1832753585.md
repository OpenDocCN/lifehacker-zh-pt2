# 关于 Spotify 的最新变化，你需要知道的一切

> 原文：<https://lifehacker.com/everything-you-need-to-know-about-spotifys-latest-chang-1832753585>

最近，Spotify 更新了它的界面，把它的“重复”按钮埋在它的移动应用程序的子菜单深处——人们很生气。Redditors 和 [Twitter](https://twitter.com/search?q=spotify%20repeat&src=typd) 用户都在互联网上表达他们的不满。



“我已经支付了多年的保费，如果这是永久性的，我可能会停止，”Redditor u/[ChummySquash](https://www.reddit.com/r/spotify/comments/aseivi/dear_spotify_if_it_aint_broke_dont_fix_it/)写道。"隐藏重复播放和随机播放是你对音乐应用程序所能做的最糟糕的事情."

但是你怎样才能再次找到重复按钮呢？最重要的是，你为什么要这样对我们，Spotify？你需要知道的是:

### 我现在怎么复读歌曲？

这并不复杂，但确实很烦人。你不能在正常的“正在播放”屏幕下重复播放一首歌曲，你必须按右上角的三个虚线按钮。在这里，您会发现歌曲和表演者播放下方的“重复”选项。

### 还有其他变化吗？

他们还将“进入队列”选项移到了同一个子菜单中。如果你想知道你的播放列表上有什么，你必须在那里找到它。除了重复选项，你还可以在“正在播放”屏幕和子菜单上找到“分享”选项，这样你就可以在 WhatsApp、Tweet 或 Instagram 上向关注你的朋友分享你的音乐品味(谁不想知道你一直在听呢？？)

### 他们为什么要这样对我？

因为他们可以。很明显，他们比以往任何时候都更重视与朋友和家人分享您的内容，而不是您的收听体验质量。“我们一直在测试新产品和新功能，目前没有什么可以分享的，”Spotify 就其变化告诉 [Verge](https://www.theverge.com/2019/2/19/18230222/spotifys-new-interface-repeat-button-user-complaint) 。

### 我能阻止它发生吗？

不。改变已经发生了。在 IOS 上测试我的 Spotify 应用程序时，重复按钮暂时再次出现，然后又很快消失了(这让我对他们重新考虑自己的决定抱有一丝希望)。目前，你可以在 Spotify 社区的版块中投票支持一个帖子，要求撤销这些更改。
