# 你不需要马上变得富有成效

> 原文：<https://lifehacker.com/you-dont-have-to-be-productive-right-now-1842378468>

我们被告知，艾萨克·牛顿在一个瘟疫年 完成了他最好的作品 [。当剧院关门时，莎士比亚可能已经把](https://www.washingtonpost.com/history/2020/03/12/during-pandemic-isaac-newton-had-work-home-too-he-used-time-wisely/) [变成了十四行诗](https://www.theatlantic.com/culture/archive/2020/03/broadway-shutdown-could-be-good-theater-coronavirus/607993/) 。现在你认识的每一个人都在家里工作(或者不工作)的时候从事某种项目。但是你知道你还有什么选择吗？*不* *做任何那种事。*

Watch

当你周围的商店关门，感染人数攀升的时候，呆在家里是一种奇怪的、前所未有的、可怕的情况。如果你不知道你的工作或整个社会发生了什么，感到有点压力是正常的？也许，我知道这是一个疯狂的想法，对自己宽容点？

我成年后几乎一直在家工作，我可以告诉你，即使在最好的情况下，这也不会让我有太多的空闲时间。当然，我没有通勤，但这意味着我错过了一个小时的阅读或听播客的时间，而我的办公室朋友认为这是理所当然的。这些天来，随着社交距离的完全生效，我的孩子和我一起在家。有更多的清洁工作要做，更多的争吵要分手，我和我的配偶要重新协商谁在做什么 [脑力劳动](https://lifehacker.com/how-to-share-the-mental-load-of-chores-with-your-part-1795657878) 。

我想，既然孩子们的活动已经取消，我就不用带他们去了，这样可以节省一些晚上的时间，但我也会在下午 6 点从家里的办公室出来，在完成工作的同时努力跟上新闻，这让我身心俱疲。让我休息一会儿，放松一下。

这不是暑假。如果你不是一个上流社会的、自我陶醉的大学生，没有人为你做家务(咳咳，牛顿)，没有理由期待这是一个放松的、富有成效的时间。如果投身于一个爱好或项目能帮助你应对压力，那太好了！潜开！但是你绝对没有义务利用这段困难的时间，也没有义务为“浪费”这段时间而感到内疚。照顾好自己，让自己摆脱困境。
