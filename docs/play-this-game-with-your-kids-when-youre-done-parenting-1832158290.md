# 当你不再为人父母时，和你的孩子玩这个游戏

> 原文:[https://life hacker . com/play-this-game-with your-kids-when you-done-parenting-1832158290](https://lifehacker.com/play-this-game-with-your-kids-when-youre-done-parenting-1832158290)

有时候你只是需要休息一下。也许一个孩子让你整夜睡不着觉，或者在最近的历史中，在你醒着的时候，每个小时都有一个孩子吊在你的腿上。也许你已经不知道如何让你的孩子多享受一秒钟，你真正想做的就是躺下。

Watch

这时 [作家兼播客希拉里·弗兰克](https://offspring.lifehacker.com/im-author-and-podcaster-hillary-frank-and-this-is-how-1831839109#_ga=2.239880448.2043571648.1548763754-224320171.1513692312) 说你应该播放“我屁股上是什么？”游戏。

弗兰克最近在 NPR 节目《新鲜空气 上接受了特里·格罗斯的采访，谈及她的新书 [怪异育儿胜](https://longestshortesttime.com/books) 。当格罗斯问弗兰克，在她仍在使用的书籍研究中，她是否遇到过任何特别有用的育儿建议时，弗兰克提出了以下建议:

> 弗兰克:“有一个妈妈编的游戏叫‘我屁股上是什么？’你玩的方式是当你觉得你需要休息，而你的孩子想玩的时候，你面朝下躺在沙发上。你告诉你的孩子在房子里随便找一个东西，放在你的屁股上，然后你要猜它是什么。我和女儿一起玩这个，它让我补充能量。"
> 
> 格罗斯:“你猜对了吗？”

> 弗兰克:“不，不可能猜对。”

她的女儿不仅仍然喜欢玩这个游戏——她还负责在最后收拾好所有的物品。

另一个建议是，如果你觉得自己今天/本周/本月已经完全厌倦了为人父母这件事，去看看 Reddit 上这篇关于意外厌恶为人父母的宣泄帖 。这个可怜的家伙有一对 8 个月大的双胞胎，支持和同情的评论会让你感到温暖和模糊。
