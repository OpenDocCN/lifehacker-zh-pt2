# 看《网飞》的时候学一门新语言

> 原文:[https://life hacker . com/learn-a-new-language-while-watch-网飞-1832564649](https://lifehacker.com/learn-a-new-language-while-watching-netflix-1832564649)

试图学习一门新的语言，但找不到时间？给你的疯狂观看一个全新的目的，在马拉松式地观看网飞表演或电影时，学习日语、西班牙语或德语的基础知识。谁说网飞的时间必须是无产出的？

Watch

使用谷歌浏览器扩展， [语言学习与网飞](https://languagelearningwithnetflix.com) ，你现在可以使用网飞作为翻译。它是如何工作的？下载 [扩展](https://languagelearningwithnetflix.com) 后，当电影或节目开始时，您选择您感兴趣学习的语言的字幕。

在电影的下面，对话会出现在你正在学习的语言中，同时会有你已经会说的语言的翻译。如果您将鼠标悬停在一个单词上，您还会看到它的各种定义(就像阅读各种字典条目一样)。从这里开始，你可以阅读整部电影或电视节目的翻译。

如果你想成为这门语言的大师，还有一些额外的特性。如果你是初学者或中级，你可以选择一个适合你的词汇水平(作为初学者，它会把你可以跳过的复杂词汇变灰，以支持基本的、简单的词汇)。而如果进展太快，你可以放慢字幕速度，甚至自动播放每个字幕，这可能会让你的电影时间增加几个小时(或者几天？).如果你不确定看什么，但仍然渴望学习一门新语言，还有一个推荐高质量电影字幕的 [目录](https://languagelearningwithnetflix.com/catalogue.html#language=English&country=) 。

毫无疑问，它是为那些在任何给定语言方面有一些专业知识的人准备的工具。作为一个初学者，单独使用这个扩展是很困难的。它也受到网飞可用字幕语言的限制。

这也是一个很大的干扰。当你忙于一遍又一遍地重复台词或在上面徘徊时，你可能会失去对故事情节的关注。我建议用它来看你已经看过的节目和电影，而不是你应该去看的那部新的、时髦的网飞原著。然而，如果你愿意花时间，这是一个非常方便的工具(可能只会让你对网飞上瘾)。

你可以用网飞的 [主页](https://languagelearningwithnetflix.com/) 下载关于语言学习的 Chrome 扩展。
