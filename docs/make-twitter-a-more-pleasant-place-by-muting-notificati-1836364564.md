# 通过屏蔽某些账户的通知，让 Twitter 成为一个更令人愉快的地方

> 原文:[https://life hacker . com/make-Twitter-a-more-pleasant-place-by-muting-notificati-1836364564](https://lifehacker.com/make-twitter-a-more-pleasant-place-by-muting-notificati-1836364564)

Twitter 是一个与世界各地的人们联系的好地方。它也可能是机器人和垃圾账户向你发送垃圾推文的好地方，你希望你一开始就没有看到这些推文。

Watch

你可能没有意识到的是，你可以过滤你在服务上看到的通知，特别是在某种程度上，从那些唯一目的是欺骗你的人那里清除大量的机器人和帐户。

要找到它们，进入 Twitter 的设置菜单，然后选择通知。在那里，您可以将未关注您的人、您未关注的人、拥有新帐户的人、拥有默认个人资料照片的人以及尚未确认其电子邮件地址或电话号码的人的通知静音。

我个人选择只让新账号和默认头像的人静音。我在 Twitter 上与我不关注的人和那些不关注我的人进行了很多精彩的对话，所以我个人认为没有理由让这些人保持沉默。确认电子邮件或电话号码也是如此。我不确定这是否会破坏对话。

至于刚开户还没选好个人资料图片的人？这些人十有八九会是我不想与之交流的人，也有可能是为了成为巨魔而创建账户的人。删除这些内容可以减少很多混乱，并且可以确保你在通知中看到的确实是你想要看到的。
