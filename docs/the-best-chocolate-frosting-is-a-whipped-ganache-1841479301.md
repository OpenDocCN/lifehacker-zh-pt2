# 最好的巧克力糖霜是搅打过的 Ganache

> 原文：<https://lifehacker.com/the-best-chocolate-frosting-is-a-whipped-ganache-1841479301>

如果我觉得在做蛋糕的过程中不得不偷工减料，我更有可能 [伪造蛋糕](https://lifehacker.com/how-to-make-a-boxed-cake-mix-taste-homemade-1839165427) 而不是糖霜。商店里买的糖霜比盒装的混合蛋糕更容易被发现，而且糖霜是一个更容易的项目——你只需要搅拌一个 ganache。

你可能知道，Ganache 是由等量的奶油和巧克力混合而成的。你把热奶油倒在切碎的巧克力上，它会融化，然后你把这种丝滑的物质倒在任何你认为会从浓郁光滑的巧克力中受益的东西上。ganache 通常会提供一种与糖霜不同的效果，但是你可以稍微搅拌一下，让它变得蓬松、易涂抹。(这对我来说太棒了，因为大多数自制的糖霜需要将黄油放出来，这样它就可以变暖到室温，而这在基因水平上我似乎做不到。)

首先，用等量的奶油和巧克力(牛奶或黑巧克力)制作一个标准的 ganache。想赚多少就赚多少！(每种 8 盎司足够你吃一层蛋糕了。)将巧克力切碎，放入立式搅拌器的碗中(如果你打算使用手动搅拌器，则放入搅拌碗中)。在调味汁锅中加热奶油，直到边缘开始形成泡沫，然后将其倒在巧克力上，搅拌直至光滑。加入一撮盐，再次搅拌，然后将碗放入冰浴中，偶尔搅拌直到冷却(大约一小时)。冷却后，用搅拌器在中火上搅拌，直到看起来像糖霜。立即使用。
