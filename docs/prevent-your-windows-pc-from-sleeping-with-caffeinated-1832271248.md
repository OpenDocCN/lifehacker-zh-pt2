# 防止你的 Windows 电脑带着咖啡因睡觉

> 原文:[https://life hacker . com/prevent-your-windows-PC-from-sleeping-with-caffeinated-1832271248](https://lifehacker.com/prevent-your-windows-pc-from-sleeping-with-caffeinated-1832271248)

我对 Mac 应用程序 [**安非他命**](https://lifehacker.com/macos-app-of-the-week-amphetamine-1825108890) 有着极大的热爱，因为它是防止你的 MacBook 因为任何特殊原因而睡眠(或启动屏幕保护程序)的好方法。我很惭愧地承认，我花了一年时间，如果不是几年的话，才意识到我没有在我的 Windows 桌面和笔记本电脑上安装类似的应用程序。

Watch

这款名为 [**含咖啡因**](http://desmondbrand.com/caffeinated/) 的应用只在 Windows 7 上测试过——其作者声称——但过去一个月我一直在 Windows 10 上使用它。它完美地工作。下载并解压这个小程序，你会得到一个可执行文件。把它藏在某个文件夹里，在你的开始菜单或桌面上创建一个快捷方式(以便于访问)。

当你启动这个应用程序时，它会在你的任务栏上弹出一个咖啡杯的小图标。您还会看到一个方便的弹出窗口，详细介绍应用程序的一些设置:

默认情况下，简单地加载应用程序不会打开咖啡因。如果你在 Windows 中设置了屏保和睡眠，你的系统仍然会屏保和睡眠。如果你没有勾选 Caffeinated 的设置窗口中的“启动时激活”按钮，你必须右键单击图标，告诉应用程序你希望果汁流动多长时间(时间范围从五分钟到无限期)。

无论你的选择是什么，咖啡因会让你的系统保持警觉，并在你设定的时间内保持活跃。如果您不希望在进行演示或离开办公桌一小会儿时触发系统的电源选项，这是一个非常有用的小工具。诚然，安非他命的触发功能更强大，但咖啡因——就像一杯快速浓缩咖啡——可以毫不费力地完成任务。
