# 剥烤辣椒最简单的方法

> 原文：<https://lifehacker.com/the-easiest-way-to-peel-roasted-peppers-1845128996>

烤红辣椒丝滑香甜，但是坚韧、片状、烧焦的果皮，可以说，并没有给餐桌带来多少东西。而且，即使它们没有那么坚韧的薄皮，它们也经常会剥落，碎成烧焦的植物物质的小黑点，你可能不会觉得特别有吸引力。幸运的是，烤辣椒非常容易剥皮——你只需要把它们盖上一会儿。

Watch

你用什么覆盖它们取决于你自己。我通常把它们扔在碗里，用保鲜膜密封起来，但封闭的纸袋也很好，带盖的罐子或特百惠类容器也一样。一旦它们被盖住，辣椒会自己蒸发，这样会使皮肤变松，让你可以用手擦掉，或者如果你担心你的手被弄脏，可以用纸巾擦掉。(如果你想捕捉那种超级烟熏味，你可以在一碗水中将它们去皮，然后像使用 [烟熏肉](https://lifehacker.com/an-easy-way-to-peel-charred-peppers-that-leaves-you-wit-1791441814) 一样使用那碗水。)

他们不需要被覆盖那么长时间；你只是想让他们冷静下来。对于我这个周末烤的两个波布拉诺，15 分钟就足够了，尽管如果你选择了整个啄(将近 37 杯辣椒)，你可能需要半个小时。

一旦它们被剥开，就可以吃了。将它们切碎，加入蘸酱、沙拉、沙拉、烤土豆或任何你认为会受益于它们的甜味和烟熏味的食物中。(这个周末我用我的烤波布拉诺做了一个烤波布拉诺荷兰酱，这是我强烈推荐的。在淋入融化的黄油之前，将切碎的辣椒和几个蛋黄在食品加工机中混合。)
