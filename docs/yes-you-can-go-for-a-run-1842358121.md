# 是的，你可以去跑步

> 原文：<https://lifehacker.com/yes-you-can-go-for-a-run-1842358121>

呆在家里。这是 [自我隔离](https://vitals.lifehacker.com/what-does-self-quarantine-actually-mean-1842268976) 和社交距离的第一法则。但是过一会儿你可能会渴望出去走走，我有好消息告诉你:如果你能独自完成所有的事情，出去跑步是很好的。

Watch

记住，阻止新冠肺炎病毒传播的方法是把你自己当成链条中的一环。你不想在任何地方感染病毒，如果你真的生病了(不管你知不知道)，你也不想把它传染给任何人。因此，最佳实践是:

*   与其他人保持六英尺的距离
*   不要碰别人碰过的东西

如果你系好鞋带，走进树林，或者只是在空荡荡的街道上慢跑，你就是金色的。现在也是骑自行车、散步或远足的好时机。尝试一次观鸟。

不要去人多的地方或者和一大群人一起跑步(很多有组织的比赛，比如马拉松，已经被取消了)，但是如果你只是想出去慢跑，那就尽情享受吧。
