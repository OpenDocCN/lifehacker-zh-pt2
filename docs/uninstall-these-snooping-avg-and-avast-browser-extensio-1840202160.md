# 卸载这些窥探 AVG 和 Avast 浏览器扩展

> 原文:[https://life hacker . com/uninstall-these-snooping-avg-and-avast-browser-extensio-1840202160](https://lifehacker.com/uninstall-these-snooping-avg-and-avast-browser-extensio-1840202160)

我们总是建议尽量简化您的扩展列表，因为您永远不知道什么时候会有东西试图从您的浏览器中吸走数据——尤其是超过它应该能够访问的数据。这正是 Mozilla 用四个看起来“不错”的扩展提出的问题——Avast 在线安全、 **AVG 在线安全**、 **Avast 安全价格**、和 **AVG 安全价格**——你应该从你的浏览器中卸载它们，直到它们被修复，至少是这样。

Watch

正如 [ZDNet](https://www.zdnet.com/article/mozilla-removes-avast-and-avg-extensions-from-add-on-portal-over-snooping-claims/) 报道，在 AdBlock Plus 的创建者 Wladimir Palant 报告这些扩展收集了过多的数据后，Mozilla 从其在线画廊中删除了这四个扩展。Palant [写道](https://palant.de/2019/12/03/mozilla-removes-avast-extensions-from-their-add-on-store-what-will-google-do/) :

> “监视你的用户显然违反了谷歌和 Mozilla 让扩展开发者签署的条款。所以昨天我向 Mozilla 和 Google 报告了这四个扩展。Mozilla 立即禁用了扩展列表，因此在 Mozilla 附加组件站点上再也找不到这些扩展。然而，Mozilla 并没有将这些扩展列入黑名单，声明他们仍在与 Avast 沟通。因此，对于现有用户来说，这些扩展仍然是活跃的，并继续监视用户。”

你可以在这里 阅读更多关于帕兰特的总体发现 [，但这里有他发现的一个片段:](https://palant.de/2019/10/28/avast-online-security-and-avast-secure-browser-are-spying-on-you/)

> *"当 Avast Online Security extension 处于活动状态时，它将从 Avast 服务器请求有关您访问过的网站的信息。在这个过程中，它将传输数据，允许重建您的整个网络浏览历史和您的大部分浏览行为。发送的数据量远远超过了扩展功能的必要量，尤其是与谷歌安全浏览等竞争解决方案相比。*

> Avast 隐私政策涵盖了这一功能，并声称这是提供服务所必需的。由于匿名化，存储数据被认为是不成问题的(我不同意)，Avast 没有做出任何声明来解释它保留数据多长时间。"

不管怎样，Avast 的一位发言人告诉 ZDNet，它正在与 Mozilla 合作修复其扩展，使其“完全兼容和透明”。你仍然可以在 Chrome 网上商店找到所有四个扩展，但 Opera 已经跟随 Mozilla 的足迹，也暂时从它的扩展库中删除了它们。

### 与此同时，如何处理这些扩展

如果你已经在你使用的任何浏览器中安装了这四个扩展中的任何一个，卸载它们。你可能甚至不需要*的*扩展来审查你正在访问的网站，因为这一切都可以用 [的一点常识](https://lifehacker.com/protect-yourself-against-amazon-prime-day-phishing-scam-1836308780) 来完成。

如果你仍然觉得你需要一个扩展来告诉你网站是否合法，你可以尝试使用类似于 **Windows Defender 浏览器保护**(用于 [Chrome](https://chrome.google.com/webstore/detail/windows-defender-browser/bkbeeeffjjeopflfhgeknacdieedcoml) )或稍微复杂一点的**应用程序保护扩展**(用于 [Chrome](https://chrome.google.com/webstore/detail/application-guard-extensi/mfjnknhkkiafjajicegabkbimfhplplj?hl=en-US) 或 [Firefox](https://addons.mozilla.org/en-US/firefox/addon/application-guard-extension/) )，将不受信任的网站转储到 Microsoft Edge 的一个独立实例中。(Y 你还必须在你的系统上安装 Edge，这样才能工作。)

至于价格监视扩展，我不能告诉你最后一次真正为我省钱是什么时候。如果你必须有某种小工具来告诉你正在考虑购买的物品的潜在交易，那么**Honey**([Chrome](https://chrome.google.com/webstore/detail/honey/bmnlcjabgnpnenekpadlanbbkooimhnj?hl=en-US)， [Firefox](https://addons.mozilla.org/en-US/firefox/addon/honey/) )是一个非常受尊重的扩展。不过，如果我是你，我会稍微搜索一下来找到最划算的交易。
