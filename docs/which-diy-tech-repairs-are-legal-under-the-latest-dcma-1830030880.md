# 根据最新的 DCMA 版权规定，哪些 DIY 技术修复是合法的

> 原文:[https://life hacker . com/which-DIY-tech-repair-is-legal-under-the-latest-dcma-1830030880](https://lifehacker.com/which-diy-tech-repairs-are-legal-under-the-latest-dcma-1830030880)

每三年，美国政府都会听到来自公民个人、小企业、技术专家和大公司的争论，讨论法律上应该允许用户对其拥有的技术进行何种修理或修改。最近的一次听证会是在 2018 年早些时候举行的，美国版权局本周发布了最终裁决，让用户能够合法地对智能手机、智能家居中枢等进行某些维修。

### DMCA 第 1201 节

这听起来可能很奇怪，但由于 2001 年通过的数字千年版权法(DMCA)第 1201 节，如果公司发现用户对其设备进行维修或操作侵犯了其版权，公司可以起诉用户。然而，该法也赋予公民向国会上诉的权利，要求免除某些维修和其他程序，否则将违反 DMCA 第 1201 条。

2015 年的裁决赋予设备所有者解锁和越狱某些智能手机、平板电脑、电视和其他智能设备的能力；更改 3D 打印机和机动陆地车辆上的软件；并允许用户对所有上述设备以及某些医疗植入物进行安全性研究。

今年，版权局听取了支持的论点:规避 HDCP 对 HDMI 视频信号的版权保护；越狱智能家居设备；并使用户能够绕过制造商在智能手机上安装的数字锁，等等。你可以阅读 DMCA 第 1201 条款的详细解释，法律细则，以及今年关于 iF [ixit 博客](https://ifixit.org/blog/11951/1201-copyright-final-rule/) 听证会的第一人称叙述。

### 2018 年版权裁决中的变更

以下是 2018 年最终裁决中授予(和未授予)的权限列表:

**法律:越狱并在语音助手设备上安装新固件**

法律现在允许用户破解他们的语音助手设备。这包括几乎所有由 Alexa 驱动的东西——你的亚马逊 Echo 扬声器、亚马逊 Fire TV Cube 等。—以及其他语音助手。这项裁决意味着你可以绕过这些设备上的数字版权管理来进行维修或安装新软件。

**法律:越狱/寻根智能手机**

像语音助理设备一样，现在为了自己解决问题而对智能手机进行 root/越狱是合法的——这在以前的某些情况下会使你的保修无效。同样，购买的新智能手机“锁定”到网络或提供商，现在可以立即解锁。以前，只有二手手机可以解锁。

看看这项豁免是否会影响某些制造商销售他们的手机，这将是一件有趣的事情，例如威瑞森强迫用户激活他们网络上的“解锁”设备，如 Pixel 3 或 Pixel 3 XL，以便实际解锁它们。

**法律:越狱/根植智能家电和智能家居系统**

智能家电和集线器也获得了一般的“维修”豁免，这扩展到修复固件以修复问题。iFixit 的文章强调了 Revolv 应用程序和智能家居中心作为例子，其中 [谷歌通过在 2016 年关闭 Revolv 服务器来强制和永久封锁](https://revolv.com/) 。有了这一变化，持有 Revolv hub 的用户可以根系统的固件来恢复其可用性，这同样适用于任何智能家居设备。

**法律:维修机动车及其软件**

你可以合法地修复/修改任何带马达的地面车辆的软件——汽车(包括特斯拉和其他电动汽车)、卡车、拖拉机、建筑车辆、摩托车等。这一变化还使用户能够访问他们车辆的远程信息诊断数据，以便诊断机械和软件问题。

**法律:第三方维修**

不仅上面列出的设备的所有者可以自己进行维修，而且现在第三方服务提供商和公司在没有被起诉的情况下进行维修也是合法的，令人困惑的是，在这一变化之前，这是不真实的。

**非法:分发规避 DRM 的工具**

奇怪的是，破解 DRM 的软件或越狱工具仍然是非法发行的。用户下载和使用破解其个人设备的工具是完全合法的，但如果你被发现开发或上传此类软件，你仍可能面临法律后果。

**非法:游戏机维修**

自己修理游戏机还是违法的。虽然您可以执行相同的许可升级，例如用更大的硬盘替换 PlayStation 4 的库存硬盘，但更换光驱或其他组件不在此列。这种替换和对基于硬件或软件的修复的任何修复仍然必须由制造商执行。

**非法:改变水上和空中交通工具和智能设备上的软件**

机动陆地车辆、智能手机、智能电器和语音助理设备的豁免不适用于飞机、船只或其他非陆地车辆/智能设备。

**非法:规避 HDMI 复制保护**

绕过 HDMI 视频信号使用的 HDCP 保护仍然是非法的。
