# 融化的冰淇淋是英国奶酪的绝佳替代品

> 原文：<https://lifehacker.com/melted-ice-cream-is-a-great-substitute-for-creme-anglai-1838669114>

上周日晚上，我戴着口罩在看《英国烘焙秀》的老节目，对着电视大喊大叫。我的一个朋友是这个节目的新成员，所以我们跟她聊了聊。当一名参赛者正在给他们的雪蛋(一种莫名其妙地被称为 [真正的](https://www.greatbritishchefs.com/recipes/mrs-beeton-snow-eggs-recipe) 的甜点)上菜时，我达到了一种沸点。

立刻，我感到一股温暖、金色的自豪感在我的血管中涌动。我的直觉是正确的，任何不同意我吃蛋羹的人都会不同意泰莎公主本人。这就像那次 [阿尔顿·布朗批准我吃](https://twitter.com/altonbrown/status/1177022457766711300) 的腊肠一样，但更好，因为我更看重女性的认可，而不是男性的认可。

总之。如果你不熟悉的话，英国奶酪是一种用蛋黄、某种乳制品、糖和香草制成的甜奶油沙司，这些都是你在香草冰淇淋中可以找到的东西。正如艾娜在她的新书 [*中解释的那样，像专业人士一样烹饪*](https://www.amazon.com/Cook-Like-Pro-Recipes-Cooks/dp/0804187045/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/melted-ice-cream-is-a-great-substitute-for-creme-anglai-1838669114&asc_source=&tag=kinjalifehackerlink-20) ，“香草冰淇淋本质上是冷冻的英国奶油。我把这个过程倒过来，最后得到了法式鲜奶油！”非常聪明。而且，作为一个人——虽然状态略有改变——最近把一些非常昂贵的冰淇淋放在冰箱而不是冰柜里，这也是一个非常适用于我生活的技巧。

但是，需要注意的是，你必须使用好的冰淇淋。任何含有大量填充物和稳定剂的东西都会以令人不快的方式融化，所以买阿根达斯或者更好的。把容器放在冰箱里——或者如果你赶时间的话，放在柜台上——让冰淇淋融化，然后用勺子舀上任何需要奶酪的甜点。你可以把它用在雪蛋上——我想这也是英国人喜欢吃的东西——但是热面包布丁听起来更有吸引力。
