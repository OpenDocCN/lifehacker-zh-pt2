# 我们最喜欢的泳池面条的非常规用法

> 原文：<https://lifehacker.com/our-favorite-unconventional-uses-for-pool-noodles-1844418121>

你知道吗，你只能点六包或更多的泳池面？因为我学到了惨痛的教训。

当我们出发去 [测试你是否能在戴着一顶用泳池面条做成的帽子](https://lifehacker.com/pool-noodles-are-not-a-good-way-to-enforce-social-dista-1844180889) 的同时保持适当的社交距离时，我最终得到了一大包泡沫意大利面，而我没有泳池可以嬉闹。



我没有让面条被浪费掉——因为我甚至不知道谁有游泳池——我开始为这些彩色的超大号布卡蒂尼找到其他用途。

*   因为它们只是圆形的泡沫，你可以在面条的长度方向切开一条缝，把它放在家具的锋利边缘上。
*   你可以用同样的方法盖住梯子的横档来保护你的小腿免受撞击。
*   要装满一个桶，把面条放在水龙头上，做一个很长的漏斗或 DIY 软管。
*   将面条切成大约一英尺长，然后再切成一半长；在园艺时，将制成的泡沫垫用作膝垫，或用作电脑的腕托。
