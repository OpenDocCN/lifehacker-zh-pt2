# 我怎么屏蔽邻居的 Wifi？

> 原文:[https://life hacker . com/how-do-I-block-my-neighbors-wifi-1833464189](https://lifehacker.com/how-do-i-block-my-neighbors-wifi-1833464189)

就像万圣节电影中的麦克尔·麦尔斯一样，生活中有些事情(看起来)是不可避免的。税收。政治。无论你在哪里，你邻居的 wifi 网络都强到足以在你的家里或公寓里给你一个不可用的信号。

Watch

我最近在 Tech 911 中谈了很多关于 wifi 的问题，只要你们——强大的 Lifehacker 读者——继续对这种连接类型有疑问，我就会继续提供建议。无线网络可能会令人困惑和沮丧，尤其是当你感觉邻居的 wifi 网络比你自己的网络连接更好的时候。

正如 Lifehacker 的读者 Sean 写道:

> “嗨，有个问题，我住在一个公寓楼里，在我公寓的某些地方，我邻居的 wifi 信号比我自己的强。我能做些什么来隔离或淹没其他信号吗？”

当然可以！面对你的邻居，评估你和他们之间有什么墙。用混凝土把这些墙砌起来，这样就可以阻止你邻居烦人的无线信号了。又或许挂上 [一串镜子](https://blog.plumewifi.com/five-things-in-your-home-that-are-killing-your-wifi-35a5cd296e7) ？

我开玩笑，但是有很多材料 会妨碍 wifi 信号，如果你打算改造你的公寓，这可能值得记住。我想你可以试试 [wifi 屏蔽漆](https://gizmodo.com/new-paint-could-block-wi-fi-from-nosey-neighbors-5139315) ，这听起来更像是一个噱头。你也可以发疯，把你的公寓关在法拉第笼里，这将会产生破坏你手机信号的额外效果。(这里希望你用 wifi 打电话。)

首先，我真正要做的是检查你邻居无线网络的实际信号强度。您可能会看到它作为连接选项在您的设备上弹出，但这是一个微弱或无关紧要的信号，因为您处于 wifi 网络范围的边缘。抓住 [一个免费的应用](https://lifehacker.com/which-wifi-channels-should-i-use-for-my-wireless-networ-1832788063) ，让你 [扫描无线信号](https://lifehacker.com/scan-for-competing-wifi-networks-with-this-free-android-1833078315) ，看看会发现什么。我敢打赌，你的网络看起来比你邻居的信号强得多——除非他们在离你公寓最近的墙壁附近设置了一些疯狂供电的路由器或接入点。

如果你和你的邻居关系很好，你可以随时走过去，让他们查看他们路由器的高级设置，并 [调低其发射功率](https://www.smallnetbuilder.com/basics/wireless-basics/32228-an-experiment-in-wireless-power-and-range)——如果这个选项存在的话。(并不是所有的路由器都允许你这么做)。这应该会将 wifi 泡沫拉回一点，并且不会对邻居的性能产生明显的影响。如果你在自己的公寓里看到很强的信号强度，我想他们的公寓里信号很好。

你可以礼貌地要求他们把路由器移到公寓更中心的位置。根据他们的设置，这可能可行，也可能不可行。然而，你可以随时让他们知道，这将确保他们的整个生活区得到一个像样的信号。

至少，你可以让他们把离你最近的路由器或接入点 [放在一个特定的无线信道](https://lifehacker.com/which-wifi-channels-should-i-use-for-my-wireless-networ-1832788063) 上，你可以使用不同的信道来进行你自己的无线设置。你仍然会看到他们的网络(除非你礼貌地要求他们“ [隐藏](https://lifehacker.com/is-hiding-your-wireless-ssid-really-more-secure-5636856) ”它)，但它不应该干扰你自己的速度太多，如果有的话。这是我会选择的解决方案，因为它保证了最好的共存，也就是说，你保持了睦邻友好的关系，你仍然可以为你自己的设备提供很好的连接。

你也可以得到领土:在你的公寓周围建立一个无线网状系统，给每一寸空间一个难以置信的无线信号，让你的邻居 [不得不处理](https://www.youtube.com/watch?v=B3lsJmwNO40) *你的*设置。或者将尽可能多的设备切换到 5GHz(并确保您的网络不在与他们相同的频道上运行)。

为了测试你的调整，你可以随时点击[【fast.com】](https://fast.com/)来检查你邻居的 wifi 开着和关着时的速度——如果他们愿意试验的话——或者你尝试的任何其他巧妙的配置。

不过，总的来说，我建议协商频道，除此之外，只要找到另一个接入点(或更强的路由器)就可以了，无论你在哪里，它都能给你提供你需要的连接。除此之外，不要再担心邻居了，感谢你没有住在一个充斥着各种频道相互竞争、供电良好的网络的公寓楼里。
