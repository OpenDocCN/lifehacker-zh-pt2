# 尝试阅读儿童书籍来理解复杂或无趣的话题

> 原文:[https://life hacker . com/try-reading-a-children-book-to-understand-complicated-1834365747](https://lifehacker.com/try-reading-a-childrens-book-to-understand-complicated-1834365747)

如果你想学点什么，但在阅读成人参考书时有困难，可以考虑选择儿童版本。

詹姆斯·霍尔茨豪尔目前已经赢得了超过 110 万美元的《危险边缘》策划费，这使他成为该节目有史以来第二大赢家，仅次于肯·詹宁斯。在本周接受*《纽约时报》*采访时，霍尔茨豪尔谈到了他的一些成功秘诀，包括他是如何了解这么多不同领域的知识的。

“我有一个阅读儿童书籍获取知识的策略。他对《泰晤士报》说:“我发现，在成人参考书中，如果不是我感兴趣的主题，我就无法进入其中。”。“我在想，图书馆里有什么地方可以让我找到适合不感兴趣的读者的书？嘣。儿童区。”

这是我个人实际上以一种有点不同的方式使用的策略:学习编码。我喜欢学习如何编码的概念，但是即使是“最简单”的成人程序，在了几堂课之后，总是以对我来说有点太难理解而告终。当我开始接触时，以孩子为中心的编码课程对我来说更容易通过，之后为我在成人版本中取得更大的成功做准备。

很明显，你不会在每一个主题上都找到儿童读物，但是如果你正在为一本儿童读物所讲述的事情而奋斗，它可能正是你所需要的，让你走上正轨。
