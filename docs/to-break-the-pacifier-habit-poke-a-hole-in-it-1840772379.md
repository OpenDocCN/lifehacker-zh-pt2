# 为了打破安抚奶嘴的习惯，在上面戳一个洞

> 原文：<https://lifehacker.com/to-break-the-pacifier-habit-poke-a-hole-in-it-1840772379>

奶嘴是一个很棒的东西。它帮助我们最小的孩子安慰自己入睡，而其他任何东西都不能做到这一点。不过，最终你会希望他们能够自我安慰。他们从婴儿成长为蹒跚学步的孩子，你渴望一张没有嘴巴的照片。随着牙齿一颗接一颗地生长，你可能会担心长时间使用安抚奶嘴会开始影响他们的牙齿排列。



你可以去冷火鸡。你可以一个接一个地“失去”他们。你可以搭上“米朵琪仙女”的顺风车，在某个晚上把它们都堆在外面，然后收集起来(作为报酬，还会留下一份特殊的奖励)。你甚至可以举行 [告别仪式](https://lifehacker.com/help-your-toddler-through-a-tough-transition-with-a-far-1829181419) 。

或者你可以用一种安全的方式破坏它们，只要一点点——在乳头顶端戳一个小洞。在奶嘴上戳一个小洞会释放空气，让吮吸的满足感降低。这可以有效地帮助孩子对他们以前的朋友失去兴趣，但是有几个警告:

*   保持洞很小(最好用针)，并确保你没有留下任何可能被吞咽的碎片。
*   不要剪掉整个尖端。这曾经是一种常用的 paci 脱机方法，但随着时间的推移，小碎片可能会脱落，成为窒息的危险。

如果他们说他们的 binky 突然感觉不一样了，或者不如以前好了，简单地告诉他们，这是因为他们正在长大，正在摆脱这个习惯。
