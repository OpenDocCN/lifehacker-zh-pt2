# 你可以一次点完所有的马丁尼配菜

> 原文:[https://life hacker . com/you-can-all-the-martini-garnies-once-1843982095](https://lifehacker.com/you-can-have-all-the-martini-garnishes-at-once-1843982095)

虽然不像“摇晃或搅拌”那样令人讨厌问题“橄榄还是麻花？”被真诚地询问的次数要多得多，因为大多数调酒师不会让顾客选择摇晃的马提尼。(好。选择装饰看起来像是一种审美——你感觉明亮、阳光还是复古性感？—但橄榄和麻花都以微妙但可感知的方式改变了马提尼的味道。但昨晚在准备我的洗澡时间马提尼时，我选择用另一个问题来回答这个问题:为什么不两者都要呢？

每种装饰带来的好处并不相互排斥。从一条柑橘皮中榨出的油使你的马提尼酒更加明亮和醉人的芳香，而附着在一颗(或三颗)橄榄上的剩余盐水提供了一种咸咸的平衡来抵消所有令人振奋的酒精。(橄榄也有作为零食的功能，我们很喜欢！)用这两者来完成一杯马提尼酒会提供更多样的风味和香味，反过来会让带出隐藏在你的杜松子酒里之前未被注意到的风味和香味。

即使是肮脏的马提尼酒也能从中受益。事实上，我认为在这种情况下*尤其*受欢迎——一点光亮可以帮助消除橄榄卤水带来的侵略性盐分和臭味。但是不管你选择混合哪种马提尼酒都要确保在喝之前轻轻挤压一下就能把油挤出来。在你的饮料中加入一点未榨的香料当然会让它看起来更有味道，但是是精油让它散发出香味。(然而，没有必要挤橄榄。你可以不碰橄榄。)
