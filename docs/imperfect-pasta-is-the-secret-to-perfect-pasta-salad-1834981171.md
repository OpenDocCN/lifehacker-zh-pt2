# 不完美的意大利面是完美意大利面沙拉的秘密

> 原文：<https://lifehacker.com/imperfect-pasta-is-the-secret-to-perfect-pasta-salad-1834981171>

像配菜 “涂有蛋黄酱的碳水化合物”类别中的其他条目一样，意大利面沙拉令人难以置信地两极分化。好吃的时候，吃的停不下来；坏了就不能吃了。我发现真正好吃的意大利面沙拉的秘密不是 [自制蛋黄酱](https://lifehacker.com/how-to-make-your-own-dukes-style-mayonnaise-1833804415) 或一罐好泡菜——而是*真正难吃的*意大利面。特别是煮过头的意大利面。



对我来说，一批坏的意大利面沙拉的唯一定义特征是脆脆的面条，这是因为淀粉在低温下凝固。(如果你曾经直接从冰箱里吃过剩下的意大利面，你就知道我在说什么了。)要克服这一点，你需要做的就是让意大利面沙拉面条的口感超过发酸的程度——热的时候越软，冷的时候越嫩。

要做有史以来最好的意大利面沙拉，将一壶盐水煮沸，加入你的意大利面，比包装上说的至少多煮一两分钟。当你咬一口时，面条应该几乎没有任何阻力——但不完全是糊状的——并且它们应该明显比开始时大。显然，煮熟的意大利面总是比干的大，但是煮过头的意大利面会膨胀:

从这里开始，你需要做的就是让面条冷却一点，然后根据你的喜好调味。这一批特别的意大利面沙拉是冰箱里的垃圾:一些需要使用的芫荽叶和欧芹根香蒜酱，超熟的 giardiniera，解冻的冷冻豌豆，一块马苏里拉奶酪的残余，以及一个果酱鸡蛋。味道很好——即使在冰箱里放了一夜，意大利面仍然非常嫩。
