# 该不该用查痣 App？

> 原文：<https://lifehacker.com/should-you-use-a-mole-checking-app-1836049240>

那颗痣看起来可疑吗？你可以下载一个应用程序来检查单个的痣，或者跟踪你身上的痣，但这些应用程序都不能代替真正的皮肤科医生检查。



### 了解应用程序的工作原理

声称可以分析你的痣或帮助你找出哪些人需要去看医生的应用程序有三种类型:

*   让你拍照并发送给皮肤科医生的应用，通常是收费的。例子: [先皮](https://www.firstderm.com/) ， [皮肤科](https://apps.apple.com/us/app/dermatologistoncall-online/id635315374) 。(这些通常适用于任何皮肤状况，而不仅仅是痣或疑似癌症)
*   分析痣的照片，并自动评估它患癌症的风险是高还是低的应用程序。例子: [SkinVision](https://www.skinvision.com/) 。
*   应用程序不会判断你的痣，但允许你随着时间的推移进行比较。其中一些可以让你拍摄大面积皮肤的照片，比如你的整个背部，以便随时间进行比较。例子: [MiiSkin](https://miiskin.com/) ， [UMSkinCheck](https://www.uofmhealth.org/patient%20and%20visitor%20guide/my-skin-check-app) 。

一些应用程序结合了这些功能，但重要的是要知道应用程序实际上会为你做什么。咨询现实世界皮肤科医生的人可能比用算法分析照片的人更可信，但仍不如亲自去看皮肤科医生。让你追踪自己痣的应用程序需要你更多的关注和判断。

### 应用程序可能会出错

人无完人，医学筛查测试也是如此。应用程序增加了一些额外的不确定性。

首先，无论一个应用程序吹嘘的“人工智能”、“机器学习”或“分形分析”有多先进，它仍然可能是错误的。 [一项研究](http://file:///Users/eskwarecki/Downloads/43-1-381-2-10-20190127%20(1).pdf) 发现，使用真正皮肤科医生的应用程序对 88%的癌性痣和 97%的良性痣都是正确的。

乍一看，这听起来不错，但这意味着 3%没有皮肤癌的人可以被告知他们患有皮肤癌；12%需要做检查的人会被告知他们可能很好。这还只是向皮肤科医生发送照片的应用程序。自动分析照片的应用程序成功率更低。

### 你的手机摄像头还不如皮肤科医生的眼睛

虽然算法可以很好地识别典型的痣，但它们并不总是知道如何处理看起来不像良性或癌症样本的图像。

你的手机摄像头是另一个问题。虽然相机很好，但它们不能像皮肤科医生使用的皮肤镜那样提供详细的视图。(然而，你可以花 99 美元或 299 美元购买一台名为 [MoleScope](https://www.molescope.com/) 的家用相机，以拍摄更高质量的照片。)

还有一个事实是，你在抓拍单个痣的照片，而不是和真正的皮肤科医生在一起。你可能担心你手臂上的一颗痣，但是医生注意到你背上的一个不同的点。

### 你还是要依靠自己的判断

虽然应用程序可以暗示痣是否可能有问题，但最终你会做出自己的判断，决定是否首先拍照，以及是否寻求专业帮助。

出于这些原因，知道你在鼹鼠身上寻找什么是很重要的。[ABCDE 助记符](https://lifehacker.com/check-your-body-for-skin-cancer-with-the-abcdes-of-mela-1716612273) 可以帮上忙:寻找*不对称*，边界*的形状*，痣的*颜色*，它的*直径*(又名大小)，以及它是否看起来是*在进化*，还是随着时间而变化。

像 MiiSkin 和 UMSkinCheck 这样的照片跟踪应用程序可以帮助你跟踪最后一个，无论痣是否随着时间的推移而变化，但它们不会为你诊断皮肤状况。如果你看看这些应用程序上的细则，没有一个声称能够取代医生。所以注意那些痣，涂上你的防晒霜，当你可以的时候， [让一个真实的人](https://vitals.lifehacker.com/how-to-find-a-free-skin-cancer-screening-near-you-1825921687) 检查一下。
