# 最新的不确定健康趋势是注入 CBD 的健身装备

> 原文：<https://gizmodo.com/the-latest-iffy-wellness-trend-is-cbd-infused-workout-g-1838982838>

不缺少被填充、浸泡或喷洒大麻二酚的产品，大麻二酚是大麻中不改变精神的成分，也称为 CBD 。最新的趋势肯定会吸引健康人群:CBD 填充的健身服，据说可以在你酸痛的肌肉开始疼痛之前缓解它们。虽然这些产品是 CBD 主流时刻的最明显标志，但它们也象征着 [这个行业在如此短的时间内变得多么喧闹。](https://gizmodo.com/should-you-your-mom-and-your-dog-be-using-cbd-1834092614) 



今年 8 月，总部位于纽约的 Acabada Active 公司首次亮相全球，推出了融入 CBD 元素的“奢华前瞻性服装”系列

该公司的服装从 125 美元的运动胸罩到 275 美元的连体裤，据说所有这些都是在葡萄牙一家使用 100%可再生能源的工厂制造的(Acabada 在葡萄牙语中恰好是“成品”的意思)。然而，CBD 100%来自美国——这一区别在 2018 年底成为可能，当时该国将大麻作物的种植合法化，大麻作物几乎不含四氢大麻酚，这是一种臭名昭著的让我们飘飘欲仙的成分。

据该公司称，由于“微胶囊技术”将 CBD 液滴收集在微小的容器中，每个 Acabada 片据说包装了多达 25 克的 CBD。当穿着者运动并摩擦衣物时，微胶囊破裂，并定期用 CBD 浸泡他们的皮肤。至于寿命，CBD 的供应应该可以持续大约 40 个“高强度的穿洗周期”，之后人们仍然会有一件耐用的运动服。

像市场上许多 CBD 的供应商一样，Acabada 小心翼翼地不公开吹嘘其产品将治疗或治愈任何疾病，以免与食品药品监督管理局的 [指令](https://www.fda.gov/consumers/consumer-updates/what-you-need-know-and-what-were-working-find-out-about-products-containing-cannabis-or-cannabis) 相冲突，即基于 CBD 的产品不得以未经批准的医疗声明出售。事实上，并不存在 FDA 批准的 CBD 局部版本。然而，Acabada 的广告确实告诉潜在买家“接受你最喜欢的 CBD 外用产品提供的消炎、镇静和缓解肌肉疼痛的好处”，以及其富含 CBD 的服装将“有助于在活动开始前消除疼痛，促进愈合。”

这些产品如此诱人的部分原因是它们有一点真实性。CBD(以及 THC 和大麻)绝对可以而且很可能被证明有助于治疗某些疾病。例如，去年 CBD 被批准作为液体形式的抗癫痫药物口服。对 CBD 的实验室研究也发现了它可以消炎的证据，包括对 [人类皮肤](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC4151231/) 的研究，而一些对人类的研究表明，它可以帮助缓解 [慢性疼痛](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC6535492/) 。你不用走很远就能找到健身房的老鼠，他们声称 CBD 通过加快恢复时间和减轻疼痛使他们的锻炼更容易进行。

但是据加州大学洛杉矶分校大麻研究项目的研究主任 Ziva Cooper 说，目前可用的研究还没有接近证实 Acabada 的营销。

她在电话中告诉 Gizmodo，“我们没有好的证据表明，从对人类的对照研究来看，CBD 将有助于加速愈合过程。”“如果 CBD 确实对愈合或肌肉疼痛有影响，我们还不知道它在局部给药时被身体吸收的程度。”

这是非常关键的一点，因为一种药物对身体的影响会因其吸收方式的不同而大相径庭。例如，局部用药可能会减少副作用，但只对用药部位有效。其他药物需要非常特殊的设计才能通过皮肤影响身体，否则 [根本不起作用](https://www.statnews.com/2017/08/09/fentanyl-falling-ill/) 。

不要指望 Acabada 会很快进行双盲对照临床试验。该公司向 Gizmodo 提供了内部研究，表明其产品如预期那样分泌 CBD，不会对佩戴者造成刺激或在洗涤时泄漏；他们还有数据显示，他们的微囊化 CBD 能够穿透皮肤，在那里它应该与细胞中发现的天然大麻素受体相互作用。Acabada Active 首席执行官 Seth Baum 告诉 Gizmodo，客户调查基本上是积极的，38 名受访者中有 32 人表示，他们在使用 Acabada 产品后感觉好了一点或好多了(总共有 100 人被要求做出回应)。

但是根据 Baum 的说法，证明 Acabada 产品中的 CBD 实际上为客户做了任何事情并不在他们发布之前的优先列表上。“我们没有试图验证 CBD 的概念和功效——这超出了我们的工资级别，”他说。“有些人理解 CBD，相信 CBD，并且已经是倡导者了。这就是我们要关注的人。”

加拿大安大略省的健康政策专家蒂姆·考尔菲尔德认为，鲍姆的理论基础几乎可以从每个兜售这些产品的人身上看到，他最近被带到了对抗流行文化中的伪科学 的 [。](https://www.theglobeandmail.com/life/article-has-tim-caulfield-become-the-canadian-nemesis-of-pseudoscience/) 

“就在不久前，我们还在谈论大麻的疯狂和危害，”考尔菲尔德告诉 Gizmodo。“但在相对较短的时间内，CBD 已经成为这个数万亿美元健康产业的支柱，它的好处是假设的，它的功效是假设的——它已经接受了这个健康光环。”

有理由认为，公众对 CBD 的兴趣可能会带来一些好处，即使其中很多最终会被放错地方。读者给我发来电子邮件，称赞 CBD 在帮助他们解决药物使用问题或慢性疼痛方面的好处，并补充说其他方法都不起作用。库珀确实认为公众的关注将有助于调动像她这样的科学家所需的资源，以正确理解 CBD 对人类的影响。似乎鲍姆和他的公司是明确地向已经皈依的人说教，而不顾医学共识。

但是，如果希望最终是假的，或者是为了赚钱而产生的，那它又有什么用呢？虽然这些产品中有多少甚至含有足够的 CBD 来影响身体还存在争议，更不用说积极影响了，但是购买 CBD 炒作可能会有实际的成本。

“人们可能会使用无效的 CBD，而不是接受有效的治疗。或者他们可能使用被污染的中央商务区，这是经常发生的。如果它是有效的，那么它也可能是有害的——没有什么是完全良性的，”考尔菲尔德指出，引用了人们服用 [假](https://jamanetwork.com/journals/jama/article-abstract/2684620) 或 [误贴](https://jamanetwork.com/journals/jama/fullarticle/2661569?mod=article_inline) CBD 的记录实例，以及服用足够 CBD 治疗癫痫发作的人经历的相对 [轻微但常见的副作用](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC5569602/) 。

虽然 Acabada 作为第一家专门的 CBD 服装公司已经进入了市场，但它不太可能长期成为唯一的游戏。它与比利时公司 Devan 合作，制造注入 CBD 的产品，这是一项被称为 R-Vital 的技术，Devan 自 2019 年 1 月以来一直在 [广告](https://fashionmagazine.com/fashion/cbd-infused-clothing/) 。
