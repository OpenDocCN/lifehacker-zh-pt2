# 使用此 iOS 快捷方式滚动一个新的龙与地下城角色

> 原文：<https://lifehacker.com/use-this-ios-shortcut-to-roll-a-new-dungeons-dragons-1830290340>

一个精神掠夺者刚刚吃掉了你——或者你的团队——而你正在寻找一个全新的*地下城&龙族*角色回到战斗中。或者你有一个新鲜的新游戏即将推出，你需要一点灵感为你的下一个角色的背景和个性。与其花一个小时盯着一张空白的角色表，不如用 iOS 快捷方式“ [WTF 是我的 DnD 角色](https://www.icloud.com/shortcuts/afe1ce9cedb04d948adfe818aa2285f3) ”(由 Reddit 用户“ [mvoviri](https://www.reddit.com/user/mvoviri) ”创建)节省一点时间。



剧透:如果你不喜欢调皮话，这个快捷键对你来说是*不是*。

安装它，在你的 iOS 快捷方式库中点击它，你会得到一个预先生成的背景故事(由恰当命名的)让你对一个全新的角色产生兴趣。

如果你不喜欢快捷方式为你拉出来的东西，点击“确定”并选择“他妈的不，给我一个新的”选项来生成一个新的背景故事。重复直到满意为止。

当你准备好积累你的新角色的属性时——力量、智力、魅力等等——你可以让快捷方式为你完成。只需点击“地狱耶，滚动一些统计，”你会得到一个随机数字列表，你可以分配到你认为合适的不同统计。你只有一次机会，所以做好你的选择。

一旦你完成了，快捷方式会自动添加你得到的任何种族修饰语(基于你的角色的预先生成的背景)，并给你一个你是谁的小总结。

然后你可以和你的同伴(和/或地下城主)分享你的总结，尽管我建议你把它放到一个笔记文件里，这样你就不会忘记你是谁。你可能仍然想手动填写一份真实的角色表(或者使用方便的[iOS](https://itunes.apple.com/us/app/fight-club-5th-edition/id901057473?mt=8)[app](https://itunes.apple.com/us/app/fifth-edition-character-sheet/id967650851?mt=8))，但至少你已经提前做了一些工作——如果你最近没有太多创意，这尤其有用。
