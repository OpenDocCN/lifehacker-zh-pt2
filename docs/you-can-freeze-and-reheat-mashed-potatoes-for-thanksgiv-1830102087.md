# 感恩节你可以冷冻和加热土豆泥

> 原文:[https://life hacker . com/you-can-freeze-and-reheated-the-pasteur-pasteur-thanksgiv-1830102087](https://lifehacker.com/you-can-freeze-and-reheat-mashed-potatoes-for-thanksgiv-1830102087)

土豆因为加热不好而有点臭名昭著。烤土豆、炸薯条和土豆煎饼是它们刚出炉时最好的样子，要让它们恢复往日的辉煌相当棘手——有些人会说“不可能”。然而，土豆泥是一个例外，这是非常幸运的，因为这意味着你可以在感恩节(或随机的星期二)制作、冷冻和加热它们。

为什么捣碎器在其他捣碎器失败的地方做得更好？他们的秘密和我的一样:他们充满了黄油和奶油。正如我的脂肪保护我免受俄勒冈州一周的大雪，这对奶油乳制品保护土豆在冰箱里不会变质。

像平时一样简单地烹饪你的土豆，使用大量的黄油和奶油——如果你喜欢，你可以使用我的食谱——让它们冷却，然后把它们放进冷冻袋，慢慢地吸，这样袋子就可以平放了。将它们储存在冰箱里，直到你准备好食用它们。

在你准备端上来的前一天晚上(所以，感恩节的前一天晚上或者随便一个周一)，把它们从冰箱里拿出来，让它们在冰箱里解冻。你有两种重新加热的选择:

*   **在烤箱里:**把土豆摊放在砂锅或其他烤箱安全的盘子里，在上面撒上黄油，再撒上一点奶油。(对于四磅土豆，我用了两汤匙黄油和四分之一杯奶油。)盖上盖子，让它们在 350 度的烤箱中加热半个小时，中途搅拌一下。慢煮可以防止烤焦或变粘，添加的黄油和奶油可以防止它们变干。
*   在慢炖锅里:将土豆倒入慢炖锅里，小火加热两个小时。一个小时后偷看他们，如果他们看起来有点干，加一点奶油。

把土豆和火鸡一起端上来，配上大量的肉汁，或者干脆用碗盛着吃，不加佐料，这是你的权利。
