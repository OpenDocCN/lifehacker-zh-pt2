# 用 Booky.io 整理浏览器的书签

> 原文：<https://lifehacker.com/organize-your-browsers-bookmarks-with-booky-io-1832365252>

保持你的浏览器的许多书签有组织是一个永无止境的斗争。相信我，我理解这种痛苦；不管我安装了多少扩展，也不管我发誓多少次我绝不会让我溢出的书签工具栏达到这种状态，它总是发生。总有一天，我会学会的。

Watch

与此同时，我偶然发现了一个整洁的小网站，可以帮助你管理你的浏览器杂物: [**Booky.io**](https://booky.io/) 。要开始，只需创建一个免费帐户，将浏览器的当前书签导出为一个简单的 HTML 文件，然后将它们导入网站。(不，没有办法通过浏览器扩展自动同步你的书签，但我希望这项功能迟早会实现。)

当您导入书签时，Booky.io 会考虑您(希望)已经用来组织书签的文件夹结构。这些成为网站上的“类别”，你可以在你的仪表板上随意移动。(是的，如果你想把“工作”和“不工作”分成不同的类别，你甚至可以有多个仪表盘)

一个重要的细节:导入似乎不适用于嵌套文件夹，或者主文件夹中的次级书签文件夹。如果是这样的话，您需要分别导入它们。

通过 booky.io，你可以在类别内或类别之间拖放书签，尽管该网站缺乏批量移动功能，这可以让你更容易地组织你庞大的网站列表，以及更容易在你的类别中排序书签的字母排序功能。但是，如果有助于找到特定的书签分类，您可以对您的类别进行颜色编码。

我认为使用 booky.io 最好的方法是把它设置成你浏览器的主页(或者一个你可以很容易访问的书签或者大头针标签)。我可能只会在我知道我总想在某个时候访问的服务中保存书签——我的*常客*，可以说，不一定是一个我只是偶然发现的网站，我想保存并在以后阅读。如果你不想一次导入所有浏览器的书签，你可以使用 booky.io 的 [bookmarklet 或 Chrome extension](https://booky.io/help) 来导入主网站的片段列表。

虽然还不清楚 booky.io 的下一个版本可能会在什么时候推出，或者它会有什么新功能，但开发者几个月前发布了他们正在开发该服务的新版本。
