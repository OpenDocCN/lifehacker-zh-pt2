# 当你陷入困境时，使用“五更法则”

> 原文:[https://life hacker . com/use-the-five-more-rule-when-you-feeling-stick-1829911156](https://lifehacker.com/use-the-five-more-rule-when-youre-feeling-stuck-1829911156)

人们很容易被所有的事情弄得不知所措。所有你不得不回复的人(通常以悲伤的开场白“抱歉延迟回复…”).所有你应该写的字。所有你承诺在跑步机上跑步的时间，所有你说要准备的食物，所有你几个月来一直想扔掉的壁橱杂物。

也许你已经开始了一项任务，但却陷入了困境，被你面前巨大的山峰所麻痹。与其放弃，不如告诉自己:**再来五个**。在一篇关于 [的文章中，卫报解释了“五更法则”](https://www.theguardian.com/lifeandstyle/2018/oct/14/the-lost-art-of-concentration-being-distracted-in-a-digital-world)

> 这是学习更好地集中注意力的简单方法。事情是这样的:每当你想放弃的时候——就再做五次——五分钟，五次练习，五页——这将扩展你的注意力。这条规则将你推过挫折点，帮助你建立精神集中。这是一种训练形式，也是完成某事的一种方式。

我一直在做我自己的事情。我称之为“五个茁壮成长”——俗气的名字，但它坚持。数字 5 多少让人觉得可以控制。不想锻炼吗？做五个面向下的狗。你不能让自己清理整个冰箱吗？清除五个项目。被放在你床头柜上的那本巨大的未打开的小说嘲笑？只读五页。你在给自己一个跳跃的开始——一旦开始，你可能就不会停下来。(但不要这样告诉自己。刚刚五点。很简单，对吧？)
