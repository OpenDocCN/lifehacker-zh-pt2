# 你现在可以在 Pinterest 上找到合法的疫苗信息

> 原文:[https://life hacker . com/you-can-now-find-legit-vaccine-information-on-Pinterest-1837670851](https://lifehacker.com/you-can-now-find-legit-vaccine-information-on-pinterest-1837670851)

直到最近，Pinterest [还屏蔽了其平台上关于疫苗](https://gizmodo.com/pinterest-quietly-blocks-all-vaccine-related-searches-1832754950) 的搜索结果。但是今天， [公司宣布](https://newsroom.pinterest.com/en/post/bringing-authoritative-vaccine-results-to-pinterest-search) 将提供“权威的疫苗结果”，而不是其他的。

Watch

好消息是，你现在可以从世界卫生组织、疫苗安全网、疾病控制和预防中心以及美国儿科学会找到信息。如果你搜索“疫苗”或类似“麻疹”的关键词，他们的结果就会出现。这些搜索还会在页面上显示一条消息:

> 关于这个话题的 pin 经常违反我们的社区指南，该指南禁止有害的医疗错误信息。
> 
> 因此，我们将搜索结果限制为来自国际公认的健康组织的 pin。
> 
> 如果您正在寻求医疗建议，请联系医疗保健提供者

其他搜索，如“癌症治疗”，仍然没有返回结果，只有这样的消息:

> 关于这个话题的 pin 经常违反我们的社区指南，该指南禁止有害的医疗错误信息。在某些情况下，我们可能选择不显示这个主题的任何搜索结果。
> 
> 如果您看到任何您认为违反 Pinterest 准则的内容，请向我们报告。我们的政策和执行指南由国际认可的机构提供信息，包括美国疾病预防控制中心、世卫组织和 AAP。
> 
> 如果您正在寻求医疗建议，请联系医疗保健提供者。

虽然 Pinterest 的新政策肯定比他们之前的政策有所进步，但 Pinterest 仍然不是循证健康信息的仙境。如果你搜索“麻疹”，你会得到经过审查的疫苗信息。但是如果你搜索“水痘”，结果包括一些不科学的别针，其中一个 [声称](https://healthy44healthy.club/this-vitamin-will-protect-you-from-shingles-and-chickenpox-2/) 维生素 C 可以预防疾病。

也许他们还在努力。几分钟前，搜索“癌症”,找到了据称会导致癌症的食物。我刚才重新加载了页面，现在我得到的大多是与十二宫杀手有关的徽章。另一方面，搜索词“癌症疗法”没有被屏蔽，并出现了一些不太科学的观点(比如姜茶可以杀死癌细胞)。要报告违反 Pinterest 指南的 pin，请单击该 pin 并使用三点菜单报告“医疗错误信息”
