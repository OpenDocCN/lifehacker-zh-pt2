# 如果你发现一只小鸟该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-you-find-a-baby-bird-1843547313>

每年春天，树上和灌木丛里都有鸟巢，而那些鸟巢里又塞满了小鸟。如果你发现一只小鸟，它不是在巢里，它不一定需要你的帮助。以下是你需要知道的。

### 如果鸟明显受伤

如果你能分辨出这只鸟受伤了，比如它显然断了一条腿或一只翅膀，不要担心列表中的其他部分——打电话给你当地的野生动物康复师。这里有一个各州列表，或者你可以谷歌搜索“野生动物康复师”加上你的位置。

如果你不能识别受伤，或者你不确定，因为也许这只鸟看起来有点不舒服和笨拙，继续读下去。

### 如果这只鸟有羽毛并且在跳来跳去

恭喜你，你找到了一只雏鸟！一只雏鸟可能不会飞，但它们可以跳跃、行走或在树枝间飞来飞去。如果它们是栖息的鸟(像你典型的鸣禽)，它们的脚趾足够强壮，可以抓住树枝或你的手指。

雏鸟是笨拙的少年鸟。正如康奈尔鸟类学实验室指出的 ，它们在父母的监督下离开巢穴，可能会花几天时间四处跳跃，然后才知道如何独立生活。父母密切关注着它们，但它们可能同时要照看四五个后代。

这些鸟不需要你的帮助；他们相处得很好。(即使你能找到鸟巢，也没必要把它们放回去。他们会马上跳出来。)如果你想确定父母会不会回来，那就坐在旁边看着就行了。你会看到他们回来。

### 如果鸟儿没有羽毛，无助

雏鸟是需要呆在巢里的鸟。它们刚刚被孵化出来，如果它们是全新的，它们的眼睛可能仍然是闭着的，它们可能还没有任何羽毛。

这些鸟需要回到巢里。它们不能独自远行，所以如果你找到一个，巢就在附近。不过，它可能是隐藏的。小心地把鸟放回窝里。

有一种说法认为鸟类会拒绝任何闻起来像人类的后代。 [这不是真的，科内尔报道](https://www.allaboutbirds.org/news/if-i-handle-a-baby-bird-will-the-parents-abandon-it/) ，所以要毫不犹豫地把雏鸟还给父母。

如果你找不到巢，你可以做一个。 [按照这里的指示重建一个巢穴](http://www.helpbabybirds.ca/s/do/fallenNest.html) 。如果整个鸟巢从树上掉了下来，这也是你应该做的。
