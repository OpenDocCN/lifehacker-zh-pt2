# 来自真正的佛罗里达人的最佳飓风提示

> 原文：<https://lifehacker.com/the-best-hurricane-tips-from-real-floridians-1844930459>

似乎 2020 年不会变得更糟，今年我们可能会看到更强大的飓风登陆。由于这些风暴可能会袭击一些没有风暴的地区，这里有一些来自我们的风暴专家工作人员的建议:佛罗里达人。

作为一个土生土长的佛罗里达人，我经历过无数的飓风、热带风暴和热带低气压。现在我住在纽约，我总是对大风暴前冲向肉店的人数感到震惊。(不要买生肉！如果你失去电力，它会变坏的！你们这些东北笨蛋！)

要记住的主要概念是，风暴过后，你可能会几天或几周没有电、wifi 或饮用水，所以一定要准备一些现金、电池、瓶装水和罐头食品(带手动开罐器)。

你还想确保你房子外面的任何东西都不会成为风中的高速抛射物，所以把所有庭院家具都带进来，盖上窗户，如果你有重的东西，就把它们扔进游泳池。

然后你需要的只是一些书、手电筒和一个毯子城堡来度过最糟糕的时期。关于所有这些技巧和更多，请看上面的视频。
