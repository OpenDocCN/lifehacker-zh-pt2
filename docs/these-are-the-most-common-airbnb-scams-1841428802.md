# 这些是最常见的 Airbnb 骗局

Airbnb 的问题在于，该平台忽视了对出现在其网站上的至少一些公寓和房东进行彻底审查。因此，骗局被识破了，而(字面上)付钱的是客人——只要问问英国的一对夫妇就知道了，他们爱上了根本不存在的奢华的 Airbnb 房产 ，在这个过程中几乎损失了 12000 美元。

或者问问任何一个在 [假酒店住宿的客人，副记者艾丽·康蒂最近的](https://www.vice.com/en_us/article/43k7z3/nationwide-fake-host-scam-on-airbnb) 报道揭露了一些 Airbnb 房东制造的全国性骗局。(别忘了，主持人也很容易被骗。在 Airbnbhell.com 的上，一个由主持人和嘉宾共同撰写恐怖故事的网站上，一些主持人报告说他们自己被骗了。)

在报道之后， [Vice 要求](https://www.vice.com/en_us/article/epgvm7/airbnb-scam-how-to-tell) 的读者描述他们自己在 Airbnb 的经历——资深撰稿人 [安娜·梅兰](https://www.vice.com/en_us/contributor/anna-merlan) 在他们收到的近 1000 封电子邮件中，总结了一些最常见的 Airbnb 骗局。

我们从这些骗局中学到了什么？好吧，骗子们很聪明，他们会利用 Airbnb 的宽容来检查房源。下面是一些你可能要提防的常见骗局，以及尽可能解决这种情况的方法:

### 最后一分钟“偷梁换柱”到另一处房产

你向主人预订了一处房产。在您预定到达的前几天(甚至几个小时)，您预订的酒店由于一些不可预见的问题而不再可用，无论是管道问题还是住在那里的最后一位客人的明显问题。为了帮助你，房东会给你提供他们出租的另一处房产，与你最初预订的房间相比，这通常是一个垃圾房。

**解决方法:**拒绝备选选项，确定主持人取消。(如果您取消，*您*有责任支付相关的取消费用。)如果你陷入困境，请联系 Airbnb 寻找替代选择。

### 资金转移

你给一个主人发信息询问他们的房产。他们要求在 Airbnb 平台之外付款，如果你同意，可能会提供更好的价格。最后，公寓很糟糕，但因为你同意了 Airbnb 条款之外的交易，所以你不受保护，也没有资格获得退款(或者这是一个完全虚假的房产，你完全被骗了)。

**解决方案:**像 [我们已经写了一遍又一遍](https://lifehacker.com/avoid-these-red-flags-while-booking-an-airbnb-1838842601) ，永远不要在 Airbnb 的平台之外付费。(事实上，尽可能保留所有给 Airbnb 的消息，以便他们可以使用你的消息历史来解决任何问题。)

### 对你从未造成的损害收费

经过多天的逗留，你离开的财产在原始状态。几天后，你通过 Airbnb 的信息平台收到了一份投诉，要么是主人，要么是 Airbnb，声称你损坏了家中的一些东西，并负责赔偿主人。

**解决方案:** [在离开](https://lifehacker.com/take-photos-of-your-airbnb-before-leaving-1833950599) 之前，给你的财物拍些照片和视频，发给你的主人，以示礼貌的“感谢”，并在他们索赔时保护自己。或者把它们保存起来，并附上一些有时间戳的证明，比如一份当前的报纸或主人确认你离开的信息。(事实上，你还应该养成一个习惯，在到达后立即给你的财产拍照，并将任何问题发送给你的主人；也许损害*是合法的*但是你前面的客人要负责。)
