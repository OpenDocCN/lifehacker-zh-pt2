# 如何加快你的 Android 热点连接

> 原文：<https://lifehacker.com/how-to-speed-up-your-android-hotspot-connection-1836597319>

能够将您的智能手机变成一个微型接入点，可以为其他设备(如您的笔记本电脑)提供互联网连接，这是有史以来最好的功能之一。它救了我很多次，当我需要使用笔记本电脑做一些事情(比如编辑 Lifehacker 的帖子)，但无法访问任何附近的 wifi 时，这种情况就更少见了。

虽然在你的 [iOS](https://support.apple.com/en-us/HT204023) 或 [Android](https://support.google.com/android/answer/9059108?hl=en) 智能手机上设置 wifi 热点非常容易，但如果你是 Android 用户，你会想要做一点小小的调整，以确保你连接到*和*智能手机热点的设备获得最佳速度。(对不起，iOS 用户；你这次运气不好。)

正如 [CNET](https://www.cnet.com/how-to/drastically-speed-up-your-android-phones-hotspot-with-this-simple-setting/) 在最近的一篇文章中解释的那样，你需要确保你的 Android 设备正在 5GHz 频段上创建其第二 wifi 网络，而不是 2.4GHz。正如 [我们之前解释过的](https://lifehacker.com/which-wifi-band-should-i-use-for-my-devices-1832635625) ，对 wifi 频段的一种过于简单的理解是，2.4GHz 可以以较低的速度提供更长的范围，而 5Ghz 可以在较短的范围内提供极快的速度。

虽然你应该经常测试设备使用 2.4GHz 还是 5GHz 频段更有意义，但这更多是针对你在家里或公寓走动时的情况——例如，你可能在远处获得 2.4GHz 比 5GHz 更强的连接。当您将智能手机设置为 wifi 热点时，您的智能手机很可能离其他需要上网的设备很近。在这种情况下，强制该热点为其 wifi 网络使用 5GHz 而不是 2.4GHz 是最有意义的。

要设置这个，你需要导航到你的 Android 设备上的 wifi 热点选项，很可能在它的**设置** > **网络&互联网**屏幕中找到(或者在你的特定 Android 上叫它什么)。例如，这是像素 3 的外观:

一旦你打开你的 wifi 热点(或开始设置过程)，你会想要寻找一个配置或高级设置屏幕。同样，不同的 Android 设备有不同的屏幕，但找到你的热点设置应该不难。在 Pixel 3 上，这就像在你打开和关闭 wifi 热点的地方点击“高级”部分一样简单。

一旦到了那里，寻找一个允许你改变热点的 wifi 波段的选项。如果你以前从未这样做过，那么你的热点的 wifi 网络默认为 2.4GHz。你会想把它设置为 5GHz。

虽然你可能不需要打开或关闭你的热点来确保你的改变生效，但这并没有坏处。如果你真的很好奇，那么你可以使用一个 [wifi 扫描应用](https://lifehacker.com/which-wifi-channels-should-i-use-for-my-wireless-networ-1832788063) 来确认你的热点网络运行在 5GHz，而不是 2.4GHz。如果你想测量速度差异，你也可以从连接到你的热点 wifi 网络的设备上运行“之前”和“之后” [速度测试](https://lifehacker.com/how-to-make-sure-youre-getting-the-internet-speeds-your-1827763762) 。
