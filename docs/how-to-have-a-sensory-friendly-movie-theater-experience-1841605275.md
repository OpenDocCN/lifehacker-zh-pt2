# 如何拥有感官友好的电影院体验

> 原文：<https://lifehacker.com/how-to-have-a-sensory-friendly-movie-theater-experience-1841605275>

电影院对任何人来说都是一种过度刺激的体验。有大于生活的行动，弥漫着黄油爆米花的气味，音量调到一个可笑的水平。但是对于一个有感觉处理障碍的孩子——或者因为其他原因感觉敏感的孩子——这可能是一种压倒性的经历，父母可能会选择完全避免它。



这就是为什么，一个月有几次，一些电影院会打开灯光，调低声音，让孩子们起床，根据需要四处走动。

AMC 剧院与自闭症协会 合作，推出了其 [感官友好电影](https://www.amctheatres.com/programs/sensory-friendly-films) 计划，该计划在每月的第二个和第四个周六放映一部适合家庭观看的电影，并在周三晚上为成年观众放映一部电影。本周六接下来是*索尼克* *刺猬*在 [近 200 个参赛地点](https://www.amctheatres.com/programs/sensory-friendly-films) 。

同样，根据其网站 的说法，富豪影院的“我的方式日场节目”是一个“安全的空间，在这里我们的客人可以在欣赏好莱坞最新电影的同时，通过唱歌，哭泣，跳舞，走动，说话或喊叫来自由表达自己”。演出时间为 3 月的第二个和第四个星期六的上午 10:30，地点为 [参与地点](https://www.regmovies.com/static/en/us/promotions/my-way-matinee) ，每张票的折扣价为 6.5 美元。
