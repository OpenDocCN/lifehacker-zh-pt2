全新的 Instagram 更新终于为该应用程序添加了一个备受期待的功能:用户可以在一个 Instagram 故事中上传多张照片的拼贴画，并使用新的“布局”模式来创建更多引人入胜的故事帖子。

Instagram 的故事帖子布局模式可能会缺少一些定制选项或一些第三方替代品拥有的额外功能，但这是一个更方便的选项。你不必安装任何额外的东西，在多个应用程序之间周旋，或者摆弄大量的复制和粘贴——现在这一切都发生在 Instagram 的主应用程序中。

### 如何在 Instagram 中创建多照片故事帖子

Instagram Stories 的布局功能今天在全球范围内推出，但具体的可用性可能会因您的设备和地区而异。如果你无法使用以下步骤找到新选项，请确保你已经在 [iOS](https://apps.apple.com/us/app/instagram/id389801252) 或 [Android](https://play.google.com/store/apps/details?id=com.instagram.android&hl=en_US) 上更新了 Instagram 应用。如果该选项仍然不可用，请耐心等待，它应该很快就会出现。

对于那些已经有这个选项的人来说，它是这样工作的:

1.  打开 Instagram，点击应用程序左上角的相机图标，开始新的故事帖子。
2.  向左滑动底部的选项转盘，直到找到**“布局”**
3.  拍几张照片放在你的故事帖子里(总共最多可以放六张)。
4.  选择你想用于拼贴画的任何网格状布局。有几个布局选项可供选择(而且，不，你不能编辑你布局中的每一张照片)。
5.  与常规的 Instagram 故事类似，你可以添加其他元素，如文本、标签、过滤器、表情符号、绘图、gif 等。
6.  对布局满意并准备好分享后，点击**“发送到故事”**
