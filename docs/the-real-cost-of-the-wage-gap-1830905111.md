# 工资差距的真实成本

> 原文：<https://lifehacker.com/the-real-cost-of-the-wage-gap-1830905111>

每个人都熟悉这样一个统计数据:在美国，女性的平均收入是 79 比 T1，80 美分比一个白人男子的 1 美元 T2 T3，但是工资差距的真实成本要比这微妙得多。

根据全国妇女和家庭伙伴关系 2017 年的数据，美国男性平均每年比美国女性多赚 10169 美元，尽管这因地点、种族和行业而异。在整个职业生涯中，光是工资一项，普通男性加起来就比其女性同行多赚 50 万美元左右。

但那只是基本工资。如果你考虑到 401(k)缴款、债务偿还、退休收入和工资增长——所有这些或多或少都取决于最初的工资——差距就会急剧扩大，并对女性财务生活的各个方面产生连锁反应。如果这一差距被拉平，普通的美国职业女性将有足够的钱支付 14 个多月的儿童保育费用，或者可以在不到三年的时间里还清她的学生贷款。
