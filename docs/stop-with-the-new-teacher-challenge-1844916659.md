# 停止“新教师挑战”

> 原文:[https://life hacker . com/stop-with-the-new-teacher-challenge-1844916659](https://lifehacker.com/stop-with-the-new-teacher-challenge-1844916659)

在最新一轮*中，人类到底发生了什么？*、现在抖音上有父母给他们的孩子看残疾人或患有影响他们外貌的疾病的人的照片。然后他们假装这些陌生人是他们孩子的新老师。因为，很明显，如果一个五岁的孩子对一个有伤疤或斗鸡眼的人或一个看起来严重醉酒的人做出恐惧的反应，那就太搞笑了。

当我写这篇文章时，标签为# [新教师挑战](https://www.tiktok.com/tag/newteacherchallenge?_d=secCgsIARCbDRgBIAIoARI%2BCjxLKQOAJN9liIagLYB%2BtpjctIXXJ0zJUit8%2BxzCKN194g7ePI1lC5%2BjLS%2F2UQp1Hxutott%2B0tDH35X6%2B%2BsaAA%3D%3D&language=en&preview_pb=0&share_item_id=6856858618357107974&share_link_id=E983CEAA-DEFA-410B-87F7-24B456C73C0C&timestamp=1598994751&tt_from=copy&u_code=0&utm_campaign=client_share&utm_medium=ios&utm_source=copy&source=h5_m) 的视频在抖音有超过 4200 万的点击量。抖音和其他社交媒体频道上的一些视频是公开反对的人，包括 [利兹·维拉斯奎兹](https://www.instagram.com/p/CDpMeamB2Qk) ，一位作家和励志演说家，她出生时患有极其罕见的先天性疾病，她的形象被用来向孩子们介绍他们的“新老师”但令人不安的是，大量带有该标签的帖子是人们试图震惊他们的孩子，让他们产生恐惧或厌恶的反应，这样他们就可以发布这种反应，让全世界观看并跟着笑。

对于做这个挑战的父母来说，他们可以张贴他们孩子的甜蜜，包容，无所畏惧的反应？那也不酷。这并不能说明你是一个多么好的父母——尤其是当 [你不断地用“你觉得她怎么样？”来刺激他们时，你几乎无法掩饰你的失望](https://www.tiktok.com/@mandi.lyn/video/6856858618357107974?_d=secCgsIARCbDRgBIAIoARI%2BCjxLKQOAJN9liIagLYB%2BtpjctIXXJ0zJUit8%2BxzCKN194g7ePI1lC5%2BjLS%2F2UQp1Hxutott%2B0tDH35X6%2B%2BsaAA%3D%3D&language=en&preview_pb=0&share_item_id=6856858618357107974&share_link_id=E983CEAA-DEFA-410B-87F7-24B456C73C0C&timestamp=1598994751&tt_from=copy&u_code=0&utm_campaign=client_share&utm_medium=ios&utm_source=copy&source=h5_m) 他们的回答是，“每个老师都很棒。”反应不是重点；参与一个因为人们的外表而嘲笑他们的挑战才是重点。

其中一些照片看起来像是面部照片。他们中的一些人是残疾人或病人。有些被 PS 过，让它们更震撼。都不好笑。

做这样的事情积极地教导孩子们如何成为网络恶霸。嘲笑一个孩子对一个陌生人出现的负面虚拟反应并不比嘲笑他们对杂货店里的某个人有这样的反应好。

我知道我可能是在向唱诗班说教，我认为任何认为这种事情很有趣的人都不太可能被关于通过模仿移情 来帮助孩子发展更多移情 [的争论所动摇。但是如果你想用](https://lifehacker.com/want-to-raise-empathetic-kids-show-em-how-its-done-1831050481) [新教师挑战赛](https://www.tiktok.com/tag/newteacherchallenge?_d=secCgsIARCbDRgBIAIoARI%2BCjxLKQOAJN9liIagLYB%2BtpjctIXXJ0zJUit8%2BxzCKN194g7ePI1lC5%2BjLS%2F2UQp1Hxutott%2B0tDH35X6%2B%2BsaAA%3D%3D&language=en&preview_pb=0&share_item_id=6856858618357107974&share_link_id=E983CEAA-DEFA-410B-87F7-24B456C73C0C&timestamp=1598994751&tt_from=copy&u_code=0&utm_campaign=client_share&utm_medium=ios&utm_source=copy&source=h5_m) 做任何事情，用它来告诉孩子们，如果或当那些照片中的人发现他们在互联网上被无数陌生人嘲笑时，他们会有什么感受。用它向孩子们展示如何在社交媒体上报告这类欺凌内容。然后和他们一起疯狂“报道”。
