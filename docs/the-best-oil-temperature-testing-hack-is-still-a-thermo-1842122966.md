# 最好的油温测试工具仍然是温度计

> 原文：<https://lifehacker.com/the-best-oil-temperature-testing-hack-is-still-a-thermo-1842122966>

黑客都是关于花更少的时间，使用更少的设备，让一个人的生活更容易。但是有时候没有捷径或替代品。有时候你不得不走远路，或者买合适的工具。一种这样的工具是数字温度计。



温度控制是烹饪中非常重要的一部分，你很难控制一些你无法测量的东西。温度计是测量温度最简单、最明显的方法，但这并不妨碍人们尝试其他更有创意的方法。

就在今天早上，我正在浏览抖音烹饪视频集，因为我非常年轻和时尚，这时这个麦乐鸡欺骗视频引起了我的注意。

bOfvBY">从表面上看，这些黑客似乎不错，我喜欢被愉快的流行音乐提醒的想法！—但我最初有些担心。</p><p class="sc-77igqf-0 bOfvBY">首先，并非所有的筷子和木勺都是一样的，涂层、处理和清洗后残留的湿气会影响它们在热油中的反应。另一方面，爆米花是一种有机植物物质，它并不以一致性而闻名。当里面的水变成蒸汽时，爆米花就会爆开，这发生在 212 华氏度——远低于 325 华氏度，这是油炸任何东西的绝对最低温度。(我通常以 360℉为目标，因为当你向油中添加食物时，油会冷却。)</p><p class="sc-77igqf-0 bOfvBY">玉米达到爆裂温度的速度取决于单个玉米的含水量(可以变化)和玉米粒用油加热的时间，这让我认为这不是最可靠的方法。</p><p class="sc-77igqf-0 bOfvBY">总之。我没有坐在沙发上思考理论上的含水量，而是决定拿起我的数字温度计，实际测量一些东西。我拿了些冷油，倒在荷兰烤箱里，加了三粒爆米花粒。我还在那里放了一把(无涂层的)木勺。当油温达到 278 华氏度时，第一个果仁爆开了。最后一次爆炸是在华氏 283 度。如果你在这个温度下开始油炸食物(这个温度太低了)，温度会降得更低，你的食物会吸收太冷的油，导致油腻、恶心的食物。</p><p class="sc-77igqf-0 bOfvBY">然而，勺子在 330 华氏度左右开始冒泡，这在 325-350 华氏度的油炸范围内，尽管对我来说还是有点太冷了。再说一次，我只能说这对于那把勺子来说是正确的，而且声称它对所有木制烹饪工具都有效(尽管它似乎对约书亚的筷子也有效)也不会感到舒服。我<em>可以</em>测试我所有的勺子，看看哪个气泡在什么温度下，但我需要一个温度计来这样做。</p><p class="sc-77igqf-0 bOfvBY">我不会告诉约书亚停止用筷子检查他的煎炸油。约书亚做得很好，至少他没有用一块爆米花，这是一个非常糟糕的黑客行为。</p><p class="sc-77igqf-0 bOfvBY">但是，除非你已经有一根可靠的筷子(或其他一些木头)，否则就买一个该死的数字温度计吧。更安全(尤其是煮肉的话)，更精准，不依赖于一片玉米的含水量，挺好的。</p> </body>
