# 当你在工作中陷入谈话时，试着站起来

> 原文：<https://lifehacker.com/when-youre-stuck-in-a-conversation-at-work-try-standin-1840698534>

我现在一个人在家工作，很光荣。在我之前；然而，我和大多数人一样在办公室工作，也和大多数人一样，我不喜欢和同事进行冗长的谈话。



特别是一位同事，他过去经常来我的办公桌前聊天，嗯，几乎无所不谈。她显然是在试图逃避她需要做的工作，或者可能已经完成了，所以她决定在我的桌子旁边告诉我她最近的假期，她午餐吃了什么，以及在大厅对面工作的那个家伙如何“如此讨厌”。

你知道谁很烦人吗？她。

我绝对不介意和同事聊天，我自己也做了很多，但她总是设法在最糟糕的时候出现，不明白她打断了什么，然后呆得比她受欢迎的时间长得多。

在向另一个朋友讲述了这个故事后，她提出了一个很好的建议:试着站起来。

当你站起来的时候，坐在你桌子旁边的人会认为你有事要做，并希望决定结束谈话离开。

根据您办公室的设置，您的里程可能会有所不同。对我来说，我会利用她来访的第二分钟来决定给我的咖啡续杯还是复印几份。是的，有时你会冒被跟踪的风险，但有时这是结束谈话的好方法。

站起来本身就传达了一个信息，那就是你要去某个地方。取决于你的办公桌在哪里，你甚至可以站起来，让那个人继续走他们的路，然后又坐回去。

当然，每 10 个人中就有一个人不明白你的暗示，无论你去哪里都跟着你。在这种情况下，可能是时候进行一次严肃的谈话了，也许 [一副降噪耳机](https://lifehacker.com/how-do-i-stop-coworkers-from-distracting-me-while-i-m-t-5869367) 。
