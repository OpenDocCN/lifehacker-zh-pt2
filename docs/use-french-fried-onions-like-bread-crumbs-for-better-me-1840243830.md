# 使用像面包屑一样的法式炸洋葱做更好的肉丸

> 原文：<https://lifehacker.com/use-french-fried-onions-like-bread-crumbs-for-better-me-1840243830>

面包屑很好——高贵，甚至。没有它们，肉丸将会是密集的小圆球，而肉面包将会是一块块的肉。不会有酥脆的裹着面包屑的煎鱼，也不会有烤鸡肉。我不想生活在一个没有面包屑的世界，但这并不意味着我不想有时改变它，用法式炸洋葱——你可以在 [*那*砂锅](https://lifehacker.com/how-to-make-the-creamiest-vegan-green-bean-casserole-1840029669) 上找到的那种——来代替它们。



法式炸洋葱块一旦粉碎，就能发挥面包屑的许多功能，而且尝起来像洋葱，这是一种有趣的 T2 口味。你可以将它们混合到肉中，制成额外美味的肉丸和肉卷，在油炸前将一片鱼从其中穿过，或者在一些黄油中用真正的面包屑炒它们，制成[-的沙拉浇头](https://skillet.lifehacker.com/radicchio-makes-a-superior-thanksgiving-salad-1839866691) 。

然而，我最兴奋的是一个更洋葱，更美味的肉丸。你甚至不需要调整你的尺寸，除非你习惯于使用极淡的面包屑，在这种情况下，你可能想减少盐。只要抓起一罐法国的(Trader Joe 的品牌也不错)，让食品加工机做它的事情，然后量出你想要面包的洋葱屑。洋葱块会把你的球粘在一起，同时增加一大堆鲜味。如果有负面影响的话，我看不出来。
