# Mozilla 的新 Firefox Android 应用目前只支持这些扩展

> 原文：<https://lifehacker.com/mozillas-new-firefox-android-app-only-supports-these-ex-1841673071>

Mozilla 即将从谷歌 Play 商店移除其新的 Firefox Android 浏览器的夜间测试版，只留下 [Firefox Preview](https://lifehacker.com/everything-you-need-to-know-about-firefox-preview-on-an-1835950429) 供用户试用。现在从夜间版本切换到这个浏览器是值得的，或者第一次瞥一眼，但有一个小问题:目前只支持一个扩展，你的许多最爱可能不会跳转到这个新版本的 Firefox。

Watch

根据 Mozilla 最近发布的消息，在 Mozilla 的“ [推荐扩展](https://support.mozilla.org/en-US/kb/recommended-extensions-program) ”计划中，只有 97 个扩展将在新版 Firefox for Android 最终发布稳定版本时得到支持。现在，你在 Firefox 预览版中只能看到一个: [uBlock origin](https://lifehacker.com/how-to-make-ublock-origin-even-better-at-ad-blocking-in-1839964889) 。Mozilla 也没有承诺未来会将推荐包中没有的扩展添加回 Firefox mobile，尽管该公司表示愿意尝试。

这是什么意思？如果你已经在当前版本的 Android 版 Firefox 中使用了扩展，你会想要将它们与 Mozilla 的列表进行交叉引用，看看它们是否会出现在今年春天发布的新版应用中。否则，你需要为这些附件找一个替代品。

为了让你开始，这里有一些 Mozilla 列表中我们最喜欢的。现在就开始试用吧，这样一旦新的火狐版本登陆 Android，你就会熟悉它们了:

*   [**黑暗阅读器**](https://addons.mozilla.org/en-US/firefox/addon/darkreader/?src=featured)——自动开启黑暗模式，即使在一些没有原生黑暗模式的网站上。
*   [](https://addons.mozilla.org/en-US/firefox/addon/decentraleyes/?src=featured)**-防止你的数据被谷歌和其他公司收集。**
*   **[](https://addons.mozilla.org/en-US/firefox/addon/ghostery/?src=featured)**——屏蔽广告和追踪器。****
*   ****[**HTTPS 无处不在**](https://addons.mozilla.org/en-US/firefox/addon/https-everywhere/?src=featured) -为通常可能不具备 HTTPS 加密功能的网站添加该功能，以实现更安全的浏览。****
*   ****[**NoScript 安全套件**](https://addons.mozilla.org/en-US/firefox/addon/noscript/?src=featured) -提醒您页面上运行的潜在恶意 web 脚本，并让您控制它们是否运行。****
*   ****[](https://addons.mozilla.org/en-US/firefox/addon/privacy-badger17/?src=featured)**——屏蔽隐形追踪器。******
*   ******[**uBlock Origin**](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/?src=featured)-在不降低设备性能或网站加载速度的情况下阻止广告和其他屏幕元素。******
