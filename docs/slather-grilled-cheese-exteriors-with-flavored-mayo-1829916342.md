# 在烤奶酪外层涂上大量调味蛋黄酱

> 原文：<https://thetakeout.com/slather-grilled-cheese-exteriors-with-flavored-mayo-1829916342>

“重新发明轮子”这句话经常被提起，但老实说，这种方法是一个新发现——至少对烤奶酪来说是这样。



诚然，在过去的几个月里，我谈论蛋黄酱的次数比我一生中还多。要么是在蛋黄酱/番茄酱口味测试 期间，要么是在蛋黄酱/花生酱口味测试 [期间，我的同事布莱恩说他用蛋黄酱而不是黄油煎他的烤奶酪。](https://thetakeout.com/peanut-butter-and-mayo-sandwiches-1829345467)

就像我在 之前听说过但从未尝试过的其他 [蛋黄酱一样，我立刻被激起了兴趣。布莱恩对这种方法的功效坚信不疑。我喜欢他的想法，因为对于孩子们，我做了很多烤奶酪，最糟糕的是总是在面包上涂上柔软但可能不够柔软的黄油，而不刺穿它。蛋黄酱？显然，像梦一样蔓延。](https://thetakeout.com/peanut-butter-and-mayo-sandwiches-1829345467)

然后*外卖*编辑彭凯文彻底颠覆了整件事，建议用 [蛋黄酱](https://thetakeout.com/how-to-make-lazyoli-the-emergency-aioli-for-urgent-dip-1826428604) 或*调味*蛋黄酱来煎你的烤奶酪。那会是什么味道？除了天堂，那是什么？

所以我拿了一些胡椒和墨西哥胡椒蛋黄酱，搭配我家里的沙拉蛋黄酱和普通蛋黄酱，然后开始油炸。一些细节立刻变得清晰起来。首先，大约一汤匙的量就可以涂在每一片面包上，否则它会粘糊糊的。此外，蛋黄酱似乎比黄油煮得更快，所以在烹饪足够长的时间让奶酪变得美味和融化的同时不烧焦边缘有点棘手。Bryan 建议经常翻转；我还用了比平时更低的火来煎烤奶酪，用中火而不是中火。

虽然普通的蛋黄酱美味地将烤奶酪包裹在舒适的脂肪层中，但调味蛋黄酱无疑将事情带到了一个新的水平。胡椒版本是令人信服的 mustardy，sriracha 增加了一些热量，但 chipotle 也有一些美味的香草和香料现在很好地点缀在面包的外部。通过在烤奶酪上涂上蛋黄酱，你可以得到蛋黄酱的全部味道，而不会被粘稠的调味品所困扰。

我有点不敢相信它有多好。你仍然可以在你的烤奶酪中添加任何你喜欢的东西——番茄、培根、火鸡——但是外面的调味蛋黄酱会让你的普通烤奶酪变成美食选择。尝试你喜欢的任何一种 sriracha 和 chipotle 肯定是为喜欢辛辣食物的人准备的，但你也可以尝试 [garlicky aioli](https://thetakeout.com/how-to-make-lazyoli-the-emergency-aioli-for-urgent-dip-1826428604) 或 [更厚的日本风味](https://thetakeout.com/my-chemical-dependence-on-japanese-kewpie-mayo-1798251538) 。或者， [做你自己的调味蛋黄酱](https://thetakeout.com/spread-the-love-tips-to-make-you-a-homemade-mayo-maste-1823928507) ，也许用一种草药成分，然后在里面油炸。可能性——至少对于烤奶酪来说——实际上是无穷无尽的。你以后会感谢我的，不客气。
