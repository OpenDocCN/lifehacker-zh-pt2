# 在关税导致价格上涨之前应该储备什么

> 原文：<https://lifehacker.com/what-to-buy-now-before-the-next-round-of-tariffs-1834782036>

如果有一个我们反复听到的关于贸易战的信息，那就是生活将 [变得更加昂贵](https://twocents.lifehacker.com/how-increased-china-tariffs-might-cost-you-money-1834670162?_ga=2.140631730.1451944291.1557756156-173178311.1556715687) 。这可能是现在购物的一个理由。



特朗普总统最近一轮将美国从中国进口的关税从 10%提高到 25%，其中包括一些可能会让你不知不觉的上涨，比如食品和清洁产品的关税。床垫和行李等物品也将大幅涨价。

但这还不是全部，伙计们:经济学教授 Katheryn Russ [告诉美国消费者新闻与商业频道](https://www.cnbc.com/2019/05/14/what-consumers-should-buy-now-as-the-trade-war-heats-up.html) ，另一轮额外类别的关税增加可能会在初秋开始。这些涨价尤其会影响到电子产品，可能会让你更难坚持假期预算。

路透社指出，“拟议的清单将涵盖几乎所有先前关税未触及的消费品”。

在预期的下一轮价格开始飙升之前，什么值得购买？如果你已经计划在以下领域购买，你可能希望尽早购买。

*   返校用品:衣服、鞋子、背包和转笔刀
*   割草机
*   体育用品和野营设备
*   电子产品:视频游戏机、电视、笔记本电脑、平板电脑和智能手机

全国零售联合会的大卫·弗伦奇在接受《美国消费者新闻与商业频道》采访时说:“如果你计划六周后进行一次大采购，你可能想今天就进行。”。

但当然，精明地做到这一点需要手头有现金。如果你还没有计划好的话，不要跑出去为你全家的新电视充电。如果你真的在市场上购买上述物品，请步行，不要跑步。
