# 为什么越来越多的视频显示器依赖 USB-C

> 原文：<https://gizmodo.com/why-more-and-more-video-monitors-are-relying-usb-c-1844110486>

就在我们似乎都已经决定使用一种行业标准的视频输入/输出时，又出现了另一种让事情变得复杂的标准——现在你会发现 USB-C 正在与现代计算和游戏显示器上的 HDMI 和其他端口竞争。在这里，我们将解释为什么会发生这种情况，以及从长远来看对消费者的利弊。



关于 USB-C 显示器的兴起，要理解的第一点是，严格来说，USB-C 只是端口物理接口的规格，它独立于通过电缆流向端口的技术。连接到 USB-C 端口的电缆通常会使用 USB 技术，但并不总是如此，尤其是在视频和显示器方面(USB 在这方面做得不太好)。

USB-C 端口可以接受传输各种音频和显示协议的电缆，包括 Thunderbolt、DisplayPort 和 HDMI(尽管这比前两种少得多)。这是制造商可能将 USB-C 端口而不是 HDMI 端口放在显示器上的主要原因之一:它的功能更加多样。

还有其他原因:USB-C 端口比 HDMI 端口更小，因此它们占用的空间更少(尽管这在笔记本电脑上比显示器更重要)。USB-C 也不在乎你用哪种方式插入电缆，这一点对于 HDMI 来说就不一样了——当你试图把手伸到显示器后面插入东西时，这是一个关键的考虑因素。USB-C 电缆在更长的距离上也能更好地工作。

作为一个额外的奖励，这些视频技术的末端带有 USB-C 插头，除了视频和音频之外，通常还可以支持某种 USB 传输。例如，一种应用方式是将触摸屏输入从连接的显示器反馈到笔记本电脑。从显示器引出的 USB-C 电缆也可以用来给设备充电和传输数据，将它们变成有效的 USB-C 集线器。

当然，USB-C 标准的所有这些好处——多功能性、紧凑性、易用性——意味着它们现在在笔记本电脑上变得非常普遍，尤其是那些更小更轻的笔记本电脑。这反过来推动了显示器的采用，因为制造商希望迎合最大的市场份额。

现代 MacBooks 通过 USB-C 上的雷电 3 提供 do 视频输出，而最近的 Surface 硬件通过 USB-C 输出 DisplayPort 1.4。在笔记本电脑上保留全尺寸的 HDMI 端口只会增加体积。

虽然 USB-C 视频的兴起给我们提供了更多的选择，但也增加了更多的困惑。当你真正选择 USB-C 显示器，或者试图在 USB-C 和 HDMI 显示器之间进行选择时，有一个警告:有很多令人困惑的组合，所以在你的研究中要尽职尽责。从你的源设备(可能是你的笔记本电脑)开始，弄清楚它是如何输出视频的，然后从那里开始。

混淆的主要原因是，并不是所有的 USB-C 端口 [都是相同的](https://gizmodo.com/usb-c-is-a-mess-how-to-navigate-it-1826978043) :仅仅因为一个端口是 USB-C 形状的，并不能真正保证它。并不是每个 USB-C 口都能给你的设备充电并提供视频输出(或视频输入)。这完全取决于制造商做出的选择。

顺便说一下，这也适用于电缆——给你的手机充电的 USB-C 电缆不一定能够从你的笔记本电脑上获取视频并将其放在大屏幕上。当你浏览产品列表时，这一点应该很清楚，但通常不是这样，所以你需要对你买的东西小心。

换句话说，仅仅因为你的笔记本电脑带有一个提供视频输出的 USB-C 端口，并不意味着你可以直接将其连接到 USB-C 显示器上。谷歌 Pixelbook 就是一个很好的例子——你可以使用 USB-C 连接器来连接第二台显示器，但你需要在某个地方安装一个适配器。

当你浏览笔记本电脑、电缆和显示器时，请寻找“Alt 模式”的参考资料——这是 USB-C 连接支持比 USB 更多的功能(包括 Thunderbolt 和 DisplayPort 音频和视频传输)。如果你幸运的话，你可以用一根 USB-C 线把你的电脑和显示器连接起来；如果没有，你需要一个适配器。

当然，HDMI 在版本、线缆和实现方面都有问题，但 USB-C 正在努力在复杂性方面超越它: [例如，查看这篇](https://dancharblog.wordpress.com/2019/02/23/microsoft-surface-usb-c-port-capabilities/) 令人眼花缭乱(并且经过充分研究)的关于各种微软 Surface USB-C 视频输出选项的指南，或者 [这篇精彩深入的](https://www.bigmessowires.com/2019/05/19/explaining-4k-60hz-video-through-usb-c-hub/) 以及必要复杂的关于添加集线器如何影响 USB-C 视频质量的解释

抛开购物挫折和困惑不谈，HDMI 连接器标准不会很快从显示器(或电视)上消失——有太多的视频控制台、蓝光播放器和有线电视盒会发生这种情况。然而，它在笔记本电脑上可能会变得更加稀缺，因为制造商希望节省空间(同时也知道用户可以广泛使用 USB-C 加密狗和集线器)。

遗憾的是，当涉及到从一个地方到另一个地方获取视频时，最初提供更简单、更方便的设备连接和充电方式的标准变得如此复杂，但我们希望随着旧设备的淘汰和 USB-C 变得更加普遍，会出现更多的清晰度。现在，只要记住笔记本电脑或显示器上友好的 USB-C 端口可能有各种不同的作用，这需要仔细研究。
