# 如何鼓励孩子的善良

> 原文：<https://lifehacker.com/how-to-encourage-kindness-in-kids-1837409755>

我们通过自己树立善良的榜样来教育孩子善良，和他们谈论善良，当我们发现他们善良时表扬他们。我们也可以用毛绒球来教它。



一位老师向 Edutopia 分享说，当她在教室里增加了一个仁慈的罐子和一堆彩球后，她的学生开始减少争吵，更多地互相帮助，整个教室的气氛变得更加平和。每当一个学生做一些善良的事情，比如分享，分发论文或对同学说一些好的话，他们可以把一个球放在瓶子里。

当罐子装满时，全班将会得到一份特别的款待或玩游戏来庆祝:
