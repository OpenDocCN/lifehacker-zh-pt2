# Todoist 最新“基础”更新中的最佳功能

> 原文:[https://life hacker . com/the-best-features-in-todoists-new-foundations-update-1839232738](https://lifehacker.com/the-best-features-in-todoists-new-foundations-update-1839232738)

我们最喜欢的 [待办应用](https://lifehacker.com/five-best-to-do-list-managers-5924093)——恰当地命名为 [待办](https://todoist.com)——今天进行了一次大升级。你将看到它在未来 24 小时左右开始推出，所有使 Todoist 更容易使用的新功能都值得等待尝试。

Watch

根据 Todoist 的说法，所有这些功能都将出现在所有版本的服务中——免费的*和* premium。对于那些仍在尝试任务管理的人来说，这是一个好消息，他们不愿意为 Todoist Premium 支付每月 3 美元。

### **新 Todoist 部分**

现在，您可以将单独的“部分”添加到您正在处理的项目中，这样您就可以更好地组织项目中的各种任务。当然，你总是可以用缩进、标签和优先级来给你的各种任务排序，但是这些奉献部分让它变得无比简单。

### **向“任务视图”问好**

Todoist 现在有一个专门的“任务视图”，你可以用它来更好地处理你告诉自己必须完成的许多事情。您可以在这个专用视图中编辑任务的所有常规细节—截止日期、优先级和注释—新的专用子任务选项卡使您更容易跟踪需要完成的任务的微小细节。

### **新的 iOS 和 Android 技巧可以更快地添加任务**

如果你在 [iOS](https://apps.apple.com/us/app/todoist-to-do-list-tasks/id572688855) 或 [Android](https://play.google.com/store/apps/details?id=com.todoist&hl=en_US) 上使用 Todoist 应用程序，你现在有更多的技巧来将任务添加到你强大的待办事项列表中。Todoist 重新设计了它的“快速添加”选项，现在你可以选择直接从界面添加任务提醒。你也可以拖动大红色的“添加”按钮到你的任务列表的任何地方——或者作为一个现有任务的子任务——在那里放入一个新的项目。这目前只能在 iOS 上运行，但 Android 对该功能的支持正在进行中。

### **停止搜寻共享标签**

这是一个微小的调整，但它很有用。现在，每当您使用与他人(如您的工作伙伴)共享的标签时，这些标签将出现在您的标签下拉菜单下单独的“共享”部分。这将帮助你更容易区分别人需要知道的任务和你可能只关心的任务。
