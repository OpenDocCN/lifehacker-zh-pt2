# 在哪里免费申报州税和联邦税

> 原文:[https://life hacker . com/where-to-file-state-and-federal-taxes-for-free-1822381090](https://lifehacker.com/where-to-file-state-and-federal-taxes-for-free-1822381090)

这是正式的纳税季节。美国国税局将于 1 月 27 日星期一开始处理个人纳税申报单，这意味着你有几周的时间进行组织。

除非你拥有一家企业，有多种收入来源，或者有大量的扣除额或复杂的投资，否则没有必要聘请专家为你报税——免费的在线软件就足够了(你应该电子申报: [根据美国国税局](https://apps.irs.gov/app/understandingTaxes/student/whys_thm06_les03.jsp#taxTrivia) 的说法，纸质申报的错误率为 21%，相比之下，电子申报的错误率为 0.5%)。任何年收入低于 69，000 美元的报税人都可以使用免费的报税软件。

由于美国国税局在 12 月宣布了一项协议，今年找到免费文件程序应该会容易一点。该协议禁止 [的免费文件程序的参与者在搜索结果中隐藏他们的免费选项](https://twocents.lifehacker.com/here-is-turbotaxs-actual-link-to-file-for-free-1834380008) 。

以下是您提交 2019 年州税和联邦税的一些免费电子申报选项:

*   [**信用因缘**](https://www.creditkarma.com/tax) :无限制。
*   [**H&R Block Free File**](https://www.hrblock.com/ffa/?otpPartnerID=5359&campaignId=pw_mcm_5359_0001):年龄在 50 岁以下，AGI 为 69，000 美元或以下，或者您有资格获得所得税抵免。
*   [**Intuit TurboTax 免税文件**](https://freefile.intuit.com/):36，000 美元或以下的 AGI，或者你有资格获得所得税抵免。
*   [](https://www.olt.com/main/oltstateff/default.asp)**:6.9 万美元以下的 AGI。**
*   **[**TaxAct**](https://www.taxact.com/applications/free-file/)**:**年龄在 56 岁以下，AGI 为 59，000 美元或以下，或者您有资格获得收入抵免。**

**对于上面列出的所有项目，如果你是现役军人，AGI 限额是 69，000 美元。**

**还有其他几个项目提供免费的联邦纳税申报表，但只为选定的州提供免费的州纳税申报表。你可以在这里 找到完整的列表或国税局批准的自由文件选项 [。](https://apps.irs.gov/app/freeFile/jsp/index.jsp)**

**如果你的 AGI 高于 69，000 美元，你可以使用国税局的 [免费文件填写表格](https://www.irs.gov/filing/free-file-do-your-federal-taxes-for-free) 进行联邦申报。它不会像为低收入人群所做的那样带你走完整个过程，但它会为你做算术并免费归档。**

**此外，如果您的收入不超过 56，000 美元、有残疾或英语能力有限并需要帮助，您可能有资格参加国税局的 [志愿者所得税援助计划](https://www.irs.gov/individuals/free-tax-return-preparation-for-you-by-volunteers) ，在该计划中，国税局认证的志愿者提供免费税务准备。美国国税局的 [老年人税务咨询](https://www.benefits.gov/benefit/722) 项目与之类似，但专门针对“老年人特有的养老金和退休相关问题”**
