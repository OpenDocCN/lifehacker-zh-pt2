# 为所有订阅自动续订设置提前警告

> 原文:[https://life hacker . com/set-up-advance-warnings-for-all-subscription-auto-renew-1831882226](https://lifehacker.com/set-up-advance-warnings-for-all-subscription-auto-renew-1831882226)

这些年来，我不可能是唯一一个因为忘记在免费试用(hello，Skillshare)结束时取消某项服务，或者因为希望使用该产品而推迟订阅的人。

Watch

万事达卡 [最近宣布](https://newsroom.mastercard.com/2019/01/16/free-trials-without-the-hassle/) 将要求公司获得客户的批准，在试用期结束时向他们收取经常性订阅费用。商家还必须通过电子邮件或短信发送收据，以及如何取消的说明。账单将包括商家的网站或商店的电话号码以及每笔费用。

最初的公告没有明确表示这项服务仅适用于实物产品，如每月维生素订阅和护肤样品盒。万事达卡不会要求提供数字订阅的商家，如网飞(和 Skillshare)，通知客户或发送收据。

万事达发言人 [告诉 *The Verge*](https://www.theverge.com/2019/1/17/18187227/mastercard-digital-subscription-auto-renewal) 说，这种区别是由于保健品的“模糊性”。当然，如果你用你的 Visa 或 Amex 来支付订阅费，这也帮不了你。

对于任何不符合条件的服务，将自动续费服务视为产品的“免费”试用: [在注册后立即设置日历提醒，](https://lifehacker.com/set-calendar-alerts-to-avoid-being-charged-for-your-fre-1823088482) 这样你就可以提前决定自己*是否真的*想多付一个月(或一年)的费用。

如果你忘记了你已经注册了什么以及你的订阅何时更新，你也可以 [使用一个预算应用](https://twocents.lifehacker.com/find-and-purge-unwanted-recurring-charges-with-this-vir-1795742670) 来帮助你。例如，Truebill 在一个列表中向您显示每月的重复费用，包括收费日期和金额，您可以直接在应用程序中取消订阅。(当然 Truebill 本身是需要付费订阅的。)Trim 通过短信或脸书提供类似的服务。

另一种选择？如果你只是想试用一个产品或服务，那么一旦注册就取消它。一些订阅将允许您在试用期 — 结束时保留访问权限，或者，如果您已经被收费并且无法获得退款(同样是 Skillshare)，则可以在付费期结束时保留访问权限。
