# 如果你有一段时间没有玩电子游戏了，从哪里开始呢

> 原文：<https://kotaku.com/where-to-start-if-you-havent-played-video-games-in-a-wh-1839035160>

也许你已经离开游戏很长时间了，现在想重新投入游戏。这可能会感觉非常令人生畏:有新的游戏，新的游戏机，新的俚语，很多青少年，Twitch 是怎么回事？这里有一些关于从哪里开始的提示。

Watch

我在高中玩了很多游戏，但在大学期间我没有时间或资源玩很多游戏。在我高三的时候，我最终在抽奖中赢得了一台免费的 Xbox 360，并决定重新尝试一下。这里有几个对我有效的方法，对我认识的几个有类似经历的人也有效。

### 从现在最火的游戏开始

当我拿到我的 Xbox 360 时，世界上最火的游戏是 *Skyrim* 。我以前从未玩过上古卷轴游戏，只是模糊地知道这个游戏是关于什么的。但我知道每个人都在谈论它，所以我决定把它捡起来。

玩热门的新游戏会让你和其他玩游戏的人有话题可聊。游戏网站会谈论它，论坛上的人也会谈论它，这可以帮助你再次感觉自己是社区的一部分。尽管对我来说, *Skyrim* 并不是一种超凡的体验，也没有成为我最喜欢的游戏，但我能够从它进入对我更有吸引力的游戏。

### 从你期待的最后一场比赛开始

那年夏天，我把我的 Xbox 360 带回家，让我弟弟在手术恢复期间玩。他想玩*刺客信条*和*黑色情人节*，因为这是他记得上次投资游戏时最想玩的两个游戏。(我还特别高兴地向不知所措的母亲解释说，是的，新游戏每个确实要 60 美元*。)*

当我哥哥整个夏天都躺在客厅的床上时，他跟我说要回到他从十几岁开始就没有去过的心态。我不会说他现在是一个大玩家，尽管他已经开始在他的新电脑上玩一些游戏。

重拾你曾经感兴趣的游戏可以帮助你回忆起最初是什么吸引你去玩游戏。如果你有一段时间没有玩游戏了，这些游戏中的一些可能也很便宜。有时候，当我寻找我在发布时不玩的游戏时，我也发现它们已经被移植到 PC 上，所以我可以在闲暇时在笔记本电脑上玩。

### 阅读关于游戏的文章，看看什么听起来最有趣

我有时会把我重返游戏归功于前 Kotaku 自由编辑 Kirk Hamilton。作为一个对新媒体艺术感兴趣的自命不凡的大学生，我试图为一个潜在的大学项目考虑我可以玩的游戏。我的研究让我 [采访了戴维·雷登](https://kotaku.com/the-stanley-parable-turns-video-game-storytelling-on-it-5829254) ，斯坦利寓言*的开发者，我读到的东西让我想立刻投入到游戏中。*

*阅读对开发人员的采访会特别有帮助。有时听到别人对自己艺术的热情会激起你的兴趣。阅读 Wreden 对斯坦利寓言的关注程度让我对游戏感到兴奋，这也让我对整个游戏感到兴奋。*

### *和朋友一起玩*

*如果你认为你可能会被控制绊倒或对不能完成游戏感到紧张，有一个朋友一起玩会有很大帮助。当《生化奇兵无限版》问世时，我的一个游戏经验仅及于《T2》的朋友告诉我，她想去看看，但又担心自己不擅长。我和我的室友已经为我的主机买了它，所以我们邀请她一起玩。*

*我的朋友确实在控制方面有问题，最终在大多数枪战中把控制器还给我，但从那以后，她开始涉足越来越多的游戏。对我来说，对一个游戏感到沮丧是我从游戏中休息的最快方式。有一个朋友来指导你可以完全省略这个问题，并帮助你重新发现今天游戏中发生的所有令人兴奋的事情。*
