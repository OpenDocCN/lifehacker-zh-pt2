# 当你戴着面具时如何与孩子交流

> 原文：<https://lifehacker.com/how-to-communicate-with-kids-when-youre-wearing-a-mask-1845067940>

“口罩”( Mask-muff)是我刚刚编造的一个术语，但它是真实存在的。如果这种布料是为了阻挡一堆微小的呼吸道飞沫，它也一定会让我们的话语变得模糊。当你站在一个戴着面具的人旁边时，很难听清每个单词，如果你是推荐的六英尺(T3)或更远(T4)的人，就更难了。更糟糕的是，我们也失去了看到一个人的嘴组成单词来填空的好处，尤其是孩子，他们依赖于说话和语言发展以及社会互动。

当你和自己的孩子在一起时，你可能不会经常戴口罩，但是如果你是一名教师，一名儿童保育人员，或者你经常在蒙面的情况下与其他孩子互动，你可以做一些事情来更清楚地与他们交流。

### 像幼儿园老师一样说话

被任何人听到，尤其是被孩子们听到的最好方式是从*放慢*开始。如果你匆忙说出你的话，他们的会被你的面具搞乱。慢慢地，故意地说。你也可以专注于让你的声音更有表现力，这样孩子们就能更好地理解你的语气。你戏谑的声音可能有点傻，你严厉的声音可能有点严肃。

我要说的是:像幼儿园老师那样说话，用他们标志性的大而明亮、清晰的声音，即使是最吵闹的五岁大的孩子也能听懂。

### 更多地说出情感的名字

我见过几个朋友在社交媒体上发布面具自拍——通常附有说明，坚称他们实际上是在微笑。有时候能看出来，有时候看不出来。但是，除非一个人的眼睛皱成一团咧着嘴笑，或者透过狭窄的缝隙瞪着你，否则仅仅通过眼睛来诠释情感是很有挑战性的。

因此，就像我们在照片说明中所做的那样，更有意识地命名我们的情感会有所帮助——比如*和*让我们的孩子命名他们的。佩里·克拉斯博士为《T4 时报》撰写文章:

> 杜克大学心理学和神经科学助理教授莎拉·盖瑟(Sarah Gaither)在一封电子邮件中说，“现在大多数学校都要求戴口罩，儿童和成人应该开始练习通过大声说出他们的情绪来更加明确地表达。”她建议说，孩子们会更好地理解人们的眼神，以及从语调中理解情感内容。
> 
> 但除此之外，她写道，“父母和老师也可能想更经常地问孩子他们的感受。”

面具实际上给了我们一个很好的理由来更频繁地描述我们的感受，并鼓励孩子们也这样做。从长远来看，识别和命名情绪的额外练习实际上有助于增强他们的情商。

### 使用更多的身体暗示

现在你的声音更有表现力了，你把情绪命名为左和右，加入你身体的其他部分。

指着你正在谈论的事情，手势更大一点，高兴时鼓掌，惊讶时眉毛抬得更高一点。也许你会看起来有点像在后排为观众表演的舞台演员，但是如果你增加一点肢体语言，你更有可能表达你的观点。

### 最后，不要太担心

在寻求与任何人沟通的过程中，面具都不是一个很好的配件，更不用说孩子了。但是，尽管令人沮丧，尽管我们担心孩子们在面具后面失去了有价值的视觉线索，但这种情况不会永远存在，也不会一直存在。当他们现在在世界上四处移动时，他们失去了那些视觉线索，但是他们在家里仍然经历了许多无面具的时刻。

香港科技大学副教授、发展心理学家伊娃·陈博士 [告诉《泰晤士报》](https://www.nytimes.com/2020/09/14/well/family/Masks-child-development.html) 说，没有证据表明来自面部覆盖更广泛的文化背景的孩子在识别面孔或情感方面会更差:

> 虽然孩子们通常会在别人说话时注意他们的嘴，但“到目前为止，这不是孩子们交流和学习的唯一线索，”她说，并引用了 2012 年的一项研究显示，当戴上面具时，孩子们也能读懂面部表情。

换句话说，他们在面具后面失去的，会通过眼神、语调、和肢体语言来弥补。
