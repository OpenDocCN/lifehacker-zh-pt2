# 用“吹牛文件”克服冒名顶替综合症

> 原文：<https://lifehacker.com/overcome-imposter-syndrome-with-a-brag-file-1835103632>

总有一天，我会辞去在 Lifehacker 的工作(因为太优秀而被解雇)，当我申请一份新工作时，我需要收集一份作品集并更新我的简历。不会花很长时间，因为我已经收集了我最好的作品。无论你的作品是否公开，你都应该这样做。你会有一个更强大的投资组合，更少的繁重工作。



### 随用随存

当我特别得意的一个 [我的](https://lifehacker.com/how-to-get-a-cat-to-like-you-1828501761) [实用](https://lifehacker.com/dick-moves-for-winning-board-games-1819785482) [诀窍](https://lifehacker.com/how-to-play-an-rpg-anywhere-any-time-1832830780) 或者 [我的](https://lifehacker.com/bartenders-should-shush-people-1825479385) [勇敢](https://lifehacker.com/every-sponge-is-bad-1832899231) [见解](https://lifehacker.com/you-shouldn-t-have-to-read-these-books-in-high-school-1828549376) [棋子](https://lifehacker.com/stop-singing-happy-birthday-to-you-1822029507) ，或者 [我的](https://lifehacker.com/im-behavioral-economist-benjamin-ho-and-this-is-how-i-1830562135) [每隔一两个月，我还会保存表现良好的作品，这样我就可以展示那些指标，而不仅仅是我个人的最爱。](https://lifehacker.com/an-opera-singers-tips-for-hitting-high-notes-1821914631)

无论你要拿出什么，都要复印一份放入文件夹、书签或笔记系统中。第一次当一份草稿让你感到自豪时，或者在一个项目完成后立即这样做。而且存太多总比存不够好。这总比筛选你做过的所有文件要好。

如果你的产出更多的是一个过程，而不是一个成品，那就养成记录的习惯。拍摄照片和视频，录制音频，将电子邮件转发到单独的帐户。这是把你的作品放在社交媒体上的一个很好的理由，即使你不得不开一个单独的账户，因为你不想在你的主账户上吹嘘。

### 保留一份本地副本

如果你的作品在网上，养成把它复制到 Evernote 的习惯，或者使用像 [Authory](https://authory.com/) 这样的服务来识别和备份你的网上作品。虽然 Lifehacker 可能会比我活得长，但我已经为十几个不复存在的网站写过文章，而且我已经永远丢失了其中的一些作品，因为我没有备份它们。

如果你的工作是在公司所有的电脑或账户上完成的，那就准备好你可以随身携带的个人拷贝。 [备份你的工作 Gmail。](https://lifehacker.com/how-to-back-up-your-gmail-and-view-mbox-files-1827660389) 任何不能被起诉或解雇的东西都要保留。

你可以保留几种不同的格式，但最好能选择一种:Evernote、Google Drive、文件夹等等。

### **也保留一个公开的吹牛文件**

糟糕的是，你不得不一直推销自己，考虑你的下一个收入来源，但你做到了。我们大多数人从毕业到退休都无法保住一份工作。因此，要有一个可公开访问的吹牛文件，包括投资组合和广告页面。

比你的私人档案更有选择性。寻找你的投资组合中最薄弱的项目，删除它，并重复直到满意为止。确保你投资组合中的第一个项目比一般项目要好。(如果你的投资组合是按时间顺序排列的，在顶部添加一个“特色”栏目。)

请朋友或同事选出最强和最弱的项目。你会惊讶的。

### 保存赞美

你需要挽回的不仅仅是你的实际工作，而是你因为工作得到的所有认可和赞扬。把祝贺邮件、反馈、奖励、任何别人对你工作的好评保存在一个广告文件夹里。

你可以在公开场合宣传你的工作或申请新工作时引用赞美，但你也可以在加薪谈判或绩效评估中使用它。你不需要每次都明确地指向赞美。例如，在绩效评估之前，你应该通过检查你的吹嘘文件来刷新你的记忆，看看哪些项目*其他*人认为你做得很好。如果你向老板要求加薪，他们不想听你个人最喜欢的成就，而是他们当时称赞的成就。你提醒他们的是工作，而不是表扬。

尤其重要的是，要从值得尊敬的来源那里获得赞扬和认可，这些来源是未来的雇主、客户或观众可以信任的。这些是即使陌生人也能欣赏的赞美，而不是来自任何人的反馈。书上到处都是广告词。医生在网上发布好评。老师们分享学生们发自内心的卡片。无论你从事哪一行，都有一些适合你公开发表的赞美。当它进来的时候保存它。

### 浏览你自己的吹牛文件

有时候，你需要卖给的人就是你自己。你需要提醒自己你已经完成了什么。或者你需要重新审视你是否在做让你自豪的工作。

今年冬天，我觉得自己陷入了一个窠臼，好像我只是在推销无聊的故事，所以我查看了一下我的吹牛文件，寻找一些方向。看到 [一个关于桌面 RPG](https://lifehacker.com/role-playing-games-for-people-who-don-t-like-dungeons-1824267470)的老帖子，想起研究起来有多好玩。但这一次我想为一个新职位玩一个游戏。于是我投了一篇关于 [如何玩一个没有规划的 RPG](https://lifehacker.com/how-to-play-an-rpg-anywhere-any-time-1832830780)的文章。在我在这里写作的两年里，这可能是我最喜欢的作品。现在它在吹牛文件里。
