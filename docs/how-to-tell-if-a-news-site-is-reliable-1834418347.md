波因特研究所 正在通过其 [国际事实核查网](https://www.poynter.org/ifcn/) 提供一些指导，该网已经编制了 [一份不可靠新闻网站的名单](https://www.poynter.org/ifcn/unreliable-news-index/) 它称之为 [UnNews](https://www.poynter.org/ifcn/unreliable-news-index/) 。”

他们是这样做的:

> 为了创建索引，我们合并了五个主要的列表，然后删除了不再活跃的网站。我们只使用由知名记者或学者策划的公开名单，其中包含原始数据(而不是来自其他名单的信息)，陈述了他们的入选标准，并定义了他们如何对不同的网站进行评级(更多信息，请参见我们的 [方法](https://hearingvoices.com/wip/fakenews/ifcn-fake.html#methodology) )。

在索引的“第一阶段”列表中有 500 多个“UnNews”网站。下一阶段将致力于“自动移除不活跃的网站，通过跟随 URL 重定向(这通常会导致新的假新闻方案)来添加网站，并收获相关的域名，” [报道称](https://www.poynter.org/ifcn/unreliable-news-index/) 。

这份名单是可搜索的，这些网站都有标签，将它们标记为有偏见、不可靠、阴谋、讽刺或点击诱饵。
