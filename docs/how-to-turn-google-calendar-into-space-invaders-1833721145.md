# 如何将谷歌日历变成太空入侵者

> 原文：<https://lifehacker.com/how-to-turn-google-calendar-into-space-invaders-1833721145>

谷歌 2019 年的愚人节噱头非常有趣。不仅可以把 [地图变成*蛇*](https://lifehacker.com/how-to-access-the-snake-easter-egg-in-google-maps-1833700845)T5】的游戏，还可以把谷歌日历变成街机经典*太空入侵者*。 



1.  要玩游戏，首先在您的网络浏览器 中打开 [谷歌日历](https://calendar.google.com/calendar/r)
2.  点击**档位图标**打开设置下拉菜单
3.  点击**“玩游戏”**

你会看到一个弹出框，解释如何玩(使用左右箭头键移动，空格键射击)，以及一个听起来很不错的借口来证明玩游戏而不是实际完成你的待办事项列表上的任务。

在经典的*太空入侵者*中，重点是阻止敌人通过你的飞船(或者在这种情况下，弹出通知)，但是当你完成连续的波时，敌人的速度和数量会增加。完成的波浪越多，分数越高。这个版本的游戏将你的谷歌日历任务作为敌人入侵者，并用额外的任务来填补空白，这些额外的任务有一些愚蠢的名字，如“鸭嘴兽”、“开瓶器”和其他随机名词。发射你的激光伴随着可爱的“皮尤！皮尤！”音效，如果有任务从你身边悄悄溜走，还有一个“哇哇哇”的游戏。

虽然我们不能声称这是一个以任何方式提高生产力的工具，但这个游戏是一个整洁的小时间杀手，有一些聪明的触摸，比大多数每年充斥互联网的 [愚人节胡说八道高出一步。](https://gizmodo.com/in-banning-april-fools-day-pranks-microsoft-is-the-her-1833606918#_ga=2.245410946.762019268.1554125258-3846207152.1521480874)T9】
