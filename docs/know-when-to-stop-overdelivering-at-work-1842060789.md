# 知道什么时候停止在工作中过度表达

> 原文：<https://lifehacker.com/know-when-to-stop-overdelivering-at-work-1842060789>

当我处于各种职业生涯的早期阶段时——无论是作为行政助理，还是后来，当我重新成为自由作家时——我都是关于超额交付的。我在一些商业书籍中读到，成为一名有价值的员工的秘诀是总是比预期的更快更好地扭转局面。更进一步，如果你不介意这些陈词滥调的话。



我确实成为了一名有价值的员工——后来，成为了一名有价值的自由职业者。我以工作又快又好而闻名，这正是你在当今快节奏、知识驱动的工作环境中需要的东西。

但是我仍然尽我所能地超额完成任务:提前交自由职业任务，总是同意接受最后一分钟的文章，等等。

然后事情发生了变化。

从技术上讲，有两件事改变了。首先，我的日程表上有足够多的自由职业者的工作，我不能有效地继续提前完成它。我仍然赶上了所有的截止日期，但是我不再提前三天把草稿发给编辑了。

其次，我到了职业生涯中人们开始找我的阶段。我不再需要为我得到的每一份工作进行游说，而是会收到邀请我考虑接受新客户的电子邮件。

我认为，这就是你可以停止*过度交付*并开始*交付*的关键。

[爱丽丝·博耶斯](http://www.aliceboyes.com/) 博士，前临床心理学家，著有《T4》健康思维工具包和*焦虑工具包*，她在 [哈佛商业评论](https://hbr.org/2020/03/dont-let-perfection-be-the-enemy-of-productivity) 中解释说，虽然过度表达在你职业生涯的某些时候可能是一项有用的技能，但永久过度表达是有代价的:

> 明白总是追求卓越的代价是什么。还有什么是你没有时间、精力、注意力和意志力去做的？也许是你自己的健康，你的远大目标，或者你的家庭。如果你认为成本很高，试着用经验法则来判断你什么时候会超额交付。例如，你可能决定在十种情况中的三种情况下你有这样做的冲动，但在其他七种情况下没有。

她建议改变思维模式，在讨论的时间框架内，给你的雇主和/或客户他们想要的东西——因为这毕竟是你们双方都同意的。在没有额外报酬的情况下做额外的工作不会像你想象的那样对你的职业生涯有所帮助，尤其是如果你已经在你的职业生涯中建立了专业声誉和技能。

换句话说:一旦和你一起工作的每个人都知道你能完成任务，你就不用担心超额完成任务了。
