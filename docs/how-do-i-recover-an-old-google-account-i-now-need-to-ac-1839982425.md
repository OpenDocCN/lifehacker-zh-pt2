# 如何恢复我现在需要访问的旧 Google 帐户？

> 原文：<https://lifehacker.com/how-do-i-recover-an-old-google-account-i-now-need-to-ac-1839982425>

我们有时都会忘记密码。它发生了。如果你使用密码管理器，这根本不是问题——事实上，我认为你不会记得你的又长又复杂的密码。我当然不知道。但有很多人不使用密码管理器，这意味着如果他们的密码(和备份方法)失败，他们将面临失去旧帐户访问权限的风险。

Watch

这正是 Lifehacker 读者 **Rebecca** 发给我们的场景。她写道:

> “我的谷歌账户已经被锁定了至少 2-3 年，但它仍然与我的雅虎邮箱相连，所以我仍然可以通过我的雅虎链接发送和接收电子邮件，但我的谷歌驱动 id 中有一些东西，比如 back。我不记得我的密码了，我的旧电话号码也不在了，你能帮我吗

嗯，我很欣慰，你仍然可以在某种程度上访问你的谷歌账户，尽管不是以你希望的方式。这个问题一开始有点令人困惑，但是让我们来看看你的选择。

首先，我假设你不知道你的谷歌账户的密码。条条大路通 [谷歌的账户恢复工具](https://accounts.google.com/signin/recovery) 即使你*不知道与账户相关的 Gmail 地址，但这将使你恢复账户变得更加困难。例如，您必须首先提供您可能甚至不知道的信息，因为您可能已经首先使用了任一选项来恢复您的帐户:*

输入你的电话号码，最好是你的名字，谷歌会向那个号码发送一条短信来验证你就是你。但是，这对于您的情况没有帮助，因为您没有访问旧号码的权限。如果你记得你的恢复邮件，这也是一样的；Google 会发出一条验证信息，你就可以开始账户恢复过程了。

问题？这只是为了帮助你记住你的谷歌账户是什么。你将*仍然*必须经历密码重置过程，这可能会被证明是有问题的。我的意思是。首先，您需要输入您的电子邮件地址(因为您无法再访问您的电话号码):

你必须输入你的名字，这应该很容易，然后输入你的电子邮件地址(你可以从雅虎上查到)。你会得到一个验证码，然后你必须输入到帐户恢复过程中，就像以前一样。

在我的例子中，然后我被要求输入我的密码——我不知道，并指出这一点。然后谷歌让我输入我所知道的那个账户的最后一个密码。我想你也不知道。(我不会。)如果是这样，你必须点击“尝试另一种方式”，在我的情况下，谷歌随后向我提出了其他问题，比如问我与该账户相关联的第一个电话号码，我创建账户的月份和年份，等等。

这个过程繁琐吗？绝对的。不幸的是，这是你能做的最好的了；除了其账户恢复工具，谷歌不提供额外的支持，无论你读过多少关于人们找到一些神奇的客服号码(例如，如果他们注册了一个 [Google One 计划](https://lifehacker.com/google-one-is-now-open-for-everyone-but-is-it-a-good-d-1826049257) ，他们会得到什么)，他们可以打电话寻求额外的帮助。不用麻烦了。工具修不好，那就修不好。

我希望不是这种情况，但我有点明白谷歌在这方面严格要求的原因。这是安全问题。如果它的恢复工具问你的一切仍然不足以证明你确实拥有这个有问题的账户，那么在电话里的一个催人泪下的故事不应该说服谷歌让你进去。如何阻止另一个人想出一些聪明的方法来说服客户支持代理，他们真的是你(并且应该进入你的帐户)？

无论如何，我希望谷歌的工具足以让你进去。如果没有，你就被困住了。我会不断想办法验证你是不是你——例如，你的第一封 Gmail 邮件可能就在你的雅虎收件箱里，这可以帮助你找出你是什么时候注册的账户(回答一个验证问题)。也许你已经在另一个服务上使用了你的旧 Gmail 密码，仔细检查你在任何地方都使用的一些常用密码——如果你这样做的话——可以帮助你进入你的谷歌账户。

如果你登录了，我建议你点击你的 [账户设置](https://myaccount.google.com/security) ，设置一个新的恢复电话号码和电子邮件地址——并更改你的安全问题，以防你也忘记了。并在接下来的几年里保持领先地位；如果您的恢复信息过期，您最不想做的事情就是努力重新进入您的 Google 帐户。此外，考虑使用 [密码管理器](https://lifehacker.com/the-five-best-password-managers-5529133) 来存储你所有的登录信息，这样你就再也不用担心忘记了。
