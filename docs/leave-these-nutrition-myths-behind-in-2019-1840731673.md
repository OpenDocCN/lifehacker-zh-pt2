# 2019 年，把这些营养神话抛在脑后

> 原文：<https://lifehacker.com/leave-these-nutrition-myths-behind-in-2019-1840731673>

当我们开始新的一年，也可以说是新的十年时，我认为有几样东西我们可以放心地不要。这是我在本世纪初相信的，或者可能想要相信的三个营养神话——以及我为什么要放弃它们。

### 神话:特定的化合物使食物“对你有益”

记得超级食物吗？任何抗氧化值高的东西都被认为对你特别有益。流行语变了，但概念没有变:例如，我们不断听说咖啡或红酒中的类黄酮使食物特别健康。

但是这些想法往往来自孤立地观察一种食物成分的研究——通常是实验室研究，与人类无关。比如，好吧，如果你从红酒中提取一种化学物质并给老鼠服用，也许老鼠血液中的炎症相关生物标志物水平会稍微低一些。那又怎样？这并不意味着有喝红酒习惯的人会比不喝酒的人更健康。

我们不吃营养素，我们吃食物；我们不仅仅吃食物，我们过着丰富的生活，包括数百种食物和数百种考虑，除了可能降低某种血液化学物质的水平。

吃你的蔬菜，吃各种各样的食物，你知道该怎么做。如果你在争论是否吃这些浆果和 T2 浆果，你是在浪费时间。

### 神话:eto 对你的新陈代谢有神奇的作用

酮和其他低碳水化合物饮食的受欢迎程度起伏不定。记得阿特金斯吗？它的诱导期基本上是严格的酮饮食，可以追溯到 20 世纪 70 年代。

在本世纪初，生酮饮食——碳水化合物含量低到足以产生某种血液化学物质的饮食——仍有可能对我们的新陈代谢产生特殊影响。但是从那以后，我们看到了一些严格的研究来检验这个假设，他们发现低碳饮食没有生化优势， [也没有任何特定的饮食](https://www.thelancet.com/journals/landia/article/PIIS2213-8587(15)00367-8/fulltext) 。

所有的饮食似乎都一样有效(或者一样糟糕，说实话——大多数人都会反弹他们长期失去的体重)，只要他们限制相同数量的卡路里。所以真正重要的是选择一种你能坚持的饮食方式，无论是低脂肪还是低碳水化合物还是其他。

### 误区:用餐时间非常重要

我确信，用餐时间之所以成为一件事，是因为人们在寻找一种简单的优化方式。也许你在吃正确数量和种类的食物方面有困难，但至少你可以控制时间。或者也许你想向自己证明你对锻炼是认真的，所以你花时间计划完美的锻炼后摇晃。

正如低脂肪和低碳水化合物饮食似乎同样有效一样，没有确凿的证据表明间歇性禁食比一天多餐更好或更差。如果喜欢早餐，可以吃；如果没有， [跳过就没事了](https://vitals.lifehacker.com/why-breakfast-is-not-the-most-important-meal-of-the-da-1682222302) 。

虽然有证据表明锻炼前后的用餐时间很重要，但与你吃什么和吃多少的基本原则相比，它的影响很小。如果你不能在锻炼完 后立即摄入 [30 克蛋白质，你的肌肉并不会萎缩死亡。(我清楚地记得在健身房结束后，我看了看时钟，以确保我在假定的 30 分钟窗口内得到了奶昔。)原来，获得那个蛋白质的窗口是](https://lifehacker.com/the-ideal-amount-of-protein-to-eat-after-exercise-1791262350) [大概几个小时长](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3577439/) 。所以，大多数人可以把我们的锻炼计划安排在两餐之间——比如说午餐和晚餐，或者早餐和第二顿早餐——而不是任何特定的时间。
