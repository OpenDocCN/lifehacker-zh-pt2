# 2019 年你学到的最大的金钱教训是什么？

> 原文：<https://lifehacker.com/whats-the-biggest-money-lesson-you-learned-in-2019-1840372946>

也许你已经开始反思 2019 带你去了哪里，你想在 2020 年关注什么。你正在计划一次旅行，或者想出如何定期回到健身房，或者考虑开始一项爱好。但是你想过你的钱吗？

当我五月份开始为 Lifehacker 写关于金钱的文章时，我用 [介绍了自己，问了你两个问题](https://lifehacker.com/meet-our-new-personal-finance-writer-1834563704) 。我想知道你在 2019 年迄今为止为你的钱做的最好的事情，我想知道你想在这里以两美分的价格了解什么。

很多很多的回答说明了你有多关心改善你的财务状况，不管你是关注储蓄、开始投资，还是考虑如何照顾你孩子的教育。这让我明白了你有多想继续学习如何最好地管理你的钱。

因此，再过几周就要年底了，我想回到最初的讨论。回想一下整个 2019 年:你买了什么，你没买什么，你搞砸了什么，你搞砸了什么但在它让你付出代价之前想出了如何修复。

我不指望你有一大堆财务决议；我也不指望你会买下新年决心。

但是如果你准备分享(无论是第一次还是第二次)，我有两个问题要问你:

1.你今年学到的最大的金钱教训是什么？你是如何把它运用到自己的财务状况中的？

2.在接下来的一年里，你想改善财务状况的一件事是什么？

首先，我要告诉你我今年学到了什么。天哪，这是不是很无聊:我知道了拥有一大笔应急基金有多重要。

10 月份，我不得不去急诊室，幸运的是，结果证明是小问题。然后，在我的急诊室之旅几周后，我遭遇了一场车祸。我很好，但是我的车最终彻底报废了。这是我的第一辆新车，几个月前我刚刚还清了贷款。我非常喜欢没有车贷的生活。

突然间，我觉得自己陷入了一个昂贵的厄运漩涡。在我的保险没有完全覆盖的自付额和医疗检查之间，我为后续的就诊和检查支付了近 1000 美元。虽然我的保险公司为我的车开了一张很好的支票，但我知道我仍然需要拿出几千美元才能得到另一套车轮(但这次是一套二手的)。

我的应急基金一次又一次地挽救了局面。我突然很高兴我没有屈服于诱惑，把所有的现金从我的高收益储蓄账户中取出来，用在我慢慢减少的学生贷款余额上。

通过抑制让我的钱做点什么事情的冲动，而不是坐在那里，警惕和准备，我保护了我的每月预算。

当急诊室要求支付我的共付额时，我毫不犹豫地打开了我的钱包(尽管实际上这样做花了我一分钟)。当我的直觉告诉我在购买另一辆汽车之前等待几个星期时，我出去买了一辆自行车和一个新头盔，这样我就可以在我的邻居周围走动了。如此等等，直到一切恢复正常。

当然，现在你不会感到惊讶，我 2020 年的目标是建立应急基金，同时继续努力实现我的债务偿还目标。但是看到我的应急基金的力量，我彻底相信了它的重要性。

关于我已经说得够多了。说说你吧。你学到了什么？你是怎么学会的？2020 年你想用你的钱做什么？
