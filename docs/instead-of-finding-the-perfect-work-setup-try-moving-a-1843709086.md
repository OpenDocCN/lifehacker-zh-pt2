# 与其寻找完美的工作环境，不如试着四处走动

> 原文：<https://lifehacker.com/instead-of-finding-the-perfect-work-setup-try-moving-a-1843709086>

你很难适应你的临时家庭办公室吗？我们已经吸收了这样的想法，即有一个理想的“人体工程学”设置，如果一切都得到适当的支持，并设置到正确的高度，我们就再也不会不舒服了。但是如果坐在一个正确的位置上对我们来说也不好呢？相反，我提出一个小小的建议:四处走动。

Watch

姿势和背部疼痛专家实际上已经说了一段时间。没有强制性的正确姿势，只有一套我们应该注意的宽松的指导方针:不同的人坐得不同是可以的。姿势随心情变化没关系。如果你感到舒适，你就不会真的“坐错了”有很多正确的坐姿和站姿。

事实上，在一个位置上坐几个小时可能并不好，不管那是什么位置。所以与其追求完美，不如尝试坐立不安。

以下是我个人的认可:我职业生涯的大部分时间都是在家工作，我从未买过一把别致的办公椅。我常用的椅子是从餐厅偷来的有衬垫的木椅。当我疯狂打字时，我可能会坐直，身体前倾。当我想思考的时候，我会把它挪到离我的桌子更近的地方，向后靠，同时把脚搁在桌下的搁脚凳上。

有时我会把笔记本电脑带到沙发上，在那里我大概会循环三四个姿势:当我侧身躺在沙发上时，腿伸在前面；一条或两条腿叠在我下面；或者有时我甚至会像正常人一样坐直，脚放在地板上。人体工程学分析师会讨厌我的。但是每当我开始感到僵硬的时候，我就会稍微动一下。(如果你盘腿而坐，这一点尤其重要；如果你不想让它们麻木的话，你得把它们移开。)

所以，如果你很难找到合适的工作岗位，考虑一下你可能已经找到几个了。尝试不同高度的桌子，不同的椅子，时不时地看看沙发或床。这个问题有很多正确答案，不一定非要选。
