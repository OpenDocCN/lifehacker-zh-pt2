# 为什么你应该冷冻西红柿而不先去皮

> 原文：<https://lifehacker.com/why-you-should-freeze-tomatoes-without-peeling-them-fir-1844714372>

“番茄太多”不是我目前面临的问题，因为我缺少一个可以种植任何番茄的花园(T2)。但是我知道有很多人现在被自家院子里的工厂生产的大量爱情苹果弄得不知所措，显然“多吃点西红柿三明治”不是一个足够积极的缓解计划。



显然，这种令人尴尬的西红柿财富可以制成罐头或 [做成酱](https://skillet.lifehacker.com/make-tomato-sauce-in-your-instant-pot-with-just-two-ing-1828971840) ，但最简单、最懒惰的保存方式是将它们放入冰箱。事实上，它甚至比我最初怀疑的更容易也更懒:根据烹饪的乐趣的编辑(他们非常聪明，非常好)，你甚至不用剥它们:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1293648588023595009&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1293648588023595009&autosize=1) 

这基本上与香蕉相反，香蕉在进入冰箱前必须去皮，除非你希望在某个时候对香蕉感到非常沮丧。(我的意思是，剥一个冷冻香蕉不是不可能的；就是不理想。)这也意味着剥它们甚至比通常的“切一个小 x 然后焯一下”的方法更简单，后者已经很简单了！你所要做的就是把它们冷冻在某种托盘上——烤盘就可以了——然后放进冷冻袋。解冻后，鱼皮会自动滑落，不需要热烫。

该出版物的 [关于“番茄过剩”](https://twitter.com/TheJoyofCooking/status/1293657921516052480) 的整个对话值得一查，因为欢乐的编辑充满了番茄 y 小贴士——包括一个很好的利用悲伤、不合格的番茄、的技巧，这些番茄今年在俄勒冈州因某种原因大量存在。(如果你发现自己正面临着一大堆不合格的西红柿，不要担心——我们已经分享了许多方法，你可以用这些方法让它们尝起来美味可口。)
