# 这些是美国最昂贵的机场

在这项新的研究中，TPG 调查了美国最繁忙的 50 个机场，并根据机票的平均成本对它们进行了排名，使用了运输统计局提供的数字，以及你可能预计要支付的其他费用，如停车、行李车、到最近的市中心的交通、茶点和休息室使用权。

如果你是纽约人，领跑者可能不会那么令人惊讶；纽瓦克自由国际机场和约翰·肯尼迪国际机场被评为美国最昂贵的机场。根据 Points Guy 的执行编辑主任 Scott Mayerowitz 的说法，交通成本是纽瓦克和 JFK 排名的重要因素。

“一般来说，纽约是一个昂贵的地方，”Mayerowitz 在电话中说。“但更重要的是，坐地铁去纽约机场的费用非常昂贵。纽约确实有通往所有三个机场的公共交通选择，但我不认为这些对大多数人来说是理想的选择。我和家人住在曼哈顿，我总是会考虑它会花掉我多少钱。”

与此同时，拉斯维加斯的麦卡伦国际和奥兰多国际被认为是最实惠的，主要是基于他们的平均机票费用。下面你会发现最贵和最便宜的机场是如何排列的，感谢积分专家。
