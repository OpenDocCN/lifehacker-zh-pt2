# 笔记本电脑玩不了游戏该不该买外接显卡？

> 原文：<https://lifehacker.com/should-i-buy-an-external-graphics-card-if-my-laptop-can-1844405307>

我喜欢在合理的分辨率和质量设置下玩电脑游戏，而不会让我的腿或桌子着火，这就是为什么我在购买新东西时通常不会考虑笔记本电脑的游戏功能。当我的台式机(及其独立的 Nvidia RTX 卡)可以满足我的一切需求时，为什么还要烦恼呢？



然而，有很多人想让 T1 在笔记本电脑上玩游戏，而这些笔记本电脑并不能让最新的游戏看起来更好(甚至可能不可玩)。在过去的几年里，你不得不开始攒钱买一台性能更好的新笔记本电脑。如今，有了第二个选择:一个外部显卡，你可以连接到你的笔记本电脑，以提高其游戏能力。

这就是我们本周的 [Tech 911](https://lifehacker.com/c/tech-911) 问题，由生活黑客读者“ **Ghost11** 提交他们写:

> 我有一个关于 eGPU(外接显卡)的问题。所以我想买一个 eGPU，我发现谷歌的搜索结果总是显示“附件”。我很困惑。我的问题是:
> 
> *   *eGPU 机箱包括显卡和里面所有的硬件吗？还是说我必须单独购买所有的硬件？*
> *   *另外，附件只是一个空盒子吗？*
> *   我应该去看看什么都有的游戏盒吗？

### eGPU 听起来很简单，但是可能会有点复杂

我很乐意帮忙，幽灵发信人。一个“外部显卡”，简称 eGPU，可以有三种不同的包装:需要您提供自己的显卡的机箱，带有可更换显卡的机箱(这样您就可以升级)，以及带有锁定显卡的机箱(这样您就只能使用现有显卡)。

你可以想象，这三种设置之间的成本差异会很大，但并不总是如此。例如，考虑一下华硕的[ROG XG Station 2](https://smile.amazon.com/dp/B06WGRT5GS?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/should-i-buy-an-external-graphics-card-if-my-laptop-can-1844405307&asc_source=&tag=kinjalifehackerlink-20):一个大而结实的盒子，里面装有你以 550 美元左右的低价供应的显卡*。这款 [千兆字节的 AORUS 游戏盒](https://smile.amazon.com/Gigabyte-AORUS-Gaming-Graphic-GV-N1070IXEB-8GD/dp/B073PYMG3G?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/should-i-buy-an-external-graphics-card-if-my-laptop-can-1844405307&asc_source=&tag=kinjalifehackerlink-20) 价格稍高一点(+250 美元)，但它附带一个显卡。这个 [雷蛇 Core X 配一个镭龙 RX 580](https://smile.amazon.com/Razer-Thunderbolt-External-Graphics-Enclosure/dp/B08BPKDDTP/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/should-i-buy-an-external-graphics-card-if-my-laptop-can-1844405307&asc_source=&sa-no-redirect=1&tag=kinjalifehackerlink-20&th=1) ？$575.*

我提到这些，因为你可能*可能*会说，如果你正在看的外部 GPU 的价格是天文数字——1000 美元以上，那么它就带有内置显卡。在此之下，不能保证你看到的东西默认带有一个卡，或者如果它是一个附件，需要*你*提供 GPU。

无论你在哪里购物，当你通读 [机柜的规格](https://egpu.io/best-egpu-buyers-guide) 和/或描述时，一个 eGPU 是否带有卡应该是显而易见的。如果你不确定，值得看一些评论，甚至访问制造商的网站，以确认你买的是盒子+GPU，而不仅仅是盒子(如果你需要的话)。否则，花更少的钱买一个更暗的外壳可能会让你最大限度地利用你可以花在一个伟大的显卡上的钱——供思考。

### 你需要 eGPU 吗？

总的来说，我觉得你根本就不应该买外置 GPU。对于那些给你足够好的性能的东西来说，它们可能贵得吓人。这一点，你必须弄清楚“一个伟大的显卡”和“一个昂贵的显卡，其通常令人难以置信的性能受到外部 GPU 和我的笔记本电脑之间的连接的瓶颈”之间的细微差别

正如卡梅隆·福克纳(Cameron Faulkner)在 The Verge 2019 年对 [的评测中所写的那样:Razer Core X Chroma 外置 GPU](https://www.theverge.com/2019/5/17/18624843/razer-x-chroma-gpu-enclosure-thunderbolt-3-gaming-review) :

> 虽然雷电 3 能够快速移动数据，但它给 GPU 的带宽带来了明显的瓶颈。它在 PCIe 上支持四个通道，比大多数台式机主板支持的 PCIe 的 16 个通道少得多。(有些笔记本电脑的雷电 3 端口的带宽甚至更低，所以你必须查看制造商的规格，以了解你的笔记本电脑到底能做什么。)当你在笔记本电脑的显示器上玩游戏时，单根电缆发挥着双重作用:电脑必须将数据发送到 GPU 进行处理，然后接收处理后的数据进行显示。玩密集型游戏是一个巨大的工作量，最终，即使是最强大的显卡也难以显示真正令人印象深刻的帧速率。尽管如此，我没有注意到游戏中的任何波动。”

如果一个外部 GPU 仅仅是 100-200 美元的事情，我会说冒险吧。当你花 500 多美元，甚至 1000 美元购买一个机箱和一个显卡时，你得到的是一个临时的、不完美的问题解决方案——考虑到 [和](https://www.howtogeek.com/295442/the-best-ways-to-connect-an-external-graphics-card-to-your-laptop) 所涉及的软件特性，这个解决方案甚至可能不直接与你的系统兼容。

像所有的台式电脑用户一样，你最终将不得不升级你的 eGPU，以全质量播放最新最棒的游戏。到那时，可能会有更好的连接类型可以使用(Thunderbolt 4？！？！！？！？！)而且无疑是更好的圈地。新的设置又浪费了 500 多美元；再加上您之前支付的费用，您就可以构建一个体面的游戏台式机，在未来几年支持各种更新、更快的显卡。

然而，我意识到，并不是每个人都想要一个专用的游戏盒(然后不得不购买显示器、鼠标、键盘，其中一个有足够的 RGB 风扇来重现夏日阳光的亮度，等等。)但是，如果这不是一个选项，并且 eGPU 是游戏的一个不太实际的权宜之计，你应该怎么做？

### **我不会放弃游戏流媒体服务**

听我说完。两大游戏流媒体服务——Stadia 和 GeForce Now——每月花费不多(分别为 10 美元和 5 美元)。如果你选择谷歌，你每个月可以获得大量免费游戏，并且可以在一系列不同的设备上玩你购买的任何游戏，包括你的电脑、兼容安卓系统的手机和电视(当然，如果你有 Chromecast Ultra 的话)。

英伟达的服务也是如此，减去每月免费的游戏。你将能够在你的 Windows、Mac 或兼容的 Android 设备上玩你可能已经拥有的许多游戏(从 [到一个相当大的标题列表](https://www.gamewatcher.com/news/nvidia-geforce-now-games-list) ),以及 Nvidia 的 Shield 流媒体盒子。

如果你是一个超级玩家，或者喜欢任何其他没有出现在流媒体服务上的游戏( [或者被取消的](https://kotaku.com/activision-blizzard-pulls-its-games-from-nvidias-stream-1841638082) )，那么这两个流媒体服务都不会对你有太大帮助。但是如果您喜欢这两种服务中的任何一种，那么它们可能值得研究，而不是目前更复杂的 eGPU 设置。随着微软的 [xCloud 服务](https://kotaku.com/game-pass-ultimate-is-getting-video-game-streaming-in-s-1844403046) 也即将推出，我们值得等待，看看这些服务是否能填补更复杂、更昂贵的游戏 eGPU 设置的空白。

有了流媒体，你将能够以一个外部显卡设置的成本获得数年的游戏流媒体，而你需要的只是一个像样的互联网连接来体验 [相当好的游戏质量](https://lifehacker.com/how-well-does-google-stadia-work-we-did-a-speed-test-1839870376) 。我认为现在这比 eGPU 更经济，至少在该技术允许你通过简单的即插即用式连接在笔记本电脑上获得完整的、不受限制的桌面性能之前是如此。当那一天到来的时候，你可以随心所欲地走出去。
