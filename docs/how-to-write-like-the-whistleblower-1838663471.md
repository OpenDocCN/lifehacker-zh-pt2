# 如何像告密者一样写作

> 原文：<https://lifehacker.com/how-to-write-like-the-whistleblower-1838663471>

这是不正常的:一名政府雇员写了一份很好的备忘录。哈佛大学写作中心主任简·罗森茨韦格在《纽约时报》专栏中称赞了长达九页的举报人投诉 反对特朗普政府，并列举了四种技巧，让投诉清晰有效。

### 轮廓清晰

其中的三种技巧——紧急的介绍、强有力的、有联系的副标题和主题句——反映了文章清晰的线性结构。你可以抽出副标题和主题句，为文档要点创建一个要点大纲。

这使得文档易于阅读，难以忽略。这也有助于信息的传播，因为新闻媒体可以为更广泛的受众准确地总结投诉——这些受众将阅读两页的文章，而不是九页的政府备忘录。

### 它是活动的

另一种技术——带主语和动词的主动句——将原因和结果分配给特定的人。举报人认为特定的人做了特定的事情，违反了特定的规则。当他知道*某人*做了*某事*但想不出是谁时，他才使用被动语态。他指出了这一点，清楚地说明了他抱怨的范围，而不是含糊地暗示责备:

> 据这些官员称，他们还"清楚地"得知，总统不想会见 zelensky 先生，除非他看到 zelensky 在任时"选择如何行动"。我不知道这个指导是如何传达的，或者由谁传达的。

这明显是为了衬托 [总裁沙拉](https://splinternews.com/trump-spews-absolute-nonsense-before-walking-out-on-rep-1838637249) 对“越来越多的人在说”这种搅浑水的话的喜爱

### 你也可以这样写

如果你想用一个强有力的结构写作，你有两个选择:精心勾画轮廓，然后在轮廓内写作，或者先写作，然后将你的写作重新安排成一个清晰的轮廓。

根据你的风格和心情，其中一个会比另一个感觉不那么可怕。有时它是提纲:你还不需要做“真正的”写作，你只是列出你以后想写的想法！如果你真的进展顺利，你会以较小的增量继续写大纲，直到你不小心写完了大部分。提纲法对于有说服力和解释性的写作特别有效，比如备忘录、论文和电子邮件。但是它也可以用在创造性写作中，如果它不会扼杀有趣的想法，让你的工作变得无聊和可预测的话。

有时候自由写作就不那么可怕了，按照你想到的顺序写下你的想法。你不必真的去写，你只是先把所有有趣的事情说出来，然后你会把它们按顺序排好，然后再填空。这样更难“意外地”写出一整篇文章，但它给惊喜留下了更多的空间，也让你在还不确定的时候发现你真正想说的话。当你有说什么的冲动，但又不知道说什么的时候，不要自我编辑，直接说出来就好了。然后放下你的草稿，一个多小时后回来，编辑。重复这个循环几次。

你经常在这两者之间写作:写完整的句子，试着让每个句子都有意义，并试着在写作时进行组织。这本身没有什么错。但是如果你开始讨厌写作的过程，试着只*只*自由写作或者只*只*概述。是的，你在推迟困难的部分，但是在这个过程中你让它变得不那么困难了。
