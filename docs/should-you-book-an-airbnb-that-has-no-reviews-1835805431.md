# 该不该订没有点评的 Airbnb？

> 原文：<https://lifehacker.com/should-you-book-an-airbnb-that-has-no-reviews-1835805431>

在浏览一个特别漂亮的 Airbnb 房源时，你可能会对自己说:这个地方好得令人难以置信。你浏览了房源的照片，了解了它的便利设施——包括一个游泳池和一个日光浴平台——大约就在这个时候，你发现房源的评论为零。



在很大程度上，有一个非常简单的解释:该列表是新的，还没有收到足够的或任何客人来查看它。然而，在其他情况下，你完全有可能被一个 [骗局](https://www.usatoday.com/story/money/2019/05/19/airbnb-scam-hits-some-users-charges-them-fake-reservations/3732422002/) 收买，尽管 [Airbnb 声称](https://www.airbnb.com/help/article/1308/does-airbnb-perform-background-checks-on-members) 至少对其部分成员进行背景调查。在预订评论很少或没有评论的 Airbnb 之前，仔细看看房源并在完成预订之前联系房东是很重要的。

### 避免在 Airbnb 的直接信息系统之外联系

在大多数情况下，申请 Airbnb 相当简单:你选择日期，请求预订，然后主人接受你的请求。(对某些房源使用 Airbnb 的“即时预订”功能，这个过程会自动发生。然而，对于绝大多数房源，主人必须亲自接受预订。)

你可能会发现主人会在接受请求之前或之后主动联系你，甚至会问你要一个联系你的电子邮件地址。这就是为什么这可能会引发一个危险信号，特别是关于零评论的列表:你的主人将*已经*有能力在确认预订后给你发电子邮件，不应该需要你的个人地址。

Airbnb 会在你和你的主人确认你入住后，在他们之间建立匿名的临时电子邮件地址；发送到您的临时 Airbnb 电子邮件地址的任何电子邮件都将被转发到您的个人电子邮件地址。(除非您决定提供您的真实电子邮件地址，否则您的主机无法找到您的真实电子邮件地址。)

一旦你从 Airbnb 的消息系统转移到个人电子邮件地址，你就不再受 Airbnb 的保护。如果房源有问题，Airbnb 通常可以查看你的信息记录来帮助解决问题，这给那些通过个人电子邮件地址联系的人带来了问题。

当然，并不是每封邮件都是骗局；过去，我曾向那些评论*优秀*的主持人提供过电子邮件地址，这些主持人提供过路线、照片，或者要求提供我的全名以供识别。(例如，在日本，法律要求主人出示你的护照复印件。)然而，如果你面对的是一个没有评论的主持人，你没有理由不能礼貌地要求保持与 Airbnb 消息系统的对话。此外，你应该在网上搜索目的地关于 Airbnb 房源的特定法律，以防主人要求提供旅行证件——最简单的方法就是搜索这个国家和“Airbnb”

如果您通过电子邮件联系，请仔细检查电子邮件，避免点击要求您提供信息的链接，这可能是网络钓鱼欺诈的一部分。Airbnb *还*警告其客人注意 [预付费用骗局](https://www.airbnb.com/help/article/199/what-should-i-do-if-someone-asks-me-to-pay-outside-of-the-airbnb-website)——如果你通过 Airbnb 以外的服务付费，主人会提供特别好的交易。这是另一个巨大的危险信号:你不应该直接向主机支付任何费用。

### 仔细检查清单本身

如果一个价格好得令人难以置信，特别是当它与你目的地的其他可用房源相比时，这是另一个有点怀疑的理由(尽管新主机可能会提供稍微降低的价格以保持竞争力)。预订前，你还应该问自己以下几个问题:

*   主持人对*其他*列表有评论吗？如果是这样的话，检查一下这些列表，看看它们的评论有多好。
*   如果你反向搜索一些房源的照片，它们会出现在其他地方吗？这是一个巨大的危险信号，除非他们可能在像 VRBO 这样的竞争对手上挂牌出售房屋。缺少照片是另一个危险信号。我有一个朋友在哥斯达黎加的 Airbnb 预订了房间，只有两张照片，没有一张显示墙壁不是从地板到天花板的。这让我和其他客人有了一种特别亲密的感觉。
*   照片包括提到的便利设施吗？如果有人承诺给你一个热水浴缸或第二张床，那就值得通过照片来确认。
*   住宿的描述写得好吗？缺乏细节的简短描述可能是你重新考虑住宿的理由。
*   主持人有没有警告你你不要 联系[物业管理](https://www.reddit.com/r/AirBnB/comments/863icq/new_airbnb_host_with_no_reviews_wondering_if_this/) 或者租赁办公室？这是另一个危险信号。你不想在你的 Airbnb 周围踮起脚尖，尤其是如果它是不合法的。
*   一旦有了地址，对位置的描述准确吗？你可能想确认这一点，以确保上市没有欺骗你。

### 向主机发送消息

在你请求之前，给一个没有评论的主持人发一条简单的信息，问一些简单的问题，比如“这个列表是新的吗？”或者只是请求更多的列表本身的照片。你可以根据他们的反应(以及他们反应的速度)来判断这个主持人是否可信。

“试着发送一条信息，确认你感兴趣的日期是否有空&提出一个列表中没有回答的合理问题(比如离杂货店的远近)，”u/lemonsqueezee 在最近的 [Reddit 帖子](https://www.reddit.com/r/AirBnB/comments/98kt6e/newly_created_host_with_no_profile_pic_and_no/) 中建议。

如果你真的过得很愉快，给主人一个好的评价；这将有利于主人和未来的客人寻找一个住宿的地方，这样他们就不会像你以前一样被留在同一个位置。
