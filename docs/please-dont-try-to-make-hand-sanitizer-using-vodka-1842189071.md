# 不要试图用伏特加制作洗手液

> 原文：<https://lifehacker.com/please-dont-try-to-make-hand-sanitizer-using-vodka-1842189071>

如果你在过去几周去了当地的杂货店或药店寻找洗手液，你可能会遇到问题。对冠状病毒的担忧导致洗手液几乎在任何地方都被抢购一空，如果你最近访问了社交媒体，几乎每个人都有一个你应该如何制作自己的配方。除非你不应该喝，尤其是伏特加。

上周，我们的健康编辑带着一些洗手液的配方出去兜了一圈，遇到了问题 。她指出，很容易搞砸洗手液的配方。最容易的失败之一就是没有喝足够的酒。你自己动手做的洗手液也会对你的皮肤造成伤害。像 Purell 这样的产品之所以存在，是因为人们通常不会尝试在家里制作自己的产品。

制作自己的洗手液需要外用酒精，雄心勃勃的 DIY 者也从商店里买了很多，这导致其他人建议用伏特加代替酒精。

我所在州的一名政客甚至在推特上分享了伏特加的配方。听起来很合理。伏特加就是酒精，对吧？除了你在商店里买的伏特加已经被水稀释成合理的饮料，这意味着它不再足以杀死微生物和作为洗手液。如果你用它来做洗手液，你本质上只是在浪费你的伏特加。

伏特加品牌 Tito's 周四在 Twitter 上分享了一份 PSA，指出根据 CDC 的规定，洗手液需要含有至少 60%的酒精。既然铁托的是 40%，那就达不到你需要的标准。

蒂托和疾控中心都指出，“在大多数情况下，用肥皂和水洗手是去除细菌的最好方法。”伏特加留着喝吧。

*2020 年 3 月 11 日更新:这篇文章中的一句话最初是说稀释伏特加“意味着它不再强大到足以杀死细菌并作为洗手液使用。”虽然这是真的，但洗手液也可以抵御病毒，包括冠状病毒，而不仅仅是细菌。感谢那些指出我们命名检查了错误的微生物的人。*
