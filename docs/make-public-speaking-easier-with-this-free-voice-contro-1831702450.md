# 用这个免费的声控提词器让公开演讲变得更容易

> 原文:[https://life hacker . com/make-public-speaking-easy-with this-free-voice-contro-1831702450](https://lifehacker.com/make-public-speaking-easier-with-this-free-voice-contro-1831702450)

如果你不习惯公开演讲，提词器可能是你武器库中的有用工具。提词器可以帮助你保持演讲在正确的轨道上，让一切进行得更顺利。只有一个问题:大多数免费的网络提词器会自动滚动，这意味着如果你在阅读时出于任何原因暂停，你的提词器就会离开你。

Watch

这个星期我遇到了一个很好的选择: [提词提示我](http://teleprompt.me) 。这款免费的网络提词器使用你电脑的内置麦克风来听你说的话，然后在你说话时滚动屏幕。如果你在音乐厅里面对大众演讲，这不会给你带来太多好处，但如果你在客厅里拍摄 YouTube 视频，那么这是一个完美的解决方案。

要使用它，你只需使用谷歌 Chrome(它只适用于 Chrome)访问该工具的网站，键入一些文本或粘贴到你在其他地方创建的脚本中。专业提示:你可以在最右边的窗口中操作，而不是在屏幕中央。

工具中有文本后，您可以更改其大小和颜色，或者使用页面顶部的小控件翻转文本。

一旦你输入了所有的文本，你所要做的就是阅读。当网站听到屏幕上显示的单词时，它将开始滚动。因为它是由你的讲话控制的，你可以用比使用自动滚动提词器更自然的速度说话。

我用一些不同的文本(包括这篇文章)进行了测试，结果完美无缺。

如果你已经在使用自动滚动的网络提词器，那么它绝对值得一试。
