# 今天是您提交 Equifax 违约和解索赔的最后机会

> 原文：<https://lifehacker.com/today-is-your-last-chance-to-file-an-equifax-breach-set-1841155678>

拖延者们，注意了:今天是对 Equifax 数据泄露提起集体诉讼的最后期限。

Watch

如果你一直生活在岩石下——老实说，我们不会评判你——Equifax 在 2017 年宣布，1.47 亿人的个人和财务数据已经 [被泄露](https://twocents.lifehacker.com/surprise-the-equifax-hack-affected-millions-more-peopl-1823427395) 。

当结算细节在 2019 年 7 月 发布时，信用局为想要提交索赔的人提供了两种选择:现金支付或免费信用监控。

违规的规模意味着几乎任何人都可以提出索赔。如此多的人都这么做了，令人痛苦的是，很明显，和解协议提供的每人“最多 125 美元”的现金支付最终将远低于这个数字。也许，如果你幸运的话，你会在几个月后收到一张 [、6 美元或 7 美元](https://lifehacker.com/youre-probably-not-getting-more-than-7-from-the-equifa-1840542322) 的支票。

但是，如果你还想提交索赔，一切都还没完。除了现在微不足道的现金选项， [你可以选择通过益百利获得](https://www.equifaxbreachsettlement.com/faq) “至少四年”的三局信用监控。除此之外，在 Experian 的四年监控之外，您还将获得该局对您信用的六年 Equifax 监控。

即使你觉得你的信用已经被很好地锁定了——你已经 [冻结了你的信用](https://twocents.lifehacker.com/what-to-know-about-freezing-and-unfreezing-your-credit-1829474191) ，你的银行设置了交易提醒，并且通过 [免费服务](https://twocents.lifehacker.com/which-credit-monitoring-service-should-you-use-1815054928) 开启了信用监控提醒——它可能仍然值得接受结算的监控报价。它包括高达 100 万英镑的保险，以覆盖身份盗窃或欺诈，你可以随时查看你的 Experian 信用报告(尽管它仍会每月更新一次)。

要检查您的资格，请点击 [这里](https://eligibility.equifaxbreachsettlement.com/en/Eligibility) (您必须输入您的姓氏和大部分社会保险号)。如果您选择免费信用监控，您将收到如何注册的说明。
