# 使用 Chrome 扩展“简化”清除 Gmail 的混乱

> 原文：<https://lifehacker.com/remove-gmails-clutter-with-the-chrome-extension-simplif-1834309296>

有时，最好的黑客是最简单的。毫不奇怪，Chrome 扩展 [**Simplify**](https://chrome.google.com/webstore/detail/simplify-gmail/pbmlfaiicoikhdbjagjbglnbfcbcojpj) 是我的新宠之一，因为它将所有杂乱的东西都放入了 Gmail 和灭霸，让它们湮没无闻。

Watch

如果你是一个普通的 Gmail 用户，你可能甚至不会认为这项服务的网络体验一开始就那么混乱。我明白了。自从 Gmail 问世以来，我就一直在使用它，感觉很熟悉，尽管我确实同意 Simplify 开发者迈克尔·莱格特(Michael Leggett)的观点，即左上角的巨大图标和标志有点招摇。

“去看看任何桌面应用程序，告诉我有多少在左上角有一个巨大的他妈的标志。来吧。纯粹是自我，纯粹扯淡。放下标志。让我休息一下，”莱格特在接受快公司的采访时说。

还有谁能比莱格特更好地批判清理 Gmail 呢？莱格特不仅是 2008 年至 2012 年 Gmail 的首席设计师，也是谷歌收件箱的创始人。我觉得非常有趣的是，他现在似乎讨厌谷歌对 Gmail 设计的所作所为；至少，我只能假设这是他的扩展 Simplify 的动机，它去掉了 Gmail 整个界面的大部分。区别如下:

哇-砰。这些变化并不惊人，因为 Simplify 并不是一个完整的 Gmail 转换。然而，可怕的标志不见了，就像整个 Gmail 右侧面板一样(现在可以通过右下角的三点图标访问)。撰写按钮也不见了，取而代之的是右下角那个漂亮的加号图标，这几乎让人觉得它可能在召唤谷歌失败的社交服务的鬼魂。此外，您的电子邮件现在直接位于浏览器的中央。我起初不喜欢这个，但是它已经在我身上成长。

我仍在摸索我对 Simplify 的感觉，但这的确是一种展示 Gmail 的有趣的新方式。如果没有别的，我喜欢 Leggett 明确说明了扩展的作用，因为数据隐私和漂亮的网页一样重要。 [他写](https://simpl.fyi/gmail/) :

“你应该一直非常小心你的电子邮件使用的扩展或应用程序。Simplify 主要是一些 CSS 加上一点 Javascript 来应用 CSS。没有追踪器。不发送或共享任何数据。从来没有广告。保证。并且所有代码都可以在[*Github*](https://github.com/leggett/simplify/tree/master/gmail)*上查看。”*

如果未来的简化版本允许你挑选你想要去除、居中或修改的内容，那么莱格特将得到加分。
