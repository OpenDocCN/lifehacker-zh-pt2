Plex 最近 [发布了新的桌面客户端](https://www.plex.tv/blog/desktop-af/?AID=13099277&PID=7293383&SID=UUwpUdUnU68137YYwYg) ，旨在将其 Windows、Mac 和 web 客户端以及独立的 Plex 媒体播放器整合到一个应用程序中。对于 Windows 用户，这个新版本的 Plex(目前)在 Windows Store 上不可用。以下是对您的意义以及 Plex 更新程序带来的所有新功能的快速概述。

### 新的 lex 取代了旧的应用程序

Plex 的新应用完全取代了之前的 Windows 和 Mac 桌面应用。事实上，从今天起，Plex 不再出现在 Windows 商店中，也不清楚这个版本是否会在未来某个时候出现。因此，如果你现在使用的是 Plex 的 Windows Store 版本，卸载它——因为 Plex 没有更新它——并获取 [独立版本，而不是](https://www.plex.tv/media-server-downloads/#plex-app) 。

Plex 也将逐步淘汰其 Plex 媒体播放器应用程序，以支持在一个屋檐下打包所有内容。这也意味着几个 [特征的变化](https://www.plex.tv/blog/desktop-af/?AID=13099277&PID=7293383&SID=UUwpUdUnU68137YYwYg) :

> “对于电视布局爱好者来说，我们希望明确一点:长期计划是让新的桌面应用程序取代 Plex Media Player，成为我们唯一的桌面解决方案。新的桌面应用程序明显缺乏电视模式，这意味着我们将停止支持这款应用程序的传统 HTPC 设置(使用连接到电视或家庭影院的桌面计算机)。那里。我们说了。这对我们来说标志着一个时代的结束，如果我们说这不是有点苦乐参半，那我们就是在撒谎。但是振作起来。我们研究了大多数人是如何使用这款应用的，你们中的大多数人都会对流媒体设备和我们的新玩家有相同(如果不是更好)的体验。”

如果你还没准备好过渡， [Plex 媒体播放器](https://forums.plex.tv/t/plex-media-player/120475/84) 将在 2020 年 1 月 30 日前接收更新。之后就是过渡时间了。

### 新 Plex 的新功能

在播放媒体方面，新的 Plex 桌面在功能上等同于其他平台上的应用程序，如 Android 和 iOS，但增加了 Plex web 客户端的服务器管理工具。这意味着您将能够在所有电脑上通过一个应用程序添加、删除和组织服务器的媒体文件。

更重要的是，新应用程序带有改进的“离线”模式——现在称为“下载”而不是“同步”——可以比以前更快地将媒体转储到您的设备上。你会得到更多关于你的下载的信息，而且它们(希望)不会像 Plex 过期的“同步”功能一样遭受那么多的错误。不过，有一点没有改变，那就是你仍然需要成为 [Plex Pass](https://www.plex.tv/plex-pass/) 的订户才能离线观看媒体。
