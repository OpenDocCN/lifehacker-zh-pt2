# 尽快安装 2019 年 10 月的 Android 安全更新

> 原文:[https://life hacker . com/why-you-should-install-the-December-2019-Android-securit-1838848808](https://lifehacker.com/why-you-should-install-the-october-2019-android-securit-1838848808)

谷歌每月的安卓安全补丁通常只针对谷歌设备，但三星、摩托罗拉、LG、Oppo、华为和小米都在推出自己版本的 2019 年 10 月安全更新，以修补几款安卓智能手机上存在的重大零日安全漏洞。那些有易受攻击的手机的人应该确保他们在未来几天的某个时候一有补丁就下载。

Watch

这个漏洞——在安全补丁说明中显示为 CVE-2019-2215——允许黑客远程找到并完全控制一台设备，尽管它需要受害者首先安装一个受感染的应用程序(或者黑客使用该漏洞结合基于 Chrome 的漏洞来部署攻击)。该漏洞存在于以下手机上，尽管谷歌的 Project Zero 警告说其他手机也可能受到影响:

*   谷歌像素、像素 XL、像素 2 和 2 XL
*   三星 Galaxy S7、S8 和 S9
*   华为 P20
*   运行安卓奥利奥的 LG 机型
*   摩托罗拉摩托 Z3
*   Oppo A3
*   小米 A1，红米 5A，红米 Note 5

谷歌将于周二开始推出 2019 年 10 月的安全补丁，其他制造商可能会在未来几天内推出自己的版本。留意自动更新通知，或者通过进入手机的设置应用程序并搜索“系统更新”来自己检查补丁(具体途径将因您的设备和 Android 版本而异。)

Google Project Zero 报告称该漏洞已被成功利用，这引发了一些关于谁在使用它以及为什么使用它的大问题。这个漏洞本身是由以色列网络安全公司 NSO 创造的，该公司否认它或它的任何客户——主要由政府团体和国家安全组织组成——正在积极利用这个漏洞。

虽然普通 Android 用户不太可能成为利用该漏洞的人的目标，但它足够严重，一旦 2019 年 10 月的安全更新在他们的特定设备上可用，每个人都应该安装它，同时使用上述任何智能手机的人都应该格外小心。这意味着抵制安装不明来源应用程序的冲动，安装一个好的反病毒应用程序，并且聪明地对待你的浏览——甚至可能 [考虑使用非 Chrome 手机浏览器](https://lifehacker.com/how-vivaldis-new-android-browser-stacks-up-against-the-1837991187)。
