# 如何启用 VLC 最好的隐藏功能

> 原文:[https://life hacker . com/the-best-hidden-features-of-VLC-1654434241](https://lifehacker.com/the-best-hidden-features-of-vlc-1654434241)

VLC 无疑是我们最喜欢的媒体玩家之一(还有 [也是你的](https://lifehacker.com/five-best-desktop-media-players-397135) )。虽然它可以处理你扔给它的几乎每部电影，但它还可以做更多的*。你不需要成为超级用户来理解它所有不可思议的内置特性；你只需要知道它们的存在。我们很高兴向您展示神奇的 VLC 可以在您的台式电脑上做的一切。*

*Watch*

### ***1。如何下载 VLC 的 YouTube 视频***

*有很多第三方应用程序、浏览器扩展和网站可以用来下载 YouTube 视频。您也不需要它们中的任何一个，因为您可以直接通过 VLC:*

1.  *在 YouTube 上找一个视频——比如 [这个](https://www.youtube.com/watch?v=Rn1WnrH-pdw)——然后从你浏览器的地址栏复制网址。*
2.  *在 VLC，进入媒体>打开网络流。*
3.  *将 YouTube 链接粘贴到框中，然后单击播放。您可能需要等待一段时间才能开始播放视频。(如果它不起作用，或者在*之后很长时间*都无法加载，你可能想试试这个修复，看看是否有帮助。)*
4.  *在“工具”下，单击“编解码器信息”。*
5.  *在“位置”框中，右键单击文本块，然后单击“全选”。将此文本复制到剪贴板。*
6.  *返回浏览器，将链接粘贴到地址栏中。这将直接在 YouTube 的服务器上打开源文件。*
7.  *在视频播放时右键单击视频，然后选择视频另存为。*

### ***2。如何用 VLC (Windows)录制视频剪辑***

1.  *加载一个视频——可以是任何东西。在它开始之前点击暂停。*
2.  *右键单击视频，然后单击查看。然后，点击高级控制。*
3.  *开始播放您的视频，并按下播放器中的红色录制按钮。当它的背景变成蓝色时，您正在录音。*
4.  *再次按下红色录音按钮停止录音*
5.  *去找到你创建的剪辑。在 Windows 的现代版本中，它会在你的“视频”用户文件夹中。*

* * *

### ***3。如何用 VLC 捕捉你的桌面***

*如果你试图向某人展示在他们的 Windows 或 Mac 系统上做某事的指令，这是一个很好的技巧——尤其是如果你不能在网络会议上共享你的屏幕。*

1.  *在“媒体”下，点按“打开采集设备”*
2.  *点击“捕获模式”下拉菜单并选择“桌面”*
3.  *修改帧速率。对于桌面录音来说，15 f/s 可能就足够了，但对于更快节奏的运动来说，可能需要 30 f/s。*
4.  *单击“播放”旁边的下拉箭头，然后选择“转换”*
5.  *在“Profile”下拉列表中，选择 h.264 或 h.265。*
6.  *在这一步，您可以单击工具图标来修改此配置文件的设置。在这里你可以修改分辨率或比特率。我们现在将使用默认设置，但是如果您需要调整最终产品，您可以稍后回到这里。*
7.  *在“目标”框中，选择一个位置来放置完成的文件。*
8.  *单击开始。*

*VLC 现在将捕捉你的桌面幕后提要。让它在你记录工作空间的时候运行。完成后，您可以点按播放器控制中的“停止”按钮来结束录制。*

### ***4。如何用 VLC 转换视频文件***

*VLC 也有一个相当不错的内置视频转换器，如果你需要快速修复，没有时间去忙乱类似的东西。*

1.  *在“媒体”下，单击“转换/保存”*
2.  *在文件选择部分添加要转换的文件。*
3.  *单击“转换/保存”*
4.  *在“设置”部分，在“配置文件”下拉列表中选择要转换的文件类型。(我们建议使用. MP4、h.264 或 h.265。)*
5.  *在 Destination 下给出文件的名称和位置。*
6.  *单击开始。如果你愿意，你可以在 VLC 境内观察转换的进程(并得到一个估计的持续时间)。*

### ***5。如何用 VLC 录制您的网络摄像头***

*你的网络摄像头可能已经配备了拍照和录制视频的软件，但 VLC 可能更好。你不仅可以选择几种不同类型的录制格式，还可以根据需要调整一些精细的设置——可能比你的网络摄像头软件本身允许的还要多。*

1.  *在“媒体”下，单击“打开捕获设备”。*
2.  *在“捕获模式”下拉列表中，默认情况下应选择 DirectShow。如果不是，请选择它。*
3.  *对于“视频设备名称”，选择您的网络摄像头。*
4.  *对于“音频设备名称”,选择您的麦克风——如果您没有使用单独的麦克风，它也可能是您的网络摄像头。*
5.  *点按“高级选项”*
6.  *如果您想要使用设备附带的软件来控制输入设置，请选取“设备属性”*
7.  *否则，为“视频输入帧速率”输入一个值对于流畅的视频，30 是一个很好的经验法则，尽管如果你不关心质量，你可以使用更少。*
8.  *单击确定。*

*此时，您有两个选择。您可以单击播放通过 VLC 播放实况视频，并按下红色的录制按钮(如前所述)根据需要录制片段。您也可以从下拉菜单中选择“转换/保存”,并选择您希望记录的文件放在哪里。*

*这两种方法各有优点。前者允许你预览你的视频，并在短时间内进行剪辑。然而，这种方法需要耳机，因为它可以创建一个反馈回路。这也会导致在速度较慢的电脑上录制速度更慢。*

*使用 Convert/Save 方法可以避免反馈问题，但它也不能提供太多关于您正在看什么或何时完成录制的信息。您可以通过在播放器中按 stop 来停止录制，但没有指示表明您当时仍在录制。*

### ***6。如何在 VLC 订阅播客***

*你可能不认为 VLC 是一个播客管理器，但如果你经常使用它，它实际上非常方便。要添加播客，您需要该节目的 RSS 源。举个例子，我们将为 Lifehacker 的 [升级](https://art19.com/shows/the-upgrade-by-lifehacker) 播客使用一个 RSS 提要:*

*[`https://rss.art19.com/the-upgrade-by-lifehacker`](https://rss.art19.com/the-upgrade-by-lifehacker)T4】*

*把它复制到你的剪贴板，然后打开 VLC，执行以下步骤:*

1.  *在 VLC 的边栏中，向下滚动直到您看到“播客”。*
2.  *将鼠标悬停在播客上，然后点按右侧的加号。*
3.  *粘贴要添加的节目的 RSS 源 URL。*
4.  *单击确定。*

*现在，你选择的播客将出现在播客侧边栏部分。点击一个节目的名称，你会看到一个可用剧集的列表。双击其中任何一个开始播放。*

### ***7。如何使用 VLC 向您的 Google Home 扬声器或 Nest Hub 发送信息***

*我前几天刚刚发现了这个，它是我最喜欢的小技巧之一，可以将音频从我的电脑传输到我家周围的各种 Google Home 扬声器。(尽管我可以对着扬声器大喊来加载 Spotify 播放列表，但并不是我所有的音乐和视频都可以通过这项服务进行流媒体播放。)*

*要通过 VLC 在你的房子周围播放音乐或视频，请下载你想要的任何内容。从那里，点击 Playback 菜单并选择 Render 选项，然后从任何可用的扬声器中选择。就这么简单，但是您可能没有想到在这个笨拙命名的选项下寻找它。*

### ***8。如何通过 VLC 修复在错误时间出现的字幕***

*如果你正在看一部有 VLC 的电影，而字幕全部关闭了——根据角色说对白的时间，要么太早，要么太晚——你可以在《VLC》中直接解决这个问题。*

1.  *当一个演员开始说一个你知道你能认出的句子时，按 Shift+H*
2.  *当相关字幕弹出时，按 Shift+J*
3.  *按 Shift+K 修复延迟。*

*(如果句子前弹出字幕，交换步骤一和步骤二。)*

### ***9。如何在 VLC (Windows)中将视频设置为桌面壁纸***

*如果你想让你的 Windows 10 壁纸更上一层楼，试试这个吧:*

1.  *打开 VLC*
2.  *点击工具>首选项*
3.  *点击顶部的视频部分*
4.  *在“输出”下，选择“Direct3D9 视频输出”*
5.  *关闭并重新打开 VLC*
6.  *加载视频并开始播放。右键点击视频，选择视频，然后“设为壁纸”*
7.  *为了获得最佳效果，请确保您的视频的大小(或长宽比)与您的桌面相匹配。*

### *10。如何在 ASCII (Windows)中观看您的所有视频*

*在下一次聚会上，放一段视频作为背景，给你的极客朋友留下深刻印象...在 ASCII 中。*

1.  *在 VLC 加载视频* 
2.  *导航到工具>首选项>视频*
3.  *在“输出”下拉菜单中，选择“彩色 ASCII 艺术视频输出”*
4.  *沉浸在你的无聊中*

*不管怎么说，我在 Windows 10 的 64 位版 VLC 上根本无法运行这个程序。你可能在 32 位版本或不同的视频类型上运气更好，但我不会在 64 位系统上安装不同版本的 VLC*只是为了这个小技巧。**

### ***11。如何在 VLC (Windows)上定制循环视频***

*如果您只想循环视频的一部分而不是全部，您可以在 VLC 中自定开始点和结束点:*

1.  *打开您的视频并导航到您想要循环开始的位置。(轻按“e”以逐帧前进。)*
2.  *右键单击视频，然后单击查看。然后，点击高级控制。*
3.  *点击一下小 A/B 图标。这就设定了你的起点。*
4.  *导航到您希望循环停止的位置，然后再次单击 A/B 图标。您现在已经设置了结束点，循环将自动开始播放和倒回。*
5.  *最后一次单击 A/B 图标，删除该循环。*
