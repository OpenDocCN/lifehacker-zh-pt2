# 如果你要去南瓜地里，带上一条毛巾

> 原文：<https://lifehacker.com/if-youre-going-to-a-pumpkin-patch-bring-a-towel-1839230390>

我是众所周知的“室内儿童”，这意味着我经常对任何类型的户外活动准备不足，即使是那些面向儿童的活动。这周末的情况就是如此，我去了一块南瓜地，结果浑身都是泥。你*无法*阻止泥浆，但你可以减轻它，而最好的办法就是多带一条或条毛巾。



我们一开进停车场，路标就在那里了。满是泥块的家庭在柏油路上闲逛，在水坑里打转他们的鞋子，用湿巾擦拭他们的孩子。“这些家庭一定很笨拙，”我想。

然而，我很快意识到，平衡和姿态并不能保护我远离泥泞。原来，生长在湿土中的植物经常被湿土包围，而湿土会把你吃掉。一条毛巾——或者吸水毯——不会让你保持干净，但是它会让你的南瓜之旅变得容易得多。

泥巴的主要问题不是它让你变得泥泞，而是它让你的南瓜变得滑溜溜的，很难拿着，这很糟糕，因为这些沉重的圆形东西已经很难拿着了。用毛巾把南瓜包起来，便于携带，还能让你少沾点泥，因为你不用把南瓜贴在身上。也可以挑个大点的南瓜，因为不会被自己的上半身力量所限制。(一个大的宜家包也是一个很好的南瓜托，但是毛巾也可以根据需要用来擦手，而且不是每个人都有宜家包。)
