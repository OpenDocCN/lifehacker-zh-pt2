# 参加 538 的科学人格测验

> 原文:[https://life hacker . com/take-fivethirtyeghts-scientific-personality-quick-1831807767](https://lifehacker.com/take-fivethirtyeights-scientific-personality-quiz-1831807767)

孪生完成；“十年前的我与现在的我”的迷因将于周五结束；是时候让一种新的自恋时尚充斥我们的社交媒体了。是时候进行 [FiveThirtyEight 的新人格测验](https://projects.fivethirtyeight.com/personality-quiz/) 了，基于大五人格属性。 [与迈尔斯-布里格斯](https://vitals.lifehacker.com/its-time-to-stop-using-myers-briggs-at-work-1830232770#_ga=2.203131411.320302203.1547570792-1222396525.1547242489) 等测试不同，五大的概念是基于数十年的科学研究。

Watch

去年，fivethirteight[解释了五大](https://fivethirtyeight.com/features/most-personality-quizzes-are-junk-science-i-found-one-that-isnt/) 背后的研究，但是现在他们已经发布了他们自己的研究。他们说，一个好的测试使用陈述性语句和从“完全同意”到“完全不同意”的一系列答案。它不会试图用神奇的问题探索你的灵魂，问你最喜欢的颜色或哪棵树代表你对未来的希望。它返回的结果把你放在一个范围内，而不是像迈尔斯-布里格斯(或者霍格沃茨分院帽)那样给你分配一种性格类型。

所以做个 FiveThirtyEight 测试，分享给你的朋友。你必须 [截屏你的结果](https://lifehacker.com/how-to-take-a-screenshot-or-picture-of-whats-on-your-co-5825771) ，或者使用该网站的内置工具创建一个群链接，你可以与你的朋友、家人或同事分享——任何你想在分析测验结果的幌子下批评其个性的人。我的在上面。拖我。
