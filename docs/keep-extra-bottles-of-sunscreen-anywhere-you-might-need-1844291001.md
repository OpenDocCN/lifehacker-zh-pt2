# 在任何你可能需要的地方多放几瓶防晒霜

> 原文：<https://lifehacker.com/keep-extra-bottles-of-sunscreen-anywhere-you-might-need-1844291001>

我们很多人几乎没有像应该涂防晒霜 那样经常涂防晒霜 [。有时这是因为我们没有足够经常地重新涂抹(你应该每两个小时在阳光下多涂一点，如果你在游泳，应该多涂一点)或者我们没有首先涂抹我们真正需要的量(如果你从头到脚都是沙滩装，你应该使用超过一盎司)。](https://lifehacker.com/all-your-sunscreen-questions-answered-1827579794)

但有时我们不涂防晒霜仅仅是因为手边没有一瓶。例如，你可能会忘记带防晒霜，直到你离开家——到那时，跑回家从浴室橱柜里拿出瓶子已经太晚了。如果你确实记得把防晒霜扔进你的钱包、背包或沙滩包里，它很可能会留在那里直到第二天早上(或者，老实说，在接下来的三天里)，这意味着你很可能会告诉自己没有时间去找它，即使你确实想到要涂上一些。

当然，有一个显而易见的解决方案。买几瓶不同大小的防晒霜，在你需要防晒霜的地方留一瓶。

浴室里有一个瓶子。车里有一个瓶子。在你的钥匙链上放一个小瓶子，在你的钱包或背包里放一个旅行瓶，在你的沙滩包里放一个超大瓶。

生产力专家劳拉·凡德卡姆 最近解释了这种方法如何帮助她的家人涂抹更多防晒霜:

> 理论上，我们可以在泥浆室里放一个巨大的喷雾瓶和管子，因为那是我们通常离开房子的地方。但是这些年来，我发现如果防晒霜非常方便的话，它更有可能从瓶子里出来，涂在身体上。因此，我们现在有一个藏在后门，准备好任何人进入后院。我们在后门廊也有一瓶，以防有人从后门溜出去而没有注意到藏酒的地方。我们车里有瓶子。孩子们要去露营吗？他们的包里有管子。多一点意味着我们不用到处找。我们不会把防晒霜从一个地方搬到另一个地方。所以它被使用了。

把一瓶防晒霜放在你可能需要的任何地方，就像给未来的自己留下一份礼物。多年来，我一直在我的浴室、钱包和背包里分别放几瓶防晒霜，我不知道有多少次多放一瓶防晒霜拯救了我的皮肤。所以，如果你买得起多瓶防晒霜，那就开始把它们藏在每一个地方:浴室、化妆室、背包、沙滩手提包、杂物箱、时髦的腰包以及任何你认为可以用得到的地方。
