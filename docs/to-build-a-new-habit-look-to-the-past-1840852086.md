# 要养成一个新习惯，回顾过去

> 原文：<https://lifehacker.com/to-build-a-new-habit-look-to-the-past-1840852086>

如果你想在日常生活中增加一个新习惯，值得花时间反思一下你是如何养成当前习惯的，以及你是否可以用同样的技巧将另一个习惯或行为融入你的日常生活。



ClearerThinking.orgT3[的道格·摩尔和斯潘塞·格林伯格最近进行了一项研究，他们让近 500 人使用 23 种常见习惯养成技巧中的一种。例如，一些人在完成习惯后给自己奖励，而另一些人使用激励性的短语或咒语。](https://www.clearerthinking.org/)

当这项研究完成后，一项技术脱颖而出:hbit rreflection。正如摩尔和格林伯格解释的那样，在 [快公司](https://www.fastcompany.com/90446008/we-studied-the-best-way-to-actually-make-a-new-habit-stick) :

> 这种快速简单的方法帮助参与者平均每周比其他人多练习 0.7 次——比第二有效的方法提高了 140%。此外，那些使用习惯反射技术的人比研究中的其他任何一组人都更满意他们在养成习惯的目标上取得的进展。

什么是 hbit rreflection？听起来的确如此:花些时间来反思你之前是如何将一个积极的习惯融入生活的。

> 写下你从过去的情况中学到的关于如何成功养成新习惯的任何东西，或者你用来帮助实现这种改变的任何策略，这些策略可以应用到你的新习惯中。

一旦你理解了你过去是如何成功养成习惯的，你就可以使用同样的工具和策略来培养你现在想要养成的任何习惯。

也许你是那种需要把习惯堆起来的人，詹姆斯清晰式。也许日常口头禅对你有用。也许你需要使用 BJ·福格的[Bs "蜂群](https://lifehacker.com/use-the-swarm-of-bs-technique-to-successfully-form-new-1840617907) 方法，同时用多种行为来支持你的习惯。

摩尔和格林伯格的研究中有趣的一点是，那些选择了习惯培养策略而没有先进行习惯反思的参与者不如那些进行了习惯反思的参与者成功。

> 一些研究参与者被随机分配习惯支持技术；其他人选择了自己的。被随机分配技术的参与者表现得和那些自己选择技术的参与者一样好，而且研究参与者使用习惯反射比自己选择要好。

换句话说:你不能因为一项技术听起来不错，或者因为它听起来像你会做的事情，或者因为你认为它在过去可能有效，就选择它。你必须经历 hbit-r 反思过程，对自己的行为诚实以对，才能成功实现新的行为改变。

因此，在你开始安排、堆叠和优化你的 2020 年习惯之前，安排一些时间来反思你当前的习惯是如何到位的，你尝试并放弃的习惯发生了什么，以及你真正需要做什么来让你的新习惯坚持下去。

然后用你学到的东西开始培养新的习惯，一次一个。
