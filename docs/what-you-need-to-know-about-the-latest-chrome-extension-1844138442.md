### 如何保护您的浏览器(和您的数据)免受假冒插件的攻击

这是一段时间以来发现的规模较大的恶意软件活动之一。 [利用恶意浏览器扩展来监视人](https://lifehacker.com/uninstall-these-eight-browser-extensions-that-stole-dat-1836539093) 并不是什么新鲜事，但 [变得越来越常见](https://lifehacker.com/check-chrome-and-remove-any-of-these-70-malware-extens-1841692874) 。事实上，如此多的扩展被牵涉其中——而且其中大部分可以在谷歌的 Chrome 商店买到——这令人担忧，但还是有办法保证你的安全。

### 坚持知名来源

浏览 Chrome 网上商店时，最安全的做法是 [坚持使用由认证发行商](https://lifehacker.com/how-to-tell-legit-chrome-extensions-from-malware-1825423984) 开发的知名扩展。是的，这可能会限制你下载一个听起来超级酷的扩展，它可以做你真正真正在寻找的事情，但它也会让你的数据更加安全。

显然，你愿意用多少安全来换取舒适是你自己的事——我们有时也推荐不太知名的扩展——但是信任一个有良好记录的体面的独立开发者是一回事，而下载你看到的第一个扩展完全是另一回事，因为它听起来很有趣而不关注关于谁创建了它(以及他们想从你那里得到什么)的任何其他细节。

Chrome 扩展商店有一个“By Google”搜索过滤器，只对第一方扩展有用，Mozilla 有一个推荐的 Firefox 插件列表，如果你不想冒险进入更可怕的水域，你可以随时信任这些插件。

如果你这样做了，最好还是把你的安装限制在浏览器官方商店的扩展上。像谷歌这样的公司尽最大努力审查他们允许进入数字市场的附加产品，但是正如 Awake 的报告告诉我们的那样，可疑的开发者很容易绕过隐私政策和安全功能。

尽管如此，如果一个扩展在你浏览器的官方插件商店上，而不是从某个随机网页或弹出广告下载的话，它更有可能是合法的——只要确保你下载的是你认为你正在下载的: [检查扩展的名称、描述和细节是否都匹配](https://lifehacker.com/how-to-tell-legit-chrome-extensions-from-malware-1825423984) ，看看该扩展的评论听起来是否比真实的更虚假。如果有疑问，不要安装它——或者四处寻找一个更知名的替代产品。

### 检查权限

像手机应用程序一样，你应该对任何要求超出其广告用途的权限的扩展持怀疑态度。同样，应该不惜一切代价避免执行多余或不必要任务的扩展。

在 Awake Security 的报告中列出的几乎每一个扩展都要求至少一个粗略的或过分的许可。例如，许多人想要“截屏、读取剪贴板、获取存储在 cookies 或参数中的凭证令牌、抓取用户击键(如密码)。”

当你第一次添加一个扩展到你的浏览器时，会弹出一个通知，列出它能做什么。如果您对它的询问感到不舒服，请单击“取消”停止安装。如果你怀疑你正在使用的分机进行恶意活动，请立即报告。

### 寻找隐私政策

如果扩展能够访问敏感信息*和*，根据扩展的预期用途，这样做是必要的，请确保发布者的隐私政策清楚地概述了如何访问、存储数据和保护这些数据。隐私政策应该在扩展的商店描述中链接，或者在出版公司的官方网站上提供。如果不清楚或者找不到那个信息，那就不要安装。

如果你*发现*有隐私政策，你应该相信它吗？嗯，对于一个可疑的开发者来说，当他们真的不尊重你的隐私时，说他们尊重你的隐私并不难，但是公然忽略一项政策只是你可以用来衡量一个扩展的可信度的又一个数据点。

### **尝试开源选项**

虽然我们的一般建议是坚持使用众所周知的、经过验证的扩展，但这并不意味着较小的第三方插件或非官方下载位置本身就很危险。然而，他们确实需要格外小心地接近 T2。许多非常安全的扩展可以从独立开发者那里获得，比如在 [XDA 论坛](https://forum.xda-developers.com/) 或者 GitHub 上。

虽然你不能保证这些平台上的扩展是安全的，但这些开源项目通常有透明的代码和隐私政策，这使得它们更容易审查。如果你不知道你在找什么，做你的研究:阅读一些论坛。看看推特。点击 Reddit。在安装之前，看看是否有人对你想安装的扩展*提出了什么危险信号。*
