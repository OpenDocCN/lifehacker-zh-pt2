# 使用人工智能添加景深到任何使用此应用程序的照片

> 原文：<https://lifehacker.com/use-ai-to-add-depth-of-field-to-any-photo-using-this-ap-1833023809>

如果你有一部较新的智能手机，那么它可能会有拍摄 3D 图像的能力，或者至少有一些景深的图像。



要做到这一点，你的手机需要有两个摄像头。如果没有，那你就只能拍二维照片了。还是你？ [Dpth](https://www.dpth.app/) 是一款适用于 iOS 和 Android 的智能手机应用程序，它可以拍摄普通的 2D 照片，并使用人工智能将它们转换成图像，你可以在其中摆弄景深。

在尝试了一些照片后，我可以告诉你，你的里程数将根据你尝试使用的照片类型而有所不同。一切都在焦点上的照片效果最好，因为这个工具允许你对照片进行更多的调整。

例如，我拍摄的已经有一点散景效果的照片，比如我的狗或啤酒，实际上没有太大变化，而完全对焦的照片给了我更多的玩法。
