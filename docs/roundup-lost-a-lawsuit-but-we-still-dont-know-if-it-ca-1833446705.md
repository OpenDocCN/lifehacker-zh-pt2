# Roundup 输掉了一场官司，但我们仍然不知道它是否会导致癌症

> 原文：<https://lifehacker.com/roundup-lost-a-lawsuit-but-we-still-dont-know-if-it-ca-1833446705>

本周 [陪审团发现](https://www.wsj.com/articles/jury-finds-bayers-roundup-weedkiller-caused-mans-cancer-11553030151) 一名男子在自家院子里使用草甘膦除草剂农达是导致他多年后患上癌症的“重要因素”。但是法庭案例不是科学研究，科学家们在草甘膦是否真的应该被认为是致癌物的问题上仍然存在分歧。



正如我们在 之前报道的 [，世界卫生组织的国际癌症研究机构研究了证据，决定将草甘膦添加到“可能”致癌的物质清单中。名单上的其他东西包括红肉、热咖啡和人类乳头瘤病毒。](https://vitals.lifehacker.com/no-your-cereal-is-not-full-of-weedkiller-1828424844)

但是其他组织做了他们自己的分析并得出了不同的结论。美国环境保护署、加拿大卫生部和欧洲食品安全局都认为草甘膦不可能致癌。 [关于草甘膦的个别研究对大多数癌症](https://plantoutofplace.com/2018/08/glyphosate-and-cancer-revisited/) 给出了喜忧参半的结果，少数研究倾向于它确实会导致非霍奇金淋巴瘤的可能性。

[最近的研究](https://www.sciencedirect.com/science/article/pii/S1383574218300887) 发现，在工作中接触最多草甘膦的农场工人中，患非霍奇金淋巴瘤的风险为 2.8%，相比之下，普通人群中只有 2%。(那项研究的作者之一 [在博客中解释了这里的工作](https://racheltalkstox.wordpress.com/) 。)这是一个很小的风险变化，它可能不适用于普通消费者，他们只在自家后院或作为食物上的微量残留物遇到草甘膦。但是科学家们确实发现这种联系是可以察觉的。关于草甘膦科学证据的最新总结，我推荐汤姆·菲尔波特在《琼斯母亲》杂志 的 [《这位解释者》。](https://www.motherjones.com/environment/2019/03/glyphosate-roundup-cancer-non-hodgkin-lymphoma-epa-panel-hardeman-lawsuit-jury-verdict/)

与此同时，拜耳公司(拥有 Roundup，购买了前身为孟山都的公司)发表了一份声明，称有大量证据表明草甘膦是安全的，不会致癌。诚然，这是真的。

事实是，法庭案件不是科学研究。在最近的诉讼中，陪审团没有时间等待更多的结果，也没有科学专业知识来回答一个科学家尚未达成一致的问题。所有的诉讼真正的意思是，它将更容易带来未来的诉讼。在我们大多数人接触的水平下，草甘膦可能仍然是安全的...但是这需要更多的科学来确定。
