# 你应该多烤些鸡心

> 原文：<https://lifehacker.com/you-should-grill-more-chicken-hearts-1844233671>

我从不回避吃肉的本能体验。我发现吃带骨的排骨、牛排、内脏，甚至是字面上的都是一种享受。带骨牛排对于日常消费来说有点贵，而且需要做很多准备工作——但是鸡心很便宜，容易串起来，烹饪起来没有挑战，而且用途惊人。另外，他们是非常好的。



你可以花大约 1 美元 50 美分买到一磅鸡心，甚至可能更低，这取决于你在哪里买肉。(我在一家亚洲杂货店以每磅 1.35 美元的价格买到了我的。)你可以让你的屠夫清理它们，或者你可以自己修剪掉多余的脂肪和小动脉。我留下了大部分脂肪，但修剪了动脉。(它们很容易被发现；它们看起来像小白管。)

一旦清洗干净，你可以腌制或不腌制——它们都很好，我真的很欣赏它们。它们是最后一分钟完美的一口大小的肉，可以完善你的烧烤菜单，不管你是给它们短暂的关注，还是让它们在美味的卤汁中浸泡 24 小时，它们都很棒。

腌泡汁不需要精心制作。我发现等量的酱油和醋(雪利酒或苹果酒)加三汤匙糖效果很好。( [这些](https://skillet.lifehacker.com/stop-overthinking-your-marinades-1825768994) 也不错。)只需在香甜扑鼻的液体中浸泡一个小时就足以彻底调味，但你可以将它们放在里面整整 12 个小时。你也可以完全跳过浸泡，用 揉搓 [调味，但你可以保持比这更简单。鸡心的味道是黑色的，有牛排味，但没有野味，质地令人愉快耐嚼，只需要撒上大量的犹太盐来强调这两种品质。](https://skillet.lifehacker.com/four-simple-spice-rubs-you-can-make-in-five-minutes-or-1797698100)

要烹饪你的心，把它们串在(浸泡过的)竹串或金属串上，然后把你的烤架烤得又热又好(调高)。每面烤三分钟左右，再久一点，坚硬的小南瓜会从耐嚼变成有弹性。(我比较谨慎，一有颜色就尝一尝心脏样本。)立即上桌，用或不用你最喜欢的酱汁——烧烤和红烧都很有趣——用牙齿把它们从串肉扦上拔下来。串肉扦不是骨头，但这样吃还是感觉很原始。
