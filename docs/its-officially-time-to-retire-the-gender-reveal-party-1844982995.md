# “性别揭秘派对”正式退休了

> 原文：<https://lifehacker.com/its-officially-time-to-retire-the-gender-reveal-party-1844982995>

如果你在过去十年中的任何时候怀孕过，你可能听说过性别揭秘派对的概念。这种趋势可以追溯到一位母亲，她在 2008 年举办了被广泛认为是第一次性别揭示派对。 [詹娜·卡维尼迪斯](https://twitter.com/HighGlossSauce?ref_src=twsrc%5Egoogle%7Ctwcamp%5Eserp%7Ctwgr%5Eauthor) 切蛋糕露出粉色糖霜(是个女孩，乡亲们！)，以及她随后的博客帖子——以及这个想法本身——迅速传播开来。现在是时候停止了。

这些派对庆祝婴儿的性别作为他们主要的识别特征——毕竟，那时我们对他们知之甚少。说，婴儿可能会发现多年后，我们都错了，但他们仍然肯定会看到照片，从一个党的意思是专门庆祝的事情，他们不是。

在劳动节的周末，一对夫妇和他们的小孩前往加利福尼亚州的 El Ranch Dorado 公园，启动某种设备，显示他们未出生的孩子的性别，而心爱的人拍下了这一时刻。如 [彭博报道](https://www.bloomberg.com/news/articles/2020-09-07/california-fire-sparked-by-device-to-reveal-baby-s-gender) :

> ...他们使用的装置引发了一场野火，烧毁了数千英亩土地，迫使人们逃离洛杉矶东部的一个城市。
> 
> 这场大火促使拥有 5.4 万人口的尤卡伊帕市及其周边地区的部分居民撤离。滴水直升机被带来，但火势被证明是顽固的——到周一早上，火势扩大到 11.5 平方英里(30 平方公里)，现场的 500 多名消防队员只能控制最低限度的火势。
