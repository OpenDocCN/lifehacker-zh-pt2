# 如果你认为你的孩子是色盲，该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-you-think-your-child-is-color-blind-1840208029>

小孩子在他们最初的几年里学了很多东西。他们正在学习走路、说话和识别各种各样的东西——人、动物、数字、字母和颜色。事实证明，我们在英语中使用颜色的方式——通常是在名词前，或者在名词之前——没有帮助。我们倾向于说，“有一辆蓝色的汽车”，而不是“有一辆蓝色的汽车”，这使得孩子更难辨别汽车的是什么使它成为“蓝色”



但是如果你的孩子似乎总是混淆红色和棕色，即使你的直觉告诉你他们现在应该能够区分这两种颜色了，那该怎么办？你可能想知道他们是否有某种形式的色盲。

### 什么是色盲？

首先，“色盲”这个术语有点用词不当。这意味着患有这种疾病的人看不到任何颜色。但是最常见的是，色盲是某些颜色的缺乏，通常是红色和绿色，但有时是蓝色，根据美国眼科学会 T2 T3 的说法。例如，一个红绿色盲的人可能会将红色和绿色视为同一种颜色，他们可能会努力识别其中含有红色或绿色痕迹的颜色，如紫色和棕色。

如果你和我一样，对这种现象如何在眼睛内发生感到好奇，那么 [AAO](https://www.aao.org/eye-health/diseases/what-is-color-blindness) 就在这里:

> 在视网膜中，有两种类型的细胞检测光线。它们被称为视杆细胞和视锥细胞。视杆细胞只能探测光和暗，对弱光非常敏感。视锥细胞检测颜色，并集中在视觉中心附近。有三种视锥细胞可以看到颜色:红色、绿色和蓝色。大脑利用这些视锥细胞的输入来决定我们的颜色感知。

> 当一个或多个视锥细胞缺失、不起作用或检测到与正常颜色不同的颜色时，就会发生色盲。当所有三个视锥细胞都不存在时，就会发生严重的色盲。当所有三个视锥细胞都存在，但一个视锥细胞工作不正常时，就会发生轻度色盲。它检测到与正常颜色不同的颜色。

### 谁面临风险？

男孩患色盲的比率比女孩高得多，据估计，大约十分之一的男性患有某种程度的色盲。 [Hazel Health](http://www.hazel.co/) 的首席医疗官 Rob Darzynkiewicz 告诉 [Romper](https://www.romper.com/p/how-to-tell-if-your-kid-is-color-blind-what-to-do-if-they-are-18688514) 如果一个孩子的外祖父是色盲，概率会增加:

> “[色盲]是基于遗传的。它是从你的父母，尤其是你的母亲那里传下来的。红绿蓝视觉的基因可以在性染色体 x 上找到。女性有 XX，男性有 XY。因此，如果一个 X 基因工作不正常，而另一个 X 基因工作正常，你可能会没事。所以女生很少色盲。对于男孩来说，他们只从母亲那里得到一个 X 基因，如果是错误的基因，他们就有患色盲的风险。”

### 你如何测试它？

因为色盲实际上并不影响视力，它可能会在常规的医生预约或视力筛查中被遗漏。除非你有色盲的家族史，或者你有其他理由怀疑它，比如经常很难识别某些颜色，否则可能根本没有必要测试色盲。但是如果你这样做了，你可以在家里和你的孩子一起进行视觉测试作为初步检查；他们只需要足够大到能够识别数字——或者至少能够用手指描线——通常在 4-6 岁左右。

这些测试被称为 [石原测试](https://www.color-blindness.com/ishiharas-test-for-colour-deficiency-38-plates-edition/) ，是以日本东京大学教授石原北本忍的名字命名的，他在 1918 年为军方开发了筛查工具，根据 AAO 。它们在网上随处可见，由彩色点状图案组成，显示出中间的数字或漩涡线，没有色盲的人也能看到。

如果您的孩子看不清数字或图案，眼科医生也可能会对色盲进行额外的测试，以帮助诊断病情。

### 如果你的孩子*是*色盲，现在怎么办？

先天性色盲没有治疗方法——相对于后天获得的色盲，后者可能有治疗选择。它不应该导致任何重大的残疾或健康问题，但它可能会导致尴尬或沮丧。Darzynkiewicz 告诉 Romper

> “你总是可以每天调整你的思维:‘我们停在路灯旁边’而不是‘我们停在红色汽车旁边’或者‘让我们试试那件条纹衬衫’而不是‘让我们试试那件绿色衬衫’。如果他们感到沮丧，这可能是关于颜色的感觉，所以不要害怕问。让他们知道你在那里，你关心他们，想从他们的经历中学习。”

此外，以下是一些家长可以为他们的色盲儿童，尤其是小学色盲儿童提倡的方法:

*   让老师意识到你的孩子能看到什么，这样他们就能避免依赖于辨别某些颜色的游戏或活动。
*   请老师在教室里显眼或重要的公告板和其他显示器上标注颜色。
*   给任何可能让他们感到困惑的马克笔、彩色铅笔或其他美术用品贴上标签，这样他们就可以更容易地选择他们被告知要使用的颜色。
*   请老师用黑色写在白板上(或用白色粉笔写在黑板上)，而不是用彩色。

最重要的是，和你的孩子谈谈他们的情况，让他们明白他们看颜色的方式和大多数人不同，寻求帮助是可以的。
