# 准备吃剩的肉做沙拉的最好方法

> 原文：<https://lifehacker.com/the-best-way-to-prep-leftover-meat-for-a-salad-1840511798>

我最喜欢的寒冷天气餐形式之一是一大块烤肉，搭配淀粉和一些绿色或新鲜的植物物质。我通常会吃烤肉的剩菜，我几乎从不重新加热，因为我第一次做得很完美，该死的。(我可以用沉浸式循环器重新加热吗？是的，但是那需要一些时间，而且我很少想到在我非常饿之前开始做饭。)



实际上，对剩下的烤肉的最高要求是沙拉。冬天的这几个月让我们倾向于只吃烤肉、炖肉、意大利面和面包。R 用一堆树叶包装一些羊肉、牛肉或猪肉会让你觉得很高尚，即使你已经被 [打得有点过了。](https://skillet.lifehacker.com/the-best-eggnog-is-made-with-tequila-1840348883)

这并不像把它切成片然后和植物部分一起扔掉那么简单——当然不是这样——但也不会更困难。要把一块老烤肉变成真正美味的沙拉蛋白，你只需要一点油和一点盐。

油尤其重要，特别是如果你的烤肉已经在冰箱里放了一段时间，或者如果你正在用更干燥的剩余部分。把你的烤肉切得越薄越好，淋上足够的油，轻轻地涂在肉片上，然后给它一点按摩，让它醒过来。这不仅能恢复干燥部位的光泽，还有助于疏松烤肉中的脂肪部分。(如果我记得有机化学中的一件事，那就是物以类聚。)撒上一些盐，也许是一点点胡椒粉，或者你周围的任何其他调料，把肉放在一边，准备剩下的沙拉。

剩下的材料由你决定——我通常用这个 oppo rtunity 来清理我的冰箱——但我发现一个时髦的蓝奶酪、一根葱(或超级切片洋葱)和某种水果(苹果、浆果或核果)是一个成功的组合。用 [简单的调味汁](https://skillet.lifehacker.com/vinaigrettes-need-way-less-oil-than-you-think-1837655884) 搅拌，食用，整个冬天重复。
