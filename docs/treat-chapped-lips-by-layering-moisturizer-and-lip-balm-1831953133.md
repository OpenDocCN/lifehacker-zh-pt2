# 通过涂抹保湿霜和润唇膏来治疗嘴唇干裂

> 原文:[https://life hacker . com/treat-chapped-lips-by-layering-moister-and-lip-balm-1831953133](https://lifehacker.com/treat-chapped-lips-by-layering-moisturizer-and-lip-balm-1831953133)

完美的 [润唇膏](https://lifehacker.com/know-which-ingredients-in-lip-balm-actually-cause-dry-l-30762383) 既要薄到能渗入你的嘴唇，又要厚到能在嘴唇上形成一层防水保护层。换句话说，它并不存在——但是你可以用一点点巧妙的分层来伪造它。

Watch

如果你的嘴唇现在极度干燥，试着在你裸露的嘴唇上擦一点你常用的面部润肤霜或眼霜，*，然后*在上面涂一层厚厚的唇膏。乳液和面霜比润唇膏含有更多的水和保湿成分，但更重要的是，它们的配方能真正被皮肤吸收。在上面涂上一些蜡质和闭塞的东西会减缓水分蒸发的过程，把水分保持在你需要的地方:你干燥、干裂、痛苦的嘴唇上。这项技术对于任何有顽固嘴唇皲裂的人来说都是天赐之物，但对于不能使用羊毛脂的人来说尤其方便。蜡质绵羊皮脂治愈干燥皮肤的效果在地球上独一无二，但它是一种相当常见的过敏原；试图找到一款没有它的有效润唇膏比你已经拥有的两种产品更难，也更贵。

你喜欢的任何保湿霜在这里都可以，只要不含活性成分。当然，润肤霜中视黄醇、化学去角质剂(AHAs、BHAs 或其他)和维生素 C 衍生物的浓度很低，但不值得冒险。不要把你最喜欢的视黄醇霜涂在你的嘴唇上——用面部油、补水精华液、眼霜、护手霜，甚至身体乳来代替。你的嘴唇会感谢你。
