# 要存更多的钱，像乐观主义者一样思考

> 原文：<https://lifehacker.com/to-save-more-think-like-an-optimist-1834242942>

一项新的研究表明，积极思考的力量也会对你的银行存款余额产生积极影响。

Watch

[杂色傻子](https://www.fool.com/retirement/2019/04/21/why-a-bright-outlook-in-life-can-help-you-save-mor.aspx) 报道:

> 高达 90%的乐观主义者表示，他们已经为购买大件商品存了钱，相比之下，只有 70%的悲观主义者这样认为。61%的乐观者已经建立了应急基金，而只有 43%的悲观者这么做了。乐观主义者也更有可能找到存钱的方法，即使在资金紧张的时候，75%的人说他们在存钱的时候变得有创造力，相比之下，60%的悲观主义者这样做过。

这一数据来自弗罗斯特银行和积极心理学研究员米歇尔·吉兰 最近的一项研究，表明乐观者比悲观者拥有“更健康的财务状况”的可能性高七倍

为什么会这样？好吧，乐观主义者可能更容易相信他们可以对自己的未来做出积极的选择——这包括为未来存钱的选择。

或者，正如弗罗斯特银行所说:

> 乐观主义者并不完美。他们仍然面临财务压力(平均每年 81 天)，大多数人只有粗略的财务计划(54%)而没有详细的计划(35%)。他们发现朝着财务目标前进很重要——58%的人对他们的个人财务状况感到乐观，因为他们正在朝着财务目标前进。

如果你更悲观，有可能训练你的大脑更积极地思考——或者至少哄骗它暂时乐观。这一切都归结于你选择感知世界的方式(或者你告诉自己你将感知世界的方式)。你是在寻找乌云还是希望？你把别人想得最坏，还是想得最好？

根据弗罗斯特银行的报告，乐观主义者更有可能将负面事件或财务挫折视为学习的机会，然后利用他们所学到的知识重新致力于他们的财务目标。

你呢？你认为自己是乐观主义者还是悲观主义者？你的人生观有没有影响到你的理财习惯，你有没有通过改变理财习惯来改变你的人生观——或者相反？
