# 使用 Windows 快速助手对电脑进行远程故障排除

> 原文：<https://lifehacker.com/remotely-troubleshoot-a-pc-with-windows-quick-assist-1842865257>

很多时候，无法亲自进行故障诊断。也许你和对方住在不同的城市，或者你在不同的办公室工作 或者世界因为疫情而被封锁。无论情况如何，一个有用的 Windows 10 功能可以帮助你远程帮助你不懂技术的同事、朋友和家人。

它叫做 Windows Quick Assist，你可以用它通过互联网访问另一个用户的桌面，帮助他们解决他们的计算机可能遇到的任何问题。该工具为提供了几个有用的功能，包括高亮显示其他用户屏幕的特定部分，甚至管理他们的系统设置。

### 如何使用 Windows 快速助手

要使用 Windows Quick Assistant，两个人都需要在他们的台式机或笔记本电脑上安装 Windows 10，并且两个设备都必须打开并连接到互联网(但是，他们不必在同一个网络上)。只要您符合这些标准，您就可以通过以下步骤使用 Windows 快速助手连接到其他人的电脑:

1.  从 Windows 开始菜单或任务栏搜索栏中搜索**“快速助手”**。
2.  点击**“快速辅助”**运行程序。
3.  选择**“给予帮助”**
4.  按照屏幕上的说明登录您的 Microsoft 帐户(如果您没有，请创建一个)。
5.  登录后，您将获得一个六位数的访问代码。为下一节准备好这个。

现在我们将跳到你想帮助的那个人身上。

1.  在对方的电脑上，让他们搜索并运行快速协助，就像我们在上一节中所做的那样。
2.  快速协助打开后，让对方点击**“获取协助”**
3.  让他们键入您将提供给他们的六位数访问代码，然后点击**“提交”**
4.  出现提示时，让他们点击**“允许”**。

如果一切顺利，您现在应该可以通过快速助手访问对方的桌面。从这里开始，你作为助手可以点击其他人的电脑，做需要做的事情。两个用户都可以在应用程序中进行一些快速辅助操作:

*   **注解:**这个功能可以让你在对方的桌面上画画。这对于记下听下笔记或者高亮显示你想引起他们注意的屏幕部分很有用。当快速辅助关闭时，您所做的任何注释都将被删除。
*   **暂停:**暂时暂停快速辅助。再次单击此处继续。
*   **重新启动:**重新启动对方的电脑，但不禁用快速辅助。帮助者将在他们的机器重新启动后重新获得对其他人的桌面的访问权。
*   **停止:**这将结束快速辅助会话。您也可以随时关闭快速协助窗口以结束快速协助。
*   **任务管理器:**让帮助者打开对方的任务管理器。如果你需要检查他们正在运行什么程序，并且不想解释什么是**“CTRL+ALT+Delete”**的意思，这是非常有用的【】。

当您准备关闭 Windows 快速助手时，只需按下**“停止”**和/或关闭应用程序的窗口即可结束连接。请注意，如果您希望将来重新连接，您需要重复上述步骤。
