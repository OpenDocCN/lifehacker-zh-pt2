# 什么时候我应该扔掉有凹痕的罐头食品？

> 原文：<https://thetakeout.com/when-should-i-throw-away-a-dented-can-of-food-1832374592>

我强调不要从亚当·桑德勒电影中获取个人理财的线索，所以我从来没有尝试过《老爸》中桑德勒指导他的儿子把意大利面条罐头扔在地上的特技(“有凹痕的罐头半价”)。这不仅是杂货店省钱的一种不正当的方法，而且很可能是不安全的；我们都被告知不要吃严重凹陷的罐头食品。为什么？

 [https://thetakeout.com/embed/inset/iframe?id=youtube-video-nEDLGQ7qxwI&start=0](https://thetakeout.com/embed/inset/iframe?id=youtube-video-nEDLGQ7qxwI&start=0) 

“永远不要购买或使用鼓胀、生锈、渗漏或凹陷很深的罐头，”美国农业部食品安全和检验服务[的技术信息专家 Janell Goodwin 警告我。大的凹痕经常出现在罐头内部的一个点上——我们看不到——这可能导致包装上的小破裂，从而使细菌进入罐头并污染食品。因为我们通常看不到一个很深的凹痕是否破坏了罐子，所以我们经常被建议为了安全起见，扔掉所有有很深凹痕的罐子，以及接缝处有凹痕的罐子。另一方面，只有小凹痕的罐子是安全的。](https://www.fsis.usda.gov/wps/portal/fsis/home)



你怎么能确定一个凹痕有多深呢？美国农业部说 一个凹陷很深的罐头是你可以“把手指伸进去”的全国杂货商协会的执行副主席格雷格·费拉拉说，因为这些罐头会导致“食品安全问题”，杂货店会把它们从货架上撤下来。但是如果你不愿意扔掉那些食物，你还有一个选择。蒙特利尔麦吉尔大学科学与社会办公室主任 Joe Schwarcz 指出，你可以煮沸罐头的内容物，这足以杀死微生物或任何可能由肉毒杆菌产生的毒素(这种细菌的神经毒素可以导致肉毒中毒)。

现在，如果你想真正了解有凹痕的罐子，让我们卷起袖子:不是所有的凹痕都是一样的。没有大边缘或折痕的侧面凹痕？可能没什么风险。接缝上有侧凹吗？有风险。没有底缝的罐子底部有凹痕？不冒险。顶部和底部都有接缝的罐子底部有凹痕吗？有风险。你可以看看我所说的 [这个方便的视觉指南，它可以(真正地)从科罗拉多州维尔德县公共卫生部&的环境中削弱](https://www.weldgov.com/UserFiles/Servers/Server_6/File/Departments/Health%20and%20Environment/Enrivonmental%20Health/Food%20Safety%20(for%20Regulated%20Facilities)/About/Dented%20Can%20Handout.pdf) 。即使在所有这些照片和令人沮丧的解剖后，该机构最终敦促谨慎:“当有疑问时，扔掉它！”
