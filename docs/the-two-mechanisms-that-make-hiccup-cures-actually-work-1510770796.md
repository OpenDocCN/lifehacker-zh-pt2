# 治疗呃逆的两种机制实际上是有效的

> 原文：<https://lifehacker.com/the-two-mechanisms-that-make-hiccup-cures-actually-work-1510770796>

打嗝无疑是令人讨厌的，它真的会扰乱你的一天。你可能有自己的家庭疗法来让他们停止。如果它们有效，那很好，但是你可能会奇怪为什么一种方法比另一种更有效。事实证明， [英国广播公司进行了一项调查](https://www.bbc.com/future/article/20140127-do-hiccup-remedies-work) ，找出需要发生什么才能让你不再打嗝。



大笑、大量饮酒、快速进食，或者仅仅是一种奇怪的自发反应，都可能导致你的横膈膜痉挛，从而使空气涌入肺部，导致你的声带突然闭合，并发出打嗝的“嗝”声。它们令人讨厌，令人生厌，而且似乎总是发生在最不合适的时候。打嗝没有真正的治疗方法，但真正有效的家庭疗法分为两类:提高血液中二氧化碳水平的疗法和刺激迷走神经的疗法。

### **二氧化碳含量增加**

如果你曾经有过像屏住呼吸或对着纸袋呼吸这样的打嗝停止技术，很可能是因为这些方法提高了血液中的二氧化碳(CO2)水平，抑制了横膈膜的痉挛 [据 BBC](https://www.bbc.com/future/article/20140127-do-hiccup-remedies-work) 报道:

它们有时会起作用，但是研究人员不确定为什么。一些人认为这是转移身体注意力的一种方式，这样身体就会担心二氧化碳的积累；其他人提出打嗝可能首先是由低水平的 CO2 引起的，因此高水平会抑制打嗝。

### **刺激迷走神经**

人们用来消除打嗝的许多其他家庭疗法——比如那些涉及喝水或有人吓你的疗法——都属于这一类。这是因为它们刺激迷走神经，迷走神经从你的大脑延伸到你的胃，有助于协调你的呼吸和吞咽。下面是英国广播公司在调查中发现的 :

> 这条神经与打嗝过程有关，但你可以通过刺激神经来扰乱事件链，让它向大脑发送信号，告诉它关注这种新的感觉。这就是诸如大口喝水、咬一个柠檬或吃碎冰等补救方法的用武之地。同样，拉舌尖、把手指放在耳朵里或轻轻按压眼球，都可以刺激迷走神经。把它想象成通过让其他戏剧性的事情发生来转移身体对打嗝的注意力。同样的逻辑也适用于给别人一个惊吓。

基本上，科学家们并不完全确定为什么呃逆治疗有效，但教训是，如果某事对你有效，继续做下去。
