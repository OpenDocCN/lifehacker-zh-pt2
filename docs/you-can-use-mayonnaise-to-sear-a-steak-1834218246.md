# 你可以用蛋黄酱烤牛排

> 原文：<https://lifehacker.com/you-can-use-mayonnaise-to-sear-a-steak-1834218246>

我是在密西西比州北部长大的，因此，我非常喜欢被称为“蛋黄酱”的白色乳液。它不仅是三明治中必不可少的，而且它还能让[超级湿润的巧克力蛋糕](https://skillet.lifehacker.com/3-unexpected-things-you-can-do-with-mayo-that-arent-put-1798356585) ，还有——我最近才知道——非常棒的烤牛排。

阿尔顿·布朗我很沮丧我第一次错过了，因为“蛋黄酱牛排”是我最讨厌的。如果你抛开你正在研究的任何蛋黄酱恐惧症，想想蛋黄酱中实际上有什么*(大量的油、一些鸡蛋、糖和调味料)，它作为一种烧焦剂是完全有意义的。糖和蛋白质有助于褐变，而油则起到了油的作用。当我周五晚上在丹佛牛排上尝试它时，我发现它在几分钟内创造了一个极好的外壳，而且我向你保证，没有一点蛋黄酱的味道。*

如果你想象一块涂有蛋黄酱的牛肉，要知道你只需要一点点。就像你用一层薄薄的中性油来处理你的锅一样，你用一层薄薄的——几乎半透明的——调味品涂层来处理你的牛排。只要用油漆刷蘸一些蛋黄酱，两面都涂上，然后像平常一样调味。在一个灼热的，未经处理的锅里烤焦(因为牛排是经过处理的，你知道)，然后煮到你想要的熟透程度。漫威在梅奥脆皮，狼吞虎咽地吃你的牛排，并根据需要重复(也许可以用鲑鱼 试试 [)。](https://lifehacker.com/the-secret-to-perfectly-grilled-fish-mayonnaise-1709142603?_ga=2.145696500.1386177238.1555939040-1723114163.1524514905)
