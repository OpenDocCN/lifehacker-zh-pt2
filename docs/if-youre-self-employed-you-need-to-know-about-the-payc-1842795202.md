# 如果你是自由职业者，你需要了解工资保障计划

> 原文：<https://lifehacker.com/if-youre-self-employed-you-need-to-know-about-the-payc-1842795202>

本月早些时候，联邦政府推出了 [薪酬保护计划](https://www.sba.gov/funding-programs/loans/coronavirus-relief-options/paycheck-protection-program-ppp) ，该计划让小企业主有机会申请小企业管理局(SBA)贷款，旨在帮助企业在冠状病毒疫情期间发放工资。如果贷款被用于某些指定的费用——工资、抵押贷款利息、租金和公用事业——贷款有可能被部分或全部免除。



今天，工资保护计划(PPP)对个体经营者、独立承包商和个体经营者开放。 [下面是如何应用](https://www.sba.gov/sites/default/files/2020-04/PPP%20Borrower%20Application%20Form.pdf) ，下面是你需要知道的:

## 你的业务需要受到疫情的影响

我是一个个体经营者，但是我能够在整个新冠肺炎疫情继续我的自由职业工作。这意味着我没有资格申请 SBA 贷款。如果你的自由职业或独资企业受到疫情的影响，你可能有资格申请。

## 该贷款仅涵盖某些类型的业务费用

如果您完成 PPP 申请，以下是您同意的内容:

> 这些资金将用于保留工人和维持工资或支付抵押贷款利息、租赁付款和公用事业付款，如工资保障计划规则所规定；我明白，如果资金被故意用于未经授权的目的，联邦政府可能会追究我的法律责任，例如欺诈指控。

换句话说:你不能用这笔钱购买广告、升级笔记本电脑或预订即将到来的会议的旅行(老实说，所有即将到来的会议都已经取消了)。薪酬保护计划顾名思义就是:在疫情和随后的经济复苏期间保护人们薪酬的一种方式。

## PPP 项目旨在保护个体经营者的收入

鉴于许多个体经营者不向雇员发放工资，你可能会问自己是否真的有资格在工资保障计划下获得小型企业管理局贷款。幸运的是，你的薪水也很重要——所以如果你在过去几周收到的客户支票少了，你可能有资格申请贷款。

下面是 [关心法案](https://www.congress.gov/bill/116th-congress/house-bill/748/text) 对此的解释:

> [...术语“工资成本”是指[与雇员工资支票相关的一长串要求，其后是]支付给独资经营者或独立承包商的任何报酬或收入的总和，即工资、佣金、收入、自雇净收入或类似报酬，其金额在一年内不超过 100，000 美元，在涵盖期内按比例分配[...]

换句话说:如果你是一名自由职业者或个体经营者，但没有得到足够的薪水，你可能有资格获得薪水保障。

## 你可能会获得部分或全部贷款豁免，这取决于资金是如何使用的

根据薪水保护计划，小型企业管理局的贷款有资格获得豁免，这取决于钱是怎么花的。我再引用一下贷款申请:

> 我理解，将为记录的工资成本、担保的抵押贷款利息支付、担保的租金支付和担保的公用事业提供贷款豁免，并且不超过 25%的豁免金额可用于非工资成本。

如果你想得到原谅，明智地使用你的资金。

## 申请过程和贷款分配可能需要一段时间

假设你非常积极主动，今天 [填写了你的 PPP 申请](https://www.sba.gov/sites/default/files/2020-04/PPP%20Borrower%20Application%20Form.pdf) 。你什么时候能收到贷款？有点不好说；这个项目是全新的，政府现在正在处理很多事情(我们不都是)，还有很多小企业主需要财政帮助。正如 [快公司](https://www.fastcompany.com/90489017/paycheck-protection-program-loans-for-self-employed-what-to-know-about-the-sba-application-process) 报道:

> 小型企业管理局、贷款人，以及基本上所有参与向人们发放贷款的人，似乎都被大量的申请淹没了。运气好的话，你开始得越早，你就会过得越好。

如果你对薪水保障计划有其他问题——我猜你有——从 [小企业管理局的计划概述](https://www.sba.gov/funding-programs/loans/coronavirus-relief-options/paycheck-protection-program-ppp) 开始，然后阅读 [贷款人和借款人常见问题](https://www.sba.gov/sites/default/files/2020-04/Final%20PPP%20FAQs%20for%20Lenders%20and%20Borrowers%204-8-20_0.pdf) 。如果您已经通过 PPP 申请了小型企业管理局贷款，无论您是独资经营者还是经营其他类型的小型企业，我们都希望听到您对该流程的想法。
