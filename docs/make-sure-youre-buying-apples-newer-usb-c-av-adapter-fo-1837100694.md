# 请确保您购买的是苹果新款 USB-C AV 适配器，适用于 Mac 和 iPad Pro

> 原文:[https://life hacker . com/make-sure-you-buying-apple-newer-USB-c-av-adapter-fo-1837100694](https://lifehacker.com/make-sure-youre-buying-apples-newer-usb-c-av-adapter-fo-1837100694)

苹果公司更新了其 [USB-C 数字 AV 多端口适配器](https://support.apple.com/en-us/HT207806) ，这是一种可以插入 Mac 或(新款)iPad Pro 的加密狗，因此你可以连接外部显示器或标准 USB 设备。新版多端口适配器增加了对 HDMI 2.0 的支持，包括:

*   4K UHD 和 60 赫兹视频播放
*   HDR 支持(HDR10 和杜比视界)

这些是对以前的 USB-C AV 多端口适配器的出色升级，以前的 USB-C AV 多端口适配器仅支持 HDMI 1.4b。虽然苹果已经停止了旧版本的适配器，现在在其官方商店(在线或零售)销售新版本，但旧适配器从其他商店货架上消失还需要一段时间。为了确保您挑选的是较新的加密狗，请务必检查包装上的**型号 A2119** (旧型号为 A1621)。

Watch

虽然该适配器可以与任何具有 USB-C 连接的 Apple 设备配合使用，但如果您有以下设备之一，您将只能最大限度地利用 HDMI 2.0 连接(以 60Hz 驱动 3840 x 2160 分辨率的显示器)。

*   15 英寸 Mac Book Pro (2017 年款或新款机型)
*   Retina iMac (2017 年款或新款机型)
*   怪兽
*   iPad Pro

您的设备还必须运行 macOS Mojave (10.14.6 或更高版本)或 iOS 12.4(或更高版本)。
