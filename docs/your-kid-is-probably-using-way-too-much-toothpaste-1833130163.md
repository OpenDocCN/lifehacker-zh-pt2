# 你的孩子可能用了太多的牙膏

> 原文：<https://lifehacker.com/your-kid-is-probably-using-way-too-much-toothpaste-1833130163>

当我的女儿蹒跚学步时，她讨厌刷牙，但一旦我们发现她喜欢的牙膏味道(当然是粉红色的泡泡糖)，她的视角立即转移到“OOOH 糖果！”每次我挤一些到她的牙刷上，她都抱怨，“太少了！”她想要一整口的东西，虽然我知道我不能让她吃，但我只是模糊地知道为什么。

Watch

现在有来自疾病控制和预防中心的确认:这对牙齿不好。长期摄入过多的氟化物会使儿童的永久性珐琅质结构变色和凹陷，这种情况被称为氟中毒。许多年轻的刷牙者处于危险之中。根据 [疾病预防控制中心最近的一项研究](https://www.cdc.gov/mmwr/volumes/68/wr/mm6804a3.htm?s_cid=mm6804a3_e) ，近 40%的 3 至 6 岁儿童使用的牙膏超过了推荐量，这是“一个豌豆大小的量”3 岁以下的儿童应该使用米粒大小的“涂片”。
