# 为了提高记忆力，整理你的照片

> 原文:[https://life hacker . com/to-improve-memory-recall-curate-your-photos-1841334875](https://lifehacker.com/to-improve-memory-recall-curate-your-photos-1841334875)

这几天拍照片的时候，都没怎么想。我知道会自动保存到云端；我知道它会自动保存一个时间戳。这不像过去，拍照的目的是把照片打印出来，然后把照片塞进相册的袖子里。

Watch

但这也意味着人们很容易忘记你所拥有的照片，或许还有它们本该纪念的经历。有研究表明，当你拍照时，你可能不会像在没有相机的情况下经历了同样的经历那样记得很清楚。

安吉拉·拉什布鲁克 [在《元素》上写了一些关于](https://elemental.medium.com/social-media-is-messing-with-our-memories-fae0c14c6b0d) 的研究。一项研究表明，当我们拍照时，我们更多地考虑的是人们会如何看待这张照片(例如，人们是否会在 Instagram 上喜欢它)，而不是我们实际上想要注意或记住什么。

我认为这不一定是件坏事。任何优秀的摄影师都会在拍照时考虑，照片可能会传达给那些看到照片挂在画廊或发布在网上的人。在我们自己的生活中，思考照片的目的是什么，以及你拍照的目的是什么，而不是无意识地抓拍，是有意义的。

我喜欢琳达·汉高告诉 Elemental 的一个解决方案，她研究拍摄和分享照片的心理学。“汉高说，如果将少量照片汇集成一本相册，照片会更好地激发你的记忆。”

然后当你想提醒自己你的假期，或者你孩子的第一年，或者另一个难忘的经历时，你可以回顾这些照片。你可能不会翻阅去年的成千上万张照片，但也许你会选择其中最好的，并不时回头看看。你甚至可以打印一本相册，或者在墙上挂一些你喜欢的照片。这样做，你会有更多的机会去思考为什么那些时刻对你来说是特别的。
