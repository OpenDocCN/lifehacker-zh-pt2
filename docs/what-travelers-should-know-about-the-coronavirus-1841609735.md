# 旅行者应该了解冠状病毒的哪些知识

> 原文：<https://lifehacker.com/what-travelers-should-know-about-the-coronavirus-1841609735>

导致新冠肺炎(我们现在称之为)的冠状病毒仍然主要在中国传播，在其他国家只有大约 2%的病例。尽管如此，如果你正在旅行，你可能会对接触到病毒的可能性感到紧张。以下是世界卫生组织对旅行者的建议，他们在最近的 [在线 Q & A](https://twitter.com/WHO/status/1226810780323958784) 中讨论了这些建议。



### 你可能不需要取消你的旅行

在拥挤的地方，比如机场，你感染任何疾病的风险都比呆在家里要高。但是对于几乎所有地方的旅行者来说，感染新冠肺炎病毒的风险几乎为零。

不过，世卫组织官员建议与航空公司核实，因为许多航空公司已经暂停航班或减少了运行的航班。而且有些国家对回国的旅客实行隔离，所以要查看相关政策。

一些机场正在对返回的旅客进行病毒筛查，所以如果你乘坐的是那种 ，那么 [就要小心了。可能会有台词。](https://lifehacker.com/build-in-extra-time-to-clear-customs-at-coronavirus-scr-1841492343)

### 除非生病，否则不要戴口罩

口罩在防止健康人感染呼吸道疾病方面作用很小，但对于防止病人将病菌传染给他人却很有好处。

### 使用与季节性流感相同的预防措施

导致新冠肺炎的病毒以与季节性流感相似的方式传播，通过咳嗽或打喷嚏时进入空气中的飞沫传播。同样的预防措施也适用:

*   经常洗手，包括吃饭前。
*   如果你的手看起来很干净，你可以用洗手液代替洗手。
*   对着纸巾咳嗽或打喷嚏，然后扔掉纸巾，或者如果没有纸巾，对着你的手肘咳嗽，而不是喷向周围或你的手。
*   与看起来生病的人保持距离——至少三英尺远。
*   为了保护婴儿，让他们靠近你，远离其他可能生病的人。
*   如果*你*生病了，做上面所有的事情，尽可能避免与他人接触。如果你认为自己可能患有新冠肺炎，或者担心自己的健康，请寻求医疗帮助。

好消息是，这种病毒似乎是通过空气传播的，所以飞机上的空气流通不太可能在机舱里传播。

### 理解病毒在表面上是短命的

如果含有病毒的液滴沾到表面，比如门把手，你就有可能被感染。这就是为什么手部卫生很重要。

但是这种病毒似乎只能在表面存活不到 30 分钟。这意味着包裹和行李内容不被认为是危险的。
