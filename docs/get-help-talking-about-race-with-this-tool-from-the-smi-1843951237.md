# 使用史密森尼学会的这个工具来帮助谈论种族

> 原文:[https://life hacker . com/get-help-talking-about-the-tool-from-the-SMI-1843951237](https://lifehacker.com/get-help-talking-about-race-with-this-tool-from-the-smi-1843951237)

当然，到现在为止，我们都看到了谈论种族的迫切需要——与我们的孩子，与我们的父母，与我们的朋友，甚至与我们自己。如果唯一阻碍你的是你根本不知道从哪里开始，史密森尼国家非裔美国人历史和文化博物馆已经推出了一个新的在线门户网站，这将有所帮助。

Watch

这个数字门户网站包含工具、在线练习、视频指导、文章和 100 多种为教育工作者和家长量身定制的多媒体资源，在乔治·弗洛伊德去世后全国爆发抗议活动之前，它就已经在工作中了。看到眼前的需要，博物馆提前了“ [谈论种族](https://nmaahc.si.edu/about/news/national-museum-african-american-history-and-culture-releases-talking-about-race-web?fbclid=IwAR1skR1fJb47d-iTszVLa26cw9XaWv6N7uBk0T450ngnWo3fDz5q2IFHoZI) ”的上映日期，以提供人们可以立即使用的资源:

> 谈论种族建立在博物馆教育者几十年的工作基础上。它是来自历史、教育、心理学和人类发展等领域的广泛研究、调查、咨询和教育资源的结果。它包括领先的专家、活动家、历史学家和思想领袖发表的关于种族、平等和包容的研究。

[谈论种族](https://nmaahc.si.edu/learn/talking-about-race) 首先让读者反思自己一生中围绕种族的旅程和经历——这是重要的第一步。此外，还有各种各样的文章、视频和活动，可以帮助教育用户了解美国的种族不平等——并武装你与你的孩子谈论这个问题。我特别推荐:

*   [本义交代了“白度”](https://nmaahc.si.edu/learn/talking-about-race/topics/whiteness)
*   [这篇文章讲述了不同类型的种族主义](https://nmaahc.si.edu/learn/talking-about-race/topics/being-antiracist) (以及如何成为一名反种族主义者)
*   [本页视频关于](https://nmaahc.si.edu/learn/talking-about-race/topics/being-antiracist)
*   [这个关于社会身份的活动](https://sites.lsa.umich.edu/inclusive-teaching/2017/08/24/mapping-social-identity-timeline-activity/) 你可以在家里和你的孩子一起做
*   [本《种族平等工具词汇》](https://www.racialequitytools.org/glossary)

和孩子谈论种族问题可能很难，尤其是如果你以前从未有过这样的经历，但这也是必要的；这些工具可以为你提供入门所需的知识。
