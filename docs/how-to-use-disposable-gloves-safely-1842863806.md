# 如何安全使用一次性手套

> 原文：<https://lifehacker.com/how-to-use-disposable-gloves-safely-1842863806>

美国疾病预防控制中心并不建议将 手套作为日常预防措施，但我知道很多人在外出时都选择戴一次性手套，例如， [在杂货店购物](https://lifehacker.com/what-precautions-are-you-taking-at-the-grocery-store-1842840991) 。如果你最终要使用手套，正确佩戴是很重要的。否则，他们可能会做更多的伤害他们的好。



### 
手套到底有什么作用

你不会因为皮肤上染上冠状病毒而生病；你会因为脸上的洞感染了病毒而生病。这就是我们洗手的原因:如果病毒沾到你的手上，你在摸脸之前先把它洗掉。

所以戴手套的目的不是让病毒远离你的皮肤。这是为了让病毒留在你的手套外面，这样你就可以脱下手套，重新拥有一双干净的手。“需要的时候不换手套和不洗手没什么区别，” [微生物学家西蒙·克拉克在对话中写道](https://theconversation.com/coronavirus-the-danger-of-latex-gloves-134701) 。

也就是说，最好的做法是在戴手套前后洗手，以确保你没有污染你的手套，反之亦然。

### 你仍然需要注意你接触的东西

如果你触摸门把手，然后触摸你的脸，你可能刚刚把病毒放在你的脸上——不管你有没有戴手套。

戴手套的一个危险是，人们倾向于认为他们受到了保护，然后他们会不小心用同一副手套接触干净和脏的东西。你见过三明治店的人戴着手套拿走你的钱，然后用同样的手套去做下一个人的三明治吗？这正是你*不想*做的事情。

因此，如果你戴着手套购物，在触摸手机或钱包之前，你应该先摘掉手套。或者如果你戴着手套碰了那些东西，确保之后 [清理它们](https://lifehacker.com/the-dos-and-donts-of-disinfecting-your-phone-1842400895) 。



### 知道如何安全地脱下手套

如果一只手套的外表面被污染了，当你脱下手套时，你不想用那只干净的手去接触它。所以技术在这里很重要。

有很多视频展示了正确的技术，但我喜欢这个，因为它使用粘性烧烤酱来说明问题。*当你试图让干净和肮脏的表面相互远离时，表现得好像所有东西都沾满了烤肉酱。你能用那只烤肉酱手摸摸你的脸吗？你会用它触摸你的手机吗？我不这么认为。*
