# 如何在各种金融挫折中生存下来

> 原文：<https://lifehacker.com/how-to-survive-every-type-of-financial-setback-1838625506>

即使你认为你在财务上做的一切都是正确的，事情仍然可能出错。失业、离婚和疾病都会对你的财务状况造成严重破坏，车祸、自然灾害也是如此——好吧，你明白了。



如果一个意想不到的紧急财务状况让你花光了应急基金里的所有钱，甚至更多，你会怎么做？你如何从金融挫折中恢复过来，无论你是被追尾、离婚，还是发现自己正处于又一次衰退之中？

在 Kiplinger，特许金融顾问 Mike Piershale 建议立即采取行动:

> 从面对现实开始。这可能说起来容易做起来难，尤其是当你宁愿避免焦虑和内疚来面对你没有做好财务准备的现实时。

由此，Piershale 提供了你可能认为是标准的财务建议:

*   做个预算。如有必要，削减一切不必要的开支。
*   想办法增加收入。没有工作就去找工作；如果你这样做了，得到一个侧面拥挤。
*   开始将尽可能多的钱投入到受此次挫折打击最大的财务领域，无论是偿还债务、重建应急基金，还是努力实现退休储蓄目标。
