# 为了让自己感觉更好，学习一项新的小技能

> 原文：<https://lifehacker.com/to-feel-better-about-yourself-learn-a-small-new-skill-1838530362>

如此多的自我保健都是以自我为中心的，无论是像洗个好澡这样的放纵还是像预约医生这样的工作。但是，当你把注意力从自己身上移开一分钟，学习如何做一些有其自身原因的事情时，会发生什么呢？开棍子，学编织， [烤点面包](https://skillet.lifehacker.com/the-perfect-time-to-get-into-bread-making-is-right-now-1837899445) 。



这个概念有个朗朗上口的名字，；你甚至可以读 [一整本书](https://www.goodreads.com/en/book/show/32509887-micromastery) 了解一下。当然，潜在的想法是永恒的。

我自己也是个技能收藏家。我从来没有真正想过为什么，直到我读了凯尔西·奥斯古(Kelsey Osgood)的一篇关于掌握微操的散文(她和一个朋友创建了一个俱乐部；他们的第一个项目是缝补衣服。)有理由相信，参与一个项目可以让你陷入冥想般的 [心流状态](https://lifehacker.com/how-finding-flow-helped-me-decide-what-i-should-do-with-1789344272) 这有助于心理健康。完成这个项目，或者掌握一些小技巧，可以增强你的自信。每一项小技能都可以轻松提升你学习下一项技能的能力。

就拿缝补衣服来说吧，奥斯古最开始做的那个。我从我妈妈那里学到了一些关于缝纫的东西，在高中的家政课上又学到了一些。每一项迷你技能都让我为下一项技能做好了准备，现在我是那种发现一架子的衣服因为拉链坏了而全都打折到 3 美元时会咯咯笑的人。(真人真事。)上周，我用一种日本缝补技术修补了儿子破旧的睡衣，这种技术叫做，我在 Instagram 上追踪了它的标签一段时间。这需要一点阅读和实验才能弄明白，但我喜欢学习它，现在这是我收集的另一个小技巧。
