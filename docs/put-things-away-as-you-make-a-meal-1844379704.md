# 做饭时把东西收拾好

> 原文：<https://lifehacker.com/put-things-away-as-you-make-a-meal-1844379704>

为自己或家人做一顿饭，往往是它本身的回报；没有什么比把你所有的技能、爱和关心放进一道菜里，然后你可以吃，看着别人享用更好的了。但是做饭也有不好的一面:总是会弄得一团糟。这就是为什么，如果你不想在吃饭的时候被食品组装过程的碎屑弄得满柜台都是，不想被即将到来的烦人任务所奚落，你应该尽最大努力在吃饭的时候清理干净。

我们已经 [推荐过在](https://lifehacker.com/clean-while-you-cook-1823544956) 之前做饭的时候清理厨房的其他地方，而 Skillet 的编辑 Claire Lower 为为什么做饭的人 [也应该洗碗](https://lifehacker.com/why-the-person-who-cooks-should-also-do-the-dishes-1788558731) 做了一个非常有说服力的案例，但是我的建议有点不同。因为与电视厨师精心制作的人工一尘不染的厨房相比，我甚至不能在不产生巨大混乱的情况下快速制作午餐。

我昨天做了简单的三明治作为午餐，并设法组装了更多的垃圾(所以我也用了炉子的一部分):三个调味瓶，一袋青菜，几罐泡菜和黄椒及其散落的盖子，几包奶酪和蔬菜熟食片，一个堆满番茄内脏的砧板。在我坐下来和孩子们一起吃饭之前，我花了几分钟时间把所有的东西都收拾好，我突然想到，如果我一用完就把所有的东西都收拾好，就会少很多麻烦(并加快我的三明治消费)。涂上蛋黄酱和芥末，然后放回冰箱。加入生菜；放回保鲜盒里。加入泡菜和辣椒，在你失去它们之前盖上盖子。合上面包，冲洗切菜板，扔掉西红柿的两端。如果你从不*制造*混乱，那感觉就不像*收拾*混乱了。另外，午饭后你可以开始洗碗。万岁。
