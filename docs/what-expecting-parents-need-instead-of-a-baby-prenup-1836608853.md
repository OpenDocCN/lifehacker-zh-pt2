# 准父母需要什么来代替“婴儿婚前协议”

> 原文：<https://lifehacker.com/what-expecting-parents-need-instead-of-a-baby-prenup-1836608853>

“婴儿婚前协议”的想法是由《亚特兰大的真实主妇》中的波莎第一次带入流行意识的，显然，它现在是最新的(哦，上帝[【momtroversy】。](https://abcnews.go.com/GMA/Family/momtroversy-baby-prenups-good-idea-expecting-parents/story?id=64086500) “这只是一对怀孕夫妇之间的书面协议，在孩子出生前列出双方对孩子的期望。

我的看法:坏名字，好主意。

称之为婚前协议毫无意义——最明显的原因是婚前协议字面上的意思是婚前协议，即婚前签订的合同。在我看来，这也定下了一个过于僵硬的基调。对于任何合作关系，你都必须留有重新评估和改变的空间。

也就是说，创造一份“婴儿婚前协议”的谈话是那些准父母们应该进行的。在分娩前的九个月里，我们经常会考虑婴儿的名字、婴儿房的装饰以及一些问题，比如“我们应该选择慢跑婴儿车还是可以折叠成小行李箱的那种？”我们忘记了(或者可能否认了)照顾新生儿的日常工作——喂食时间表、洗瓶子的工作，以及迅速淹没我们的无情任务。这可能是我们经常陷入过时的性别角色和/或感到劳动力不平衡的原因之一——根据戈特曼研究所的研究，67%的夫妇在他们宝宝生命的前三年里彼此变得非常不快乐。

因此，当你和你的伴侣都处于平静、合作的状态时，达成共识是有意义的。(阅读:当你在凌晨 4 点摇着一个尖叫的婴儿时，不是讨论谁最适合整理打嗝布的时候。)写下一个计划是明智的，知道它是有可塑性的——当你了解到你对特定宝宝的需求时，你可以重新评估事情。

你的计划应该包括什么？你可能想从的家务分工开始。谁来洗瓶子零件？谁将管理访问者的日程表？在这里，你应该讨论是否有你想承担或不想承担的职责*。*你还应该注意每隔一段时间交换一下角色(我的丈夫总是那个给我们的婴儿修剪指甲的人，今天我仍然害怕地看着那个婴儿指甲钳)。

其他可以谈论的话题可能不那么明显:你将如何给彼此“休息”(也许你可以确保每个伴侣每月至少有一个晚上或早上与朋友出去)，你将如何处理潜在的超越姻亲的行为，以及你对身体接触和性的期望(你可能来自*非常*不同的地方)。像这样的文件显然不应该被用来互相指责，但是当你被扔进一个混乱的世界时，它可以给你一些清晰的责任。记住:你们是一个团队的。
