# 希望支持您当地的餐馆吗？特殊订单可能会有帮助。

> 原文：<https://thetakeout.com/how-to-support-local-restaurants-with-special-order-pur-1843004150>

Instagram 告诉我，去年我工作的一家餐厅因冠状病毒而关闭。我的脑海里浮现出在可预见的未来，我那些失业的厨师同事们，那些没有薪水就无法生活的洗碗工，还有那些用小费付房租的酒保和服务员们。不可否认的是，我同时也在想，这家餐厅的胡萝卜鹰嘴豆泥会变成什么样，这是一种深受人们喜爱的酱料，它支撑着我度过了许多漫长的服务时光。



这不是真正的鹰嘴豆泥，虽然塔希尼参与。胡萝卜和大蒜被腌制，直到它们成为胡萝卜和大蒜的浓缩精华。我在这家餐馆工作已经一年了，但这种美味的菜肴仍然充满了我的梦想。在我的极乐世界的幻想中，载着我的船去天堂的河流是由光滑的胡萝卜鹰嘴豆泥组成的，想到它可能只存在于我的幻想中，我感到非常难过。我不认为我是唯一一个对餐馆菜单上的东西有这种感觉的人。

在新冠肺炎时代，作为一个餐馆迷，你可能会感到无能为力，你自己的天堂般的幻想，一艘烤奶酪形状的船载着你在一个小瀑布下，与启发你的三明治店一起被残酷地剥夺了。餐馆正处于岌岌可危的境地，我们希望尽我们所能帮助它们生存下去。但是答案可能不像经常叫外卖那么简单；一个人能吃多少是有限制的，而且大多数煮熟的主菜不能长时间食用。我们可以做的一件事是重新思考顾客和餐馆如何能够互惠互利，因为传统的食品服务模式已经过时了。例如，我心爱的鹰嘴豆泥一直只是大盘菜的一小部分，餐馆通常不会向提出要求的顾客出售一桶桶调味品，因为这些项目已经包含在大盘主菜的成本中。但在目前的形势下，企业可能更愿意扮演美食杂货商的角色，向你出售你想买的任何东西。菜单已经被简化和改变，商业模式也在转变，每个人都想找到增加销售额的方法。

公平的警告:不能保证特殊订单不会立即被拒绝。但是，问一问那个早餐点是否能买到打装的奶油蛋卷又有什么坏处呢？根据你所在地区的规定，餐馆也可能会卖给你一些当地饲养的鸡蛋和一磅早餐香肠。作为一个经常看到有人要求购买增值食品而不是菜单上的菜肴的厨师，我可以告诉你，接电话的人目前的压力水平是决定订单是否会履行的最重要因素。除此之外，这里还有一个剧本，教你如何支持当地企业，提出这些特殊要求，同时又不疏远餐厅员工。

仔细考虑这家餐馆。如果它的外卖业务仍然相当稳定，试着在淡季打电话，比如他们开门的第一个小时。请记住，如果餐厅是特许经营或连锁企业的一部分，员工可能没有权力心血来潮地取消菜单。如果他们说不，这不是针对个人的。

确定你想要什么。浏览当前菜单。只要你想要的商品有相关的价格，你就应该能够购买它。例如，假设你喜欢你家附近烧烤店的通心粉和奶酪配菜。在你的极乐世界的幻想中，载你去天堂的船漂浮在起伏的奶酪通心粉河上。你希望买下奶酪酱，然后在家里再造一个苹果。只要有与配菜相关的成本，餐厅就会知道酱汁的成本。为了算出菜单上的价格，食物的每一部分都要计算成本，从酱汁中的黄油、面粉、牛奶和奶酪到上面的棕色面包屑。如果一份包括两盎司的奶酪酱，他们可以将酱的成本从菜的其他部分中分离出来，并简单地将成本提高到卖给你 32 盎司。浴盆(或你选定的任何一部分尺寸)。然而，如果在主菜价格范围内提供了一些配菜，而你不能以一定的价格单独购买通心粉配菜，他们可能不知道奶酪酱的价格，当你询问时，他们可能会说不。这种逻辑适用于任何没有菜单价格的东西，比如免费提供的调味品，或者在晚餐到来之前推出的面包和黄油。

灵活一点，理解餐厅面临的限制。即使您点的是当前菜单中的某样东西，也可能不适合批量订购。餐馆通常提前几天工作，并依靠大批量。你可能希望从当地酒吧购买一些自制的切达奶酪，因为在你的天堂幻想中，你骑着一只巨大的切达奶酪去天堂。制作香肠是一项大工程，所以这家酒吧大概一周不会做超过一次。他们可能会在某一天准备好所有的配料，然后在第二天研磨、混合和填充香肠。如果香肠被熏制，这个过程甚至会延续到第三天。如果你打电话要 20 根香肠，而他们没有 20 根多余的，而且几天内也做不了更多，他们会说不。但是如果你询问购买香肠的事宜，并明确表示在特定日期不需要，他们可能会记下你的电话号码，在下次计划做香肠时给你打电话。

买足够多的东西才值得，不要指望买到便宜货。虽然看起来用这种方式购买食物要比热腾腾端上桌的花费少，但实际上餐馆的花费并没有减少。外卖订单所需的包装材料不是免费的，准备和上菜的劳动将被重新分配到包装你购买的任何东西上。预计该物品的价格将远远高于杂货店的等价物，因为它是由一个真实的人从零开始制作的。考虑到劳动力成本，请确保订购大量产品。例如，让我们说在你的极乐世界的幻想中根本没有船，而是一个辣焦糖浸泡的滑梯，直接带你进入天堂。完成一杯这种辣焦糖酱的订单需要与四杯订单一样多的劳动力，这两份订单可能都需要更多的劳动力，而不是厨师通常需要两分钟来装上餐厅菜单上的辣焦糖排骨。所以当你下单的时候，问问他们卖给你多少才合理。当你这样做的时候，询问一下预期的保质期和储存建议。也许最好在冷冻前将食物分成几份，或者干脆不冷冻？

**和气地问。**这是最重要的指导方针，对你得到你想要的东西的可能性有着最大的影响。告诉他们这件物品对你意味着什么。告诉他们你的极乐世界幻想！最后，告诉他们你想支持他们的业务。他们会明白的。
