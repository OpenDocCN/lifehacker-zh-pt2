# 质疑你的假设，做出更好的决定

> 原文：<https://lifehacker.com/make-better-decisions-by-questioning-your-assumptions-1836700026>

当你发现自己有两个看似不相容的目标、需求或欲望时，问问自己怎样才能同时实现这两个结果。

虽然你做的每一个决定都排除了另一种选择或可能性，也就是说，你花在午餐上的钱不能存入你的个人退休帐户，你在健身房花的时间也不能用来打扫厨房，但我们许多人都太快就认为某些目标自然会相互冲突。

如果我们在工作中接受更多的责任，我们将不得不花更少的时间在家里。

如果我们有了孩子，我们就不能去旅行了。

如果我们想存够钱退休，我们就买不起拿铁了。

辩证提问——自苏格拉底和柏拉图以来就存在的一种辩论方法——允许你调查自己的假设，并找出违背这些假设的情景。

正如领导力教练杰西·索斯特林解释的那样，在 [战略+商业](https://www.strategy-business.com/blog/Develop-better-leadership-habits-to-thrive-in-an-age-of-information-overload?gko=568c0) 中:

> Pham 想获得下一次晋升，但她觉得为了获得晋升而做出的工作量和生活质量的妥协是不可持续的。两种结果——要么获得晋升，要么放弃晋升以保持更好的工作生活平衡——都与她的价值观(成就、成功、家庭、健康等)一致。)而且每个人都觉得自己很重要。然而这两种结果似乎是不相容的。一个辩证的问题为两者的结合提供了空间:“我如何在以更健康的方式设计我的领导生活方式的同时，继续作为一名领导者成长并在职业生涯中进步？”Pham 用这个问题来积极地引导她的想法和行动，使结果包含两种结果都令人满意的元素。结果是有意识地选择每隔一周延长工作时间，每两周减少一次旅行。通过协商这种灵活性，她可以与朋友和家人重新联系，并恢复她曾经忽视的瑜伽练习。

请注意，Pham 的解决方案实际上包括做出排除某些选项的选择——她将在几周内工作更多时间，这意味着她无法花时间与朋友和家人在一起，但她将在其他几周内减少工作时间，并可以利用这些时间与她关心的人联系。

当然，不是所有人都能选择自己的工作时间。这并不意味着我们不能用辩证的方法来调和我们生活中出现冲突的其他方面。如果你问自己“我如何才能让我每天的拿铁咖啡*和*为我的退休基金做贡献”，用一个经典但老套的例子来说，你可能会找到另一个你能够削减支出的领域——或者如果你发现你的预算太紧张，你可能会开始问自己 [如何才能赚更多的钱](https://twocents.lifehacker.com/how-to-make-more-money-in-2019-1831844993) 。

这个方法很像著名的*高效能人士的七个习惯* : [认为双赢](https://www.franklincovey.com/the-7-habits/habit-4.html) 的第四个习惯。在这种情况下，每个人都必须发生什么才能得到他们想要的？需要做出什么选择，需要设定什么界限，需要考虑什么可能性？

所以，下次你发现自己不得不在两条看似不相容的道路之间做出选择时，开始问问自己是否有第三条你还没有考虑过的道路。怎样才能同时得到 A 和 B？你对这种情况的假设是什么，如果你假设相反的情况会发生什么？有没有办法可以调整 C，让 A 和 B 更加兼容？

一旦你知道了这些问题的答案，你就会对如何前进有更好的理解。
