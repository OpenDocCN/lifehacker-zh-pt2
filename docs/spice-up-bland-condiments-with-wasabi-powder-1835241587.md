# 用芥末粉给清淡的调味品增加味道

> 原文：<https://lifehacker.com/spice-up-bland-condiments-with-wasabi-powder-1835241587>

当谈到基于胡椒的辛辣“红色”热量时，我是一个懦夫，但我对芥末和辣根的清窦“绿色”热量有很高的阈值。除了在寿司上抹一点绿色酱，我还喜欢在身边放一罐这种粉末，用来蘸酱、调料和蛋黄酱。



当我在 Trader Joe's 看到一罐芥末蛋黄酱时，我开始痴迷于在白色奶油调味品中添加芥末粉。这是一个很好的产品，但我有一个小冰箱，在任何给定的时间都已经有两三种不同的蛋黄酱。另外，我最后总是会往里面加更多的芥末。

我不会在我的冰箱里放满芥末味的东西，这些东西永远也满足不了我的需求，我会在我的橱柜里放上芥末粉，然后在闲暇时把它加入到牧场调料、酸奶油、奶油奶酪和蛋黄酱中。(撒上爆米花和薯片也很好。)

至于你应该添加多少，这取决于你和你的耐受水平，但请记住，芥末需要几分钟才能发挥其全部潜力。开始时，每四分之一杯调味品加入四分之一茶匙，搅拌，静置五分钟，然后品尝，并根据需要进行调整，让它在每次添加之间静置。一旦你对自己的承受范围有了概念，你可以更大胆地添加，但在测试自己的极限之前，要花时间了解自己。
