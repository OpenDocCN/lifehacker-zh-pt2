# 如果您仍在使用 Windows 7，则需要此安全更新

> 原文：<https://lifehacker.com/if-youre-still-using-windows-7-you-need-this-security-1839612838>

黑客正在使用 BlueKeep 侵入 Windows 电脑，安装一个加密货币挖掘器。该漏洞只会影响运行 Windows 7、Windows Server 2008 R2 和 Windows Server 2008 的系统。

Watch

如果你运行的是这三个版本的 Windows，你应该 [从微软下载并安装一个补丁来纠正这个问题](https://portal.msrc.microsoft.com/en-US/security-guidance/advisory/CVE-2019-0708) 。补丁从 5 月份就开始发布了，所以很有可能你已经更新了。

也就是说， [ZDNet 报道](https://www.zdnet.com/article/bluekeep-attacks-are-happening-but-its-not-a-worm/) 安全研究人员刚刚发现了第一次利用该漏洞的大规模黑客活动，所以一些 Windows 用户没有收到备忘录。

当微软在 5 月份发布关于这一漏洞的警告时，它认为这可能会被用作一种自我传播的蠕虫。相反，黑客正利用它来挖掘加密货币，一些攻击可以追溯到 10 月 23 日。

虽然加密货币挖掘远不像利用漏洞攻击那样是灾难性的攻击，但你仍然应该确保你的计算机已经安装了补丁，这样你就不会容易受到攻击。
