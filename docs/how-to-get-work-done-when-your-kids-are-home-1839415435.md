# 当你的孩子在家时，如何完成工作

> 原文：<https://lifehacker.com/how-to-get-work-done-when-your-kids-are-home-1839415435>

每个人都有一个在家工作的计划，直到你的孩子流鼻涕和轻微发烧醒来。在这一点上，许多父母会告诉你，这只是为了度过一天。但是有时候，用正确的心态和策略，你可以拥有一切。



我在家工作。一个月有几次，我也在观察一个生病或在家的孩子，因为他们的老师正在培训(如果你家里有一个婴儿和你在一起， [让这一块成为你的向导](https://lifehacker.com/how-to-get-work-done-when-youre-home-with-a-baby-1823152539) )。我仍然必须完成工作，如果我的孩子不连续看三部*的《飞人》*电影，我会很高兴。虽然我在下雪天和生病的时候会看很多电视，但我也制定了一系列策略，可以在你努力工作和照看孩子的时候帮到你。

### **管理每个人的期望**

确保你诚实地说出你需要 T1 做什么，你希望 T3 做什么。然后在一天开始的时候列出你想要的和需要的清单。我做这个。它有助于管理我自己的期望，当我在晚餐前只检查了清单上的两件事时，它让我感觉更好。

我也会问孩子们他们需要做什么，他们想做什么，这样他们会觉得自己对发生的事情有发言权。然后我们谈论我不能跳过的一件事，无论是面试还是电话会议。如果他们知道我需要在一天的晚些时候打电话，这可以让他们更容易关门，并希望在时间到来时安静下来。专业提示:宣布你可能是 [une 预期加入](https://lifehacker.com/disclose-that-your-kid-is-with-you-when-you-start-a-pho-1833124685) 的电话会议。

### **找一个艺术项目**

我总是让我的孩子为我或我的妻子创作一幅画或用乐高建造一些东西。我还帮助设置，以避免争夺空间或资源。为什么房子里总是只有一支蓝色马克笔？

当我知道我有一个电话会议时，我就把炉子上的计时器打开。这给了我的孩子一些东西，当他们感觉我的电话永远打不完时，他们可以检查。关键是要留出时间，一旦电话和艺术项目结束，让他们告诉你他们所做的故事。

### **准备一个“住宿包”**

当我知道接下来的六个小时我会在机场或动物园时，我从不缺少零食或活动。用同样的方式对待你的房子，准备一个惊喜的零食、书籍或小玩具包。我选择了活动书——我小时候喜欢的那种带有隐形墨水的书——和小罐姜汁汽水，这是为孩子们的手设计的。一点点自发性对对抗幽居症大有帮助。

### 让他们做午饭

列出一些成分和界限(*允许用刀吗？*)；但除此之外，让他们决定你们一起吃什么。他们会喜欢这种自由，而且这种自由总是比他们想象的要长。告诉他们你会摆桌子洗碗。

### **举行展示和讲述**

当你为工作完成一项任务时，让你的孩子坐在一旁。他们可能知道你的职业名称，但不知道实际需要做什么。我会让他们帮我打一个字来完成一个句子。这就是为什么你在读单词“blork”

完成一项任务可能会花费你更多的时间，但是分享你们实际做的事情会很有趣。毕竟，你们是一伙的。
