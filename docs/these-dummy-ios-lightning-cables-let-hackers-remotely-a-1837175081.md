# 这些虚拟的 iOS 闪电电缆让黑客远程访问你的设备

> 原文：<https://lifehacker.com/these-dummy-ios-lightning-cables-let-hackers-remotely-a-1837175081>

硬件黑客可能不会像最近的安全漏洞 一样受到关注，但它们可能同样具有毁灭性。你可能会想象攻击者偷偷使用一个虚拟的 USB 密钥来渗透计算机，但正如 Vice 最近的一份报告 [所描述的，即使像苹果的 Lightning 电缆这样无害的技术也可以被修改来利用漏洞并控制你的设备。](https://www.vice.com/en_us/article/evj4qw/these-iphone-lightning-cables-will-hack-your-computer)

虚拟闪电电缆——以其创造者——一位被称为 [MG](https://mg.lol/blog/) 的安全研究员的名字命名为“O.MG 电缆”——在最近的 Defcon 黑客大会上流传。O.MG 电缆在其他方面看起来很普通，是用额外的硬件改装的闪电电缆，当它们插入时，黑客可以远程访问你的 PC 和 iOS 设备。

O.MG 电缆与真实的东西没有什么区别，它们甚至带有标志性的粘合环，你可以在新的苹果电缆上找到。改装的电缆也可以正常工作，让你通过 USB 为设备充电或从 iOS 设备传输文件。

无论是你的电脑还是你连接的设备都不会注意到有什么不对劲。除了解剖电缆以寻找额外的硬件，检测您正在使用 O.MG 电缆的唯一方法是当您意识到您的设备被利用后。即使你碰巧发现一个攻击者远程运行你电脑上的终端窗口，O.MG 电缆也包括一个 killswitch，它可以禁用植入的硬件，从而消除追踪攻击来源的任何可能性。

### 保护自己免受奥镁电缆(和朋友)的伤害

这里有一个好消息:这些假电缆并不普遍，即使如此，也很难从苹果商店“意外”购买一根，因为黑客需要花费一些努力才能将改装电缆植入合法包装中——至少目前如此。这一点，以及这些改进的闪电电缆都是手工精心制作的(到目前为止)。

O.MG cable 漏洞很像使用 USB 密钥或其他外部设备来远程访问 PC。这种策略比一些人想象的更常见，但你可以采取一些常识性的预防措施来保护你的数据和设备的安全:

*   不要随便捡起你发现的任何技术，即使它看起来是合法的包装
*   不要接受你不信任的人送的来路不明的充电器、USB 加密狗或类似组件作为礼物，并且*绝对*不要将它们插入你的设备。同样，只向你信任的人借充电器。
*   只从网上合法的来源购买你的电缆，或者更好的是，从你可以确保包装没有被篡改的地方购买。
*   当涉及到您已经拥有的电缆时，请将您的设备、电缆、USB 加密狗和其他组件放在附近，并在公共场合保持安全。
