# 让《哈利·波特:巫师团结》运行得更快

> 原文：<https://lifehacker.com/make-harry-potter-wizards-unite-run-faster-with-this-h-1835714647>

游戏开发商 Niantic 正式放弃 [*哈利波特:巫师团结*](https://harrypotterwizardsunite.com/) 周四下午对于 iOS 和 Android 来说，这意味着像我(也许还有你)这样的人在午休时间玩巫师，收集分散在城镇各处的魔法生物、事物和人。



如果你已经玩过这个游戏，你可能也会有一个很大的观察:它真的很慢。

我的午餐时间游戏就在游戏上线后，这意味着成千上万的想要成为巫师的人蜂拥至该应用程序注册，并走上街头，看看游戏到底是怎么回事。当很多人同时玩游戏时，会给游戏服务器带来一点压力。虽然这款游戏没有像《精灵宝可梦 Go》一样崩溃，但它的运行速度非常慢。

如果你有同样的问题，一个简单的解决方法是将游戏的资源下载到你的手机上。在你玩游戏的时候，游戏仍然需要连接到 Niantic 的服务器，但它需要传输到你的设备的数据量会少很多，从而让游戏运行得更快。

下载游戏资源的选项并不明显。要找到它，你需要深入了解游戏的设定。首先，确保你使用的是 Wifi，这样你就不会在旅途中因下载资源而遭遇巨额数据费用。

*   一旦你加载了游戏，点击屏幕底部中间的行李箱。
*   在下一个屏幕上，点击左上方的齿轮，调出设置菜单。
*   在“设置”菜单的底部，有一个“下载所有资源”的选项单击该按钮，将游戏的所有资源下载到您的手机上。

一次下载所有资源意味着当你外出时你不必下载那么多，这可能会使游戏加载速度更快。

在我下载之后，我注意到加载 Foundables 时的滞后时间少了很多,游戏似乎总体上运行得更好。当然，在我第一次启动到我下载资产的一两个小时内，使用这款应用的人数可能会减少，但为了便于讨论，我认为下载是值得的。

当你在设置菜单中时，你也可以在应用程序中调整一些其他的东西； y 你可以关闭音乐或音效，管理你的通知，以及其他事情。我个人选择关闭开场序列。当你在那里下载资产的时候，四处挖掘看看你还想定制什么，然后再回来挥舞虚拟魔杖。
