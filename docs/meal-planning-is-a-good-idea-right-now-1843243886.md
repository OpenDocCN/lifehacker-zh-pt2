# 现在计划饮食是个好主意

> 原文：<https://lifehacker.com/meal-planning-is-a-good-idea-right-now-1843243886>

“没有计划”曾经让人感觉奢侈——享乐主义、甚至——但“事情”已经发生了变化，我们现在缺乏结构的生活开始(或继续)让人感到有点压抑，或者至少是极度沉闷。简而言之，这种感觉就像没有什么可期待的，这可能会影响我们与食物的关系。

Watch

有空闲时间完善你的烹饪技巧是社交距离为数不多的令人愉快的方面之一，但这并不意味着养活自己很容易。如果你根本不工作，或者已经转移到在家工作，你通常的世俗书立已经被拆除，让你漂浮在干意大利面和豆子的海洋中。这就是饮食计划能有所帮助的地方。

### 识别你的检疫陷阱

要明白这个“计划”可以根据你的需要而严格或宽松。它可以像节日主题的夜晚一样简单——周一的烤宽面条或者周日的鸡肉——或者你可以每天为自己和孩子准备一份真正的午餐。如果一个计划不能解决任何问题，它就只是一个烦人的待办事项清单，所以首先要确定“事情的状态”是如何影响你的烹饪和饮食的。

最简单的方法就是列一个清单。例如，我的列表是这样的:

*   在思考和写了一整天关于食物的东西后，我对食物和烹饪感到厌恶
*   持续的低调焦虑和突然爆发的抑郁
*   不能去商店买最后一分钟的食材或冲动购物
*   对通过应用程序点餐感到内疚

你的清单可能会有所不同，尤其是如果你有孩子，不习惯在家工作，或者根本不习惯自己做饭。但是一旦你发现了这些问题，你就可以解决它们。我发现问题通常分为两类——后勤和情绪——饮食计划对这两方面都有帮助。

### 如果是物流问题

我一直在阅读人们有“所有这些空闲时间”——我甚至在上面写了它，它是如此根深蒂固！—但对于重要员工、在家全职工作的人或必须为子女教育提供便利的人来说，情况并非如此。如果是这样的话，传统的膳食计划方法——花一段时间烹饪你计划吃的大部分食物——可能是正确的方法。再次，尝试识别潜在的陷阱。如果你不知道从哪里开始，试着从我的 [关于这个主题的其他帖子](https://lifehacker.com/create-a-meal-plan-youll-actually-stick-to-1821880607) 中回答这些问题:

> 你喜欢吃丰盛的早餐吗？我最喜欢的《星际迷航》中的“令人捧腹的误会”之一是，当[Crusher 博士不断试图给 Picard 提供新的有趣的早餐](https://www.tor.com/2013/01/25/star-trek-the-next-generation-attached/) 时，他就像“嗯，好吧”一样，然后他们被绑架并心灵相通，结果他们都宁愿喝咖啡和吃羊角面包(Picard 自从嫁给他最好的朋友后就爱上了 Bev)。我的观点是，有些人无法忍受早上第一件事就是吃太多的早餐，吃一片烤面包和一个苹果而不是隔夜即食燕麦酸奶和腌制炖水果盘是可以的。
> 
> **你会吃剩饭吗？**如果你不会，你的伴侣或室友会吗？有些人就是不喜欢剩菜剩饭，这些人不应该一周都吃一大堆剩菜，因为这将导致一种特殊的悲伤。(我一直以为自己是剩饭剩菜，直到我离婚后才意识到是我丈夫在吃剩饭剩菜，而不是我，从那以后我已经适当地调整了我的烹饪方式。)
> 
> 你喜欢在工作日做饭，还是愿意一次做完？有些人宁愿坐着听一个关于协同作用的三小时的 Power Point 演示，也不愿在工作日的晚上切菜，但有些人觉得切菜切块相当有禅意。如果工作日晚上做饭让你感到紧张，你可能会想在周日抽出时间来做一周的全部或部分准备工作和烹饪。

举例来说，如果你知道自己讨厌做早餐，但喜欢做晚餐，那就挤出点时间煮些鸡蛋或者做一大堆 [冷冻三明治](https://skillet.lifehacker.com/make-two-weeks-worth-of-breakfasts-with-one-sheet-pan-1819690171) ，在每周(或每两个月)购物之前，挑选几个新的、令人愉快的食谱作为晚餐*。如果你讨厌剩菜，考虑做一大堆同一种蛋白质，然后储备不同的碳水化合物(米饭、玉米饼、意大利面)和酱料，这样即使你每天晚上都吃同样的东西，也不会觉得你是。*

此外，尽管目前社交媒体上展示的烹饪氛围非常“都市居家”，但不要忘记储备零食(用于)并购买一些冷冻餐(如比萨饼、墨西哥卷饼或——呃——一大包薯条)。在任何情况下，不要因为购买 [预切蔬菜或以其他方式准备的食物](https://lifehacker.com/its-okay-to-buy-prepared-foods-1834422853) 而感到内疚。即使你有“这么多空闲时间”，你也可能宁愿花时间看书或做拼图，而不是切洋葱，因为切洋葱会让人难受。你也可以买面包。购买面包仍然合法。

### 如果是心情的问题

即使事情对你个人来说“没那么糟糕”，也有很多糟糕的事情正在发生，如果这让你悲伤、焦虑、疲惫或者只是总体上脾气暴躁也没关系。拿我来说，我无法预测我每天会有什么样的感觉，以及它会如何影响我吃饭和做饭的欲望。前一分钟我还在哭，下一分钟我就在从头开始做自己的早餐香肠——真是过山车！

A.A .牛顿写了 [一个完整的指南](https://lifehacker.com/how-to-feed-yourself-when-youre-really-depressed-1823560127) 告诉你如何在抑郁的时候喂自己，它可能对你有帮助，尤其是如果你以前从未经历过急性抑郁。饭菜根据容易程度被分成几类——从“比淋浴容易”到“比淋浴难”——特色是味噌汤(我目前最喜欢的)、营养丰富的烤蔬菜和非常容易的烘焙食品。

如果你通常喜欢烹饪，但却很难在这项活动中找到乐趣，给自己买一个新的厨房玩具，并围绕它来计划你的饮食。我很少用我的立式搅拌机烘烤，但我买了绞肉机附件，男孩，我很高兴能自己制作香肠和汉堡混合物。如果你有一段时间没有使用某个特定的设备了，掸掉它的灰尘，看看它是否能激发你的灵感。

这也有助于产生外部影响，比如虚拟食谱俱乐部，作为一种饮食计划，只要确保你没有过度扩张自己。记住，现在不是“优化”或“充分利用”任何东西的时候。养活自己和家人其实完全够了。
