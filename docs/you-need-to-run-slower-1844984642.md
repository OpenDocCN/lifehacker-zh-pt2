# 你需要跑慢一点

> 原文：<https://lifehacker.com/you-need-to-run-slower-1844984642>

如果跑步很难，这是给你的——特别是如果你是一个跑步新手，或者如果你已经跑了一段时间，但你似乎不能跑得更快。你需要学习的最重要的事情是:你跑得越多越快，你跑得越慢越多。



每个人都开始得太快。你并不孤单。这就是曾经的我:每次出门都要跑三英里，总是希望打破上次的记录。从来没有真正打败过它。

当你开始时，你有两种速度。跑步快；走路很慢。所以一旦你一直在“跑”(比如说，你已经达到了你的沙发到 5K 程序 的 [的终点)，你就一直在跑*快*。](https://lifehacker.com/all-the-questions-youll-have-when-you-start-couch-to-5k-1830857969)

如果是这样的话，你做得很好，你已经走得很远了。现在你还有一个任务:*他妈的慢下来*。

### 正常跑步应该是什么感觉？

正常的训练跑应该感觉*轻松*。是的，你可能想成为一个跑得快的人，但是训练跑和比赛是有区别的。

训练跑步轻松、缓慢、悠闲；赛跑是快速、艰苦、费力、困难、痛苦的。比赛对你的身体来说很难，只有在你准备好的情况下才能进行。不是每天。

我把这种混乱归咎于 5K 沙发。“5K”是比赛距离。如果你跑得最快的朋友说他们跑了 25 分钟 5 公里，那就是他们训练并达到顶峰的比赛。他们没有以那样的速度进行日常跑步，即使是 5 公里(3.1 英里)的距离。可能他们的三英里训练跑是 30 分钟或更长。

在 reddit 上我经常看到有人问如何跑得更远。他们说他们一直在尝试，但他们无法越过一定的距离(比如两三英里)。如果这是你的问题，你肯定是跑得太快了。你不应该以这么快就让你筋疲力尽的速度前进。在 30 分钟轻松跑结束时，你应该停下来，因为时间到了，而不是因为你的身体不行了。

### 等等，跑步训练从来都不是很快？

偶尔你可能会在训练中跑一会儿，但那不是你的谋生之道。

看看任何有经验的跑步者的训练计划。可能会有一个快速工作日(交替进行短时间的快速间歇和一些轻松的恢复)或一个节奏跑(以稳定的、有点累人的速度跑步)，但其他一切都意味着轻松的里程。这意味着至少 80%到 90%的跑步速度应该像散步一样悠闲。对于初学者级别的程序，这个配速可能是你 100%的里程数。

轻松的跑步正在增强你的体质。他们 [不必努力做好自己的工作](https://lifehacker.com/exercise-shouldnt-feel-horrible-1844763622) 。慢跑很容易做，也很容易让你的身体恢复。你不应该在每次去健身房的时候都竭尽全力，也不应该在每次系鞋带的时候都去跑步。

### 好吧，多慢才算够慢？

有几种方法可以衡量这一点。经典的是:跑步的时候可以随便聊几句吗？如果你不能说出完整的句子，你可能跑得太快了。

你担心跟不上你的训练伙伴吗？不断地选择走在你前面的人，加快步伐，以便你能赶上他们？太快了。你不想要一个你必须强迫自己保持的速度。你想要一种感觉你可以永远保持下去的速度。

你可以在筋疲力尽前保持大约一个小时的配速就是你的*节奏配速*，你需要把它留到节奏训练中。这意味着轻松跑的配速是你可以无限期保持的。

你能四处看看，享受阳光或风景吗？很好。

你是否呼吸沉重，期待着结束或者下一次休息时间？太快了。

超能力者会超过你吗？信不信由你，这是个好兆头。意思是你已经设法慢下来了。

### 但是我跑不了那么慢！

有两件事我想对你说。

首先，是的，你可以。我推荐一个小练习来解决这个问题。你将交替中速跑一分钟，然后慢速跑一分钟。(不用看表，你可以在直道上中速跑，在弯道上慢速跑。或者使用城市街区或电线杆。任何间隔相对较近的东西都可以。)

随着时间的推移，你会意识到，即使当你以“正常”速度运行时， [的速度也比](https://lifehacker.com/run-smarter-not-faster-how-to-find-your-gears-for-b-1706084781) 的速度慢。可能会感觉慢得令人抓狂。记住，如果一个步行者超过你，*那是你开始明白的信号*。如果你找不到合适的配速，试着保持同样的节奏，但缩短步幅。一步步来。一步一步来。

第二，如果你尝试了上面的方法，仍然不能跑得足够慢，那么根据需要进行散步休息。将你的整体努力水平保持在可以继续对话的水平。当你发现自己喘不过气来时，走路。忘记你的速度，努力保持轻松。

### 你正在学习成为一名更好的跑步者

几乎每个人都会经历这个过程。我记得有一天，在一个沙发到 5 公里风格的计划中，我决定看看我是否可以放慢一点，是否可以跳过步行休息。这很有效，但在那之后的*年*我才知道如何以足够慢的速度完成所有的跑步，以至于我不觉得自己快死了。我在一系列的短跑/步行间歇跑完了我的第一次 10K 比赛。但还是算了！我还是跑完了全程。

你跑得越慢，你就会越快发现轻松跑和快速跑的区别，你就会越快在增加里程和变得更快方面取得进展。这确实是一个违反直觉的过程，但它是值得的。在轻松的日子里，你真的不能跑得太慢。坚持下去，很快你就能跑得更快。
