# 下飞机的正确礼仪是什么？

> 原文:[https://life hacker . com/what ' s-the-correct-礼仪下飞机-1838911167](https://lifehacker.com/whats-the-correct-etiquette-for-getting-off-a-plane-1838911167)

回到 2018 年 8 月，边疆航空公司 航班上的两名乘客在下飞机时相互威胁，最终演变成一场争吵；据目击者称，一名乘客在试图从头顶的行李箱中拿包时撞到了另一名乘客，引发了打斗。

我们的社交媒体编辑蒂姆(Tim)最近在着陆后试图拿他的包时遇到了类似的(尽管不那么敌对)分歧。蒂姆没有在那里等着，而是决定在乘客被允许站立时，径直走到他那排座位后面的行李箱前——另一名乘客公开表达了对他行为的不满。

“我把事情掌握在自己手中，我会再做一次，”蒂姆冷冷地说。

读者们，蒂姆是个怪物吗？《华盛顿邮报》 最近报道了下飞机礼仪以及谁有权先到达登机口的话题。假设你乘坐的航班前面只有一个出口，下飞机的基本规则不应该让人感到意外:乘客在前，后面跟着乘客。首先是靠过道的座位，然后是中间和靠窗的座位，这不用说。(前面的乘客总是优先于后面的乘客，即使他们在过道里。)制定这些规则的主要原因是为了让的乘客能够以最少的摩擦下机，并确保航班之间的平稳过渡。

然而，对于这些规则，我有几个例外。首先，如果你花了很长时间才解开安全带，其他人在你前面是完全没问题的。没有人应该为你服务，如果你挣扎着从座位上站起来，这可能会减轻你的压力。

如果你有一个快速连接，我也会让你通过，假设你让其他乘客知道你的请求；是的，你会遇到一名乘客，他会对这个要求发牢骚，但在大多数情况下，我们都有过冲向登机口的经历。(邮报建议让服务员提前知道这一点，这样他们可以为你提供登机口信息，或者在着陆时给你找一个靠前的座位。)

蒂姆的情况很独特，他提出了强有力的理由；毕竟，如果他没有抢先抓住他的包，他将不得不最后一个下飞机。不幸的是，这并不被认为是最佳实践。正如我们之前所写的，头顶上的隔间空间是先到先得的商品；如果你最后一个登机，你就倒霉了。。如果你登机晚了，不得不把包放在身后，对不起，蒂姆，你就惨了。

另一方面，他确实有几个解决问题的备选方案。第一个答案应该是一些礼貌的推搡，同时告诉他的乘客需要拿他的包。如果包离他的座位足够近，他也可以请一位乘客帮助他。(诚然，他这样做不会交到朋友，但其他人可能会更有同情心。)

另一个更容易的选择是等到他看到机会去拿他的包，而他后面的人都被拖住了。当然，如果你的包在身后那么远，你能做的也就那么多了，但这或许值得一试。

因此，我们向我们的读者提问:当乘客在飞机着陆时从座位上跳起来时，你会感到恼火吗？下飞机的正确方法是什么？蒂姆是个无情的反社会者吗？
