# SEP 个人退休帐户限额在 2019 年增加

> 原文：<https://lifehacker.com/the-sep-ira-limit-is-increasing-in-2019-1830310964>

对于那些有闲钱的人来说，2019 年 SEP 个人退休帐户的缴费限额将增加 1000 美元，从 55，000 美元增加到 56，000 美元。



SEP IRAs 是面向小企业主和员工(包括自由职业者)的传统退休账户。捐款是免税的，而且，和传统的个人退休帐户一样，你的投资是延税的，直到你提取资金，在这一点上，它们被作为收入征税。

对于 SEP 个人退休帐户，雇主缴纳的费用不能超过以下两项中的较低者:

*   薪酬的 25%
*   2019 年 56000 美元

这就是它们对自由职业者如此有吸引力的原因:如果你可以使用它，你可以比传统的个人退休帐户节省和投资更多的钱，这可以让你在退休后更有优势(更不用说它更简单，因此得名——建立一个 SEP)。

不利方面:50 岁以上的人没有额外缴款，如果你为一个小雇主工作，你就要依赖他们为你缴款。

solo 401(k)s 的缴费限额只能由没有员工的企业主开设，2019 年的缴费限额也提高了 1，000 美元至 56，000 美元。这些账户还允许 50 岁以上的企业主追加 6000 美元的捐款(当然，数学上 [比](https://www.nerdwallet.com/blog/investing/what-is-a-solo-401k/) 更复杂)。
