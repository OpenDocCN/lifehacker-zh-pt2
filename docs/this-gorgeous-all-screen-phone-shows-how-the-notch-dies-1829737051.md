# 这款华丽的全屏手机展示了缺口是如何消失的

> 原文：<https://gizmodo.com/this-gorgeous-all-screen-phone-shows-how-the-notch-dies-1829737051>

当你看到苹果、三星和谷歌最新手机的设计时，很难区分今年的产品和 2017 年的过时产品。与此同时，除了三大巨头，其他所有手机制造商似乎都满足于跳上有缺口的屏幕潮流，满足于类似的迭代美学。就在 2018 年似乎将成为手机真正无聊的一年时，Oppo 推出了 Find X，这款设备同时显示了智能手机的创造力并没有死亡，并埋葬了没有办法避免那些愚蠢的屏幕剪切的想法。



从上到下，Find X 都是优雅简约的典范。它的锥形玻璃侧面看起来和感觉都很棒，而沿着显示器外部延伸的薄黑色轮廓突出了所有完美的屏幕空间，就像一幅更好的画作上的一个漂亮的框架。即使在背面，Oppo 也努力通过隐藏在烟雾玻璃下的柔和色彩来实现微妙的精致。Find X 对书呆子更有吸引力的唯一标志是一个细长的月牙形塑料片，它通常位于摄像头所在的位置。

所有这些事情一起工作，以确保没有什么会分散你对 Find X 屏幕的注意力，这正是你想要的。Find X 采用生动的 AMOLED 面板，分辨率为 2340 x 1080，覆盖 6.42 英寸，提供了迄今为止智能手机上最具沉浸感的观看体验之一。唯一的小缺点是，Find X 的峰值亮度为 425 尼特，比你在新的三星 Galaxy 或 iPhone 上获得的亮度低 20%至 30%。当在室内或阴凉处时，这不是一个问题，但如果你在室外花很多时间盯着你的手机，那就不理想了。

要让真正的烟花绽放，你需要做的只是解锁手机或打开相机应用程序，让 Find X 的电动相机像邦德恶棍藏身处的秘密入口一样弹出。隐藏式摄像头不仅履行了传统的拍照职责，因为 Find X 没有板载指纹传感器，它还可以用作面部扫描仪，这样你就不必一天输入 50 次 pin 码。

现在，有些人可能会对 Find X 的摄像头从隐藏的洞中升起的半秒钟左右感到恼火。但是，当我每次需要使用手机的摄像头时，Find X 会让我脸上露出笑容，因为它的内部有一个奇妙的技术噪音，听起来就像是过去台式机上的电动 CD 托盘。有时候，真正让你开心的是一些小事。

不幸的是，尽管有这些令人讨厌的优点，相当不错的电池寿命不到 12 小时，规格包括骁龙 845，8GB 内存和 128GB 存储空间(以及即将推出的一款拥有 [世界上第一个 10GB 内存](https://www.xda-developers.com/oppo-find-x-10gb-ram/) )的型号，Find X 仍然有许多缺陷。它在美国没有正式销售，所以取决于你在哪里买到，手机可能没有预装谷歌 Play 商店这样的重要东西，这意味着你需要自己下载 APK 的文件。

Oppo 的 Color OS 也有点落后于时代，因为它基于 Android 8.1 而不是 Android 9。但更令人沮丧的是，手机附带了一个充满膨胀软件的完整文件夹，默认情况下，它的通知不会显示任何有用的内容，所以你需要逐个定制每个应用程序的首选项。此外，你需要花 925 美元左右去第三方经销商那里买一台，Find X 缺乏无线充电、microSD 卡插槽和任何防水功能都是明显的疏漏。

还记得我之前说过的在户外很难看到屏幕吗？好吧，Find X 的问题甚至不止于此，因为我觉得对于那些适度冒险的人来说，这款手机可能很难通过。许多人不仅证明了 Find X 的机箱比你的典型手机更容易弯曲，它也不是很智能，因为如果你不小心试图在它在你的口袋里时解锁，机动相机会弹出来，并在放弃之前沮丧地挣扎几秒钟。

当相机打开时，你也可以轻轻摇动手机，听到弹出滑块在外壳中发出轻微的卡嗒声。即使有一些情况下 Find X 也能正常工作，但我怀疑它们中的任何一个是否能够在手机掉落到相机上的情况下提供足够的保护。所有这些事情对 Find X 的长期耐用性来说都不是好兆头，所以希望它们能在未来的迭代中得到完善和改进。

Find X 相机的图像质量也不错。这款手机在解锁时识别我的脸从来没有任何问题，但尽管在前面提供了一个巨大的 25-MP 摄像头，在后面提供了 16-MP 和 20-MP 摄像头，整体图像质量肯定低于这个价格范围内的其他设备。

在与 Galaxy Note 9 和 S9+的正面交锋中，Find X 总是落后几步。虽然我不会说 Find X 的照片令人失望，但它们始终缺乏竞争对手拍摄的照片所具备的那种额外的清晰度和色彩饱和度。

但即便如此，我还是想澄清一点:尽管存在缺陷，Find X 仍是一款迷人的设备。Oppo 本可以稳扎稳打，生产出可能更实用或更实惠的产品，但它没有。相反，Oppo 抓住了一个机会，在这一年里，当人们围绕 notches 和 bezels 展开了无数次互联网大战时，Find X 做了自己的事情。它实现了全屏梦想。现在，Oppo 有了一些创新和独特的东西来建立。即使你考虑到三星可能在年底前推出可弯曲屏幕手机，毫无疑问，Oppo Find 是今年最重要的智能手机之一。
