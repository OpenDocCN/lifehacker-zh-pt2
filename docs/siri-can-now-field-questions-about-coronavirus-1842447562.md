# Siri 现在可以回答关于冠状病毒的问题

> 原文：<https://lifehacker.com/siri-can-now-field-questions-about-coronavirus-1842447562>

如果你对冠状病毒有疑问，现在你有另一个来源可以得到答案:Siri。

Watch

苹果本周末更新了 Siri，加入了一些关于冠状病毒的筛选问题。虽然这绝对不能代替与医生交谈，但询问虚拟助理“我有冠状病毒吗？”会提示助手带您完成一系列关于您症状的问题。根据你的回答，助理可能会建议你联系医疗专业人士，或者告诉你监测你的症状。

Siri 还会为你指出远程医疗应用，这些应用可以帮助你从家中连接到医疗服务提供商。它提供的所有信息都来自疾控中心。

如果你是安卓用户，谷歌 [也有一个新冠肺炎门户](http://google.com/covid-19) ，里面有关于病毒的信息以及预防技巧和本地资源。Google Assistant 也可以提供这些信息，尽管直接访问 hub 要高效得多。它还通过一个真实创建的网站 提供湾区人民的 [测试信息。该网站预计将来会推广到其他地区。](https://www.projectbaseline.com/study/covid-19/)
