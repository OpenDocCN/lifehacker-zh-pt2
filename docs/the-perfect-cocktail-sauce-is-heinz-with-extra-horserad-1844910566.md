# 完美的鸡尾酒酱是亨氏加额外的辣根

> 原文：<https://lifehacker.com/the-perfect-cocktail-sauce-is-heinz-with-extra-horserad-1844910566>

在吃(和做)了近三十年的虾鸡尾酒后，我得出了一个不可避免的结论:真的很难做得比亨氏鸡尾酒酱更好，尤其是如果你额外添加辣根的话。我尝试过更好、更贵的鸡尾酒酱品牌，我自己做了，但我总是手里拿着象征性的帽子回到亨氏。(男朋友这个周末犯了买海狸牌鸡尾酒酱的错误，我现在还有点恼火！)

Watch

这是有道理的。鸡尾酒酱主要是番茄酱，亨氏制作的番茄酱最好，所以他们的鸡尾酒酱达到了甜味和扑鼻味的完美平衡也就不足为奇了。另外，“自制”鸡尾酒酱并不是真正的自制，对吧？就算是自己做“自己的”鸡尾酒酱，也得从番茄酱说起，而我们已经知道 [自制番茄酱烂](https://skillet.lifehacker.com/nobody-wants-your-homemade-ketchup-1827140540) 。(你问最好吃的番茄酱是什么？亨氏。是*亨氏*。)

所以，除非你打算自己做番茄酱来制作鸡尾酒酱——我不建议你这么做——那就买亨氏吧，根据你的喜好来调配。尽管我很喜欢亨氏鸡尾酒酱，但我承认它需要额外的辣根。(实际上，我从来没有遇到过对我来说够辣味的鸡尾酒酱，这包括那些声称根部很重的酱。)

因此，无论你买的是哪种鸡尾酒酱，在冰箱里放一小罐调制好的辣根酱都是必要的。如果你想知道多少辣根是“足够的”辣根，这很容易:你想添加足够的辣根，这样鸡尾酒酱的颜色就会变浅一整个色度。你不想要红酱——不，不——你想让那东西变成深粉色。然后，只有到那时，它才准备好给你的小虾吃。
