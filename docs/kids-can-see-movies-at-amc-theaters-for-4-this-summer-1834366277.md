# 今年夏天，孩子们可以花 4 美元在 AMC 影院看电影

> 原文：<https://lifehacker.com/kids-can-see-movies-at-amc-theaters-for-4-this-summer-1834366277>

从现在起到学校开学，AMC 将提供一部适合儿童观看的电影、爆米花、一杯饮料和一包“Footi Tootis ”,每个孩子 4 美元，外加税。该协议仅在周三有效，是 AMC“夏季电影营”推广活动的一部分， [福克斯 11 台](http://www.foxla.com/news/kids-can-get-a-movie-ticket-drink-and-popcorn-for-4-this-summer-at-amc-theatres) 报道。

Watch

“夏季电影营”在全国各地的许多不同的影院举行，是白天专门为儿童放映的电影。预计今年夏天上映的电影包括《彼得兔》、《如何训练你的龙:隐藏的世界》、《乐高电影 2:第二部》等。

您可以通过 [访问 AMC 的网站](https://www.amctheatres.com/summer-movie-camp#participating-theatres) 或直接联系影院，了解您当地的 AMC 是否参与促销活动。

像大多数打折商品一样，你应该预料到这些商品的门票会卖光，所以如果你感兴趣，就早点买。
