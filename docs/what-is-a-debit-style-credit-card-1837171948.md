# 什么是借记卡？

> 原文：<https://lifehacker.com/what-is-a-debit-style-credit-card-1837171948>

有没有希望有一种方法来建立信用，而没有无时不在的入不敷出的诱惑？也许你寻求用信用卡购物的安全，但是喜欢能够看到你的借记卡上可用于购物的实际现金余额。

一种被称为借记卡的新型信用卡可以提供两者的优点。

借记式信用卡要求使用附带的支票账户。信用卡的信用额度将反映你在支票账户中的金额，防止你的消费超过你可以立即偿还的金额。与此同时，该卡还提供额外服务，比如建立你的信用记录，让你获得奖励。

把它看作是对一些传统的管理支出方法的更新。

*   [**有担保的信用卡**](https://twocents.lifehacker.com/how-to-choose-a-secured-credit-card-1836196338) **:** 需要存款才能将你的信用额度设定到那个数额。
*   签帐卡:要求你每个月支付全部余额。一些美国运通卡仍然以这种方式工作。
*   **优化信用卡支付:**有些人每个账单周期支付一次，有些人则选择每次购物时都用信用卡支付。少数创业公司寻求自动化高频支付，但现在不存在了——Debx 在 2018 年被竞争对手 [收购，Debitize](https://twocents.lifehacker.com/debitize-connects-your-credit-card-to-your-bank-account-1803144679) 收购，然后 Debitize 在 2019 年被 Trim 收购。
*   [**预付借记卡**](https://lifehacker.com/use-a-prepaid-debit-card-to-curb-impulse-sending-1588277919) **:** 你决定在借记卡上放多少现金，然后根据需要再充值。

借记卡还没有广泛使用。网上银行应用程序 [Zero](https://zero.app/) 最近推出了 Zerocard，允许你开启或关闭借记功能。当它打开时，你的余额每月会自动从你的支票账户中支付。当它关闭时，你可以选择花比你的支票账户更多的钱，你的零卡就像一张典型的信用卡。

这项特殊服务的一个大缺点是:要获得零卡，你需要下载零应用程序，该程序只能在 iOS 上使用。如果你不支付全部余额，你将面临 24.99%的可变利率。

如果你已经对自己的信用使用感到自信，借记卡可能对你没有吸引力。但是，如果你刚刚开始建立和使用信用卡，一张与你的支票账户绑定的信用卡可能会很有价值。虽然以这种方式工作的卡还不常见，但可以肯定的是，银行业的颠覆者随时都会跳上混合动力车。
