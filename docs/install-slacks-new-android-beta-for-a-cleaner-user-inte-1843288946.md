如果可以的话，我会尽量不在手机上懈怠——通知会变得有点多——但如果我这样做了，我肯定会在 Android 上这样做。我还会确保我注册了 Slack 的 Android 应用程序的测试版，因为 Slack 正在测试的新 UI 增强功能使应用程序更容易导航。

这是那种“眼见为实”的事情，我稍后会向您展示其中的变化。一般来说，Slack 在应用程序的底部添加了一个小小的导航栏，让你可以快速访问 Slack 频道、DMs、提及和“你”部分的列表——在那里你可以启用“请勿打扰”/“离开”模式，设置状态，编辑你的个人资料，等等。

将这些元素统一在一个导航行中比点击 Slack 的其他小图标要容易得多，包括其应用程序常规版本左上角的三格汉堡。测试版是一种优雅、更简单的体验，我在使用它的前五分钟内就已经爱上了它。

### 如何刷入新的松弛时间

除了底部可爱的导航栏之外，Slack 还对应用程序的滑动进行了一些修改。在一个频道内向右滑动，你会像往常一样调出你的频道列表。再滑动一次，你会看到这个新屏幕，而不是你的工作区的直接列表:

下一个有点混乱。如果你在任何不是频道的主要导航元素中——主页、DMs、提及或你——向左滑动会带你回到你最后看的频道。如果你已经在一个频道中，向左滑动没有任何作用。

### 如何加入 Slack 测试版

如果你在安卓系统上，只需打开 Google Play，搜索，然后寻找“加入测试版”链接:

一旦加入，给它一分钟左右的时间，刷新 Google Play 上的 Slack 列表。你将看到一个可用的应用程序更新。安装它，强制关闭设备上的 Slack(如果它是打开的)，然后重新打开它，享受全新的(测试版)用户界面。
