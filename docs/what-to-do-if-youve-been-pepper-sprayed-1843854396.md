# 如果你被胡椒喷雾了该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-youve-been-pepper-sprayed-1843854396>

胡椒喷雾(包括品牌名称 Mace)会导致眼睛和鼻子极度刺痛和灼烧感。如果你被喷了胡椒喷雾，你需要把化学物质冲出来。水是有用的，但不是唯一的选择。

### 胡椒喷雾里有什么？

胡椒喷雾之所以这样叫是因为它含有辣椒素，这种化学物质使辣椒尝起来很辣。如果你曾经在切完墨西哥辣椒后揉过眼睛，你就会明白了。它还会刺激身体的其他部位，包括口腔、鼻子和外阴的粘膜(在去洗手间和擦拭之前，一定要洗手*。*

辣椒素是油性的，你可能记得化学课上(或厨房经验中)说过，辣椒素在其他油或乳化混合物(如肥皂或牛奶)中溶解最好。肥皂是去除皮肤中辣椒素的最佳选择，但是你需要对你的眼睛更细致一点。

### 怎么把胡椒喷雾从眼睛里弄出来？

摘下隐形眼镜——最好是用干净的手——如果你戴着的话。不要揉眼睛。(如果可以的话，不要摩擦或触摸任何东西。)反复眨眼。

[和催泪瓦斯](https://vitals.lifehacker.com/how-to-flush-your-eyes-if-youre-tear-gassed-1843826826?rev=1591105729128) 一样，用水冲洗眼睛是让胡椒喷雾喷出来的一种有效且安全的方法。(有一个朋友帮助你是最理想的，因为你可能看不清东西。)尽管水不会与这种化学物质结合，但水流仍能以物理方式将辣椒素从眼睛中排出。为此，确保直接喷眼睛，而不是脸的其他部位——如果你的额头上有胡椒喷雾，你不会想把它*送进*你的眼睛。

除了水，你也可以用盐水或一点无泪婴儿洗发水混合在水中(提前在喷雾或喷水瓶中混合)。

不太推荐的是牛奶和 50/50 的抗酸剂混合物，如 Maalox 和水。目前还不清楚这些是否真的比上面的选项更好，一般来说，最好避免将任何不必要的东西放入眼中。如果你使用抗酸剂，确保它是不加香料的。特别是薄荷味，会进一步刺激眼睛。

在寒冷的天气或没有水的地方，用浸过植物油的布或纸巾会有帮助；用它擦眼睛。前线健康，一个在抗议活动中提供医疗支持的组织，在这里有一个更 [完整的处理胡椒喷雾的指南](https://www.frontlinewellness.org/guidelines-for-treating-pepperspraychemical-irritants) 。

同样值得注意的是，梅斯制造了他们自己品牌的去污喷雾，尽管他们没有说里面有什么。 [这里可以买到](https://www.mace.com/products/takedown-relief-mk-iii) 。他们说这“大大减少了胡椒喷雾消毒的时间。”

### 从你的皮肤和衣服上取下胡椒喷雾

如果你触摸你的脸或衣服的另一部分，然后触摸你的眼睛，你会再次经历烧灼感。

肥皂是去除皮肤上胡椒喷雾残留物的最好方法。可以考虑携带洗洁精和纸巾擦拭皮肤。你也要清洗你的衣服，并计划清洗或丢弃任何其他可能被喷洒过的物品。小心脱掉你的衣服，如果你不在家，把你的衣服和其他被污染的物品放在塑料袋里，防止它们把化学物质传播给其他人或物体。

和催泪瓦斯一样，胡椒喷雾在短期内会让人感到疼痛，但从长期来看基本上是无害的。然而，它会导致一些人的并发症，包括哮喘患者的呼吸问题。如果有人呼吸困难，寻求医疗帮助。
