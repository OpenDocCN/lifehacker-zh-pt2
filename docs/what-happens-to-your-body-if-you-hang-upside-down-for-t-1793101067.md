# 如果你倒挂太久，你的身体会怎么样

> 原文：<https://lifehacker.com/what-happens-to-your-body-if-you-hang-upside-down-for-t-1793101067>

如果你想知道倒挂太久是否会致命，答案是肯定的。如果你不及时直立起来，你的身体会发生什么。

当你倒立时，你一翻身，血液就开始涌向你的头部。最终，这会导致血管破裂或引发脑出血。但这还不是最大的问题。大多数并发症是由窒息或缺氧引起的。

当你倒立时，你的器官会给你的肺增加额外的重量，使呼吸更加困难。心脏问题也可能出现，因为你有更多的血液流向心脏，这意味着你的心脏比平时泵得更努力。悬空太久也对你的眼睛有害。当你头朝下时，你眼睛里的压力会加倍，这会导致视力问题。

当然，你必须倒挂几个小时或更长时间才会发生这些事情，所以没有必要让孩子们完全避开单杠。但是如果你正在 [挂一些圣诞灯](https://lifehacker.com/the-most-common-holiday-injuries-and-how-to-avoid-them-1790272397) ，去爬山或者在高处做一些工作，确保有人在附近以防万一。
