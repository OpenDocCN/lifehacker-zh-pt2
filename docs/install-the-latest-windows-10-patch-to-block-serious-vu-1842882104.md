### 如何安装 2020 年 4 月的 Windows 安全更新

该更新应该对每个人都可用，如果您启用了自动更新，有些人可能已经安装了它。但是如果你还没有更新，或者不确定如何更新，这里有安装补丁的方法:

1.  打开 Windows 开始菜单。
2.  点击齿轮状的**设置**图标。
3.  转到**更新和安全>“Windows Update”**查看补丁。(你可能会看到一个黄色的**"！"**当您打开设置菜单时的通知(如果更新已经挂起)。
4.  点击**“下载”**安装补丁。
5.  等待它完成，然后根据提示重新启动您的电脑。
