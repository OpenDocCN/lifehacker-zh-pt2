# 雷雨来临前拔掉电子设备的电源

> 原文：<https://lifehacker.com/unplug-your-electronics-before-a-thunderstorm-1832986570>

拐角处的坏天气？你可能不得不忍受一个没有电子干扰的雨天——也就是说，如果你还想在第二天打开电视的话。



今天，CNET 提出了一个我们很多人在雷暴中问自己的问题:当暴风雨来临的时候，我应该拔掉 PS4、Xbox 和其他电子设备的电源插头，还是冒着把它们都烤焦的危险？

嗯，事实证明这是一个非常真实的风险，你可能不得不重新考虑你的雨天活动。为什么，到底是？嗯，如果闪电击中电杆，它将释放出一股能量，烧毁其路径上的一切。

正如 [CNET](https://www.cnet.com/how-to/unplug-during-a-thunderstorm-fact-or-myth/) 提到的，在北美，任何插在墙上的东西都被设计成使用 129 伏电压，最高可达 169 伏。电涌意味着数百万伏的电压——所以可以肯定地说，你的平板电视或冰箱再也见不到阳光了。

但是这回避了另一个重要的问题:这种情况实际上多久发生一次？关于风暴造成的电力激增的数量，这里没有大量的研究(尽管在 2014 年，一场雷暴在 [芝加哥](https://www.chicagotribune.com/suburbs/northbrook/news/ct-nbs-comed-power-surges-tl-0110-story.html) 的一个地区爆发了大范围的电力激增，影响了大约 244 名居民，并永久性地损坏了炉子、空调插座和它后面的炉子——这在 2018 年再次发生在*同一个*社区)。

同时，我们知道雷暴是非常频繁的。根据美国国家强风暴实验室的数据，美国一年大约有 100，000 次雷暴，而在任何特定时刻，世界上有 2，000 次雷暴正在发生。甚至连国土安全部的都建议你在暴风雨来临时拔掉不必要的电子设备。

所以，当你在天气预报中听到一场雷雨正扬起它丑陋的头时，你该怎么办呢？首先，在雷雨前拔掉电子设备的插头，而不是在雷雨期间(否则，你可能会直接经历电涌)。 [国家气象局](https://www.weather.gov/media/owlie/ttl6-10.pdf) 甚至建议避免在暴风雨中洗澡或淋浴，这样你就不会冒着接触到水的风险。

你不必 [拔掉](https://www.weather.gov/media/grr/brochures/nwsthunderstorms&lightning.pdf) 所有的电子设备，但是。你可以也应该让任何有助于你获取天气信息的电子设备开着(当然，一台没有电源插头的笔记本电脑也可以)。但是，请远离任何插在上面的电缆或电线。

至于电涌保护器，正如所指出的，它们并不能承受暴风雨的电压——所以拔掉你的 Xbox 仍然是你最好的选择，这样你就可以改天再玩堡垒之夜。
