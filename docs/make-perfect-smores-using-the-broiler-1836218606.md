# 用烤鸡做出完美的三明治

> 原文：<https://lifehacker.com/make-perfect-smores-using-the-broiler-1836218606>

在夏天，人们总是试图让其他人在外面吃饭，或者“户外”，如果你愿意的话。虽然我不喜欢在玫瑰园吃塔可钟(我昨晚就是这么做的！)，我确实认为有些“户外食物”在室内准备时，即使不更好，也一样好。其中一种食物是 s'mores。

作为一个前卫的思想家，我一直认为斯莫尔比夏天更重要，应该也可以在常规的周二享用。我也一直认为在室内烹饪更好，因为室内没有虫子和灰尘——这是我在烹饪和吃饭时尽量避免的两件事。但是，即使那些令人毛骨悚然的爬虫和泥土不是问题，你的室内厨房仍然有一个主要优势:烤鸡。

在营火上做饭既有趣又浪漫，但它缺乏细微差别。不受控制的火焰使得控制棉花糖的烘烤度变得有点困难，并且它使得融化巧克力——你不能用棍子串起来——变得几乎不可能(尽管把一个放有一块巧克力的全麦饼干放在*靠近*的地方，火可以帮助融化它)。

在烘烤器的作用下，你的棉花糖以可控的均匀速度变成褐色，而你的巧克力正在融化。烤鸡也烤全麦饼干的边缘，我认为这是一个很好的接触。为了最大限度地增加烤面包和饼干的覆盖面，我建议在把棉花糖粘在饼干上之前，把它纵向切成两半。

两个字的告诫:时刻保持警惕。根据你的烤鸡设置的高度，以及你的烤饼离加热元件的距离，烤焦和融化会非常快，通常在几分钟之内。作为一个控制狂，我只是徘徊在烤箱前，当它们达到刚刚过去的金黄色时，就把它们拉出来，但你做到了。

### 简单的室内 S'mores(四个，但根据需要扩大)

**配料:**

*   4 块全麦饼干
*   1 块好时巧克力，分成 4 块，每块 3 段
*   4 块棉花糖

**指令:**

将每个饼干掰成两半，将每个棉花糖垂直切成两半。将棉花糖放在四块饼干上，在剩下的一半上面放上巧克力。把每个人都放在一个薄烤盘上，然后放在离加热元件几个槽的架子上。你希望它很近，但不要太近。关上烤箱，打开烤箱，观察并等待。当软糖烤到你喜欢的程度，巧克力融化时，把锅拿出来，把它们夹在一起，然后享用。你甚至可以带他们出去，如果你喜欢的话。
