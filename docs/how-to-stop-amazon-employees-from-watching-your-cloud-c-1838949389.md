# 如何阻止亚马逊员工观看你的云摄像头录像

> 原文：<https://lifehacker.com/how-to-stop-amazon-employees-from-watching-your-cloud-c-1838949389>

最近来自彭博 的一份 [报告解释说，亚马逊在印度和罗马尼亚的员工使用云摄像头记录来“训练”设备的人工智能，使其能够更准确地从日常活动中识别安全威胁——例如，有人打碎窗户。虽然可能令人毛骨悚然，但也不是闻所未闻；对于我们强加给智能设备和人工智能的所有存在性恐惧，它们仍然需要人类的输入来“学习”。](https://www.bloomberg.com/news/articles/2019-10-10/is-amazon-watching-you-cloud-cam-footage-reviewed-by-humans)

然而，真正的问题是这些录音首先是如何获得的，以及它们是否被安全地存储，事实证明亚马逊在这两方面都做得不好。去想想。

### 有些事情不合情理

亚马逊声称，除了你自己，任何人都无法看到云摄像头的视频，除非你出于故障排除的目的发送一段视频记录。此外，在任何产品描述页面或用户协议中，都没有说亚马逊员工可以访问你的云摄像头记录。

然而，正如匿名亚马逊员工告诉彭博的那样，亚马逊没有向员工解释剪辑是如何选择的，根据他们的描述，该公司似乎不太可能只是使用“自愿提交”的镜头。不出所料，录音通常包含大多数人希望保密的信息或主题——包括性、电话交谈和个人身份信息。

但是，即使我们假定亚马逊是无辜的，假设所有被观看的视频都是用户自愿上传的，这也是可悲的错误处理。虽然报告称亚马逊禁止员工在观看视频的场所使用智能手机，但彭博的消息人士反驳了这一说法，甚至声称视频在员工之间共享，并经常上传到互联网上。

然而，事情是这样的:尽管彭博的报告给消费者敲响了警钟，但并不令人惊讶。我这么说不是为了告诫云摄像头用户或减轻亚马逊的罪责——该公司显然没有对其客户完全诚实——但这里有一个模式。我们知道亚马逊使用录音来训练其 AI 产品，包括 Alexa 设备和 [环形安全摄像头](https://lifehacker.com/protect-your-privacy-from-your-own-cloud-security-camer-1831684103) ，无论用户是否意识到这一点；可能凸轮只是最新的例子。

### 如何让你的亚马逊 AI 产品更安全

公平地说，亚马逊并不是唯一一家因错误处理用户记录而有罪的公司。然而，由于用户抗议和政府怀疑，其他公司谷歌和苹果已经停止了对其人工智能驱动产品的类似“训练”方法，并扩大了隐私控制。亚马逊已经投降，让用户对他们的 Alexa 语音记录有更多的控制权，但该公司经常被曝光在处理用户隐私方面表现不佳。但是，有一些方法可以减轻潜在的安全问题:

*   正如我们对 [环型摄像机用户](https://lifehacker.com/protect-your-privacy-from-your-own-cloud-security-camer-1831684103) 的建议，永远把你的云摄像头安装在你不介意被别人看到的地方。避开像卧室和浴室这样的地方，除非，不管出于什么原因，你必须有一个安全摄像头。
*   如果可能，白天在家时禁用设备。否则，当你不想录下什么的时候，就把这些吸盘粘起来。
*   如果你不得不发送一个剪辑供审查，对你发送的内容要有选择性，并始终假设有人会浏览这些片段。最明智的做法是干脆不要发送任何视频。
*   如果你有一个 Alexa 设备: [更新你的安全设置](https://lifehacker.com/prevent-amazon-from-eavesdropping-on-your-alexa-convers-1833974108) ，选择不让 [亚马逊](https://lifehacker.com/how-to-prevent-amazon-from-reviewing-your-alexa-convos-1836938142) 访问你的数据，并定期 [删除你的 Alexa 语音历史](https://lifehacker.com/you-can-now-tell-alexa-to-delete-your-conversations-1835097576) 。

### 其他选项

如果你不愿意对你的设备和录音进行微观管理，或者你担心不断有报道称隐私协议乏善可陈，那就不要使用亚马逊的产品。

正如我前面所说，谷歌和苹果已经停止了他们自己的一些基于人类的人工智能训练计划。这绝不是对任何一家公司的全力支持——[两家公司都有自己的隐私问题](https://lifehacker.com/how-to-delete-voice-recordings-with-alexa-google-assis-1836977240)——但在这方面，它们似乎更容易接受用户反馈，并对你的数据提供比亚马逊更多的控制。 [就这一点而言，谷歌的 Nest 产品是一个不错的选择](https://lifehacker.com/how-to-prepare-for-the-changes-coming-to-your-nest-acco-1834828917) (尽管就一般隐私而言，我仍然建议尽可能选择苹果而不是谷歌)。

对于 DIY 的观众，我们也有一些关于设置你自己的 [自制视频安全系统](https://lifehacker.com/build-your-own-pan-and-tilt-security-camera-with-a-rasp-1784708079) 的提示——只要确保你的 [遵循法律要求](https://lifehacker.com/make-sure-youre-recording-people-with-your-home-securit-1787040532) 如果你走这条路。
