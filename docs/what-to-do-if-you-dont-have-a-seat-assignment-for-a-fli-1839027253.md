# 如果没有航班的座位分配，该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-you-dont-have-a-seat-assignment-for-a-fli-1839027253>

当你买机票的时候，航空公司经常会故意忽略掉所有你需要承担的其他费用，比如一顿蹩脚的盒饭的价格，不稳定的无线网络，以及经常出现的实际座位分配。



在网上办理捷蓝航空公司航班的登机手续时，我们的总编辑弗吉尼亚最近面临着一个两难的问题:她是否应该为一个座位分配支付 90 美元，仅仅是为了保证自己有一个靠窗的座位；否则，她会冒险前往机场，依靠空姐的突发奇想来抢到一个喜欢的座位。

“这个骗局是新的吗？!"她问 Lifehacker 的工作人员。从我们的一些回复来看，这并不是航空公司独有的经历——我也曾处于这种不安的境地，面临着购买过道座位的选择，以缓解我对被中间座位拖累或不知何故被完全排除在航班之外的恐惧。

为什么会这样？好吧，显而易见的答案是，航空公司很可能理解这种恐惧，并利用这一点从你的机票上赚点外快。(出于更实际的原因，航空公司可能还会将 [部分的](https://lifehacker.com/how-to-reserve-blocked-seats-on-airplane-flights-1832791620) 座位隔开，为那些拥有精英身份的乘客、机组人员或有特殊需求的乘客腾出空间，比如家人和孩子，这样他们就不会被分开。)

老实说，作为乘客，你的选择是有限的；是的，你*可以*购买升舱，并保证给自己一个特定的座位。比如说，如果你想绝对确定自己会坐在靠过道的座位上，你可能会从中受益。或者，如果你只是想有一点额外的腿部空间，并有能力支付溢价。

但如果你愿意放弃绝对保证的靠过道或靠窗的座位，那就不要让步。为什么？嗯，在这种情况下，你可能不是唯一的乘客。如果你等着接受任务，你可能只会得到一个过道或窗口。(如果给你一个中间的座位，你可以*然后*利用这个机会，如果你愿意，可以购买升级。)

根据航空公司的不同，如果您刚刚购买了一个航班，并且没有座位分配，通常情况下，您应该等到航班起飞前 24 小时办理登机手续的窗口打开时(并且被锁定的座位变得可用)。通常，航空公司会在这个时候自动给你分配一个座位——这很可能是一个靠窗或靠过道的座位——让你不必为一个高级座位付出额外的钱。

如果你在值机开始后没有座位分配，那就等到你到达机场。我在捷蓝航空公司的航班上遇到过这种情况，我在航空公司柜台得到了一个靠过道的座位。(一位捷蓝航空的客服代表向我证实，即使你没有提前预订，也能保证你在航班上有一个座位。)

如果你想知道你是否能得到一个靠走廊的座位，考虑一下你出发和到达的城市，以及你什么时候出发；如果你飞往或离开一个受欢迎的旅游目的地，你被困在中间座位的几率可能会更高。如果你的旅行日期在工作日，你可能会比在繁忙的旅行日比如周日旅行更幸运。

如果你有特殊需要，试着打电话联系你的航空公司，告诉他们你需要特殊住宿；根据你的具体需求，他们可能会尽力为你找到最好的座位。(在从东京出发的 14 小时飞行前，我食物中毒了；我联系了航空公司通知他们，虽然他们当时不能给我安排座位，但我非常方便地找到了洗手间旁边的过道座位——这可能只是纯粹的运气。)

并且永远永远下载你的 [航空公司的 app](https://lifehacker.com/always-download-and-use-your-airlines-app-when-your-tra-1838810884) 。你也许可以用这个应用抢到一个更好的座位，而不必在门口排很长的队。
