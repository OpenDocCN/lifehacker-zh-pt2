# 现在有一个全国性的数据库，里面有可信的被指控虐待的牧师

> 原文：<https://lifehacker.com/theres-now-a-nationwide-database-of-priests-credibly-ac-1841363810>

一个名为 [可信指控](https://projects.propublica.org/credibly-accused/) 的 ProPublica 项目提供了一个可搜索的天主教牧师数据库，顾名思义，这些牧师已经被确定为涉嫌虐待者。并非每个教堂或教区都公布了名单，但很多都公布了，数据库中有超过 6000 个名字。



ProPublica 的总裁 [在时事通讯](http://go.propublica.org/nsu) 中写道，有力量将来自许多地方的列表结合起来。一些神职人员被悄悄地从一个地方转移到另一个地方，但在这里你可以按名字搜索，看到来自全国各地的人。

该数据库包括 178 个教区(教会团体)和秩序，共有 6470 万成员。另外 41 个教区和秩序，服务于 900 万天主教徒，没有公布名单。在 [工具的首页](https://projects.propublica.org/credibly-accused/) 可以看到哪些组有和没有发布名单。

我从小去天主教堂，我从一份 [2018 年宾夕法尼亚大陪审团报告](https://www.nytimes.com/2018/08/14/us/catholic-church-sex-abuse-pennsylvania.html) 中得知，我小时候认识的几名牧师被可信地指控犯有性虐待罪。我查了 ProPublica 的数据库；那些名字在里面，尽管教堂和教区的名字没有出现。不过，它们被正确地归属于匹兹堡教区。

“看到施虐者的名字被公布，对幸存者的影响是深远的，”BishopAccountability.org 的安妮·巴雷特·道尔在新闻简报中说。“它给出了验证。这立即将他的羞耻和自责的负担转移到了犯罪者身上，这是应该的。”如果有一个名字在你童年的时候就引起了你的注意，也许可以考虑查一下。
