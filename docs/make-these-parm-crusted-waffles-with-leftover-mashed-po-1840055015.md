# 用剩下的土豆泥做这些烤华夫饼

> 原文：<https://lifehacker.com/make-these-parm-crusted-waffles-with-leftover-mashed-po-1840055015>

我不知道你能不能看出来，但是我最近买了一个华夫饼干机，并且一直在测试华夫饼干的极限。(火腿事实证明，华夫饼干做得很好。)我已经计划将所有的 [剩余馅料](https://lifehacker.com/turn-leftover-stuffing-into-crispy-cheesy-waffles-1839968945) 烤成华夫饼，土豆泥似乎是合理的下一步。





我知道你要说什么——“吃剩的土豆泥？？那些是什么？?"我求你不要说出来。如果你为感恩节做了适量的食物(也就是说，太多了)，你会吃剩下的东西，包括土豆泥。那些土豆应该是鸡蛋饼，而且要加大量的奶酪。parm 在松软的土豆周围形成一层外壳，帮助华夫饼保持形状。味道也特别好。要做帕尔马干酪土豆泥华夫饼，你需要:

*   土豆泥
*   磨碎的帕尔马干酪

将四分之一杯土豆泥舀成一个小球，然后轻轻压平，形成一个三英寸的圆盘。将圆盘的两面都压入奶酪中，根据需要重复，将小圆盘放在一边。

将不沾饼干的烙铁加热到第二高的温度。你可能需要稍微摆弄一下这个，但是我把我的 Cuisinart 华夫饼干制作机设置为 4(满分 5)，这是完美的。一旦华夫饼干机被加热，把一个土豆圆盘放在中间，关闭华夫饼干机。

不要动它，直到你的华夫饼干机指示它已经做好了。我不知道你的华夫饼机是怎么工作的，但是我的灯从红色变成绿色。与填充华夫饼不同，你不能在烹饪过程中检查这些部分，因为土豆泥会太软，外壳不会完全成型，华夫饼会撕成两半。华夫饼干做好后，轻轻地从制作机中取出(筷子可以帮助你进入凹槽)，立即食用。这些非常适合搭配蔓越莓酱，我想你也留下了。
