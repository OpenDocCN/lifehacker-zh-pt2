# 如何删除你所有的脸书旧帖子

> 原文：<https://lifehacker.com/use-these-tools-to-delete-all-of-your-old-facebook-post-1843887925>

脸书有一个全新的删除旧帖子的工具，但它不是很好。它不仅在脸书的桌面网站上不可用——因为*推出了*——而且即使是这样，现在也有更好的工具可以用来删除你在脸书的旧帖子，而不用删除脸书本身。



在我们开始之前，先说一句:我不同意脸书关于允许重要人物提供内容的立场，否则外行人会被踢出这项服务，但我仍然使用脸书与我的朋友和当地社区保持联系。我不会评判任何想离开服务的人，但我也不会推迟帮助那些留下来的人。

### 脸书的“管理活动”是如何运作的

如果你可以通过脸书的 iOS 或 Android 版本访问该功能，或者脸书 Lite，在那里它可能现在适用于 [iOS](https://apps.apple.com/tr/app/facebook-lite/id1393838612) 或[Android](https://play.google.com/store/apps/details?id=com.facebook.lite&hl=en_US)——你会在你的活动日志中找到它。

调出你的个人资料，点击蓝色“添加故事”框旁边的三个点，然后点击活动日志。在那里，点击“管理活动”，然后点击弹出的“你的帖子”选项。现在，您将进入此屏幕，在这里您可以点击想要存档或删除的单个帖子:

当你在脸书上存档内容时，只有你知道它还在继续——你和脸书。你的朋友看不到它。垃圾选项应该是显而易见的，但是也有一个小问题；任何你扔掉的东西都会被扔进一个数字回收站，为你的帖子开始 30 天倒计时。一旦时间到了，帖子就永远消失了。如果你改变了主意，只要把你的帖子扔进垃圾箱，在你可以的时候取消删除。

由于这个工具上没有“全选”选项，如果你打算清理你的帐户，请做好大量点击的准备。我打赌你会在清空你的帖子之前放弃，所以你最好现在就不要费心了。有更好的解决办法。

### 脸书的网站给你一个更好的删除旧帖子的工具

如果你使用的是脸书的桌面版本，我很喜欢它的“管理文章”工具，你可以在你的个人资料的“文章”部分的右边找到它:

点击它，你会得到一个类似于你的脸书应用程序的窗口，允许你按日期过滤帖子，选择一大组帖子(或全部)，然后删除它们。后一个选项只有在你点击“下一个”按钮时才会出现，前提是你已经过滤了只有你发表的帖子。如果不小心拉进了别人的帖子，选项会变灰。

你不能用这种方法存档旧文章，所以只能删除或者什么都不做。然而，如果你不想让其他人看到它，你真的认为你会那么在乎你 10 年前发布的东西吗？

### 如果你真的想删除年复一年的帖子

您可能需要比这些工具所能提供的更多的火力。考虑到你可能在脸书发了多少帖子，以及你可能想删除多少，手动点击或点击帖子是浪费一个下午或一年时间的好方法。

相反，尝试一个类似 [**社会图书管理员**](https://lifehacker.com/social-book-post-manager-is-a-must-have-for-pruning-you-1823952493) 的 Chrome 扩展。自动删除旧帖子需要一些时间，但这比试图手动删除大量旧内容要好得多。

否则，你可以随时 [选择核选项](https://lifehacker.com/how-to-delete-your-facebook-account-a-checklist-1829436824) :删除你的账户，重新注册服务。如果是这样，确保你先下载了所有的数据，并列出你回来后想重新联系的所有朋友或团体，这样你就不会错过任何人。
