# 用“拇指数字”技巧教孩子写数字

> 原文：<https://lifehacker.com/teach-kids-to-write-numbers-with-the-thumb-numbers-tr-1844340879>

对一些孩子来说，学习如何写数字是一个真正的挑战。有这么多东西要学！各种奇怪的形状！人们可以看到这是多么令人沮丧。但是一个 [职业治疗助理](https://www.facebook.com/kevindelores.koster/posts/1416642591855683) 可能已经想出了一个帮助那些孩子的方法。她称之为“拇指数字”法。

Watch [https://lifehacker.com/embed/inset/iframe?id=fbpost-https%3A%2F%2Fwww.facebook.com%2Fkevindelores.koster%2Fposts%2F1416642591855683&autosize=1](https://lifehacker.com/embed/inset/iframe?id=fbpost-https%3A%2F%2Fwww.facebook.com%2Fkevindelores.koster%2Fposts%2F1416642591855683&autosize=1) 

不过，这种特殊的方法只对右撇子有效，所以她也对左撇子孩子有一些建议。他们可以翻转右手，掌心向上，用左手写数字。如果这太不舒服，她还建议让他们练习只用食指在沙子或面粉之类的东西上描绘数字，以去除精细运动技能成分，感受数字的形成。

如果你点击她的照片，你还会看到她用同样的技巧用一把尺子，尺子上画着一只伸出手指的手——[，就像这个](https://funandfunction.com/one-finger-spacer-25-pack.html) 。它通常用于帮助孩子们在写句子或练习写单词时学会在单词之间留足够的空间，但它也可以兼作拇指数字指南，供右撇子*或*左撇子使用。
