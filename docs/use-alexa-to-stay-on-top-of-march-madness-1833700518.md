# 使用 Alexa 保持在三月疯狂的顶部

> 原文：<https://lifehacker.com/use-alexa-to-stay-on-top-of-march-madness-1833700518>

2019 年 NCAA 男子锦标赛周日继续进行。周日，德克萨斯理工大学和弗吉尼亚大学赢得了四强席位，今天奥本将对阵肯塔基州，杜克将对阵密歇根州立大学，以确定同样的结果。



如果你错过了最近几场比赛，或者你需要温习一些琐事，亚马逊已经为其个人助理加载了锦标赛的具体信息。与 Alexas 的其他一些技巧不同，该功能可以开箱即用，无需启用特殊技能即可使用。

这里最好的办法是问 Alexa 你需要知道什么，然后看看会发生什么。" Alexa，疯狂三月是怎么回事？"会给你一个当前三月疯狂状况的简要概述。你也可以问比赛在哪里进行，最后四场比赛的时间表是什么，以及当天比赛的比分是多少。

Alexa 非常了解球队和比赛的历史，知道如何解决与朋友之间的比赛时间冲突，如果你想对自己的表现感觉更好，你可以询问她自己的观点。亚历山大让冈萨加赢得了一切。

她也有统计信息，所以她可以告诉你不同比赛之间的赔率是多少，并且加载了实时更新，所以你可以大喊“Alexa，杜克比赛的比分是多少？”当你打扫客厅的时候，即使你没有开电视，你也能发现。

最后，如果你只是需要在你的球队输了之后缓解情绪，她有几个篮球笑话可以帮助缓解痛苦。
