# 这些 Chrome 扩展将清理你泛滥的 Gmail 收件箱

> 原文：<https://lifehacker.com/these-chrome-extensions-will-declutter-your-overflowing-1842416460>

泛滥的电子邮件永无止境。即使有充足的隔离时间，整理你的 Gmail 帐户并到达零收件箱感觉像是一个遥不可及的幻想，但它不需要成为一个:有许多优秀的(而且大部分是免费的！)第三方浏览器扩展将帮助您筛选全天候的雪崩。

***注意**:如果你在桌面上看这个，点击下面的箭头来浏览幻灯片。如果你在手机上，就继续滚动吧！*

Gmail 会将你没完没了的电子邮件转储到一个没完没了的列表中，它简朴的界面会让清理你凌乱的收件箱感觉力不从心。

一个名为 [**Sortd**](https://www.sortd.com/) 的 Chrome 扩展解决了这个问题。它把你的收件箱变成了一个拖放式的工作空间，你的未读邮件在一端，列表栏在另一端。您可以随意自定义这些列。例如，您可以为需要立即回复的电子邮件创建一个栏，为可以等到周末的邮件创建另一个栏。

一旦你配置好了你的公告板，就可以很容易地把收到的邮件移到合适的列表中，并根据需要处理它们，而不是一条一条地筛选你的收件箱。视觉概览提供了一个更好的视角，可以看到哪些电子邮件可以等待，哪些是时间敏感的。

Sortd 还为你的 Gmail 带来了许多方便的任务管理工具，包括一个日程安排工具，可以让你安排电子邮件并记录它们的截止日期。在收件箱的顶部，它甚至显示了当天和接下来一周到期的电子邮件数量。你也可以在邮件中附上个人笔记，并在你撰写回复时参考。

Sortd 的基本功能是免费的，但如果你想获得高级功能，如电子邮件跟踪和创建无限列表的能力，你必须支付 6 美元的月费。

你可以把 Chrome 扩展[](https://www.dragapp.com/)**看成是 Sortd 的删减版。它把你的 Gmail 收件箱变成了一个 [看板](https://lifehacker.com/productivity-101-how-to-use-personal-kanban-to-visuali-1687948640)——基本上是更多的组织栏目。让 Drag 区别于 Sortd 的是，它有一个不那么吓人的设计，并且就在 Gmail 里面，而不是把你重定向到一个不熟悉的新页面。**

**您可以随心所欲地个性化您的拖板，甚至为单独的项目创建多个拖板。当您查看电子邮件时，您可以使用顶部标签的快速访问行将其移动到特定的栏或板。像 Sortd 一样，你也可以给个人邮件添加笔记和任务。**

**只要你是唯一使用它的人，拖动就是免费的。对于团体来说，无限用户的月费是 99 美元，小团队的用户是 15 美元。**

 **如果你不喜欢看板，但仍然觉得在你的收件箱中安装一个任务管理工具很有趣，那么[**active inbox**](https://www.activeinboxhq.com)很适合你——如果你也不介意每月支付少量费用的话。

ActiveInbox 允许您根据优先级处理电子邮件。它会在你的收件箱上方固定一个新窗口，列出你的邮件，按照你指定的截止日期排序。一旦您建立了当天的任务，您可以使用 ActiveInbox 来激活“请勿打扰”模式，该模式会“阻止”您接收新电子邮件提醒。单独的焦点模式隐藏了 Gmail 界面中所有不必要的元素，比如侧边栏。

既然你生活在暂时的数字隔离中，你就不会被源源不断的新邮件分散注意力，并且能够首先专注于你的优先事项列表。此外，ActiveInbox 提供了专门的选项卡，用于记笔记和添加子任务。

ActiveInbox 是一款高级应用，但你可以免费试用 14 天。在那之后，你每个月至少要支付 4.16 美元——仅仅相当于一杯咖啡的现金来容纳你那满满的收件箱。

这里有一个漂亮的小 Chrome 扩展，可以让你在收件箱里阅读几行邮件，而不是一行一行地点击。使用 [**Gmail 邮件预览**](https://www.gmail-message-preview.com/) ，你可以一次查看多达三行的每封电子邮件——这很有用，尽管这会让你的收件箱感觉很糟糕。如果你觉得大胆，你甚至可以直接在收件箱里显示每封邮件的全部内容。

这个小小的调整最终会为你节省大量时间，因为对于大多数电子邮件来说，一两行文字就足以表明你是需要存档、回复还是扔进垃圾桶。这个扩展可以免费使用，所以尽情发挥吧。

一旦你设法给你的 Gmail 带来一些秩序，你的下一步应该是确保你的收件箱不再是垃圾的垃圾场。为此，你需要 [**燃烧器电子邮件**](https://burnermail.io/) 。

这个超级有用的扩展允许你快速生成一个一次性的电子邮件地址，只要你必须提交一封电子邮件来注册一些东西。换句话说，只有傻瓜才使用他们真实的电子邮件地址——或者那些喜欢垃圾邮件和邮件列表的人。

您可以选择将各种一次性电子邮件帐户的更新转发到您的实际电子邮件地址。当你不再需要它们时，你只需点击一个按钮就可以退订你的假邮件。一次性电子邮件的免费版本可以让你生成五个 BS 地址，并为你的冒名顶替者邮箱提供七天的历史记录。
