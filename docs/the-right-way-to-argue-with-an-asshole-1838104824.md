# 和混蛋争论的正确方式

> 原文：<https://lifehacker.com/the-right-way-to-argue-with-an-asshole-1838104824>

赢得与混蛋争论的最好方法就是完全忽略他们。但这并不总是可能的。有时你需要说服某人停止争论并离开，或者你需要缓和与你需要共处的人的争论。有时候，你需要在一个非常糟糕的想法流行起来之前就把它扼杀掉。律师 ryanmeer 说，当你真的要和一个混蛋争论时，你必须像争论一个混蛋 T4 T5 那样去争论 T2。



在 Tumblr 上一篇最近成为热门推特帖子 [的帖子](https://twitter.com/kerplunkthemunk/status/1171481487192907780?s=21) 中，米尔解释说，当有人恶意争论时，你无法通过善意的争论击败他们。他们不是来权衡两个对立的观点的，他们是来因为你不同意他们而对你大喊大叫的。你不能和玩绳球的人下棋。

米尔说，要打败一个混蛋，你必须放弃任何让他们相信你是对的希望。“人类有一件很酷很可怕的事情，”他说:“*确定性与事实*无关。当人们确定无疑时，他们就成了混蛋。”给他们事实只会给他们当混蛋的材料。取而代之的是你必须让那些混蛋按照他们自己的规则输。你必须不公平。

米尔列出了几个混蛋战术，你需要在他们对你使用这些战术时或之前对他们使用:

*   让他们定义每一个术语，并挑出他们的定义。
*   不管他们问你什么问题，让他们先回答，否则他们就是懦夫。
*   责备他们太情绪化。
*   如果他们试图开一个玩笑或使用一个基于令人憎恶的观点的委婉语，假装你不明白，直到他们把它说出来。
*   嘲笑他们。
*   让他们离开。

在米尔的博客 上阅读这些战术 [的更全面的解释。正如米尔指出的，这种方法对混蛋很有用，因为这是混蛋的战术手册。不要向没有要求的人发泄。](https://cornichaun.tumblr.com/post/185990171607/how-to-argue-like-an-asshole)
