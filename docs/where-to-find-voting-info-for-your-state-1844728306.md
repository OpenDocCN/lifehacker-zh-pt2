# 在哪里可以找到您所在州的投票信息

> 原文：<https://lifehacker.com/where-to-find-voting-info-for-your-state-1844728306>

2020 年大选即将到来，虽然 11 月 3 日是一个很容易记住的日期，但如果你打算提前投票或通过邮件投票，选举日可能会提前几周。由于每个州的投票规则和截止日期都有所不同，现在是时候制定一个计划了。



你可以在政府网站上搜索或浏览一页又一页，找出你需要做什么和什么时候做——或者你可以利用一个全面的、按州分类的资源，它已经把你需要知道的一切都汇集在一个地方。

DAME 的伙计们整理了一份在各州 投票的优秀指南，包括以下信息:

*   选民登记截止日期
*   选民身份要求
*   提前投票日期
*   邮寄和缺席投票信息，包括要求投票的截止日期
*   投票站位置
*   轮询工作应用程序
*   投票权
*   选举官员的联系信息

只需向下滚动到您所在的州并打开。页面上已经列出了关键日期，但对于其他信息，您将被重定向到您选择的州的网站上的相关页面。请注意，有些资源可能不适用于整个州，例如，犹他州的链接投票工作者应用程序就特定于犹他州县。(犹他州实行电子邮件投票制，所以亲自投票还是有限的。)

你可以在“投票拯救美国”网站找到另一个有用的选民信息指南 。向下滚动到“了解您所在的州”部分，从下拉菜单中选择您所在的州，查看关键日期并访问关于提前投票和邮寄投票的常见问题。

你也可以 [检查你是否登记投票](https://votesaveamerica.com/be-a-voter/)——尽管你可能也想直接确认你的州选民名册，特别是当成为选民工具告诉你你没有在你当前的地址登记的时候。

一旦您在日历上标记了重要的选举日期和截止日期，使用我们的指南 [注册投票](https://lifehacker.com/how-to-register-to-vote-1843476040)[通过邮件投票](https://lifehacker.com/how-to-vote-by-mail-1843745080) 和 [在疫情](https://lifehacker.com/how-to-safely-vote-in-person-this-year-1844701684) 期间亲自(安全地)投票，以确保您的投票被计入。
