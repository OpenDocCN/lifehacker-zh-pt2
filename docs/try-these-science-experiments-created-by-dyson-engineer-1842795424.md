# 试试这些由戴森工程师创造的科学实验

> 原文：<https://lifehacker.com/try-these-science-experiments-created-by-dyson-engineer-1842795424>

戴森的工程师想知道: [你的孩子准备好迎接挑战了吗](https://www.jamesdysonfoundation.com/resources/challenge-cards.html) ？嗯，也许他们以前不是，但他们现在已经回家一个月了，他们已经无聊透顶了。现在是尝试新事物的好时机。



詹姆斯·戴森基金会设计了 [44 项科学与工程挑战](https://www.jamesdysonfoundation.com/content/dam/pdf/US%20challenge%20cards%20with%20cover.pdf?)——可能更为人所知的是科学实验——孩子们可以在家里做。它们包括许多你已经在厨房或房子周围拥有的原料或工具。挑战包括创建:

*   气球烤肉串
*   水下火山
*   瓶子里的龙卷风
*   气象气球
*   灭火器
*   跳舞的葡萄干
*   熔岩灯
*   隐显墨水
*   纸板船
*   水钟

如果你或你的孩子和我一样，你可能会喜欢看工程师们 自己做一些实验，这样你就不用工作了。 [机械工程师创造的意大利面桥](https://youtu.be/Pp9U6lyoIqg) ，有人吗？

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-Pp9U6lyoIqg&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-Pp9U6lyoIqg&start=0) 

还可以观看工程师们搭建 [硬纸板椅子](https://youtu.be/FuAqA_GhVQ0)[潜望镜](https://youtu.be/ra1HmKMXmGY)[气球车](https://youtu.be/2x-ixR7E3xA) 等等。

除了每个挑战易于遵循的指示和插图，基金会还在 [可下载的 PDF](https://www.jamesdysonfoundation.com/content/dam/pdf/US%20challenge%20cards%20with%20cover.pdf?) 中提供了一个清单，这样孩子们可以在完成每个挑战时勾选方框，并跟踪他们完成了多少。
