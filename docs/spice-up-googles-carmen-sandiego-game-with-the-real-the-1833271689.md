# 用真正的主题曲给谷歌的卡门·桑迪戈游戏增添趣味

> 原文：<https://lifehacker.com/spice-up-googles-carmen-sandiego-game-with-the-real-the-1833271689>

我心里一直有一个老派的“寓教于乐”游戏卡门·桑迪戈，因为小时候它们教了我一些像样的地理知识。我也喜欢这部很棒的电视节目(有很棒的主题曲 )，尽管我很确定我做了噩梦，梦见自己出现在节目中，并且在大结局中被一个难以置信的困难的大陆卡住，就像 [非洲](https://youtu.be/GsSCNjbNy4I?t=1423) 。

Watch

如果你是新网飞版《卡门·桑迪戈》系列的粉丝——很棒的动画，但仍然错过了那首朗朗上口的 Rockapella riff——你现在可以试着自己抓住小偷中的小偷大师。通过 [谷歌地球](https://earth.google.com/web/data=CiQSIhIgYmU3N2ZmYzU0MTc1MTFlOGFlOGZkMzdkYTU5MmE0MmE) 在网络或服务的 [安卓](https://play.google.com/store/apps/details?id=com.google.earth) 或 [iOS](https://play.google.com/store/apps/details?id=com.google.earth) 应用程序上启动谷歌与霍顿·米夫林·哈科特合作开发的第一款游戏 [。](https://www.blog.google/products/earth/your-mission-gumshoe-catch-carmen-sandiego-google-earth/)

(如果你用的是后两种，你会想在应用程序底部寻找小卡门·桑迪戈图标。)

虽然谷歌的游戏很有趣，并通过询问证人和环游世界向旧的教育娱乐游戏致敬，但这并不是我希望拥有的完美的卡门·桑迪戈体验。例如，没有任何声音效果，所以当你在新的地点拉夏洛克时，你不会听到你的鞋子 的*啪嗒啪嗒啪嗒* [。](https://youtu.be/vpSTIWL7LG4?t=278)

为了补救这一点，我建议 [打开这个网站](http://carmensong.ytmnd.com/) ，让它在你做你的侦查工作时在后台播放。你会感觉好一点的，我保证。

我喜欢这个游戏如何让你在真实的世界中缩放，而不是教育标题的原始艺术，它应该代表你正在参观的各种地标。卫星图像提供了一个很好的，真实的近似你正在看的东西和它有多大，以及附加的背景，向你显示地标在你所访问的任何城市中的实际位置。如果你不太擅长国际旅行，那么意识到一些你从小就崇拜的目的地就在星巴克(或 [必胜客](https://www.nytimes.com/2018/05/30/travel/world-famous-destinations-depictions.html) )旁边总是很有趣的。

如果你仍然没有被新的卡门·桑迪戈体验所打动，不要担心。你仍然可以通过 [点击 archive.org](https://archive.org/details/softwarelibrary_msdos?and%5B%5D=carmen&sin=)并查看任何几十年前的卡门·桑迪戈作品:*卡门·桑迪戈在哪里*，*在美国的过去卡门·桑迪戈在哪里*，*在时间上卡门·桑迪戈在哪里*，*在迪斯尼乐园卡门·桑迪戈在哪里，因为我想要一条救济鞭，我厌倦了等待*等等。
