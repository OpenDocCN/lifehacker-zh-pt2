# 通过删除这些段落来改进每篇文章

> 原文:[https://life hacker . com/improve-every-essay-by-deleting-these-paragraphs-1837747554](https://lifehacker.com/improve-every-essay-by-deleting-these-paragraphs-1837747554)

“一开始就是这个词。道与神同在，道就是神。”如果你在写弥赛亚的传记，这是一个很好的开头。对于一篇大学入学申请论文，或者一篇关于生三个孩子的快乐的中型文章，或者公司时事通讯上的一个观点专栏来说，就不那么好了。当你在写一篇文章的时候，以几个热身段落开始是可以的。但是记得删掉。

### 修改你的介绍

盖伊继续她的思路，提供更多的论文写作建议，但现在，让我们尝试一个清理任何论文的具体策略:删掉第一段，可能是最后三段。

有时你会在第一段试图证明你的作品的存在。如果这个作品是好的，它证明了自己的存在。杀了它。

有时候，第一段是你试图将你的特定主题联系起来的地方。在这种情况下，您可以保留它，但将其下移几段。以一件具体的事情开场，然后循环回到一般。在过去的一两年里，我在自己的 Lifehacker 帖子里养成了这个习惯。

有时你反过来做:讲述一个个人轶事作为一个大话题的引子。你可能也应该把这个引子下移。如果这剥夺了个人故事的目的，删除它。

有时候你用第一段开玩笑。或者至少我知道。将笑话加入到文章的其余部分——现在你已经有了一篇稍微有趣的文章了！—或者杀光他们。这些是每个人都让你去杀的宠儿。顺便说一下，这是一个无益的比喻。编辑不是杀婴，是节育。

### 修正你的结论

你也可以删除最后一段。见鬼，你可以删掉最后三个。

如果你的最后几行没有增加新的信息，删掉它们。突然结束总比像巴特·辛普森的 [“利比亚是一个充满反差的国家”那样毁掉你的作品要好【T2](https://frinkiac.com/meme/S09E14/199248.jpg?b64lines=IElOIENPTkNMVVNJT04sIExJQllBIElTIEEKIExBTkQgT0YgQ09OVFJBU1RTLiAKVEhBTksgWU9VLg==)

如果你的最后几行大大扩展了你文章的范围，删掉它们。一个令人尴尬的大结论是一个好迹象:你积累了所有的赛车动力，你在弯道上飞驰，在直道上狂奔。但你需要在驶下悬崖前踩下刹车。

在过去的几段中，我已经将一些 Lifehacker 的帖子赶下了悬崖。我给出一些具体的建议，我把它说清楚，我把我的观点讲清楚，我提供必要的警告，然后我试着把这些具体的建议变成生活的咒语。我的 [糟糕的咖啡随笔](https://lifehacker.com/bad-coffee-will-make-you-a-happier-person-1822250111) 是好东西，但我不应该把教训延伸到包括“买蹩脚的衣服”和“买蹩脚的手机”这超出了文章的范围！这削弱了我的观点。如果再给我一次机会，我会删掉那一段。去做同样的事。
