# 使用 LICEcap 将您的屏幕录制为 GIF

> 原文：<https://lifehacker.com/record-your-screen-as-a-gif-using-licecap-1840204678>

**Windows 和 MAC OS:**[LICEcap](https://www.cockos.com/licecap/)是一个简单的应用程序，用于录制你屏幕的一部分，并将其保存为 GIF 格式。当您不能轻松共享视频文件时，或者当您想要将您的录制内容发布到网页上，并且视频上传太麻烦时，它非常有用。



LICEcap 看起来像一个应用程序窗口的空框架。你把它拖到你想要记录的区域，然后点击*记录*。然后，您可以像平常一样与窗口进行交互。我录下了自己玩 Flash 游戏的过程:

如果你在 Photoshop 中清理任何杂散帧，该应用程序也是一种非常粗略的将视频剪辑转换为 GIF 的方法。我录制了我的一个旧 YouTube 视频，只是通过播放 LICEcap 帧中的视频，然后在 Photoshop 中修剪掉开头和结尾。这张照片比我想象的要压缩，但是它表达了我的意思

在 LICEcap 中，您可以选择是否可视化您的鼠标点击。您还可以添加文本，在录制时移动帧，使用键盘快捷键暂停录制，以及选取最大帧速率。您必须使您的窗口适合 LICEcap 窗口的边框；在 MacOS 上，我无法让 LICEcap 在全屏模式下工作。这是一个快速和简单的录音快速和肮脏的应用程序。
