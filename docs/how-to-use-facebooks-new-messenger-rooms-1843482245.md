# 如何使用脸书的新“传达室”

> 原文：<https://lifehacker.com/how-to-use-facebooks-new-messenger-rooms-1843482245>

如果你使用脸书或 Facebook Messenger，你可能会看到提示你试用 [新的“消息室”功能](https://about.fb.com/news/2020/04/introducing-messenger-rooms/) 的通知。Messenger Rooms 基本上是 Zoom、Teams 或 Google Meet 等视频聊天应用的免费且更简单的替代品。它们不是一个独立的产品，而是直接集成到 Facebook Messenger 中。每个房间同时支持多达 50 个用户。

我们中的许多人一直在远程工作和社交，Messenger Rooms 提供了通过我们的电话和电脑相互联系的另一种方式——但没有你会在其他视频会议应用程序中找到的一些多余的方式。

### 如何开始一个新的信使室

#### **在 iOS/Android 上**

1.  打开 Messenger 应用
2.  点击**“人物”**标签
3.  点击**“创建房间”**
4.  要邀请其他人，点击**“共享链接”**和选择您想要共享房间链接的联系人。您还可以指定是否希望任何具有该链接的人都能加入您的房间(不需要脸书帐户)，或者是否希望您的房间仅限脸书用户使用。

#### 桌面上的

*   **打开信使上途经 messenger.com 的站点(或)**
*   **点击左侧菜单顶部的**“开始新房间”**摄像头图标。**
*   **将弹出一个新窗口。点击**“继续为【你】”**开始开房。**
*   **若要邀请其他人，请将房间窗口顶部的 URL 复制、粘贴并发送给您想要邀请的任何人。**

### ****如何在 Messenger 中加入房间****

**在手机 app 或桌面上加入房间是一样的。只需轻按或点按邀请链接即可打开房间，然后点按“以[用户名]身份加入”(一次只能在一个传达室。)房间只接受邀请，但请记住，只要有链接，任何人都可以加入房间——您不能将房间限制为您的朋友的子集(或任何特定的朋友列表)。**

**一旦您进入 Messenger 的房间，就可以很容易地对其进行控制。屏幕上的按钮包括:**

*   **麦克风静音开/关**
*   **相机开/关**
*   **切换到所有参与者视图** 
*   **红色的“挂断”气泡。桌面用户如果愿意，也可以关闭浏览器窗口结束通话。**

**如果你是 messenger 聊天室的主持人，你可以点击聊天应用程序或桌面版本上的参与者图标来锁定聊天室——防止其他人加入，如果他们碰巧访问了你的链接。你也可以随时结束聊天，把所有人赶出你的房间。**

**在 M essenger R ooms 的**桌面**版本上，还有一个设置菜单，如果你需要修理麦克风或摄像头，它提供了几个音频/视频选项，以及一个启用屏幕共享模式的按钮(基于应用程序的版本没有)。与此同时，在**基于应用程序的 messenger rooms** 上，你可以点击左下角的一个按钮，为你巨大的闲逛——呃，房间快速拍照。**
