# 股票亏损不算，直到你卖出

> 原文：<https://lifehacker.com/stock-losses-dont-count-until-you-sell-1841493530>

我的投资最近一直在反弹，尽管它们呈下降趋势；过去一周，我投资组合的总体回报率从 10.3%升至 9.3%。

Watch

但我不会卖出——如果你担心冠状病毒爆发、弹劾裁决、所有关税问题或任何其他突发新闻可能会对你的投资组合产生长期负面影响，这里有一条来自莎拉·奥布莱恩的建议，她是 [【美国消费者新闻与商业频道】](https://www.cnbc.com/2020/02/06/there-is-a-time-to-dump-stocks-and-move-to-cash-some-experts-say.html)的个人理财记者:

> 长期投资者——即那些几十年内都不需要钱的人——应该记住，尽管股市可能会波动或进入长期低迷，但除非你卖出，否则你在账面上看到的损失不会被锁定。

我以前从来没有这样说过——当市场下跌时，我的座右铭是“当股票打折时，是时候投入更多的钱了！”

但我真的真的很喜欢这个概念。

股票亏损直到你卖出才算。

如果你可以买入并持有，然后*继续持有*，那么市场最终反弹的可能性很大。也就是说，我不是投资顾问，我很确定我必须告诉你，你不应该把这当成实际的投资建议。

这只是你下次醒来发现你的投资再次贬值时要考虑的事情。

(此外，确保你有一些现金留在应急基金中，这样你就可以将这些投资留在市场上，而不是在股票下跌时卖掉，因为你需要支付抵押贷款或其他费用。)
