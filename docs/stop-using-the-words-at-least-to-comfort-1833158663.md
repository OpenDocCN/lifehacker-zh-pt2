# 不要再用“至少”来安慰人了

> 原文：<https://lifehacker.com/stop-using-the-words-at-least-to-comfort-1833158663>

当有人向你倾诉时——从真正糟糕的事情到普通的糟糕的一天——你接下来说的话绝对不应该是:“至少……”

Watch

“至少……”之后的任何内容往好里说是最小化，往坏里说是冒犯。

我得了重感冒。至少不是流感。

我今天失业了。至少你可以从工作中休息一下。

我流产了。至少你知道你可以怀孕。

大多数用“至少……”的人试图安慰是用心良苦的。他们认为指出“好的一面”是有帮助的。但是痛苦中的人不想看到光明的一面；他们希望被倾听和理解。

“至少”可能是一件令人同情的事情，但它不是 T2 感同身受的事情。根据 [布琳·布朗](https://brenebrown.com) 的说法，差异是实质性的，这位研究教授研究了勇气、脆弱、羞耻和移情。在这个动画短片 中，她很好地解释了:
