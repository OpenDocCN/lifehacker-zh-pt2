# 在炎热的天气里你应该避免饮酒吗？

> 原文：<https://lifehacker.com/should-you-avoid-alcohol-in-the-heat-1836499730>

天气越热，饮料似乎越提神——包括一杯冰啤酒或一杯 frosé。但是出于安全考虑，你可能会想在热浪来袭时尽量减少饮酒。



酒精会让你脱水，真的。它会阻止你体内一种叫做 [抗利尿激素](https://www.yourhormones.info/hormones/anti-diuretic-hormone/) (ADH)的天然激素的释放。正常情况下，抗利尿激素会阻止你排出太多的水。这意味着，当你喝很多水时，你会尿很多，失去的水分甚至比你摄入的还要多。

如果你只喝一两杯，并确保保持水分，这没什么大不了的，但你喝得越多，你就会越脱水，你对自己的判断就越难控制。因此，作为一个安全提示，最好避免饮酒，或者在非常热的日子里尽量少饮酒。脱水会使人更容易得热病，在极端情况下会危及生命。

如果你在游泳池或海滩上降温，你必须格外小心。不是扫兴，但是 70%与户外娱乐相关的溺水死亡与酒精有关 。当你喝醉的时候，你更容易做傻事，你的平衡和协调性也受到了影响。

所以，如果你打算今年夏天去海滩喝酒，至少要聪明一点。多喝水，不要喝太多酒，和那些足够清醒的人呆在一起，如果你把自己置于危险之中，他们会注意到。
