# 这个病毒菠萝黑客有一些问题

> 原文:[https://life hacker . com/this-viral-菠萝-hack-has-some-issues-1833110354](https://lifehacker.com/this-viral-pineapple-hack-has-some-issues-1833110354)

在我的生活和职业生涯的这个阶段，我觉得我已经掌握了如何削水果和吃水果的方法，但每隔一段时间，我就会对我以前没有考虑过的方法感到惊讶(比如用一品脱玻璃杯 削芒果)，这种方法非常有效。然而，我对这条病毒式推文中展示的技术并不感冒。

Watch [https://lifehacker.com/embed/inset/iframe?id=twitter-1102704616142225408&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1102704616142225408&autosize=1) 

首先，我不相信一些预切片没有发生。当那只无形的手去摘下一小片整齐的金色水果时，你可以看到*其他的部分也一起拿了起来*，对我来说，这是恶作剧的象征。

但我还是想试一试，所以我买了一个成熟的菠萝，切掉它的顶部，然后把我的双关节拇指伸进去。它不起作用。正如你从我们的头部图像中看到的，我只能撕下一小块皮肤，只附上最悲伤的水果。为了确保我没有“做错”,我寻找了其他视频，找到了一个显示该技术获得中等(更真实)成功的视频。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1103386798686945280&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1103386798686945280&autosize=1) 

虽然它在技术上“有效”，Twitter 用户 Mia 确实注意到它非常混乱，并且需要“一些技巧”(我怀疑长指甲在这里也有帮助。)我决定再次尝试我的菠萝，并设法——在真正开始吃之前用拇指轻轻戳了几个地方——吃了一小块。(果汁到处都是。)

我怀疑用超级熟的菠萝更容易做到这一点，但这也忽略了一个事实，那就是吃这种多汁的水果是一种混乱的方式，而且它需要“巧妙处理”的事实意味着它不是一种更容易、更简单或更精简的做事方式。它也不适合分享，也不是提供给一群人的好方法。几只不知道清洁程度的手伸进去撕开一个菠萝，一次一片是不卫生的、低效的、可笑的，尤其是当你想到我们是唯一用刀子的物种时。因此，我的专业意见是，这不是一个好的黑客，但事实上，是古怪的。
