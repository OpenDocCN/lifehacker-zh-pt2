# 你应该让木法沙成为你的锁屏

> 原文：<https://lifehacker.com/you-should-make-mufasa-your-lock-screen-1839897803>

当我的朋友科尔花太多时间穿着内衣玩视频游戏，或者在 YouTube 上看别人玩视频游戏时，他会想象他的祖父叹着气，翻着白眼，说“我为了这个参加了一场战争？”

Watch

我想，正是这种精神让 Reddit 用户的 alwaysmyday [把他们的锁屏](https://www.reddit.com/r/SuccessionTV/comments/dkzox0/put_this_as_my_phone_locked_screen_to_remind/) 换成了洛根·罗伊的照片，这位反对《T4》继任的父亲。redditor 表示，这是一个有用的提醒，提醒人们远离手机。你也可以这样做，不管你对失望的父母形象的品味如何。对你来说可能是:

*   木法沙，对辛巴在大象墓地玩耍感到失望
*   米兰达·普里斯特利，对安妮·海瑟薇诋毁天蓝色感到失望
*   戈登·拉姆齐小声嘀咕着“该死”，因为一个有前途的*厨艺大师*选手做了一道糟糕的菜
*   汉克·希尔，对鲍比不会投棒球感到失望
*   朱迪·丹奇的 M，很失望邦德没能控制住它
*   玛吉·史密斯在她扮演的所有角色中，对有人用错了叉子感到失望，或者嫁入了一个不得不买家具的家庭

如果你想 [制作一个动画锁屏](https://lifehacker.com/how-to-make-an-animated-lock-screen-for-your-phone-1827183651) ，找一个好的静止图像，甚至是 GIF。想象一下这个角色看着你在他妈的手机上浪费生命。感到羞耻。让这种羞耻驱使你远离手机，回到现实生活中，或者至少回到你的电脑前。

常见的“手机并非天生就是坏的”警告也适用。只有当你真的想为了你自己好而不使用手机，但是你不可能一整天都关机或者把它放在家里的时候，才这样做。(疯狂的是，人们过去整天只是走来走去，手边却没有一个小小的通信-信息-娱乐中心。)不赞成的数字建议欢迎。
