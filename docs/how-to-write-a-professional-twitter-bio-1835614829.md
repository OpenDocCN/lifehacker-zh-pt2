# 如何写一篇专业的推特简历

> 原文：<https://lifehacker.com/how-to-write-a-professional-twitter-bio-1835614829>

你的 Twitter 账户应该反映你的个人生活，还是你的专业知识？可以说，答案是“两者都有”——但要找到如何平衡这两种身份并不总是那么容易。



在自由职业者 的 [上，艾米丽·高德特解释了如何创建一个 Twitter feed，让人们了解你的专业工作，同时提醒他们应该喜欢(并关注)你的所有理由:](https://contently.net/2019/06/12/resources/how-to-use-twitter-writer-professional-freelance/)

> 你的订阅不必读起来像 LinkedIn 的个人资料——事实上，确保它不是！你应该在 Twitter 上发布你希望人们喜欢的内容，但你也应该在你的订阅中加入观点、见解和关于你职业活动的更新。

虽然她的建议是专门为自由职业者和内容创作者设计的，但它也适用于传统员工。高德特建议用你的推特简历来吸引人们对你专业工作的注意，同时突出你的一个个人兴趣，例如:

> 我还选择了三个主要关键词来描述我的专业职能:两个严肃(编辑，播客主持人)和一个戏谑(地下城主)。尽管让编辑们知道我是一个精通龙与地下城的玩家并没有什么帮助，但这确实增强了他们对我的“个人品牌”的整体理解

在阅读了高德特的建议后，我更新了我的推特简历——现在是“作家、编辑、教师”。在 Lifehacker、Vox、Bankrate 等网站从事自由职业。《普通人传记》的作者。偶尔有人付钱让我唱歌。”

这准确地反映了如果人们在 Twitter 上 [关注我](https://twitter.com/HelloTheFuture) 会得到什么；链接到我的自由职业工作和即将到来的课程，我正在起草的小说的更新，以及关于我正在排练的音乐的笑话。

在这个世界上，雇主和潜在客户经常使用社交媒体来联系和审查他们未来可能合作的人，拥有一个反映你的职业和个人生活的社交媒体可以极大地促进职业发展。例如，如果我的 Twitter 除了一串指向我的文章的链接之外什么也没显示，我会给人一种超级没有人情味的感觉(而且很无聊，这对一个想以作家身份赚钱的人来说不是好事)。

不过，我的推特还远未达到最佳状态；我不发布很多图片，虽然我经常转发和分享其他人的作品，但我并不总是提供将简单的转发变成联系时刻的个人见解。

如果这听起来像是额外的工作，那么...确实是。而是把它当成新版本的闲聊；离开办公桌和待办事项的必要时间，与团队中的其他人交流职场友好故事。

只是在 Twitter 上，你的团队包括你职业和个人领域的所有人——以及任何最终可能喜欢你的工作并决定追随你的人。
