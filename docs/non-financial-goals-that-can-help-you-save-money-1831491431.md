# 可以帮你省钱的非财务目标

> 原文:[https://life hacker . com/non-financial-goals-that-can-help-you-save-money-1831491431](https://lifehacker.com/non-financial-goals-that-can-help-you-save-money-1831491431)

在 401(k)计划、529 计划、个人退休账户、高收益账户和为你工作的应用程序之间，不缺少省钱的方法。有效使用这些账户和应用是长期财务健康的关键。

Watch

也许你 2019 年的金钱目标是节省更多的钱，通过利用所有这些选项:你正在 [增加你的 401(k)贡献](https://twocents.lifehacker.com/increase-your-401-k-or-ira-contributions-for-2019-1830819672) ，或者使用像 [Digit 或 Qapital](https://lifehacker.com/saving-app-showdown-qapital-vs-digit-1790189978) 这样的应用程序为你的下一次旅行存钱。但是在严格的财务领域之外，你还可以做很多其他的事情来帮助你实现你的底线，同时实现你的另一个目标或愿望。

### 学习一项新技能

这可能看起来违反直觉，因为你可能认为你需要新的工具、书籍或经验来开始学习新的东西。例如，如果你想做饭，难道你不需要一本烹饪书和配料，一个速溶锅和一套新的刀具吗？

毫无疑问，你会需要一些东西，但对我们来说，节省的时间来自于学习新东西的时间投入。如果你在厨房里闲荡，或者在读一本关于罗马衰落的新历史书，你可能不会去网上购物，也不会突然去买“一样东西”。

同样，也许你只是想发现新的事物或兴趣。正如我 [在这里](https://twocents.lifehacker.com/how-to-prepare-for-a-no-spend-month-1831401525) 写的，“挑一个你一直很好奇的话题，从你的图书馆借关于它的书，在 YouTube 上搜索视频，听关于它的播客片段，等等。阅读作者认为 [激发了你最喜欢的作者](https://lifehacker.com/make-your-own-family-tree-for-creative-inspiration-1829708944) 。”同样，尝试拓展你的音乐或电影品味，或者参加你所在城市的免费艺术展。

理想情况下，如果目标是学习一项技能或关于一个新的话题，它将需要磨练。这可能意味着花几周或几个月的时间和精力在某件事情上，而不是花钱。

### 投资你的友谊

成为一个好朋友或家庭成员并不需要金钱。有很多便宜或不贵的活动你可以做，不要介意只是和你的朋友在那里，为你的朋友。

Buzzfeed 有一个 [的伟大列表](https://www.buzzfeed.com/rachelwmiller/the-ultimate-guide-to-showing-up-for-other-people?utm_term=.dmgrLl3bZ#.leB63poew) 的方式来表达真正的欣赏，这是我们这里 [总结的](https://lifehacker.com/how-to-maintain-your-friendships-1827484649) 。以下是一些基本知识:

*   为重要的日期设置日历提醒，并确认它们。
*   对他们的兴趣感兴趣。
*   有一个每月食谱俱乐部。
*   不方便的时候支持他们。

### 参加阅读挑战

免费娱乐的一个明显资源是图书馆(或者图书馆 app， [Libby](https://lifehacker.com/five-best-audiobook-services-1688572545#_ga=2.236532353.1699283556.1546610522-594046802.1524762060) )。但是与其告诉自己下班后你会去图书馆买一本新的小说——因为你不会——不如给自己一个挑战。

例如，在过去的三年里，我的目标是每个月至少读两本与工作无关的书。今年，我通过限定它们必须由女性作者来提高它(明确地说，我可以也愿意阅读男性作者的书籍，但它们不计入目标)。这个挑战给了我动力，让我更频繁地寻找新书，花更多的时间阅读。不，这里没有什么真正的事情，但是找到新的女性作家和阅读新的作品给了我巨大的满足感。

你可以选择参加 Goodreads 挑战赛，在那里你的朋友和其他追随者可以看到你的进步，参加[Book Riot Challenge](https://bookriot.com/2018/12/12/2019-read-harder-challenge/)，或者找一个朋友来监督你。

### 创造更多的艺术

每周/每月/每年都有艺术挑战，但最好的参与方式可能是自己动手。选择一个你想提高的领域——写作、摄影、绘画、雕塑等等。—承诺每天在其中创造一些东西，无论你认为它会挑战和激励你多长时间。

如果你在寻找更有条理的东西，可以包括:

*   一年每天拍一张。
*   每天学一个新单词，坚持一年。
*   每天写 1000 字，坚持一个月。
*   每天写*点东西*(灵感来源:查看作者 [奥斯汀·克莱恩的博客](https://austinkleon.com/) )。
*   每周尝试一种新的食谱(或者尝试完整的朱莉*茱莉&茱莉亚*食谱，并按照你选择的食谱进行工作)。更多 inspo 见 Reddit 的 [烹饪 52 周](https://www.reddit.com/r/52weeksofcooking/) 线程。
*   [每天创建一个图案，持续 100 天](https://makeitindesign.com/100daysofpattern/) 。
*   每天创建一个调色板，无论是基于你的服装、照片还是你遇到的其他场景。

还有一些比较有想法的 [这里](https://dayzeroproject.com/feature/61/big-list-of-daily-weekly-monthly-challenges)[这里](https://kincavelkorner.wordpress.com/about/52-things-in-52-weeks-2013/) 。

根据你的兴趣，有数不清的方法来投资你的时间和精力，而不用花很多钱。如果你正在尝试其他方法或者有一个旧的备用方法，请在评论中告诉我们。
