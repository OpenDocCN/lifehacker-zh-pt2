# 如何在“随处看电影”上与朋友一起看电影

> 原文：<https://lifehacker.com/how-to-watch-movies-with-your-friends-on-movies-anywher-1844548979>

电影无处不在最近推出了 [新的“一起看”功能](https://moviesanywhere.com/watch-together) ，让多人在互联网上同步观看电影之夜。这项新功能可以在你的桌面上运行，也可以通过 Movies Anywhere 的 iOS、Android 和智能电视应用程序运行，这项服务上的所有电影都可以供团体观看派对观看——只要你自己拥有它们。

### 如何在任何地方设置“一起看电影”会话

首先，每个人都需要注册自己的“随处电影”账户——即使是那些将作为嘉宾观看的人。每个帐户都需要至少观看 PG-13 电影的权限，这一设置可以通过其家长控制来更改 [。](https://help.moviesanywhere.com/hc/en-us/articles/115004565726-How-do-I-restrict-movie-watching.)T3】

能够参与观看会话的总人数取决于是否每个人都已经拥有该电影。最多可以有九个人一起观看，前提是每个人的电影 Anywhere collection 中的功能演示都是解锁的，这要求你要么购买电影 Anywhere 的标题，要么通过兑换物理副本中包含的代码来解锁。如果你已经了解了所有这些，你可以跳到下面的**“开始一起观看电影的会话”**。

如果不是每个人都拥有这部电影，您最多只能和另外三位客人一起观看。这些客人都不需要拥有这部电影，这很酷，但你需要先通过烦人的新“屏幕通行证”功能借给他们每个人。

### 通过 Movies Anywhere 的“Screen Pass”功能租借电影

电影 Anywhere 的借阅功能目前仅限于 [Screen Pass beta 用户](https://moviesanywhere.com/what-is-screen-pass) ，目前只能借阅某些片头。《漫威》和《星球大战》等著名系列电影目前不包括在内，这意味着如果你想作为一个群体观看，每个人都需要拥有自己的《T4》复仇者联盟 4：终局之战。

也就是说，任何人都可以请求并获得借出的副本，即使他们不在测试版中。Screen Pass lending 也不限于一起观看；用户可以在收到电影后的 14 天内，或者在点击“播放”后的 72 小时内，随时观看他们“借用”的电影，无论是独自观看还是集体观看

1.  若要借出标题，请在 Movies Anywhere 应用程序中访问您的 Movies Anywhere 储物柜，或者通过 moviesanywhere.com 访问它。
2.  滚动到**“我的屏幕合格电影”**部分。
3.  从库中选择要共享的电影。
4.  点击**“详细信息”按钮，然后选择“屏幕传递”**
5.  复制链接并与最多三个其他用户共享。
6.  其他用户可以在“随处电影”应用程序中或通过网站接受借出的电影。

### 随时随地开始观看电影

一旦每个人都建立了自己的“随处看电影”帐户，并购买或租借了电影拷贝，您就可以开始“一起看”会话了。

1.  要主持“一起观看”会话，请打开“随处电影”应用程序或在桌面浏览器上转到。
2.  导航到您的电影储物柜。
3.  滚动并选择您想要观看的影片，然后选择“一起观看”
4.  会弹出一个网址；复制链接并与您想与之一起观看的人分享。
5.  主持人可以随时开始播放电影。所有回放—暂停、倒带、快进—都由主持人控制，但来宾可以发送暂停、倒带等请求。所有参与者，如果他们想对某个场景做出反应，请点击或轻触表情菜单。
