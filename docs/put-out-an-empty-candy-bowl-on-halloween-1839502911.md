# 在万圣节放一个空的糖果碗

> 原文：<https://lifehacker.com/put-out-an-empty-candy-bowl-on-halloween-1839502911>

需要一些最后一分钟的万圣节糖果，因为你要么太便宜而买不起，要么你自己吃光了？这里有一个窍门:你实际上不知道。



相反，拿出一个空碗。把它想象成你房子的服装:你正在把它装扮成一个房子，这个房子*正在*分发糖果，但是已经用完了。你犯了一个多么诚实的错误啊！只是想在偷懒的同时给邻居家的孩子带来一点欢乐，现在看看发生了什么。

用一个适当的标志来推动叙述:比如“请只拿一个，谢谢。万圣节快乐！”在“一”下面划几下线。然后，不给糖就捣蛋的人会对那些比他们先来的贪婪的孩子发火，而不是对无辜的小你发火。
