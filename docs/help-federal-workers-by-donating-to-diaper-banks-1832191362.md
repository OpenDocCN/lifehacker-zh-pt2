# 通过向尿布银行捐款来帮助联邦工作人员

> 原文：<https://lifehacker.com/help-federal-workers-by-donating-to-diaper-banks-1832191362>

尽管在政府关门期间一个月没有拿到薪水的联邦工作人员有望在本周拿到工资，但尿布银行仍在不知疲倦地为他们的家庭分发尿布。



北卡罗莱纳州 尿布银行的官员预计在 1 月和 2 月期间将向家庭额外发放 20 万片尿布，主要是因为政府关闭了，导致许多联邦雇员忙于支付基本生活费用。

“我们知道，我们州的许多员工已经开始靠薪水度日了，”DBNC 的执行董事米歇尔·奥尔德说，该公司经营着三个地区分公司，还经销女性卫生用品和其他个人护理用品。"停薪留职，即使是短时间的停薪留职，也会让这些家庭处于弱势地位。"

尿布银行通常部分通过企业合作，部分通过当地的尿布运动，部分通过捐款来保持库存。现在，这些捐款对尿布银行来说和以往一样重要，因为尿布银行能够保持库存，以帮助已经有需要的家庭，以防自然灾害，并为另一次潜在的政府关闭做准备。

“每天我们都听到联邦工作人员不得不在支付住房和供暖费用或为孩子提供足够的尿布之间做出选择。”老说。“对支持的需求处于关键水平，并将持续一段时间，因为家庭试图重组并重新站起来。”
