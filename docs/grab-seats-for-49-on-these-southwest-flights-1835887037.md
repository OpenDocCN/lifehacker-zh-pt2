# 在这些西南航班上以 49 美元的价格抢到座位

> 原文：<https://lifehacker.com/grab-seats-for-49-on-these-southwest-flights-1835887037>

就在生日特卖一周后，西南航空公司将在 T2 举行另一场今秋航班特卖，将于今晚午夜结束。

Watch

只要 49 美元，你就可以买到从纳什维尔到亚特兰大的单程票(反之亦然)。其余的销售仍然相当不错，从旧金山到拉斯维加斯的旅行只需 65 美元。

在您预订之前，您还应该了解一些其他条款和条件。停电日期适用，这意味着您将无法在感恩节当天或前后预订。票也不能退，所以不能反悔。

以下是我们发现的一些其他交易:

*   达拉斯到塔尔萨:59 美元
*   拉斯维加斯到凤凰城:59 美元
*   奥克兰到洛杉矶、伯班克或圣地亚哥:64 美元
*   孟菲斯到巴尔的摩:99 美元
*   纽约(拉瓜迪亚)到亚特兰大:108 美元

如果你打算今年秋天去海外旅行，这里有一些值得注意的优惠:

*   洛杉矶到墨西哥 Cabo 圣卢卡斯:151 美元
*   劳德代尔堡到巴哈马拿骚:184 美元
*   奥克兰到墨西哥巴亚尔塔港:201 美元
*   奥兰多到特克斯和凯科斯的普罗维登夏莱斯:205 美元
*   休斯顿到利比里亚，哥斯达黎加:225 美元

要查看你附近的交易，看看西南航空的 [网站](https://www.southwest.com/html/promotions/nationwide_sale_NonstopConnect_190624.html) 并过滤当地机场的结果。与几乎所有其他航空公司不同的是，西南航空*也不收取托运行李的费用，所以这是一笔特别划算的交易。*
