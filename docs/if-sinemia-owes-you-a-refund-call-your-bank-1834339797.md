# 如果 Sinemia 欠你一笔退款，打电话给你的银行

> 原文：<https://lifehacker.com/if-sinemia-owes-you-a-refund-call-your-bank-1834339797>

电影票折扣订阅服务公司 Sinemia 是 MoviePass 的主要竞争对手，它今天宣布将停止在美国的运营——这将在今年最大的电影 [之一](https://lifehacker.com/celebrate-avengers-endgame-with-this-google-easter-egg-1834328500) 上映的当天影响各地的电影观众。



“作为 Sinemia，我们的愿景是通过创建电影票订阅服务，为电影观众和电影行业增加价值，帮助尽可能多的电影观众享受负担得起的更好的电影体验，”Sinemia 在给订户的电子邮件中写道。"[...]尽管我们的团队尽了最大努力，但作为一家初创公司，如果没有资源和足够的资本来满足不断增长的运营和法律成本，我们很难继续为美国的电影观众提供服务。”

换句话说，他们真的没钱了，从声明来看，Sinemia 没有发布任何关于退款的公告。有什么办法能把你的钱要回来吗？

你应该采取的第一步是立即给你的银行打电话，提出退款；告诉他们你为从未提供的服务付费。据一条 [Reddit 线索](https://old.reddit.com/r/Sinemia/comments/bhej59/sinemia_ending_operations_effective_immediately/) 称，许多通过 Sinemia 支付年度会员费的用户正在通过 Chase 获得按比例退款。(还有，你应该通知你的银行阻止该公司的任何*进一步*交易，以防万一。)

鉴于该公司刚刚申请破产，你暂时不太可能直接从 Sinemia 拿回你的钱。目前，与你的银行联系是你最好的选择。
