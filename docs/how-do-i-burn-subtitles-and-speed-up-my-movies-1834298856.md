# 如何刻录字幕并加快电影播放速度？

> 原文：<https://lifehacker.com/how-do-i-burn-subtitles-and-speed-up-my-movies-1834298856>

> 背景:我通常在 VLC 观看播放速率为 1.33 倍或更高的电影/节目。(时间就是金钱，amiright？)为了加快播放速度，以确保我能够拾取所有的对话，我将字幕加载到 VLC 中。srt 文件。我有一台 Plex 服务器，可以将视频文件从我的电脑传输到 Roku 连接的电视上。我的最终目标是能够在我的 Roku 连接的电视上以更快/可变的回放速度播放视频文件，同时还能看到字幕。

乐意帮忙！我提前说一下，因为我已经有一段时间没有做大量的视频编码了。如果没记错的话——借用一句老日本*铁厨*秀的话——你应该可以使用流行的(开源的)工具 [手刹](https://handbrake.fr/) 将字幕刻录到你的视频文件中。

一旦你下载并安装了它，就可以用它来播放视频。我用的是我从网上下载的疾速追杀的拷贝，因为有人已经制作了字幕。SRT)为它归档。(对于一个普通的 ol '视频，我懒得自己做那个；原谅我，狮门影业。)

要将字幕刻录到您的视频中，请点击手刹中的(适当命名的)“字幕”选项卡，点击导入 SRT ( [，假设您有一个](https://handbrake.fr/docs/en/latest/advanced/subtitles.html) )，并选中“刻录”框，这将确保您的字幕被硬编码在您的电影之上。

哒哒！

至于你问题的第二部分——提高视频的播放速度——我最初有点不知所措。(我通常会像你一样做:提高我的软件播放器 VLC 的速度，而不是改变原始视频)。

然而，我做了一些磕磕绊绊，我认为你实际上正在寻找的工具是 [**【公理**](https://github.com/MattMcManis/Axiom) ，一个对 FFmpeg 编码器有用的 GUI。手刹也用 FFmpeg，所以两个程序是...在执行上相似，只是在接口上差别很大。

下载并运行 Axiom——无需安装——点击 Input 找到并加载您的电影文件。在字幕选项卡上，将编解码器切换到“刻录”并使用小加号图标来查找和选择您的。SRT 档案。

从那里，点击“视频”标签，向下滚动一点，直到你看到“速度”设置。把它设置成你想要的，尽管我实际上建议先去“过滤器”标签。原因如下。很明显，你会希望你的加速视频有同步的音频，调整后者速度的设置实际上是“滤波器”标签中的“速度”选项。滑块很麻烦——至少，它一次只让我上下移动三个增量——它对应的是一个百分比。换句话说，设置为“50”意味着你的音频速度减半；将其设置为“150”意味着增加 50%，以此类推。

记住您在那里设置的值(为了举例，让我们说“140”，然后返回到 Video 选项卡。现在，调整速度以匹配—对于我的例子是 140，这要求我从下拉菜单中选择“Custom”并手动输入。

您可以(也应该)调整 Axiom 中的其他设置，以确保您的编码获得正确的视频和音频质量(以及正确的输出格式)。不过，我喜欢 Axiom 的是它的预览功能。按下那个按钮，你就能看到你的设置是否给了你想要的那种视频。(具体来说，仔细检查您的视频和音频是否同步非常有用。)

值得一提的是，你可能想在其他地方保存一份未经修改的电影副本，以防 Plex-Roku 版本-有一个速度调整功能。
