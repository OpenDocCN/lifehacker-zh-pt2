# 通过将 Disney+安装为渐进式网络应用程序来更快地启动它

> 原文：<https://lifehacker.com/launch-disney-faster-by-installing-it-as-a-progressive-1842728846>

如果你在 Windows 或 macOS 上看了很多 Disney+的，考虑将该网站安装为一个渐进式网络应用(PWA) ，以便更快地访问你最喜欢的电影和节目。

Watch

PWAs 把一个网站和 [一个全功能的应用](https://lifehacker.com/this-free-disney-mac-app-resumes-your-shows-as-soon-as-1840206512) 区分开来。将 Disney+安装为 PWA 会为网站的独立版本创建一个易于访问的桌面快捷方式。当启动时，PWA 将不仅绕过你的浏览器主页并移除它的大部分 UI，与通过传统方法查看网站的 相比，它将提高速度和稳定性。

更好的是，由于 PWA 是基于网络的，安装将占用 在你的 PC 上微不足道的空间，并且实现它非常容易——尽管我们通过 [Edge Chromium](https://lifehacker.com/our-favorite-tips-for-microsofts-new-edge-chromium-brow-1841041653) 安装 Disney+作为 PWA 是最成功的。理论上，其他浏览器如谷歌 Chrome 浏览器也可以通过 PWA 来支持流媒体服务，但这个选项现在似乎不存在——至少，不能通过像 Edge Chromium 浏览器那样优雅的实现方式来实现。

幸运的是，如果你不愿意，你可以通过 Edge Chromium 安装 Disney+ PWA，而不必更换默认浏览器。当你双击新的快捷方式时，Disney+仍然会在 Edge Chromium 中 *加载*，但是你的其他设置不会受到影响；你仍然可以在日常驾驶中使用 Chrome、Firefox、Brave 或任何你喜欢的东西。(声明一下， [Edge Chromium 不算半- 坏](https://lifehacker.com/our-favorite-tips-for-microsofts-new-edge-chromium-brow-1841041653) 如果你在讨好新浏览器。)

### 如何在 Windows 上安装 Disney+作为 PWA？

1.  在一个边缘铬标签中打开[【disneyplus.com】](http://disneyplus.com)。
2.  登录您的 Disney+个人资料。
3.  点击**“…”浏览器右上方的**打开 Edge 菜单。
4.  转到**应用程序>“将此网站安装为应用程序”**
5.  键入应用程序快捷方式的名称，可以是任何名称。
6.  点击**“安装”**安装应该几乎瞬间完成。

一旦安装了 PWA，你的桌面上就会有一个指向 Disney+的快捷方式，如果你喜欢，你可以把它钉在任务栏或开始菜单上。双击桌面快捷方式，直接进入迪士尼+主页页面并开始播放——如果你在那里需要新的节目，请务必查看我们的 [月度综述](https://lifehacker.com/tag/disney-plus#!) (共期)迪士尼+即将上映。
