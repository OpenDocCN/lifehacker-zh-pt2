# 现在是“新闻饮食”的时候了

> 原文：<https://lifehacker.com/now-is-the-time-for-a-news-diet-1843522123>

你多久看一次新闻？一次是在你醒来的时候，可能是——另一次是在你开始工作之前，还有一次是在你需要休息的时候，还有一次是在午餐的时候，还有一次是...嗯，你明白了。

Watch

问题是，大多数时候，新闻并不都是新的。我们还在疫情。我们仍然处于经济危机中。我们仍然不知道事情会在何时或如何发生变化。

这意味着是时候改变我们自己的习惯了。具体来说，我们每小时都在新闻和社交媒体网站上打转的习惯，希望我们能看到一些可能让我们对所有这些不确定性感觉更好的东西。

因为——你可能已经知道了——我们很可能会看到一些让我们感觉更糟的事情。一篇推测经济会变得多么糟糕的文章。一个关于参观公园是否安全的 Twitter 帖子，在回复中有来回的争论。一篇关于疫情处理得有多糟糕的专栏文章。

所以是时候考虑新闻饮食了。例如，如果你每小时检查一次你的网站，试着每天检查一次或两次。也许你会在工作前和午饭后看新闻(因为你不想在休息前给自己压力)，然后暂停看新闻和社交媒体，直到第二天早上。

我知道很难停止阅读新闻，尤其是如果这是你在工作任务之间放松大脑的默认方法——但是因为新闻现在不是很放松，你甚至没有给自己你需要的精神休息。(这只是如今如此难以高效工作的众多原因之一。)当你需要休息阅读和滚动的时候，试着找些别的东西来阅读和滚动。我发现，无论是在笔记本电脑上还是在智能手机上，阅读休闲、舒适的电子书，都会让我感到“我需要盯着一些与工作无关的词，敲敲它们”的痒感，而不会让我暴露在新闻和社交媒体的猜测和焦虑循环中。

如果你是一个播客听众，找一些与新闻无关的播客，或者考虑找一些别的东西来听。(此时，我已经退订了大部分我以前最喜欢的播客，换成了我十几岁时听的经典百老汇演员专辑和乐队。)如果你喜欢开着电视作为背景噪音，那就看电视游戏节目网，而不是 CNN。如果社交媒体是你与朋友和家人保持联系的方式，筛选你的关注列表，直到它们只包含你真正关心的人——并考虑阻止某些引起焦虑的词，这样它们就不会出现在你的订阅源中。

因为当有值得知道的消息时，无论是有效的疫苗还是新一轮的刺激资金，相信我们，你会知道。
