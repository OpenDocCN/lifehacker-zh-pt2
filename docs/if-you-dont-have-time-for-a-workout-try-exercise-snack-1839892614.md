# 如果你没有时间锻炼，试着用运动“零食”来代替

> 原文：<https://lifehacker.com/if-you-dont-have-time-for-a-workout-try-exercise-snack-1839892614>

如果你没有时间锻炼，欢迎加入俱乐部。(我经常锻炼，但有时我会被在特定的地点和时间穿着运动服完成整个锻炼所花费的时间和精力所淹没。)一个办法:运动零食。

Watch

这是锻炼的一个小部分:绕着街区走一圈，在没人注意的时候在办公室做一组俯卧撑，安装一个门口引体向上的拉杆，每当你走过时做一两次。

零食有什么好？嗯，这取决于你的其他锻炼程序是什么样子的，如果你有一个的话。如果你目前什么都不做，零食是一个很好的方式来培养一个完整的锻炼习惯。如果你已经从健身中休息了一段时间，零食可能会刺激你的食欲，让你错过健身房。打个比喻来说，即使你现在按照固定的时间表进行锻炼，零食仍然是有营养和美味的。

小吃的美妙之处在于缺乏承诺。你不必换衣服，因为你可能不会出汗。如果你只打算做十次，然后放下哑铃回到电脑前，你不必为多组令人筋疲力尽的哑铃练习做心理准备。

几年前，在有了孩子让我改掉了去健身房的习惯之后，我又重新开始了锻炼。 [现在看我](https://vitals.lifehacker.com/im-beth-skwarecki-lifehacker-health-editor-and-this-i-1838284294) 。我的门廊上还有一对哑铃，门口还有一个引体向上的杠铃，我会时不时地拿它们当零食。在写这个故事之前，我甚至绕着街区走了一圈。所以试试零食吧，因为锻炼不一定非要孤注一掷。
