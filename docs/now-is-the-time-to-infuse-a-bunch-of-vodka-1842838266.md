# 现在是时候注入一些伏特加了

> 原文：<https://lifehacker.com/now-is-the-time-to-infuse-a-bunch-of-vodka-1842838266>

现在我们都在家里喝酒，你可能已经注意到你的酒吧推车并不像你最初认为的那样多样化。出去喝酒的一个主要优势是，他们有制作各种鸡尾酒所需的所有材料，而满足这种多样性通常超出了家庭饮酒者的预算。





然而，输液可以让你混合东西而不用买更多的瓶子，如果你有搅拌机，你可以在几分钟内完成。搅拌机输液既快又非常有效。叶片提取新鲜，干净的味道和美丽的颜色，当你过滤掉所有的固体，你就有了一个全新的注入伏特加完美的啜饮或混合成一个完整的鸡尾酒。(我很少混合这些——只是把它们放在冰箱里冷藏，然后用小玻璃杯喝。)

我以这种方式制作了[辣根伏特加](https://skillet.lifehacker.com/make-crazy-potent-horseradish-vodka-in-your-blender-1831650872)[佛手伏特加](https://skillet.lifehacker.com/blend-this-crazy-looking-citrus-right-into-your-booze-1832119969) 和 [大黄伏特加](https://skillet.lifehacker.com/rhubarb-vodka-is-the-best-thing-you-can-make-with-your-1836196220) ，但现在是你受冰箱里东西的启发，摆弄自己的*创意搅拌机的时候了。新鲜的药草或任何多叶的东西都是很好的开始，但是嫩茎、髓皮和美味的根也是很好的选择。(我一直梦想着一杯番茄皮伏特加；我认为果皮会增加微妙的鲜味和美丽的颜色。)*

*就数量而言，一杯半的叶子或草药对于 750 毫升的瓶子来说通常是足够的，但当你开始时，可以将它减少到每 250 毫升半杯，直到你找到你喜欢的。与缓慢浸泡的浸剂不同，搅拌机浸剂的一个优点是，你可以混合、品尝，并根据需要添加更多的调味剂，所以从少量的配料开始，然后慢慢添加。我得到了惨痛的教训，在测试辣根输液时 猛烈地挖掘了我的鼻窦。*

*除了伏特加和任何你注入的东西，你可能还想加一点糖或蜂蜜来使它变得圆润光滑。你可以添加一点点来减少味道(就像我用 [【山葵】](https://skillet.lifehacker.com/make-crazy-potent-horseradish-vodka-in-your-blender-1831650872) )或者你可以添加很多味道(就像我用 [大黄](https://skillet.lifehacker.com/rhubarb-vodka-is-the-best-thing-you-can-make-with-your-1836196220) )。如果你正在处理一片亮绿色的叶子，挤几下柠檬汁可以帮助防止褐变。*

*每次冲泡的过程都是一样的:把所有的东西都加入搅拌机，搅拌直到固体成分被破坏，然后通过咖啡过滤器过滤到瓶子里。如果保存在冰箱里，这些输液最好在几天内使用，但我已经在冰箱里保存了几个星期，没有问题。*
