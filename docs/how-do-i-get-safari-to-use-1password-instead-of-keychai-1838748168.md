# 如何让 Safari 使用^ 1 密码而不是钥匙串？

> 原文：<https://lifehacker.com/how-do-i-get-safari-to-use-1password-instead-of-keychai-1838748168>

欢迎回到 Tech 911，Lifehacker 的 [每周技术建议专栏](https://lifehacker.com/c/tech-911) 。本周，我们要解决一位读者的问题，他发现当他的浏览器提示他保存一项他不愿意使用的服务的密码时，这很烦人。这听起来像你吗？请继续阅读。



生活黑客读者约翰写道:

> *“我读了你关于 1 密码和瞭望塔的文章，我想知道你是否知道任何方法来同步 1 密码和钥匙串？我在 Mac、iPhone、iPad 和我工作的 Windows PC 上都有 1 个密码。当我在 Safari 中的网站上输入凭据时，它似乎会提供将凭据保存到 Keychain 或 1Password，似乎是随机选择的，而不会同时保存到两者。因此，我经常需要一个不容易找到的密码，因为它在 Keychain 而不是 1Password 中。”*

啊，我很清楚这个问题——到处都是你的登录名和密码，这使得很难记住你使用哪个服务来保存哪个密码。

我最初的建议是像一个坏习惯一样丢掉钥匙链。我并不反对，但是如果你已经有了一个密码，你应该假装 Keychain 不存在。没有必要在两个服务上分开你的密码，因为那会导致很多混乱，就像你现在所经历的。

如果我是你，我会先检查一下 [这个导入工具](https://discussions.agilebits.com/discussion/68239/how-do-i-transfer-my-icloud-keychain-passwords-to-1password) ，希望它能帮助你顺利地将 Mac 的钥匙串密码导入 1Password。你也可以手动操作。打开“钥匙串访问”(在 Spotlight 中键入),查看您的“登录”和“iCloud”钥匙串，看看您存储了什么。双击条目以找到您的用户名和登录名，在 1 密码中输入新条目，然后重复。如果您有很多登录，这可能需要一些时间，但这是值得的。

从那里，打开 Safari，启动它的偏好设置，然后点击“自动填充”标签。关闭所有选项，其中 [应该让 Safari](https://support.1password.com/disable-browser-password-manager/) 停止提示你关于钥匙圈的事情。(您也可以取消选中“密码”选项卡下的选项。)

我假设你正在使用它，但为了以防万一，请确保你也为你的 Mac 安装了 1Password 7 [，并在 Safari 中打开了扩展(通过偏好设置中的“扩展”标签)。这样，1Password(而非 Safari)会提示您在网站上保存密码。](https://support.1password.com/safari/)

坦白地说，我觉得在 Safari 上使用 1Password 有点麻烦(也许这只是我的问题)。您也可以完全切换到不同的浏览器，如 [Chrome](https://www.google.com/chrome/) 或 [Firefox](https://www.mozilla.org/en-US/firefox/new/?redirect_source=firefox-com) ，这可能会使使用 1Password(而不是 Keychain)保存密码的过程更加容易。
