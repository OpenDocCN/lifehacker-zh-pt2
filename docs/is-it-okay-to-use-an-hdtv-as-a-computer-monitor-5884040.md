# 可以用高清电视作为电脑显示器吗？

> 原文：<https://lifehacker.com/is-it-okay-to-use-an-hdtv-as-a-computer-monitor-5884040>

每次去买新的电脑显示器，我都会问自己同一个问题:我为什么要这样做？为什么不通过我已经有的巨大的显示器，或者说我的大屁股电视来运行所有的东西呢？



从理论上来说，通过电视播放电脑图像似乎是最好的解决方案，但在你从购物清单中划掉“全新的桌面显示器”之前，有一些因素你应该考虑。这并不是说你*不能*做这个，或者做好这个；这并不是一个在*所有*情况下都有效的完美解决方案。

### 你在用电脑做什么？

为了帮助您确定是否应该将您的 PC 连接到 HDTV，首先考虑您通常在您的 PC 上做什么。你的日常使用会受到大量变量的影响，但你有可能属于两个基本类别中的一个:普通电脑用户(T2)或娱乐用户(T4)。日常可能包括从发送几封电子邮件到整天在 Photoshop 中工作的所有事情，而娱乐意味着你的大部分时间都花在看电影或玩游戏上。

一旦你花了一点时间自我反省，让我们更深入地挖掘一下高清电视对这两类用户意味着什么:

#### **不经常玩游戏或玩流媒体的普通电脑用户**

如果你把大部分电脑时间花在浏览网上文章、阅读电子邮件、写作或其他类似的事情上，那么大尺寸高清电视的最大问题就是分辨率。记住，说到显示器， [分辨率最重要](https://lifehacker.com/is-the-multiple-monitor-productivity-boost-a-myth-5616859) 。一台 32 英寸的高清电视*可以*显示与 27 英寸显示器相同的分辨率(假设它们都是 1080p)，但放大了 5 英寸。这使得一切都有点模糊，不利于阅读。走到一个 42 英寸的 1080p 显示器前，你会讨厌你所注视的东西——它缺乏清晰度，你会质疑你的生活选择。

另一个问题是人体工程学和高清电视的庞大尺寸。正如我们在 之前概述的 [一样，你希望你的视线水平在显示器顶部下方大约两到三英寸处。除非你非常高或者坐得特别高，否则这对于大屏幕来说是很困难的。把一台 44 英寸的电视放在你的桌子上，你会伸长你的眼睛和头四处看你的数字桌面上的一切，这可能会很快变得很烦人。](https://lifehacker.com/how-to-ergonomically-optimize-your-workspace-30833302) 

尽管如此，有很多例子表明 YouTube 的观察者和程序员已经转向高清电视，而不是传统的电脑显示器，并为此大放异彩。以下是他们通常寻找的东西:

*   最低 4K 分辨率
*   最低 60Hz 刷新率
*   支持 [色度 4:4:4 二次采样](https://www.rtings.com/tv/learn/chroma-subsampling) (避免文字模糊)
*   HDMI 2.0(和)到 [实现前面提到的所有](https://hometheaterreview.com/what-you-need-to-know-about-hdmi-20/)
*   低输入延迟——事实上，只有在设置好电视后，您才能真正进行自我测试。要么读一堆关于你将要购买的东西的评论，要么找一个有强有力的退货政策的零售商，以防像你的鼠标移动这样的事情的滞后让你发疯。
*   VESA 安装。如果你把一个巨大的电视放在你的桌子上，你可能想把它放在一个更好的位置，而不是坐在一个支架上(这可能需要你稍微抬起头)。
*   内置扬声器。这个是可选的，但是如果你的电脑没有替代的声音设置，内置扬声器也无妨。

在 PC 端，你需要确保你的系统至少能以 60Hz 的频率输出 4K 的分辨率。如果您有大约第七代或第八代英特尔 CPU(如果您运行的是集成显卡)或像样的独立显卡，您的系统应该能够做到这一点，只需查看规格，看看它能做什么(在什么连接上)。你很可能会为此使用 HDMI，因为我不希望大多数现代电视都带有 DisplayPort 连接，因为大多数电视制造商不会假设你会将它连接到你的计算机上。如果你的台式机或笔记本电脑上只有有 DisplayPort，你可能需要 [一个有源适配器](https://www.startech.com/AV/display-and-video-adapters/Displayport-Adapters/dp-hdmi-2-adapter-4k~DP2HD4K60S) 。
