# 为什么你不应该让名人通过社区给你发短信

> 原文：<https://lifehacker.com/why-you-shouldnt-let-celebrities-text-you-via-community-1841259489>

定向广告，那种可能出现在你的脸书 feed 或 Twitter 时间线上的广告，会给人一种咄咄逼人的感觉；毕竟，他们经常使用个人信息出现，比如你的位置、年龄或搜索历史。由于一项新服务在社交媒体上掀起波澜，他们可能会更密切地锁定你。



最近，如果你偶然发现一条推文或 Instagram，其中一位名人发布了他们的“电话号码”,以便他们可以直接与你互动和发短信，这不是你想的那样——你没有与一位名人进行私人聊天，而是实际上*订阅了一份自动更新的简讯，将通过短信发送。*

早在 10 月，记者泰勒·洛伦茨为《纽约时报》详细描述了它是如何运作的。如果你用这个号码给一位名人发“短信”，你很可能注册了一项由名为 [社区](http://gocommunityapp.com/) 的科技创业公司运营的服务；他们代表名人和其他知名人士帮助管理与大量受众的直接消息传递，作为他们社交网络的一种“货币化”方式。

一旦你通过短信选择加入，你会收到什么样的更新？嗯，看情况。有时候，它只是一串随机的照片和文本，就像演员阿什顿·库彻给洛伦茨的那样。我注册了 *Pod Save America 的*乔恩·费儒更新，刚刚收到一堆关于他们巡演的更新。

如果你真的非常非常幸运，你可以从相关名人那里得到回复，但是不要指望这个。考虑到他们可能收到的信息数量，光是回复每一条短信似乎就需要付出巨大的努力。大多数时候，您可能只是这些自动更新的接收端。

如果你是某个名人的特别粉丝，也许注册并不是一个坏主意。但是我们会帮你一个忙，现在告诉你这可能不值得。你收到不仅仅是自动更新的文本的几率非常低，而 [社区保证](https://www.nytimes.com/2019/10/15/style/celebrity-phone-numbers.html) 他们不会把你的个人信息卖给其他第三方，嗯， [我们在](https://www.theatlantic.com/technology/archive/2018/12/facebooks-failures-and-also-its-problems-leaking-data/578599/) 之前都听说过，不是吗？
