# 红氢手机是为另一个现实制造的手机

> 原文:[https://gizmodo . com/the-red-hydrogen-one-a-phone-made-for-a-alternate-r-1830015204](https://gizmodo.com/the-red-hydrogen-one-is-a-phone-made-for-an-alternate-r-1830015204)

吉姆·詹纳德是个有趣的家伙。在开始销售摩托车零件的单人生意后，他决定转向护目镜和眼镜。一晃 25 年过去了，奥克利已经成为世界上最大的太阳镜品牌之一。2007 年，在以 21 亿美元出售 Oakley 后，Jannard 转向了他的另一家公司，同年，Red 推出了第一台 4K [影院级数码相机](https://www.red.com/shot-on-red) 。从任何角度来看，这两家公司都是非常成功的企业，所以对于他的第三个支点，Jannard 认为他会把目光放得更高，制造一款与市场上任何其他产品都不同的智能手机。但这一次，我认为吉姆可能有点走得太远了，因为红氢 one 不是制造你可能想买的东西，而是具有雄心勃勃的 3D 显示屏，感觉像是一部从不同维度传送过来的手机。

由扇形铝框架和凯夫拉尔面板(即将推出的钛包层模型)制成，氢一号看起来像一辆坦克，感觉更重。它的背部和侧面有棱纹凹槽，你可以在上面磨碎奶酪，在背部有一个红色的宝石，周围有雕刻的金属，上面写着红色媒体机器。所有这些结合起来提供了一种罕见的耐用性，这是很长时间以来我第一次觉得有一部手机永远不需要外壳来保持安全。简而言之，氢手机与现代玻璃夹层智能手机完全相反。

但古怪之处还不止于此，因为除了用于数据和充电的 USB-C 端口，Hydrogen One 还配有一个(喘气)耳机插孔和一个可以用指甲拔出的 microSD 卡托盘，而不是依赖于那些在你得到新手机时立即消失的小托盘顶杆。在它的两侧，除了分别用于调高和调低音量的按钮外，右边还有一个内置指纹识别器的锁定按钮和一个专用的相机按钮——非常适合 Red 制作的手机。

所有这些都只是 Hydrogen One 亮点功能的前奏:它的 4 视图显示屏。那么 4View 代表什么呢？我不完全确定，瑞德的网站也没有详细说明。但是，4View 应该提供的东西是相当不错的。4View 是 Red 声称的“世界上第一个全息屏幕”，其组件包括所有最热门的流行语。它由纳米技术通过独特的光场屏幕提供动力，无需眼镜即可获得身临其境的 3D 图像。(对于那些想了解 4View 实际工作原理的人，请点击 查看对这项技术的创造者之一 [的采访。)](https://www.slashgear.com/red-hydrogen-one-phone-display-light-by-leia-interview-23551068/amp/)

可悲的是，这些花哨的词语和增加的深度在正常的图片或视频中不会出现，这意味着这只能亲自体验。对我来说，Hydrogen One 的显示器真的只是一个比你在新的任天堂 3DS 上看到的稍微好一点的版本。这是真的，即使不戴眼镜，图像看起来确实会跳出屏幕，在半空中盘旋，但这不是我期待在一年多[炒作](https://gizmodo.com/red-s-first-smartphone-still-looks-like-a-fake-star-tre-1802683503) 后看到的那种革命性的显示。

在我看来，使用该技术的静态图像看起来比视频更好，照片中的雕塑或树木等线条硬朗的东西在使 4View 真正流行方面做得最好。但是仍然有很多问题。在正常的 2D 模式下，Hydrogen One 的 5.7 4 视图显示屏分辨率相对较高，为 2560 x 1440。但是当你打开 3D 时，你可以立即看到屏幕的分辨率明显降低到可以看到单个像素的程度。最重要的是，没有什么东西看起来那么清晰，总有一种微弱的彩虹色隐藏在背景中，使事物看起来更梦幻而不是真实。但我可能有点厌倦了，所以为了更好地了解其他人的想法，请查看我们的视频，看看 GMG 的同事和他们对 4View 的第一次体验(查看上面的视频)。

Red 知道单靠 4View 显示屏不足以销售这款手机，因为除了其他 Hydrogen One 用户通过 Holopix 应用程序或其他一些工具相互分享照片之外，可供查看的 3D 内容有限。氢网络应用程序提供了大约 20 个电影片段，你可以观看，而红色莱娅阁楼应用程序增加了对游戏的支持，如但仅此而已。(哦，美国电话电报公司也给从他们那里购买氢气的人免费提供 4 个版本的 *Ready Player One* 和 *Fantastic Beasts* )。因此，为了帮助充实可用的 4View 内容，Red 确保 Hydrogen One 不仅可以显示 3D，还可以捕捉 3D。

使用前后双摄像头，氢气可以拍摄 3D 静止图像和视频，尽管有一些限制。对于前置摄像头，你只能在纵向拍摄 3D，这其实没什么大不了的。但在回中，你被限制在仅仅是风景拍摄，这就更烦人了。一些问题可能源于该技术仍处于开发周期的早期，例如 4View 视频录制在剪辑中途关闭，因为手机认为外面太亮了。但即使你可以忽略这些问题，氢的标准图像质量并不令人印象深刻。

在一个光线充足的面包店，在几乎理想的条件下，Hydrogen One 的相机拍摄出的照片看起来黯淡无光，毫无生气，总体上比我从 Pixel 3 获得的照片更差。接下来，当我拍摄一些南瓜的照片时，氢南瓜在动态范围方面很挣扎，这可以从它的高光溢出和阴影中曝光不足的细节中看出。到了晚上，事情变得非常糟糕，氢一号在多个位置重复产生严重模糊和嘈杂的图像。甚至手机的相机应用程序也有一个尴尬的用户界面。要拍摄 3D 照片，你需要按下 2D 按钮来切换拍摄模式，要切换到视频模式，你需要按下相机按钮。其实意义不大，也肯定不直观。似乎也没有显示网格线来帮助构图的选项，虽然我很欣赏红色包括显示直方图的能力，但这可能是我能说的关于手机摄像头的最好的事情。

我想，在看到像 [Essential](https://gizmodo.com/the-essential-phone-is-too-expensive-to-have-a-camera-t-1798408448) 和 [Razer](https://gizmodo.com/razers-powerhouse-phone-for-gamers-is-delightfully-diff-1820817305) 这样的首次手机制造商努力提高图像质量之后，我应该不会感到惊讶。但鉴于红色的传统，我期待一个更好的表现。

理论上，所有这些都可以通过 Hydrogen One 的模块化附加系统来解决，该系统有望在某一天让手机连接到该公司超级昂贵的相机，或连接独立的传感器和相机制造商的第三方镜头，包括尼康、佳能、索尼、徕卡等。但是那些 mod 还不存在，也不知道什么时候会到。

然而，事情开始变得真正糟糕的是，对于一部售价 1300 美元的手机来说，它的规格从第一天起就过时了。它采用了去年的 Snapdragon 835 芯片，尽管 6GB 的内存和 128GB 的存储空间很好，但它仍然运行 Android 8.1，Red 表示 Android 9 要到 2019 年才会到来。(目前，它的安全补丁可以追溯到 7 月，所以希望 Red 在 11 月 2 日上市之前解决这个问题。)它的液晶显示屏也不是很鲜艳，没有现代 OLED 面板的对比度。即使是 Hydrogen One 的 4500 毫安时大电池也受到低效组件的影响，因为 Red 的手机运行时间为 11:51，比 iPhone XS Max (13:07)或 Galaxy Note 9 (14:11)少一两个小时，仅比 Pixel 3 XL (11:24)好一点。

最终，Hydrogen One 是一款非常结实的手机，内部部件陈旧，没有防水或无线充电功能，屏幕炫丽，仅比任天堂 3DS 好一点点，整体图像质量让 Red 自己的名字都感到羞愧。如果这听起来不像是在另一个现实中变出来的电话，我不知道什么才像。虽然 Red 仍有机会在未来赎回 Hydrogen One 及其技术，但要等到很多不叫 Jim Jannard 的人真正想拥有这款手机，还需要很长时间。
