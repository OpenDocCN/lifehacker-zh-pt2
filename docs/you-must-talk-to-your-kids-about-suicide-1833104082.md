# 你必须和你的孩子谈论自杀

> 原文：<https://lifehacker.com/you-must-talk-to-your-kids-about-suicide-1833104082>

作为一名家长，我的孩子——或者我认识的任何一个孩子——可能会考虑自杀，这种想法是我想推得远远的。不。天啊，不。不可能。

然而，这种心态是危险的。

因为孩子们在自杀。在青少年中，2009 年至 2017 年间，自杀死亡增加了 33%。在过去的一周，来自 [、俄亥俄州](https://www.whio.com/news/local/kettering-fairmont-mourns-death-student-from-suicide/l98jwzFkjKJK4U3vj0tANM/) 、 [加州](https://www.ocregister.com/2019/03/05/aliso-viejo-elementary-middle-schools-closed-after-apparent-suicide-by-13-year-old-boy/) 和 [纽约州](https://nypost.com/2019/03/03/nyu-student-committed-suicide-in-dorm-room-cops/) 的学生死于明显的自杀。

关于防止年轻人自杀，我们还有很多不知道的，但我们知道的是:*沉默无助于*。

对我们来说，和我们的孩子谈论自杀是很重要的，当我们这样做的时候，使用这些词:“自杀”，“自杀”，“伤害自己。”

“不要害怕直接问(你的孩子)是否在考虑伤害自己或自杀，” [谢丽尔·金](https://healthblog.uofmhealth.org/wellness-prevention/how-to-talk-to-children-and-teens-about-suicide-a-guide-for-parents) 写道，她是密歇根医学院的精神病学教授，也是美国预防青少年和儿童自杀的主要专家之一。“这些问题是在关心和希望帮助他们解决痛苦的背景下提出的。”

在一篇关于 [如何与孩子谈论自杀](https://healthblog.uofmhealth.org/wellness-prevention/how-to-talk-to-children-and-teens-about-suicide-a-guide-for-parents) (你应该完整阅读)的文章中，她提供了以下示例提示供父母使用:

*   “考虑到你现在正在处理的事情，以及学校里发生的事情，你有没有想过伤害自己或自杀？”
*   “我想知道你是否有自杀的念头。”

像这样的问题使你的孩子的经历正常化，帮助他们理解没有禁忌的想法。金建议，问完问题后，保持冷静并倾听。如果你的孩子表现出痛苦，她建议用“我很抱歉你这么痛苦”来回应。让我们想想我们能做什么。让我们看看我们能做些什么来解决这个问题。我想帮上忙。让我们得到帮助，让我们一起学习更多。”

一个常见的误区是，询问某人关于自杀的问题会让他们产生自杀的想法，如果他们还没有这个想法的话。几项研究发现， [这不是真的](https://www.nimh.nih.gov/health/publications/suicide-faq/suicideinamericafaq-508_149986.pdf) 。在一篇名为 [【谈谈吧】](https://thebloggess.com/2019/02/18/have-the-talk/) 的漂亮博客文章中，作者珍妮·劳森进一步分析了为什么这个假设是有缺陷的。

> 和你的孩子谈论性不会让他们发生性行为。谈论美洲驼并不意味着它们就是美洲驼。和你的孩子谈论自杀不会让他们自杀。和你的孩子谈论精神疾病并不会让他们患上精神疾病。然而，它确实给了他们工具来帮助识别那些可能会使他们困惑或害怕的事情。这可能有助于他们认识到自己或朋友身上的东西。这可以拯救一条生命。

你也可以鼓励你的孩子和他们的朋友谈论自杀，并给他们这样做的工具。凯瑟琳·珀尔曼是一位育儿专家，也是蔻驰 家庭的创始人，她告诉我，“孩子们经常知道他们的朋友有些不对劲。他们看到社交媒体上的帖子或群聊中的文本。但他们不知道如何处理。”

她建议教孩子们自杀的警告信号，并告诉他们可以问这个问题。事实上，它可以挽救一条生命。

皮尔曼说:“反应过度总比错过干预的机会好。”

如果你正在考虑自杀，或者担心你的朋友或爱人，请拨打国家自杀预防热线 1-800-273-8255。
