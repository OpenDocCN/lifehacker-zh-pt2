# 防止玉米饼壳和融化的奶酪一起裂开

> 原文：<https://lifehacker.com/melt-cheese-in-crunchy-taco-shells-to-keep-them-from-sp-1836828069>

他们有很多名字。“脆皮玉米卷”、“盎格鲁玉米卷”，甚至还有“妈妈玉米卷”，但不管你怎么称呼它们，脆皮玉米卷提供了一种独特的——有些令人沮丧的——家庭用餐体验。是的，我说的是可怕的壳裂。



在烤箱里加热会有帮助。热量激活了所有的油脂，使贝壳变得更加柔韧，不容易破碎。但是，如果你真的想强化酸奶油和调味牛肉输送系统的结构，在填充之前在外壳中融化一点奶酪。

这个想法——来自 Reddit 用户[u/hy tec 9](https://www.reddit.com/user/hyteck9/)——很好，因为它给了我一个借口在我的玉米卷上放更多的奶酪，但它也很好，因为它很有效。虽然我的壳确实破了一点点，但是裂缝是沿着壳的顶部发生的；奶酪中的额外脂肪使底部变得格外柔韧，直到最后一口都保持完整。

用奶酪做壳胶很简单。沿着底部撒一点切碎的东西，沿着烤箱安全容器的边缘支撑它(我用的是铸铁锅)。在华氏 350 度的温度下烘烤，直到奶酪融化，然后像往常一样加入更多碎奶酪，尽情享用。
