# 免费给医院里的孩子寄一张情人节卡片

> 原文：<https://lifehacker.com/send-a-valentine-to-a-child-in-the-hospital-for-free-1832363184>

不要强调情人节或者你的生活中是否有爱，试着送一点爱给生病的孩子。



许多个体医院运行程序，在那里你可以发送带有鼓励信息的个性化卡片给正在为节日康复而奋斗的孩子们。SlickDeals 有 [编了一批最大的](https://slickdeals.net/f/12738058-children-s-hospitals-send-valentine-s-day-card-to-a-sick-child-free)；卡片的类型各不相同，但一般来说，医院会为你提供一些选择，一个写你的名字和电子邮件地址的地方，以及一个个性化信息的部分。

有一个小缺点:

> 通过提供您的电子邮件地址，一些(并非所有)组织*可能会*自动注册(或自愿要求)参与者接收来自您提供信息的组织的定期电子邮件更新。您可以随时选择不订阅或取消订阅(如果自动注册)。

如果你想避免麻烦，你也可以使用一次性电子邮件。卡片本身将只显示你的名字，而不是你的电子邮件地址，并将“随机分发给病人或在整个医院展示”，而不是发给特定的个人。试着给一群医院，而不仅仅是你所在社区的医院，发送一个乐观的信息，并且知道你正在让某人的情人节变得更加愉快。
