# 立即注册投票

> 原文：<https://lifehacker.com/register-to-vote-now-1838360979>

今天是全国选民登记日，如果你以前没有登记过，这是在总统选举前登记投票的绝佳机会。今天，全国各地的志愿者都在协调努力，让人们意识到投票的重要性，以及报名参加投票有多简单*。根据 [NVRD 的网站](https://nationalvoterregistrationday.org) 显示，去年的这一天有超过 80 万选民登记——随着 2020 年的临近，这个数字可能会继续增长。*

*要登记，只需访问 [全国选民登记日的网站](https://nationalvoterregistrationday.org/register-to-vote/?source=nvrd-topnav) ，填写你的地址、邮箱、电话号码；您将被重定向到您所在州的注册网站，在那里您将提供类似的详细信息。(整个过程需要两分钟。)如果你没有驾照或非驾驶州 ID，你可能需要额外填写一份申请来登记投票。关于你所在州的这些细节和截止日期，一定要查看 [Vote411 的网站](https://www.vote411.org/) 。*

*如果你想亲自登记，找一个今天在你附近发生的选民登记活动。注册后，继续在社交媒体上分享消息，这样如果你的朋友和家人还没有注册，他们也会这样做。*

*请记住，如果你之前已经登记投票，你必须*而不是*重新登记，除非你在过去四年中搬家、改名或者没有投票。如果你在 2016 年总统选举中投票，并且没有改变地址，你可能是安全的。当然，你可以通过在线搜索“选民登记状态”和你所在的州，输入你的姓名、地址、生日和任何其他必要的信息，来检查你现有的选民身份。*
