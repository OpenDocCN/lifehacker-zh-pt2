医疗保健代理人是指在您自己无法做出决定的情况下，可以为您的医疗保健做出决定的人。它有时被称为持久医疗委托书或医疗保健代理。

它经常与 [生前遗嘱](https://twocents.lifehacker.com/how-to-create-a-will-1797833946) 捆绑在一起，但它是一个可以独立存在的单独文档。

当需要讨论治疗方案和重大决策(如维持或终止生命支持)时，你的代理人是医疗专业人士的首选。你的代理人不需要和你有血缘关系，但应该是能为你争取最大利益的人。他们也应该是住得离你比较近的人，这样如果需要的话，他们可以很容易介入。

即使你不属于冠状病毒爆发期间的高风险人群，如果未来出现困难情况，拥有这些文件之一也可以建立一些秩序。提前就您的医疗愿望和您信任的执行人做出一些决定，可以确保您的愿望在您昏迷或无法说话时得到尊重。

如果您没有选择医疗保健代理人，负责您的护理决定的人将取决于您所在州的法律，但通常默认为配偶、父母或子女或兄弟姐妹。

“当谈到冠状病毒时，你和你爱的人不需要使用医疗代理的可能性仍然存在，”家庭财务应用程序的编辑主任艾莉森·卡德说。“但即使在最好的情况下，这也是一种简单的方式，让你可以与你爱的人谈论对你来说重要的事情，并对自己的决定有发言权。这是另一项任务，我会把它放在“无论如何都要做好”的桶里，所以冠状病毒可能只是一个最终完成它的好借口。”(Fabric 提供了一个在线意愿创建工具。)
