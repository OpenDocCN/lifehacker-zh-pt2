# Edison Mail 的安全灾难对您的电子邮件意味着什么

> 原文：<https://lifehacker.com/what-edison-mails-security-disaster-means-for-your-emai-1843526187>

流行的第三方电子邮件客户端 Edison Mail 最近允许少数 iOS 用户访问不属于他们的电子邮件地址。这不是黑客攻击或泄露，而是一个 bug——而且是一个相当大的 bug。



这个错误是由最近的一次更新引起的，Edison 向 T2 保证，T3 只推给了“一小部分”用户。此次更新包括一个新的账户同步功能，无论出于什么原因，都可以将随机的电子邮件账户同步到无关的数据，并允许用户完全访问陌生人的收件箱、联系人和其他信息，而无需登录。只是有点...发生了。

### 爱迪生邮件用户现在需要做什么

Edison 取消了其糟糕的更新，并将 Edison Mail 恢复到之前的版本。如果你仍然能够在 iOS 应用程序中看到其他人的电子邮件，那么你不应该这样做，但请检查应用程序商店，确保你有该应用程序的所有最新补丁。

这个漏洞只影响了 iOS 用户，但我们只是在谈论那些可以*看到* *他人邮件的人。*暴露的账户可能来自连接到任何 Edison Mail 应用程序的任何平台，因为 Edison Mail 是跨平台的，允许您将 Gmail、iCloud、微软和其他电子邮件服务的多个电子邮件账户连接到一个收件箱。
