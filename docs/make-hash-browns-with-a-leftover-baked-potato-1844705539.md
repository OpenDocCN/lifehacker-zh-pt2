# 用剩下的烤土豆做土豆煎饼

> 原文:[https://life hacker . com/make-hash-browns-with-a-left-baked-potato-1844705539](https://lifehacker.com/make-hash-browns-with-a-leftover-baked-potato-1844705539)

我知道大多数读到这里的人从未听说过“剩下的烤土豆”，老实说，我很喜欢你这么说。但我是一个意志薄弱的人，她的眼睛远远大于她不小的肚子，我已经知道过度订购，过度购买，过度准备，特别是如果我订购，购买，或准备牛排晚餐。

对我来说，牛排晚餐有三个主要组成部分(不包括马丁尼):带骨里脊肉、大块红褐色肉和楔形沙拉。这三种食物本身都很丰盛，所以——*耶*——有时候我会吃剩菜，有时候那些剩菜大部分是，但不完全是，刮下来的——烤土豆。如果你从未经历过这些，那么做得很好，但是你错过了。大部分--刮出来的烤土豆是极好的早餐土豆，如果你一口气吃完你的土豆，你永远不会有这种体验。

这并不令人惊讶。最好吃的脆皮土豆都是 [煮两次](https://lifehacker.com/how-to-make-restaurant-worthy-fries-1832361045) 。炸薯条是这样，烤土豆也是这样。这就是为什么冷冻土豆煎饼比油炸过的方块家庭薯条好吃得多，这也是为什么剩下的烤土豆薯条炸得如此漂亮(而且很快)。大部分工作已经完成；你所要做的就是把你的剩菜脆成金黄色的土豆泥状，和一些鸡蛋一起吃。

准备工作很简单:把你晚餐剩下的土豆切成一口大小的块，然后放入一两汤匙培根油(或其他高热友好油脂)中煎炸。如果一小块奶酪、青葱和酸奶油进入锅中，融化并与油脂融合，那就更好了。将土豆块煎至金黄色且超级酥脆，然后和一起享用一份完整的早餐。它们在早餐玉米卷里特别好吃。
