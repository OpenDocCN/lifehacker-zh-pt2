# 如何在家中使用旅行配件

> 原文：<https://lifehacker.com/how-to-use-your-travel-accessories-at-home-1843134914>

虽然我们中的大多数人暂时不坐飞机，不自驾游，也不入住酒店，但我们仍然可以利用我们收集的所有旅行配件，让我们的生活在旅途中更加轻松。

两周前，我们在积分盖伊 [的朋友推荐了](https://thepointsguy.com/guide/travel-accessories-use-at-home/) 特定的旅行配件，可以在家里购买和使用，但当你可以使用你已经拥有的产品时，就没有必要购买特定的物品。如果你的空行李箱里有以下任何东西，现在就把它拿出来使用。

### **用压缩袜让你的血液流动**

压缩袜对你的腿和脚踝施加温和的压力，这有助于促进血液流动。当你连续几个小时坐在飞机上的时候，它们是很棒的；当你在家工作时(尤其是当你不得不限制外出时)，它们对于长时间坐在办公桌前同样有用。

### **重新路由您的 wifi——户外**

如果你在家工作，你可能厌倦了白天盯着同样的四面墙——但是如果你的 wifi 没有延伸到你可能足够幸运拥有的户外空间呢？

忙碌的旅行者的第二天性就是把你的手机作为无线热点。通过将你的手机连接到你的笔记本电脑，无论你去哪里，你都会有一个连接。另一个选择是使用旅行路由器——一种手持设备，允许你连接到现有的信号以扩展你的无线覆盖范围。

### **用包装立方体包装你的工作站**

在家工作既麻烦又有压力。一天下来，你的家庭办公室(或者厨房柜台、餐桌或床)可能最终看起来像是被海啸袭击了一样。一个解决方案:包装立方体。传统上用于整理行李的包装立方体可以在家里使用,当你需要同样层次的组织来整理你的无数工作项目时。考虑:

*   为你的电子产品和电源线指定一个包装立方体。这将防止你的电线缠绕在一起，保护你的电子产品。
*   将重要文档组织到单独的多维数据集中。
*   使用较小的立方体来存放钢笔/铅笔、回形针以及其他办公用品和你在工作中使用的附件。
*   将你的立方体堆在桌子或书桌的一角，以保持该区域的整洁，减少混乱。

一切都将在第二天、准备好，如果你的工作空间是一个不断移动的目标，你可以轻松地运输它们。

### 给那几绺(头发)涂上肥皂沫

如果你和我一样，喜欢收集酒店的洗发水/护发素和乳液(你知道，好闻的那种)，现在是时候使用它们了。你很快就会登上飞机或者去泡温泉的可能性很低，但是当然你可以犒劳自己去一次自己的化妆温泉。

随着我们所有的发型迅速变成禁忌， 使用你在储藏的洗发水和护发素可能会给你在一天或一周结束时需要的呼气时刻。当你做完这些后，拿起你一直留着的一团洗手液，因为它对美国运输安全管理局来说足够小了，它会让你洗得太多的手恢复活力。
