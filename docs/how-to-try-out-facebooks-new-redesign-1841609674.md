# 如何尝试脸书的新设计

> 原文：<https://lifehacker.com/how-to-try-out-facebooks-new-redesign-1841609674>

脸书正在让用户测试其即将到来的重新设计的测试版，预计将于今年晚些时候全面推出。你可以通过登录脸书并选择你的新闻订阅顶部的“查看脸书测试版”来启用新的外观——尽管该选项可能不是对所有人都可用。

Watch

如果您在新闻订阅中没有看到该选项，您也可以尝试单击右上角的箭头图标，并从下拉菜单中选择“查看脸书测试版”。

更新后的用户界面重新排列并简化了脸书的界面，使其更容易打开 Messenger 对话，并加入了新的图标和按钮。第一次启用测试版设计时，你会得到一个快速浏览和变化解释，如果你感兴趣，还有机会切换到黑暗模式。

如果你打开了测试版的重新设计，但没有感受到新的外观，你可以撤销更改，轻松恢复“经典”的脸书用户界面。只需点击页面右上角的个人资料图片，打开快捷菜单，然后选择“切换到经典脸书”当你第一次打开新设计时，脸书实际上向你展示了如何做到这一点，但如果你忘记或没有抓住设置在哪里，这是值得指出的。

像这样的变化可能会导致分裂，所以脸书让用户回到他们想要的旧网站设计是很酷的。你还能继续使用旧界面多久还不清楚，但我怀疑只要新设计处于“测试”阶段，就有可能恢复使用旧界面。
