# UPS 将为您打包并返还您的旧互联网路由器

> 原文：<https://lifehacker.com/ups-will-pack-and-return-your-old-internet-router-for-y-1838972844>

如果你要更换互联网服务提供商，升级到新设备，或者有任何其他原因要运回你的 ISP 提供的调制解调器和路由器，那么大多数最大的 ISP 会让你把它放在 UPS，甚至不用装箱。这里有一些可以，一些不可以，以及在哪里可以得到你需要的设备。



### 康卡斯特

允许无箱 UPS 卸货。 [按照这些指示](https://www.xfinity.com/support/articles/returning-your-equipment) ，在标题“顺便去一家当地的 UPS 商店”

> 您可以将您的设备带到商店，由 UPS 代表将您的设备打包并寄回给我们，或者您可以自己用一个纸板箱和您在家里打印的预付 UPS 运输标签包装设备。

### AT&T U-verse

允许无箱 UPS 卸货。 [跟着左上角的](https://www.att.com/media/att/2012/support/pdf/att121140829-2_wireless_gateway_swap_3800_to_3801_without_voice.pdf) 走。

> 将原始 Wi-Fi 网关和电源线带到最近的 UPS 商店。带上装箱单上的美国电话电报公司账号。

> UPS 将扫描您的设备，提供收据，包装并将设备运送到美国电话电报公司，而无需您支付任何费用。

### 频谱(又名宪章又名时代华纳有线电视)

允许无箱 UPS 卸货。 [按照这些方向走。](https://www.spectrum.net/support/tv/returning-your-spectrum-equipment)T3】

> UPS 商店将免费处理包装和运送回我们。只需带上您的设备，告诉 UPS 您是 Spectrum 客户，他们会处理剩下的事情！记住保留收据以便跟踪。

### 威瑞森电信

要求您使用“退货工具箱”来处理 UPS 投递，但您可以通过电话获得。 [按照这些方向走。](http://www.verizon.com/support/smallbusiness/tv/fiostv/general+support/account+issues/questionsone/129064.htm)T3】

> 当您遵循威瑞森提供的 Fios 退货工具包中的说明时，退货是免费和简单的。贴上更换/断开套件中包含的预付运输标签。把它放在最近的 UPS 商店。
> 
> 如果您需要额外的退货/换货箱，请致电 1.800.VERIZON (1.800.837.4966)。

### 边界

需要一个返回箱(根据要求提供)用于 UPS 的卸载。 [遵照这些指示。](https://frontier.com/helpcenter/categories/account/manage-my-account/how-to-return-frontier-equipment)T3】

> 请致电 1.800.921.8101，告诉我们您希望对您的边境服务做出何种改变。我们将确定您拥有的所有前沿设备，如宽带路由器、DVR 和机顶盒，并告诉您需要将哪些物品返还给我们。我们将通过 UPS Ground 向您发送一个预付费邮箱和写有完整地址的标签。

我们没有发现[century link](https://www.centurylink.com/home/help/internet/modems-and-routers/print-a-return-label-and-send-back-your-modem.html)[Sparklight](https://support.sparklight.com/hc/en-us/articles/115010179727-How-do-I-cancel-disconnect-or-change-my-service-)[Consolidated](https://www.consolidated.com/support/locations)[Mediacom](https://www.allconnect.com/providers/mediacom/customer-service)[TDS](https://tdstelecom.com/support/account/account-cancellation.html)或 [Windstream](https://www.windstream.com/support/my-account-and-payments/How-do-I-return-my-Windstream-equipment) 的无箱式 UPS 落客的公开信息。 [Cox 的退货页面](https://www.cox.com/residential/support/returning-your-cox-equipment.html) 提到 UPS 送件是“搬到 Cox 未服务区域的客户”的一个选项

ISP 让你在取消服务后迅速归还旧设备，否则他们会向你收费，这也是 Lifehacker 建议你 [购买自己的调制解调器和路由器](https://lifehacker.com/is-it-ever-a-good-idea-to-rent-a-modem-5949278) 的一个原因。

当您放下设备时，一定要有一个跟踪号码，以防 ISP 丢失您的设备并试图向您收取费用。记住不要把你的愤怒发泄在客服代表身上，他今天比你更糟糕。
