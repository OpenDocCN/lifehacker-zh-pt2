# 如何以及为什么开始写日记

> 原文:[https://life hacker . com/why-you-should-keep-a-journal-and-how-start-yours-1547057185](https://lifehacker.com/why-you-should-keep-a-journal-and-how-to-start-yours-1547057185)

一些历史上最有影响力的人详细记录了他们的生活，包括查尔斯·达尔文、玛丽·居里、托马斯·爱迪生和弗里达·卡罗。这些日记有两个目的:为后代留下永久的记录，并且，对于写日记的人来说，可能是一种宣泄。

Watch

即使你认为你两者都不需要，写日记也有很多好处，你可以从今天开始享受。也许你想给你的孩子留下一些东西来讲述你的故事和你的成就。也许你更实际，想要一种方法来利用你的创造力。也许你只是想要定期写作带来的宣泄。不管是什么，这些都是很好的理由。这就是为什么你可能想要定期坐下来记下你的想法。

### **经常写作有益于精神健康**

写作对你的心理健康有奇效。除了让你的创造力源源不断——这是一个单独的话题，我们很快就会谈到——定期写作可以给你一个安全的、宣泄日常生活压力的阀门。我们已经从创造性写作的角度讨论了在 之前写作的一些 [精神和情感上的好处——但是你不一定要写小说才能得到它们。](https://lifehacker.com/harness-the-mental-creative-and-emotional-benefits-of-5855019)

例如，我们提到过 [记牛逼日志可以为你的自尊创造奇迹](http://lifehacker.com/keep-a-journal-of-awesomeness-to-boost-your-self-esteem-5877106) 。定期写作不仅让你感觉良好，还能帮助你在一个安全的环境中重新体验你所经历的事件，在那里你可以没有恐惧或压力地处理它们。

根据 [心理中枢](https://psychcentral.com/lib/the-health-benefits-of-journaling/) 记日记可以帮到你:

*   阐明你的想法和感受
*   更好地了解自己
*   减轻压力
*   解决与他人的分歧
*   更有效地解决问题

事实上，有如此多的数据表明写日记对精神和情感有好处，以至于顾问、社会工作者和治疗师经常鼓励他们的病人这样做。 [这项来自《精神病治疗进展](http://apt.rcpsych.org/content/11/5/338.full)*杂志的研究是一项伟大的实验，也是对当前该主题研究的坚实总结。研究人员指出，三到五次 15-20 分钟的时间足以帮助研究参与者处理创伤、压力或其他情绪事件。*

*写日记对患有严重疾病的人特别有效，比如癌症 。事实上，这种做法很受欢迎， [有一个日志治疗中心](https://journaltherapy.com/) 致力于定期写日志对心理健康的好处，包括治疗和个人设置。*

*这不仅仅是你写的东西。你如何写作也是一个因素。 [爱荷华大学的这项研究表明](http://transformationalchange.pbworks.com/f/stressjournaling.pdf) 记录压力事件有助于参与者处理他们经历的事件。然而，关键是关注你在想什么和感受什么，而不是仅仅关注你的情绪。简而言之，当你讲述你的个人故事时，你会得到写日记的最大好处，而不仅仅是写下你自己的感受。这是一个很好的例子，说明讲述你自己的个人故事可以给你的幸福带来巨大的变化。*

### *写日记有助于发挥你的创造力*

*写日记的创造性好处也是有据可查的。你可能听说过，提高写作水平的最好方法就是坚持写下去。这是真的，但是好处不仅仅是创造更好的句子。比如 [定期写作可以帮助你学会有效地处理和沟通复杂的想法](http://www.edutopia.org/blog/writing-executive-function-brain-research-judy-willis) 。它还可以帮助你记忆重要的信息，并激发新的想法。换句话说，写下你的经历不仅有助于你处理它们，还有助于你发现乍看之下可能不明显的机会。它还能帮助你学会将复杂的经历分解成相关的、有用的、有条理的信息。*

*即使你不认为自己发生了什么特别的事情，记日记这个行为本身就能帮助你头脑风暴 。你有多少次发现自己写的东西表面上看起来很枯燥，但在你处理它的时候却把你引向了其他思想、观点和记忆的蜘蛛网？每当你坐下来的时候，有规律的写作就会打开那些机会的大门。*

### ***即使你不做创造性的工作，定期写作也有实际的好处***

*定期写作也是有用的，可以提醒你犯过的错误，你引以为豪的成就和你想要记住的伟大时刻。例如， [记工作日记可以作为对错误和成功的跟踪记录](https://lifehacker.com/keep-a-work-diary-to-minimize-mistakes-and-document-suc-5816473) 。当你情绪低落时，这些书面记录会派上用场，但当你感到失落时，它们也能帮助你调整你的个人航向。拿起你的工作日记，回顾一下你做得很好的事情——你也许能找出一个你想在职业生涯中遵循的模式。*

*同样，那些成就和令人敬畏的时刻不仅仅提升了你的自尊，在和老板谈论加薪 的时候，它们给了你加薪或升职的充分理由。你不需要成为一个有创造力的工作者来欣赏回顾你做得好的事情，以及你需要继续努力的事情。在别人指出你的错误之前看到它们是一件很棒的事情，记录你的成就可以确保它们永远不会被忽视。*

*常规写作不仅仅适用于工作。写日记是培养良好习惯的好方法，因为它迫使你意识到自己的行动和行为。如果你想注意你吃的东西，记食物日记是保持密切关注的好方法——一种被证明可以帮助人们吃得更健康的食物。同样， [只需写下发生在你身上的积极的事情](http://lifehacker.com/write-down-three-positive-things-about-each-day-every-5975693) 或 [跟踪你的情绪](http://lifehacker.com/fill-out-this-one-minute-form-every-day-and-find-out-wh-5901651) 可以帮助你识别生活中可以重复的好模式，你应该腾出时间去做——更不用说那些让你感觉糟糕或让你失去游戏的事情了。*

### ***你应该选择哪种媒体，为什么选择***

*一旦你决定写日记，你的下一个决定就是写日记的媒介。你有很多选择，对一个人有效的对另一个人无效。你必须选择一个最适合你的。以下是几个选项:*

#### ***纸质笔记本***

*如果你喜欢写下自己想法的感觉，纸质笔记本可能是你最好的选择。写纸质日记让你对自己的写作有完全的物理控制，它给你最大的隐私，因为当服务关闭或受到威胁时，你的日记被“黑客攻击”或“丢失”的可能性很小。然而，纸质日志意味着你没有备份以防你的工作发生意外——盗窃、火灾，或者只是一个丢失的背包意味着你的日志永远消失了。*

*如果你不仅仅想要一个普通的空笔记本，如果你已经在使用纸质笔记本做待办事项和笔记，那么 [子弹日记生产力方法](https://lifehacker.com/the-bullet-journal-productivity-method-empowers-your-pa-1169313228) 非常适合，前面提到的[八九不离十](http://www.sorta.la/) 有独特的可移动页面的笔记本。如果你担心自己太忙而没有时间写日记，考虑一下五分钟日记，这是一个纸质笔记本，它会给你一句激励性的话，然后给你每天写一些提示，比如“今天我很感激”，“什么会让今天变得很棒？”和“今天发生的 3 件大事”*

#### ***日志和日记应用***

*如果你无法将自己与手机或笔记本电脑分开，有很多应用程序承诺隐私和安全，以及一个伟大的写作环境。在 之前，我们已经推出了一些 [，但其中一些突出的应用包括](http://lifehacker.com/how-can-i-keep-a-personal-private-journal-online-5975750)[【Penzu】](http://penzu.com/)，这是一款带有移动应用的在线私人日志应用，以及 [Day One](http://dayoneapp.com/) ，这是一款好看的 iOS/OS X 应用，可以感知位置，让你添加照片等等。如果你喜欢免费和开源，试试 [RedNotebook](http://rednotebook.sourceforge.net/) 。这是一个奇妙的 [维基风格的日志工具](http://lifehacker.com/rednotebook-is-a-wiki-style-journaling-tool-5688063) 它是跨平台的。*

*当然，你根本不用用 app。你可以在 Dropbox 中保存一个加密的文本文件，使用 [Evernote](http://evernote.com/) 或 [Google Keep](http://keep.google.com/) ，或任何其他你喜欢的笔记应用程序。你甚至可以用 [使用](http://lifehacker.com/how-to-automatically-archive-your-life-with-ifttt-and-e-5935658) 任何你喜欢的工具 [来运行你自己的定制日志系统](http://lifehacker.com/roll-an-automatic-journal-with-ifttt-drafts-instagram-5993748) ，但是请记住，你越是自动化这个过程，你实际上就越少*记录日志*，所以你得不到完全相同的好处。*

#### ***写博客***

*写博客是 [获得写日记](http://www.scientificamerican.com/article/the-healthy-type/) 好处的另一个好方法，不管你是开始为自己扬名，还是只是公开表达你的想法和感受。写博客为尽可能多的读者打开了大门，但同时也牺牲了隐私。如果这是您的首选路线，您有大量的工具和主机可供选择，包括免费的和付费的。*

*免费选项很多，像 [WordPress](https://wordpress.com/create/?sgmt=gb&utm_source=google&utm_campaign=google_wpcom_search_brand_desktop_us_en&utm_medium=paid_search&keyword=wordpress&creative=353006487005&campaignid=998785131&adgroupid=53026924047&matchtype=e&device=c&network=g&cpn=998785131&device=c&pl=&targetid=kwd-295456403946&locationid=9004331&gclid=Cj0KCQjwn_LrBRD4ARIsAFEQFKtmSIID-4asbEIgjcV9yQE1YVzZT3ULPCpZ4SEw_uQKVU3JTleT38MaAhv0EALw_wcB&gclsrc=aw.ds) ， [Tumblr](https://www.tumblr.com/) 和 [Medium](https://medium.com/) 。它们都提供不同的外观，迎合不同的观众，为不同类型的人设计。无论你选择什么，保持个人博客可能不会附带写作提示或花哨的移动应用程序(尽管有些会)，但它们可以附带社区，以及与世界分享你的故事的选项。*

*无论你选择哪种方式写日记，都有明显的好处。你不必成为思想领袖、著名艺术家、政治家、科学家或名人，你的思想和经历才有价值——任何人都可以从中受益。*
