# 我们最喜欢的 12 个 Chrome 扩展

> 原文:[https://life hacker . com/our-12-favorite-chrome-extensions-1844729409](https://lifehacker.com/our-12-favorite-chrome-extensions-1844729409)

Chrome 网上商店 [的确是一团糟](https://lifehacker.com/stop-installing-extensions-from-the-chrome-web-store-1844678014) ，但这并不意味着你*不应该*使用合法的扩展来改善你的浏览器体验——无论是在 [谷歌 Chrome](https://lifehacker.com/why-chrome-uses-so-much-freaking-ram-1702537477) *还是*资源更友好的替代品，微软的[Edge Chrome](https://lifehacker.com/our-favorite-tips-for-microsofts-new-edge-chromium-brow-1841041653)。

虽然让你的浏览器充满你能得到的所有有用的扩展很有诱惑力，但关于“资源”的那一点并不仅仅是我开玩笑的蹩脚尝试:在某些时候，你会想要对你的选择有战略性，因为你可能不想 [让你的浏览器慢下来](https://lifehacker.com/use-google-chrome-s-task-manager-to-figure-out-what-s-s-1823227377) 。我觉得我现在有太多的扩展了，我希望这篇文章能说服我从浏览器中删除一些。

也就是说，我认为每个人都应该知道一些扩展。它们对你来说可能是显而易见的；它们可能是全新的。你必须从某个地方开始，这就是为什么我要分享我一直使用的扩展，希望它能给你一些宝石(或填充套件)来尝试。

如果你有关于其他扩展的建议，你会不惜任何代价捍卫，请在评论中留下它们！我一直在寻找下一个伟大的扩展，我会非常高兴地看看它并与你的 life hacker 粉丝们分享——当然，如果它很棒的话。

### [**Chrome 远程桌面**](https://chrome.google.com/webstore/detail/chrome-remote-desktop/inomeogfingihgjfjlpeplalcfajhgai?hl=en)T5】

让我们从这里开始。还记得以前，如果你想远程拨号进入你的系统，你必须在电脑上安装烦人的 VNC 应用程序吗？如果你想在远程计算机上移动鼠标，就像坐在屏幕前一样打字，你可能必须想出一种方法来穿透你家或办公室的路由器。你认为这些年来我们有多少个动态域名账户？

回到今天，对于拥有多种设备的人来说，没有一个扩展比 [**Chrome 远程桌面**](https://chrome.google.com/webstore/detail/chrome-remote-desktop/inomeogfingihgjfjlpeplalcfajhgai?hl=en) 更有用了。将它安装在你的各种台式机或笔记本电脑上，确保这些系统链接到你的谷歌账户，并将它们设置为主机——这实际上只需要点击一两个按钮并设置 PIN 码。然后你可以使用 [这个有用的网站](https://remotedesktop.google.com/access) 来访问这些系统，无论它们在哪里，假设它们是开机的。你甚至可以使用一个 [Android](https://play.google.com/store/apps/details?id=com.google.chromeremotedesktop&hl=en_US) 或 [iOS](https://apps.apple.com/us/app/chrome-remote-desktop/id944025852) 应用程序粗略地点击你的屏幕，调出你需要启动的任何程序。

这是*分机，如果我要出城，出于任何原因需要拨回我的家庭桌面，我会使用它——当下载完成时关闭它，或者确保我已经加载了我最喜欢的闲置游戏(并设置显示器在一分钟左右后关闭)。你可能不会每天都使用 Chrome 远程桌面，但你会很高兴当你需要它的时候它就在那里。*

 *### [**1 密码**](https://chrome.google.com/webstore/detail/1password-extension-deskt/aomjjhallfgjeglblehebfpbcfeobpgk)T5】

您不需要*使用*1 password 作为您的密码管理器。有替代方案；这正是我过去多年来一直坚持的。我喜欢 [**1 Password 的**simple Chrome extension](https://chrome.google.com/webstore/detail/1password-extension-deskt/aomjjhallfgjeglblehebfpbcfeobpgk)，因为它可以方便地访问我多年来保存在应用程序中的许多登录名和密码。该扩展是 1Password 的 [桌面应用](https://1password.com/downloads/) 的配套产品，您也需要在 PC 上安装该应用。否则，您将需要独立的[**1 password X**](https://chrome.google.com/webstore/detail/1password-x-%E2%80%93-password-ma/aeblfdkhhhdcdjpifhhbdiojplfjncoa)扩展名来代替。

### [**谷歌邮件检查器**](https://chrome.google.com/webstore/detail/google-mail-checker/mihcahmgecmbnbcchbopgniflfhgnkff)T5】

假设你使用 Gmail——谁不使用 Gmail？—那么 [**谷歌邮件检查器**](https://chrome.google.com/webstore/detail/google-mail-checker/mihcahmgecmbnbcchbopgniflfhgnkff) 就是你浏览器的必备扩展。安装它，它会在你的主工具栏上弹出一个小图标，告诉你有多少未读的 Gmail 邮件。就是这样。你不能改变一个选项，也不能以任何方式改变计数，所以让这激励你去零收件箱。

### [](https://chrome.google.com/webstore/detail/enhancer-for-youtube/ponfpcnoihfmfllpaingbgckeeldkhle?hl=en-US)**增强器**

**大概有 50 种不同的 Chrome 扩展可以用来定制你的 YouTube 体验。我试过很多，我发现自己因为各种原因讨厌它们——它们停止工作，它们杂乱，它们的用户界面很糟糕，它们的设置很难弄清楚，它们感觉很糟糕，等等。**

**在过去的一年里，我一直坚持使用的是 YouTube 的[](https://chrome.google.com/webstore/detail/enhancer-for-youtube/ponfpcnoihfmfllpaingbgckeeldkhle?hl=en-US)**增强器，因为它满足了我对这种扩展的大部分要求。它让我跳过烦人的 YouTube 广告；每当我观看视频时，它都会自动让播放器进入更大的影院模式；而且它甚至让播放器变成了更小的画中画显示器，这样当我向下滚动阅读平淡的 YouTube 评论时，我仍然可以观看视频；它让我自动默认任何视频的最高质量播放。****

****它的设置菜单虽然有点复杂，但很有意义，你甚至可以用它来调整 YouTube 的颜色和主题，编辑默认播放速度，并用鼠标滚轮控制音量。很好。****

## [**链接到文本片段**](https://lifehacker.com/how-to-create-a-bookmark-that-jumps-to-specific-text-on-1844113624)

这个 [我在](https://lifehacker.com/how-to-create-a-bookmark-that-jumps-to-specific-text-on-1844113624) 之前已经讲过了，就不多赘述了。 [**链接到文本片段**](https://chrome.google.com/webstore/detail/link-to-text-fragment/pbcodcjpfjdpcineamnnmbkkmkdpajjg) 扩展允许你创建你自己定制的锚链接——超链接将指引 Chrome(或 Edge Chromium)用户到网站上文本的确切位置。它非常有用，尤其是如果你是学生、研究人员或者其他人，你可能不想每次都在巨大的文本流中寻找特定的段落。

### [OneTab](https://chrome.google.com/webstore/detail/onetab/chphlpgkkbolifaimnlloiipkdnihall?hl=en)

哦是的。如果你厌倦了处理标签混乱——这正是为什么我现在不给你看我的完整浏览器的截图——你绝对需要[**【OneTab**](https://chrome.google.com/webstore/detail/onetab/chphlpgkkbolifaimnlloiipkdnihall?hl=en)。这种扩展是生产力的救星。有了它，你可以将所有打开的标签转储到一个单独的标签列表中，然后保存在扩展中或者导出到它自己的列表中。HTML 文件 [妥善保管](https://lifehacker.com/organize-your-browser-tabs-with-onetab-but-back-them-u-1837740779) 。虽然我仍然推荐给你想保留的网站做书签，但 OneTab 至少可以帮助你更好地管理你每天(或每周)的噩梦，还可以节省你系统上的宝贵资源。

## [会话好友](https://chrome.google.com/webstore/detail/session-buddy/edacconmaakjimmfgnblocblbcdcpbko?hl=en)

你会说“救命恩人”吗？有时候你的浏览器会崩溃；这是软件的本质。您通常会收到恢复之前打开的标签页的通知。然而，有时这个过程可能会出一点差错，实际上您最终什么也恢复不了。这就给你留下了一个令人不快的任务，你要在历史记录中找出这些标签全部打开的时间，然后你可以一个接一个地弹出它们。

或者，你可以像我这样做:安装 [**会话伙伴**](https://chrome.google.com/webstore/detail/session-buddy/edacconmaakjimmfgnblocblbcdcpbko?hl=en) ，只需点击你的标签的一个实时备份，然后启动一切，就像你的浏览器失控之前一样。我数不清会话伙伴救了我多少次。为此，它位于我的浏览器“必备”扩展列表的最顶端。

### [](https://chrome.google.com/webstore/detail/the-great-suspender/klbibkeccnjlkjkiokjodocebajanakg?hl=en)

**我从来没有运行过这些数字来看看 [**这个伟大的吊杆**](https://chrome.google.com/webstore/detail/the-great-suspender/klbibkeccnjlkjkiokjodocebajanakg?hl=en) 让你的浏览器减少了多少 CPU 和内存占用。理论上，这个扩展听起来很可靠:它基本上停止了你不经常使用的标签上的任何活动，当你准备重新访问它们时，只需点击一下就可以恢复它们。我承认，即使这样也不足以让 Chrome 在我每次使用它的时候不把我的台式电脑变成喷气式发动机，但是 [每一点都有助于](https://developers.google.com/web/updates/2015/09/tab-discarding) ，对吗？**

 **### [微笑永远](https://chrome.google.com/webstore/detail/smile-always/jgpmhnmjbhgkhpbgelalfpplebgfjmbf?hl=en)

如果你想让杰夫·贝索斯变得更富有，至少你可以确保你在亚马逊上购买的东西对你所在地区的慈善机构有所裨益。通过 [**始终微笑**](https://chrome.google.com/webstore/detail/smile-always/jgpmhnmjbhgkhpbgelalfpplebgfjmbf?hl=en) ，你的浏览器将始终指向 smile.amazon.com 而不是 Amazon.com，这意味着你的任何购买都会给你选择的慈善机构一点现金。他们会成为百万富翁吗？没有，但我在过去一两年里购买的东西已经为我最喜欢的当地慈善机构筹集了近 100 美元。如果有 1000 个人和我一样购物，并且都做同样的事情，那就是 10 万美元的额外收入。如果一万人使用这个扩展...

无论如何，如果你在亚马逊购物的话，请安装笑脸。它不会花费你任何东西，越多人这样做，慈善机构得到的免费资金就越多。就这么简单。

### [隐私獾](https://chrome.google.com/webstore/detail/privacy-badger/pkehgijcmpdhfbdbbnkijodmdjhbjlgp)

我不喜欢在网络上被跟踪，我会竭尽全力——知道这可能仍然是徒劳的——阻止脚本、cookies 和其他废话跟踪我，了解我喜欢的一切。正如电子前沿基金会描述的[](https://chrome.google.com/webstore/detail/privacy-badger/pkehgijcmpdhfbdbbnkijodmdjhbjlgp)**:**

> ***“从更高的技术层面来说，Privacy Badger 会记录在您访问的页面中嵌入图像、脚本和广告的“第三方”域名。Privacy Badger 寻找跟踪技术，如唯一识别 cookie、本地存储“超级 cookie”、通过图像像素的第一方到第三方 cookie 共享以及画布指纹。如果它观察到一个第三方主机在三个不同的网站上跟踪你，Privacy Badger 将自动禁止来自该第三方跟踪器的内容。”***

**这个扩展、AdGuard(运行在我的 Raspberry Pi 4 上，我通过它路由我的互联网流量)和 uBlock Origin 是我减少网络烦恼的“神圣三连胜”。在这三者之间，我过着一种没有广告和废话的网上生活——我希望如此。**

 **### [子块原点](https://chrome.google.com/webstore/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm?hl=en)

你做到了这一步，那么我们来谈谈核武器—[**uBlock Origin**](https://chrome.google.com/webstore/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm?hl=en)。正如我提到的，这是我用来摆脱恼人的广告和网络跟踪的三大工具之一。使用如此多的阻塞工具和扩展似乎有些多余，但是我发现这些工具在捕捉一切方面都做得相当不错。虽然，是的，有比 uBlock Origin 更友好的“”扩展允许你支持做广告的网站，但有时你需要的是手榴弹，而不是手术刀。

### [重定向器](https://chrome.google.com/webstore/detail/redirector/ocgpenflpmgnfapjedencafcfakcekcd?hl=en)

我把这个留到了最后，这样你就可以很容易地跳到下一页的评论里，告诉我你有多讨厌脸书，讨厌我提出任何与脸书有关的建议。

[**重定向器**](https://chrome.google.com/webstore/detail/redirector/ocgpenflpmgnfapjedencafcfakcekcd?hl=en)*本身并不是脸书的扩展；当你在地址栏中输入任何你想输入的内容时，它只允许你的浏览器启动一个特定的 URL。例如，你可以设置它，这样在地址栏输入“lh”总是会跳转到好的 lifehacker.com，节省你大量的击键(如果你不使用地址栏的自动完成功能)。*

*不过，我最喜欢 Redirector 的一点是，它终于允许我在脸书默认一个按时间顺序排列的 feed，一个我之前在这里描述过的黑客。我对此很自豪，因为这是我和我的朋友们一直抱怨的事情。有了重定向器，你可以跳转到脸书的网址，每次你访问脸书时，它都会给你一个按时间顺序排列的提要——也就是说，把网址键入地址栏。**********
