# 抄写反奴隶制的信件来帮助历史学家

> 原文：<https://lifehacker.com/transcribe-anti-slavery-letters-to-help-historians-1844070062>

波士顿公共图书馆收藏了 19 世纪反奴隶制积极分子往来的信件。唯一的问题是:它们是手写的，这使得它们难以阅读，也无法搜索。这就是你进来的地方。

图书馆正在寻找志愿者来帮助转录其反奴隶制手稿项目 中的 [字母。当您登录时，您将看到一封信，其中的脚本行带有下划线(或者，如果您是第一个转录此文档的人，您将自己给它们加下划线)。点击，你可以输入文件中的那一行似乎在说什么，或者你可以同意之前志愿者的转录。多人处理每个文档时，你们每个人都在检查彼此的工作。](https://www.antislaverymanuscripts.org/)

图书馆写道这些文件的重要性:“大量的信件记录了美国和英国主要废奴主义者在 50 年间的互动，从而建立了一个档案，全面记录了 19 世纪波士顿以及海外反奴隶制运动的历史，直到美国内战结束和奴隶制废除。”

根据图书馆的说明，一些写信者还讨论了妇女在废奴运动中的作用，这在当时是有争议的，一些人描述了他们遇到的来自不想废除奴隶制的人的阻力。通过帮助转录信件，你可以帮助历史学家和读者更好地了解这些历史。
