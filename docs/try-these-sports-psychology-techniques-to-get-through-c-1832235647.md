# 尝试这些运动心理学技巧来度过分娩

> 原文：<https://lifehacker.com/try-these-sports-psychology-techniques-to-get-through-c-1832235647>

我曾经参加过马拉松比赛，并在同一年生了一个孩子，我发现这两件事在精神上非常相似。在这两种情况下，一开始都不会太痛，但你知道你必须保存体力。你会到达一个点，你觉得你的身体不能再往前走了，然而还有更远的路要走。你永远无法真正预测最后会发生什么。



没有什么能真正让你为这两种经历做好准备。分娩课程尝试过，但并不是每个人都喜欢。当我疼痛时，我不在乎以特定的方式呼吸。想象一个快乐的地方对我来说毫无意义。按摩已经过时了——别碰我。

这就是为什么我喜欢塞蕾娜·所罗门在《纽约时报》上发表的关于 [运用运动心理学技术分娩](https://www.nytimes.com/2019/01/15/well/family/using-sports-psychology-for-childbirth.html) 的文章。(需要明确的是，这是针对任何类型的劳动，而不仅仅是“自然”分娩。即使你百分之百确定你会要求硬膜外麻醉，你也必须先在家中和车里度过几个小时的宫缩。)运动心理学家向她推荐的一些策略:

*   挑战无益的信念。所罗门害怕剖腹产，所以她收集了剖腹产者的正面故事。事实证明，她的一些担心是没有根据的，这有助于看到人们可以经历这种经历并没事。
*   确定最难的部分是什么，并为每个部分制定一个可视化的计划。对于运动员来说，这可能是一场艰苦的比赛。对所罗门来说，那是去医院的出租车。
*   有一个你可以用作重置按钮的动作，一个连接大脑和身体的东西。所罗门把她的呼吸和手的位置联系起来。

我在分娩时想出了一个类似的动作:我会在每次宫缩时尽可能用力地弯曲脚趾。我身体的其余部分保持放松，但我把所有的压力和疼痛都集中在我的脚趾上。

如果你有克服疼痛的经验(运动或其他)，意识到你可能已经有了一些应对策略。知道你正在做什么，想象困难的部分，并以你自己的大脑和身体感觉最好的方式对情况做出反应。
