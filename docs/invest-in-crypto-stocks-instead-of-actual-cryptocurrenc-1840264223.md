# 投资加密股票，而不是真正的加密货币

> 原文：<https://lifehacker.com/invest-in-crypto-stocks-instead-of-actual-cryptocurrenc-1840264223>

就在短短几年前，购买加密货币风靡一时。理论上，你可以在比特币或其他流行的加密货币上花一点钱，然后以惊人的回报率套现。

但你可能已经注意到，比特币和它的一些同时代人现在正处于后炒作阶段，正如《Coin Talk》的杰伊·凯斯宾·康此前指出的。加密不再是一个可行的快速致富方案()我的意思是，曾经是吗？如果你寻求短期收益，专家提醒你不要这样做。

但也许你仍然对加密货币的发展感兴趣，你想分散你的投资组合。你可以通过股票投资加密货币，而不是购买任何硬币。从逻辑上讲，这比 [记住你的访问密钥](https://gimletmedia.com/shows/reply-all/z3hl9r) 和 [安全存储你的数字货币](https://lifehacker.com/how-to-store-your-bitcoin-as-securely-as-possible-1821051421) 要容易得多。它潜在的风险更小。但是你应该这样做吗？

从设计上来说，股票市场仍然有的风险，但是如果你正在读这篇文章，并且只有一个退休储蓄账户，你可能已经在玩这个游戏了。加密货币股票可能是在不接触最终产品的情况下投资这项新技术的一个选择。

华尔街日报最近将 Overstock.com 公司、英伟达公司和 Hut 8 矿业公司列为受欢迎的加密货币股票。这些公司通常不会发行自己的加密货币，比如比特币或以太坊。相反，他们专注于开发支持加密货币交易的区块链技术。

但是这篇论文也指出仍然有很多波动。在 2018 年加密泡沫破裂后，加密相关股票下跌，尽管 Overstock.com 和 Hut 8 Mining 已经稳定，但它们的趋势远远低于以前的峰值表现。

至于他们参与的加密游戏，例如，Overstock 是一家电子商务零售商。它还经营着一家名为 tZero 的 [区块链子公司](https://www.sltrib.com/news/2019/11/16/stock-utah-based/) ，并计划明年以这个名字发行自己的货币。加密货币的其他参与者都是股票市场上众所周知的名字:微软、Visa、PayPal 和高盛都在 [雅虎！《金融》的“顶级加密赌注”列表](https://finance.yahoo.com/u/yahoo-finance/watchlists/top-crypto-bets/)

注册会计师赖利·亚当斯(Riley Adams)在博客 [Young 和](https://youngandtheinvested.com/) 上写道:“仅仅因为该公司主要专注于加密和区块链，并不意味着它不能有其他业务，甚至不能进入加密价值链的其他环节，以分散仅在有限数量的加密货币内运营的商业风险。”。

基本上，就像你希望分散投资组合以降低一次糟糕的股票购买将使你的收益化为乌有的风险一样，企业也希望通过分散投资自己公司的方式来降低风险。亚当斯说，通过关注特定货币背后的技术，“它极大地降低了任何一种货币在大量投资后破产的风险。”

Adams 建议查看那些从事支付便利化或区块链基础设施的公司，而不是那些专注于某一特定类型加密货币的公司。

如果你 [不想跟踪个股](https://lifehacker.com/stop-investing-in-individual-stocks-use-etfs-or-mutual-1783963972) ，你或许还能从这种加密货币发展中分得一杯羹。雅虎上的许多公司。例如，可以在指数基金 内找到基于公司规模或行业的列表。
