# 你可以只花 49 美元预订西南航空公司的航班

> 原文：<https://lifehacker.com/book-a-southwest-flight-for-as-little-as-49-1835649819>

夏天快开始了——这意味着如果你还没有计划好今年秋天的假期计划的话，现在可能是时候了。直到明天午夜，西南航空公司正在大减价，庆祝公司的生日，今秋单程机票价格低至 49 美元。



虽然机票是不可退票的，而且你也不能在感恩节前后预订停电日期的机票，但一些非常便宜的航班上有很多座位可供预订，包括从亚特兰大到纳什维尔和长滩到圣何塞的单程机票，每张 49 美元。

以下是我们发现的其他一些值得注意的交易:

*   圣何塞到加州伯班克:49 美元
*   纳什维尔到亚特兰大:49 美元
*   旧金山到洛杉矶:64 美元
*   旧金山到拉斯维加斯:65 美元
*   密尔沃基到纳什维尔:71 美元
*   华盛顿到亚特兰大:89 美元
*   纽约(拉瓜迪亚)到芝加哥:98 美元
*   加州奥克兰到墨西哥 Cabo 圣卢卡斯:185 美元
*   巴尔的摩到波多黎各圣胡安:191 美元
*   新泽西州纽瓦克到墨西哥坎昆:222 美元

这种机票是双向的，也就是说你可以预订从圣何塞到伯班克的旅程，反之亦然，每段旅程只需 49 美元。记住你应该在日期上灵活；一些特别作弊的机票要求你在工作日乘飞机离开。

你可以看一下西南航空公司的网站 ，找到一些离你最近的好去处——不要等太久。
