# 通过在 Twitch 上与人聊天来练习一门新语言

> 原文：<https://lifehacker.com/practice-a-new-language-by-chatting-with-people-on-twit-1836770913>

学习一门新语言可能会很复杂，尤其是如果你没有地方练习你正在学习的新词汇和语言规则。



本周，Duolingo [宣布与 Twitch](https://www.twitch.tv/duolingo) 建立新的合作关系，这可能会让寻找练习空间变得更容易一些。

通过新的合作伙伴关系，Duolingo 已经将许多 streamers 确定为内部人员，这是全球大使计划的一部分，就像平台上的主持人和贡献者一样。他们都不是语言专家，但他们会多种语言，或者正在学习第二语言，他们鼓励观众在 Twitch chat 中练习他们知道的语言。

多林哥有 [这里列出的所有经过验证的飘带](https://duolingo.space/ourstreamers) 以及他们说什么语言或者正在学习什么语言。

如果你正在学习一门新的语言，那么试着在实际对话中说，而不仅仅是通过语言学习课程，这是非常宝贵的。Duolingo 非常适合帮助你记下基本的词汇和语法，但是像这样的东西也非常适合做一些练习。

如果你不太喜欢视频游戏，我还推荐 Beelinguapp ，它可以让你一边听你选择的语言的有声读物，一边看那本书的英文文本。这比不上和另一个人用那种语言交谈，但有助于沉浸其中。
