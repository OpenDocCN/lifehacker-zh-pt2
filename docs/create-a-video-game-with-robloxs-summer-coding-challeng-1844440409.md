# 用 Roblox 的夏季编码挑战创建一个视频游戏

> 原文:[https://life hacker . com/create-a-video-game-with-robloxs-summer-coding-challeng-1844440409](https://lifehacker.com/create-a-video-game-with-robloxs-summer-coding-challeng-1844440409)

如果你的孩子曾经想设计自己的视频游戏并制作动画，但不知道从哪里开始，Roblox 有一个免费的夏季编码挑战，可以教他们基础知识。这项名为“ [”的挑战为孩子们提供了简单易懂的指导，帮助他们学习如何创建一个带有自定义动画的游戏，然后通过使用移动设备在现实生活中移动来获得游戏等级。](https://blog.roblox.com/2020/07/build-it-play-it-animations)

Watch

[Roblox 在其博客上解释了](https://blog.roblox.com/2020/07/build-it-play-it-animations) :

> 这个挑战是关于运动的，无论是在游戏中还是在现实生活中。我们将一步一步地向您展示如何制作一个游戏，让玩家在赛道上比赛，并通过用他们的移动设备移动 IRL 来获得关卡。他们的头像是怎么移动的？这取决于你。你可以从头开始制作动画。他们会飞吗？倒着跑？侧手翻？可能性就像你的想象一样无限。

首先，孩子们需要下载免费的 Mac 或 PC 版 [Roblox Studio](https://www.roblox.com/create) 。(如果他们对 Roblox 完全陌生，他们还需要 [注册一个账户](https://www.roblox.com/account/signupredir) 。)接下来，他们可以 [从三个教程](https://www.roblox.com/islandofmove) 中选择一个与自己的动画经验和技能水平(初级、中级或高级)相匹配的教程。

该教程充满了插图，图形和常见问题的故障排除说明，所有这些都将有助于孩子们浏览工作室和创建他们的原创动画。

一旦他们的动画完成，孩子们可以在 7 月 31 日之前提交给 [胜利和庆祝动画大赛](https://developer.roblox.com/en-us/resources/build-it-play-it/2020summer/animation-contest) ，Roblox 的最爱将出现在官方*移动之岛*游戏中。获奖者将根据他们的动画创意，以及是否符合庆祝活动的主题和整体质量来选择。
