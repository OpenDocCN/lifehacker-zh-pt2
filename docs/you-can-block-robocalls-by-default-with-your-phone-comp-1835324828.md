# 你可以通过你的电话公司默认屏蔽自动电话，但这需要你付费

> 原文：<https://lifehacker.com/you-can-block-robocalls-by-default-with-your-phone-comp-1835324828>

如果你接到的机器人电话比真实电话多，你可能会感谢联邦通信委员会周四通过的新规定。立即生效，电话公司将被允许(并被鼓励)自动屏蔽自动电话。



到目前为止，消费者可以注册 robocall blocking，但电话公司并没有将其作为所有人的默认选项。根据新规定，电话公司可以自动屏蔽不需要的电话，只要他们的客户已经被告知这种做法，并有机会选择退出。这些公司还可以提供屏蔽任何不在用户联系人名单上的电话号码的服务。

联邦通信委员会主席 Ajit Pai 在 Youtube 视频中解释说，遏制诈骗机器人电话是他保护消费者的首要任务。“几乎每个有电话的美国人，甚至是联邦通信委员会的人，都会收到，”他说，他的表情表明，也许大多数电话都是直接打给他的，他只是厌倦了这些废话。

Pai 还表示，预计电话公司将在今年晚些时候推出改进的来电显示技术，这将使区分垃圾电话和你可能真正想要接收的电话变得更加容易。

## 你不要以为这是免费的

该裁决的一个不清楚的方面是，如果有的话，为客户的帐户自动启用 robocall 阻止将会花费多少成本。Pai 已经 [先前评论过](https://arstechnica.com/tech-policy/2019/05/ajit-pais-robocall-plan-lets-carriers-charge-for-new-call-blocking-tools/) ，“我们当然鼓励公司免费提供这种服务”，但这完全是和风细雨。

所有四大运营商都提供付费版本的呼叫阻止功能，如果你不选择退出，这很可能成为你每月账单上的主要内容。

*   威瑞森提供呼叫过滤器，这是一款基于应用的呼叫拦截器，免费提供基本功能，但对识别未知号码等高级功能收费 2.99 美元。
*   [AT & T](https://www.att.com/features/security-apps.html) 为客户提供免费的欺诈拦截和可疑垃圾邮件警告，并为增强的来电显示和反向号码查找收取每月 3.99 美元的费用。
*   [Sprint 的](https://www.sprint.com/en/shop/services.html?id16=robocall&id16=robocalls) Premium Caller ID 服务每月 2.99 美元自动筛选 robocallers。
*   [T-Mobile 的](https://www.t-mobile.com/resources/call-protection) Scam ID 和 Scam Block 服务是免费的，但识别电话号码并将选定类别的呼叫直接发送到语音邮件的 Name ID 每月收费 4 美元。

威瑞森、美国电话电报公司和 T-Mobile 都明确表示这些服务只对后付费用户开放。

当然，除了大家伙之外，还有一大堆你可以下载的应用程序和 [方法，你可以自己用](https://lifehacker.com/how-to-prevent-robocalls-and-minimize-phone-spam-1828192361) 来对抗机器人。与此同时，在接下来的几个月里，密切关注你的电话账单，留意默认服务的变化和账单的增加。
