# 这个月的健身挑战主题是:楼梯

> 原文：<https://lifehacker.com/this-months-fitness-challenge-theme-is-stairs-1834511269>

这个月，我们的 [挑战](https://lifehacker.com/c/lifehacker-fitness-challenge) 你走楼梯。整个 5 月，我们将为您提供以楼梯为中心的锻炼和训练，并将其添加到您的日常活动中。第一个，很简单:走或跑上一堆楼梯。

Watch

### **休闲选项**

等待楼梯在你的日常生活中出现，走楼梯而不是电梯或自动扶梯。如果你要上十几层楼，而你还没准备好汗流浃背地穿上工作服，那就乘电梯走完大部分路程，然后走上最后一两层楼。

### **户外活动选项**

如果你住在一个多山的地方，寻找室外楼梯会有一点挑战。有的徒步路线全是石阶；一些城市在某些陡峭的街道旁有巨大的楼梯。找一条轻快的路线去远足。

### **激烈选项**

围绕你最喜欢的楼梯进行高强度的间歇训练。我知道我不是唯一一个在我的登山重复日因雨取消时跑上楼梯的人。即使你只有一段楼梯，你也可以这样做:轻快地走或跑上去，然后慢慢下来。

请记住，走下楼梯会比走上楼梯对你的腿来说更难，即使当时你并没有那种感觉。(众所周知，这些“古怪的”肌肉收缩会导致第二天的疼痛。)第一次别紧张，让我们知道结果如何。
