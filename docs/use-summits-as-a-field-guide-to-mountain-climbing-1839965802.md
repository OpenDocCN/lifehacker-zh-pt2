# 使用“顶峰”作为登山的野外指南

> 原文：<https://lifehacker.com/use-summits-as-a-field-guide-to-mountain-climbing-1839965802>

[峰顶](https://summits.com/) 是一座山一座山的攀登指南。Summits 数据库中的每座山都有自己的页面，上面有详细的定量和定性攀登方向，这使得它成为准备参观某座山甚至选择攀登哪座山的好方法。

Watch

每座山的页面都包括该山的自然历史、周围环境的描述、照片库和该山的人文历史:谁爬过它、当地人对它的态度、该山在当地习俗和宗教传统中的特点。有支持设施指南、当地法规和所需许可证，以及建议的路线。Summits 的目标是让登山者为某一特定山峰的完整体验做好准备。

该网站根据一个 [峰顶指数](https://summits.com/about-summits-index) 对每座山进行评级，该指数根据每座山的高度、地形难度、可用的支持设施和地理可达性(首先到达这座山的难度)来计算每座山的攀登难度。

虽然该网站目前只覆盖东南亚，但它计划为世界各地的山脉建立索引，使用众包信息对从主要山脉到邻近山脉的一切进行评级。
