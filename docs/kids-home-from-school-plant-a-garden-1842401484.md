# 现在是时候和你的孩子一起种植一个花园了

> 原文：<https://lifehacker.com/kids-home-from-school-plant-a-garden-1842401484>

现在新冠肺炎关闭了一切，我们都需要处理可能发生的压力的方法，而我们通常的应对机制都消失了。对于有孩子的父母来说更是如此，现在学校和托儿所都关门了，操场不再安全，游戏日期也取消了。



与此同时，许多工作场所已经转移到在家工作，期望你将继续以同样的速度工作，抛开所有额外的父母义务。

[霍普·贾伦](https://twitter.com/HopeJahren) ，生物学家，《实验室女孩》和《更多的故事》两本书的作者， [有一个建议](https://twitter.com/HopeJahren/status/1237743547182350336) 给那些带着孩子困在家里的父母。

id="h17962">考虑你的可用空间</h3><p class="sc-77igqf-0 bOfvBY">第一步是考虑你可用的空间。对我们中的一些人来说，这意味着一个院子，对其他人来说，这意味着一个阳台，对其他人来说，这意味着一个窗台。即使你只有一个<span> <a class="sc-1out364-0 hMndXN sc-145m8ut-0 fBlGIv js_link" bOfvBY">下一步是考虑你想种什么植物。最简单(也是最便宜)的开始方式是购买几包种子，无论是在苗圃、杂货店、五金店还是网上。</p><h3 class="sc-1bwb26k-1 kpbNNd" id="h17963">容易种植的蔬菜</h3><p class="sc-77igqf-0 bOfvBY">Jahren 指出，一些最好的蔬菜是豆类。"任何种类的豆子都很容易种植！"她说。豌豆或四季豆很容易种植，萝卜也一样。“甜菜很适合种植，因为它们大多能照顾好自己，”Jahren 说。"别忘了吃上面的东西！"</p><p class="sc-77igqf-0 bOfvBY">其他容易种植的蔬菜 包括胡萝卜、西红柿、黄瓜、南瓜和沙拉蔬菜。西葫芦也长得非常好，以至于你的邻居会因为你把棒球棒大小的南瓜卸到他们身上而讨厌你。</p><h3 class="sc-1bwb26k-1 kpbNNd" id="h17964">考虑生长条件</h3><p class="sc-77igqf-0 bOfvBY">当谈到室外花园的种植时，重要的是要考虑生长条件。你种植什么和什么时候种植取决于你住在哪里和一年中的什么时候。这个国家的每个地区都被划分为不同的区域，这是一种衡量什么植物可以在你的地区生长，以及每种植物的生长季节是什么时候的方法。重要的是弄清楚你处于哪个<span> <a class="sc-1out364-0 hMndXN sc-145m8ut-0 fBlGIv js_link" href="https://www.almanac.com/content/plant-hardiness-zones" target="_blank" rel="noopener noreferrer">区</a> </span>，这决定了不同植物的生长季节。你可能还需要对你的土壤进行一些改良，比如添加堆肥，以使你的植物开心。</p><p class="sc-77igqf-0 bOfvBY">也就是说，如果这些听起来太多，不要想太多。最糟糕的情况是，当你想通这一切的时候，你可能会浪费一些种子。最好的情况是，你的努力会得到一些蔬菜或沙拉。</p><h3 class="sc-1bwb26k-1 kpbNNd" id="h17965">制定一个成长计划</h3><p class="sc-77igqf-0 bOfvBY">一旦你有了种子，也想好了要在哪里播种，下一步就是制定时间表。正如 Jahren 建议的那样，阅读种子包装上的说明，写下每种种子的种植和收获建议，然后制定一个春夏秋的滚动时间表。</p><p class="sc-77igqf-0 bOfvBY">一旦你想通了这一切，拿起你的花园铲，拿起你的种子袋，开始种植吧！再过一会儿，如果一切顺利，你会得到吃你的成果的满足感。</p><p
