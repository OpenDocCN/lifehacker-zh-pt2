# 11 月的挑战是:卧推

> 原文：<https://lifehacker.com/novembers-challenge-is-bench-press-1830153476>

我们已经 [蹲起](https://vitals.lifehacker.com/julys-fitness-challenge-is-squats-1827051624) 和 [硬拉](https://vitals.lifehacker.com/how-was-your-month-of-deadlifts-1830124247) ，所以是时候解决 [卧推](https://lifehacker.com/everything-you-need-to-know-to-master-the-bench-press-s-1769031371) 了。我们为每个人都留有空间:即使你从未尝试过，也许有点害怕把酒吧摔在你的脸上，你仍然可以加入。

可能最简单的开始方法是用一对哑铃，一手一个。这种锻炼肌肉的方式与单个大杠铃不同，所以即使你习惯了杠铃，你也可以尝试哑铃卧推来改变一下节奏。

不过，问题就在这里。一旦你对这个动作感到舒适，你会发现你可以卧推比你可以轻松携带和操作的重量要大得多。你如何进入位置？如果你不能完成最后一个销售代表，你会怎么做？

对于答案，我喜欢在 [这个视频](https://youtu.be/v8y8dQZMJXw?t=383) o 中关于举重的提示。(从 6:23 开始快速浏览，或者从头开始详细了解每一步的重要性。)

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-v8y8dQZMJXw&start=383](https://lifehacker.com/embed/inset/iframe?id=youtube-video-v8y8dQZMJXw&start=383) 

安全举哑铃的关键步骤:

*   把哑铃从架子上拿下来，放在地板上。
*   一旦你坐在长凳上，把哑铃放在你的脚上。
*   站起来硬拉它们，坐下时把它们放在膝盖上。
*   抬起一个膝盖，然后另一个膝盖，将哑铃举到肩膀的高度。
*   向后倾斜，做重复动作，然后双脚一起踢，帮助你坐起来。
*   如果你不能完成一个 rep，尽量不要放弃重量；把它们放在你的胸前，向上踢你的膝盖，就像你在比赛结束时那样。

准备好试试了吗？如果你是一个有经验的举重运动员，你知道该怎么做。如果你是这个动作的新手，从轻开始，慢慢来，看看你是否能适应一对哑铃。无论哪种方式，你都可以继续上个月的挑战，将硬拉和卧推纳入同一项锻炼。我们走吧！
