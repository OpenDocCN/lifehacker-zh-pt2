# 如何帮助你的孩子建立自尊

> 原文：<https://lifehacker.com/how-to-help-your-kids-build-self-esteem-1841909386>

你的孩子真的是一个特别的，神奇的小人类。他们是一个令人难以置信的泰然自若的舞者或外向的人，这是你在数学上永远也不会有的。你知道你的孩子很棒；你只是想确保*他们*知道他们有多棒。想要赞美他们所做的所有美好的、令人印象深刻的事情是很自然的。但是，如果我们*过度赞扬*，随着时间的推移，我们实际上可能会伤害他们的自尊。

Watch

《你的孩子正在倾听:他们需要从你这里听到的九条信息》一书的作者吉姆·泰勒告诉今天的父母，试图通过不断地赞美孩子来帮助他们建立自尊是不可取的。相反，孩子们需要的是学会如何发展自己的能力。

> 泰勒说，自尊来自感受爱和安全感，来自发展能力，尽管父母经常给孩子灌输前两种成分，但能力——擅长某事——需要时间和努力。“尽管我们很想这么做，但我们不能通过表扬让孩子变得有能力，”他说。
> 
> 事实上，过度表扬孩子，弊大于利。“我们降低了他们的门槛，”泰勒说。“如果你一直告诉你的孩子她已经做得很好了，你就是在说她不再需要逼自己了。但是自信来自于实践，来自于尝试、失败和再尝试——来自于实践。”

### 让他们多练习

一个在家里或学校只觉得自己能胜任自己的角色或期望的孩子，不一定能胜任新的环境。这就是为什么随着年龄的增长，让他们接触各种环境中的活动或兴趣是个好主意。

专门治疗青少年 的临床心理学家芭芭拉·格林伯格说:“我认为，当孩子们能够在各种不同的环境中与不同的人相处时，他们会开始感到更有能力。”。

给他们工作和责任是另一种让他们练习能力的方法。对于一个青少年来说，这可能意味着一份兼职工作，但青少年和小孩子可以在成功处理家庭琐事中找到一些增强自信的力量。

格林伯格说:“他们可能会为此与父母争吵，但私下里，他们对此感觉良好。”。“他们觉得有必要。”

### 帮助他们评估自己的表现

当然，你可以告诉他们他们在学校戏剧中的表现是一流的，或者他们在球场上的防守在这个赛季中有了很大的提高。格林伯格说，但最终，你不会在那里为他们评估他们的每一次表现，所以他们最好学会如何自我评估。

“你可以问孩子，‘你觉得怎么样？’或者‘还能更好吗？’或者‘你觉得下次你能做些什么？’”她说一个附带的好处是，他们学会了阅读社会线索。"

如果他们学会如何——并定期练习——这种自我评估，他们可能会在与同事、老师或老板发生冲突时，更好地认识到问题并纠正错误。

### 记住奋斗是好的

我们知道我们应该表扬努力而不是结果或能力(T2)。(这就是“我看到你今天早上练习了多少数学题”和“我很骄傲你考得很好！”).然而，当你的孩子真的擅长某件事时，就很难把它卷进去了。但是，过于关注表现的孩子可能会开始减少冒险，或者在迎接新挑战时犹豫不决，以免他们做得不如自己希望的那样好，或者不如他们认为我们期望的那样好。

我们希望他们努力奋斗，至少有时候是这样。奋斗是好的；奋斗是克服障碍的机会。失败——或者仅仅是犯了一个错误——是一个再试一次的机会。泰勒在 [中解释了今天的父](https://www.todaysparent.com/family/parenting/how-to-build-your-childs-self-esteem/) :

> “很多父母都有这种想法，”泰勒说。“他们认为奋斗和失败会伤害孩子的自尊，但实际上这是帮助建立自尊的黄金机会。"

所以，尽管你不想给你的孩子一个超出他们能力的任务，让他们失败，但你确实想帮助他们设定目标，鼓励他们尝试新的挑战，冒一点风险。这有助于培养他们的综合能力，有能力的孩子会更自信。
