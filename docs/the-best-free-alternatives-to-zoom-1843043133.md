# 变焦的最佳免费替代品

> 原文：<https://lifehacker.com/the-best-free-alternatives-to-zoom-1843043133>

虽然自冠状病毒爆发以来，视频会议服务 Zoom 的受欢迎程度已经飙升，但最近关于其许多 [安全](https://gizmodo.com/zoom-accused-of-misrepresenting-security-measures-in-ne-1842756394) 和 [隐私](https://gizmodo.com/zoom-bombings-started-off-as-pranks-now-someone-could-1842644645) 问题的新闻可能会让你对使用它感到紧张。在上面的视频中，我分享了三个你应该尝试的免费选择。

### [不和](https://discordapp.com/)

Discord 最初是为游戏玩家创建的语音聊天工具，现在已经取得了长足的进步，将它的形象 重塑为一个集语音、视频和文本于一体的平台，面向所有类型的用户。

任何使用过 Slack 或微软团队的人都会立刻熟悉 Discord 的界面。Discord 服务器可以有不同的项目或讨论通道，语音通道使服务器上的任何人都可以非常容易地快速接通电话。多亏了最近更新的，多达 100 人只需按一下按钮就可以加入视频通话。使用 Go Live 功能也可以共享您的屏幕，尽管您不能像 Zoom 一样保存通话记录或更改网络摄像头背景。

多年来，我一直使用 Discord，主要是作为我们一起玩视频游戏时与朋友聊天的一种方式。R 然而，最近我们已经把这个工具用在了从举办桌游之夜到一起看电视节目的各种事情上(一个朋友甚至流着自己组装了一台电脑)。对我们来说，这是一种非常简单的保持联系的方式。

### [Skype](https://www.skype.com/en/)

还记得 Skype 吗？嗯，它仍然存在，而且它实际上是 Zoom 的一个很好的免费替代品。微软拥有的 VOIP 服务已经推出了[Meet Now](https://www.skype.com/en/free-conference-call/)功能，支持多达 50 人的视频通话。最棒的是，无论是否有 Skype 帐户，任何人都可以通过邀请链接加入通话。

Skype 提供您可能需要的所有基本功能:记录通话、屏幕共享、模糊您的网络摄像头背景，以及最近更新的 [支持虚拟背景](https://lifehacker.com/how-to-customize-your-background-in-skype-video-calls-1843022022) 。这是一个很棒的全方位平台，可以让你和所有人保持联系，从不懂技术的父母到你的小企业团队。

### [思科 Webex](https://www.webex.com/)

乍一看，Cisco Webex 看起来像一个企业工具，更多的是为了明天的预算会议，而不是为了今晚的欢乐时光。尽管如此，其慷慨的免费层为提供了许多与 Zoom 相同的功能:日历集成、多达 50 人的视频通话和无数的来电选项。在冠状病毒疫情期间，限制通话 40 分钟的自由层限制已经取消，最多 100 人可以进行视频通话。

Webex 对于小型企业或中型团队来说是一个很好的免费选择，但它也是与朋友和家人保持联系的一个很好的方式。
