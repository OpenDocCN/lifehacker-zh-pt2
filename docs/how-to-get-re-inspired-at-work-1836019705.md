# 如何在工作中重新获得灵感

> 原文：<https://lifehacker.com/how-to-get-re-inspired-at-work-1836019705>

我们当中有谁没有在某一天醒来时发现自己深深地陷入了一个老式的工作场所的窠臼？这当然会发生在你从第一天开始就有点犹豫不决的工作上，但是你四年前得到的理想工作也不能幸免。无论我们开始时有多积极，总有一些事情会让我们陷入“平静”，那就是年复一年的例行公事、重复和熟悉。



Arrow Global Entertainment 的首席执行官 Taffi Dollar 为《分钟杂志》 写道，每个人都会遇到这些平静期，这些停滞期，我们在工作中变得不那么兴奋，效率也更低。她有一个建议:去实地考察。

> 如果你想在工作中完成某件特定的事情，但你要么没有动力去做，要么不知道该如何去做，那就出去做一些研究。
> 
> 找一个你认识的人，或者你认识的人，他拥有你想要的智慧或智力。然后，去观察他们的行动。例如，如果你想重组你的组织，去拜访一家已经有类似结构的公司。

当然，你可以实地考察一下你一直想参加的年度会议。或者约一个同行业的同事喝咖啡，交流一下想法。但这真的需要更多的人际关系网，可能还不足以让你走出困境。你需要的是一些真正新鲜的想法，来自那些最有可能拥有这些想法的人。

### 与受影响的人交谈

无论你在哪个领域工作，你所做的(或你的雇主所做的)很可能是为了解决某种问题。为了摆脱停滞不前的状态，去见见受这个问题影响最大的人。只要有可能，就去找他们，而不是把他们带给你。

例如，如果你在一家教育软件公司工作，去你附近的学校。会见老师，询问他们的技术障碍，并在他们教学时观察他们。你可能会更深刻、更及时地了解教师现在需要什么来提高课堂效率。

也许最重要的是，你会有一种新的灵感，你的工作最终会帮助老师们更好地完成他们的工作。只要记得把这些人当成你在野外的焦点小组；对他们有礼貌，尽你所能奖励他们的时间。

### 寻求外部专家

当然，你和你的同事可能是你所在领域的专家。当你做一件事的时间足够长时，你会开始觉得自己对这件事了如指掌。但很可能有人知道的比你还多。在你当地的大学里，可能有一位大学教授专门研究你所在的行业，每年都会就一些你可能想旁听的话题做讲座。也许有一个研究人员正在研究你最感兴趣的角度，也许那个研究人员会允许你去他们的办公室看看他们的工作。

至少，很可能还有其他与你的目标人群或客户一起工作的团队，他们或许能够对你试图解决的任何“问题”提供见解。想一想，除了你的竞争对手，还有谁可能在你的领域拥有知识、专长或经验，并找出他们。

去参观，参加会议，问问你是否能成为墙上的苍蝇。这些实地考察可能没有你在幼儿园时参观南瓜园那么令人兴奋(没有什么能比这更令人兴奋)，但走出办公室，沉浸在你所在行业的主题中，可能正是你需要为你的工作重新注入活力的。
