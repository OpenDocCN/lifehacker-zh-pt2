# 如何判断自己是否无聊

> 原文:[https://life hacker . com/how-to-tell-if-you-boring-1835650232](https://lifehacker.com/how-to-tell-if-youre-boring-1835650232)

无聊的人无处不在——在工作中，在你孩子的学校，甚至可能在家里。但也许唯一比被困在与单调先生的无聊对话中更糟糕的事情是发现，实际上，无聊的人是*你*。

Watch

不要惊慌。你现在可能很无聊，但这并不意味着你必须永远无聊。你只需要意识到警告信号，并承诺保持航向正确。

心理学家芭芭拉·格林伯格在 [【今日心理】](https://www.psychologytoday.com/us/blog/the-teen-doctor/201905/10-ways-determine-if-you-are-boring) 给了我们一份简短的问卷，你可以用它来判断你是否让你周围的人感到无聊。在她的问题中，一些特定的主题浮现出来供你思考。

### 你的倾听技巧

不无聊不仅仅是说你脑海中突然出现的所有有趣的事情。而是成为对话中真正的参与者。这意味着积极倾听对方在说什么，而不是做白日梦，想着如何插话，把话题引回自己身上。

好奇心也是你的朋友。当你提出真诚的后续问题时，人们知道你在听，并且对他们所说的真正感兴趣。"那么，在你去新奥尔良的旅途中，你吃过的最好的东西是什么？"或者“哦，那个项目听起来很有挑战性；多给我讲讲！”

### 说得太少还是说得太多

我们试图在孩子们很小的时候就传授他们“轮流”与他们交谈的习惯(如果你还没有看过那个关于父亲与他的宝贝儿子 交谈的 [视频，现在是时候了)。然后我们长大了，发现自己要么完全主导了一场对话，要么在某种程度上喃喃自语。两者都很无聊。](https://qz.com/1639907/the-viral-video-of-a-dad-talking-to-his-baby-demonstrates-a-crucial-parenting-skill/)

在表达对某个话题的共同兴奋或兴趣，而又不完全抢走话题之间有一个平衡。

### 你的脆弱

你不需要为了变得有趣而泄露每一个深藏不露的秘密，但是允许自己通过分享恐惧或焦虑变得脆弱，可以建立与他人的情感桥梁。如果你太封闭了，别人可能很难和你交流。

这并不意味着——我重复一遍，*这并不意味着*——你应该不停地抱怨你糟糕的同事或懒惰的配偶，而你对面的那双眼睛却无动于衷。那不是脆弱。那太无聊了。振作起来，把谈话转移到一个新的话题上。

如果你觉得谈话嘎然而止，你可以不时地拿出一些轶事、故事或问题，这也没有坏处。

### 身体语言

你能感觉到有人没那么投入你的谈话，对吗？他们看手机有点不礼貌，环顾四周示意餐厅服务员停下来，或者通常看起来心不在焉或不感兴趣。注意你当前的话题是否已经不受欢迎。

换句话说，人们是否似乎*希望*在你身边？如果他们这样做，你可能不无聊。

如果你还不确定，去拿 [格林伯格的全问卷](https://www.psychologytoday.com/us/blog/the-teen-doctor/201905/10-ways-determine-if-you-are-boring) 。
