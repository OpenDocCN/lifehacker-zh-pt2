# 用这张 GIF 更好的呼吸

> 原文:[https://life hacker . com/use-this-gif-to-breathe-better-1833297415](https://lifehacker.com/use-this-gif-to-breathe-better-1833297415)

凯蒂·霍金斯-加尔在她的简讯 [《我可爱的笨脑袋](https://mailchi.mp/848017378986/dont-forget-to-breathe?e=0cc9645d81) 中写到花时间让*呼吸*的重要性。你知道，不是那种为了生存而进行的无意识呼吸，而是深思熟虑的缓慢呼吸，帮助你脚踏实地，感受你在这个世界上的位置。

“你上一次注意自己的口气是什么时候？霍金斯-加尔写道:“现在花几分钟宝贵的时间专注于你的呼吸。“用鼻子吸气，停顿，然后用嘴慢慢呼出。重复。”

她接着建议，如果你觉得有困难，就用这张 GIF 的 [来练习。](https://media.giphy.com/media/krP2NRkLqnKEg/giphy.gif)

看着它。吸气，呼气。我已经觉得平静多了。

如果这一切对你来说感觉不太好，那也没关系。一开始，我也是。但是根据哈佛医学院的说法，深呼吸和练习呼吸技巧有助于缓解压力。

首先，你需要确保你做得对。哈佛医学院说“当你深呼吸时，从鼻子进入的空气完全充满你的肺部，下腹部上升。”

一开始可能会觉得不自然，但它在几个方面是有益的。哈佛健康 写道 [“深腹式呼吸鼓励充分的氧气交换——也就是说，进来的氧气与出去的二氧化碳进行有益的交换。"毫不奇怪，它可以减缓心跳，降低或稳定血压."](https://www.health.harvard.edu/mind-and-mood/relaxation-techniques-breath-control-helps-quell-errant-stress-response)

以下是哈佛健康推荐的做法:

> ***第一步。*** 找一个安静、舒适的地方坐下或躺下。首先，正常呼吸。然后试着深呼吸:用鼻子慢慢吸气，当你充满肺部时，让你的胸部和下腹部上升。让你的腹部充分扩张。现在用嘴慢慢呼气(或者用鼻子，如果感觉更自然的话)。
> 
> ***气息专注于修行。*** 一旦你采取了上述步骤，你就可以继续有规律地练习控制呼吸了。当你闭着眼睛舒适地坐着时，深呼吸和有用的图像混合，也许还有一个帮助你放松的焦点词或短语。

当你专注于你的呼吸时，你不会专注于其他压力或负面的刺激。对她来说，霍金斯-加尔说，这有助于她应对紧张的海外旅行。

“我很感激我能够参加这次旅行，以及所有让我来到这里的焦虑和兴奋的时刻，”她写道。"那些时刻提醒我要呼吸，这反过来让我脚踏实地，活在当下."
