# 按日期限制谷歌搜索结果的最快方法

> 原文：<https://lifehacker.com/the-fastest-way-to-limit-google-search-results-by-date-1841894204>

一个措辞恰当的谷歌搜索可以在几秒钟内显示出最相关的结果。但是那些被认为最符合你的搜索查询的结果的问题是，它们并不总是那个主题的最新结果。

Watch

这种事经常发生在我身上。我经常在谷歌上搜索关于个人财务的信息，但是像税率这样的数据每年都会变化。如果我得不到最近的*和*相关结果，我会花很长时间才能得到我需要的信息。

这就是为什么我对 Reddit 用户 [familytreebeard](https://www.reddit.com/r/LifeProTips/comments/f7yg7i/lpt_if_you_want_google_to_return_only_very/) 的这个提示很满意。您可以在文本框中为您的搜索添加时间限制。以下是要做的事情:

> 您可以在任何搜索中使用“之前:<year>”和“之后:<year>”来过滤任何年份范围。比点击“工具”菜单并手动选择时间范围要快得多。</year></year>

该线程强调在搜索软件或编程信息时使用这种技术，这可能在短短几年内发生很大变化。

让我举一个简单的例子来说明它有多有用。上周，当我写关于自动化你的信用卡支付的文章时，我提到了信用卡发行商可以收取的最高滞纳金。

这并不能代替你的努力和仔细检查搜索结果。但是如果你经常进行搜索，需要某个时间段的结果，或者只是想要最新的信息，这是最快的方法。
