# 使用 Ring 的新控制中心来阻止警方的请求

> 原文：<https://lifehacker.com/use-rings-new-control-center-to-block-police-requests-1841432223>

感觉很多人都不会放弃他们昂贵的摄像机门铃，不管 Ring 做什么——要么是因为他们花了很多钱买了一个，而且他们很固执，要么是他们不想花更多时间摆弄螺丝刀和电线来安装其他东西。

那很好。我们都会在生活中做出决定，不管是好是坏。但如果你坚持使用你的戒指设备，你绝对应该看看该公司刚刚在其 [Android](https://play.google.com/store/apps/details?id=com.ringapp&hl=en_US) 和 [iOS](https://apps.apple.com/us/app/ring-always-home/id926252661) 应用中首次亮相的新控制中心。

一旦你可以访问它(通过你的应用程序左上角的汉堡图标)，你会看到许多选项都值得研究。

我会从最底层开始。向下滚动到**“社区控制”**部分，点击**视频请求**。将开关转到“禁用”以阻止当地警察机构通过 Ring 的“邻居”服务从您的各种 Ring 设备请求视频。(如果您想知道您当地的哪个警察部门(如果有)可能需要这类信息，请点击“查看活动机构地图”选项。)

这不会阻止警察使用 [传票或](https://techcrunch.com/2020/01/30/ring-security-control-center/) 搜查令来获取你的视频。这只是为了防止他们利用邻居来索取镜头，这可能很烦人——或者是你不想参与的事情。

从那里，回到控制中心，记下三个**“授权客户端设备】、**、**、**和**、【关联账户】、**选项。在每一个，你会得到一个杀死开关，你可以用来快速删除任何和所有与你的戒指帐户的联系。如果你没有使用双重认证来保护你的帐户，请查看这些列表，以确认没有其他人访问过你的记录。

最后，点击**双因素认证**。您应该已经使用这个来保护您的帐户，即使 Ring 之前从未提示您这样做。(它很快就会为新账户自动启用 2FA，但 Ring 不会强制现有账户使用 2FA。)您可以使用此屏幕来验证您*正在使用双因素身份验证*。如果没有，你可以也应该设置它。

诚然，Ring 的实现实际上是两步认证，这是一种不太安全的设置，当你登录时会收到验证码。与物理设备相比，有人更容易获得你的电话号码，比如你的指纹锁定智能手机上的认证应用程序，但两步认证总比什么都没有好。
