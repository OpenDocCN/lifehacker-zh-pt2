# 用本笃会创造一个特别的曼哈顿

> 原文：<https://lifehacker.com/make-an-extra-fancy-manhattan-with-benedictine-1841978492>

有时我太专注于杜松子酒和阿玛罗酒，以至于忘记了我刚刚坐在那里等着被消费的完美的威士忌。实际上，这有点奇怪，因为我曾经很喜欢曼哈顿和曼哈顿变体。



这是一种非常多变的鸡尾酒。你可以更换威士忌(从黑麦威士忌换成波旁威士忌或者其他一些)或者你可以更换苦艾酒的成分。今天我们要做的是后一种，用甜苦艾酒换成本笃会酒，这是一种由僧侣酿造的奇特的草药利口酒。它甚至比甜苦艾酒还要甜，所以半盎司就足够了。要做到这一点，你需要:

*   两盎司半黑麦威士忌
*   1/2 盎司本笃会
*   两个破折号安古斯图拉苦酒

将所有材料加入到装满冰块的搅拌杯中，搅拌至完全冷却。你要在这里搅拌很多，因为这是很多乙醇，稀释是非常需要的。滤入冰过的双座玻璃杯，如果有的话，用一条柠檬皮装饰。
