# 现在你可以把不想要的亚马逊商品退回给美国柯尔百货公司

> 原文：<https://lifehacker.com/now-you-can-return-unwanted-amazon-items-to-kohls-1836138613>

下次你想把东西退给亚马逊，你不一定要把所有东西都重新打包，然后去邮局。从本月开始，除了阿拉斯加州安克雷奇的商店之外，亚马逊允许在美国柯尔百货公司的任何地方通过邮寄和当面退货。

Watch

在美国柯尔百货公司退货并不像带着它走进去那么简单。为了在商店退货，你首先需要在亚马逊的在线退货中心开始在线退货。在那里，你会告诉亚马逊你打算在哪一个美国柯尔百货公司下车。

一旦你在网上完成退货设置，亚马逊会给你一个二维码，供你在商店使用。

之后，您只需将想要退货的商品带到美国柯尔百货公司的商店，并向店员出示二维码，即可完成退货。

与邮寄物品不同，物品可以自行退回美国柯尔百货公司，不需要运输箱(尽管如果你愿意，你当然可以随身携带运输箱)。这意味着，如果你在决定退货之前回收了箱子，你就不必为了退货而去翻另一个装运箱或想出如何安全地包装东西。

美国柯尔百货公司将处理退回商店的任何物品的包装和运输。面对面的回访也是免费的，尽管这并不包括你在美国柯尔百货公司时留下的任何冲动购物。
