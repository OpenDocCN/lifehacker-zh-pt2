# how to write a condolence note 1833449660

> 原文:[https://life hacker . com/how-to-write-a-comotion-note-1833449660](https://lifehacker.com/how-to-write-a-condolence-note-1833449660)

当你认识的人失去一个亲近的人时，很难知道该做什么或说什么。悲伤是一种可怕而不确定的东西，如果你没有经历过，伸出手会感觉像是超越了界限或提醒某人他们试图忘记的事情。

但是悲伤是孤立的，也是痛苦的，重要的是让失去亲人的朋友和熟人知道你在想他们和他们所爱的人，不管这感觉有多尴尬。如果你认识他们失去的人，这是真的；如果你没有，这也是真的。

“大多数悲伤的人，不管他们是否意识到这一点，他们最大的恐惧之一就是失去的人会被遗忘，” [詹尼弗·苏斯](https://jennifersoos.com/) 说，她是得克萨斯州圣安东尼奥的一名婚姻和家庭治疗师。“任何时候有人想起他们，承认他们，谈论他们，以任何方式召唤他们，这就像是恐惧的安慰。”

因此，尽管看起来忽视损失会让哀悼中的人更容易继续生活，但这实际上是你能做的最糟糕的事情。虽然对某人说“我为你的损失感到难过”似乎有些老套，但至少，承认失去的人曾经存在，现在他们不存在，会让你悲伤的朋友感到被倾听和被认可。“我认为这不会经常发生，”苏斯说。

下面是怎么做的。

<label class="bxm4mm-13 juykRM">Advertisement</label>

### 发送一张卡片——或者任何一种信息

或者电子邮件，或者短信，这取决于服丧的人属于哪一代，以及你对他们的了解程度。苏斯说:“如果你是千禧一代，而且是同龄人，发短信没问题。”“电子邮件比短信好，因为你必须坐在电脑前才能收到。我坚定不移地属于 X 一代，所以我更喜欢手写的便条，但我承认这是一代人的偏好。”

基本上做点什么总比什么都不做好，但是大家都喜欢一张牌。(同样值得注意的是，文本可能会在死后的洪水中迷失，这一点要记住。)

### 不知道该说什么也没关系(但这里有一些你可能想说的事情)

没有人真的知道该对哀悼中的人说些什么，尤其是当他们所爱的人意外死亡的时候。在你的信中承认这一点完全没问题。苏斯说:“在我看来，说‘正确’的话就是事实。”。“这是我们在治疗、支持团体和研究中一遍又一遍地从失去亲人的人那里听到的。他们说的是，‘我希望人们只是说，‘我不知道该说什么，但我很抱歉。’。我不知道该说什么，但我为你感到难过。我不知道该说什么，但我在乎你。"

你想让对方知道你在想他们，你在他们身边，你可以倾听他们。你不指望让他们感觉更好，因为当你悲伤时，没有什么会让你感觉更好。但是你不想感到被忽视或孤独。“说‘我不知道该说什么’似乎很容易，”苏斯说。“我们担心这会让一个人感到尴尬、不快或不安。但事实是，我们不知道该说什么。没有什么话能让它变得更好，除非你有一个神奇的咒语让死人复活。”

<label class="bxm4mm-13 juykRM">Advertisement</label>

所以，请随意坚持老生常谈，只要它们是真实的。“人们可以说的一件事是，‘我不知道该说什么，但我爱你，我在你身边，’”新奥尔良的心理学家玛里琳·门多萨博士说。“或者，‘我在想你，给你送去爱。’或者，“无法用语言表达我对你的损失有多难过。我很同情你。"

正如苏斯所说，“你真正想要传达的是，‘我在这里支持你。我见证了你正在经历的，我承认它，我没有忽视它，我不需要完美的语言来帮助你。你被看到，你被听到，我只是简单地通过承认来与你分担这种痛苦的负担。"

### 不要谈论你自己

这应该是显而易见的，但吊唁信不是广播你自己生活更新的时候。门多萨说:“这张纸条应该是给死者家属的。”“不要包括关于你自己的东西，比如我刚找到一份新工作，或者这是我的新地址。”类似地...

### 不要说，“我知道你的感受”

同情哀悼中的人是很有诱惑力的，尤其是如果你自己也经历过失去。但是悲伤对每个人来说都有不同的表现，如果有人声称知道你的具体感受，而他们并不知道，这可能是无效的。“不要说，‘我知道你的感受’，因为即使你失去了母亲，你也不知道另一个人的感受。门多萨说:“你知道自己的感受。

提供建议也没有帮助。“没有人真的想要建议，”苏斯说。"它做出了一些微妙的判断，即悲伤的方式有对错之分."

如果你想用“至少”这个词，千万不要。“我总是告诉人们，如果你要说‘至少’，你应该停下来，”苏斯说。“你要证明为什么它不应该是痛苦的，这是无效的。”

### 不要想当然地知道哀悼中的人和他们失去的人之间的关系

通常在吊唁卡上附上一条信息，评论死者对失去亲人的人来说是多么特别。但是如果你不知道死者或者不太了解他们和你写信的人的关系，你可能要避免做任何假设。

门多萨说:“如果你不认识这个人，评论他们可能不是最好的事情。”“有时候并不是很特别的关系。”当然，如果你确实知道这个人和你的朋友很亲近，请随意说出来。你也可能想包括死者的名字。门多萨说:“当你失去亲人时，人们通常不会提到这个人的名字，但这对他们来说意义重大。”“所以你可以这样说，‘我知道大卫对你来说是个特别的人。’"

### 提供具体的帮助，如果有的话

人们在冲突时经常说的一句话是，“如果有什么我能帮忙的，请告诉我。”这并不是说这一定是一件不好的事情，但是一个穿着丧服的人是不会接受你的工作的。苏斯说，不要提供模糊的帮助，“给他们一些具体的东西，让他们告诉你是否有帮助”。问他们你是否能给他们带晚餐，或者喂他们的猫，或者照看他们的孩子，或者带他们去喝一杯，听他们倾诉。

如果你对他们足够了解(并且知道这不会是一种打扰)，不用被要求就可以随意做一些事情。“我有一个非常好的朋友，她的一个最亲密的朋友最近去世了，”苏斯说。“我没有问，我只是出现在她家，并带来了晚餐。”

### 在紧接的哀悼期结束后第二次伸出手

大多数人在听到这个消息后都会向失去亲人的人伸出援手，这当然是正确的做法。但是在某人去世后，一切都变得模糊不清，虽然收到大量短信、电子邮件和可爱的卡片非常好，但很难在那一刻欣赏那些发自内心的信息。苏斯建议，在最初的哀悼期表面上结束后，第二次接触那些悲伤的人。

“事实是，坦率地说，人们在(死后)最初的几天或几周内，真的记不起太多东西，”苏斯说。“当你听到这个消息时，先联系一下，然后马上在你的日历上标记 4-16 周。那是大多数人需要更多支持和鼓励的时候。”

<label class="bxm4mm-13 juykRM">Advertisement</label>

苏斯说，几个月后，大多数与死者不亲近的人都会继续生活，但对于那些悲伤的人来说，痛苦还没有消退。“时间和悲伤的可怕之处在于，震惊开始消退的时候，恰好是其他人继续前进的时候。苏斯说:“那时你真的会陷入‘哦，天哪，这是我的真实生活，真糟糕。’。“那些在 3 个月、4 个月或更长时间内出现的信息、笔记和文本可能比最初的接触更有意义。”

所以，几个月后，发个短信，带你朋友去吃饭，带瓶酒，带部好电影去他们家，让他们给你讲讲他们爱的人。不要觉得你在强迫某人重提不愉快的事情，或者提醒他们想忘记的事情。他们真正想要的，是不被遗忘。
