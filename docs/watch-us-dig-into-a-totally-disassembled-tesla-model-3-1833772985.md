# 我们挖掘了一辆特斯拉 Model 3 和雪佛兰 Bolt，以了解它们的秘密

> 原文:[https://jalopnik . com/watch-us-dig-into-a-total-disablished-Tesla-model-3-1833772985](https://jalopnik.com/watch-us-dig-into-a-totally-disassembled-tesla-model-3-1833772985)

如果你仍然不相信电动汽车至少会成为我们汽车未来的一个重要组成部分，那么，好吧，我想你可以享受你那充满汽油的美好幻想生活了。如果你像大多数有眼睛和大脑的人一样，你知道电动汽车会一直存在，深入了解它们是有意义的。这就是为什么当我们听说我们可以获得市场上最受欢迎的三款电动汽车，并将其拆成零部件时，我们知道我们必须利用这个机会。好家伙，我们做到了。

将这三辆车——一辆雪佛兰 Bolt、一辆宝马 i3 和一辆特斯拉 Model 3——如此彻底和仔细地肢解的地方是 [蒙罗](https://jalopnik.com/the-fascinating-company-that-tears-cars-apart-to-find-o-1787205420) 、 [这家工程公司，当主要汽车制造商想要确切地知道他们的竞争对手是如何制造汽车的时候，他们会求助于](https://jalopnik.com/the-fascinating-company-that-tears-cars-apart-to-find-o-1787205420) ，直到最后一个微小的螺栓和螺钉。

我们接触到的三款电动汽车是一个特别好的选择，因为它们代表了一种非常不同的电动汽车制造方法。当然，在一个理想的世界里，我们会有更多的机会。例如，日产聆风(Nissan Leaf)可能是路上最常见的电池电动汽车，因此其中一辆就不错了。好消息是，在一般的设计和建造中，我认为叶片最像螺栓，所以它不会构成一个完全独立的类别。

此外，如果我们能够获得一个 [大众 MEB 平台](https://jalopnik.com/the-fascinating-engineering-behind-vws-electric-car-pla-1829257860) 那就太好了，但是因为那些还没有提供，我真的不能太沮丧它不在那里。我认为这个新平台与特斯拉的非常相似，但可能没有那么紧密的组件集成，这意味着默认情况下大众平台不包括 frunk(我认为它可能包括 frunk，并且[ID Buzz 肯定显示](https://jalopnik.com/holy-crap-volkswagen-will-build-that-new-microbus-1796226189) 它不需要太多的引擎盖)，但在特斯拉平台的可维护性方面可以提供一些改进。

当然，我只是推测。然而，对于 Bolt、Model 3 和 i3，我们根本不需要推测，因为我们可以看到一切。事实上，我们看到了太多，以至于我们的视频编辑让我们收回一些视频中强烈的极客感，但出于理智的原因，我将交给大卫给你一些我们无法塞进视频的真正、真正深入的东西。

### 大卫·特雷西:

我不会真的进入太深的细节，因为这实际上只是对这三款电动汽车之间的一些差异的高层次观察(好吧，所以 i3 在技术上是一款混合动力车，因为它有增程器)，但我会指出一些我们看到但没有包括在视频中的有趣的事情。

我已经清楚地表明了我有多喜欢冷却系统 ，我们在视频中谈到的一件事是 i3 如何为电池利用直接制冷剂冷却设置，Model 3 和 Bolt 都将液态乙二醇冷却剂送入电池组。不过，值得注意的是，Model 3 和 Bolt 还利用了连接到 HVAC 系统的所谓“冷却器”中的制冷剂。

这样的设置意味着 Bolt 和 Model 3 的电池是主动冷却的，而不是被动冷却的，如果他们只是通过汽车前部的散热器输送液体乙二醇，就会出现这种情况。你可以在这张冷却系统图中看到 Model 3 的液体冷却剂到制冷剂的冷却器，我之前在另一篇文章 中 [解释过。](https://jalopnik.com/the-tesla-model-3s-superbottle-easter-egg-is-a-fascin-1830992728)
