# 你应该马上在你的任天堂账户上设置两步验证

> 原文：<https://lifehacker.com/you-should-set-up-two-step-verification-on-your-nintend-1842967392>

如果你有一个任天堂帐户，现在是一个很好的时间来确保你在那个帐户上设置了两步验证来保证它的安全。

Watch

任天堂让两步验证成为其账户的可选选项，但最近开始了一个小型的社交媒体活动，鼓励用户选择加入。

据任天堂生活报道，本周末，许多任天堂 Switch 用户发布了他们账户上发生的第三方登录事件，其中一些导致了资金损失。任天堂最近还向日本的账户持有人发出警告，称通过任天堂账户进行的信用卡欺诈有所增加。

我自己的账户在 3 月下旬被泰国的某个人访问了。虽然我幸运地在任何损害发生之前发现并更改了我的密码，但我的安全密码显然不够安全，我很尴尬我没有在未授权登录之前进行两步设置

要在您的帐户上设置两步验证， [转到任天堂网站](https://accounts.nintendo.com/2fa/authenticator) ，然后登录您的帐户。

之后，进入“登录和安全设置”，然后进入“两步验证”并点击“编辑”

虽然一些服务使用你的电话号码进行两步服务，但任天堂只允许通过认证应用程序进行两步验证。

当你点击任天堂网站上的“两步验证设置”时，你会收到一封电子邮件，说明如何使用谷歌认证器完成验证过程——这也应该可以在 [任何你喜欢的第三方应用](https://lifehacker.com/how-do-i-switch-from-one-2fa-authentication-app-to-anot-1842296754) 上运行。

作为这个过程的一部分，如果你需要访问你的帐户，并且手边没有这个应用程序，你会得到一些代码作为备份。请确保将它们存放在安全的地方，以便在需要时可以找到。

两步认证将使你登录你的任天堂账户变得稍微困难一点，但它将使成为潜在攻击者的无限困难。
