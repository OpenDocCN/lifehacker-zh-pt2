# 如何治疗严重晒伤

> 原文：<https://lifehacker.com/how-to-treat-a-bad-sunburn-1844138553>

如果你每天 [涂防晒霜](https://vitals.lifehacker.com/all-your-sunscreen-questions-answered-1827579794) ，适当地涂防晒霜，每隔几个小时再涂一次(或者每当你出汗、游泳或擦毛巾的时候)，那么你就不需要这篇文章。但是我们在这里。你被烧伤了，你很受伤。现在怎么办？

让晒伤消失是不可能的，但是在你等待它痊愈的时候，你可以做一些事情来帮助它感觉更好。这真的是在愈合——晒伤是一种真正的伤害，你的身体需要从中恢复。

### 远离阳光，涂上防晒霜

你需要做的第一件事是*不要让事情变得更糟*。如果你在阳光下，当你发现自己被晒伤了，马上离开太阳。如果你直到第二天早上才看到发红，请确保在出门前涂抹防晒霜。

美国皮肤病学会 [推荐](https://www.aad.org/public/everyday-care/injured-skin/burns/treat-sunburn) 穿防晒- 也穿防晒面料。一些海滩专用的衣服可能会在标签上贴上 SPF 标签，但一般来说，当你把它举到阳光下时，你会想要一些不透明的东西。

### 考虑一种含有芦荟、大豆或炉甘石的乳液

芦荟凝胶在晒伤的皮肤上感觉很好， [AAD 也推荐它，](https://www.aad.org/public/everyday-care/injured-skin/burns/treat-sunburn) ，还有含有大豆的乳液。炉甘石洗剂也可以感觉舒缓。

[氢化可的松乳膏](https://www.webmd.com/drugs/2/drug-10402-859/hydrocortisone-topical/hydrocortisone-topical/details) ，药店有售，是一种抗炎药物，可以减轻肿胀、瘙痒和发红。无论如何，手头有一个防蚊虫叮咬的产品是很好的，但是你也可以用它来治疗晒伤。

### 不要在你的皮肤上涂任何刺激性的东西

有很多不同的家庭疗法，你可能应该跳过所有这些。例如，不要在晒伤处涂醋或金缕梅。许多专家建议不要使用局部麻醉剂，因为它们也会刺激皮肤。(我个人*喜欢我的含利多卡因的芦荟胶，但考虑到自己被警告过。)*

### *洗冷水澡或淋浴*

*最有效的缓解方法之一也是最简单的:洗个冷水澡。不要擦洗，因为那会刺激皮肤，不要使用多余的肥皂。当你从浴缸里出来时，拍干你的皮肤，并涂上保湿霜。*

*顺便说一下，这些应该是凉爽的浴池——T2 不冷。冰浴和冰袋会刺激皮肤。对自己宽容点。*

### *照顾好你的整个身体*

*由于晒伤是一种需要愈合的损伤，你需要给你的身体一个恢复的机会。保持水分并考虑服用布洛芬等消炎药来暂时减轻疼痛和红肿。*

*晒伤通常伴随着中暑或脱水，所以要留意这些症状。如果你出现寒战、发烧、虚弱、头晕或脉搏加快或呼吸急促，寻求医疗帮助。*
