# 正念冥想并不适合每个人

> 原文:[https://life hacker . com/正念-冥想-不适合所有人-1844751749](https://lifehacker.com/mindfulness-meditation-isnt-for-everyone-1844751749)

冥想有众所周知的好处，既有研究支持，也有轶事趣闻。但是生活中很少有事情对每个人都有效，而且有理由相信冥想可能会使一些人的心理健康恶化。如果是你，我在此允许你退出。无罪恶感。

提醒我说这是《新科学家》的一篇文章 ，关于一项研究(尚未在网上发布)发现一些人——8%的人，他们说——经历了冥想的负面影响，比如焦虑增加。与此同时， [去年的一项研究](https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0216643) 发现，25%经常冥想的人有时会感到“与冥想相关的不愉快经历”

值得做的事情偶尔会令人不愉快，这当然是可能的；我经常锻炼，并认为这有助于我的身心健康，但如果你问我是否有过不愉快的锻炼经历，我肯定会回答是的。糟糕的一天并不会让整个追求变得毫无价值。

但是很多冥想练习都有一种奇怪的僵硬感。我记得整个关于顶空的介绍系列都是一个温和的声音告诉我该做什么，并安慰我如果我按照指示去做，我最终会感觉更好。一些应用程序奖励 [条纹](https://lifehacker.com/you-dont-truly-build-a-habit-until-after-you-break-your-1844686414) 或者强迫你以某种方式或某种顺序进行冥想。

另一方面，如果你走自由形式，在思考或听音乐的时候专注地呼吸，这可能会导致我们中的一些人过度思考。心理学家凯蒂·斯帕克斯 [告诉](https://www.newscientist.com/article/2251840-mindfulness-and-meditation-can-worsen-depression-and-anxiety/) 新科学家，引导冥想可能“更安全”，有助于让你的想法保持在一个好的地方。或者，正如我的编辑在与我分享链接时所说的，“现在也许比以往任何时候都更不想活在当下？”

有一点是清楚的:虽然冥想帮助了很多人，但认为它是灵丹妙药或大脑的必要维护可能是错误的。后者是我们在过去的 中提到过的 [观点，但我认为这值得注意——冥想作为一种维护对我们许多人来说是一个有益的比喻，但它不是万能的解决方案。](https://lifehacker.com/why-meditation-should-be-your-brains-scheduled-mainten-1633217329)

因此，如果你试图通过冥想来消除焦虑或抑郁，你会发现它只会让事情变得更糟，而不是更好——放弃吧。或者尝试不同的类型。将你自由形式的冥想换成引导式的，反之亦然。最后，如果你放松的事情没有让你放松，也许是时候相信你的直觉，继续前进了。
