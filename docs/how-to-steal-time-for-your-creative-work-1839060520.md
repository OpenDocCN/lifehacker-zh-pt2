# 如何为你的创造性工作挤出时间

> 原文：<https://lifehacker.com/how-to-steal-time-for-your-creative-work-1839060520>

创造性类型的工作-生活平衡更像是工作-工作-生活。你必须为你的工作职责和非工作职责找时间，为你关心的人和你支持的社区找时间，为休息和恢复找时间，为你正在做的大创意项目找时间。

时间从何而来？

德里克·西弗斯(Derek Sivers)，他最出名的要么是 CD Baby 的创造者，要么是给了我们 [“这要么是一个*地狱耶*，要么是一个*不*](https://sivers.org/hellyeah)决策框架， [认为你必须偷走它们](https://sivers.org/uncomf) :

> 做你想做的东西需要很多个小时。时间不会突然出现。你必须把它们从舒适中偷出来。无论你以前做什么都很舒服。这不是。这会让*真的*不舒服。

西弗斯具体指的是那些会比我们意识到的更多地消耗我们一天时间的舒适的分心事物:社交媒体、电视节目、漫无目的的在线滚动等等。他不是说“不要花时间放松”；西弗斯只是指出，在许多情况下，我们选择了“舒适的”消费他人的创造性产出，而不是“不舒适的”创造自己的产出。

然而，同样值得注意的是，我们可以从不适中偷走这些时间。有很多很多方法可以优化、自动化甚至消除耗费我们大量时间的繁重工作——而你恰好在一个最好的网站上了解更多关于这些所谓的“生活窍门”这里有几个例子可以帮助你开始:

我目前正在起草一部小说，我决定每个工作日为这个特别的项目偷两个小时:一个小时在清晨，一个小时在下班后。第一个小时从舒适中被偷走；我不得不更早醒来，这意味着要想出如何在稍短的时间内获得相同质量的睡眠。这意味着在我的笔记本电脑上设置关机定时器，以确保我在睡觉前不会“上网”，在睡觉前调暗灯光并降低温度，以支持我身体的生理节奏，等等。

一个相关的好处是，我睡得更香了，半夜醒来的次数也比很久以前少了。

第二个小时是从不适中偷出来的。在工作日结束和晚上的工作之间要求一个完整的小时意味着学习如何优化工作任务，避免分心，并进行有效的休息——因为良好的休息和良好的工作会议就像网上杂货店送货和膳食计划一样，这是我为我的创造性工作窃取时间的另一种方式。

和以前一样，这种生活黑客还有很多额外的好处。首先，我用更少的时间完成了更多的工作。我也不那么不舒服了，例如，我很少会有那种懒散、恶心的“我不敢相信我花了一个小时浏览了一堆我都不记得的东西，我还没有解决我面前的问题”的感觉。

我知道我的个人生活窍门不会对每个人都有效。(我是一个独居的自由职业者，所以我对自己的日程和环境有很大的控制权。)但是如果你想花更多的时间做创造性的工作——或者真的做任何事情——你就必须花更少的时间做其他事情，这意味着决定何时以及如何偷回几个小时。

找到你感到无意识地舒服或沮丧地不舒服的时候，并从那里开始。
