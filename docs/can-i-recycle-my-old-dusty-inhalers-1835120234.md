# 我可以回收我的旧的，布满灰尘的吸入器吗？

> 原文：<https://lifehacker.com/can-i-recycle-my-old-dusty-inhalers-1835120234>

如果你是一个哮喘患者，你可能会有一堆用过的吸入器。你可能也不清楚到底如何处理它们——它们毕竟含有药物，当被扔进垃圾填埋场时，通常被视为危险废物。

Watch

在最近的一个 Reddit 帖子中，当 u/michiganbell_3 发布了一张药店用过的吸入器的收集箱的照片时，用户们目瞪口呆，他们第一次意识到他们的塑料箱和铝罐*可以回收利用——但这里有一个警告:这并不容易做到，至少在美国*

作为背景，吸入器在进入垃圾填埋场时特别危险，因为它们是受压的；如果 [刺破](https://www.redbags.com/inhaler-recycling/) 就会爆炸或者引起火灾。许多类型的吸入器也含有医疗废物，如果它们从罐中泄漏出来，会污染当地的水源。一些老版本的吸入器可能还含有消耗臭氧层的氯氟烃，但这些已经被逐步淘汰。

吸入器应该被适当地回收以避免这些危险，这似乎是合适的；然而，美国并没有现有的全国性的吸入器回收计划(这个问题中的 Redditor 可能来自像英国这样的国家，那里有*一个现有的吸入器回收计划)**。***

*那么，如果你在美国，你在回收吸入器方面运气不好吗？也许吧。与其他药物不同，CVS 和 [沃尔格林](https://www.walgreens.com/topic/pharmacy/safe-medication-disposal.jsp) 等一些主要零售商不会接受吸入器作为他们药品回收计划的一部分。事实是，如果你*确实*碰巧找到一家愿意接受你的吸入器的药店，他们有可能最终会把它们扔进垃圾桶。*

### *找一家当地药店或危险废物处理机构*

*你应该做的第一件事是查看吸入器上的处理说明；这些可能会详细说明您可以和不可以用您的特定设备做什么。*

*接下来，在网上搜索你附近的当地药店，而不是大盒子零售商，并与他们联系，了解他们是否会接受你的吸入器，以及他们将如何处理它们；他们可能把它扔进垃圾箱*或*带到危险废物处理设施。这不是传统意义上的回收，但至少他们会妥善处理你的吸入器，不会对环境造成问题。*

*也可以使用缉毒署 [网站](https://apps2.deadiversion.usdoj.gov/pubdispsearch/spring/main?execution=e1s1) 上的定位器查找当地药店；这通常是为了找到受控物质(而不是吸入器)的投放位置，但是，您可能会发现它对您的搜索有所帮助。*

*如果搜索结果不理想，搜索一下当地的路边回收项目，联系他们，看看他们是否接受吸入器；毕竟，每个程序都有不同的运作方式。他们可能会要求你联系一家药店退回你的吸入器(如果你的药店搜索已经没有结果，这是没有帮助的)，或者他们可能会建议附近的危险废物设施。*

*如果你想完全避免这一步，只需自己寻找你附近的危险废物转运设施；一旦你积累了相当数量的吸入器，这样做可能会有帮助。*

*如果你别无选择，把它扔进垃圾桶可能不是你能做的最糟糕的事情；美国食品和药物管理局 甚至在其网站上建议将吸入器丢弃在你的家庭垃圾中。为了确保它们不会泄漏到垃圾填埋场，试着用塑料袋将它们包起来，并与泥土或猫砂混合在一起。为了你自己的隐私，也一定要去掉标签上的任何个人信息。*
