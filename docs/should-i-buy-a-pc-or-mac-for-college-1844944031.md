# 大学该买 PC 还是 Mac？

> 原文：<https://lifehacker.com/should-i-buy-a-pc-or-mac-for-college-1844944031>

是的，我们本周正在涉入这些水域:十多年前，经典的 PC 或 Mac 辩论催生了无数来自苹果和约翰·霍吉曼的搞笑广告。(我肯定微软会乞求不同意苹果的解释，但我认为整个 T4 会同意 Zune 根本不应该存在。)

Watch

本周 Lifehacker 的 Tech 911 建议专栏的问题来自一位读者，她正试图在这两个重量级产品之间做出选择，以购买下一台大型电脑。给这个决定增加了一个小小的转折，那就是她打算为学校买一台电脑——我很惊讶这种事情还没有发生，但我想每个人都是从开始做起的。正如**罗震环**所写:

> “PC 和 Mac 哪个更适合大学？许多人说是 PC，因为大多数大学使用许可费用更低的 PC 兼容软件，而 Mac 通常不兼容或开源。感谢您的参与。”

### PC 还是 Mac？这要看情况，但是让我们开始吧

首先，我们来谈谈术语。Mac 也是 PC——当然是“个人电脑”的缩写。我认为我们在这次对话中应该使用的术语是 Windows vs. Mac，但即使这样也有点欺骗性。毕竟，苹果电脑可以运行 Windows 操作系统；不过，一台运行 Windows 操作系统的普通非苹果笔记本电脑不能运行 macOS 操作系统。至少，如果没有一些令人头疼的 hackintosh 问题，它是做不到的，但是我现在不会讨论这个问题。

所以，我对你的问题的第一反应是，如果你知道你需要一台能同时运行 Mac *和* Windows 应用的电脑，你就需要苹果的产品。句号。这是确保兼容一切的最简单的方法，并且设置 Boot Camp 在 Mac 上运行 Windows 虽然偶尔会很麻烦——并不太难。在两个操作系统之间切换就像你的 Mac 启动时按住按钮一样简单。

然而，苹果电脑的问题是你要支付苹果税。即使有了 [教育折扣](https://lifehacker.com/how-anyone-can-score-apples-educational-discount-1824123674)——请不要*而不是*跳过教育折扣如果你是学生、教育工作者或父母，想为自己或孩子买苹果设备——你仍然在寻找一台昂贵的笔记本电脑。一台运行速度一样快的非苹果的纯 Windows 笔记本电脑无疑会让你花费更少。当然，它看起来不会那么好，但是你可以在同样快速的系统上省钱。你只是不能运行 macOS。

也要记住，苹果 macbook——我假设你要的是笔记本电脑，而不是 Mac 台式机——在你挑选的时候升级会更贵。不幸的是，苹果并不像对待台式机 那样宽容笔记本电脑的 DIY 升级。

如果需要的话，不能保证你能自己升级你的非苹果笔记本电脑。而且也不能保证买一台升级版的 Windows 笔记本电脑不会花你一大笔钱。不过，一般来说，你可以找到很棒的 Windows 专用笔记本电脑——超极本、经济型笔记本电脑、游戏笔记本电脑等等——它们允许你以后用更好或更大的硬盘和更多内存来升级它们。如果这是你未来四年要用的东西，这些都是很好的选择。

我不确定我对你的软件相关问题有一个很好的答案。我已经有一段时间没有以学生或雇员的身份走进学术界了，但我确实记得 macOS 和 Windows 都有相当慷慨的折扣和/或免费软件。我建议你现在就去看看你的大学信息技术部门(或其他什么部门)能提供什么。如果你需要的所有软件都只能在 Windows 上使用，这可能会对你的购买决定产生一点影响。如果你专业的关键软件只能在 Mac 上运行，那就这样吧。

不过，还有技术支持的角度。坦率地说，苹果的技术支持选项通常比大多数其他电脑制造商更好、更快、更容易处理。我的意思是，你不能只是走到东芝商店——当然，一旦疫情结束——然后在那里获得系统帮助或授权更长的维修过程。

对于其他笔记本电脑制造商，您将不得不打电话、通过技术支持，并自己将系统运回来(理想情况下，在您等待该制造商向您发送 RMA 包装盒和标签之后)，等等。不要让我开始做排除 Windows 系统故障的噩梦，所有垃圾笔记本电脑制造商都喜欢在他们的设备上倾倒垃圾...

### 但是，不要忘记谷歌

如果你已经进入了苹果的生态系统——例如，你是一个自豪的 iPhone 用户——那么 MacBook 是一个你可以利用的所有有用的同步选项的显而易见的选择。我并不是说你应该坐在教室里，用 Mac 的 Messages 应用程序和朋友发短信，但你在纯 Windows 系统上不会得到这种支持。同样，如果你是安卓用户，Windows 10 中的 [手机](https://www.microsoft.com/en-us/p/your-phone/9nmpj99vjbwv) 应用可能会成为你新的最好的朋友。用 Android 和 macOS 过双平台的生活有点烦。

我们还忘记了晚宴的第三个受邀者:谷歌和 Chrome 操作系统。如果你现在就能买到 Chromebook(它们供不应求)，这可能是一个可行的选择，如果你只是需要一台能够处理基本功能的笔记本电脑:笔记、电子邮件、文档、电子表格等等。如果你需要任何特殊软件，比如 Adobe 套件、特殊工程软件或一些重型音频编辑应用程序，那你就没那么幸运了。但是，如果你需要一些轻便便宜的东西来学习，Chromebook 可能是一个不错的选择。

值得一提的是，我有一台 MacBook 和一台 Windows 笔记本电脑，我可以交替使用它们。我敢打赌，这对你的决定没有帮助，但我真的不介意长期坚持其中一个。事实上，我已经这样做了——在较长的假期里带上 MacBook，或者当我知道我要去我想去的地方玩游戏时，带上 Windows 笔记本电脑。

这是我现在能给你的最好的建议:弄清楚你的下一台笔记本电脑需要做的主要事情，弄清楚哪个平台能最好地支持它们，然后照着做。从本质上来说，纯 Windows 笔记本电脑和 MacBooks 一样好；这完全取决于*你*比硬件多得多。
