# 如何成为更好的朋友

> 原文:[https://lifehacker.com/how-to-be-a-better-friend-1835324667](https://lifehacker.com/how-to-be-a-better-friend-1835324667)

我们都有这样的习惯:陷入工作和睡眠的循环，忘记回复朋友的短信，或者错过生命中重要的约会。但是保持我们之间的友谊是很重要的；毕竟，有很多研究表明，友谊对我们的幸福有益，甚至可能延长我们的 [寿命](https://www.thefader.com/2017/03/08/science-of-friendship) 。鉴于成年后结交新朋友有多难，以下是如何巩固你已经拥有的友谊。

### 启动计划

如果你很古怪，是那种总是不愿意做出承诺的人——是时候改变了。首先，朋友很容易将这种不愿意制定计划的行为理解为“我对见你不感兴趣”，即使这不是真的。(如果它*是*真的，嗯， [那就另说了](https://lifehacker.com/how-to-dump-a-friend-1832297628) 。)

显然，对抗这种态度最简单的方法就是接受朋友的任何邀请。另一个解决办法是更频繁地启动计划，这样你的朋友就不会觉得你在回避它们。你应该启动*具体的*计划——时间、日期和地点——这样你就不只是发出你想见面的模糊信号。更好的是，如果可以的话，每周或每月定期去一次，这样你就会把它变成你日常生活的一部分。

如果你因为任何原因取消了计划，下次你需要成为发起计划的人；除非有紧急情况，否则最后一分钟取消是不明智的。不管怎样，保持联系是很重要的，即使只是随便发个短信。

### **根据他们的条件制定计划**

这给我们带来了另一个重要的提示:愿意制定不仅仅满足你需求的计划。当你遇见一个朋友时，决定一个对他们来说方便的时间和地点(不要在你公寓附近的街区)。这是一个简单的表示，表明你也关心他们的需求。

出于这个原因，我尽量制定对我的朋友来说方便的计划，无论是长途跋涉 45 分钟的地铁到布什维克，还是去他们工作附近的餐馆。

### 记住重要的日期

你是那种过了一个星期才祝另一个朋友生日快乐的朋友吗？它发生了，虽然这是一个特别犯罪的例子。正如 u/C8H10N4O2xlife 在最近的 Reddit 帖子中建议的那样(在 之前，我们已经写过 [)，不要忘记那些重要的日子，比如生日或你朋友生活中的其他重要事件。](https://lifehacker.com/how-to-maintain-your-friendships-1827484649)

他们写道:“很多时候，我会在日历上提醒 4-6 个人提到的任何大事——家庭旅行、他们期待的音乐会、他们担心的最后期限。”“为你不太亲近的人做这件事可能会很奇怪....但这帮助我记住了与那些 4-6 人的交往，他们似乎真的很感激。”

约会到了，给他们发一条简单的短信，问问进展如何——这些小动作其实很有意义。你表现出你在想你的朋友，当他们不在你身边时，他们仍然在你心中。

### 放下电话

如果你想成为一个更好的朋友，你也必须学会如何更好地倾听。(对我们大多数人来说，这可能说起来容易做起来难。)正如《快速公司》 所写的那样，有时候，我们可能倾向于倾听，因为我们是出于礼貌，而不是因为我们真的好奇。

麻省理工学院领导中心的执行主任哈尔·格里格森告诉《快公司》:“如果我们结束了一次谈话，却没有学到什么令人惊讶的东西，那我们就没有真正在听。”。

没有议程的参与是关键；不带判断和干扰地倾听。等到他们讲完一个故事，然后问问题(当然是真诚的问题)。

### 放弃坏朋友

为了成为一个好朋友，你也应该有对你好的朋友。我有一些更古怪的朋友，他们在我们见面或迟到一个小时之前改变了几次计划。

随着时间的推移，我认识到，当我知道他们不可避免地会改变或取消计划时，通过短信进行为期一周的交流并不总是值得的。从我 20 出头开始，我已经把我的友谊减少到三个我可以依靠的好朋友。

换句话说，成为好朋友的一部分就是学会何时对一些朋友说不，并把你的时间和精力花在其他人身上。你不需要和你喜欢的每一个大学朋友或同事都保持联系——毕竟这是脸书的目的——所以要认识那些对你最重要的人。
