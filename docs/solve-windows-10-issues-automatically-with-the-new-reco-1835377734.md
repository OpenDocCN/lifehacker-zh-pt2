# 使用新的“推荐故障排除”功能自动解决 Windows 10 问题

> 原文：<https://lifehacker.com/solve-windows-10-issues-automatically-with-the-new-reco-1835377734>

你应该 [安装最近的 Windows 2019 年 5 月 10 日](https://support.microsoft.com/en-us/help/4028685/windows-10-get-the-update) 更新，如果你还没有。该更新有一个很酷的新功能，建议故障排除，使诊断和解决常见的电脑问题更容易。



建议的故障诊断会自动识别并修复严重的电脑问题，甚至可以建议用户进行一些小的更改来提升电脑的整体性能，例如关闭或卸载危险的应用程序、更新驱动程序、删除不必要的文件等等。至少，它应该减少用户可能花费在浏览互联网搜索结果页面上的时间，以找出为什么他们的系统运行缓慢，或者为什么他们的互联网连接一直中断。

我们将向您展示如何启用推荐的故障排除，并解释您可以调整该特性设置的各种方法。

### 如何开启 Windows 10 推荐故障排除

Windows 10 将自动执行它发现的任何关键修复，但在打开建议的故障诊断后，用户将能够让他们的 PC 自行处理非关键的变化。

1.  单击快速启动图标或按 Windows 键调出开始菜单
2.  单击齿形设置快捷方式。
3.  进入**隐私>诊断&反馈>“推荐故障排除”**
4.  将建议的故障排除滑块转到**“开”**

从这里，您将能够选择 Windows 处理任何建议的修复程序的默认行为:

*   **修复问题前询问我:**当发现问题时，Windows 将向您发送桌面通知，并询问您是否希望修复该问题。(这是默认行为。)
*   **无需询问即可为我修复问题:**所有修复将自动完成，无需在之前或之后向您发送通知。Windows 将自动应用它认为最好的解决方案。
*   问题解决后通知我:同上，但问题解决后 Windows 会发送通知。
*   **只为我修复关键问题:**只应用关键修复，其余的将被忽略。您将不会收到任何通知。

只要建议的故障排除功能处于打开状态，您就可以通过转到**设置>隐私>诊断&反馈>“建议的故障排除”，**然后点击“查看历史记录”来查看建议的修复列表和任何更改的历史记录然后，您可以按时间顺序浏览 Windows 已经做出或建议的所有修复，以及它们是否成功解决了问题。这对于即使应用了 Windows 推荐的修复程序后问题仍未解决，或者您错过/忘记了以前的通知的情况很有帮助。

我们应该注意到，如果您已经将建议的故障排除设置为仅修复关键问题，或者如果您完全关闭了该功能，您仍然可以通过使用诊断&反馈设置页面中的内置“故障排除”选项来手动获得建议的修复。您将被要求选择您正在运行手动故障诊断的问题的性质(如互联网连接、硬件等。)然后单击“运行故障排除”Windows 将为您提供它认为最好的解决方案，就像自动推荐功能打开时一样。H 但是，这些手动扫描不会保存在您的故障诊断历史记录中以供将来参考。
