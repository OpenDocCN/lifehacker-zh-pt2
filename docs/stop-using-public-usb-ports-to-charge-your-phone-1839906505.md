# 停止使用公共 USB 端口给手机充电

> 原文：<https://lifehacker.com/stop-using-public-usb-ports-to-charge-your-phone-1839906505>

电池电源是非常珍贵的东西之一，尤其是当你在旅行时，需要使用机场为数不多的电源插座之一来为你的设备充电。



本周，洛杉矶县地方检察官办公室发布了一项警告 ，提醒每个人不应该让设备一直充电的一种方式:公共 USB 端口。

我们 [早在五月](https://lifehacker.com/how-hackers-can-steal-your-data-at-airports-1834976072) 就警告人们不要使用公共 USB 端口。

问题是，公共 USB 端口可能会被黑客攻击，这样当你充电时，他们就会在你的手机上安装窃取数据的恶意软件。这种被称为“果汁劫持”的黑客可能会导致骗子获得你的密码、个人信息等。不值得为了几分钟的糖果粉碎，对不对？

如果您使用其中一个端口(不要！)，请确保您不同意让该端口访问您设备上的数据。根据您的设备，您可能会在连接时看到一个弹出窗口，询问您是否信任该设备。你没有。

一个更好的选择总是简单地将你的手机或平板电脑插入交流电源插座。我强烈建议你在假期旅行前带上一个小型电涌保护器 。这种保护装置只占用很少的袋子空间，在与人协商让你共享一个电源插座时，它可以成为一个强大的工具。*正如这篇文章的一些评论者所指出的，* [*一个 USB 避孕套*](https://www.amazon.com/PortaPow-3rd-Gen-Data-Blocker/dp/B06XGM6LJB?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/stop-using-public-usb-ports-to-charge-your-phone-1839906505&asc_source=&tag=kinjalifehackerlink-20) *或者没有数据功能的电缆也是一个好主意，尽管你需要记住使用它们。*T15】

像便携式电池这样的东西也可以成为你的朋友，甚至当你在远离插座的世界里时，它们也可以帮助你补充一点能量。我是这款摩尔菲电池 的超级粉丝，如果你需要的话，它甚至可以给你的笔记本电脑充电，但是如果你只关心你的手机，你可以买一些更小更便宜的电池，也可以完成这项工作。
