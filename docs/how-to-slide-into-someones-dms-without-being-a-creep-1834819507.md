# 如何不让人讨厌地溜进别人的 DMs

> 原文：<https://lifehacker.com/how-to-slide-into-someones-dms-without-being-a-creep-1834819507>

互联网给了我们前所未有的接触每个人的机会。我们不仅可以在多个社交网络平台上看到我们的前任在做什么，而且我们现在(不管是好是坏)也可以看到我们最喜欢的名人和人物的短暂想法、偷拍照片和语无伦次的漫谈。我们还可以直接进入他们的 DMs 系统。

作为曾经给乔纳森·泰勒·托马斯寄过一封手写信件的人，她在一份《虎扑 T1》中找到了这个地址，这是一种危险的情况。除非你喜欢的人的个人资料被锁定，否则回复他们发布的每件小事的诱惑会非常强烈，很容易让人觉得你很讨厌。即使是在正常的、不出名的迷恋中，滑入 DMs 也是一件棘手的事情。

这并不意味着不能或不应该这样做，但一定的克制是必要的。事实上，我的一个好朋友和前编辑——艾米丽·麦库姆斯(Emily McCombs)以前是 xoJane 的记者，现在是 T2《赫芬顿邮报》的记者——和一个在几个州之外和她分手的男人保持着长期的恋爱关系。而且，作为一个已经在互联网上呆了一段时间的有魅力的女人，她已经收到了许多这种类型的信息，有些是好的，有些是坏的，并且好心地分享了一些滑动提示。

### DM 发送消息

发送任何形式的不请自来的信息都意味着你不得不接受没有任何回应。如果这让你深感不适，就不要发了。即使收件人以友好的方式回复，也不一定是邀请继续进行更长时间的对话。如果他们有兴趣这样做，他们会继续下去。“当一个人第一次溜进目的地管理系统时，我们很清楚你对他感兴趣。艾米丽解释道，并补充道，如果你希望事情朝着调情的方向发展，你必须要有耐心，尤其是当你和一个女人约会的时候。“礼貌是好的，只要你在该更上一层楼的时候听从我们的暗示。请记住，每个女性都会在互联网上不断收到厌女症和物化的信息——不要再增加了。”(这是不是意味着你可以对男人公开露骨？不，在那里也要保持尊重，但这种事情对女性和非男性- 介绍人来说特别累人。)

### 不要勉强

最好的对话是关于*事情*和*事情*，对某人外貌的评论不是最好的切入点- 。为了让事情更自然，艾米丽建议有一个话题，Instagram 或 Twitter 通常可以提供一个。“我会对一个故事做出回应——这感觉更自然，就像你刚刚开始一场对话。除了“我喜欢你的胸部”之外，你还有一个承认的理由。“(Twitter 没有故事，这使得在这个平台上实现这一点更加困难，但你可以随时回复推文，前提是你不要以回复- guy 的方式这样做。)

另一件需要考虑的重要事情是，你发信息的人是否愿意接受这种交流方式。如果他们的个人资料上贴满了他们重要的另一半的照片或参考资料，他们可能不想搭讪，所以保持中立和尊重，听从他们的领导。(你永远不知道某人是否是一夫一妻制，但不要假设他们不是。)解渴陷阱可能是更辣的东西的机会，但有时不是，因为人们出于各种原因发布各种自己的照片。即使你确定这个人在吸引调情的注意力，也要保持冷静和尊重，而不是闯进来，大放厥词。“老实说，我在 Instagram 上有过几次调情，所以我对此持开放态度，”艾米丽解释道。“当他给我发信息时，我去了他的页面，看了他的照片，我觉得他很可爱，所以我回复了他。鉴于我们之间的距离，我不希望这是一个比调情，但这些都很有趣！此外，他也很有礼貌——他没有立刻表现出性欲，也没有说任何猥亵的话。他等着我把它带向性感的方向。我不介意收到调情的短信，但这就像亲自接近一个女人一样——不要说任何你不会走到她面前当面说的话。”

### 不要期望太多

没有人欠你什么，仅仅因为你发了一条信息并不意味着你会得到回复，即使它是一条非常好的信息。这是你必须接受的事情。事实上，一个不回答本身就是一个回答。根据艾米丽的说法，如果有人没有回复你的第一条信息，他们很可能不会回复你的第二条信息。她说:“有些人不停地给我发短信，我不感兴趣，所以我不回复。”“最疯狂的是，他们一直不停地吸毒。接受暗示吧！如果一个女人从不回应你，她可能对你的关注不感兴趣。整个线程只是你在自言自语！”

当然，并不是所有的 DM 都与性有关，也许你根本就不想调情。就我个人而言，我收到了许多信息，只是对我的写作的赞美或对一个笑话的回应，这些总是受欢迎的，尤其是当我没有感到回应或继续一场长时间谈话的压力时。给收件人你的信息空间是很重要的，让他们知道你尊重这种空间可以走得很远。对艾米丽来说当然如此，”我在 Instagram stories 上发布了奇怪的口渴陷阱，他回复了一个。他只是回答了我问的一个问题，然后称赞了我的写作，并说了类似“我现在退出你的 DMs。“我们在 DMs 和短信上聊了大约一个月，然后我去芝加哥看他。我得说，那时事情已经很严重了。”

纠缠、抱怨或以其他方式要求这个人给你关注从来都不是一件好事，甚至不要想尝试这种艺术家的胡说八道。“我也不会回应挖苦的赞美或否定的话语管理系统——不要因为我的'自信'而称赞我，因为我是一个露出皮肤的大码女人，比如说，，”艾米丽说。最终，一切都归结于尊重对方的界限和时间。对好人说好话是好事，但要求回应就是要求占用他们的时间，他们可能没有多余的时间，或者他们只是想以其他方式利用他们的时间。发送一封好的 DM 的关键是冷静、冷静，并且对已经发送但从未收到回复感到舒服。我从未收到乔纳森·泰勒·托马斯的回复，结果一切对我来说都很好。
