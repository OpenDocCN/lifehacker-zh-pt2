# 我对丰田佳美有意见，它太小家子气了

> 原文：<https://jalopnik.com/i-have-a-complaint-about-the-toyota-camry-and-its-super-1835739156>

尽管你对它嗤之以鼻，但 2019 款 [丰田凯美瑞](https://jalopnik.com/tag/toyota-camry) 确实是一款 [好车](https://jalopnik.com/2018-toyota-camry-heres-why-its-still-the-sedan-king-1826940096) 。尤其是你跟 [300 马力发动机](https://jalopnik.com/look-at-this-nearly-40-000-camry-with-300-horsepower-1828716522) 一起订购的时候。它是轿车之王，因为它既可靠又宽敞。对此我只有一个抱怨。而且是小家子气。



*( **完全披露:**丰田借给美国一辆 2019 款凯美瑞一个星期，还加满了油。)*

是后备箱。或者更具体地说，行李箱锁释放按钮。几乎所有的汽车都有。你在行李箱盖上摸索它，通常在牌照上方，直到你找到它，推动它并释放闩锁，使盖子弹开。如果没有，你只需按一下按键。教科书上的东西。

好几次，我的手指沿着行李箱盖的底部，寻找按钮。我找不到。我手指碰到的唯一东西是备用摄像头。所以我想凯美瑞没有按钮，只是用钥匙代替。

然而！

凯美瑞有这样一个按钮，但它不在它应该在的地方，也就是正中间。想想你会在哪里找到释放按钮。行李箱盖的中间，对吗？没错。很高兴我们意见一致。

奇怪的是，凯美瑞的按钮放在行李箱盖的右侧。我是打开后备箱偶然看到才发现的。

当然，在这个角度很容易发现。但是，当你另一只手的钥匙上有一个非常好的按钮时，谁会去找那么久呢？你的手指会擦伤牌照灯，但没有成功，你会停下来用钥匙代替。

据推测，丰田将按钮移到了一边，因为中间的空间现在被备用摄像头占用了。但是有没有办法把它们都设计成中心位置呢？我不知道。

把这个凯美瑞按钮的位置放在“不要乱动”列表下。以前的条目包括保持齿轮选择杆作为一个实际的杠杆，而不是一个愚蠢的拨号，并给我一个手套箱拉手，而不是触摸屏按钮。

需要说明的是，这丝毫不会影响凯美瑞的驾驶、操控或其他享受。这只是最微不足道的烦恼。

谢谢你今天迁就我的小毛病。
