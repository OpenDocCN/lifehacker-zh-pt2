# 使用 Alexa 来帮助你用这个技能冥想

> 原文：<https://lifehacker.com/use-alexa-to-help-you-meditate-with-this-skill-1841097886>

今年，我试着冥想更多，这涉及到对你能在互联网上找到的所有冥想工具的深入探究，特别是那些 f re e 工具。



我最近遇到了一个我非常喜欢的 Alexa 技能: [正念冥想](https://www.amazon.com/Stoked-Skills-LLC-Mindful-Meditation/dp/B0784GXF2M?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/use-alexa-to-help-you-meditate-with-this-skill-1841097886&asc_source=&tag=kinjalifehackerlink-20) 。

该工具可以免费使用，并附带五种不同的冥想，长度从 3 分钟到 20 分钟不等。还有一个巨大的好处:他们使用讲师的声音，而不是 Alexa 的机器人声音。

以下是它提供的选项:

*   三分钟冥想正念呼吸
*   四分钟身体扫描，关注身体意识
*   五分钟放松冥想
*   十分钟内在智慧冥想
*   20 分钟的完全冥想，专注于身体的感觉、声音、呼吸、感觉等等

要使用它，您需要在 Alexa 设备上启用它。之后，你可以说“Alexa，打开正念冥想”，然后按照提示进入你想要的冥想，或者你可以说“Alexa，打开 10 分钟冥想”，直接进入你有兴趣尝试的选项。

我发现这些非常令人愉快，以至于我不介意一遍又一遍地做同样的提示，这是我尝试过的其他选项所没有的。

如果你家里有一个 Alexa 设备，那么它可以成为你的正念程序的一个很好的补充。
