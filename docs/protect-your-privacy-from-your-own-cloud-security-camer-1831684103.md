# 从您自己的云安全摄像头保护您的隐私

> 原文：<https://lifehacker.com/protect-your-privacy-from-your-own-cloud-security-camer-1831684103>

有多少人会在笔记本电脑的摄像头上贴上便利贴，因为他们担心有人会监视他们？在当今的技术世界，黑客访问网络摄像头和 USB 麦克风的传言一直存在，但当负责联网摄像头的制造商正是泄露你的视频的制造商时，情况就更糟了。



正如截击 最近的一份报告 [中详细描述的那样，亚马逊环形家庭安全摄像头存储了来自室外和室内摄像头的未加密视频记录。Ring 还雇佣了一些人来观看这些潜在的敏感视频，并对其内容进行标记，以帮助其虚拟邻居观察网络“邻居”的“机器学习”能力。](https://theintercept.com/2019/01/10/amazon-ring-security-camera/)

更令人沮丧的是，这些视频本可以通过简单的复制和粘贴分享给任何人。那些有权访问 Ring 数据库的人只需要知道一个用户的电子邮件地址，就可以进入他们的摄像头所看到的一切的直播画面。

根据 Ring 向 The Intercept 提供的一份声明，“这些视频完全来自 Neighbors 应用程序的公开共享 Ring 视频(根据我们的服务条款)，以及一小部分 Ring 用户，他们提供了明确的书面同意，允许我们为此目的访问和利用他们的视频。”

尽管如此，这仍然是一个很好的提醒，这种安全监控摄像头——旨在 [保护我们家的安全](https://lifehacker.com/four-tips-from-ex-burglars-on-how-to-protect-your-home-1789280763)——带有严重的隐私隐患。

## 云联网相机有什么隐私小技巧吗？

在这些情况下，你很难防止外人访问你的云连接摄像头。除了从家中移除基于云的摄像头和监控设备之外，没有什么万无一失的方法可以让控制云服务的公司无法访问它们。你只需要相信那些保护你的数据的人正在做正确的事情，就像你会相信[【脸书](https://lifehacker.com/dont-delete-facebook-just-be-smarter-on-facebook-1823922407)[【优步】](https://gizmodo.com/report-uber-can-track-any-customer-using-its-god-view-1660525110) 等等。

如果一家公司正在收集未加密的数据，那么该公司理论上有权访问所述数据包含的任何内容。即使一家公司阻止其员工查看其客户的摄像头反馈，黑客——或政府——仍有可能找到进入的途径。

### 假装麦克风很热，摄像机正在拍摄

如果你仍然想享受云连接相机的好处，在你离开时保护你的家，最好把所有相机、麦克风和类似设备视为任何人都可以观看或收听。有可能在某个时候，某个人可能*会*被——不管他是否有恶意或淫荡的意图。让安全摄像头和其他记录设备远离你想要完全隐私的地方，比如你的卧室，不要在它们面前做任何你不想有一天在互联网上曝光的事情。

虽然你在智能扬声器或电脑显示器上的网络摄像头前说的话也是如此，但你是否愿意用便利换取隐私，这取决于你自己。如果有人知道我对 Alexa 吼了什么，我不会那么担心，但我可能不会面对浴室建一个窝。

### 如果有疑问，用胶带粘起来

这听起来可能有点偏执，但是如果你怀疑一个设备，你应该关掉它，拔掉它，或者禁用它。在你的网络摄像头上贴上胶带或便利贴。当你不使用智能设备时，关闭它们(或者至少 [关闭它们的麦克风](https://lifehacker.com/how-do-i-use-a-smart-speaker-without-an-assistant-spyin-1831237434) )。不需要时拔掉 USB 麦克风。断开与陌生 wifi 网络的连接。配置用户设置以共享尽可能少的数据。不要将敏感数据存储在未受保护的云驱动器上。你在自己的隐私墙上创造的开口越多，滥用的可能性就越大。

### 注意用户协议和隐私政策

并非家中的每一个摄像头都受到了威胁，一些公司竭尽全力确保他们的用户放心——包括加密你的数据，对你的数据如何存储或使用过于透明，或者从一开始就不收集数据。

例如，脸书——尽管因利用用户数据销售广告而臭名昭著——对其新的 Alexa 驱动的门户视频聊天设备如何处理你的视频数据 很坦率。当然，当做出这样的承诺时，总是有可能出现精美的印刷、公关宣传和/或漏洞，但在亚马逊的环形相机中，没有提供这样的承诺。你可能需要挖掘才能发现一家公司到底在用你的数据做什么或不做什么，但这是值得调查的。

### 托管您自己的安全摄像头设置

像 Nest 相机或亚马逊戒指这样的产品的吸引力在于它们的便利性、可负担性和简单性。设置它们很容易，你可以毫不费力地将它们连接到你家中的其他设备，别人会为你存储所有数据。如果你不介意跑腿，你可以建立自己的系统，确保你之外的任何人都无法访问你的视频或你的订阅源。

在你着手这样做之前，确保你的预期设置 [符合法律要求](https://lifehacker.com/make-sure-youre-recording-people-with-your-home-securit-1787040532) 。至于网络本身，你需要 [使用](https://lifehacker.com/build-your-own-pan-and-tilt-security-camera-with-a-rasp-1784708079) [Arduino](https://www.arduino.cc/) 或 [Raspberry Pi](https://www.raspberrypi.org/blog/turn-your-pi-into-a-low-cost-hd-surveillance-cam/) 小型机构建自己的摄像头和安全设备 ，或者购买可以将录像转储到家庭服务器或 SD 卡的摄像头。你还需要给你的房子装上电线 [自己安装这些设备](https://medium.com/@Cvrsor/how-to-make-a-diy-home-alarm-system-with-a-raspberry-pi-and-a-webcam-2d5a2d61da3d) ，以及设置你自己的 [硬盘或服务器来存储视频录像](https://reolink.com/record-security-camera-for-24-7/) 。最后，你会想要调查可以帮助你的开源软件——尽管你也会想要审查你正在安装的东西，或者阅读他人的评论，以确保软件本身不包含一些令人不快的系统后门。
