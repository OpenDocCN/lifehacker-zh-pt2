# 如何观看今晚的民主党总统候选人辩论

> 原文：<https://lifehacker.com/how-to-watch-tonights-democratic-presidential-debate-1840490748>

在几名候选人 [因当地劳资纠纷威胁抵制](https://www.vox.com/policy-and-politics/2019/12/17/21025026/democratic-debate-strike-loyola-marymount-tom-perez-unite-here) 今晚的辩论后，官方宣布:2019 年最后一场民主党总统辩论重新开始。迄今为止第一次辩论，我们将在洛杉机的罗耀拉·玛丽蒙特大学见证七位候选人登上舞台*。*

这包括前副总统乔·拜登、南本德市长皮特·布蒂吉格、明尼苏达州参议员艾米·克洛布查尔、佛蒙特州参议员伯尼·桑德斯、亿万富翁汤姆·斯泰尔、马萨诸塞州参议员伊丽莎白·沃伦和企业家杨安泽。

与此同时，包括新泽西州参议员科里·布克和前住房和城市发展部长朱利安·卡斯特罗在内的候选人被排除在外，未能达到民主党全国委员会的标准。今晚的辩论也将是加州参议员卡玛拉·哈里斯而不是 首次出现，他在本月早些时候退出了竞选( [以](https://www.npr.org/2019/12/03/784443227/kamala-harris-drops-out-of-presidential-race) 缺乏资金为由)。

你对今晚由 PBS 和政治共同主持的三小时辩论有什么期待？嗯，可能会有更多关于弹劾和医疗保险或 T2 ll 的讨论。刚刚结束了对他在麦肯锡 的 [咨询工作的批评，以及争取黑人选民](https://slate.com/podcasts/what-next/2019/12/the-pete-buttigieg-campaign-thrust-the-mckinsey-into-the-spotlight) 的 [斗争，但蒂吉格很可能也会在一段时间内坐在这个棘手的位子上。](https://www.theroot.com/pete-buttigieg-is-a-lying-mf-1840038708)

如果你想了解 2019 年民主党辩论的气候结论，PBS 和 CBS 都将在美国东部时间晚上 8 点开始播出这场辩论。 [、PBS](https://www.pbs.org/) 、 [、CNN](https://www.cnn.com/) 、 [、政治](https://www.politico.com/) 也将在各自网站上在线直播辩论。如果你有 SiriusXM，辩论将在 116、454 和 795 频道同时播出。如果你想快速回顾一下上一场辩论， [我们来回顾一下](https://lifehacker.com/the-best-and-worst-moments-from-last-nights-democratic-1839972613) 。
