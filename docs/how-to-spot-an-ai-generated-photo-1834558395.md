# 如何识别人工智能生成的照片

> 原文：<https://lifehacker.com/how-to-spot-an-ai-generated-photo-1834558395>

我第一次向还是学龄前儿童的儿子解释说，卡通人物并不是真正的卡通人物，而是在电脑上绘制和制作的，这让他大吃一惊。看起来如此真实的东西怎么可能是假的？我想我现在知道他的感受了；最近，我对人工智能生成的照片了解得越多，我的大脑就越想反抗整个观念。

Watch

也许你听说过 [这个人不存在](https://thispersondoesnotexist.com/) 这个网站，它的特色是真实的( [但完全是假的](https://www.inverse.com/article/53280-this-person-does-not-exist-gans-website) )照片，照片上的人肯定存在于这个世界上，但却不存在。用计算机算法生成的人工智能照片已经变得如此先进，以至于区分真假变得越来越难。

当然，这可以为一些电影和电子游戏制作特效。我们正在规范为其他娱乐目的制造假人的做法，比如像这样的网站引诱我们去寻找我们的假替身。

但如果落入坏人之手，这些算法也可能有助于制造宣传和其他虚假媒体。在一个充满假新闻的时代，我们不得不学会不总是相信我们读到的一切；现在，我们也不应该相信我们看到的一切。

由杰文·韦斯特和卡尔·博格斯特伦 创建的网站[，旨在教导用户不要那么信任摄影图像，而要对潜在的合成“照片”更加挑剔:](https://lifehacker.com/how-to-detect-bullshit-1803144313?_ga=2.144665973.1940509479.1557157607-1723114163.1524514905) 

> 电脑不错，但是你的视觉处理系统更好。如果你知道要找什么，你一眼就能认出这些假货——至少目前是这样。用于生成它们的硬件和软件将继续改进，人类在伪造和检测的军备竞赛中落后可能只需要几年时间。

仍然有一些迹象表明 [一张照片是假的](http://www.whichfaceisreal.com/learn.html) 。韦斯特和博格斯特伦说，在我们假设一张照片是真实的之前，我们应该注意以下几点:

*   **水渍**。看起来有点像水斑的闪亮斑点是 NVIDIA 目前生产的“StyleGAN 算法”的一个显著特征。
*   **诡异的背景**。如果背景看起来像一张破损的照片，或者有奇怪的形状或其他扭曲，它可能是假的。
*   **不对称**。算法有各种各样的对称问题，包括面部毛发、耳环、织物，尤其是眼镜。
*   **头发**。假的人的头发可能看起来太直，有条纹，或者周围有光环或光晕。
*   **荧光出血**。寻找从背景渗透到照片中人的头发或脸上的亮色。
*   **牙齿**。这是我首先关注的事情之一。不仅仅是牙齿看起来不完美，而是它们很奇怪(比如，三颗门牙)，不对称，在一些照片中几乎有像素化的外观。

就目前而言，韦斯特和博格斯特伦在 [上说哪个脸是真实的](http://www.whichfaceisreal.com/learn.html) ，仍然有一个银弹可以让你用来判断网上的一张脸是否属于一个真实的人:

> 目前，我们还不知道有任何软件可以从多个角度生成同一假人的多个图像。所以如果你想确定你喜欢的人是一个真实的人，坚持看两张或更多的照片。如果他有一张大头照*和一张自己和老虎的照片*，他就是一个真实的人。当然，给你写信的人可能不是照片中的人，但照片中的人肯定是真实的。

准备好测试你识别真人和假人的能力了吗？你可以在这里 [参加他们的考试](http://www.whichfaceisreal.com/index.php) 。
