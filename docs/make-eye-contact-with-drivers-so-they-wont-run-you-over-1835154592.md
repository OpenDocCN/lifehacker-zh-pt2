# 和司机进行眼神交流，这样他们就不会从你身上碾过

> 原文:[https://life hacker . com/make-eye-contact-with-drivers-so-they-won-run-you-over-1835154592](https://lifehacker.com/make-eye-contact-with-drivers-so-they-wont-run-you-over-1835154592)

Lifehacker 的工作人员在纽约时代广场工作，这意味着我们经常处于被路过的汽车撞上的威胁之下。就在今天，一辆 SUV 差点撞倒我，尽管它闯了红灯。

Watch

好了，在 [Reddit](https://www.reddit.com/r/LifeProTips/comments/btk8a7/lpt_when_you_cross_a_road_always_make_eye_contact/) 上，u/cinema7D 有一个必要的(也可能是显而易见的)建议，下次你过马路时:与迎面而来的汽车司机进行眼神交流，这样他们就不会从你身上碾过。是的，你可能在进行眼神交流，以确保他们不会把你撞倒*，但重要的是他们在短暂的凝视中与你的目光相遇，这样他们就知道这是你的通行权。*

*当然，光是眼神交流是不够的，你也不想让自己产生一种虚假的安全感。行人可能会产生错觉，认为我们的目光已经被司机看到了，特别是如果他们戴着太阳镜，车窗是有色的，等等。*

*为了提高你的胜算，举起一只手，向司机挥手或示意一个友好的“停下”手势。这往往会迫使他们进行眼神交流，或者至少，让他们更有可能看到你。在理想的情况下，他们向你招手，这样你就知道他们看到你了，并且给了你优先通行权。*

*如果你骑着自行车，一辆车可能会在转弯处横穿你的道路，正如 u/ [trueanalytic](https://www.reddit.com/user/trueanalytic/) 所写的那样，**永远**也要看看路过的车轮。“这给了你最早的迹象，表明你即将变成一块薄饼，因此也是最有可能做出反应的时间。”如果你从一辆车的后面过来，无法与司机进行眼神交流，这可能是你避免碰撞的最好工具。*
