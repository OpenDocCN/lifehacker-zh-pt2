# 问那个咸咸的女服务员:当酒保“买”一杯饮料时，谁真正付钱？

> 原文：<https://thetakeout.com/who-pays-when-bartender-buys-comp-drink-1837906235>

> 嗨，小咸，我上周订婚了，我的朋友带我去一家不错的鸡尾酒吧喝酒庆祝。酒保举杯祝贺我们，然后当我拿到账单时，我看到他请了我的第一杯酒。我给了额外的小费，但不足以支付整杯饮料的费用。现在我想知道这是否意味着他失去了我的鸡尾酒的成本？实际上是从他的工资里扣的，还是酒吧只吃这个成本？我不知道这是怎么回事。谢谢你，瑞秋



嗨瑞秋，

拥抱、亲吻、祝福、庆祝等。关于你的订婚。我的建议是:尽可能多地参加免费的婚礼蛋糕品尝会。

至于优惠券是如何起作用的，是餐馆或酒吧在消耗成本。你不是在从酒保的薪水里偷钱，但是给 [小费有点额外的好处](https://thetakeout.com/ask-the-salty-waitress-how-much-should-i-tip-on-a-free-1822848886) 因为嘿，免费饮料。毕竟，酒保还是得制作和端上来。

大多数餐馆每周或每月都会留出一定比例的预算用于优惠券，这样员工就可以奖励忠诚的顾客或给他们的妈妈买一杯饮料，而不会让生意出现赤字。我甚至在一家餐厅工作过，那里的经理让每个人都知道我们一个月的运营费用远远低于预算，所以我们应该给常客和业内朋友提供更多的饮料。把所有的免费饮料都送出去，让别人开心一天，这让我萎缩的小心脏感到温暖。现在，我曾在一些地方工作过，那里的服务员和调酒师有权在每班或每周发放一定金额的礼券，还有一些地方的礼券需要经理批准。通常有一种方法可以在 POS 系统中跟踪这些赠品(这是销售点，有时也是你知道的一部分)。有时，它甚至会根据报酬类型进行细分:餐厅投资者的报酬、常客的报酬、行业朋友的报酬。然后，在月底，所有者和经理审查所有的费用，确保它们符合预算，并且没有人滥用这个系统。

所以这就是它的长与短，女孩。如果你们中的任何人想知道如何才能得到自己的免费轮次——请不要假装订婚，那只是悲伤——这里是我的绝密建议:

…做一个好人。记住，好人不会要免费的饮料。
