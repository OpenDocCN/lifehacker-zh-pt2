# 3 岁其实是最好的年龄

> 原文:[https://life hacker . com/age-3-is-actually-the-best-age-1840901074](https://lifehacker.com/age-3-is-actually-the-best-age-1840901074)

有小孩的父母经常说，可怕的两岁孩子在三岁时什么都没有。当他们长到三岁时，我们开始称他们为“三个管理者”，这是对他们日益不满的性格的一种肯定。但是这个词远远不足以描述每天家里有一个三岁的孩子是什么样的。他们不能像自己希望的那样交流，这让他们很恼火。他们不断试图做可能杀死他们的事，而你阻止了他们的死亡，这让*非常*恼火。

Watch

但我今天不是来告诉你三岁的孩子有多难相处。如果你养育过一个，你已经知道了(如果你没有养育过一个，你会认为我在夸大其词)。相反，我在这里告诉你，三岁不是最糟糕的年龄；其实是最好的。现在我将捍卫这一立场。

### 他们在那一年成长和发展得如此之快

在他们三岁和四岁生日之间的一年里，孩子们学习和成长了很多。突然间，他们的词汇里有了数百个单词(而不是几十个)，他们用短句交谈，回答简单的问题。他们开始给颜色命名，他们找出早晨、下午和晚上的区别，他们甚至能稍微数一下 T1。他们开始帮助穿上自己的衣服，骑三轮车，踢球，扔球和接球。他们可以双脚交替爬楼梯！

老实说，的发展里程碑列表非常令人印象深刻，随着所有这些增长，难怪他们不时会有一些不满。如果你的大脑每天都在不断学习新单词和掌握新的运动技能，你也会这样做。三岁就累死了。

### **三位教练让你为其他年龄段做好准备**

每个时代都有起起落落，有回报的时刻，也有挑战。但是没有一个像三岁那样，它的存在是为了让你为以后的所有年龄做好准备，并培养你的耐心。当然，现在他们长大了，他们会合法地和你顶嘴，或者不做作业，或者眼睛一直盯着智能手机屏幕。但实际上，这与你不得不像拖足球一样把他们拖出餐馆/商店/朋友家的(许多)时间相比根本不算什么，因为他们会因为这样或那样不合逻辑的原因尖叫和打闹。

如果你已经成功地度过了三岁，只有正常数量的吼叫和父母的崩溃，你就准备好应对父母可能带来的任何其他问题。

### 这是你最终渴望的一年

尽管听起来像杂货店里的老太太，她的任务是建议年轻父母“珍惜”最年轻的岁月，因为他们“成长得太快”——总是没有注意到父母凌乱或疲惫的外表——但我还是要说出来。总有一天，你会因为三岁而憔悴。

三岁是他们看起来真的很小的最后一年。到他们四岁的时候，随着他们完整的句子，稍微完整的推理技能和学前教育课程，他们已经失去了许多幼稚的品质。随着你远离发脾气，开始忘记及时去任何地方或做任何事情有多难，留给你的将是他们看起来如此渺小和珍贵的照片和他们所有可爱的错误发音的视频。

尽管当时这似乎是不可能的，但你会希望再一次，你能体验到他们“帮助”你叠毛巾是多么兴奋。或者他们想让你带着他们去任何地方，尽管他们完全有能力走路。或者努力自己拉上夹克的拉链。

总有一天，他们会觉得自己成长得太快了。
