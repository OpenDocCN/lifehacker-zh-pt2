# 商店里买的肉桂卷面团可以做很好的华夫饼干

> 原文：<https://lifehacker.com/store-bought-cinnamon-roll-dough-makes-an-excellent-waf-1840340363>

肉桂卷是非常标准的圣诞节早餐，但是那些用商店买来的面团做的面包很少像自制面包那样柔软黏腻。这并不是说它们不好，但我认为罐装小圆面包比华夫饼干好得多。



烤肉桂卷的好处有很多。它们做起来非常快——我的每个只需要四分钟——你可以一次做一个，这非常适合胃口不同的小群体。不要要求每个人在圣诞节早上，当兴奋情绪高涨，父母昏昏欲睡的时候一起吃饭，你和你的家人可以在心情好的时候，甚至在礼物之间，快速烤一个肉桂卷。任何没有用完的面团都可以储存在冰箱里，直到你又想吃华夫饼。

如果你觉得肉桂卷在早上的第一件事有点太恶心，我有一个好消息:waffling 过程产生的脆角和裂缝缓和了甜味，而丰富的面团使内部保持良好和蓬松。结果是一个枕头肉桂味的华夫饼，无论是在甜蜜还是美味的早餐情况下都是完美的。你可以把它裹上糖浆(或者自制的奶油奶酪糖衣)，或者用两个华夫饼干做一个非常令人兴奋的早餐三明治(有点像花式麦里德尔)。

只要打开一管面团，把你的(不粘的)华夫饼干制造机加热到高温，在制造机中间放一个肉桂香味的圆盘。如果你的有点粘，先用一些不粘喷雾。关紧烤箱，烤华夫饼干，直到外面变成浅棕色，你可以很容易地把华夫饼干从烤箱里拿出来，大约需要 3-5 分钟。用硅胶或木制抹刀将华夫饼取出，立即与黄油和糖浆、奶油奶酪糖衣或早餐三明治馅料一起食用。根据需要重复。
