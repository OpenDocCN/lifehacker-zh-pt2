# 如何笼络别人的猫

> 原文：<https://lifehacker.com/how-to-win-over-someone-elses-cat-1830434400>

猫不是狗。如果你想和一只新的猫交朋友，你不要把它当成一只狗。你耐心地找出对这只猫有效的方法，但你并不指望每次都有效。但是当你最终赢得一只猫的芳心时，和它成为朋友的缓慢过程会让你受益匪浅。

今年早些时候，我就帖子 [“如何让猫喜欢你”与三位猫行为专家进行了交谈](https://lifehacker.com/how-to-get-a-cat-to-like-you-1828501761) 这个帖子太受欢迎了，所以我们决定在生活黑客视频制作人亚当·鲍尔斯(Adam Powers)的猫科斯莫(Cosmo)身上试试这个技巧。在上面的视频中，我不仅用零食和玩具讨好 Cosmo，还给他空间和耐心。正如你将看到的，即使是像 Cosmo 这样友好的猫，也不总是容易的。首先，他是一个爱嚼口香糖的人。

现在，科斯莫和我永远是朋友了——有时会忽略对方，坐在沙发的两端，但也会依偎在一起，讨要虾点心。有时科斯莫也要虾。
