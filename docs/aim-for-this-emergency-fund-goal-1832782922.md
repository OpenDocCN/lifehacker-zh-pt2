# 瞄准这个应急基金目标

> 原文:[https://life hacker . com/aim-for-this-emergency-fund-goal-1832782922](https://lifehacker.com/aim-for-this-emergency-fund-goal-1832782922)

说到遵循标准的财务建议，我们中有多少人能说我们实际上在一个应急基金中存了三至六个月的生活费？

Watch

这是一个崇高的目标，但如果这对你目前的财务状况来说太过沉重，我可以建议一个不同的应急基金基准:你的健康保险免赔额。

您的 [免赔额](https://www.healthcare.gov/glossary/deductible/) 是在您的保险生效之前，除了某些预防性福利之外，您必须为医疗服务支付的金额。根据非营利组织员工福利计划国际基金会的 [报告](https://www.ifebp.org/aboutus/pressroom/releases/Pages/Average-Health-Care-Deductible-Nearly-$1,500-for-Individual-Coverage-Through-an-Employer-Plan.aspx) ，2018 年单一保险的平均可扣除额为 1491 美元，家庭保险为 2788 美元。

这比三到六个月的生活费用低得多(你的确切免赔额显然会有所不同)，但确保了如果你或你的家庭成员有医疗紧急情况，你可以支付(大部分)账单，而不必依赖信用卡、个人贷款或在美国黯淡的金融生态系统中越来越多的筹款活动。你仍然需要支付 [共付额和](https://twocents.lifehacker.com/the-health-care-terms-you-need-to-know-1828084354) 共保额，但至少你可以支付最初的账单。

如果你有 HSA 或 FSA，当然要尽可能用这些基金来支付费用，但不要把它们包括在你的应急基金总额中。这样你就有了一个额外的安全网来帮助你支付任何的意外账单。

这个数字只是一个起点。如果您的医疗紧急情况导致丧失能力或无法工作，您将需要更多的储备来帮助您支付日常开支。但是当许多美国人甚至不能支付 400 美元的紧急费用 时，这是一个很好的起点。
