# 最好的牛排沙拉不需要花哨的调料

> 原文：<https://lifehacker.com/the-best-steak-salad-doesnt-need-a-fancy-dressing-1836883456>

一份普通的沙拉可以通过一种极好的调料来制作，但是我最喜欢的沙拉之一——牛排沙拉——几乎没有调料。我喜欢用柠檬汁和——更重要的是——锅里的汁液来完成我的里贝叶状绿叶蔬菜，而不是搅拌油醋汁或奶油芝麻酱。



毕竟，潘汁是美味的，但如果我说这种策略优雅的懒惰不是其魅力的一部分，那我就是在撒谎。涂有黄油的大理石牛排是你的最佳选择，因为你需要这些热脂肪来腌制你的蔬菜。就像平常一样用平底锅煎你的牛排，加入至少一汤匙(或两汤匙)黄油来帮助形成外皮，然后取出牛排，让它静置。当它休息的时候，拿起你的(普通的，未穿衣服的和未装饰的)绿叶，直接从锅里淋上汁液，确保给每片叶子一点爱(如果需要，你可以扔)。在上面挤半个柠檬的汁，准备沙拉的其他成分。

至于其他非牛排配料，你需要一份洋葱(如葱、洋葱或大葱)、一份甜味水果(西红柿或草莓)和一份原味奶酪。把它们切碎并撒在蔬菜上，然后把你的牛排切片，把它们也放在上面。花点时间惊叹你为自己做的美食，然后狼吞虎咽地吃下去。
