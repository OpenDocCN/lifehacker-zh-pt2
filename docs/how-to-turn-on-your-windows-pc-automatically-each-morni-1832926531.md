# 如何每天早上自动打开 Windows PC

> 原文：<https://lifehacker.com/how-to-turn-on-your-windows-pc-automatically-each-morni-1832926531>

我们大多数人都有早晨的习惯，而你可能会在某个时刻使用电脑。虽然按下电源按钮并不难，但等待系统启动，然后输入 Windows 密码登录桌面，这些都是早上必须要做的非常耗时的步骤。如果幸运的话，你可以自动完成整个过程。

这个计划的一个问题是笔记本电脑制造商和主板制造商倾向于不同的标签设置。然而，下一次你打开你的笔记本电脑或台式机的电源时，按下任何你需要的按钮来加载它的 BIOS 而不是你的操作系统。

一旦进入，你将会寻找任何与能量相关的设置。这些可能被标记为简单的“开机菜单”，或“电源调度”，或诸如此类的东西。还可能有一个“RTC 报警”的“开机”设置，“RTC”的开机设置，或简单的“报警恢复”

不管它叫什么——如果选项存在的话——你应该能够安排一个你希望你的系统每天自动启动的时间。如果你计划在周末睡懒觉，但想确保你的笔记本电脑或台式机已经准备好，可以做你早上准备工作时要做的任何事情，你甚至可以安排哪*天*发生这种情况。

不过，你还没完。如果你真的想要自动启动，你也必须禁用密码提示，防止 Windows 直接进入你的桌面。在 Windows 10 中，你不能选择*或*设置密码——至少，如果你使用微软账户登录，你就不能设置密码，无论如何你都应该这样做，以便在不同系统之间同步你的设置。

这个过程会让你付出一些安全代价，但是如果你的台式机或笔记本电脑在一个房间里，并且你是唯一一个与之交互的人，你不应该担心那么多。(你仍然有密码和/或 PIN 码，只是启动时不再需要。)

要禁用 Windows 的密码提示，只需点击开始按钮并输入“netplwiz”取消选中名为“用户必须输入用户名和密码才能使用这台电脑”的选项，然后单击“确定”您将被要求输入您的 Windows 10 密码，然后您的系统将使用该密码自动让您登录。换句话说，错别字这部分，整个流程就不行了。

一旦你完成了这两个步骤——在你的 BIOS 中设置一个开机时间表，绕过你的 Windows 密码——每天早上你都会被你的桌面按照你创建的时间表问候。如果你的显示器或笔记本电脑足够亮，并且和你睡觉的地方在同一个房间，也许你甚至可以把它当作闹钟。
