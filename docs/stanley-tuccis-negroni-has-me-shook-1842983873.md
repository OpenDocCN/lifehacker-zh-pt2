# 史坦利·图齐的尼格罗尼震撼了我

> 原文：<https://lifehacker.com/stanley-tuccis-negroni-has-me-shook-1842983873>

他们说规则是用来打破的，但是在你能有效地打破规则之前，你必须理解规则。鸡尾酒有很多规则，也有很多违反规则的人，但史坦利·图齐违反了尼格罗尼规则，这让我开始怀疑我是否真的理解了这种饮料。(令人惊讶的是，我非常喜欢规则。我的规则往往与其他人的不同，但一旦它们被确立，它们就会被*遵循*。)

bOfvBY">虽然未经冷冻的苦艾酒也让德沃伊卡犹豫了，但她觉得如果有人能完成一个摇摇欲坠的尼格罗尼，那就是图奇。“我的理论？他是意大利人。他可以做他想做的事情，”她通过短信告诉我。“这就像他们拥有的魔力。当我去意大利的时候，他们打破了几乎所有的规则:咖啡和他们的开胃酒鸡尾酒，结果总是很完美。实际上，他用了一片橘子，而不仅仅是我喜欢的果皮。这不符合“现代经典”的规则，但我这么做了，而且更喜欢这样。我也很欣赏他有一种仪式，而不是这种完全不敬的把酒放进嘴里的方式。”</p><p
