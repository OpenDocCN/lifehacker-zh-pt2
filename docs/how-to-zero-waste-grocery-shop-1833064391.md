# 如何零浪费杂货店

> 原文：<https://lifehacker.com/how-to-zero-waste-grocery-shop-1833064391>

随着零浪费杂货店的兴起，现在购物时减少对环境的影响比以往任何时候都容易。但是*到底是什么*【零浪费】你问？



简而言之，这是关于产生更少的副产品(不一定是零——这只是目标)。想想所有那些塑料袋，所有那些你在购物之旅结束时最终会扔掉或回收的包装。这就是零废物倡议者试图减少的。

但是你能像全食超市一样在大众商店零浪费购物吗？是啊！尽管我们称之为“零”浪费(有时你不能放弃使用商店里的每一片塑料或纸张)。

通过致力于减少你的影响，你可以帮助减少容器和包装，它们大约占美国所有垃圾填埋场废物的 23%[。这里是你需要知道的关于零废物和如何在杂货店购物时最小化你自己的影响的一切。](https://www.cnbc.com/2018/10/19/zero-waste-markets-want-to-shake-up-grocery-shelves-and-your-shopping.html)

### 考虑你的消费

在你开始探索之前，想想你自己的消费和购买习惯——零浪费不是一个放之四海而皆准的命题。

布鲁克林零废物杂货店的老板卡特琳娜·博加蒂列娃(Katherina Bogatireva)在电话中说:“首先要做的就是看看你自己的垃圾袋。”。“这是知道应该避免什么和应该替换什么的唯一方法。这是个人的事情。解决办法不是某天醒来说，‘我要实现零浪费’，把我所有的东西都扔掉，换成不锈钢或玻璃容器。那不是最好的做法，事实上恰恰相反。”

相反，简单地了解你最浪费的地方。例如，如果你在购物后有很多塑料，零废物购物是你可以帮助抵消它的地方，通过尽可能经常地重复使用那些塑料袋或包装或购买未密封的产品。

### 批量购买

零浪费购物的一个重要组成部分是批量购买。不，这并不意味着买太多卫生纸，以至于你的壁橱和车库里堆满了卫生纸。它的意思是 [购买食物](https://zerowastenerd.com/how-to-buy-groceries-zero-waste/) 这些食物通常装在可舀的容器中出售，没有包装。全食商店有整个过道专门用于散装箱，一些商店完全致力于散装购物(在网上搜索附近的散装商店，或者你可以咨询 [零废物呆子](https://zerowastenerd.com/bulk-location-list/) 或 [无垃圾者](https://www.litterless.com/wheretoshop) )。

你可以在散装部门找到像糖、醋、坚果、大米、格兰诺拉麦片和燕麦这样的必需品。更好的商店也可能有散装肥皂、香料或宠物食品的通道。这对减少你的影响有什么帮助？你减少了你和商店原本会使用的大量包装(此外，你可以选择你想要的确切数量的产品，而不会因为你吃不完而导致食物变质)。

### 自带容器

理想情况下，零浪费的要点是尽可能重复使用，这意味着带上你自己的容器来收集食物。是的，你可以带自己的玻璃或塑料容器去大多数商店——如果你想确定，去任何健康食品店或全食超市。

在你往容器里放东西之前，找一个会称重的工作人员，这样你就可以支付正确的金额。(如果你担心这会很尴尬，让他们知道你正在努力减少塑料消耗！如果你经常在商店购物，他们会在几次访问后习惯你的要求。玻璃瓦罐是一个安全的赌注，装农产品和水果的布袋也是，正如野生极简主义者 T2 T3 推荐的那样。

买奶酪，面包，肉？向柜台后面的人提供您的容器。在奶酪的情况下，他们可能会为您提供一个没有玻璃纸包装的楔子。许多杂货店的面包店的后面也有未开封的面包。

当然，随身携带一个可重复使用的袋子。在你的车里或背包里多放一个包，这样你就不会缺少。

### 参观农贸市场

如果所有这些都失败了，你可以做一些事情来减少你在大众杂货店之外的影响。Bogatireva 推荐了农贸市场，它允许你购买和支持当地的农民，通常允许你购买没有不必要包装的产品。

她补充说，如果你的杂货店里没有散装产品，那就尽可能买大包装的不易腐烂的东西——比如大米、格兰诺拉麦片或你经常吃的东西。你可以减少包装，也不用去商店跑几趟。

如果你想更认真地减少你的影响，试试用我们的便捷指南制作堆肥。如果这看起来有点雄心勃勃，那就退而求其次，在网上搜索附近的堆肥投放点。
