# 馅饼面团不需要被黑

> 原文:[https://life hacker . com/pie-dough-dont-need-to-be-hacked-1840002661](https://lifehacker.com/pie-dough-doesnt-need-to-be-hacked-1840002661)

出于我永远无法完全理解的原因，馅饼制作中唯一有人试图破解的部分是最容易的。混合面团再简单不过了；你所做的只是用一点水将冷脂肪和面粉粘合在一起。这不是一个需要黑客入侵的过程。

另一方面，做馅饼是一件棘手的事情。但是没有人把同样神奇的、科学的黑客能量带到真正烦人的部分。把你的面团擀成一个完美的圆形 然后等待——等你的冰箱开始工作，等你的馅饼烤好，然后等它完全冷却——这是做馅饼的基本要素，你不能加快它们的速度。一旦你的面团碰到柜台，你的命运就掌握在上帝和热力学定律的手中。也许这就是为什么有这么多搅拌面团本身的技巧和诀窍；我们都渴望一种控制感，即使这只是一种错觉。

我并不是说你最喜欢的馅饼食谱是错误的，如果它依赖于这些技巧的话。馅饼是 90%的经验和 10%的运气，如果你已经知道什么是有效的，你就为生活做好了准备。这本书是写给新手们的，他们是有抱负的馅饼天才，在放弃购买冷冻馅饼之前，他们每年都会从无数的谷歌搜索结果中筛选一两次面包皮食谱。(我说这话是顺便零判断；冰冻面包皮是真正的生活帮。)你不需要食物加工机或伏特加，甚至是刨丝器来做一个很棒的馅饼皮——事实上，这些东西最终可能会让你的生活变得更艰难。

### 糕点搅拌机有点烂

面团搅拌器——有时被称为面团分切机——是馅饼皮黑客，我讨厌它们。我用了一年，因为我认为这是正确的事情，但它从来没有节省我的时间或以任何明显的方式使我的生活更容易。刀片似乎总是绕着黄油块弯曲，而不是切进去，这意味着我最终用手完成了这项工作。也就是说，我是一个理性的人，能够自我反省，因此认识到这可能更多地反映在我选择的工具上，而不是所有的面团搅拌器上。如果你爱你的，忽略我——但是如果你正在考虑买一个，我只是不认为他们是必要的。

### 磨碎的黄油比它的价值更麻烦

在过去的几年里，有一个巧妙的方法很受欢迎，那就是 [用箱式磨碎机](https://lifehacker.com/use-a-cheese-grater-to-get-super-flaky-pie-crust-1557282356) 磨碎冷冻的黄油棒，而不是用手指把它揉进面粉里。我认为这有两个问题。首先，磨碎一根黄油需要一段时间，这给了你足够的时间来融化或软化你拿着的一端。第二，磨碎的黄油块比大块脂肪块融化得快得多，所以你必须更加珍惜成品面团，以免到处涂抹。我只是不确定你能从中得到什么。

### 伏特加是用来喝的，不是用来做馅饼的

伏特加面包皮肯定是最受欢迎的，而且它的宣传很有说服力。由于面筋分子只有在有水的情况下才会连接起来，理论上讲，用伏特加代替一些水会产生一个面筋发育最少的柔韧面团。然后，一旦你的馅饼放进烤箱，伏特加就会完全蒸发，留下一层易碎的外皮。很好。

事情是这样的。和所有涂黄油的面团和面糊一样，产生过多面筋的绝对风险微乎其微，尤其是当你用手搅拌面团的时候。出于对过度工作的恐惧而不小心低估了面团的价值要常见得多——我认为这更糟糕。糕点需要一点面筋的发展，因为这是它们形成结构的地方。未充分加工的外壳可能会剥落，但祝你好运让它保持在一起。坚持用水做面包皮，把你的伏特加留到马提尼酒或 [一种奇特的注入](https://lifehacker.com/make-crazy-potent-horseradish-vodka-in-your-blender-1831650872) ，在那里你会真正体会到它的味道。

### 把食物加工机收起来

许多人喜欢在食品加工机中制作馅饼面团:你只需将配料倒入，脉动，然后就可以了。这并不是说它不起作用，而是它远没有想象中的那么高效。制作馅饼面团最耗时的部分是冷却面团并擀开——混合和揉捏大概需要一分钟。一旦你拿出食品加工机，把面团搅拌在一起，你并没有为自己节省多少时间和精力。

然后是*其他*食品加工黑客: [用一些面粉将黄油中的绝对 bejesus 闪电般地加工成糊状](https://www.seriouseats.com/recipes/2011/07/easy-pie-dough-recipe.html) ，然后将糊状物加入更大块的面团中。这被认为是创造了一个超级灵活的面团，仍然烤出片状和招标，它很可能——但我不卖。食品加工机非常强大，加热速度非常快；根据你特定的机器和批量大小，似乎很容易不小心融化黄油糊。再加上你还要刮到碗里，和剩下的食材叠在一起，这在我听来就是 [多余的菜](https://lifehacker.com/stop-hand-washing-your-dishes-1830493047) 。

### 有一个秘密——而且非常简单

当我说完美的馅饼皮只有一个秘密时，我是根据经验说的:依靠揉捏步骤，而不是最初的混合，来混合脂肪。与其花几分钟时间费力地用指尖将黄油切进面粉中，不如将冷黄油块扔进干配料中，轻轻搅平。在水中混合，把整个易碎的，蓬松的东西倒在一个柜台上，用你的手的根部涂抹面团几次，直到它粘在一起，就像这样:

法国人称这种技术为 fraisage，但我称之为常识:冷脂肪可以做成酥皮糕点，而你的手是面团接触到的最温暖的东西。你用那些滚烫的小手指接触裸露的黄油(或猪油或起酥油)的时间越少，它就越不容易融化。一旦它被面粉完全覆盖，你就不必那么小心了——因此有了*脆皮*。

这就是你要做的全部。馅饼制作过程充满了过度思考和事后批评的机会；不要把最简单的部分复杂化。你要担心的已经够多了。
