# 售价 400 美元的谷歌 Pixel 3a 完全重新定义了物有所值

> 原文:[https://gizmodo . com/the-400-Google-pixel-3a-completely-redefines-great-ban-1834573047](https://gizmodo.com/the-400-google-pixel-3a-completely-redefines-great-ban-1834573047)

去年， [Pixel 3 向](https://gizmodo.com/google-pixel-3-review-the-other-way-to-make-a-killer-p-1829749807) 证明了使用软件来制造一款功能强大、有思想的手机是可能的，有时甚至比那些塞满了最新高端组件的竞争对手更容易使用。但对于一部起价 800 美元的手机来说，如果能有一些额外的额外功能就好了，比如反向无线充电或背面多一两个摄像头。通过 Pixel 3a，谷歌将所有漂亮的 Pixel 软件和人工智能增强功能都塞进了一部价格只有一半的手机。在这个过程中，谷歌做出了市场上最超值的手机。

Pixel 3a 的起价仅为 400 美元(Pixel 3a XL 的起价为 480 美元)，你想知道的第一件事是谷歌要做出什么样的牺牲才能获得这么低的价格。从外面看，这些差异很难发现，因为 Pixel 3a 保留了 Pixel 3 的商标设计，一直到手机的双色背面、颜色鲜艳的电源按钮和柔软的触感。只有当你把 3a 和普通的 Pixel 3 并排放在一起时，你才会注意到它是由聚碳酸酯(塑料)而不是玻璃制成的，这使它变得更加温暖，不那么滑，同时也减轻了重量。

与此同时，对于 Pixel 3a XL(这是我在这篇评论中测试的)，谷歌新的中端手机和标准 Pixel 3 XL 之间的最大区别是缺少一个凹槽。取而代之的是，Pixel 3a XL 在其 6 英寸显示屏周围采用了更大的边框(而不是 Pixel 3 XL 上的 6.3 英寸)。谷歌还将 Pixel 3 的一个前置扬声器移到了手机的底部，这让 3a XL 有点突出，但谢天谢地，新的底部安装的扬声器已经与顶部的耳机配对，这样手机仍然可以提供立体声。

在 Pixel 3a XL 上，谷歌还将手机的分辨率从 Pixel 3 XL 的 2960 x 1440 下调至 2160 x 1080，这意味着如果你仔细观察，你会发现文本和照片的清晰度略有下降。但这真的很挑剔，因为 Pixel 3a 屏幕仍然采用了有机发光二极管显示屏，它不缺乏我们在最初的 Pixel 3 屏幕上获得的任何丰富和充满活力的颜色。

至于它的内脏，Pixel 3a 最大的变化是使用了高通骁龙 670 处理器，而不是骁龙 845 芯片，但同样，除非你真的试图给系统施加压力，否则很难在日常性能中找出任何大的下降。如果你仔细观察，偶尔你会看到在应用程序之间切换或滚动菜单时有一点点口吃或滞后，而像 *PUBG Mobile* 这样的游戏运行起来不像在芯片更强大的手机上那样流畅。

在一部 400 美元的手机上，这种打嗝是相当无害的。Pixel 3a 的 4GB 内存和 64GB 存储符合你对这个价格范围内手机的期望。说到电池寿命，Pixel 3a 的 3，000 毫安时电池在我们的视频播放测试中持续了令人钦佩的 11 小时 51 分钟，而 Pixel 3a XL 的 3，700 毫安时电池在 12:43 的表现甚至更好。对于 Pixel 3a 来说，这比我们从常规 Pixel 3 XL (11:24)获得的时间长半个小时，而对于 Pixel 3a XL 来说，几乎是整整一个小时。

至关重要的是，谷歌没有对 Pixel 3a 的后置摄像头进行改动，该摄像头的传感器和光学元件与标准 Pixel 3 相同。然而，你确实错过了标准 Pixel 3 正面的辅助广角自拍摄像头。虽然 3a 没有谷歌专用的 Pixel Visual Core 芯片来帮助处理图像处理，但谷歌定制了 3a 的照片管道，以确保 3a 不会在与常规 Pixel 3 的正面交锋中失利。这是一个相当大的声明，但在比较了两款手机拍摄的大量照片后，谷歌完全做到了。400 美元的手机从来没有这么好的摄像头。

在几乎所有的情况和环境下，Pixel 3a 都产生了与我从标准 Pixel 3 获得的相同的色彩丰富的 HDR+照片，同时还保留了 Pixel 3 的所有特殊拍摄模式，如 Super Res Zoom、Photo Booth 模式(现在有一个可爱的新吻检测功能)，以及最重要的夜视功能。谷歌甚至在 Pixel 3a 上添加了一个新的延时功能(这也是老款 Pixel 手机的目标)，让你可以将长时间的录制变成明显更短的剪辑，并可以选择 5 倍、10 倍、30 倍甚至 120 倍的速度。

为了测试这一点，我使用延时模式捕捉了日落时逐渐减弱的光线，因为它冲刷了附近的一些山脉和田野，将一个五分钟的片段浓缩成了一个 15 秒钟的短片。创建定时拍摄再简单不过了，我所要做的就是选择定时拍摄的速度，然后点击记录。现在要明确的是，手机上的延时模式并不是一个突破性的功能，但最终的剪辑清晰流畅，唯一分散云加速运动注意力的是昆虫云，它们显示为穿过画面的小黑点。我最大的遗憾是，我真希望我有更多的时间来抓住一个更长的记录。

对于那些仍然渴望谷歌旧 Nexus 手机的 Android 粉丝来说，你的冠军来了。Pixel 3a 有塑料背面，内置耳机插孔，纯 Android 体验，感觉像是 [Nexus 5](https://gizmodo.com/nexus-5-review-the-best-is-still-the-best-especially-1458003288) 的真正精神继承者。你唯一没有得到的是可拆卸电池，这是几乎没有手机了。我对 Pixel 3a 唯一真正担心的是，它是否会遭遇与 Pixel 3 早期相同的质量控制问题和缺陷。

400 美元的 Pixel 3a 不仅填补了预算/中端市场的一大空白，我还认为，任何之前考虑购买 300 美元 [Moto G7](https://gizmodo.com/the-moto-g7-is-so-close-to-budget-perfection-1833663097) 的人都应该认真考虑为 Pixel 3a 多花一点钱。虽然 Pixel 3a 没有像 [一加 6T](https://gizmodo.com/oneplus-6t-review-the-no-bullshit-best-deal-in-smartph-1830066240) 那样的纯计算能力，但在谷歌的 HDR+摄影、第一天更新的可用性以及谷歌照片中的双面和无限图片存储等好处之间，Pixel 3a 可能是一个更好的价值，也更友好地使用。

最终，虽然谷歌做出了一些妥协，创造了 Pixel 3a，但每一个变化都是有意义的，都是经过深思熟虑的。当你考虑 Pixel 3a 更实惠的价格标签时，所有这些权衡都是完全值得的。对于一部 400 美元的手机来说，Pixel 3a 是一个胜利。它看起来像一个像素，感觉像一个像素，它肯定像一个像素一样拍照。
