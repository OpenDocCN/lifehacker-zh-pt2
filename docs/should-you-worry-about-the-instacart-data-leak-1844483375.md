# 你应该担心 Instacart 数据泄露吗？

> 原文：<https://lifehacker.com/should-you-worry-about-the-instacart-data-leak-1844483375>

如果你是 Instacart 购物者，黑客可能会在网上出售你的数据。根据 Buzzfeed 最近的一份报告，至少有两个黑客社区上有 278，531 个 Instacart 账户以 2 美元一个的价格出售。每个账户包括全名、信用卡号码的最后四位数字和订单历史。这些列表出现在 4 月份,并不断更新新账户，就在昨天。

Watch

Instacart 否认其数据库遭到破坏，并指出，如果出现漏洞，它会立即通知用户，并自动强制他们更改密码，以防止不必要的访问。然而， [Buzzfeed 与](https://www.buzzfeednews.com/article/janelytvynenko/instacart-customers-info-sold-online) 多个账户泄露的个人进行了交谈。这些用户证实，这些黑暗网络卖家提供的信息是正确的，但表示 Instacart 从未联系过他们，也没有自动重置他们的密码。

这意味着 Instacart 要么在漏洞问题上撒谎，从未在第一时间发现漏洞，要么近 300，000 名 Instacart 用户受到各种外部网络钓鱼欺诈的影响。然而，这三种情况最终都导致了同一个结果: insta cart 账户数据被窃取并在网上出售，用户需要确保他们的账户安全。这包括:

*   更新您的密码，从受影响的帐户开始，然后对与其链接的任何帐户和/或使用相同密码的帐户重复。(R 记住总是 [让你所有的新密码都是唯一的](https://lifehacker.com/how-to-create-secure-passwords-that-arent-impossible-to-1825048324) 。)
*   [开启双因素认证](https://lifehacker.com/no-one-knows-about-two-factor-authentication-and-privat-1838913065) 和其他任何登录安全选项。
*   [使用加密密码管理器](https://lifehacker.com/the-five-best-password-managers-5529133) 轻松安全地存储您的登录信息。
*   用 [检查任何其他受影响的帐户和电子邮件地址，我已经被](https://lifehacker.com/how-to-make-sure-your-passwords-havent-been-stolen-1837305758)Pwned，并且更新那些。
*   监视可疑的金融活动和未经授权的帐户登录。
*   如果您看到未被识别的交易，请确保立即联系您的银行。

虽然 Instacart 用户可能不需要采取比这些更激烈的措施，但在接下来的几个月里，我会继续关注的可疑指控。你不必每天都扫描它们，但是每个月当你付账的时候，快速浏览一下你相关的信用卡账户是个好主意。

像往常一样，确保你尽一切努力避免安装粗略的应用程序和浏览器扩展，访问并非你所想的网站，或者回复含有你真实信息的虚假邮件。钓鱼攻击 [可以很老练](https://lifehacker.com/the-complete-guide-to-avoiding-online-scams-for-your-l-5420356) ，但是你只要有一点常识就能打败其中的大部分。
