# 重新加热炸薯条的最好方法

> 原文:[https://life hacker . com/the-best-way-to-re-heating-French-fries-1840958947](https://lifehacker.com/the-best-way-to-reheat-french-fries-1840958947)

冷薯条似乎是为数不多的不可回收的剩菜之一。一旦油炸食品开始冷却，蓬松的淀粉颗粒中的水分就会向外壳移动，使油炸食品内部呈颗粒状，外部呈糊状。这真是一个可怕的情况。

试图在烤箱中重新加热它们没有太大帮助——它只会让它们尝起来像热纸板，而不是冷纸板——重新油炸它们需要一点油才能达到普通的效果。重新加热薯条并获得最大松脆度的最好、最优雅的方法是把它们扔进我们最喜欢的便利器具:华夫饼机。

现在，我必须强调，这些华夫饼干薯条与真正新鲜的薯条不一样，因为没有一种简单的方法来重新水合这些淀粉颗粒。但是在华夫饼干机中粉碎它们会赶走浸湿表面的水分，同时还会创造出松脆的角落和缝隙(额外的表面积)，其质地非常类似于家庭油炸式早餐土豆。

只要把你的华夫饼机加热到最高温度，在中间放一把冷的、悲伤的薯条，然后把它们夹在热盘子里。一旦你看到暗金色的薯条袋，立即装盘食用。蘸番茄酱或辛辣蛋黄酱很好吃，但作为煎鸡蛋的平台也特别令人愉快，而且其超脆的特性使其成为浇在肉汁中的理想选择。
