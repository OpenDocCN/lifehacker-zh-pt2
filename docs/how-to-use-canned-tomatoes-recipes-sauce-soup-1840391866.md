# 夏天很短暂，但它永远是罐装番茄的季节

> 原文：<https://thetakeout.com/how-to-use-canned-tomatoes-recipes-sauce-soup-1840391866>

对我们许多人来说，西红柿是夏季 的完美食物，当花园和农贸市场洋溢着各种颜色、形状和大小的超熟多汁甜味时。我觉得这非常讨厌。因为我？我不喜欢生西红柿。我不喜欢它们的粉状糊状果肉，也不喜欢它们黏糊糊的裹着果冻的种子，也不喜欢它们薄薄的表皮，它们会不舒服地卡在我的牙齿之间，在我的上颚形成一层漆膜。在夏天，每个人都被束缚并决定把新鲜的西红柿放在任何没有被钉死的东西上，我总是把它们移到一边或摘下来，或挥走共用的盘子，或对意式面包或无尽的 caprese 说不。我不吃西班牙凉菜汤或新鲜的沙拉，如果你试图递给我一杯番茄汁，我很可能会吓得退缩。



说了这么多，我确实喜欢任何不是生的番茄。我喜欢番茄酱 [番茄酱](https://thetakeout.com/easy-homemade-pasta-sauce-recipe-versions-1838374011) 还有烤番茄和番茄汤。我喜欢烤番茄酱、填充番茄和番茄酱。我喜欢把它们晒干，包在油里，密封起来做成果酱，我非常非常喜欢罐头。罐装番茄是一种美丽和可能的东西，如果你的储藏室里有高质量的罐装番茄，你就离各种美味的东西不远了。

对我来说，罐装西红柿是超级熟和熟的完美结合。我倾向于寻找欧洲的西红柿，尤其是意大利的西红柿，众所周知意大利的西红柿更好。圣马尔扎诺是番茄完美的起点，我总是在标签上寻找 PDO 或 DOP 的名称，因为显然任何不可靠的罐头厂都可以把他们的番茄称为圣马尔扎诺，如果他们喜欢的话，但只有那些已经通过欧盟认证的番茄才被允许在标签上标注这些名称。在罐装西红柿中寻找这种东西可能看起来很傻，特别是当一些国产罐头的价格低至 99 美分时，但是如果你每磅花 4-7 美元购买新鲜的祖传西红柿，为什么不在你的罐装西红柿中寻找同样质量的东西呢？它仍然会花费你价格的一小部分。

我喜欢欧洲罐装西红柿的一点是，它们通常比你在杂货店能买到的任何新鲜西红柿都更成熟，味道更浓。这是因为他们不必前往罐头厂。取而代之的是让它们在藤上完全成熟，然后在采摘后的几个小时内进行加工:它们只需简单烹饪，让果皮滑落，然后包装在它们自己的汁液中。一般我喜欢买整只去皮的西红柿，因为我觉得它们更百搭；我的储藏室里通常都有完整的李子或罗马风味的西红柿和完整的樱桃西红柿。我可能还会准备一些番茄酱和双倍浓缩番茄酱。

即使你喜欢当季的新鲜番茄，那个季节也很短，尤其是在寒冷的州。一个高质量的番茄罐头会让你在接下来的一年中保持快乐。

加入切碎的洋葱、芫荽叶、辣椒和酸橙汁，罐装番茄很容易变成沙拉。玛塞拉·哈赞著名的神奇番茄酱 就是罐装的西红柿、洋葱和黄油，它是意大利面食中最美味的酱料。加入一些高汤和香料，你就有了完美的番茄汤。加入一些豆类、蔬菜和小的意大利面形状，就成了蔬菜通心粉汤。

有时候，我会把整个去皮的樱桃番茄过滤，涂上盐、糖和橄榄油，在慢火上烤两到三个小时，直到它们变成酸甜的小番茄葡萄干，放在你的下一个奶酪盘或熟食拼盘上，或者作为沙拉配菜。

为了快速做出美味的意式面包，我将一罐完整的李子番茄沥干并切碎，用大蒜和洋葱炒一下，然后用红辣椒片、茴香籽和牛至调味，如果我真的觉得很美味，还可以用一些切碎的油浸橄榄或马槟榔来调味。

对于最美味和最温暖心灵的冬季晚餐，撕开一条酸面团或丰盛的乡村面包，将其放入橄榄油、盐和胡椒中，烘烤直到面包片边缘变得焦黄酥脆，但内部仍然柔软。然后将两罐去皮的樱桃番茄和它们的汁液混合，用盐和胡椒调味，并将其全部堆在烤盘底部，顶部放上完整的甜或热意大利香肠和一些切片的洋葱或葱。用叉子或刀的顶端戳香肠，这样它们就不会裂开，然后在 400 度的烤箱里烤，直到香肠外面变成棕色，里面达到 180 度。用真正优质的橄榄油和切碎的新鲜欧芹、罗勒或薄荷点缀，你会在碗里看到天堂。
