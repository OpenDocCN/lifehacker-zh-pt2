# 把蔬菜放在架子上烤，以获得最大的松脆度

> 原文：<https://lifehacker.com/roast-vegetables-on-a-rack-for-maximum-crispiness-1839330580>

水是酥脆的敌人，完美的金黄色烤蔬菜——这很糟糕，因为蔬菜中含有大量的水。如果你只喜欢那些脆脆的边缘，试着去拿一个冷却架。更多的空气流通意味着所有的水分都会蒸发掉，所以你的蔬菜每次都会变得香脆。

Watch

基本上就是这么简单，但是有一些技巧可以充分利用这种技术。首先，不要害怕提高烤箱温度:425 华氏度-450 华氏度是最佳温度。至于准备，蔬菜薄片比大块蔬菜效果更好。照片中的美味南瓜又嫩又好吃，但如果我把切片做成 1/4 英寸厚而不是 1/2 英寸，它会变得更脆。此外，一定要用大量的油来搅拌你的蔬菜——甚至比你通常使用的多一点。没有热锅来促进褐变，每片蔬菜都需要一层厚厚的脂肪，这样它们才会变脆，而不是脱水变成蔬菜皮。

你可以把任何蔬菜放在架子上，但是我发现，嗯，*蔬菜越湿，*蔬菜就越有利于准备。(如果你曾经打开烤箱检查烤好的蔬菜，得到一大股蒸汽，你就知道我的意思了。)像西兰花、花椰菜和卷心菜这样的十字花科植物是最成功的——我的意思是，看看那些小花！—但是红薯和胡萝卜也很棒。加入一点点 [玉米淀粉](https://skillet.lifehacker.com/get-extra-crispy-roasted-vegetables-by-adding-a-little-1829270955) 或 [发酵粉](https://skillet.lifehacker.com/make-this-buffalo-cauliflower-dip-for-super-bowl-sunday-1832245628) ，你就可以享用香脆的烤蔬菜天堂了。
