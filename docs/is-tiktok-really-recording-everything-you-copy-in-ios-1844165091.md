# 抖音真的会记录你在 iOS 中复制的所有东西吗？

> 原文:[https://life hacker . com/is-tiktok-really-recording-everything-you-copy-in-IOs-1844165091](https://lifehacker.com/is-tiktok-really-recording-everything-you-copy-in-ios-1844165091)

苹果的 [iOS 14](https://lifehacker.com/get-apples-ios-14-and-macos-big-sur-wallpapers-right-no-1844137276) 有一个很棒的新隐私功能，每当一个应用程序检查到你 iPhone 的剪贴板上有什么内容时，它就会给你发送一个通知。这听起来像是一个聪明的补充，但它也发现了几个应用程序的一些潜在的令人不安的行为。

Watch

看看 Twitter 用户 Jeremy Burge 发布的这个例子，它显示了他在使用抖音时收到的所有 iOS 14 剪贴板警告:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1275896482433040386&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1275896482433040386&autosize=1) 

一些开发人员调查了这个问题，看看到底发生了什么。根据他们的发现和随后在 Twitter 上的讨论，这些应用似乎只是在读取剪贴板，而不是抓取、复制或保存他们在那里找到的数据——理论上是这样的。

一个应用程序这样做是有正当理由的，所以如果你在你最喜欢的应用程序中看到这个通知，我不会感到惊慌。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1276146864895778821&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1276146864895778821&autosize=1) 

一些开发者甚至发现他们的应用程序正在 [意外触发通知](https://twitter.com/twolivesleft/status/1275776460918157315) — ，即使他们没有在第一时间访问剪贴板——so很可能 iOS 14 中有新的 API 需要由应用程序实现，以防止意外行为的过度通知。

尽管如此，人们不禁要问，这些应用中有多少在偷偷访问 iOS 剪贴板，以及它们是否在我们不知情的情况下，在 Android 和旧版本的 iOS 上做着同样的事情。然而，我敢打赌，当看到这样的通知时，大多数人都会有一个问题:你应该担心抖音和其他应用程序可能会窃取你的剪贴板数据吗？

根据该公司的说法，我会对抖音说“不”。但我不会将这种全面的信任 [延伸到每一个应用](https://www.mysk.blog/2020/03/10/popular-iphone-and-ipad-apps-snooping-on-the-pasteboard/) ，尤其是如果你倾向于对来自未知开发者的未经测试、轻率地审查的应用进行修补的话。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1276190791002550272&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1276190791002550272&autosize=1) 

在一个完美的世界里，苹果会调整它的通知来更清楚地表明什么时候一个应用程序正在*扫描*你的剪贴板以查看是否有任何东西在上面，而什么时候它在主动*将*剪贴板上的信息用于另一个目的。即便如此，应用程序总是能够访问剪贴板而没有限制。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1276147520675160064&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1276147520675160064&autosize=1) 

目前，在你打开或使用应用程序之前，明智的做法是三思而后行。也许最好不要复制和粘贴重要的细节——你的电子邮件地址、你的密码、你的 2FA 代码等等。—如果可以避免的话。
