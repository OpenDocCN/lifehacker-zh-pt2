# 更新你的亚马逊 Echo Buds，防止它们过热

> 原文：<https://lifehacker.com/update-your-amazon-echo-buds-to-prevent-them-from-overh-1844405144>

亚马逊的 Echo Buds 刚刚获得了一个重要的软件更新，可以防止过热问题，并延长电池寿命。然而，你必须鼓起一点勇气在第一时间安装更新，因为在这个过程中你的 Echo Buds 可能会过热。

Watch

在给 Echo Buds 所有者的一封电子邮件中，亚马逊表示，过热只会在耳机处于充电状态的罕见情况下发生。亚马逊尚未透露问题背后的确切原因，但这已经足够危险，该公司正在要求所有 Echo Bud 所有者尽快安装版本 318119151 的软件更新。然而，更新过程需要将 Echo Buds 插入充电盒 15-30 分钟或更长时间——这种精确的情况有时会导致首先 过热。

幸运的是，软件更新是自动进行的，所以如果你最近给你的 Echo Buds 充电，你可能已经安全了。你可以在 Alexa 应用程序中确认你的 Echo Buds 的软件版本。打开应用程序，点击**“设备”**选项卡，然后进入**所有设备> Echo Buds。**

如果软件版本是 318119151 或者更高，你就没什么好担心的了。但是，如果是旧版本，您需要立即更新 Echo Buds。

### 如何更新你的亚马逊 Echo Buds

虽然亚马逊表示过热问题“非常罕见”，但在装有过时软件的 Echo Buds 上仍有可能发生。在执行更新时，请留意它们；如果有任何东西看起来、感觉到或闻起来很热，立即中止进程，然后 [联系亚马逊支持](https://www.amazon.com/gp/help/customer/display.html?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/update-your-amazon-echo-buds-to-prevent-them-from-overh-1844405144&asc_source=&nodeId=GJ97XK5PV93NNX5N&tag=kinjalifehackerlink-20) 寻求进一步帮助。

在你开始之前，你的充电盒和 Echo Buds 都需要有 30%以上的电池电量，并且在更新过程中，Echo Buds 需要保持在你的手机 25 英尺以内。

1.  打开亚马逊 Echo Buds，通过蓝牙将其连接到智能手机或平板电脑。
2.  在您的设备上打开 Alexa 应用程序，并确保 Echo Buds 显示在**“设备”**选项卡中。
3.  将 Echo Buds 放入充电盒中，然后盖上盖子。
4.  回声芽应该会在接下来的三十分钟内自动更新。
