# 你应该用亚马逊微笑吗？

> 原文：<https://lifehacker.com/should-you-even-bother-using-amazonsmile-1840347699>

如果你想支持一个非营利组织而不捐钱，你可以通过志愿服务或传播他们的工作来贡献你的时间。或者，你可以在亚马逊上购物来支持你最喜欢的慈善机构。

等等，什么？

亚马逊于 2013 年启动了亚马逊微笑计划，据该公司称，自那以来，该公司已向“数十万个慈善组织捐赠了超过 1.55 亿美元”。

作为背景，亚马逊仅在 2019 年第三季度就带来了 700 亿美元的收入。将该公司六年来捐赠给慈善机构的金额与它在一个三个月的销售收入进行比较，感觉非常可笑。但是，即使你只看 [利润](https://www.statista.com/chart/14887/amazon-quarterly-profit/) 而不是收入，亚马逊每年也能赚到几十亿。

这种差异是否意味着这只是给予的表象，你应该避开亚马逊微笑？

不完全是，尽管这不应该是你慈善捐赠的首选。

亚马逊*是否在 [捐赠给你选择的参与慈善机构的“合格产品”价格的 0.5%](https://venturebeat.com/2019/10/24/amazon-earnings-q3-2019/)之外*支持慈善机构。公司 [本周](https://press.aboutamazon.com/news-releases/news-release-details/amazon-surprises-hundreds-charities-fulfilling-their-amazonsmile) 宣布，它为各种组织完成了亚马逊产品愿望清单，并将在这个假期向支持无家可归者的组织捐赠 100 万美元的现金和实物。

“这是一个很好的选择，因为它是免费的，”慈善导航 的沟通经理阿什利·波斯特在谈到通过亚马逊网站购物时说。“把钱留在桌子上是愚蠢的”，因为亚马逊是根据你的购买量做出贡献的，她解释道。

波斯特承认，没错，一美元只值几分钱。“它永远不应该取代真正的捐赠，”她说。虽然一些组织通过亚马逊网站获得了巨大的成功，但波斯特说，这些往往是更大的“家喻户晓”的名字。如果你选择的慈善机构是地区性或地方性的，他们可能看不到参与的影响。

但这并不意味着你需要避开亚马逊来换取直接的贡献。如果你已经打算在亚马逊上购物，你可以一起使用它们。波斯特建议，把它用在你的假日购物上，但是省下一点额外的钱，直接捐给你心中亲近的慈善机构。说到慈善捐赠，现金是王道，因为它为组织提供了最大的灵活性来满足其当前的需求。

一如既往，你应该在捐赠前通过查看像 [慈善导航](https://www.charitynavigator.org/) 或 [引导之星](https://www.guidestar.org/) 这样的目录来 [研究](https://twocents.lifehacker.com/how-to-not-get-scammed-when-donating-to-charity-1827017641) 一个组织的使命和他们如何使用他们的资金。
