# 这些是 2019 年的税级

> 原文：<https://lifehacker.com/these-are-the-2019-tax-brackets-1831021335>

美国国税局宣布了 2019 年的税率和收入等级，除了因通货膨胀而对收入范围进行小幅调整外，它们看起来与去年相同。



请记住，这些是 2019 纳税年度的括号，这意味着它们将影响你在 2020 年申报的税收。

自 2018 年减税与就业法案生效以来，七个等级和边际税率不受影响。然而，如上所述，收入范围发生了变化，略有不同。

根据税法，标准扣除额增加了一倍，单身申报人的标准扣除额也将增加 200 美元，已婚夫妇共同申报的标准扣除额将增加 400 美元，分别达到 12，200 美元和 24，400 美元。

如果你想知道 2019 年几个更重要的财务数字，这里是你的 [401(k)，IRA](https://twocents.lifehacker.com/these-are-the-new-401-k-and-ira-contribution-limits-fo-1830175445) 和 [SEP IRA](https://twocents.lifehacker.com/the-sep-ira-limit-is-increasing-in-2019-1830310964) 贡献限额。你的 401(k)缴款将减少你的应税收入:用你一年的缴款乘以你的边际税率，你就会得到你在税收上节省的金额(例如，如果你在 401(k)中投入 5000 美元，而你的税率为 24%，你将节省 1200 美元的税收)。
