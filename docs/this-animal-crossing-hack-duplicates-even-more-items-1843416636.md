# 这个动物穿越黑客复制了更多的物品

> 原文：<https://lifehacker.com/this-animal-crossing-hack-duplicates-even-more-items-1843416636>

好了，已经一周了，任天堂还没有干掉 [最后一个*动物穿越:新视野*](https://lifehacker.com/duplicate-all-your-items-in-animal-crossing-because-no-1843260796) hack 允许你免费复制某些物品。万一它真的出现了，或者如果你想要另一种方法，让你可以复制你拥有的每一个物品——小故障的圣杯——另一种方法出现了。下面是怎么做的。

Watch

和往常一样，在阅读描述的同时(或之后)看一段视频可能是你理解并完美执行它的最佳选择。首先，你需要招募一些朋友或家人来帮助你。你们都需要坐在同一个房间里，其中两个或更多的人“注册”到你的岛上——换句话说，他们需要能够在你的岛上拥有房子，你们都将通过一个共享的开关屏幕进行游戏。

(如果你想自己控制额外的玩家，你也可以自己执行这个小故障。但那不觉得孤独吗？)

对于大多数人来说，这可能会排除这种物品重复的黑客行为，除非你愿意在你的填满的岛上对村民进行一点修剪，但也许免费、无限制的物品的前提太好了，不能错过。

无论如何，一旦你准备好了，通过游戏的邮件系统发送一到三个你想要复制的物品给你岛上的一个现实生活中的玩家。一旦他们确认收到，使用你的 NookPhone 上的“呼叫居民”应用程序召唤他们到你身边。

从那里，正如 [任天堂生活](http://www.nintendolife.com/news/2020/05/yet_another_animal_crossing_new_horizons_item_duplication_glitch_has_been_found) 所描述的:

> 让邮件接收者成为领导者，让他们打开他们的邮件。他们应该打开礼物并将物品扔在地上，之后你应该立即再次使用 Call Resident 应用程序与不同的玩家而不是邮件收件人玩游戏。一旦他们加入游戏，就结束对话；你应该会发现这些物品仍然在地上，如果你重复这个过程，你的邮件收件人仍然会在他们的邮箱里有这些物品，准备好让你一遍又一遍地做整个事情。”
