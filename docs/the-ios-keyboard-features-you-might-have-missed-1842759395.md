# 你可能错过的 iOS 键盘功能

> 原文:[https://life hacker . com/the-IOs-keyboard-features-you-may-has-miss-1842759395](https://lifehacker.com/the-ios-keyboard-features-you-might-have-missed-1842759395)

iOS 13 中的默认 iOS 键盘充满了你可能会错过 T2 d 的有用功能。在上面的视频中，我分享了我一直使用的三个功能。

### 轻松移动打字光标

没有什么比试图用你的手指精确地移动微小的打字光标更烦人的了。幸运的是，有一种更简单的方法:只要长按空格键，你就会看到字母键逐渐消失。按住手指的同时，只需向任意方向移动，就能轻松控制那个讨厌的光标。

### 特殊字符

长按某些字符会调出特殊字符的子菜单，如“é”。这在多语言对话和关于神奇宝贝的长篇大论中尤其有用。(有趣的事实:这一招在桌面 OS 中也有效。)

### 轻扫以键入

鉴于一些预测算法可能有多糟糕，滑动打字起初似乎并不直观。但是 iOS 上的预测惊人地准确，而且一旦你进入状态，滑动比敲击要快得多——尤其是如果你只能用一只手的话。
