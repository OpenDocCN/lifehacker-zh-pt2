# 你真的需要设置自动信用卡支付

> 原文：<https://lifehacker.com/you-really-need-to-set-up-automatic-credit-card-payment-1841766991>

嘿，你付了这个月的信用卡账单，对吧？

如果这个问题让你坐得更直一点，并再次猜测你是否真的支付了你的信用卡……嗯，我一点也不惊讶。

Watch

事实证明，即使 [我们在](https://lifehacker.com/automate-your-credit-card-payments-already-1828654501) 之前已经讨论过，你应该自动管理你的财务，这样你就不会错过付款，但你们中的许多人仍然错过了你的到期日。个人理财推荐网站 [WalletHub](https://wallethub.com/credit-cards#survey) 的一项新调查发现，18%的成年人预计在 2020 年至少会错过一次信用卡到期日。

也不是因为他们没有足够的钱支付。受访者报告错过信用卡付款的首要原因就是忘记了。高收入者(10 万美元或以上)因健忘而错过付款的可能性是低收入者(2.5 万美元或以下)的两倍。

避免这种常见失误的最简单的方法是做你能为自己设置的最好的财务技巧:在到期还款前几天安排你的信用卡定期还款。你不必确切知道你的余额会是多少；把它设置成打到你的银行账户上，支付最低要求的金额。一旦你看到交易完成，它会提醒你登录并在自动付款后付清所有余额。

WalletHub 首席执行官 Odysseas Papadimitriou 说:“这至少可以消除健忘这一潜在原因。”但是你仍然要确保你的支票上有足够的钱来支付你的最低余额。30%的受访者表示，没有足够的钱导致他们延迟付款。

如果你的信用卡到期日在每个月的某个困难时期到来——比如说，第三个月，你刚刚用第一天的薪水支付了房租——你可以要求你的信用卡发行商 [更改你的到期日](https://lifehacker.com/most-bill-providers-and-credit-card-issuers-let-you-cha-1784711419) 。你通常可以通过在线登录你的帐户并查看你的帐户设置来实现这一点；如果您不想通过电话询问，也可以使用您卡上的实时聊天客户服务选项。

当你登录时，检查你的信用卡收取的滞纳金。WalletHub 发现，只有 49%的人在选择申请时会考虑信用卡的滞纳金。快速提醒你不按时付款会损失多少额外的钱，可以促使你尽快把钱凑齐。通常，如果信用卡收取滞纳金，第一次将收取 29 美元，六个月内的任何其他滞纳金将收取 40 美元——这些费用的法定上限。

如果你的逾期付款确实是一次性失误，那么要求免除你的滞纳金是值得的:根据调查，几乎 90%的人说他们试图免除信用卡滞纳金是成功的。但是，如果你是一个惯犯，不要指望这么幸运，即使你有一张宣传没有滞纳金的卡。帕帕迪米特里欧说:“‘不收滞纳金’实际上通常意味着你第一次错过到期日时不收任何费用。”。“在那之后，所有的赌注都取消了。”
