# theres no such thing as weird meat 1844249803

> 原文：<https://lifehacker.com/theres-no-such-thing-as-weird-meat-1844249803>

大约三年前，我从猪头上取下一张猪脸，用香草、皮和香料将它包起来，放在料酒里浸泡了几个小时，做了一份猪肉罐头。这个过程被录了下来，拍成了视频(我第一次为 Lifehacker)，让很多人非常气愤。

我受到了一些威胁，其中一个是针对我的猎犬，它短暂地出现在视频中，但大多数反应都是震惊、厌恶和恐惧。(这段视频甚至因为“冒犯性内容”在一段时间内被脸书屏蔽掉了。)其他人只是“被吓到了”，这是你能做的最无聊的事情。



我对素食者和纯素食者的强烈反对并不感到惊讶。如果你已经决定肉类不适合你，那么一个以动物屠宰和烹饪为特色的视频可能不是你渴望的内容，我理解你不高兴背后的思维过程。然而，我发现令人难以置信的荒谬的是，当面对动物尸体时，许多食肉者表现出恐惧和厌恶。如果你不喜欢你的食物有一张脸，你可能不应该吃有脸的东西。

制作一块干净的小鸡块比制作一块特殊的猪肉要残忍得多(对环境也有负面影响)。(我猜有可能存在更人道的鸡块，但绝大多数是大规模生产和工厂化养殖的。)我在视频中使用的头是从离我住处几英里远的一个当地农民那里买来的，属于一头猪，它的饲养方式比你在杂货店里能买到的大多数肉都要人道得多。这种与肉类的脱节在美国是一个问题，特别是对于那些看着自己未来的晚餐被包装在整洁的泡沫塑料包装中长大的白人来说。

每当我写一种稍微不属于美国白人饮食的肉时，就会有一点反弹。(在评论中没有那么多，因为 Skillet 的评论者大多数时候都很好，但这种情况在 Twitter 上屡见不鲜。)任何种类的内脏，任何有大量骨头和结缔组织的东西(如鸡爪)，以及任何不是鸡肉、牛肉、猪肉或羊肉的东西都会引起广泛的不满，从呕吐脸表情符号到公然的种族主义和阶级歧视。



种族主义和阶级歧视不是偶然的。根据 [科学美国人](https://blogs.scientificamerican.com/a-blog-around-the-clock/offal-is-good/) 的说法，肌肉和内脏的这种划分可以追溯到奴隶制时代:

> 美国的另一个方面是历史性的——在很长一段时间里，每当农场的动物被屠宰，主人得到牛排，奴隶得到内脏...我注意到，尤其是在南方，乡村俱乐部的白人尤其瞧不起杂菜，他们对杂菜的厌恶几乎是发自内心的，带有一点阶级歧视和种族歧视的色彩。

这继续渗透到今天人们对肉类的态度中。虽然肝、猪肠和牛肚在美国南部有些正常，但你不会在中产阶级白人家庭的晚餐桌上找到它们。我在密西西比长大，每年夏天都去看望我的父亲和祖父母，虽然他们会谈论我们家的成员吃器官肉，但我从来没有吃过。我从未被直接告知这种肉是“给穷人吃的”，但我知道我祖母指的家庭成员是穷人，我年轻的心灵自己做出了这种联系。



可以预见的是，在过去的十年里，内脏在上流社会的美食家中变得相当受欢迎，因为上流社会的美食家总是试图将自己区分为不仅是“正宗”的欣赏者，而且不同于落后的“普通”中产阶级。你可以找到纽约、 [波特兰](https://pdx.eater.com/maps/16-places-to-stomach-offal-around-portland) 等城市的“最佳杂碎”名单，推荐的餐馆通常分为两个不同的类别:餐馆和食品车，提供有色人种制作的价格合理的食物，以及昂贵的、白人拥有的、经常获得提名的餐馆，这些餐馆出现在各种花哨的“名单”上区别不在于准备的质量，而在于上菜的环境、盘子，以及(有时)更小、“更整洁”的份量。(通常有一个“法国”，质量花哨的内脏；法国化是让白人接受事物的简单方法。)

在一个充斥着种族主义和阶级歧视的国家，对食物保持客观几乎是不可能的。食物是我们小时候学习“他人”的第一种方式。甚至在幼儿园的时候，美国孩子就“知道”PB&J 比肝泥香肠更“正常”，移民和有色人种的孩子从很小的时候就不得不听白人评论他们的午餐盒。这种种族主义充其量是令人讨厌的，但对一个孩子来说可能是毁灭性的。

拆除压迫系统需要的不仅仅是拓宽你的味觉，但是当谈论内脏和其他对你来说不寻常的肉类时，你当然可以从你的词汇中删除“怪异”和“恶心”这两个词。除了种族主义和阶级歧视，这没有任何意义。如果你是一个肉食者，看到有人使用动物的每一个部分——甚至是有眼球的部分——应该会让你感到兴奋。肉类不是放在塑料包装的泡沫塑料托盘里的整洁的小产品。这是一个他妈的动物。它有骨头，有脸，有内脏，有生命。如果这些让你恶心，那么也许吃肉不适合你。
