# 如何切换到 Twitter 的新 iOS“黑暗模式”

> 原文：<https://lifehacker.com/how-to-switch-to-twitters-new-ios-dark-mode-1833647727>

Twitter 今天宣布，其 iOS 应用程序有几个黑暗模式的扩展选项，如果你不想整天盯着应用程序的常规白色背景而失明的话。你现在有两个选项可供选择——结合了黑色和深灰色的**“暗淡”**模式，以及将背景设置为黑色的**“熄灯”**模式、。



此外，你现在可以使用新的**“自动黑暗模式”**功能，在晚上自动将你的 iPhone 或 iPad 切换到黑暗模式，在白天切换回“正常模式”。

要开始使用所有这些功能，首先要确保你的 [Twitter](https://itunes.apple.com/us/app/twitter/id333903271?mt=8) 应用通过 App Store 更新到最新版本。然后，为了以防万一，我会 [强制关闭](https://support.apple.com/en-us/HT201330)Twitter 应用，如果你的设备上有打开的话。重新启动 Twitter，点击左上角的个人资料图标，点击左下角的小灯泡，打开和关闭常规黑暗模式——“昏暗”版本。您现在应该会看到一个关于“自动黑暗模式”的提示，看起来像这样:
