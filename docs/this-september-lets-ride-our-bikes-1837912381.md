# 今年九月，让我们骑自行车吧

> 原文：<https://lifehacker.com/this-september-lets-ride-our-bikes-1837912381>

我们已经完成了跑步和游泳，所以让我们把九月的健身挑战集中在骑自行车上。如果你没有自行车，不要担心——室内自行车是非常允许的。今天，我们有一些给初学者的建议。

Watch

如果你想尝试室内自行车，一个团体班是一个不错的开始。教练可以帮助你正确地调整自行车——确保早点到教室，并解释你是新来的(或者你有一段时间没来了)。花时间问问题，总比发现自己在一个小时里不知所措、不自在，而周围的人似乎都知道自己在做什么要好。

如果你有一辆自行车，但很久没骑了，把它拖出来，给它一次调整。我们有一份 [指南，指导你自己做这件事](https://lifehacker.com/how-to-get-your-broken-neglected-bike-ready-for-spring-5898638) ，或者你可以带着它去当地的自行车商店。在把你的生活交给它之前，确保一切都正常运转。

也许你住在一个城市，你认为你会骑自行车上下班，或者也许只是更经常地骑着你的自行车随机出差。我们有一个在市区安全骑车的指南，许多建议也适用于其他类型的地形。

如果你想买一辆新自行车， [我们也为你准备了](https://lifehacker.com/the-beginners-guide-to-picking-the-perfect-bicycle-1782443592) 。那么告诉我们:H 你将如何庆祝这个九月的自行车运动？
