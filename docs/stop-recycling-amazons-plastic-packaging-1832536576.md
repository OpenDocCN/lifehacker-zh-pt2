# 停止回收亚马逊的塑料包装

> 原文：<https://lifehacker.com/stop-recycling-amazons-plastic-packaging-1832536576>

回收实际上有点复杂——绝不像把塑料扔进塑料回收箱那么简单。例如，大多数路边回收项目不接受塑料袋。



而现在，你又多了一个可以避免回收的东西:亚马逊的塑料包装。

据《华盛顿邮报》 报道，亚马逊的新型不可回收塑料邮件正在“搞糟”美国的一些回收设施，消费者将它们与其他塑料一起扔进了回收站。

“亚马逊包装遭遇了与塑料袋相同的问题，塑料袋在我们的回收系统中无法分类，会被机器卡住，”国王县固体废物部门的项目经理丽莎·塞潘斯基告诉华盛顿邮报。“把它们剪下来需要劳动力。他们必须停止机器。”

邮件上的纸质标签也是不可回收的，这使得问题更加复杂。

在过去的一年里，亚马逊开始用塑料邮件向世界各地的顾客运送一些包裹，作为其纸板盒的替代品。这是为了节省运输卡车和飞机的空间(根据一些专家的说法，这种塑料总体上减少了温室气体的排放)。

尽管如此，塑料也产生了另一个问题:一大堆不可回收的塑料需要处理，进入我们的垃圾填埋场(尽管亚马逊计划在未公开的时间推出一款完全可回收的纸质邮件)。

那么，你应该如何处理亚马逊的塑料邮件呢？简单——把它们扔进垃圾桶，这样它们就不会污染可回收物品和干扰机器。 [How2Recycle 的定位器](https://www.how2recycle.info/sdo) 可能会帮你找到附近愿意接受包装的店(提前打电话确认材料被接受，安全起见)。

如果没有，请将包装重新用于您自己的运输和退货。
