# 做饭的真正关键是冷冻一切

> 原文：<https://lifehacker.com/the-real-key-to-meal-prep-is-freezing-everything-1830778739>

这是一个老生常谈的建议，让你吃得更健康、花钱更少的最有效的方法之一，就是集中精力，自己准备食物。这很好，因为这是真的！

问题是，一个错误的 Instagram 或 Pinterest 喜欢上任何与烹饪、预算、健康或健身有关的东西，你的 discover feed 很可能会被过度的#mealprep 帖子淹没，提出让你吃东西的精心计划...一周中的每一天都是一样的早餐、午餐和晚餐。

这不是生活的方式。除了这样一个事实，即你通过创造这样一个多余的摄入量使你的营养餐变得有点缺乏营养，食物是生活中最大的乐趣之一，一周五天都吃同样的东西会减少快乐的潜在来源。(同时也增加了你仍然在 Chopt 浪费 13 美元的可能性，因为你无法忍受连续第四天吃你带来的任何 Tupper weed "prep "餐。)

有一个简单的解决方案，就在你的冰箱里。这个系统是这样工作的:任何时候你做一大堆东西(汤，米饭和豆子，一个玉米煎饼碗，一份炒菜....实际上几乎是任何东西)，把一些单独的部分放在上面——贴上用餐信息和日期的标签，如果你真的有条理的话——然后把它们扔进冰箱安全保存。

这样，你就慢慢建立了一个健康的、自制的冷冻晚餐的宝库，同时避免吃太多的任何一种食谱，以至于你开始讨厌它。例如，一周吃几次素食辣椒，然后带一份冷冻辣椒回来，几周后上班时再加热，那时你又想吃了。

对于大量的剩菜来说，这也是一个聪明的策略——不要强迫自己在感恩节后的几天里只吃火鸡和土豆泥，而是把一餐量的食物扔进一个容器，冷冻起来，然后在一月下旬享受感恩节的舒适食物。诸如此类。你仍在尽职尽责地为自己创造营养丰富、价格合理的膳食，但现在你的做法是，让自己把它们分开，享受不同的食物，避免让自己痛苦。不要冷冻和重复加热任何东西。
