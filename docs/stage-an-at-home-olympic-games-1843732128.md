# 在家举办奥运会

> 原文:[https://life hacker . com/stage-an-in-home-Olympic-games-1843732128](https://lifehacker.com/stage-an-at-home-olympic-games-1843732128)

尽管 2020 年的奥运会被取消了，但如果你在自家后院举办奥运会，你的家人就不必在今年夏天没有激烈的体育赛事了。你不需要能够完成一个后手翻或冒险让你的孩子扔标枪来完成这个动作——只要在创造性的活动中找到一些乐趣，用你的手机作为计时器，拿一张纸来记录分数。

### **举行开闭幕式**

奥运会开幕式和闭幕式的精彩程度不亚于奥运会本身，所以别忘了将它们融入你的主场比赛中。让孩子们参与进来，创造自己的旗帜、奖牌和火炬。

让每位参赛者选择他们想代表的国家或吉祥物。选择一个真正的国家是很棒的——当然可以打开额外教育机会的大门——但它们绝对是虚构的(我本人更倾向于代表纳尼亚和霍格沃茨参赛)。

此外，就像真正的奥运会一样，你可以让你的比赛持续几天。一天举行开幕式，另一天举行闭幕式。选择一个长达一个周末的比赛，以确保每个人都可以加入到乐趣中来。

### **计划多样化的活动阵容**

你会想在你的家庭奥林匹克运动会上安排各种各样的活动，这些活动不一定都是体育性质的(事实上，你越有趣越有创意越好)。你可以自己构思，也可以整合现有的游戏——这完全取决于你。根据每个参与者的年龄，游戏可以从简单到半挑战性:

*   比赛看谁能用吸管最快喝完一大杯水。或者举行一场饮食比赛。
*   利用你最大的室内地板空间，看谁能跳得最远，用胶带标记每个人的跳跃。
*   时间一 [寻宝游戏](https://offspring.lifehacker.com/buy-yourself-30-minutes-of-peace-with-an-alphabet-scave-1842593033) 看谁能最快找到所有物品。如果你的孩子有不同的年龄或能力水平，你可以允许他们在团队中做。
*   看谁能最快地从房子后面走到前门——蒙上眼睛，在同伴的指导下。
*   制作纸飞机，测量哪一个飞得最远。
*   如果你有游泳池，举办你自己的游泳比赛。

选择是无穷无尽的，设置也不需要太复杂——重点是和家人一起享受天伦之乐。

### **每个人都是赢家**

运动员训练了一辈子，为的是争夺奖牌，在真正的奥运会上代表他们的国家。在主场奥运会上，每个人都是赢家。不要让你孩子的好胜心战胜他们。想办法让比赛轻松一些，确保每个人最后都能拿到奖牌。真正的奖励是让每个人都站起来，动起来，希望能一起笑。
