# 让这些感恩节经典更加精彩

> 原文:[https://life hacker . com/make-these-thanksgiving-classics-even-better-1830236674](https://lifehacker.com/make-these-thanksgiving-classics-even-better-1830236674)

几乎每个感恩节餐桌上都可以找到某些菜肴，Lifehacker 餐桌也不例外。对于我们的小家庭餐，副主编爱丽丝·布拉德利负责土豆泥和填料——火鸡中的两个图标——我负责帮助她使它们发光。用两种秘密的(有人会说是“极化”)配料，一块随机的金属，加上一点舞蹈，爱丽丝和我煮出了一些美味的填料 ，和最美味、最梦幻的土豆泥。
