# 我真的需要锻炼前补充吗？

> 原文：<https://lifehacker.com/do-i-really-need-a-pre-workout-supplement-1835693628>

逛逛任何一家维生素商店，你会发现自己迷失在药丸和粉末的过道中，每一种都声称通过增强你的、能量或恢复来有益于你的锻炼。健身前补充剂，旨在跑步或举重前消耗，现在特别热，新品种不断涌现。但是他们到底是做什么的呢？它们值得争论或者花费吗？

### 锻炼前补充剂到底是什么？

锻炼前补充剂是你在锻炼前消耗的任何可能提高你表现的产品；例如，咖啡甚至可以被认为是锻炼前的补充。

然而，对于许多健身爱好者来说，突然想到的各种健身前补充剂以药丸或粉末的形式出现，标签如“ [极端](https://cellucor.com/products/c4-extreme-energy) ”和“ [点燃 AF](https://www.gnc.com/nitric-oxide/beyondRawLitAF.html)”；这些补充剂通常含有碳水化合物、甜菜根汁和氨基酸(其中包括支链氨基酸，这是一个你在网上购买补充剂时偶然发现的熟悉名称)，以及咖啡因。

肌酸是一些锻炼前补充剂中包含的另一种成分，但许多 [人](https://www.menshealth.com/uk/building-muscle/a750240/what-is-creatine/) 将其作为单独的补充剂消费。

### 它们真的有用吗？

虽然许多 [品牌](https://www.alaninu.com/products/pre-workout?variant=20246772383842&currency=USD&keyword=&creative=355427913249&gclid=EAIaIQobChMIr8uPwb744gIVkkYNCh3z0gzUEAkYASABEgLXKPD_BwE) 声称这些补充剂有助于你在健身房“最大限度地提高你的表现”，但这是一个棘手的说法。因为锻炼前补充剂没有受到美国食品和药物管理局的严格监管，任何公司都可以在没有证据的情况下做出非常夸张的声明。

就个人而言，这些成分似乎应该起作用，至少基于研究。咖啡因的功能由 [减轻疲劳](https://bjsm.bmj.com/content/early/2019/03/29/bjsports-2018-100278)；最近一项关于咖啡因对运动表现影响的不同研究的总括性综述发现，一剂咖啡因可能有助于你跑得更远，举起更多的重复次数或更多的重量。

正如 [GQ](https://www.gq.com/story/pre-workout-supplements-guide) 所写的，碳水化合物通过为你的肌肉提供糖原来工作，例如，你的身体会在举重时使用糖原。在锻炼前几个小时吃一顿富含碳水化合物的食物 [显示](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC4042570/) 可以提高耐力锻炼的表现。

与此同时，对甜菜根汁 [*的研究似乎*](https://jissn.biomedcentral.com/articles/10.1186/s12970-017-0204-9) 支持了持续时间非常短的高强度锻炼(想想短跑或任何其他涉及快速爆发强度的锻炼)可以提高表现的说法，尽管几乎没有研究。

氨基酸补充剂稍微复杂一点。我们自然产生许多氨基酸——蛋白质的组成部分——但我们确实*而不是*产生其中的九种，被称为必需氨基酸。BCAA 补充剂，其中三种氨基酸，在商店里很常见，经常吹嘘 [声称](https://www.gnc.com/protein_fitness/352126.html?mrkgcl=1098&mrkgadid=3317266399&product_id=352126&flagtype=nonbrand&vendor=none&channel=ppc&adpos=1o1&creative=304499128079&device=c&matchtype=&network=g&gclid=EAIaIQobChMIku_Dydr44gIVDiOGCh006AKYEAQYASABEgJiCvD_BwE&gclsrc=aw.ds) 包括对肌肉增长的好处，但研究是相互矛盾的。一项 [2017 s](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC5568273/https://www.ncbi.nlm.nih.gov/pmc/articles/PMC5568273/https://www.ncbi.nlm.nih.gov/pmc/articles/PMC5568273/https://www.ncbi.nlm.nih.gov/pmc/articles/PMC5568273/) [研究](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC5568273/https://www.ncbi.nlm.nih.gov/pmc/articles/PMC5568273/) 得出结论，BCAA 补充剂单独不能刺激肌肉生长，尽管当同时消耗*所有*必需氨基酸时可能会刺激肌肉生长。

理论上，所有这些成分加在一起应该有助于提高能量，对吗？看来是这样基于 [一](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC4947244/) [几](https://jissn.biomedcentral.com/articles/10.1186/s12970-014-0040-0) [的研究](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3361498/) 。然而，埃默里大学的一位医学教授认为，这主要是一种安慰剂效应，可能会让你在健身房中获得心理优势。
