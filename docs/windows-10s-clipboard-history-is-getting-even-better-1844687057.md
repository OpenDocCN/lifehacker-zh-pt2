# Windows 10 的剪贴板历史变得更好了

> 原文：<https://lifehacker.com/windows-10s-clipboard-history-is-getting-even-better-1844687057>

Windows 10 有一个内置的剪贴板历史，随着 Windows 10 Build 20185 的即将发布，它将变得更加方便，目前可供 Windows 内部人员测试。



我们之前已经讨论过剪贴板历史，这里是我的坦白:我说的“我们”是指“我”。即使我写了这篇文章，我也完全忘记了剪贴板历史，直到今天，当我意识到:1)它是存在的，2)我没有启用它，无论出于什么原因。

[哎呀](https://lifehacker.com/where-do-i-find-my-clipboard-history-on-a-pc-or-mac-1838529039)T3】。

所以，我回来提醒你，Windows 10 的内置剪贴板历史非常容易设置和访问。你只需要记住一个与你已经用来复制和粘贴的 **CTRL+V** 非常相似的快捷键。不要按那个，而是按 **Windows 键+V** 。当你这样做的时候，你会在你的屏幕上看到这个弹出窗口(如果你还没有启用剪贴板历史):

单击那个蓝色的大按钮打开它—就这么简单。你也可以访问 Windows 10 的**设置>系统>剪贴板**来启用剪贴板历史以及微软方便的剪贴板同步功能，以防你想在不同的 Windows 10 系统之间保持历史一致。(请确保您已经对其余部分执行了相同的步骤，以确保此功能已启用，并检查您是否在每个部分上都使用了相同的 Microsoft 帐户登录。)

现在，你只需要将自己从令人安慰的 **CTRL+V** 中解放出来，继续前进到未来的 **Windows 键+V**——这正是我第一次报道 Windows 10 的剪贴板历史时遇到的问题。如果能够从文本、图像或 HTML 代码的历史记录中复制和粘贴还不足以帮助你进行转换，Windows 10 Build 20185 将进一步增加这一优惠。

正如 [Windows 最新](https://www.windowslatest.com/2020/08/08/windows-10-getting-windows-10x-emoji-panel/) 报道，微软正在扩展剪贴板历史，为(可搜索的)表情符号、符号、kaomoji 和 gif 添加标签。对于一个人的剪贴板来说，这感觉几乎太*太*强大了，把它变成了你希望在你使用的任何消息应用程序上给朋友发短信时看到的那种界面。

尽管如此，这仍然是一个很有用的火力，可以节省你自己查找 [表情符号](https://www.emojicopy.com/) 或[gif](https://giphy.com/)的时间。我不能保证我会在生活中的每一天都使用这些新的剪贴板历史功能，但我在和朋友聊天时确实会查找一堆表情符号和 gif。在整个窗口范围内做到这一点的能力似乎很有帮助。

如果你现在就想使用这个新的剪贴板历史，你必须注册微软的 Windows Insider 程序(通过 Windows Update)。一旦你做到了这一点，注册开发频道更新以获得 20185 版本。希望你不会出什么差错。
