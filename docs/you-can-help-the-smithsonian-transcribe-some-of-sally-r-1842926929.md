# 你可以帮助史密森学会抄写一些萨莉·赖德的笔记

> 原文：<https://lifehacker.com/you-can-help-the-smithsonian-transcribe-some-of-sally-r-1842926929>

国家航空航天博物馆目前收藏了超过 38，640 页的资料，涵盖了萨莉·赖德作为物理学家、宇航员和教育家的职业生涯。这个庞大的收藏只有一个大问题:大部分是手写或用打字机打的，无法搜索。



博物馆估计所有的内容大约占用 23 立方英尺；然而，它可能很难搜索，也很难访问。

博物馆已经扫描了所有的文件，但现在 [它正在寻求帮助，以使扫描的档案更容易被其他人访问](https://sova.si.edu/record/NASM.2014.0025) 。

如果你想参加，你可以通过史密森尼学会的转录中心注册帮助。数字中枢于 2013 年推出，此后被用于许多转录项目。

目前， [你可以转录 1979 年至 1981 年期间莱德的航天飞机训练笔记](https://transcription.si.edu/project/24924) ，以及她担任的美国宇航局各委员会的一些笔记。

很多页面都是打字机页面，很容易阅读。你只需要花时间重新输入他们的内容。

注册转录服务是免费的，只需要创建一个用户名和提供一个电子邮件地址。

除了即将完成的《莱德笔记》,博物馆还在寻找志愿者参与其他一些项目，包括一战日记的抄本。您可以在这里 [浏览所有可用的项目](https://transcription.si.edu/browse?page=2&filter=all&sort=) 。

如果你正在寻找一种在家打发时间的方式，花几分钟(或几个小时)来打发时间可能是一种有趣的活动，可以帮助人们更好地获取未来几年的历史资料。
