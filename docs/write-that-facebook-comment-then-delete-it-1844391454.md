# 写下脸书评论，然后删除它

> 原文：<https://lifehacker.com/write-that-facebook-comment-then-delete-it-1844391454>

多年来，我们都试图离开脸书。 [科学告诉我们应该](https://www.nytimes.com/2019/01/30/health/facebook-psychology-health.html) ，也许我们中的一些人 [甚至设法做到了](https://www.businessinsider.com/people-who-delete-facebook-dont-return-using-zoom-chat-apps-2020-4)——至少直到疫情 [把我们拖回](https://www.vox.com/2020/4/29/21241601/facebook-coronavirus-pandemic-users-advertising-growth-making-losing-money-users-q1-2020-earnings) 作为所有常规的社交活动，更不用说教堂服务和尊巴课程，搬到了网上。即使你 [不能完全戒掉脸书](https://www.marketwatch.com/story/want-to-delete-facebook-read-what-happened-to-these-people-first-2018-07-27) (不过如果你想的话， [我们可以告诉你怎么做](https://lifehacker.com/how-to-delete-your-facebook-account-a-checklist-1829436824) )有一个简单的方法可以让你在网上的体验——就此而言，几乎在网上的任何地方——好一点。

如果你猜测它涉及到评论，那么，这显然不是你第一天上网。多年来，极端在线者的明智建议是“永远不要看评论”，但最近我一直按照一条可能更重要的法令生活:永远不要留下评论。你可以把它打出来，但是在你真正发布它之前，你绝对应该*删除它*。

尤其是在疫情时期，但也是从远古时代开始，在社交媒体上发表评论变得越来越丑陋。昨天， [《大西洋月刊》认为，北卡罗莱纳州支持面具和反对面具的团体](https://www.theatlantic.com/technology/archive/2020/07/facebook-reopen-north-carolina-masks/613978/) 之间正在进行的争斗是在线文化战争中与 COVID 相关的新战线的象征，但你可能不需要媒体指出任何例子，只要在自己的订阅中漫步就可以了。

昨天，我和一个高中朋友的“朋友”发生了争执——这个我从未见过面的人认识一个我已经 20 多年没有当面说过话的人——他告诉那些表达对秋季开学的学校的担忧的人，他们“生活在恐惧中”，害怕一种“存活率为 99.9%”的病毒，“病例数被夸大了，这里正在发生更大的事情。”这些评论对我来说是如此具有煽动性，以至于我打破了我的社交媒体规则，留下了一条评论，并在接下来的几个小时里后悔不已。事情是这样的:留下脸书的评论感觉很好...一分钟。我知道我是*对的*，她是*错的*，我有事实可以证明。不幸的是，她对我也有同样的感觉，并继续再次回应，用她自己(在我看来)歪曲和捏造的事实(没有双关语)来支持她的论点。十条评论下来，心情很差，后悔整件事。不用说，没有打开思维，也没有找到共同点。

我应该做的——也是我一直努力做的——是打出我愤怒但合理的回应，让它停留 10 秒钟，然后在点击“返回”之前删除它。这个效果出奇的好。概述我的论点并指出其他人错误的所有原因，我同样感到满意，但我不必处理负面后果——即不可避免的来回，在最好的情况下，我们同意不同意，在最坏的情况下，会变成浪费时间的堆积，我会把这种压力带到线下世界。也许你准备好争辩说，如果我们不纠正我们在网上遇到的错误信息，事情只会变得更糟——我可以看到你准备好评论的手指。但问题是，有很多证据表明网上的争论是不值得的*。加州大学伯克利分校和芝加哥大学的研究人员最近的一项研究( [由《南方生活》](https://www.southernliving.com/healthy-living/facebook-arguments) 报道)表明，在线交流的非人性化方面使得我们很难将我们不同意的观点视为来自一个“有能力的人”根据这项研究，与面对面辩论相比，网上辩论使我们认为那些我们与之战斗的人——通常只不过是陌生的名字和小小的个人资料照片，也许还覆盖着美国国旗——是“相对无脑的”。南方生活笔记:* 

> [这项研究]要求 300 名志愿者阅读、观看视频或聆听关于有争议话题的争论...【然后】回答关于他们不同意的观点的问题。当他们阅读不同的意见时，参与者对反对意见完全不屑一顾，认为他们“无知或无情”当他们*听到*或看到有人表达了他们不同意的观点时，他们的反应会更加友善和温和。

搜索你的感觉，你知道这是真的。网上吵架很烂。突然停止很难，而“键入然后删除”的策略可能是你避免卷入另一场全面战争所需的美沙酮。这不仅仅适用于脸书；在网上任何地方争论都是徒劳的，从脸书，到推特，到那篇让你生气的博文的评论区。你应该停下来。我们都应该停下来。
