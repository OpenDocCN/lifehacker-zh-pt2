# 我们现在称之为“物理距离”

> 原文:[https://life hacker . com/were-calling-it-physical-distanding-now-1842454451](https://lifehacker.com/were-calling-it-physical-distancing-now-1842454451)

我们*刚刚*习惯了“社交距离”这个词，但也许是时候继续前进了。在周五 的 [简报会上，世界卫生组织的玛利亚·范·克霍夫赞同一种开始变得越来越普遍的措辞变化:我们需要练习“身体距离”而不是社交距离](https://twitter.com/WHO/status/1241048714971475968)

Watch

重点没有改变:在身体上远离其他人。避免人群、聚会和任何你可能在他人六英尺之内的场合，或者你可能触摸他人触摸的东西的场合。确保或鼓励物理距离的措施是中国能够如此迅速地控制其爆发的的一部分。(广泛的测试和接触追踪是其他主要组成部分。)

但是，通过称之为物理距离，公共卫生官员希望传达我们并没有试图切断自己的社交。拥有一个支持性的社会生活比以往任何时候都更重要，以应对这些混乱时期的压力，所以我们应该积极寻求社会关系。

所以给朋友发短信吧。打电话给你妈妈。 [举办虚拟欢乐时光](https://lifehacker.com/organize-a-virtual-happy-hour-with-friends-over-video-c-1842334402) 或 [在保持距离的同时，玩一个有橱窗招牌的游戏](https://offspring.lifehacker.com/host-a-social-distancing-neighborhood-scavenger-hunt-1842427054) 。参加一个 [Peloton](https://vitals.lifehacker.com/peloton-is-offering-a-90-day-free-app-trial-so-you-can-1842394973) 或 Instagram 直播训练课程，让你知道你正在和其他人一起锻炼，即使只是远程锻炼。接近其他人，但要在远处做。
