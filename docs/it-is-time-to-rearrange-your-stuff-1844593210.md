# 是时候重新整理你的东西了

> 原文：<https://lifehacker.com/it-is-time-to-rearrange-your-stuff-1844593210>

如果你在 covid 之前没有在家工作过，那么你现在很有可能会这样做，而且你可能已经厌倦了查看你生活空间的内部。我已经在家工作了大约五年，作为一个很容易厌倦的人，陈旧的内部装饰是我非常擅长应对的事情。这可能看起来太明显了，但是给你的住所注入新生命的最简单的方法就是在没有计划或议程的情况下在四处移动你的东西。

即使你确定每样东西都已经放在了它最有用的地方，试着移动它。即使你确信椅子只能在那个角落里工作，试着移动它。即使那张桌子一直放在那里，试着移动它。即使你搬走的东西最终没有留在他们的新家，沿着这条路，你很可能会找到你认为已经丢失的东西，或者你需要扔掉的东西，或者受到启发将更多的东西搬到新的地方。

我最近用一些我看厌了的扬声器电线做了这件事。我的前夫在我刚搬进这间公寓时帮我安装了音响系统，他人很好，但是他这样做的方式给我留下了很多非常明显的电线侵入我的空间。我最终解决这个问题的旅程始于一个偶然的想法— “我想知道这一根线是否可以插在其他地方？“—在我意识到之前，我已经把每一根电线都移到了一个不同的插座上，在那里我再也看不到它们(或者被它们绊倒)。近五年来，我一直看着那些又笨又丑的电线，从来没有质疑过它们需要在那个确切的地方，因为它们一直都在那里(我曾诚实地认为我的前任选择了最好的地方把它们插上，因为他一直是我们关系中的技术支持)。电线被移动后，我觉得很神奇。我没有意识到他们制造的视觉噪音让我有多心烦意乱。我现在的空间真的感觉变大了。我在娱乐中心后面的电线中找到了一些我之前指责我男朋友丢失的太阳镜。

移动一个东西常常会滚雪球般地变成移动几个东西。当我在移动电线时，我发现自己在重新定位一个装满猫玩具的猫托架，以便找到一根特定的电线。该运营商一直坐在娱乐系统和旁边的一个小角落，使看起来相当拥挤。没有好的理由让它在那里；它已经成为那个房间视觉景观的公认部分。但是电线的成功迁移让我感觉相当大胆——大胆到质疑一切！——所以我把猫驮架搬到了浴室附近小走廊的手推车上。现在娱乐中心附近的角落看起来整洁多了，也不那么拥挤了。

所以只要移动一些东西。不一定是什么大事，但也可以是。质疑每个物体的位置。把它移动几英寸，或者几英尺。重新摆放架子上的东西，打开盒子，将小摆设移到房间的另一边。只是移动*的东西，*没有一个心中明确的目标，然后移动第一次移动发现的东西。不断移动物品，直到空间变得清新。我建议从电线开始。
