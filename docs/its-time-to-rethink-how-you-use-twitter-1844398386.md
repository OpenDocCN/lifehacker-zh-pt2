# 是时候重新考虑你如何使用 Twitter 了

> 原文：<https://lifehacker.com/its-time-to-rethink-how-you-use-twitter-1844398386>

昨天，Twitter 的安全遭到巨大破坏，导致各种高调的账户——从比尔·盖茨到巴拉克·奥巴马，甚至是一直沉默的苹果——在 Twitter 上发布骗局，引诱人们向匿名比特币钱包汇款。对于我们一段时间以来看到的最大的社交媒体黑客攻击之一，你无能为力，但现在你可以采取*步骤来抵御未来的灾难。* 



嗯，除了 [停用你的推特账号](https://help.twitter.com/en/managing-your-account/trouble-deactivating-twitter-account) ，那就是。

### 推特到底怎么了？

推特真的，*真的*这次搞砸了。这是毫无疑问的。我敢肯定，品牌、有影响力的人和名人都很愤怒，他们的安全措施被绕过了——甚至是 Twitter 为超级名人账户设置的更强保护——诈骗信息以他们的名义发布。

让我换个说法:“成功的诈骗短信。”与出现在被攻破账户上的信息相关联的比特币地址从轻信的用户那里获得了的巨额捐款。(这让我想到了推特灾难后的第一条建议:*互联网上没有人想给你免费的比特币或钱，尤其是当他们要求你先给他们钱的时候。来吧，大家。)*

### 我能做些什么来防止这种袭击吗？

我对[的故事](https://gizmodo.com/apple-musk-gates-bezos-and-everyone-you-ve-ever-hea-1844396384) 背后的骇客比对后效更感兴趣。因为即使你做对了*每件事*——强密码、双重认证、停用对你不再使用的第三方应用的访问，以及 [启用密码重置保护](https://twitter.com/settings/security) ，你仍然可能成为这次攻击的目标。有那么严重。

删除 Twitter 当然是一个选择，但感觉不是最好的选择。不管你喜不喜欢，这个*就是*事实上的全球实时更新交流平台。从抗议活动到 [天气](https://www.theverge.com/interface/2020/7/15/21325708/twitter-hack-global-security-crisis-nuclear-war-bitcoin-scam) ，再到——不幸的是——国际外交和其他可怕的政治决策，Twitter 是一个非常有用的信息来源。它允许那些通常不会见面的人进行对话，尽管它的存在意味着每个人，包括那些持有可悲观点的人，都有了代言人和无限的放大潜力。

如果你能给我一点时间离开我的演讲台——事实上，很多人*威胁*离开社交网络，但是 [从来没有真的离开过](https://lifehacker.com/dont-delete-facebook-just-be-smarter-on-facebook-1823922407)——如果你打算继续使用 Twitter，我会告诉你*应该采取的一些措施。*

### 清理您的 Twitter 订阅源

我曾经赞同这样一种理论，即把一个人的个人生活上传到网上让全世界都可以看到，也为一个人生活中的各个时刻创造了一个极其有用的记忆、思想和感情的档案。诗意。然而，我在社交媒体上呆得越久，我就越意识到我并不真的关心挖掘毫无意义的过去、T2 和 T3 的记忆。

我从来不会回头去看我在 2014 年发了什么微博，我也不认为我会需要或想要去数字记忆里寻找过去的任何东西。强大的记忆往往会伴随着一个人；关于我多年前点的塔可钟的古怪推特毫无价值。我也不需要或希望人们看到我几年前在推特上发的东西，并对我的帖子感到悲伤。

所以，我删除了我的旧推文。我们已经讨论过如何做到这一点 [之前的](https://lifehacker.com/delete-old-tweets-and-protect-your-online-data-with-jum-1834240043) ，我很高兴地报告，我的新宠(但挑剔的)工具 [**半个人**](https://semiphemeral.com/) ，做了一件体面的工作，消除你发布到 Twitter 上的旧思想(以及你的各种喜欢和转发)。还有很多其他的服务我没有尝试过，比如更简单的[**tweet delete**](https://tweetdelete.net/)，所以如果我们推荐的东西不符合你的口味，你有很多选择。

这样能防止以后有人闯入你的账号吗？不。但是当他们进入时，他们能下载你的整个数字生活并进行分类吗？没有。(不管怎样，真的会有人*希望*这么做吗？谁知道呢。)

更重要的是:为什么要给服务提供你不再需要或关心的数据？不要让它永生；熄灭它。
