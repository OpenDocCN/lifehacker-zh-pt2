# 即使你不喝鸡尾酒，也可以制作鸡尾酒糖浆

> 原文：<https://lifehacker.com/make-cocktail-syrups-even-if-you-dont-drink-cocktails-1841912429>

我的冰箱里装满了糖浆。除了必不可少的枫糖，还有一些便宜的“煎饼糖浆”，一些 [浆果蜂蜜](https://lifehacker.com/make-a-jewel-toned-honey-with-two-ingredients-1841002930) ，一瓶简单糖浆，一瓶蜂蜜糖浆。从历史上来说，后两种主要用于鸡尾酒，但实际上每个人都应该在他们的冰箱里有它们，不管他们是否喝酒。

Watch

你看，鸡尾酒糖浆的美妙之处在于它很容易溶解。与结晶糖或纯蜂蜜不同，这些稀释的甜味剂不需要大量搅拌或加热就能分散。这在厨房里很有帮助。

它们也非常容易制作。简单来说，只需将等量的糖和水(一杯一杯)放入调味汁锅中，用中高火加热，直到糖溶解。对于蜂蜜，使用三份蜂蜜和一份热水的混合物。两者都会无限期地保存在冰箱里。

我不喜欢喝水，但是这些糖浆可以帮助我保持水分。最近，我一直在制作单份柠檬水和酸橙饮料，根据我的心情，用少量蜂蜜或简单的糖浆使它们变甜。我可能还会加入生姜糖浆，因为一旦天气变暖，生姜柠檬水听起来会很不错。

在食物方面，我一直在使用醋油沙司中的蜂蜜糖浆，我把简单的东西搅拌成酱料，既有 [花生](https://skillet.lifehacker.com/use-an-almost-empty-jar-of-peanut-butter-to-make-a-dipp-1841891764) 和 marinara(不要告诉我的意大利男友)。糖让所有东西——甚至是美味的食物——味道都很好，一茶匙左右的糖浆可以让一道菜味道更完整。

你喜欢冰咖啡还是茶？你只需要手头有简单的东西。这是没有办法的。虽然用微波炉制作一点点 的东西很容易，但制作一瓶*并懒洋洋地将其搅拌到你的早晨冰镇饮料中要容易得多。生活已经够艰难了；用糖浆让你的生活更简单。*
