# 你可能不需要的 Lifehacker 认可的 Mac 应用程序

> 原文:[https://life hacker . com/the-life hacker-apps-you-may-not-need-1831745295](https://lifehacker.com/the-lifehacker-approved-apps-you-might-not-need-1831745295)

如果我可以在每篇 Lifehacker 文章上加上一个免责声明，那就是:不是每一个黑客、技巧或应用程序都适合所有人。本着这种精神，以下是我已经停止使用的 Lifehacker 批准的应用程序。我没有在我的新 Mac 电脑 上安装它们，我很好奇你们中有多少人也放弃了它们。

### **熊**和**简写**

当我真的喜欢苹果的默认应用时，我感到一种正常的羞愧。但是当我的几个 Lifehacker 同事使用第三方笔记应用程序如[](https://bear.app/)**和[**simple note**](https://simplenote.com/)时，我对**笔记**很满意。它很容易通过 iCloud 同步，可以搜索，使用文件夹。我甚至已经习惯了糟糕的、不可改变的默认字体。因为至关重要的是，Notes 支持基本的文本格式。**

**每一个像样的 Notes 竞争者都是纯文本或降价的。这个我真的不懂！所有旧式的文本编辑器都有所见即所得的格式:Word，Google Docs，TextEdit，Notepad。见鬼，Mac 的 **Stickies** 应用支持文本格式。为什么“notes bundle”应用程序不能在不求助于 Markdown 之类的伪代码解决方案的情况下处理这个问题？为什么用粗体和斜体这么难？我不想求助于像 Evernote 这样沉重的、基于订阅的应用程序。说到这里:**

### ****Evernote****

**我大概只是 [用 **Evernote** 错了](https://lifehacker.com/ive-been-using-evernote-all-wrong-heres-why-its-actual-5989980#_ga=2.55285163.167576501.1547477492-1222396525.1547242489) 。但是现在完全不用了。我过去常常把有趣的“灵感”归档到这个剪贴簿应用程序中，保存可能会被删除的图像、想法和网页。但是当我触及免费版 的极限(每个用户两个设备，没有离线移动访问)时，我意识到我可以轻松地使用更简单的应用程序，而不是每月支付 8 美元购买 Evernote Premium。**

**我的绝大部分笔记都是文本，所以我对笔记没什么意见(偶尔会对基于列表的笔记感兴趣)。如果我在做一个视觉项目，我有时会把东西保存到一个私人的 Tumblr(免费的)。如果你需要结合两者，你可以使用 **OneNote** 或 **Google Keep** 。或者，如果你像我一样，你真的不需要一个万能桶。**

### ****卡铺****

**我喜欢 Flexibits 的日历应用 [**奇幻**](https://flexibits.com/fantastical) ，所以我也想喜欢他们的最新应用。[**card hop**](https://flexibits.com/cardhop)本应重塑我与联系人的互动方式。我可以从 Cardhop 应用程序启动每一个交互，而不是在不同的通信渠道中搜寻，将它用作命令行，有点像 Spotlight 或 Chrome 地址栏。**

**但是正如我当时所写的，关于 Cardhop 最难的事情是记住使用它。我无法打破使用默认联系人应用程序的习惯，尽管它很差，或者在各种应用程序中开始对话，即使我后来发现自己在搜索旧消息——我是不是通过谷歌视频聊天、Twitter DM 或文本给科尔发送了那个链接？**

**几个星期后，我没有使用 Cardhop 的免费评论版。所以我不建议花 20 美元买它。对我来说，弄清楚 Siri 的哪些任务更快或更慢已经够难了。我没有足够的心理空间来使用另一个命令行。**

### ****火狐****

**也许你真的应该 [切换到优先考虑隐私的浏览器](https://lifehacker.com/why-you-should-switch-from-google-chrome-to-firefox-1821879163) ，而不是让你进入谷歌的服务。 [**火狐**](https://www.mozilla.org/en-US/firefox/new/) 在速度和功能上赶上了 **Chrome** 。但在它落后的几年里，我变得依赖于很多 Chrome 扩展，我并不渴望寻找和校准它们的 Firefox 等价物。**

### ****由 OmmWriter 提供的 omm bits****

**对于那些发明了一款“不受干扰”的新写作应用程序的人，政府会给他们发放保密奖金吗？如果我的写作项目不涉及在线研究、添加链接或任何其他应用切换任务，我会试着用物理笔记本来写。于是我停止了对这些全屏幕、“锁定”的应用程序的实验，比如[comm writer](https://ommwriter.com/)和[comm bit](https://ommwriter.com/ommbits/)，它们据说会迫使你去写东西。**

### ****香草****

**如果你想要一个便宜的应用程序来暂时隐藏你不常用的菜单栏项目，你可能会选择 [**【香草】**](https://matthewpalmer.net/vanilla/) 而不是更成熟的应用程序 [**调酒师**](https://www.macbartender.com/) 。但从长远来看，香草的极简界面感觉*太过*极简，所以我又换回来了。**

### ****盖章****

**这款价值 10 美元的 [**图章**](https://freeyourmusic.com/) 应用可以在苹果音乐、谷歌音乐、Tidal 或 Spotify 等流媒体服务之间移动音乐。但免费网络服务 [**调我的音乐**](https://www.tunemymusic.com/) 也是如此。**

### ****键盘大师****

**[**键盘大师**](https://www.keyboardmaestro.com/main/) 是一个复杂的控制中心，通过键盘快捷键、手势，甚至通过插入耳机来触发功能。它可以管理多个剪贴板条目，定位您的窗口，录制宏以自动完成，或配置自定义事件通知。这就像本地化的 [**IFTTT**](https://ifttt.com/) 。**

**但是如果你和我一样，只是把它作为一个价格过高的文本替换工具，那么你和我一样，可以通过 Mac 的内置文本替换功能来解决问题。(内置版本在某些领域不行，包括地址栏。)**

### ****AppZapper****

**我过去常常删除应用程序来释放硬盘空间，但现在我可以通过同步我装满视频的巨大文件夹或我臃肿的流媒体音乐收藏的本地下载来清理更多空间。 [**但这些天来，我不会首先删除许多应用程序。**](https://www.appzapper.com/)**

**我认为一些基本的东西改变了。几年前，当我换到一台新电脑时，我不得不把我的应用程序和我的其他文件一起复制过来，或者从光盘上重新安装。现在，我在网上购买所有的应用程序，因此更容易下载最新版本，并在我的电子邮件中找到许可证密钥。(多年来，最后一个延期付款是我所有的盗版 Adobe 应用程序，但现在我为 Creative Cloud 付费。)**

**我可能会保留一台电脑四五年，但时间不够长，应用程序不会堆积得太多。然后当我拿到新机的时候，如果我很久没用一个 app 了，我就不拿过来了。我从来没有被这个系统烧伤过。**

**我就抛弃 AppZapper 一事询问了 Lifehacker 的技术编辑大卫·墨菲(David Murphy)。他个人认为，如果你已经为它支付了 13 美元，你还不如使用它。但是他推荐免费的替代品[**【app cleaner】**](http://freemacsoft.net/appcleaner/)。**

### ****以此类推****

**还有很多其他应用程序我没有移植到我的新电脑上:专门的媒体播放器，比如 **Cog** 和**Miniplay**；免费音频编辑**无畏**；像 YouTube 下载器 **VideoDuke** 和菜单栏待办应用 **ThoughtTrain** 这样流产的实验。搬新电脑就像搬新家一样；你扔掉了很多杂物，避免了携带它们的麻烦。你在最后一台旧电脑上留下了哪些应用程序？**
