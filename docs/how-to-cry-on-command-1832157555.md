# 如何在命令中哭泣

> 原文：<https://lifehacker.com/how-to-cry-on-command-1832157555>

我不太爱哭，但当我哭的时候，我会放声大哭。

我的脸变红了，我的衣服被汗水浸湿了，我的眼睛闭上了。我是丑哭的大阪直美。



但是我不经常哭。其实我一般都是要做好计划的。每次《饥饿游戏》上映，Rue 死的时候我都会哭。如果我听新航的《深呼吸》，我会哭到最后一个音符。如果我想起高中，想起那有多糟糕，我会不由自主地流下眼泪。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-pabK7LRANiU&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-pabK7LRANiU&start=0) 

今天，福布斯写了为什么我们应该开始更经常地哭泣，以及它缓解压力的特性，这让我想到:如果这对我们中的一些人来说不是那么容易呢？

不像女演员布莱丝·达拉斯·霍华德可以为家得宝的出价哭泣，如果哭泣感觉像是一个不可能的壮举呢？我们能在命令下哭泣吗？还是这是我们与生俱来的？我为我们这些(真的)需要哭泣的人收集了一些技巧:

### 运用你的记忆

最明显的工具就是用自己的回忆来刺激眼泪。回忆那些让你不舒服、难过甚至有点害怕的瞬间(合理范围内)。记得当时是什么样子，如果你需要一个触发器，试试音乐或者一种气味，它会提醒你正在讨论的记忆。

电视连续剧明星 Ana Brenda Contreras 告诉《纽约时报》的你的童年常常能激发一场痛哭。

> “他们告诉你把自己想象成一个世界上没有人关心你的孩子，”孔特雷拉斯说。这是关于与你感到脆弱和被遗弃的那部分联系起来。这很有效，即使你有一个快乐的童年。"

### 打哈欠让自己流泪

当你打哈欠时，有时你的眼睛开始流泪。据《赫芬顿邮报》 报道，这是对眼睛周围腺体受到压力产生眼泪的一种生理反应。

在柯南 的后续采访中，霍华德说打哈欠对她的哭泣技巧至关重要。通过一遍又一遍地打哈欠，你最终会开始流泪。用这些眼泪来好好啜泣，想想会有帮助的情感和记忆。

### 掐自己

当回忆不起作用时，强行流泪。轻捏自己，拉一把鼻毛，眼睛不眨地盯着一面墙，或者切一个洋葱，直到挤出眼泪。现在不要发疯。你只需要一个小小的触发器让眼泪流出来，从那开始，你就准备好了。

### 使用催泪棒

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-A7tl_O48gto&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-A7tl_O48gto&start=0) 

女演员安娜·法瑞丝在《詹姆斯·柯登深夜秀》节目中分享了这个诀窍:当所有方法都失败了，你极度渴望大哭一场时，就用催泪棒吧。泪棒是由薄荷醇制成的，它会刺激你的眼睛流泪。这可以擦在眼睛下面(不要直接擦到眼睛里，因为你实际上会引起比你需要的更多的疼痛)。

如果在所有这些建议之后，你还是哭不出来，那也没关系。有些人只是觉得不可能流泪(我会为我们俩哭个够)。寻找其他方式来释放压抑的情绪，比如去跑步或向朋友发泄。我们都需要一个出口，有时这并不总是涉及一个无法控制的哭泣。
