# 如何在 Android 上识别 Chrome 浏览器中的假地址栏

> 原文:[https://life hacker . com/how-to-spot-a-fake-address-bar-in-chrome-on-Android-1834388305](https://lifehacker.com/how-to-spot-a-fake-address-bar-in-chrome-on-android-1834388305)

虚假登录页面是从用户处骗取登录凭据的一种常见方法。如果一个网站看起来是合法的，你的肌肉记忆很容易被激活，你就开始输入你的用户名和密码，而不检查 URL 是否正确(或者网站是否合法)。让事情变得复杂的是一个新问题，最近由开发者吉姆·费舍尔(Jim Fisher)描述，这表明网站使用虚假地址栏让你认为你在你不在的地方是多么容易。

通常，你可以瞥一眼地址栏左边的挂锁图标，判断一个网站是否可信。然而，不要盲目相信那个小图形，因为网络钓鱼者已经设计出一种方法，让移动网页在 Chrome 中显示虚假的地址栏，其中包括挂锁图标*和替换网址*。众所周知，这个“初始栏”取代了你浏览窗口中真正的地址栏。如果你不太注意，你可能会认为你的浏览器工作正常。

这整个技巧是可能的，因为移动版 Chrome 的 UI 经常在你向下滚动页面时消失，网站开发者可以覆盖并阻止 UI，包括 URL 栏，重新出现。正如费希尔所描述的:

> 这很糟糕，但情况会变得更糟。通常情况下，当用户向上滚动时，Chrome 会重新显示真实的 URL 栏。但是我们可以欺骗 Chrome，让它不再显示真正的地址栏！一旦 Chrome 隐藏了地址栏，我们就将整个页面内容移入一个“滚动监狱”——也就是一个带有溢出的新元素:scroll。然后用户认为他们在页面中向上滚动，但实际上他们只是在滚动监狱中向上滚动！就像《盗梦空间》中的一个梦，用户认为他们在自己的浏览器中，但他们实际上是在浏览器中的浏览器中。
> 
> 但更糟糕的是！即使有上面的“滚动监狱”，用户也应该能够滚动到监狱的顶部，此时 Chrome 将重新显示地址栏。但是我们也可以禁用这种行为！我们在滚动监狱的顶部插入一个非常高的填充元素。然后，如果用户试图滚动到填充区，我们将它们向下滚动到内容的开始处！看起来像是页面刷新。

谷歌正在修复这个聪明的编码 ，但是你仍然有一些技巧来揭示这些讨厌的初始栏:

*   你可以强制 Chrome 应用程序显示用户界面，即使网站通常会阻止它。你所要做的就是在 Chrome 应用打开时锁定手机屏幕，然后解锁。这将重置 Chrome 应用程序窗口，以便显示用户界面。如果 URL 是假的，你会看到两个显示的 URL 栏——上面的是真实的 URL，下面的是初始栏。
*   如果您在浏览时打开了多个标签，请密切关注标签图标中显示的数字。开始栏在这里经常会显示不正确的数字。
*   Chrome Android 中新的黑暗模式也使得发现开始栏变得更加容易。当 [黑暗模式被启用](https://lifehacker.com/how-to-enable-chromes-new-dark-mode-on-android-1834339091?_ga=2.181123175.1978754842.1556560510-1864099547.1536779123) 时，地址栏和其他 UI 元素将是黑色的，所以(假的)白色地址栏更容易被发现——或者反之亦然如果你使用的是普通的 Chrome 移动 UI 主题，而假的地址栏是黑色的。在 Chrome mobile 中使用 [阅读器模式](https://lifehacker.com/how-to-install-google-chrome-s-experimental-reader-mode-1834171339?_ga=2.181123175.1978754842.1556560510-1864099547.1536779123) 、更简单的 UI 模式或者改变地址栏外观的替代主题时也是如此。
