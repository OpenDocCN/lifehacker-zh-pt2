# 如何解决每个常见的拉链问题

> 原文：<https://lifehacker.com/how-to-fix-every-common-zipper-problem-5975700>

自从拉链被发明以来，它的变化并不大，我们都遇到的问题也是如此。从卡住的拉链到咬不紧的牙齿，以下是如何解决你在使用拉链时会遇到的所有问题。

### 你的拉链卡住了

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-seDQuPlKi9I&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-seDQuPlKi9I&start=0) 

当你的拉链卡住时，感觉就像被什么东西卡住了。拉链可能根本拉不下来，除非你修好它，否则你会被困在夹克里。

解决这个问题的最好方法是什么？抓起一支石墨铅笔，用笔尖在牙齿上摩擦。再试一次，应该可以。如果这不起作用，是时候使用润滑剂了。Windex 很好，因为它不是油基的，但你也可以使用肥皂或唇膏。

从拉链一直向上开始，慢慢将润滑剂涂到牙齿上。然后再把拉链往下拉一点，重新拉上，继续这样做，直到拉链完全拉下。这对于 [修复卡在面料本身的拉链](http://lifehacker.com/how-to-fix-a-stuck-zipper-5846669?_ga=2.89566811.1940509479.1557157607-1723114163.1524514905) 来说尤其得心应手。

### 牙齿合不上(或者一直张开)

拉链齿合不上是最烦人的问题之一。该问题可能由几种不同的情况引起。有时，上述使用铅笔(或一块肥皂)的技巧会使牙齿变得足够光滑，这样它们就会再次工作。

如果没有，滑块可能工作不正常。首先，仔细检查以确保一块布或线没有卡在拉链里。接下来，看个别牙齿。如果它们中的任何一个伸出来，拿一把钳子把它们移回原位，这样它们都是直的。

如果牙齿又直又干净，那就看看滑块本身吧。随着时间的推移，滑块开始分开，当这种情况发生时，它停止将拉链齿固定在一起。拿几把钳子，试着合上滑块，直到它再次卡住牙齿。

如果你的拉链在一条牛仔裤上，解决方案就有点复杂了。如果可以的话，需要 [去掉底部的金属保险杠，换上一针](http://www.threadsmagazine.com/item/3738/how-to-replace-a-broken-zipper) ，或者 [如果底部缺齿就在中间打结](http://tlc.howstuffworks.com/home/how-to-mend-clothes5.htm) 。不幸的是，这只适用于能接触到底部保险杠的裤子。

如果这样做失败了，或者你正在处理的裤子不能达到整个拉链长度，你可能需要 [完全更换拉链](http://www.threadsmagazine.com/item/3738/how-to-replace-a-broken-zipper) 。虽然你可以用钳子、剪刀和线自己动手，但在大多数裁缝那里，更换一条裤子的拉链只需要 5-10 美元。

### 拉链拉不上了

裤子拉链的一个常见问题是拉链拉不起来。这可能会导致各种尴尬的局面。不幸的是，你不能真正永久地解决这个问题，除非你彻底更换拉链。

也就是说，有两个简单的临时修复方法。最简单的就是将 [的钥匙圈穿过拉链拉到裤子上，扣上](http://lifehacker.com/keep-your-loose-zippers-up-with-a-key-ring-5856178?_ga=2.147155319.1940509479.1557157607-1723114163.1524514905) 。这以一种简单的方式保持拉链拉上。如果你喜欢多一点弹性，你也可以试试橡皮筋。

### 滑块断了

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-FztgKTHtjQY&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-FztgKTHtjQY&start=0) 

如果滑块完全脱落，或者没有正确关闭齿，那么您需要更换它。要取下滑块，使用 [点钳子](http://www.youtube.com/watch?v=w5AEBhwIqDc) 。完成后，重新安装新的拉链滑块，将其滑回到齿上。就这样，你完了。

更换滑块通常非常简单，应该只需要几分钟的时间。

### 拉链拉断了

当拉链被拉断时，拉上拉链会变得非常困难。好消息是，这几乎是最简单的修复方法。你可以把一个回形针，一个钥匙圈，甚至一个 [电话线接头](http://lifehacker.com/replace-a-broken-zipper-slide-with-a-telephone-wire-con-5880289?_ga=2.147155319.1940509479.1557157607-1723114163.1524514905) 变成拉链头。只要把它穿过滑块上的拉环，瞧，你就有了一个新的拉链头。当然，这并不是最时尚的解决方案，但至少你可以穿上和脱下你的衣服。
