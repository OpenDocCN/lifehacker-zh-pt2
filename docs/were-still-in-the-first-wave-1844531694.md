# 我们还在第一波

> 原文：<https://lifehacker.com/were-still-in-the-first-wave-1844531694>

尽管几个月来一直被告知这种新型冠状病毒不同于我们在现代应对的任何病毒，但它仍然诱惑我们从流感 的角度来看待它。(尽管它们绝对不是一回事。)这很有意义，因为我们熟悉流感、和流感季节，它们似乎给了我们一种理解新冠肺炎疫情的方式。虽然有一些相似之处(如两种疾病对高风险健康人群来说都特别危险)，但世界卫生组织(世卫组织)希望我们知道新冠肺炎在一个主要方面与流感不同:它不是季节性的。我们都经历了一个大浪潮，而且这个浪潮还在继续。

### 没有所谓的“COVID 季节”

当这种新型冠状病毒在 2020 年初开始传播时， [一些人推测，一旦天气变暖袭击](https://www.npr.org/sections/goatsandsoda/2020/04/09/830297538/scientists-try-to-figure-out-if-summer-will-slow-the-spread-of-covid-19) ，疫情就会自行消失。当然，抓住那一点点希望是很好的，但正如流行病学家和公共卫生专家马克·利普西奇博士、流行病学教授和哈佛大学 T.H. Chan 公共卫生学院传染病动力学中心主任指出的那样，[COVID 不是这样工作的](https://ccdd.hsph.harvard.edu/will-covid-19-go-away-on-its-own-in-warmer-weather/) 。

今天在日内瓦的一次虚拟简报会上，世卫组织发言人[Margaret Harris 博士为我们](https://www.reuters.com/article/us-health-coronavirus-who-idUSKCN24T16U) 解析了一下:

> “人们仍然在考虑季节。我们需要弄清楚的是这是一种新病毒...他的儿子表现得不一样。夏天是个问题。这种病毒喜欢全天候。”

### 他的静止不动不会很快结束吗

最重要的是，有很多关于新冠肺炎不同“浪潮”的说法，类似于我们认为在 1918 年疫情流感 T3 期间发生的 [。这就是一旦传播得到控制，消散一段时间后，](https://www.cdc.gov/flu/pandemic-resources/1918-commemoration/three-waves.htm) [又一轮新的感染将开始](https://www.cebm.net/covid-19/covid-19-epidemic-waves/) 。因为像纽约市这样的地方——美国最初的震中——已经到了能够重新开放的程度，看起来像亚利桑那州和佛罗里达州这样的地方的感染率上升是我们被警告的“第二波”。但是世卫组织官员想让你知道事实并非如此。

事实上，世卫组织官员已经尽力确保他们不会把新冠肺炎的复兴(就像目前发生在香港的那场)描述为“浪潮”这是有原因的:使用“海浪”这个术语听起来好像新冠肺炎的传播超出了人类的控制，而此时我们知道协调一致的集体行动可以减缓传播，T5 路透社报道 。

下面是 Harris 在同一个虚拟简报中对那个 说的话:

> “我们处在第一波浪潮中。这将是一个大浪潮。它会上下波动一点。最好的办法是把它弄平，变成一个在你脚下轻轻拍打的东西。”

### 我们能做什么？

尽管新冠肺炎不像流感一样具有季节性，但这并不意味着我们对今年的流感季节准备不足。这就是为什么今年接种流感疫苗比以往任何时候都更重要 。正如哈里斯所说:“如果你已经有很高的呼吸系统疾病负担，而呼吸系统疾病又增加了，那会给卫生系统带来更大的压力。”除此之外，她还强调了继续保持公共健康措施的重要性，比如保持身体距离、戴口罩和避开人群。(是的，我们所有人，是的，在夏天也是。)

尽管认为 COVID 是季节性的或呈波浪式出现可能令人放心，但这似乎不是这种疾病的工作方式。就这么定下来吧，因为正如 Lifehacker 的高级健康编辑 Beth Skwarecki 几个月来告诉我们的那样，， [群体免疫救不了我们](https://vitals.lifehacker.com/herd-immunity-wont-save-us-1843006237) ，事情 [可能会变得更糟](https://lifehacker.com/its-gonna-get-worse-1844002403) 。
