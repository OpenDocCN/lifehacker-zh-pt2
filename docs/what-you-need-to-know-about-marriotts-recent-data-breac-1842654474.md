# 关于万豪最近的数据泄露，你需要知道什么

> 原文：<https://lifehacker.com/what-you-need-to-know-about-marriotts-recent-data-breac-1842654474>

3 月 31 日星期二，万豪国际宣布可能发生了的数据泄露，泄露了超过 520 万名客人的信息。这是该酒店集团在过去两年中第二次遭遇如此大规模的违规事件。

Watch

在这次事件中，该公司认为护照信息、驾照号码和信用卡信息是被盗信息的而不是部分。根据他们的官方网站，可能被泄露的信息包括:

*   联系方式(如姓名、邮寄地址、电子邮件地址和电话号码)
*   忠诚度账户信息(例如，账号和积分余额，但不是密码)
*   其他个人详细信息(例如，公司、性别和出生日期和月份)
*   合作关系和附属关系(例如，关联的航空公司忠诚度计划和号码)
*   偏好(例如，住宿/房间偏好和语言偏好)

这不是万豪第一次经历大规模数据泄露，两年前我们报道过。

如果您受到了最近一次入侵的影响，您应该做以下事情:

**1。立即更改您的密码。**

**2。注册** [**网络观察者**](https://www.webwatcher.com/)——一个监控你的个人信息并在未经你授权的情况下通知你的网站。

**3。建立一个信用提醒**来监控你的分数的任何变化或者你没有授权或申请的新账户/贷款。
