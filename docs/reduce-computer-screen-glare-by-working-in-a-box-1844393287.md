# 通过在盒子里工作来减少电脑屏幕眩光

> 原文：<https://lifehacker.com/reduce-computer-screen-glare-by-working-in-a-box-1844393287>

不可避免的是，每周有几天下午的某个时候，我儿子会突然跑进我的办公室，问我是否愿意在他在充气泳池玩耍的时候去后院外面工作。他让我在阴天做这个吗？没有；你看，阴天可不是游泳的好日子。因此，我最终走到了外面，在阳光下，有着不稳定的 wifi，屏幕上覆盖着太多的眩光，我发现自己不断地定位和重新定位我的笔记本电脑，试图找到一个角度，让我在工作时感到最少的挫败感。至少，我*用*就是这么干的。现在我把我的笔记本电脑放在一个盒子里。



这个想法来自于 Reddit 用户 u/benlouislebuT3，他说:

> 因为天气晴朗，我想在外面工作，但是太亮了，我看不到屏幕。把你的笔记本电脑放在盒子里可以挡住光线，所以你的笔记本电脑屏幕很亮。它还能让笔记本电脑更凉爽。

这两点他都是对的。虽然你可以使用像 u/benlouislebu 这样的非常好看的纸箱，但我发现一个旧盒子也可以做到这一点，这个旧盒子曾经装过我们最喜欢的当地啤酒之一(大声喊出这个恰如其分的名字[Always Sunny Pale Ale](https://thebrewworks.com/brews/always-sunny-pale-ale/))。当然，在我们一年一度的家庭海滨之旅中，来回运送食物和饮料有点累——但它遮挡了阳光，让我的笔记本电脑摸起来更凉爽*和*，它提醒我的邻居我喜欢冰镇啤酒。这个箱子很重。

正如一些 Reddit 评论者指出的那样，如果通风口似乎被堵塞并且超时工作，你可能需要在纸板上切几个孔以通风，并且/或者将你的笔记本电脑从纸板上抬起来(也许可以通过在纸板下面滑动两个较小的纸板)——特别是如果你打算像这样设置一段时间的话。
