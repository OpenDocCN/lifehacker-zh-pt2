# NASA 想付钱让你在假火星上隔离 8 个月

> 原文：<https://lifehacker.com/nasa-wants-to-pay-you-to-isolate-on-fake-mars-for-8-mon-1843598007>

如果，在冠状病毒疫情爆发两个多月之后，你对自己说，“嗯，隔离生活对我来说挺管用的，”NASA 可能正好有适合你的工作。太空总署 [正在寻找申请人](https://www.nasa.gov/feature/nasa-seeking-us-citizens-for-social-isolation-study-for-moon-and-mars-missions/) 进行一项社会隔离研究，为未来的火星和月球太空飞行任务开发方法和技术。以下是你需要了解的关于这份工作的一些情况，我们现在突然比几个月前更有资格做这件事。

### 细节

成功的申请者将在俄罗斯莫斯科与一个小型国际团队一起度过八个月的隔离期。这个“模拟任务”的目的是帮助美国宇航局了解更多关于禁闭和隔离对人类的身体和心理影响。在此期间进行的研究将包括使用虚拟现实执行机器人操作以及各种任务。

这个想法是利用这些信息来准备对月球的探测任务以及未来对火星的探测任务，这些任务将持续很长一段时间。参与者在俄罗斯的隔离时间将获得不同级别的报酬，这取决于他们是否已经与美国航天局建立了联系。

### 如何申请

除了愿意花八个月假任务，申请人还必须 [满足以下要求](https://www.nasa.gov/feature/nasa-seeking-us-citizens-for-social-isolation-study-for-moon-and-mars-missions/) :

*   美国公民身份
*   年龄在 30-55 岁之间
*   精通俄语和英语。
*   有硕士，博士学位。医学博士或完成军官培训。(尽管具有学士学位和其他特定资格的参与者也可以考虑，例如相关的额外教育、军事或专业经验。)

欲了解更多信息，并申请该职位， [访问美国宇航局网站](https://www.nasa.gov/analogs/want-to-participate) 。

### 宇航员能教会我们什么是隔离

美国宇航局这项研究的目的是了解人类如何应对与世隔绝的生活。几周前， [我们在 Lifehacker](https://lifehacker.com/how-to-cope-with-isolation-according-to-an-astronaut-1842549279) 采取了类似的(尽管是相反的)方法，当时作家雷切尔·费尔班克采访了前宇航员[d·马歇尔·波特菲尔德](https://engineering.purdue.edu/ABE/people/ptProfile?resource_id=7380) 以获得关于不习惯被孤立的人如何在这个疫情期间应对的指点。他的建议包括制定一个常规，保持活跃，与朋友和家人保持联系，并有一个目标。也许*你的*目的可以是为你的火星任务做准备。假火星。
