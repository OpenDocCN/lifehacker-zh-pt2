# 使用床垫打气筒来获得更好的火鸡皮

> 原文：<https://lifehacker.com/use-a-mattress-pump-for-better-turkey-skin-1830570820>

将一串黄油 塞在感恩节火鸡的表皮下是确保它湿润可口的绝妙方法，但这需要你的手有点脏。如果你想花更少的时间 [把皮和肉](https://skillet.lifehacker.com/how-to-make-a-delicious-thanksgiving-turkey-1830465631) 分开，那就买一个空气床垫泵吧。

Watch

虽然这并没有完全否定动手(或真的，“动手”)工作的需要，但使用气泵确实使任务变得容易得多，而且它确实可以帮助你分离即使是最小、最幼稚的手也可能无法触及的区域。

只要拿一个(干净的)空气床垫(或其他电动)泵，用手指在皮肤和肉之间开一个小口，然后将泵的(非常干净的)尖端插入那个开口。扳动开关，以一种有点令人毛骨悚然但有点令人愉快的方式观看火鸡皮气球。在你挤黄油的过程中，你可能仍然会遇到一些结缔组织，但是我想你会发现——就像我一样——这将是你用过的最松的火鸡皮。(我知道这听起来很恶心，但在这种情况下是好事。)此外，所有的气流都有助于干燥皮肤，这意味着它会更容易变脆，黄油般酥脆的皮肤是一件非常好的事情。(只是记得充分消毒后，你的气泵；用稀释的漂白剂溶液擦拭就可以了。)
