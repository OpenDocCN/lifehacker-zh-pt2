# 让这些医生告诉你你是否快死了

> 原文:[https://life hacker . com/let-thes-doctors-tell-you-if-you-dead-1830740911](https://lifehacker.com/let-these-doctors-tell-you-if-youre-dying-1830740911)

每当你头痛或胸痛得厉害时，你可能会直接冲向 WebMD，疯狂地进行研究，陷入全面恐慌之中。

这就是为什么心脏病专家马克·艾森伯格和克里斯·凯利写了 [*我要死了吗？！*](https://www.harpercollins.com/9780062847607/am-i-dying/) *，*冷静的对话式指导，准确地引导你进入平静的水域，找到你症状的可能原因。在上面的视频中，医生向我们解释了并非所有的胸痛都是心脏病发作，并非所有的胃痛都是急性阑尾炎，以及如何最好地向医疗专业人员描述你的症状。
