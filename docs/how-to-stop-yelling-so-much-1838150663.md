# 如何停止大喊大叫

这个建议只是给你的，不是给我的，因为我绝对不会对我的宝贝孩子大声嚷嚷，什么样的人会做这种事？？

好吧，我吼了，我吼了，对一些事情有一些吼。尽管事实上我知道得更多，但这并不奏效(奇怪的是，孩子们在早上并没有走得更快，只是因为你大喊他们应该在早上走得更快)，我的声音有时还是会变大。看起来想要变得更有耐心和实际上变得更有耐心并不会自动地联系在一起。

心理健康顾问、凌乱母亲的创始人阿曼达·鲁特在播客 中告诉 [力量，她决定尝试一种新策略来抑制自己的叫喊。她在家里到处张贴物理提醒——她能看到和触摸到的东西——提醒她不要对她的孩子大喊大叫:](https://www.strengthinwords.com/podcast/how-to-stop-yelling-at-your-kids/)

> 我儿子当时最喜欢的颜色是黄色，所以我用建筑纸剪下黄色的心形，贴在我家周围的“热点”上；在他的卧室门上帮助我睡觉，在我的厨房冰箱上吃饭...在我车里的后视镜上。诸如此类的地方，这样我就可以一整天都碰到他们。

她用这些心形图案来提醒自己保持冷静，同时也作为一种愤怒检测工具。每当她遇到这种情况时，她都会对自己的情绪进行一次快速的内部检查，以帮助识别她的愤怒是否正在积聚，是时候实施冷静策略了(她列出了 50 个""你可以从她的网站上请求)。

我认为心形是一个特别甜蜜的选择，但你发布的提醒可以是任何引起你注意的东西，让你考虑与孩子的互动是否变得紧张——停止标志，他们最喜欢的节目中的角色或带有安抚咒语的便利贴。

作为奖励，这为我们的孩子树立了一个积极的榜样，让他们学会如何管理自己的情绪——努力做得更好永远都不晚。
