# 何时使用液体绷带代替创可贴

> 原文：<https://lifehacker.com/when-to-use-liquid-bandages-instead-of-band-aids-1826485377>

当我在药店无聊的时候，我会买一些我不需要的骗人的东西。这就是我最终发现液体绷带的原因，事实证明我确实需要它，因为它对我手上的小伤口、水泡和任何伤口都非常有效。液体绷带用一层直接敷在伤口上的透明液体代替了普通绷带。



有 [多个品牌](https://www.amazon.com/s/ref=nb_sb_noss_2?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/when-to-use-liquid-bandages-instead-of-band-aids-1826485377&asc_source=&field-keywords=liquid bandage&tag=kinjalifehackerlink-20&url=search-alias=aps) 使用不同的化学品，它们以喷涂或涂抹的形式出现。我选择 Walgreens 商店的油漆品牌，但就像普通绷带一样，你会找到自己最喜欢的。

### **微小的割伤和虫咬**

过去我们推荐过用胶带或指甲油覆盖被虫子咬过的地方的，或者用强力胶封闭伤口的。液体绷带只是那种材料的优化版本，添加了防腐剂。更不容易刺激伤口，更容易涂抹。(梅奥诊所 [经常使用皮肤粘合剂](https://mayoclinichealthsystem.org/hometown-health/speaking-of-health/should-super-glue-be-in-your-first-aid-kit) 而不是缝线，因为这样更快，痛苦更少。)

你仍然可以透过液体看到伤口。因此，如果你有一个明显的伤口，你可能更喜欢用创可贴来隐藏它。

### **手部伤口**

我们大多数人整天用手，常规绷带容易脱落。对于手指伤口，我们必须使用那些时髦的指关节和指尖绷带。但是现在我们都用触摸屏了，绷带很讨厌。其中大部分都不能在你的智能手机上运行。它们也不适合正常打字。

液体绷带小而不显眼(这也让它非常适合面部)，而且它不会让你的手指在触摸屏上隐形。它会弄糟触控 ID，所以激活一个不同的指尖，停止触摸穿过痛苦，兰博。

### **水泡**

就像普通绷带一样，液体绷带有助于治疗和预防水泡。(安德鲁·韦尔提到音乐家使用强力胶来保护他们的手指免受弦乐器的伤害。)如果你需要做的只是减少摩擦，但普通绷带不断脱落，那就涂上大量液体绷带。

就像不干胶邮票和更好的麦片盒子顶部一样，液体绷带是那些微小的技术进步之一，在你使用它的前三次感觉改变了生活，然后成为生活的米色背景噪音的一部分。但现在，这让我觉得自己像个自愈的变异人。我很酷，就像金刚狼，而不是在我点击鼠标的手指上长了只猫头鹰的笨蛋。
