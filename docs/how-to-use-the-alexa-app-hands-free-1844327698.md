# 如何免提使用 Alexa 应用程序

> 原文:[https://life hacker . com/how-to-use-the-Alexa-app-hands-free-1844327698](https://lifehacker.com/how-to-use-the-alexa-app-hands-free-1844327698)

根据设计，大多数 Alexa 驱动的设备可以完全通过开箱即用的语音命令来控制——只需说“Alexa ”,你的 Echo 扬声器、Fire TV 或其他 Alexa 设备就会做出响应，不需要按按钮或点击。罕见的例外是 Alexa 智能手机应用程序，它要求用户点击一个“Alexa”按钮来启用数字助理。或者至少，它曾经是这样的:亚马逊最近为 Alexa iOS 和 Android 应用程序添加了一个新的免提选项，但你需要手动打开它才能利用它。

### 如何打开 Alexa 应用的免提模式(iOS 和 Android)

1.  在 iPhone 或 Android 设备上打开 Alexa 应用程序。
2.  点击左上方的三行菜单图标打开侧边菜单，进入手机上的**设置> Alexa。**
3.  点击菜单顶部**“启用 Alexa 免提”**旁边的开关。
4.  然后，该应用程序会请求允许访问您手机的麦克风和设备音频。轻按“打开”，然后允许应用程序始终监听您的麦克风，并允许它在出现提示时访问设备音频。
5.  授予权限后，设置将打开。您可以随时通过点击**“启用 Alexa 免提”**开关来关闭它。

当 Alexa 的免提模式打开时，你只需说“Alexa”就可以调用助手——不需要额外的按钮点击。你必须先加载应用程序，所以这不太像使用一个总是收听的 Alexa 智能扬声器，但你仍然可以通过让 Siri 或谷歌助手为你启动它来保持这个过程完全不干涉。撇开这个怪癖不谈，启用这个功能会让你在智能手机上使用 Alexa 更加方便。只要确保你关注 Alexa 的响应能力:语音助手是 [容易受到不想要的激活](https://lifehacker.com/you-should-mute-your-smart-speakers-mic-more-often-1844250811) ，这一事实对用户隐私有着严重的影响。幸运的是，有几种方法可以 [最小化 Alexa 的窃听](https://lifehacker.com/how-to-delete-voice-recordings-with-alexa-google-assis-1836977240) 。
