# 如何在家中为孩子创造学习空间

> 原文：<https://lifehacker.com/how-to-create-a-learning-space-for-kids-at-home-1844517672>

今年秋季开学后，我们的孩子很有可能会花一周的至少一部分(或者大部分，或者全部)时间在家学习。尽管我们设法在临时的学习场所度过了春天，在那里他们完成了零星的任务，并参加了 T2 组织的电话会议，但这是不同的。他们开始了新的年级，有了新的老师和新的课程，我们不再处于只能勉强维持到年底的模式。他们将不得不——咽——合法地从家里学习 T4。所以他们需要一个专门做这些事情的空间。

Watch

如果你的孩子还没有一个固定的区域来做他们的功课，现在是一个好时机来看看你的家，想出你可以把他们安置在哪里——记住他们的舒适，空间的功能和任何你可以添加的东西，让它感觉有点乐趣。

### 舒适

春天，我的儿子养成了习惯，每天抱着他的 Chromebook 趴在我的床上一两个小时，完成他三年级的作业。他年轻的背部可以在一天中的一小部分时间里处理这样的位置，但今年他将被期望在家坐着工作更长的时间。所以我开始寻找的第一件东西就是一张桌子和一把舒适的椅子。

今年秋天，你家里可能已经有了一张书桌或桌子，可以重新用作学习空间，但是如果你还没有一把合适的椅子，并且你至少有能力在一件东西上挥霍，那就把它变成吧。检查他们使用的椅子和桌子的高度是否会让他们在工作和/或大部分时间打字时感到舒适。

如果可能的话，把它们放在窗口或窗口附近。如果说我从多年在家工作中学到了什么，那就是自然光和外面的世界对一个人的心理健康有好处，尤其是现在。

### 功能

至少可以说，知道我们将不得不让我们的孩子在这个学年继续工作是一个令人生畏的想法。然而，这是真的。会有很多事情让他们从工作中分心，包括他们的兄弟姐妹，你自己的工作电话，街道上叮当作响的垃圾车，靠近满是小吃的厨房，以及他们宁愿看电视的事实。我们唯一的希望就是给他们提供尽可能多的功能空间。

如果可能的话，特别是如果你有不止一个孩子需要整天做作业和视频聊天，为每个孩子找一个半私密和安静的空间，这样一个孩子在幼儿园的尖叫声就不会破坏他第一次尝试乘分数的注意力。如果孩子们需要在同一个房间工作，可以考虑在另一个房间指定一个空间专门用于视频通话，这样无论谁在工作都可以安静地进行。

他们的工作空间也应该足够大，能够放得下他们的电脑(如果他们有)和他们经常使用的任何学校用品，比如铅笔、蜡笔、纸张、尺子、活页夹和练习册。如果他们有带抽屉的桌子，那就太好了；如果没有，拿出一些小容器或办公用品来让他们存放他们的用品。

如果他们很年轻，而你在这段时间也在家工作，你需要考虑他们将在哪里工作与你的工作地点的关系，特别是当他们感到困难时，你将是主要帮助他们解决问题的人。他们需要足够近，以便在需要时寻求帮助，但不要*太近，以至于整天都在分散对方的注意力。这将会棘手，并根据你孩子的年龄和他们需要多少帮助而变化，但这是要记住的事情。*

### 乐趣

最后，试着让空间变得有趣一点。我给儿子订了一张 [明黄色的办公桌](https://www.overstock.com/Home-Garden/Lyndale-2-drawer-Helix-Legs-Office-Desk-by-iNSPIRE-Q-Bold/9641382/product.html?kwds=kids%20desk&refccid=OXQ3FLUZW2D2KGSG364KETX73Y&rfmt=&searchidx=3) 和一张 [霓虹绿色的椅子](https://www.amazon.com/gp/product/B0047FXY8Y/ref=ppx_yo_dt_b_asin_title_o01_s00?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-create-a-learning-space-for-kids-at-home-1844517672&asc_source=&ie=UTF8&psc=1&tag=kinjalifehackerlink-20) (这两张都是他挑的)，还有一些五颜六色的办公用品和 [这些表情符号杯垫](https://www.amazon.com/gp/product/B01ALARLCS/ref=ppx_yo_dt_b_asin_title_o00_s01?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-create-a-learning-space-for-kids-at-home-1844517672&asc_source=&ie=UTF8&psc=1&tag=kinjalifehackerlink-20) 。但是你也可以把你已经拥有的家具漆成明亮的颜色，挂上一个小软木板、白板或海报来装饰。

接下来，在他们的桌子上放一盏时髦的灯，或者在他们的工作空间下面放一块彩色地毯。给他们和他们非疫情时代的朋友拍几张照片。用一株小植物营造氛围，用一个压力球驱散焦虑，让一切变得圆满。尽你所能让这个空间明亮迷人，这样他们每天坐下来开始工作就不会有太多的思想斗争。
