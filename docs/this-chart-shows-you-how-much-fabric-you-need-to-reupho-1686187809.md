# 这张图表显示了更换家具需要多少布料

> 原文：<https://lifehacker.com/this-chart-shows-you-how-much-fabric-you-need-to-reupho-1686187809>

更换旧家具——无论是你已经用了多年的沙发，还是你从路边捡来的沙发——都能让一件家具重获新生。问题是，根据物品的类型和形状来计算出你到底需要多少布料是一个挑战。

这些图表帮助你了解普通椅子、沙发和凳子需要多少布料。

设计网站 [Honey & Fitz](https://shop.honeyandfitz.com/) 已经收集了可打印的面料数量，用于重新装饰流行的座椅形状。如果你第一次考虑重新组装某样东西，一定要做好调查， [知道你需要的工具和你需要遵循的步骤](http://workshop.lifehacker.com/reupholster-a-chair-part-one-the-teardown-1660495228) 。

我们不能保证重新抛光的过程会顺利进行(同样，还有其他文章)但是至少你会有合适数量的布料来完成这项工作。
