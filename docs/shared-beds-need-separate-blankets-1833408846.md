# 共享床需要单独的毯子

> 原文：<https://lifehacker.com/shared-beds-need-separate-blankets-1833408846>

一个又大又漂亮的被子，整齐地铺在你和伴侣共用的床上，在白天可能看起来很漂亮。但是到了晚上，就会出现混乱。一个人会太热，一个人会太冷。更糟糕的是，他们中的一个可能会在晚上翻来覆去，拉着冷空气钻进被窝。简单的解决办法是:分开毯子。

Watch

两张毯子的情况看起来仍然不错:要么在床脚折叠一张，要么在上面铺上更大(或更漂亮)的毯子。你可以用更小的毯子，而不是两个人一起睡；当我们从大号床升级到特大号床时，我保留了我的大号被子。它完美地躺在床上，没有挂在床沿上，晚上它只属于我。(我老公更喜欢轻薄的毯子或者床单。)

这一招也适用于给客人惊喜。如果你有半夜爬上床的孩子，帮你自己一个忙:手边多准备一条毯子。如何阻止他们溜进来是另外一个问题 ，但至少如果他们有自己的被子，他们就不会用冰冷的小脚在你的被子下面打洞了。
