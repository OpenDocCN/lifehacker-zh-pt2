# 不，杰夫，我们不需要去太空拯救人类

> 原文:[https://gizmodo . com/no-Jeff-we-dont-need-to-space-to-save-humanity-1834668636](https://gizmodo.com/no-jeff-we-dont-need-to-go-to-space-to-save-humanity-1834668636)

我热爱太空。我对人类更多地探索太空感到非常兴奋，我甚至可以接受我们 [重返月球](https://gizmodo.com/we-must-go-to-the-moon-and-bring-back-our-piss-1833660539?_ga=2.147111543.1776964966.1557490742-2075852810.1537814466) 作为开始。太空是一个等待科学发现的广阔领域；一个让我们 [探索宇宙奥秘](https://gizmodo.com/tag/astrophysics)[了解自己](https://io9.gizmodo.com/a-psychological-transformation-that-only-astronauts-exp-1655217228) 的机会。

空间不是也永远不会是拯救我们这个物种的关键。

然而，这一主张——我们需要大胆地进入最后的前沿，以拯救人类，或许还有地球本身——已经成为某些具有宇宙意识的科技兄弟的流行教条。2016 年，埃隆·马斯克 [公布了他的愿景](https://gizmodo.com/this-is-how-elon-musk-plans-to-build-a-city-on-mars-up-1787146547?_ga=2.204835350.740568268.1557143383-229968814.1545242016) ，以在火星上建立永久的人类存在，他将使人类成为“多行星物种”的项目描述为对冲我们对灭绝的赌注的最佳和唯一方式。昨晚，杰夫·贝索斯在一次 50 分钟的演讲中进一步拓展了这一思路，表面上是为了宣布 [蓝色起源的新月球着陆器](https://gizmodo.com/jeff-bezos-reveals-lunar-lander-designed-for-sustained-1834654065) 。事实上，这个演讲是对贝佐斯将人类从蓝色大理石有限资源的枷锁中解放出来的愿景的全面概述。

“在人类历史上，地球对我们来说一直很大，”贝佐斯告诉聚集在一起的观众。“那不再是真的了。地球不再大了。人性之大。对我们来说似乎很大，但它是有限的。”

在迅速将贫困、饥饿、无家可归、污染和过度捕捞描述为“迫在眉睫的问题”后，贝佐斯让人们了解了人类面临的真正危机。

“一个非常基本的长期问题是，我们将耗尽地球上的能源，”贝佐斯说。“我们不想停止使用能源，但这是不可持续的。”根据贝佐斯的说法，阻止我们最终不得不“用太阳能电池覆盖整个地球表面”的唯一方法是走出我们的家园。

“如果我们进入太阳系，实际上我们有无限的资源，”贝佐斯说，并解释说我们可能有“一万亿人”，包括“一千个莫扎特和一千个爱因斯坦。”似乎将人口数量提高两个数量级还不够雄心勃勃，贝佐斯接着建议我们通过将数百万人塞进旋转的虚空城市来实现这一目标——也被称为 [奥尼尔圆柱](https://gizmodo.com/this-is-what-it-might-be-like-to-fly-through-a-giant-sp-1701786771)——在这些城市中，我们将拥有城市、庄稼和重建的国家公园。据推测，一个仁慈的公司将确保持续的空气供应。

亚马逊人说，地球将最终被划分为“住宅和轻工业区”,而所有破坏性的工业将超越我们星球的蓝色光泽。

“我们可以两者兼得，”贝佐斯说。我们将为子孙后代保护好地球，但人类不必放弃“一个充满活力和增长的未来”。

听着，我明白。这一切听起来非常令人兴奋，硅谷的亿万富翁最喜欢的莫过于宣扬一点技术乐观主义。但是我们是不是*也许*有点超前了呢？你知道，在考虑如何在寒冷、黑暗的虚空中再造一百万个口袋大小的版本之前，我们难道不应该试着弄清楚如何在这个唯一支持人类生活的星球上可持续地生活吗？

贝佐斯对许多 21 世纪的问题不屑一顾，但这些是真实的人类现在面临的真实危机。如果我们不控制失控的气候变化并找出如何 [防止一百万个物种灭绝](https://earther.gizmodo.com/everything-is-fucked-major-new-extinction-report-finds-1834547635) 我们很有可能永远不会接近地球“有限资源”的极限，因为，你知道，我们都会死。

坦率地说，我们离用尽地球资源还很远。贝佐斯在他的演讲中也承认了这一点，他漫不经心地说，我们可以把整个内华达州变成一个太阳能农场，并以此为整个人类企业提供动力。美国能源部计算出美国海岸线 的风中有 [足够多的能量供每个美国家庭使用。我们为支持能源转换而开采的](https://gizmodo.com/the-battle-to-bring-offshore-wind-power-to-america-1787885358?_ga=2.123896460.740568268.1557143383-229968814.1545242016) [稀土金属](https://earther.gizmodo.com/the-dirty-truth-about-green-batteries-1833922990) 在地壳中非常丰富，虽然我们目前只能获取其中的一小部分，但在地球上开发新的资源或设计我们的技术来更有效地利用它们，都是比从土星环中获取相同金属更合理的解决方案。我们可以把开发奥尼尔汽缸所需资源的一小部分投入到一个强大的聚变能源项目中，在一个世纪左右的时间里，我们可能会用海水为整个城市供电。

也许贝佐斯甚至可以投资一些亚马逊目前正在使用的资源？

我们不要忘记，最关键、最有限的资源是在地球上找到的，而不是在其他地方:生物多样性。我们不能离开复杂而微妙的生命网络而生存，我们的物种是这个网络的一部分——数万亿的微生物通过土壤和海洋维持和循环养分，植物产生氧气并滋养我们，无数的物种让我们开发出新的药物或获得新技术的灵感。我们还没有想出如何防止相当大一部分完全不可替代的生物多样性在地球上消失，我们想尝试在真空中从头重新设计它？

我们的物种目前在简单模式下扮演上帝，我们失败了。将我们从一个位于太阳系的适居带的富含水和氧气的星球上解放出来并不会让事情变得更容易——反而会让事情变得更难。

杰夫·贝索斯和我的一个共同点是我们都喜欢 [*【广阔无垠】*](https://io9.gizmodo.com/the-expanses-third-season-hits-amazon-next-month-belte-1831948459) ，这是一部设定几个世纪后人类遍布太阳系的科幻系列片。数百万人生活在火星上，这是一个代际地球改造项目的过程中，无数的小殖民地分散在整个小行星带。这部剧很棒，但我怀疑我们对它的核心信息有不同的理解。我想，在这部剧里，贝佐斯看到了人类通过永久摆脱地球引力而获得自由的模板。我看到人类在一个充满敌意的前沿挣扎求生，他们的身体以不可预测和危险的方式变化着，只是一个钢壳和几平方英尺的空气，在寒冷黑暗的真空中痛苦地死去。

有一天我们可能会扩展到地球以外，那将是令人难以置信的令人兴奋。但是这个星球将永远是我们的家。在我们考虑去其他地方冒险之前，我们需要想办法让它适合居住。

如果我们失去了它，就没有重来的机会。
