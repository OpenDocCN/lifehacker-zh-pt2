# 如何给恐龙称重

> 原文：<https://lifehacker.com/how-to-weigh-a-dinosaur-1844930286>

在恐龙阶段中期和一个孩子交谈是非常棒的。他们几乎肯定比你知道更多关于恐龙的事实和数据，但他们也有很多无法回答的关于的问题，这些生物在灭绝前发生了什么。这是有道理的:很难想象这么大的东西会到处跑。



但是它们到底有多大呢？多亏了一个新的研究项目，我们现在有了一个更好的想法——尽管这反过来又提出了一个古老的问题:如何称一只灭绝已久的恐龙的重量？这是我们所知道的。

### 恐龙称重的基础知识

虽然你可能从来没有具体想过恐龙有多重，但这是古生物学家一个多世纪以来一直试图解决的问题。因此，当来自新英格兰大学 [和皇家安大略博物馆](https://www.rom.on.ca/en/about-us/newsroom/press-releases/how-to-weigh-a-dinosaur) 的研究人员在《生物评论 杂志 [上发表他们的发现时，这是一件大事。](https://onlinelibrary.wiley.com/doi/abs/10.1111/brv.12638)

尽管我们很想报道他们的方法涉及时间旅行和巨大的规模，但事实并非如此。事实上，根本不涉及秤或实际称重。相反，研究人员回顾了自 1905 年以来使用的恐龙体重估计技术，发现尽管采取了不同的方法，但许多方法产生了惊人的相似结果。这意味着我们目前对恐龙大小的概念可能是相对准确的。

如果你想知道这些技术实际涉及到什么，这里有更多来自安大略皇家博物馆 的信息:

> 尽管多年来尝试了一系列不同的方法来估计体重，但它们都归结为两种基本方法。科学家们要么测量并缩放了活体动物的骨骼，如手臂(肱骨)和腿(股骨)骨骼的周长，并将其与恐龙进行比较；或者他们已经计算出了三维重建的体积，这个体积接近于动物在现实生活中的样子。关于哪种方法“更好”的争论已经在文献中激烈展开。
> 
> 研究人员发现，一旦对缩放和重建方法进行整体比较，大多数估计都是一致的。明显的差异是例外，而不是规则。

### 体重只是一个数字，那么这有什么关系呢？

简而言之，因为它让我们对恐龙的长相和行为有了更清晰的了解。新英格兰大学古科学研究中心 [成员尼古拉斯·坎皮奥尼博士在](https://www.rom.on.ca/en/about-us/newsroom/press-releases/how-to-weigh-a-dinosaur) 的一份声明中说:“体型，尤其是体重，几乎决定了动物生活的所有方面，包括它们的饮食、繁殖和运动。”。“如果我们知道我们对恐龙的体重有一个很好的估计，那么我们就有了一个坚实的基础来回顾性地研究和了解它们的生活。”

话虽如此，这项研究背后的研究人员强调，像人类一样，每种恐龙的大小、形状和质量都是独一无二的，所以不可能精确确定每种恐龙的精确重量。“只有通过结合使用这些方法，并通过了解它们的局限性和不确定性，”坎皮奥尼解释说，“我们才能开始揭示这些和其他早已灭绝的动物的生活。”
