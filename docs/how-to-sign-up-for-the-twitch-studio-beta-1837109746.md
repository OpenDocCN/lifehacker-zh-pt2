# 如何注册 Twitch Studio 测试版

> 原文：<https://lifehacker.com/how-to-sign-up-for-the-twitch-studio-beta-1837109746>

本周早些时候，Twitch 宣布计划为游戏玩家开发一款新的一体化流媒体应用程序，名为 Twitch Studio，是该公司的第一款广播软件。



目前，Twitch 广播公司正依靠第三方软件将他们的流媒体提供给观众。Twitch Studio 将理想地使加入 Twitch 和开始一个流，以及实际广播该流的过程更加无缝。

Twitch Studio 还没有为黄金时间做好准备，当它准备好的时候，它将针对新用户，但现在，该公司已经开放了测试版测试。

测试版仅限 PC，需要邀请才能加入。你可以在 Twitch 的网站 [这里](https://www.twitch.tv/broadcast/studio) 注册，有可能获得邀请。

注册页面显示，随着时间的推移，Twitch 将开放更多的测试点。所以，如果你没有马上收到，就要留意那封邮件。
