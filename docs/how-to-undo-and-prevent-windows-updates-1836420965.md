# 如何撤消和阻止 Windows 更新

> 原文：<https://lifehacker.com/how-to-undo-and-prevent-windows-updates-1836420965>

Windows 10 重大更新 [并非一帆风顺](https://lifehacker.com/back-up-windows-10-before-installing-the-october-update-1829556930) 。如果你拥有微软的 Surface Book 2，你可能会有点恼火地知道 [不能更新](https://docs.microsoft.com/en-us/windows/release-information/status-windows-10-1903#536msgdesc) 到微软操作系统的最新迭代 Windows 10 版本 1903，这要感谢英伟达 GPU 的一个错误。



公平地说，当一家公司为安装在硬件配置和功能差异很大的电脑上的操作系统推送更新时，这样的错误肯定会出现。微软通常会很快提供补丁来修复这些问题，但有时如果您扣动扳机并破坏了系统，您需要立即进行修复。

### 如何回滚 Windows update

如果你行动迅速，你可以完全撤销主要的 Windows 更新。安装最新版本的 windows 后，您只能获得 10 天的 Windows，因此，如果您的系统开始出现重大问题，请迅速采取行动..

1.  通过单击 Windows 开始菜单中的齿轮图标，或按“Windows+I”键，打开 Windows 10 设置菜单。
2.  点击**“更新&安全”**
3.  点击侧边栏上的**“恢复”**选项卡
4.  在**“回到之前版本的 Windows 10”**下点击**“开始”**

5.  按照屏幕上剩余的说明完成回滚。

虽然 Windows 10 注意到你*可能*在启动回滚前不需要备份你的文件， [它永远不会伤害](https://lifehacker.com/how-to-back-up-your-computer-automatically-with-windows-1762867473) 。

### 如何卸载次要的 Windows 更新

您还可以卸载自最近一次构建以来应用的所有 Windows 更新。只有当你知道哪个更新导致了这个问题，才使用这个，因为过分的删除会导致更多的问题。

1.  从 Windows 开始菜单点击齿轮图标，或者按“Windows 键+I”，打开 Windows 10 设置菜单。
2.  点击**“更新&安全”**
3.  打开**“Windows Update”**选项卡
4.  点击“更新历史”，然后点击**“卸载更新”**
5.  在出现的新屏幕中，从列表(在“Microsoft Windows”下)中找到您要删除的更新，然后单击**“卸载”**

### 最后一招:重装 Windows

如果以上选项都不起作用，最后的选择是“重置”或重新安装 Windows， [我们在这里有一个指南](https://lifehacker.com/the-ultimate-guide-to-reinstalling-windows-from-scratch-1832897572) 。重置 Windows 将保留您的文件和应用，而全新安装将恢复到 Windows 10 的纯版本，删除您的文件/应用，并将您的所有硬件驱动程序和系统设置重置为出厂默认值。全新安装可能会解决您的软件问题，但这是一个繁琐的过程，应该是您的最后手段。

### 阻止自动更新

一旦您回滚到安全的 Windows 版本或卸载了有问题的 Windows 更新。您可能希望阻止 Windows 自动重新下载和安装它们。只要你运行的是 2019 年 4 月版的 Windows 或更高版本，你就可以在家庭版的操作系统中这样做。(这个功能是 Windows 10 Pro 自动内置的。)

若要暂停更新，请返回 Windows Update 屏幕并查找选项。你不会错过的:
