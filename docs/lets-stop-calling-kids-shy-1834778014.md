# 让我们不要再叫孩子们“害羞”了

> 原文：<https://lifehacker.com/lets-stop-calling-kids-shy-1834778014>

我从小就觉得自己很害羞。这是一个很容易得出的结论，因为每当我处在一个让我感到不舒服的情况下——特别是当我被期待与一个我几乎不认识的人互动时——总有人会不可避免地说，“哦，她很害羞。”或者，更糟的是，他们会直接问我，“你害羞吗？?"



(小贴士:问一个害羞的人他们是否害羞会让这个人感觉不太舒服，而不是更舒服。)

作为一个害羞的人，我很难理解为什么我会去大学学习新闻，在这个职业中，我每天都要和许多陌生人交谈，通常是关于困难或有争议的话题。直到我意识到我并不“害羞”，我只是内向。长时间和很多人在一起会让我精疲力尽，但是我确实喜欢和新的人接触和交流。

那是因为，就像 [作家鲍勃·莱利在他的博客](https://99parent.com/dont-like-word-shy/) 上说的，害羞是一种感觉，而不是一种特质。

> 说某人害羞意味着害羞是个人存在的一部分，而不是在与人交谈或参加他人面前的活动时感到不舒服的暂时状态。

> 这并不是说我不重视这种感觉——在我生命中的大部分时间里，我确实有过这种感觉，但在一天结束时，这只是一种不舒服的感觉。
> 
> 就是这样。
> 
> 孩子们认为自己“害羞”的问题是，这将害羞从一种感觉转化为他们身份的一部分。当负面特征成为孩子身份的一部分时，很难摆脱它。

你可能会感到害羞；这并不意味着你*害羞。然而，当一个孩子在家庭聚会上躲在她妈妈的腿后面时，我们似乎都无法控制自己。我们必须说一些关于这个孩子的事情，他应该像其他孩子一样跑来跑去，兴高采烈地问候所有这些大家庭成员(陌生人)。*

没关系，你可以说点什么。你可以说，“哦，她现在一定很安静。”如果你想让你的好意更进一步，你可以加上:“我明白；我有时候也觉得很安静。”

因为在某些时候，我们都会感到害羞，帮助我们克服这种害羞的最好方法是让我们周围的人在我们所处的地方遇见我们，而不是把我们叫出来，给我们贴上难以摆脱的标签。
