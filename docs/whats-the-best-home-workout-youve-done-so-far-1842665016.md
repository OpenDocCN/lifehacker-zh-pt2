# 到目前为止，你做过的最好的家庭锻炼是什么？

> 原文：<https://lifehacker.com/whats-the-best-home-workout-youve-done-so-far-1842665016>

伙计们，我们需要谈谈。虽然我的身体一如既往地需要锻炼，但我的大脑已经疲惫不堪。我在 [独自进行力量训练](https://vitals.lifehacker.com/how-to-plan-your-strength-training-when-the-future-is-u-1842649573) ，但我必须承认，我喜欢把如何锻炼的决定权交给屏幕上的某个人的想法，他会确保我玩得开心。



因此，在这个月的健身挑战中，我们将一起享受乐趣。我想听听你迄今为止做过的最好的家庭锻炼。不一定是最累人或最有效的，但却是你最开心的一次。

我们的主编在 Instagram live 上对瑞安·赫夫顿 的汗水淋漓的训练赞不绝口。舞蹈指导和导演跳舞，你跟着跳。我自己还没有尝试过，但是 [秃鹫](https://www.vulture.com/2020/04/twerking-through-the-apocalypse-with-ryan-heffington.html) 的这个采访描述了它的感觉:“它是自由的。这是治疗。”这也是为了让不会跳舞的人也能接触到。

一些健身房和工作室也有直播课程，包括。 [这里有一个热情教练的跳绳训练](https://www.instagram.com/p/B-ebovfnxRX/) 。或者你可能更喜欢阿德里安 的 [瑜伽舒缓的声音，她为本月设定了一个主题“养育”。](https://www.youtube.com/playlist?list=PLui6Eyny-Uzy9eeFCGMO7dJNX4TZuQ1VM)

所以让我们知道你一直在努力。任何东西都是公平的游戏，只要你觉得它是令人愉快的，并且你的读者朋友也能看得懂。我们会每周挑选一个锻炼项目或频道一起做，然后再聚在一起讨论进展如何。我们应该先尝试什么？
