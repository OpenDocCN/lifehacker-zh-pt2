# 用这个网站停止用你的病毒之手触摸你该死的脸

> 原文：<https://lifehacker.com/use-this-website-to-stop-touching-your-damn-face-with-y-1842154454>

我喜欢简单的小工具，帮助你更好地改变你的行为——番茄红素；所有那些提醒你喝水的 apps 现在，好老“不要碰你的脸”点 com。



对于那些害怕冠状病毒的人，或者只是想养成更好的习惯，因为它与任何传染病有关，我建议暂时保留“ [不要碰你的脸](https://donottouchyourface.com/) ”作为固定的浏览器标签。或者，更好的办法是，让它在你孩子的电脑上运行，这样他们就不会再用粗糙的、没洗过的手去摸自己的脸了。

它是这样工作的:打开这个网站，它会要求你对你的脸进行快速视频记录。简单。

然后，它会要求你把你的(干净的)手放在脸上，拍摄第二段视频。加倍容易，因为它比原始视频花费更少的时间。现在，每当你把你的脏手举到你的脸上，给它一个快速的轻拍或抚摸，你会得到一个非常明显的警告——如果你启用了它们，还有一个桌面通知。

只要你的标签留在浏览器的后台，警告就会一直存在(假设 [你没有挂起它](https://lifehacker.com/the-great-suspender-frees-up-memory-by-suspending-brows-1702778962) )。

我喜欢这个概念，但这个网站有两个怪癖值得一提。首先，由于所有这些面部训练都是直接在你的浏览器中进行的，而不是在某个地方的服务器上，所以每次重启浏览器时，你都必须拍摄你正常面部和被手遮住的面部的视频。烦人？是的，但这肯定比有人在某个地方的某个服务器上记录了成千上万人的脸要好。

第二，不完善。有时候，当我的手没有靠近我的脸时，我会收到可怕的警告。我责怪我有点蹩脚的摄像头和照明。如果可以的话，确保你在教学过程中处于一个光线充足的地方，以免你的电脑无缘无故地出现故障。

虽然我看不到很多人会永远使用这个方法，但这仍然是一个有趣的摆脱坏习惯的小方法。照我说的做，不要照我做的做，好吗？
