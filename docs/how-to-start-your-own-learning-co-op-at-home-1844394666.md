# 如何在家里建立自己的学习“合作社”

> 原文：<https://lifehacker.com/how-to-start-your-own-learning-co-op-at-home-1844394666>

通常，选择很简单——孩子们要么去学校学习，要么在家上学。通常*、*这两个基本选项中至少有一个符合你家庭的生活方式或你孩子的需求。直到今年。今年，如果孩子们去上学，我们会担心每个人的身体健康，如果他们不去上学，我们也会担心他们(和我们)的心理健康。但是还有另一个选择，一个不错的小妥协，许多家庭开始探索——形成他们自己的小型学习“合作社”



### 一学*什么——现在*？

合作社通常是指由它最终服务或受益的成员所拥有和经营的企业。我们在这里使用的术语有点松散，但想法是相似的——形成一个由大约 3-6 个孩子组成的小组，他们可以一起学习。许多学校将在秋季开放面对面学习，但也会提供虚拟选择。对于那些想让孩子呆在家里，但又想让他们(T2)有一些社交活动的父母来说，以及那些(T4)也必须工作，这使他们无法(T5)促进(T6)虚拟课堂作业的父母来说，合作社可能是答案。

上周，这个话题出现在我们的 [后代脸书集团](https://www.facebook.com/groups/2018785615043946) 中，下面是一位家长如何建立她的合作社:

> 将聘用持有过期教师资格证的人员；幸运的是，她是其中一个男孩的姑姑，她的社交泡沫是众所周知的。小组中的许多父母在家工作，他们真的需要稳定和安全。每周在一个人家里上课，有 5 个孩子。

### 与其他家庭联系

很有可能，如果你正在考虑为你的孩子建立一个小的学习小组，你可能已经有了一些邀请谁加入的想法。你孩子的朋友或同学，邻居，他们垒球队的孩子，你教堂的孩子，公交车站的一些孩子。

不是每个人都会接受这个想法，所以在你找到一两个愿意和你一起迈出这一步的家庭之前，你可能需要四处打听。如果你不感兴趣，有办法将提问范围扩大到你的朋友和邻居之外。尝试寻找脸书当地的育儿团体，或者使用 Nextdoor 应用程序，看看附近是否有家庭对此感兴趣。

你也可以联系你学校的家庭教师协会，看看他们是否知道有其他家庭在努力协调这件事；也许他们甚至可以向你学校的 listserv 发送一个通知来帮助家庭联系——特别是如果你让你的孩子参加学校的在线项目，而不是让他们退出家庭学校，这将影响学校未来的资金。

### 寻找老师或辅导员

这可能是建立你自己的学习舱最具挑战性的部分——谁来运行它？理想情况下，你会像我们脸书小组中的家长一样幸运，其中一个孩子的阿姨有过期的教师资格证和空闲时间。其中一个家庭可能有一位全职父母，他可以让这个小组继续工作。或者，小组中的成年人可以灵活安排工作时间，这样父母可以轮流工作。

如果没有，你需要雇人来管理这个团队。从你的圈子开始，然后努力走出去。你认识退休教师吗？有人在上学时间有保姆吗？你认识的高中毕业生中有谁在开始上大学之前需要一份工作度过一个间隔年？当地有没有被裁掉的日托员工或者幼师？与小组中的其他家长一起集思广益，找出一个合适的人选。

你实际支付给这个人的报酬将取决于你住在哪里，你要求他们工作的时间以及这项工作需要的专业水平。

### 设定一些基本规则

建立你自己的合作社的很大一部分将是制定基本规则。你应该只与家人和一名辅导员组成小组，他们在采取什么措施来保证每个人的安全和健康方面有着相同的想法。每个人都应该分享他们在疫情期间见到的人，并披露他们可能接触到该病毒的任何。如果你足够担心去探索这些选择，你很明显你现在不会想要和那些反面具或者所有关于人群的人组成一个合作组织。

即使每个人都决定尽可能远离团队，也没有办法做到这一点并保持零 T2 风险。我们现在都知道了。因此，你们还需要一起决定学习环境的样子。孩子们仍然应该保持六英尺的距离，每个人都应该戴口罩，特别是在室内，有吃零食、午餐和室外休息的时候(只要有可能)。主持人应确保所有孩子都定期彻底洗手，每次课后应对他们的工作场所和高度接触区域进行消毒。

最后一点:我意识到这个帖子充满了特权，我总是犹豫要不要写一些对每个人来说都不是财务或后勤选择的东西。但是，我们脸书小组的一位家长指出了这些学习合作社可以为整个社区带来的潜在好处——更多的合作社意味着教室里的孩子更少，这意味着那些必须上学的孩子可以分散得更远，每天接触的人更少。
