# 如何停止讨厌玉米糖

> 原文：<https://lifehacker.com/how-to-stop-hating-candy-corn-1838855471>

玉米糖是坏糖。至少这是我多年来坚持的观点，每年 10 月，在我经常光顾的每个社交媒体平台上，公开宣称它是“垃圾”。但是现在有很多真正重要的事情需要大声疾呼，在网上为糖果疯狂似乎是愚蠢的。无论在哪里，人们都应该找到快乐，即使这种快乐是以季节性款待的形式出现的。

所以今年我取消了反对玉米糖的活动，取而代之的是试图理解它的吸引力，甚至可能享受它。尽管 Lifehacker 对这种像软糖一样的糖果有相当大的分歧，但两位鉴赏家——我们的执行主编 [弗吉尼亚·史密斯](https://kinja.com/vksmith) 和副主编 [爱丽丝·布拉德利](https://kinja.com/alicebradley1)——还是很友好地分享了他们的品尝笔记和上菜建议。

首先，要知道不是任何品牌都可以。真正的玉米糖爱好者知道 Brach's 是国王，因为 Brach's 在他们的食谱中加入了一点点盐，当你处理这样一种甜食时，这是至关重要的。“商店里没有品牌玉米，”弗吉尼亚告诉我。爱丽丝对此也深有感触，而且不止一次提到过。

但是，如果盐的味道还不够，甚至 Brach 的味道像未加控制的糖膏，不要烦恼——弗吉尼亚有一个解决方案。将这些小粒与蜂蜜烤花生搭配。弗吉尼亚多年来一直试图说服我尝试这种咸甜豆类和甜甜糖的组合，坚持认为它们的组合味道与黄油手指惊人地相似。

我终于尝试了配对，她没有说谎，一点也没有。虽然我仍然不喜欢玉米糖本身，但我现在对这种组合很感兴趣，并确定了玉米和花生的绝对最佳比例，即每两个完整的(而不是减半的)蜂蜜烤 bois 对应一个玉米粒。即使你认为你看不起蜜饯玉米，我也强烈建议你尝试一下这种搭配。它尝起来确实像黄油手指，但感觉更有节日气氛。
