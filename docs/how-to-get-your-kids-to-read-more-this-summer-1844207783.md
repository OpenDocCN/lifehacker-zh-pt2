# 如何让你的孩子在这个夏天多读书

> 原文：<https://lifehacker.com/how-to-get-your-kids-to-read-more-this-summer-1844207783>

让我们的孩子坚持阅读是一个挑战。但是考虑到刚刚过去的学年，你可能会特别担心你需要帮助他们提高阅读水平，至少这样他们就不会在秋季到来的几个月里退步。这些年来，我们已经写了很多关于阅读的东西，所以我为你们整理了一些我们给小孩子和大孩子的最佳建议，以及一些我们认可的整体技巧。

### 给小孩子的

小孩子通常喜欢阅读。他们喜欢与父母或看护人一对一的时间，他们喜欢翻页，一遍又一遍地读同一本书，这样他们就知道接下来会发生什么。然而，你可能仍然会发现自己墨守成规，每天在同一时间阅读同一本书。

这里有一些策略可以让你们俩(如果你们一起阅读)或他们(如果他们单独阅读)的阅读体验焕然一新:

### 对大孩子来说

当谈到正常的阅读时间时，年龄较大的孩子可能更难接受。[在九岁左右，许多孩子为了好玩](https://lifehacker.com/why-kids-stop-reading-for-fun-by-age-9-and-what-to-do-1833554143) 而停止阅读。他们的日程安排变得满满的(或者说，至少，这是疫情之前的真实情况)。大人们已经不再给他们读书了。他们开始认为这不仅仅是个人享受，更是对他们的期望。但是你仍然可以和你的大孩子做一些事情来让他们继续阅读，即使这些事情有点不合常规:

*   一旦你的孩子学会阅读，不要停止给他们读书
*   [开始与他们交谈](https://offspring.lifehacker.com/start-a-conversation-journal-with-your-kids-1844038268)
*   [帮他们找笔友](https://offspring.lifehacker.com/how-to-get-your-kid-a-pen-pal-1840932649)
*   让他们熬夜阅读

### 在任何年龄，欺骗他们一点

我不是说你应该全力欺骗你的孩子去阅读，但我是说，当文字是一本书面书籍或杂志的一部分时，阅读并不仅仅是“有价值的”。

我的儿子经历了一个巨大的神奇宝贝阶段，在此期间，他会仔细阅读卡片，阅读那些非常高级的单词和名字。当他的老师给他的班级分配额外的夜间和周末阅读时，我们决定研究他的神奇宝贝卡片至少会占那段时间的一部分，特别是因为他大声地向我们读出了事实。

弄清楚他们现在喜欢什么，用语言来满足他们的热情。获得关于他们最喜欢的运动或话题的书籍。买僵尸书或者魔法书或者笑话书。把他们弄成漫画——[漫画*是*书](https://offspring.lifehacker.com/comic-books-are-books-1822641467) ，你懂的。搞定他们 [狂库](http://www.madlibs.com/) ！

阅读不应该感觉像是一件苦差事，所以没有理由让它成为一件苦差事；这里灵活一点是好的。
