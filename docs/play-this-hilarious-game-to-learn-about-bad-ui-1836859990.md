# 玩这个有趣的游戏来了解糟糕的用户界面

> 原文：<https://lifehacker.com/play-this-hilarious-game-to-learn-about-bad-ui-1836859990>

如果一个网站曾经因为太多的弹出窗口、设计愚蠢的菜单或无数次证明你是人类的测试而让你烦恼，那么网页游戏 [用户 Inyerface](https://userinyerface.com/game.html) 会在让你发笑的同时让你抓狂。

Watch

这款游戏来自设计和技术公司 [Bagaar](https://www.bagaar.be/) ，挑战你简单地注册一个网站——同时克服可怕的、可怕的设计形式和菜单。你必须手动避开弹出窗口，找到隐藏的链接，避免破坏你的工作的按钮，并满足荒谬的密码要求。

这个游戏只需要几分钟，但那感觉就像是几个小时甜蜜的痛苦。(这让我们想起了臭名昭著的 Flash 游戏 [青蛙分数](http://twinbeard.com/frog-fractions/) 。)用户 Inyerface 演示了一个设计糟糕的应用程序或网站，感觉就像一个具有挑战性，但又非常不受欢迎的视频游戏。下次你不得不为你的保险公司、社区剧院或地方政府浏览一个糟糕的网站时，记住这个游戏，也许你太阳穴上的一点点刺痛就会消失。

如果你认识一个设计师，给他们发一个链接。他们会喜欢你的。
