# 你应该知道的瑜伽术语

> 原文：<https://lifehacker.com/the-yoga-terms-you-should-know-1831084530>

正如芭蕾舞者需要学习几句法语，跑步者需要能够听到和说出瑞典语单词 *fartlek* 而不会咯咯笑得太多，练习瑜伽的人如果能够理解瑜伽课和瑜伽视频中常见的梵语和其他术语，就会做得最好。

如果你是瑜伽老手，我很想听听你在开始时犯了哪些错误，你会给初学者什么建议，让他们在第一堂课上遵循。同时，这里有一些每个人都应该知道的术语:

*   Namaste :印度的一种问候方式，大致意思是“我向你的神性鞠躬。”它通常开设和关闭瑜伽课程；准备双手合十，低头。(环顾四周，做其他人都在做的事情。)

*   连续运动:从一个姿势到另一个姿势的连续运动。一朵花。
*   阿斯汤嘎，比克拉姆，哈他，尹，艾扬格:这些都是不同类型的瑜伽。 [这张图](https://lifehacker.com/find-the-right-yoga-style-for-you-with-this-chart-5991512#_ga=2.188372776.1055601077.1544468859-3846207152.1521480874) 可以帮你弄清楚哪个才是适合你的。

瑜伽中的体式叫做*体式*，重音在第一个音节，听起来像“啊”有些有普通的英文名字，而有些你可能会听到梵文名字。

*   **下犬**:双手着地，屁股朝天的那种。你会经常做这个。
*   **拜日式**:这是一个流程，你站得很高，弯下腰，以平板姿势结束，然后努力恢复到站得很高。有变化，但 [这里有一个视频的一个常见的做法](https://www.yogajournal.com/videos/salute-the-sun) 。
*   **山式**:挺立，也叫山式。
*   想象做一个俯卧撑，但是做了一半就停下来。如果可以的话。
*   **Updog** :不，这不是 [开玩笑](https://knowyourmeme.com/memes/updog) ！这是仰面犬的简称，一种小腿放在地板上，上身垂直的姿势。眼镜蛇的姿势也差不多: [下面是两者的合照](https://www.self.com/story/fitness-difference-between-upward-facing-dog-cobra-pose) 。
*   Utkatasana，或椅子式:像蹲一样，但更优雅。假装你身后有一把看不见的椅子，然后坐在上面。如果那太难，就假装它是一个高脚凳。
*   [**Virabhadrasana**](https://en.wikipedia.org/wiki/Virabhadrasana_I)**或者战士一、二、三**:这些都是站姿。在战士 I(“战士一”)中，你的一只脚在垫子的前面，一只脚在后面，你面向前方。这就像一个弓箭步。在战士 II 中，你的胸部转向垫子的一侧。而战士三是独腿平衡。 [下面是三个](https://www.doyouyoga.com/the-ultimate-guide-to-the-warrior-poses-69050/) 的指南。
*   **Savasana** :尸体姿势，基本上仰卧着什么也不做。许多瑜伽课程都以一个尴尬的长时间的 savasana 结束。不，教练没有抛弃你。大概吧。你可以睁开眼睛检查。
