# 你应该放一个吉百利鸡蛋在火上

> 原文:[https://life hacker . com/you-should-set-a-Cadbury-egg-on-fire-1842064191](https://lifehacker.com/you-should-set-a-cadbury-egg-on-fire-1842064191)

复活节糖果是最好的糖果。没有其他糖果有这个范围！复活节糖果不仅包含巧克力和水果口味的糖果，你也不能否认蛋形糖果客观上是有趣和美味的。(里斯的鸡蛋花生酱和巧克力的比例最好；树和心可以咬我的屁股。南瓜很好。)

吉百利的复活节做得很好，尽管我从来没有完全喜欢过奶油蛋。(我更喜欢迷你鸡蛋女孩，虽然我也喜欢焦糖填充的男孩，尤其是一大杯冷牛奶。)并不是说奶油鸡蛋不好；它们只是一个小小的一维空间，而这个一维空间是一个甜蜜的监狱。

如同任何其他令人作呕的糖果(以及一般的)一样，最好的做法是将 [放在火上](https://skillet.lifehacker.com/you-should-brulee-your-next-glazed-doughnut-1833377289) 。除了纯糖，焦和焦糖化还能让你的舌头有所探索，这是*乐趣*。多亏了软糖馅料和易融化的牛奶巧克力，一个烤焦的奶油鸡蛋有很大的 s'mores 味道。把它放在全麦饼干上，开心点。

为了获得最好的吉百利奶油鸡蛋体验，您需要一点佐餐糖。我知道这听起来很疯狂，因为我只是不停地谈论这些东西有多甜，但方丹不喜欢形成外壳，巧克力外壳上一点额外的糖可以防止它在你形成焦糖外壳时完全燃烧和融化。(它仍然会燃烧，但会燃烧得更少。)您还可以冷冻鸡蛋，以防止巧克力融化——一个小时应该足够了。

要做这些烤焦的时令美食，你需要:

*   至少一个吉百利奶油鸡蛋
*   一堆餐桌上的糖
*   锋利的薄刀片

如果你不想要融化的鸡蛋，把它放在冰箱里一个小时，但是我认为融化的鸡蛋也可以。打开鸡蛋，找到垂直环绕它的接缝。将你的刀片压入鸡蛋的缝隙，打开鸡蛋，将任何错误的软糖舀回巧克力外壳中。快速将鸡蛋的软糖面朝下压入糖堆中。放在火焰安全的表面上，用厨房的火把糖在鸡蛋上轻轻来回扫动。照吃不误，把它放在全麦饼干上，或者放进冰淇淋里做季节性圣代。
