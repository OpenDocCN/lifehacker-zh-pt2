# 使用共享的谷歌日历来计划与朋友的时间

> 原文：<https://lifehacker.com/use-a-shared-google-calendar-to-plan-time-with-friends-1836665639>

我们很多人都想和朋友多呆一会儿——但在晚上或周末大家都有空的时候，往往要来回跑好几圈才能到达。



共享的谷歌日历可以解决一些日程安排问题。不一定是你很难找到每个人都能聚在一起的日子，而是你必须等待每个人回复关于周三或周四是否可行的群发短信(总有一个朋友不经常回复短信，这意味着其他人不得不在他们的日程上安排所有可用的日期，直到他们有机会回复，等等。等等。等等。).

在 [《大西洋》](https://www.theatlantic.com/family/archive/2019/07/planning-friend-hangouts-google-calendar/594586/) ，托里·莱瑟姆写道共享的谷歌日历如何增进了她的友谊:

> 忠于形式——对组织的热爱是我的定义特征之一——我创建了一个共享的谷歌日历。每个人都可以标记他们何时会出城或被占用(用“OOO”来表示，这是“不在办公室”的职场说法)，也可以提前为更大的活动(晚餐、生日聚会、周末旅行)写下“保留”。令我欣慰的是，我的朋友们只需一点哄骗就接受了这个想法。“检查 G-cal”和“把它放在 G-cal 上”已经成为我们聚会时常用的口头禅(即使这些话有时是讽刺性的，还会翻白眼)。

我曾使用共享的谷歌日历来计划与前男友的约会，所以我知道这个系统是可行的——尽管我目前的社交团体没有共享 G-Cal，但我们仍然定期向对方发送谷歌日历邀请，以建议和正式确定聚会地点。

这就是我喜欢使用 G-Cal 的原因:

*   它通过电子邮件传递，而不是文本。我是那种不常回短信的朋友。我的手机永远是面朝下的，处于静音状态，所以我可以专注于我的笔记本电脑，完成我的工作。这包括检查和处理我的电子邮件，这意味着我更有可能看到和回复活动邀请。(是的，我知道你也可以在笔记本电脑上收到短信通知。当我意识到它们在那里的时候我就把它们关掉了。)
*   它帮我把朋友的活动放在我的日历上，并负责提醒。我不必记得将晚餐讨论从短信链转移到我的日历上，也没人需要发送提醒短信。
*   建议不同的时间或改变计划变得非常容易。如果一个朋友想在我不方便的时候出去玩，我可以使用 G-Cal 的“提议新时间”功能来同意出去玩——这让每个人都感觉很好——同时提供一个对我们双方都合适的时间。如果我们有一个共享的谷歌日历，我们甚至不必做这一步(假设 [我们保持我们的日历准确更新](https://lifehacker.com/use-your-calendar-to-record-what-actually-happened-1835943219) )。同样，如果一个朋友邀请我吃午饭，突然有事，重新安排时间也是非常容易的。
