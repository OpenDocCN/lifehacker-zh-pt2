# 为什么 Windows 不能识别我的 Android 手机？

> 原文：<https://lifehacker.com/why-wont-windows-recognize-my-android-phone-1844869321>

你有机器人。*检查。*你有一台电脑。*止，止。*你想把你的安卓系统连接到你的电脑上，从电脑上抓取文件(或者添加文件，就像你花了几个小时设计的那些铃声一样)。*检查三次。*将手机插入台式机或笔记本电脑的 USB 端口，然后...什么都没发生。



欢迎来到《生活黑客》读者斯图的科技地狱小角落，他在本周的 [科技 911 Q&](https://lifehacker.com/c/tech-911)中描述了这个小角落:

> 多年来，我一直用 USB 线在我的三星 S7 手机和电脑之间传输文件，一切正常。然后几周前我发现 Windows 10 文件浏览器不再识别我的手机。它在设备管理器中得到识别，在我的 Windows 10 笔记本电脑上运行良好。我和三星和微软打了几个小时的电话，都没有解决方案。下面是我试过没有成功的:
> 
> *   *卸载和重新安装手机驱动程序。*
> *   *使用几种不同的 USB 电缆*
> *   *使用不同的 USB 端口*
> *   *更新整个 Win 10 操作系统*
> *   *确保手机中的 USB 调试已开启*
> *   *确保手机处于 MTP 模式*
> 
> 可能有一些我忘记了，但如果你有任何想法，我会很高兴听到。

### 将 Android 连接到 PC 应该很容易，对吧？

Stu，如果这个回答转专栏有点短，我提前道歉，因为你已经做了很多我会建议的故障诊断。这太棒了！你已经领先游戏六步了。即使你不能想出一个可行的解决方案，事实上你可以在你的“请不要解决这个问题”列表中划掉一些想法，这是朝着正确的方向迈出的一步，这有助于我们缩小范围，尝试一些其他的技巧。

鉴于*没有什么*能够解决你的问题，并且你已经尝试了我推荐的所有常用步骤，我的第一个想法是将你的手机重置为出厂设置。我知道这是以核武器为先导，而不是外科手术式的打击，但我想知道简单地从你的 Android 上从头开始是否可以解决现在让 Windows 10 无法看到你的手机的任何设置。

首先使用 [三星云](https://www.samsung.com/us/explore/samsung-cloud) 备份你的工作手机。一旦你完成， [工厂重置](https://www.verizon.com/support/knowledge-base-203263/#:~:text=Press%20and%20hold%20the%20Volume,select%20Wipe%20data%2Ffactory%20reset.) 它，我建议当你第二次设置它的时候不要从任何备份中恢复。做你已经做过的事情——启用 USB 调试，将手机设置为 MTP 模式，等等——并将其连接到 Windows 10 系统。如果你的笔记本电脑*还是*看不到手机，我感觉我们可以排除手机是罪魁祸首了。差不多了。

我会尝试的一件事是，看看你是否能让任何 Windows 10 系统识别你的手机。即使我们处于隔离期，也许你可以借一个朋友的笔记本电脑或台式机，用我刚才描述的方法插上电源，看看会发生什么。如果它出现在文件浏览器中，那么我们可以合理地假设你的笔记本电脑或 Windows 10 操作系统有问题。如果没有，嗯，*射*。我不知道出了什么问题。也许让你的 Galaxy S7 扎根并安装第三方 Android 操作系统 的 [可能会解决你的问题？我们在这里得到了极端的解决方案，但是如果你经常需要在你的手机和笔记本电脑之间传输文件，这是一个令人烦恼的问题。](https://lineageos.org)

如果另一台 Windows 10 电脑允许你访问你的 Android，而你的不允许，我最初的想法会是说“去他妈的”，然后 [重新安装*你的*版本的 Windows 10](https://lifehacker.com/the-ultimate-guide-to-reinstalling-windows-from-scratch-1832897572#:~:text=The%20fastest%20and%20easiest%20way,10%20ISO%20directly%20from%20Microsoft.) 。再说一次，这是另一个核武器，但我经常发现从头安装 Windows 10 比尝试解决操作系统的一个特别棘手的问题花费的时间更少。此外，您还可以获得一个可爱、干净、快捷的工作系统带来的所有额外好处。

然而，在使用这种方法之前， [克隆您现有的驱动器](https://lifehacker.com/how-to-migrate-to-a-solid-state-drive-without-reinstall-5837543) 。如果你的 Windows 10 重新安装没有解决你的问题，你可以用你的克隆版本覆盖它，以恢复你的旧操作系统(和你的数据/文件/设置/应用程序等)。)

当你这么做的时候，我会确保你已经安装了你的笔记本电脑制造商可能提供的任何 BIOS 更新或更新的 USB 驱动程序。如果您还没有尝试过，这也有可能解决您的问题。(如果是一台较旧的笔记本电脑，Windows 10 用于 USB 驱动程序的版本很可能比笔记本电脑制造商网站上的版本更新。请不要安装，比如说，某种五年前发布的驱动程序包。)

我怀疑——这似乎被其他报道 证实了——某种主要的 Windows 更新破坏了三星 MTP 的驱动程序。所以，在你淘汰并重装 Windows 10 之前，考虑一下试试这个建议(步骤 7)。在设备管理器中找到您连接的 galaxy S7，即使它被列为“未知设备”，并尝试使用“让我从列表中选择...”来更新驱动程序方法。选择“安卓手机”，并选择“MTP USB 设备”选项，而不是“MTP 设备”

否则，你可以尝试使用像[**air droid**](https://www.airdroid.com)T5】或[**super beam**](https://play.google.com/store/apps/details?id=com.majedev.superbeam)这样的应用程序/服务，将你的文件从安卓系统传到笔记本电脑。你也可以选择一个便宜的 microSD 卡和 USB 读卡器来管理设备之间的文件，这种方式可能比直接连接更麻烦。最后，如果这一切都让你抓狂，你可以在你的笔记本电脑上双启动一个早期版本的 Windows 10(甚至，比如说，Windows 8.1 或其他版本)。从生产力的角度来看，这确实不理想，但这确实是一个解决方案。
