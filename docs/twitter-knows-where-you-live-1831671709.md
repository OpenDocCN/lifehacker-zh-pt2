# 推特知道你住在哪里

> 原文：<https://lifehacker.com/twitter-knows-where-you-live-1831671709>

全世界真的需要看到你发微博的地方吗？我通常不会考虑我在 Twitter 上分享了多少位置数据，因为我认为没有人会关心我的推文和我在哪里推文。然而,《连线》杂志的一份新报告让我重新思考我的方法。事实证明，通过你的推文准确定位*你住在*并不困难。

Watch

正如《连线》的故事所述，来自伊利诺大学和希腊研究与技术基金会的一名研究人员 [创造了一个名为“LPAuditor”或“位置隐私审计员”的工具](https://arxiv.org/pdf/1901.00897.pdf) ，以利用许多用户多年来一直在 Twitter 上分享他们精确的 GPS 坐标的事实——即使他们的推文附有更通用的“你居住的巨大城市的名称”。[按](https://www.wired.com/story/twitter-location-data-gps-privacy/)条:

> 多年来，选择给推文添加地理位置标签的用户，即使是像“纽约市”这样地理位置广阔的地方，也会自动给出他们精确的 GPS 坐标。用户不会看到 Twitter 上显示的坐标。他们的追随者也不会。但 GPS 信息仍将包含在推文的元数据中，并可通过 Twitter 的 API 访问。
> 
> 直到 2015 年 4 月，Twitter 才在其应用中改变这一政策。现在，用户必须选择加入分享他们的精确位置——根据 Twitter 发言人的说法，只有很少一部分人会这么做。但是直到今天，人们在更新之前共享的 GPS 数据仍然可以通过 API 获得。

哦。谢天谢地，Twitter 让删除你发送的任何位置数据变得非常容易，也许是时候尝试一下这个功能了。要找到它，请访问您的 Twitter 帐户设置中的“”屏幕。此设置应该靠近页面顶部:

我建议关闭“带位置的推文”，因为你可能不需要与任何人分享你的精确位置——无论是你的粉丝还是 Twitter 的 API。完成后，点击大的“删除位置信息”按钮即可。

虽然当其他人使用 Twitter API 试图确定你去了哪里时，这肯定会帮助你保持更多隐私，但 [Twitter 注意到](https://help.twitter.com/en/using-twitter/tweet-location) 删除你的位置数据可能不是一个万无一失的方法:

> *“需要注意的是，删除 Twitter 上的位置信息并不保证该信息会从第三方应用程序或外部搜索结果中的所有数据副本中删除。此外，此设置不会删除通过直接消息共享的位置。*

尽管如此，总比什么都没有好，尤其是因为 Twitter 似乎没有兴趣代表你修补这个位置数据问题。当《连线》杂志联系 Twitter 询问为什么特定的 GPS 坐标仍然存在于该公司的 API 中，甚至在该公司于 2015 年 4 月转向选择加入模式后，该公司的回应是:

> “我们觉得不经人们同意就单方面改变他们的推文是不合适的。”

没错。我敢打赌，很少有人会在意，甚至会注意到，Twitter 已经移除了 2014 年发布的一条推文中的地理标签。然而，用户可能会更关心，如果他们的地址因为旧数据仍然可以通过 Twitter 的 API 获得而被散布在网络上。
