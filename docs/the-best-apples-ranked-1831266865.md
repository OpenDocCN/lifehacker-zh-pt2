# 最佳苹果，排名

> 原文：<https://lifehacker.com/the-best-apples-ranked-1831266865>

我们都对最好的苹果有自己的看法，但这是正确的看法。我的资历:我吃过很多苹果。我很幸运地品尝了古董品种的苹果和实验研究品种的苹果。如果你想知道最好的杂货店苹果，我可以告诉你。如果你想要真正最好的苹果，这些很难找到，但是我可以告诉你去找哪一个。



### 最好的杂货店苹果

**1。花式包里那些贵的。**这些是有商标名称的“俱乐部苹果”，其中 [种植者必须保持某种俱乐部的会员资格才能种植它们](https://www.npr.org/sections/thesalt/2014/11/10/358530280/want-to-grow-these-apples-youll-have-to-join-the-club) 。俱乐部的要求之一是他们必须非常擅长质量控制，所以他们往往是以良好的状态出现在商店的好品种。我从来没有对这些失望过。SweeTango 就是这样一个品牌。

**2。甜脆饼干。抱歉，甜心。你很美味，但是你太大了，所以你不可能是第一名。你不适合任何人的苹果切片器，俱乐部苹果通常只是更脆一点，更多汁一点，更甜一点。**

**3。粉红女郎。**第三名不好补；如果粉红女郎已经过了她们的全盛时期，她们很容易被下面的一个亚军超过。

**荣誉奖:**帝国，富士，加拉，爵士

### 最好的苹果馅饼

**1。史密斯奶奶。这个苹果有一系列的特点，使它不适合作为休闲零食:太酸、太干、太不平衡。我们的食品和饮料编辑克莱尔·洛喜欢这些加盐的；我相信这是一个需要合作才能发光的好苹果。和奶酪一起吃也不错。它在馅饼中特别好，它的酸味被糖平衡了，它的干燥意味着它很容易做出质地良好的馅饼，永远不会变得粘稠。**

**2。金色美味。**这是一个非常遥远的第二名，但是克莱尔注意到 [在馅饼烘焙测试](https://skillet.lifehacker.com/forget-red-delicious-here-are-the-apples-you-should-us-1732697809) 中表现良好。不过，要让它们保持新鲜，因为悲伤、过季的金色美味一点也不好玩。

**荣誉奖:**任何你喜欢的酸苹果，包括 Jonagold、MacIntosh 或其他地方列出的零食苹果。陆奥号，也被称为 Crispin，与 Golden Delicious 味道相似，可能会做得很好。

### 最好的鲜为人知的苹果

这些不太可能出现在你当地的超市里，但是要留意农贸市场和任何你发现奇怪特产的地方。

**1。罗克斯伯里·拉塞尔。这是一个绿色的小苹果，有粗糙的褐色斑点。看起来很丑。味道好极了，又酸又多汁。**

**2。阿肯色州黑人。**另一种难看的:这些是深红色，存放时变暗到近乎黑色。换句话说，它们看起来像已经变质的苹果。但是咬一口，哇，它们又脆又多汁，非常完美，即使在储存了几周或几个月后也是如此。

**3。考克斯的橘子皮聘。这是一个奇怪的苹果，但我认为每个人都应该尝试一次。它有一种水果味，尝起来几乎和橘子皮兹糖一模一样。我无法解释；这种苹果品种可以追溯到 1830 年，所以味道完全是天然的，不像(颤抖)格斗。** 

**荣誉奖:**阿什米德的内核，扎贝尔高·雷纳特，自由，北方间谍，怀恩萨普

### 最差的苹果

只是提醒一下，我们都讨厌蛇果，这个苹果擅长看起来像苹果，但不擅长尝起来像苹果。

然而，要得到那种湿纸板的味道， [我们需要达到几样东西](http://www.washingtonpost.com/wp-dyn/content/article/2005/08/04/AR2005080402194.html) 。首先，它应该是现在流行的厚皮、暗红色的后代之一，*而不是*[原来好吃的苹果](https://usdawatercolors.nal.usda.gov/download/POM00001917/screen) 更有条纹，记录暗示，实际上很好吃。

它还应该在完全成熟之前采摘，并在可控气氛中保存数月。把它放在便利店的水果篮里，它就会成为我们名单上的最后一个。不要试图从树上摘新鲜的蛇果，或者诸如此类的废话。这完全会打乱我们的排名。
