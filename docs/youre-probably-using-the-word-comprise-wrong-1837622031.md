# 你可能用错了“包含”这个词

> 原文：<https://lifehacker.com/youre-probably-using-the-word-comprise-wrong-1837622031>

在我深情地解释了 [“谁”和“谁”](https://lifehacker.com/how-to-remember-the-difference-between-who-and-whom-1837581203) 之间的区别后，一位亲爱的读者意识到了一个事实:我一直在进行一场 [一个女人的圣战](https://lifehacker.com/tag/grammar-police) 来净化我们这个世界上所有可怕的语法。我只知道那个叫“Fabian Knockwurst”的人来找我，要求“做*来组成* next”，我很乐意答应。



这是一个特别好的话题，因为“comprise”的不正确用法现在非常普遍，很可能*就差*一点就可以被接受了。所以让我们把这看作是一次语法上的孤注一掷——让你们使用自己真正想要的词的最后努力。那就是:“作曲”

“包括”是一个动词，意思是“由…组成”、“由…组成”或“由…组成”为了检查你是否正确地使用了它，把“组成”换成“包含”或“包含”

这个队由各种技能水平的运动员组成。(这个队由各种技能水平的运动员组成。)

然而，这个团队并不是由所有技能水平的球员组成的。不，团队是由各种技能水平的玩家组成的*。Compose 是一个动词，意思是“弥补”我知道，我知道；*如此之近。*但不一样。碎片组成整体；整体由片断组成。*

听着，如果其他的都失败了，记住“包括”这个词永远都不会是正确的。 [语法女孩](https://www.quickanddirtytips.com/education/grammar/comprise-versus-compose) 这样说:

> 正如你不能说，“房子包括七个房间”，你也不能说，“房子由七个房间组成。”你必须说，“房子由七个房间组成。”

还是坚持用"是由"。我不做评判。
