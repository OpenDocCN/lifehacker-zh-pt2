# how to successfully start a new job remotely 1844247623

> 原文：<https://lifehacker.com/how-to-successfully-start-a-new-job-remotely-1844247623>

很难想象有哪个职业没有受到新冠肺炎全球疫情的影响。无论你是丢掉了工作，转变为在家工作，还是作为一名重要员工不得不遵循新的协议，大多数人在过去的几个月里都经历了工作生活的重大变化。尽管由于新型冠状病毒的爆发，失业率飙升，信不信由你，一些公司仍在招聘新员工。

假设你是他们中的一员，祝贺你！你完成了看似不可能的任务，并设法在疫情期间找到了一份新工作。这是一个不小的壮举。但这也可能是你以前从未做过的事情——尤其是如果你的新职位是经理的话。那么，你该如何为自己的成功做准备呢？



根据临床心理学家、《T4》的作者洛里·沃特利 博士的说法，心态决定一切，接受一份新工作，领导一个团队——即使是在这些新的、陌生的环境中——也不例外。“这肯定是一个令人兴奋的绝佳机会，”她告诉 Lifehacker。“当我们能够将不同寻常的环境视为机遇时，我们就能克服焦虑。做到这一点的一个有效方法是，专注于你希望在新工作中实现的目标，并让你的新团队成为他们的领导者。”

不管新职位是什么，Whatley 说重新规划意想不到的障碍对成功是至关重要的——关注最终结果，而不是达到所述结果的问题。“训练你的思维远离那些让你害怕的事情，因为你凝视的地方就是你的方向，”她解释道。为了找到更多关于远程开始新工作的策略，我们采访了许多专家。这是我们学到的。



### 远程管理团队的新工作建议

如果你擅长管理一群人，确保一切高效运转，你可能知道如何成为一名实体办公室的经理，但虚拟办公室呢？这里有一些让你成功的方法。

#### 与你的团队单独联系

人力资源专家反复强调的一点是，沟通在远程工作中非常重要。当然，这在办公室也很重要，但是当你不能亲自看到你的团队时，你的一些信息可能会丢失或被误解。这就是为什么威利·格里尔建议与你的团队成员单独沟通，他创立了 [、产品分析师](https://theproductanalyst.com/) ，并在公司初创时担任人力资源团队的负责人。他告诉 Lifehacker:“在远程领导团队的过程中，你有责任亲自向他们传达关注点或任务。”“最好时不时地和他们联系，询问他们所经历的挑战，以便能够带来更舒适的工作环境。”

#### 让你的老板给你安排一个长期经理

想了解在新工作单位做经理的内幕吗？JookSMS 的营销经理克里斯多佛·普拉萨德建议，让你的新老板帮你联系一个以前担任过类似职位的人，并从他们那里获得关于你具体工作情况的建议。他告诉 Lifehacker 说:“(这)可以阻止一个人得到所有新人需要问的问题。”。“新员工需要对诚实地提问感到自在，而不必担心与人接触。如果你试图回避小事情，它会滚雪球般变成更大的问题。记住，团队希望你成功。”

#### 设定明确的期望

与团队有效沟通的一部分是为他们设定明确的期望。“向每个人解释你的工作风格和你对团队的期望，”Chegg 前高管、[Imagine Easy Solutions](https://techcrunch.com/2016/05/02/chegg-acquires-imagine-easy-solutions-the-company-behind-easybib-bibme-and-citation-machine/)的创始人尼尔·塔帕瑞艾(Neal TapariaI)告诉 Lifehacker。“例如，如果划分家庭和工作生活很重要，让你的团队知道你不希望他们在下午 6 点以后回复邮件。”



#### 领导就像你和你的团队在同一个房间里一样

如果你远程开始一份新工作，Loanry 的首席执行官伊桑·陶博(Ethan Taub)说，你必须记住，主要的区别是你和你的同事不在一个房间里。“一个团队需要清晰和理解的领导——你必须成为他们的代言人，”他告诉 Lifehacker。“你必须激励他们去做工作，就像你们在同一个房间里一样。这是完全可能的，只要你也感受到动机。”



### 给任何开始远程工作的人的提示

无论你是高级经理还是低级员工，任何人都可以采取一定的步骤来开始远程工作。这里有几个。

#### 准时开始

这似乎是显而易见的，但你绝对需要准时出现在一个偏远的位置。与面对面的工作不同，你不能再用“通勤不方便”作为迟到的借口。仅仅因为你可以穿着睡衣工作，并不意味着你可以不守时。



#### 把你的问题列成一个清单

无论你在开始一个新职位时接受了多少培训，通常在你学习一份新工作时，都有很多东西需要保留和应用。问题不可避免地会出现，所以当问题出现时，把它们写下来， [KISSPatent 创新咨询](https://kisspatent.com/) 的创始人兼首席执行官德沃拉·格莱泽博士建议。把你的问题列成一个清单，这样你就不会忘记它们。此外，向你的经理询问他们回答你问题的首选地点。也许他们希望你一有问题就问，但也许他们更希望你带着你的清单去一个预先安排好的会议，一口气解决掉一堆问题。

#### 了解你的同事

开始一份新的面对面工作最有压力的部分之一是被扔进一个其他人都已经相互认识的办公室生态系统。当你们不在同一个办公室时，了解每个人——他们的工作方式、幽默感以及其他可以帮助你们更有效地合作的基本信息——可能会特别令人生畏。“我不是说你们应该成为朋友，而是要意识到他们是谁，他们的时区是什么(这一点超级重要！)，他们的职责是什么，以及任何其他对你工作很重要的信息，”格莱泽告诉 Lifehacker。“基本上，有很多问题！你问得越多，就越容易。”

当你远程开始一个新的职位时，在了解你的同事方面，你可能看起来像是摆脱了困境，但实际上恰恰相反。花点时间和他们接触——尤其是那些你将直接共事的人——开始相互了解的过程，即使你们不在同一个房间里。MyCorporation.com 机场 [的运营总监达纳·凯斯(Dana Case)建议通过 Zoom 与你的新同事见面，并继续定期进行面对面的视频聊天，以保持联系。她补充道，如果你在监督他们的工作量，这一点尤其重要。](https://www.mycorporation.com/)
