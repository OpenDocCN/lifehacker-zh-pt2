# 如果你在抗议时被逮捕，该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-youre-arrested-while-protesting-1843921685>

参加抗议时了解自己的权利很重要。在上面的视频中，我们采访了律师 Maryanne Kaishian，她是布鲁克林辩护服务公司的一名高级律师，讲述了如果你发现自己在抗议活动中被逮捕，应该采取的正确步骤。

### 在手臂上写下重要的电话号码

在参加抗议之前，Kaishian 建议在手臂上写下律师或法律服务机构的联系电话。这确保你知道如何联系他们，即使警察没收了你的手机。

### 援引你保持沉默的权利，并要求与你的律师交谈

如果你发现自己在抗议中被逮捕，请行使你保持沉默的权利。向当局提供你的姓名和地址是可以的，但不要回答任何其他问题，并立即要求与你的律师交谈。

凯希安说，她经常告诉她的客户把她作为一个借口。她建议使用“我的律师说我不能和你说话，我需要给她打电话”这样的短语，这样可以转移你回答问题的压力。

### 不要同意搜查你的手机

警察可能会没收你的手机，但你不必同意搜查你的个人设备。没有搜查令，执法部门无法进入你的手机，但他们仍可能试图向你施压或恐吓你解锁。这就是为什么最好像触控 ID 或 Face ID 那样 [关闭生物识别解锁功能](https://lifehacker.com/the-phone-settings-you-need-to-know-before-protesting-1843829849) ，取而代之的是设置一个又长又复杂的密码。

凯希安警告说，警察仍然可以搜查你的口袋和包，所以不要带任何有价值的东西或任何你不想让警察发现的东西去抗议。

### 不要给 DNA 样本

在逮捕过程中，警察可能会试图通过擦拭你的脸颊来获取你的 DNA。不要同意提供 DNA 样本，即使警察让它看起来像一个标准的逮捕程序。

脸颊拭子是获取 DNA 的一种常见方式，但警方有其他方法你应该小心，包括向你提供水瓶、香烟或口香糖。最好不要接受警察的任何东西，尤其是在逮捕过程中。

### 捕捉事故并获取徽章号码

Kaishian 建议和一个朋友或一群朋友一起去参加抗议活动，并同意互相帮助。这包括在警察逮捕或虐待的情况下捕捉镜头。当局可能会声称拍摄逮捕过程是非法的，但除非你在身体上妨碍了一名警官，否则你有权站在一旁捕捉整个瞬间。

如果你被逮捕了，并且知道一个朋友或陌生人正在附近拍摄，大声讲述发生了什么也是有用的。这可以提供事件的背景，并可以帮助你的案件在任何诉讼下来。

你也应该试着得到任何参与逮捕或虐待的警察的名字和警徽号码。虽然这并不经常导致警官的反响，但 Kaishian 表示，这有助于建立一个不当行为的书面记录，对未来的案件有所帮助。

### 为被捕的现实做好准备

了解你的权利，遵循上述所有步骤，并采取一切预防措施，可能仍然不能保护你免受执法人员的虐待。在一个完美的世界里，每个人都知道自己的权利，当局每次都会完美地遵循程序。但这不是一个完美的世界，逮捕是混乱的，可怕的，有压力的。要知道，事情可能不会一帆风顺，即使你做的每件事都是对的。也就是说，了解并行使你的权利是你减轻风险的最好工具。
