# 凯特的多功能野餐酱可以简化你的春季野餐

> 原文：<https://thetakeout.com/recipe-kates-all-purpose-picnic-sauce-mayo-lime-juice-1833644410>

我的房子很小。我哥哥曾经告诉我，我住在一个小房子里，“但就像，不是一个*实际的* [小房子](https://en.wikipedia.org/wiki/Tiny_house_movement) ”谢了兄弟。但是它的内部面积不足，但它有一个相当大的后院，有足够的空间来建一个花园，养 T7 只鸡和烧烤聚会。在春季和夏季，我会冲动地给朋友发短信，让他们过来，通常是在我查看冰箱里的东西之前。这意味着我做了很多野餐和后院菜单的即兴创作，这导致了凯特野餐酱的诞生。



野餐酱最初是烤鱼玉米饼的奶油调味品，加入了芫荽叶和酸橙汁。虽然它绝对是墨西哥食物的美味佳肴，从鸡肉玉米饼到智利五香蔬菜到 esquites，但野餐酱很快证明了自己是各种后院美食的多功能伴侣。当客人已经在路上时，我在最后一刻做的另一道菜是某种沙拉或玉米沙拉，与我们正在烧烤的东西形成了脆脆的对比。野餐酱也适用于此:把它混合起来，加入切成薄片的卷心菜或玉米粒或切碎的西红柿，然后——奶油沙拉或配菜沙拉。

让野餐酱为你展现它的力量:将它添加到意大利面中制成奶油意大利面沙拉，或者添加到土豆中制成非传统的土豆沙拉。它为火鸡或烤鸡三明治做了一种辛辣的调味品。真的，这种野餐酱在很多场合下救了我的命，当朋友们发短信说，“嘿，我们带了另外三个人，10 分钟后到！”我不得不争夺更多的食物。它由简单的成分和简单的比例组成；甚至不要费心去测量东西，只是边走边品尝。基本内容如下，但根据你的喜好进行调整。哦，如果你找到酱的另一个应用，我等不及要听了。

* * *

## 凯特多功能野餐酱

*   一份酸奶油
*   两份蛋黄酱
*   少量喜欢的辣酱
*   一把切碎的芫荽叶
*   很少榨酸橙汁
*   盐和胡椒

将所有这些混合在一起。奉上好评如潮。拍拍自己的背。
