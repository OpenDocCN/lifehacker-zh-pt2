# 如果一家航空公司改变你的航班，你该要求什么

> 原文：<https://lifehacker.com/what-to-ask-for-if-an-airline-changes-your-flight-1839895475>

如果一家 [航空公司无意中把你](https://lifehacker.com/what-to-ask-for-when-youre-bumped-from-a-flight-1836448891) 赶下飞机，你可能会要求比你想象的更多的东西，比如餐券、酒店住宿或现金。但是如果一家航空公司突然决定完全改变你的航班起飞时间，你有什么权利呢？它发生了；在 Reddit 上， [u/kaileen](https://www.reddit.com/r/airlines/comments/9xm5y4/frontier_airlines_changed_a_flight_by_more_than/) 分享了他们的家人在航空公司最后一刻改变行程的经历。

“我哥哥在 8 月份从边疆航空公司买了一张机票，”他们写道。“原定的航班是本周六早上 5:45。他刚刚收到一封电子邮件，内容是他周日早上 6:00 出发的新行程...如果这不是一个错误，有什么可以做得到折扣或某种形式的不便赔偿？我知道 Frontier 并不以适合飞行而闻名，但这似乎太荒谬了。”

这很荒谬——但不要指望任何补偿。一般来说，在最好的情况下，你可以要求免费重新预订另一个航班，或者得到退款，至少你可以用退款重新预订另一家航空公司的航班。

以下是美国几家主要航空公司的一些政策，你一定要仔细看看，这样你就能更好地了解当你的出发时间发生重大变化时该怎么要求:

*   [【美国](https://blog.flydealfare.com/american-airlines-schedule-change-policy-make-journey-comfortable/) :如果你新的出发时间比你原来的时间少了 90 分钟**，他们会在相同的“票价区间”内为你提供同一天的另一个航班，也就是说一个经济舱座位换成另一个经济舱座位。如果您的新航班距离原定起飞时间超过了 **90** 分钟，那么您可以选择重新预订任何舱位、其他日期的任何 AA 航班，甚至是 AA 合作伙伴航空公司的航班。(正如 [指出的那样](https://thepointsguy.com/2017/03/when-an-airline-changes-itinerary/) 写道，AA 通常非常宽容，即使是很小的日程变动。)要申请退款，退款可能会以优惠券的形式出现，您的预定航班必须在您最初出发后超过 **60** 分钟。**
*   达美:达美的政策不是那么明确。根据 [达美航空](https://www.delta.com/us/en/legal/contract-of-carriage-dgr#22re) 的运输合同，如果您的预定出发时间超过您原定时间的 90 分钟，达美航空将“将乘客运送到达美航空下一个航班的目的地，该航班在最初购买的服务等级中有座位。”然而，据我与之交谈的一位客服代表称，乘客可以免费改签他们的航班“如果他们的出发或到达时间延迟了 **30** 分钟或更长时间。”退款也是一样。

*   [美联航](https://www.united.com/ual/en/us/fly/help/faq/schedule-changes.html#rebookingoptions) :“如果您的预定出发或到达时间改变了 **30** 分钟或更长时间，我们很乐意尝试找到其他满足您需求的航班选择。”(注意:这是唯一一家表示如果您的到达时间也发生变化，它会为您重新预订的航空公司。)对于退款，如果您的预定出发或到达时间改变了**两个小时**或者影响了您转机的能力，您可以申请退款。
*   边境 :如果你的出发时间改变了一个小时或者更少，你需要支付重新预订和机票差价的费用。如果您的起飞时间变更超过 **60** 分钟*和*您的航班变更**在其预定起飞的 72 小时**内，您可以免费重新预订；但是，如果你的航班在 72 小时后变更，你将不得不支付任何机票差价。超过三个小时，你也可以要求退款。

如果您想更改航班或要求退款，请了解影响您的政策，并立即联系您的航空公司要求更改。然而，值得注意的是，如果预定的变化影响了你中途停留的能力，航空公司可能会试图更好地适应你(尽管没有规定具体的时间限制)。

此外，如果你在不同的航空公司预订了两段旅程，可能会错过转机，要知道在这种情况下你的第二家航空公司并不欠你什么。在这种情况下，您可能想尽快查看我们的 [航空公司改签费指南](https://lifehacker.com/heres-what-youll-pay-to-change-your-flight-1836410208) 。
