# 如何将我的工作笔记本电脑连接到我的家用显示器？

今天，我要回答来自 Lifehacker 评论者 [**crs486**](https://lifehacker.com/1842430111) 的一个问题，它涉及到我最不喜欢的话题:让多个显示器很好地播放。

### 这个问题

“我无论如何也不能让第二台显示器工作。它是使用适配器连接的，因为我的机器没有连接。我也试过 HDMI，那种方法也不行。显示器连接到其他机器时工作正常。如果这台笔记本电脑是传统的坞站，会有什么不同吗？我没有使用扩展坞。”

### 难以置信的正确答案

嗯。这个问题还有一点要解释，所以我们开始吧。“有一个适配器”，我想你的意思是你正在做某种 USB-to-what 适配器，以强制在没有自己的 HDMI 端口或 DisplayPort 的笔记本电脑上建立第二个视频连接，或者 Thunderbolt，就此而言。

我很高兴你已经用其他机器测试了显示器，因为这排除了你的硬件的任何机械缺陷或其他讨厌的问题。我们可以缩小到适配器和连接本身，这就是我怀疑问题所在。

你没有具体说明，但我假设你的系统没有 USB-C 连接。如果是的话，这个*应该*很容易。一个 [简单的 USB-C 转 HDMI 适配器](https://smile.amazon.com/Adapter-QGeeM-Thunderbolt-Compatible-Pixelbook/dp/B07JW7GT7H/ref=sr_1_3?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-do-i-connect-my-work-laptop-to-my-home-monitor-1842471071&asc_source=&dchild=1&keywords=usb c to hdmi&qid=1585067637&s=electronics&sr=1-3&tag=kinjalifehackerlink-20) 就是你所需要的一切。在您的情况下，适配器本身可能有问题，这是您可能想要检查的东西，比如说，Amazon 或任何其他出售它的地方。如果很多人都报告了适配器的问题，这可能是一个垃圾设备。这当然会发生，你只需要一个新的适配器。(我用 [这个](https://smile.amazon.com/Qidoou-Multiport-Converter-Compatible-Chromebook/dp/B07KMC3DTL/ref=sr_1_10?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-do-i-connect-my-work-laptop-to-my-home-monitor-1842471071&asc_source=&dchild=1&keywords=usb c to hdmi&qid=1585067671&s=electronics&sr=1-10&tag=kinjalifehackerlink-20) ，对我来说一直很好用。)

如果你的笔记本电脑只有 USB 2.0，我几乎愿意打赌你的适配器有问题。要么是动力不足，要么是工作不正常，要么是和你的操作系统不兼容。所述适配器可能具有您需要安装的驱动程序，以便您的操作系统使用它，因此您可能需要检查制造商的网站。如果我是对的，简单的“USB 到 HDMI”电缆在这种情况下是行不通的——我假设你没有，因为适配器是关键组件。如果是这样，这就是你的问题所在。

你提到你的笔记本电脑有一个扩展坞。我会调查一下，因为它可能有一个内置的 HDMI 端口或其他显示连接，你可以用它来连接第二个显示器。(如果没有我会很惊讶，但是你永远不知道！)

除此之外，可能是硬件连接*正常工作，无论什么原因，你的操作系统无法识别你的第二台显示器。您可能需要点击一些键盘命令来启用多显示器设置(通常是 Function + whatever 键，该键的图标看起来像两个矩形显示器，中间用竖线隔开)。或者你可能需要直接在你的操作系统中设置一些东西。*

在 Windows 10 中，你应该可以在**设置应用>显示**中找到任何相关的设置。寻找“多显示器”部分，并确保您复制或扩展了您的显示器，而不是简单地在您的笔记本电脑上显示它( **Windows 键+ P** 也是检查这一点的有用快捷方式。)如果你使用的是 Windows 7 或 Windows 8，你需要右击桌面，选择“屏幕分辨率”来找到类似的设置。

在 macOS 中，只需点击左上角的苹果标志，然后**系统偏好**，然后**显示**。如果您的系统检测到第二台显示器，您将在此屏幕中找到配置设置的所有选项。如果您看不到任何东西，请尝试按住 Option 键并选择现在可见的“检测显示器”按钮。

如果你仍然没有找到任何成功，我仍然愿意打赌，这是你的适配器。你可以继续测试新的电缆和适配器，看看这是否可行，或者你可以弃船。在你做后一项工作之前，我至少会再尝试一次，但请尝试检查另一台笔记本电脑/显示器设置上的适配器——即使你不得不在离家工作/隔离期间将它放在朋友的邮箱中。

如果你的笔记本电脑最终是罪魁祸首，你可能不得不暂时转向单屏幕，尽管这很烦人。或者，如果你必须有额外的显示屏，也许你可以尝试使用第三方应用程序 来连接平板电脑作为第二屏幕。
