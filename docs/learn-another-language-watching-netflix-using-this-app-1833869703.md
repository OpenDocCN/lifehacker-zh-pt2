# 使用此应用程序观看网飞时学习另一种语言

> 原文：<https://lifehacker.com/learn-another-language-watching-netflix-using-this-app-1833869703>

学习一门新语言很难。学了八年法语后，我在这门语言上是一个糟糕透顶的失败者，直到我在这个国家花了一个月的时间经常使用它。看到在上下文中使用的单词和短语会比仅仅在教科书中阅读更容易理解。



[Lingvo](http://lingvo.tv/) 是一款让你使用网飞、YouTube 和亚马逊视频来帮助你更快学会一门语言的应用。

这项服务实际上是一个 Chrome 扩展和一个基于网络的智能手机应用 的组合。

有了它，你可以通过浏览器使用 Chrome 扩展观看另一种语言的电影，并带有该语言的字幕。

在你的手机上，同样的字幕会同时出现。当你听到或看到你不理解的东西时，你可以在你的智能手机上点击这个词来获得快速的定义，而不必停止电影。

如果你的语言水平很高，那么你可以全力以赴去看一部成人电影，但如果你刚刚起步，那么儿童节目可能是最适合观看的。

显然，你需要一些语言知识才能让这种语言非常有用，但如果你正在学习一种语言，需要一点帮助来更好地掌握它，那么这可能正是医生(或法语老师)所要求的。
