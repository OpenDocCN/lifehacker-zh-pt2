# 是时候试试杰斐逊硬拉了

> 原文：<https://lifehacker.com/its-time-to-try-a-jefferson-deadlift-1839987677>

到目前为止，在我们的怪异托举挑战中，我要求你做一个 [单手硬拉](https://vitals.lifehacker.com/lets-get-weird-and-try-a-one-handed-deadlift-1839717740) 和一个 [单手哑铃抓举](https://vitals.lifehacker.com/this-weeks-weird-lift-is-the-one-hand-dumbbell-snatch-1839893931?rev=1574372309060) 。现在有一个解脱:在本周的举重中，我会让你把双手放在杠铃上。我们在做杰斐逊硬拉。



这部电梯不是以我们的第三任总统命名的，而是以一个名叫查尔斯·杰弗逊 T3的马戏团强人命名的。这是一种不对称的硬拉，你必须使用你的核心肌肉来防止杠铃在你举起时旋转。

好吧，你应该这么做。这个需要杠铃。它在地上，你抓住它，然后站起来。唯一的问题是你必须一只脚站在横杆的两边。

 [https://lifehacker.com/embed/inset/iframe?id=instagram-B5JTXaShdfc&autosize=1](https://lifehacker.com/embed/inset/iframe?id=instagram-B5JTXaShdfc&autosize=1) 

这意味着你要么面对杠铃的一端，一只手在前面，一只手在后面；或者，你像平时做硬拉一样用手，但是一只脚走在杠铃前面，而不是两只脚都在杠铃后面。两种方式都可以，或者你可以找到两者的混合体。这是我在健身房举重，体重 135 磅(下图)，而 [在杠铃](https://www.instagram.com/p/B3SsUoAhTJz/) 上举重 100 公斤(220 磅)。

可能需要一些实验来找出最适合你的身体姿势。有些人采取宽大的姿态；我发现双脚并拢更成功。我喜欢像做传统的硬拉一样准备，然后我一只脚跨过横杆。然后我把我的后脚向外倾斜一点，当我弯腰捡起它的时候，试着保持身体居中。如果我的右脚向前，我会左手握拍，右手握拍。当杠铃升高时，如果需要，我会移动臀部，这样杠铃就可以一直向上，而不会撞到我的腿。

我的意思是，是的，你把标准提高到了胯部。看起来真的，真的很傻。你可能会认为这里的人可能有解剖学上的问题，但我从未听到有人抱怨过。相反，你会听到人们在互联网上对这种奇怪的电梯大加赞赏。(我是说， [看看这个家伙](http://usawa.com/jefferson-lift-origin-techniques/) 。还有 [这个](https://www.t-nation.com/training/how-to-do-the-jefferson-deadlift) 。还有 [这个](https://fitnessvolt.com/31354/jefferson-deadlift/) 。)

这就像有一条不成文的规则，我们不会取笑这个电梯，即使它是愚蠢的。所以我们在这里。你要非常认真地对待这次托举，好吗？试一试，让我们知道进展如何。
