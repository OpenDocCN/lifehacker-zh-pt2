# 通过多林哥的课程学习《权力的游戏》中的高级瓦雷利亚语

> 原文：<https://lifehacker.com/duolingos-high-valyrian-lessons-will-help-you-speak-lik-1834125892>

流行的语言学习应用 Duolingo 正在利用*权力的游戏*第八季的宣传，发布了其高瓦雷利亚语课程 的 [更新版本，使粉丝更容易在最短的时间内学习维斯特洛最古老的语言之一。](https://www.duolingo.com/course/hv/en/Learn-High%20Valyrian) 



High Valyrian 于 2017 年首次添加到 Duolingo，但新版本通过扩大词汇量，更新课程，甚至全语音录音和发音来充实课程。这门课程是由语言学家大卫·j·彼得森创建的，他也为电视节目《权力的游戏》开发了一门完整的语言。Peterson 还为 Duolingo 课程贡献了所有录制的语音剪辑，因此您可以确信一切都是正确发音的。

### 如何用多林果开始学习高缬草

1.  安装并打开 Duolingo app ( [安卓](https://play.google.com/store/apps/details?id=com.duolingo&hl=en&referrer=utm_source%3Dduolingo.com%26utm_medium%3Dduolingo_web%26utm_content%3Dtext_link%26utm_campaign%3Dfooter_site-map)[iOS](https://itunes.apple.com/app/duolingo-learn-spanish-french/id570060128?mt=8)[Windows](https://www.microsoft.com/en-us/p/duolingo-learn-languages-for-free/9wzdncrcv5xn?rtc=1&activetab=pivot:overviewtab))
2.  点击左上角的“课程”标志，然后点击“添加”
3.  向下滚动到列表底部，然后轻按“高瓦雷利亚语”
4.  课程介绍结束后，你将开始上第一课。在课程中轻按扬声器图标，以听到大声朗读的单词和短语。

该课程的新版本包括 38 个模块，每个模块包含各种小课程，涵盖各种语法和词汇主题。之前已经开始学习该课程的任何人都将自动收到新的课程材料。

像其他多林戈语课程一样，高瓦雷利亚语课程旨在快速完成，所以你只需要每天花几分钟来学习坦格利安语。一旦你掌握了高级瓦雷利亚语，你就可以学习多林戈语的 33 种其他语言中的一种——包括克林贡语和 T3，为那些想进一步增加他们幻想语言技能的人提供帮助。就我个人而言，我希望有一天我们会有一门 [辛达林](https://lotr.fandom.com/wiki/Sindarin) 课程。
