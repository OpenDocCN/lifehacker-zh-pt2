# 永远不要从空白页开始

> 原文：<https://lifehacker.com/never-start-with-a-blank-page-1839325924>

创意块结束了， [如果你想要的话](https://www.youtube.com/watch?v=CbKsgaXQy2k) 。如果你在一次头脑风暴或一个项目中没有想法，你只需要使用这个简单的系统:消费东西，做笔记，并随身携带这些笔记。以下是如何有效地做到这一点。

Watch

作家、演员和圣诞精灵大卫·塞达里斯说他的工作更多的是学习技能而不是特殊才能:“每个人都有自己的眼光。唯一不同的是，我在前面的口袋里放了一个笔记本。”正如写作老师大卫·佩雷尔解释的那样 ，塞达里斯记下了他生活中所有有趣的事情，每一个他想探索的杂念。他偶尔会坐下来，读他的笔记，并把好的复制到他的电脑上。

塞达里斯不需要坐在那里，试图思考事情的头脑风暴会议。当他写任何东西时，他已经有了所有这些想法，这更多的是选择写什么而不是写什么的问题。他可以开始重复他已经写下的东西。

读到这里，我意识到我最糟糕的工作是从一张白纸开始的，而我最好的工作是从我到处收集的一大串想法开始的。通过从各处获取想法，我发现了我感兴趣的东西，以及我想成为“临时专家”的东西。当我看着所有这些想法时，我开始看到我可以建立的联系，我可以把事情放在其中的结构——我想出了更多的话要说。

你不可能永远准备，最终 [研究变成拖延](https://nesslabs.com/start-now) 。不要试图收集整个项目所需的所有东西。只要收集你需要开始的东西。
