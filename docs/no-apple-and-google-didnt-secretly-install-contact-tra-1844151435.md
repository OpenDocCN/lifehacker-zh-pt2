# 不，苹果和谷歌没有偷偷在你的手机上安装联系人追踪应用

> 原文：<https://lifehacker.com/no-apple-and-google-didnt-secretly-install-contact-tra-1844151435>

最近的社交媒体帖子(我不会链接到这些帖子，但你可以通过 Twitter 搜索很容易地找到它们)声称苹果和谷歌在未经你允许的情况下在你的手机上安装联系人追踪应用程序。这些帖子甚至展示了令人信服的截图，并暗示两家公司的声明表明他们一直在开发技术来帮助追踪接触者；这一证据被用作超级军团在 T4 隐藏你手机上的应用程序并跟踪你去的任何地方的证据。



现在，看，这是真的，一些早期的接触者追踪应用程序执行了黑幕数据收集计划，在最近的抗议活动中，有人担心“接触者追踪逮捕”。B 但是关于后门追踪功能被偷偷安装在你的设备上的传言是不真实的。

让我直截了当地说:目前在你的 iPhone 或 Android 设备上没有隐藏的联系人追踪应用程序 。

虽然一些设备*已经*更新了与新冠肺炎暴露跟踪相关的专用设置，但所有增加的都是联系人跟踪应用*的能力，你可以安装*在系统级别与你的设备交互，以便他们可以发送警报和暴露通知。

《福布斯》网络安全记者戴夫·温德 亲自测试了这一点，我在我的 Pixel 3a XL 上也有类似的新冠肺炎主题设置。上述设置不仅在默认情况下是关闭的，而且谷歌引入的更新没有在我的设备上安装任何不需要的位置跟踪应用程序(即使我打开了设置，也不会安装任何东西；它只影响寻找那些权限的应用。

在这一点上，一个联系人追踪应用程序出现在你的手机上的唯一方式是你手动下载并安装它，并授予它追踪你的数据的权限。如果你不想安装，你可能永远也不会安装，不管怎样，因为手动 [联系追踪](https://vitals.lifehacker.com/what-is-contact-tracing-1842984055) 工作正常。

但是，即使你的设备上有新的联系人追踪功能，它也会使用你的手机和应用程序已经记录的 GPS 和应用程序活动信息。甚至有办法让 [关闭这样的追踪功能](https://lifehacker.com/psa-your-phone-logs-everywhere-you-go-heres-how-to-t-1486085759) ，删除像 [苹果](https://lifehacker.com/i-asked-apple-for-everything-it-knows-about-me-and-her-1831045096)[谷歌](https://lifehacker.com/how-to-automatically-delete-your-google-data-and-why-y-1839614022)[脸书](https://lifehacker.com/find-out-how-facebook-tracks-you-even-when-youre-not-us-1841306557) 和 [推特](https://lifehacker.com/how-to-stop-twitter-from-sharing-your-data-with-adverti-1842794674) 这样的公司理所当然地从你那里收集的数据。

我并不是说你*需要*去马上关闭所有这些功能；我只是指出，我们在一定程度上控制了哪些数据被跟踪，数据去了哪里以及谁可以访问这些数据——更不用说我们首先选择使用哪些应用程序或设备了。只要这些选项仍然可用，你就不必担心在上世纪最糟糕的全球疫情期间，你的手机是否在监视你。我不怪人们担心无处不在的数字监控的威胁，但是不要让这些担心变成相信虚假的谣言。
