# 三星 Galaxy S10 的指纹锁不再安全了

> 原文：<https://lifehacker.com/the-samsung-galaxy-s10s-fingerprint-lock-isnt-very-safe-1839133478>

最近 [发现](https://www.bbc.com/news/technology-50080586) 三星 Galaxy S10 和 S10+有一个重大安全漏洞，可以轻松绕过他们的指纹锁。在“1”到“不好”的范围内，我们在这一点上肯定是偏向右边的。

Watch

公平地说，指纹传感器和其他生物识别安全功能并不是无懈可击的；黑客可以成功绕过这些安全措施，尽管工作量相当大。然而，Galaxy S10 的指纹传感器可以通过简单地添加一个由硅胶、钢化玻璃或塑料制成的屏幕保护器或手机壳来欺骗。来自保护材料的干扰显然足以迷惑传感器或者任何人的手指点击都可以解锁手机。*(唉。)*

作为对投诉的回应，三星已经开始了内部调查，但在官方修复之前可能还需要一段时间——鉴于这个问题的广泛影响，希望不会。目前，该公司建议 Galaxy S10 和 S10+用户仅使用三星配件和手机壳，但该建议实际上只会有助于让 S10 的传感器更容易识别你自己的指纹。这实际上不会提高你手机的安全性。

最简单的解决方案是放弃使用 Galaxy S10or 或 S10+的保护配件，但这不会有太大作用，因为有人可以很容易地拿起你的手机，自己套上一个保护套来利用指纹漏洞。虽然切换到 [不同的锁屏方法](https://lifehacker.com/whats-the-best-way-to-unlock-your-android-phone-1828856042) 在这一点上可能是明智的，但这些也可以被足够精明的黑客绕过。要格外小心你的手机的下落和你让谁使用它的，并准备一个应急计划来保护你的数据，以防你的手机或身份被盗 。
