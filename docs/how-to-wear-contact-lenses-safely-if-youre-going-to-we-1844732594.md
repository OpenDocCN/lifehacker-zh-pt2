# 如何安全地佩戴隐形眼镜，如果你无论如何都要戴的话

> 原文：<https://lifehacker.com/how-to-wear-contact-lenses-safely-if-youre-going-to-we-1844732594>

早在 4 月份，我们就报道了美国眼科学会(AAO)的建议 ，隐形眼镜佩戴者暂时换回眼镜，原因是疫情。除了避免任何涉及将手指插入眼睛的事情，美国眼科协会还指出，隐形眼镜事故 [可能需要医疗护理](https://vitals.lifehacker.com/if-you-wear-contacts-consider-switching-to-glasses-for-1842912934) ，如果可能的话，我们应该尽可能远离医疗机构。



但是，正如《生活骇客》副主编乔丹·卡尔霍恩所说，“唯禁欲眼镜教育是行不通的——人们会戴隐形眼镜。”如果那是你的情况，而你*现在需要*戴隐形眼镜(人各有因！)，以下是如何做到尽可能安全。

### 你当然需要洗手

这真的不应该是什么新鲜事——尤其是在疫情期间——但是你绝对必须在接触隐形眼镜然后把它们戴到眼球上之前洗手。“重要的是要记住，尽管对冠状病毒有很多担忧，但常识性的预防措施可以显著降低你被感染的风险，”眼科医生兼美国眼科协会发言人 Sonal Tuli 博士在该组织网站 的一篇文章中说。"所以要多洗手，并遵循良好的隐形眼镜卫生习惯."

### 别忘了你的箱子

如果你的隐形眼镜盒外面覆盖着某种浴室和/或手指污垢，除非你打开盒子后再次洗手，否则这些粘性物质会进入你的眼睛。尽管看起来箱子的内部永远是干净的，但你需要做的不仅仅是把用过的接触水放入水槽中。

美国眼科协会建议用无菌隐形眼镜溶液 (不是自来水)冲洗你的盒子内部，然后让空盒子敞开晾干。密苏里大学圣路易斯视光学学院的退休教授兼验光师埃德·班尼特博士更进一步，他建议每周扔掉你的病例。

### 避免揉眼睛

像洗手一样，这是另一个现在适用于每个人的方法。虽然没有证据表明戴隐形眼镜会增加你患新冠肺炎病的风险，但是图利指出，戴隐形眼镜的人比普通人接触眼睛的次数更多，所以这一点要记住。
