# 通过这个网站了解你缴纳的每一分钱的去向

> 原文：<https://lifehacker.com/learn-where-every-penny-of-your-taxes-go-with-this-site-1830201711>

有没有想过你的税金到底去了哪里？虽然我对政府资助的项目有一个大致的概念，但我从来没有看一下我的钱具体是如何被州政府和联邦政府使用的。



[我的税去哪了？](https://www.wheredomytaxesgo.co/) 是一个为你分解信息的网站。随着时间的推移，你输入你的税前收入，国家，申报状态，然后它生成一份报告，你的现金去了哪里。

例如，如果我今年赚了 60，000 美元，并作为一个单身人士在加利福尼亚州申报，我的收入中有 2，219.07 美元将进入社会保险，40，21 美元进入农业部，1266.73 美元进入国防部。

这些数字是使用 2019 年拟议联邦预算设定的。它没有给出你可能要支付的确切金额，但它把它分解成比你以前可能见过的更详细的内容。

值得一试的是:输入收入时不能用逗号。我第一次尝试时这样做了，它将 60，000 美元注册为 60 美元。这在网站上有记录，但我第一次肯定忽略了这一点。

不管你对你所看到的感到高兴还是沮丧，请确保你本周出去投票，让别人听到你的声音。
