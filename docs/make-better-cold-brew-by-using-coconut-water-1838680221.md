# 用椰子汁做更好的冷泡茶

> 原文：<https://lifehacker.com/make-better-cold-brew-by-using-coconut-water-1838680221>

冷煮咖啡不是我喜欢的冷咖啡。虽然它确实有效，但就咖啡因而言，它的味道可能有点淡，有点刺鼻，让我想起一点点酒。它感觉没有完成，就像它需要一些东西来完善它，而椰子汁就是这样。

Watch

这是两件我不喜欢分开做的事情在一起工作得很好的例子。就其本身而言，椰子水在口中感觉有点粘稠，但当用于冷煮咖啡时，它增加了急需的口感和淡淡的椰子味。它非常令人愉快，而且——如果宣传是可信的话——比你通常喝的咖啡多一点水分。(而且，即使你把椰子水的价格考虑在内，它仍然比从你的咖啡店里买冷的 brew 便宜。)

要制作它，你所要做的就是将粗磨咖啡浸泡在椰子水中一整夜(至少 12 小时，但你可以达到 24 小时)。四分之一杯咖啡豆和一杯椰子水的比例产生了一种相当浓缩的酿造，你可以用一点普通的水或牛奶稀释它。加入甜炼乳可以得到真正美味的东西，或者加入*焦糖*甜炼乳 [椰奶](https://skillet.lifehacker.com/make-this-vegan-one-ingredient-caramel-sauce-in-your-i-1823718112) 得到带有萨摩亚饼干味道的含咖啡因饮料。
