# 用这个工具找到完美的学校或学院

> 原文:[https://life hacker . com/find-the-perfect-school-or-college-with this-tool-1838440365](https://lifehacker.com/find-the-perfect-school-or-college-with-this-tool-1838440365)

为你的孩子选择合适的学校是一个重大的决定。从第一个幼儿园班开始，直到他们进入大学，我们希望他们在一个学术健全的环境中，有高质量的老师。提供参加俱乐部、运动和其他活动的机会。一个多样化的、安全的、有良好设施的学校，为他们的大学生活或大学毕业后的生活做准备。

Watch

不过，这是一个很大的问题，尤其是如果你要搬到一个新的地方，或者想将这所高中与这条路上的另一所高中进行比较。或者你的高中生试图缩小他们的大学选择范围。但本周我偶然发现了一个网站，它似乎为我们做了很多工作:。

你可以使用 Niche 来比较 K-12 学校*和*学院，这使得它对于家长和高中生都具有独特的价值。你可以按州、市、公立和私立搜索 [最好的学校和地区](https://www.niche.com/k12/rankings/)——或者按最好的老师、最好的大学预科和最有活力的学生生活搜索。

你可以在许多类别中搜索 [最好的大学](https://www.niche.com/colleges/rankings/) :文理学院、社区学院、私立大学和“最物有所值”等等。或者你可以按专业、州或学术搜索顶级学校。

自从我们的儿子开始上 K-5 学校以来，这个网站对我和我丈夫一直在进行的一场讨论非常有帮助:我们希望他在初中和高中时去哪里？他的小学恰好与我们区的三所中学相连，所以他的朋友可能会分散开来。因为我们可能会在他接近中学的时候搬家，如果我们决定更喜欢一所学校而不是另一所，我们可以根据学校的界限来决定他去哪里。

这些学校都相当具有可比性，但利基在其分析中提供的信息可能实际上有助于我们朝着一个方向发展。它从一长串来源中获取数据，这些来源包括美国教育部、美国人口普查局、布鲁金斯研究所以及它自己的家长和学生调查。每个学校、地区或学院也有来自家长、学生和老师的评论。

当我这么做的时候，我决定去看看一所很有可能在我儿子未来的大学候选名单上结束的学校。至于学术、多样性、教授、价值、地点、学生生活*和*派对场景。很高兴知道。
