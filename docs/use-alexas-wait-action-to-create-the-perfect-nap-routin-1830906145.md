# 用 Alexa 的“等待”动作创建完美的午睡程序

> 原文：<https://lifehacker.com/use-alexas-wait-action-to-create-the-perfect-nap-routin-1830906145>

我是例行程序的忠实粉丝——也就是说，数字例行程序，因为它们是一种便捷的方式来启动一系列动作，而不必在每次你想做某事时手动设置一切。

想想我最喜欢的活动之一，“午睡”我不需要起床，关掉灯，放一些舒缓的音乐(或 [雷雨声](https://support.google.com/googlehome/answer/7364558?hl=en) )来屏蔽外界，也不需要打开房间的风扇，我只需在我的亚马逊 Echo 或谷歌主页上大喊一声命令，它就能为我做到这一切。哒哒。

多亏了 CNET 的 [Ry Crist](https://twitter.com/rycrist/status/1070411804097032195) 最近的一条推文，我了解到亚马逊现在在其例程中支持等待时间。换句话说，你可以在你的行动之间插入一大块时间，这让你在你设定的程序中变得更有创造力。这是午睡爱好者的梦想。

### 如何在你的 Alexa 程序中设置延迟动作

继续那个睡觉的例子，在你的 Echo 设备上设置一个新的午睡程序非常简单。在你的智能手机上打开 Alexa 应用程序(不是[)Alexa 的网页界面](https://alexa.amazon.com/spa/index.html?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/use-alexas-wait-action-to-create-the-perfect-nap-routin-1830906145&asc_source=&tag=kinjalifehackerlink-20) ，唉)，点击左上角的三行“汉堡”图标来打开侧边栏，点击例程，点击右上角的加号图标来创建新的东西。

建立一个例程是一个非常简单的过程:创建一个触发器——例如，一个语音命令或一个预定的时间——并按顺序添加你的回声设备应该执行的所有操作，比如当你想睡觉时关掉你卧室的灯。

Alexa 的新“等待”动作很棒，因为你现在可以给自己一个内置的计时器来午睡。例如，你可以创建一个程序，立即关掉房间里所有的灯，打开 sleepytime 播放列表几分钟，帮助你入睡。然后，你可以在你通常午睡的时间内添加一个“等待”动作——比如说， [三十分钟](https://lifehacker.com/for-the-best-benefits-take-an-ultra-brief-nap-5928593)——然后添加另一个动作，比如开始你最喜欢的起床歌曲，打开你的灯，或者让你的回声说，“嘿，醒醒。”如果你通常午睡固定的时间长度，或者想要坚持这个习惯，那么创建一个这样的例行程序要比每次你要临时睡觉时设置一个智能手机定时器容易得多。

至于其他哪些例程可能会从延迟中受益，我将把它留给您的创造力。我可能会把我的标准早间安排分开，这样我一醒来就能看到天气预报，但是 30 分钟后当我不那么昏昏沉沉的时候，我会收到早间新闻。如果只是套路支持 [耳语模式](https://offspring.lifehacker.com/new-parents-have-alexa-whisper-to-you-1829871715) ，虽然；我真的可以用一些延迟的 Alexa 短语吓坏我的室友。
