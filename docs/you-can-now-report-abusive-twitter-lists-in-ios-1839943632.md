# 你现在可以在 iOS 中报告辱骂性的 Twitter 列表

> 原文：<https://lifehacker.com/you-can-now-report-abusive-twitter-lists-in-ios-1839943632>

Twitter 的列表功能使得浏览该平台的噪音变得容易，但它也带来了一个主要的烦恼——列表充满了被滥用的可能性，尤其是当你发现自己是其他人“吸 Twitter 的人”列表中的持卡人时。

Watch

我们的意思是。用户可以将他们关注的帐户组织成列表，根据他们想要的任何子类别或主题创建更容易理解的提要。这可以让 Twitter 成为一种不那么令人难以抗拒的体验，尤其是对于那些关注大量涵盖各种主题的账户的人来说。

然而，Twitter 列表也被一些用户滥用，将这一功能变成一种类似监视的工具，对某些群体进行分类和监控。例如，在线仇恨组织的成员可以将边缘化人群的账户收集到列表中，并轻松地与他人分享这些账户，还会给这些列表起一些非常不好听的名字。

为了遏制列表作为骚扰手段的使用，Twitter 正在 [推出新的报告工具](https://help.twitter.com/en/safety-and-security/report-abusive-behavior) ，当用户被添加到有害或辱骂列表时，该工具将让用户进行标记。该功能现已在 iOS 上运行，并将很快应用于 Android 应用程序——尽管这两个应用程序版本的流程是相同的。

### 如何举报有害或滥用的 Twitter 列表

能够报告有害名单是一个明智的举动，但 Twitter 在 2017 年就停止通知用户他们已经被添加到名单中。你仍然可以查看自己被添加到了哪些列表，只需点击主边栏菜单中的**“列表”**选项，然后进入**“成员”**

1.  要报告辱骂列表，请在 iOS 中打开列表，然后选择**“...”**图标。
2.  点击**“报告”**，然后选择**“这是滥用或有害的。”**
3.  在所提供的字段中提供任何额外的详细信息或解释，然后提交报告。
4.  一旦收到报告，您将会收到通知，并且您将会收到来自 Twitter 的任何其他行动的更新。

希望这将有助于进一步遏制许多 Twitter 用户每天经历的猖獗滥用。尽管如此，正如我们在过去所看到的，要从根本上阻止骚扰，需要的不仅仅是报告虐待行为的新方法。
