# 如果你现在在家工作很糟糕也没关系

> 原文：<https://lifehacker.com/its-okay-if-you-suck-at-working-from-home-right-now-1842498164>



从学校停课到企业倒闭，到大规模裁员，再到担心生病，每过一个小时都会让我们担心一些新的事情。在这种疯狂的情况下，我们中的许多人也在与雇主打交道，他们希望我们从办公室工作平稳过渡到在家工作，并像一切正常一样继续工作。

我们不能。

“在家工作”和“疫情期间在家*工作”是不一样的现在发生的事情在现代美国历史上是前所未有的，没有充分的生产力绝对是百分之百没问题的。有压力是完全正常的。*

如果现在很难对任何事情心存感激，请记住，我们这些可以在家工作的人是幸运的。许多人失业了。越来越多的人不得不去工作——无论他们是在医院、杂货店工作，还是从事许多其他重要工作中的一项——在那里他们有生病的风险。

但这并不意味着这不难。美国经历如此规模的疫情已经有一个世纪了。在我们的有生之年，我们周围的世界从来没有这么长时间如此封闭，我们当然也从来没有在我们能去哪里、能做什么、能拥抱谁的问题上受到这么多限制。我们从来没有像现在这样如此担心我们所爱的人的健康和安全。

我们以前经历过国家紧急状态。我们一直担心我们无法控制的因素带来的危险。但我们从未面临过这种无处不在的生存威胁。指望人们继续工作，就像什么都没发生一样，这绝对是病态的。

在过去的几周里，我们听说艾萨克·牛顿在隔离瘟疫期间发展了卓越的理论。 [莎士比亚](https://www.theatlantic.com/culture/archive/2020/03/broadway-shutdown-could-be-good-theater-coronavirus/607993/) 当死亡人数迫使剧院关闭时，莎士比亚可能会转向十四行诗。

对牛顿和莎士比亚有好处。我们不是他们。首先，我们中的许多人不得不做饭、打扫卫生和照顾孩子，同时*还要*从事有偿工作，让有个栖身之所。我们没有充裕的时间来提出万有引力理论或写出一部文学巨著。

尽管我很想利用这些奇怪的环境来专注于我的激情项目，但我不能。首先，我没有时间。(见上。)如果你也不知道，那就意味着你和其他人一样生活在现实世界中。

在你因为效率不高而自责之前，让我们来看看这不是一个正常的在家办公环境的所有原因。

除非你没有额外的家庭责任，没有你需要担心的家庭成员的健康，也没有因经济衰退而增加的工作压力——而且你还拥有一种超人的能力，可以忽略不断传来的坏消息——那么你现在要处理很多额外的担忧和责任。

这是*不*正常。这不是一个典型的在家工作的设置。我们大多数人都在勉强度日，试图让收支平衡，同时试图让我们周围的人免受致命的无形威胁。

现在，这意味着在照顾孩子、在家上学、做饭、打扫卫生和控制你的焦虑之间周旋，同时纠结于我们需要采取什么措施来保持健康，哦，是的，我三十分钟后有个会议，但是我的儿子刚刚摔倒了，嘴唇裂开了，正在流血，哦，我的天哪，我刚想起来我明天要交一份报告，我必须找时间完成它，不是吗？太多了。

在这些已经加剧的日常担忧之下，还有一些引发了深深的、尖叫的恐惧:你可能会生病和死亡，留下依赖你的人；你可能在不知情的情况下感染了你的祖母或正在接受化疗的表兄。每次出门都充满危险。每一个微小的互动都带有不祥的含义。

这不是“在家工作，因为我们不必通勤，所以更有效率。”这是一种现代瘟疫。如果有人试图告诉你相反的情况，直截了当地告诉他们他们错了。同时，你也要提醒自己。
