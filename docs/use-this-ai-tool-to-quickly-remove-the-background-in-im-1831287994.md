# 使用这个人工智能工具，不用 Photoshop 就可以快速去除图像中的背景

> 原文：<https://lifehacker.com/use-this-ai-tool-to-quickly-remove-the-background-in-im-1831287994>

去除图像中的背景通常需要知道如何使用照片编辑软件，或者寻求其他人的帮助。现在有了另一种解决方案:使用 AI。

Watch

[Remove.bg](https://www.remove.bg/) 是一个网站，其唯一的目的是从图像中移除背景。要使用它，你只需上传一张有人物的图片，该网站就会自动计算出人物在图片中的位置，然后移除背景，给你留下一个可下载的带有透明背景的主题 PNG。

正如你想象的那样，你的里程可能会有所不同。最初发现工具 的 The Verge 、 [指出，它对背景模糊的图像有困难。](https://www.theverge.com/2018/12/19/18148020/remove-bg-ai-background-foreground-without-photoshop) 

也就是说，如果你有一张前景清晰的图片，那么编辑照片就容易多了。
