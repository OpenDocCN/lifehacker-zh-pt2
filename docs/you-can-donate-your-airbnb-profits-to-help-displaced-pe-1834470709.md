# 你可以捐出你的 Airbnb 利润，帮助流离失所的人找到住房

> 原文：<https://lifehacker.com/you-can-donate-your-airbnb-profits-to-help-displaced-pe-1834470709>

想要回馈的 Airbnb 主机现在可以捐出一部分——或全部——利润来帮助受灾难、冲突或重大疾病影响的人们。



该公司宣布了一项名为“”的新举措，主办方可以从每笔支出中自动扣除一定比例(从 1%到 100%)来帮助有需要的人找到临时住房。目前，Airbnb 正在与以下组织合作:

*   [**国际救援委员会**](https://www.rescue.org/) ，帮助难民在移居美国后，以支持、住房和教育重建生活。
*   [**万众一心**](https://www.allhandsandhearts.org/) ，一个援助受自然灾害影响的社区的救灾组织。
*   [**许愿基金会**](https://wish.org/) ，为患有严重疾病的儿童实现愿望(包括临时旅行和住宿)。

“”是 Airbnb 的 [开放家园计划](https://www.airbnb.com/openhomes) 的一个分支，在该计划中，东道主可以向被疏散者、救援人员和难民捐赠使用他们的房屋。但是对于那些负担不起免费招待但仍想提供帮助的人来说，捐赠计划可能更合适。

所有捐款都可以免税，并将直接捐给非营利组织(Airbnb 将支付任何处理费用)，这些钱将有助于提供临时住房或医疗需求的差旅费用。
