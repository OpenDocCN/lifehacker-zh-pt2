# 如何在人们讲话时阻止群组 FaceTime 窗口调整大小

> 原文:[https://life hacker . com/how-to-stop-group-face time-windows-from-resizing-when-p-1843572742](https://lifehacker.com/how-to-stop-group-facetime-windows-from-resizing-when-p-1843572742)

如果你最近进行过群组互动，那么你可能已经体验过让参与者的窗口根据谁在说话而改变的功能。这个想法类似于 Zoom 在默认情况下的做法，因为它会让说话者成为关注的焦点，但如果你正在与多人进行激烈的对话，它也会很快变得令人讨厌。

Watch

随着昨天 iOS 13.5 的发布，苹果还增加了关闭群组 FaceTime 通话功能的能力，该功能可以自动改变这些窗口的大小。

要修复它:

*   转到设置。
*   选择 FaceTime。
*   向下滚动到“自动突出”部分。
*   关闭“说话”功能。

一旦您这样做，您的群组 FaceTime 通话中的所有窗口将保持在对话期间的位置和大小。它们仍将层叠在彼此之上，而不是平铺开来。

如果有人在讲话，而你想关注他们，你可以在通话过程中双击他们的窗口，让窗口变大。您还可以拖动窗口，使单个扬声器在您的个人屏幕上更加突出。

你需要 [在你的设备](https://lifehacker.com/how-to-unlock-your-iphone-even-faster-while-wearing-a-f-1843162575) 上安装 iOS 13.5 才能看到你设置中的选项。

随着 FaceTime 的更新，iOS 13.5 还让你戴着口罩时解锁 iPhone 变得更容易，并增加了对苹果和谷歌通知 API 的支持，未来可能会用于跟踪新冠肺炎的暴露情况。
