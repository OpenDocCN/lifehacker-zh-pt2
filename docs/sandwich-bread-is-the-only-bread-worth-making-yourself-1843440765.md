# 三明治面包是唯一值得你自己做的面包

> 原文：<https://lifehacker.com/sandwich-bread-is-the-only-bread-worth-making-yourself-1843440765>

在我所在的地方，现在获得新鲜出炉的面包和长棍面包比找到面粉更容易，所以我买长棍面包和面包。我做的法棍面包和 boules 永远不会比我花三四块钱就能买到的好吃，我只是不太享受烘焙的过程。然而，三明治面包是个例外。

三明治面包不仅是最简单、最值得做的面包之一，也是普通面包师新手能做的为数不多的味道比他们能买到的更好的面包之一。即使是我最喜欢的切片袋装三明治面包品牌，也总是有点干——吃起来就像吃一块廉价的薄海绵——而自制的三明治面包则柔软蓬松。(如果你当地的面包店做了很好的白色三明治面包，那就一定要买；我的没有。)

就面包而言，自制三明治面包也相当快。除非你用酸面团发酵剂——这会增加一天的时间——三明治面包可以在一个下午做出来。它在一个碗里混合，在一个简单的面包盘中烘烤，不需要立式搅拌机。

这也是一种实用的多功能面包。它(显然)是三明治和法式吐司(以及其他国家起源的吐司)的完美之选，但它的自制性意味着它也适合更高级的应用。把它烤熟，配上美味的奶酪，把它撕成大块，浸在辛辣的橄榄油里，或者涂上黄油和烤大蒜，做一个极其潮湿的大蒜面包。

就食谱而言，我通常使用 Peg 的这个 [阿米什白面包食谱，这是我用 AP 面粉做的，效果很好；它也可以被切成两半，做成一条面包。如果你没有酵母但有发酵粉，亚瑟王面粉有一个很好的](https://www.allrecipes.com/recipe/6788/amish-white-bread/) [发酵粉三明治面包配方](https://www.kingarthurflour.com/recipes/sourdough-sandwich-bread-recipe) 。如果你想体验白色三明治面包的凯迪拉克，投资一点乳制品，做 [日式牛奶面包](https://food52.com/recipes/30962-hokkaido-milk-bread) 。

最后，还有精神上的回报:做三明治面包会让你觉得自己是一个有成就的面包师，因为——不像你第一次做法棍面包——你的第一条白色三明治面包会看起来像它应该看起来的样子。你会立即取得胜利。成功了！你厨师帽子上的一根羽毛。自制三明治面包真的值得你花费时间和精力。它奖励你一个更好的产品，现在感觉更好不是很好吗？
