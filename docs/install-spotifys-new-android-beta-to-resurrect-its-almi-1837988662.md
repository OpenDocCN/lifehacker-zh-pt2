# 安装 Spotify 新的 Android 测试版，复活其全能的小工具

> 原文：<https://lifehacker.com/install-spotifys-new-android-beta-to-resurrect-its-almi-1837988662>

谁会想到一个小小的插件——更确切地说，是删除它——会激怒这么多 Android Spotify 粉丝？这正是该公司最近决定杀死其 Android widget 后发生的事情，这里有一个好消息:它回来了，你现在就可以下载带有恢复 widget *的 Spotify 版本*。



有什么问题吗？你需要获得一个 Spotify 测试版，这意味着你必须注册*进行测试，并处理潜在的错误软件。如果你想提前使用这个小工具，这就是你要付出的代价，最终所有 Spotify Android 用户都会得到回报(感谢上帝)。*

如果这个小工具不足以让你试用 Spotify 的测试版——它应该是——如果你目前运行的是 Android 10，还有一个功能值得探索。正如 [Android Police](https://www.androidpolice.com/2019/09/09/spotify-seekable-audio-progress-bar/) 报道的那样，Spotify 应用的测试版还允许你通过 Android 通知栏中的进度条在你正在听的歌曲中前进和后退。

要开始使用 Spotify 测试版，它可以提前解锁这两个功能:

1.  点击 [此链接](https://groups.google.com/forum/#!forum/spotify-android-beta/join) 加入谷歌 Spotify 测试组。
2.  一旦你成为谷歌群组的一员，点击 [这个链接](https://play.google.com/apps/testing/com.spotify.music) 。
3.  点击蓝色的大“成为测试者”框

最后，确保您已经在 Google Play 中将应用程序更新到最新版本。然后你*应该*拥有授权你访问小部件的版本，以及更新的进度条。
