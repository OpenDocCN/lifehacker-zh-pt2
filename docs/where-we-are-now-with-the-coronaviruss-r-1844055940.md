# 追踪你所在州的冠状病毒 R₀

> 原文：<https://lifehacker.com/where-we-are-now-with-the-coronaviruss-r-1844055940>

还记得冠状病毒还是全新的时候，我们都担心它的 R <sub>0</sub> 可能会变成 T5T1】吗？R <sub>0</sub> ，读作“R naught”，是一个衡量每个感染者倾向于传染给多少人的指标。为了阻止疫情，我们需要把这个数字降到小于 1。一个名为 [Rt.live](https://rt.live/) 的工具根据最近的数据显示了每个州当前的生育数量。

从技术上讲，R <sub>0</sub> 指的是“基本”繁殖数，也就是说，在一个假设的空白人口中，有多少人会被感染。真实世界的数字称为 R <sub>e</sub> 为有效再现数，或 R <sub>t</sub> 为给定时间的再现数。

想一个连锁信，或者多层次的营销方案。如果你和我两人一根绳子，他们两人一根绳子，很快我们的计划就会在世界范围内传播，呈指数增长。但是如果普通人决定，不，我不会把这个传递下去，这个计划——或者疫情——将会失败。

这些场景之间的分界线是 R <sub>t</sub> 为 1。如果每个人只感染另外一个人，疫情不会加速，但也不会消失。R <sub>t</sub> 为 1 意味着我们已经使曲线变平了。这是一个重要的里程碑，但不是胜利。

在 [Rt.live](https://rt.live/) 时，我们可以看到哪些状态的 R <sub>t</sub> 低于 1，而哪些状态的 R 仍高于 1。到目前为止，新泽西州做得最好，R <sub>t</sub> 约为 0.8。亚利桑那州表现最差，R <sub>t</sub> 为 1.11。

老实说，仍然令人担忧的是(对我来说)这些数字都很相似，而且相对较高。一旦传播速度开始减慢，州长们就开始开放各州。为 1 的平坦 R <sub>t</sub> 真的不够好。

但这确实表明封锁和面具以及其他一切都在起作用。 [估计冠状病毒的 R<sub>0</sub>](https://lifehacker.com/what-is-the-coronaviruss-r0-and-why-does-it-matter-1841264885)T5】把它放在 2.0 到 2.5 左右。通过将 R <sub>t</sub> 降低到 1.0 或更低，我们已经消除了一批新的感染，并大大减缓了疫情的进程。这是一个开始。
