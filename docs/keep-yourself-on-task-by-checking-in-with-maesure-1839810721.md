# 通过有限度的检查来保持自己的工作状态

> 原文:[https://life hacker . com/keep-yourself-on-check-in-with-Mae sure-1839810721](https://lifehacker.com/keep-yourself-on-task-by-checking-in-with-maesure-1839810721)

[maesture](https://maesure.com/)是一款免费的、基于网络的微型时间追踪器。每十分钟，Maesure 问你在做什么，你告诉它。这是像 [RescueTime](https://www.rescuetime.com/) 这样复杂的时间追踪器的一个简单的替代品。但我喜欢它作为一种保持专注的方式。

Watch

与更多功能齐全的时间追踪器不同，Maesure 不会主动观察你打开了哪些应用和网站。它唯一的功能是时不时弹出一个浏览器通知，询问你在做什么。但有时，我发现，这就是我保持正轨所需要的。在我屏幕的肩膀上轻轻一拍。

您可以自定义签入的时间间隔。如果您的答案与上一个相同，您只需在通知上单击“是”即可。

当 Maesure 问你在做什么时，你只需要诚实。如果 Maesure 发现你心不在焉，那就承认这一点，然后回到你预定的任务中去。如果你在做一件“非生产性”但令人满足或放松的事情，你不必内疚——只要诚实就好。Maesure 会累计你在各种任务上花了多少时间，这样你就可以知道你是否花了太多时间聊天、看 YouTube 或调整字体。

如果您创建了一个帐户，您可以跟踪过去的日子，并从不同的设备进行检查。您还可以将任务分类到文件夹中。

除此之外，没什么花哨的。没有图表，没有分析。你的时间追踪器不会让你分心。你可以通过关闭标签来关闭它。在 Windows 上，你可以 [安装 Maesure 作为 app](https://maesure.com/apps) 。
