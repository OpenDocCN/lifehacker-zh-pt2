# 如何使用 Google Home 挑选万圣节服装

> 原文：<https://lifehacker.com/how-to-use-google-home-to-pick-a-halloween-costume-1830054821>

如果你还在想万圣节要穿什么，谷歌可以帮你。Google Assistant 目前已经加载了一个测试，来帮助你确定最适合你的万圣节服装。要访问它，问一个谷歌助理设备“万圣节我应该是什么？”然后你将参加一个小测验来帮助缩小你的选择范围。



如果你已经有了一个万圣节计划，谷歌主页上还有一些其他万圣节特有的功能。

具体来说，你可以问谷歌关于怪物的事实，或者鬼魂或狼人的声音是什么样的。

谷歌还可以提供最近的南瓜补丁的方向，如果你还没有得到一个南瓜，谷歌 Home Hub 可以播放视频，解释如何为吸血鬼等流行服装进行面部彩绘。

如果你只是在寻找一个有趣的万圣节相关活动，Google Home 也有一个自己选择冒险的寻宝游戏，你可以进入谷歌助手的虚拟鬼屋，扮演寻找糖果的不给糖就捣蛋的人。

当 Google Home 发现储藏室里的糖果不见了，它会寻求用户的帮助来调查哪位客人(木乃伊先生、德古拉伯爵或友好的女巫威洛)应该对丢失的糖果负责。你可以说“嘿，谷歌，不给糖就捣蛋”来激活它
