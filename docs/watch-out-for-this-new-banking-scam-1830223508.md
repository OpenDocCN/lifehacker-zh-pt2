# 小心这种新的银行诈骗

> 原文：<https://lifehacker.com/watch-out-for-this-new-banking-scam-1830223508>

收到银行发来的短信，告诉你你的账户被锁定了？在遵循它的指示之前，你可能要三思而行。



据安全 上的 [Krebs 称，这是因为诈骗者正在使用伪造文本和无卡自动取款机的组合，从不知情的人那里窃取数千美元。这个骗局让他们在相当短的时间内大赚了一笔:根据 Krebs 的说法，骗子在不到两周的时间里从伊利诺伊州、密歇根州和俄亥俄州的大约 125 名第五三银行客户那里窃取了 68，000 美元。](https://krebsonsecurity.com/2018/11/sms-phishing-cardless-atm-profit/)

工作原理是这样的:诈骗者给银行客户发短信，声称他们的银行账户已经被锁定。这些短信包括一个链接，让客户进入一个类似银行网站的地方，在那里他们输入自己的账户凭证，“包括用户名、密码、一次性密码和个人识别码，”Krebs 报道说。一旦骗子掌握了这些信息，他们就可以在无卡自动取款机上利用这些信息来窃取顾客的钱。

无卡自动取款机让顾客用他们的智能手机取现金，而不是依赖借记卡。它们还不是非常普遍，但被大多数大银行使用，如 [大通](https://www.chase.com/digital/atms/cardless-nfc) 以及像 Fifth Third 这样的地区性银行，以方便客户。据报道，它们比标准的自动取款机更安全——或者至少金融业是这么说的——但是，正如所有新的金融创新一样，骗子们已经找到了利用它们的方法。如第五个例子所示，在某些情况下，骗子只需要用户名、密码和个人识别码就能盗取现金。

正如克雷布斯所建议的，永远不要“回复通过电子邮件、短信或电话发送的个人或财务信息。”如果你收到令人困惑的信息或电话，挂断或忽略它，并直接联系你的银行或金融机构。
