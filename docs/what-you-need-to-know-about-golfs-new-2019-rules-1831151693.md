# 关于高尔夫 2019 年的新规则，你需要知道什么

> 原文：<https://lifehacker.com/what-you-need-to-know-about-golfs-new-2019-rules-1831151693>

2019 年将带来许多变化，包括高尔夫运动的一些重大变化。

美国高尔夫协会和圣安德鲁斯皇家和古代俱乐部，以及 PGA 巡回赛和其他人的意见，为这项运动制定了新的规则，将于明年实施，所有的目标都是使这项运动对玩家更友好，，更容易理解，动作更快， [报道*《环球邮报》&*](https://www.theglobeandmail.com/sports/golf/article-golf-introducing-new-rules-for-2019-to-make-the-game-more-player/?cmpid=rss&utm_source=dlvr.it&utm_medium=twitter)。

至于具体的变化，这些变化取决于你是娱乐性还是职业性地玩游戏。以下是一些高层次的亮点。你也可以在 R & A 的网站 上获得 [所有规则的概要。](https://www.rules.golf/) 

### 丢弃程序

从 1 月 1 日开始，在休息时，高尔夫球员将从膝盖高度而不是肩膀高度放下球。

### 双重打击

击球两次不再会被罚分。即使技术上需要两次击球，只有一次击球会计入高尔夫球手的总得分。

### 出界

对于休闲球员，高尔夫俱乐部可以制定一个当地规则，球员可以在球出界的附近扔一个新球，并被罚两杆。

### 偶然运动

现在，除非“几乎可以肯定”球被移动了，否则导致球在果岭上移动或当你试图找到球时，不会受到处罚。

### 国旗别针

现在，如果高尔夫球手在洞内击打无人看管的旗杆，将不会受到处罚，并且在推杆时不再需要看管或移除旗杆。

### 修复钉痕

高尔夫球手现在可以修复果岭上的钉痕和其他损坏。

### 找一个球

如果你像我一样，经常丢球，现在你就没有时间去找球了。传统的 5 分钟高尔夫球搜索时间现在已经下降到 3 分钟。
