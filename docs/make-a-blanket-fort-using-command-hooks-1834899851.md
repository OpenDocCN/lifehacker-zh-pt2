# 使用指令钩制作毯子城堡

> 原文：<https://lifehacker.com/make-a-blanket-fort-using-command-hooks-1834899851>

我 6 岁的女儿第一次尝试做一个毯子城堡时，她有着孩子般快乐的雄心。不幸的是，她没有工程师的结构专业知识，因为整个建筑在三分钟内倒塌，她哭了。

Watch

大多数毯子城堡的缺陷在于地基:用来支撑毯子的家具。我的孩子使用厨房凳子，这是一个很好的高度，但很容易在任何匆忙的动作中倒下(例如，一个孩子站起来跑向浴室)。我决定做一些简单的研究来建造一个更好的毯子城堡，一个可以享受几个小时甚至几天的城堡。给我印象最深的方法？使用命令挂钩。

指挥钩让您可以将堡垒固定在墙上或天花板上，这意味着您不必担心您的孩子被倾翻椅、落地灯或花盆压碎。他们也给你灵活性，让你的堡垒变大或变小。当然，挂钩是可拆卸的，你可以在家里的不同区域尝试不同的结构，决定你理想的结构。(房间角落特别适合打造舒适的角落。)

要做一个简单的堡垒，你需要:

*   命令挂钩。我用了一个 [小丝钩](https://www.command.com/3M/en_US/command/products/~/Command-Small-Wire-Hooks/?N=5924736+3294529207+3294772789&preselect=5584742+3293786499&rt=rud) ，但是你用的钩数取决于你希望你的堡垒是什么样子的。
*   字符串。我用了面包师的麻线，但钓鱼线或任何结实的绳子都可以。
*   轻薄的床单和毯子。薄纱襁褓毯是夏季堡垒的绝佳选择。
*   晒衣夹

1)将指挥钩挂在墙上或天花板上。

2)将绳子的一端系在指挥钩上，另一端系在另一个钩上或安全底座上。(我把另一端绑在楼梯栏杆上。)确保绳子绷紧。

3)将毯子盖在绳子上。用晒衣夹将布料固定住。

4)将毯子的两端固定在地毯上，将它们粘在沙发垫上，以此来固定城堡的底部。

5)通过添加更多的毯子和衣夹来制作一个通往城堡的门。

这个周末，我和我的孩子们在我们的堡垒里呆了几个小时，看书，打美发沙龙。我可能会再坚持一段时间——当我需要片刻的禅坐时，我自己会退回去。嘿，这里很安静。
