# Shift 键会将您的键盘快捷键反转

> 原文:[https://life hacker . com/the-shift-key-kicks-your-keyboard-shortcut s-into-reverse-1839939967](https://lifehacker.com/the-shift-key-kicks-your-keyboard-shortcuts-into-revers-1839939967)

在我 [在 Reddit](https://old.reddit.com/r/LifeProTips/comments/dy257y/lpt_if_you_accidentally_press_the_spacebar_and/f7xw6ap/) 上看到它之前，这个简单的东西对我来说从来没有点击过:在 Windows 和 MacOS 的许多键盘快捷键中，添加 *shift* 键会使快捷键做与它通常做的相反的事情。

Watch

几个例子(在 Mac 上，用 cmd 替换 ctrl 或 alt):

*   ctrl-z =撤销；shift-ctrl-z =重做
*   space =向下翻页；shift-space =向上翻页
*   ctrl-tab(Mac 上相同)= next tabshift-ctrl-tab =最后一个制表符

*   alt-tab =下一个 appshift-alt-tab =最后一个应用程序
*   ctrl-t =打开新标签页；shift-ctrl-t =重新打开上次关闭的选项卡

你会注意到，几乎在任何移动的快捷方式上，你都可以添加 *shift* 来移动到相反的方向。

有时 shift 会给动作带来不同的变化:

*   ctrl-n =新建文件/文档/项目；shift-ctrl-n =新建文件夹/智能项目
*   ctrl-v =粘贴；shift-ctrl-v =剥离格式并粘贴(非常适合在文档和电子邮件之间移动文本)
*   ctrl-r =刷新；shift-ctrl-r =刷新*真的很难*

因此，将 *shift* 添加到你最喜欢的键盘快捷键中，穿着高跟鞋向后做所有事情。
