# 紧急疏散时要带什么

> 原文：<https://lifehacker.com/what-to-pack-during-an-emergency-evacuation-1839418212>

周末，加州北部的毁灭性火灾迫使大约 20 万人撤离——据《华盛顿邮报》报道，这是索诺玛县历史上最大的一次撤离。与此同时，就在南方几百英里处， [盖蒂大火](https://losangeles.cbslocal.com/2019/10/28/getty-fire-mandatory-evacuations-ordered/) 已经吞噬了洛杉矶地区至少 500 英亩的土地，并迫使数千人撤离。

虽然大火继续迫使人们离开家园，但在 Twitter 上，洛杉矶市应急管理部门发布了易于记忆的 evacuation⁠—which 打包提示，可以应用于你可能遇到的任何紧急情况，如飓风或停电期间离开你的家。为了你的利益，现在就把这个列表打印出来(或者写下来放在某个安全的地方)，这样你就可以急着查阅，而不用惊慌失措地去找 wifi 或者电再去找。如果你陷入困境，就遵从这些建议——或者他们所说的“六个 P”。

显然，你应该根据你的特殊需要和紧急情况来定制这个列表。如果是停电，没有照片和其他大事记你大概也能活下来。(一定要查看我们的 [停电备考指南](https://lifehacker.com/how-to-prepare-for-a-power-outage-1836610333) ，具体来说。)

另一方面，如果你可能一次离开家几天，或者紧急情况有更严重的后果，如果你有足够的时间，考虑在撤离前打包一个" [旅行袋](https://lifehacker.com/how-to-pack-an-evacuation-go-bag-1819669582) 。除了上述物品，你还应该带上食物和水。

具体到火灾，联邦应急管理局也在推特上提供了类似的建议。他们建议卷起车窗，关闭通风口，以避免吸入烟雾，并为宠物打包任何必需品，包括食物和水碗。

别忘了注册你所在地区的本地警报系统。许多地方，包括 [洛杉机](https://www.lacounty.gov/emergency/alert-la/) 和 [索诺马县、](https://socoemergency.org/home/emergency/stay-informed/socoalert/) 都设置了紧急警报，这样你就可以收到更新的短信或电子邮件；在网上搜索一个离你最近的，现在就注册吧。
