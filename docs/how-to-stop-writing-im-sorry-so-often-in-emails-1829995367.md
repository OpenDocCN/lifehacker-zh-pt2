# 如何停止在电子邮件中频繁写“对不起”

> 原文：<https://lifehacker.com/how-to-stop-writing-im-sorry-so-often-in-emails-1829995367>

上周，我发了一封电子邮件，其中写道“对此我很抱歉！”这是我的自动回复——我忘了什么东西，这个人简短的信息表明他们对我不太满意。但是经过一分钟的考虑，我说出了“对不起”,并告诉他们我会尽快按照他们的要求去做。我做到了，任务完成了，每个人都很好。没有伤害，就没有犯规。



几天后，我仍然在想那封邮件，想知道我的回复是否粗鲁。要明确的是，粗鲁不是我的本意；当我花额外的一分钟去思考我的回应时，我意识到我没有做任何值得道歉的事。但是作为一个女人(尤其是一个年轻的女人回应一个年长得多的人，理论上更有权威)，你习惯于扮演某些角色。

而自动道歉者是最常见的，也是最难破解的。女人被教导要被动，让每个人都舒服。我们的默认是道歉，即使真的没有什么可道歉的。我这么说是因为我已经不止一次向无生命物体道歉了。

写利亚费斯勒为 [石英](https://qz.com/work/1314825/theres-a-gmail-plug-in-to-make-you-stop-apologizing-so-much/) :

> 正如无数的研究证明的那样，女人比男人更频繁地道歉，主要是因为男人被社会化地认为他们没什么好道歉的。

> 这种习惯在网上无缝转换，其中“女性语言”体现在 [女性不停地使用感叹号](https://qz.com/938165/emoji-piece/) ，令人安心的表情符号，以及打出的“对不起！！!"性别语言学家苏珊·赫林称这种“模糊限制语”为数字交流中的 [情感劳动](https://qz.com/1030976/in-honor-of-national-emoji-day-heres-the-feminist-case-for-not-using-%F0%9F%91%8D/) 。

这是我在过去的一年里更加意识到的事情，并且已经采取措施去改变。扮演温顺的取悦者的角色非常 2017 艾丽西娅。这并不是说我再也不会道歉了——有些行为确实值得道歉。但是回复别人的邮件不够快，或者完成任务的方式和别人建议的不一样，都不会上升到那个水平。

当然，少说“对不起”的关键是意识到你正在这么做。在匆忙回应之前先打个拍子会有帮助。这就是为什么花额外的时间回复信息成为我在过去一年左右养成的最有用的习惯之一的原因之一。我写一个回复，坐下来或者简单地切换标签，然后再回来。

我喜欢的另一个技巧是:用“谢谢”代替“抱歉”，就像石英 建议的那样。而不是写“对不起！”例如，你可以写，“谢谢你指出这一点。” [Gmail 也有一个插件](https://qz.com/work/1314825/theres-a-gmail-plug-in-to-make-you-stop-apologizing-so-much/) ，它可以指出你用“对不起”、“我只是”和令人不快的被动句“我不是专家，但是”来贬低自己的例子
