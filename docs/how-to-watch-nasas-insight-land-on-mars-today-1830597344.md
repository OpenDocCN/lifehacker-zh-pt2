# 如何观看今天 NASA 的洞察号登陆火星

> 原文：<https://lifehacker.com/how-to-watch-nasas-insight-land-on-mars-today-1830597344>

美国国家航空航天局今天将在火星上着陆一艘宇宙飞船，你可以实时观看这一过程。

Watch

美国宇航局的火星内部探索使用地震调查，大地测量和热传输着陆器(是的，这是一个满嘴的)预计将在美国东部时间下午 3 点/太平洋时间下午 12 点左右在火星着陆。美国国家航空航天局简称它为 InS ight，你也应该这么做。

自 2012 年以来的首次火星着陆(美国宇航局将好奇号火星车降落在火星上)，这次着陆将开启洞察号为期两年的任务，其主要目标是研究火星的内部。希望通过研究这颗行星，科学家们能够更多地了解所有岩石世界是如何形成的，包括我们的地球。

如果你想亲眼目睹着陆(你确实想)，你可以在美国宇航局的 b 广播电视台在线观看这一事件。广播时间设定为东部时间下午 2-3:30/太平洋时间上午 11-12:30。你可以在 [这里](https://mars.nasa.gov/insight/timeline/landing/watch-online/) 找到直播。

如果你想和其他太空爱好者一起体验这一活动，美国各地以及一些国际地点也有很多观看派对，你可以和其他人一起观看。

值得注意的大选项是时代广场(在纳斯达克屏幕上)、史密森尼博物馆和洛杉矶中央图书馆。

你可以点击 查看 [观看派对选项的完整列表。](https://mars.nasa.gov/insight/timeline/landing/watch-in-person/)
