# 方便面和汤应该分开煮吗？

> 原文：<https://thetakeout.com/should-you-cook-instant-ramen-noodles-soup-separately-1834765171>

*欢迎来到*的外卖*的* [***亚洲面条周***](https://thetakeout.com/tag/asian-noodles-week) *，向 O.G .的面食碗致敬。*

* * *

我可以花几百字来构思这个故事，但是标题简洁明了。我过去常常在煮面条的同一壶水中加入调料包。几年后，我会分开做汤，认为这样会提高最终产品的质量。但是这真的有什么不同吗？我们的亚洲面条周是一些 A/B 测试的完美借口。



我买了两包札幌一品轩品牌的盐味拉面——不是最高档的面条(每包 1 美元),但也是一个不错的底线。对于我的第一批，按照说明，我将两杯水煮沸，加入面条，煮三分钟，然后从火上移开——此时水是浑浊的——并将调料包倒入锅中。对于第二批，我也在两杯水中煮面条，但单独煮了第三锅，我将它倒入其中一个调料包，然后与煮好的面条混合在一起。我同时做了所有这些菜；目标是近距离体验两碗面条。

左边的碗的特点是汤料和面条水在同一个锅里煮。右边的碗分别盛着汤料和面条。你可能会注意到体积上的差异——两者都是用两杯水煮的，尽管在一炉中蒸发的液体更多。

但是仔细观察，不透明度有明显的不同:

这两种汤尝起来都含有大量的盐和防腐剂。但是左边的那个——不仅仅是我的视觉偏见——对液体的粘度稍微大一点。它在味觉上感觉更饱满，而右边的汤有着清晰而稀薄的质地。浏览面条砖的成分表，我怀疑罪魁祸首可能是木薯淀粉。但这是基于猜测，所以我求助于*外卖*的常驻 [拉面主，迈克·萨坦诺夫](https://thetakeout.com/c/the-ramen-lord) 。他在一封电子邮件中回复道:

“据我所知，方便面本来就是用同样的水煮着吃的。他们肯定会释放一些淀粉，甚至一些油，但据我所知，这是故意的。奇金拉面，最初的速食拉面是由 [安藤桃福](https://www.cupnoodles-museum.jp/en/osaka_ikeda/about/) 自己做的，实际上是将调料涂在面砖上，而不是放在单独的小袋中。你只需要加入沸水来煮，就可以了，不需要过滤。一些制造商现在建议一开始就添加调味料，就像 Shin Ramyun 上的说明一样。但也有例外；我看过其他一些挂面，确实建议沥干。我认为这完全取决于制造商。但是最初？这是一个快速的容器的事情。”

莎廷诺夫的措辞“有意为之”很有道理。这是我的结论:在这种情况下，我实际上更喜欢一体化锅的版本。它的丰富程度有明显的不同。但这也是因为说明书上是这么规定的。也许另一个品牌的速食拉面会建议煮沸后沥干，但那是另一个品牌。再一次，最简单的答案胜出了:遵循最佳结果的指示。
