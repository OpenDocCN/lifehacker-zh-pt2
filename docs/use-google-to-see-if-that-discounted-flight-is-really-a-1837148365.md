# 用谷歌看看打折航班是否真的划算

> 原文：<https://lifehacker.com/use-google-to-see-if-that-discounted-flight-is-really-a-1837148365>

最近，机票交易和销售开始频繁出现。然而，仅仅因为一个航班在“打折”，并不意味着打折价格实际上是一笔交易。



本周，谷歌推出了新的航班价格洞察。它不会对每个航班都有效，但对于你可能正在考虑的许多航班，谷歌可以让你知道航班是低，高，还是传统的典型。

所以，假设，如果一家航空公司提供一张从你的家乡飞往夏威夷的 500 美元的机票，你可以判断这是该航班通常价格的巨大折扣，还是只是广告宣传的交易，而实际上与一年中这个时候的平均价格持平。

例如，我查找了一个飞往檀香山的 528 美元的夏威夷航班。在屏幕的底部，它告诉我 528 美元是这个航班的典型价格，类似航班的平均价格在 465 美元到 710 美元之间。

有了这些信息，我可能会去预订它。如果谷歌说价格很高，或者我接近“典型”成本窗口的顶部而不是底部，我可能会四处寻找不同的航班或稍微调整一下我的旅行日期。

除了基本信息，您还可以查看特定航班的价格历史，以更好地了解价格是上涨还是下跌。看起来，如果我在大约 45 天前扣动火奴鲁鲁航班的扳机，我可以花大约 425 美元得到它。这是下次的好信息。
