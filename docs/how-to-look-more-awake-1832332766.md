# 如何看起来更清醒

> 原文:[https://lifehacker.com/how-to-look-more-awake-1832332766](https://lifehacker.com/how-to-look-more-awake-1832332766)

我是一个可怕的睡眠者。

光线唤醒了我，我公寓外任何路过的汽车也是如此。我通常会在床上扭动一个小时后入睡，试图找到理想的睡姿(我确信这并不存在)。尽管我至少喝了四杯咖啡，但白天我总是很累。

Watch

在大学里，我不止一次假装醒着，让一只眼睛睁着，与我的教授保持持续的目光接触，同时让另一只眼睛休息(并交替)。

换句话说，我很擅长努力让自己看起来清醒，尽管我肯定不是。如果你正在努力避免在会议中打哈欠或在电脑前睁着双眼，这里有一些在你最需要的时候保持清醒的建议。

### 往脸上泼冷水

C 老水会 [收紧你的毛孔](https://www.huffingtonpost.com/2015/01/16/fake-looking-great-when-tired_n_6445532.html) ，让你的皮肤看起来更均匀，给人看起来更清醒一点的样子。

同样，早上洗冷水澡被证明可以减轻压力，增加血液循环，让你感觉更清醒。定期喝水也能帮助你保持警觉，因为脱水会导致疲劳。

### 用去角质产品和爽肤水对抗皮肤问题

当你睡眠不足时，你很可能会发现皮肤上堆积了一层油(你也更容易得)痤疮。正如 [GQ](https://www.gq.com/story/how-to-not-look-tired) 所建议的，用一种好的去角质剂来解决这两个问题，这将有助于清除你脸上的死皮。之后，使用可以去除多余油脂的爽肤水，这样你会看起来更像一个正常的、在办公室里体面的人。

### 干燥时使用润肤霜

为了避免干燥或脱皮，使用好的保湿霜(最好是轻薄无味的)。这将有助于在一个漫长的夜晚后对抗你脸上的一些黑斑或不平。找不致粉刺的，无油的，这样就不会长痘了(我去过，不太愉快)。它也不必是一个昂贵的品牌，因为许多药店出售物美价廉的选择。

### 散步

不要在办公桌前挣扎，每小时花五分钟绕着办公室走一圈。这可以改善你的情绪，减少疲劳，甚至有助于消除饥饿感。它还会让你的 [心率加快](https://greatist.com/live/how-to-stay-awake) ，让你的脸上恢复一点颜色，至少让你维持一小会儿。这也将向你的同事证明你仍然可以站得笔直！暴露在阳光下也有助于提高工作效率，所以如果可能的话，到户外或者窗户边。

### 坐直

有时候，你需要看起来更清醒的是*认为*更清醒。这听起来很简单，但是通过坐直和保持双脚着地来改善你的姿势是度过一天的好策略。你在强迫自己留在位置上并保持警惕(这也被证明是一种情绪和 [生产力助推器](https://www.fastcompany.com/3021985/the-science-of-posture-why-sitting-up-straight-makes-you-happier-and-more-product) )。当你太舒服的无精打采的时候，你会想打瞌睡，所以当你有疑问的时候，好好想想。

### 冷却你的眼袋

如果你非常想隐藏眼袋，试着用冰袋或冷勺子冷却至少 20 分钟。这样可以减少眼睛下方血管的(使人显得疲倦)。眼袋通常是一种遗传特性，所以虽然你可能永远也不会摆脱它们，但这有助于改善它们的外观。

### 疏忽

如果所有这些都失败了(而且你还有一个房间)，就睡一个的午觉。说到看起来更清醒，没有什么比睡觉更好的了。小睡会对学习、记忆和表现有立竿见影的效果，但是无论你做什么，都不要超过 30 分钟。再久一点，你可能会感觉比以往任何时候都更虚弱(谁想这样呢？).
