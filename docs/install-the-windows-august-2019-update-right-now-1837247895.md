# 立即安装 Windows 年 8 月更新

> 原文：<https://lifehacker.com/install-the-windows-august-2019-update-right-now-1837247895>

Windows 10 用户:花一些时间检查你是否已经在你的电脑上安装了微软 2019 年 8 月的安全更新。虽然你可能已经被提示更新你的系统，但仔细检查一下也无妨。

Watch

最新的 Windows 更新 修复了微软旗舰操作系统当前版本中发现的一长串漏洞。谢天谢地，大多数都是最小的风险，但这次更新解决了 Windows 10 的远程桌面协议(RDP)中的四个严重漏洞，其中两个类似于今年早些时候发现并修补旧版本 Windows 的 的 [Bluekeep 漏洞。](https://lifehacker.com/update-pcs-running-older-version-of-windows-immediately-1835149208)

与 Bluekeep 漏洞不同，这些漏洞(稍微)不太令人担心，因为 RDP 功能在 Windows 10 上默认是关闭的。尽管如此，如果它们像 Bluekeep 一样危险，那么你应该确保你已经立即更新了 Windows。

您可以通过执行以下操作来查看您的系统是否有任何挂起的更新:

1.  打开 Windows 开始菜单
2.  单击设置图标
3.  转到**更新和安全> Windows 更新**
4.  如果有新的更新等待安装，您会看到“更新可用”
5.  单击“下载”开始该过程，但请记住，您需要重新启动您的电脑，并等待更新生效。

一般来说，您可能还希望对系统的关键文件进行快速备份，以防 Windows 更新导致任何问题(正如人们经常知道的那样)。虽然快速安装 Windows 更新很重要，但正如我们刚刚谈到的，如果你想给微软一点缓冲时间来修复已经失控的更新，你也可以给自己几天的延迟时间(通过 Windows Updates“[”暂停更新](https://lifehacker.com/the-best-new-features-in-windows-10s-may-2019-update-1834927597) ”功能)。
