# 什么是“炸弹旋风”？

> 原文:[https://lifehacker.com/what-is-a-bomb-cyclone-1833291460](https://lifehacker.com/what-is-a-bomb-cyclone-1833291460)

一场 [“炸弹旋风”周三袭击科罗拉多州](https://www.usatoday.com/story/news/nation/2019/03/14/blizzard-powerful-storm-hit-central-u-s-heavy-snow-winds/3159959002/) 。虽然这个术语听起来像是科幻小说中的东西，但它是真实的——“炸弹气旋”是一个描述炸弹形成后果的科学术语，这是另一个我很感激现在在我的词汇中的词。因此，为了理解“炸弹气旋”,你首先需要了解炸弹的形成。

### 爆炸发生

[据国家海洋局](https://oceanservice.noaa.gov/facts/bombogenesis.html) 报道，一场海啸 sis “描述了一场迅速增强的中纬度气旋。”

这是海洋服务的完整定义。

> 当中纬度 [气旋](https://oceanservice.noaa.gov/facts/cyclone.html) 迅速增强，在 24 小时内下降至少 24 毫巴时，就会发生【爆炸成因】。毫巴测量大气压。当冷气团与暖气团碰撞时，例如温暖海水上方的空气，就会发生这种情况。这种快速增强的天气系统的形成是一个称为炸弹形成的过程，它产生了所谓的炸弹气旋。

我把 T1 称为“炸弹”,因为它是一种迅速发展的爆炸性风暴。Bombogenesis 是 bomb 和 cyclogenesis 两个词的组合。所有的风暴都被认为是旋风，“创世纪”意味着开始。所以炸弹形成是爆炸性风暴的开始。

### 炸弹 c yclone

气压越低，风暴就越强，或者说“炸弹气旋”。更强的风暴意味着大风和大量降雪。

在丹佛的例子中，当来自北方的北极气团遇到当时覆盖该地区的温暖潮湿气团时，气压迅速下降，炸弹气旋形成。

这本质上意味着这是一场迅速出现的强风暴，而不是一场每个人都能追踪其逼近的缓慢风暴。

根据美国国家气象局的数据，虽然我们不太经常听到炸弹气旋，但通常每年至少发生一次。随着气候变化，随着冷空气继续向比过去更南的 T2 方向移动，我们可能会开始更多地看到它们。
