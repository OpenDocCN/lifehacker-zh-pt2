# 如何雇佣虚拟保姆

> 原文：<https://lifehacker.com/how-to-hire-a-virtual-babysitter-1842836439>

那些同时进行的“在家工作”和“养育子女”对你来说怎么样？如果你能时不时地从保姆那里得到一点帮助，那不是很好吗？我们甚至不再关心约会之夜了；不用在任何特定的时刻不停地扮演两个截然不同的角色，那就太好了。



这里有一个解决方案，它不是很棒，但却是 T2 的东西——把它归档到“总比没有好”的文件夹中，这个文件夹现在已经塞满了。这个半有用的建议是:雇佣一个“虚拟”保姆。

虚拟保姆不如真实的东西好，同样，虚拟欢乐时光、 [、虚拟生日派对、](https://lifehacker.com/how-to-celebrate-your-kids-birthday-during-the-pandemic-1842511577) 和虚拟的任何东西都不如真实的东西好。但是如果过去的几个月没有一次又一次地降低自己的标准，那么我肯定是做错了。

虚拟保姆不能把你从出去吃饭(反正你也做不了)或跑腿(你应该尽可能限制)中解放出来。但是他们可能能够与一个小孩互动并让他分心足够长的时间，让你完成一些事情。他们可以打发一点时间，他们可以是一个不同的人，让孩子与之互动，如果他们很好，他们实际上可以让孩子比你想象的更有趣。

### 设定现实的期望

在你雇佣虚拟保姆之前，不要抱有不切实际的期望会有所帮助。这不是一个真正的全职儿童保育解决方案。提前与保姆讨论哪种类型的活动最有可能吸引孩子的注意力；Sittercity 为我们提供了一份很好的清单 ，推荐虚拟保姆可以尝试的活动，包括玩西蒙说和我窥探这样的游戏，一起看书，主持假装烹饪节目，甚至帮助他们做作业。慢慢来，每次尝试一两个小时，看看效果如何。

请记住，如果你的孩子足够小，他们通常需要不断的监督，当他们与虚拟保姆互动时，你仍然需要留在附近，因为那个人无法保证他们的人身安全。

### 找到合适的

找到合适的虚拟保姆和找到合适的真人保姆一样重要——甚至更重要，因为他们真的需要不断地与你的孩子互动和联系。想想他们生活中的谁可能会很好地融入这个角色，无论是你过去雇过的保姆，最喜欢的阿姨，邻居，教练，甚至是他们的日托或幼儿园老师。

很多人现在都失业了，如果他们没有整天照顾自己的孩子(或者他们的时间表有一定的灵活性，想赚点外快)，这对每个人来说都是一个不错的选择。你可能还会想到你生活中的青少年，他们呆在家里，与朋友隔绝，感到厌倦。你的侄女或侄子或朋友的大孩子可能会喜欢“照看”一个小家伙，并赚取一些额外的视频游戏钱。

### 你也可以雇佣一个“专业人士”

如果你生活中的每个人都和你一样忙碌，那么许多为父母匹配保姆的公司，如[【Care.com】](https://www.care.com/)[sitter city](https://www.sittercity.com/sitters/virtual-babysitting-activities-for-kids)和 [UrbanSitter](https://www.urbansitter.com/) ，现在都在鼓励会员根据需要请求并执行虚拟保姆服务。因为这些网站上没有专门选择“虚拟”保姆的选项(至少目前没有)，所以未来的保姆可以在他们的简历中添加“虚拟保姆”，以表明他们愿意提供这项服务，父母可以给潜在的保姆发消息，询问他们是否愿意尝试。

此外，由于虚拟保姆没有单独的审批程序，Care.com 的一名代表告诉我，他们将通过相同的背景和安全检查，已经到位，成为该网站上的正式批准的护理员。
