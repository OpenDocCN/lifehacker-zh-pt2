# 为什么没做完的工作能让你成为更好的员工

> 原文：<https://lifehacker.com/why-leaving-work-undone-can-make-you-a-better-employee-1835652698>

你如何知道你在工作中是否做对了事情？你可能想问问自己还有什么没做。

Watch

在[prosper globa](https://thriveglobal.com)l，杰西卡·希克斯指出了如果你想保持高效率并保持良好的工作生活平衡，应该避免的三个职场习惯，其中一个职场坏习惯是 [每天督促自己完成清单上的每一项任务](https://thriveglobal.com/stories/how-to-become-more-productive-at-work-change-habits/) :

> 当一天结束的时候，如果你还没有完成你想要完成的所有事情 ，你可能会感到沮丧。虽然你最初可能会感到沮丧，但有一两个未完成的任务实际上可能是一个迹象，表明你正在成功地对一天中的 进行优先排序。也许你无法安排会议，但你完成了一份第二天到期的重要报告——在这种情况下，你能够完成最紧迫的任务，并可以安排第二天的会议，那时你会重新充满活力并有一套新的优先事项。

正如希克斯所说，通过适应“未完成的事情”，你会更好地理解你的优先事项(不管是什么，你都要完成的任务)和你的技能组合(你自然倾向于和/或避免或拖延的任务)。

换句话说:如果你花了一天的大部分时间来完成一个大项目，但是没有提交你的费用报告，你可能已经搞清楚了你的优先事项。这些费用报告必须在某个时候归档——标准建议是每周或每两周留出一大块时间专门用于这类管理工作——但你成功地避免了让它们打断你最重要的任务。

另一方面，如果你发现自己整天都在做那些不重要的事情，并且回避那些占你工作大部分的项目，那就有必要弄清楚为什么你没有优先处理最重要的工作。

例如，也许你不清楚项目的下一步。也许你是一个效率低下或缺乏动力的团队的一员，或者也许你知道完成你的项目意味着与一个你不喜欢的经理进行一次冗长的批评会议。

也许你生活中有其他压力，让你很难专注于比查看邮件更复杂的事情。也许你没有足够的睡眠、阳光或社交活动。

也许你的工作日被会议和签到穿插得太多，以至于你没有足够的时间给你的项目以它所需要的深度关注。

但也许你只是在推迟你最不喜欢的任务。我们中的一些人喜欢提交费用报告，但讨厌写季度报告。我们可能喜欢组织而不是创造，喜欢完成几十个小任务而不是一个大项目，或者喜欢团队工作而不是独自工作。

如果你总是发现自己在逃避大部分工作——他们雇佣你来完成的工作——而优先考虑工作中更有成就感的其他方面，这可能是一个信号，表明你处于错误的位置，应该开始寻找一份更合适的工作。

所以，看看在一个典型的工作日结束时，你还有什么没有完成。如果是低优先级的工作，你可能会成为比你意识到的更好的员工。如果是高优先级的事情，可能是时候找一份你可以花时间做你擅长的工作的工作了——无论是为一家提供你需要完成它的结构的公司做类似的工作，还是让你专注于你自然优先考虑的工作。
