# 旅游时如何避免被骗

> 原文:[https://life hacker . com/how-to-avoid-get-scammed-when-traveling-1835913288](https://lifehacker.com/how-to-avoid-getting-scammed-when-traveling-1835913288)

本周在希腊， [卧底警察](https://www.independent.co.uk/travel/news-and-advice/greek-restaurants-tourist-rip-off-mykonos-overcharge-raids-tax-inspectors-a8973411.html) 正在打击向游客过度收费的餐馆；一名美国游客在 [猫途鹰](https://www.tripadvisor.com/ShowUserReviews-g659660-d12991346-r674941074-Dk_Oyster-Platys_Gialos_Mykonos_Cyclades_South_Aegean.html) 上说，他在米克诺斯岛的一家餐馆吃了六瓶啤酒和三份沙拉，被要价 830 欧元。

让我们面对现实吧——作为游客，我们都曾成为偶然骗局的受害者。例如，我已经同意支付超过必要的出租车费，可能还会花高价买一份沙拉。

正如 u/donlemonny 2019 在最近的 [Reddit 帖子](https://old.reddit.com/r/LifeProTips/comments/c5pkmx/lpt_when_traveling_international_look_up_what_the/) 中建议的那样，当你在海外旅行(甚至在国内旅行)时，你应该总是准确地研究目的地的常见骗局，这样你就不会*成为游客被诈骗的受害者。*

这些骗局并不总是显而易见的，像扒窃这样的非法行为也是如此；这可能很简单，比如多收你的水或者分发混音带来换取现金“捐赠”(一种常见的纽约骗局)。

在最近的一篇文章中，我们收集了一些其他熟悉的骗局，这些骗局来自于访问过全球各个城市的读者。这里有几个值得注意的:

*   “芝加哥:停车费太贵了，我亲爱的游客，如果你不开车进城，就不要去租车了。在市区停车两个小时很容易就要 50 美元。”——
*   “早在 2010 年，我就在保加利亚发现了这一点，但我想每个国家的旅游目的地可能都是如此。这是一个瓶装水骗局，人们重复利用他们在垃圾桶里、街上等地方找到的旧水瓶。并重新注入标准自来水。”- [AG74683](https://kinja.com/ag74682)
*   “在多伦多，有一个女人到处告诉人们她们很漂亮，并提供一张贴纸以换取捐款。她有时说是为了慈善事业，有时又不说要去哪里，但钱总是会进她的口袋。《多伦多太阳报》(Toronto Sun)在《贴纸女士》(Sticker Lady)上写了一篇文章，揭露了她是如何用自己收集的钱为自己在郊区的房子和大众汽车提供资金的。-[cccccccc](https://kinja.com/biteme657)T3】
*   “在上海，如果你在南京东路上...如果有人找你聊天，那就是个骗局。不要去任何地方喝茶，看艺术展，或者帮助任何人练习英语。每杯茶要花数百美元，艺术品会很烂，价格会高得让人内疚，而‘练习我的英语’这句话总是会引出前面提到的两件事。”- [ulic14](https://kinja.com/ulic14)

如果你想在你的目的地研究一些熟悉的骗局，最简单的方法是在猫途鹰这样的旅游网站上整理评论；举例来说，如果你计划去参观埃菲尔铁塔，你应该通读一些更具批判性的评论。(一些猫途鹰的贡献者警告游客们不要排队“为了更快登上高塔而骗人。)

否则，就使用 Reddit 这样的网站进行在线搜索，找出你应该不惜一切代价避免的其他骗局。这些骗局不应该把你吓跑——在参观一些旅游热点时知道会发生什么是很好的。
