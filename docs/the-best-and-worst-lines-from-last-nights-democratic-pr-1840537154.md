# 昨晚民主党总统候选人辩论的最佳和最差台词

> 原文：<https://lifehacker.com/the-best-and-worst-lines-from-last-nights-democratic-pr-1840537154>

如果你错过了昨晚的民主党总统候选人辩论，这里有一个快速的回顾:伊丽莎白·沃伦谈到了她提议的财富税，伯尼·桑德斯称川普为“病态骗子”，每个人都讨厌葡萄酒洞穴。好奇我们在说什么吗？下面，是 2019 年第六场也是最后一场辩论中我们最喜欢的几个解释和俏皮话。



### [杨安泽讨论总统候选人缺乏色彩的原因](https://www.youtube.com/watch?v=o6xDVej9zFs&t=14s)

在新泽西参议员科里·布克和前住房和城市发展部长朱利安·卡斯特罗都没有资格参加昨晚的辩论——以及加州参议员卡玛拉·哈里斯本月早些时候退出——之后，杨安泽仍然是台上唯一的有色人种。在讨论总统竞选中缺少有色人种的代表时，他批评了眼前的问题。

“黑人家庭的平均净资产只有白人家庭的 10%，”他说。“对于拉丁美洲人来说，这一比例为 12%。如果你是一名黑人女性，你死于分娩并发症的可能性会增加 320%。这些数字定义了我们国家的种族。问题是，为什么我是这个舞台上唯一的有色人种候选人？不到 5%的美国人向政治活动捐款。你知道你需要捐什么给政治活动吗？可支配收入。”

### [沃伦为自己的年龄辩护，在竞选筹款会上喊出 Buttigieg](https://www.youtube.com/watch?v=NQQswRLws4k&t=16s)

在连续第二场辩论中，包括拜登、桑德斯和沃伦在内的候选人被问及他们的年龄，因为这与成为总统有关。当被问及如何权衡时，沃伦说出了这样一句俏皮话:“我也会成为有史以来就职的最年轻的女总统。”

后来，沃伦指责 Buttigieg 最近在纳帕谷和一个“葡萄酒洞”举办的“900 美元一瓶的葡萄酒”筹款活动。她说，“葡萄酒洞里的亿万富翁不应该选择下一任美国总统。”但蒂吉格回应称，沃伦无法通过她自己作为百万富翁的“纯度测试”。

后来，桑德斯加入沃伦的努力，呼吁拜登和 Buttigieg 支持他们的竞选捐款人。“我的好朋友乔，他是我的好朋友，他收到了 44 位亿万富翁的捐款。皮特，另一方面，他落后了，皮特。你只有 39 个亿万富翁捐款。所以，皮特，我们期待你。我知道你是一个精力充沛的人，也是一个好胜的人，看看你能否在这个问题上与乔较量。”

### Klobuchar 批评 Buttigieg 关于候选人经历的言论

在一次关于赔偿的讨论中，不知何故，话题转回到 Buttigieg，Amy Klobuchar，批评市长在之前的辩论中使用了明显的语言。

“市长，当我们进行最后一场辩论时，你基本上是在嘲笑 100 年的舞台经验，”她说。"...我只是认为，当你审视自己如何评价一个能把事情做好的人时，你应该尊重我们的经验。”后来，克洛布查尔认为总统候选人应该有能力赢得选举。但是蒂吉格回应道...如果你想谈论获胜的能力，那就试着组成一个联盟，让你以迈克·彭斯印第安纳州同性恋者 80%的选票重新当选。”

然而，Klobuchar 并不确定，并提到了 Buttigieg 在 2010 年竞选印第安纳州财政部长时的失利。“再说一遍，我愿意，市长...如果你在印第安纳州赢了，那是一回事。你试过了，但你输了 20 分。对不起这只是数学上的。”

### 候选人赠送礼物并要求道歉

在向台上的七位候选人提出的一个问题中，PBS 新闻小时节目主持人兼辩论主持人朱迪·伍德拉夫本着节日的精神提出了一个特殊的问题:有没有一位候选人你想请求原谅，或者有没有一位候选人你想送给他一份礼物？嗯，答案和问题一样有用。以下是他们回答的部分内容:

*   杨:“我很乐意给你们每人一本我的书。”
*   Buttigieg: “我想我们所有人最终都会想要同样的东西。我们知道，与我们所拥有的相比，这里的任何人成为美国总统，对未来和国家来说都是一份礼物。”
*   **沃伦:**“我会请求原谅。我知道有时候我真的很激动，有时候我有点激动。我真的不是故意的。”
*   拜登:“我和我妻子有一份 20 到 100 人的电话清单，我们至少每周或每月打电话告诉他们，我在这里。我给他们我的私人电话号码。他们和我保持联系...我会给这里的每个人一份礼物的原因是，他们想像我一样做一些事情，让他们的生活变得更好，因为有很多人受到了非常非常严重的伤害。”

*   桑德斯:“但我认为，我们所有人需要给美国人民的礼物是对特朗普政府现实的一个非常非常不同的愿景。我们需要提出的愿景是建立一个基于爱和同情，而不是贪婪和仇恨的政府和国家。”
*   克洛布查尔:“好吧，我会在你们任何时候对我发火的时候请求原谅。我可以直言不讳。但我这样做是因为我认为在这里挑选合适的候选人非常重要。我愿意...作为民主党人，我们必须记住，如果我对此感到激动，那是因为我内心深处非常相信，我们必须让人们与我们站在一起，而不是将他们拒之门外。这是我们在这次选举中能够给美国的礼物。”
*   施泰尔:...团队合作的礼物。因为这里的问题是，我们如何一起改变这个框架？我们如何一起打败这个腐败犯罪的总统？”
