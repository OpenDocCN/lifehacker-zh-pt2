# 用 2020 火星车把你的名字送上火星

> 原文：<https://lifehacker.com/send-your-name-to-mars-with-the-2020-rover-1834974442>

如果你曾经渴望去火星旅行，现在是你的机会。嗯，好吧，*你*不能去火星旅行，但是你的*名字*可以，这是你能得到的最接近的了。



[火星 2020 漫游者](https://mars.nasa.gov/mars2020/) 正准备于明年 7 月开始它为期 7 个月的火星之旅，美国宇航局称它将带上你的名字:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1130959203802255362&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1130959203802255362&autosize=1) 

一旦到达，它将花费至少一个火星年(大约 687 个地球日)收集数据和岩石和土壤样本进行研究。

加入 200 多万的名字(包括我的！)已经报名的， [去 NASA 的网站](https://mars.nasa.gov/participate/send-your-name/mars2020/) ，输入你的姓名、国家、邮政编码和邮箱地址，然后点击“把我的名字发给火星。”您可以下载或打印您的登机牌。

如果你有年幼的孩子，他们可能会从中得到乐趣，这将是一个谈论太空探索和 [一起了解](https://mars.nasa.gov/mars2020/) 任务的好机会。

但如果你想单飞，那也没问题。
