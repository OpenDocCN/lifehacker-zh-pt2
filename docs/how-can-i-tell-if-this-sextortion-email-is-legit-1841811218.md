# 我怎样才能知道这封“色情”邮件是否合法？

> 原文:[https://life hacker . com/how-can-I-tell-if-this-sextortion-email-is-legit-1841811218](https://lifehacker.com/how-can-i-tell-if-this-sextortion-email-is-legit-1841811218)

垃圾邮件是令人讨厌的，但是如果你使用一个还算不错的电子邮件提供商，你就不应该在你的收件箱里看到太多垃圾邮件。至少，我发现很少有东西能逃过 Gmail 强大的垃圾邮件拦截器，让我考虑打开它。

另一方面，当你收到一封听起来很不愉快的电子邮件时，你会怀疑自己。如果 Gmail 没有将其标记为垃圾邮件，即使它听起来像垃圾邮件，它真的是垃圾邮件吗？然后，发送者甚至会加大赌注，正如《生活黑客》读者**梁**所描述的:

> *“您好，抱歉打扰，我最近收到了一封来自我自己的色情邮件，我试图阻止它，但我不能，当我回复邮件时，它进入了我自己的帐户。请问我是被黑了还是只是垃圾邮件？还有，黑客是怎么用我的账号发邮件的？”*

梁，好消息来了。你没有被性扭曲。你只是被垃圾邮件，可悲的是，可能有很多人陷入这种胡说八道，最终发送金钱，比特币，登录，或者谁知道还有什么给这些骗子。

我怎么知道这个？好吧，让我们坦诚相待。除非你的系统中有某种恶意软件触发你的网络摄像头，制作录像，然后将这些录像发送给第三方，而第三方*也*知道你是谁，否则我真的不认为有人会发现你做了任何你可能不想公开的事情。你可以通过问自己一个简单的问题来验证这一点:在过去的几个月里，我在电脑前裸体了多少次？

作为一个推论，你可以这样问自己:如果有人有我的罪证照片或视频，为什么他们在没有向我出示任何证据的情况下威胁我？一个模糊的“抓到你了”感觉像是大多数人会忽略的东西；一封陌生人发来的电子邮件中出现了一张你裸露臀部的真实照片，作为黑客入侵的证据，这是很难忘记的——而且可能会给攻击者带来更有利可图的结果。

但是你没有得到这种性扭曲证据的原因是因为它不存在。你没有被黑。如果你愿意，你可以在你的系统上运行一个快速的 [杀毒和反恶意软件扫描](https://lifehacker.com/use-these-antivirus-and-anti-malware-apps-instead-of-av-1841264690) ，并更改你用来保存照片的任何云存储服务的密码，但我不会在这上面花太多时间。你正被无聊的垃圾邮件所引诱，但是一个随机的互联网人实际上并没有你的任何生动的照片。

至于邮件是你自己发的，这很容易做到。快速的谷歌搜索会发现很多网站，你可以使用 [来欺骗发送者](http://www.sendanonymousemail.net/) 。所以，再说一次，我不会用你收到了一封来自“你自己”的电子邮件这一事实来证明有人闯入了你的电脑或电子邮件账户，拿到了你秘密收藏的性感自拍。这*是*一个聪明的把戏，可能会骗过一些人，但骗不了你——至少，再也不会了。

事实上，我敢打赌，你收到的电子邮件很可能是垃圾邮件发送者群发的“普通”格式信件，或“喷雾祈祷式”攻击。即使 99.5%的收件人忽略该邮件，向一群数百万泄露的电子邮件地址发送垃圾邮件的成本也很低，而且轻松的回报值得花费少量时间来发送一吨电子邮件。你可以在这里 看到一些典型的色情邮件 [的例子；你的可能看起来很像这些。](https://cofense.com/sextortion/)

虽然一些 Lifehacker 的读者可能会叹息，并想知道为什么我们甚至懒得谈论这个问题——因为这是如此明显的垃圾邮件——我认为记住不是每个人都有相同水平的技术专业知识更有用。这就是为什么你的祖父母可能一直打电话给你，询问他们收到的各种垃圾邮件。或者，更糟的是，为什么他们自己也受到了一些攻击。

像这样的垃圾邮件利用了一种原始的、数字时代的恐惧，我敢打赌它比你典型的“便宜买这些药吧！1!!"废话。如果你*知道*你已经在你的设备上做了任何事情，包括你的裸体、你的半裸身体、你朋友的裸体，或者任何其他有趣的组合，那就更是如此。但是吓人的垃圾邮件还是垃圾邮件；直到有人咳出一张你处于危险姿势的照片，然后你应该与当地警方讨论，删除邮件，继续你的一天。你很好。
