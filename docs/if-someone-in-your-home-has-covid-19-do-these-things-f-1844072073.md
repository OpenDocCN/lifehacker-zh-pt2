# 如果你家里有人患有新冠肺炎，先做这些事情

> 原文:[https://life hacker . com/if-有人在你家里做了新冠肺炎做的这些事情-f-1844072073](https://lifehacker.com/if-someone-in-your-home-has-covid-19-do-these-things-f-1844072073)

如果和你一起生活的人被诊断出患有新冠肺炎病毒，或者看起来很可能患有这种病毒(例如，如果他们正在等待测试结果)，你突然有了一项重要的工作:照顾这个人，同时降低他们将病毒传播给你或其他任何人的风险。以下是你得到诊断后应该做的事情。

### 弄清楚你能在多大程度上彻底隔离感染者

如果可以，给病人自己的卧室和浴室。如果你有一个客房，或者如果你的房子有多个卧室，你可以随意安排谁睡在哪里，这将是最理想的。

### 指定一个关键人物

如果你家有多人，选择一个人与生病的人互动。(同样，如果有人风险更高，他们就不应该担任这个角色。)

如果你是关键人物，你将是病人的主要看护人，你将是为他们打扫卫生、洗衣服、给他们带食物等等的人。这最大限度地降低了对家中其他人的风险。

### 确保这个人仍然有社会关系

有了所有的预防措施，感染者很容易花很多时间独处。现在不是亲自拜访的时候，但是当你准备好临时病房的时候，要确保这个人有足够的机会和其他人交流。确保他们手边有电话或设备，可以在室内或室外与他人进行 Facetime 或聊天。

托雷达斯博士指出，沟通对一个人的心理健康很重要，同时也是其他人关心他们的机会。如果你和一个生病的朋友面对面交流，并注意到他们看起来异常疲惫或呼吸急促，这将是一个重要的警告信号，表明他们可能需要帮助。

### 收集你的补给

你去商店的次数越少越好。下面是 [关于](https://lifehacker.com/what-supplies-to-have-at-home-in-case-you-get-sick-1842582934) 的几件事:

*   清洁产品，包括消毒剂和漂白剂，用于清洁表面
*   肥皂和洗手液保持双手清洁
*   温度计
*   退烧药，如泰诺，如果需要的话
*   面具，如果你还没有的话

医疗专业人士 对脉搏血氧仪是否是必需品存在一些分歧。(这是一种无痛夹在手指上的装置，用光来测量血液中的氧含量。)托拉斯博士推荐了一种方法，他指出，一些新冠肺炎患者的血氧饱和度低到了危险的程度，却不会感到气短。如果你使用一个，这是一个好主意，告诉你的医生如何解释结果。

### 将传播的可能性降至最低

虽然许多新冠肺炎病例是在家庭内部传播的，但托拉达斯博士指出，也有很多这样的例子，家庭中有一个人被感染，但由于小心谨慎，没有传染给他人。所以付出额外的努力来降低传播的风险是值得的。

冠状病毒传播的主要途径(不是唯一途径)是通过呼吸道飞沫。换句话说，当有人靠近你呼吸、喊叫或咳嗽时。所以你可以降低风险:

*   保持六英尺的距离。
*   如果可以的话，打开窗户或者在外面呆一会儿。
*   口罩有帮助。当你密切接触时，感染者和其他家庭成员都应该戴上口罩。
*   塑料面罩也可以防止额外的水滴溅到你的脸上，包括你的眼睛。如果你有，就用它。

病毒也可能通过表面传播，因此要防止:

*   在接触病人的物品后，或者在给病人洗衣服或打扫卫生后，经常戴手套和/或洗手。
*   当你和他们接触时，考虑穿上一件防寒服，防止水滴溅到你的衣服上。例如，这可能是一个塑料雨披。
*   消毒病人接触的表面，尤其是经常接触的东西，如门把手和工作台面。

### 帮助他们跟踪和监控症状

轻微的冠状病毒病例不需要像感冒或流感那样的护理。但是新冠肺炎可能会变得严重，甚至致命，所以记录他们的健康状况很重要。

记下症状，这样你就可以在寻求医疗护理时回答问题——例如，症状是从哪一天开始的？

一定要注意这个人需要紧急护理的迹象。那些预警信号，按照疾控中心的说法， [包括](https://www.cdc.gov/coronavirus/2019-ncov/if-you-are-sick/care-for-someone.html) :

*   呼吸困难
*   胸部持续疼痛或压力
*   新的困惑
*   无法醒来或保持清醒
*   带蓝色的嘴唇或脸

如果可以的话，在带新冠肺炎病人去医院之前，一定要提前打电话。

根据当前的家庭隔离指南，采取这些预防措施。如果您的医疗保健提供者没有给你具体的指示，那么 [CDC 关于何时结束家庭隔离的页面在这里](https://www.cdc.gov/coronavirus/2019-ncov/if-you-are-sick/end-home-isolation.html?CDC_AA_refVal=https%3A%2F%2Fwww.cdc.gov%2Fcoronavirus%2F2019-ncov%2Fprevent-getting-sick%2Fwhen-its-safe.html) 。目前，据说病人可以在退烧三天后结束家庭隔离，*和*他们没有症状，*和*他们首次出现症状已经至少十天了。健康的护理人员在首次接触病毒后应在家呆 14 天。
