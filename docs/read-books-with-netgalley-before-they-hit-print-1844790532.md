# 在书籍付印前用 NetGalley 阅读

> 原文：<https://lifehacker.com/read-books-with-netgalley-before-they-hit-print-1844790532>

在我成为 Lifehacker 之前的日子里，我是美国最大的连锁书店的内容创建者——我肯定你知道那家——每天来到办公室就像书呆子的圣诞节:我的邮箱不断收到出版商寄来的包裹，里面有几个月内不会上市的书籍的预读本(或“ARCs”)。进入这些独家邮件列表可能会很棘手(这肯定是我最怀念的额外待遇)，但如果你的手机上有电子阅读器或电子书应用程序，你可以通过[netgally](https://www.netgalley.com/)捕捉一点那种魔力。

号称“一个书评人和其他专业读者可以在书籍以电子版或数字版形式出版之前阅读它们的网站”，NetGalley 主要是作为出版商——从印度到“五大”——的一种方式存在，通过将它们分发给广泛的“有影响力的人”网络来为即将出版的书籍制造声势，这些人希望在他们的出版物上，在他们的博客上，甚至在 Goodreads 上评论它们。(从经验上讲，成为图书影响者肯定不如成为 Instagram 影响者光鲜，但至少你不用担心你的自拍游戏。)以下是参与行动的方法。





### **如何加入 NetGalley**

虽然 NetGalley 表面上是在寻找“专业读者”，但互联网的奇迹——以及 Goodreads 和其非亚马逊控制的替代品 等书评聚合网站的奇迹——意味着*任何人*都可以成为专业读者。当然，你不需要提供任何诚意来注册账户。只需前往 NetGalley 并创建一个。你需要提供一些基本信息——你的名字、年龄、电子邮件地址——但是如果它要求你的“公司”,你可以随意输入你的博客或者 Goodreads 的名字。你也可以随便编点什么，尽管诚实——实际上拥有一个评论平台——会帮助你获得更受欢迎的书评。为了阅读预发本，你还需要写一份简短的简历，列出你最感兴趣的阅读类型。无论你提供什么信息，一旦你注册并确认了你的账户，你就可以开始浏览即将出版的图书目录，并提供电子下载。有些书只有在你请求审阅并获得出版商批准的情况下才会被交付给你(这有助于作为一名审阅者拥有一个公认的声音，无论是在 Goodreads 上还是在你自己的博客或 YouTube 频道上)。其他的——事实上是很多——任何人都可以免费下载并立即阅读。只需点击“查找标题”并查找 [“立即阅读”链接](https://www.netgalley.com/catalog/readnow?sort_books=pop.desc&text=&target=section) 。

诚然，这些免费提供的图书中的大部分将是你从未听说过的书籍，其中许多来自 T2 的小出版商，但这并不总是一件坏事。你也会在这里找到主要出版商和作者的书籍；在撰写本文时，最受欢迎的免费图书之一是 [*第九步站*](https://www.netgalley.com/catalog/book/200855) ，作者是一批主流出版的科幻作家，包括雨果奖提名者马尔卡·奥尔德和弗兰·怀尔德。

书籍可以以各种格式下载——包括有声读物——或者你可以在 iOS 和 Android 上可用的原生[NetGalley Shelf app](https://netgalley.zendesk.com/hc/en-us/articles/360048125994-How-to-use-the-NetGalley-Shelf-app)中阅读它们。请注意，这些特殊的提前拷贝通常包括拷贝保护，这将需要您跳过一些环节才能将它们加载到您的设备上，但该网站提供了一些 [好的常见问题](https://netgalley.zendesk.com/hc/en-us/articles/115003990954-I-need-access-to-Digital-Review-Copies-How-do-I-get-started-) 来帮助您开始。你也想等到准备好阅读时再下载，因为它们通常会在几周或一个月后过期。

如果你没有一个好的方法来证明你作为一个书评人拥有强大的追随者，不要对看到超级著名作家的书籍列表过于兴奋——这些几乎总是在“批准”的基础上，出版商将不愿意确定任何随机寻找的人获得免费的副本。也就是说，这是值得一试的 NetGalley 还提供了一些小技巧来提升你的个人资料，并有希望让你进入必读的 ARC。
