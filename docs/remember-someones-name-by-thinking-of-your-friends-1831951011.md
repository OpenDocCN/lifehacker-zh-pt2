# 通过想起你的朋友来记住某人的名字

> 原文：<https://lifehacker.com/remember-someones-name-by-thinking-of-your-friends-1831951011>

我不擅长记名字。你也是。每个人都不擅长记名字。你已经读过一百万个记住别人名字的技巧，但没有一个有用。但也许这个会！这是我听过的最简单、最不花哨的技巧之一，我以前也曾用过，但并没有认真思考过。 [Redditor analogHedgehog 称](https://www.reddit.com/r/AskReddit/comments/aij1ar/compared_to_everyone_else_your_age_what_life/eeo8uj1/) :

> 我不会试图记住一个人的名字，我只会在脑海中把他们和一个我认识的同名的人联系起来。所以，如果我遇到一个叫丹尼尔的人，我会试着记起我的老实验伙伴丹尼尔。这一点似乎在我脑海中更加突出，因为我可以将我刚刚遇到的人与复杂的记忆联系起来，而不是仅仅一个我不可避免会忘记的单词。

Watch

你想起一个同名的朋友，你想象这两个人在一起。它如此简单，几乎算不上是一个黑客——一些最伟大的黑客都是如此。正如 redditor 所说，你与你认识的人有着千丝万缕的联系，而不是押韵的记忆法，甚至不是名人的名字。

正如 analogHedgehog 所承认的，有一点需要注意的是:“我现在唯一有困难的名字是那些我以前没听过的名字。”你可能不得不求助于一个名人，或者一个声音相似的人。或者你必须知道他们的姓，并想象他们和你的朋友有关系。
