# 为什么你应该冷藏红酒

> 原文：<https://lifehacker.com/why-you-should-chill-red-wine-1837385067>

很少有人认为红酒是需要冷藏的东西。毕竟，红酒通常是在“室温”下供应的，但这一数值会因房间、房屋和气候的不同而大相径庭。但是大多数红酒都受益于至少在冰箱里的一点时间，尤其是在炎热的月份。

Watch

为了找出葡萄酒应该在什么时候冰镇，以及应该冰镇到什么程度，我联系了来自 [Shalom Y'all](https://www.shalomyallpdx.com) 和之前的 [Enoteca Nostrana](https://enotecanostrana.com) 的 Ian Ferrier。(他还为我提供了许多杯葡萄酒，同时试图在这个问题上教育我，并取得了不同程度的成功。)

### 偏于冷静

“每种葡萄酒都有自己特定的理想温度范围，但一般来说，你会更倾向于较冷的一端，而不是较热的一端，”费瑞厄解释道，“因为喝一杯热起来的葡萄酒总是很好的。”就度数而言，“你总是希望你的红色至少在 60 到 70 度的范围内。你不想要比这更热的葡萄酒的主要原因是，当你把鼻子伸进酒杯时，芳香开始转移，酒精的感觉开始接管。你尤其不想要更温暖的大红酒，因为它的酒精含量往往更高。总的来说，把它想象成喝一杯热的帕布斯特和一杯冰的。你尝的东西不一样，口感也不一样。”

### 酒越淡，你就能越凉快

“一般来说，”费瑞厄解释说，“你可以冷却更浅的红酒，降低到传统的白葡萄酒温度[49-55℉]。你不希望一个大红色的葡萄酒变得这么冷，因为，除了其他因素之外，冷冻葡萄酒可以增加单宁结构的感觉，可以使葡萄酒变得苦涩和尖锐。浅红色可以增加酸度，增强结构，展现更多花香。每种葡萄酒都不一样，都有自己的最佳点。”

### 为了泡泡而做

如果你喜欢泡泡，寒冷是你的朋友。“就起泡的红葡萄酒而言，就芳香、结构等而言，葡萄酒本身通常遵循相同的原则。但最大的区别是溶液中的二氧化碳，”费里尔告诉我。“如果你的酒是凉的，二氧化碳很好地留在溶液中，你会得到极好的气泡，如果是室温，当你打开它时，你的瓶子可能会爆炸一点。除此之外，这就像温苏打水和冷苏打水的对比——碳酸化是粗糙和不愉快的。葡萄酒变暖后可能会很好喝，但在打开传统的‘香槟风格’软木塞时，你会想要一杯冰镇葡萄酒。”
