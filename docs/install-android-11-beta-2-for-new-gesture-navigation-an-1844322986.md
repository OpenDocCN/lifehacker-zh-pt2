# 安装 Android 11 Beta 2 以获得新的手势导航等功能

> 原文:[https://life hacker . com/install-Android-11-beta-2-for-new-gesture-navigation-an-1844322986](https://lifehacker.com/install-android-11-beta-2-for-new-gesture-navigation-an-1844322986)

Android 11 beta 2 来了。这意味着我们离发布预览版只有一个测试版了，也就是所谓的“如果你想在尽可能稳定的环境中尝试新特性，你想要得到的测试版。”从那里开始，它就到了 Android 11 最终版本。如此接近！

Watch

如果你之前已经在 Android 11 测试版的早期阶段 [为自己](https://lifehacker.com/four-ways-to-install-the-new-android-11-public-beta-1843983079) 注册了，我强烈建议你今天检查一下你设备的系统更新设置。(如果你想在今天的测试中迈出第一步，很容易 [设置](https://www.google.com/android/beta) ！)

一旦你完成并安装了最新的测试版——下载应该不会花很长时间——有一些你想知道的新特性。最重要的一个是一个新的导航手势，使 Android 的手势导航模式更容易接受。

如果你因为是超级用户而拥有大量主屏幕，你现在可以从屏幕底部向上滑动，返回主屏幕(就像你以前在两键或三键导航中点击虚拟主屏幕按钮一样)。

其他两个关键手势保持不变:在屏幕上的任何地方向上滑动都会拉起你的应用抽屉，从设备底部向上滑动会打开你最近打开的应用的 Android 视图。简单。

### 其他值得了解的 Android 11 beta 2 更新包括:

*   **屏幕录制**现在，除了拍摄你正在做的任何事情的视频之外，你还可以拍摄你的 Android 的音频
*   Android 11 重新设计的**媒体播放器**现在是你的通知面板中的默认选项；以前，您必须启用开发人员选项来解锁它
*   谷歌正在一点点清理安卓的份额表。如果某个应用程序之前提示您多个共享选项，这些选项现在将被隐藏在该应用程序的图标下。你可以点击它来展开选项菜单，然后点击你想要的选项。
*   您现在可以编辑**应用** [**对话**](https://lifehacker.com/the-13-best-new-features-of-android-11-1844019029)——一种不同类型的通知——的设置，通过设置>应用&通知中的专用部分。你首先要调整设置*，让*一个应用的“对话”通知出现在这里，这可以通过从你的 Android 屏幕顶部下拉后长按这些通知来完成。
*   对画中画窗口的新限制——感谢谷歌——意味着你不能像以前那样放大你的小视频。
