# 给每个三明治涂上厚厚的一层黄油

> 原文：<https://lifehacker.com/slather-every-sandwich-with-butter-1840731274>

我是黄油 的忠实粉丝，但我没有充分发挥它的潜力。这是一种明显的土司配料和烹饪脂肪，但它也是一种非常奇妙的(并且被犯罪性地低估了的)三明治涂抹料。

Watch

如果你是一个英国人，你可能已经这样做了，所以不要再看这个了，去快餐店或酒吧或其他地方吧。但是你们其余的人——那些没有在三明治上涂黄油的人——再等一会儿。

黄油和大多数东西一起吃味道很好。配奶酪很好吃。和肉一起吃很好。配蔬菜很好吃。主要是脂肪，它比蛋黄酱更好地排斥水，它增加了一点奶油，而不会用分散注意力的味道压倒你的三明治。它让你放在上面的每一个三明治都更加丰富，而不那么潮湿。坏处在哪里？

我是在建议你用黄油代替蛋黄酱吗？不，我建议你两者都用。(我在那方面有点享乐主义！)然而，如果你是一个讨厌蛋黄酱的人，你绝对应该使用它；我一直想知道讨厌蛋黄酱的人是如何忍受这么干的三明治的。制作熟食沙拉三明治，如鸡肉或金枪鱼？那些会很潮湿，所以你会想在三明治的两面都涂上黄油。

如果你真的想从中得到乐趣，就去买些复合黄油吧。(我们有 [布里黄油](https://skillet.lifehacker.com/brie-butter-is-the-best-compound-butter-1833264190)[黑蒜黄油](https://skillet.lifehacker.com/how-to-make-black-garlic-butter-1833065260)[鸭肉黄油](https://skillet.lifehacker.com/this-compound-butter-is-basically-duck-frosting-1838492514)[培根黄油](https://skillet.lifehacker.com/make-bacon-butter-and-put-it-on-everything-1825658580) ，还有 [酸奶黄油](https://skillet.lifehacker.com/why-you-should-add-a-little-yogurt-to-your-butter-1832877504) ，都很棒。)如果你担心冷黄油会撕破你的面包，使用 [蔬菜削皮器](https://skillet.lifehacker.com/use-a-vegetable-peeler-for-perfectly-buttered-toast-1840614871) 。(虽然真的，你应该在任何时候都有至少一种室温的黄油。)没有理由不在你的三明治上涂黄油( [甚至素食者的](https://skillet.lifehacker.com/these-fake-butters-are-good-actually-1834313812) )。相信我，我知道我在说什么。我是头号黄油迷。
