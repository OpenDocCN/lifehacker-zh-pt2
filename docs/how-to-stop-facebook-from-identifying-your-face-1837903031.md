# 如何阻止脸书认出你的脸

> 原文：<https://lifehacker.com/how-to-stop-facebook-from-identifying-your-face-1837903031>

如果你最近在脸书，你可能会注意到你的订阅中有一个新通知，解释了该公司如何使用其面部识别技术和管理附带的数据。作为其中的一部分，脸书已经重新命名了设置，如果你不喜欢这个概念，你需要知道如何关闭在照片中找到你的脸的功能。



该通知将发送给之前启用了脸书“标签建议”功能的用户，以及新用户。还有一个脸书官方博客如果你没注意到这个警告，并且你很想了解更多关于脸书人脸识别的政策。

正如博客文章所解释的，“标签建议”设置现在被更名为“面部识别”，以更好地体现该功能的功能。关闭它会禁用脸书的自动标记建议和照片查看功能，但您仍然可以手动在照片中标记(或被标记)。

## 如何在脸书的网站上查找和编辑面部识别权限

最近的通知包括一个快捷链接，直接将您带到面部识别设置，但也很容易找到您自己。

### 在桌面上

1.  登录。
2.  点击页面右上角的向下箭头图标，然后进入**设置>人脸识别。**
3.  点击“您是否希望脸书能够在照片和视频中认出您”旁边的**“编辑”**
4.  从下拉框中选择**“否”**禁用面部识别，然后点击**关闭**

## **如何在脸书应用中查找和编辑面部识别权限**

1.  打开脸书应用程序
2.  在主屏幕上，点击三个堆叠的点图标。
3.  向下滚动并点击**“设置&隐私”**
4.  进入**设置>隐私>人脸识别。**
5.  点击“您是否希望脸书能够在照片和视频中认出您”框。
6.  在下一个屏幕上，如果您想禁用该功能，请选择**“否”**，然后轻按几次**“返回”**箭头以返回应用程序主屏幕。
