# 如何使用 Alexa 有用的新日历功能

> 原文:[https://life hacker . com/how-to-use-alexas-useful-new-calendar-features-1830151503](https://lifehacker.com/how-to-use-alexas-useful-new-calendar-features-1830151503)

你的亚马逊 Alexa 设备现在可以更好地配合你的日历了。

亚马逊今天宣布了一些 Alexa 更新，特别是一个新的允许你浏览你的日历可用性，另一个允许你将你的日历纳入日常事务。

Watch

### 日历集成到例程中

现在，您可以将日历添加为例程中的一个动作。例如，如果您已经有了一个例行程序，为您提供当天的天气、交通和日历事件，现在您可以让该例行程序在特定时间自动为您播放，而不是要求您说出触发短语来开始事情。你可以在 Alexa 应用程序的例程部分设置你喜欢的时间来代替触发短语。

这里的想法是，你不必记得实际上说触发短语了。对于我们这些经常忘记，但仍然喜欢每天更新的人来说，这是一个坚实的优势。

### 浏览日历可用性

这是一个我个人非常兴奋的新功能。Alexa 已经可以告诉你你的日历上有什么，但现在她可以告诉你什么时候你有一些空缺。例如，如果你正在和一位同事谈论安排一次会面，你可以问“Alexa，星期三我什么时候有 30 分钟时间？”如果你的时间表排得很满，你也可以问一些更宽泛的问题，比如，“Alexa，我这周什么时候有空？”

### 提醒 API

最终的更新更多的是针对开发人员而不是个人，但应该会带来一些积极的技能更新。现在，企业能够提醒顾客一些事件。例如，Kayak 可以提醒你有一趟即将到来的航班，NHL skill 可以让你知道你最喜欢的球队今晚有一场比赛。亚马逊和一些合作伙伴一起发布 API:Kayak，TV Guide 和 NHL。它也适用于亚马逊蓝图，所以你也可以将提醒整合到你自己建立的技能中。
