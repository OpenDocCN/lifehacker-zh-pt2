# Windows 10 2019 年 5 月更新中最好的新功能

> 原文：<https://lifehacker.com/the-best-new-features-in-windows-10s-may-2019-update-1834927597>

微软今天正式推出了 Windows 10 2019 年 5 月更新，供所有人使用——也就是说，所有已经几个月没有进行测试的人。获取所述更新是容易的。单击开始按钮，键入“更新”，单击弹出的“检查更新”选项，然后在随后的屏幕中单击不容错过的“检查更新”按钮。如果你看到一个“Windows 10，version 1903”更新的列表，那就是你想要的那个。

虽然我预计这次更新不会有任何灾难(不像以前的版本)，但那些不愿意冒险的人也应该考虑在安装 之前备份他们的系统。我一直都想，但是我太兴奋了。

随着下载逐渐进入您的系统并开始安装，下面快速浏览一下 2019 年 5 月发布的版本。

### Windows 沙盒

最后，微软将这一令人敬畏的安全功能融入了 Windows 10。前提是完美的:你在一个隔离的桌面环境中运行应用程序，因此如果它们做了什么坏事，它们不会搞乱你的实际电脑。然而，理论上，这项功能对大多数人来说并不像 T1 那么有趣，因为微软只为 Windows 10 Pro 或 Windows 10 Enterprise 用户发布了这项功能。(不是针对个人，Windows 10 家庭用户。你需要能够通过操作系统利用虚拟化，而你那微不足道的 Windows 版本并不支持。)

不过，别担心。对于那些受到微软战略阻碍的人，总有第三方应用程序。也是免费的！否则， [开始使用 Windows 沙盒](https://techcommunity.microsoft.com/t5/Windows-Kernel-Internals/Windows-Sandbox/ba-p/301849) 需要一点跑腿的工作，但是花这些时间是值得的。

### 走向光明(主题)

我对任何操作系统、应用程序或网站中的黑暗主题都很着迷，所以这一个让我有点痛苦。微软已经在 Windows 10 的五月更新中打包了一个新的 light 主题。访问它很容易——尽管您可能不想使用默认的“主题”

单击开始按钮，输入“个性化”，然后单击出现的“主题和相关设置”选项。看啊！是主题！点击“Windows (light)”立即切换到新的外观(和壁纸)。我喜欢让我的论文保持原样；如果您同意，请点击左侧边栏上的“颜色”。

在这里，您将看到为 Windows 10 操作系统的各个部分设置“亮”和“暗”模式的几种不同方式。“选择您的颜色”选项允许您在系统范围内选择设置浅色或深色。如果你想得到具体的，选择“自定义”你现在可以为 Windows 10 的系统 UI *和*应用选择你想要的风格了。

例如，将前者设置为“暗”，后者设置为“亮”，会给你一个黑色的任务栏和开始菜单，搭配一个白色的文件浏览器和设置屏幕。你可以随意搭配。(为了一致起见，我认为全系统的“选择你的颜色”选项创造了最好的外观，但这只是我的看法。)

### 暂停未来的 Windows 更新

这个很奇怪，但是听我说完。Windows 10 Pro 或 Enterprise 的用户可以选择暂停所有 Windows 更新一段时间(35 天，截至我写这篇文章时)。在这段时间结束时，你被迫安装你一直推迟的任何更新，然后才能重新启用延迟机制。

对于企业用户来说，这是一个很好的选择，因为在你到处推广 Windows 10 更新之前，你可能想确认关键系统不会受到重大 Windows 10 更新的负面影响。对于大多数家庭用户来说，这是一个可怕的选择——至少，我是这么认为的，直到微软一遍又一遍地搞砸了 [前面提到的 2018 年 10 月更新](https://gizmodo.com/microsoft-says-its-fixed-that-windows-10-october-update-1829642118) 。

现在，微软为所有 Windows 10 用户提供了一种快速简单的方法来暂停 Windows 更新七天。在 Windows Update 屏幕上寻找正确的选项。你不会错过的。

点击它，你将延迟一周更新。再点一下，可以再加一周。您可以将其混合长达五周的时间，与 Windows Pro 和企业用户通过其(以前的)独家高级选项获得的时间相同。

### 卸载更多垃圾应用

典型的 Windows 10 安装会将各种你可能不需要或不想要的应用打包到你的系统中。你可以从一开始就卸载其中的一些软件——Skype、Weather、OneNote 等等——但不是全部。虽然你仍然无法安装一个准系统版本的 Windows 10 而不占用任何垃圾或营销主题的应用程序，但由于 5 月份的更新，你*确实*有机会卸载更多的应用程序。正如 [保罗·瑟罗特](https://www.thurrott.com/windows/windows-10/199366/here-are-the-best-improvements-in-windows-10-version-1903) 今年早些时候所写的，现在有资格获得数字砧板的应用包括:

*   3D 浏览器
*   计算器
*   日历
*   凹槽音乐
*   邮件
*   电影和电视
*   绘制 3D
*   剪切和草图
*   便笺条
*   话音记录器

很有趣，音乐很棒。

### 复制到云剪贴板

我知道，我知道。没有什么比 Windows 10 剪贴板更性感了。但是，实际上，有些东西*比默认的*更性感，你从来没有真正想过的隐形剪贴板。(不管怎么说，我们当中有多少人在完成了**CTRL+C**之后，还做了比 **CTRL + V** 更多的事情？)

微软在 5 月大更新中更新了 Windows 10 的云剪贴板的 UI。我敢打赌，你甚至不知道 Windows 10 *一开始就有*基于云的剪贴板。(奇怪的是，默认是关机的。)

要改变这一点，打开你的设置应用，然后导航到“系统”和“剪贴板”打开“剪贴板历史”以及“跨设备同步”选项。现在，如果你用 Windows 键+ V 键而不是 **CTRL + V 键**，你会得到一个小的弹出窗口，在这里你可以选择你想要粘贴的内容——可以是你刚刚复制的内容，也可以是你之前复制的内容。“同步”功能是不言自明的，它也应该与 [Android](https://play.google.com/store/apps/details?id=com.touchtype.swiftkey&hl=en_US) 上的相关 SwiftKey 应用程序一起工作。(目前还不知道这项功能是否会在 iOS 上实现。)

### 找到完美的符号

我曾经讨厌在我的文章中加入一个破折号，而不得不在键盘上键入一个 ALT 码。现在我已经换成了 tenkeyless 键盘，这个过程就更烦人了——我必须设置一个热键组合才能访问这个可爱的“，”直到我重新启动系统，它最终都会出错。

现在多亏了 Windows 10 月更新，我可以打 **Windows 键+；**弹出一个更新的表情符号面板，现在包含了一堆符号。我将展示它的力量:🙌Ă' ↯Ω.

### 让你的鼠标漂亮(或疯狂-巨大)

这是我在 Windows update 中看到的古怪功能之一，但我喜欢它。Windows 10 五月更新现在允许用户自定义他们的鼠标指针——我知道是野生的。听我说完。调出设置，然后点击轻松访问。点击“光标和指针”，你可以调整一些参数，包括它的大小和颜色。想要一个巨大的绿色霓虹灯鼠标指针？为什么不呢？狂野地生活。
