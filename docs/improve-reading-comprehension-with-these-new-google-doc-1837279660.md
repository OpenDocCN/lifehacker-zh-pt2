# 使用这些新的谷歌文档字体提高阅读理解能力

> 原文：<https://lifehacker.com/improve-reading-comprehension-with-these-new-google-doc-1837279660>

当你在写文档时，要考虑到你的读者可能很难理解你键入的内容——这个问题影响到 70%的普通人群。你不是一个糟糕的作家；这可能只是你的字体选择。

Watch

Google Docs 现在支持“Lexend”系列字体，该系列字体由 Thomas Jockin 和教育治疗师 Bonnie Shaver-Troup 设计，旨在通过使用改进形状、大小和间距的字体来帮助解决阅读流畅度(速度、准确性和表达)的问题。

虽然当你选择了符合他们阅读速度的字体版本时，你的读者将获得最大的好处，但你需要进行“”的流畅测试来优化，如果你想让读者更容易处理你的文档，切换到 Lexend 仍然是一个很好的选择。W 有七种不同的 Lexend 字体可供选择，对于希望尽可能提高孩子阅读理解能力的父母来说，这是一个不错的选择。

如果你有兴趣了解更多关于 Shaver-Troup 的研究，该研究导致了 Lexend 的*，我推荐查看这份 [白皮书](https://revreading.org/wp-opt-perf/) ，它更详细地介绍了她的阅读能力工作。Jockin 和 Shaver-Troup 一起建立了 Lexend 字体家族，他最近在 [Medium](https://medium.com/type-thursday/fonts-are-harming-7-out-of-10-people-this-one-font-can-change-this-injustice-83d61e3ba3f9) 的一次采访也值得一读:*

> *...教育系统是建立在假设学生在三年级之前学习如何阅读的基础上的。从那时起，阅读被认为是学习的，现在阅读是学习的辅助工具。但是，如果学生没有获得这种阅读技能，会发生什么呢？他们被留下来照顾自己，或者被归类为发育障碍，因为他们阅读困难。阅读能力和智力被混为一谈，因为教育领域不理解印刷术的物质条件对阅读能力和理解的影响。”*
