# 使用 AltStore 安装 NES 模拟器和替代 iOS 应用程序

> 原文：<https://lifehacker.com/install-nes-emulators-and-alternative-ios-apps-with-alt-1838631801>

作为一个使用 Android 作为主要移动操作系统的人，我倾向于关注谷歌产品 的 [替代应用/服务，但这并不意味着我不喜欢 iOS 和 iPad 上苹果应用的第三方替代品。不幸的是，苹果对其产品的“围墙花园”方法意味着 iOS](https://lifehacker.com/the-best-competitors-to-every-first-party-google-app-an-1834172092) [上可用的第三方和/或开源应用程序比 Android](https://lifehacker.com/the-comprehensive-guide-to-quitting-google-1830001964) 上的要少得多——而且大多数都需要你破解你的 iPhone，这是一个乏味的、违反保修的项目。

然而，软件开发商 Riley Testut 新发布的开源应用商店项目，恰当地命名为 [AltStore](https://altstore.io/) ，可以通过一个巧妙的变通方法下载并安装在你的 iOS 设备上。

AltStore 相当新，只包含 Testut 在发布时开发的少量软件，但这包括 Testut 的 Delta emulator 应用程序，可以让你在 iPhone 上玩 NES 游戏(以及 SNES、任天堂 64、Gameboy 和 Gameboy Advance 游戏)。未来将会有更多的应用程序被添加到 AltStore 中，但这是一件大事；仿真软件在官方应用商店是被禁止的，过去不越狱的话几乎不可能安装在 iPhones 上。

AltStore 能够通过利用一些 iOS 漏洞而存在，例如通过 Mac/Windows 应用程序将软件下载到手机的能力。这是一个聪明但不稳定的解决方案，苹果可能会在某个时候轻易关闭整个系统。由于这种潜在的可能性，如果你对 AltStore 提供的东西有一点点兴趣，立即安装它可能是明智的。开发者可以 [查看 AltStore Github 页面](https://github.com/rileytestut/AltStore) ，如果他们很好奇，想了解更多关于这个项目的信息。

### 如何安装 AltStore 和 AltStore 分布式应用程序

安装 AltStore 比破解你的 iPhone 要容易得多(也安全得多)，但比你下载普通的 App Store 要花更多的精力。尽管如此，一旦你把所有的部分都准备好了，整个过程是快速而轻松的。这是你需要的:

*   你的 iPhone。
*   运行 Mac 或 Windows 10 的 PC(Windows 10 版本的软件在技术上仍处于测试阶段。)
*   连接两者的 USB 电缆。

完成所有设置后，您现在可以在 iPhone 上安装 AltStore:

1.  将你的 iPhone 插入你的电脑。
2.  前往[https://altstore.io/](https://altstore.io/)并为您的 PC 操作系统安装 AltServer 软件。
3.  运行安装程序，然后在您的计算机上打开 AltServer 应用程序。
4.  我们现在将通过 AltServer 将 AltStore 应用程序发送到您的 iPhone。在 Mac 上，单击工具栏中的 AltServer 图标打开下拉菜单，然后单击**“安装 alt store”**并单击您的设备。在 Windows 10 上，从图标托盘中点击 AltServer 图标(你可能需要点击**“显示更多图标”**，然后将鼠标悬停在**“安装 alt store”**上，选择你的设备。
5.  等待 AltStore 应用程序安装到您的 iPhone 上。
6.  保持手机连接和 AltServer 在 PC 上运行，打开 AltStore 应用程序开始下载应用程序。
7.  与从 App Store 安装应用程序不同，您需要将手机连接到 PC，并运行 AltServer 来安装新应用程序，并每隔几天更新一次应用程序，因此请确保不要从 PC 上卸载 AltServer。
