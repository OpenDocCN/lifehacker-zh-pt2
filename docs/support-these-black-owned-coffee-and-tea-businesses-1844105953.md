# 支持这些黑人拥有的咖啡和茶叶生意

> 原文：<https://lifehacker.com/support-these-black-owned-coffee-and-tea-businesses-1844105953>

即使你习惯了——并且忠诚于——一家特定的咖啡连锁店，也可以扩大你的视野，尝试一家当地的独立咖啡店。既然你这么做了，为什么不把它变成黑人的生意呢？如果你正在你的地区寻找一个新的地方喝咖啡或喝茶(或者知道当你最终再次旅行时去哪里)，这里有全国各地许多黑人拥有的咖啡店。

Watch

### [啜一口，细细品味，芝](https://www.sipandsavorchicago.com/)T3】

这家风城迷你连锁店不仅有来自世界各地的经过认证的公平贸易咖啡，还有各种茶、摩卡、拿铁和混合饮料可供选择。在芝加哥南区有四个地点，分别是布朗斯维尔、海德公园、罗森沃尔德和南环路。

### [创业北，明尼阿波利斯](http://www.venturenorthbwc.org/coffee.html)T3】

Venture North 不仅仅是一家咖啡店:它还是一家自行车店。如果这还不够，它也是一个非营利组织，通过自行车项目、课程和工作来帮助社区中的年轻人。

### [A.T .绿洲咖啡&茶叶店，凤凰](https://atoasiscoffee.com/)T3】

这家店专注于小批量的埃塞俄比亚咖啡，以及奶昔、奶昔和即食食品。

### [【波比叔叔的咖啡和书】，费城](https://www.unclebobbies.com/)

虽然咖啡店部分业务目前已经关闭，但你仍然可以通过他们的网站订购书籍和其他物品，以帮助维持运营，直到它可以再次提供 java 服务。

### [【新奥尔良咖啡】](https://www.backatownnola.com/)

如果你喜欢咖啡、爵士乐和户外用餐，Backatown Coffee 是一个不错的选择。这里有免费 WiFi，所以一旦一切恢复正常，这可能是完成一些工作的好地方。

### [保险箱咖啡馆，华盛顿特区](https://coffycafe.com/)

除了咖啡和茶，Coffy Café还以其薄饼闻名。虽然你现在不能亲自享用，但是可以送货上门。

### [、纽约市](https://www.instagram.com/brooklynperkcoffee/)布鲁克林公园

如果你发现自己在布鲁克林需要一些咖啡，这是去的地方。想要点不一样的吗？从红色天鹅绒咖啡到杏仁牛奶香蕉拿铁，再到一种名为“独角兽”的东西(由抹茶、红色 shiso 和苹果制成)，菜单上有适合每个人的东西。

### 其他位置

这只是这个国家黑人拥有的咖啡店和茶馆的一小部分。还有 [几个](https://www.delish.com/food/g32872385/black-owned-coffee-shops/) [其他的](https://shoppeblack.us/2020/05/black-owned-coffee-brands-online/) ( [更全面的](https://sprudge.com/support-these-black-owned-coffee-companies-updating-164145.html) ) [一一列举](https://www.newsweek.com/black-owned-coffee-shops-us-alternatives-starbucks-1510435) 有哪些 [值得](https://ny.eater.com/2020/6/5/21281798/nyc-black-owned-restaurants-list-guide) 一看。也许你会找到一个新的喜欢的地方。
