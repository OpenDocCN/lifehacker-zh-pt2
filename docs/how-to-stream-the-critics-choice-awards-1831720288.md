# 如何获得评论家选择奖

> 原文：<https://lifehacker.com/how-to-stream-the-critics-choice-awards-1831720288>

评论家选择奖将于今晚 7 点播出。与金球奖一样，颁奖典礼一直是奥斯卡最佳预测者之一。今年的节目由泰雅·迪格斯主持，将于美国东部时间晚上 7 点在 CW 电视台正式开播。



如果你有一台电视机，那么最简单的方法就是使用你的有线电视盒或者通过数字天线无线收听。如果你订阅了有线电视，并且刚好不在家，你也可以从 CW 的网站上观看颁奖典礼。

如果你没有电视或有线电视，那么你仍然有一些流媒体选择。

不幸的是，无论是 Sling 还是 PlayStation Vue 都没有提供 CW，所以如果你想使用其中任何一个，你今晚的节目就要倒霉了。

也就是说，Hulu With Live TV、YouTubeTV 和 DirecTV 现在都提供了播放 CW 的功能。如果你还没有尝试过这三个中的任何一个，你可以注册一个免费的试用版，它将让你度过今晚的颁奖典礼和下周，而不用支付一分钱。一周后，这三项的费用是每月 40 美元。

值得注意的是，这三种服务在大多数市场上都有网络,但并非所有市场都有。在你注册之前，确保你所在的地区是提供 CW 的地区之一。
