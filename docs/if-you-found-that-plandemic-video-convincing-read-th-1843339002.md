# 如果你觉得这个“规划的”视频很有说服力，也可以看看这个

> 原文：<https://lifehacker.com/if-you-found-that-plandemic-video-convincing-read-th-1843339002>

外面有一个视频声称显示一名科学家被政府压制。我一直看到人们一边分享，一边说“你觉得这个怎么样？”或者“我不知道这样说对不对，但值得一听。”如果你想了解更多，但又担心被宣传所欺骗，我为你写了这篇文章。



该视频是一部名为 [Plandemic](https://plandemicmovie.com/) 的即将上映的电影的“第一部分”，由制片人和病毒学家朱迪·米科维茨之间的一次采访组成。她说新冠肺炎·疫情不知何故是在实验室里被创造出来的，或者是被允许故意传播的(她的故事有点变化)。这里有一些微妙的事实，但几乎所有可行的建议——例如，是否戴口罩或接受疫苗——都没有得到支持，这很危险。如果你不知道该不该相信这段视频，或者视频中的哪些信息是真的，哪些是假的，我们将为你一一解析。

### 更大的真理

在我们进入任何细节之前，让我们来谈谈这个视频的吸引力背后的假设。首先:在这个国家，金钱主宰一切。大型制药公司拥有比它应该拥有的更多的权力。

这个视频将美国政府描绘成不可信任的，这种说法我想每个人都可以认同，尽管不同政党的人可能对他们不信任政府的哪些部分有不同的想法。

我认为这个视频触及的最大事实是，我们，公众，听到了关于这个疫情的困惑和矛盾的信息。我们正在经历一些真正前所未有的事情，很多事情都没有意义。这是不可避免的——这确实是地球上的新事物。但这也让我们渴望确定性，没有人会因为想要寻找明确的信息而受到指责。

但是正如我们以前在这里讨论过的， [真正的专家讨论关于我们处境的事实的不确定性](https://vitals.lifehacker.com/how-to-tell-whos-full-of-shit-when-theyre-talking-about-1842293545) 。今天活着的人不可能有 100%的答案，所以我们应该对那些说他们能解释一切的人保持怀疑。

我们渴望确定性。我们可能更喜欢听到证实我们偏见的事实，或者让我们觉得我们可以控制世界上正在发生的事情。我们渴望任何能让我们生活中的这一刻感觉不那么怪异的东西。记住，如果你想让 T1 相信你所听到的，你应该提高警惕。

### 一些重要的背景

这个视频是专业制作的。看起来像纪录片。它花了将近十分钟的时间把 Mikovits 描述成一个在她的领域里被迫害和被压制的专家。当她开始谈论新冠肺炎的时候，我们已经开始了解她，并站在她的一边。

一些制作专业纪录片的人会在她做出声明时检查她的声明，并且会采访不止一个人以获得不同的观点，所以这是我们的第一个线索，事情可能不像他们看起来的那样。(我们已经联系了制作公司，询问他们如何审查她的说法，如果他们有回应，我们会更新这篇文章。)

关于媒体，最重要的问题之一是它来自哪里，是谁制作的，以及在两个方向上追踪资金。这是谁付钱的？谁会受益？

我希望在接下来的几周内，我们会了解到更多关于谁委托和资助了这个视频，但是到目前为止，这些信息很少被公之于众。我们知道接受采访的科学家 Mikovits 正在销售一本 [的书](https://www.amazon.com/Plague-Corruption-Restoring-Promise-Science/dp/1510752242/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/if-you-found-that-plandemic-video-convincing-read-th-1843339002&asc_source=&tag=kinjalifehackerlink-20) ，这是目前亚马逊上最畅销的书。

她的书是与小罗伯特·F·肯尼迪领导的组织“儿童健康防御”合作出版的。该组织的网站以反科学的阴谋论为特色，包括声称强烈暗示 [疫苗和手机信号塔会让儿童患上老年痴呆症](https://childrenshealthdefense.org/news/why-are-more-younger-adults-developing-alzheimers-and-dementia/) 。它的 [常见问题页面](https://childrenshealthdefense.org/about-us/frequently-asked-questions/) 称群体免疫是“荒谬的”,并表示疫苗是否能拯救生命是“有争议的”。这些完全是不真实的陈述。

那么这是反 vaxxer 的宣传吗？它当然似乎来自那个背景，但在视频中，采访者问 Mikovits 她是否是“反疫苗”的，她说不是。对于普通观众来说，这似乎令人放心。但是如果你花时间听了反对者的话，你就会知道这是一个巨大的危险信号。“我不反对疫苗，但是……”是科学否认者版本的“我不是种族主义者，但是……”

在我们进入 Plandemic 视频的内容之前，还有一点需要注意。有时人们问我我的钱从哪里来。当我写有争议的话题时，我被指责为各种利益的“托”。所以明确一点:我的收入大约 98%来自为这个网站写稿。其余的来自一本偶然的书版税支票。我的书是关于[基础遗传学](https://www.simonandschuster.com/books/Genetics-101/Beth-Skwarecki/Adams-101/9781507207642) 。我对有争议话题的立场不会影响销售数字。

[公司政策](https://g-omedia.com/editorial-policy/) 和 [职业道德](https://www.spj.org/ethicscode.asp) 禁止我允许我的故事的主题或来源影响故事本身，我绝对不能接受金钱来歪曲报道。在公司的广告部门和我这边的人——编辑——之间甚至有一道“防火墙”。在这场战斗中我没有参与。我只是希望人们健康，消息灵通，我跟随他们带我去的地方的事实。现在我正在与你分享它们。

### 米科维茨的背景故事有多少是真实的？

不偏不倚的报道讲述了一个与她在《规划》中讲述的背景故事完全不同的故事。如果你想更多地了解她，这篇最近华盛顿邮报的文章 很有用。

在她的职业生涯早期，Mikovits 发表了一篇被视频称为“重磅炸弹”的文章，声称慢性疲劳综合征是由病毒引起的。后来发现 [别的实验室无法复制她的结果](https://science.sciencemag.org/content/333/6038/35.1.full) ，她自己的实验室也无法复制。文章被 [收回](https://science.sciencemag.org/content/334/6063/1636.1) 。但是 Mikovits 继续争辩说，病毒会导致各种慢性疾病，包括自闭症，而且它们可以在疫苗中找到。她的说法没有证据。

她的故事的其他元素被视频中的讲故事夸大了(或者更糟)。《计划》中的一个场景显示了一个特警队，而米科维茨谈到了在没有逮捕令的情况下被逮捕——但镜头是来自一个不相关的事件 的 [，当时的一则新闻报道称她](https://onscene.tv/santa-ana-swat-team-raids-house-possibly-related-to-murder/) [在](https://mynews4.com/news/local/judy-mikovits-turns-herself-in) 自首。

她所参与的事件在当时的新闻故事和科学期刊中被广泛报道； [下面是 1985 年《科学》](https://science.sciencemag.org/content/230/4726/640) 上的一篇文章，内容是关于谁最先证实了艾滋病毒结果的争论。(没有理由相信她所声称的出版延迟会导致生命损失。)

别再说米科维茨了。她说的是真的吗？

### 新冠肺炎病毒有迹象表明是在实验室中制造出来的吗？

不会。冠状病毒有很多种，科学家们早就知道，偶尔，感染一种动物的病毒可以通过微小的基因变化感染另一种动物(如人类)。正如 Mikovits 所说，没有理由相信进化需要“800 年”。由于动物到人类的传播，出现了几种疫情病毒。SARS、MERS、伊波拉和多种疫情流感病毒株(包括 1918 年和 2009 年的病毒株)只是其中的几个例子，科学家几十年来一直警告说，总会有另一个病毒株即将到来。大卫·奎门 在 2012 年写了一本书，讲述了几个科学家团队研究这些事件，并试图预测和理解接下来可能发生的事情。

是的， [中国武汉有一个实验室，研究冠状病毒](http://english.whiov.cas.cn/) 。它成立于 SARS 疫情(也是由冠状病毒引起的)之后，是的，美国资助了它，因为如果你对预防流行病感兴趣，你可能会做的一件事是资助对过去导致大规模流行病的同类病毒的研究。这没什么可疑的。

科学家们已经多次揭穿了“T2 T3”病毒是人工制造的说法。它的基因组显示了它与动物病毒的明确关系，而 [与来自武汉实验室的冠状病毒样本](https://www.businessinsider.com/theory-coronavirus-accidentally-leaked-chinese-lab-2020-4) 没有关系。

### 福奇博士真的是所有事情的幕后策划者吗？

米科维茨不断提到福奇的名字，但从未真正将他与她讲述的故事联系起来。此人经营着一个 [硕大的](https://oma.od.nih.gov/IC_Organization_Chart/NIAID%20Organizational%20Chart.pdf) 政府机构；他又不是在她隔壁的办公室里亲自干涉事情。

在整个视频中仔细听他的名字，你会明白我的意思。它从来没有真正连接到任何东西。例如，Mikovits 声称福奇精心策划了一场掩盖(尚不清楚的真相)，称人们“得到了巨额报酬”听起来很诡异。但她随后澄清说，她的意思是研究人员的实验室得到了 NIAID 的资助。这不是回报。这是很多科学得到资助的正常方式。如果有任何理由相信在资金分配中有什么粗略的事情发生，我们还没有看到任何证据。

### FDA 和大型制药公司禁止廉价药物作为赚钱策略吗？

Mikovits 声称，制药公司只对他们可以申请专利并收取大量费用的药物感兴趣。诚然，制药公司的商业行为包括抬高药品价格，专利也有所帮助，但遗憾的是，如果她认为制药公司需要与美国食品和药物管理局合谋，从市场上撤出廉价药物来实现这一目标，那么她所知甚少。如果一种药物有效，最明显的举措就是享受需求的增长，并收取更高的费用。

我是说，我们都知道。还记得 daraprim 吗，这种药的价格被 Martin Shkreli 臭名昭著地抬高了？近 700 美元一粒，零售价(少用一张优惠券，但那是一个 [整另一个球拍](https://vitals.lifehacker.com/drug-pricing-is-a-mess-and-we-re-all-paying-for-it-1758338047) )而且根本没有专利。肾上腺素也没有专利，但是看看肾上腺素笔的价格。

收取一吨毒品的费用和往常一样，完全合法。不需要阴谋论来解释。

至于她具体的说法:Mikovits 提到了一种廉价的、古老的药物 suramin，据说可以治疗自闭症。那次试验的实际结果是 [而不是她描述的](https://www.spectrumnews.org/news/doubt-greets-reports-suramins-promise-treating-autism/) 。

她还提到羟氯喹是一种正在被压制的潜在新冠肺炎治疗方法。实际上， [现在有 169 个试验正在进行中](https://clinicaltrials.gov/ct2/results?cond=COVID&term=hydroxychloroquine&cntry=&state=&city=&dist=) 。它被广泛用于适应症外，以至于需要它治疗狼疮等其他疾病的人的供应处于危险之中。尽管她在视频中声称， [并没有任何确凿的证据证明它有效](https://vitals.lifehacker.com/chloroquine-is-not-a-ready-to-go-wonder-drug-for-covid-1842471341)——只是希望如此。

这里有一个事实:生产 remdesivir(目前最有前途的新冠肺炎药物之一)的公司几乎肯定会收取不合理的每剂 [费用，因为他们可以](https://www.reuters.com/article/us-health-coronavirus-gilead-pricing/will-gilead-price-its-coronavirus-drug-for-public-good-or-company-profit-idUSKBN22I2NM) 。不需要阴谋:这就是我们系统建立的方式。任何认为大型制药公司需要阴谋来赚钱的人都没有注意到大型制药公司实际上是如何运作的。

### 戴口罩会让我们更容易感染冠状病毒吗？

不。视频上说口罩“激活”了病毒。证据在哪里？没有了。(顺便说一下，这是脸书在删除视频时作为错误信息引用的一个具体说法。该公司 [告诉《纽约时报》](https://www.nytimes.com/reuters/2020/05/07/technology/07reuters-health-coronavirus-tech-video.html) “暗示戴口罩会让人生病可能会导致迫在眉睫的伤害，所以我们删除了这段视频。”。)

### 呆在室内会削弱我们的免疫系统吗？

不。为什么会呢？有一种理论认为，儿童在成长过程中接触微生物会影响他们成年后的免疫功能，但目前还没有发现当我们呆在室内时免疫力会下降的现象。无论如何，我们的房子里有大量的微生物

顺便说一下，这部分视频中引用的医生是加州贝克斯菲尔德的丹尼尔·埃里克森博士和阿丁·马西博士。Plandemic 让这两位看起来像是在代表许多医生说话，分享常识性的观察，但事实并非如此。美国急救医学学会(AAEM)和美国急救医师学会(ACEP)发布了一份声明，其中他们“联合强烈谴责”医生的视频(在 Plandemic 发布之前自行发布)发布错误信息。

### 流感疫苗是否会让你更容易感染冠状病毒？

据我们所知没有。没有关于流感疫苗和新冠肺炎的研究，研究流感疫苗对感冒(其中一些是由冠状病毒家族的其他成员引起的) [的影响的研究发现了混合的结果](https://healthfeedback.org/claimreview/claim-that-flu-vaccine-increases-coronavirus-infection-is-unsupported-misinterprets-scientific-studies/) 。

### 新冠肺炎的死亡人数被夸大了吗？

很难知道真实的数字，但由于测试仍然没有广泛使用，我们更有可能低估而不是高估。

不过，这是一个简单的问题。你不需要测试结果就能看出，自疫情开始以来，在每个国家的 中，各种原因导致的死亡都在飙升。

### 如果医院诊断出一个新冠肺炎患者或者给他们戴上呼吸机，他们会得到更多的钱吗？

是的，因为医院对需要更多资源的治疗收费更高。住院的新冠肺炎患者往往比其他疾病的患者病情更重，住院时间更长。当 Mikovits 谈到医疗保险为新冠肺炎病人给医院 13000 美元时，那不是礼物或奖金。这是他们护理的价格标签。

还有另外一件事，这是所有阴谋论的一个缺陷:你如何让每一个制药公司和每一个 T2 政府参与诈骗？你怎么能让 T4 的一个政府在一个问题上形成统一战线呢？
