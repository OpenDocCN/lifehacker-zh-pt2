# 土豆泥是新的油炸面包丁

> 原文：<https://lifehacker.com/tater-tots-are-the-new-croutons-1839332942>

如果你熟悉我的作品，你就会知道，当谈到用土豆代替面包时，我不会感到不安。我喜欢经典的面包丁，但我认为没有必要将这种类型局限于一种碳水化合物。



就像任何好的油炸面包块一样，马铃薯块松脆而脆，但它们的内部也很柔软，而且这些内部是由土豆制成的。它们甚至比“普通的”油炸面包丁更容易制作，后者从一开始就不是 [和](https://skillet.lifehacker.com/how-to-make-croutons-with-whatever-stale-bread-you-happ-1819789828) 的挑战。按照包装袋上的说明来烹饪，确保每个蛋糕之间有足够的空间，这样就不会有人被蒸了。用铝箔衬你的烤盘，容易清理(和特别脆的底部)。

一旦蛋糕滚烫，变成褐色，把它们放在碗里，加入干香草、大蒜粉，或者——如果你是我或者任何像我一样的人——酸奶油和洋葱爆米花调料 。任何你想用作品调味的东西。把它们放在沙拉里，然后吃掉。任何沙拉都得益于托托，但一份 [早餐沙拉](https://skillet.lifehacker.com/the-case-for-eating-salad-for-breakfast-1796685555) 实际上是在乞求它。
