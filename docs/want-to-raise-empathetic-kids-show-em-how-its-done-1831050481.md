# 想培养有同情心的孩子吗？让他们看看该怎么做

> 原文：<https://lifehacker.com/want-to-raise-empathetic-kids-show-em-how-its-done-1831050481>

我儿子昨晚问我今天要写些什么。我刚刚在纽约时报 上读到这篇关于父母如何培养有同情心的孩子的文章，它引起了我的共鸣。作为一名家长，我非常想把我的儿子培养成那种在别人有需要时挺身而出的人，但这是我能真正影响的吗？



“我要写父母如何帮助他们的孩子变得有同情心，”我说。“你知道那个词是什么意思吗？”

他没有，所以我解释说，当你有同理心的时候，你就能理解和关心别人的感受。“这是人们想要互相帮助的原因，”我解释道。

“哦，你会写我吗？我感同身受，”他说。

这是一个 8 岁的孩子，刚刚加入了一个非营利组织 [的“少年委员会”，该组织为寄养在](https://www.onesimplewish.org/) 的孩子实现愿望。上周，在一个上学的晚上，我们来回开了三个小时的车，这样他就可以参加他的第一次“会议”他和他的董事会成员装饰了长统袜，并为一个圣诞派对打包了零食，这个派对是为了让那些在系统中被分离的兄弟姐妹们聚在一起而组织的。

同样在上周，他花了自己的一大笔钱来满足一个被看护儿童的愿望，并向学校的玩具捐赠活动捐赠玩具。

他一直是个敏感的孩子，天生对事物有很深的感受。但我也把定期练习布施作为我们家的头等大事。

我们是养父母，所以我的儿子亲眼目睹了这些孩子所遭受的创伤。他每年都和我一起为有需要的孩子买玩具。他把钱放在红色救世军的桶里，他把罐头食品带进食品捐赠处，他翻遍他的玩具来捐赠他不再玩的东西。

但是，我仍然想知道:你天生就有同情心，还是后天习得的？就像育儿的许多方面一样，我对着墙扔了一堆东西，希望其中一些能粘上来。

事实证明，成为一个善解人意的人是 [先天*和*后天](https://www.nytimes.com/2018/12/10/well/live/how-to-foster-empathy-in-children.html?rref=collection%2Fsectioncollection%2Fwell-family&action=click&contentCollection=family&region=rank&module=package&version=highlights&contentPlacement=2&pgtype=sectionfront) 的结合。我们确实对此有一些影响。

> (海伦)里斯博士和她的合作者的研究表明，我们每个人生来都有一定数量的神经元参与移情反应。但是，这种适当关心他人的潜力是实现了还是受到了破坏，很大程度上是由早期生活经历塑造的，从出生开始一直持续到整个童年。
> 
> 那么，怎样才能给孩子灌输健康程度的同理心呢？“同理心是一种可变的特质，它是可以被教授的，”里斯博士告诉我。“我们生来都有某种天赋，但根据环境因素，这种天赋可能会明显上调或下调，”她说，尤其是孩子的照顾者树立的榜样。

这并不意味着你必须跑出去成为养父母(尽管如果你这样做了，那是很棒的！).你可以从承认你孩子的情绪开始，表现出关心，并在你的社区中树立有益的行为榜样。

哈佛医学院(Harvard Medical School)精神病学副教授、《移情效应 的作者里斯博士说，这可能很简单，就像给生病的邻居送饭，或者当朋友或兄弟姐妹擦伤膝盖时，让你的孩子帮忙拿创可贴一样这本书为在孩子发展的每个阶段建立移情模型提供了指导。

> 里斯博士建议，对于年龄较大的孩子，父母可以带他们去施粥所帮忙，或者去养老院看看。她写道:“引导孩子更好地理解他人的感受，永远都不晚。”。
> 
> 同样重要的是，父母要表现出对自己孩子的同情，承认他们的关切和感受，并认识到他们对安全的需要。例如，她说，“当孩子害怕狗时，不要说‘别害怕，他不会咬你的’，而是说‘你害怕狗吗？“你害怕什么，”这证实了孩子的恐惧，而不是否认它们。"

然而，里斯博士也警告说,*有可能走得太远:*

> 父母不应该反应过度，不能容忍“孩子生活中哪怕一秒钟的不快乐”，以免这种被误导的同理心剥夺了孩子发展成功生活所必需的勇气、毅力和韧性。

我认为这意味着更多的关心，更少的溺爱。
