# 如何处理税务表格 1095-C

> 原文：<https://lifehacker.com/what-to-do-with-tax-form-1095-c-1832460735>

《平价医疗法案》要求适用的大型雇主——或全职雇佣至少 50 名员工的组织——向员工发送 1095-CT3[“雇主提供的健康保险报价和覆盖范围”。如果你已经为一家公司全职工作了至少一个月，除了 W2 到来的纳税时间(你可能已经有了)之外，你还会收到表格。](https://www.irs.gov/forms-pubs/about-form-1095-c)



虽然 ACA 对没有医疗保险的税收惩罚基本上不再有效，但你仍然需要报告你有保险的证明，至少在 2018 纳税年度。

这是因为 2017 年减税和就业法案从 2019 年开始将 [没有保险的罚款减少到 0 美元](https://lifehacker.com/the-parts-of-the-affordable-care-act-that-will-stay-ch-1793055229#_ga=2.99978461.2140879112.1549460470-594046802.1524762060) 。根据 [消费者报告](https://www.consumerreports.org/taxes/irs-form-1095-c-health-coverage/) 显示，今年你仍然在钩上。

雇主必须在 3 月 4 日之前把表格寄给你。如果你有 2018 年全年的雇主健康保险，你真的不需要担心它；更多的是供参考。

如果你没有参加雇主提供的保险，也没有其他必要的保险，你可能需要表格上的参考代码，在这种情况下，你可能欠 2018 年的税收罚款(2017 年的金额是每个成人 695 美元，加上每个 18 岁以下儿童 347.50 美元):

但是如果你通过市场购买保险，你需要等待 1095-A 表格来报税。

今年你报税后，你就不用再担心表格的问题了。
