# 今天起飞前检查航班延误情况

> 原文：<https://lifehacker.com/check-for-flight-delays-before-flying-today-1833714457>

今早去机场吗？在登机口安顿下来之前，你可能需要再次确认(甚至三次确认)你的航班起飞时间。



今天，包括美国航空、达美航空、捷蓝航空、西南航空和联合航空在内的主要航空公司报告了全系统的运营问题，导致一些滞留在停机坪上的乘客延误长达两个小时。

在一条推文中，美国航空公司表示，该问题涉及机场数据中断，软件 [测量重量和平衡](https://abc7ny.com/travel/technical-issue-delays-flights-for-multiple-airlines/5228469/) ，以便航班可以安全起飞。然而， [西南](https://www.cnbc.com/2019/04/01/southwest-delta-united-airlines-say-they-are-experiencing-systemwide-outages.html) 表示，问题已经解决，达美航空确认数据现在运行正常，没有取消计划。

尽管如此，作为对这一消息的回应，美国联邦航空局 的 [在推特上表示，乘客应该直接联系他们的航空公司，了解航班信息和任何必要的更新(佛罗里达州的](https://twitter.com/FAANews/status/1112681600788119553) [一个机场](https://abc7ny.com/travel/technical-issue-delays-flights-for-multiple-airlines/5228469/) 报告称，仅今天早上就有多达 55 次延误)。

去机场之前你应该做什么？尽快打电话给你的航空公司并下载你的航空公司的应用程序以获取任何更新。查看 Twitter 上的机场信息也不是一个坏主意——你可能会遇到一些愤怒的传单，上面有一些有用的信息。
