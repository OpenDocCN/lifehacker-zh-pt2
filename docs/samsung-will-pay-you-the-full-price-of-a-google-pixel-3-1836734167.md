# 如果你以旧换新，三星将支付你一部谷歌 Pixel 3 的全价

> 原文：<https://lifehacker.com/samsung-will-pay-you-the-full-price-of-a-google-pixel-3-1836734167>

三星正在为最新的三星 Galaxy、谷歌 Pixel 和 iPhone 智能手机提供慷慨的以旧换新优惠，以及大量旧款手机的交易。这包括支付你一部普通谷歌 Pixel 3 的全价——截至发稿时的 600 美元 。

我们说“掷骰子”，因为这些数字是折价物可以获得的最高可能值。与任何以旧换新计划一样，你的手机必须处于全新状态才能获得全部价值。如果有任何磨损的迹象，收到的实际折价退款将低于列出的价格，无论这些迹象有多小。

一旦你把你的设备发送到三星， [就是这样](https://www.samsung.com/us/trade-in/frequently-asked-questions/) 。只要三星愿意为*提供一些*价值，如果三星认为它不值得它宣传的全部价值，你就不能拿回你的折价物。Android Police 最初注意到了新的以旧换新优惠，他提出了重要的一点:三星过去在兑现这类以旧换新交易方面做得很差。

如果你愿意尝试三星的促销活动，你可以通过该公司新的“下一代” [Galaxy 预购页面](https://www.samsung.com/us/mobile/reserve/) 开始以旧换新过程。
