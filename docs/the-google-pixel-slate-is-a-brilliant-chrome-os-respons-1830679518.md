# 谷歌 Pixel Slate 是 Chrome 操作系统对 iPad 的杰出回应

> 原文:[https://gizmodo . com/the-Google-pixel-slate-is-a-brilliant-chrome-OS-respons-1830679518](https://gizmodo.com/the-google-pixel-slate-is-a-brilliant-chrome-os-respons-1830679518)

Chromebooks 不应该花很多钱。 [这是我们喜欢他们的一大原因](https://gizmodo.com/why-i-bought-a-chromebook-instead-of-a-mac-1778403065#_ga=2.237087750.768447776.1543244064-171684553.1542638922) ！它们总是足够好*，*，而且总是很便宜——想想不到 500 美元。但搭载最新版本 Chrome 操作系统的 Pixel Slate 为价格更高的 chromebook 提供了一个近乎完美的案例。这款平板电脑可以变成一台笔记本电脑，外加一个 160 至 200 美元的配件，起价 600 美元，无论是作为笔记本电脑还是平板电脑，它的表现都非常好，让人觉得它几乎总是物有所值。

通常情况下是不会这样的！谷歌有一个坏习惯，就是生产运行 Chrome 操作系统的设备，对于你所得到的东西来说太贵了。像 Pixelbook(售价 1000 美元)和更早的 Pixel Chromebooks 这样的 Chrome OS 设备的价格特别高。我们过去喜欢 Chrome OS 设备，因为它们是非常好的小电脑，价格非常低。花更多的钱买 chrome book——即使是像 Pixelbook 这样出色的产品——也不合理。

与 Pixel Slate 一起改变的是 Chrome OS 本身。它变得如此之好，以至于它可以很好地对抗 iOS，甚至在较小程度上对抗 Windows。缓慢但肯定的是，Chrome OS 正在成为一个配得上谷歌为其生产的昂贵设备的操作系统——即使 1000 美元对于 Chromebook 来说仍然太贵了。

这就是为什么我应该注意到我审查的具体像素板单元是 [、1000 美元的第八代 i5 版本，具有 8GB 内存和 128GB 固态硬盘](https://store.google.com/us/product/pixel_slate_specs?hl=en-US) 。1000 美元，我认为 Pixel Slate 不值得。对于大多数人来说，800 美元的 m3 CPU、8GB 内存和 64 GB 固态硬盘，或者更便宜的 700 美元的英特尔赛扬 CPU、8GB 内存和 64GB 固态硬盘，都值得考虑。

这是因为 Pixel Slate 的竞争对手不是其他 Chrome OS 设备，如 [Pixelbook](https://gizmodo.com/the-google-pixelbook-made-me-alter-my-existence-1819663762#_ga=2.70614807.603952265.1543323449-422248777.1525353160) 或 [价格实惠的三星 Chromebook Plus v2](https://gizmodo.com/samsungs-chromebook-plus-v2-is-an-incredible-budget-lap-1827675148) ，而是像 iPad Pro 和微软 Surface Pro 这样的平板电脑第一二合一设备，它们的起价分别为 800 美元和 900 美元。像这些设备一样，Pixel Slate 作为平板电脑出售，并额外提供键盘套。和那些设备一样，放在膝盖上用也没那么爽。(有一个可选的 160 美元的键盘附件，与官方 200 美元的键盘套相比，提高了笔记本电脑的性能，它只是使用了蓝牙，而不是 Pixel Slate 的磁性键盘连接器。)此外，就像那些设备一样，你必须提醒自己，如果你想在这该死的东西上打字，真正的成本要多 150 到 200 美元。

但这是 Slate 与 iPad Pro 和 Surface Pro 的区别:它真的可以作为平板电脑或笔记本电脑使用。另外两个没有！iPad Pro 是一款完美的平板电脑，但它的键盘外壳和对触摸输入的关注只能让它成为笔记本电脑的可怜替代品，Surface Pro 是一款非常酷的类似笔记本电脑的设备，但 Windows 在平板电脑使用要求的纯触摸模式下使用时仍然非常糟糕。与此同时，Pixel Slate 从专注于触摸的平板电脑模式过渡到专注于鼠标的笔记本电脑模式非常简单，你会奇怪为什么花了这么长时间才恢复正常。

我把这款设备作为平板电脑一次用了几个小时。当我将键盘向后翻转时，操作系统平稳地过渡到了触摸友好模式，我的所有 Android 和 Chrome OS 应用程序都可以通过点击获得。键盘一回到我的手指下，Pixel Slate 就回到了笔记本电脑模式，回到了更传统的 Chrome OS 鼠标和键盘布局。不像 Windows 设备，它从来没有，从来没有搞砸了过渡。我真的想把事情搞砸！

但 Pixel Slate 只是勇敢地在两种模式之间切换，它做得非常好，我发现自己与 Pixel Slate 的交互方式有所不同，甚至在鼠标可用的情况下也使用触摸模式。一般来说，当我在设备上处于笔记本模式时，我的手指很少伸出去触摸显示屏，而是停留在键盘和触控板上。在 Pixel Slate 上，我从来没有想过要快速点击显示屏来打开应用程序，或者向上滑动来访问应用程序抽屉或通知面板。将触摸和鼠标输入结合在一起感觉很自然，这要归功于 Chrome OS 即使在鼠标模式下也能保持图标较大且对触摸友好的方式——这是 touch Windows 10 应该考虑复制的设计。

当你想到像素板上的玻璃有多光滑时，这是令人印象深刻的。它有一个 12.3 英寸的显示屏，分辨率为 3000×2000 像素，每英寸像素为 293ppi，比最新的 iPad Pro 每英寸像素更多(11 英寸和 12.9 英寸都有 264ppi)或 12.3 英寸的 Surface Pro (267ppi)。显示屏很漂亮，但谷歌选择覆盖显示屏的玻璃出奇地滑，合上时键盘壳会滑向它，我的手指和 100 美元的 Pixelbook 笔滑过它的速度有点太快了。Surface 和 iPad 都选择了更有质感的玻璃，这样你的笔或手指就不会感觉像在滑冰。

性能方面，Pixel Slate 做得很好。像 Surface Pro 上那样的 CPU 密集型软件并不可用，我们用来测试笔记本电脑存储速度的完整版 Photoshop 也不可用。但这没关系，因为这些应用程序在 iPad Pro 上也不可用。现在可以使用的是 WebXPRT 2015，这是一个综合的基准测试，可以通过一系列通常在 web 浏览器中执行的任务来运行设备，如图像大小调整和电子表格工作。在那里，它的得分为 485，与运行第八代英特尔芯片的其他设备一致，略高于 Pixelbook 的 453，远远领先于 iPad Pro 的 381。我们尚未测试当前的 Surface Pro，当我们测试时会更新。

有趣的是，Pixel Slate 已经足够快了。它可以运行多个标签页，并向我的电视播放视频，而且速度永远不会变慢。就像一个好的平板电脑一样，它很简单。它的电池寿命也很长。当以 200 尼特的设定亮度播放 YouTube 视频时，我们测试的像素板持续了 10 小时 48 分钟。这仍然低于 iPad Pro——它持续了 12 小时 37 分钟，但远远好于 Pixelbook (8 分 53 秒)或我们之前最喜欢的 Chrome OS 设备三星 Chromebook Plus v2 (7 分 42 秒)。

所以我会推荐它而不是 iPad Pro 吗？对很多人来说是的！尤其是如果你正在考虑购买 iPad Pro，11 英寸的起价为 800 美元，12.9 英寸的起价为 1000 美元。Pixel Slate 的起价为 600 美元，在笔记本电脑和平板电脑模式之间的切换要流畅得多。这是一款价格低廉的平板电脑，作为一款坚固的工作设备，它的功能相当灵活——尽管我建议至少多花 100 美元从 4GB 内存升级到 8GB。Pixel Slate 很好地完成了平板电脑的工作，苹果应该注意到这一点。如果你是一个不依赖 Windows 的 Surface Pro 助手，那么 Slate 也值得一看。

至于那些只是想要一台好的 Chrome OS 设备，并想花比三星 Chromebook Plus 所需的 500 美元多一点的人...事情有点复杂。Pixel Slate 可能很棒，但它首先还是一个平板电脑！对于笔记本电脑爱好者来说，这毫无意义，尤其是考虑到去年的 Pixelbook 价格下降，起价仅为 700 美元。电池续航时间没有那么好，但 Pixelbook 肯定是笔记本电脑工作的优秀设备，而且如果考虑到 Pixel Slate 的键盘外壳的成本，它要便宜得多。但 Pixel Slate 并不是为笔记本电脑设计的。它被设计成一个功能最好的 2 合 1 平板电脑，在这方面，它做得非常好。如果你想要一台平板电脑，又不想和苹果打交道，那么这就是你想要的 Pixel Slate 适合你。
