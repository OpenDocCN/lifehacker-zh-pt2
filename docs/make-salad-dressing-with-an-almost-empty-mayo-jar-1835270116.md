# 用几乎空了的蛋黄酱罐做沙拉酱

> 原文：<https://lifehacker.com/make-salad-dressing-with-an-almost-empty-mayo-jar-1835270116>

说到在家制作调料，醋油沙司通常是最容易得到的选择。他们所需要的只是一点不太精确的测量、普通的厨房配料和一些摇动(最好在罐子里)。但是通过用一点蛋黄酱代替或补充你的色拉油，你可以在几分钟内做出奶油般梦幻的沙拉酱。顺便说一下，粘在蛋黄酱罐边缘的“最后一点”通常是最完美的量。



类似于我们如何清洗 [果酱罐](https://skillet.lifehacker.com/use-an-almost-empty-jam-jar-to-flavor-a-cocktail-1834080513) 和 [蜜熊](https://skillet.lifehacker.com/make-a-vinaigrette-inside-an-almost-empty-honey-bottle-1834980660) ，你所要做的就是向几乎空了的容器中加入配料，摇一摇，然后倒酒。考虑到“最后一点”并不是一个标准的衡量标准，我总是加入比我认为我需要的量更少的醋，然后品尝并调整以获得我想要的味道和粘度。至于香草和其他调味品，这几乎是你自己选择的冒险情况，尽管我喜欢为牧场调料添加一撮味精。要制作你自己的超级快速奶油调味酱，你需要:

*   一罐几乎空了的蛋黄酱
*   2 茶匙-1 汤匙雪利酒或香槟醋
*   1 茶匙蜂蜜
*   1 茶匙第戎芥末或石磨芥末
*   几撮新鲜切碎的香草
*   少许味精(可选)
*   一撮盐
*   胡椒调味

向罐子里加入两茶匙你想要的醋和所有剩余的配料。摇晃直到所有东西都充分乳化。如果调料太稠，再加一茶匙醋。如果太稀，加一点油。根据需要品尝和调整调味料，再次摇晃，然后放在一堆绿色蔬菜(或冰山楔)上。
