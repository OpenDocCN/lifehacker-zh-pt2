# 如果你住院或残疾，如何投票

> 原文:[https://life hacker . com/how-to-vote-if you-hospital-or-disabled-1830236833](https://lifehacker.com/how-to-vote-if-youre-hospitalized-or-disabled-1830236833)

如果你因为身体残疾无法到达投票站或面临额外的挑战，仍然有办法确保你的投票被计算在内。

事实上，你有受联邦保护的权利去做，而且有许多服务设立来帮助任何在这个过程中需要帮助的人， [据](https://www.forbes.com/sites/tarahaelle/2018/11/05/hospitalized-or-disabled-you-can-still-vote-and-receive-federally-protected-help/#fe4fd90756c2) 记者 Tara Haelle 为福布斯:

> 1965 年的《选举权法案》、1990 年的《美国残疾人法》、1993 年的《全国选民登记法》和 2002 年的《帮助美国投票法》保障了包括精神病患者在内的残疾人的投票权。随后的每一部法律都加强和扩大了这些权利，以确保每个有资格投票的人都可以不受侵犯其公民权利的障碍地投票。

精神疾病说明非常重要，因为许多人没有意识到他们有权帮助投票，即使他们正在与不太明显的健康问题作斗争，从焦虑发作到精神病。这种保护还包括“智力、发育或精神残疾”

Bazelon 心理健康法律中心出版的全国残疾人权利网络和自闭症自我辩护网络推荐了一份 [指南](http://electionaccess.org/en/resources/publications/458/) ，供任何想了解有心理健康问题的个人所拥有的具体权利的人使用。

### 谁能帮忙？

关于谁能帮你投票，只有两条规则:他们不能是你的雇主或工会领袖。但是你可以问一个朋友、同事，甚至一个陌生人。投票工作人员应该接受培训来帮助你，如果有人劝阻你或试图阻止你投票，他们就违反了联邦法律。您可以向全国选民热线 866-OUR-VOTE (1-866-687-8683)举报干扰。

### 在医院

如果你在医院，只要你是注册的，你仍然有投票权。一些医院有训练有素的工作人员提供帮助，但如果他们没有，你可以联系当地的选举委员会提供帮助。工作人员也可以代表你打电话——如果他们拒绝或做任何其他事情来阻碍你，他们也违反了法律。PatientVoting.com 网站对您的特定 [状态](https://www.patientvoting.com/) 有更详细的说明，还有紧急缺席投票的 [选项](http://www.ncsl.org/research/elections-and-campaigns/absentee-voting-in-case-of-a-personal-emergency.aspx) 。在健康紧急情况下，可能很难主张你的投票权，但如果你有毅力，没人有权阻止你。
