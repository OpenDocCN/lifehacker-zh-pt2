# 使用这些 Mac 应用程序从 YouTube 流式播放音乐

> 原文：<https://lifehacker.com/stream-music-from-youtube-with-these-mac-apps-1843304857>

在过去的几年里，谷歌让 YouTube 更难用作音乐播放器，原因显而易见。一些允许你在不看视频的情况下从 YouTube 上播放音乐的扩展和应用程序 [慢慢消失在以太](https://lifehacker.com/how-to-turn-youtube-into-the-ultimate-streaming-music-s-1624383344) 中。与此同时，谷歌开始推其 [YouTube 音乐](https://music.youtube.com/) 服务——类似的功能，只是不免费。

然而，除了简单地在浏览器中加载 YouTube 音乐之外，你还有一些选择可以提供更多的功能。让我们快速浏览一下 YouTube 音乐的三个最受欢迎的应用和扩展。

### [VOX 音乐播放器](https://vox.rocks/) (macOS)

[**Vox**](https://vox.rocks/) 是一款用于 Mac 的 [高级音乐播放器](https://lifehacker.com/vox-is-a-powerful-but-minimal-music-player-for-mac-1040029727) 专为那些想要播放无损音乐而不是压缩 MP3 的人设计。我们将使用它的秘密武器:它能够播放来自任何给定链接的音乐，包括，你猜对了，YouTube。

要使用该功能，您不需要登录 Vox Cloud 或为该应用的高级服务付费。打开 Vox app 后，使用键盘快捷键 **Command + U** 调出一个文本框。只需粘贴你想听音乐的视频的 YouTube 链接，然后按回车键。

Vox 将立即开始播放 YouTube 链接上的音频。您可以重复这一过程，将更多的 YouTube 曲目添加到您的队列中，但这可能会很繁琐。取而代之的是，把你想听的所有视频放到 YouTube 播放列表中，然后把那个链接复制到 Vox 中，把整个视频添加到你的队列中。

一旦你的队列建立起来，选择所有的曲目并点击右键打开上下文菜单。现在，转到“复制到收藏”部分，并选择“创建新收藏”选项。

您可以将 Vox 收藏视为播放列表。创建后，它们将显示在“收藏”选项卡中。现在，您已经从当前队列中创建了一个收藏，您可以返回 Vox，只需导航到收藏菜单，就可以开始播放 YouTube 上的音乐或音频，而无需打开 YouTube 网站或 YouTube 音乐应用程序。您可以重复这个过程来创建多个 c 收藏。

作为一个音乐播放器，Vox 的功能非常丰富。您会发现菜单栏快捷键、均衡器、标准回放控制等等。然而，要获得对媒体控制键的支持，你必须安装 [VOX 媒体控制扩展](https://vox.rocks/mac-music-player/control-extension-download) 。这将使 VOX 与键盘和耳机上的媒体键硬链接。即使您正在使用其他媒体应用程序(如 iTunes)，媒体键也将始终适用于 VOX。

### [YT 音乐 App](https://ytmusic.app/) (macOS)

免费开源的 [**YT 音乐应用**](https://ytmusic.app/) 是你在 Mac 上能获得的类似 Spotify 的体验。是的，它仍然是 YouTube 音乐的网页包装器，但它比 YouTube 音乐的 Chrome PWA 应用程序做得好得多。

首先，你可以独立于 Chrome 使用该应用程序，它支持媒体键、键盘快捷键、触摸栏控制和新曲目的通知。此外，YouTube 音乐的所有常规功能都存在并可用。

就我个人而言，我是 YT 音乐应用程序的 Dock 图标选项的粉丝。当你右击应用程序的 Dock 图标时，你可以快速播放音乐或跳到下一首曲目。当你将媒体键映射到 Vox 等另一个应用程序时(或者当 Chrome 中的视频接管它时)，这个小功能就派上用场了。

### [YT 音乐迷你](https://chrome.google.com/webstore/detail/yt-music-mini-for-youtube/jcdmcpefbebkldopabclefjaggejocff) (Chrome)

如果你用 Chrome 听 YouTube 音乐，可以考虑抢一下[**YT Music Mini**](https://chrome.google.com/webstore/detail/yt-music-mini-for-youtube/jcdmcpefbebkldopabclefjaggejocff)扩展，让你的体验好一点。不用使用 YouTube 音乐网站来控制你的播放，你可以使用这个扩展来播放/暂停，跳到上一首和下一首曲目，并调整音量。您还可以查看您的队列并切换到另一首曲目，甚至可以在播放您最喜欢的歌曲时朗读歌词。
