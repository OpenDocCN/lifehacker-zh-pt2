# 使用 Alexa 来找出什么啤酒和葡萄酒来搭配你的晚餐

> 原文：<https://lifehacker.com/how-to-use-alexa-to-figure-out-what-beer-and-wine-to-pa-1841561679>

当你做饭时，至少在我家，一个经常出现的问题是我们打算用它喝什么。



虽然有些搭配相当明显，比如浓郁的红酒配牛排，或者 IPA 配油腻的披萨，但其他搭配就有点难以捉摸了。例如，你用拉面或者你妈妈发给你的柠檬鸡肉意大利面做什么？

当你想要在配对时仔细检查你的检查或者不知道从哪里开始时，Alexa 可以是一个可靠的朋友。

有很多不同的技巧可以帮助你找出从火鸡到西班牙肉菜饭的正确搭配。两个最好的，在我看来，还有 [寻酒器](https://www.amazon.com/Bloop-Entertainment-Wine-Finder/dp/B07CLH4WF3?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-use-alexa-to-figure-out-what-beer-and-wine-to-pa-1841561679&asc_source=&tag=kinjalifehackerlink-20) 和 [酿酒台](https://aax-us-east.amazon-adsystem.com/x/c/Qq-liQQScRg9mgj0ueso1DYAAAFwLAyvqwEAAAFKAZhzjGA/https://www.amazon.com/ricardohdz-Brewers-Table/dp/B01HAKRFR8/ref=sr_1_2?creativeASIN=B01HAKRFR8&linkCode=w61&imprToken=2917KKLUQTAeBSdHs4fJYQ&slotNum=0&ascsubtag=b70615cc59238ab406d8bc7561a79c1801a6edad&ie=UTF8&keywords=beer%20pairing&qid=1535937732&s=digital-skills&sr=1-2&tag=lifehackeramzn-20) 。对于这两种情况，你只需询问该技能你正在准备的菜肴配什么啤酒或葡萄酒，它就会给出一个有根据的建议，告诉你应该拿出什么样的搭配。

我要坦率地说，两者都不完美。虽然两者都可以轻松处理大量的日常食物，但如果你倾向于吃一些偏离典型美国食物的食物，也很容易难倒两者。

在这种时候，我建议考虑一种可能有类似风味的更基本的食物，并询问一下。尤其是品酒技巧，对于它不知道的大部分事情，都会建议用霞多丽。虽然这可能没有错，但霞可以搭配很多东西，我也不会认为这是一种彻底的成功。

也就是说，如果你在做饭，但没有头绪，两者都可以提供一些相当不错的建议，帮助你找到正确的方向。
