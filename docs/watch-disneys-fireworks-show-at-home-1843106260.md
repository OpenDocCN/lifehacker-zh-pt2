# 在家看迪士尼的烟火秀

> 原文：<https://lifehacker.com/watch-disneys-fireworks-show-at-home-1843106260>

这些天我一直在想迪士尼世界。在疫情飓风来袭之前，我和丈夫正计划今年秋天带着我们九岁的儿子去那里旅行。早在二月份，当我们和陌生人一起排着长队，整天触摸扶手似乎是个好主意时，我们是多么天真。

Watch

这些天来，我们不知道我们的孩子什么时候会回到学校，更不知道什么时候在拥挤的迪斯尼公园会安全，但迪斯尼正在尽最大努力让我们体验一点公园的体验。它的最新礼物是一个预先录制的视频“后快乐地生活”，这是在一切都变得糟糕之前运行的压轴魔术王国烟花表演。

迪士尼将这段 18 分钟长的视频发布在 [、博客](https://disneyparks.disney.go.com/blog/2020/04/disneymagicmoments-virtual-viewing-of-happily-ever-after-at-walt-disney-world-resort/) 和 [YouTube](https://www.youtube.com/watch?time_continue=102&v=5bN5b11H4_s&feature=emb_logo) 上，供我们观赏。(看那些人站得多近啊，我的天。)

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-5bN5b11H4_s&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-5bN5b11H4_s&start=0) 

这和亲临现场是不一样的。烟火在银幕上永远不会像在现实生活中那样令人印象深刻。但是音乐和城堡上令人难以置信的灯光和图形使它值得一看，特别是因为这种体验是免费的和 T2 的。

如果这还不足以解渴你的迪士尼之渴， [你可以观看关闭前迪士尼乐园正在进行的](https://www.youtube.com/watch?v=rR-KP8KH1Ng&feature=emb_logo)[神奇的事情](https://disneyland.disney.go.com/entertainment/disneyland/magic-happens-parade/?CMP=ILC-DPFY20Q2wo0326200002A) 游行:

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-rR-KP8KH1Ng&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-rR-KP8KH1Ng&start=0)
