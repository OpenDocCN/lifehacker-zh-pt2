# 什么是支架？

> 原文:[https://lifehacker.com/what-is-a-stent-1838746584](https://lifehacker.com/what-is-a-stent-1838746584)

伯尼·桑德斯在心脏周围的动脉中放置了两个支架，导致他取消了竞选活动。放置支架是一个相当简单的医疗程序— 让我们来看看需要做些什么。

### 谁会得到支架？

冠状动脉支架相当普遍，每年约有 200 万人接受该手术。它们可以在心脏病发作时放置，但也常用于防止动脉阻塞的人心脏病发作。

他的团队告诉记者，伯尼在经历胸痛后接受了支架治疗。在经历胸痛后，患者可能会接受 [冠状动脉造影](https://www.mayoclinic.org/tests-procedures/coronary-angiogram/about/pac-20384904) ，该造影使用 X 射线寻找堵塞。如果医生发现堵塞，他们可能会建议放置支架。

### 支架到底是什么？

支架是一种网状管，可以放置在部分阻塞的动脉内以保持其开放。通常情况下，医生会将一根细管插入患者手腕或腹股沟的动脉中，并将其一直穿到心脏周围的动脉。一旦管子到达堵塞处，他们就给气球充气来扩大空间。球囊周围有折叠的支架，但球囊膨胀后，支架会粘在原位以保持动脉开放。(设备的其余部分被移除。) [这里有一段来自梅奥诊所](https://www.youtube.com/watch?v=I12PMiX5h3E) 的视频，展示了动脉内部发生的情况:
