# 了解您的网络，第 2 课:了解路由器的管理页面

> 原文：<https://lifehacker.com/know-your-network-lesson-2-understanding-your-router-5831529>

大多数路由器都带有某种管理网站，你可以通过在你最喜欢的网络浏览器中键入它们的 IP 地址，并输入你的登录名和密码来访问——你可以从 *admin/admin* 或 *admin/password* 更改登录名和密码，对吗？如果你幸运的话，你有一个“更智能”的路由器，它更容易通过应用程序进行配置，但大多数人(以及他们的旧的非网状 wifi 设备)可能会被网站所困扰。

除非制造商在体验中建立了一个“简单模式”，否则路由器的管理页面第一眼看上去会让人不知所措，尤其是如果你不习惯你会发现的许多术语的话。我们今天的目标是帮助您设置路由器最重要的功能，并解释您应该了解的所有其他细节。

您可能不会使用路由器管理页面中的每个部分，但了解它们是什么以及在哪里可以找到它们将有助于您创建最佳的家庭网络。

### **命名您的路由器**

虽然这看起来微不足道，但实际上你需要知道一些关于命名路由器的事情。首先，您的路由器和无线网络的名称*是不同的*。命名您的无线网络实际上是命名路由器广播的 [服务集标识符](http://en.wikipedia.org/wiki/SSID)(SSID)——当您想要连接到您的 wifi 网络并上网时，您在计算机上选择的标识符。如果您可以选择更改它，您的路由器的*名称、*或*主机名、*就是它如何向网络上的其他设备标识自己。SSIDs？超级重要。路由器名称？不重要。

### **基本 wifi 配置和安全性**

配置路由器时，您需要采取几个步骤，这些步骤都非常简单。首先，您需要做出一些基本的设置决定。一般来说，您可以在路由器管理页面的“无线”选项卡中找到这些设置，或者类似名称的内容—查找“无线”和“设置”和/或“设置”的组合。

一旦你到了那里，你要做的第一件事就是选择一个 SSID。随便挑个名字；这是你的无线网络。你要么被要求使用一个 SSID 来覆盖你的所有无线网络，你的路由器将决定设备是在 2.4GHz 还是 5GHz 上连接，或者你可以为你在每个频带上运行的每个无线网络设置单独的名称。我们更喜欢后者，因为知道你在连接什么是很好的，但是很多人为了额外的方便而选择前者。

您不需要担心调整路由器的广播模式，除非出于某种原因，您没有看到路由器理论上可以支持的最高连接类型(如 802.11ac 或 802.11ax)。如果没有，请确定它已设定为允许来自这些设备的连接。

至于“安全性”，你会希望使用 WPA2，理想的 WPA2-PSK 与 AES 加密(如果你被要求得到具体的)。这将为您提供最新、最安全的安全性，保护您的 wifi 网络免受潜在攻击者的攻击——尽管有人试图强行进入您的无线网络的可能性非常低。

“频道”选项变得有趣起来。你可以让你的路由器自动选择最好的无线信道在 2.4GHz 和 5GHz 上使用，但你可能也想自己做一个简单的扫描，确保它的选择是最有意义的。我们已经在前面的 中讨论过 [，这很容易做到。同样，你可能希望你的路由器自己决定其信道宽度，但如果你看到它](https://lifehacker.com/which-wifi-channels-should-i-use-for-my-wireless-networ-1832788063) [使用 2.4GHz 的 40Mhz 信道](https://lifehacker.com/the-10-best-ways-to-boost-your-home-wifi-5931743)——而且你周围还有其他竞争的无线网络，比如你的邻居——你可能想做个好人，手动将其设置为更友好的 20MHz。你不想成为那个为别人的关系 造成悲伤的人 [，更别说自己的设备了。](https://community.tp-link.com/en/home/forum/topic/163064)

最后，如果你被要求改变你的发射功率，抵制住改变任何东西的冲动。

### 是这样吗？

如果你只是想让你的无线网络运行起来，你已经做得差不多了。您已经完成了基础工作:

*   命名您的 SSIDs
*   设置密码
*   为您的网络选择正确的加密方式
*   设置您的 wifi 频道(或告诉您的路由器为您设置)

但是，正如您将看到的，您可以在路由器上进行更多的设置。让我们再接触一些可能有意思的东西。

### **访客网络**

虽然大多数人会把自己的 wifi 密码透露给朋友、室友和客人，但你不会。如果您的路由器支持访客网络，您可以设置一个——类似于您设置初始 wifi 网络的方式——然后您可以使用它将这些人与您的*实际*无线网络上的设备隔离开来，包括彼此。

一旦你厌倦了提供免费 wifi，更改密码(或完全停止访客网络)就很容易了。当您这样做时，您现有的任何设备都不会受到影响，只有您的客人会受到影响。如果您对自己的无线连接感兴趣，这是一项值得探索的伟大功能。

### **USB 共享**

即使是半花哨的路由器*也应该*配备 USB 端口，这允许你插入一个设备——比如便携式硬盘或打印机——然后你可以与网络上的其他系统共享。也就是说，这可能是一个大多数人都不会在意的功能。您的打印机可能已经按原样连接到您的网络；如果不能，你可能会使用一台个人电脑进行所有的打印(或者通过 USB 将你的打印机连接到任何你需要打印的设备上)。

尽管如此，如果您的路由器支持此功能，还是值得探索一下，因为这是一种与网络上的所有设备共享兼容 USB 设备的便捷方式。这不是我安装新路由器时的首要任务，但很高兴知道如果你愿意，你可以做这件事。

### **基于云的访问**

这个很棘手。一些路由器带有云连接，您可以使用云连接从任何有互联网连接的地方访问路由器的设置。你甚至可以访问你的网络设备，以防你需要把一个文件从你的家庭桌面转移到你的工作电脑上。

大多数人——即使是那些拥有网状路由器的人，你通常会配置一个应用程序，允许你从任何地方访问他们的设置——可能不需要在他们设置好路由器后再去摆弄它。最好不要向云开放你的路由器和家庭网络——那会增加你不需要的安全风险。(你甚至可以用某个特定路由器制造商提供的一些疯狂的密码和任何一种双因素认证选项来锁定你的帐户，但大多数人可能没有那么在意安全。)

更好的选择是使用更简单的服务，如谷歌的 [Chrome 远程桌面](https://remotedesktop.google.com/) 来远程访问你的系统或来回传输文件。(远程桌面没有内置的文件传输功能，所以你必须将你想要访问的文件上传到云服务。)尽管如此，这仍然是一个更安全的选择，因为它与你的谷歌账户*和*绑定在一起，而不是将你的路由器开放给云。

### **高级设置**

如果您在路由器的管理页面上没有看到一堆看起来神秘而模糊的设置，很可能它有某种“高级”选项卡或模式，您可以启用它来查看您可以做的所有事情和配置。虽然大多数正常人可能不需要摆弄他们路由器的高级设置(那么多)，但有很多东西值得了解。

#### **DNS**

在路由器设置的某个地方，您可能会看到一个选项，要么让路由器自动获取其 DNS 地址(从您的 ISP 处),要么使用您手动输入的地址。这个选择很大。你的 ISP 的 DNS 可能比你可能使用的要慢(并且潜在的不安全/更多窥探)。切换到第三方 DNS 提供商，这是性能和隐私为导向的；这是我们设置新路由器时首先要做的事情之一。

#### **DHCP 服务器**

您的路由器还应该有一个连接到它的所有设备的列表，该列表将包括它们的 IP 地址，并且很可能包括它们的 MAC 地址——可能类似于“22-33-44-55-66-47”的唯一标识符如果你需要追踪网络上的设备，路由器管理页面的这一部分是必须访问的。

类似地，关闭*路由器的 DHCP 服务器(并禁用其防火墙)就是如何将路由器转变为一个更笨的接入点。这是假设您的路由器还没有某种您可以轻松切换的“接入点”模式。不过，一旦你做到了这一点，你就可以通过将这个路由器变成的哑接入点连接到另一个*路由器*来 [扩展你的网络](https://lifehacker.com/how-to-extend-your-wi-fi-network-with-an-old-router-915783308)——可能是更新更好的东西。*

#### **NAT**

NAT 代表 [网络地址转换](http://en.wikipedia.org/wiki/Network_address_translation) 。您可能知道您的本地 IP 地址不同于互联网上的 IP 地址。在您的本地网络中，它们通常看起来像 192.168.x.x 或 10.0.x.x，但实际上它们可以是任何东西，因为它们是本地的。NAT 是将外部 IP 地址转换到您的本地网络，这样您就可以在广阔的互联网范围内与人交流。

端口转发与此相关，因为默认情况下，外部的任何人都不能访问您的本地机器。但是，您可以使用端口转发为网络上的某些机器打开某些端口。例如，如果您试图让 BitTorrent 达到最高速度，您可能需要将端口转发到您的下载设备(可能是您的 PC)，然后告诉您的 BitTorrent 客户端应该使用哪些端口进行连接。

#### **QoS**

QoS 代表服务质量。它旨在帮助您设置网络上设备消耗带宽(出站，非入站)的限制。换句话说，当用户和服务独占其他设备或服务所需的带宽时，QoS 允许您设置规则并限制用户和服务。

这在理论上听起来很棒，但实际上可能对你没什么用。在你的路由器上测试设置，看看你的优先设备*是否真的*看到任何速度提升；如果没有，请随时将其关闭。

#### **IPv6**

如果您的路由器允许您打开 IPv6——大多数现代路由器都应该这样做——您应该这样做。假设您的 ISP 支持，它可以无缝使用。你可能看不到任何减速带，但是你会帮助你的网络适应未来。毕竟，我们不可能永远生活在 IPv4 世界里。

#### **固件更新**

最后，你会想知道你去哪里更新你的路由器固件。对于大多数非云路由器，这可能是一个手动过程:您从制造商的网站下载固件，将其上传到路由器，并等待它安装。如果你幸运的话，你的路由器可以直接检查其制造商的固件，并让你知道是否有新的东西要下载。更好的是路由器可以自动更新自己，因为你不必担心这个过程。

为什么这很重要？更新的固件可以为您提供新的功能，修复错误，并修补攻击者可能用来危害您的网络的明显安全漏洞。这是一项如此重要的任务，假设你的路由器不能自我更新，你甚至应该每三个月左右设置一个日历提醒。它几乎不需要你花时间去做，如果你不得不花几分钟去安装新的固件，时间会稍微多一点，但是你不想让它溜走。
