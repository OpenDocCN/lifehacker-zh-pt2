# 这个应用程序就像艺术版的 Shazam

> 原文：<https://lifehacker.com/this-app-works-like-shazam-for-art-1838123110>

当你听到一首你喜欢却又不太能定位的歌曲时，Shazam 成了许多人解决问题的方法。你所要做的就是打开应用程序，让它听几秒钟，通常你会在几秒钟内看到标题和艺术家。

Watch

现在，一些应用程序开发人员正在为真实的艺术再现这种体验。*《纽约时报》* [突出了该领域的几款应用程序](https://www.nytimes.com/2019/09/11/arts/design/smartphone-art-app.html) 、sspecifically[Magnus](http://www.magnus.net/)，这款应用程序可以为你看到的一件艺术作品判断出艺术家是谁。更好的是，它还能给你那件艺术品的名称，以及它最后卖了多少钱。这个价格标签是你在博物馆地板上看不到的信息。

该应用程序向您显示拍卖和画廊销售的历史和近期价格，因此您可以快速了解艺术家的受欢迎程度这些年来是如何上升(或下降)的。它还有一个额外的好处，那就是让你知道你在家里拥有一件艺术家作品的可能性有多大。
