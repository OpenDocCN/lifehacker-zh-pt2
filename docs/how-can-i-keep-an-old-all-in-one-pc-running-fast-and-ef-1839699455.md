# 如何让旧的多功能一体电脑快速高效地运行？

> 原文:[https://life hacker . com/how-can-I-keep-a-old-all-in-one-PC-running-fast-and-ef-1839699455](https://lifehacker.com/how-can-i-keep-an-old-all-in-one-pc-running-fast-and-ef-1839699455)

我总是很高兴在我的收件箱里收到“帮我解决一个技术问题”的信件，最近我收到了不少与 PC 维护有关的信件——可能是由我上个月回答的 [这个技术 911 问题](https://lifehacker.com/how-can-you-keep-your-old-desktop-pc-running-well-1839141116) 引发的。今天我又带来了另一个，它扭转了任何想要升级的人都面临的“如何让旧的东西感觉新鲜”的困境。

Watch

Lifehacker 的读者 Michele 写道:

“只要读一读你写的关于保持旧电脑平稳运行的文章。我有一台用了 5 年的联想 AIO，不知道如何让它平稳运行？没什么可脱吹的等等。我想从预装的 Windows 8 升级到 Windows 10，但我也不确定该如何进行。你能帮助一个技术能力有限的 69 岁老人吗？”

我准备告诉你，事实上，你可以拆卸一些一体化系统并升级它们。这不是世界上最差的工艺，看你的厂家对你有多好了，但也不是不可能。

然而，既然你自己承认你的技术能力有限，我怀疑你可能不打算使用螺丝刀。(如果您想要挑战，请访问联想的网站(可能是其支持网站 )获取针对您的特定一体机的“”硬件维护手册。)

如果你想提升你的系统硬件，我猜你会想提升它的内存，如果可能的话，提升到你的系统支持的最快和最大的内存。我在这里不会说得太专业，因为你可能不会走这条路，但如果你还没有 8GB，我会选择最低 8GB(如果可能的话)——你可以通过从 Windows 控制面板调出 System 来检查。然后你可以使用一个简单的网站，如 [至关重要的](https://www.crucial.com/) 来查看什么样的内存适合你的系统，以及你实际上可以在你的系统中安装多少内存。(换句话说，你可能连 8GB 都做不到。)

如果你的系统还没有固态硬盘，我也会在其中安装一个，但这将是一个更棘手的升级——你必须将当前的硬盘克隆到一个新的硬盘，备份并恢复所有数据，或者干脆重新安装 Windows 并从头开始。这不是一个最有趣的过程，尤其是如果您以前从未做过，而且这一切都假设您的系统中甚至有空间容纳 2.5 英寸驱动器(或者想购买一个转换器来安装 3.5 英寸托架)。

现在，让我们把所有这些建议都放到一边，因为硬件升级听起来不在你的考虑范围内。

### 向 Windows 10 迈出重要一步

在你的情况下，我会做的是跳到 Windows 10。这样做你不会看到巨大的性能提升——假设你的系统 [甚至可以处理](https://support.microsoft.com/en-us/help/4028142/windows-10-system-requirements) Windows 10。(我猜可以。)让你的系统感觉速度更快的是，你将对它 [进行一次彻底的春季大扫除](https://lifehacker.com/the-ultimate-guide-to-reinstalling-windows-from-scratch-1832897572) 。你将抛弃你系统中积累的所有垃圾程序和其他垃圾数据；重置任何可能会降低你的电脑速度的设置；而且，如果没有别的，给你甚至更强的保护(以 Windows Defender 的形式)来防止任何潜在的性能抑制恶意软件感染。

我们之前讨论过如何安装 Windows 10——重新安装，真的——同样的技巧也适用。确保你已经 [备份了](https://lifehacker.com/how-to-back-up-your-files-now-that-crashplan-isnt-an-op-1798320345) 你的关键数据，包括任何重要的东西，比如你的文档、图片或者桌面文件夹(或者任何你保存关键文件的地方)。如果你健忘的话，你可能还想写下一个你想确保以后重新安装的关键程序的列表，并确保你已经保存了你的浏览器书签(理想情况下保存到云端)以及任何你仍然想要访问的打开的标签。

一旦你准备好扣动扳机，看看是否有办法免费获得[Windows 10](https://lifehacker.com/the-best-ways-to-get-windows-10-for-free-1837174464)(或者便宜)。否则，将安装介质插入您的系统——无论您是购买了一张 CD， [自己刻录了一张](https://www.microsoft.com/en-us/software-download/windows10) ，还是 [使用闪存盘](https://www.microsoft.com/en-us/software-download/windows10) 来安装 Windows 10——然后开始安装。为了简洁起见，我稍微掩饰了一下这个过程，但是安装 Windows 10 [并没有那么难](https://lifehacker.com/how-to-set-up-your-windows-laptop-from-scratch-1826146907) 。我的建议是在出现提示时进行全新安装，而不是升级，因为这将为您提供基本的准系统 Windows 10 体验，然后您可以根据自己的喜好进行调整。

(例如，我发现如果我在一次干净的安装后忘记了重新安装一个程序，那么它可能并不重要，我可能早就应该从我的系统中卸载它了。)

你也可以使用类似 [Ninite](https://lifehacker.com/the-best-way-to-quickly-install-apps-on-a-new-windows-p-1836244140) 的工具快速下载并安装你可能经常使用的应用。如果你遇到任何麻烦，试试 Windows 10 的新“ [推荐故障排除](https://lifehacker.com/solve-windows-10-issues-automatically-with-the-new-reco-1835377734) ”功能。它不会解决所有问题，但它是有意义的。

哦，别忘了把你所有的数据从你之前备份的地方拷贝回来。这一点非常重要——确保你设置了快速备份你的东西的服务，因为这一步很容易被遗忘(导致灾难性后果)。
