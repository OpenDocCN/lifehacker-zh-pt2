# 楼梯挑战怎么样？

> 原文：<https://lifehacker.com/how-was-the-stairs-challenge-1835156158>

我们的 [楼梯挑战](https://vitals.lifehacker.com/this-months-fitness-challenge-theme-is-stairs-1834511269) 结束了，快告诉我们进展如何吧！你做了 [弓步下半身锻炼](https://vitals.lifehacker.com/how-to-get-a-leg-workout-with-just-a-set-of-stairs-1835008484) 吗？试试我们的 [阶梯难度越来越高的俯卧撑](https://vitals.lifehacker.com/get-better-at-push-ups-by-doing-them-on-stairs-1834847608) ？



正如 [读者 hellhog 指出的](https://vitals.lifehacker.com/1834862409) ，每一级楼梯俯卧撑使用的肌肉略有不同。当平板俯卧撑使用你的胸部时，下降俯卧撑(手低于脚)更多地强调你的肩部肌肉。用健身房的术语来说，这就像卧推、倾斜卧推和头顶卧推之间的区别。或者换一种说法，这就是立式压力机和卧式压力机的区别。这是利还是弊取决于你想得到什么样的锻炼。如果你正在寻找变化，楼梯级数可能是一个很好的方法。

如果你想在这方面继续做更多的练习，试着在几个不同高度的楼梯上、木板上或者 [侧板上](https://www.acefitness.org/education-and-resources/lifestyle/exercise-library/100/side-plank-modified) 。我们只是简单介绍了你可以在楼梯上做的上半身和下半身的锻炼，所以如果你已经找到了你喜欢的人，请在评论中说出来吧！
