# 你应该对‘寄生眼虫’或脸蝇感到恐慌吗？

> 原文：<https://lifehacker.com/should-you-panic-about-parasitic-eye-worms-or-face-flie-1839692676>

我第二次想到眼睛里有昆虫的糟糕经历是在现实生活中的一次遭遇，当时我在一条小道上跑步时正面撞上了一团昆虫。我无法摆脱眼睛里仍然有东西的感觉，几个小时后我终于提取了一个死(？)用棉签从我的眼睑内侧抓出一只小昆虫。唯一能打败那个的: [今早在 Gizmodo 上看脸苍蝇](https://gizmodo.com/a-second-woman-has-gotten-parasitic-eye-worms-and-sure-1839670950) 。

最近，加利福尼亚的一名妇女在一条小径上迎面撞上了一群苍蝇，后来发现其中一只苍蝇将虫子驱逐到了她的眼睛里。“她回忆起拍掉她脸上的苍蝇，并把它们从她嘴里吐出来，”案例报告说。在接下来的一个月里，她和两位眼科医生在她的眼睛里一共发现了四条虫子。

我想明确一件事:脸蝇和眼虫是两种不同的东西，它们一起产生了这种特殊的身体恐怖。脸蝇喝动物的眼泪。眼虫生活在果蝇的脸上，它们的口器里，当果蝇进食时，眼虫会逃到眼睛里。时不时地，如果一只苍蝇落在人类的眼睛上，像你或我或加州步行者这样的人可能会以一只眼睛充满微小的蠕虫而告终。

### 好消息是:这真的很少见

眼虫有好几个种类，最近的这个案例之所以有新闻价值，部分是因为和 [去年报道的另一个案例](http://www.ajtmh.org/content/journals/10.4269/ajtmh.17-0870) 是由不同种类的眼虫引起的(*结膜吸吮线虫*，而不是以前在人类眼睛中发现的种类(*加利福尼亚吸吮线虫*)。

这些蠕虫主要感染动物；当他们发现一个人的眼睛，这似乎是一个错误。蠕虫不能在你的眼睛里繁殖，所以即使你被感染了，它们也不会永远生活在那里。

对你和我来说，带回家的信息相当简单:如果你的眼睛被刺激，你不知道为什么，或者如果你发现，你知道，你的眼睛里有虫子，请寻求医疗护理。你可以在 [这个 CDC 页面](https://www.cdc.gov/dpdx/thelaziasis/index.html) 上看到一张*蝾螈*蠕虫的照片。他们是小家伙，就像你把一根面条缩小到几乎看不见的尺寸。

对于那里的医生来说，最近案例研究的作者建议将眼虫送到寄生虫学实验室，不要鉴定为“是的，那是眼虫”，而是要鉴定到物种水平。因为科学家们想知道两年内的两起病例是否意味着古洛沙锥虫变得越来越普遍，如果是这样，我们真的应该密切关注(抱歉)。
