# 把花压成“检疫标本室”

> 原文：<https://lifehacker.com/press-flowers-for-a-quarantine-herbarium-1843399113>

草本植物制作迷人的时间胶囊。 [在你附近的博物馆或大学](https://en.wikipedia.org/wiki/List_of_herbaria) 里，可能有一个房间里放着一页又一页的旧的大纸，每张纸上都小心翼翼地贴着一整株干枯的植物。这些是恐龙骨骼或动物标本的植物等价物:为未来保存的曾经活着的标本，它们的所有细节都完好无损，因为它们不是图画或印记，而是真正的植物本身。



如果你像我一样，每天散步，你可能会注意到你以前从未做过的事情——或者对它们格外关注。我知道邻居的粉红色山茱萸最近开始在我附近开花，大蒜芥菜开始占据我后院的一个角落(这是入侵性的，我应该真的拔掉它)，我女儿心爱的蒲公英已经盛开了几个星期。

我没有精力写太多日志，但我一直在拍我看到的东西。现在我知道了检疫植物标本室，我可能会开始敦促一些植物创建一个自然剪贴簿，它是由实际的自然碎片制成的。

历史学家伊莱恩·艾尔斯(Elaine Ayers)创建了一个合作的 [检疫植物标本项目](https://drive.google.com/drive/u/1/folders/1XhXEZDwUqOlVc8wHgQb38ic8O08J4e0j) ，每个人都可以投稿。按下你自己的标本，扫描或拍照，并通过电子邮件添加到收集。(这里的指令是 T5T7】。)

无论你是否参与这个项目，你都可以为自己创建一个植物标本室，作为一个你如何经历这个季节的时间胶囊。植物标本通常包括植物采集的时间和地点的记录；你可以添加尽可能多的额外细节。你也不必拘泥于野生标本。室内植物也很重要，我注意到收藏中有一张新收获的花园芦笋的照片，上面写着“这些标本将在存档后食用。”

就像 [地图](https://lifehacker.com/why-you-should-draw-a-map-of-your-life-in-quarantine-1843261143) 和 [适合孩子的时间胶囊](https://offspring.lifehacker.com/create-a-pandemic-time-capsule-with-your-kids-1843109937) 一样，植物标本室可以是一种与你周围环境联系起来的方式，让你稍微活在当下，同时有意识地创造一个你可以在未来回归的记录。
