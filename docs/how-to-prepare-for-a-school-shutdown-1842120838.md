# 如何为学校停课做准备

> 原文：<https://lifehacker.com/how-to-prepare-for-a-school-shutdown-1842120838>

随着引起世界关注的冠状病毒继续传播，美国各地的学区——特别是在病毒确诊病例的地区——正在考虑关闭的可能性。有小孩的工作父母问他们自己，如果学校关闭，他们将如何对待他们的孩子。



全国各地的一些学校已经关闭，以应对当地的冠状病毒病例， [官方称为新冠肺炎](https://vitals.lifehacker.com/the-coronavirus-has-an-official-name-covid-19-1841601530) ，随着疾病的传播，很可能会有更多的学校效仿。但是仅仅因为学校关闭并不意味着你可以暂停你的工作。所以现在是时候开始计划冠状病毒儿童护理应急计划了。

### 从工作开始

如果你有一份办公室工作，可以根据需要远程办公，你可能是这种情况下最幸运的人之一。如果是你，和你的老板商量一下，如果学校停课并且 [准备好你可能](https://lifehacker.com/be-prepared-to-work-from-home-during-the-coronavirus-ou-1842088431) 需要的任何设备或材料，无限期在家工作是否是一个选择。

但是许多工人没有这种灵活性，所以现在就问你的老板 [如果学校停课，公司的政策将是什么](https://vitals.lifehacker.com/what-your-workplace-should-be-doing-about-the-coronavir-1842064369) 。如果学校关闭，企业会效仿吗？你会用完假期或病假吗？(如果是这样，现在再仔细检查一下你有多少钱。)他们会给员工提供额外的病假吗？如果你必须呆在家里陪孩子，你需要不带薪休假吗？如果他们为人父母，*他们的*计划是什么？

### 拜访你的村庄

好消息是，你并不孤单。如果学校关闭，你将成为众多争相寻找日间托儿服务的人中的一员。联系你孩子朋友的其他父母，问问他们有什么计划——也许你可以和另外一两位父母交换照看孩子的职责，或者一位父母有一个负责任的少年，他可以在必要时挺身而出。

如果你别无选择，问问大家庭成员或邻居他们的时间表是否有弹性。你会希望避免大型聚会或集体活动——这样做的目的是减缓疾病的传播——但是如果你或你的伴侣不能照看你的孩子，而且他们还没有大到可以独自呆在家里，你就需要后援了。

### 密切注意学区在说些什么

如果你还没有在社交媒体上关注你孩子的学校或学区，现在是时候了。你可能会在那里找到关于学校关闭的最新信息——许多学校已经在谈论他们将如何打电话，并正在考虑在线学习选项，以便在学校关闭时尽可能多地继续上课。

如果到目前为止你所在的地区没有任何消息，特别是如果你所在的地区已经确诊了该疾病，那么你可以联系学校管理部门或地区领导，询问他们正在考虑什么。
