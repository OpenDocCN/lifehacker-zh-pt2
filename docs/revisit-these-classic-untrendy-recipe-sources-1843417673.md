# Allrecipes 没有任何问题

> 原文：<https://lifehacker.com/revisit-these-classic-untrendy-recipe-sources-1843417673>

美食写作是一种非常受个性驱动的体裁，已经有一段时间了。菜谱开发者、作家和电视主持人发展出狂热的追随者和粉丝并不罕见，谁写的菜谱会成为你选择晚餐做什么的一个重要因素。

Watch

这很好，也很有意义——食谱作者通常因为他们的食谱*好而出名——但这可能是有限制的:像其他人一样烹饪同样的病毒菜可以建立一种社区感，但这也让发现的兴奋感消失了。专注于一个人的食谱也会阻止你尝试其他人的食谱，在某些情况下，甚至是整个文化的食谱。(如果你喜欢的人是白人，这一点尤其正确。)*

不知名的、无品牌的食物内容目前并不流行，但它有它的优点。首先，你可以真正客观地评价食谱，因为你对它的创造者没有情感依恋。当没有让你想象中的烹饪死党“失望”的风险时，这个过程会感觉不那么珍贵，也更没有人情味。

我要说的是，匿名(或接近匿名)内容有其优势。它允许你把你的感觉从食物中分离出来，并且承认有时候好的——而不是最好的——已经足够好了。这里有一些我最喜欢的食谱资源，它们完全没有品牌。我会说我是一个“斯坦”，除了我不确定，确切地说，我会是谁。

### [全部接收](https://www.allrecipes.com)

Allrecipes 不是一个性感的网站，但它是一个*有用的*网站。缺乏高度程式化的品牌，它真的是*所有*食谱。据我所知，他们的特色就是一个著名的人物——厨师约翰，他很棒——但他的内容并不比其他内容更受欢迎。(他在网站上有很多食谱，但它们不是通过横幅图片强加给你的。)

Allrecipes 上的食谱很容易获得，几乎没有自我意识，丰富而美味。我把这个网站作为尝试新菜式的起点，从来没有对它们失望过。( [阿米什人最喜欢的白面包是](https://www.allrecipes.com/recipe/6788/amish-white-bread/) 。)

### [烹饪的快乐](https://www.amazon.com/Joy-of-Cooking/dp/0743246268?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/revisit-these-classic-untrendy-recipe-sources-1843417673&asc_source=&tag=kinjalifehackerlink-20)

“烹饪的快乐”是一个永恒的图标，虽然两个非常真实的人编辑了这本大部头书中的内容，但你可能不知道他们长什么样。挤满了食谱，完全没有照片(有一些插图)这本书感觉有点复古，尽管去年刚刚发布了一个完全更新版本的[。(照片，编辑跟我说过一次，其实约会一本食谱。如果你不相信，看看 90 年代的烹饪书。)](https://www.amazon.com/Joy-of-Cooking/dp/0743246268?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/revisit-these-classic-untrendy-recipe-sources-1843417673&asc_source=&tag=kinjalifehackerlink-20)

这本书包含了许多不同的食谱和技术——从真空温度计到剥松鼠皮的方法——以及来自许多文化和烹饪的食谱(总是正确地注明这道菜的来源)。你可以连续几个月每天用“快乐”烹饪新的食谱，但你不一定知道它们都来自同一个地方。我认为，这是一件好事。

### [开箱食谱](https://lifehacker.com/the-best-recipes-youll-find-on-the-back-of-the-box-1790210846)

许多“家庭”食谱最初是在混合物、炼乳罐或调味汁瓶的背面发现的。事实上，我敢肯定，你至少能说出一个你认为是你祖母或婶婶原创的食谱，却发现它印在一盒蛋糕粉或一包巧克力片上。我家的玉米砂锅 就是这种情况，它(显然)是 Jiffy 的人为了卖出更多亮蓝色的盒子而创造的。Tollhouse 饼干、费城奶酪蛋糕条、康乃馨柠檬条和 Libby 南瓜派只是这种类型的几个惊人的例子。一些食谱——比如米饼和 Chex Mix——是如此的符合时代精神，以至于我们忘记了它们是由一个品牌为了销售更多的谷物而创造的。我并不是说你应该用盒子的背面来制作完整的食物，但是要留心；你永远不知道你会发现什么宝藏。

食谱作者与观众建立的关系和信任是美好的，但从你最喜欢的领域扩展到不那么时髦、受个性驱动的领域，作为一种味觉清洁剂，可能是有用的。你可能会发现一个新的最喜欢的食谱或菜肴，也可能不会。如果真的发生了灾难性的事情，你总是可以跑回你最喜欢的地方。他们甚至不会知道你离开了。
