# 如果你讨厌戈尔，就看看这些恐怖电影吧

> 原文：<https://lifehacker.com/watch-these-horror-movies-if-you-hate-gore-1839354134>

如果你是那种一看到血就恶心的人，你可能会认为恐怖电影不适合你——但你应该知道这种类型不全是血腥和胆量。举个例子，像*出去，一个安静的地方*和*灵异活动*这样的电影(大部分)是没有血腥的。



如果你正在寻找一个没有血腥和 stabbing⁠的恐怖 movie⁠，我们有一些建议。从一些员工最喜欢的开始，下面你会发现我们的无戈尔电影列表，不会让你胃里翻腾。

### 《活死人之夜》 (1968)

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-ob8vZhSjES8&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-ob8vZhSjES8&start=0) 

听我们说完:如果你在寻找一部不血腥的经典，乔治·A·罗梅罗的《活死人之夜》可能是你的答案。很大程度上是因为这部电影，我们现在有了像*《行尸走肉》*——或者基本上任何有僵尸的现代电影。至于血腥，以今天的标准来看，这是一部温和的电影。

### ***苦难* (1990)**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-UbSvZbB5Glc&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-UbSvZbB5Glc&start=0) 

凯茜·贝茨是国宝，苦难就是最好的证明。这部电影是一个粉丝痴迷的两个小时的探索，基本没有惊吓和血腥。剧透警告——当她开始把詹姆斯·卡安的角色绑在床上时，最好把目光移开。

### ***(1996)***

 *[https://lifehacker.com/embed/inset/iframe?id=youtube-video-kWin2LZkvrA&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-kWin2LZkvrA&start=0)* 

*在 90 年代末，主流的恐怖类型很大程度上属于以年轻人为中心的血腥戏剧，如电影《惊声尖叫》、《T2》、《我知道你去年夏天做了什么》、《T3》——但很少有人能与《坎皮高中戏剧》、《T4》相提并论。看它是因为它有多诡异，多精彩。*

### **布莱尔女巫项目* (1999)*

 *[https://lifehacker.com/embed/inset/iframe?id=youtube-video-a_Hw4bAUj8A&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-a_Hw4bAUj8A&start=0)* 

*在布莱尔女巫的《T2》中，你不会看到任何血腥场面——但这正是这部电影特别恐怖的地方。这部电影主要是关于你的想象力——还有什么比这更可怕的呢？*

### *小妖精 (1984)*

 *[https://lifehacker.com/embed/inset/iframe?id=youtube-video-dz5V8KzhQn8&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-dz5V8KzhQn8&start=0)* 

*如果你想要比你的标准恐怖片稍微轻松一点的东西， *Gremlins* 是一个很好的开始，作为一部探索当一群 Furbys 来到生活中并肆虐时会发生什么的电影。*
