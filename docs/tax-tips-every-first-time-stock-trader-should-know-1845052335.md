# 每个首次炒股的人都应该知道的税务提示

> 原文：<https://lifehacker.com/tax-tips-every-first-time-stock-trader-should-know-1845052335>

随着免佣金的经纪公司和 [股票交易应用](https://twocents.lifehacker.com/should-you-delete-your-stock-trading-app-1844748240)——就像流行的罗宾汉平台——继续增长，数百万新的股票交易者可能没有为明年的税收做好准备。



当你刚开始炒股时，你可能不会考虑你的账户会如何影响你的税收。例如，Robinhood 只提供应税经纪账户——这与你的延税 401(k)不同。原因如下:每次你卖出一只股票，它可能会立即影响你的税收。

如果你今年一直呆在家里，通过一个或多个交易应用程序买卖股票，这里有一些重要的税务提示需要知道。

### 投资所得税

当你出售股票获利时，或者当你收到股息时，你可能会获得投资收入，而这与你工作时的工资不同。不同之处在于，根据你持有股票的时间长短，你可能需要按照的资本利得率缴税。

当你持有一只股票一年或更长时间后，你卖出股票获利时，你将欠下长期资本利得税，通常不会超过 15% 。这可能比你的常规所得税便宜，取决于你的税级。但是，如果你持有一只股票不到一年，然后卖出获利——这对于股票交易者来说可能很常见——你可能会欠短期资本利得税。这些税率与你的常规所得税相同，即“普通收入”税率。

### **您可能会欠下季度预估税**

因为我们的税收是一个量入为出的系统，当你出售股票获利时，你也可能全年都欠季度估计税。你可以使用 [这张工作表](https://www.irs.gov/publications/p505#en_US_2019_publink1000194747) 来计算你的欠款——或者你可以向税务专家寻求指导。

### 你可以用亏损抵消税收收益

随着年底的临近，可能会用其他亏损的股票来抵消你的一些资本收益，这就是所谓的资本损失。这些损失与短期和长期资本收益具有相同的时间表，或“持有期”，但你必须在年底前出售这些股票，以“实现”这些收益或损失。

你可以在这里 了解更多关于净资本利得或损失税规则[——但如果你感到不确定，你应该与税务专家合作。](https://www.irs.gov/publications/p544)

### 你可以扣除一些资本损失

如果你的资本损失超过了你的资本收益，你可以从你的固定收入中扣除最多 3000 美元(或者 1500 美元，如果已婚的话)的资本损失。你可以在 [附表 D](https://www.irs.gov/pub/irs-pdf/f1040sd.pdf) 上看到更多关于如何扣除资本损失的细节。
