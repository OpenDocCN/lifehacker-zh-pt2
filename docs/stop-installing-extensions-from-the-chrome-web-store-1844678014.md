# 停止使用 Chrome 网络商店查找扩展

> 原文：<https://lifehacker.com/stop-installing-extensions-from-the-chrome-web-store-1844678014>

这听起来违反直觉，但请听我说完。如果你正在寻找一个新的浏览器扩展来尝试，或者你想下载一个推荐给你的，停止使用 Chrome 网络商店来寻找它。



虽然我对你，一个精通技术的生活黑客读者，不会被 Chrome 网上商店的诈骗扩展所欺骗非常有信心，但 [AdGuard](https://adguard.com/en/blog/fake-ad-blockers-part-3.html) 最近报告称，大约 300 个流氓扩展吸引了近*8000 万* Chrome 用户(不包括任何机器人)。

虽然这些 扩展中有许多 [是你典型的“任何有头脑的人都应该避免的蹩脚扩展”，但也有许多足够接近合理的扩展，从而导致问题。例如:](https://gist.github.com/ameshkov/238e45af398dfbae242fd32fe4894516)

*   【YouTube 的广告屏蔽
*   【YouTube 广告拦截器
*   **α-阻滞剂**
*   **UBlocker**
*   **广告拦截—停止每个网站上的广告**
*   **Adblocker-X**
*   **AdBurner**
*   阿德贝尔
*   **合金**
*   **Adkill**

这些都是伪造的列表，已经从 Chrome 网上商店中删除，但这只是因为 AdGuard 能够引起谷歌的注意。在那之前，这些扩展诈骗从一开始就积累了数百万用户。声明一下，列表中没有一个扩展名是合法的——你想要的扩展名是“*是“[Youtube 的广告屏蔽](https://chrome.google.com/webstore/detail/adblock-for-youtube/cmedhionkhpnakcndndgjdbohmhepckk?hl=en) ”，作者是 AdRemover，但是一个普通人怎么会知道呢？*

*感觉我们以惊人的频率写这类故事。对我来说，这表明 Chrome 在 Chrome 网上商店中有一个相当大的问题。这感觉有点像狂野西部，任何人都可以上传任何扩展，要么是另一个恶意软件的克隆，要么只是稍微有点不同，以免让谷歌怀疑有什么事情发生。*

*很明显，当“广告拦截器”、“广告拦截器”和“广告拦截器”都有相似的描述和图标时，显然有什么事情发生了。但是，如果你不是很懂技术，那么，你如何将恶意软件与必备的扩展分开呢？*

*AdGuard 很好地总结了我们通常会提供的指导:*

> **   *"If you plan to install a browser extension, please think twice. Maybe you really don't need it?**   *Only install extensions from developers you trust.**   *Don't believe the description you read in the extension.**   *Reading users' comments doesn't help either. Most malicious extensions have good comments, but they are malicious.**

*我很高兴看到他们的最后一点与我最近一直在想的相呼应:*

> **   *Don't use the internal search of Chrome Web Store, just follow the links on trusted developers' websites.**

*说一个谷歌实体，在所有事物中，在搜索方面并不值得信任，这感觉有点奇怪(与其使用它，你应该只使用...谷歌)。我不建议普通人使用 Chrome 网上商店寻找他们不知道的东西；即便如此，被恶意软件感染的几率还是太大了。*

*我的意思是，我只是简单地搜索了一下“屏蔽广告”，就像任何正常人可能会做的那样，这些结果似乎都有点粗略:*

*不要从 Chrome 网上商店安装，除非你绝对确定你安装的是合法的扩展。为了确保你不会被任何其他蹩脚的扩展所诱惑或愚弄，找到一个扩展的原始开发者——一个网站、一个 GitHub、一个 Twitter 账户，无论什么——或者一个你信任的评论网站，并使用他们提供的链接从谷歌获取你的扩展。你会因此更安全。*
