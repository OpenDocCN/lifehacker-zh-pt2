# 什么是托管，什么时候需要托管？

> 原文：<https://lifehacker.com/what-is-escrow-and-when-is-it-necessary-1838050821>

当你的钱被“托管”时，你的钱去了哪里？谁负责这件事？这意味着你有麻烦了吗？听起来你有麻烦了。钱还在，对吧？



即使你以前从来没有处理过托管，你可能会在某个时候。如果你在买房或与有问题的房东打交道，契约可能会发挥作用。如果你正在买卖任何有价值的东西，你可能也会听到这个术语:一辆车，一艘船，一件艺术品。如果大量的金钱和资产正在转手，如果出现“代管”也不要惊讶。

但这是什么呢？把它想象成金钱的地狱，但更像是一个*乐于助人的*金钱的地狱。当你的钱被托管时，这仅仅意味着一个公正的第三方保管着它。这通常发生在你把钱换成有价值的东西时，以保持买卖双方的诚实，直到交易正式开始。

### 房屋买卖双方如何使用托管服务

当你买房子的时候，你可能会把你的钱放在第三方保管处，以确保卖家会接受你的出价，完成任何销售条款，比如维修。同时，卖方可能会将房契交由第三方保管。一旦交易完成，卖家拿到钱，你拿到房契。托管官员确保交易时间顺利进行。

你的贷款人也可以为你维持一个托管账户。一些贷款机构可以每月为财产税等账单筹集资金，然后在到期日临近时提交这些款项。你的贷款人说，“我们将持有这笔钱，直到你需要它，但我们不会把这笔钱用于我们的业务。”

### 代管如何为租房者和房东工作

现在，你可能会想“嘿，我没什么价值。这对我不适用。”但是，即使你刚刚开始建立你的财务基础，你可能会遇到托管。

假设你公寓的加热器坏了，你的房东说他们会修理。但那是 11 月，现在是 12 月。这是个问题。如果你的房东违反了你对安全和舒适的基本需求，你可以扣留你的租金。但是你不能简单地忽略每月的第一天，花掉那些钱。相反，你可以通过当地法院系统申请将租金支付给第三方托管。

俄亥俄州的一家非营利性律师事务所社区法律援助解释说，在申请支付托管费用之前，你必须向你的房东提供书面通知。但如果你的房东是出了名的难以接近，这是你应该知道的最后一招。

底线是:如果你在第三方托管中处理资金，那是因为双方都想确保交易公平完成。你没有麻烦，你的钱还在。
