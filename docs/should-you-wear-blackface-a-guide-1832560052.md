# 该不该穿 Blackface？一个向导

> 原文：<https://www.theroot.com/should-you-wear-blackface-a-guide-1832560052>

当我意识到白人有多爱黑脸的时候，我已经 20 岁了.

我早就知道这是一个受欢迎的白人万圣节活动，因为每年都会有一个高中老师或 [银行高管](https://www.theroot.com/blackface-is-a-choice-friends-praise-sc-bank-executive-1830074140) 扮成坎耶·韦斯特或埃丝特·罗尔出现在当地的白人秋季节日上。有意思的是，当有黑人在场时，这些人从不穿仿黑人服装...或者他们有，只是他们没有活着出来讲这个故事。(听说吉米·霍法就是这样。他打扮成 Shaft 和 Shaft 去了万圣节前夕的聚会...好吧，也许我说得太多了。)

无论如何，根据皮尤研究中心 的一项新的 [民意调查，大约三分之二的美国白人认为黑脸在某些场合是可以接受的，而大多数黑人认为黑脸是“永远不能接受的”全国各地的学校都在疯狂地翻看旧年鉴，却发现 80 年代的大多数白人兄弟会要求至少一个人在脸上涂上鞋油才能喝酒。](http://www.pewresearch.org/fact-tank/2019/02/11/about-a-third-of-americans-say-blackface-in-a-halloween-costume-is-acceptable-at-least-sometimes/ft_19-02-11_blackfacehalloween_whiteadultstwice_2/)

每当有人因为戴黑脸被抓，他们的借口总是:“嗯，我不知道用一群人的肤色作为我周六晚上恶作剧的道具有什么不对！我不应该这样做吗？”

所以，为了帮助全美国的 [【梅根·凯利】](https://www.theroot.com/you-mean-to-tell-me-megyn-kelly-was-right-this-whole-da-1832429158) 和其他白人，Root 整理了这张方便的流程图来帮你决定是否应该戴黑脸。请将它打印出来，与朋友分享，并作为插页包含在下一次全球白人会议的广告手册中。
