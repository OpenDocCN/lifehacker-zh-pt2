# 使用“经典”食谱作为模板制作更好的沙拉

> 原文：<https://lifehacker.com/make-better-salads-by-using-a-classic-recipe-as-a-templ-1837986043>

对一些人来说，做一道好沙拉是很直观的；对其他人来说，这是一个令人惊讶的挑战。在一堆叶子上混合和搭配各种口味和质地可能看起来不是太令人畏惧的任务，但很容易导致沙拉太辛辣、太咸，或者——绝对最糟糕的——淡而无味。

Watch

你不需要厨师的头脑就能做出美味的沙拉。正如《华尔街日报》的指出的，通过使用经典的沙拉食谱作为模板，你可以磨练你的沙拉制作技能，然后扩展到制作你自己的完美平衡的一堆蔬菜(和其他东西)。

### 我说的“模板”是什么意思？

要使用沙拉作为模板，你必须得到一份书面的食谱，看看它的每一种成分，考虑每一种成分给沙拉碗带来了什么，然后*再*想出具有类似特征的其他成分。像任何好的实验一样，一次只改变一个变量——即主要成分——是个好主意。(你不必像在实际科学实验中那样严谨——我们是在做午餐，而不是或类似的东西。)你也可以添加你认为沙拉缺少的东西，不管是营养上的还是味道上的。

### 怎么做

让我们拿一个树叶图标——我们将使用我最喜欢的楔形图标——来分解它。楔形沙拉包括:

*   冰山的四分之一
*   西红柿丁
*   脆皮碎培根
*   红洋葱
*   蓝纹奶酪调料(可能还会额外放一些碎末)
*   韭菜

这是一种丰富的咸味沙拉，令人惊叹，但你可以立即看到一些成熟的修改区域。如果你想在里面多放一点健康食品，显然可以从生菜开始。冰山可能会提供很多凉爽的嘎吱声，但它主要是水。当切成楔形时，紫色的菊苣看起来很漂亮，它提供了一种很好的苦味，有助于去除所有的脂肪和盐。你也可以用任何切碎的沙拉蔬菜做一个“解构”楔子，只是要确保选择稍微重一点的东西，因为蓝纹奶酪和培根可以压住更娇嫩的叶子。

至于其他互换，葱属部门是食谱中另一个值得研究的部分。香脆的炸青葱或炸蒜片都会带来红洋葱的辣味，以及一些坚果式的风味。大葱或青蒜(显然)可以代替细香葱。如果你想变得真正疯狂，你可以用草莓代替西红柿。乍一看，这可能听起来很疯狂，但它们有相似的酸度和甜度，而且——如果你是一个执着于美学的人——它们也是红色的，这很好。

在这里，我唯一不会乱动的东西是蓝奶酪酱和熏肉。也许我可以被说服使用脆皮火腿，但是火腿比培根贵得多，而且油炸火腿意味着你会忘记丝滑的脂肪，而丝滑的脂肪至少是这种特殊腌制猪肉产品的一半。

### 你还可以用什么沙拉？

如果你不精通沙拉知识，请允许我提供一些沙拉的图标，以及一些建议的替换和补充:

#### 凯撒什锦色拉

不喜欢凤尾鱼？用 [蓝纹奶酪](https://lifehacker.com/make-a-vegetarian-friendly-caesar-dressing-with-blue-ch-1829765284) 代替你的调料。或者把长叶莴苣换成另一片叶子(我真的只在《可怕的长叶莴苣回忆》中这么做)。

#### 科布沙拉

水煮蛋不错，但 [日式拉面蛋](https://skillet.lifehacker.com/how-to-make-a-ramen-egg-1836419141) 更好，还有 [酱蛋](https://skillet.lifehacker.com/make-appetizer-worthy-hard-boiled-eggs-by-wrapping-them-1831021259) ，或者直接 [腌制蛋黄](https://skillet.lifehacker.com/grate-egg-yolks-like-cheese-for-a-rich-flavor-topping-1711826331) 。

#### 尼斯沙拉

试着用芦笋代替通常的绿豆，或许试着用烤小鱼做土豆。我不会省略橄榄，但我会加入角豆或刺山柑来增加咸味。

#### 西瓜和羊乳酪沙拉

使用不同的咸奶酪，如 cotija，并添加石灰或辣椒粉(或 Tajin)以获得额外的积分。

当然，你选择作为模板的沙拉不一定是大写的“经典”任何来自知名沙拉来源的沙拉食谱都会对你有好处。修补直到你知道什么有用(什么没用)，然后继续制作你自己的蔬菜制品。
