# 为你挑食的人创建一份“菜单”

> 原文：<https://lifehacker.com/create-a-menu-for-your-picky-eater-1842058400>

我坚信，让挑食的人吃东西没有太多的窍门。挑食的父母需要更加努力地为他们的孩子提供健康、均衡的膳食，这样他们的孩子每天早上、中午和晚上都不会大吵大闹。当然， [食物选择](https://offspring.lifehacker.com/let-your-kid-eat-with-food-picks-1836892766) 可能是一个孩子开始啃食甜瓜所需要的一切。但下一个孩子可能会被“ [品尝盘子](https://offspring.lifehacker.com/use-a-taste-plate-to-get-picky-eaters-to-try-new-food-1796917875) ”说服或者也许你会有更好的运气，在 [维恩图](https://offspring.lifehacker.com/feed-your-picky-kids-and-you-with-venn-diagram-meals-1836499243) 中想到你家人的饭菜。



很有可能，如果你的孩子很挑剔，你已经尝试了几种不同的策略并取得了不同的成功。今天，我在这里给你提供多一个选择:为你的小孩创建早餐、午餐和晚餐“菜单”。

这个想法来自克莉茜·泰根，他在 YouTube 的 [视频中说，“相信我，我*知道*这很荒谬。我很清楚这很疯狂。”但我觉得这不比把食物放在棍子上，](https://www.youtube.com/watch?v=YM-idYI1I0w&feature=emb_logo) [用他们的食物制造迷宫](https://offspring.lifehacker.com/make-food-fun-for-your-picky-eater-1834749221) ，或者 [抛硬币](https://offspring.lifehacker.com/get-your-little-kids-to-eat-their-dinner-by-flipping-a-1834054976) 更疯狂。嘿，它对 Teigen 的女儿 Luna 有效。这意味着它对你的孩子有效吗？谁知道呢！他们很挑剔！但我认为值得一试:

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-YM-idYI1I0w&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-YM-idYI1I0w&start=0) 

Teigen 使用了一个三环活页夹和一个层压机(但你也可以使用 [护板](https://www.staples.com/Staples-5-1-2-x-8-1-2-Sheet-Protectors-Clear-25-Pack/product_616224?cid=PS:GooglePLAs:616224&ci_src=17588969&ci_sku=616224&KPID=616224&gclid=CjwKCAiAnfjyBRBxEiwA-EECLKA8uCF6aKE19rMHklqpD62B5GEQMpkBQO4cfPl5zeDZ54jE-XrnjRoCocUQAvD_BwE) 代替)。她拍下了她知道卢娜喜欢的各种健康、均衡的膳食，比如早餐吃水果麦片，西兰花烤鱼条，或者意大利面条和肉丸。吃饭的时候，泰根翻看活页夹，让卢娜挑选她想要的。

这会适得其反吗？是的，当然可以。如果你在蓝莓吃完的第二天就离开了，那你就是在自找麻烦。在你交出菜单之前，自己快速浏览一遍，确保你拿出了任何你手头没有的食材(或者没有时间/精力去做)。

Teigen 还继续叠了一堆一美元的钞票，这样 Luna 就可以“支付”她的餐费了。(看起来她可能在使用她的新层压机时很开心。)如果你也认为这是一个练习数钱和用钱的机会，那就去做吧，但这不是必须的。只要让他们吃点健康的东西就足够了。
