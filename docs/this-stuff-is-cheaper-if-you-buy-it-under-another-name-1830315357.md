# 如果你用别的名字买，这东西会便宜些

> 原文：<https://lifehacker.com/this-stuff-is-cheaper-if-you-buy-it-under-another-name-1830315357>

“我知道如果你搜索‘马厩垫’的话，健身垫会更便宜，”redditor MMAJakob 在 r/节俭网站上说。“如果你把婚礼也称为生日派对，那它就便宜多了。 [有哪些其他的东西，当你称它为别的东西时会变得更便宜？](https://www.reddit.com/r/Frugal/comments/9uxfx1/cheaper_names_for_things/) “下面是一些来自跟帖的最佳答案。

当然，用户 mareish 回答说，健身房的垫子已经比马垫便宜了——每样东西的价格都不一样，所以不要在没有快速仔细检查的情况下就按照下面的提示行动。

*   我们从你们都想知道的那个开始:是的，你可以用医用折叠楔代替解放者性爱楔。谢谢你的提示，Psydaranta。
*   婴儿很贵。塞布丽娜说，你可能想用小狗垫代替一次性尿布，用狗粪袋代替尿布袋来装尿布。kurinevair666 说，成人护垫可能更便宜。君特 1001 说，嘉宝布尿布就像打嗝布一样。

*   萨布丽娜说，任何时候你发现自己在寻找“婴儿”产品时，也可以尝试购买成人版本。你的宝宝不需要自己专用的婴儿版搅拌机、加湿器和白噪音机。塞布丽娜指出，不管标签上是否有婴儿，无色无味的洗涤剂都一样好用。当你买婴儿床床单之类的婴儿用品时，先去综合商店，再去专门的婴儿用品店。
*   不要买烧烤手套，hellonasty 说；买焊接手套。
*   jreed26 表示:“与资金池相关的一切都是巨大的赚钱机会。例如，你应该在好市多购买普通袋装盐，而不是“池盐”。

*   [拖拉机供应](https://www.tractorsupply.com/) 在这个线程中得到命名很多。他们将宠物用品作为农场用品廉价出售，redditor hungaryforchile“发现了整整一磅(或更多？)的粉红色喜马拉雅岩盐，在马区的一家拖拉机用品店卖 8 美元。对于干燥和开裂的鞋跟，奶奶推荐拖拉机供应的袋装香油，而不是护足霜。jreed26 买了一个真正的拖拉机配件来代替昂贵的泳池灯:在一家五金店，他们发现了同样的“拖拉机灯”。

*   如果你不觉得丙酮洗甲水太刺鼻，“你可以在五金店买到一夸脱纯丙酮，价格和化妆品区的 6-8 盎司一瓶一样，”wiredandwiser 说。(稀释它，然后根据你的舒适程度逐步增加。)
