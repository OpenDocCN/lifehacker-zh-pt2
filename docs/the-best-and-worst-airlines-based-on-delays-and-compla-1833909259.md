# 基于延误和投诉的最佳和最差航空公司

> 原文：<https://lifehacker.com/the-best-and-worst-airlines-based-on-delays-and-compla-1833909259>

有没有经历过长时间的延误，却在错误的州着陆，滞留在没有食物的停机坪上？我有！我不建议这样做。

如果你经历过任何版本的这种地狱般的场景，你就会知道并非所有的航空公司都是平等的。根据今年的 [航空公司质量报告](https://airlinequalityrating.com) ，一项比较美国九大航空公司的研究，当你通过某些航空公司预订航班时，你现在有充分的理由重新考虑你的旅行计划。

在新的报告中，每家航空公司都根据其 2018 年的误处理行李、准时到达、非自愿拒绝登机(或被撞)和客户投诉的比率进行了排名。换句话说，如果他们搞砸了，这很可能反映在报告中，并没有帮助他们的排名。

哪家航空公司是最大的违规者？好吧，你可能有你最不喜欢的，但你可能需要重新考虑边疆航空公司和美国航空公司的航班，这两家公司在准点表现、行李和客户满意度方面排名都很差。

另一方面，达美航空(Delta Airlines)和捷蓝航空(JetBlue)排名最佳，前者在所有航空公司中颠簸的乘客最少，在准点起飞方面得分较高。

根据研究人员的说法，在 2017 年至 2018 年期间，航空公司在所有类别中都有所改善，除了准点表现。鉴于广为人知的联合航空事件已经迫使许多航空公司重新考虑他们自己的碰撞(和超额预订)政策，非自愿碰撞的急剧下降不应该是一个意外。
