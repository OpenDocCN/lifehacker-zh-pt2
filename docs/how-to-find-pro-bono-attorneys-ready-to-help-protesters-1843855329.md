# 如何找到愿意帮助抗议者的公益律师

> 原文：<https://lifehacker.com/how-to-find-pro-bono-attorneys-ready-to-help-protesters-1843855329>

抗议——即使是和平抗议——伴随而来的风险之一是你可能会被逮捕。是的，第一修正案赋予了我们和平集会的权利——但是仅仅因为《权利法案》中有所规定并不意味着你不会发现自己处于这些权利被侵犯的境地。

Watch

在你参加下一次抗议之前，如果你在抗议 时被警察拦下 [，阅读、保存和/或记住](https://www.aclu.org/know-your-rights/protesters-rights/#i-was-stopped-by-the-police-while-protesting) [ACLU](https://www.aclu.org/) 的行动清单是个好主意。这些建议包括“确保让你的手看得见”和“如果可以，写下你记得的一切，包括警官的徽章和巡逻车号码以及他们工作的机构。"

美国公民自由联盟也提供了这条重要的建议:

> 如果你被捕了，你有权问为什么。否则，说你希望保持沉默，并立即要求律师。没有律师在场，什么都不要说，什么都不要签。

当然，如果我们在抗议中被逮捕，我们大多数人都没有可以立即联系的律师——你甚至可能不确定你是否能支付律师费。这意味着，在你参加下一次抗议活动之前，你还应该保存或记住一位愿意帮助抗议者的公益律师的电话号码。

Twitter 用户已经开始编制准备帮助有需要的抗议者的律师名单:

现在，Ashley Holub 的 [无偿服务律师电子表格](https://docs.google.com/spreadsheets/d/1w24baP-I-DcL6KOPuYkxuCeL_5hbkX6tryucgj6MaOw/edit#gid=0) 包括了代表 18 个州和哥伦比亚特区的 60 名律师——如果你在列表中没有看到你所在的州或城市，在 Twitter 上快速搜索“无偿服务”和你所在城市的名字应该可以帮助你找到更多的资源。

因为如果你要参与和平集会的权利，你会想知道如何在需要律师之前联系律师。
