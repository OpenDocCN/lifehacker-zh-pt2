# 用观察笔记本提高你的写作水平

> 原文：<https://lifehacker.com/improve-your-writing-with-an-observational-notebook-1839921441>

如果你正在起草 NaNoWriMo，你觉得你的小说缺乏其他作者似乎自然包含的那种感官细节——这种写作让你感觉你就在故事里面，与人物在一起，感受他们的感受——可能是时候开始携带观察笔记本了。

Watch

这种笔记本与其说是一本日记，不如说是一种快速记下你所看到的、听到的、闻到的、尝到的、感觉到的——你明白了。一旦你开始观察这种感官细节，你就能在自己的写作中运用你扩展的感官词汇。

[文学枢纽](https://lithub.com/) 最近分享了作者莉迪娅·戴维斯的新散文集 [*散文集一*](https://us.macmillan.com/books/9780374148850) ，其中戴维斯 [解释了这种做法的好处](https://lithub.com/lydia-davis-ten-of-my-recommendations-for-good-writing-habits/) :

> 这将提高你的观察力和表达能力。建立了一个富有成效的反馈回路:通过记笔记的习惯，你必然会更多地来观察；观察得越多，你就会有更多的东西要记下来。

戴维斯建议你用笔记本记下具体、详细的观察结果，从独自坐在咖啡店里的人的姿势，到你自己的咖啡冒出的蒸汽暂时使你鼻子前面的空气变浓的方式。(当然不是说你一定要在咖啡店里做这种观察工作，但是咖啡店里的作家都是陈词滥调是有原因的。)

这是戴维斯的笔记本中的一个例子，关于风如何影响户外庭院:

> 昨天的大风吹走了女人的长发，女人的长裙，树的花冠，在户外吃饭时，餐巾从膝盖上掉了下来，生菜从盘子上掉了下来，馅饼从盘子上掉在人行道上。

当你花时间观察和描述你周围的环境，以及这些环境如何影响你的身体和你周围的人的身体时，你会有更多的细节和感官记忆可供你起草小说时借鉴。

他们说你应该写你所知道的，毕竟——所以用你的观察笔记本来确保你知道刮风的一天对酥皮糕点有什么影响，或者一杯热咖啡对你的鼻孔有什么影响，或者独自坐在熙熙攘攘的咖啡馆里对某人的肩膀有什么影响。

然后回到你的 NaNoWriMo 草稿，看看有多少观察结果最终进入了你的故事。
