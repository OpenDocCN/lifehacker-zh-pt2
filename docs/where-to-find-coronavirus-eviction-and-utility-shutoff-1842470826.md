# 在哪里可以找到您所在州的冠状病毒驱逐和公用事业关闭规则

> 原文：<https://lifehacker.com/where-to-find-coronavirus-eviction-and-utility-shutoff-1842470826>

随着冠状病毒 [疫情](https://lifehacker.com/tag/coronavirus) 的发展，许多城市和州已经暂停出租驱逐和公用事业关闭。但是到处都有这么多与流行病相关的新闻，很难不分心地找到这些信息。



普林斯顿大学的驱逐实验室 创建了一个城市和州驱逐政策变化的目录。该目录在顶部有几个联邦政策的变化，随后是按字母顺序排列的州列表。该目录包含行动类型和生效日期，以及更多详细信息的链接。

随着更多细节的出现，这份名单会不断更新。

这里只是目录的一部分，让你了解不同地方的住宿条件有什么不同:

例如，一些地区列出了暂停驱逐的具体日期，而另一些地区则无限期暂停驱逐。一些地方已经宣布不会停水或断电；其他人指定所有实用程序将保持打开。

该网站还提供了联邦政策变化的链接，包括像美国农业部食品援助项目 这样的机构。

如果你有想要驱逐实验室团队添加到列表中的信息，你可以 [给他们发送一个通知](https://evictionlab.org/contact/) 。
