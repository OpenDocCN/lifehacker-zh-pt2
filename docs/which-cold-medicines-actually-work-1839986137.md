# 哪些感冒药真的有效？

> 原文：<https://lifehacker.com/which-cold-medicines-actually-work-1839986137>

当你因为感冒而感觉糟糕的时候，你只是想要一些东西来治愈你——如果不能治愈你，至少可以帮助你暂时感觉好一些。不幸的是 [很多药方都是安慰剂](https://vitals.lifehacker.com/how-to-harness-the-placebo-effect-when-you-have-a-cold-1839965981) ，但是药店里有些东西比其他的更有效。

### 最好的解充血药在柜台后面

如果你鼻塞，**伪麻黄碱**才是真货。过去，你可以在商店的货架上找到它。Sudafed 是一个品牌名称。(苏达菲，伪麻黄碱，懂吗？)但是 [伪麻黄碱可以相当容易地转化为甲基苯丙胺](https://www.acsh.org/news/2017/09/12/why-sudafed-behind-counter-meth-chemistry-lesson-11812) ，因此 2006 年的一项法律限制了其销售。如果你想要好东西，花时间向药剂师出示你的身份证。

### 蜂蜜效果出奇的好

蜂蜜不会治愈你的感冒，但它似乎可以缓解喉咙痛和咳嗽，至少有一点点。研究经常显示它比非处方咳嗽糖浆更有效。(这是因为蜂蜜很棒还是因为止咳糖浆有点糟糕？老实说，可能是后者。)但是不要用蜂蜜止咳糖浆——是的，这是一种东西——只要抓一些真正的蜂蜜，混合到热水或茶中。那更便宜也更容易。

顺便说一下，关于蜂蜜的一个重要警告: [给 1 岁以下的婴儿服用蜂蜜被认为是不安全的](https://www.poison.org/articles/2010-jun/dont-feed-honey-to-infants) 。肉毒中毒的风险很小，婴儿尤其容易感染。蜂蜜对 T4 没什么帮助，所以不值得冒这个险。

### 大多数药物的疗效都不确定

去氧肾上腺素是一种减充血剂，取代了非处方产品中的伪麻黄碱。它似乎工作得不是很好，根据一些研究，它 [根本不起作用](https://www.forbes.com/sites/daviddisalvo/2015/10/26/the-popular-over-the-counter-cold-medicine-that-science-says-doesnt-work/#4ac439a74753) 。

止咳剂右美沙芬同样不是很有效，愈创甘油醚也不是，愈创甘油醚是一种“祛痰药”，旨在稀释粘液，使其更容易咳出。一份 Cochrane 综述得出结论，没有足够的证据表明任何非处方咳嗽药是否真的有效。

泰诺和布洛芬确实有助于缓解疼痛和退烧，尽管它们可能不是你想到感冒药时想到的东西。(泰诺，或称乙酰氨基酚，是许多感冒药成分混合物的一部分。如果你正在联合用药，确保你没有重复使用这种成分。)

### 架子上有许多无用的东西

几乎每个儿童感冒药都是垃圾。含有减充血剂和抗组胺剂 [的药物被认为对 4 岁以下](https://vitals.lifehacker.com/dont-bother-with-cold-medicines-for-toddlers-1684259625) 的儿童不安全。但是，如果你在医药行业，那么“天然”品牌可以用维生素、蜂蜜和草药补充剂的混合物来填补这个空白。这些其实也没什么用，但不知何故它们的价格好像是 8 美元一瓶。

在儿童和成人部分，你会发现顺势疗法，这是相当多的骗局和 [应该是非法的](https://www.consumer.ftc.gov/blog/2016/11/homeopathy-not-backed-modern-science) 。注意任何标有“顺势疗法”或成分以“X”或“C”计量的东西，而不是像毫克这样的真正单位。这些不像标签上有时暗示的那样，是简单的天然补充剂；他们是美国食品和药物管理局政策中一个奇怪漏洞的受益者，这个漏洞是 [允许他们作为药物上市，同时声称通过魔法](https://vitals.lifehacker.com/what-homeopathy-is-and-why-its-useless-1795694290) 起作用。我希望这是我编的，但我没有。节省你的钱。
