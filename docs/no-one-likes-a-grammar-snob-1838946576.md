# 没有人喜欢语法势利者

> 原文：<https://lifehacker.com/no-one-likes-a-grammar-snob-1838946576>

当你寻求写作建议时要小心。有些人想以正确的方式去做——决定你想对谁说什么，并帮助你这样做。但是有些人只是想让你符合他们“正确”写作的想法。他们是学究，认为表达自己只有一种正确的方式，像所有学究一样，他们宁愿正确也不愿聪明。他们都不是。



每个知识领域都有学究:打着教育他人的幌子守门的人。但它们经常出现，而且很明显，在语言领域，每个人都有经验，所以任何人都可以假装是专家。编辑皮帕·贝利在一篇关于语言进化的文章中对它们进行了探讨。

> 毫不动摇的迂腐不是对语言的热爱，而是一种更丑陋的东西:一种永远正确并贬低他人的欲望。它是优越的，精英主义的；语言不是交流的工具，而是身份的工具。

一个学究认为他们的语言是以现在的形式从高处传下来的。他们忽略了在他们说这种语言时形成这种语言的所有变化，并谴责此后的每一次变化。他们认为语言像物理定律一样是固定的。这就好比认为一块钱值一块钱是因为绿墨水的价值。这是史蒂夫·马丁关于访法的一个笑话:“这些人，好像他们对每件事都有不同的说法！”

学究们未能认识到语言总是不完美的，它不存在于使用它的人之外，它完全是由文字构成的。甚至《创世纪》也说上帝带走了所有的动物，“把它们带到亚当面前，看他会给它们起什么名字。无论亚当怎样称呼每一种生物，那就是它的名字。”规定主义终极之书的第二章，它明确地说上帝没有给动物命名，但是*等着看人类会怎么称呼它们*。

你同样可以剖析任何知识领域的任何形式的迂腐。但是在语言中特别容易识别，因为我们每天都在不断地使用语言，忽略了“规则”，让语言进化。你到处都可以看到好的写作拆分不定式或以连词开头或使用太多单词，好的演讲结巴、跳跃、拖尾和中断。你会看到糟糕的写作和糟糕的演讲，不管是好是坏，也会改变我们对正确事物的共同理解。

所有这些语言都有很多规则。例如，以和其他人一样的方式拼写单词，或者只在你的观众已经接受或者可以暂时腾出空间的意义上使用它们，会有所帮助。但是每一种特定的语言规则都是灵活的，一些写作可以一次打破其中的许多规则，但仍然成功。在英语中，我们最崇敬的作家是莎士比亚，他将 422 个单词引入英语； [你可以在这里看到它们全部](https://www.litcharts.com/blog/shakespeare/words-shakespeare-invented/) 。他工作在英语飞速发展的时代。感谢互联网，我们正处于另一个快速变化的时代。

学究尖叫道:“如果你不遵守我的规则，那你就没有规则了！”这完全是不真实的。仅在说英语的人群中，每天就有数百万人进行语码转换，用一种方式与朋友交谈，用另一种方式与家人交谈，用另一种方式与同事交谈。它们遵循的不是一套规则，而是几套交叉的规则。他们有自己的个人语言习惯，最喜欢的短语，他们避免使用的单词，而不期望其他人分享他们的偏好。学究看到一种不是他们自己的方言，没有想象力去认它是方言。

学究们支持当权者的语言，侮辱无权者的语言，认为这根本不是语言，这并非巧合。 [迂腐是政治。](https://lifehacker.com/everything-is-political-and-always-has-been-1818724298) 学究们会通过指出语言的其他政治用途来为自己辩护——比如 [避免使用侮辱残疾的词语](https://lifehacker.com/learn-correct-terminology-with-the-disability-style-gui-1838449345) ，或者 [使用中性代词](https://lifehacker.com/how-to-use-gender-neutral-pronouns-1821239054)——并声称它们与学究相同。但事实恰恰相反。这些“规则”使得我们的语言包含更多的人，而迂腐却排斥人。这是这场战争的本质。

好消息是学究们正在输，已经输了。语言学的亮点是像 [格雷琴·麦卡洛克](https://gretchenmcculloch.com) 这样的人，她的书 [*因为互联网*](https://www.amazon.com/Because-Internet-Understanding-Rules-Language/dp/0735210934/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/no-one-likes-a-grammar-snob-1838946576&asc_source=&tag=kinjalifehackerlink-20) 庆祝了语言的近期演变，作为一个古老而永无止境的过程的一部分。像皮帕·贝利这样的编辑在他们的工作中，即使是在最主流的出版物中，也会大声疾呼并避免卖弄学问。每当这份乏味的报纸首次尝试一个词时，人们都会关注 Twitter 账户 [《纽约时报》](https://twitter.com/nyt_first_said?lang=en) 来欢呼。通过有趣的社交媒体账户，词典编纂者利用他们自己的文化权威来削弱自己。他们宁愿聪明也不愿正确。他们都是。
