# 如何教孩子区分事实和观点

事实和观点之间的区别是当今的热门话题。近年来越来越明显的是，这是一个很难理解的概念，甚至对一些成年人来说，更不用说孩子了。因此，无论何时，或者是否，你孩子的学校决定解决这个差异，父母在家加强课程是个好主意。

## 什么是事实？

简单来说，事实就是可以被证明为真或假的东西

*   狗是哺乳动物。

*   爱莉安娜·格兰德是一名歌手。
*   有七大洲。

*   草是绿色的。

这些都很容易得到:它们不仅是事实，而且你可以证实它们是真的。但是像这样的陈述呢？

*   鹦鹉生活在地下。
*   哈里·斯泰尔斯是一个贝壳。
*   北达科他州是一个国家。
*   鸡块是液体。

你的直觉可能反对称这些陈述为“事实”，因为你说，事实应该是真实的。是的，这当然是“事实”的一个定义但是当你谈论事实而不是观点时，有时，事实是错误的。

这是因为关键在于信息是否可以证实——即使在这个过程中，你证实它们是假的。罗布·莱尔德在伊利诺伊州的弗里波特高中教高中英语，他说，当他想到事实陈述不一定是真实的时，他的学生们有点震惊。

“你可以把某件事说成事实，也可以把它说成是错误的，”他说。“这可能是你犯的一个错误，也可能是一个谎言，但你把它当成了事实。”

莱尔德说，解释这一点的诀窍在于解释事实和观点之间的关系。这就引出了一个问题:

## 什么是观点？

与此同时，观点是*不能*被证明为真或假的东西:

*   棒球是一项可怕的运动。
*   粉红色是一种极好的颜色。
*   安娜比艾尔莎好。
*   《我的世界》很奇怪。

我个人可能会想这些事情，我可能有很好的理由来解释为什么我会这么想，但这并不能使它们成为事实，因为它们是不可证明的。另外，你可能有充分的理由认为:

*   棒球是最好的运动。
*   粉红色是一种难看的颜色。
*   埃尔莎比安娜好。
*   《我的世界》很有趣。

这里令人困惑的部分是，观点可能看起来非常明显——这就是莱尔德如何将事实和观点联系起来的。

## 两者之间的关系

我们举个足球的例子:说新英格兰的战绩是 13 胜 3 负，克利夫兰的战绩是 3 胜 13 负。现在考虑一下这句话，“新英格兰比克利夫兰好。”那是事实还是观点？

莱尔德的大多数学生都说“事实”，这是不正确的。

“为什么这么说？”莱尔德会问他的学生，他们会回答，“因为新英格兰赢了更多的比赛。”

“新英格兰*赢得了更多比赛，”莱尔德说。“这是事实，”这支持了*的观点*，即新英格兰是一支更好的足球队。“你用事实来发展观点。尽管这是显而易见的——没有人会争辩说克利夫兰比新英格兰好——但这仍然是一种看法。你还是需要用事实来支撑。”*

## 如何以及为什么处理事实与观点

你还可以在这个 Pinterest 板 中找到 [工作表集合，由语言病理学家、教育家和](https://www.pinterest.com/teachtutors/teaching-fact-vs-opinion/) [Teach Tutors](https://www.teachtutors.com/) 的创始人达纳·莫纳汉策划，这是一个使用同伴导师的信息和资源网站。

这一课在今天尤其重要，因为许多人不知道从哪里获得可靠的、真实的信息。

“这是我从成人和儿童身上看到的首要问题，”莱尔德说。“我如何知道这是否是可靠的信息？”

这些辨别假新闻 的技巧，通常是伪装成事实或不真实事实的观点，包括检查来源和使用事实核实网站的重要性。莱尔德还建议进行废话测试(这在他的学生中很受欢迎):

*   **C** :信息 C *是当前的*？
*   **R** :信息 R *是否可靠*？
*   **A** :这个信息是由 A *权威*来源共享的吗？
*   **P** :信息的*目的*是什么？是陈述事实还是说服你？

当莱尔德让他的英语或新闻学学生写一篇议论文或专栏时，他会指导他们选择一个主题，但不要形成观点。他说，首先，做研究，然后用这些事实来发展你的观点。

“我认为人们在社交媒体上犯的罪，学生们经常犯的罪，是他们有自己的观点，他们去挖掘支持他们观点的事实，”他说，“他们很难改变他们的想法，因为他们带着大量的偏见。”
