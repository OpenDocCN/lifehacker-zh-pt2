# 用味精改善飞机餐

> 原文：<https://lifehacker.com/use-msg-to-improve-airplane-food-1836392738>

精神上，飞行对我来说不是问题。让我感到不安的是以每小时数英里的速度被抛向空中的身体状况。即使你没有经历鼻窦压力或晕车，坐在嘈杂的飞机上也可以改变你对周围世界的看法。事实上，它可以改变你品尝 T2 食物的方式，老实说，这是很难接受的。

Watch

根据纽约时报 T3 的报道，“与在地面上相比，我们在空中对甜味和咸味食物的敏感度下降了 30%。”然而，鲜味更容易被发现，这也解释了我在飞机旅行中突然想喝番茄汁和油炸玉米饼的原因。这也解释了为什么零食在飞机上不会受到同样的影响，*但*它给了我们一个机会去适应、反击，并增加我们旅行零食的鲜味含量。

我想你知道我想说什么。如果鲜味是你在飞机上唯一能真正品尝到的东西，那么鲜味就是我们应该在食物中增加的东西。这很容易用一点味精来完成。你只需要一点点，所以装几茶匙在小瓶、罐头或袋子里，撒在适当的饭菜和小吃上。(但是，请准备好回答关于白色晶体的问题。我发现这些小盐罐 引起的怀疑最少。)把味精放在你悲伤的航空餐上，放在一袋薯片里摇一摇，或者撒在奶油芝士条上。你甚至可以用它来增加水果的味道——在苹果上试试，它的效果非常类似于帕尔马干酪。
