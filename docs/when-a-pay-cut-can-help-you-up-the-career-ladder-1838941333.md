# 什么时候降薪可以帮助你提升职业

> 原文：<https://lifehacker.com/when-a-pay-cut-can-help-you-up-the-career-ladder-1838941333>

五年后你想在哪里？你有达到目标所需的技能吗？如果你不这样做，你可能不得不从职业阶梯上爬下来，然后才能重新爬上去——是的，这可能意味着减薪。



Okta 首席产品官迪亚乔利(Diya Jolly)采取了三次故意减薪，作为她长期职业战略的一部分。 [正如她在美国消费者新闻与商业频道](https://www.cnbc.com/2019/10/04/i-climbed-corporate-ladder-by-taking-lower-titles-and-less-pay-3-times.html)所解释的:

> 在我职业生涯的三个阶段，我已经退了一步，接受了更低的头衔和薪水。没有任何背景，我知道这听起来像一个错误的决定。但我在公司阶梯上走的每一步都是有意的，帮助我取得了更大的长期成功，无论是在成就还是薪酬方面。每一个新角色都教会了我长期发展所需的技能，让我离成为产品领导者的最终目标更近了一步。

如果你正在考虑做一个类似的决定，问问你自己你现在需要什么工作来帮助你得到你将来想要的工作。确定你是否走上了你想要的工作的正确职业道路的最好方法之一，就是看看目前担任这份工作的人的职业历史和技能。他们的职业轨迹与你的相符吗，还是他们都在你从未担任过的特定职位上工作过？

如果你和其中一个人有私人或工作上的联系，那就有必要向他们询问信息——或者，如果你不想给这种对话贴上正式标签，看看他们是否愿意一起喝杯咖啡或聊聊天。你会想知道拥有那个特定的职位是什么感觉(以确保它是你真正想要的)，并弄清楚在给自己定位一个类似的角色之前，你是否需要更新你的技能。去问问你正在面试的人，他们是否建议你在职业阶梯上往下走一步；他们可能会提出一些替代方案，比如课程或证书。

不降薪也有可能找到你想要的工作。不要去参加工作面试和薪资谈判，假设你将不得不接受一份比你之前收入更低的工作。然而，要为这个话题的出现做好准备——准备好解释你为什么寻找这种类型的角色，你目前的技能如何帮助公司解决问题，以及你希望从短期和长期的职业发展中获得什么。

最终，你会想要建立一份职业历史和一套技能，让你发展关系网和声誉，去你想去的地方——即使这需要短暂的减薪。正如乔利所说:

> 为了实现你的目标，你需要关注长期发展，而不仅仅是下一次晋升。确定达到目标所需的技能，并留出时间来培养这些技能，是你获得工作保障的最佳方式。

如果你最终接受了降薪，我们有一些建议来帮助你管理你减少的收入——并为你未来的职业和财务做好准备。如果你做了正确的决定，你会在工作中更快乐，并且很快会发现自己已经处于一个重新开始攀登职业阶梯的位置。
