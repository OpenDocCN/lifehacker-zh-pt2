# 这个聊天机器人帮助你与生活中的疫苗否认者交谈

> 原文：<https://lifehacker.com/this-chatbot-helps-you-talk-to-the-vaccine-deniers-in-y-1834169246>

不想给孩子接种疫苗的父母可能看起来与我们如此疏远，以至于很难理解他们是如何形成自己的观点的。但他们大概都有当时在身边的朋友和家人，却不知道 [说什么](https://lifehacker.com/how-to-make-sure-your-friends-dont-become-anti-vaxxers-1832815146) 。



羞辱不起作用，不管你多么希望它起作用。但是你可以成为一个有帮助的，支持性的谈话的人，引导这个人远离阴谋论，走向以证据为基础的方法来保持他们孩子的健康和安全。在你开始谈话之前，你可以使用威尔的“谈话教练”来帮助你练习说什么。

它不是一个充满反暴力比喻的复杂人工智能，而是一个引导对话，引导你找到更多有用的回答和问题。例如，当朋友提到他们因为某些风险而不打算接种疫苗时，你不应该带着争论跳入英雄模式。相反，教练建议说:“好吧，你为什么会这么想？”

倾听一个人是进行富有成效的谈话的第一步，就共同的目标达成一致也是如此，比如保护孩子的安全。教练还包括一些避开话题破坏者的建议——当他们谈论大型制药公司时，不要咬。

如果你谈论的是你的孩子，有一个版本的机器人可以帮助你和你的伴侣进行对话。Verywell 还有一份与你的医生 交谈的 [指南，你可以把它带到办公室，就你关心的问题进行深入讨论。](https://files.verywellhealth.com/doctor-discussion-guides/vaccination_doctor_discussion_guide.pdf)
