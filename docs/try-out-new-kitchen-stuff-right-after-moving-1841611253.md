# 搬家后马上尝试新的厨房用品

> 原文：<https://lifehacker.com/try-out-new-kitchen-stuff-right-after-moving-1841611253>

每一次搬家最糟糕的部分是试图像一个正常人一样生活在之间满是盒子的门槛空间里。但是，事实证明，这段恼人的时间恰恰有一个好处:它提供了投资新炊具的绝佳机会，因为你别无选择，只能每天使用它。

我刚刚结束了三年来的第二次跨国搬家，多亏了一点小小的运输问题，我的东西还要一周才能到达。所以我买了一个碳钢煎锅和一把抢救用的切肉刀，这两样我以前都没有过，在我剩下的财产慢慢回到我身边的时候，我可以用它们。由于除了一个借来的炖锅，我没有其他炊具，我不得不在每一个可以想象的烹饪场景中测试我购买的两个炊具。(我还必须学习如何对碳钢 进行时效处理，这比我想象的要容易得多。不出所料，我非常喜欢我的新宝宝，迫不及待地想把他们介绍给家里的其他人——但如果我不是，这充其量只是暂时的不便。这是最理想的试用期。这并不是说你应该在每一个吸引你眼球的小玩意上砸下一大笔钱——搬家已经够贵了。为了达到最大效果，你买的任何东西都必须填补你现有收藏的一个漏洞，所以犒劳一下自己，买几样你多年来一直想要但由于某种原因一直推迟的东西。无论你是想学一门全新的技能，比如磨刀，还是只是想看看在铸铁或碳钢上烹饪是什么样子，当最后一个盒子被打开时，你就会确切地知道*你对它的感觉。*
