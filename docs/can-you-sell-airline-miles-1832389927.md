# 你能卖航空里程吗？

> 原文：<https://lifehacker.com/can-you-sell-airline-miles-1832389927>

假设你正在黑市 搜寻一些泄露的密码，这时你发现了一个航空里程的宝库。“有意思，”你想，“我也有一些航空里程想卖掉！我该怎么做呢？”

Watch

根据，出售里程本身并不违法，假设它们属于你。如果你在网上向一个人出售里程，警察不会来敲你的门。但是这是有风险的，应该避免:

> 如果你想把积分卖给你在网上认识的人，你必须真的为他们买票，并相信他们会还给你。
> 
> 由于忠诚计划禁止出售或交换积分，如果航空公司抓住你，他们有权将你逐出他们的计划，取消你的机票并寻求偿还。

忠诚计划本身禁止出售里程(授权网站，如 [、points.com](https://www.points.com/how-it-works/exchange-points/)，允许你将里程从一个忠诚计划转换成另一个)。如果你真的出售它们并被抓住，你可能会被罚款:据报道，一名高级会员 [去年被罚款 50 万英里](https://thepointsguy.com/news/aadvantage-member-fined-500k-violating-terms/) ，因为他在几年的时间里出售他的。

这就是为什么，正如 CreditCards.com 指出的，为了出售你的里程，你必须为一个陌生人(或者你的销售对象)买票，并让他们给你付款。如果你曾经在网上随机购买过任何东西，你就会知道从安全角度来看，这不是一个理想的情况。 [骗局比比皆是](https://www.cnbc.com/2018/06/14/online-shopping-can-be-a-minefield-of-scams-heres-how-to-protect-yourself.html)——如果它们随着你的里程消失，那你就倒霉了。即使它成功了，你拿到了你的钱，如果航空公司发现了，它仍然会惩罚你。

“越来越复杂的工具正被用来识别不寻常的活动(如以你不认识的人的名义预订的昂贵奖项)，如果你被抓住，你的账户可能会被关闭，”积分专家 报道说。"当这种情况发生时，你可能会一无所获，所有的奖励都会被没收."

然而，您可以根据计划， [与配偶或家庭成员](https://twocents.lifehacker.com/what-to-know-about-sharing-airline-miles-with-family-an-1828223485) 分享里程，无需盈利。坚持这一点，而不是你在 Facebook Marketplace 上遇到的那个人。
