# 为现在困在家里的孩子制定一个白天的学习计划

> 原文：<https://lifehacker.com/create-a-daytime-academic-schedule-for-kids-stuck-at-ho-1842357145>

随着全国各地的学校关闭，家长们争相在家工作或寻找儿童看护，人们很容易将这视为一个漫长的大雪天。用一些小吃把他们撑在电视机前，安然度过。但是专家一致认为，在接下来的几周(或更长时间)里，父母能为孩子做的最好的事情就是让他们马上有一个时间表。

Watch

全国各地的学校在是否提供在线学习选项上会有所不同；即使有，也会有学生无法参与，因为他们缺乏必要的技术。但是，比给他们布置作业或表现得像一个临时家庭学校老师更重要的是制定一个时间表，在某种程度上反映他们正常的学校生活——然后坚持下去。

艾米·乔伊斯和玛丽-简·威廉姆斯为《华盛顿邮报》 撰稿:

> Melanie Auerbach 是该地区 Sheridan 学校的学生支持主任，她有一些关于如何在中断期间让孩子们走上正轨的想法。她的主要建议是什么？制定一个时间表，并坚持下去。
> 
> 奥尔巴赫说:“夏季大脑缺乏一个时间表，一个常规，睡眠。“他们不会忘记如何阅读。…他们已经忘了如何学习。寒假过后，当孩子们回来时，他们需要一周的时间来调整。夏令时过后，他们需要一周时间来重置。改变他们的日常生活会有很大的不同。”

因此，虽然你不需要站在白板前教长除法，但你可以帮助他们安排一天的时间，让他们保持学术思维，设定开始时间、休息时间和一天中不同的“主题”。这里有一个例子:

我为我九岁的儿子制定了一个类似的时间表。他起初很抗拒，因为，当然，他*想把这件事当成下雪天或请几天病假。但是我们一起想出了这个计划:*

我原来的作息时间有更多的创意玩耍时间，更少的阅读时间，没有休息时间，这是他可以自由支配的时间。这些调整足以让他入伙。对于不识字或不会看时间的孩子，你可以建立一个类似的图表，用图像来表示活动，并用计时器来帮助他们知道什么时候该做下一件事。

如果他们在学校关闭期间要和父母或其他照顾者呆在家里，这种时间表会很有用，但如果他们的照顾情况每天都在波动，这也有助于给他们的生活带来更多的一致性。

现在也是深入挖掘你的壁橱、抽屉和地下室的好时机，以恢复旧的艺术和手工艺项目、科学工具包、拼图或其他他们已经忘记的学习游戏。我拿出了几个我儿子早已忘记的选项，让他在一天的“创造性游戏”中有更多的可能性。

他知道他也可以利用这段时间来画漫画书或建造某种乐高杰作——而且他负责弄清楚在这段时间里每天要做什么——但我想给他一些新鲜的想法。因为看起来我们要长期坚持下去了。
