# 提醒:性病检测在疫情期间仍然很重要

> 原文：<https://lifehacker.com/reminder-std-testing-is-still-important-during-the-pan-1845080398>

疫情打乱了我们生活的许多部分，包括性传播疾病的检测。在诊所关闭和供应短缺之间，许多测试被推迟，因为测试率在过去几个月中大幅下降。所以，这是对你的提醒，如果你有患性病的风险，你应该确保寻求检测，即使这比通常更不方便。

Watch

疾控中心对性病检测的建议是 [此处](https://www.cdc.gov/std/prevention/screeningreccs.htm) 。这些建议根据你是谁以及你每年是否有多个性伴侣而有所不同。我们中的许多人应该至少每年做一次淋病、衣原体检查，在某些情况下还要做一次、艾滋病毒和梅毒检查。

不幸的是，衣原体和淋病检测试剂盒现在供不应求。卫生与公众服务部最近给医疗服务提供者 发了一封信，建议包括孕妇和男男性行为者在内的高危人群优先接受检测。他们还指出，如果有人确诊患有衣原体或淋病，不需要等待测试结果就可以让他们的伴侣开始治疗。

更有趣的是，现在阿奇霉素 也短缺 [，阿奇霉素是一种常用于治疗衣原体和淋病的抗生素。每次你在 Youtube 上看到一个庸医告诉他们的追随者服用“羟氯喹和阿奇霉素和锌”来预防 COVID (](https://www.accessdata.fda.gov/scripts/drugshortages/dsp_ActiveIngredientDetails.cfm?AI=Azithromycin%20Tablets&st=c&tab=tabs-1) [其中不起作用，顺便说一下](https://vitals.lifehacker.com/that-viral-video-about-a-covid-19-cure-is-wrong-1844533029) )，他们正在影响性传播感染患者获得必要药物的可能性。

其他性传播感染的检测不一定会受到短缺的影响，尽管一些诊所可能不会恢复全面运营，这使事情变得复杂。如果你有症状，或者你属于高危人群，联系你的提供者或者你当地的性健康诊所，问问他们能做些什么。通常他们会给你一些选择。例如，一些诊所因检查和其他常规护理而关闭，但仍会进行筛查。

还有艾滋病毒的家庭测试和其他性传播感染的邮寄测试，如在 [让我们检查一下](https://www.letsgetchecked.com/us/en/home-std-test) 提供的测试。出售这些测试的公司通常不接受保险，但是*接受 FSA 和 HSA 卡。如果你在今年春天疫情的事情结束时推迟了性传播感染测试，现在就花点时间检查一下你的选择，如果你需要的话，确保去做一次测试。*
