# 尝试这些卧推变化

> 原文：<https://lifehacker.com/try-these-bench-press-variations-1830588845>

厌倦卧推了吗？希望 [我们的挑战](https://lifehacker.com/c/lifehacker-fitness-challenge) 一直让事情保持足够有趣，你不会*梦想*厌倦卧推...但是也许你想在举重的时候锻炼一些其他的肌肉。这里有一些你可能想尝试的新闻变化。



### 拒绝新闻

我实际上从来没有做过这个，因为 [看起来有点恐怖](https://www.bodybuilding.com/exercises/decline-barbell-bench-press) 。也许我应该试试。你躺在一张倾斜的长椅上，让你的头低于你的臀部，然后把重量直接推到天花板上。这比标准的卧推更能锻炼你的胸部和背阔肌(背部两侧)。确保有一个观察员，或者更好的是一个带安全装置 的 [架子，这样就不会有杠铃掉在你脸上的风险。(你也可以用哑铃来做这个动作，或者其他任何动作。)](https://vitals.lifehacker.com/how-to-bench-press-safely-without-a-spotter-1830337071#_ga=2.104519945.849348195.1542638360-1456718367.1520458611)

### 倾斜压力机

这张正好相反: [板凳斜向上](https://www.bodybuilding.com/exercises/barbell-incline-bench-press-medium-grip) ，所以像是从 la-z-boy 压过来的。如果你不能决定今天是要卧推还是顶推，这是一个完美的妥协。

### 高架压力机

这个不完全是卧推的变化，但感觉它属于同一个俱乐部。你把杠铃(或哑铃)从肩膀上举过头顶。你可以站着或坐着做这个动作。站立需要其他肌肉更加稳定，但是如果你想真正专注于手臂和肩膀而不需要下半身的帮助，坐着可能会更容易。

* * *

那么，你是如何进行卧推训练的？我目前的日常训练包括杠铃卧推、头顶站立式卧推(有时是杠铃，但我发现我更喜欢哑铃)，偶尔我会做一圈重量板器械，包括肩部卧推和倾斜卧推。你最近在举什么？
