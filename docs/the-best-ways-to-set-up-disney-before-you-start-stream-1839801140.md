### 安装 Disney+应用程序后，我需要调整哪些设置吗？

为什么，是的！事实上，一旦你在智能手机或平板电脑上点击 Disney+,你要做的第一件事就是点击它的设置菜单并做一些更改。(你也可以通过网络/你的电视应用程序来做这件事，我现在无法登录任何一个来展示给你看。啊，发射日。)

首先，也是最重要的一点，你应该为任何将要访问你的账户的人——当然是家庭成员——设置不同的用户配置文件。你可以有多达七个，每个都可以有自己漂亮的小迪士尼主题图片。父母也可以将帐户标记为属于“孩子”，这将他们在服务上看到的内容限制为家庭友好的内容。

回到主设置屏幕，我建议点击应用程序设置，这会弹出这个屏幕:

我已经设置了我的首选项，所以你看到的不是你安装应用程序和/或首次启动 Disney+服务时的默认设置。我已经将我的设备设置为只通过 wifi 播放 Disney+,因为我不想烧光我的蜂窝数据计划，但如果你对你将观看的内容有点紧张，你也可以选择“自动”或“保存数据”模式。

类似地，我只通过 wifi 从 Disney+下载内容，但我确实调整了应用程序以抓取高质量的视频——而不是它的默认设置。这会占用我的 iPhone 更多的空间，但这就是我的方式。(您可以在特定电影或电视节目信息页面的“详细信息”选项卡上查看其可用的格式。)

虽然您可以从 Disney+下载任何您想要的内容，但您必须至少每 30 天将您的设备登录到互联网一次，以重新认证许可证。对于大多数人来说，这应该不是问题，但是如果你打算休长假，并且不能忍受远离你最喜欢的经典动画，这是一个很好的参考。
