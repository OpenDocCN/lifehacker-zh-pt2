# 这些假黄油真的很好吃

> 原文：<https://lifehacker.com/these-fake-butters-are-good-actually-1834313812>

90 年代对脂肪来说是一个黑暗的时代，但我记得对黄油来说尤其暗淡。一瓶邪恶的"我不敢相信这不是黄油"喷雾在我童年的厨房里一直存在，我把这种乳黄色的液体既用作食用油又用作调味剂。我的祖母仍然用熏肉油烹饪，但一桶乡村瓦罐代替了黄油作为配料。这不太好。



但是不含乳制品替代品的世界已经发生了变化，T2 出现了新的人造黄油。有 [美代子的欧式养殖纯素黄油](https://miyokos.com/collections/vegan-butter) (由发酵腰果奶油制成)和基于蔬菜的 [融化有机](http://meltorganic.com) ，这两种黄油都声称其作用和味道像真正的黄油。

我喜欢一款 [腰果基假奶酪](https://lifehacker.com/i-ate-a-bunch-of-vegan-cheese-and-it-was-actually-quit-1731244460) ，我喜欢融化产品上的包装，所以我把它们都买了下来，还有叶 olde standards Smart Balance 和 Earth Balance。他们的第一个测试很简单:他们被放在一片简单的法国面包上食用，这样味道和口感就能得到恰当的评价。

首先，我尝试了美代子的，因为我很兴奋地尝试了不是真正的腰果黄油*腰果黄油*(至少不是花生酱的意思)。它是块状的，很像别致的培养黄油。我打开了它。它看起来不像黄油，而是灰白色的雕塑粘土——事实证明，这也是我成长过程中的重要特征。直接从冰箱里拿出来，它比牛奶黄油更容易传播，因为这种性质的非乳制品通常会这样做，但不像乡村瓦罐那样容易传播。

然后是真心话时间。

读者，如果没有人告诉我这是腰果，我不会马上知道。它是奶油状的，略带咸味，带有你通常在培养黄油中发现的酸奶味。我吃了一些涂有奶油的面包，然后又去了美代子餐厅做比较。乳制品的味道更浓郁，余味更浓，但只是轻微的。美代子完成得更干净一点，再次让我想起酸奶。我印象深刻。我又吃了两块，然后想起我还有更多假黄油要尝。

接下来我尝试了融化，它装在一个桶里，看起来和我处理过的所有其他黄油替代品几乎一样。它缺乏有教养的唐，在我嘴里融化得有点太快，但味道是廉价的奶油，只有非常轻微的植物油般的回味。

然后我尝试了地球平衡和智能平衡，被吓坏了。与美代子的和 Melt 相比，它们的合成余味非常突出。

不用说,地球和智能天平都没有让进入第二轮——吐司试炼。(实际上它是油炸面包，但是“吐司审判”打字起来更有趣。)两者都声称像“真正的”黄油一样融化和变成褐色，所以我把它们涂在更多的法国面包上，点燃煎锅，每面都煮几分钟，直到金黄酥脆(见上图；美代子的在上面，下面融化。它们都比真正的 T4 黄油需要更长的时间来变成褐色，但是一旦变成褐色，它们尝起来就和奶油面包没有什么区别了。搭配野外烧烤的 [纯素炒面](https://skillet.lifehacker.com/two-grilled-cheese-alternatives-for-people-who-cant-eat-1795388350) ，你就有了最能骗过的纯素烤奶酪(尽管可能不是全部)。

当用作食用油时，美代子的和融化的之间的微妙之处就消失了— 但是考虑到它们在我当地的布格杂货店是同样的价格，我会首先选择美代子的，这是发酵腰果的另一个胜利。我太喜欢它了，我可能不会和我的素食主义者朋友分享它，尽管我会宣扬养殖腰果的好处。(我的乳糖不耐症男朋友可以吃点，我猜。)
