# 使用此浏览器扩展预览您的 Google 搜索结果

> 原文：<https://lifehacker.com/preview-your-google-search-results-using-this-browser-e-1833693596>

毫无疑问，我的谷歌搜索结果中的第一个链接从来都不是我真正在寻找的东西。因此，我通常的做法是在浏览器上打开所有看起来可能很合适的搜索结果，然后逐一浏览。根据我搜索的内容，可能会有很多。



[谷歌结果预览器](https://chrome.google.com/webstore/detail/google-results-previewer/mkmjdljkedjpedbceoaaghdmcnipdcjf/related?ref=producthunt) 是 Chrome 的一个扩展，让这个过程变得更简单。Chrome 扩展允许你通过将鼠标悬停在网页上来预览网页，而不是打开每个单独的链接来查看搜索结果。这样，当你最终打开那个额外的标签或者点击一个链接时，你就知道你进入了什么。

预览最初显示为搜索结果页面右下角的一个小窗口，你可以从我最近对“华夫饼三明治”的重要搜索中看到。

如果你喜欢你看到的东西(不可否认，我对这个基督山华夫饼三明治很感兴趣)，你可以悬停在那个窗口上，获得更大的视图，就像传统的网页一样可以滚动。

要真正打开链接并进入页面，你必须像平常一样在搜索结果中点击它。

这是一个巧妙的技巧，让我的搜索更有成果，或者至少涉及的标签更少。

当你不想使用它的时候，像所有的 Chrome 扩展一样，你也可以通过浏览器工具栏上的一个按钮来关闭它。
