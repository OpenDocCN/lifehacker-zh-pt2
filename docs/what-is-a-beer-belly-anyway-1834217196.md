# 到底什么是啤酒肚？

> 原文：<https://lifehacker.com/what-is-a-beer-belly-anyway-1834217196>

拥有一个大肚子是 [与](https://www.health.harvard.edu/staying-healthy/abdominal-obesity-and-your-health) 有更大的心脏病和死亡风险，但要了解啤酒肚(实际上你不能责怪啤酒)，你必须看看它的内部。



### 一些脂肪就生活在皮肤下面

皮下(字面意思是“皮下”)脂肪在你的身体里无处不在。你的耳垂是脂肪做的。这是你可以触摸和挤压的脂肪。这可能很烦人，但这不是你被警告过的有害腹部脂肪。

你的身体把皮下脂肪放在你的基因和激素告诉它的任何地方。女人更倾向于把它放在屁股和大腿上，男人更多放在肚子上，但是很多人都是两者兼而有之，或者两者都不符合。对于一个正常健康的身体来说，所有这些都是可能的，尽管如果你也有内脏脂肪，你可能会注意到你的皮下腹部脂肪更多。

### 你的器官周围的脂肪是需要担心的类型

如果你听说过“腹部脂肪”不好，那就不要再去想什么是爱柄了，开始想想你的腹部里面有什么。与健康风险最密切相关的脂肪是包围你器官的内脏脂肪。这块肥肉是里面的*，在一堵肌肉墙后面。如果你戳你的腹部，你会感觉到柔软的皮下脂肪，然后你会感觉到肌肉壁。内脏脂肪已经过了那个。*

正是这种脂肪 [增加了你患心脏病、胰岛素抵抗和慢性炎症的风险](https://www.health.harvard.edu/staying-healthy/taking-aim-at-belly-fat) (或者可能是你已经处于风险中的信号)。这就是为什么 [腰围是你健康风险](https://lifehacker.com/the-right-way-to-measure-your-waist-size-and-check-your-1778119942) 的一个有用指标。它并不完美，但比《身体质量指数》或《总重量》更好地讲述了这个故事。

### 还有其他方法可以让肚子肿胀

啤酒肚可能来自其他原因，我不仅仅指怀孕。一些健康状况会导致您的肝脏或其他器官增大。其中一个特别与饮酒有关。

一生酗酒会导致肝脏损伤，称为肝硬化。在极端情况下，这可能导致腹水，即腹部内的液体积聚。如果你有腹水，你的医生可能会排出液体，但这是一个严重的情况，通常表明你需要肝移植。

除此之外，啤酒 [和啤酒肚](https://www.nature.com/articles/ejcn200939) 并没有什么特别的联系。如果你摄入大量的酒精，你可能会摄入大量的卡路里，这会让你摄入更多的这两种脂肪；你也可能有其他心血管疾病的风险因素。但你不能把啤酒肚归咎于啤酒。
