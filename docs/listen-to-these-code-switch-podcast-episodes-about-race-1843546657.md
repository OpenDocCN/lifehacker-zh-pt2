# 和你的孩子一起听这些关于种族的“代码转换”播客片段

> 原文:[https://life hacker . com/listen-to-these-code-switch-podcast-episodes-about-race-1843546657](https://lifehacker.com/listen-to-these-code-switch-podcast-episodes-about-race-1843546657)

随着这个奇怪的学年慢慢接近尾声，我们都在深入挖掘对我们的孩子有教育意义和吸引力的内容——但我们不必亲自策划，因为现在没有时间做这些。没有盯着屏幕看的教育内容会得到加分，如果它解决了一个重要的社会问题，如美国的种族和种族主义，会得到额外的加分。

Watch

NPR 播客展示了有色人种记者关于种族的对话，涉及从政治、流行文化到历史和体育的方方面面。它通常面向成年观众，但许多代码转换剧集可以被家庭用来进行更深入的种族对话。于是，NPR 在[【Spotify】](https://open.spotify.com/playlist/45yPXWm0KwvoDfAZogkZXh)上创建了一个 19 集播放列表 ，让家长们开始体验:

> 我们梳理了剧集，以确保它们没有亵渎、图形引用和其他成人内容。(虽然谈论种族和种族主义总是很复杂，所以父母们，在这里用你们的判断力。)我们的剧集从来没有所有的答案，我们希望这些能为一些老式的餐桌讨论开辟空间。

在 NPR 的网站上，Code Switch 的助理编辑娜塔莉·埃斯科瓦尔(Natalie Escobar)也从播放列表中概述了该团队最喜欢的 10 集，为父母提供了每个主题的简要总结以及为什么它对孩子有好处。例如，当冠状病毒从中国传播时，亚洲人和亚裔美国人经历了仇外心理和种族骚扰，这一集有助于解释孩子们可能已经在自己的生活中看到或经历过的种族主义。或者“ [《两个学区的故事》](https://www.npr.org/2019/09/11/731867149/a-tale-of-two-school-districts) ”可以帮助学生理解为什么他们的校舍和课本质量与朋友的不同。

其他主题包括:

*   [黑人运动员如何利用他们的平台](https://www.npr.org/2019/02/27/698102193/on-the-shoulders-of-giants) 来抗议美国历史上的不公正
*   移民政策 的 [波动如何给无证移民及其家庭造成压力，尤其是在“混合身份”家庭中](https://www.npr.org/2019/11/12/778505580/status-update)
*   爱探险的朵拉 如何帮助美国迎来多元文化儿童节目的浪潮
*   志愿学校融合计划，至今仍然存在

你可以在 [NPR 的网站](https://www.npr.org/sections/codeswitch/2020/05/05/850777529/the-very-best-code-switch-episodes-for-kids) 上阅读埃斯科瓦尔的剧集摘要，在 Spotify 上查看 [给孩子们的完整播放列表。](https://open.spotify.com/playlist/45yPXWm0KwvoDfAZogkZXh)
