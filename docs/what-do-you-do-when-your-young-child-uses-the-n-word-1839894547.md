# 当你的小孩使用 N 字的时候，你会怎么做？

> 原文：<https://lifehacker.com/what-do-you-do-when-your-young-child-uses-the-n-word-1839894547>

有时父母希望他们能和孩子重新进行一次对话。“大话”会在我们最意想不到的时候给我们当头一棒。本周的 [家长咨询](https://lifehacker.com/c/parental-advisory) 问题来自一个孩子的家长，他在 6 岁时第一次听到并使用了 N 字。但是，他们是否以最好的方式处理了这件事？



他们写道:

> 我儿子(6 岁的时候)在 YouTube 上听到了 N 字。一个游戏玩家用它来亲切地斥责刺猬索尼克中的一个角色(我们再也不看那个人的频道了)。他立即重复了一遍。我吓坏了，我们不得不长谈为什么你不用那个词。永远不会。但是他很难理解为什么这个单词比其他单词差那么多。例如，几个星期后，我们走进一家快餐店，他大声说“为什么我们不用 N 字呢？”所有人都转过头来盯着我。
> 
> 我应该如何导航呢？

亲爱的家长，

事实上，几周后你儿子仍在考虑这件事，这表明他真的很困惑。这可能源于他第一次使用时你对他的反应程度。我不知道你到底说了什么，但你用了“吓坏了”这个词，所以我猜他是被高度反应吓到了。你的信息可能在你强烈的情绪中迷失了。

你肯定不希望你的孩子长大后成为那种会用脏话的人。但对他来说，在那一刻，这真的只是另一个词——特别是如果他听到你认为听起来充满爱意的语气。他不可能像你一样理解它的意思，因为他没有你所拥有的上下文。所以他需要的是语境。

我联系了亚利桑那州立大学人文项目创始主任尼尔·莱斯特 ，征求他的意见。莱斯特在他的职业生涯中研究、写作和教授了大量的非裔美国人文学、文化和其他与种族相关的主题，包括 N 字。

莱斯特说，“更少的是，‘你不用这个词’，更多的是，‘这是这个词的问题所在。’”“这意味着你必须谈论种族问题。”

莱斯特说，仅仅强调这是一个我们从来不用的词的问题是，你可能会无意中鼓励与你试图完成的相反的事情。现在，有了这么大的反应，你可能会让这个词看起来很尖锐，而不是成功地告诉人们为什么它的使用是如此令人不快。

你问你应该如何处理这个问题，理想情况下，我会说在你儿子接触到 N 字之前，你就已经和他谈过种族问题了。这将包括关于奴隶制和民权运动的适龄对话——在 6 岁时，任何种族的孩子都可以理解歧视固有的不公平。寻找及时的理由来解决这个话题——解释新闻中报道的抗议背后的原因，或者谈论(并详述)他们在学校学到的黑人历史。

如果在你儿子接触到 N 字之前，你已经和他谈过美国的种族关系和歧视问题，这将是在你之前谈话的基础上的另一个机会。这时你会停下 YouTube 视频，说，“哇，我刚听到一个我不喜欢的词。你知道这意味着什么吗？让我解释一下为什么我们不用这个词。记得我们说过...?"

但既然听起来第一次不是这样的，莱斯特建议你现在做的*是让自己做好准备，以便能够更有效地 [与他就种族](https://lifehacker.com/how-to-talk-to-young-kids-about-race-1797811655) 展开对话。他说，同样重要的是“自我评估”你自己对种族的感觉，以便在谈论这个话题时更加自如。*

*“我不知道我会不会提出来，”他说，“但她需要一些工具来武装自己，以便(在未来)与他进行适龄的对话。”*

*一个很好的切入点是莱斯特的文章 [教导宽容](https://www.tolerance.org/author/dr-neal-a-lester) 、[*中的 N 字:通过历史的断开连接？*](http://The N-Word: Connected Through Historical Disconnect?)*以及 [本期采访他关于](https://www.tolerance.org/magazine/fall-2011/straight-talk-about-the-nword) 这个词的历史和用法。**
