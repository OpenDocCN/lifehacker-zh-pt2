# 使用谷歌助手的翻译模式用某人的语言与他们交谈

> 原文：<https://lifehacker.com/use-google-assistants-interpreter-mode-to-talk-to-someo-1840397512>

下次你想和说另一种语言的人交谈时，你也许可以只用你的手机。

Watch

谷歌今天开始在 iOS 和 Android 设备上推出谷歌助手的解释器模式。该功能允许你与说另一种语言的人进行常规对话，并让应用程序实时翻译你们两人所说的话。由于它已经内置在助手中，你也不需要在手机上安装翻译或其他应用程序。

要使用该功能，你可以说“嘿，谷歌，做我的德语/法语/意大利语翻译”或“嘿，谷歌，帮我说日语”，你的手机(如果有该功能)将启动谷歌翻译，并开始听你的对话。

谷歌一直在拉斯维加斯的凯撒宫和旧金山机场的凯悦酒店等几家酒店的门房和前台测试这一特殊功能，并通过谷歌家庭设备和智能显示器进行测试。

这种体验类似于谷歌翻译中已经提供的“对话”功能，但更加无缝。

作为该对话功能的经常用户，我还要指出，为了让它成功工作，你会希望在尽可能安静的地方进行这些对话，并尽可能使用简单的语言。

如果你曾经使用过任何人工智能转录工具，你就会明白它有其局限性。一台电脑，而不是人类，正在处理这些翻译。你对计算机越简单，它对你的翻译就越好。说得太快，使用复杂的语言，或者在有很多背景噪音的地方说话，都不会给你带来好的结果。

这里面的限制，可以使在餐馆点餐的过程或者在你无限期错过火车后弄清楚该做什么变得更容易。
