# 一般买家真的应该在乎性能吗？

> 原文:[https://jalopnik . com/should-the-average-buyer-really-care-performance-1837302362](https://jalopnik.com/should-the-average-buyer-really-care-about-performance-1837302362)

作为 Jalopnik 的常驻购车专家和专业购车者，我会收到电子邮件。很多邮件。我决定挑几个问题，试着帮帮大家。本周，我们将讨论买家优先、二手车延期保修谈判以及 trims 之间的性能比率。

Watch

首先，大多数人真的关心自己新车的“性能”吗？他们应该吗？一位读者写道:

> “有一天，我在选购一辆跨界车，我告诉销售人员，我将试驾 CR-V 和 CX-5。本田的销售人员试图更好地描述他的产品，他说……“嗯，本田有涡轮发动机，而马自达没有，所以它更快。”
> 
> 尽管两辆车的马力数相似，加速时间也相差不远，但这让我开始思考。购买普通汽车的人会在乎这些东西吗？"

作为一个在网上阅读汽车之前很久就已经在杂志上研究过四分之一英里和 0-60 英里/小时的人，即使我也承认大多数买家真的不在乎一辆车能在 7 秒内达到 60 英里/小时，而另一辆车能在 7.2 秒内达到 60 英里/小时。显然，购买高性能车的人想知道这些事情，但让我们现实一点——通常买家在走进经销商之前就已经知道规格表了。绝大多数汽车购买者在不同程度上优先考虑款式、功能和价格。

购车是情绪化的，一个品牌或一辆车的“感觉”或它的形象，也是决策过程中的一个因素。纯粹基于事实来销售汽车不会给你带来任何顾客。

如果你是一个无法决定买哪辆车的人，并且你在四分之一英里的时间里汗流浃背，你要知道:两种车都开，买你最喜欢的。不要让纸的东西动摇你太多。

接下来，哪辆车的性能变体与普通版本的比例最好？

> “我有一个理论。嗯，我想更多的是一种假设，我希望你有一些聪明的方法来测试它。
> 
> 这就是:G 级的“性能”变体与基础的比率最高。我觉得每个 Si 对应 100 个 Civics，每个 r 对应 10 个 Civics，每个 E63 对应 50 个 E350。一百万个 320 代表一个 M3。一千架 F-150 战斗机换一架猛禽。
> 
> 但似乎每隔一个 G 级是 G63 AMG。这是为什么呢？贵版和基础版之间有没有其他看起来接近 1:1 的型号？

这是一个有趣的思维练习，不科学地说，似乎你在街上看到的大多数奔驰 G-wagens 都是 full tilt AMG 版本。当然，人们购买它们并不是因为他们需要 G-wagon 的 AMG 性能，而是因为这些汽车经常吸引某些喜欢展示财富形象的买家，他们希望向世界展示他们能够购买最昂贵的汽车。

这也与这样一个事实有关，与思域或 F-150 或其他车型相比，没有多少 G-wagens 开始生产，所以比例显然会有利于生产数量较少的汽车。

我不确定哪个型号的普通版本和性能版本的比例最接近。自然，我们不得不把兰博基尼和法拉利这样的外国车排除在外，因为每辆车都是某种性能版本。然而，如果我大胆猜测的话，我会说，我认为性能更高的特斯拉 Model S 汽车比电池更小的汽车更多。

最后，如果你购买延长保修，经销商会不会在二手车价格上更加灵活？

> “我想买一辆 2016-2017 款奥迪 SQ5 或 Jeep GC Summit，价格在 3.5-4 万美元之间。虽然我计划无论买什么都要一个独立的 PPI，但我更喜欢 CPO，因为它包含了延长保修期。据我所知，保修是经销商的高利润部分，因此似乎有机会通过谈判获得更多保修范围而不是更低的价格来增加交易价值。也就是说，如果我同意支付他们的车辆标价，要求奥迪或吉普经销商延长保修 1-2 年是否合理？”

所以理论上，这是可行的。然而，保额大的保修不是几百美元，而是几千美元。现在你可以将 3000 美元的保修价格砍到 2200 美元。但是，如果你要把它计入二手车的销售价格，经销商需要有 2200 美元的利润才能在广告价格内运作，而他们通常没有。无论你如何划分，6 万美元以下二手车的典型议价幅度都在数百美元左右。可能有几个人敲了一千左右的竹杠。我建议你先关注车价，然后是保修。
