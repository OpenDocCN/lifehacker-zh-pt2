# 如果你想养成新习惯，那就少养成一些习惯

> 原文：<https://lifehacker.com/if-you-want-to-build-new-habits-build-fewer-habits-1835099898>

你试图创造的习惯越多，就越难保持下去。

我早就知道了。去年，我建立了一个电子表格来帮助我跟踪一系列生活指标(睡眠、营养等。)以及我想优先考虑的活动和习惯。这是一个交易，如果我做了习惯，我把细胞变成绿色；如果没有，我就把牢房变成红色。



我很快了解到 [没有完全绿的一天](https://www.nicoledieker.com/2019/03/29/how-to-create-the-systems-structure-in-which-to-do-your-best-work-part-3/) 。换句话说，没有哪一天我会去锻炼*和*看书*和*和朋友聊天*和*练钢琴*和*按时睡觉*和*等等。

这就是为什么我不是“不要打破链条”习惯追踪者的超级粉丝。对我来说，七天中有五天打到已经足够好了。

有趣的是，当生活变得紧张时，我学会了改掉哪些习惯。即使我热爱我的钢琴，练习经常是首先要去的习惯之一；感觉就像在我所有其他工作的基础上增加了更多的工作。

另一方面，锻炼通常是最后一个消失的习惯。只要我坚持早上的瑜伽练习和每周的 Les Mills 课程，我可以让其他事情顺其自然——当我太忙而不能去基督教青年会的时候，我会满足于“只要我能去散步 20 分钟。”

事实证明，保持你的核心习惯有很多好处，即使是在困难时期。正如杰基和约翰·科尔曼在《哈佛商业评论》中解释的那样:

> 对我们来说，过去的一年是彻底确定优先事项的一年。我们不断优化——确定我们最重要的优先事项和活动，同时不情愿地、痛苦地削减重要但不紧急的事情。
> 
> [...]
> 
> 与你的长期目标一致的每日或每周习惯可以让你保持在正轨上，即使很难提前思考，它们可以在不稳定的时期增加稳定性。我们每个人都有规律的做法，我们试图保持我们的生活结构，保持身心健康，并确保我们有意识地接近生活。这些习惯，在任何时候都很重要，在我们最忙最乱的时期必不可少。

即使在不困难的时候，确定你的核心习惯(相对于你更有抱负的习惯)也是至关重要的。我最近开始使用应用程序 [Exist](https://exist.io) ，它使用数据来绘制你的各种活动之间的相关性——例如，当我摄入较少的蛋白质时，我的静息心率会上升。

Exist 还可以让你跟踪习惯和其他自定义标签，我最终染上了“重新开始综合症”，并创建了一大堆我想要跟踪的新习惯和标签。

好吧。

事实证明，添加所有这些新的令人兴奋的习惯 [让我更难坚持我的核心习惯](https://www.nicoledieker.com/2019/05/16/i-love-the-exist-app-and-i-super-love-what-it-taught-me-about-priorities/) ，仅仅是因为在完成“每天同一时间上床睡觉”的习惯之前，你只能勾掉这么多习惯。

所以我停下来，筛选，重新排序。

这并不是说你不能添加新习惯，或者你必须消除一个旧习惯，然后才能重拾另一个。像 James Clear 这样的习惯专家建议:一次培养一个习惯[把它们堆在一起](https://jamesclear.com/habit-stacking) 来提高完成率和效率:

> 你早上的日常习惯可能是这样的:
> 
> 1.  在我倒完早晨的咖啡后，我会沉思六十秒。
> 2.  在我冥想 60 秒后，我会写下一天的待办事项。
> 3.  写完一天的待办事项清单后，我会立即开始我的第一项任务。

但是在你做这些之前，有必要问问你自己，无论发生什么，哪些核心习惯是最重要的。

这样，你就可以在他们周围建立你的余生——当你的余生遇到阻碍时，他们仍然会在那里。
