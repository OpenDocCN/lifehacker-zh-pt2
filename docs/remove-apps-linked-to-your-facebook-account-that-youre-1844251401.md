# 删除链接到你的脸书帐户但你不使用的应用

> 原文：<https://lifehacker.com/remove-apps-linked-to-your-facebook-account-that-youre-1844251401>

如果你曾经使用你的脸书帐户登录过第三方应用或网站，你应该花点时间更新一些安全设置。



脸书最近发现，已经被移除的应用程序——无论是在 90 天不活动后自动移除还是由用户自己手动移除——仍然可以访问一些脸书数据。 [脸书关于这个错误的博客文章](https://about.fb.com/news/2020/07/improving-data-limits-simplifying-terms/) 用了一个具体的例子，某人使用一个健身应用程序“邀请他们家乡的朋友去锻炼，但我们没有意识到他们的一些朋友已经好几个月不活动了。”这个应用可以在没有得到用户授权的情况下收集更多的数据，仅仅是因为它以前被使用过。

虽然看起来没有任何恶意，但像这样的错误仍然可能导致安全问题。幸运的是，脸书立即修复了这个问题，并没有发现泄露或数据处理不当的证据，但为了安全起见，现在是时候删除任何与你的脸书帐户相关联的你不再使用的应用程序了。

### 如何取消第三方应用与你的脸书帐户的链接

1.  在 iOS 或 Android 上打开脸书应用程序，或者如果你在桌面上，在浏览器中前往 Facebook.com。
2.  打开设置菜单，然后向下滚动并点击/轻触**[**应用和网站**](https://www.facebook.com/settings?tab=applications&ref=settings) **。”**这将打开一个新菜单，其中列出了您使用脸书登录或以某种方式授权访问您的脸书帐户的所有第三方应用和网站。**
3.  **移除你不再希望链接到你的脸书帐户的任何应用(注意:在从你的脸书帐户取消链接之前，请确保备份和/或删除这些应用/网站中你希望保存的任何数据。)**
4.  **如果你没有看到任何列出的应用程序，这意味着你要么从未使用脸书登录过第三方应用程序，要么脸书已经为你从你的帐户中删除了这些应用程序(如果应用程序在 90 天内没有被访问，脸书会自动从用户帐户中删除这些应用程序)。已移除/已取消链接的应用程序可能仍保留以前收集的数据，但在移除后将无法跟踪您。**

****“应用和网站”**菜单还允许您编辑您的帐户通过脸书与第三方应用的默认交互。**

**点击**应用、网站和游戏**或**游戏和应用通知**面板下的**“编辑”**按钮，开启/关闭功能。关闭应用、网站和游戏也将从你的帐户中删除所有外部应用。这样做还可以防止你使用嵌入式脸书快捷方式分享/喜欢来自其他网站或平台的内容，但这是阻止其他网站(就此而言，还有脸书)跟踪你的帐户活动的最佳方式。**
