# 是什么导致了亚马逊雨林大火？

> 原文：<https://lifehacker.com/what-caused-the-amazon-rain-forest-fires-1837511635>

周一下午 3 点，巴西最大的城市[S](https://www.buzzfeednews.com/article/skbaer/sao-paulo-darkness-amazon-fires)[o Paolo](https://www.buzzfeednews.com/article/skbaer/sao-paulo-darkness-amazon-fires)【陷入黑暗，因为正在进行的大火正在摧毁数千英里外的亚马逊雨林。今年，巴西目睹了创纪录数量的野火。该国总统雅伊尔·博索纳罗表示，该国政府缺乏扑灭这些火灾的资源，这引起了人们对雨林破坏和土著人土地毁坏的更多关注。

但是到底是什么引起了这些火灾呢？这很简单，也是可以预见的，是环保主义者和一位政治家之间争论的根源。

### 是什么引起了这些火灾？

人为砍伐森林。正如美国有线电视新闻网报道的那样，环境保护主义团体和研究人员都认为，当地的养牛场主、农民和伐木工人故意点火为牛清理土地(巴西亲商/怀疑气候的总统鼓励他们这样做)。

在给纽约时报 的一份声明中，一个政府机构表示，自 2019 年初以来，超过 1330 平方英里的森林覆盖已经被清除(与去年同期相比增加了 39%)。一般来说，这些火灾与巴西对牛肉生产的依赖有关；这个国家提供了近 20%的世界牛肉出口 。

与此同时， [Bolsanaro](https://www.theguardian.com/world/video/2019/aug/21/jair-bolsonaro-claims-ngos-may-be-setting-amazon-rainforest-on-fire-video) 指责非营利组织，认为他们在今年早些时候削减了他们的资金后，点燃了“ [让](http://brazil presidential election jair bolsonaro) ”政府难堪(他尚未提供证据 来支持这些说法)。

至于气候变化在火灾中的作用的争论，它肯定没有帮助，但这不太可能有任何联系(尽管它可能使火灾变得更糟，总体上造成 [更热和更干燥](https://www.c2es.org/content/wildfires-and-climate-change/) 的条件)。

### 现在有多少起火灾？

截至上周，估计有 9000 起野火。正如[BBC](https://www.bbc.com/news/world-latin-america-49433767)报道的那样，巴西在 2019 年已经出现了创纪录的森林火灾数量——迄今为止已经记录了超过 75，000 起(与去年同期的 40，000 起相比，仍然很多)。7 月和 8 月也标志着亚马逊干旱季节的开始，这使得它比多雨的月份更容易发生火灾。

### 我为什么要在乎？

亚马逊雨林 [被认为是](https://www.nytimes.com/2019/08/23/world/americas/amazon-fire-brazil-bolsonaro.html) 我们星球的“肺”它吸收 [百万吨的碳排放](https://www.bbc.com/news/world-latin-america-49433767) 。科学家们估计它产生的氧气约占世界氧气总量的 6%到 9%。更糟糕的是，它向周围地区排放烟雾和碳，危及植物、动物和居住在盆地的土著居民。

### 我能帮什么忙？

虽然你真的不能做任何事情来阻止目前的火灾，但你可以做一些事情来帮助预防火灾和保护雨林:

*   首先，你可以向雨林行动网络的 [保护一英亩计划](https://www.thecut.com/2019/08/how-to-help-the-amazon-rainforest-fires-in-5-ways.html) 捐款。这些资金将捐给致力于保护雨林地区和土著人民的组织。
*   你也可以向亚马逊手表 捐款，这是一个非营利组织，也保护雨林和土著居民。
*   或者，由 [雨林信托](https://www.rainforesttrust.org/our-work/current-projects/) 资助一个项目，其中包括拯救亚马逊的 [土著土地](https://www.rainforesttrust.org/projects/saving-indigenous-lands-in-the-amazon/) 。
