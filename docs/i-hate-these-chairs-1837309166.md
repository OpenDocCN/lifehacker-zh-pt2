# 我讨厌这些椅子

> 原文：<https://jezebel.com/i-hate-these-chairs-1837309166>

我不知道从什么时候起，这种特殊的椅子成了美国所有休闲餐厅的首选椅子。我不知道他们叫什么，也许是“小酒馆”的椅子？我只知道我恨他们，我害怕看到他们。

Watch

我不知道为什么这些被诅咒的座位突然成了餐饮业的首选椅子。根据你遇到这些椅子的地方的类型来判断——你知道，在大都市地区的高价“高架”烧烤店之类的地方——显然有一个共识，它们唤起了一种时髦的休闲。

但它们与放松或休闲截然相反。每当我看到这些椅子中的一把，我都会懊恼地轻轻叹口气。提醒你，还不足以转身离开。刚刚够发泄一些轻微的烦恼，接下来的一个小时我要么坐在我的椅子前面，要么用两块金属挖进我屁股外面柔软多肉的地方。太惨了！

当然，在某些方面，这把椅子比其他椅子更好。它坚如磐石，直到你有时在可爱的地方遇到的那些摇摇晃晃的破旧别致的椅子，总是让我觉得这将是我最终被甩在地板上的时候。它们的侧面也没有扶手，这意味着在某些方面，它们更容易接近——没有人会被这些东西卡住。然而，在试图放松到那些愚蠢的金属条的无情怀抱中时，很难想到这些椅子的可取之处。

但是它们不仅仅是我们这些有大屁股的人的问题，当我向 G/O 媒体摄影总监 Elena Scotti 要一张库存照片时，我发现了这一点。“去他妈的那些椅子，”她回答说，并指出它们是糟糕的户外椅子，因为它们会变热，然后会受伤。当你第一次坐下时，或者如果餐厅的空调开得太大，它们也会很冷。金属要么感觉太过临床化，要么很土气；《Jezebel》的编辑朱丽安·埃斯科韦多·谢泼德说:“坐在那些椅子上之后，我觉得我需要打一针破伤风疫苗。”

另一名 Jezebel 员工抱怨说，他们总是太低；另一个人问道，“有人曾经舒服地坐在一把会扎进你后背的硬金属椅子上吗？”不要！没人！不是一个人！这些椅子糟透了，没人喜欢！让他们离开这里！
