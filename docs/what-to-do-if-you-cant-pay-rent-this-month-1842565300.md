# 如果你这个月付不出房租怎么办

> 原文：<https://lifehacker.com/what-to-do-if-you-cant-pay-rent-this-month-1842565300>

虽然看起来没完没了，但 2020 年 3 月已经快过去了。这个月的第一天又来了，自从你上次付房租以来，发生了很多变化。我们在练习物理距离，政府通过了一项大法案来刺激经济，冠状病毒迫使 [失业率](https://www.washingtonpost.com/business/2020/03/26/unemployment-claims-coronavirus-3-million/) 创历史新高。



如果你因为这个月不能按时付房租而感到压力，我们理解。虽然许多州和城市已经暂停驱逐，但这并不意味着如果你停止支付房租，驱逐过程是不可避免的。

如果你现在需要一只手来维持生计——并且想尽可能地讨好你的房东(毕竟，他们可能也有抵押贷款要还)——这里有一些尽快交房租的策略。

### 转移一些钱

在你等待政府刺激计划支票或失业救济金的时候，一些预算的重新分配可能会帮助你度过难关。

CFP 的高级财务教练凯特·豪威尔顿在的 SmartPath 中指出，在的六个月里，联邦学生贷款的支付可以腾出一些现金用于基本的生活开支。你也可以兑现信用卡积分来支付食品杂货或汽油等物品，然后将这些现金用于支付房租。另一种选择？豪厄尔顿说，暂时减少你的退休金，直到你的收入稳定下来。

别忘了给你的公用事业提供商 打电话:你也许可以几个月不交电费、煤气费或水费，而不会有服务被关闭的风险，尽管你可能需要在疫情之后偿还这些余额。

### 寻求当地援助

如果你不确定在你所在的地区哪里可以找到租房援助，你可以从查看当地报纸的网站或者打电话给当地房管局寻求指导开始。你也可以咨询一下救世军社会服务机构、美国红十字会(T2)或天主教慈善机构。

“有可用的社区资源，但这取决于你住在哪里，作为租户[现在]你有什么权利或援助，，律师，[莱斯利·泰恩](https://attorney-newyork.com/)，说。“例如，佛罗里达州的奥兰治县为因疫情而休假的居民提供一次性租金支付。在纽约市，审计长斯科特·斯金格要求相关居民联系他的办公室，以便与当地提供帮助的机构取得联系。

一旦你研究了如何调整自己的财务状况，以及在社区中哪里可以得到帮助，是时候和你的房东谈谈了。

### 向你的房东求助

如果支付你的全部租金甚至部分租金是不可能的，问问你的房东或物业经理他们能提供什么选择。

“我不会建议马上提供一些东西，”律师莱斯利·泰恩提醒道。“看楼主愿不愿意，如果需要更多，那就开口要。”

房地产投资者、的创始人玛丽娜·瓦蒙德(Marina Vaamonde)说，没有一个放之四海而皆准的解决方案，即使是对同一房东的不同房产的居民也是如此。“每个租户都面临着一系列需要解决的不同情况，”她说。

这有助于通知你的房东你已经采取的步骤或计划采取的步骤，以赶上并保持当前的租金水平。如果一旦你的失业支票开始到达，你将能够支付你的余额，请注意这一点。如果你已经通过当地的组织申请了帮助，那就提及此事。如果你可以支付这个月的租金，但担心下个月的租金到期前你的工作时间会被削减，那就早点告诉你的房东这个情况。

Vaamonde 说，她的团队已经能够向一些租户推荐政府和非营利租金援助项目。“知道我们的一些租户被解雇，我们已经决定根据具体情况，我们将推迟支付部分或全部款项，直到他们能够从现有项目中获得援助，”她解释说。

如果你的房东使用在线支付系统，他们可能会调整你的租金。科林·克鲁克 ，他住在加州，在俄勒冈州的波特兰租了一套房子，用 Cozy 来管理他的房客。他说，他可以关闭滞纳金或推迟它们，给租户一个未来租金的信贷，并启动一个付款计划，允许租户按周分期付款。

即使你的房东使用更老派的方法，他们也可能愿意将你的租金分成几期，免除滞纳金或者破例接受信用卡付款。只要确保得到他们书面提供给你的任何住宿。
