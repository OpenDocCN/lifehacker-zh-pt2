## 如何文件编码器。c 勒索软件工程

该勒索软件被称为 Android/Filecoder。c，最早是通过 HTML 链接和二维码出现在 Reddit 和论坛线程上的。它通常伪装成成人内容或可下载的“性模拟”虚拟现实应用程序。事实上被感染的人。apk 将勒索软件转储到您的设备上，*然后*试图通过短信将自己传播到受害用户的存储联系人。

发完短信后。c 加密并锁定用户手机上的几乎所有文件，使其无法使用。然后，它要求比特币赎金，以重新获得控制权。根据 Welivesecurity 对该应用程序代码的分析，确切的赎金可能在 90 美元至 190 美元之间，甚至更高，这取决于比特币的当前价值。

虽然加密和赎金似乎是真实的，但该应用程序还声称将在 72 小时后删除被勒索的数据。Welivesecurity 无法确认这是否属实。然而，它确实证实了文件编码器。c 使用了一种难以破解的加密方法——更糟糕的是，删除勒索软件应用并不能撤销赎金。关于 的完整解释，可以阅读 [Welivesecurity 的报告。](https://www.welivesecurity.com/2019/07/29/android-ransomware-back/)

## 避免勒索软件和其他恶意软件的提示

再说一遍，文件编码器。c 似乎还没有传播得很远，但它仍在活跃地传播。以下是一些让你免受这种和其他令人讨厌的 Android 恶意软件攻击的建议:

### 对来自联系人的随机短信和链接保持怀疑

*   文件编码器发来的假短信。c 声称你的暴露照片会出现在其他应用上，但是除非你对分享这样的内容满不在乎，否则这是不太可能的。
*   这些短信一开始看起来和听起来都很奇怪，所以你应该能够判断出你的朋友实际上没有发送它。如果短信来自你不常联系的人，那就更有理由避免点击短信包含的任何链接。
*   如有疑问，不要下载，点击模糊链接或安装。因为你的朋友突然发短信给你。打电话给你的朋友，问问发生了什么事——你甚至可以提醒他们自己感染了恶意软件，以此来帮他们一个忙。

### 在扫描或点击之前，解码二维码并检查 HTML 链接源

*   免费的 [二维码桌面阅读器&生成器](https://www.codetwo.com/freeware/qr-code-desktop-reader/) 让你在扫描之前先看看什么是二维码。这个工具本身并不足以保证您的安全，但是它至少可以帮助您在打开代码之前看到代码将要做什么。
*   对于 HTML 链接——尤其是像 bit.ly 链接这样的短链接——使用像 [ScanURL](https://scanurl.net/) 这样的服务来诊断它们。
*   一眼就能发现可疑/虚假的全长 HTML 链接的一个简单方法是在 URL 中使用过多的“%”符号。

遵循上面的步骤是有帮助的，但是我们也建议完全避免 HTML 链接和 QR 码，如果它们感觉奇怪或者你不能确定它们是什么。

### 不要下载或安装随机的 APK 文件

*   坚持使用经过验证的应用程序商店或可信的网站，如[【APK 镜像】](https://www.apkmirror.com/) 。
*   恶意软件和勒索软件尤其臭名昭著的是 [伪装成人内容](https://lifehacker.com/be-careful-about-sideloading-popular-android-apps-unle-1836669206) 。对这些链接要格外小心，不要安装听起来奇怪的淘气应用程序——尤其是当你把它们下载到你的设备上的时候。
*   使用 [反病毒 app](https://lifehacker.com/double-check-that-your-android-antivirus-app-actually-w-1833414047) 防止安装恶意 app。
