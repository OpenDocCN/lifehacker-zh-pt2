# 不能去健身房怎么健身

> 原文：<https://lifehacker.com/how-to-work-out-when-you-cant-go-to-the-gym-1842299641>

无论你是在 [自我隔离](https://lifehacker.com/what-does-self-quarantine-actually-mean-1842268976) ，你的健身房已经取消了活动作为社交距离的措施，或者你只是想小心谨慎，除了去你通常的健身房或健身班之外，有其他锻炼的选择也是不错的。这里有一些让你自己保持活跃的方法。

### 到户外去

社交距离的概念，简单来说，就是和其他人保持六英尺的距离，不去碰其他人碰过的东西。虽然这通常被总结为“呆在家里”，但只要你能遵守这些规则，离开你的房子 实际上是没问题的。

这意味着你仍然可以独自去跑步或骑自行车，或者到大自然中去远足或进行其他户外活动。(如果出于安全考虑，你要带一个伴侣，要注意你们两个可能会分享细菌。也就是说，如果你们两个人都感觉很好，或者你们已经住在一起，那么任何一次外出都没有太大的风险。)

### 用应用程序或视频替换群组课程

怀念那种集体健身的氛围吗？幸运的是，没有实际的组，你仍然可以得到它。 [我们测试了几款应用](https://lifehacker.com/get-a-group-fitness-experience-on-your-own-with-these-a-1841700813) ，它们可以在你的耳边、手机或平板电脑屏幕上放置一名教练。在健身房的健身车上试一试(前后彻底擦拭)或者去外面跑一跑。

你也可以在网上找到训练。如果你独自一人，下面是一些最好的视频 。当你完成了这些，别忘了我们的经典指南 [最佳 YouTube 锻炼频道](https://vitals.lifehacker.com/the-best-youtube-channels-you-can-work-out-to-right-now-1751069875) 。

### 用最少的设备进行力量训练

健身房里所有的重量都很方便，但它们不是训练肌肉的唯一方式。请记住，您在家的选择包括:

*   [体重练习](https://lifehacker.com/this-table-of-exercises-shows-you-how-to-get-fit-withou-1525971528)
*   [悬浮训练师](https://vitals.lifehacker.com/amp-up-your-bodyweight-workout-with-these-suspension-ex-1751397400) (像 TRX)
*   [阻力带](https://vitals.lifehacker.com/use-resistance-bands-for-travel-friendly-workouts-1758183618)
*   好老的 [哑铃](https://vitals.lifehacker.com/work-every-part-of-your-body-with-this-dumbbell-exercis-1764033240) 和 [壶铃](https://vitals.lifehacker.com/get-a-full-body-kettlebell-workout-with-these-exercises-1774731446)

所有这些都是很好的权宜之计，直到你回到你的日常生活中。也就是说，如果你在地下室或备用房间里有一些空间，这可能是开始组装你梦想中的家庭健身房的理想时间。T3】
