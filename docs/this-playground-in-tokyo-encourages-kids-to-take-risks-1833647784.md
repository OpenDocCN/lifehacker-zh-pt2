# 东京的这个游乐场鼓励孩子们去冒险

> 原文：<https://lifehacker.com/this-playground-in-tokyo-encourages-kids-to-take-risks-1833647784>

我喜欢把自己想象成一个反直升机的家长，但是当我第一次带我一岁半的女儿去附近的游乐场时，听起来就像是在看一部恐怖片。"*喘息！*不要踩在那里！别靠在那上面！哦，天啊！*喘息！*哦，天哪！”她才蹒跚地走上三级台阶，我就准备坐下了。



在我在东京参观的世田谷游乐园，没有这种徘徊。在这个 [冒险游乐场](https://offspring.lifehacker.com/take-your-kid-to-an-adventure-playground-1822980170) ，这里的座右铭是“自由玩耍，风险自担”，妈妈们在野餐毯子上聊天，而他们的孩子——我猜是 3 岁到 5 岁——爬上爬下用废弃的木头搭建的建筑，用*真*锯锯树枝，在熊熊的篝火上帮忙做午饭。虽然许多父母可能会看到这一幕，看到分裂和诉讼，但这里的父母似乎对他们孩子的能力充满信心。“这就是童年应该有的样子，”我想。

根据日本 [*大都会杂志*](https://metropolisjapan.com/adventure-playgrounds/) 的说法，冒险游乐场于 20 世纪 70 年代来到日本，作为“倡导城市儿童在过度安排和受监管限制的生活中寻求更大风险和自由玩耍的一种方式”。世田谷游乐园是由游乐园志愿者在 1995 年神户地震后创建的。对日本来说，那是一段黯淡的时光——一些成年人反对公园，T7 批评孩子们在灾难发生后这么快就在操场上“玩地震”。但是专家们开始意识到假装游戏对孩子有好处，尤其是在悲剧发生后。儿科医生开始提倡玩耍来帮助孩子释放压力，预防创伤后应激障碍。今天，全国各地都有几个冒险游乐场——社区成员被邀请向公园捐赠工具和废料，这样孩子们就可以建造堡垒、海盗船或他们能想象的任何东西。

从柏克莱到纽约州再到内布拉斯加州，美国也有越来越多的冒险游乐场。如果可以的话，检查一个。在你离开之前，教你的孩子 [危险和风险](https://offspring.lifehacker.com/teach-your-kids-the-difference-between-danger-and-risk-1833069026) 的区别，温习一些 [短语来代替“小心”](https://offspring.lifehacker.com/what-to-say-to-kids-instead-of-be-careful-1816590594) 并确保带一套换洗衣服——你的孩子*会*弄脏。
