# 让我们做一些沙袋搬运

老实说，我不做核心练习。没有仰卧起坐，没有木板；我确信我的腹肌可以应付它们，但我的大脑就是不感兴趣。(如你所知，我是 [的忠实信徒，不做任何你真正讨厌](https://vitals.lifehacker.com/exercise-shouldnt-feel-horrible-1844763622) 的运动。)而是让你的核心努力工作的事情*同时*你在做别的事情？当我做这些的时候，我感觉很糟糕。

Watch

所以今天，有了我们的沙袋，那东西就是沙袋扛。当你搬运重物时，你的整个身体都在工作。取决于你如何保持它，你会让你的腹肌和背部参与进来，所有那些模糊的其他“核心”肌肉围绕在你的躯干两侧和你的胸腔内侧，我们被告知这些肌肉应该存在。你的肩膀和臀部在做了大量的搬运后可能会感到奇怪的疼痛。他们很烂，但是好的一面。

所以，我们开始吧。我选择了四种不同的方式来搬运我 50 磅重的沙袋，你可以在这个视频中看到我是怎么做的:

中的进位是:

*   肩扛(两边都做)

*   我只能说:祝你好运

对于肩扛和背扛，你需要把沙袋放在膝盖上，然后摆好姿势，就像我们在挑战的第一周讨论的。开销更棘手。你把它举到肩膀的高度，但是你必须设法把它举到你手的*顶部*。我发现把包靠在我的胸前，小心地把手放在它下面，然后慢慢地把它压在头顶上是有效的。任何时候我想跑得太快，最后都会掉下来。

开销是最好的，如果你能让它工作。你在锻炼你的肩膀和整个核心，并时刻意识到你的平衡。

另一个选择，没有显示，是用一只手抓住沙袋(如果你的包有把手的话，抓住把手)，像提箱子一样提着它。你会觉得很不平衡，这才是重点。只要确保两边的距离相同。
