# 如果你在飞机上生病了该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-youre-sick-on-a-plane-1836421256>

不管你是晕机还是吃了不该吃的东西，在飞机上生病是最糟糕的。以下是如何度过这场磨难的方法。



### 要一个呕吐袋

航空公司不再总是在每个座位上都放一个呕吐袋，所以不要指望一接到通知就会有呕吐袋。当你开始感到恶心时，集中你的思想，开始制定计划。按下呼叫按钮；空服员宁愿现在给你找几个呕吐袋和纸巾，也不愿过一会儿给你收拾残局。一般来说，最好让船员知道发生了什么，而不是试图隐瞒你生病的事实。

### 打开通风口

脸上的凉风会有帮助，所以打开座位上方的通风口，并考虑脱掉外套或毛衣。(还有第二个原因:如果你觉得你真的可能会呕吐，这减少了你需要担心溅出的物品数量。)

想想还有什么能让你感到舒服:也许是一小口姜汁汽水；闭上眼睛听播客，而不是试图阅读。(如果你晕机，阅读往往会使情况变得更糟。)不要喝酒，不要吃东西，除非一些椒盐卷饼或饼干能让你的胃舒服。

### 切断通往浴室的线路

是的，在等着的人面前跑去洗手间是不礼貌的。但是如果你排队等着却没赶上，对每个人来说都更糟。是的，我们可以在这里谈论呕吐或腹泻。

如果你要在浴室呆一会儿，一定要让别人知道，这样他们就可以检查你了。如果你没有和朋友或家人一起旅行，给空姐提个醒。另外，要注意当你在里面的时候他们不能着陆，所以你最终还是要出来。

### 如果需要，寻求医疗帮助

空乘人员真的会问飞机上是否有医生(或其他合格的医疗专业人员)，以防出现医疗紧急情况。

每架飞机都携带一个急救包和基本用品， [一个医疗箱，供医疗专业人员](https://www.asma.org/asma/media/AsMA/Travel-Publications/Medical%20Guidelines/In-Flight-Medical-Care-July-2016.pdf) 使用。该工具包包含处方药和专门的医疗设备。如果你病得很重，或者如果你认为飞机着陆时你的情况可能会特别糟糕，那么寻求帮助是值得的。例如，医生可能会给你抗恶心或抗腹泻的药物，或者在极端情况下，甚至会给你静脉注射液体以防止你脱水。

如果事情变得很严重——比如你得了危及生命的疾病，而不仅仅是晕机或食物中毒——在极少数情况下，飞机可能会改道，以便你可以更快地去医院。无论飞机上是否有医生，机组人员通常都能从地面上的专业人员那里获得医疗指导，以帮助确定你的病情，以及当你在空中时他们能为你做些什么。

根据当地的法律法规，某些国家的航空公司工作人员可能会向您提供非处方药。问问无妨，但如果他们说帮不了你，也不要惊讶。下次，带上你自己的晕船药。
