# 使用应用程序更好，还是用传统的方式更好？

> 原文:[https://life hacker . com/is-it-better-to-use-a-app-or-do-it-the-old-fashion-1835382063](https://lifehacker.com/is-it-better-to-use-an-app-or-do-it-the-old-fashioned-1835382063)

你有没有拿出手机写下一个清单，然后想“这种方式真的比以前更好吗？”

嗯，我们有同样的想法，让 Lifehacker 的工作人员 [尼克·道格拉斯](https://kinja.com/nickdouglas) 和 [阿布·扎法尔](https://kinja.com/abuzafar93) 面对面测试简单的任务，模拟对数字。

从使用卷尺测量 iPhone 的测量应用程序开始，尼克和阿布测试了七项任务，包括存入支票、看到很远的距离和使用指南针。

判决结果？没那么简单。在某些方面，应用程序让我们的生活变得更加轻松，但仍有许多杂务不一定是科技的天下。在某些功能上，比如分摊餐馆账单，这款应用轻松胜出。但在测量或写笔记等基于身体的任务中，数字并不总是更好。
