# 如何假装自己不嗨

> 原文:[https://life hacker . com/how-to-decough-you-not-high-1834122277](https://lifehacker.com/how-to-pretend-youre-not-high-1834122277)

我们大多数人都会遇到这种情况:你兴奋起来，以为在正常生活打扰你之前，你有足够的时间冷静下来。然后你记得你不能取消的牙医预约。或者你妈妈打电话提醒你今天是你叔叔的葬礼。临时工中介发短信给你说你刚得到一份收入丰厚的工作，一小时后开始。好莱坞来电，你将成为一个明星，但首先你必须找到你的裤子，并进入豪华轿车，就在外面空转。你会怎么做？！

你要假装清醒**，这就是你要做的，我们来告诉你怎么做。拿出你的望远镜，给自己倒一杯冰咖啡，准备好测试这些表演技巧。一切都会好的。

嗯，也许不是我们当中最好的 T2。

注:假装清醒并不等于清醒——所以你不能开车，特克斯。
