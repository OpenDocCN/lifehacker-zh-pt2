# 如何写一封完美的电子邮件

> 原文：<https://lifehacker.com/how-to-write-the-ultimate-canned-email-1832879435>

如果你收到了很多至少应该粗略回复的邮件，你可能会觉得没有一封格式信邮件可以取代所有的邮件。这里有一个聪明的解决办法，借用科幻作家罗伯特·海因莱茵的话。



艺术家和艺术家的鼓吹者 Austin Kleon 收集了来自名人和组织 的 [股票拒绝。有的短小精悍，让人缴械；E. B .怀特曾“出于秘密原因”拒绝成为联邦委员会成员。文学杂志 *Zyzzyva* 有一封格式信，对被拒绝的投稿非常友好和诚实。包括海因莱茵在内的一些人使用了一个由几十个答案组成的长长的清单:](http://tumblr.austinkleon.com/post/120472862666)

这种解决方案在印刷品上看起来有点不友好，但对于电子邮件来说，这是一个绝妙的主意。像上面那样做一个巨型的罐装回复列表，放在 [Gmail 的罐装回复](https://lifehacker.com/use-gmails-canned-responses-for-smarter-autoresponders-5165371) ，或者一个方便复制粘贴的空间。然后粘贴到你的回复里，*删除所有不适用*的部分。嘣，你就有了一封个性化的邮件。
