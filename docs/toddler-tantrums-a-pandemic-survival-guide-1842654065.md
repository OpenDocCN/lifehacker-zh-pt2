# 幼儿发脾气:疫情生存指南

> 原文：<https://lifehacker.com/toddler-tantrums-a-pandemic-survival-guide-1842654065>

养育年幼的孩子已经够难了。在这个疫情时期，养育年幼的孩子是一个全新层次的困难。在工作、照顾孩子和疾病快速传播和经济快速崩溃带来的担忧之间，父母们已经到了崩溃的边缘。没有人会责怪他们为了自己的孩子 努力保持一点理智。



从我们听到的一些家长来看，孩子们也感受到了这种不断加剧的混乱的影响，导致他们的行为面临额外的挑战。从越来越多的发脾气，到兄弟姐妹间的争斗，再到睡眠时间被打乱，小孩子们开始意识到这个世界现在并不安全。

为了了解这种疫情可能会如何影响幼儿，我们求助于儿童发展专家 [莎拉·凯特·贝尔曼](https://education.utexas.edu/faculty/sarah_bearman) ，德克萨斯大学奥斯汀分校教育心理学助理教授，她的研究重点是有破坏性行为、创伤、焦虑和抑郁的儿童。

除了她专业的儿童行为规范，Bearman 自己也有一个四岁的孩子，这意味着她现在和我们一样深陷困境。鉴于她的时间有限，我们通过电子邮件与她交谈，所以她可以在仅有的空闲时间里回答我们的问题。

### 压力可能是一个恶性循环

Bearman 说:“当家庭环境高度温暖和一致，父母能够适应孩子的需求时，孩子表现得最好。”

当父母压力很大时，他们很难给孩子提供一个稳定的环境。这导致了一个循环，在这个循环中，孩子比平时表现得更好，父母很难对他们的反应保持耐心，反过来，每个人都变得越来越紧张和不开心。

如果这听起来很熟悉，你并不孤单。要处理的事情很多，而且当你身处其中时，你会觉得无法打破这种模式。

### 先戴上自己的氧气面罩

“父母需要善待自己，”Bearman 说。"即使在最好的情况下，为人父母也是一项艰难的工作。"

尽管这很难做到，但 Bearman 建议父母们找到一些自我照顾的方式，无论是去散步，洗个热水澡，甚至只是抓住一些独处的时间。正如她所指出的，父母需要先照顾好自己，然后才能照顾别人；想想空姐(曾经，叹气)建议你先戴上自己的氧气面罩，再帮孩子戴上他们的。

“我们都在尽最大努力，”Bearman 说。

### 保持常规，增加一些娱乐

为了打破这种父母负担过重和孩子脾气暴躁的循环，Bearman 建议父母每天留出一点时间专注于孩子的游戏活动，像体育播音员和运动员一样描述他们正在做的事情，同时找到表扬他们的方法。

“这是一种与孩子互动的有趣方式，我不建议父母长时间这样做，但即使每天五分钟这种高质量的关注也可以帮助孩子的发展，减少以后出现问题的可能性，”Bearman 说。“这有点像维生素——它能促进亲子关系，有助于大脑健康发育。”

除了这项活动，Bearman 建议尽可能坚持日常活动。保持规律的就寝时间、午睡时间和进餐时间有助于缓解孩子的焦虑，给一天注入一点平静。

说真的，在这个可怕的世界里，我们都需要一点冷静。
