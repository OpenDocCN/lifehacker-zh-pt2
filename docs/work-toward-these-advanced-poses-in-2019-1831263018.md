# 在 2019 年朝着这些高级姿势努力

> 原文：<https://lifehacker.com/work-toward-these-advanced-poses-in-2019-1831263018>

[瑜伽挑战赛](https://vitals.lifehacker.com/decembers-fitness-challenge-is-yoga-1830827794) 进行的怎么样了？你是否正在掌握家庭练习的窍门，或者去健身房或瑜伽馆参加课程？我们在一个月的时间里只能做这么多，你真的可以花几年的时间来达到一些更有雄心的体式。所以在一年中的这个时候，放眼未来是很棒的。

Watch

我想谈谈目标姿势，你想做的事情，但你还没有完全做到。也许你需要更灵活，也许你需要更强壮，或者也许你只需要花更多的时间，学习如何让这个体式为你的身体服务。

我很想听听你有哪些进球姿势，或者你最近成功做到的那些。我喜欢手臂平衡，在 [乌鸦](https://www.yogaworks.com/blog/2013/07/24/breaking-down-the-pose-bakasana-crow-pose/) (巴卡萨纳)我可能不太擅长*，但我觉得自己就像一头野兽，任何时候我都能让它发生。*

 *[https://lifehacker.com/embed/inset/iframe?id=youtube-video-tmTrSGBvaoE&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-tmTrSGBvaoE&start=0)* 

*当我第一次尝试这个体式时，我的前臂平放在地上，这意味着你的小腿和鼻子只比垫子高一英寸。这是一个获得自信的好方法，可以尝试更传统的姿势。从那里开始，你可以换一个更好的姿势，或者尝试这些变化。有一次，我上了一堂课，在一个小时的时间里，我们逐渐形成了一种乌鸦的亲戚，叫做飞鸽式(你可以 [点击](https://www.tummee.com/yoga-poses/bakasana/variations) 链接查看图表)，这是一种令人惊叹的新目标式。*

*一些可能值得尝试的目标姿势:*

*   ***灵活性**:向前折叠，花环式，舞者式(见这些和更多T3T5】)* 
*   ***对于力量**:真正深入到椅子式或 chaturanga，或专注于保持任何艰难的姿势更长的时间(一些想法 [这里](https://www.yogajournal.com/poses/types/strength) )*
*   ***平衡**:除了像乌鸦一样的手臂平衡，像 [战士三](http://www.yogabasics.com/asana/warrior-iii/) 或者树式这样的站立平衡也很棒。*

*在 12 月的最后几天练习一个新的瑜伽姿势可能正是你所需要的，让你从假日的混乱中转移注意力，但一如既往，慢慢来，意识到你自己的能力。你的目标姿势是什么？*
