# 当孩子顶嘴时，正确的回应方式

> 原文：<https://lifehacker.com/the-right-way-to-respond-when-kids-talk-back-1838214468>

我的一个朋友曾经长篇大论地说她不会抚养一个无礼的孩子。“这是不可能的，”她说。“他不会顶嘴的。尊重对我来说太重要了，我不会容忍他的粗鲁。”她的儿子当时大约一岁。我的比她大几岁。于是，我笑了。



“是的，听起来很棒，”我告诉她。"如果你发现了这个秘密，就告诉我，好吗？"

她还没有经历的是一个非常现实的困境:1) 想要制定法律，因为我不会抚养一个这样跟我说话的孩子；2)想要建立一个更好的沟通方式，因为他们更可能像我们一样做，而不是像我们说的那样做。

孩子们，他们会顶嘴。从发展上来说，他们应该。父母蔻驰·梅根·莱希为 [《华盛顿邮报》](https://www.washingtonpost.com/lifestyle/on-parenting/when-a-9-year-old-back-talks/2019/09/17/1ba4d168-d568-11e9-9343-40db57cf6abd_story.html) 写道，当顶嘴越来越多时，实际上可能意味着孩子已经准备好更多的独立和控制。

> 似乎一夜之间，你的孩子就准备好独自去朋友家，整理自己的作业，选择自己所有的衣服，并对他们的饭菜有自己的看法。虽然这可能会感到有挑战性，但最好将孩子的意见和反对视为改变的邀请，而不仅仅是单纯的不当行为。将顶嘴视为一种需要更好理解的交流形式是有益的。

尽管如此，往好了说这很烦人，往坏了说这很让人恼火。幸运的是，我们可以做一些事情来减少顶嘴，缓和局势。

### 松开缰绳

一整天，孩子们都被呼来唤去。从他们早上的例行公事，到他们在学校的课程表，到把他们的鞋子放好，做作业，喂狗，他们总是被指导着。所以当我们告诉他们该上楼去洗澡时，他们会发出一个大大的“哈鲁姆普”!跺着脚走上楼梯，大喊，“我讨厌。服用。洗澡！淋浴。是。所以。蠢！”我们不应该完全惊讶。

就像我们试图让我们的孩子选择避免发脾气一样(亲爱的，今天你想穿橙色还是紫色的衬衫？)，我们可以找到方法让我们的大孩子做决定，并接管他们日常生活的一些控制权。你想今晚洗澡还是明天早上？你想在作业前还是作业后吃点心？我们压制和支配得越少，他们就越放松。

### 选择你的战斗

并不是所有的回话都是平等的，所以要划清界限。你能忍受什么，什么是不能接受的？瑞秋·西蒙斯是《好女孩的诅咒:用勇气和信心抚养正宗女孩 》一书的作者，她给《纽约时报》的提出了以下建议:

> 西蒙斯女士说，决定什么时候让事情顺其自然——随口说说、耸耸肩或翻白眼——以及什么时候回击。当一些事情违背家庭的道德价值观时，如侮辱父母的外貌或拒绝做家务，应该发生推回。

我当年是翻白眼女王，但我爸妈好像不怎么在意。事实上，他们觉得这很有趣，没有什么比你的父母喜欢它更能扼杀一个好的眼球转动了。不要理会那些小事，但是要准备好(冷静地)在更大的事情上坚持自己的立场。

### 不为所动

这是我最喜欢的建议，来自积极育儿解决方案 的创始人艾米·麦克里迪:“当你的孩子正在上演一场值得托尼奖的戏剧表演时，你的工作就是做一个无动于衷的观众。”

[麦克里迪](https://www.positiveparentingsolutions.com/parenting/five-steps-to-put-the-brakes-on-back-talk) 在这里说我的语言。这是我们可以真正为他们示范如何应对冲突的地方:通过冷静、坚定和善良。

> 简单地说，“你对我说话的方式让我感到受伤。当我听到那种语气时，我会走开。当你能对我毕恭毕敬地说话时，我们可以再谈。”然后走开。
> 
> 下次再发生这种事，甚至不需要警告——直接离开房间。你发出了拒绝参与权力斗争的信息。当没有人一起战斗时，就没有战斗了！

如果你一时纠结，试着引导你内心的。当他们走低，我们走高。当他们顶嘴时，我们保持冷静。
