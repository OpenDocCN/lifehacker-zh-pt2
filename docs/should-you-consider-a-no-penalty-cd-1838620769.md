# 你应该考虑一张无罚 CD 吗？

> 原文：<https://lifehacker.com/should-you-consider-a-no-penalty-cd-1838620769>

如果你只知道一件关于存单的事情，你大概知道这种投资是有固定期限的。你可以将钱存入一张 CD，期限为九个月、一年、两年或更长，但无论你选择哪一种，在期限结束之前，你都不能存取这笔钱——那是 CD 的到期日。如果你需要你的钱，你会支付费用。

定期存单通常是有吸引力的储蓄工具，因为它们提供略高的利率，以弥补将资金锁定一段时间的不便。

但是无罚存单允许你在期限内随时提取资金，不收取任何费用。你也可能会看到这些被列为流动、无风险或可存取的 CD。它们的利率与传统的定期存款利率没有太大差别。

### 无罚款 CD 的预期利率是多少

一年期定期存款利率目前在 2.3%的高端和 0.25%的极低端之间。这些回报并不令人激动:如果你以 2.25%的利率获得 500 美元的 12 个月定期存款，你将获得 11 美元的巨额收益。但是，关于财务可预测性，有一些很棒的东西值得一提，尤其是当你试图专注于储蓄的时候。

无罚款 CD 不像储蓄账户那样灵活——你通常必须一次取出所有的钱并关闭 CD，而不是在你需要时取出零碎的东西——而且它们可能很难找到。如果你的银行或信用合作社不提供免罚 CD，请选择网上银行选项。例如，高盛的 Ally 和 Marcus 都向他们提供 2%的利率。马库斯雷需要 500 美元才能开始，而艾丽没有最低限额。

如果你愿意为了方便而牺牲一点利益，那就值得考虑你不想看到和想到的短期储蓄。

### **谁应该考虑无处罚 CD**

#### **如果你有一笔健康的应急基金，并想继续存钱**

储存 3-6 个月的开支 以防紧急情况，这是一个很好的经验法则，但是你不需要把所有的现金都放在手边。除了要有足够的钱来帮助你度过汽车故障或意外的兽医之旅，你可能想把这笔钱存在一个不太容易存取的单独的储蓄账户中。随着美联储降息，高收益储蓄账户的利率一直在下降。但是一些无惩罚存款的利率至少和那些储蓄账户一样好。

#### **如果你在存钱买特别的东西**

夏威夷的大型家庭聚会两年内不会举行，但是你知道你现在应该开始存钱了。或者你知道你将在未来一两年内购买一辆新车，但你并不着急，因为它在你目前的汽车中一帆风顺。一旦你存了一大笔钱——许多 CD 都有 500 美元或 1000 美元的最低存款额——你可以把它放在 CD 里，看着它(缓慢但肯定地)增长一年。再存一笔可观的钱？您不能像使用储蓄帐户那样继续向 CD 添加内容，但您可以打开另一张 CD。

#### **如果你的高收益储蓄账户让你失望了**

如果你一直在观察高收益利率在 [后下跌，美联储调整利率](https://lifehacker.com/what-the-federal-reserve-interest-rate-cut-might-mean-f-1836790877) ，你可能会考虑跳槽到一个更好的报价。但是，一张无罚息存单可以给你一个有竞争力的利率，同时仍然保持你的现金可获得性。CD 将提供整个期限的固定利率，而不是应对可能使你的回报更难预测的浮动利率。
