# 你现在可以改变 Alexa 的语速

> 原文：<https://lifehacker.com/you-can-now-change-alexas-speaking-speed-1837075956>

现在，如果 Alexa 对你的反应太快，你可以让她慢下来。

周三，亚马逊为智能助理推出了一项新功能，允许你在她回应你时要求她放慢或加快速度。总的来说，有七种不同的速度可供选择；其中两个比 Alexa 的默认速度慢，四个更快。



要改变 Alexa 说话的速度，只需说“Alexa，说慢点”或“Alexa，说快点。”如果你最终不喜欢你所选择的，你也可以选择说“Alexa，以你的默认速率说话。”

在更改速度之前，您选择的速度将保持默认值。

在宣布功能 的博客文章，亚马逊表示，它从一些客户那里听说，如果 Alexa 可以放慢速度，以便他们可以更好地理解她的回答，那些重听和年长的客户会很感激。另一方面，一些盲人和弱视客户习惯于更快地处理信息，并希望 Alexa 说得快一点。 

当你从 Alexa 获取大量信息时，改变她的速度肯定会派上用场。较慢的速度可以让你有时间记笔记，而较快的速度可以让你快速浏览大量内容——比如说你的日程安排、比如说 — 。

现在，所有 Alexa 用户和设备都可以改变 Alexa 的语速。
