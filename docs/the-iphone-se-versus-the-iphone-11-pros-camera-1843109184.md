# 400 美元的 iPhone SE 的摄像头如何与苹果最贵的手机相抗衡

> 原文:[https://gizmodo . com/the-iphone-se-vs-the-iphone-11-pros-camera-1843109184](https://gizmodo.com/the-iphone-se-versus-the-iphone-11-pros-camera-1843109184)

新的第二代 iPhone SE 是你能买到的最便宜的 iPhone，这意味着你必须做出一些牺牲，特别是在相机方面。

但是，花 400 美元而不是 1000 美元买一部基础型号的 iPhone 11 Pro，你放弃了什么？嗯，你只有一个 1200 万像素的后置镜头，而不是三个，还有一个 700 万像素的自拍镜头，而不是 1200 万像素的深度感应相机。这意味着没有 Face ID，但有些人更喜欢用指纹传感器解锁手机。(现在我在公共场合都要戴口罩，我也是其中之一。)

据新手机的 [iFixIt 拆机](https://www.ifixit.com/Teardown/iPhone+SE+2020+Teardown/133066) 报道，SE 的相机镜头直接从 3 岁的 iPhone 8 上取下。但 SE 也有 [iPhone 11 Pro](https://gizmodo.com/try-to-hate-the-iphone-11-pro-1838252691) 的 A13 仿生处理器，它可以实现一些幕后的相机魔法。

当我将 iPhone SE 的照片与 1100 美元的 iPhone 11 Pro Max(这些锁定期内我手头上唯一的新 iPhone)进行比较时，我没想到会大吃一惊。但 SE 有自己的优势，特别是在人像模式下拍摄时。
