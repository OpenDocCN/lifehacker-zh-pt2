# 用这个故事解释一个丢失的填充动物玩具的新外观

> 原文：<https://lifehacker.com/explain-a-lost-stuffed-animals-new-appearance-with-this-1844496354>

每一个有挚爱“爱人”的孩子的父母都生活在恐惧中。心爱的东西——最喜欢的毯子、毛绒动物或其他毛绒玩具——倾向于跟随蹒跚学步的主人去任何地方。这意味着他们有办法在最不方便的地方迷路，比如你在公路旅行中停下来的加油站、露营地、海滩上的 Airbnb，或者干脆消失在空气中。

Watch

如果你已经有了一个复制品，每一两周你都用金惠秀#1 清洗和轮换，这样这对可爱的双胞胎穿得均匀，从一个蹒跚学步的孩子挑剔的眼睛看起来不那么明显，我为你鼓掌。但很难猜测是毛绒企鹅、软软的兔宝宝还是角落里缝着鸭子的软毯会抓住他们的心。你不能什么都买两件。

当你意识到你最好复制哪一个爱人时，第一个人可能已经失去了一只眼睛，在不同的地方用笔或记号笔刷了几下，总体上看起来有点乏味。这不是一个关于如何找到一个重复的 lovey 的帖子——那个帖子就在这里 。*这个*帖子是关于如何在肮脏的原版金惠秀失踪后介绍全新的金惠秀。答案当然是神奇肥皂。

这是 Reddit 用户 u/SteffyJeffy [对这篇文章](https://www.reddit.com/r/LifeProTips/comments/hwayht/lpt_if_you_give_your_child_an_stuffed_animal_and/) 的建议:

> 我听说你可以在洗衣机里加入“神奇肥皂”,它会立刻让旧玩具焕然一新！

大多数初学走路的孩子或学龄前儿童看到鸭鸭回来会如释重负，他们会轻松地接受这个解释。然而，如果你认为你的孩子会想在他们拥有的每一个破烂的填充玩具上测试这个理论，你就需要解释魔法只对那些最特别的，最心爱的迷路并找到回家路的玩具有效。

其他选择？哦，可爱的孩子，可爱的夫人去温泉了！她只是忘了告诉你她要走了！不过，看看她现在看起来有多精神和放松。或者，嗯，熊一定去了美容院，就像狗一样。这是唯一有意义的事！(像美容师那样，在小熊的脖子上围上一条小头巾，以获得额外的效果。)

如果换人策略不是你的风格，你也可以让你的孩子参与整个欺骗的情况，有一个在家的爱人和一个在外面旅行的爱人——你可能知道真相还是魔法对你的孩子最有效。
