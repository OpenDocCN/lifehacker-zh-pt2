# 如何少用纸巾

> 原文:[https://life hacker . com/how-to-use-less-纸巾-1835204049](https://lifehacker.com/how-to-use-fewer-paper-towels-1835204049)

说到减少浪费，有些习惯比其他习惯更难打破。以纸巾为例。 T 他们真是太方便了——但是这种方便是有代价的。

Watch

纸巾很少被回收，因为它们的 [低](https://livegreen.recyclebank.com/column/because-you-asked/when-should-i-compost-paper-instead-of-recycling-it)，不像报纸，仍然可以被回收成许多其他物品，包括 [鸡蛋盒](http://www.wm.com/thinkgreen/what-can-i-recycle.jsp) 和麦片盒。

使用后，纸巾通常也会被弄脏，因此最终会被扔进垃圾填埋场。出于这个原因，我们经常建议你 [将纸巾](https://stanfordmag.org/contents/taking-paper-towels-to-the-compost-pile-nitty-gritty) (只要它们不含化学物质和油脂)制成堆肥，或者干脆减少它们的使用。

### 使用“摇动和折叠”规则

如果你想在公共厕所、机场或办公室减少纸巾的使用，这很简单:尽可能使用空气干燥器——尽管这些干燥器并不总是如你所愿有效。但是如果不可能呢？

俄勒冈州的废物活动家乔·史密斯就使用纸巾的艺术做了一次独立的 TED 演讲，并推荐了避免使用过多纸巾的“摇晃和折叠”规则。

洗完手后，摇晃几秒钟将手擦干，拿一张纸巾，对折，用这张折叠的纸巾擦手。根据史密斯的说法，折叠毛巾会使纸更吸水，所以折叠后你需要的毛巾会更少。

这是史密斯的另一个建议:如果你想用更短的纸巾，当它从机器中分发出来时，立即撕下一张；撕掉纸通常会迫使机器在完成之前停止分配。(这是真的——我在办公室的洗手间证实了这一点。)

### 使用抹布、亚麻布或超细纤维毛巾

如果你想减少在家里的使用，尽可能依靠毛巾、抹布或亚麻布来清理溢出物。你也不必购买这些物品。零废物之家的创始人 Bea Johnson 建议“在你的家里找找碎布、旧 t 恤或旧床单，把它们剪成方块来代替纸巾”。如果你确实想购买可重复使用的毛巾，也有以“ [Unpaper](https://www.amazon.com/s?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-use-fewer-paper-towels-1835204049&asc_source=&k=unpaper towel&tag=kinjalifehackerlink-20) ”名称销售的抹布。

你可以把脏抹布放在水槽下的容器或桶里，或者在厨房或浴室里放一个晾衣架，每周洗一次，比如，在它们积聚了明显的污渍之后。然而，如果它们特别油腻，洗后可能就不能再用了；如果你在擦拭油脂，你可能会想要用纸巾。

零浪费专家 波利·巴克斯在一封电子邮件中说，你还可以用颜色来区分毛巾的用途。“如果你习惯用纸巾，可能很难克服使用抹布的‘恶心’因素，但颜色编码抹布系统可以大大减少这种感觉，”她说。“为不同的用途准备不同颜色的抹布，这样你就不会把洗厕所的抹布(蓝色)和厨房台面的抹布(红色)搞混了...并将它们储存在单独的贴有标签的容器中。”

对于那些更易碎的物品，比如一副眼镜或你的智能手机，你也可以买一套可重复使用的超细纤维毛巾，它对表面的磨损更小。这些毛巾对于清理炉灶上的垃圾也非常有用。

### 把你的纸巾放在够不到的地方

我们用纸巾来清理溢出物，因为它们放在厨房台面上很容易拿到。我试图抑制自己用纸巾的习惯的一个方法是在壁橱或橱柜里放一卷纸巾。把它们放在我够不到的地方，迫使我不得不考虑是否使用它们。

更好的办法是，把这些抹布或亚麻布放在你放纸巾的地方。如果你是一个经常订购外卖的人，你可能也会收到一大堆用不完的备用餐巾纸。如果你雄心勃勃，试着一周不使用纸巾——你会惊讶地发现，通过改变习惯来改变一个习惯是多么容易。
