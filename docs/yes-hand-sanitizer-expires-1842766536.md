# 是的，洗手液过期了

> 原文：<https://lifehacker.com/yes-hand-sanitizer-expires-1842766536>

现在，我们大多数人已经翻遍了家里的角落和缝隙，寻找被遗忘的洗手液。我们发现了一些样品瓶，这些样品瓶是从谁知道什么事件中拿出来的，我们从汽车后备箱的急救箱里取出了一个迷你瓶子，并掸掉了占了水槽下空间的半空瓶子。但是这些古老的消毒剂真的有效吗？洗手液会变质吗？不幸的是...



### 洗手液随着时间的推移会失去效力

事实证明，尽管洗手液并没有变“坏”，但随着时间的推移，它会失去效力。 [疾控中心的建议](https://www.cdc.gov/handwashing/show-me-the-science-hand-sanitizer.html) 是，如果没有肥皂和水，那么酒精含量至少为 60%的洗手液是很好的替代品。

正是这个 60%的数字带来了一个问题:酒精是一种挥发性化合物，这意味着它很容易蒸发。这意味着那瓶在你的运动包底部放了多久的洗手液现在可能比标签上说的含酒精量少。

### 检查有效期

洗手液 [受食品药品监督管理局](https://www.healthline.com/health/does-hand-sanitizer-expire#hand-sanitizer-vs-handwashing) 监管，要求瓶上印有批号和有效期。这一有效期通常是从生产日期算起的 2 到 3 年，是对活性成分(洗手液的活性成分是酒精)下降到标签上所印含量的 90%以下的估计。如果你的洗手液含有 70%的酒精，那么有效期就是这个百分比下降到 90%的时候，也就是 63%。这仍在有效范围内，但幅度较小。

开瓶后， [酒精的百分比会随着暴露在空气中而下降](https://www.prevention.com/health/a32052322/does-hand-sanitizer-expire/) 。因此，如果你有一瓶在某个早已被遗忘的日期打开的酒，即使它还没有过期，酒精含量也很有可能会显著下降。随着酒精百分比下降到 60%以下，消毒剂的效果会变得更差，尽管它仍然会有一些效果。

### **正确存放洗手液**

重要的是 [正确存放洗手液](https://safety.lovetoknow.com/household-safety-tips/does-hand-sanitizer-expire-fast-facts-safety-tips) 。这包括避免阳光直射，因为这会导致酒精蒸发。让洗手液远离小孩也很重要，因为一些品牌的颜色和气味会让人混淆。美国食品和药物管理局 [报告称，自疫情开始以来，打给毒物控制中心](https://www.fda.gov/consumers/consumer-updates/safely-using-hand-sanitizer) 的电话增加了，主要是由于意外摄入洗手液。请记住，如果它是彩色的或者闻起来很香，你的孩子很有可能会尝试喝它。避免将洗手液存放在高于华氏 73 度的温度下也很重要，因为它被视为易燃液体。

### 肥皂和水是有效的

如果你担心你的洗手液的有效性，你最好也经常洗手(这是你应该做的事情)。正如疾病预防控制中心指出的，洗手液在某些情况下效果很好，但是不能代替普通的肥皂和水。当你的手很油腻或者很脏的时候，洗手液也不太好用。

也就是说，如果你没有肥皂和水，但是你有一瓶旧的洗手液，那也比什么都没有强。只要确保在第一时间用肥皂和水清洗，并且停止触摸你的脸。
