# 一边用“Scener”看网飞和 HBO，一边视频聊天

> 原文:[https://life hacker . com/video-chat-while-watch-with-scener-1843483388](https://lifehacker.com/video-chat-while-watching-netflix-and-hbo-with-scener-1843483388)

前几天，我在想“G 奥什，自从我看了《创:遗产》后，它肯定已经永远消失了。“然后一个朋友在*动物穿越:新视野*里发现 [赛博服](https://villagerdb.com/item/cyber-suit) 在卖，她在我们岛聊里说了*同样的事情*。我们的解决方案？让我们一起看 *Tron: Legacy* ，用一个 **Chrome** 浏览器扩展来模拟我们实际上同时处于同一个空间的效果。

Watch

嗯， [**场景器**](https://chrome.google.com/webstore/detail/scener-%E2%80%93-virtual-movie-th/lkhjgdkpibcepflmlgahofcmeagjmecc) 技术上来说不会让我做*那个*，因为华丽的 *Tron* 续集在网飞或 HBO 上不可用，并且扩展仅与这两个流媒体服务一起工作。但是我还是要安装它，用它来主持共享观看夜。就是那么好。

该扩展完全免费使用(甚至还有 HBO 的 WarnerMedia 的 [加持](https://techcrunch.com/2020/05/14/scener-now-lets-you-co-watch-hbo-or-netflix-in-a-virtual-theater-with-up-to-20-people/) ，所以你不应该担心它会因为其中一家流媒体公司过得不好而从互联网上消失 。安装它就像安装任何 ol' Chrome 扩展一样简单:在 [Chrome 网络商店](https://chrome.google.com/webstore/detail/scener-%E2%80%93-virtual-movie-th/lkhjgdkpibcepflmlgahofcmeagjmecc) 上调出扩展，并通过提供您的姓名和电子邮件地址在 Scener 上获得创建一个新的帐户。

从那里，你可以点击 Chrome 工具栏中的场景器图标，这将启动你的弹出式电影院，分为两部分——主放映部分和侧边栏:

这个屏幕有点无聊，因为你必须创建或加入一个“剧院”——你和朋友们的小观察室——才能让扩展做任何事情。点击“创建”提示，直到你被要求选择网飞或 HBO (HBO Now 和 HBO Go for now 目前还不知道 Scener 是否会与即将上映的 [HBO Max](https://lifehacker.com/everything-you-need-to-know-about-hbo-max-1843322402) 合作。

然后你必须在场景器较大的“剧院”部分(左侧)登录上述服务。完成后，你会在右边栏看到一个邀请链接和一个房间代码。与你的朋友分享，并挑选一些给每个人看。一旦您开始播放电影或电视节目，您正在观看的内容将与您“剧院房间”中的朋友完美同步

通过对着你的网络摄像头麦克风大喊，或者，如果你喜欢，在 Scener 的内置聊天室中疯狂地输入，随意添加尖刻的评论。如果你厌倦了当虚拟放映员，点击你的实时图像左上角的远程按钮，将它传递给其他人，他们将完全控制你的观看会话:

就是这样。虽然 Scener[很难成为与朋友同时观看电影和视频的唯一](https://lifehacker.com/watch-shows-on-hulu-disney-and-more-with-your-friends-1842924419)选项，但它是一个可靠、简单的解决方案——对于网飞和 HBO 的粉丝来说，聊天框使它比 [Vemos](https://vemos.org/) 略具吸引力，尽管该扩展支持更多的流媒体网站。
