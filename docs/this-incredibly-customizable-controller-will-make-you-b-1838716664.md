# 这种难以置信的可定制控制器将使你更好地在开关

> 原文：<https://gizmodo.com/this-incredibly-customizable-controller-will-make-you-b-1838716664>

对于那些喜欢玩比 Switch 的 Joy-Cons 更实在的游戏的人来说，任天堂是一款出色的独立控制器。Pro 控制器一直是我向任何寻找 Switch 游戏手柄的人推荐的，但现在不是了。 [8BitDo 的 SN30 Pro+](https://www.8bitdo.com/sn30-pro-plus/) 并不完美，但它最大的吸引力——它能够被 [完全定制和重新编程](https://gizmodo.com/this-super-customizable-controller-might-be-a-secret-we-1836304487)——远远弥补了它的几个缺陷。

如果你是玩着超级任天堂(Super Nintendo)、世嘉创世纪(Sega Genesis)或 N64 等经典游戏机长大的，你可能会记得第三方控制器非常糟糕。如果我去朋友家玩电子游戏，我总是确保带上自己的控制器，以免被山寨产品困住，注定要在真人快打输掉比赛。(有时木匠的工具*是罪魁祸首*。)

谢天谢地，第三方控制器的状况已经有所改善，同时在价格上仍然具有竞争力。如果你需要几个额外的控制器来玩分屏多人游戏，现在没人会抱怨被一个杂牌游戏手柄卡住了。但有了 8BitDo 的新 SN30 Pro+，现在实际上很难推荐任天堂 70 美元的硬件，因为 50 美元就可以买到一个开关友好的无线控制器，手感更好，按钮的触觉反应更好，可以与其他游戏系统配合使用，甚至可以完全重新配置和定制，以适应最痴迷的游戏玩家。

SN30 Pro+看起来像是 Switch Pro 控制器和 PS4 DualShock 游戏手柄的混搭，带有并排的模拟操纵杆和一个方向板，配有一组四个动作按钮。实际上，我更喜欢 Switch Pro 控制器的不对称模拟杆布局，但 SN30 Pro+上的方向板感觉更模糊(以一种好的方式)，更适合玩复古 2D 游戏所需的密集按钮操作，而光滑的操作按钮重新唤起了 N64 和 SNES 自己的控制器的美好回忆。

另一个重大改进是，与 Switch Pro 控制器和 Joy-Cons 不同，SN30 Pro+包括一对具有广泛行程的模拟触发器，以便在玩赛车游戏等游戏时，您可以更好地控制加速度。这是一个 Switch 库中许多游戏都支持的功能，但任天堂的实际控制器硬件却不支持，这很奇怪。SN30 Pro+还包括运动控制和隆隆声。两者都不如任天堂自己的实现有效或细致，但这些也是我个人不关心的特性。

与过去的第三方控制器不同，8BitDo 的构建质量令人满意地稳定，我越来越多地发现自己正在接触该公司的游戏硬件。SN30 Pro+还包括一个可充电电池，可以通过 USB-C 电缆充电，但它只能提供大约 20 小时的游戏时间。任天堂的 Pro 控制器号称有 40 个。但是 8BitDo 的电池组是可拆卸的，所以如果你在游戏中遇到紧急情况，你可以把它换成备用电池或一对现成的 AA 电池。

将 SN30 Pro+与交换机、电脑甚至安卓设备同步并不复杂，但它需要在每个特定设备通电时按下不同的按钮组合。这是一个我总是需要先在网上查找的过程，所以我希望 8BitDo 能够在他们的控制器背面打印或蚀刻这些指令和按钮组合，以便于参考。然而，一旦同步，只要 SN30 Pro+和我的交换机都通电，它们就会自动配对。

在这一点上，我仍然认为它是 Switch Pro 控制器和 SN30 Pro+之间的一个掷硬币，但对于后者，你也可以得到 8BitDo Ultimate 软件，从某种意义上说，可以成为真正的游戏规则改变者。如果你曾经进入一个游戏的设置去重新映射几个按钮或者颠倒一个游戏杆的行为，那么你应该很熟悉终极软件能做什么。但是，虽然只有少数游戏在其设置中提供了良好的控制器定制水平，但 SN30 Pro+将它带到了一切。

8BitDo Ultimate 软件适用于 Windows 和 macOS 电脑，让游戏玩家可以定制控制器的几乎每个方面，并保存各种配置以方便以后访问。您可以重新映射任何按钮，调低或增加隆隆声反馈，编程自定义宏以通过一次按键触发一系列命令，调整模拟操纵杆和触发器的灵敏度，并通常使控制器完全按照您喜欢的方式运行。

我明白为什么这种可定制性对一些游戏玩家来说很重要，但 SN30 Pro+的最佳功能看起来不像是我实际使用过的东西，直到我下载了*星球大战:绝地武士 II:绝地弃儿—* 一款我非常兴奋地看到移植到交换机上的游戏，直到我玩了它。在 PC 上，游戏的前几个任务，当你的角色不使用原力或光剑时，用鼠标很容易击败，但用游戏手柄玩则更具挑战性。以至于我立即寻找作弊代码，从一开始就给我的角色力量、光剑和公认的不公平优势。

更糟糕的是，在*绝地武士 II 的*设置中，翻转操纵杆的 Y 轴进行瞄准是不可能的:这个选项是很多游戏玩家(包括我自己)更喜欢的。沮丧之余，我转向 SN30 Pro+,果然，反转操纵杆上的 Y 轴是一个简单的一键修复，重新排列动作按钮也是如此，因为出于某种原因，我在玩游戏时总是不小心激活武器选择菜单。在一个我准备彻底放弃的游戏中，花了整整两分钟来修复和补救一些缺失的功能。

如果说 SN30 Pro+有什么我想改进的地方，那就是现在它只能在电脑上使用 8BitDo Ultimate 软件应用程序进行配置，并且必须反复使用 USB-C 电缆连接和重新连接控制器来测试你的调整变得乏味。这也使得当你离开笔记本电脑时，很难加载其他配置。一个无线工作的移动版本的应用程序将是一个很大的改进，因为你可以转向你的 iOS 或 Android 手机，这总是触手可及。

这也有助于在控制器上进行 A/B 测试，可以上传两种不同的配置，并通过开关快速切换，这样你就可以很容易地测试哪个在游戏中玩得更好。但即使有这些建议的改进，我仍然对 SN30 Pro+印象深刻。8BitDo 终于设法做了一件这个老派游戏玩家从未想过可能的事情:它创造了一个不仅比任天堂的 [更好，而且也更便宜的](https://www.8bitdo.com/sn30-pro-plus/) 控制器。

### **自述**

*   这是一款第三方控制器，性能超过了任天堂自己的 Switch Pro 控制器，价格便宜 20 美元。
*   8BitDo Ultimate 软件提供了令人印象深刻的控制器定制水平，但它只能在 Windows 或 macOS 计算机上工作。
*   SN30 Pro+确实包括动作控制功能，但感觉不如 Switch 的 Joy-Cons 细致入微。
*   包括模拟触发器，任天堂的控制器都没有，尽管这是开关支持的一个功能。
*   依靠蓝牙，所以它也可以连接到个人电脑，安卓设备，甚至树莓派。
*   可拆卸充电电池，不能充电的时候可以换 AA。
*   没有内置 NFC，所以你将无法使用这个控制器来加载 Amiibo figure 内容。
