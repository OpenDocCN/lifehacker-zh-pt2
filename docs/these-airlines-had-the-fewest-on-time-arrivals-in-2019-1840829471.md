# 这些航空公司在 2019 年的准点到达次数最少

> 原文：<https://lifehacker.com/these-airlines-had-the-fewest-on-time-arrivals-in-2019-1840829471>

如果你在不久的将来乘坐 Frontier 或 JetBlue，如果你的航班不能准时到达，不要感到惊讶。数据收集机构去年跟踪了超过 560 万次航班，根据准点到达情况对所有北美航空公司进行了排名。报告发现，边疆航空公司和捷蓝航空公司在让旅客在时间到达目的地方面表现最差。



2019 年，只有 74.4%的 Frontier 航班和 74.8%的 JetBlue 航班按时成功抵达。同时，达美航空公司和阿拉斯加航空公司排名最好，分别有 85.69%和 82.01%的航班准时到达登机口。

Cirium 还调查了全球航空公司的准点到达情况，达美航空公司排名第三(也是唯一一家排名第一的美国航空公司)；俄罗斯航空公司俄罗斯国际航空公司(Aeroflot)是 2019 年唯一最准时的航空公司，以 86.68%的航班准时到达率勉强击败达美航空和日本航空。

然而，这里还有一个重要的因素:当达美航班延误时，预计它会*非常*延误。尽管达美航空的准点率最高，但航班的平均延误时间为 58 分钟，而阿拉斯加的平均延误时间仅为半小时。(捷蓝航空的平均延误时间高达 75 分钟。)航空公司以 [增加航班时间](https://www.cntraveler.com/story/airlines-are-padding-flight-times-to-avoid-being-late) 而闻名，以防延误，所以尽管达美航空可能会让你在时间到达目的地，但你仍然可能会经历一些起飞延误。
