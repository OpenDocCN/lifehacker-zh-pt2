# 如果你想优先考虑某件事，把它放在你能看到的地方

> 原文:[https://life hacker . com/if-you-want-to-prioritize-something-put-it-where-you-c-1841954967](https://lifehacker.com/if-you-want-to-prioritize-something-put-it-where-you-c-1841954967)

有两个非常好的技巧可以帮助你记住每天早上出门时需要携带的东西。其中一个是给自己一个口头禅，比如“电话、钱包、钥匙”，另一个是把你需要随身带的东西放在你的门前。

Watch

这个想法是，你将无法打开你的门而不推撞需要在当天下午邮寄的包裹，或者无法转动你的门把手而不干扰便利贴，上面写着“你有你的盒饭吗？”你已经让需要记住的东西*高度可见*，这意味着你更有可能记住它。

事实证明，这种方法也适用于不太明显的优先事项——比如你的价值观，或者你想如何支配时间。

正如纽约大学心理学副教授、新书《更清晰、更接近、更好:成功人士如何看待这个世界》 、 [的作者艾米丽·鲍尔西提斯最近告诉格雷琴·鲁宾](https://gretchenrubin.com/2020/02/emily-balcetis-author-interview/) :

> 我发现的最有效的策略之一是我称之为“物化”我的实践。我们优先考虑那些进入我们日程表的事情。我们在当天的任务清单上看到的事情。我们对那些只记在脑子里的东西漠不关心。

当 Balcetis 想学习如何打鼓时，她把练习时间安排在日程上，就像她安排工作会议或预约医生一样。我以同样的方式对待我的待办事项和日历；如果我想完成某件事(不管是个人的还是职业的)，我会把它列在清单上，如果可能的话，把它分配到特定的时间。

为什么？因为这些是我生活中我想优先考虑的部分，不管我是在赶一个自由职业者的最后期限，确保我及时赶到基督教青年会上课，还是和一个朋友联系。(是的，我是那种在日历上写着“用某某人签到”的人——是的，这很有效。)

如果事情发生了，而项目没有完成，不管是因为另一个任务比预期的时间长，还是因为另一个优先事项强调了它的重要性，我都会把它从待办事项列表和日历中删除。这样，我就能看到我真正优先考虑的是什么。

简而言之，我每天都以我希望完成的事情的清单开始，以我实际做了什么的记录结束一天——如果某个项目一直从清单上掉下来，我必须问自己生活中是否有什么东西失去了平衡，或者那个项目是否不够重要，以至于不能完成。

但是清单本身就相当于门把手上的便利贴或者放在你鞋子旁边的打包好的运动包；一个预先准备好的提醒，提醒我不想忘记的事情。

这就是为什么它有效。
