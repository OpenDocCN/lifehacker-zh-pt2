# 如果你拥有一个 Pixel，你可以从谷歌获得 500 美元

> 原文：<https://lifehacker.com/if-you-own-a-pixel-you-could-get-up-to-500-from-googl-1834783480>

谷歌同意支付 725 万美元来解决 Pixel 智能手机所有者对该公司提起的集体诉讼 。该诉讼称，谷歌故意销售焊接连接有缺陷的 Pixel 手机，导致麦克风普遍故障。谷歌已经解决了这一案件，该公司可能欠 Pixel 所有者每台设备高达 500 美元。即使你没有资格获得全部 500 美元，谷歌也有可能至少欠你*一些*钱。

Watch

该诉讼涉及 2017 年 1 月 4 日之前在美国购买的任何原装 Pixel 或 Pixel XL 手机，无论是否存在缺陷。转售/二手购买不包括在内。如果您清除了这些要求，以下是和解协议中商定的各种支付选项:

*   所有原始 Pixel 设备的所有者都有资格获得至少 20 美元的赔偿，不管你的手机是否有缺陷的麦克风。
*   如果你购买了一个麦克风损坏的 Pixel 设备，并从谷歌收到了第二个有缺陷的 Pixel 设备，你有资格获得全额 500 美元的付款。
*   那些只拥有一个缺陷像素的人有资格获得高达 350 美元的手机赔偿。
*   如果您在手机上支付了保险免赔额，那么您也有可能获得全额免赔额报销。

### 如果你不认为有人欠你钱

该和解协议还有待法庭批准，这意味着条款可能会改变。与此同时，您可以 [注册更新和解协议](https://chimicles.com/google-pixel-smartphone-microphone-defect-class-action-investigation/) ，这样您就不会忘记在领取集体诉讼现金时提交索赔。(如果那个链接不管用，因为 Chimicles Schwartz Kriner&Donaldson-Smith LLP 的网站已经乱七八糟， [试试这个](https://chimicles.com/chimicles-schwartz-kriner-donaldson-smith-llp-files-motion-for-preliminary-approval-of-7-25-million-class-action-settlement-in-google-pixel-litigation/)——回头试试原来的链接，以防万一。)

这还不是谷歌潜在支出的结束:该公司还同意再向 [支付 965 万美元，以解决今年](https://www.engadget.com/2017/04/21/lawsuit-takes-aim-at-google-huawei-over-nexus-6p-battery-issues/) 关于 Nexus 6P 电池关键问题的类似诉讼。对 Pixel 2 手机上类似麦克风故障以及原始 Pixel 和 Pixel XL 老化的调查也在进行中，但要找到任何潜在的解决方案还需要一段时间。
