# 向保释基金捐款时要小心虚假的 Venmo 骗局

> 原文：<https://lifehacker.com/beware-fake-venmo-scams-when-donating-to-bail-funds-1843823866>

明尼苏达自由基金，一直在救助明尼阿波利斯的抗议者，已经成为本周捐赠的热门事业。太好了！但是一定要去他们真正的网站——不要向以他们的名义流传的 [假网站](https://twitter.com/MNFreedomFund/status/1266550293157892100) 捐款。

Watch [https://lifehacker.com/embed/inset/iframe?id=twitter-1266550293157892100&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1266550293157892100&autosize=1) 

对于你想捐赠的任何事业来说，这都是一个很好的建议:总是检查你要捐赠的组织，并确保你已经找到了最直接的捐赠方式。

### 找到该组织的官方网站或经过验证的社交媒体帐户

确保你面对的是组织本身。在社交媒体上，你经常可以看到还有谁关注这个账户。肩负相同使命的组织通常会相互关注和支持，您也可能会注意到受信任的朋友在关注该帐户。

### 教育自己他们做什么，他们需要什么

来自朋友或有影响力的人的支持是很好的，但是让自己了解一个组织所做的工作总是合适的。确保他们是你想要支持的群体，并从他们的网站上找到支持他们的最佳方式。

例如，食物银行 [可以用钱做比食物捐赠](https://lifehacker.com/donate-money-not-food-to-your-local-food-bank-1830561819) 更多的事情。有些人可能还会向你推荐其他基金。例如，明尼苏达自由基金会得到了如此多的捐赠，他们要求捐赠者分享爱， [列举了其他几个组织](https://twitter.com/MNFreedomFund/status/1266936872674172929) 现在也可以使用帮助。

你可能还想在 Guidestar 或 Charity Navigator 等网站上查看一个不熟悉的组织 ，以确保它们是合法的。

### 以他们要求的方式捐赠

当你准备好给予时，按照组织要求的任何方式去做。如果他们的网站上有一个大大的捐赠按钮，那是你最好的选择。如果他们提供了多种选择，请注意某一个是否被列为首选。当有疑问时，最直接的选择是最好的。
