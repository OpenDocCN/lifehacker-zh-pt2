# 根据马丁·艾米斯的说法，如何正确使用同义词词典

> 原文：<https://lifehacker.com/how-to-use-a-thesaurus-correctly-according-to-martin-a-1836891843>

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-UnWIsWoUQuk&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-UnWIsWoUQuk&start=0) 

马丁·艾米斯使用同义词词典。这位英国小说家和评论家被认为是一种势利小人，而同义词库被认为是业余作家试图听起来聪明而滥用的危险工具，但正如你可能已经猜到的那样，埃米斯使用同义词库的方式与大学新生非常不同。在上面的大思考视频(via [开放文化](http://www.openculture.com/2019/07/martin-amis-explains-how-to-use-a-thesaurus-to-actually-improve-your-writing.html) )中，他描述了使用同义词库的更好方法。



就像 [上百万篇关于写作风格的文章警告](https://slate.com/human-interest/2014/08/writing-clearly-in-student-papers-the-right-click-thesaurus-and-rogeting-are-ruining-writing.html) 的那样，同义词词典的危险在于，你可能会用一个不恰当的词替换一个恰当的词。你可能会用自命不凡的“抄写员”来代替非常好的词“作家”，或者用“停止”来代替“停止”，或者一些其他类型的学期论文废话。

你这样做可能是因为你想听起来更聪明，因为你不知道越简单的单词通常越好。或者你可能会这样做，因为你害怕在一篇作品中多次使用同一个词。

第二个习惯——用不同的同义词替换一个单词的每个实例，以避免重复说同一个单词——被称为 [【优雅的变化】](https://en.wikipedia.org/wiki/Elegant_variation) ，但不是一种恭维。它通常出现在糟糕的新闻报道、博客帖子以及那些认为自己有一天会写书的人写的被改写的 Yelp 评论中。非盟特别警告不要这样做。(同一个词用几次就好了；读者不会像你一样注意到重复。或者用代词。)

取而代之的是，Amis 使用一个词库来匹配他的单词的风格和节奏。他可能会发现他现在的单词对于句子来说太长或太短。或者他可能想避免一个尴尬的押韵或头韵。他指的是弗拉基米尔·纳博科夫，他把他的小说*的英文标题*改成了*的*。大声说出来，你会听到改善。

他还通过在字典中查找单词来仔细检查自己的用词——不一定是检查它们的定义，而是检查它们的词源。例如，他说，*中的*拉皮德*来自拉丁语中的石头，所以虽然他可能指的是破旧的房子，但他不会指破旧的树篱。*

你不必像埃米斯那样小心用词。但是如果你真的想写得更好，多注意你的单词的节奏，并研究它们的用法和词源。你的同义词词典和字典只需点击一下鼠标，好好使用它们。
