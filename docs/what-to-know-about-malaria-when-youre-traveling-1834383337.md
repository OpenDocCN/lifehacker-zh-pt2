# 当你在旅行时，应该知道哪些关于疟疾的知识

> 原文：<https://lifehacker.com/what-to-know-about-malaria-when-youre-traveling-1834383337>

疟疾是世界上许多热带地区的地方病，如果你在旅行中感染了这种疾病，它会导致严重的并发症。所以，如果你计划去热带地区旅行，以下是你需要知道的。



### 弄清楚你的旅行计划是否会给你带来风险

西非是疟疾传播风险最高的地区，非洲、亚洲、大洋洲的其他地区以及南美洲的部分地区也可能构成疟疾风险。但这取决于你确切的旅行地点和时间。

所以，当你知道你即将进行一次热带旅行时，去看看懂旅行医学的医生或诊所。他们会帮助你计算出你的目的地患疟疾的风险有多大，根据你的计划，他们可能会推荐预防疟疾的药物。即使你以前去过那个地区，或者小时候在那里生活过，你仍然有危险。(疾病预防控制中心称这种情况的人为“VFR 旅行者”，意为“拜访朋友和亲戚”，他们有一个关于你患疟疾风险的专页。)

为了预览健康专家可能会告诉你什么，请查看来自 CDC 的这张表，它详细列出了世界各地的疟疾风险和推荐的药物。正如本风险评估指南[中所解释的，建议还将取决于你计划在该地点停留多长时间，以及你自己的健康状况。](https://www.cdc.gov/malaria/travelers/risk_assessment.html)

### 如果医生给你开了药，那就吃吧

您可能会获得全程服用的疟疾药物(“化学预防”)。如果有，就拿去吧。这并不能保证你会避免疾病，但它大大提高了你的机会。如果你在服药期间感染了疟疾，你的病情可能不会那么严重。

如果你服药不一致，你仍可能感染疟疾，但症状出现可能需要更长时间。帮你自己和你的医疗保健提供者一个忙，按照建议服用药物。

在某些情况下，如果您感染了疟疾，并且无法获得医疗保健服务，您也可能会获得 [【备用紧急】药物](https://www.fitfortravel.nhs.uk/advice/malaria/sbet-standby-emergency-treatment-for-malaria) 来治疗疟疾。知道如何使用，随身携带，但不要以此为借口，跳过你的其他药物治疗。

### 避免蚊虫叮咬

不管你是否在服药，避免蚊虫叮咬是很重要的。(疟疾是由一种微小的寄生虫引起的，这种寄生虫通过蚊子在人与人之间传播。驱蚊剂很重要:疾病预防控制中心推荐使用避蚊胺、IR3535、柠檬桉树油或匹卡瑞丁的 T2 T3 驱蚊剂。

纱窗和其他屏障也可以帮助你避免蚊虫叮咬。如果你不确定会有什么样的住宿条件，或者如果你知道你会在一个蚊子和疟疾常见的地方，你可以带一顶经过杀虫剂处理的蚊帐，在你睡觉的时候让蚊子远离你的身体。(塞在床垫下面。传播疟疾的蚊子在晚上最活跃。

预防蚊虫叮咬的常识也适用:考虑穿长袖衬衫，尽可能多地遮盖裸露的皮肤；蚊子活跃时留在室内；用蚊帐盖住婴儿背带。

### 注意症状

即使在旅行归来后，你也可能患上疟疾。至少持续三个月，留意发烧或类似流感的症状， [包括](https://www.cdc.gov/malaria/about/disease.html#uncomplicated) 头痛、寒战和肌肉无力。一定要告诉你的医生你去过疟疾流行的地区。
