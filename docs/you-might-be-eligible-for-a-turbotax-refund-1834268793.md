# 你可能有资格获得涡轮退税

> 原文：<https://lifehacker.com/you-might-be-eligible-for-a-turbotax-refund-1834268793>

如果你的收入低于 34，000 美元，并向 TurboTax 缴纳了税款，那么是时候打电话要求退款了。



[正如 ProPublica 解释的](https://www.propublica.org/article/how-to-get-turbotax-refund) :

> 如果你是去年收入低于 34，000 美元的数百万美国人之一，你应该可以使用免费版的 TurboTax。如果 TurboTax 让你购买付费版本，那就值得给公司打个电话。

致电 **888-777-3066** 与 TurboTax 代表交谈并获得退款。ProPublica 报告称，已有多名纳税人成功申请并收到了退款，因此，如果您符合条件，请拨打电话。

如果你想知道更多关于 TurboTax 如何使用 ProPublica 所谓的“欺骗性设计”从有资格免费申报的纳税人那里赚钱， [阅读这个详细的解释者](https://www.propublica.org/article/turbotax-just-tricked-you-into-paying-to-file-your-taxes) :

> 我们仔细查看了 TurboTax 网站的源代码，注意到了一些奇怪的事情。尽管我们点击了“免费担保”选项，并满足了免费申请的所有要求，但该公司已经将我们标记为潜在的付费客户。
> 
> 在源代码中，TurboTax 将我们标记为“NONFFA”那代表“非自由文件联盟”换句话说，我们终究没有走上免费申请的轨道。

如果你今年的收入低于 34，000 美元，并向 TurboTax 申报，你能免费申报吗？还是软件指导你使用它的付费选项之一？
