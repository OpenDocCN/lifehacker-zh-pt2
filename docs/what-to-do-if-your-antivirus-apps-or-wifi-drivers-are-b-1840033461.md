# 如果您的防病毒应用程序或 Wifi 驱动程序阻止了 Window 10 更新，该怎么办

> 原文:[https://life hacker . com/what-to-do-if-your-anti-virus-apps-or-wifi-drivers-are-b-1840033461](https://lifehacker.com/what-to-do-if-your-antivirus-apps-or-wifi-drivers-are-b-1840033461)

微软正在阻止某些运行旧版本 AVG 和 Avast 杀毒软件以及过时的高通 WiFi 驱动程序的个人电脑安装 Windows 10 版本 1809、1903 和 1909。原因如下:由于潜在的漏洞和不兼容问题，微软不再允许 Windows 10 在任何和所有系统上自动更新。

这些特定的阻止措施更进一步，彻底阻止使用旧版本 AVG 和 Avast 防病毒产品以及某些高通 wifi 驱动程序的用户手动安装 Windows 10 更新。但这并不是一件坏事，因为没有稍微改进的 Windows 10 版本总比根本无法工作的版本好，对吗？

不幸的是， [ZDNet 报道](https://www.zdnet.com/article/microsoft-blocks-windows-10-upgrades-for-these-avast-and-avg-users/) 许多用户已经数月无法升级到最新版本的 Windows 10。如果你遇到了这个问题，那么你的防病毒应用或网络适配器驱动程序可能已经过时，需要升级，然后才能安装新版本的 Windows 10。

### 如何更新 AVG 和 Avast 杀毒软件

Windows 10 更新块会影响运行以下应用程序 19.5 或更低版本的 AVG 和 Avast 用户:

*   Avast 高级安全
*   Avast 总理
*   Avast 互联网安全
*   Avast 免费杀毒软件
*   AVG 终极版
*   平均互联网安全
*   AVG 杀毒软件免费

[Avast](https://support.avast.com/en-ww/article/253) 和 [AVG](https://support.avg.com/SupportArticleView?l=en&supportType=home&urlName=AVG-Antivirus-Windows-10-update) 拥有支持页面，其中包含相应软件更新和安装指南的链接，可以让您的软件保持最新并解决 Window 10 更新问题。

### 更新高通 wifi 驱动程序以修复 Windows 10 更新问题

如果你的 AVG 或 Avast 防病毒应用是最新的(或者你一开始就没有使用它们的软件)，那么你的 wifi 驱动程序可能会阻止你安装 Windows 10 更新。您可以通过检查驱动程序更新来解决此问题。方法如下:

1.  在 Windows 任务栏搜索框中，搜索**“设备管理器”**并打开。
2.  向下滚动到**“网络适配器”**，然后单击箭头图标显示所有已安装的组件。
3.  从列表中找到您的高通设备，然后右键单击其名称并选择**“更新驱动程序”**
4.  单击**自动搜索更新的驱动程序**
5.  让过程运行。如果安装了新的驱动程序，您现在应该是最新的，并且能够成功安装新的 Windows 10 更新。否则，您需要手动下载并安装网络适配器所需的驱动程序。你可以在微软的支持页面 上找到这样做的说明。
