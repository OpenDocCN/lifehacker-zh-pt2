# 最好的饮食计划是自私

> 原文：<https://lifehacker.com/the-best-meal-planning-hack-is-selfishness-1838070043>

在后即时锅食谱的某些时候，“膳食计划”变成了“打包五或六份相同的午餐，假装很兴奋地把它们都吃光”的代码。当然，如果你真的*想把它们都吃掉，那是一回事——但是看到关于 [如何坚持饮食计划](https://lifehacker.com/create-a-meal-plan-youll-actually-stick-to-1821880607) 的文章几乎和饮食计划本身一样多，那就有问题了。解决方法可能很简单，就是把自己放在第一位。*

*我认为自私是一种被犯罪低估的生活帮，尤其是在烹饪方面，但它不是那种带你去广播的；没有人想被视为自私，尤其是女性。想象一下，当我的老朋友黛比·佩雷尔曼为《纽约时报》撰写的一篇名为《T2》、《T3》、《现实生活中的家庭膳食计划》、《T5》的优秀文章中提出这一点时，我有多高兴整篇文章对任何有孩子的人来说都值得一读，但对我来说，她的第一条建议是改变游戏规则的:“找一个你非常非常想做的食谱。”她接着提出了一个强有力的理由，说明她可以一次不理会别人想要什么(强调是我的):*

> *我没有说，“找一个你的孩子可能会接受贿赂，同意吃的食谱，”或者“找一个能在你回家的六分钟内，在每个人都崩溃之前做好的食谱。”...如果这不是你真正想吃的菜，就很难激励自己去做。首先关注你的渴望。*成为一丁点儿(喘息！)自私。**

*家庭厨师经常把别人的需求放在自己的需求之前。而不是问，“今晚吃晚饭我兴奋吗？”而是“我的孩子/配偶/室友的偷吃贼会怎么想？”不管你要养活多少人，计划一周的膳食都需要大量的脑力和体力劳动；你也许会对最终结果感到满意。不要围绕你应该吃什么来计划你的一周——当然，这是避免你认为你不应该吃什么的代码——试着优先考虑你想要什么。*

*这在理论上听起来很棒，但应用起来就是另一回事了。最明显的障碍是，你不可能一直取悦所有人，尤其是一桌挑食的人。但除此之外，“膳食计划”的概念与饮食和减肥文化如此纠结，以至于“膳食计划”在功能上已经等同于“节食”——大批量食谱往往以大量藜麦、蒸蔬菜和鸡胸肉为特色，而不是奶酪和意大利面等有趣的东西。事实上，一份好的提前制作食谱有两个确切的要求:它必须容易放大或缩小，而且它必须能够在冰箱里放置几天而质量不会严重下降。就是这样。不管你的营养需求是什么，有成百上千的食谱符合你的所有标准。*

*对你有理由感到兴奋的食谱进行优先排序不仅仅是一次简单的烹饪准备；这也是学习烹饪的最好方法。早期的成功让你想要坚持下去，尝试新事物，这只能帮助你提高。你做得越多，你就越擅长，也就越容易调整食谱来适应你家里的每个人——但在那之前，稍微自私一点没什么错。*
