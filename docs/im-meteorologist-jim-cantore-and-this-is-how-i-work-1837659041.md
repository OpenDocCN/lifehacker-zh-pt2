# im meteorologist jim cantore and this is how i work 1837659041

> 原文：<https://lifehacker.com/im-meteorologist-jim-cantore-and-this-is-how-i-work-1837659041>

当吉姆·康托尔抵达一个城市时，就会成为新闻。在像飓风道林这样的重大天气事件期间，新闻媒体甚至 [推测](https://www.miamiherald.com/news/weather/hurricane/article234465647.html) 关于他的 [下一个目的地](https://www.newsobserver.com/news/weather-news/article234734712.html) 。因为除了出现在像 AMHQ*和风暴故事*这样的节目中:下一章*之外，Cantore 还是天气频道现场报道恶劣天气的负责人。他到处都有超级粉丝，当他对天气感到兴奋时， [就会像病毒一样传播。我们和吉姆谈了他长期的气象事业，他疯狂的起床时间，以及他最大的错误。](https://www.youtube.com/watch?v=PdRWGMyeSYY)*

**地点:**佐治亚州亚特兰大
**当前职位:**气象频道电视网的摄像机气象学家
**当前电脑:**我在总部的办公桌上有一台戴尔笔记本电脑、一台显示器和一个扩展坞。尽管人们可能认为我有最棒的机器，但我真的没有。我只需要一台可靠的电脑，让我可以访问所有我喜欢的网站，这些网站我会全天候在不同的标签页中打开。
**当前移动设备:**我有一台苹果 iPhone 8。我现在不想买新的 iPhone，因为我知道很多功能都不一样，我是那种没坏就不要修的人。我只需要一部能让我访问 Twitter 并与我最喜欢的风暴观察者交流的手机，我很好。
**一个词最能描述你的工作方式:**勤奋

#### 请告诉我们一些你的背景以及你是如何走到今天这一步的。

我从小就对天气非常着迷，所以当我到了上大学的年龄时，我父亲鼓励我学习气象学。我照做了。我去了林登州立大学，在学校的电视台工作，当我毕业时，我得到了气象频道的工作邀请，我很兴奋。我渴望继续在电视台工作，直到我退休，所以我可以说我的职业生涯一直在气象频道工作。

#### 你的工作职责是什么？

我是一名业务预报员，我研究有关天气的数据，作为一名播音员，我必须让这些数据和信息容易被美国公众理解。当我在现场报道时，我会在大风暴和天气事件期间让公众了解最新情况，以一场比赛一场比赛的方式分享每一次更新，有点像体育广播员在棒球比赛期间分享更新。

#### 带我们看看最近的一个工作日。

我每天在凌晨 3:15 醒来，这是一个可怕的午夜，这是我工作中最困难的部分。那么早起床会使你疲惫不堪。

我起床后做的第一件事就是走到电脑前，阅读当天天气的所有数据。然后我做好准备，开车去总部，在我的演出开始前一个小时到达。我总是在进门之前就知道我要说什么。

一到办公室，我就化妆，戴上麦克风，然后主持 *AMHQ* [ *美国早间总部*。在我的节目结束后，我会参加电视网的早间编辑会议，如果有时间的话，我可能会去为我们的新节目*《风暴故事:下一章》录音。*

在总部呆了一整天后，我回家小睡一会儿，然后去好好锻炼，然后吃晚饭。我通常在晚上 8 点左右放松一下，9 点就睡着了，然后醒来，第二天再做一遍。

#### 你生活中离不开哪些应用程序、小工具或工具？

我离不开 [雷达望远镜](https://www.radarscope.app) 和 [气象虫](https://www.weatherbug.com) 。这些天气应用程序对我的个人生活和工作都很有帮助。天气不会停止。

#### 你的工作空间是什么样的？

在气象频道总部，我坐在靠近新闻室和工作室的开放式平面图中。我有一张桌子，笔记本电脑，扩展坞，两台显示器和一条电话线。我使用一个显示器显示当前天气，另一个显示器永久显示我的所有访问网站。相比站立式办公桌，我更喜欢坐在符合人体工程学的椅子上。

#### 你最喜欢的捷径或窍门是什么？

使用 Twitter 在旅途中快速收集天气新闻。

#### 带我们经历一个你在工作中的有趣、不寻常或挑剔的过程。

工作中一个挑剔的过程是戴上我的麦克风和 IFB；显然我从来没有把它们正确地穿上，所以楼层主管总是重做，哈哈。

#### 帮助你完成工作的人是谁，你如何依赖他们？

天气制作人！没有他们，我的工作是不可能的。他们加载图形，当我需要时，他们给我实时信息，等等。我们通过独特的工作室技术保持联系，使我们能够实时沟通。

#### 你在工作中犯过的最大的错误是什么，你是如何处理的？

几年前，我在推特上发布了我得到的信息，但不知情的是，这些信息是错误的，所以看起来我误导了我的追随者，但这不是我的意图。我当时向我的粉丝道歉，现在，我总是三次检查我分享的信息，所以我希望永远不会再遇到同样的情况。

#### 你如何记录你必须做的事情？

我的行政助理让我及时了解自己的工作。她是一个很好的合作伙伴，我很感激她的帮助。

#### **如何充电或休息？**

我在北乔治亚有一座湖边别墅，我去那里，在水上和阳光下度过时光。我喜欢花时间和我的孩子在一起，我也做了很多景观美化。

#### 你目前在读什么，或者有什么推荐？

我不太爱看电影，但我喜欢看老战争片。HBO 的兄弟乐队是我下一个要看的节目。我从来没有狂看一个节目，因为我很难坐得住，但我希望这是我狂看的第一件事。

#### 你喜欢听什么音乐？

我喜欢 80 年代的摇滚，像 AC/DC 和布鲁斯·斯普林斯汀。我也喜欢乡村音乐。

#### 你还希望谁来回答这些问题？

迈克·特劳特，我的朋友和培训中的气象学家。(他也是洛杉矶天使队的中外场手。)

#### 你收到过的最好的建议是什么？

我爸爸告诉我“做你喜欢做的事情”，因为你每天醒来都要去工作，所以最好是你喜欢的事情。这就是他鼓励我学习气象学的原因，我很高兴他这么做了。

#### 你仍在努力解决的问题是什么？

生活平衡。努力成为最好的父亲、雇主、朋友、伴侣以及介于两者之间的一切总是一个挑战。我已经学会了简化我的生活来更好地胜任这些角色，这真的很有帮助。
