# 使用这个免费的 Android 应用程序扫描竞争的 Wifi 网络

> 原文：<https://lifehacker.com/scan-for-competing-wifi-networks-with-this-free-android-1833078315>

**Android / iOS:** 一位生活黑客读者在最近的一篇 Tech 911 帖子中提醒我存在 Ubiquiti 的 WiFiman 应用程序——谢谢， [里克](https://lifehacker.com/1832831554) ！—如果你是一名[**Android**](https://play.google.com/store/apps/details?id=com.ubnt.usurvey&hl=en_US)用户，我全心全意地推荐它。该应用也适用于[**iOS**](https://itunes.apple.com/us/app/ubiquiti-wifiman/id1385561119?mt=8)，它有一些有趣的功能，但对于了解更多关于减缓连接速度的 wifi 拥塞的信息来说，它远没有那么有用。

Watch

首先，也是最重要的，这款应用是免费的。而且在安卓上功能齐全。安装它，接受一个许可，你将立即跳转到你的设备在你周围地区可以找到的所有 wifi 网络列表。您将看到它们的 SSIDs，包括您的邻居正在破坏的任何隐藏网络，以及它们的信号强度，以及它们是开放的还是安全的。

点击其中任何一个，你会看到一个应用程序检测到的信号强度的小运行图，以及所有在 2.4GHz 和 5GHz 范围内发射 SSID 的 wifi 无线电的列表。(记住，2.4GHz 网络活在 2400-2500MHz 之间，5GHz 网络活在 5200-5800MHz 之间。)

该应用程序真正的亮点是它的“频道”选项，你可以在屏幕底部找到它。点击它，你会得到你附近所有 wifi 网络的漂亮图表，包括它们的信号强度，它们的主要信道，以及它们的信道有多宽。

你的目标？确保您的 wifi 网络运行在来自其他无线网络的干扰最少的信道上(理想情况下，2.4GHz 网络为 1、6 或 11，5GHz 网络为较低的信道)。

如果你曾经使用过像 [Fing](https://lifehacker.com/ios-android-app-of-the-week-fing-network-scanner-1825205516) 这样的应用程序来搜寻你网络上的一切，那么这个应用程序的发现功能应该看起来很熟悉。WiFiman 将扫描您的网络，并报告它发现的一切——尽管这个过程可能需要一点时间。

最后，“速度测试”功能正如其名称所暗示的那样。它会检测各种服务，并报告你的连接质量，包括下载/上传速度，以及当你试图连接到谷歌，脸书和 Twitter 时，你的连接有多长时间的延迟。

至于 iOS 版本的 WiFiman，苹果对 iOS 内允许做什么应用的限制在很大程度上限制了它的功能。更糟糕的是，这款应用最有用的功能并不可用——主要是 wifi 扫描，其中包括该应用超级方便的跨频道 wifi 网络图。您仍然可以发现网络上的设备并运行速度测试，但仅此而已。
