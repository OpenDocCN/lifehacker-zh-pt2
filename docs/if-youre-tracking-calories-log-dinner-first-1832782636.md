# 如果你在追踪卡路里，首先记录晚餐

> 原文：<https://lifehacker.com/if-youre-tracking-calories-log-dinner-first-1832782636>

跟踪你吃的东西很像预算:你一整天都有一定量的卡路里或营养物质可以“消耗”。计划是关键，这可能有点伤脑筋:如果我在早餐上消耗了太多卡路里怎么办？晚餐时我会饿吗？

Watch

所以这里有一个小窍门，不管你的目标是什么都适用(包括如果你想吃更多的东西，又害怕晚餐会是一大份):首先追踪晚餐。

不要在你的追踪应用中把每天从早到晚都算在一起，而是把它当成从下午到下午的 24 小时。我第一次听说这个是从 redditor[u/woodsea](https://www.reddit.com/r/xxfitness/comments/aj77a1/what_are_some_underrated_tips_for_recomp_or/eetiwjx/)那里。假设你在某个星期一开始追踪:

*   忽略早餐和午餐。把你的晚餐记录在今天(星期一)的记录上。
*   第二天醒来，吃早餐，登录昨天(周一)的条目。
*   晚餐时间，开始追踪今天(周二)的条目。明天，也就是星期三，你将填写该条目的其余部分。

现在，午餐成了最容易预算的一餐:你知道你有多少卡路里或宏。如果你搞砸了，最后不得不吃比计划少的一餐，你不会饿很久——几个小时后你会重新开始第二天的工作。
