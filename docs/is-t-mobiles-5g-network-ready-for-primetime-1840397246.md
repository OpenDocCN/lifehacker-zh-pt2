# T-Mobile 的 5G 网络准备好迎接黄金时段了吗？

> 原文：<https://lifehacker.com/is-t-mobiles-5g-network-ready-for-primetime-1840397246>

又是一个大 5G 承诺的一年，尽管我们已经看到第一批 5G 设备上市，并且在过去的 12 个月里，越来越多的广告宣传 5G 速度的荣耀，但这项技术仍然不值得普通智能手机用户花钱。对于 T-Mobile 的大多数新“全国 5G”网络来说尤其如此。

Watch

T-Mobile 可能称其全国 5G 网络为 5G，但这不是真正的 5G。与其他运营商一样，T-Mobile 使用所谓的“低频带”5G，而不是更快的“毫米波”(mmWave)或高频带 5G。后者是“真正的”5G 技术，因为它利用未使用的无线电频率频谱来实现更快的通信和数据速度。而低频带 5G 在技术上是*的* 5G，但在性能上比不上 mmWave。

权衡之下，低频 5G 覆盖范围更广，更容易启动和运行，因为在某些情况下，运营商可以借用现有的网络技术。另一方面，毫米波 5G 需要全新的硬件，而潜在的智能手机速度参赛者必须与毫米波固有的有限范围相抗衡。

尽管美国电话电报公司和威瑞森正忙于建设将提供“真正”5G 体验的高频段网络(这是一个缓慢的过程)，但 T-Mobile 一直在利用其对低频段频率的访问，以更快的速度扩展其 5G 网络。但是，您通过其网络获得的性能可能会有很大差异。

### T-Mobile 的 5G 网络在许多地方比 4G 快不了多少

PCMag 的 Sascha Segan 最近 [发布了 T-Mobiles 5G 网络](https://www.pcmag.com/news/372478/t-mobile-5g-secrets-revealed-heres-where-it-doesnt-work-w?utm_source=muckrack&utm_medium=pr&utm_campaign=5gcitiesmclaren) 使用即将发布的一加 7T Pro 迈凯伦 5G 智能手机进行为期一周的测试的结果。Segan 在 T-Mobile 的网络上访问并测试了超过 22 个美国城市的 5G 速度，结果并不理想。

根据 Segan 的测试，T-Mobile 网络中的大多数城市通常比 4G 速度快不了多少，在某些情况下，甚至比 4G 速度慢*——尽管有几个城市的 5G 速度明显优于 4G:*

*   *爱达荷州博伊西*
*   *得克萨斯州埃尔帕索*
*   *田纳西州孟菲斯*
*   *内布拉斯加州奥马哈*
*   *内华达州里诺*
*   *德克萨斯州拉伯克*
*   *华盛顿州斯波坎*

*T-Mobile 为其低频段 5G 使用 20MHz、30MHz 或 40MHz 的频谱，使用 40MHz 频谱的城市(几乎没有电视信号干扰)往往会为您的移动网络速度提供坚实的提升。在所有其他城市，5G 充其量只是略有改善，许多城市与 4G 相比根本没有改善。在纽约，5G 比 4G 慢得多。*

### *一些 5G 实际上并不是 5G*

*这种黯淡的速度似乎首先挫败了 5G 网络的整个目的，并让人质疑升级到 5G 是否值得——除非你碰巧是 T-Mobile 的 5G 手机客户，并且住在 T-Mobile 可以接入 40MHz 频谱的少数几个城市之一。*

*然而，Segan 的结果凸显了 5G 的另一个主要问题:运营商仍然可以合法地称之为 5G——即使它几乎没有超过 4G，甚至比 4G 慢*——因为 5G 指的是信号类型，而不是网络速度。这使得客户很容易认为他们正在获得 5G“速度”，而他们只是使用一种发送和接收 5G 信号的设备。速度取决于网络性能，而不仅仅是手机的性能。**

**更糟糕的是:正如我们之前报道的那样，许多 5G 网络和设备会在没有警告的情况下将用户降至 4G LTE，但仍然声称自己在 5G 上；事实上， [一些设备直截了当地谎称你是否在 5G 上](https://lifehacker.com/your-smartphones-5g-icon-might-be-bogus-1834581479) 。这不是一个好的外观，它应该让你思考一下，在不久的将来，你是否甚至需要购买一个 5G 设备。在许多情况下，5G 实际上没有太多优势。**

### **慢慢变好**

**尽管 T-Mobile 从 PCMag 获得的 5G 成绩单平平，但 5G 仍有增长空间。网络性能和可用性越来越好，还有大量未开发的 5G 频率，公司将在 2020 年及以后开始使用。**

**5G 设备也变得越来越好，一些设备甚至可以同时结合 4G 和 5G，速度比单独使用 4G 或 5G 更快——例如 Segan 用于测试的一加 7T Pro 5G 迈凯轮。它们仍然不太值得涨价，但明年有望更接近。**

**但即使技术在慢慢改进，5G 的物流问题也必须在它成为许多公司追求的普遍标准之前得到解决。这些挑战包括材料问题，如 5G 设备的高功率要求和高昂的价格标签，以及基础设施问题，如毫米波 5G 信号容易被多云天气(甚至你的手)等日常障碍阻挡。**

**正是因为这些问题，我在 2019 年的最后几周对 5G 持相同的看法，以至于我在 2018 年底: [简直不值得](https://lifehacker.com/5g-phones-are-coming-soon-but-you-should-wait-to-buy-o-1831406561) 。至少现在还没有。**
