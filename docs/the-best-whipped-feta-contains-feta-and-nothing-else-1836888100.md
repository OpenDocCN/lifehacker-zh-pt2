# 最好的搅打羊乳酪只含羊乳酪，不含其他成分

> 原文：<https://lifehacker.com/the-best-whipped-feta-contains-feta-and-nothing-else-1836888100>

像其他消费品一样，食物也受资本主义之风的影响，这导致了与季节性无关的趋势。搅打羊乳酪目前很流行，但也很美味，所以我——这类东西的仲裁者——对它的流行很满意。

当一种食物发现自己拥有某个时刻时，它也会发现自己被写了很多。这种情况发生在 [切片水果](https://www.eater.com/2019/7/31/20748861/cut-fruit-summer-act-of-love) (因为这关系到你的父母有多爱你)和搅打羊乳酪上。网上有很多搅打羊乳酪的食谱，但我看到的每一个都有一个共同点:它们都用另一种更好的乳制品来源补充羊乳酪，有时数量相等。有奶油奶酪的食谱(搅打的和块状的)，酸奶的食谱，labneh 的食谱，和重奶油的食谱。自然，我都试过了。

在吃了许多不同的羊乳酪和其他乳制品的组合后，我发现在羊乳酪中加入另一种乳制品确实会让它变得更丝滑，但也会减弱羊乳酪的味道。这可能会让那些容易不知所措的人更容易接受，但作为一个喜欢羊乳酪的人，我有点被冒犯了。我希望我的搅打羊乳酪轻盈、易涂抹，而且羊乳酪向前冲。我也喜欢不用测量任何东西。

最好的搅打羊乳酪就是用食品加工机的刀片搅打的羊乳酪。起初，看起来你只是在做碎羊乳酪，但是要有耐心。奶酪将开始在碗中滚动，然后以奶油状、可涂抹的蘸酱的形式沉淀到边缘。刮一下碗的边缘，尝一尝奶酪，然后拿一些可以吃的零食。如果羊乳酪本身对你来说太难消化，你可以加入一汤匙奶油奶酪或 labneh(每六盎司羊乳酪)，或者淋上一点橄榄油，再来一次，但我鼓励你在开始调整之前尝试另一种食物。蘸有搅打过的羊乳酪的樱桃番茄是一种完美的双组分小吃；我看不出有任何理由让其他口味参与进来。
