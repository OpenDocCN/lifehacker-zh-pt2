# 当事实无法说服某人时，如何使用讲故事

> 原文:[https://life hacker . com/how-to-use-storyling-when-facts-fail-to-confident-som-1844807082](https://lifehacker.com/how-to-use-storytelling-when-facts-fail-to-convince-som-1844807082)

不是所有的争论都值得你花费时间和精力。在某些情况下——比如有人根本不同意你的观点——你说什么或做什么都无法改变他们的想法。但在其他情况下，你完全有可能提出自己的观点，让别人同意(或至少理解)你的观点。本能地，你可能会依靠事实或数据来做你的说服工作，但是统计数据很容易被操纵，而且不是每个人都对扔给他们的信息反应良好。事实证明，讲好故事可能是赢得人心的关键。以下是你需要知道的。

### 为什么要讲故事？

在最近发表在《通讯报道 杂志上的一项研究 [中，研究人员为参与者播放了一段电台采访的短片，内容是一名来自北卡罗来纳州的 66 岁男子——一名狂热的猎人和渔民——谈论他个人如何受到气候变化的影响。它包括这样的线:](https://www.tandfonline.com/doi/abs/10.1080/08934215.2020.1799049?journalCode=rcrs20)

> “鳟鱼需要冰冷、清澈、干净的水。过去我钓过鳟鱼的地方曾经有很多鱼，现在变暖了，鱼不像以前那样在那里了。这让我非常非常难过。”

听完他的故事后，被认为是保守或温和的参与者报告说，他们更关心气候变化，并且更确信这是由人类造成的。研究人员发现，那些在听采访片段时感受到担忧和同情的人最终改变了他们的想法。

这些发现与 7 月份发表在《环境政治 杂志上的一项研究相似，该研究涉及分析 1996 年至 2015 年间国会议员关于气候变化的演讲内容。事实证明，民主党人用事实和科学证据来处理这个问题，而共和党人则依靠讲故事，包括图像和情感。研究人员得出结论，共和党人正在“以最终可能更有效的方式沟通。”

这里的要点是，人们似乎对个人叙述比对统计数据更敏感。所以，下次你试图用科学或逻辑来说服某人某事时，试着把事实和数字编织成一个令人信服的故事。
