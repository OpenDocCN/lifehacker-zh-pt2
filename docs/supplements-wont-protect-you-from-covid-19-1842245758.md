# 补充剂不会保护你远离新冠肺炎

> 原文：<https://lifehacker.com/supplements-wont-protect-you-from-covid-19-1842245758>

接骨木糖浆？维生素 C？如果你正在寻找冠状病毒的额外保护，请把它们放回药店的货架上，回家洗手。是的，即使奥兹医生只是告诉你买他们。

Watch

“补充剂从未被证明能击败冠状病毒，”奥兹博士说，正确的是，《今日秀》中的。然后他又莫名其妙地继续推荐观众服用包括维生素和 [接骨木](https://lifehacker.com/will-elderberry-syrup-help-your-cold-1832119226) 糖浆在内的补充剂。

还有一封 [连锁信流传在](https://www.snopes.com/fact-check/zinc-lozenges-coronavirus/) 周围，其中包括服用锌锭以保护自己免受冠状病毒感染的建议。各种有影响力的人，有些有医学证书，有些没有，一直在推特和 Fa 上征求关于维生素或补充剂 [甚至特定饮食](https://vitals.lifehacker.com/no-the-keto-diet-wont-give-you-special-protection-from-1842057362) 的建议，好像它们可以抵御病毒。

但事实是，这些补充剂中的每一种治疗或预防任何疾病的证据都很微弱，甚至不存在。即使是维生素 C [对预防感冒也没有多大作用](https://lifehacker.com/immune-boosting-supplements-won-t-protect-you-from-ba-1804490868) ，尽管它被誉为某种神奇的免疫增强剂。

说到新冠肺炎，疗效的证据从可疑到完全没有。这种病毒仅在三个月前被发现；没有人进行过任何试验来确定补充剂是否能预防它。(目前正在进行抗病毒药物的试验，但那是另外一种情况,。)

你花在购买补品 [上的时间、精力和金钱是一种浪费](https://lifehacker.com/the-placebo-effect-is-no-reason-to-buy-junk-1774878033) 。关注那些真正有效的事情，比如 [洗手](https://vitals.lifehacker.com/how-to-actually-wash-germs-off-your-hands-1838626406) 和 [关注你所在社区的其他人](https://vitals.lifehacker.com/how-to-help-keep-people-in-your-community-safe-from-cov-1842220774) 。
