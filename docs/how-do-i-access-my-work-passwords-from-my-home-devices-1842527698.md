在今天的文章中，我将回答来自 Lifehacker 读者 Daisy 的一个有趣的电子邮件问题。我怀疑你们中的一些人目前也在经历这个问题。

### 问题是:

> *“有没有办法把我 Chrome 保存的密码(工作)移植到我的 iPhone/iPad(家用)上？手动重新输入每一条简直要了我的命。谢谢！”*

### 最重要的答案是:

哦是的。让我们现在就解决这个问题，因为没有理由让你不得不忍受不止一次地输入又长又复杂的安全密码。无论你做什么，不要以此为借口把你又长又复杂的安全密码改成更简单的。不，不，不:方便是安全的敌人。

这个问题的超级简单的答案是，你只需要在你的 iPhone 和 iPad 上安装 Chrome，在你的工作系统上用你用来访问浏览器的账号登录，同步你的密码。就这么简单。

稍微复杂一点的是，你可能不想让你在 iPhone 或 iPad 上用 Chrome 做的任何事情同步到你的工作账户。不过这没什么——你可以在你的设备上同步多个账户，尽管在 iOS 和 iPadOS 上切换账户不像在 Android 上那么容易。您必须注销一个帐户，并在登录另一个帐户之前暂时“关闭同步”。老实说，这个过程很痛苦，不像在你的桌面上设置和切换多个 Chrome 档案/谷歌账户那么简单。

相反，我可能会建议你在 Chrome 登录你的工作账户时，调出你的 [密码管理器](http://passwords.google.com) 。单击齿轮图标，然后单击导出密码，这会将它们以. CSV 文件的形式全部转储到您的桌面上。

从那里，注销你的工作账户，在 Chrome 中登录你的个人账户——在你的台式机或笔记本电脑上——并在地址栏中输入以下内容:`chrome://flags/#PasswordImport`

一旦按下 enter 键，您应该会看到如下所示的屏幕:

打开那个标志，重启你的浏览器，点击右上角的三点图标，选择设置，调出你的 Chrome 设置。点击“自动填充”部分下的“密码”，然后点击“保存的密码”旁边的三点，弹出一个新的“导入”选项，如下所示:

然后，您就可以加载。CSV 文件，包含您之前保存在 Chrome 实例中的所有工作密码，您已将其绑定到您的个人帐户。现在，点击左上角的“你和谷歌”并选择“管理同步”，以确认你的密码将同步到你使用个人帐户登录 Chrome 的任何地方。

你现在应该可以在你的“家庭”版 Chrome 上调出你的工作密码了——也就是说，在你的 iPhone 或 iPad 上——只要后者的账户也从你的 Chrome 账户同步密码(你可以在每台设备上的 Chrome 应用程序的设置菜单中查看)。

老实说，你可能只想硬着头皮把你所有的密码都转移到 [一个可靠的密码管理器](https://lifehacker.com/the-beginners-guide-to-1password-1794464866) 。这样，无论您在任何设备上使用什么帐户，您都可以访问它们。这个过程将是完全不可知的，只要你安装了密码管理器的应用程序，你就能得到你的密码。就这么简单；不要再为 Chrome 账户和其他东西大惊小怪了。

至少我会这么做。事实上，我现在就是这么做的，来管理我在 Lifehacker 上使用的密码和我在其他任何事情上使用的密码:我将它们全部转储到 1Password 中，并让应用程序管理我的登录，不管我使用的是什么设备或浏览器实例。我强烈推荐；一个可靠的密码管理器值得你每年花 35 美元来使用它吗？这还不到每月一杯 bougie 咖啡价格的一半，而且这种快乐会持续更长时间。
