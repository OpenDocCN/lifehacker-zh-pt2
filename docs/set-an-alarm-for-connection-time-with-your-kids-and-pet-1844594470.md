# 为与孩子和宠物的“交流时间”设置闹钟

> 原文：<https://lifehacker.com/set-an-alarm-for-connection-time-with-your-kids-and-pet-1844594470>

仅仅因为你现在一直在家和你的伴侣、孩子或宠物在一起，并不意味着你在花时间和对方在一起方面做得更好。事实上，相反的情况可能是真的；你可能会觉得现在有一点点*太多*的时间在一起。但是数量并不等同于质量，由于每个人都躲在家里各自的角落里(或者经常到处乱跑)，留出一些专门的时间来联系——通过设置闹钟——可能是个好主意。



可能你已经在家里建立了一个很好的与他人和动物联系的系统。也许你每晚都有家庭聚餐，每周和你的伴侣一起玩拼字游戏，你遛狗的频率增加了一倍。在这种情况下，你可能不需要这种技巧(除非你想给与你所爱的人的联系增加更多的结构)。

然而，如果你的猫因为你的漠不关心而生气，几个星期以来一直忽视你，而且这些天你对你的伴侣说的每一句话似乎都带有某种语气，你可能想试一试。

这很简单。决定你想对谁倾注一点额外的关注，并弄清楚在你的时间表中你可以把它放在哪里。也许你所需要的只是重新评估并在精神上重新投入更多的优质时间。但是如果你知道你会在一个大项目开始的时候放弃这个承诺，那就在你的手机上设置一个闹钟。

闹钟提醒你这件事对你来说很重要，足以打断你的一天(在你最方便的时候)。如果你想确保它在可控范围内，那就在 10、15、或 20 分钟后再设一个闹钟，提醒你该回去工作或开始做饭了。重点不是在你的待办事项清单上增加更多，而是让你一天中的一小部分更有目的，这样你就能从你的投资中获得最大的情感回报。

闹钟可以提醒你去和孩子们玩一个快速的午间游戏——去钓鱼，或者让狗在外面跑来跑去玩捡东西，或者和你的伴侣坐下来一起吃午饭。希望你只需要闹钟提醒一周左右，直到这些小联系成为你一天中的例行公事。
