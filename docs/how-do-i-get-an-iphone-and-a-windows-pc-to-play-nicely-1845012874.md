### 如何让 Windows PC 和 iPhone 和平共处

好主意，戴夫！所以，这里有一个隐藏的秘密:很久以前——我甚至不能告诉你那是什么版本的 iPhone 或 iOS 我虔诚地在我的 Windows PC 上使用 iTunes。这是我的主要音乐播放器，这意味着这个*肯定是在 Spotify 之前的*,我的决定可能是我大学时代的一种坚持，那时 iTunes 是从你的宿舍朋友那里听(或偷)音乐的最简单的方式。

我不知道我什么时候离开了 iTunes，但它发生了...在某个时候。我真的不记得什么时候停止在我的 Windows 系统上安装 iTunes 了。大概也是在我停止越狱 iPhone 的时候(还记得越狱吗？).

关键是，我已经不知道多久没用 iTunes 了。这有一个很好的理由:我真的不在手机上看电影，因为我认为在小屏幕上看电影毫无意义，我通过 Spotify 播放我所有的音乐。即使情况相反，我可能会直接从 iTunes 商店下载我购买的电影，并订阅 Apple Music(或者下载我以前通过同名应用购买的专辑)。

至于 iPhone 的备份，我*应该*使用 [iTunes](https://lifehacker.com/if-youre-an-iphone-loving-windows-user-youll-want-itun-1825606649) 在我的电脑上进行备份，因为你永远不会有太多的备份。我倾向于不这样做有几个原因:除了备份之外，我从来不需要把我的 iPhone 插到我的电脑上，所以我很健忘。除此之外，我还经常摆弄我的电脑(包括比大多数人更频繁地重新安装 Windows ),以至于如果我的 iPhone 出了问题，我都不想依赖我桌面上的手机备份。

最后，也是最重要的一点，通过 iCloud 自动备份手机里的 [简直简单得离谱。是的，它占用空间——每个月花掉我宝贵的 3 美元，苹果——但是它就像任何好的备份一样，眼不见，心不烦。](https://lifehacker.com/delete-old-iphone-icloud-backups-to-free-up-space-1720384238) 

因此，出于所有这些原因，我并不真正在 Windows 上使用 iTunes。现在都没装。我就是这么不在乎。如果我需要音乐、电影、电视节目或应用程序，我会直接从苹果手机上下载。我可能只会安装 iTunes 来删除手机上新的(自己创建的) [铃声](https://lifehacker.com/how-to-get-the-iphone-xs-exclusive-new-ringtone-on-your-1820080432) ，我已经有一段时间不需要这么做了。

你可能想知道我是如何把 iPhone 上的各种图片和视频传到 Windows 电脑上的。这是一个很好的问题，也很容易解决。我*可以*用 iCloud 同步所有文件并下载，但苹果的界面很烦人(似乎总是想让我双重验证是我登录了我的账户)。

相反，我只是将我的照片和电影备份到[【Google 相册】](https://photos.google.com/) 。它速度快，免费，而且很容易从我使用的任何台式机或笔记本电脑上访问。如果我因为任何原因需要在桌面上下载照片，我会批量下载；否则，我就让他们坐在云端，使用谷歌优于苹果的搜索功能来找到我需要的任何东西。

就是这样！虽然我希望有一个神奇的子弹，让我可以像苹果粉丝一样在 macOS 上通过 Windows 与我的朋友交谈，但我不会屏住呼吸等待这一天的到来。老实说，这是我作为一个 Windows 用户最大的烦恼:我不想一直拿起手机回复短信。谢天谢地，我的大多数朋友都在 Facebook Messenger 上聊天，所以至少我不会一直在我的桌面和 iPhone 之间跳来跳去。

我在想我是否遗漏了什么。一般来说，如果我因为任何原因不得不把文件放到我的 iPhone 上，我会使用让我通过云来做这件事的应用程序。因此，如果我需要带一堆 PDF 到某个地方，我只需将 PDF Expert 与 Google Drive 文件夹同步，从我的桌面上传我的 PDF，然后就可以收工了。就像我说的，我基本上不会因为任何原因将我的 iPhone 直接连接到我的 Windows 桌面或笔记本电脑，因为我觉得在 2020 年我们的云时代，你不会真的有 T1。
