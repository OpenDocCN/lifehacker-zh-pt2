# 做一下这个测试，学习一下旅行时如何给小费

> 原文：<https://lifehacker.com/take-this-quiz-to-learn-how-to-tip-when-you-travel-1837507692>

当我们在餐馆、咖啡店或发廊时，许多人都知道如何给小费。我们旅行的时候呢？除了出租车和拼车司机，还有谁能得到小费？我们是否在不经意间忽略了让我们的旅行平稳舒适的服务人员？

Watch

《华盛顿邮报》 有一个测试可以帮助你判断你是一个精明的旅行小费者还是一个刻板的人。这里有一个示例问题:

> 你从租车场乘坐免费班车到终点站，司机把你的包从车上卸下来放在路边。你给小费吗？

如果你好奇的话，我答错了——但我确实正确回答了关于给酒店客房服务员小费的问题！

关于小费，我们很多人都依赖外部提醒来知道什么时候给小费以及给多少:我们把零钱放在小费罐里，填写收据底部的行，或者选择应用程序中出现的选项之一。当你旅行的时候，很多提醒都消失了；免费班车上没有触摸屏，提示你给司机小费。(至少，现在还没有。)

所以像这样的测验可以帮助你决定下次旅行时什么时候该给小费，以及你需要带多少额外的现金。
