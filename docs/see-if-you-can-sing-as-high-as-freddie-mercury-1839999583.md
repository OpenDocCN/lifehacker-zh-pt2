# 看看你能否唱得和佛莱迪·摩克瑞一样高

> 原文：<https://lifehacker.com/see-if-you-can-sing-as-high-as-freddie-mercury-1839999583>

我以为办公室闹鬼。透过通风口传来的猫或婴儿刺耳的哀鸣声。窗外建筑的机器？天啊，可能是天花板上一只垂死的老鼠。我带来了我们的副主编爱丽丝和我一起听。“我听说了！”她说。她回到她的办公室；我在办公室的各个房间里蹑手蹑脚地走动。问我们的创意制作人希瑟是否也听到了。她笑了。昨天我们都在玩 [打高音](https://vole.wtf/high-notes/) ，这个游戏看你能唱多高。她试图战胜它。

Watch

点击高音为你播放一种音调，并说出能唱出这种音调的著名歌手的名字。然后你试着唱出来。在我的真实声音中，我敲击得相当快，只比弗兰克·辛纳屈快几个音。用假声我可以走得更远，直到我像卑鄙的比约克印象一样刺耳尖叫。我能和真正的高音歌手匹敌的唯一方法就是靠作弊，即使那样我也达不到顶峰。惠特尼·休斯顿能唱得比我吹口哨还高。

这个网络玩具来自于 [VOLE.wtf](https://vole.wtf/) ，一个特别圆滑和有创意的消磨时间的网站。当你弄断了你的喉咙后，试试 [把足球踢回给一些残忍的英国孩子](https://vole.wtf/high-notes/) ，这是一个非常令人紧张的游戏。或者，对于一个不太像听力测试而更像是千篇一律的卡拉 ok 之夜的唱歌游戏，试试 YouTube Music 的，试着像佛莱迪·摩克瑞那样唱歌。
