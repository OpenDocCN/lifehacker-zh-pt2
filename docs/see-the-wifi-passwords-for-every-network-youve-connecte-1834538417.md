# 查看你使用此 Android 应用连接到的每个网络的 WiFi 密码

> 原文：<https://lifehacker.com/see-the-wifi-passwords-for-every-network-youve-connecte-1834538417>

我比大多数人更倾向于更换电子产品，这意味着当我带着笔记本电脑或平板电脑去拜访朋友时，很可能不是我上次去的那台笔记本电脑或平板电脑。



这是一种职业风险，也让那些不得不第 9000 次告诉我密码的朋友们感到烦恼，很可能是在深夜，当我意识到我没有密码的时候。是的，根据你手头的小工具，有很多解决这个问题的方法，但由于小工具黑客 ，我最近遇到了一个很棒的 [:使用 WiFi 密钥恢复应用程序在我的手机中找到密码。](https://android.gadgethacks.com/how-to/see-passwords-for-wi-fi-networks-youve-connected-your-android-device-0160995/)

如果我的手机或 Android 平板电脑之前已经连接到网络，那么我可以很容易地自己查找密码，这样我就可以使用该应用程序连接其他设备。Android 设备在默认情况下不会这样做，但该应用程序是一个简单的解决方法。

使用时， [从谷歌 Play 商店](https://play.google.com/store/apps/details?id=aws.apps.wifiKeyRecovery) 下载应用程序(您的手机 [需要 root](https://lifehacker.com/tag/android-rooting?_ga=2.240641027.1262130619.1557137664-870847336.1557137664))并运行应用程序。你必须给予该应用程序超级用户访问权限，但当你这样做时，你将能够看到你的设备曾经连接过的所有 WiFi 网络，以及这些网络的密码是什么。密码是可搜索的，您可以使用附加应用程序通过 QR 码与另一台设备共享密码。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-Pb0NGCKFa_M&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-Pb0NGCKFa_M&start=0) 

该应用程序还提供了通过文本文件存储整个列表的选项。我个人对无限期授予应用程序超级用户权限有点担心。如果你也是，我建议下载列表，然后卸载应用程序或限制其权限。如果你需要一个流氓密码，你可以随时重新启用它，但当你不需要的时候，你没有给应用程序完全的控制权。
