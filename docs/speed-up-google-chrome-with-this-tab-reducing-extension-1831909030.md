# 使用这个减少标签的扩展来加速谷歌浏览器

> 原文：<https://lifehacker.com/speed-up-google-chrome-with-this-tab-reducing-extension-1831909030>

我现在在电脑上打开了大约 70 个标签，误差不大。我把 Chrome 标签当做临时书签。例如，我打开的几个标签页是我以后想读的故事，还有几个是我目前正在计划的旧金山啤酒周活动。这不是最有效的工作方式，但这是我选择的方式。不幸的是，这也是一种消耗大量能量的方式。



我的标签习惯已经不止一次在我的 MacBook 上成为一个问题，原因很简单，就是让所有的标签都开着，尤其是使用谷歌 Chrome，会占用大量的电脑内存。

[一个标签](https://chrome.google.com/webstore/detail/onetab/chphlpgkkbolifaimnlloiipkdnihall) 是一个 Chrome 扩展，它允许你本质上保持所有标签打开，而不是实际上保持所有标签打开。火狐 也有 [。](https://addons.mozilla.org/en-US/firefox/addon/onetab/?src=search)

当你使用它时，它会将你所有打开的标签页转换成一个列表。当您需要再次访问它们时，您可以恢复您需要的单个选项卡，或者一次恢复所有选项卡。该扩展的创建者说，使用它可以节省高达 95%的电脑内存。当然，这取决于你现在经常打开多少个标签页，但不管怎样，应该会节省一些内存。

如果你愿意，标签列表也可以转换成网页，这样你就可以和朋友分享了。这意味着，如果我愿意的话，只要创建一个链接网页，我就可以将我现在开设的十几个啤酒活动发送给朋友。或者我可以直接把研究发给我自己，然后永久关闭所有的标签页。

虽然可共享功能很酷，但真正的好处是节省了内存。如果你像我一样经常打开，这可能意味着拥有一台功能正常的计算机和经常看着死亡的旋转车轮之间的区别。
