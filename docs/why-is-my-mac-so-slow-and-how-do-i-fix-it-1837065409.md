# 为什么我的 Mac 很慢，我该如何解决？

> 原文：<https://lifehacker.com/why-is-my-mac-so-slow-and-how-do-i-fix-it-1837065409>

在本周的 [**技术建议专栏**](https://lifehacker.com/c/tech-911) 中，我们正在解决一个最古老也是最常见的问题——我们在网上看到了这个问题，它一次又一次地被朋友和爱人纠缠着:“我的电脑很慢，我不知道如何让它再快一点。”

Watch

好吧，为了弗吉尼亚的 的荣誉，当她在 Lifehacker Slack 中 ping 我以获得我对她慢得像糖蜜一样的 Mac 的意见时，她确实说得更有说服力了。她是这样说的:

> “我用了 5 年的苹果电脑有时速度太慢，几乎无法运行。很明显，我的系统被垃圾应用程序弄得乱七八糟，但我不知道从哪里开始。我该怎么办？”

事实上，我们前几天刚刚在 Lifehacker Slack 上进行了一场类似的辩论——是否那些“清理你的系统，让它更快！”你在网上看到的广告应用是值得安装的(有时)，或者完全是伪造的(通常)。

作为一个 Windows 爱好者，我的脑海中立即浮现出“注册表清理器”和“内存优化器”，它们向你承诺，如果你只需一次性支付 50 美元的费用，就能获得令人难以置信的性能提升，因为这是一款改变你生活的软件，而事实证明它只不过是垃圾科学。(如果你幸运的话，那是；他们运行的“清理”实际上可以 [搞乱你的 Windows 注册表](https://support.microsoft.com/en-us/help/2563254/microsoft-support-policy-for-the-use-of-registry-cleaning-utilities) ，给你带来比你之前经历的更大的问题。)

我对推荐任何一种承诺能让你的电脑回到曾经完美状态的应用感到紧张。对我来说，你只需要考虑两种清理工具:一种是 [帮助你找到并删除你已经忘记的旧文件](https://lifehacker.com/find-out-whats-taking-up-space-on-your-windows-computer-1826717296) (或者不管什么原因仍然存在于你系统中的巨大临时文件)，还有常识。

在 Mac 上， [**磁盘清单 X**](http://www.derlien.com) 是基于 Windows 的 WinDirStat 或 WizTree 的一个很好的替代品，可以帮助你找出哪些文件夹和文件占用了你系统上的大部分空间。在我的情况下，类似这样的应用程序帮助我意识到，我在某个时候不小心制作了一个文件夹的几千兆字节文件的额外副本——我立即删除了它，以释放我的主 SSD 上的一些空间。

至于“常识”这一点，我主要指的是花时间去了解你的系统和你的系统上有什么。我的意思是。

### 也许是时候升级你 Mac 的硬件了(如果你可以的话)

我是一个硬件爱好者，所以当我面对一台运行缓慢的 Mac 电脑时，我首先会考虑的是，无论我想用它做什么，我的系统是否动力不足。我首先从记忆开始。通常，8GB 内存是一个很好的最低目标。除非您在系统上运行大量要求苛刻的应用程序，否则您可能不需要更多。

您可以通过 Mac 的“系统信息”屏幕查看您有多少。寻找第一页上的“内存”行，这应该很明显:

当您在这里时，记下您的 Mac 的型号标识符、处理器名称和处理器速度。如果您的内存不足 8GB，请使用此信息进行快速网络搜索，看看您是否可以将 Mac 升级到更大的内存。(在一些 Mac 机型上，苹果将组件焊接到主板上，使它们更难拆卸和更换。)

也有很多网站可以提供帮助，比如 OWC 的便捷的“”工具。如果您看到列出的升级类型—如内存—那么您可以合理地升级它，不会有太大问题。

除此之外，如果你的 Mac 已经老到可以摇动硬盘而不是固态硬盘，这种升级是显而易见的。如果你从旋转盘片转移到小芯片，你绝对会看到性能的提升，这是任何人都可以为任何台式机或笔记本电脑(如果允许的话)进行的一次伟大的升级。

您的 Mac 也有可能存在某种散热问题。确保系统的所有通风口都没有粘性物质、灰尘和碎片。如果你想变得狂野，你可以 [打开你的系统](https://www.youtube.com/watch?v=vOJjAho26JA) 并尝试清洁任何内部风扇(或重新使用热糊)，但这可能超出了大多数人的能力。

### macOS 本身呢？

如果你的 Mac 硬件不错，但你的系统仍然感觉迟钝，你有几个选择。首先，也是最明显的，是这个问题:你运行的是最新版本的 macOS 吗？升级到 [macOS Mojave](https://lifehacker.com/what-you-need-to-know-before-and-after-installing-mac-1829271754) (或者，很快，卡特琳娜) [可能会给你一个性能提升](https://lifehacker.com/how-is-your-macos-mojave-experience-on-your-older-mac-1829333329) 。

接下来，点击**系统偏好** > **用户&群组**并点击“登录项目”标签，来看看 macOS 上运行的应用程序。这里不应该有大量的应用程序；如果有太多，修剪列表可以帮助您释放一些系统资源。

您也可以尝试扫描 Mac 的驱动器，查看是否有任何可能导致速度变慢的异常情况。打开**磁盘工具**应用程序，在你的硬盘上运行“急救”选项，它会检查错误并修复发现的任何问题。

虽然这在您第一次启动系统时可能对您没有帮助，但是您也可以启动**活动监视器**来查看是否有任何进程占用了您的处理器(按“% CPU”排序)或内存(按 memory 选项卡上的“Memory”排序)。如果有些东西看起来不合适——可能是云同步应用占用了整整 1gb 的内存，使用了 75%的处理器，或者做了其他看起来很奇怪的事情——更新或卸载/重新安装该应用可能会解决你的问题。

如果问题仍然没有被发现，你最好的办法就是从头开始重新安装 macOS。这听起来工作量很大，但实际上并不难做到——至少，您可能会节省时间，并实际提升 Mac 的性能，而不是尝试其他模糊的故障诊断步骤来尝试和纠正您尚未发现的问题。

在你开始之前，确保你已经通过 [时间机器](https://support.apple.com/en-us/HT201250) 、 [iCloud](https://support.apple.com/en-us/HT204025) ，或者任何其他 [存储服务](https://lifehacker.com/google-one-is-now-open-for-everyone-but-is-it-a-good-d-1826049257) 或者你使用的备份技术备份了你 Mac 的关键文件。因为我们正在重新安装 macOS，你 [不需要担心](https://lifehacker.com/all-the-folders-you-can-probably-exclude-from-your-macs-1792939535) 保存你整个硬盘的内容。只保存您无法重新安装的数据，如您尚未保存到云中的文档或媒体等。

如果你健忘，别忘了拍摄并保存 Mac 上已安装应用程序列表的截图，这样你就可以轻松地重新安装它们，让你的系统恢复原样。同样的道理也适用于任何你不希望以后必须查找的超关键设置，比如你公司的 VPN(或诸如此类的东西)的登录细节。

当你准备好重新安装 macOS 时，重启你的系统并按住 **Command+R** 键。这将引导您的 Mac 进入恢复模式，并允许您安装先前安装在 Mac 上的最新操作系统。(如果你想升级到你*还没有*安装的最新版本的 macOS，按住 **Option+Command+R** 即可。)

在那里，选择“重新安装 macOS”选项，并按照所有提示进行操作。很简单。一旦完成，你将经历 macOS 的简短的初始设置屏幕，然后你可以开始 [重新安装你所有的应用](https://lifehacker.com/how-to-set-up-your-macbook-from-scratch-1825920271) 并重新连接你所有的账户。

希望在经历了这一切之后，你的 Mac 能感觉好一点。如果没有，硬件升级将是你最好的选择——或者可能是一台全新的 Mac。
