# 如何处理你现在拿到的所有税务文件

> 原文：<https://lifehacker.com/what-to-do-with-all-the-tax-documents-youre-getting-rig-1831988627>

到目前为止，你已经收到了附有“重要税务信息”和年终投资报表的电子邮件和信件，并且可能还会收到更多。你知道你需要这些来提交 2018 年的纳税申报表，但在那之后，你会怎么处理它们？



在[humbled dollar](https://humbledollar.com/2019/01/paper-chase/)结束时，Ross Menke 根据文件的重要性及其法律要求，为个人和财务文件建议了一个三“桶”系统:

### **永远的文件**

这个桶是用来存放所有需要花很大力气才能替换的文档的。，包括您的(以及您家庭成员的):

*   保障
*   出生证明
*   社会保障卡
*   房契
*   汽车标题
*   结婚证书
*   威尔斯
*   委任书
*   捐税收入
*   人寿保险政策
*   医药费收据如果你有 HSA(因为，如我昨天写的，你可以在以后的年份里报销医药费)

另一个项目应该在这里，根据门克:罗斯个人退休帐户的贡献历史。“你每年存入罗斯个人退休帐户的钱可以随时提取，而且免税免罚，”门克写道。“你想记录自己贡献了多少，因为 20 年后可能很难记起。”

### 支持税务材料

这是你现在收到的邮件。保留任何税务相关文件和其他信息三到七年，[IRS 建议](https://www.irs.gov/businesses/small-businesses-self-employed/how-long-should-i-keep-records) ，包括:

*   W-2s
*   1099s
*   证明 [你主张的任何扣款](https://twocents.lifehacker.com/all-the-tax-deductions-you-can-take-for-2018-1831808127)

如果你没有归档，就永远保存它们(尽管你可能会遇到比丢失文件更大的问题)。

### 一年期文件

门克说，这个桶是为你的其他记录保留的:

*   每月银行对账单
*   信用卡对账单
*   账单
*   工资单

真的没有理由保存这些超过一年。现在，你把这些文件存放在哪里？我们这里有 [一些建议](https://twocents.lifehacker.com/how-to-organize-your-financial-documents-1831339097) 。门克建议，另一个主意，尤其是对那些终身博士来说，是买一个保险箱或防火保险箱。
