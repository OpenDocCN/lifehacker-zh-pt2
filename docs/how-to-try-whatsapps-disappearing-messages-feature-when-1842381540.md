# 当 WhatsApp 的“消失信息”功能终于到来时，该如何尝试

> 原文：<https://lifehacker.com/how-to-try-whatsapps-disappearing-messages-feature-when-1842381540>

对 WhatsApp 粉丝来说，有好消息也有坏消息。我们先从前者开始:这款应用很可能很快就会有一个奇特的小功能，让你在一段时间后自动删除群发和私人信息。有趣的是:*你*决定这段时间有多长。

Watch

嗯，是这样的。根据 [WABetaInfo](https://wabetainfo.com/whatsapp-beta-for-android-2-20-83-and-2-20-84-whats-new/) 的截图，你可以设置你的信息在一个小时、一天、一周、一个月或者一年后消失。虽然我想对 24 小时内的时间进行更多的粒度分析，以防你希望你的消息在几个小时后自动删除，这样人们会看到它，但它不会永远存在，这个功能的存在仍然比没有好。

关于那个。据 WABetaInfo 报道，WhatsApp 目前正在测试自动删除功能。因此，如果你使用的是普通版本，你很可能在应用程序中找不到这个功能。如果你有幸成为 [测试者](https://play.google.com/apps/testing/com.whatsapp) ，你*可能*看到它，但你也可能看不到；我从 [APKMirror](https://www.apkmirror.com/apk/whatsapp-inc/) 中抓取了据称具有该功能的应用程序的测试版(2.20.84)，但仍然没有结果。

因此，虽然您不得不等待这很烦人，但这个特性很快就会为每个人带来。当它命中时，访问它是容易的。调出一个 WhatsApp 群聊或私信，然后点击群名打开它的选项。在媒体可见后，你会看到一个新的“删除消息”选项，如 WABetaInfo 的截图所示:

启用后，所有消息的时间旁边都会有一个小时钟图标。这意味着这条信息，像生活中的所有事物一样，最终会永远消失。
