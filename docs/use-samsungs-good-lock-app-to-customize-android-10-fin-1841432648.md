# 用三星的‘好锁’App 定制 Android 10，最后

> 原文：<https://lifehacker.com/use-samsungs-good-lock-app-to-customize-android-10-fin-1841432648>

通过 One UI 2.0 更新，Android 10 终于可以在几款最新的三星智能手机上使用，但正如许多三星用户发现的那样，最新版本的 Android 在发布时不支持 One UI 的 Good Lock 定制应用。幸运的是，该应用程序最近更新了对 Android 10 和 One UI 2.0 的支持，你现在可以从 [Galaxy Store](https://galaxystore.samsung.com/detail/com.samsung.android.goodlock) 下载新版本。

Watch

如果你以前从未使用过 Good Lock，它是一款三星专属的应用程序，包含各种模块和小部件，可以让你自定义用户界面，个性化手势控制，并为你的手机添加通常无法通过谷歌 Play 商店应用程序获得的新功能。一些亮点包括:

*   **Bixby 例程:**让你用 Bixby 助手创建自动系统例程。
*   **ClockFace:** 包括几个时钟面，可以应用到你的锁屏和其他时钟小工具上。
*   **单手操作+:** 允许您修改和创建新的手势控制。
*   **QuickStar:** 允许您更改状态栏、通知菜单和通知行为的外观和布局。
*   **声音助手:**让您控制音量、均衡器平衡、蓝牙设置和其他单个应用程序的音频选项。
*   **任务改变器:**个性化您最近的应用程序菜单。
*   **主题公园:**制作自己的 UI 主题。

还有许多其他模块可供您控制一切，从曲面屏设备上的屏幕边界到定制免提汽车模式等等。请注意，一些模块尚未从 Android 10 和 One UI 2.0 中更新——例如 Lock Star，它可以让你定制锁定屏幕的外观和布局——所以不是所有的*模块目前都可以工作，尽管你现在仍然可以获取它们，希望它们很快会更新。*

### 如何使用好的锁定模块来定制您的三星手机

当你直接从 Galaxy Store 下载 Good Lock 应用程序时，会包含完整的 Good Lock 模块包，但如果你下载了该应用程序或者你居住的地区没有完整的包，也可以从[【APK 镜报】](https://www.apkmirror.com/uploads/?q=samsung-electronics-co-ltd) 零零碎碎地获得它们。对于那些不能一次获得完整软件包但不想逐个安装模块 apk 的人，XDA 开发者论坛用户 Xantrk 有一个名为 NiceLock 的 [专门启动程序，它会为你下载并安装所有模块 apk。](https://forum.xda-developers.com/galaxy-note-9/themes/nicelock-module-download-links-module-t3923844) 

要使用其中一个模块，首先下载并安装好锁是你还没有，然后启动应用程序，找到并使用你已经下载的所有模块的列表。点击您想要使用的模块，然后根据需要调整设置。
