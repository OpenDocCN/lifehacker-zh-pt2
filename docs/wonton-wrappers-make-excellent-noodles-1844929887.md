# 馄饨皮是极好的面条

> 原文:[https://life hacker . com/wonton-wrappers-make-excellent-noodles-1844929887](https://lifehacker.com/wonton-wrappers-make-excellent-noodles-1844929887)

你可能知道，馄饨包装纸是薄而精致的面粉、水和鸡蛋，用于制作顶级食品，如油炸馄饨、馄饨汤和仰光螃蟹(仅举几个例子)。但是这些小方块可以很容易地放入煨汤、酱汁或普通的水中，做成一堆堆美味有弹性的小面条。

Watch

事实上，用馄饨皮做面条可能是你能用它们做的最简单的事情，特别是如果你刚刚学会如何正确地包馄饨或者正在完善你的包法，并且容易撕坏它们。不小心撕开的面皮可以做成很棒的面条，但是你也可以故意撕开它们——半个面皮就可以做成一根大小完美的面条。

包装纸煮得很快，很像新鲜的意大利面(因为它们基本上是新鲜鸡蛋意大利面的薄片)，所以在你把它们放进沸腾的液体之前，确保你已经准备好吃它们了。如果你打算在喝汤的时候吃它们，只需在你打算端上汤之前一分钟把它们放进去。(它们会浮在上面，煮熟后会变得有点透明，但在烹饪一分钟后尝一尝，看看它在你牙齿之间的感觉如何。)如果你想做冷面碗，先把它们煮两分钟，然后沥干，在冷水下冲洗，停止煮的过程。你也可以把它们放在调味汁里煮大约一分钟，或者 [把它们和黄油](https://lifehacker.com/use-broken-lasagna-noodles-in-your-stews-1839135260) 一起端上桌，和丰盛的炖肉一起享用(炖肉季节即将来临)。
