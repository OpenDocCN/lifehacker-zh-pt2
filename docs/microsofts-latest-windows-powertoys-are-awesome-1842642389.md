# 微软最新的 Windows PowerToys 太棒了

> 原文：<https://lifehacker.com/microsofts-latest-windows-powertoys-are-awesome-1842642389>

微软忙碌的开发人员已经逐渐发布了有趣的新工具供 Windows 用户使用。这些 [PowerToys](https://github.com/microsoft/PowerToys/) 往往会给你的操作系统增加一点古怪但实用的功能，而且你完全可以免费试用。我们已经介绍了他们以前的[产品](https://lifehacker.com/bookmark-this-site-to-try-microsofts-new-windows-powert-1834749716) 中的一些，但是最新增加的两个是最好的。



抓取最新的 [PowerToys 安装程序](https://github.com/microsoft/PowerToys/releases/download/v0.16.0/PowerToysSetup-0.16.0-x64.msi) ，一起来认识一下**图像缩放器**和**窗口行者**。

### Image Resizer

默认情况下，当你安装该应用程序时，微软 PowerToys 中的所有各种实用程序都是启用的；如果有一些你不需要或不想要的，你可以在应用程序设置中快速调整。我只是想摆脱这一切。

这意味着 PowerToys 安装完成后，您就可以开始使用 Image Resizer 了。只需打开文件资源管理器，找到一个(或多个)你想调整大小的图像，然后右击它。你可以选择一个新的“调整图片大小选项”

当你这样做时，你会得到这个方便的小弹出窗口:

最有趣的选项是“自定义”预设，它允许您选择“适合”、“填充”或“拉伸”图像，以便在您设置的像素参数内工作。我从来不会使用“拉伸”，但是如果你需要自动裁剪的东西，“填充”是有用的，“适合”，如果我是正确的，意味着你的结果图像将被调整大小，使其至少满足两个值中的最低值。

你甚至可以允许图像调整程序覆盖原始图像——显然，除非你确定你在做什么，否则我不建议你点击这个选项。你也可以考虑点击左下角的“设置”链接，这可以让你为你的编码图像和新图像的文件名指定自定义大小，首选文件类型和质量设置。强大的东西！

### 窗户巡视员

不，这不是塞尔达传说游戏。这个功能允许你搜索所有打开的应用程序，而不是通过传统的 ALT+Tab 显示来浏览各个窗口。它比简单的 ALT+tab 浏览你的窗口更有用吗？如果你有两个开着的窗户，没有。如果你有二十个，有。(你有二十个，是吗？)

要调出窗口漫游器，您需要使用不同的键盘快捷键:Windows 键+ CTRL。开始键入任何打开的应用程序的名称，你将能够使用键盘的箭头键在潜在候选项之间上下移动。这样做将调出每个应用程序的实时视图，使你很容易找到你要找的应用程序。

就是这样！最终，这项功能将被纳入微软正在开发的一个新的“启动器”项目——理论上，它将取代你的标准 Windows Key+R 快捷键(弹出很少使用的“运行”框)。
