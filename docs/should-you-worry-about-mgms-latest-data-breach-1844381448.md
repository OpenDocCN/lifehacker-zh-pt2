# 你应该担心米高梅最近的数据泄露吗？

> 原文:[https://life hacker . com/should-you-worry-mgms-latest-data-breach-1844381448](https://lifehacker.com/should-you-worry-about-mgms-latest-data-breach-1844381448)

一个包含超过 1 . 42 亿米高梅度假村客人的个人信息的数据库在网上出售给 T2。这些数据在暗网上以 2900 美元的价格出售，包括敏感的客人信息，如姓名、电话号码等等——尽管似乎不包括财务信息。

目前还不清楚这个信息是否包括 2019 年类似违规事件中曝光的 1060 万嘉宾的相同数据，还是全部为新信息。也不确定数据库最初是如何获得的。一个俄罗斯黑客网站上的帖子声称，这些信息是从 Night Lion security 的 DataViper 中窃取的——这是一种索引泄露数据的服务，去年 [也遭到了自己的入侵](https://krebsonsecurity.com/2020/07/breached-data-indexer-data-viper-hacked/)——但 Night Lion 创始人 Vinny Troia [告诉 ZDNet](https://www.zdnet.com/article/hacker-breaches-security-firm-in-act-of-revenge/) 该公司的数据库中根本没有米高梅度假村的信息。另一种可能性是，这些信息也是在去年米高梅云服务器最初的漏洞中获得的。

无论如何，如果你去过米高梅酒店，你可能会担心这个大数据泄露对你意味着什么。

### 你应该担心吗？

所有的数据泄露都应该认真对待，但是这些 MGM 度假村的数据泄露虽然敏感，但如果你一开始就有强大的数据安全实践，并不一定会给你带来风险。

好消息是，似乎没有财务信息被泄露，因为你没有创建用户名或密码来住在酒店，你不需要担心有人闯入你的其他帐户与凭证填充。

也就是说，泄露的信息*包括姓名、出生日期、*电话号码、电子邮件地址和实际地址，b 但还有很多其他地方可以获得这些信息——有些信息是公开的——如果有人真的想找到的话。光是这些数据还不足以进行全面的身份盗窃，但肯定可以帮助某人的企图。

这就是为什么保持你的密码的唯一性和没有任何个人信息是很重要的。和往常一样， [限制你自愿添加到你的各种账户和社交媒体档案中的个人信息](https://lifehacker.com/how-to-commit-internet-suicide-and-disappear-from-the-w-5958801) 的数量。

### 客人应该做什么

如果你住在米高梅度假村或米高梅大酒店，你需要加强你的普通账户安全。 [开启双因素认证](https://lifehacker.com/no-one-knows-about-two-factor-authentication-and-privat-1838913065) 和所有账户的最高安全设置， [更新你的密码](https://lifehacker.com/how-to-create-secure-passwords-that-arent-impossible-to-1825048324) 和 [使用密码管理器](https://lifehacker.com/the-five-best-password-managers-5529133) 如果数据中的任何内容可能会帮助他人破解你的账户，并从不需要的账户中删除任何个人信息。

为了以防万一，你还应该留意可疑银行和其他金融活动，如果发现任何情况，立即联系这些机构。这些预防措施将帮助你尽早发现未经授权的登录企图和其他活动——无论你的个人信息是否在泄露的 MGM Resorts 数据库中。
