# 为什么你应该尝试单腿硬拉

> 原文：<https://lifehacker.com/why-you-should-try-single-leg-deadlifts-1829863791>

如果你一直在为我们的硬拉挑战 而挣扎，因为你不容易进入有杠铃和重物的健身房，这是特别为你准备的。不过，我们所有人都会受益。是时候试试单腿硬拉了。

Watch

单腿平衡从两个方面增加了这项练习的难度。首先，你需要更多地使用这些小的稳定肌肉来保持直立。第二，你有一条腿在做双倍的工作，所以即使是很轻的重量也会很有挑战性。

我喜欢《断肌》 中的这个视频来演示一些不同的选项。首先，他们让你从站姿开始，然后*降低*重量，而不是从地板上的重量开始。这样更容易找到自然的动作。他们还使用两个壶铃，在你站立的腿的两侧，这样至少你的上半身可以保持平衡。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-7uA1PLNIWK8&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-7uA1PLNIWK8&start=0) 

取决于你想做什么，你可以把你的脚趾放在你站立的脚附近的地面上；把你的脚趾放在那只脚后面的地面上；或者将你的自由腿向后推，使它完全不触地。

保持你站立的腿完全伸直是很诱人的，但是我们中很少有人有这样的力量和灵活性(两者都需要！)才能稳定在那个位置。允许你的膝盖有些弯曲，试着仍然把这个动作当成硬拉，而不是直腿的 [花样滑冰](https://www.google.com/search?q=ice+skating+spiral&tbm=isch) 。
