# 是时候设定你的 2019 年金钱目标了

> 原文:[https://life hacker . com/its-time-to-set-your-2019-money-goals-1830990890](https://lifehacker.com/its-time-to-set-your-2019-money-goals-1830990890)

既然我们已经谈论了设定目标， [回顾了我们今年的表现](https://twocents.lifehacker.com/did-you-accomplish-your-2018-money-goals-1830981445) ，是时候谈谈明年了。为此，我 2019 年的金钱目标是...

Watch

[鼓声]

尽可能地省钱！耶！原创，我知道。但是，为了在接下来的几个月里真正完成这个目标，我需要从 [开始，把](https://lifehacker.com/how-can-i-turn-vague-goals-into-actionable-to-dos-5925801) 分解成我可以划掉的每周或每月的小目标，正如我在这里写的。

这包括:

*   **每月自动存入罗斯个人退休帐户**。我上周自动完成了我的投稿，所以我已经在路上了。也许你 [上周也增加了你的 401(k)或 IRA 缴款](https://twocents.lifehacker.com/increase-your-401-k-or-ira-contributions-for-2019-1830819672) ？我还在 7 月份的日历上放了一个提醒来增加我的捐款。
*   **带着午餐去上班**。这可以进一步细分为寻找简单、美味、适合旅行的食谱(你好，MTA )，或者储备便宜的外卖午餐。我的一个子目标是多抽点时间去 Trader Joe，而不是回到我家附近价格更高的杂货店。不，我不认为带午餐去 [会让你变得富有](https://twocents.lifehacker.com/its-ok-to-buy-the-damn-coffee-1825722875) ，但是我知道我经常很懒，这可能是我省钱的一个简单方法，尤其是在纽约。
*   减少饮酒和夜间外出。这个对我的健康和钱包都有好处。相反，我会试着在 [和](https://twocents.lifehacker.com/try-these-inexpensive-socializing-ideas-in-the-new-year-1830747956) 度过一些不贵的夜晚，或者专注于我自己喜欢做的活动。
*   **没有新衣服**。我的意思是，我有足够的，我会没事的。
*   用 DollarBird 追踪我的消费。我最近 [开始使用 DollarBird](https://twocents.lifehacker.com/use-dollarbird-for-easy-hands-on-budgeting-1830944057#_ga=2.233377822.1055601077.1544468859-3846207152.1521480874) ，这是我第一个真正喜欢使用的预算应用程序，因为它相对简单，低投入。
*   寻找赚更多钱的方法。没有什么比补充你的现金流更有帮助了。我正在寻找更多的方法在 2019 年赚更多的钱，并希望在新的一年里写更多的文章。

这在很大程度上可以归结为 [建立更好的日常生活](https://twocents.lifehacker.com/how-to-stop-making-excuses-and-finally-get-your-finance-1672029475#_ga=2.254730624.948678826.1544450486-594046802.1524762060) ，以一种有助于而不是阻碍成功的方式安排我的一天。当然，有一百万种方法可以做到这一点，我只选择了几种。其中最重要的是提醒自己，我是为了自由而，而不仅仅是因为 [一些金融专业人士对](https://twocents.lifehacker.com/how-much-money-you-need-to-save-by-the-time-youre-35-1826106733) 说。而且，你知道， [只是在做](https://lifehacker.com/do-one-thing-now-1830771564#_ga=2.242344202.948678826.1544450486-594046802.1524762060) ，而不是在说。

这也是认识到我不会一直坚持所有这些小目标。我(肯定)会搞砸的。但是如果总的趋势是我想去的方向，那就相当不错了。
