# 如何举办烹饪书交换派对

> 原文：<https://lifehacker.com/how-to-host-a-cookbook-swap-party-1834750636>

承认吧:你陷入了墨守成规。昨晚是玉米卷星期二，今晚是肉卷，星期四是烤鸡。又来了。

Watch

我们都经历过。连我 8 岁的孩子都厌倦了我的菜单。就在前几天，他还在说，“意大利面和肉丸*又来了*？我们每周都有这样的活动。”(对此，我非常友好和成熟地回答道，“那就开始喜欢除了意大利面、热狗和披萨之外的东西，我会改变一下，嗯？”)

不是你没有足够的烹饪书；事实上，你藏了很多。只不过你已经浏览了一百遍，你已经把你喜欢的三个食谱做了书签，剩下的只是 *meh* 而已。或者他们有你通常手头没有的成分。或者它们是一个 8 岁孩子不会吃的食谱。

解决方法:举办一次烹饪书交换派对。

食谱交换派对是一个常规读书会和一个讨厌的办公室礼物交换以最美的方式碰撞的交汇点。你不必真的去读任何东西，你可以带着有用的东西回家。

### 基本规则

以下是我写这篇文章时制定的官方食谱交换聚会规则:

**1。邀请两个或更多的人**。你至少需要三个人来完成这项工作，但是你邀请的人越多，你最终得到的选择就越多。

**2。每个人可以带尽可能多的烹饪书。无论你带多少，你都要带多少。**

**3。别带着你所有的小破车。那些你在杂货店收银台发现的季节性“书”(它们更像杂志)不算数。好吧，你可以带着它们，以防有人特别想要一大堆万圣节甜点，但它们不算在你的书里。带好书。**

**4。收藏你最喜欢的食谱。对这本书的下一个主人来说，这是一件有礼貌的事情，它会帮助你记住在你把它带回家之前制作你自己的副本。**

**5。提供小吃和饮料。因为这是个派对。**

在你的烹饪书交换聚会结束时，你将度过一个有趣的夜晚，你将会有一些新的晚餐想法。
