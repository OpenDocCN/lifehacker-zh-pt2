# 什么是剧吐？

> 原文:[https://lifehacker.com/what-is-hyperemesis-1833267505](https://lifehacker.com/what-is-hyperemesis-1833267505)

艾米·舒默告诉《纽约时报》记者 ，自从怀孕以来，她已经呕吐了 980 次。她患有妊娠剧吐，就在几年前，剑桥公爵夫人凯特·米德尔顿 [也患有这种疾病](https://www.today.com/health/prince-williams-reveals-not-much-has-helped-pregnant-duchess-kate-t116827) 。

[妊娠剧吐顾名思义就是](https://americanpregnancy.org/pregnancy-complications/hyperemesis-gravidarum/) ，如果你知道医学上的行话:妊娠过多(hyper)呕吐(呕吐)(gravidarum)。这是妊娠恶心的一种极端形式，被戏称为“晨吐”有些人在怀孕期间根本不会恶心，另一方面，有些人恶心得无法停止呕吐。

正常情况下，怀孕期间的恶心在 5 周左右开始，到 12 周左右就会消失。你可能偶尔会呕吐，但你可以把大部分食物咽下去。然而，如果出现呕吐，你很难把食物咽下去，并且会严重脱水。当其他人的恶心感减轻时，你的恶心感达到了顶峰。有时它会持续整个孕期。

呕吐会导致体重减轻，如果在怀孕期间持续，还会导致其他并发症，如肝脏和食道受损。治疗的目的是帮助你处理症状，直到恶心消退(或者直到你有了孩子或终止妊娠)。你可能会被开出 [IV 液体、电解质和维生素](https://www.merckmanuals.com/professional/gynecology-and-obstetrics/abnormalities-of-pregnancy/hyperemesis-gravidarum) 来替代你不能通过食物获得的东西，并且你的医生可能还想排除其他可能导致类似症状的情况。像 Diclegis 和 ZofranT5 这样的抗恶心药物可能是一种选择 。幸运的是，如果这是你第一次怀孕，你不能保证下一次会有呕吐。复发率从 13%到 81%不等。
