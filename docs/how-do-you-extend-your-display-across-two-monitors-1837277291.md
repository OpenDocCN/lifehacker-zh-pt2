# 如何在两台显示器之间扩展显示？

> 原文:[https://life hacker . com/how-do-you-extend-your-display-cross-two-monitors-1837277291](https://lifehacker.com/how-do-you-extend-your-display-across-two-monitors-1837277291)

在本周的 [**Tech 911**](https://lifehacker.com/c/tech-911) 专栏，Lifehacker 对所有极客的建议专栏中，我们正在解决一位读者的问题，他对自己的梦想设置有问题:多显示器安排让他的生产率翻倍(或偷懒)。

Watch

我能感受到他的痛苦。多年来，我一直使用两台显示器——即使在以前的工作中，我很少给任何人额外的显示器——直到我最终升级到我的 34 英寸超宽显示器。如果或者当你的操作系统变得繁琐时，多显示器设置可能会有点烦人，但是你遇到的任何问题应该不会花很长时间来修复。

Lifehacker 读者 **Sameer** 写道:

> 我需要一些关于 Windows 10 的帮助。我使用双显示器(几年前)。显示器工作正常，直到最近我升级了我的 Windows 10，现在两个屏幕都显示相同的应用程序。控制面板中的检测不起作用。一个是戴尔，另一个是华硕显示器。你能帮忙吗？

好消息是，Sameer。我不认为你遇到了永久的问题。我非常怀疑升级到 Windows 10 已经用低效的多显示器设置诅咒了你。可能发生的情况是，升级只是重置您的设置，而 Windows 正在处理您的显示器，无论它现在想要什么。

要解决这个问题，您有几个选择。你没有提到你的系统是运行集成显卡还是 AMD/Nvidia 显卡，所以我就从解决这个问题开始。如果你使用的是第三方显卡，确保你已经安装了最新的驱动程序(从 [AMD](https://www.amd.com/en/support) 或者 [Nvidia](https://www.nvidia.com/Download/index.aspx) )。作为其中的一部分，您可能会选择升级您当前使用的任何驱动程序(“快速安装”)或运行“全新安装”，在安装过程中清除任何现有设置。跟后者走，以防万一。

接下来，调查任何一家公司的控制面板——通过右键单击 Windows 桌面并选择 Nvidia 控制面板或 AMD 镭龙设置找到。在这里，您应该找到某种管理多显示器设置的选项，或者至少找到一种方法来检查您的两个显示器是否以您不希望的方式配置(克隆或扩展)。因为我使用的是 Nvidia 显卡，所以我的系统看起来是这样的:

在我的例子中，我可以右键单击我的一台显示器，然后选择是要克隆该显示器(主显示器上显示的内容会出现在另一台显示器上)还是扩展该显示器，我相信这就是您正在寻找的设置。

如果您在控制面板中看不到此选项，您想要一个更简单的解决方案，或者您使用集成显卡，您也可以通过 Windows 本身完成相同的步骤。点击开始按钮，输入“显示”，调出 Windows 10 的显示设置。单击开始菜单中出现的第一个选项。您将看到一个如下所示的屏幕:

向下滚动一点，您应该会看到一个下拉框，允许您选择是否要复制或扩展您的显示器(或者，无论出于什么原因，只在您的第一个或第二个显示器上显示图片)。我敢打赌，在您的情况下，这已经被更改为“重复”，您将会选择“扩展”，而不是:

如果设置此选项*仍然*不起作用，您可能需要在故障诊断方面有点创意。当我遇到多显示器设置问题时，一个通常有效的技巧是关闭系统电源，拔掉辅助显示器，然后在只连接一个显示器的情况下打开系统电源。从那里，你可能还想重新安装你的图形驱动程序(如果适用)，使用一个“干净”的安装，所以没有任何奇怪的设置给你的问题。

完成后，再次关闭系统电源，插入第二台显示器，然后再次打开系统电源。你可能不得不通过你的 AMD 或 Nvidia 控制(或 Windows 10)再次对设置大惊小怪，但这个*应该*修复你一直存在的任何问题。你也可以咨询 Windows 10 的设备管理器，确保那里没有任何奇怪的事情发生(比如，你的一个显示器有两个实例——如果有，试着卸载一个？)

另外，我会再次检查你是否正确地插上了显示器。我不认为这是一个问题，因为在你安装 Windows 10 之前，一切都很好，但是值得一提。

如果你没有显卡，两个显示器都应该插入主板的接口。如果你有显卡，把两个显示器都插上。不要一个显示器插在显卡上，一个插在主板上。我怀疑如果你这样做，你甚至不会在两个显示器上都看到一张图片，但如果你这样做了，这可能就是为什么你不能在两个显示器上扩展你的图片的原因。
