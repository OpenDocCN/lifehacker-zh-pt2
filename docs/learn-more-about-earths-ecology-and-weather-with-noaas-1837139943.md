# 使用 NOAA 的新应用程序了解更多关于地球生态和天气的信息

> 原文：<https://lifehacker.com/learn-more-about-earths-ecology-and-weather-with-noaas-1837139943>

美国国家海洋和大气管理局(NOAA)是一个科学的政府机构，主要关注海洋和水道以及地球大气的状况。



该组织有一个名为“球体上的科学”的产品，它将一些数据显示在一个房间大小的大球体上。它通常出现在博物馆里。如果你曾经在博物馆的大型地球仪投影上看到过类似风的图案，那很可能是球体上的科学。

现在你可以在 iPad 或手机上获得类似的体验。

[SOS 探索者](https://apps.apple.com/us/app/sos-explorer/id1358724422#?platform=ipad) 是 NOAA“球体科学”的手机版 iOS 和 Android 版免费提供，有了它，你可以看到描绘从大气风暴和气候变化到土星环等一切事物的动画图像。

你知道，复杂的环境问题，用一点视觉辅助工具就能更好地理解。

以下是一些可用的样本数据集:

最近的飓风季节

鸟迁移

地震活动

冰雪

洋流

历史海啸

脸书友谊

空中交通

太阳系的行星

气候变化模型

360°水下图像

有了这个应用程序，你可以选择你想看的东西，然后在应用程序的虚拟地球仪上显示出来。如果您愿意，您可以放大不同的部分，并根据您的兴趣定制您的体验。

这并不能完全取代博物馆的体验，但如果你对地球科学感兴趣，这可能是一件有趣的事情，可以玩一会儿，并有可能在这个过程中学到一些东西。
