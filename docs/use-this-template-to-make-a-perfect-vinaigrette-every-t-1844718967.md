# use this template to make a perfect vinaigrette every t 1844718967

> 原文：<https://lifehacker.com/use-this-template-to-make-a-perfect-vinaigrette-every-t-1844718967>

一份好的调味汁应该是咸、甜、酸三种味道的强烈结合。在一个或另一个方向上犯太多错误，你的沙拉就会遭殃。然而，制作美味的调味汁并不难，使用上面(和下面)的模板可以帮助你每次都做好。

### 你可能用了太多的油

正如我尊敬的同事 A.A .牛顿在 之前指出的 [，大多数调味汁配方需要太多的油而不是足够的酸。牛顿通过将她遇到的所有油醋汁配方中的油减半来解决这个问题，但是这个策略只适用于现有的配方。](https://lifehacker.com/vinaigrettes-need-way-less-oil-than-you-think-1837655884)



如果你想做你自己独特的特制调味汁，最好从一份醋和一份油的比例开始，而不是经常推荐的一份醋三份油。这可能看起来像酸过度，但请记住，一个好的调味汁应该太强烈，不能单独享用。这是调料，不是西班牙凉菜汤。半杯调料对于一份家庭大小的沙拉来说足够了，所以从 1/4 杯油和 1/4 杯酸开始。

对于油，你可以用一些有味道的东西，像橄榄或榛子；或者你可以使用 [一种更中性的选择](https://lifehacker.com/why-olive-oil-isnt-always-the-best-oil-for-vinaigrette-1834552898) 像葡萄籽、鳄梨或菜籽油，并真正让其他成分发光。如果你想变得真正疯狂，你可以混合油——像烤芝麻和葡萄籽，以扩展前者的味道——只要确保你不超过 1:1 的油--酸比例。

对于酸来说，几乎任何醋都可以，尽管我会避免纯白醋。如果你想要非常酸的调味汁，你可以加入一汤匙白醋和三汤匙其他东西，但是我告诉你这真的很酸，我不是在开玩笑。柠檬和酸橙汁也很可爱，尤其是搭配蜂蜜的时候。



### 你必须用一些芥末

一份好的芥末酱会给你的调味汁增加味道，更重要的是，它是稳定的乳化调料的关键。没有它，你的油和醋会在很短的时间内彼此分离，无论你如何用力摇晃。(将中的[和](https://skillet.lifehacker.com/a-mason-jar-is-the-only-tool-you-need-for-perfectly-emu-1796819745)中的摇匀，效果最佳。)含 [大量粘液的芥菜](https://skillet.lifehacker.com/how-to-choose-the-right-mustard-for-your-vinaigrette-1832912147) 乳化效果最好，所以选择第戎或其他全谷物选项，最大化其稳定效果。一汤匙就够了。

### 你需要加点糖，亲爱的

一点点糖让一切都变得更好吃，沙拉也不能幸免于这一科学事实。液态甜味剂进入溶液最快，所以选择糖浆。我用枫糖浆、龙舌兰糖浆、纯糖浆和蜂蜜来加甜油醋汁，所有这些都非常美味。你只需要一汤匙。

### 加入盐和其他调料

盐不只是让东西尝起来咸，它让东西尝起来好，你需要 1/4 茶匙盐来让你的调味汁唱歌。你可以在这里停下来，你会有一个非常美味的调料，但我知道你们中的许多人不会(为什么你应该)。其他干调味料——如孜然、辣椒粉、大蒜粉、白胡椒等。—从 **1/4 茶匙**开始，根据口味需要添加更多。对于新鲜的香草和葱(如大蒜和葱)，加入切碎的它们，**一次一茶匙**，直到你达到你想要的效力水平，记住它们在调料中的味道。



### 以一点常识结束！

为每个类别选择最有味道的选项可能很诱人，但请记住，这都是关于平衡的，选择一种或两种味道来突出，以防止事情变得混乱。如果你有超级美味的浸泡醋，考虑使用温和的油。如果你想展示柠檬，也许放弃大蒜。此外，检查你的醋的成分是否有添加剂，如果你不确定它们的味道，也许可以用一片面包尝一尝。调味米醋，顾名思义，预先用盐和糖调味，所以你可能不需要通常的盐量。幸运的是，做调味汁时很容易品尝，所以每次加入一点你不熟悉的配料，全部摇匀，然后品尝并根据需要调整。
