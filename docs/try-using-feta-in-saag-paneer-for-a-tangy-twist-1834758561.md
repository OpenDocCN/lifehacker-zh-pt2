# 试着用羊乳酪来做一次强烈的扭转

> 原文：<https://lifehacker.com/try-using-feta-in-saag-paneer-for-a-tangy-twist-1834758561>

不管你是谁，也不管你以什么为生，把晚餐端上桌都是一件不可能完成的任务。在一整天的食谱开发( [或与我悲伤的大脑](https://skillet.lifehacker.com/how-to-feed-yourself-when-youre-really-depressed-1823560127?_ga=2.104056644.516558604.1557914692-amp-VOgg_tjC2YsJPdc1F95aeA) )之后，我宁愿啃一块奶酪或干脆不吃晚餐，也不愿弄脏更多的盘子。所以我看了很多美食电视，买了很多烹饪书，总是在寻找一些东西来让我摆脱我永久的工作日晚餐无聊。如今，没有人比 Priya Krishna 做得更好。



克里希纳神奇的新食谱，[](https://www.priyakrishna.me/indianish)*，是我很久很久以来买的最好的一本书。像每一本伟大的食谱一样，它不仅仅是食谱的集合:它是印第安-美国食材和技术的速成班，是给微波炉的情书，是自制酸奶的宣言，是一部真正感人的家庭历史，因为它是大声笑出来的欢闹。但真正让它与众不同的是，因为没有更好的词，它令人难以置信的“可烹饪性”大多数食谱都是由 Priya 的妈妈 Ritu 创作的，她养活了她的四口之家，并经常举办晚宴，同时全职工作是一名航空公司软件工程师。这意味着*印度风味*中的每一道菜都是快速、简单、值得欣赏的大众美食——这正是每个疲惫的家庭厨师所需要的。*

*如果 Instagram 的 [是](https://www.instagram.com/explore/tags/saagfeta/) 的话，那么*的印度式*海报儿童就是 [Ritu 的 saag feta](https://www.youtube.com/watch?v=E4G5Y2ghqEQ) ，这是 saag paneer 的一个聪明的转变，用大块的咸羊乳酪代替 paner，这是温和而有弹性的——有点像奶酪。你可以在本书 [中找到完整的食谱，或者在 Bon Appétit](https://www.bonappetit.com/story/saag-feta-indian-ish) 上找到，并且你也可以在 Bon Appétit 测试厨房里观看 Priya 的制作:*

 *[https://lifehacker.com/embed/inset/iframe?id=youtube-video-E4G5Y2ghqEQ&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-E4G5Y2ghqEQ&start=0)* 

*这种互换直接有效。羊乳酪不仅更容易找到，它也更咸、更香，而且通常比潘妮尔更美味，当它与肉汁融合在一起时，你得到的质感是难以超越的。就像 saag paneer 一样，有大量的大块软奶酪，但也有少量半融化的羊乳酪流过肉汁。而且，就像这本书里的其他东西一样，萨格羊乳酪有着无与伦比的努力回报比:整个事情在大约半小时内完成，包括准备，味道如此之好，以至于它可能会永远毁了你的萨格潘尼尔。我知道对我来说是的。*

*我对书中的食谱做了一个很大的改变，就是研磨我的香料，而不是像食谱建议的那样，把它们扔进整个肉汁中。除非你有 Vitamix(或者类似的)，否则我建议你也这样做；用我的棒式搅拌机搅拌再多也不能粉碎整个豆蔻和芫荽豆荚。除此之外，我还做了一些小调整:肉汁里多放了些羊乳酪、大蒜和酸橙，还在锅里放了一小撮阿勒颇胡椒片( [一种温和、果味浓郁的红辣椒](https://www.bonappetit.com/story/what-is-aleppo-pepper) )，以增加颜色和味道。如果你以前从来没有做过 chhonk，它基本上是整个和/或磨碎的香料在油中油炸的混合物或 [酥油](https://skillet.lifehacker.com/make-ghee-in-the-microwave-because-youre-lazy-and-also-1834730652) 最后淋在盘子上作为点睛之笔。*

*以下是制作萨格羊乳酪的简单方法。将切好的羊乳酪放入超级美味的菠菜肉汁中...*

*...在酥油中放入整粒孜然籽和磨碎的红辣椒...*

*...洒满全身。与印度香米、酸橙、酸奶和香菜一起食用。就是这样。*

*最精彩的部分？这种天才的互换远远超出了萨格·帕尼尔。无论你是把它搅拌成肉汁，还是和五香蔬菜一起搅拌，块羊乳酪都有着帕内的奶油般的美味，还带有额外的令人上瘾的咸味。我不知道你怎么想，但是马塔尔羊乳酪、羊乳酪提卡马萨拉和羊乳酪戈壁听起来对我来说非常棒——我迫不及待地想尝试一下。*
