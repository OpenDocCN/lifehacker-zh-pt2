# 如果我在 1 月 15 日之前不升级 Windows 7，我会完蛋吗？

> 原文：<https://lifehacker.com/am-i-screwed-if-i-dont-upgrade-windows-7-by-january-15-1840903120>

截止日期很可怕。*我知道*。微软给 Windows 7 用户抛出了一个大难题:在 2020 年 1 月 15 日之前， [更新到更现代的操作系统](https://lifehacker.com/quickly-upgrade-windows-7-to-windows-10-for-free-with-t-1840843214) ，否则你将永远不会收到安全更新。最终，微软甚至将在全年内开始禁用 Windows 7 的关键服务——比如互联网双陆棋*和*互联网跳棋。



我对此有点轻松，但微软放弃 Windows 7 *是很多人担心的原因。Lifehacker 读者 **Douglas** 最近带着这个问题写道:*

> “我读了你写的关于 Windows 7 的文章，现在我真的负担不起升级。如果我现在不升级会是灾难性的吗？提前感谢您的帮助"

灾难性的？不会。如果你仍然坚持使用 Windows 7，你很可能会用它来解决一些简单的需求——发送电子邮件、浏览网页等。或者，您可能有一些特定的应用程序无法在未来版本的 Windows 上运行，或者您甚至可能在旧硬件上运行 Windows 7，由于某种原因(速度或可用空间)，旧硬件无法处理升级。

Windows 7 将在 1 月 15 日继续运行。然而，现在微软不会再为操作系统发布任何安全更新，Windows 7 将更容易受到攻击，这是事实。毫无疑问。

然而，我愿意冒险说，你可以用一些好的常识来缓解大多数问题。用 Lifehacker 的话来说，这意味着认真考虑你家里的网络安全设置，并尽一切努力确保软件不会进入你的机器，从而利用任何无法修补的漏洞。

如果我还在使用 Windows 7，我会选择核选项。我会在我的系统上安装一些 Linux 的变种，并在虚拟机上运行 Windows 7，从而给我一个全新的、更新的系统来处理日常活动，并访问 Windows 7 来使用我需要使用的任何超级特定的应用程序或服务。这不是一个棘手的过程，但设置这种配置可能会超出许多仍在使用 Windows 7 的人的理解范围。或者你的系统运行虚拟机的能力太弱。(下周我可能会为此写一篇指南，敬请关注！)

否则，在您坚持熟悉 Windows 7 的同时，您可以做许多其他事情来尽可能地保护自己免受攻击。这份清单相当全面:

*   确保您使用的路由器仍在从制造商那里接收固件更新(比如，不是六年前的东西)。保持你的路由器更新和 [使它尽可能安全](https://lifehacker.com/how-to-make-your-wifi-router-as-secure-as-possible-1827695547) 。
*   不要用 [管理员账号](https://support.microsoft.com/en-us/help/2663817/how-to-determine-your-user-account-type-in-windows) 处理日常任务。使用标准帐户，并在需要更改任何设置时切换到管理员帐户(如果需要)。完成后切换回来。
*   使用在 Windows 7 停产后仍会更新的防病毒软件。(我推荐 [BitDefender 的免费版](https://www.bitdefender.com/solutions/free.html) 。)
*   用免费的反恶意软件工具[定期扫描你的系统。保持此实用程序更新。](https://www.malwarebytes.com/pricing/)
*   尽可能保持你系统的 BIOS(如果不是旧的话)
*   保持关键的 Windows 安全功能(如 Windows 防火墙)，禁用或删除不需要的功能(Java、Adobe Flash Player、Quicktime、Shockwave Player 等)。类似地，清除你藏在系统中的任何其他垃圾——浏览器工具栏、你不用的应用程序等等。
*   不要下载你不认识的文件。不要打开您的浏览器为您下载的文件。
*   跳过网络浏览器，使用考虑到你的 [隐私](https://gofirefox.com/) 和 [安全](https://brave.com/) 的浏览器。运行 [uBlock Origin](https://github.com/gorhill/uBlock) 和[uBlock Origin Extra](https://github.com/gorhill/uBO-Extra#ubo-extra)、[Privacy Badger](https://www.eff.org/privacybadger)、以及[HTTPS Everywhere](https://www.eff.org/https-everywhere)最小值。
*   使用 [第三方 DNS](https://lifehacker.com/how-to-browse-faster-and-more-securely-with-cloudflar-1824256064) 帮助 [您在浏览时更加安全](https://www.cloudflare.com/learning/dns/what-is-1.1.1.1/) 。
*   在 [沙盒](https://www.sandboxie.com/) 中运行未知应用，同时确认它们可以安全使用。
*   [马上备份你的系统](https://lifehacker.com/first-look-at-windows-7s-backup-and-restore-center-5144757) 及其关键文件。将备份保存在一个安全的位置，以防您的 Windows 7 系统受到某种可怕的恶意软件的攻击，您必须从头开始。
*   考虑 [加密](https://www.veracrypt.fr/en/Home.html) 你存储在系统上最重要的数据。
*   保持你系统的应用尽可能的更新。
*   看看你是否可以 [黑进你的方式获得几年免费的 Windows 7 更新](https://lifehacker.com/how-to-extend-your-windows-7-security-updates-past-janu-1840313094) 。

还有一个更疯狂的想法。如果你出于某种原因不需要你的 Windows 7 机器上的网络，那么就断开它。只要你不到处乱插 u 盘到你的电脑上，你的系统被黑客攻击的几率就会直线下降。

我不想出来说，从 1 月 15 日开始，有一种完美的方式来保护 Windows 7(除了从网络上拔掉你的计算机)。我猜想大多数攻击将从用户活动开始；比如，某种网络钓鱼企图或恶意软件会让你陷入困境，这就是流氓应用程序感染你的系统并利用你的旧操作系统的方式。如果你有一些网络安全常识，你应该能够避免大多数(或所有)这些情况。

这并不是说你应该永远坚持 Windows 7。没有一个操作系统是百分之百安全的，但是我宁愿选择微软正在积极修补的操作系统，而不是没有修补的操作系统。相反，即使你升级到了 Windows 10，你也应该*继续*遵循本文中适用的说明。拥有可靠的安全实践是抵御恶意软件、病毒和其他令人不快的数字问题的最佳防线之一。
