# 停止玩糖果粉碎，开始玩更好的游戏

> 原文：<https://lifehacker.com/stop-playing-candy-crush-and-start-playing-better-games-1835009574>

多亏了智能手机和平板电脑，我们现在都是游戏玩家。我是游戏玩家，你是游戏玩家，我们的妈妈也是游戏玩家，他们的妈妈也可能是游戏玩家。所以是时候我们开始好好利用这些游戏了。

在上面的视频中，我提出了删除“糖果粉碎”等“愚蠢”游戏的理由，并推荐了“智能”游戏，这些游戏将帮助你提高技能，放松和冥想，或弯曲大脑中的神经连接。

我推荐的应用有:

*   Elevate ( [iOS](https://itunes.apple.com/US/app/id875063456?mt=8) ， [Android](https://play.google.com/store/apps/details?id=com.wonder&referrer=af_tranid%3D0TvMGdFVfdbhlWLXlkWAHg%26pid%3Delevateappcom%26c%3Delevateappcom) )
*   Lumosity ( [iOS](https://itunes.apple.com/app/id577232024?mt=8) 、T5】Android)
*   Orixo ( [iOS](https://itunes.apple.com/us/app/orixo/id1449230416?mt=8) 、T5】Android)
*   我爱色相([iOS](https://itunes.apple.com/us/app/i-love-hue/id1081075274?mt=8)[Android](https://play.google.com/store/apps/details?id=com.zutgames.ilovehue&hl=en_US))
*   NYT 填字游戏( [iOS](https://itunes.apple.com/app/id307569751?referrer=mat_click_id%3D6278ae8ea0c64999a52ca2e32bc11099-20190524-16948%26link_click_id%3D660510433899369351&mat_click_id=6278ae8ea0c64999a52ca2e32bc11099-20190524-16948) ，T5】Android)
*   bonza([iOS](https://itunes.apple.com/us/app/bonza-word-puzzle/id662053009?mt=8)[Android](https://play.google.com/store/apps/details?id=air.au.com.minimega.bonza&hl=en_US))

[虽然“大脑训练”游戏](https://slate.com/technology/2014/01/lumosity-cognifit-cogmed-are-these-kinds-of-brain-training-games-worth-it.html) 有多有效仍有争议，但很难说在 Elevate 中做数学题或解决 NYT 填字游戏不比玩像 Crossy Road 这样的游戏的僵尸模式 *tap* 、 *tap* 、 *tap* 更好地利用时间。
