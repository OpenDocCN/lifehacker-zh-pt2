# 关于高收益回报支票账户应该知道什么

> 原文：<https://lifehacker.com/what-to-know-about-high-yield-reward-checking-accounts-1830681732>

利率在一点点上升，但是如果你满足一定的要求，你可以通过转换到高收益回报支票账户获得大银行的两倍~两个百分点的收益。

由较小的地区银行和信用合作社提供的吸引客户的高收益回报支票账户提供高达百分之五的利率。要获得资格，你通常需要低于某个余额阈值(比如 10，000 美元到 25，000 美元，尽管它可能会更低)；报名直接存款；并且每月进行一定数量的借记交易。如果你这样做了，你将从更高的利息和联邦存款保险公司的保险中获益。

最后一部分是这个账户对某些人特别有吸引力的地方。CDs 和货币市场也可以提供相对较低的利率，但安全性较高；这些支票账户有相同的安全级别*和*更高的回报。跟踪利率的存款账户 [报告称利率](https://www.depositaccounts.com/checking/reward-checking-accounts.html) 处于五年来的最高水平。

“每个需要某种紧急储备账户的人肯定会考虑这些类型的工具，”财富顾问和注册财务规划师黛安·皮尔森告诉。

显然，除了利率之外，还有其他因素需要考虑。你会想查看与任何潜在新账户相关的 ATM 费用，以及在线支付选项、账户维护费等。(特别是这些费用可能会抹去你获得的利息。)你可能无法获得每个信用合作社的资格，尽管有些会相对容易，要求一次性捐款才能加入。只要确保你检查他们的银行足迹，如果进入一个物理分行对你很重要，并且你了解利率的要求。在你注册之前，也要查看一下银行或信用合作社的利率变化历史(你可以通过谷歌搜索找到)。

但是如果你有一点 [现金在](https://twocents.lifehacker.com/where-should-you-keep-significant-short-term-savings-1827685508) 中，你想从中赚取一点利息，这些账户是不错的选择。你会在这里找到 [最佳选项](https://www.depositaccounts.com/checking/reward-checking-accounts.html) 。
