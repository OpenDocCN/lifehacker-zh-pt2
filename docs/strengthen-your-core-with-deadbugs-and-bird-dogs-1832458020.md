# 加强你的核心与死臭虫和鸟狗

> 原文：<https://lifehacker.com/strengthen-your-core-with-deadbugs-and-bird-dogs-1832458020>

欢迎来到我们第二周的 [非跳板挑战](https://vitals.lifehacker.com/februarys-fitness-challenge-is-core-exercises-that-are-1832265424#_ga=2.159372154.1194826643.1549230290-1456718367.1520458611) ，在这里我们尝试一些你应该知道的被低估的核心练习。今天我们要做两个以生物命名的练习，这两个练习都比看起来要难。

第一个是长期生活黑客最喜欢的，这个死 bug。你开始仰卧，手臂和膝盖指向天空。伸出一只胳膊和一条腿，使它们与地面平行(但不要接触地面)，并保持几秒钟。

当你做这个动作时，你的背部会自然地想要拱起，所以你的腹肌正在努力防止这种情况。出于这个原因，它被认为是一种“反伸展”运动，与 ab 轮运动属于同一家族。它还可以帮助你专注于移动你的手臂和腿，而不是通过伸展或扭动你的背部来欺骗运动。当你表演死 bug 的时候，你的臀部和肩膀不应该离开它们在地板上的位置。

为了使死 bug 更容易，只做腿。你甚至可以考虑用手抓住什么东西。为了让它们更难，看看这些带有阻力带 的变化 [。](https://www.stack.com/a/9-dead-bug-variations-that-build-an-athletic-core)

### 鸟狗

把死臭虫翻过来，它就变成了一只鸟狗。你从双手和膝盖开始，背部保持中立。伸出一条腿和另一只手臂，然后回到起始位置。
