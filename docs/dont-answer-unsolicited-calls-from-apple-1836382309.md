# 不要接听苹果公司主动打来的电话

> 原文:[https://life hacker . com/don-answer-unsolicited-calls-from-apple-1836382309](https://lifehacker.com/dont-answer-unsolicited-calls-from-apple-1836382309)

最近，你*可能*收到了一条来自“苹果”的紧急语音邮件，声称你设备上的“金融活动”处于危险之中。

Watch

别担心——你的苹果账户很好，公司也不会打电话给用户，亲自警告他们 iCloud 账户或 iPhones 上的可疑活动；正如 [Snopes](https://www.snopes.com/fact-check/apple-support-call-on-suspicions/) 最近所写的，这是一种常见的欺骗骗局，骗子模仿合法公司的号码来骗取你的个人信息。

如果你接起电话(就像我最近做的那样)，你很可能会听到一条假想的苹果客户支持热线自动发出的欢迎信息。当你最终联系到“代理”时，他们会询问你的个人信息，比如你的 iCloud 账户信息。

这个数字本身看起来并不可疑；事实上，在 iPhone 上，如果你*确实*使用苹果合法的客户服务网站请求回电以寻求帮助，任何随后的虚假电话都将被索引为之前来自苹果*实际*支持热线的电话；换句话说，假电话看起来和合法电话没什么区别。

那么你能做些什么来避免这个骗局呢？正如苹果公司在其 [网站](https://support.apple.com/en-us/HT204759) 上所写的，你永远不应该接到该公司任何主动打来的电话。显然，让任何声称来自苹果的意外来电转到你的语音信箱，不要透露任何个人信息。此外，正如我们在 之前所写的那样，回叫任何机器人电话时要小心，这些电话可能会以你的名义(在你不知情的情况下)收取通行费。

苹果公司建议在其 [网站](https://support.apple.com/contact) 上举报任何网络钓鱼诈骗，这样你也可以帮助其他人避免类似的骗局。
