# 选民压制是真实的，它确实影响了黑人

> 原文：<https://www.theroot.com/voter-suppression-is-real-and-it-really-affects-black-p-1830001232>

我想我们都同意 2018 年中期选举至关重要。这意味着我们的投票比以往任何时候都重要。但是，当你已经注册并准备好参与这个良好的民主，但不能的时候，会发生什么呢？

再来说说选民压制。

选民压制很简单——这是一套用来阻止选民行使投票权的策略。美国有很长很长的压制选民的历史。

多亏了 1870 年批准的第 15 修正案，黑人在内战 后被允许投票。由于种族歧视法，黑人 [被迫参加文化测试，缴纳人头税](http://americanhistory.si.edu/brown/history/1-segregated/white-only-1.html) ，以及其他投票障碍。

快进到 2018 年。今天，压制选民的形式可以是选民身份证法、选民名册清洗、投票站关闭和故意误导选民的自动电话等等。

2006 年之前——奥巴马之前！— [美国 50 个州](https://www.bustle.com/p/is-there-voter-suppression-in-2018-heres-what-it-could-look-like-in-the-midterms-10620006) 没有一个州要求选民提供身份证投票。今天， [34 个州要求或要求你有某种 ID](http://www.ncsl.org/research/elections-and-campaigns/voter-id.aspx) 。 [乔治亚](https://www.theroot.com/actually-no-one-really-knows-anything-about-voter-sup-1829868393) 有“完全匹配”顾名思义，你身份证上的名字必须与你选民登记上的名字完全一致。没有变化，没有拼写错误。可悲的是，这次选举周期的选民压制策略并没有止于乔治亚州。
