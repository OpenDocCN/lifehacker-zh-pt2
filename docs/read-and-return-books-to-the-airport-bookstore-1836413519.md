# 阅读并还书到机场书店

> 原文:[https://life hacker . com/read-and-return-books-to-the-airport-books-1836413519](https://lifehacker.com/read-and-return-books-to-the-airport-bookstore-1836413519)

你在等你的航班登机。也许你在中途停留，或者你的航班延误了，或者你只是想消磨一些时间，所以你走进一家无论你在哪个城市看起来感觉都一样的机场书店。最近出版的精装书的桌子在呼唤你的名字，但你知道你可以一回家就从图书馆预订其中一本，所以你不能证明全价购买的冲动是正当的。或者可以吗？

许多机场商店看起来都一样的原因是因为它们在 100 多个机场经营着 850 家商店和 170 家餐厅和酒吧。他们有一个鲜为人知的计划，航空旅客应该利用 : t 他阅读和返回计划。

### 它是如何工作的

你在机场买了本书。你可以在飞机上或旅行后的几天/几周/几个月内阅读。你在购买后的 6 个月内退货，你会得到 50%的退款。

你需要你的原始收据(用它做书签，这样你就不会弄丢了)，但是你可以把书归还到任何一个帕拉迪丝·拉加代尔书店——你购买它的那个书店或者你要去的任何地方。然后，商店会以低于封面价格 50%的价格转售这些“二手”书籍(这样你也可以通过这种方式达成交易)。

公司发言人告诉我，任何以“破旧状态”归还的书籍仍将获得 50%的退款，并将捐赠给当地图书馆，“作为公司鼓励社区扫盲承诺的一部分。”

所以，不要把它想象成*买*一本书，而更像是*在紧要关头需要什么来打发时间时租*本书。
