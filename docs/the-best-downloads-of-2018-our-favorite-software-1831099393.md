### [为什么要从谷歌 Chrome 切换到火狐](https://lifehacker.com/why-you-should-switch-from-google-chrome-to-firefox-1821879163)

尽管 Chrome 仍然占有市场份额，但 Mozilla 今年在 Firefox 上做了一些非常有趣的事情。我一直想把我数量惊人的标签切换到 Mozilla 的浏览器上，看看 Firefox 在管理内存方面比 Chrome 好多少——也许这将成为 2019 年的一个解决方案。除此之外，Mozilla 今年在其浏览器和浏览器插件中大力推动了 [隐私](https://lifehacker.com/keep-your-data-secure-with-mozillas-newest-tools-1829304577) 和 [安全](https://techcrunch.com/2018/11/15/mozilla-adds-website-breach-notifications-to-firefox/) ，随着整个世界(OK，Edge)转向 Chromium，是时候比以往任何时候都更加热爱 Firefox 了。一个渲染引擎的世界。
