# twitter 新的固定列表让你可以丢弃远程推文

> 原文：<https://lifehacker.com/twitters-new-pinned-lists-let-you-ditch-promoted-tweets-1842092496>

众所周知，Twitter 的列表功能是一种很好的方式，可以从整天对你大喊大叫的互联网陌生人的混乱中提取有用性，新的更新使得在 iOS 或 Android Twitter 应用程序上访问你的列表变得更加容易。



首先，确保你已经通过谷歌 Play 商店或苹果的应用商店将你的 Twitter 应用更新到最新版本——这是你的选择。启动应用程序，点击左上角的你的照片，然后点击列表。

从那里，点击多达五个你想“钉”在你的主 Twitter 屏幕上的列表。

完成后，点击左上角的后退箭头。现在，你应该可以在你的订阅源顶部看到你的锁定列表，你可以在应用程序上左右滑动来访问它们。(如果你没有看到它们，请退出应用程序，在 iOS 或 Android 上将其从最近的应用程序列表中划掉，然后再次尝试启动 Twitter。)

就是这样！虽然这一功能不会修复 Twitter 并治愈它的所有烦恼——只有 [静音](https://lifehacker.com/make-twitter-a-more-pleasant-place-by-muting-notificati-1836364564) 才能真正做到这一点——但这是朝着正确方向迈出的一步。作为一个额外的奖励，我相信这些列表都默认为一个按时间顺序排列的提要*和*没有 [烦人的推广推文](https://lifehacker.com/private-lists-are-the-best-way-to-stay-organized-on-twi-1838834186) 。这本身就是一个很好的理由，让你在 Twitter 应用中尽可能多地添加有用的列表。
