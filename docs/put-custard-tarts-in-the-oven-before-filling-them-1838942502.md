# 在填充蛋挞之前，先把它们放进烤箱

> 原文：<https://lifehacker.com/put-custard-tarts-in-the-oven-before-filling-them-1838942502>

不管是填满水果还是不加装饰的，我最喜欢蛋挞的一点是它们整洁迷人的外观。与乡村的美国风格的馅饼不同，这些整洁的小甜点在被稳定、细致的手填满时看起来是最好的。为了让你的蛋挞看起来整洁得体，在馅料之前把它们放在烤箱里。



如果说重温英国烘焙展的老剧集有什么好处的话，那就是我发现了我以前错过的小技巧和窍门，这个蛋挞策略是一个微小但强大的游戏改变者。如果你以前做过蛋挞，你会知道即使是最少量的蛋挞溢出也会使你的面包皮看起来烧焦和不均匀，降低你的蛋挞的魅力因素(和你家人的荣誉)。大部分溢出发生在去烤箱的路上，所以把你的面包皮放在烤箱的架子上，*然后*用蛋羹填满它。你的蛋挞会因此而变得更好，你也不必为难看的斑驳的外皮感到羞愧。
