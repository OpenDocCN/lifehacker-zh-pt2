# 享受一次技术升级吧

> 原文：<https://lifehacker.com/treat-yourself-to-a-tech-upgrade-1844574128>

时局艰难。失业率高得惊人；学校无法决定如何安全开放，或者根本不应该开放；我们在阻止第二波新冠肺炎浪潮方面做得如此糟糕，以至于我们仍然在骑第一波 。我们中的许多人很幸运地仍然被雇用，由于在家工作和凑合使用可能远不理想的技术设备，他们变得非常疯狂。

Watch

虽然我不建议现在就用一台 2000 美元的新笔记本电脑来增加你的预算(除非你真的需要一台新的),但为了你的心理健康，让自己至少有一次合理的升级来帮助你解决一个长期的技术问题是值得的——特别是如果你能在这个过程中学习一些新东西的话。

### 很容易对你的愿望清单说“不”

我知道当我为钱感到压力时，我会倾向于保持沉默。我的美食塔可钟午餐变成了普通的自制火鸡三明治。我白天喝酒变成了周末喝酒(用纯苏打水+酒的组合)。我从各种愿望清单上购买的所有物品都回到了“眼不见，心不烦”的领域。

很多人现在对未来感到焦虑，这是有充分理由的。他们可能暂时有工作，但担心随时会失业。也许他们被困在一个对他们的心理健康有问题的生活环境中——例如，不得不在满是室友的房子里执行隔离条例。或者也许他们的生活在其他方面很好，但简单的事实是 [*我们都在一个没有结束迹象的疫情*](https://www.cnbc.com/2020/08/02/dr-birx-warns-us-in-new-phase-of-coronavirus-pandemic-with-extraordinarily-widespread-cases.html) 本身就给他们造成了足够的压力。我明白你的意思，真的。

### 你不必为了升级而打破你的银行账户

但是当我说“升级”的时候，我不是指“新车种类”——除非你很有钱，那是你缓解压力的方式。对于其他人来说，合适的升级可能是廉价和小型的。你可以或多或少地做些事情，以适应你的个人预算和环境，但如果你有办法，你不应该武断地阻止自己购买愿望清单上的东西，因为疫情让你在精神上处于一个糟糕的境地。

由于我是一个极客，我最近用我的“我应该升级一些东西，以改善我的心情，并最终完成一个项目”的方法来证明为我的汽车购买一个售后后视摄像头是正确的。我没有疯狂地花钱——33 美元，外加一些小工具的费用——但这个项目让我忙了几个小时。我正在自学如何安装和连接，而不是花钱请人帮我安装。

### 将采购与项目相结合

如果你想让你的“升级”有回报的感觉，那就结合一个项目:挑选你想买的东西*和*学习如何做。这并不是说，你不会从购买任天堂 Switch 的那个支架或那套新的餐盘中找到快乐。但是，一件能让你学会一些新技巧的东西——即使是简单的家庭维修，或者在你公寓的两点之间架设以太网电缆以设置新的接入点，或者获得一个钻刷附件并弄清楚如何使用它来深度清洁你的浴室——会让你感觉更值得，而不像是冲动消费。

换句话说，你不太可能为了减轻焦虑而进行购物的负罪感之旅；你将会得到一些你需要的东西来开始你一直想要解决的项目。一旦你完成了那个项目，你会有一个新玩具，一个从你的任务清单上划掉的任务，以及一种至少应该持续一周*的邪恶成就感——或者不管你使用你刚刚做的任何东西多长时间。*

*说真的。去挂几个你从来没去布置过的架子。学习如何给一幅画装框。在一个总是有点暗的房间里重做照明。买一个树莓派，让你的房子自动化。或者无视我所说的一切，拿起你在过去两年中一直想要的乐高玩具。*没事的*。*

*我非常支持基于项目的升级，但我也希望你开心。如果这意味着订购你一直想买的艺术印花，甚至是你真正喜欢的连帽衫，那就去做吧。让自己快乐起来。允许自己升级，因为这是你应得的。也许有一天，一旦我们都恢复正常，你可以拿起你愿望清单上的第二件事来庆祝。(希望是蛋糕。)*
