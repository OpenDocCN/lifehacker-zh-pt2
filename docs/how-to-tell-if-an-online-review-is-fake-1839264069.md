# 如何辨别一篇网上评论是不是假的

> 原文:[https://life hacker . com/how-to-tell-if-an-online-review-is-fake-1839264069](https://lifehacker.com/how-to-tell-if-an-online-review-is-fake-1839264069)

用户评论非常有用，尤其是在护肤品和美容产品领域。毕竟，这是你抹在脸上的东西。你珍贵的脸！但是当其中一些评论被证明是假的，你怎么能确定你能相信你读到的任何东西呢？

护肤品公司 Sunday Riley 被指控两年来发布虚假产品评论，现已与美国联邦贸易委员会达成和解。根据联邦贸易委员会的投诉，同名品牌的首席执行官桑迪·赖利(Sunday Riley)指示员工和实习生在丝芙兰的网站上为该公司的产品建立账户并留下评论，其中一些产品的价格超过 100 美元。

根据联邦贸易委员会去年提交的诉状，赖利甚至给她的员工发电子邮件，一步一步地说明如何使用 VPN 来隐藏他们的身份，因为他们建立了假账户来留下五星评论，同时“非常热情，看起来不像一株植物”。

此外，员工应该跟踪其他(合法的)评论，“不喜欢”那些负面的评论，并撰写评论以压倒任何负面评论。“收到评论时，也要阅读。一份备忘录上写道:“如果你注意到有人说我不喜欢它的 x，写一篇评论说相反的话。”

推假评论怎么处罚？嗯，没什么。该品牌必须清楚地表明，谁在推销公司产品时得到了报酬，并应该指导其员工进行披露。联邦贸易委员会甚至无法就和解达成一致，该机构的三名委员投了赞成票，两名委员投了反对票。持不同意见的委员们发表了一份声明，称根据时尚法 的规定，联邦贸易委员会应该进行罚款，称这一解决方案奖励了不良行为者，伤害了诚实的公司。

### 如何识别假评论

虚假的周日赖利评论专注于新产品，试图获得更多的关注和销售。所以，为了快速检查，注意新发布产品的评论数量。“留意最近推出的产品，它们会突然获得数千条正面评价，”DealNews.comT1 的消费者分析师迈克尔·博恩布莱特(Michael Bonebright)说。

一些网站，如亚马逊，有一个早期评论程序，明确标记由评论者撰写的新项目的评论，这些评论者有时会因他们的努力而获得报酬。“如果你看不到任何迹象表明卖家利用该项目获得了初步评价，那就是一个危险信号，”他说。不是每个在线商店都会有这样的程序，但是许多主要的零售商会指出一个评论者是否提前收到一个产品或者免费评论。

不管你在哪里浏览，寻找在一堆评论中出现的重复单词或短语。“亚马逊在清除这类假货方面做得很好，但其他网站就没那么努力了，”Bonebright 说。

如果你对书面评论没有信心，Bonebright 建议寻找一个你可以观看的视频评论:“虽然这些评论者经常因他们的评论而获得报酬，但你仍然可以看到产品，并更好地了解它的实际价值。”
