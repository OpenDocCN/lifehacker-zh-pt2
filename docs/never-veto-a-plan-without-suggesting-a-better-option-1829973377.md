# 在没有提出更好的选择之前，永远不要否决一个计划

> 原文：<https://lifehacker.com/never-veto-a-plan-without-suggesting-a-better-option-1829973377>

与你的朋友、爱人和浪漫的伴侣一起制定计划应该是一个简单的过程，甚至可能是一个愉快的过程。然而，很多人都非常糟糕，犹豫不决，无法掌握或解释基本细节，沟通能力差，以及纯粹的懒惰。



我不能假装用一篇博文就能解决所有这些问题，但我可以用一个基本的要求来尝试消除至少一个常见的、令人费解的祸害:如果和你一起制定计划的人建议你做一些不适合或不吸引你的事情，尽管说出来！但是如果这确实是你想要见的人，而你否决了他们的建议，那么你有责任 100%提出一个替代方案。如果你不这样做，参与你计划的另一方(或多方)完全有权利认为你 a)没有真正的兴趣和他们在一起，或者 b)是一个麻烦的讨厌鬼，应该被排除在未来的绞刑之外。公平就是公平。

如果不是很直观，我们甚至可以通过几个例子来演示这种情况会如何发生。不想去被建议的街区？好的，对你和你的约会对象都方便的另一个地点是哪里？暗地里讨厌保龄球之类的？完全可以理解，也许每个人都可能喜欢不同的活动(也许你的朋友甚至会帮忙找一个，如果你坦诚地告诉他们你的感受)？

没心情吃披萨吗？非常酷，但是你还想吃什么呢？在每个人的价格范围内，哪里有一些餐馆你可以买到那个东西？(如果你有挑食规则或者有特殊的饮食习惯，这些规则适用于你十倍。满足你的需求是你的责任，而不是别人的。天啊。)

这里有一个简单的问题:桌子上的日期或时间不符合你的时间表吗？一切都好！建议一些替代方案！诸如此类。履行你的社会契约有时需要一点额外的努力和考虑，即使你在发短信和赶时间。如果这太难处理，也许你最好呆在家里。
