# 低轮廓轮胎正在毁掉现代汽车

> 原文：<https://jalopnik.com/low-profile-tires-are-ruining-modern-cars-1836309462>

去年夏天我在法国拥有的大众 Up GTI[是一个梦想](https://jalopnik.com/the-2018-volkswagen-up-gti-is-as-fun-as-tiny-cars-get-1826079451) 。它也有 17 英寸的车轮和低轮廓轮胎。在离海岸不远的地方，我碾过了路上一块看似无害的小石头。大约在接下来的 10 秒钟内，我就知道 [出事了](https://jalopnik.com/erik-shilling-is-having-a-very-nice-time-in-france-1825829425) 。



如今，低轮廓轮胎风靡一时，主要是为了美观，因为汽车制造商想要突出“性能”或其他什么。但是大多数人，甚至那些自认为是超级超超性能的人， [都不需要这些](https://jalopnik.com/bring-back-big-sidewalls-1831232854) 的垃圾橡胶块。

当然，他们可能会在赛马场提供一些优势，但大多数人不会每天都在赛马场。大多数人在道路上驾驶他们的汽车，这些道路会形成坑洞，这对低轮廓轮胎来说是致命的。你喜欢驾驶一辆轮胎低矮的汽车的压力吗？这种轮胎随时可能被一些看似无害的障碍物撞倒。那就想尽一切办法，把你的钱投进低规格轮胎。

对于我们其余的人，我提议禁止。胎侧更大的轮胎可以承受更多的虐待，而且看起来也更好。当你碾过一个坑洼时，你不会得到一大笔修理费，正如我们的工程朋友杰森·芬斯克最近在他的特斯拉 Model 3 性能中解释的那样:

 [https://jalopnik.com/embed/inset/iframe?id=youtube-video-H6sPc9dFsGw&start=70](https://jalopnik.com/embed/inset/iframe?id=youtube-video-H6sPc9dFsGw&start=70) 

这个视频并不令人惊讶——2600 美元更换一辆豪华汽车的两个轮胎和两个车轮是正确的——但芬斯克关于 Model 3 轮胎无用的说法完全正确。在他被困在路边的 1 小时 40 分钟里，坑坑洼洼让他的两个乘客侧轮胎瘪了下去，但并没有让其他人瘪下去。

这是因为这些人要么设法避免了坑洼，要么轮胎具有适当的侧壁。或者他们只是运气好。无论如何，驾驶低轮胎的汽车会招致灾难，尤其是像 Model 3 这样的汽车，正如芬斯克指出的那样，Model 3 的重量超过 4000 磅，给其 20 英寸的米其林 Pilot Sport 4S 轮胎带来了更大的压力。(问题中的轮胎是相当低的 35 轮廓轮胎，准确地说是 235/35/20。它比 195/40/17 那次炸了我负责的大众汽车还要低调。)

这不是特斯拉的问题，也不是大众的问题，也不是任何人的具体问题，这是任何配有低规格轮胎的汽车的问题，尤其是在豪华车这些天如此急切地想要炫耀其性能的时候，而且越来越重。我还会注意到我的本田飞度有着大胎侧的轮胎。我从未刺破它们，即使在纽约崎岖的道路上追逐。

当那辆带低底盘的 GTI 爆胎时，我停在路边，徒劳地在后面寻找备胎。相反，我所能找到的只是一些修补漏气的粘性物质，它们根本不起作用。这辆车我总共买了大约一个星期。

匆忙地将这些东西从我们的生活中移除。
