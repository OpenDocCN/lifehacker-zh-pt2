如果你更喜欢从一本书里学习，他们有两本特别好的书推荐。如果你想要一个在线课程，他们有几个，有不同的方法和界面。我特别被这个描述吸引住了:

> 可汗学院有一门 [介绍 HTML/CSS:制作网页](https://www.khanacademy.org/computing/computer-programming/html-css) 的课程，它以一种超级酷的格式打包。这就像视频一样，你可以听到教师向你讲述学习过程，但你看到的是一个真实的文本编辑器和真实的输出。有时候是老师在控制代码，有时候是为了挑战而中断，让你自己接手并编辑代码。

正如指南所说，你可能会通过结合多种方法来学习。作家 Chris Coyier 将 web 开发比作演奏音乐:你和老师、视频或和弦图一起工作，但你也可以自己演奏。如果你想得太多，就会开始感到害怕，所以使用这样的列表来选择一个起点，暂时忽略其他的事情是有帮助的。如果 web 开发吸引了你，你很快就会自己拓展业务。
