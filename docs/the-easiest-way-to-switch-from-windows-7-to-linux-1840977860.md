### 在我们开始之前:准备好离开 Windows 7

为了让这篇文章不超过 100 万字，我假设您已经从现有的 Windows 7 安装中保存了您的关键数据和其他所有需要的东西。(无论如何，你应该一直备份你的系统，所以这并不奇怪)。

如果你对切换感到紧张，你可以先创建一个我们将要使用的 Linux 发行版的 live CD(或 live USB)， [Linux Mint](https://linuxmint.com/) 。事实上，无论如何您都必须这样做才能安装它，所以最好现在就把它去掉。通过在计算机启动时引导到 Live CD 而不是 Windows 7，您将能够探索使用 Linux Mint 的感觉，就好像您已经在您的系统上安装了它一样。您在操作系统中所做的任何事情都不会在重启之间持续——都是临时的——但这至少让您能够在完全提交之前尝试这个 Linux 发行版，看看您是否喜欢它。

对于大多数人来说，我建议创建一个你已经安装的所有 Windows 应用程序的列表，并将其保存到某个地方的云存储帐户，以及任何其他合适的关键数据(例如，你的文档)。将你的照片上传到云存储服务上——如果足够的话，可以使用谷歌照片，如果你需要保持照片的原始质量，可以使用在线存储服务器。完成这一部分需要花很多时间，因为你只有一次机会(除非你采取了第二步，使用 Backblaze 这样的服务来自动保存你的所有内容，或者在其他地方克隆了你的整个驱动器)。

最后，记下您的 Windows 7 产品密钥。如果你不知道它在哪里或者是什么，使用 [ProduKey](http://www.nirsoft.net/utils/product_cd_key_viewer.html) 实用程序来找到它。

### 让 Windows 7 回到您全新的 Linux 桌面上

假设您的系统可以处理它，在 Linux 桌面上虚拟化 Windows 7 是一种很好的方式，可以让操作系统在没有 Linux 替代品来满足您的任何需求的时候继续运行。为此，我们将使用 VirtualBox 直接在 Linux Mint 中运行 Windows 7 的一个实例。听起来很复杂，其实不然。

要开始，请打开 Linux Mint 的软件管理器。您应该会在首页看到 VirtualBox 的列表；如果没有，搜索一下。

安装这个应用程序很简单——只需点击按钮，按照提示的要求去做。

启动 VirtualBox，你会看到一个枯燥的空白界面。通过点击 New 按钮，选择 Windows 7 作为你的操作系统，并给你的虚拟机起一个有用的名字来解决这个问题。

系统会询问您希望为虚拟机分配多少内存。Linux Mint 至少需要大约 2GB 才能顺利运行，如果你运行的是 32 位版本的操作系统，Windows 7 应该至少有 1GB 的内存，如果你升级到 64 位版本，应该有 2GB 的内存。

接下来，您将被要求为 Windows 7 创建一个硬盘。同样，64 位版本的操作系统(最低 20GB)需要的空间比 32 位版本的操作系统(最低 16GB)稍多。您还需要考虑要在 Windows 7 实例上存储多少其他内容，如数据、应用程序等，并做出相应的决定。

如果你选得不好，不要惊慌失措。您可以稍后嵌入此虚拟硬盘。生活还要继续。

一旦你点击“创建”，你就完成了一半。您需要在列表中单击您的虚拟机，然后单击“设置”图标。这里有很多可以玩的东西，但是我建议查看两个关键部分:“系统”，如果您想提高虚拟机的速度，它将允许您为虚拟机分配额外的处理器，以及始终至关重要的“存储”。

一旦你进入“存储”，你会想要花一点时间在 Linux Mint 中打开一个网页浏览器，导航到微软的网站，在那里你将能够下载 [一个完整的磁盘映像](https://www.microsoft.com/en-us/software-download/windows7) (。ISO)的 Windows 7。是的，为此您需要您的产品密钥。

一旦你下载了那个。ISO 到 Linux Mint，回到 VirtualBox 的存储设置。单击“存储设备”下的光盘图标，然后单击窗口右侧外观相似的光盘图标。选择“选择虚拟光盘文件”，然后找到 Windows 7 .ISO。

你可以在设置中做更多的事情，比如共享一个来自 Linux Mint 的文件夹(比如“下载”)，如果你愿意，你可以在 Windows 7 中找到它，但是你现在已经检查了所有的基本要求。您可以通过主屏幕上的绿色大“开始”箭头启动虚拟机，开始安装和配置新的 Windows 7 系统。

之后，回到最基本的:确保你已经从 Windows Update 获取了你需要的任何更新，并且已经安装了你打算使用的任何应用。我的建议是，不要再次被 Windows 7 迷住。最初，这个操作系统可能看起来比 Linux Mint 更熟悉，但是如果你能在 Linux 中完成和在 Windows 中相似的任务，选择后者。首先，你的性能会更好，而且你可以在一个更安全的操作系统中做任何事情。

### 帮助您开始使用 Linux Mint 的应用程序

如果你还有点“我下一步做什么？”每当你推出 Linux Mint，我都理解。让我们用 [让你了解一些有用的应用](https://lifehacker.com/how-to-install-software-on-linux-1822447190) 。如果我想像使用 Windows 一样使用 Linux Mint，我会点击前面提到的软件管理器，并获取这些应用程序:

*   网络浏览器: Chromium 浏览器(或 Firefox)
*   媒体播放器: VLC(或 Spotify，流媒体)
*   **音乐库组织者:**克莱曼婷
*   **云存储:** Dropbox
*   **图像编辑器:** Gimp
*   **电子邮件:**雷鸟
*   **Wine** (用于在 Linux 上运行一些 Windows 应用)
*   文字处理/电子表格:图书馆办公室
*   **密码管理器:** 1Password X 或 Keepassx
*   **BitTorrent:** Qbittorrent

当然，也有很多其他值得安装的(Steam？不和？).这些都是基本的，但是 Linux Mint 的软件管理器使得查找和安装更多软件变得非常容易。
