# 教孩子“打小报告”和“报告”的区别

> 原文：<https://lifehacker.com/teach-kids-the-difference-between-tattling-and-reportin-1830251957>

我从来不喜欢“打小报告”这个词也许是因为我的俄亥俄州东北部口音确实弱化了“a”这个硬音，使得这个词本身听起来几乎和练习一样令人讨厌。**。**

**Watch**

**或者可能是因为当你的孩子第一次开始向他的兄弟姐妹和朋友告状时，很难解释为什么你不想知道有人在做他们不应该做的事情。毕竟，作为父母，我们基本上整天都在告诉他们停止做他们不应该做的事情。**

**不，我们不想知道约翰尼用了一个不好的词，即使我们会告诉他，如果我们自己听到了。但是等等，是的，我们确实想知道约翰尼是否在玩火柴。把他的手指放在你脸上？ *Nah* 。想爬上屋顶？*是的*。**

**我的儿子是独生子，但是我和我的丈夫做过几次养父母。在我儿子五岁的那一年，我们照顾着一个四岁的孩子，流言蜚语在我们家蔓延开来。最后，为了有意识地避免说“t attling ”,我开始使用这个短语，“有人感到悲伤或不安全吗？没有吗？回去玩吧。”但即使这样也不完全正确。很多讨厌的事情会让一个人难过。**

**我最近在 Reddit 上偶然发现一位家长也有同样的困境。“3InaRow”有一个四岁的儿子，他承认自己的八卦很有用:**

> **老实说，我很喜欢！有时候我需要去洗手间或者从楼上拿东西，他已经尽量减少了家庭活动室里的混乱，因为他在事情变得太糟糕之前警告了我。**

**一位名叫“恐龙靴”的小学教师，用我所见过的关于打小报告和举报的最佳解释之一跳了出来:**

> **报告是有益的信息，孩子觉得分享这些信息很重要(欺凌弱小、制造混乱等)。).通常你也会发现那很有用，即使那不是什么大事。打小报告只是想让别人陷入困境。也许这是真的，被打小报告的孩子不应该做他们正在做的事情，但也许这真的没那么重要。**

**“DarlinMermaidDarlin”还为孩子们提供了一系列问题，让他们在打小报告之前问自己:**

*   **是真的吗？**
*   **有帮助吗？**
*   **重要吗？**
*   **有必要吗？**
*   **善良吗？**

**“2007 年 11 月”的另一个简单而直接的解释是:**

> **搬弄是非:只是想让别人陷入困境。告诉:寻找某事/某人的好处。**

**这是最后一个定义，用我八岁儿子的话来说:“打小报告就像，‘哇，他拿了我的玩具！这不是什么大不了的事情，你可以自己处理。报告是很重要的事情，比如当有人流血或呕吐时。”**
