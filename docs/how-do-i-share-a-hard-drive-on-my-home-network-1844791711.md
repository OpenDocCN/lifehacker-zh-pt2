# 如何在家庭网络上共享硬盘？

> 原文：<https://lifehacker.com/how-do-i-share-a-hard-drive-on-my-home-network-1844791711>

我永远不会责怪问了一个感觉“基本”的技术问题的人，因为我们都必须从某个地方开始。我们中很少有人出生时就知道如何在 Windows 中访问网络共享，什么是工作组，或者如何映射网络驱动器。



在本周的[**Tech 911**](https://lifehacker.com/c/tech-911)T5】Q&A 中，Lifehacker 读者 Matthew 提出了一个相对简单，但却很重要的关于家庭网络的问题。让我们开始吧:

> “我对建立一个非常基本的家庭网络很感兴趣——实际上只是一个共享硬盘，我可以从家里的任何一台计算机上访问它——但是我不知道如何开始。
> 
> *简单到只需将外置硬盘插入路由器背面的 USB 端口即可吗？这似乎太容易了。还是我需要一个通过以太网端口连接的特殊硬盘？如何将它配置为“出现”在我的设备上？我不期望它是自动的，但是我不熟悉使它工作所需的配置设置。是需要特殊的软件，还是所有必要的工具都内置在 Windows 中？*
> 
> 我渴望了解更多。"

### 如何在 Windows 中设置驱动器和/或文件夹共享

Matthew，我很高兴你能对广阔的网络世界有更多的了解。我是从报道硬盘的科技新闻开始的，并在 *Wirecutter* 用我的路由器指南大获成功。你的问题激起了我的两大兴趣，我很乐意帮助你。

您有两种主要的选择来与家中的其他计算机共享硬盘。您可以将此硬盘连接到您的台式 PC(内部或外部)，或连接到您的笔记本电脑(可能是外部)，然后通过 Windows 将该硬盘共享到您的网络。你所需要的一切都直接内置在 Windows 中，我很高兴现在就带你完成这个过程。

为了便于讨论，并且因为我周围没有备用的外部机箱，我将假设您已经以某种方式将该驱动器连接到您的系统。如果您使用系统的一个免费 SATA 端口，您将获得最佳性能；通过 USB 3.0 或 USB-C 连接，外置硬盘的性能最佳，而且更加便携，但性能可能不会那么好。

现在你已经完成了，你可能需要点击 Windows 的计算机管理屏幕来给你的硬盘分配一个字母。(如果您购买了自带的外置硬盘，它可能已经为您初始化，您可以在文件资源管理器中找到它。如果没有，请继续这些步骤。)

在你的开始菜单中输入**“计算机管理”**，然后在出现的屏幕中点击**“存储”**下的**“磁盘管理”**:

在此窗口的底部找到您的驱动器，如果它是一个全新的驱动器，它应该被标记为“未分配”空间。右键单击，选择“New Simple Volume”，并在屏幕上单击，直到您可以选择为其分配一个驱动器号(无论您想要什么)和一个文件系统(选择“NTFS”)。

现在，当你打开文件浏览器，你会看到你的新硬盘，准备好所有你想转储的数据。但是你真的想在你的网络上分享它，这需要我们再多跳几圈。

你现在应该可以在文件浏览器中右击任何文件夹或驱动器，选择**“授予访问权限”**，然后点击**“高级共享”**

在出现的屏幕中，点击**“高级共享”**按钮，勾选**“共享此文件夹”**框、，随意给你的驱动器或文件夹起一个令人兴奋的名字。

完成这些步骤后，我在另一个连接到我的网络的 Windows 系统上打开了文件浏览器。尽管我在这两个网站上都登录了同一个 Microsoft ID(这对于这个场景来说是一个重要的事实),但一开始我看不到这个网络共享。我不得不点击地址栏，输入“\”，然后输入我的第一个系统名(“圣骑士”)来调出共享。从那里，很容易右键单击它并选择**“映射网络驱动器”**，这使它总是出现在文件浏览器中我的第二个系统的驱动器列表的下面。

(如果您忘记了第一个系统的名称，只需访问**控制面板>系统**并查找您的“设备名称”)

明白了吗？很好。请记住，使用这种方法，用户必须在相关的原始系统上拥有帐户，才能访问共享驱动器/文件夹。否则，你必须去你的**控制面板>网络和共享中心>更改高级共享设置**，关闭**“所有网络”下的**“密码保护共享”****这个*应该*然后让你网络上的任何人访问这个文件夹，这可能是你在家里大多数情况下想要做的。

### 选项二:专用的网络连接存储设备

既然我们已经讨论了选项一——这是一次怎样的旅行——我们来讨论选项二:网络连接存储。

前几天我读了一篇文章，有人在网络存储上拉屎，我甚至无法看完，因为他们令人难以置信的大量错误开始渗透到我的台式电脑中，让我感到焦虑。我当然是在开玩笑，但是他们仍然是非常不正确的。他们认为网络连接存储是愚蠢的，最好使用 USB 连接存储，并通过 Windows 设置软件 RAID。

当然，你可以走那条路。是的，NAS 盒可能会变得很贵，尤其是当您考虑到驱动器的成本时。但是他们也不一定非得是的高价。我认为它们是一种投资。至少，我在 2013 年购买的 QNAP NAS 盒子仍然很强劲。我只升级了内存，这花了我大约 20 美元左右，用它来存储万亿字节的数据和播放太多(合法)电影没有任何问题。

NAS 盒允许您将硬盘驱动器放入一个易于使用的外壳中，这样可以容纳它们并保护它们的安全。你可以把 NAS 盒放在家里任何你想放的地方，把它连接到你的网络上，它包含的操作系统应该可以完成大部分的工作。它会找到你的驱动器，帮助你初始化它们，并给你一个简单的方法来分享它们的内容。我总是发现在我的 NAS 设备上为所有的 vario us 文件夹设置共享非常容易，如果我需要的话，为每个文件夹的不同用户设置访问限制也是轻而易举的事情。

拥有 NAS 盒的另一个好处是，每当我想与另一个系统共享一些东西(或在家里四处传播)时，我不必启动我的台式机或笔记本电脑。在我概述的第一种情况下，你总是需要启动你的台式机或笔记本电脑系统，以便其他人访问你的共享文件夹。如果你，比如说，在他们复制文件的时候重启你的系统——或者如果系统崩溃了——你只是把它粘在了那个人身上。

如果你愿意，你可以让一个 NAS 盒子整天开着(我怀疑它会比一台增强的台式机或笔记本电脑消耗更少的资源，特别是如果你启用它的省电模式)。或者，您可以使用任何像样的 NAS 盒的设置来根据您想要的任何时间表打开和关闭它。如果你愿意的话，你甚至可以使用手机应用程序来唤醒和睡眠你的 NAS，这比台式机或笔记本电脑容易处理得多。

如果您希望将多个驱动器组合成一个巨大的存储阵列(或者包含额外冗余以在驱动器出现故障时保护您的阵列)，我认为 NAS 盒比任何一种基于 Windows 的 RAID 设置都好得多，您可以将它应用到更简单的基于 USB 的外部存储设备。

### 如何决定哪种共享设置适合您

让我们言归正传。如果您需要一个随时可用的网络共享，供家中多人访问，我会选择专用的网络连接存储设备。我不会把我的信任放在一个基于 USB 的 24/7 全天候使用的外壳上，也不会让我的台式电脑(尤其是笔记本电脑)整天运行。在这种情况下，您还不如投资购买一台低功耗的服务器，而 NAS 设备基本上是一种简单易行的方式。

然而，如果你认为你不需要那么大的火力，并且你只是偶尔需要从你的其他计算机和设备访问硬盘驱动器的内容，我会首先简单地从我的桌面 PC 添加和共享驱动器。如果这不是一个选项，我会寻找一些外部的东西，我可以连接到任何我想要的系统。

当然，如果你这么做了，为什么还要对网络共享大惊小怪呢？如果你家里有人需要“通用 USB 驱动器”上的文件，只要走过去，拿到它，插上电源。或者，你可以使用共享的 Google Drive 账户、Dropbox 或任何你喜欢的其他 [存储服务](https://lifehacker.com/google-one-is-now-open-for-everyone-but-is-it-a-good-d-1826049257) 在系统之间共享你的文件。见鬼，你甚至可以在家里创建自己的文件同步设置，不需要使用任何额外的硬盘或网络设置。

但这至少是有很多选择的乐趣所在；现在，您可以选择一个最适合您的解决方案。理想情况下，它是为您节省最多时间、金钱和技术难题的方法。如果你还有后者，请告诉我，我会很乐意继续帮助你。
