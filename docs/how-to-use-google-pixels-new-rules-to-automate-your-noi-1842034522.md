# 如何使用谷歌 Pixel 的新“规则”来自动化你嘈杂的 Android

> 原文:[https://life hacker . com/how-to-use-Google-pixels-new-rules-to-automate-your-noi-1842034522](https://lifehacker.com/how-to-use-google-pixels-new-rules-to-automate-your-noi-1842034522)

谷歌 Pixel 用户将发现一个新的 [【规则】功能从今天](https://www.blog.google/products/pixel/more-pixel-features-dropping) 开始添加到他们的设备中。规则类似于 [iOS 快捷键](https://lifehacker.com/try-out-iphone-automation-with-this-list-of-150-ios-sh-1833153395) ，允许你在特定位置时创建自动设置更改的开关——尽管相比之下，Android 规则更多地局限于快捷键。目前，您只能设置规则:

Watch

*   打开“请勿打扰”模式。
*   将通知设置为静音。
*   将通知设定为振动。
*   将通知设置为响铃。

新的“规则”菜单还有一个选项，可以根据您使用手机和更改设置的方式来打开规则建议，例如，如果您的手机注意到您离家时经常调低铃声音量，它可能会建议您创建一个规则来静音室外通知。

至于手动设置您自己的规则:

1.  进入**设置>系统>高级>规则**(您需要授予规则在后台运行的权限，并在第一次打开规则页面时对您的设置进行更改。)
2.  点击**“添加规则”**
3.  点击**“添加位置的 Wifi 网络”**选择设置更改的触发位置。你可以在保存的网络列表中进行选择，或者通过谷歌地图搜索找到一个位置。您还可以修改该位置周围的影响半径区域。
4.  最后，页面底部还有一个选项，允许您在规则发生变化时启用推送通知。
5.  完成所有设置后，点击菜单底部的**“添加”**完成规则设置。
6.  您制定的规则会列在**设置>系统>高级>规则下。**您可以通过点击规则旁边的垃圾桶图标来删除规则。

新的规则菜单包含在最近的 [谷歌像素功能下降](https://www.blog.google/products/pixel/more-pixel-features-dropping) 中，它本身是 2020 年 3 月 Android 像素设备安全补丁的一部分。你现在可以去**设置>系统>高级>系统更新下载。**
