# 使用此扩展阅读网站时，查看单词的定义和发音

> 原文：<https://lifehacker.com/view-definitions-and-pronunciations-for-words-while-you-1835025366>

即使是我们当中最热心的读者，偶尔也会碰到我们不太知道意思的单词。在这种情况下，我的典型做法是将这个词复制/粘贴到另一个标签页的谷歌中，看看会出现什么。本周，我遇到了一个 Chrome 扩展，它去掉了中间的人(或中间的标签)，而是在同一个页面上处理操作。



谷歌词典名为，“一旦安装，你只需双击网站上的任何一个单词，就可以调出这个单词的定义。定义将出现在现有文本上方的浮动方块中。如果你想知道它是怎么说的，你也可以弹奏它的发音。

除了允许你当场学习一个定义之外，这个扩展还允许你存储你已经查找过的单词的历史，允许你以后参考它们，这样你就有可能永远记住它们。您还可以允许其他第三方扩展访问历史记录，以做一些类似创建抽认卡的事情，供您复习。

它也适用于外语单词，并会自动提供你选择的语言的定义。该历史记录可以随时以 CSV 格式下载。

支持的语言包括阿拉伯语、巴西葡萄牙语、中文、捷克语、荷兰语、法语、德语、印地语、意大利语、日语、韩语、俄语、斯洛伐克语、西班牙语和土耳其语。

总而言之，这是一个非常简单的扩展，但却是一个非常方便的东西。
