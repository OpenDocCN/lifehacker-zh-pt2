# 用每天的“屁股谈话时间”来抑制低级幽默

> 原文：<https://lifehacker.com/curb-potty-humor-with-a-daily-butt-talk-time-1839638124>

五岁的时候便便很有趣。好吧，我知道，所有年龄的人都可能喜欢一个好的屁笑话。但是偶尔，*可能会让*厌倦你孩子的无聊笑话——尤其是在餐桌上——如果你对此反应强烈，你可能只会怂恿他们。他们试图激怒你；这是一次权力之旅。



[纽约时报](https://parenting.nytimes.com/preschooler/kids-potty-humor?module=editors-picks&action=click&region=2) 解释道:

> “首先，孩子们知道单词有意义，然后他们知道某些单词有特殊的力量，”劳伦斯·科恩博士说，他是《》一书的作者他说，身体机能有一种特殊的强度，这种强度延伸到描述它们的词语上。使用浴室有强烈的感官体验，结合“非常秘密的隐私和秘密”,孩子们看到成年人对这些话的反应。
> 
> “孩子们总是在尝试权力，让成年人大笑或尴尬，这对孩子来说是一件非常强大的事情，”他说。

对于便壶式幽默在 3 至 6 岁年龄段引起他们的兴趣，还有其他解释:他们实际上可能对浴室事故感到焦虑，他们可以通过笑话释放一些焦虑。他们也处于一个发展阶段，在这个阶段荒谬是有趣的，对学龄前儿童来说，没有什么比身体机能更荒谬的了。

如果你同意便便和撒尿是完全滑稽可笑的，那么无论如何，让我们跟着它们一起撕吧。不过，如果你想把它拨回来，临床心理学家玛雅·科尔曼有一个建议，她与《T2 时报》 分享:实行每天“屁股谈话时间”

> 它是这样运作的:每天五分钟，任何事情都是游戏。孩子们被鼓励说任何事情，不管有多下流或粗俗，父母应该用笑声和游戏来支持。如果这个笑话很有趣，就笑吧。如果你的孩子叫你一张臭脸，你可能会假装吃惊地说，“什么？我以为我是一朵娇嫩的花。”其效果是减少这些词的情感负荷。

科尔曼说，这也有助于在不恰当的时候(比如餐桌上或教堂)引导孩子。她只是告诉他们，“哎呀，我们需要把这个留到‘屁股谈话时间’去。”"
