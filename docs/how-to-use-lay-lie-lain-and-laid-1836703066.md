# 如何使用躺，躺，躺，躺

> 原文：<https://lifehacker.com/how-to-use-lay-lie-lain-and-laid-1836703066>

偶尔，你会想躺下来小睡一会儿，对吗？不，你没有。你刚刚犯了混淆你的谎言和谎言的罪。



(在我们开始之前，你可能想参加 [语法女孩的六题谎言与谎言测试](https://www.quickanddirtytips.com/lay-versus-lie-quiz) 。然后你可以在最后再考一次，看看自己提高了多少。)

“lay”这个词总是和一个宾语成对出现；意思是放下东西。一个常见的例子是你把一本书(物体)放在桌子上。撒谎不需要目标；意思是采取休息的姿势。你躺下(没有对象)好好休息一晚。

### “把它放在我身上”

如果你不能直接记住这条规则，记住“撒谎”和“说谎”之间区别的一个流行的记忆方法是想出这样一句话，“把它放在我身上。”“Lay it on me”是一种俚语说法，“跟我说说吧”，但它也可以提醒你 *lay* 与一个物体(它)搭配。

### 这就是事情变得棘手的地方

还不算太糟，对吧？但是现在如果需要用 lay 或者 lie 的过去式或者过去分词呢？是啊，这就是风险所在。

#### 撒谎/躺下/躺着

Lie 是现在时态。我要去躺下。

Lay 是过去式。昨晚，我躺得太早，无法入睡。

Lain 是过去分词。她彻夜未眠。

#### 铺设/铺设/铺设

Lay 是现在时。我应该把婴儿放在婴儿床里。

Laid 是过去式。我把邮件放在厨房的桌子上。

而 laid 是*也是*的过去分词。*我每周都把报告放在同一个地方。*

我不能帮你记住这一点；你只需要大量练习使用它，直到它成为你的第二天性。把它写下来，放在你的电脑旁边，或者在浴室的镜子上贴一张便利贴，坚持一周，把它记在心里。

然后你就可以一劳永逸地结束整件事了。
