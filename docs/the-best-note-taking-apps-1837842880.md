### [Evernote](https://evernote.com/)T3】

在我们对竞争者的呼吁中，Evernote 获得了最多的选票，但它也有一些可以理解的警告，我们一会儿就会谈到。

首先，好处:Evernote 有各种各样的选项来让你的想法和工作有条理，并支持文本、音频和视频笔记。但是 Evernote 最好的特性可能是它强大的跨平台支持，允许你从多种设备和网络浏览器访问你的所有文档。Chrome、Firefox 和 Safari 也可以配备 Evernote Web clipper 插件，该插件可以将整个网页保存到您的文档中以便快速访问，而 iOS 版本包括 Siri 集成。如果你想获得跨平台集成的更细粒度，15 美元的 Evernote 高级版支持 Google Drive、微软团队、Outlook、Slack 等。高级版还包括额外的有用功能，如离线笔记访问。

显然，Evernote 在纸面上是一个很好的选择，我们也同意 Evernote 是同类应用中整体表现最好的。我们还建议将 Evernote 与 Saferoom 配对，后者在这个列表中有自己的名字。

### [OneNote](https://products.office.com/en-us/onenote/digital-note-taking-app)T3】

[微软的 OneNote](https://products.office.com/en-us/onenote/digital-note-taking-app) 也得到了不少好评，这并不奇怪；就像 Evernote 一样，OneNote 在任何笔记应用程序列表中都排名靠前。

它也赢得了那些推荐。OneNote 不仅支持文本和网页剪辑，如果您使用平板电脑或触摸屏，还可以手写笔记和扫描物理文档。

<label class="bxm4mm-13 juykRM">Advertisement</label>

由于这是微软的产品，你将能够在与你的微软帐户相关的所有设备上同步你的笔记，包括 iOS 和 Android 设备。但是，即使你没有登录到同一个帐户，你仍然可以享受与 Microsoft Office 应用程序(如 Excel 和 Word)的原生集成。这是一个很好的选择，也是 Evernote 的可行替代方案，尽管我们会再次推荐它和 Saferoom 一起使用，你可以在下面读到。

### [Zoho 笔记本](https://www.zoho.com/notebook/)

像这里列出的其他应用程序一样， [Zoho Notebook](https://www.zoho.com/notebook/) 支持许多类型的笔记和媒体嵌入，但它以灵活的列表和组织选项与众不同。大多数其他应用程序只是在一个位置堆叠或收集您的笔记，并限制组织或工作流程选项，但 Zoho Notebook 使用颜色编码和可定制的功能来创建不同类型的笔记，如清单、电子表格、媒体剪辑等。每种类型的笔记都有自己的特性、视觉设计，甚至还有特定的手势控制，用于快速的 UI 导航。

Zoho Notebook 在 Android、iOS、Mac 和 web 客户端上是免费的，尽管目前没有 Windows 应用程序。

### [收纳盒纸](https://www.dropbox.com/paper)

[Dropbox Paper](https://www.dropbox.com/paper) 不像列表中的其他一些应用程序那样具有相同的跨平台支持或功能广度，但它是协作的绝佳选择，因为它允许多个用户访问和编辑相同的笔记。

它只能在 Android、iOS 和网络浏览器上使用，但它集成了主要的 Dropbox 云存储服务，允许你将文件从 Dropbox 文件夹添加到你的笔记中。

### [安全屋](https://getsaferoom.com/)

<label class="bxm4mm-13 juykRM">Advertisement</label>

好吧——从技术上来说， [安全室](https://getsaferoom.com/) 本身并不是一个笔记应用，下载并安装它本身不会做太多事情。相反，Saferoom 用于加密 Evernote 或 OneNote 等应用程序中的笔记，甚至是 Microsoft Word 文档中的笔记。

您的文件通过零知识密码保护进行加密—基本上，您创建一个只有自己知道的密码。密码不存储在其他地方，但 Saferoom 确实支持像 LastPass 这样的 [密码管理器应用](https://lifehacker.com/the-five-best-password-managers-5529133) 。不过，请确保不要忘记您设置的任何密码，因为如果您忘记了，就没有办法恢复您的密码或加密文件。

尽管笔记应用本身并不是一个笔记应用，但我们已经将它包括在我们的列表中，因为很少有笔记应用拥有内置的加密或隐私设置——你知道我们有多喜欢我们的数据隐私解决方案。

### 其余的

这里有几个，咳咳，值得注意的应用程序，只获得了少量的投票，但仍然值得一试。

<label class="bxm4mm-13 juykRM">Advertisement</label>

[**simple note:**](https://simplenote.com/)想让事情变得又快又好，简单？使用简单的笔记。UI 是最小化的(以一种好的方式),基本上你可以用 Simple Note 做的唯一的事情就是创建和编辑新的笔记，但是如果这真的是你所需要的，它可能是最好的选择。此外，您的笔记是云同步的，因此您可以从多个设备访问它们。当然，其他应用程序可以做得更多，并打包了大量的组织工具，但 Simplenote 非常适合快速记下想法或写下最后一分钟的购物清单。简单笔记可以在 Android、iOS、Mac、Windows 和 Linux 上使用。

[**Joplin**](https://joplinapp.org/)**:**Joplin 是一款免费开源的笔记应用，可在 Windows、Mac、Android、iOS 和 Linux 上使用。它可以让你创建多种类型的笔记、提醒和提醒，所有这些都可以与 Dropbox、NextCloud、OneDrive 和 WebDAV 等云服务同步。该应用程序还包括一个 web clipper 工具，甚至支持手写笔记，但在我们看来，Joplin 的最佳功能是内置的端到端加密，用于保护您的笔记隐私。

**Google Keep / Apple Notes:** 尽管是具有不同功能的不同应用，但 Google Keep 和 Apple Notes 分别作为 Android 或 iOS 上的股票笔记解决方案工作得很好。两者都在同一操作系统的其他设备上与云同步，并且都集成了各自母公司的专有服务(Google Drive for Keep 和 iCloud for Apple Notes)。这个列表中的其他应用程序可以扩展你的功能集，并允许你从更广泛的平台上访问你的笔记，但 Google Keep 和 Apple Notes 仍然是很好的选择。
