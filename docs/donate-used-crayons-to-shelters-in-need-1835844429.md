# 向有需要的收容所捐赠用过的蜡笔

> 原文:[https://life hacker . com/把用过的蜡笔捐给需要帮助的收容所-1835844429](https://lifehacker.com/donate-used-crayons-to-shelters-in-need-1835844429)

去任何一家有儿童菜单的餐馆，你很可能会看到蜡笔散落在桌子上，一天用完就扔掉。

Watch

虽然蜡笔算不上是主要的污染源，但它们对环境也没什么好处；大多数是由石蜡制成的，这种物质你可能会在 [美容产品](https://www.amazon.com/Paraffin-Wax-Works-Green-Treatment/dp/B00SLMLOSQ?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/donate-used-crayons-to-shelters-in-need-1835844429&asc_source=&tag=kinjalifehackerlink-20) 中找到，是原油或石油的副产品。它们不会生物降解，这意味着它们会在未来几年里留在垃圾填埋场。

幸运的是，蜡笔可以很容易地重新利用，以减少浪费，并捐赠给一个良好的事业。不要扔掉那些坏掉的或用过的蜡笔，试着把它们捐赠给像蜡笔倡议 这样的项目，这些项目在任何情况下都接受蜡笔；这些蜡笔随后将被熔化，再制造成新的蜡笔，最终分发给全美的儿童医院

[运送](https://thecrayoninitiative.org/wp-content/uploads/2018/08/170623_TCI_UPS-Shipping_R00.pdf) 它们相当简单；购买(或重复使用)一个箱子，从蜡笔倡议的 [网站](https://cvshipping.ups.com/Announcements.asp) 创建一个 UPS 运输标签，填写在线表格(这将要求你计算箱子的重量)，然后将箱子放在任何一家 UPS 商店。

类似地，其他程序如 [疯狂蜡笔](https://crazycrayons.com/recycle-program/) 将接受用过的蜡笔，这些蜡笔将被重新利用并在他们的商店中转售。

如果你想在当地产生影响，你也应该在网上搜索你附近的收容所、医院或其他接受蜡笔的设施(或现有的存放容器)。如果你有很多坏掉的蜡笔，你甚至可以自己 [再利用](https://www.thesprucecrafts.com/how-to-recycle-old-crayons-into-new-crayons-1388486) 它们，将它们熔化后倒入模具或松饼罐中。
