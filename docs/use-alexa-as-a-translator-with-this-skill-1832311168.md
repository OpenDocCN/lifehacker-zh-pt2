# 使用 Alexa 作为这个技能的翻译

> 原文：<https://lifehacker.com/use-alexa-as-a-translator-with-this-skill-1832311168>

在过去的几年里，我开始在世界各地旅行，这意味着学习新的语言。



虽然我肯定已经到了一个年龄，流利地掌握一门全新的语言是不可能的(老实说，我 18 岁时就是这样)，但我总是在去一个新地方之前努力学习一门语言的基础知识。并学习如何正确地发音。如果你在同一条船上，这个 [Alexa 技能翻译过来的](https://www.amazon.com/Translated-Labs/dp/B01N9BZJPZ?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/use-alexa-as-a-translator-with-this-skill-1832311168&asc_source=&tag=kinjalifehackerlink-20) 可以帮上忙。

有了它，你可以用英语说些什么，并要求翻译成另一种语言。

它目前支持美国西班牙语阿拉伯语、巴斯克语、巴西葡萄牙语、加泰罗尼亚语、中文、捷克语、丹麦语、荷兰语、芬兰语、佛兰芒语、法语、加拿大法语、加利西亚语、德语、希腊语、希伯来语、印地语、匈牙利语、冰岛语、印度尼西亚语、意大利语、日语、韩语、挪威语、波兰语、葡萄牙语、罗马尼亚语、俄语、斯洛伐克语、西班牙语、西班牙语拉丁美洲、瑞典语、瑞士德语、泰语、土耳其语和威尔士语。

该应用程序只适用于将英语翻译成另一种语言，而不是相反。如果你理解某事有困难，你也可以要求它说得慢一点，这样你就能听懂。

这种技能不会教你一门新的语言，为此我推荐 Duolingo，但它可以帮助你学会如何快速正确地说出一些东西，或者快速解决关于一些东西的正确发音的争论，这比拿出谷歌翻译要容易得多。
