# 玩谷歌涂鸦游戏档案中的所有游戏

> 原文：<https://lifehacker.com/play-all-the-games-in-googles-doodle-games-archive-1843109788>

当我们都被困在室内时，谷歌正在复活旧的谷歌涂鸦游戏来娱乐我们。在接下来的两周里，这个搜索巨头计划将一些老的流行涂鸦作为其每日特色涂鸦。但是如果你不介意做一些挖掘，你也可以在谷歌的档案中找到大量的游戏。



### 对于懒人:只需每天访问谷歌

这些回归的涂鸦游戏中的第一个是为胡萝卜 编码的 [，它是为了教孩子们(以及任何年龄的编码新人)关于编程和游戏开发而开发的。玩家引导一只兔子在各种类似益智游戏的关卡中前进，使用拖放界面来“编码”兔子的动作，沿途收集胡萝卜。这是对基本脚本语言的有趣而有效的介绍。](https://www.google.com/doodles/celebrating-50-years-of-kids-coding)

你可以通过桌面浏览器上的点击“谷歌”标志，为胡萝卜和任何其他返回的涂鸦进行编码。这些涂鸦将在未来几周内每天循环播放，所以请继续查看当天的内容。哦，如果你错过了一个也不用担心；所有归还的涂鸦都可以被找到，并在重新发布后通过点击谷歌标志和滚动搜索结果顶部的列表来重放，而任何剩余的涂鸦都会显示为“即将到来”

下一场比赛是板球 ，它最初也是在 2017 年举行的，以庆祝当年的 ICC 冠军奖杯锦标赛。玩家在快节奏的板球迷你游戏中竞争，这些游戏的特色是团队——还有什么？—蟋蟀。在接下来的两周里，请注意这一点，还有更多涂鸦游戏。就我个人而言，我希望 [大食尸鬼决斗](https://www.google.com/doodles/halloween-2018) 卷土重来，我有几笔帐要算。

### 对于专门的:通过谷歌涂鸦档案爬行

说到这个，如果你碰巧知道过去几年中你最喜欢的任何涂鸦的名字，包括永远有趣的，你总是可以自己在谷歌的 [巨大的涂鸦档案](https://www.google.com/doodles#archive) 中搜索它们。那包括经典的像 [快画](https://quickdraw.withgoogle.com/) ， [侏儒折腾](https://www.google.com/doodles/celebrating-garden-gnomes) ，甚至还有这个疯狂的 [魔方](https://www.google.com/doodles/rubiks-cube) 。你甚至可以 [做音乐](https://www.google.com/logos/doodles/2017/fischinger/fischinger17.9.html?hl=en&doodle=undefined) ，如果你愿意， [完成一个填字游戏](https://www.google.com/logos/2013/crossword/crossword13.html) ，或者玩 [不-真-Flappy-Bird](https://www.google.com/doodles/halloween-global-candy-cup-2015) 。
