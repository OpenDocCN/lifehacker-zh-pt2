# 在家为你的孩子举办“主题周”

> 原文：<https://lifehacker.com/host-theme-weeks-for-your-kids-at-home-1843111667>

在疫情的这个时候，你的孩子可能会错过学校，你肯定会错过你的孩子过去上学的时间。现在日子都混在一起了，学校能够以一种我们以前无法完全欣赏的方式改变事情。但是你可以从你学校的“让事情保持有趣”剧本中吸取经验:把经典的学校精神周的想法带进你的家里，稍微改变一下——主题周。

Watch

第一步是确保每个人都参与到游戏中来。在我们城市目前的居家订单还剩大约四周的时候，我让每个人在一张纸上写下一个主题想法，然后把它们放进一顶帽子里。然后，我儿子从帽子里抽出了第一个建议。

在过去的两周里，我们度过了三明治周(我的建议)和任天堂周(我女儿的提议)。这些想法伴随着主题食物、活动和——也许是最重要的——每天期待的东西。

在精神周期间，你的孩子可能会梳着疯狂的头发或穿着最喜欢的球队球衣去上学。每一天通常都有一个建议的活动或存在的理由。模仿这个想法，试着每天做一件事(食物、活动、电影、游戏或手工)来庆祝本周的主题。这样做的目的是增加一点乐趣或分散注意力，而不会给你的日程增加压力。

三明治周当然是以食物为中心。我们吃了香肠三明治。我们做了香蕉面包。我们温和地辩论热狗是否是三明治，这导致热狗和烤面包成为我女儿画的一本小杂志的明星。

剩下的鸡肉沙瓦玛重生成了一个三明治，里面有山羊奶酪、萝卜和全谷物芥末。我们在任天堂 Switch 上玩了《烹饪过度 2》——在这个游戏中，你们必须像厨师一样在厨房里合作，以满足不断增长的订单。只以泪洗面过一次。

任天堂周不仅仅是关于技术——尽管它开始于额外的屏幕时间。Just Dance 2020 免费开放了一个月的无限图书馆，我妻子让孩子们在锻炼模式下运动 30 分钟。我从我儿子那里得到了一个关于在 YouTube 上看别人玩视频游戏的教程。接下来，我们将建造 [八位灵感乐高壁画](https://www.instructables.com/id/LEGO-Mario-Sprite-Mosaics/) ，并装饰像超级马里奥人物一样的糖饼干。

我们仍然有神奇宝贝周(我打赌我们会做另一个 [神奇宝贝狩猎](https://lifehacker.com/how-to-organize-your-own-pokemon-hunt-1842858176) )和历史周。当考虑不同的主题时，选择可以有很多方向的更广泛的类别。在喜爱的电影、书籍、食物或流派中寻找你的主题。

*星球大战，*举例，就是为了光剑对战，尤达语音课和十几部电影准备观看而打造的。自制的 [黄油啤酒](https://thetakeout.com/wizarding-world-harry-potter-hot-butterbeer-taste-test-1832920999) 、适合散步和魔法课的有声读物可以让哈利波特周变得生动起来。神秘周可能会有一个 [字母寻宝游戏](https://lifehacker.com/buy-yourself-30-minutes-of-peace-with-an-alphabet-scave-1842593033) ，谜语和怪异的故事，在你的院子或客厅露营。

如果没有别的，主题可以帮助一周周过去，让每一天都感觉不同。
