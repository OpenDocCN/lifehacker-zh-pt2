# 将皇家糖衣放在调味瓶里，这样更容易装饰

> 原文:[https://life hacker . com/put-royal-ice-in-sauce-bottles-for-easy-decorating-1840342237](https://lifehacker.com/put-royal-icing-in-sauce-bottles-for-easier-decorating-1840342237)

装饰圣诞饼干是我非常想享受的一项活动，但我缺乏精细运动技能和短暂的注意力持续时间使它变得困难。填充——然后*使用*——裱花袋也相当可怕。没有什么窍门可以把艺术天赋灌输到这些没有天赋的手里，但是一个 [可挤压的调味瓶](https://skillet.lifehacker.com/you-need-to-get-some-squeeze-bottles-in-your-kitchen-1826541753) 比一个糕点袋更容易填充和使用。

Watch

即使你已经掌握了灌装那些薄薄的袋子( [一个大量杯](https://skillet.lifehacker.com/pumpkin-icebox-cake-is-a-perfect-thanksgiving-dessert-f-1839859088) 可以帮忙)，从 [餐厅供应店](https://skillet.lifehacker.com/the-top-5-restaurant-supply-store-essentials-ranked-1797727937) 抓一些挤压瓶还是有很多好处的。例如，如果你有孩子，我想你会发现容易握持和挤压的塑料瓶对他们的小手和发展协调感更友好。坚硬的墙壁意味着它们可以独立站立，让装填和储存变得无比简单。虽然我知道一些糕点和裱花袋在技术上是可以重复使用的，但我从来没有享受过清洁袋子的乐趣。

一旦装满，很容易控制挤压瓶上施加的压力，这让我尽我所能装饰饼干(这仍然不像，令人惊讶，但这是我的问题)。挤压瓶可以用于管道和注水，有些甚至带有完整的尖端，让你可以将它们剪成你想要的大小。一旦你完全冰透了，只需将剩余的糖倒入下水道，然后用另一种美味的液体(如鸡尾酒糖浆、沙拉酱或你最喜欢的酱)重新装满它们，这样就可以方便、精确地分配了。
