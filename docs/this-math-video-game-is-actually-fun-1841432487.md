# 这个数学电子游戏其实很有趣

> 原文：<https://lifehacker.com/this-math-video-game-is-actually-fun-1841432487>

几个月前的一天，我儿子放学回家，问他是否可以玩一个朋友告诉他的新游戏。这种情况一个月会发生几次——经常导致我研究游戏，然后用我的答案让他失望，因为*哇，不适合年龄*——所以我没有马上提交。但事实证明，这个名为 [神童](https://www.prodigygame.com/) 的特殊游戏，既是一个真正有趣的怪物战斗、探索游戏*又是一个数学技能游戏* 。



Prodigy 是一款奇幻风格的游戏，有点像神奇宝贝和《我的世界》创意模式的结合，中加入了一些数学方程式。为了赚钱定制他们的房子，购买宠物或赢得奖品，他们必须打败怪物。为了打败怪物，他们必须正确回答数学问题才能获得咒语。Prodigy 表示，这些问题与 1-8 年级的国家级课程相一致，包括共同核心和 TEKS，是根据每个孩子的优势和劣势量身定制的:

> Prodigy 使用分级测试来确定您孩子的技能水平，在您的孩子玩耍时向他们提出更难或更容易的问题，以找出他们的位置。
> 
> 放置测试会立即开始，并在游戏运行时在后台运行。它从您或您的孩子在设置帐户时选择的年级的下一个年级开始，然后向上确定您孩子的实际年级。测试结束后，难度水平会继续调整，根据孩子的表现重新考虑先决条件或向前跳跃。

我的儿子今年 9 岁，目前最感兴趣的是帮助一些叫做浮尸的生物重建家园，我想是？——被傀儡师消灭。我不知道，这个游戏里有很多东西:战斗、营救、宝石、神话生物、购物、装饰和去其他世界旅行。对我来说，这听起来像是另一种语言，但我可以从他谈论这个游戏的兴奋方式看出他对这个游戏很着迷。

至于能够与其他用户在线聊天，这一直是我最关心的问题之一，Prodigy 确实有消息选项，但它非常受限制:

> 虽然聊天和朋友列表不能被禁用，但您的学生总是可以在“离线”模式下玩游戏，在这种模式下，所有社交元素都被禁用，游戏由电脑玩家填充。这总是可以在登录后从世界选择界面访问。
> 
> 只有被认为安全的预定义句子被添加到游戏中。用户不能键入或发送他们想要的任何内容。这是出于学生/儿童的隐私和安全原因而设置的。任何其他用户都不能输入或看到任何个人信息或不适当的语言。

孩子们可以免费玩这个游戏，但是某些功能和选项在基本会员下是不可用的。因此，如果他们喜欢这个游戏，请注意，他们很快就会想升级到额外的高级会员资格(一年每月 4.99 美元，六个月会员每月 7.95 美元，定期按月计划 8.95 美元)。

我们从免费会员开始，我跟踪了他大约一个月的实际玩了多少。Prodigy 每周提供他们玩了多少，他们练习了哪些数学技能以及他们在不同类别中的熟练程度的总结。一旦我确信我的儿子会坚持下去，我就把他升级为每月会员。现在他已经拥有并使用了几个月(他实际上在游戏中花了更多的时间，因为额外功能已经解锁)，我可能会将他升级为半年或年度计划。

我儿子大多喜欢在他的 Chromebook 上玩网页版的 Prodigy，但它也可以作为一个应用程序在 [iOS](https://apps.apple.com/us/app/prodigy-kids-math-game/id950795722) 和 [Android](https://play.google.com/store/apps/details?id=com.prodigygame.prodigy&hl=en_US) 上使用。当我问他是否会向其他孩子推荐这本书时，他说他肯定会推荐，因为“它教孩子们做数学，太棒了。”
