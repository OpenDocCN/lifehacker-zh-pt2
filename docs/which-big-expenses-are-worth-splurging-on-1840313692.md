# 哪些大额支出值得挥霍？

> 原文：<https://lifehacker.com/which-big-expenses-are-worth-splurging-on-1840313692>

我们都被告知要尽可能多存钱。但是，有没有一些东西是值得多花点钱的呢？

在上面的视频中，个人理财专家和*我会教你变得富有*作者 Ramit Sethi 给出了他关于如何挥霍的建议。他提出了“金钱刻度盘”的想法——在你真正想要的东西上把刻度盘调到最大，在其他事情上把刻度盘调到最小，想象你的生活就是这样。你能在游轮上挥霍你的积蓄吗？当然，但是那样你一整年都不会去吃一顿美味的晚餐。想要那块高档手表吗？去拿吧，不过也许可以搬进便宜点的公寓。

Sethi 说你优先考虑的是问“为什么？”。你想买房子？为什么？想要辆新车吗？为什么？把这些长期目标记在脑子里，让你的支出形象化，让你保持在正轨上。
