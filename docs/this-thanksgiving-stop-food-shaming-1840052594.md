# 这个感恩节，停止食物羞辱

> 原文：<https://lifehacker.com/this-thanksgiving-stop-food-shaming-1840052594>

无论你和食物的关系是什么样的，它可能会在感恩节前后加强。如果你坚持一种特殊的饮食，你必须决定是继续下去还是说去他妈的，休息一天放纵一下。如果你家人的饮食习惯惹恼了你，那该多好啊！你会整天看着他们做饭和吃饭。我要提出一些激进的建议:让我们就此休战吧。

Watch

所以，当你吃第三片馅饼时，不要说你是“坏”的。不要在餐桌上提到卡路里——那只是你和我的秘密。

但你知道还有什么吗？如果别人和你有不同的饮食习惯，那就离他们远点。你表哥带了花椰菜而不是土豆泥？太好了，他们带来了他们想吃的食物，就像我们其他人一样。如果这让你很困扰，也许你应该自己带土豆来。(专业建议:总是自愿带上你最想吃的东西。)

我谦恭地推荐今天的以下规则。如果你想讨论让每个人都加入他们，那是你的事。但是这些应该可以作为你的个人道德准则，每个遵守它的人都会让这顿饭对其他人来说更好一点。这是:

*   想放纵就放纵吧。
*   想吃就吃健康的。
*   以上两个选项随意混搭。
*   不要评论任何人的食物选择。
*   一如既往，永远不要评论别人的身体。

顺便说一下，这包括孩子们。(“自从我上次见到你以来，你又高了一英尺！”是最后一条规则的例外。)他们还有 364 天可以吃蔬菜。不要逼他们多吃。不要在他们面前谈论你自己的食物焦虑。只是 [让他们安静地吃他们该死的](https://offspring.lifehacker.com/let-your-kids-eat-nothing-but-rolls-on-thanksgiving-1840002589) 面包卷，像其他人一样。
