# 利用星期五来计划接下来的一周

> 原文：<https://lifehacker.com/use-friday-to-plan-the-following-week-1839890542>

我们很多人都没有什么仪式来帮助我们从工作周中过渡出来，让周末尽可能放松。也许你去欢乐时光；也许你点外卖；如果你是我，也许你会收拾好你的公寓，点上一支香烛。



生产力专家 [劳拉·凡德卡姆](https://lauravanderkam.com/) 建议在你的周五待办事项清单上再增加一项: [规划下周](https://lauravanderkam.com/2019/11/tranquility-by-tuesday-planning-on-fridays-and-batching-those-little-tasks/) 。

> 长期的博客读者知道，我总是建议人们在周五下午计划他们的一周。这是一个很好的时间来回顾即将到来的一周，并列出职业和个人的优先事项。

Vanderkam 解释说，周五下午——或者当你的工作周结束时，如果你周一到周五不工作——是评估下周可能会是什么样子的绝佳时间，你可以如何安排这一周，不仅包括你眼前的责任和任务，还包括一些我们倾向于推迟的长期的大计划。

换句话说:如果你一直渴望开始一个新的锻炼计划，让家人恢复正常工作日晚餐的习惯(或者如果不起作用，早餐)，或者会见一个你几个月没见的朋友，花周五下午的一部分时间把这个目标放在日历上。

Vanderkam 还建议利用这个计划会议来“为下周的日程分类” 寻找可以委托、删除或拒绝的项目，然后寻找可以批量处理的差事和任务。也许你可以腾出一些时间，用于工作项目或个人目标——或者，你知道，你可以简单地利用这些时间放松一下。

我倾向于按月而不是按周进行这种类型的计划，它对帮助我确定每个月我能承担多少自由职业者和社会责任非常有用。如果你也做这些类型的计划会议，他们如何为你工作？总的来说，你的工作和生活平衡得更好了吗？还是你发现无论你如何计划和安排优先顺序，工作和生活都成了障碍？
