# 现在是开始健身的最佳时机

> 原文：<https://lifehacker.com/now-is-the-best-time-to-start-working-out-1830680894>

如果你有哪怕是最微小的暗示，你可能想在新的一年里——或者任何时候——保持身材，不要等到一月。养成健身习惯的最佳时间就是现在，那时你将拥有属于自己的地方。

Watch

从技术上来说，“现在”总是开始的最佳时机，但是 12 月是特殊的。如果你等到一月份，健身房会挤满了人。今年的头两周，每个人都认为他们开始了一个惊人的新健身习惯...然后到了二月，他们中的大多数人已经把这件事忘得一干二净了。不要因为在高峰期开始就让自己的生活变得更加艰难。课程会爆满，你得 [跟人争蹲架](https://vitals.lifehacker.com/alternative-exercises-for-when-that-guy-will-not-get-ou-1828150316#_ga=2.247439556.1188746829.1543243014-1456718367.1520458611) ，很难有什么真正的乐趣。所以现在就开始。

12 月，天气晴朗的健身房人下班后会去买礼物，周末会去参加节日派对。他们早上也不想离开温暖的被窝。到了之后，蹲架就空了。你可以选择跑步机。健身房将是你的囊中之物。

在抢购之前，你会有一个月左右的时间。而那个月是一个 [完美的小时间框架](https://lifehacker.com/make-a-resolution-for-the-last-six-weeks-of-the-year-1830555411#_ga=2.3083632.1188746829.1543243014-1456718367.1520458611) 来设定一个小目标并坚持下去。发誓每周去健身房三次，新年前你会有至少 12 次锻炼。开始一个 [couch-to-5k](https://lifehacker.com/get-off-the-couch-and-start-running-with-the-c25k-app-1825351919) 跑步计划，你将在一月前完成一半。

是的，12 月可能会很忙，压力很大，但请记住，健身往往会帮助我们的大脑管理压力。在你的日历上标记出你的锻炼，并把这些约会留给你自己。

不要担心会错过优惠:任何一家提供一月特价的健身房或工作室都会在整个十二月进行销售。如果他们不提供，就问。
