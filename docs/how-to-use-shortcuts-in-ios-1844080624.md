# 如何在 iOS 中使用快捷键

> 原文：<https://lifehacker.com/how-to-use-shortcuts-in-ios-1844080624>

快捷方式应用程序是一个强大的自动化工具，可以帮助你优化生活，节省大量的日常任务时间。在上面的视频中，我分享了一些我使用快捷方式的方法，以及我用来了解它们的资源。

### 什么是捷径？

快捷方式允许您创建一个脚本，通过简单地询问 Siri 或按下按钮来自动运行许多任务。例如，如果我说“嘿 Siri，现在是工作时间”，那么下图的工作时间脚本将自动运行。“勿扰模式”会一直打开到下午 5 点，Siri 会说“又是一天，又是一美元！”

快捷方式也可以设置为基于特定标准触发，而不是按下按钮或通过 Siri。当我还在通勤上班的时候，我曾经在上下班的时候设置了两个自动装置。这些脚本将根据我手机的位置运行。

这些是快捷方式能做什么的非常简单的例子。更复杂的脚本可能有几十个步骤长，可以用来做一些事情，如 [为你的 WiFi](https://shortcutsgallery.com/shortcuts/share-wi-fi/) 或 [生成一个可共享的二维码，自动化你整个早上的例行程序](https://www.reddit.com/r/shortcuts/comments/go2bzh/final_version_of_my_good_morning_shortcut_p/?utm_source=share&utm_medium=web2x) 。如果你有兴趣学习如何建立自己的快捷方式，苹果全面的 [快捷方式用户指南](https://support.apple.com/guide/shortcuts/welcome/ios) 是一个很好的起点。

### 哪里可以找到快捷方式

幸运的是，你不必自己建立快捷方式。该应用程序有一个图库页面，苹果在其中展示了许多预建的快捷方式。像 [快捷方式子编辑](https://www.reddit.com/r/shortcuts/) 、 [快捷方式库](https://shortcutsgallery.com/) 和 [共享快捷方式](https://sharecuts.app/) 这样的网站也是找到强大而有用的快捷方式的好地方。

所以，开始考虑你的日常任务吧。如果你的手机设置为自动完成，哪些会节省你的时间？如果你的妈妈收到一句“我爱你”，她会跟你一样唠叨吗？每隔几天就收到你的短信？

自动化快乐。
