# 用这个网站免费创建图形

> 原文：<https://lifehacker.com/create-graphics-for-free-with-this-site-1843677054>

在和孩子在家呆了几个月之后，许多父母期待着和他们一起度过这个夏天...孩子们全职在家。取决于你住在哪里，以及你个人打算在温暖的月份里对社交距离有多严格，特别是青少年可能已经达到厌倦的顶峰。是时候让他们 1)玩一些新的有创意的东西，2)装饰自己卧室的墙壁或更新他们的社交媒体档案了。要做到这两点，把他们介绍给 [海报我的墙](https://www.postermywall.com/) 。



在海报我的墙上，用户可以按主题 浏览图形和视频模板，如生日、音乐会和乐队、励志、毕业、家庭或体育。或者他们可以 [按大小](https://www.postermywall.com/index.php/posters/sizes) 搜索——从 Instagram 帖子或 YouTube 缩略图到海报或传单的大小。

无论如何，我都不是平面设计师(这一点很明显)，但在摆弄一个空白模板来看看事情是如何运作的过程中，我在大约 10 分钟内为 Instagram 制作了这个小图形:

是的，所以你可以看到为什么我倾向于模板，其中 [有很多](https://www.postermywall.com/index.php/posters/gallery?utm_source=nav&utm_content=browsetemplates&utm_medium=link&utm_campaign=templategallerynav) 。但是对于更有设计天赋的人来说，你可以 [从头开始定制你的图形](https://www.postermywall.com/index.php/posterbuilder) 。选择大小(海报、横幅、相册封面、Instagram 帖子或故事、脸书封面照片或共享图片或 YouTube 频道封面照片，等等)。然后通过网站搜索股票图片，上传你自己的照片，选择背景颜色并添加你的文字。

当然，你可以购买的高级订阅。但是，除非你是一个企业主，试图利用海报墙进行营销和品牌推广，否则基本的免费版本可能是你的家人所需要的。你们这些孩子将主要使用它来创建自己的图形，设计一些社交媒体图像，也许还会打印一些海报挂在他们的卧室里。Lifehacker 的一名员工的十几岁的儿子甚至用它为他的朋友为他缝制的刺绣补丁创建了一个图像。

父母还可以使用海报我的墙来创建数字 [儿童生日聚会邀请函](https://www.postermywall.com/index.php/posters/search?s=kids+birthday&tt=all)—甚至社交距离 [生日游行邀请函](https://www.postermywall.com/index.php/posters/search?s=birthday+parade&tt=all#1e83e0dde0adde459d97a7029bfe33b6) 。大多数模板都是免费的，但如果你想选择定制他们的模板，一些设计师会收取几美元。还要注意的是，很多免费设计的底部会有一个小小的“Made with PosterMyWall.com”水印，但不会太分散整体设计的注意力。
