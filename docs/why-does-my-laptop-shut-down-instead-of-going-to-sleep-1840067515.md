# 为什么我的笔记本电脑会关机而不是进入睡眠状态？

> 原文：<https://lifehacker.com/why-does-my-laptop-shut-down-instead-of-going-to-sleep-1840067515>

你在使用一个设备(通常是你的电脑)时会遇到的最令人沮丧的问题之一是，当它开始做以前从未做过的事情时。通常，这是你不希望*It 部门做的事情:它需要 15 秒以上的时间来启动，它发出一些奇怪的噪音，或者它无法启动你过去三个月一直在使用的应用程序。诸如此类的怪事。*



雪上加霜的是，Lifehacker 读者 **Melissa** 给我们发来了这个令人困惑的问题:

> 我的笔记本电脑已经用了大约 5 年了。这是一个华硕，运行 Windows 10，杀毒 ESET，使用 Chrome 作为主要浏览器。
> 
> 我总是将设置设为睡眠(或休眠？它对我来说是可互换的)当我走开的时候。我不在公共场合使用笔记本电脑，所以除了我的密码之外，不需要任何隐私设置。我去工作，它睡觉。我回到家，按下空格键或电源按钮，它就从我离开的地方开始浏览互联网。
> 
> 在过去的一两个月里，这种情况发生了变化。我想知道是否有一个 Windows update 在某个地方翻转了设置，或者我的计算机是否有问题。每次我离开超过 30 分钟，电脑就会完全关机。我按下电源按钮，它带我通过华硕加载屏幕，我必须等待一切重新启动，包括 ESET 软件。我必须重新打开 Chrome 并恢复我的浏览会话。

> *到底怎么回事？*

这绝对是一个古怪的问题，但是你的问题中也有一线希望。我不认为你做过任何会导致你的笔记本电脑突然关机而不是睡眠的事情；我认为这完全是你的笔记本电脑的问题。

不过，我的第一个倾向是把你带到你的笔记本电脑的“电源和睡眠设置”屏幕——点击 Windows 开始按钮后输入这个短语即可进入。在那里，向下滚动一点，直到你看到“附加电源设置”链接，然后点击它。

在出现的屏幕中——您的控制面板的“电源选项”部分——单击左侧边栏上的“选择合上盖子会做什么”链接。然后，您应该会看到如下所示的屏幕:

如果您按下电源按钮和/或合上笔记本电脑的盖子没有设置为睡眠，请确保在您的笔记本电脑接通电源或使用电池时设置为睡眠。这应该能解决你的问题。

然而，如果所有这些设置都已经设置为“睡眠”，情况就复杂了。在这种情况下，我认为这个问题更加古怪。

你会想要打开你的“设备管理器”,在你点击“开始”按钮后输入这个短语。在那里，展开您的系统设备，寻找“英特尔管理引擎接口”如果找到了，右键单击它，选择“属性”，然后单击“电源管理”选项卡。取消选中“允许电脑关闭此设备以省电”，看看这样是否能解决问题。

否则，这台电脑中的电脑可能需要更新的驱动程序。单击“驱动程序”选项卡，然后更新驱动程序，以查看是否可以获得比您当前使用的版本更新的版本(您可以在驱动程序版本行上看到)。如果您的系统没有发现任何问题，您可能需要从英特尔 下载 [更新的驱动程序并手动安装。](https://downloadcenter.intel.com/download/28679/Intel-Management-Engine-Driver-for-Windows-8-1-and-Windows-10)

如果这些都不起作用，您的最终选择是要么回滚到 11 版之前的驱动程序(如果可以的话)，要么完全禁用英特尔管理引擎接口。这是同一个驱动程序选项卡中的两个选项。

我希望这些答案中的一个能够解决你的问题，但是如果你仍然有笔记本电脑关机而不是睡眠的问题，请回信。我们将为您的设备找到合适的数字 NyQuil。
