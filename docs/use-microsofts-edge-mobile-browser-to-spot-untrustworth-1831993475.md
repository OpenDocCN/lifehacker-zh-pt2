# 使用微软的 Edge 移动浏览器发现不可信的新闻来源

> 原文：<https://lifehacker.com/use-microsofts-edge-mobile-browser-to-spot-untrustworth-1831993475>

微软正在将自己的 [假新闻之战](https://lifehacker.com/learn-about-fake-news-with-this-web-game-1823229554) 扩展到移动战线。其 Edge 浏览器应用程序的最新更新包含了一个新闻标记功能，该功能以前只在桌面版本上作为可选插件提供。

Watch

该功能的移动和桌面版本都是基于 NewsGuard——一种浏览器扩展，使用记者来发现假新闻，而不是机器学习算法或其他第三方网站和公司[。这种方法有利也有弊，但结果是你让真实的人来检查和验证你正在阅读的网站。](https://lifehacker.com/fight-fake-news-with-this-chrome-extension-1826810600)

一些浏览器(如 Edge 的移动版本)预装了扩展。其他的会要求你自己添加。在下一节中，我们将向您展示如何为所有可用的浏览器启用/禁用该扩展，并且我们将解释如何使用它。

### 如何启用或禁用 NewsGuard

**边缘移动**

*   你需要运行 Edge 最新的手机浏览器。在谷歌 Play 商店的或者苹果的 [应用商店购买。](https://itunes.apple.com/us/app/microsoft-edge/id1288723196?mt=8)

*   打开移动设备上的浏览器，点击**“...屏幕右下方的**图标，然后进入**设置>新闻评级**，打开**的“地址栏显示评级”选项**要禁用 NewsGuard，只需反向操作即可。

**边缘桌面**

你也可以在 Edge 的桌面版上获得 NewsGuard。

*   从 [Windows store](https://www.microsoft.com/en-us/p/newsguard/9nwp4lmmkfkt?rtc=1&activetab=pivot%3Aoverviewtab) 下载扩展，安装完成后打开 Edge。

*   你应该会看到一个对话窗口，询问你是否愿意打开新闻评级选项；点击**“是”**如果您没有看到这一点，请点击**…”浏览器右上角的**图标，然后点击**【扩展】**并启用 NewsGuard。

Chrome、Safari 和 Firefox

对于那些想在不同浏览器上使用 NewsGuard 的人，你需要为[Chrome](https://chrome.google.com/webstore/detail/newsguard/hcgajcpgaalgpeholhdooeddllhedegi)[Firefox](https://addons.mozilla.org/en-US/firefox/addon/newsguard/)或 [Safari](https://itunes.apple.com/us/app/newsguard/id1438657064?mt=12) 安装插件。然而，这些版本不像 Edge 的移动版本那样在 URL 中显示网站的评级，而是给你一个图标供你查看(与你的浏览器的其他扩展在同一工具栏上)。

在任何桌面浏览器上，您都需要从浏览器中卸载 NewsGuard 附加组件才能将其删除，这通常只需右键单击扩展的图标，然后单击“从浏览器中删除/卸载”，尽管在某些情况下，您可能需要在浏览器的插件/扩展设置中找到它。

### 如何使用 NewsGuard 的新闻评级

浏览时，寻找图标(无论是在浏览器的地址栏还是其他地方)。此图标反映了基于 NewsGuard 评估标准的网站评级。

颜色编码的含义如下:

*   **绿色:**可信网站。通过全部或大部分 NewsGuard 的 [评级标准](https://www.newsguardtech.com/ratings/criteria-for-and-explanation-of-ratings/) 。

*   **红色:**通过很少(或没有)可信评级的必要要求。

*   **蓝色:**网站主要以用户生成的内容为特色。

*   **橙色:**此图标表示文章/网站是讽刺性的，不要当真。

*   **灰色:**网站仍在审核过程中。

点击 NewsGuard 图标(或者将鼠标悬停在桌面浏览器上),会显示一个网站被给予特定评级的原因的快速解释。您也可以点击“获取完整营养标签”选项，获取更详细的解释。

虽然你会看到几乎所有主要媒体和你遇到的大多数其他来源的评级，但不是所有的网站都会有评级，尤其是非新闻网站(在这种情况下，图标将是白底黑字)。但是，您可能会遇到没有评级的新闻来源。在这种情况下，您可以通过点击或点击 NewsGuard 图标并点击“提交”来提交页面进行审查
