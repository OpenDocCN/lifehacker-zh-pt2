# 据 Lifehacker 的工作人员说，隔离舒适电视

> 原文：<https://lifehacker.com/quarantine-comfort-tv-according-to-the-lifehacker-staf-1842667167>

自从美国许多城市发布就地避难的命令以对抗新冠肺炎病毒的传播以来，已经过了很长的时间，而且李也不打算在短时间内恢复正常。现在是时候逃避到让你感觉良好的电视节目中去了——或者至少让你感觉到某种东西，而不是压倒性的存在恐惧。



“抚慰”对一百万人来说意味着一百万件事，但最重要的是，我认为舒适电视需要被人们所熟悉。在一个延长的隔离场景中，充足也是关键——应该会有很多集，看起来不可能全部完成。无论你的个人电视美食是什么类型——轻松的喜剧、没用的真人秀、自然纪录片或声望剧——现在都不是潜入陌生和可能令人失望的东西的时候。坚持看你已经知道你喜欢的节目，那些有足够多的剧集让你看很长时间。

对我来说，这意味着回到我的荒岛秀:*美国人* ( [都在黄金视频](https://www.amazon.com/The-Americans-Season-1/dp/B00B8P8O9K?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/quarantine-comfort-tv-according-to-the-lifehacker-staf-1842667167&asc_source=&tag=kinjalifehackerlink-20) )和*山中之王* ( [一样，但 Hulu](https://www.hulu.com/series/king-of-the-hill-52b8dd8a-eff2-4ed2-9b8d-7c0039df1c53) )。我对詹宁斯一家、希尔一家和他们无数的伙伴们简直百看不厌；两部剧中的每一个角色对我来说都是如此熟悉，以至于看着他们就像和老朋友在一起。(或者，对美国人来说，是一个极度不正常的大家庭。)两个跑了足够长的时间给我周现成的娱乐也无妨。

下一个家庭电视节目是:[*【葫芦】*](https://www.hulu.com/series/justified-2faf68df-e153-4e47-b117-a048427b250f)*，我男朋友很喜欢，但我从未看过——如果我们对失去和悲伤进行残酷的冥想，也许是 [*剩菜* (HBO Go)](https://play.hbogo.com/series/urn:hbo:series:GVU2g4AhF347DwvwIAURT) 。作为犯了痛的粉丝，遭遇和奥莉薇娅·柯尔曼，然而另一场 [*西洋镜* (Hulu)重看](https://www.hulu.com/series/peep-show-6c41fe11-ae9b-4c22-bf66-16b159961e28) 似乎也在所难免。如果你在寻找更多的灵感，这里还有一些 Lifehacker 的员工在隔离期间关注的事情。*
