# 让我们做一些臀部伸展运动

> 原文：<https://lifehacker.com/lets-breathe-into-some-hip-opener-stretches-1842428304>

这几天世界有点乱，但是 Lifehacker 健身挑战赛还是一如既往的为你而来。这个月我们一直在练习柔韧性，现在我们已经学完了和 [脚踝灵活性](https://vitals.lifehacker.com/how-to-improve-your-ankle-mobility-for-squats-1842315361) 是时候打开我们的臀部了。

我喜欢瑜伽把某些伸展动作称为“开场动作”这就转移了对一定程度的灵活性的关注。相反，它让我想尝试一个职位，看看它能给你什么机会。也许你已经翘着二郎腿坐了很久了。如果你反其道而行之会怎么样？那使你能做什么或感觉到什么？

所以我认为这是一个很好的时间来尝试一些伸展运动，让我们的腿和臀部以我们在办公桌和沙发上花费时间时没有机会的方式运动。试试这些:

**[**膝盖向胸部伸展**](https://yourpaceyoga.com/blog/single-knee-to-chest/) 。仰面躺着，一个膝盖贴在胸前，另一条腿平放在地上(或者尽可能平放)。**

****[**缠角姿势**](https://www.yogajournal.com/poses/bound-angle-pose) 。双脚并拢坐在地板上。如果你能让脚贴近身体，那就太好了；如果没有，找一个舒服的角度。(如果你的脚底不能放平，就把你的脚跟并拢。)为了做额外的伸展运动， [躺下，把你的后背也放在地板上](https://www.yogajournal.com/poses/reclining-bound-angle-pose) 。****

****[**用带子把腿拉长**](https://dailyburn.com/life/fitness/best-hip-stretches-relieve-tightness/) 。如果你没有瑜伽带，任何东西都可以，包括一条毛巾或一条套在脚上的运动裤。仰卧，将一条腿伸直举在空中(如果你的腿指向前方而不是伸直，也没关系)。轻轻地拉紧皮带，将腿向自己的方向拉，拉伸腿筋，然后让腿向一侧倾斜。****

****希望在这些伸展运动中花点时间呼吸会帮助你在身体和精神上感觉好一点。****
