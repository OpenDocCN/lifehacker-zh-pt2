# 做烤茄子不需要烤架

> 原文：<https://lifehacker.com/you-dont-need-a-grill-to-make-grilled-eggplant-1844139154>

众所周知，我喜欢茄子 ，多年来，我在烈日下做了所有的准备——只有一个明显的例外。由于家里没有空间放烧烤架，而且强烈厌恶在夏天打开我的烤箱，烤茄子一直是我的烹饪大白鲸。直到现在。

Watch

我新喜欢的茄子技术来自于洛杉矶 夜市的主厨 [Kris Yenbamroong，我在 Instagram](http://www.nightmarketsong.com/welcome) 上关注他 [主要是因为他令人惊叹的烹饪故事。在他的保罗茄子食谱中，他使用了一个我从未见过的天才技巧:他没有用盐腌茄子，而是用油炸茄子，而是把完全没有准备好的茄子片扔进一个完全没有油的盐锅里。煮熟的茄子片被涂上大量的橄榄油、大蒜、辣椒片和香草，就像千层面一样。很明显，我必须尝试一下。](https://www.instagram.com/krisyenbamroong/?hl=en)

我第一次尝试这种方法的时候，我手头只有几棵悲伤的、软绵绵的茄子，它们已经在我的冰箱里放了一个多星期了。通过这种准备，即使是那些古怪的老样品也尝起来*非凡的*——第二批，我为了本文的目的用超新鲜茄子做的，甚至更好。当我刷掉一片热茄子上多余的部分时，我学到了一个有趣的方法，氯化钠晶体疯狂地保持热量；当茄子碰到热盐时，所有从茄子上掉下来的水分都会蒸发掉。表面保持相对干燥，这大大有助于变深，甚至变褐。这不是木炭烧烤，但结果和我吃过的最好的烤茄子一样又香又甜，而且口感丝滑。

这项技术本身非常简单。将茄子纵向或横向切成 1/4-1/2 英寸厚的片。用中高火吃一个不粘锅(或者一个非常 [风干的铸铁](https://lifehacker.com/why-you-should-season-your-cast-iron-the-wrong-way-1843447192) 锅)。天气热的时候，在锅里撒一两撮食盐(或细海盐)。将茄子片放在上面。几分钟后，把它们翻过来。当它们两面都被烤焦时，把它们转移到碗里或盘子里。重复直到茄子吃完，根据需要加入盐。

现在是有趣的部分:调味。因为你还没有使用任何油，一个必不可少的元素是你选择的有味道的油——它会保持茄子鲜嫩多汁，而不是像皮革一样。除此之外，没有错误的答案。就连这道菜的做法也由你决定:像 Yenbamroong 一样，在茄子上涂上油和香料，或者把它切碎，像沙拉一样混合在一起。

如果你已经有了香精油、大蒜和香草的最佳组合，那就用它吧。如果没有，这里有一些建议:

*   芝麻油、大蒜、姜、葱、辣酱和米醋
*   橄榄油、柠檬汁、大蒜、刺山柑、橄榄和欧芹
*   花生油、鱼露、酸橙汁、甜辣椒酱、香菜和碎花生
*   酸奶、橄榄油、莳萝、柠檬汁和大蒜
*   [自制辣椒油](https://skillet.lifehacker.com/how-to-make-your-own-chili-oil-1842478518) ，酱油，江醋和芝麻
*   橄榄油、熟番茄切片、罗勒和香醋

适当的数量和比例会根据你的喜好和你做了多少茄子而变化，所以边做边品尝吧。一旦调味完美，你就可以马上吃茄子，或者把它放在冰箱里几个小时。无论哪种方式，你都会有一道非常美味的蔬菜，非常适合搭配肉类、面包和奶酪、米饭或单独食用。
