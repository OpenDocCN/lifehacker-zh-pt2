# 你应该租哪种婴儿用品而不是买？

> 原文：<https://lifehacker.com/which-baby-gear-should-you-rent-instead-of-buy-1840857087>

生孩子是一项昂贵的努力。医疗保健费用和为婴儿第一年的生活做准备的费用占了一大笔钱。随着婴儿的成长，他们的需求也在变化。没有办法预测你的孩子会多快过渡到这些阶段，这可能会带来一大堆不可预见的成本，包括衣柜和汽车座椅的升级。



如果你最初几个月的预算有点紧张，你经常带着婴儿旅行，或者你发现自己在和一个快速成长的孩子打交道，那么租赁二手婴儿用品可能是你的一个好选择。但是出于安全和卫生的原因，也有一些婴儿用品是绝对不能借或租的。

### 如何租赁婴儿用品

租赁婴儿用品的生意已经演变成一种独立的零工经济。有些服务是与个人提供商签约的，就像你通过 Airbnb 租房子一样。供应商有责任在每次返回后清洁设备，并确保其仍可运行。这个领域中比较受欢迎的公司之一是 [BabyQuip](http://www.babyquip.com) 。该网站让你选择各种各样的婴儿用品，包括汽车座椅、婴儿床、游戏围栏、换衣垫、婴儿监视器、门锁、便携式秋千和玩具。

BabyQuip 已经存在好几年了，它的一些提供商也出租设备有同样长的时间。他们知道他们有什么，他们的设备有保险。一旦装备被归还，你可以留下一个评论给其他人，让他们考虑你与那个特定的婴儿装备提供商的经历。

婴儿装备的价格各不相同。提供商按天收费，外加送货费和安装费。如果你要长期租用，他们会提供几天的折扣，但是事先做一个粗略的计算是值得的。在某些情况下，直接买东西更划算。

一些制造商提供个人租赁计划。如果您的宝宝仅在几个月内需要某种特定的产品，这种方法是最好的。例如， [史努比智能睡眠器](https://www.happiestbaby.com/products/snoo-rental) 很受精疲力尽的父母们的欢迎，他们需要帮助他们摇着婴儿入睡。然而，一次性购买一个要花费 1300 美元，这对于预算有限的家庭来说是非常昂贵的，尤其是考虑到宝宝长大后会买不起。但是史努会让你租卧铺来省钱。它需要至少一个月的承诺，但每月 112 美元的起价，在六个月的推荐最大使用量内租赁更实惠。

你也可以在你的区域内寻找一个婴儿用品租赁服务。 [HelloBundl](https://www.hellobundl.com/) 服务于洛杉矶和奥兰治县地区，而 [湾区婴儿设备(B.A.B.E)](https://bayareababyequipment.com/about-us/) 和 [一切为婴儿租赁服务](https://anythingforbaby.com/) 分别与旧金山湾区和凤凰城的客户合作。如果你在旅行，试着在你去的地方找一个服务。如果你要去迪斯尼世界，你可以试试婴儿最好的朋友，它为在奥兰多地区度假的家庭提供服务。甚至还有 [国际出租服务](https://travelmamas.com/rent-baby-travel-gear/) 供家庭出国旅游。

如果你决定走独立经营的路线，发电子邮件或打电话询问租赁服务的政策。询问他们如何确定他们的设备是安全的，他们是否检查汽车座椅没有被召回或 [过期](https://lifehacker.com/childrens-car-seats-have-expiration-dates-1796238663) 以及他们如何消毒婴儿床床垫。如果你的宝宝受伤或感染，询问他们的保险单包括哪些内容以及你应该承担哪些责任也是有帮助的。

### **适合租赁的设备**

租赁任何易于清洁或消毒的物品既容易又安全。这包括:

*   衣服
*   玩具
*   浴缸
*   换衣桌
*   高椅
*   布尿布(前提是它们有可拆卸的衬垫，并且每次使用后都经过消毒)
*   抄袭
*   操场
*   散步者

婴儿床、游戏场和婴儿车的一个警告是检查它们是在联邦法规对这些特定产品类别变得更加严格之后制造的。

### **你应该买的东西**

有很多方法可以租到婴儿用品，但这并不意味着你应该租。这些东西你最好自己去买，或者至少，如果你是租的话，仔细检查一下。

*   **吸奶器**。吸奶器有交叉污染的风险，根据 [FDA](https://www.fda.gov/medical-devices/breast-pumps/buying-and-renting-breast-pump) ，“医院级泵”这个术语甚至没有得到官方认可。如果你不得不租一个，事先和你的医生或认证的哺乳顾问谈谈。确保你看到的模型不是一次性的。
*   **汽车座椅**。你可以为快速旅行租一个汽车座椅，但当你联系租车服务时要考虑到这一点:抱着你的小家伙的泡沫材料会随着时间的推移而降解。你会想问汽车座椅是否出过事故，它有多老了。美国儿科学会的暗示年龄不超过 10 岁。最后，一定要查看国家公路交通安全管理局的网站 ，上面列出了召回信息和单个汽车座椅的安全评级。
*   **婴儿床床垫**。最好不要使用用过的婴儿床床垫，尤其是如果是通过长期租赁的租赁服务。婴儿会弄脏自己，而且不能保证最后一个使用床垫的人会使用吸水垫。婴儿床床垫也会随着使用变得更加柔软。根据宝宝的年龄，你会想要一些坚固的东西来帮助预防婴儿猝死综合症(SIDS)。

当你浏览婴儿用品租赁世界时， [消费者召回](http://recalls.gov) 的官方网站是一个方便的链接，可以存档以仔细检查任何安全问题。
