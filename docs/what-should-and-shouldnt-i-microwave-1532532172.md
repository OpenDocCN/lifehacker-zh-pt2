# 我该不该用微波炉加热什么？

> 原文:[https://life hacker . com/what-should-and-should nt-I-microwave-1532532172](https://lifehacker.com/what-should-and-shouldnt-i-microwave-1532532172)

微波炉要么是厨房的救命稻草，要么是危险的辐射机器，这取决于你问的是谁。事实上，它们对某些目的来说是伟大的，但对另一些目的来说却不是。以下是如何确定哪些该用微波炉加热，哪些该跳过。

亲爱的生活黑客，我听说用微波炉加热塑料容器或塑料包装的食物很危险，因为会污染食物的化学物质很可怕。这是真的吗？还有哪些东西在微波炉里使用安全或不安全？

签名，紧张的核武器

亲爱的神经紧张者，微波炉是一种强大的电器，伴随着这种力量而来的是巨大的潜在危害——比如火花或爆炸，是的，可能还有化学物质渗入你的食物。不过，别担心，你的微波炉很可能不会要你的命。只要留意这些安全微波烹饪的建议。

### **集装箱**

通常情况下，你需要把食物和饮料放在某个东西里用微波炉加热。以下是最适合用作容器的材料和应该避免的材料。

**做:**

*   玻璃容器。这些可能是最好用的，因为它们在微波炉中的安全性是毫无争议的。
*   大多数纸盘、毛巾和餐巾纸。然而，由于一些纸巾是用塑料制成的，一些纸盘和纸杯涂有塑料，美国国家卫生基金会(NSF)国际 [建议](http://www.nsf.org/newsroom/cooking-with-a-microwave) 只使用那些标有微波安全的产品。
*   陶瓷。大多数由适当配方的釉料制成的陶瓷在微波炉中是安全的，但是根据今日秀 ，我们在处理中国制造的陶瓷时应该小心。“由于中国没有对该行业进行充分的监管，一些制造商正在使用含有大量铅和砷的釉料——微波加热时，这些危险的元素会浸出到食物中，”T5 文章解释道 。此外，请注意:盘子、杯子或其他带有金属漆的容器也可能产生火花。

*   根据美国国家科学基金会的说法，蜡和羊皮纸，以及微波烹饪袋也被认为是好的。

**不要:**

*   [铝箔](http://science.howstuffworks.com/aluminum-foil-in-the-microwave.htm) 。从技术上来说，你可以在微波炉中使用铝箔(有趣的事实:热口袋微波容器和类似的食物 [有铝内衬](http://mentalfloss.com/article/32032/why-cant-you-put-metal-microwave) )，但它 [可能不是最明智的做法](https://www.youtube.com/watch?v=lhpndQJkNk0) 而且可能会着火。
*   金属容器(如罐头食品)，原因同上。
*   杂货店买的棕色纸袋。根据美国农业部[:“它们不卫生，可能引起火灾，并能释放有毒气体。高温可能会导致袋子着火，从而导致烤箱起火...纸袋中的墨水、胶水和回收材料在受热时会散发出有毒气体。相反，使用购买的烤箱烹饪袋。”](http://www.fsis.usda.gov/wps/portal/fsis/topics/food-safety-education/get-answers/food-safety-fact-sheets/poultry-preparation/turkey-alternate-routes-to-the-table/CT_Index)
*   一次性存储容器，如外卖容器、人造黄油桶或酸奶容器。他们 [很可能没有“微波炉保险箱”标签](https://www.health.harvard.edu/staying-healthy/microwaving-food-in-plastic-dangerous-or-not) ，所以你不应该冒这个险。
*   塑料垃圾袋、垃圾桶或胶卷盒。我不知道人们会考虑这些烹饪器皿，但是内布拉斯加大学 T2 分校 T3 警告不要用这些东西来微波烹饪

**有争议:**

*   塑料制品。这是人们害怕导致癌症的一大原因。甚至不含 BPA 的产品(尽管其他研究表明 [可能不像之前认为的那么糟糕](http://www.npr.org/blogs/thesalt/2014/02/26/283030949/government-studies-suggest-bpa-exposure-from-food-isn-t-risky) )。然而，关于微波或洗碗塑料对健康的影响，还没有定论。环境工作组 [建议不要用任何种类的塑料容器微波食品或饮料。然而，美国食品和药物管理局批准微波使用的容器是基于他们对化学物质浸出的测量；哈佛健康报告](https://www.ewg.org/enviroblog/2008/04/cheatsheet-bisphenol-bpa) :“每磅体重的最大允许量比实验室动物终生使用的有害量低 100-1000 倍。只有通过这项测试的容器才能显示“微波炉安全”图标，或“微波炉安全”字样，或大意为它们被批准用于微波炉的字样。
*   泡沫聚苯乙烯。同样，一些聚苯乙烯泡沫塑料产品标有“微波炉安全”，但当有疑问时，不要用微波炉加热。
*   如果你决定用塑料容器微波，只要确保它们没有破裂、老化或变色 。当用保鲜膜覆盖食物时，确保塑料不会接触到食物。

### **食物和饮料**

有趣的是，即使微波加热某些食物也可能对你的健康有害或导致事故，尽管总的来说微波食物很棒 [，甚至可以帮助某些食物保留更多的营养](http://io9.com/do-microwave-ovens-kill-nutrients-in-food-1443585868) 。

**做:**

*   培根，最好是用一个 [倒扣的碗](http://lifehacker.com/use-a-bowl-to-cook-crispy-bacon-in-the-microwave-5990895) 。
*   [冷冻披萨](http://lifehacker.com/when-its-actually-best-to-cook-a-frozen-pizza-in-the-mi-5992719) 。
*   鸡蛋， [炒鸡蛋](http://lifehacker.com/make-breakfast-in-a-mug-in-just-one-minute-5797936) 或 [水煮鸡蛋](http://lifehacker.com/make-sunny-side-up-or-poached-eggs-in-the-microwave-1454422584) 。
*   羽衣甘蓝，薯片的健康替代品
*   或者 [薯片](http://lifehacker.com/make-potato-chips-in-the-microwave-without-sacrificing-5492617) 。
*   柠檬和酸橙， [从中获取更多的汁液](https://lifehacker.com/refrigerate-then-microwave-citrus-to-get-the-most-poss-651845002) 。
*   [碗里的爆米花](http://lifehacker.com/microwave-popcorn-in-a-bowl-and-plate-to-reduce-unpoppe-5970874) 。
*   [蔬菜，尤其是茄子](http://lifehacker.com/cook-better-with-your-microwave-than-your-stove-375262)T3。
*   像 [松糕](http://lifehacker.com/make-a-surprisingly-moist-sponge-cake-in-the-microwave-1028723713) 和 [布朗尼](http://lifehacker.com/make-a-brownie-in-the-microwave-in-six-minutes-5985062) 这样的甜点。
*   除了下面的项目，几乎任何东西。

**不要:**

*   葡萄。显然 [它们会爆发出火焰](http://www.youtube.com/watch?v=vCNNqgKqnaQ) 并变成一个发光的等离子气体球。
*   蛋壳完整的鸡蛋或煮熟的鸡蛋。 [这里的问题](https://www.incredibleegg.org/cooking-school/tips-tricks/microwave-wisdom/) 是压力在密封的鸡蛋“容器”中积聚，会导致它爆炸(即使煮熟的鸡蛋被剥皮)。最好在加热前在鸡蛋上多处穿孔。
*   干辣椒。显然，辣椒中的辣椒素化学物质是易挥发的，会着火。

**注意:**

*   用微波炉烧水时要小心，因为水可能会“过热”，可能会爆炸。在杯子 [中放一根筷子或其他棍子可以防止这种](http://lifehacker.com/prevent-super-heated-exploding-water-in-a-microwave-wi-1377512762) 。
*   和鸡蛋一样，切或戳一些更容易因为内部压力而爆炸的东西，比如热狗、香肠和土豆。另外，确保排空所有容器，以便蒸汽能够逸出。

### **其他物体**

你可能已经知道大多数金属不应该放入微波炉，除非你想要火花和灯光秀。你不应该放进微波炉的其他东西的清单很长，但也有一些不寻常的，非破坏性的方法来使用你的微波炉。

**做:**

*   [湿海绵，给它消毒。](https://lifehacker.com/heat-a-wet-sponge-or-rag-in-the-microwave-to-make-clean-5859200) 确保你先挤出洗碗皂。
*   盆栽土，要 [给它](http://homeguides.sfgate.com/microwave-potting-soil-48342.html) 消毒，特别适合做种子。
*   洗碗巾放在一个带拉锁的袋子里 [DIY 热水袋](http://lifehacker.com/microwave-dish-towels-in-a-ziploc-for-an-emergency-diy-1452349946) 。
*   [DIY 热袋](http://lifehacker.com/make-a-diy-microwave-heat-bag-1508417772) (内装大米、豆子等。).你也可以用这些东西装满一只袜子。
*   美容产品。真正简单的 [注意事项](http://www.realsimple.com/food-recipes/tools-products/14-surprising-uses-for-your-microwave-10000001035388/page4.html) 你可以对热油护发素、热脱毛蜡甚至面膜等产品进行短暂预热。
*   柠檬水。它竟然 [清理你的微波炉](http://lifehacker.com/use-a-lemon-to-make-cleaning-the-microwave-easy-1525960405) 。

**不要:**

*   干海绵，会着火。
*   几乎所有在 [上做过的事情，用微波炉加热是个好主意吗？](https://www.youtube.com/playlist?list=PLU4IMu04MIlJgB6Aaj07q-5iXuHGVzFAR) 显示。微波专家测试了数百种物品，包括会说话的贺卡、鞭炮、喷漆、皮卡丘(玩偶)和一个丙烷罐。这些通常导致融化，火焰，火花和烟雾。
*   生物。说够了。

**有争议:**

*   [肥皂](http://www.youtube.com/watch?v=7lAOOwMNodA) 。如果你不想惊讶或浪费一块肥皂，这真的是值得商榷的。
*   光盘。有时候你可能只想 [看一场灯光秀](http://www.youtube.com/watch?v=0JkCIfLE_-M) 或者毁掉一张 CD。
