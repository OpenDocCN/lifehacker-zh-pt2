# 播客《反叛的脚步》中的最佳政治行动建议

> 原文：<https://lifehacker.com/the-best-political-action-tips-from-the-podcast-rebel-s-1830239367>

你投票了。太好了！这几乎是参与民主的基本水平！这可能是你仍然感到焦虑和相对无力的原因。你下一步做什么？你如何更多地参与政治，尤其是在你没有很多时间或金钱的情况下？



试试，这是一个有限的系列播客，讲述如何参与政治，特别是左翼政治，尽管它的课程适用于多种意识形态。这是一个两小时的政治组织介绍，分为七个简短，友好的插曲。以下是我最喜欢的系列课程。

## 找到合适的参与程度

在第 4 集“加入一个团队”中，无政府主义者莎拉谈到了开发一个浏览器扩展( [没有法西斯主义的平台](https://chrome.google.com/webstore/detail/no-platform-for-fascism-t/khpfcpkmoldaeiknfjcgjfcnkojobicd/related?hl=en) ，它帮助人们报告违反 YouTube 服务条款的仇恨内容)。任何人都可以下载扩展，做一点低努力，低承诺的政治工作。但是这个工具背后的组织包括可以相互信任的人之间的闭门会议。

这种多层次参与的策略，每个小组被赋予与其信任和承诺程度相适应的职责，适用于各种各样的项目。不要太快冲进去自己组；寻找你可以帮忙的现有团体，并弄清楚你想参与到什么程度。你可能会发现，离开公司中心一两步会让你感觉最舒服。你还会发现，并不是每个人都能立即参与到每个计划会议中。政治团体必须小心政府的监视和来自反对派团体的攻击，所以他们不能邀请每一个新成员参加敏感的战略会议。和任何人际关系一样，你需要先建立信任，然后才能期待完全的接触和责任。

## 在正确的媒介中工作

几个消息来源告诉莉兹，在采取行动时，他们已经 [抛弃了一种媒介，转而使用另一种](https://rebelsteps.com/episodes/2-decorate-your-neighborhood.html) 。你需要根据你的目标受众、你的技能和你的风险偏好来选择你的媒体。有时候印刷杂志是有意义的，但是现在一个博客或者一个脸书页面可能会接触到更多的人。

这不仅仅是为了上网而放弃 IRL 活动。如果你和年轻人聊天，你会选择 Instagram、Snapchat 或 Twitter，而不是脸书。但是你也可以选择在校园里招聘，而不是在办公室外面。

除了你的目标，想想你的技能和风险。例如，如果你担心被逮捕或被骚扰，或者如果你知道的唯一设计是 MS Paint，那么你可能不会选择 wheatpasting 海报。你可以选择制作 gif 和 memes，就像 [进步数据](https://twitter.com/DataProgress) 在其科学图表之间发布的政治评论:
