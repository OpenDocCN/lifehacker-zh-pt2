# 使用这张地图找出哪些恐龙生活在你的地区

> 原文：<https://lifehacker.com/use-this-map-to-find-out-which-dinosaurs-lived-in-your-1844964437>

像许多孩子一样，我也经历了恐龙阶段。这些巨型蜥蜴怪物代替人类在地球上游荡的想法让我既害怕又好奇。所以当我的老师告诉我们恐龙也生活在我们的区域时，我需要了解更多。这主要包括在森林里挖洞寻找恐龙骨头，然后几周后找到所说的洞，并试图说服自己它们实际上是恐龙脚印。(在我内心深处，我知道真相。)



但那都是在互联网出现之前。今天的孩子们只需点击几下鼠标，就可以获得 [各种恐龙和其他信息](https://lifehacker.com/how-to-view-3d-dinosaurs-in-your-own-home-1844245800) ，包括 [这张地图](https://www.deseret.com/u-s-world/2020/9/1/21408531/dinosaur-map-hometown-interactive) 显示了哪些恐龙生活在你所在的地区。以下是使用方法。

### 如何使用恐龙地图

首先，这幅地图的功劳归于加州古生物学家 [伊恩·韦伯斯特](https://dinosaurpictures.org/ancient-earth#260) ，他也是许多 [其他有趣科技项目](https://www.ianww.com) 的幕后推手。使用 [映射](https://dinosaurpictures.org/ancient-earth/#240) 非常简单。进入你的家乡(或任何你有兴趣探索的地区)并准备了解一些事实。地图默认为 2.4 亿年前，但你可以通过页面顶部的下拉菜单进行更改。你也可以通过右边的下拉菜单选择一个特定的时期，从“第一次多细胞生命”一直到“恐龙灭绝。”

在您输入地名的框的正下方，您将根据在该地区发现的化石，找到一些生活在该地区的恐龙和其他早期生物的例子。另一个有趣的功能是把它带回到 7 . 5 亿年前，看看你的城市在地球上的地理位置。然后当你把日期改得越来越近时，你会看到大陆漂移是如何发生的以及我们是如何到达地球上现在的位置的。

除了作为一个有用的教学工具，韦伯斯特还把地图设计得很有趣，他告诉 CNN :

> 他说:“这是为了激发人们对科学家的兴趣，并有望赢得他们的尊重，他们每天都在努力更好地了解我们的世界和它的过去。”。“它还包含有趣的惊喜，例如美国过去是如何被浅海分割的，阿巴拉契亚山脉过去是非常高大的山脉，可以与喜马拉雅山脉相媲美，佛罗里达过去被淹没。”
