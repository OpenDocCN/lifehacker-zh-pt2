# 使用浸泡循环器，轻松制作荷兰酱

> 原文：<https://lifehacker.com/make-hollandaise-sauce-the-easy-way-with-an-immersion-c-1836184914>

呆在家里几乎总是比出去好，尤其是在中午之前。出去吃早午餐意味着排队、喝淡含羞草酒，以及(取决于机构)和孩子们在一起。但是如果这个地方有好吃的本尼迪克特鸡蛋，所有这些都是值得的，主要是因为荷兰酱是出了名的挑剔。

即使你是乳液大师，在炉子上做荷兰酱也需要*温和的*加热、*持续的*搅拌，以及比我希望在周末早晨提供的更多的关注。但是——在真空浸泡循环器的帮助下——你可以跳过所有的看护，直接喝含羞草饮料。

不管你喜欢哪种荷兰酱，真空烹饪会让你的生活变得无比轻松。我从 [ChefSteps 食谱](https://www.chefsteps.com/activities/can-t-f-it-up-eggs-benedict) 开始，它涉及到大量的质量测量，我惊讶于食谱中烹饪部分是如此的简单。你只需要把所有的配料都扔进一个袋子里，在袋子里煮一个小时，然后倒入搅拌机(或者用乳液搅拌机搅拌)，搅拌成完美、均匀、美味的荷兰酱。为了确保这种方法也适用于其他非 ChefSteps 食谱，我尝试了一个来自 [allrecipes](https://www.allrecipes.com/recipe/25227/creamy-hollandaise-sauce/) 的食谱，然后我又尝试了 [这个美食网食谱](https://www.foodnetwork.com/recipes/tyler-florence/hollandaise-sauce-recipe-1910043) 。一切都很完美。我确实发现酱汁比平时浓一点，但我不介意，如果你喜欢更稀的酱汁，你可以随时添加额外的一汤匙水。

选择你最喜欢的含有蛋黄、黄油和柠檬的食谱。把它们都放进一个可密封的冷冻袋里，然后放在 147 华氏度的水浴中一个小时。将它转移到搅拌机中，搅拌，然后放在你最喜欢的本尼迪克特上，让你感到安慰的是，你不必离开你的房子，穿好衣服，甚至站在炉子旁边。
