# 设定新的“家庭疫情规则”

> 原文:[https://life hacker . com/set-new-family-疫情-rules-1843242288](https://lifehacker.com/set-new-family-pandemic-rules-1843242288)

现在孩子们经常出现在屏幕上吗？熬夜？多睡会儿？整天整夜穿着睡衣？大家好，欢迎来到*疫情育儿网，*这个游戏展示了一切都是虚构的，分数并不重要。

如果听起来好像我有点崩溃，那是因为我不得不暂停工作一会儿，清理掉洒在我床底下的水——洒在一些书和报纸上，还有一大堆灰尘。地毯上的湿灰尘...真了不起。

我通常会允许我九岁的儿子在我的床上玩电脑游戏，一边吃薯片，一边在他旁边的地板上放满一杯水吗？不，我不会；听起来像是一团乱麻在等着发生！但这一次，他们呼吁改变优先事项。正常的家庭规则在这里不再适用，因为现在不是正常时期。如果你在担心你是如何系统地打破了你在第一周结束前制定的每一个屏幕时间规则，你可以停止。这并不是说你违反了规则；目前，你生活在疫情规则下。

我们知道生活会在某个时候回归正常。当那个幸福的时刻到来时，我们的孩子可能会认为每天连续玩四个小时《我的世界》游戏仍然是一件可以接受的事情。但是我们可以通过现在就和他们讨论疫情规则来帮助控制他们的期望。

把疫情规则想象成假期规则。并不是说疫情本身就像一个假期，而是住在里面的经历就像 T2，我们在度假的时候，我们的生活发生了巨大的变化。然而，当假期结束时，我们吃饭、睡觉和娱乐自己的方式会在一夜之间恢复正常。

根据你孩子的年龄、他们的脾气、他们的个人需求和你的优先考虑，每个家庭的疫情规则看起来会有所不同。也许你是一个严格的全餐和零食都在厨房餐桌上吃的家庭，但现在说厨房餐桌是一个临时办公室，所以不管怎样，把你的零食放在沙发上。一旦疫情结束，办公用品被送回真正的办公室，那些椒盐卷饼就又回到了桌子上。

也许你已经决定他们可以在周末拥有无限的屏幕时间——只要他们完成家务，或者在外面玩一会儿，或者做一些体力活动。那也行！这是疫情育儿法！

如果你的孩子足够大，你甚至可能想听听他们对疫情规则的看法——在讨论了它们的临时性之后——以增加认同。我发现我的儿子很愿意坚持 [一周内的学术常规](https://offspring.lifehacker.com/create-a-daytime-academic-schedule-for-kids-stuck-at-ho-1842357145)—*如果*我在的几个 30 分钟的休息时间里给打个盹，他就可以做他想做的事情(看电视)。我也曾经严格限制他一周嚼多少口香糖，但是现在他可以在自己的房间里随时嚼口香糖。为什么？我不知道；因为这让他开心，而我现在有更重要的事情要担心。

也许你更经常地对晚餐吃鸡块、通心粉和奶酪说“是”，或者你让他们晚睡一会儿，或者把他们的玩具扔得满地都是。我不是提倡你为了好玩而养成一堆坏习惯，但是你努力培养和坚持的那些好习惯可能和疫情不太相配。为了每个人的心理健康，可能需要做一些调整。

一旦这一切都过去了——当孩子们不可避免地被突然回到常规的规则所融化或抵制时——根据需要重复这句话:那是疫情规则，亲爱的。L ife 现在恢复正常，和所有的正常规则都适用。
