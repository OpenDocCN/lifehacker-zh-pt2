# 为什么你应该写一份恐吓清单

> 原文：<https://lifehacker.com/why-you-should-write-an-intimidation-list-1833204602>

我年轻的时候几乎没有什么不令我害怕的。独自在黑暗中睡觉，在我的舒适区之外上课，告诉我爱的人我爱他们，在啦啦队比赛中做后手翻——所有这些事情都给我带来巨大的压力，以至于我干脆避开它们。我睡在姐姐的房间里，上了很多我知道我会学得很好的课，而且从来没有在没有观察员的情况下表演过后手翻。

现在我对以上所有的事情都相当满意了(嗯，除了后手空翻)，而且没有什么事情是我仅仅因为害怕而避免的。但我确实经常反思我被恐吓打败的时候，不管是因为我觉得自己做不到而不去上课、参加体育活动、不去交谈、不去谈恋爱还是做其他事情。

正如玛丽亚·阿德科克斯为《T2》杂志撰写的文章和杰奎琳·麦克莱恩为《T4》杂志撰写的文章一样，盘点那些让你害怕的事情——以及为什么会这样——是有益健康的。是处理你的财务，还是反击一个令人恼火的老板？是和心爱的人进行一次特定的对话，还是承认自己知道的并没有自己以为的那么多？想想什么让你害怕，并写下来。

### 写下你的恐吓清单

和大多数事情一样，你的清单越具体越好。McElhone 建议花一周时间认真思考你的日常生活，考虑你为什么做或不做某些事情。

麦克莱恩写道:

> 我鼓励你试着深入一点，不要停留在你会想到的表面上。“冒险”、“失败”、“自信的人”都令人生畏——我明白了。但是还有什么呢？哪些具体的事、人、经历，让你感到恐惧？你是否每天都遇到一些困扰你的事情？
> 
> 你平时有没有回避做某件事，回避见某个人，或者回避谈论某个话题？是因为你害怕吗？

然后想清楚*为什么*这个人或那个情况会让你害怕。你害怕什么？当你明白了这一点，你可以采取必要的措施来解决这个问题，如果你愿意的话。

我的可能看起来像:

*   在阅读/活动后向作者介绍自己:我可能会问错问题，在我欣赏的人面前出丑。
*   写小说/以一种我以前从未尝试过的体裁写:我可能会发现我实际上做不到，所有关于我自己的最糟糕的恐惧都会被证实(我没有创造力、聪明、有才华等等)。).
*   **收拾行李搬到加州**:如果我没有一个由朋友和同事组成的固定社区，会发生什么？
*   **公开演讲**:请不要让我做。

最后，我要补充一点，恐吓不一定是坏事。有少数作家和创作者让我感到害怕，但这是因为他们的知识和/或艺术的广度和深度是如此之大。从这方面来说，他们是鼓舞人心的。

正如 McElhone 所写的，这种练习可以帮助你在生活和行动中找到你没有意识到的共同主题。“也许你会发现，你其实并不害怕你想象中的自己，”她写道。“也许你会发现，你所认为的你所害怕的，离征服只有一步之遥。这就是自我成长的起点。”
