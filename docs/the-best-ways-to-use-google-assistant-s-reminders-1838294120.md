# 使用谷歌助手提醒的最佳方式

> 原文：<https://lifehacker.com/the-best-ways-to-use-google-assistant-s-reminders-1838294120>

不管怎样，这些数字助理有什么意义呢？当然是为了让生活更简单。特别是谷歌助手，它不仅可以帮助回答生活中的紧迫问题和控制智能设备，还可以充当你自己的个人提醒机器人。



Assistant 提供了大量可编程的命令来帮助你上路，它的提醒可以在任何 Android 智能手机或安装了 Google Home 应用程序的 iPhone 上使用，也可以在兼容的 Google 智能扬声器和智能显示器上使用。它们很容易设置，而且这个特性不仅仅是大喊“嘿，提醒我做点什么。”不过，助理也做得很好。

### 对于忙碌的人来说

似乎不记得完成日常杂务，拿起干洗时，或补充您的宠物最喜爱的食物藏匿？谷歌助手可以帮助你处理所有这些任务。您可以按日期、日期和时间或按位置为自己设置特定的提醒。

首先，你需要打开助手的 [个人结果](https://support.google.com/googlenest/answer/7684543) ，可以在 iOS 和 Android 的家庭应用中找到。您必须为想要使用此特定功能的每个特定移动设备、智能扬声器或智能显示器打开该功能。然后在 Home app 的助手设置面板中设置 [语音匹配](https://support.google.com/googlenest/answer/7342711?hl=en) ，让助手在你设置提醒时识别你的声音。

从那里，触发该功能是很容易的。比如说，“好吧，谷歌，提醒我每晚 7 点调低恒温器的刻度。”助手将回复您，让您知道提醒已被存档。然后，它会在每晚的同一时间在智能扬声器或智能显示屏上发出提醒。如果你是用手机提醒的，你会看到一个弹出的通知。

若要设置基于位置的提醒，请尽量具体。例如，如果你在一个小镇，你可以让谷歌提醒你在任何时候去特定的杂货店买牛奶。如果你在一个大的郊区或市区，这些特定的区域设置会变得更加棘手。相反，尝试提及您希望提醒弹出的街道地址。对于你家或办公室附近的提醒，你可以将这些地址输入谷歌地图以获得更高的准确性。

### 对于有太多事情要做的学生

作为一名学生，当你要学习人生课程、参加聚会、准备考试时，很难跟踪每一件要学习的事情、要会见的人和要交谈的导师。助手的提醒是帮助你坚持工作的一个很好的方式，谷歌会保持一个长期运行的记录，直到你把它们标记为完成。不幸的是，没有一个单独的应用程序可以在不查看你的谷歌日历的情况下查看你迄今为止所做的提醒。但由于提醒页面在技术上是一个网络应用程序，你可以将一个特定的链接加入书签，放在手机的主屏幕上，以便于访问。

在手机浏览器中导航至[](http://assistant.google.com/reminders/mainview)*。(请注意，此 URL 在桌面浏览器上不起作用。)如果你在 Android 设备上使用 Chrome 浏览器，点击溢出菜单中的添加到主屏幕选项。然后 Chrome 会引导你创建一个书签小工具，放在你主屏幕的任何地方。其他浏览器可能允许您将链接添加为书签，尽管在主屏幕上放置小部件的附加功能可能不可用。*

*iOS 用户可以通过一个简单的书签访问这个特殊的提醒页面。在您的 iPhone 或 iPad 上，打开 mobile Safari，导航到上面的链接，然后从菜单中选择“添加到主屏幕”以在您的应用程序旁边添加快捷方式。*

### *对于沮丧的父母和室友*

*提醒不仅限于你。该助手还可以为家庭成员或室友编写提醒程序。唯一的先决条件是，你需要将那些人添加到你的 [谷歌家族](https://support.google.com/families/answer/6286986?hl=en) 群中，你可以通过[*【families.google.com】*](http://families.google.com)来设置。*

*如果是柏拉图式的情况，你可以将个人谷歌账户链接到家庭小组中的智能扬声器或智能显示器。这是通过 Google Home 应用程序设置的:点击主页视图——最左侧的底部标签——然后点击设置齿轮图标。在这里，点击第二个选项以查看家庭成员，然后点击屏幕右上角的加号以输入您要添加的人的电子邮件地址。Google 会邀请他们加入您的家庭群组。*

*要设置特定的提醒，可以这样说，“好的，谷歌，提醒弗兰克每周四晚上 8 点去倒垃圾。”如果助理接受了您的提醒，他会做出回应。当提醒的时间到来时，智能扬声器或智能显示器会发出一个信号，并宣布有一个对弗兰克的提醒。弗兰克还会在他们的移动设备上收到通知，前提是他们安装了 Android 或 iOS 版的助手应用程序。基于位置的提醒在这里也是一样的，就像你为自己设计一个一样。*

*家长们，您可以使用 [家庭链接](https://families.google.com/familylink/) 为您家中任何 13 岁以下的孩子设立一个特定的谷歌账户。他们可以得到提醒，这样助理就可以让他们负责每天早上整理床铺或者记住每周的家务。*

*目前，此 [特定功能](https://support.google.com/assistant/answer/9387035) 仅在美国、英国和澳洲可用。此外，如果你使用的是通过工作或学校提供的谷歌账户，它也不会起作用。*

### *对于健忘的人*

*不记得邻居的名字了？还是你的 wifi 密码？助手可以帮忙。请谷歌记住一个具体细节。措辞是，“记住我的邻居的名字是鲍勃。”当你想让助手回忆起这些信息时，明确地问:“我的邻居叫什么名字？”它会用你要求它记住的东西来回应。* 

*为了获得最佳效果，请尽可能简化信息。要求助理记住 wifi 密码的特定大写字母和空格可能会使编程过程变得复杂。但在很大程度上，这是一个有用的工具，可以记住那些不总是能立即访问到的共享信息。*

### *对于愤怒的记录员来说*

*谷歌助手提供了一个简单的“自我提醒”功能。请谷歌“记笔记”，它会将笔记保存在助手应用程序中。当你准备好访问这些笔记时，问谷歌，“我的笔记是什么？”它会在您的移动设备或智能显示器上显示该列表。你也可以使用“给自己写笔记”的命令，谷歌会让你选择是将笔记保存到谷歌 Keep 还是 Gmail。* 

*想要一种更有效的记笔记方式吗？你会想注册 IFTTT，这是一项有用的服务，可以与无数的联网设备和应用程序配合使用。IFTTT 的工作有点像拖放式编程，因为您可以设置参数，“如果发生这种情况，那么就会发生这种情况。”您不需要任何编程方面的工作知识，但是您将需要您计划用来授予访问权限的每个服务的帐户名和密码。更多关于 IFTTT 入门的信息，请查看我们的 [初学者指南](https://lifehacker.com/the-beginners-guide-to-ifttt-1819624556) 。*

*你可以通过谷歌助手创建一个 IFTTT 公式来归档笔记，无论是通过 iOS 和 Android 的移动应用程序，还是在你的桌面浏览器中。在本例中，我们将采用后者。*

*对于公式中的“If This”部分，搜索 Google Assistant 选项，然后选择“说出带有文本成分的短语”选项。在第一个空白处，在“你想说什么？”键入您要对助手说的话，以便它知道这个特定的命令。在其后输入一个美元符号($)，表示助手将等待您的输入。它可以是你觉得容易记忆的单词的任意组合。* 

*有一个空间让你决定你想让助手说些什么来回应你的命令。同样，这可以是向您指示该特定命令正在使用的任何内容。你可以在你希望助手重复你所说的地方键入美元符号，这样你就可以确认将被存档的短语。完成后，选择“创建触发器”*

*接下来，系统会提示您选择操作服务。IFTTT 的好处在于它提供了很大的灵活性。例如，如果你使用谷歌文档，你可以让你的笔记同步到一个特定的文档，或者在你每次写笔记的时候同步到一个新的文档。或者，如果你使用 Evernote、Todoist 或 OneNote 等服务，你可以在上面同步你的口头笔记。对于专业人士来说，这个功能也适用于 Trello 和 Asana。*

*如果您选择 Google Docs，IFTTT 将允许您选择文档名称和内容显示方式，以及将文件放在 Google Drive 中的什么位置。完成后，选择“创建操作”以固化公式。*

### *对于苹果提醒用户*

*我们明白——你宁愿不用谷歌提醒，因为你是苹果提醒的忠实用户。幸运的是，通过一点 IFTTT 的魔力，你可以设置一个小程序，将提醒直接归档到苹果提醒中。谷歌提供了官方的 [助理提醒 IFTTT 应用](https://ifttt.com/applets/WkZFqD8r-add-a-new-to-do-in-your-iphone-s-reminders-app-with-google-assistant) 。您可以像设置其他公式一样设置它；只需稍加编辑，就能确保每项服务都与你各自的谷歌和苹果账户同步。*
