# 使用牙线作为应急鞋带和厨房线

> 原文：<https://lifehacker.com/use-dental-floss-as-emergency-shoelaces-and-kitchen-twi-1838674334>

你知道“牙线舞”不仅仅是所有孩子都在跳的流行舞蹈吗？

事实上，简陋的牙线可以在家里到处使用。在厨房里，用未调味、未打蜡的牙线系住鸡肉或烤肉。它也非常适合切软奶酪、西瓜和蛋糕。(说真的，现在谁有奶酪丝？)你也可以用它把粘在锅里的烘焙食物刮出来。

如果你发现自己突然需要一根新鞋带，牙线可以解决这个问题。我们系好鞋子，有规律地系好，然后在一天的剩余时间里走来走去，没有一只鞋子断了！真正不可思议的东西。
