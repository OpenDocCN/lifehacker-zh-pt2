# 如何在疫情期间掌控孩子的监护权

> 原文：<https://lifehacker.com/how-to-navigate-child-custody-during-the-pandemic-1842473167>

在最正常的情况下，与前伴侣共同抚养孩子可能是一种尝试；当你的孩子的转移可能意味着他们或你暴露于冠状病毒时，共同养育是一个全新的挑战。

很少有监护权协议包括通过疫情导航育儿计划的规定，然而我们在这里，得到的命令是尽可能远离他人，在某些领域，完全呆在家里。所以什么时候你会说，“够了，孩子们会一直呆在这里，我不在乎你喜不喜欢”？

来自 [【布林克利·摩根](https://www.brinkleymorgan.com/) 的佛罗里达州家庭法律师肯·戈登说，在他的 25 名律师事务所中，没有人真正开始将如何处理新冠肺炎疫情期间监护权的语言写入育儿计划。不过，在某些案件中，尤其是那些仍处于早期阶段、可能只有临时法庭命令的案件，事情开始升温。

“理性解决方案和非理性解决方案之间的界限是模糊的，”戈登说。

### 可以选择和孩子隔离吗？

全国各地的父母每天醒来都在想，他们是否应该带着孩子去拜访另一位父母，或者孩子是否应该从一家到另一家，甚至是在同一地区。许多地区的居民被告知“就地躲避”，或者除了必要的外出之外呆在家里。但是拜访父母算是“必要的”吗？

“比如说，妈妈(或爸爸)说，‘我有孩子，我在看新闻，我相信这次疫情很严重，我选择就地躲避，’”戈登说。“嗯(如果你是另一方家长)，你打算怎么办？”

他说，最合理的答案可能是...没什么。警察不太可能介入监护权纠纷——他们会告诉你通过法院系统处理任何问题。你可以打电话给你的律师，试图得到一个听证会，但像其他事情一样，法院很大程度上采取了虚拟听证会，这很难协调，因此很难得到。

当这一切都结束后，戈登说，他认为选择和孩子呆在一起，让他们和另一方保持一定距离的一方父母，不太可能被判藐视任何先前存在的法院命令的协议。

“对他们来说，被判藐视法庭……意味着他们不得不故意违抗法庭命令，”他说。“在这个特殊时期，疫情和所有人都疯了，人们真的会感到不安或担忧不是没有道理的……所以为了证明他们的行为是轻蔑的，你知道，祝你好运。”

但是他说要记住，特别是对于那些仍在敲定抚养协议的父母来说，尽管他们在这种情况下可能不会被发现藐视法庭命令，但他们在这种时候的行为会被法庭注意到。

### 关注孩子的最大利益

当然，即使你决定和你的孩子呆在一起，这并不意味着父母应该利用这个机会限制和另一方的所有接触。如果在你住的地方有必要就地躲避，你应该尽一切努力确保另一方父母可以经常通过电话和视频聊天联系。

“通过‘这就是我所说的以孩子为中心’的过滤器，过滤掉你作为共同父母试图掌握的任何决定，”他说。

现在是时候保持耐心，认识到紧张局势可能正在加剧，并努力度过这个不寻常的时期，而不进一步加剧任何预先存在的敌意。

“这有点像西部荒野，不幸的是，会有很多人利用这种情况，”戈登说。“但他们可能认为自己在做正确的事情。”

基本上，这里没有硬性规定什么时候做什么，什么时候决定与任何人的任何接触，包括另一个父母，是太多的接触。但是现在是时候深入挖掘，尽你所能成为最好的共同父母，无论你是在家带孩子的一方还是最终被关在门外的一方。
