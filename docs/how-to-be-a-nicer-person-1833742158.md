# 如何成为一个更好的人

> 原文：<https://lifehacker.com/how-to-be-a-nicer-person-1833742158>

有没有觉得你*可能*是个混蛋？

让我们面对现实吧——我们都有这样的时刻，我们或许可以更友善一点。

Watch

也许有人在路上拦住你，你冲他们大声咒骂。也许有人偷了健身房的最后一台跑步机，你希望他们从未出生。也许有人在工作中再次偷了你的午餐，你让你的整个办公室变得懒散，你亲自找到了罪魁祸首，并像 Trunchbull 一样强迫他们吃了一整个巧克力蛋糕。

如果你曾经想成为一个更好的人，这并没有你想象的那么难。作者 [Austin Kleon](https://austinkleon.com/2019/03/09/how-to-be-kind/) 在他的博客上讨论了这个话题——其核心并不涉及向慈善机构捐款或更多地对人微笑——而是需要对我们周围的人多一点同情心。

试着更有礼貌一点？以下是你需要知道的如何表现得好(也要表现得好)——这样你就不会再对别人大喊大叫了。

### 假定每个人都是无辜的

在 Kleon 的文章中，他提到了作者兼牧师，，他提供了一些成为一个更善良的人的关键建议。其中一个教训尤其重要:假定人们是无辜的。

马丁在一段视频中说:“当对某人如何表达某事或某人做什么有疑问时，给他们加号。”。

不要假设你周围的所有人都是出于自私、肮脏的目的，而要假设最好的情况。你对他们的行为会反映出这一点——而不是消极的攻击性(即回避它们，简短的回答，偶尔翻白眼，等等。)，比起怀疑他们的可信度，你会更坦诚地与他们交流，并给予他们更多的尊重。

对于焦虑的人来说，紧张也会让人变得虚弱，所以它还有一个额外的好处，那就是释放你自己头脑中的精神空间。

### 不要无谓地谈论人

马丁提供了另一个解决屁眼问题的可靠建议:尊重任何缺席的人。

“换句话说，不要在背后议论别人，”他说。“我认为没有什么比诋毁他人更能损害我们的精神生活了。这是一种非常严重的缺乏同情心的行为，当然，这会让另一个人感到痛苦。”

在节日聚会上喝一两杯酒后发表自己的意见是很诱人的，但保持沉默会让你受益更多。坦白地说，你也会看起来更好——说别人坏话会让你看起来更糟，没有人想看起来像办公室里的狗屎。

是的，发泄的感觉很棒(尤其是喝酒的时候！)而如果你需要一个发泄机制，那就让别人去做吧。听一段发泄的对话本身就是一种放松的练习。也许他们会分享你内心深处的观点。

### 不要期待回报

在 Reddit [的帖子](https://www.reddit.com/r/AskReddit/comments/42b8t4/nice_people_of_reddit_any_tips_on_how_to_be_a/) 、u/ [中，SnowHesher](https://www.reddit.com/user/SnowHesher) 这样描述成为一个更好的人:

> “不要陷入‘秘密合同’的谬论。许多“好人”相信，如果他们善待每一个人，他们也会被善待。因此，当他们为人们提供帮助时，他们希望以帮助或友谊的形式得到补偿。”

换句话说，如果你想成为一个更善良的人，你不能对每一件好事都附加条件。为什么？这是肤浅的，你最终是在给别人和他们长期能为你提供的东西赋予金钱价值。

相反，尽你最大努力去做一件事，而不要承认这件事在将来会给你带来什么好处。是的，这很难(见:在《老友记》第一集里，菲比做了一件乐善好施的事，却没有以某种方式为自己谋利)，但不期待回报对不那么混蛋非常重要。

### 多原谅别人

本着尽可能假定他人无罪的精神，当人们承认错误时，你应该同样容易地原谅他们。你不想成为那个无法释怀的人。心怀怨恨会让你在感情上和关系中无法继续前进。

正如我们在 之前写的 [，宽恕也不意味着和解。你不必成为最好的朋友——这只是一种相互尊重的行为。所以能放手就放手吧。](https://lifehacker.com/how-to-forgive-someone-who-has-wronged-you-1671192403)

如果你想不开，也没关系！也许是一个冷酷的前任，在这种情况下，*他们是*混蛋，你的怨恨是完全合理的。
