# 你可以免费参加这 600 门在线课程

> 原文：<https://lifehacker.com/you-can-take-these-600-online-courses-for-free-1830066444>

你不再需要成为哈佛学生或普林斯顿毕业生，就能从这些精英机构的教学中受益。你可以通过 [MOOCs](http://mooc.org/) 在线访问一些他们最受欢迎的课程。



MOOCs(大规模开放在线课程)已经存在多年，但现在有数百门课程可供任何人在线学习，而且(大部分)是免费的。参加这些课程不会让你有资格获得学位，但是如果你有时间，并且一直对 [黑洞](https://www.class-central.com/course/coursera-astro-101-black-holes-11460) 或 [计算机如何工作](https://www.class-central.com/course/coursera-how-computers-work-12188) 感到好奇，比如说，你可以报名参加世界上一些最好的高等教育机构的课程。

从麻省理工学院到哈佛大学，再到伯克利音乐学院，很多大学都提供这些课程，而且很多都是自定进度的，所以你可以在任何方便的时候选修。一些例子:

*   [区块链技术](https://www.class-central.com/course/edx-blockchain-technology-11428?utm_source=qz&utm_medium=web&utm_campaign=new_courses_october_2018) 来自加州大学柏克莱分校
*   [DNA:生物的遗传密码](https://www.edx.org/course/dna-biologys-genetic-code) 来自水稻
*   [哈姆雷特的鬼魂](https://www.edx.org/course/hamlets-ghost)来自哈佛的
*   [精益研究面试技巧](https://www.class-central.com/course/edx-lean-research-skills-for-conducting-interviews-11441?utm_source=qz&utm_medium=web&utm_campaign=new_courses_october_2018) 来自 麻省理工
*   [普通话要领](https://www.edx.org/course/mandarin-chinese-essentials) 出自 MandarinX
*   [宇宙之心——社会中的机器人:是福还是祸？](https://www.class-central.com/course/edx-mind-of-the-universe-robots-in-society-blessing-or-curse-11345?utm_source=qz&utm_medium=web&utm_campaign=new_courses_october_2018) 来自代尔夫特理工大学
*   [概率:基本概念&离散随机变量](https://www.edx.org/course/probability-basic-concepts-discrete-random-variables-0)来自普度
*   [问题解决、编程和视频游戏](https://www.class-central.com/course/coursera-problem-solving-programming-and-video-games-11500?utm_source=qz&utm_medium=web&utm_campaign=new_courses_october_2018) 来自阿尔伯塔大学

还有 [多了很多](https://www.edx.org/course/) 。这篇文章解释了 [如何免费注册](https://www.class-central.com/report/coursera-signup-for-free/) 。请注意，虽然许多课程的视频仍然是免费的，但有些只提供付费部分，如作业和成绩。

你可以在这里搜索 [新课程](https://www.class-central.com/universities?utm_source=qz&utm_medium=web&utm_campaign=new_courses_october_2018) ，Quartz 提供的这个 [是根据学科内容分解的](https://qz.com/1437623/600-free-online-courses-you-can-take-from-universities-worldwide/) ，包括编程、工程、人文和数据科学。快乐学习。
