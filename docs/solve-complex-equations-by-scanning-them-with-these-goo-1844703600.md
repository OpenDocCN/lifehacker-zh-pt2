# 用这些谷歌应用程序扫描复杂的方程来解决它们

> 原文：<https://lifehacker.com/solve-complex-equations-by-scanning-them-with-these-goo-1844703600>

Google Lens 的一项新功能即将推出，它可以帮助学生学习数学和 STEM 相关的主题。 要使用 t he 新的“作业”搜索模式，你所要做的就是扫描你需要帮助求解的方程。它将在谷歌镜头 Android 应用程序中提供，并通过 [谷歌视觉搜索](https://lens.google.com/) 在谷歌照片、谷歌助手和谷歌 iOS 应用程序中提供。它现在也可以在谷歌拥有的 Android 和 iOS 上的苏格拉底教育应用程序中获得。



虽然家庭作业助手函数为你解出扫描的方程，但它不*只是*给你正确的答案。谷歌透镜和苏格拉底都概述并解释了所涉及的步骤，以帮助你学习如何自己解决问题。它们还提供了与你正在处理的方程类型相关的链接。例如，如果你扫描一个要求你求解 x 的线性方程，比如“x-4=10”，结果将包括在线教程甚至 YouTube 视频的链接，这些视频解释了如何解决类似的数学问题。

谷歌还创建了“超过 100 个 STEM 概念”的互动 3D 模型，教师和学生可以将其用作视觉学习辅助工具。这些模特将可以从谷歌镜头的家庭作业模式和安卓和 iOS 上的谷歌搜索应用 访问 [。](https://support.google.com/websearch/answer/9817187?co=GENIE.Platform%3DAndroid&oco=1)

### 如何通过谷歌视觉搜索获得数学和科学帮助

谷歌镜头的家庭作业模式目前还不可用，但谷歌已经概述了一旦它到来该如何使用:

1.  打开谷歌镜头应用( [安卓](https://play.google.com/store/apps/details?id=com.google.ar.lens&hl=en_US) )或使用谷歌视觉搜索谷歌照片( [安卓](https://play.google.com/store/apps/details?id=com.google.android.apps.photos&hl=en_US) 、 [iOS](https://apps.apple.com/us/app/google-photos/id962194608) )、谷歌助手( [安卓](https://play.google.com/store/apps/details?id=com.google.android.apps.googleassistant&hl=en_US) ) 或 [谷歌 iOS 应用。](https://apps.apple.com/us/app/google/id284815942)

2.  滑动到底部菜单的**“家庭作业”**模式——这是毕业帽图标。
3.  给你需要帮助的数学或科学方程式拍张照片。如果询问，点击以突出显示特定等式。
4.  将弹出一个新菜单，解释如何一步一步地解决问题，并提供其他学习资源的链接(如 3D 模型查看器)。

### 如果你不想等谷歌...

苏格拉底教育应用程序也使用相同的功能，现在就可以使用。下面是使用方法:

1.  安装苏格拉底式 app ( [安卓](https://play.google.com/store/apps/details?id=com.google.socratic)[iOS](https://apps.apple.com/app/apple-store/id1014164514))。
2.  打开应用程序，给你的作业拍张照片。
3.  用手指突出显示需要帮助的特定等式。
4.  该应用程序将向你展示如何求解方程，并提供其他有用资源和在线教程的链接。
