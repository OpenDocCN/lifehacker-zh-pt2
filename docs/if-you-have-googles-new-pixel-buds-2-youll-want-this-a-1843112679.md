# 如果你有谷歌的新 Pixel Buds 2，你会想要这个应用程序

> 原文：<https://lifehacker.com/if-you-have-googles-new-pixel-buds-2-youll-want-this-a-1843112679>

第二代像素芽是 [最后这里是](https://store.google.com/us/config/pixel_buds) 。虽然大多数人仍然需要等待他们的订单从谷歌的在线商店出现——因为，你知道，大多数实体店目前都关闭了——你可以通过下载刚刚推出的 Pixel Buds 应用程序来为他们做准备。

Watch

新的谷歌 Pixel Buds 应用程序 [出现在谷歌 Play 商店](https://play.google.com/store/apps/details?id=com.google.android.apps.wearables.maestro.companion) 上，在 Pixel Bud 2 本周发布之前完全没有宣布，一些 Pixel 用户报告说它会自动出现在他们的设备上。(事实上，您可以在蓝牙音频设置中直接找到它的功能。)

然而，如果你没有 Pixel，如果你想解锁新 Pixel Buds 的全部功能，该应用程序是必须下载的。这是因为该应用程序会单独显示每个耳塞和手机套的电池电量和连接状态，并允许您访问其他设置和功能，例如:

*   **寻找设备** -你的 Pixel Buds 2s 会像你的智能手机一样响，如果它们丢失了，希望能更容易找到它们。但是，它们需要与您的设备配对并打开。
*   **触摸控制** -关于如何使用 Pixel Bud 2 的触摸控制的教程。
*   **自适应声音** -根据周围环境的噪音水平调整入耳式音量。

还有修改谷歌助手控制和偏好设置的选项，打开或关闭入耳式检测，以及进一步的应用内故障诊断支持。

不幸的是，最初的 Pixel buds 不被支持，所以如果你已经订购(或计划订购)新的 Pixel Buds，现在安装谷歌 Pixel Buds 应用程序 更像是一个预期的举动。
