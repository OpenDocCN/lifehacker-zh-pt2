# Marmite 让任何蘸酱更好

> 原文:[https://life hacker . com/marmite-makes-any-dip-better-1838741952](https://lifehacker.com/marmite-makes-any-dip-better-1838741952)

Marmite 是调味品世界的 Fernet-Branca。就像苦味的 amaro 一样，超咸酵母酱也是极端两极化的，人们倾向于要么在发自内心的层面上鄙视它，要么全心全意地爱它。但是，即使你不是一个在烤面包上涂果酱的人，你也可以用它来给奶油涂酱和蘸酱增添魅力。

Watch

marmite 的味道最好描述为“如果酱油，但太多了怎么办？”它又稠又粘，超级咸，回味出奇的苦。这不完全是“令人愉快的”，但它是美味的。当与奶油、脂肪含量更高的配料(如酸奶油或奶油奶酪)一起调和时，它简直美味无比。

你不需要大量的马麦酱来活跃蘸酱；每杯蘸酱加半茶匙会增加一种深度的、烤过的、美味的味道，你不能把你的手指放在上面，但是再加多一点，蘸酱就会开始尝起来像马麦酱(这可能没问题，取决于你是谁)。加到 [洋葱蘸](https://kinja.com/api/profile/getsession?redirect=https%3A%2F%2Fskillet.lifehacker.com%2Fsetsession%3Fr%3Dhttps%253A%252F%252Fskillet.lifehacker.com%252Fmake-a-super-bomb-onion-dip-without-a-mix-1795872077) ，加到 [queso](https://skillet.lifehacker.com/3-cheesy-dips-that-are-way-better-than-whatever-chipotl-1818528845) ，加到一个 [芝士球](https://skillet.lifehacker.com/you-should-make-cheeseballs-for-your-holiday-party-1820847699) 。见鬼，你甚至可以把它搅进一点奶油奶酪里，做一个特别美味的百吉饼。(我今天早上才做的。很好。)
