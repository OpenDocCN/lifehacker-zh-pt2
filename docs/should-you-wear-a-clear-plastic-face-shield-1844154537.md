# 你应该戴一个透明的塑料面罩吗？

> 原文：<https://lifehacker.com/should-you-wear-a-clear-plastic-face-shield-1844154537>

当事情重新开始时，口罩是我们用来保护彼此安全的重要工具之一。但是面罩呢？这些是透明塑料，为你的脸提供一个防溅屏障。它们可以很好地补充口罩，但不能替代口罩。



### 面罩保护你免受他人飞沫的伤害

口罩最大的缺陷之一——尤其是布口罩——是它们不能很好地阻挡其他人的呼吸道飞沫。你不能只是戴上面具就认为自己是安全的；只有当另一个人也戴着的时候，你才会受到保护。

但是面罩是防溅塑料，它确实提供了额外的一层保护，如果人的水滴击中面罩而不是你的脸。请记住，盾牌的本质是它的底部是开放的。举例来说，如果坐在椅子上的人抬头看着你并打了个喷嚏，这个盾牌对你没什么作用。

### 面罩并不能很好地保护他人免受你的伤害

当你的呼吸飞沫从你的嘴 [和鼻子](https://lifehacker.com/put-your-mask-over-your-damn-nose-1844132568) 中散发出来时，一个贴身的口罩将会捕捉到大部分的飞沫。但是面罩的底部是敞开的，这使得它比面具更容易泄漏。

使用面罩 [的医护人员将*与*口罩](https://www.cdc.gov/coronavirus/2019-ncov/hcp/using-ppe.html) 配合使用。盾牌保护他们，也保持他们的面具清洁，所以他们可以继续戴更长时间。(这本身并不好——口罩真的应该一次性使用——但是很多医院仍然供应不足。)

### 面罩还有其他优点

除了保护你的嘴和鼻子，面罩还可以阻挡可能携带病毒的飞沫进入你的眼睛。它们也让你几乎不可能不小心碰到你的脸或面具。

面罩也很容易清洁——只需擦拭即可。它们也比口罩更安全，因为你不会不小心在下巴上戴了一个面罩，或者让鼻子露出来。

所以当你外出时，你应该戴口罩吗？一些专家认为你应该使用 ，或者是除了口罩之外，或者是在没有口罩的情况下代替口罩。

如果你将与可能生病的人密切接触，比如你在家照顾一名新冠肺炎患者，面罩绝对是一个好主意。一些医学专业人士，包括一些管理新冠肺炎测试的人，已经戴了几个月了。
