# 用特殊的玩具让换尿布变得更容易

> 原文：<https://lifehacker.com/make-diaper-changes-easier-with-a-special-toy-1830385875>

我的儿子刚刚庆祝了他的第一个生日，从他出生的第二天起，他就是一个真正的活跃分子。(医院在他出生时的阿普加分数旁边写着:“神经过敏。”)



早在这个孩子六个月大的时候，他就开始换尿布了，儿科医生把这比作试图把一只章鱼塞进袋子里。当我们珍贵的、蠕动着的快乐包裹几乎从换衣台上飞下来，然后又从我们的床上飞下来的时候——而且还是在同一天满身便便的时候——我意识到问题已经失控了。我向专家寻求解决办法:妈妈们的腰带下有成千上万的尿布。

### 进入:分散注意力的玩具

我了解到，除了给你的孩子变速(通过练习变得更容易)之外，关键是在你换尿布的地方放一个特殊的玩具。当——也只有当——你把它们擦干净，擦干净，换上干净的尿布时，你才把这个玩具给他们。

这是专为该特定活动保留的独家限时优惠。在短暂、珍贵(如果不值得纪念的话)的时刻，你需要一个玩具来让他们的小手和小心思完全被占据。

以我的经验来看，最有效的是一个互动的小玩意，当你处理大大小小的业务时，它会让你的小家伙惊叹不已、分心并参与其中。想想按钮、灯或声音。对于我们的儿子来说，当你把他放下的时候，他就像离开水的鱼一样四处乱扑腾，一个音乐填充章鱼 创造了奇迹(我知道其他父母也非常喜欢这个神奇的软体动物)。

其他非常成功的干扰物包括玩具手电筒、婴儿电话、拨浪鼓或小型乐器，如铃鼓或马拉卡。把它像珍宝一样端上来:“嘿，嘘，孩子——看看这个！但是要快，因为我们一结束这里的工作，我就要把它藏起来。”

为了在你家外面给婴儿换尿布，在你的尿布包里放一个玩具哨子，一本柔软的活动书或者一条 [出牙项链](https://www.amazon.com/Teething-Necklace-Greeting-Natural-Silicone/dp/B074KG23XB/ref=sr_1_22?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/make-diaper-changes-easier-with-a-special-toy-1830385875&asc_source=&ie=UTF8&keywords=teething necklace for mom&qid=1541555684&s=toys-and-games&sr=1-22&tag=kinjalifehackerlink-20) ，上面有五颜六色的咀嚼珠。如果你的宝贝突然爆胎，一个笔记本，一张对折的纸，或者你的钱包在极度绝望的时候都很好用——任何容易打开和关闭的东西都会让他们忙起来，至少暂时如此。

为了让事情保持有趣——也为了让你的孩子不会厌倦——继续寻找那些真正吸引他们注意力的东西，并且在他们成长的过程中不要太笨重。这样，你就可以偶尔把你的特殊玩具藏起来，以确保换尿布不会在你家里造成灾难。
