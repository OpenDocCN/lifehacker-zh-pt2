# 现在是买车的好时机吗？

> 原文：<https://lifehacker.com/is-now-a-good-time-to-buy-a-car-1842648636>

最近看过新车广告吗？它们可能以诱人的价格和慷慨的融资选择为特色。这并不是因为汽车制造商的巨大改变；相信我，他们还是想赚钱的。汽车现在很便宜，因为冠状病毒 让人们呆在家里——推迟了他们可能不得不购买新车的计划。



与去年同期相比，汽车销量下降了大约 35%——这是自 10 年前经济衰退以来的最低水平。我们仍处于冠状病毒疫情的最严重时期，这意味着消费者对就业和金融稳定的不确定性不太可能很快导致“正常”的汽车购买。

汽车制造商和经销商正在提供各种各样的交易，试图吸引你在这个时候买车。即使从技术上来说，展厅并不对公众开放供其浏览，人们仍然可以选择购买汽车。

那么，价格有多好呢？

嗯，它们还不那么令人兴奋——至少，如果你只看价格标签的话。你可能会得到一个比平时更好的交易，或者你会发现从汽车的窗户价格开始谈判会更容易。

但真正吸引人的是:融资提议。

《凯利蓝皮书》和 [的执行出版人卡尔·布劳尔(Karl Brauer)说:“我们已经看到许多制造商向买家提供财务援助，从 0%融资到延期付款不等。](https://www.kbb.com/)

布劳尔说，仅仅为了一笔交易而考虑购买是不值得的。但是如果你想买一辆新车，他说现在对购买者的激励是“特别的”。他说:“六七年的零利率贷款累积起来就是真正的金钱储蓄，而以零利率推迟 60 或 90 天的首次付款会增加储蓄。”布劳尔指出，一些汽车制造商，如菲亚特克莱斯勒、大众和起亚，同时提供两种激励措施。

现在购买最大的好处是什么？你可以在自己舒适的家中完成大部分工作。布劳尔说，你仍然应该试驾车辆，同时仍然练习负责任的物理距离。(提供家庭试驾和送车服务的经销商会定期消毒汽车。)“买车的其他所有环节现在都可以在网上完成，包括融资、保险，甚至签署最终文件。消费者应该能够找到提供所有这些服务的经销商。”

像往常一样，在你决定购买之前，一定要做好调查。 [Edmunds](https://www.edmunds.com/) 、Autotrader 和 [消费者报告](https://www.consumerreports.org/cars-) 提供了大量的评级和评论，我们在 [Jalopnik](https://jalopnik.com/) 的朋友总是有好的建议。凯利蓝皮书关注汽车制造商的融资激励和支付援助计划。

这些困难时期的一个小好处是:汽车制造商和经销商不太可能在疫情之后放弃他们的数字销售工具，这意味着下次你准备购买时，你可能需要在停车场花更少的时间。
