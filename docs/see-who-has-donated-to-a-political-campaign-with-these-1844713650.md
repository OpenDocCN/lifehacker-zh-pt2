# 使用这些工具查看谁为政治活动捐了款

> 原文：<https://lifehacker.com/see-who-has-donated-to-a-political-campaign-with-these-1844713650>

悄悄了解某人的政治和他们真正关心的问题的一个方法是弄清楚他们资助了哪些竞选活动。有时你会发现 [意想不到的惊喜](https://www.snopes.com/fact-check/trump-donated-kamala-harris/) 或者了解到某人把钱投在哪里并不总是仅仅与党派有关。或者你可能会发现，一个宣扬某种价值观的组织 [用它的美元](https://popular.info/p/kelly-loeffler-is-running-against) 投票是不同的。



如果你想知道你的家庭成员、邻居、雇主或喜欢的公司是否为各种政治候选人捐过款(以及捐了多少)，你可以通过搜索公开的财务披露记录轻松找到这些信息。

这是这项工作的最佳工具。

### 如何跟踪对国家运动的捐赠

所有参议院、众议院和总统候选人(以及任何被授权为他们筹款的委员会)都必须向联邦选举委员会报告竞选捐款。

FEC 有 [一个可搜索的捐助者数据库](https://www.fec.gov/data/receipts/individual-contributions/?two_year_transaction_period=2020&min_date=01%2F01%2F2019&max_date=12%2F31%2F2020) ，让你可以通过接收者或捐助者浏览捐助者，并通过邮政编码进一步缩小搜索结果，捐助者职业，捐助金额等等。

然而，仔细阅读联邦选举委员会的报告可能会感到有些力不从心。另一个更容易管理的选择是非营利、无党派的响应政治中心(Center for Responsive Politics)，该中心跟踪美国选举和政治中的资金，并编制游说者、政治行动委员会(PAC)、利益集团和其他捐助者的详细支出报告。

该中心的网站，OpenSecrets.org，有一个可搜索的竞选捐款数据库，告诉你一个捐款人给了 T4 哪些竞选或候选人。您也可以使用高级搜索选项，改为按收件人、进行排序，或者按活动周期和捐赠者所在的州缩小结果范围。

另一种浏览投稿的方式是用中心的 [获取本地！工具](https://www.opensecrets.org/states/)T3。从下拉菜单中选择候选人所在的州，然后使用选项卡按捐赠者、候选人、城市地区等列出报告。这个工具还有一个邮政编码搜索功能，可以让你将捐款的范围缩小到你的邻居。

### 如何跟踪对州和地方竞选活动的捐赠

联邦选举委员会和 OpenSecrets.org 只跟踪联邦竞选和候选人(尽管你也可以用它们来查看对政党和政治行动委员会的捐赠)。如果你想知道谁给一个州或地方的竞选活动捐了款，你需要去国家政治金钱研究所。

使用 FollowTheMoney.org 的网站、、并不十分直观，但是有几种浏览和搜索的方法。一种是向下滚动地图，选择你感兴趣的州；这将打开一个带有州概况的新标签，在那里你可以选择办公室类型(例如州长)来缩小特定种族的捐赠细节。

另一个选择是滚动地图到工具区，使用下拉菜单选择你的州和选举周期。最后，试着在主搜索栏中输入一个名字——这将显示出个人的贡献，如果他们也是候选人，详细说明他们收到的钱。

如果你在 FollowTheMoney 上找不到你想要的，你可以直接去 [你所在州的竞选财务或选举委员会网站](https://www.followthemoney.org/resources/state-disclosure-agencies) 搜索披露表格。

也有基于州的组织，像弗吉尼亚州的 [公共访问项目](https://www.vpap.org/) ，追踪地方选举中的资金。在谷歌上快速搜索“竞选披露[州]”，如果这些组织在你所在的地区存在的话，应该可以帮助你追踪到它们。
