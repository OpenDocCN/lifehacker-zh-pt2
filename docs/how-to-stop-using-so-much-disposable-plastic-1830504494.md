# 如何停止使用这么多一次性塑料

> 原文:[https://life hacker . com/how-to-stop-use-so-much-disposable-plastic-1830504494](https://lifehacker.com/how-to-stop-using-so-much-disposable-plastic-1830504494)

我们正在用塑料淹没这个世界。它 [冲上了我们的海滩](https://www.smithsonianmag.com/smart-news/waves-garbage-are-washing-beach-dominican-republic-180969747/) ，它 [在垃圾填埋场被掩埋了几个世纪](https://slate.com/news-and-politics/2007/06/do-plastic-bags-really-take-500-years-to-break-down-in-a-landfill.html) ，它漂浮在海洋上，形成一团微小的微粒 [体积是得克萨斯州的两倍](https://www.theoceancleanup.com/great-pacific-garbage-patch/) 。其中大部分——在美国占 75%——从未被回收。回收它仍然需要制造新塑料能量的 10%——并非一无所有。一开始少用一点就好了。

## 用可重复使用的袋子代替塑料袋

带着 [可重复使用的农产品袋](https://www.amazon.com/Reusable-Produce-Bags-Fruits-Veggies/dp/B002UXQ7QQ/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-stop-using-so-much-disposable-plastic-1830504494&asc_source=&tag=kinjalifehackerlink-20) 去杂货店，说 [艾米丽 M-M](https://twitter.com/metalgf/status/1063093199969927170) 。把手提袋放在你的车里或钱包里。(拿折叠起来小的。)重复使用你的拉链，至少是干的。

这里有一个重要的警告:不要购买塑料替代品，除非你准备使用它。如果你不使用它们， [可重复使用的购物袋只是垃圾](https://lifehacker.com/how-to-actually-use-your-reusable-grocery-bags-1821390104) 的一个新创新。拒绝免费赠品或找到一个好的用途。(我把家里莫名其妙堆积起来的所有普通袋子都变成了“赠品袋”，里面存放着我捐给慈善商店的所有捐款。)

## 更换你的另一个一次性塑料

尝试一些 [可重复使用的保鲜膜](https://www.amazon.com/Organic-Reusable-Food-Wraps-Etee/dp/B077Z1VVLL/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-stop-using-so-much-disposable-plastic-1830504494&asc_source=&tag=kinjalifehackerlink-20) 代替保鲜膜，像[Kasia Mychajlowycz](https://twitter.com/xokasia/status/1063124870492553218)。尝试用条形肥皂代替塑料瓶装沐浴露，比如 Aimee Louise Sison。寻找用纸而不是塑料卖同样东西的品牌。下次你举办聚会的时候，看看你是否有足够的真正的菜肴来覆盖所有人。

当你买一次性物品时，再次倾向于纸。

## 携带自己的食物容器

简单水平:自己做咖啡，如果你想在路上喝，用热水瓶。一个热水瓶，你保存多年。在办公桌上放一个玻璃水瓶，在包里放一个 [可折叠水瓶](https://lifehacker.com/the-platypus-plusbottle-collapses-to-save-space-is-the-5895986) 。

中级:自带可重复使用的银器，比如 [克里斯汀修士](http://ana cecilia alvarez) 。这有点奇怪，但作为奖励，你再也不用那种真的很差的塑料叉子了——那种看起来是用来装零食物的叉子。

难度:安娜·塞西莉亚·阿尔瓦雷斯 说 [带自己的外卖容器去餐馆。处理你自己的狗食袋，而不是让工作人员为你的剩菜剩饭去取一些新鲜的垃圾。这有点尴尬，需要更多的计划，但嘿，你可以选择自己的特百惠。](https://twitter.com/_llorona/status/1063126376360755200)

## 少点外卖

这对我来说很难。虽然阿尔瓦雷斯是正确的，我们通过外卖和送货的实际吨塑料，切换到更多的家庭餐是一个实际的时间承诺。但我肯定没有充分考虑到每次我在街对面点印度菜时会扔掉多少垃圾。

## 查找你当地的回收规则

糟糕的回收让我疯狂。Lifehacker 办公室的一些人——当然不是 Lifehacker 的员工！—把该死的*塑料袋*扔进蓝色垃圾桶。你不能这么做！大部分地方都没有！你在给回收中心制造问题，你在破坏其他人的好工作！

是的，令人沮丧的是不同的城市和州有不同的回收规则。但这不是随意的——不同的回收设施有不同的能力——你真正需要学习的是你居住和工作的地方的规则。

如果你想回收塑料袋，你需要多做一点点工作:把它们放在你的口袋或钱包里，然后放到最近的杂货店或连锁药店。很多连锁店的前面都有塑料袋回收箱；一些州强制要求这样做。Twitter 的 [@shityeahitscool](https://twitter.com/shityeahitscool/status/1063107570729185280) 指向[PlasticFilmRecycling.org](https://www.plasticfilmrecycling.org/)，在这里可以找到附近的落客地点。养成这个习惯并不难。最糟糕的情况是，当你在杂货店时，你的口袋里有几个袋子，但你又忘记带手提包了。

这些把戏不会拯救世界。塑料只占我们国家垃圾的 10%,我们每个捡塑料的人都生活在大量企业和工业垃圾的阴影下。但是实践和规范谨慎消费是健康的，减少对石油的需求会进一步削弱这些工业巨头的实力。
