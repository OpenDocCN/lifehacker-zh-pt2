# 科技公司首席执行官的生物黑客，评级

> 原文：<https://lifehacker.com/tech-ceos-biohacks-rated-1834586864>

似乎每隔几周就有一个新的关于技术兄弟如何度过早晨的采访。我们完全想从成功人士的习惯中挖掘生活窍门——毕竟，你现在看到的是 [我如何工作](https://lifehacker.com/c/how-i-work) 的家。但是这些生物黑客有多有效呢？这里有一个固执己见的纲要:

### 很早就醒了

**判决:**瓦克

为什么:你需要睡眠，该死的！如果你工作或聚会到深夜，不要试图在凌晨 5 点或 4 点或 3 点醒来。如果没有一整夜的睡眠，你只是在欺骗自己。早起工作是没有效率的，这是愚蠢的。



这个规则有一个例外:如果你是一个早起的人，并且你总是准时上床睡觉，你可以在任何你想要的时候醒来。杰夫·贝索斯说 他醒得很早，但他也优先考虑每晚睡八个小时。*那是*一个黑客。

### 禁食

**判决:**可以是黑，但要看情况。

**为什么:**已经有一个词表示当你饿的时候不吃东西，因为你希望食物的缺乏对你的身体产生一些影响，而 [就是*节食*](https://jezebel.com/dont-be-fooled-by-extreme-fasting-1832787660) 。间歇性禁食可以是控制你摄入热量的一种方法。从这个意义上来说， [是一个 hack](https://vitals.lifehacker.com/we-finally-have-long-term-data-on-an-intermittent-fasti-1794877196) 。

但是禁食也很容易失控，成为一种饮食紊乱的形式。它也因其精神上的好处而被过度宣传。正如任何节食者都会告诉你的，一段时间不吃东西会导致一种专注的、轻微的欣快感，这种感觉开始表现在“饥饿”和“饥饿”之间。
