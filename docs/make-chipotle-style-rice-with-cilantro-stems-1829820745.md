# 用芫荽茎做墨西哥风味米饭

> 原文：<https://lifehacker.com/make-chipotle-style-rice-with-cilantro-stems-1829820745>

自从 2017 年 的 [大溃败之后，我就没有再去过 Chipotle 与奶酪有关的心碎需要最长的时间才能愈合——但我仍然对他们的香菜石灰饭情有独钟。幸运的是，这并不难复制，而且可以用芫荽叶茎做成*和*。](https://lifehacker.com/3-cheesy-dips-that-are-way-better-than-whatever-chipotl-1818528845) 



*这是《与克莱尔一起吃垃圾》的一部分* [*，这是一个生活黑客系列，克莱尔·劳尔说服你将你的厨余转化为可食用的美味。*](http://lifehacker.com/tag/eating-trash-with-claire)

芫荽叶得到了很多人的喜爱(和憎恨)，但它的叶子通常得到所有的关注。然而，茎有一吨的味道，它们可以用来调味卤汁，股票，salsas，和我个人最喜欢的大米。只需将半杯切碎的茎加入三杯生米中，加上一茶匙盐、少量橄榄油和月桂叶，然后像往常一样烹饪。(如果你没有电饭锅，看看我们的烹饪指南。 )米饭煮好后，用叉子弄松，然后加入四分之一杯鲜酸橙汁和一汤匙柠檬汁搅拌。尝一尝，如果需要的话再加一点盐，在你舒适的厨房里用它来做你自己的玉米煎饼碗。(我推荐用 [腌猪肉](https://skillet.lifehacker.com/sous-vide-carnitas-and-eat-tacos-all-summer-long-1825761771) 或 [这种泡菜猪肩](https://skillet.lifehacker.com/this-kimchi-pork-roast-will-warm-you-to-your-bones-1829755972) ，搭配大量的 [烤](https://skillet.lifehacker.com/grill-vegetables-before-pickling-or-marinating-them-1828057997)[腌玉米](https://skillet.lifehacker.com/you-should-put-pickled-corn-on-everything-1827544519) 。)
