# 用这本书帮助孩子学习重要的生活技能

> 原文:[https://life hacker . com/help-kids-learn-important-life-skills-with this-book-1844305679](https://lifehacker.com/help-kids-learn-important-life-skills-with-this-book-1844305679)

新书 [*发布的时机:65 在你长大之前学习的极其有用、超级重要的技能*](https://www.storey.com/books/how-to-be-a-person) 真的再好不过了。我敢肯定，作家凯瑟琳·纽曼(Catherine Newman)并没有预料或打算让她这本专注于教授孩子们基本生活技能的书在疫情期间发行，但我们还是来了。随着我们的孩子回家过暑假，以及未来几个月不同程度的隔离，这似乎是他们学习如何切洋葱、写信封地址或缝纽扣的最佳时机。

Watch

当我第一次开始通读*如何成为一个人的时候，*我只是想从中挖掘一些有用的想法与大家分享。但是当我读到第一部分的时候——标题是“其他生物:如何照顾你生活中的人、宠物和植物”——我哽咽了。整件事读起来像是一个值得信赖的导师给你的温和建议。她在引言中写道，这是直接对孩子们说的:

> 听着，我们理解也许这本书不是你自己选的。也许你的成年人甚至消极主动地在你生日的时候把它连同一副橡胶手套和一个活塞一起送给了你。(不客气！)但这真的没关系，因为我们相信，许多我们倾向于认为是家务事的任务可能是真正值得的——特别是如果你有机会自信地走向它们，而不是被唠叨的推土机推着走，否则就是你的妈妈或爸爸。

纽曼的写作清晰而直接——有时甚至很有趣——从不觉得自己对孩子有优越感或偏见。每一部分都有 2-4 页长，以准图形小说的风格编写，并配有可爱的插图(由 Debbie Fong 编写)，因此很容易消化。她涵盖了各种主题，分为七章:

*   其他生物:如何照顾你生活中的人、宠物和植物
*   正确表达:如何友善地表达你的观点
*   脏东西:如何清洁和护理你的家
*   可食用的食物:如何做饭并在厨房里找到自己的路
*   你穿着那件？:如何清洁和护理你的衣服
*   你的两分钱:如何获得、给予和花钱
*   有用的技能:如何做基本的重要事情

这本书真正打动我的是，它非常适合那些在生活中没有强大榜样的孩子，他们可能会觉得向朋友或老师询问如何做简单的日常事情很尴尬，如包装礼物或炒鸡蛋，更不用说更复杂的事情，如学习如何省钱。这本书的格式很容易浏览，让孩子们迅速找到最能引起他们共鸣的话题。

纽曼还写道，“事实是，做有意义的事情感觉很好，被欣赏的感觉也很好，这些技能很大程度上保证了这两种好感觉的结合。”
