# 黑客如何在机场窃取你的数据

> 原文：<https://lifehacker.com/how-hackers-can-steal-your-data-at-airports-1834976072>

有没有想过在机场使用流量很大的公共 wifi 网络上网是不是一个坏主意？多半是 [安全](https://www.wired.com/story/hotel-airport-wifi-safe/)；只要您连接到正确的网络，您的信息可能不会有风险。另一方面，你*可能*想重新考虑把你的手机插到门口的 USB 充电站。



网络犯罪分子可能会在你不同意或不知情的情况下使用这些网站下载你的数据并安装恶意软件，正如本周《福布斯》 所解释的那样——这种犯罪被称为“ [果汁劫持](https://money.cnn.com/2017/02/15/technology/public-ports-charging-bad-stop/index.html) ”

IBM Security 负责 X-Force 威胁情报的副总裁 Caleb Barlow 告诉《福布斯》[“插入公共 USB 端口有点像在路边找到一把牙刷，然后决定把它放进嘴里。“你不知道那东西去过哪里。记住，USB 端口可以传输数据。”](https://www.forbes.com/sites/suzannerowankelleher/2019/05/21/why-you-should-never-use-airport-usb-charging-stations/#78a6b5995955)

虽然这完全有可能，但你还不需要恐慌。没有大量的报告显示在机场发生了抢果汁事件(尽管如果在我们不知情的情况下发生，这可能是一种比我们知道的更频繁的犯罪)。

如果你不喜欢用路边的牙刷刷牙，或者为了安全起见，你可能应该使用你的标准充电器，并将其插入插座。据哈佛大学伯克曼互联网与社会中心的安全技术专家兼研究员布鲁斯·施奈尔(Bruce Schneier)称，你还可以购买一个 [USB 避孕套](https://shop.syncstop.com/products/usb-condom?variant=35430087052) ，这是一种通过切断数据引脚来阻止通过 USB 电缆进行数据交换的设备，只有 [允许电源引脚](https://int3.cc/products/usbcondoms) 连接。

你还应该在任何旅行中携带一个便携式电池设备——它可以是一个真正的生活救星。
