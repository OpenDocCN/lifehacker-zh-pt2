# 你现在可以免费在线参加常春藤联盟的课程

> 原文：<https://lifehacker.com/you-can-now-attend-ivy-league-classes-online-for-free-1842907160>

如果你一直想上哈佛，现在可能是最佳时机:哈佛大学 [向所有报名者免费提供 64 门不同的在线课程](https://online-learning.harvard.edu/catalog/FREE?page=1) 。

课程设置包括从网络编程和音乐理论到气象学和下一个疫情的规划。

这些课程的时间长短不一，但通常是五到十周的课程。

点击课程主页上的课程，您将获得更多关于所花时间的信息，以及材料的先进程度和您希望在课程结束时学到的东西。

例如，如果你报名参加“神经科学基础:神经元的电特性”课程，你可以按照自己的节奏每周学习大约 3-5 个小时。有一个条件:虽然你可以免费参加、的课程，但如果你支付费用，你只能在课程结束时获得一份经过验证的结业证书。

除了哈佛，许多其他常春藤联盟的大学——包括布朗大学、康奈尔大学、普林斯顿大学、耶鲁大学和哥伦比亚大学——也提供免费的在线课程。

像哈佛一样，这些课程包括从工程和数学到艺术和设计课程。 [查看这些机会的完整列表(有数百个！)上](https://www.classcentral.com/collection/ivy-league-moocs)班中央。
