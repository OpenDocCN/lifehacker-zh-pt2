# 要开始早起，承认这很糟糕

> 原文：<https://lifehacker.com/to-start-waking-up-early-acknowledge-that-it-sucks-1840899880>

如果你很难早起，接受一个简单的事实可能会有所帮助:在第一个清晨起床就像撕掉创可贴。

Watch

你可以试着用我们的睡眠和早晨相关的技巧让事情变得简单。给自己 [充足的睡眠时间](https://lifehacker.com/sleep-opportunity-is-the-key-to-actually-getting-enou-1821151153) ，晚上避免 [屏幕、咖啡因和酒精](https://lifehacker.com/get-better-sleep-and-start-mornings-strong-with-the-10-1762499118) ，设置一个 [实际有效的闹钟](https://lifehacker.com/this-evil-alarm-clock-app-is-the-only-reason-i-m-awake-1793682623) 。但是如果你没有心情在闹钟响的时候醒来，养成这个习惯的唯一方法就是*真的起床*。

前海军海豹突击队·乔科·威林克说:“过攻击性的日子比被攻击好得多。”。这里他有一点 [鼓舞人心的话](https://www.cnbc.com/2019/02/19/ex-navy-seal-how-to-get-out-of-bed-when-you-dont-feel-like-it.html) 关于早上 4:30 醒来的秘密。这个秘密是:你决定什么时候起床，如果 4:30 是你的决定，那么，你坚持。

我知道这很糟糕。意志力被高估了，所以在大多数情况下，我会说如果你需要依靠意志力，你可能已经让自己失败了。但是如果你是一个试图早起的夜猫子——这是可能的！—你需要明白一开始会很糟糕，无论如何你都需要站起来。几天后，你的身体会适应新的起床时间。但是建立这种习惯的唯一方法就是不管你喜不喜欢，都要准时起床。
