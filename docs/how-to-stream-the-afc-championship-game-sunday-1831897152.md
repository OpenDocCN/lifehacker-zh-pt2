# 如何直播周日的亚冠比赛

> 原文：<https://lifehacker.com/how-to-stream-the-afc-championship-game-sunday-1831897152>

新英格兰爱国者队将于本周日东部时间 6:40 在亚足联冠军赛中迎战堪萨斯城酋长队。这场比赛的获胜者将继续参加今年的超级碗比赛。



这场比赛正在哥伦比亚广播公司播出，所以只要你有一台电视和至少一个数字天线，你就可以开始了。有线电视用户也可以登录哥伦比亚广播公司的体育应用程序，并在他们的电脑上捕捉所有的动作。

但是如果你没有我所说的这些“电视”呢？你还有选择。以下是如何在流媒体服务上玩游戏:

### CBS 所有访问

CBS All Access 是传统电视服务之外观看比赛的最便宜的选择之一。这项服务每月 5.99 美元，不含广告每月 9.99 美元。这 200 个市场只有[live](https://cbsi.secure.force.com/CBSi/ViewArticle_allaccess?popup=true&aId=kA0E0000000PrrH&categories=CBS_Entertainment%3AWatching_Live_TV&template=template_cbsvod&referer=cbs.com/vod&data=&cfs=SFS_FT)有。

### DirecTV Now、YouTube TV、Hulu with Live TV、富波电视

在大多数市场，DirecTV Now、YouTube TV、Hulu with Live TV 和富波电视都提供对 CBS 的访问。绝对值得注意的是，我在那里说了“大部分”。有可能你的市场不会提供 CBS 访问其中一个或所有这些，所以一定要在游戏时间之前检查。

这四家公司都提供每月 40 美元的服务，如果你以前没有用过，还可以免费试用。富波电视是一个以体育为中心的网站，如果你只想观看现场体育赛事，这可能是一个不错的选择。富波电视的第一个月是 40 美元/月，但随后价格略有上升，达到 45 美元/月。

### playstation vue(游戏机视图)

PlayStation Vue 在大多数市场都提供 CBS 服务(再次确认你的服务是否包括在内)，但价格比其他一些替代产品稍高，每月 45 美元。
