# 给你欣赏的同事写更多的感谢信

> 原文:[https://life hacker . com/write-more-thank-you-aware-notes-to-colleges-1837845232](https://lifehacker.com/write-more-thank-you-notes-to-colleagues-you-appreciate-1837845232)

专业交流包括很多感谢。对于电子邮件来说，这是一个必不可少的结尾，即使是那些不需要收件人采取行动的邮件。但是，你上一次在 37 封邮件之外坐下来给同事写感谢信是什么时候？

Watch

自由作家艾米丽·莫纳科 [最近为缪斯](https://www.themuse.com/advice/sending-thank-you-note-each-week-changed-how-i-work) 写了一篇关于她努力增加感谢信数量的文章。她从朋友和家庭成员开始，但很快将每周练习扩展到她的专业网络成员。她感谢她为撰写文章采访过的专家，一位她钦佩其工作的记者，以及一位影响了她工作流程的前主管。她说，虽然她不指望笔记能带来任何回报，但摩纳哥经常发现笔记对她的工作关系有影响，甚至是微妙的影响。举个例子，给一个她喜欢一起工作的编辑写一封短信。

> 当一个和我非常喜欢的编辑一起工作的项目结束时，我给她发了一封正式的“感谢”邮件。她写了回信，鼓励我再投一次，填补她编辑日程表上一个意想不到的空缺。发票归档后很久，我的便条提醒了她我是谁，感觉我们的关系已经从交易关系演变成更像真正的同事。

我喜欢摩纳哥的努力，因为它迫使你花时间思考你未来清单上的待办事项。也许你已经完成了一个大项目，有了一个专业的突破，或者从过去回忆起一个最近适用的教训。花点时间感谢那些帮助你进步的人——不管是直接的还是间接的——可以帮助你看到你已经走了多远，并且可以促使你 [承认你的成就](https://twocents.lifehacker.com/keep-track-of-your-work-accomplishments-throughout-the-1823436243) 。

一封真诚的感谢信不需要花太长时间，否则它会成为你日程表上的另一个负担。如果你不喜欢手写的信件，也不需要用蜗牛邮件。根据你们的关系，你可以发一封简短的电子邮件，甚至发一条短信。没有必要滔滔不绝——你所要做的就是告诉收件人你为什么欣赏他们或他们的工作。
