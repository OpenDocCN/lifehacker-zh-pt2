# 使用这些拉丁短语听起来更聪明

> 原文：<https://lifehacker.com/use-these-latin-phrases-to-sound-smarter-1838745733>

韦氏词典希望我们在日常生活中更多地使用拉丁语，我们认为没有理由不这样做。(好吧，M-W 并没有确切地说*说*，但是他们已经公布了一份他们最喜欢的拉丁短语的列表，所以这是一个公平的结论。)



其中一些你可能已经知道了。有些你可能听过，但不太记得意思。有些是全新的，但值得立即实施。

### 临时的，因此临时的

**定义**:在此之后，因此因为此。它是“声称一件事引起另一件事只是因为它先发生的逻辑错误。”换句话说，相关性不等于因果关系。

韦氏词典说 我们要感谢亚里斯多德开始对这些不合逻辑的论点进行分类。我走到外面，天开始下雨了。我在外面散步真的*导致*下雨了吗？可能会有这种感觉，但真正相信这是一个逻辑谬误。

(如果你仍然不相信你需要把这个短语加入到旋转中，大声跟我说出来: *Post hoc，ergo propter hoc。*那可是好东西。)

### 为了奥古斯塔的苦恼

**定义** : [通过困难走向荣誉](https://www.merriam-webster.com/words-at-play/top-10-latin-words-to-live-by/per-angusta-ad-augusta) ，通过考验走向胜利，或者通过困难走向伟大的事情。这是“不劳无获”的另一种说法

这让我想起了我丈夫和我在寄养家庭中常说的一句话:“我们可以做艰难的事情。”Per angusta ad augusta 会成为一句很好的口头禅或家庭格言。

### **韦尼，维迪，维奇**

**定义** : [我来了，我看见了，我征服了](https://www.merriam-webster.com/words-at-play/top-10-latin-words-to-live-by/veni-vidi-vici)T5。

我们今天可能会在玩笑中使用这个短语，也许是在一次午餐自助餐的特别史诗般的表演之后。但是，如果它像人们普遍认为的那样，是由朱利叶斯·恺撒首先写的，那么它的意思就相当于字面上的意思。他在公元前 47 年左右取得快速胜利后，在一份战争报告中使用了这句话。

### 酒醉吐真言

**定义**:[酒里有真相](https://www.merriam-webster.com/words-at-play/top-10-latin-words-to-live-by/in-vino-veritas) ”或“酒里有真相”这种观点认为，喝多了的人会降低自制力，更倾向于说出自己的真实想法。

这个短语有时会延伸到“在葡萄酒中，在水中”或者，“酒中有真理，水中有健康/良好的感觉。”下次你或你爱的人喝多了，开始倾诉你内心深处的想法时，用这句话作为道歉或解释。

### 货物出门概不退换

**定义**:让买家当心。

韦氏词典说 这个词来自早期的罗马法，当时买家被敦促在购买前真正检查商品，以免被敲竹杠。我们非常喜欢这个，所以我们在英语中采用了“警告”这个词，意思是“禁止某人做某些事情或行为的警告”

亚马逊想到这个短语， [出于某种原因](https://lifehacker.com/how-to-ensure-your-amazon-purchase-isnt-dangerous-1838370086) 。

### 白愁飞的非

**定义**:爱情征服一切。

我们把这个短语——这是你在英语中经常听到的另一个短语——归功于罗马诗人维吉尔，他写道:“爱征服一切；让我们也臣服于爱情吧，“就在第一个千年开始之前， [据韦氏词典](https://www.merriam-webster.com/words-at-play/top-10-latin-words-to-live-by/amor-vincit-omnia) 。从那以后，这个短语被许多诗人使用，并出现在 13 世纪晚期的《T4 坎特伯雷故事集》中。
