# 这两个问题将帮助你决定是否要做某事

> 原文：<https://lifehacker.com/these-two-questions-will-help-you-decide-if-you-want-to-1830411844>

对于我们的大部分雄心壮志来说，在“想要”和“做”之间是有差距的。在你开始尝试关闭它之前，问自己这两个问题。他们可能会节省你一些时间。

根据教练彼得·布雷格曼的说法，如果你不能对两者都说是，你正在考虑的事业可能不适合你。在《哈佛商业评论》的一篇文章 中，布雷格曼分享了一件趣事，他在一次田径比赛后和他 10 岁的儿子聊天，他在比赛中“名列前茅”他的儿子对他的时间感到失望，所以 Bregman 给他这个思考练习:

> “我有两个问题要问你，”我说。“一:你想做得更好吗？”
> 
> 如果答案是“不”，那么尝试指导将是徒劳的(这是我过去犯过的一个错误)。
> 
> “是啊”他说。

> “这是我的第二个问题:你愿意感受付出更多努力和尝试新事物的不适吗？这种不适会让你感到怪异和不同，而且不会马上奏效。”

布雷格曼说，如果你不能对这两个问题都回答是，你就不能做出改变。无论你的目标是什么——跑得更快，找到一份新工作，交朋友——你都必须想要超越，并且愿意以不同于你现有的方式做事，同时体验其中隐含的不适。最终，他的儿子不确定，并决定等到下一季再做决定，对于一个十岁的孩子来说，这是一个相当成熟的反应。但是对于我们其他人来说，这是决定我们是否准备好成长的好方法。

学习新的东西意味着在身体上、智力上和情感上锻炼你已经萎缩的肌肉。当你发现这一点时，可能会导致沮丧或尴尬。冒险可能会令人害怕，尤其是如果你习惯于在目前的水平或职位上取得成功。Bregman 说首先要面对做新的事情意味着什么，问问你自己是否真的愿意去做。当然，说“是”并不能保证你会成功，但是在艰难的时刻，你可以记得你相信这是值得的。那就推过去。
