# 愚蠢的运动动作，排名

> 原文：<https://jezebel.com/stupid-exercise-moves-ranked-1840835700>

欢迎来到一年中的这个时候:当一群自称身材不佳的人发誓要变得更苗条或只是身材更好(或者更简单地说，*不那么糟糕的身材*)时，这意味着全国各地的健身房都充斥着新会员和打着健康和福祉旗号包装的社会结构体的虚假承诺。身材好是可以的！感觉你能在好日子里和鲁斯·巴德·金斯伯格摔跤并且赢了是很好的。(我想我可以。一次好的锻炼应该提供运行你的身体机器所需的适当剂量的内啡肽，你的身体是一个由韧带、肌肉、骨骼和血液组成的迷宫。但老实说，有些锻炼动作很蠢。有些确实如此，人们也同意。



当我播放一段在家锻炼的视频时，我不可避免地会想，“带着它离开这里。”或者“这可行吗？”我和下一个体力不支的人一样喜欢挑战——事实上，在最初厌恶 burpees 和登山者之后，我已经开始喜欢他们了，就像 barre class 一样。但是拜托，让我们理智一点。

大多数最愚蠢的运动动作都包括跳跃。而且我说的是那些不包括设备的。我已经成功地完成了抱膝跳——一种需要巨大冲力的高膝跳，而我很少有这种冲力——也许一生中有两次。驴踢也是一样，就是当你四肢着地——是的，驴踢的方式——试着抬起你的后腿而不摔倒在你的头上。再一次，像驴子一样。视频里的教官会说:“驴踢 30 秒！”我会说，“不”，而是做一些变化，比如原地跳跃，这是一个很好的锻炼。(这就是为什么跳千斤顶是最好的。)

这里有一个最愚蠢的锻炼动作的排名，其标准完全由我决定，但部分原因是:我不喜欢做它们。在你说“最愚蠢的运动是不运动”之前，我已经想到了。这些是按照从最愚蠢到最不愚蠢的顺序排列的。

* * *

1.两条腿的驴踢腿

2.抱膝跳跃

3.方块跳跃

4.土耳其服装/任何服装动作

5.横杠移动时，你将一条腿向后抬高

6.V-ups

7.拍肩板(教练告诉你做的时候不要旋转臀部，但是教练几乎总是在旋转。)

8.手枪深蹲
