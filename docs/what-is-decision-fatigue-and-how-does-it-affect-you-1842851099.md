# 什么是“决策疲劳”，它对你有什么影响？

> 原文：<https://lifehacker.com/what-is-decision-fatigue-and-how-does-it-affect-you-1842851099>

就在几个月前，我们会和朋友出去吃早午餐，在商店短暂停留买一加仑牛奶，看望我们的祖父母，所有这些都不会有第二个想法。

然后冠状病毒出现了。然后是关于社区传播的报道，以及缺乏可用的测试。接下来，人们开始担心是否会有足够的医院资源给生病的病人 ，以及我们一些最脆弱的朋友和家人面临的风险。接下来是 [确诊病例数和死亡人数不断上升](https://www.nytimes.com/interactive/2020/us/coronavirus-us-cases.html) 。

伴随所有这些担忧而来的是问题。点外卖 安全吗？我需要给我的杂货 [消毒](https://vitals.lifehacker.com/you-dont-need-to-sanitize-your-groceries-1842528397) 吗？ [病毒在表面](https://vitals.lifehacker.com/how-long-does-the-coronavirus-live-on-different-surface-1842316128) 持续多久？我该不该 [戴口罩](https://vitals.lifehacker.com/what-to-know-if-youre-wearing-a-cloth-mask-in-public-1842706701) ？去拜访我的父母安全吗？这种咳嗽是因为过敏还是我得了 [新冠肺炎](https://vitals.lifehacker.com/do-you-have-covid-19-or-something-else-1842382063) 病？

卡耐基·梅隆大学工程和公共政策教授巴鲁克·菲施霍夫 说:“这对我们来说更难，因为我们必须去寻找证据。”他研究人们为什么会做出这样的决定。“你担心你错过了什么。”

生活在疫情意味着做出大大小小的各种艰难决定。无论是决定去杂货店，点外卖还是去看望生病的亲人，我们以前会不假思索地做的普通行为已经成为高风险的决定，通常没有明确的正确或错误答案。

### 太多的决定导致情绪和精神紧张

“这些都是合理的困难决定，”菲施霍夫说，并补充说人们不应该为与它们斗争而感到难过。“感觉不好是雪上加霜，”他说。

这增加了我们决策的复杂性，导致了决策疲劳，这是当我们被迫做出太多选择时出现的情绪和精神紧张。决策疲劳是当我们感到压力或疲劳时，思考一个决定变得更加困难的原因。

“这些都是困难的决定，因为风险通常很高，而我们需要掌握不熟悉的信息，”菲施霍夫说。

但是如果这些听起来太多，我们可以采取一些行动来减少决策疲劳。首先，最好尽量减少你一天中做的小决定，比如晚饭吃什么或穿什么。你必须做的小决定越少，你就有越多的带宽来做大决定。

对于这个特殊的危机，你可以采取更多的步骤来减少你的决策疲劳。

### 寻找可靠的新闻来源

外面有很多错误的信息，还有很多阴谋论，所有这些都令人疲惫和困惑。为了抵消这种额外的压力，菲施霍夫建议挑选一些可以信任的专家。

“媒体绝对至关重要，”他说。他的建议是找到那些拥有致力于获取正确事实的记者和编辑的媒体。他还建议避免社交媒体上流传的更疯狂的阴谋论。

“你的本能是试图理解它们，即使你认为这很荒谬，”他说，并补充说，当你研究完阴谋论时，“你知道的比开始时少。”

### 对自己宽容点

后知后觉是 20/20。回顾和猜测决策是很容易的，比如没有比你做的更早行动。这就是所谓的后见之明偏差，这也是菲施霍夫建议我们应该努力避免的，因为我们都在尽我们所能做出最好的决定，考虑到有限的信息和这场危机不断变化的性质。

“不要事后批评你所做的决定，”菲施霍夫说。"尽你所能，对自己宽容些。"
