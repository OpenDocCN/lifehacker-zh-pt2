# 电话里不是社会保障局

> 原文:[https://life hacker . com/thas-not-the-social-security-administration-on-the-pho-1830111974](https://lifehacker.com/thats-not-the-social-security-administration-on-the-pho-1830111974)

根据联邦贸易委员会的说法，城里有一种新的骗局需要警惕:要求个人信息的电话据称来自社会保障局。但不是 SSA，是骗子。

Watch

下面是它的工作原理，根据 [消费者报告](https://www.consumerreports.org/scams-fraud/fake-calls-from-social-security/) :

> 来电显示显示 SSA 的客户服务号码 800-772-1213，尽管它们实际上可能来自任何地方。这种使用合法电话号码来欺骗消费者的行为被称为“欺骗”

骗子然后说他们来自 SSA，并要求您的社会安全号码作为您的档案，或者说他或她需要“更多信息以增加您的福利金，或者如果您不确认他或她所拥有的信息，他或她将威胁终止您的福利。”

[很像国税局的](https://twocents.lifehacker.com/the-irs-isnt-calling-you-1822798693) ，SSA 是不会给你打电话的。不要上当！

如果你接到电话，挂断并拨打 800-772-1213。这是 SSA 的真实号码，但诈骗者可以重新路由它，让它看起来像他们在该组织工作。然后你可以问他们是否真的需要你的信息。

和大多数骗局一样，老年人和不会说英语的人面临的风险最大，所以要保持警惕。你也应该在 [SSA 的网站](https://oig.ssa.gov/report) 上举报这个电话，或者向 FTC 举报。
