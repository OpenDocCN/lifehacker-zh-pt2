# 因此...学校怎么样？

> 原文:[https://lifehacker.com/so-hows-school-going-1845094739](https://lifehacker.com/so-hows-school-going-1845094739)

昨天我带我儿子去看医生。这是和一个新医生的预约，一个我们从未见过的专家，护士停顿了一下，直视我的眼睛。“所以，”她说。“学校过得怎么样？”

Watch

这不是医生办公室里常见的闲聊，“哦，他几年级？”或者“你最喜欢的科目是什么，伙计？”这是一个真实的问题，是一个家长问另一个家长的——如果要我猜的话。学校过得怎么样？你是怎么做到的？我们这些 T2 人是如何做到这一点的呢？

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1306222332847906817&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1306222332847906817&autosize=1) 

昨天，我认识的一位母亲告诉我，她曾短暂地考虑过对她二年级的孩子说*见鬼去吧*和 [非学校教育](https://lifehacker.com/should-you-give-unschooling-a-try-1843184632) 。她上五年级的大儿子表现不错，但她的小儿子呢？是啊，没那么多。

“他在每一个层面上都在挣扎，远程学校对他来说完全不合适，”她告诉我。但是，当面对面的学校不是一种选择(因为疫情)，但在家上学也不是一种选择(因为工作)，除了看着他日复一日地奋斗，或者完全放弃正规教育，还剩下什么？

“[在]春天，我告诉两个男孩只是为了得到他们的出勤率，我不在乎之后发生了什么，”这位母亲说。“但是‘去他妈的’方法是不可持续的，如果我们不能让远程学校工作，我不知道还有什么其他方法。”

我花了几个月的时间试图找到走出这个特别令人烦恼的精神密室的方法，但是如果有钥匙的话，我还没有找到——为我自己或其他任何人。*如果这都不行，还有什么？*

就在我写这篇文章的时候，我儿子的四年级老师正在给我们传达另一个新策略:一份纸质作业清单，帮助学生记录每天需要做的事情。我儿子的作息时间是混合的，一周三天在家，两天在学校(和他一半同学的作息时间相反)。这学期我们已经进行了三周，而我们*仍然*试图弄清楚一些基本的事情，比如“事情什么时候到期？”

昨天我没有一个好的答案给那个护士——或者说至少我不能在我的孩子面前给出一个好的答案——但是被问到还是很好的。很高兴被提醒这是不正常的，即使感觉生活好像就在我们周围，好像我们现在应该已经适应了。

所以，在评论里告诉我:学校过得怎么样？远程学校，或者混合学校，或者家庭学校，甚至 [非学校教育](https://lifehacker.com/should-you-give-unschooling-a-try-1843184632) 进展如何？

什么有效，什么无效？

你希望你能做些什么不同的事情？

如果这个不行，还有什么？
