# 在 Lyft 应用程序中为你从不记得的地址设置“自定义快捷方式”

在过去的几个月里，我不得不去镇上几个不同的医生那里预约。我住在旧金山，没有车，所以去那些约会总是意味着付钱给他们。毫无疑问，在他们每一个人之前，我都准备出门，然后意识到我实际上不知道我要去哪里。

是的，我以前去过那里，但在大多数情况下，是在几周或几个月前。

最终发生的是，我必须回到里面，打开我的笔记本电脑，登录到我的常规医生的患者门户网站，找到她原来的转诊信息和新的地址。是的，我可以把地址放在约会的日历条目中。是的，我可以在我的手机上有这个信息的文档。事实是我不知道，也不太可能知道。

最近，我发现了 Lyft 的一个功能，它解决了我将要做的事情:自定义快捷方式。

当你使用拼车服务去某个地方，如餐馆或商店时，输入目的地的名称就可以到达那里。当你要去别的地方，比如我的医生在一个巨大的医疗大楼里的办公室，输入他的名字不会让你去任何地方。

通过快捷方式，我可以添加那个医生的地址，然后贴上“正畸医生”或“治疗师”的标签。一旦你这么做了，当你需要它的时候，它就会永远存在于你的应用程序中。

当涉及到医生时，这个特殊的功能对我来说很棒，但它也可以方便地用于像保姆的地址或你一年去两次的朋友家这样的事情。

要添加快捷方式，请启动 Lyft 应用程序，然后点击页面左上角的汉堡菜单。

在那里，点击出现的菜单底部的“设置”选项，然后在下一页点击“管理所有快捷方式”。该页面有一个“添加自定义快捷方式”按钮轻按该按钮，然后输入您不记得的地址。一旦你提交了它，Lyft 会提示你为它键入一个名字。

在这个假实例中，我为“保姆凯伦”创建了一个条目完成后，点击角落里的勾号，你就可以开始了。

现在，每当你启动 Lyft 时，你可以在应用程序中输入“保姆凯伦”，直接导航到她的房子，而不需要记住她的地址。

对于像我这样健忘的人来说，这是一个很好的功能,但对于老人和孩子，或者任何其他可能使用这款应用程序的人来说，这真的很方便，他们可能偶尔会去非商业场所，在那里他们可能不知道自己头顶上的地址。
