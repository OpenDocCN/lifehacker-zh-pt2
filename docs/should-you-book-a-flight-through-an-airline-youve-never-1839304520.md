# 你应该通过一家你从未听说过的航空公司预订航班吗？

> 原文：<https://lifehacker.com/should-you-book-a-flight-through-an-airline-youve-never-1839304520>

在你寻找特别划算的航班的过程中，你可能会遇到看起来像虚构的航空公司——例如，贝尔斯金航空公司(Bearskin Airlines)，它在加拿大安大略省运营，非常真实。或者 [座椅航空](https://www.chair.ch/) ，一家飞遍欧洲的瑞士航空公司，它不是你想象中的虚构。甚至还有 [奥凯航空](https://www.okair.net/#/) ，提供飞往中国和泰国的航班，嘿，至少他们对自己的服务质量很诚实。

但是，通过一家你一无所知的航空公司订票值得吗？嗯，那要看情况，尤其是你的预算、舒适的必要性和旅行计划。在你决定通过不熟悉的航空公司订票之前，请记住以下几点:请务必确认您预订的机场，并阅读小字。

### 检查费用和合作航空公司

首先，在你预订之前，一定要检查你的预订中可能不包括的费用——想想托运行李、手提行李或仅仅是预订座位的费用。Allegiant Airlines 是一家在全美运营的廉价航空公司，在机场打印登机牌将收取费用。(这是 [真的](https://www.allegiantair.com/popup/taxes-and-fees) ，价格 5 美元。)

最后，你的机票可能看起来像是一笔交易，但当航空公司收取你最初不知道的费用时，可能就不那么划算了。在这种情况下，最好的做法是在网上搜索该航空公司和“可选服务费用”或更一般的“费用”如果你有任何疑问或者航空公司的网站将你引入歧途，请联系航空公司代表，向他们询问行李或座位预订费用。

之后，在网上搜索这家航空公司，看看它是否与一家大型航空公司有合作关系；如果你在航空公司有积分，你可以用这些积分购买机票。例如，美国航空公司与一些较小的航空公司合作，包括塔希提岛航空公司、阿提哈德航空公司和 InterJet。您可以使用这些航空公司的 AA 里程预订航班。

### 确认机场并检查评论

如果你乘坐的是一家小型的通勤航空公司，那么你计划的路线中的机场可能不是主要的枢纽机场。正如 [美国消费者新闻与商业频道](https://www.cnbc.com/2019/05/17/budget-airlines-what-you-need-to-know-before-booking-your-ticket.html) 所建议的那样，在确认您的预订之前，一定要仔细检查两个机场。你不想因为不知道机场在哪里而在目的地外一个小时到达目的地——更好的办法是，为了保险起见，在地图上标出你从机场出发的酒店。(如果你需要租车，一定要确保机场提供租车服务。至少，确保你能从机场搭车到汽车租赁机构。)

不言而喻，检查航空公司的评论是非常重要的。你可以使用像这样的网站找到几乎所有航空公司的评论。下面是对奥凯航空公司一次航班的评论:“座位有点狭窄，我不得不在过道里伸腿...没有任何形式的屏幕，只有座位口袋里的杂志，对于支线航班来说是可以接受的，但这并不意味着应该有口香糖把页面粘在一起！没有吃的或喝的，没有瓶装水，没有袋装坚果。唯一的空乘全程隐身。”如果几篇这样的评论吓跑了你，也许你应该花点时间考虑一下你的其他选择。

猫途鹰也为航空公司众包评论，帮助你避免一次特别糟糕的飞行。“整架飞机都很脏，但等你看到厕所时就知道了，”MartaKent 在评论 [地中海航空](https://www.tripadvisor.com/Airline_Review-d13150558-Reviews-Air-Mediterranean#REVIEWS) 时写道。"...在某个时候，我甚至看到乘务人员在一个厕所里倒热水试图疏通。(你应该在飞机的厕所里倒热水吗？)" [好问题](https://lifehacker.com/how-to-unclog-a-toilet-when-there-is-no-plunger-1820623969) ！

最后，在你订票之前，确保你知道航空公司的退款政策，以防你的旅行没有按计划进行。一般来说，美国航空公司坚持 24 小时规则，允许你在一天内取消航班而不产生费用。在其他地方(大多数美国航空公司在 24 小时后)，如果你决定改变或取消你的计划，如何或是否退款完全由他们决定。出于这些原因，在订票前一定要看小字，如果你有任何疑问，再次，直接联系航空公司或寻找其他旅客的评论，他们可能已经处理了该航空公司的客户支持噩梦。
