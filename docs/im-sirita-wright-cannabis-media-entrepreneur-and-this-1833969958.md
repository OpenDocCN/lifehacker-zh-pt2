# im sirita wright cannabis media entrepreneur and this 1833969958

西里塔·赖特在执行一项任务。大麻法律正在迅速变化，这意味着企业家正在将注意力转向蓬勃发展的行业。然而，目前大多数大麻企业主都是白人——这是西里塔想要帮助改变的一个事实。通过她的媒体公司 [EstroHaze](https://estrohaze.com/) ，西里塔和她的联合创始人为有色人种妇女提供了一个空间，让她们了解大麻产业提供的众多生活方式和商业机会。正如她在 NBC 采访中所说的，“我不会道歉。大麻拯救生命，创造就业机会，是有色人种获得世代财富的巨大机会。”这是她工作的方式。

#### 首先，告诉我们一些你的背景，以及你是如何走到今天这一步的。

2005 年，我在弗吉尼亚州阿灵顿的玛丽蒙特大学(Marymount University)完成本科学业时搬到了纽约，当时我正在学习时装营销，后来搬到了纽约，在《VIBE》杂志的时装部实习。在完成实习，在时尚界做了各种工作，并活着度过了 2008 年的金融危机后，我转向并开始在社交媒体上从事自由职业。2014 年，我成为了黑色企业的社交媒体经理。在那里，我遇到了我的联合创始人卡莉·怀尔德和萨丰·弗洛伊德，我们开始了一个名为 [EstroHaze](https://estrohaze.com/) 的大麻主题播客。

2016 年底，我们申请了一个名为 Canopy Boulder 的种子阶段大麻加速器。我们是 2017 年春季毕业生， [EstroHaze](https://estrohaze.com/) 是一家成熟的媒体公司和咨询机构，制作以有色女性为特色的大麻商业和生活方式为中心的内容。

#### 带我们看看最近的一个工作日。

上周二，在起床并做了我早上的例行工作(刷牙、洗脸、洗澡/穿衣、遛狗、喝水、喝咖啡)后，我查看了我的电子邮件和当天的日历，并准备与我的联合创始人萨丰会面，参加在纽约举行的一次合作会议。当我们走进会场时，我接到了一个与我合作的选角机构的电话，确认我第二天是否有空做替身演员。会议结束后，我们前往市中心参观了位于翠贝卡的一个令人惊叹的 CBD 商店/合作空间“每日归来”。之后，我们回到布鲁克林，给我们的联合创始人兼首席执行官 Kali 打电话，就我们的会面和下一步计划进行沟通。从那里，我喝了一轮大麻烟和茶，然后回家——从萨芬家步行大约需要六分钟。我遛狗，然后查看可能需要我关注的电子邮件。我查看了我们的社交资料，以确保没有任何我们需要关注的评论，查看了我第二天的日历，做出了任何调整，并开始了放松的例行公事。



#### 除了你的手机，还有什么应用程序、小工具或工具是你生活中不可或缺的？

我不能没有备用的手机充电器、书籍和营销播客，比如尼尔·帕特尔和埃里克·苏的营销学院的、我的 FENTY Beauty lipgloss 和我(合伙人)的 PuffCo Peak vape。

#### 你的工作空间是什么样的？

我办公桌前的那面墙是我的灵感之墙，挂满了时尚照片、实体日历、感人的文字和其他图像，它们激励着我，让我走进办公室时会心一笑。我在那里花了很多时间，所以它需要给我一种很棒的感觉。我用文件夹和剪贴板来把我的项目分开，用文件夹来存放财务和个人事务，还有成堆的彩色书籍——包括一本未经授权的德雷克的书。

#### 围绕大麻的法律变化对你的业务有什么影响？

他们让我们能够围绕大麻宣传以及大麻与健康如何携手并进进行更开放的对话。随着对话变得更加主流，[我们更有能力]帮助打破围绕大麻和使用这种植物的人的污名——特别是已经被污名化的社区。这使我们能够谈论社会正义，并找出修复毒品战争造成的损害的方法。



#### 帮助你完成工作的人是谁，你如何依赖他们？

我的联合创始人都是了不起的女性，她们都是坏蛋，能把事情做好。当我们进入流程并开始执行我们的想法时，我们的业务就会繁荣。我们继续学习、成长，并在必要时进行调整。创业不易，需要身边有强大的团队；我和我的联合创始人以前不仅一起工作过，而且还一起生活过，我们真的需要深入挖掘，为我们的业务奠定基础。

#### 你如何记录你必须做的事情？

我分层工具来帮助我保持正轨。我有每周和每月的 Trello 板，我所有的日历都同步到一个主日历上。我每周制定我的时间表，每天看我的日历。我非常相信时间分块、批量任务和使用生产力工具，如 [Pomodoro chrome 扩展](https://lifehacker.com/timedoser-is-a-pomodoro-timer-for-chrome-1639626091) 。

#### **如何充电或休息？**

我真的很喜欢到大自然中去。也许是因为我是个女童子军或者处女座，但是在公园或者散步有助于我充电。我最喜欢参观的地方之一是布鲁克林植物园。我认为这是一个神奇的地方。那里的每个人都很开心，面带微笑。孩子们在玩耍，每个人都在学习植物，空间周围流动的能量非常特别，感觉不像是在纽约。



#### 你最喜欢的副业是什么？

我现在最喜欢的副业是我的营销播客 [野蛮媒体](https://anchor.fm/siritasavage) 。真的是创业者和创意人的营销常识。我分享的战术和战略可以快速实施，并在帮助您了解和简化整体营销工作的同时，实现快速成功。
