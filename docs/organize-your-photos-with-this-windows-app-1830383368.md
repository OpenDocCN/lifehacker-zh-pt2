# 使用此 Windows 应用整理您的照片

> 原文：<https://lifehacker.com/organize-your-photos-with-this-windows-app-1830383368>

Windows:如果你有一个又一个文件夹的照片，你发誓有一天你会整理它们，我不会感到惊讶对我来说，那天是这个周末。不管出于什么原因，我想花几个小时来处理我一生的摄影历史，我已经把它们放入我的网络存储设备上的一个大的“分类”文件夹中。

我害怕这项任务，因为我发现像 Lightroom 或 Capture One 这样的程序很有压力。虽然它们让按 EXIF 数据(拍摄时间或地点、使用的相机等)组织照片变得非常容易，但弄清楚编目的细微差别，以及如何让虚拟文件夹或过滤器映射到电脑或存储设备上的真实文件夹，让我感到头疼。

这就是为什么我很兴奋地发现了 Windows 应用程序[**PhotoMove**](https://www.mjbpix.com/automatically-move-photos-to-directories-or-folders-based-on-exif-date/)，它成了我凌乱的图片档案的救命稻草。免费版可以彻底快速地分析你保存照片的地方，读取任何和所有 EXIF 数据，并自动创建和移动(或复制)你的照片到文件夹中。该应用程序几乎不需要时间安装，并且很容易理解——比学习一个真正的编目应用程序要快得多，你可以(或者应该？)还是用。

PhotoMove 的免费版本中没有太多需要调整的设置，但它们很重要。首先，你选择你想让应用程序分析的文件夹(包括子文件夹！)，以及照片最终存放的目标文件夹(PhotoMove 创建的新文件夹)。

在这款应用的免费版本中，你只能选择两种文件夹结构，这种粒度——根据拍摄时的年、月、*和*日期将图像分类到文件夹中——可能会令人讨厌。不过，有一个简单的解决办法。一旦你完成了这个过程，PhotoMove 就会根据照片拍摄的日期创建大量的文件夹，你只需在文件浏览器中找到它们的父文件夹(月份)，进行通配符搜索(" *。*)，并将 Windows 找到的所有照片移动到月份(或年份)文件夹，这取决于您的个人偏好。

可能说服我购买 PhotoMove 专业版(仅售 9 美元)的一个功能是它的副本查找器。换句话说，如果 PhotoMove 检测到两张图像具有相同的名称或匹配的 EXIF 数据(这是你的偏好)，该应用的付费版本可以自动重命名副本，让副本覆盖原始副本，或将副本移动到一个特殊的“你会想要手动查看这个”文件夹。如果你不擅长整理照片，这可以帮你节省很多时间。

最后，当 PhotoMove 在照片中检测不到任何 EXIF 数据时，它也可以发挥类似的作用。如果是这样的话，它可以根据文件本身创建的时间将照片移动到有组织的文件夹中——这不是最糟糕的主意——或者将所有这些照片移动到一个特殊的“未知”文件夹中。你需要这款应用的专业版来实现这些功能，但是 9 美元对于这些便利来说并不算多。

至于我的组织会议，我很高兴 PhotoMove 能够毫无问题地处理我庞大的照片库的大约 90%。它无法处理的照片——我有预感这些照片会被处理——没用多长时间我就可以手动移动到正确的文件夹中。至少，我花的时间比从零开始组织所有 25，000 张图片要少得多。
