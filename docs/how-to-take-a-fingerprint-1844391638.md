# 如何采集指纹

> 原文：<https://lifehacker.com/how-to-take-a-fingerprint-1844391638>

国际旅行 需要大量的文件和身份证明，这通常包括获取的指纹。我是一个经验丰富的旅行者，我知道我不是唯一一个不得不不止一次采集指纹的人，包括电子指纹和纸张上的墨水指纹。

这个过程可能会很耗时，但实际上有一种正确的方法可以让你的指纹第一次就正确无误，而不会有任何也就是说，不需要做三次或四次直到指纹技术人员可以得到一个干净的印象。以下是如何在第一次尝试时采集正确的指纹，然后上路。



许多【坏】指纹都是皮肤太油或者太干燥的结果。 [建议](https://www.atlanticscreening.com/livescan/fingerprintrejections/#:~:text=1.),after%20you%20wash%20your%20hands.) 在提取指纹之前，不要洗手或使用洗手液。然而，由于我们生活在一个全球化的疫情，我强烈建议你两者都做—但是当你之后保湿的时候，使用无油乳液。

当你将手指放在将记录指纹的表面上时，无论是电子的还是纸张，重要的是不要将手指压在中间。从外角开始， 在纸张或屏幕上滚动手指，同时用力向下按压，直到手指舒适地居中。

如果你的指纹被电子照相，不要开始移动你的手指，直到你从设备或操作它的人那里得到许可。如果你在一个印台上，操作它的人可能会 给你精确的指示如何放置你的手指；在疫情之前，他们甚至可能已经用他们的手物理引导你的手指。
