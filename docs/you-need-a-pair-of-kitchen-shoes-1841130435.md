# 你需要一双厨房鞋

> 原文：<https://lifehacker.com/you-need-a-pair-of-kitchen-shoes-1841130435>

我四肢的血液循环非常糟糕，这意味着我从十月到五月一天 24 小时都穿着厚袜子和拖鞋。今年，我得到了极其舒适的圣诞拖鞋——然后在为家人做晚饭时，立刻把各种屁话泼在拖鞋上。悲剧。

幸运的是，的污渍似乎都不是永久的，但我回家后做的第一件事就是翻出几双旧球鞋，粘在火炉旁。不管你有没有自己的花式拖鞋，如果你经常做饭，我强烈建议你也这样做。你的厨房可能比你家里的其他房间更危险，所以留一双鞋子只是为了做饭是很有意义的。它们不仅能让你的脚趾不被溅出和溅出，让你的袜子远离洗碗水的水坑，还能提供足够的保护，防止物体坠落，防止严重的悲剧发生。它们就像你脚上的围裙。

不需要出去买花里胡哨的厨师木屐；任何一双能承受打击的闭趾鞋都可以。我的是一双破旧的低顶货车，鞋跟被塞得很低，这样我就可以快速进出。他们被打得很惨，我可以把任何东西溅到他们身上，也不在乎——当我做完饭后，我会把他们踢走，放在厨房里他们应该在的地方。
