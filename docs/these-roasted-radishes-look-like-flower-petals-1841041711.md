# 这些烤萝卜看起来像花瓣

> 原文：<https://lifehacker.com/these-roasted-radishes-look-like-flower-petals-1841041711>

每个人都曾在某个时候被诱惑去吃百花香(被禁止的薯片)。干花瓣和香料的混合物看起来总是很脆，如果有点灰尘的话。你不应该吃百花香，但你应该吃这些烤萝卜片，这肯定会挠痒痒。

Watch

任何烤过的萝卜片都会卷曲起来，看起来像花瓣，但是西瓜萝卜片卷曲起来看起来最漂亮。你烤的时间越长，它们就会变得越黑，但我理想中的萝卜花瓣是边缘呈褐色，中心是充满活力的粉红色。你切得越均匀，它们就越容易变成褐色。

烤萝卜花瓣在吐司(鳄梨或其他)上看起来很棒，但你也可以把它们放在蔬菜馅饼、砂锅菜上，或作为一大块烤肉的配菜。它们尝起来有点像愤怒的薯片——又脆又咸，但是有点辣，淀粉含量少一点。要制作它们，您需要:

*   你想要多少萝卜就有多少
*   橄榄油
*   盐

将烤箱加热到 375 华氏度，清洗并修剪萝卜。用一把锋利的刀或曼陀林切片机，尽可能薄地(均匀地)切萝卜。将切片放入碗中，淋上足够的橄榄油。将切片平铺在铺有羊皮纸的烤盘上，在每片上撒上少许盐。烤至边缘卷起，变成深金黄色。从烤箱中取出，用纸巾擦干，然后食用你的蔬菜百花香。
