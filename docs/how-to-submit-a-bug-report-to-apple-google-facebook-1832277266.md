# 如何向苹果、谷歌、脸书、推特、微软等提交错误报告

> 原文：<https://lifehacker.com/how-to-submit-a-bug-report-to-apple-google-facebook-1832277266>

安全漏洞、硬件漏洞、系统崩溃、功能受损——每一个硬件或软件都容易出现错误和漏洞，很可能你在科技生涯中的某个时候不幸遇到过这些问题。当大多数人苦笑、忍受并等待问题自行解决时，您也可以通过报告您的发现来采取更积极的方法来处理 bug 和其他安全灾难。



问题？当您遇到问题时，您可能不知道如何或在哪里提交错误报告。为了使这个过程更容易，我们已经查看了最常用的应用程序、服务和硬件制造商，并将他们的错误报告工具合并到一个大列表中。

### 关于错误报告的一些提示

虽然我们的列表解释了如何提交常用应用和服务的错误报告，但它并不详尽。如果您没有找到您想要的，这里有一些快速错误报告提示和最佳实践:

*   一些应用和程序将允许你直接发送崩溃/错误报告。如果你正经历频繁的崩溃，并且这个选项是可用的，利用它。通常情况下，这些自动报告将包含您必须手动包含的信息，从而使该过程变得更加容易。
*   如果可能的话，记下任何弹出框或错误代码(或截图)。详细说明错误、错误或崩溃是如何发生的，并确保在适用的地方包括您的硬件/软件规格。这些细节将有助于包含在您的错误报告中(在某些情况下可能是必需的)。
*   如果你在论坛或留言板上提交一个 bug，一定要阅读任何张贴指南，这通常要求你进行初步搜索，看看你的特定 bug 是否已经被报告。虽然重复报告有助于更快地修复 bug，但一些 bug 报告论坛对如何提交相同 bug 或错误的报告有严格的要求。
*   如果你正在寻找一家公司的 bug bounty 计划或者如何提交一个安全相关的漏洞，这些链接通常可以在 [Bugcrowd](https://www.bugcrowd.com/bug-bounty-list/) 或者 [Hackerone](https://hackerone.com/) 上找到。请记住，这些程序更适合于高层次的问题和主要错误，而不是您的平均性能问题，因此有严格的提交指南。

### 如何(以及在哪里)提交错误报告

#### **Adobe (Creative Cloud，Photoshop，Illustrator，Premiere Pro，Acrobat Reader 等。)**

1.  访问 [Adobe 的 bug 提交和特性请求表。](https://www.adobe.com/products/wishform.html)T3】
2.  从下拉菜单中选择产品，并同意 Adobe 的条款和条件。
3.  然后，您将被带到该特定产品的错误/功能请求页面。完成表格并提交。

安全漏洞可以直接上报给

#### **AMD**

1.  如果你遇到硬件问题或显卡相关的崩溃，使用 AMD 的专用 [错误报告调查页面](https://www.amdsurveys.com/se/5A1E27D23A3DE966) 。
2.  选择您的产品。
3.  按照说明提交错误或错误报告。确保包含任何崩溃或错误文本。

AMD 坚持认为公司不会直接回复错误报告，但是更频繁报告的问题有更高的几率在未来的更新中修复。

#### **苹果**

*   你可以通过 [同一个报告页面](https://developer.apple.com/bug-reporting/) 发送所有苹果产品的 bug、错误或漏洞报告。
*   如果您需要提交产品的总体反馈，请使用此页面的。
*   对于与帐户相关的问题，请使用苹果开发者网站 上的 [联系表格。](https://developer.apple.com/contact/)

#### **体式**

*   您可以在此页面 提交与 Asana 的 app 或网页性能 [相关的 bug 或错误。](https://asana.com/support)
*   如果发现潜在的安全问题，提交给 [Asana 的赏金程序](https://asana.com/bounty) 。

#### **Dropbox**

1.  访问 [此页面](http://dropbox.zendesk.com/) ，为 Dropbox 的支持系统创建一个新账户。
2.  填写表格以提交帮助单。

也可以联系 Dropbox 的 [客服](https://www.dropbox.com/support) ，或者搜索 [Dropbox 服务台](https://www.dropbox.com/help) 。有关帐户和功能的问题或顾虑。

#### **DuckDuckGo**

1.  转到 DuckDuckGo 的用户反馈 [提交表单](https://duckduckgo.com/feedback) 。
2.  选择您的问题的性质。对于错误和漏洞，选择“我需要报告一个安全问题”
3.  按照说明并填写所需的表格来提交您的报告。

#### **Evernote**

1.  访问 [Evernote 的支持页面](https://www.evernote.com/SupportLogin.action) ，使用 Evernote 账户凭证登录。
2.  提交您的支持票。

也可以将安全漏洞直接发送到[【security@evernote.com】](mailto:security@evernote.com)。

#### **脸书**

1.  参观脸书的 [求助中心](https://www.facebook.com/help/) 。
2.  点击“政策和报告”下拉菜单下的“报告脸书问题”。
3.  按照说明，填写所有必需的信息，提交您的问题。

你也可以将安全漏洞发送给脸书的漏洞奖励计划，如果你想要你(合格的)努力的一点回报。

#### **谷歌、安卓、Chrome 和其他谷歌服务**

Google 有很多产品，其中一些有专门的 bug 提交页面，而其他的使用通用的提交表单。

**Chrome:**

1.  在台式机或笔记本电脑上打开 Chrome。
2.  点按浏览器右上角的三个堆叠的点图标。
3.  转到帮助>报告问题。
4.  写你的错误报告，确保包含尽可能多的信息。
5.  点击“发送”

**安卓操作系统漏洞和其他谷歌应用/服务:**

1.  访问谷歌关于 [报告安全漏洞](https://www.google.com/appserve/security-bugs/m2/new) 的页面。
2.  选择最适合您问题的选项，并按照说明提交您的报告。

谷歌也有一个 bug 赏金计划，你可以在这里 了解更多 [。(您还可以使用“报告安全漏洞”工具来发送这些信息。)](https://www.google.com/about/appsecurity/reward-program/index.html)

#### **Instagram**

您可以通过以下方式报告您的 Instagram 问题:

1.  在 Instagram 应用程序中，转到您的个人资料。
2.  轻按右上角的三条堆叠线图标。
3.  点击齿轮形状的设置图标。
4.  向下滚动并点击选项屏幕“支持”部分的“报告问题”。
5.  按照说明并填写所有必需的信息。

如果你想报告一个与安全相关的问题，你可以将这些提交给脸书的赏金计划。T3】

#### **升**级

使用此页面的 [提交 Lyft 应用程序和网站的问题和错误](https://help.lyft.com/hc/en-us/articles/115013082448-Issues-with-the-app-or-website) 。Lyft 也有 bug 赏金程序，但是 T5T7】是邀请才有的。

#### **微软**

**Windows 10**

1.  从“开始”菜单打开 Windows 10 反馈中心，或者在桌面任务栏的搜索栏中键入“反馈中心”。
2.  单击“报告错误”,并按照说明发送您的问题。

**Xbox**

如果你是一名 [Xbox Insider](https://support.xbox.com/xbox-one/console/xbox-insider-hub) ，你可以通过按住控制器上的 Xbox 按钮并使用“报告问题”选项来报告错误。

**安全漏洞**

*   如果你发现任何微软产品的安全问题， [提交给微软的赏金计划](https://www.microsoft.com/en-us/msrc/faqs-bounty) 。该计划涵盖了微软的所有产品和服务。
*   您也可以将微软产品和服务的安全漏洞发送到此电子邮件地址:[【secure@microsoft.com】](mailto:secure@microsoft.com)

#### **Mozilla**

*   你可以从任何浏览器提交一个 Firefox 相关的 bug 到 [Mozilla 的 bug 报告页面。](https://bugzilla.mozilla.org/)T3】
*   基于安全的顾虑/漏洞应该报告给 Mozilla 的 bug 赏金计划或 security@mozilla.org 的。

#### **网飞**

网飞让用户 [通过一个简单的在线提交表格报告播放 bug 或性能问题](https://help.netflix.com/en/node/104515) ，而其 [bug 赏金程序](https://bugcrowd.com/user/sign_in) 处理安全漏洞。

#### **任天堂**

*   通过任天堂官方 [支持联系页面](https://en-americas-support.nintendo.com/app/contact) 提交软件 bug、错误、硬件问题以及其他问题。
*   另一方面，与任天堂网站相关的错误或问题应通过此 [网络表单提交。](https://www.nintendo.com/corp/webform.jsp)T3】

#### **英伟达**

*   你可以在 Nvidia 的支持论坛上发一个帖子，提醒 Nvidia 关于他们的硬件、性能或游戏兼容性的错误或错误。遵循本论坛帖子 中概述的 [步骤和规则，确保通读要求。](https://devtalk.nvidia.com/default/topic/522835/linux/if-you-have-a-problem-please-read-this-first)
*   对于一般支持和故障排除，使用 [Nvidia 支持页面](https://www.nvidia.com/object/support.html) 。
*   最后，你可以通过 [这个网站](https://www.nvidia.com/en-us/security/) 或者邮箱:【AppSecSupport@nvidia.com】[发送安全漏洞给英伟达](mailto:AppSecSupport@nvidia.com)

#### **Paypal 和 Venmo**

将所有的 bug 和漏洞报告提交给 [Paypal 的 bug bounty](https://www.paypal.com/bugbounty/) 程序。

#### **Reddit**

*   在 Reddit 的专用 [r/bugs subreddit](https://www.reddit.com/r/bugs/) 中发布 bug。

*   将安全漏洞发送到，也可以使用 [bug bounty subreddit。](https://www.reddit.com/r/bugbounty/)

#### **松弛**

*   Slack 众多 app 中的一个 app 上与技术性能相关的 bug 可以发送到[【feedback@slack.com】](mailto:feedback@slack.com)或者 [这种联系方式。](https://slack.com/help/contact)
*   向 [Slack 的赏金计划举报安全漏洞。](https://slack.com/report-vulnerability)T3】

#### **Snapchat**

通过 Snapchat 应用程序报告 bug 实际上有点新奇:

1.  [摇动你的设备](https://support.snapchat.com/en-US/a/shake-to-report) 调出“Bug 和建议”提交菜单。
2.  选择相关选项，并制作您的报告。

你也可以用 [这个网页表单](https://support.snapchat.com/en-US/i-need-help) ，或者通过 [Snapchat 在 Hackerone](https://hackerone.com/snapchat) 上的 bug bounty 程序，从你的网页浏览器发送 bug 或者漏洞报告。

#### **索尼**

**一般索尼产品:**

*   通过 [索尼的赏金计划](https://secure.sony.net/) 公开所有 bug 和安全漏洞。

游戏机:

*   与 PlayStation 服务和硬件相关的技术错误可以通过几种方式提交给 PlayStation 的支持团队，包括在线、在 Twitter 上、通过电子邮件、聊天、通过电话等。 [查看本页](https://support.playstation.com/s/contact-support?language=en_US) 找到与你最相关的方法。

#### **Spotify**

*   将应用或网络性能问题发送给 [Spotify 的支持团队](https://support.spotify.com/us/article/how-can-i-contact-spotify/?in_footer) 。

*   向 [Spotify 的赏金程序](https://www.spotify.com/us/bounty/) 报告安全问题。

#### 特雷罗

在这里 了解是什么造就了一份优秀的 Trello bug 报告 [，然后通过](https://help.trello.com/article/709-reporting-trello-bugs) [Trello 的支持页面](https://trello.com/contact) 提交您的问题。

#### **Tumblr**

*   通过 Tumblr 的支持页面 发送错误报告。
*   如果你发现了一个安全问题，提交给 Tumblr 的 [漏洞奖励程序](https://tumblr.zendesk.com/hc/en-us/articles/234583348-Bug-Bounty-Program) 。

#### **抽动**

*   将技术 bug 和性能问题发送给 [Twitch 支持。](https://help.twitch.tv/customer/portal/emails/new)T3】
*   通过 [将安全漏洞和赏金提交给 Twitch](https://www.twitch.tv/p/security)。

#### **推特**

*   Twitter 的帮助页面有一个专门的 [错误报告提交表](https://help.twitter.com/forms) ，用于处理网页版和应用程序相关的错误。你也可以把你遇到的问题发微博到@TwitterSupport。
*   Twitter 有一个 [黑客一号赏金程序](https://hackerone.com/twitter) ，用于发现、报告和修复安全漏洞。

#### **优步**

所有的安全漏洞都可以提交给 [优步的黑客一号赏金计划](https://hackerone.com/uber) 。

#### **阀门(蒸汽市场)**

*   [遵循本论坛帖子](https://steamcommunity.com/groups/SteamClientBeta/discussions/0/864957817327818370/) 中的规则，在 Steam 论坛上提交 bug、漏洞或崩溃报告。
*   Valve 也有一个 [黑客悬赏页面](https://hackerone.com/valve) 寻找安全漏洞。

#### **Wordpress**

*   如果你的 Wordpress 站点遇到了错误或者你在 Wordpress 后端遇到了问题， [在这里](https://make.wordpress.org/core/handbook/testing/reporting-bugs/) 提交错误报告。
*   如果你发现的问题是安全漏洞，提交给 [Wordpress bounty 程序](https://hackerone.com/wordpress) 。
