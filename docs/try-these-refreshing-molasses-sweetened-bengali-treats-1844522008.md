# 尝尝这些清爽的糖蜜加糖孟加拉点心

> 原文：<https://lifehacker.com/try-these-refreshing-molasses-sweetened-bengali-treats-1844522008>

随着气温的持续上升，我们对清凉消费品的需求也在增加。虽然你可能习惯于在像姜饼这样的烘焙食品中食用糖蜜，但这种黑色的美味甜味剂可以用来制作美味的夏季美食。



对于来自孟加拉国的人们来说，糖蜜被称为“goor”有三种受欢迎的古尔，鉴赏家可以通过颜色识别它们。根据杂货店国际部的活跃程度，你应该能在当地找到以下三种糖蜜。如果没有，你可以在网上购买它们的固体形式，然后在炉子上用小火加热糖蜜使其液化。(购买选项链接如下)。

第一种糖蜜是甘蔗汁第一次煮沸后剩下的糖浆，它是所有糖浆中最轻的。这种糖蜜被称为“[”aaker goor](https://www.indianproductsstore.com/karuppatti-palm-jaggery-panai-vellam.html)，“这种糖蜜通常尝起来非常甜，通常被用作煎饼和华夫饼的糖浆，或作为新鲜水果的蘸料——这是一种完美的夏季甜点。你可以整个夏天都把水果浸在糖蜜中，并且非常快乐，但是当有提神的饮料和凉爽的奶油布丁时，为什么要停下来呢？

### Jeera pani 甚至比柠檬水还要好

第二种颜色较深的糖蜜是从椰枣中提取的，被称为“T0”*【khejurer goor】。一种在孟加拉人中很受欢迎的配方被称为“jeera pani”，这是一种用孜然、罗望子和薄荷制成的甜味浓郁、口感复杂的饮料。它很有冲击力，很酸，非常提神。*

要做够两个人吃的，你需要:

*   2 杯水
*   一个柠檬的汁
*   1/4 杯糖蜜
*   1/4 杯罗望子酱
*   一撮烤孜然籽
*   10-15 片薄荷叶调味。
*   糖来调味

将除薄荷和糖以外的所有原料放入大罐中，搅拌直至糖蜜溶解。将薄荷叶在搅拌机中粉碎，加入到饮料中。让薄荷浸泡大约三十秒，尝一尝，如果需要的话，加入更多的薄荷和一点糖。如果你喜欢少一点的饮料，把薄荷叶滤出，然后在上菜前放入冰箱冷藏。

### 古尔·巴斯是你期待已久的免煮布丁

最后，我们有“ [”帕塔利古尔](https://www.desiauthentic.com/products/thati-bellam-palm-jaggery?variant=31633707827246&currency=USD&utm_medium=product_sync&utm_source=google&utm_content=sag_organic&utm_campaign=sag_organic) ，“一种从椰枣树的汁液中提取的浅色到中等颜色的糖蜜。这种类型的糖蜜是以硬化的形式出现的，所以当你把它添加到饮料和菜肴中时，你必须更有创造性。幸运的是，你只需要一点点米饭和一些奶粉(或全脂)就可以做出这道被称为“goor bhath”的冷奶油米饭布丁。要做够两个人吃的，你需要:

*   硬币大小的糖蜜，分解成 4 小块
*   两杯煮熟的茉莉香米(这种米饭会给你的布丁带来最好的味道)
*   一杯冷水
*   2 汤匙奶粉，最好是 [Nido](https://www.nestlenido.com/en/products/nido-fortificada) (或者，可以用 1/2 杯全脂牛奶代替奶粉和水。)

将糖蜜切成四分之一大小的小块，放在一边。米饭煮熟后，在室温下，将糖蜜撒入米饭中。你可以边吃边尝——加的越多，越甜——所以根据你的喜好调整量。接下来，加入牛奶。如果用奶粉，先和水混合再加到米饭里，一定要用冷水；温水意味着不冷不热的布丁，这是不可取的。你希望布丁看起来像奶油，但不要太稠——应该有点稀——所以根据需要添加更多的牛奶(或奶粉和水的混合物)。作为甜点享用，作为清淡甜美的早餐，或加入新鲜水果作为清爽的季节性午餐或晚餐。
