# 特朗普在加州野火问题上犯了什么错误

> 原文：<https://lifehacker.com/what-trump-gets-wrong-about-californias-wildfires-1830391393>

加州北部的营地大火刚刚成为该州历史上最致命的一次；南方的伍尔西大火正在吞噬马里布；我们的总统在推特上拐弯抹角地谈论伐木业。欢迎来到 2018 年。

从消防员到 CNN 气象员每个人都知道这是错误的，错误的，错误的，所以让我们来分解一下:

### 这些不全是“森林”火灾

营火确实始于森林地区，但南加州的大火并非如此。正如帕萨迪纳消防协会回复的 一样，这些被更好地描述为“城市界面火灾”——换句话说，这些火灾始于在易燃植被中建造房屋的地方。

“荒地火灾”是一个更准确的术语，它包括灌木丛火灾、牧场火灾等等。因此，如果你想象除了树在燃烧之外什么也没有，那么你一开始就有了错误的想法。

### 这些大规模、致命且代价高昂的野地火灾有一个重要原因

气候变化。我们真的要再复习一遍吗？

加州十大破坏性火灾 中，有九场发生在过去的 15 年里。*五个*是 2017 年和 2018 年的。

这些火的食谱需要 [几种常见的配料](https://www.wired.com/story/the-terrifying-science-behind-californias-massive-camp-fire/) :

*   非常靠近易燃植物的房屋
*   干燥天气使植物变得格外易燃
*   急流产生的高压风，每年这个时候都会出现。

加利福尼亚越热，就越干燥。极度的干燥使得植被——不仅仅是树，还有灌木和草——更容易燃烧。土地越干燥，着火所需的火花就越小。

然后，干热的阵风吸收了更多的水分，推动火势向前发展，将余烬抛向沿途的房屋和其他燃料。这位恼怒的气象学家 比我能解释得更好。

### “森林管理”可能是伐木业的一个口号

包括内政部长瑞安·津克在内的政界人士一直倡导“燃料减排管理”。思路似乎是:去掉森林，就不可能有森林火灾。

实际上，这意味着开放土地用于伐木。但是，正如外面的 [在这里](https://www.outsideonline.com/2244376/could-logging-reduce-wildfires) 解释的那样，伐木并不能有效清除野火赖以生存的燃料。伐木工人想拿走最大的、最有商业价值的树，留下树苗和灌木。猜猜什么东西最容易着火？是啊。树苗和灌木。

### 预防火灾和帮助受灾社区需要资金

还不清楚你能从图片中删除哪些联邦资金，并使消防管理更好。燃烧的大部分区域是联邦土地 。直到 2020 年，美国林务局 [仍然不得不削减自己的防火预算](https://www.hcn.org/articles/wildfire-fire-funding-fix-includes-environmental-rollbacks) 来筹集资金扑灭已经燃烧的大火。(这种被称为“火借”的做法最终以国会今年早些时候通过的一项法案而告终。)

随着火势变得越来越严重，扑灭野火的成本每年都在增加 。只是 2018 年生活的另一个令人兴奋的方面。
