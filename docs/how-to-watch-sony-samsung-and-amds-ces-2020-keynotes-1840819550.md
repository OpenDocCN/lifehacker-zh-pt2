# 如何观看索尼、三星和 AMD 的 CES 2020 主题演讲

> 原文：<https://lifehacker.com/how-to-watch-sony-samsung-and-amds-ces-2020-keynotes-1840819550>

消费电子展周二在拉斯维加斯正式开幕；然而，周一一些大公司将举行新闻发布会，展示他们在展会上发布的一些最大的产品。



就电视机而言，今年肯定是一个大年，预计将有来自不同制造商的 8000 台电视机展出。我们也可能会看到大量新的智能电器和其他设备，让我们的家在新的十年里变得更加智能，可能还会有更多的新相机、手机和技术配件。

虽然大多数品牌只允许媒体进入他们的 CES 2020 新闻发布会，但也有一些品牌提供直播，任何人都可以观看和跟随。

以下是如何收看 AMD、索尼和三星今年的重大发布:

## 超微半导体公司

AMD 的新闻发布会将于美国东部时间周一下午 2 点/5 点举行。

该公司计划展示如何让 2020 年成为“游戏玩家、创作者和移动 PC 用户难忘的一年”。

你可以在该公司专用的 CES 网站 上注册观看其 CES 主题演讲 [的直播。](https://www.amd.com/en/events/ces)

## 索尼

索尼的主题演讲定于美国东部时间下午 5 点/8 点举行。

该公司将在自己的网站上直播整个事件。

至于你可以期待什么，我们可能会看到 PlayStation 5 的预告片。我们肯定会看到新的电视，可能还有新的摄像设备和耳机。

## 三星电子

索尼发布会后不久，三星的 CES 2020 发布会将于美国太平洋时间下午 6:30/美国东部时间晚上 9:30 拉开帷幕。三星计划在其专用的网站 以及 [网站](https://www.facebook.com/samsungelectronics) 上播放 [。](https://news.samsung.com/global)

该公司对此次活动的官方声明是，它计划“展示在体验时代通过互动和智能技术丰富人们生活的创新方式”，不管这意味着什么。

三星 2020 年的公告预计将主要集中在新电视上，特别是新的 8k 型号。除了其他东西，我们还可能会看到一些智能电器。

三星在 也有一个专门发布其 CES 2020 公告的网站。
