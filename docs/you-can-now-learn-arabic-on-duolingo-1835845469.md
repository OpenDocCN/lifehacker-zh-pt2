# 你现在可以在多林哥上学习阿拉伯语

> 原文：<https://lifehacker.com/you-can-now-learn-arabic-on-duolingo-1835845469>

从今天开始，你可以使用 [多林戈](https://www.duolingo.com/) 学习阿拉伯语。

与该服务的其他语言课程类似，阿拉伯语课程在五分钟的课程中教你语言，感觉更像是一场游戏而不是一堂课。这些课程适应你的学习风格，并提供即时反馈，这样你就可以在学习过程中对自己的语言有一个良好的感觉。



阿拉伯语通常被认为是一种“难”学的语言，部分原因是它的语法非常复杂。阿拉伯语也使用不同于英语的字母表，这使得以英语为母语的人学习起来比使用相同字母表的法语或意大利语更复杂。

阿拉伯语课程将教授现代标准阿拉伯语，但多林戈指出，这将是一个不太正式的阿拉伯语口语版本。Duolingo 将其描述为“不是出现在诗歌或正式新闻广播中的版本，而是一旦新闻播音员停止朗读他们的脚本并开始与他们的受访者交谈时就会使用的版本。这是一种可以在对话中使用的语言版本，但也可以用于最广泛的阿拉伯语使用者。”

根据 Duolingo 的说法，研究表明，34 小时的 Duolingo 课程相当于大学一门语言的一个学期。这意味着如果你致力于学习语言，那么你就有可能学到和实际上课一样多的东西。当然，不同之处在于，Duolingo 是免费的，你可以在自己的时间里做一些事情。

如果你想今天就开始学习，Duolingo 的阿拉伯语课程是 [，既可以在线学习，也可以通过该公司的移动应用](https://www.duolingo.com/) 学习，后者适用于 iOS、Android 和 Windows Phone。
