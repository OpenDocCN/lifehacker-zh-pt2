# 关于不太常见的新冠肺炎症状，你需要知道什么

> 原文：<https://lifehacker.com/what-you-should-know-about-the-less-common-covid-19-sym-1843241146>

“典型的”新冠肺炎症状是发烧、干咳和气短，这些症状在几个月前首次报道，当时普通美国人只是通过一些零星的新闻报道熟悉这种疾病。从那以后，我们对新冠肺炎有了更多的了解，美国疾病控制和预防中心也大大扩展了官方症状清单 。这对你意味着什么？

新冠肺炎症状扩展列表包括:

*   发冷
*   寒战中反复颤抖
*   肌肉疼痛
*   头痛
*   喉咙痛
*   味觉或嗅觉的新丧失

CDC 指出，如果患者出现咳嗽和呼吸急促，或者出现上述至少两种症状，他们可能已经感染了冠状病毒，应该进行检测。



但是还有其他报告的疾病症状——如[“covid toes”](https://health.clevelandclinic.org/are-covid-toes-and-rashes-common-symptoms-of-coronavirus/)或 [中风风险增加](https://www.usatoday.com/story/news/health/2020/04/29/coronavirus-covid-19-caused-strokes-young-patients-nyc-doctors/3046447001/)——我们仍在了解它们有多罕见或常见，即使我们添加到列表中。随着我们理解的加深，如果你对自己的健康有任何担忧——无论你认为自己是否有新冠肺炎症状，倾听你的身体并与你的医生交谈真的很重要。与此同时，如果你读到一种新的或可怕的症状，这里有一些事情要记住。

### 大多数研究都是针对住院患者的

“这是一个完全进化的领域，”匹兹堡大学医学中心的传染病医生加文·哈里斯说。"大多数研究都是在住院病人身上进行的."

哈里斯指出，病情严重到需要住院的病人不一定会出现与病情较轻的人相同的症状。对于一些罕见的症状，我们仍有很多了解。

哈里斯还指出，新冠肺炎和其他呼吸系统疾病之间有很多重叠，因此没有一种症状是这种疾病独有的。相反，通常是症状的组合表明患者患有新冠肺炎，其诊断必须通过测试来确认。

“发烧不是一个普遍的症状，”哈里斯说。数字有所不同，但即使是病入膏肓需要住院治疗的病人也不总是发烧。

### 人们可能以不同的方式经历同样的疾病

正如任何疾病一样，人们通常不会以完全相同的方式生病。一些新冠肺炎患者会有干咳和气短，但没有发烧。其他患者可能会出现味觉或嗅觉丧失、头痛和干咳等症状，但不会出现呼吸急促。

除了疾病预防控制中心列出的症状，医生还报告了[【covid toes】](https://www.nytimes.com/2020/05/01/health/coronavirus-covid-toe.html?smid=tw-share)症状较轻的患者表现出脚趾肿胀。还有一些关于 [神经症状](https://www.nytimes.com/2020/04/01/health/coronavirus-stroke-seizures-confusion.html) 如中风、癫痫发作和精神混乱，以及 [心血管问题](https://khn.org/news/mysterious-heart-damage-not-just-lung-troubles-befalling-covid-19-patients/) 的报道。

这些都是医生已经注意到的症状，但我们还不太清楚病毒为什么以及如何引起这些症状。我们——以及医学界——仍处于危机模式，我们还需要一段时间才能真正了解所有这些症状的流行程度。重要的是要记住，我们所知道的东西仍在进化——新冠肺炎是一种复杂的疾病，可能会导致许多不同的并发症。

### 你是自己身体的专家

谈到监测症状，病人最了解自己的身体。哈里斯说:“随着症状的发展，人们必须思考自己的情况，以及这与你习惯的情况有何不同。”

如果你有季节性过敏，但你开始出现不太像你习惯的症状，这可能是一个信号，它实际上是别的东西。同样，如果你开始感到不舒服，那也是要和你的医生谈谈，并密切关注的事情。

“人们是自己身体的专家，”哈里斯说。“警惕真的很关键。”

### 呼吸急促是一个危险信号

说到治疗病人，哈里斯最关心的是一个症状。

“呼吸急促是一个危险信号，”哈里斯说。呼吸急促表明病人没有获得足够的氧气。虽然不是所有的病人都会呼吸急促，但这是哈里斯发现的最令人担忧的症状，因为它最能暗示病人正在发展可能导致住院的并发症。

随着这场危机的结束，我们无疑将了解更多与新冠肺炎相关的不同症状，以及这些症状在疾病如何影响我们的身体方面可能意味着什么。在那之前，密切关注你的健康状况并尽你所能帮助减缓传播是很重要的，这样像哈里斯这样的医生就可以拥有他们所需要的资源来保护我们所有人的安全。
