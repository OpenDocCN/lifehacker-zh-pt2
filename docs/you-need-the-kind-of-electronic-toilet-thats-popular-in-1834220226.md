# 你需要日本流行的那种电子马桶

> 原文：<https://lifehacker.com/you-need-the-kind-of-electronic-toilet-thats-popular-in-1834220226>

每个人都会大便。但不是每个人的大便都一样。

例如，在日本，在瓷王座上打理生意是一种全新的体验。想象一下，如果你愿意，下面的场景:

你冲进浴室，充满了墨西哥卷饼和遗憾。当你靠近时，你的马桶座会自动升高。当你坐着的时候，你的臀部不是被一个白色的学步车一样冰冷的东西抓住，而是被一个温暖的马桶座圈抓住。除臭剂在碗中喷洒，准备掩盖即将到来的气味。如释重负之后，你按下附近控制台上的一个按钮。坐浴盆会洗你的屁股，然后快速吹干。当马桶自动冲水时，你在水槽边洗手，你回到你的生活中，感觉比以前更快乐、更干净、更轻盈。

这种奢华的浴室体验不是幻想。这是日本的现实，这要归功于高科技电子厕所的普及。相比之下，美国的浴室感觉自石器时代以来就没有进步过。

在上面的视频中，Lifehacker 的撰稿人尼克·道格拉斯(Nick Douglas)参观了日本最大的电子马桶制造商之一 TOTO 的东京展厅，以了解更多关于这些令人惊叹的马桶的信息，为什么它们如此受欢迎，以及西方如何才能赶上。
