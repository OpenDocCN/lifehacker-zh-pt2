# 父母如何对抗性别刻板印象

> 原文：<https://lifehacker.com/how-parents-can-fight-gender-stereotypes-1839225144>

父母有足够的理由反对传统的性别刻板印象。我们希望我们的女孩从事 STEM 职业，我们希望我们的男孩在表达悲伤或恐惧时感到舒适，我们也希望不要假设他们的外表性别是他们内在的性别认同。



但正如#MeToo 运动反复强调的那样，在我们的国家努力应对性暴力盛行的同时，还有另一个原因来积极避免坚持传统的性别刻板印象:到了青春期，这些刻板印象可能会开始助长性暴力。 [《华盛顿邮报》报道](https://www.washingtonpost.com/outlook/2019/09/17/stop-sexual-violence-we-need-change-how-we-engage-with-kids/?arc404=true) :

> 研究表明，与其他十几岁的男孩相比，那些认同强烈性别刻板印象的人——例如，男孩想崇拜女孩是很自然的，女孩应该用自己的外表和身体来吸引男人——更有可能对女孩做出性评论并抓住女孩的身体。这部分是因为在青少年时期，男性的性别刻板印象开始融入男性主导、侵略和性冷漠的观念，而女性的观念开始以性和吸引力为中心。

我们对性别的关注始于性别揭秘派对，小女孩穿的“爸爸的小公主”t 恤，蹒跚学步的男孩穿的装满卡车的箱子。但随着孩子们长大，并开始内化玩具是如何针对特定性别进行营销的，男孩是如何被推向体育运动的，女孩是如何被鼓励参加芭蕾舞或音乐的，这种情况仍在继续。

孩子们从我们和他们周围的世界中获取线索，这些线索帮助他们开始对自己和他人进行分类。我们不可能避免所有这些，但是父母可以做一些事情——特别是在我们的孩子很小的时候——来影响他们对这些规范的接受程度。

### 你的话很重要

这里有一个简单的开始——当你谈论你的孩子和其他孩子时，尽可能用“孩子”、“朋友”或“学生”替换“女孩”和“男孩”不断地按性别给孩子分类会夸大其重要性。

这并不是说，当你看到或听到一些强化刻板印象的东西时，你不应该承认性别，甚至不应该指出来；你应该。你可能会说这样的话，“嗯，我想知道为什么这个故事中的科学家都是男孩，”这可以让他们批判性地思考性别角色在书籍和节目中是如何描绘的。当他们把刻板印象从学校带回家时——“粉红色是女孩的颜色”或“恐龙是男孩的”——问他们是否能想到为什么这可能不是真的。你可能会问:“你不知道有喜欢粉色的男生吗？”或者“嗯，你的表妹艾玛喜欢恐龙，而且她也不是男孩。”

通过在家里质疑这些刻板印象，你也在帮助他们发展在世界上质疑它们的技能。随着年龄的增长，和他们谈论性别歧视，就像你和他们谈论种族歧视一样，这样，正如《邮报》指出的，他们就不会得出自己的结论:

> 在没有任何性别歧视知识的情况下，孩子们更可能将他们在社会中看到的等级差异，例如我们从未有过女性总统的事实，归因于性别之间的先天差异。许多人会认为女性没有当选是因为她们不是好领导，或者不想当总统。

当我们调整谈论性别的方式时，让我们也放弃使用“假小子”这个词一个喜欢短裤和长裤而不是连衣裙的女孩，是运动型的，不是特别喜欢洋娃娃，但喜欢爬树，就是有这些偏好的女孩。她不需要被这些事情定义或贴上“男孩”的标签。

### 计划男女合演

这种情况发生得很无辜——你的小女孩和幼儿园的另一个小女孩变得不可分割。你喜欢另一个女孩的父母，于是频繁的玩耍约会开始了。这太棒了！但是特别是当他们还很小，在课间休息时还没有完全按性别分开的时候，你应该鼓励跨性别的互动。

主持幼儿园男孩和女孩的游戏日。让所有邻居的孩子放学后过来玩，或者给你的孩子报名参加男女同校的运动或俱乐部。当他们长大后，几个女孩想在休息时加入男孩的踢球游戏，这似乎不是什么大不了的事。

### 在他们所在的地方见他们

如果你的女儿喜欢公主裙——越蓬松越闪亮越好——没关系！如果你的儿子生活和呼吸火车，没问题！关键是不要引导他们远离他们自然感兴趣的、传统上与他们性别相关的事物。正如《大西洋月刊》 的 [视频中的爸爸们所说，关键是让他们做自己。](https://www.theatlantic.com/video/index/576091/gender-roles-kids/)

> 我不是说，“哦，如果你的女儿们喜欢兔子，你就不能和她们谈论兔子；你只能跟他们谈科学。”在他们所在的地方遇见他们，男孩也一样。我绝不是想暗示孩子们拥抱他们喜欢的事物是错误的，只是因为它落在一边或另一边；只是不应该由我们来强制。

与其他任何事情一样，孩子们大多会学习父母为他们树立的榜样，所以更多地根据每个伴侣喜欢做什么或擅长什么来分担家务和职责，而不是按照传统的性别角色，是一个很好的榜样。
