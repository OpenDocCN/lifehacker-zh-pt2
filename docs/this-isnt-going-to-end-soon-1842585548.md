# this isnt going to end soon 1842585548

> 原文:[https://life hacker . com/this-not-going-to-end-soon-1842585548](https://lifehacker.com/this-isnt-going-to-end-soon-1842585548)

我不想过分严肃，但是让我们现实一点。如果一个地方说他们关闭两周，或者如果一个四月的活动被推迟到五月，这里是我的建议:不要指望它。

当我孩子的学校说他们要关闭两个星期时，我把它理解为“关闭，等待进一步通知。”当然，出于规划的目的，他们必须确定一个日期来重新评估，但这不是我真正关心的。当然，他们不得不延长关闭时间——我不断听到家长们抱怨日期的改变，好像他们期望并应该得到一个明确的消息。伙计们，我们在未知的领域。

这个疫情是新的，这些关闭和留在家里的订单前所未有。这不像流感季节，我们知道它什么时候开始，什么时候达到高峰，持续多长时间。没有人知道会发生什么。我们只知道这才刚刚开始。

早在二月份，关闭有可能是短暂的。“流行病学家说，如果有可能挥动魔杖，让所有的美国人坐在相距六英尺的地方呆在原地 14 天，整个流行病就会停止，”纽约时报 [上周报道](https://www.nytimes.com/2020/03/22/health/coronavirus-restrictions-us.html?action=click&module=Spotlight&pgtype=Homepage) 。但是社会并没有停滞不前。一些学校仍然开放，一些州没有关闭非必要的企业，一些人继续参加聚会，各地的人们都在保持必要的互动，如寻求紧急医疗护理。

所以我们不知道接下来会发生什么，但很明显，我们的疫情曲线仍在上升。不管你看的是哪个追踪器，也不管测试是捕获了大部分案例还是只捕获了极小的一部分: [每一个图都在攀升](http://91-divoc.com/pages/covid-visualization/) ，案例不可能突然暴跌为零。我们会在这里待一段时间。

多久了？没人能说清楚，老实说，现在就对任何具体的预测下结论还为时过早。也许九月份学校会重新开学，也许不会。也许检测工作最终将能够赶上足以找出多少人真的有冠状病毒，这将使有效的接触追踪，以杜绝韩国式的传播。即使在最好的情况下，我们也需要一段时间才能真正恢复正常。

所以，我们现在的处境是，没有办法准确地规划未来。我们所能做的就是一天一天的慢慢来。在我写这篇文章的时候，我偶然看到了 Liz Neeley 的一篇关于如何与你的朋友和家人谈论冠状病毒 的精彩文章，她在文章中强调了前战俘 James Stockdale 的一句名言:

> 斯托克代尔后来 [将](https://www.jimcollins.com/media_topics/TheStockdaleParadox.html) 的幸存归功于他“从未对故事的结局失去信心”，不像那些“说‘我们会在圣诞节前出去’的人”圣诞节来了，又走了。然后他们会说，‘我们会在复活节前出去。复活节来了，又去了。然后是感恩节，然后又是圣诞节。他们因心碎而死。"

如果斯托克代尔是对的，拥抱不确定性可能会帮助我们度过难关。
