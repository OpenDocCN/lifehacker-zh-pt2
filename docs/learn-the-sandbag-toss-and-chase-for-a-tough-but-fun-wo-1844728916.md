# 学习“沙袋投掷和追逐”这一艰难但有趣的运动

事情已经到了这一步，在沙袋健身挑战赛的这一周，我尝试了臭名昭著的。户外。在酷热中。如果我活下来了，你也可以。

折腾和追逐很简单。你捡起沙袋，把它扔得越远越好，然后向它跑过去。然后你重复这个过程，就像一场乒乓球比赛，你没有对手，只有你自己，球的重量是 20 或 50 或 100 磅，而不是什么都没有。

沙袋并不适合投掷，所以你不能把它们扔得很远。好消息是:你的跑步时间很短。坏消息是:这只会让投掷更加接近。

总之，这是我像个十足的傻瓜一样扔着一个 50 磅重的沙袋:

原来也没那么差！我很高兴在 10 次重复后休息，但这不是一种“倒在一滩汗水中”的锻炼。当然，袋子越重，越结实。我发现我可以在大约 50 磅的时间内(一分钟)重复 10 次 35 磅的袋子。作为比较，我深蹲 200+磅，力量清洁 130 左右。根据你的力量增加或减少，如果你的包很轻就多做几次，如果你的包很重就少做几次。

由于你在短时间内努力工作，然后给自己一个休息的机会，如果你讨厌有氧运动，这种间歇运动可以称为“调节”，如果你喜欢，可以称为“HIIT”。

哦，开始前先修剪一下你的指甲。那些吸盘会卡住的。
