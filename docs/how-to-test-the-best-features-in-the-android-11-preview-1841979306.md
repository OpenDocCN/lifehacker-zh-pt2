# 如何测试 Android 11 预览版中的最佳功能

> 原文:[https://life hacker . com/how-to-test-the-best-features-in-the-Android-11-preview-1841979306](https://lifehacker.com/how-to-test-the-best-features-in-the-android-11-preview-1841979306)

这么多 Android 设备 [还在等待 Android 10](https://lifehacker.com/find-out-when-your-samsung-device-is-getting-android-10-1840235349) ，很难相信 Android 11 的第一个 [预览版已经面世](https://developer.android.com/preview) 。Android 11 目前没有官方发布日期，普通用户还有一段时间才有机会试用公共测试版，但早期测试人员已经发现了 Android 11 将在今年晚些时候推出的一些功能。以下是一些最值得注意的例子:

*   **事故跟踪(仅限 Pixel):**一旦升级到 Android 11，所有 Pixel 设备都将收到一个以前 Pixel 4 专属的汽车安全应用程序版本——尽管用户也可以 [立即安装该应用程序的早期版本](https://lifehacker.com/how-to-get-googles-car-crash-detection-app-on-older-pix-1841834525) ，如果他们不想等待的话。
*   **聊天气泡:**旧版本的 Android 使用类似 Facebook Messenger 的气泡来快速访问短信对话，它们似乎在 Android 11 中卷土重来。然而，它们将如何工作以及看起来会是什么样子还不清楚。
*   **可定制的分享菜单:**用户将能够个性化 Android 11 分享菜单中列出的应用和联系人。
*   **黑暗主题定时器:**虽然在 Android 10 上已经有 [应用](https://lifehacker.com/use-this-hack-to-create-dark-mode-timers-in-android-10-1838101522) 可以 [做到这一点](https://lifehacker.com/force-android-10s-dark-mode-on-individual-apps-with-dar-1841577354) ，但在 Android 11 中似乎将原生提供用于打开或关闭黑暗主题的自动定时器。
*   **5G 连接增强:**Android 11 预览版包含新的 API，应该可以让开发者和制造商更容易地开发 5G 应用和功能。 [购买 5G 手机可能仍然是个坏主意](https://lifehacker.com/heres-everywhere-your-new-5g-phone-can-get-5g-speeds-1840679254) ，尽管 Android 11 让它们变得更好一些。
*   **运动感应媒体播放控制(仅限 Pixel):**Pixel 手机还将能够使用简单的手势控制来播放/暂停音频播放，而不必点击通知栏控制。
*   **新的应用权限选项:** Android 11 看起来会让用户授予一次性的应用权限。例如，如果你只想让一个应用程序在特定时间访问你的相机，你可以点击“只有这一次”选项。该应用程序将能够使用该功能一次，然后将不得不在未来再次询问。
*   **屏幕记录:**用户将能够使用操作系统的内置功能记录他们的屏幕活动，而不是第三方应用程序。这项功能最初是为 Android 10 计划的，但后来又回到了 11(至少目前是这样)。

### 早早试用 Android 11

如果你真的想尽早试用 Android 11，你可以尝试下载并在你的 Pixel 手机上安装当前的开发者版本(仅支持 Pixel 2、3、3a 和 4 设备)，但首先要确保你知道你将会遇到什么。

安装开发者版本不同于将你的设备注册到 Android beta 测试中，更不用说复杂得多了。如果你尝试任何超早期的 Android 11 版本，你将被迫重置你的手机，所以在尝试之前请备份你的设备。 谷歌也强烈建议你不要在主手机上使用 Android 11 开发者 buildAndroid 11 仍处于开发初期，大多数应用程序或功能都无法正常运行(如果有的话)。

也就是说，如果你对稳定性问题和限制没有问题，你可以在这里 [下载 Android 11 开发者版本](https://developer.android.com/preview/download) ，然后 [按照这个页面上的步骤](https://android.gadgethacks.com/how-to/complete-guide-flashing-factory-images-android-using-fastboot-0175277/) 将其安装到你的 Pixel 设备上。
