# 感恩节你需要一碗克莱门汀

> 原文：<https://lifehacker.com/you-need-a-bowl-of-clementines-at-thanksgiving-1839925024>

我对我不能吃的餐桌摆饰没有耐心。花很漂亮——我一直这么认为——但是在桌子中间放一瓶不可食用的绿色植物只会给自己将来制造麻烦；问题是“我把这个东西放在哪里来给食物腾出空间？”

小柑橘类水果没有问题，只有解决方案。一年中的这个时候，你可以看到克莱门蒂和温州蜜柑，它们的叶子还连着，它们是美丽的、实用的中心装饰品。

功能很多，但首先是健康的功能。感恩节经常有蔬菜，但很少有水果。是的，有蔓越莓酱，但是你真的吃了多少？几勺？如果你是个超级粉丝的话，可以喝四分之一杯？吃一片水果会让你感觉更健康，更阳光，吃一片柑橘类水果会让你觉得你在积极地抵御坏血病或其他一些海盗疾病。即使你除了馅料和淹没在肉汁中的土豆以及多块馅饼之外什么也没吃，你仍然可以回顾你的一天并想“至少我吃了四个克莱门蒂”，然后谷歌一下维生素 C 的好处以进一步减轻任何不必要的负罪感。

克莱门汀和温州蜜柑也是感恩节前的理想小吃。他们不需要你做任何准备工作，给你的客人一些事情做——削水果——这是一项让周围空气闻起来令人愉快的活动。不像蘸酱和熟食板，没有人会吃足够多的柠檬来破坏大餐。它们也非常适合扔给饥饿的孩子，除非他们真的很小，否则他们完全有能力在你继续做饭的时候剥水果并喂自己。

如果你不喜欢餐桌中央的装饰品，你可以用这些小水果作为(可爱的、划算的、可食用的)座位架。用铅笔在水果上写下每位客人的名字。(不用担心，墨水不会透过皮肤漏出来的。)感恩节我在祖母家做过一次，让我告诉你，她被迷住了。

所以，是的，买一堆柠檬和/或温州蜜柑(*带叶子*)，把它们放在碗里，然后把碗放在桌子上。您的中间拼接现在已排序。
