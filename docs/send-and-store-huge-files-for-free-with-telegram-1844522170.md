这并不新鲜，但是 [Telegram](https://telegram.org/) 昨天宣布提高你可以在它的服务上发送的文件的大小限制——并且，再次，免费存储在云中*——到一个巨大的 2GB。对于一个完全免费的云存储服务来说，这太疯狂了。当然，如果你以前从未尝试过，也有足够的理由使用它。*

*如果你已经在使用 Telegram，那么对于单个文件来说，这项服务从之前的 1.5GB 提升到 2GB 可能没什么大不了的。这仍然是一个令人难以置信的提醒，这个聊天应用程序是一个与朋友和/或自己分享文件的伟大工具。*

*唯一的问题是。您的朋友(或您的其他设备)需要安装一个电报帐户，以便下载您在聊天中放置的任何内容。没有办法创建 Telegram 上托管的文件的公共链接。毕竟这不是 Google Drive。*

*不过，我很感激 Telegram 有一个可移植的 Windows 应用程序(除了一个可安装的应用程序之外)，以及你可能想在其上使用它的其他平台的应用程序。甚至还有 [一款网络应用](https://web.telegram.org/) 。*

### *如何在 Telegram 上与自己共享文件*

*在 Telegram 上进行设置很容易:你只需提供你的电话号码，不需要用户名或密码(尽管如果你愿意，你可以*创建一个用户名；它允许其他人更容易地在服务上找到你)。**

*一旦你做到了这一点，创建一个与自己聊天的最简单的方法——这将成为你的文件的转储地——就是简单地创建一个只有你在里面的新的私人频道。*

*Telegram 仍然会给你一个邀请链接，允许你邀请其他 Telegram 用户加入所述频道，但是如果你所寻求的只是一种简单得不能再简单的方法来在你自己的系统和设备之间快速共享文件，就忽略它。*

*我现在使用的是 Windows 客户端，我只是拖放了一些文件到我的 Mac 电脑上。这样做的速度和把它们放入 Dropbox 文件夹一样快，但是我真的不用担心文件有多大或者它们占用了多少空间。令人愉快。*

*所述文件基本上永远存在，因为我怀疑你会达到“聊天中消息太多”的限制——无论那可能是什么——并且你可以上传的文件数量没有限制。我读过一些关于人们在几年内将 2TB 的数据放入电报聊天中的报道——都是可搜索的(通常在描述中使用标签组织),并且都是可立即下载的。而且是免费的。*
