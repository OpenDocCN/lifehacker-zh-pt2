# 关于学生贷款利息减免你应该知道什么

> 原文：<https://lifehacker.com/what-you-should-know-about-the-student-loan-interest-wa-1842358284>

周五，特朗普总统表示，他将免除联邦学生贷款的利息(T2)。但这一举措并不像听起来那么令人兴奋。



你仍然需要像往常一样支付你的联邦学生贷款。不同之处在于，你的全部还款都将计入你的本金，而不是通常的本金加日利息。

你将会在你的学生贷款上看到主要储蓄的唯一方法是如果你在利息免除有效的时候支付一大笔钱，足以显著地减少你的本金。这样，当兴趣恢复时，你将有一个较低的余额，利息可以增长。

但是如果你现在担心你的财务状况，你可能不会考虑为你的学生贷款余额支付一大笔钱。所以，当你坚持定期还款的时候，把利息减免看作是减缓贷款余额增长的一种方法。

如果你很难支付最低还款额怎么办？利息减免对你来说有一点点实际的好处。正如罗恩·利伯在《纽约时报》的中所写的那样，处于延期期的联邦贷款通常会产生利息——也就是说，由于财政困难，还款已经暂停。现在，豁免就像是给你的贷款划上一个 100%的句号——既包括你的还款，也包括你的余额因利息而增长的速度。

您不必为您的合格贷款做任何事情来获得利息豁免，您将在大约一周内看到您的贷款状态的变化(利息冻结追溯到周五)。

但是 Lieber 指出了一些不符合豁免条件的贷款:

*   私人贷款
*   联邦家庭教育贷款
*   你们学校持有的珀金斯贷款

还有一个更大的未知数:我们还不知道当豁免期结束时，被豁免的利息是否会重新计入贷款。预计教育部将很快敲定计划并公之于众。

如果你预计在支付最低学生贷款时会有困难，联系你的贷款人 [讨论你的选择](https://lifehacker.com/what-you-should-know-if-you-need-to-delay-your-student-1792867238) 。如果你发现自己在冠状病毒爆发期间失业了一段时间，很可能你可以暂停支付学生贷款，直到你的收入稳定下来。
