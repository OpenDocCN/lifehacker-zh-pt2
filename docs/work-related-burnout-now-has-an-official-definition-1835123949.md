# 与工作相关的倦怠有了新的官方定义

> 原文:[https://life hacker . com/work-related-burnout-now-has-an-official-definition-1835123949](https://lifehacker.com/work-related-burnout-now-has-an-official-definition-1835123949)

我们大多数人都经历过一种模糊的被工作“耗尽”的感觉，但是现在对于这种感觉的真正含义有了一个具体的定义。世界卫生组织最近更新了他们的 [国际疾病分类编码](https://www.who.int/health-topics/international-classification-of-diseases) (ICD-11)，将倦怠定义为一种具有三个维度的综合征。

Watch

新的定义并不意味着倦怠(或者，他们称之为“烧尽”)是一种疾病；是 [将](https://icd.who.int/browse11/l-m/en#/http%3a%2f%2fid.who.int%2ficd%2fentity%2f129180281) 归类为“影响健康状况的因素”这是新的定义:

> 倦怠是一种综合征，被认为是由长期工作压力导致的，并且没有得到成功的管理。它以三个维度为特征:1)能量耗尽或衰竭的感觉；2)与工作的精神距离增加，或与工作有关的消极或愤世嫉俗的感觉；3)专业效能降低。职业倦怠特指职业环境中的现象，不应该用来描述生活中其他领域的经历。

如果你觉得自己符合倦怠的标准，下次你寻求医疗或心理健康护理时，你可能会想提一下。就其本身而言，倦怠不被认为是一种医疗状况，但它代表了你生活中可能需要应对的额外压力。 [需要的话寻求专业帮助](https://vitals.lifehacker.com/your-basic-guide-to-seeking-help-for-your-mental-health-1829656946) ，或者考虑 [从工作中抽出一个“心理健康日”](https://vitals.lifehacker.com/how-to-take-a-mental-health-day-from-your-job-1826316891) 是否可以给你足够的喘息空间来考虑你的选择。
