# 哪些公司现在还在招人？

> 原文：<https://lifehacker.com/which-companies-are-still-hiring-right-now-1842889382>

在大规模裁员和一连串的招聘冻结之间，飙升的失业率让许多人怀疑现在找工作是否值得。如果你正在找工作，你可能会觉得你面对的是数百万人在冠状病毒关闭期间争夺任何微薄的工作。

Watch

但即使是现在，一些行业仍在引进新员工。虽然该平台上的职位发布总数比一个月前下降了 40%，但 Snagajob 分享了几个招聘人数大幅上升的领域。

自 3 月 11 日以来，物流和仓库岗位增加了 156%以上，零售、运输和杂货岗位也是如此。医疗保健和便利店头寸略有下降，降幅均不到 5%。下滑幅度最大的是临时员工职位(下降 74%)、娱乐(46%)、酒店(34%)、休闲餐厅(30%)和清洁服务(23%)。

想尽快开始新的工作吗？

Snagajob 有一个页面专门介绍公司立即招聘的职位，顶级职位包括杂货店店员、客户服务代理、收银员和送货司机。招聘员工的公司包括 Ace Hardware、达美乐、UPS 和 Walgreens 等。 [Glassdoor 的博客](https://www.glassdoor.com/blog/companies-hiring/) 也呼吁公司在疫情期间招聘，而 [Monster](https://www.monster.com/career-advice/article/in-demand-jobs-hiring-now) 也强调了公司现在正在集体招聘。
