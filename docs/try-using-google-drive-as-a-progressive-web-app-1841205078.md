# 尝试使用 Google Drive 作为渐进式网络应用程序

> 原文：<https://lifehacker.com/try-using-google-drive-as-a-progressive-web-app-1841205078>

Google Drive 现在可以作为桌面和移动的渐进式网络应用程序(PWA)使用，为用户提供了在常规浏览器中启动该服务的另一种选择。pwa 看起来和感觉上都像精简版的应用程序，尽管它们基本上存在于浏览器的精简版中。它们就像是应用程序和网站之间的中间地带；虽然它们仍然需要互联网连接才能打开和运行，但它们比应用程序占用的空间更少，处理速度通常比加载整个浏览器(及其许多标签)要快得多。



你现在可以使用 Chrome 浏览器在 Windows 10、macOS 和 Chrome OS 上安装 Drive PWA，不过 Windows 10 用户也可以通过 Edge Chromium 安装该应用程序。

*   [在 Chrome 标签中打开 Google Drive](https://www.google.com/drive/) 。
*   进入驱动器后，单击书签星形图标旁边的地址栏中的 **"+"** 图标。
*   将出现一个弹出窗口，询问您是否要安装驱动程序。点击**“安装”**
*   将出现一个新的驱动应用程序窗口，其界面与网站版本相同。快捷方式也将被添加到您的桌面上。

Android 用户可以安装 Drive PWA 的移动版本，方法是在 Chrome 移动浏览器中打开 Drive，点击汉堡图标，然后**“添加到主屏幕”，**但我们建议继续使用现有的 Android 应用。(比较好用)

与通常完全在自己的窗口中运行的其他 pwa 不同，某些驱动器文件将在 Chrome 标签中打开，而不是在应用程序中打开(这在桌面和移动上都存在)。所以，你现在运行的是一种 PWA 体验，但如果你总是在处理 Google Drive，它仍然是有用的。
