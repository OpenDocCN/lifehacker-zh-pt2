# 加入这个没人听说过的社交网络

> 原文:[https://life hacker . com/the-new-social-network-sunshine-looking-yet-another-1836912567](https://lifehacker.com/the-new-social-network-sunshine-looks-like-yet-another-1836912567)

新的社交网络 [【阳光】](https://sunshine.social/) 看起来又是一个 Twitter 的克隆，但它是一个极端极简主义者。事实上，它没有 T4 的消息。相反，阳光一次显示一篇文章。一旦你和这篇文章互动(或者点击“下一步”)，你就不会再看到它了。(您可以在用户的个人资料页面上看到用户的所有帖子。)它的用户不到一千人，注册非常容易。和你所有的朋友一起入侵它，打造你自己的小社交网络。(via [产品搜索](https://www.producthunt.com/posts/sunshine-8) )
