# 最好的飞机点心是柑橘

> 原文：<https://lifehacker.com/the-best-plane-snack-is-a-mandarin-orange-1836378421>

飞行是一件乏味的事情，毫无魅力可言。座位不好。空气不好。食物是糟糕的。让自己从所有糟糕的事情中转移注意力是度过飞行的唯一方法，而且(惊喜，惊喜)我喜欢用食物转移注意力。



但是光分散你的注意力是不够的。最适合这项工作的食物是采用多感官分散注意力方法的食物。这就是为什么最好的飞机点心是柑橘。没错，它尝起来像阳光，但它也向你周围的污浊空气中释放出美妙的柑橘油，让你的个人空间变得更加明亮、令人精神振奋，几乎令人愉悦。

它们还被密封在自己的包装中，这意味着你可以确保在水果接触你的舌头之前，没有细菌接触过水果的果肉。我更喜欢 Cutie 品牌的橘子，因为它们通常是最容易剥的，而且一年到头都很容易在杂货店里找到。就数量而言，我喜欢每小时至少带一个，尽管我通常最少带四个，因为一次不可能少于四个柑橘。
