# 使用此扩展按取消政策和登记时间搜索 Airbnbs

> 原文:[https://life hacker . com/search-for-airbnbs-by-cancellation-policy-and-check-in-1840958303](https://lifehacker.com/search-for-airbnbs-by-cancellation-policy-and-check-in-1840958303)

当你在找 Airbnb 的时候，大多数情况下内置的搜索功能都很好用。有时候；但是，您还有一些标准搜索未涵盖的住宿规定。

Watch

例如，也许你想找一个可以提前入住或者为长期入住提供折扣的旅馆。 [Airplus 是 Chrome 的扩展](https://chrome.google.com/webstore/detail/airplus-advanced-options/bdcdjfoindheggeigmgocjlglbphnkdn?ref=producthunt) ，它给网站增加了额外的搜索层，这样你就可以更容易地显示信息。

有了它，你可以指定入住或退房的时间，酒店是否提供行李寄存服务，以及你是否可以办理入住手续，或者你是否会等待主人的到来。

当你在 Airbnb 上搜索时，你仍然会得到你通常会得到的所有搜索结果，但不符合你添加的标准的选项将被显示为灰色，让你马上知道它们不会是最合适的。

更好的是，一旦你有了这些搜索结果，你可以在页面上点击鼠标右键，按评级对结果进行排序，这样你就可以立即看到什么是对你来说“最好”的属性。

将鼠标放在一个酒店的缩略图上也会显示基本的主机和酒店信息，这样你就可以在点击进入实际发布之前，从搜索页面得到一个基本的概念。

如果你正在寻找一个住宿的地方，并关心诸如取消政策或你是否能停留一天或一个多月之类的事情，那么它可以成为帮助你减少平均搜索时间的一个很好的资源。
