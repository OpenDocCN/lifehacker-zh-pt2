# 如果你想送货上门，现在就加入亚马逊生鲜和全食超市的等候名单吧

> 原文：<https://lifehacker.com/if-you-want-grocery-delivery-get-on-the-amazon-fresh-a-1842840589>

如果你正在考虑第一次尝试亚马逊生鲜或全食超市的送货或提货服务，请注意，在使用这两种服务购买食品之前，你可能需要在等候名单上花一些时间——所以你可能想现在就把自己放在名单上。



正如亚马逊在最近的一篇博文中宣布的:

> 我们暂时要求新的亚马逊生鲜和全食超市送货和取件客户注册邀请，以使用在线杂货送货和取件。我们每周都在增加产能，每周都会邀请新顾客来购物。

换句话说:如果你以前从未通过亚马逊生鲜或全食超市订购过，你就必须请求邀请。是的，即使您是，您也有资格获得免费的亚马逊生鲜和全食超市配送，作为您的亚马逊 Prime 会员的一部分。

我以前在亚马逊下过新鲜订单，所以亚马逊现在让我通过这项服务购物没有任何问题(除了有限的送货时段)。当我试图下我的第一个全食超市订单，看看会发生什么时，亚马逊告诉我，我需要注册才能收到通知，告诉我什么时候可以购物。

换句话说，是一份候补名单。

这个名单是全新的，所以仍然有可能获得一个体面的位置(尽管谁知道自从亚马逊宣布以来已经有多少人注册了)。一旦你离开了等待名单，你可能就能利用亚马逊新的送货窗口功能，假设他们那时已经启动并运行了:

> [...]在接下来的几周，我们将推出一项新功能，让顾客能够争取时间购物。该功能将为交付客户提供一个虚拟的“排队位置”，并允许我们根据先到先得的原则分配交付窗口。同时，我们将继续尽快增加容量。

亚马逊还提醒购物者，如果你不想等待购物邀请或送货窗口，你可以亲自去全食超市——当然，如果你的健康、风险因素和责任允许的话。请记住，许多全食超市是在减少营业时间的情况下营业的，所以去之前一定要检查一下时间表。
