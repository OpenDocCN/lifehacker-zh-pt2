# 这些是最好的机场

> 原文：<https://lifehacker.com/these-are-the-best-airports-to-get-stuck-in-1838630521>

2019 年北美机场满意度研究 于上周发布，同时发布的还有美国机场排名。

Watch

这项研究 考察了旅行体验的方方面面，从你在机场办理登机手续和通过安检开始，到你的购物和饮食选择，以及到达登机口有多复杂。

机场分为大型、大型和中型机场。

实际上，这些信息可能不会给你带来太多好处。在选择机场时，我们都可能会受到我们居住的地方和我们要去的地方的限制。也就是说，当你想在哪里建立联系时，这个列表会派上用场。

我知道当我旅行的时候，我经常要在几个不同的机场中选择我计划中途停留的地方。我个人经常选择飞越亚特兰大进行国际旅行，只是因为它将来自西海岸的国际航班一分为二，不像纽约市的机场那么难导航，而且我知道 T2 T3、T4 T5 的确切位置。).

我还会故意在拉斯维加斯停留，因为那里有成吨的食物，如果你真的觉得无聊，还有老虎机。另外，维加斯拥有世界上最多的酒店房间(说真的)，所以如果我真的被困住了，至少我可以一边赌小马，一边在院子里的冰冻草莓代基里借酒消愁。

在你熟悉的机场转机也有很多好处，尤其是当你在处理一个紧密的转机时。同样的道理也适用于选择美味食物的机场。在最近的一次中途停留期间，由于航班严重延误，我只能选择在必胜客就餐。具体来说，一家必胜客在延迟四小时后关门一小时。不理想。

你想要一个营业时间很晚的有食物选择的机场。当然，当你在那里的时候，你可能不吃东西，但是如果你吃的话(或者如果你意外地被困住了)，有这个选择是很好的。

以下是该研究中提供最佳整体体验的顶级机场列表。下次你预订中途停留的旅程时，如果其中一个是你的选择，它可能是一个安全的赌注。
