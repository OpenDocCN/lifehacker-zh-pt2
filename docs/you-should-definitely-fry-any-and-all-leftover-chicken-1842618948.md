# 你绝对应该油炸所有剩下的鸡皮

> 原文：<https://lifehacker.com/you-should-definitely-fry-any-and-all-leftover-chicken-1842618948>

我从来没有完全信任过不喜欢吃烤鸡皮的人。(这里面包括我妈。)无论是刚出炉还是从好市多烤肉店(Costco rotisserie)吐出来的脆皮，都是家禽消费的亮点之一。那些留下它的人是*傻瓜*。



如果我一个人的话，我通常会一下子把所有的皮都剥下来，然后马上吃掉，但是显然有些人觉得这样很恶心。可悲的是，一旦皮肤在冰箱里冷却，它就会变得像皮革一样，有粘性，而且整体上不太好。但是如果你被黏糊糊的剩余皮肤卡住了，你可以给它一个新的——我敢说更好吗？—油炸生活。

任何种类的油炸鸡皮都是美味的，但是已经煮熟的鸡肉的油炸鸡皮是特别的。它已经煮过一次了，所以大部分的脂肪已经呈现出来，胶原蛋白也融化了，所以它很容易就变脆了。它通常也已经调味了，随着鸡皮的油炸，这些调味料会变得浓缩，收缩成一个完美的鸡片。即使是鸟底部灰色松弛的皮肤也能在几分钟内变成金黄色和松脆的东西。

制作它很容易。只需(轻轻地)剥掉煮熟的鸡肉的皮，然后将鸡肉块放入冷的不粘锅中。把锅放在炉子上，把火调到中火，把肉片煎至金黄酥脆，翻转一次，两面都煎好(用筷子煎最好)。转移到纸巾，盐(如果需要的话)和三明治或沙拉中享用，撒在意大利面上或单独享用。
