# 查看亚马逊的“最想要的礼物”页面，寻找礼物灵感

> 原文:[https://life hacker . com/check-amazons-most-wish-for-page-gift-inspiration-1834717973](https://lifehacker.com/check-amazons-most-wished-for-page-for-gift-inspiration-1834717973)

如果你一直在亚马逊愿望清单上为将来的购买藏东西，你可能会无意中为完全陌生的人提供礼物想法。

亚马逊将其愿望清单和注册表中的所有“”最想要的物品汇编成一个巨大的数据库 [，用户可以按类别](https://www.amazon.com/gp/most-wished-for/ref=zg_mw_unv_k_0_k_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/check-amazons-most-wished-for-page-for-gift-inspiration-1834717973&asc_source=&tag=kinjalifehackerlink-20) 进行搜索。结果——每天更新——是一个礼物想法的宝库，你可以为即将到来的生日、假期或特殊场合仔细阅读。

也许你最好的朋友喜欢烘焙，但你知道她已经得到了所有的基本用品，所以你去了亚马逊“最想要的东西”的“”部分这就是你找到这些有趣的 [硅胶烘焙垫](https://www.amazon.com/AmazonBasics-Silicone-Macaron-Baking-Mat/dp/B077XR38W3/ref=zg_mw_kitchen_5?_encoding=UTF8&asc_campaign=InlineText&asc_refurl=https://lifehacker.com/check-amazons-most-wished-for-page-for-gift-inspiration-1834717973&asc_source=&psc=1&refRID=C9RE720V02XVG4Y9NWM7&tag=kinjalifehackerlink-20) 的地方，她从来不知道她一直需要这些东西。

你的侄子快 3 岁了，而你却不知道 3 岁的孩子喜欢什么？转到“ [玩具和游戏](https://www.amazon.com/gp/most-wished-for/toys-and-games/ref=zg_mw_nav_0?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/check-amazons-most-wished-for-page-for-gift-inspiration-1834717973&asc_source=&tag=kinjalifehackerlink-20) ”，然后转到“ [学前](https://www.amazon.com/gp/most-wished-for/toys-and-games/251909011/ref=zg_mw_nav_t_1_t?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/check-amazons-most-wished-for-page-for-gift-inspiration-1834717973&asc_source=&tag=kinjalifehackerlink-20) ”子类别，当你带着 [一套假装看家设置](https://www.amazon.com/Melissa-Doug-Housekeeping-Organizing-Confidence-Building/dp/B00FZMDAO6/ref=zg_mw_251909011_20?_encoding=UTF8&asc_campaign=InlineText&asc_refurl=https://lifehacker.com/check-amazons-most-wished-for-page-for-gift-inspiration-1834717973&asc_source=&psc=1&refRID=HV6QQDR5S3T1HB4J3VBP&tag=kinjalifehackerlink-20) 到达派对时，成为一个意想不到的英雄。

你一定知道有人会喜欢这个 [格鲁特花盆](https://www.amazon.com/B-BEST-Guardians-Drainage-Succulents-Halloween/dp/B07D1KVD2V/ref=zg_mw_home-garden_20?_encoding=UTF8&asc_campaign=InlineText&asc_refurl=https://lifehacker.com/check-amazons-most-wished-for-page-for-gift-inspiration-1834717973&asc_source=&psc=1&refRID=NAABGTMFZBSTZV8YJ75N&tag=kinjalifehackerlink-20) 。它目前在“家庭和厨房”排行榜上排名第 20 位，所以有更多的人希望得到它，这可能超出了人们的想象。

有(及子类别)几十个品类，包括汽车、书籍、草坪和花园、电子产品、宠物用品、办公用品和工具。

(不得不说:你实际上不必直接从亚马逊购买这些商品。如果你不是亚马逊的粉丝也没关系；你仍然可以利用他们的想法*。*)
