# 在计划一次旅行之前，查看一下国务院的旅行建议

> 原文：<https://lifehacker.com/check-the-state-departments-travel-advisories-before-pl-1835295227>

计划今年夏天来一次的单人旅行？在你独自去之前，为了你自己的安全，做好你的功课，尽你所能去研究。最近，正好赶上繁忙的夏季旅游季节，美国国务院在其 [网站](https://travel.state.gov/content/travel/en/traveladvisories/traveladvisories.html/) 上更新了一些旅游建议。



如果您以前从未审查过它们(您应该这样做)，该部门会根据以下四个建议级别之一对每个国家进行评级:

*   第一级:采取正常的预防措施
*   第二级:加强防范
*   第三层:重新考虑旅行
*   第四级:不旅行

美国国务院根据当地犯罪、恐怖活动、民间动乱、健康和自然灾害/天气的可用公共信息进行评级，这意味着在去你可能不熟悉的地方之前进行检查是一个好办法。

至少，他们每年更新一级和二级警告，每六个月更新三级和四级警告的国家，尽管他们经常根据正在发生的事件更频繁地更新。如果你正在计划一次旅行，这里有你可能要考虑的二级以上国家的最新排名。(考虑到当前的事件，列出的一些国家可能比其他国家更明显。)

当然，咨询系统有一个固有的缺陷，即在一些国家总是存在地区差异。(美国国务院最近取消了地区警报。为此，你应该访问他们的网站，查看一个国家的咨询；你可以在那里找到关于当地风险的信息。)也就是说，你仍然应该仔细考虑任何去目的地的旅行计划，主要建议。

如果你想在国外保持安全，总是报名参加国务院的 [智能旅行者报名计划](https://step.state.gov/) 或 STEP 您将收到最新的旅行提醒，并在紧急情况下拥有资源。

此外，联系目的地当地的美国大使馆或领事馆总是有帮助的，他们也可以为你提供旅行中可能需要的任何信息或资源。

要更仔细地了解每个国家的建议，你应该访问国务院的网站[。](https://travel.state.gov/content/travel/en/traveladvisories/traveladvisories.html/)
