# 如果你担心冠状病毒，注射流感疫苗

> 原文：<https://lifehacker.com/if-youre-worried-about-coronavirus-get-your-flu-shot-1841201798>

冠状病毒可能会成为头条新闻 ，但迄今为止，它对我们大多数人来说并没有那么大的风险。全世界只有极少数病例，如果你没有去过中国武汉，你的咳嗽很可能只是普通感冒。不过，如果你担心，用你紧张的精力去做:注射该死的流感疫苗。

Watch

需要说明的是，流感疫苗并不能预防冠状病毒。那是一种不同的病毒。但是两者的早期症状是相似的:呼吸道症状(如咳嗽)和发烧。如果你注射了流感疫苗，如果你开始咳嗽，你可能会安心一点。流感疫苗并不完美，但它可以预防许多流感病例，并使其他人不那么严重。

这对其他人也有好处:如果你在冠状病毒爆发期间寻求流感护理，你会给医院工作人员带来更多工作，并可能给自己带来额外的担忧。但是你可以用你应该注射的疫苗来预防流感...为什么不呢？

注射流感疫苗还有一个更重要的原因:流感已经杀死，而且几乎肯定会在今年杀死，比新型冠状病毒多得多的人。每年大约有 20，000 到 60，000 名美国人死于流感，其中大多数是非常年轻、非常年老或免疫功能低下的人。他们经常从健康的人那里感染，这些人懒得去注射流感疫苗。因此，如果你认为一种猖獗的、致命的病毒是一件坏事，那就已经接种了流感疫苗。
