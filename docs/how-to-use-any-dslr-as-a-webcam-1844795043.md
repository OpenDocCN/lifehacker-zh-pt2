# 如何使用任何 DSLR 作为网络摄像头

> 原文:[https://life hacker . com/how-to-use-any-dslr-as-a-web cam-1844795043](https://lifehacker.com/how-to-use-any-dslr-as-a-webcam-1844795043)

新冠肺炎继续。为了迫使我们中的许多人在家工作和社交，大多数大相机品牌都发布了官方软件，可以将你的 DSLR 变成一个 T2 网络摄像头，可以与 Zoom、OBS、Facebook Messenger 和其他应用程序一起使用。

我们已经强调了一些这方面的例子，但是有这么多公司发布应用程序，我们决定把所有*目前可用的软件汇总到一个帖子里。我们还将介绍一些可行的第三方替代产品，以防你的品牌或特定型号不受专有技术支持。*

### 安装

在讨论软件之前，我们先来谈谈硬件。无论你使用哪种相机或应用程序，你都需要在开始之前收集一些东西:

*   显然，你需要相机、一台电脑和正确的应用程序，但你还需要一根 USB 线将相机连接到电脑。具体类型取决于您的相机，但设备附带的 USB 电缆效果最佳。如果你没有那个，就去查阅用户手册或者厂商网站，看看你需要什么。
*   你也应该考虑使用三脚架或类似的支架来固定你的相机。
*   如果你在寻找最佳的图像，寻找可用的操作指南，从你的特定相机中获得最佳的视频质量。你可能还会找到镜头和配件推荐。

最后要注意的一件事是:这里列出的所有应用都还在测试阶段。期待最少的额外功能，有限的兼容性(我们会注意任何主要的例外)，以及在某些情况下，错误的性能。我们还注意到应用程序是否有额外的要求，如安装额外的软件或更新相机的固件。

随着这些警告的消失，让我们继续关注软件。

### 索尼(Windows 10)

我们先从索尼新出的 [成像边缘](https://support.d-imaging.sony.co.jp/app/webcam/en/) 网络摄像头 app 说起，既然它 [刚出](https://www.theverge.com/2020/8/20/21377018/sony-camera-webcam-software-imaging-edge-windows-10) 。该软件相当标准，尽管它允许一些相机在连接到 PC 时通过 USB 充电，这是一个不常见的功能。该软件只能在 Windows 10 上运行，因此 Mac 用户需要尝试第三方解决方案(我们将在列表末尾的处推荐一对)。

下载链接和支持的摄像机列表 [在这里](https://support.d-imaging.sony.co.jp/app/webcam/en/download/) 可以找到，设置 [演练可以在](https://support.d-imaging.sony.co.jp/app/webcam/en/instruction/) 这里找到。

### 佳能(Windows 和 Mac)

如果你有一个更新的佳能 EOS 或 PowerShot 相机，那么佳能的 EOS 网络摄像头测试版可能是你最好的选择。该应用于今年早些时候发布，但最近的更新也扩展了对 Mac 的支持。

我们有 [在 Windows](https://lifehacker.com/turn-your-canon-dslr-into-a-webcam-with-this-free-app-1843162662) 上下载安装 app 的指南。Mac 用户会在佳能的网站 上找到类似的 [预排。](https://1.shortstack.com/VChq6Z)

佳能官方的兼容相机列表和 [所有相关的下载链接都在这里](https://www.usa.canon.com/internet/portal/us/home/support/self-help-center/eos-webcam-utility/) 。

### 富士胶片(Windows 和 Mac)

富士胶片的 X 摄像头应用程序 可以免费下载，尽管一些用户可能会发现它的支持有点有限。该工具可用于 Windows 10 和 Mac，但只有少数富士的最新相机兼容。它也只适用于一系列精选的视频应用，但大多数流行的应用都包括在内，如 Zoom、Google Meet、Skype 和 OBS。

完整的 [兼容性信息](https://fujifilm-x.com/en-us/support/compatibility/software/x-webcam/) 、 [下载链接](https://fujifilm-x.com/en-us/support/download/software/x-webcam/) [和设置说明](http://fujifilm-dsc.com/en/manual/webcam/) 在富士胶片的网站上有。

### GoPro (Mac)

GoPro 的应用程序仍处于测试阶段，非常简单，但它至少可以将你的 GoPro HERO8 Black 变成 Mac 上的网络摄像头。Windows 版本正在开发中，但目前还不可用。你还需要手动在你的相机上安装测试版固件，但是我们的向导会带你完成整个安装过程 。

### 尼康(视窗)

尼康的 beta 网络摄像头实用程序是另一个 Windows 专用的应用程序。 [官方下载页面](https://downloadcenter.nikonimglib.com/en/download/sw/176.html) 包括安装步骤，并列出所有支持的相机型号。尼康也有一个很好的实时 [流媒体指南](https://www.nikonusa.com/en/learn-and-explore/live-streaming-with-nikon-cameras.page) ，提供如何最大化相机视频质量的技巧。

### 奥林巴斯(Windows)

奥林巴斯的 OM-D 网络摄像头测试版再次只在 Windows 10 上发布。相机支持仅限于五种 OM-D 相机型号:E-M1、E-M1X、E-M1 马克 2 号、E-M1 马克 III 和 E-M5 马克 2 号。奥林巴斯还选择让它的应用程序只支持视频录制，所以如果你希望能够传输/录制你的声音，你需要使用你电脑的内置麦克风或设置一个外部麦克风。

你可以从奥林巴斯的网站 下载 OM-D 网络摄像头测试版。该页面还向您展示了如何安装和配置该应用程序，并提供了录制音频的提示——尽管我们的 [播客启动指南](https://lifehacker.com/how-to-start-your-own-podcast-1709798447) 也可以在这方面提供帮助。

### 松下(Windows 和 Mac)

松下的 Lumix Tether 已经允许用户将他们的相机连接到他们的 Windows 或 Mac 电脑上进行拴系拍摄(在期间，你的电脑充当外部显示器，在你拍摄照片后立即查看照片)，但该应用程序现在包括流媒体的实时视图模式。它只适用于六个松下相机——DC-GH5、DC-G9、DC-GH5S、DC-S1、DC-S1R 和 DC-S1H——但应该适用于任何流媒体或视频聊天应用。

下载链接和安装演示可在的 [处获得。](https://www.panasonic.com/global/consumer/lumix/lumixtether.html)

### 第三方应用程序(Windows 和 Mac)

到目前为止，几乎每个大牌相机品牌都有自己的网络摄像头应用，但如果你不想依赖第一方软件，你不必依赖第一方软件——特别是考虑到上面列出的应用只支持特定的相机，在某些情况下是 Mac 或 Windows 独有的。

如果上面的应用程序都不适合你的相机，这里有几个第三方选项可能会有用:

*   [**【spark ocam(Windows):**](https://sparkosoft.com/how-to-use-dslr-as-webcam)仅支持佳能和尼康相机，但可以与两家公司的网络摄像头应用程序中不包含的旧型号配合使用。Sparkocam 的免费版本在视频上显示水印，但你可以购买高级版本(1 年许可证 70 美元，终身访问 100 美元)来移除水印。
*   [**【CamTwist】(Mac):**](http://camtwiststudio.com/)CamTwist 可以免费下载，应该可以与各种相机配合使用。不幸的是，它*不能*与 64 位程序一起工作，所以整个应用程序的支持是有限的。尽管如此，如果这是你唯一的选择，还是值得一试。

### 使用捕获卡

为了完整起见，还有一种方法可以将你的 DSLR 用作网络摄像头:使用外部采集卡。这将*可能*工作，即使相机不支持我们上面列出的任何应用程序。一旦你把所有东西都插上电源，你的电脑就会自动检测视频源。

不幸的是，使用采集卡是一个更昂贵的解决方案——你需要卡本身，以及合适的 HDMI 和/或 USB 电缆来连接一切。线缆不会花费太多，但是 [像样的捕捉卡起价在 100 美元](https://www.elgato.com/en/gaming/cam-link-4k) 左右。

也就是说， $100 买一张采集卡比你买一台新 DSLR 只是为了把它当网络摄像头用要便宜得多。也就是说，你可能有一部智能手机。如果你的老 DSLR 不能胜任这项任务，为什么不用它作为你的网络摄像头呢？
