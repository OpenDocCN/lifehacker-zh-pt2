# 通过谈论来提高你对某事的记忆

> 原文：<https://lifehacker.com/improve-your-memory-of-something-by-talking-about-it-1834025854>

记忆力是我们都在不断努力提高的东西之一。随着年龄的增长，这也是那些不断恶化的事情之一。

Watch

如果你想学些东西，帮助你记住的最好方法之一就是把它教给别人。

本周 [*《纽约时报》*刊登了一篇关于如何提高记忆力的文章](https://www.nytimes.com/2019/04/12/smarter-living/how-to-improve-your-memory-even-if-you-cant-find-your-car-keys.html) 。它的一个建议？告诉某人你想记起的事情。在最近的一项研究中，参与者被要求在面前有或没有笔记的情况下讲一堂课。一周后，他们回到测试机构，接受了一次突击测试，测试内容是他们所学的内容。那些被迫用自己的话讲述原始课程的人，而不是那些用笔记讲授相同课程的人。

最重要的一点是:谈论某事有助于你记住它。用你自己的语言表达不仅能帮助你理解一个主题，而且用你自己的语言向别人解释的过程也能帮助你在以后获取信息。

告诉别人并不意味着你必须就这个话题全力以赴。相反，考虑在酒吧向朋友解释这件事，或者在晚餐时向你的伴侣解释。目标只是用你自己的话解释一些东西，以便其他人能够理解。

NYT 文章的另一个好建议？学习结束后，在黑暗中坐几分钟，让大脑放松。研究表明，当你让你的大脑休息，并在刚学完知识后马上处理它时，记忆力会提高 10-30%。
