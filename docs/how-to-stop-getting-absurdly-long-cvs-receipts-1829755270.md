# 如何停止收到冗长的简历收据

> 原文：<https://lifehacker.com/how-to-stop-getting-absurdly-long-cvs-receipts-1829755270>

CVS 以其长得离谱的收据而闻名。如果每次买口香糖都得到相当于树枝的东西让你烦恼，以下是你可以停止这种疯狂行为的方法。

Watch

Vox 记者雷切尔·舒格对这些收据的长度和历史做了一次的深度调查，甚至将它们与其他违规者进行了对比。虽然许多公司不小心烧掉了纸张，但简历收据尤其具有争议性，因为它们已经成为社交媒体最重要的社区之一:投诉者的宠物问题。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1047466215122329600&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1047466215122329600&autosize=1)  [https://lifehacker.com/embed/inset/iframe?id=twitter-1048423303252140032&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1048423303252140032&autosize=1) 

包括吉米·基梅尔先生，他甚至将宝贵的电视时间奉献给了奥巴马总统来解决这个问题。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-Efc8UnyEHzM&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-Efc8UnyEHzM&start=0) 

事实是，并不是所有的简历收据都那么长。要获得这些疯狂的奖励，你需要成为 ExtraCare 奖励计划的成员。收据打印出所有潜在的储蓄和会员可用的优惠券。CVS 完全知道这些抱怨，但他们也知道他们忠诚的客户群，他们声称“继续倾向于纸质选择。”

CVS 已经做了一些重新设计，他们说自从实施以来已经节省了“30 亿英寸”的纸张。然而，如果这还不够好，你可以留下纸张，仍然可以获得奖励——CVS 现在提供 [数字收据](https://www.cvs.com/content/digital-receipt?WT.tsrc=affiliate&cjevent=96771de3d08811e8822502970a1c0e0d&CID=AFF:2617611:7598037:12897382) 。不幸的是，你不能在网上注册；你得让收银员帮你结账。

一位负责的扑克玩家通过电子邮件告诉 Lifehacker，如果你不想在真实的生命线上等待，另一个选择是下载 CVS 应用程序。然后你可以“进入额外护理环境”。在电子邮件首选项下，任何人都可以选择接收电子邮件收据——他们再也不会收到 6 英尺高的纸质收据了(除非他们在结账时要求一张)。”

这个应用程序附带了一些有趣的额外功能。例如，您可以在手机上浏览您的奖励，然后直接将它们发送到您的 ExtraCare 卡上。比把它们揉成一团，然后忘在包底要有效得多。
