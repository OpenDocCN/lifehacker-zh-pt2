# 如何和孩子谈论学校重新开放(或不开放)

> 原文：<https://lifehacker.com/how-to-talk-to-kids-about-schools-reopening-or-not-1844483946>

不知何故，现在已经是 7 月底了，这意味着我们的一些孩子再过*周*就要回到教室了——或者不会。全国各地的学区都在宣布返校计划，要么完全面对面，要么完全在线，要么两者兼而有之。你的孩子想知道 T4 会是什么样子。所以即使你和我一样，还不知道孩子的学年会如何架构，也是时候和他们谈一谈了。

### 问问他们的感受

如果你还没有，现在是和你的孩子坐下来谈谈这个学年可能会是什么样子的好时机。请记住，即使你整个夏天都没有和他们谈过这件事，他们——和你一样——也可能已经仔细考虑过了，想知道并等着听他们是否会回到教室，以及如果他们回去会是什么样子。所以先问他们对这个学年的感觉如何，或者他们的希望和恐惧是什么。

回到朋友身边，他们可能会非常兴奋，但又担心一天中的大部分时间都要戴着面具。经过这么多月的物理距离和呆在家里，他们也可能害怕再次在这么多人面前冒险。谈论他们的感受或担忧很重要，这样你就可以帮助支持他们。

### 解决不确定性

如果到目前为止 2020 年教会了我们一件事，那就是在不确定性中生活几个月的能力——今年秋天也不会有什么不同。孩子们想知道返校会是什么样子，但事实是我们真的不知道。即使你们的地区已经宣布了具体的计划，到现在为止，我们都知道事情会很快发生变化。因此，我们可以告诉我们的孩子我们*知道什么*知道什么，以及到目前为止已经决定了什么，但我们也应该承认，一切都是可以改变的。

谈论家庭和社区中每个人继续保持灵活性的必要性。我们的灵活性和适应能力最终会让我们安全健康。指出他们在灵活性方面已经做得很好的方式，例如在春天切换到虚拟学习，并调整他们与朋友和家人沟通的主要方式。

### 关注在他们的控制范围内是什么

*然而，没有人能生活在完全不确定的环境中；孩子们需要对 T2 的某些事情有一些控制权。谈论他们可以为新学年做些什么准备。他们可以和你一起挑选学习用品。如果他们要去教室，他们可以选择一些有趣图案的新面具(我和我儿子昨晚刚刚订购了几个《我的世界》和神奇宝贝面具)。如果他们呆在家里，他们可以帮你搭建一个学习角，放一张桌子、一把椅子和他们的学习用品。*

*如果他们在学校，谈论管理者、员工和老师将为他们采取的安全预防措施，然后谈论他们可以做些什么来保护自己和他人的安全。经常洗手、戴口罩、咳嗽或打喷嚏时捂着手肘、尽可能保持距离以及避免触摸自己的脸，这些都是孩子们可以经常练习的事情，以保证自己和他人的安全。*

### *把你的负面情绪留给自己*

*理解我们的孩子现在对学校的感受是很好的，但是父母应该小心不要把我们自己的负面情绪发泄到他们身上。这就是儿童心理学家 Nicole Beurkens 博士传达给 [美国广播公司密西根](https://www.wzzm13.com/article/news/health/coronavirus/talking-to-kids-about-schools-reopening/69-4571c256-371c-45af-808a-f2a70be9f17b) 站的信息:*

> *“在所有这些事情中，父母要记住的最重要的事情之一是，我们作为成年人和父母如何对待它，将对我们的孩子对此的感受产生巨大的影响。所以，不管我们自己的个人感受如何——我们的压力可能越来越大，我们可能会感到焦虑或沮丧——我们都希望真正避免在孩子面前谈论学校和计划的负面内容。”*

*你可能会认为你的学校在处理返校问题上很鲁莽，你可能会认为他们反应过度，或者你可能会认为学区是由一群白痴管理的——你可能是对的。但是把咆哮留到孩子们睡觉的时候，你可以向你的伴侣或朋友发泄。*
