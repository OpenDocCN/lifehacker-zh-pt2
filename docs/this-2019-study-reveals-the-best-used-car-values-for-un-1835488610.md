尽管新车与二手车的争论将继续在评论区激烈进行，直到我们都化为灰烬(看看我的帖子，叶强大，绝望)，但我总是至少在专业背景下逐案处理这个难题。通常，如果一个买家正在购买一辆价格在 25，000 美元至 30，000 美元之间的二手非豪华车，一旦你考虑到新车型的折扣，二手车就不会有太大的价值。然而，低于 20，000 美元的范围是二手车真正发光的地方。

虽然现代伊兰特(Hyundai Elantra)和丰田卡罗拉(Toyota Corolla)等汽车的基础型号如今配备相当不错，但预算有限的人不必牺牲空间、动力或设施，如果他们愿意接受一些折旧曲线陡峭的二手车的话。

根据*iSeeCars.com 的说法，*这些车型是最划算的，因为这些车的价值大约是三年前原价的一半。

榜单上占据主导地位的是大多来自美国和韩国品牌的中型和大型轿车。它们也是非常好的车，只是从来都不太受欢迎。还有大众和菲亚特，从可靠性的角度来看，这可能有点像赌博。

对于新车购买者来说，这也是一些有用的信息。他们可能希望避免上述车型，因为当汽车以旧换新时，他们可能会感到震惊，但二手车买家可以花不多的钱获得舒适的乘坐体验。
