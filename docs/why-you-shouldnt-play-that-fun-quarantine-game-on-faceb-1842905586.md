# 为什么你不应该在脸书玩“有趣的隔离游戏”

> 原文：<https://lifehacker.com/why-you-shouldnt-play-that-fun-quarantine-game-on-faceb-1842905586>

如果你最近在脸书，你可能会看到大量的【有趣】游戏，建议你告诉每个人你住过的所有街道的名字，或者你拥有的所有汽车的名字，或者你出生那年流行的歌曲。或者建议你上传高中照片的那个怎么样？

Watch

从表面上看，所有这些建议似乎都是你可以与朋友分享的趣事。事实上，所有这些帖子都可能被黑客用来进入你的私人账户。

当然，发布它的朋友可能不是黑客。他们可能看到了另一个朋友发的帖子，他们感到无聊，决定也参与进来。

然而，如果你留心的话，你会发现所有这些有趣游戏的答案也是你在网站上验证身份以重置密码时可能输入的内容。当你在互联网上发布这些答案时，你就让那些不是你朋友的人很容易通过简单的互联网搜索找到它们，并利用它们来对付你。

《出生时流行的歌》可以给一个黑客过生日。你给那张毕业照加的标题是“1979 年北方高中老鹰队！” 告诉别人你哪年毕业，你学校的吉祥物是什么。像你长大的街道和你第一辆车的品牌和型号这样的事情是不言自明的。

考虑一下你发布的内容，如果答案是你可能需要的安全问题的一部分，那么**不要发布**。

如果你绝对不能阻止自己参与(同样，你不应该)，确保你只与你的朋友分享这些答案，而不是朋友的朋友或整个世界。你可以点击帖子上时间戳旁边的图标，将帖子的隐私改为“朋友”

当然，最好的办法就是选择不看这些帖子，或者在你看到的任何一个帖子的评论中添加一个链接。虽然现在足够天真，但当你试图找回你的帐户时，这个“有趣的隔离游戏”将不再有趣。
