# 惠普的 YouTube 频道现在是技术支持和修补的宝库

> 原文：<https://lifehacker.com/hps-youtube-channel-is-now-a-treasure-trove-for-tech-su-1836820670>

要么是某个实习生度过了最无聊的硅谷夏天，要么是惠普的 YouTube 团队已经为权力而疯狂。不管是什么原因，该公司最近在 YouTube 上掀起了一场，上传了*数百个*详细的技术支持视频，涵盖了拆卸、修理以及以其他方式搞乱其众多笔记本电脑、台式机和其他产品的无数方法。

Watch

事实上，我甚至可以说，如果你对惠普笔记本电脑有问题，或者你想要一点指导，告诉你如何为普通用户做一些更激烈的事情(比如更换硬盘或内存)，你应该完全忽略惠普的网站，只需访问它的 YouTube 频道。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-nC-ImDwEV7o&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-nC-ImDwEV7o&start=0) 

如果你是一个比较认真的修理者，惠普的 YouTube 频道是一个金矿，里面有关于如何修理你拥有的任何惠普产品的大多数重要内部组件的教学视频。想要更换 HP EliteBook x360 1040 G5 的整个显示屏吗？简单——嗯，在视频中看起来很简单，至少:

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-TrfQHhkdkHw&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-TrfQHhkdkHw&start=0) 

不可否认，惠普的视频确实掩盖了一些重要的步骤，比如告诉你“移除主板”，而没有实际显示那个人正在攻击将主板连接到系统机箱的许多螺钉。也就是说，我很欣赏视频的简洁:你以一种干巴巴的、几乎是像 T1 一样的方式获得了基本事实。即使你没有拆卸昂贵物品的丰富经验，你(可能)也不会感到很失落——假设你已经以某种方式获得了你需要的任何替换零件，那么在惠普的帮助下，你可能会省去昂贵的维修店之行。

与其费力地浏览惠普的许多技术视频，不如确保你使用的是 YouTube 方便的频道搜索功能。我也忘了，但是很容易找到。在任何频道的页面上查找“关于”右侧的放大镜。点击它，键入你想查找的任何惠普产品的名称，你就能很快看到惠普是否上传了任何关于如何拆解它的视频。

我不建议订阅这个频道，因为如果惠普保持上传速度，它会淹没你的 YouTube。相反，把它做为书签，这样你就可以随时记得查阅它，如果你想开始摆弄你的笔记本电脑的部件的话。
