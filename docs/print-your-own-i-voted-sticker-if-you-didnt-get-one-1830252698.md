# 如果你没有“我投票了”的贴纸，那就印上自己的吧

> 原文:[https://life hacker . com/print-your-own-I-voted-sticker-if-you-not-get-one-1830252698](https://lifehacker.com/print-your-own-i-voted-sticker-if-you-didnt-get-one-1830252698)

在 [可能发生在](https://lifehacker.com/the-most-common-voter-suppression-scams-1830095243?rev=1541007456527#_ga=2.175117154.1552136910.1541451956-3846207152.1521480874) 投票站的最微小的不公正事件中，我所在地区的投票站不会发放那些小小的“我投了票”的贴纸，而今天许多人都自豪地佩戴着这些贴纸。但是没关系——不管你是提前投票、邮寄缺席选票，还是你的社区投票工作人员用完了贴纸，以下是如何打印你自己的。

Watch

Slate 有一张 [贴纸图片](https://slate.com/culture/2018/11/i-voted-stickers-2018-midterm-elections-print-at-home.html) ，可以下载打印。如果你喜欢，可以把它们贴在贴纸上，或者在必要时用普通的纸和胶带凑合着用。(Slate 每年都做这些——抓住 [他们 2012 年的](http://www.slate.com/blogs/browbeat/2012/11/06/no_i_voted_stickers_at_your_polling_station_print_your_own_here.html) 来个复古氛围。)

今年的时尚选项包括“我提前投票”，“泰勒·斯威夫特告诉我投票，所以我投票了”，“我投票是因为有一天我想回去关心那些愚蠢的狗屎。”或者，从 [教师网站](https://www.teacherspayteachers.com/Product/I-Voted-Printable-Election-Stickers-for-Mailing-Labels-402078) 下载一张普通的老式美国国旗“我投票了”贴纸。另外，它们符合标准的邮寄标签模板。

如果没有找到适合自己的消息，就抓一个类似 [这种](https://www.stickergiant.com/templates) 的空白贴纸模板，自己写标语。
