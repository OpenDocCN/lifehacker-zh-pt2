# 只用一个引体向上的杠铃锻炼你的核心力量

> 原文：<https://lifehacker.com/work-your-core-with-nothing-but-a-pull-up-bar-1832825036>

欢迎回来！我希望你们都喜欢这个月 全力以赴，无所事事。今天，我们来谈谈你可以在引体向上或一套体操环上做的运动。



在这些练习中，你主要是锻炼腹肌和臀部屈肌(连接躯干和大腿的腿部肌肉)。这里有一些我最近一直在尝试的，你也可以。

### 悬膝或抬腿

悬挂在横杆上，抬起你的腿，这样你的大腿至少与地板平行。如果你保持膝盖弯曲，你会看起来像坐在一张想象的椅子上。这是更简单的版本。如果不能一路抬高膝盖，就尽量往高处走。

让你的腿伸直会更难。你能把腿抬起来让你看起来像字母 L 吗？没有吗？我也没有。中间的选择是抬起弯曲的膝盖，然后伸直双腿，同时放下。或者，抬起弯曲的膝盖，将腿向外伸直，然后用弯曲的膝盖再次放下。

### 膝部圆圈或挡风玻璃雨刷

同样，如果你是一只野兽，你可以用直腿做这些，否则弯曲膝盖。用你的膝盖(或脚趾)在空中画一个巨大的圆圈，把它们举到你的左边，然后尽可能高，然后举到你的右边，然后放下。

对于挡风玻璃雨刷版本，跳过圆圈的底部，来回就可以了。

### 中空货舱

好了，现在我们进入稍微容易一点的领域。你可以在任何地方练习，包括躺在地板上，但悬挂着也是的乐趣。你的目标是收缩腹肌，使骨盆和胸腔向对方倾斜。你的躯干会呈现一个轻微的“c”形。

### 脚趾到横杆和膝盖到肘部

印象最深的吊 ab 练习是我根本不会做的一个: [脚尖对杠](https://www.youtube.com/watch?v=yCqJyD6wwy0) 。从字面上看，你只需转动臀部，抬起你的直腿，这样你的脚趾就能碰到你悬挂的横杆。

如果你做不到，试试膝盖弯曲的方式:膝盖对肘部。

这两样我都做不了。如果可以，我向你致敬。
