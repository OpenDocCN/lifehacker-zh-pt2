# 问咸味女服务员:我该给炭火盆厨师小费吗？

> 原文：<https://thetakeout.com/should-i-tip-the-hibachi-chef-teppanyaki-restaurant-1830721885>

> 嗨，咸味，我的问题很简单:我们应该给炭火盆厨师小费吗？如果有，多少？服务员和厨师会平分小费吗(我听说他们会，所以当我围着炭火盆吃饭时，我会给额外的小费)...你能为我澄清这个灰色地带吗？
> 
> 谢谢，艾琳

亲爱的艾琳:

你做到了。你难倒了咸咸。

我看了你的问题，眨了两下眼睛，整了整裤袜，还是不知道答案。该死的。我从来没有在那些 Benihana 地方工作过，所以我没有任何这方面的第一手经验。但我打了几个电话。



我接触到的两家炭火/铁板烧餐馆说，厨师是服务员、厨师和调酒师分享小费的一部分。其中一家餐馆还表示，除了留在账单上的小费之外，人们还会直接用现金给厨师小费，这很常见。(我猜他们对那天的洋葱火山表演印象特别深刻。)

如果服务员和厨师共用小费，我认为你可以留下标准的 18-20 %,原因如下:在我去过的炭火餐厅，地板上的服务员没有标准餐厅那么多，因为厨师基本上既给你上菜又做饭。大多数情况下，服务员从酒吧拿饮料并处理账单。因此，当服务员、炭火盆厨师和调酒师平分小费时，应该是相当公平的。

如果你想在“晚餐和表演”方面投入一些额外的钱，我永远不会告诉你不要。但是从一些火盆连锁店的菜单价格来看，他们已经把这个因素考虑进去了。
