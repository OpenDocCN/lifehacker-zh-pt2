# “驱逐暂停”到底是什么意思？

> 原文：<https://lifehacker.com/what-does-an-eviction-pause-really-mean-1842496524>

全国各州和城市决定在冠状病毒疫情期间暂停驱逐和公用事业关闭。但如果你面临被驱逐的风险，暂缓令可能只是一种临时保护。

Alieza Durana 说，各州和各市提出的住宿条件存在一些问题。杜兰娜在普林斯顿大学的驱逐实验室 工作，该实验室不断更新暂停驱逐和关闭的地方名单 [。](https://lifehacker.com/where-to-find-coronavirus-eviction-and-utility-shutoff-1842470826)

“在大多数州，用来阻止驱逐的机制留下了漏洞和/或没有 100%解决问题，”她说。

有几种情况会影响正在被驱逐的人:

*   在法院已经停止驱逐听证会的地方，案件可能会在几周内恢复。
*   一些地方的房东现在仍然可以提出驱逐申请，在春季或夏季的晚些时候进行审理。
*   已经发布暂停令的州可能会让地方政府来执行，这可能会让寻找信息的租户感到困惑。
*   房东仍然可以迫使房客搬走，或者通过忽视他们的出租单元进行报复。

政策之间的差异会让你很难知道你所在的地区正在发生什么——以及什么时候你需要做什么。例如，Durana 说在加州，租户必须证明他们因为新冠肺炎而无法支付。

“即使有暂停令，压力仍然存在，”Durana 说，他解释说，在大多数地方，暂停令解除后，租户和房东的下一步是什么还不清楚。

当你看到像义大利 这样的国家 [已经停止所有的抵押贷款支付时，整件事会令人沮丧。为什么不在这里实施类似的措施，并将它们推广到租房者身上呢？杜兰娜解释说，这部分是由于美国政府的分权制度，因为总统还没有在联邦层面采取行动颁布全面禁止驱逐的禁令，所以有时在州一级会出现谁有权暂停这一进程的问题。](https://thebasispoint.com/america-wont-get-coronavirus-mortgage-relief-like-italy-uk-but-will-get-delayed-tax-filing/)

如果你在冠状病毒爆发期间或之后有被驱逐的风险，你可以访问 [Just Shelter](https://justshelter.org/) 来寻找你所在州的住房资源。(Just Shelter 由《被驱逐者 》的作者、 [*和驱逐实验室的首席研究员马修·德斯蒙德(Matthew Desmond)共同创立)。*](https://www.evictedbook.com/) 

你也可以通过法律援助协会在你所在的地区寻找低成本的法律帮助。LawHelp.org 有一个按州分类的目录，帮助你查找资源。
