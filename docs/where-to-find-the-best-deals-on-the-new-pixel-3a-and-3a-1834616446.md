## Pixel 3a 和 3a XL 规格和定价

Pixel 3a 和 3a XL 是去年推出的高端 Pixel 3 和 3 XL 智能手机的更精简、更实惠的版本。3a 和 3a XL 的规格相似，包括:

*   **内存:** 4GB 内存
*   **内部存储:** 64 GB
*   **芯片组**高通骁龙 670
*   **OS:** [基于谷歌的 Android OS](https://lifehacker.com/how-to-turn-any-android-smartphone-into-a-google-pixel-1833921883)
*   **后置摄像头:**220 万像素，f/1.8 镜头
*   **前置摄像头:**800 万像素
*   **集成耳机插孔**
*   **电池:**3000 mah(像素 3a)；3700 毫安时(像素 3a XL)
*   **屏幕尺寸:** 5.6 英寸，2220x1080 分辨率(像素 3a)；6 英寸，2160x1080 分辨率(像素 3a XL)
*   **价格:**像素 3a:400 美元；像素 3a XL:480 美元

虽然 3a 版本的功能不如高端 Pixel 3 手机——由于处理器较慢，没有防水功能，也没有无线充电功能——但它们有一个共同的重要方面:它们的摄像头。这些是一些最好的智能手机相机 ，具有当前的功能，如夜视、人像模式和顶部拍摄。他们也会收到即将到来的延时模式。
