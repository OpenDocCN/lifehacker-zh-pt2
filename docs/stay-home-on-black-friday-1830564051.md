# 黑色星期五呆在家里

> 原文:[https://lifehacker.com/stay-home-on-black-friday-1830564051](https://lifehacker.com/stay-home-on-black-friday-1830564051)

黑色星期五的狂欢可能比在消费者“假日”购物更受欢迎。我们现在都知道，砸门广告并不像宣传的那样是真正的大优惠，打折的电视不值得把人踩在脚下(至少，我希望我们都知道这一点)。

Watch

但是黑色星期五 的 [【消亡】，这个已经写了很多年的话题，也被过分夸大了。我们继续在每年的长周末花费破纪录的金额。去年，](https://twocents.lifehacker.com/black-friday-is-dying-and-that-s-a-good-thing-1743114275) [美国人仅在 24 小时内就在网上花费了 50 亿美元。](https://qz.com/1456840/black-friday-by-the-numbers-savings-store-openings-and-more/)

那么，销售真的值得吗？那得看情况。 [专家称](https://www.nytimes.com/2018/11/13/smarter-living/how-to-tell-if-black-friday-deals-are-worth-buying.html) 小家电和智能家居小工具将于今年上市，研究指出 [合法旅游优惠在“旅游星期二”](https://twocents.lifehacker.com/check-travel-prices-before-black-friday-1830537278) 消费主义周最新加入。

“黑色星期五”、“网络星期一”之类的节日不仅为购物者提供了优惠，还只是引诱我们尽早开始假日购物的一种方式。零售商已经在 [大肆宣传他们的销售](https://lifehacker.com/the-psychological-reasons-we-shop-like-crazy-on-black-f-1789101905) 好几个星期了，但这并不意味着他们真的值得你花钱。例如，Loft、the Gap 等零售商打六折。，与其说是一年一次的抢断，不如说是 [双月事件在这一点上](https://www.racked.com/2016/4/13/11371544/love-letter-to-loft-ann-taylor) 。研究指出 [大部分假期销售](https://www.thebalance.com/what-is-black-friday-3305710) 发生在感恩节前一周*。而正如我们之前 [所涵盖的](https://twocents.lifehacker.com/when-a-sale-isnt-actually-worth-it-1830094661) ，很多销售都是骗人的。*

> 非营利倡导组织消费者支票簿(Consumers ' Checkbook)对某些商品的价格进行了独立分析，发现 JCPenney、Kmart、Macy's 和 Sears 等商店的商品几乎每周都在“打折”,这意味着它们从未以“原价”上市。销售价格或多或少是“真实”价格。

也就是说，如果你已经想好了一些东西或经历，或者如果你喜欢感恩节后和家人一起购物的仪式，那就想尽一切办法去做(如果你真的想要最划算的交易，就使用价格跟踪工具，比如 [骆驼](https://camelcamelcamel.com/) 或 [看不见的手](http://www.getinvisiblehand.com/) )。与其说这是指令性的，不如说是反思性的(如果我说我没有从我最喜欢的商店里挑选过一个包，那我就是在撒谎)。正如杂色傻瓜写的那样，在过去的几年里，零售商有正当的理由在这个时候增加销售，尽管现在在整个假日季节都或多或少有相同的销售(而且往往更早)。会有更多的“交易”总是有的。

那么，这个周末你会呆在家里，还是去逛商店？你有你正在寻找的特价商品吗？
