# 使用 Chrome 扩展将你的推文转化为给政治家的信息

> 原文：<https://lifehacker.com/turn-your-tweets-into-messages-to-politicians-with-this-1832897027>

现在，我们比以往任何时候都更有在推特和脸书上发泄政治情绪的习惯。



想把所有的怨恨转化成更有用的东西吗？这是你表达愤怒的机会(如果你谨慎使用的话)。
