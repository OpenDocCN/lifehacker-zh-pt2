# 使用 Chrome 扩展将你的推文转化为给政治家的信息

> 原文:[https://life hacker . com/turn-your-tweets-into-messages-to-politics-with this-1832897027](https://lifehacker.com/turn-your-tweets-into-messages-to-politicians-with-this-1832897027)

现在，我们比以往任何时候都更有在推特和脸书上发泄政治情绪的习惯。

想把所有的怨恨转化成更有用的东西吗？这是你表达愤怒的机会(如果你谨慎使用的话)。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-803397508349198337&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-803397508349198337&autosize=1) 

如果你想正式表达你的不满，现在你可以下载免费的 Chrome 扩展 ePluribus，它可以将任何刻薄的推文、脸书消息，甚至新闻故事转换成发给当地国会议员的电子邮件。

它是如何工作的？在 eplusibus '[网站](https://www.epluribus.io/) 下载扩展后，你必须创建一个“公民 ID”，或登录，它使用地址信息来找到你当地的代表(如果你担心隐私设置，他们不会给出这些信息)。

接下来，公司会给你发一张有验证码的实体卡，来验证你的地址。利用这一点，ePluribus 将确保您是一位政治家所在城市/州的经过验证的选民，而不是垃圾邮件发送者，每当一位政治家收到您的消息时，就会显示垃圾邮件发送者(但是，如果您热衷于开始，您仍然可以发送消息而无需验证您是选民)。

一旦你设置好了，你就可以点击 ePluribus 图标，或者直接去 Twitter 或脸书写你的信息。在每条消息的下方，您可以选择您当地的代表(以及川普和迈克·彭斯)作为收件人。

该网站还支持许多新闻网站，如《纽约时报》和《NPR 》,运营方式略有不同。读完一个故事后，你会发现向你的代表发送消息的选项，这看起来很像一个评论区。在这里，你还可以选择匿名发送消息，不像你的社交 feed，你的消息显然是公开的。如果收件人在寻找你的评论的上下文，他们还会收到一个故事的链接。

这与直接向您当地的销售代表发送信息有何不同？首先，你是一个合格的选民。你的代表(或任何阅读你信息的人)会知道他们实际上能为你个人带来改变，你更有可能得到回复。

这比发一条推文到深渊里更进了一大步。我们之前 [提到](https://lifehacker.com/the-best-ways-to-contact-your-congress-people-from-a-f-1788990839) 众议员不太可能在推特和其他社交媒体上查看你的信息(尽管，正如《纽约时报》报道的那样，打电话可能仍然是联系当地国会议员的最佳方式)。

那么给你的州议员发电子邮件有什么礼节呢？首先，不要像你的私人阅读器一样给他们发垃圾邮件。他们也有工作。如果你想让别人有效地听到你的声音，伸出手时要使用普通的礼貌。要知道，大多数时候，是政客的幕僚在阅读你的信息，所以没有必要发一封粗鲁的邮件。言简意赅，如果可能的话，描述一下这个特殊的故事是如何影响你个人的。

正如《纽约时报》 报道的那样，也不要害怕谈论已经在社交媒体上流传的话题(或者感觉有人已经替你做了这件事)。力量来自数字，每封邮件都可以带来改变。
