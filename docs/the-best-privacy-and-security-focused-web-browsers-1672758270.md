无论你的首选浏览器是 Chrome 还是 Firefox，你都有几个选项来帮助你更安全地浏览——假设，也就是说，你准备放弃其他人都在使用的版本，尝试一些新的东西。这里有一些值得尝试的浏览器。

### [**勇敢**](https://brave.com/)T5】

这款适用于 Windows、macOS、iOS 和 Android 的独立浏览器基于 Chromium——这意味着如果你是 Chrome 的粉丝，你会感觉非常熟悉。首先，迁移到 Brave 会让你基本上脱离谷歌的生态系统，所以你可以松一口气，因为你知道你不会通过你的登录帐户向谷歌发送大量数据。

更重要的是，Brave 的默认设置非常适合那些想要更多数据隐私，但又不想实际*学习*如何获取数据的人——或者想出他们需要安装的所有扩展来减少他们最喜欢的浏览器中的跟踪。默认情况下，Brave 会阻止所有广告和第三方跟踪，其内置的 [HTTPS Everywhere](https://www.eff.org/https-everywhere) 功能可确保您始终连接到您正在浏览的网站的最安全版本( [，如果适用的话还有](https://www.eff.org/https-everywhere/faq#when-does-https-everywhere-protect-me-when-does-it-not-protect-me) )。

如果你想奖励你最喜欢的网站的创建者的工作，也欢迎你选择(它不会跟踪你或收集你的信息)。虽然对于一个注重隐私的浏览器来说，也有自己的广告设置可能看起来很奇怪，但这一举动似乎是一个合理的妥协，在隐私和便利与内容创作者面临的财务现实之间取得了平衡。

### [**Tor 浏览器**](https://www.torproject.org/projects/torbrowser.html.en)T5】

如果你想在你访问的网站上保护你的匿名和隐私，并与恶意跟踪 cookies 和广告做斗争，Tor 是一个很好的选择。对于那些不熟悉的人来说，Tor [通过一系列中继](https://lifehacker.com/what-is-tor-and-should-i-use-it-1527891029) 来路由你的流量，这些中继是为了尽可能匿名地保持你的真实身份和计算机。它是 [不完美](https://lifehacker.com/get-tor-s-browser-update-to-stop-leaking-your-real-ip-a-1820190316) ，它 [当然也有它的缺点](http://lifehacker.com/how-can-i-stay-anonymous-with-tor-1498876762) ，但是如果匿名是你的最终目标，Tor 浏览器是一个很好的选择。

Tor 浏览器基于 [Firefox Quantum](https://lifehacker.com/why-you-should-switch-from-google-chrome-to-firefox-1821879163) ，是开源的，预配置为访问 Tor 网络。绝大多数内置插件和服务已经被禁用或删除，重要的是你要让它们保持原样，否则你想要保密的数据可能会泄露到你访问的网站。否则，Tor 会阻止所有第三方追踪器，在你关闭浏览器时清除 cookies 和历史记录，并试图让所有浏览器用户看起来都一样，以阻止任何类型的广告相关的 [指纹](https://lifehacker.com/how-to-reclaim-your-digital-privacy-from-online-trackin-1820878546) 。

Tor 可用于 Windows、macOS、iOS 和 Android，当您需要在不受信任的系统上安全地访问 web 时，您甚至可以获得便携式版本。也就是说，Tor 是为物理和数字匿名而设计的，而不是安全和加密。您在使用它时所做的事情可能会暴露您的匿名身份(发送电子邮件、登录网络服务等)，虽然 Tor 网络内部的通信是加密的，但是一旦您离开网络，您的数据就是明文的(如果没有以其他方式加密的话)。)

### [**火狐量子**](https://www.mozilla.org/en-US/firefox/)T5】

不错的老火狐。如果你使用网络有一段合理的时间，你肯定听说过 Mozilla 的 Chrome 大竞争对手——尽管，奇怪的是，目前是谷歌自己为这个非营利组织的 [和](https://gizmodo.com/new-firefox-defaults-to-google-ends-friendship-with-ya-1820469923) 提供了大量资金。

除了历史教训之外，Firefox Quantum 还提供了许多直接内置在浏览器中的隐私增强功能。默认情况下，浏览器的增强跟踪保护功能会根据 [Disconnect 的列表](https://disconnect.me/trackerprotection) 阻止第三方跟踪 cookies，浏览器现在附带了 [反指纹功能](https://blog.mozilla.org/firefox/how-to-block-fingerprinting-with-firefox/) ，使广告公司更难围绕你的习惯建立档案。还有一个 [反密码挖掘](https://blog.mozilla.org/futurereleases/2019/04/09/protections-against-fingerprinting-and-cryptocurrency-mining-available-in-firefox-nightly-and-beta/) 功能，以防你担心你访问的网站正在获取你系统的资源以赚取微薄的利润。

默认情况下，浏览器不会屏蔽广告，但是 Mozilla 推荐 Firefox 扩展,“T2”,“T3”可以帮助你实现这一点。阅读 Mozilla 的数据收集政策，你也会受到启发——使用它推荐的技术来关闭 Firefox 浏览器中的任何和所有数据收集，尽管它可能是无辜的。如果你是脸书的忠实粉丝，Mozilla 非常乐意通过一个方便的 Firefox 扩展来帮助你减少社交网络如何追踪你在做什么。

对于移动浏览，我们建议查看一下 [Firefox Focus](https://lifehacker.com/how-to-block-tracking-cookies-with-firefox-focus-1831598569) ，这是一个更加精简和以隐私为中心的 Firefox 版本。

### [**【duck duck go】**](https://duckduckgo.com/app)

这款仅支持 Android 和 iOS 的浏览器 [包含隐私功能](https://lifehacker.com/how-to-maximize-your-browsing-privacy-using-duckduckgo-1830659232) :拦截广告、杀死第三方追踪者、尽可能强制更安全的 HTTPS 连接，以及用特殊字母等级评估你访问的每个网站的隐私特征。如果你看到一个大大的“F”，可能是时候去一个更关心数据收集的网站了。

我们尤其喜欢浏览器底部中间的小火焰图标。点击它，你会被询问是否要清除所有的标签和数据。再次点击该选项进行确认，你在手机浏览器上所做的一切都会被删除。这是一个令人难以置信的快速和简单的方法来清除你的痕迹，并确保浏览器没有存储任何你不想要的数据。如果你觉得很懒，你也可以设置浏览器在每次重启应用时自动清除所有数据。

虽然你可以通过扩展为你的 [Chrome](https://chrome.google.com/webstore/detail/duckduckgo-privacy-essent/bkdgflcldnnnapblkhphbgpggdiikppg?hl=en) 或 [Firefox](https://addons.mozilla.org/en-US/firefox/addon/duckduckgo-for-firefox/) 浏览器添加类似的反跟踪功能，但这并不等同于拥有一个成熟的浏览器。

### **或者:调整 Chrome 或 Firefox 以保护隐私**

如果你不太热衷于下载一个全新的浏览器，将你所有的书签、扩展和其他数据转移到上面，从头开始，不要担心——你总是可以调整 Firefox 或 Chrome 成为你想要的浏览器。例如:

*   告诉你的首选浏览器 [以匿名或私密模式](http://lifehacker.com/start-any-browser-in-private-browsing-mode-5530828)启动。
*   启用 [您的浏览器可能拥有的任何附加功能](https://lifehacker.com/how-to-enable-content-blocking-filters-and-privacy-exte-1835242670) ，但默认情况下不会运行，以确保您的数据更加安全。
*   将默认搜索引擎改为注重隐私的网站，比如 [DuckDuckGo](http://duckduckgo.com/) 或 [Startpage](https://www.startpage.com/en/) 。
*   安装 [扩展](https://lifehacker.com/how-to-reclaim-your-digital-privacy-from-online-trackin-1820878546) 可以帮助你保护自己的隐私。
*   使用广告拦截器，如 [广告拦截加](https://lifehacker.com/the-best-browser-extensions-that-protect-your-privacy-479408034) 。
*   当您关闭浏览器时，自动清除您的缓存和历史记录。
*   到处使用 HTTPS/SSL[y](https://lifehacker.com/https-everywhere-updates-to-keep-you-secure-on-thousand-1695541941)[ou can](http://lifehacker.com/why-should-i-care-about-https-on-facebook-or-other-web-5745086)。
*   [用 VPN 加密你所有的流量](https://lifehacker.com/how-to-find-a-trustworthy-vpn-1833045522) 。
