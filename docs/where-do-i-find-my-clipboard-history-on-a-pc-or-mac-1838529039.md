# 在 PC 或 Mac 上哪里可以找到我的剪贴板历史记录？

> 原文：<https://lifehacker.com/where-do-i-find-my-clipboard-history-on-a-pc-or-mac-1838529039>

欢迎回到 Lifehacker 的技术建议系列的另一部分， [**技术 911**](https://lifehacker.com/c/tech-911) 。如果你有一个特别令人困惑的问题，你可以写信提出你的棘手问题，我们会给你一些难回答的问题、一点幽默和一点挠头。



本周，生活黑客读者 **Sunny** 写道:

> *“很多时候，我把一些东西复制到我的剪贴板上，在粘贴之前，我不小心通过复制其他东西覆盖了它。如何找到我复制到剪贴板上的内容的“历史”？有没有什么 app 可以帮忙的？”*

你忘了说明你用的是什么操作系统，Sunny，不过没关系！我将介绍 Windows 和 macOS，这样每个人都可以知道如何恢复他们不小心在剪贴板上覆盖的内容。(是的，这也是我的一个问题，所以我很高兴我们在讨论这个问题。)

首先，我假设你使用的是 Windows 10(如果你是 Windows 用户)，你运行的是最新版本的操作系统。打开**设置**应用，点击**系统**，向下滚动左侧边栏，点击剪贴板。我已经启用了我的剪贴板历史记录，所以你的屏幕应该看起来有点像这样——但是如果它没有被选中，你将想要启用剪贴板历史记录。

向下滚动一点，您也可以使用此屏幕来清除您的剪贴板历史:

现在，每当你按下 Windows 键+ V(而不是 CTRL + V)，你的剪贴板历史会在你的屏幕上弹出。单击列表上的任何内容，将它粘贴到光标所在的位置。就这么简单。

### 在 macOS 上启用剪贴板历史记录

你需要一个第三方应用程序才能在你的 Mac 上启用真正的剪贴板历史记录——内置的剪贴板只存储一个项目。您可以探索的一些选项包括 [**CopyClip**](https://apps.apple.com/us/app/copyclip-clipboard-history/id595191960?mt=12) ， [**Paste**](https://pasteapp.me) (这是超级贵的)，[**Alfred 4**](https://www.alfredapp.com/)，[**PasteBot**](https://tapbots.com/pastebot/)，以及开源的[**Clipy**](https://github.com/Clipy/Clipy?ref=producthunt)。

它们都很容易使用，并且它们应该给你一个简单的方法来保存你复制的东西(如果没有其他的东西的话，只能保存很短的时间)。我可能会首先选择免费或便宜的应用程序——尤其是那些带有某种试用版的应用程序——这样你就可以在付费之前感觉它是否符合你的需求，如果你决定走这条路的话。真的，这应该是操作系统本身的一个主要内容，如果苹果在某个时候将这个功能转储到 macOS 的未来版本中，我不会感到惊讶。
