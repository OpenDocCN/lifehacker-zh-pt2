# 2018 年最佳三种成分鸡尾酒

> 原文：<https://lifehacker.com/the-best-3-ingredient-cocktails-of-2018-1831369062>

恭喜每一个坚持到 2018 年底的人，我敢说你值得喝一杯(或者七杯)。如果你还在绞尽脑汁，试图找出如何庆祝今年被遗忘，让自己平静下来，凝视我们今年制作和享用的许多美味的三种成分的鸡尾酒。有很多选择。



### [**麦琪·霍夫曼的边车名叫**](https://skillet.lifehacker.com/3-ingredient-happy-hour-a-sidecar-named-desire-1823807036#_ga=2.209566930.1352987868.1545790970-1157707288.1539189461)

首先由新奥尔良 SoBou 的劳拉·贝卢奇开发，这种饮料是一种多汁的水果味鸡尾酒，我们熟知的，是非常 riff- 的能力。好的，加糖的杏果酱是让这个婴儿唱歌的关键，所以跳过便宜的东西，犒劳自己。

### [**我的血腥情人节**](https://skillet.lifehacker.com/3-ingredient-happy-hour-my-bloody-valentine-1822853432)T5】

如果那罐醉醺醺的柑橘对你来说看起来很熟悉，那是因为你跟着 [*和克莱尔*](https://lifehacker.com/c/eating-trash-with-claire) 一起吃垃圾，你知道里面装着 [浸了血橙的苦艾酒](https://skillet.lifehacker.com/preserve-fruit-in-oxidized-vermouth-1822599722) 。你可以诚实地直接啜饮罐子里的东西，并且完全快乐——我是这样做的，我也是——但是我认为情人节需要一些更强烈的东西。

### [**枫老土**](https://skillet.lifehacker.com/3-ingredient-happy-hour-a-maple-old-fashioned-1823051491)T5】

我们将使用一小块枫糖浆，而不是白糖块——反正我也没有。如果你仔细想想，枫糖浆是一种现成的鸡尾酒甜味剂，完全不需要你做任何准备工作或煮沸，比蔗糖做的东西更有深度。

### [](https://skillet.lifehacker.com/three-ingredient-happy-hour-a-muddle-free-faux-jito-1824028231)

**杰森的饮料没有使用一大堆薄荷叶和糖，而是从布兰卡·门塔(Branca Menta)那里获得了甜味和清爽的薄荷味，布兰卡·门塔是 Fernet 更容易接近的表亲。杰森在他的鸡尾酒里加了苏打水，以获得更真实的莫吉托体验，但那会让我们有四种成分，所以我们稍微调整了比例，以制作一种介于莫吉托和代基里之间的饮料，它是*美味的*。** 

**[注:很多人不喜欢这种苦涩而振奋人心的经典薄荷饮料，但我最近又尝试了一次，很高兴地报告说，它仍然是我的最爱。]**

### **[**黑莓果酱树莓**](https://skillet.lifehacker.com/3-ingredient-happy-hour-a-blackberry-jam-bramble-1824303394#_ga=2.18316153.1352987868.1545790970-1157707288.1539189461)T5】**

**尽管美国发明了鸡尾酒，英国发明了荆棘，世界因此变得更美好。这种饮料通常由杜松子酒、柠檬汁、简单的糖浆和黑莓利口酒制成，这很好，但成分太多了。此外，我不想再买一瓶利口酒，所以我拿了一罐果酱，完全不加糖浆。(别担心，我调了杜松子酒来弥补甜酒的不足。)**

### **[**杰弗里·摩根塔勒的《爱尔兰再见》**](https://skillet.lifehacker.com/leave-the-workweek-behind-with-an-irish-goodbye-1829716083)**

**由克莱德·科姆的杰弗里·摩根塔勒创作的《爱尔兰再见》包含了爱尔兰威士忌、白百合和西纳尔。这三者并没有争夺主导权，而是和谐地创造出一种有点水果味、有点木头味、相当僵硬的鸡尾酒。**

**[注:我带了一瓶这些去万圣节派对，和一个穿得像 [摇滚](https://ftw.usatoday.com/2017/05/dwayne-johnson-the-rock-saturday-night-live-snl-fanny-pack-turtleneck-haircut-recreation-photo) 的男人分享。]**

### **[**【车酒 kalmotxo】**](https://skillet.lifehacker.com/stay-cool-and-tipsy-with-a-cheerwine-kalimotxo-1827036088)**

**我想要能吸引眼球的东西，我希望它有趣。这种由勃艮第色调的樱桃味软饮料，Cheer Wine 制成的 kalitmotxo 就是所有这些。为了突出樱桃的因素，并保持饮料的低温而不稀释，我加入了冷冻樱桃作为第三种成分。**

### **[**【萨莎·皮特拉斯克的宇航员(via Robert Simonson)**](https://skillet.lifehacker.com/celebrate-independence-day-with-a-cosmonaut-1827219810#_ga=2.45397988.1352987868.1545790970-1157707288.1539189461)**

**Petraske 的父母和祖父母都是共产党员，根据他的讣告 ，这导致他既“不信任资本主义，又同情劳动阶级”虽然这款鸡尾酒并没有真正的俄罗斯特色——它更多的是对当时无处不在的世界主义的重复——但我仍然非常喜欢啜饮这些酸酸的、略带甜味的杜松子酒，而不是一杯接一杯地喝淡啤酒。**

### **[**葡萄手术**](https://skillet.lifehacker.com/make-a-grape-surgery-cocktail-to-celebrate-the-best-mem-1830946924#_ga=2.235306561.1352987868.1545790970-1157707288.1539189461)T5】**

**这款鸡尾酒让我将对这种完美迷因的热爱和对协和葡萄汁的热爱结合在一起，这是一种童年主食，给我带来了极大的安慰，让我的嘴巴很开心。(警告:任何谈论协和葡萄汁的人都有被封杀的危险。)我决定使用伏特加，一种中性口味的烈酒，而不是试图寻找一种补充或增强果汁的烈酒，因为葡萄汁不需要增强。**

### **[](https://skillet.lifehacker.com/the-kir-pettilant-is-the-perfect-new-years-eve-drink-1831367189)**

****你所需要的是一点黑醋栗，一些起泡酒，和一条柠檬皮作为装饰。如果你想犒劳一下自己，就拿些大写字母 c 的香槟，做一杯 Kir Royale，但我不太喜欢在优质香槟里掺任何东西。****
