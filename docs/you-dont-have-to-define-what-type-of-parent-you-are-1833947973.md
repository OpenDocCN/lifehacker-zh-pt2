# 你不必定义你是哪种类型的父母

> 原文：<https://lifehacker.com/you-dont-have-to-define-what-type-of-parent-you-are-1833947973>

十年前，当妈咪博客越来越受欢迎，我们都应该感到团结在生存的保护伞下时，妈咪战争的现象爆发了。这些所谓的战争，围绕着一个人如何选择父母，在全职妈妈与职业妈妈之间，母乳妈妈与配方奶喂养者之间，穿婴儿装的妈妈与婴儿车爱好者之间，以及同睡父母与婴儿床使用者之间展开了斗争。



虽然很多尖酸刻薄的话要么是捏造的，要么是被媒体 渲染的，但我在自己的生活中看到了它的影子。父母——大多数是母亲——被分成小部落，在那里他们可以在自己特定的育儿策略中感受到认可和支持。当面临成为父母这一改变人生的转变时，这是完全正常的，人之常情。但是当我们向对方寻求支持、同情和建议时，我们也开始给自己贴上这样或那样的父母标签。我们开始比较。

当我的儿子还是个婴儿的时候，我不仅知道他的小伙伴们是否接受母乳喂养，而且知道具体是几个月。我知道谁在用布换尿布，谁坐在那些有毒的一次性用品里(我可以这么说，因为他是其中之一)。我知道谁哭着睡着了，谁肯定没有。

对我们中的许多人来说，我们部落中的妈妈们似乎突然比我们做得好得多。为人父母，尤其是在网上，变成了一场竞争。

加剧这一切的是 Instagram 的“影响者”,他们开始向我们展示，如果我们有免费的衣服和完美的灯光，我们可以多么完美地养育孩子。作家凯瑟琳·杰泽尔-莫顿在《T2》《T3》和《T4》《T5》中这样描述。

> 影响者时代带来了即时完美的生活，每个人都知道这是精心策划的，但我们喜欢遵循——并批判性地拆除——无论如何。

很快，我们都差不多受够了——他们的家不可能一直看起来那么漂亮！— 和*真正的妈妈们寻找新的方式来定义自己。一些渗出“我做得很好”的东西，但也有“但是伙计们，这太难了”*

*于是“完美不完美”运动诞生了。 [杰泽-莫顿写道](https://www.thecut.com/2019/04/online-moms-mommyblogs-instagram.html) :*

> *《完美不完美》声称拒绝完美主义的陷阱，经常出现在自我接纳的“旅程”或战胜逆境的故事旁边。它经常被用来描述一种整体的生活方式:忘记快乐的家庭主妇的例行公事，拥抱家庭生活的混乱和爱。*

*给自己贴上“完美不完美”父母的标签意味着自我接受，是的，但这听起来仍然是错误的。*

*现在我的儿子八岁了，我们住在不同的州，我很高兴地告诉大家，我完全不知道他的朋友马修是否是母乳喂养的，或者他的另一个朋友马修在婴儿时期是否在托儿所，或者他的另一个朋友马修是否通过婴儿引导断奶法开始吃固体食物。(我们这一带有很多马修斯。)* 

*更好的是，除了试图判断他们的父母是否是好人之外，我不可能不关心他们是如何长大的。*

*你不需要定义你是什么样的父母，无论是对你还是对其他人。没有人永远是那种类型的父母。你可以什么都做一点。或者你可以暂时是一堆东西，然后是一堆别的东西。当你的孩子真的需要的时候，你可能会在某一天用直升飞机把他送过来，让他在一周的剩余时间里自由活动。如果它是为你的家庭工作的，它不需要适合一些整洁的小盒子或吸引人的标签。*

*几个月来，我的婴儿只吃自制的有机水果、蔬菜和蛋白质浓汤。作为一个蹒跚学步的孩子，他吃了一大堆微波恐龙形状的鸡块。*

*当谈到他消费什么样的媒体和他如何消费时，我是非常严格的。但是我真的不限制屏幕时间。*

*当他在家的时候，我不让他玩在线多人游戏，但是当他在外面和他的朋友玩的时候，我根本不怎么监督他。*

*我在培养，*和*我以大喊大叫而闻名。*

*我的标准因星期几、风向和我的心情而异。我完全不知道我是什么样的父母，除了希望是一个体面的。*

*我不知道如何从零到完美来定义自己。你也不需要。*
