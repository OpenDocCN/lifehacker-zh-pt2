# 我从 iOS 相册中删除了照片，现在它们永远消失了？

> 原文：<https://lifehacker.com/i-removed-photos-from-an-ios-album-and-now-theyre-gone-1843908428>

我喜欢玩寻宝游戏——只是不是当搜索的对象是“我想永远保存在我的设备上，但我现在找不到的照片”虽然当你从文件夹中删除照片或者删除文件夹或成批照片时，你的手机应该非常智能，但有时整个过程可能会有点混乱。



当你的照片消失时，即使你并不想让它们消失，这也是乐趣的开始。对于本周的[**Tech 911**](https://lifehacker.com/c/tech-911)T5】Q&A，我就让 Lifehacker 读者**司徒**解释一下:

### 问:我的 iPad 的照片去了哪里？

> 我妻子在她的 iPad 上有两个相册有相同的照片，今天她删除了一个相册中的照片，两个相册的内容都被删除了！我们已经在最近删除的照片中寻找，但它们也不在那里，请问有什么建议吗？

### (希望)没有全部丢失

好消息是。我不相信你的照片会永远消失。事实上，我想不出有什么办法可以让删除照片、相册或相册中的照片，从而将它们从你的设备中永久删除。

像我这样的 Windows 用户知道这种恐惧——多亏了我们中的一些人不顾后果地使用的 ol' Shift+Delete 热键，我们偶尔会发现*做了* 却不小心永远删除了一些东西(哎呀)。另一方面，你的 iOS“最近删除”文件夹既是你的回收站，也是你的救星。

然而，为了确保我没有胡说八道，我完全按照你在提示中描述的那样做了。我拍了几张照片，在照片中创建了两个文件夹，然后把这四张照片放在一个文件夹中，另外两张放在另一个文件夹中。当我试图删除第二个文件夹中的两张图片时，我得到了这样的提示:

点击“从相册中删除”，这就是所有将发生的事情；它们会留在你的相机胶卷中，它们只会从相册中消失，它们是从中删除的。如果你点击“删除”，就像你可能做的那样，图像*将*从那个文件夹中消失，并且你放置它们的任何其他文件夹*和*将从你的相机胶卷中消失。然而，它们仍然不会永远消失。不完全是。

当我调出最近删除的文件夹，这是在**照片>相册**中找到的，如果你滚动到最底部，呃，最近删除的图片确实还在那里。我想你的问题有一个非常简单的答案，那就是苹果如何显示被删除的图片。

当您第一次打开该文件夹时，您会看到所有最近删除的图像——过去 30-40 天内的任何图像——最下面是最早的图像，因为它们从现在到到期的时间最短。在我的 iPhone 上，我最近删除的文件夹在删除了那两张照片后加载时是这样的:

然而，一个 s 滚动到最近删除的顶部，显示出我刚刚删除的图像，准备用我的手指轻敲几下恢复。

这就是我对你为什么会错过那些倒霉的照片的解释。我不知道有什么机制会让它们在从文件夹中移除后就消失了。我唯一的另一个想法是，如果你正在查看一个共享文件夹，而该文件夹的创建者删除了其中的图像，那么它们确实会从你的相机胶卷和你最近删除的文件夹中消失。他们*将*仍然出现在共享文件夹创建者设备上的相机胶卷或最近删除的文件夹中，因此他们也不会真的永远消失。

C 检查 [iCloud](https://www.icloud.com/photos/) 看看你丢失的照片是否已经保存在那里——包括你的相机胶卷*和*最近删除的文件夹。如果这仍然不能打开你的照片——并且你还没有将它们备份到电脑或单独的云服务(如 Google Photos)——你可以尝试给一个几天(或几周)没用的设备加电。如果它被设置为与 iClouD 同步，它们仍然会在那里。将它们保存在其他地方并发送给自己。

假设这些选项都不起作用，从最近的备份中恢复你的 iPad 也应该恢复导致你的照片消失的任何更改。请确保保存您在过渡期间创建的任何内容——其他照片、笔记等——因为当您将恢复到旧备份时，它们会消失。
