# 了解“系统的”和“系统的”之间的区别

> 原文：<https://lifehacker.com/learn-the-difference-between-systemic-and-systematic-1844185555>

鉴于最近关于种族不公正的讨论和示威,“系统性的”和“系统性的”这两个词已经出现了很多，有时可以互换使用。虽然它们有相同的词根，但它们也有不同的意思。如果你不太确定每个单词的确切使用时间， [米农·福格蒂(又名“语法女孩”)](https://www.quickanddirtytips.com/education/grammar/systemic-or-systematic) 和 [语法学家](https://grammarist.com/usage/systematic-systemic/) 的好心人已经为我们拼出来了。以下是需要了解的内容。



### 一些背景

先说词根:系统。根据，“系统”一词最早记录于 17 世纪初，源自希腊语*s stma*，“由几个部分合成的整体”这个词有几种不同的用法，根据韦氏词典的说法，包括“一种社会、经济或政治组织或实践的形式”。这个定义适用于讨论种族主义等行为。现在，让我们来分解一下“系统性”和“系统化”的区别。

### 有系统的

正如 Fogarty 解释的那样， [【系统的】](https://www.quickanddirtytips.com/education/grammar/systemic-or-systematic) 是这两个词中更古老也更常见的，可以追溯到 17 世纪 70 年代的英语。这个词用来形容某件事是彻底的和有意的，有条不紊的或按照计划实施的。她举了这些例子:

> 医生开始了系统的治疗计划。
> 
> 结束系统性歧视是一个有价值的目标。

### 系统的

现在我们来讨论一下“系统性”。自 19 世纪早期以来，这个词就在英语中使用，Fogarty 说 似乎是医生第一次创造这个词来描述整个生物系统中发生的事情——比如病人的身体，或者消化系统。这方面的一个例子是:“他有系统性感染。”

从那以后,“系统性”一词就被用在了政府或商业部门等整个系统的多个部分 中。根据 [语法学家](https://grammarist.com/usage/systematic-systemic/) 的说法，“系统性”是两个词中更具体的一个，意思是“系统范围内的或深深根植于系统中的”例如，这就是它在种族主义方面的用法:它 [通常指的是](https://grammarist.com/usage/systematic-systemic/) “难以逆转的习惯或过程，因为它们已经成为一种制度。”

### “系统性”与“系统性”

考虑到这一点，Fogarty 说，一些问题既是系统性的*又是系统性的*——因此有一些重叠——但它们是不可互换的。她用这个 [概括了](https://www.quickanddirtytips.com/education/grammar/systemic-or-systematic) :

> 如果你想说某件事是有条理的，有组织的，有目的的，那就称之为系统化。

> 如果你想说某件事很普遍，影响到某件事的很多部分，那就叫系统性。
