# 网飞刚刚发布了新的家长控制

> 原文：<https://lifehacker.com/netflix-just-released-new-parental-controls-1842749179>

如果说现在是网飞让父母更多地控制孩子的收视体验的好时机，那就是现在，我们的孩子在家看的节目比以往任何时候都多。他们已经掌握了一些基本知识，有助于确保孩子们不会观看不合适的内容，或者你不想让他们看到的节目。

Watch

在 [网飞新的家长控制](https://media.netflix.com/en/company-blog/more-choice-and-control-for-families) 下，你可以:

*   用 PIN 码保护个人资料，防止儿童使用
*   按成熟度等级过滤标题
*   过滤掉您不想让他们观看的个别节目或电影的标题

*   检查您孩子的观看活动
*   关闭自动播放单集和自动播放预览

网飞还指出，你屏蔽的任何节目或电影在他们的个人资料中都是不可见的。所以你可以根据需要使用“哦，奇怪，网飞一定是把那个拿掉了”。

要访问家长控制，请在您的笔记本电脑或移动设备上转到“帐户”，然后向下滚动到“个人资料和家长控制”。
