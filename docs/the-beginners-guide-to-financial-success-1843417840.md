# 金融成功入门指南

> 原文：<https://lifehacker.com/the-beginners-guide-to-financial-success-1843417840>

理财最难的部分是开始。

一旦你建立了一些结构，检查你的钱就成了例行公事，并且在这个过程中更容易做出决定。但是“结构”并不意味着建立一次预算，然后就忘记了。这并不意味着开立一个退休账户，并希望它能让你成为百万富翁。



你必须定期保管好你的钱。为了让你保持在正确的轨道上，你会想要访问(并重温)你的财务工具箱中最重要的部分。

### 编制预算

如果你不知道你在做什么，你就不知道该拿你的钱做什么。

有一百万种方法来预算你的钱，但是如果你正在寻找一种非判断性的方法开始，尝试一下方法，它让你把你的钱分成四类来快速了解什么是有效的，什么是无效的。

如果一开始处理太多，把事情分成两周一次的而不是一个月一次。

如果你准备好迎接返璞归真的终极挑战，那么用纸质支出账本来做传统的事情怎么样？保持它，即使是一个月，也会确保你每天都关注你的预算。

### 省钱

一旦你控制住了你的预算，是时候加大你的储蓄力度了。

那么，每个人都在谈论的这些应急基金是怎么回事呢？专家说，你应该存足够的钱来支付三到六个月的开销，以防你失业或不得不进行意外购买。但最终，正确的数量取决于你。显然，越大越好。但是，如果省下六个月的房租/水电/食物/交通费用的想法让你喘不过气来，那么任何一笔钱都会有助于提供一个财务缓冲。

一个存放应急基金的好地方是一个高收益的储蓄账户，在你享受尽可能多的紧急时刻的同时赚取利息。 [下面介绍如何选择一个](https://lifehacker.com/how-to-choose-a-high-yield-savings-account-1835461088) 。

如果这些天你的预算感到更加充裕，这并不意味着你可以自由支配。每次你得到加薪的时候，确保你把钱用在了最重要的地方。

预算组织，储蓄支撑，信贷管理？是时候投资了。想象一下交易员在纽约证券交易所的大厅里大喊大叫——你的经历不会是这样的。相反，作为初学者，你最好的选择是 [建立一个不需要太多维护的简单投资组合](https://lifehacker.com/how-to-build-an-easy-beginner-set-and-forget-investm-1686878594) 。
