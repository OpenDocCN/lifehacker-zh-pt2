# 发送重要邮件时，最后添加收件人

> 原文:[https://life hacker . com/when-sending-an-important-email-add-the-recipient-last-1840873886](https://lifehacker.com/when-sending-an-important-email-add-the-recipient-last-1840873886)

当你正忙着起草一封重要的电子邮件时，也许是发给未来的招聘经理，不小心在之前*发送出去是搞砸最简单的方法。也许有一个明显的拼写错误——你可以发誓之前没有——或者你忘记附上你的简历。现在，疯狂的后续邮件流程开始了，而不是以你快乐的方式离开。“非常抱歉，”你写道，然后以书面形式解决问题。* 

Watch

这里有一个显而易见的提示，为了你的理智，值得重申一下:千万不要在重要邮件的开头把收件人的电子邮件地址写在“收件人”栏里。很容易会不小心按下“发送”键，在发完一封你真的希望没有发的邮件后发现自己情绪失控。回复邮件时，我通常会立即删除收件人的电子邮件地址，以避免这种情况，这通常是这里的首要建议。

我们的生命特征编辑 Beth 将电子邮件地址放在正文的第一行。“这样我就可以在准备好的时候进行剪切和粘贴，而不用占用主题栏，”她说。“过去，我习惯在地址(beth@whatever.comx)末尾加个 x，但 Gmail 不允许你这么做。”

或者，你也可以像《边缘》杂志主编凯文·阮承认的那样，保持电子邮件的质量。“我在文本编辑器中起草我所有的电子邮件，因为我总是担心我会不小心太快地点击发送，”他在 [Twitter](https://twitter.com/knguyen/status/1214671984841383936) 上写道。如果这似乎太费力了，为什么不使用 [这种方便但礼貌的](https://lifehacker.com/use-these-convenient-but-polite-canned-email-responses-1840829836) 电子邮件回复呢？
