# 最简单的视频聊天应用，适合与技术斗争的人

> 原文:[https://life hacker . com/the-simple-video-chat-apps-for-people-fighter-wi-1842780065](https://lifehacker.com/the-simplest-video-chat-apps-for-people-who-struggle-wi-1842780065)

社交距离对于降低新冠肺炎的感染曲线和保护我们的安全至关重要，但这并不意味着这很容易。当然，我们有社交媒体、视频聊天和一系列随时可用的娱乐选项，但对于缺乏技术技能的人来说，要搞清楚所有这些东西更难。

对于老年人来说尤其如此，他们已经遭受了越来越多的孤独，并且可能会与我们大多数人用来保持联系的快速技术变化作斗争。幸运的是，通过正确的应用程序和正确的设置设备，我们可以帮助所有经验水平的人与朋友和家人保持联系。

本指南重点介绍了包括文本、语音和视频聊天功能的流行应用程序，以及无需注册额外帐户的易用界面。因此，虽然 Skype、Zoom、Discord、Telegram 和 [等其他类似的应用](https://lifehacker.com/find-the-best-video-chat-app-with-this-mega-guide-1842732323) 可能是显而易见的选择，如果使用 PC 或智能手机是你的第二天性，那么对于有点不喜欢科技的人来说，它们可能太难处理了。

在我们讨论应用程序的建议之前，我们先来谈谈如何让智能手机(或任何小工具)更容易使用。

### 辅助功能选项

您拥有的大多数设备都包含许多辅助功能选项和自定义设置，使它们更易于使用。这些 [辅助选项](https://lifehacker.com/tag/accessibility) 可以做任何事情，从让按钮和文本变大；添加有用的快捷方式；简化应用程序界面；甚至隐藏无关的功能、通知和应用程序。这些设置通常也会反映在您安装的应用程序中，尽管有些应用程序可能需要您配置特定的辅助功能设置。无论哪种方式，这些选项都不应该被忽略，因为它们将使使用和解释新设备或应用程序变得更加容易。

我们有关于安卓 和 [iOS/iPadOS](https://lifehacker.com/though-not-located-in-the-iphone-accessibility-tap-a-l-834682011#!) 上众多 [选项的指南，帮助你入门。Gadget Hacks 也有一个很好的指南，可以帮助老年人或技术恐惧症患者安装智能手机。他们的指南侧重于 Android 手机，但一般提示(大部分)与 Android 平板电脑以及 iPhones 和 iPads 相关。](https://lifehacker.com/the-best-new-accessibility-features-in-android-10-1839118262)

说完这些，让我们来看一下对应用程序的建议。

### [FaceTime](https://support.apple.com/en-us/HT204380) (iOS、iPadOS、macOS)

FaceTime 是你的 iPhone、iPad 和 Mac 上包含的标准视频聊天应用程序，因此对于已经拥有苹果设备的人来说，它将是最简单的选择，并且只会*与其他苹果用户联系。*

您不必安装任何额外的应用程序，您启用的任何 iOS 辅助功能选项也将应用到 FaceTime。该界面通常易于使用和学习，特别是如果这个人已经熟悉了 iMessage 等其他苹果应用程序。

然而，如果你需要一个选项来联系其他平台上的人，如 Android 或 Windows，你会想要使用下面的跨平台解决方案之一。

### Facebook Messenger Lite([Android](https://play.google.com/store/apps/details?id=com.facebook.mlite&hl=en_US)、T5】iOS/iPadOS)

Facebook Messenger Lite 是 Facebook Messenger 的一个更小更时尚的版本，删除了几个额外的功能，专注于短信和视频/语音通话。你仍然可以在 Messenger Lite 中发送图片和视频，但黑暗模式和线索式消息回复等功能不可用。它还缺少一些额外的动画(如个人资料图片滑动到聊天底部，表示用户已经看到了一条消息)，因此加载和阅读对话要简单得多。Facebook Messenger Lite 甚至放弃了 Messenger 移动应用程序上的浮动聊天气泡通知，这可能会惹恼或迷惑一些用户。

Messenger Lite 是跨平台的，无论你在哪个设备上使用它，它的界面基本上都是相同的，这应该有助于保持那些拥有不同设备的人的体验一致。该应用程序没有 PC 版，但如果需要，可以使用主要的 [Messenger web 客户端](http://messenger.com/) 。

就像核心的 Messenger 移动应用程序一样，您可以在没有脸书帐户的情况下使用 Messenger Lite，并且很容易从您的设备给联系人打电话或发短信。这款应用是免费的，它的所有功能都是免费的——只要确保视频通话通过 wifi 进行，以减少数据使用量并避免潜在的超龄费用。

### WhatsApp ( [安卓、](https://play.google.com/store/apps/details?id=com.whatsapp&hl=en_US)[iOS/IP ados](https://apps.apple.com/us/app/whatsapp-messenger/id310633997)、 [Windows、Mac](https://www.whatsapp.com/download/) )

WhatsApp 是另一个免费的、脸书所有的选择，但它在一些关键方面不同于脸书信使。

最值得注意的是，WhatsApp 的开箱即用隐私设置比 Facebook Messenger 更好。所有 WhatsApp 消息和通话数据都受到端到端加密的保护——这基本上意味着它们只能被对话中的用户读取或访问。如果你担心数据隐私，WhatsApp 可能是最简单的选择。当然，也有其他信息应用有更严格的隐私控制，但它们不像 WhatsApp 那样直观，也不像 WhatsApp 那样受欢迎。

WhatsApp 的界面也很容易使用，有些人可能会发现它甚至比 Facebook Messenger Lite 已经简化的界面更容易使用。像 Messenger 一样，Whatsapp 可以用来联系 WhatsApp 的其他用户，以及保存在设备上的联系人，你在该服务上做的一切都是完全免费的——消息、语音通话和视频聊天。

WhatsApp 除了它的 Android 和 iOS 版本之外，还有专门的 Windows 和 Mac 应用程序，加上 [一个 web 客户端](https://web.whatsapp.com/) ，所有这些都可以同步到同一个配置文件。

### [奥斯卡资深](https://www.oscarsenior.com/)

这最后一个选择与前三个有点不同，但仍然值得注意。Oscar Senior 不仅仅是一个视频聊天或信息应用程序；这是一个完整的数字护理包。

Oscar Senior 基本上是一个迷你的二级操作系统，包括许多功能，让老年用户与家人、朋友和护理人员保持联系。它的设计是为了让技术经验最少的人也能轻松使用。Oscar Senior 可用于打电话、发短信和语音聊天，以及跟踪重要的护理数据。该界面使用易于找到的大按钮和带有大文本的小部件，启动视频通话只需按几下按钮。

该服务有几个支付等级选项，需要相当多的设置才能开始，但由于新冠肺炎，该公司目前提供两个月的免费试用。它可以在 iOS、Android、Mac 和 Windows 上使用。

### 智能音箱不是一个应用程序，但它是一种保持联系的简单方式

另一个可靠的选择是完全抛弃应用程序和手机。给你那些被技术困扰的爱人安装一个智能扬声器，他们就可以通过简单的语音请求给你打电话。(你甚至可以为他们写下触发词和标语，以帮助他们适应他们的设备)。

你可能希望你试图联系的人拥有一个和你使用相同平台的智能音箱；如果你是 iOS 用户，那就是 HomePod。如果你已经投资了亚马逊的生态系统，你的朋友或爱人应该买一个回声设备。如果你使用安卓系统，或者有自己的谷歌智能家居设备，他们可能会想要一个谷歌家庭扬声器。

这些扬声器都可以用来打常规电话，只要它们的主人也有一台符合要求的设备——通常是运行该公司的操作系统或在兼容的智能手机上使用正确的应用程序。你可能需要自己为你的朋友或亲戚做这个设置，因为对于那些不太懂技术的人来说，这可能有点复杂，但回报是值得的。智能扬声器是保持联系的一个伟大而简单的解决方案。
