# 我想升级 Windows 7，但我害怕丢失我的程序

> 原文：<https://lifehacker.com/i-want-to-upgrade-windows-7-but-im-scared-to-lose-my-p-1841044394>

本周每个人都在谈论 Windows 7。即使你对个人电脑操作系统的内部运作完全不感兴趣，你的注意力无疑已经被微软的转变所吸引:该公司 [不再支持带有安全更新](https://lifehacker.com/am-i-screwed-if-i-dont-upgrade-windows-7-by-january-15-1840903120) 的 Windows 7。



虽然你不应该因为恐惧而惊慌失措，购买一台全新的 Windows 10 电脑，但对你来说，迁移到一个最终将比已经过时的 Windows 7 安全得多的操作系统是很重要的。但是人们不愿意这样做的最大原因之一是因为他们的程序不能在更现代的操作系统上运行。

对此，Lifehacker 的读者唐纳德写道:

> “在回顾你关于免费的 Window 10 升级的文章时，我有一个问题:我是必须把我所有的程序都保存在一个磁盘上，还是我的应用程序也可以一起升级？”

这是一个很好的问题，因为当你 [试图从 Windows 7 升级到 Windows 10](https://lifehacker.com/quickly-upgrade-windows-7-to-windows-10-for-free-with-t-1840843214) 时，应用兼容性*可能会成为一个问题。我承认，我从来没有进行过这样的升级，因为我总是选择“干净”的安装——清除你硬盘上的所有数据，安装一个全新的、未经改动的 Windows 10 版本，绝对没有任何第三方应用程序。(我稍后会谈到原因。)*

通常情况下，Windows 7 到 Windows 10 的升级过程*应该*带着你所有的应用一起上路。然而，有些情况下它可能不会出现——你的应用可能不会出现，你可能无法选择在升级过程中携带它们的选项，或者你的升级甚至可能不会工作，除非你首先删除由于任何原因无法跳转到 Windows 10 的应用。

我怀疑你遇到第一个和第三个问题的次数可能比第二个多。(如果我是对的，如果你试图从一个版本的 Windows(如 Home)升级到完全不同的版本(如 Enterprise)，这种情况可能会发生，这对大多数人来说是一种罕见的情况。)好消息是，你可能会以某种方式得到提示，升级过程不能(或不会)将它们带到 Windows 10。

这就是为什么我建议如果可以的话，先在 Windows 10 中测试你的关键应用程序，因为你至少会对现代操作系统中什么可行什么不可行有所了解。这样，当你升级时，丢失的应用程序——或者你必须在升级前卸载*的应用程序——就不会令人惊讶了。你就能为这种可能性做好准备。*

我有几个其他的建议，也许能帮到你。首先，我打开命令提示符，键入“wmic”，按 Enter 键，然后键入:

`output:C:\Users\Davem\Desktop\InstalledProgramsWMIC.txt product get name,version`

然后，将我的 c:\位置替换为您想要保存它的位置。TXT 文件，完成后按 Enter 键。然后，您将获得一份安装在您电脑上的所有程序的便捷列表，当您升级到 Windows 10 时，您可以使用它来交叉引用您系统中仍然存在的程序。

(如果你不喜欢这个选项，你也可以通过 Windows 7 控制面板浏览你的已安装程序列表，然后截图。将它们保存在某个地方，然后在检查新安装的 Windows 10 时参考它们)。

为了实现这个完整的循环，我推荐在*结合*中采取这一步，运行一个全新安装而不是升级安装的逆向建议。原因如下。如果你担心你的所有应用程序是否都将升级到 Windows 10，更不用说在 Windows 10 上运行了，你将不得不根据你在系统上安装了多少东西进行大量的抽查。那需要一些时间。为什么不花时间只重新安装你知道你需要的应用程序，而不是检查应用程序是否存在和功能？

我敢打赌，你的电脑上可能有很多你已经忘记并且可能不需要的东西——你曾经使用过但现在不需要的应用程序、你在某个时候安装但忘记了的服务、旧文件、随机数据、垃圾文件*、*等等。转移到一个新的操作系统是一个*完美的*时间来重新开始，因为它允许你非常具体和精确地向你的宝贵、干净的电脑中添加新的项目。

我现在不打算详细介绍这样做的步骤，因为我们已经详细介绍了，但是我鼓励您考虑一下。如果你有一大堆专门的应用程序需要重新安装，这意味着你要做一些额外的跑腿工作，但至少基本的在你的系统上安装和运行起来非常简单。你可能会花一个下午的时间根据自己的喜好重新配置 Windows 10，但我认为你将获得的简化体验是值得的。(之后，您可以随时 [映像您的系统](https://lifehacker.com/back-up-and-clone-your-hard-drive-with-macrium-reflect-1825289970) 以创建一个“准系统”版本的 Windows，如果需要，您可以稍后重新安装，这可以节省您大量的安装时间。)

最后，我衷心建议您在运行升级之前对您的 Windows 7 PC 进行完整备份。无论您对硬盘进行镜像还是克隆，这都是您的事情，但我会复制每一位和每一字节的数据，以便在发生灾难时，将*恢复到*与 Windows 7 完全相同的版本，这是一个简单的过程。只有当您发现即使采用虚拟化，也绝对无法运行 Windows 7 中的关键应用程序时，我才会这么做。你怎么小心都不为过。
