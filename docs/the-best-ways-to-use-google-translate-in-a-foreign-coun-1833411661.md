# 在国外使用谷歌翻译的最佳方式

> 原文:[https://life hacker . com/the-best-ways-to-use-Google-translate-in-a-foreign-coun-1833411661](https://lifehacker.com/the-best-ways-to-use-google-translate-in-a-foreign-coun-1833411661)

如果你使用谷歌翻译只是复制粘贴文本，那你就错过了。该应用程序可以直观地翻译标志和菜单，保存您最常用的短语，等等。

### 离家前下载你的语言

即使你非常确定你在目的地会有数据连接，在谷歌翻译应用程序中下载你的目标语言也只需要一分钟。

只需在主屏幕上点击该语言的名称；它可能会说“检测语言”在该屏幕上，您可以点击您可能需要的每种语言旁边的下载图标。( [这里是谷歌的官方说明](https://support.google.com/translate/answer/6142473?co=GENIE.Platform%3DAndroid&hl=en) )。)

现在你可以在飞越海洋时翻译海关表格。或者当你认为法国的手机网络很糟糕，但实际上你只是忘记了打开国际漫游时，在巴黎到处翻译东西。问我怎么知道的。

### 使用摄像机拍摄标志和菜单

这个功能就像魔术一样。按下小相机按钮，把你的手机放在你的眼球和你要阅读的标志、菜单或表格之间。谷歌将即时翻译单词，插入全大写英语来代替其他语言的单词。

实时浏览您的屏幕，或轻按快门按钮来定格图像。这种模式非常适合菜单或列表:拍照后，你可以用手指突出显示文本，并获得这些单词的翻译。

不幸的是，这款相机不太适合拍摄大块文本。我尝试用这种方式阅读西班牙语杂志，发现该应用程序可以单独翻译每行文本。在杂志文本的窄栏中，这意味着它一次只能翻译几个单词，并且完全误解任何用连字符连接并分成两行的单词。

我们的育儿编辑米歇尔·吴(Michelle Woo)在东京寻找感冒药时指出，谷歌翻译并不能很好地帮助她解读标签，但它确实帮助她排除了提到鼻窦或鼻腔症状的包装，这不是她要找的药物。

### 保存你可能再次需要的短语

如果你只是键入一个单词或短语，谷歌会翻译出来，而你不必按下额外的按钮。但是如果你真的按了回车键，这个应用程序会把这个短语和它的翻译保存在你的历史记录中。

启动你的收藏夹，你将能够在屏幕底部的“保存”标签中轻松找到它们。

### 不要把任何超级重要的东西托付给它

谷歌翻译是一个方便的助手，但错误肯定仍然是可能的。我们的东京团队发现它在菜单方面很棒，但在药店还不足以帮助他们选择购买什么。“最后，一位会说英语的员工给我展示了正确的方法，”米歇尔说。

最近的一项研究 发现，谷歌翻译非常擅长将医院出院说明从英语翻译成西班牙语和汉语，但它有时会出现灾难性的错误——例如，当说明实际上说停止服药时，却告诉病人继续服药。在棘手的情况下，一定要找到一个能够翻译关键信息的人。
