# 为什么我的下载速度在不同的电脑之间差别如此之大？

> 原文：<https://lifehacker.com/why-do-my-download-speeds-differ-so-drastically-between-1844324163>

在一个完美的世界里，你会为超高速互联网服务付费，将你的电缆调制解调器连接到一个相当高速的路由器，并为你家中的每一个设备实现令人难以置信的下载速度。不幸的是，我们生活在现实世界中，而现实世界往往描绘出一幅截然不同的画面。

在现实世界中，你为数据封顶的“无限”互联网服务支付了过多的费用，最终你要花更多的钱从你的 ISP 那里租用一个电缆调制解调器。它是一个路由器/调制解调器的混合体，为你的许多设备提供一个一般的连接...嗯，*你家的*部分。你考虑怎样才能获得一个无线连接来覆盖盲点，最后买了一个蹩脚的扩展器，帮助你实现你花钱买的速度的一小部分。你想知道为什么这如此复杂。

Lifehacker 读者 **Jon** 可能没有*这么糟糕的设置，但他对本周 T5】Tech 911T7&Q&A 的问题提醒我，我们生活在一个普通人不得不经历许多磨难才能(或许)建立快速互联网连接的世界。他写道:*

> 我有两台电脑连接到同一个新的托管 Netgear 交换机。我所有的设备都支持 1000Gbps。举个例子，这里有两张我在两台电脑上运行速度测试的截图:首先是 PC1，然后是 PC2。
> 
> 我不明白通过相同的路由器/电缆调制解调器在相同的子网上测试的巨大速度差异？

### **对性能不佳的网络进行故障排除是一件轻而易举的事**

Jon 实际上给我发来了更多关于他已经研究过的各种事情的细节，以便最大限度地提高他的 1Gbps 互联网计划的连接速度。(是很多！两个电缆调制解调器，两个网状系统，大量的以太网和同轴电缆。一次雷击。这是个不小的故事。)然而，为了简洁起见，这是他的问题的根源:不一致的性能，即使是在有线连接上。

至少，我假设是有线的。让我们先弄清楚:如果 PC #1 通过以太网连接，PC #2 通过无线网卡连接，这就是你的问题所在。在 wifi 上获得一致的、令人难以置信的快速信号要复杂得多——有许多[可以影响](https://lifehacker.com/which-wifi-band-should-i-use-for-my-devices-1832635625)[的质量，你的连接](https://lifehacker.com/how-to-check-your-wifi-signal-strength-1843047801) 和 [产生的速度](https://lifehacker.com/do-i-need-a-new-cable-modem-if-im-stuck-working-from-ho-1842455120) ，这是我在许多专栏中提到的。

速度测试报告不同的结果也有可能只是因为... [他们做](https://lifehacker.com/why-do-internet-speed-tests-report-different-results-1834814879) 。然而，奇怪的是，在同一个理论测试地点进行*相同的*速度测试，却得到如此大相径庭的结果。我想你已经运行了很多次测试，并且看到了你的两台电脑在性能上的巨大差异。

我可能首先要做的是尝试一些其他的速度测试，以防 [速度测试](https://www.speedtest.net/) 出现某种异常。我一直喜欢用但是还有很多其他的。您想要确定您是否在多次测试中看到您的两个系统之间有相同的疯狂变化。如果是这样的话，我们就可以开始缩小故障诊断的范围，找出与出问题的系统潜在相关的问题。如果不是，或者如果结果相反，那么您可能想要后退一步，考虑一些一般性的故障排除(或者只是处理奇怪的差异)。

为了便于讨论，让我们假设您的各种测试表明 PC #2 仍然比 PC #1 慢。我们可以做很多事情来尝试解决这个问题，所以让我们开始吧。

S 从交换两台电脑使用的以太网电缆开始——或者通过另一条 新的、便宜的 Cat 6 电缆连接。也许您在速度较慢的电脑上使用的网线有问题。对于你使用它的距离(以及你希望达到的速度)来说，它也可能被评为太低。你甚至可能有某种松散的连接。尝试不同的电缆。运行您的测试。看看会发生什么。

我假设 PC #1 和 PC #2 都以相同的方式连接到您的路由器。如果 1 号电脑有直接的以太网连接，但 2 号电脑连接到电力线适配器，这就是你如何设法给它一个互联网连接，而不需要在你的家里到处布线以太网电缆，这可能就是瓶颈所在。当您运行测试时，确保您直接从路由器连接到桌面。

在进行速度测试之前，不要忘记重置网络上的所有东西。重置你的调制解调器、路由器、个人电脑和两者之间的任何其他设备——甚至重启你作为网络一部分使用的任何交换机，(为了让我的以太网连接的台式电脑再次恢复速度，我不得不这么做。是的，开关通常是哑的，但是它们*有时会*出问题。

如果这还不能解决问题，请深入挖掘。尝试将以太网电缆连接到交换机或路由器的不同端口。您可以使用第二个以太网端口吗？试试看会发生什么。就硬件而言，我想不出其他什么了——除非你正在使用某种独立网卡或 USB 适配器在 PC #2 上获得互联网连接，这也可能是你的故障点。

在软件方面，检查是否有任何固件更新可以安装在你的调制解调器、路由器、交换机或主板上——仅举几例。有可能有一个更新说已经修复了一个会降低你速度的缺陷。也有可能一个设置或另一个出现了故障，刷新更新的固件(通常会将你的设置恢复为出厂默认设置)可以帮助修复那个。

否则，考虑重置你的调制解调器到它的默认值，然后是你的路由器、，甚至可能是你的 BIOS 和/或操作系统。是的，这是“删除所有内容并重新安装”的方法，这是我推荐作为最后手段的一个选项，但这个选项总是摆在桌面上供你尝试。( Ar 你的系统上有没有你不认识或不用的奇怪的网络主题应用程序？卸载它们 [可能有助于](https://www.reddit.com/r/HomeNetworking/comments/cyb9bb/strange_ethernet_problem_slow_only_on_one_computer/eyr86r0/) 。)

C 检查你的电脑(或你的 网络接口)制造商是否提供了任何你可以安装的更新驱动程序。虽然我不认为这是导致您的第二台 PC 运行缓慢的原因，但这是一个易于探索的修复方法。当你这样做的时候，把 2 号电脑上的 DNS 设置更新到更快的速度(比如 [谷歌 DNS](https://lifehacker.com/how-to-enjoy-the-new-web-speed-boost-from-google-dns-an-5835775) 或者 [Cloudflare](https://lifehacker.com/how-to-browse-faster-and-more-securely-with-cloudflar-1824256064) )而不是你从 ISP 那里得到的任何东西。

你也可以尝试用 [重置你的 TCP/IP 栈](https://support.microsoft.com/en-us/help/299357/how-to-reset-tcp-ip-by-using-the-netshell-utility) ，以管理员身份打开命令提示符，输入 [以下](https://grok.lsu.edu/Article.aspx?articleid=19483) :

`netsh int ip reset`

我不会建议你买任何新东西，除非你已经用尽了所有的故障诊断选项。如果你碰巧有一个朋友有的以太网适配器，你可以使用，看看这是否能提高你的速度。便宜的 [基于 PCI 的](https://www.amazon.com/TP-Link-1000Mbps-Gigabit-Ethernet-supported/dp/B003CFATNI/ref=sr_1_3?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/why-do-my-download-speeds-differ-so-drastically-between-1844324163&asc_source=&dchild=1&keywords=gigabit ethernet card&qid=1594334758&s=electronics&sr=1-3&tag=kinjalifehackerlink-20) 或者基于 USB 的都可以，尽管我总是更喜欢前者，因为它提供了更少的潜在复杂性。(谢天谢地，如果你不得不买一个，一个像样的千兆卡不到 20 美元就能买到——这是让你的网速翻倍的小小代价。)

不过，我只能想到这些了。希望这些技术中的一个能解决你的棘手问题。即使他们不这么做，一个系统上 400Mbps 也不算太差。我知道这不是你希望的 700+ Mbps，但你只会在下载大文件时才会真正注意到速度变慢。否则，如果在这个问题中找到任何一线希望的话，,我认为延迟(和 DNS)对你典型的网络浏览体验的影响将超过这种速度差异。
