# 借助黑洞基于区块链的服务，免费安全传输文件

> 原文：<https://lifehacker.com/securely-transfer-files-for-free-with-black-holes-block-1837879668>

我们之前已经介绍过许多文件传输服务 ，但是我们最近偶然发现了黑洞，一种相对较新的服务使用基于区块链的 Blockstack 网络进行用户登录、文件存储和交付。(最好的一点:是免费的。)

Watch

所有上传到黑洞的文件首先被加密，然后数据被拆分并在其他 Blockstack 用户之间共享。这意味着你的文件和个人数据是完全保密的，即使是对黑洞。其他用户只能访问您的公共用户名。

就像几乎所有其他文件传输服务一样，黑洞上传仍然有大小限制。目前的免费版本限制你的上传到 512MB，但即将推出的 Plus 版本将限制增加到 8GB。与其他服务相比，这些服务的价格略低——例如， [Dropbox Transfer](https://lifehacker.com/is-dropbox-transfer-better-than-other-free-file-sharing-1836249598) 免费账户的起价为 2GB 文件，许多其他服务允许高达 10GB 或更大的文件。

也就是说，黑洞对区块链的依赖意味着所有用户拥有无限的总存储量；下载链接永不过期；任何人都可以从任何浏览器下载上传的文件，即使他们没有黑洞或 Blockstack 帐户。(不过，如果你想上传或发送文件，你仍然需要注册)。你很难找到另一种免费提供所有这些功能(或其中任何一项)的服务。

黑洞既有网络客户端，也有 windows 应用。这两个版本都具有上述所有功能，但 Windows 应用程序有一些独特的好处，如传输历史记录，删除和禁用链接的能力，以及受密码保护的文件。

如果你有兴趣尝试一下黑洞，这里有注册和上传文件的方法。

1.  前往 [黑洞. run](https://blackhole.run/)T3】
2.  登录/创建 Blockstack 帐户。
3.  选择 web 或 windows 客户端。
4.  单击以上传文件，然后从浏览器中选择要传输的文件。你也可以从你的桌面拖放一个文件到上传窗口，而那些使用 Windows 应用程序的用户也可以点击突出显示一个文件，然后按“CTRL + B”来上传它。
5.  等待上传完成。(这可能需要一段时间，具体取决于文件的大小和互联网连接的强度。)
6.  一旦完成，你会得到一个私人下载链接。只有有链接的人才能下载文件，但链接不会过期，所以可以下载的次数没有限制。不要忘记，你可以从黑洞 Windows 应用程序中删除以前上传的文件。
