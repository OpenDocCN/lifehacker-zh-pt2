# 这些是 2019 年新的 401(k)和 IRA 缴费限额

> 原文:[https://life hacker . com/these-is-the-new-401-k-and-IRA-contribution-limits-fo-1830175445](https://lifehacker.com/these-are-the-new-401-k-and-ira-contribution-limits-fo-1830175445)

退休储蓄者可以在他们的 401(k)和个人退休账户中再存 500 美元，因为美国国税局会根据 2019 年的年度生活成本调整 T2。

Watch

这意味着那些参加 401(k)、403(b)和大多数 457 计划的人在 2019 年可以贡献总计 19，000 美元的税前收入，而传统的个人退休帐户和罗斯贡献限额现在是 6，000 美元。

50 岁以上的人仍然可以在个人退休帐户中多缴纳 1，000 美元，总计 7，000 美元，而 50 岁以上的人在 401(k)s、403(b)s 和 457 计划中的追缴限额仍然是 6，000 美元，这意味着老年工人在 2019 年可以缴纳总计 25，000 美元。固定缴款计划(包括 401(k)计划)的总限额(包括雇主缴款和利润分享)现在是 56，000 美元。

罗斯个人退休帐户的收入也在逐步增加，单身和户主的收入为 122，000 美元至 137，000 美元，共同申报的已婚夫妇的收入为 193，000 美元至 203，000 美元。
