买车，尤其是二手车，可能是一件棘手的事情。有许多事情要考虑，其中最主要的是你将如何维护它。如果前任车主在保养汽车方面很糟糕，你会遇到什么样的头疼事？以下是如何避开前面的子弹。

不用说，我们头也不回地离开了汽车和经销店。

现在轮到你了。这些是你发现的危险信号。
