# 为患有痴呆症的亲人创建音乐播放列表

> 原文：<https://lifehacker.com/create-a-music-playlist-for-a-loved-one-with-dementia-1835688112>

看着心爱的人因痴呆而衰老可能是一种可怕、孤立和绝望的经历。但是即使患者失去了认知功能——思考、记忆和推理——他们仍可能与他们对音乐的热爱保持联系。



Reddit 用户 u/sportznut1000 表示，他的妻子在临终关怀医院工作，并参加了一个关于如何更好地治疗痴呆症患者的课程:

> 她与我分享的一件突出的事情是，当给痴呆症患者一个带有个人音乐的 mp3 播放器(和避免分心的耳机)时，他们如何变得(沉浸在)音乐中，即使只有几分钟也能利用音乐回忆起他们生活中与这首歌相关的快乐时刻。我猜大脑中储存音乐的部分是最容易患痴呆症的部分之一。

你可能会想象那会是什么样子，但看到它的行动是令人难以置信的。查看此视频由 [音乐&记忆](https://www.alzheimers.net/2013-06-04/music-therapy-for-dementia/)T3】组织:

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-fyZQf0p73QM&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-fyZQf0p73QM&start=0) 

现在有一种推动让痴呆症患者更容易获得音乐的运动，如英国的“痴呆症音乐 2020 ”运动，旨在明年让所有痴呆症患者都能获得音乐。

为了制作你自己的播放列表，痴呆症患者音乐网站上有 [一些让你开始的提示](https://www.musicfordementia2020.com/2018/10/10/create-a-playlist-listen-to-recorded-music/) :

> 个性化的播放列表可以为个人提供舒适和享受的来源，并有助于确保音乐收听是以人为中心的。在考虑为痴呆症患者设计播放列表时，有证据表明音乐存在“记忆碰撞”。似乎患有痴呆症的人对他们大约在 10 岁到 30 岁之间欣赏和听到的音乐保持着最清晰的记忆。

清单越个性化越好，所以寻找线索，哪首特定的歌曲或哪种类型的音乐最能引起接收者的共鸣。你可能已经有了一些想法，但你也可以问问其他家庭成员，翻翻老照片，看看他们是否去过任何喜欢的音乐会，或者翻翻他们的个人唱片或 CD 收藏。

痴呆症音乐建议当你所爱的人听音乐时，尤其是第一次听音乐时，与他们坐在一起，仔细观察他们的反应。

> 这可能是眼睛睁开或四处移动，手指或脚趾敲击，面部表情的变化。他们可能会变得更加警觉或说话。他们可能会变得更放松或更敏感。将此人回应的任何歌曲放入播放列表。
> 
> 留意红旗歌曲——音乐是强大的。它可以把人们运送到另一个时间或地点。这是一个很好的礼物，但你不想把某人带回一个糟糕的地方。眼泪并不总是负面的，但是如果有人因为某首歌而变得非常激动或痛苦，你应该停止这个会话并丢弃那首音乐。记得记下红旗歌曲，这样就不会再播放了。

[Reddit 用户 u/AdorableLime](https://www.reddit.com/r/LifeProTips/comments/c2rxfb/lpt_make_a_personal_music_playlist_for_a_loved/) ，自称他们是专业的老年人看护者，认同音乐对痴呆症患者的好处。

“令人惊讶的是，我的大多数病人从未忘记他们年轻时最喜欢的歌曲，他们甚至会随着音乐一起唱，”u/AdorableLime 说。“这是激发他们生活欲望的最好方式之一，让他们度过一段快乐的时光，远离每天都在忘记你是谁和他们在哪里的压力。”
