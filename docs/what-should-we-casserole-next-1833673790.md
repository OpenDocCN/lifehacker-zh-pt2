# 接下来我们应该做什么菜？

> 原文:[https://life hacker . com/what-should-we-砂锅-next-1833673790](https://lifehacker.com/what-should-we-casserole-next-1833673790)

大家好，欢迎回到 [*会砂锅*](https://lifehacker.com/tag/will-it-casserole) [*？*](https://lifehacker.com/tag/will-it-casserole) ，我们看到什么饭、什么菜、什么菜系可以重新想象改造成砂锅的栏目。上个月美味可口的砂锅豌豆汤已经吃完了，这意味着我们必须转向一道新的热菜。

这个月，我感觉到一些非常抽象的东西。我想提出一个概念，越开放的解释越好。一如既往，我有一些想法，但请在评论中留下您的原始想法:

*   **乔氏砂锅:**我跟你说实话，这个概念的灵感来自我周四晚上的晚餐，那是四种不同类型的乔氏中式冷冻食品，煮熟后倒入一个碗里。我不后悔，并愿意拓展中国以外的业务。
*   塔可钟砂锅菜:所有塔可钟食品基本上都是同样的五种配料的重新组合，所以我看不出砂锅菜有什么不好。我们可以用实际的塔可钟产品、自制的复制品或者两者的混合来建造它。
*   7-11 砂锅菜:挑战在于用 7-11 店特有的东西制作砂锅菜。我对这方面的进一步方向持开放态度。

我意识到我所有的“概念”仅仅是商店和餐馆，但我对此相当满意。如果你有更多晦涩难懂的想法，我很想听听，并鼓励你和全班同学分享。(如果你有其他商店和餐馆，我也欢迎。)
