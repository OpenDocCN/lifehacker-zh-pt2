# 根据 Reddit 的说法，你做任务的顺序是错误的

> 原文:[https://life hacker . com/the-tasks-you-do-in-the-order-on-1835336490](https://lifehacker.com/the-tasks-youre-doing-in-the-wrong-order-according-to-1835336490)

[“什么是不需要你做任何新的事情，只需要改变你已经在做的事情的顺序的‘生活黑客’？”](https://old.reddit.com/r/AskReddit/comments/bxgvjz/what_is_a_lifehack_that_doesnt_require_you_to_do/) 问 Reddit 用户 DetroitHustlesHarder。Redditors 回答了大量好的时间管理建议，不需要额外的工作，只是以更好的顺序做同样的工作。你需要做的只是控制你的时间表，和一些自我控制。

Watch

*   “首先将附件添加到电子邮件中，”LionelHutz4 说。那就不能忘了附上。
*   Redditor -Axon- adds:在你写完邮件之前，不要填写“收件人:”行，这样你就不会意外地发送得太快。

*   katchibeauty 说，在你开始烹饪之前，拿出你所有的配料，然后边做边清理(比如在食物变硬之前立即冲洗罐子)。

*   特德牧芬说，煮熟玉米后，剥去玉米皮。
*   jumpsteadeh 说:“在把药丸放进嘴里之前，先往嘴里灌点水。”“这样就不会有半干的表面让药片粘在上面了。”
*   “在做其他家务之前先洗衣服，”Marcusaralius76 说。"当机器运行时，从技术上讲，你是在同时做多项家务."

*   PhoneSteveGaveToTony 说:“在去度假之前，彻底打扫你的家。
*   用户 sequoiastar 说，周五晚上打扫，而不是周日晚上。你越早做周末家务，你就越不会有周日焦虑。
*   (抱歉)PM_UR_NUDES_4_RATING 说:“去杂货店购物前先吃饭。”。当然，那是当你没有食物的时候，但是至少买一块糖吧。
*   NoLifeKing_RL 说，淋浴时，在清洗身体之前放入护发素，之后冲洗干净。
*   吃青蛙，第 37 节说:在一天的早些时候做最不愉快的任务，剩下的一天会感觉容易些。另外，你不会有那种恐惧笼罩着你。
*   或者，就像 RandomlyMethodical 建议的那样，起床后马上做一点家务，获得一些动力，“以‘把事情做完’的态度投入到一天的剩余时间中。”
*   “前一天晚上做好你的咖啡,”旁观者说。从个人经验来看，我们同意:在你晚上的例行公事中放入水和泥土，然后设置一个计时器，或者在第二天早上你去淋浴之前打开你的洗衣机。你会感到一阵轻松，因为你不必在喝咖啡前摆弄勺子和水龙头。
*   或者，如果你是一个喜欢喝可可的孩子，听听 Paperfoldingfractal 和 Kimmelane:不要先倒入所有的粉末，或者所有的牛奶。开始用一点牛奶，然后一些粉，然后一些牛奶，等等。
