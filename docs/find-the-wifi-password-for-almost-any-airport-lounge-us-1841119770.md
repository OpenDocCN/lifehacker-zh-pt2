# 使用此免费地图查找几乎任何机场候机室的 WiFi 密码

> 原文:[https://life hacker . com/find-the-wifi-password-for-almost-any-airport-lounge-us-1841119770](https://lifehacker.com/find-the-wifi-password-for-almost-any-airport-lounge-us-1841119770)

幸运的是，我们现在所处的位置是，美国大多数机场都以某种形式提供免费 WiFi。是的，有时你必须看广告才能到达那里，但它就在那里。

也就是说，有时候你最终会发现一个没有 WiFi 的机场，或者一个有免费 WiFi 但有时间限制的机场。在这种时候， [WiFox 谷歌地图](https://www.google.com/maps/d/u/0/viewer?mid=1Z1dI8hoBZSJNWFx2xr_MMxSxSxY&ll=41.165686589932584%2C-94.88615571532335&z=4) 可以帮上忙。

有了它，你可以搜索世界上任何一个机场，看看如何连接那里的 WiFi。

在某些情况下，你只是要找到关于每个人都可用的公共 WiFi 的信息。然而，在其他情况下，您可能还会找到机场一些休息室的密码，或者如何规避时间限制的详细信息。

例如，对于伦敦的希思罗机场，地图上有高级休息室、Admirals Cub、阿提哈德休息室、阿联酋航空休息室、维珍大西洋俱乐部会所以及另外六个私人网络的密码。

根据你在机场的具体位置，有时休息室的 WiFi 会比免费网络更好。其他时候，这可能是你唯一的选择。

正如你所想象的，WiFi 密码会改变，所以在使用该工具连接时，会有一些明确的命中和未命中。

地图中的所有信息，以及附带的 [iOS](https://apps.apple.com/us/app/wifox/id1130542083) 和 [Android 应用](https://play.google.com/store/apps/details?id=com.foxnomad.wifox.wifox&hl=en_US) (均售价 1.99 美元)都是用户生成的，因此当你发现这些差异时，你可以报告它们，这样就不会有人遇到问题。当您找到一个尚未被报告的网络时，您也可以提交密码，这样其他人也可以使用它。

如果你是一个经常出差的人甚至是一个不经常出差的人，它可以成为你口袋里的一个很好的工具。
