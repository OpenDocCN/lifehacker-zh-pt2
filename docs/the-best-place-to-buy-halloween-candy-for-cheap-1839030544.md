# 廉价购买万圣节糖果的最佳地点

> 原文：<https://lifehacker.com/the-best-place-to-buy-halloween-candy-for-cheap-1839030544>

如果你打算在今年的万圣节分发糖果，你可能应该全力以赴，分发全尺寸的名牌糖果。你不仅会成为你所在街区的英雄，还会让你其他更廉价的邻居感到力不从心，这是一个有趣的奖励。为了买到最便宜的万圣节糖果——和每个孩子最多的笑容——去你当地的餐馆用品店。

Watch

餐厅用品商店规则。除了 [盛汤的容器和不令人恼火的保鲜膜](https://lifehacker.com/the-top-5-restaurant-supply-store-essentials-ranked-1797727937) ，他们以有竞争力的价格出售大量的消耗品，你不必拥有一家餐馆或任何类型的会员资格就可以在大多数地方购物(去之前打电话核实)。这意味着你可以以低廉的价格买到大量的巧克力。虽然不同商店和不同地区的价格不同，但我发现我当地智能食品服务仓库的糖果比通常的大盒子连锁店更便宜。例如，你可以花 24.57 美元买到一盒 36 包的里斯花生杯。这相当于每包 66 美分，比我当地的 Target 和沃尔玛的 Reese 便宜几分，在沃尔玛，一个全尺寸的两杯包装分别为 72 美分和 68 美分。这看起来差别不大，但是每个孩子几分钱加起来，特别是如果你住在一个非常容易被贩卖的“不给糖就捣蛋”的社区。

除了便宜但奢华的糖果，你还可以买一些派对用品，比如——我不知道——给成年人的一百多个小塑料酒杯。拉着一群孩子到处给他们装糖是一项艰苦的工作，他们的父母也应该得到一点奖励。
