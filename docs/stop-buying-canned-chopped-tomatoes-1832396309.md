# 不要买罐装的切碎的西红柿

> 原文：<https://lifehacker.com/stop-buying-canned-chopped-tomatoes-1832396309>

我一直不明白罐装切碎的西红柿有什么意义。它们的味道很少像去皮的同胞一样好，而且在烹饪时它们从来没有完全分解，在所有其他成分都变成糊状后很长时间都保持着立方体的形状。罪魁祸首？氯化钙。



这种化合物应该可以防止番茄在罐头里变成糊状，但是它做得有点太好了。也许你想让热的、异常坚硬的西红柿漂浮在你的炖菜和酱汁中，但我不想——我想让我的西红柿融化。(注意:一些完整的番茄也用氯化钙处理过，但是更容易发现那些没有用氯化钙处理的。看看标签就知道了。)此外，使用整个去皮的家伙意味着我只需要保存一种类型的番茄。接招吧，资本主义。

切光滑、去皮的李子西红柿似乎是一件很麻烦的事情，但实际上很容易控制。只需拿一把干净的厨房剪刀，直接插入罐头(如上图所示)，将西红柿切成你想要的大小，然后加入你的汤、炖肉或酱汁中。如果你想要大块的可辨别的番茄，不用氯化钙也是可以实现的。简单地保留一些你自己切的西红柿，在烹饪过程中稍晚一点加入。它们会很厚，但不会令人不安地坚硬，因此你的酱汁味道会更好。(想要碎番茄？用你的手。是的，这很乱，但却是一种奇怪的宣泄。)
