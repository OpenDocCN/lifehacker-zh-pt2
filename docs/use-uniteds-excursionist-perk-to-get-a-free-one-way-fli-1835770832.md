# 使用里程时，使用美联航的“短途旅行者优惠”获得免费单程航班

> 原文：<https://lifehacker.com/use-uniteds-excursionist-perk-to-get-a-free-one-way-fli-1835770832>

如果你在用奖励里数兑换联航的国际旅行，有一个简单的方法让你的虚拟货币更有价值:计划一次至少三条腿的旅行。



本周 [Travel + Leisure 发表了一篇](https://www.travelandleisure.com/travel-tips/points-miles/how-to-book-flights-with-united-excursionist-perk) 的报道，讲述了一个鲜为人知的美联航员工福利，名为“ [短途旅行福利](https://www.united.com/ual/en/us/fly/mileageplus/awards/travel/types.html) ”这项特权允许您在预订涉及多个城市的旅行时获得一张免费的单程奖励机票。

例如，如果我想在回家之前预订从旧金山到苏格兰，然后从苏格兰到伦敦的航班，那么我可以获得从苏格兰到伦敦的免费航班。

为了享受这一优惠，您不能在您交易的起始地所在的 MileagePlus 规定区域内。因此，如果你在北美开始你的旅程，那么这种额外津贴必须是针对北美以外的旅行，反之亦然。

您的旅行必须在出发的同一地区结束，并且免费单程航班的服务舱和奖励类型必须与之前的单程航班相同或更低。

如果你有足够的里程来获得奖励机票，并想尝试一下，你可以通过美联航的网站预订机票时使用高级搜索收费来找到这个选项。你需要选择多城市选项，然后输入你的目的地。如果您的旅程符合交易条件，您将看到其中一段里程为零。
