# 使用 Any.do 的“智能购物清单”让购物再次充满乐趣

> 原文：<https://lifehacker.com/make-shopping-fun-again-with-any-dos-smart-grocery-list-1831614324>

iOS: 现在有如此多的任务管理应用，选择一款最适合你的应用本身就是一项任务——你可能会想在待办事项列表中或者，老实说，在一个巨大的电子表格中跟踪它。然而，如果你是一个总是在家里购买杂货的人，你会在 Any.do 的应用程序中发现的一个全新功能可能会将其提升到你的(应用程序)购物清单的首位。



在我展示 Any.do 令人惊叹的新购物清单功能之前，我有一个简短的说明。你现在只能在 any.do 的应用程序的 [iOS](https://itunes.apple.com/us/app/any-do-to-do-list-calendar/id497328576?mt=8) 或 [Android](https://play.google.com/store/apps/details?id=com.anydo&hl=en_US) 版本上找到它。该工具的网络版总有一天会到来， [该公司称](https://blog.any.do/2018/10/shared-grocery-list/) 。

在我看来，值得等待。

若要开始，请确保您在 iOS 上安装了最新版本的 Any.do 应用程序。打开它，在左上角寻找一个看起来像购物篮的新图标:

如果你是第一次使用这个功能，点击它，你会被提示“满足未来的购物清单”。点击“演示”开始，这就是乐趣所在。

您将看到一个小的文本输入，其中预先填充了一些您可能想在商店购买的标准商品:牛奶、鸡蛋、面包、黄油等等。你也可以输入你自己的购物需求——当你这样做的时候，你可以自由地指定细节。我很高兴地看到，在我只输入了“Haribo”的前几个字母后，any.do 就正确地自动推荐了“Haribo gummi bears”无论您需要什么，只要点击弹出的商品，就可以快速将它们添加到您的购物清单中。

只是，你不会像大多数任务管理应用那样，把你需要买的东西添加到一个巨大的购物清单上。相反，Any.do 会自动按类别对项目进行排序。这意味着，当你去商店购物的时候，当你看到清单的底部并意识到你忘了带罐装豌豆时，你不必跑回“水果/蔬菜”通道。这是一个非常方便的功能，效果非常好——至少，这个应用程序正确地将我的购物清单上的所有东西分类。

您也可以从设备的剪贴板将项目导入到您的购物清单中。如果你正在寻找某一特定食谱所需要的确切信息，这是很有用的。我确实发现，在商品名称前添加一个度量单位的食谱——例如，“1/4 杯调制番茄酱”——会混淆 Any.do 的分类机制，这意味着该商品被归入一个包罗万象的“其他”类别。一旦我把这个项目重新命名为“番茄酱”，这个应用程序就正确地把它归类到了“餐具室”类别。

Any.do 使与他人(如您的室友或家庭成员)创建协作列表变得容易。当然，他们需要自己的 Any.do 账户，但这样你们就能为同一个购物清单做贡献了。

我建议为每个人都需要去的每个主要商店创建不同的购物清单，比如单独的“Trader Joe's”和“Whole Foods”购物清单，这样你就不会对应该在哪里买什么感到困惑(如果你很挑剔的话)。虽然您不能将项目分配给与您共享列表的特定人员(但愿如此！)，至少这样可以避免大家在你只需要一个的时候，在回家的路上买一个面包。
