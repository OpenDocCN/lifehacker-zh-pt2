# 你可能完全吃错了枸杞

> 原文:[https://life hacker . com/you-possible-eating-goji-berries-all-wrong-1844002919](https://lifehacker.com/youre-probably-eating-goji-berries-all-wrong-1844002919)

自从进入 21 世纪以来，枸杞才在美国声名鹊起，被吹捧为具有无数健康益处的超级水果。但是中国人已经把枸杞作为茶和滋补品的重要成分享用了几千年。根据中国传统医学哲学，枸杞子滋阴明目，对肝肾有益。现代科学告诉我们，这些浆果含有全部九种必需氨基酸，是维生素 A、维生素 C、铁、锌和纤维的良好来源。

像美国人选择的许多亚洲配料一样，枸杞已经获得了神话般的超级食物地位。它们经常出现在果汁吧、健康食品咖啡馆和全食超市的货架上。干瘪的浆果被当作时髦的异国葡萄干，撒在从沙拉到巴西莓碗的所有东西上，或者混入格兰诺拉麦片。

作为一名美籍华人，我总是对此感到恼火。枸杞不是葡萄干。这就是它们的销售方式，因为精致的浆果很容易腐烂，你很难发现它们是新鲜的。在亚洲，枸杞在食用前会被重新水合。这并不难——你只需加水，等 20 分钟。

对于一个中国人来说，把枸杞干果拌在什锦沙拉里或者和酸奶冻糕一起吃，就像嚼干意大利面一样有意义。干枸杞很难嚼，会卡在你的牙齿里，一旦重新水合，味道就没有那么好了。一旦变得丰满和柔软，它们就变得更加多才多艺，在中国烹饪中经常被用来装饰美味的菜肴，如豌豆苗或脆皮鸡。你也可以把它们折叠成米饭或谷物菜肴，或者把它们加入调味汁或蘸酱中。(如果你喜欢额外的奶油质地，试着用你选择的牛奶或牛奶替代品代替水将浆果再水合。)复水的概念并不是枸杞独有的:其他中国美食，如海参、燕窝和雪耳都是晒干后出售的，需要复水，几个世纪以来，干燥一直是一种有用的食物保存方法。

你所知道和喜爱的市售枸杞，就是*枸杞、*鲜红色橙子，原产宁夏。就像有许多食物一样，人类已经广泛繁殖了这一品种，但还有许多其他的，包括*黑果枸杞*，一种在喜马拉雅山脉发现的野生黑色枸杞浆果，类似于四川胡椒。它的深紫色标志着有效的抗氧化剂，但你很难在美国找到它。

虽然对这些脆弱浆果的需求持续增长，但它们的保质期很短，只有少数美国农民尝试过商业化种植枸杞。索诺马的 Goji Farm USA 是最近最成功的，但即使是他们也没有在全国范围内推广。所有商业品牌的枸杞干(像 Navitas Organics 和 Viva Naturals) 都来自中国。在美国，要找到难以捉摸的鲜枸杞，你最好的办法是去加州的农贸市场，或者去 T4 自己种植。

枸杞是一种带刺的木本灌木。它是茄科的一部分，与番茄有亲缘关系。对于这样一种稀有的水果，如果你有一个排水良好的土壤花园，它们实际上非常容易在家里生长。在苗圃寻找凤凰之泪或深红之星枸杞幼苗，开始种植吧。枸杞可以抵御炎热的夏天和寒冷的冬天，只要你的花园没有太多的雨水，它们就有一个漫长的收获季节，从仲夏开始。如果你能保护它们免受蚜虫和鸟类等食肉动物的侵害，你将有几个月稳定的新鲜枸杞供应——不需要补充水分。
