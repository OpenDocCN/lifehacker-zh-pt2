# 你不知道的最好的 iPad 键盘快捷键

> 原文：<https://lifehacker.com/the-best-ipad-keyboard-shortcuts-you-didnt-know-about-1834729560>

如果你在 iPad 上使用外置键盘，你可以利用 Mac 上的一些键盘快捷键。 [甜蜜的设置](https://thesweetsetup.com/best-ipad-keyboard-shortcuts/) 列出了对 iPad 用户特别重要的好的键盘快捷键，包括这些不太明显的:

Watch

*   **按住向下箭头**调出软件键盘，无需移除。
*   **Command-H** 转到主屏幕。
*   **Command-shift-T** 重新打开上次关闭的标签页。
*   **Command-shift-D** 在 mail 中发送电子邮件。

Sweet 设置还列出了 Bear、Ulysses、Adobe Lightroom CC 和 Things 3 等应用程序的快捷方式。

棘手的是，虽然许多快捷方式在 iPad 和 Mac 上是相同的，但有些略有不同，或者工作方式有所不同: **Command-tab** 只在你最近的十个应用中循环。你必须习惯工作流程中的小调整。
