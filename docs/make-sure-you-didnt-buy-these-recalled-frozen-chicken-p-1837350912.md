# 确保你没有购买这些召回的冷冻鸡肉饼

> 原文：<https://lifehacker.com/make-sure-you-didnt-buy-these-recalled-frozen-chicken-p-1837350912>

泰森公司正在召回超过 39，000 磅冷冻鸡肉饼，原因是担心它们被“外来物质”污染

Watch

这些馅饼以 Weaver 品牌出售，分布在密苏里州、马萨诸塞州、康涅狄格州、新罕布什尔州、纽约州、佛蒙特州、俄亥俄州、宾夕法尼亚州、德克萨斯州和弗吉尼亚州。

此次召回具体涉及 2019 年 1 月 31 日生产的 26 盎司全熟鸡肉饼的重新密封袋，保质期为 2020 年 1 月 31 日。

受影响的袋子将在标签上印有批号 0319PBF0617、0319PBF0618、0319PBF0619、0319PBF0620、0319PBF0621、0319PBF0622、0319PBF0623 或 0319PBF0600。

如果你对召回有疑问，你可以直接联系泰森食品公司，电话是(855) 382-3101。
