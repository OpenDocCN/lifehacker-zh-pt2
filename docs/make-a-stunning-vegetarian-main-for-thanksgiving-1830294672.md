# 为感恩节做一道真正令人印象深刻的素食大餐

> 原文：<https://lifehacker.com/make-a-stunning-vegetarian-main-for-thanksgiving-1830294672>

你可能已经注意到，我总是关心素食者 的感恩节困境，这种关心也延伸到我的素食同事。对于我们的小员工感恩节，我们的(素食)视频制作人， [亚当·鲍尔斯](https://lifehacker.com/im-adam-powers-lifehacker-video-producer-and-this-is-1829200075) 想带来一道无肉菜，而*我*想确保它尽可能地有趣和美味。进入[garsquashroom](https://skillet.lifehacker.com/introducing-the-garsquashroom-the-vegetarian-answer-to-1820552234)——以组成它的大蒜、南瓜和蘑菇命名——这是一种植物性的尤物，吃起来和说起来一样有趣。
