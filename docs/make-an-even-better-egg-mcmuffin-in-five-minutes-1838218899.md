# 在五分钟内做出更好的鸡蛋松饼

> 原文：<https://lifehacker.com/make-an-even-better-egg-mcmuffin-in-five-minutes-1838218899>

我最喜欢的早餐之一是麦当劳的鸡蛋松饼，但没有汽车意味着我不得不从 UberEats 订购，没有什么比麦当劳的外卖更让我觉得自己是个垃圾了。幸运的是，我已经知道了如何用我的垃圾烤箱在大约五分钟内做出一个麦当劳式的三明治。

我说“麦克芬顿式”是因为它比快餐早餐三明治看起来更乱，但那是由于烹饪方法，它从马蒂·马西森的 [博洛尼亚碗](https://lifehacker.com/matty-mathesons-bologna-bowl-is-very-satisfying-1829668475) 中得到启示。我是博洛尼亚碗的超级粉丝，但我并不总是有心情以四片博洛尼亚开始我的一天，所以在一个命运攸关的早上，我做了一个迷你博洛尼亚碗，只用了一片博洛尼亚。我把午餐肉压在一个小碗里，加了一片美国奶酪，在上面打了一个鸡蛋。然后我把整个东西放进微波炉加热两分钟。

不出所料，这道菜非常好吃，而且——多亏了腊肠包裹鸡蛋和奶酪的方式——我可以把整个东西放在一片吐司上，方便食用。太完美了。腊肠边缘很脆，奶酪完全融化了，鸡蛋也煮透了，但蛋黄还是有点软。这让我想起了鸡蛋松饼，但用了腊肠而不是加拿大培根。很明显，我不得不尝试用加拿大培根。

它工作得很好。然而，我用了两片加拿大熏肉，因为它们非常小，一片不足以覆盖 ramekin 的底部。但是过程是完全一样的:我把肉和奶酪放在一个微波安全的小容器里，在上面打一个鸡蛋，然后用微波炉加热两分钟。(关于微波鸡蛋的一个注意事项:我有一个非常便宜的微波炉，并且在加热整个鸡蛋时没有任何问题，但是你可能想要打破一点蛋黄，特别是如果你有一个别致的大功率微波炉。蛋黄加热过快会导致蒸汽在膜下积聚，从而导致蛋黄爆裂，这是肮脏和糟糕的。戳蛋黄会打破膜，这将有助于蒸汽逸出。)

几分钟后，奶酪已经转移到顶部，包住了鸡蛋，我把它从微波炉中拿出来，让它冷却，同时我烤了一个英国松饼并涂上黄油。(它需要这个冷却期；这东西出来是熔化的。)松饼做好后，我用黄油刀在松饼边缘划了一下，然后把早餐圆盘滑到面包上。

它没有真正的麦当劳松饼的无菌分层，但每一口都有鸡蛋、奶酪和肉，而且——除了往嘴里塞奶酪——这是我为自己做的最快的早餐。我想我更喜欢博洛尼亚的想法。博洛尼亚很难打败。
