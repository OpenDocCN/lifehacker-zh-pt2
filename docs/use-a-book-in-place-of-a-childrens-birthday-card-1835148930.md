# 用一本书代替儿童生日卡

> 原文：<https://lifehacker.com/use-a-book-in-place-of-a-childrens-birthday-card-1835148930>

当到了打开生日礼物的时候，孩子们关心卡片的程度就像他们关心他们的 Cap'n Crunch 包装盒上的营养价值一样。它通常被扔在一边，信息消失在堆积如山的废弃包装纸中。

Watch

这就是为什么我喜欢 Reddit 用户 fouronthedice 的这个想法 :不要把自己拖到 Hallmark 过道，拿一本便宜的儿童书籍，在封面上写下你的生日信息。这本书现在成了卡片。很功利，很有意义——每次孩子读故事，她都会看到你特别的备注。

不管怎么说，贺卡价格过高，大约要花 5 美元——如果你有一张突然出现在你面前并会唱歌的贺卡，价格会更高。你可以找到价格差不多经典书籍，如 [*费迪南的故事*](https://www.amazon.com/Story-Ferdinand-Munro-Leaf/dp/044845694X/ref=zg_mg_2970_33?_encoding=UTF8&asc_campaign=InlineText&asc_refurl=https://lifehacker.com/use-a-book-in-place-of-a-childrens-birthday-card-1835148930&asc_source=&psc=1&refRID=YSTTQS8HYMM602W51WX9&tag=kinjalifehackerlink-20)*[*饥饿的毛毛虫*](https://www.amazon.com/Very-Hungry-Caterpillar-Eric-Carle/dp/0399226907/ref=zg_mg_2970_1?_encoding=UTF8&asc_campaign=InlineText&asc_refurl=https://lifehacker.com/use-a-book-in-place-of-a-childrens-birthday-card-1835148930&asc_source=&psc=1&refRID=6B01CM4SAAPTFB9JH5QS&tag=kinjalifehackerlink-20) 或 [*哦，该想的你可以想！*](https://www.amazon.com/Oh-Thinks-You-Can-Think/dp/0394831292/ref=pd_sim_14_2/131-1179522-8606045?_encoding=UTF8&asc_campaign=InlineText&asc_refurl=https://lifehacker.com/use-a-book-in-place-of-a-childrens-birthday-card-1835148930&asc_source=&pd_rd_i=0394831292&pd_rd_r=e99c2312-83af-11e9-b31d-a3211b75eab9&pd_rd_w=HYe4W&pd_rd_wg=AhByK&pf_rd_p=90485860-83e9-4fd9-b838-b28a9b7fda30&pf_rd_r=D84S0PFRXQHZRJSBSFT8&psc=1&refRID=D84S0PFRXQHZRJSBSFT8&tag=kinjalifehackerlink-20) 或者，如果你的孩子是生日接收者的朋友，他可能会分享他自己图书馆的一本书。*

*在你的信息中，包括你为什么选择这本书，并注明年份。你可以把它放在一个大信封里，或者在封面上贴一张便条，这样孩子就知道往里面看了。然后把它包含在你的礼物里。这种情感会比一张昂贵的卡片持续更久，卡片上有一只猫，上面写着“祝你生日快乐！”*
