# 如果你被敲诈了该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-youre-getting-blackmailed-1832462318>

所以你发了裸体照。我们生活在一个数字时代，现在裸照比以往任何时候都更像是约会的货币。



但是当你的前任——或者在杰夫·贝佐的情况下，一个杂志出版商——威胁要在互联网上公布你的照片时，你会怎么做？

这是一个棘手的情况(至少可以这么说),当你几乎无法控制时，很容易感到脆弱，但是有*种方法可以从你的最大利益出发来阻止这种情况发生。*

非自愿色情，或 [报复色情](https://www.cybercivilrights.org/revenge-porn-laws/) ，在大多数州都是犯罪，尽管各地的惩罚不同(如果照片拍摄时你还未成年，这在任何地方都是犯罪)。

我采访了刑事辩护律师杰弗里·利希曼 (是的，也是埃尔·查普的实际律师)，关于如果你因为那些不那么 PG 分级的照片而被勒索时应该采取的步骤，以及如果所有其他方法都失败了该怎么办。

### 记录任何互动

根据李奇曼的说法，当有人为了钱追捕你时，切断所有联系是很诱人的，但这不一定是最好的行动。

“你需要做的第一件事是保存有关勒索企图的一切，”他说。“如果是电话，你会想把它录下来。如果是电子邮件，或者短信，不管是什么，你都需要以某种方式记住。在这条路上的某个时候，你会需要它。”

也不要付一分钱。“你永远不能付给他们任何东西，但你必须让他们说个不停，这样你就有足够的证据证明你对他们有影响力。”

但是为了录制他们的音频，你必须检查你所在州的录音许可法。例如， [加州](https://www.mwl-law.com/wp-content/uploads/2013/03/LAWS-ON-RECORDING-CONVERSATIONS-CHART.pdf) 是一个双方同意的州，这意味着除非双方同意，否则你不能对通话进行录音。然而，大多数州都是一方同意——你可以在不通知另一方的情况下记录通话。

### 找个律师

李奇曼说，现在还不要去报警。“整个想法是尽量避免将此事公开，如果有人因勒索你而被捕，这将是一个公开的事情，很不幸，勒索企图的全部意义将会实现，即使你将此人关进监狱。”李奇曼建议，如果可能的话，找一个刑事辩护律师来帮助你的案子。

“通常，律师会做的是写一封信，展示我们所拥有的，解释这个人的所作所为是非法的，并且你有他们犯罪的证据，”他说。“你的律师会明确表示，他们不怕使用这些证据。现在[...]是不是有人疯到愿意为这种勒索企图拿自己的自由冒险？通常不太可能，但肯定有可能。”

### 联系网站管理员

如果所有这些都失败了，你的照片在互联网上可怕地流传，你可以联系发布照片的网站。

“很多时候，如果它是不合适的或者是被盗的，你可以让网站删除它，”李奇曼说。“如果某人的手机或电脑遭到黑客攻击，通常情况下，你可以让发布该信息的网站删除它，因为基本上，这是犯罪的产物。”

为了联系网站管理员，在网站首页的某个地方找一条“联系我们”的线，上面有电子邮件或电话信息。 [谷歌支持的](https://support.google.com/websearch/answer/4628134?hl=en) 也提供了另一种解决方案:

> 您可以执行 Whois(“谁是？”)使用谷歌搜索网站所有者。去 google.com 搜索`**whois**` [`**www.example.com**`](http://www.example.com) 。联系网站管理员的电子邮件地址通常可以在“注册人电子邮件”或“管理联系人”中找到

或者，你可以使用上面的 Whois 搜索来联系网络主机。一旦你联系上了，解释你的问题，他们应该可以帮助你。否则，如果你有律师，让他们发出的下架通知。这些通知意味着你拥有这些照片的版权，一个请求将迫使他们把它们拿下来。

如果你愿意多付一点钱，你可以使用类似于 [DMCA 保护&撤下服务](https://www.dmca.com/signup/createtakedown.aspx?ref=sol7dcf-top) 的服务，它会代表你直接向网站所有者发出照片撤下通知。

### 什么都不要付

同样，因为这需要重新迭代，所以不要付钱给你的勒索者。也不要屈服于给钱来换取照片不被曝光或被删除的协议的想法。

李奇曼说:“我认为，那些试图通过谈话来摆脱困境、试图通过推理来摆脱困境的人，是最容易受到虐待的人。”。“你需要对他们穷追不舍，他们需要明白，如果这种情况继续下去，你会尽你所能把他们送进监狱。”

如果你想避免照片被泄露，有时候最好不要拍任何照片，尤其是在这个我们的个人数据隐私备受关注的时代。
