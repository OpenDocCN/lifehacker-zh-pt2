> 原文：<https://lifehacker.com/this-font-blurs-profanity-and-prompts-you-to-use-more-i-1844887233>

如果你是一个喜欢骂人的人，你可能会发现自己无意中在邮件中加入了一句脏话——或者至少，你想这么做。根据你在哪里工作，和谁一起工作，这可能会引起不满，甚至会自动触发电子邮件监控系统，如果你的公司有这样的系统的话。帮助遏制(或至少变得更加意识到)这种习惯的一种方法是下载一种模糊诅咒语并促进反种族主义和包容性语言的字体。以下是需要了解的内容。

#### 诅咒控制

这部分很简单。你只需键入一个单词或句子，字体就会自动模糊它认为是侮辱、诽谤或其他攻击性语言的内容。棘手的是，它对冒犯性语言的构成有自己的想法。例如，“他妈的”和“屁眼”是模糊的，而“屎”和“废物”却不是。“灌洗”和“混蛋”都是模糊的。

在其他情况下，它会自动将他们认为具有攻击性的词语替换成其他词语。比如，“母狗”自动变形为“母狗”，“c*nt”变成“阴户”，“公鸡”变成“公鸡”。

#### 包容性语言和反欺凌

从技术上讲，脏话部分只是字体主要目的的一个组成部分，鼓励包容性语言，以帮助防止网络欺凌。有一个强大的反种族主义的组成部分，它改变了任何攻击性的更具包容性的术语。

它还改变单词或短语，使它们更友好。例如，它将“我讨厌你”改为“我不同意你。”而“你很丑”变成了“你没有传统意义上的美”，而“你很笨”变成了“你很傻”。另一方面,“你疯了”,保持原样。
