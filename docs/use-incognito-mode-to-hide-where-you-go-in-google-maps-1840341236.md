# 使用隐姓埋名模式来隐藏你在谷歌地图中的位置

> 原文:[https://life hacker . com/use-incognito-mode-to-hide-where-you-go-in-Google-maps-1840341236](https://lifehacker.com/use-incognito-mode-to-hide-where-you-go-in-google-maps-1840341236)

有很多原因让你想在谷歌地图中追踪你去过的一些地方，但也许不是所有的地方。你可能不想让你的朋友和爱人(可以使用你的电话或账户)知道你去了当地的药房。也许你正在为某人计划一个惊喜聚会，需要去几次场地做准备。你也可能对位置追踪有所怀疑；那也可以。

Watch

隐姓埋名模式，就像你在 Chrome 浏览器中找到的一样，允许你关闭应用程序的位置跟踪功能(暂时)。虽然你可以*也可以* [永远关闭这个](https://lifehacker.com/psa-your-phone-logs-everywhere-you-go-heres-how-to-t-1486085759) ，如果你愿意，我发现隐姓埋名模式是一个很好的妥协。我喜欢知道我去过哪里，但也许谷歌不需要知道我去过的所有地方。

要开始使用隐名模式，请确保您运行的是最新版本的谷歌地图。该功能已经为 [Android](https://play.google.com/store/apps/details?id=com.google.android.apps.maps&hl=en_US) 版本的应用程序推出了一段时间，今天刚刚开始向 [iOS](https://apps.apple.com/us/app/google-maps-transit-food/id585027354) 用户推出。

打开谷歌地图，在右上角寻找你的照片。轻点它。

一旦你这样做了，你应该会看到选项“打开隐姓埋名模式”点击它。

现在你会在屏幕底部看到一个小方框，告诉你隐姓埋名模式是怎么回事。值得一读。

为了更容易看出你正在使用隐姓埋名模式，请再次查找你的照片。取而代之的是一顶有趣的小侦探帽和眼镜，因为你们都是神秘和伪装的(至少对谷歌地图而言)。你的谷歌地图也会在应用程序顶部显示你正在运行隐身模式。
