# 使用这款梵蒂冈认可的应用程序与教皇弗朗西斯一起祈祷

> 原文：<https://lifehacker.com/pray-with-pope-francis-using-this-vatican-approved-app-1831912700>

如果你一直想和教皇一起祈祷，现在有一个应用程序可以实现这一点。

教皇方济各本周末发布了“ [点击祈祷](https://clicktopray.org/)”app。正如它听起来的那样，祈祷应用程序允许来自世界各地的人们(如果他们是祈祷者的话)聚集在一起为某个问题祈祷。

Watch

教皇发布了这款应用，它是教皇全球祈祷网络(祈祷的使徒身份)的官方应用，以配合 1 月 22 日开始的世界青年节。

来自站点:

> 点击祈祷是教皇的祈祷平台，他的全球祈祷网络。这个应用邀请你一起为他的意图祈祷，他的意图是解决当前人类面临的挑战和教会的使命。因为“每一天都是不同的”，该应用程序为你提供了一年 365 天的祈祷，以帮助你发现上帝如何召唤你生活的每一天。加入我们，与教皇一起祈祷，与基督的心联合，我们成千上万的世界各地的人现在形成了一个真正的全球祈祷网络。

该应用程序分为三个部分:与教皇一起祈祷，每天祈祷，与网络祈祷。

教皇计划在“与教皇一起祈祷”部分推出“为教会使命祈祷的请求”。“每天祈祷”版块提供每日祈祷建议，“与网络一起祈祷”版块为人们提供张贴他们自己的祈祷请求的地方。

该应用程序有西班牙语、英语、意大利语、法语、葡萄牙语和德语版本。当你浏览这些网络请求时，你会很快看到一种新语言的帖子。

如果你宁愿不把应用程序放在手机上，也有一个网站，你可以通过手机应用程序做你能做的一切，而不用下载任何东西。
