# 如何培养一个乐观的孩子

> 原文：<https://lifehacker.com/how-to-raise-an-optimistic-child-1839888690>

从政治角度来说，我们的孩子成长在一个完全疯狂的时代。我们已经设法培养了一个又一个他们将不得不应对的危机——阿片类药物流行、自杀率上升、大规模枪击、我们星球的毁灭；我可以继续。总感觉没什么好乐观的。但是如果有一个时候我们都可以乐观一点，那就是现在。

Watch

一个人可能天生倾向于悲观或者乐观 ，但是父母可以做一些事情来鼓励和加强他们孩子(和他们自己)的乐观情绪。

### 停止抱怨

这是一个很好的开始，但是如果你不是天生的乐观主义者，这将是一个挑战。你那台破电脑总是死机，你厌倦了这场雨，这条路上的交通又一次变得一团糟。人生，就是烦。如果你必须发泄，试着把它留给你的伴侣或朋友，在孩子们上床睡觉后。我们是孩子最大的榜样，告诉他们如何看到光明的一面远不如我们自己去实践有效。

试着少抱怨，多赞美。不要打电话给经理抱怨你刚刚上的室温炸薯条，而是打电话给他们称赞你的服务员是多么友好和周到。你的孩子会注意到——你也会感觉很好。

### “大小合适”坏消息

我个人认为这个消息令人震惊。我是个成年人。在媒体工作。因此，对我们的孩子来说，这一定会更加令人困惑和焦虑，因为他们没有我们这样的背景和尺度感来判断事情到底有多糟糕。常识媒体的育儿编辑 Caroline Knorr 为《华盛顿邮报》的写道，我们需要为他们“调整”新闻的大小:

> 当悲剧在世界某个地方发生时，每当我们打开电视，打开社交媒体，查看电话通知，或者走过鼓吹耸人听闻标题的超市报摊时，我们都会重温这一幕。父母明白，媒体为了吸引眼球和点击而放大事物。但是孩子们不一定了解信息来源、赞助商和观众之间的关系。你对新闻的反应会影响孩子对新闻的处理。通过解释最大的声音能吸引最多的听众，帮助你的孩子正确看待事物。

### 教他们“大脑链”

如果孩子不是天生擅长某件事，他们会很快变得消沉。我们常说“熟能生巧！”但是不，大量的练习并不一定能达到完美。然而，练习*会使*进步*。所以下次你听到他们抱怨他们数学很烂或者他们画的猫一点也不像猫，告诉他们他们还在 [构建大脑链](https://lifehacker.com/what-to-teach-kids-instead-of-practice-makes-perfect-1839260441) 。*

每次我们练习某事，我们都在大脑中建立一个新的链条，随着时间的推移，这个链条变得越来越粗，直到它变得如此强大，以至于以前对我们来说很难的事情现在变得更加自动化。你可以给他们各种例子；他们在某项运动上取得了怎样的进步，他们现在的阅读速度比他们刚开始学习时快了多少，甚至他们过去穿衣服有多难，现在他们对此毫不犹豫。这些都是他们日积月累建立起来的链条。

### 记住，乐观不是虚假的希望

这可能违反直觉，但我们不需要为了让孩子乐观地思考而提升他们的自尊。儿童心理学家塔玛·钱斯基是《让你的孩子从消极思维中解放出来》一书的作者，他告诉,《让你的孩子从消极思维中解放出来》一书的作者,《让你的孩子从消极思维中解放出来》一书的作者,《让你的孩子从消极思维中解放出来》一书的作者,《让你的孩子从消极思维中解放出来》一书的作者,《让你的孩子从消极思维中解放出来》一书的作者,《让你的孩子从消极思维中解放出来》一书的作者,《让你的孩子从消极思维中解放出来》一书。这样你的孩子就能为他所面临的一切做好准备。”

这就是为什么承认他们的挑战——就像父母特蕾西·雷纳特 在她家搬家和她的儿子努力结交新朋友时所做的那样——比自动跳到“一切都会好的”要好

> “搬到一个新的地方重新开始很有挑战性，”她解释道。“交朋友需要时间。”之后，Matt 不再抱怨，并积极采取措施解决问题。他让妈妈放学后带他去最近的操场，让他在附近骑自行车去见住在附近的孩子。几周之内，他长出了一些新芽。“他突然意识到事情会解决的，”雷纳特说，“他最终教会了我一两件关于乐观的事情。”

### 鼓励他们独立

为人父母最难的事情之一就是不断地放手。他们开始完全需要你的一切，一点一点地，他们变成完全独立的成年人，不需要你的帮助就能在这个世界上发挥作用。当然，我们知道这是最终目标，但是鼓励他们的独立——甚至是在这个过程中鼓励一些冒险——真的对他们的乐观很有好处。

一个孩子越能自己尝试和征服，他们对自己就越有信心，也越能拥抱可能性。
