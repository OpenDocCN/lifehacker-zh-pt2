# 搅拌瓶是一个很好的鸡尾酒调酒器

> 原文：<https://lifehacker.com/the-blender-bottle-makes-an-excellent-cocktail-shaker-1843967431>

果汁瓶是“健康人”拥有和使用的物品，所以我从来没有花太多时间去想它。但是塑料瓶里的那个小搅拌球注定不仅仅是混合蛋白粉——那个搅拌球是为了酒精的伟大而制造的。



你可能知道， [贝丝·斯克沃茨基](https://kinja.com/bethskw?_ga=2.138428979.1285311211.1591629595-329965069.1585339145) 是一个健康的人。贝丝也是一个喝啤酒的人，她没有鸡尾酒调酒器——但是她*有一个鸡尾酒调酒器，因为她经常举起很多很重的东西，这需要很多蛋白质，所以她喝很多蛋白质奶昔。但是像我们中的许多人一样，贝丝也需要时不时地喝一杯有趣的鸡尾酒，最近她用她的搅拌瓶调制了鸡尾酒中最具挑战性的鸡尾酒之一——拉莫斯杜松子汽酒。*

剧透:它成功了，而且非常成功。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1266531681873334274&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1266531681873334274&autosize=1) 

如你所见，拉莫斯是一种泡沫鸡尾酒。它的顶部有一个泡沫圆顶，搅拌瓶里有一个小搅拌球——专为...打碎蛋白粉什么的，我不知道——做了一个很棒的建造穹顶的工作。它充气并搅动，本质上是将蛋白搅打成蛋白酥皮。

事实上，如果你有一个搅拌瓶，你并不真的需要鸡尾酒调酒器；如果贝丝可以用一个杯子做一杯蓝魔鸡尾酒，你就可以用一个杯子做一杯代基里鸡尾酒、或者任何需要充分摇晃的饮料。如果你在酸溜溜的 里享受一个 [蛋清，那就特别有帮助；那个搅拌球不仅使制造泡沫变得更加容易，而且你实际上可以看到瓶子内部的情况，这可以直观地跟踪泡沫的形成。只需将所有酸味原料加入搅拌瓶中，不加冰摇一会儿(也叫“干摇”)，直到你看到泡沫，然后加入冰，摇至冷却。倒入玻璃杯，就大功告成了。](https://skillet.lifehacker.com/the-sour-is-the-perfect-quarantine-cocktail-1843185396)

那个搅拌球贡献了两样东西——暴力和空气——这就是你制造泡沫所需要的，不管有没有鸡蛋。那个小钢丝球甚至可以帮助无蛋饮料产生美丽的泡沫，就像山姆·比索尼不久前用他的搅打咖啡所做的那样。它还可以帮助你制作任何超清爽和蓬松的饮料。(严格来说，山姆的 frappe 不含酒精，但如果你努力尝试并相信自己，它可能是含酒精的。)

搅拌瓶没有内置滤网，但没关系——你可以买一个。更好的是，用一个筛子过滤你的饮料——它不仅可以用来调制鸡尾酒，而且我非常喜欢多功能的厨房工具。
