# 南瓜冰箱蛋糕是懒人的完美感恩节甜点

> 原文：<https://lifehacker.com/pumpkin-icebox-cake-is-a-perfect-thanksgiving-dessert-f-1839859088>

冰箱蛋糕不是传统的感恩节甜点，但也许应该是。这些蛋糕本质上是一堆鲜奶油和饼干——这是一件好事——而且根本不需要放在烤箱里，这意味着你有更多的空间放砂锅菜。



这款蛋糕 的馅料非常宽容，你可以调整南瓜和香料，以适应你对每一种的渴望。加入果泥，尝一尝，如果你需要更多的季节性口味，可以加入更多，记住蛋糕在冰箱里的时间会减弱一些味道。把它(和姜味饼干)堆在你能找到的任何盘子、盘子或平底锅上，然后让整个“蛋糕”在冰箱里冷藏至少四个小时，直到饼干变软，蛋糕变得可以切片。
