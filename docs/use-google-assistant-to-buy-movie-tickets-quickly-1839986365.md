# 使用谷歌助手快速购买电影票

> 原文：<https://lifehacker.com/use-google-assistant-to-buy-movie-tickets-quickly-1839986365>

Android 用户现在可以让谷歌助手购买电影票并为他们预订座位，这要归功于一项新的谷歌双工功能，该功能使人工智能能够自己访问和导航 Chrome 网页——就像双工人工智能允许谷歌助手代表你打电话给餐馆或酒店预订。



推出时，该功能支持 AMC 和 MJR 影院等 70 多家影院连锁，以及 Fandango 或 Movietickets.com 等购票服务——即使他们没有应用程序，或者他们的网站通常没有针对移动浏览器进行配置。这不仅比直接浏览这些公司的网站更容易，而且可能比自己购买和预订座位更快——这可能有助于你在非常受欢迎的电影《T2》、《T3》售罄之前获得一个座位。

### 如何使用新的机票订购功能:

*   在您的 Android 设备上，打开谷歌助手，搜索附近的电影放映。
*   在结果页面上，点击你想要购买门票的演出时间，一个新的带有购买选项的框将会弹出。

*   轻按您想要使用的服务或网站旁边的 Google Assistant“购买门票”图标。
*   你需要确认你想使用谷歌助手为你买票，然后接受任何网站 cookies。

不像谷歌的 AI 给你打个电话、订个票，买电影票需要多一点输入。Google Assistant 会问你一些问题，比如你想要多少张票，或者要预订哪些座位(如果适用的话)。

在购买门票之前，你还必须确认最终订单和支付信息。尽管如此，这是一个更简单(也更快)的*很多*!)而不是自己滚动浏览多个页面并输入所有这些信息。
