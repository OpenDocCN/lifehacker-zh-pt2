# 制作动画 gif 的完整指南

> 原文：<https://lifehacker.com/the-complete-guide-to-making-animated-gifs-1503276993>

有时，对群聊或电子邮件的最佳回应是一个强有力的 GIF——一个小动画，它可以比你键入的内容更好地表达你的感受。虽然你可以在你最喜欢的消息应用程序中轻敲几下手机键盘，找到大量适合各种场合的 gif，但如果你可以制作自己的 gif，为什么要使用相同的“deal with it . gif”meme 呢？

Watch

你每天捕捉和消费的所有照片和视频，你可能有一个内容宝库，可以很容易地转化为可共享的互联网艺术。你可以用来制作 gif 的工具大多是免费的，比以往任何时候都更容易使用。这里有一些制作 gif 的最佳方法，不管你的技术专长是什么*还是*设备——台式机、笔记本电脑还是手机。

### 一个 GIF 制作者统治所有平台:Giphy

#### **什么事**

Giphy 是一个基于网络的工具，让您为互联网创建和主持简单的动画艺术。你不仅可以将照片和 20 秒以下的视频转换成 gif，还可以将现有的在线视频转换成剪辑(有一些限制)。比如原视频不能超过 15 分钟，也不能违反 [数字千年著作权法](https://support.giphy.com/hc/en-us/articles/360020287091-GIPHY-DMCA-Copyright-Policy) ，或者 [Giphy 的社区准则](https://support.giphy.com/hc/en-us/articles/360020286811-GIPHY-Community-Guidelines) 。

#### **给谁的**

任何想要创建基本的 gif 并且不想担心它的帧速率的人。

#### **平台**

Giphy 适用于任何网络浏览器，所以你可以在任何电脑或移动设备上访问这个 GIF 制作工具。Giphy 也可以作为手机应用程序用于 [Android](https://play.google.com/store/apps/details?id=com.giphy.messenger&hl=en_US) 和 [iOS](https://apps.apple.com/us/app/giphy-the-gif-search-engine/id974748812) 。

#### **它如何在您的台式机或笔记本电脑上工作**

打开 [Giphy](https://giphy.com/) ，点击 [Create](https://giphy.com/create/gifmaker) 开始制作 GIF。如果你已经制作了一张 GIF 图片，并想上传到网上使用和分享，只需点击“ [上传](https://giphy.com/upload) ”。

#### **在手机上如何工作**

从您选择的应用程序商店安装 GIPHY 应用程序后，打开应用程序并点击屏幕底部的绿色“+”符号以创建 GIF。您可以抓拍新照片，或者使用照片库中的照片制作基本的 GIF。您还可以拖放动画贴纸、文本和照片过滤器来增强您的内容。一旦你对你的 GIF 满意了，你可以把它保存到你的设备上或者上传到 GIPHY 上，这样你就可以在网上分享 GIF 了。

### **高级 Android GIF 艺术家:GIF 制作-编辑**

#### **什么事**

GIF Maker-Editor 是一个全面的 GIF 应用程序，可以让你很好地控制你的创作，从调整 GIF 每秒的帧数到定制文件大小。你可以在照片上应用各种有趣的装饰，但贴纸和效果没有竞争对手的应用那么时髦(这里没有小胡子贴纸)。

作为一个免费的应用程序，GIF Maker-Editor 对恼人的广告相对较轻，但它确实将图像的分辨率限制在 448×578 像素。如果你想用 720p 制作 gif，那么你需要升级到专业版(2.99 美元)。

#### **给谁的**

想要对自己的 gif 动脑筋的 Android 用户。停止在你的照片上添加简单的贴纸，加入大联盟——制作一些动画！

#### **平台**

[安卓](https://play.google.com/store/apps/details?id=com.media.zatashima.studio)

#### **工作原理**

使用照片库中的照片或新拍摄的照片和视频，只需点击应用程序主屏幕上的任何矩形按钮，即可开始 GIF 制作之旅。(应用程序的圆圈图标链接到你完成 gif 的文件夹。)

您可以通过点击“图像转换为 GIF”按钮，选择尽可能多的照片，然后点击右上角的“复选标记”,将静态图像转换为精彩的幻灯片。从这里开始，屏幕底部变成了一个侧扫菜单，让你可以更好地控制你的 GIF。你可以用贴纸装饰 GIF，在 GIF 上涂鸦，微调 GIF 的速度，甚至可以将现有的 GIF 插入到你的新创作中。

当你对你的 GIF 感到满意时，将文件保存在你的设备上或者发布到你选择的社交媒体平台上。如果你有一个现有的视频或者想拍摄一些新的内容来转换成 GIF，这个过程是一样的。

这个应用程序也可以很容易地改变你的 GIF。如果你改变了主意，你可以点击“编辑 GIF”或“快速编辑 GIF”按钮来编辑你的文件。你的 GIF 文件太大了，无法在线分享吗？您可以点击“压缩 GIF”或“编辑 GIF”来调整分辨率，从而进一步压缩 GIF 文件。

### **对于高级 iOS GIF 美工:ImgPlay**

#### **什么事**

当你创建下一个 GIF 时，免费的 ImgPlay 应用程序为你提供了许多选项，从它每秒可以显示的帧数(高达 30fps)到你可以应用的许多 meme 就绪模板——甚至是视频。虽然不是 iOS 专属的应用程序，但这个版本可以很容易地将你最喜欢的实时照片和回旋镖转换成 gif。

#### **给谁的**

那些对 iPhone 的 stock Live Photo-GIF 转换器(或你为制作 GIF 而创建或下载的任何其他 [快捷方式](https://lifehacker.com/try-out-iphone-automation-with-this-list-of-150-ios-sh-1833153395) )不满意的人。

#### **平台**

[iOS](https://apps.apple.com/us/app/imgplay-gif-maker/id989843523) 和 [Android](https://play.google.com/store/apps/details?id=me.imgbase.imgplay.android)

#### **工作原理**

这款应用程序会自动按照文件类型(视频、照片、实时照片、连拍照片和 GIF)组织设备上已经存在的所有内容，以便您一打开应用程序就可以开始挑选所有相关文件来创建下一个 GIF。如果你想在你的下一个 GIF 中拍摄一些新的图像，你只需要点击应用程序底部的“相机”来启动你的照片会话。

一旦你选择了你想在 GIF 中使用的图像或视频，界面将转换成三个区域。顶部栏显示 GIF 中的所有框架，中间包含您试图编辑的内容，底部菜单显示您可以使用的所有设计选项，包括称为“画布”的便捷模板

不幸的是，这个免费应用程序会在你的 gif 上打上水印，然后你才能在互联网上分享它们，但当你升级到 ImgPlay Pro 应用程序(4.99 美元)时，水印就会消失。

### **在你的电脑上赠送任何东西:ScreenToGif**

#### **什么事**

ScreenToGif 是一个免费的开源程序，用于记录你电脑上显示的任何内容。无论你是在观看你最喜欢的 Twitch 直播，参加网络会议，还是在数字画板上涂鸦，这个小应用程序都可以帮助你将你正在观看的内容转换成 GIF。 [无限可能](https://lifehacker.com/how-to-turn-any-part-of-your-windows-desktop-into-a-gif-1825787640) 。

#### **给谁的**

**Windows** 希望记录自己的数字生活(如史诗游戏大战或 YouTube 视频中的病毒时刻)并与朋友和家人分享的用户。 **Mac** 用户可以给 [GIPHY Capture](https://apps.apple.com/us/app/giphy-capture-the-gif-maker/id668208984?mt=12) 一次尝试访问类似的功能。

#### **平台**

screen toif:[【windows】](https://www.screentogif.com/)。
phy capture:[【MAC】](https://apps.apple.com/us/app/giphy-capture-the-gif-maker/id668208984?mt=12)

#### **工作原理:**

安装最新版本的应用程序后，打开它并选择你想要记录的内容类型:录音机，网络摄像头，(草图)板或(内容)编辑器。比方说，你正在尝试制作西蒙妮·比尔斯(Simone Biles)在 2019 年首次登陆的历史性体操动作的 GIF。你需要做的是:

1.  在电脑上打开您想要录制的视频
2.  打开 ScreenToGif 程序，选择“录像机”
3.  拖动 ScreenToGif 界面，使其位于视频上方。请随意调整框的大小，以便您的视频在矩形框内可见。
4.  将视频排队到你想录制的地方。播放视频。
5.  点击“红色”圆圈(在 ScreenToGif 框架上)开始录制。
6.  点击“蓝色”方块(在 ScreenToGif 框架上)停止，这将打开程序的其余部分，允许您编辑和保存 Gif。
7.  如果你想在互联网上分享你的高分辨率 GIF，你必须把它上传到一个图片托管服务，比如 [Imgur](https://imgur.com/upload) 。否则，保存您的工作，您就大功告成了！

### **对于严肃的 GIF 动作:Adobe Photoshop**

#### **什么事**

Photoshop 是行业标准的照片处理软件。对于想要廉价(或免费)制作 gif 的人来说，这绝对不是一条路，但是 Photoshop 为你提供了许多编辑和优化动画的强大工具。

#### **给谁的**

如果你是一个严肃的摄影师，想要制作一些高分辨率的 gif 来打动，或者你想对你的 gif 的图像质量或视觉效果做一些认真的调整，Photoshop 是你的朋友。

#### **工作原理**

如果你只是想把你正在看的一些有趣的视频或电影制作成 GIF，可以从使用 [VLC 媒体播放器](https://www.videolan.org/vlc/index.html) 从完整的视频文件中提取一个剪辑开始。

1.  在 VLC 打开视频
2.  在视图下，选择“高级控制”。
3.  将播放头放在您想要录制的动作之前大约 5-10 秒。
4.  按下红色的录制按钮。
5.  按视频上的播放。
6.  完成动作录制后，再次按下录制按钮。
7.  在你的电脑上找到剪辑(可能在你的常规“视频”文件夹中)

现在是有趣的时候了。Photoshop 允许您对动画的单个帧做任何事情，就像您对常规图像所做的一样。与其变得太疯狂，不如从制作一个简单的动画 GIF 开始:

1.  在 Photoshop 中，选择**文件** > **导入** > **视频帧到图层…**
2.  在弹出的对话框中，使用左右手柄将剪辑修剪到您想要导入的确切帧。
3.  (可选)使用文本工具或滤镜来编辑和修改单个框架
4.  要导出 GIF，请选择**文件** > **保存为网页格式…**
5.  在这里，您将有许多选项可以调整。您应该做的主要更改是在“GIF”下面的第一个下拉框中选择“Adaptive”如果它还没有被选中。否则，调整你认为合适的，但默认选项应该给你一个体面质量的 GIF。
6.  在底部，在“循环选项”下，确保“永远”被选中。
7.  单击“保存...”给你的动画一个名字，并把它放在某个地方。

就是这样！完成后，你可以把它上传到 [Imgur](https://imgur.com/upload) 或者任何你喜欢的文件托管服务——或者直接把它们放在你发给老板和/或朋友的电子邮件里。

这五种制作 gif 的方法应该涵盖了最基本的需求，但是总会有更新更好的技巧出现。如果你有喜欢的方法、应用或工具，请在评论中告诉我们。
