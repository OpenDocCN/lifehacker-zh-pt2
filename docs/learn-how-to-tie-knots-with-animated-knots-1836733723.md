# 学习如何用“动画结”打结

> 原文：<https://lifehacker.com/learn-how-to-tie-knots-with-animated-knots-1836733723>

[动画绳结](https://www.animatedknots.com/) 是一个了解绳结一切的完美网站:如何打结，它们的用途，以及如何为特定的工作选择最好的绳结。

Watch

该网站列出了 196 种绳结，包括绳子、带子、细绳、电线、鞋带和领带。您可以按类型(如弯曲、末端循环和滑动与抓握)或按活动(如划船、攀爬和装饰)浏览。或者 [按字母顺序浏览所有绳结](https://www.animatedknots.com/complete-knot-list) 。

新手要查一下[系包](https://www.animatedknots.com/package-tying-knot)[系鞋带](https://www.animatedknots.com/shoelace-bow-knot-surgeons)[系后帘](https://www.animatedknots.com/curtain-tie-back-knot)[打结线](https://www.animatedknots.com/underwriters-knot) 。或者学习 [基本绳结](https://www.animatedknots.com/basic-knots) 在各种情况下使用。

对于每一个结，你可以观看一步一步的打结指南，上面有照片和书面说明。您可以像播放动画一样以多种速度播放照片，并水平或垂直翻转。每个结的页面还列出了它的用途、历史、优缺点和技术细节。也有叙述视频和替代结或技术的链接。

该网站还提供技术文章，包括绳索安全详细指南[绳结和绳索、划船、钓鱼和攀爬的术语](https://www.animatedknots.com/terminology) ，以及多种 [指南](https://www.animatedknots.com/rope-materials)[类型](https://www.animatedknots.com/rope-properties)[绳索](https://www.animatedknots.com/which-rope) 。

如果你需要离线使用绳结，你可以花 5 美元在 iOS 应用程序 或安卓应用程序 中获得动画绳结 [。但是如果你能上网，这个网站在手机上也很好用。](https://apps.apple.com/us/developer/grog-llc/id376302652)

该网站没有进入许多精心和愚蠢的方式来打领带，只是 [四个常见的领带结](https://www.animatedknots.com/how-to-tie-a-tie-necktie-knots) 和一个领结。但是如果你在教一个小孩打领带，你应该试试 Lifehacker 的视频。如果你需要打领结，学学保罗·F·汤普金斯的。
