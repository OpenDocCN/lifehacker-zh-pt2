# 接下来我们应该做什么菜？

> 原文：<https://lifehacker.com/what-should-we-casserole-next-1831769527>

大家好，欢迎回到 [*将它砂锅*](https://lifehacker.com/tag/will-it-casserole) ，在这个栏目里我们看到哪些饭菜、菜肴、菜系可以重新想象，改造成砂锅。又是新的一年，这意味着是时候挑选我们 2019 年的第一个砂锅项目了。



鉴于我们正处于 [冬季](https://jezebel.com/bad-winter-starts-now-1831436102) 最糟糕的部分，我们需要一个舒适的砂锅，让我们感觉像在吃一个拥抱。我们需要从里到外温暖我们的东西，并且在最后一口时点燃快乐的火花(以防你有冲动去近藤你的冰箱)。一如既往，我有想法:

*   **烤奶酪&番茄汤:**我们可以用几种不同的方法来做这道菜，但我想象有很多硬皮面包，很多不同的奶酪，还有一个简化的番茄汤。
*   **法式洋葱汤砂锅:**和上面的相似，但是用了大量的焦糖洋葱，并且用牛肉代替了番茄(很明显)。
*   **拉面砂锅:**很多个月前就有人推荐这个了，是个常青树的创意。蘑菇、各种蔬菜和海藻可能会出现，或者我们可以走 [罗伊·蔡](https://cooking.nytimes.com/recipes/1016583-perfect-instant-ramen) 路线，加入美国奶酪。

这些我都完全可以接受，但我很想听听你想和全班分享的任何创意砂锅概念——越安慰越好。让我眼花缭乱！
