# 使用该网站快速创建您自己的自定义表情符号

> 原文：<https://lifehacker.com/quickly-create-your-own-custom-emojis-with-this-website-1840114297>

表情符号可以是推文或信息的有趣补充，但正确的表情符号并不存在。 [表情符号制作者](https://jamstack.studio/emoji) 是一个简单的工具，可以提供帮助。

Watch

该工具提供了大量不同的选项来从头开始构建自己的表情符号。例如，你可以创建一个惊讶的小丑或哭泣的便便表情符号。它的 [类似于表情符号生成器](https://lifehacker.com/make-your-own-custom-emoji-using-this-site-1830634983) ，但是有一个更简单的界面。

建立一个包括挑选一个空白的脸模板，然后添加一些东西，比如眼睛、嘴或者眼泪，让它真正成为你的。

选项都是内置表情库中已经有的标准内容；然而，该工具允许您以独特和不同的方式将它们放在一起。

不幸的是，一旦你完成了，你唯一的选择就是下载你的作品作为图像，然后以这种方式分享，而不是把它添加到表情键盘上。尽管如此，这仍然是一件有趣的事情。
