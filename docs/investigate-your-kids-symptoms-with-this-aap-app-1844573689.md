# 使用此 AAP 应用程序调查您孩子的症状

> 原文：<https://lifehacker.com/investigate-your-kids-symptoms-with-this-aap-app-1844573689>

鉴于这些天除非绝对必要，否则我们都有点犹豫是否要去医生办公室、医院或紧急护理，随着流感季节的临近，我们继续乘着疫情的浪潮，有一款应用程序可能会让你方便使用:[The kids doc app](https://apps.apple.com/us/app/kidsdoc-from-the-aap/id373964536)。

Watch

KidsDoc 应用程序由美国儿科学会创建，是一个症状索引，你可以通过字母列表、身体区域或关键字搜索来搜索。当您点击一个症状时，例如耳朵充血、头痛或眼睛肿胀，您会获得更多信息，更详细地解释该症状以确认准确性(加上可能更合适的其他症状的列表)。从那里，你可以了解潜在的原因，下一步该做什么和整体护理建议。您也可以将信息通过电子邮件发送给自己(或其他人)。

您可以为每个孩子创建迷你档案，记录他们常用的药物、剂量、处方号和其他注释。还有一个“急救”部分，你可以在那里查找如何治疗各种伤害或疾病，如烧伤或割伤。该应用程序甚至存储了您的孩子可能使用的常见非处方药的剂量表，包括对乙酰氨基酚、布洛芬和苯海拉明(苯海拉明)。

正如该应用的免责声明正确指出的那样，你在该应用上找到的指南是一个起点，而不是诊断和治疗你孩子疾病或伤害的全部。

> 没有什么职业指南是完美的。因此，在使用 KidsDoc 时，您需要运用自己的最佳判断和常识。你要对你孩子的健康负责。做出安全谨慎的决定。如果有疑问，打电话给你的医生。如果你认为你的孩子有紧急情况，打 911 或者去急诊。

该应用售价 1.99 美元，适用于 [iOS](https://apps.apple.com/us/app/kidsdoc-from-the-aap/id373964536) 和 [Android](https://play.google.com/store/apps/details?id=com.gss.android.KidsDoc&hl=en_US) 。
