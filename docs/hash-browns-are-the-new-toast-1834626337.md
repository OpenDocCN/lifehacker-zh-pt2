# hash browns are the new toast 1834626337

> 原文：<https://lifehacker.com/hash-browns-are-the-new-toast-1834626337>

说到早餐土豆，我有很多想法和感受。说到薯饼，只有两种可以接受的形式。我的意思是，我不相信有“坏”土豆这种东西，但如果你要声称你的菜单上有“土豆煎饼”，它们最好是切碎的、金黄色的，或者是像快餐一样的肉饼。这些是薯饼的规则，而且很死板。

在家里，我喜欢把土豆饼放在冰箱里，因为我发现它们让我的早晨更加可爱。除了搭配早餐三明治，它们还是各种早餐食品的绝佳配送系统。如果你想把它放在吐司上，你*应该*把它放在一个棕色肉饼上，在我看来，这样会更好。



我有什么建议吗？我当然知道。几乎任何东西都比剁碎和油炸的土豆要好，但这些是我最喜欢的土豆泥配料:

### **Labneh +除了百吉饼调料以外的一切**

这是我个人最喜欢的一种——浓稠、扑鼻的酸奶，带有咸味和美味。

### 甜椒奶酪+香葱

一个不错的午餐选择。你可以用任何洋葱绿色的东西代替细香葱。

### 本尼迪克特浇头的任何配置

荷兰酸辣酱和荷包蛋是必不可少的，但享受第三个组成部分。加拿大培根是经典的选择，但菠菜、熏鲑鱼或一堆切成薄片的火腿也非常美味。



### 奶油芝士+烟熏三文鱼

加入刺山柑和切碎的红洋葱可以加分。

### 培根+鸡蛋+奶酪

在上面再放一层土豆泥，就可以做三明治了。

### 熏牛肉+煎蛋+炒韭菜

这是一个你可以捡起来的杂凑。玉米牛肉也是一个好主意(显然)。

### 鳄梨

我的意思是，如果你要像对待烤面包一样对待一些东西，你可能会一直这样下去。



至于土豆煎饼本身，只需按照包装上的制作说明——我的是在烤箱里做的——让它们冷却几分钟，然后加入配料，特别是以乳制品为基础的配料，如 labneh、奶油奶酪和甜椒奶酪。虽然融化的甜椒奶酪实际上是一种不错的，所以也许你不需要等待那一个。
