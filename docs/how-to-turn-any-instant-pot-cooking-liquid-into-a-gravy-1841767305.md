# 如何把任何即时锅煮液体变成肉汁

> 原文：<https://lifehacker.com/how-to-turn-any-instant-pot-cooking-liquid-into-a-gravy-1841767305>

速食锅对很多东西都有好处，但我主要用我的来煮大块的有很多结缔组织的硬肉。该设备使它们从骨头上脱落，但是，由于它是密封的烹饪环境，这不可避免地在插入物的底部产生大量美味但非常稀薄的液体。



把它简化成釉料是你可以探索的一条途径，但这需要一段时间，我没有耐心。减少也意味着强化口味，这可以转化为过咸或过甜的酱汁。但是，我的主要问题是需要多长时间，特别是当你只用两种配料就能把烹饪液体变成美味的肉汁时。

实际上，这里有两种双成分方法。你可以做一份玉米淀粉浆(两份冷水+一份玉米淀粉)，或者你可以做一份法式和花式玉米淀粉酱，也就是“生的 T4 面糊”虽然这听起来有点冒险，但你所要做的就是将等量的黄油和面粉捣碎，直到你得到均匀的糊状物。通常每种一汤匙就足以使食谱中的 jus 变稠，但我通常会将量增加三倍，并将多余的部分放在冰箱里，直到需要时再用。

如果你用的是玉米淀粉浆， [将液体煨开](https://skillet.lifehacker.com/cornstarch-does-its-best-work-when-its-hot-1841311617) ，滴入一点你的淀粉混合物，降低温度并搅拌。调味汁冷却后会变稠，所以每次加一点点。(你可以用勺子舀一点到冷盘上，看看它是如何凝固的。)如果你用的是*奶油面糊，*掰下小块面糊，放进去，搅拌融化黄油。面粉会均匀分散，把水状液体变成可以用勺子舀的东西。你可以想加多少就加多少，做成稀沙司或粘稠的肉汁来配你的美味肉片。
