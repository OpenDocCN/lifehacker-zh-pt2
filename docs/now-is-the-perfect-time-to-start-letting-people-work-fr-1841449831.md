# 现在是让人们在家工作的最佳时机

> 原文:[https://gizmodo . com/now-is-the-perfect-time-to-start-let-people-work-fr-1841449831](https://gizmodo.com/now-is-the-perfect-time-to-start-letting-people-work-fr-1841449831)

虽然在家工作对一些人来说是一种奢侈，但对另一些人来说却是必须的。全球目前正在见证其“最大规模的在家工作实验”，正如彭博所说的，由于武汉冠状病毒的快速传播，武汉 T5 比 2002-03 年非典爆发时杀死的人还多。中国各地的许多企业正迫使员工呆在家里，以保护他们的健康，防止病毒传播。对于习惯于面对面集思广益会议的团队来说，这可能是一个挑战，但远程工作有可能通过不仅仅是遏制武汉病毒来保护公众健康。

这是因为远程工作是减少温室气体排放的最简单、最具成本效益的方式之一。和冠状病毒一样可怕的是， [气候](https://earther.gizmodo.com/the-climate-crisis-may-have-helped-spawn-massive-locust-1841385871) [变化](https://earther.gizmodo.com/miami-is-fucked-1841431975) [是](https://earther.gizmodo.com/goosebumps-researchers-capture-first-video-from-under-1841311460) [另一个](https://earther.gizmodo.com/a-heat-wave-has-fire-danger-rising-again-in-australia-1841354142) [巨大的](https://earther.gizmodo.com/women-face-pervasive-violence-in-the-face-of-the-enviro-1841333396) [问题](https://earther.gizmodo.com/miami-is-fucked-1841431975) 已经盯上了最脆弱的。对抗它需要改变我们生活的方方面面，包括我们离开家去办公室的频率。

事实上，美国政府在 2000 年就开始要求员工远程工作，部分原因是为了让疫情的系统到位，全球工作场所分析公司(Global Workplace Analytics)总裁凯特·李斯特向 Earther 解释说，这是一家帮助促进远程工作实施的研究和咨询集团(注意，武汉病毒尚未达到疫情病毒的水平，但肯定会很快达到 T5 病毒的水平)。然而，最近几年，联邦政府一直将远程办公视为减少个别机构温室气体排放的一种方式。现任总统的使命是破坏这一进程，但那是另一回事。

在家工作的碳效益很大程度上取决于一个人如何去上班。如果你像我一样坐火车去上班，呆在家里并不能减少你的温室气体排放。如果你是 76%独自开车上班的美国人中的一员，那么每周有几天呆在家里可以大大减少你个人的碳足迹，同时也减少了路上太多汽车造成的拥堵和污染。

在像美国这样一个痴迷于汽车的地方，交通运输占全国温室气体排放量的的最大一块，占 28.9%。提高燃油效率和改善公共交通往往被誉为解决方案，但它们需要投资。这当然不一定是一件坏事，但是如果工作允许的话，员工一周在家呆一两次几乎不需要花费什么。来自全球工作场所分析公司的研究 [显示](https://globalworkplaceanalytics.com/brags/news-releases#FJ) 虽然数据显示远程办公在过去十年中增加了 115 %,但只有 7%的雇主为大多数员工提供这种选择。如果我们要拯救世界，这还不够。

同一份报告还显示，即使是这一相对较小的远程工作者群体——美国约有 390 万远程工作者——每年也能减少相当于 60 万辆汽车的温室气体排放量。发表在 2018 年 杂志上的另一项研究 [发现，在芝加哥，远程办公每天减少约 695 吨温室气体排放和 367 磅颗粒物，这是一种可能导致肺部和心脏问题的危险空气污染物。](https://www.sciencedirect.com/science/article/pii/S1361920918301226#!)

“我们需要这种政策，如促进公共交通，促进积极的交通，如自行车、踏板车等——当然还有远程办公——作为一种廉价的政策，我们可以实施，同时获得有希望的结果，”研究作者 Ramin Shabanpour，伊利诺伊大学芝加哥分校土木工程的访问研究助理教授，告诉 Earther。

然而，在家工作并不完美。科学仍然没有密切关注这些好处是如何被增加的家庭取暖和制冷需求或在工作日跑来跑去做杂事抵消的。Shabanpour 的研究模拟了一些场景，但仍有更多的分析需要完成。

“总的来说，在家工作可以减少上下班的交通，特别是如果这种旅行是通过汽车进行的，在家工作可以减少通勤的碳排放，”滑铁卢大学的副教授马库斯·莫斯(Markus Moos)在给 Earther 的一封电子邮件中写道。“然而，这项关于在家工作的研究还发现，人们在家工作时会选择新的旅行方式，所以净效应仍然有些模糊。”

在理想情况下，积极因素足以鼓励雇主制定灵活的工作政策。然而，李斯特说，省钱才是真正的推动力。幸运的是，如今经济和环境齐头并进，因此企业也看到了设定和实现内部可持续发展目标的财务效益。这些在家工作的措施可能会吸引投资者(和消费者！)对公司的环境和社会影响感兴趣。

“我走进一些公司，对他们说，‘嘿，我们可以用一种非常简单、容易、受欢迎和便宜的方式来满足你们的减排要求。这是一个真正的卖点。"

员工也喜欢这个选项。我每周至少在家工作两次。如果我感冒了，如果我进来，我的同事会很生气。有时候，我穿着睡衣工作。太棒了！ [研究发现](https://globalworkplaceanalytics.com/resources/costs-benefits) 给员工提供选择权可以提高他们的生产力水平，让他们更开心，并提高员工的忠诚度。因此，当公司实施远程工作政策时，也有助于节省开支。

在中国，面临尝试这种工作实践任务的雇主可能会意识到，他们其实真的很喜欢这种工作。考虑到中国许多人在家工作的原因，这些感觉可能有点复杂，但它也可以提供一些关于远程工作的气候好处的重要教训。中国是世界上最大的温室气体排放国，因此希望总部设在中国的公司能够更深入地了解这种在家工作的实验如何成为一种更持久的减排措施。

都柏林工程咨询公司 Arup 的土木工程师 Paul O'Keef fe 对这些好处进行了研究，他告诉《地球》杂志说:“在很多情况下，当好事发生时，那是因为坏事。”。“人们对此的反应表明，变化可能是好的，也可能是可能的。”
