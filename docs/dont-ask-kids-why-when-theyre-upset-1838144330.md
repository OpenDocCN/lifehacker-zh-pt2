# 当孩子们不开心的时候，不要问他们“为什么”

> 原文：<https://lifehacker.com/dont-ask-kids-why-when-theyre-upset-1838144330>

在过去的五分钟里，你的孩子在做数学作业时一直在小声嘟囔；最后，他明显沮丧地摔下铅笔。“你为什么变得如此沮丧？”你可能会问。或者你的学龄前儿童冲下楼梯奔向你，泪水顺着她的脸颊流下；“你为什么难过？”你问。

Watch

我们的意图是好的。我们看到我们的孩子在处理一种困难的情绪，我们想知道是什么导致了这种情绪(因此，我们可以如何帮助修复它)。但是，根据戈特曼研究所 的说法，我们非但没有找到他们问题的根源，反而可能实际上让他们处于守势。

[在这篇文章](https://www.gottman.com/blog/kids-not-listening/) 中，斯蒂芬妮·卢米斯·帕帕斯写道，我们应该停止问孩子“为什么”他们此刻感到沮丧，这个想法在阿黛尔·费伯和伊莱恩·马兹利什的书《 [如何说话》中有所概述，这样孩子们就会听话了...而且听着这么会说话的小孩](https://www.amazon.com/How-Talk-Kids-Will-Listen/dp/1451663889/ref=tmm_pap_swatch_0?_encoding=UTF8&asc_campaign=InlineText&asc_refurl=https://lifehacker.com/dont-ask-kids-why-when-theyre-upset-1838144330&asc_source=&qid=&sr=&tag=kinjalifehackerlink-20) :

> 尽管有些孩子可以解释他们当时的感受，但许多孩子不能。对于那些孩子来说，问为什么只会让事情变得更糟:除了他们最初的痛苦，他们现在必须分析原因，并提出合理的解释。孩子们常常不知道他们为什么会有这样的感觉。在其他时候，他们不愿意告诉你，因为他们担心，在成年人眼里，他们的理由不够充分。(“你在哭吗？”)

通过问*为什么？我们让他们处于尴尬的境地，潜在的信息可能是他们需要向我们证明他们的感受。*

想想当一个朋友打电话给你，听起来很悲伤的时候，你用的词。“你听起来很沮丧，”你可能会说，或者，“听起来你过了*天*。”我们通常不会直接跳到“你为什么生气？”即使我们问了一个问题，我们也倾向于问更多关于情况的问题(“发生什么事了吗？”或者“怎么回事？”)而不是关于*为什么*他们会有这样的感觉。他们有那种感觉是因为那是他们的感觉。

我自己也为此感到内疚。我平常的“怎么了？”很可能是，“你看起来很沮丧。”当他们沮丧地放下铅笔时，我们可以更谨慎地选择措辞:“看来你今天的家庭作业额外具有挑战性。”这让他们对自己表达的感觉有了肯定，同时也减轻了谈论这种感觉的压力。(“这么大的数相乘好难啊！”)

当我们的孩子处理棘手的情绪时，更少的“为什么”实际上可能会让我们更容易接近，并帮助他们感到更多的倾听和理解。
