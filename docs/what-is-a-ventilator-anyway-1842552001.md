# 到底什么是呼吸机？

> 原文：<https://lifehacker.com/what-is-a-ventilator-anyway-1842552001>

众所周知，世界各地的医院都缺乏呼吸机——由于冠状病毒而导致呼吸衰竭的患者可能无法获得呼吸所需的帮助。



与感染总人数相比，需要积极呼吸帮助的新冠肺炎患者人数仍然相对较少，但美国和其他国家正在准备定量供应机器，要求制造商加快生产，甚至用水肺面罩制造临时呼吸器。一些医院正在想办法让多个病人共用一个呼吸机。

那么什么是呼吸机，你需要多严重的病情才需要呼吸机呢？

### 通风机 101

一个健康的人可以以正常的速度吸入氧气和排出二氧化碳，以保持身体的功能。但是一个患有疾病或肺部受损的人不能在正常的空气中快速或用力呼吸以支持他们身体的氧气需求。呼吸机通过产生压力打开肺部，通过管道将氧气输送到肺部，并清除肺部产生的二氧化碳来完成这项工作。

当你呼吸时，你的横膈膜和胸部的其他肌肉收缩，这使你的胸部扩张，并产生负压，将富氧空气吸入肺部。当这些肌肉放松，胸部恢复正常尺寸时，你呼出二氧化碳。如果你的身体不能自己产生负压，可以使用呼吸机来产生正压——将空气推入。

在重症监护环境中使用的机械通气机是靠电力运行的大型复杂设备，并且连接到中央氧气源。该机器的主要部分是一个监视器，带有刻度盘、显示器和设置选项，可以根据每个患者的呼吸需求进行调整。

外壳内部是一个风箱，就像你在电视上看到的救护人员使用的急救袋一样，推动含氧空气通过一系列管道进入肺部。

在空气到达病人体内之前，它会先通过一个加湿器，加湿器会对空气进行加热和加湿。从那里，含氧的空气被推动通过一个管道，将它输送到肺部，并去除产生的二氧化碳。

接受机械通气的患者通常被插管——这意味着他们有一根管子通过他们的嘴插入他们的气道。一旦气管内导管就位，末端就会膨胀以封闭气道，从而防止任何气体或液体在空气被推入肺部时向后泄漏。

插管的病人在机械通气的部分或全部时间里也至少被轻度镇静。除了喉咙里插着管子的不适感之外，它还会让你感觉自己像是被 [窒息、窒息或溺水](https://pdfs.semanticscholar.org/bc2c/bb841d453fd438e3b3360e07b9e7573f1101.pdf) 。

有一些侵入性较小的呼吸辅助方式，例如通过面罩输送氧气，但对于新冠肺炎患者来说，补充氧气可能还不够。

佐治亚州萨凡纳市的麻醉师 Brian Redmond 博士说，新冠肺炎会影响呼吸系统，所以已经有肺病的人已经落后半步了。吸烟者、老年人和其他呼吸系统疾病患者的肺部效率较低，可能会受到病毒导致的呼吸问题的更大打击。

### 为什么呼吸机短缺？

当没有全球疫情的时候，仍然有病人在使用呼吸机。但是这些人中的许多人只需要几个小时(例如手术期间)或几天的机器，然后它们就被传递给下一个病人。呼吸衰竭的新冠肺炎患者通常需要一周或更长时间的机械通气，因此机械通气的速度不同。

“出现瓶颈的原因是因为有太多的病人，当他们继续工作时，他们会呆很长时间，”雷德蒙说。

此外，呼吸机并不是一劳永逸的设备。它需要不断的监测和频繁的微调，以平衡每个患者的压力、容量、氧气百分比和呼吸频率——这意味着让很多人使用呼吸机会加重已经繁忙的医院、护士和医生的资源负担。

新冠肺炎面临的另一个障碍是，医务人员 [每次与病人互动时，都必须穿着个人防护装备](https://www.cdc.gov/coronavirus/2019-ncov/infection-control/control-recommendations.html)——罩衣、口罩、手套、护目镜。雷德蒙说，这意味着他们不能只是走进去转动一个转盘来调节机器。插管时也有暴露于病毒的风险。

长期使用机械通气的患者也有风险，但雷德蒙说这不是冠状病毒的主要担忧。

“我们只是想让这些人活下来，”他说。
