# 使用你的椅子进行家庭锻炼

> 原文：<https://lifehacker.com/use-your-chair-for-a-home-workout-1844574724>

欢迎来到本月最后一篇健康挑战文章。我希望你已经享受了这些 [迷你](https://vitals.lifehacker.com/get-up-and-do-some-arm-circles-1844420029?_ga=2.107776386.1248358329.1595879433-1985951192.1595879433) [从你的一天中休息](https://vitals.lifehacker.com/get-out-of-your-chair-and-do-some-downward-dogs-1844493696) ，尽管它们有时有点傻。今天，我有一个更愚蠢的挑战:把除了屁股以外的东西放在椅子上。



例如:往前挪，让你的屁股离开椅子，手放在座位上。现在你可以做，一个极好的三头肌训练。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-HCf97NPYeGY&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-HCf97NPYeGY&start=0) 

为了更进一步，你们当中有健身雄心的人可以伸直手臂，将屁股离开椅子，直接悬停在座位上方。(如果你的椅子比较窄，你可能需要两把椅子挨着，给自己足够的空间。)

现在抬起你的脚。如果你非常强壮，你可以做一个 L 型坐姿，把你的腿伸向前方。如果你还没有达到目标——我也没有——那就弯曲双腿，当你变得更强壮时，试着伸直双腿。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-5yL9v36Z2W0&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-5yL9v36Z2W0&start=0) 

别忘了椅子也是俯卧撑的完美平台。如果你还不擅长在地板上做这些动作，把你的手放在椅子上让这个动作更容易；用这种方法，你仍然可以练习在平板姿势中保持身体挺直——比用膝盖做俯卧撑好得多。

或者，如果你很擅长俯卧撑，把这个翻过来。把你的*脚*放在椅子上，做抬脚俯卧撑。你可以保持你的身体在一个平板的位置，或者通过进入一个屈体姿势开始做倒立俯卧撑，上半身垂直。或者 [像这个](https://www.youtube.com/watch?v=mz0TeyqDg7E) 一样，双手绕着椅子走，而不是做俯卧撑。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-mz0TeyqDg7E&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-mz0TeyqDg7E&start=0) 

可能性是无限的。椅子也可以很好地用于踏步，或者在健身房需要长凳或箱子的任何运动。只需 [确保你选择的椅子适合运动](https://lifehacker.com/how-to-prevent-injuries-when-youre-working-out-at-home-1843106512)——在有轮子的办公椅上做箱型跳跃可能不太好。哎哟。
