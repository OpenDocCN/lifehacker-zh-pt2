# 在去外国之前学习这两个单词

> 原文：<https://lifehacker.com/learn-these-two-words-before-visiting-a-foreign-country-1831543906>

在你下一次大型国际旅行之前，你可能已经下载了，但是在去任何一个说外语的地方之前，有两个重要的词需要知道。

Watch

现在，你可能在想那些显而易见的术语:“你好”和“再见”；“请”和“谢谢”；“救命”和“迷路”

但是没有！

我说的是“推”和“拉”这两个词——这两个词会让你在走近一扇简单的门时不显得像个傻瓜。(同样，当你去公共浴室的时候,“男人”和“女人”这两个词会让你免于尴尬，但这些词通常都配有有用的象形图。)

在许多语言中，这些单词惊人地相似——即使在英语中，它们都是四个字母，以“P”和“U”开头，所以认为你可以通过视觉分辨出哪个是“push”和“pull”是行不通的。

因此，我给了你一个在 Duolingo 上最流行的语言中使用“push”和“pull”的指南:

*   **法语**:推= *推*，拉= *拉*
*   **西班牙语** : Push = *Empujar* ，Pull = *Tirar*

*   **德国**:推= *按*，拉= *拉*

*   **意大利语** : Push = *Spingere* ，Pull =*tirate*

*   **土耳其语**:推= *推*，拉= *拉*

*   葡萄牙语:推= *推*，拉= *拉*

*   **瑞典文**:推= *推*，拉= *拉*

*   **Norwegian** : Push = *Trykk* , Pull = *Trekk*

*   **冲**:推= *推*，拉= *拉*

所以现在你不会是那个在挪威应该徒步旅行而去 T2 旅行的游客了。
