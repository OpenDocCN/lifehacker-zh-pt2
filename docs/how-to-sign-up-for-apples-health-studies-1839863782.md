# 如何报名参加苹果健康研究

> 原文：<https://lifehacker.com/how-to-sign-up-for-apples-health-studies-1839863782>

今年秋天早些时候，苹果公司宣布，他们将与大学合作，推出关于听力、心脏健康和生殖健康的三项新研究。现在， [有一个应用程序可以让你注册](https://apps.apple.com/us/app/id1463884356) 。

Watch

这款应用名为 [苹果研究](https://apps.apple.com/us/app/id1463884356) ，你必须更新到 OS 13.2 才能安装。除了三项新的研究，该应用还显示了苹果心脏研究的更新，其 [结果](https://www.nejm.org/doi/full/10.1056/NEJMoa1901183) 最近发表。

通过 app，我报名参加了听力学习。该应用程序给出了研究的简要描述和要求，然后在你输入你的姓名和联系信息后，你会看到一份知情同意书。这更详细地解释了研究的目的、对您的要求、研究将持续多长时间以及参与的风险和益处。

如果你曾经参与过一项研究，这是很熟悉的。您可以随时停止参与。尽管苹果和他们的合作伙伴承诺保护你的数据安全，但每项研究都会告诉你，数据泄露的风险总是非零。也有了解你健康状况的风险。例如，如果参与研究让我思考并担心我的听力，我可能会想去检查一下我的听力，这可能会导致我花钱或暴露健康问题。

与此同时，尽管这些数据将被用于开发产品，但这项研究不向参与者支付报酬，我们也没有获得任何利润。

该应用程序随后请求访问健康应用程序收集的关于环境声音水平、耳机噪音水平和噪音通知的数据。

当你加入一项研究时，应用程序会向你呈现“任务”第一天，它要我回答关于自己的人口统计学问题，做一个两分钟的“研究设置”任务，并进行 10 分钟的初步调查。还有一个跑马灯，让我知道我参加这项研究有多长时间了(一天)。根据我被分配到研究的哪一部分，我要么会收到通知，要么不会，当我的手机检测到我暴露在噪音中时，会提示我查看我的噪音水平数据。

你也可以随时重新阅读同意书，该应用程序会将你链接到隐私政策和关于该研究的常见问题。我觉得自己见多识广，而且这项研究的任务似乎不会引人注目(只要我一直戴着手表，不与他人分享我的账户，而他们要求你不要这样做。)如果你有兴趣帮助科学(和苹果)了解你的健康，这是一个相当顺利和方便的参与研究的方式。
