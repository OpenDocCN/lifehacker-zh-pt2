# 现在是献血的好时机

> 原文：<https://lifehacker.com/now-is-a-good-time-to-donate-blood-1844120751>

美国目前正面临严重的血液短缺。由于对冠状病毒的担忧，许多献血活动被取消，也许是因为献血者 T2 感觉 T3 对报名尤为犹豫。虽然在封锁期间对血液的需求有所减少，因为许多选择性手术被推迟，对血液的需求已经恢复，但是献血还没有回到疫情之前的水平。



“我们的库存减少了一半，”美国红十字会生物医学服务总裁克里斯·赫鲁达告诉纽约时报 。“我们开始进入一个危急的局面。”根据 Hrouda 的说法，红十字会通常有五天的供应量，现在只有不到两天的供应量。

如果你有机会献血，现在是献血的好时机。为了确保献血者的安全，诊所正在采取额外的安全预防措施来防止感染。包括红十字会在内的许多诊所经营者也采取了额外的步骤 [筛查捐献者是否存在抗新冠肺炎](https://www.redcross.org/about-us/news-and-events/press-release/2020/red-cross-announces-testing-of-all-blood-donations-for-covid-19-antibodies.html) 抗体。通常在 7 到 10 天内，捐献者就可以得到抗体测试的结果。

(快速提醒:新冠肺炎抗体是过去被感染过的迹象，而不是活动性感染的迹象。如果你感觉不舒服，或者你最近接触过新冠肺炎病毒，你就不应该献血。)

### 谁有资格捐赠

一般来说，任何 17 岁以上，体重至少 110 斤的健康人都有资格献血。你可以每 56 天捐献一次全血。仅捐献血浆或血小板也是一种选择。

在您的捐赠地点可能会有一些额外的限制，所以请查看您当地的小中心的指南。使您不合格的常见因素将是低铁水平，如果您正在服用某些药物，或者如果您曾在特定国家居住或旅行过。美国食品和药物管理局最近放宽了一些反对同性恋和双性恋捐赠者的规定，以及旅行限制。

取决于你住在哪里，可能会有很多选择。M 由于需要保持适当的安全预防措施，包括体温检查、限制预约次数以确保足够的物理距离以及增加卫生预防措施(带上口罩 ) ，提前预约将尤为关键。
