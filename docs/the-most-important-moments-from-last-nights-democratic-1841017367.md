# 昨晚民主党辩论中最重要的时刻

> 原文：<https://lifehacker.com/the-most-important-moments-from-last-nights-democratic-1841017367>

昨晚，六名总统候选人在爱荷华州的得梅因为 2020 年 race⁠—and 的第七场民主党辩论登台，事情变得尴尬起来。在辩论中，两位候选人讨论了与伊朗日益紧张的关系、气候变化和医疗保健等问题。然而，任何人都可以谈论辩论后伊丽莎白·沃伦和伯尼·桑德斯在当晚快结束时的交流(以及汤姆·斯泰尔，他在辩论后成为了 [迷因](https://twitter.com/oneunderscore__/status/1217300549151920130) )。以下是昨晚民主党总统候选人辩论中一些最精彩和最糟糕的时刻，以防你错过了所有(令人不快的)行动。



### 伯尼·桑德斯否认曾告诉伊丽莎白·沃伦女性不可能赢得选举

本周，所有的目光都聚焦在伯尼·桑德斯身上，伊丽莎白·沃伦在的一份声明中证实了 [桑德斯曾向她表达过他不相信一个女人能赢得总统大选。在辩论中，桑德斯否认说过这话。然而，沃伦站出来抗争，并对关于女性*可能*获胜的观点加倍下注。“我认为谈论谁能赢的最好方法是看人们的获胜记录，”她说。“那么一个女人能打败唐纳德·特朗普吗？看看这个舞台上的男人。他们总共输掉了 10 场选举。在这个舞台上，唯一赢得过所有选举的人是女性...艾米和我。”](https://twitter.com/KristenOrthman?ref_src=twsrc%5Etfw%7Ctwcamp%5Etweetembed%7Ctwterm%5E1216879913078329344&ref_url=https%3A%2F%2Fwww.usatoday.com%2Fstory%2Fnews%2Fpolitics%2Felections%2F2020%2F01%2F13%2Felizabeth-warren-says-sanders-didnt-think-woman-could-president%2F4461106002%2F)

### 皮特·布蒂吉格回应了关于他缺乏黑人选民支持的问题

在竞选的大部分时间里，皮特·布蒂吉格未能赢得黑人选民的支持。(根据最近《华盛顿邮报》/益普索的调查 ，在接受调查的民主党黑人选民中，蒂吉格仅获得 2%的支持，在主要民主党总统候选人中排名第二。)当被问及他在黑人选民中的地位时，Buttigieg 回答说:“最了解我的黑人选民都支持我，”他说。“这就是为什么我在南本德得到了最多的支持。这就是为什么在我的社区中参与竞选的当选黑人官员中，大多数人都支持我。”

### 乔·拜登讨论了他是否会会见朝鲜

川普总统自上任以来，已与朝鲜金正恩多次会面(并成为第一位进入朝鲜国家的在任总统)。当被问及是否会在没有任何前提条件的情况下会见金正恩时，拜登表示不会，并批评了特朗普的行为。“瞧...我们给了他所寻求的一切，合法性，”他说。“总统出现了，会见了他，给他合法性，削弱了我们对他的制裁...我不会见“最高领袖”，他说乔·拜登是条疯狗，应该用棍子打死。”桑德斯毫不迟疑地打断了他。“除此之外，你喜欢他？”

### 艾米·克洛布查尔开始关注医疗保健

Amy Klobuchar(在讨论医疗保健时):“我自己的父亲，我知道他的长期护理保险何时结束，然后我们为他准备了一些储蓄。他住在养老院。他结过三次婚——完全是另一个故事——所以没什么可说的。”
