# 把糟糕的家务结合起来，让它们变得更好

> 原文：<https://lifehacker.com/make-terrible-chores-better-by-combining-them-1835203170>

在任何时候，至少有一个打给公用事业公司的电话是我推迟的。无论是 [打电话给 ConEd](https://lifehacker.com/how-to-make-your-home-as-energy-efficient-as-possible-1831263910) 更新读数，还是打电话给频谱(偷偷换了名字，时代华纳有线，我们仍然知道是你)问 [为什么我的互联网费用](https://lifehacker.com/five-steps-to-help-you-cut-your-cable-bill-1791298146) 一直在上涨，我害怕这些电话，因为我知道它们会永远持续下去。



也有我不惜一切代价避免的家务，因为它们很无聊，我不想做。这些包括从 [刷洗我的浴缸](https://lifehacker.com/scrub-your-bathtub-and-shower-in-minutes-with-a-cordles-1459751196) ，到 [清理我的灶台](https://skillet.lifehacker.com/clean-your-stove-while-the-oven-is-on-1829820800) ，到 [清理我的衣柜](https://lifehacker.com/this-flowchart-helps-cleans-your-closet-with-quick-deci-1613207640)——当你一个人生活，周围没有人评判你的时候，这些事情真的很容易拖延。

这就是为什么我设计了一个系统来一石二鸟。首先，我打电话，当我不可避免地被搁置时，把我的电话切换到扬声器模式，然后开始工作。烦人的音乐是理想的配乐来完成像叠衣服这样无聊的任务，或者 [清理冰箱](https://skillet.lifehacker.com/clean-out-your-fridge-by-baking-some-bread-1823311147) 。

然后，当我按照电话接听的顺序等待与代表通话时，我会尽可能多地完成工作。其中一部分包括欺骗自己，让自己认为这是某种游戏(诚然相当无聊)，赢家是我和我节省的时间。
