# 使用这个 Chrome 扩展防止拖延

> 原文：<https://lifehacker.com/prevent-procrastination-with-this-chrome-extension-1832723418>

我发现自己最近陷入了恶性循环。每当我有空闲时间时，我通常会浏览相同的、陈旧的网站列表，吸收任何新内容来缓解我的无聊。我会在脸书、Twitter、YouTube、Reddit 和 Gmail(偶尔还有 Slickdeals)之间来回切换，直到我最终提醒自己这很愚蠢，也许我应该出去走走，或者做一些更有成效的事情。

Watch

感谢 Chrome 扩展 [**HabitLab**](https://habitlab.stanford.edu/) ，我已经能够回调我的边缘强迫行为。这是一项由斯坦福大学支持的实验，它提供了大量不同的鼓励——有些温和，有些不那么温和——来防止你沉迷于那些占据你太多注意力和时间的网站。在 之前，我们已经讨论过 [这个扩展，但是我想更深入地说明为什么 HabitLab 是一个数字救星。](https://lifehacker.com/be-more-mindful-of-the-time-you-waste-online-with-habit-1828118354)

安装该扩展后，你将跳转到一个页面，该页面提供了关于你可能想要限制的网站的建议。你可以添加任何你想要的网站，但我喜欢预先填充的建议下面还有一个小计数器，可以估计你在这些网站上花了多少时间。(对自己重复:*不要看脸书的号码。不要看脸书的数字。不...)*

一旦你选择了你想从生活中移除的网站，你会被要求设置扩展的烦恼程度。如果你需要尽可能轻微的接触来鼓励你停止总是查看相同的几个网站，扩展可以*跟踪你在这些网站上花费的时间。就这么简单。如果你需要很多帮助，这个扩展甚至会在你打开任何选定的网站 60 秒后自动关闭它们。*停止。浏览。Reddit。**

除了你选择的“扩展攻击”设置，你还可以启用其他“轻推”来帮助你保持专注。我喜欢 HabitLab 提供的各种选项，包括在你加载网站时提示你输入你想在网站上呆多长时间，一个迫使你在网站加载前等待几秒钟的“看门人”(希望阻止你继续下去)，以及一个防止你在过多滚动一次后这样做的“滚动冷冻器”。

你甚至可以在启用它们之前尝试每一个轻推，以防你在做决定之前想看看它们做了什么。如果你是一个 JavaScript 高手(或者喜欢 [复制粘贴](https://github.com/habitlab/habitlab/wiki/Writing-Interventions) )，你甚至可以创建 [你自己的定制推送](https://github.com/habitlab/habitlab/wiki/Share-Interventions) 来鼓励你离开电脑去做其他事情。例如，如果你真的不想浏览网站，你可以告诉 HabitLab 把网站的内容上下颠倒。

一些轻推对于不同的网站有不同的效果。在脸书上，这个扩展可以把关于你浪费时间的温馨提示直接放到你的新闻订阅里，这样你就不会错过了。如果你是一个推特迷，HabitLab 可以隐藏你的所有信息。拖延*解决了*。

在扩展的设置中，你还可以看到你一次在“禁止访问的网站”上呆了多长时间，以及每次轻推对你离开这些网站的效果如何。如果你不在乎自己浪费了多少时间，但需要保持工作效率，你甚至可以将扩展设置为仅在特定时间启用提醒。
