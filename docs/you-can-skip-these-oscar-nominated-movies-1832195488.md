# 你可以跳过这些奥斯卡提名的电影

> 原文：<https://lifehacker.com/you-can-skip-these-oscar-nominated-movies-1832195488>

你没看过*波西米亚狂想曲*！？那很好。Lifehacker 的工作人员收集了我们 2018 年的集体观影体验，并命名了奥斯卡提名电影，你可以放心跳过——包括我们自己甚至没有看过的电影。你可以安全地跳过任何电影，这就是娱乐的方式。但是你特别可以跳过这些！



<section class="q35npn-0 MZtuu">![](../Images/aef335eb562a1da6957ce25a574b4aa1.png)

#### *一个明星的诞生*

我很高兴解除你想见一个明星的诞生的任何冲动。那些告诉你这很棒的人被轻薄的灯光和夸张的音乐所操纵。人物和故事都很单薄。布莱德利·库珀大约有三条线，它们都是听不见的。Lady Gaga 会唱歌，但你已经知道了。这是一个乏味的爱情故事，感觉过时了，脱离了复杂的，分层的故事和丰富的人物刻画，比如说，*黑人三 k 党人*或*你能原谅我吗？*更像是*一个明星是**无聊的*** 那个明星是我，而不是你，因为你不会去看这部无聊的电影。

主编梅丽莎·基尔希

</section>

#### *黑骑士*

看完《黑色党徒》，我发现了很多缺陷——平淡无奇的对话，尴尬的弧线，驱使主角犯菜鸟错误的怪异动机，突然变成好人然后又突然变成坏人的坏人——肯定是因为贴近现实生活的素材。然后我读了导演布茨·瑞列关于这个故事是如何被粉饰以使种族主义警察看起来更好的文章。真正的警察对黑人民权组织的敌意不亚于他们对三 k 党的敌意。这里没有什么值得讲述的故事。知道什么值得讲吗？莱利的*抱歉打扰你*，可能是 2018 年最具文化意义的电影，奥斯卡提名为零。

尼克·道格拉斯，专职作家

</section>

<section class="q35npn-0 MZtuu">![](../Images/aef335eb562a1da6957ce25a574b4aa1.png)

#### *狗岛*

*狗岛*还不错。它只是不如潮人上帝/时尚偶像韦斯·安德森的另一个基于动物的粘土动画仙境，*神奇的狐狸先生*。而*福克斯先生*拥有所有合适的明星(斯特里普！克鲁尼！)，*狗之岛*充斥着安德森的头号玩家，你大部分时间都会在手机上谷歌名人录(*哦，那是默瑞·亚伯拉罕的声音？*)。这并不是说我不喜欢韦斯·安德森电影——事实上我喜欢它们！——这不是他最好的一部，甚至也不是他最好的动画作品。该说的都说了，该做的都做了，这将成为安德森经典中的一个脚注，让位于*布达佩斯大饭店*、*月出王国*和*皇家特南鲍姆*。

乔尔·卡恩，高级视频制作人

</section>

<section class="q35npn-0 MZtuu">![](../Images/aef335eb562a1da6957ce25a574b4aa1.png)

#### *绿皮书*

绿皮书是一部表面上关于种族关系的电影，这部电影导致白人演员获得奥斯卡最佳男演员提名，黑人演员获得最佳男配角提名。这告诉了你这部电影的本质，如果你看过金球奖，你就会知道这位白人导演认为种族主义可以“解决”，如果人们可以交谈并更好地了解彼此的话(他显然也认为在片场拿出他的鸡巴给人们惊喜很有趣)。这种心态在现实中没有依据，但却是好莱坞喜欢的。事实上，它非常喜欢这部电影，我敢肯定几个月后我们会被邀请去看类似的电影——所以如果你错过了这部电影，你真的不会错过任何东西(除了一部关于种族关系的电影，在这部电影中，白人主角认为在现实生活中说 N 字是可以的)。马赫沙拉应该得到更多。

艾丽西娅·亚当奇克，个人理财作家

</section>

<section class="q35npn-0 MZtuu">![](../Images/aef335eb562a1da6957ce25a574b4aa1.png)

#### *老虎钳*

我会说跳过*反之*因为看着很烦。节奏和音调都无处不在。有时候我觉得我在看一个过于聪明的 powerpoint 演示。如果你看电影是因为你喜欢看克里斯蒂安·贝尔*的表演*，那就去看看其他的克里斯蒂安·贝尔电影。如果你喜欢看他和艾米·亚当斯互动，那就看看*《美国骗局》*什么的。我不知道。我不擅长写电影。

食品和饮料编辑克莱尔·洛

</section>

<section class="q35npn-0 MZtuu">![](../Images/aef335eb562a1da6957ce25a574b4aa1.png)

#### *波西米亚狂想曲*

我没有看过《波西米亚狂想曲》,但是我可以很坦然地说，除非你绝对崇拜《女王》,并且渴望尽可能多地了解《女王》《T2 》,否则你不需要看这部电影。我们才华横溢的 editor-in-chief Melissa 最近指出，真实的音乐会镜头或纪录片总是比传记片更令人满意，这是我打算通过前进来生活的一条规则。

弗吉尼亚·k·史密斯，执行主编

</section>
