# 用 Nduja 做真正美味的鳄梨吐司

> 原文:[https://life hacker . com/make-true-terrible-avocado-toast-with-nd uja-1844446601](https://lifehacker.com/make-truly-terrific-avocado-toast-with-nduja-1844446601)

尽管我已经是千禧年的长者，但我从来没有对鳄梨吐司着迷过。(至今 [我还租](https://twocents.lifehacker.com/how-to-afford-a-house-when-youre-spending-all-your-mone-1795231082) ！)很好——没什么*毛病——但我通常更喜欢用勺子直接从鳄梨坚韧的果皮中舀进嘴里。*

Watch

但是命运对我另有安排。作为最近多吃蔬菜、少去商店的努力的一部分，我最近注册了“不完美食品”,这是一个不错的小程序，可以让农产品(如果你喜欢，也可以是肉类和奶制品)送到你家门口。几周前，我发现我可以添加一管 [La 凯西亚的](https://laquercia.us/cuts_specialties_Nduja_Americana) 来补充我平时收集的水果和蔬菜。你最好相信是我加上去的。

我不知道我是否相信命运，但似乎有更高的力量将辛辣的发酵猪肉酱放在不完美盒子里的鳄梨旁边。我爸爸也刚刚给我带来了一个他在面包店买的很好的面包，我不能忽视这些标志。我不得不做鳄梨吐司，我不得不用恩多佳做。

我烤了一片厚面包，然后在上面放了一层薄薄的热的、有味道的可涂抹的 salume，然后在上面捣碎半个鳄梨。我用一点片状盐吃完它，咬了一口，然后听到了类似哈利路亚合唱的声音。各种口味完美搭配。是的，酱料又辣又咸，但也有超熟辣椒的果香。它很丰富，但从来没有尝起来很重。nduja 中强烈的味道去除了所有的脂肪，保持了咬起来的平衡，突出了鳄梨的甜味。

除非你住在一家真正的意大利熟食店附近，否则找到“恩杜嘉”可能有点棘手。事实上，La Querica 品牌甚至不是真正的 nduja，而是一种用该公司生产的熏火腿和 speck 剩下的零碎材料制成的“辣熏火腿酱”。(不过，这很好。)如果你能找到真正的交易，抓住它。如果你能找到的只有“Nduja Americana ”,那就去捡吧。如果你都找不到，你可以 [用商店里买的腌肉和卡拉布里亚辣椒自制](https://lifehacker.com/make-this-spicy-prosciutto-spread-immediately-1828857194) 非常接近。无论你属于哪一代人，这三种选择都会很好地满足你的鳄梨吐司需求。
