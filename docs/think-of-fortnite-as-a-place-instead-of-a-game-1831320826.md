# 把堡垒之夜当成一个“地方”而不是一个“游戏”

> 原文:[https://life hacker . com/think-of-fort nite-as-a-place-代替-a-game-1831320826](https://lifehacker.com/think-of-fortnite-as-a-place-instead-of-a-game-1831320826)

有很多家长对这种现象感到绝望，那就是。这不仅仅是一个孩子们可以从一系列设备(电脑、手机、PlayStation、Xbox 或任天堂 Switch)上免费玩的生存社交游戏。*《堡垒之夜:皇室战争*也因有点上瘾而获得名声( [取决于你问谁](https://www.abc.net.au/news/2018-07-11/fortnite-is-addiction-really-a-thing/9981528) )。

Watch

但也许我们想错了。我们所看到的青少年沉迷于连续几个小时的在线射击游戏，从而腐蚀他们的大脑，他们可能只是把这看作是与朋友“闲逛”的一种方式。

至少 [那是作家欧文·威廉姆斯的《乘](https://char.gd/blog/2018/fortnite-is-the-new-hangout-spot) :

> 我们一起玩，但我们真的只是在一起玩。堡垒之夜有内置的语音聊天，所以它是无缝的，只需跳到语音上，谈论一天，生活，无论发生什么，甚至没有真正意识到这一点。我们一起玩游戏，每十分钟从战斗巴士上下来开始新的一轮，但让我们在那里的是我们都在一起度过时间。
> 
> 自从我搬到海外后的四年里，我在堡垒之夜的语音聊天中与世界另一端的朋友交谈的时间可能比我给他们打电话的时间还多。即使是住在同一个城市的亲密朋友，我们在游戏中也能更频繁地聊天，一起玩几局，同时讨论新的内容。

威廉姆斯并不是唯一一个认为堡垒之夜的内在价值是能够定期与朋友联系的人。他在科技博客 [上发表的文章](https://char.gd/blog/2018/fortnite-is-the-new-hangout-spot) 受到了来自 @anoopr 的的这条推文的启发:
