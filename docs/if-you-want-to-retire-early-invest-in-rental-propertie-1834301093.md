# 如果你想早点退休，投资出租房产

> 原文：<https://lifehacker.com/if-you-want-to-retire-early-invest-in-rental-propertie-1834301093>

如果你追求财务独立和提前退休，你可能听说过 4%规则:当你计算你的投资组合是否能支付你的生活费用时，假设 4%的提款率。



[CampfireFinance](https://www.campfirefinance.com/4-percent-rule/) 很好地总结了为什么 4%法则有效，至少在理论上是这样:

> 4%法则是基于两个金融平均值。
> 
> 首先，4%规则说你的股票投资组合将以平均每年 7%的速度增长。第二，因为*的平均通货膨胀率是 3%* ，你可以安全地提取 4%的增长，留下 3%来跟上通货膨胀。
> 
> 因为你只花了你投资组合的平均增量，理论上你永远不会没钱。

然而，Paula Pant 在做了计算——如果你投资租赁物业，你将能够提取 6% 而不是 4%。这意味着退休后你将获得更大的月支出，你甚至可能比你预期的更早退休。

她是这样解释的:

> 如果你在指数基金上退休，你可以根据 4%的提取规则来计划你的退休。
> 
> 但是，如果你退休后租房住，你可以使用相当于 6%的提款规则。
> 
> 即使这两项投资的总回报相同，情况也是如此。
> 
> 为什么？两个原因:(1)出租物业的回报偏向于收入流而不是资本增值，以及(2)出租物业没有收回本金的风险。

[她的完整帖子](https://affordanything.com/six-percent-is-the-new-four-percent/) 涵盖了为什么租赁物业比指数基金带来更多的现金流，为什么租赁是比股票风险更低的投资，以及即使你没有很多钱可以投资，你如何开始租赁物业。(剧透提醒:寻找中西部的独栋住宅。他们仍然负担得起！)

如果你已经开始投资租赁物业，作为创造被动收入和/或提前退休的一种方式，让我们知道你是否认为 6%规则是准确的——以及除了把钱投入股市之外，我们是否都应该开始购买租金。
