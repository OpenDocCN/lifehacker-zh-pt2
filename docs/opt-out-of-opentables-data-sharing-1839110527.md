# 选择退出 OpenTable 的数据共享

> 原文：<https://lifehacker.com/opt-out-of-opentables-data-sharing-1839110527>

OpenTable 是在白天或晚上的任何时间预订餐馆的一个方便的选择，尤其是如果你想避免通过电话与任何人交谈。但如果你有一个 OpenTable 账户，你很可能会允许该平台使用和分享你的数据，包括你的餐饮偏好和销售点信息。



您的 OpenTable 帐户会自动选择将您的餐饮资料和偏好与同一餐厅集团的其他机构以及“餐厅合作伙伴和分支机构”共享默认设置还会与不明确的第三方共享您的个人信息，OpenTable 会从您就餐的餐厅的销售点终端接收数据，这些数据可能会被汇总并与所述餐厅共享。

如果你不想让 OpenTable 访问这么多信息，你必须手动选择退出。为此，请打开 OpenTable 应用程序。在“主页”选项卡上，单击右上角的个人资料图标。点击“关于你”，向下滚动到定制餐饮体验部分。在那里，您可以关闭与您的用餐体验、第三方共享和销售点信息相关的共享设置。单击保存。

要通过 OpenTable 的网站选择退出，请登录并单击右上角的开关打开我的个人资料。转到帐户详情>用餐偏好，取消选中您想要退出的设置的所有复选框。
