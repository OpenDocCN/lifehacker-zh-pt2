# 如何为自己做饭感到兴奋

> 原文：<https://lifehacker.com/how-to-get-excited-about-cooking-for-yourself-1835125167>

“一个人做饭”被认为是最悲伤的一种烹饪，但你有没有试过为讨厌西红柿的人做饭？不管你的关系状态如何，你都得吃饭，你如何看待这项活动的一个简单的转变可以把自我保护的行为变成真正有营养和娱乐的事情。

Watch

但首先，你必须放松。饮食和烹饪中有很多道德问题，你必须允许自己 [走捷径](https://lifehacker.com/its-okay-to-buy-prepared-foods-1834422853) ，出去吃饭，搞砸一顿饭。为一个人做饭的好处是*没人能看到你*。这意味着你的食物可以不在 Instagram- 上，是的，但它应该鼓励你尝试精神错乱的、挑战极限的风味组合，以及 [具有挑战性的食谱](https://skillet.lifehacker.com/why-you-should-try-challenging-recipes-right-now-1831874153) 你肯定会 [注定失败](https://skillet.lifehacker.com/why-failure-is-a-useful-kitchen-tool-1797643711) 。一旦你接受了会有一些陷阱，你就能找到一些乐趣。

### 了解满足感对你意味着什么

不是每个人都对同样数量或种类的食物感到满足。一些人对数量感到满意——想一想超大份的蔬菜沙拉——而另一些人，比如我的母亲，喜欢少量的更丰富、更密集的食物。然后就是我这样的人，被综艺满足了。认清自己是什么样的吃货，然后 [购物，并据此做好计划](https://skillet.lifehacker.com/how-to-buy-cook-and-store-food-for-one-1787145463) 。

### 不要给自己太大的压力

如果烹饪对你来说是一项困难的活动，允许自己*而不是*去烹饪。也许你一周只能处理一个食谱。那很好。为其他晚上储存一些冷冻或免煮的选择，或者点一些该死的泰国菜。烹饪越令人愉快，你就越有可能去做，所以不要因为那些晚上你不能完全从头开始做一顿饭而对自己进行评判。

### 读书

尽管我在网上写关于食物的文章，但我更喜欢平静的、没有评论的食谱作为一个食谱提供系统。找一本合适的烹饪书能引发对食物的痴迷。如果没有奈杰尔·劳森的《如何成为家庭女神》,我就不会是今天的我——一个在推特上大喊奶酪的女人。我被 Nigella 本人以及她对食物和烹饪的方法深深吸引，于是我开始阅读她的几本书，一路走来，在厨房里变得更加熟练和舒适。

虽然有专门为单身食客写的食谱，但罗艳芳的《T2 的独舞》《T4 的独舞》《T5 的独舞》尤其引人注目，不要觉得你必须坚持小的格式。你足够特别，值得拥有独一无二的特殊食物，在冰箱的帮助下，你可以批量制作大量的食谱。我一生中最快乐的时光之一是我为自己准备了一整块猪肩肉，然后吃了几个月。

### 拥抱所有的自由

孤独是规则。虽然我有几个(两个)口味几乎相同的朋友，但事实是许多人的食物偏好、敏感性和饮食可能与你的不匹配，不必适应所有这些是非常自由的。当你为一个人做饭时，你被允许按照你认为合适的方式制作你的盘子。作为一个嫁给讨厌西红柿、蘑菇、海鲜和未腌制猪肉的男人的人，这是一种宣泄，但即使你没有因为挑食而受到伤害，这也很好。此外，只为你做饭意味着你可以买到更好、更贵的食物。为一个人购买扇贝和牛排的价格是为两个人购买的一半。这只是数学。

### 变得稍微社会化

我喜欢让人们远离我的厨房，但有时我会让他们进入我的思想和内心。与他人讨论食谱和烹饪，即使你不与他们一起吃饭，也会带来各种关于(也许是自己)的新发现。你可以成立一个烹饪书俱乐部，让每个人根据同一本烹饪书制作一份食谱，或者只选择一个松散的主题； [组织一次食谱交流会](https://skillet.lifehacker.com/how-to-host-a-cookbook-swap-party-1834750636)；或者在你工作的地方建立一个以食谱为主题的休闲频道，如果你有机会的话。

我经常说“有人做饭时，我会做得更好。”这部分是因为我喜欢炫耀，但这反映了许多人如何看待独自烹饪和饮食。为某人做饭是一种爱的行为，没有理由不把这种爱指向你自己。当然，不是每顿饭都需要是一封可以吃的情书；有些食物只是燃料，学会如何平衡这两个极端可以让在家吃饭压力更小，更令人满意。
