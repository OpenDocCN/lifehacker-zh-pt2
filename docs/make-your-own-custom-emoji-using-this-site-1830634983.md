# 使用这个网站制作你自己的表情符号

> 原文:[https://life hacker . com/make-your-own-custom-e moji-using-this-site-1830634983](https://lifehacker.com/make-your-own-custom-emoji-using-this-site-1830634983)

有时候，智能手机上的表情键盘并没有你想要的东西。

表情符号生成器是一个网站，让你利用他人的片段来组合你完美的表情符号。例如，你的手机有一个呕吐男表情符号，但是一个戴着光环和单片眼镜吹着派对喇叭的呕吐男怎么样？没想到会这样。

界面使用起来超级简单。你从 12 个基本面部中选择一个使用，然后添加你自己的眼睛、嘴巴和配件，赋予它你自己的个人风格。

一旦你创建了完美的表情符号，你就可以直接从网站上把它保存到你的手机上，用它来回复朋友，就像你使用股票表情符号一样。

不幸的是，你保存的作品不会显示在你手机的表情键盘上。取而代之的是，你必须把它作为一张保存在手机上的图片。该网站确实给了你一个小版本和一个大版本，所以虽然你的里程数可能会因你的设备而异，但那个小表情符号可以看起来非常接近真实的东西。
