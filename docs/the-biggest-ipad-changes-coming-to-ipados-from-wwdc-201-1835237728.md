# 2019 年 WWDC 的 iPad 最大的变化来自 iPadOS

> 原文：<https://lifehacker.com/the-biggest-ipad-changes-coming-to-ipados-from-wwdc-201-1835237728>

在周一举行的年度全球开发者大会(简称 WWDC)上，苹果公司就其所有产品的未来做出了一些重大宣布，但其中最大的一个事实是，iPads 现在拥有自己的操作系统 iPadOS。iPad 将能够走自己的路，而不是使用与 iPhone 相同的软件 iOS。根据演示，这将使它更好地遵循区分手机和笔记本电脑的道路，成为家庭电脑的可行替代品，用于基本任务，如打字和浏览网页。

在其成立的第一年，iPadOS 和下一个版本的移动浏览器 [iOS 13](https://lifehacker.com/all-the-big-ios-13-changes-apple-announced-at-wwdc-19-1835216984) 有很多重叠。这两款设备都有所改变，尽管它们主要面向平板电脑用户。(手机上的三指捏手势？来吧)。当然，iPad 本身也有一些变化。让我们来看一下大的要点，这样你就能知道今年你的 iPad 会有什么了。

### **平滑移动边缘**

随着切换到 iPadOS，iPad 主屏幕将略有改观。平板电脑底部的常用应用栏更紧密，允许更多的应用程序和更像笔记本电脑的感觉。你还可以锁定窗口小部件——当你在 iOS 上从应用程序的第一页向左滑动时看到的磁贴——允许你通过快速滑动将它们带到应用程序的主屏幕上。

### **多任务改造**

通过 iPadOS 的转变，iPad 最大的改进是多任务处理能力，让你可以像在笔记本电脑上一样在 iPad 上工作(或娱乐)。你现在可以在一个 iPad 屏幕上打开多个窗口。这意味着你可以同时使用两个应用程序，或者在同一个屏幕上显示两个屏幕。

此外，如果你点击你正在使用的应用程序，你可以访问“App Exposé”，这是一个类似于 MacOS 上的 mission control 功能的任务管理器 — 有趣的事实:Mission Control 过去在 Mac 上被称为 exposé—，它可以让你从所有打开的应用程序中进行选择，包括同时打开多个应用程序的屏幕。您可以打开每个应用程序的多个迭代，因此您可以在单独的 Word 文档和另一个“空间”之间切换，或者在 Word 和另一个应用程序的屏幕之间切换。基本上，这是一种模拟(然后组织)拥有定义笔记本电脑或台式机体验的一堆打开的窗口的体验的方式。(好吧，也许只是*我的*笔记本电脑体验。不要被评判- y)。

iPad 的 Slide Over 功能可以让你在屏幕上附加一个辅助应用程序，现在还可以在不同的应用程序之间轻松切换。通过从屏幕底部拖移手指，您可以看到所有使用 Slide Over 打开的应用程序，这让您可以轻松地挑选出您需要的内容。您也可以沿着“滑过”标签的底部左右滑动，以在它们之间循环。能够像标准应用程序一样轻松地在应用程序上滑动，这使得该功能更易于使用。

### **将 iPad 用作第二块屏幕**

如果你有一台笔记本电脑和一台 Mac 电脑，名为 Sidecar 的新的 [Mac 应用](https://lifehacker.com/all-the-macos-10-15-catalina-announcements-from-apples-1835210038) 可以让你将 iPad 用作笔记本电脑或台式机的第二个触摸屏。即使连接到 Mac，iPad 的行为仍然像 iPad 一样，这意味着您现在有了一个基于触摸的 MacOS 机器界面。此外，Sidecar 支持 Apple Pencil，这表明你可以使用 Sidecar 来弥合更强大的计算机和基于触摸的平板电脑之间的差距。对我来说，这不是一个很大的好处，但我想，在 iPhones 和 iPads 上长大的用户(或任何喜欢触摸控制而不是键盘和鼠标/触摸板的人)会发现这很自由。

### **新文件应用让你像搜索 Macbook 一样搜索平板电脑**

新的文件应用程序在 iPadOS 和 iOS 上都可以使用，但对你的平板电脑来说更重要。新的应用程序经过重新设计，模仿了 MacOS 的文件系统，允许你使用苹果的所有标准文件视图 — 图标视图、列表视图和分栏视图，像在笔记本电脑上一样组织和搜索文件。您可以组织文档组，并独立于它们所依赖的应用程序来访问它们:文件还支持插入设备的能力，如 USB-C 硬盘驱动器，并直接上传或下载文件，绕过应用程序和 iOS 共享菜单。如果你使用笔记本电脑，并喜欢它们的工作方式，这将使生活变得容易得多。

### **别担心，iPadOS 是 iOS 13 的好东西**

仅仅因为 iPad 在自己的操作系统上运行，并不意味着 iPad 用户会错过下一版本 iOS 宣布的新功能。[iOS 13](https://lifehacker.com/all-the-big-ios-13-changes-apple-announced-at-wwdc-19-1835216984)的所有重大改进，从黑暗模式，到新的位置数据安全升级，到用苹果按钮登录，到那些甜蜜的，甜蜜的 Memoji，都将在 iPadOS 上实现。

随着时间的推移，这两条道路可能会更加分歧，但目前的区别似乎更多的是为 iPad 启用对手机没有意义的新功能，而不是删除或缩减任何一款设备的有用工具。
