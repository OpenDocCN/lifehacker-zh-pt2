# 如果医生只告诉你减肥，考虑第二种意见

> 原文：<https://lifehacker.com/if-a-doctor-only-tells-you-to-lose-weight-consider-a-s-1837202095>

众所周知，医生在看更大的病人时，有时会忽略体重 。如果你认为你真正的健康问题被忽视了，那就值得寻求第二种意见。

我之所以会想到这一点，是因为喜剧演员珍·柯伦 [昨天在推特上](https://twitter.com/jencurran/status/1160961368142405632) 讲述了她如何差点错过癌症诊断，因为她看的第一位专家将她的问题归咎于她最近的怀孕和她的体型。
