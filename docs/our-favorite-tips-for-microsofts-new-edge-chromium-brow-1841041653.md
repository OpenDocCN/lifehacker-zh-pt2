# 我们对微软新的 Edge Chromium 浏览器最喜欢的提示

> 原文：<https://lifehacker.com/our-favorite-tips-for-microsofts-new-edge-chromium-brow-1841041653>

微软终于发布了基于 Chromium 的 Edge 网络浏览器的官方稳定版本。尽管这并不意味着你的老派优势正在消失，微软*将* [开始小批量推出](https://blogs.windows.com/msedgedev/2020/01/15/upgrading-new-microsoft-edge-79-chromium/)Chromium 浏览器，最终在今年夏天通过 Windows Update 取代它。



当然，您也可以今天就下载。我建议这样做。这是一个伟大的，快速的浏览器，比微软的老版本 Edge 好用得多。您可以获得更多定制外观和感觉的选项。你可以在 Chrome 用户的网上商店 使用所有有趣的扩展功能 [，你还可以玩很多额外的功能。](https://chrome.google.com/webstore/category/extensions)

当然，这意味着你必须从一个浏览器切换到另一个，这可能会感觉不和谐。为了帮助你，我们创建了一个 Edge Chromium 须知的快速指南。

### 我在哪里可以得到边缘铬？

如果你不想等微软把它更新的浏览器扔到你的腿上，你可以现在就自己下载。访问 [本网站](https://www.microsoft.com/en-us/edge?form=MO12GB&OCID=MO12GB) ，为您的操作系统选择正确版本的浏览器，并下载。运行安装程序后，系统会提示您从现有浏览器导入数据。我建议这样做，至少可以让您不必从头开始设置一切。(如果你在“导入”和“继续而不导入”两个选项之间点击，Edge 甚至会向你展示一个预览。)

然后你会被要求定制你想要的新标签页的外观，你有三个选项可以选择:灵感、信息和聚焦。像以前一样，点击这些选项中的任何一个都会实时地重新配置浏览器，给你一个改变后的样子的例子——这是一个我开始喜欢的功能。

假设你将在你拥有的其他设备上切换到 Edge，那么你将被询问你是否想要在登录的设备上同步你的密码。最后，Edge 会询问你是否可以利用你的浏览历史来改进个性化广告、搜索结果和新闻条目。仅此而已。没有额外的宣传，Edge 已经准备好了。现在让我们自定义它。

### 值得关注的边缘铬设置

点击右上角你名字旁边的三点图标，然后点击设置。如果你是 Chrome 用户，下一个屏幕应该看起来有些熟悉。

首先，我建议访问你的**个人资料**部分的“同步”部分，以确保你在 Edge 中所做的一切都与你的微软账户同步——以防你在不同的机器之间切换(或者甚至在你的 [安卓](https://play.google.com/store/apps/details?id=com.microsoft.emmx&hl=en_US) 或 [iPhone](https://apps.apple.com/us/app/microsoft-edge/id1288723196) 上使用 Edge)。你会注意到有些数据*还没有*同步:你的浏览历史，你打开的标签，你的扩展等等。这些选项即将推出，但微软还没有准备好。

在左侧边栏的**隐私和服务**部分，你可以从三种不同的模式中选择 Edge 应该如何处理各种浏览器跟踪器。选择你的毒药，但我稍后会有一些建议，让它更上一层楼。

这也是你可以在每次关闭浏览器时告诉 Edge 清除各种数据的地方——对于注重隐私的人来说，这又是一个方便的举措。点击“选择每次关闭浏览器时要清除的内容”，您将能够从以下选项中进行选择:

回到“隐私和服务”主屏幕，确保选中发送“请勿跟踪”请求的复选框，即使网站完全自愿服从这一指定。

一直到底部，点击“地址栏”，它允许你离开微软的 *meh* Bing 搜索引擎——当然是 Edge 的默认——并切换到更有用的东西(谷歌)或私人( [DuckDuckGo](https://lifehacker.com/how-to-maximize-your-browsing-privacy-using-duckduckgo-1830659232) )。

现在，点击左侧边栏的**外观**部分。在这里，你可以选择让你的浏览器有一个亮或暗的主题，或者只是跟随 Windows 当时正在做的事情。如果需要更大(或更小)，您还可以调整字体大小，并让您的收藏夹栏贴在您打开的每个标签页上(而不仅仅是新标签页)。

启动设置标签上的**很简单，但是很重要；这是你如何选择你希望你的浏览器在启动时显示的内容，可以是一个新的标签，你上次离开浏览器时打开的所有内容，或者一个特定的页面。我是中间选项的忠实粉丝，这就是为什么我现在在 Chrome 的后台有如此多的浏览器标签。如果你喜欢更整洁的体验，每次关闭浏览器时，Edge 中的默认选项会删除每个打开的标签页。**

类似地，**新标签页**部分将允许您定制您之前选择的任何新标签页——或者选择一个新的，如果您不喜欢您的选择。如果你选择了“定制”选项，你就不能定制太多的*内容，但是你至少可以让自己访问当天的新闻，例如，不会有大量的报道充斥在你浏览器的新标签页上。如果你讨厌你在 Edge 得到的日常形象，因为你不喜欢乐趣，你也可以把它关掉。*

*跳到你的设置的**系统**部分，我想提醒你注意“当 Microsoft Edge 关闭时继续运行后台应用程序”选项。我是一个纯粹主义者，我不喜欢应用程序在后台运行，如果它们真的不需要在那里，所以我把它关掉了。我怀疑我会注意到我的边缘加载时间的增加，也不会影响我的扩展或日常使用。此外，如果你的扩展遇到了奇怪的问题，请取消选中“使用可用的硬件加速”——在我的情况下，这是让我的“从网页创建 PDF”扩展工作的唯一方法。*

*最后，请务必注意“关于 Microsoft Edge”部分。和 Chrome 一样，每当你想手动检查浏览器更新时，这是你想要访问的部分。*

### *微软 Edge 里也有标志吗？*

*是的。像 Chrome 一样，你可以在浏览器的地址栏中输入“edge://flags/”,点击 Enter，然后加载一些高级和/或预览功能，这些功能可能会启用，也可能不会启用。现在想到的一个可能值得一试的是“ [标签冻结](https://lifehacker.com/enable-these-hidden-security-and-performance-features-i-1840363078) ”，或者是 Chrome 的一个功能，它会在不活动五分钟后自动暂停你的背景标签，从而节省你宝贵的系统资源。*

*我也喜欢“声音内容设置”，它可以让你通过右键菜单将整个网站静音，而不仅仅是单个标签。*
