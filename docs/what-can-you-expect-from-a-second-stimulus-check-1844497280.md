# 你能从第二轮刺激计划中期待什么？

> 原文：<https://lifehacker.com/what-can-you-expect-from-a-second-stimulus-check-1844497280>

随着国会为下一个刺激方案制定细节，有一点已经得到了确认——另一轮刺激检查。正如 [华尔街日报](https://www.wsj.com/articles/second-stimulus-payment-could-deliver-cash-even-faster-11595522449) 报道的那样，国会仍然需要批准第二笔付款，决定细节，并获得特朗普总统的签署——这可能需要几周时间。但一旦发生这种情况，第二张支票可能会更快到达，因为 IRS 系统已经在第一张支票上就位。

### 你会得到多少钱？

尽管遭到共和党人的反对，众议院民主党人在五月通过了英雄法案。该法案包括向成年人支付 1200 美元，同时向最多三名受抚养人每人支付 1200 美元。昨天，《国会山报》报道说，共和党人还将在他们的新刺激法案中包括 1200 美元的直接支付，该法案将于周一到达参议院。

昨天，部长史蒂文·姆努钦 [告诉记者](https://www.bloomberg.com/news/articles/2020-07-23/republican-plan-would-send-more-1-200-stimulus-checks-in-august) 共和党的法案将包括“与上次完全相同的条款”如果发生这种情况，共和党人可能不会同意将家属津贴从 500 美元提高到 1200 美元。

### **谁将晋级**

虽然我们还不知道谁有资格获得第二笔付款，但我们知道两党的立法者将考虑你的年龄、收入、公民身份、婚姻状况和受抚养人，以确定你是否收到支票以及支票的金额。

《英雄法案》一直是谈判的起点——它反映了《关爱法案》,单一收入限制为 7.5 万美元至 9.9 万美元。还为第一轮支付中遗漏的人提供支付——大学生、17 岁以上的受抚养人、领取社会保障残疾收入的人和提交联邦纳税申报表的非公民。

很难预测共和党法案中会包含的确切门槛。如果条款与《保健法》相匹配，收入限制将与民主党的法案相同。但是一些共和党人，包括麦康奈尔 ，已经呼吁降低限额。

### **如何处理这笔钱**

不管国会同意什么，如果你有资格获得另一笔付款，你可能会比你预期的更快收到钱。如果你的收入没有受到疫情的影响，以至于你需要立即使用它来支付你的账单，你很快就会决定如何处理这笔意外之财。

“如果可以的话，考虑将刺激计划中的一大部分存起来，以建立应急基金，”马克·雷耶斯说，他是 [阿尔伯特](https://albert.com/) 公司的财务顾问专家。他说，在不确定的经济形势下，储蓄可能比挥霍在有趣的事情上更好。

雷耶斯说，一旦你攒够了至少三个月的生活费，接下来就要考虑还清高息债务了。但是，如果你目前失业，他建议只把钱存起来用于基本支出——比如食物、房租、公用事业或医药。

如果你真的不需要钱， [你随时可以把钱捐给](https://twocents.lifehacker.com/where-to-donate-your-stimulus-check-1842862639) 。
