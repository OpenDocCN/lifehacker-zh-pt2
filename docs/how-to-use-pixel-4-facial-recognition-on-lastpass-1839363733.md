# 如何在 LastPass 上使用 Pixel 4 面部识别

> 原文:[https://life hacker . com/how-to-use-pixel-4-face-recognition-on-lastpass-1839363733](https://lifehacker.com/how-to-use-pixel-4-facial-recognition-on-lastpass-1839363733)

尽管如此新，但许多第三方应用程序已经支持 Pixel 4 的一些独特功能。其中一个应用程序恰好是我们最喜欢的密码管理器 之一，LastPass，它支持 Pixel 4 的生物识别解锁功能。

Watch

假设你已经在 Pixel 4 上设置了面部识别数据，LastPass 用户可以通过他们的微笑杯进入他们所有保存的应用和账户——但首先，你需要在 LastPass 中打开设置。

1.  打开 [LastPass 安卓 app](https://play.google.com/store/apps/details?id=com.lastpass.lpandroid&hl=en_US) 。
2.  点击三叠点图标。
3.  点击**“设置”、**，然后点击**“安全”**
4.  向下滚动并打开**“使用生物识别解锁”**

打开此选项后，当您登录到与 LastPass 同步的应用程序或网站时，您会看到一个新的提示，要求使用您的面部。对准你的脸，一旦你的手机识别出是你，然后点击**“确认”**无需密码即可登录。

现在，尽管这很方便，但我们不能不提的是，Pixel 4 的面部识别比其他手机上的要不安全得多。谷歌表示，它正在寻找一个修复方法，但可能需要几个月的时间才会有任何变化，所以请谨慎使用该功能——尤其是如果它可以被您的密码管理器使用的话。
