# 远离婚礼贷款

> 原文：<https://lifehacker.com/stay-away-from-wedding-loans-1835841764>

如今孩子们如何支付婚礼费用？有贷款。一个持续一天的派对。《华盛顿邮报》 调查了通过个人贷款来支付婚礼费用的做法。虽然这不是一个令人惊讶的阅读，但仍然令人毛骨悚然。

Earnest 的大卫·格林在接受《华盛顿邮报》采访时表示:“人们背负着越来越多的债务，他们想结婚，但却没有足够的资金。”他说，去年婚礼融资贷款的需求翻了两番。文章指出，夫妇平均借款 16，000 美元，期限为三年，利率从 7%到 18%不等。SunTrust 的贷款服务公司 LightStream 表示，其贷款资助的婚礼数量是一年前的三倍。类似的贷款也可以通过 Prosper 和 Upstart 获得。

“无论是定制书法的请柬，还是简单的 Instagram 设置，婚礼费用都会迅速增加——即使是最节俭的夫妇有时也需要一些额外的现金，”[Earnest](https://www.earnest.com/personal-loans/wedding-loans)婚礼专用贷款页面上写道，这触动了那些不知道如何调整婚礼预算的准新娘或准新郎的心弦。

### 为什么婚礼贷款现在如此受欢迎

这里有一个完美的风暴，使这些婚礼主题贷款如此受欢迎。人们结婚的年龄越来越晚，这意味着他们的父母可能已经退休，无法或不愿意做出贡献，就像传统曾经规定的那样。婚礼变得越来越昂贵:根据 [和](https://www.prnewswire.com/news-releases/couples-spend-an-average-of-33-931-on-weddings-creating-completely-personalized-celebrations-with-emphasis-on-reclaiming-and-redefining-tradition-according-to-the-knot-2018-real-weddings-study-300795628.html) 的最新统计，一场“普通”婚礼的费用为 3.4 万美元。美国人有更多的学生贷款债务和更高的生活成本。人们只是不像以前那样留出婚礼储蓄资金。

使用信用卡支付婚礼费用也很常见。但是个人贷款有一些吸引人的地方:它每个月有一个固定的付款额，并承诺如果你坚持到底，你将在什么时候还清。信用卡账单会继续上涨，因为每个月的利息都会卡在余额上。

### 为你的婚礼借新债是个坏主意

但是，尽管有种种缺点，信用卡可能比贷款更有意义。如果你在考虑收取婚礼费用，你可能已经有了自己名下的信用额度。信用卡限额可能比你能获得的任何新贷款都要低，这限制了你的债务额度。即使对于超级持卡人来说， [的平均信用额度](https://www.fool.com/the-ascent/credit-cards/articles/average-credit-card-limit-america-credit-score/) 最高也在 9500 美元左右——远低于平均婚礼贷款。

贷款和信用卡都有相似的利率，它们都向厌恶债务的人发出了相同的信号:远离债务。为婚礼负债不是投资。只是入不敷出而已。

如果你为了买房子而负债，当你还清债务时，你就拥有了可以增值的财产。如果你负债买车，车会贬值，但至少你可以开着它在城里到处逛。如果你为了买婚礼而负债，你得到的只是一个派对。
