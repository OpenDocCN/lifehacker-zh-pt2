# 这些美国航空公司在 2019 年颠簸了最多的乘客

> 原文：<https://lifehacker.com/these-u-s-airlines-bumped-the-most-passengers-in-2019-1839694572>

超售航班回来了，宝贝！2017 年春天，在 [之后，一名男子被强行拖下超额预订的联合航空](https://gizmodo.com/united-ceo-doubles-down-blames-passenger-who-got-his-a-1794203925) 航班，联合航空首席执行官*指责该男子*，航空公司在只出售飞机上实际座位数量方面有了显著改善。到 2018 年年中，只有事件发生前的六分之一的“颠簸”乘客。但是，据旅游网站 UpgradedPoints.com 介绍， [的颠簸率又翻了三番](https://upgradedpoints.com/us-airlines-most-likely-to-bump-you-data-study/) 。

今年 3 月，在两起坠机事件导致数百名乘客和机组人员死亡后，美国终于加入世界其他国家的行列，停飞了所有波音 737 Max-8 和 Max-9 飞机。就在白宫让他们停止之前，西南航空和美国航空 [告诉 Gizmodo](https://gizmodo.com/which-u-s-airlines-operate-the-boeing-737-max-8-1833200705) 他们计划继续飞行这些飞机，非常酷，并且已经向波音公司订购更多。

随着可供选择的飞机越来越少，航空公司又开始超额预订更多的航班。你想让他们做什么，只有当他们真的有座位给你时才赚钱吗？也许你应该在迫使他们停止飞行波音死亡陷阱之前就想到这一点！

UpgradedPoints 表示，2018 年，最有可能超额预订和取消航班的航空公司是 Frontier 和 Spirit，紧随其后的是阿拉斯加、PSA 和美国航空。但在 2019 年，Frontier、Spirit 和 Alaska 都有所改善。随着波音 Maxes 停飞，美国和西南航空超额预订了更多的乘客。

因此，截至 2019 年年中，最有可能无意中取消你的航班的航空公司是梅萨航空公司(其运营方式为**美国之鹰**和**联合快递**)、PSA 航空公司(也称美国之鹰)、**忠实航空公司**和**美国航空公司**。

好消息是，从统计数据来看，你仍然不太可能是被撞的乘客之一。即使是在梅萨，也只有 0.01162%的乘客会不由自主地被撞。(这些统计数据不包括自愿颠簸。)很可能不会发生在你身上。

正如我的同事贝丝·斯克沃茨基所说，坏消息是:“应该是零！还有哪个市场的人会在购物时付款，然后在最后一分钟把它签上来？”

对于一个似乎无法持续盈利的行业来说，超额预订是一种必要的罪恶，还是一个正在谋杀地球 的行业的一种更不道德的选择，这取决于你。但是 UpgradedPoints 说，避免颠簸最安全的赌注是 Delta。
