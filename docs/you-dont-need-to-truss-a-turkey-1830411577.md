# 你不需要把火鸡捆起来

> 原文:[https://life hacker . com/you-dont-need-to-truss-a-turkey-1830411577](https://lifehacker.com/you-dont-need-to-truss-a-turkey-1830411577)

烹饪一只完美的感恩节火鸡是一个相当高的要求，每个厨师都有自己的技巧来制作脆皮和多汁的肉。无论你今年如何与你的火鸡争吵，有一个普遍的火鸡真理将会节省你的时间和精力:俄罗斯是一个骗局。

火鸡体积巨大，脂肪含量相对较低，这使得它们很难完美地烘烤。将你的火鸡分解成各个部分，几乎可以保证均匀的烹饪结果，但大多数人更喜欢简单的准备和诺曼·洛克威尔式的美学，一整只捆绑整齐的烤火鸡。这很好，但你应该知道捆绑是一种纯粹的美学选择，它使一项已经很困难的任务变得更加困难。

绑住火鸡的腿会堵住空腔，阻止空气在火鸡最密集的部位自由流通。这对你的感恩节大餐至少有三个重大影响。首先，如果你是一个爱填馅的家庭，堵住腔只会让馅更难煮熟。第二，腿已经比火鸡的其他部分需要更长的时间来烹饪；把它们挂在乳房旁边意味着它们需要更长的时间，在这一点上，那些乳房可能是干的。第三，捆扎是将皮和脂肪的宝库折叠起来，而不是将其暴露在需要变脆的热量下。火鸡的表皮和脂肪含量已经很低了；你不应该把脆皮留在桌子上。

旧习惯很难改变，所以如果更少的烘烤时间和额外的松脆外皮还不足以说服你，也许这个可以:跳过桁架是有史以来最简单的火鸡，特别是如果你已经干腌过它。你可以把它从冰箱里拿到柜台上，再拿到烤箱里，完全不需要额外的工作。当然，你仍然可以把腿绑起来拍照——如果你划破了几处皮肤，这是隐藏证据的好方法。
