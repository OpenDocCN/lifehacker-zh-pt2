# 这些电子游戏可以让你抚摸狗狗

> 原文：<https://lifehacker.com/these-video-games-let-you-pet-the-dogs-1833127272>

正如我们的姐妹网站 Kotaku 解释的那样， [所有的视频游戏都应该让你抚摸动物](https://kotaku.com/all-video-games-should-let-me-pet-the-animals-1826089191) 。可悲的是，许多游戏都有狗、狗，但却没有办法去抚摸它们。直到国会采取行动，取缔这一疏忽，一个新的推特账户， [你能摸狗吗？](https://twitter.com/CanYouPetTheDog) ，告诉你哪些游戏做，不让你宠狗。它有图片。

Watch [https://lifehacker.com/embed/inset/iframe?id=twitter-1103362511875784705&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1103362511875784705&autosize=1) 

这个账户(仅两天后就有超过 18，100 名粉丝，12 条非常受欢迎的推文)特别有用，如果你试图在爸爸模式 下 [玩视频游戏，避免在孩子面前玩游戏时出现暴力。宠物狗狗的游戏有*红色死亡救赎 2* 、*刺客信条三*、*我们最后的人*、*最终幻想八*。](https://offspring.lifehacker.com/how-to-play-far-cry-new-dawn-in-dad-mode-1832662007)

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1103060562278318082&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1103060562278318082&autosize=1) 

“你能摸摸狗吗？”很简单。根据账户简介，“手动按键导致抚摸的视觉表现是确认所必需的。”没有关于如何提交你自己的信息，但是大概你可以@-给他们发你自己的信息，并希望得到转发。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1103411388406472704&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1103411388406472704&autosize=1) 

互联网是一个如此糟糕的地方，以至于“垃圾火灾”成了陈词滥调。然而还是有纯粹的乐于助人的人在里面，告诉我们哪里可以养更多的狗。

目前没有“能不能宠猫？”账户。
