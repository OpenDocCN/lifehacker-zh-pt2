# 不要被骗去支付这额外的 40 美元报税费

> 原文:[https://life hacker . com/don-get-dricked-to-paying-this-extra-40-tax-return-1832227421](https://lifehacker.com/dont-get-tricked-into-paying-this-extra-40-tax-return-1832227421)

当你在网上或通过代理报税时，如果你不能免费报税，你通常可以选择从联邦退税中扣除报税费用。

Watch

虽然这对于大多数报税者来说是一个方便的选择，但是如果你不想被另一笔费用打击的话，你可以跳过这一步，用你的信用卡支付。

用退款支付很容易，但这也是税务公司从申报者身上赚更多钱的一种方式。尤其是涡轮税，让报税者对客户所说的“欺骗性”做法感到沮丧。如果你使用该软件，并从你的退款中扣除申报费用，你将被收取 39.99 美元的退款处理费，此外还有 TurboTax 收取的报税费用。

一些客户表示，TurboTax 没有明确说明还有另一种不产生额外费用的支付方式。

“这 39.99 美元的所谓退款处理费是误导性的，TurboTax 网站也是误导性的，”一名申报人在 [公司的常见问题页面](https://ttlc.intuit.com/questions/4173150-should-i-be-charged-39-99-for-filing-and-39-99-for-option-to-pay-with-refund) 上写道。“为什么没有通知或免责声明说，如果你用信用卡支付，就不会收取 39.99 美元的费用？”

经营 TurboTax 的 Intuit 公司的企业沟通高级经理 Ashley McMahon 去年告诉底特律自由新闻报 说，这项费用“是为那些手头没有信用卡的顾客提供的额外便利——而且完全是可选的。顾客也可以选择用信用卡、借记卡或预付卡付款。”

别介意，在自己的网站上，TurboTax 承认这个过程并不清楚。“尽管名称有些误导，但您不需要退款处理来处理您的退款！如果你没有(或不想使用)信用卡或借记卡，这只是一种支付涡轮税的便捷方式，” [公司称](https://ttlc.intuit.com/questions/4204503-my-friend-filed-her-taxes-and-was-only-charged-the-39-99-tax-to-come-out-of-her-refund-i-did-the-same-thing-and-now-i-am-charged-twice) 。

事实上， [Intuit 的社区公告板](https://ttlc.intuit.com/community/Get-started-with-TurboTax/39-99-Refund-Processing/m-p/2244) 上充斥着客户对相同费用的抱怨。

### 退税预付款

然而，这不仅仅是涡轮税。据自由新闻报道，其他公司，包括 H&R 布洛克、TaxAct 和 TaxSlayer，当你选择用你的申报表来支付申报费用时，也会收取类似的费用。它被宣传为“无现金支出”，这使得它对那些目前不想支付(或没有现金可用)的人来说很有吸引力。

然而，如果你选择了那个选项，你实际上是在你的回报上收到了一笔预付款，正如我在这里写的 ，这从来都不是一个好主意:短期贷款总是会有很高的利率，这基本上就是 39.99 美元的费用。你可能不会“自掏腰包”支付任何东西，但没有什么是免费的。

TurboTax [提供了移除费用](https://ttlc.intuit.com/questions/1900517-how-do-i-remove-refund-processing-fees-in-turbotax-online-and-change-my-payment-method) 的说明，但请注意，“一旦您的电子申报单处于待决或已接受状态”，您就不能移除该费用这是你需要提前意识到的事情。例如，如果你去 H & R 街区办公室寻求帮助，你会想马上告诉他们，你不想花钱申请退款。你宁愿用卡或支票付款。
