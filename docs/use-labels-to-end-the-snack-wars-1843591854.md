# 用标签结束零食大战

> 原文：<https://lifehacker.com/use-labels-to-end-the-snack-wars-1843591854>

在这个社会距离和家庭隔离的时代，小事变得格外重要。阳光明媚的一天，在附近散步。通过视频聊天与爱人联系。确保你那该死的弟弟不会再把所有好吃的麦片吃光。



当我还是个孩子的时候，我的亲哥哥实施了他所谓的“24 小时规则”，即如果我有某种零食或剩菜，并且没有在 24 小时内吃完，其他人(他)就可以吃了。即使作为一个成年人，我也必须保护我丈夫留下的某些珍贵的剩菜，他的生活哲学是“请求原谅，而不是允许”。

好吧， [一个妈妈](https://www.instagram.com/p/CAdQBMJDfv0/) 已经受够了这些废话——以及由此引发的所有争斗。她想出了一个解决办法，拿出了她的记号笔，现在里维、博、寓言和阿彻都有了自己的草莓酸奶盒，所以关于谁在霸占好东西的争论终于可以停止了:

 [https://lifehacker.com/embed/inset/iframe?id=instagram-CAdQBMJDfv0&autosize=1](https://lifehacker.com/embed/inset/iframe?id=instagram-CAdQBMJDfv0&autosize=1) 

丽贝卡·伍尔夫在 Instagram 上写道:“当每个人都最喜欢同一种谷物时，它就被隔离了。”“我花了两个月的时间在谷物战争的战壕中斡旋，才弄明白这一点，但老狗最终会学会的。”

也许你也在处理“谷物战争”——或者也许在你的家里是烧烤片战争或巧克力片格兰诺拉麦片棒战争。无论什么零食已经成为一种有价值的商品——并且有包装——现在都可以贴上标签，这样就不会再有任何关于哪个食物属于哪个家庭成员的错误了。
