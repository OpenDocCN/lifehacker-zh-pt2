# 用味噌包住鸡蛋，做成开胃的煮鸡蛋

> 原文：<https://lifehacker.com/make-appetizer-worthy-hard-boiled-eggs-by-wrapping-them-1831021259>

对于煮熟的鸡蛋，蛋黄提供了所有的味道。尽管这是一种可靠且容易获取的蛋白质来源，但白葡萄酒味道太淡，没有调味就一点也不好玩。盐(显然)效果很好，但是如果你想让鸡蛋白充满坚果的鲜味，你应该用味噌。



我的绝对与 [完全](https://lifehacker.com/make-this-simple-miso-marinade-for-all-of-your-meaty-ne-1796723345) [对豆酱的投入](https://skillet.lifehacker.com/all-the-delicious-reasons-you-should-buy-and-use-miso-i-1788462313) 是 [不是](https://skillet.lifehacker.com/improve-almost-any-thanksgiving-dish-with-miso-1830473857) [一个](https://skillet.lifehacker.com/you-should-definitely-add-miso-to-your-thanksgiving-stu-1820053100)[秘密](https://skillet.lifehacker.com/improve-your-favorite-thanksgiving-pies-by-adding-a-lit-1830138102) ，而是把鸡蛋包在馅料里直到我看到 [三民](https://skillet.lifehacker.com/im-samin-nosrat-host-and-executive-producer-of-salt-f-1830721316) 在*盐、脂肪、酸、热*上做的时候我才想到。(这确实是一部不断奉献的电视剧。)在*盐*这一集里，Samin 遇到了 Nancy Singleton Hachisu，他们两人开始用味噌包裹煮熟的鸡蛋，然后让它们挂四个小时，然后轻轻地去除糊状物。我亲自尝试了一下，我可以确定结果是一个咸的、有味道的、美味的鸡蛋，甚至不需要撒一点氯化钠。如果你在想“哇，要用很多味噌才能包住一打鸡蛋”，你是正确的，但是，一旦从鸡蛋中取出，味噌可以用于其他目的，所以这是一个非常划算的使用方法。

味噌腌制的鸡蛋本身就很完美，但我想不出更好的方式来增加你的芥末蛋游戏 的乐趣，而且在节日派对上没有什么比芥末蛋更令人愉快的了。(它们也很漂亮，因为味噌让它们变成了令人愉悦的褐色。)
