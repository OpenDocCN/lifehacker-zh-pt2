# 为家庭杂务创建一个“招聘”公告栏

> 原文：<https://lifehacker.com/create-a-help-wanted-bulletin-board-for-family-chores-1841833307>

在学校度过漫长的一周后，孩子们不一定想在周末做家务。他们宁愿看电视也不愿吸尘。或者建造一个新的乐高玩具而不是折叠毛巾。(孩子们，我们也是。我们也是。)他们对日常工作之外的家务活特别抵触——你想让我喂狗、清空洗碗机、拔草？！



尽管如此，让孩子们参与进来还是很重要的；他们是家庭的一部分，一个家庭就是一个团队。每周，团队都有额外的杂务或工作需要处理。伊丽莎白·艾利森(Elizabeth Allison)为 [恐怖妈咪](https://www.scarymommy.com/household-help-wanted-bulletin-board/) 写道，当她厌倦了儿子们抱怨耙草、浇水或打扫卫生时，她想出了一个新系统:她创建了一个家庭“招聘”公告板，并将它挂在她家交通繁忙的区域。

### 它是这样工作的

在一周的时间里，每个家庭成员写下他们预计在周末需要帮助的家务，并把它们钉在黑板上。这些任务应该“范围合理”，艾利森的家人将其定义为能够在一小时内完成。那可能是清理一个壁橱，练习视觉单词，或者做一些庭院工作。

求助请求(每人两个)必须在周五晚上发布，并在周日晚上完成。

[佳佳解释道](https://www.scarymommy.com/household-help-wanted-bulletin-board/) :

> 尽管每个人都会在一周内仔细阅读招聘启事，但没有人会在周六早上之前做出任何承诺。那时，每个家庭成员都在两份张贴的工作请求上签上自己的名字。我发现，当我的孩子可以选择工作时，他们有更强的控制感，也更积极地履行自己的职责。为此，大人们最后选择，这样孩子们就有更多的任务可以选择。

乔布斯获得了——胜利！—完成后划掉，整个家庭在周末聚会，反思完成了多少，并感谢彼此的帮助。有傲气！还有感恩！有和谐！最重要的是，抱怨少了。

Allison 说，这个系统的伙伴关系方面甚至给了她多一个与孩子交流的机会

> 更重要的是，没有人被分流到房子的一个角落独自工作，这在我们使用木板之前有时会发生。相反，我们每个人都喜欢工作时的陪伴和交谈。我的孩子们不止一次自发地讲述了在学校发生的故事，当时他们正忙着在我身边扫地或洗碗。对我来说，这些自发的谈话是我们现在处理家务方式最快乐的意外结果。

董事会也给她的孩子时间，让他们在心理上准备好他们的周末确实会包括一些工作——能够选择他们帮助的事情可以给他们一种对任务的主人翁感。
