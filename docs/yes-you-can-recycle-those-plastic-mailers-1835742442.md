# 是的，你可以回收那些塑料邮件

> 原文：<https://lifehacker.com/yes-you-can-recycle-those-plastic-mailers-1835742442>

在任何一天整理你的邮件，很可能你已经收到了很多垃圾——从垃圾邮件到不必要的包装材料。幸运的是，你可以很容易地回收一些讨厌的塑料邮件。



作为背景，许多这些邮件是由#2 塑料，高密度聚乙烯，或#4 塑料，低密度聚乙烯制成的。大多数路边回收项目不接受邮寄者，因为他们可能会缠住回收机械(尽管许多项目可能会指定他们*将*接受硬质 2 号或 4 号塑料，如一些塑料瓶，因为这些不太可能造成这种复杂情况)。

但是塑料邮件*可以被*许多接受塑料袋的杂货店或大盒子商店回收，这些塑料袋是由相同类型的塑料制成的。家得宝、全食超市和塔吉特只是接受塑料邮件箱的商店的几个例子。你可以使用 [Earth911 的搜索定位器](https://search.earth911.com/) (并过滤#2 或#4 塑料)或 [How2Recycle 的目录](https://www.how2recycle.info/sdo) 找到离你最近的商店。

在你放下它们之前，检查一下你的邮件是否有回收标志，找出它的塑料类型。接下来，在网上搜索一下你自己的路边项目对 2 号或 4 号塑料的规定。正如 [How2Recycle](https://www.how2recycle.info/sdo) 所写的，你应该从邮寄者身上拿掉任何纸质标签(这可能会让分拣过程更容易)。如果你有一个用气泡包装的纸质邮件，这种包装也可以和塑料袋一起回收利用。只需从信封上剪下包装纸，然后分别回收。如果你还有问题，去商店了解一下他们的政策也无妨。
