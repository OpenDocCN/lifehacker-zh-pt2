# 使用“O&O 关闭 10”在 Windows 10 中增加您的隐私

> 原文：<https://lifehacker.com/increase-your-privacy-in-windows-10-with-o-o-shutup10-1833945565>

**Windows:** 虽然一个普通的 Windows 10 安装通常并不那么难管理，但你的操作系统下可能会有很多让你恼火的东西。也许这是你希望关闭的一两个功能，或者也许你关心微软在你的电脑上收集什么样的数据——或者它在哪里。



你可能甚至不知道你的操作系统中有哪些选项是你可以调整(或关闭)的，这就是这个巧妙命名为 [**O & O【关闭】10**](https://www.oo-software.com/en/shutup10) 的应用程序发挥作用的地方。这是一个简单的应用程序，它使得调整 Windows 10 的各个方面变得非常容易，这些方面通常被隐藏起来，或者普通人无法访问。更重要的是，该应用程序带有一些有用的警告，这样你就不会意外禁用一些你不应该禁用的东西(如自动更新)。

要开始，你所要做的就是下载应用程序并运行它。就是这样。没有值得一提的装置，这已经让我很激动了。当应用程序加载时，它看起来像这样:

你会看到一堆你可以打开和关闭的不同选项——有些可能已经启用了——还有一个方便的“推荐”栏，给你一些建议，告诉你是否应该*真的*改变那个设置。不过，我喜欢 O & O ShutUp10 的一点是，你可以通过简单地将鼠标悬停在每一行上并点击来获得关于每个设置含义的更多*信息，就像这样:* 

虽然你可能不应该只是浏览和启用所有推荐的东西*一起*，但我会用那个绿色的小勾号作为你探索这个应用的指南。启用任何相关设置，你就*可能*没事了。然而，一旦你开始进入黄色的“限量”类别，事情就有点危险了。例如，您可能不想禁止所有应用程序访问您的麦克风或摄像头，或者您可能想这么做。请记住，下次你准备参加视频会议时，你已经切换了这个设置。

类似地，关闭应用程序访问你位置的功能听起来可能对隐私有好处，直到应用程序开始为你提供另一种语言的内容，因为它不知道你实际上在哪里。然而，你*可以*阻止 Windows Update 下载第三方硬件驱动程序，假设你自己一直在做这种事情，并直接从制造商那里获得你的组件的最新驱动程序。

至于那个吓人的红色“不”感叹号，我就不去管它了。让应用程序在你的系统后台运行是有帮助的，也是好的，就像 Windows Updates 一样——除非你真的想在安装它之前看看其他人是否对一个主要的 Windows 更新有任何问题，我想。

如果你感到焦虑，你可以使用该应用程序来帮助你创建一个系统还原点(通过操作下拉菜单)，如果你想在做出任何调整之前得到一点保护。你还可以点击快速撤销你在应用程序中所做的所有更改，将自己恢复到普通的 Windows 10 体验。
