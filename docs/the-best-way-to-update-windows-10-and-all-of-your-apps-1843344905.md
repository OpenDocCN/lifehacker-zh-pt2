更新你的电脑和上面的应用程序，是每个人都应该知道如何做的最基本的维护工作之一。虽然你的操作系统应该通过代表你运行大多数更新来让你轻松做到这一点，但没有理由让它享受所有的乐趣。掌控您自己的命运:安装您自己的更新，并在您的操作系统决定之前获得最新的功能(和安全补丁)。

### 如何更新 Windows 10

要开始更新您的操作系统，请单击“开始”菜单，键入“更新”，然后选择“检查更新”出现的屏幕看起来有点像这样(如果你使用的是 Windows 10 的 [亮暗模式](https://lifehacker.com/how-to-make-the-most-of-microsofts-windows-10-october-u-1829524192) ，显然会有所不同):

单击“检查更新”按钮。根据 Windows 10 找到的内容，任何更新都可能会自动开始下载和安装。如果它是一个相当大的或主要的更新，你也可能不得不自己手动开始下载和安装。

这是更新 Windows 10 的基本方式。但是你还可以做更多的事情！我们继续。

### 考虑在大量更新发布时(或之前)暂停 Windows 更新

所以，微软还没有在 Windows 重大更新上取得的最佳成绩。如果有一个出现在你的待安装列表中——你会知道它的名字，通常是 Windows 10 的“功能更新”,后面是新的版本号——你可能会考虑按下那个大的“暂停更新 7 天”按钮。当你这样做的时候，更新将在一周内不会下载或自动安装，这给了你一点时间来看看是否有任何你需要知道的关于所述更新的灾难性错误。如果没有，让我们撕；当这一周结束时，你将没有真正的选择，因为在允许你再次暂停之前，你将被要求安装最新的更新:

### 确保 Windows 不会在您需要时重新启动您的电脑

你肯定想点击“更改活动时间”，这让你定义你通常使用你的电脑的时间。这样，Windows 就不会在这些宝贵的时间里安装更新并自动重启系统来完成这个过程。虽然你可以让 Windows 自己计算出你的“活动时间”，但我更喜欢强硬的方法:在我醒着的任何时候都不要更新我的电脑。

为了进一步防止意外重启，请单击 Windows Update 主窗口中的“高级选项”。您会看到这个屏幕:

我建议启用“更新通知”，这将会给你一个小小的提示，提醒你系统需要重启才能完成更新。在这里，打开“更新 Windows 时接收其他 Microsoft 产品的更新”为什么这没有被设置为默认的“是”，我永远也不会知道。功能和安全更新是一件好事。

### 
如何更新 Windows 10 电脑上的应用程序

当你想要更新你的各种 Windows 10 应用时，你通常有三个选择。

#### 对于您从 Microsoft 商店购买的 Windows 10 应用程序

只需启动微软商店应用程序，点击右上角的三点图标，然后选择“下载和更新”单击蓝色的“获取更新”按钮即可。

#### 对于从其他地方下载的应用程序

如果你幸运的话，你的应用程序——比如说谷歌 Chrome——会自动检查更新，并在更新出现时安装它们。你可能需要手动进行检查，但如果这个选项存在，它很可能隐藏在应用程序的设置中，甚至可能是“帮助”或“关于”菜单。

如果你的应用程序*不具备*自我更新的能力，你将不得不不时手动查询应用程序的开发者，看看是否有新版本存在。你也可以使用一个超级有用的程序来自动完成这个过程，比如 [**【修补我的电脑】**](https://patchmypc.com/home-updater-download) ，它可以帮你检查和安装。

我大约一个月运行一次这个应用程序，以确保我使用的所有东西都尽可能地更新。使用起来再简单不过了；我将它设置为一启动就检查任何潜在的更新，并静默安装任何更新。我不需要点击任何确认或任何东西。我只需等待，我所有合格的应用程序——这是一个相当全面的列表——都可以获得他们需要的任何更新，而无需我额外输入任何信息。

#### 更新应用程序:高级模式

如果你想变得真正厉害，你可以使用一个包管理器安装你所有的第三方应用程序，比如 [**【巧克力】**](https://lifehacker.com/install-and-update-all-your-windows-apps-at-once-with-t-1827588626) ，这样就可以通过一个单一的、易于管理的用户界面轻松更新它们。

软件包管理器并不适合所有人，但是像[**Chocolatey GUI**](https://chocolatey.org/packages/ChocolateyGUI)这样的应用程序使这个过程比使用命令行更容易处理，如果你不喜欢的话。
