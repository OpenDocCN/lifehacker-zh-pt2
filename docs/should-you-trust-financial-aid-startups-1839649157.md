# 你应该相信财政援助创业公司吗？

> 原文：<https://lifehacker.com/should-you-trust-financial-aid-startups-1839649157>

随着美国学生贷款债务加起来超过 1.5 万亿美元，难怪很多人想让支付大学费用更容易——也许还能更便宜一点。但随着科技领域涌现出一些雄心勃勃的初创公司，打算彻底改变我们支付大学学费的方式，普通借款人可能会怀疑它们是否合法。



我与非营利性组织学生贷款顾问协会(TISLA) 的创始人贝齐·马约特进行了交谈，以获得一位在金融援助领域工作了 20 多年的人的观点。

以下是这些新创业公司提供的服务，以及你在签约之前应该知道的事情。

### 创业公司承诺的财政援助是什么

一些网站声称可以免费帮你填写助学金申请。

在过去的几年中，一个获得了很多关注的是 [弗兰克](https://withfrank.org/) ，它允许你通过其承诺“简单英语”(或西班牙语)说明的门户网站提交免费的联邦学生援助(FAFSA)申请。申请人提交他们纳税申报表的照片，系统会剔除不适用于该申请人的问题，以加快流程。的发言人德鲁·克尔说，学生填写 FAFSA 的平均时间“略超过四分钟”，该网站的错误率低于 FAFSA 官方网站。

FAFSA 档案服务是完全免费的。那么它是怎么赚钱的呢？目前有三种方式，第一种是 Frank 可以从推荐给通过网站填写 FAFSA 的学生的学校获得佣金。

第二种是提供会员资格。弗兰克会员每月支付 19.90 美元，提前获得一部分经济资助，以填补支付过程中的任何缺口。

弗兰克还为那些认为他们应该得到更多联邦资助的学生提供经济资助上诉服务。它的价格为 144.95 美元，克尔解释说，这些客户可以获得个人服务，包括为他们写的上诉信。

与此同时，一项名为 [Mos](https://www.mos.com/) 的服务承诺其平台“使用一种表格，将你申请到你有资格申请的所有政府大学财政援助项目。”尚不清楚它在这一点上是如何赚钱的:它在常见问题中说，它不出售客户数据；对于没有在网上充分解释的“服务计划”,有一个退款政策。(我们已经向 Mos 寻求澄清，如果他们有回应，我们将更新这个帖子)。

### 没有他们怎么申请

但是还有另外一种免费申请经济资助的方法:简单地自己填写 FAFSA。随着 [截止日期](https://twocents.lifehacker.com/if-you-want-federal-student-aid-fill-out-your-fafsa-no-1838749614) 越来越适应大学申请日历和更漂亮的网站，这几年来申请变得越来越容易导航。你甚至可以通过一个 app 申请联邦援助 [。](https://twocents.lifehacker.com/you-can-now-file-fafsa-from-your-phone-1829435594)

“你永远都不应该为帮助付费，”马约特说。她说，大多数声称会免费帮你填写 FAFSA 的公司都是合法的，但在她看来，它们没有必要。“[FAFSA]的计算就是计算，”她说。

马约特说，每个州都有某种形式的大学入学办公室，可以帮助你免费跳 FAFSA 舞。你甚至可以参加你所在州的 FAFSA 日，在那里不同的地方提供免费的研讨会，帮助你填写 FAFSA，并协助申请奖学金。

她说，这些服务的用户可能会被推销给他们申请的联邦学生援助以外的各种贷款项目。Frank 在其隐私政策中解释了它如何使用您的个人信息，并允许您选择退出 [信息共享](https://withfrank.org/privacy-policy/) 。

### 你不需要为上诉信付费

“我觉得这有点掠夺性，”当我问她关于为你写经济援助请求书的收费项目时，马约特说。"他们利用恐慌的学生。"她曾与许多借款人合作，他们提交了上诉信，并向他们保证不会根据你写得有多好来评判你。“他们在寻找是什么导致了你的挣扎，你在做什么来改善它，你是否会见了学术顾问，”她说。

马约特解释说，许多需要对援助决定提出上诉的学生是那些改变专业的学生，他们需要超过四年的时间才能毕业——因此需要学术顾问的签字。但是其他学生可能正在处理 家庭收入变化、医疗债务，或者他们只是可能已经从另一所学校得到了更好的提议。

马约特警告说，如果你选择使用外部公司提交你的 FAFSA 或浏览你的选项来支付大学费用，“确保你理解条款和条件”。此外，阅读大量评论，运用常识。“任何时候你在考虑一个金融产品，无论是金融援助应用还是信用卡，你都需要做功课，”她说。
