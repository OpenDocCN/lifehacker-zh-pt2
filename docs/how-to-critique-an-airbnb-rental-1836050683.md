# 如何评论 Airbnb 租房

> 原文：<https://lifehacker.com/how-to-critique-an-airbnb-rental-1836050683>

在专门介绍 Airbnb 上噩梦经历的网站[【AirbnbHell.com】](http://airbnbhell.com/airbnb-guest-stories/)上，客人们分享他们与 [蚂蚁和蟑螂横行](https://www.airbnbhell.com/infestation-refund-charge/) [ns](https://www.airbnbhell.com/infestation-refund-charge/) 、拥有“假” [海边风景](https://www.airbnbhell.com/french-airbnb-features-fake-seaside-view/) 的单元，以及根本不存在的公寓住在一起的故事。



“我女儿通过 Airbnb 网站在新西兰奥克兰预订了一家 Airbnb。一位 [的评论者](https://www.airbnbhell.com/fake-airbnb-in-auckland-new-zealand/) 写道:“当她的车带她去那个地址时，大楼里一片漆黑。“没人开门。她无法通过电话联系到所谓的主持人...当她想知道发生了什么事时，有人开车来到附近的一处房产，告诉她那栋楼不是 Airbnb，他不得不告诉很多人这一点。

在很多情况下，比如对不存在的便利设施或公寓过度承诺，在 Airbnb 上写差评是合理的。在不那么极端的情况下，这是一个棘手的决定。与在猫途鹰或 Yelp 上写评论不同，在 Airbnb 上写评论更加个人化——毕竟，你是在评价某人的家。

“如果你在 Yelp 上评论一家餐厅，你不会直接谈论店主，但在 Airbnb 上，你基本上是在给主人评级，就像你在给酒店评级一样，”T he Points Guy 的编辑艾米丽·麦克纳特告诉。

一个简单的评论，尤其是对那些很少或没有评论的列表，会对主机的声誉产生负面影响。出于这个原因，如果你在 Airbnb 过得不太愉快，试着在你决定发帖之前把问题告诉主持人*,并在情况允许的情况下，在评论中尽可能保持公正。*

### 给主持人一个解决问题的机会

在旅途中，你应该给主人一个机会来解决你遇到的任何问题；意外时有发生，主人不能为一些不幸做好准备。例如，在东京逗留期间，我和我的朋友发现我们 Airbnb 的热水器停止工作。(这种情况也发生在隆冬，所以这是一个非常寒冷的时间没有温水。)

当然，在决定发布任何关于这个家的评论之前，我们联系了主持人。当我们在*天*都没有收到回复时，我们决定在我们的评论中加入一条意见，即沟通不理想，主持人理解这种情况。另一方面，一个试图尽快解决问题的主持人应该得到比你想象的更好的评价。

“例如，如果一位客人在评论中说互联网关闭了，但房东立即做出了回应，并联系了有线电视公司进行修复，这并没有造成任何伤害，”丹尼尔·博茨为 [Budget Travel](https://www.budgettravel.com/article/how-to-decode-an-airbnb-review) 写道。"事实上，这是一个好的*主持人的标志."*

除非在某些情况下——比如房源承诺有一个游泳池，而你只有一个浴缸——你应该允许主人解决任何问题，并在你自己的评估中记录这些互动(无论是积极的还是消极的)。然而，如果租金跟你想象的不一样，那就值得联系 Airbnb，看看你是否能让 T2 得到退款。

### 保持平衡，并提供如何改进的建议

当我最近住在伦敦的 Airbnb 时，我从来不需要闹钟来叫醒我；早上 6 点，我隔壁的单位开始施工，整个早上我都会在悦耳的钻井声中醒来。虽然这令人沮丧，但我仍然留下了一个平衡的评论，指出了其他伟大的位置和设施。(不过，我确实明确表示噪音是一个问题。)

写负面评论时，你应该尽可能公平，这是为了主人的利益，最终也是为了你自己的利益。“包括我自己在内的许多主持人都会阅读客人留给其他主持人的评论。如果你听起来吹毛求疵，你会被拒绝预订。”u/ [maccrogenoff](https://www.reddit.com/user/maccrogenoff/) 在 Airbnb 的一个 subreddit 中写道。"注意要坚持事实，不要吹毛求疵。"

你也可以选择在写完评论后私下给主持人发信息——在这里你可以给主持人提供如何改进的建议。如果你住在 Airbnb，很少或没有评论，一个新主人可能会受益于一条解释如何更好地欢迎客人的私人信息。(一开始的差评也会影响他们再次主持的能力。)

### 保持简洁

假设你有一次三星级的经历，但不确定如何进行评估；当所有这些都失败时，保持评论简洁，让其他客人从字里行间去体会。

“虽然 XXX 是一个可爱的人，但我不认为共享托管环境是他们最好的地方，”一名用户 [在 Airbnb 社区论坛上建议](https://community.withairbnb.com/t5/Hosting/Writing-a-polite-but-negative-review/td-p/828187) ，并回应一名*主持人*与一名客人的糟糕经历。

没有理由你不能用同样的建议来改善在 Airbnb 的糟糕体验。不一定要充满敌意甚至消极——只是总结你旅行的诚实评论——但要准备好主人对你的回应的回应，这也在网站上公开。换句话说，不要做一个混蛋，除非情况绝对需要。
