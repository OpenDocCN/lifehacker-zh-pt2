# 开着电视睡觉真的对你有害吗？

> 原文：<https://lifehacker.com/is-falling-asleep-with-the-tv-on-really-that-bad-for-yo-1840613894>

在一个完美的世界里，我们都应该在睡觉前至少半小时关闭电子设备，洗个澡，读一本放松的书，冥想，然后在刚洗过的熏衣草香味的床单上昏昏欲睡。但对我来说——以及大约 60%的美国人——实际上，这个过程包括在我们试图入睡时看电视。

Watch

我知道，我知道——开着电视睡觉不仅仅是一首不被人欣赏的比利·周歌曲 :这也应该是一个可怕的习惯。我们被指示 [从卧室](https://www.becomingminimalist.com/18-darn-good-reasons-to-get-the-tv-out-of-the-bedroom/) (或者一开始就不要在那里放电视)，这很好，但多亏了流媒体服务，我们也可以从笔记本电脑、平板电脑和手机上“看电视”，所以基本上随时都可以访问。真的有那么糟糕吗？

### 为什么看着电视睡着会让人感觉舒服(对一些人来说)

虽然我不能代表那些开着电视睡着的人，但对我来说，很多吸引力与我的焦虑和 PTSD 有关。无论我做多少呼吸练习，或者读多少无聊的文章，只要我觉得自己快要睡着了，我的大脑就会播放我不想去想的所有事情的蒙太奇画面:最后期限、失败、生病的朋友和亲戚、政治、超级细菌，如果我在电子邮件中无意中冒犯了某人，我们死后我们的身体会发生什么——所有我试图从脑海中赶走的东西，以便在一天中保持高效。然后，当我在半夜醒来，脑海中出现生动的 PTSD 闪回时，我需要用其他东西来替代那些图形图像，坚持看一集《30 块石头》或《黄金女孩》会有所帮助。

据睡眠城市 的首席研究官罗斯·麦克道尔说，我并不孤单。她说:“许多人已经习惯了开着电视入睡，以屏蔽其他声音，或者在入睡时转移自己的想法或焦虑感。”"另一些人觉得电视的声音很舒缓，可能很难安静地入睡。"

### 为什么专家说看着电视睡着不好

在听取了各种睡眠和心理健康专家的意见后，很明显他们不喜欢人们开着电视睡着。例如，丽贝卡·考恩博士，一位有执照的专业顾问，强调睡眠卫生很重要，因为它与我们的心理健康相关。此外，她解释说，开着电视睡觉会扰乱褪黑激素的产生，褪黑激素是一种帮助我们睡得香甜的激素。“以我的专业观点来看，我不认为在任何情况下睡觉时开着电视是可以的，”考恩说。

同样，注册护士和更好睡眠委员会的睡眠专家 Terry Cralle 解释说，睡前看电视会增加我们的睡眠债——例如，如果我们被一个节目吸引，继续看下去而不是去睡觉。她还指出，当我们睡觉时，我们的大脑仍然在处理声音，所以整晚开着电视会扰乱我们的睡眠。同样，《8 小时睡眠悖论的作者马克·伯亨内博士说，在深度睡眠之前的睡眠阶段，我们的大脑意识到声音，并可能对它们做出反应。他告诉 Lifehacker 说:“在深度睡眠的梦境中，音频甚至会影响你所做的梦。”

Burhenne 还提到电视发出的蓝光是另一个缺点，因为它是大脑保持警觉的信号，可能会阻止我们进入深度睡眠。“最终，当你睡觉时，你要避免刺激大脑，”他说。“你的身体需要高质量的睡眠，通过清除不必要的细节和巩固你真正需要的信息来‘重新格式化’大脑。”

### 睡眠不是万能的

当然，开着电视睡觉并不理想，但并不是每个人都认为这应该是一个严格的规则。例如，睡眠研究员珍妮·乔伊博士 说，在公布睡眠建议时，不会考虑个体差异。“就像没有一种特定的饮食适合每个人一样，也没有一种完美的睡眠方式适合每个人，”她说。

是的，对一些人来说，这意味着开着电视睡觉。但乔伊说，如果一个人在一天的早些时候出去，在某个时候做些运动，电视的蓝光可能不会扰乱他们的昼夜节律，他们可能会在开着电视的情况下睡得更好。她建议使用 [睡眠活动图](https://www.verywellhealth.com/what-is-actigraphy-3015130) 来检查你开着电视的睡眠质量。如果这表明你睡得很好，那么你应该可以开着电视睡着——只要你的伴侣不被它打扰。她说:“如果一个人在这种情况下睡得很好，那么开着电视睡觉本身并没有什么不好。”

事实上，乔伊指出，在很多情况下，睡着时开着电视可能是有益的。例如，电视可以淹没外部噪音，如犬吠声、警笛声、邻居打架声和枪声。同样，堪萨斯睡眠 的睡眠科学认证教练伊娃·科恩说，对于患有 PTSD 等精神健康疾病的人来说，从电视节目的倒叙中转移注意力在某些情况下可能会有帮助。但是，像大多数其他专家一样，她强调最好不要在睡觉前两小时内看电视，并咨询治疗师以找到更好的应对机制。

### 改尝试什么

如果你是一个不开电视绝对无法入睡的人，并且不愿意尝试其他方法，我们采访的几位专家建议你在晚上戴上蓝色遮光眼镜。虽然理论上很棒， [蓝光阻隔眼镜真的没什么用](https://lifehacker.com/do-computer-glasses-really-work-5980509) ， [美国眼科学会也不推荐](https://lifehacker.com/does-blue-light-actually-affect-your-health-1839035806) 。

相反，Cralle 建议确保你的电视睡眠定时器是开着的——理想状态是 30 分钟，最多一个小时。她还鼓励我们在入睡时思考我们在看什么。“如果你认为为了放松和入睡，你必须在睡前看电视，那就看一些无聊的东西:没有政治、枪击/暴力或‘侦探小说’，”她说。她建议选择“一部低调的情景喜剧——甚至更好，一部你已经看过一百万遍的重播”，而不是那些可能激怒你或让你想继续看下去的东西

但是如果你想睡觉时不开电视，那么什么是有效的替代品呢？Joy 说她更喜欢通过耳机而不是电视来听 [双耳节拍音乐](https://lifehacker.com/i-dose-sends-binaural-beats-to-your-brain-398342) 。尽管研究很少，[2018 年的一项小型研究](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC6165862/) 确实发现双耳节拍有助于提高睡眠质量。其他人，像考恩，建议听着音响或放松的音乐入睡，而克莱尔说其他人很幸运听着有声读物或播客入睡。

科恩建议，不要把注意力集中在声音上，而是通过睡前阅读小说故事来引导你生动的想象力。“这将有助于你更好地表达自己的情绪，降低压力水平，”她解释道。

如果睡觉时开着电视的部分吸引力在于它可以兼作夜灯，那么你可能想看看一些新的睡眠照明选项，包括反映你昼夜节律的智能灯泡( [，像 GE](https://www.cbyge.com/pages/pdp-tunable-white) 的这些灯泡)或应用程序激活的照明。

所有这些都需要反复试验才能找到最适合你的睡眠系统。如果你是一个不看电视就绝对睡不着觉的人，而且这并不影响你的睡眠质量，你要知道这个习惯不会让你失眠。
