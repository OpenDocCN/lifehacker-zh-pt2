# 现在你可以用谷歌找到一个新冠肺炎考试中心

> 原文：<https://lifehacker.com/now-you-can-find-a-covid-19-testing-center-using-google-1842946499>

现在，当你在谷歌上搜索新冠肺炎时，你可以找到位于你附近的病毒检测站点的信息。



谷歌周五推出了搜索测试网站的功能。该搜索引擎现在能够显示美国 43 个州的 2000 多个新冠肺炎考试中心的信息。

这只是卫生当局批准发布的表面位置，因此无法显示您附近的所有测试中心。对于缅因州、密苏里州、新泽西州、俄克拉荷马州、俄勒冈州和宾夕法尼亚州，目前无法显示任何结果， [*The Verge* 报道](https://www.theverge.com/2020/4/17/21225828/google-showing-covid-19-testing-centers-search-results) 。

增加测试中心是谷歌搜索结果众多变化的一部分。现在，无论何时你搜索任何与新冠肺炎相关的东西，你都会在页面左侧看到一个 SOS 警报栏。从那里你可以快速导航到关于病毒的信息，包括症状、治疗、新闻、统计数据，是的，你可以在那里进行测试。

点击“测试”标签，在旧金山显示这些信息。

当你点击进入时，你会在地图上看到所有的新冠肺炎测试中心，以及在那里测试的要求。

本月早些时候，苹果还开始在苹果地图中显示新冠肺炎测试中心。像谷歌一样，被收录的考试中心旁边也列有信息，表明在该特定设施进行考试的要求。
