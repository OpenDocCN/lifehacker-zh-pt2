# 知道当一个名人被指控骚扰这个 Chrome 扩展

> 原文：<https://lifehacker.com/know-when-a-celebrity-has-been-accused-of-harassment-wi-1832103707>

我承认我不了解很多名人的八卦新闻。这意味着我通常不知道一个演员在和谁约会，或者他们刚刚有了一个孩子。我也不知道什么时候有人被指控性骚扰或性侵犯，除非这件事引起了温斯坦级别的关注。

Watch

[在 Chrome 扩展中发现骚扰者](https://chrome.google.com/webstore/detail/uncover-harassers/lafpdaabjfbkfhkcmlellicnacnhkgjc) ，它可以帮助我知道我正在阅读的人被指控骚扰。这个想法是，你可能会宣传某个特定的名人或他们的作品，而没有意识到他们被指控做了一些可怕的事情。

该扩展会自动突出显示你正在阅读的新闻文章或其他媒体中涉嫌骚扰者的名字。当你将鼠标悬停在某人的名字上时，会出现一小段信息，解释他们被指控(或定罪)的罪名。还有一个相关新闻文章的链接，你可以从中了解更多信息。

这不是万无一失的。为了扩大范围，某人不仅需要被指控犯罪，还需要在主要出版物上报道该犯罪。

目前，扩展包括美国，印度和巴基斯坦的名人。美国名人的主要来源是《纽约时报》，所以它也主要针对“大人物”，而不是每个名人。如果你遇到某人没有被包括在内，应该被包括在内，还有一个在线表格，你可以在那里提出补充建议。

值得注意的是:填写表单并不保证会添加一个名字。为了让它进入扩展，创作者将从一个有信誉的来源用一个新闻故事核实事情。

也就是说，如果你不是那种“知情”的人，这可以帮助你保持消息灵通。
