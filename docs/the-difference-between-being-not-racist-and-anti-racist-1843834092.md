# “不是种族主义者”和“反种族主义者”的区别

> 原文：<https://lifehacker.com/the-difference-between-being-not-racist-and-anti-racist-1843834092>

种族主义是影响有色人种安全和福祉的瘟疫。当我们继续目睹种族主义导致的不公正时，有一点是清楚的:我们的白人盟友仅仅“不是种族主义者”是不够的；他们必须是反种族主义者。如果你想知道有什么不同，我们将为你分析一下。

### **种族主义的定义**

种族主义有许多层面；这一点应该永远被理解。种族主义的基本定义是基于你的种族优于他们的信念而对人们的公然歧视、偏见和不公正待遇。

种族主义不仅仅是通过仇恨言论或身体行为来残酷对待人们，它还通过就业流程、不公平的银行业务、医疗保健、社会经济差距和刑事司法系统系统地表现出来。

下面是由 美国国家非裔历史博物馆&文化概述的不同类型的种族主义 [:](https://nmaahc.si.edu/learn/talking-about-race/topics/being-antiracist)

1.  **个人种族主义**是指个人的信念、观点和行为，这些信念、观点和行为以有意识和无意识的方式支持或维护种族主义。例子包括相信白人的优越性，不雇用有色人种，因为“有些事情感觉不对劲”，或者讲种族主义笑话。
2.  **人际种族主义**发生在个体之间。这些是微侵犯的公开表达，通常涉及诽谤、偏见或仇恨的言语或行为。
3.  **制度上的种族主义**基于一个组织的周围和内部。这些待遇、政策或做法是歧视性的、不公平的、有偏见的，是基于种族的，给予白人比黑人更好的结果。尽管制度政策可能没有明确规定种族，但它们的意图是在有色人种社区之外建立优势。
    例如:与白人学生占主导地位的学校相比，有色人种学生上学更频繁的学校系统将会资金不足且过于拥挤。

4.  结构性种族主义是跨越机构和社会的整个种族偏见体系。它允许白人享有特权，同时给有色人种带来极大的不利。
    例如:在主流电影、媒体和娱乐中将有色人种描绘成罪犯或“暴徒”的种族刻板印象。

### **非种族主义者和反种族主义者的区别**

不做种族主义者意味着在你人性的某处，你意识到所有人都有尊重和宽容的平等权利。当你*不是种族主义者*时，你不会认为自己更好或更值得这个世界上的任何东西，因为尽管我们不都一样或有相同的经历，但我们有权获得同样的 [“生命、自由和追求幸福”](https://www.ushistory.org/declaration/document/)

然而，仅仅因为你可能不是种族主义者，并不意味着你不是同谋或参与维护由于系统性种族主义而给予的利益和特权

另一方面，反种族主义意味着你在积极地做反对种族主义的工作。《国家行动委员会国际视角:妇女和全球团结》 [申明](http://www.aclrc.com/antiracism-defined) “反种族主义是通过改变制度、组织结构、政策、做法和态度来确定和消除种族主义的积极过程，以便公平地重新分配和分享权力。”

### **为什么反种族主义势在必行**

仅仅说“我不是种族主义者”是不够的。如果来自边缘化社区的人不会因为种族主义制度导致的国家批准的谋杀、监狱和失败的医疗实践而失去生命，那就太好了。虽然种族主义是一个普遍而致命的问题，但你必须积极努力与之斗争。为了不助长种族主义，你必须公开谴责它并要求改变。

作为一名反种族主义者，你承认有些东西被严重破坏了，必须被修复。反种族主义需要真正的战略和实践来挑战和反对所有种族主义。

以下是一些反种族主义的方法:

*   当你在朋友、家人或同事中听到种族主义笑话时，大声说出来。
*   向需要支持的知名事业捐款，以帮助边缘化群体。
*   了解你的特权从何而来，以及它如何对他人产生负面影响。
*   更加积极地消除种族主义制度(服务不足的学校、刑事司法改革、医疗保健宣传)

种族主义不是一夜之间发生的，也不会就此结束。但是，如果我们都积极地尽自己的一份力量，我们就能真正改变这个世界。
