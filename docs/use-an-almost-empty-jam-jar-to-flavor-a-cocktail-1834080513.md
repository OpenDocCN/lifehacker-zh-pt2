# 用一个几乎空了的果酱罐给鸡尾酒调味

> 原文：<https://lifehacker.com/use-an-almost-empty-jam-jar-to-flavor-a-cocktail-1834080513>

果酱是极好的鸡尾酒配料。它也非常粘，这意味着从果酱罐中取出那些最后看似无关紧要的部分需要一些创造性的问题解决方案。幸运的是，乙醇是一种很好的溶剂。

Watch

用果酱残渣给酒调味其实很容易。只要拿一盎司半或两盎司你最喜欢的酒，倒在罐子里，摇，摇，摇。

用细网筛过滤你的新水果浸液，像使用任何调味酒一样使用它。我做的樱桃杜松子酒只加一点补药就很完美了，但如果放在树莓酒或[【spritz】](https://skillet.lifehacker.com/add-some-sparkle-to-your-weekend-with-a-cranberry-sprit-1830342077)(上面放些起泡酒)里也会很可爱。

如果事情很糟糕，你没有精力调一杯味道浓郁的鸡尾酒，知道一盎司浸有橘子果酱的龙舌兰酒是一杯不错的酒，芒果保存的朗姆酒也是。事实上，我想不出任何组合会是一个糟糕的镜头；你想做什么就做什么。
