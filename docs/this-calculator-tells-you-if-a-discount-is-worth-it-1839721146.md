# 这个计算器告诉你一个“折扣”是否值得

> 原文：<https://lifehacker.com/this-calculator-tells-you-if-a-discount-is-worth-it-1839721146>

这几天感觉好像所有的东西都在打折。但是并不是所有的销售都是平等的。在假日期间，认识到这一点尤其重要，因为购物假日的特点是复杂的折扣方案，有时也会带来同样复杂的限制。



如果心算不是你的强项，看看 [黑色星期五计算器](https://www.omnicalculator.com/finance/black-friday) 。“很多这些看似真实的销售往往是数学陷阱，”波兰博士生汉娜·帕姆拉说，她为 Omni Calculator 项目建立了这个模型。她包括了九种不同的折扣方案——打九折、买一送一、购买多件商品打折等等——来帮助你决定一个折扣是否值得你花时间和金钱。

虽然这个计算器是以黑色星期五命名的，但你可以在任何时候使用它来计算一个假定的销售价格。

为了检验这个计算器，我在鞋类零售商 DSW 测试了一个简单的“所有商品八折”销售。我输入了一双运动鞋的原价 69.99 美元，20%的折扣，以及我要支付的 7%的附加税。计算器算出节省了 14.98 美元，gif 上有一只正在打盹的加菲猫问我是否值得起床。

我可能已经完成了将商品添加到我的购物车、输入我的运输信息以确定税收，以及添加促销代码的步骤。但是让我们面对现实:到那时，我可能会点击额外的两个按钮来购买鞋子。

但这很简单。让我们试试更复杂一点的。计算器可以处理以下折扣场景:

*   %关闭
*   第二个或第三个产品的%折扣

*   二对一
*   三比二
*   三分之四
*   双倍或三倍折扣(例如:50%折扣+ 30%折扣)
*   多个单位的折扣

买一送一打五折怎么样？我在鞋子嘉年华上查看了网上销售情况(我并不想让这成为打折鞋店的一场战斗，但我们在这里)以测试计算器的 BOGO 功能。当然，我需要一双雪地靴来应对佛罗里达漫长的冬天，这要花掉我 99.99 美元。但是如果第二件商品打五折，也许我会买一双 Doc Martens 牌的。这是一笔好交易吗？

计算器告诉我，我将节省 48 美元多一点。对于两双鞋，我会为它们节省 23.7%的全价。当然，省钱的最好方法是只买我真正需要的那双。但是如果你的购物清单不仅仅包括你自己物质上的东西，计算一下可以帮助你最大化你的预算。

当你开始比较假日销售时，有一件事需要警惕:警惕价格通胀。我们以前写过这样的报道:在黑色星期五和网络星期一之前的几周，零售商通常会 [略微提高价格](https://lifehacker.com/check-the-prices-for-your-black-friday-buying-now-1830338477) ，这样他们以后提供的折扣看起来会更好。

“零售商会做任何事情来吸引你的注意力，让交易看起来比实际情况更好，”Pamula 在电子邮件中说。将一点点 [预先研究](https://lifehacker.com/five-best-price-tracking-tools-1692745053) 与一些数学强化结合起来，你将处于一个有利位置，可以明智地花钱，而不会被一笔糟糕的交易所欺骗。
