# 与嘶嘶作响的共产主义者聚会

> 原文：<https://lifehacker.com/party-with-the-fizzy-communist-1839541767>

关于运行 Skillet，我最喜欢的事情之一是我与读者建立的关系。除了评论区——这是互联网上一个令人惊讶的冷门——可爱的人们经常给我发短信或电子邮件，告诉我建议、要求和食谱。

有一次这样的人最近给我发来了下面这条信息，信息是“想念你。共产主义者“lol”，这既是一个坚实的燃烧，也是一个非常甜蜜的情绪。(不清楚这个名字的确切来源，但我显然很喜欢它。)

正如你所看到的，这是一种四种成分的鸡尾酒( [经典](https://www.grahamandtonic.com/recipes/all/the-communist) )，但杜松子酒、樱桃和柑橘的组合是一个非常可重复的模板。我试着把柠檬放进去，把橘子放进去，但读起来是苦的，所以我加入了起泡的柑橘汽水，以确保酸甜口味都有。要做到这一点，你需要:

*   1.5 盎司杜松子酒
*   3/4 盎司樱桃利口酒
*   大约 2 盎司的水，或者其他柑橘汽水

在加满冰块的鸡尾酒摇壶中摇合杜松子酒和利口酒，直到变凉。滤入一辆双座轿车，加入苏打水。
