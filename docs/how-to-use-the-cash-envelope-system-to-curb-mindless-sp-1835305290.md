# 如何使用现金信封系统来抑制盲目消费

> 原文：<https://lifehacker.com/how-to-use-the-cash-envelope-system-to-curb-mindless-sp-1835305290>

预算的某些部分很容易设定和坚持。你的租金不会波动。你的健身会员可能是一个可预测的数额。你大概知道你需要买多少汽油才能在这个工资周期来回工作。



但是，那些固定的和可预测的开支会让你觉得你已经征服了你的预算，更灵活的分类会让你建立的整个系统崩溃。你说你这个月要花 50 美元买衣服，那为什么有这么多 H&M 包放在走廊里？你朋友的生日晚宴本来应该是一件低调的事情，直到有人点了三份开胃菜和一瓶葡萄酒，并说“我们平分吧，伙计们。”

当你的预算出错时，你可以花时间反思、调整，并在下个月再次尝试——这就是你的预算作为一份活文件的美妙之处。但是如果你正在寻找一个额外的自我控制的方法，那就放弃旧的方法，试试预算的现金信封系统。

信封系统不仅仅是我妈妈在 90 年代实践的东西！预算管理方法是由金融大师戴夫·拉姆齐推广开来的。我不喜欢拉姆齐宣扬的一切，但我承认，他控制你支出的一些策略是可靠的。

## 如何使用现金信封系统

创建现金信封系统的第一步是计算出你的每月预算。你可以决定哪种方法适合你，以及给你生活的各个部分分配多少钱。不要偷看你邻居的卷子。这是你自己独特的预算。没有错误的答案。

然后选择你想跟踪或控制你支出的类别。当谈到网上消费时，也许你有点太“加入购物车”了。也许你在假期觉得自己格外慷慨，但最终却因为你的慷慨而得到了一张信用卡。

改用现金信封的流行类别包括“杂货”、“礼物”、“娱乐”和“外出就餐”在用这些现金购物之前，你在月初设定放入信封的现金数量。

当你没钱的时候，你就倒霉了。不要放过去，不要多打自动提款机，不要偷出其他信封。你这个月就完了。你必须等到日历翻过来才能取出新的现金来填满你的信封。

至于你的交易工具，拉姆齐出售一系列 [产品](https://www.daveramsey.com/store/product/starter-envelope-system?int_cmpgn=no_campaign&int_dept=dr_blog_bu&int_lctn=Blog-Text_Link&int_fmt=text&int_dscpn=envelope_wallet_system_product_review_blog-starter_envelope_system_prod_link) 来让你的现金有条不紊。但你不需要花 14.99 美元或更多(加上运费)买一本花哨的假支票簿。你可以使用正面写有类别的普通旧信封。每次你买东西的时候，把总数写在信封上，或者把收据塞进信封。

这不是一个到处挥舞大量现金的公益广告。你带多少取决于你的预算。如果你认为一个信封对于你的日常出入来说太花哨了，把每一类的账单用一个回形针夹在钱包里，同时附上一张标明类别的纸条。

这里有一个信封系统在实践中的例子:你出门去看电影。你可以带着信封或剪好的钞票去“娱乐”如果是同花顺，那么你和你的约会对象可以每人得到一份爆米花。如果不是，那就去体验电影人生吧！—也许你应该在出门前在口袋里放些零食。信封主宰一切！

## 信封系统起作用是因为它很痛

当我和朋友在酒吧见面时，我经常第一个到达，因为我是一个焦虑的过度成功者。当我的朋友来的时候，我可能已经喝了一半的酒，而那时我已经是一个无拘无束、慷慨大方的富翁小姐了。“哦，我有一个标签打开了！”我会说。“就记在我的账上吧！”三十四美元，三个小时后，我们离开了欢乐时光。这很好，直到我回头看这个月，意识到我在快乐时光花了多少钱。

取而代之的是，我在出门时从我的大“酒吧和餐馆”信封里拿出 20 美元——我把它留在家里会用得更久。我点了一杯饮料，而不是在酒吧里记下模糊的账单。酒保给我酒，告诉我价钱。我给他们一些钱，他们给我一些零钱。现在我可以看到这个特别的夜晚我的预算还剩多少了。

信封系统之所以有效，是因为它不好玩。触摸钞票并把它交给收银员 [比其他支付方式更能伤害你大脑和灵魂内部的](https://twocents.lifehacker.com/save-money-by-embracing-the-pain-of-paying-1828933622) 。这不会阻止你进行每一项购买，但必须拿出现金可能会帮助你抑制超出你预期的挥霍冲动。没有人想面对一个还剩一个月的空信封。

## 网购呢？

那些你送货上门的餐盒，自动装运的狗粮，你在易贝买的廉价裤子。你可以给我一百万个理由说“信用卡比现金好，而且，你不能在网上用现金！”

所以也许这个方法不适合你。或者可能是为你准备的，但仅限于几个特定的类别:花式咖啡、室内植物、运动鞋、庭院销售。如果退订老海军不断的电子邮件足以让你远离网上购物的冲动，也许现金信封可以帮助你更进一步。

说到底，戴夫叔叔对这种预算系统的论点是正确的:“信封系统之所以有效，是因为它迫使你有意识地决定把钱花在哪里，怎么花。”你所花费的只是几个信封——如果你重复使用那些你收到的垃圾邮件，你也可以节省信封。
