# 这个网站使用人工智能来发现隐私政策中的问题

> 原文：<https://lifehacker.com/this-site-uses-ai-to-find-issues-in-privacy-policies-1838335480>

每当你注册一个新的应用或服务，你可能也同意一个新的隐私政策。你知道，那种不需要阅读就可以快速滚动的难以置信的长文本块？

Watch

[Guard](https://useguard.com/) 是一个使用人工智能解读超长隐私政策的网站，然后着重指出其中可能存在问题的任何方面。

一旦它通读了一个网站或应用程序的隐私政策，它会根据该政策给服务打分，并就你是否应该使用它提出建议。它还带来了与公司相关的任何丑闻的新闻故事和任何安全威胁的信息。

例如，Twitter 对这项服务的评级为 D。Guard 建议您避免使用该应用程序。最大的威胁？该公司的隐私政策规定，它可以出售或转让您的信息。

目前，你只能看到 Guard 决定分析的服务的评级，包括 youTube、Reddit、Spotify 和 Instagram 等大多数主要应用。然而，如果你对某个特定应用程序的评分感兴趣，你可以提交给服务并要求完成。

随着支持服务列表的增长，这可能会成为查看您在手机或电脑上使用什么以及了解您的数据如何被使用的更可靠的资源。
