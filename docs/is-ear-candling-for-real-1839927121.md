# 耳烛是真的吗？

> 原文：<https://lifehacker.com/is-ear-candling-for-real-1839927121>

很多人发誓说耳烛很有效，如果你想做好人并且不使用棉签 ，这种水疗服务看起来是清除耳垢的合理方式。不幸的是，这些说法好得令人难以置信。

### 有用吗？

耳烛的基本概念是，管状的蜡烛会产生真空，从耳朵里吸出耳垢。(一些从业者声称有其他好处，我们可以说，这些好处有些牵强。在这里，我们将把重点放在更合理的东西上。)

但是 1996 年的一项研究 发现，如果你将耳烛插入一个可以测量压力的模型耳朵，不会产生真空。嗯。

这项研究还测试了蜡烛底部堆积的蜡状污垢。如果这项技术按照预期的方式发挥作用，那么这些东西应该主要是耳垢。但是化学分析发现它是...蜡烛蜡。与此同时，接受过耳烛检查的人耳朵里的耳垢并没有减少。

简而言之，行不通。2006 年的一篇评论认为“没有数据表明它对任何病症都有效。”

### 有什么风险？

好吧，所以没用。但是有没有造成什么伤害呢？不幸的是，它可以。

如你所料，蜡烛烧伤并非闻所未闻。你的头发离火焰很近，如果从业者不把它举起来(或者如果他们把它楔在适当的位置，但它会掉下来)，你可能会被烧伤。

但是，即使一切都如设计的那样正常工作，烛照实际上可能会给你的耳朵增加耳垢。请记住，没有吸力，所以蜡更有可能向下而不是向上。有大量的案例研究表明，人们在耳烛后耳朵里有蜡烛蜡； [这里有一个](https://www.ncbi.nlm.nih.gov/pubmed/23321892) 。

FDA 已经 [警告](https://www.fda.gov/consumers/consumer-updates/dont-get-burned-stay-away-ear-candles) 反对这种做法。他们说:“在过去的十年里，FDA 已经收到了烧伤、耳膜穿孔和耳道堵塞的报告，这些都需要通过使用耳烛进行门诊手术。”

### 底线

耳烛不能清除耳垢，实际上可能会增加耳垢(更不用说烧伤的可能性了)。美国听力学学会 [称这种做法](https://www.audiology.org/news/ear-candles-and-candling-ineffective-and-dangerous) “无效且危险”，所以最好避免。
