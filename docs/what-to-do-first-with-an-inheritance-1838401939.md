# 继承遗产首先要做什么

如果你收到一大笔遗产，你会怎么做？如果你想在财务上谨慎一点，答案是什么也不做，至少一年内不做，T2。

> 布拉德利建议新继承人进入“自由决策区”至少几个月到一年的时间来考虑各种选择。在此期间，她建议创建一个她所谓的专业人士智囊团，比如财务顾问、会计和律师。
> 
> 专家说，在计划到位之前，继承人应该将任何现金和人寿保险收益存入联邦保险的银行账户，而不是与配偶的联合账户。

是的，你没看错——专家建议将遗产存入你自己的银行账户，而不是与配偶共享的账户。(如果你把你的遗产完全分开，在离婚的情况下，你更有可能被认为是“单独财产”——尽管你想和律师见面了解全部细节。)把你的钱放在家里除了你没人能拿到的地方，然后不要碰它。

等待一年不仅能防止你把遗产挥霍在一时的奢侈品上，还能给你一个标准的回应，让人们以为你的新财富会让你更愿意接受经济援助的请求。不，你不能给你的朋友或表亲(或配偶或孩子)额外的现金来资助他们最新的商业想法或其他任何东西。你今年不会花掉任何遗产。

在这一年结束后，在你超越了围绕你的损失和遗产的直接情绪后，你将能够对如何处理这笔钱做出更深思熟虑的决定。你不会因为悲伤、激动、琐碎或恐惧而采取行动，你会有时间与会计师和律师讨论遗产带来的所有选择和影响。

所以不要马上花掉遗产。即使你想用遗产来解决眼前的财务问题，比如还债，给自己几个月时间咨询财务专家，了解与遗产相关的一切，从税收到婚姻财产法。这样，你就不会浪费指定给你的一分钱。
