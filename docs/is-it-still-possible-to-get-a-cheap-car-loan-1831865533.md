# 还有可能拿到便宜的车贷吗？

> 原文:[https://jalopnik . com/is-it-尚能得到一笔便宜的汽车贷款吗-1831865533](https://jalopnik.com/is-it-still-possible-to-get-a-cheap-car-loan-1831865533)

作为 Jalopnik 的常驻购车专家和专业购车者，我会收到电子邮件。很多邮件。我决定挑几个问题，试着帮帮大家。这个星期，我们正在讨论，消失廉价汽车贷款，现金购买的最佳策略，找到一个廉价的家庭汽车。

首先:所有的超廉价贷款都发生了什么？

> 我有一个关于当前利率的小问题，对于合格的买家来说，现在的利率是多少？希望在 60-72 个月内融资 5.4 万美元，并在 36 个月内还清贷款。此外，2%的贷款会一去不复返吗？

利率正在上升，预计短期内不会下降。在二手车上，我看到即使拥有一级信用的人也能从经销商那里获得大约 4.5%的利率，有时从信用社获得的利率更低。对于新车，除非制造商提供激励性融资，否则价格可能会徘徊在 3.5%至 4%左右。超级廉价贷款确实来自汽车制造商，但不像过去那样普遍。一些品牌仍然提供低年利率贷款，甚至零利率融资，但这些优惠可能不符合你正在购买的车型。

我发现越来越多的二手车买家拥有一级信用，并从信用合作社获得大量贷款，经销商告诉他们“是的，我们不能碰它。”这意味着，如果他们没有进行预先批准，他们可能会为二手车支付更多的利息。然而，对于新车，经销商通常可以达到或超过小银行提供的利率。就像买车一样，货比三家是值得的。

接下来，如果您支付现金，获得最低价格的最佳方式是什么？

> 所以，我是一个老练的买家，会以低于发票的价格谈判。我不需要建立我的信用等...我想付现金。
> 
> 以“低于发票”的价格范围获得报价的最佳策略是什么？

在我开始之前，我相信你们中的很多人都在想“这有什么区别吗？交易就是交易，不管怎样，价格应该是一样的。” [正如我在之前的文章](https://jalopnik.com/what-you-should-know-about-being-a-cash-buyer-at-a-deal-1737301990) 中所讨论的，经销商除了从金融办公室获得担保等其他产品外，还从贷款中赚钱。因此，如果你带着现金出现，这是他们失去的利润领域，因此，如果一些经销商知道他们不会从财务办公室得到更多的钱，他们可能不会愿意对汽车价格打折。但这并不意味着即使支付现金，你也不能买到划算的东西。

大多数真正普通的汽车可以低于发票价格购买，而其他汽车的最佳“交易”价格是发票价格或略高于发票价格。这真的取决于特定型号的市场。当谈及最佳交易时，现金买家或其他人会征求同一辆车的一系列报价，看看谁愿意提供最低的价格。然而，你可能不想一开始就告诉他们你只付现金。我想说的是“我可以支付现金，但如果有动机的话，我可能愿意融资。请根据现金交易发送数字。”这样你就可以比较不同公司的报价。

虽然我会说，有时你只付现金会把钱留在桌上。最近，我有一个客户在寻找一辆新的卡罗拉，她可以支付现金，但丰田提供 1500 美元的回扣，如果你与丰田融资。那笔回扣在一笔现金交易中丢失了。她拿走了贷款和多余的存款，然后马上付清了所有的钱。

最后……为什么 15000 美元以下的好家用车这么难找？

> 我主要开摩托车上下班，这样就把车留给了我妻子使用。然而，我们下周将迎来一个宝宝，这将改变我们应该买哪辆车的整个游戏。理想情况下，我们想要一辆跨界车，以便于让婴儿进出汽车座椅，以及打包所有我们需要的垃圾。我在网上搜索过很多跨界车，但我很难找到一辆里程不到 50，000 英里、低于我们 15，000 美元的户外预算、车龄为 3 到 5 年、仍有清晰名称的跨界车。我对这些参数现实吗？如果没有，除了预算，我应该先改哪一个？我真的应该关心找一辆干净的有牌子的车吗？我应该考虑轿车而不是跨界车吗？

这些都是很好的问题，我能理解你的沮丧。如果你没有在给定的参数内找到你想要的，你需要灵活。但是哪些参数是灵活的是棘手的部分。第一，只关注干净标题的车。虽然有一些宝石有损坏历史和重建标题，有很多质量的汽车没有这些瑕疵，所以最好只是过滤掉它们。

至于轿车与跨界车的难题，一辆好的轿车可以很好地作为家用车，鉴于四门市场目前正在挣扎，你可能会发现一辆较新的轿车，与价格类似的跨界车相比，里程数更少。专注于几款中型车型，带它们去兜风，看看空间是否合适。

最后，当谈到年份和里程时，三到五年的范围是合理的，但不要太在意 50，000 英里的上限。在这个价格区间，条件比里程更重要。你可以拥有一辆行驶里程为 35，000 英镑的汽车，但它被打得很烂，而一辆行驶里程为 70，000 英里的汽车却可以有更长的寿命，因为它保养得很好。我并不是建议你看里程表上超过六位数的汽车，但是将你的网络扩展到 75，000 英里可以让你有一个真正稳定的旅程。只要确保你买之前检查过就行了。
