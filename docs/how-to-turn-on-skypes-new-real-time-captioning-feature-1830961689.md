# 如何打开 Skype 新的实时字幕功能

> 原文：<https://lifehacker.com/how-to-turn-on-skypes-new-real-time-captioning-feature-1830961689>

如果您耳聋或听力不好，那么拨打或参与 Skype 通话可能是一件复杂的事情。本周， [微软推出了一个功能](https://blogs.skype.com/news/2018/12/03/introducing-live-captions-and-subtitles-in-skype/) 让事情变得简单一点:标题和字幕。

Watch

该功能为音频和视频通话添加了实时字幕，使那些可能听不清对方聊天的人有更好的机会了解谈话内容。它还提供 20 种语言的实时翻译，以应对与你聊天的人不会说你的语言的情况。

一旦打开，您将看到字幕/说明在通话过程中滚动显示正在说的内容。字幕是由人工智能完成的，所以它们并不完美，但它们明显比什么都没有好。

要打开它们，你有两个选择:

### 就打一个电话

如果您只需要正在进行的通话的字幕，您可以在通话过程中按+按钮，然后从菜单中选择“打开字幕”来打开字幕。

### 永久开启

如果你想默认开启字幕功能，你可以在应用程序中点击你的个人资料图片，然后选择设置。在那里，选择呼叫，然后选择字幕，然后打开“显示所有语音和视频呼叫的字幕”按钮

Android (6.0+)、iPhone、iPad、Linux、Mac、Windows 上的 Skype 版本 8 和 Windows 10(版本 14)上的 Skype 都提供实时字幕。

该功能正在逐步推出，因此如果您有 Skype 版本 8，但没有看到它，请等待几天，然后再试一次。
