# 这是你应该努力争取的信用评分

> 原文：<https://lifehacker.com/this-is-the-credit-score-you-should-actually-be-strivin-1838037288>

每个人都想在某方面做到完美。但当涉及到你的信用评分时，那完美的 850 分可能不值得你焦虑。事实上，你的分数只要足够好就能打动贷款人。

Watch

CreditCards.com 的分析师泰德·罗斯曼在最近的一篇文章中提到了这一点，他在文章中解释了他选择新信用卡的策略 。“一旦你达到 740 以上，你就有资格享受最优惠的条件，”他说。他解释说，对你的分数有一点影响的行为——比如申请新的信用卡——可能会让你的分数降低几分，但不足以让你担心你的信用档案的健康状况。

但是 740 比最高分低了 110 分。你真的能获得贷款、信用卡利率或其他银行产品的最佳利率吗？感觉就像获得了 B+而不是 A？

我问罗斯曼为什么 740 是一个神奇的数字。他说:“任何超过 740 分的都被认为是优秀的，贷方不会区分超过 740 分的。”。那是因为有一种 [的评分方法](https://twocents.lifehacker.com/why-your-credit-scores-at-the-three-bureaus-are-all-dif-1735517744) 供贷方在你申请信贷时用来确定你的信用程度。

VantageScore，你在使用免费信用评分工具时经常看到的数字，是由三家信用报告机构合作的，该方法有两个版本可以用来计算你的分数。

但是贷方更频繁地使用 FICO 来生成信用评分。和 VantageSCORE 一样，FICO 也有几个不同版本的评分模型。罗斯曼解释说:“不同的贷款机构使用不同的公式，通常是因为他们对一个公式感到满意，并希望避免升级的成本和麻烦。”。

益百利教育主管罗德格里芬(Rod Griffin)表示，抵押贷款机构可以使用早在 FICO 2 之前的评分模型，而 FICO 8 和 9 是最常用的 FICO 模型。他认为关键得分基准是 750。“这取决于分数的类型和贷方的门槛，”格里芬说。

只是在这种情况下增加了一个问题，一些评分模型的最高分已经达到了 900 多分。

但是现在不要担心 850 和 900。与其把你的数字分数作为一个财务目标，更重要的是想想是什么因素构成了这个分数。

如果你至少按时支付你的最低余额，并且不使用太多已经提供给你的信用，你可能会有一个可靠的信用评分，尽管实际数字可能会根据你看的地方而有所不同。所以，不要超出你的信用额度，为你的月结单设置自动转账。一旦你达到了 750 分，你就可以自信地说你已经在班上名列前茅了。
