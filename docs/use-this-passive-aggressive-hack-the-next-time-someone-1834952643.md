# 下次有人借你的笔时，用这种被动攻击的方式

> 原文:[https://life hacker . com/use-this-passive-aggressive-hack-the-next-time-someone-1834952643](https://lifehacker.com/use-this-passive-aggressive-hack-the-next-time-someone-1834952643)

你是否厌倦了人们在工作中借用你的东西——比如一支昂贵的钢笔，配有你喜欢的 0.38 毫米线宽和舒适的握感——并且永远不会归还？

Watch

如果你曾经借给同事一支你非常想要还回去的钢笔，这里有一个小提示:在你把它交给同事的时候，不要盖上笔帽。正如 u/jothebest75 在 [、Reddit](https://www.reddit.com/r/LifeProTips/comments/brdqba/lpt_when_letting_someone_borrow_our_pen_or_marker/) 上解释的那样，如果没有上限，他们更有可能在使用后归还。

他们可能不想让你的钢笔墨水弄得满手都是或者在他们的包里，这是在提醒你的借书人:这支钢笔不属于你，如果你拿走了，你就毁了我们俩的钢笔。事实上，大多数人并没有打算偷走你珍贵的书写工具，而是忘记了。我家里有一把牙医和医生办公室的笔，肯定是签字后不经意带走的。在给别人一支笔之前不盖上笔帽只是一个简单的提醒，提醒他们用完后要还回去。

如果有人问你借笔，你也可以说不，尤其是如果这是一支自来水笔，不像你的标准圆珠笔。(就说没墨水了。)

为了减少你自己抢笔的情况，只要有可能，就从酒店和银行开始携带那些促销笔，但是特别是当你知道你将乘坐带有海关表格的国际航班或者在 DMV 排队的时候。在这种情况下，不管有没有笔帽，钢笔都是很有价值的商品。
