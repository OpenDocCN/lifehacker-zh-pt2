# 用 30/30/3 法则看看你能买得起多少房子

> 原文：<https://lifehacker.com/see-how-much-home-you-can-afford-with-the-30-30-3-rule-1844997314>

如果你一直在疫情寻找一个新家，你并不孤单。随着 [创纪录的低抵押贷款利率](https://www.wsj.com/articles/30-year-mortgage-rate-reaches-lowest-level-ever-2-98-11594908357)[更多的人在家工作](https://www.dallasfed.org/-/media/documents/research/papers/2020/wp2017.pdf) ，许多人正在考虑升级。事实上，根据最近一份全国房地产经纪人协会的报告，6 月份现房销售跃升了 20%以上，7 月份飙升了 25%。问题是，买房不仅仅是支付每月的抵押贷款——随着财产税和房主保险费的上涨，每月的抵押贷款也会增加。

根据 HomeAdvisor 的 2019 年家庭支出状况 [报告](https://www.homeadvisor.com/r/wp-content/uploads/2019/06/StateOfHomeSpending2019.pdf) ，你还必须支付持续的房屋维修和保养费用——平均每年 1105 美元。但是许多房主会花更多的钱，所以专家建议每年留出房屋价值的 1-4%来支付。

毫无疑问:买房是有风险的，尤其是当你不得不超支预算的时候，在经济低迷时期，风险可能更大。虽然抵押贷款有自己的一套负担能力指导方针，但可能值得考虑更保守的方法。

[金融武士](https://www.financialsamurai.com) 的山姆·多根建议遵循 [30/30/3 法则](https://www.businessinsider.com/use-30303-rule-determine-what-house-you-can-afford-2020-9) 来衡量你能负担多少。多根说，你每月的抵押贷款支出不应超过总收入的 30%。他还建议将房屋价值的 30%存起来——20%作为首付以避免私人抵押贷款保险，10%作为紧急缓冲。他还建议将房产价值限制在家庭年收入的三倍以内。

举个例子:假设你的家庭年收入为 75，000 美元。这意味着根据 Dogen 的指导方针，你的家庭月总收入是 6250 美元，你的抵押贷款不应该超过 1875 美元。你还应该把你的房子的购买价格限制在 75，000 美元的三倍——或 225，000 美元——并且在购买房子之前，你应该有 225，000 美元的 30%——或 67，500 美元。

虽然多根的建议对普通家庭来说可能很难，但计算数字可能仍然是一项值得做的工作。毕竟，很难预测经济或者你的工作稳定性在未来会是什么样子。在房子上花更少的钱可能不是一件坏事，尤其是如果你或你的伴侣经历了收入的损失。
