# 提醒:防晒喷雾可能易燃

> 原文：<https://lifehacker.com/reminder-sunscreen-spray-can-be-flammable-1844169292>

气溶胶喷雾通常是易燃的，包括许多防晒霜和杀虫剂。标签上用小字写着，我猜你一般不会看，因为你在烤架或 tiki 电筒旁防晒。

Watch

防晒喷雾通常包含酒精(或类似的挥发性化学物质)，以帮助将喷雾物质从罐中带到皮肤上。一旦它在你的皮肤上，它会产生浓厚、湿润的光泽，这是*好的*——这意味着你已经使用了足够多的。

但是虽然它是湿的，它仍然是易燃的。喷雾本身在空气中也是易燃的，当我们谈论火灾时，值得一提的是，如果过热，气溶胶罐可能会爆炸，所以让那罐防晒霜远离营火。

几年前，一名男子涂了防晒霜后立即在烤架上烧伤了自己，导致一名新闻工作人员让一名消防员点燃了一个人体模特的手臂 。

要明确的是，只有*湿*喷雾才是易燃的。一旦防晒霜在你的皮肤上变干，似乎就不会有任何危险了。拿着烤架的人来说， [很可能是一个有故障的喷嘴导致太多的喷雾被分配](https://www.scientificamerican.com/article/fact-or-fiction-sunscreen/) ，结果喷雾没有在你合理预期的时间内变干。

所以，今年夏天当你使用喷雾防晒霜时，要远离明火，确保喷雾完全干燥后再去烧烤。
