# 腌制葡萄的美妙时光

> 原文：<https://lifehacker.com/pickle-grapes-for-a-surprisingly-good-time-1839061793>

一些泡菜——黄瓜、洋葱，甚至胡萝卜——是显而易见的，但葡萄不是显而易见的泡菜。葡萄是一种令人惊讶的泡菜，一种意想不到的泡菜，一种耐人寻味的泡菜。如果你在发酵过程中不动它们，它们就会变成冒泡的泡菜。



它们也是很容易做的泡菜。虽然“乳酸发酵”听起来有点科学(可能含有大量乳制品)，但这是一个非常简单的(不含牛奶)过程。“乳酸菌”指的是乳酸杆菌，这是一种天然存在的细菌，当置于有氧(无氧)环境中时，它会将糖转化为乳酸。乳酸给泡菜一种很好的味道，盐抑制了有害细菌的生长，同时让我们有益的细菌发挥作用。

最常见的乳酸发酵泡菜是卷心菜和胡萝卜，但是一个推特好友建议用乳酸发酵一些葡萄，我现在痴迷于这种美味、扑鼻的食物，非常适合做奶酪盘。虽然你可以在液态盐水中乳酸发酵葡萄，但你可以通过使用真空密封器来放弃这一切，这为我们友好的细菌创造了一个奇妙的环境。要制作这些意想不到的泡菜，你需要:

*   葡萄(洗净)
*   无碘食盐(碘能抑制发酵)
*   真空密封器和真空袋

决定你要腌制多少葡萄并称重。记下质量，称出相当于你的葡萄质量 2%的盐量。(例如，如果你有 100 克葡萄，你需要 2 克盐。).如果你喜欢，把葡萄切成两半，然后把它们扔进真空袋里。加入盐，把所有的东西混合在一起，然后密封袋子。让葡萄在室温下静置。一天后(或者更快)，它们会开始冒泡。大约一周后，泡沫将开始消退。一旦他们这样做了，你的葡萄就可以享用了。

如果你把葡萄整个放在一边，一些气泡可能会被困在它们的皮下。这就形成了气泡泡菜，这很有趣，但效果会随着时间的推移而消退，所以直接从袋子里取出一些放在嘴里。将味道浓郁且略带异味的葡萄与奶酪和腌肉一起食用，或将它们切碎做调味汁或酸辣酱，将盐水用于腌泡汁、调味汁和调味汁。把泡菜放在冰箱里，至少可以保存几个月。
