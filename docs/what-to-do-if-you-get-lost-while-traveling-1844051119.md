# 如果你在旅行中迷路了该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-you-get-lost-while-traveling-1844051119>

旅行中有各种各样的方式会让你迷路。你在火车上睡着了，错过了很多站，当你醒来的时候，火车已经不再服务了。你的手机没电了，谷歌或苹果地图不再是一个选择。也许你的手机彻底丢了——你唯一的地图也不见了。

事情会发生，但你不必感到害怕或焦虑——如果你迷路了，还有很多事情可以做。

### **计划好，不要慌**

预计每次旅行都会出错；这样你就知道如果真的发生了该怎么做。为您的设备准备一个额外的充电器或电池；规划出如何在没有手机的情况下往返住处；有多余的现金以防你不得不跳上出租车；知道你在火车上的站；随时准备好紧急联系人。

### 随身携带你的手机

有时你可能会想，如果你在某个特定的地区没有服务，或者你的电池没电了，你想让它充电，那么你的手机就没有意义了。但是如果你有选择的话，你不应该不带手机去任何地方。

无论是在酒店还是咖啡店，总有人会有适合你手机的充电器。即使没有服务，也可以连接到任何有 wifi 的地方。接入 wifi 可以帮助你找到一个翻译，告诉你你迷路了，你想去哪里。电话是无价之宝。随身携带手机。

### 携带酒店/住宿卡

如果你在一个非英语国家，每家酒店都会有他们的信息，包括名字和地址，用母语书写。如果你迷路了，你可以给别人看地址，看看他们是否能帮你回到你想去的地方。可以在世界任何地方帮助你。

在中国，我带着一辆优步走错了酒店，手机快没电了。我给错了酒店的前台看了我本来要去的酒店的卡(我的普通话仅限于“你好”、“再见”、“谢谢”)。很容易看出我走错了地方，所以他们不仅通过地图告诉我酒店的位置，还在我的手机在他们桌子上充电时为我叫了一个优步。

### 学习行话

知道如何用任何语言寻求帮助总是好的。

*   “我需要警察。”
*   “我迷路了。”
*   “你会说英语吗？因为我需要帮助。”

如果说不同语言的人能够理解你的需求，他们会非常乐意帮助你——即使是最基本的需求。你也可以用母语和英语写下一些短语；随身携带，指出你需要的短语，这样人们就能大致了解他们如何能提供帮助。

### 专心

无论我去世界上的哪个地方，我都看到人们如此专注于他们的手机屏幕，或者没有完全集中注意力，以至于他们错过了他们的火车站，或者走在了一个完全不同的方向，他们不应该去的地方。

意识到你周围的环境，注意地标、街道名称或者你是如何到达一个地方的，可以帮助你回到你需要去的地方。
