# 工作中什么让你抓狂？

> 原文：<https://lifehacker.com/whats-making-you-crazy-at-work-1832475937>

也许你的老板是个混蛋。也许你管理的团队有怪异的内部冲突。也许你对自己下一步的职业发展犹豫不决。也许你的隔间邻居快把你逼疯了。



不管怎样，我们都在工作——我们都可以时不时地听取一些工作上的建议。

介绍“人力资源”，Lifehacker 的新工作建议专栏，由记者 Rob Walker 撰写。你可能通过他在 2013 年至 2018 年期间在《纽约时报》*担任的 [【工作学家】](https://www.nytimes.com/column/workologist) 的智慧分配角色来了解罗伯。他还写了几本书，包括将于 5 月出版的《注意的艺术》。*
