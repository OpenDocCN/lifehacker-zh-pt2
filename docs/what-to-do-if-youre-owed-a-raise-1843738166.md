# 如果你被要求加薪，该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-youre-owed-a-raise-1843738166>

自从冠状病毒封锁开始以来，我们这些有幸保住工作的人一直在投入大量时间和精力工作。我们一直在学习新的技能，包括用变焦开会的艺术；我们还帮助我们的公司在不确定性和焦虑加剧的时期保持高效。



是时候向老板要求加薪了吗？在疫情期间，这可能吗——或者合适吗？

如果你因为封锁错过了年度绩效评估，这个问题会变得更加重要。如果你应该和你的上司讨论加薪，但没有机会，你可能会问自己是否有一个开始谈话的好方法。

我向一位经理的 艾莉森·格林求助，询问她对要求加薪的时机是否合适的看法，她证实了我们许多人心里已经知道的事情:不合适。

“我想说，现在，在大多数行业，有太多的不确定性和担忧，这不是大多数人要求加薪的好时机，”格林告诉我。“雇主们正试图弄清楚如何留住员工，他们可能需要在哪些方面裁员，以及他们的财务状况在两个月后会是什么样子。很有可能会被认为对这种情况听不进去。”

格林为这种情况提供了一个例外。如果你的雇主相对没有受到疫情冠状病毒和相关经济不确定性的伤害，如果你的公司没有裁员或面临任何财务风险，那么提出加薪的话题可能没什么问题。某些行业目前表现非常好，这些行业的员工可能相应地增加了他们的工作量——这意味着要求加薪可能是合适的。

但是再等一会儿也可能是个好主意——因为，正如格林提醒我们的，你可能不会永远等下去。“这并不意味着我们正在进入一个长达数年的人们不能要求加薪的时期，”格林解释道。“当我们仍处于如此多的不确定性之中时，现在真的是时候了。”

我的建议？在你的日历上记下，今年晚些时候重新考虑要求加薪的想法。如果您知道您的组织何时创建年度预算，请尝试安排您的请求时间，使其在预算最终确定之前完成。如果你的日程表上已经有一个绩效评估，这是提出来的好时机。如果你错过了你的绩效评估，问问你的主管他们什么时候(或者是否)愿意重新安排；这会给你很多信息，让你知道你的雇主优先考虑的是什么。

然后继续做你已经在做的事情——完成你的工作，赶上最后期限，与你的团队成员沟通，成为那种值得你要求加薪的员工。
