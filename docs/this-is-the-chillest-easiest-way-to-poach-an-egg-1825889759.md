# 这是最酷、最简单的煮蛋方法

> 原文：<https://lifehacker.com/this-is-the-chillest-easiest-way-to-poach-an-egg-1825889759>

在我成年后的某个时候，我变成了一个早上 6:30 醒来的人。我没有选择这条路——我宁愿一直睡觉，谢谢——但我正试图充分利用我的身体突然拒绝睡懒觉的状况。



早起让我不得不重新审视自己对早餐的看法。我以写食物为生，所以有人可能会认为我每天都有丰盛美味的早餐。有一个可能是错的:我花了几个小时才承认咖啡和水不是食物，而当这种情况发生时，我已经极度饥饿，只有最快速、最简单的食物才能解决问题。对我来说，这一直是鸡蛋和烤面包，感谢我想象中的妈妈爱丽丝·梅德里奇，我现在有一个重要的荷包蛋时刻。

荷包蛋对所有经验水平的厨师来说都是白鲸。每个人都有自己喜欢的方法——过滤掉多余的蛋白！往水里加点醋！制造一个水漩涡！—但是这些都不能简化过程。在我看来，复杂、过于繁琐的技术与荷包蛋的简单完美截然相反。进入 [爱丽丝·梅德里奇的不失败技巧](https://food52.com/blog/22023-how-to-make-simple-poached-eggs-you-don-t-need-vinegar) ，这是她从前男友那里学来的:

> “以前一个男朋友教我做荷包蛋。相反，我是看着他做的。他会把鸡蛋一个一个敲碎，放进一个浅锅里，盖上锅盖，关掉火，把面包片放进烤面包机，然后去刮胡子。他会溜回厨房，把胡子刮干净，拿起烤面包，把鸡蛋放在上面。没什么——还有美好的回忆。”

(他听起来很梦幻。干得好，爱丽丝！)

十多年来，我一直非常成功地用水煮鸡蛋，但我说这种方法改变了我的生活并不夸张。它快速、简单、高度可重复——这正是我一天中第一顿饭所需要的。下面是怎么做的。

向有紧盖的煎锅或炖锅中加入约两英寸的水；用大火煨一下。如果你像我一样把切片面包放在冰箱里 ，现在是时候拿出几片放进烤面包机了。



轻轻敲开冰箱里的冷鸡蛋，一次一个直接放入沸腾的水中——或者如果你喜欢，先放入一个 rame kin/茶杯/汤容器中——然后盖上盖子，关火。启动你的烤面包机。

三分钟后，打开盖子检查你的鸡蛋。大鸡蛋就要做好了；巨型蛋可能还需要一两分钟。

在吐司上涂上厚厚的黄油，然后将纸巾(或干净的厨房用毛巾)折叠几次。用漏勺将鸡蛋从煎锅中取出，用毛巾擦干多余的水。

将鸡蛋转移到涂黄油的吐司或盘子里。用盐和胡椒调味，因为辣酱是用来煎鸡蛋的。

这种方法的轻松性质应该足以说服你尝试它，但如果放弃过滤器、醋和漩涡的想法让你发痒，这里有三个更多的条目在“优点”栏中。首先，它完全消除了那些劣质的、细小的蛋清，这些蛋清会导致难看的荷包蛋——当你舀起鸡蛋时，它们会沉入底部，并从剩余的蛋清中完全脱落。将勺子压在煎锅的侧面，可以整齐地切掉任何残留的残渣。

第二，在完全静止的水中煮鸡蛋会温和而均匀地使蛋清蛋白变性，而不是像连续煨煮那样一蹴而就。这可以防止白色粘到该死的煎锅的侧面和底部，所以清理起来容易多了。

第三，这种方法使得扩展变得轻而易举。如果你要举办一个大型的早午餐，并且非常喜欢吃本尼迪克特鸡蛋，只需要使用更多的水和一个更大的锅——并且计划在将鸡蛋加入水中之前将它们打碎到一个可倾倒的容器中。

和所有鸡蛋烹饪一样，根据你的口味调整温度和时间。我喜欢一个荷包蛋，里面有完全凝固的蛋白和果酱般的蛋黄，所以用猛火煨就可以了。然而，如果你想要一个更经典的荷包蛋——蛋清，全液态蛋黄——把水一直烧开，敲碎鸡蛋，*然后*盖上锅盖，关火。他们会煮得快一点，所以两分钟后入住。

注意:一旦你掌握了这个技巧，你会发现所有东西看起来都像一片烤面包。这对我来说很好；毕竟，从炒饭到一堆炒青菜，一切都更像是一顿饭，上面有一个荷包蛋。
