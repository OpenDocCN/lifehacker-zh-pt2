# 如何看三星本周的 Unpacked 活动

> 原文：<https://lifehacker.com/how-to-watch-samsungs-unpacked-event-this-week-1836956742>

周三，三星预计将推出 Galaxy Note 10，这是其广受欢迎的触控笔智能手机的下一代产品。



如果你没有被邀请参加该公司在纽约举行的活动，你可以在网上看到所有的活动。

三星将于美国东部时间 [周三下午 4 点在网站](https://www.samsung.com/global/galaxy/events/unpacked/?cid=afl-ecomm-cjn-cha-092118-52057&cjevent=7eac15dfb70d11e982a001930a1c0e11) 直播整个发布会。

传言称，Galaxy Note 10 将进行多项升级，包括手机触控笔上的内置摄像头(称为 S-pen)和四个后置摄像头。这种笔还可以提供远程控制手机的功能。

预计三星将推出两种不同版本的手机，一种是 6.3 英寸显示屏，另一种是 6.8 英寸显示屏；这意味着将有比 6.4 英寸 Note 9 更大和更小的选择。

据传，这两款手机都有三星的 Infinity-O 显示屏和高通的骁龙 855 处理器。

Note 10 Plus(较大的那个)预计也将有一个 4300 毫安时的大电池。普通的 Note 10 会有一个 3,500mAh 的电池。

当然，现在所有这些都只是谣言。如果你是三星的粉丝，请在周三收看，看看这些传言中有哪些成为了现实。
