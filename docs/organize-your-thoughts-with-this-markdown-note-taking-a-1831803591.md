# 用这个 Markdown 笔记应用程序整理你的想法

> 原文：<https://lifehacker.com/organize-your-thoughts-with-this-markdown-note-taking-a-1831803591>

我承认，我并不像应该的那样精通 Markdown，但这个令人敬畏的免费笔记应用程序让我想了解它的复杂性。这款应用同时适用于 Windows 和 Mac，是一款成熟的笔记记录*体验*，比你在系统中找到的任何默认应用都要好得多。你只需要学会降价就能充分利用它，但这听起来比实际情况要糟糕得多。

Watch

从你安装并启动它的那一刻起， [**值得注意的**](https://github.com/fabiospampinato/notable) 就非常愿意牵着你的手，让你跃入一个新的笔记应用程序的可怕而未知的世界。这听起来可能过于戏剧化，但让我们面对现实:如果你已经花了几年时间将你的想法倾倒在 Evernote 或 Mac 的 note 应用程序上，你会抵制转向新的东西。你应该。冒险一试。

我喜欢很多值得注意的东西。首先，它的创造者 Fabio Spampinato 以预填充笔记的形式为你提供了一个应用程序的教程，这很有创意。当你阅读这些说明时，你不仅在学习该应用程序如何工作，而且如果你像我一样喜欢修补，你可以直接跳到他创建的笔记的具体细节，看看该应用程序的笔记组织功能是如何工作的——以及 Markdown。

您创建的所有便笺都保存在计算机上的一个文件夹中，该文件夹是您在程序首次启动时指定的。您放入备忘录中的任何附件都存在于另一个文件夹中。虽然这听起来很混乱——当然是在你的硬盘上——但如果你利用该应用的许多技术来组织你的笔记，一切都会变得井然有序。

例如，您可以将笔记分配给无限数量的笔记本，或者您想要保存在同一位置的主题想法集合。您还可以给便笺添加任意数量标签，这是组织标签内容的第二种方法。最后，你甚至可以将笔记标记为“收藏夹”，以防你需要另一种方法来找到你的宝石，或者将重要的笔记钉在你所有的笔记本和标签上，这样你就不会丢失它们。这款应用甚至还带有内置搜索工具，以防你需要更多的帮助来找到一些东西。

至于 Markdown 方面，我发现学习该语言的语法最简单的方法是将所有值得注意的教程笔记保存在一个新的笔记本中。如果你需要快速复习如何倾斜文本、建立列表或将短语转换为超链接，你可以参考它们。(你也可以将网上众多 [降价备忘单](https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet) 中的一张复制粘贴到新便笺中。)

无论你如何思考这些代码，你都会发现 Markdown 是一种快速有效的方式来整理你的想法——至少比点击各种工具栏按钮要好得多。
