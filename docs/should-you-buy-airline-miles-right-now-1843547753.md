# 你应该现在购买航空里程吗？

> 原文:[https://life hacker . com/should-you-buy-airline-miles-right-now-1843547753](https://lifehacker.com/should-you-buy-airline-miles-right-now-1843547753)

无论你有多怀念旅行，你可能都不会对即将登上飞机的前景感到兴奋。但是航空公司知道，最终，人们将再次开始飞行——当那个时刻到来时，他们希望座位上有屁股。因此，你可能已经看到促销邮件发送到你的收件箱，为你最喜欢的航空公司提供大折扣的里程。

但是，如果你不确定什么时候你会再次感到舒适，购买里程是一笔好交易吗？或者你省钱的努力会白费吗？

简短的回答是:跳过它，除非你心中有一个特别的旅行。但即使你有 2020 年末或明年的旅行计划，购买积分仍可能是一场赌博。

忠诚计划顾问、的创始人戴夫·格罗斯曼说:“如果你现在就在计划一次旅行，即使是明年，并且看到你想要的航班现在就有奖励，显然可以节省里程，这可能是值得的。”但即使这样，你也需要记住，你的航班可能会被取消，你可能会坐在里程。"

但是如果你不仔细阅读航空公司的里程估价，你可能最终会为你的最终旅行支付过高的费用。

的信用卡专家史蒂文·戴斯勒说:“现在最大的问题是，当人们开始旅行的几个月后，机票价格会是什么样子。”。“他们会在现在的地方盘旋吗？或者他们会向上或向下移动？这最终将决定你购买的积分的价值，以及你是否得到了一笔好交易。”

戴斯勒认为，购买航空里程的最佳时机是当你非常接近奖励航班，并且需要少量里程来度过难关的时候。但是，如果你打算提前六个月或更长时间预订奖励航班，就很难确定你需要多少额外积分。(如果你想知道与去年相比，现在一家航空公司的里程值是多少，请查看来自的估价指南。)

如果提供优惠的航空公司是你经常乘坐的那家航空公司，而且你有这家航空公司的信用卡，你或许可以通过正常消费来赢取里程，而不是预先支付。

但是，即使你有一家最喜欢的航空公司，购买积分而没有使用它们的计划也是一种风险，因为航空公司可能会使它们的里程和积分贬值。正如我们 [最近对](https://lifehacker.com/how-credit-cards-rewards-will-change-after-the-pandemic-1843484915) 的解释，航空公司(以及任何其他提供奖励的企业)可以随时改变他们的积分或里程数。因此，虽然你现在可能可以以低价获得大量里程，但当终于到了飞行的时候，你可能会发现，你的里程并没有达到你希望的那么远。

除了价值，当我们都开始思考疫情之外的生活会是什么样时，还有一件事需要考虑:你最喜欢的航空公司还会存在吗？

到目前为止，世界各地的航空公司已经从政府那里收到了超过 850 亿美元的资金 ，航空业可能需要数年时间才能从航空旅行的突然下滑中恢复过来。

格罗斯曼建议在从航空公司购买大宗商品之前，先检查一下该公司的财务状况。
