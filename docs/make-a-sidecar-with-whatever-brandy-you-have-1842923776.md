# 用你所有的白兰地做一个边车

> 原文:[https://life hacker . com/make-a sidecar-with what-brandy-you-have-1842923776](https://lifehacker.com/make-a-sidecar-with-whatever-brandy-you-have-1842923776)

最近我一直在做很多酸味的，仅仅是因为它们几乎适用于任何烈酒，而且不需要太多配料。酒精、柑橘和甜味剂组成了整个饮料，但当你为更多的酒精添加糖浆时，事情变得更有趣了。

Watch

使用库拉索岛作为糖源的边车可能是这方面最经典的例子。它通常是用法国白兰地做的，但是，因为去卖酒的商店并不是一次必不可少的旅行，所以我不喝那种酒。不过，我有梨子白兰地。你猜怎么着？那工作得很好。如果你有另一种不同的白兰地，那也很好。

见鬼，你也许可以喝一种稍微不同的橙色利口酒(但我不会用柑橘伏特加)。我只有库拉索干葡萄酒，顾名思义，没有那么甜，这是一种非常好喝的饮料。这是一个相当宽容的鸡尾酒甲酸，这就是我要说的。要做到这一点，你需要:

*   2 盎司白兰地
*   3/4 盎司库拉索芦荟汁(如果你用的是干的，增加到 1 盎司)
*   3/4 盎司柠檬汁

将所有材料加入鸡尾酒摇壶，加冰摇至非常冷。滤入冰过的 coupe 型玻璃杯。
