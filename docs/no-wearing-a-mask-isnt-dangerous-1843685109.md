# 不，戴口罩并不危险

> 原文：<https://lifehacker.com/no-wearing-a-mask-isnt-dangerous-1843685109>

关于口罩的神话正在蔓延，混杂在我们都需要知道的戴口罩的关键建议和事实中，分散了我们的注意力。许多这些神话与阴谋论直接相关，当你花一分钟思考它们 时 [就会分崩离析，但它们仍然流传，因为它们看起来像是*可能是真的。因此，让我们来看看几个神话和一个事实。*](https://vitals.lifehacker.com/how-to-spot-a-conspiracy-theory-1843463080)

### *误解:口罩会让你吸入过多的二氧化碳(或氧气不足)*

*空气含有多种气体的混合物。主要是氮气(我们的身体根本用不到)，还有少量的氧气和二氧化碳。我们身体的细胞需要氧气；每次呼吸我们都会消耗一点氧气，并释放一点二氧化碳。*

*如果你被困在一个封闭的空间里很长时间，是的，有可能看到这个空间里的二氧化碳水平上升。这就是为什么你不让孩子把塑料袋套在头上。但是有一些病毒帖子表明，高二氧化碳或高碳酸血症是戴口罩的常见后果，它会让你感到昏昏欲睡或困惑。*

*但是口罩不是塑料袋。你呼出的微量二氧化碳不会被布口罩或外科口罩截留。N95 口罩如果佩戴得当，并且在面部周围有一个紧密的密封，有时会使有某些健康状况的人呼吸困难——但这不是因为二氧化碳的积累。这是由于体力上的努力，不得不*更用力地呼吸*以通过面罩材料获得足够的空气。即使是布料或外科口罩，当它们因你的呼吸而变得潮湿或温暖时，有时也会感到不舒服。但这不是健康风险。*

*在 之前，我们已经 [写过这个问题，但只是为了额外的视角，我问了 Saskia Popescu，感染预防学家，她最近回答了读者的面具问题，是否高碳酸血症是普通公众或她工作的医院工作人员的问题。一点也不，她说。“我有医护人员整天戴着 N95，”她说他们不会遇到这些问题。所以我不太确定(对高碳酸血症的担忧)从何而来。"](https://lifehacker.com/do-masks-make-you-breathe-too-much-carbon-dioxide-1843421831)*

### *误解:你会因为呼吸呼出的病毒而生病*

*这个很奇怪。它出现在 [Plandemic](https://lifehacker.com/if-you-found-that-plandemic-video-convincing-read-th-1843339002) 中，声称口罩“激活”你呼出的病毒。然后我看到了另一个病毒帖子，声称口罩将你呼出的病毒重定向到你的鼻子，在那里它们可以进入你的大脑。啊？*

*这不仅毫无意义，还假设你呼出的是新冠肺炎病毒。如果是这样的话，你已经病了，那么口罩怎么会对你造成进一步的危险呢？* 

*但为了防止我遗漏了什么，我也向波佩斯库询问了这个问题。也许有一些事实启发了这个神话？口罩可能会以某种方式加剧病毒性疾病吗？*

*“事情不是这样的。我的意思是，对不起，我不知道还能怎么说，但事情不是这样的，”她说。*

### *误区:口罩不能阻挡病毒*

*病毒非常非常小。不仅仅是微观的，而是小到典型的显微镜根本看不到。*

*布质面具的编织非常松散，你经常可以看穿它。显然，这无法阻止病毒单独传播。我听说有人把这种现象比作“蜜蜂穿过铁丝网”但是从那里，反口罩的人得出结论，这就是你需要说口罩不能阻止病毒的所有信息。*

*但这不是对正在发生的事情的准确描述。病毒不会像蜜蜂一样飞来飞去。导致新冠肺炎的病毒作为呼吸飞沫的一部分离开我们的身体。想象一下有人在一束阳光中打喷嚏。你会看到数以百万计的微小粘液/唾液/天知道是什么发出微光，它们向前喷出，然后轻轻地飘向地面。那些是呼吸道飞沫。*

*布口罩*做*阻挡呼吸道飞沫。当你把口罩戴在你的嘴和鼻子前面时，它会阻止大部分细菌。(当被感染者戴上口罩时最有效，他们被推荐给每个人，因为你可以在不感到恶心的情况下感染和传播病毒。)*

*因此，一个更好的比喻可能是 [把口罩比作裤子](https://couriernewsroom.com/2020/05/01/we-asked-health-experts-about-the-wear-your-mask-the-urine-test-meme/) :如果有人在你腿上撒尿，而你们两个都没穿裤子，你就会被淋湿。但是如果撒尿的人穿着裤子，大部分的尿只会浸湿他们自己的裤子，而不会流到你身上。*

*(有时人们会问，当内衣不能阻止屁时，口罩如何阻止病毒，而不是蜜蜂穿过铁丝网。好吧，携带屁味的分子很小——比病毒还小——而且它们在空气中漂浮而不会停留在液滴中。所以他们可以自由地穿过你的内衣，到某人的鼻子里，在那里你可以声称谁闻到了它，谁就交易了它。)*

### *真相:口罩不一定能保护你免受其他人的病毒感染*

*这是真的，但是我把它包括在列表中，因为它经常和其他的神话一起被讨论。你不戴面具保护自己。口罩最大限度地减少(不是完全阻止)佩戴者的细菌传播。*

*我的面具保护你；你的面具保护了我。*

*因此，要负责任地戴上口罩，你需要意识到它的局限性。如果有人在你附近呼出小飞沫，或者如果你在一个有很多人唱歌或喊叫的封闭空间，戴上口罩并不能保护你免受病毒感染。尽管如此，戴一个还是比没有强。*

*你离喷吐飞沫的人越近，这些飞沫可能就越大，这就是为什么如果你在照顾生病的亲人，口罩可能会给你一点保护。但是面具不能代替其他好的做法，比如物理距离。保持距离*和*戴上面具，你将尽最大努力保护他人的安全。*
