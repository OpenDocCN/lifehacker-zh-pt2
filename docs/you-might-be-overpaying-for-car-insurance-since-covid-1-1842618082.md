# 由于新冠肺炎让我们少开车，你可能为汽车保险多付了钱

> 原文：<https://jalopnik.com/you-might-be-overpaying-for-car-insurance-since-covid-1-1842618082>

既然新冠肺炎不让我们开车，而它正在破坏其他一切，你不妨看看你是否能在汽车保险上省钱。如果你今年的里程数减少，你可能会支付更少的费用。或者，也许是时候调整自己的风险评估了。



你的汽车保险费用不仅仅是根据你的个人驾驶记录和你想要的保险额来计算的。你住在哪里，你有什么样的车...我们可以对这些算法的工作原理做一个完整的调查，但现在让我们把注意力放在对大多数人来说正在发生变化的东西上:年度里程。

打电话给你的保险公司的代表，看看你是否能因为今年里程数的大幅减少而降低保费。您的保险公司可能只会在续保时进行上述调整；我现在有四辆汽车的国家农场，公司的人告诉我，我只能每六个月改变我的里程数。(我还没有问他们是否会为疫情破例，我也没有屏住呼吸。)

但是，也许你的保险公司是不同的，或者也许你自己的保单的续保时间到了！值得一问。我*确实*有几辆车的保费普遍较低，因为它们的年里程数很低，这是我知道这是一件事的唯一途径。我必须每六个月自我报告一次里程表读数。

如果你发现你现在不能改变你的里程数，但你仍然希望降低成本，也许是时候降低你的保险额度了。保险公司通过评估风险来决定向你收取多少费用，当你决定获得何种程度的保险时，你也在做同样的事情。

计算可以归结为:支付便宜的保险，如果发生事故，可能会欠很多钱；或者购买好的保险，每月支付很多钱，但不必担心灾难降临。

现在，我和其他人一样知道，你无法预测你什么时候需要或者想要一份好的保险。毕竟，这就是保险存在的原因。不是老生常谈，但有时事情会在你最意想不到的时候发生。因此，我不会明确建议任何人降低你的汽车保险范围，以在短期内节省一些钱。

但是这个*是*重新评估你个人汽车保险情况的好时机。有第二辆在疫情期间不会有太多动作的车吗？也许你不需要全部的综合保险和责任险。得到了一整辆车的收藏那肯定是在可预见的未来得不到锻炼了吧？如果你的一些车被封存了，也许你想暂停它们的保修。

即使你一个月只开车去杂货店两次，你仍然有可能遭遇车祸。但是，如果你不再每天通勤或驾车出游，这种可能性就会大大降低。所以，对你来说，降低保险额度的风险是值得的。

更多的陷阱需要考虑:当你的车停在你自己的车库里时，坏事情仍然可能发生。如果你没有为它们支付保险，你就不会被覆盖。此外，如果你的汽车没有保险，但仍然注册为道路使用，可能会有州政府对保留车牌但放弃保险的罚款。

你需要考虑的另一件事是你的车是否有留置权。如果你正在付款，有可能进行一定程度的保险是你贷款的一个条件。所以，在削减保险之前，你也要确保知道那里发生了什么。

这里真正的要点是:当谈到保险时，教育你自己到底要支付什么。即使你在选择保单时已经这么做了，冠状病毒让许多汽车靠边站，这也是重新审视它的明智时机。

对于我来说，有太多的变量和个人情况可以轻松地给每个人发放汽车保险行动计划。但是，在这个灾难时刻，每个人都想省钱，你的汽车保险在这一点上需要重新考虑。
