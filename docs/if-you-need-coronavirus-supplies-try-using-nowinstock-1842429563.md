*这些彩色的小图标非常重要，因为它们决定了通知你的方式和时间。从左至右依次是:*

*   ***#1:** 物品有货时启用文本提醒* 
*   ***#2:** 商品有货时启用浏览器提醒*
*   ***#3:** 商品有货时收到邮件*
*   ***#4:** 商品有货时激活接收提醒*(重要！)**

*所以，在这种情况下，如果我想收到关于我什么时候能买到那种甜味 Purell 的短信，我会点击第一个和第四个图标。就这么简单。你可以在你的 [我的账户](https://www.nowinstock.net/users/myAccount.php) 页面查看你的全部追踪者列表，在那里你可以调整提醒或者完全删除这些项目。*
