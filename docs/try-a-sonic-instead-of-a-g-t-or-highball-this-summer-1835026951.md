# 今年夏天，试试“索尼克”,而不是 G&T 或高杯

> 原文：<https://lifehacker.com/try-a-sonic-instead-of-a-g-t-or-highball-this-summer-1835026951>

如果你最近在高档杂货店或烈酒商店浏览搅拌器，你一定会发现各种各样的奎宁水。我个人最喜欢的是发烧树的接骨木花补品。试试吧，太棒了。滋补品可以很好地帮助带出你正在喝的酒的味道，但它们有时也可能有点令人无法抗拒，特别是当你真的想品尝你正在混合的鸡尾酒的味道时。



本周在日本旅行时，我对一种滋补饮料产生了兴趣:音速小子。在谷歌上快速搜索一下就会发现，超音速已经流行了一段时间，但这是我以前从未见过的。

Sonic 代表苏打和奎宁水。要制作一种饮料，你基本上要用传统饮料中一半的奎宁水，用苏打水代替，使它成为一种音速饮料。

我的音速小子是在东京的 Nikka Blender 酒吧。我喝的不是杜松子酒，而是掺有妮卡威士忌的高杯酒。当我在温暖的月份喝威士忌时，高球(通常是白酒和苏打水)已经成为我的长期选择，但我从未考虑过将威士忌与奎宁水混合。

我在这里告诉你它很好吃，你绝对应该试一试。

一位东京酒保给了我一个很好的建议:不要使用昂贵的奎宁水。虽然我心爱的接骨木花滋补品非常适合纯 G&T，但如果你正在制作索尼克，你会想要一些更传统的东西，没有特别大胆的味道。

街边便利商店的 2 美元一瓶的滋补品就够了。

拿一瓶便宜的滋补品和一瓶苏打水，准备好去发现你的夏季新宠吧。
