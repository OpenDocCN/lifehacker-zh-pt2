# 如何在 2019 年大修版下降之前使用新的 Winamp Beta

> 原文：<https://lifehacker.com/how-to-use-the-new-winamp-beta-before-the-overhauled-ve-1829974853>

上周，Radionomy 首席执行官 Alexandre Saboundjian 透露，该公司计划彻底改革经典音乐播放器 WinampT3，该播放器以 iTunes 的 Windows 优先替代品而闻名，使其对我们更加注重流媒体的音频媒体世界更加重要。该版本将于 2019 年推出。



与此同时，旧的 Winamp 回来了！也就是说，它现在兼容 Windows 8 和 Windows 10，这使它成为不使用旧硬件的人更可行的选择。据 Techcrunch 报道，说实话，尽管从许多方面来看，Winamp 是一个过时的工具，自 2013 年以来就没有更新过，但它在全球范围内拥有超过 1 亿每月用户的忠实粉丝群。

目前的 beta 版， [Winamp 5.8](https://winamp.com/) ，就是你以前认识的那个 Winamp。它仍然是一个有趣的和非常可定制的音频和视频媒体播放器，如果你仍然在收集费西合唱团音乐会的盗版或通过 RSS 订阅播客，它可能值得你花时间。

### Winamp 实际上是做什么的？

Winamp 是一个“通用媒体播放器”。它可以播放音频和视频文件，更重要的是，它为您提供了一种在单一位置访问这些文件的简单、直观的方式。你可以把你的 CD 和盗版音乐放在一个“你的音乐”的目录里回到专用 MP3 播放器的时代，你还可以用它来管理你设备上的文件。如果你还有一个——或者想把音乐文件保存在安卓设备上——发疯吧。

在其全盛时期，Winamp 与同行相比脱颖而出，因为它具有高度的可定制性。在浏览器扩展特别普遍之前，Winamp 的大量软件插件可供选择，它们可以给你一切，从 [改进性能](https://www.stereotool.com/) 和 [热键](https://winampplugins.co.uk/plugins.html) 到 [广播音频](http://www.opus-codec.org/) 。

不过，它的许多最受欢迎的功能纯粹是装饰性的。你可以使用数以千计的官方和用户自定义选项，更换播放器的浏览器窗口和字体。当然，还有可视化工具——一种把歌曲的正弦波变成某种激光表演的工具。

### 我为什么要尝试 Winamp(或者回去)？

虽然普通音乐爱好者一般不太需要音频库，但 Winamp 现在对那些仍然需要的人来说，就像它在鼎盛时期一样有用。无论您是下载高质量的音频文件、需要筛选个人音频记录，还是将播客存储在播客“应用”较少的台式机或笔记本电脑上，都有许多使用案例可以让您轻松地对音频媒体进行分类。

既然现在购买音乐已经不那么普遍了，那么不带商店的音乐播放器就更有吸引力了。我年轻时把音乐储存在 iTunes 里，但苹果的现代销售策略有时会让人感到压抑

从技术上讲，存储和播放视频文件也是如此。我认为 Winamp 主要是一个音乐播放器，而明年的重新启动将专注于音频，尽管如此，Winamp 在这个领域并不重要。

### 什么变了？

正如我们之前提到的，Winamp 已经五年没有更新了，直到上周，所以，根据你上次使用它的时间，可能不会太多。

除了兼容 Windows 8 和 10，Winamp 最新版本的最大变化是取消了该应用程序的付费服务 Winamp Pro。Winamp Pro 对更高质量的服务收取月租费，例如以高比特率翻录音频 CD 的能力和某些视频编解码器，我怀疑 2018 年许多人会为此付费。

如果你一直在关注 Winamp 并且想要一份详细的变化列表，你可以在 [Winamp 论坛](http://forums.winamp.com/showthread.php?t=453675) 查看。

### Winamp 会怎么样？

明年，Radionomy 将推出 Winamp 的现代化版本。根据 Saboundjian 的说法，该版本将“通用音频播放器”的概念现代化，其中包括整合数字广播和 Spotify 和 Apple Music 等基于订阅的音乐服务的库。

这将如何工作？你的猜测和我的一样好。

如果真的如此，那会让许多听众的生活轻松很多。有趣的是，我认识的大多数人都通过多种音乐服务、YouTube 和下载播客来听音乐。有些人甚至仍然下载歌曲和/或购买唱片。如果 Winamp 可以为每个人编译所有这些东西，并使其可搜索，它将是一个必备的应用程序。
