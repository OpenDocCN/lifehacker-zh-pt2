# 为你的孩子建立一个安静的角落

> 原文：<https://lifehacker.com/set-up-a-calming-nook-for-your-kid-1834121704>

在漫长的一天之后，当我们心情不好或者只是需要一点独处的时间时，我们都需要一个小地方来减压。对你来说，这可能意味着洗个热水澡放松一下，或者在你最喜欢的安静的地方读本书。对于孩子来说，一个专用的平静区域可以成为过度刺激、权力斗争和兄弟姐妹竞争中的绿洲。

我知道，我知道。他们的卧室或游戏室或者 [儿童洞穴](https://offspring.lifehacker.com/how-to-create-the-ultimate-kid-cave-1832334325) 已经有足够的空间。他们——以及他们的东西——已经占据了房子的大部分。但是，在他们卧室、游戏室甚至是主要生活区的角落里，一个安静的小角落可以更少地用于存放玩具，而更多地用于放松和补充能量。

在学校度过了艰难的一天？在开始做作业之前，在安静的角落呆上 15 分钟。因为连续第五天下雨不能在外面玩？把一本书拿到角落里呆一会儿。对你周围世界的不公平感到愤怒？宁静角落！

当然，如果你打算把家里的一个角落奉献给宁静，有几件事要记住，以确保最终的结果确实让人感觉宁静。

### 保持中立

把一些明亮、愉快的原色放到孩子的角落里可能很有诱惑力。你希望它是一个快乐的空间！但是，鲜艳的颜色可能会过度刺激，过度刺激不是我们在这里要做的。选择中性颜色和天然材料的家具和装饰品(多用木材，少用塑料)。

### 想软了

蓬松的地毯。一些大抱枕。一把豆袋椅和几个最喜欢的毛绒动物玩具。柔软的材料可以让你躺在上面拥抱，让这个角落更加舒适诱人。

### 限制书籍和玩具

如果你的孩子将这个空间主要用作阅读点，那么很容易拉一个书架，放上大量的材料。相反，通过在几个软篮子里放几本书来保持选择的简单和最小化。将大部分书籍存放在另一个房间或壁橱里，并不时地轮换选择。

如果你在角落区域有书架，拿出书，添加一些家庭照片、喜爱的纪念品和平静活动，如拼图、积木或烦躁玩具。一两株小植物也能给空间增添一点温暖。

### 让那里有光

更具体地说，让有*自然*光。对于你的角落来说，最重要的考虑因素之一可能是它是否靠近一扇能让大量自然光进来的窗户。你想要一个明亮通风的空间，而阳光是营造温馨氛围的最便宜、最简单、最有效的方式。
