# 什么时候“小额”购买仍然不值得？

> 原文：<https://lifehacker.com/when-is-a-small-purchase-still-not-worth-it-1831399349>

当你阅读大量的个人理财文章时，你不可避免地会到达这样一个地步:你不能在街角的熟食店买一瓶水或买汽油，而不开车四处寻找每加仑便宜两美分的东西，而不感到羞耻。毕竟，这些都是不必要的购买，每个人都知道花 3 美元你不需要的东西会导致人们不满足，不富裕的生活。或者什么的。



事实上，购买小的或便宜的东西并不总是那么重要。有时它们真的丰富了我们的生活，有时却不是。变得更大的负担是精神上的努力去弄清楚*这个*购买是否“值得”，以及这到底意味着什么。

对于简单的美元，特伦特·哈姆 [写了几条聪明的指导方针](https://www.thesimpledollar.com/when-to-sweat-the-small-stuff/) 来决定是否值得在没有精神痛苦的情况下花钱购买不贵的东西。

### 不要泄露个人信息

也许脸书和谷歌已经知道我们所有的希望、恐惧和最深的秘密，但这并不意味着你必须心甘情愿地把你的信息给所有人。

哈姆写道:“如果有人要我的电话号码或地址，或者更糟糕的是，要我的社会安全号码或驾照号码，那就不值得节省几分钱或赚取几分利息。”“这不仅增加了垃圾电话和垃圾邮件的可能性，还增加了身份盗窃的风险。我不同意这些。”

### 注意它是否会影响你经常使用/做的事情

我们所有人都过着不同的生活，但一般来说，这意味着如果你能在家务、孩子、房子、为大多数人开车等方面节省一些钱。，至少值得考虑。

“例如，我经常开车，所以省钱的策略，比如让我的轮胎保持在最大推荐压力，坚持维护计划，对我来说是很大的价值，”哈姆写道。“我整天都在喝饮料，所以训练自己多喝水是另一个省钱的好方法。”

### 如果不影响您经常使用/做的事情，请跳过

哈姆将此定义为每周少于两次，但显然会因人而异。他写道:“如果这种节俭策略影响了我不常做的事情，并且每次只为我节省了几分钱，那可能就不值得担心了。”“我不担心在去咖啡店这样的事情上省钱，因为我几乎从不去咖啡店。”

理由是因为在不经常影响你生活的东西上省几个便士是不值得的。

### 如果新的任务/惯例只需要很少甚至不需要投资，就参与进来

精打细算并不好玩，但是如果某样东西的替代品可以帮你省一点，而且不需要太多努力(或者你的个人信息)，那就值得一试。

同样，如果你必须注册一些东西，但它不需要更多的个人信息，但它节省了少量的钱，考虑这样做。哈姆写道:“这是一个相当广泛的范畴，因为它包括诸如提高住宅能效、打电话降低手机费用以及对信用卡进行零利息余额转账等。”。或者 [增加百分之一的退休储蓄](https://twocents.lifehacker.com/increase-your-401-k-or-ira-contributions-for-2019-1830819672) 。

### 如果需要持续投资，则跳过

如果你不得不经常提醒自己做一些不同的事情来节省一些钱，这可能不值得争论。一如既往，这些任务和行为对每个人来说都是不同的，你需要自己决定货币门槛。一个例子可能是剪下一些你可以购买仿制药的东西的优惠券。
