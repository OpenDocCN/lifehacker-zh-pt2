# 必胜客的 Book-It 计划仍然存在

> 原文：<https://lifehacker.com/pizza-hut-s-book-it-program-is-still-around-1833131581>

作为一个 90 年代在农村工人阶级地区长大的爱读书、好奇的孩子，我的生活围绕着三件事:我的足球队、图书馆和必胜客。后两者有着内在的联系，原因我稍后会解释(第一个肯定有助于燃烧后一个的热量余波)。看，在一年中辉煌的六个月里，我接触到了最精彩的骗局。



每周，我都可以带一张小纸条，上面写着我读过的三本书的名字，还有我妈妈的签名，然后从我的老师那里得到一张特殊的邮票，老师总是有点怀疑我是否经常这样做，但无法忽视我的小书桌上不断堆积的书籍。一旦我的文书工作就绪，接下来就是激动人心的部分:拿着我签好的单子去几个镇外的必胜客，用它换一份免费的儿童披萨，然后我就可以一个人吃了。

这绝对是不可思议的，而且对我的父母也是一个巨大的帮助，他们并没有在“让我们带孩子们出去吃午饭”的钱里游泳。它叫做预定它！骗走了我无尽的免费披萨(并因为我的书呆子习惯而得到回报)仍然是我最美好的回忆之一。

我认为，就像我的童心和美国民主的幻想一样，预订吧！在 2000 年代初的某个时候，由于缺乏兴趣而消亡，被送进了公司历史的墓地。但是，你猜怎么着？我错了。*必胜客现在还这么干！*没错:这个深受喜爱的项目依然生机勃勃，他们甚至建立了一个异常紧张的 [网站](https://www.bookitprogram.com/resources) 来帮助老师和家长跟踪他们孩子的进步。

预定吧！简单中透着精致:老师设定每月阅读目标，达到目标的孩子会得到免费披萨的奖励。K-6 年级(5-12 岁)的小学和家庭学校的学生有资格获得，父母被鼓励与还没有自己阅读的孩子一起阅读(这仍然算作比萨饼)。我记得当我因年龄原因退出这个项目时，我简直是*怒不可遏，但幸运的是，我身边有一个小妹妹，她的阅读速度足以让我们俩在接下来的几年里享受披萨。*

所以，美国的成年人和大兄弟姐妹们，请注意:如果你的生活中有孩子，或者想鼓励他们多读书，或者手头有一个饥饿的书虫，那就从必胜客的书中吸取教训，沉迷于一些老式的贿赂吧。
