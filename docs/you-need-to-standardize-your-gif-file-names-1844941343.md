# 你需要标准化你的 GIF 文件名

> 原文：<https://lifehacker.com/you-need-to-standardize-your-gif-file-names-1844941343>

30 岁以上的人，gif 是中年千禧一代的数字等价物:大器晚成的人，现在过度工作并被误解。今天，它们是短信和社交媒体线程中的标准互联网货币，由于它们不再是新奇的事物，我们对它们已经变得有点懒惰了。我在和你说话，那些粒状的、不连贯的、过度使用的反应的发送者们，他们的应用程序集成自动地暗示着这些反应。



我要说明的是:如果你搜索一个单词，并使用 10 大反应 GIF，你就不好笑。我就更直接了:你甚至没有用你的詹妮弗-劳伦斯式的反应来取悦任何人。现在我只想说:在你发送另一个道森怀疑的哭泣，乔纳·希尔的尖叫，或克莉茜·泰根畏缩的 GIF 之前，把你的手机扔进他妈的河里。

这里有一个好的 GIF 的快速纲要。 [完美的 GIF](https://i.imgur.com/5M2Qb4o.gif) 是一个无缝的循环。如果你几乎不知道循环从哪里开始，你已经找到了(或者更好，制作了)一个漂亮的 GIF。取而代之的是，一个伟大的 GIF 有一个明显的循环，但在其有限的框架空间讲述一个故事。它甚至可以与一个 [击掌，就像这个](https://media.giphy.com/media/xUPGclJ5wKOJmENxcs/giphy.gif) 。最差的 gif 都是循环太短，使其显得不平稳，令人不快， [像这张](https://tenor.com/JKNI.gif) 。一个糟糕的 GIF 如果在你的屏幕上时间太长，几乎会让你晕车。

过度使用的 gif 在视觉上类似于无意义的填充词，或者用“有趣”来回复朋友的消息。(“有趣”是对你被告知的任何事情最空洞、最令人愤怒的回应，但那是另外一个故事了。)你比那更好——或者至少我希望你想成为——所以这里有一个简单的方法来创建你自己的反应 gif 的战争箱，这些 gif 对你和你的幽默感是个人的，*和*还没有被地球上所有的 70 亿人看到。

你已经知道 [怎么做 GIF](https://lifehacker.com/the-complete-guide-to-making-animated-gifs-1503276993) 了，我就不重点说那个了；相反，我们需要谈谈当你已经*有*这些 gif 时该怎么做，这样你就可以轻松地使用它们。这始于组织，也止于组织:标准化你的 GIF 文件名，包括来源、角色名、物理描述符和情感基调。当你发现或制作一个很棒的新 GIF 时，每次都要花额外的时间以一种标准化的方式给它命名(并且，很明显，把它们保存到你设备上的一个适当的文件夹中)。

我们都尝过在搜索 GIF 时很难找到它的挫败感——互联网文化就相当于在你的舌尖上有一个单词，但你却不记得了。从至少三个不同的角度组织你的文件名将有助于确保你总是能找到它。这是我看到的样子。这个方法对我的大脑有效，但是你可以根据自己的情况来定制:

[霓虹创世纪 Evangelion-Gendo Ikari-Fire Glasses-冥思苦想. gif](https://gfycat.com/abandonedphonyasianporcupine)T3】

回忆不起来电视剧，可以回忆人物；如果回忆不起来人物，可以回忆一下*你个人*是如何描述情感基调的，等等。通过用个性化的、标准化的命名约定保存你的 gif(或者任何图片文件),我保证你会在需要的时候找到你需要的。文件名会很长，但这是值得的。

所以当你保存 gif 的时候要组织好它们，这样当你需要它们的时候，它们就会在你身边。你不必每次都自己制作 GIFs 虽然你可以，我向那些这样做的人致敬——但是请不要再做艾玛·斯通 T3 了。
