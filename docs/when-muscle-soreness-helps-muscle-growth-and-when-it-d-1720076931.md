# 肌肉酸痛什么时候有助于肌肉生长(什么时候没有)

> 原文：<https://lifehacker.com/when-muscle-soreness-helps-muscle-growth-and-when-it-d-1720076931>

剧烈运动后，伴随肌肉酸痛而来的是某种自豪感——就好像你如此用力地推动你的身体，以至于你仍然能感觉到运动在发挥它的魔力。这很好，但 [酸痛可能不是](https://vitals.lifehacker.com/why-your-muscles-get-sore-and-what-you-can-do-about-it-1680937155) 肌肉增长的最佳指标。我们的意思是。

可怕的 DOMS(或称 [延迟发作肌肉酸痛](https://www.ncbi.nlm.nih.gov/pubmed/12617692) )是一种相当新的锻炼后几天*你感觉到的那种邪恶的僵硬和局部疼痛。(为了记录在案，我们具体谈论的是举重引起的疼痛。)当它来的时候，它打击很大，但大多数人都津津乐道于它，认为已经取得了严重的肌肉增长和进步。*

但事实未必如此， [根据身体重组](https://bodyrecomposition.com/training/doms-muscle-growth.html/) :

> 需要考虑的四件事:
> 
> 1.DOMS 通常在训练周期的开始是最差的，特别是有新动作的时候，但是明显的和增加的增长通常出现在周期的末尾，这时 DOMS 不再发生。

> 2.一些肌肉，三角肌就是一个例子，很少会因为某些原因而疼痛，但它们生长得很好。DOMS 不是必需的。
> 
> 3.很少锻炼的人，比如每周锻炼一次身体部位的人，经常报告惊人的 DOMS。但是很多都长不好。
> 
> 4.训练更频繁的人(即每个肌肉群每周两到三次)总是报告较少的 DOMS，但更多的增长。
> 
> 基本上，以上似乎表明，不仅 DOMS 与增长无关，而且在大多数情况下，你会得到更好的增长(T1)，而减少 T2(T3)

此外，2012 年的一项名为 [*的研究显示，白细胞、细胞因子和卫星细胞:它们在离心运动后的肌肉损伤和再生中发挥了什么作用？*](https://www.ncbi.nlm.nih.gov/pubmed/22876722) 发表在《运动免疫学评论*杂志上，论述了肌肉中 [卫星细胞](https://www.ncbi.nlm.nih.gov/pubmed/12757751) 的激活“似乎与肌肉损伤标志物没有直接关系”然而，研究人员也指出，最初的肌肉损伤创造了卫星细胞再生和建造更大肌肉所需的刺激。*

*这里值得注意的是，无论如何，DOMS 也不是*肌肉损伤的最佳指标*。* 

*然而，记住生物学并不是非黑即白的。根据你看的是短期的还是长期的训练适应，上面所说的会有所不同。另外，每个人都是不同的，对锻炼会有不同的反应。*

*那么，这对你意味着什么？力量教练 [Greg Nuckols](http://www.strengtheory.com/about/) 建议在短期内， [避免过多的肌肉损伤](http://vitals.lifehacker.com/three-common-fitness-traps-to-avoid-and-what-to-do-ins-1680448743) 为了让你更高频率的训练有所回报；你的肌肉会长得很好，短期内不会有持续的、显著的损伤。*

*在很大程度上，时不时的高度肌肉损伤确实很有帮助，但是不要把每一次锻炼都花在推上，直到你一瘸一拐地走出健身房。这可能会损害你充分进行下一次锻炼的能力，更不用说，从长远来看，对你的关节有负面影响。一如既往，听从你的身体。*
