# 这些天气应用程序是苹果拥有的黑暗天空的绝佳替代品

> 原文：<https://lifehacker.com/these-weather-apps-are-great-alternatives-to-the-apple-1842591377>

“黑暗天空”刚刚宣布被苹果收购，出售给超级本地天气应用的粉丝带来了一些大问题。首先，由于显而易见的原因，Android 应用程序将彻底消失。你可以继续使用它，直到 7 月 1 日，订户将被退还他们在订阅上花费的任何钱，这些钱很快就会变得一文不值。

Watch

第二，同样重要的是，黑暗天空的 API 将在明年消失。据该公司称，它将一直活跃到 2021 年，但从今天开始不接受任何新的注册。换句话说，你最喜欢的利用黑暗天空信息运行的天气应用程序，无论是胡萝卜天气还是 Hello 天气，都必须切换到另一个数据源——谁能说这对它们的准确性意味着什么。

虽然开发者有一点时间来重组，但如果你使用 Android，你将不得不寻找另一个选择，而且很快。即使你是一个自豪的黑暗天空 iOS 用户，我也希望应用程序在某个时候会发生变化，这可能会让你不那么激动。见鬼，苹果甚至可能会选择将其淘汰，并将 Dark Sky 的功能直接整合到原生 iOS 天气应用中。天空是极限——这是双关语。

与此同时，这里有一些其他的天气应用程序值得一试——假设你 [信任*他们中的任何一个*](https://www.vice.com/en_us/article/gy77wy/stop-using-third-party-weather-apps)你的位置数据。(你的设备内置的天气应用也很不错！)

### 胡萝卜天气([iOS](https://apps.apple.com/us/app/carrot-weather/id961390574)[Android](https://play.google.com/store/apps/details?id=com.grailr.carrotweather&hl=en_US))

是的，胡萝卜天气利用了黑暗天空的数据，入门费是 5 美元——这比你目前在 iOS 上为黑暗天空支付的费用要高。然而，如果你支付 5 美元的年费，你可以在应用程序中的许多不同数据源之间切换:AccuWeather、ClimaCell、Foreca、MeteoGroup、Aeris Weather 或 WillyWeather。我喜欢这种灵活性，它让我可以看到哪个来源对我的位置最准确。

### 
你好天气( [iOS](https://apps.apple.com/us/app/hello-weather/id978393692) ， [Android](https://play.google.com/store/apps/details?id=com.helloweatherapp&hl=en_US) )

第二节，和第一节一样:“你好天气”也利用了“黑暗天空”的 API，但这种情况肯定会很快改变。这款应用看起来很棒，就像胡萝卜天气一样，它允许你将你的天气数据源切换到你认为合适的其他服务。这款应用程序可以免费使用，但你必须支付月费(1 美元)或年费(9 美元)才能使用它的高级功能，包括切换上述数据源。天气是要花钱的，但我不介意帮助三个开发者的团队，他们把这个应用做成了“一个热情的项目，而不是铁杆资本主义”

### 风大( [iOS](https://apps.apple.com/us/app/windy-com/id1161387262) ， [Android](https://play.google.com/store/apps/details?id=com.windyty.android&hl=en_US) )

我自己很少使用 Windy 应用程序，但它的网站是 [看起来绝对很漂亮](https://www.windy.com/?37.294,-121.900,5)——当你工作时在第二个显示器上保持同步的完美事物。该应用基本上是该网站的一个扩展，完全免费，可以满足你的基本天气需求。虽然对于那些对查看你所在地区的可视化风力条件和雷达及卫星天气地图感兴趣的人来说，它确实更有用，但每次启动应用程序时，它都会提供一个简单的五天预报，你可以滚动查看在给定时间你所在地区的条件可能是什么样的。点击你所在的位置，可以调出更多的细节——如上面的截图所示——或者为你想要的任何地区设置天气和温度条件的警报。

### 天气-天气频道([iOS](https://apps.apple.com/us/app/weather-the-weather-channel/id295646461)[Android](https://play.google.com/store/apps/details?id=com.weather.alps&hl=en_US))

第一点很难出错——至少在 App Store 上。如果我没弄错的话，天气频道实际上提供了你会在苹果默认应用程序中找到的天气信息，但如果你宁愿比你通常使用的信息多一点，而不是干净的外观，它的官方应用程序是一个不错的选择。基本服务不需要花费任何成本，但是付费移除广告和获取更精确的预测(和更多数据)将花费你每月 4.99 美元或一年 30 美元。哦。

### 天气地下( [iOS](https://apps.apple.com/us/app/weather-underground-forecast/id486154808) ， [Android](https://play.google.com/store/apps/details?id=com.wunderground.android.weather&hl=en_US) )

我已经提到了“三大”天气应用中的两个——天气频道和 Accuweather——所以通过引入地下天气来结束三方的斗争才是公平的。这三款应用中的一款很有可能满足你对最新天气预报的需求，但它们不同的精确度才是最重要的。我建议调查所有的应用程序，或者一个你更喜欢的利用它们的数据流的应用程序，看看哪一个最适合你居住的地方。(或者欺骗并使用[](https://www.forecastadvisor.com/)****)。)****

**无论如何，回到地下天气:这有点类似于天气频道的应用程序，这是合适的，因为 TWC 拥有两者。它给你预测和一个体面的用户界面，但它充满了广告——毫无疑问，这是一种向你追加销售 4 美元/月或 20 美元/年订阅费的方式，以消除它们。我不喜欢这一点，尽管地下天气最终比 TWC 自己的应用程序便宜得多，但如果它是你所在地区最准确的提供商...好吧。这很难被忽略，即使它的用户界面一般。毕竟，比起屏幕上显示的内容，你可能更关心可能从天上掉下来的东西。**
