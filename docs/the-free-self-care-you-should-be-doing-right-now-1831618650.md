# 你现在应该做的免费自我护理

> 原文：<https://lifehacker.com/the-free-self-care-you-should-be-doing-right-now-1831618650>

下面说说如何照顾好自己。不是公司以此为借口让你买他们的屎的“自理”，而是如何覆盖你身心的基本需求。



### 给自己喂些好的食物，这样你就不会那么渴望垃圾食品了

我们都有一种我们可能知道应该少吃的食物。我不会去评判，但如果你读到这封信时想到了什么，你已经知道是什么了。也许是饼干。也许是冰淇淋。也许是从一个绝对不健康的地方买的外卖，但是当你饿的时候，它们总是在那里。

在你开始为此自责之前，花一分钟时间计划一下你希望你的饮食是什么样的，现实一点。我不在乎是高热量还是低热量，是食肉动物还是素食主义者；但是给你自己找一些好的食物，并且早点经常喂给自己吃。蔬菜。蛋白质。

如果你面前有健康的食物， [多吃点](https://vitals.lifehacker.com/healthy-meals-should-have-bigger-portions-not-smaller-1699243990) 。那么当它发生时，你就不会被那诱人的东西诱惑了。如果你决定犒劳自己，没什么大不了的，因为你已经有了如此健康的饮食。

### 设定一个现实的锻炼时间表

锻炼对我们非常有益。它降低了我们患心脏病和糖尿病的风险，它让我们有一个充满活力的晚年，它有时会消除抑郁或焦虑。底线是，你的生活有一些锻炼总比没有好。

所以在你的日历上标记一些。如果你把锻炼作为一天或一周中的最后一件事，它将会从你的日程表上掉下来。弄清楚你能管理的最少时间，然后在你的日历上划掉那些时间。

没有健身房？试试，你可以在家里做，或者查看一下 [我们的指南，在酒店或公寓楼里的那种基本健身房](https://vitals.lifehacker.com/how-to-get-a-good-workout-in-a-crappy-hotel-gym-1830405477) 锻炼。

### 清洗一件东西

无所事事地坐在自己的杂物堆里是很糟糕的，但是我们并不都有精力[玛丽近藤- 风格](https://lifehacker.com/the-life-changing-magic-of-tidying-up-mind-hacking-adv-1749135755#_ga=2.62807596.1274489844.1546958312-1723114163.1524514905) 大清洗。

所以: C 精益一件事。这可能是一个你经常使用(或者希望可以使用)的水平面，比如你的餐桌、办公桌或者你很确定曾经是梳妆台的那堆东西。花 20 分钟，你会惊讶它有多干净。(拍前后图！)然后宣布它是干净的，从现在开始，当你看着它时，它不会是“那张乱七八糟的桌子”，而是“我干净的桌子，但上面有三样东西我应该放好。”

### 安排大事

今年你需要去看什么医生？在你的心理日历上还有哪些重大事件？选择一个日历(手机上的默认应用程序就可以了)，输入重要的事情，基本上包括任何你可能忘记的事情。那个你不用再担心八个月的最后期限？从现在开始设置七个月的提醒。

### 获得充足的睡眠

你可以决定什么时候睡觉，什么时候起床，但是要确保这两个数字相互匹配。

当你熬夜的时候，无论是工作还是聚会，你都会跨越一条线，从“今晚会少睡一会儿”到“明天真的会很糟糕”通过练习，你会找到那条线；你可能已经知道它在哪里了。不要告诉任何人你是少数能靠四小时睡眠生存的人；你知道你不是。去睡觉吧。

### 检查某人，或者让某人检查你

有时自我照顾往往是别人应该为我们提供的照顾的权宜之计， [谢拉·洛夫在《副》](https://tonic.vice.com/en_us/article/zmdwm4/the-young-and-the-uncared-for-v25n4) 中指出。例如，为了找到内心的平静，我们会求助于泡泡浴和冥想应用——但如果我们生活在一个可以依靠良好的精神健康覆盖的社会，情况会怎样呢？

认识到你的社会、家庭或朋友可以帮助你的方式，以及你可以帮助别人的方式。你应该去食物银行吗？你能捐给食物银行吗？

如果你需要一些你自己不能提供的东西，想办法去得到它。第一步可能是接近一个朋友或家庭成员，他们可以帮助你想出下一步该做什么。如果你正经历一段困难时期，你知道人们会问他们如何能帮上忙，那就列出一张你需要帮助的清单。

另一方面，如果你有时间、精力和金钱，考虑为其他人争取更好的照顾。这可能是检查你的朋友，也可能是打一些政治电话。毕竟，建设一个更美好的世界是自我关怀的终极目标。
