# 你可能得到了减税，即使感觉不像

> 原文：<https://lifehacker.com/you-probably-got-a-tax-cut-even-if-it-doesnt-feel-like-1834046892>

你今年得到减税了吗？根据税收政策中心的新数据，事实上，大多数美国人在减税和就业法案下缴纳的税更少了——但由于我们许多人获得的退款更少(或根本没有退款)，可能感觉不像。



[为*《纽约时报》*报道](https://www.nytimes.com/2019/04/14/business/economy/income-tax-cut.html) :

> 税收政策中心 [估计](https://www.taxpolicycenter.org/model-estimates/individual-income-tax-provisions-tax-cuts-and-jobs-act-tcja-february-2018/t18-0026)65%的人依法纳税较少，只有 6%的人纳税较多。(其余的人认为他们的税收没有什么变化。)
> 
> [...]
> 
> “大多数人没有意识到税后工资的增长，或者至少没有将其归因于减税，”[H&R 街区分析师内森·里格尼]说。他说，他们中的一些人可能意识到现在他们正在报税，但“发现你在一年中收到了几千美元，但你已经花掉了，这没有什么安慰。”

不幸的是，尽管有上面的引述，我们中的许多人并没有得到“几千美元”根据美国人口普查局 的统计，如果你是收入在 47，110 美元到 77，552 美元之间的美国中产阶级家庭，你的平均减税额只有 780 美元。高收入者获得的减税幅度更大，收入最高的 1%的美国人获得的减税高达 3 万美元。

所以。你今年交的税少了吗？这些节税体现在你的工资中了吗，或者你最终得到了退款？如果你在 2018 年确实收到了略高的薪水，你是否觉得你得到了税收优惠——或者你仍然觉得你错过了承诺的减税？
