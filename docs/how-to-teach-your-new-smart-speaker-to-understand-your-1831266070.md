# 如何教你的新智能音箱理解你的声音

> 原文：<https://lifehacker.com/how-to-teach-your-new-smart-speaker-to-understand-your-1831266070>

并非所有的智能音箱都是生而平等的。事实上，有些人根本不具备被称为“聪明”的资格——但他们在回答你的问题方面越来越好，假设他们能理解你在说什么。

Watch

投资公司 Loup Ventures 该公司去年也进行了这项测试，并对结果进行了比较，以了解每款智能音箱的性能提高了多少。

虽然今年所有接受测试的助理都能理解分析师提出的 99%以上的问题，但这并不意味着每个人都能与他们的智能扬声器进行很好的对话。如果你的努力理解你在说什么，它不会有太多的机会给你正确的答案。我们稍后将讨论培训技巧。首先，让我们快速浏览一下 Loup Ventures 的测试结果:

*   **第一名:谷歌助手**
    谷歌助手正确回答了 88%的问题——高于 2017 年 12 月测试的 81%——并理解了所有 800 个问题。在 Loup Ventures 的分析中，它是表现最好的数字助理。
*   第二名:苹果的 Siri
    Siri 今年正确回答了 75%的问题，而去年的正确率为 52%。

*   第三名:亚马逊的 Alexa
    Alexa 去年答对了 64%的问题，今年答对了 73%。
*   第四名:微软的 Cortana
    Cortana 去年的正确率为 56%，今年为 63%。

## **如何训练 Alexa 学习你的声音**

1.  打开 Alexa 智能手机应用程序。

2.  点击左上角的**菜单图标**打开侧边栏，然后进入**设置> Alexa 账户>识别语音。**
3.  确保**“自动识别语音”**已打开，然后轻按**您的语音。**
4.  点击**开始**，然后**选择设备，**并点击您想要训练的回声(确保静音或关闭您附近的任何其他回声)。
5.  按照提示，当指示时大声说出短语。
6.  完成练习后，轻按“完成”以结束。Echo 将处理您的语音数据(大约需要 10 分钟)。
7.  您还应该在应用程序中的**设置>账户的 Echo > Echo 位置**和**设置>您的位置**下设置您的 Echo/Alexa 设备位置，这样您将为您询问的本地和基于导航的问题获得最佳结果。

**提示:**你可以通过 Alexa 应用向亚马逊发送对 Alexa 性能的反馈。Alexa 智能手机应用程序的主屏幕显示了按时间顺序排列的“卡片”，每张卡片都是你向连接的 Alexa 设备发出的命令。你可以在每张卡上点击“是”或“否”，让亚马逊知道 Alexa 是否正确理解了你的命令。

## **如何训练谷歌助手**

1.  在你的 Android 设备上点击并按住 **Home 键**，直到谷歌助手窗口弹出。
2.  点击**指南针图标**，点击右上角的**“更多”图标**(三个堆叠的点)，然后点击**设置>助手。**
3.  点击**“语言”、**并设置适当的语言。如果您的未显示，请轻按“添加语言”回到助手选项卡，在“**助手语音”下做同样的操作**确保“语言”和“助手语音”屏幕上的语言和地区相匹配。

4.  返回“助理”选项卡，然后向下滚动到**“助理设备”**并点击您想要训练的设备。
5.  轻点**语音模式**。
6.  如果您有以前的声音模型，点击**“删除声音模型”将其删除**否则，点击**重新训练声音模型**开始训练过程。
7.  按照屏幕上的指示，以随意、对话的语气(在安静的地方)说出短语。
8.  完成后，向下滚动并点击**语音匹配**，然后点击**添加设备**。选择您希望与您的语音关联的设备(该设备需要与您的电话连接到同一个 wifi 网络)。遵循屏幕上的说明。

这些其他步骤是可选的，但启用它们可以改善您与 Google Assistant 的交互。

*   在助手选项卡中，点击**“继续对话”**并打开设置，这将允许您向谷歌助手提出后续问题，而不必每次都重复“好的，谷歌”或按住 Home 键。
*   在“个人信息”标签下，确保设置你的家庭和工作地址、交通方式和支付方式。为重要的联系人设置昵称或快捷方式，这样谷歌助理就会知道你说的“妈妈”和“爸爸”是指谁。

## **如何训练 Siri**

1.  在你的 iOS 设备上，进入**设置> Siri &搜索**。

2.  确保**“语言”**和**“Siri 语音”**选项设置为正确的语言/地区。
3.  关闭**“收听‘嘿 Siri’”、**，然后再次打开。

4.  按照说明进行操作，并在出现提示时说话。
5.  一旦你完成了这些，返回到**设置> Siri &搜索**，然后浏览你想要连接到 Siri 的每个应用程序，并确保在应用程序的设置中打开了**“配合 Siri 使用”**。
6.  最后，进入**设置>隐私>位置服务**，确保**位置服务**和**分享我的位置**已启用。这样，Siri 将知道你在哪里，何时在家(如果你还没有在 iOS 设备上设置家庭地址，你可以在“通讯录”应用程序中以你的名字设置。您还可以保存其他位置的地址，如工作地点、学校等。)

**提示:**当 Siri 念错一个单词的时候，不要犹豫去纠正她。你可以说“Siri，你不是这么说的”，然后用正确的发音说出这个单词。Siri 会根据你的声音提供一个发音列表。选择与单词最匹配的单词，或者轻按“再次告诉 Siri”再试一次。

## **如何训练 Cortana**

**在你的电脑上:**

1.  单击或点击 **Windows 搜索栏**打开 Cortana。
2.  点击或轻触**设置图标**。
3.  将弹出一个新的“与 Cortana 对话”窗口。在这个窗口中，打开**“嘿 Cortana。”**
4.  Cortana 会征求你的同意。点击**“确定。”**
5.  回到“与 Cortana 交谈”窗口，点击**“学习我如何说‘嗨，Cortana’。”"**
6.  按照说明操作。

**在 Windows 10 手机或平板电脑上:**

1.  点击或轻触 **Windows 搜索栏**打开 Cortana
2.  点击或轻触**笔记本图标>设置**
3.  在设置菜单中，打开**“Hey Cortana”**
4.  选择**“学习我的声音”**
5.  遵循指示

如果您的语言和地区设置匹配正确，Cortana 将会工作得最好，因为根据微软的帮助页面 ，Cortana 仅在特定地区 [提供特定语言:](https://support.microsoft.com/en-us/help/4026948/cortanas-regions-and-languages)

*   澳大利亚:英语
*   巴西:葡萄牙语
*   加拿大:英语/法语
*   中国:中文(简体)
*   法国:法语
*   德国:德语
*   印度:英语
*   意大利:意大利语
*   日本:日语
*   墨西哥:西班牙语
*   西班牙:西班牙语
*   英国:英语
*   美国:英语

您可以仔细检查这一点，并在 Windows PCs 和移动设备上轻松进行调整。

**在 PC 上:**

1.  打开 Windows 开始菜单，然后点击**设置>时间&语言>地区&语言。**
2.  请确保您的区域和 Windows 显示语言正确；如果没有，请从下拉菜单中选择正确的选项。如果您的语言未包含在内，您可以点击**“添加语言”**下载新的语言包，并选择您希望安装的语言包。下载和应用新的语言包需要几分钟时间
3.  接下来，单击**语音**选项，并确保语音语言选项与 Windows 显示语言相匹配。
4.  如果您更改了这些设置中的任何一项，您需要注销 Windows，然后重新登录。(您也可以重新启动计算机来应用更改)。

**在 Windows 10 移动设备上:**

1.  打开设备的设置菜单。

2.  转到**时间&语言>语言**。如有必要，更改所选语言。如果您的语言不在列表中，点击**“添加语言”**，然后点击您希望安装的语言包。

3.  在“语音”选项下，确保所选语言与上一步中的语言相匹配。

4.  进行这些更改后，您可能需要重新启动设备。
