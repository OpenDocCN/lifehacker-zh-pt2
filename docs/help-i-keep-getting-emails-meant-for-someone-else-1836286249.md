# 救命啊！我不断收到给别人的邮件

> 原文：<https://lifehacker.com/help-i-keep-getting-emails-meant-for-someone-else-1836286249>

欢迎回到 [**Tech 911**](https://lifehacker.com/c/tech-911) ，Lifehacker 的每周技术建议专栏，它总是在寻找解决你的挑战——极客、奇怪或其他。本周，我们来看看一个认错人的案例。

这种情况我太熟悉了。我有一个相当普通的名字(感谢你毁了我的谷歌排名，)和一个与这个名字相近的电子邮件地址(感谢你盗用我的名字，在 Gmail 中击败我的大卫·墨菲)。因此，我有时会收到别人的邮件，而不是给我的，我的分身有时会收到我的邮件，而不是给他的。

Lifehacker 的读者 Michael 也有类似的问题:

> 看了你关于寄给别人的邮件的文章后，我想就我的问题征求你的意见。
> 
> 我不断收到那些把我的地址作为他们地址的人的电子邮件。

> 要么是打字错误，要么是他们没有电子邮件地址，而我的电子邮件地址是他们第一个想到的。在我的家乡，这是一个常见的昵称。
> 
> 有时候这很有趣，就像有一次一个音乐会推广人联系我，提供一个演出的机会。有时这很严重，就像房地产经纪人发来一份时间敏感的租赁协议。主要是它很烦人，就像那个帮我注册外卖服务的快餐迷...在另一个城市。
> 
> 如果我很慷慨，我会回复并告诉他们这封邮件是错的。其他时候，它们被标记为垃圾邮件。我用快递服务更改了密码，然后关闭了账户。
> 
> 但是处理这件事的最好方法是什么？"

老实说，你无法阻止其他人使用你的电子邮件地址注册服务——不管是不是搞错了。如果你养成了将发件人列入白名单的习惯，这是你可以用各种应用 或 [疯狂过滤器](https://webapps.stackexchange.com/questions/36257/how-to-accept-email-only-from-white-listed-addresses-in-gmail) 轻松做到的事情，你可能会错过一封你*希望*收到的来自某个以前从未困扰过你的人(或某家公司)的电子邮件。

如果这些身份错误的电子邮件有一些共同之处——例如，可能有人在介绍中使用了你名字的变体——你可以将包含该文本的邮件过滤到一个独立的文件夹中，而不是收件箱中。否则，我想不出有什么好办法来阻止这类邮件进入你的收件箱。某种" [跳过这个额外的环，以证明你真的打算给我发消息](https://lifehacker.com/make-your-own-email-spam-filter-1829437006) “设置并不真正适用于这种情况，因为你没有被*垃圾邮件*本身。这是一个简单的认错人的例子，比大多数人更容易发生在你身上。

### 也许 Gmail 是罪魁祸首

如果你在 Gmail 上收到了这些误发邮件——你没有指明——有可能有人混淆了 Gmail 在电子邮件地址中忽略的各种符号，比如句号。在这种情况下，向 david.murphy@gmail.com 或 davidmurphy@gmail.com 发送邮件会到达完全相同的 Gmail 帐户，即使这是两个“不同”的地址。如果是这种情况，你*可以* [设置过滤器](https://www.reddit.com/r/GMail/comments/bc3ucy/i_receive_mails_intended_for_other_people/) 来捕捉和丢弃发送到错误地址的消息——确保如果你实际使用的地址是 davidmurphy@gmail.com，你永远不必处理任何发送到 david.murphy@gmail.com 的消息。

### 自动回复你的“这不是我”

至少，你可以在查看邮件时使用键盘宏工具，比如[](https://www.autohotkey.com/)****。按下键盘上的几个按钮，你就可以粘贴一条预先生成的消息，通知发件人他们联系错了人。当你回复目标错误的邮件时，这会节省你一些时间。****

### ****取消订阅、报告或过滤****

****除此之外，如果发件人变得令人讨厌，你可以通过邮件中的链接退订他们的邮件(如果适用)。你也可以举报它们是垃圾邮件——如果你在使用 Gmail 这样的服务,“举报垃圾邮件”功能可能会提示你取消订阅。如果情况变得更糟，你可以过滤发件人，这样他们发送的所有信息都将进入你的垃圾箱，但你的问题听起来更像是来自多个发件人的一次性问题，而不是一个发件人在你的收件箱中塞满了给其他人的邮件。****

****无论如何，确保你在这个问题上保持领先，因为如果有人用你的电子邮件地址注册了一个相当受欢迎的服务，比如脸书、优步等等，你会想迅速采取行动。不得不 [联系](https://www.facebook.com/help/contact/255904741169641) 公司来纠正这个问题会很烦人，但这样做总比有人冒充你好。(你也可以在服务本身上执行密码重置，因为你拥有访问电子邮件地址的权限，并自己接管帐户。)****

### ****删除键是你的朋友****

****不管怎样，我有点言过其实了。如果我是你，我要么回复这些邮件来纠正错误(如果你感觉很好)，要么干脆忽略/删除它们。这种事情时有发生。如果这些不正确的电子邮件太多了，有一个好方法可以让你把它们过滤掉，这是值得研究的。****

****无论你做什么，不要改变你的电子邮件地址。一寸都不要放弃。****
