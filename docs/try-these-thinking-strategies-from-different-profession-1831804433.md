# 如果你被卡住了，试试这些不同职业的思考策略

> 原文：<https://lifehacker.com/try-these-thinking-strategies-from-different-profession-1831804433>

“我对这个案子的想法变得非常紧张，”在《大勒保斯基》中的那个家伙说，他发现另一个勒保斯基持有赎金。如果你发现那条线是“大情绪”，用我们这个时代的说法，那么试试这些从各种职业聚集的 [二十五个有用的思考工具](https://www.scotthyoung.com/blog/2018/12/04/25-thinking-tools/) 。

励志作家斯科特·h·杨(Scott H. Young)列出了与 25 种不同职业相关的心智模型:一位经济学家经常考虑人们对激励的反应，而一位哲学家则考虑“直觉的意外后果”。一个好的老师能够理解了解他们所知道的事情是什么样的，一个好的记者在做出进一步的结论之前会核实事实。但重点不是总结职业。重点是记住这些心智模型，并把它们用在你自己的问题上。

如果你的创造力受阻，你可能不会从一个艺术家的模型中得到答案，而是从一个人类学家那里，他们通过沉浸在一个陌生的文化中来学习。或者你可以采用企业家的快速原型法:快速测试许多想法，看看哪些是最有前途的。

如果你的工作太分散，你可以使用程序员的方法，尝试自动化你的任务——即使这意味着找到一个通用的解决方案，在每项任务上只做一件像样的工作。如果你在领导一个团队方面有困难，也许你需要像销售人员一样思考，弄清楚其他人想要什么，并从那里逆向工作，使之与你想要的相匹配。

杨的清单并不确定，所以你可以找到自己的心智模型来借鉴。这些模型能以令人惊讶的方式与你的问题相匹配。我有一个创造性的项目，需要更多的士兵方法:“常规和纪律防止致命的错误。”
