# 完美自制爆米花的秘诀是铝箔

> 原文：<https://lifehacker.com/the-secret-to-perfect-homemade-popcorn-is-aluminum-foil-1844483799>

爆米花不仅仅是一种小吃；对我来说，这是一种生活方式。我对这种生活方式的执着意味着我总是对学习新方法感兴趣，以加快 或 [的速度，或者提高](https://lifehacker.com/youre-not-using-enough-oil-to-pop-your-popcorn-1827137012) 我的技术——我最近发现了我认为是我完美爆米花程序的最后一部分。



当然，我是从 YouTube 上学会的。在新冠肺炎隔离大约一个月后，Polygon 视频制作人 [布莱恩·大卫·吉尔伯特在他的个人频道](https://www.youtube.com/watch?v=3fG8rNHUspU) 上发布了一个关于“胡椒”的视频，这是一种标志性的调制食品，使用老干妈辣椒酥和 gochugaru 来获得美味、辛辣的感觉。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-3fG8rNHUspU&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-3fG8rNHUspU&start=0) 

几个批次后，我决定 pepcorn 不是我的果酱，但它甚至没有关系，因为这个视频是一个不间断的自制爆米花黑客游行。我最喜欢的:BDG 没有用普通的锅盖，而是用一张铝箔盖住炒锅，然后用刀捅开。

对我来说，这是天才。穿孔箔片的工作原理就像一个内置的释放阀，允许刚好足够的蒸汽聚集来启动爆裂，然后立即释放多余的蒸汽，这样蒸汽就不会滴回锅里。另外，你不必玩“半开有多宽”的游戏游戏，或处理一个热，沉重的锅盖滴着冷凝。Jiffy Pop 一直都是对的。

箔黑客兼容任何炉灶爆米花例行；你所要做的就是用一张箔纸代替你常用的锅盖。卷曲边缘将其密封，用一把锋利的小刀刺透，然后加热。很快，你就能吃到非常酥脆、蓬松的爆米花，不会有水滴破坏派对。
