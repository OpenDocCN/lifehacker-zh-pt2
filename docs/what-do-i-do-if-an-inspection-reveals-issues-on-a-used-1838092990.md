# 如果检查发现二手车有问题，我该怎么办？

> 原文：<https://jalopnik.com/what-do-i-do-if-an-inspection-reveals-issues-on-a-used-1838092990>

作为 Jalopnik 的常驻购车专家和专业购车者，我会收到电子邮件。很多邮件。我决定挑几个问题，试着帮帮大家。本周，我们将讨论二手车检查中发现的问题，取消保修，以及经销商需要多长时间找到一辆车。

首先，当购买前检查发现一些问题时，最佳行动方案是什么？

> “我想知道当 PPI 出现问题时会发生什么。如果需要进行小的维修或保养，卖家通常会在购买前处理好，还是会通过谈判降低价格？
> 
> 如果需要更昂贵的维护/修理费用(例如，发动机服务之外),费用可能高达数千美元，该怎么办？"

购买前的检查对于确定二手车的状况非常重要。有时这些检查会发现一些问题点。细节的严重性将决定你的行动方针。如果是像汽车需要新刹车这样的小问题，你可以让卖家在出售前更换这些部件，或者通过谈判降低价格来弥补成本。

如果检查发现了一些重大问题，比如汽车需要重新安装整个引擎，你可能会想走开去找另一辆车。

接下来，电梯套件会自动取消保修吗？

> “我刚买了一辆 2019 款斯巴鲁傲虎，想给它加一个 2 英寸的升降套件。这会自动使我的保修失效吗？”

根据 Magnuson-Moss 保修法案，修改本身不会使保修无效。然而，如果你的车有问题，需要保修，如果问题与你的改装有关，汽车制造商可能会拒绝索赔。所以如果你的导航系统很奇怪，那也不会和你的升降工具有关系。但是如果你的悬挂或刹车出了问题，你可能会陷入困境。

最后，如果经销商说他们可以找到一辆车，那么典型的周转时间是多少？

> “我想买一辆蓝色的 2020 款现代伊兰特。我找到一个经销商，显示有现货，准备购买。我们在停车场转了几分钟，销售员拿到了车牌号，走进去告诉我这辆车已经在前一天卖掉了。我不着急，他们告诉我应该只需要几天就能拿到车。我填写了文件，然后又被告知“一两天后才能找到”。已经五天了，他们告诉我他们还没有找到。我不认为他们在拖后腿，因为他们一直很好，销售和财务都表示，他们很高兴与我合作如此容易，因为我知道我想要什么，并获得了 0%的认可。
> 
> 他们在获得汽车方面遇到问题正常吗？天真的我以为会有更多的汽车装运，我会确认车辆以及一个粗略的时间框架。让我好奇的不是到达的时间，而是定位和确认的时间。"

通常，如果经销商库存中没有您想要的汽车，他们可以在另一家商店找到一辆，要么购买该车，要么以旧换新。在一些罕见的情况下，这可能需要一些时间。

我不认为蓝色现代伊兰特是一辆很难找到的车，尽管也许这种特殊的装饰和颜色组合在你所在的地区并不流行。如果经销商在五天内没有回复你，要么他们忘记了，要么他们找不到你的车。

最好的办法是使用现代的网站或第三方列表网站，如 Autotrader 或 Cars.com，来找到你想要的车。然后打电话给经销商确认供货情况。
