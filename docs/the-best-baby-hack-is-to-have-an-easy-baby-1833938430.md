# 最好的育儿方法是生一个简单的宝宝

> 原文：<https://lifehacker.com/the-best-baby-hack-is-to-have-an-easy-baby-1833938430>

发表在 [学术儿科学](https://www.academicpedsjnl.net/article/S1876-2859(18)30541-2/fulltext) 上的一项新研究证实，挑剔婴儿的母亲比“容易”婴儿的母亲更容易出现抑郁症状。一份 [新闻稿](https://labblog.uofmhealth.org/rounds/mothers-of-fussy-babies-at-higher-risk-of-depressive-symptoms) 称，研究人员从 8200 多名儿童及其父母中提取数据，发现“婴儿越不容易安抚，母亲就越痛苦”。事实上，照顾难相处的婴儿确实需要付出情感代价。

亲爱的父母，如果你在凌晨 3 点读到这封信，因为在你摇晃他、用襁褓包裹他、用越来越具有威胁性的语气给他唱摇篮曲后，你的婴儿仍然不停地哭，你可能会对着屏幕大喊，“谢谢，明显船长”。我明白了。但是也许在科学发现中有一个令人欣慰的信息，那就是 ***不是你*。**

我的第一个孩子很难相处。她讨厌重力，每当我停止让她在健身球上弹跳，她就会尖叫。(想起来了，我现在对健身球的联想不好，一般的锻炼也是。)我花了很多天浏览脸书，感觉所有其他初为人父母的人都在一起——他们在旅行，获得晋升，开始兼职，宣布有更多的孩子。哭泣和睡眠不足，我一直在想:“我怎么了？”

现在，六年过去了，我有了一个新宝宝，他很“随和”(目前为止)。他经常微笑，通过盯着自己的手指自娱自乐，从七周大开始，他每天晚上睡七到九个小时。我已经惊呆了。在过去的几个月里，虽然我们仍然有典型的婴儿挑战，但我已经能够让 T2 计划生日聚会，加入 T4 的读书俱乐部，并开始改造我的浴室。对我来说，这是一次完全不同的育儿经历，然而我对此却无能为力。那孩子朝这边来了。生个容易的孩子。这就是诀窍。

当然，你可以做一些事情来让和婴儿在一起的生活变得更容易管理——例如，制定一个计划[争取支持](https://offspring.lifehacker.com/why-you-need-a-postpartum-party-more-than-a-baby-shower-1827362347?_ga=2.147935287.1035541305.1554683099-1940201396.1552611778)[冷冻一大堆晚餐](https://skillet.lifehacker.com/give-the-new-parents-in-your-life-the-gift-of-food-bec-1793306157)[去户外](https://offspring.lifehacker.com/how-to-get-just-enough-exercise-to-help-stave-off-postp-1797827555) 。但也要知道你可能只是有一个难相处的孩子。虽然这种认识不会对你日夜醒来或剧烈绞痛(我很抱歉)有太大帮助，但它可能会帮助你放弃任何你做得不够，或你不够，或你的生活应该看起来像别人的想法。(注意:不管宝宝的气质如何，如果你正在经历 [产后抑郁症](https://americanpregnancy.org/first-year-of-life/forms-of-postpartum-depression/) 的症状，你应该立即和你的医生谈谈。)

基本上，如果你的宝宝很难，那就怪抽签的运气，而不是你自己。然后坚持住——我会告诉你，即使是最挑剔的婴儿也可以成为一个善良、可爱、最令人愉快的孩子。当这种情况发生时，你应该*肯定*为此得到一些荣誉。
