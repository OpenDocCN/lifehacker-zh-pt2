# 什么是脸书的新“安静模式”,它是如何工作的？

> 原文：<https://lifehacker.com/what-is-facebooks-new-quiet-mode-and-how-does-it-work-1842797471>

脸书正在为其 iOS 和 Android 应用程序推出新的“安静模式”功能，这是一个再好不过的时机了。现在，我们中的很多人比平时更依赖手机——谢谢，隔离！——虽然像脸书这样的应用程序帮助我们保持联系，但有一种东西是*也有*联系。



像任何习惯一样，很难戒掉脸书瘾——即使使用它不再有趣或令人满足。新的安静模式有望让削减变得更容易——也就是说，一旦这个功能向你推出。

### 脸书新的安静模式实际上做了什么

当你启用安静模式时，应用程序的界面隐藏在全屏通知后面，提醒你设置已打开，并显示距离安静模式结束还有多长时间。

很简单，对吧？好吧，如果你感到不安，你可以点击“管理安静模式”按钮，进入你的设置，并随时禁用该功能。不幸的是，你并没有被这个应用程序*拒之门外*,但它确实会迫使你通过几道关卡重新获得访问权限。

安静模式会使大多数(但不是全部)脸书通知静音。据 [脸书的帖子](https://about.fb.com/news/2020/04/coronavirus/) 称，法律要求该公司允许某些通知通过——比如与隐私政策变化或账户安全有关的通知——但可以肯定的是，当安静模式处于活动状态时，新帖子、messenger 回复和其他时间线交互的通知将被压制。

这适用于你的 iOS 或 Android 设备发送的通知和徽章，以及脸书应用本身发送的通知和徽章。静音活动仍然会显示在通知标签中，以防您决定查看您的朋友在做什么；在安静模式下，你不会得到任何提示。

### 如何启用脸书的安静模式(一旦你可以)

您可以在脸书应用程序中打开静音模式:

1.  点击**【汉堡】**菜单标签。
2.  向下滚动并点击**“你在脸书的时光”**
3.  点击**“安静模式”旁边的滑块**

您可以设置每次运行安静模式时保持活动状态的时间。你还可以找到设置自动静音模式定时器的工具，以及让脸书提醒你屏幕时间限制的选项。

虽然安静模式在人们最需要它的时候突然出现，但脸书的推出是出了名的不均衡，安静模式也不例外。如果你幸运的话，它可能已经存在于你的手机应用程序中了，但是我们大多数人可能还得再等一会儿。脸书预计所有 iOS 用户将在未来几周内拥有它，而大多数 Android 用户将不得不等到 5 月份的某个时候。

### 让脸书闭嘴的其他选择

对于那些还没有安静模式，但需要从没完没了的头条新闻和热门话题中立即解脱的人来说，还有其他策略可以减少你的脸书使用和屏幕时间。

脸书的**“静音通知”**选项会像静音模式一样让系统级警报静音，尽管应用内通知仍然会出现；如果你想避免可怕的“叮”声，你需要关闭这个应用程序这也可以在“你在脸书的时光”菜单下打开或关闭。

其他手机设置，包括[翻转到 Shhh](https://www.androidpolice.com/2019/11/27/flip-to-shhh-comes-to-the-pixel-2-with-latest-digital-wellbeing-beta-apk-download/) 甚至 [飞行模式](https://lifehacker.com/use-airplane-mode-as-an-alternative-to-silencing-your-p-5783265) ，也会暂时暂停通知。最后一根稻草是，你可以卸载脸书应用程序，甚至 [删除你在脸书的个人资料](https://lifehacker.com/how-to-delete-your-facebook-account-a-checklist-1829436824) 。稍微休息一下不是坏事，对吧？
