# 你现在可以在 16 英寸的 MacBook Pro 上安装 Windows 10 了

> 原文:[https://life hacker . com/you-can-now-install-windows-10-on-your-16 英寸 macbook-1839961622](https://lifehacker.com/you-can-now-install-windows-10-on-your-16-inch-macbook-1839961622)

我知道，我知道；如果你刚刚花了 2000 多美元买了一台新的 16 英寸 MacBook Pro，这听起来有点神圣，但我们生活在一个小猫和小狗可以成为朋友的时代。如果你想在同一个系统上运行 macOS 和 Windows 10，你应该觉得你可以做到这一点，而不会惹恼苹果之神(或粉丝)。

Watch

至少，是这么想的。然而，如果你最近试图在你最近购买的 MacBook Pro 上安装 Boot Camp，你可能会有点沮丧，因为你无法让它工作。不幸的是，你的 Mac 太新了，驱动程序还不能让 Boot Camp(和 Windows 10)在上面运行。

Apple 已经解决了这个问题，现在您应该可以运行 Boot Camp Assistant 而不会出现任何问题。事实上，你不需要在你的 Mac 上运行系统更新或者类似的东西来开始。打开新兵训练营助理。就是这样。

作为新兵训练营过程的一部分，你需要确保你已经下载了一个。ISO 的 [最新的 Windows 10 版本](https://www.microsoft.com/en-us/software-download/windows10) ，你要确保你的驱动器上至少有 40GB 左右的可用空间用于新的 Windows 分区。

除此之外，在你的 MacBook 上安装 Windows 10 是一个非常简单的过程。当你启动 MacBook 时，不要忘记按住选项 ，这样你就可以快速切换到一个或另一个操作系统(或者通过系统偏好设置将 [设置为你的默认启动盘](https://support.apple.com/en-us/HT202796) 】用于你通常使用的任何操作系统)。
