# 用这个 Alexa 技能学习关于足球的一切

> 原文:[https://life hacker . com/learn-everything-about-football-with this-Alexa-skill-1831495342](https://lifehacker.com/learn-everything-about-football-with-this-alexa-skill-1831495342)

足球是我从未真正喜欢过的一项运动。我是一个典型的朋友，试图召集军队参加棒球和曲棍球比赛，在我的家乡北卡罗来纳州，篮球是一种宗教……但足球总是避开我。我想这不是我的运动。

Watch

这意味着当我和朋友们一起看比赛时，我会变成那个问特定术语是什么意思或者和特定球员有什么交易的人。现在有了一个应用程序。或者说是一种技能。

[NFL](https://www.amazon.com/dp/B07MBDWX9Q?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/learn-everything-about-football-with-this-alexa-skill-1831495342&asc_source=&tag=kinjalifehackerlink-20)菜鸟指南是国家橄榄球联盟创造的 Alexa 技能，帮助像我这样的菜鸟和永久菜鸟学习足球。一旦启用，该技能可以回答一些基本问题，如“一个队有多少球员？”以及“跑卫和外接员有什么区别？”以及更具体的琐事，如“亚伦罗杰斯有多高？”

这项技能包含了关于规则、处罚、得分打法、评论员行话、以前的超级碗、球队阵型和其他基本琐事的信息。这个想法是，如果你在观看比赛时有一个与足球相关的问题，你可以得到一个快速(准确)的答案。

在 NFL 的季后赛期间，skill 还将有一个名为“游戏计划”的每周 5 分钟的播客，为观众准备下一轮季后赛。

它本来是用来看 NFL 比赛的，但它可以在看任何人比赛时派上用场——尤其是如果你是一个不停提问的人。
