# 当技术支持人员就电脑问题打电话给你时，这很可能是一个骗局

> 原文：<https://lifehacker.com/when-tech-support-calls-you-about-a-pc-problem-it-s-pr-1830798500>

现在是技术安全 PSA 时间。《纽约时报》 最近的一篇报道提供了关于虚假技术支持骗局正在演变和扩张的有趣见解。“经典”的例行程序，即在你的电脑屏幕上弹出一个窗口，为你不知道的“病毒”或其他紧急问题提供技术支持，不仅一直在流通——这是一个增长的行业。



所以，对于那些担心弹出窗口、聊天框甚至是你最近接到的电话的人来说，深呼吸，跟我重复。

“放松。这是一个骗局。我的电脑没有问题。除非你先打电话给客户支持，否则他们永远不会打电话来。”

想想吧。微软、苹果——任何可能为你的笔记本电脑或台式机提供技术支持的公司——都有数千万客户。没有一个技术支持团队有时间搜索和检查每台机器上的每一个潜在问题。如果你想得到他们的帮助，你必须通过他们的重重关卡:打电话给技术支持，稍等一会儿，解释说你已经重启了你的电脑 T2 T3。我希望我们生活在这样一个世界里，每当你有问题时，硬件制造商就会出现，但遗憾的是事实并非如此。

如果一个骗子甚至懒得去模仿一个可能为你提供技术支持的公司呢？这本身就是一个危险信号。

如果你收到这些信息，你能做的最好的事情就是避开它。您可以将基于电子邮件的诈骗标记为垃圾邮件，并向您的电子邮件提供商或您雇主的 IT 部门举报。除此之外，最好的策略是删除它们，忽略它们，或者挂断电话，继续前进。
