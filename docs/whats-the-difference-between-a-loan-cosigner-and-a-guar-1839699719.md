# 贷款联署人和担保人有什么区别？

> 原文：<https://lifehacker.com/whats-the-difference-between-a-loan-cosigner-and-a-guar-1839699719>

一些金融术语非常相似，以至于它们最终可以互换使用。但这并不意味着它们是完全一样的东西。一个特别常见的混淆是:共同签名人和担保人的区别，如果你的个人财务记录不够好，他们都可以帮助你获得贷款。

让我们来分解一下:一位 **联署人**同意，如果主要借款人还款有困难，他将对贷款负责。这是典型的人与年轻的信用历史或那些修复他们的信用需要一个共同签署人贷款。例如，一个学生申请大学私人贷款很可能需要一个联合签名人，因为他们没有长期的工作和财务历史。

联署人不同于**共同借款人**。后者用于两个人想要共同拥有的东西，比如一辆车或者一个家， [Bankrate 解释](https://www.bankrate.com/loans/cosigner-vs-co-borrower-difference/) 。如果你有一个连署人，这并不意味着那个人有权在任何时候开你的车。只是意味着你们都有责任为此付出代价。

那么 **担保人**呢？那些在类似的情况下发挥作用，最初的申请人没有足够强大的信用历史来自己获得贷款或公寓租赁。但是担保人更像是一个后备计划，而不是联署人。

“区别在于责任，”福古斯金融集团 的创始人迈克尔·福古斯说。连署人只要在债务上签字，就对债务负责，而债权人不需要采取任何额外的行动。只有在债权人用尽从原借款人处收款的所有其他选择之后，担保人才对债务承担责任。”

听到你需要一个共同签署人来申请贷款可能会感到害怕，但如果有人愿意和你签署，这实际上比你被要求选择担保人更灵活。正如诺洛在 中解释的那样，如果借款人还款有困难，担保人可以暂时介入帮助。然而，当担保人被叫来时，这意味着贷款已经处于不良状况。

如果你需要一个联署人或担保人贷款，你会想选择一个健康的信用档案，以提供你的备份。如果你被要求为某人联署或担保，你既要相信借款人有能力偿还，也要愿意在他们无力偿还时接管还款。你正在帮助的贷款会出现在你的信用报告上，所以你应该鼓励借款人继续还款。
