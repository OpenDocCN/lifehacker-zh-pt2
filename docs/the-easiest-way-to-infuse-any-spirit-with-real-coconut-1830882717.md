# 将真正的椰子味注入任何烈酒的最简单方法

> 原文：<https://lifehacker.com/the-easiest-way-to-infuse-any-spirit-with-real-coconut-1830882717>

如果“椰子朗姆酒”这个词引起了一阵恶心，很可能你对椰子味烈酒的体验仅限于马里布(或其他同样令人作呕的东西)。但是真正的、美味的椰子味是一种享受，它可以被添加到任何烈酒中——不仅仅是朗姆酒——通过用一点椰子油“清洗”酒。



### 怎么能用油洗东西？

“洗脂”是一种在 2000 年代中期到后期突然流行起来的技术，产生了各种各样多肉的鸡尾酒和一大堆培根波旁威士忌。这是一个非常简单的技术——你取一种美味的脂肪(如培根油或椰子油),将其添加到一些烈酒中，让它浸泡，然后冷冻整个东西，这样脂肪就会浮到顶部并凝固。刮掉变硬的脂肪， [多亏了一种被称为“乙醇”的两亲物](https://drinks.seriouseats.com/2014/04/science-fat-washing-spirits-cocktails-how-fatwashing-works.html) (一种既爱脂肪又爱水的分子)，人们得到了一种调味酒。

### 那*口感*呢？

我从来没有在我喜欢的许多脱脂鸡尾酒中注意到浮油的特征，但这个过程确实给你的酒一种“圆”或“软”的特征。如果说有什么不同的话，我发现洗去脂肪会让一种精神更容易享受——是的，我说的是酒——因为它能缓和苛刻，增加乐趣。

### 那么，任何椰子油都可以吗？

*没有*，我很高兴*你*问了这个问题。你必须使用原生的，*而不是*精炼椰子油，因为精炼椰子油有一种中性的味道，这种味道对于烘烤饼干 很好，但是对于给朗姆酒(或杜松子酒、伏特加或威士忌)调味很糟糕。对于 750 毫升的酒精来说，大约 3/4 杯椰子油就足够了。把它融化掉(在炉子上或者在微波炉里)，把它和酒一起倒入一个大罐子里，摇匀。让它静置至少三个小时或一夜，每隔一段时间摇晃一下罐子，然后把它放入冰箱 [直到它被冷冻](https://theconcourse.deadspin.com/rainbow-cake-recipe-inspires-comment-apocalypse-1592575661) 。(我说的“它”指的是石油，不是酒；酒不会结冰。)如果你能很容易地打开油箱盖，那就打开它，否则就在固体油上挖一个洞，通过这个洞把酒倒出来，用一个衬有几层干酪布的筛子过滤。

用你精致的 - but- still- 热带烈酒为你的鸡尾酒(和人生)增添一点阳光，或许与几个朋友分享。我特别喜欢用椰子杜松子酒做的马提尼，尽管我建议用酸橙而不是柠檬做配菜。(当然不是橄榄。)
