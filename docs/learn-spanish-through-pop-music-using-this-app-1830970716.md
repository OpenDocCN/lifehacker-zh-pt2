# 使用此应用程序通过流行音乐学习西班牙语

> 原文：<https://lifehacker.com/learn-spanish-through-pop-music-using-this-app-1830970716>

我上幼儿园的时候，我们班参加了一场引人入胜的演出《 [50 个俏皮的美国](https://www.youtube.com/watch?v=WhDrGnjacvA) 》，一首关于，你猜对了，50 个州的歌。这首歌的一部分(我指的是 75%)需要按照字母顺序唱出所有 50 个州的名字。现在我快 30 岁了，多年来我一直让酒吧的老主顾们高兴，因为我仍然能够毫无停顿地、完美地背诵全部 50 个州。



音乐是学习东西的好方法。没有什么比一首朗朗上口的曲子更能让人记忆犹新。

[Lirica](https://play.google.com/store/apps/details?id=com.taplyrica.lirica) 是一款将这一理念应用于外语学习的应用。该应用程序采用安立奎·伊格莱希亚斯和埃尔维斯·克雷斯波等艺术家的音乐，并使用它来帮助你学习西班牙语词汇和语法。谷歌最近强调该应用程序是 [Google Play 的“2018 年最佳隐藏宝石”之一](http://ttps://play.google.com/store/apps/topic?id=campaign_editorial_bestof2018_best_hidden_gem)

应用程序中的歌曲，分为初级，中级和专家。

在每首歌里，你可以选择你想学的内容。例如，当我选择埃尔维斯·克雷波的“Suavemente”时，我可以选择学习西班牙语的小词、副词、现在时态或代词动词。

在每一课中，你都学习语言的一部分，类似于使用 Duolingo 或其他语言学习应用程序，但使用音乐。每一课都包括一些小游戏，你可以从中学习歌曲中实际所说的内容。

你也可以只听带字幕的音乐，字幕显示的是西班牙语或西班牙语下面有英语翻译。

你可能无法熟练使用这款应用，但这是一种快速学习新短语的有趣方式。
