# 遵循这个通用的理财建议

> 原文：<https://lifehacker.com/follow-this-one-size-fits-all-financial-advice-1832269698>

Lifehacker 的一个基本原则是，并不是每个黑客都适合每个人。在金钱建议上尤其如此，因为每个人的个人情况和目标都会和其他人有很大的不同。例如，当你使用信用卡时，你可能会超支，所以关于最佳奖励卡的建议对你不起作用。



但是有一些经验法则对几乎每个人来说都是有意义的。这些包括:

### 不要累积信用卡债务

这是显而易见的，但它值得重复:如果你用一张卡“支付”一些东西，因为你的银行账户里没有钱，你会让自己陷入债务，最终你会付出更多。同理，你不应该入不敷出，你应该总是按时付账，即使你只能 [支付最低还款额](https://twocents.lifehacker.com/decoding-your-credit-card-bill-1827545220) 。但是那些不用说，呃，写作，对吧？

你可能认为这适用于所有的债务，但事实并非如此。 [“好”债务和坏账](https://lifehacker.com/what-is-good-debt-1823647125) 是有区别的。简而言之:良好的债务，如抵押贷款或汽车贷款，通常会有助于你的信用评分，因为它是由有形的东西担保的，而坏账，如信用卡债务，会损害你的信用评分，因为它不是。

学生贷款债务是一个更棘手的话题——对许多人来说，它可能是好债务，因为它可能有助于提高信用评分，并带来更多机会。但如果它压抑地掩盖了你做出的每一个财务和职业决定，那也可能是件坏事。正如所有的事情一样，在两个极端之间有灰色的阴影和变化。

例外情况:你正处于一场灾难/紧急情况中，除了使用信用卡之外别无选择。

### 为你的 401(k)贡献你的雇主匹配

401(k)账户名声不好，其中一些是有根据的。它们的运营成本可能很高，提供的投资选择也有限，此外，它们也无法替代大多数员工的固定福利计划。不是每个人都有一个或一笔闲钱可以存起来。但是，它们也是提供给全职员工的最有效的金融工具之一。

如果你对你的 401(k)期权和费用不满意，这是一个合理的批评，那么至少为比赛做出贡献，这样你就可以获得全额补偿。正如我之前写的:

> 你应该把那笔钱看作是你报酬的一部分。当你同意雇主付给你 4.9 万美元而不是 5 万美元时，你能接受吗？没有吗？你的 401(k)匹配也是这样。突然间你百分之二的贡献变成了百分之四。不管怎样，他们应该给你钱，但是他们已经把这个要求加在上面了，所以无论如何你都应该这样做。如果你没有为比赛投入足够的资金，你基本上就失去了一部分薪水。

此外，401(k)降低了你的应税收入，给你减税，如果你不动你的钱，你将受益于复利。

例外情况:短期内你需要一些钱，但是一旦你赶上了，就计划增加你的贡献。

### 当你收到账单和财务文件时，打开它们

上周，我登录我的富达账户下载了一些税单。当我这么做的时候，我看到我的旧雇主的账户上的钱已经被完全清空了。余额:0 美元。我一时冲动，然后给一个老同事发了条短信。结果公司换了供应商，如果我打开邮件的话，我会知道的。这个悲伤故事的结局是:打开你的邮件，尤其是如果它来自你的银行或其他与你有业务往来的金融机构。你会省去很多头痛(和震惊)的事情。

例外情况:现在，小偷指望你打开你的邮件，这也意味着你需要提防诈骗。对于学长来说更是如此。一般来说，跳过任何承诺给你一定回报的事情，或者，如果你有学生贷款，声称他们可以让他们消失。

还有什么你想忽略的吗？ [信用卡便利支票](https://www.creditkarma.com/credit-cards/i/credit-card-convenience-check/) 。不要使用它们。

### 首先支付你自己

这是每一篇“如何省钱”的文章中都会提到的技巧之一，但那是因为这是真正省钱的最好方法。在你花费多余的开支之前，先支付自己，从长远来看，你会取得成功，能够每天生活得更舒适，没有压力。

这意味着从你的薪水中自动储蓄，最好是在你收到薪水之前。更好的办法是，让这笔钱更难获得:把它放在一个单独的高收益账户中，或者使用 Digit 这样的应用程序来存钱。

例外情况:你至少存了 1000 美元或相当于一个月开支的钱，你想还清一些债务。

说到无聊...

### 你的财务应该很无聊

你并不真正了解的快速致富计划和复杂的投资产品可能看起来好得不像真的，因为它们确实如此。许多金融服务和产品的存在仅仅是因为它们让其他人赚了很多钱。屡试不爽的最佳产品和策略相对简单明了。 [这篇文章详细介绍了这些策略](https://lifehacker.com/your-finances-should-be-boring-1823831040) 以及它们如何帮助你积累财富。拿不准的时候， [跟着杰克博格尔](https://twocents.lifehacker.com/john-bogles-investing-philosophy-is-still-the-best-1831838039) 。

例外情况:你已经很富有了，你喜欢玩日内交易。开个玩笑，你也别那么做。

### 在你签字(或购买)之前阅读所有内容

虽然这一点似乎也显而易见，但它也适用于许多不太明显的情况，从住房到信用卡再到贷款:

*   问问你的房东，如果你的室友中途搬出去了，或者你失业了，需要提前终止合同，会发生什么。确保你的租约上写清楚了。
*   你没有意识到你的汽车贷款包括延长保修期。
*   一周试用期后，您的健身房会自动续订一年的会员资格。
*   你的发薪日贷款成本远远超过你签约时的理解。
*   当你还在学校的时候，你的学生贷款利息就开始累积了。
*   你使用的信用卡有一个延期利息政策，在一定的月数后生效。
*   例如，在购买房屋时口头同意的事情并没有包含在你的成交文件中。

**异常**:无。

### 要知道过去的表现并不能保证未来的结果

在撰写关于长期投资的文章时，Lifehacker 推荐低成本、多样化的共同基金和 ETF，它们在历史上以比主动管理基金更低的成本获得了更好的回报。但是就像生活中的大多数事情一样，有一个很大的警告:结果是不确定的。没有人能够预测股票市场——或者你的银行、老板、配偶等等。—会的。在研究任何类型的基金、信用卡或银行账户时，记住这一点很重要。

同样，要知道除了死亡和税收，没有什么是可以保证的。如果有人试图向你推销一种“保证，100%发生”回报或结果的产品，说废话，然后走开。

**异常**:无。一切都在变化，没有什么是一成不变的。
