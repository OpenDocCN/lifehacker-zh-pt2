# 如何为 LGBTQ+旅行者使用 Tinder 的新安全功能

> 原文：<https://lifehacker.com/how-to-use-tinders-new-safety-features-for-lgbtq-trave-1836700020>

约会应用对你在 旅行时的 [交往很有帮助，但它们也可能是一个重大的安全风险。这就是为什么 Tinder 推出了一个新的“](https://lifehacker.com/use-dating-apps-while-traveling-to-see-a-new-place-1829466559?_ga=2.179927969.447315495.1563817596-1864099547.1536779123) [旅行者提醒](https://blog.gotinder.com/safety-update-introducing-traveler-alert/) ”功能，该功能是与国际女同性恋、男同性恋、双性恋、跨性别者和双性者协会( [ILGA](https://ilga.org/) )合作开发的，以保护那些将同性行为定为刑事犯罪的国家和地区的 LGBTQ+用户。



国际旅行者要遵守他们所去国家的法律，即使这些法律是不宽容和倒退的。可悲的现实是，许多政府公开压迫 LGBTQ+群体——常常到了暴力的地步。虽然 Tinder 的旅行者提醒是一种令人讨厌的自我审查形式，但它旨在帮助用户保持安全。

### Tinder 的旅行者提醒是如何工作的

如果 Tinder 识别出 LGBTQ+用户位于 69 个迫害同性行为的国家之一，或者如果用户试图通过 Tinder Passport 功能与位于这些国家之一的人进行匹配，Tinder 会提醒他们潜在的安全风险，并让用户选择暂时隐藏他们的个人资料。

如果你仍然选择公开显示你的个人资料，Tinder 会自动在你的个人资料中隐藏你的性别和性取向。一旦你离开这个国家，你的个人资料将会再次公开。

Tinder 还更新了约会安全提示 ，增加了隐私建议和更多旅行者警告信息。正如该公司所描述的:

> “如果你选择与这些国家的陌生人交往，一定要格外小心——因为一些执法机构已经知道使用约会应用程序作为潜在陷阱的工具。一些国家最近还出台了法律，将同性约会应用程序或网站上的个人通信定为刑事犯罪，如果这种通信导致性接触，甚至会加重处罚。
> 
> *访问* [*ILGA 世界*](https://ilga.org/maps-sexual-orientation-laws) *查看各国最新的性取向法律，并考虑捐款支持他们的研究。”*
