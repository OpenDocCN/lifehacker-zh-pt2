# 我在食谱中使用哪种盐有关系吗？

> 原文:[https://the take out . com/which-type-of-salt-to-use-kosher-table-sea-1839063480](https://thetakeout.com/which-types-of-salt-to-use-kosher-table-sea-1839063480)

一些食谱在配料表中指定了犹太盐、罐装盐或海盐。但是，当你一边搅拌热锅，一边试图在手机上阅读食谱，并在厨房里与朋友进行半连贯的对话时，你很容易抓起最近的一瓶食盐。你可能会想，它仍然是盐，我用什么类型真的重要吗？

根据专家的说法，是也不是。

烘焙需要精确，这就是为什么你会想要严格遵循食谱的盐建议，根据凯特琳·彭泽姆格、 [*外卖*撰稿人](https://thetakeout.com/salt-grinders-are-bullshit-and-other-lessons-from-grow-1798259994) 和 [作者](https://thetakeout.com/importance-of-salt-on-spice-book-penzeymoog-1831585194) *关于香料:建议、智慧和历史与一粒盐*。她说盐的质地各不相同——想想粗盐和细盐相比有多少——这意味着一汤匙的粗盐和一汤匙的细盐可能不一样。称量配料有助于缓解这种差异，但是如果食谱给出了它的体积测量，如果可能的话，遵循指定的盐。如果烘焙食谱没有指定盐的种类，你只能用你喜欢的种类。

对于其他烹饪应用，如 [烤蔬菜](https://thetakeout.com/glazed-roasted-vegetables-recipe-1838712443) 或烹饪时调味，犹太盐通常应该是你的首选。它更大的表面积意味着它可以更均匀地融化在食物上，并且比摇动器更容易用手指分配。

芝加哥 [肯德尔学院](https://www.kendall.edu/) 烹饪学校的厨师教练杰克·坎贝尔告诉*外卖*“如果你的手指稍微潮湿，即使食盐中有防结块剂，它也会粘在你的手指上或结块，食物的某些地方会有盐块。“因为犹太盐较粗，你可以在牛排上看到犹太盐，所以你可以看到某些区域是否过多或过少。”

坎贝尔解释说，某些品牌的食盐中的抗结块剂也会在清汤等精致的菜肴中产生轻微的混浊，一些味觉非常敏锐的厨师实际上可以检测出食盐中添加的碘。他说，正因为如此，犹太食用盐作为默认的烹饪用盐更受欢迎；PenzeyMoog 同意。

“就我个人而言，我 95%的烘焙和烹饪食物都使用犹太盐，”她说。“犹太盐的质地非常好，你可以用手指捏一捏。当你在餐桌上使用时，你实际上需要的量更少，因为它分布得非常均匀。”

PenzyMoog 将食用盐的质地比作一小块冰——“它有点反弹，碰到哪里就融化，融化的效果不是很好”——而犹太盐就像雪花，一路融化，更均匀地覆盖在你的食物上。

在餐桌上或上菜前给最后一道菜调味怎么样？这时你就可以拿出你藏起来的粉色喜马拉雅盐、黑色盐、烟熏盐、调味盐、巨大的马尔东水晶等等。调味盐有助于调味，所以当你尝过所有现有的味道平衡后，最好把它们留着给最后一道菜调味。像或 fleur de sel 这样又脆又宽的盐也是如此，PenzyMoog 说，这些盐更有可能放在你的食物上而不融化，给每一口鞑靼牛肉或带黄油的花式面包增加松脆的质地和盐分。她说，海盐是一个模糊的术语，没有任何规定或精确的定义，所以它可以通过传统方式从盐水中开采或干燥，或者通过加速的工厂流程来制造。作为消费者，几乎没有办法知道，所以如果你真的喜欢它的质地或味道，就只花高价购买。

人们很容易被诱惑去挥霍那些拥有美丽的晶体结构、宝石般的色泽和有趣味道的美食盐。如果你喜欢他们让你的食物尝起来的味道，那就去做吧。但是在那些日子里，当你在厨房里跑来跑去，在电话铃响、烤箱哔哔作响、狗叫着要你注意的时候，努力把晚餐端上桌——不要担心你的盐选择。

“所有盐的 95%是氯化钠，不到 5%的差异是其他矿物质或藻类的存在，”PenzeyMoog 说。

底线:对于大多数烹饪应用，使用犹太盐，如果你有它。但如果没有，手头有什么总比没有盐好。
