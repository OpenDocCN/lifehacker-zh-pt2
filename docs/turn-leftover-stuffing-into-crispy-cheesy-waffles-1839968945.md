# 把剩下的馅料变成松脆的奶酪华夫饼

> 原文：<https://lifehacker.com/turn-leftover-stuffing-into-crispy-cheesy-waffles-1839968945>

感恩节剩菜本身就是一种食谱类型，尽管我喜欢三明治，但准备一些其他稍微有点创新的剩菜策略也无妨。如果你想把剩菜当早餐吃，这一点尤其正确。



填充华夫饼只需要一些剩余的填料和华夫饼机，但我的*填充华夫饼也需要培根(你可以用小块)和奶酪。(切达干酪在华夫饼的边缘变脆，这非常令人愉快。)在上面浇上 [香喷喷的鲜奶油](https://skillet.lifehacker.com/the-case-for-savory-whipped-cream-1837069612) ，这种鲜奶油已经注入了你身边的任何香草，你几乎认不出昨天的湿面包了。*
