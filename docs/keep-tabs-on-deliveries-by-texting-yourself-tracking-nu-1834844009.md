# 通过在 iOS 中给自己发送跟踪号码的短信来跟踪送货情况

> 原文：<https://lifehacker.com/keep-tabs-on-deliveries-by-texting-yourself-tracking-nu-1834844009>

**iOS:** 这是一个小技巧——一个*的小技巧*，如果你愿意的话——它已经在 iOS 上可用多年了，但如果你不想注册应用或服务来跟踪收到的包裹，这是值得提醒的。(如果你知道一些特别的东西正在路上，并且你想要一个简单的方法来跟踪送货司机，这也是适用的。)



当您收到来自 UPS、USPS 或 OnTrac(我今天测试了这三种服务)的包裹时，请复制包裹追踪号。打开邮件，用那个号码给自己发一条短信。这将它转换成一个可点击的链接，这使得从任何承运商那里检查所述包裹的状态变得容易得多。就这么简单。

我想，你也可以在笔记中做同样的复制/粘贴过程，但这比你发给自己的文本更容易忘记。然后，你必须进去删除这条短信，而不是在收到新信息时，让你的文本从你的视线中消失，从你的脑海中消失。

至少，这是我的论点。就像我说的，我注册了各种各样的“ [让我知道它正在路上](https://lifehacker.com/how-to-track-incoming-packages-like-a-pro-for-free-1829681982) ”服务，还有大量的 [其他应用](https://itunes.apple.com/us/app/17track/id1004956012?mt=8) 你可以通过一个界面来跟踪收到的包裹。但是，如果你想密切关注一批货，这个简单的方法是不会错的。

当然，有一点要注意。我用一个联邦快递跟踪号码(事实上有几个)尝试了这个小技巧，信息认为它们是电话号码(可能是因为它们的长度)。令人失望的是，没有办法告诉 iOS 你想要像跟踪包裹一样跟踪一串数字，而不是一个联系人。如果你和联邦快递的司机直呼其名，这个小把戏可能不适合你。
