# 收集迪士尼角色签名的最佳方式

> 原文：<https://lifehacker.com/the-best-ways-to-collect-disney-character-autographs-1836597609>

我和我的大家庭去过几次迪士尼世界，包括我的儿子和三个侄女。在他们四到七岁的时候，获得角色签名是一件非常重要的事情。我们做了那个年龄组的所有其他父母似乎都在做的事情:我们买了一本 [迪士尼签名簿](https://www.shopdisney.com/world-of-disney-autograph-book-with-pen-1489664?pagePath=%2Fsearch&pageType=SearchPage&algorithm=filter%3Afalse%2Csort%3AD-gross_margin&position=3%2C5) 他们手里拿着签名簿，有条不紊地穿过公园。

Watch

虽然这很有趣，但当我现在回想起来，我想我本可以更有创意地收集孩子们的签名和他们最喜欢的角色的照片。

### 让他们在空白贺卡上签名

如果你想把迪士尼的魔力延伸到你的旅行之外，一位母亲有了一个天才的想法，她带了一叠卡片让角色们签名。她上周在 [的一个坏妈妈播客](https://www.maximumfun.org/one-bad-mother/one-bad-mother-episode-312-still-traveling-kids-plus-princess-hair-heather-burris) 中描述了这一新的育儿举措:

> “我们刚刚去了迪斯尼世界进行家庭旅行，在我们到达那里之前，我买了一叠空白卡片——有些是生日卡片，但有些只是空白卡片。当我们离开的时候，我让角色们在卡片的底部签名。因此，一年中断断续续地，或者每当我想起或者为其他孩子的生日聚会，我们可以只写一张纸条，并让它由奥罗拉或米老鼠或其中一个冷冻人物签名。然后我的孩子们可以全年都有特别的生日贺卡，我真的很期待我们的迪士尼世界之旅。"

### 带一本“年鉴”给他们签名

我们的后代脸书育儿小组的一名成员建议带一本有角色照片的书，就像这本动画角色初级百科全书，这样角色就可以在他们的个人页面上签上自己的名字。

“我在网上的其他地方读到过这种想法，许多角色非常了解这本书，甚至会在他们故事中的对手角色的页面上写一些愚蠢的东西，”小组成员埃里卡说。

### 让他们在不是书的东西上签名

签名书和年鉴是经典的选择，但角色不仅限于签名书。取而代之的是，你可以带一件物品给所有的角色签名，这样你的孩子以后可以穿、用或展示，比如一件迪士尼 t 恤、一个枕头套或一张海报。

### 让他们与喜爱的玩具互动

和跳跳虎或巴斯光年或奇普和戴尔站在一起摆姿势很可爱；拍下这些角色很傻的照片，与你的孩子*和*互动，他们最喜欢的填充动物或玩具甚至更可爱。

在你开始拍照之前，让你的孩子挑选一个小玩具交给角色。你最终会得到一套甜蜜的、主题鲜明的照片，这些照片展示了对你的孩子有意义的东西，并展示了人物的个性。
