# 征服你庞大的“稍后阅读”清单的最佳工具

> 原文：<https://lifehacker.com/the-best-tools-for-conquering-your-massive-read-later-l-1838097178>

“以后再看”清单是一个重要的生产力工具。将文章发送到一个单独的应用程序供以后阅读，可以让你在工作时集中注意力，并让你在停机时可以在手机上做一些有成效的事情。但是保存一篇文章比阅读它更容易，所以“稍后阅读”列表上的故事(就像应用程序 [、Instapaper](https://www.instapaper.com/) 和 [Pocket](https://app.getpocket.com) )往往会堆积起来。这里有三种不同的策略来完成你的阅读清单，以及执行它们的工具。



### 读取队列

如果你不得不在你的阅读清单应用程序中搜寻你真正想要阅读的项目，你的清单中有太多杂乱的东西。是时候要么读那些你每次都跳过的文章，要么就让它们过去。

如果你有良好的自制力，你可以像浏览一个被忽视的收件箱一样浏览你的阅读清单，并把所有不是立即感兴趣的东西存档。无情地浏览一遍——如果你后悔存档了一篇文章，你随时可以在以后挖掘它。(更有可能的是，你会永远忘记这件事，然后*你会没事的*。)

如果你没有那个自制力，你需要一个类似 [阅读队列](https://readingqueue.app) 的 app，一次只会给你看一个项目。在应用程序删除某个项目之前，您只能跳过该项目三次。这样你会花更多的时间阅读，而不是花更少的时间寻找可以阅读的东西。

### 从邮件

如果问题是你从来不打开你的阅读列表应用，那么试着把文章发送到你的电子邮箱。

如果问题是你从来不打开你的阅读列表应用，那么试着把文章发送到你的电子邮箱。Chrome 和 Firefox 的 [Mailist](https://mailist.app) 扩展功能类似于 Instapaper 或 Pocket，但它不是将文章保存到应用程序中，而是将它们编译成专为你准备的电子邮件简讯。

如果你有相反的问题——收件箱里堆满了你一直想看的简讯——那么 [使用这些指令](https://lifehacker.com/how-to-send-email-newsletters-to-instapaper-1836602754) 将这些简讯发送到 Instapaper。

像这样的技巧可能不会永远奏效，因为你的大脑开始忽略阅读列表电子邮件，就像它忽略阅读列表应用程序一样。但是如果它们能工作几个星期，它们会帮你在你的清单上留下一个凹痕。(我非常相信“目前有效的工具”)

### 加速器

Instapaper 可以强迫你用“速读”工具快速阅读你的文章。查看文章时，点击底部菜单右端的分享按钮。在灰色共享选项行中，点击速度计图标。

口袋没有速读。但它支持 iOS 的第三方速读 app， [加速器](https://apps.apple.com/us/app/accelerator-speed-reader/id675410630?ign-mpt=uo%3D4) 。

还有更奇怪的选择来完成你的阅读清单。Pocket 和 Instapaper 都有朗读功能，所以你可以像听机器人播客一样听 Siri 风格的自动语音朗读文章。你甚至可以采用我们的帖子中的一些技巧，和朋友或伙伴一起大声朗读文章。我们个人不会走这么远，但对某些人来说是行得通的。

进入“超高效”模式，浏览、存档和删除不需要的文章是很有趣的。但是一次通常不足以永远清除你的阅读清单。选择一天中有规律地在手机中穿梭的时间——在厕所里，在火车上，在排队时——每次都让自己处于这种状态。(如有必要，设置每日提醒。)剩下的时间，你可以高枕无忧，看一整篇文章。但在那几分钟里，你处于终结者模式:寻找并摧毁。
