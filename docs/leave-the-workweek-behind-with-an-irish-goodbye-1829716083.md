# 用一句爱尔兰再见来告别工作周

> 原文：<https://lifehacker.com/leave-the-workweek-behind-with-an-irish-goodbye-1829716083>

各位周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。今天我们要调制一杯自信而美味的鸡尾酒，来帮助我们忘记一周工作中的烦恼。



作为一个不太喜欢测量酒精含量的人，我倾向于要求三种成分等量的鸡尾酒。由克莱德·科姆的杰弗里·摩根塔勒创作的《爱尔兰再见》包含了爱尔兰威士忌、白百合和西纳尔。这三者并没有争夺主导权，而是和谐地创造出一种有点水果味、有点木头味、相当僵硬的鸡尾酒。要自己做一个，你需要:

*   1 盎司功率爱尔兰威士忌(或你最喜欢的品牌)
*   1 盎司辛纳
*   1 盎司白百合

将所有原料倒入加冰的混合杯，搅拌至完全冷却。滤入一个双座玻璃杯。
