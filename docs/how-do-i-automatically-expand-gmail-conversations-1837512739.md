# 如何自动扩展 Gmail 对话？

> 原文：<https://lifehacker.com/how-do-i-automatically-expand-gmail-conversations-1837512739>

在 Lifehacker 本周的 [技术支持专栏](https://lifehacker.com/c/tech-911) 中，我们正在解决 Gmail 中一个相对简单的问题——但如果你对你的电子邮件的外观非常挑剔，这个问题可能会非常烦人。(讲究一点也可以！)

虽然这个问题是一位生活黑客读者*从技术上来说是*提出来的，但我欠你的，读者们，我必须承认，这个问题实际上是执行主编 [弗吉尼亚·史密斯的](https://kinja.com/vksmith) 妈妈提出来的。从技术上来说，她把问题发给了弗吉尼亚，而不是我，但我非常喜欢这个问题，认为它值得讨论。史密斯女士写道:

> *“请就如何处理那些烦人的 gmail 字符串做个黑客吧。他们让人抓狂。”*

经过进一步调查，她澄清说:

> 他们不显示一些回复，而是重复另一些，把最后一封邮件的答案放在字符串的底部，等等。通常你甚至不能得到最新的消息。非常令人抓狂。”

我认为弗吉尼亚的妈妈最终的意思是，她希望有更多的选项来定制 Gmail，如果有的话。好消息是:它们确实存在，无论你是在修改 Gmail 本身的设置，还是使用第三方扩展来拼凑你最喜欢使用的布局。

为了具体解决史密斯女士的担忧，一次查看一个线程中所有电子邮件的一个简单选择是禁用 Gmail 的对话视图。为此，请在浏览器中打开 Gmail，然后单击设置图标。点击设置(单词)，向下滚动你的眼睛，直到你看到“对话视图”选项。

关掉它，你收到的每一封邮件都会在你的收件箱里单独出现。点击一个线程中最近的电子邮件将显示整个线程的内容(假设人们已经在自己的回复中保留了回复历史)，并且它将按照时间倒序排列——最近的邮件在顶部，随着你在链中向下移动，越来越旧。

如果你不喜欢这种方法，还有其他选择。当您打开电子邮件对话时，您可以随时单击邮件右上角的向上箭头/向下箭头按钮来手动展开所有邮件。它变得乏味，但它是可选的。

我认为一个更好的技巧是通过 Gmail 的设置菜单启用 [键盘快捷键](https://lifehacker.com/the-19-most-important-gmail-keyboard-shortcuts-1836311313) 。然后，按下分号(；)会展开一个线程中的所有消息，按下冒号，或者 shift +分号，应该会折叠所有消息。这不如自动扩展对话中所有信息的一些功能或扩展流畅——仍在努力寻找——但在你的键盘上敲一个键就很接近了*。*

### 如何将新的 Gmail 邮件从底部移到顶部

至于你的问题的整个“在帖子底部回答”方面，我想你可能会更高兴，如果你能反转 Gmail 对话的提要，把最近的邮件放在顶部，旧邮件放在较低的位置。幸运的是，有一个 Chrome 扩展可以帮你交换这个功能: [Gmail 反向对话](https://chrome.google.com/webstore/detail/gmail-reverse-conversatio/kfgepjmmgamniaefbjlbacahkjjnjoaa?hl=en) 。(对 [火狐](https://addons.mozilla.org/en-US/firefox/addon/gmail-reverse/?src=search) 也可用。)CloudHQ 也有 [Gmail 对话线程反转](https://chrome.google.com/webstore/detail/gmail-conversation-thread/fkplphcelnmocakmefjmpnohnfgkibkk) 扩展，增加了一个按钮，你可以反转(或还原)对话的消息。

这应该包括你在给弗吉尼亚的信中提到的一切，史密斯女士。然而，还有很多其他方法可以根据你的喜好修改 Gmail。(我很享受 [简化 Gmail](https://lifehacker.com/remove-gmails-clutter-with-the-chrome-extension-simplif-1834309296) 扩展，比如。)

像往常一样，在使用浏览器扩展之前，在 Gmail 的设置或界面上摸索一下，看看你是否能激活一种新的方式来与你的邮件互动。否则，先阅读一个扩展的评论，看看它是否受欢迎，并运行一个快速的网络搜索，以确保您不会将收件箱的钥匙交给一个不道德的扩展作者。你怎么小心都不为过！
