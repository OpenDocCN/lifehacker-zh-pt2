# 最新的 Android 9 安全更新包括对 Pixel 手机有用的功能补丁

> 原文：<https://lifehacker.com/the-latest-android-9-security-update-includes-helpful-f-1830862271>

周二， [谷歌为即将到来的 12 月、安卓安全更新发布了补丁说明](https://source.android.com/security/bulletin/pixel/2018-12-01) ，该更新将于今日开始向所有安卓设备推出。这些每月更新通常会修复运行 Android 9 的设备的潜在漏洞，但本月的更新除了正常的安全更新外，还包括 Pixel 手机的特定补丁。这些补丁包括提高相机性能，修复屏幕闪烁错误和应用程序在后台运行时意外关闭等几个问题。



虽然大多数补丁是针对 Pixel 3 设备的，但也有一些适用于以前的 Pixel 型号，因此您会希望在更新可用时尽快应用更新以利用。以下是 12 月份 Android 9 安全更新中 Pixel 手机的所有变化:

**改进的内存性能和后台应用关闭修复**

一些 Pixel 2 和 3 用户发现他们的手机会过早关闭后台运行的应用程序，但据报道，这一新的更新解决了这一问题。谷歌将这一变化称为所有 Pixel 2、2XL、3 和 3XL 手机“在特定情况下提高了内存性能”，这将更新这些手机上的 RAM 管理，专门防止不必要的应用程序关闭。

**环境显示闪烁修正**

12 月安全更新的另一个主要功能补丁旨在防止 Pixel 3 设备的环境锁定屏幕上出现的屏幕闪烁，谷歌称之为“改进的总是显示触发”。

**更好的相机性能**

Google 正在为 Pixel 2 和 Pixel 3 兜售改进的相机捕捉性能，这也是 RAM 管理修复的一部分。Pixel 3 用户也可以期待在他们安装更新后重新装备自动聚焦行为和改进的快门性能。

**媒体和蓝牙改进**

在整个 Pixel 手机系列中，各种媒体应用和功能都有一些变化。首先，谷歌调整了连接蓝牙设备时的音量行为，这适用于所有 Pixel 手机。此外，通过蓝牙连接时，原始 Pixel 或 Pixel 2 系列机型的所有者应该会看到更快的解锁响应。

至于 Pixel 3 特定的媒体更新，谷歌表示，用户可以期待在使用 Pixel 3 的媒体应用程序时“改善 HDR 的轮廓”，以及通过 USB-C 端口插入音频设备时更好的检测。

**像素点更新**

为 Google Home 挑选了 Pixel 支架配件的 Pixel 3 用户会很高兴听到谷歌在你的手机与支架对接时“提高了通知可视性”(尽管《邮报》不清楚这些改进具体是什么)。您还应该注意到更好的热门词汇识别和性能。

**安卓汽车性能**

最后，12 月份安全更新中的最后两个补丁适用于 Android Auto 功能，这些功能扩展并提高了对 Pixel 2 和 Pixel 3 设备的兼容性，并对一些车辆在使用 Android Auto 时的 Pixel 3 音频性能进行了全面改进。

### 如何下载 12 月的 Android 9 安全更新

如上所述，更新和所有适用的功能补丁将于 12/5 今天开始在所有 Android 设备上推出。当系统更新可用时，您会收到系统更新通知，此时您需要做的就是运行更新。

然而，谷歌也为其所有更新发布了[【OTA】](https://developers.google.com/android/ota)和 [工厂图像](https://developers.google.com/android/images) ，12 月的更新目前对那些喜欢手动将操作系统更新闪存到他们的设备的人开放。
