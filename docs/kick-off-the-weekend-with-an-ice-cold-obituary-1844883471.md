# 以冰冷的讣告开始周末

> 原文：<https://lifehacker.com/kick-off-the-weekend-with-an-ice-cold-obituary-1844883471>

茴香是一种黑暗、危险、虚无的味道。也许是因为我习惯了在当地的哥特酒吧喝苦艾酒，也可能是因为直到 2007 年(我 21 岁那年)苦艾酒在美国还是非法的，但这种茴香酒有一个明显黯淡的名字并不令我惊讶。(不能只有我一个人有这种感觉；苦艾酒和香槟酒的组合被称为“下午的死亡”！)



讣告鸡尾酒是一种马提尼变体，通常含有一点苦艾酒调味。我只说“通常”是因为我也见过用 pastis 做的，这是我今天用的，因为这是我用过的。(我最初买了一瓶给 [莫骨灰盒](https://lifehacker.com/add-a-little-pastis-to-your-vesper-in-honor-of-melissa-1843483318) 启程我们亲爱的前主编梅丽莎·基尔希，从那以后几乎没怎么喝过。)

就像本专栏中的大多数东西一样，它很容易制作，但是你真的需要搅拌它。如果没有适当的冷却和稀释，讣告读起来就像是含有大量酒精，所以至少搅拌一分钟。要制作这款闷闷不乐的马提尼，你需要:

*   2 盎司杜松子酒
*   半盎司苦艾酒或茴香酒
*   半盎司干苦艾酒

将所有原料加入装满碎冰的混合杯中，搅拌一分钟。过滤成一杯(最好是冰镇的)鸡尾酒。如果你需要装饰，可以用一条柠檬皮装饰，如果你感到年轻和情绪化，可以用丁香装饰。
