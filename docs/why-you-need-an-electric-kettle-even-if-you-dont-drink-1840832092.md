# 为什么不喝茶也需要电水壶

> 原文：<https://lifehacker.com/why-you-need-an-electric-kettle-even-if-you-dont-drink-1840832092>

烧水是一项基本的厨房技能，任何人都能够掌握；你只需要一个锅，一些水和一个炉子。不一定是需要被黑的流程，但是是可以更快的流程。



如果你曾经拜访过英国的朋友或家人，你可能已经注意到在那个国家几乎每个人都有一个电水壶。我一直认为这是因为他们需要茶，而且他们需要尽快喝完，但是这个设备有多种非常有用的非茶用途。

### 如果你是真空

大多数时候，将一个小水浴加热到 145 华氏度左右是我的浸没循环器能够胜任的任务，但更大的项目——比如加热足够的水以将整个猪肩肉浸没在 185 度的水浴中——对我来说可能需要太长时间。在浴缸中加入一两壶开水，可以在几分钟内迅速将水温升至目标温度。

### 如果你现在需要意大利面

把一大锅意大利面水烧开所花的时间总是比我想象的要长，我觉得这很烦人。如果你的炉子需要太长时间来满足你的面食消费需求，拿起你的水壶，通过向几乎但不完全沸腾的水中加入沸水来加速这个过程，有效地加倍这项任务。(显然，你可以用水壶烧开任何大量的水，即使它不是用来煮意大利面的。)

### 如果你是一个咖啡势利者

它可能被称为茶水壶，但是爱喝咖啡的人也可以喝一杯，作为一种享受。大多数傲慢的咖啡方法——比如法式压榨或倒出来——需要的水刚刚低于沸点，电水壶比炉子更快到达，这在早晨非常重要，因为每一分钟都很重要。(另外，从水壶里倒比从平底锅里倒要容易得多。)简单地将水烧开，让它平静一会儿，然后像往常一样制作你的花式咖啡饮料。(额外收获:大多数水壶都很小，可以放在你工作的桌子上，这样就不需要喝不新鲜的自助咖啡了。)

### 如果你真的喜欢方便面

有时候(当你喝醉的时候)你需要一份拉面，而且你现在就需要。众所周知，我用微波炉加热我的拉面水，但不是每个人都有这样的柜台空间。一个小电水壶显然可以在这里有所帮助。

电水壶你最喜欢的用途是什么(除了泡茶)？我当地的酒吧专门为热棕榈酒准备了一个，这可能是对这种设备的最高要求。
