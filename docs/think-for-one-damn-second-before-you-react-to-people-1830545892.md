# 在你对别人做出反应之前，先好好想想

> 原文:[https://life hacker . com/think-for-one-damn-second before-you-react-to-people-1830545892](https://lifehacker.com/think-for-one-damn-second-before-you-react-to-people-1830545892)

内森·w·派尔是纽约著名的礼仪卡通片《T2》的幕后策划者，他有一个建议给你们所有人，如果你们能闭上该死的嘴一秒钟的话。实际上，这就是秘诀:当有人说了什么，现在轮到你说话，你准备好情绪化地回应并升级对话时，也许闭上你的大嘴一小会儿。你能做到吗？这里是内森的多部分 Instagram 帖子，如果你能费心点击并关注别人一次 :

内森的建议， [最初发表在 BuzzFeed](https://www.buzzfeed.com/nathanwpyle/this-is-why-i-pause-before-i-speak) 上，刚刚在 Instagram 上疯传，因为成千上万的人停止了“我我我”的行为，只有一点点时间，并意识到“哦，嘿，也许我应该接受这个。”这恰好是这篇技巧的第二部分。

当内森在谈话中停顿时，他会思考自己的参照系和他人的参照系，以及这些参照系如何可能对任何感知到的分歧负责。然后，他会思考如何通过和谐的讨论来调整这两个参照系，而不是立即产生分歧。

这是一个简单的建议，但它是一个很好的提醒，因为我们都可以不那么快争论，多一点相互理解，而不是那么急于在彼此的脸上，对不对？正确你抓到那个家伙了吗？你在听吗？
