# 从酒吧拿杯垫

> 原文：<https://lifehacker.com/take-coasters-from-bars-1831839564>

杯垫是一件聪明的东西，但却是一件愚蠢的东西。生活中最大的讽刺之一是，最昂贵的杯垫通常是由坚硬、厚重的材料制成的，而不是吸收水分，让水分聚集，直到无法控制，溢出到你的表面。解决办法很明显:不要买杯垫；相反，T2 从酒吧里偷了它们。

Watch

“偷”这个词有点夸张。首先，这些纸圆圈或方块很快就会变脏，这意味着它们在使用一两次后就会被扔掉。酒吧通常期望——甚至希望——你带着它们，因为它们几乎总是有酒吧的名字在上面，所以当你把它们带出去的时候，你基本上是在帮助它们做广告。(他们绝对不会在乎某个啤酒或白酒品牌投放的促销杯垫。)如果你还在纠结从酒吧拿杯垫的道德问题，那就直接问酒保你能不能拿一个。他们不会说“不”

除了免费之外，从酒吧拿纸杯垫还有很多好处。它们比昂贵的硬材料杯垫(如哑瓷砖)吸水性更强，它们又薄又轻，而且容易存放，它们让你看起来像一个去过许多酷又有趣的地方的酷又有趣的人。它们是我最喜欢收集的纪念品之一，它们让我想起了我在全国各地各种精致场所度过的所有美好时光。(哦，它们是免费的。我说过它们是免费的吗？)
