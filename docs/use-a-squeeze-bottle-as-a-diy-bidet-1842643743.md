# 使用挤压瓶作为 DIY 坐浴盆

> 原文：<https://lifehacker.com/use-a-squeeze-bottle-as-a-diy-bidet-1842643743>

我不知道你怎么样，但是我的手纸快用完了。我的柜子里只剩下两卷了，我当地的杂货店除了空架子什么都没有。两周前，当我最后一次亲自去杂货店时，没有任何卫生纸——本周，当我在网上订购时，我可以选择在某一类别中订购任何可用的商品，卫生纸类别中没有任何商品。



这意味着是时候发挥创造力了。我的马桶没有坐浴盆附件，但我熟悉一般概念，并且在过去专门使用过水(当我在研究生院时，我在印度的海得拉巴大学花了一个学期教莎士比亚)，所以我知道我真正需要的是一个好方法，将适度有力的水流对准我的下体。

进入自行车瓶。顶部有喷嘴的那种湿软的，所以你骑的时候可以直接把水流挤到嘴里。事实证明，你也可以将水流对准身体的其他部位——这真的很有效。你可能需要做一些弯曲和移动来使你自己和瓶子处于正确的角度，但是你可以自己解决这个问题。

然后，做你在使用标准坐浴盆时可能会做的事情:把所有正确的地方都弄干净，坐在马桶上直到不再滴水，如果你真的想的话，用一点卫生纸把自己拍干(我没有这样做，试图为真正的 TP 紧急情况节省卫生纸供应)，然后继续你的一天。在某些情况下，你甚至不需要冲厕所，但是你需要洗手。每一次。

如果你没有自行车瓶，你可能会有另一种类型的 squeezy 瓶可以工作——同样，我相信你会自己找出这一部分。回答另一个问题，我知道你会暗自焦虑:如果你有那种不能用一股水流清理干净的如厕经历，你可以随时去冲个澡。(我们中的一些人无论如何都会这样做，在那些特殊的厕所经历之后，甚至连卫生纸都不能解决。)

不，我可能再也不会用这个自行车瓶子喝水了，即使在卫生纸供应恢复正常之后。但至少它现在很好地服务于我，如果不是完全为了它的预期目的。
