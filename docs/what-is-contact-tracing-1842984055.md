# 什么是联系追踪？

> 原文：<https://lifehacker.com/what-is-contact-tracing-1842984055>

接触者追踪是遏制疾病爆发的关键工具。对新冠肺炎来说，这也是世界卫生组织放松距离限制的指导方针等计划的关键部分。那么什么是接触追踪，它是如何工作的？

Watch

### 追踪接触者可以发现感染疾病风险增加的人

如果我们知道一个人生病了，很可能他们把疾病传染给了其他人。接触追踪是寻找这些人的经典流行病学工具。这很简单，也是典型的低技术含量:你采访一些人，找出被感染者接触过的人。

《传染病》,一部最近异常火爆的电影，《T2》[有几个场景](https://www.youtube.com/watch?v=Jt5Kl9j7D54) ,展示了一个戏剧化版本的联系追踪。凯特·温斯莱特扮演的角色询问一位晚期病人的同事和搭档，在她死前的某段时间里，她和谁有过接触。
