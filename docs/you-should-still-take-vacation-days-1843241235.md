# 你仍然应该休假

> 原文：<https://lifehacker.com/you-should-still-take-vacation-days-1843241235>

如果你在政府关闭期间一直在家工作，那么过去的八周可能会比平时多一倍的忙碌和压力。毕竟，你面临着一项艰巨的任务:在学习新工具的同时保持工作效率，找出如何与远程同事沟通，试图平衡工作效率和养育子女(更不用说在家上学了)，以及在日常任务(如购买杂货)比平时复杂得多的情况下维持家庭运转——再加上在全球疫情期间与生活相关的所有焦虑。

这意味着是时候休假了。

正如肯纳迪·西尔考克斯在会议上解释的那样，如果你的工作是那种你有能力申请一个长周末的工作，那么这是值得的——即使你不会离开家。

> 想一个你会去度的常规假期。当你在沙滩上舒展身体或在泳池边阅读时，目的是为了放松。在家休息一天可能不会有同样的感觉，但是如果你过去几周一直在加班，留出一些时间在电视前放松一下也不是一个坏主意。

如果你不喜欢看电视，那就以温暖的天气为借口，在户外呆一整天，探索公园、小径或任何安全距离的地方(既远离你的家，也远离其他人)。

西尔考克斯还建议，不要把你的 PTO 申请框定为休假本身，而是更多的心理健康日。如果你的老板是那种能理解你最想要的是没有计划或义务的一天的人，那种你(和你家里的其他人)可以让你的头脑和身体都得到休息的一天，你可能会在周五或周一休假。记住，即使是从日常事务和工作中稍微休息一下，也有助于防止精疲力尽。

如果你现在不能休假，不管是为了度假还是心理健康，看看你是否能把即将到来的周末定为“无计划日”把它当成你需要的假期/休整日:一个休息、阅读、狂看电视节目、出去享受阳光、和孩子玩游戏或做任何你“如果我有更多时间”清单上的事情的时间。

然后继续为下个月安排一个没有计划的日子——因为从现在开始，你可能也需要休息几周。
