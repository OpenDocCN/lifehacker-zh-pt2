# 问那个咸咸的女服务员:恶，我的服务员告诉我她生病了

> 原文：<https://thetakeout.com/my-server-told-me-she-s-sick-1834726573>

> 亲爱的咸:我们最近参加了一个晚上的毕业典礼。我们在演讲开始前就吃了东西，但在听完没完没了的演讲后，我们需要吃点点心。去当地的小餐馆，那里很拥挤——毕业典礼上的每个人都有同样的想法。很好，我们不着急，我们等 45 分钟然后就坐。
> 
> 服务员说的第一句话是“请原谅。我今天真的病了，但是我不能请一晚上的假，因为我需要钱。”
> 
> 我们的行动是什么？我们已经投入了 45 分钟，并且知道在那么晚的时间里等待其他所有的开放将是残酷的——我们将回到队伍的末尾。我们要求移到另一张桌子吗？太挤了，谁知道要花多长时间？我们向经理投诉吗？如果我们把服务器送回家(她需要钱)或者更糟(暂停)呢？被解雇了？).
> 
> 我们不情愿地坚持了下来。没有人真的我真的想在那个时候找到别的地方。我留了下来，吃得很少，从那以后一直用维生素 C 轰炸自己。手指交叉。
> 
> 在这种情况下，你建议我们应该怎么做？当服务员需要钱的时候，他们是如何处理感冒的？她当初为什么要告诉我们？(最后这个是反问句，但我还是问了)。

> 感谢，票房告罄

亲爱的恶心死了，

带病上班对这个行业的许多人来说是一个糟糕的现实——疾病控制中心说 在过去的一年中，我们中有 20%的人在工作时呕吐或腹泻，呃——但是没有必要传播它。当然，你的服务员可能患有不具传染性的疾病，比如偏头痛、背痛或 UTI，她试图解释为什么她看起来心不在焉。但是，仍然没有理由说自己生病了，让你的桌子认为他们会得流感。



我敢肯定服务器不是开着铃来工作的。餐馆让员工几乎不可能打电话请病假，通常是因为我们没有病假，或者因为我们在最后一刻需要休假而被责骂。我被告知要找人代班，否则就等着被扫地出门吧。你可能会想:*但是健康代码*！实际上，生病的工人来上班并不一定违法。FDA 的 [食品代码](https://blogs.cdc.gov/yourhealthyourenvironment/2016/12/05/food-workers-working-while-sick/) ，这是州和地方食品代码的基础，*建议* 有呕吐和腹泻症状的工人不要工作。但那只是常识。一些县有健康文件，这是餐馆员工培训的一部分，员工可能会被要求签署一份协议，承诺在生病时告诉他们的经理。如果他们不这样做，他们可能会面临后果。当然，这些是否被强制执行…

那么，当你发现伤寒玛丽是你的服务员时，你能做什么呢？如果餐馆不是很忙，我会说在你点菜之前，你应该让经理换一下桌子。如果你害怕给服务器带来麻烦，就说是因为你想坐在窗边，或者你所在的地方太冷了。但是在一个繁忙的餐馆里，你必须权衡你是想等在另一个服务站坐下，还是冒着感染凯撒沙拉旁边的细菌的风险。

我个人会要求我的食物带走。这不是一个完美的解决方案，但也许食物可以直接装在厨房的外带盒里，这样可以减少服务员接触你的盘子和用她的细菌口气和你说话的时间。我同情她的原因，因为我也经历过——感觉很糟糕，但生活在对我的经理的恐惧中——但当然，想到你可能会让别人生病也很糟糕。

你问生病的工人是如何处理的，答案和你作为顾客所做的一样:多洗手，抱最好的希望。
