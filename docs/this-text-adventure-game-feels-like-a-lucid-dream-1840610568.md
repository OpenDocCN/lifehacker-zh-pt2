# 这个文本冒险游戏感觉像一个清醒的梦

> 原文：<https://lifehacker.com/this-text-adventure-game-feels-like-a-lucid-dream-1840610568>

人工智能不仅仅是反乌托邦算法；这也有利于创造性的娱乐，就像这个人工智能为一堆不存在的小说 写开头的句子。现在，一个人工智能随时准备引导你通过一个古老的学校文本冒险游戏，一个在你开始尝试玩它之前并不存在的游戏。

Watch

人工智能知道这些故事会如何发展。如果你输入“开门”，它要么会告诉你门是锁着的，要么会打开门并描述里面的房间。如果你试着和客栈老板说话，它会告诉你对话是怎么进行的。

当你玩 [AI 地下城 2](https://www.aidungeon.io/2019/12/aidungeon2-is-here.html) 时，你处于玩家的角色中，但你也拥有作为地下城主的所有创造力。想跳上猫头鹰飞走，抄近路去山上的城堡吗？我做到了。就像清醒梦一样。无论你采取的行动多么有创意或出人意料，故事都会给你一个“是的，而且”

最棒的是，它可以在你的手机上运行(最初的人工智能地牢是一个基于网络的游戏，需要花很长时间才能加载)，所以你可以一次玩一点，只要你有空。在 [iOS](https://apps.apple.com/us/app/ai-dungeon/id1491268416) 和 [Android](https://play.google.com/store/apps/details?id=com.aidungeon) 上是免费的。
