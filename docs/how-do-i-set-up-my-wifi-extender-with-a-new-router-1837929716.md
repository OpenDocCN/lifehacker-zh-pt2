# 如何使用新路由器设置我的 Wifi 扩展器？

> 原文:[https://life hacker . com/how-do-I-set-up-my-wifi-extender-with-a-new-router-1837929716](https://lifehacker.com/how-do-i-set-up-my-wifi-extender-with-a-new-router-1837929716)

在本周 Lifehacker 的 [技术建议专栏](https://lifehacker.com/c/tech-911) 中，我们将回到我们最喜欢的话题:无线网络。这一次，读者在让一个重要的旧设备与一个重要的新设备一起工作时遇到了一些问题。如果你曾经升级过你的路由器，或者打算升级，这种情况应该再熟悉不过了。

Lifehacker 的读者阿什利写道:

> “我看到了你的一篇关于 safari 的文章，可能需要一些建议。所以我阿姨和我住在同一个地方，相距不到 200 英尺。她在过去的一年里一直有 WiFi，我的扩展器总能接收到信号，没问题。直到一周前她买了新路由器。所以我的问题是，如何重新连接扩展器？如果可能的话，我想在有人指导下自己做这件事。提前感谢！”

我将从好消息开始:这个问题应该很容易解决，除非出现任何不可预见的情况。首先，我将假设您的新 wifi 路由器相当不错，并且从您设置了无线扩展器的地方接收信号没有任何问题。

让您的扩展器重新连接到您的 wifi 路由器的最快方法是让您的阿姨在她的新路由器上使用与她在旧路由器上使用的完全相同的 SSID 和密码。扩展器不知道任何更好的；它应该可以毫无问题地连接到无线网络。

然而，有一个小警告可能会搞乱这个过程。如果您之前将扩展器设置为连接到路由器的 2.4GHz 网络，而您的阿姨(无论出于何种原因)只在她的路由器上使用 5GHz，则此配对将不起作用。(更有可能的情况是，您的阿姨*仅*在她的路由器上使用 2.4GHz，而您之前已经将扩展器设置为连接到路由器现在不存在的 5GHz 网络。)

您可以随时调出您的无线扩展器的设置，进行任何您需要的更改以使连接再次工作——如果您的阿姨重命名了 wifi 网络，请连接到新的 SSID，或者调整您的扩展器用来连接到您的路由器的无线频带 的 [。我建议在网上搜索扩展器的型号，并找出制造商的说明，因为无论您在网络浏览器中输入什么来访问其设置，都可能会有所不同。](https://lifehacker.com/which-wifi-band-should-i-use-for-my-devices-1832635625)

(顺便说一下，我建议不要使用扩展器的“双重连接”选项，如果它有一个的话——或者任何类似名称的选项。为了获得最佳性能，请仅将扩展器连接到路由器的 2.4GHz *或* 5GHz 网络。当你创建从你的扩展器射出的*第二个* wifi 网络时，如果适用的话，将其隔离到相反的频段。例如，如果您连接到 5GHz 的路由器，您的设备应该只连接到 2.4GHz 的扩展器。)

如果你对扩展器的主设置屏幕有点困惑，或者你只是不想对它大惊小怪，尝试一下核心方法:工厂重置扩展器。通常，您可以通过按住其重置按钮(假设有)5-10 秒钟，直到扩展器上的灯开始不同程度地闪烁，来完成此操作。不过，为了确保你正确地完成了这个过程，请查看你的设备手册。

完成后，您可以按照制造商对扩展器的初始设置说明进行操作，这应该可以很容易地将其连接到您阿姨的新路由器上——毕竟，您第一次就让扩展器正常工作了。

如果您仍然遇到连接问题，请尝试将扩展器移近您阿姨的路由器(或让她将路由器移近扩展器)，看看是否有帮助。并确保您已将扩展器和路由器更新到最新的固件版本。您应该能够在每个设备的基于 web 的设置屏幕中检查路由器和扩展器使用的固件。然后，您可以查看设备制造商的支持网站，了解它提供的最新固件是否是您的设备正在使用的固件。如果没有，更新一下！
