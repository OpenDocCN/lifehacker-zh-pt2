# 如何判断是否可以相信最新的冠状病毒头条

> 原文：<https://lifehacker.com/how-to-tell-if-you-can-trust-the-latest-coronavirus-hea-1844609639>

每天都有这么多关于新冠肺炎·疫情的文章，人们很容易被大量的信息淹没。鉴于此，现在比以往任何时候都更难找出哪些是真正提供高质量报告、值得关注的。那么，我们如何知道该读什么，该相信谁呢？

[华盛顿大学生物学教授卡尔·博格斯特伦](https://www.callingbullshit.org/) 博士(以及升级版 上的 [前嘉宾)，近日](https://lifehacker.com/how-to-detect-bullshit-1803144313) [与《科学美国人》](https://www.scientificamerican.com/article/how-to-evaluate-covid-19-news-without-freaking-out1/) 就如何消费和评价新冠肺炎新闻报道而不“抓狂”(他们的原话)。作为识别和辨别谎言的专家，博格斯特伦分享了一些技巧，帮助你找出哪些 COVID 新闻值得阅读和相信。这里有一些我们最喜欢的。

### 等 12-18 个小时去读一篇关于“突发新闻”的文章

在一些新闻故事中，速度和即时报道是有意义的，比如公众人物的死亡或重大的全球事件。但这种即时“突发新闻”报道并不总是最佳选择——例如，如果它涉及与正在进行的全球疫情有关的医学或科学细节。博格斯特伦建议我们放慢脚步:

> “高质量的信息与几分钟到几小时的及时性没有太大关系。这与这些信息被审查、三角分析和呈现的程度有很大关系。我鼓励人们在这样的危机中做的是慢下来[阅读]12 小时前——或 18 小时前或 36 小时前——由一名多年来一直报道传染病的专业记者撰写的报纸故事，他与一群专家交谈，综合正在发生的事情，解释事情，并将其置于背景中。
> 
> 我鼓励人们转向他们信任的传统媒体来源，而不是转向 Twitter 或脸书或 WhatsApp，因为当你这样做时，你确实会获得更近一点的信息，但这些信息的质量要低得多。你很容易受到互联网上传播的任何谣言的影响，这可能是一个大问题。"

### 寻找并关注有健康和/或传染病报道经验的特定记者

不管在疫情之前记者的节奏是什么，几乎所有的事情都转向了 COVID 报道。这里的问题是，不是每个人都有处理这类报告的背景和训练。这一点，加上对速度的需求，意味着许多 COVID 文章可能包含不准确的信息或分析。

博格斯特伦说，对他来说，找到最好的新闻来源归结为分析个别记者。他阅读的名单中包括 STAT 的海伦·布兰斯维尔 ，因为，他说，她有 20 多年的传染病报道经验。他 [告诉《科学美国人》](https://www.scientificamerican.com/article/how-to-evaluate-covid-19-news-without-freaking-out1/) “她理解整幅画，并出色地展示了它。”。“我认为这是一个找到那些你信任的声音，然后依靠那些声音的问题。”

### 改变观点或建议是一件好事

报道疫情的主要挑战是不断发展的科学和不断变化的公共卫生信息。有一天，我们被告知不要戴口罩，几个星期后，他们是强制性的——诸如此类的事情。一些媒体消费者可能会认为，如果某个出版物或记者改变了他们对某件事的立场或建议，就意味着他们的报道不准确。但是，正如博格斯特伦指出的，事实恰恰相反:

> “首先要认识到，由于科学在变化，你从健康专家那里得到的建议也会随着时间而变化。你会看到有人说，‘你不能相信(美国国家过敏和传染病研究所主任安东尼)福奇，因为他在 2 月份说的是一回事，而在 7 月份又说了另一回事。’这完全是倒退。你不能信任的人是那些没有改变观点和建议的人，尽管他们有更多的证据。那些基于证据改变观点和建议的人，是做科学的人，是给出好建议的人。"
