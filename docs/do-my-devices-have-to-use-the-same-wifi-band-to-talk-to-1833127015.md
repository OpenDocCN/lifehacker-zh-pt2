# 我的设备必须使用相同的 Wifi 频段才能相互通话吗？

每个人都有关于无线网络的问题，这完全没关系。关于 wifi，你可能还有很多不知道的，即使你已经成功设置了路由器，现在正坐在那里，在你的笔记本电脑、台式机、智能手机或平板电脑上浏览 Lifehacker。



在这个 [**Tech 911**](https://lifehacker.com/c/tech-911) 建议专栏中，我之前已经讨论过一些更大的问题，如果你想在家里获得最快的连接，你需要了解这些问题，包括你应该使用哪个 [无线频段](https://lifehacker.com/which-wifi-band-should-i-use-for-my-devices-1832635625) 来获得最佳连接，以及你的 wifi 网络应该使用哪个 [无线信道](https://lifehacker.com/which-wifi-channels-should-i-use-for-my-wireless-networ-1832788063) 。这些都是相当复杂的话题，但这并不意味着我们应该忽视基础。

最近，Lifehacker 读者 **Daron** 给我发来了以下问题:

> “我的路由器旁边有一台无线打印机。2.4 还是 5？我一直认为笔记本电脑和打印机必须在同一个频道上才能正常工作。”

如果我是你，我会把我的无线打印机调到 2.4GHz，但如果你真的想的话，你也可以调到 5GHz。我不知道你用的是什么打印机，但是我要猜测所有的无线打印机都可以连接 2.4GHz 的 wifi 网络，大概有一些*不能*连接 5GHz 的网络。老实说，这是选择一个而不是另一个的唯一原因。

由于你的打印机离你的路由器如此之近，任何一个频段都应该足够快以供你的打印机使用。(我想你不会发送大量消耗大量带宽的打印任务；如果你是，或者如果你有一个无线 - 交流打印机，并想使用它的最大潜力，你会想去 5GHz。)

至于你的第二个评论，那是一个我很乐意消除的无线神话。无论您使用的是 2.4GHz 还是 5GHz，您连接的设备都应该能够互相看到，这没有问题。有*有*一些情况下它们可能不会，但这通常不会出现在默认的路由器设置中。例如，如果您正在运行客人网络，连接到该网络的设备可能看不到连接到您的主网络的任何东西(默认情况下，或者由于您在设置客人网络时切换的设置)。

你的路由器也可能带有某种“ [接入点隔离](https://routerguide.net/complete-ap-isolate-guide-how-to-improve-your-wireless-network/) ”功能——类似的交易。如果您启用了该功能，但您可能不应该启用，那么连接的设备应该无法看到网络上的任何其他设备。他们将能够访问互联网，但仅此而已。

否则，连接到 2.4GHz 或 5GHz 的决定是一个范围和速度的问题。虽然这不是一个有保证的规则，但我发现 2.4GHz 可以让你比 5GHz 的范围更大一点，尽管速度更慢，但如果你在两者的范围内，请选择 5GHz 以获得最快的连接。如果你的设备很简单，不需要那么高的速度，就把它们放在 2.4GHz 上，让你的 5GHz 频段免费提供给需要带宽的设备。
