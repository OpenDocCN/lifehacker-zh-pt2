# 用烧焦的洋葱做更好更快的洋葱蘸酱

> 原文：<https://lifehacker.com/make-a-better-faster-onion-dip-with-burnt-onions-1843998650>

洋葱蘸酱可能是我最喜欢的蘸酱，但是将洋葱焦糖化既烦人又无聊。因为我已经有点烦了，讨厌无聊，所以我现在非常喜欢烤焦洋葱的生活方式，并且在做洋葱蘸酱的时候用烤焦的洋葱代替焦糖洋葱。

Watch

请不要误解:我确实喜欢焦糖洋葱的味道，但我不想做一种必须同时被呵护和忽视的东西。搅拌太多，它们需要一个多小时才能焦糖化；搅拌的次数太少，它们会烧焦(仍然需要超过 45 分钟才能完全焦糖化)。但是通过接受不耐烦和故意烧焦这些小家伙，你可以在很短的时间内做出非常美味的洋葱蘸酱。

老实说，一旦你把焦糖洋葱和大量酸奶油和/或蛋黄酱混合在一起，它们的许多细微差别就会消失。然而，烧焦的洋葱的浓郁、烘烤、美味、略带苦味的味道有点难以混淆，不管你用多少脂肪去涂抹它。(这就是为什么他们在 [牧场调料](https://skillet.lifehacker.com/transform-any-creamy-dressing-with-burnt-onion-1837851760) 中工作得非常好。)烧洋葱蘸酱不需要特别的食谱；任何需要焦糖洋葱的食谱(比如这个)都可以。只需使用相同数量的洋葱，用中高火在少许油中烹饪，直到它们看起来像这样:

让洋葱冷却，把它们加入酸奶油中，用伍斯特郡酱、味精和一些洋葱和大蒜粉调味。如果你想要更顺滑的蘸酱体验，你可以把一半的酱加入酸奶油中。一旦所有东西都混合在一起，在冰箱里冷藏几个小时，让蘸料凝固，味道融合。配上大量的薯片。
