# 这是我们见过的最聪明的缩放黑客

> 原文：<https://lifehacker.com/this-is-the-smartest-zoom-hack-weve-seen-yet-1843418059>

让一个无聊的青少年想出一个我见过的最巧妙的方法来在缩放聊天中“在场”——特别是如果聊天是在较大的一边，并且有人没有太注意你在网格视图中的网络摄像头。



既然对大多数人来说，我们已经处于隔离期的第*周，我觉得我们大多数人都宁愿做其他事情，也不愿再经历一次视频会议。我知道它们的重要性，真的，但有时你只是需要休息一下。或者也许外面看起来美极了，而你的身体要求你去散步，而不是坐在笔记本电脑前进行一天中第四个小时的演示。*

不管是什么原因，从这本四年级的书里借一页(感谢 [Reddit](https://www.reddit.com/r/funny/comments/ghujvl/today_one_of_my_4th_grade_students_renamed/) )。如果你想错过一个 Zoom 会议，或者至少给人留下这样的印象:由于*的连接问题，你无法参加*，做两件事。首先，将手指放在 **ALT + V** 和 **ALT + A** 键上，分别关闭网络摄像头和音频。(Mac 上的 **Command + Shift + V** 和 **Command + Shift + A** )。)

然后，右键单击自己并选择“重命名”，如果它可用的话。一旦你准备好“离开”会议，把你的名字换成“重新连接…”但是现在还不要单击“确定”。点击 **ALT + V** 然后 **ALT + A** 来删除你的视频和音频，然后点击 OK 来更改你的名字。幸运的话，这一切看起来会非常无缝——你的麦克风和网络摄像头突然切断，你现在正在艰难地努力“重新连接”。

这是要忽悠大家吗？绝对不行。当你非常确信主持 Zoom 会议的人以及参与会议的人不太了解这项技术时，我会保留这个技巧。(如果你真的有连接问题，你会留在缩放会议中，但你的网络摄像头视频会被及时冻结。就不会有任何“联系……”或者“重新连接…”在你冰冷的脸上贴上纸条。)

最终，你断开的自我会退出会议。所以，如果有人知道这些细节，你的“重新连接……”诡计可能不起作用。你总是可以来个组合拳:放弃“重新连接…”恶搞，等几分钟，然后完全离开会议。后来被问到的时候，就说你当时在纠结你的 wifi 什么的。

我的意思是，这是一个放弃会议的相当大的努力，但它至少比制作自己*观看*假会议 的循环视频 [需要更少的努力。我喜欢它的欢乐，但你必须把它做好，才能让人相信。如果有人问你问题怎么办？](https://lifehacker.com/how-to-create-a-looping-video-of-yourself-that-attends-1842843207)
