# 去餐馆前采取这些预防措施

> 原文：<https://lifehacker.com/take-these-precautions-before-going-to-a-restaurant-1843565306>

仅仅因为餐馆再次开门 [并不意味着去](https://lifehacker.com/just-because-you-can-go-out-doesnt-mean-you-should-go-1843523899) 就安全了。关于在疫情期间开餐馆是一种新的情况，正如在新冠肺炎的所有事情中，我们边走边学。在你预订晚餐之前，这里有一些你需要考虑的风险因素和预防措施。



### 评估你自己的个人风险

如果你是高风险人群或者和高风险人群住在一起，那么每次外出都要格外小心，特别是在非必要的旅行中，比如去餐馆吃饭，在那里你可能会和不在你家的其他人长时间接触。I 如果你对在餐馆吃饭的风险感到不安，无论是对你还是对你家里的人说—[说不](https://lifehacker.com/how-to-say-no-to-socializing-1843532583) 都是完全可以接受的。

你也要考虑你居住的地区。如果最近有新病例上升，或者你居住的地区有很多病例，你就需要特别小心。

评估整体风险也很重要——不仅是对你自己，也对餐厅里的其他人(T2)来说。如果你认识生病的人，或者如果你的工作让你有更高的患病风险，为了保护你周围的人，你可能要避免外出就餐。

### 考虑替代方案

我明白了:去餐馆给你一个机会去吃你没有做过的食物并且和朋友在一个不是你家的地方共度时光。考虑到过去几个月的艰难，难怪人们想出去吃饭。

但是在你做出保留之前，值得考虑的是你是否能以一种风险更小的方式实现同样的目标。例如，你可以在你最喜欢的餐馆叫外卖，在 [和朋友在公园](https://vitals.lifehacker.com/how-safe-is-it-to-socialize-outdoors-1843339830) 见面，在那里你将受益于置身于一个不是你家的环境中，并且吃你不必烹饪的食物，但也将在一个有更多空间的地方保持适当的物理距离。

如果放松是目标，那么尽可能减少焦虑的来源是值得的。如果你能找到不用担心生病而享受美食的方法，你会享受更多。

### 出发前收集信息

一些餐馆会比其他餐馆有更好的防范措施。在你预订酒店之前，你会想知道他们是如何处理重新开业的。C 查看他们的网站或社交媒体账户，或者干脆打电话询问。如果他们似乎还没有考虑清楚— 或者你对他们的预防措施感到不舒服— 那么你就要重新考虑你的选择了。

美国疾病预防控制中心发布了针对即将重新开业的餐厅的指导方针，这是一个有用的起点，可以用来衡量你最喜欢的餐厅是否符合标准。餐厅应该采取的预防措施包括为高风险员工提供额外保护，额外的卫生和消毒程序，确保安全布局，允许顾客练习物理距离以及持续监控员工健康，包括计划在员工生病时该做什么。

### 如果一家餐馆看起来很拥挤，不要进去

如果你到了一家餐馆，它看起来拥挤到没有人能保持六英尺或更远的距离，不要进去。桌子离得够远吗？餐车挤得太近了吗？人都戴口罩了吗？工作人员最起码都要戴口罩，用餐者不吃饭的时候戴口罩也是个不错的主意。

### 非常注意你接触的任何表面

一个环境中的人越多，人们接触桌面、玻璃器皿和门把手等表面的可能性就越大。注意你接触到的任何表面并带上洗手液以防你不得不接触高接触表面，这很重要。

### 考虑在外面吃饭

如果可以的话，最好是在户外用餐。 在室内吃饭意味着和其他人一样呼吸同样的空气——而 [证据表明病毒在封闭空间的空气中逗留](https://quillette.com/2020/04/23/covid-19-superspreader-events-in-28-countries-critical-patterns-and-lessons/) ，即使你坐在离别人六英尺远的地方——而室外提供循环的新鲜空气，这可以降低你的风险。

### 不要呆太久

传播的风险在很多方面是一个概率游戏。你与其他人在一个封闭的区域呆得越久，你就越有可能感染。在餐馆呆一个小时比在那里呆三个小时风险要小。如果你要去餐馆吃饭，限制你在那里呆的时间会有所帮助。如果在用餐结束时，你仍然不想回家，去附近的公园(只要不太拥挤)而不是在餐桌前逗留。

正如我们之前所讨论的，风险并不是一个非全有的场景，而是 [存在于](https://lifehacker.com/think-of-risk-as-a-spectrum-1843543882) 的光谱上。在你做任何决定之前，重要的是要了解什么是相对风险，以及你需要采取的预防措施，以尽可能保证自己的安全。
