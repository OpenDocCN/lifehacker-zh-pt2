# 你现在可以购买一个家庭医疗检查包

> 原文：<https://lifehacker.com/you-can-now-buy-an-at-home-medical-exam-kit-1839858327>

与医生的视频访问是一种不出家门就能获得医学意见(有时是处方)的廉价而快捷的方式。但是医生不能对你进行身体检查，这就限制了它们的用处。一套价值 300 美元的数字医疗设备似乎可以改变这种情况。

Watch

该套件来自 TytoCare，在百思买门店和 [在线](https://www.bestbuy.com/site/tytocare-tytohome-medical-exam-kit-white/6332714.p?skuId=6332714) 销售。它包括一个带有摄像头和传感器的设备，包括一个温度计和三个附件:一个听心肺音的数字听诊器，一个看耳朵的耳镜，一个让摄像头看到喉咙后部的压舌板。

该公司表示，他们的医生可以诊断“耳朵感染、喉咙痛、发烧、感冒和流感、过敏、红眼病、恶心、便秘、哮喘、支气管炎、上呼吸道感染、蚊虫叮咬和常见的皮肤疾病，包括接触性皮炎、皮疹和尿布疹。”常规的远程医疗服务也可以做到这些，但是这个工具包可以让医生更好地看看，比如说，你的喉咙痛实际上是什么样子。

一个应用程序会带你完成使用设备的过程，并与 Tyto 连接进行视频访问。每次出诊 60 美元，类似于其他远程医疗服务。(许多保险计划包含视频访问。)Tyto 指出，你可以用 FSA 的钱购买这款设备。
