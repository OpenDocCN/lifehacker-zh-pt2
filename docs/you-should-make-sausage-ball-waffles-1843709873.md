# 你应该做香肠球华夫饼

> 原文:[https://life hacker . com/you-should-make-香肠球华夫饼-1843709873](https://lifehacker.com/you-should-make-sausage-ball-waffles-1843709873)

隔离期间我一直在做两件事:做香肠和(不太成功地)重新整理我的厨房。前者已经成为一种困扰——我几乎完善了我的早餐香肠——但后者把我带到了一个被遗忘的收藏库—,里面有亮蓝色的快餐盒，包括但不限于饼干混合物。显然，一切都井井有条——我需要做香肠球。

问任何一个来自美国南部的人现在是不是香肠- 球季，他们很可能会告诉你现在是*而不是*香肠- 球季。香肠、烘焙混合物和碎奶酪的混合物通常被烘焙成嫩的小块，作为假日聚会的小吃或圣诞节早晨的点心。再加上天气真的开始变暖了，这意味着大多数人并不热衷于打开烤箱吃咸咸的、奶酪味的、多肉的小吃。

当然，这就是华夫饼熨斗的用武之地。现在可能不是香肠- 球季，但却是香肠- 球*华夫饼*的*时代*。当用华夫饼干烙铁(它不会像烤箱一样加热你的家)压制时，肉、奶酪和饼干混合物会变平，成为一块分割的松脆华夫饼干，可以作为鸡蛋或枫糖浆的输送系统。在我拍了你上面看到的照片后，我立刻把半熟的鸡蛋打碎在两个华夫饼干之间，加了一片奶酪，享受了一顿特别的早餐三明治。我想在本尼迪克特的情况下也会有用。

这些华夫饼很容易做，但是味道好的香肠会给你最好的结果。我用了这个 [Allrecipes 食谱](https://www.allrecipes.com/recipe/21649/sausage-balls/) 的比例，可以放大也可以缩小，但是下面会祝福你四个华夫饼。要制作它们，您需要:

*   1/2 杯饼干烘焙混合物，如 Jiffy 或 Bisquick
*   1/4 磅猪肉香肠
*   1/4 磅切达干酪丝

将华夫饼烤模加热至中等温度。当它变暖的时候，把所有的材料放入碗中，混合并捣碎，直到所有的材料都粘在一起，变成肉和奶酪。将混合物分成四等份，并把它们做成 1/2 英寸厚的圆盘状。按照生产商的说明，用华夫饼干熨斗将它们烤熟，直到它们变脆，外表变成棕色。在上面放上鸡蛋，浸在糖浆里，或者站在厨房里直接用手吃。
