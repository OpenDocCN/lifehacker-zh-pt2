# 戴夫·皮尔奇正在主持虚拟阅读和汲取教训

> 原文：<https://lifehacker.com/dav-pilkey-is-hosting-virtual-readings-and-drawing-less-1842661151>

戴夫·皮尔奇——裤衩船长和狗人儿童系列图书的作者和插图画家——和我们大多数人一样，被困在家里。因此，他与国会图书馆 的 [团队合作，为我们的孩子朗读，并帮助他们学习画一些他最喜爱的角色。](https://loc.gov/engage)



从今天开始，皮尔基发布了各种每周一次的短视频，在视频中，他给迷你上绘画课，朗读他的一本书中的一章(配有出色的角色声音和音效)，或者讲述他如何创作出一个角色的故事([)里尔·皮蒂是基于他的妈妈](https://www.youtube.com/watch?v=KP8GhJ0rQ94&list=PLpAGnumt6iV6fcun8XGk41DqOMZLs-qT2&index=2&t=0s) ！).

在这里他正在读的一章[狗人放出](https://www.youtube.com/watch?v=B9oCTSDe6Cg&list=PLpAGnumt6iV6fcun8XGk41DqOMZLs-qT2&index=3)的:

 [https://lifehacker.com/embed/inset/iframe?id=youtube-list-B9oCTSDe6Cg%2FPLpAGnumt6iV6fcun8XGk41DqOMZLs-qT2&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-list-B9oCTSDe6Cg%2FPLpAGnumt6iV6fcun8XGk41DqOMZLs-qT2&start=0) 

皮尔基是我们其他一些最受欢迎的作家、插图画家和名人中的最新一位，他为我们的孩子打破了单调的家庭生活。他们也可以 [和莫一起涂鸦](https://offspring.lifehacker.com/doodle-with-childrens-book-author-and-illustrator-mo-wi-1842376859) 或加入或为常规故事时间。

可以在 YouTube 上观看皮尔基的 [全集；国会图书馆将在每周五上午放入新的录像。Pilkey 还在 Scholastic 的“](https://www.youtube.com/playlist?list=PLpAGnumt6iV6fcun8XGk41DqOMZLs-qT2) [【戴夫·皮尔奇在家里】](https://kids.scholastic.com/kids/books/dav-pilkey-at-home/) ”网站上为孩子们提供了额外的内容，包括教孩子们如何画狗人的可下载副本、书籍封面着色页和其他活动。
