# 如何尽早尝试 Firefox 73 的新缩放功能

> 原文:[https://life hacker . com/how-to-try-Firefox-73s-new-zoom-features-early-1840663384](https://lifehacker.com/how-to-try-firefox-73s-new-zoom-features-early-1840663384)

可定制的缩放级别是火狐最好的辅助功能之一。用户可以根据每个网站调整放大尺寸，每次访问网站时，浏览器都会记住并应用首选的缩放级别。对于有视觉障碍的用户来说，这使得导航和阅读页面变得更加容易，但是对于您访问的每个新页面来说，不得不调整这一点是很烦人的。此外，放大太多还会扭曲网站上的图像和其他图形元素，破坏缩放功能的好处。幸运的是，Firefox 将很快添加两个新的设置来修复这些缺点。

第一个是通用缩放设置，允许你设置你在 Firefox 中访问的所有网站的默认放大倍数，另一个是纯文本缩放选项，只会放大文本，并尽可能保持图像和其他视觉效果不变(尽管如果你放大太多，页面布局肯定会有一些变化)。德国火狐开发者 [【索伦·亨茨舍尔](https://www.soeren-hentzschel.at/soeren-hentzschel/)(via[gHacks](https://www.ghacks.net/2019/12/26/firefox-73-gets-an-ui-option-to-set-a-global-zoom-level/))在最新的火狐夜间测试版本中发现了这两种设置。

你可以通过 [下载 Firefox Nightly](https://www.mozilla.org/en-US/firefox/channel/desktop/) build 来尝试这两种设置，这是 Firefox beta 版本中最前沿也是最不稳定的。只有当你不介意潜在的错误或崩溃时才使用它。那些喜欢使用稳定的公开发布版本的人将在 2020 年 2 月 11 日 Firefox 73 发布时获得该功能(这比看起来要快得多)。

### 如何尽早测试 Firefox 的新缩放设置

以下是如何在 Firefox 中找到新的设置。我假设这个过程在 Firefox 73 的最终版本中也是一样的，但是可能会有一些细微的差别。

1.  [每夜下载安装火狐](https://www.mozilla.org/en-US/firefox/channel/desktop/) ，然后打开一个新的浏览器窗口。
2.  进入**菜单>设置>常规。**(或使用地址栏进入**关于:首选项**)。
3.  在设置菜单中，向下滚动到缩放子标题，然后单击**“默认缩放”**旁边的下拉框，选择您的首选级别(可以是 30%到 300%)。
4.  如果您只想将这些更改应用到网站上显示的文本，请选中“仅缩放文本”旁边的框
5.  这些更改将应用于您在 Firefox 中访问的所有网站和浏览器设置页面。如果需要，你仍然可以使用缩放热键来进一步调整网站的放大倍数，Firefox 会记住这个变化。
