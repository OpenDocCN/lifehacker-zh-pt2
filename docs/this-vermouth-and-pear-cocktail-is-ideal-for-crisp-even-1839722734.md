# 这种苦艾酒和梨鸡尾酒是清爽夜晚的理想选择

> 原文：<https://lifehacker.com/this-vermouth-and-pear-cocktail-is-ideal-for-crisp-even-1839722734>

比起真正的水果，我更喜欢梨，主要是因为——像鳄梨一样——它们的“啊，该死的耶”窗口很短。但是梨白兰地从来没有让我失望；味道一直很好，口感从不粗糙，ABV 徘徊在 40%左右。



这是一种纯的、令人陶醉的梨子酒，在鸡尾酒中用苹果白兰地代替它是改变你季节性饮酒习惯的好方法。这种苦艾酒味重的鸡尾酒是星星上的即兴重复，是我在罗伯特·西蒙森的 [*三成分鸡尾酒*](https://www.amazon.com/3-Ingredient-Cocktails-Opinionated-Enduring-Cocktail-ebook/dp/B01MS373D3?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/this-vermouth-and-pear-cocktail-is-ideal-for-crisp-even-1839722734&asc_source=&tag=kinjalifehackerlink-20) (一本非常好的书)中找到的苹果白兰地曼哈顿。这种饮料是半甜的苦艾酒，所以挑一种好的。( [再来一次](https://skillet.lifehacker.com/get-devilishly-drunk-with-the-defender-1839362396) ，我必须推荐 Carpano Antica 配方，冰镇后可以单独享用。)等量的苦艾酒和白兰地平衡并混合在一起，没有一种会被另一种压倒。要制作这个星空梨，你需要:

*   1.5 盎司梨白兰地
*   1.5 盎司甜苦艾酒
*   两个破折号安古斯图拉苦酒

将所有材料倒入混合杯，搅拌至冷却。挤进一辆小轿车。重复，也许这次是苹果白兰地。
