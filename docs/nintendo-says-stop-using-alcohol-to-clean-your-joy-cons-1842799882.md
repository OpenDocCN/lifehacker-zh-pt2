# 任天堂说停止用酒精清洗你的游戏机

> 原文：<https://kotaku.com/nintendo-says-stop-using-alcohol-to-clean-your-joy-cons-1842799882>

你可能已经听说了目前正在进行的全球疫情。因此，你可能会用酒精和消毒湿巾之类的东西来清理你的东西，包括你的开关。任天堂说不要，至少如果你的首要任务不是破坏系统的完成。对许多人来说，可能不是。

Watch

任天堂客服 [根据 *Kotaku* 的翻译昨天在推特](https://twitter.com/nintendo_cs/status/1248407634211790849) 上写道:“最近，客户一直在问他们是否可以用酒精给他们的任天堂 Switch 游戏机和游戏机消毒。”。“非常抱歉，请避免使用酒精，因为酒精可能会导致塑料零件褪色或变形。”

但不仅仅是酒精。该公司接着说，其他清洁剂也有潜在的风险。“我们也不建议使用不含酒精的消毒片，因为根据成分不同，它们可能会损坏塑料部件，”任天堂写道。

对于那些过去试图改造他们的欢乐合唱团或者使用奇怪的化学物质来清洁他们的人来说，这并不是什么令人惊讶的消息。漆面非常不稳定，很容易擦掉。新冠肺炎 [的传播很严重，尽管](https://gizmodo.com/trump-finally-admits-covid-19-is-serious-blames-europe-1842281267) ，想要让你的电子产品 [免于成为病毒的潜在携带者](https://kotaku.com/how-to-clean-and-sanitize-your-gaming-gear-1842365692) 是聪明的，尤其是对于那些经常与室友或爱人共用交换机的人。

疾控中心说:“每天清洁和消毒经常接触的表面。”"这包括桌子、门把手、电灯开关、工作台面、把手、书桌、电话、键盘、马桶、水龙头和水槽."为此，苹果 [上个月更新了其 iPhone 支持页面](https://gizmodo.com/apple-says-its-a-ok-to-use-clorox-wipes-on-your-iphone-1842216982) 告诉用户，与其之前的建议相反，用某些酒精和基于 Clorox 的湿巾“温和地”擦拭他们的设备是可以的。

那么你可以用什么来擦拭你的欢乐大会呢？“出于清洁目的，请使用柔软的干布，”该公司写道，如果你担心你的开关可能被污染，你绝对应该忽略这条建议。当这一切结束时，没有人会赢得最质朴的特别版欢乐大会的奖项。

任天堂没有立即回复置评请求。
