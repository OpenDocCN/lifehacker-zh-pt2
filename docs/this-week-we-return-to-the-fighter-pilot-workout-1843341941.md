# 本周我们回到战斗机飞行员训练

> 原文:[https://life hacker . com/this-week-we-return to the fighter-pilot-workout-1843341941](https://lifehacker.com/this-week-we-return-to-the-fighter-pilot-workout-1843341941)

我们开始了本月的健身挑战——重温经典家庭训练——用 [科学七分钟训练](https://vitals.lifehacker.com/lets-revisit-the-seven-minute-workout-1843201042) 重温。本周，我们将历史追溯到更久远的加拿大机构，该机构最初是为驻扎在没有健身房或锻炼空间的地方的军事飞行员发明的。我打赌我们很多人都有同感。

这种锻炼在军队之外变得流行起来，它之所以成为经典是有原因的。这里是 [解释](https://csclub.uwaterloo.ca/~rfburger/5bx-plan.pdf) 套路的完整小册子 PDF。不仅介绍中充满了对*为什么*你应该首先费心锻炼的精彩(如果过时的话)解释，小册子还提供了一个完整的锻炼计划。有六张图表；这个想法是，你从最简单的一个开始，随着你变得更健康，逐渐过渡到下一个。每个图表也给你一个等级，所以你有一个适合自己的分数和一个努力的目标。根据你的年龄，有平民和飞行人员的推荐分数。

我没有按照说明去做，而是直接跳到了图表 3。这些练习有点奇怪，所以请仔细阅读说明。我把仰卧起坐换成了仰卧起坐，因为这些被认为是潜在不安全的(虽然我不知道它到底有多严重)。这张图表也有一个动作看起来像一个向下的狗，但它实际上是一个俯卧撑，在你把屁股伸到空中之前，你的下巴和额头在地上轻轻一拍。我不能 100%确定我做对了。

为了完成锻炼，你在分配的分钟数内做每项锻炼，并数你的重复次数。这意味着两分钟用于脚趾- 触摸伸展；仰卧起坐、背部伸展和奇怪的俯卧撑各一分钟；六分钟原地跑步，中间穿插半膝弯曲。

我的结果是:脚趾触地、背部伸展和跑步都得了 A+...但是俯卧撑上一个实 D。是我不擅长这些还是我做错了？可能两者都有一点。

总的来说，我发现这种锻炼比七分钟锻炼更平衡，尽管它花在力量动作上的时间更少。我抱怨说那个在腿部没有挑战性，这个一点也没有。(是的，你在跑步，但是在你跳到图 4 中的跳跃之前，没有什么比蹲下或弓步更好的了。)这项锻炼最初并不是为了让更健康而设计的，只是为了在你没有更好的选择时帮助你保持健康，所以我想它在这一点上起到了作用。

总之，我有点喜欢这个。如果你想在家里找点事做，我认为比七分钟锻炼更好的选择是。它根据你的健康水平和缩放，总是给你一个期待的目标。图表 6 是“冠军运动员”一路平安。
