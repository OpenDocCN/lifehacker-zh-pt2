# 在一些机场餐厅使用您的优先通票可享受 28 美元的优惠

> 原文:[https://life hacker . com/use-your-priority-pass-for-28-off-at-some-airport-rest-1833471990](https://lifehacker.com/use-your-priority-pass-for-28-off-at-some-airport-rest-1833471990)

如果你使用优先通行证进入机场候机室，你可能会对免费食物印象不深，这取决于你在哪里。但是有一种方法可以在不花更多钱的情况下获得更高质量的食物。

Watch

你可能没有意识到的一个好处是，优先通票与一些机场餐厅合作，提供餐饮积分，通常为 28 美元，正如莎莉·弗伦奇为 [Wirecutter](https://thewirecutter.com/money/credit-cards/maximize-priority-pass/) 所写。这有助于缓解休息室过度拥挤的状况，并给会员带来额外的好处。法语解释道(重点是我的):

> 有些餐馆还允许你带一个客人，从你的账单上扣除你和一个幸运的朋友总共 56 美元(即使你的朋友没有优先通票)。
> 
> 你只需出示你的卡，点菜单上的菜，就可以享受 28 美元的优惠。任何超过 28 美元的费用都是免费的，小费总是额外的(**你应该给全额小费，而不是你支付的折扣金额**)。

如果你提前到达机场(或中途停留)，并且不想在候机室里处理潜在的低水准服务，这是一笔很好的交易。弗伦奇还提到，她会利用这一福利为以后的生活准备食物:

> 有时我在机场并不饿，就像在去澳大利亚昆士兰的途中在悉尼停留时那样。当我抵达悉尼时，我去了 Movida(一家提供 36 澳元优惠的餐厅)，买了一个烤南瓜和鹰嘴豆泥三明治(14 美元)和一个巧克力羊角面包(6 美元)，我留着第二天在斯普林布鲁克国家公园 徒步旅行时吃。

Priority Pass 的 [网站注明](https://www.prioritypass.com/en/blog/2017/benefits-you-didnt-know-you-had) 优惠“可能因地点和/或时间而异”积分员列出了美国和海外所有的 [优先通票餐厅](https://thepointsguy.com/news/priority-pass-adds-new-restaurant-us/) 。

要成为 Priority Pass 会员，你要么需要在它的网站上注册(你需要支付 [年费](https://www.loungebuddy.com/blog/priority-pass-the-ultimate-guide/) ，然后每次访问都要付费)，要么通过 [旅行奖励卡](https://twocents.lifehacker.com/these-credit-cards-give-you-free-airport-lounge-access-1820374723) 来访问它，如果你有一张的话，比如大通蓝宝石储备卡，或者美国运通的白金卡，这是一种额外津贴。
