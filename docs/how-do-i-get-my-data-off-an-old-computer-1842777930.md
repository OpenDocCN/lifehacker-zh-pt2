# 我如何从旧电脑上获取数据？

> 原文：<https://lifehacker.com/how-do-i-get-my-data-off-an-old-computer-1842777930>

无论你是升级还是购买一台全新的台式机或笔记本电脑，总有一天你将不得不与使用多年的电脑说再见。你们中的大多数人会在永远离开退休系统之前尝试从退休系统中提取数据。如果你很匆忙——因为你和我一样对建立更新、更快、更漂亮的东西感到兴奋——你会忘记这样做，你会发现自己被卡住了。



至少，这是生活黑客读者科里提出的情况，他最近向 [**理工 911**](https://lifehacker.com/c/tech-911) **:**

### 我想删除旧电脑上的所有内容，但是...

> *“你好大卫，*
> 
> 我有一台较旧的 gateway tower，但卖掉了附带的显示器。
> 
> 我有一台笔记本电脑和一台电视，我试着使用 HDMI 转 VGA 适配器，以为它会在我的电视或笔记本电脑上显示，但它不起作用。我想在回收利用我的旧电脑之前，先把它清理干净。

> 如何在旧电脑上查看内容，有什么建议吗？"

### 为什么你的旧电脑收不到信号

好消息是:您有一些解决方案来解决这个问题。首先，让我们讨论一下什么是行不通的。将台式机连接到笔记本电脑不会有任何作用，因为笔记本电脑的连接是输出。无论你使用什么连接，把你的台式电脑和你的笔记本电脑连接起来都不会在你的笔记本电脑上显示你的桌面图片。这在理论上是一个伟大的想法，但在技术上是行不通的。

我从未尝试过使用 VGA 转 HDMI 适配器——事实上，这是我第一次听说有人真的使用它，考虑到这种连接类型有多古老。我的怀疑是这种技术行不通，或者说失败的可能性很大，因为 VGA 是模拟信号，HDMI 是数字信号。在两者之间转换是可能的，但是无论你使用什么适配器，都必须接受 VGA 信号并进行转换。如果你只有一根简单的电缆，一端是 VGA 连接，另一端是 HDMI 连接，那可能就不行了。

也有可能你选择的适配器很糟糕，因为没有更好的表达方式。如果你买了廉价的适配器，或者买了一个还没有被很多人测试过的适配器，它可能不能做你想要它做的事情，因为这就是事情的发展。

### 适配器会有所帮助，但还有更好的解决方案

这是一个棘手的情况，因为我通常推荐的 USB 转 DVI 适配器，然后连接到 DVI 转 HDMI 电缆，或者更简单的 USB 转 HDMI 适配器，可能会比您愿意花在解决方案上的钱更多。至少，在我花 30-50 美元购买适配器之前，我可能会四处打听，看看我的朋友中有没有带 VGA 输入的旧显示器。

有一点值得检查的是，你的旧台式机是否有 S-Video 端口。这不太可能，但如果可能，并且您的电视有一个，您可以通过这种方式将它们连接在一起。希望渺茫，但值得探索。

否则，我认为你最好的选择是登录你所在地区的 Craigslist 网站，购买别人便宜的旧显示器。在你付钱之前，确保他们可以确认它可以使用 VGA 信号，但这将是从你的系统中获得图像的最佳和最可靠的方法。这样，你就可以开始仔细研究你的数据，清理你的系统。

看看我所在地区的 Craigslist，人们正在以大约 15-30 美元 的价格出售他们旧的、蹩脚的 VGA 显示器 [。你可能在你住的地方找到和那一样便宜的东西；请记住，重要的是它能工作，而不是它看起来漂亮，有一个巨大的屏幕，或任何类似的东西。把它连到你的电脑上，保存或删除任何你想要的东西，然后和你的旧电脑一起回收。](https://sfbay.craigslist.org/search/sss?sort=priceasc&query=vga%20monitor)

### 要获取旧数据，请尝试移除硬盘

摆弄显示器和监视器是一种方法。由于您关心的只是硬盘上的数据，因此您也可以打开台式机，断开并移除硬盘，然后将硬盘放入外部机箱。我会推荐一个，但我不确定你的驱动器是否使用古老的 IDE 连接或 SATA。(你可以找到处理这两者的 [机箱和](https://smile.amazon.com/Hard-Drive-Enclosures-IDE-SATA-Accessories/s?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-do-i-get-my-data-off-an-old-computer-1842777930&asc_source=&rh=n:160354011,p_n_feature_keywords_browse-bin:5871549011&sa-no-redirect=1&tag=kinjalifehackerlink-20) 坞站，所以这几乎是一个有争议的问题。)

将旧台式机的硬盘安装到其中一个硬盘上，通过 USB 将硬盘连接到新的台式机或笔记本电脑上，您应该可以访问您的所有文件。保存您想要的任何内容，擦除驱动器，并在完成后将其和您的旧系统一起送去回收。

虽然这听起来像一个过程，但从您的台式电脑上移除硬盘驱动器就像断开几根电缆并拧下四个螺钉一样简单。取出硬盘时，请注意不要过度挤压。不要把它倒过来或做任何类似疯狂的事情，并确保在摆弄系统内部之前，您已经触摸了您的(金属)外壳的位置以释放任何静电。

如果我是你，我会走那条路，因为它既容易又省钱。与其花 20-30 美元买一个你永远不会再用的显示器，你可以选择一个你在未来某个时候肯定会用到的坞站，无论你是从另一个驱动器上下载文件，还是将文件复制到一个空的备份驱动器上，等等。这比一个蹩脚的旧显示器要实用得多，但是它需要为你的情况做更多的跑腿工作。你能做到的。我有信仰。
