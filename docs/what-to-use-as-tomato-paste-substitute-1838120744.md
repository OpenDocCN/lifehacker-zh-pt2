# 我可以用什么做番茄酱的替代品？

> 原文：<https://thetakeout.com/what-to-use-as-tomato-paste-substitute-1838120744>

几乎没有人记得买番茄酱。这是你认为理所当然的食品柜项目之一，假设总会有几个罐子藏在你橱柜的最深处。你以为你有番茄酱，但是，就在你需要的时候，你发现它不见了。

Watch

不幸的是，番茄酱没有替代品，因为这完全取决于你用它做什么。西红柿天然富含谷氨酸盐，所以当你把它变成糊状时，你最终会得到一种勃艮第色的鲜味浓缩物，它可以给你做的任何东西增加一股美味的冲击力。它可以立即给汤、炖菜和酱汁一种“炖一整天”的味道，这是我们大多数人没有时间实现的，或者给炒菜、三明治、沙拉酱——几乎所有的“S”食物添加某种 X 因素。

我建议你买一个可以放在冰箱里的可挤压的番茄酱，不仅因为它可以保存一段时间，而且它会提醒你更经常地使用这种神奇的成分。(Mutti 是个好牌子， [在这里找到，](https://www.amazon.com/Mutti-Double-Concentrated-Tomato-4-Pack/dp/B07JRCSTKT?asc_campaign=InlineText&asc_refurl=https://thetakeout.com/what-to-use-as-tomato-paste-substitute-1838120744&asc_source=&language=en_US&linkCode=ll1&linkId=f8f5adbaf6f47eb3b458aaac165d9d2a&ref_=as_li_ss_tl&tag=kinjatakeoutlink-20) 但绝不是唯一值得一试的。) [摆弄一下](https://thetakeout.com/recipe-canned-crab-pasta-caramelized-tomato-paste-1831589202) ，在你正在做的任何东西中加入一两汤匙，可以让它美味可口。但是，如果你的冰箱里没有方便的管子，或者如果你正在阅读这篇文章，因为你需要立即得到帮助，而谷歌告诉你阅读这篇文章，这里有一些替代方法可以在紧要关头发挥作用:

### 番茄减量

如果你有一些压碎的西红柿 [(这些很好)](https://www.amazon.com/Colavita-Italian-Crushed-Tomatoes-Recart/dp/B077K8J467?asc_campaign=InlineText&asc_refurl=https://thetakeout.com/what-to-use-as-tomato-paste-substitute-1838120744&asc_source=&language=en_US&linkCode=ll1&linkId=6857db7a472477e166d29532533e12b0&ref_=as_li_ss_tl&tag=kinjatakeoutlink-20) 或番茄酱 [(又名过滤西红柿)](https://www.amazon.com/Pom%C3%AC-Strained-Tomatoes-7-05-Pack/dp/B08C1LK2CW?asc_campaign=InlineText&asc_refurl=https://thetakeout.com/what-to-use-as-tomato-paste-substitute-1838120744&asc_source=&language=en_US&linkCode=ll1&linkId=93bdd82548fbb3d9b230cc1708971ad9&ref_=as_li_ss_tl&tag=kinjatakeoutlink-20) ，你可以在 20 分钟或更短时间内制作自己的酱。将你食谱中给出的番茄酱量乘以 5，然后将一定量的罐装番茄倒入一个小平底锅中。将它加热至沸腾，然后放入文火煨，每隔一两分钟搅拌一次，以防烧焦。(除了谷氨酸盐，西红柿也有很多天然糖，如果你让它们粘在锅底，它们就会燃烧。)

一旦体积减少到原来的五分之一，你就可以自己做番茄酱了——但现在你有了一罐用了一半的番茄，而你与它毫无关系。这就是为什么我推荐使用这种方法，如果你正在做某种番茄沙司，或者一大锅需要慢炖的东西，比如 T2 汤或者辣椒。直接把整罐都倒进去就可以了，因为像这样的菜真的不需要太紧的尺寸。一点点额外的番茄酱真的只能让东西比你原来的食谱味道更好。

### 鱼酱或伍斯特郡酱

如果你想给你的菜肴增加一点味道，就拿一个餐具吧。番茄酱也没有给派对带来甜味，但当谈到鲜味时，很少有东西能与 [鱼露](https://thetakeout.com/unexpected-ways-to-use-fish-sauce-cooking-1833471750) 相比。这方面的其他好替代品:Maggi 酱 [(一种很棒的调味酱)](https://www.amazon.com/Jugo-Maggi-Seasoning-Sauces-Bottles/dp/B00EN2A3RQ?asc_campaign=InlineText&asc_refurl=https://thetakeout.com/what-to-use-as-tomato-paste-substitute-1838120744&asc_source=&dchild=1&keywords=Maggi Sauce&language=en_US&linkCode=ll1&linkId=28de81ca4c39617ef7b13ad9a6f74676&qid=1611355126&ref_=as_li_ss_tl&s=grocery&sr=1-2&tag=kinjatakeoutlink-20) ， Vegemite [(你可以在环保包装中买到)](https://www.amazon.com/Vegemite-Australia-Friendly-Packaging-Creations/dp/B08MLJ81X8?asc_campaign=InlineText&asc_refurl=https://thetakeout.com/what-to-use-as-tomato-paste-substitute-1838120744&asc_source=&language=en_US&linkCode=ll1&linkId=ef4eae14cbe2bb871798c5122e3df218&ref_=as_li_ss_tl&tag=kinjatakeoutlink-20) ，凤尾鱼酱，或者一点蘑菇汤底 [(此处有售)](https://www.amazon.com/Better-Than-Bouillon-Mushroom-Base/dp/B01MXHLSZD?asc_campaign=InlineText&asc_refurl=https://thetakeout.com/what-to-use-as-tomato-paste-substitute-1838120744&asc_source=&language=en_US&linkCode=ll1&linkId=4733a74a1c0f3d1035c4dac2ee3ab70f&ref_=as_li_ss_tl&tag=kinjatakeoutlink-20) 。这些东西都不能完全取代番茄酱，而且都非常强烈，所以边吃边加一点。

## 哈里萨

鲜味可能不是你伸手去拿番茄酱的原因；有时你会寻找甘美而略带甜味的背景调，比如在**锅酱**或**快速炒**中。在这种情况下，捣碎的烤红辣椒效果很好，但就我个人而言，我喜欢使用辛辣、芳香的 harissa，这是我的十大调味品之一。其他九个是什么？一个真正的女士从不告诉所有人。

### 蕃茄酱

当有人问我 [番茄酱](https://thetakeout.com/a-blind-taste-test-to-determine-the-best-ketchup-1825133916) 是否可以作为番茄酱的替代品时，我最初说不可以——番茄酱不是纯粹的番茄酱，而是番茄、醋、糖、香料和一堆其他有趣东西的混合物。然后我问这个人他们通常用番茄酱来做什么，当他们说 [菜炖牛肉](https://thetakeout.com/recipe-oma-beef-goulash-1832468499) 时，我改变了主意。这仍然不是一个完美的替代品，但是在像辣椒粉这样强烈的烟熏香料的菜肴中，番茄酱的糖可以帮助圆一些香料的侵略性，令人难以置信的是，这实际上可以使它们尝起来更好。因此，虽然番茄酱不是所有东西的最佳替代品，但如果你做的任何东西都使用了大量的香料，如孜然、墨西哥辣椒或液体烟，试着加入一点番茄酱，看看你会喜欢它。
