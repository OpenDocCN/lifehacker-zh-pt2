# 使用 Pixel 4 的“实时字幕”功能转录静音视频

> 原文：<https://lifehacker.com/use-the-pixel-4s-live-caption-feature-to-transcribe-mut-1839362089>

谷歌的 Pixel 4 最酷的部分之一是其人工智能辅助的实时字幕功能，允许你的智能手机自动为你的应用程序中正在播放的任何音频添加字幕。这让你可以阅读 YouTube、Twitter 等媒体资源上播放的内容。

Watch

正如 Android Police 最近发现的 一样，当你的 Pixel 4 的音量被调低或者一个应用被静音时，音频转录甚至可以工作。这是因为现场录音是由音频数据生成的，而不是由手机麦克风采集的声音生成的。I 如果你喜欢将自动播放的视频静音(或者如果你的浏览器默认不静音)，但不想错过偶尔播放的视频中的重要信息，这种静音和直播字幕的组合非常适合你。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-v8cLk2W3vY8&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-v8cLk2W3vY8&start=0) 

要打开 Pixel 4 的实时字幕，请前往**设置>声音>实时字幕**，然后点击滑块打开该功能。当手机识别到正在播放内部音频时，会自动通知您并开始添加字幕。

对于有听力障碍的人来说，这是一个很好的辅助选项，但所有用户都将受益于直播字幕，包括 Pixel 3 和 3a 用户，他们将在即将到来的更新中获得该功能。至于静音方面，这很简单:当你使用网络浏览器时，只要把手机的音量调到最大。你也可以打开它，点击 Chrome 地址栏右边的锁图标(例如)，点击网站设置，屏蔽整个域名的声音。
