# 卸载这些窥探你的 iPhone 或 Android 的 VPN 和广告拦截应用

> 原文：<https://lifehacker.com/uninstall-these-vpn-and-ad-blocking-apps-that-spied-on-1842274746>

最近的一项 [Buzzfeed 调查](https://www.buzzfeednews.com/article/craigsilverman/vpn-and-ad-blocking-apps-sensor-tower) 发现，几个流行的 Android 和 iOS VPN 和广告拦截应用程序秘密收集用户数据，并将其发送到 Sensor Tower，这是一个为应用程序开发者提供的跨平台数据分析服务。窥探应用包括:

Watch

*   Adblock Focus (Android 和 iOS)
*   免费和无限制的 VPN (Android)
*   Luna VPN (Android 和 iOS)
*   移动数据(Android)

这些应用能够通过获得设备文件和文件夹的 root 访问权限来跟踪用户数据——这是苹果和谷歌都不允许应用正常进行的事情，但开发者利用了一个简单的漏洞来绕过限制。

许多应用程序使用误导性的通知，承诺额外的功能——比如 Luna VPN 提供免费的 YouTube 广告拦截——如果用户在安装应用程序后从第三方网站下载额外的文件。这些“额外功能”将要求额外的权限，包括根访问权限，以便运行。

苹果和谷歌都禁止了各自商店中的违规应用，并正在调查这些应用背后的开发公司，以回应 Buzzfeed 的调查结果。用户应该卸载上面列出的应用程序，特别是如果你从第三方来源而不是 App Store 或 Google Play 下载了任何应用程序。这些应用程序都没有透露他们的数据跟踪行为，也没有透露他们与传感器塔的连接。它们不是 Sensor Tower 仅有的应用程序，只是每个平台上最近可用的应用程序。如果您安装了其他 VPN 或广告拦截应用程序，并且您不确定是谁开发的，请快速进行网络搜索，看看 Sensor Tower 是否是开发者。如果是这样，删除应用程序。

这也是另一个提醒，要注意 app 权限，尤其是 root 访问权限。除非你*绝对确定*你正在安装的东西是安全的，并且出于某种特定的原因你需要授予应用 root 访问权限，否则不要这样做。这样做可以让应用程序直接访问手机上的基本所有内容——所有文件、文件夹和应用程序——并完全访问它想做的任何事情。

即使一个应用程序正在请求“正常”权限，它们也可能是不必要的。VPN 不需要访问你的联系人或相机，广告拦截器也不需要访问你的文件夹。注意和 [细看一个 app 索要的权限](https://lifehacker.com/check-if-your-android-vpn-is-collecting-personal-data-1833108243) 。如果其中任何一个看起来不必要或可疑，拒绝请求并卸载它。
