# 写正面评价而不是负面评价

> 原文:[https://life hacker . com/write-a-positive-review-而不是-a-negative-review-1830771188](https://lifehacker.com/write-a-positive-review-instead-of-a-negative-review-1830771188)

如果你在某件事上有过糟糕的经历——比如去餐馆、看书或去沙龙——在留下负面评价之前花一分钟。值得吗？

Watch

在我们关于 [写作创作者感谢信的帖子](https://twocents.lifehacker.com/send-creators-thank-you-notes-this-holiday-season-1830721570) 这个假期， [交通锥 3 号](https://twocents.lifehacker.com/1830731480) 提供了这个伟大的建议:

> 我得到并实践的一条建议是，每当你感到不得不对某件事留下抱怨或负面评价时，花点时间对其他事情留下积极的[重新]看法。两个原因，消极倾向有时占主导地位，当人们不高兴时，他们更有动力说出来，所以这有助于扯平。其次，它有助于你自己的精神健康。让你不太可能沉湎于消极。

如果你没有什么好话要说，这是对经典建议的扭曲...相反，写一些关于其他事物的好话。不要沉浸在消极中会对你的心理健康更好，而且它会帮助你做一些事情，比如在算法中浮出水面。

这并不是说你无视所有的不良行为或糟糕的服务——有时服务员真的是歧视性的，或者服务不像商家声称的那样。但是如果你有一次性的糟糕经历， [花点时间](https://lifehacker.com/think-for-one-damn-second-before-you-react-to-people-1830545892) ，然后考虑对你确实喜欢的东西留下积极的评价。你会感觉好一点。
