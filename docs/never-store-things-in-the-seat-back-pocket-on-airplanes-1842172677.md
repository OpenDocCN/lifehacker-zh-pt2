# 千万不要在飞机上把东西放在座椅靠背的口袋里

> 原文:[https://life hacker . com/never-store-things-in-the-seat-back-pocket-on-plants-1842172677](https://lifehacker.com/never-store-things-in-the-seat-back-pocket-on-airplanes-1842172677)

冠状病毒使得航空旅行成为许多人正在重新考虑的事情。当登上飞机时，越来越多的人也考虑给他们的私人空间一点擦拭，试图抓住任何可能由坐在那里的最后一名乘客留下的细菌。一个你可能不会考虑的地方:座椅靠背的口袋。

Watch

对于经济舱乘客来说，座椅靠背的口袋是最受欢迎的存放物品的地方之一，从瓶装水到手机、书籍和笔记本电脑。这也可能是飞机上最脏的地方之一。

[知情人最近揭露了](https://www.insider.com/flight-attendant-why-you-shouldnt-use-plane-seat-pockets-2018-11)[Reddit 帖子](https://www.reddit.com/r/AskReddit/comments/87g3ai/flight_attendants_of_reddit_what_are_some/dwd8342/) 询问空乘人员乘客应该知道的航空旅行中最令人不安的秘密。

第一，我已经知道:短途航班通常只是翻个身，而不是深度清洁。虽然由于冠状病毒，现在可能有点不同，但平均而言，你的短途航班只会有一名机组人员经过，清除明显的垃圾并折叠安全带。他们没有擦拭每一张餐桌或扶手。上一个坐在你座位上的人可能感冒了，或者甚至在托盘桌上给他们的婴儿换尿布。

我们都应该带上抗菌湿巾，把我们的空间彻底擦干净，而不仅仅是在病毒在世界上传播的时候。

她指出的另一件事是:你永远不要把任何东西放在座位的口袋里。

虽然飞机最终得到了深度清洁，但对于座椅靠背口袋来说，唯一发生的事情是垃圾被拿出来了。这意味着如果你前面的人把他们的脏纸巾放在里面，当你在起飞前把它放在口袋里时，这些细菌就有可能跑到你的 iPad 上。

作为一名空姐，她也见过人们把脚放在那些口袋里。所以如果组织的事情没有得到你也许别人的脏脚趾会。

虽然现在清理飞机上的空间是绝对好的建议，但这也是你在旅途中应该记住的事情，即使没有病毒威胁要应对。
