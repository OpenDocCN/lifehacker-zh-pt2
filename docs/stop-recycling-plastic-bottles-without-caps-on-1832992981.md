# 停止回收没有盖子的塑料瓶

> 原文：<https://lifehacker.com/stop-recycling-plastic-bottles-without-caps-on-1832992981>

随着 [回收行业](https://www.wired.com/story/since-chinas-ban-recycling-in-the-us-has-gone-up-in-flames/) 的不断变化，围绕回收塑料的规则并不总是一成不变的(至少在美国是这样)。首先，你不能回收塑料袋或吸管，现在 [黑色塑料](https://lifehacker.com/think-twice-before-recycling-black-plastic-1832868090) 甚至成为一个问题。我们怎么知道回收的正确方法呢？



鉴于这种困惑，在最近的一个 [Reddit 帖子](https://old.reddit.com/r/LifeProTips/comments/aw5z2c/lpt_whether_you_recycle_or_just_straight_up_throw/) 上，用户 u/[insert-quote-here](https://old.reddit.com/user/insert-quote-here)提倡在回收时取下瓶盖是有道理的，但这就是为什么这通常是一个错误的举动——通过将瓶盖从瓶子上分离，你实际上已经将它扔进了垃圾桶。

为什么？通常情况下，当瓶盖进入回收设施的分拣流程时，它们会因为体积小而被分拣出来，然后被送往垃圾填埋场。因此，瓶盖现在是在我们的海洋中发现的最常见的物品之一(当被海洋生物摄入时，它们也会出现在食物链中)。

部分困惑在于我们认为拧开讨厌的盖子可以让回收者的工作变得更容易，但这实际上是浪费你的时间和精力。瓶盖和瓶子通常在用水设施中分开(切碎后，它们根据漂浮能力分开。瓶盖密度较低，会漂浮。瓶子下沉)。

帽子也很值钱。它们由 [聚丙烯塑料](https://www.plasticsmakeitpossible.com/plastics-recycling/plastics-recycled-happens-recycled-plastics/) 制成，这是一种制造商梦寐以求的塑料类型，用于从储藏箱到砧板的所有东西，所以扔掉它们是对一种非常有用的资源的浪费。

那么回收塑料水瓶的正确方法是什么呢？首先，打开瓶盖，从底部向上挤压瓶子，直到大部分空气被去除。拧上盖子，然后把瓶子扔进回收箱！这很简单，只要你记住这两者是属于彼此的。和往常一样，在网上搜索你当地家乡的回收项目，找出上限的情况，以防你的是一个罕见的例外。
