# 通过在航班起飞前痴迷地检查座位来获得升级

> 原文：<https://lifehacker.com/score-an-upgrade-by-obsessively-checking-seats-before-a-1836497476>

我能给那些不想为“高级”座位的额外腿部空间支付愚蠢费用的乘客的最好建议之一是:*经常检查你的座位分配*。



我的意思是。我最近预订了阿拉斯加航空公司飞往纽约的航班，这让我可以选择航班的座位。很明显，挪到伸腿空间以外的座位需要花钱，快速的 [座位专家](https://www.seatguru.com/) 查询让我知道只有~~平民~~普通经济舱座位——甚至没有出口排。

我很有策略地安排了我的座位，选择了飞机中更靠后的靠窗座位——这一排已经有人坐在过道座位上了——希望航班不会卖得很好，没有人会想要可怕的中间座位。我还经常在阿拉斯加航空公司的应用程序中查看可用座位，至少每周一次，直到航班起飞，以及飞行那周的每一天。

虽然我最初的计划因为有人坐在中间而失败，但我笑到了最后。在我乘坐飞机的前几天，阿拉斯加航空公司突然向所有人开放了两个出口。我很快换了一个座位，找到了一排出口，在五个多小时的飞行中，我享受了 6 英尺 5 英寸的腿部空间。(我在回家的路上也玩了同样的把戏。)

这能在任何载体上工作吗？不。西南航空已经出局了( [显然是](https://www.southwest.com/html/generated/help/faqs/boarding_the_plane_faq.html) )，像联合航空这样的航空公司即使为最微不足道的座位升级也不会让你免费换到更贵的座位:

随着您的航班日期越来越近，仍然有必要查看是否有空位。也许高级座位会降价。也许你会从中间的座位跳到过道或窗口，因为有人决定升级。如果你乘坐阿拉斯加航空公司的航班，你甚至可能得到梦寐以求的出口排。

值得一提的是，我还学到了这个诀窍——勤奋，真的——对你想参加的活动也很有效。就我而言，我想在旅途中观看托尼奖获奖影片《哈德斯敦》( Hadestown )( T1 ),但在我能去的那几天，所有的票都卖光了(除了 Ticketmaster 上那些可怕的、标有“验证转售”和“高级座位”的选项)。

在我旅行的两周前，我再次心不在焉地查看 Ticketmaster 网站上的这些日期，我看到了一个奇迹:大约有 6-8 个正常价格的包厢座位。后来我打电话给 Ticketmaster，我看到*的座位很棒，而且不用支付黄牛票费——或者冒着那天的风险——我可能还没买到票就卖光了。*

到目前为止，对座位的狂热在我纽约之行的每一个例子中都得到了回报，它教会了我检查，再检查，甚至更多检查的价值。你永远不知道会发生什么。
