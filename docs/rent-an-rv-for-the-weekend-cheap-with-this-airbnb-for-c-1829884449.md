# 用这个“露营车 Airbnb”租一辆周末用的廉价房车

> 原文：<https://lifehacker.com/rent-an-rv-for-the-weekend-cheap-with-this-airbnb-for-c-1829884449>

如果你想租一辆车或一个住的地方，你有很多选择。但是如果你想租一辆兼作住宿地方的车呢？那么你的选择就会变得更加有限。

Watch

[Campervancentral](https://www.campervancentral.com/#!/) 宣称自己是“露营车和房车世界的 Airbnb”，有了它，你可以从你附近的个人那里找到并租用露营车或房车。这是一种类似于 Getaround 和 Airbnb 的东西，前者提供别人的汽车租赁服务，后者提供别人的房屋租赁服务。

这项服务目前只在网上提供(没有应用程序)，你可以使用你的位置或邮政编码来搜索房车。

现在在英国有很多选择，特别是在公司总部所在地。例如，你可以花 50 英镑在爱丁堡租一辆柴油房车，可以睡四个人，还允许养宠物。

就像 Airbnb 和 Getaround 这样的服务一样，if 允许车主将他们当时可能不会使用的东西货币化，并允许周末需要房车的人以比通过传统渠道低得多的价格获得房车。
