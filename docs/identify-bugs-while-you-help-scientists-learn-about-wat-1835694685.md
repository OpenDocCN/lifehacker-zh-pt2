# 在帮助科学家了解水质的同时识别细菌

> 原文：<https://lifehacker.com/identify-bugs-while-you-help-scientists-learn-about-wat-1835694685>

有些昆虫只会生活在最新鲜、最干净的水中。其他人在任何又脏又旧的沼泽里都很快乐。科学家可以用水源中的虫子作为水质指标，在各种公民科学项目中，你也可以。



但是你必须知道哪些是水虫，这就是一个新的非常详细的网站的目的，这个网站叫做北美常见淡水大型无脊椎动物图集。(除非你喜欢看虫子的图片，否则不要点击那个链接。)

该网站有一个识别键，所以你可以回答几个快速的问题，并迅速找到正确的生物群——例如，如果它有三条尾巴，它就在 [豆娘目](http://macroinvertebrates.org/taxa-characters/odonata-larva) 中。然后你可以详细浏览不同的物种；每种生物都被拍摄了数千次，所以你不会错过任何东西。(还是那句话，除非你真的*真的*喜欢看虫子的图片，否则不要点击这些链接。)

该网站突出并解释了相似物种之间的差异，比传统的野外指南更容易使用。只有最常见的物种被包括在内，所以这不是一个详尽的列表，但它旨在方便学生和公民科学家——或者任何在溪流中发现虫子并想知道它是什么的人。
