# 教育你的孩子，每项任务都有开始、中间和结束

> 原文：<https://lifehacker.com/teach-your-kid-that-every-task-has-a-beginning-middle-1836922721>

如果我试图帮助我六岁的女儿玛吉成为一个更有条理的思考者，我的指导似乎来得太晚了。“到处都是棋盘游戏棋子，*，*”我会告诉她，在她逃离棋盘游戏并开始三种不同活动的几个小时后，我注意到了这一场景。作为一个目前拥有大约七本不同小说书签的人，在如何完成你开始的事情上，我可能不是最伟大的老师。但我想让我的孩子走上一条不同的道路——有条理可以减轻压力，增强信心，让人们有更多的时间去做他们喜欢的事情(即使现在，我女儿喜欢的是把手指伸进蓬松的粘液里)。

发育儿科医生、《养育有组织的孩子》一书的作者达蒙·科尔布博士写道:“有组织的孩子不是突然出现的——他们是被抚养长大的。”在他最近由美国儿科学会出版的书中，他解释说训练应该从婴儿期开始，有固定的程序。随着孩子年龄的增长，这些惯例将帮助他们预测他们的下一步行动，而来自妈妈或爸爸的提示越来越少。父母可以帮助锁定例程的一种方式，根据 Korb 的 [是**告诉孩子，每项任务都有开始、中间和结束**。他举了一个例子:一个孩子拿出一个玩具，玩着玩具，然后把玩具放好。他写道:“思维混乱的人可能无法识别任务中固有的顺序，可能需要持续的强化。”](https://www.yahoo.com/lifestyle/yes-third-grader-making-own-193600140.html)

孩子们已经从他们的故事中理解了开始、中间和结束的概念。当我晚上给玛姬读书时，我们经常以一句戏剧性的“eeeee end！”然后合上书，结束了这个故事和漫长的一天。完成每一个任务都要发出同样的感觉。如果你的孩子想抢零食，你可以问:“开头、中间和结尾是什么？”你的孩子可能会说，他们会从冰箱里拿出一包奶酪(开头)，吃掉它(中间)，然后扔掉包装纸(结尾)。帮助他们把这个过程想象成一个故事板，并看到整个过程。是关于不断强化。

随着孩子的成长，他们会有多个任务在进行中，这也很好。如果他们对万事万物都有开始、中间和结束有了深刻的理解，他们就能更好地衡量自己能做什么和不能做什么，并据此设定界限。

在我现在的家里，有很多事情卡在了中间，甚至卡在了开头。在玛姬的房间里，我看到了一些半成品的挂链和一系列我们一直想在今年夏天一起看但一直没有机会看的电影。尽早认识到结局会帮助我们结束这些任务，给我们更多的精神空间去进入下一个故事。
