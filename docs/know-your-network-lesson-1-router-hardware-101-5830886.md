# 了解您的网络，第 1 课:路由器硬件 101

> 原文：<https://lifehacker.com/know-your-network-lesson-1-router-hardware-101-5830886>

购买和配置一个新的无线路由器可能是一个令人麻木的练习。如果你对自己的极客技能非常自信，你可能不需要太多的手把手，但是有很多人对无线网络几乎一无所知。他们几年前买的一些四四方方的设备可以让他们在马桶上看 YouTube，这才是最重要的。

至少了解家庭网络如何运行的基本知识是很重要的，因为当您的连接开始变得疯狂时，这将帮助您排除故障(或升级)。它还将有助于确保您从所有网络设备中获得尽可能好的速度——如果您每个月已经为高速互联网计划(或光纤)投入了相当大的一笔钱，这将非常重要！)

我们将从了解您网络中最重要的设备开始:您的路由器。

### **什么是无线路由器**

您的路由器是将您的家庭网络连接在一起的粘合剂。它通过以太网电缆或无线连接将您的所有计算机相互连接。

路由器不同于调制解调器:你的调制解调器连接你到互联网，而你的路由器连接你的电脑。当您将路由器连接到调制解调器时，您就可以与网络上的所有计算机共享互联网连接。

一些调制解调器带有内置路由器，但情况并非总是如此。如果您的台式机或笔记本电脑没有内置路由器，您不应该将它们直接连接到调制解调器，因为路由器具有许多旨在保护您的设备免受外界影响的功能。您的调制解调器没有。你也可能向你的 ISP 付费，从他们那里租借一个蹩脚的调制解调器(或调制解调器/路由器),所以购买你自己的牛逼调制解调器*和*路由器——分开购买——是你为你的家庭网络所能做的最好的事情。

连接到路由器的设备——电脑、平板电脑、智能手机、智能家居扬声器、游戏系统等等——被称为*客户端*。您的路由器为网络上的每个客户端分配了自己的 IP 地址，这有助于您的路由器定向流量。网络内的客户端获得一个*本地* IP 地址，而你的调制解调器获得一个*全球* IP 地址。全球 IP 地址就像街道地址，而本地 IP 地址就像公寓号码:一个让您找到与世界其他地方相关的建筑，而另一个让您找到在建筑群内的具体位置。这些地址确保来自外部世界的正确信息到达您网络上的正确设备。

路由器有许多不同的功能，因此我们将介绍一些最常见的路由器规格，以及它们如何影响您的家庭网络。

### **有线与无线**

由于有线连接通常比无线连接更快、更可靠，因此您需要使用以太网电缆来硬连接任何不需要移动的计算机或设备，如台式 PC 或游戏机。它们不太适合你拿起来四处移动的设备，比如笔记本电脑和智能手机。为此，我们使用无线连接(通常称为 wifi)。你在这些设备上体验到的速度取决于许多不同的因素:你离路由器有多远，你的设备和路由器之间有什么，你的设备支持什么样的速度，还有哪些无线信号可能会干扰你的连接，有多少设备也连接到你的路由器，等等。

对于大多数人来说，路由器仅仅是让互联网在他们的设备上发生的东西——这是看待无线网络的一种简单方式，但也很公平。然而，即使是世界上最快的无线路由器，给你的智能手机提供最清晰的连接，如果你甚至没有为非常快的互联网连接付费，也不会感觉非常快。更糟糕的是，如果你的 ISP(或你试图访问的网站或服务)有问题，你的路由器将无法让奇迹发生，你也不应该因为你经历的任何缓慢而责怪它。

### **无线吞吐量**

吞吐量是路由器传输数据的速度。您的无线连接的传输速度取决于它使用的无线*标准*。当今最常见的标准是 802.11n 和 802.11ac(也分别称为“无线 N”和“无线 ac”，或“Wi-Fi 4”和“Wi-Fi 5”)。Wireless AC 比 wireless N 快得多，现在你能买到的大多数普通路由器都支持这两种标准。(换句话说，你的下一次升级可能不应该是只支持 wireless N 的路由器，因为你正在积累的所有无线交流设备都将被较慢的 wireless N 速度卡住)。

不过，如果你把大部分时间花在浏览网站和下载东西上，这可能不是问题。对大多数人来说，你从 ISP 处获得的网络计划可能比你的路由器所能处理的速度要慢。换句话说，如果你为一个 50Mbps 的互联网计划支付 50 美元/月，那么一个支持 1,600Mbps(理论上)最高速度的无线 AC 路由器在大多数情况下不会给你带来速度提升。

也就是说，最好是有一个至少能支持你连接到它的设备的最快连接的路由器。不用深入探究无线交流速度是如何分类的(那些可怕的 A#####数字)，这里有我们之前分享过的简单建议:

> *“对于大多数希望覆盖一个合理大小的家庭或公寓的人来说，一个强大的 AC1200 或 AC1750 路由器可能就足够了——例如，如果你拥有支持 AC1750 全速运行的新款 MacBook Pros，肯定是后者。"*

在路由器的包装盒上看到较大的数字并不一定意味着它是你能买到的最快的路由器。考虑一个极端的例子:TP-Link 的 [Archer AX11000](https://www.tp-link.com/us/home-networking/wifi-router/archer-ax11000/) 路由器。这听起来好像比一种“更简单”的路由器快几个数量级，比如它的 [Archer A9](https://www.tp-link.com/us/home-networking/wifi-router/archer-a9/) ，一种 AC1900 路由器。

Archer AX11000 通过两种方式获得这个数字。首先，它允许您创建三个独立的无线网络——两个在 5GHz 频段，一个在 2.4GHz 频段。它将所有这些的最大理论速度相加(4，804；4,804;和 1，148Mbps)得到 10，756，或者 AX11000(有点四舍五入。)

听起来令人印象深刻？嗯，你需要一个无线 AX 设备来从路由器的顶级性能中受益，而你可能没有。即便如此，无线干扰、你家或公寓的设置、吞噬你吞吐量的其他设备以及你的网速——是的，又一次——意味着你可能甚至注意不到 450 美元的 AX11000 路由器和 90 美元的 AC1900 路由器之间的区别。为什么要交附加费？

### **有线吞吐量**

现在你应该关心的有线速度只有一种:千兆位，或 10/100/1000 Mbps。如果你的路由器只支持 10/100，或“快速以太网”，那就太慢了，你应该让它在商店的货架上积灰，而不是“升级”到一个。千兆连接可以让你在互联网上更快地传输数据，如果你在家里传输文件或流媒体电影，这是非常有用的。如果你碰巧购买了速度超过 100Mbps 的互联网，你将需要千兆网络来为你的有线台式机、游戏机和其他设备提供尽可能快的速度。

令人欣慰的是，无线交流路由器通常带有千兆网络，所以这甚至不应该是一个很难做出的决定——或者说任何决定，真的。

### **范围**

无线路由器只能到达这么远。如果你有一所大房子，路由器在房子的一边，你可能无法从房子的另一边访问网络。

这就是事情变得有点奇怪的地方。Wireless N *通常比 wireless AC 的覆盖范围更大，但在您的情况下可能并非如此，这取决于有多少其他设备在 2.4GHz 频段上产生干扰。由于 wireless AC 的速度比 wireless N 快得多，因此在更远的距离上，您实际上可能会体验到更好的速度，尽管 wireless N 可能会让您走得更远并保持连接。如果有疑问，请确保在路由器的最大范围内尝试 2.4GHz 和 5GHz 的无线网络。一种选择可能行得通，而另一种则行不通。* 

也就是说，有许多其他方法可以远程连接到您的网络。无线扩展器(也称为 [无线中继器](https://lifehacker.com/turn-your-old-router-into-a-range-boosting-wi-fi-repeat-5563196) )是你可以买到的产品，它们确实如你所说的那样——进一步扩展你的网络。或者，你可以 [买一个电力线适配器](https://lifehacker.com/how-to-ditch-slow-wireless-speeds-and-go-completely-wir-5782479) ，它可以让你用家里的电线把远处的设备通过以太网电缆连接到你的路由器上(从而获得比无线更快的连接)。

### **端口数量**

路由器后面有两种端口:局域网端口和广域网端口。你的广域网端口连接到你的调制解调器(也就是连接到互联网的设备)，而局域网端口连接到你的电脑和其他客户端。大多数路由器有四个 WAN 端口。如果你的有线设备多于一个路由器所能容纳的，你可以用一个 [有线开关](http://www.newegg.com/Store/SubCategory.aspx?SubCategory=30&name=Switches) 把它们都插上。交换机就像路由器的电源板:它允许你插入比路由器最初允许的更多的设备。

### **波段数**

正如我们在本文中已经提到的，无线路由器在 [无线电波段](http://en.wikipedia.org/wiki/ISM_band) 上广播，所有无线 AC 路由器(和大多数无线 N 路由器)可以在两个波段上广播。这些被恰当地称为*双频*路由器。古老的路由器只能在 2.4Ghz 频段上运行，就像旧笔记本电脑和许多智能家居设备一样，而双频带路由器可以在 2.4Ghz 频段和 5Ghz 频段上连接。

5Ghz 频段非常棒，因为它的干扰较少，因为大量其他设备——从其他网络到蓝牙、无绳电话到微波炉——都工作在 2.4Ghz 频段。5Ghz 频段的主要缺点是，因为它使用更高的频率，所以不太擅长穿透墙壁。如果您使用 5GHz，您只能实现路由器的最高无线交流速度，但根据它(和您的设备)的速度，2.4GHz 可能足以满足您的一些设备。

### **无线安全**

除非你不介意陌生人占用你的带宽并可能访问你的网络文件，否则你应该始终用密码保护你的无线网络。WPA2 是目前最安全的无线加密类型——自 2006 年以来，在带有 wifi 标志的设备上强制使用——所以如果可以的话，请确保使用 WPA2-PSK。

### **黑客能力**

如果你喜欢在路由器上刷新新固件，你最好获得一个兼容第三方固件的固件，如 DD-WRT 或 [番茄](https://lifehacker.com/turn-your-60-router-into-a-user-friendly-super-router-344765) 。确保你的路由器在 [DD-WRT 的支持设备列表](http://www.dd-wrt.com/wiki/index.php/Supported_Devices) 或 [番茄的支持设备列表](http://en.wikibooks.org/wiki/Tomato_Firmware#Supported_devices) 中，如果你想走这条路。

### **到了买新路由器的时候**

如果你有一个特别旧的路由器，你可能会阅读大量的上述信息，并决定是时候升级了。由于影响无线性能的变量如此之多，所以在购买新产品时，您只需记住两点:

*   购买一台支持您使用的设备的最快速度的路由器(或者计划很快购买*，不打算三年后购买)*
*   *从零售商那里购买一个有很好的退货政策的路由器*

*第二点似乎有点显而易见，但我认为这是至关重要的。尽管我很愿意相信人们会在他们的房子或公寓周围的各个地方对他们的新路由器的性能进行基准测试，但他们没有。大多数人都打算买个路由器，设置好基础，塞到一个不太理想的地方(比如:不是自己居住的集中地点)，过几个月感觉不是很快就烦了。既然他们花了一大笔钱买了一个新路由器，他们就会处理它——或者买一个蹩脚的扩展器来补充它。*

*当你购买一台新路由器时，把它安装在你居住的中心位置(或者尽可能靠近它)。不要把它塞在柜子里，或者用一大堆垃圾堵住它。然后，在你的房子或公寓周围走走，看看你喜欢做的任何事情的连接速度如何:在你的家庭办公室下载游戏，在你便便时观看 4K·网飞的流媒体，浏览 YouTube 或从床上抽搐，等等。 [跑点速度测试](https://lifehacker.com/how-to-make-sure-youre-getting-the-internet-speeds-your-1827763762) 。看看一切都是什么感觉。如果你对路由器的性能满意，那太好了。如果你不是，退回去，换一件更好的——最好是其他专家已经测试过并与竞争对手比较过的东西。* 

*你也可以试着走网状路线——在你家周围放置多个接入点来扩大覆盖范围。试试看一个好的路由器是否能给你你需要的速度和范围。如果没有， [你有的是其他选项](https://lifehacker.com/the-best-ways-to-boost-your-home-wifi-5931743) 。*
