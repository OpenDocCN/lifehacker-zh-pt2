# 这种复合黄油基本上是鸭子糖霜

> 原文：<https://lifehacker.com/this-compound-butter-is-basically-duck-frosting-1838492514>

鸭油和黄油是我最喜欢的两种脂肪，所以直到今天，我还没有把它们混合成一种美味的酱，这让我非常困惑。当我称之为“宏伟”时，你可能会认为我夸张了，但我向你保证，所有其他的赞美都是不够的。当你把鸭油和黄油混合在一起，搅打它，你会得到一种奶油般光滑的酱料，最好的描述是“鸭子糖霜”

鸭肉糖霜对什么有益？玉米松饼，不在松饼里的玉米，还有爆米花。除了玉米以外的食物也很好吃，比如所有的面包。你可以在里面煎鸡蛋，涂在华夫饼上，或者搅拌成意大利面。*可以捣成土豆*。我在里面蘸盐了吗？是的。我真的喜欢这次经历吗？也是的。

正如我前面提到的，需要大量的鞭打。你也要确保你的两种脂肪完全处于室温，否则你会得到悬浮在鸭油中的小块黄油，在你嘴里会感觉油腻——而不是奢侈。要做鸭肉糖霜，你需要:

*   8 盎司(2 支)室温优质盐黄油(如果你只有盐，加入 1/2 茶匙盐)
*   1/3 杯室温提炼的鸭油(Y 你可以从你的屠夫那里买到这个或者 [自己提炼](https://skillet.lifehacker.com/render-poultry-scraps-and-gain-excellent-cooking-fat-1821958410) 。)

将黄油切成 16 块(每块 8 块),和鸭油一起放入搅拌机的碗中。使用打蛋器，把脂肪打得很高，根据需要刮下两边，直到黄油颜色变浅，你就有了蓬松、光滑、有光泽的涂抹。艺术地涂抹在盘子上，用几片百里香叶装饰，享受你最喜欢的碳水化合物。
