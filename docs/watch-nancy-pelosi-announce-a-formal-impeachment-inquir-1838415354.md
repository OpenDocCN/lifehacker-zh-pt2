# 观看南希·佩洛西宣布对总统的正式弹劾调查

> 原文：<https://lifehacker.com/watch-nancy-pelosi-announce-a-formal-impeachment-inquir-1838415354>

在犹豫了几个月之后，今天下午，众议院议长南希·佩洛西宣布计划对唐纳德·特朗普总统发起正式弹劾调查。

Watch

她说:“总统迄今采取的行动严重违反了宪法，特别是当总统说，‘第二条说我可以做我想做的任何事情。...总统必须对此负责。没有人可以凌驾于法律之上。"

这一消息紧随特朗普与乌克兰的关系，以及特朗普与乌克兰总统的通话，其中涉及特朗普推动调查前副总统乔·拜登(Joe Biden)和他的儿子，以及一名举报者的更大投诉。(正如美国有线电视新闻网 报道的那样，告密者已经同意在一个未知的日期作证，但最早可能在本周。)

虽然今天早些时候 [川普已经同意](https://www.cnbc.com/2019/09/24/trump-says-he-authorized-the-release-of-complete-fully-declassified-and-unredacted-transcript-of-ukraine-call.html) 公布电话记录，但这显然没有阻止佩洛西继续调查。

如果你想看完整的演讲，CNN 在 [YouTube](https://www.youtube.com/watch?v=4lon4fZaoXI) 上有。
