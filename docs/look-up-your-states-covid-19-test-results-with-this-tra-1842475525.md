# 用这个追踪器查找你所在州的新冠肺炎测试结果

> 原文：<https://lifehacker.com/look-up-your-states-covid-19-test-results-with-this-tra-1842475525>

毫无疑问，美国的新冠肺炎 病例比检测显示的要多得多，但检测仍然是开始了解疫情扩散程度的重要方式。一个名为 [Covid 跟踪项目](https://covidtracking.com/data/) 的在线工具已经收集了每个州的测试数据，如果你想了解进展情况的话。

Watch

跟踪器覆盖所有的州加上 DC，以及美国领土，如波多黎各和美属萨摩亚。

这个追踪器是查找你所在州的数据的一种便捷方式，但是它也有很多警告。例如，各州可能使用不同的指导方针来决定测试谁，并且他们不会以相同的方式报告结果。正如你在上面的截图中看到的，每个州的数据都有一个注释。

跟踪器还保留历史数据的记录。每个州的数字截图每天都会被拍摄几次( [这里是纽约州的](https://covidtracking.com/data/state/new-york/#history) )，这样你就可以查看这些数字随着时间的推移是如何变化的。一些州将结果按县分开或提供地图；其他人只是给出几个数字或“还没有案件”的信息。

追踪器的创建者写道:“信息是不完整和不一致的，所以我们对我们发现的东西和我们如何处理它保持透明。”在目前混乱的情况下，这可能是所有人能做的最好的事情了。
