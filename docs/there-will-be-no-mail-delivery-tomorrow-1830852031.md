# 明天将没有邮件递送

> 原文：<https://lifehacker.com/there-will-be-no-mail-delivery-tomorrow-1830852031>

美国邮政局 [宣布](http://about.usps.com/newsroom/statements/120318-usps-operating-status-national-day-of-mourning.htm) 将在明天为纪念前总统老布什的全国哀悼日暂停邮件递送。全国各地的邮局也将关闭。



前总统布什于周五晚间去世，他的追悼会将于周三在华盛顿国家大教堂举行。

“我们将在那一天提供有限的包裹递送服务，以确保我们的网络保持流动，我们不会在繁忙的假期剩余时间里经历任何可能对我们的客户或商业伙伴产生负面影响的包裹递送业务，”邮政服务于 [在其网站 e](https://go.skimresources.com/?id=33330X911647&xs=1&isjs=1&url=https%3A%2F%2Fwww.usps.com%2Fnational-day-of-mourning%2Fwelcome.htm&xguid=65fad53de6fdf065319486e151e1c074&xuuid=544eb081eeb89af963240cd03f859acb&xsessid=&xcreo=0&xed=0&sref=https%3A%2F%2Flifehacker.com%2Fthere-will-be-no-mail-delivery-tomorrow-1830852031&pref=https%3A%2F%2Fkinja.com%2Fdashboard&xtz=300&jv=13.12.1-stackpath&bv=2.5.1) 上宣布。

为了纪念已故总统，股票市场也将关闭。
