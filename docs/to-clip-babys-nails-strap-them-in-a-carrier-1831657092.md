# 要修剪婴儿的指甲，把它们绑在一个托架上

> 原文：<https://lifehacker.com/to-clip-babys-nails-strap-them-in-a-carrier-1831657092>

如果你在为剪孩子的指甲而挣扎，你并不孤单。这是一项必要的、经常性的任务，令许多父母害怕。我们第一次在控制我们的小女儿锋利的小爪子上失败得如此悲惨，以至于我手里拿着剪刀出现在她一个月大的检查上，请求护士和儿科医生帮助我；他们拒绝了。



当我们的第二个孩子在 NICU 度过一段时间时，我再次希望超级细心的工作人员会处理好这件事，但唉，这是他们没有也不会做的一件事。对于新生儿和熟睡者来说，这个问题很容易解决:当你的宝宝打盹的时候，小心地剪头发。但是，一旦这些流氓变得更加活跃和抗拒，你就可能遇到真正的麻烦——特别是如果像我 14 个月大的儿子一样，你孩子的能量水平可以与一场不间断的全面龙卷风相媲美。

出于彻底的绝望，在做了十年的父母后，我终于发现了一个让我们看起来不像是被猫拖进来的东西的窍门。为了保持指甲美观整齐，我找到了一个绝对有效的指甲修剪方法，只需要几分钟就可以轻松搞定。

把你的小老虎绑在婴儿背带里，面朝外，剪下来。虽然孩子们仍然可以非常自由地挥动他们的胳膊和腿，但是全身的安全带使任何抵抗不仅无效，而且非常容易管理。

在我使用这个技巧的经验中，孩子已经足够分心四处张望了，就这一次，他们不在乎你在对他们的手指甲或脚趾甲做什么。
