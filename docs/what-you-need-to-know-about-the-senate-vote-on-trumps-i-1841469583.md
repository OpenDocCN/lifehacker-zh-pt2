# 关于参议院对特朗普弹劾案的投票，你需要知道什么

> 原文：<https://lifehacker.com/what-you-need-to-know-about-the-senate-vote-on-trumps-i-1841469583>

本周在政治方面，爱荷华州的党团会议进行得很顺利，特朗普总统在昨晚的国情咨文中吹嘘说，经济是“有史以来最好的”。

另一条来自《颠倒报》的消息，今天下午，特朗普的参议院弹劾审判将在民主党和共和党领导人之间几个月的反复之后迅速结束。好奇发生了什么事，以防你已经完全忘记了或者只是不能保持直线？下面，回答你关于特朗普总统弹劾现状的问题。

### 我迷路了。最近发生了什么事？

很多。简而言之，在过去的两周里，参议院的日程非常繁忙，其中包括听取民主党经理和 [川普的辩护团队](https://www.npr.org/2020/01/28/800559407/trump-lawyers-wrap-up-opening-arguments-in-senate-impeachment-trial) 的开场辩论，一场由参议员 准备的 [书面问题会议，一场关于是否传唤证人](https://www.nytimes.com/2020/01/30/us/politics/impeachment-trial-trump.html) 作证的 [辩论(就像前国家安全顾问约翰博尔顿一样)，以及一系列的](https://www.nbcnews.com/politics/trump-impeachment-inquiry/senate-vote-calling-witnesses-fails-ushering-trial-endgame-n1127966) [结案辩论](https://www.washingtonpost.com/politics/2020/02/03/5mf/)

### 哦该死的。那么今天发生了什么？

今天早上，包括俄勒冈州参议员杰夫·默克利和参议院少数党领袖查克·舒默 [在内的领导人在参议院投票决定是否判川普总统有罪或无罪释放他滥用权力和妨碍国会的指控之前发表了闭幕词](https://www.cnn.com/politics/live-news/trump-impeachment-trial-02-05-20/index.html) 。根据宪法，罢免总统需要三分之二的多数票——但不要屏住呼吸。共和党在参议院占据多数席位，川普预计将被无罪释放*主要是*按照党派路线，正如 [NPR 写](https://www.npr.org/2020/02/05/801695409/watch-live-trump-on-brink-of-acquittal-by-u-s-senate) 。

### 特朗普在国情咨文中有没有提到他被弹劾的问题？

没有，但他确实谈了很多关于股票市场、创造就业机会和经济的问题，还批评了全民医疗保险和为“非法移民罪犯”提供庇护的美国城市。他演讲后不久，众议院议长 [南希·佩洛西](https://www.cnn.com/2020/02/05/media/nancy-pelosi-speech-rip/index.html) 撕掉了他的演讲稿。(如果你想阅读文字稿，纽约时报的有一份完整的副本。)

### 我想看投票。有办法让我播放吗？

是的，将在美国东部时间下午 4 点开始流式投票。 [CNN](https://www.cnn.com/politics/live-news/trump-impeachment-trial-02-05-20/index.html) 也全天提供实时更新。
