# 请在烹饪排骨之前去掉薄膜

> 原文：<https://lifehacker.com/please-remove-the-membrane-before-cooking-ribs-1844052342>

就多肉烹饪 项目而言，排骨是一个相当简单的项目。无论你是吸烟、烧烤、烘烤还是腌制，准备一个令人印象深刻的架子是大多数家庭厨师可以轻松掌握的事情。但是有一件事许多新手排骨厨师忘记做，它可以大大破坏否则愉快的排骨吃的经验。

如果你看一排排骨不太多肉的那一面，你可能会注意到它有点发亮——就像有一层薄薄的纱布覆盖在肉和骨头上。那是因为有，有点。是一层膜(叫腹膜，yum！与胶原蛋白和结缔组织不同，它在烹饪时不会软化。

当打开时，薄膜就像一块非常薄的耐嚼的塑料包装，用它的韧性破坏你的肋骨体验。L 留着它是对食客的一种犯罪，无论什么动物为了给你它的排骨而死。幸运的是，这层膜很容易移除。你甚至不需要一把刀。(虽然我*曾*见过有人试图剪掉它，真的没有理由让刀片卷进去。)

要移除肋骨膜，你只需要双手和纸巾。将架子翻转过来，让多肉的一面朝下，然后，从肋骨的较小一端开始，用指甲将闪亮的膜与多肉的骨头分开。一旦你剥了一小块，用纸巾抓住它，用另一只手按住架子较小的一端，然后拉。薄膜会马上脱落，你的肋骨会因此变得更好。
