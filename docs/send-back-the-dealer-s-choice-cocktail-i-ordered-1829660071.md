# 问咸味女服务员:我点的“经销商之选”鸡尾酒可以退吗？

> 原文：<https://thetakeout.com/send-back-the-dealer-s-choice-cocktail-i-ordered-1829660071>

> 亲爱的咸味:最近在一家不错的鸡尾酒吧，我让酒保给我做了一份菜单外的饮料，以波旁酒为基酒。我最后喝了一杯对我来说太甜的鸡尾酒。在这种情况下，我只是完成了一半并支付了费用，但我想知道——我会不会因为对它不满意而要求其他东西，或者我实际上是因为点了“经销商的选择”而坚持要它？
> 
> 谢谢，对波本威士忌感到失望

亲爱的沮丧的，

我对此犹豫不决。一方面，你支付了 12 美元或这种鸡尾酒的任何费用，所以你应该得到一杯你满意的饮料。另一方面，你孤注一掷，让吊裤带麦克吉格掌管你的酗酒命运，没有任何保证。你做了你的床，但是你在床上睡觉要付钱吗？

Watch

为了记录在案，我假设你是一个正常的鸡尾酒顾客，而不是一个过分挑剔，公主和豌豆，没有什么是永远完美的，足够的屁股痛。我猜你点了经销商的选择，因为你喜欢鸡尾酒，想尝试一些新的东西，或者因为你已经尝试了菜单上的其他波旁酒，想混合起来。

假设你是一个通情达理的人——不要证明我错了，伙计——我认为给老吊带裤一些反馈是公平的。这就是我这么说的原因:菜单外或经销商选择的饮料并不像听起来那么即兴。很多时候，调酒师利用这些时间来尝试他们可能想添加到菜单中的饮料，或者试驾对经典配方的调整。如果我是背带裤，我会想知道我的新食谱是一个哑弹，还是我的老式即兴重复烂透了。

但是要有礼貌——你知道，你用蜂蜜抓到更多的蜜蜂。问一问饮料里还有什么，并利用这个机会说一些像“哦，我想要一些不那么甜的东西。”运气好的话，吊裤带会领会这个暗示，提供一些别的东西。如果没有，打电话问问酒吧能不能给你做点别的。(不用说，只有当你在之前问*你已经喝了 3/4 的鸡尾酒时，这个方法才有效。)*

希望你的酒保不会对这个反馈太苛刻。我遇到过一些自命不凡的“调酒师”，他们认为自己调制的每一种鸡尾酒都表达了某种宏大的愿景。呸。他们从事酒类和招待行业，这意味着让客人开心。如果一个酒保不愿意冒险去赢得你的信任，那么他们就不应该尝试经销商的选择。
