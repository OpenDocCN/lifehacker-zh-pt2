# 用这个免费的 Windows 应用程序替换文件资源管理器

> 原文:[https://life hacker . com/replace-file-explorer-with-this-free-windows-app-1844168762](https://lifehacker.com/replace-file-explorer-with-this-free-windows-app-1844168762)

这些年来，Windows 的文件资源管理器得到了发展和扩展，但核心功能基本保持不变。它还是和以前一样又笨又丑。

Watch

微软正计划在即将到来的 Windows 10X 版本中给文件浏览器一些 TLC，但那些想要一个精简的——或者只是更好的——外观的——选项现在可以转向第三方替代方案，如 [**文件 UWP**](https://www.microsoft.com/en-us/p/files-uwp-preview/9nghp3dx8hdx?) 。

公平的警告: Files UWP 仍处于测试阶段，所以通常的警告在这里适用；该应用程序可能不会总是像它应该的那样工作，可能会出现一些错误，一些功能仍在开发中。但即使处于不完整的状态，UWP 文件也有很多可以提供的，尤其是作为一个免费的应用程序。

### 连接

文件 UWP 是精简和易于使用，但我注意到的第一件事，当我加载它是多么好的应用程序看起来。与其说是华而不实；相反，它看起来像一个原生的 Windows 10 应用程序。我甚至可以说，它比微软的文件浏览器更好地融入了 Windows 10 的用户界面。这有助于让 UWP 感觉文件交换更加自然。

在功能上，UWP 文件看起来像是 Windows 10 系统菜单和 Edge Chromium 的混合体。左侧菜单包含一个熟悉的文件夹快捷方式列表，包括:

*   桌面
*   文档
*   下载
*   图片
*   音乐
*   录像
*   本地驱动器
*   OneDrive
*   以及具有常用和最近打开的项目的快捷方式的“主页”屏幕。

当你在“文件”“UWP”中打开一个新标签时，你会看到这个主屏幕，这是与“文件浏览器”最大的区别。“文件 UWP”允许你在一个应用程序窗口中以标签的形式打开多个文件夹，比“文件浏览器”更像一个网络浏览器。这使得桌面更有条理(尽管我发现多窗口方法更适合我，因为我每天都要在多个文件夹中输入和输出文件)。也就是说，在“文件 UWP”标签之间移动文件一点也不难，即使不优雅，也可以快速拖动文件。

### 其他功能

当谈到基本的文件管理，UWP 文件做你所期望的一切，但有些功能出现在不同的位置比你可能习惯。

例如，右键单击可以移动、复制、共享、删除、重命名和查看文件和文件夹的属性，但您需要单击应用程序窗口右上角的 **"+"** 来创建新文件夹。这需要一点时间来适应，但该按钮还包括制作新文本文档或新图像文件的快捷方式，这是一个很好的补充。

还有一个“布局模式”按钮，可以让您快速调整文件夹中文件的外观和顺序。坦率地说,“文件 UWP”实际上以一种逻辑的方式列出了下载文件夹的内容，而不是“文件浏览器”喜欢的怪异方式(至少在我的笔记本电脑和 PC 上),这一事实足以说服我继续安装这款应用程序——即使“文件 UWP”在一些方面受到限制，需要一些时间来适应。

一个让我印象深刻的问题是必须按住 shift 键或 CTRL 键来选择多个文件。虽然有一个快捷的“选择”菜单，可以让您快速全选、反转选择或清除选择，但您不能像在文件资源管理器中那样单击并拖动。你也找不到在 WinRar、Photoshop 等软件中打开文件的内置快捷方式或快速选择选项。哦，UWP 文件加载有点慢，缺乏一些文件浏览器的更深入的搜索功能。

也就是说,“文件 UWP”已经是“文件浏览器”的一个更时尚、更好看的替代品——而且是免费的。它可能不会立即取代文件资源管理器，但潜力是存在的，并且随着开发的继续，它将成为一个可靠的补充。你可以从微软商店 下载并亲自试用 [。](https://www.microsoft.com/en-us/p/files-uwp-preview/9nghp3dx8hdx?)
