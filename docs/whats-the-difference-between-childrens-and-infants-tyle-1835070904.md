# 儿童和婴儿的泰诺有什么区别？

> 原文：<https://lifehacker.com/whats-the-difference-between-childrens-and-infants-tyle-1835070904>

如果你曾经不得不为一个小家伙买止痛药，你可能已经注意到有分开的“儿童”和“婴儿”产品瓶。在对乙酰氨基酚(泰诺)的情况下，瓶子的内容是相同的。

不过，它们以前是不同的。在 2011 年之前，婴儿版本是指用滴管定量，而儿童版本是用勺子定量。这意味着婴儿的版本更集中。婴儿的 [被改为与儿童的](https://www.fda.gov/drugs/drug-safety-and-availability/questions-and-answers-important-change-concentration-over-counter-otc-liquid-acetaminophen-marketed#q2) (160 毫克/5 毫升)相同的剂量，所以现在你不能把两者混在一起，给错剂量。

因此，儿童和婴儿的对乙酰氨基酚现在是一回事。NPR [报道](https://www.npr.org/sections/health-shots/2019/05/27/726327937/tylenol-for-infants-and-children-is-the-same-why-does-1-cost-3-times-more) 婴儿扑热息痛的价格仍然是原来的三倍，据称是因为婴儿的包装更贵。但即使你更喜欢滴管，你也可以通过重新填充更便宜的儿童版本来节省几块钱。
