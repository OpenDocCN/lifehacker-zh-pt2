# 孩子们也需要工作空间人体工程学

> 原文：<https://lifehacker.com/kids-need-workspace-ergonomics-too-1844944467>

不再只有成年人会一次在电脑屏幕前呆上几个小时；随着本学年孩子们在家学习的时间增加，通常是使用 Chromebooks 或其他笔记本电脑，确保它们尽可能设置在最佳的人体工程学位置非常重要。在年轻时养成良好的习惯将会减少现在和将来发生疼痛或受伤的可能性。



### 基础知识

适用于成人的基本人体工程学原理同样适用于儿童。一个中性的姿势是最好的，在这个姿势中，肌肉、骨骼、神经和血管受到的压力最小。这意味着坐在一起:

*   他们的脚平放在地板上(或者放在凳子或一摞书上，如果你孩子的脚够不到地板的话)
*   椅子抬高，使他们的膝盖与臀部大致齐平，膝盖后部舒适地靠在椅子的前边缘
*   显示器与眼睛平齐，距离大约一臂远
*   他们的背部由椅背(或椅背和椅子之间的枕头)支撑，直到肩胛骨
*   前臂与地面平行，手腕弯曲最小

一个很好的经验法则:如果它看起来很笨拙，它可能不符合人体工程学。

### 你不需要昂贵的设备

无论你看到你的孩子有什么姿势问题，比如无精打采，他们的脚不能平放在地板上，或者笔记本电脑太低，不能与他们的眼睛平齐，都可以用家里的小道具来解决。职业治疗师斯泰西·拉姆费尔特向 [通心粉小子](https://southcharlotte.macaronikid.com/articles/5c7d71ae4ef51723cbe86478) 解释道:

> 如果孩子懒散地坐在椅子上，枕头可以用来支撑他们。把枕头放在背后可以让他们在座位边缘坐直。除了枕头，一堆书可以把他们的笔记本电脑提升到更高的屏幕位置，以防止懒散。[I] 还推荐一种能够减少颈部弯曲的文件架。

> 如果你的孩子的脚不能接触到地板，盒子、板条箱或一摞书等物品可以很快解决这个问题。建立家庭作业站的目的是保持一个中立的姿势，以尽可能减轻颈部、背部和四肢的压力。

她建议首先尝试这些简单、便宜的方法，如果姿势问题持续存在，只使用更昂贵的工具，如儿童尺寸的键盘和鼠标。

### 提醒他们起床活动

别忘了孩子的身体是需要运动的。职业治疗师 Meredith Chandler 为人类工程学健康协会写道:

> 除了主动无精打采，只要有运动，孩子坐着就没有“错误”或“正确”的方式。
> 
> 很少有孩子会在学校里有目的地整天坐在一个严格的 90-90-90 的位置上，因为他们天生就适合运动。
> 
> 只要坐立不安或移动不经常打断学术学习，那么教育者应该允许孩子在学校里到处走动。

所以这是虚拟学习的一个好处——除非他们现在正在和其他学生打电话，否则再多的坐立不安或起身在客厅里走几圈都不会打扰他们的同学。鼓励孩子经常休息，随着音乐跳舞，原地跳跃，或者跳过后院。
