# 为什么您应该使用 Windows Defender 的勒索软件预防

> 原文：<https://lifehacker.com/why-you-should-use-windows-defenders-ransomware-prevent-1837311176>

默认情况下，Windows Defender 中的一个重要安全功能被禁用:勒索病毒防护。这令人惊讶，因为 [勒索软件是一个严重的威胁](https://lifehacker.com/a-crash-course-in-ransomware-and-how-to-protect-yoursel-1783372708) 它锁定你的设备，阻止你访问你的数据，直到你付钱给攻击者。当你考虑到勒索软件保护早在 2017 年 10 月的 Windows 10 更新中就已添加，这就更令人惊讶了。

Watch

对于为什么勒索病毒防护被默认禁用， [正如 gHacks，](https://www.ghacks.net/2017/10/24/configure-controlled-folder-access-in-windows-10/) 所指出的，最好的解释是该功能可能容易出现误报。这是一个合理的担忧，但鉴于勒索软件攻击的破坏性，我们同情那些急于保护自己数据的人。

在 之前，我们已经介绍过 [一般的勒索软件防范技巧，但是 Windows 10 的内置选项为你的电脑增加了一层额外的保护。幸运的是，Windows Defender 不仅包括勒索软件保护，用户还可以根据自己的需求扩展或限制其覆盖范围。方法如下:](https://lifehacker.com/protect-yourself-against-new-android-ransomware-that-pr-1836824864)

1.  打开开始菜单，然后单击设置图标。
2.  在设置菜单中，进入**更新&安全> Windows 安全>病毒&威胁防护**。
3.  向下滚动到勒索软件防护，点击**“管理勒索软件防护”**
4.  在下一个菜单中，启用**“受控文件夹访问”**
5.  默认情况下，受控文件夹访问仅保护某些文件夹:文档、图片、视频、音乐、桌面、收藏夹。您可以通过在同一窗口中单击“添加受保护的文件夹”来将勒索软件保护扩展到其他文件和文件夹。
6.  导航到要添加到勒索软件防护的文件夹。对每个要保护的文件夹重复上述步骤。

现在您已经设置了受控文件夹访问，Windows Defender 将监视哪些程序正在访问受保护的文件夹以及其中存储的文件。然后它会阻止可疑程序试图访问。虽然这让您放心，但问题是一些“可疑”程序实际上并没有恶意。为了避免误报，您可以将程序添加到“受控文件夹访问”白名单中。

1.  打开“开始”菜单，然后单击“设置”图标。
2.  进入**更新&安全> Windows 安全>病毒&威胁防护>管理勒索软件防护**。
3.  向下滚动并点击**“允许应用程序通过受控文件夹访问，**”，然后找到所需程序并将其添加到列表中。你必须对每个你想授予访问权限的应用程序重复此过程。
4.  您还可以单击**“阻止历史”**查看 Windows Defender 已阻止访问您的受保护文件的程序列表。如果您无法识别列出的程序，或者不确定为什么会有人试图访问您的文件，请卸载该程序。

即使 Windows Defender 提供了微调的勒索软件保护，您也应该确保定期备份您的文件(这样，如果勒索软件来袭，您就不会被锁定任何重要内容)。例如，您可以使用外置硬盘，或者通过 OneDrive 设置数据恢复，这可以在受控文件夹访问设置中启用，就在我们之前讨论过的同一个“勒索软件防护”屏幕上。如果你还没有设置 OneDrive，你将需要设置 one drive，并且你只能获得 5GB 的可用空间，但如果遇到勒索软件攻击，你将能够恢复你的关键数据。
