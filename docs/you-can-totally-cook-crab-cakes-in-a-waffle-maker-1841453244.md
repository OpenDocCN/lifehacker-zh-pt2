# 你完全可以用华夫饼干机做蟹饼

> 原文：<https://lifehacker.com/you-can-totally-cook-crab-cakes-in-a-waffle-maker-1841453244>

蟹肉蛋糕最初看起来并不像是一个很好的 waffling 候选人。毕竟，好的蟹肉是尊重蟹肉的美味的。如果你花时间购买了一磅所说的肉来制作你自己的蟹饼，这不是你所寻求的烹饪方法；如果你碰巧在海鲜部看到一些预制的蟹饼，那么 waffling 是一条很好的路径。

你看，许多商店制作的蟹饼除了螃蟹之外还含有大量的材料——主要是粘合剂——这使得它们比真正的蟹饼更有“蟹味油条”的味道。他们很好。我会吃它们，不会抱怨，但它们并不能真正满足我对蟹肉蛋糕的渴望。

但这是平庸可以转化为力量的例子之一(这是我正在做的事情)。一个五美元的蟹饼并不是什么珍贵的东西，你应该毫不犹豫地把它夹在热华夫饼干机的盘子里，让它变成一个海鲜型的碳水化合物平台，有着最脆的波峰和波谷。这是一道很好的开胃菜，但你能想象它作为一个本尼迪克特的基础吗？这个世界可能还没有准备好。

不像我其他大多数糊涂的儿子，这只需要稍微低一点的烹饪温度，以免烤焦外面。许多准备好的蟹饼会沾上一些健康的面包屑或面包屑，而且会很快变黄。把你的熨斗调到中火，在盘子之间打碎蛋糕，每隔几分钟检查一下，直到它变成棕色，可以完整地从熨斗上取下(用筷子把它从凹槽中取出)。配上柠檬，一些柠檬蘸酱，或者在上面做你最好的早午餐。
