# 什么是红潮？

> 原文：<https://lifehacker.com/what-is-a-red-tide-1829710196>

自去年以来，有毒的“赤潮”一直影响着佛罗里达州的海滩，现在情况尤其糟糕。我们在地球的朋友深入报道这个主题，但是如果你对什么是赤潮以及为什么每个人都在谈论它感到困惑，我们有你基本问题的答案。

### 什么是赤潮，为什么这么叫？

“红潮”实际上是一群漂浮在水中的微小生物。海洋中总是充满了微生物，但当我们谈到赤潮时，我们会想到某些物种，它们有时看起来是红色的，可以用它们自然产生的毒素充满海水。美国佛罗里达州的这次赤潮是一种叫做[](https://en.wikipedia.org/wiki/Karenia_brevis)*的物种造成的。*

*每个个体都是一个单细胞，有一条用来游泳的小尾巴。它可以像植物一样利用阳光制造食物，所以它被认为是一种藻类。因为它是海洋中的微小生物，它也是浮游生物的一种。*短吻海葵*在显微镜下是褐色的，但是当海水中充满了短吻海葵时，我们可以看到海水略带红色——或者 T5】有时是黄色或者棕色 T7】。*

*其他无害的藻类也能把水变成有趣的颜色，所以与其说是“赤潮”，不如说是“有害的藻华”。*

### ***是什么原因导致它“开花”？***

*藻华是大量的藻类——数以百万计的微小生物充满了特定区域的海水。当条件合适时，包括水中大量的营养物质，藻类繁殖。想象一下，如果每次你走进一个吃到饱的自助餐，你就可以生一堆孩子，这样你的家人就可以占领餐厅，吃掉里面的所有东西。*

*短吻海葵每年都会在海洋中开花，并被冲上佛罗里达的墨西哥湾海岸。这不是那种伴随肥料流失的水华——这是目前影响佛罗里达州的*另一种*水华(是的，他们有两种)。但是赤潮可能会以另一个为食，化肥可能会发挥作用。也就是说，通常是其他因素，包括洋流，导致每年佛罗里达水域出现赤潮。 *K. brevis* 依靠二氧化碳生长，喜欢温暖的水，所以气候变化可能是今年水华变得特别糟糕的部分原因。*

*今年的花开始于去年 10 月，并没有像往年一样在今年春天放晴。很有可能情况会变得更糟，因为赤潮季节又开始了。*

### *它到底有多毒？*

**K. brevis* 制造一种被称为 [brevetoxin](https://en.wikipedia.org/wiki/Brevetoxin) 的神经毒素(一种影响神经和大脑的毒素)。这种毒素可以杀死动物，包括鱼类和鸟类。在最近的赤潮中，海牛和海龟也被发现死亡。*

*吃了被毒素污染的贝类的人会患上 [神经性贝类中毒](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC2579735/) ，所以不要从受赤潮影响的地区采集或食用贝类，这一点很重要。这种毒素无嗅、无色、无味，不会被烹饪破坏。症状包括恶心、呕吐和腹泻，以及面部、手和脚的刺痛或麻木。*

*我们也可以吸入毒素。当充满藻类的海浪冲击海滩时，它们会破坏一些藻类细胞，并将它们的毒素释放到空气中。患有哮喘或其他肺部疾病的人风险最大，但健康人也会受到影响。你会注意到的症状是呼吸困难，也许还有眼睛发炎。*

*受赤潮影响的海滩通常会张贴一个标志，警告呼吸该地区的空气，特别是如果你有哮喘等肺部疾病。水里可能全是死鱼，所以不管怎样，你也不想在那里游泳。*

### *飓风让情况变得更好还是更糟？*

*现在下定论还为时过早，但美国全国广播公司报道称，许多地区的水样中的 T4 短杆菌数量有所减少。据一位与他们交谈过的生物学家说，飓风可能“像搅拌机一样”混合了海水，稀释了赤潮生物。*

*另一方面，飓风导致的洪水可以将营养物质推入大海，供短臂海葵食用。 [史密森尼报告](https://www.smithsonianmag.com/smart-news/hurricane-michael-could-worsenor-alleviatefloridas-toxic-red-tide-outbreak-180970521/)2004 年四次飓风袭击佛罗里达州后，赤潮增加，但卡特里娜飓风过后，赤潮减少。所以我们只能等等看。*
