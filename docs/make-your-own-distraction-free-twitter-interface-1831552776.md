# 制作你自己的不受干扰的 Twitter 界面

> 原文：<https://lifehacker.com/make-your-own-distraction-free-twitter-interface-1831552776>

Twitter，一个让你对世界和自己感觉糟糕的地方，仍然对一些事情有用。你可能只会在上面关注他们真正的朋友，或者把它作为一个新闻聚合器，或者一个关注收集艺术、音乐或有趣视频的账户的地方。但 Twitter 会非常努力地让你关注更多的人，“发现”更多的“趋势”，并点击广告。以下是如何把自己不喜欢的部分剪掉。

Watch

作家 John Pavlus 构建了一个自定义样式表，隐藏了很多 Twitter 界面。你也可以使用它，如果你遵循 Pavlus 网站 :
的这些说明

1.  为 Chrome 安装手写笔扩展
2.  安装 Pavlus 的样式表， [驯服 Twitter](https://userstyles.org/styles/156933/tame-twitter)
3.  从扩展菜单中打开手写笔，并编辑样式表

正如该网站解释的那样，你可以恢复某些元素，但默认情况下，它会屏蔽很多元素，甚至是推文、赞或转发的功能。(以上，我手动重新启用了几个功能。)所以如果你愿意的话，你可以把 Twitter 变成一个没有互动性的纯链接博客，不分心不分心。
