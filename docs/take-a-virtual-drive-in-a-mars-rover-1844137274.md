# 乘坐火星漫游车进行虚拟驾驶

> 原文：<https://lifehacker.com/take-a-virtual-drive-in-a-mars-rover-1844137274>

如果你曾经渴望成为现实生活中的马克·瓦特尼——收集火星上的岩石和泥土样本——现在是你的机会了。借助美国宇航局的 [AI4Mars 项目](https://www.zooniverse.org/projects/hiro-ono/ai4mars) ，你可以乘坐好奇号火星车在红色星球上进行一次虚拟旅行。



由美国宇航局喷气推进实验室的一个团队设计的模拟程序，要求用户 [帮助分类火星地形](https://www.zooniverse.org/projects/hiro-ono/ai4mars/classify) 。你将标记好奇号从火星表面拍摄的图像(例如，沙子或基岩)，科学家将使用众包数据训练未来的火星车更好地绕过危险的障碍。

火星漫游者有陷入沙子的倾向——有时无法自己出来——所以该项目旨在使漫游者像自动驾驶汽车一样。为了在没有人类船员的情况下探索太空，这些车辆必须知道“在哪里开车、着陆、睡觉和冬眠是安全的。”

当您打开分类工具时，系统将指示您选择不同的表面类型(岩石或土壤)，并使用包含您选择的类型地形的绘图工具标注表面图像的区域。一旦你确定了图像中的所有内容，选择“完成”进入下一张照片。

有一个有用的教程弹出窗口(点击“教程”)和一个讨论板，您可以在这里提出问题或对该工具进行故障诊断。

同样的众包概念已经被用于训练地球上城市景观和房屋号码的神经网络。

需要澄清的是，你不会真的驾驶好奇号在火星表面飞行，甚至虚拟的——这是而不是视频游戏。但是你*将*近距离观察火星表面，同时为科学发现做出贡献。
