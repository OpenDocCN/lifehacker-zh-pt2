# 你可以用 FSA 的钱做 23andme 的 DNA 测试

> 原文：<https://lifehacker.com/you-can-use-fsa-money-for-23andme-dna-tests-1839775389>

如果你一直在考虑进行 DNA 测试，为你提供健康报告，好消息是:美国国税局已经裁定，你可以用你的 FSA 资金支付部分费用。



今年早些时候发布的裁决 指出，祖先 DNA 测试不被视为医疗保健，但如果测试提供了健康信息(如 23andme 的健康+祖先服务)，一定比例的费用可以从您的医疗保健灵活支出账户中支付。

23andme 想要 FSA 的钱，所以他们计算了一下数据，他们想让你知道，在 199 美元的测试费用中，117 美元用于支付测试的医疗费用。如果你以折扣价购买了测试，或者如果你已经购买了血统服务并支付了健康升级费用， [他们有一个方便的计算器](https://www.23andme.com/fsa-hsa-eligibility/calculator/) 来确定有多少费用是符合 FSA 的。然后，您可以打印适合提交报销的收据。

到目前为止，美国国税局只对 23 项 DNA 测试做出了具体裁决，但这项裁决表明，人们可以认为其他与健康相关的 DNA 测试也应该有资格。

根据该裁决，国税局的“医疗护理”概念是宽泛的，包括“如在没有医生建议的情况下，对没有疾病症状的个人进行全身扫描。”大多数情况下，消费者 DNA 测试在医学上没有用处；即使 23andme 测试表明你有某种疾病的遗传风险，你仍然应该进行临床等级测试 [以查明这是否是真的](https://vitals.lifehacker.com/mail-order-genetic-tests-may-turn-up-a-lot-of-false-pos-1824126192) ，并且可能还应该与你的医生和遗传顾问谈谈。
