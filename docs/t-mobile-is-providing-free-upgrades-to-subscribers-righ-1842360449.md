# T-Mobile 现在向用户提供免费升级

> 原文：<https://lifehacker.com/t-mobile-is-providing-free-upgrades-to-subscribers-righ-1842360449>

冠状病毒的传播不仅仅影响公共健康和全球经济；它还会影响互联网和蜂窝网络。如此多的人被困在家里，其他人因旅行禁令而流离失所，数据使用量上升，带宽堵塞，服务在世界各地中断。



所有“四大”移动运营商都试图通过免除账单滞纳金和承诺不会因无力支付而终止服务(多亏了冠状病毒)来从经济上帮助客户。然而，如果你是 T-Mobile 的用户，你会得到一些额外的好处。

首先，T-Mobile 正在扩展其 600 MHz 频谱，以应对网络使用量的增长。这也包括农村社区的客户。此外，T-Mobile 客户将在未来至少 60 天内获得以下免费服务升级:

*   所有 T-Mobile 和 Metro by T-Mobile 的数据计划正在升级为无限智能手机数据(不包括漫游数据)。
*   T-Mobile 和 Metro by T-Mobile 将获得额外的 20GB 移动热点或网络共享数据。

在此期间，T-Mobile 也在拓展一些特殊项目。这包括:

*   使用 T-Mobile 授权程序的学校和学生将在未来 60 天内每月获得至少 20GB 的数据。
*   T-Mobile 上的 Lifeline 用户将在未来两个月内每月获得额外的 5GB 免费数据。
*   所有当前 T-Mobile 和 Metro 的 T-Mobile 客户均可免费拨打第三级受影响国家的电话。

最后，T-Mobile 还通过授予 Sprint 用户未来 60 天的免费漫游权限来帮助被疫情取代的 Sprint 客户 。这是一个重要的举措，因为区域隔离和旅行禁令已经使成千上万的旅客滞留在非本地地区。
