# 那么，究竟什么是政变呢？

> 原文：<https://lifehacker.com/so-what-is-a-coup-anyway-1838703704>

周二晚上，面对众议院民主党人对弹劾调查越来越大的压力，川普总统在推特上说，对 T2 的调查实际上是一场“政变”

“随着我每天了解越来越多，我得出结论，正在发生的不是弹劾，而是政变……”何 [写了](https://twitter.com/realDonaldTrump/status/1179179573541511176) 。"...意图剥夺人民的权力，他们的投票，他们的自由，他们的第二修正案，宗教，军事，边境墙，以及他们作为美利坚合众国公民的天赋权利！”



### 什么是政变？

政变(coup)是政变(coup)的缩写，通常用来指突然非法夺取政府权力。美国历史上只发生过一次政变——1898 年的 [威尔明顿种族暴动](https://www.theatlantic.com/politics/archive/2017/08/wilmington-massacre/536457/) ，当时一群白人至上主义者推翻了当地政府，杀害了数十名黑人居民。

在这种情况下，弹劾质询是 而不是非法的，也不是突然的。(正如我们在 之前写的 [，这是一个需要几个步骤的宪法程序，包括众议院和参议院的投票和审判。)鉴于目前的参议院构成，特朗普总统实际上不太可能下台。正如我们在](https://lifehacker.com/how-to-impeach-a-president-1834123053) [Splinter](https://splinternews.com/impeachment-is-not-a-coup-1838698183) 的朋友所说，真正的政变少了很多繁文缛节。

### 特朗普以前用过这个词吗？

在 [二月](https://www.newsweek.com/trump-tweets-coup-impeachment-civil-war-fox-news-1462555) 中，川普第一次用这个词引用福克斯新闻频道的肖恩·哈尼蒂；那个月，汉尼蒂指责前联邦调查局局长安德鲁·麦凯布质疑川普与俄国的关系，从而激怒了 T4。在一条使用汉尼蒂语言的 [推特](https://twitter.com/realDonaldTrump/status/1097690731882446849?ref_src=twsrc%5Etfw%7Ctwcamp%5Etweetembed%7Ctwterm%5E1097690731882446849&ref_url=https%3A%2F%2Fwww.newsweek.com%2Fdonald-trump-tweets-quote-sean-hannity-says-andrew-mccabe-committed-treason-1335029) 中，特朗普将麦凯布的行为称为“政变”。

今年 4 月，特朗普在接受肖恩·哈尼蒂采访时再次使用了这个词，谈到了福克斯新闻频道对 2016 年大选的影响。“这是企图推翻美国政府，”他告诉汉尼提。"...我认为它远远大于水门事件。”( [Politifact](https://www.politifact.com/truth-o-meter/statements/2019/apr/29/donald-trump/was-donald-trump-target-coup-no/) 认定，既然穆勒的调查没有越过法律界限，那就不是政变企图。)

据《新闻周刊》 报道，川普至少在其他五个场合使用了政变，要么是引用保守派的话，要么是关于他与俄罗斯的所谓关系(或两者都有)。

### 特朗普通过使用这个词的目的到底是什么？

从他的推文来看，他正试图论证弹劾调查的非法性(尽管到目前为止没有什么是非法的)。如果他认为政变是企图夺取他的权力，他是对的，但他要么是误导了他的追随者，要么是故意将宪法规定的弹劾程序定义为反对总统的非法行为。
