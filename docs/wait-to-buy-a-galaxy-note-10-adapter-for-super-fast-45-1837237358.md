# 等下买个 Galaxy Note 10+适配器超快 45W 充电

> 原文：<https://lifehacker.com/wait-to-buy-a-galaxy-note-10-adapter-for-super-fast-45-1837237358>

为手机购买新充电器可能会令人困惑。通常情况下，你可以使用你的设备附带的任何东西，但它可能不是一个可以给你最快最好充电速度的适配器。这些通常更贵，你的智能手机制造商不会那么慷慨。



三星即将推出的 Galaxy Note 10+就属于这种情况，它支持高达 45W 的充电功率——尽管该设备附带的电源适配器最多只能输出 25W。这仍然很棒，但是没有你能得到的那么快。

令人沮丧的是——即使对于最近试图为 Galaxy Note 10+提供一些充电器建议的 Android Police 来说——第三方充电器可能*看起来*像是以你的智能手机可以处理的最快速度充电。但事实证明，当你插上手机时，情况可能并非如此。

对于 Galaxy Note 10+来说，这意味着声称可以在 45 瓦以上充电并支持 USB 供电标准第三版(15 伏和 3 安培)的第三方充电器可能仍然无法获得你预期的最快充电速度。正如 Android Police 所描述的:

> 据中国科技博主 Ice Universe 称，这里的一个障碍是你的充电器是否支持可编程电源或 PPS——这允许充电器输出可变电压，而不是 5V、9V、15V 等典型的固定电平。问题是充电器可以兼容 PD3.0，但选择使用固定电压，而不是使用 PPS。”

换句话说，为你的设备购买完美的充电器将是一件令人烦恼的事情，最好的情况是:

此外，考虑用你最终得到的充电器进行你自己的测试。您应该能够在不到一个小时的时间内从空到满，或者至少比使用随附的 25W 充电器为您的设备充满电所需的大约一小时零五分钟更快:
