# 如何在这个感恩节观看每一场 NFL 橄榄球赛

> 原文：<https://lifehacker.com/how-to-watch-every-nfl-football-game-this-thanksgiving-1840082010>

如果你今年在奶奶家庆祝感恩节，那么你可能忽略了一个重要的细节:你将如何观看足球比赛。



今天有三场不同的比赛。东部时间下午 12:30，芝加哥对阵底特律，接下来是东部时间下午 4:30，水牛城对阵达拉斯，东部时间晚上 8:20，新奥尔良对阵亚特兰大。

好消息是，所有这些游戏都在主要网络上播放，所以如果奶奶有有线电视，甚至一个像样的数字天线，你应该可以在她的电视上收听。以下是您的选项概要:

## 电视机

这是每个游戏和频道的官方概要。这些游戏应该可以在大多数市场的这些频道上免费获得。

## 车站应用程序

如果你的祖母没有有线电视，但你有，那么你可以通过使用你的有线电视 ID 登录，在各个电视台的应用程序上播放游戏。

## 流媒体服务

所有主流流媒体服务都提供对福克斯、哥伦比亚广播公司和全国广播公司的访问。唯一的问题是，他们不会在所有市场都提供这种接入，所以在签约之前，你需要仔细(甚至三次)检查你的市场是否有你想要的电台。

每项服务都提供免费试用，所以你可以注册，今天观看，明天取消服务，不收取任何费用。如果你想购买流媒体服务，那么最便宜的应该是每月 25 美元的 Sling。带直播电视的 Hulu 每月 40 美元，富博电视台和 Playstation Vue 每月 50 美元起。
