# 关于智能手机安全，杰夫·贝索斯能教给你什么

杰夫·贝索斯可能比你富有(很多),但他没有你聪明——至少，在基本的智能手机安全方面没有。你可能会读到并嘲笑(然后在你查了贝佐斯 [每分钟赚多少](https://qz.com/work/1410621/jeff-bezos-makes-more-than-his-least-amazon-paid-worker-in-11-5-seconds/) 后感叹)，沙特王储穆罕默德·本·萨勒曼最近通过 WhatsApp 给贝佐斯发了一个视频文件。《卫报》描述道，该文件实际上“包含了侵入贝佐斯手机的恶意软件，并在数小时内泄露了大量数据”。

虽然大多数人可能不会发现自己成为可疑朋友的直接网络安全攻击的对象，但贝佐斯被骗的轻松程度令人震惊，但并不令人惊讶。这也是一个很好的提醒，永远不要忘记你可以用来保护自己免受恶意软件攻击的最重要的安全措施之一。准备好了吗？

### **不要打开或下载你没有要求的文件**

就是这样。简单。嗯，没那么容易。如果某个随机的电话号码或网络陌生人给你发来一份文件，避开它应该不需要太多意志力。你很容易加载收到的随机文件或视频，尤其是当它们伴随着听起来很粗略的信息时。

当一个朋友——尤其是一个新朋友——给你发了一些东西来看，比如一个看似无辜、有趣的视频，我理解你为什么会放松警惕打开它。毕竟，他们有什么理由用恶意软件攻击你呢？他们从哪里得到这个恶意软件的？那不是一只看起来很滑稽的猫吗？你为什么不想看看它在做什么？

在这种情况下，很难提出重大的安全建议，说“永远不要看任何视频”是不现实的。甚至暗示一个平台可能不如另一个平台安全也无济于事。据称，贝佐斯被黑的手机是一部 [iPhone](https://www.ohchr.org/EN/NewsEvents/Pages/DisplayNews.aspx?NewsID=25488&LangID=E) ，而不是安卓系统——如果你认为苹果的移动平台牢不可破，你可能已经猜到了。攻击的载体显然是 WhatsApp 本身，正如联合国人权调查人员 [描述的](https://www.ohchr.org/EN/NewsEvents/Pages/DisplayNews.aspx?NewsID=25488) :

> *法医分析评估认为，入侵可能是通过使用在其他沙特监视案件中发现的一种著名间谍软件产品进行的，例如 NSO 集团的 Pegasus-3 恶意软件，该产品被广泛报道为由沙特官员购买和部署。这与其他信息是一致的。例如，使用 WhatsApp 作为平台将 Pegasus 安装到设备上的做法已经得到了充分的证明，并且是脸书/WhatsApp 对 NSO 集团提起诉讼的主题。*

正如《纽约时报》 所言, [*甚至不清楚贝佐斯是否自己打开了视频文件(和恶意软件),或者接收视频文件的简单行为是否足以利用 WhatsApp 的一个漏洞。换句话说，简单的接收隐藏在无害内容中的恶意代码就足够了。你可以尽你所能不去接触它，但这可能并不重要——除非你完全避开这条信息，这与一个*消息应用*的目的相反。*](https://www.nytimes.com/2020/01/22/technology/jeff-bezos-hack-iphone.html)

### 避免劣质手机恶意软件的其他方法

正如我所说的，很难提出全面的安全建议，而不会在某种程度上影响您与应用程序、服务和朋友的基本交互方式。我不会说“永远不要再打开视频文件”，因为那很傻。如果你收到了一些不请自来的信息，避免发送，但是如果一个朋友在 WhatsApp 上给你发了一个视频，你该怎么办？

真实对话:在大多数情况下，打开这样的文件可能是安全的。如果有一些巨大的恶意软件在 WhatsApp、Signal 或任何你喜欢的消息服务上流行，你会听说的——相信我。这些一次性攻击不太可能是你的好朋友发起的，只有*稍微*更有可能是你最近遇到的和不太了解的人发起的。我认为，我们谈论的是 0.01%到 0.05%的差异。

你可以选择不使用第三方消息应用，坚持使用手机的默认方式，但这是一个相当大的要求。例如，我通过 Facebook Messenger 和普通短信与朋友平等交谈；摒弃前者是行不通的。一些第三方应用程序，如 Signal，为您的日常信息提供了强大的保护(通过其内置的端到端加密)。你为什么不想这样？

这些第三方应用程序可以有自己的 的问题 [，但你的智能手机的默认消息服务](https://www.cnet.com/news/whatsapp-telegram-had-security-flaws-that-let-hackers-change-what-you-see/) 的 [也是如此。我愿意说，对于媒体附件之类的东西，后者可能比前者更安全，但不足以保证忽略你在 WhatsApp 等第三方应用程序中发送的任何和所有内容。无论如何，除了完全删除应用程序，你真的做不到。](https://www.wired.com/story/imessage-interactionless-hacks-google-project-zero/)

我*会*做的是为你最常用的信息服务设置一个快速谷歌提醒。通过这种方式，您可以随时了解有关该服务最近发现的漏洞或问题的任何新闻，这可以帮助您决定是否值得暂停该服务，或者切换到另一个服务，直到这些问题得到解决。

### 关闭自动下载

一个可能帮助你的设置是关闭你的消息应用程序包含的任何自动下载功能，这据称使贝佐斯陷入困境。以 WhatsApp 为例，你有一个[不同的](https://faq.whatsapp.com/en/android/23248698/?category=5245251) [方式](https://faq.whatsapp.com/en/iphone/28041114) 来阻止媒体自动下载到你的设备上。虽然我无法证实这无论如何都会保护贝佐斯，但保留自动下载默认设置可能会让视频恶意软件 [逃脱 iOS 和 Android 上的数字沙箱](https://www.independent.co.uk/voices/jeff-bezos-phone-hack-mbs-mohammed-bin-salman-jamal-khashoggi-whatsapp-a9299516.html) 。

### 跟踪您的数据使用情况

很容易看出你是否突然在 iOS 或 Android 上吞噬了太多的蜂窝数据，这可能是你的设备出了问题的迹象，像贝佐斯一样。没有一个很好的方法来了解你的 iOS 设备需要多少 wifi 数据，除非你可以通过你的路由器来分析这一点，但 Android 用户应该能够在操作系统内本地完成这一点(或通过第三方应用】T1)。

正如《泰晤士报》所描述的那样，你会希望留意你的设备正在发射的数据量的不切实际的增长:“在发送后的 24 小时内，贝佐斯先生的 iPhone 开始发送大量数据，这比他正常的数据使用量增加了大约 29，000%。"

如果你看到你的数据使用量急剧增加，并且你没有做太多额外的下载或流，考虑恶意软件的可能性。这仍然是一个遥远的可能性，但我可能会考虑抓一个或两个扫描仪应用程序，看看他们是否能找到什么，或者甚至重置我的设备的出厂设置(假设这可以清除它)。
