# 今年七月，我们致力于平衡

> 原文：<https://lifehacker.com/this-july-were-working-on-balance-1836017358>

是时候从操场 进来 [开始](https://vitals.lifehacker.com/the-june-fitness-challenge-is-playgrounds-1835210163) [Lifehacker 健身挑战赛](https://lifehacker.com/c/lifehacker-fitness-challenge) 的新篇章了。七月，我们回到了你可以在阴凉处，在任何空闲时间做的事情:平衡练习。

Watch

我们将从单脚平衡开始。简单点说，就是看你一只脚能站多久。如果你能坚持 30 秒以上，试着通过以下方式增加难度:

*   双臂交叉在胸前
*   闭上眼睛
*   站在不太稳定的地面上(像湿软的地毯而不是硬木地板)

一如既往，请了解自己的极限，保持安全。

用一只脚保持平衡可能很困难，如果你还没有准备好迎接上述挑战，有一些方法可以让平衡变得更容易。尝试:

*   抓住椅子或其他坚固的支撑物
*   用一个手指触摸墙壁，如果你只是需要一个身体上的提醒来提醒你在太空中的位置
*   让你自由的那只脚的一个脚趾着地

我从一个 [儿童瑜伽视频](https://lifehacker.com/cosmic-kids-teaches-kids-yoga-through-interactive-stori-1791432544) 中找到了最后一个的变体。在树式中，通常是将脚掌放在站立腿的一侧，教练要求孩子们将脚掌放在地板上，将脚跟放在站立的脚上。即使是我的小家伙们在试了几次后也能做到，所以如果树的姿势让你感到困惑，那就试试吧。
