# 把那些甲壳类动物的壳留下来做调味酱

> 原文：<https://lifehacker.com/save-those-crustacean-shells-to-make-a-sauce-base-1844520024>

这个周末，我和男朋友做了蟹酱——一种红色的酱，里面装满了美味的螃蟹。是的，有大块的邓杰内斯肉，但是酱汁的真正主角是螃蟹*壳*，我们捣碎它，用洋葱和茴香煨它，做一个非常难消化的基础。



我并不是说肉不是最终产品的重要组成部分，但是肉是在最后一刻被扔进去的——为了防止这种微妙的蛋白质在一锅热的、酸的、碾碎的西红柿里煮过头——和没有时间赋予太多的味道。虽然“普通的”番茄酱加蟹肉仍然很美味，但加蟹肉的*蟹*风味酱更好吃。

我们使用汉克·肖的蟹酱配方，但是用来制作底料的方法可以适用于任何需要海鲜味道的酱料，你可以使用几乎任何甲壳类动物的壳来制作。虾、龙虾、小龙虾和任何当地的螃蟹都会很好吃；你只需要一个沉重的木勺(或金属马铃薯捣碎器)。将贝壳与炒过的洋葱、大蒜和香草捣碎，煨煮，然后过滤并加入你的调味汁中调味。汉克的基础呼吁一点茴香烈酒，这是一个不错的主意；你当然也可以使用一些葡萄酒或在冰箱里放了几个月的苦艾酒。

你需要多少贝壳？没有那么多。我们用三个邓杰内斯贝壳做了我们的底座，结果得到了大约一夸脱的底座，其中只有一半我们用来做一大批酱。然而，底部确实可以很好地冷冻，所以在你有至少三只螃蟹、六只龙虾或 20 只左右的虾来一次做一大堆之前，保存你的壳(也在冰箱里)可能是值得的。(它确实有点让厨房发臭。)你也可以将贝壳组合成一个多种甲壳类动物的底座——我认为这是个好主意。要制作这种味道浓郁的贝壳水，你需要:

*   3 汤匙橄榄油
*   1 个洋葱，切丁
*   甲壳类动物的壳(大约 3 只螃蟹、6 只龙虾、20 只虾或一种混合物)
*   4 瓣蒜，切碎
*   一片 3-4 片月桂叶，一个茴香球的顶端，或者几根你认为和海鲜一起吃味道会很好的香草
*   半杯某种酒，是葡萄酒、苦艾酒还是茴香烈酒

用中高火加热大汤锅中的油，加入洋葱，煮至半透明。加入贝壳和大蒜，用一个大木勺或土豆捣碎器把之前的捣碎。煮大约五分钟，然后加入月桂叶、茴香和/或香草，以及你选择的任何酒精。加入足够的水，覆盖所有东西大约一英寸，然后炖一个小时。

用漏勺过滤掉大的固体，然后用一个有奶酪布或纸巾衬里的筛子进行精细的过滤。下一次你做一种可以从海洋风味中获益的调味汁时，加入一两杯你的基料。如果酱料准备不是固有的，简单地将基料冷冻成巨大的立方体或小汤料容器，然后根据需要将它们倒入未来的酱料。
