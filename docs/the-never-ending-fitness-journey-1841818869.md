# the never ending fitness journey 1841818869

> 原文：<https://jezebel.com/the-never-ending-fitness-journey-1841818869>

在 2020 年第一个令人沮丧的几周，狗仔队拍摄的阿黛尔的照片登上了小报。其中一个粗糙的镜头显示了一个快乐的女人，和著名的朋友哈里·斯泰尔斯在安圭拉岛的海滩上嬉戏。在照片中，阿黛尔看起来很开心，她也看起来很瘦——有些粉丝猜测太瘦了，表达了通常只有减肥才会有的虚假担忧。据报道，阿黛尔瘦了 100 多磅，这是值得注意的，但更值得注意的是媒体接近她的身体的方式，好像这与她的外表没有任何关系，而是与她的内在精神状态有关。一月份， [的一位“内部人士”告诉](https://people.com/health/adele-transformation-not-about-losing-weight-source/) *人*“这从来就不是减肥”。“她的体重减轻了，因为她减少了饮酒，吃了更多真正的食物。但是她现在也喜欢自己的身体变化。她更自信了，穿着也不同了，而且总体上看起来更开心了。”没有迹象表明阿黛尔的转变是暂时的:我们现在被健身之旅的第一站所束缚——这段旅程的含义是，这是一场史诗般但永无止境的奥德赛。

“健身之旅”现在无处不在，从 [的博客文章](https://www.amelietahiti.com/blog/my-fitness-journey) 到 [的 YouTube](https://www.youtube.com/watch?v=N9z5xqAP7ek) ，这里充满了认真的、长达 35 分钟的货币化视频。在社交媒体上，健身之旅很少是男性的专利。取而代之的是对减肥的刻意重新包装，加入了情感元素。从以前为你服务的身体中创造出一个新的身体，对身体和精神都有好处。当然，如果你生活在你目前居住的身体里，你会很好，但是发现是否会有一两个腹肌从你柔软的腹部冒出来不是很有趣吗？



尽管锻炼会释放内啡肽，使身体感觉良好，并暂时缓解任何精神压力，但这一事实在当代健身之旅的概念中明显缺失，这意味着保持身材并不明确与精神或身体健康有关，而是更广泛地与健康有关。健身之旅是一项精神事业，但需要外在和内在的动力。为了在这无尽的转变之旅中获得完全的成功，你的动机和意图必须是纯洁的。仅仅为了减肥而开始健身之旅是不足以维持旅程的；一定有更强的内在动力推动你走向这条路。健身之旅的流行说法是，如果保持身材的唯一动机是为了看起来性感，那么你最终会失败。

2019 年的电影*布列塔尼跑马拉松*挑选了健身旅程中的黑暗面。在这部电影中，布列塔尼(吉莉安·贝尔饰)是一个刻薄、戒备森严的女人，她对自己的身体感到不安，并对自己的爱情价值深感怀疑。在医生严厉地告诉她，她的身体质量指数把她列入“肥胖”类别后，布列塔尼开始跑步。一个街区变成了三个，不可避免地变成了一英里。看电影健身动机有用。如果你以任何方式抵制“目标”或“动机”的想法，那么布里特妮的成功之路感觉是可以管理的。

在电影中，健身实际上就是打理花园。那个花园是你的身体，入侵者是自我怀疑、渴望和认为自己永远不够的缓慢心碎。布里特妮的身体形象问题，强调了她减肥和健康的整个旅程，没有被忽视，也没有以其他不太敏感的电影制作人可能会有的方式掩盖。不认为你的身体——非常好，非常可爱——值得爱和关注的绝望导致从内部腐烂。布列塔尼通过自我毁灭来对抗这种腐败，但最终改变了她的生活，所以她最终对自己的恨稍微少了一点。虽然在电影的结尾，布列塔尼似乎处于一个更好的位置，但健身之旅的一个重要部分是首先考虑是什么导致了疼痛。为了实际的自我改善而自我改善是一回事，但是不管你现在看起来如何，如果不与这个世界将永远把你视为一个胖子的事实抗争，这就不一定是成功的秘诀。作为健身之旅的一部分，减肥和塑身的女性将永远与以前的自己的阴影抗争——她们旅行前的身体是一个迫在眉睫的威胁。但这部分旅程被忽略了，因为它不太符合健康博客和 Instagram 运动爱好者认可的闪亮授权叙事。

Casse y Ho，博客背后的流行健身博主，在 YouTube 上记录了她自己的个人健身之旅，她开始健身的部分原因是她对自己的身体感觉不好。何的视频的叙述与互联网上的其他视频相同:她进行了一次健身之旅，因为她对自己的身体不再感到舒适。对何来说，她的身体就是她的生计，她非常受欢迎的 YouTube 频道可以证明这一点。“首先，我想说我没有饮食失调，”她说。“我没有身体意象障碍。我不讨厌自己。这段旅程不是为你，而是为我。”
