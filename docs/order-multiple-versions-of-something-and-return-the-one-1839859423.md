# 订购某样东西的多个版本，并退回不想要的版本

> 原文：<https://lifehacker.com/order-multiple-versions-of-something-and-return-the-one-1839859423>

我知道每种潜在产品都没有无限的版本，但有时感觉就是这样——所以当你在购买一件新毛衣、一个新电水壶或一盏新灯时，你怎么知道你选择的是最好的呢？



是的，你可以花时间浏览客户反馈，交叉检查产品评论网站，如的库存，并在 Instagram 上搜索产品的图片。我已经在这种事情上浪费了*几个小时*，试图确定哪种电器真的最耐用，哪双运动鞋能提供最大的缓冲，以及哪些亚马逊的评论实际上是可信的。

然后我意识到，我可以只订购两三个版本的相同基本产品，自己检查，并退回我不想要的产品。

当然，在将这一策略付诸实施之前，您需要检查退货/退款政策。你还需要确保你的财务状况能够承担你计划购买的所有物品的费用——并且在你的日程表中有足够的时间退回你不打算保留的物品。当你只需要一个光疗灯的时候，你最不希望的事情就是在你的家里(和你的信用卡上)被三个光疗灯困住。

但当我意识到我不必再花一秒钟的时间阅读鞋子评论——我可以买下我正在考虑的所有三双鞋，在家里试穿，然后把其中两双寄回去的那一刻——改变了游戏规则。

值得注意的是，一旦鞋子、电器或闪亮的灯真的到了，你可能会发现自己想要保留不止一件。这实际上是一种屡试不爽的销售技巧；这个想法是，一旦你家里有了一件产品，即使只是为了测试它，它也会成为你的东西——你开始告诉自己，你更想要这件东西，而不是钱。

所以不要买你买不起的东西，即使你告诉自己以后会把它退回去。(你也应该知道 [据报道亚马逊已经封禁了多次退货的人的账户](https://www.cnet.com/news/amazon-is-banning-shoppers-who-make-too-many-returns/) ，所以...明智点，好吗？)

但是如果你能负担得起一次购买一堆不同的产品，并且有足够的时间退回所有不适合你的东西，这确实是一个简化购物的好方法。
