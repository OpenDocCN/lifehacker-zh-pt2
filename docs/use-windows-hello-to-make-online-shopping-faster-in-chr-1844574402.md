# 使用 Windows Hello 在 Chrome 中更快地进行在线购物

> 原文：<https://lifehacker.com/use-windows-hello-to-make-online-shopping-faster-in-chr-1844574402>

每当你在网上购物时，不得不键入你的信用卡号和 CVC 码，这可能是一件痛苦的事。然而，将这些信息保存在你的浏览器中，却没有足够的安全性来隐藏它，这也是一个糟糕的举动。要解决这个问题，可以考虑将 Windows Hello 与谷歌 Chrome 配合使用，这是一种更安全、快捷的购物方式，前提是你的笔记本电脑或台式机上有某种指纹识别器。



事实上，基于生物特征的支付信息认证即将变得更加重要，因为谷歌也正在为安卓用户推出这一功能 。

Windows Hello 使用[fido 2 协议、](https://lifehacker.com/update-google-play-services-to-enable-touch-id-sign-ins-1832874607) 安全技术，允许用户使用替代认证令牌(如生物数据、USB 密钥或 [手机](https://lifehacker.com/use-your-android-phone-as-a-security-key-for-signing-in-1833946643) )登录他们的设备、应用和网站。如果你的电脑上没有兼容的摄像头或指纹扫描仪，或者你不想给出你的生物特征数据(这一点 [我很同情](https://lifehacker.com/preview/worried-about-the-spread-of-surveillance-technology-th-1844565194?rev=1596152893201) )，Window Hello 也可以让你使用你的 Windows 10 PIN 码。

无论你使用何种登录方式，Windows Hello 都能让 Chrome 用户跳过网上购物最烦人的部分:填写你的信用卡和 CVC 号码。你不再需要每次结账时都掏出钱包，或者占用宝贵的思维空间去记忆所有这些数字——这就是记忆的工作方式，对吗？

听起来很方便，但是你需要先设置好。

### 在 Windows 10 上设置 Windows Hello

1.  从 Windows 开始菜单中打开**“设置”**。
2.  进入**账户>登录选项。**
3.  选择您想要使用的 Windows Hello 选项(面部、指纹或 PIN)，然后单击**“设置”**
4.  遵循屏幕上的说明。
5.  设置好身份验证方法后，您就可以在 Chrome 中打开 Windows Hello 了。

### 在 Chrome 上启用 Windows Hello

1.  在 Windows 10 上打开 Chrome(确保 Chrome 更新到最新版本)。
2.  前往**chrome://设置/自动填充。**
3.  点击**“支付方式”**
4.  如果您还没有添加信用卡信息，请添加。
5.  打开**“Windows Hello”**开关。
6.  点击**“使用 Windows Hello”**确认。
7.  使用 Windows Hello 进行身份验证。
8.  打开设置后，一切都准备好了。

下次你去网上支付时，会出现一个 Windows 弹出窗口，要求你使用你的面部/指纹/PIN 进行身份验证。按照提示，你应该很好去。如果你愿意，你也可以跳过使用 Windows Hello，像平常一样插入你的卡信息。

如果你无法访问 Windows Hello 或者选择不使用它，仍然有很多方法可以让 Chrome 中的帐户管理更容易——比如新的 [密码自动填充](https://lifehacker.com/schedule-downloads-on-android-with-this-chrome-flag-1844442037) 功能。 [谷歌不再是浏览器山之王](https://lifehacker.com/yes-you-should-quit-using-google-chrome-1844408232) ，但正是这些便利让 Chrome 保持相关性。
