# 你不应该为了睡着而玩电子游戏(但不管怎样，这里是方法)

> 原文：<https://kotaku.com/you-shouldnt-play-video-games-to-fall-asleep-but-heres-1841914026>

我们都做过。前一分钟，你看着吉姆对德怀特说了些刻薄的话；下一分钟，你醒来时看到一个屏幕，问你“你还在看*【办公室】*(美国)吗？”



当网飞指责你听着它的产品睡着时，你可能会觉得有点不好意思，但是使用媒体来帮助睡眠是很常见的。在 2016 年发表在[*行为睡眠医学*](https://www.tandfonline.com/doi/abs/10.1080/15402002.2014.963582)*的一项研究中，调查了 800 多名 18 岁至 94 岁的成年人，四分之一的受访者表示他们使用音乐入睡，约三分之一的人列出了电视，近 40%的人提到拿起一本书。10%的人会玩电子游戏。研究发现，使用媒体打瞌睡的参与者在匹兹堡睡眠质量指数上得分更高，这意味着他们比那些独立入睡的人睡得更差。研究还发现，那些听着媒体入睡的人睡得晚，但也起得晚，这意味着，尽管他们可能睡得不好，但他们不会因为游戏或电视而失眠。(对 2000 年代中期的中等射手的粉丝来说，有趣的事实是:研究人员称这一结果为“时间转移”)*

*然而，这就是睡眠——我们大多数人都没有得到足够的资源。2015 年，睡眠研究学会和美国睡眠医学学会这两个备受尊敬的睡眠研究组织发表了一份联合共识声明，将理想的最低睡眠时间从普遍接受的八小时下调至七小时。但是，根据美国疾病控制和预防中心的研究，35%的美国成年人经常忽略这一点。在 2014 年由[【NSF】](https://www.sleepfoundation.org/press-release/lack-sleep-affecting-americans-finds-national-sleep-foundation)进行的一项民意调查中，45%的受访者称睡眠不足影响了他们的“日常活动”除此之外，经常缺少睡眠还会 [增加你的血压](https://insights.ovid.com/crossref?an=00006842-201907000-00009)[削弱你的免疫系统](https://academic.oup.com/sleep/article/40/1/zsw019/2952682)[抑制你的性欲](https://splinternews.com/here-are-all-the-ways-sleep-deprivation-is-killing-your-1793846458)[让你不那么警觉](https://www.ncbi.nlm.nih.gov/pubmed/21075236)[增加你的食欲](https://www.ncbi.nlm.nih.gov/pubmed/26612385) 。* 

*市场上有大量的睡眠黑客。如果看情景喜剧重播比盯着天花板想你在早些时候的会议上说的蠢话要好，那么几轮*堡垒之夜*能以同样的方式使用吗？该不该用电子游戏来入睡？*

*“作为一名睡眠医生，我的第一个冲动是，‘嘿，不要玩电子游戏。关掉它。“拿起一本书，”美国睡眠医学研究院的发言人尼顿·维尔马博士在电话采访中告诉 *Kotaku* 。"但是两周前我玩电子游戏的时候睡着了."(好奇的读者:在*塞尔达传说:野性的呼吸*中，维尔马正在玩庞多的保龄球迷你游戏。)*

id="h289297"><strong>视频游戏如何影响你的睡眠周期</strong></h2><p class="sc-77igqf-0 bOfvBY">在嗜睡学家中，<span></span>普遍认为，打瞌睡的实际行为是两个身体过程的结果:“过程 S”睡眠和“过程 C”睡眠，或者你可能知道的昼夜节律。睡眠专家、NYU 医学院助理教授阿齐兹·塞夏斯博士在上周的一次电话会议上告诉<em> 处于高峰而过程 C 处于平静时——你就会睡着。但是玩电子游戏会因为几个原因使你的 C 进程紊乱，最终阻止睡眠。</p><h4 class="sc-1bwb26k-1 kpbNNd" id="h289298">视频游戏发出蓝光。</h4><p class="sc-77igqf-0 bOfvBY">许多因素——噪音、光线、咖啡因——都会对 C 过程睡眠产生影响。当然，即使是最令人兴奋的未知场景也不含咖啡因，但大多数视频游戏都有大量的声音和灯光。此外，你可能用来玩视频游戏的屏幕会发出所谓的蓝光。“我们的眼睛对特定的波长很敏感。特别是波长为 500 纳米的蓝光，”维尔马说。"蓝光会影响昼夜节律."有证据表明，<span> <a class="sc-1out364-0 hMndXN sc-145m8ut-0 eRdxJX js_link" href="https://www.health.harvard.edu/staying-healthy/blue-light-has-a-dark-side" target="_blank" rel="noopener noreferrer">蓝光</a></span>——从你的手机或任天堂 Switch 等屏幕上发出的光——会干扰睡眠。</p><h4 class="sc-1bwb26k-1 bOfvBY">噪音是一个更阴险的因素。“噪音作为一种烦恼的角色和噪音作为一种生理刺激的角色是有区别的，”塞夏斯说。他提出了城市新移民的想法:当你第一次搬到一个人口密集的城市时，喇叭和警笛可能会让你很困扰。不过，在那里住上一段时间，你就会不再注意到它们；那些相同的喇叭和警笛作为白噪音消失在背景中。尽管如此，你的身体还是会听到它们，你的大脑会把它们记录为“微声音”即使你没有意识到，它们也会让你精神振奋。同样，视频游戏的声音——战略游戏的环境声音，或者神奇宝贝战斗的背景噪音——会让你的大脑保持警惕，不管音量有多低(除非静音)。</p><h4 class="sc-1bwb26k-1 kpbNNd" id="h289300">一个“惊喜”因素会让你保持清醒。</h4><p class="sc-77igqf-0 bOfvBY">如果你打算在睡觉前玩游戏，你可能会认为像<em> Gris </em>或<em> Abzu </em>这样的放松游戏会比最新的<em>战役</em>-填空或<em>战争</em>-无论什么都好。但是，根据维尔马的说法，游戏的内容比游戏对你来说有多新更重要。“如果我刚刚得到这个游戏，我就开始玩它，我不可能睡着，”维尔马说。“一切都是新的，未知的，一切都是惊喜。这些东西让你清醒。”</p><p class="sc-77igqf-0 bOfvBY">然而，当你从事重复性的行为时——为获得经验值而努力或为获得战利品而耕作——你不会像投入一个未知的游戏时那样建立更多的预期。“在某种程度上，几乎没有什么可期待的，”维尔马说。那能让你睡着。<br/></p><h2 class="sc-1bwb26k-1 kpbNNd" id="h289301"><strong>为什么你应该将视频游戏视为助眠工具</strong></h2><p class="sc-77igqf-0 bOfvBY">鉴于上述情况，可以肯定地说，你不应该在睡觉前玩游戏，对吗？嗯，没那么快。尽管有这些缺点，游戏还是有一些令人昏昏欲睡的真诚，尽管这主要是因为它们不像其他常见的替代品那么糟糕。</p><h4 class="sc-1bwb26k-1 kpbNNd" id="h289302">视频游戏不会像睡眠药物一样带来依赖风险。</h4><p class="sc-77igqf-0 bOfvBY">尽管电子游戏对你的睡眠有负面影响，但依靠媒体作为睡眠辅助手段实际上可能是一种比非处方睡眠药物健康得多的替代方式。褪黑激素补充剂、苯海拉明(Aleve PM)之类的药丸或安必恩之类的处方药都是短期解决方案。维尔马说:“服用安眠药的最大风险是，你今天可以服用，明天仍然可能难以入睡。”。"安眠药只对那天晚上有帮助."</p><p class="sc-77igqf-0 bOfvBY">由于睡眠药物的短期效果，形成依赖并不困难。在周一使用它，你可能会安然入睡。然后你在星期二跳过它，很难入睡。有什么能阻止你在周三使用它呢？周四？整个周末？</p><p class="sc-77igqf-0 bOfvBY">“如果有一种不用安眠药就能可靠入睡的方法，那几乎总是更好，”维尔马说。“现在，是我给那些睡不着的人开电子游戏的处方吗？不。但如果你可以选择一边玩视频游戏一边睡着，或者吃安眠药，是的，我会选择视频游戏。”</p><h4 class="sc-1bwb26k-1 kpbNNd" id="h289303">玩游戏胜过无所事事。</h4><p class="sc-77igqf-0 bOfvBY">游戏的惊喜因素会让你保持清醒，但玩电子游戏可能实际上比盯着天花板更有效。尽管你可能听说过，数羊并不能诱导睡眠。国家睡眠基金会 建议，如果你不能在 20 分钟内入睡，你应该起来做一些“放松”的事情一个快速的冥想可能会让一个人放松；在<em>边境 3 </em>的试验场跑来跑去对其他人来说可能是放松。玩一个你熟知的游戏或游戏的一部分可以有助于睡眠的镇静效果。正如维尔马所说，一定要避开任何新奇的事物。</p><p class="sc-77igqf-0 bOfvBY">回放一个旧的电子游戏甚至可能是比阅读一本新书更好的睡眠方式。不知道会发生什么迫使你翻过这一页。接下来的事情你知道，这是日出，你渴望知道是否克沃斯知道风的名字。(也就是说，重读一本已经破烂不堪的书可能是更好的选择。没有蓝光，没有微声音，没有惊喜。)<br/></p><h2 class="sc-1bwb26k-1 kpbNNd" id="h289304"><strong>如何利用视频游戏帮助你入睡:</strong></h2><p class="sc-77igqf-0 bOfvBY">玩一个你熟悉的游戏。值得一提的是:一个可预测的夸夸其谈的游戏会比一个不可预测的安静游戏更有效。<strong> </strong>“这就像在看<em>宋飞</em>的重播，”维尔马说。“这个节目我已经看过一百万遍了。我知道它会如何结束。如果我要在睡觉前看什么，那太棒了。但如果我正在看一部我一直在等着看的电影，而且我不知道它将如何结束，我就不可能睡着。”</p><p class="sc-77igqf-0 bOfvBY">玩一个低压力的迷你游戏。以<em>的《野性的呼吸》</em>中 Pondo 的小游戏为例。就是重复。它需要最低限度的参与。“我只是在等待，从某种程度上来说，这并不令人兴奋，”维尔马说。"令人惊讶的是，正是这些小游戏组件让我昏昏欲睡."(研磨也恰好属于这一类。)</p><p class="sc-77igqf-0 bOfvBY">不要把大屏幕放在你的卧室里。维尔马说:“大多数睡眠医生给出的典型建议是，在卧室里尽可能少放设备、电视和大屏幕。住在小公寓里或者和室友住在一起可能会自然地排除这种可能性，但是如果你能把平板电视放在客厅里，那就这么做吧。如果你只能在卧室里玩游戏，较小的屏幕，如开关或电话上的屏幕，发出的蓝光总量比大屏幕少。</p><p class="sc-77igqf-0 bOfvBY">调低亮度。使用智能手机，你可以打开“夜间模式”，通过将屏幕的颜色输出转移到光谱的“更温暖”一端来减少蓝光的输出。电子游戏并没有给你这样的选择。即使是任天堂 Switch 的基本黑色背景主题，你可以在“系统设置”中找到，也帮不了你。毕竟，当你打开你的开关时，你不会花时间盯着菜单看，不是吗？然而，根据维尔马的说法，调低亮度会有所帮助。</p><p class="sc-77igqf-0 bOfvBY">调低音量。正如塞夏斯所说，你要避开那些听觉微障碍。如果可以的话，试着玩一个没有声音的游戏。对于几乎要求你听的游戏，尽可能低的音量会有所帮助。</p><p class="sc-77igqf-0
