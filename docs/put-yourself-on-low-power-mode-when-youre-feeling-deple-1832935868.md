# 当你感到精疲力尽时，让自己进入“低功耗模式”

> 原文:[https://life hacker . com/put-you-on-low-power-mode-when-you-feel-deple-1832935868](https://lifehacker.com/put-yourself-on-low-power-mode-when-youre-feeling-deple-1832935868)

如果你正在经历一个巨大的转变或者只是感到精疲力竭，进入“低功耗模式”会有所帮助我在休产假的时候，在播客上听到了这个建议，立刻感到安慰。低功耗模式— *是*。在此期间，你只能执行最关键的功能，而将其他所有事情推迟到你重新充满电的时候。你在保存你的能量，这样你就不必完全关闭。(这个词指的是 iPhone 的省电模式，针对非苹果用户。)

电池的类比很有道理。如果你处于低功耗模式，并在暴风雪中迷路，你会阅读维基百科对《真正的侦探》第一季大结局的总结吗？你会做一个小测验来找出你是哪个泰勒·斯威夫特吗？你会不会突然记起你的打印机没有墨粉了，然后在亚马逊上订购一些当天发货的墨粉？你不会的。你会简单地打电话求助，并试图保持自己的温暖。在生活中——比方说，如果你正在适应一份全新的工作——你可能会在家里进入低功耗模式。Y 你会继续洗衣服、洗碗，但是清理你的垃圾抽屉却要等一等。

这有助于向你身边的人宣布你正处于低功耗模式。在播客中，鲁宾解释说，她的一名团队成员告诉她，在生完孩子后的几周内，她将处于低功率模式。鲁宾明白这意味着什么——她是可以联系到的，但范围有限。当然，有时候你可能想完全脱离，这也没关系。但正如鲁宾解释的那样，完全脱离网络会让一些人感到压力更大。低功耗模式可以帮助你记住降低亮度有时是必要的，并且这种设置是暂时的。
