# 如果你的 MacOS 卡塔利娜安装卡住了，尝试这个修复

> 原文：<https://lifehacker.com/if-your-macos-catalina-installation-gets-stuck-try-thi-1838980635>

您在安装 macOS Catalina 时遇到过任何问题吗？具体来说，你有没有遇到过任何奇怪的场景，升级——不应该花*那么长的时间——似乎比感觉合理的时间要长得多？*

Watch

如果是这样，您的安装可能有问题。虽然你可以自由等待，甚至让你的 Mac 运行一整夜，看看这种缓慢的情况是否会自我纠正，但你很可能不得不代表你的 Mac 说情。

如果你的 Mac 挂在“设置你的 Mac”屏幕上，这应该在 App Store Catalina 下载完成和安装开始后不久出现，你有一个很好的解决方案让你的 Mac 再次屈从于你的意志:重置它。按住 Mac 的电源按钮，直到它关闭，然后再次打开。正如 MacRumors [描述的](https://www.macrumors.com/how-to/fix-macos-catalina-stuck-install/) :

> *”在 Mac 进入“设置您的 Mac...“荧屏，‌macOS Catalina‌的安装基本完成了。我们不建议立即重启，但是如果在屏幕前停留半小时或更长时间，请关闭它，因为它不会自己解决问题。*
> 
> ‌macOS Catalina‌安装需要一些时间，所以不要在设置屏幕弹出之前重启，但根据大量报告，如果它卡住了，以这种方式重置似乎是让你的 Mac 重新启动并运行的一种安全方式。"

我通常不会建议你在对 Mac 进行故障诊断时首先尝试这个技巧——特别是如果它是任何类型的冻结安装或更新——但在 Catalina 的情况下，它似乎产生了奇迹。要知道，当你突然关闭系统电源时，你确实会冒搞坏你的电脑、经历数据丢失或者引发任何其他可能发生的问题的风险。然而，如果你没有任何其他希望让 Catalina 安装，这一次似乎值得冒一次险。

同样，这也是为什么我们建议*在*你在 Mac 上安装主要系统更新之前做一个大的备份。如果你不想对此大惊小怪，你可以等待几周(或几个月)看看苹果是否更新了 Catalina 来解决这些问题。当然，你会错过 [有趣的功能](https://lifehacker.com/all-the-macos-10-15-catalina-announcements-from-apples-1835210038) 比如 Sidecar，但是你会活下来。
