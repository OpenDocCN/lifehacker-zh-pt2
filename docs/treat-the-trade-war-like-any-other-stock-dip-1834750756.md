# 像对待任何其他股票下跌一样对待贸易战

> 原文：<https://lifehacker.com/treat-the-trade-war-like-any-other-stock-dip-1834750756>

股票市场已经提高了警惕，因为中国将无限期提高关税。周一，标准普尔 500 经历了自一月以来最糟糕的一天，纳斯达克经历了今年最严重的下跌，道琼斯指数下跌了 617 点。苹果股票 [下跌近 6%](https://www.nytimes.com/2019/05/13/business/global-markets.html) ，公司债券市场利率上升，大豆价格下跌。事实上，贸易战是导致股市大约一周 下跌的罪魁祸首。恐慌时刻到了，对吧？

Watch

其实，请不要惊慌。一些股票今早已经有所反弹。在贸易谈判期间，它们可能会上下波动。“这不是明天就能解决的事情，任何说他们知道事情会如何发展的人都是在撒谎，”卢肯投资分析公司的格雷格·卢肯告诉美国消费者新闻与商业频道 。

据美国消费者新闻与商业频道报道，周一，花旗告诉客户，“正在进行的贸易突发事件可能会在短期内继续影响股票，但我们认为市场可能已经消化了这一点。”

因此，市场将经受住这些“突发事件”你的钱呢？

### 贸易战期间如何处理你的投资

上周，妮可 [分享了米歇尔·辛格特里](https://twocents.lifehacker.com/how-increased-china-tariffs-might-cost-you-money-1834670162) 的建议:几周内都不要看你的投资组合。等到贸易战结束。事情会有一段时间不稳定，所以不要过于频繁地检查它而让自己心痛。

但是，你可能会对我说，“我不能！我必须管闲事！我太紧张了，没耐心！请把烤箱手套戴在我手上，这样我就不会做出任何鲁莽的事情了！”

我们 2015 年 的 [建议(你甚至还记得*那次*股市跳水吗？我要猜不)仍然成立:不要在恐慌中卖出。相反，重新检查你的分配。](https://twocents.lifehacker.com/what-to-do-with-your-money-when-the-stock-market-plumme-1726155457)

正如 CFP·利亚·斯内尔当时告诉 Lifehacker 的:

> 考虑做一些研究，制定一个既舒适又适合你在上涨和下跌市场的风险承受能力的配置，这样下次市场波动性增加时，你就能在财务和情感上做好更好的准备。

如果你现在正在投资寻求高额回报，打电话给你的财务顾问寻求指导。趁现在股价低的时候抓住一些已经下跌的股票可能会增加你未来的收益。把它当成是苹果公司的折扣——或者任何你一直关注的热门公司。

但如果你是长期投资，比如你以后的退休，一个能经受住下跌和跳水的投资组合从长远来看对你最有利。这可能意味着如果你的接近退休，将资金转移到更安全的投资领域(通常是债券而不是股票)。或者你用完了 Rolaids，只是想感觉一切都会好起来。
