# 如何判断你是否受到了脸书最新黑客攻击的影响

> 原文：<https://lifehacker.com/how-to-tell-if-you-were-impacted-by-facebooks-latest-ha-1829716986>

上个月，脸书透露，其工程团队发现了一个问题，黑客能够利用该网站的“查看为”功能中的漏洞。该功能允许用户看到他们的个人资料对其他人来说是什么样子，但黑客能够利用它来窃取脸书访问令牌并接管人们的账户。访问令牌的工作方式有点像一组数字密钥，它允许你保持登录脸书应用程序，而不是每次想访问网站时都输入密码。

黑客攻击影响了该服务上的 5000 万个账户。当时，作为预防措施，脸书让近 9000 万个账户的访问令牌失效，并通知被注销的用户为什么会发生这种情况。

那时，脸书已经开始调查这个问题，所以还不确定哪些用户确实受到了影响。

虽然调查仍在进行中，但脸书现在已经通知用户，如果他们的任何数据因黑客攻击而受损。您也可以通过 [点击此处](https://www.facebook.com/help/securitynotice?ref=sec) 和滚动至页面底部来自行查看。在那里，你可能会看到下面两条消息中的一条:你没事了，或者你的帐户受到了影响，这是黑客可能已经访问的数据。

我很幸运没有受到攻击，但我的一位 Lifehacker 同事就没那么幸运了。

不像其他攻击，你可能想改变你的密码或监控你的信用卡信息，这一次窃取数据如你的姓名和联系信息，以及潜在的你列出的性别、宗教、你被标记的位置和你喜欢的脸书页面(仅举几例)。

虽然这种可能意味着你将面临更多的垃圾邮件，但对于脸书黑客你无能为力，除非更加注意你给服务的数据，改进你的电子邮件和 [电话垃圾邮件](https://lifehacker.com/how-to-prevent-robocalls-and-minimize-phone-spam-1828192361) 过滤器，或许 [完全重新考虑使用【脸书】](https://lifehacker.com/how-to-protect-yourself-after-facebooks-recent-hack-1829490273#_ga=2.121878543.505100129.1539268837-396842925.1520800403) 。
