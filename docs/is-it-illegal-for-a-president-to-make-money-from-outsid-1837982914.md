# 总统在任期间从外部企业赚钱违法吗？

> 原文：<https://lifehacker.com/is-it-illegal-for-a-president-to-make-money-from-outsid-1837982914>

上周，美国副总统迈克·彭斯访问了爱尔兰都柏林，参加了几场会议——但令人困惑的是，他住在 200 英里外 T3[附近的一家川普酒店。就在昨天，美国空军因在苏格兰川普酒店过夜加油而受到审查。特朗普本人最近](https://www.vox.com/policy-and-politics/2019/9/3/20847247/mike-pence-ireland-doonbeg-trip-180-miles-away-trump-hotel) [提议](https://www.usatoday.com/story/news/politics/2019/08/26/trump-and-g-7-presidents-florida-golf-resort-may-host-2020-gathering/2118304001/) 在他位于佛罗里达州多拉尔的酒店举办下一届七国集团峰会。“我在各地都有很多酒店，人们使用它们是因为它们是最好的，”特朗普在回答关于彭斯最近入住的问题时说。

诸如此类的案件现在引发了民主党人对特朗普总统的更大规模调查，特朗普总统通过利用他的办公室引导客户转向他的企业，从他的总统任期中获利。根据《纽约时报》上周末发表的一篇报道 ，川普的房产已经接待了外国领导人、国会共和党议员、游说者和政治候选人等数以千计的来访。根据 [响应政治中心](https://www.opensecrets.org/trump/trump-properties?cycle=All) 的分析，他们已经花费了不到 1900 万美元来使用特朗普的财产，其中大部分是在特朗普已经成为总统之后。

但是，有什么潜在的规则可以阻止总统在任期间获利吗？国会民主党人正在做什么来阻止它？

### 有什么问题？

正如 [时报](https://www.nytimes.com/2019/09/07/us/politics/trump-hotel.html) 所写，自特朗普成为总统以来，外国领导人、游说者和其他共和党领导人在特朗普房产的支出大幅增长；消费者权益团体 [Public Citizen](https://www.citizen.org/) 估计，自 2017 年 1 月以来，仅政治候选人和政党组织就在特朗普的房产上花费了至少 560 万美元，而在特朗普竞选总统之前的四年里，这一数字仅为 11.9 万美元。

很多争议都指向外国政党的支出。根据宪法中一套模糊的条款，即所谓的 [薪酬条款](https://fas.org/sgp/crs/misc/IF11086.pdf) ，在任总统不能接受任何外国政党的任何“礼物”，比如说，以领导人预订几个酒店房间或在 Trump properties 举办会议的形式支付。

此外，薪酬条款不允许总统接受多于或少于他的工资，包括来自联邦或州政府的工资，这带来了像彭斯在爱尔兰逗留这样的严重问题(由美国纳税人支付，然后直接流入特朗普组织的口袋)。然而，从技术上讲，美国总统在任职期间继续经营企业并不违法，而且在过去，总统们采取的是各种各样的(或 [盲目信任](https://www.forbes.com/sites/kellyphillipserb/2017/01/12/whats-a-blind-trust-anyway-and-why-wont-it-work-for-president-elect-trump/#6f8408f725b7))；总统为了避免任何利益冲突，放弃了按照自己的信任行事或管理自己信任的能力)。特朗普并没有坚持这种荣誉制度。

《华盛顿邮报》记者大卫·法伦霍尔德告诉“他每年从住在他酒店的外国政府那里得到至少一百万美元”。“而强制执行的机制是联邦法院，他们在尝试处理这个他们以前从未处理过的全新法律领域时，行动非常非常缓慢。实际效果是特朗普逍遥法外。”

当然，这是独立的，但与特朗普自己住在他的酒店时的支出密切相关。但目前，没有公开的纳税申报单，我们真的不知道他花了多少纳税人的钱来充实自己的企业。

### 接下来会发生什么？

针对川普提起了三起援引薪酬条款的诉讼；其中两个已经被撤销，国会民主党人的一个仍然有效。

然而，在彭斯的逗留和七国集团峰会拟议地点的刺激下，民主党人表示，他们现在正在调查这两起违反条款的案件。“委员会不认为美国纳税人的资金应该用来充实特朗普总统、他的家人和他的公司，”监督和众议院改革委员会主席伊莱贾·e·卡明斯在给白宫的一封信 中写道 [。与此同时，](https://oversight.house.gov/sites/democrats.oversight.house.gov/files/2019-09-05.EEC%20to%20Mulvaney-WH%20re%20Doonbeg.pdf) [众议院议长南希·佩洛西](https://www.speaker.gov/newsroom/9419-1) 在一份声明中称川普的房产为“腐败的粪坑”
