# 通过径赛训练跑得更快

> 原文：<https://lifehacker.com/run-faster-with-track-workouts-1834167188>

我很少会觉得自己比跑步时更强壮、更快或更糟糕。今天，我想和你分享这种感觉。



在本月的 [健身挑战](https://lifehacker.com/happy-spring-were-running-again-1833723734) 中，我们将了解不同类型跑步锻炼的好处。即使是在赛道上你可以做的短而快的事情，你也有很多选择。

### 如何在跑道上跑步

一个标准的跑道是 400 米，或者大约四分之一英里，如果你跑在内道(跑道上最短的一条)。大多数高中都有跑道，当地居民可以随时使用，但要确保遵守张贴的规则。总的来说，如果你在课余时间出现，而他们并没有参加真正的田径比赛，那么这个地方就是你的了，你可以随心所欲地跑或者走。

所以，你可以跑一圈(“我今天跑 400 米，”你可以告诉你的跑步朋友)，或者你可以一次跑两圈(800 米)。重点不是全力以赴跑完你选择的距离，而是以最快的速度跑完这段距离，感觉你可以一次又一次地跑完这段距离，中间有一些休息。

不过，在你开始之前，先热身。慢跑(或散步)四圈；那是一英里。在间歇期间，你可以选择是散步、慢跑还是只是站着。你选择哪一个可能取决于你的锻炼目标，但一般来说，我喜欢走和跑同样长的时间。如果一个间歇跑了两分钟，我会在下一个之前走两分钟。

### 尝尝我的最爱

好吧，我有两个最喜欢的锻炼，它们都涉及短时间间隔。最短的是 100 米间隔。这是赛道直线部分的距离，所以找一条线标记终点，一条线标记起点，从一条线跑到另一条线。

但如果我只能选择一项径赛训练，我会选择 200 米。径赛上的一条线直接穿过，通常标有“完成”它在直道的尽头，当你逆时针跑的时候。从那里正好在赛道的一半开始(将有一系列交错的线，因为每个车道都有自己的起跑线)。

从这个起点跑到终点，也就是半圈。记住，要走得足够快，以至于你像是*wheee this fast！！*但要慢到你能坚持到最后而不崩溃。然后走回起点，从头再做一遍。如果你是第一次，试试这四种；随着你的身体习惯了锻炼，你可以做更多。
