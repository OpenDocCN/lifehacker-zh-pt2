# 使用“查看身份”查看您在脸书上公开共享的内容

> 原文：<https://lifehacker.com/use-view-as-to-see-what-youre-sharing-publicly-on-faceb-1834917501>

它回来了——算是吧。脸书已经正式恢复了它的“查看”功能，这是一种让别人看到你的个人资料的方式。这是一个很棒的隐私工具，但有一些新的令人讨厌的限制。

Watch

在其最初的版本中，脸书的“查看方式”选项是一种很好的方式，可以反复检查你的隐私设置是否如你所愿，特别是如果你试图阻止特定的人(如你的父母)看到你生活的各个方面，而你希望他们不要看到。你可以从你的任何一个朋友的角度看到你的个人资料是什么样的，你也可以看到你向偶然发现你的个人资料的任何随机互联网用户公开分享了什么信息。

听起来很棒，对吧？直到 [脸书](https://gizmodo.com/50-million-facebook-accounts-affected-in-massive-securi-1829394250?_ga=2.111087366.2008167612.1558447521-2127637903.1532515930) 报告称,“查看身份”中的一个漏洞可能允许攻击者接管你的帐户或以你的身份登录许多其他网站(通过脸书的单点登录功能)。这场隐私混乱影响了至少 5000 万用户，脸书立即关闭了该功能。

脸书现在让“观点”起死回生——希望比以前更安全一点。然而，截至我们写这篇文章时，你只能在脸书的 Android 或 iOS 应用程序上访问它，它不如以前那么强大。你看不到特定的朋友可以在你的个人资料上看到什么，但你至少可以确保你不是故意公开分享信息。

### ios

您可以在您的个人资料页面上找到“查看身份”。点击右下角的汉堡图标，然后点击顶部的“查看您的个人资料”,即可导航至该页面。(或者，作为快捷方式，只需点击主脸书屏幕上“您的想法”状态框左侧的个人资料图片。)

在那里，点击齿轮图标和蓝色的“添加故事”图标右边的三点图标。你会看到一个弹出的小菜单，在顶部有“查看方式”。点击它，你会看到你的脸书个人资料页面与世界分享的所有信息。

### 机器人

点击脸书应用程序主屏幕上“添加帖子”框左侧的个人资料照片，调出您的个人资料，或者点击脸书应用程序右上角的汉堡图标，然后选择顶部的“查看您的个人资料”。现在，在您的姓名和简介文本下方的图标行上，您会看到一个大的“查看为”按钮，位于“添加到故事”的左侧。点击它以查看您个人资料的公开视图。

### 不要忘记调整您共享的内容

如果你想开始限制任何人可以看到的内容，iOS 版本的“查看为”功能并不能让你轻松编辑你的个人资料，但当你使用“查看为”选项时，你会在 Android 上看到一个又大又胖的“编辑个人资料”按钮。

我认为，当你拿着手机查看别人能看到什么时，用你的台式机或笔记本电脑编辑你的脸书设置会更快，但这只是我的看法。然而，没有消息表明“查看身份”是否会再次出现在你的脸书个人资料网站上。我联系了脸书，要求澄清这一功能，但他们从未回复。也许我们的友谊是紧张的。
