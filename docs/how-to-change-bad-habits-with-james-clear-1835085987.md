# 如何改变坏习惯，与詹姆斯清楚

> 原文：<https://lifehacker.com/how-to-change-bad-habits-with-james-clear-1835085987>

你会听到很多关于如何改变习惯的事情——这将花费你 21 天的时间！在你能做出大的改变之前，你必须做出小的改变！奖励自己！否定自己！咬紧牙关！去做吧！



你尝试了所有这些事情，但你仍然试图建立一个有意义的健身程序。你仍然在抽烟，或者在电视前吃东西，或者比你想的睡得晚，或者对你的配偶发脾气，或者咬指甲，或者[在这里填写你自己令人沮丧/可耻/顽固的习惯]。

我们想弄清楚我们怎样才能真正改变行为，生产力专家、心理学家和善意的同事们所公布的方法中，哪一种真正有效。

为此，我们采访了《一年的生产力》的作者[](https://alifeofproductivity.com/a-year-of-productivity/)*和 *Hyperfocus* 关于戒除习惯的五个步骤。《原子习惯》一书的作者 James Clear 帮助我们理解了我们的朋友是如何阻碍我们改变的。和伦敦国王学院心理学高级讲师 [本·加德纳](https://kclpure.kcl.ac.uk/portal/benjamin.gardner.html)**一起，我们终于弄清了 ol 的“21 天养成/打破一个习惯”的想法。***
