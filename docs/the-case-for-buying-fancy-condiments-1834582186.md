# 购买高档调味品的理由

> 原文：<https://lifehacker.com/the-case-for-buying-fancy-condiments-1834582186>

他们说多样化是生活的调味品，但是我确实过着我的生活，我从实际的调味品、调料和调味品中得到不同的乐趣。这就是为什么各种罐子和挤压瓶占据了我冰箱的一半，但这也是为什么我几乎每天晚上都在家吃饭。

Watch

除了没有时间，人们不为自己做饭的一个主要原因是，除非你把烹饪作为一种爱好，否则它会有点无聊和单调。如果储存真正好的鱼酱让在家吃饭更令人向往，那最终是自我保健，你不应该为此感到难过。简而言之，调味品让食物变得更有趣，每个人都有自己的最爱，你应该得到那个最爱的*最佳版本*，即使购买它不符合你对“财政负责任的成年人”会做的事情的愿景。任何鼓励你在家做饭和吃的东西都是好东西，如果一瓶进口酱油或一罐本地制造的发酵泡菜能让你这么做，那就把它放在你的购物车里。如果购买名牌番茄酱给你带来快乐，那么你应该得到它，即使普通版本便宜整整一美元。

节俭没有错，但如果你的预算(和冰箱)中没有为快乐留出空间，你会认为在家吃饭很无聊，没有味道，这将导致订购外卖，这几乎总是更贵和/或没有营养。(然而，当你需要或者只是想吃的时候，点外卖是完全正常的，但是如果你的目标是更多地在家吃饭，购买你真正喜欢的调味品可以帮助你做到这一点。)我是在说买你家里所有调味品中最贵的版本吗？不，那会很过分，但是挑选至少一瓶真正特别的东西意味着你的家庭用餐体验会感觉更特别。(此外，“特殊”并不总是意味着“昂贵”，但有很多重叠。)

如果你发现自己在想“哦，我不知道，10 美元一瓶这种超级美味的苹果醋似乎有点多，尽管它会让我一周都吃沙拉，”照我做的做，用鸡尾酒(或其他你不眨眼就买的无聊东西)来描述购买。每当我对“奢侈”食物犹豫不决时，我会问自己“这能买多少杯鸡尾酒？”答案通常是一。然后我问自己，“鸡尾酒会持续多久？”答案通常是“只是这瓶奇特醋能用多久的一小部分”，这让我有足够长的时间来考虑事情，让我在没有负罪感的情况下购买。

告诉自己“家里有食物”是成年人的一部分，但硬币的另一面是你可以控制家里的食物。有些食物可能很无聊——因为营养*有时很无聊——但是如果真正优秀的调味品可以缓解这种无聊，那就买下来。你值得被娱乐和享受你的食物，这意味着你应该得到昂贵的酱油。*
