# 以下是孩子们应该花在“远程学习”上的时间

> 原文：<https://lifehacker.com/heres-how-much-time-kids-should-spend-on-remote-learnin-1842879856>

可以理解的是，很多家长对这种在家学习的情况感到沮丧( [我仍然拒绝](https://lifehacker.com/dont-call-this-homeschooling-1842791078) 把我们现在所做的称为“在家上学”)。U p 在我的社交媒体上， [父母担心在疫情期间，他们会让孩子在学业上不及格，因为他们也在家工作，或者他们在外面工作，或者他们只是对此刻的一切感到焦虑。毫无疑问，我们孩子的学业进步是目前被忽视的事情之一。](https://lifehacker.com/if-you-re-struggling-to-parent-right-now-you-re-not-al-1842641280)

一些州和个别地区正在向家长提供比其他地区更多的指导、支持和结构，这意味着我们中的一些人完全不知所措，而其他人则希望他们有更多的资源或方向。伊利诺伊州教育委员会[发表了一份全面的(深思熟虑的)报告，内容是关于远程教育现在应该是什么样子——还有，如果*不像*那样适合所有人也没关系。](https://www.isbe.net/Documents/RL-Recommendations-3-27-20.pdf)

这份 62 页的报告 中的建议是由伊利诺伊州数十名行政人员、教师*和学生*组成的委员会起草的。他们以这些至理名言开始:

> 在教育工作者努力实施这些建议的时候，我们呼吁每个人对所有人都表现出灵活性和优雅。此时此刻，我们都需要树立韧性、批判性和创造性思维、深思熟虑的响应能力和同理心的榜样，以确保学生在个人、学术和语言方面继续成长。

布道，伊利诺伊州。*说教。*

无论如何，委员会接着给出了一个有用的图表，列出了孩子们应该按年级参加远程学习活动的最小和最大时间范围，以及我们合理预计他们应该一次集中注意力多长时间:

委员会还建议教育工作者为家庭提供额外的可选工作、参与机会和丰富机会，但前提是*也*明确工作是可选的，不会对学生的成绩产生负面影响。事实上，报告是这样描述评分的:

> 评分应关注学习的连续性，并优先考虑连通性和对学生和员工的关怀。所有学生都应该有机会重做、补充或再次尝试完成、显示进度或尝试完成远程学习阶段之前指定的作业。在这个前所未有的时期，我们应该把重点放在让孩子们在情感和身体上安全、有饭吃并参与学习上。

该报告还为教师提供了一个框架，以便他们为不同年级的学生以及接受特殊教育的学生制定教学计划。

但是这个图表，以及整个文件的整体基调，有助于家长们把它作为一个提醒，提醒他们我们不能或者不期望每天都进行几个小时的家庭教育。相反，如果你的幼儿园老师在 30 分钟内(分成更小的时间段)学得很好，你就做得很好。
