# 如何像星际舰队军官一样为人父母

> 原文:[https://life hacker . com/how-to-parent-like-a-starfleet-officer-1840338010](https://lifehacker.com/how-to-parent-like-a-starfleet-officer-1840338010)

有时候，当你蹒跚学步的孩子因为他们要香蕉而大发脾气，然后你给了他们一个香蕉，你可能会想到:*坐在我面前的这个不理智的人是谁或什么？就好像我在和一个无法适应新环境的外星生命形式打交道。那是时候引导你内心的星际舰队军官了。*

当然，你可以试试老式的耐心。那总是一个可靠的选择。但是当耐心耗尽的时候，你还有另一个选择。 [李曼·凯斯勒](https://www.facebook.com/LtKessler/posts/841298424195) ，我只能假设他是 TNG 的粉丝，他在脸书上解释道:

 [https://lifehacker.com/embed/inset/iframe?id=fbpost-https%3A%2F%2Fwww.facebook.com%2FLtKessler%2Fposts%2F841298424195&autosize=1](https://lifehacker.com/embed/inset/iframe?id=fbpost-https%3A%2F%2Fwww.facebook.com%2FLtKessler%2Fposts%2F841298424195&autosize=1) 

我们的婴儿、幼儿、小孩、青少年？他们都是来自另一个太阳系的外星使者，虽然相当聪明，但就是不习惯我们星球的奇怪方式。

作为一名星际舰队代表，你的职责是在他们适应地球的过程中保证他们的安全。如果在他们开始学习我们的方式时，你能给他们一些特殊的角色来帮助他们建立信心并鼓励他们进步，那就更好了。比如负责萝卜的执行官:

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-bnt01eFqpAI&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-bnt01eFqpAI&start=0) 

正如 [皮卡德船长曾经说过的](https://www.youtube.com/watch?v=8AYY5dpzNlU) ，“每一个星际舰队军官的首要职责就是真相，无论是科学真相还是历史真相或者个人真相。这是星际舰队赖以存在的指导原则。”

你的工作就是向我们的小外星人大使传授这些真理。就这样吧。
