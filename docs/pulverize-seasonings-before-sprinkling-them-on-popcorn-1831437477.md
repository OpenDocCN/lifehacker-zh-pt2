# 将调味料粉碎，然后撒在爆米花上

> 原文:[https://life hacker . com/粉化-调味料-在将它们洒在爆米花上之前-1831437477](https://lifehacker.com/pulverize-seasonings-before-sprinkling-them-on-popcorn-1831437477)

人们低估了爆米花的多功能性。没错，这是一道完美的小吃，但当搭配一杯葡萄酒和一些烤橄榄 时，它也是一顿不错的晚餐。而且，鉴于它是一整粒谷物，我认为没有人会因为你早餐吃它而责备你。(基本都是燕麦片。)这几乎是完美的，让它更完美的最好方法是在你开始洒之前把你的调味料粉碎。

Watch

你看，当磨成细粉时，盐和调味料以前所未有的力量粘在果仁上。闪电般地将混合物遗忘——比如你下面看到的辣甜盐混合物——意味着它会粘在你的零食上，而不是(悲惨地)掉到碗底。

即使你用的是普通的盐，将它分解成粉末会大大改善你吃爆米花的体验，但是当你用它搭配更粗的混合物时，这种方法真的很棒，特别是如果它们里面有少量的香草或种子。如果你需要一些灵感，试试下面这些:

*   Furikake
*   油炸香草
*   旧海湾
*   一切百吉饼调料
*   营养酵母
*   普罗旺斯香草
*   肉桂和糖(先把糖自己闪一下；肉桂已经够好了。)
*   黑胡椒和柑橘皮

无论你选择哪种口味包装的朋友，方法都是一样的。向食物加工机的碗中加入一勺调味料，将其搅拌成细粉，根据需要击打碗的边缘，让每个人都动起来。让灰尘沉淀一会儿——这样你就不会把它吸入——然后把粉末倒在一个筛子上，接住任何游离的大块。把它洒在热的、蓬松的、松脆的谷粒上，倒上一杯酒，享受你那非常有营养的全谷物碗。
