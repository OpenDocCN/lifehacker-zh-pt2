# 甜泡菜沙拉酱。是的，你听到了，甜泡菜沙拉酱。

> 原文：<https://thetakeout.com/recipe-how-to-make-sweet-pickle-salad-dressing-1835257671>

那是 90 年代。沙拉上的橄榄油和醋是一种东西。自制调料的爆炸还没有从帕妮丝蔓延到中美洲。因此，当一种自制的(我不严格地使用这个词)调料来到我们家的餐桌上时，我们都兴奋不已。

泡菜汁调味酱不是我们成长过程中餐桌上各式各样的沙拉酱之一——它是唯一的一种。当然，我们的冰箱里总是放着一瓶意大利叉骨调味酱，它有多重功能，比如牛排腌泡汁和切片西红柿酱。但是，在这种神奇的刺鼻调料出现之前，没有任何美味的东西被搅拌、摇晃或浇在生菜上。

味道很浓，很甜，像奶油一样。这是我们最喜欢的生菜:冰山。为了让它更令人难忘，这种沙拉通常会搭配一顿特别的晚餐——排骨、炸鸡、烤牛肉——这是一顿牛排餐厅级别的晚餐，我妈妈使出浑身解数。我们觉得自己是这个街区最幸运的孩子。

具有讽刺意味的是调料是由两种我讨厌的东西制成的:奇迹奶油和甜泡菜汁。我家没有蛋黄酱。这是奇迹鞭，也是唯一的奇迹鞭。我直到 20 岁才知道自己喜欢蛋黄酱，因为奇迹威化。检查它们里面的微小红点(你可能需要一个珠宝商的环才能看到它们)，我拒绝让那令人作呕的甜，奶油的厚厚的污渍任何三明治。

我对做这道菜很紧张——如果我的记忆更多的是基于情感而不是味道呢？如果很恶心呢？只有一个方法可以知道...还有什么比把泡菜汁洒在裤子上更好的方式来开始探索甜味调料呢？事实证明，很难从罐子里倒出满满一汤匙泡菜，同时又抑制住一大勺泡菜。以为倒掉比较容易？不对！我的柜台上堆满了泡菜汁。但这是值得的。调味汁香甜扑鼻，是卷心莴苣的完美搭配。相信我，这看似简单，却非常美味。(非常重要:不要用莳萝泡菜汁，要用甜小黄瓜的汁。它需要糖。)

我对妈妈原来的调料做的唯一“升级”是添加了——等等——盐和胡椒。一点全脂牛奶使得这个神奇的搅打器更容易均匀地搅拌到泡菜汁中。我尝试了几个不同的版本，以确保我没有错过泡菜汁节拍。我用了相同的比例——一份泡菜盐水混合两份蛋黄酱:

**甜黄瓜汁+海尔曼蛋黄酱:**就是不好吃——错过了奇迹鲜奶油的甜味。

**犹太莳萝泡菜汁+海尔曼蛋黄酱:**不可怕，但不好吃。比如水牧场。

**Cornichon 果汁+ Hellman 的蛋黄酱:**corni chon 你们这些鬼鬼祟祟的小绿魔！这是一匹我非常喜欢的黑马。请注意，这和我原来的调料味道完全不同，但有自己的优点。这是你的安慰奖敷料，如果你没有奇迹鞭和甜黄瓜汁，继续使用你的花式裤子泡菜。

**甜黄瓜汁+奇迹鞭:**经典，美味至极。

## 德怀尔家庭甜泡菜沙拉酱

*   1.5 汤匙。甜小黄瓜榨出的汁
*   3 汤匙神奇鲜奶油
*   1 茶匙不足。牛奶
*   盐和胡椒

将所有材料搅拌在一起，与切碎的卷心莴苣一起搅拌。
