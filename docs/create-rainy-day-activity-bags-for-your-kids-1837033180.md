# 为你的孩子制作雨天活动包

> 原文：<https://lifehacker.com/create-rainy-day-activity-bags-for-your-kids-1837033180>

我完全同意让孩子们感到无聊(这对他们有好处)并对他们自己的娱乐负责(他们有一大堆玩具)，但偶尔为他们提供一点灵感也不错。尤其是当他们因为恶劣的天气被困在里面已经好几天了，他们几乎要爬墙了(很快你也会这样)。



一位父亲在推特上分享了一个新想法，这个想法不需要你付出太多努力，鼓励他们的创造力，最重要的是，让他们忙碌一段时间。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1158633746423320578&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1158633746423320578&autosize=1) 

他的信中写道:

> 亲爱的孩子们:
> 
> 我有一个非常重要的任务给你。如你所见，我给 3 个包编号，1、2 和 3。

> *   从第一个书包里拿出一张纸条，作为你的作业。
> *   从第二个袋子里拿出一张纸，作为你作业的主题。
> *   从第三个袋子里拿出一张纸条，写下你任务的场景。
> 
> 没有重画。保留你拿到的第一张纸！如果你愿意，你们可以互相帮助，但我希望回家后能看到结果。祝你好运，希望机会永远对你有利。
> 
> 爱你的，爸爸

他在第一个袋子里装满了各种各样的活动:画画、写故事、拍照、拍电影、创作舞蹈。

他在第二个袋子里装满了活动的主题:你的父母、圣诞老人、太空人、小丑、鬼魂。

他在第三个袋子里装满了动作:骑恐龙、打熊、结婚、枕头大战、在月球上行走。

取决于他们拿出什么，他可能会回家看圣诞老人与熊搏斗的解释性舞蹈，或者是关于邻居小丑去月球旅行的故事。

你可以用他们的朋友或家人或季节性主题的名字来个性化这些想法(例如，僵尸、大蜘蛛和骷髅可能会在万圣节前后出现在活动包中)。

或者，如果他们觉得特别有创意，他们可以想出自己的创意添加到袋子里。
