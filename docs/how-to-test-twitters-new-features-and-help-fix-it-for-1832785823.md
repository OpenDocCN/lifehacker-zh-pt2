# 如何测试 Twitter 的新功能(并帮助修复它)

讨厌推特？看腻了推文乱序？对它的界面有长期的抱怨。对特朗普的荒谬推文感到恼火？

你可能无法解决所有这些问题，但你可以尽自己的一份力量。本周，Twitter 宣布推出一个“原型”测试程序 Twttr，主要关注回复和用户互动方式的变化。这个项目对公众开放，不过你必须申请才能被接受。

这里是你需要知道的关于加入 Twitter 的新测试模式的一切，包括彩色编码的推文(！):

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1098341544501731329&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1098341544501731329&autosize=1) 

### 有什么样的变化？

虽然 Twitter 尚未分享其新界面的任何截图，但一些变化包括圆形、缩进式的回复，这样它们看起来更“平易近人”和“ [”聊天式的](https://www.engadget.com/2019/02/20/twitter-replies-redesign-beta-test/) ”(这看起来有点像现在出现的直接消息)。对话也将被标上颜色，这样你就可以更容易地从所有回复的蓝色用户中识别出灰色的原创者的推文。

也许最大的变化是，当你穿越你的时间线时，股票期权和参与度也将被隐藏起来。你必须点击 tweet 才能在原型模式下查看它们。

### 这些变化好吗？

当然可以。仅仅基于阅读这些变化，他们会觉得有点像发短信(区分原始海报和回复是个好主意)。不过，隐瞒订婚是一个大胆的举动。有时候，你想看到受欢迎的推文是有原因的，隐藏起来会模糊这一点(我如何跟上最新的迷因？？)

### 我该如何申请？

您可以通过 Twitter 的 [申请表](https://survey.twitterfeedback.com/survey/selfserve/53b/190114#?) 申请。据 [TechCrunch](https://techcrunch.com/2019/02/20/twitter-opens-applications-for-its-new-testing-program-focused-on-fixing-conversations/) 报道，他们只接受了几千名申请者，其中大部分人会说英语和日语。如果你被录取，你将在未来几周内收到一封电子邮件。

一旦您被邀请，您将能够通过封闭的表格或在支持人员处发推文的方式发送反馈。

### 这些变化会推广到所有用户吗？

也许吧！如果你加入这个项目，说出你对它的新功能的支持(或鄙视)，几个月后你很可能会在 Twitter 上看到这些变化。
