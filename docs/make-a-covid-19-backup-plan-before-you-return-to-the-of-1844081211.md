# make a covid 19 backup plan before you return to the of 1844081211

> 原文：<https://lifehacker.com/make-a-covid-19-backup-plan-before-you-return-to-the-of-1844081211>

在家工作了几个月后，你可能会对回到办公室的前景感到兴奋。共享微波炉！饮水机聊天！会议室预订已满！过去的所有不便，但新常态的新奇。

但是，尽管一些办公室开始开放，冠状病毒病例仍在增加，我们还远远没有摆脱困境。在你对旧的办公室惯例感到太舒服之前，你需要计划如果有人生病会发生什么。在这种情况下，这个人就是你，或者你为其提供护理的人。



### **公开项目**

当你拥有一个项目或项目的一部分时，你可能习惯于保密细节，直到你的部分准备好了——或者到了与其他人合作的时候。但是如果你有可能不在办公室，几个星期都不能工作，你需要分享你正在做什么，以及你在这个过程中的位置。

快公司 的阿特·马克曼建议建立一个共享文档，列出你正在做的项目、你在其中的角色以及每个项目的现状。每周更新一次，这样一个指定的工作人员就可以从你停止的地方继续工作。

理论上，你的主管和其他经常和你一起工作的人会知道你的项目进展如何。但是如果他们不能打电话来询问你的问题，你不想让他们猜。



### **不要让你的队友去寻宝**

当您设置共享文档以提供项目状态更新时，请努力进行充分沟通。这并不意味着你必须在日记中详细记录你每个项目的进展——你只是想在你不在的时候让其他人更容易跟上进度。

客户联系信息和首选联系方式？列出来。你在等什么截止日期和组件吗？列出那些。如果你有任何“如果”的想法，而你没有时间去思考或提出来讨论，在你的共享文档中记录下来。

这个想法是让你的团队对你的项目有一个整体的了解，而不仅仅是你外出时的任务清单。

### **通知关键方**

如果你经常与客户、供应商或其他同事交流，确保指定一个人与他们联系，并让他们知道你可能会外出几周。您的指定人员可以概述您不在期间的通信选项。



至于其他的？别担心。花五分钟时间查看你的电子邮件，设置一条 [不在办公室的消息](https://lifehacker.com/use-out-of-office-messages-even-when-youre-not-on-vacat-1841997428) ，告诉你如果有重要的事情要联系谁。然后关掉你的通知，休息一下。
