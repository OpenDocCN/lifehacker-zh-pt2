# 如何正确使用分号

> 原文：<https://lifehacker.com/how-to-use-a-semicolon-correctly-1836184008>

分号可能被认为是所有标点符号中最冷漠、最吓人的一个。它比逗号强，但缺少句号的结尾。在任何你可能想要使用它的情况下，你实际上并不需要*使用它。所以何必呢？那很简单；分号使写作更有趣。*



想象一下，像这样的逗号给读者一个轻微的停顿，句号给读者一口气。分号介于两者之间；稍长的停顿，但不是完全的呼吸。当你想强调两个想法之间的联系时，它可以给你的写作注入不同的节奏。

### 把它当作一个连接器

分号的主要作用是连接两个独立的从句。这两个都是完全形成的语句，每个语句都有自己的句点。这完全没问题。如果你愿意，就去做吧。但是，如果这两个分句关系密切，并且你想强调它们之间的联系，你可以用分号将它们连接起来。

**正确**:我儿子每天晚上晚饭都吃热狗。他很快就会变成吸血鬼。

**也纠正**:我儿子每天晚上晚饭吃热狗；他很快就会变成吸血鬼。

### 把它用作“超级逗号”

您还可以使用分号作为一种额外的逗号来分隔包含其他逗号的列表中的项目。在这种情况下，分号有助于避免混淆。

例 1 :在我们的旅行中，我们将去参观乔治亚州的萨凡纳；南卡罗来纳州的查尔斯顿；和默特尔海滩。

例 2 :今年夏天我打算只吃煮得很熟的玉米棒；马苏里拉奶酪、番茄和罗勒三明治；还有一碗碗新鲜的草莓、蓝莓和哈密瓜。

### 不要用连词

当你的两个独立分句用连词连接时，千万不要用分号。连词本身已经是连接单词了，所以用一个连词加上分号是多余的。在这些情况下，请改用逗号。

**不正确**:本学期考虑选五门课；但后来我意识到我将没有社交生活。

**正确**:我考虑过这学期选五门课，但是后来我意识到我会没有社交生活。

然而，你可以在其他非连接词或短语前使用分号，如然而，此外，此外，此外，或例如。(我告诉他他要变成该死的热狗了；然而，他又吃了一个。)

看，没有理由被分号吓倒；当你知道如何使用它时，它只是你写作武库中的又一个工具。
