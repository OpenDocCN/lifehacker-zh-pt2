# 使用此应用程序快速查找附近的摩托车

无可否认，我不愿意接受正在渗透我们城市的小型摩托车。在我住的旧金山街区，你必须时刻保持警惕，确保没有经验的骑车人不会在人行道上撞倒你，而且在周末，人行道也经常被成堆的自行车堵塞。我讨厌他们。但是在我的家乡北卡罗莱纳州，那里的人要少得多，我看到了它的吸引力。

如果你是一个负责任的滑板车骑手，你知道，一个使用自行车道并自带头盔的人，那么滑板车可以让出行变得更容易，尤其是在像旧金山这样的城市。

虽然每个滑板车公司的应用程序可以帮助你找到你附近的一个滑板车，但 [应用程序滑板车地图](https://scootermap.com/) 可以帮助你找到你所在地区目前可用的所有滑板车，无论它们背后是什么公司。

该应用最初是为合同工创建的，他们需要找到需要充电的滑板车(大多数滑板车公司付钱给个人，让他们在家里给滑板车充电)，但它本周推出了一种针对骑手的模式。你还可以使用该应用程序骑任何 Lime，Bird 或 Skip 滑板车，这样你就不必启动另一个应用程序来跳上去。但是，您需要事先将您的帐户连接到踏板车地图。

该应用程序可以查找 Bird，Lime，Voi，Skip，Jump，Lyft，Razor，Scoot，Wheels，GoX 和 Wind scooters。如果你非常讨厌这些公司中的一家(或一群)，你也可以从你的搜索结果中过滤掉它们。默认情况下，地图还会显示踏板车、电动自行车、手动自行车和轻便摩托车。你也可以过滤这些，这样地图就不会显示自行车，例如，如果你更愿意坚持使用踏板车，反之亦然。

现在肯定有很多应用程序集成了这些公司的产品(城市地图是其中最好的，我认为)，但没有一个有 Scooter 地图那么多。如果你是一个经常使用滑板车的人，或者是一个有抱负的人，那么它绝对值得一看。
