# 何时预订酒店可以获得最优惠的价格

> 原文：<https://lifehacker.com/when-to-book-a-hotel-to-get-the-best-price-1835729684>

每当我计划去某个地方旅行时，一旦我预订了机票、火车票或租车，就感觉很“正式”，但我从来不确定何时预订酒店[。我是否应该尽可能提前做这件事，以获得最好的交易？或者我应该等到我真正到达目的地后，在今晚的旅馆或另一个临时地点碰碰运气？事实证明，为了获得最优惠的价格，实际上可能有一个预订住宿的“正确”时间。](https://lifehacker.com/tag/hotels) 



根据来自旅游研究 app [Hopper](https://www.hopper.com/) 的 [一项新研究](https://media.hopper.com/research/top-4-hotel-booking-myths-debunked) ，预订酒店的最佳时间取决于你的目的地城市。以下是他们的发现:

### 更大的城市

在分析了数百万家酒店的价格后，他们得出的结论是，如果你要去一个有很多商务旅行者的大城市旅行，你需要提前一到两周预订 ，比如 [【纽约】](https://lifehacker.com/your-best-new-york-city-tips-1818638405)[波士顿](https://lifehacker.com/the-best-boston-hacks-from-our-readers-1819849348) 或 [芝加哥](https://lifehacker.com/the-best-hacks-for-chicago-according-to-our-readers-1819377480) 。在这种地方，在你入住前的三到六个月，酒店价格只会有一点点波动。真正的价格下降发生在旅行前的 12 周。

那么为什么不预订呢？根据这项研究，在这些城市，最优惠的价格往往在最后一刻才能买到。问题是，它不一定能保证你喜欢的房间类型:你预订的越接近入住日期，酒店被预订的可能性就越大，所以你可能最终不会入住你最喜欢的房间。如果你只对酒店感兴趣，不在乎得到什么样的房间，那么在这些城市，最后一分钟预订可能是你的最佳选择。

如果你碰巧提前预订，然后看到价格下降， [有一种方法](https://lifehacker.com/get-cheaper-hotel-prices-after-you-book-using-this-site-1831909284) 你可能会得到更便宜的价格，只要你原来的预订有免费取消的选项。这项服务叫做 Pruvo。为了充分利用这一机会，您可以像往常一样预订酒店，然后将您的确认信发送到 save@pruvo.net 的。从那时起，Pru vo 将跟踪该预订的价格。如果价格下降，他们会发电子邮件通知你。然后，您必须取消原来的预订，并以较低的价格重新预订。这是一个多一点的工作，但最终可以节省你很多钱。

### 小城市和度假目的地

如果你要去一个小城市度假、周末度假或去海滩，你会希望尽早预订酒店。在这种情况下，价格在入住前三个月左右最低，之后可能会上涨，尤其是对那些等到最后一分钟的人来说。
