# 以一杯马拉希诺马提尼开始漫长的周末

> 原文：<https://lifehacker.com/kick-off-the-long-weekend-with-maraschino-martini-1835011457>

假日周末——尤其是夏季的周末——通常与烧烤和啤酒联系在一起。我总是下来吃一些烧烤，但啤酒让我打喷嚏，我不明白为什么我要在一个长周末改变我的饮酒习惯。

Watch

但是马提尼——我最常喝的饮料——不太阳光。事实上，我大部分时间都是坐在黑暗的酒吧里喝，尽量避免围绕我的纹身闲聊。就马提尼酒而言，大多数添加物倾向于(如 dirty)或甚至更醉(如 [伏特加-y vesper](https://skillet.lifehacker.com/3-ingredient-happy-hour-the-classic-boozy-vesper-1797339960) )。为了让我的酒更明亮一点，我决定加入一点我最喜欢的樱桃味利口酒——Luxardo maraschino。

要明确的是，我不建议你整个周末都喝这个。尽管这是一种更友好、更诱人的马提尼酒，但它仍然是一种马提尼酒，你至少应该和几罐拉克罗伊酒交替饮用。要制作这款果香浓郁的酒，你需要:

*   2 又 1/4 盎司干杜松子酒
*   3/4 盎司干苦艾酒
*   1/4 盎司 Luxardo maraschino 利口酒

将所有东西倒入加满冰块的混合杯中，搅拌直到杯子非常非常凉。应变成轿跑，悠闲地享受，然后用补水的东西追。整个长周末根据需要重复。
