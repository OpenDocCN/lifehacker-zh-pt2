# 这本免费的涂色书帮助孩子们学习他们身体部位的正确名称

> 原文：<https://lifehacker.com/this-free-coloring-book-helps-kids-learn-the-correct-na-1833891599>

尽早教你的孩子正确的身体部位名称 是很重要的。你儿子有阴茎，不是“小鸡鸡”你女儿有阴户，不是“小淑女”

这有许多原因。首先，我们的身体是惊人的，所以为什么要用代码谈论它们呢？此外，知道正确的术语有助于他们在感觉不舒服时进行交流——例如，你的孩子可以告诉你，确切地说是什么东西在燃烧或发痒。也许最重要的是，这让他们对身体自主有了更深的理解。那些被告知生殖器真实名称的孩子——以及什么样的触摸是不可以的——可能不太容易受到性虐待。(我从一开始就试着教我女儿这些名字。有一次，妹妹在换尿布，问:“我用这个尿不湿膏做什么？”她两腿悬空，脱口而出:“你把它放在我的阴道上了！”)

[Amaze](https://amaze.org/) ，一个“消除性教育中的尴尬”的伟大资源，发布了 [一个免费下载的涂色书](https://amaze.org/2019-coloring-book/?utm_medium=frontpage_promo&utm_source=website&utm_campaign=coloring-book) ，其中包括一页插图 [男孩和女孩的身体部位](https://amaze.org/wp-content/uploads/2019/03/2019_AMAZEJR_COLORING_BOOK_thumb-01.jpg) 。这是向小孩子介绍解剖学上正确的名字的简单方法——我的意思是，谁不喜欢涂颜色呢？你可以在和你的孩子谈论[同意](https://offspring.lifehacker.com/these-videos-teach-kids-about-consent-in-an-age-appropr-1819448970) 和 [愉悦](https://offspring.lifehacker.com/give-kids-specific-language-to-talk-about-pleasure-1822150699) 时使用它。

虽然这可能不是我们与自己父母的谈话类型，但它们是必要的，也许现在比以往任何时候都更有必要。只要有助于我们启动讨论，我都会欣然支持。
