# 在新的一年里，尝试这些廉价的社交想法

> 原文：<https://lifehacker.com/try-these-inexpensive-socializing-ideas-in-the-new-year-1830747956>

在经历了一两年越来越无意义的消费后，我新年的目标之一就是尽可能多的存钱。在我生活的某些方面，比如在外面吃饭或者买衣服，很明显我需要削减什么，而且有很多简单的方法来省钱。

Watch

我预见会给我带来麻烦的一个领域是和朋友出去玩。当外出活动是酒吧之夜或在新餐馆吃饭时，外出一两个晚上会打破你的预算，尤其是当你的朋友不像你这样对钱敏感的时候。

这就是为什么我们众包一些便宜的悬挂选项。奥黛丽·冈萨雷斯提出的这个建议听起来特别有趣:

> 我的一个朋友想出了办一个烹饪俱乐部的主意。每个月，小组里都会有人自愿示范如何做一道最喜欢的菜，无论是鸡肉馅饼、辣椒粉还是黑豆菠菜辣酱。我们甚至让人们演示简单的蛋糕装饰技术，或带我们了解创建自己的家庭花园的基本知识。由于繁忙的日程安排和年幼的孩子，我们提前几周安排烹饪俱乐部，因为我们有足够多的人，我们只需要每年志愿者一次或两次。其余的时间，你可以去参加社交活动，学习如何制作新东西，还可以免费吃饭。

这是一个多么好的与朋友保持同步的方式，提供舒适的*和*展示你的烹饪能力。你也可以坚持更传统的聚餐。

另一个我们之前写过的建议: [创办一个短篇小说俱乐部](https://lifehacker.com/start-a-short-story-club-instead-of-a-book-club-1827514968) 。这不涉及读书俱乐部的时间承诺(或费用)，而且，正如尼克所写的，你可以用你的选择变得更有创造性和前卫。

电影俱乐部也可以。受到播客的启发，我和一些家庭成员开始一起浏览 [AFI 百部最佳电影](https://www.afi.com/100Years/movies.aspx) 。我们每隔一周挑选一部电影，边看边讨论。你也可以建立类似的东西，调整时间框架来满足你的需要。最棒的是，你可以在网上免费找到许多电影，或者以不到一杯咖啡的价格租借。
