# 用快速文本支持 USPS

> 原文：<https://lifehacker.com/support-the-usps-with-a-quick-text-1844749588>

如果你担心美国邮政署​明显缺乏处理即将到来的邮寄选票所需的资金，一条短信可以帮助你。在 Twitter 上， [@Resistbot](https://twitter.com/resistbot) 主动提出通过短信或直接信息交换的方式代表你给代表们写信。

Watch

(如果你想，“等等，USPS 怎么了，”请 [花点时间到这里赶上](https://lifehacker.com/help-save-the-usps-with-one-call-1844726967) 。我知道*现在一切看起来都很重要，但这很可能会影响选举的结果，因为大量的选票没有计算在内。)*

当然，你也可以(也应该)给你的代表打电话——[这里有一个简单的脚本和指示【](https://lifehacker.com/help-save-the-usps-with-one-call-1844726967)】告诉你怎么做。但大量涌入的信件也无伤大雅，从头到尾只花了两分钟就完成了。 [上手](https://twitter.com/resistbot/status/1294667532796911621) ，简单来说:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1294667532796911621&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1294667532796911621&autosize=1) 

[Resistbot](https://resist.bot/) 是一个 501(c)4，帮助选民直接向他们选出的代表和报纸发送信件。以下是 Resistbot 代表我提交的信件(我今天早上的签名编号是 1，175，630):

要发送您的信件，您需要提供您的姓名、您的邮寄地址(以确认哪些参议员和众议员应该收到您的信件)、和您的电子邮件地址，以便接收确认码。

如果提供这些信息让你不舒服，但你仍然想支持美国邮政，那么 [买些邮票](https://store.usps.com/store/results/stamps/_/N-9y93lv)；这是一个简单的方法，即使你以后才需要邮票，现在也可以给它注入一点资金。
