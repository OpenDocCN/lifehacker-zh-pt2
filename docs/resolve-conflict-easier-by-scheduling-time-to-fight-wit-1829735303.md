# 通过安排时间和你的伴侣吵架，更容易解决冲突

> 原文：<https://lifehacker.com/resolve-conflict-easier-by-scheduling-time-to-fight-wit-1829735303>

没有人想吵架(或者至少没有人是你想交往的对象)，但是吵架是会发生的，而且吵架是任何关系中健康的一部分。虽然一直吵架不是一件好事，但不吵架也不是一件好事。当你和你的伴侣在影响你们关系的重要问题上进行健康的建设性的争吵时，它甚至可以让你们走得更近，而不是更远。

那么，什么是“健康的”战斗呢？*时间* [近日采访了心理学家](http://time.com/5402188/how-to-fight-healthy-partner/) 一探究竟。这些生理学家提出了一个有趣的建议:安排好你的冲突。

这是一个我认为很奇怪的建议，但在思考了一会儿之后，我可以接受。与其在紧张的时候吵架，不如在吵架开始的时候安排一个时间让你结束。

你不会把战斗推迟到下周二，但是给一些时间让每个人思考他们的感受和他们为什么生气，然后带着这些深思熟虑的想法回到对话中。这样，你就有希望避免一时冲动的伤人言论，而是能够找到问题的根源，提出一个双方都尊重的解决方案。

### 想想你打架的原因

我们经常以一次又一次完全相同的争吵而告终。虽然争吵的主题理论上可能会改变，但如果你看看争吵的根源，你会发现它们会再次出现。

时间给出的例子是一对夫妇在一周的“下午 5:30 吵架”,因为一方想谈论他们的一天，而另一方想从一天的工作中减压。当第二个人回避谈话时，第一个人感到被忽视和不被重视，但事实上，这两个人都是问题的根源。

不要每天吵架或者一遍又一遍的吵架，看看是什么最终导致了一个问题，然后努力创造一个让每个人都开心的妥协。例如，当同样的争斗再次开始时，也许每个人都可以离开几分钟，然后重新集合。

当你们吵架时，一定要听你的伴侣说什么，问问题，当你错了，学会正确的道歉方式。就像我们都有自己的爱情语言一样，我们也都有自己的道歉语言。
