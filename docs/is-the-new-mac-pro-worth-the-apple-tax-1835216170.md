# 新款 Mac Pro 值苹果税吗？

> 原文：<https://gizmodo.com/is-the-new-mac-pro-worth-the-apple-tax-1835216170>

Mac Pro 是一个声明。它的透气外壳让人想起厨房工具，是对 Power Mac G5 的致敬和进步，粉丝们亲切地将它比作奶酪刨丝器。这是一款配备了新一代至强处理器的苹果设备，似乎终于按时推出，并与其竞争对手步调一致。尽管如此，6000 美元基本款配备的微不足道的 GPU 和 SSD 似乎与其行业领先的设计不协调。当我走过苹果公司为展示这款设备而设立的展厅时，我不停地问自己，它值不值这么高的苹果产品价格？



最好的苹果产品价格昂贵，但有足够的高级抛光，他们觉得值得。我爱我的 MacBook Pro:我喜欢未被充分利用的 Touch Bar 和 [电脑运行多快](https://gizmodo.com/the-new-apple-macbook-pro-is-so-good-the-price-is-almos-1827802621) 。我喜欢众多的霹雳港。同样， [iPad Pro](https://gizmodo.com/the-new-ipad-pro-is-the-last-great-tablet-1830380453) 和 [iPhone XS](https://gizmodo.com/the-new-ipad-pro-is-the-last-great-tablet-1830380453) 都是昂贵的设备，感觉它们物有所值。从硬件到软件的体验都足够好，你可以证明其成本是合理的。

但是其他设备不值得——主要是 MacBook 和 MacBook AirT3。它们是硬件过时的中型设备，这使得它们虚高的价格看起来令人厌恶。

Mac Pro，起价 6000 美元(！)感觉它很容易被归为后一类——价格过高，动力不足。基本型号有一个基于老化架构的新 GPU，一个微不足道的 256GB 存储，值得注意的是， [和一个大约 1200 美元的 CPU](https://ark.intel.com/content/www/us/en/ark/products/193747/intel-xeon-w-3225-processor-16-5m-cache-3-70-ghz.html) 。(苹果吹嘘的 28 核处理器价格是零售价的两倍多，可能会导致该机价格相应上涨。)如果你想自己制造这款设备，基本型号中的其他组件可能会卖到 2500 美元以上。因此，乍看之下，苹果税显得非常离谱。在脑子里算了一下后，我发现自己目瞪口呆。

放大一些细节，这个价格就开始有意义了。当您查看它的输入和输出时，会发现它有很大的灵活性和吞吐量，尽管您可能一眼就看不到它。与典型的 Windows 机器相比，Mac Pro 的背面没有端口。有几个千兆以太网端口和 USB-A 端口，加上两个雷电 3 端口，一切都很干净，看起来很稀疏。(另外两个雷电 3 端口位于设备顶部。)

这很不寻常，因为雷电 3 端口并不经常出现在台式机主板上——尤其是消费类主板。你必须去专业市场才能找到它们。Mac Pro 上巨大的 8 个 PCIe 插槽也是如此。你更有可能在苹果的竞争对手 [、惠普](https://store.hp.com/us/en/ConfigureView?catalogId=10051&langId=-1&storeId=10151&urlLangId=&catEntryId=3074457345618619819&quantity=1) 和戴尔制造的专业工作站中发现这一点。想要体验视频的普通游戏玩家或消费者不需要这种灵活性。如果你是一名专业人士，需要直接将光纤连接到你的计算机，或者需要大量的 GPU 来处理 8K 渲染，你才真正需要这些插槽。

你和我？我们不需要那玩意。

同样，我们也不需要 1.5TB 的内存，你可以把它装进 Mac Pro。你和我永远都不需要 1.5TB 的内存。但是有人用中等格式的数字图像或 8K 视频项目工作吗？这对于他们来说一直是一个白日梦，直到昨天，大多数工作站只能支持苹果公司在 Mac Pro 的 12 个 DIMM 插槽中安装的昂贵 ECC RAM 的一半。

考虑到所有这些，以及建造一台类似的机器需要多少成本，Mac Pro 开始看起来不仅合乎逻辑，而且实际上是负担得起的。一台类似于 6000 美元版本的 Mac Pro 的基本型号的设备，惠普的[价格将超过 9000 美元，戴尔的](https://www.dell.com/en-us/work/shop/desktop-and-all-in-one-pcs/precision-7920-tower/spd/precision-7920-workstation/XCTOPT7920US_4?selectionState=eyJPQyI6InhjdG9wdDc5MjB1c180IiwiTW9kcyI6W3siSWQiOjMsIk9wdHMiOlt7IklkIjoiRzNEUUVWVSJ9XX0seyJJZCI6MTMsIk9wdHMiOlt7IklkIjoiMTBHTlRXSyJ9XX0seyJJZCI6NDksIk9wdHMiOlt7IklkIjoiVlBSTyJ9XX0seyJJZCI6MTQ2LCJPcHRzIjpbeyJJZCI6IlM2MTUyIn1dfSx7IklkIjo2NjYsIk9wdHMiOlt7IklkIjoiVEhCUENJRSJ9XX1dfQ%3D%3D) 价格接近 6300 美元。这还不包括额外的雷电 3 端口，或支持 1.5TB 内存的能力。

这不仅仅是机器规格和 IO 吞吐量的问题。正是这些小事让我觉得苹果的溢价可能是值得的。考虑一下这个庞然大物是如何组装和设计成气冷式的，这样你就不需要为闭环水泵而烦恼了。诚然，苹果完全控制了演示区，但当我昨天看到 Mac Pro 运行时，当我在三台显示器上实时观看 8K 镜头时，我没有听到风扇嗡嗡作响。这箱子摸起来很凉。这给我留下了深刻的印象。

同样令人印象深刻的是 after burner FPGA(FPGA 是一种处理器，通常为特定的利基任务而构建，就像控制台仿真器中的 FPGA 一样)。Apple 构建的定制卡旨在加速专业电影编辑使用的 ProRes 编解码器。从表面上看，这似乎不可思议，因为它允许计算机处理 8K 的连续镜头而不会出现停顿。Moor Insights & Strategies 的分析师 Anshel Sag 通过电子邮件告诉我，对于制作公司和视频编辑来说，“加速视频处理是一件大事”。但是我们仍然不太了解加力燃烧室本身。Sag 说:“在主题演讲中，关于它的细节很少。”

我们确实知道所提供的 GPU。在基本模型中发现的镭龙 Pro 580X 在理论上是相当脆弱的。它使用了 2016 年首次推出的旧 [北极星架构](https://www.amd.com/system/files/documents/polaris-whitepaper.pdf) ，8GB 的 DDR5 RAM 显得微不足道。值得注意的是，苹果在一次场外演示中展示的 Mac Pros 没有一个使用这种 GPU，与提供的其他卡相比，这不是它性能的好兆头。(苹果选择展示基于较新的 Vega 架构、功能更强大、价格更贵的显卡。)

但是我们不应该把 580X 排除在外。我一直在想苹果将每个 GPU 打包到其中的 MPX 模块。通过 MPX 模块，有可能从旧的 GPU 中获得更多的性能。有了 Mac Pro，你不会只是去买一个 GPU 卡，然后把它扔进去。(我相信理论上你可以。)相反，你会得到已经打包到 MPX 模块中的 GPU，该模块旨在使 GPU 的核心比 AMD 的股票卡更冷。它为芯片提供了雷电 3 的内部支持，同时也允许苹果和 AMD 建立定制的 GPU，将一个或多个 GPU 配对在同一个主板上。

那真的很酷！它是否足够酷，几乎可以弥补使用 580X 而不是像 AMD 上周在 Computex 上宣布的基于 Navi 的新 GPU 那样的东西？我不确定。

很多事情我都不确定。我非常矛盾。有足够的工程实力和足够令人印象深刻的技术规格，使 MacPro 看起来值得苹果税。尤其是在我目睹的演示中。

但是，直到我们打开这个东西，并尝试在受控的情况下实时渲染一些 8K 的镜头，很难称这台机器是不可思议的。目前，它只是令人印象深刻，即使它看起来像我妈妈在厨房里放的旧炉排。
