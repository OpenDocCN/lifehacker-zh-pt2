# 因为缩放不能保证你的隐私，尝试这些替代方案

> 原文：<https://lifehacker.com/which-video-chat-apps-offer-the-best-privacy-and-encryp-1843886613>

有许多免费的群组视频聊天和电话会议应用程序 。他们中的许多人兜售他们的隐私和安全功能。仅仅因为他们禁止其他用户或潜在黑客进行某些通信，并不意味着你的谈话不受这些公司本身或他们合作的其他组织的保护。

例如，Zoom 现在对免费用户限制其端到端加密(E2EE)。该公司表示，这样做是为了遏制滥用，并采取行动打击巨魔和黑客，但这也削弱了那些使用 Zoom 会议进行安全通信的人的隐私。

其他流行的应用程序，如 Discord，也不加密用户通信——即使是发生在私人服务器上的通信。这有助于 Discord 将仇恨和白人至上主义团体从其平台上移除，但这也意味着，通过 Discord 服务器努力创造积极运动的活动人士、记者和社区组织者没有这种行动通常需要的隐私。

如果你正在寻找 Zoom 或 Discord 的免费和加密替代方案，Paul Schreiber、Jacob Hoffman-Andrews 和 Yael Grauer 的 [视频会议指南](https://videoconferencing.guide/) 将帮助你找到适合你需求的应用。它对 28 个最常用的视频聊天应用程序进行了并排比较，并涵盖了每个应用程序的功能、限制、安全性和定价信息(如果适用)。关注开源软件是最明智的，因为这些应用程序通常没有广告，代码透明，更容易看到它对你的数据做了什么。

至于具体的建议，这里有三个来自视频会议指南的开源选择，得到了我们的支持。

### [**Jitsi**](https://jitsi.org/)T5】

Jitsi 是这个列表中最“传统”的免费开源视频会议应用。它的界面让人想起其他电话会议应用，但有一些关键的不同——最重要的是，它是透明的 E2EE 和安全功能，你可以 [在这里](https://jitsi.org/security/) 了解更多信息。它还让*通话中的所有*用户能够在有人未经邀请进入私人会议的情况下，将其他参与者静音或移除。

它在一次通话中支持多达 75 名参与者，并让你无限制地记录会议，这比其他电话会议应用程序更多——甚至是一些付费应用程序。Jitsi 可以在你选择的浏览器中自由使用，或者作为安卓、iOS 甚至开源 F-Droid OS 的应用 下载。你也可以通过一个简单的电话加入活跃的 Jitsi 会议。

### [**信号**](https://signal.org/#signal)T5】

Signal 经常得到安全通信的认可，我同意它通常得到的认可，特别是考虑到最近的增强，如 PIN 安全性。也就是说，与其他应用程序相比，它的视频聊天功能有限，因此在这个特定的列表中排名较低。Signal 仅支持两个用户进行实时视频通话，尽管其视频通信使用与一对一和群组短信相同的隐私和 E2EE 设置。通过 Signal 发送的媒体也是加密的，即使在群聊中也是如此，这有利于安全地共享视频、照片和录制的音频剪辑。媒体附件和文本甚至可以设置为在查看后或经过一定时间后自动删除。

你可能无法通过信号主持一个大型的集体电话会议，但是对于那些需要私人交流的人来说，这仍然是一个很好的资源。 [信号在安卓、iOS/iPadOS、Windows、Mac、Linux 上都有](https://signal.org/#signal) 。

### [](https://jami.net/)

**伊斯兰大教堂可能是目前最安全的视频聊天选择。它不仅在加密措施和隐私政策方面完全免费、开源和透明，而且是同类应用中唯一使用完全对等分发模式而不是更常见的基于服务器的模式的应用之一。这意味着*伊斯兰大教堂上的所有*通信纯粹发生在相关设备之间——永远不会向服务器发送任何信息。这保持了所有用户的匿名性和所有通信的私密性，甚至允许用户在连接到较小的本地网络时自由通信，而无需通过互联网发送数据。** 

**也就是说，伊斯兰大教堂的 P2P 分销模式也有一些特质，可能会限制其可行性。由于外部服务器不便于通信，伊斯兰大教堂对群组规模、通话记录、视频质量甚至数据传输速度都没有固有的限制，但你仍然会受到你的设备和点对点连接强度的限制。所有参与通话的用户还需要使用伊斯兰大教堂应用——它不像 Signal 和 Jitsi 那样支持与应用生态系统之外的用户进行交流。也就是说，伊斯兰大教堂可以在 Android、iOS、Mac、Linux 甚至 Android TV 上使用，所以找到一个兼容的设备应该不会太难。**
