# 如何在酒店获得升级

> 原文：<https://lifehacker.com/how-to-get-an-upgrade-at-a-hotel-1834579101>

我会向任何人要求升级，无论是餐馆里更好的桌子还是飞机上的头等舱座位。然而，获得酒店升级也有其自身的困难。你无法准确评估在你逗留期间有哪些空房或哪些房间会开放。



最近，NBC 新闻 采访了礼宾专家和《T4》[礼宾机密](https://www.amazon.com/Concierge-Confidential-Secrets-Serving-Champagne/dp/1250002737?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-get-an-upgrade-at-a-hotel-1834579101&asc_source=&tag=kinjalifehackerlink-20) 的作者迈克尔·法齐奥，询问客人如何在酒店获得更好的服务，在这里，额外的毛巾布长袍和房间升级等便利设施都是可以买到的。法齐奥斯在纽约洲际酒店的礼宾柜台后站了七年，他解释了为什么在与酒店员工打交道时，时机和礼貌大有帮助。

### 待在正确的日子里

如果你渴望升级，你可能会在周末有更好的运气；根据 Fazio 的说法，工作日通常是酒店最忙的时候，因为有商务旅客(所以在周五要求升级可能对你有利)。

早点入住也比晚点入住好。当客人登记入住时，在一天开始时可能有比接近结束时更多的升级选择，此时任何可能的房间升级都更有可能被采取。正如 [Travelz oo](https://www.travelzoo.com/blog/the-ultimate-list-of-ways-to-get-a-hotel-upgrade-2/) 所写，大多数酒店的清洁时间是早上 7 点到下午 3 点。如果你的入住时间是下午 3 点，那么在那之前到达也无妨，因为那时你获得更好房间的机会更大。

### 加入忠诚度计划

如果你是一个经常出差的人，选择同一家连锁酒店并加入它的忠诚计划会有所帮助；你可能会因此得到一些房间升级。按照 [TravelZoo](https://www.travelzoo.com/blog/the-ultimate-list-of-ways-to-get-a-hotel-upgrade-2/) 的建议，确保你的预订附有你的账户，并且在登记入住时不要害怕提及你的状态。

另一方面，预订一家全新的酒店也可能让你升级，因为他们试图招揽未来的业务并获得积极的评价。我最近住在火奴鲁鲁的一家新的精品酒店，甚至没有要求就获得了升级。

### 直接向酒店预订

根据法齐奥的说法，只要有可能，你应该直接预订酒店。下面是他的推理，正如 [NBC 新闻](https://www.nbcnews.com/better/lifestyle/how-get-better-service-hotel-according-career-concierge-ncna1001401) 写道:

> 酒店往往看不起通过打折旅游网站预订的客人。Expedia、Priceline 和 Orbitz 等廉价旅游网站和聚合网站的价格很低，但酒店有时不喜欢看到你从聚合网站上预订房间。这就像你进来时身上贴着一个红字，上面写着“便宜的价格”。

所以下次你考虑住酒店的时候，直接打电话给他们，要求预订房间。(通常情况下，他们会愿意与任何第三方预订网站上的价格相匹配。)办理入住手续时，看看他们是否愿意满足升级的要求。

### 提到你写评论

正如任何一家 [餐厅](https://www.vice.com/en_us/article/gvm37w/as-a-chef-i-wish-yelp-didnt-exist) 的服务员或酒店服务员可能会告诉你的那样，评论者是一个讨厌鬼。但正如法齐奥所说，委婉地让员工知道你是一名点评者，可能有助于你获得升级。你应该*而不是*用差评威胁他们。而是在你入住的时候礼貌的问酒店工作人员怎么写点评，这是一个足够明显的暗示。

正如我们之前所写的，如果你真的想要升级，就提出要求吧！“听起来很简单，但只要确保他们知道你会欣赏它(并知道这是一种常见的做法)就会增加你的胜算，” [戴夫](https://santorinidave.com/)[圣托里尼戴夫](https://santorinidave.com/) [。酒店和旅游评论网站 com](https://santorinidave.com/) 在一封邮件中表示。

如果你的房间不是你期望的那样——或者只是很糟糕——那就更有理由问了。“如果你觉得你应该得到提升，不要不好意思要求和经理谈谈，”他补充道。

任何时候都要有礼貌。礼貌可能是获得升级的最有效工具。
