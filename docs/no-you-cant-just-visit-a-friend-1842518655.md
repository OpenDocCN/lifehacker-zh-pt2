# 不，你不能“只是拜访”朋友

> 原文：<https://lifehacker.com/no-you-cant-just-visit-a-friend-1842518655>

本周不要去你朋友家。求你了。即使你和你的朋友都年轻健康，即使“只是一个朋友”，即使“如果我生病了也没关系，这是一个我愿意冒的风险”，即使“我们会小心，洗手，保证！”



我们正处于疫情的中心，生活在一个在几乎所有可以想象的指标上都搞砸了反应的国家。我们开发的最初测试没有成功。在有人发现之前，病毒已经在华盛顿州传播了数周。与此同时，本届政府的反应缓慢而随意，各州被迫自己动手做准备。美国对这次疫情的反应以最糟糕的方式挑战了所有的预测，这种拖延将会付出生命的代价。

这个疫情会变得多糟糕取决于我们所有的行动。是的，我们的行动。这包括你和我，你的朋友，你的父母和你的邻居，所有人都需要尽自己的一份力量来拉平曲线，防止医院不堪重负。

尽管现在压力很大，但严峻的现实是，我们正处于疫情的开端，而不是结束。最坏的情况还没有到来。我们仍然没有足够的测试。我们医护人员的个人防护设备严重不足。我们没有足够的重症监护室床位和呼吸机来应对激增的病例。我们都必须努力防止这种情况进一步蔓延。

我们的医生和护士站在第一线，冒着生命危险，为的是不让人们死亡。他们没有足够的资源。他们 [已经在做出难以想象的决定](https://www.vox.com/2020/3/22/21189896/coronavirus-in-us-masks-n95-respirator-doctors-nurses-shortage-ppe) 关于如何护理没有合适防护装备的病人。在短缺的情况下，医院已经被迫制定 [计划，决定谁能得到呼吸机，谁不能得到](https://www.michiganradio.org/post/draft-hospital-letter-leaks-outlining-who-gets-ventilators-shortage) 。如果这种情况持续下去，医生将不得不决定哪些病人将获得呼吸机，哪些不获得，因为没有足够的呼吸机给每个人。与此同时，意大利已经被 [强制配给呼吸机](https://www.statnews.com/2020/03/16/coronavirus-model-shows-hospitals-what-to-expect/) ，他们的 [人均医院床位比我们的](https://www.businessinsider.com/coronavirus-hospital-beds-per-capita-us-south-korea-italy-2020-3) 多。

医生和护士们希望人们呆在家里，以保证他们的安全，这样他们就可以工作来保证我们的安全。他们值得我们尊敬。更重要的是，他们需要我们呆在家里。这意味着不能和朋友见面。即使你觉得自己很健康，你仍然可以 [传染](https://www.discovermagazine.com/health/asymptomatic-carriers-are-fueling-the-covid-19-pandemic-heres-why-you-dont) ，你仍然可以让别人生病，甚至 [年轻并不意味着你不会出现严重的并发症](https://vitals.lifehacker.com/young-people-can-get-serious-complications-from-covid-1-1842415846) 。

[物理距离](https://vitals.lifehacker.com/were-calling-it-physical-distancing-now-1842454451) 硬。身体上的距离是孤独的。我们的生活有太多新的限制，太多我们不得不放弃的舒适，太多我们不得不推迟的庆祝活动。人们取消了生日聚会、毕业典礼、家庭团聚、千载难逢的假期和婚礼。对我们所有人来说，正当我们比以往任何时候都更需要这些友谊的时候，我们的正常活动已经嘎然而止，我们不得不放弃那些安慰我们的仪式。

我们都在一起。对我们来说，这意味着呆在家里。
