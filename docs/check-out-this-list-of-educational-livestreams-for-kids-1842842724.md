# 查看这份儿童教育直播列表

> 原文：<https://lifehacker.com/check-out-this-list-of-educational-livestreams-for-kids-1842842724>

每天，我们都会听到一个新的教育直播，我们的孩子可以或者应该收听这个直播。虚拟博物馆之旅，参观动物园和科学演示——甚至还有 [虚拟体育课](https://offspring.lifehacker.com/how-to-burn-off-that-little-kid-energy-1842620326) 。资源就在那里；挑战在于保持它们都是直的。



幸运的是，已经有人在为我们做这一部分了。我们通过 [kottke.org 的快速链接](https://kottke.org/quick-links/2020-04) 发现的 [Everywhere School](https://everywhereschool.live/) ，在疫情期间每天都在监管教育直播。有循环时间、烹饪课、艺术课、音乐会、舞蹈课——等等。

[列表](https://everywhereschool.live/) 很容易扫描，从当前正在直播的流开始，然后是一天剩余时间即将到来的预定流。如果你是一个真正的提前计划型的人，你也可以提前扫描，看看明天或本周晚些时候会有什么内容。在大多数情况下，即使视频流不是实时的，你也可以点击组织或主持人的 YouTube 频道或脸书页面，查看他们收藏的以前的视频。

请记住，和往常一样，你仍然应该审查你的孩子在看什么。我没有看到任何看起来不合适的视频，但 T2 的随处学校在其网站上说，虽然他们尽最大努力确保内容适合孩子，“但有可能不小心包含了具有挑战性或攻击性的内容。Everywhere School 将尽最大努力审查它推荐的视频流，但我们无法控制显示的内容。”

所以，关注这些事情，给他们的选择你父母的认可，这就是他们所说的。如果你知道其他应该包含在他们列表中的直播，你可以 [在这里](https://airtable.com/shrFfyYXe9mhnzwPQ) 提交。
