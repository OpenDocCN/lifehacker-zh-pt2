Windows 操作系统...没有那么多。虽然你的新台式机或笔记本电脑几乎肯定会配备 Windows 10，但操作系统的价格可能已经包含在你为系统支付的任何费用中。打造自己的 PC，你得自己掏钱买一份 Windows 10。如果你从旧的操作系统升级，你是否需要为新版本的 Windows 付费的问题会变得令人困惑。

为了减轻你的经济负担，这里总结了你可以用来获得免费或打折版 Windows 10 的所有最佳方法。

### 免费:无需激活即可运行 Windows 10

是的，你可以不激活 [安装并运行 Windows 10](https://www.microsoft.com/en-us/software-download/windows10) 。你的系统不会在 30 天后关闭，微软也不会把你锁在你的应用程序或其他类似疯狂的事情之外。你会被*唠叨到*激活你的 Windows 10 版本，你将无法个性化你的桌面——也不能个性化它的颜色——但这是微软为你运行一个未激活的操作系统版本设置的最大障碍。

你仍然可以获得所有最新的 Windows 更新，尽管微软当然保留在未来某个时候改变这种方式的权利(或者如果你愿意，可以将你锁定在操作系统的其他部分之外)。

### 免费:从旧得多的 Windows 版本升级

早在 Windows 10 首次亮相时，微软就大谈特谈其旧操作系统——特别是 Windows 7 或 Windows 8.1——的用户如何免费升级到 Windows 10。虽然微软不再谈论那种过期的“促销”，但这种伎俩仍然适用于许多人。老实说，这甚至算不上什么把戏。

只需在旧版本的 Windows 上运行 Windows Update，如果提示您更新到 Windows 10，请执行该操作。否则，使用微软的 [更新助手](https://www.microsoft.com/en-in/software-download/windows10) 或 [媒体创建工具](https://www.microsoft.com/en-in/software-download/windows10) 来运行您的更新:

我已经很多年没有这样做了，但我相信你会想确保你是在更新 Windows，而不是清理你的硬盘并安装一个全新版本的 Windows 10。这应该可以确保 Windows 10 在你迁移到新版本的操作系统时，正确地为你提供免费升级——一个可以与你的 Microsoft 帐户相关联的数字许可证。

在开始此过程之前，还值得记下( [或找到](https://lifehacker.com/magical-jelly-bean-keyfinder-finds-product-keys-for-all-1488686776) )您的 Windows 7/8.1 产品密钥，以防您在升级期间或之后需要它来激活 Windows 10。你不应该，但我会把它放在手边*以防万一。*

如果你选择全新安装而不是升级，你也可以尝试使用 Windows 7 密钥激活 Windows 10——home for home、pro for pro 等等。

Windows 10 Education 与 Windows 10 Enterprise 非常相似，对于大多数人在日常 Windows 生活中所做的事情来说，这无疑是大材小用。尽管如此，如果你是一名学生——你可以通过 Hub 服务 上的 [(微软自己也链接到这个服务)来证明这一点——你或许可以获得一份免费的 Windows 10 教育版，没有任何附加条件。](https://onthehub.com/microsoft-windows-10-education-for-students/)

(我和我的母校尝试过这种方法，希望我的校友电子邮件地址能让我获得一些免费的 Windows 10。唉，我被提示登录学校的门户网站以访问软件产品，但显然我不能这样做，因为我不再去那里了。值得一试！)

### 打折:买便宜的 OEM 钥匙

这是“买家当心”的终极举措。你可以在网上找到非常便宜的 Windows 10 OEM 密钥。这方面的问题很多。首先，从技术上讲，你不应该能够购买这些，但有人通过向你出售(希望如此)有效的密钥来赚钱，这些密钥是他们通过一些不太光彩的方式获得的——未使用的批量密钥、在 Windows 10 价格低得多的地方购买的灰色市场密钥、被盗密钥等等。最糟糕的情况是，你正在购买一个已经被使用过的密钥——或者在一个流行的 BitTorrent 列表中找到的一个普通密钥——你将失去你支付的微不足道的金额。

第二，您的 OEM 密钥将锁定一台机器。你仍然可以进行一些合理的升级——一个新的显卡，也许是一个不同的硬盘，或者更多的内存——但如果你做了一些激烈的 PC 手术，如更换主板，你可能会发现你的钥匙不再工作。Windows 会认为您试图在另一台机器上使用该密钥，并且会对此表示不满(就像您致电 Microsoft 客户服务代表试图解决此问题一样)。

最后，你的密钥可能会工作一段时间，但微软可能会随时取消它——特别是如果你所在的不那么合法的地方试图将密钥转售给其他人。你永远不知道什么时候你可能会被标记，即使是在你购买钥匙后不久，无论你从哪里购买它都可能不太愿意帮助你拿回你的钱(鉴于这种情况)。
