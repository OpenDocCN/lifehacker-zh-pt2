# 迎头赶上一座小山

> 原文：<https://lifehacker.com/tackle-a-hill-head-on-1845026700>

是什么让山变得可怕？有时候，很陡。你担心在到达顶峰之前会失去动力。如果是这样的话，读一读 [我们的跑步指南，不要打击你的信心](https://vitals.lifehacker.com/how-to-run-hills-without-crushing-your-confidence-1783126331) 。慢慢来，一小步一小步地走，在你努力达到顶峰的过程中，专注于保持轻松稳定的努力水平。

还有，停下来走走也没事。你认为超级跑者如何完成 100 英里的山地越野跑？我向你保证，不是每一步都跑。

有时候让一座山变得可怕的是它有多长。你可以使用同样的策略，但是调整自己的节奏变得更加重要。在一个真正长的山坡上，我喜欢保持尽可能低的努力水平— [慢跑如此缓慢](https://lifehacker.com/you-need-to-run-slower-1844984642) 我的心率不比我在平地上时高。如果需要的话，我还是会出去走走。

但是对于我们这些生活在丘陵地区的人来说，还有一个问题经常出现:那就是山丘是无尽的，你总会在最没有准备的时候遇到一个。

这是我本周为自己解决的挑战。我住在一个多山的街区，没有路可以绕过它。我甚至不能做一些简单的事情，比如把最大的山放在跑步的开始到中间，因为在某个地方总有*另一座山*。所以我出发了，我知道我是在和一堆又陡又长又不舒服的小山作战。我的挑战不是冲向或者保持一个好的速度，而是仅仅为了生存而不讨厌旅程。一些有帮助的策略:

*   计划一次彻底的撤退。这样，每一个山坡我都跑上去，我知道我也会跑下来。
*   让第一段成为上坡。这意味着我在回来的路上有一段下坡路。
*   在我的脑海中保持一段持续的独白，假装我正在和一个讨厌小山的假想朋友一起跑步。

这可能很傻，但最后一个让我坚持到了最具挑战性的部分。“这是最后一次上坡了，”我告诉他，但有时我是在撒谎。他对我坚持喝下去的能力感到敬畏。我差点把他甩在后面，但是阿姆的《直到我崩溃》来的正是时候(多亏了先进的虚拟朋友技术，他可以通过我的耳机听)，他成功了。

不管你对山的恐惧是什么，我希望你本周挑战它。去找一个陡峭的山坡，在上面重复跑，或者找一个长的山坡，坚持到底，或者跑一系列不可预知的山坡，坚持到底。如果你住在一个没有希望的平坦的地方，为什么不慢跑上体育场的楼梯，或者在下班后去一个停车坡道？试一试，让我们知道进展如何。
