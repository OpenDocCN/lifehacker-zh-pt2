# 如何与难相处的前任共同养育子女

> 原文：<https://lifehacker.com/how-to-co-parent-with-a-difficult-ex-1838221744>

所以，你和一个混蛋有了孩子。你可能不是故意的。也许在孩子出生之前，他们并不那么混蛋——或者也许当你不试图让渺小的人类生存下去时，他们的混蛋倾向更容易处理。无论如何，你现在在这里，既期待又害怕共同抚养孩子的漫长旅程。



努力建立积极的、相互支持的共同养育伙伴关系应该是我们的目标。但是有时候呢？你被一个混蛋缠住了，再多的同情、耐心或沟通都无法改变这一点。不过，还是有一些策略和界限，可以让你和这个混蛋的育儿过程不那么痛苦。

(现在，这些建议真的只适用于你离婚/分居的情况。如果你仍然和一个猪头住在一起，那是另一篇文章的主题。)

### 使用应用程序

你可以做的最简单的事情之一就是使用共同养育应用程序来简化沟通，避免面对面的争斗。希望*选择*一款应用不会引发争论(不过，我确信这并非不可能)。使用专门用于共同养育的应用程序的好处包括:

*   不可删除或更改的信息(即，如果您需要，您可以获得相关文档)
*   轻松扫描和发送文档的能力
*   共享日历
*   可以选择要求改变下车时间，或者要求报销费用

最重要的是，它将所有东西都放在一个中心位置。不再有连珠炮似的短信，接着是一封恶毒的电子邮件，接着是三个未接来电。至少理论上是这样。

有很多选择，但我们的 [后代养育脸书集团](https://www.facebook.com/groups/2018785615043946/) 特别推荐 [AppClose](https://appclose.com/?fbclid=IwAR01_mjIEk0mWNkvcFeRniowiLu8qD4L-ED8ygasEym5o62hVxVmQIuKn5c#Co-ParentingApp) 和 [共同养育者](https://coparenter.com/) 。

### 选择你的战斗

就像对待你的孩子一样，你将不得不和你的共同父母发生冲突。不幸的是，你不能每次在你的前任无理要求换日子的时候都把他们拖上法庭。但是你*可以*决定哪些问题对你来说是最重要的(也许是接种疫苗，或者孩子将去哪所学校或教堂)。如果你知道父母中的另一方有不同的首要问题，也许一些谈判或妥协是可能的，这样每个人都得到一点他们想要的。

另外，记住这个混蛋不会把每件事都做对(也就是说，用你会做的方式)。也许他们允许太多的屏幕时间，或者他们让孩子熬夜到很晚，每次回来后他们都筋疲力尽。我知道，每次都很烦人，但至少他们接种了疫苗。

### 走大路

你我都知道，我们都知道你不是混蛋。你的共同父母是。没有必要参与他们的荒谬行为。当面交谈时，尽量保持简短的交流，声音尽量中性(你可以稍后打电话给你妈妈，让她大声咆哮)。当他们通过你的应用程序向你发送一条特别愚蠢的信息或请求时，放下电话，让你的愤怒消退一会儿，然后再做出回应。

当你*做*回应时，保持简短和甜蜜。(或者至少简短亲切。)

“这一切都是为了创造一点点距离，这样即使他们真的出轨了，你也可以冷静下来，用合理、深思熟虑的回应进行交流，” [后代小组成员](https://www.facebook.com/groups/2018785615043946/) 朱莉说。

最后，永远不要在你的孩子面前谈论另一个父母是多么的混蛋。那会让你成为混蛋..
