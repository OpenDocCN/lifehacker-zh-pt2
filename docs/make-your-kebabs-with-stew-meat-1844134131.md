# 用炖肉做你的烤肉串

> 原文：<https://lifehacker.com/make-your-kebabs-with-stew-meat-1844134131>

现在不是炖肉的季节，一点也不是，然而杂货店继续出售炖肉，公然无视天气。这将是粗鲁的，几乎是嘲讽，除了炖肉可以用于非炖肉的应用，尤其是烤肉串。

Watch

这不是我自己想出来的原创想法。昨天我和我的朋友丹·伯克纳 聊天，他是一个多产的烤肉串(我想还有音乐)制作者，因为我知道他做了很多烤肉串，我想知道他用了什么腌泡汁。他(大方地)分享了两种卤汁的细节——一种巴尔干，一种四川，但我(自私地)继续提问。“你用的是哪块羊肉？”我问。“炖肉，”他回答道，“因为便宜。”

啊哈。

炖肉不仅便宜，而且已经被切成块，作为一个懒惰的便宜货，我对这两个特点感到高兴。牛肉炖肉是最常见的，但你也可以找到猪肉炖肉(肩肉)和羊肉炖肉(也是肩肉，但来自绵羊而不是猪)，所有这些都适合烤肉串。

如果你一想到烤羊肉串就开始抓狂，请冷静下来。虽然炖肉通常通过低慢的烹饪过程(比如炖)变嫩，但你可以用隔夜腌泡汁用化学方法嫩化你的肉块。我们这里有一份便宜又简单的卤汁清单，我相信网上还有更多让你尝试的。 [混合羊乳酪](https://skillet.lifehacker.com/add-feta-cheese-to-your-brine-for-delicious-juicy-chic-1782694195) 或羊乳酪包装在 中的 [液体也可以，就像](https://www.youtube.com/watch?v=FD59sjs9LPE) [洋葱蘸酱](https://skillet.lifehacker.com/marinate-your-chicken-in-leftover-onion-dip-1844133748) 一样，这是我正在谈论的新事物。只要确保你让你的肉在卤汁中炖至少八个小时，然后再串起来烧烤，以获得最嫩、最美味的结果。
