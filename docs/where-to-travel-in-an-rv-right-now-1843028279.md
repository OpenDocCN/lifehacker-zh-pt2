# 现在开着房车去哪里旅行

> 原文：<https://lifehacker.com/where-to-travel-in-an-rv-right-now-1843028279>

对于全国各地的旅行者来说，跟上最新的限制已经成为一项日常任务。有这么多停飞的航班和警告指示当穿越州界时该做什么，我们中的许多人都梦想着逃离。房车(休闲车)生活是现在寻找房车的方式吗？

Watch

2019 年， [美国超过 900 万人](https://www.thewanderingrv.com/rv-industry-statistics-trends-facts/) 拥有房车；超过 4000 万人还报告说，他们定期去房车露营。如果你是这数百万人中的一员，并希望在冠状病毒疫情期间利用你的房车， [CDC 指南](https://www.cdc.gov/coronavirus/2019-ncov/travelers/travel-in-the-us.html) 建议现在**不是公路旅行的时候**。最好把它停在你的院子里使用。

然而，对于 100 多万以房车*为家的美国人来说，这里是你可以安全乘坐的地方:*

### **国家公园**

国家公园是以公园为单位开放或关闭的。搜索公园的完整列表，查看您选择的目的地的条件 [这里](https://www.nps.gov/planyourvisit/alerts.htm) 。

然而，即使是在 开放的 [公园也专注于防止拥堵和过度拥挤并且要求任何游客:](https://www.nps.gov/aboutus/news/public-health-update.htm)

*   仅在指定区域停车
*   把你的访问安排在不忙的时候
*   与其他访客保持物理距离
*   把你带进公园的所有东西都带出来

请记住，创建这些公园是为了保护自然环境，包括植物和动物。关心你正在穿越的地区，清理好自己的垃圾，留意可能在你的房车附近或周围的动物。

### **旅行时停放房车的其他地方**

如果你在这次疫情期间住在或使用你的房车，你可以利用其他的选择。

#### **沃尔玛**

除了继续营业提供食品杂货、清洁产品、电子产品和服装在关闭期间出售，沃尔玛还在停车场的指定位置欢迎房车游客。你应该在到达之前联系商店的经理，以确保你不会被拖走，或者如果你需要固定一个仪表板标志来展示。

#### **卡车停靠站**

卡车停靠站也为房车旅行者提供空间。它们是一个很好的选择，因为它们是专门为忙碌和驾驶大型车辆的人设计的。请注意，卡车停靠站确实有礼貌规则供所有人遵守:

*   停在真正的停车位上
*   慢慢开进停车场
*   离开前检查你的车辆
*   燃料岛不是用来停车的，除非你在加油

留意你的周围环境，锁好门，待在光线充足的地方，练习安全的物理距离，以确保你的安全。如果你在旅行中需要使用任何地方的设施，戴上口罩，每次都要洗手。
