# 在这个网飞的复制品上观看经典的公共领域电影

> 原文：<https://lifehacker.com/watch-classic-public-domain-movies-on-this-netflix-clon-1842709466>

我喜欢良好的社区努力，并给予元过滤用户所有可以给予的“支持”，他们花时间浏览公共领域电影的列表，并设计了一个网飞克隆来呈现它们。

Watch

这个名为“[【vole flix](https://vole.wtf/voleflix/)”的网站，如果你热爱古典文学，是你浏览器书签栏的必备内容。这包括 1932 年的经典*最危险的游戏*，以及*恐怖小店*(20 世纪 60 年代，非音乐版)，永远令人愉快的 *Manos:命运之手*，以及*汽船 Willie* ，这是未来几个月内你可能会去的迪士尼公园。

这个网站可能看起来有点像网飞，但绝对不是网飞。你确实有一个各种各样的“队列”，你可以把你想看的电影添加到一个列表中，但是没有“账户”可言。换句话说，不要疯狂地建立一个收藏夹列表，因为有可能你的浏览器会在某个时候清空这个列表，毁掉你花在创建个人标准集上的时间。一次加*一部*电影，也许两部，但仅此而已。

此外，由于这些电影相当老了，不要指望在你加大马力的 4K 显示器上得到清晰的图像。这些老游戏的音频有时会有点困难，所以如果你在观看 Voleflix 时使用 5.1 环绕声设置，你应该确保调高音量，尤其是中置扬声器。你甚至可能想稍微调整一下亮度和对比度，以便更好地观看电影，不过要确保你首先记下你弄乱的任何设置的原始数字，并在完成后切换回它们。

否则，享受一次经典电影之旅。如果你录制了自己的 MST3K 附带评论(如果它很有趣),并将其发送给你的朋友，让他们在观看时收听，这将会加分。(否则，我不知道还有什么方法可以通过 Manos...)
