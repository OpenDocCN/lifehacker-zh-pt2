# 如何设定对新保姆的期望

> 原文：<https://lifehacker.com/how-to-set-expectations-with-a-new-babysitter-1839831186>

和一个新保姆开始一段关系可能有点伤脑筋。你希望你会喜欢他们，你的孩子也会喜欢他们，这种感觉是相互的。你希望这是一段美好友谊的开始，所以你最不想做的事情就是在友谊开始前就制定法律。但是你必须。

Watch

好吧，“制定法律”并不是完全必要的，但是坦诚说出你的期望是很重要的——即使所说的保姆是你同事的孩子或者你姐姐的朋友，这样做感觉有点尴尬。因为在他们一再违反屏幕时间规则后，不得不晚些时候做这件事会尴尬得多，他们甚至不知道你已经违反了规则，或者没有问他们的男朋友就邀请他们过来，因为他们认为这很好。

### 把这些都写下来

当你第一次带领他们进入厨房并交出两页说明书时，你会看起来像 A 型父母吗？是啊，可能吧。不过，这实际上是一件好事，因为你可以用你的彻底性作为自嘲和缓解尴尬的借口。“我知道，考虑到我只离开两个小时，这有很多话要说，”你可能会笑着说。“把它写出来对我更有帮助，因为当我试图出门时，我总是记不住要告诉我们的保姆所有的事情！”

然后，口头上打亮点。你不需要一字一句地指导他们，但你应该指出你的优先事项，无论是孩子在某个时间上床睡觉，还是限制垃圾食品。如果这是你希望经常雇佣的人，你可能需要在这里多花一点时间，确保他们了解你的家规，你的“客人”政策，你孩子的就寝时间，等等。如果这可能只是一次性的情况，或者你认为你不会经常依赖的人，你可能会决定放松一点。

### 要解决的主题

当你写下你的规则和期望宣言时，会有很多内容要涉及。苏西·泽尔丁是智能坐椅的共同拥有者和运营总监，她告诉[【Care.com】](https://www.care.com/c/stories/4331/babysitter-responsibilities/)你首先要考虑你对保姆的期望:

> 泽尔丁说，沟通是确保你的保姆理解别人对你的期望，并觉得自己有能力扮演好自己角色的关键。
> 
> 她解释道:“最重要的是，不要用一份他们不知道的责任清单来蒙蔽你的保姆，然后他们来到你家，感觉设置和最初沟通的不一样。”

举个例子，你可能会要求他们准备好孩子的晚餐，清理餐桌上的碗碟，但你不会要求他们装载和运行洗碗机。一旦这些期望明确了，你就可以制定一些基本规则了。这里有一些问题供你考虑:

*   您想设置什么屏幕时间限制？
*   你的孩子对食物过敏吗，或者你对孩子什么时候可以吃有限制吗？
*   你能接受保姆在照看孩子时邀请(经批准的)客人过来吗？接私人电话呢？
*   你希望他们不要在社交媒体上发布你孩子的照片吗？
*   他们应该怎样——出于什么原因——管教你的孩子？
*   你希望他们在多大程度上与你的孩子互动/玩耍？(如果他们不知道从哪里开始，你可以建议他们一起做一些活动。)

### 如果你一开始没有这么做

也许你认为这将是一个一次性的情况，这个人变成了一个定期保姆。或者你认为他们知道你不在的时候你不想让孩子们一直呆在电视机前，但是你很确定事情就是这样。你越早解决越好。

一两天内一条简单的短信应该可以:“嘿，孩子们和你在一起很开心！我想提一下，当我们回来时，我注意到他们正在看一个节目，我们通常不会让他们在下午 6 点后出现在屏幕上。忘记告诉你是我的错！仅供下次参考！”或者你可以在他们下次来的时候说:“哦，顺便说一下，我们真的在努力减少屏幕时间。我们过去对此太松懈了！你能确保在下午 6 点前关掉所有的电子设备吗？”

如果你真的需要重新开始，你可以打印出你一开始就应该给他们的那两页笔记，并说:“我知道，我应该在几年前就这样做，但生活太疯狂了！很抱歉让你悬而未决，直到现在。您做得很好，但希望这将有助于解决您对我们如何解决的任何问题(在此插入棘手问题)。”

很有可能，如果你有一个值得继续留在身边的保姆，这就是你需要做的全部事情。此外，随着时间的推移，事情会发生变化，孩子会成长，规则也会发生变化。这些对话将会在进行，甚至与长期保姆进行，所以对它感到舒适是有好处的。
