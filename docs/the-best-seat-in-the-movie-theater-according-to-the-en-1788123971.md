# 据一位 THX 工程师说，这是电影院中最好的座位

> 原文：<https://lifehacker.com/the-best-seat-in-the-movie-theater-according-to-the-en-1788123971>

虽然去 [很容易，但只要呆在](https://lifehacker.com/what-would-make-you-actually-go-to-the-movies-1821605002) 里，穿着睡衣坐在沙发上打开你最喜欢的流媒体服务，实际上*去看电影*看起来就像是一件大事。那，还有 [票价失控](https://lifehacker.com/heres-what-would-make-going-to-the-movies-worth-the-mon-1821647467) 的事实。(相关:T11】RIP movie pass。 )



所以，如果你要克服穿上户外裤和离开家的麻烦，你最好充分利用你的经历，从在剧院里挑一个好座位开始。但是具体在哪里呢？根据大多数剧院的设置方式，它将位于后面三分之二的位置，在这一排的中间。

正如 [THX 的](https://www.thx.com/) 全球技术总监史蒂夫·马茨 [向秃鹫解释的那样，](https://www.vulture.com/2016/10/how-to-find-best-seat-movie-theater.html?mid=twitter_vulture) 技术人员通过从座位上的几个点进行麦克风测量，在一个正在建设的影院中测试声音和视频质量。虽然我们的目标是让房子里的每个座位都有最好的体验，但你离他们测试的地方越近，你的座位就越好。这通常意味着你想坐在这一排的中间，离屏幕大约三分之二的距离。

> 这一组麦克风包括一个“主麦克风”，用于设置回放音量、扬声器定时和其他需要在一个位置进行测量和校准的参数。为了获得最佳音质，您需要尽可能靠近这个话筒的位置:大约在后面三分之二的位置，在这一排的中间。“THX 把每个座位都设计得很好，但大多数人坐在靠近主麦克风的位置会更好，”马茨指出。"然后随着电影院的爆满扩展到其他地区."

当然，无论如何，许多人已经自然而然地被这个领域所吸引(工程师们并不是随意选择这个地方的)。如果你想减少争夺最佳座位的竞争，试试 [找一家使用预订座位](http://lifehacker.com/movie-theater-showdown-amc-vs-regal-1776728825) 的影院。
