# 无需付费墙，从哪里获取冠状病毒新闻

> 原文：<https://lifehacker.com/where-to-access-coronavirus-news-without-a-paywall-1842300436>

说到冠状病毒新闻， [不乏误传](https://lifehacker.com/how-to-tell-whos-full-of-shit-when-theyre-talking-about-1842293545) 和不良演员。如果你想从可靠的来源学习，你可能会担心你会很快用完你这个月的免费文章，然后不得不相信一堆推文。



我们稍后将讨论付费新闻的重要性。现在，让我们把重点放在实用性上:你可以免费阅读主要新闻媒体。

点击新闻媒体的名称，直接进入该出版物的免费报道。

《纽约时报》 免费提供新冠肺炎报道。

《华盛顿邮报》 有一个免费的电子邮件简讯，提供冠状病毒相关的新闻。时事通讯中链接的所有文章都是免费的。

[《洛杉矶时报》](https://www.latimes.com/california/liveblog/coronavirus-live-updates) 冠状病毒 liveblog 免费提供。

根据该报的一条推特消息，芝加哥论坛报 免费提供一些冠状病毒内容。

为冠状病毒文章创建了一个免费版块。

[《大西洋》](https://www.theatlantic.com/category/what-you-need-know-coronavirus/) 正在免费提供所有冠状病毒相关的文章。

[明星](https://www.thestar.com/news/world/coronavirus.html) 正在给与冠状病毒有关的免费文章贴上红色的“免费数字访问”标签。

[守护者](https://www.theguardian.com/world/coronavirus-outbreak) 没有付费墙。

[STAT](https://www.statnews.com/tag/coronavirus/) ，覆盖健康和医药，免费提供冠状病毒保险。

另一个可靠的赌注:看看你家乡的报纸。许多人正在提高冠状病毒文章或整个网站的付费墙，以努力让当地社区了解情况。

如果你家乡的报纸碰巧是那些还没有取消收费的报纸之一(看看你，芝加哥论坛报和波士顿环球报)，你可以通过在一个 [匿名窗口](https://lifehacker.com/how-to-get-past-paywalls-in-chromes-incognito-mode-1835513589) 打开你想阅读的链接来绕开规则。
