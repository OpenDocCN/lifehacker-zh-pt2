# 使用此图表跟踪所有新冠肺炎候选药物和疫苗

你听说过 [羟氯喹](https://lifehacker.com/chloroquine-is-not-a-ready-to-go-wonder-drug-for-covid-1842471341) ，但你知道 Remdesivir、Roivant、Athersys 吗？如果你在关注一种可能的疫苗，你知道有超过 12 种候选疫苗正在进行试验或者希望很快开始吗？ [这个来自医疗保健新闻网站 STAT](https://www.statnews.com/feature/coronavirus/drugs-vaccines-tracker/) 的追踪器已经覆盖了你。

追踪器并没有包括每一种可能的药物和疫苗，但它列出了一些“最常被谈论的”,并对它们进行了分类，将发展最深入的放在图表的顶部。

对于药物来说，最重要的是 Remdesivir，它已经开始了第三期和第三期试验。它之前在 SARS、MERS 和埃博拉病毒上进行了测试，希望它可以作为一种通用的抗病毒药物。现在它正在新冠肺炎患者身上进行测试，尽管在中国的 3 期试验由于缺乏合格的患者而暂停。

在疫苗中，名列前茅的是 Moderna 的 mRNA 疫苗 ，该疫苗于 3 月份开始其第一阶段试验。没有一种候选疫苗已经开始第二阶段试验，这意味着我们还需要一段时间才能知道它们是否有效，更不用说在全球范围内生产和销售了。我们可能距离第一种疫苗还有几年时间(有史以来最快的疫苗 [用了四年](https://www.nationalgeographic.com/science/2020/04/why-coronavirus-vaccine-could-take-way-longer-than-a-year/) )，但有了这张图表，我们可以关注这个过程是如何进行的。
