我一直想学习如何编码。我想至少学会足够的编程语言来创建一些基本的自动化功能——或找出一些简洁的方法来控制我拥有的各种 LED 照明套件。

Watch

虽然你可以使用大量的选项(付费和免费)来开始编码，但微软又增加了一大批免费的教学视频。公司上传了一个 44 集的系列(！)到 YouTube 上为初学者配音的[Python](https://www.youtube.com/playlist?list=PLlrxD0HtieHhS8VzuMCfQD4uJ9yne1mE6)，你可以通过观看来熟悉这种编程语言。

每个视频的长度从 3 到 10 分钟不等，这是你在电话会议(希望你有一份更好的工作)、吃午餐或者想拖延时间时观看的完美长度。因为它们很容易理解，所以如果你没有完全理解微软想要分享的概念，它们非常适合多看几遍。(你也可以在 YouTube 评论中提问，前提是你不介意费力地浏览 YouTube 评论中的典型问题。)

微软也有 [一个单独的 GitHub 页面](https://github.com/microsoft/c9-python-getting-started) ，如果你想要更多的帮助、指导或代码样本，你可以查看。虽然对于那些以前从未真正涉猎过的人来说，编码可能显得有点不可思议，但是自学基础知识是完全可能的——你只需要花时间去尝试。
