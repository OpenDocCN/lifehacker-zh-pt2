# “Verbage”不是一个词

> 原文:[https://lifehacker.com/verbage-is-not-a-word-1837711169](https://lifehacker.com/verbage-is-not-a-word-1837711169)

如果你曾经不得不停下来问自己，“嗯，这里哪个词是正确的...*啰嗦*或*啰嗦*，“我为你准备了完美的记忆术:*啰嗦*是*垃圾*。

Watch

另一方面，“Verbiage”是一个名词，指的是使用太多单词的口语或书面语。说某事有“太多赘言”是多余的，因为赘言就是罗嗦；这是一篇长篇大论的演讲或文章。

例子:“简单的事实很难解读，因为它们隐藏在如此多的误导性措辞之下。”

“Verbage”只是一个常见的拼写错误，因为可能有些人不发“I”的音，或者他们更喜欢它的样子。但这是不对的，所以不要再用了。

对于那些已经准备好评论并告诉我这是语言如何演变的人来说，如果这个错误足够常见，那么非单词就变成了单词，等等等等，我会让 Maeve Maddox 在 [的日常写作技巧](https://www.dailywritingtips.com/do-we-really-need-%E2%80%9Cverbiage%E2%80%9D-and-%E2%80%9Cverbage%E2%80%9D/) 让你明白:

> Dictionary.com 的定义将“verbage”描述为将 *verbiage* 与*垃圾*同化，这不值得那些以编写一种标准英语为目标的作家去考虑。我们已经有了丰富的词汇可以传达词汇的无价值，其中之一就是**垃圾**。

看到了吗？Verbage 是垃圾，我们已经有很多垃圾了。
