# 用哈里萨做一道更烟熏、更辣的甜椒奶酪

> 原文:[https://life hacker . com/make-a-smokier-spicier-pimento-cheese-with-ha rissa-1841974951](https://lifehacker.com/make-a-smokier-spicier-pimento-cheese-with-harissa-1841974951)

我从来没有遇到过我不喜欢的甜椒奶酪，可能是因为很难搞砸蛋黄酱、各种奶酪和甜而芳香的辣椒的组合。我认为标准配方不需要太多调整，但是如果你想让你的甜椒酱更大胆、更烟熏、更辣，那就拿一管哈里萨吧。

如果你不熟悉烟熏胡椒酱，现在是熟悉它的好时机。主要用于北非和中东地区的烹饪，一管 harissa 的成分可能会因其原产国而略有不同，但你最有可能找到烟熏辣椒，大蒜，芫荽和香菜。这些都是很好的东西。我讨厌写食物的比喻——这是一种挑战——但这确实是一种“让一切都变得更好”的调味品(烤鸡、任何谷类食物、炒鸡蛋、血腥玛丽和最好的辣蛋黄酱都会立即浮现在脑海里。)

芳香的辣椒酱带来了相当多的热量，但乳制品为主的甜椒奶酪可以承受。每杯一汤匙是我喜欢的比例——它增加了一种美好的，挥之不去的烟熏香料，补充了甜甜椒和丰富的咸切达干酪。辣酱不是尝起来像“哈里萨奶酪”，而是通过给它们一些经得起考验的东西来增强已经存在于酱料中的味道，让你更加欣赏它们。你可以把它加到一批自制的中，但这也是吸满一桶商店买的东西的最好、最简单的方法。
