# 为什么伊丽莎白·沃伦的学生债务计划没有那么激进

> 原文：<https://lifehacker.com/why-elizabeth-warrens-student-debt-plan-isnt-that-radic-1834240902>

你可能听说过伊丽莎白·沃伦提出的计划 取消现有学生债务，给新生免费大学教育的机会，并通过“超级百万富翁税”(对财富超过 5000 万美元的家庭每年征收 2%的税)资助这一切。

你可能也看过菲利普·克莱恩在《华盛顿检验者》上发表的病毒式社论，他声称“伊丽莎白·沃伦取消学生贷款债务的计划将是对所有努力偿还贷款的人的一记耳光。”T3】

好吧。事实证明，我们中有很多人会非常高兴地看到别人的学生贷款被免除，即使我们努力偿还自己的贷款。在推特上，对克莱恩高度渲染的推文 的回应包括“监管食品的计划将是对所有死于沙门氏菌的人的一记耳光”和“让事情变得更好的事情是，这样做是为了让更多的人不再受苦。这是一件好事，我的好伙计。”

克莱因的社论还认为，“那些更负责任的人会理所当然地对那些可能更挥霍的人现在将获得政府救助的想法感到愤怒，”这同样是错误的。

不是人们可能会感到愤怒的部分——我的意思是，他们中的一些人可能会——而是负债累累的毕业生和即将入学的学生将刚刚得到政府的救助。

因为，正如乔尔·马西斯在本周提醒我们的 ，政府长期以来一直在资助大学教育。

事实上，我们当前的学生债务危机部分与政府减少了以前对学院和大学的资助有关:

> 如今不断增加的学生债务很大程度上是政策选择的结果。简而言之，学生债务增加是因为大学学费上涨，而大学学费上涨很大程度上是因为全国各地的州立法机构已经慢慢放弃了资助公立学院和大学的承诺。
> 
> 大衰退是这个故事的反面角色之一；这促使立法机构削减了对高等教育的资金投入，而且在很大程度上，这些削减再也没有恢复。根据 [一项分析](https://www.cbpp.org/research/state-budget-and-tax/unkept-promises-state-cuts-to-higher-education-threaten-access-and) ，2018 年公立学院和大学的国家拨款比 2008 年的水平低 70 亿美元——这是在调整通货膨胀后的数字。公立四年制大学得以生存的一个方法是在那十年间平均提高 36%的学费。并非巧合的是，在 2004 年至 2017 年期间，美国的 [学生债务翻了两番](https://www.pgpf.org/blog/2018/07/the-facts-about-student-debt) ，从 3450 亿美元增至近 1.4 万亿美元。
> 
> 今天的年轻学生并不比他们的前辈更负责任。他们得到的帮助要少得多。

伊丽莎白·沃伦不是唯一一个致力于降低大学费用和免除学生贷款的总统候选人。市场观察 列出了每位民主党总统候选人的高等教育政策平台，每位候选人目前都在宣传大学的可负担性——此外，许多人已经发表声明，表示他们希望尽可能多地免除学生债务。

因此，如果你觉得政府可能会帮助其他人减少大学费用的想法有点先发制人，请记住，政府过去曾提供过更多的帮助。这种让大学变得更负担得起的新举措更像是回到我们过去的状态，就像亚历山大·奥卡西奥-科尔特斯(Alexandria Ocasio-Cortez)提出的税收计划一样——该计划重组了边际所得税税率，让最富有的美国人支付高达 70%的税率——实际上是试图让我们回到 20 世纪 50 年代和 60 年代的税收等级。

这些拟议中的经济变革可能不会让你对今年支付的税款或所有用于学生贷款的钱感觉更好，但(正如经济学家可能会提醒我们的那样)这些都是沉没成本。现在是时候考虑未来了。
