# 何时检查自己是否饥饿

> 原文：<https://lifehacker.com/when-to-check-if-youre-hangry-1832430313>

我一天中最关键的时刻是到家后的前三分钟。我正处于人生的最低点。如果我被要求做决定或解决问题，我会回答无用或急躁。我消除不了那个小蠢期。但是我已经找到了解决办法，通过关注我的身体。



回家的头几分钟通常是我一天中身体最不舒服的时候。我穿过地铁站和狭窄的人行道，拎起杂货，然后跑上楼梯，我仍然会因为夏天的炎热或冬天的大衣而感到过热。所以我学会了推迟任何真正的讨论——甚至是晚餐计划——直到我放下我的东西，冷静下来，也许吃了一只温州蜜柑。在这三分钟里，我的智商会恢复 10 分。

有时这种身体检查被称为暂停:检查你是否饿了/生气了/孤独了/累了。愤怒和孤独是很难解释的(我总是很生气，队长)，所以我喜欢先检查一下身体因素。我是否因为一个直接的身体问题而产生强烈的情绪？我需要水吗？我需要尿尿吗？我需要有人捏我的肩膀吗？我需要*放下东西*吗？在《科学美国人》中，大卫·普兰斯将这种身体意识 [【内视知觉】](https://blogs.scientificamerican.com/observations/weve-lost-touch-with-our-bodies/) 称为“内视知觉”，这个词已经存在了 100 多年，但却没有得到应有的重视，取而代之的是“正念”你可以通过传统的“正念”活动来发展你的内观，比如冥想，也可以通过简单的停顿式检查来发展。

当我和 Lifehacker 的同事提出这个问题时，每个人都立即举了一个例子。在晚上的某个时候，高级视频制作人乔尔·卡恩意识到他需要拿出他的隐形眼镜，这样他才能清醒地思考。工作人员作家乔希·奥坎波从健身房回来后无法完成工作:“我太放松了，以至于无法完成任何事情。”

临时育儿编辑梅根·沃尔伯特(Meghan Walbert)晚上会变得暴躁，除非她有吃零食的休息时间。健康编辑 Beth Skwarecki 知道，当她毫无理由地打开新的浏览器标签时，她需要吃东西。

财经作家艾丽西娅·亚当奇克(Alicia Adamczyk)在早上通勤时，“对地铁上的每个人都出汗和恼火，”除非她给自己足够的时间喝杯咖啡，“还有适当的时间让我的润肤霜渗入，这样当我被挤在人群中时，就不会让我汗流浃背。”

“早上我需要提醒自己不要检查我的身体，”贝丝说。“醒着的前 15 分钟，任何一点点的疲劳或肌肉酸痛或鼻塞，我都确信我要死了，起床是没有用的。”

作家布伦丹·黑塞(Brendan Hesse)用“一种近乎持续的、高度集中的自我反思思维循环，反复检查我为什么和如何感觉，以及那种状态和我的想法之间的最终凝聚力”来处理 ADHDADHD 药物可以使这个循环安静下来，但也使他对饥饿和温度等事情的意识变得更少。

作家 Aimee Lutkin 做了一个反向检查:如果她度过了一个糟糕或怪异的夜晚，她更有可能吃得过多。她说，“我在晚上会变得更加情绪化，所以我从不在天黑后发送重要的电子邮件。”

《科学美国人》的 Plans 说，我们现在的文化在避孕方面很糟糕。就我个人而言，我总是对身体需求可以控制我的情绪这一想法感到恼火。但是，我的许多坏情绪都可以通过物理修复来解决或缓解:如果我妥善安排我的办公桌，我就不会那么紧张；如果我看得比自己的身体更远，我就不会在人群中惊慌失措 。

在剪辑中，伊迪丝·齐默曼 [自嘲地强调了“只是”一词在描述这些感受](https://www.thecut.com/2019/02/what-is-interoception.html) 时的用法:只是我们肌肉酸痛，只是饿了，只是累了，好像吃饭、睡觉和拥有身体都是边缘活动。我们喜欢把自己想象成装在容器里的大脑，或者是可以依靠气体运行的超级英雄。但我们不是超级英雄。如果我们想让任何人认为我们是，我们最好吃点零食，穿上毛衣。
