# 用啤酒烹饪的统一理论只有一个简单的规则

> 原文：<https://thetakeout.com/unified-theory-of-cooking-with-beer-rule-guide-1838149318>

虽然将啤酒与特定的食物搭配起来介于艺术和科学之间，但将啤酒作为一种配料来烹饪却不那么费力。我做过 [熟](https://thetakeout.com/what-s-the-best-beer-to-cook-with-1829392862) 和 [啤酒烤](https://thetakeout.com/recipe-breakfast-stout-muffins-beer-baking-1829345041) ，虽然后者需要精确和深谋远虑——毕竟是烘烤——但前者非常简单。记住一条关键规则，你就很难被引入歧途。



这一条规则，我称之为我的用啤酒烹饪的统一理论(商标正在申请中)，将在你 95%的用啤酒烹饪的实验中很好地为你服务:

如果啤酒是棕色或琥珀色的，没有太多的啤酒花，而且足够可口，你可以用它来烹饪。

现在让我们一条一条地检查这条规则。棕色和琥珀色啤酒的色泽来自酿造它们所用的特制麦芽。这些麦芽比浅色麦芽烘烤程度更高，它们的颜色来自加热和干燥过程。像想到烤面包一样想到麦芽谷物:颜色越深，烘烤、烘烤、焦糖化或事件烧焦的味道就越强烈。一般来说，棕色和琥珀色的啤酒会提供我们在美味菜肴中喜欢的饼干、坚果、太妃糖和牛奶巧克力 [美拉德](https://en.wikipedia.org/wiki/Maillard_reaction) 口味，而不会转向烧焦、烧焦或烧焦的领域。

但是一些棕色和琥珀色的啤酒也很好喝，这就是我们用啤酒烹饪的统一理论的第二部分。啤酒花在食物中味道很好——有啤酒花油比萨饼，我喜欢在辛辣的沙拉调味汁中加入啤酒花——但它们通常比麦芽更难烹饪。这是因为啤酒花含有增加苦味的α酸——更不用说啤酒花本身的实际味道了，它可以从西瓜味和水果味到松木味和树脂味。在你的调味汁中加入大量的异丙醇，你可能会奇怪为什么它突然尝起来像松油。坚持喝麦芽啤酒，你会避免苦松木陷阱。

最后，就像厨师会告诉你不要用你不能忍受喝的酒做菜一样，不要尝试用啤酒做菜，那是、不新鲜的，或者对你来说不好吃的。一瓶不是你最喜欢的，但总体来说味道还不错的啤酒可以用来烹饪，就像你在冰箱里放了几个星期的啤酒一样。但是如果你品尝不良口味，不要把它放在你的食物中——就像你不会用 [腐臭黄油](https://thetakeout.com/is-it-safe-to-leave-butter-unrefrigerated-1821925356) 或 [陈奶](https://thetakeout.com/how-long-should-you-keep-milk-past-the-cartons-date-1823081376) 烹饪一样。

记住这三个简单的要素，你就能为绝大多数烹饪应用选择合适的啤酒。至于用在什么菜里，可能性很大。炖菜和辣椒可以从深色啤酒的烤麦芽中受益，而用琥珀色的麦芽啤酒为烤肉脱釉会带来一些饼干和焦糖的丰富。在许多情况下，你甚至不需要量出啤酒——给这道菜一点点，品尝，然后添加。那就请我过去吃饭吧。
