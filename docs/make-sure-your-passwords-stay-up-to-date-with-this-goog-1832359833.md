# 确保您的密码与此 Google Chrome 扩展保持同步

> 原文:[https://life hacker . com/确保你的密码与 this-goog-1832359833 保持同步](https://lifehacker.com/make-sure-your-passwords-stay-up-to-date-with-this-goog-1832359833)

在这个企业黑客和大量密码泄露的时代，拥有一个新的、未经授权的密码和拥有一个难以破解的密码一样重要。为此，谷歌刚刚宣布并推出了一个新的 Chrome 扩展，它有助于确保你的密码不会被泄露，并仍然保持你的账户安全。

Watch

这个名为 [密码检查](https://chrome.google.com/webstore/detail/password-checkup/pncabnpcffmalkkjpajodfhijclecjno) 的扩展程序会在你使用时获取你的用户名和密码，并根据泄露的凭证对其进行检查。在其 [公司博客](https://www.blog.google/technology/safety-security/google-password-checkup-cross-account-protection/) 上，谷歌表示，它用来检查你账户的被攻破的密码库包括 40 亿个不安全的名称和密码组合，你知道，这是一个很大的数目。

谷歌声称，密码检查不允许任何人读取或存储你的安全信息。注册表和您当前的安全信息都经过多次加密和哈希处理，并使用一种叫做 [blinding](https://en.wikipedia.org/wiki/Blinding_(cryptography)) 的技术进行交叉引用，这种技术允许扩展将您的信息与列表中的条目进行匹配，而无需读取或存储任何一项。您的密码状态只能通过使用分机本地存储在设备上的密钥来访问，因此，从理论上讲，您是唯一知道分机显示什么的人。

Password Checkup 确实会收集一些常规数据，包括谷歌数据库中的密码被检查了多少次，以及在显示密码不安全后是否会被更改。谷歌在其 [安全博客](https://1.bp.blogspot.com/-6PKL5w3qBEc/XFkw1XVbA9I/AAAAAAAAMis/mquatdaaXcsQsrvLlfUDM9SjA849AKUewCLcBGAs/s1600/technical_infographic.png) 上对这一过程有详细的解释，如果你想了解更多的话。

谷歌并不是第一家提供检查你的账户信息是否被泄露的服务的公司。1Password 是我们最喜欢的 [密码管理器](https://lifehacker.com/1passwords-updated-password-manager-is-a-must-have-for-1826379117) 之一，它提供了一个名为瞭望塔集成的功能，可以根据检查您存储的密码，这是一个巨大的、可公开搜索的泄露密码数据库。

密码检查和瞭望塔集成之间有一个重要区别——谷歌的选项是免费的。你可以随时 [手动检查你的账户](https://lifehacker.com/how-to-secure-your-accounts-after-the-massive-collectio-1831835847) 是否已被破解，但密码检查会重复自动进行，这增加了你及时发现错误密码以保护自己的可能性。
