# 用罐装桃子汁做贝里尼

> 原文：<https://lifehacker.com/make-a-bellini-with-the-juice-from-canned-peaches-1844053608>

桃子罐头里的桃子从技术上来说和你在杂货店能买到的新鲜的克林斯通样品是一样的，但是它们是非常独特的产品。桃子罐头比它们的绒毛罐头更加柔软易入口，这听起来令人不快，但实际上令人愉快。我可以一整天都吃那些桃子。



桃子罐头的另一个显著特征是它们包装的液体。有时是果汁，有时是淡糖浆，有时是浓糖浆，但总是很美味，把它倒进下水道是只有傻瓜才会做的事，尤其是因为你可以把它倒进起泡葡萄酒里，并称之为“贝里尼”

那是谎言吗？算是吧。“真正的”贝里尼是由泥状白桃(有时浸泡在葡萄酒中)和普罗塞克制成的，这绝对不是那个。我不会说它“没用”，但我会说它“容易接近”，这也是我对自己的描述。你也可以称之为“节俭”或“低浪费”，如果你想找些东西来称呼它的话。

总之。这很容易做。虽然大多数贝里尼食谱都是加入两盎司果泥，然后在果泥上加四盎司普罗塞克，但我喜欢在起泡酒后加入糖浆/果汁，因为糖浆/果汁比酒更浓，当你在较轻的成分后加入较重的成分时，饮料会自然混合。两盎司的果汁/糖浆是一个很好的量，尽管如果你的桃子是浓糖浆的话，你也可以尝试一盎司半。无论哪种方式，你都应该冷藏所有东西——果汁/糖浆、普罗塞克和玻璃杯——然后将两种冰镇液体倒入冰镇玻璃杯中，尽情享用。罐装桃子糖浆和果汁可以在冰箱里保存大约一周，所以你可以做一些，不过我建议把它们转移到一个单独的容器里，而不是储存在一个敞开的金属罐里。
