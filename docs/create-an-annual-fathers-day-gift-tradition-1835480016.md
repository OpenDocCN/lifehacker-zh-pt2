# 创造一年一度的父亲节礼物传统

> 原文：<https://lifehacker.com/create-an-annual-fathers-day-gift-tradition-1835480016>

我认为买父亲节礼物比买母亲节礼物更有挑战性。每年，向妈妈表达感激之情的礼物都很常见，也很受欢迎:珠宝、鲜花、一株植物、足疗或按摩的礼券。但是父亲节又来了，我们被垃圾创意轰炸，比如钢笔或烧烤配件。(一个人需要多少威士忌石头？答案是零。他不需要威士忌。)



理想情况下，一份好的父亲节礼物是对他为人父母的一种充满感情的肯定，也是他真正喜欢的东西。他不想要“世界上最伟大的父亲”的奖杯。所以你需要的是一个每年都可以建立的年度礼物的想法，这个想法变得如此根深蒂固，几乎成了一个笑话，但却是他期待的事情。这份年度礼物将结束你一直在谷歌搜索的所有疯狂的、最后一刻的“父亲节独特礼物想法”。

### 扭曲的年度照片

我的一个朋友在她最大的孩子还是个婴儿的时候就开始了父亲节的传统:用一些大的“D”和“a”摆姿势，然后制作一个“D-A-D”照片拼贴画(上图是她的女儿)。你可以从任何工艺品商店买到这样的信，或者简单地从硬纸板上画出来，让你的孩子装饰它们。

我高兴地从她那里偷了这个主意(谢谢，克里斯！)从我儿子小的时候起，每年六月，我们都会出去给爸爸拍照。每年，我都会把新的放在相框里，把旧的叠放在后面(每年我都会把它们看一遍，对生活充满美好和感动)。有一年，我们的养子在照片里。两年后，我们的狗出现了。去年，我儿子穿着他的空手道制服完成了整个拍摄。今年，嗯，我丈夫可能会读到这个，所以我要说的是，他的儿子把它自己设计成只有他会的样子。

我丈夫把相框放在他工作的桌子上，我这么多年来一直这样做，他知道在父亲节的前一周把它带回家，这样我就可以把新照片放进去，重新包装，就像它是一个全新的礼物一样。

你也可以用年度照片做各种有趣的变化。当然，你可以做标准的坐在玩具熊旁边的小孩，看看他们成长了多少。或者，你可以全力以赴地推出一个主题，就像我们的 [后代养育脸书小组](https://www.facebook.com/groups/2018785615043946/) 的成员斯蒂芬妮·硼想做的那样:

“我奇怪的想法是让她每年打扮成不同的超级英雄，”硼说。“如果我能让她一直这样做，即使她长大了，那该有多傻？”

不要担心这些照片带来的坏脾气或不幸。我每年都用这些照片制作一张照片卡片，和礼物一起赠送，这样我的丈夫既可以看到快乐的、被框住的儿子，也可以看到有点烦恼的儿子。事实上，正如我们小组的另一个成员所建议的，从过去的一年中找到你孩子的一张有趣的照片，并把它变成一个迷因，并把它框起来，这也可能是一个不错的年度礼物想法。

### 定制古怪的东西

我有好消息和坏消息。坏消息是，订购这些定制面袜并在父亲节前及时收到已经太迟了。好消息是，正如你已经看到的，他们 100%值得等待。

可以定制 DivvyUp 制作的 [，最多三面。你知道他每年都想要一副这样的眼镜，不管你的孩子采用了什么样的新签名。](https://www.divvyupsocks.com/facesocks) 

在很长一段时间里，我儿子的两颗门牙两边的牙齿都不见了，他养成了闭上眼睛做兔子笑的习惯，我们觉得很好笑，溅到袜子上看起来会很好。(迈克，只要你说一句话，一双鞋就会被送到这里。)

像这样以真正夸张的方式定制任何东西都是成功的。为了用定制的照片礼物打出全垒打，你必须全力以赴——越花哨越好。当然，你可以给他 [一个个性化的开瓶器](https://www.shutterfly.com/personalized-gifts/bottle-openers/gallery-of-two-bottle-opener?productCode=1328389&categoryCode=1363886&skuCode=1328390) 。他喜欢啤酒，他喜欢他的孩子，他会用它，这很好。或者你可以给他 [一条围裙](https://www.etsy.com/listing/667090698/personalized-apron-custom-photo-aprons?ref=shop_home_active_1&pro=1) 上面画着他孩子的巨脸，从上到下溅满整个画面。谁不想这样呢？

因此，选择一个你知道会让他微笑、大笑甚至流泪的传统，你未来的父亲节礼物采购将会容易得多。
