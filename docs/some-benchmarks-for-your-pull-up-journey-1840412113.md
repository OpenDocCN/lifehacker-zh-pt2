# 你引体向上之旅的一些基准

> 原文：<https://lifehacker.com/some-benchmarks-for-your-pull-up-journey-1840412113>

你想做引体向上吗？你是否已经做了一些，但还想继续前进？在我们的年终健康基准系列中，这里有几个级别的引体向上测试来衡量你的进步。



### 如果你还不能做一个

如果你不能做引体向上，我们将做定时悬挂。最简单的方法是设置你的手机给你录像，然后你可以检查你挂在吧台上的时间。或者，用老办法，看钟。(不过，不要把密西西比算在内:我们在压力下并不总是保持稳定的计数。)

从这些版本中选择，或者选择多个版本。我把它们从最容易到最难排序。

*   **死挂**:随便抓个引体向上的杠，能撑多久就撑多久。这可以锻炼你的抓地力，让你习惯挂在吧台上。(如果连这个都太高级了，那就把脚搁在什么东西上，比如凳子。)
*   **主动悬挂**:抓住杠铃后，将肩胛骨向下拉，使背部活动。当你再也坚持不住这个位置时，就结束计时。
*   **屈臂上吊**:进入最高位置做引体向上(你可以跳或爬上去)，尽可能长时间保持下巴在杠上的姿势。你可以通过反握使手掌面向你来使这变得容易一点。

如果你修改了这些练习中的一个，记下你是如何做的(例如，脚踩在踏板上做悬空动作)。如果你有一个先前的基准，将你的时间与你今年早些时候做的事情进行比较，并保存你的结果以供明年参考。

### 如果你已经做了引体向上

首先，恭喜你！无论你的健康水平如何，即使做一个引体向上也是一项重大成就。这里有三种方法来衡量你的进步；用你觉得最合适的。

*   **数数你的引体向上**。越多越好。
*   **数一数你的辅助引体向上**。如果你只能做几个完整的引体向上，这是一个很好的选择，因为一个和两个之间的差距太大了。在健身房使用阻力带或引体向上器械，并计算重复次数。确保记下你使用的是哪种阻力带或器械的设置。
*   **找到你能增加的最大重量，同时做一个完整的引体向上**。如果你做了 8-10 个以上的引体向上，那就把它当作一个有重量的练习。大多数健身房都有一条带链子的腰带，可以让你把重物挂在腰上。你也可以用背包或者双脚夹着哑铃来增加重量。你能举起多少？

再次，做适当的笔记。如果你做负重引体向上，考虑记录你的体重和增加的重量，因为毕竟你是在要求你的肌肉移动整个你+重量系统。

写下你做了什么，如果你有数据的话，和去年比较，期待一个强大的 2020 年！
