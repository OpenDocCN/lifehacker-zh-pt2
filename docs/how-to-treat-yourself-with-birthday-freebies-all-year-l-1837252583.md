# 如何用全年的生日赠品犒劳自己

> 原文：<https://lifehacker.com/how-to-treat-yourself-with-birthday-freebies-all-year-l-1837252583>

如今，世界上的每一家零售和快餐连锁店似乎都有一个忠诚度计划，好的连锁店提供的额外津贴之一是你的生日赠品或折扣。这些程序很少验证你的出生日期。



这意味着你的生日可以是你想要的任何时候。

将这些免费赠品和折扣在一年中分散开来，可以让你享受购物或外出就餐的即时满足感，而不会对你的钱包造成太大冲击。这个想法的起源完全是自私的:我的生日在 12 月，所以当我的生日优惠券出现时，我会把钱花在给家人买礼物或去旅行看他们上。虽然这些生日礼券可以很好地用作礼物，但它们通常只适合我自己的喜好，而不是我家人的喜好。我的爸爸，不管他有多爱我，都不会对我从 Ulta 得到的任何免费产品留下深刻印象，你知道吗？

下次你注册一家零售商的 [忠诚俱乐部](https://lifehacker.com/which-store-loyalty-programs-are-actually-worth-it-1776536641) 以获得他们为你来到地球提供的任何甜蜜或半甜蜜的折扣，记下一个随机的生日。为了简单起见，在一月一号，如果你渴望得到一份免费赠品，就在你注册后的那个月过生日。

这个技巧的专家水平是根据季节来计算你的折扣。如果你想用一双新鞋开始你的新年健身计划，你可以在一月份注册获得你的生日优惠券。

我是不是应该把这个建议留给 [恶周](https://lifehacker.com/tag/evil-week) ？大概吧。但是，嘿，如果我们与这些我们喜欢购买的公司交换我们的购买历史，我们至少应该能够做的是充分利用这些额外津贴。
