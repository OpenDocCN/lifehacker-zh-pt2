# 广告拦截的最佳浏览器(非 Chrome)

> 原文：<https://lifehacker.com/the-best-browsers-for-ad-blocking-that-arent-chrome-1835130559>

好像我们没有足够的理由让 [远离 Chrome](https://lifehacker.com/why-you-should-switch-from-google-chrome-to-firefox-1821879163) ，谷歌已经加倍努力改变内容屏蔽 [浏览器扩展](https://lifehacker.com/how-to-install-chrome-extensions-on-an-android-browser-1834279173)——包括广告屏蔽——在 Chrome 中的工作方式。



导致这种情况的技术变化有点复杂，但它们主要是由于一个新实现的系统，名为 Manager V3，用于解释 Chrome 扩展如何工作。这种转变导致旧的 API 被弃用，并打破了许多广告拦截应用所依赖的技术。这迫使开发者转向效率较低的内容拦截方法，这意味着在一个页面上部署更多的广告，即使你正在运行多个广告拦截扩展。

这个规则的唯一例外将针对企业版 Chrome 用户——这意味着只有一小部分用户能够在 Chrome 上获得完全无广告的浏览体验。当然，Chrome 在 2018 年推出了自己的原生广告拦截功能，但仅限于违反某些预定义显示规则的广告，其他的就不多了。用户已经转向第三方扩展来屏蔽其余部分。

尽管技术变化复杂，但变化背后的原因很简单:谷歌通过广告赚了很多钱，因此降低第一方浏览器的广告拦截器功效可以确保他们从更多广告中赚钱。然而，谷歌不会站出来这么说，而是将这些变化归咎于性能问题。不管是哪种情况，Manifest V3 都意味着广告拦截器在 Chrome 中的效果会差得多。

那么，是时候找一个新的浏览器了。让我们考虑一些替代方案。

## 最佳广告拦截浏览器

### [勇敢](https://brave.com/)

勇敢是目前为止屏蔽广告最聪明的选择。这款浏览器专为无广告体验打造，内置广告拦截功能和反跟踪技术。它几乎在互联网的任何地方都使用安全 HTTPS。此外，与大多数其他选项相比，不加载广告意味着更快的浏览体验。

具有讽刺意味的是， [Brave 最近开始了一项新的新颖的选择性加入广告服务](https://lifehacker.com/get-paid-to-watch-ads-in-the-brave-web-browser-1834332279) 。这是一种与你通常在在线广告中发现的完全不同的模式，甚至向选择在浏览时定期查看少量广告的用户付费。Brave 希望新系统将彻底改变在线广告的工作方式，但好消息是，Brave 浏览器中的网页仍然完全没有广告，即使你决定获得定期广告。如果我们被迫在广告体验中做出选择，我们会很容易地选择 Brave 的当前模式，而不是 Chrome 的页面广告。

### [火狐](https://www.mozilla.org/en-US/firefox/new/)T3】

当人们在寻找 Chrome 的替代品时，火狐很可能是第一个想到的浏览器。也有很好的理由:Firefox 的默认隐私设置比 Chrome 的现成设置好得多，而且你可以自定义大量设置来微调你的浏览体验。Firefox 的移动版本也是目前表现最好的移动浏览器之一。

值得简要注意的是，尽管有 [内容屏蔽功能](https://lifehacker.com/how-to-block-auto-play-videos-using-firefox-1833493848) 和附加功能，火狐最近在浏览器扩展 方面也有自己的 [问题。也就是说，无论是作为 Chrome 的替代品](https://lifehacker.com/if-your-firefox-extensions-arent-working-update-your-b-1834552458) 还是其他，我们仍然给 Firefox 我们的一般推荐— [。](https://lifehacker.com/the-best-competitors-to-every-first-party-google-app-an-1834172092)

### [戏曲](https://www.opera.com/)

Opera 可能没有 Firefox、Brave 或 Chrome 那么受欢迎，但对于广告屏蔽人群来说，它是一个值得的选择。

像 Brave 一样，Opera 在默认情况下会屏蔽广告，尽管它使用的是广告屏蔽器的集成版本，而不是专有的后端技术，所以它不如 Brave 那么高效或快速，但仍然有效。

Opera 也是少数几个在桌面和移动设备 上都有免费内置 [VPN 的浏览器之一，尽管它可能会导致页面加载速度变慢，而且你需要修改一些设置，以便在所有浏览器场景中默认开启。](https://lifehacker.com/how-to-set-up-opera-s-mobile-vpn-for-secure-browsing-on-1832436689)

### 铬和边缘

微软新推出的 [铬基 Edge](https://lifehacker.com/how-to-beta-test-microsoft-s-new-chromium-based-edge-br-1833889177) 也是一个可能的替代方案。它有广告拦截插件，尽管它仍然依赖谷歌 Chrome 的大部分基础设施。

或者，如果你想要一个没有谷歌和微软的 chromium 体验，也有一个 [开源 Chromium](https://www.chromium.org/Home) 浏览器。你在谷歌 Chrome 中使用的许多相同的浏览器扩展也兼容 Edge Chromium 和开源 Chrome，包括一些广告拦截器。

### 托尔怎么样了？

Tor 被誉为最注重隐私的浏览器 ，但它不会默认屏蔽广告。但当 Tor 仍然显示广告时，这些广告将无法跟踪你，这是一个优势。Tor 完全匿名化您以及您在在线追踪者、广告和 ISP 面前的形象，从而使您无法与其他 Tor 用户区分开来。使用任何种类的浏览器扩展——包括广告拦截器——都是不被允许的，因为这会使你与其他使用 Tor 的人有所区别，从而更容易跟踪你。并不是说你*不能*屏蔽 Tor 中的广告——你可以——而是它会削弱 Tor 的功能。

### 全都试试

尝试所有这些选项来找到你最喜欢的可能是值得的。事实上，我们建议在任何时候至少安装两到三个浏览器，因为有些浏览器在某些任务上比其他的更好。尽管如此，无论你将哪个浏览器作为首选，那些想要减少在线广告(以及与之相关的一切)的人都应该避免使用 Chrome。
