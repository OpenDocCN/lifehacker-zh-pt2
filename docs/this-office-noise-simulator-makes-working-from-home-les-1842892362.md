# 这个办公室噪音模拟器让在家工作不再那么孤独

> 原文：<https://lifehacker.com/this-office-noise-simulator-makes-working-from-home-les-1842892362>

设计可爱的微型网站[Reichenbergerstr 121](https://imisstheoffice.eu/)模拟办公室噪音——如果你在家工作，并且你想念在办公室、咖啡厅或其他人与你一起工作的任何地方工作的声音，这很方便。



原色的同事形状在现代主义的工作空间里跳跃。你点击同事和办公家具来触发聊天、打字、倒水、办公室乒乓、电话铃响、椅子发出的小屁声，当你在椅子上移动时，椅子会发出这种声音，然后故意再次移动来证明是椅子。有时他们会主动发出噪音。你可以用低沉的德语切换周围的工作场所对话。网站网址是 imisstheoffice.eu，(创作者是一家名为 [kids](https://www.thekids.eu/en/) 的瑞士和德国创意机构；网站名称是他们柏林办事处的地址。)

这是一个通用的解决方案。为了完美地复制我多年来工作过的办公室，我的共鸣板需要一些额外的东西

*   两个试图在开放式办公室里闲聊的人的窃窃私语
*   拉克罗伊罐头的开瓶
*   办公室里的每个人都认为复古的夏季果酱是一种 bop
*   平庸但勤奋的克里格机器咯咯作响
*   与一个叫费利西亚的人一起工作的营销经理得意洋洋地喊着“再见费利西亚！”每天 3 至 30 次
*   两个人为在大厅里撞到彼此而道歉
*   C-SPAN 广播一场国会听证会
*   神秘的笑声来自一个大家都是好朋友的地方
*   我的编辑想吃一顿有史以来最安静的午餐

我现在和一个伴侣以及一个蹒跚学步的孩子在一间公寓里工作；我和我的搭档每天都有多次视频会议。幸运的是，很多在开放式办公室工作的 [应对策略](https://lifehacker.com/how-to-stay-productive-in-an-open-working-environment-1443536319) 都适用。我们在早上分享我们的时间表，并制定一个计划来分享婴儿、好桌子和广播。我们尽可能多地使用耳机。当我们不想被打扰时，我们让彼此知道。我们尽最大努力让家有点，但不太像办公室。

如果办公室的声音对你来说是地狱，试试迅流状态推荐的几百张器乐专辑，或者我列出的 [非常好的背景噪音发生器](https://lifehacker.com/the-best-free-background-noise-generators-for-working-1836258033) 。
