# 知道“高级经济”和“超级经济”的区别

> 原文：<https://lifehacker.com/know-the-difference-between-premium-economy-and-economy-1835462892>

如果你不是经常乘坐飞机的人，飞机上的座位选择会让你很困惑。



例如，在最近一次飞往苏格兰的航班上，我看到一位女士在空乘人员面前崩溃了，因为她没有得到豪华的毯子、枕头或配套设施，而这些都是提供给她后面几排经济舱+座位上的高级经济舱乘客的。她认为这两者是一样的，她因为预订了飞机后面几排的座位而受到不公平的惩罚。“我和他们花了一样多的钱买票？”不止一次被挑衅性地吼过。她错了。

事实上，那些坐在高级经济舱的人为那些小奢侈品支付了更多的机票钱。多得多。

虽然肯定没有“为什么我坐在中间？”普通经济舱乘客的观点是，在我的航班上，愤怒的乘客并不孤单。本周, [康泰纳仕旅行者分析了两种票价的](https://www.cntraveler.com/story/whats-the-difference-between-premium-economy-and-economy-plus?fbclid=IwAR2uaxSogmod7FDrtfcXYBt_79KvbfXOmaIUUvRvQfxuzBLynuYUYX81_LM) 差异。

### 超级经济舱

几乎每家主要航空公司都提供经济舱座位，可以让你有几英寸的额外腿部空间。座位位于飞机的前部，这一区域的乘客通常可以更早登机，吃到稍微好一点的小吃(想想更好的格兰诺拉麦片和香蕉)，还可以免费喝酒，即使在国内航班上也是如此。

超级经济舱机票和传统经济舱机票的价格差异取决于您乘坐的航班，但国际航班的价格可能在 20-30 美元到数百美元之间。

如果您不确定是否需要升舱，您通常可以在登机口升舱到经济舱+,有时比您最初预订座位时花的钱要少，前提是您在办理登机手续时还有座位。

### 高端经济

溢价经济是一种完全不同的动物。我个人认为，最好的方式来描述这种类型的座位是“商务”，但这在技术上是不正确的。高级经济舱的座位没有头等舱或商务舱的座位好，但也比经济舱的座位好。美联航称这部分为“高级经济舱”，达美航空称之为“高级精选”

在飞机上，座位在自己的客舱中是分开的，提供额外的腿部和座位空间，稍微倾斜一些(尽管你不能躺下)，在长途航班上通常还有餐点和便利设施包等东西。

这个区域的座位价格通常是经济舱的两倍，所以你要为这种舒适付费。许多航空公司也有 2-4-2 布局的座位，这意味着你很有可能坐在中间的座位。就我个人而言，我会选择经济舱+的靠过道座位，而不是高级经济舱的中间座位，但如果你和靠过道的人一起旅行，这可能是一个不错的提议。

不管怎样，注意你预订的东西，确保你在你认为你在的地方买了一张票。
