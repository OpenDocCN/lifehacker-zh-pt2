# 如何安排你的朋友约会

> 原文：<https://lifehacker.com/how-to-set-your-friends-up-on-a-date-1832601558>

所以终于发生了。你遇到了一个很棒的人。他们聪明、有魅力、有工作——而且他们是你那毫无必要的吹毛求疵的朋友的完美伴侣。我们都有过这样的经历，你有一个朋友很适合另一个朋友，你肯定他们会一见如故。但是设置它们说起来容易做起来难，不像在繁忙的地铁站台或婚宴上的浪漫喜剧。

我和约会教练康奈尔·巴雷特谈了这次约会的来龙去脉，以及如果事情不完全按计划进行，如何让自己摆脱困境。

### 看生活方式因素

首先，在你撮合任何人之前，重要的是你要考虑未来的约会对象有什么共同点。根据巴雷特的说法，当你在结交朋友时，他们可能需要在五个方面与某人保持一致:年龄、幽默感、职业抱负、身体吸引力和政治。

不，他们不需要在这些类别中的任何一个上是平等的(不是每个人都想和他们的复制品约会，我们都有自己的偏好)。这是关于采取一种客观的方法，弄清楚他们是否能成为一对好夫妻。你能想象你的朋友们手拉着手或者互相开玩笑吗？如果是这样，你很幸运。

当然，根据巴雷特的说法，虽然他们在大多数方面可以匹配，但每个人都有自己的交易破坏者，巴雷特曾经介绍过两个政治信仰不同的相似的朋友。

“(我的朋友)说‘我就是无法想象自己会和一个投票给唐纳德·特朗普的人在一起’，他们甚至不谈政治，”他说。“对我来说，这是一个很好的提醒，如果这五大巨头中有一家不结盟，它们就不太可能有共同的未来。”

因此，在你开始炮制你的总体规划之前，确保他们确实有足够的共同点(并且他们在即将到来的选举中的偏好不会破坏交易)。

### 衡量他们的兴趣

假设你的朋友对你的另一个朋友或熟人感兴趣。或者你坚持认为他们会喜欢对方。

首先，不要把潜在的约会框定为一个圈套。“我会把这个想法说成‘我有一个人可能真的很适合你，你可能会喜欢他。“这是你愿意接受的吗，””巴雷特说很少有人会说‘不，谢谢。’可以说，你可以通过丢下手帕来引诱他们。"

也要衡量对方的兴趣。问他们是否有兴趣，不要有太大的压力。如果是的话，让他们交换电话号码，或者用其他不会让他们感到尴尬的方式联系他们(脸书朋友的请求是好的，尽管不完全是私人的)。

但如果这是单方面的事情会怎么样呢？好吧，让他们你的朋友知道这不是私人恩怨。正如《欲望都市》第一集曾经告诉我的那样，有时候，撒谎也是更好的选择。

### 保持低调

所以他们很感兴趣，正在安排约会！看看你，你这个邪恶的策划者。当你的朋友最终采取行动时，无论你做什么，都不要试图让情况变得更有压力。

让他们自己交流，不要过多介入(除了回答偶尔的八卦问题)。巴雷特说，不要仅仅因为你参与其中，就把你的朋友安排在一个大的团体中，比如邀请你参加的聚会。

“你不希望他们在有很多其他人的大型活动中见面，因为他们会非常害羞，感觉所有的目光都在看着他们，”他说。“这听起来不错，但会造成很大压力。”

什么是完美的第一次约会？一杯速溶咖啡或一杯葡萄酒。压力更小，如果他们喜欢，约会可以进展到另一杯咖啡或杯子。所以告诉你的朋友保持轻松，慢慢退出群发短信。

### 支持你的朋友

所以约会糟透了。或者不是每一方都感兴趣。或者只是他们的时间未到！那也没关系。我已经交了朋友，第一次约会后总是很尴尬，尤其是和共同的熟人。

“你想告诉人们这是关于化学，”巴雷特说。“如果一个人确实喜欢另一个人，而另一个人只是不喜欢他或她，那会让人觉得很私人。那感觉像是对你的价值和价值的判断，那是非常痛苦的。这就是为什么约会很糟糕。”

那么你应该如何应对呢？告诉你的朋友这和他们无关。这是一个“适合”的问题，巴雷特补充说。“把每一次约会和你采取的积极行动视为双赢。每一种方法。每条调情短信。你的每次约会。每一次右挥。大家都是双赢。你想爱上采取好的行动，而不去评判结果。”

如果两个朋友都不感兴趣呢？嗯，那很简单。你们可能无法一起参加同一个派对，但有时，一种联系可以带来另一种联系。

如果双方都感兴趣，那么，你就找到了媒人。退出这个场景，让他们在几年后因为你出色的工作给你发一份婚礼请柬。
