# 使用此地图追踪亚马逊雨林火灾

> 原文：<https://lifehacker.com/track-the-amazon-rainforest-fires-with-this-map-1837901523>

八月份，大约有 66，000 起大火肆虐亚马逊雨林，其中大部分是人为砍伐的结果。在 [最初拒绝](https://www.nytimes.com/2019/08/27/world/americashttps://www.nytimes.com/2019/08/27/world/americas/brazil-amazon-aid.html/brazil-amazon-aid.html) 数百万的对外援助后，巴西总统雅伊尔·博索纳罗最近 [禁止](https://www.cnn.com/2019/08/29/americas/brazil-amazon-bolsonaro-fire-ban-intl/index.html) 使用火来清理土地，包括雨林中的土地，为期非常保守的 60 天。

如果你想更好地了解灾难的范围，本周， [Buzzfeed News](https://www.buzzfeednews.com/article/benking/amazon-fires-map-august) 利用美国宇航局的可用数据，绘制了一张 8 月份亚马逊地区所有火灾的地图。使用地图，您可以将滑块拖动到任意日期，以查看当天火灾的规模和数量。

正如《华盛顿邮报》的所指出的，重要的是要注意一些在这样的卫星地图上并不明显的事情。首先，尽管公众认为亚马逊的中心正在燃烧，但许多火灾发生在雨林的边缘，而不是中心。(你会注意到火灾主要发生在亚马逊盆地的南部，大致是秘鲁下半部与巴西东北部马拉尼昂州的交界处。)

此外，最近的这些火灾大多发生在前几年已经被砍伐的地区；正如 [帖子所写的](https://www.washingtonpost.com/opinions/2019/09/05/were-thinking-about-amazon-fires-all-wrong-these-maps-show-why/) ，这些土地大部分被清理出来用于农业。出于这个原因，你不能必然地认为火灾与最近的森林砍伐有关——这一过程可能在火灾之前就已经在该地区发生了。

截至 [周三](https://www.nbcnews.com/think/opinion/amazon-rainforests-are-still-fire-if-trump-won-t-act-ncna1049556) ，雨林仍在燃烧。在 Bolsonaro 上周禁止用火来清理土地后的 48 小时内， [独立报](https://www.independent.co.uk/news/world/americas/amazon-fires-brazil-ban-burning-rainforest-climate-change-jair-bolsonaro-a9088396.html) 报道称，巴西大约有 4000 起新的*森林火灾，其中近一半发生在亚马逊地区。如果你想看火灾的实时地图，试试全球森林观察的地图；搜索过去 24 小时内的火灾警报。和我如果你想帮助保护雨林或生活在雨林盆地的土著社区，看看我们的指南寻找并捐赠给可信的组织。*
