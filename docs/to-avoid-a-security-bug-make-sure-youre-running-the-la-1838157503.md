# 为了避免安全漏洞，请确保您运行的是 LastPass 扩展的最新版本

> 原文：<https://lifehacker.com/to-avoid-a-security-bug-make-sure-youre-running-the-la-1838157503>

如果您是 LastPass 用户，您应该确保您当前运行的是最新版本的浏览器扩展。



[在今天](https://blog.lastpass.com/2019/09/lastpass-bug-reported-resolved.html/) 的一篇博文中，LastPass 承认了扩展中的一个 bug，该 bug 可能会允许恶意网站欺骗浏览器给它你在另一个网站输入的凭证。这可不是你想发生的事。

LastPass 表示，一旦它意识到这个由谷歌漏洞研究员发现的问题，它就开发了一个修复程序，并已部署到该扩展中。这个问题只影响了 Chrome 和 Opera 上的扩展。

该公司指出“不需要用户操作，你的 LastPass 浏览器扩展会自动更新，”但是确保自动更新已经到达你那里总是一个好主意。

要检查，单击 Chrome 右上角的三个点，然后选择更多工具，然后选择扩展。从那里，向下滚动，直到你看到 LastPass，点击细节按钮，然后点击“更新”手动更新到最新版本的扩展。

在宣布该问题的博客帖子中，LastPass 也提供了一些自己的安全提示。具体来说，不要点击你不认识的人的链接，永远不要重复使用你的 LastPass 主密码，并为每个帐户使用唯一的不同密码。你知道，基本的计算机安全。

该公司还建议始终为 LastPass 和你的其他在线账户启用多因素认证。如果你还没有设置好，现在是时候检查你所有的账户并确保你设置好了。
