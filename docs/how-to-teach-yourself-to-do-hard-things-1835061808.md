# 如何自学做难的事情

> 原文：<https://lifehacker.com/how-to-teach-yourself-to-do-hard-things-1835061808>

(据说)熟能生巧，但是我们当中有谁从来没有花大量的时间去练习一些他们甚至没有接近完美的东西呢？拿我来说，我十几岁的时候打了很多网球，在很多乐队里用单簧管摔跤了很多年，你猜怎么着？我在这两方面都没有超过平均水平。

Watch

太糟糕了，软件开发人员大卫·麦克弗的“ [完全通用的学习做困难事情的系统](https://www.drmaciver.com/2019/05/how-to-do-hard-things/) ”当时没在身边帮我。他创造了这个循序渐进的系统来帮助我们学习如何做困难的事情。“困难的事情”可能是从学习下棋到跑马拉松的任何事情；我们在谈论什么困难的事情并不重要。他这样解释他的过程:

> 这个系统“总是”在某种意义上起作用，要么“最终”你会发现为什么这个目标对你来说是不可能的，要么你会成功，但它是非常无益的那种最终，不能保证它不会花费无休止的长时间。更有可能的结果是，要么你会相对快速地成功，要么感到厌倦并放弃，但这没关系——这个系统的设计是为了让你在前进的每一步都遵循它，即使你没有实现最终目标，你也会从中受益。

因此，这不是一个保证，而是一个结构化的计划，如果你不知道如何开始掌握困难。他提供了两种方法——一种方法是如果你已经知道成功是什么样子，另一种方法是如果“成功”的定义更主观。在这两种情况下，您将遵循以下步骤:

> 1.找一些像一样难但容易的事情。

> 2.修改简单的事情，让它在你觉得困难的某一方面像困难的事情。
> 
> 3.做修改过的东西，直到它不再难。
> 
> 4.如果卡住了，请执行下列操作之一:
> 
> *   回到第三步，选择一种不同的方式来解决这个问题。
> *   递归地将学习做困难事情的一般系统应用到你所坚持的事情上。
> *   去问问专家或者橡皮鸭的建议吧。
> *   如果你在尝试了前三个之后仍然停滞不前，很可能你已经达到了某种自然的难度极限，可能无法取得进展。
> 
> 5.如果原来困难的事情现在变得容易了，你就完成了。如果没有，请返回步骤 2。

这个想法是，不要一下子把注意力集中在改善困难的事情上，你可以把问题分开，一点一点地掌握它的挑战性方面。

或者你发现这对你来说太难了，但是嘿，至少你尝试过了。
