# 如何使用开关的缩放功能使游戏文本变大

> 原文：<https://kotaku.com/how-to-use-the-switch-s-zoom-feature-to-make-game-text-1839259385>

任天堂 Switch 可以在旅途中玩这么多优秀的游戏，这很好，但有些游戏的菜单就像食品标签上的小字一样难以阅读。谢天谢地，这个开关有一个设计巧妙的变焦功能。



任天堂在四月 增加了变焦选项 [，但它仍然很容易被错过。那太糟糕了，因为它可以大幅改善在手持模式下玩游戏的体验。](https://kotaku.com/nintendo-adds-a-zoom-feature-to-the-switch-1834120841)

…你可以放大。

Bethesda 的 2016 *Doom* 游戏在 Switch 的掌机模式下运行确实不错，但是如果你想读取游戏的绝杀文本呢？当然，当然，阅读*末日*传说并不重要，但是想象一下它是重要的。当你看到的时候最好放大。

以下是如何使用缩放功能的方法:

首先，你需要在常规设置菜单下进入交换机的系统设置，并打开缩放选项。

这将设置双击主屏幕按钮的选项，以便在使用开关时显示缩放界面。请注意，缩放在系统范围内起作用，因此您可以在任何游戏中使用它，同时浏览 eShop 或者——这可能太元了——调整系统设置。

玩游戏的时候最得心应手。比方说，你正在玩《巫师 3》,你决定读一本游戏内的知识书籍。这些文字并不是很小，但是放大一些可能会有帮助。

所以，双击 home 键，缩放界面就会出现。

当你这样做的时候，游戏不会暂停，但是它会暂停系统操纵杆和按钮的正常功能。双击 home 键可以让你控制缩放本身，而不是控制你的角色或游戏菜单。X 和 Y 按钮放大或缩小游戏的图形，而左控制杆定位缩放的图像。对于巫师的书来说，知识显示在右边。正确调整后，缩放看起来像这样:

这个缩放级别可以通过再次点击 home 按钮来锁定。这样做会导致系统的操纵杆和按钮重新控制游戏。然后你可以在放大的时候玩游戏，这没什么用，除非有一些游戏真的不占用屏幕上的任何东西。

双击 home 键取消缩放，让游戏再次完整显示。

开关会记住您使用缩放功能的最后一个位置，这意味着您可以设置一个首选缩放，然后通过双击主屏幕按钮反复切换到该位置或远离该位置。在玩动作冒险游戏时，你可以用它来轻松地放大一会儿小地图。

在一个完美的世界里，Switch 的屏幕分辨率足够高，game designer 的用户界面设计足够仁慈，你的视力足够好，你就不需要使用 Switch 的缩放功能了。我们并不生活在那个世界里，但我们确实生活在一个拥有一些非常好的游戏的世界里，这些游戏在切换时几乎没有任何问题。

如果他们的问题是微小的文本，掌握系统的缩放功能应该会让事情好很多。
