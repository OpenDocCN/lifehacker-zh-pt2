# 向克赖斯特彻奇枪击案的受害者捐款

> 原文：<https://lifehacker.com/where-to-donate-to-help-the-victims-of-the-christchurch-1833322533>

新西兰克赖斯特彻奇市发生毁灭性的恐怖袭击，造成 49 人死亡。以下是你如何帮助受此次大规模枪击事件影响的社区。



### **新西兰伊斯兰信息中心**

作家伊斯克拉·汗分享了一篇内容广泛的《脸书邮报》 关于如何最好地支持正在处理新闻的穆斯林人民和社区的文章；她的帖子包括两个经过验证的捐赠链接。一笔是给[NZ IIC——新西兰伊斯兰信息中心](https://www.nziic.org/) 的钱。在他们的运动中，他们写道:

> 所有的收益都将用于满足悲痛家庭的短期需求。无论多少钱都无法让他们的亲人起死回生，但我们确实希望以某种方式减轻他们的负担。

可以在这里 捐赠 [。](https://www.launchgood.com/project/support_for_the_families__victims_of_the_new_zealand_mosque_shootings?fbclid=IwAR1BxJm3t6KnlKKovrpaA-XU4e_rog8vUzyKQL183Ra61OFzNkMLygOCKmk#!/)
