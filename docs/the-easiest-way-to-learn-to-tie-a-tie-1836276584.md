# 学习打领带最简单的方法

> 原文：<https://lifehacker.com/the-easiest-way-to-learn-to-tie-a-tie-1836276584>

如何正确地打领带被证明是成年后的一个挑战。

因为很多人很难记住这个过程的步骤(而且每次参加正式活动都需要搜索说明)，我们想出了一个记住如何打领带的方法。就像许多孩子用故事学习系鞋带一样，只要跟着这个简单的韵脚:

*   想象你的房子里有一只猫和一只老鼠。
*   当猫打盹时，老鼠飞快地跑了过去。
*   猫醒了，一场追逐开始了。
*   猫跳了起来，看到了晚餐。
*   绕着椅子，不是一次而是两次。
*   当老鼠藏起来的时候，猫往里面看。
*   尽管它在扭动，这只可怜的猫还是被紧紧地卡住了。

想象一下，领带宽的一端是猫，窄的一端是老鼠。上面的视频解释了押韵的每个部分如何指导你做什么，因为它非常简单，我们教一个 5 岁的孩子做它——完全靠他自己！
