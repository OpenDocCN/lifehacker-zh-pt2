# 如果你被一个虐待你的伴侣孤立，去哪里寻求帮助

> 原文：<https://lifehacker.com/where-to-get-help-if-youre-isolated-with-an-abusive-par-1842401757>

在冠状病毒疫情 期间，几周的大部分时间都呆在家里，这种想法足以让你厌倦家中的每个人。但是如果你和一个有虐待行为的人住在一起，呆在家里对你来说可能会更不舒服——或者完全危险。

Watch

家庭暴力支持组织正在张贴关于其热线电话和短信支持选项的提醒，承认与施虐者隔离会加剧本已紧张的局势。

可以拨打国家家庭暴力热线 1-800-799-7233。你也可以访问或者发短信 LOVEIS 到 22522。倡导者可以帮助你 [谈论你的选择](https://www.thehotline.org/help/what-to-expect-when-you-contact-the-hotline/) 并提供资源帮助你制定计划以保持安全。
