# 我们在 2019 年给出的最佳技术建议

> 原文：<https://lifehacker.com/the-best-tech-advice-we-gave-in-2019-1840662610>

52 周，52 [Tech 911](https://lifehacker.com/c/tech-911) 专栏——减去一两周我在度假，根本无法从远处回答你那些狂野疯狂的科技问题。虽然 Lifehacker 完全是关于服务和生产力的，但我在我们每周的 Tech 911 技术建议专栏中发现了一种特殊的快乐。那不是因为我写的。嗯，不是*，只是*，因为是我写的。相反，我喜欢有机会直接和 Lifehacker 的读者交谈。我喜欢听到影响你的硬件和服务的古怪问题，试图解决它们让我觉得自己像是在一个数字版的 Stump the Chumps(对那些过去热衷于汽车话题的人来说)。



尽管如此，我今年回答了很多疯狂的问题。但在我们总结我最喜欢的一些之前，2019 年的最后一个请求是:如果你想让我在未来的科技 911 专栏中考虑你的科技问题， [**给我发电子邮件**](mailto:david.murphy@lifehacker.com?subject=Tech%20911) **！我期待在 2020 年有很多话题可以聊。**

### [如何屏蔽邻居的 Wifi？](https://lifehacker.com/how-do-i-block-my-neighbors-wifi-1833464189)T3】

我住在硅谷，尽管如此，我的房子还是受到了大量无线网络的轰炸。谢天谢地，它们都集中在一个或两个无线频道上，给我留下了一点自由来建立我自己的不受影响的无线网络。如果你住在公寓楼里，或者大城市，我很同情你。你不仅会受到辐射，还可能时不时会遇到各种无线问题。

这是 2019 年最受欢迎的 Tech 911 帖子，我可以理解为什么。还有，不对，你可能不应该买“挡 wifi 漆”来解决。

* * *

### [如何搜索 PDF 评论？](https://lifehacker.com/how-can-i-search-through-pdf-comments-1833299295)T3】

布勒。布勒。这东西开着吗？*布勒？*这是 2019 年最不受欢迎的 Tech 911 帖子，让我有点惊讶。要么是我是唯一一个曾经处理大量 pdf 文件的人，要么是苹果公司——我的前雇主——对加价有点不满意。然而，PDF 通常很难使用，我很惊讶没有更简单(免费)的方法来搜索 PDF 评论。

答案？当谈到 pdf 时，很难抗拒 Adobe，但是如果您需要知道文档注释中的内容，至少有一些免费(或更便宜)的解决方案可供您使用。

* * *

### [如果无法从 Windows 7 升级到 Windows 10，该怎么办？](https://lifehacker.com/if-you-cant-upgrade-from-windows-7-to-windows-10-what-1835696084)T3】

现在还有人运行 Windows 7 吗？我不认为这个数字有那么高，但从这个问题收到的评论数量来看，这是一个非常有争议的话题。此外，如果每次有人说“告诉他们运行 Linux 就好了”时我都有五分钱，我现在就可以辞职。

解决办法？如果你的系统过时了，你担心它不能运行 Windows 10，你可以运行简单易用的 Linux 版本。你也可以备份你的系统，升级到 Windows 10，看看你是否喜欢，然后继续。你*应该*还能让 [也免费升级](https://lifehacker.com/the-best-ways-to-get-windows-10-for-free-1837174464) 。

* * *

### [如何让你的旧台式 PC 保持良好运行？](https://lifehacker.com/how-can-you-keep-your-old-desktop-pc-running-well-1839141116)T3】

我绝对喜欢这类问题，因为这是我也很纠结的事情——至少在上个月我升级电脑之前是这样。全面改造你的台式机或笔记本电脑系统会花费一大笔钱；这可能不在每个人的预算或技术能力范围内。然而，您可以做很多事情来从旧系统中挤出更多的性能和寿命，我在这篇文章中探讨了很多选项。(我也推荐看看评论，里面也充满了很棒的想法。)

### [如何从旧手机中保存重要的语音邮件？](https://lifehacker.com/how-do-you-save-an-important-voicemail-from-an-old-phon-1836883299)T3】

我真的很喜欢这个问题，因为我确信这是很多人在人生的某个阶段都在纠结的事情，或者说，*将会*纠结。如果您有一份重要的数据，您如何确保它永远存在？每当我把旧照片、视频或文档上传到云端时，我都会思考这个问题，我相信随着年龄的增长，我会更多地思考这个问题。

答案？从旧手机中保存语音邮件并不容易，但这是可能的。如果说我从撰写所有这些 Tech 911 建议专栏中学到了什么，那就是总有一个答案；这可能不是一个简单的解决办法，但你总可以做些什么。

### [将智能手机留在无线充电器上安全吗？](https://lifehacker.com/is-it-safe-to-leave-my-smartphone-on-my-wireless-charge-1834138691)T3】

我对 Tech 911 问题特别感兴趣，我会问自己(如果我有一个建议专栏可以帮我回答这些问题的话)。在这种情况下，我一直很好奇我是否一直在给我的智能手机*过度充电*，因为没有更好的方式来形容它，每天让它长时间处于充电器上。整个问题源于我的一部谷歌 Pixel，它基本上一天 20 小时都在充电器上，其电池最终膨胀并从智能手机上吹掉了外壳。这不是暴力，但绝对是...鼓鼓囊囊。如果智能手机的外壳没有在一定程度上固定住凸出的外壳，我会很快注意到这一点。

无论如何，你可能不应该每天全天给手机充电，但无线充电器不会在一夜之间毁掉你的设备。至少，不应该。
