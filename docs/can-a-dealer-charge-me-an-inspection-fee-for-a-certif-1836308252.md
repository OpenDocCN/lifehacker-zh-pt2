# 对于经过认证的二手车，经销商可以向我收取“检测费”吗？

> 原文:[https://jalopnik . com/can-a-dealer-charge-me-an-inspection-fee for-a-certif-1836308252](https://jalopnik.com/can-a-dealer-charge-me-an-inspection-fee-for-a-certif-1836308252)

作为 Jalopnik 的常驻购车专家和专业购车者，我会收到电子邮件。很多邮件。我决定挑几个问题，试着帮帮大家。本周我们讨论了认证车的额外费用，有问题的 CarMax Miata，以及为什么本田不生产更多的 Rs 型车。

首先，经销商可以对认证车收取额外费用吗？

> “上周，我在一家本田经销商处跟进一辆 2014 款本田思域，行驶了 34k 英里，状况良好。这辆车被宣传为经过认证，这对我来说根本不是什么大事。事实上，我觉得这种营销废话很有趣。
> 
> 然而。当我们谈到数字时，销售单上有一项“检验费”当我问这是什么(除了 699 美元的文件费之外)时，财务人员告诉我，这是汽车检查和认证的费用，这是本田汽车认证的要求。
> 
> 啊？你能解释一下那个吗？是真的还是假的。"

经销商确实需要向制造商支付一笔费用，让汽车通过认证程序，但这笔费用应该已经包含在广告宣传的汽车价格中。“检查”和“修复”是二手车交易的一部分。后来收取这些费用是一些汽车经销商的常用策略，这样他们就可以公布低于市场价值的价格，并用额外的费用来弥补。

事实上，如果一辆汽车被宣传为“认证”，而经销商向你收取“认证费”或“检查费”， [他们可能违反了 FTC 关于广告的规定。](https://jalopnik.com/heres-how-you-can-and-should-report-shady-dealerships-t-1822508371) 这有点像去汉堡王点了广告上标价 4.99 美元的巨型培根汉堡，收银员说培根要多收 50 美分。

我遇到过类似的情况，其他经销商想收取这些费用，往往当你推后，他们放弃了费用。如果他们没有，就把你的生意带到别处去。

当然，这可以追溯到一个核心的购车建议，那就是在你去经销商那里之前，无论汽车是新的还是旧的，都要得到一份详细的书面价格清单。这样，如果一个经销商拉这个额外费用的把戏，你可以抓住它，然后继续前进，在你浪费时间开车去经销商。

接下来:在什么情况下，你会放弃一辆有问题的汽车的销售？

> “我想买一辆 Miata NC，我在 CarMax 找到了一辆价格为 15，000 美元、行驶里程为 51，000 英里的 2013 款车。然而，当他们把它运到我当地的 CarMax 时，他们决定应该给引擎盖上漆，并修理几个鼎。所以我等着那件事完成。当我试驾它时，当我回到停车场时，乘客前轮胎的刹车开始磨，所以他们说他们会更换转子，刹车片和卡钳。对于一辆行驶里程相对较低的汽车来说，这是不是太多危险信号了？里面的塑料也有磨损，这表明它可能没有得到很好的照顾。我想选择 CarMax，以避免从个人或可疑的二手车市场购买，但我不知道自己在做什么。”

这里所发生的是发生在几个人身上的事情，他们认为 CarMax 二手车比其他来源的二手车好得多。 [我在以前的专栏](https://jalopnik.com/are-carmax-cars-really-worth-the-price-premium-1832984118) 中提到过这个问题，尽管 CarMax 的论点与此相反，但没有明显的证据表明从 CarMax 购买二手车会*保证*更高质量的样品。这辆特别的车似乎过着艰难的生活，我建议转移到其他线索。

在 15，000 美元的范围内，有许多信誉良好的二手车经销商和私人卖家将提供优质的 NC Miata。你只需要有耐心，一旦你找到一个好的候选人，让它检查。

最后，为什么本田不生产更多的思域 Rs 呢？

> “为什么本田不生产更多思域 Type R 车型？我在半径 100 英里的范围内搜索了又搜索，每次，我只找到 4 到 5 种 Rs 出售，它们都有从 4K 到 10K 的加价。我想买一辆，但是不加价，因为那辆车已经用了两年了。是什么在拖本田的后腿？”

本田正在制造更多的 Rs 型， [因为我们已经涵盖了下一个车型年将获得价格上涨](https://jalopnik.com/the-honda-civic-type-rs-price-has-gone-up-almost-2-500-1836241220) ，但你的问题更多地与正在制造的汽车数量有关。如果需求这么高，本田为什么不直接增产呢？

答案很复杂，但基本上思域 Type R 对本田来说是一款低销量、利润相对较低的车。他们在每种型号上都赚了一些钱，但是利润并不像一个满载的飞行员那样大。因此，从商业角度来看，本田没有超级动力去生产更多利润不高的汽车。请记住，这些加价对本田没有好处——那是经销商的纯利润。

也许随着下一个车型年的到来，它可能会迫使剩余库存的价格下降。如果你不着急，等一会儿是值得的。
