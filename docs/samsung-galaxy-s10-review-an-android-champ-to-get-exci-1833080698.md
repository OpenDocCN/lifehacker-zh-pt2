# 三星 Galaxy S10 评测:令人兴奋的安卓冠军

> 原文：<https://gizmodo.com/samsung-galaxy-s10-review-an-android-champ-to-get-exci-1833080698>

如果说 [Pixel 3](https://gizmodo.com/google-pixel-3-review-the-other-way-to-make-a-killer-p-1829749807) 是谷歌展示软件优先的手机设计方法有多好的最高成就，那么 [Galaxy S10](https://www.samsung.com/us/mobile/galaxy-s10/) 则是向硬件之神致敬。Galaxy S10 采用了令人厌倦的凹口自拍摄像头，重新设计了带有三种不同镜头的后置摄像头，以及令人难以置信的电池续航时间，感觉像是智能手机技术成熟的庆典。

令人印象深刻的是三星在 Galaxy S10 中塞进了多少东西，尤其是 S10+。尽管与 [Galaxy Note 9](https://gizmodo.com/samsungs-galaxy-note-9-is-the-best-big-phone-1828348138) 的屏幕尺寸几乎相同，但 S10+拥有略大的 4100 毫安时电池(Note 9 为 4000 毫安时)，前后额外的摄像头传感器，以及反向无线充电(三星称之为无线电源共享)，机身实际上薄了 10%。现在，我不是那种通常会关心 7.8 毫米和 8.8 毫米之间差异的人，但当它们在你手中时，你肯定会注意到。对于一部包装了这么多新功能的手机来说，重 6.1 盎司的 S10+也轻得令人惊讶。

S10 的一切美学都很到位。三星新的 Infinity-O 打孔自拍相机是去年的 的 [的更时尚、更少干扰的版本。三星还调整了将手机固定在一起的铝制底盘，使其握起来更舒适。甚至三星的棱镜色选项也名副其实，尤其是棱镜白配色，根据光线的不同，它会从电紫色变成热粉色再变成淡蓝色。自从三星用埋在显示屏下的超声波扫描仪取代了后置指纹，背部的干扰功能就更少了。](https://gizmodo.com/the-incredible-evolution-of-the-smartphone-notch-from-1829852261)

S10 最吸引人的是它的屏幕。对于 2019 年，三星将 S10 的屏幕称为动态 AMOLED 显示屏，这是一个组件，在 [DisplayMate](http://www.displaymate.com/Galaxy_S10_ShootOut_1S.htm) 的屏幕测试专家给了他们迄今为止的最高分。虽然所有这些数字和评级都很重要，但只要看看 S10 屏幕呈现出的令人惊叹的绚丽色彩就足以证明，S10 的显示屏是市场上最好的。它支持 HDR10+内容，在某些情况下，显示器可以超过 1，200 尼特，以及 HDR10+视频录制。结合顶级的照片拍摄，S10 具有神奇的能力，可以让你所有的本土内容在屏幕上弹出，比 iPhone XS 或 Pixel 3 更好。

在某些方面，这是一个颠覆性的技巧，因为它往往会使内容在 S10 的屏幕上看起来比在其他设备上看起来更生动。另一方面，S10 的屏幕还具有改进的蓝光过滤器，可以更好地减少颜色，从而使人更难入睡。

为了避免这个缺口，三星不得不取消过去两代 Galaxy 手机上使用的虹膜识别传感器。如果你愿意，你仍然可以用你的脸解锁手机，但要验证在线支付等更敏感的操作，你需要依赖 S10 的屏幕内超声波指纹传感器，三星声称这是世界上第一个经过 FIDO 认证的生物传感器。

这是我愿意做的交易，因为我更喜欢用手指触摸来确认购买，而不是像 Face ID 那样。凭借 S10 的屏幕内传感器的速度和可靠性，基本上没有任何负面影响。真的，S10 的超声波扫描仪唯一一次错过是当我试图解锁手机时，它的屏幕完全关闭，也没有告诉你触摸哪里的小指纹图标。但在习惯了手机大约一天之后，这个问题就自行解决了，现在我有点喜欢能够在不需要任何明显线索的情况下解锁 S10 的隐秘特性。

至于那些新的后置摄像头，三星在 S10 的工具包中添加了一个超宽镜头。这是我们之前在 [LG V40](https://gizmodo.com/lgs-5-camera-phone-offers-a-snapshot-of-the-incredible-1829488124) 和 [华为 Mate 20](https://gizmodo.com/huaweis-mate-20-pro-is-a-technical-marvel-1830346004) 上看到的一个举动，它将三后置摄像头整合为每个高端手机都应该有的东西。这三个都非常清晰，尽管超广角由于其 123 度的视野而在边缘处略有失真。但是广角镜头的真正影响是它如何补充其他两个摄像头，因为现在有了三个不同焦距的相机，S10 比以前更好地处理生活中任何精彩的瞬间。

一个很好的例子发生在我代表 MWC 在巴塞罗纳的时候。当我在去那些看起来没完没了的手机发布会的路上时，我能够停下来一分钟，用所有三个镜头给富丽堂皇的加泰罗尼亚国家艺术博物馆拍了三张照片。在不到 10 秒的时间里，我从远处拍摄到了博物馆的特写，一张照片捕捉到了我看到的景色，第三张照片捕捉到了广阔的环境。这是谷歌和苹果最好的手机目前无法捕捉的一系列镜头，它突显了一个看似多余的额外传感器如何能够记录一个宁静的时刻，即使在繁忙的日程安排中。

也就是说，如果图像质量是你唯一关心的事情，在某些情况下，S10 的相机达不到 Pixel 3 所能提供的。在 S10+、Pixel 3 和 iPhone XS Max 之间的并排枪战中，Pixel 3 在捕捉背景中起重机的细节和前面道路的纹理方面做得最好。与 iPhone XS 不同的是，S10+和 Pixel 3 都没有对透过中间喷泉的蓝光表现过度。

三星还更新了手机的场景优化器，它可以智能地检测各种物体和环境，以帮助优化相机设置。现在，S10 可以识别猫和狗、鞋和汽车的区别，或者在非常黑暗的情况下自动打开手机的明亮夜晚设置。S10 的问题是，除非手机自己检测到这些东西，否则你无法自己激活这些照片调整。这给亮夜模式带来了一个问题，因为即使在低光的情况下，除非真的很暗，否则你不会得到任何帮助。

通常，这并不是一件大事，因为主摄像头的最大光圈为 f/1.5，S10 自然比 Pixel 3 上的 f/1.8 光圈后置摄像头收集更多的光。但当我设置烛光拍摄来测试这一点时，虽然 S10 在没有打开任何特殊模式的情况下表现优于 Pixel 3，但我无法直接比较三星的亮夜模式和谷歌的夜视功能，直到我将一支蜡烛移到房间的对面。因此，在一切恢复正常后，我再次尝试，发现在明亮的夜晚和夜间模式激活的情况下，嗯，我们只能说 [谷歌的夜视](https://gizmodo.com/googles-incredible-night-sight-mode-was-worth-the-wait-1830432540) 是真正的交易。

尽管《夜视》已经问世六个月了，它的效果也不应该再让我感到惊讶，但谷歌的计算摄影仍然有办法让我感到惊讶。现在三星已经升级到三个后置摄像头，感觉它可以改善未来 Galaxy 手机摄影的最大方式是进一步加强图像处理。虽然 S10 的三个摄像头都很坚固，但仍有一点改进的空间。

S10 配备了高通 855 芯片组、8GB 内存和 128GB 存储标准，比速度最快的上一代 Android 手机高出 15%至 25%。这个凸起本身可能不是每个人都注意到的，但与三星的 Android 9 用户界面皮肤配合使用，S10 感觉比[S9](https://gizmodo.com/samsung-galaxy-s9-review-the-best-android-phone-dont-n-1823575944)更快、更灵敏。一个 UI 还包括方便的界面改进，使按钮和图标更容易单手触及，以及有用的附加功能，如当有人给你发送文本时，弹出通知会在手机的主屏幕上产生小气泡。

最后，如果我没有提到 S10 的电池寿命有多好，那我就大错特错了。在我们的缩减测试中，标准的 Galaxy S10 持续了 14 小时 45 分钟，而 S10+持续了 15 小时 9 分钟。认真地。这甚至比我们从 Galaxy Note 9 获得的 14:09 更好，也比 Pixel 3 的 10:50 长得多。

在现实世界中，S10 的寿命更长。在 MWC 期间，我使用 S10+作为我的主要设备，在不断使用手机检查空闲时间、查找方向、拍摄照片、检查笔记和我需要做的其他事情之间，报道贸易展览是我可以让手机执行的最繁重的任务之一。

但即使如此，S10+一次充电也能坚持近两天。在第一天入睡前达到 60%后，S10 终于在第二天晚上 10:30 左右睡着了。当然，这不是一个完整的 48 小时，但即使没有任何特殊的电池节省模式打开，一个晚上不插电也不像以前那样判了死刑。除非你是故意试图杀死手机，一天的电池只是觉得它不再是一个问题。不再需要携带两部手机、一个充电盒或额外的电池组。S10 的电池续航时间给你自由。

那么你要这些多余的果汁做什么？当然，要确保你所有的其他小玩意都加满了。虽然 S10 的无线 PowerShare 功能传输电量约为 5 瓦(或略高，取决于正在充电的设备)，这是无线充电速度的最低要求，但该功能更多的是实用性，而不是尽快充电。它非常适合帮助朋友的低能量手机，但 wireless PowerShare 的最佳用途可能是在旅行时作为无线充电板的替代品。有了它，当你离开家几天时，你不需要为 Galaxy 手表或其他支持 Qi 的智能手表带底座。把手机翻过来，点一下快速设置菜单里的 PowerShare 图标，就行了。请记住，当它没有插上电源时，如果手机的电池电量不足 30%，无线 PowerShare 将无法工作。

[去年年底我对三星](https://gizmodo.com/samsung-cant-afford-to-hold-back-anymore-1831178495#_ga=2.126569162.694137766.1551836211-1220318864.1537376527) 很苛刻，但理由很充分。尽管 S9 很好，但作为世界上最大的智能手机生产商，我觉得三星可以做得更好。通过 Galaxy S10，三星实现了这一目标。即使与华为 Mate 20 Pro 相比，Galaxy S10 也有更好的屏幕，更清晰的摄像头，更光滑的设计，甚至更长的电池寿命——所有这些都不需要牺牲手机的耳机插孔。

阻止任何人购买 S10 的最大因素是价格，因为普通 S10 的起价为 900 美元，这对于一部手机来说是一大笔钱。但当你将它与价格没有下降、不太复杂的硬件仍需 800 美元的[【Pixel 3】](https://gizmodo.com/google-pixel-3-review-the-other-way-to-make-a-killer-p-1829749807)和售价 1000 美元的[【XS](https://gizmodo.com/the-iphone-xs-is-forever-1829305262)iPhone 相比，Galaxy S10 最能让所有这些钱物有所值。
