# 应该去掉大蒜胚芽吗？

> 原文:[https://life hacker . com/should-you-remove-the-大蒜胚芽-1830071615](https://lifehacker.com/should-you-remove-the-garlic-germ-1830071615)

如果你切过一些大蒜，你可能会注意到在蒜瓣中间有一个小芽，叫做“胚芽”与其他致病细菌不同，这种细菌不会伤害你，但去除它会让你的菜味道好一点(有时)。

Watch

当你的大蒜是新鲜和年轻的，胚芽是苍白和柔软的，并没有真正影响味道，但随着大蒜的老化，胚芽变成绿色和苦涩的，最好把它弄出来。如果你介于两者之间——并且有一种淡绿色的大蒜“青少年”——这取决于你会用大蒜做什么。

如果你要把大蒜生着吃——比如说一个 [优质蒜泥](https://lifehacker.com/aioli-is-not-fancy-mayonnaise-1829347023)——最好总是去掉胚芽，除非你手头有最年轻、最新鲜的大蒜。如果你只是简单地烹饪大蒜，那么去除细菌也是一个好主意，因为一点点热量不足以消除苦味。但是如果你要烹饪大蒜——也许是 [烤整个头部](https://skillet.lifehacker.com/how-to-roast-a-whole-head-of-garlic-1820407305)——保留胚芽也不错，就像长时间烹饪驯服这头苦涩的野兽一样。(如果 [它发芽了](https://skillet.lifehacker.com/plant-sprouted-garlic-bulbs-and-eat-the-greens-1826402986) )，不过，继续种吧，为了美味的蒜苔。)

去除胚芽很容易——将蒜瓣切成两半(或捣碎),用指甲或削皮刀将其撬出，然后像往常一样继续准备大蒜(和生活)。
