# 最好的棕色黄油的秘密

> 原文：<https://lifehacker.com/the-secret-to-the-best-browned-butter-1840046998>

棕色黄油 让一切尝起来更好——当然，除非你根本尝不出来。它浓郁的烤味很容易在除了黄油以外的菜肴中消失。解决办法很简单:加牛奶就行了。



我从博客作者李漫荻那里学到了 [这个天才的把戏](https://ladyandpups.com/2019/06/15/extra-browns-browned-butter/) 。它的工作原理是这样的:当你将黄油变成褐色时，真正油炸并变成褐色的部分是牛奶蛋白。加入纯牛奶可以让乳脂更好地发挥作用，所以你会得到更高比例的松脆的棕色部分，这让棕色黄油有了明确无误的味道。你得到的越多，它们在完成的菜肴中就越强。

制作额外的棕色黄油就像制作普通的东西一样，除了你在开始时倒入一些牛奶。没有神奇的比率。上面的照片是两根棒加大约半杯全脂牛奶，但是你可以根据你的需要使用更少或更多。密切关注它，并经常搅拌——一开始它看起来很奇怪，很苍白，但一旦牛奶煮沸，你就会变成金黄色。完成后，把这些松脆的小块搅拌成任何需要一点活力的东西。这一次，你一定会知道他们在那里。
