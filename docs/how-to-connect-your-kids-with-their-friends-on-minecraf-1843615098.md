# 如何让你的孩子和他们在《我的世界》的朋友联系起来

> 原文:[https://life hacker . com/how-to-connect-your-kids-on-minecraf-1843615098](https://lifehacker.com/how-to-connect-your-kids-with-their-friends-on-minecraf-1843615098)

如果你的孩子和我的一样，他们已经花了几千个小时的隔离时间来玩《我的世界》。但是如果他们只是在单人模式下玩《我的世界》，他们就错过了——不仅错过了大量有趣的游戏选项，还错过了与朋友社交的机会。《我的世界》就像一个虚拟的游乐场，孩子们可以在那里一起玩耍，但没有疾病可以传播。

Watch

然而，与真正的游乐场不同，多人《我的世界》运行起来可能有点麻烦。所以我整理了这个一步一步的小指南，尽可能容易地建立一个共享的、专属的《我的世界》世界，没有陌生人或任何不想要的玩家要对付。按照这些指示，你的孩子和他的朋友将在几秒钟内一起玩在线游戏…好吧，几分钟....好吧，最多一小时。

### 第一步:确定每个人使用的《我的世界》版本

虽然从技术上讲，《我的世界》有 14 个不同的版本，但几乎每个人都在玩《《我的世界》:Java 版》或《《我的世界》:基础版》。让你的孩子和他们的朋友组成一个在线《我的世界》小团体的第一步是确定每个人都在运行哪个版本的游戏。

如果你在游戏机、手机或平板电脑上玩《我的世界》，你就是在玩基岩游戏。如果你在 macOS 上玩，你几乎肯定是在玩 Java 版。如果你在装有 Windows 10 的电脑上玩，你可以玩任何一个版本。

虽然 Java 版的任何用户都可以与任何其他 Java 玩家一起加入游戏，并且任何基岩玩家都可以在任何控制台上玩任何其他基岩，但 Javas 和 Bedrocks 不能一起玩(如果没有复杂的解决方法)。所以，可悲的是，如果你孩子的朋友泰勒在他的 Mac 上玩 Java 版，而你的孩子在他的任天堂 Switch 上只有基础版，那么是时候交一个新的好朋友了。

### 第二步:至少有一个玩家需要订阅《我的世界》领域

《我的世界》领域 Plus 是基岩《我的世界》的基于订阅的官方服务器主机，而《我的世界》领域:Java 版是 Java 版本。对于基础玩家来说,《领域》是迄今为止创建一个共享的、持久的世界并向你的朋友开放的最简单的方法。Realms 还允许你在公共服务器上玩，以及让你下载皮肤和纹理包来改变游戏的外观和感觉。

Java《我的世界》玩家已经可以在公共服务器上玩游戏，下载 mod、皮肤和纹理包，而无需订阅服务器，但是如果你的目标是为你的孩子和他的朋友建立一个尽可能少麻烦的共享世界，订阅服务器就是答案。

你不需要订阅服务器就可以在其他人的共享世界里玩，所以 10 个朋友中只有一个人需要每月支付 7.99 美元，然后邀请其他人在私人世界里玩。订阅费只允许你一次创建一个世界，但如果你愿意，你可以每月再付 8 美元创建第二个世界。

你是否也需要订阅你的主机的在线服务来使用《我的世界》领域取决于你的主机。在 Xbox One 上，你不需要 Xbox Live Gold 来使用领域，但在任天堂 Switch 上，你*需要*任天堂 Switch 在线订阅。Java 玩家再也不需要另一个订阅就可以在线玩了。

在这两个版本的游戏中，你都可以建立自己的服务器和/或使用第三方托管服务来建立一个共享的《我的世界》世界，但是如果你需要这个基本的操作指南，我不建议你建立自己的服务器。这可能会变得很复杂，除非你和/或你的孩子想要解决技术问题，否则我建议不要这么做，至少现在不要。

### 第三步:创建你的世界并订阅服务器

好了，是时候为你的孩子和他的朋友创造一个世界了。我们开始吧:

1.开始《我的世界》。

2.点击“播放”

3.点击“新建”

4.点击“创造新世界”

5.单击“在领域上创建”

6.在 10 人游戏和双人游戏之间选择

7.你现在必须订阅服务器，所以拿出信用卡。

### 第四步:邀请他们的朋友

#### 要在同一台主机上邀请朋友:

1.点击《我的世界》菜单屏幕上的“播放”。

2.单击新创建的领域名称旁边的铅笔图标。

3.点击“成员”

4.从您控制台的朋友列表中选择您的人加入您的服务器。

#### 要在其他控制台上邀请朋友:

1.确保您的家长控制允许跨平台连接。

2.打开《我的世界》。

3.点击“播放”

4.点击你新创建的世界旁边的铅笔图标。

5.转到“成员”

6.点击“共享链接”

7.复制唯一的网址，通过电子邮件、短信或信鸽发送给朋友。

### 第五步:享受网上《我的世界》的乐趣

一旦你的孩子的世界在服务器上被创建，并且其他玩家被邀请，所有被邀请的玩家将能够在任何时候进入这个世界，杀死大量的骷髅，并且建造各种各样令人敬畏的东西。
