# 2019 年最佳转售名单是所有卡车和一辆令人惊讶的保时捷

除非你买一些有收藏价值的异国情调的东西，否则你可以打赌，一旦你把它开出停车场，你可爱的新车就会贬值。但是有些车比其他车承受得更好。 [凯利蓝皮书发布了他们 2019 年的最佳转售名单](https://www.kbb.com/new-cars/best-resale-value-awards/) 这基本上是你所期待的，除了一个德国品牌的意外进入。

今年的榜单与去年的非常相似，只是在排名上有些洗牌。这不会让任何人感到震惊，因为皮卡和 4x4 SUVs 倾向于统治美国这片土地。但我们在 2019 年关注的是:

**1。2019 年丰田塔科马**—60 个月的转售价值为 62.2%

**2。2019 款 Jeep 牧马人**—60 个月的转售价值为 58.3%

**3。2019 丰田 Tundra**—60 个月内转售价值 56.9%

**4。2019 GMC Sierra**—60 个月的转售价值为 56.7%

**5。2019 款丰田 4 runner**—60 个月的转售价值为 56.3%

**6。2019 年雪佛兰 Silverado**—60 个月的转售价值为 53.7%

7 .**。2019 款本田 Ridgeline**—60 个月内转售价值的 51.3%

**8。2019 雪佛兰科罗拉多**—60 个月内转售价值 50.9%

**9。2019 年福特 F 系列**—60 个月的转售价值为 50.6%

10。2019 款保时捷 Macan—60 个月的转售价值为 50.0%

在凯利蓝皮书，他们也有 36 个月 的 [转售价值。](https://www.kbb.com/new-cars/best-resale-value-awards/best-resale-top-10-cars/)

今年 [吉普牧马人](https://jalopnik.com/the-2018-jeep-wrangler-didnt-get-soft-it-got-much-bett-1821160227) 跃过丰田苔原占据第二的位置，而塔科马仍然是王者。常年的福特 F-150 下降了几个等级，排在第九位，而 [本田 Ridgeline](https://jalopnik.com/the-2018-honda-ridgeline-proved-itself-on-a-700-mile-ca-1828175665)——或者我喜欢称这种卡车为聪明人，他们知道他们不需要拖走整个房子——获得了几个名次。去年溜进第 10 名的 [斯巴鲁 WRX](https://jalopnik.com/the-subaru-wrx-sti-still-matters-1819925240) ，已经被 [保时捷 Macan](https://jalopnik.com/the-2019-porsche-macan-handles-better-than-a-crossover-1830850009) 取代。

Macan 在转售名单中名列前茅并不令人意外，因为一般来说，豪华车的销量往往会大幅下降，Macan 是保时捷销量最大的车。然而，在前三年，它保留了 65%的残值，这比名单上的许多皮卡都要好。

虽然有些人可能不喜欢保时捷最畅销的汽车是跨界车，[Jalopnik 的大多数员工都认为 Macan 更像是一辆掀背车](https://jalopnik.com/the-2019-porsche-macan-handles-better-than-a-crossover-1830850009) ，尽管是一辆非常昂贵的而不是笨重的 CR-V
