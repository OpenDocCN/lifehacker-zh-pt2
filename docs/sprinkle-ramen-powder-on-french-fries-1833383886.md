# 将拉面粉撒在薯条上

> 原文：<https://lifehacker.com/sprinkle-ramen-powder-on-french-fries-1833383886>

我是一个很有品位的世俗女人，这就是为什么我旅行中最喜欢的事情之一是看看国际麦当劳的当地人才能提供什么，特别是在蘸酱和 McFlurry 选项方面。我在英国吃过吉百利麦片粥，在法国吃过蘸 La 酱咖喱的鸡块，但这些来自香港的拉面风味的“摇摇”薯条 确实很特别。

Watch

这些薯条是与 Nissin——the Cup o ' Noodles/Top Ramen people——的合作，它们听起来就像是:用拉面粉调味的麦当劳薯条。不是将整批薯条涂上美味的混合调料，而是将粉末放在一个单独的小袋中，还有一个袋子用于摇动，以确保调料均匀分布(因此得名)。

这是如此简单、明显的配对。作为一个既喜欢 [【味精】](https://skillet.lifehacker.com/put-msg-in-everything-you-cowards-1831721707) 又喜欢 [用拉面粉](https://skillet.lifehacker.com/season-rice-eggs-popcorn-and-more-with-instant-ramen-1791624851) 调味的著名粉丝，坦率地说，我感到震惊和震惊的是，我还没有尝试过。与我尝试过的大多数国际快餐不同，这是一种很容易在家里复制的东西。点一些薯条 [或者自己做](https://lifehacker.com/how-to-make-restaurant-worthy-fries-1832361045)——然后撒上你最喜欢的一包拉面粉，如果你和我一样，你已经收集了很多。

你甚至可以把煎炸油和调料搭配起来。例如，如果你喜欢用 schmaltz 烹饪薯条，试试鸡肉口味，或者用牛肉粉搅拌牛油薯条(遗憾的是，这对虾不起作用，因为没有虾油这种东西)。此外，我不明白为什么我们必须只吃薯条——为什么不把所有的土豆都拉面化？拉面调味的土豆煎饼配一个松软的鸡蛋听起来不错，还有拉面土豆泥。

只是不要在纸袋里摇晃捣碎器。那不会有好结果的。
