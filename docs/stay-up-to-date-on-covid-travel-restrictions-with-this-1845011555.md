# 使用此交互式地图了解最新的 COVID 旅行限制

> 原文：<https://lifehacker.com/stay-up-to-date-on-covid-travel-restrictions-with-this-1845011555>

自从新冠肺炎·疫情开始，旅行就变得一团糟。考虑到旅行限制是遏制传染病传播的重要组成部分，这是有道理的。起初，大部分人确实呆在原地，避免乘飞机旅行，呆在他们的家里(或第二个家)。

Watch

但不久之后，在 Instagram 上快速滚动会显示人们在海滩上度假，并配有类似“别担心，我们在保持社交距离！”或者是一张标签为“#逃离城市”的古色古香的小镇主要街道的快照在某一点上，感觉就像这个国家的一半人在不可思议的美丽的地方过着他们最好的生活，而我们其余的人却呆在家里，连续几个月盯着同样的墙壁，每次离开家都穿着个人防护装备。

这就是说，在疫情旅行一直令人困惑。各州和联邦政府的指导方针是不断变化，这可能很难跟上，即使对那些习惯于以旅行为生的人来说也是如此。如果您需要一些帮助来跟踪一切，您可能会对联合航空公司的 [新互动地图](https://www.united.com/ual/en/us/fly/travel/restrictions-map.html) 感兴趣——这是一种帮助我们应对不断变化的 COVID 旅行限制的资源。以下是你需要知道的。

### 如何使用美联航的互动地图

在美联航的网站 和他们的移动应用程序上都可以看到这种交互式彩色编码地图 [，它旨在指示目的地是否关闭、部分开放、或完全开放，并将注明前往该地点是否需要任何测试或自我检疫。](https://www.united.com/ual/en/us/fly/travel/restrictions-map.html)

还有 [特殊滤镜](https://hub.united.com/2020-09-03-united-helps-customers-navigate-travel-restrictions-with-new-online-interactive-map-2647446779.html) 让用户可以看到某个地区的地方法规，包括那些涉及社交距离和口罩强制执行的。另一个过滤器显示哪些酒店、餐馆、和其他机构对公众开放的信息。

不管你是否真的在不久的将来乘坐美联航(或任何其他航空公司)的航班，该地图都是 COVID 旅行信息上一个易于使用的更新资源。
