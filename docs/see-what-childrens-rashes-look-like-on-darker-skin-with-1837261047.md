# 用“棕色皮肤问题”看看深色皮肤上的儿童皮疹是什么样的

> 原文：<https://lifehacker.com/see-what-childrens-rashes-look-like-on-darker-skin-with-1837261047>

如果你曾经在你孩子的皮肤上看到一些不寻常的东西，当你焦急地等待与儿科医生的预约时，你可能会转向互联网寻求指导。在健康教育网站上，你会找到各种常见皮肤病的图片，包括湿疹、手足口病、水痘、脓疱病和麻疹。但是这些图像几乎都是白人病人的特写。(要自己看，只需在梅奥诊所、WebMD 和美国皮肤病协会网站上快速搜索一下这些情况。甚至连肿块、疮和皮疹的插图也只显示在白皙的皮肤上。)



多年来，这让艾伦·布坎南·维斯很沮丧，她的蹒跚学步的儿子是混血儿。她试图在网上搜索照片，以参考水痘和荨麻疹等疾病，但她告诉我，“即使加上限定词‘黑孩子身上的水痘’也会产生大部分高加索人的例子。”最近，她决定做一些事情来帮助其他面临类似障碍的父母，并开始自己收集照片。她的项目 [棕色皮肤问题](https://www.instagram.com/brownskinmatters/) 是一个 Instagram 账户，里面装满了非白色皮肤的皮肤病参考图片。你可以看到一个西班牙裔黑人小孩被蚂蚁叮咬后的样子。或者病毒性疾病如何在一个黑白混血的孩子身上显现。

从照片上看，很明显，不同肤色的人看起来情况不同。在一个以患有水痘的黑人儿童为主题的帖子上，一个人评论道:“谢谢！我妈妈(白人)总是说她不确定我们是否真的得了水痘，因为它们看起来不像她想象的那样。但是儿科医生说我们有。”

虽然韦斯不是医学专业人士，但她正与医生一起审查观众提交的照片。她强调，关于棕色皮肤问题的信息仅用于教育和参考目的，而不是诊断。“我只是一个普通人，非常爱她的儿子，希望他和其他长得像他的人能得到公平的代表权和资源，”韦斯说。
