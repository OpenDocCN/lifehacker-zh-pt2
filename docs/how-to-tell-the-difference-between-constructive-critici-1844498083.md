# 如何区分建设性的批评和欺凌

> 原文：<https://lifehacker.com/how-to-tell-the-difference-between-constructive-critici-1844498083>

在“取消文化”中，生活在一个极端政治极化的社会中，收到来自其他人的反馈并不罕见。事实上，即使是“反馈”这个词也感觉很委婉，因为现在这么多的交流——无论是在社交媒体上还是面对面——都可以变得热烈起来(说得委婉一点)。虽然被告知你做错了什么和/或在某些方面可以改进通常感觉不好，但从某人那里得到建设性的批评和被欺负是有区别的。我们脆弱的自我会让即使是温和的批评也感觉像是人身攻击，但这不一定是正在发生的事情。那么建设性的批评和恃强凌弱有什么区别呢？这里有一些有帮助的见解。



### 建设性的批评对欺凌

在进一步讨论之前，我们先来谈谈这两个术语的定义。以下是 Maham Barbar 如何在《每日新闻》 的一篇 [文章中进行分解:](https://blogs.arynews.tv/differentiate-between-criticism-and-bullying/)

> “建设性批评是一个对他人的工作提出善意评论的过程，这样他们就可以知道自己的优势和劣势，这有助于他们未来的改进。
> 
> 恃强凌弱是一种不受欢迎的攻击性行为，被用来剥削他人。这种行为反复出现，会影响人的身心健康。"

在区分这两者时——无论是作为反馈的给予者还是接受者 Barbar 建议我们记住以下几点:

#### 话

是的，用词很重要。建设性的批评通常至少包括一些鼓励的话，因为这个想法是为了帮助别人进步。另一方面，恃强凌弱可能涉及侮辱性的言论，并且只关注弱点。

#### 音调

这在个人面前比在 Twitter 或评论区容易得多，但语气也可能意味着善意的反馈和全力以赴的欺凌之间的区别。根据巴巴尔的说法，保持礼貌是防止进入恃强凌弱领域的一种方式(只要不是讽刺)。

#### 保持简短

没有人想听别人没完没了地谈论自己的缺点——即使是以非常礼貌的语气——所以如果你给了某人建设性的批评，表达你的观点，但不要白费力气。一旦你表明了你的观点，继续前进。

同样地， [巴巴尔建议](https://blogs.arynews.tv/differentiate-between-criticism-and-bullying/) 只对一个人实际上可以控制的事情提出建设性的批评。她写道:“批评别人无法改变的事情，然后走开会让他们感到无助，这实际上是一种欺负行为。”但是如果你坚持在可以改进的地方与交流，那么对话可能会有激励作用。
