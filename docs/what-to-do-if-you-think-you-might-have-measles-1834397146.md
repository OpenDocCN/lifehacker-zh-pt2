# 如果你认为自己可能得了麻疹，该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-you-think-you-might-have-measles-1834397146>

首先，深呼吸。不要惊慌。很有可能，几乎所有阅读这篇文章的人都没有麻疹。但是你可能想知道如果那个统计异常值是你(或者你的孩子)该怎么办。

### 了解最常见的早期症状

麻疹最著名的是它的小红点皮疹，但皮疹不会出现，直到你有其他症状三到五天。疾控中心将常见的早期症状列为:

*   高烧(可能飙升至 104 华氏度以上)
*   咳嗽
*   流鼻涕
*   红色的水汪汪的眼睛

在皮疹出现之前，口腔内有时会出现微小的白色斑点，称为 [科普利克氏斑](https://en.wikipedia.org/wiki/Koplik%27s_spots) 。(这些对医生来说是一个有用的线索，但是如果你认为你得了麻疹，谷歌图像搜索口腔皮疹可能不是最好的利用你的时间。给医生打电话。)

症状通常需要 7 到 14 天才能开始显现。同时，你可能不知道你已经被感染了，但是在你注意到症状的几天前，你就开始变得有传染性了。

### 位置很重要

你必须从某个地方感染麻疹。如果你住在当前麻疹爆发的地方，你的医生会比住在距离任何已知病例数百英里之外的地方的医生更加怀疑发烧和皮疹。

如果附近没有麻疹病例，你可能要想想你最近是否去过爆发疫情的地方。最近的国际旅行也可能让你处于危险之中:疾病控制中心 [目前有针对以色列、乌克兰、日本、巴西和菲律宾的麻疹旅行建议](https://www.cdc.gov/measles/travelers.html) 。

### 给你的医生打电话，但不要亲自去

任何医生都不想在候诊室里看到一个人传播麻疹病毒。它的传染性如此之强，以至于在你离开房间后，空气中的飞沫中可能会漂浮着足够多的病毒，以至于在你 两小时后进入房间 [的人可能会感染麻疹。](https://www.cdc.gov/measles/hcp/index.html)

所以，如果你认为你可能得了麻疹，不要去拜访——打电话。没有初级保健医生？打电话给紧急诊所，或者你最喜欢的远程医疗提供者——呆在家里直到有人给你另外的指示。

### 疫苗呢？

如果你最近接种了麻疹疫苗[，那么你有 97%的几率免疫(如果你只接种了一剂而不是推荐的两剂，那么有 93%的几率免疫)。这意味着一些接种了疫苗的人仍然有可能感染这种疾病。](https://vitals.lifehacker.com/do-you-need-another-measles-vaccine-1834241586)

但是没有接种过疫苗的人，几率几乎相反: [90%没有接种疫苗的人*会*得病](https://www.childrensdayton.org/the-hub/90-percent-chance-getting-measles-if-you-are-exposed-and-unvaccinated) 。

未接种疫苗的人包括年龄太小而不能接种疫苗的儿童，或者由于任何原因而没有及时接种疫苗的儿童。他们还包括免疫系统有问题的人，他们不能安全地接受注射。所有这些人都依靠我们其他人接种疫苗来阻止麻疹传播。

因此，如果你有可能接触过麻疹，并出现发烧和皮疹(或其他特征性症状)，请立即通知你的医生，并准备在家里呆一段时间。否则，你会传染给其他人。
