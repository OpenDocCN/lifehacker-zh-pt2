# 今天就用谷歌的“Android Flash 工具”试用 Android 11 吧

> 原文：<https://lifehacker.com/try-android-11-today-using-googles-android-flash-tool-1843953684>

我们中的许多人都渴望安装 Android 11 Beta 1——下一代 Android 操作系统的公共测试版，它可能比 Android 11 开发者预览版稳定得多，包含的错误也更少。虽然一些幸运的 Android 用户已经得到了它，但我们其余的人现在还停留在 Android 11 开发者预览版 4 上。



不过，这并不可怕！如果你想生活在 Android 的最边缘，在其他人之前尝试各种新功能——当然，在这个过程中要冒一点稳定性的风险——你现在就可以玩 Android 11。

虽然我通常会敦促你等待面向公众的 Android 11 Beta 1 发布，但我不会阻止你现在试用测试版。事实上，我打算自己安装在我的(非主要)Android 智能手机上；这毕竟是 *Lifehacker* 。

哦，我也要告诉你怎么做。

### 哪些设备可以运行 Android 11 开发者预览版？

抱歉，三星粉丝们。目前，只有**谷歌 Pixel 4 / 4 XL、Pixel 3a / 3a XL、Pixel 3 / 3 XL、**和 **Pixel 2 / 2 XL** 智能手机可以使用 Android 11 开发者预览版。但是，嘿，你得到了比克斯比！如果你仔细听，你几乎可以听到从你家附近的像素所有者发出的嫉妒的声音。

### 如何开始安装 Android 11 开发者预览版？

令人欣慰的是，谷歌已经让用新的测试版更新你的像素变得非常容易。首先，你需要安装 [Chrome](https://www.google.com/chrome/) 或[Edge Chrome](https://www.microsoft.com/en-us/edge?form=MA13DE&OCID=MA13DE)，如果你还没有使用这两种浏览器的话。然后，您需要针对您的特定设备点击以下链接，这将启动谷歌基于浏览器的 Android Flash 工具:

*   [像素 2](https://flash.android.com/build/6455311?target=walleye-user&signed)
*   [像素 2 XL](https://flash.android.com/build/6455311?target=taimen-user&signed)
*   [像素 3](https://flash.android.com/build/6455311?target=blueline-user&signed)
*   [像素 3 XL](https://flash.android.com/build/6455311?target=crosshatch-user&signed)
*   [像素 3a](https://flash.android.com/build/6455311?target=sargo-user&signed)
*   [像素 3a XL](https://flash.android.com/build/6455311?target=bonito-user&signed)
*   [像素 4](https://flash.android.com/build/6455311?target=flame-user&signed)
*   [像素 4 XL](https://flash.android.com/build/6455311?target=coral-user&signed)

如果你用的是 Windows，你可能需要在开始之前额外做一步:安装 [这些 USB 驱动](https://developer.android.com/studio/run/oem-usb) (或者 [这些](https://developer.android.com/studio/run/win-usb) )。如果在安装驱动程序时，您没有在 Windows 的设备管理器中看到您的 Android，请确保您已经通过**设置>连接的设备> USB** 将 USB 模式设置为“文件传输”。

然后你需要确保你已经在你的 Pixel 上设置了开发者模式(通过**设置>关于手机**，滚动到底部，并开始大量点击“构建号”)。您还需要在新的开发者选项中启用 USB 调试功能(**设置>系统>高级>开发者选项**)。最后，确保您在同一个屏幕中启用了 **OEM 解锁**。

完成后，点击 Android Flash 工具中的“添加新设备”,你就可以开始用测试版操作系统刷新你的 Android 了。

找到您的设备后，您只需点击它即可开始更新过程:

不过，在您单击该按钮之前，您可能需要单击您的版本右侧的铅笔图标，并选择擦除设备选项——如果您想从头开始使用测试版操作系统，这很有用。完成后，您还应该启用锁定引导程序的选项。最后，我建议在开始之前，禁用浏览器中的广告拦截器，或者将谷歌的 Android Flash 工具列入白名单，否则你可能会遇到奇怪的“网络错误”

在安装过程中，*不要触摸你的 Android，除非你被要求做一些事情*。去吃午饭或做点什么，这样你就不会在这个过程中摆弄你的设备了。

你将被要求在智能手机上解锁你的 Android 引导程序，这是你在下载和安装更新时需要做的最多的事情。记住，不要碰你的设备。尽管你很想这么做，但如果它看起来没有任何作用，就不要碰你的设备。Android Flash 工具会让你知道什么时候需要做些什么——比如重新锁定引导程序。耐心点！
