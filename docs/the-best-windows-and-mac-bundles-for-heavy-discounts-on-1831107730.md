# 最棒的 Windows 和 Mac 套装，在应用程序和游戏上提供大幅折扣

> 原文:[https://life hacker . com/the-best-windows-and-MAC-bundles-for-heavy-discounts-on-1831107730](https://lifehacker.com/the-best-windows-and-mac-bundles-for-heavy-discounts-on-1831107730)

应用捆绑包是一个很好的方式，可以节省一大笔有用的应用和游戏的钱。这个概念很容易理解:开发者或第三方店面将一堆有趣的软件打包到一个数字盒子里，然后以很大的折扣出售，比你单独购买每个应用程序的成本要低得多。有时候，你打一个统一费率就能获得一批应用；其他时候，你可以自己定价，你支付的费用决定了你能获得多少应用程序。

应用捆绑包的问题是有很多垃圾。许多网站非常愿意包装垃圾，称之为交易，甚至一些提供应用捆绑包的更合法的公司也会提供听起来有点奇怪的应用，或者更糟的是，复制另一个免费应用(或你的操作系统)将免费提供的功能。

小心行事很重要。研究你得到了什么，确定你想要的是否真的是一笔交易，对你安装在台式机或笔记本电脑上的东西格外挑剔。为了帮助您远离骗局，我们收集了一些值得研究的主要 Windows 和 Mac 软件包:

### 通用捆绑包

**[**卑微的捆绑**](https://www.humblebundle.com/)** 

**啊，卑微的包裹。这个网站一开始是一种廉价的方式来获得许多令人敬畏的独立游戏和鲜为人知的游戏，现在几乎是家喻户晓的名字，而且该网站早已扩展到包括许多其他类别的捆绑包。当我输入这个的时候，你可以以非常低的价格获得一堆维加斯的编辑应用——音频和视频，而不是旨在帮助你处理沙漠派对后遗症的应用。我应该指出，这是您自己设定的价格—Humble Bundle 最广为人知的特性之一(也是省钱的最佳方式之一)。**

**[**狂热**](https://www.fanatical.com/en/bundle)T5】**

**你喜欢游戏吗？是的，你有！你喜欢蒸汽吗？我希望如此，因为所有狂热的游戏打折包是为阀门的服务，期间。购买捆绑包，获得在 Steam 上解锁游戏的代码，玩游戏，购买更多捆绑包，重复。就这么简单。此外，所有捆绑包都是固定价格——这里没有“说出您将支付的价格”选项。** 

**[](https://www.indiegala.com/)**

****这个网站比大多数网站有更多的游戏和捆绑交易，但你可以在这里的一些陌生人销售中找到一些瑰宝，比如目前游戏和书籍的“仅限成人销售”。我不会在工作中点击那个，但我会在 Steam 中搜索 Indiegala 各种捆绑包中的一些游戏，在购买前检查它们的评级和评论。这是确保你得到的实际上是一个体面制作的游戏，而不是一些蹩脚的，充满 bug 的体验的最好方法之一。****

### ****仅限 Mac 的捆绑包****

****[](https://www.macheist.com/)****

******我不熟悉 MacHeist 的上一个捆绑包中的大多数应用程序，一个“黑色星期五”主题的版本，该网站目前也没有提供新的捆绑包供你下载。尽管如此，它仍然值得书签和不时检查。在谷歌上快速搜索一下上一个捆绑包中的一些应用，比如[【Acorn 6】](https://www.macworld.co.uk/review/photo-editing/acorn-5-review-3680153/)，会发现其中一些确实是不错的选择。其他的就没那么多了，但这就是应用捆绑包的代价:你可以便宜地得到很多应用，即使你不喜欢(或从来不使用)其中的一些。******

******[](https://stacksocial.com/bundles)******

******一样的交易。你会发现很多课程捆绑包、偶尔的游戏捆绑包和一些应用程序，但这并不意味着你应该走出去，把你的钱堆在 Social 上，因为一些应用程序听起来很有用(而且不贵)。当我为这篇文章查看这个网站时，我对它的数字棋盘游戏捆绑包比对它的基本 Mac 捆绑包更感兴趣，但这只是我的看法。******

******值得称赞的是，StackSocial 网站确实在其描述的底部列出了一个应用程序的评论，所以在你做出购买决定时要考虑到这一点。如果捆绑包中的一堆应用程序从未被审查过，并且你无法通过网络搜索找到太多关于它们的信息(或者没有多少人审查过苹果应用商店中的应用程序)，你可能会想要推迟。将网站加入书签，保持耐心，等待一个绝佳的机会出现。******

******[](https://bundlehunt.com/)******

******BundleHunt 的 Mac 捆绑包自助餐为您提供了大量打折应用供您选择。你预先支付一个“解锁价格”就可以获得折扣，然后你可以将你想要的任何应用添加到你自己创建的捆绑包中——网站不会为你捆绑。我之前已经 [谈论过 BundleHunt](https://lifehacker.com/find-discounted-mac-apps-in-bundlehunts-new-app-pack-1830279655) ，并试图为你可能想要查看的应用提供一些建议。最终，你将不得不做自己的研究，以确保你买的东西是便宜的*和*惊人的，但这是你为，呃，大打折扣的价格所付出的代价。******

******[**Setapp**](https://setapp.com/)T5】******

****Setapp 并不像我们一直使用的那样是一个真正的捆绑包，因为你要支付每月一次的订阅费来访问应用程序，而不是以一笔很低的费用获得一堆应用程序。也就是说，如果你注册了 Setapp(10 美元/月)，你会有很多*的应用可以玩——以至于你可能甚至不再需要购买应用捆绑包。点击这里 查看完整的应用程序列表，这比你在任何常规应用程序捆绑包中找到的都多。由于 Setapp 总是在更新它的列表，你将有新的软件可以定期使用。*****
