# 如何在飞行中减少浪费

> 原文：<https://lifehacker.com/how-to-reduce-waste-while-flying-1835267109>

航空公司产生大量垃圾。想想包裹你的食物的塑料包装，你的耳机和毯子，还有扔在过道里的吸管、餐巾和糖包。 [美国大陆控股公司](https://www.bloomberg.com/news/articles/2019-06-05/united-s-planet-flight-takes-a-step-toward-garbage-free-skies) 。联合航空公司的所有者说，他们平均一次飞行产生 65 磅的垃圾，而且大部分垃圾不太可能被回收。

为了纪念今天的世界环境日！——该航空公司今天早上推出了一个从芝加哥到洛杉矶的“绿色”航班，使用混合生物燃料飞行，购买碳抵消，并通过可回收杯等方法减少他们的整体垃圾。然而，值得注意的是,只是一次飞行，而不是一次持续的努力。

虽然零垃圾旅行不是我们很快就能实现的未来，但每个旅行者都可以通过改变习惯来减少自己的垃圾；这通常就像拒绝使用尽可能多的一次性垃圾和携带可重复使用的替代品一样简单。

### 冷冻任何食物，拔掉电器插头

在登机前，你应该在家做几件事来减少你的个人浪费。首先，如果你要进行为期一周的旅行，你应该在冰箱里冷冻任何食物，或者提供给可能需要的人。

此外，在离开之前拔掉一些电子设备的插头，包括那些带有外部(或“电源砖”)、任何处于睡眠或待机模式的设备、小电器(如微波炉或咖啡机)以及调制解调器或路由器——所有这些都需要电力。如果可能的话，一定要关掉所有的空调和你的热水器，因为它们在不使用的时候也会耗尽电能。

### 打包零废物工具包

减少你自己浪费的最简单的方法就是做好准备，用可重复使用的物品打包你自己的零浪费工具包。(在这种情况下，零浪费只是一个目标，因为实际上根本不可能产生任何浪费。)“想想你在旅行中可能会遇到的一次性塑料物品，并带上一些可重复使用的物品，” [零废物专家](https://pollybarks.com) 波利·巴克斯通过电子邮件说。“考虑用可重复使用的冷热旅行杯代替水瓶或一次性咖啡杯，一把真正的叉子，布餐巾，可重复使用容器中的零食等。”

可重复使用的水瓶或水壶是关键；你可以在通过安检后加满油，跳过机场昂贵的塑料版本。(你可以让航空公司的服务员给你倒酒，而不是用杯子。)也要自带零食；你可以拒绝任何椒盐卷饼或饼干，乘客可以在随后的航班上使用。

当然，你也应该拒绝任何其他一次性垃圾，如吸管或餐巾纸；如果服务员给了你一张餐巾纸，那就把它带在身边以备后用，这样至少不会浪费掉。(如果他们给你一整罐汽水，你就把它带走，这样你以后可以回收利用。)你也可以带上自己的一副耳机和一件 [毛衣](http://trashisfortossers.com/how-to-travel-without-waste-tips-for-the-eco-conscious-traveler/) ，以减少你不得不从打开的航空公司的耳机或毯子中使用的一些塑料。

另一方面，不要仅仅为了减少垃圾而拒绝国际航班上的食物(T2)。— 根据 [当地法律](https://www.usatoday.com/story/travel/columnist/cox/2019/02/22/airline-food-garbage/2941319002/) 的规定，任何 [未吃完的食物和垃圾](https://thepointsguy.com/news/what-happens-to-uneaten-food-on-airplanes/)很可能会被焚烧。然而，如果你乘坐国内航班或廉价航班，通常未吃完的食物和饮料会在晚些时候的航班上使用。

### 购买碳补偿

购买机票时，你可能会选择购买碳补偿；这些是对绿色组织的捐赠，帮助抵消你旅行中的碳排放。巴克斯说:“一个组织计算你的碳排放成本，你支付等量的金钱。”

正如 [阿法尔](https://www.afar.com/magazine/everything-you-need-to-know-about-carbon-offsets?utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+AfarMagazine+%28AFAR+Magazine%29) 所写，这是一件有点棘手的事情。批评者认为这只是在向问题扔钱，根据钱来计算补偿有点模糊。此外，许多倡议包括植树来抵消你的排放，这可能需要很多年才能吸收大量的碳。(换句话说，他们就是没那么有效率。)

尽管如此，总比什么都没有好，尽管当你决定购买这些补偿时，值得考虑你要捐赠给谁。“这不是一个完美的解决方案(这只是一个长期的解决方案，不会减轻你旅行的即时影响)，但它可以帮助你感觉更好，并支持有价值的项目，”Barks 补充道。

例如，联合航空公司与国际保护组织合作，后者将计算你在某次飞行中的碳足迹，得出货币价值，并将其捐赠给全球森林和野生动物保护工作(而不仅仅是植树)。

巴克斯建议使用 [特拉帕斯](https://www.terrapass.com/carbon-footprint-calculator) 来计算你的碳数。一旦你有了这样的想法，你也可以向与你的航空公司无关的组织捐款；例如，你可以使用 [Green-E 的](https://www.green-e.org/certified-resources/carbon-offsets) 审查组织列表，这样你就可以找到一个有价值的事业。
