# 当你飘飘欲仙时可以尝试的怪异网络体验

> 原文：<https://lifehacker.com/try-these-weird-web-experiences-when-youre-high-1834148683>

虽然我已经很久很久没有嗑药了，但我仍然记得我嗑药时的那些愚蠢的冒险经历。那是在大学时代——很久很久以前——当我们品尝各种(难吃的)食物时，我们只有超级粉碎兄弟、肉搏、熔岩灯和壁球来娱乐自己。啊，更简单的时代。

Watch

如今，如果我想爽，我会扑通一声坐在电脑前。这并不是因为我不合群；你嗑药的时候可以在网上做很多奇怪的事情。

我不会说这是当你兴奋时值得看的东西的权威列表。很多经常分享绿叶的人可能会有一个文件夹，里面装满了有趣的网络书签，当你抽大麻的时候，这些书签会让你感觉完全在那里。请在评论中告诉我一些你最喜欢的，我会在以后的文章中重点介绍它们。

否则，这是我在被击中后击中的东西。

### [电羊](https://electricsheep.org/#/download)

这个必备的屏幕保护程序给你一个不断旋转的随机动画系列。即使你没有嗑药，你也会认为你的电脑出问题了。如果我不是苹果的空中屏保( [被黑为 Windows](https://github.com/cDima/Aerial) )的粉丝，我会毫不犹豫地使用它。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-t6jlhqNxRYk&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-t6jlhqNxRYk&start=0) 

### [演示场景](http://www.pouet.net/prodlist.php?order=views)

如果你真的不喜欢屏幕保护程序，看看这些不可思议的编码作品——通常伴随着古怪的音乐节拍——它们将大量的动画压缩到一个小小的文件中。(相关的 [子编辑](https://www.reddit.com/r/Demoscene/) 也是找到它们的好资源。)

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-rFv7mHTf0nA&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-rFv7mHTf0nA&start=0) 

### [运行一些流体实验](https://haxiomic.github.io/GPU-Fluid-Experiments/html5/?q=UltraHigh)

我不高，最近十分钟一直在 [玩这个网站](https://haxiomic.github.io/GPU-Fluid-Experiments/html5/?q=UltraHigh) 。这太棒了。

### [雷兹无限](http://rezinfinite.com/)

当你兴奋的时候试着给这个游戏一个机会。如果你在虚拟现实中玩这个游戏，而且不呕吐，会有加分。

快乐的树。还需要我多说吗？如果 24/7 鲍勃·罗斯抽搐流的舒缓声音不为你做，我不知道你烤过的自己还需要什么。把那个灌木丛里的魔鬼打出来。

这是你的房子。好吧。这是每个人的房子。有什么比实时观看地球更好的方式，让你感觉自己在太空中漂流，被烤得神志不清？

### [我的醉厨](https://www.youtube.com/watch?v=zL0OD1lf5Dc)

现在我在交叉淡入淡出，但是听我说完；尽管《我的醉酒厨房》的主持人在炮制她偶尔看起来很糟糕的作品时并不兴奋，但她肯定不是处于正常状态。也许这将激发你在开始感到可怕的零食之前，创造你自己的烹饪实验。

### 那些古老的瓦雷兹介绍

回到过去，非法获得的应用程序和游戏的档案通常带有安装程序、破解工具或其他一些小的可执行文件。这些会以一些聪明的小动画和歌曲为特色，被“场景”中的团体用来为他们的发布而自我鼓励。虽然动画通常很短，但音乐却非常吸引人——也是美妙的背景噪音，让你度过一个愉快的夜晚。查看 [Flashtro](http://flashtro.com/) 、 [KeyGenMusic](http://www.keygenmusic.net/) 和 [WAB](http://www.wab.com/) 获得你需要的所有 chiptunes。

### 制作一些分形

啊，分形——让我想起了《侏罗纪公园》这部小说。通过 [创造](https://chaoticafractals.com/) (或 [通过](https://codeparade.itch.io/marblemarcher) )源源不断的无限图案来丰富你的广生党。就好像这个世界 [在建造自己](https://fractaltodesktop.com/index.html) 一样，人。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-9U0XVdvQwAI&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-9U0XVdvQwAI&start=0) 

### 制作自己的音乐

如果你更喜欢创作音乐而不是盯着生动的图像，可以试试 [拼接声音](https://splice.com/sounds/beatmaker)——一个你可以在浏览器中使用的手柄小音序器——或者更简约的 [采样器](http://sampulator.com/) 。还有 [Incredibox](https://www.incredibox.com/demo/v4) ，如果你觉得傻的话。而如果你只是想用最小的努力获得环境噪音，那么 [双耳打击机](https://mynoise.net/NoiseMachines/binauralBrainwaveGenerator.php) 绝对不会错。
