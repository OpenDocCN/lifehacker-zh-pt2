# 用味噌改善几乎所有感恩节菜肴

> 原文:[https://life hacker . com/improve-almost-any-thanksgiving-dish with-miso-1830473857](https://lifehacker.com/improve-almost-any-thanksgiving-dish-with-miso-1830473857)

有些人对感恩节大餐的口味很挑剔——他们想要经典的，他们想让他们品尝经典。当我建议在馅料中加入味噌时，那些人 [冲我](https://skillet.lifehacker.com/you-should-definitely-add-miso-to-your-thanksgiving-stu-1820053100) 大喊大叫，但我很固执，我会继续谈论味噌在几乎任何感恩节菜肴中都有多好。

虽然发酵豆酱在日本料理中最常见，但它有一种鲜味，咸味，略带坚果和甜味，与火鸡、填料、南瓜、任何砂锅，甚至南瓜派搭配都非常美味。它增加了复杂性，盐，甜坚果味，如果你想吃更黑的品种，还有一点臭味，但没有压倒菜里其他更“传统”的味道。

将味噌加入你的食物就像用勺子搅拌你正在做的任何东西的液体成分一样简单——看看你，浓缩蘑菇汤——但是我们有一些非常具体的想法，并且(显然)愿意分享:

*   **馅饼:** [在黄油、蛋羹或水果馅饼的浸汁中加入一点儿果浆](https://skillet.lifehacker.com/improve-your-favorite-thanksgiving-pies-by-adding-a-lit-1830138102) 作为人人都会争抢的甜点。(在南瓜派里面唱的不错，但是在山核桃里面唱的，反正是上等派。)
*   **砂锅菜:**搅拌几茶匙(或者用汤匙搅疯！)的深色物质(红色或黄色)加入你的面酱、浓缩汤或 [酸奶油](https://skillet.lifehacker.com/an-ode-to-jiffy-corn-casserole-1830231857) 中。
*   **馅料** : [我帮爱丽丝](https://skillet.lifehacker.com/make-these-thanksgiving-classics-even-better-1830236674#_ga=2.55162411.1037275332.1542039900-1157707288.1539189461) 为我们的员工餐做了 [这种味噌蘑菇馅料](https://skillet.lifehacker.com/you-should-definitely-add-miso-to-your-thanksgiving-stu-1820053100) ，他们很喜欢，但每杯肉汤两汤匙在任何馅料配方中都是不错的。
*   **任何纯素食菜肴:**味噌的真正优势在于为无肉餐增添了美味。土豆泥中的白色物质非常美味，拌入橄榄油的味噌搭配任何烤蔬菜都很棒，尤其是抱子甘蓝和南瓜。这也使得 [的素食肉汁](https://www.epicurious.com/recipes/food/views/mushroom-miso-mustard-gravy-231051) 的美味无比。(真见鬼，一份 [素食肉汁太好了](https://skillet.lifehacker.com/five-ways-to-break-your-gravy-ennui-1820629338) 。)
*   **火鸡:**我不知道你是否听说过这种鸟，但据说它本身味道不怎么样，喜欢腌泡汁。 [这种三种成分的味噌酱](https://skillet.lifehacker.com/make-this-simple-miso-marinade-for-all-of-your-meaty-ne-1796723345) 是我最喜欢涂在任何一种肉上的东西，它的甜和咸让每个人都想狼吞虎咽。

显然，如果你在做汤——比如奶油胡萝卜——也要放一些进去。味噌实际上是用来做汤的。
