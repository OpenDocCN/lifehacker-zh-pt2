# 使用旧的学校索引卡来捕捉新的想法

> 原文：<https://lifehacker.com/use-old-school-index-cards-for-capturing-new-ideas-1838074984>

我尝试过 Evernote 和 Scrivener、paper planners 和 Moleskines，以及世界上几乎所有的应用程序来捕捉我的想法。但是我不断回到一个简单的、被低估的工具上来捕捉那些似乎不适合其他地方的半生不熟的想法和想法。最完美的地方是不起眼的索引卡。

Watch

当然，索引卡在我们的个人组织系统中来来去去。很少有人用它们来记录家庭食谱，但索引卡仍然不仅仅是制作卡片或在演讲前记录要点的媒介。你可能听说过 [把事情做完](https://lifehacker.com/getting-things-done-ten-years-in-1795707084)【GTD】，这个系统让你写下每一个待办事项，这样你的大脑就不需要一次处理太多信息。这个系统的一个扩展是使用索引卡来跟踪每天的任务和记录，记录你喜欢的引用，或者记录你学到的东西。你把它们都放在一个盒子里，按日期或笔记类型分类。它被恰如其分地称为 [堆索引卡](https://lifehacker.com/the-pile-of-index-cards-system-efficiently-organizes-ta-1599093089) 系统。

这就是我偏离组织轨道的地方。我用索引卡记录所有不实用的笔记。提醒和待办事项放在我的计划器里，对某个专业领域的研究放在谷歌文档的目录里。但是随机的、不成熟的想法会出现在索引卡上。我把它们堆在一起，但是那堆东西没有组织好。这是一个大混乱。

当我不赚钱的时候，我会尝试写小说和个人散文，为日常生活中出现的问题、俏皮话或简短场景准备一个专用笔记本的想法令人生畏。 [你有多少本笔记本](https://lifehacker.com/these-are-the-best-notebooks-1835480916) 前三页都填好了就藏起来，结果却被抛弃了？我讨厌这样做。

所以我带走了空白页的压力。取而代之的是一张索引卡。这是我思考的舒适空间。如果我觉得整洁，我会用有衬里的那一面。感觉潦草？我把它翻到空白的一面。你可以往任何方向写，我不在乎。像 GTD 一样，它有助于在你的大脑有机会忘记这个想法之前，把它从你的脑海中抹去。

当我需要一些写作灵感时，我会抽出那叠卡片。他们中的大多数是神秘的，在写作中除了我没有人能破译。其中一个简单地说，“钥匙断了。”两个字。我知道它的意思，而且由于索引卡的篇幅有限，我觉得我不需要在笔记本的剩余部分展开这个短语。我过会儿会得到它。需要的时候我会用的。

这有点像 Twyla Tharp 在她的书*创作习惯*中解释的项目研究过程。你为每个项目开一个新的盒子，所有的材料，不管是剪报、照片还是笔记，都放进盒子里。你不断添加，直到项目完成，在这一点上，你标签的盒子，并储存起来。

我以为我只是一个受 Twyla 启发的拿着一叠索引卡的怪人，直到我了解了喜剧演员菲莉丝·迪勒的插科打诨文件 。这是一个有 48 个抽屉的卡片目录，每一个都装满了她几十年来按类别整理的笑话——大约 40，000 个笑话！笑话文件对喜剧演员来说并不罕见，但对我来说，这是我需要拥抱我的索引卡的顿悟。迪勒拥有她的方法。是时候拥有我的了，不管在别人看来有多乱。

最有效的组织系统就是为你工作的系统。你可以从别人那里获得灵感，测试他们的方法，但是你可能最终会发现你不需要别人给你指路。
