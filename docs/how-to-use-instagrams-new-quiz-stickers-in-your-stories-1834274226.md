# 如何在你的故事中使用 Instagram 的新问答贴

> 原文：<https://lifehacker.com/how-to-use-instagrams-new-quiz-stickers-in-your-stories-1834274226>

两年前，Instagram 在 Instagram Stories 中增加了对投票的支持。今天，它将这一功能向前推进了一步，引入了一个新的“测验”功能，允许您在自己的故事中向朋友提出多项选择问题。

Watch

(你已经可以用投票的方式做到这一点了，你可以提供两个选项，让你的朋友在它们之间投票。有了这个功能，你将有多个选项可供选择，以及一个正确的答案。)

为了测试这个特性，我设计了这个有趣的测试，测试我的狗是否是世界上最可爱的狗。

左边的照片是我创建它时看到的，而右边的照片是我的朋友在我的故事中看到的。

要创建测验，您可以按贴纸按钮(与调出 gif 和日期的按钮相同),然后从出现的选项中选择“测验”。如果你打开 Instagram，没有看到测验选项，请确保你当前运行的是最新版本的应用。

在那里，将出现一个测验框。您在框的顶部键入一个问题，然后在下面键入问题。要创建测验，您至少需要有两个问题，但最多可以包含四个问题。

一旦你输入了所有的答案，你就可以点击页面上的“正确”答案。当你这样做时，它会亮起绿灯(只为你)，这样你就知道你选择了哪一个。

之后，你就像平常一样发表东西。当你的朋友看到你的帖子时，他们可以从你的测验中选择答案。他们会立即看到他们是否正确，他们的答案会发送给你，这样你也可以看到他们做得如何。
