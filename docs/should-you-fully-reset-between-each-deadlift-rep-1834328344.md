# 你应该在每次硬拉之间完全复位吗？

> 原文：<https://lifehacker.com/should-you-fully-reset-between-each-deadlift-rep-1834328344>

我一生中有过三次不同的下背部受伤，都是因为糟糕的硬拉形式。直到几年前，在参加了一个举重健身房后，我才终于了解到什么可能是我持续背痛的最大原因:一触即发的硬拉动作，以及在每次练习之间轻轻擦过地板



如果你读过 Reddit[r/fitness threads](https://www.reddit.com/r/Fitness/comments/64ic0u/difference_between_touch_and_go_deadlift_and_full/)，你会发现重复之间的完全重置是唯一可以接受的硬拉方式；这是真的，不像一触即发，你在硬拉的时候休息一下会受益更多。

### 完全重置有助于你获得离地力量

正如阿维·西尔弗伯格在[【PowerliftingTechnique.com】](https://powerliftingtechnique.com/touch-and-go-vs-reset-deadlifts/)上所写的那样，硬拉的最底部范围——将杠铃提离地面——需要你的股四头肌和脊柱直立肌(沿着你的背部长度)的巨大力量。通过重复之间的完全重置，你需要这两个区域的最大力量来承担每次重复的重量。

然而，如果你坚持一触即发的硬拉，你很有可能“反弹”你的代表，或者通过将重量反弹离地板来辅助硬拉的底部。你利用的是反弹的动量，而不一定是你的力量。

西尔弗伯格写道:“如果你经常习惯于从底部得到‘反弹’，那么在拉一个代表时就不会有反弹，”这对竞争性力量举重运动员或其他专注于提高一个代表最大值的人来说是一个问题。通过重置硬拉训练，你可以更好地准备离开地面(从个人经验来看，我可以说这通常是提升中最困难的部分)。

### 复位是必要的形式，以避免受伤

在重置代表之间，你可以很容易地纠正你的形式；根据私人在线教练 k .阿莱莎·费瑟 的说法，在关键时刻，你没有那么多时间来纠正航向。通过使用触球，如果你失去了每个代表之间的紧张或紧绷，你的形式就有危险。

她在电话中说，“如果你在下降的过程中失去了紧张感，并且每次都是这样，那么当你进行到第八次时，你的状态可能会完全恶化。”“每次重复之间的重置可以确保你有最大的张力，你处于正确的状态，并且每次重复都来自最佳的基础设置。在每一项练习中，质量应该优先于数量，这就是重置的作用。”

换句话说，通过触地硬拉受伤的风险更大(2018 年的一项研究[发现](https://www.ncbi.nlm.nih.gov/pubmed/29489730) 你的后链，你身体后部的肌肉，如腿筋和臀肌，*可能*在弹跳硬拉过程中比完全复位的提升风险更大)。

### 重置时，最好暂停几秒钟

有没有一种情况下冒险可能会有所帮助？是的，事实证明，你可以用它们来减轻重量，做更多的额外有氧运动。“它可以用来发展速度和力量，”私人教练肯尼·马哈迪奥在电话中说。"但是你需要更加注意技术."

然而，根据 Fetters 的说法，如果你打算安全地提高你的最大单次成绩，重置至少一到两秒可能是关键。“不一定要很大，”她说。“只要确保你处于正确的位置，你的脊椎是好的，把那些背肌固定住，然后有目的地进入下一个练习。对于硬拉，很多人都在鼓捣，但你必须控制好形式，给自己一些时间。”如果你特别有野心，你甚至可以尝试 [赤字硬拉](https://barbend.com/deficit-deadlifts-benefits/) ，这需要站在一个高架平台上，可以作为让体重离开地面的练习，尽管不是每个人都适合。

如果你不得不降低重量来固定那些完全重置的代表，那也没关系——毕竟这都是以形式的名义。
