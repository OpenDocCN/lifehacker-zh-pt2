# 如何在 Chrome OS 上观看画中画视频

> 原文:[https://life hacker . com/how-to-watch-picture-in-picture-videos-on-chrome-OS-1842096378](https://lifehacker.com/how-to-watch-picture-in-picture-videos-on-chrome-os-1842096378)

对许多人来说，画中画(PIP)视频播放几乎是一项普遍功能——除非你拥有一台 Chromebook。在那里，该功能只对几个应用程序有效，不像你会在 Android、iOS、Windows 和 macOS 上的 [网络浏览器和](https://lifehacker.com/how-to-watch-picture-in-picture-videos-in-firefox-and-c-1840272901) 视频流应用程序中获得的体验。

然而，支持者似乎越来越多。最新的 Chrome OS 更新 [增加了对网飞](https://www.androidpolice.com/2020/03/02/chromebooks-get-pip-for-netflix-ambient-eq-coming-to-more-devices/) 流的自动画中画支持，这意味着未来有望有更多的应用支持画中画。但与其他平台不同的是，在其他平台上切换到 PIP 大多是自动的，在网飞的 Chrome OS 应用程序中启用 PIP 的方法与在 Chrome 中观看 Youtube 视频的方法不同。以下是如何在两个支持 PIP 的 Chrome OS 应用上启用 PIP。

### 网飞

在 Chrome OS 应用程序中观看网飞时，只需切换到另一个应用程序，视频就会缩小到一个更小的窗口中并继续播放。您还可以轻按/点按视频以返回全屏幕、播放/暂停，或者将它拖到屏幕的其他部分(如果它妨碍了另一个应用程序)。

### 谷歌浏览器

Chrome 浏览器本身是唯一一个支持 PIP 的 Chrome 操作系统应用。该功能将允许你在 PIP 窗口中观看几乎任何视频内容，这意味着只要你通过浏览器观看，几乎所有支持 Chrome 的视频平台或网站都可以在 PIP 窗口中播放。

你需要先下载并安装这个 [官方 Chrome 扩展](https://chrome.google.com/webstore/detail/picture-in-picture-extens/hkgfoiooedgoejojocmhlaklaeopbecg) 。然后，当你在看视频时，按下**“Alt+P”**将视频弹出到一个单独的较小的视频播放器中。即使你浏览其他标签和应用程序，视频也会继续播放，并且可以在屏幕上拖动。轻按或点击视频也会调出基本的播放功能。
