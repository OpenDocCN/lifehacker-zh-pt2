# 用鸡尾酒苦味制作更好的柠檬水

> 原文:[https://life hacker . com/make-better-lemonade-with-cocktail-bitters-1844039249](https://lifehacker.com/make-better-lemonade-with-cocktail-bitters-1844039249)

我的酒吧推车上有大约七瓶鸡尾酒苦味酒，但在制作鸡尾酒时，我真的只使用两瓶——ango stura 和 orange。这并不是说其他口味不好，只是我很少想要一种用葡萄柚或薰衣草苦味剂制成的老式饮料。但是柠檬水是完全不同的故事。

Watch

鸡尾酒苦味剂是制作定制柠檬水最简单的方法。你不需要给任何乱七八糟的水果榨汁，也不需要煨煮任何注入植物的糖浆。你所要做的就是摇几滴你喜欢的苦味酒到杯子里，T1 s T2 T3 可选。经典的苦味酒实际上可以增加一点草本的深度，但是味道越古怪，柠檬水就越有趣。

除了巧克力，我想不出一种在这里不适用的苦啤酒。葡萄柚苦味能带来气泡柠檬水的感觉；花香苦味给你的饮料一种花园派对的感觉。像大黄、蔓越莓和樱桃这样的水果朋友可以做出很棒的粉红色柠檬水。如果你想要真正的*和轻微的植物性——芹菜苦味剂可以提供一点点诱人的苦味。想来杯辣柠檬水吗？伸手去拿胡椒味的苦啤酒！甚至山核桃苦味也能增加一种不错的坚果风味。*

但我最喜欢在柠檬水中加入苦味剂的一点是它是多么简单。你可以做一大罐原味柠檬水，并在每一杯里加不同的苦。作为一个喜欢变化、讨厌重复的人，这让我兴奋不已。事实上，如果聚会再次被允许的话，我建议你在你的玉米片酒吧或其他酒吧旁边放一个柠檬水和苦味酒吧。人们喜欢 DIY 食物和饮料吧。

此外，苦味剂让柠檬水看起来很美。审美不是一切，但人们必须欣赏既美丽又美味的饮料。
