# 标签组是 Chrome 多年来最好的新功能，以下是使用方法

> 原文：<https://gizmodo.com/tab-groups-is-chromes-best-new-feature-in-years-and-he-1842944249>

在经历了浏览器的开发版和测试版之后，一个重要的新功能即将出现在我们大多数人都在使用的谷歌 Chrome 稳定版中:标签组。它可能会永远改变你浏览网页的方式。

***更新:**我们最初声明标签组已经随着 Chrome 81 为每个人推出，但似乎还不是这样。据我们所知，越来越多的人正在获得这项功能，但如果你还没有在 Chrome 中看到它，请在地址栏中输入"Chrome://flags/# tab-groups "，点击**输入**，你就可以在那里启用这项功能。*

标签组似乎出现在当前版本的谷歌浏览器的中——至少对某些人来说——所以首先你需要确保你运行的是最新版本的浏览器。打开 Chrome 菜单，然后选择**帮助**和**关于谷歌 Chrome** 。如果您尚未运行 81 版或更高版本，您应该会看到更新软件的提示。

顾名思义，选项卡组允许您...分组选项卡。很可能我们大多数人在任何时候打开的标签都比实际可行的要多，新功能让你可以更好地控制如何管理这些标签。各种第三方扩展在过去已经解决了这个问题(下面有更多关于这些插件的内容)，但是现在 Chrome 提供了一些内置的帮助。

开始使用选项卡组最简单的方法是在 Chrome 的选项卡标题上右键单击(或者在 Mac 上用**Ctrl+单击**)——你应该会看到一个将**添加到新组**的选项。你会看到标签有一个彩色的轮廓，旁边有一个彩色的点。

如果您不喜欢分配给它的默认颜色，单击圆点来命名您的全新选项卡组(一个将出现在选项卡栏上的名称)，并更改颜色。此菜单上的其他选项允许您关闭组中的所有选项卡，取消组中所有选项卡的组合(删除组)，以及向组中添加新的空选项卡。

例如，你可能有一个标签组用于工作内容，一个标签组用于社交媒体。如果你正在同时研究两个或更多的主题，或者试图处理多个项目，或者只是试图将你的商务和休闲网页浏览分开，那么这些都是选项卡分组可以派上用场的场景。

一旦你创建了你的第一个组，右键单击(或**Ctrl+单击**)任何标签页标题，你就可以选择将该标签页添加到一个现有的组中，也可以创建一个新的组(在这里聪明地命名你的标签页组是有好处的)。如果您从已经在选项卡组中的选项卡打开链接，新的选项卡将进入同一个组。

您也可以将选项卡添加到组中，只需单击并将其拖入组中即可。选项卡可以用同样的方式在组之间移动。锁定的选项卡是豁免的，不能被分组(如果您尝试将一个锁定的选项卡分组，它会自行解除锁定；您锁定的分组选项卡将不再被分组)。

选项卡组实现的一个很酷的技巧是一起移动选项卡——单击并拖动选项卡组标签，您可以一次移动它的所有选项卡(或者将它们拖到一个全新的窗口)。顺便说一下，你可以在没有标签组的情况下做到这一点，通过用**Ctrl+单击**(或者在 Mac 上用**Cmd+单击**)选择多个标签，然后四处拖动它们。

随着时间的推移，Chrome 开发团队无疑会为标签组添加更多的功能。目前，这不是最稳健的解决方案:例如，简单地拖动单个标签会将其从一个组中拉出，并且没有办法保存标签组或在关闭标签后将它们作为一个组恢复。

尽管如此，这是一个充满希望的开始，表明谷歌工程师正在思考更好的方法来管理我们每天都在处理的大量标签。试一试，你可能会发现它比你想象的更有用。

如果你需要的不仅仅是标签组所能提供的，大量的 Chrome 扩展覆盖了同样的领域。 [集群](https://chrome.google.com/webstore/detail/cluster-window-tab-manage/aadahadfdmiibmdhfmpbeeebejmjnkef) 让你按窗口组织标签，这样你就可以把内容相似的标签分组在一起，然后批量打开和关闭它们。这也使得在标签页中搜索变得更加简单。

[标签管理器加](https://chrome.google.com/webstore/detail/tab-manager-plus-for-chro/cnkdjjdmfiffagllbiiilooaoofcoeff) 使用类似的方法来帮助你基于标签所在的浏览器窗口管理标签——如果你想关闭标签组，但又很容易恢复它们，这很好。作为一个额外的奖励，它将帮助你发现你已经打开的重复标签，并且可以(可选地)限制你总共可以打开多少个标签。

[托比](https://chrome.google.com/webstore/detail/toby-for-chrome/hddnkoipeenegfoeaoibdmnaalmgkpip) 也值得一提，是最全面的。它从本质上改变了 Chrome 中标签和书签的管理方式，为您提供了一种新的、简化的方式来跟踪您需要跟踪的网页。这可能需要一点时间来适应，但如果适合你，它可能会对你的在线生产力产生很大的影响。
