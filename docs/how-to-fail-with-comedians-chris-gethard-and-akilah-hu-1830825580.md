# 喜剧演员克里斯·盖特哈德和阿基拉·休斯的《如何失败》

> 原文：<https://lifehacker.com/how-to-fail-with-comedians-chris-gethard-and-akilah-hu-1830825580>

这是我们第一次现场录制升级，在纽约布鲁克林的贝尔之家 的 [观众面前表演。加入我们的是喜剧演员克里斯·格哈特(](https://www.thebellhouseny.com/)[](https://www.amazon.com/Lose-Well-Chris-Gethard-ebook/dp/B076P4Q652/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-fail-with-comedians-chris-gethard-and-akilah-hu-1830825580&asc_source=&ie=UTF8&keywords=lose well chris gethard&qid=1543860435&sr=8-1&tag=kinjalifehackerlink-20)*， [《克里斯·格哈特秀》](https://www.trutv.com/shows/the-chris-gethard-show/index.html) )和阿基拉·休斯( [“显然是阿基拉”](https://www.youtube.com/user/smoothiefreak) )，以及 [Gimlet Media 的播客《点头》的布列塔尼·卢瑟和埃里克·艾丁斯](https://www.gimletmedia.com/the-nod) 我们谈论如何不失败就不能成功，以及如何克服最尴尬的失败。另外:我们玩了几轮“我从未有过”(升级版)和一个新游戏“Lifehacker 拯救你的生命”它是非常有趣的。*
