# 使用松饼罐和鸡蛋盒来增强野餐体验

> 原文：<https://lifehacker.com/use-muffin-tins-and-egg-cartons-for-an-enhanced-picnic-1844750480>

这是户外用餐的夏天。从人行道到后院到公园，我们都在享受温暖的天气，吸入新鲜空气。然而，保持野餐(和你的手指)干净整洁是一件很有挑战性的事情——尤其是当家的舒适远离你的时候。

因此，我们收集了一些我们最喜欢的野餐技巧，来充分利用你的生日聚会、浪漫午餐或户外玩耍日。首先，在你的毯子下面铺上一块塑料桌布，以防之前的降雨弄湿了草地。



冷冻一些水瓶，用作篮子里的冰袋(或者更好的是，一个看起来像篮子的冷却器，因为篮子不是很隔热)。把瓶子放在食物的顶部，随着冷空气向下流动，一旦它们融化，你就可以喝了。你回来时，冰箱会更轻。T2:那就像是一个三连击。

至于打包你的食物和饮料，保持一切整洁，看看上面视频中的所有技巧。
