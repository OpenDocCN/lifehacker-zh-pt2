# 如何写一封出色的求职信

> 原文：<https://lifehacker.com/how-to-write-a-great-cover-letter-1836000052>

几周前，我的一个密友在脸书上发布了一份工作，她的办公室正好有一份工作，恰好与我另一个正在找新工作的朋友完全匹配。我把信息传递给第二个朋友，并把他们联系在一起。



按照惯例，在这种情况下，一号朋友愿意回答二号朋友关于演出的任何问题。然而，他唯一的问题让我很惊讶。他问，“这是那种看求职信的办公室吗？”

显然，他现在的办公室倾向于把求职信扔进垃圾桶，只关注申请人的简历和他们在门户网站上填写的内容。他想知道他是否应该花时间在求职信中写一些有思想的东西，或者只是发送一些简短而甜蜜的信息，这些信息可能永远不会或几乎不会被阅读。

她的回答大致是“亲爱的上帝，请写一封求职信。”

本周快公司 [发表了一篇关于](https://www.fastcompany.com/90367147/is-writing-a-cover-letter-worth-it) 写求职信是否仍然值得的故事。简短的回答？是的，是的，它是。

虽然肯定有一些地方不重视求职信，但大多数企业仍然如此。它们很重要。有趣的是，我知道有几次我参与一个职位的招聘过程时，我都是从求职信开始的，如果它没有让我惊叹，或者至少没有让我感兴趣，我就从来没有出现在别人的简历中。

除了注意到求职信的重要性，《快速公司》还就如何写一封好的求职信提出了一些建议，首先要确保你的求职信是写给正确的人，而不是泛泛而谈的“你可能会关注的人”花额外的几分钟找出招聘经理是谁，或者至少是正确的团队。付出努力。

除此之外，不要一开始就对你申请的工作做一般性的解释，而是使用你的开场图，用你对工作或公司感兴趣的一个方面来吸引人们的注意力。之后，接着用具体的例子说明为什么你认为自己是这份工作的最佳人选。

正如我们在 之前提到的 [，了解你的读者和了解你自己也很重要。](https://lifehacker.com/how-to-write-a-cover-letter-that-employers-will-actuall-5880545) 

保持专业，但也尽量保持原创，毕竟你想脱颖而出。
