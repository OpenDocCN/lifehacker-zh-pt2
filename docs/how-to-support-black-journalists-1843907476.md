# 如何支持黑人记者

> 原文：<https://lifehacker.com/how-to-support-black-journalists-1843907476>

本周,《纽约时报》选择发表阿肯色州参议员汤姆·科顿的一篇极具争议且危险的专栏文章，他在文章中主张对抗议者使用军事力量。正如许多记者所注意到的，在没有背景、没有 [对科顿的许多不准确的说法](https://twitter.com/jenvalentino/status/1268543700789014534) 进行事实核查，并且没有对危险、非法的立场提供任何反驳的论据的情况下，决定刊登这篇文章，将 b 缺乏记者置于危险之中。



尽管有这些批评，社论版编辑詹姆斯·班纳特选择了 [在他的决定上加倍下注，在一篇冗长的](https://www.cnn.com/2020/06/03/media/new-york-times-tom-cotton-op-ed/index.html) [推特帖子](https://twitter.com/JBennet/status/1268328278730866689?s=20) 中为他的行为辩护。作为回应，该报的一些工作人员已经通过在推特上发布专栏截屏来抗议，并附上一个简单的声明:“这样做将布莱克@纽约时报的工作人员置于危险之中。”

鉴于《纽约时报》的这一决定，以及《纽约时报》随后拒绝承认或纠正他们的错误(也许通过撤下这篇文章)，你可能正在寻找方法来表达你对这些缺乏新闻记者的支持。一种方法是捐赠给帮助新闻工作者的组织或者订阅新闻工作者拥有的报纸。

### 向支持记者的组织捐款

新闻编辑室倾向于不成比例的白人和男性，这一事实影响了新闻报道的内容和方式。如果我们希望新闻报道更准确地反映我们的世界，那么我们绝对必须支持和鼓励来自被报道群体的新闻工作者——包括缺乏新闻工作者，他们在职业生涯的关键时刻需要支持，尤其是在他们刚刚起步的时候。正如美国劳动力的许多方面一样，机会往往会流向那些拥有最好关系和最多财力的人。

支持 b lack j 记者的组织有[Ida b . Wells Society](https://idabwellssociety.org/)，通过地区研习班和实习来支持有色人种记者；全国黑人记者协会，通过奖学金和实习来支持 b 缺乏新闻工作者；以及 [梅纳德学院](https://www.mije.org/) ，该学院为新闻编辑室提供如何公正准确地报道有色人种社区的培训项目。所有这些组织都需要你的财政支持。现在和未来

### 订阅当地黑人拥有的报纸

如果你想通过订阅一份 b lack 拥有的独立报纸、、[来表示你的支持，这份由非裔美国文学读书俱乐部编制的列表](https://aalbc.com/newspapers/)、是一个很好的资源，会帮你在自己的社区内找到一个新闻资源。
