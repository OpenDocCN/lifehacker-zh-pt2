# 当你在为飓风做准备时，去自动取款机吧

> 原文：<https://lifehacker.com/when-youre-preparing-for-a-hurricane-go-to-the-atm-1837704659>

你在用木板封住窗户。你在储备水。你在挖沙袋。但是等等:你有现金吗？如果你正在为飓风、热带风暴或其他提前通知的天气事件做准备，你应该把去银行或自动取款机列入你的任务清单。



暴风雨过后，现金为王。在电力恢复之前，商业可能会重新开放，这将使现金持有者排到从汽油和食物到冰啤酒的各种东西的前面，当情况开始恶化时，你会不顾一切地想要打开它；米下来了。

银行倾向于在一个已知的天气威胁到来之前为自动取款机充值，而提供现金返还的零售商有时可以减少他们每次购物允许的现金返还金额。当 2012 年飓风桑迪掠过华盛顿特区时，一家杂货店 [将其现金返还限额](https://www.washingtonpost.com/business/capitalbusiness/the-cash-before-the-storm-banks-refill-atms-start-disaster-relief-efforts-before-bad-weather-hits/2012/11/02/dd420fda-22b3-11e2-8448-81b1ce7d6978_story.html) 从 200 美元降至 50 美元。

那场风暴席卷纽约市后，断电地区的居民不得不步行去寻找可以使用的自动取款机 ，并且经常发现它们被耗尽。暴风雨过后，银行可能很难知道哪些自动取款机需要充值，哪些地区已经恢复供电。与此同时，装甲卡车可能很难到达遭受风暴破坏、需要补充燃料的社区。

当暴风雨来临的时候，你手头应该有多少现金？几百美元应该足够了。如果天气事件很严重，你认为你可能需要撤离，更多的现金会更好。如果可以的话，准备足够的现金来支付一个月的日常开支。如果你能拿到各种各样的钞票——一百张、一堆 20 元、一些 10 元和 5 元的——你就能更好地应对那些需要从应急基金中提款的情况。
