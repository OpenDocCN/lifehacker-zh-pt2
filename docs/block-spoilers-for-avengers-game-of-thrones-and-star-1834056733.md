# 用这些工具阻止复仇者联盟，权力的游戏和星球大战的剧透

> 原文：<https://lifehacker.com/block-spoilers-for-avengers-game-of-thrones-and-star-1834056733>

作为一个极客，这是艰难的一周，除非你喜欢看你喜欢多年的角色，如果不是几十年的话，咬一口大的。本周一定要带着你的 mjlnir 大小的面巾纸盒子去复仇者联盟 4：终局之战(尤其是如果你和我一样，把灭霸作为你的首选。虽然他不太可能 [获得蚁人](https://youtu.be/FLtnts6i2UI?t=109) 奖，但我有点怀疑每个人都喜欢的虚无主义者不会通过这部电影。)



哦，还有*权力的游戏*。耶。愿你和朋友或同事玩“死池”的几率永远对你有利；在周日的游戏中，你可能会在你的宾果卡上划掉很多方块。

当然，没有什么比 [在你有机会亲眼目睹发生了什么之前，偶然发现一部电影、一集或你最喜欢的角色的剧透](https://www.engadget.com/2019/04/16/avengers-endgame-leak-reddit-youtube-twitter/) 更快破坏这些大型特许经营的乐趣了。虽然防止这一切的最简单的方法是本周不要上网，但这对大多数人来说不切实际。然而，你有几个选项可以用来反击剧透。

### 让你痴迷的粉丝群体停下来

你每天看的那本*权力的游戏*子编辑？总在爆料你的新闻的庞大的*《星球大战》*脸书集团？你对所有事情的不和谐渠道*复仇者联盟*？当你即将迎来你所期待的东西的首次亮相时，考虑离开团队(或服务)。你可以随时重新加入。

你会知道什么时候是休息的时候——特别是如果你听到风声说你最喜欢的节目或电影已经有了媒体预览，如果所说的节目或电影已经向大众推出，或者如果你开始看到新闻报道说泄漏和剧透正在出现。避免它们的最好方法是不要挂在有人可能张贴它们的地方(在版主删除它们之前)。

### 将 Twitter 上的许多单词和短语静音

Twitter 也是最不适合抓开车过来捣乱的地方之一。然而，Twitter 有一个很好的系统，你可以在它毁掉你的电影之前用它来解决这个问题:静音。

拉起 [本页面](https://twitter.com/settings/muted_keywords)；载入你最喜欢的电影、电视节目、视频游戏或其他任何东西的维基百科条目；让你的手指舒展一下。那是因为你会想要输入任何名字、作者、地点、情节点或其他关键字，这些关键字*可能会出现在剧透中。你想变得多疯狂和彻底取决于你——如果你是《权力的游戏》的粉丝，我很抱歉，因为有很多角色和房子需要静音——但是你静音得越多，你就越不可能被你不想知道的细节所破坏。*

如果您想在观看节目或影片后参与一些热烈的讨论，您可以将静音持续时间设定为“24 小时”、“7 天”或“30 天”，而不是“永远”

如果你在 Twitter 上很受欢迎，或者经常受到骚扰，你可能还想为各种类型的人(例如，那些不关注你的人)设置静音通知。更好的是， [关闭](https://twitter.com/settings/safety) 接收来自任何人的剧透式直接消息的功能，并且不要让任何人在“韩去世”的照片中标记你。

如果你在某种社交平台上很出名——无论是 Twitter、脸书、PlayStation，还是其他什么——你可能会考虑关闭消息、通知、聊天或其他任何你能做的事情，以防止人们用你不想看到的剧透来打击你。外面的世界很残酷。

### 清除您的 YouTube 历史记录

虽然很容易在 YouTube 上避开“剧透:[节目名称]”视频，但更棘手的是，当你在观看老剧集中最喜欢的片段时，你会看到一个相关的视频，视频标题中有一个剧透。给你一个剧透:这样做的人都是混蛋。

我稍后会谈到更复杂的方法来净化你的 YouTube feed，但是一个简单快捷的方法是删除你观看过的所有视频的历史记录。这将有助于防止 YouTube 的算法向您提供您可能不想了解的“相关”视频。

要删除您的手表历史记录，请访问 [本网站](https://myactivity.google.com/myactivity) ，点击顶部的“按日期和产品过滤”选项。

取消选择除 YouTube 之外的所有内容，然后单击蓝色放大镜。一旦你的 YouTube 观看历史被填充，点击搜索栏右边的三点图标并选择“删除结果”在弹出的框中点击“删除”。

### 移除您的子编辑(暂时)

如果你是一个 Reddit 的大用户，你正在涉入一个剧透雷区(取决于你的 subreddits 的版主有多好)。然而，你不必为了避免剧透而大规模退订；你仍然可以保持这些社区的成员身份，并在感觉合适的时候回来参与话题讨论。相反，我建议从你的首页提要中删除剧透式的子编辑。

要做到这一点，你首先要为 [Chrome](https://chrome.google.com/webstore/detail/reddit-enhancement-suite/kbmfpngjjgdllneeigpgjifpgocmfgmb?hl=en-US) 或 [Firefox](https://addons.mozilla.org/en-US/firefox/addon/reddit-enhancement-suite/) 安装有用的 **Reddit 增强套件**扩展。在启用扩展的情况下拉起 Reddit，点击右上角的齿轮图标，点击“RES settings console”以加载 Reddit 增强套件的配置屏幕。点击左边栏的“子编辑”链接，然后点击“过滤编辑”

在这个配置屏幕上，你可以过滤掉 Reddit 中的特定单词——隐藏标题中有这些特定关键词的任何帖子——以帮助你管理剧透，就像你对 Twitter 所做的一样。您还可以隐藏子编辑本身，可以在整个 Reddit 站点上隐藏，也可以只在/r/all、/r/popular 和 domain 页面上隐藏。

### 使用浏览器扩展获得无干扰体验

如果你需要更多的帮助来创建一个尽可能无干扰的网络体验，有一些很棒的浏览器扩展值得考虑。 **BlockTube** ，用于 [Chrome](https://chrome.google.com/webstore/detail/blocktube/bbeaicapbccfllodepmimpkgecanonai?hl=en-US) 和 [Firefox](https://addons.mozilla.org/en-US/firefox/addon/blocktube/) ，让你隐藏评论，不让烦人的 YouTube 用户看到，或者更重要的是，通过关键词——非常适合阻止充满剧透的聊天，只要你不介意填写关键词列表。你也可以通过视频标题中出现的单词来屏蔽视频，这样看到“DARTH·VADER 是卢克的父亲”这样的相关视频就不会毁了你的一天。最后，该扩展可以屏蔽 YouTube 的整个趋势页面，以及该服务创建的任何自动生成的播放列表。

你也可以查看 Chrome 扩展 [**删除推荐 Youtube VK 脸书**](https://chrome.google.com/webstore/detail/remove-recommendations-yo/hfnghnjkcpgagjbiheidccjnneoipbmb?hl=en) **，**哪些做了刚才那件事——删除任何和所有推荐的视频，这样你就不会被你不想看的东西(包括标题)宠坏。对于火狐来说，试着检查一下 [**去掉 Youtube 的建议**](https://addons.mozilla.org/en-US/firefox/addon/remove-youtube-s-suggestions/) 。

由于浏览器扩展无法在你的手机浏览器(或应用程序)上运行，你可能想在你的智能手机上退出 YouTube 或者根本不用它——直到你看完你的电视节目或电影。

如果你很喜欢 Twitch，一定要使用[**BetterTTV**](https://nightdev.com/betterttv/)扩展来阻止聊天中的某些关键词，以防流氓也试图破坏那里的某些东西。你也可以通过 Twitch 的设置尝试 [屏蔽来自陌生人的低语](https://help.twitch.tv/s/article/how-to-manage-harassment-in-chat?language=en_US) ，但如果你潜伏在流中，你很可能不会成为消息破坏者。

为了更集中的努力，适用于更多的网站，而不是 YouTube，请检查一下 [Chrome](https://chrome.google.com/webstore/detail/spoiler-protection-20/eelacikjiplnmdingehjfdjcfegclmkg) 和 [Firefox](https://addons.mozilla.org/en-US/firefox/addon/spoiler-protection-2-0/) 的**剧透保护 2.0** 。

像以前一样，你将添加你最喜欢的极客系列的关键词，你不想看到在网络上被提及。资料片的美妙之处在于它是剧透的核弹。剧透保护 2.0 将保护你不受整段文字的影响，在某些情况下，甚至是整篇文章的影响。有点重手，但这是拯救自己远离秘密的最好方法。
