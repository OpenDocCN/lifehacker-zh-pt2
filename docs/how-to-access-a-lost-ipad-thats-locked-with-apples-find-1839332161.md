# 我如何使用苹果的“查找我的”锁定丢失的 iPad？

> 原文：<https://lifehacker.com/how-to-access-a-lost-ipad-thats-locked-with-apples-find-1839332161>

苹果的“查找我的”功能是找回丢失或被盗设备的有力工具。如果你已经购买或以其他合法方式获得了一个仍然绑定在另一个人账户上的二手设备，这也可能是一件令人头疼的事情。在本周的 [**Tech 911**](https://lifehacker.com/c/tech-911) 栏目中，我们要解决的是后者。



生活黑客的读者张克帆写道:

> *感谢“如何设置锁定的 Mac”这篇文章。我父亲得到了一块一年前发现的平板电脑。我们把它放在失物招领处，在认领期结束后，它被归还给了我们。当然,“查找我的手机”功能并没有被禁用，我父亲联系了苹果公司，希望他们能够利用这一功能将手机物归原主。他对苹果因“隐私”问题不联系其所有者感到不安。他觉得他们是在逼迫机主购买新设备，而不是联系他们有人找到了他们的设备。它现在就放在他家没人用。他有办法让它自己运转起来吗？*

让我们把坏消息说出来。如果你盯着一个 [激活锁](https://support.apple.com/en-us/HT201365) ，你就卡住了。苹果不会帮你删除这个，你自己也没有办法。你必须联系设备的原始所有者——这似乎是你无法做到的——让他们在 iPad 上输入他们的苹果 ID/密码或密码，或者从他们的 iCloud 帐户中 [擦除并移除](https://support.apple.com/en-us/HT201441)iPad。

简洁点说，激活锁=不好。你现在自豪地拥有了一个昂贵的镇纸，而不是平板电脑。

不，把你的 iPad 启动到 [恢复模式](https://support.apple.com/en-us/HT201263) 并试图通过连接到 Mac 来重置它是没有用的。当你试图设置你的 iPad 时，你仍然会看到激活锁定屏幕，因为这是与苹果的“查找我的”功能捆绑在一起的。

从积极的一面来看，这至少对那些拥有丢失和/或被盗设备的人来说是一种鼓励。激活“查找我的”后，攻击者只能使用 iPhone 的部分功能，仅此而已。我意识到，这并不能让你拿回你的设备，但希望它至少能对潜在的 iPhone 或 iPad 窃贼起到某种威慑作用。

至于苹果不愿意解锁你的设备，禁用激活锁，甚至不愿意告诉你设备前主人的联系方式，这都是可以预料的。虽然你只是试图获得一台显然已经丢失了一段时间——很可能被遗忘了——的设备，但苹果公司没有办法核实你的故事。

考虑另一种选择。假设你最近丢失了你的 iPhone，有人捡到了它，要么是无辜地从偷你的人那里买的，要么是他们发现并决定保留的东西(如果他们没有办法联系到你)。或者，更糟糕的是，假设有人从你手中偷走了你的 iPhone。

你想让这个小偷用一个简单的故事就能抢劫任何一家苹果商店，并摧毁你唯一的安全措施——找到我的——吗？我再次意识到，如今大多数 iPhone 窃贼可能只是在出售这些设备的零件(或者希望你没有打开“查找我的”)，但我确实认为，对于一些潜在的攻击者来说，如果他们知道他们的目标设备只不过是一块砖头，假设他们缺乏拆卸它的技术知识，这至少是一个小小的挫折。

同样，我也不希望一家公司仅仅因为有人发现(或偷走)了我的手机而泄露我的个人信息——我的电子邮件、电话号码或任何其他联系我的方式。虽然我意识到这意味着我关闭了一个潜在的恢复选项，但我感觉好多了，因为我知道我可以通过 [将我的设备标记为丢失](https://support.apple.com/en-us/HT210515) 来使用“查找我的”来提供信息，而不是担心苹果会给小偷更多关于我的信息。

在一个完美的世界中，你可以向苹果公司报告设备丢失或被盗，这可能是一种允许苹果公司在设备被带到苹果商店时向*提供一些*方法来联系你的方式。不过，这有点吹毛求疵；同样，您也可以通过“查找我的”报告设备丢失。苹果公司很可能不愿意尽可能多地参与进来，以免在技术上谁拥有一台丢失的设备的问题上陷入一些棘手的灰色地带——特别是如果第二个人购买了一台之前被报告为丢失的设备。呀。

说到这个，这就是为什么每次你购买二手 iPhone 或 iPad 时，确认“查找我的”是关闭的是非常重要的。苹果 [对这个](https://support.apple.com/en-us/HT201351) 再清楚不过了，理由很充分；如果您购买的设备激活锁仍处于打开状态，当您使用帐户凭据重置和设置它时，您将无法对其执行任何操作。

虽然这个答案没有给你提供你想要的结果，但我希望它至少解释了为什么你找到的 iPad 在这一点上可能是无用的。这对你来说很糟糕，但想想好的一面——你并没有丢失你花钱买的昂贵平板电脑。相反，你根本无法使用你找到的昂贵平板电脑。
