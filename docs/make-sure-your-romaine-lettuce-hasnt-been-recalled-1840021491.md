# 确保你的生菜没有被召回

> 原文：<https://lifehacker.com/make-sure-your-romaine-lettuce-hasnt-been-recalled-1840021491>

还记得去年 召回的那些长叶莴苣吗？这是再次发生。疾病控制中心建议消费者不要再吃任何长叶莴苣，特别是生长在加利福尼亚萨利纳斯的莴苣。

Watch

如果你的冰箱里有生菜，疾病预防控制中心建议扔掉它，如果它说它是在萨利纳斯生产的，或者如果标签没有特别注明生菜是在哪里包装的。警告也已经发出，要求零售商停止销售生菜，并要求餐馆停止供应生菜。

美国农业部还召回了大量含有 T2 T3 的特定产品。

据报道，来自 16 个州的至少 40 人因莴苣中的大肠杆菌而患病，其中 28 人已经住院。

大肠杆菌的症状包括痉挛、腹泻和呕吐。

此次召回包括所有形式的长叶莴苣，包括整棵莴苣、切碎的长叶莴苣，以及含有长叶莴苣的预制沙拉和沙拉混合物。

一句话:如果你的冰箱里有生菜，但不知道是在哪里包装的，扔掉它。你可能也想在外出时避免点生菜，以防你所在的餐馆没有听说召回事件或者不经意间给你上了一些受污染的生菜。
