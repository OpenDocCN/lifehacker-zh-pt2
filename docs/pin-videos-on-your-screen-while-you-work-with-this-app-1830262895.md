# 使用此应用程序时，将视频固定在屏幕上

> 原文：<https://lifehacker.com/pin-videos-on-your-screen-while-you-work-with-this-app-1830262895>

如果您没有双显示器设置或超宽显示器，您在 PC 上进行多任务处理的选择将会受到限制。在你工作的时候放一个 YouTube 视频会更棘手，因为你在主窗口中做的任何事情——比如说 Excel——都会位于你试图观看的内容之上。并排运行两个窗口看起来也不太好。

Watch

相反，试试看 Windows 和 Mac 的应用程序[**Pennywise**](https://github.com/kamranahmedse/pennywise)——是的，以那个吓人的小丑命名。有了它，你可以将任何网站悬浮在一个单独的窗口中，它将位于你正在做的任何事情之上。这将允许你在工作的时候观看 YouTube 视频(或者 Twitch，或者 ESPN，或者其他什么),在*魔兽世界*中钓鱼，或者在你的电脑上执行任何其他乏味的任务。

设置 Pennywise 很容易。安装它，你会得到一个巨大的、怪异的屏幕，上面有一张可怕的小丑脸和一个输入。

键入或粘贴一个 URL，它就会加载到 Pennywise 的窗口中——就这么简单。

当你想看别的东西时，只需在 Pennywise 中添加一个新的 URL。如果你点击右上角的齿轮图标，你还可以调整窗口的透明度——如果你想看到你的游戏多一点点，但不想完全失去你最喜欢的流。

如果你想疯狂一下——至少在 Windows 上，也就是我测试该应用的地方——你甚至可以同时运行多个 Pennywise 实例。假设你的显示器足够大，你可以把它变成一个虚拟的股票交易桌面。你知道我在说什么:
