这封短信已经在 [疯传](https://www.businessinsider.com/president-trumps-written-memo-has-become-a-meme-2019-11) ，因为，嗯，这很荒谬，但也因为川普独特的书写风格。这有点滑稽，没有微软遇上 Helvetica 遇上迪士尼的标志字体。

如果你想像特朗普一样写作——或者在假期发送一张美好而难忘的卡片——这里有一个小窍门:你现在可以使用 [本网站](https://www.finalwordfromthepres.com/) 用特朗普的手写风格写自己的便条。只要写下你的笔记，点击“下载海报”，你就会有自己的特朗普备忘录，就像特朗普签署他的备忘录一样。
