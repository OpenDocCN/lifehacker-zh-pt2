# 通过稀释煎饼粉来制作“骗子”薄饼

> 原文：<https://lifehacker.com/make-cheater-blini-by-thinning-out-pancake-mix-1842981687>

由于花式家庭烘焙的激增，仍然很难找到面粉、鸡蛋和酵母，但混合料仍然丰富，煎饼混合料就是其中之一。我——著名的懒女人——已经是加水煎饼粉的粉丝了，特别是自从我发现我可以用它来做不太正宗，但非常好用的薄饼。



取决于你在哪里，“blini”可以指用小麦或荞麦粉制作的又大又薄的酵母发酵煎饼，但西方人通常认为它们是小而美味的(也是)酵母发酵的圆盘，搭配鱼子酱。他们很有趣。

虽然有人可能会认为像鱼子酱这样的珍贵食材配得上珍贵的煎饼，但我认为这不是必须的。非常美味的鱼子酱运输盘可以通过稀释你通常的烙饼面糊来制作；除非你们当中有一个 blini 鉴赏家，否则没有人会看不出区别。(如果他们真的知道其中的区别，那就归咎于“这些绝望的时代。”)

你使用的额外水量取决于你的混合物，但只要加入额外的四分之一杯，直到你有一个薄面糊，当你把它倒入锅里时，它保持扁平和稀薄。你可能需要测试几个出来。吃掉那些垃圾。用你觉得好吃的油煎，或者你身边有的任何东西。黄油做的薄饼更杂色；在不粘锅里放一点蔬菜或其他中性油，可以做出更均匀的薄饼。做一大堆。

一旦你有了一堆薄饼，就从冰箱里找馅料。就像烤土豆条一样，blini 是清理冰箱的一个很好的中立工具，所以拿出一大堆东西。酸奶油、泡菜、炒鸡蛋、农家干酪、各种腌制的鱼、熟食、果冻和果酱都是不错的选择。如果这唤醒了你对鱼子酱的渴望，放纵一下，下次你去杂货店的时候给自己一点奢侈。人们在“这段时间”不会给自己足够的奢侈品，所以你很可能会发现鱼子酱还有存货。
