# 你仍然可以在亚马逊上订购任何你想要的东西

> 原文:[https://life hacker . com/you-can-still-order-what-you-want-on-Amazon-1842396337](https://lifehacker.com/you-can-still-order-whatever-you-want-on-amazon-1842396337)

如果你在当地杂货店或大卖场找不到卫生纸和洗手液等必需品，你可能会在互联网上找到慰藉。至少你还可以在网上订购家居用品，然后送货上门，对吗？我们可能无法在大群体中闲逛，但那些亚马逊送货车仍然在我们的街区到处跑。

但是最近关于亚马逊出货量有限的报道可能会让你担心你是否还能从这个电子商务巨头那里 [购买必需品](https://lifehacker.com/what-to-buy-if-youre-concerned-about-the-coronavirus-1842095740) (或者你选择的非必需品)。

亚马逊宣布暂停向其仓库运送一些产品。但是现在，对你能点什么或者什么时候能点没有什么影响。

以下是亚马逊本周在其博客 上就发货优先级所说的话:

> 随着新冠肺炎的发展，我们最近看到网上购物的人越来越多，这对我们服务客户的方式产生了影响。因此，在短期内，我们决定暂时优先考虑进入我们的履行中心的家庭必需品、医疗用品和其他高需求产品，以便我们可以更快地接收、补充库存和向客户运送这些产品。我们正与我们的销售合作伙伴昼夜不停地工作，以确保这些基本产品的可用性，并继续增加交付客户订单的能力。

根据 [格言](https://adage.com/article/digital/amazon-tells-suppliers-it-wont-accept-new-shipments-nonessential-items-until-april-5/2244826) 获得的一份备忘录，向亚马逊运送商品以便亚马逊将商品运送给客户的亚马逊卖家被要求将新库存推迟到 4 月初，除非它属于几个类别。清洁用品、医疗用品和食物是该规则的主要例外。

已经从卖家运送到亚马逊的非必需品将被接受并照常销售。运送自己产品的卖家目前没有面临任何限制。

如果你打算在亚马逊上买一只棒球手套或一台笔记本电脑，你可能会发现可供选择的选项很少。但是，亚马逊不太可能在抢购最受欢迎的商品的同时，耗尽非必需品。除了运送来自第三方卖家的商品，亚马逊还出售和运送大量自己的库存。

亚马逊正在仓库 [疯狂招聘](https://www.npr.org/2020/03/16/816704442/amazon-to-hire-100-000-workers-to-meet-surge-in-demand) ，试图容纳未来几周被困在家附近的人们涌入的订单。与此同时，该公司建议消费者尽可能选择更慢的运输方式:“选择不急选项使我们能够整合订单，减少一周内在邻近地区的停留，最重要的是，首先为有最重要需求的客户服务，”亚马逊的博客写道。
