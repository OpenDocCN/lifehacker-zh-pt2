# 如果你生活在空气污染中，你就更容易受到新冠肺炎的影响

> 原文:[https://gizmodo . com/if-you-live-with-air-pollution-you-been-more-vuln-1842366934](https://gizmodo.com/if-you-live-with-air-pollution-youre-already-more-vuln-1842366934)

新冠肺炎是一种由冠状病毒引起的疾病，正在全球传播，它的一个显著特点是，它会在一些人身上引发严重和致命的疾病，而另一些人只会轻微患病。到目前为止，我们知道年龄越大或已有健康状况的人出现严重症状的风险越大。不幸的是，对许多人来说，他们每天呼吸的污染空气可能已经损害了他们的肺部，使他们更容易受到新冠状病毒的攻击。

新冠肺炎经常发烧和干咳。这种病毒攻击呼吸系统，当患者患上急性呼吸窘迫综合征时，这种病毒就会致命，急性呼吸窘迫综合征会导致肺部充满液体。这种病毒已经对 60 岁以上的人，以及患有心血管疾病、癌症和慢性呼吸道疾病的人最致命。

猜猜上面所有疾病的主要驱动力是什么？空气污染。谁是最有可能暴露在美国糟糕空气质量下的？低收入家庭和有色人种社区。谁最有可能因空气质量下降而过早死亡？ [老年人](https://earther.gizmodo.com/reducing-air-pollution-saved-the-us-24-billion-on-elde-1841267976) 。今天，黑人死于哮喘的可能性是白人的三倍。对于黑人儿童来说，这种可能性增加了 10 倍。黑人也因心脏病而遭受最高死亡率。

这就是为什么新冠肺炎疫情是一个环境正义的问题，前环境保护局局长吉娜·麦卡锡周一在推特 T3 上也这么说。

“这场危机不仅仅是一个公共卫生问题。这直接关系到社会公平和环境正义，”麦卡锡写道。“这与我们争取清洁空气、清洁水、健康环境和健康社区的努力直接相关。#COVID19 正在影响我们所有人——我们的健康和生活方式，但低收入社区和有色人种社区可能面临更多风险。”

到目前为止，还没有研究调查冠状病毒发病率较高的社区的人口统计数据。也没有任何关于空气质量差会如何加剧病毒症状的研究。哥伦比亚医学中心的儿科助理教授 Stephanie Lovinsky-Desir 告诉《地球》杂志，具有讽刺意味的是，这种病毒起源于中国， [是世界上空气质量最差的](https://www.airvisual.com/world-most-polluted-cities) 之一。

Lovinsky-Desir 说:“病毒如此致命的原因之一可能与空气污染等环境暴露有关。”

然而，我们确实知道空气污染 [会影响](https://www.ncbi.nlm.nih.gov/pubmed/31995344) 呼吸免疫系统，因为有毒颗粒会进入肺部并使其发炎。糟糕的空气质量加上新冠肺炎可能会成为一个致命的组合，因为全球经济 [依赖于燃烧有毒的化石燃料](https://earther.gizmodo.com/the-coronavirus-outbreak-has-caused-the-biggest-global-1842123322) 已经给社区带来了痛苦。

最近由于新冠肺炎的原因，正常的日常活动被关闭，实际上导致整个 [中国](https://earther.gizmodo.com/chinas-air-pollution-rates-plummeted-after-coronavirus-1842010373) 和 [意大利](https://earther.gizmodo.com/satellites-show-italys-air-pollution-dissipating-as-cov-1842316669) 的空气污染急剧下降，这两个国家的 [病例数最高](https://gisanddata.maps.arcgis.com/apps/opsdashboard/index.html#/bda7594740fd40299423467b48e9ecf6) 。然而，在没有全球健康危机的情况下，汽车定期向我们周围的空气中排放毒素，而燃烧化石燃料的炼油厂和发电厂则主动向少数人的后院排放颗粒物和致癌物质。

“如果你暴露在污染中，那么风险就会增加。罗格斯大学转化医学和科学研究所副所长、肺病医生 Rey Panettieri 告诉《地球》杂志，如果这些地区的有色人种数量不成比例，那么推而广之，他们肯定会更容易生病。“弱势群体——我在更广泛的意义上使用这个词——肯定会不成比例地受到所有疾病的影响，由于病毒传播的快速性，我认为这些人将受到不成比例的影响，所以是的，我很担心。”

所有这些空气污染的结果，在全球范围内，每年有 700 万人死亡，根据世界卫生组织的说法。在美国，空气污染十年来首次呈上升趋势。( [谢](https://earther.gizmodo.com/trumps-budget-proposal-is-an-assault-on-clean-air-and-w-1841584969) [唐纳德](https://earther.gizmodo.com/trumps-tremendous-work-on-superfund-sites-includes-not-1840795436) [川普](https://earther.gizmodo.com/trump-administration-sues-california-for-daring-to-addr-1839289632) )。)从 1999 年到 2013 年，空气质量的改善减少了老年人过早死亡的数量， [每年节省 240 亿美元](https://earther.gizmodo.com/reducing-air-pollution-saved-the-us-24-billion-on-elde-1841267976) 但他们仍然是最危险的人群。空气污染如此致命，健康专家称之为疫情，指出化石燃料是全球死亡和绝望的主要来源。

现在，新冠肺炎很可能进一步提高这些社区的死亡风险。

“我预计低收入社区的感染率和死亡率会高得多，低收入有色人种社区的感染率和死亡率会更高，因为所有预先存在的条件——包括医疗和社会条件，”乔治·梅森大学气候变化、能源和环境健康平等副教授兼全国医疗协会医疗立法委员会主席马克·米切尔告诉《地球》杂志。

这些社会条件包括 [较高的贫困率](https://aspe.hhs.gov/report/overview-community-characteristics-areas-concentrated-poverty/racial-and-ethnic-minorities-are-overrepresented-concentrated-poverty-population-and-concentrated-poor-communities-metropolitan) 、医疗保健领域的不平等，以及带薪休假的不平等。也有生活方式的做法，如多代同堂的住房，祖父母、子女和孙辈可能都生活在同一个屋檐下。这是 [在移民家庭和有色人种中比较常见的](https://www.census.gov/content/dam/Census/library/working-papers/2013/acs/lofquist-01.pdf) 。 [常规使用公交](https://www.pewresearch.org/fact-tank/2016/04/07/who-relies-on-public-transit-in-the-u-s/#:~:text=Americans%20who%20are%20lower%2Dincome,Pew%20Research%20Center%20data%20show.&text=Among%20urban%20residents%2C%2034%25%20of,with%20only%2014%25%20of%20whites.) 也是如此。

这就是为什么一些因工业活动而面临更高空气污染率的城市的社区组织者正在尽一切努力与当地人联系，并让他们了解病毒的情况。在加利福尼亚州的里士满，雪佛龙里士满炼油厂的所在地，近年来因释放有毒气体而受到攻击的，亚太环境网络(APEN)一直试图给成员提供如何保护自己免受病毒感染的重要信息。

该组织的许多成员都是老年人，许多人患有哮喘或癌症。到目前为止，该组织还不知道任何确诊的新冠肺炎病例。因为 APEN 与亚裔美国人一起工作，所以它也特别关注种族偏见和仇恨犯罪的增加，该网络在里士满的组织主管梅根·扎潘塔告诉《地球》杂志。最终，这些社区在新冠肺炎时代需要的资源——稳定的住房、医疗服务、稳定的工作——是他们永远需要的。

扎潘塔说:“我们想真正从各种角度考虑恢复能力，而不仅仅是在发生巨大灾难或疫情时的恢复能力。”。“或者当出现经济衰退时的弹性，所有这些都是相关的，或者当出现住房危机时。”

美国政府对新冠肺炎的传播只能控制这么多。然而，政府的反应——授权，改善 [公共交通](http://www.mta.info/press-release/mta-headquarters/mta-doubles-station-sanitizing-efforts-across-new-york-city-transit) 的清洁，以及向市场注入 [超过一万亿美元](https://www.newyorkfed.org/markets/opolicy/operating_policy_200312a?mod=article_inline)——显示了什么是可能的，不管疫情是否肆虐。对冠状病毒的反应让我们了解到美国政府在进入紧急状态时愿意做什么。遗憾的是，我们的领导人不得不等到灾难来实施这些政策。

米切尔说:“我们知道，我们现在有技术关闭这些[污染]设施，转而使用更清洁的可再生能源，并停止这种主要影响低收入社区和有色人种社区的暴露。”。“它将在改善健康方面产生立竿见影的效果，并有助于降低对新冠肺炎等其他疾病的易感性。”
