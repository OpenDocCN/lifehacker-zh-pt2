# 出去 5 分钟，改善坏心情

> 原文：<https://lifehacker.com/improve-a-bad-mood-by-going-outside-for-5-minutes-1831927148>

治疗坏心情的方法可能就像出门一样简单。

里贾纳大学的研究人员最近发现，在大自然中呆上五分钟就足以帮助人们改变情绪。它最近在积极心理学杂志上发表了它的发现。



总的来说，研究人员发现，离开一步，休息几分钟有助于让消极的情况更容易处理。最大的好处；然而，当你花了几分钟在外面的时候来。

研究人员进行了两项独立的测试，参与者被要求在不同的时间里坐在房间里或户外，所有人都没有电子设备让他们有事可做。

虽然在这两种情况下，负面情绪在短短几分钟后就会降低，但置身于大自然中往往会激发积极的情绪。

“有两个重要的带回家；这项研究的作者之一 Katherine D. Arbuthnott 告诉[*【psy post*](https://www.psypost.org/2019/01/spending-just-5-minutes-in-contact-with-nature-boosts-your-mood-study-finds-52948)】说:“这些天来，我向我所有的学生强调的第一件事是——当你需要情绪激励时，最快、最简单的方法就是花几分钟时间接触大自然。她从这项研究中得出的第二个结论是，维护户外空间对于公众整体的情绪健康和幸福非常重要。

这项研究并非惊天动地。我们都知道我们需要休息，这是显而易见的，在户外看一些漂亮的东西，呼吸一些新鲜空气会比坐在办公桌前更好。也就是说，当我们心情不好的时候，有多少人真的会出去走走呢？没错。

总的来说，当你遇到麻烦时，休息一下总是个好主意。获得一点空间，即使只有五分钟，通常可以让你以全新的视角和全新的精力回到问题上来解决问题。

下次不要把休息时间花在冲浪脸书上，考虑一下出去走走。你回来的时候可能会开心很多。
