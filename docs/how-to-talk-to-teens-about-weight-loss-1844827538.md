# how to talk to teens about weight loss 1844827538

> 原文：<https://lifehacker.com/how-to-talk-to-teens-about-weight-loss-1844827538>

你可能听说过成年人抱怨“quaran-ten”甚至“quarant 15”——这是一种诙谐的方式，承认我们在疫情期间与外界隔绝时体重增加了。我从我的朋友和家人那里听到了类似的情绪，并在我的社交媒体订阅中看到了它们。但是，不仅仅是成年人会注意到他们身体在这段时间里的变化——青少年现在可能也在经历体重增加和一些相关的自我意识。

Reddit 用户 [u/magneticsouth](https://www.reddit.com/r/loseit/comments/hwcsj3/recent_influx_of_teens_how_we_as_a_community_can) 注意到青少年在“ [减肥](https://www.reddit.com/r/loseit) ”版块上发帖求助减肥的人数有所上升:



> 因为我们这里的大多数人都是成年人，我们知道青少年需要更多的热量，因为他们仍在成长，比具有类似统计数据的成年人需要更多的热量。我想用这篇文章作为我们社区的一个起点，来促进和鼓励这些青少年健康和可持续的习惯，而不是关闭他们，否决他们或建议只适合成年人的赤字。

有趣的是，临床心理学家丽贝卡·卡茨 博士说，她和她的同事们还发现，越来越多的回头客和父母担心他们家里有人现在过于关注身体形象。这可能是因为实际的体重增加，或者是因为在疫情期间我们的社会交往方式发生了变化。

专门治疗饮食和体重紊乱的 Katz 说:“我认为人们花了很多时间减少实际的社会接触，更多地关注社交媒体和互联网上的图像。”"即使是你认识的人，这些图片也是经过精心策划的."

换句话说，在前疫情时代，青少年可能已经习惯于看到所有这些自拍都是在合适的角度和最合适的灯光下拍摄的，但第二天去学校，看到所有的同学都在野外，并不总是看起来像最上镜的自己。现在，这种现实检查对许多青少年来说不是一个选项。



### 让我们来谈谈增长

在我们讨论当你的孩子对自己的体重感到不安时，你应该何时担心或如何支持他们之前，让我们先来谈谈孩子的成长实际上应该是什么样的——这与身体质量指数(身体质量指数)计算器根据他们的身高和体重给出的数字无关。

还记得当我们的孩子还是婴儿的时候，他们在生命的第一年以令人难以置信的速度成长，而我们为每一卷新的婴儿肥欢呼雀跃吗？卡茨说，随着青春期的临近，类似的事情也会发生。

“在青春期发育高峰期，孩子可以长到 6-8 英寸或 40 磅，”她说。“起初，这对父母和孩子来说都有点令人担忧。”



比他们成长多少或身体质量指数计算器所说的更不重要的是(卡茨强烈建议完全抛弃他们)，他们沿着自己的个人成长曲线相当一致地成长。一个从蹒跚学步开始就处于第 95 百分位的孩子很可能会继续比他们的同龄人在身高和体重上增长得更快——因为这是他们身体自然生长的方式。卡茨将成长中的孩子的身体比作从地基开始建造房子。

“如果我们的孩子没有在身体上收集足够的物质，他们就无法成长，”她说。

### 注重全面的方法

不管你的孩子是否对他们的体重表示担忧，树立一个全面的饮食方式是很重要的；卡茨说，她甚至更倾向于不强调“健康饮食”，而支持“所有食物都适合”的心态。

“如果你吃了薯片，不会有什么可怕的后果，”她说。"每一种食物在我们的生活中都有一席之地。"



这意味着你可以模拟不同类型食物的均衡摄入，谈论食物作为我们身体所有活动的燃料，只要有可能，全家一起吃饭，这样父母就可以模拟这些习惯。在你的家里，不要谈论任何关于“卡路里”的话题。

### 让你的身体动起来

当然，有些人锻炼是为了减肥。但是定期锻炼也有利于我们整体的身体健康和心理健康——这就是为什么我们应该鼓励和支持我们的孩子多走动的愿望和需要。

“父母可以说，‘我们必须走出家门，’”卡茨说。“他们可以通过坦率地说‘我感觉有点暴躁’来鼓励这一点；我们需要离开这里。"



作为一个家庭一起四处活动——徒步旅行、骑自行车、在附近散步或者甚至进行一场史诗般的打水仗——也给你一个机会通过调整自己的速度、补水和休息来进行安全和适度的锻炼。

### 如果青少年带着担忧来找你

每时每刻都做这些事情是件好事；但是，如果你的孩子仍然来找你，说他们对自己的身体感到难为情，或者想减肥，怎么办？卡茨说，首先要以同样的方式回应孩子带给你的任何担忧——感同身受。

“你会同情他们，所以花些时间去同情他们，”她说。然后对以上所有内容重新做出承诺——家庭用餐时间，谈论食物作为我们身体所有重要活动的燃料，模拟走动对我们整体健康的重要性，并确保我们的孩子可以获得全面的膳食和零食。我们能做的最好的事情就是模型，模型，模型。

“最强有力的干预是家庭因素，”卡茨说。



最后，如果他们只是不相信你的话，安排一个时间让他们和一个值得信任的儿科医生谈谈可能会帮助他们获得他们需要的观点和客观性。

### 你应该在什么时候寻求外界的帮助？

如果你怀疑有任何形式的自伤发生，或者你有其他安全顾虑，你应该立即寻求帮助。饮食紊乱或身体形象问题的其他迹象包括不吃饭或不吃零食或拒绝吃饭。(卡茨说，鉴于父母和青少年通常不一起吃午饭，午餐往往是第一顿饭，因为它最不明显。)

身体活动增加加上焦虑也可能是一个危险信号。卡茨说，像“我必须出去跑步”或“我感觉不舒服，因为我今天没有跑步”这样的评论可能表明他们对体育活动的焦虑程度有所增加。

在这些情况下，良好的第一步是拜访他们的儿科医生，他们可以进行体重检查，并将青少年现在的体重与他们过去的体重增加趋势进行比较。医生还可以做一些实验室工作，记录一些关键的统计数据，如心率，以确定基线并检查他们的整体健康状况。



从那里，你的儿科医生可能会把你介绍给心理学家或社会工作者。如果你探索这个选项，Katz 警告说，你联系的专业人士应该接受过治疗饮食和体重紊乱的专门训练；她说，一般的治疗师在这个特定领域没有足够的知识来有效地提供帮助。

国家饮食失调协会 也可以是一个很好的起点来获得更多信息，卡茨建议年幼孩子的父母查看 [埃林·萨特关于孩子和体重的资源和书籍](https://www.ellynsatterinstitute.org/how-to-feed/child-weight-issues) ，包括 [*你孩子的体重:帮助而不伤害*](https://www.ellynsatterinstitute.org/product/your-child-s-weight) *。*
