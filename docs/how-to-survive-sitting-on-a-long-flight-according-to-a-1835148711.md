# 根据物理治疗师的说法，如何在长途飞行中幸存下来

> 原文:[https://life hacker . com/how-to-survive-to-sitting-on-a-to-a-1835148711](https://lifehacker.com/how-to-survive-sitting-on-a-long-flight-according-to-a-1835148711)

我从来不能在飞机上舒服地坐着；多年来我一直与腰痛作斗争。众所周知，在一次坐在中间座位的长途飞行后，我瘫倒在机场的地板上，就像一个刚学会走路的婴儿。

Watch

如果你也有类似的问题，这很正常——即使是最灵巧的旅行者，坐在一个除了枕头和扶手之外没有什么资源的狭小空间里也会感到不舒服。然而，你可以做一些事情来避免长途飞行后你可能经历的背部、腿部或颈部常见的身体疼痛。

我们采访了布莱克·德克森(Blake Dircksen)，他是纽约的一名物理治疗医生，在 [进行预约治疗](http://www.bespoketreatments.com/) 和 [睡前训练](http://www.nightcaptraining.com/) 的创始人，他谈到了拉伸和压缩袜子的重要性，以及为什么把一个袋子放在头顶的行李箱中对一些旅行者(即像我这样的下背痛患者)来说是一种痛苦的经历。

### 如果你能站起来，尽可能多地伸展身体

是什么让连续几个小时坐在狭窄的座位上如此不舒服？如果这还不明显，那就是你的身体不能动了。

“你的身体渴望运动，”德克森说。“当你长时间处于一个位置时，这些组织会被压缩，这些组织会缺氧...没有氧气，组织进入无氧状态，你开始积累乳酸——这是你坐得太久时的隐隐作痛感。”

德克森建议，只要有可能，就移动、伸展、起身——当然，是在通过过道的饮料车之间。

“为了避免一些疼痛，我喜欢尽可能多地运动，也许是一些 [小腿抬高](https://www.gq.com/story/ask-a-trainer-calf-raises) ，脚踝泵，扭转背部，”他说。“我不想说每个人都可以毫无保留地这样做，但一般来说，如果你处于无负荷状态，你应该能够弯曲、弯曲或旋转脊柱。这是让液体进入椎间盘并让血液流动的好方法。运动和各种各样的运动是缓解久坐疼痛的最好方法。”

如果可以的话，德克森推荐了一种针对腰痛患者的运动。“这很简单，只要把你的腿张开一点点，慢慢地向下移动，试着抓住你的鞋子或者把手放在地板上。保持这个姿势 30-45 秒，通过腹部呼吸，给背部肌肉一个放松的机会。对大多数人来说是安全的。”

出于这个原因，下次预订航班时，你可能会想选择靠走廊的座位，尤其是如果你想在不打扰邻座的情况下起身的话。

此外，如果你打算拿起你的包或把它放在头顶的箱子里，德克森把这个动作比作，一种你可能在健身房看到的常见电梯；为了避免背部过度伸展和对背部施加压力，用膝盖抬起并支撑住你的核心，这样你就不会感到任何疼痛。

### 如果你被困在座位上，使用一些方便的工具

如果你的同伴睡着了，而你不想爬过他们，德克森推荐了一些简单的工具来避免飞行疼痛。除了在座位上活动和伸展，你应该在脖子和下背部后面放一个枕头或毛衣，这样你的身体会处于一个更中性的位置(或者购买一个颈枕。比起记忆泡沫枕头，我更喜欢羽毛枕头，因为我发现记忆泡沫枕头太硬太不舒服，但这只是我的看法。尽可能向后倾斜你的座位也能帮助你的背部。

尽可能带上一双压缩袜，尤其是如果你容易出现血栓的话。“动脉有这些小肌肉帮助将血液泵送到全身。当血液转移到静脉系统时，静脉没有那些肌肉，它们依靠我们自己身体的运动和我们更大的肌肉将血液泵回心脏并再次再循环。当你坐在那里，停滞在飞机上，你缺乏运动来循环血液。这些袜子所做的只是增加压力，减少腿部的血液淤积。”

如果你身上有任何 [结](https://thepointsguy.com/news/how-to-fly-with-a-bad-back/) ，你可能会发现长曲棍球很有帮助；你可以在结上和结周围施加压力来减轻一些压力。

有什么办法可以完全避免中间座位的不适感？不太可能。但是，如果这种想法让你害怕，而你又愿意并且有能力负担得起，你可以选择在大多数航班上购买高级经济舱或商务舱座位，这可能会提供更多的腿部空间和倾斜空间。
