# 用 Gmail 过滤器创建一个联合电子邮件地址

> 原文：<https://lifehacker.com/make-a-joint-email-address-with-a-gmail-filter-1836702340>

当你有一个伴侣时，你不可避免地要注册会员资格和账户，这对你们两个都有影响:双影院订阅，或者涵盖你们两个的健康保险。令人恼火的是，许多会员资格只允许你用一个电子邮件地址注册。这里有一个 Gmail 小技巧，你可以设置后就忘记了，而不是设置第三个电子邮件帐户，或者为每次注册制定一个新的转发规则。

因此，伯特可以创建一个名为 bert+ernie@gmail.com 的地址，并告诉 Gmail 将发送到那里的所有电子邮件转发给厄尼(无论厄尼是否在 Gmail 上)。

要设置联名账户，你必须在 Gmail 中设置一个转发地址，然后创建一个过滤器。但是一旦设置好了，就不用再做什么工作了。你只需要用那个特殊的地址来注册联名账户。

### 设置转发地址

首先，你必须指定你的合作伙伴的地址作为转发地址。(这是为了防止垃圾邮件。)

1.  进入*设置*(点击右上方的齿轮图标)，打开*转发和 POP/IMAP* 选项卡。
2.  在*转发*部分，点击*添加转发地址*。
3.  输入您的合作伙伴的电子邮件地址，然后点击*确定*。
4.  Gmail 将向您的合作伙伴发送一封验证电子邮件。从他们那里获取代码，并将其粘贴到您的 Gmail 设置中。点击*验证*。

### 创建您的联合电子邮件地址过滤器

现在你可以告诉 Gmail 总是把某些邮件转发给你的伴侣。

1.  进入 Gmail 的设置(通过右上角的齿轮图标)。
2.  点击*过滤器和阻止地址*选项卡。
3.  在底部，点击*创建新的过滤器。*
4.  在新的下拉窗口中，在*到*部分，填写您的联名电子邮件名称，例如 bert+ernie@gmail.com。点击*创建过滤器*。
5.  在新的下拉窗口中，点击*转发至:*并从下拉列表中选择您的合作伙伴的电子邮件地址。点击*创建过滤器。*

### 用你的特殊地址注册联名账户

现在，当伯特报名参加情侣瑜伽时，他就可以进入 bert+ernie@gmail.com 了。厄尼也可以。无论哪种方式，电子邮件都会发送给 Bert，然后 Gmail 会自动转发给 Ernie。

请注意，Bert 的标签与 Ernie 的电子邮件地址没有任何关系。可能是 bert+weirdo@gmail.com。厄尼不一定要有 Gmail 才能做到这一点。这很好，因为厄尼仍然使用美国在线。
