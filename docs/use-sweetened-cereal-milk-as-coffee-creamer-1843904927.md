# 用加糖的谷物牛奶作为咖啡奶精

> 原文：<https://lifehacker.com/use-sweetened-cereal-milk-as-coffee-creamer-1843904927>

在我孩子的学校因疫情冠状病毒而关闭两个月后，我毫不羞愧地说，在家管理工作，同时在家教育一个 3 岁和 8 岁的孩子的挑战已经对我们的育儿惯例造成了严重破坏。然而，我有点惭愧，这对于我的后代摄入幸运符意味着什么。

在以前的时代，加糖的谷物即使没有在我们家被完全禁止，也绝对是一种主食。我们的孩子凑合着吃快速高蛋白早餐:一碗燕麦片。涂有花生酱的面包片。煮熟的鸡蛋。每次经过麦片通道时，他们都乞求并恳求给我们提供超甜的早餐，但我们总是立场坚定——直到我妻子在一次提前关闭的杂货店送货恐慌中抢走了四盒 2 美元的棉花糖麦片，一切都改变了。

事实证明，我的孩子们发现不健康的谷类食品就像我知道的那样令人满意——毕竟，作为一个 80 年代的孩子，我成长在一个时代，在这个时代，每一个新的玩具时尚或电视节目都伴随着一种可能几乎不可食用但绝对含糖的谷类食品，我成功地说服我的父母把它们都买下来(或其通用仿制品)。我们现在已经到了这样一个地步，我的儿子，他曾经更喜欢普通的燕麦片而不是加糖的东西，如果他的碗里没有足够的干棉花糖形状，他会经常扑倒在地上以示无声的抗议。然而，我们宽松的教育也有好的一面:我发现了一种喝咖啡的新方法。它发生在另一个令人绝望的停工时刻:无法确保杂货店的送货时间，我们的奶油和白糖都用完了，牛奶也少得可怜。有时我会喝黑咖啡，但这杯茶也是...可以说，不是我喜欢的品牌。一天早上，我正在想办法让它变得更可口，我看着一个儿童早餐碗里剩下的粉红色牛奶，心想...为什么不呢？

事实证明，加糖的谷物牛奶(在这种情况下，是 2%的牛奶和一碗棉花糖混合物的混合物)是一种完全有用的替代品，可以替代你可能真的希望在咖啡中加入的东西。这种酿造照顾到了挥之不去的谷物余味，虽然我也想象你可以从一碗可可泡芙的碎屑中得到一点摩卡咖啡。先用勺子(或筛子)清理掉漂浮的麦片，然后将麦片牛奶加入杯子中品尝。

我不是在暗示这是你每天都要做的事情，还有很多 [其他的面霜替代品你可能想先试试](https://lifehacker.com/these-emergency-coffee-creamers-might-be-hiding-in-your-1842649267) 。(我是甜炼乳的忠实粉丝。)但是我们生活在一个奇怪的时代。
