# 你在任何谈话中可能犯的最大错误

> 原文:[https://life hacker . com/the-maximum-mission-you-can-make-in-any-conversation-1839864717](https://lifehacker.com/the-biggest-mistake-you-can-make-in-any-conversation-1839864717)

你有没有尝试过解决朋友的问题，而他们真正想要的是发泄和获得一些同情？你有没有寻求建议，只得到同情的时候？当你的伴侣还在不断提出新的指控和抱怨时，你有没有尝试去解决一场争论？在所有这些情况下，问题是你们还没有就对话的*框架*达成一致。

这是一个如此广泛的问题，以至于分析它需要一些非常抽象的语言。LessWrong 短文 [【注意框架差异】](https://www.lesswrong.com/posts/f886riNJcArmpFahm/noticing-frame-differences) 需要一些时间来适应，可能需要几篇阅读才能理解。但是一旦点击，你将拥有一个有用的 [心智模型](https://lifehacker.com/mental-models-solve-problems-by-approaching-them-from-1682835620) ，它将在一次没有成效或令人沮丧的谈话中帮助你。

### 框架如何工作

LessWrong 作家 Raemon 说，框架是一种“观看、思考和/或交流”的方式有数不清的框架，但我们在交流中使用的一些主要框架是:

*   解决问题
*   分享感受
*   坚持权力
*   进行交易
*   只是射击狗屎

每个人，在每次互动中，都在使用一个框架。(即使框架是“填补沉默，直到电梯到达你的站。”)有时，互动中的每个人都使用相同的框架:

*   你和一个店员都在做交易，都在说着客套话。
*   你和你的同事在抱怨管理。
*   你在看一出戏，所以你接受了自己作为沉默观众的角色。

仅仅因为你们使用同一个框架并不意味着互动是愉快的。有时你们都使用相同的负面框架:

*   你和你的另一半在一场大喊大叫的比赛中倾诉所有的不满。
*   你在和你的工作对手恶语相向。
*   你和你的飞机座位搭档在争夺扶手。

你明白对方想做什么，你只是想更努力地去做。

### 帧如何变得不匹配

当对话中的人有不同的框架时，他们都更难实现他们的目标——无论目标是“谈判一项条约”还是“在派对上玩得开心”例如:

*   您希望在预算有限的情况下为特定项目集思广益，但您的同事希望为新项目提出雄心勃勃的想法。
*   你想八卦你真正喜欢的朋友，但你的伴侣想解决她认为你们友谊中的深层缺陷。
*   你想赶上一个朋友，但他想证明他比你更有成就。
*   你想表达一个想法，但是一个互联网评论者想挑剔你的语法。
*   你想解释你的总统立法议程，但你的对手想获得便宜的分数。
*   你想安安静静地坐在剧场里，但喜剧表演是有观众参与的，现在演员们想把你拽上台。
*   作为一个女人，你想做任何事情，任何事情，但是一个男人想泡你。

当双方有不同的框架时，双方将很难达到他们的对话目标——尽管一方可能比另一方更成功。除非每个人都进入相同的框架，否则对话不会朝着有成效的方向发展。

Raemon 列举了更多的例子，从明显的帧差异到微妙的差异——甚至是上面提到的帧内的差异。 [全部读完](https://www.lesswrong.com/posts/f886riNJcArmpFahm/noticing-frame-differences) 会帮助你更好的理解和认识框架差异。

### 如何解决帧差

要进入同一个框架，首先你必须注意到不匹配。我和雷蒙都没有快速解决这个问题的方法。但是当你或你的对话伙伴看起来很沮丧或像是在浪费时间时，要注意了。当你进行了一次看似富有成效的谈话，但那次谈话没有任何结果时，请集中注意力。关注那些越来越没有成就感的关系。

你还需要认识到，是否所有的对话伙伴都是抱着善意来参加这次对话的。理想情况下，每个人都想在同一个框架中。你们通常在同一个“团队”中，并且有一些共同的目标。

在这些情况下，当框架不匹配时，您可以接受并采用他们的框架，要求他们采用您的框架，或者同意同时或依次在两个框架中工作。(当你在工作会议上计划一个项目，但也闲聊几句时，你是在同时分享两个画面。)

这也适用于竞争框架:同意在没有黑客的情况下玩堡垒之夜，或者在镇议会会议上遵守罗伯特的秩序规则。这尤其有助于找到一套你们都没有发明的外部规则。如果框架没有对齐，你可以向当局申诉。

但有时你的伴侣并不在乎你是否符合他们的身材。整个对话是一个你碰巧在场的单独练习。(比如说，当有人只是想对你发泄愤怒，或者不恰当地对你动手动脚时，这是很常见的。)有时候你会意识到*你才是*那个不在乎搭配镜框的人。

在这些不守信用的情况下，明智的做法通常是脱离、重组，并考虑在更明确定义的条款下重新参与。从长远来看，你应该尽可能地避免不良的框架错配——当你无法避免时，每次都要指出来。当恶意框架被明确承认时，它们的效力会降低。

当你和你的朋友、同事、老板、家人和敌人交谈时，审视框架是一种有用的方式，可以让你从互动中退一步，评估你正在完成的事情。弄清楚为什么你会觉得自己在用头撞墙。因为没有人是一堵墙。你们只是两个用头撞在一起的人。以我的经验，这很少是理想的互动。
