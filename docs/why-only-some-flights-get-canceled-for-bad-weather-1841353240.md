# 为什么只有一些航班因为天气不好而被取消

> 原文:[https://life hacker . com/why-only-some-flights-get-cancel-for-bad-weather-1841353240](https://lifehacker.com/why-only-some-flights-get-canceled-for-bad-weather-1841353240)

当天气离世界末日还差一场风暴时，一旦你意识到它可能会影响你的即时旅行计划，你的喉咙里可能会形成一个肿块。不能在预定的时间登机，你只能想象一次又一次的延误，以及在机场的星巴克或安妮阿姨的椒盐卷饼店等待更长的时间。

有没有想过为什么在天气糟糕的时候，所有的航班都没有被取消？事实证明，这是航空公司精心策划的一场赌博。在为写的一篇故事中，作家芭芭拉·皮特森详细解释了为什么只有一些航班因为糟糕的天气而延误或取消——以及为什么你可能会乘坐其中一个航班。

虽然航空安全是机场最关心的问题(理应如此)，但航空公司也会根据飞机和机组人员的可用性以及维护需求来决定何时取消或延迟航班。听起来足够简单；恶劣的天气可能会延误机组人员按时到达预定的航班，或者飞机可能需要在起飞前除冰，这两种情况都可能导致航班延误。一种算法可能会处理大部分的决策过程，但是航空公司人员也有责任做出偶尔的艰难决定。

从这里开始，航空公司可能会考虑两件事:对消费者的影响，以及他们的底线。例如，一家航空公司可能会推迟一个航班，而不是直接取消一个航班，最终是为了他们自己的利益(翻译:他们想尽可能地保留你的钱)。

一位旅游专家向 Afar 解释说:“当它深入到微观层面时，诸如航班上有多少‘高价值’乘客，或者是否有一个大型团队前往邮轮或重要活动等考虑因素，可能会决定谁的计划将被打乱。”当然也有例外；两周前，芝加哥奥黑尔机场因降雪发布了地面停飞令，这意味着 689 次航班被取消，航空公司对此事可能没有什么发言权。

航空公司人员多久介入一次？好吧，我们不知道——但是如果你的航班被取消了，而另一个飞往同一目的地的航班没有取消，你就有理由相信算法或航空公司本身对这个决定负责。

如果你知道你的航班可能会延误或取消，可能是因为你窗外的暴风雪，选择在航空公司的网站上接收你的航班状态通知，并下载你的航空公司的应用程序，作为预防措施。如果你不确定它的状态，打电话给你的航空公司也无妨。如果你认为取消是不可避免的，现在看看你的替代选择。一旦取消通知宣布，你可以联系你的航空公司，并要求这些替代安排。
