# 你应该马上更新你的 Sprint 密码

> 原文：<https://lifehacker.com/you-should-update-your-sprint-password-right-now-1836453274>

Sprint 客户，立即更改您的账户密码——尤其是如果您拥有一部三星设备的话。



正如 ZDNet 最初报道的，Sprint 告知其客户，6 月 22 日发生了重大安全漏洞。黑客利用三星宣传网站上的一个尚未定义的漏洞来获取 Sprint 的客户信息。Sprint 尚未透露有多少账户受到牵连，或者这些信息可以被访问多长时间，所以不清楚这是多大的违规行为。我们*所知道的是，根据 Sprint 发送给一些用户的回应信息，泄露的信息包括“电话号码、设备类型、设备 ID、每月重复费用、用户 ID、账号、账户创建日期、升级资格、姓名、账单地址和附加服务”。*

在信的后面，Sprint 向客户保证“没有获得其他可能造成欺诈或身份盗窃风险的信息”，但您必须原谅我们没有分享 Sprint 的信心。说没有“实质性风险”被泄露固然很好，但泄露中包含的大量信息可能会被黑客利用来接管你的设备和账户 。

Sprint 擅自更改了可能已经受损的账户的 pin 码,但在这里我们宁愿谨慎行事。这是花点时间更新你的 Sprint 账户密码和/或 PIN 码的完美借口，即使你没有收到 Sprint 关于你的信息被泄露的通知。您可以在登录 [Sprint 的网站后，在账户设置中进行设置。](https://www.sprint.com/) 如果你需要新密码的帮助， [点击这里查看我们的指南](https://lifehacker.com/how-to-create-secure-passwords-that-arent-impossible-to-1825048324) 。
