# 如果我们混合一个超越汉堡...用牛肉吗？

> 原文：<https://thetakeout.com/what-if-we-blended-a-beyond-burger-with-beef-1836617136>

本周，*外卖*的工作人员受到了 Otto42.5 的 [评论的启发，这是一个关于](https://thetakeout.com/1836494689?) [吃肉/不可能的汉堡](https://thetakeout.com/who-eats-fast-food-meatless-vegan-burgers-data-1836490878) (不是纯素食者)以外的东西的故事。T 帽子评论:

这在许多层面上激起了我们的兴趣。但是让我们倒着来应对这个挑战:我们将首先——姑且称之为超越汉堡一半——并解释它是否有效或者是否有必要。

按照 Otto42.5 的建议，我们将 Beyond 汉堡肉饼和 85%瘦肉碎牛肉以 50:50 的比例混合。我们加入少量盐和黑胡椒调味，然后将馅饼松散地放在一起。

除了肉类，最大的打击是弥漫在你厨房里的豌豆蛋白的味道。我们认为把假肉的数量减半，可以减轻香味。这种情况没有发生。那种明确无误的气味仍然充满了力量。

在我们的铸铁煎锅上烹饪这些馅饼(用一小块黄油)，很难判断正确的熟度。尽管我们尽了最大努力，我们还是无法让牛肉呈现完美的中等粉红色。尽管做得相当好，牛肉中的脂肪仍然给汉堡增加了很多多汁性。最后，我们加了一片烟熏干酪，放在一个奶油面包上，没有额外的装饰，以更好地品尝肉饼。

怎么样？味道很好，但不令人满意。人们意识到它并不完全是 T2 牛肉汉堡，因为它尝起来无疑是超越肉类的汉堡，但有 80%的强度。但它也不是一种植物性汉堡，因为吃低胆固醇食物的想法随着添加的肉而消失了。拥有这两者，两者的优势就会相互抵消。

那么问题来了，这个假肉和真肉混合的汉堡到底是给谁吃的。在我的超市里，一包两个的 Beyond Burger 售价 8 美元，因此将它与牛肉混合以改善味道在经济上没有意义(更具成本效益——也更美味——的方式是将碎牛肉与 [切碎的蘑菇混合，这样既减少了脂肪又不牺牲味道)。](https://thetakeout.com/this-cheeseburger-has-1-3-less-fat-and-you-ll-never-not-1823132695) )而正如我们的实验所显示的，Beyond +牛肉的组合并没有改善味道。

最终，这个想法在理论上听起来不错，但实际上，结果并不令人鼓舞。事实上，将假肉和真肉结合起来的唯一理由是一个滑稽的理由。电梯推销应该是这样的:“Beyond Burgers 太贵了吗？把它拉长——用牛肉！”
