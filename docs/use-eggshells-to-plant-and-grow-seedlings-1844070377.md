# 用蛋壳来种植和培育幼苗

> 原文：<https://lifehacker.com/use-eggshells-to-plant-and-grow-seedlings-1844070377>

在检疫烘烤的流行和容易制作的鸡蛋菜肴之间，我们可能会有更多的蛋壳。不是每个人都有时间和精力去清洗和保存蛋壳，但是如果你属于这一类，我们有一个新的(字面上的)小窍门给你。我们已经介绍了几个基于蛋壳的黑客技术，比如 [用它们来让蛞蝓远离你的花园](https://lifehacker.com/keep-slugs-and-pesky-insects-out-of-your-garden-with-eg-824479727) 或者 [漂白你的衣物](https://lifehacker.com/whiten-laundry-with-egg-shells-1603108559)——但这个更可爱，因为它涉及到培育幼苗。

### 如何在蛋壳里育苗

如果你一直想拥有自己的花园，但不知道从哪里开始，这种方法可能是答案。在皮特和格里的有机鸡蛋网站 上，他们有一整套在蛋壳里育苗的便捷指南，即使你以前从未做过。以下是要做的事情:

*更多关于在家种植植物的信息，请看下面的视频:*

#### 收集你的补给品

真的，你只需要四样东西:

*   一把蛋壳，洗净，对半切开
*   鸡蛋纸盒
*   预湿种子初始混合物
*   草本植物或小花的种子

但是你如何得到一把蛋壳，清洗干净并切成两半？皮特和格里建议敲开你的鸡蛋，然后切成两半(不一定要完美)并清洗——把所有的生鸡蛋放在一个容器里，你以后可以用来烹饪或烘焙。另一个选择是像平常一样使用鸡蛋，但在使用时花时间和精力将它们减半，然后清洗它们并保存起来，直到你有足够的鸡蛋开始种植项目。无论哪种方式，你都会得到你需要的贝壳。你可以用装鸡蛋的纸箱，也可以用你拿着的另一个。

#### 在蛋壳里播种

首先在鸡蛋纸盒的每个槽中放一个空壳。然后，开始种植。按照 [他们的指令](https://www.peteandgerrys.com/blog/start-seeds-in-eggshells) :

> 用茶匙将种子混合物填满每个空蛋壳，直到顶部，在土壤和蛋壳边缘之间留出几毫米的空间。加入泥土后，用手指在泥土上挖一个小洞，为种子腾出空间。在每个洞里放入两颗种子，轻轻撒上一点土。

#### 照料你的幼苗

既然你的种子已经在土壤里，你的工作就是照顾它们。确保你把纸箱放在窗台上，或者其他光线充足的地方，最好比家里其他地方暖和。

像任何幼苗一样，水真的很重要——尽管你不想过度。这个想法是保持土壤湿润，同时不要过度浇水(记住:蛋壳没有小的排水孔)。给这些幼苗浇水的最好方法是使用喷雾瓶，每两到三天轻轻喷洒一次。

#### 修剪矮子

因为你在每个蛋壳里种了两颗种子，所以很有可能最终会长出两颗幼苗。如果是这样的话，你可以剪掉两个芽中较小的一个来帮助另一个生长。

### 如何移植秧苗

最终，你的幼苗会变得对蛋壳来说太大——通常是当它们长到几英寸高，或者长出第二对叶子的时候。当这种情况发生时，你就可以把植物搬到新家了。

#### 户外花园

如果你选择将幼苗移至室外花园，你需要做额外的准备工作。它被称为，意思是“用手在植物上轻轻摩擦，以模拟微风，并逐渐让它们在户外停留更长时间，以帮助它们适应户外的天气。”这需要你做更多的计划，但是对于你自己的美丽的户外花园来说，这只是小小的代价。

#### 陶罐/室内花园

如果你更愿意将幼苗移至陶罐或某种室内花园，你可以跳过“炼苗”过程。万一你没意识到，你要把整个蛋壳放进锅里。贝壳会自然分解，充当一种肥料，给你的幼苗提供养分。至于具体的移交过程，我将把 [移交给皮特和格里](https://www.peteandgerrys.com/blog/start-seeds-in-eggshells) :

> 当你移植幼苗时，第一步是在一个足够大的花盆或花坛中填入你用来种植蛋壳的土壤。接下来，拿起里面还有植物的蛋壳，非常非常温柔地压碎鸡蛋的外面，足够让蛋壳外层的一些碎片脱落。接下来，把蛋壳种在新的花盆或花园里，确保蛋壳的顶部完全被埋住。

如果你选择种植药草，你的植物应该在几周内长到足够大，可以被切割并用于烹饪。否则，享受看你的花成长。
