# 我们应该全年都吃蔓越莓酱

> 原文:[https://life hacker . com/we-should-eat-cranberry-sauce-all-year-1840055543](https://lifehacker.com/we-should-eat-cranberry-sauce-all-year-1840055543)

蔓越莓酱在感恩节起着至关重要的作用。没有它，盐和脂肪会主宰你的味觉，你的嘴会很快因丰富的美味而过饱和。蔓越莓酱解决了所有这些问题，因此我们的食物更好。但是为什么要把香甜扑鼻的调味品局限在一天呢？它能提供更多的东西。

如果没有别的，你应该把它放在你所有的火鸡三明治上。蔓越莓酱在剩菜三明治中有很多好处，但这不是你一年中唯一吃的火鸡三明治。见鬼，涂在其他三明治上也不错——火腿，烤奶酪，或者烤火腿和奶酪都可以从品红色酱汁中受益。

你知道有些奶酪盘会在切达干酪和蓝调音乐上加一点果酱吗？是的，蔓越莓酱也可以。它也适用于猪排或炸土豆饼(我们也应该全年都吃)，而不是通常的苹果酱。

你可以在正常的日历年购买罐装蔓越莓酱，但如果你看到假日大减价，现在也不是囤货的时候。如果你喜欢自制，你可以冷冻新鲜的小红莓(或者购买已经冷冻的小红莓),然后随心所欲地做一小份你最喜欢的食谱。如果你愿意的话，没有理由不每天享受蔓越莓酱的酸甜。时间是一种幻觉，蔓越莓酱是好的。相应地调整你的生活。
