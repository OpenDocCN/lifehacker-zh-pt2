# 你可以直接从冰箱里烤洋葱圈

> 原文:[https://life hacker . com/you-can-waffle-onion-rings-direct from-the-freeze-1844250637](https://lifehacker.com/you-can-waffle-onion-rings-straight-from-the-freezer-1844250637)

吃洋葱圈的人，在他们生命中的某个时刻，咬了一口洋葱圈，结果洋葱部分从松脆的家中滑了出来。这并不是可能发生的最糟糕的事情，但却相当令人沮丧，因为软洋葱和油炸面糊应该一起享用。晃动你的洋葱圈让这个问题成为过去。

我最初没有华夫饼冷冻洋葱圈，因为我认为这将解决洋葱圈滑脱的问题；我这样做是因为我想要五个洋葱圈，而不想打开烤箱烤五个冷冻洋葱圈。但是，当我正在吃我的五个洋葱圈时，我注意到洋葱还在原地，哦，我的朋友们，我真的*高兴了*。

多亏了华夫饼机中的热槽盘，不仅洋葱保持不动，而且洋葱圈非常脆。它们也有点扁平，这意味着如果你是那种在汉堡上放洋葱圈的人，你可以在汉堡上放更多的洋葱圈；但是它们也被挖掉了，这意味着它们能容纳更多蘸酱。但最重要的是，waffling 洋葱圈意味着你可以在几分钟内心血来潮随意吃几个洋葱圈，而不必加热任何类型的烤箱。

只需将你的华夫饼机调到中高，扔几个洋葱圈进去，轻轻按下顶部的熨斗，直到洋葱圈解冻变软，这样你就可以完全关闭华夫饼机了。煮几分钟，直到洋葱圈变成深金黄色。从华夫饼机中取出来——筷子对于把东西从凹槽中取出来很有帮助——然后享受你的戒指，不受洋葱滑落的影响。
