# 如果我被困在家工作，我需要一个新的电缆调制解调器吗？

> 原文：<https://lifehacker.com/do-i-need-a-new-cable-modem-if-im-stuck-working-from-ho-1842455120>

欢迎来到 [**理工 911 周**](https://lifehacker.com/c/tech-911) 。由于我们中的许多人被困在自己的家里或公寓里，我正在回答*至少*一个关于每天在家工作的问题。(如果你继续在我的收件箱 或评论中向我发送 [，我甚至可能会做一个大的快速总结。)](mailto:david.murphy@lifehacker.com?subject=Tech%20911)

所以，不再耽搁，今天的问题来了。我打算转述一下，因为一位朋友前几天在脸书问题上提出了这个问题，我不确定他们是否希望自己的名字与这个问题联系在一起。因此，出于隐私的考虑，我转述了他们的想法。

### 问题是:

我必须在家里为一家很棒的科技公司工作，下载和上传我的作品需要很长时间。我用康卡斯特升级了我的网络计划，但这是否意味着我还需要一个新的电缆调制解调器？

### 正确答案是:

我要选好的“看情况”然而，我首先要赞扬你实际上拥有了自己的电缆调制解调器，而不是每月花 14 美元左右让康卡斯特租给你一个他们的调制解调器/路由器组合。没有理由你应该付费去租那些你可以自己购买的东西。当然，这也意味着如果你的网络计划超出了调制解调器的能力，你就必须买一个新的电缆调制解调器。

在这种情况下，我向我的朋友询问更多信息，他提到他已经从 Comcast 获得了 300Mbps 的套餐。他之前没有提到他的服务计划，所以*有*的可能性，他需要购买一个新的电缆调制解调器，才能最大限度地提高这些速度——下载速度，也就是说，因为“300Mbps”，我们知道，并不意味着双向 300Mbps。

事实证明，他的 [网件 CM500](https://www.netgear.com/home/products/networking/cable-modems-routers/CM500.aspx) 有线调制解调器是绰绰有余的。谷歌快速搜索确认它支持高达 680Mbps 的速度——对于 300Mbps 的计划来说绰绰有余。虽然我可以进入“16x4”作为其配置的一部分意味着什么的技术细节，但这对大多数人来说是无关紧要的。它与 [康卡斯特](https://www.xfinity.com/support/devices/) 兼容，最重要的是，康卡斯特自己证明它将支持 300Mbps 的速度。

这一点很重要，因为当涉及到电缆调制解调器时，你永远不应该假设制造商是正确的。仔细检查你的 ISP 提供的兼容电缆调制解调器列表，确认你拥有的或打算购买的任何东西都可以与该公司的服务兼容*和*可以处理你期望的任何速度。如果您在列表中没有看到您的调制解调器，或者根本没有看到列表，请给他们打电话。

注意:如果你也通过你的 ISP 获得电话服务，那种你实际上是把 VOIP 电话连接到你的调制解调器上的服务，那么你可能需要一个功率更大(也更贵)的电缆调制解调器。它们是突破口；不要买一个新的，便宜的，典型的调制解调器，并假设它可以和你的 VOIP 系统一起工作。

### 但也许不是电缆调制解调器的问题...

我还认为重要的是要知道，通过你的 ISP 升级你的服务计划，并购买一个新的电缆调制解调器，可能*而不是*实际上给你带来你所期望的速度提升。如果你家里的 wifi 设置很差或者功率不足，或者如果你试图从一个离你的主路由器相当远的地方访问你的文件——或者在它和你之间有很多东西(包括墙)——那么你的无线设置可能比你的服务计划更应该受到责备。

老实说，我会先调查一下。在你的 ISP 的月账单上查找你支付的任何服务等级，并在你的房子周围走动——在不同的点运行[【fast.com】](https://fast.com/)或[【speedtest.net】](https://www.speedtest.net/)——看看你是否在你通常使用各种设备的地方获得了最高速度。如果没有，修复您的无线设置应该是首要任务，*然后*担心您的 ISP 提供的更快的互联网服务。

你会问，你是怎么做到的？首先，请确保您的路由器位于家中的中心位置。这可能需要你沿着地板和天花板铺设以太网电缆，而不是在你房子最角落的小壁橱里，离你在家工作的桌子最远。相信我，这会有回报的。(如果你被隔离了，你就有足够的时间做这样的项目！)

除此之外，如果您使用 wifi 扩展器或网格设置来扩展路由器的信号，请知道您在链中的每一个“跳”——从路由器到扩展器，到另一个扩展器，再到您的笔记本电脑——每次都可以将吞吐量减少一半。除非您的网状网络设置有专用的回程连接，并且您的接入点彼此之间的距离足够近，可以从强大的连接中受益，否则简单地将您的网络扩展到您需要无线覆盖的地方就可以让您上网，但速度可能会非常慢。

一如既往，尽可能使用有线以太网以获得最佳速度。拿一个加密狗，把你的笔记本电脑连接到墙上的以太网端口(使用坚固的 Cat6 电缆，因为我喜欢面向未来，它几乎不比 Cat5 或 Cat5e 贵多少)。使用以太网电缆将您的接入点和路由器相互连接起来，尽管这可能很烦人，但不要使用 wifi 来扩展您的无线网络。

我知道这很痛苦。但我刚刚将一所房子从全无线设置转换为有线设置，现在这家人终于可以享受他们一直支付的*千兆*以太网速度——他们*从未*拥有的速度，因为他们依赖一个蹩脚的电缆调制解调器/路由器和扩展器“pods”来尝试让他们的房子充满无线连接。他们可以通过回拨他们的互联网计划来匹配他们网络的能力，从而节省一大笔钱；或者他们可以像我一样升级他们的网络，通过以太网电缆将接入点连接到主交换机(和路由器),并实际享受他们支付的超快的速度。

这只是我啰嗦的一句话，我想说你的上网计划可能不是你首先应该责怪的家里网速慢的原因。当然，这是等式的一部分，但是你也要确保你的无线网络符合标准。如果或者当你升级你的计划并获得一个新的支持它的电缆调制解调器(如果适用)，确保你做同样的过程——在你的房子周围走动，测试你的速度，并修复你的 wifi 网络——以确保你不只是为你永远看不到的服务向你的 ISP 付费。
