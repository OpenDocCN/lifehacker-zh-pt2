# 如何成功飞待机

> 原文:[https://life hacker . com/how-to-successful-fly-stand by-1843567056](https://lifehacker.com/how-to-successfully-fly-standby-1843567056)

回想一下你最后一次在机场等待登机的情形。(还记得机场吗？还记得飞行吗？)你可能已经注意到登机口柜台上方的电视屏幕上满是乘客名单，并听到一名空服员通过对讲机喊着同样的名字。这些是希望搭乘你的航班的候补乘客的名字。

等待航班意味着你在等待你实际上没有买票的航班的下一个座位，对于预算有限的旅行者来说，这是一个很好的方式，可以花更少的钱赶上更早的航班回家——但是任何人都可以等待航班。下面是怎么做的。

### **谁能飞待机**

过去，待机飞行意味着你没有机票就出现在机场，然后等着，希望能在起飞的航班上找到一个空座位，但 9/11 之后的安全变化意味着乘客必须有预购的机票才能尝试待机飞行，而且你只能凭原始机票飞往目的地。这些天来，等待航班通常意味着你已经买了一张机票，但想看看你是否能在飞往同一目的地的更早航班上找到一个空位。然而，当然有更多的情况下，人们会想乘坐备用飞机:

*   你持有好友通行证。联票是航空公司员工给予朋友或家人的一种特殊待遇，允许他们无需支付全价机票即可乘坐飞机。使用伙伴通行证时，你不能保证有座位，必须在通过安检后将你的名字添加到登机口的候补名单中。(下面有更多关于好友通行证的信息。)
*   您错过了航班,正试图搭乘下一趟航班。这是人们等待航班的最常见原因，因为一旦你错过了最初的航班，你可能不得不等待，看看哪些后续航班(如果有的话)会有空座给你。
*   你想要升级。当您申请升舱到更好的座位时，您的名字将被列入候补名单，与其他也申请升舱的人一起。精英会员和高级会员将始终获得第一次升级的机会。
*   **一个航班被超额预订**，而你没有座位分配。当一个航班太满，票卖得比座位多时，任何没有实际座位的人都会被列入下一个座位的候补名单。

### **成为朋友**

近年来，各航空公司围绕“伙伴通行证”的政策已经收紧，这样人们就不能滥用它们，但这在旅游业中仍然是一件很常见的事情。伙伴通行证是给航空公司员工的航班通行证，供他们的家人和朋友使用。它允许乘坐通票的人只需支付机票的税费。

伙伴套票的使用规则因航空公司而异，因此如果您计划乘坐其中一家航空公司的航班，您需要了解具体的政策。还要知道，使用联票并不能保证你在飞机上有座位——它只能保证你在广播名单上有一个位置，如果有一个座位空出来，取决于你的名字在名单上的位置，你就可以坐飞机。

当您使用 buddy 套票时，您希望尽可能早地出现在机场，这样您的名字就会尽可能靠近列表的顶部，从而增加您获得座位的机会。

### **如何增加获得候补座位的机会**

你可以采取一些步骤来有效地执行待机飞行，并让自己更快地搭乘下一趟航班。例如，如果你独自飞行，而不是三个人一起飞行，你的运气会更好。将你的行李限制在手提行李而不是托运行李也将有所帮助，因为如果他们必须花时间找到你的托运行李并将其放在下一个航班上，你可能会被淘汰(从名单上除名)。

增加机会的其他方法:

*   **永远拥有“精英身份”**大多数航空公司不允许在当天免费改签基本经济舱机票，这意味着候补是您唯一的选择，而拥有精英会员身份的乘客在候补名单上始终享有优先权。如果没有精英身份，你可能会被其他拥有精英身份的人从名单上除名。不同的航空公司对精英会员资格有不同的要求，值得您在出行前进行调查。
*   **下载您的机场应用程序。**这是查看下一趟航班时间的最简单方法。他们也可能允许你做“当天航班变更”并把你的名字加入候补名单，而不需要和代理人谈。
*   你来得越早越好。机场建议您提前 2-3 个小时到达机场的原因实际上是为了帮助我们避免因安检排长队或其他机场意外事件而错过航班，但您越早到达登机口，就能越早在候补名单上登记，所以不要在最后一刻离开或在 Hudson Books 闲逛。
