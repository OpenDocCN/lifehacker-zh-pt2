# “驱蚊”应用程序不起作用

> 原文：<https://lifehacker.com/mosquito-repellent-apps-dont-work-1835813081>

几十年来，我们已经能够买到播放噪音的设备，据说可以驱蚊，几十年来，这些设备完全没有用。现在，你可以下载一个应用程序，而不是购买一个设备。你猜怎么着？还是不行。



“这都是痴心妄想。没有证据表明发声装置可以阻止蚊子叮咬，”蚊子科学家卡梅隆·韦伯 [为《对话](https://theconversation.com/the-buzz-from-your-smartphone-wont-stop-mosquito-bites-92611) 写道。

然而，iTunes 和 Google Play 应用商店都有几十个驱蚊应用的列表。有些声称可以工作，而其他的 则带有类似“重要！还没有科学证明低频声音有助于驱蚊。正因如此，这个 app 应该算是一个笑话 app。” [另一位](https://play.google.com/store/apps/details?id=com.toolsandutilities.pestrepellentsound) 有益地指出:“你也可以像恶作剧一样使用这个应用程序，用这个声音来烦你的朋友:)”

如果你真的想赶走蚊子，就打开风扇。微风让蚊子无法通过你的呼吸嗅出你。为了获得额外的保护，使用含有有效成分的杀虫剂，如避蚊胺、哌啶或柠檬桉树油 。这些——不像声波驱逐应用——实际上是有效的。
