# 你可能会因为焦虑而拖延

> 原文:[https://life hacker . com/you-may-be-delaying-because-you-anxiety-1833809567](https://lifehacker.com/you-might-be-procrastinating-because-youre-anxious-1833809567)

拖延可能是你读这篇文章的原因，但是你拖延的原因可能实际上是焦虑。

Watch

今天，我花了很长时间才开始写作:首先，我必须检查我的三个电子邮件帐户，Twitter，Instagram，给我爸爸发短信，告诉他我想起的一些事情，喝杯水，然后再次检查我的电子邮件。我很确定，是我的拖延习惯让我无法接管这个世界。我一搜索茱莉亚·切尔德的出生年份就会知道了。

对于我们这些有拖延习惯的人来说，尤其是在重要的事情上，这可能不仅仅是“偷懒”拖延的冲动与我们大脑中的 [【战斗或逃跑】反应](https://qz.com/work/1584023/how-to-tell-when-procrastinating-is-actually-anxiety/) 有关，根据 Quartz in Work 的一篇文章，焦虑是如何阻止我们完成重要项目的。拖延是行动中的“逃跑”反应。这里有一些方法可以判断你是否焦虑——而不是懒惰——然后走出困境。

### 识别症状

也许你感觉很放松，享受着不用洗衣服或做其他家务的一天——太棒了！那很可能不是焦虑。心理学家安德鲁·罗森是佛罗里达州德尔雷海滩焦虑和情绪障碍治疗中心的创始人和主任，他告诉《工作中的石英》杂志，如果你花了很多时间合理化自己的行为，甚至认为你应该做的事情毫无意义，你就可以看出这是焦虑:

> 当谈到寻找避免那种不好的感觉的方法时，人类是非常有创造力的，无论是拖延(“我明天会做的”)、转移注意力(“我先去看看推特”)还是自我破坏(“你知道吗？无论如何，这是一个愚蠢的想法。”)最后一条在分析型或大脑型的人当中特别流行，他们甚至可能没有意识到他们放弃梦想的超理性原因受恐惧影响的程度。

如果你花了很多时间在你的脑海里，为你不得不做的事情是愚蠢的辩护，这可能意味着你不喜欢逃避它。

### 检查你的理由

只要你还在拖延，就用一种可以对抗焦虑的方式来拖延。如果你已经列出了为什么你不能做某事的理由清单，把这个清单拿出来，做成一个新的清单——一个检验这些理由的合法性的清单。加州大学伯克利分校的心理学助理教授 Robin Yeganeh 对此做了一个很好的练习:

> “我们经常养成一个坏习惯，在‘给出理由’的基础上，选择更舒服的行为，而不是对我们有益的行为，”耶加内说。“例如，‘我努力工作，所以我不应该做 X’或者‘我太累了，无法在 X 上取得进展’，我会建议列出不从事更高优先级行为的所有理由，然后挑战每个理由的可信度。决定这些规则是否带来了生活中的成功，或者它们是否需要升级以有利于做出决定的成功导向的原因。”

这是一种温和的方式来面对你正在做的事情:逃避责任。这可能足以让你振作起来，这样你就可以继续做真正需要做的事情。

### 审视你的感受

这一切的核心是你最初感到焦虑的原因。根据心理学家莱斯利·康纳的说法，如果你清理冰箱下面是为了避免一个可能导致升职的项目，那么可能有更深层的原因:

> 每一次成功都伴随着权衡——更多的曝光、更多的压力、更少的自由——忽视对这些的担忧会反过来伤害我们。
> 
> “如果我们只和肯定的感觉联系起来，把矛盾或恐惧压下去，它们就会出来。但有时他们会敲门，”康纳说。
> 
> 还有一个大问题:害怕失败。

如果你发现这是你生活中的一个持续问题，你可能需要和治疗师谈谈你的焦虑，因为没有人应该生活在这种状态下，不管他们有什么义务。但是，对一些人来说，承认恐惧和焦虑可能是打破拖延循环的开始。
