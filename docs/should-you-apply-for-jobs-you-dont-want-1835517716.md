# 你应该申请你不想要的工作吗？

> 原文：<https://lifehacker.com/should-you-apply-for-jobs-you-dont-want-1835517716>

Reddit 用户 MrNumber28 [问](https://old.reddit.com/r/personalfinance/comments/bznn9z/is_it_wrong_to_apply_for_other_jobs_to_see_what/) “有时候我想知道在另一家公司工作会不会得到更多报酬。我对自己的工作很满意，但显然我更希望得到更多的报酬。我不应该货比三家，看看还能买到什么，这有错吗/有什么理由吗？”



这是一个会让你成为第二个的困境——猜测你从第一天开始所做的每一个职业选择。你应该对你所拥有的心怀感激，对你所得到的工作充满信心吗？或者你应该一直寻找更好的东西？

把 FOMO 的职业生涯加到 T1 这个不时被提起的趣闻:当你每隔几年换一份工作，而不是等着加薪时，你的工资会增长得更快。

非常繁忙的 Reddit 帖子提供了许多曾经经历过这种情况的人的警告。是的，他们同意，你应该时刻关注你所在领域的工作机会。但要谨慎行事。

## 检查你的价值

mikefaley 坚持说这不是私人恩怨，这是生意。

> 当然了。我总是，也鼓励我的员工，总是去面试。这不像恋爱中的欺骗。这是生意。作为一名员工，你应该始终对自己的市场价值有一个概念，你应该像做其他事情一样，通过做推销员来练习面试。当大难临头时，你应该做好准备。

Jasonpatudy 建议对市场进行年度检查:

> 我最好的一位老板曾经告诉我，每年至少要向其他公司申请一次，以评估我在市场上的价值。你不必每次都要求匹配，但它会让你知道当你得到晋升时要求多少。

关注你所在领域的职位描述可以帮助你发现引起你兴趣的公司。它还能帮你剔除那些你绝对不想去工作的地方，比如任何一家公司都在寻找键盘忍者、在线战士或电子表格专家 T2 T3。

## 准备练习

Jesuschin 说他们不久后的第一次采访棒极了:

> 天哪，在 11 年没有参加面试后，我在第一次面试时说出了一句脏话，当时我脑子里在想“这次肯定不行。”不得不说，这让我卸下了肩上的重担，我只是利用面试的剩余时间磨练我需要为未来的面试做些什么。

萨塞尔霍夫说，它可以帮助你为最坏的情况做好准备:

> 如果你碰巧突然失业了(不管什么原因)，你的简历是最新的，你可以放心地去面试，因为你一直在“练习”，你知道自己在就业市场上的位置。

如果你已经离开市场一段时间了，练习面试问题可以帮助你预料到——你甚至可以 [让 Alexa](https://lifehacker.com/how-to-use-alexa-to-help-you-prep-for-a-job-interview-1833024806) 帮你准备。

## 在投入太多精力之前先试水

Cogentorage 建议尽早问这个大问题:

> 如果你有一份满意的工作，但发现自己经常与招聘人员接触，从薪资要求开始。如果你对自己的价值有所了解，你可以通过这种方式剔除很多垃圾信息。

招聘人员和人事经理 [希望你在面试过程中早点问](https://lifehacker.com/most-hiring-managers-expect-you-to-ask-about-salary-by-1793242735) 关于薪水的问题，所以在你了解职位和公司的时候，不要羞于询问细节。

## 了解你所在行业的局限性

Iplaychemistry 说:

> 我所在的行业充满了相对小众的小企业。如果我四处去面试，我很容易就能在这个州为我的领域的潜在就业机会烧掉每一座桥，并在这个过程中丢掉我的工作。这个行业很重要。

T2 的 Ask a Manager 的 Alison Green 建议说，在一个小领域工作意味着你在浏览工作机会时应该格外谨慎——你可能想让潜在雇主对你的申请保密。另一个警告:如果你在申请工作时没有任何转换公司的打算，你可能会自断退路。
