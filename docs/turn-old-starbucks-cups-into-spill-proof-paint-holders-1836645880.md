# 将旧星巴克杯子变成防溢涂料架

> 原文：<https://lifehacker.com/turn-old-starbucks-cups-into-spill-proof-paint-holders-1836645880>

我喜欢看我的孩子画画。我害怕看到我的孩子画画。这两种感觉并存。每当我六岁的女儿准备在她的艺术桌上尽情享受杰森·布拉克时，我通常会在几个纸碗里挤一些颜料。但是 [母包](https://www.instagram.com/p/B0NvStvFQJH/) 共享一种保持东西更干净的方法:把颜料倒入有圆顶盖子的透明杯子里。星巴克的星冰乐杯很完美。

Watch

为什么杯子有圆顶盖子？这种形状让孩子们可以轻松地刮掉刷子上多余的油漆。如果杯子倒了，盖子可以阻挡一些飞溅。(我还会往里面扔一块石头来称它的重量。)然后，当你的孩子完成他们的杰作时，你可以洗杯子，下次再用。是时候出去喝一杯抹茶绿茶星冰乐了——当然是为了艺术。
