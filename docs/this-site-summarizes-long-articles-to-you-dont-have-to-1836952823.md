# 这个网站为你总结了长篇文章，所以你不必去读它们

> 原文：<https://lifehacker.com/this-site-summarizes-long-articles-to-you-dont-have-to-1836952823>

长篇文章是深入探讨某个特定话题的好方法。也就是说，有时候你没有能力(或者说没有欲望)立刻通读一篇 5000 字的关于某个主题的文章，但是你仍然想要得到文章的主旨，这就是 TLDR 的用武之地。



(代表太长，没看完)，总结了太长你看不完的长文，不用看了。

该网站通过输入网站的 URL 或者将文章中的文本复制/粘贴到网站中来自动总结任何文章、文本、文档、网页或文章。

根据我们的测试，文章越长，网站越好。虽然它肯定可以总结较短的内容(例如，像这篇文章)，但它提取的亮点不如它能够完成较长形式的工作那么准确。

与记叙文之类的文章相比，它更适合于较长的新闻文章。例如，*外关于摩洛哥山区谋杀案的这篇文章的工具 [摘要在其摘要中却只字未提谋杀。](https://www.outsideonline.com/2397512/hikers-killed-morocco-atlas-mountains)* 

*然而，该网站关于 [的总结，这位*日报猛兽*文章](https://www.thedailybeast.com/suspected-el-paso-shooter-patrick-crusius-dad-once-sought-to-help-gun-violence-victim) 上的埃尔帕索射手却相当准确。* 

*显然，它并不能完全取代阅读整篇文章；但是它可以帮助你在决定把它加入你的必读清单之前，先了解它的内容。*
