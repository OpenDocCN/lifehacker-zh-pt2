# 与其制定大的新年计划，不如制定小的新年目标

> 原文：<https://lifehacker.com/instead-of-big-new-years-resolutions-make-smaller-new-1840710920>

一年中的这个时候，我们所有人都开始思考如何在接下来的一年里改善我们的生活。也许我们会开始跑步或者花更多的时间和朋友或家人在一起。无论你对 2020 年的希望和梦想是什么，如果你把它们设定为更小的目标，你可能会更成功。

Watch

虽然“决心”一词仅仅意味着决定做什么或不做什么，但在大多数情况下，当人们谈论决心时，他们谈论的是一个大问题，而不是合理可实现的事情。这意味着他们可能会失败。

例如，如果你从未去过健身房，你不会仅仅因为现在是 2020 年就突然开始每天都去。

不要追求“减掉 20 磅”的崇高目标，而是把这个目标分解成可以实现的小目标，这样你就可以在一年中完成。

就减肥而言，这可能意味着每周几次午休时出去散步，在家做一顿你可以吃的健康饭菜，而不是你通常买的快餐。

这里的想法不是不要制定减肥目标，而是把它分解成更小的部分，你可以很容易地跟踪和完成，并把它作为成就而不是改变你的生活方式。

通常，我们做出的决定是为了从根本上改变我们的为人。虽然改变可能很好，但你可能不会在一夜之间改变自己的很大一部分。相反，如果你把这种改变分解成更小的可实现的部分，那么你成功的机会会更大。

努力太快几乎总是以失败告终。一个好的经验法则是从一些小的改变开始。一旦你把这些纳入你的日常生活，你就可以开始增加一些了。过一段时间，你将会实现你所追求的巨大改变，但是是以一种更容易实现的方式。
