# 你可能有权获得美国航空公司的燃油附加费退款

> 原文：<https://lifehacker.com/you-might-be-entitled-to-an-american-airlines-fuel-surc-1836851731>

如果您最近通过美国航空公司预订了英国航空公司的 SEA-LHR 奖励机票，您可能有资格获得燃油附加费退款。

随着 [的分家伙](https://thepointsguy.com/news/american-airlines-fuel-surcharge-refund/) 解释道:

> 差异似乎涉及西雅图(海上)到伦敦希思罗(LHR)航线。英国航空公司最近将承运人收取的附加费下调至 65 美元，但 AA 似乎没有在其奖励预订系统中更新新的较低费率，而是收取承运人收取的 225 美元费用。

换句话说，一些人用他们的航空公司奖励通过美国航空公司预订英国航空公司的海上 LHR 机票被多收了钱。

当 [的加里·莱夫从](https://viewfromthewing.boardingarea.com/2019/07/27/american-airlines-overcharged-fuel-surcharges-on-some-british-airways-awards-will-provide-refunds/) 的机翼上查看到美国航空公司关于附加费差异的问题时，他得到了道歉和以下回应:

> 我们的团队现在正在努力纠正这些金额，当然我们也在努力更新我们的流程，以防止将来再次发生。此外，我们正在审查我们的数据，以确定任何可能以错误金额购买机票的客户，我们将主动联系他们，退还差额。

美国航空公司可能需要几周时间来联系他们的所有客户，所以请检查您的 SEA-LHR 奖励机票，看看您支付了多少承运人收取的附加费，以及您是否应该预计退款。(如果您没有使用奖励或里程乘坐海上 LHR 航班，这一附加费问题不应适用于您——但无论如何，请随时检查您的机票。)

如果你认为你有资格获得燃油附加费退款，但在接下来的几周内你没有听到美国航空公司的任何消息，那么联系他们可能是值得的。现在就把它放在你的日历上，这样你就不会忘记——然后如果你不需要联系美国人，你可以随时 [删除提醒](https://lifehacker.com/use-your-calendar-to-record-what-actually-happened-1835943219) 。
