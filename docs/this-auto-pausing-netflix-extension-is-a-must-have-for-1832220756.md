# 这个自动暂停的网飞扩展是 Chrome 用户的必备工具

> 原文：<https://lifehacker.com/this-auto-pausing-netflix-extension-is-a-must-have-for-1832220756>

我最喜欢的网飞把戏之一很简单。当你在你的网络浏览器，而不是一个独立的网飞应用程序中观看流媒体服务上你最喜欢的电视节目时，你很可能会遇到片头字幕，不得不拖动你的鼠标，点击方便的“跳过介绍”按钮开始。我认为这太麻烦了，因为你也可以点击键盘上的“s”来激活按钮。节省的秒数*。*

*Watch*

*你不会在 [网飞公布的榜单](https://help.netflix.com/en/node/24855) 上找到这个小小的热键，但乐趣也不必止于此。为 Chrome 安装 [**网飞扩展**](https://chrome.google.com/webstore/detail/netflix-extended/gjcgfkhgpaccjpjokgpekpgpphgaanej/related) 扩展，你会得到更多可以玩的东西——这是我绝对喜欢的一个功能，对于那些似乎无法长时间专注于一个节目的人来说。*

*除了网飞的常规命令外，网飞扩展版还解锁了四个新的键盘命令——比如“向上”或“向下”来增加或减少音量，“m”来静音，“space”来暂停。扩展增加了:*

*   *“N”——代表跳到你正在看的下一集*
*   *“B”——跳至上一集*
*   *“Esc”——返回主网飞屏幕(选择当前节目或电影)* 
*   *“R”-播放随机的节目或电影*

*在网飞扩展的选项中，你可以把这四个新热键中的任何一个换成完全不同的键，如果你愿意的话。更重要的是，你还可以激活一个特殊的功能——默认情况下是关闭的——当浏览器标签不在焦点上时，自动暂停你的网飞内容。如果你需要离开一会儿检查你的电子邮件或最喜欢的社交网络，这很好，因为当你回到网飞标签时，你的节目或电影会自动恢复播放。*

*默认情况下，网飞扩展也跳过所有显示简介和摘要。如果你想手动关闭这个设置，当然是用“s”键。*
