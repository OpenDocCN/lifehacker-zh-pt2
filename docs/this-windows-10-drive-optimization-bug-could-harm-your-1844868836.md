# 这个 Windows 10 的“驱动器优化”错误可能会损害您的驱动器

> 原文：<https://lifehacker.com/this-windows-10-drive-optimization-bug-could-harm-your-1844868836>

最新的 Windows 10 更新似乎比它需要的更频繁地运行自动硬盘优化过程。虽然这是硬盘维护的一个必要部分，让 Windows 在你的 SSD 上运行 [TRIM 命令](https://blog.oo-software.com/en/defragmentation-and-ssds-trim) 也不会有什么坏处，但是过度的*整理硬盘碎片*通常是不必要的，并且会随着时间的推移影响 SSD 的寿命。



简而言之，对驱动器进行碎片整理可以重新排列数据，从而提高加载效率。然而，对硬盘进行碎片整理需要对硬盘内容进行大规模重写，重复碎片整理可能会更快地磨损您的固态硬盘，因为每次读写数据时，存储数据的微型单元 SS Ds 都会被削弱。

许多专家建议最多一个月只整理一次你的固态硬盘，许多人甚至建议不要整理你的固态硬盘。但是运行 Windows 10 version 2004 的电脑在每次连接 SSD 的时候*都在执行自动碎片整理，包括每次重启电脑。完全磨损固态硬盘需要很长的*时间，但是那些过多的碎片整理仍然是有害的。**

微软承认了这个问题，并将发布一个补丁来修复它。在此之前，您可以关闭 Window 10 设置中的自动碎片整理工具，以防止不必要的重写:

1.  在 Windows 10 搜索栏中输入**“Defrag”**。
2.  从搜索结果中单击**“整理磁盘碎片并优化驱动器”**，打开“优化驱动器”窗口。
3.  点击预定优化部分下的**“更改设置”**(如果选项改为**“打开”**，则自动碎片整理已经禁用。)
4.  取消选中**“按计划运行”**
5.  点击**“确定”**，然后点击**“关闭”**

您仍然可以通过点击列表中的驱动器并选择**“优化”来手动运行碎片整理过程**
