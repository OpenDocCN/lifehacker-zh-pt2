# 在 MacOS 上访问表情符号的最简单方式

> 原文:[https://life hacker . com/the-easy-way-to-access-e mojis-on-MAC OS-1838778227](https://lifehacker.com/the-easiest-way-to-access-emojis-on-macos-1838778227)

在试图从我的 MacBook Pro 上删除一个多年前的自定义键盘快捷键时(失败了)，我无意中在键盘设置中发现了一个方便的选项，它让访问桌面上的表情符号变得更加容易。

Watch

打开“系统偏好设置”，点按“键盘”，然后勾选“在菜单栏中显示键盘和表情符号查看器”复选框这将立即在屏幕顶部的工具栏中创建一个图标，让您可以随时访问表情符号。要将它们插入文本字段，双击您想要的表情符号。

当然，你总是可以通过使用键盘快捷键“control + command +空格键”来访问 Mac 上的表情符号，但拥有一个我可以用可点击的快捷键打开的独立窗口是一个不错的选择；我个人觉得你用键盘快捷键时出现的弹出框有点 janky。请记住，如果您想要组织工具栏快捷方式的布局，请按住“command”键，同时点按并拖移它们，以便按照您喜欢的方式重新排列它们。
