# 在社交距离中自学一项任意的技能

> 原文：<https://lifehacker.com/teach-yourself-an-arbitrary-skill-during-social-distanc-1844533192>

我一直认为，每个人都应该学习一项古怪的技能，如果不是几项的话，来教会自己在武断中找到快乐。除了享受那件事，做好任何事都不需要“理由”,特别是如果它走在挑战性和可能性之间——只要你的能力足够远，如果你足够努力，你就能做到。

从 2008 年到 2010 年，我是一名和平队志愿者，像许多志愿者一样，我发现自己长时间孤独，除了通过互联网连接，与朋友和家人断开联系。在新冠肺炎教我们这个短语之前，这是为了今天的社交距离而进行的练习，我的一点小小的满足感来自于学习一项琐碎的技能:我自学变戏法。

我们之前已经讨论过如何教自己玩 的，虽然找到学习新技能的专业指导可以使掌握该技能更容易，但考虑到学习某样东西的随意性根本不需要以“正确的方式”学习。自由是在以你自己的方式和标准掌握某些东西的过程中。例如，我的方法就是让球在空中停留一分钟。没有判断，没有适当的形式，没有指示；如果我让三个球在空中停留一分钟，*我做到了*。这是一种我从小就已经忘记的感觉，是好奇心和随后的发现所带来的与生俱来的快乐。我能做到吗？我打赌我能。

当然，做一些意想不到的随机技巧和挑战是有好处的——想象一下学习扫雷、台球、莫尔斯电码、洗牌、旋转溜溜球、解魔方或驾驶手动挡，而没有协调、专注、坚持或成为一个更有趣的人的额外好处——但这些结果是副作用，而不是目标。学习一项新技能是为了记住*你是如何*学习的——随着年龄的增长，这些技能往往会被遗忘——以及伴随这个过程的乐趣。

我几个月的社会隔离以一项新技能结束，这是我至今热爱的。我仍然时不时地捡起橘子，让它们漂浮在水面上，只是为了纯粹的快乐，看看我是否还能。(我仍然可以不停地超过一分钟。)对于我们这些被社会孤立的人来说，找到一项“无用的技能”可以重建我们小时候拥有的肌肉。选择一项技能并以你的方式掌握它。提醒自己，你可以自学，你以前自学过，你可以再自学一次。你认为你能做到吗？我打赌你可以。
