# 如何让你的旧台式电脑保持良好运行？

> 原文：<https://lifehacker.com/how-can-you-keep-your-old-desktop-pc-running-well-1839141116>

当你的台式电脑老化了，但你没有心思(或预算)去更换它，你仍然可以做很多事情来给它一个长时间的，令人满意的生活。在 Lifehacker 本周的 [技术支持专栏](https://lifehacker.com/c/tech-911) 中，我们回答了一位读者的问题，他想知道他们应该在陈旧台式机的维护清单上写些什么。

Watch

Lifehacker 读者**“BH”**写道:

> 我有一台用了将近 10 年的台式电脑。我想积极主动地照顾它，而不是让它在我面前死去。
> 
> *下面是部分描述:*
> 
> *   *赢 10；i7 CPU 860 @ 2.80GHz 处理器；*
> *   *12gb RAM；*
> *   *硬盘换成 SSD*
> *   *几年前更新显卡 NVIDIA GeForce GT 630 配 1gb；*
> 
> 过去这里很安静，但现在似乎大部分时间都能听到粉丝的声音。这是一个警告信号吗？我应该考虑更换电脑，因为它的年龄，还是我应该更换任何组件？我很欣赏你对此的想法。

好消息是，BH。你已经完成了我能想到的所有主要升级。也就是说，你已经在系统中安装了固态硬盘(这是我要做的第一件事),并提升了显卡性能。(Nvidia GeForce GT 630 并不强大，但在某些时候，你的 CPU 将成为你性能的瓶颈，所以我不会挥霍在任何更令人印象深刻或更昂贵的东西上。)

只要你的系统对你很好，允许你在上面做任何你想做的事情——即使你永远不能在 4K 玩游戏——那么你就可以暂时升级了。或者，换句话说，你需要(或想要)进行的下一次大升级将是更换所有旧的东西，并构建一台新的 PC。您已经达到了当前配置的极限(或者说，至少已经非常接近了)。

关于持续维护，你可以做很多事情来保持你的旧电脑运行在最佳状态。我写了一个指南 [关于清理你系统(包括它的风扇)中所有的灰尘和粘糊糊的东西](https://lifehacker.com/how-to-clean-your-dusty-messy-desktop-pc-1825160454) ，你绝对应该每年至少做一次。更保守地说，你可能还想移除你的 CPU 的冷却器，擦掉旧的导热膏，并且 [应用新的批次](https://lifehacker.com/how-to-fix-a-cpu-temperature-error-when-your-computer-g-1829336876) 。(不要用太多。)

我喜欢你已经注意到你的系统在做什么和它发出什么声音。由于您可能每天都在使用您的计算机，这是确定某件事情是否可能会变坏的最佳方法之一——如果您的系统听起来与几天前不同，或者您注意到某个特定组件发出声音，您可能需要进行调查。

在这种情况下，如果你能听到风扇的声音，而你以前没有听到，你的第一个选择是 [清理它](https://lifehacker.com/use-dryer-sheets-to-clean-out-dusty-computer-case-fans-5876383)—我假设我们在谈论你的一个 80 毫米机箱风扇(或者你碰巧使用的任何尺寸)。如果这不能解决问题，它可能值得更换。 [拿出一个快坏的风扇，换上一个全新的](https://kotaku.com/swapping-out-my-noisy-old-pc-case-fans-made-a-huge-diff-1825397401) 超级简单，几乎不费吹灰之力。见鬼，你甚至可以拿起一个亮着的风扇，或者一个比你以前用的更安静(或者吹出更多空气)的风扇。

CPU 冷却器上的风扇或显卡上的任何风扇也是如此。如果它们听起来很奇怪，就好好清洗一下。如果它们听起来仍然有点问题，但看起来工作正常，运行速度也不慢，那么您可能很好，但这是您需要关注的事情。您也可以随时更换出现故障的 CPU 冷却器——无论是库存冷却器还是售后冷却器。显卡上的冷却器(如果是罪魁祸首的话)更难更换(如果你能做到的话)。在易贝上买一张新卡(或者稍微升级一下)可能更好。

不过，这是我的建议。总的来说，我认为你的旧电脑完全可以进行全面升级。换句话说，我会抛弃它的大部分组件(除了固态硬盘)，并建立一些新的东西。然而，如果你过得很好，忽略我的建议。保持系统清洁，注意任何可能老化的活动部件，尽可能更换。知道如何对你的电脑进行故障诊断并保持其良好状态将会让你在今后的极客生活中受益匪浅。
