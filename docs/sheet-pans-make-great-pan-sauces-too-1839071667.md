# 薄板锅也是很好的锅底

> 原文：<https://lifehacker.com/sheet-pans-make-great-pan-sauces-too-1839071667>

很难比 [单锅饭菜](https://skillet.lifehacker.com/make-two-weeks-worth-of-breakfasts-with-one-sheet-pan-1819690171) 方便。在同一个锅里并排烘烤蛋白质和蔬菜是一顿丰盛大餐的捷径。嗯，*快完成了——调味汁在哪里？单锅还是锅，那我们为什么要给长柄锅预留 [锅酱](https://lifehacker.com/how-to-make-a-freaking-pan-sauce-1795277933) ？* 

Watch

尽管每当我烘烤原料和汤时，我都使用非常相似的方法，但直到我在 Instagram 上到处看到艾莉森·罗曼的 [醋鸡配碎橄榄酱](https://app.getpocket.com/read/2735042646) 时，我才明白了片状潘酱的概念。这是一个天才的想法:在薄烤盘上烤肉和/或蔬菜，移到盘子里，把高汤、奶油、酒、醋，甚至是普通的水直接倒入仍然热的薄烤盘。像在长柄锅里一样，用勺子刮出焦黄的部分，用盐和胡椒调味，浇在所有东西上。如果酱汁看起来太稀，把锅放回热烤箱大约五分钟来减少。

我决定在我最喜欢的薄烤盘餐上试试:马蒂·马西森的烹饪书《T2》T3 中的烤意大利香肠、土豆和洋葱。烤好后，我把食物堆在一个盘子里，用半杯自制的鸡汤，加上两汤匙苹果醋和光滑的第戎芥末给锅上光。这道菜让人欲罢不能:丰盛的香肠，浸泡在香肠汁中的脆皮土豆，以及令人上瘾的辛辣潘酱。我再也不会跳过单锅酱，我迫不及待地想看看它能为烤蔬菜做些什么。
