# 为什么你应该把我的手表换成计时器

> 原文:[https://life hacker . com/why-you-should-dump-myfitnesspal-for-cronometer-1837835080](https://lifehacker.com/why-you-should-dump-myfitnesspal-for-cronometer-1837835080)

Myfitnesspal 似乎是健身界默认的卡路里计数器，我想不出任何好的理由来解释为什么。当你试图记录你的食物时，它试图向你显示广告和文章，将你的运动卡路里放入屏幕顶部的一个等式中( [不是一个很好的跟踪方式，老实说](https://lifehacker.com/for-accurate-calorie-counts-don-t-track-your-activitie-1769445569) )，它的食物数据库是一个不准确列表的雷区。还有更好的应用程序，我最喜欢的是一个不太为人所知的: [Cronometer](https://cronometer.com) 。

Watch

我喜欢它的最大原因之一是:你可以使用该应用程序，而不必纠结于卡路里。Myfitnesspal 将你一天的卡路里显示在主屏幕的顶部，整个应用程序似乎都是围绕着你正在努力减肥的假设而构建的。但是，如果你只是想在保持健康的同时吃得更好，或者在不节食的情况下增加一些肌肉，Cronometer 可以让你减少对卡路里的关注。

相反，当我打开 Cronometer 时，我看到了一天中我的卡路里和宏量营养素(蛋白质、碳水化合物、脂肪)的条形图(伴随着非常小的数字)。付费版本中的一个功能让我可以跟踪我每餐的蛋白质含量。滑动到下一个屏幕，你可以看到一个不同的视图，显示你在一天中完成“目标”的情况。卡路里不一定要涉及；我目前包括纤维、添加糖、蛋白质和一些维生素。

基本上，我喜欢 Cronometer，因为它让我跟踪我想跟踪的东西，感觉不像是饮食文化的延伸。它不会不断地通知我，也不会有一个标签，上面全是关于低热量零食的文章，以及为什么我没有花足够的时间在大自然中。是的，这是一个食物跟踪应用程序，跟踪食物的行为本身就有问题。但这似乎是一种不那么糟糕的方法。
