# 如何最终关闭网飞的自动播放视频预览

> 原文：<https://lifehacker.com/you-can-finally-turn-off-autoplaying-video-previews-on-1841498753>

自从我开始看《网飞》以来，我就一直在等待这部电影，我很高兴这一天终于到来了。网飞最终屈服于大众需求，发布了一项功能，允许你关闭所有该死的自动播放预览。



更好的是，如果你的家庭(或者和你一起分享网飞的朋友们)在这些预告片上出现分歧，你可以逐个改变你是否想要自动播放视频。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1222629203327291393&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1222629203327291393&autosize=1) 

首先，你需要在台式机或笔记本电脑的网络浏览器上的“管理档案”中调出你的档案——如果你愿意，可以使用 [这个便捷的链接](https://www.netflix.com/profiles/manage) 。单击您的个人资料，您会看到以下两个选项:

我想你知道你想取消哪一个。这样做的时候不要太激动，但是当你浏览或观看网飞的时候，你会享受到幸福的宁静。

如果你在你的电脑上，我想你也可以使用 Chrome 扩展 [经典网飞](https://chrome.google.com/webstore/detail/netflix-classic/mokjkiliipanjhlfbpagnmlpihmaohde?utm_source=chrome-ntp-icon) 来完成类似的任务(以及其他变化)，但由于大多数人不只是在他们的台式机或笔记本电脑上观看网飞，这个经常被要求的变化是更优雅的解决方案。
