# 卸载这八个窃取了数百万人数据的浏览器扩展

> 原文：<https://lifehacker.com/uninstall-these-eight-browser-extensions-that-stole-dat-1836539093>

网络安全研究员 [萨姆贾达利](https://securitywithsam.com/)T5】最近发现大规模数据泄露，泄露了 45 家大公司和数百万个人的私人信息。Ja dali 和他的团队称之为“DataSpii ”,这次泄露是由看起来无辜的 Chrome 和 Firefox 浏览器扩展造成的，这些扩展收集并分发用户的浏览数据——显示用户私人信息和一长串公司的 URL，包括苹果，沃尔玛，亚马逊，23AndMe，SpaceX，Skype 等。(完整列表包含在 Ja dali 的报告 T11 中)。)

用于实施泄漏的八个扩展是:

*   品牌调查 (Chrome)
*   **FairShare 解锁** (Chrome 和 Firefox)
*   **蹄目〔t1〕(铬)**
*   **小组社区调查** (Chrome)
*   **面板测量**(镀铬
*   **SaveFrom.net 助手**(火狐)
*   **说吧！**(铬合金)
*   超级变焦(Chrome 和 Firefox)

Ja dali 向 Chrome 和 Mozilla 报告了跟踪活动，他们的回应是远程禁用附加软件并将其从市场上移除。然而，Ja dali 继续监控这些现已被禁用的浏览器插件的活动，却发现它们仍在跟踪用户数据，尽管它们的主要功能已被禁用。

换句话说，**卸载上面列出的任何扩展**，如果你正在使用它们中的任何一个。虽然其中一些扩展的用户不到 10 人，但至少有两个扩展的用户超过了 100 万，其余的扩展有几万到几十万用户。

这些扩展中的每一个都以不同的方式跟踪数据，并使用偷偷摸摸的策略——例如等到安装后 24 天才开始跟踪——来混淆数据收集过程。收集到的数据然后卖给任何感兴趣的买家，结束了 Ja dali 在他的 [完整报告](https://securitywithsam.com/2019/07/dataspii-leak-via-browser-extensions/) 中图解的过程:

Ja dali 还提醒了那些信息也被曝光的公司，他们能够证实 Ja dali 的发现。泄露的数据包括敏感的公司信息和泄露的用户数据，如员工姓名、地址、信用卡信息、密码和 PIN 码、存储的云文件等，甚至在某些情况下还包括纳税申报单、基因信息和病史。

### 考虑核选项来保护自己免受不良扩展的影响

虽然受影响的用户已经收到警报，但当发生这种泄漏时，明智的做法总是 [查看您的帐户活动和/或更改信息](https://twocents.lifehacker.com/what-to-do-if-theres-a-data-breach-1826450129)——即使您的数据没有具体受损。

展望未来，我们首先推荐一条建议:限制你在浏览器中使用的扩展名数量。虽然一个扩展出现在官方市场上并不一定意味着它是安全的。

虽然有很多令人惊叹和有用的第三方浏览器扩展，但也有很多人想利用你。我们并不是说使用零扩展，这是最安全的做法，但是要注意那些你*在你的浏览器中安装的扩展。也许你不需要 30 个扩展来完成你的大部分工作，一个五个的准系统——来自你认可的官方公司——就可以让你一天顺利完成。*
