# 如果你使用亚马逊的门铃设备，请更改你的 Wifi 密码

> 原文：<https://lifehacker.com/if-you-use-amazons-ring-doorbell-devices-change-your-w-1839721221>

如果你正在使用亚马逊的门铃设备，你应该立即更改你的家庭 wifi 网络密码。 [据 Techcrunch](https://techcrunch.com/2019/11/07/amazon-ring-doorbells-wifi-hackers/) 报道，门铃设备在设置期间与用户的智能手机以明文形式共享 wifi 密码——本质上是未加密的文本数据。这使得家庭网络和任何与之相连的设备容易被黑客或附近任何知道如何收集这些数据的人利用。



亚马逊在几周前悄悄地修补了这个问题，但直到现在才通知其用户和公众，所以至少有可能你的 wifi 密码被泄露了。Ring 用户应该更改他们的家庭 wifi 密码，以防止任何不必要的网络使用或访问您的设备和存储在设备上的个人数据。这样做的具体步骤取决于您的路由器型号和互联网提供商，但在网上查找说明应该很容易。

不过，更改密码只是开始；我还建议花点时间反思一下，亚马逊的智能家居产品是否值得安全方面的担忧。似乎每隔几个月，我们就会听到该公司的 [智能扬声器](https://lifehacker.com/prevent-amazon-from-eavesdropping-on-your-alexa-convers-1833974108) 、 [家庭安全产品](https://lifehacker.com/how-to-stop-amazon-employees-from-watching-your-cloud-c-1838949389?rev=1570736025537) ，甚至只是 Alexa 本身可能会错误处理——甚至泄露——你的私人信息。

这并不是说亚马逊是唯一一家应该受到这种审查的公司，但似乎他们的产品更容易受到牵连 [而不是谷歌](https://lifehacker.com/protect-your-privacy-from-your-own-cloud-security-camer-1831684103) 的类似设备，当然 [也比苹果的](https://lifehacker.com/check-out-apples-updated-privacy-site-asap-1839698198) 更多。这些亚马逊拥有的产品通常会削弱用户的信任，当你的房子被门铃、安全摄像头、消防电视和/或 Alexa 扬声器装饰起来的时候，如果你想保证你的数据安全，有一些重要的考虑要做。
