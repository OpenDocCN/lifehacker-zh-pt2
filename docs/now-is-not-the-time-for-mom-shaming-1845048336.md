# 现在不是“让妈妈蒙羞”的时候

> 原文:[https://life hacker . com/now-is-not-the-time for the-mom shaming-1845048336](https://lifehacker.com/now-is-not-the-time-for-mom-shaming-1845048336)

大约十年前，在所谓的“妈妈战争”期间，我成为了一名妈妈，当时育儿博客——其中绝大多数是女性——经常使用他们的社交媒体平台来推广他们的育儿方式。在此期间，妇女们感到迫切需要找到她们的“部落”他们的专属母乳喂养部落，他们的职业妈妈部落，他们的依恋育儿部落，他们的尿布部落。

Watch

最初是通过博客和虚拟小组与其他(通常是第一次)妈妈联系和支持的方式，演变成了植根于防御性的大声、令人厌倦的辩论。无论是尝试非药物分娩、母乳喂养还是配方奶喂养、接种疫苗、同床共眠、大哭一场、呆在家里还是在外面工作——许多人都认为对妈妈有正确的方式，对妈妈有错误的方式。

那么——万岁！——随着女性意识到，嘿，我可以按照自己的意愿养育子女，这些斗争遭到了强烈反对。你也可以！如果我们都互相支持不是更好吗？你做你！每个孩子都不一样！坚持使用任何有效的方法！

特别是，我记得这张来自康涅狄格州 的一群妈妈的照片。这些妇女成双成对，举着标语，宣称她们的育儿策略或哲学。一位母亲手里的牌子上写着:“我实行和平育儿法”；“有时我会对我的孩子大喊大叫，”旁边微笑的妈妈手中的牌子上写着。幻灯片以这样的方式继续:“我正在给我两岁的孩子喂奶”，“我从一开始就选择了配方奶。”除了“我让我的孩子想看多少电视就看多少电视”之外，“我严格限制看电视的时间。”诸如此类。

当我们意识到另一个人为人父母的方式对我们或我们的孩子影响很小(可能除了接种疫苗)，所以谁在乎呢？

然后疫情来了。世界各地的父母都不可能做出这样的选择，即实际上*可能*影响其他人的健康——他们年迈的家庭成员、朋友、邻居、同事、同学，甚至整个社区。特别是妈妈们，在冠状病毒期间，经常发现自己 [承受着精神负荷](https://www.today.com/parents/mental-load-coronavirus-pandemic-means-moms-take-more-t179021) 的冲击，妈妈们更有可能成为父母放弃自己的职业，在家管理育儿和教育事宜。

生殖精神病学家 Alexandra Sacks 最近告诉《纽约时报》 ,许多妈妈在谈到母乳和配方奶粉时能够采取的“你做你”的心态在谈到可能接触冠状病毒时已经减弱:

> 萨克斯发现了一个普遍的心理陷阱。“竞争和对他人的批评往往是不安全和不稳定的表现，”她说。疫情是两者的完美风暴:在内部，人们缺乏信心去驾驭一个新奇和不可预测的局面；从外部来看，他们缺乏急需的社会和结构支持系统。把这两件事放在一起，一个人会显化为羞辱妈妈者。
> 
> 现在检查你的信箱，但也要阅读潜台词:你是一个疲惫的母亲，渴望正常生活吗？你是一个会在日常工作之余疯狂地摆弄你孩子的在线课程的超级妈妈吗？拥有昂贵保姆或私人学习舱的特权妈妈？一个传统的妈妈愿意放弃你的事业去上家庭学校吗？

在朋友、家人和熟人之间的对话中，羞愧以微妙和公开的方式表现出来，当然也包括社交媒体上的对话。例如，你可能见过各种各样的“在疫情期间，我们不可能亲自送我们的孩子回学校”的评论，然后是“哎呀，有选择真好”的回应。

那些家长，比如我自己，已经做出了一些类似的选择(例如，混合学习还是全虚拟学习)，可能会发现自己在一个糟糕的选择和另一个糟糕的选择之间来回摇摆。当一个人面临从两个看似同样糟糕的选项中选择一个不那么糟糕的选项时，这个决定的不安全感会让一个人感觉有所防备，或者需要证明他们的推理是正确的。尤其是如果，比如说，他们最好的朋友或邻居或姐妹选择了相反的选择。

另一方面，那些因为孩子的学校缺乏选择，或者因为自己的工作或照顾孩子的情况而被迫做出决定的人，可能会对那些对今年剩下的时间有发言权的人感到不满。

通过疫情生活和养育孩子的一个结果是，我们可能会无意中开始感觉到好像我们在互相对抗。那些为了家人的健康和社区的更大利益而“格外小心”的父母，与那些不得不把孩子送到学校或托儿所的父母，因为如果他们不这样做，他们就会失去工作。家长们权衡了面对面社交的好处和感染新冠肺炎病毒的风险，选择送他们去学校——而家长们无法想象在疫情期间送孩子去教室。

无论我们做出什么决定，无论我们如何度过这段时间，现在的风险都非常高。比我们在布尿布或一次性用品、自制有机食品或震动食品之间做决定时更是如此。这些话题导致了一些炒作的“战争”，但我们也在那段时间里了解到，它不必如此。我们可以选择相互支持。即使赌注更高；*尤其是*当赌注更高的时候。

如果我们能摆脱我们的防卫，我们也能放下那些会让其他父母感到羞愧的过度解释。如果我们可以接受别人的决定不是对我们自己的评论，我们也可以同意我们都在尽力而为。
