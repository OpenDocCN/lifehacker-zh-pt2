# 如果你没有发酵粉或小苏打，试试这些替代品

> 原文：<https://lifehacker.com/if-youre-out-of-baking-powder-or-baking-soda-try-these-1843314311>

如果菜谱要求用小苏打，但你只有发酵粉，或者你有小苏打但没有发酵粉，你该怎么办？事实证明，是有选择的。你可以做快餐面包、饼干或曲奇。你甚至可以把它们都做出来，这在压力大的时候总是一个可以接受的选择。

### 碳酸氢钠

如果你没有小苏打，你可以用三倍于食谱的发酵粉。因此，如果食谱要求一茶匙小苏打，你可以用三茶匙发酵粉。发酵粉也含有一点点盐，所以将食谱中要求的盐减半也是一个好主意。

用发酵粉代替苏打粉的缺点是，你冒着烘焙食品变苦的风险，所以重要的是不要用太多。

### 发酵粉

替代泡打粉稍微复杂一点。如果你有小苏打，但是你没有发酵粉，你需要使用小苏打和一种酸，比如酒石。对于每一茶匙发酵粉，你需要用一茶匙酒石代替一茶匙小苏打。如果你没有酒石，你也可以用一茶匙发酵粉和一茶匙醋或柠檬汁的混合物来代替一茶匙发酵粉。对于液体酸，如醋或柠檬汁，将其与湿配料混合。

### 如何辨别你的小苏打或发酵粉是否还是好的

如果你正盯着橱柜后面的那包发酵粉或苏打粉，在你用它来做东西之前，很容易用 T2、T3、T4、T5 和 T6 来测试它。

对于发酵粉，在碗里放三汤匙温水，然后加入一小勺发酵粉搅拌。如果混合物嘶嘶作响，这意味着它仍然是好的。如果没有，那就意味着它过期了，应该扔掉。

对于小苏打，将三汤匙白醋倒入碗中，然后加入一茶匙小苏打搅拌。如果混合物冒泡，它仍然是好的。如果没有，就扔出去。如果事实证明你的发酵粉还是好的，但是你的苏打粉不好，或者相反，至少你知道如何用一个代替另一个。
