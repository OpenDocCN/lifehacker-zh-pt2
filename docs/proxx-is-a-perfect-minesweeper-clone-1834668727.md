# Proxx 是一个完美的扫雷艇克隆

> 原文：<https://lifehacker.com/proxx-is-a-perfect-minesweeper-clone-1834668727>

我一直不擅长扫雷，我想我上一次玩这个游戏的年龄肯定至少是我的一半。很难让*点击——你很好*、*点击——你死了*游戏玩法性感，但谷歌凭借其 [开源](https://github.com/GoogleChromeLabs/proxx) 迭代、[**Proxx**](https://proxx.app/)做了一件令人钦佩的工作。

Watch

Proxx 的有趣之处不在于它是游戏的更新版本，拥有华丽的图形和动画(就扫雷而言)。这并不是谷歌投入时间复活这款 Windows 3.1 时代经典的原因。这款基于网络的游戏的天才之处在于，它基本上可以在任何东西上运行:从你的智能手机到你藏在抽屉里积灰的老式 T9 功能手机。此外，这款游戏总共只有 100KB，在你能够与之互动之前只需要下载 18KB，并且它以每秒 60 帧的速度渲染。

“智能功能手机，如 KaiOS 手机，正在迅速普及。这些都是资源非常有限的设备，但我们尽可能使用网络工作者的方法使我们能够在这些手机上获得高度响应的体验。由于功能手机有不同的输入界面(d-pad 和数字键，没有触摸屏)，我们也实现了基于按键的界面，”该项目的开发者之一、谷歌的 [真理子·小坂](https://web.dev/proxx-announce) 写道。

要开始，只需点击 [Proxx 网站](https://proxx.app/) 并选择你的难度。如果你喜欢冒险，你也可以手动调整雷区的大小，并设置你想要多少个“黑洞”。(黑洞是你不想点击的“地雷”。)

如果你在过去的 30 多年里一直住在一个山洞里，而你不知道如何“打扫”，这里有一个快速入门。单击一个正方形。如果是地雷， [你输](https://www.youtube.com/watch?v=M5QGkOGZubQ) 。游戏结束。如果不是地雷，方块清空。如果周围没有地雷，邻近的方块也会被清除。最终，你会看到有数字的方块。这是围绕所述正方形的地雷的数量——无论是水平方向、垂直方向还是对角线方向。

如果你认为你已经找到了地雷的位置，你可以将底部的设置从“清除”切换到“标记”，并在你的网格上标记潜在的问题。(如果您在浏览器中玩，也可以右键单击来标记地雷)。标记一个地雷意味着当你切换回“清除”时你不会意外点击它，它将帮助你跟踪什么是和不是(可能的)地雷。继续清理和标记，直到你爆炸或找到所有的自由空间。享受扫雷这种永恒的挫败感。
