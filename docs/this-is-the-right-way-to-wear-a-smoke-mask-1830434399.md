# 这是戴防烟面具的正确方法

> 原文：<https://lifehacker.com/this-is-the-right-way-to-wear-a-smoke-mask-1830434399>

外科口罩不会保护你的肺部免受野火烟雾的伤害，所以如果你在烟雾弥漫的地区，确保你有合适的口罩，并且知道如何佩戴。

### 哪些面膜真的管用？

你需要一个能过滤掉小颗粒的口罩。寻找标有 P95、N95 或 R95 的那种，如果佩戴得当，它们可以过滤掉 95%的小颗粒。如果你发现更高的数字，如 P100，N100，或 R100，这些更好。

字母(P、N 或 R)描述口罩是否耐油，如果你用它来防止野火烟雾，这应该不是一个问题。你可以在 这里阅读 [更多关于掩膜评级的内容。](http://trainandcert.com/articles/osha-safety-articles/n95-and-r95-respirators-whats-the-difference/) 

你可以在药店、五金店和绘画用品店找到口罩。

以下面罩将*而不是*很好地保护野火烟雾:

*   外科口罩(那种看起来像一块正方形的布，耳朵上有环圈)是用来过滤你呼出的空气的。他们没有足够好的密封来保护你免受你呼吸的空气中的微粒的影响。
*   防尘口罩不能防护足够小的颗粒。这些通常是有一根带子的纸口罩，也可以被标记为“舒适”或“讨厌”口罩。

如果你没有合适的口罩，防尘口罩可能比什么都没有要好，因为空气中也可能有大颗粒。但是如果可以的话，给自己买一个合适的面具。

这些口罩不会持续很久，所以如果可以的话，多储备一些。如果口罩被灰尘堵塞，或者被撕破，或者被损坏得不再合适，你应该扔掉它。

### 如何戴口罩

如果你已经有呼吸问题，如哮喘或慢性阻塞性肺病，问一问健康专家你戴或不戴口罩是否会更好。通过面罩呼吸确实会让你的肺更难受。

首先检查你的面具有两条带子，而不是只有一条。(如果你买对了口罩，会的。)两条带子都绕在你的后脑勺上，一条在耳朵上方，一条在耳朵下方。面具本身适合你的鼻子和下巴。

接下来，确保你有一个好的印章。如果你的皮肤和面具之间有缝隙，当你呼吸时，烟雾会呼呼地进入，破坏了整个目的。调整面具，以获得良好的密封周围的边缘。如果你有胡子，甚至是胡茬，你可能很难找到合适的发型。他们也没有儿童尺寸的口罩。

### 面具不是完美的保护

口罩的一个缺陷是，当人们认为自己受到保护时，他们会在烟雾中停留更长时间。继续遵循其他预防措施；那是面具，不是魔法力场。

口罩只能防护空气中的微粒。烟雾也可能携带刺激性气体，而口罩不能过滤掉这些气体。

加州公共卫生部 [建议](https://www.cdph.ca.gov/Programs/EPO/Pages/protect_yourself_during_wildfire.aspx) 在空气质量不好的时候呆在室内，保持门窗关闭，关闭空调的新鲜空气入口，并考虑使用 HEPA 过滤器去除空气中的颗粒，以保持室内空气清洁。不要吸烟，不要点蜡烛，尽量不要用吸尘器打扫，因为那会扬起灰尘。
