# 如何在 Android 上使用谷歌助手的环境模式

> 原文：<https://lifehacker.com/how-to-use-google-assistants-ambient-mode-on-android-1841604179>

我们的智能手机可以显示大量信息，甚至在锁定时也能提供有用的快捷方式，谷歌助手的环境模式值得一试。它允许你使用手机的几个最好的功能，而不必解锁，节省了你的宝贵时间和用手指摸索着验证你的设备的烦恼。



环境模式基本上是一个增强的锁屏，在手机充电时出现。它显示当前的天气状况，即将到来的警报，和日历事件。您还可以控制音乐播放、使用 Google Assistant 或控制同步到手机的其他智能设备，而无需解锁您的设备。通过启动 Google 相册中保存的图片的幻灯片来设置各种屏幕保护程序，或者查看您收到的任何时间或推送通知。

联想、诺基亚、索尼、Transsion 和小米的其他几款运行 Android Oreo 或更高版本的设备已经提供了环境模式。一加最近还宣布，该功能将在一加设备 上推出——包括 OnePlus 3 和所有更新的型号——所以现在是熟悉环境模式设置过程的好时机。(很简单。)

### 如何在 Android 上使用谷歌助手的环境模式

1.  打开手机上的谷歌应用。一般预装在安卓设备上，如果没有的话可以从 Play Store 中 [抓取。](https://play.google.com/store/apps/details?id=com.google.android.googlequicksearchbox)
2.  启动应用程序，然后点击**“更多”**进入**设置>谷歌助手**，然后点击**助手**标签，向下滚动到**手机**。再次向下滚动到**“个性化”**部分，寻找**“环境模式”**选项。
3.  一旦环境模式设置打开，你所要做的就是你的设备，然后插入充电电缆。谷歌应用程序的环境屏幕应该会在你的锁定屏幕上自动弹出，你可以访问各种快捷方式、照片幻灯片和其他信息，而不必解锁你的手机。
