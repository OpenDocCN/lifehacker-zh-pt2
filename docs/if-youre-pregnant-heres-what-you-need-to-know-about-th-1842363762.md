# 如果你怀孕了，以下是你需要了解的冠状病毒

> 原文：<https://lifehacker.com/if-youre-pregnant-heres-what-you-need-to-know-about-th-1842363762>

去年的这个时候，我怀了我儿子 36 周。那是我第一次怀孕，我是一个极度循规蹈矩的人。不含咖啡因。没有午餐肉。不要在蹦床上跳。怀孕前，我喜欢做所有这些事情，但这不再只是我一个人的事了。我害怕外面的东西可能会伤害到内心可爱的小东西。

因此，如果你正在阅读这篇文章，请知道你的担忧是可以理解的，这种担忧可能因冠状病毒而加剧。你有很多事情要做，这个世界上也有很多事情要做。

### **怀孕要知道什么**

当你怀孕时，你的身体会有一些看得见和看不见的变化。其中一个看不见的变化是你的免疫系统变慢。你的身体正在做大量的工作来构建这个小人儿，这个构建过程的一部分经常会导致你的免疫系统自我抑制。

根据圣安东尼奥 Acacia OB/GYN 的 OB/ GYN 哈特姆·哈特姆，医学博士的说法，考虑到这一点，受抑制的免疫系统*可能*使孕妇更容易受到病毒性呼吸道感染，包括导致新冠肺炎的新型冠状病毒。

当讨论怀孕和新冠肺炎的时候，单词*可能*经常被使用。关于这种感染，尤其是它对孕妇的相关影响，还有更多需要了解。

哈特姆说:“根据现有的有限信息和少量新冠肺炎病例，孕妇患严重疾病的风险可能比普通人群高。

哈特姆说，由于信息的缺乏——以及所有的*月* 和*月***——**月，目前还没有专门针对孕妇的建议。

另一个可能？母亲能否在子宫内将新冠肺炎病毒传给婴儿。

“目前还不清楚，”哈特姆说。“有一些未经证实的报告称，婴儿在美国境外出生后检测呈阳性。我们不确定这些婴儿是如何被感染的，也不知道是发生在怀孕期间还是之后。”

清楚的是，在感染新冠肺炎病毒的母亲所生婴儿的有限确诊病例中，没有一个婴儿的感染检测呈阳性。这是好消息。

### **交付时要知道什么**

如果你的分娩日期快到了，医院听起来可能是你最不想去的地方，紧挨着飞机。再说一遍，你的担心是可以理解的，但也许没有必要。

如果一家医院接收了一名新冠肺炎病毒感染者，疾病预防控制中心就如何最好地降低风险和保护其他患者的安全提供了分步建议。这些步骤包括为受感染病人的到来做好准备；将病人安置在一个关着门的单间里；限制探视病人，并限制病人在医院内的活动。

如果新冠肺炎或者细菌让你重新考虑你的医院分娩计划，请记住美国妇产科学院(ACOG) 仍然认为医院是最安全的分娩场所。

### **当你的宝宝在外面的时候要知道什么**

一旦你的宝宝来到这个世界，下一阶段的担忧就开始了。新冠肺炎帮不上什么忙，但从我的经验来看，你会担心一切事情，从奶嘴是否足够干净到吸奶器是否安装正确。这只是领土的一部分。

当谈到这种冠状病毒和你的新生儿时，保持谨慎是好的，但也要意识到，出于某种原因，儿童比成人更不容易因新冠肺炎而出现严重症状。

考虑到这一点，如果你的病毒检测呈阳性或被怀疑感染了新冠肺炎病毒，你可以采取一些措施来减少接触婴儿的可能性。Hatem 建议父母在接触婴儿前洗手，如果可能的话，母乳喂养的父母在哺乳时戴上口罩。

“如果用吸奶器挤出母乳，一定要在接触任何泵部件之前洗手，”Hatem 说。“如果可行的话，也许可以考虑让身体健康的人来给宝宝喂奶。”

虽然陪审团仍未确定新冠肺炎病毒是否可以通过母乳传播，但在迄今为止的有限病例中，没有证据表明在母乳中发现了该病毒。

### **现在做什么**

虽然对这种病毒及其全面影响仍有相当多的未知，但怀孕新冠肺炎患者的确诊病例数量非常低。专家、政府和整个世界都在努力，希望保持低水平。

虽然在这种时候感到害怕是很自然的，但是要提醒自己，目前情况似乎对你有利。试着不要让任何事情剥夺你的新生宝宝带给你的 T2 的快乐。甚至新冠肺炎也没有。
