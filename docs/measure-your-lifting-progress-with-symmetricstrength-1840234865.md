# 用场地对称强度测量你的提升进度

> 原文：<https://lifehacker.com/measure-your-lifting-progress-with-symmetricstrength-1840234865>

武断的实力分类是愚蠢的。我知道这一点，但我一直在各种图表和工具上查找我的提升比较。最好玩，也最好玩，其中:。



完全清楚地说，*你*应该在训练时给*设定自己的*目标。例如，如果你想参加全国力量举重比赛，你可以查一下你需要多少总成绩，并以此为目标努力。或者，如果你只是喜欢变得更强和打 PRs 的感觉，你不需要别人的数字来监控你的进展。

但有时看看你过得怎么样很有趣。像 SymmetricStrength 这样的工具很酷的一点是，你可以用它来直观地了解你的进度。拍一张它今天如何评价你的截图，明年这个时候再拍一张。

它是这样工作的。你告诉它你训练什么样的托举，并为每种托举输入你最好的设置。(这可能是一个最大努力，但也可以与你最好的五个一起工作。)然后，你会得到一张图表，不仅显示你的托举与其他人相比如何，还显示了你不同肌肉群的力量。

该工具对你、你的托举和你的肌肉进行分级，等级范围从“低于标准”到“世界级”。不要把那些分类看得太重；它们来源于举重标准和世界纪录，以及 ExRx 图表。(ExRx )

在过去的一年里，我把自己的分数(完全随意，毫无意义)从“新手”提高到了“精通”。在随附的图表中，我的托举相互比较，引体向上是我两次最强的托举。我的深蹲是我去年最差的一次托举；现在它被拉到了前面，而顶置印刷机是落后的那个。如果你没有竞争目标，你可以用这张图表来决定要关注的提升。嗯，一年来我能提高我的媒体多少？

看看你的排名，但是记住，这只是为了好玩。
