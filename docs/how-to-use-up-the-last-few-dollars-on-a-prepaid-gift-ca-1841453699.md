# 如何用完预付礼品卡上的最后几美元

> 原文:[https://life hacker . com/how-to-use-up-the-last-number-on-a-pred-gift-ca-1841453699](https://lifehacker.com/how-to-use-up-the-last-few-dollars-on-a-prepaid-gift-ca-1841453699)

去年圣诞节，我收到了一张价值 50 美元的 Visa 礼品卡，我花了整整 48 美元买了四双新袜子(因为我现在是那种圣诞节想要袜子的成年人)。

然后我必须想出如何处理卡上剩下的 2 美元。我应该在接下来的六个月里把这张礼品卡放在钱包里，期待有一天我需要买一瓶水或一包口香糖吗？我是不是应该扔掉这张卡，并认为自己很幸运，能够使用其总价值的 96% ？

这些预付礼品卡的问题在于，它们对于网上购物并不太有用。如果我在我的购物车里添加了第五双带脚趾的袜子，总数就会超过 50 美元，我就不能用我的 Visa 礼品卡购物了。

为什么？因为在线零售商不把这类礼品卡理解为“礼品卡”——也就是说，他们不允许你选择用预付 Visa 礼品卡支付部分总额，而用信用卡或借记卡支付部分总额。如果你想使用预付 Visa 卡(或预付 MasterCard 卡，或任何其他类型的预付卡)，你必须准备在卡上完成全部购买。

这样我就剩下 2 美元了。我该拿它怎么办？

我是这样做的:我从亚马逊买了一张 2 美元的网上礼品卡。

我从雷切尔·米勒的博客中得到了这个想法，在博客中，她描述了她的 [“利用 Visa/Amex 预付礼品卡上的每一分钱的小技巧:](https://www.justgoodshit.com/blog/visa-american-express-amex-mastercard-prepaid-gift-cards-balance-amazon-hack)

> 去年收到一张大额预付 Visa 礼品卡后，我厌倦了总是查看余额，我想出了一个非常简单易行的变通办法，让你可以实际使用所述卡上的每一分钱:**一旦礼品卡上的金额少到你不想动的地步，你就可以去亚马逊为自己买一张电子礼品卡，金额正好是 Visa 礼品卡上的金额。因此，如果 Visa 卡上有 7.83 美元，你可以给自己买一张 7.83 美元的亚马逊礼品卡。**

我自己试过，米勒是对的:你可以购买一张亚马逊电子礼品卡，金额几乎不限，不管是 7.83 美元，还是我的 2 美元。然后，你所要做的就是使用预付卡支付——这是整个过程中最棘手的部分，因为你的亚马逊肌肉记忆会提示你点击或点击“立即购买”(这将把礼品卡计入你的默认支付方式)，而不是“添加到购物车”(这将允许你输入新的支付方式，即你的预付卡)。

这个技巧非常有用，以至于 Lifehacker 的 David Murphy 建议在你开始花钱之前，把你的预付礼品卡变成亚马逊礼品卡。

也就是说，我很高兴自己没有把 50 美元的 Visa 卡变成亚马逊礼品卡，否则我就无法在仁济消费 48 美元。但是把剩下的 2 美元用于我的下一次亚马逊购物很适合我——我建议你下次在这些预付卡上还剩几美元的时候也这么做。
