# 伊利湖的水危机如此严重，以至于这家啤酒厂在啤酒中添加了藻类

> 原文：<https://gizmodo.com/lake-eries-water-crisis-is-so-bad-this-brewery-is-putti-1829796809>

**俄亥俄州托莱多—** 莫米湾酿酒公司的员工发誓说，该酿酒厂有 150 年历史的大楼闹鬼。一个酒保确信她看到一个玻璃杯自己漂浮着。另一个人发誓说，她看到一个手骨瘦如柴的女人，她一转身就不见了。至少看门人认为鬼魂是友好的。不用说，员工们已经熟悉了超自然现象，甚至可能已经学会接受它。

看看他们最新的季节性啤酒就知道了。

这是一首歌颂环境保护主义者多年来努力消除的怪物的颂歌。然而，这只野兽并没有在酒吧的黑暗地下室里躲藏。它每年夏天都会在伊利湖上抬起头，这是啤酒厂的水源。我说的是湖中的恶魔般的藻类大量繁殖，或者，现在用一种活泼的酸麦芽酒纪念的来自藻类大量繁殖的生物。

伊利湖的藻华是由过多的蓝绿藻或蓝细菌造成的。俄亥俄州环境委员会(Ohio Environmental Council)西北地区主任尼古拉斯·曼德罗斯(Nicholas Mandros)说，它们在夏季出现，当时春雨将农业和暴雨径流中的氮和磷引入湖中，该委员会与该啤酒厂合作生产这种啤酒。在夏天，有充足的阳光和温暖来保持藻类的健康和茁壮成长。这给依赖湖水作为饮用水的 1100 万 T2 人带来了麻烦。

在伊利湖，藻类大量繁殖在过去的 20 年里一直是一个问题，但在过去的 10 年里，情况变得更加糟糕。2014 年，这一问题达到了新的高度，当时水中有毒的蓝绿色藻类微囊藻毒素的含量变得如此之高，以至于不能安全饮用或酿造啤酒。在托莱多，居民倒霉了三天。啤酒厂及其附属餐厅也关门了。

“微囊藻毒素是最有害的物质之一，所以你不需要在水中摄入大量的微囊藻毒素就能感受到它的影响，”Mandros 告诉 Earther。潜在的健康影响包括呕吐、恶心、腹泻和肺炎等等。

这种反复出现的健康危害激发了莫米湾新的绿色气泡啤酒。酿酒师克里斯·克拉克去年开始精心制作酸味双 IPA 的配方。最终，他找到了抹茶、猕猴桃和无毒海藻的完美平衡，可以安全食用，这给了它绿色的色调。这种啤酒在八月份首次亮相，正好赶上该湖一年一度的花开。

这是完美的夏季饮品:足够强烈，让你兴奋不已，但又有如此清新的果味，以平衡其大量的啤酒花。

“啤酒有助于在种植我们需要的作物和维护我们的淡水资源之间找到平衡，”克拉克在我亲自品尝啤酒后告诉我。

对于热爱啤酒和环境的人来说，这是一个令人沮丧的时期，不仅因为藻类大量繁殖威胁着啤酒厂所需的水源，而且因为气候变化也可能威胁到制造啤酒所需的大麦。气候变化也助长了这些藻类的生长，因为海水变暖，降雨量增加，这有助于更多的营养物质最终到达湖泊。

简而言之:情况糟透了。

但不一定要这样。我们的行动可以减少助长全国藻类大量繁殖的养分流失量，我们还可以采取措施遏制全球变暖。这就是为什么莫米湾酿酒厂的优秀员工首先调制了这种饮料:提高对问题的认识并鼓励行动。

该公司的销售和营销经理香农·莫尔说，在很大程度上，顾客喜欢啤酒。其他啤酒厂已经伸出手来学习更多关于莫米湾如何处理湖泊的水华，并把啤酒加到他们的水龙头上。

“最终，我们的目标是永远不再酿造啤酒，”莫尔在酒吧里告诉我。大约在夏季，湖水会盛开，这是一年中唯一可以买到啤酒的时候。没有花开，就没有啤酒。酿酒厂一直供应啤酒，直到他们的最新一批啤酒售完，当我本月早些时候去拜访他们时，他们的啤酒快用完了。

这种酸的双 IPA 很好吃，会被错过，但失去一瓶啤酒总比失去所有啤酒好。它的消失意味着我们离保护伊利湖又近了一步。这是值得庆祝的事情。
