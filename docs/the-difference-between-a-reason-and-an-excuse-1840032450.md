# 理由和借口的区别

> 原文：<https://lifehacker.com/the-difference-between-a-reason-and-an-excuse-1840032450>

“我他妈的受不了那些不知道‘理由’和‘借口’区别的人，”[redditor raspberrykoolaed](https://www.reddit.com/r/AskReddit/comments/e18g6m/if_ok_boomer_was_a_digital_assistant_like_ok/f8or8ci/)说的是那些令人反感的人，他们不能同情任何他们自己没有面对过的问题，认为其他人都只是在提供借口。你不会想成为那种人的。但你也不想成为一个哭哭啼啼的失望者。当你能从借口中分辨出原因时，你的整个生活将会改善。



我说的“借口”指的是一个糟糕的借口，一个为逃避指责或努力而提出的合理化。我说的“理由”是指一个能充分解决对方异议的解释。它们都被用来为一种行为辩护，但只有后者真正这样做。

我们有时都会找借口，为了挽回面子或试图摆脱困境，或者因为我们真的不想做解决问题的工作。但我们也提供理由，因为我们不是万能的，有时候别人对我们的要求超过了我们所能提供的。

如果你只看原因，而认为它们是借口，你最终会过多地责怪别人，对他们期望过高。例如，您可能会:

*   对服务人员来说是个混蛋
*   认为压迫不存在，因为你没有面对它或者你找到了个人的解决方案

*   即使是最小的错误也要责备自己，因为你总是把自己比作完美
*   用一家资金雄厚的初创公司“扰乱”一个行业，用你的短期现金流威胁优先考虑长期可持续发展的企业，然后看着它崩溃，因为你知道为什么“旧方式”是正确的方式
*   给出可怕的建议

如果你看到借口并认为它们是理由，你可能会:

*   经常迟到
*   让人们把你当屎一样对待
*   和一个永远不会让你感到特别的伴侣在一段不满足的关系中度过数年，因为他们除了你之外忙于他们生活中的一切，而你所有的朋友都在背后说你的闲话
*   被骗
*   被解雇

区分理由和借口并不容易。它们没有明确的定义，在一种情况下是借口，在另一种情况下是充分的理由。但是每当你试图从借口中分辨出一个理由时，就这样做:

*   做你的研究:如果你不明白你收到的第一个原因，深入挖掘。 [问五次“为什么”。](https://lifehacker.com/ask-why-five-times-to-get-to-the-root-of-any-problem-1755399935)T3】
*   假设对方是真诚的，直到有必要这样做。这给了你改变主意的空间，也给了他们改变主意的空间。
*   想想为什么有人给你找借口。有时你会看到他们真正的目标——比如试图挽回面子——你可以帮助他们做到这一点，同时还能从他们那里得到你需要的东西。
*   寻找相反的动机，那些让双方在某个需求是否合理的问题上产生分歧的动机。寻找取代对立动机的共同动机，看看你是否能根据这些动机来重新构建情境。
*   定期重温你的假设。与和你有*不同经历的人*一起检查你的假设是否仍然成立。做好学习的准备，而不是去捍卫自己之前的立场。

这个过程是双向的，因为你把它应用到你自己的一面，也应用到对方的一面。假设你的老板不听你错过最后期限的正当理由。问“为什么”你的原因，你可能会发现你可以通过解决根本问题来克服它们。但是也要想想你的老板可能不理解你的理由的原因——他们是否没有考虑到你的工作量，或者他们只是觉得需要维护权力。你可能需要更新他们关于你其他工作的信息，或者找到一种方法让他们感觉是在负责，同时仍然给你更多的时间。如果你是老板，你认为你的员工很懒，那么整个过程都是一样的。

如果你总是看到同一个人要求高或懒惰，或者你看到一大群人要求高或懒惰，那么你需要使用这些策略。有时候，事实证明你是对的——资本主义的运行依赖于老板们的过分要求，以及 [榨取你剩余劳动力的价值](https://www.marxists.org/glossary/terms/n/e.htm#necessary-labour) ！有时候你是错的——就像有人认为无家可归的人应该“找份工作”

你无法明确证明理由和借口之间的区别。(如果有人能做到这一点，就不会有竞争或错失机会了。)但是，如果你练习检查这种差异，你会更好地与人相处，更经常地得到你想要的东西，甚至对自己设定更好的期望。你也不会让整个 Reddit 帖子 都知道你是个混蛋。
