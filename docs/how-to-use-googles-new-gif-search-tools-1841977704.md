# 如何使用谷歌新的 GIF 搜索工具

> 原文:[https://life hacker . com/how-to-use-Google s-new-gif-search-tools-1841977704](https://lifehacker.com/how-to-use-googles-new-gif-search-tools-1841977704)

谷歌发布了几款新的 GIF 搜索工具，应该可以让你更容易找到你想要的动画图片。用户可以使用这些工具——使用来自 Tenor GIF 应用程序的搜索数据——更准确地缩小搜索标准，并根据他们想要的心情或背景获得建议。N o 更多的梳理不相关的图片，忙乱的搜索词汇，以找到正确的 GIF。

你会在这篇文章 中找到所有的搜索工具 [。它们使用简单——只需点击下拉框，输入你想要搜索的名字、动作或心情——然而，每个工具都有自己独特的搜索功能。下面是他们的做法:](https://medium.com/@smfrogers/emoting-with-pop-culture-44091bfe3d75) 

*   前两个搜索框显示了哪些情绪、行为或情感与某个特定的名人或虚构人物最为相关。第一个框以堆叠框的拼贴形式列出了排名，另一个框按排名顺序列出，最受欢迎的列在顶部。
*   下一个框让你基于一个特定的动作或情绪进行搜索，以查看该类别中最常用的 gif 和最受欢迎的名人。
*   最后一个工具可以显示一系列基于情绪谱的名人 gif 图——快乐/悲伤和笑/哭——正面的 gif 图在右边，负面的在左边。

目前，这些工具似乎只在 medium post 上可用，但它们有望在未来进入 [Tenor 应用程序和网页](https://tenor.com/) 。与此同时，它们有助于找到哪些搜索结果效果最好，而且看到某些人物和名人与哪些情绪或活动相关联也很有趣。
