# 你应该创造一个家庭格言

> 原文：<https://lifehacker.com/you-should-create-a-family-motto-1834331300>

几年前，当我们是一个九岁男孩的养父母时，我丈夫和我实施了一条家庭格言。这是在一次寄养父母培训会议上向我们一群人提出的一个建议，这是养育有创伤的孩子时经常使用的一句口头禅:“没有伤害，团结在一起，玩得开心。”

Watch

我们惊讶于它适用于如此多的情况。我们不会伤害他人，也不会伤害自己，无论是身体上还是情感上。我们在人群中团结在一起*和*我们互相支持。如果我们应该玩得开心，但我们没有，我们会想办法扭转局面。(我们的座右铭甚至带有手势，让它变得更棒。)

我特别喜欢有座右铭的一点是，它把重点从家庭规则转移回 T2 的家庭价值观。少一点“这是我们的工作”，多一点“这是我们工作的原因。”

如果你想更进一步， [作家兼播客鲍勃·莱利](https://99parent.com/how-teach-child-to-right-thing/) 建议写一份完整的家庭法典，他解释说， “可以作为他们看世界的一面镜子”，让他们决定“哪些事情符合他们的价值观，哪些事情不符合他们的价值观。”

或者你可以像作家丹尼尔·平克的家人一样，编造一个首字母缩写词。他们的是 HAHU，代表着忙碌、期待、抬头。“它最初是为我们带着三个小孩旅行而设计的，” [他告诉我们](https://offspring.lifehacker.com/im-author-daniel-pink-and-this-is-how-i-parent-1833367846?_ga=2.123702347.1584412397.1555298157-1940201396.1552611778) 。“当你在机场时，你必须*奔忙*。你不能混日子。你必须预测接下来会发生什么。而*抬头*意味着你必须看着你要去的地方，尽量不要撞到人。它已经成为我们家经久不衰的首字母缩略词——我们一直在使用它。”

我问我们的 [后代脸书集团](https://www.facebook.com/groups/2018785615043946/) 中的父母，他们是否有任何家庭格言或咒语，他们中的几个人有，他们通过各种方式找到了自己短语的灵感。克洛维斯的家庭受到她的传统的启发:“我们实际的家庭格言(苏格兰家族格言)吸引了我的孩子:Numquam obliviscar。我永远不会忘记，”“蒂芙尼承认从 [*借用了她家的座右铭“我们走吧卢娜*](https://pbskids.org/luna) *:* “总有更多要学的，总有明天。”布兰登的家庭格言摘自他姑姑写的一篇祈祷文:“带着爱生活。愉快地工作。分享你所拥有的。永远心存感激。”

创造你自己的座右铭，想想你希望它体现的价值。对你来说什么是重要的？辛苦？毅力？善良？让你的孩子参与进来，这样他们会有一种主人翁的感觉，然后选择一个简短的、与各种情况相关的短语，然后脱口而出。下面是 [一个很棒的家族格言列表，从](https://www.organizedmom.net/inspiring-ideas-new-family-motto/) 到让你开始。

一旦你选择了你的座右铭，把它展示在冰箱上，一块写字板或黑板上——任何人都能看到的地方。一年左右重温一次，看看这句格言是否还在引起大家的共鸣，或者是否应该更新。

如果你很难找到一个适合你家庭的座右铭，你总是可以采纳后代小组成员约书亚的建议:“嗯，随便吧。我们会想出办法的。”这句话在很多情况下也适用。
