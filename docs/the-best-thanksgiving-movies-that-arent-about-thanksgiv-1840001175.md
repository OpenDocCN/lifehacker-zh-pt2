# 与感恩节无关的最佳感恩节电影

> 原文:[https://life hacker . com/the-best-thanksgiviv-1840001175](https://lifehacker.com/the-best-thanksgiving-movies-that-arent-about-thanksgiv-1840001175)

感恩节就要到了，对于我们这些在节日期间避开家人的人来说，这意味着这是一个专门盯着你的电视或笔记本电脑屏幕几个小时的日子。如果你没有节日的精神，也许因为你是一个人度过的， [我们得到了](https://lifehacker.com/how-to-cope-with-spending-the-holidays-alone-1742026483) 。尽管如此，你还是不想完全忘记*那一天，对吗？* 

Watch

下面你会发现几部你应该在这个感恩节看的电影(没有额外的，不断提醒你应该和朋友或家人一起过感恩节)。有时候，最好的假期恰恰是那种你看不到任何人，舒适地躺在沙发上，汗流浃背的假期。

### *亚当斯家族价值观*

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-rVQqQuOO9yQ&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-rVQqQuOO9yQ&start=0) 

如果你正在寻找一些不那么庆祝或快乐的东西，亚当斯的家庭价值观是一个该死的宝藏，如果只是对琼·库萨克和克里斯蒂娜·里奇来说(由于这部电影，我仍然在使用大量的[gif](https://media.giphy.com/media/rNkfrxkwKzOAU/giphy.gif))。利玛窦扮演星期三亚当斯，他在夏令营时，在重现第一次感恩节大餐时发动了政变(尽管非常暴力)。

### *贝多芬*

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-ki8wHMR-yOI&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-ki8wHMR-yOI&start=0) 

《贝多芬》是一部关于一只巨大笨拙的狗的电影，它也是一种天才？被一个宠物店小偷抛弃，贝多芬被一个家庭收养，不出所料地破坏了感恩节晚餐，以及随后发生的其他狂欢，因为这个家庭多年来不知何故忽视了对他的训练。

### *飞机、火车和汽车*

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-_akwHYMdbsM&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-_akwHYMdbsM&start=0) 

史蒂夫·马丁是一个偶像，*飞机、火车和汽车，*由约翰·休斯*执导，*就是证明。如果你还没有看过，马丁的角色试图在几次可预见的交通事故后及时赶回家过感恩节。这部电影还以《T4 孤家寡人》和《酷跑》中已故的约翰·坎迪为特色。没有剧透，结局是苦乐参半，我想我要哭了？

### *你收到了一封邮件*

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-jsjZ4ZqlG4w&start=18](https://lifehacker.com/embed/inset/iframe?id=youtube-video-jsjZ4ZqlG4w&start=18) 

这是 1998 年。克林顿是总统。姜辣离开辣妹。《你有邮件》首次亮相，由汤姆·汉克斯和梅格·瑞恩主演。这部电影正好包含了一个著名的感恩节场景，在这个场景中，瑞安的角色在 Zabar's 购物时信用卡被拒绝，汉克斯必须前来救援。

### *家族之石*

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-3lgcViU45Fs&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-3lgcViU45Fs&start=0) 

由黛安娜·基顿、瑞秋·麦克亚当斯、莎拉·杰西卡·帕克和克莱尔·丹尼斯主演的《T2 家族之石》是一部关于一个女人在节日期间不顾一切地试图赢得男友家人的电影。《T4》实际上没有一部发生在感恩节前后，但是我强烈建议你，读者，还是要看这部电影。它已经成为一部狂热的电影，因为大部分角色都不太讨人喜欢，然而，它却是一部经典。在一群都是混蛋的角色中，你到底支持谁？太神奇了！
