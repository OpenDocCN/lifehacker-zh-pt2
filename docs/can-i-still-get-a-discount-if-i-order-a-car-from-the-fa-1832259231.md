# 如果我从工厂订购一辆汽车，我仍然能得到折扣吗？

> 原文：<https://jalopnik.com/can-i-still-get-a-discount-if-i-order-a-car-from-the-fa-1832259231>

作为 Jalopnik 的常驻购车专家和专业购车者，我会收到电子邮件。很多邮件。我决定挑几个问题，试着帮帮大家。本周，我们正在讨论一份新的沃尔沃 V60 订单的谈判，对一辆本田思域进行改装，并从加拿大进口一辆小汽车。

首先:如果你从工厂订购了一辆新车，你还需要支付全部标价吗？

> 所以，沃尔沃现在似乎只能通过在线订购的方式销售 V60。看起来有点像特斯拉，但目标是更喜欢选择的 V60 客户。我在想。你还能与经销商谈判并获得交易/折扣吗？该网站要求先支付 2000 美元，其余的在交货时支付，但它也声明“您的沃尔沃礼宾员将在 1-2 个工作日内与您联系，以确认您的订单。”

沃尔沃似乎确实在向特斯拉学习订购流程。我们去年报道过，至少在第一年， [新款 V60 四轮拖车将只接受订单，](https://jalopnik.com/the-2019-volvo-v60-wagon-starts-at-39-895-will-be-fact-1830818230) ，不在经销商处备货。

老实说，我不熟悉如何使用这个门户网站进行价格谈判。然而，我为客户做过一些工厂订购的沃尔沃，我和他们谈判的方式就像我在停车场谈判一辆车一样。我联系经销商，告诉他们客户想要什么，让经销商给我报价。价格最低的经销商获得订单。由于最终购买是通过经销商完成的，不管这个在线门户网站，这是我推荐订购 V60 的方法。

我还会指出，根据市场的不同，你可能不会在工厂订单上获得折扣，这完全取决于特定汽车的供求经济状况。有时候，像保时捷 911GT3T3】这样的热门汽车，获得订单是成就 T5】而不是价格。 

接下来，给一辆新本田思域装上 ECU 调校会有什么风险？

> 大概五个月前买了一辆 2018 款本田思域 Sport，很喜欢。在我购买它之后，我在网上闲逛，偶然发现了几个不同的本田思域专用网站，它们讨论了使用 Hondata 或 Ktuner 对它们进行调整是多么容易。这两个调谐器似乎都工作得很好，很容易安装，但我真的不能辨别什么可能(如果有的话)的负面影响，这些电子调谐器会对引擎。我猜问题是，他们是安全的，还是我应该期待这些调谐器提供的额外马力会过早地磨损发动机和涡轮？
> 
> 我猜想在一个相关的注意，如果这些调谐器是安全的，为什么在世界上没有本田出售他们的汽车已经这样调整？

正如你可能意识到的，这些曲调改变了电脑的程序来增加能量。本田的工程师设计这种电机是为了理想地处理一定量的功率，以平衡可靠性。

发动机可以“处理”更多的功率，但部件可能会磨损得更快。也可能不会。本田基本上可以出售已经调校过的汽车，但动力更强的思域跑车将与更昂贵的思域 Si 竞争。汽车制造商希望为他们的目标客户找到性能、价格、燃油经济性和耐用性之间的中间地带。

关于“这些曲子安全吗？”这完全取决于公司。我敢肯定，你可以找到几个论坛线程与业主讨论他们的经验，各种曲调来自不同的公司。

虽然调谐不会自动使您的保修失效，但是如果您确实调谐了它，并且您需要为您的电机或变速器提出保修索赔，本田可以辩称售后调谐导致了故障，从而拒绝保修索赔。

最后，你能从加拿大买到一辆全新的汽车吗，即使它与美国的车型一模一样？

> 我对新款 Veloster N 感兴趣，但在美国，它不会配备加热座椅。然而，在加拿大，它将配备加热座椅。对于这样的新车，在加拿大购买并进口到美国容易吗？还是购买座椅并安装更容易？谢谢你的信息。

如果一辆汽车与美国的规格版本相同，那么允许一个美国人从加拿大购买一辆汽车是合理的，但这不是它的工作方式，因为我们的汽车进口法律是荒谬的。但是，即使你能从我们北方的朋友那里得到你的 Veloster N，仅仅是加热座椅的整个过程就可能是困难和昂贵的。

售后加热座椅相当容易和负担得起，所以问一下你当地的商店，看看他们能做什么。
