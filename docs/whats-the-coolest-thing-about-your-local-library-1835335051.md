# 你们当地图书馆最酷的是什么？

> 原文：<https://lifehacker.com/whats-the-coolest-thing-about-your-local-library-1835335051>

这里有图书馆，你可以借到木偶、仪器、望远镜、装裱好的艺术品、求职面试用的领带，甚至是种植 T4 的种子。L 图书馆主持讲座，音乐会故事时间，工作坊，学习班；他们分发书籍、杂志、报纸、DVD，甚至电子游戏。他们可能主持护照申请办公室或其他政府服务。他们可能会让你免费进入博物馆，或者进入像 Kanopy 或 Hoopla 这样的数字电影图书馆。但是每个图书馆系统的服务都是不同的。你在当地图书馆能得到或做的最特别的事情是什么？

有一句老话是这样说的，“如果图书馆还不存在，它们今天也不会被批准。”奇怪的是，人们并没有说图书馆是个坏主意。这是因为他们对社会如此有益，如此有效地利用公共资源来服务整个社区，帮助人们自助，如果他们还没有存在，我们永远也不会想到他们会如此令人惊叹。

大多数图书馆随着技术和文化发展得很好。他们更新他们的资料，他们让顾客从他们的电脑和手机上找到、保留甚至消费资料，他们提供社区最需要的节目。图书馆员可能是你有幸咨询过的最友好、最乐于助人的政府雇员。

你当地的图书馆系统提供了什么让你惊讶或与众不同的东西？他们为你的社区做的最好的事情是什么？你能借到的最神奇的东西是什么，或者你能在那里做的最酷的事情是什么？你的图书馆提供数字服务吗？
