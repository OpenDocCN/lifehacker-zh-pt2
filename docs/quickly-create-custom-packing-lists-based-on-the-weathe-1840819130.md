# 使用此应用程序，根据目的地的天气快速创建定制装箱单

> 原文：<https://lifehacker.com/quickly-create-custom-packing-lists-based-on-the-weathe-1840819130>

去年，我乘飞机旅行了 15 万多英里。我还准备了很多行李箱、手提行李和旅行包，为那些旅行做准备，有时是在我去机场前几个小时或几分钟。



几年前的某个时候，我做了一个基本的打包清单，让这一切变得容易多了。我每次旅行都会带很多东西。我列了一张清单，每次出门前都会参考一下，确保我记得把包里的东西都扔了。想想药物、充电线、我的便携式热点和耳机。这是我绝对推荐的做法。

我的衣服是每次旅行的百搭单品。

我住在旧金山，那里几乎全年只有一个季节。每当我旅行的时候，这意味着我要去哪里，那里的天气如何，然后弄清楚去那里我到底需要带些什么。

[PackPoint](https://www.packpnt.com/) 是一款可以帮忙的 app。我们曾在 2014 年 简要介绍过这款应用，当时它还是新产品，只能在安卓和黑莓手机上运行。六年后，它可以在 iOS 上使用，并为 Android 用户提供了大量新功能。

在基本层面上，该应用程序会问你要去哪里旅行，什么时候去，呆多长时间，然后根据你在那里的天气情况为你的旅行制作一份打包清单。

它还会问一些关于你在旅途中可能会做什么的问题，比如滑雪，去海滩，或者去吃大餐，以及你在逗留期间是否可以洗衣服。

2.99 美元的高级版本集成了 TripIt 和 Evernote，让你可以保存自己的定制活动，例如商务会议或夜店。

对我来说，有时这个应用程序会建议我带些什么，有时它会建议我绝对想带的东西，比如远足用的杀虫剂，但可能会从我的清单中删除。

装箱单可以为你定制，也可以与你同行的其他人共享。

当真正到了打包的时候，你也可以直接在手机上检查清单上的项目，这样你就知道你已经得到了所有的东西。

当你开始考虑为一次长途旅行打包什么东西时，这可能是一个很好的起点，并且可以帮助你确保不会把重要的东西落下。
