# 在特定城市以每小时 50 美元的价格租一辆优步

> 原文：<https://lifehacker.com/rent-an-uber-for-50-an-hour-in-select-cities-1843779787>

新冠肺炎疫情在很多方面改变了我们的生活，包括我们的出行方式。对于没有车，依赖公交的人来说，更是如此。四个月前，我们可能会乘地铁去办事，或者去美国国家铁路客运公司进行一次短途旅行，但由于有限的时间表和传播病毒的可能性，这些不再是许多人的选择。但现在，美国某些城市的人们有了另一种可能:优步每小时一次。

Watch

从每小时 50 美元开始，你可以要求一个优步司机的旅程，持续 7 个小时。用户像往常一样通过优步应用程序订购顺风车，只需选择“每小时”选项，而不是 UberX、UberXL 或优步拼车。

### 关于这项服务需要了解什么

当然，还有很多附属细则。从 6 月 2 日(星期二)开始，这项服务将仅在亚特兰大、芝加哥、达拉斯特区、休斯顿、迈阿密、奥兰多、费城、凤凰城、西雅图、塔科马和坦帕湾提供(至少目前如此)。此外，50 美元/小时价格中包含的里程数因城市而异，所以 50 美元在芝加哥可能不如在凤凰城那么远。每小时的费用也不包括过路费、小费或其他附加费，如增加一个车座、。

骑手运营总监 Niraj Patel [在一份声明](https://www.uber.com/newsroom/hourly-booking/) 中表示:“我们为那些在处理重要任务时预计需要额外灵活性的骑手和司机建立了这一功能，以便他们可以获得额外的收入机会。”。“我们希望这项功能能让所有顾客在这个‘新常态’和充满挑战的时期过得更轻松。”

据 公司发布的消息称，在澳大利亚、非洲、欧洲和中东的城市成功试点该计划后，优步正在推出其新的每小时服务。优步时薪被宣传为一种在更长时间内拥有自己的车(嗯，技术上来说，是自己的司机)的灵活性——比如你需要跑腿，或者带心爱的人去医院看病——而不必在每个目的地都单独预订。

它还应该更舒适一点:选择优步小时制的人将与拥有更宽敞、更新车辆的司机匹配(即有资格享受优步舒适的那种)。只是 [别忘了你的面膜。](https://lifehacker.com/soon-youll-need-a-face-covering-to-take-an-uber-1843242345)T3】
