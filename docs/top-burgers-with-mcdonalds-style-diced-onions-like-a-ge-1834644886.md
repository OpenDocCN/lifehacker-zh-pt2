# 像天才一样用麦当劳式洋葱丁做的顶级汉堡

> 原文：<https://lifehacker.com/top-burgers-with-mcdonalds-style-diced-onions-like-a-ge-1834644886>

虽然我喜欢麦当劳，但我绝不会推荐它们为“你应该去享受美味汉堡的地方”麦当劳本身是一种东西，一种不完全是食物的东西。(你不是在吃鸡肉，你是在吃麦乐鸡。)但有几件事他们做对了，其中一件事就是在他们的芝士汉堡上切洋葱。

Watch

这有很多好处。首先，它促进了洋葱的均匀分布。如果洋葱被限制在自然形成的环状，它只能覆盖这么多的表面区域。如果你把洋葱圈摆好，让馅饼的每一部分都覆盖上等量的洋葱，你——很像这个句子——会有太多的洋葱。将洋葱切成小块可以让你精确地分散它们，加入适量的洋葱味。

然后是滑点的问题。取决于你的牙齿如何在你的头上排列，当你咬一口汉堡时，从你的汉堡上撕下一大块浇头可能是吃汉堡的一个不幸的副作用。有时候这些大块的洋葱会从你的牙齿上掉到你的乳沟里。如果洋葱是小块的，这就不会发生。结案了。

当切成小块时，在汉堡上做得更好的其他配料是泡菜(你也可以只用莳萝调味品)和生菜(大叶子喜欢滑落)。用这种方式将它们分开可以确保质地和味道的均匀分布，而且配料会留在你放的地方，直到你把它们放进嘴里。作为一个精确控制什么和什么时候东西进入我嘴里的粉丝，我认为额外的刀工作是值得努力的。
