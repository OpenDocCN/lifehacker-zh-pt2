# 使用此应用程序快速删除图片中的背景

> 原文：<https://lifehacker.com/quickly-remove-backgrounds-from-your-pictures-with-this-1843925087>

**iOS:** 在网上很容易找到关于虚化或者 [去除照片背景的指南](https://lifehacker.com/use-this-ai-tool-to-quickly-remove-the-background-in-im-1831287994) ，但是几乎所有的都需要一个桌面和图像编辑软件。然而，iOS 用户可以在手机上使用 PhotoRoom 应用程序来做这件事。



该应用程序会自动模糊背景，这样你就可以获得肖像模式的效果，即使在通常不具备该功能的设备上也是如此。它还能让你完全去掉背景，用其他图像替换，就像一个绿色的屏幕。由于该应用程序内置了调整阴影和添加其他图形元素的工具，定制背景实际上看起来非常有说服力，这些都是你通常会使用 Photoshop 或 GIMP 的东西——和 PhotoRoom 更容易使用。

PhotoRoom 在 iOS 商店 上免费提供 [，Android 版本和 web 应用程序也在开发中。用免费版编辑的图像会显示一个水印标志，但你可以通过升级到每月 9.49 美元或每年 46.99 美元的高级版来删除它。](https://apps.apple.com/us/app/photoroom-make-studio-photo/id1455009060)

### 如何使用照相变焦

1.  打开 PhotoRoom 应用程序来拍照。使用屏幕上的参考线来排列您想要保持焦点的对象。右边的按钮让你在不同的背景编辑之间进行选择，包括模糊它或者完全裁剪背景并用预加载的图像替换它。
2.  抓拍照片，应用程序会处理背景编辑。
3.  在编辑器中，添加并操作焦点物体下的阴影，并对背景做任何必要的进一步的改变。
4.  完成编辑后，保存照片。它现在可以上传或通过其他应用程序共享。
