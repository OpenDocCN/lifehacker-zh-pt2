# 如何换尿布

> 原文：<https://lifehacker.com/how-to-change-a-diaper-1843390285>

当然，很多人在成长过程中会有更年轻的兄弟姐妹或表兄弟姐妹，或者他们十几岁时会照看孩子，或者他们的朋友有孩子，或者他们只是有孩子。对这些人来说，换尿布似乎是一件不需要动脑筋的事情。但也有像我和我丈夫这样的人，他们从来没有这样做过，而是给我们自己的新生儿换了第一片尿布。

这是一件令人伤脑筋的事情，作为一个新的父母，被期望有这种瞬间的母性或父性本能*以及*知道如何换尿布。这是一件事后看来非常明显，但一开始又非常令人困惑的事情。所以首先，我将*告诉*你如何换尿布。然后我会*告诉*你如何换尿布。然后你会*知道*如何换尿布。

第一步:将婴儿放在干净柔软的表面上。如果地面很高，一定要把它们系好，并且一只手一直放在或非常靠近它们，即使它们还没有滚动(第一次滚动真的会让你大吃一惊)。

**第二步:**解开脏兮兮的尿布，然后遵循的这个战略建议[期待什么](https://www.whattoexpect.com/first-year/diapering/how-to-change-a-diaper/) :

> **对于湿尿布:**将脏尿布叠放在婴儿下面(这样干净的外面就在他或她的屁股下面了)并擦拭该区域。

> **对于便便尿布:**尽可能用尿布本身擦拭，然后折在下面，如上。抬起腿，用温水或抹布清洁宝宝的前部，确保进入所有的褶皱。然后抬起双腿，彻底清洁宝宝的屁股。

在此步骤中，请记住以下几点:

*   你总是需要从前到后擦女生。
*   男孩可以从任何一个方向被擦拭，但你会想要用额外的尿布或干净的布覆盖阴茎，以防止出现 [小便喷泉的情况](https://offspring.lifehacker.com/how-to-prevent-your-baby-from-peeing-on-you-1835940769) 。
*   当你扣紧干净的尿布时，你也要把阴茎朝下，以减少泄漏。

**第三步:**在宝宝干净干燥后(如果有必要的话，你已经涂过皮疹霜了)，把干净的尿布滑到宝宝的屁股下面。如果你用的是一次性尿布，紧固片在后面，在婴儿下面。将尿布的前部拉到婴儿的两腿之间，从尿布的后部到前部穿过腹部系紧拉环。

尿布应该系得足够紧以防止泄漏，但不要系得太紧以至于引起刺激。(如果你在下一次换尿布时看到红色标记，你就知道它太紧了。)
