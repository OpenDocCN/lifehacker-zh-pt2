# 做一堆调味盐，什么玩意儿

> 原文：<https://lifehacker.com/make-a-bunch-of-flavored-salts-what-the-hell-1842457195>

我的社交媒体目前充斥着面包、大圆饼和发面的照片。这很好，也有道理，但我的压力很少通过烘焙来缓解。然而，我确实喜欢沉浸式厨房项目，建造一个奇特的盐的收藏正是我们现在都需要的那种有趣、低风险、看似简单的任务。

### 塔可钟盐活墨西哥卷

花式盐是一种骗局，尤其是调味薄片。你可以用 [香草、柑橘皮和任何你常喝的](https://skillet.lifehacker.com/give-your-dishes-a-savory-boost-with-these-homemade-fla-1716413673) 酒来制作调味盐，但这种时候似乎需要混乱的调料，那就是 [塔可钟盐](https://skillet.lifehacker.com/make-taco-bell-salt-with-leftover-sauce-packets-1828393224) ，至少对我来说是这样的:

> 像任何用湿调味剂制成的调味盐一样，这个过程非常简单。简单地将酱汁和盐混合，在烤箱中脱水，然后粉碎。要做塔可钟盐，你需要:
> 
> *   6-10 包酱，取决于你喜欢盐的味道有多浓。六威尔给了你一种带有塔可钟香味的调料，但是你需要更多的调料来加热。
> *   1/4 杯普通食盐
> 
> 在食品加工机中混合调味汁和盐，直到你得到颜色均匀的沙状混合物。将盐摊在铺有羊皮纸的烤盘上，放入 200 度的烤箱中至少两个小时，直到盐完全干透。把它放回食品加工机，粉碎，然后把你最喜欢的垃圾食品的精华撒在松脆的零食上。

### 把这些废料利用起来

你知道吗，番茄皮、辣椒皮以及几乎所有粉碎的蔬菜碎片 都可以制成充满活力、味道鲜美的盐。嗯，它*可以*而你*应该*。上面的视频彻底解释了这一过程，但 [对于任何废料](https://skillet.lifehacker.com/use-bell-pepper-peels-to-make-paprika-salt-1824741759) 来说都差不多:

> 只需拿起果皮，称一下重量，然后加入等量的盐搅拌均匀即可。将长条放在铺有羊皮纸的烤盘上，放入 200 度的烤箱，直到它们尽可能干燥。在食品加工机中粉碎条状食品，制成精细的咸味粉末。如果盐看起来有点潮湿，不要担心，要让你的盐完全干燥，只需将盐铺在一张羊皮纸上，将羊皮纸放回烤盘上，然后放回 200 度的烤箱，直到它完全干燥。在食品加工机中再次研磨，或者用研钵和杵研磨成更细的粉末。

### 在你的慢炖锅里做一份片状精盐

用完马尔东——或者我喜欢称之为“零食盐”——可能看起来不像一场真正的“悲剧”，但这并不意味着它不令人不快。幸运的是，你可以用犹太盐、水和某种慢煮设备(我用的是速溶锅)制作自己的薄片状、松脆的晶体。

[这篇文章](https://lifehacker.com/make-flaky-finishing-salt-in-your-slow-cooker-or-instan-1831643616) 有所有的细节，但它只是简单的制作一个超饱和溶液，然后慢慢地把水赶走，让大而美丽的薄片形成。一克盐对四毫升水的比例很好，你可以根据需要放大或缩小。
