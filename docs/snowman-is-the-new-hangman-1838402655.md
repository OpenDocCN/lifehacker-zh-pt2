# “雪人”是新的“刽子手”

> 原文：<https://lifehacker.com/snowman-is-the-new-hangman-1838402655>

过去，当我们发现自己不得不打发时间的时候——这是经常发生的，因为没有手机——我们通过挂简笔画来练习拼写技能。

Watch

一方面，我们都玩过 Hangman，我们可能没有想太多，我们接触过其他更暴力的游戏，我们大多数人都很好。

另一方面，现在是 2019 年，我们的国家有一段黑暗和骇人听闻的私刑历史，*也许*我们可以选择一个更积极、对孩子友好的符号来帮助孩子练习词汇。

就像，雪人。

你玩雪人的方式和刽子手一样，只是步骤不同。 [教案](https://www.lessonplanet.com/article/teacher-education/a-politically-correct-version-of-hangman) 为我们分解一下:

> 在刽子手的游戏中，当一个孩子错过了密语的一个字母时，你在套索中画一个圆圈(代表头部)。接下来是一根长棍子(身体)，两根棍子(手臂)，两根棍子(腿)，两只眼睛和一张悲伤的脸。那总共是九个错误。当你玩雪人时，对于第一个不正确的答案，下面的(也是最大的)雪球被画出来，对于第二个，中间的雪球被加进去，对于第三个，一个雪球被加在顶部(雪人的头)。之后，雪人得到了两根棍子(手臂)，一顶大礼帽(就像雪人戴的那顶)，两只眼睛和一张悲伤的脸。瞧啊。九错！

你可以只坚持这九个项目，或者你可以从中得到乐趣，并得到你想要的细节。我记得在特别史诗般的回合中给我的刽子手添加衣服和鞋子——哇，我对自己感到畏缩——你也可以给你的雪人添加围巾、纽扣和胡萝卜鼻子。

另一个(非冬季)选项是“”太空人，玩家画一艘宇宙飞船。你也可以想出自己的角色——例如，如果雪人和宇宙飞船对你的孩子来说有点太可爱，就画一个“怪物人”。
