# 如何阻止机器人向您发送脸书好友请求

> 原文：<https://lifehacker.com/how-to-stop-bots-from-sending-you-facebook-friend-reque-1841166369>

自 1 月 1 日起，我每天早上都要清理十几个来自脸书的好友请求，都是我不认识的人发来的。



我不太清楚为什么所有这些伟大的新机器人，其中许多似乎不会说英语，突然对我的虚拟友谊感兴趣，但我知道他们的存在非常令人讨厌。

如果你和我一样，已经开始收到越来越多的垃圾好友请求，有一个快速简单的方法可以帮助你减少这些请求:改变你的隐私设置。

脸书有一个隐私设置，允许您指定谁可以向您发送请求。它默认让每个人，甚至是机器人，向你发送请求。

在您的桌面上，登录脸书，然后单击页面右上角的向下箭头。在那里，从下拉菜单中选择设置。

从左栏菜单中选择“隐私”,调出网站的隐私设置和工具。

在那里，点击“谁可以向您发送好友请求？”旁边的“编辑”按钮 t o tweak 谁被允许在你的好友名单上申请一个位置。

我建议将你的设置改为“朋友的朋友”，而不是“每个人”

假设你的朋友没有加你为好友，那么改变这个设置应该可以阻止任何 randos 加你为好友。

值得注意的是:这也使得你外出时遇到的真实的人无法向你发送好友请求，除非你已经有了一个共同的朋友。

对我来说，这没什么大不了的，因为我结交的大多数“新”朋友往往是我通过另一个朋友认识的人，但如果你刚刚开始一份新工作或搬到一个新地方，并试图结交一些新朋友，改变这一点意味着你需要成为发起朋友请求的人，而不是相反。

尽管如此，如果你被垃圾邮件请求轰炸，这是一个很小的代价。
