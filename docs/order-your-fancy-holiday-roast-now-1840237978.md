# 现在就点你的节日烤肉吧

> 原文：<https://lifehacker.com/order-your-fancy-holiday-roast-now-1840237978>

由于感恩节菜单的有限性，即使你等到最后一刻，也不可能在这个大日子里买到火鸡。(然而，解冻是另一个话题。)你在圣诞晚餐上有更多的回旋余地，这意味着你可能会想象烤排骨、一顶羊冠，甚至是一只又大又肥的鹅 。我给你的建议？现在就订购*。*

*Watch*

*你看，不像火鸡，没有全国范围内对烤肋排(或羊冠、鸭肉或鹅肉)的需求，这意味着这些东西必须订购，而且由于某个资本主义巨兽(谢天谢地)还不知道如何在当天运送一大块肉或整只鸟，你需要给你的屠夫一些时间来把它运进来。*

*没必要惊慌。今天花点时间研究和计划你的菜单——至少是蛋白质部分——然后给你友好的邻居肉店打个电话，让他们为你订购。在选择送货日期之前，一定要问清楚肉是新鲜的还是冷冻的，并且知道在你把肉带回家之前，你希望如何/是否希望你的肉被分解、捆绑或以其他方式准备。我喜欢圣诞惊喜，只要不是“惊喜！冻鸭！”*
