# 不，那封关于苹果卡的邮件不是网络钓鱼

> 原文：<https://lifehacker.com/no-that-email-about-the-apple-card-wasnt-a-phishing-at-1837239530>

每当你收到一封据称来自一家公司的电子邮件，但它要求你点击一个链接并确认一些该公司应该已经非常熟悉的信息，你完全有权利持怀疑态度。其实你应该很怀疑；常识是防范网络钓鱼企图的最佳第一道防线。

如果你是最早获得苹果卡的人之一，你可能会收到苹果发来的一封奇怪的电子邮件，要求你在申请之前验证你的苹果 ID。你可能从来没有收到过苹果公司发来的这种邮件，尤其是如果该公司把它发送到你用来注册 Apple Card 通知的电子邮件地址。

邮件内容如下:

> 你想成为第一批获得苹果卡的人，这是一种由苹果公司而不是银行创造的新型信用卡。好消息:这是你先于所有人体验 Apple Card 的机会，所以你可以帮助我们为公开发布做准备。您的提前访问邀请正在等待，但我们需要您的 Apple ID 来发送它。您提供的电子邮件地址与登录到 iCloud 的 Apple ID 不匹配。只需完成几个简单的步骤，我们就可以向您发送邀请。”

它继续要求你查找你在 iPhone 上使用的 Apple ID，然后使用 [Apple Card 网站](https://www.apple.com/apple-card/) 上的“通知我”按钮输入电子邮件地址。

在这种情况下，你的怀疑——尽管有根据——是没有根据的。这实际上是一封来自苹果公司的合法邮件，尽管你可能会收到错误的邮件。如果是发送到你用于 Apple ID 的主要电子邮件地址，那么苹果没有理由需要确认你提供的电子邮件地址，因为你在注册 Apple Card 通知时很可能使用了这个确切的电子邮件地址。然而，如果你想开始注册 Apple Card，你必须点击“通知我”按钮并再次输入你的 Apple ID。

### 对网络钓鱼邮件保持警惕

不要让苹果的这个小小的*失误*让你对这类电子邮件变得懒惰。当一家公司(突然)要求你确认它已经掌握的信息时:

*   检查发件人的电子邮件地址。确保它不是来自一些古怪的子域(如 apple.totallylegit.net ),否则可能会让它看起来很真实。
*   考虑在浏览器中键入电子邮件中提到的任何 URL，而不是单击超链接，以防电子邮件试图通过假设您在单击链接后不会注意到地址栏中的虚假 URL 来对您进行网络钓鱼。
*   点击电子邮件中的链接时，一定要查看地址栏。问问你自己:这真的是你试图访问的公司或合法网站吗？不要被 [缩短的网址](https://www.checkshorturl.com) 或虚拟字符所迷惑，比如有人用“I”或“1”代替“l”
*   不要打开你不希望收到的附件。
*   如有疑问，请直接联系公司——通过直接发送给客户支持的电子邮件、在线聊天或电话。他们将能够确认任何电子邮件请求是否合法。
