# 我们拒绝所有的饮用水黑客

> 原文：<https://lifehacker.com/we-reject-all-water-drinking-hacks-1832586287>

告诉人们多喝水是一种懒惰且无用的健康伎俩。你渴了就会喝水。为什么要把事情复杂化？



我应该可以在那里结束帖子，但由于某种原因，有 [列表](https://www.elle.com/beauty/health-fitness/advice/a19189/water-drinking-hacks/)[列表](https://helloglow.co/hydration-hacks-11-easy-ways-to-drink-more-water/)[列表](https://blog.myfitnesspal.com/20-life-hacks-for-drinking-more-water/) 在那里的水化黑客。欺骗自己多喝水的方法。(我们甚至已经发布了 [一个](https://lifehacker.com/how-to-trick-yourself-into-drinking-more-water-every-da-1678956552) 或者 [两个](https://lifehacker.com/what-the-color-of-your-pee-says-about-your-health-1477171258) 。)从这些看来，你会认为在你的生活中获得更多的水是非常重要的。不止如此。不，甚至更多。一饮而尽。

但是我们一直在补充水分。你死了吗？没有吗？ [那么](https://lifehacker.com/do-you-really-need-to-drink-more-water-1734665155) 你可能做得很好。以下是我拒绝的一些具体做法:

*   每天喝 X 杯水:或者你可以在口渴的时候喝水。
*   用一个应用程序跟踪你的水摄入量:见上图。
*   **分析你尿液的颜色**:你的尿液反映的是你的身体试图清除的东西， [而不是你现在身体里实际存在的东西](https://www.ncbi.nlm.nih.gov/pubmed/29464103) 。
*   **不算咖啡或其他饮料的水合作用:**如果你不习惯，咖啡因有一点点脱水作用。这并不重要，因为像咖啡这样的饮料是静止的，看看这个，由水制成的*。* 
*   ***忘记食物含有水分**:除非你严格控制肉干和葡萄干的饮食，否则饮料不是你唯一的水源。*
*   *将水果放入水中，让它看起来更漂亮，更有食欲:这很难尝起来像看起来那么好，而且这是浪费(通常很贵)你可以吃的水果。*
*   ***每天早上喝一杯柠檬水**:除非你 [真的喜欢柠檬水](https://vitals.lifehacker.com/lemon-water-doesn-t-do-anything-1822932908) 。我们其余的人将坚持喝咖啡。*
*   ***“智能”水瓶，告诉你什么时候喝:**加油。 [当真](https://gizmodo.com/stop-trying-to-sell-me-a-smart-water-bottle-1710270984) ？*

*水是一种很好的饮料，是苏打水等含糖饮料的绝佳替代品。但是只要你体内有某种液体，你就会没事的。*
