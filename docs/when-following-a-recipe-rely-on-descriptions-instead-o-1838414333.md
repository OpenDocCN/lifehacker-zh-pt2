# 当遵循一个食谱时，依靠描述而不是时间

> 原文：<https://lifehacker.com/when-following-a-recipe-rely-on-descriptions-instead-o-1838414333>

当你按照菜谱做的时候，你可能会注意到它提供了两条指导信息:你应该烹饪一种配料多长时间，以及在进入下一步(或上菜)之前，这种配料应该是什么样子。为了品尝最好的食物，你应该(大部分)忽略第一点，而专注于第二点。



这并不是因为食谱作者主动对你撒谎。他们规定的煎、烤或煨的分钟数可能是基于现实的，但是你应该用这个时间作为最宽松的指导。从厨房到厨房的变量实在太多了，以至于不能把数字当成真理。烤箱受热不均匀，煤气炉上的“中高温”和电炉上的“中高温”是不同的，成分的变化——如大小或水分含量——会极大地影响烹饪时间。因此，与其担心你做饭的时间是否过长(或过短)，不如关注食谱中描述物理线索 的部分，如温度、颜色和体积。(拜托，看在所有可食用的东西的份上， [买个数字温度计](https://lifehacker.com/a-digital-thermometer-makes-cooking-infinitely-easier-1795590263) 。)
