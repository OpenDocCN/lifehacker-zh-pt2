# 也许是时候停止使用闪光粉了

> 原文:[https://life hacker . com/it-may-be-time-to-stop-using-glitter-1833784383](https://lifehacker.com/it-may-be-time-to-stop-using-glitter-1833784383)

本周，欧盟 [一劳永逸地禁止](https://www.washingtonpost.com/world/2019/03/28/eu-parliament-backs-ban-single-use-straws/?utm_term=.4836afcfc4ef) 一次性塑料吸管，加入全美连锁酒店、 [餐厅、](https://www.businessinsider.com/plastic-straw-ban-why-are-there-so-many-2018-7) 、迪士尼主题公园和连锁杂货店

随着塑料吸管成为遥远的过去，聚苯乙烯泡沫塑料[一次性塑料袋](https://www.cnn.com/2019/03/31/us/new-york-state-plastic-bags-ban/index.html) 在全国范围内禁止制造绿色浪潮，什么日常用品将很快成为我们的下一个环境敌人？

事实证明，闪闪发光——是的，闪闪发光——很可能是一个值得研究的原因。

派对装饰有什么问题，除了它能爬进你地毯的每一个缝隙？大多数 [闪光产品](https://www.mnn.com/health/healthy-spaces/blogs/ban-glitter-plastic-pollution-like-microbeads) 由塑料、铝和 [聚对苯二甲酸乙二酯](http://petresin.org/faq.asp) 或 PET 制成，PET 是塑料包装中常见的一种聚酯，这已经不是一个很好的标志。

由于 glitter 的体积很小，它被认为是一种 [微塑料](https://www.independent.co.uk/environment/glitter-ban-environment-microbead-impact-microplastics-scientists-warning-deep-ocean-a8056196.html) 。这种污染物被冲走后会进入我们的海洋，通过降解变小，通过消耗进入水生生物体内，最终进入我们的食物链。(是的，我们正在吃它。)

虽然没有很多研究专门研究 glitter 本身对环境的影响(只是对微塑料的一般研究)，但根据独立的 的 [，PET 可以被分解成与人类和动物体内激素紊乱有关的化学物质。**T5】科学家们对彻底禁止闪光粉褒贬不一。(一些人呼吁禁止，称其与**](https://www.independent.co.uk/environment/glitter-ban-environment-microbead-impact-microplastics-scientists-warning-deep-ocean-a8056196.html) **[微珠](https://www.wkbw.com/news/local-news/ban-on-glitter-scientists-say-glitter-is-just-as-harmful-as-banned-microbeads-on-the-environment)** **一样对环境有害，这是一种在某些品牌的肥皂中发现的危险的微塑料。还有人说就是 [不够研究](https://news.nationalgeographic.com/2017/11/glitter-plastics-ocean-pollution-environment-spd/) 。)**

如果你发现自己正在考虑购买一件特别闪亮的东西，你该怎么办？

当然，你可以不买它。还要考虑你自己的购物习惯。亮片不仅仅出现在派对装饰中，它也出现在许多化妆品中:[丝芙兰](https://www.sephora.com/product/treasure-pot-glitter-gel-P439933?skuId=2178788&om_mmc=ppc-GG_1165716881_53825938740_pla-483945699557_2178788_257731947965_9060351_c&country_switch=us&lang=en&ds_rl=1261471&gclid=Cj0KCQjws5HlBRDIARIsAOomqA3SDE2kAdwQFURQkED8Om-CShlwdF_q6blJXaigtXUncLmBl-mhMIYaAoosEALw_wcB&gclsrc=aw.ds) 和 [凯莉化妆品](https://www.kyliecosmetics.com/collections/glitter-eyes/products/witch-famous-glitter-eyes) 所有零售的美容产品中都含有亮片。(Glossier 指出，你不应该用水洗掉你的产品，以避免污染水道，但你怎么能如此确定它最终不会进入水道呢？)

如果你还不能放弃闪光粉， [可生物降解的闪光粉](https://glitterevolution.com/) 实际上是一种东西，尽管它比普通的闪光粉贵。(一听售价将近 17 美元。)

身体肥皂连锁店已经放弃使用闪光粉，转而使用可生物降解的合成云母，这种材料像闪光粉一样闪闪发光，不会对环境造成影响。然而，云母是一种极具争议的成分——许多使用它的化妆品公司被指控使用童工开采这种材料。所以现在，如果成本是一个问题，少用*闪光粉可能是你保持绿色的最好答案。*
