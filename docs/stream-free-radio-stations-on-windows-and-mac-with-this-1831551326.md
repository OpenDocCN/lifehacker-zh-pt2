# 使用此应用在 Windows 和 Mac 上流式播放免费电台

> 原文：<https://lifehacker.com/stream-free-radio-stations-on-windows-and-mac-with-this-1831551326>

有几次我想在工作的时候听收音机——因为我偶尔会怀念通勤——我发现大多数电台的网站都很糟糕。就好像他们试图成为新闻服务、Tumblr 博客和播放音乐和视频的复杂网络工具的奇怪组合。糟糕。



相反，我很高兴我找到了应用程序 [**Odio**](https://odio.io/) ，这给了你一个找到和收听世界各地电台的简单方法——截至我写这篇文章时，共有 206 个国家的电台。该应用程序完全免费下载和使用，适用于 Windows、Mac 和 Linux。

打开应用程序，点击“国家”部分开始，尽管你也可以通过他们说什么语言或社区贡献的标签来浏览新的广播电台。我很欣赏 Odio 在你跳过不同电台时几乎不需要任何时间就能连接到一个流，因为没有什么比等待每个电台加载几秒钟更快地扼杀这个应用程序的乐趣了。

一旦你找到了你喜欢的东西，你可以查看电台的简介来了解更多信息——它以什么语言播放，它使用什么比特率和编解码器，等等。您还可以收藏您喜欢的电台，这将把它们添加到您的个人资料库中。这是一个在应用程序中保持有序的好方法(除了 Odio 的“最近查看”部分)，因为有很多*可以听。*

无论出于什么原因，如果你对你真正想听的电台没有被列入 Odio 的目录感到不高兴，你可以随时将它提交到 [社区电台浏览器](http://www.radio-browser.info/) 网站。Odio 从这里提取其电台流媒体列表，所以如果你想让 Odio 在应用程序中展示它们，你可以贡献你最喜欢的。
