# 如果你预订了去托马斯·库克的旅行，以下是退款的方法

> 原文：<https://lifehacker.com/if-you-booked-travel-with-thomas-cook-heres-how-to-get-1838358649>

周一，总部位于英国的旅游公司托马斯·库克突然崩溃，影响了 15 万名游客的计划，并解雇了数万名员工。在西班牙梅诺卡这样的机场，数百名旅客滞留在登机口。

虽然破产主要影响英国游客，但该公司提供往返美国的套餐和航班，包括纽约和奥兰多。根据的消息，托马斯·库克控股公司还拥有一家德国航空公司 Condor，该公司为巴尔的摩和凤凰城等城市提供服务。

如果你受到这个消息的影响，首先，你应该知道所有托马斯·库克的航班都被取消了，所以你可能不应该去机场。另一方面，如果你预订的托马斯·库克度假套餐包含了单独的航空预订，你的航班可能仍然有效——请联系该航空公司确认。根据该航空公司的 [推特账户](https://twitter.com/Condor/status/1175885838091280386) ，秃鹰航班目前仍在运营。

否则，如果您使用信用卡预订了您的旅行计划，请立即联系您的银行并解释您的情况，以发出退款。正如斯科特廉价航班公司的斯科特凯斯在《T2 T3》中写道的那样，你也可以看看其他航空公司提供的“救援票价”，比如维珍大西洋航空公司，它为洛杉矶、拉斯维加斯、旧金山或纽约等城市的旅客提供帮助，这样你就可以找到打折机票。

如果您的最终目的地是英国，请访问 [民航局的网站](https://thomascook.caa.co.uk/customers/if-you-have-a-future-booking-and-have-not-traveled-yet/) ，该网站为那些计划被取消的人提供了一些资源。如果您目前在国外，民航局将免费为您更改计划。(这里事情变得有点复杂了；如果您的计划受到 ATOL 保护，您只能更改计划或申请退款。ATOL-protection 意味着，如果你的英国公司出于任何原因停止公开交易，你有权获得退款。如果你是受 ATOL 保护的，你会在预订时收到证书。)
