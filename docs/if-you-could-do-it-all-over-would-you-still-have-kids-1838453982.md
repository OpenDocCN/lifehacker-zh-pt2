# 如果可以重来，你还会有孩子吗？

> 原文：<https://lifehacker.com/if-you-could-do-it-all-over-would-you-still-have-kids-1838453982>

前几天在 Reddit 上看到 [这个帖子的时候，我几乎是一扫而过。标题是，“父母:你对生孩子的真实想法是什么？在你经历了这一切之后，你还会拥有它们吗？”但我停顿了一下。我知道会有很多“*为人父母太难了，但我不会拿我的孩子换任何东西！”的回应，但肯定有些人有过遗憾。出于好奇，我点了进去。*](https://www.reddit.com/r/Parenting/comments/d95ga4/parents_what_are_your_real_thoughts_about_having/)



该帖子来自 Reddit 用户 u/olivesucks，她一直计划与丈夫生孩子。但现在，他们正在重新考虑这个选择:“我知道每件事都有积极和消极的一面，但以你今天的想法和经验:你会回到过去，取消生孩子的部分吗？”

尽管有很多预期中的“我的孩子是发生在我身上最好的事情”的回答，但也有一些人如果可以的话会取消他们的父母身份。一些父母说他们不会再这样做了，因为他们的生活已经被彻底接管了，当然，还有一位父母，u/Shoonasasi，他的孩子患有精神疾病和大脑疾病:

> 如果没有帮助，他要么进监狱，要么死了，知道他没有机会了是绝对令人心碎的，除非我们赢得彩票或者签署我们的权利，把他交给政府。
> 
> 如果我能回到过去，我不会生下他，哪怕只是为了把他从他现在被迫过的生活中拯救出来。

然后有一个回应，来自 u/dieomama(也得到其他人的响应)，他说他们今天会做出不同的选择——不是因为他们作为父母的经历，而是因为我们要留给我们的孩子什么:

> 看事实，气候危机很有可能让他们在 30 岁之前生活悲惨。我后悔在我有孩子的时候没有被更好地告知。如果我适当地通知我自己，我会避免把某人带到这个世界上。
> 
> 我仍然可以抱最好的希望。气候模型并不是 100%准确，但事情看起来对未来并不乐观。我是一名训练有素的科学家，科学似乎是可靠的。

是否抚养孩子是这样一个个人决定，是否有理由后悔也是个人和情境的。满足或遗憾可能来自我们的生活质量，我们孩子的生活质量或我们对他们未来的想象。

告诉我们:如果你有了孩子，你可以回到过去从头再来一遍，你会吗？
