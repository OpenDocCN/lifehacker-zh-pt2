# 以下是向信用机构报告已偿还债务所需的时间

> 原文：<https://lifehacker.com/heres-how-long-it-takes-for-paid-debt-to-be-reported-to-1836018898>

你刚刚支付了一大笔采购的最后一笔款项，而你似乎永远都在支付这笔款项。应该庆祝一下，对吧？来吧，当你最后一次按下“现在支付”按钮时，在你的座位上跳舞。开瓶香槟酒。打电话给你妈妈。但是不要指望你的信用报告能马上反映出还清的贷款。



我从来没有考虑过信用报告的滞后时间，直到今年初春我兴高采烈地支付了我的汽车贷款。我确信我知道更新账户和报告已付贷款需要一分钟，但在我激动的心里，我确信贷款公司会马上给三个信用局打电话说，“嘿，Lisa 提前两年还清了她的汽车贷款。划出来好不好？”

我雄心勃勃的朋友们，事情不是这样的。

这没什么大不了的。我只想让信用机构给我一颗金星。但是，如果你是一个正在积极努力提高你的信用的人，并且一直在努力摆脱贷款，那么，你希望看到你的进步被有关部门记录下来，这些部门可以决定你未来获得更多信用的能力。

那些已支付的房贷、助学贷款、车贷什么时候会体现在你的征信报告里？给它两个月的时间。

债权人不需要在每个月的某个时间提交信息，Experian 、 [TransUnion](https://www.transunion.com/blog/credit-advice/how-long-does-it-take-for-a-credit-report-to-update) 和 [Equifax](https://www.equifax.com/personal/education/credit/report/how-long-does-information-stay-on-credit-report/) 表示，通常需要 30-45 天才能反映出收益。当你的信用报告最终更新时，它会显示你最后一次付款的日期。

对于循环会计来说有点不同。如果你在付清后立即关闭信用卡，你会看到这反映在你的信用报告上。如果你支付了余额，但保持卡开着(对你的信用记录来说是一个更好的选择)，你只会看到账户当前余额为零。

如果你每年定期访问[【AnnualCreditReport.com】](http://www.annualcreditreport.com)以获得来自三个局的免费报告，了解这一点是很有帮助的。您可能希望等到知道您的帐户已关闭时再访问您的最新报告。如果你登录得太快，你仍然会看到一个平衡挂在那里。你做了这么多工作，这一点都不好玩。

不要忘记，在你完成支付后，那些已支付的项目会在你的信用报告上停留很长时间。已支付的已结清账户 [在您的报告上保留 10 年](https://twocents.lifehacker.com/how-long-different-items-stay-on-your-credit-report-1750377233) ，而诸如收款账户、逾期付款、税收留置权和民事判决等负面项目在您的账户上保留 7 年。
