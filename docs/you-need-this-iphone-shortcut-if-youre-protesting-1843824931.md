# 如果你要抗议，你需要这个 iPhone 快捷键

> 原文：<https://lifehacker.com/you-need-this-iphone-shortcut-if-youre-protesting-1843824931>

当你离开房子去抗议时，你可能很想把你的 iPhone 留在家里——我不怪你，因为当你被推倒在地或被橡皮子弹击中时，你昂贵的设备被摧毁的风险会增加——你可能需要重新考虑。你的 iPhone 可以是一个令人难以置信的新闻来源和一个强大的通信设备；如果你在出门前花几分钟时间设置一些重要的快捷方式，它也可以成为救命稻草。



马修·卡西尼利(Matthew Cassinelli)已经因为创造了数量惊人的 [iOS 快捷方式](https://www.matthewcassinelli.com/category/shortcuts/) 而在技术圈相当知名，他又回来了。这一次，他建立了一个有用的快捷方式，如果你要出去抗议，你应该考虑安装在你的 iPhone 上。(或者，至少，对于您可能想要构建的任何类似主题的快捷方式来说，这是一个很好的起点。)

他的 [版本](https://twitter.com/mattcassinelli/status/1266744618441302017) 是这个最初向抖音广播的有用快捷方式的增强，以及罗伯特·彼得森去年发布到 Reddit 的 [这个快捷方式](https://www.reddit.com/r/shortcuts/comments/9huqiw/getting_pulled_over_by_police/) :

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1266487920564695040&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1266487920564695040&autosize=1) 

轻点图标， [卡西尼利的快捷方式](https://www.icloud.com/shortcuts/ab31b23eb7954cc4858e6c1502bfdf7e) 会:

*   给你的一个联系人发一条短信，提供你的街道、纬度和经度，以及你被逮捕的消息
*   打开 iPhone 的后置摄像头，开始拍摄视频
*   完成录制后，将视频保存到 iPhone 中
*   将所述视频发送给你的前述联系人

我建议和一个朋友一起测试这个快捷方式，以确保你的 iPhone 运行快捷方式所需的任何权限都已经设置好了。同样，你需要确保它能在你的设备上完美运行。有一个花哨的“在紧急情况下通知你的朋友”的快捷方式，但它不能做你所期望的一切，这是没有意义的。

当然，您可以修改这个快捷方式的任何部分来满足您的特定需求。举例来说，你可能不会被逮捕，但如果抗议变得越来越激烈，而你正身处其中，你可能会想通知你的朋友。

在安装快捷方式之前，您需要做三件事。首先，你需要在你的 iPhone 上运行至少一个快捷方式。它可以是任何东西；只是运行一些东西。之后，你需要打开你的设置应用，点击快捷方式，然后打开“允许不可信的快捷方式”

最后，你需要点击 [这个链接](https://www.icloud.com/shortcuts/ab31b23eb7954cc4858e6c1502bfdf7e) 在你的 iPhone 上获得 Cassinelli 的快捷方式。

安装快捷方式后，打开快捷方式应用程序，然后点击新“警察”快捷方式右上角的三点图标。在快捷方式的配置屏幕中，点击右上角的三点图标，然后您可以将其重命名为*并将*添加到您的主屏幕和共享表单，以便于访问。
