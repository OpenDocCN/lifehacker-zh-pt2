# 这个奥利奥马克杯蛋糕只是一个温和的求救信号

> 原文：<https://lifehacker.com/this-oreo-mug-cake-is-only-a-mild-cry-for-help-1844297852>

大概一年一次，心情低落就做个马克杯蛋糕。它从来都不令人满意。测量一小勺一小撮的烘焙原料，却得到一大杯潮湿的类似蛋糕的东西，这有一种特别凄凉的感觉。



几周前，我对马克杯蛋糕产生了感情(几乎像时钟一样)，尽管我最终选择了“碗曲奇”，这和过去几年的马克杯蛋糕一样糟糕，如果不是更糟糕的话。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1275995685737779200&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1275995685737779200&autosize=1) 

也许如果我已经下载了抖音，事情会有所不同。也许我会早点看到这个奥利奥马克杯蛋糕的配方，也许我会做那个。或许，我可以吃一大杯奥利奥口味的不太像熔岩的蛋糕，而不是一碗潮湿的、像海绵一样但不耐嚼的饼干面糊。我本可以不去测量和搅拌，而是去泼洒和刺伤。

 [https://lifehacker.com/embed/inset/iframe?id=tiktok-6803418065066396933&autosize=1](https://lifehacker.com/embed/inset/iframe?id=tiktok-6803418065066396933&autosize=1) 

马克杯蛋糕最令人沮丧的地方是它们实际上需要多少努力。面粉、糖、膨松剂和量勺都仍然需要，但最终产品却平庸得不成比例。马克杯蛋糕用它们平淡无奇的自卑来嘲笑它们的制造者，事实证明，这对抑郁症毫无帮助。

总之。由于我没有抖音，我第一次在厨房 上看到这种食谱，在那里它被描述为“神奇”我不会用这个词来形容它，但这个蛋糕肯定比我吃过的任何其他蛋糕都好。味道和质地方面，它介于 BJ 披萨饼和融化的巧克力蛋糕之间，只是味道稍微淡一点，劳动强度也比这两者低得多。

理论上，你所要做的就是在一个马克杯里扔一些饼干，倒入足够的牛奶，填满半个马克杯(大约三分之一杯)，然后捅和捣碎，直到你有一个奥利奥污泥。杯子放入微波炉 75 秒，然后出来一个黏糊糊的巧克力蛋糕。

在实践中，我的奥利奥蛋糕杯制作经验与视频中的略有不同。我不得不用微波炉加热了整整一分半钟——开始时它非常粘稠——我还不得不额外加入几块奥利奥来做稠泥。(需要八个干奥利奥才能装满杯子，但是总共需要 10 个才能把面糊放在你想要的地方。)第一个奥利奥马克杯蛋糕是潮湿和温暖的，但没有我预期的巧克力味，所以我又做了一个，并在牛奶中加入了两汤匙好时的糖浆，这(令人惊讶地)使它更有巧克力味。

总而言之，我想我实际上更喜欢吃四五个奥利奥和一大杯冷牛奶的味道和体验，但这仍然是我做过的最好的马克杯蛋糕，它和冰淇淋一起吃会特别好。这并不完全是一种求助——更像是一种礼貌的请求——但是奥利奥马克杯蛋糕并没有假装成它不是的东西。而不是伪装成“其实挺好的！它包含了一种不可避免的绝望，这种绝望首先会导致一个人做出这样的甜点。“捅那些饼干，”它低声说道，“狠狠地捅它们，做成一堆烂泥。”如果这不是这个时候合适的心情，我不知道什么是。
