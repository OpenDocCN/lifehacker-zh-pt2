# 如何将智能手机变成网络摄像头

> 原文：<https://gizmodo.com/how-to-turn-your-smartphone-into-a-webcam-1842783844>

你使用视频通话软件的次数可能比一个月前多得多，而且你也不想成为唯一一个没有视频源的远程工作者。如果你的家用电脑没有内置摄像头，那么你可以用智能手机上的摄像头来填补这个空白。

在我们讨论软件选项之前，有必要谈一谈硬件设置。好消息是，你手机的摄像头将比大多数网络摄像头更好，但你需要确保它的稳定性——一个好的迷你三脚架或至少一本厚书可以起到支撑作用。

你通常可以在前置摄像头和后置摄像头之间进行选择，所以你显然想选择手机上的后置摄像头——它将提供最好的质量。最后，如果你选择的应用程序让你选择通过 USB 而不是无线连接，这是一个好主意，以获得最快、最稳定的反馈。

有相当多的应用声称可以让你把手机变成网络摄像头，但我们并不完全相信它们。随着如此多的笔记本电脑和一体机内置网络摄像头，而且网络摄像头本身非常便宜，这可能不是软件开发人员热衷于探索的领域。

然而，我们已经找到了一些我们喜欢的工具。第一个是 NewTek 的 NDI·HX 相机应用程序——不幸的是，目前你只能在 iOS 系统上获得它。之前有一个 Android 版本，我们希望它会在某个时候回归(也许在你读到这篇文章的时候)。

还有另一个警告，那就是 NDI 虚拟输入实用程序，它欺骗你的视频软件接受 NDI 输入作为网络摄像头，暂时只在 Windows 的[【NDI 工具](https://www.ndi.tv/tools/) 中可用。虽然有一个用于 macOS 的 NDI 工具，但目前它不包括 NDI 虚拟输入，所以不能把你的手机变成网络摄像头。

如果你碰巧用的是装有 Windows 操作系统的 iPhone，设置 NDI 应用程序再简单不过了:打开 iPhone 应用程序，然后点击 **NDI** 按钮开始播放。在你安装了 NDI Windows 工具后，启动 NDI 虚拟输入并在通知区域右键点击它的图标 iPhone feed 会神奇地显示为一个选项，无需进一步配置。

剩下要做的就是启动 Zoom、Skype 或任何其他你用来进行视频通话的软件，并选择这个新的 NDI 相机作为你的网络摄像头。例如，在 Zoom 中，点击主窗口上的 cog 图标，然后选择**视频**和**摄像机**。在 Skype 中，点击你的头像，然后点击**设置**、**音频&视频**和**摄像头**。无论你使用什么视频聊天工具，这个选项都会隐藏在视频设置中。

在你的 iPhone 上，点击齿轮图标，调出设置选项。当你在横向模式下握着手机时，从左到右，他们控制着变焦、亮度、相机闪光灯、三分法网格、麦克风静音、哪个相机用于馈送(前置或后置)以及视频馈送的质量。

如果你想用安卓手机作为网络摄像头，我们发现最完美的选择之一是 [DroidCam](https://play.google.com/store/apps/details?id=com.dev47apps.droidcam) 。同样，这只适用于 Windows，通过 [桌面客户端](https://www.dev47apps.com/) ，但设置很简单——启动 Android 应用程序，它会给你一个 IP 号码，你可以输入到 Windows 客户端，然后你就可以启动并运行了。

使用手机作为网络摄像头时，点击 Android 应用程序右上角的圆形图标可以调暗屏幕。点击右边的三个点来调整白平衡和曝光(当馈送停止时，您可以使用同一菜单上的**设置**来限制发送到您的计算机的每秒帧数并节省电池寿命)。

随着提要的运行，摄像头应该对所有使用网络摄像头的视频聊天应用程序可用。例如，它可以很好地与 Skype 和 Zoom 配合使用，尽管我们无法让 Windows 自带的摄像头应用程序识别它。

支付 4.50 美元购买 Droidcam 的高级版本，你可以从 Windows 端访问更多设置，例如缩放控制和将视频切换到镜像视图的选项。免费版本应该足以满足大多数人的需求，我们使用它没有任何问题，尽管你不得不忍受一些广告。

我们确实试验了其他几个被宣传为负责这一特定任务的应用程序，包括用于 [Android](https://play.google.com/store/apps/details?id=com.e2esoft.ivcam) 和 [iOS](https://apps.apple.com/us/app/ivcam-webcam/id1164464478) 的 iVCam 网络摄像头，以及用于 [Android](https://play.google.com/store/apps/details?id=com.kinoni.webcam2) 和 [iOS](https://apps.apple.com/us/app/epoccam-webcam-for-mac-and-pc/id449133483) 的 EpocCam(这是极少数承诺在 macOS 上工作的应用程序之一)。这些选项中没有一个像 NDI·HX 相机和 DroidCam 一样可靠，尽管你的里程数可能会有所不同。

另一种选择是简单地使用手机上的应用程序进行视频聊天，尽管这通常不如使用笔记本电脑舒适或灵活。许多视频应用也不得不减少手机上的功能和选项。但至少它会让你的朋友、家人和同事看到你的脸。
