# 你可以不用百加得制作百加得鸡尾酒

> 原文:[https://life hacker . com/you-can-make-a-Bacardi-cocktail-no-Bacardi-1844421363](https://lifehacker.com/you-can-make-a-bacardi-cocktail-without-bacardi-1844421363)

台克利酒是最普遍的朗姆酒鸡尾酒，这是有充分理由的。它很好吃，很提神，而且非常容易做。它不需要任何修改，但如果你觉得有必要改变它，你可以通过将简单的糖浆换成石榴糖浆来制作一种全新的(完全粉红色的)饮料。

这个“奇怪的把戏”把你的代基里鸡尾酒变成了百加得鸡尾酒，出于某种原因，这种曾经很受欢迎的饮料你在很多菜单上都看不到了。我认为任何好的白朗姆酒都可以，尽管百加得公司可能不同意 T2 的观点。我用了 Havana Club Añ ejo 三年，发现它非常令人愉快，因为它的朗姆酒真的非常好。百加得鸡尾酒沿袭了经典的 [酸味形式](https://skillet.lifehacker.com/the-sour-is-the-perfect-quarantine-cocktail-1843185396) ，又酸又甜，很像台克利酒。多亏了石榴皮，它也更有果味(和粉色)。我用的是商店买的石榴糖浆，但是你可以用等量的石榴汁和糖制作糖浆来制作你自己的糖浆。(就做一个简单的糖浆，但是用果汁代替水。)这个额外的步骤确实让饮料变得更加美味，我认为这是一个不错的小项目。

要制作这款充满活力、令人耳目一新的朗姆酒，你需要:

*   2 盎司白朗姆酒
*   3/4 盎司酸橙汁
*   3/4 盎司石榴糖浆

将所有东西倒入加满冰块的鸡尾酒调酒器中，摇晃直到所有东西都变得太冷而无法保存。滤入一个小酒杯，如果你愿意的话，用酸橙装饰，在喝之前花点时间凝视你的饮料的美丽。
