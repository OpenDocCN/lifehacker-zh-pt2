# 如何直播 Lollapalooza

> 原文：<https://lifehacker.com/how-to-stream-lollapalooza-live-1836909474>

本周末，超过 150 支乐队将登上今年在芝加哥格兰特公园举行的 Lollapalooza 音乐节的舞台，驯服的黑斑羚(Impala)、21 飞行员(Twenty One Pilots)和爱莉安娜·格兰德(Andrew)等明星将在几十万名与会者面前，在酷热的天气中表演。(去年，160 多名市民因中暑住院。)



如果你不能参加派对，或者因为站在那里和长长的厕所队伍而讨厌音乐节，你可以在 YouTube 上直播音乐节。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-oFjKZ2S0K4U&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-oFjKZ2S0K4U&start=0) 

整个周末，Lollapalooza 将在三个独立的频道上直播三个阶段的表演；只需访问 [Lollapalooza 的 YouTube 页面](https://www.youtube.com/channel/UC-QVOEJcRTmqXTEwUFbPRLA) 即可找到每个阶段的页面。

如果您错过了任何演出， [第一阶段的频道](https://www.youtube.com/watch?v=oFjKZ2S0K4U) 还将重播前一天的节目，直到下午 Lollapalooza 重新开始。周四的演出目前在 [重播](https://www.youtube.com/watch?v=oFjKZ2S0K4U) ，如果你想赶上 H.E.R .、Hozier 或 the Strokes 的集。

以下是几场演出的设定时间，包括一些值得注意的头条新闻(所有时间显示在 **CDT** ):

*   [第二阶段](https://www.youtube.com/watch?v=A64obrjX0sE) :可爱的死亡出租车(星期五，晚上 9:15)
*   [第三阶段](https://www.youtube.com/watch?v=ff0nG6_uhsw) :阿莱索(周五，晚上 9 点)
*   [第一阶段](https://www.youtube.com/watch?v=oFjKZ2S0K4U):21 名飞行员(周六，晚上 8:45)

*   [第二阶段](https://www.youtube.com/watch?v=oFjKZ2S0K4U) :犹大&狮子(周六，晚上 7:45)
*   [第三阶段](https://www.youtube.com/watch?v=ff0nG6_uhsw) : Madeon(星期六，晚上 9:10)
