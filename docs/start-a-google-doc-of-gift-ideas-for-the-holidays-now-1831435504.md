# 现在就开始用谷歌文档搜索节日礼物创意吧

> 原文：<https://lifehacker.com/start-a-google-doc-of-gift-ideas-for-the-holidays-now-1831435504>

在最近的所有派对、消极攻击性的家庭评论和香槟宿醉之后，2019 年的假日季可能是你现在最不想考虑的事情。

Watch

但是如果你想在送礼物方面做得更好——或者只是让整个事情对你自己来说更容易——现在就开始为你爱的人列出礼物的想法。全年都要参考它，当你或你的朋友或爱人有了一个礼物的想法并发表了评论。到 12 月 23 日，你就不用忙活了。

《生活黑客》的临时育儿编辑梅根·沃尔伯特对这一招深信不疑。"她说，我有一个名为“礼物创意”的电子邮件文件夹. "每当有人提到什么，我就用电子邮件发给自己，然后存档。"

你也可以使用谷歌文档，创建一个亚马逊购物清单，专门列出礼物的想法(当你有钱时，在一年中购买)，使用你的 notes 应用程序或任何其他适合你的系统，并且全年都可以轻松访问。(编者注:神童列表也非常适合这个职位。)

运气好的话，你的下一个假期会很顺利，你会成为你家里最会送礼物的人。非常值得努力。
