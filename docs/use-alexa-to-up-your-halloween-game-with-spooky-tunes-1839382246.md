# 使用 Alexa 来升级你的万圣节游戏，加入怪异的音乐、尖叫声等等

> 原文：<https://lifehacker.com/use-alexa-to-up-your-halloween-game-with-spooky-tunes-1839382246>

再过几天就是万圣节了。取决于你住在哪里，今晚你所在的城市可能会庆祝这个节日。



如果你有一个亚马逊 Alexa 设备，那么它可以方便地为不给糖就捣蛋的人和派对客人设置场景。

默认情况下，亚马逊的设备上有怪异的万圣节声音。要激活它，只需说“ Alexa，开始令人毛骨悚然的万圣节声音”，一个令人毛骨悚然的声音循环就会开始。

如果您的设备上启用了 Spotify 或 Pandora，您还可以通过智能扬声器要求它播放其中一项服务的万圣节播放列表。

我也是 [幽灵尖叫技能](https://www.amazon.com/Vifopex-Corp-Spooky-Scream/dp/B01N0RWGJX?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/use-alexa-to-up-your-halloween-game-with-spooky-tunes-1839382246&asc_source=&tag=kinjalifehackerlink-20) 的超级粉丝。一旦启用，你可以让它在设定的时间后尖叫。例如，你可以设置它在一小时后尖叫，当你不在房间里的时候恐吓你的配偶/室友。

如果你正在寻找一个快速、有趣的万圣节主题时刻，你也可以要求 Alexa“告诉我一个可怕的故事”,这将产生一个大约 20 秒的可怕诗歌片段。从我的经验来看，它们都没那么可怕，但听起来还是很有趣。
