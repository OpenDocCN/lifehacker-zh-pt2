# 使用 Roadtripper 和 Porsche 等应用程序选择风景优美的路线

> 原文：<https://lifehacker.com/take-the-scenic-route-with-apps-like-roadtripper-and-ro-1835154455>

一切又变绿了，我们在外面摇下了车窗，这只能意味着一件事:是时候上路了。



如果你是那种喜欢周日驾车外出或不时在人迹罕至的地方漫步的人，有几个应用程序(一个经验丰富，一个是新手)可能会帮助你在这个夏天找到自己的路。

### 公路旅行者

[Roadtrippers](https://roadtrippers.com/) ，一个网络和移动旅行规划平台，有一个流行的应用程序，可用于 iOS 和安卓。如果你计划一次公路旅行，这个应用程序可以识别沿途的兴趣点，如小径、远足、景点、公园和自然保护区。

如果你是那种提前计划的人，你可能更容易先在 [桌面版](https://roadtrippers.com/) 上创建旅程，然后在你上路时在你的应用程序上查看。(在玩桌面版的时候，我发现有一个叫 [Reptiland](https://reptiland.com/) 的地方，我们每年探亲的时候都会路过几次，现在我有义务带着我的儿子去那里。)但这款应用在运行时也很好用。

Roadtrippers 也有助于在你的城镇或任何你正在旅行或探索的地区找到一个新的风景优美的散步场所。

### 保时捷之路

风景路线导航板块的最新成员是保时捷 的 [Roads，它宣称自己是“为热爱驾驶的人建立社区的应用程序”](https://gearpatrol.com/2019/05/28/porsche-driving-app/)

它仍然是新的(到目前为止，它只在 [iOS](https://itunes.apple.com/us/app/roads-by-porsche/id1438225964?mt=8) 上提供)，并依赖用户保存和分享他们喜欢的路线。我注意到我身边还没有太多的选择——最近的一个离我家大约 23 英里。但这应该会随着时间的推移而增长，我喜欢这个应用程序的原因是用户可以添加关于驱动器的描述。您可以了解您将看到的风景类型、附近的午餐地点和道路状况，如海拔变化或车道的弯曲程度。

你也可以通过评级、驱动器数量、最新或长度来搜索更大的区域，如果你只是想在未来寻找你可能想要的风景驱动器。

当然，如果你想要一个真正经典的道路漫游体验，你可以去老派学校，拿出纸质地图。那也很有趣。
