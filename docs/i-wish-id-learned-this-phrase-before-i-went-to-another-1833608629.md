# 我希望我在去另一个国家之前学会了这个短语

> 原文：<https://lifehacker.com/i-wish-id-learned-this-phrase-before-i-went-to-another-1833608629>

为一次旅行而学习一门语言和大规模学习是不同的；Duolingo 太全面了，任何试图建立在语法基础上的东西都是如此。你需要一些短语，比如“请”和“谢谢”，“这是什么意思？”以及“洗手间在哪里？”但大多数时候，我只需要一个词就能让自己不那么像一个丑陋的美国人。我需要说对不起。这比听起来要复杂得多。



想想你用英语表达的小小歉意:“对不起”、“原谅”、“对不起”、“谢谢。”各有不同的内涵，这也要看你怎么说。“对不起”可能意味着“对不起，撞了你”，或者“实际上你应该向我道歉”，或者“请忽略我”，或者“我可以请你注意吗？”

我没有意识到我是多么依赖这些小小的变化来度过我的一天，直到我在那里，不断地撞到人，挡在路上，或者寻求帮助。我不想给人一种卑躬屈膝的印象，但我也不想看起来像一个有资格的游客。这些微小的道歉短语是口头润滑剂，让数百万人在没有持续情感摩擦的情况下近距离地继续他们的生活。如果你能很好地运用它们，你会更加自信和友好，而不害怕冒犯。

当你是一个明显的外国人时，你可以带着歉意的鞠躬和微笑离开。当我蹒跚地走过东京的街道和地铁时，我不认为我给任何人留下了可怕的错误印象。(但是谁知道呢？当地人很有礼貌，不会让人知道。)在德国或芬兰这样的地方，我不会马上被认出是外国人，所以我会更加小心。

如果可以的话，和会说两种语言的人谈谈关键短语和它们的意思。如果找不到人，就找一个包含发音音频剪辑的短语手册应用程序。iOS 和 Android 上有大量便宜和免费的软件。我最喜欢的是[co gent 的学说话系列](https://www.learnapps.co/) 。他们的应用程序 [学习日语](https://itunes.apple.com/us/app/learn-japanese-phrasebook-for-travel-in-japan/id447047877?mt=8) 包括“对不起”、“对不起(道歉)”、“对不起(同情)”、“请”用于请求和提供的版本，以及“没关系”和“没问题”的版本

在我的旅行中，我试着学了一些，但我唯一能始终记得的是“sumimasen”，意思是“打扰一下”老实说，我通常只是说“对不起”，做一个礼貌的鬼脸。每一次，我都希望自己事先多练习。我浪费了很多时间去记忆各种形式的“早上好”和“晚上好”，以及“Hajime mashite，Nicku-des”——“很高兴见到你，我是 Nick，”，这些我根本用不上。现在我不得不临时抱佛脚。

一本常用语手册仍然可能误导你进入尴尬的用法——也许你会踩了某人的脚并表达你最深切的哀悼——所以要注意人们的反应，当你撞到说英语的人时，向他们寻求建议。

记得感谢他们——在日语中可以是“domo”、“arigato”，或者你买东西时会听到的“arigato gozaimashita”——轻快地咕哝着，但重复两三次。希望你选对了！
