# 使用这些工具和 Chrome 扩展来升级你的网飞游戏

> 原文：<https://lifehacker.com/up-your-netflix-game-with-these-tools-and-chrome-extens-1842429696>

社交距离意味着我们很多人在网飞上花费了更多的时间。如果你的家用电脑上正在观看网飞，那么你会想要看看这些 Chrome 扩展。



过去的一周 [网飞派对](https://lifehacker.com/sync-netflix-streams-with-far-away-friends-using-this-c-1824048081) ，一个可以让你和朋友一起看电影的扩展，得到了很多关注。它已经存在多年了，但现在随着人们寻找一种新的观看和社交方式，它开始作为一种“新”工具再次传播。

像网飞党，也有一吨其他伟大的网飞扩展有值得一看。

如果你的网飞队列已经开始变少， [Flixab le 可以帮你找到真正值得看的东西再把它填满](https://lifehacker.com/flixable-helps-you-find-the-perfect-thing-to-watch-on-n-1822271341) 。网站允许你通过流派、IMDB 评级和发行年份搜索流媒体服务上的电影和电视节目。一旦你得到结果，你可以通过发行年份、评级、标题或者它们被添加到网飞的时间来进一步过滤它们(如果你在平台上寻找对你来说“新”的东西，这很有用)。

### [更好的浏览](https://lifehacker.com/browse-netflixs-secret-categories-with-this-chrome-exte-1828909576)

网飞已经按类别划分了内容，但也有你看不到的网飞类别。 [“更好的浏览”向您显示网飞的所有“隐藏类别”](https://lifehacker.com/browse-netflixs-secret-categories-with-this-chrome-exte-1828909576) ，这样您可以比平常更深入地搜索。例如，你可以看“动作片”和“动作惊悚片”，而不是“动作片”

### [永不落幕的网飞](https://lifehacker.com/1823683942)

厌倦了网飞问你是否还在疯狂观看吗？ [【永不落幕】网飞](https://lifehacker.com/1823683942)自动跳过电影和电视节目的介绍和演职员表，这样你就可以直接跳到下一集。它还消除了询问您是否还在观看的弹出窗口。

### [【lingvo】](https://lifehacker.com/learn-another-language-watching-netflix-using-this-app-1833869703)

如果沉迷于电视妨碍了你利用在家的时间学习新东西的计划， [Lingvo 可以帮你完成这两项任务](https://lifehacker.com/learn-another-language-watching-netflix-using-this-app-1833869703) 。Chrome 扩展是 Chrome 扩展和基于网络的智能手机应用程序的结合。有了它，你可以看另一种语言的电影，由于 Chrome 扩展，该语言的字幕会同时显示在屏幕上和手机上。当你遇到一个你不理解的单词时，你可以在手机上点击这个单词来获得快速定义，而不必暂停你的电影。
