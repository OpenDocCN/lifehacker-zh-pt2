# 使用此浏览器扩展以窗口全屏观看 YouTube

> 原文：<https://lifehacker.com/watch-youtube-in-windowed-fullscreen-with-this-browser-1844596144>

有时你想要一个电影般的 YouTube 体验，但网站的默认播放器尺寸在桌面上还有待改进。

Watch

普通播放器很好，但它很小，用户界面会分散注意力。全屏模式会将视频填满整个显示屏，但在这种模式下，您无法访问其他浏览器标签或应用程序。

中间的选择是“剧院模式”，它将视频扩展到正常的播放器尺寸之外，而不会遮挡你已经打开的其他标签。然而，它并不比普通的 YouTube 视频大多少，而且它以不方便的方式重新配置了网站的用户界面。

如果这些选项都不适合你，你可能想试试 YouTube 窗口全屏浏览器 add- on。

像 YouTube 的正常全屏模式一样，YouTube 窗口全屏模式将视频放大到正常或影院模式视图之外，但它只占用浏览器窗口，而不是整个屏幕，因此您可以继续使用其他应用程序。最重要的是，即使你移动到其他标签或应用程序，视频也会保持浏览器填充大小，因此当你返回视频时，你不必重新启用。

该插件由开发者“navi.jador”创建，他最近将 [扩展移植到了火狐](https://addons.mozilla.org/en-US/firefox/addon/youtube-window-fullscreen/) 。谷歌 Chrome 也有 [。](https://chrome.google.com/webstore/detail/youtube-windowed-fullscre/gkkmiofalnjagdcjheckamobghglpdpm)

它是这样工作的:

1.  从 Chrome 或 Firefox 插件商店下载 add- on。按照屏幕上的说明完成安装。
2.  安装了窗口全屏后，进入 YouTube 并播放一段视频。
3.  点击**“`”**键(在“1”键的正左侧)进入窗口全屏模式。再次按下可返回默认视图。

您还可以调整附加组件设置中的一些选项，根据您的喜好自定义控件和行为。

其中一个设置可以让你隐藏 YouTube 的默认“全屏”切换，你可以改变启用/禁用窗口全屏模式的热键。该插件还可以在你播放视频时自动切换。要找到这些选项，右键单击地址栏旁边的 YouTube Windows 全屏图标，然后选择“选项”

如果你不喜欢 YouTube 的默认视频布局强加给你的任何妥协，YouTube 窗口全屏是很好的，但它不是 Chrome 和 Firefox 用户唯一的替代 YouTube 观看体验。

如果你想让视频一直显示在屏幕上，还可以选择画中画(PIP)播放。与窗口式全屏模式不同，PIP 在一个更小、大小适合 T2 的播放器窗口中播放 YouTube 视频，该窗口将显示在其他屏幕应用程序上。默认情况下，YouTube 的 PIP 可以在 Chrome 和 Firefox 中使用——你不必安装任何第三方扩展，但使用它的步骤取决于你使用的是哪种浏览器。
