# 使用 Windows 10 的新 PowerToys 管理您凌乱的桌面

> 原文：<https://lifehacker.com/manage-your-messy-desktop-with-windows-10-s-new-powerto-1837940528>

“PowerToys”这个词可能不会让你跑向你的 Windows 电脑，但仅仅因为微软的古怪工具套件不性感并不意味着它不值得拍打你的系统。该公司今年早些时候宣布，它计划为 Windows 10 推出 PowerToys，前两个实际上是你日常生活中非常有用的补充。

如果你想遵循这个 Windows 10 的 PowerToys 的简短指南，在你的系统上安装 PowerToys 是很容易的。去下载。MSI 安装程序来自 [Github](https://github.com/microsoft/PowerToys/releases) ，其中包含两个实用程序，并运行它。一旦你在系统上安装了 PowerToys 并启动了应用程序，一个新的图标就会出现在你的任务栏上。(就是那种看起来像 RGB 渐变，下面有六个点的，我截图里左起第二个。)

右键单击图标并选择“设置”,开始配置两个 PowerToys 实用程序。第一个是 **FancyZones** ，允许您在系统屏幕上设置透明区域，您可以用它来以设定的形式(以设定的大小)绘制窗口。

要设置不同窗口的布局，点击 PowerToys 设置中的 FancyZones 菜单，选择“编辑区域”一旦你创建了自己的自定义布局(或者使用了标准模板)，你只需要在屏幕上拖动窗口之前按下 Shift 键。Up 将弹出你的半透明的“区域”，拖放一个窗口到它们上面，把它放到你之前设计的任何地方。(如果您还启用了“覆盖 Windows Snap 热键”选项，您甚至可以使用键盘在区域之间移动窗口。)

第二个 PowerToys 实用程序，**快捷指南**，非常容易激活，而且非常有用。默认情况下，当你安装并运行 PowerToys 时，它是启用的(你可以通过它的常规设置设置为“登录时启动”，这是我推荐的一个调整)。现在，当您按住 Windows 键几秒钟时，一个方便的弹出菜单将接管您的屏幕，并告诉您可以用来与 Windows 交互的任何和所有键盘命令。(列出的键是您在 Windows 键的*附加*中要按的键。)

如果你不喜欢快捷方式向导，你可以通过 PowerToys 的常规设置将其关闭。在“快捷键指南”的特定设置中，您还可以调整按住 Windows 键以弹出指南所需的时间，以及它的不透明度。
