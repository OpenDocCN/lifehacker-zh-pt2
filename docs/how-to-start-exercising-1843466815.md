# 如何开始锻炼

> 原文：<https://lifehacker.com/how-to-start-exercising-1843466815>

当你不锻炼的时候，阅读关于锻炼的文章就像在立交桥上仰望飞驰的火车。你想，如果我在上面*那里*，我会做得很好，但是我在下面，我没有动力。

今天我们要讨论的是如何开始锻炼，是你现在根本不做，还是你做的没有你喜欢的那么多。

### 如果你有健身背景，但已经有一段时间了

重新开始锻炼可能会很难，如果仅仅是因为你知道你离过去有多远的话。你知道如何跑步，但是你以前“轻松跑”的配速现在感觉像短跑。你很想创造新的个人记录或追逐你的旧目标，但你甚至不能这样做，直到你花几个月——也许几年——爬回到你曾经在的地方。

回到事物 [需要态度转变](https://vitals.lifehacker.com/what-to-do-when-returning-to-the-gym-after-a-long-break-1730933838) 。你可能会做以前做过的同样的练习，但是你以前的思维方式不一定是现在要采用的正确方式。对自己好一点，对自己现在的处境诚实一点。你必须放慢速度或使用比以前更轻的重量。停留在当下，专注于今天的 [比昨天的](https://lifehacker.com/what-to-expect-when-you-start-working-out-again-after-a-1751582922) 好一点。

### 如果你是锻炼的新手

开始永远不会太晚。你会有东西要学， [习惯要养成，](https://lifehacker.com/how-to-finally-start-working-out-even-if-you-hate-it-1729593891) 技能要培养。接受你正在旅行的事实，一开始事情并不容易。你也不会马上看到结果。请记住，你是在做一项长期的工作，并且一致性比你正在做的事情的细节更重要。采纳你能找到的最好的建议，但是不要害怕犯错。没有人一开始就是完美的。无论遇到什么问题，你都能挺过去。

### 从非常小的地方开始

当你开始阅读锻炼计划并和健康的朋友谈论他们做的事情时，你会很快在脑海中建立一个你“应该”做的事情的列表。这没什么不好，只要你记住一件事:*不要试图立刻改变一切*。

挑一个，就一个。 [一个轻松的人](https://lifehacker.com/make-one-of-these-small-changes-for-better-health-inste-1790710017) 。也许你可以每天做几个俯卧撑(即使只是壁式俯卧撑)，或者你可以开始去散步。

如果你想做锻炼视频，选择那些你能真正跟上的，而不是那些在头五分钟就让你筋疲力尽，让你因为跟不上而感到悲伤的视频。永远不要责怪自己身材太差，跟不上某个项目或锻炼。当这种情况发生时，意味着锻炼不适合你的水平。

我喜欢用两周的时间来思考进展。选择一些你可以在两周内加入你生活的小事情。不是“我今天要做这个致命的锻炼”，而是“我要每周做三次中等难度的锻炼。”

两个星期后，你可以添加其他内容。也许你会在不在家锻炼的日子去散步或跑步，这样你每天都有活动可做。或者你会改变你的饮食。或者你可能会把你一直在做的锻炼换成一个更难的。无论你选择什么，做一个小小的改变，再等两个星期。然后重复。

### 找个向导

5K 的 [沙发如此受欢迎是有原因的:因为它给你一个完整的*计划*。你可以坚持九周(或者更长时间，如果你重复其中的几周)，你将永远知道今天要做什么，下周要做什么。](https://lifehacker.com/all-the-questions-youll-have-when-you-start-couch-to-5k-1830857969)

我上面建议的两周结构是一种计划。你可以决定你在每个阶段做什么。如果你喜欢这样的声音，太好了！坚持下去。但是如果你有特定的目标，你会想要找到一个满足它们的计划。

[起始力量](https://lifehacker.com/starting-strength-is-an-awesome-weight-training-program-1790689993) 是一个受初学举重者欢迎的计划，因为它为每周锻炼提供了一个结构，并承诺如果你坚持下去就会取得进步。 [Stronglifts 5x5](https://stronglifts.com/5x5/) 也是这么做的，有一个特别好用的 app。这两个都不适合超级初学者——你需要有人来帮助你学习托举——但是它的结构和简单性使它们很容易掌握。

对于面向不同目标的锻炼程序列表，我喜欢 [这是 r/fitness subreddit 组装的](https://thefitness.wiki/routines/) 。刚开始的时候，只要你在做*的事情*，你选择什么项目并不重要，它可以让你持续变得更加活跃。随着你变得更加健康，你会对自己喜欢什么和不喜欢什么有更好的感觉，无论你接下来尝试什么类型的锻炼计划，你都会有一个健康的基础。

### 探索

你可能不会马上找到你真正喜欢的锻炼方式。有两种方法可以解决这个问题。

一是把锻炼看作是值得的事情，即使它很无聊。有些人真的可以进入每天做同一件事的禅境。这不会让他们兴奋，但会让他们感到不安。

相反的方法是不断尝试不同的东西，直到你找到抓住你心的东西。试试沙发到 5000 米，看看你是否喜欢跑步。没有吗？也许是时候掸掉你旧自行车上的灰尘，看看骑自行车是否适合你。没有吗？抓住一对哑铃，试着举起来。你明白了。

### 忍耐

当你开始锻炼时，你是在要求你的身体做以前从未做过的事情。酸痛是正常的。(你也可能在要求你的大脑做一些新的事情，比如选择合适的运动胸罩或者增加杠铃的重量。放自己一马。)

不过，我们来谈谈 [酸痛](https://lifehacker.com/what-to-do-about-your-sore-muscles-1832462699) 。当你尝试新事物时，你会感觉最糟糕，这是从最简单的锻炼开始的更好的理由。如果你不断改变你的日常生活，你可能会对每次改变感到不舒服。一致性会有帮助。

初学者经常把疼痛当成一个问题:一个必须积极解决的问题，就好像有一种特殊的拉伸或治疗方法可以治愈它。但事实是，酸痛就是会发生，越强越少。人们建议你做的所有缓解疼痛的事情都是为了分散你对疼痛的注意力。我最好的建议就是接受疼痛有时会发生，它会过去的。

有时候初学者会问我，一个狂热的锻炼者，*你只是一直*酸痛*？而答案是:不知道？也许吧？我经常感到这里或那里有一点点疼痛，但并不痛。除了极少数例外——比如在一场比赛之后或者在健身房度过了特别难熬的一周——我几乎没有注意到它。一旦你习惯了新的锻炼计划，你也不会。*
