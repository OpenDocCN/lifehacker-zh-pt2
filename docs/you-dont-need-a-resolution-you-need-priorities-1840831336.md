# 你不需要一个决议，你需要优先次序

> 原文：<https://lifehacker.com/you-dont-need-a-resolution-you-need-priorities-1840831336>

下定决心的日子是有大梦想的日子。你会有一个美好的一年，一个伟大的一月。你最终会养成锻炼的习惯，或者写那本书，或者打扫你的房子——只要你决定其他事情必须付出什么，这些都是可能的。



你做的每件事都需要时间、精力，通常还需要某种形式的金钱。你在健身房度过的每一个小时，都不是你写小说、做饭或整理杂物的一个小时。我相信你；我相信你能做任何事情。但是我不相信你能做所有的事情。

所以，在你朝着你的新习惯或雄心勃勃的目标前进一步之前，花一分钟来决定为了适应这个新事物，你在优先考虑什么。也许你会通过少看电视来读更多的书。也许你需要和你的伴侣协商，让他们在早上你在健身房的时候让孩子准备好，但是你接管了他们的就寝时间，而他们有一个小时的自由时间做他们的宠物项目。发挥创造力是没问题的。你甚至会发现你比你想象的更能适应。

暂时放弃一些东西也是可以的。也许你可以从健身房休息一下，腾出时间来整理一下。诀窍是要有一个重新投入的计划。也许你会把每天的健身房锻炼换成每周两次的家庭锻炼，并暂停你的会员资格一个月。然后，到了二月，又到了健身时间。最好事先计划好，而不是让它意外发生，并且没有恢复的计划。
