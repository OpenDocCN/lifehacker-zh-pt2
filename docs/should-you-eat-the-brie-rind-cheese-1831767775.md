# 你应该吃布里干酪皮吗？

> 原文：<https://thetakeout.com/should-you-eat-the-brie-rind-cheese-1831767775>

*欢迎来到* [***奶酪周***](https://thetakeout.com/tag/cheese-week)*外卖的*为期一周的奶酪好评。** 

* * *

*在这种奇怪的古老生活中，很少有东西像一块美味的软奶酪一样既怪异又美味。它是乳脂状的，通常有点古怪，有时它会有自己的小套装。但是你应该穿那身衣服吗？我总是吃那套衣服——好吧，好吧，T2 皮——因为它很好吃，但是和一群人坐在一大盘烤布里干酪周围，至少他们中的一个会留下那白色的外表。那么谁是对的呢？你能拿起一大杯布里干酪，像咬汉堡一样咬一口吗？我们应该吃果皮吗？* 

*Watch*

*据芝加哥田园工匠奶酪、面包、葡萄酒和 T3 的批发客户经理丽莎·福特曼说，答案是肯定的。“毫无疑问。只有在你真的真的不喜欢的情况下【跳过】。没有理由不这样做。”*

*外壳是白青霉菌，正如她所说的，“美丽的白色外壳”，味道很好。*

*福特曼承认，霉菌过敏可能会让人们望而却步，但即使这样也不能阻止她:“我个人对霉菌过敏，我吃果皮。这是一种不同的模式。”内布拉斯加大学林肯分校的食物过敏资源和研究网络 [支持它](https://farrp.unl.edu/resources/gi-fas/opinion-and-summaries/mold-ripened-cheeses) :*

> *根据食物过敏研究和资源项目(FARRP)主任史蒂夫·泰勒博士的说法，没有证据表明发霉的奶酪对霉菌过敏的人有潜在的危害。患有霉菌过敏的消费者通常对霉菌孢子的吸入有反应。另一方面，摄入发霉的奶酪通常包括摄入菌丝体，而不是孢子。虽然奶酪中可能确实存在一些孢子，但消化过程可能会破坏过敏原。*
> 
> *一些医生确实警告他们的病人不要食用发霉的奶酪，但这种警告并不是基于泰勒博士所知道的任何确凿的证据。他进一步补充说，环境中的主要霉菌过敏原是链格孢属，链格孢属不会是奶酪中常见的霉菌腐败物种(更可能是腐败的水果)。*

*关于你应该吃什么或不应该吃什么的问题:这应该是不言而喻的，但并不是所有的奶酪皮都应该狼吞虎咽。蜡皮奶酪，或者有塑料涂层的奶酪，显然不应该吃。“它是可以吃的，不会要你的命，”福特曼说，“但这不是一次愉快的经历。”她还建议不要吃天然皮奶酪 ，它们看起来像岩石或石头——不仅这些皮不适合食用，而且它们通常是奶酪的唯一包装，所以不太卫生。“只有真正的疯子才会吃那些...如果它看起来像一块石头，不要吃它。”(总的来说是个好建议。)* 

*但这不仅仅是因为吃布里干酪的外皮是安全的或可接受的。弗特曼说，果皮是吃布里干酪“不可或缺的一部分”。质地的差异是显而易见的，但更重要的是，布里干酪上的青霉菌皮，开始是一层白色，但随着时间的推移，变成象牙色或几乎黄色，拥有最强烈的味道。对于陈年奶酪来说尤其如此，“但即使奶酪非常非常年轻，它也会给这种体验添加一种特殊的风味。”*

*其他带皮的软奶酪也是如此。在我偶尔做一名 [高级威士忌店员](https://thetakeout.com/6-whiskey-recommendations-from-a-fancy-whiskey-shopgirl-1828587316) 的工作中，我也和高级奶酪打交道，我注意到当人们拿起一种洗了皮的奶酪时，他们经常会感到担心，因为他们可以透过包装闻到这种味道。像 [郁金香树奶油厂](https://www.tuliptreecreamery.com/) 的毛地黄(我个人最喜欢的一种)，香气非常强烈，即使是通过塑料，顾客们通常认为这意味着它已经不知何故地变了。事实完全不是这样，福特曼解释了原因:“水洗皮奶酪有一种粉橙色的外皮，吃起来棒极了...如果你喜欢臭奶酪，那将是奶酪中最臭的部分。”*

*至于烤布里干酪，如果你喜欢奶酪中浓郁的味道，就不要跳过外层，而选择黏糊糊的中层。我知道这很诱人，但正如福特曼所说，“加热奶酪会让任何强烈的味道更加强烈。”*

*她的最后一条建议也与加热有关:如果你不喜欢更臭的奶酪，就不要加热它。如果你不想要果皮，给我留着。*
