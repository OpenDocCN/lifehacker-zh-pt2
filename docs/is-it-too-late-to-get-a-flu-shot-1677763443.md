# 现在注射流感疫苗还来得及吗？

> 原文：<https://lifehacker.com/is-it-too-late-to-get-a-flu-shot-1677763443>

所以你认识的每个人都生病了，如果你生病了，你就抽不出时间去工作。吃得好、睡得好、经常洗手是有帮助的，但是现在也许你正在后悔今年没有注射流感疫苗。太迟了吗？



好消息:只要流感仍在传播，注射流感疫苗 [永远不会太晚](http://www.fda.gov/ForConsumers/ConsumerUpdates/ucm384535.htm) ，即使疫苗不是完美匹配，也仍然值得。唯一可能“为时已晚”的感觉是，如果你等得太久，药店可能最终会告罄。提前打电话确定他们有疫苗库存。

看看疾病控制中心的数据到目前为止这个季节的流感活动。很明显有很多流感还没有发生。在大多数年份， [流感活动在 2 月](http://www.cdc.gov/flu/about/season/flu-season.htm) 左右达到高峰，虽然每个季节都有点不同。流感疫苗大约需要 [两周才能生效](http://www.cdc.gov/flu/protect/keyfacts.htm) ，所以越早接种越好。这就是为什么最好在流感季节开始前注射疫苗——当它蔓延到你的社区时，你将得到保护。(这两个星期的窗口期也解释了为什么人们有时会觉得流感疫苗让他们得了流感，或者疫苗没有保护他们:如果你在这两个星期的窗口期内患了流感，疫苗没有足够的时间生效。)

但是如果疫苗与今年流行的流感病毒不匹配呢？你仍然会得到相当多的保护。这是疾病预防控制中心关于过去十年疫苗有效性的数据(今年的数据还没有出来)。近年来最低的效率是 2014-2015 年的 19%，这比零好得多。在好的年份，比如 2010-2011 年，有效性是 60%。过去十个赛季的平均值是 44%。

想想吧。你可以免于感染几乎一半的流感病毒，免费接种，或者你可以碰碰运气，要么自己感染流感，要么让它潜伏足够长的时间，把它传染给别人。这是显而易见的。

关于“如果你健康，你就不需要注射疫苗”的神话，最后说一句:流感并发症风险最高的人群是 5 岁以下的儿童、65 岁以上的老人、孕妇、哮喘或糖尿病患者或几乎任何严重的健康问题，如心脏病和癌症。如果你的生活中有这些人， [你的流感可以变成他们的流感](https://vitals.lifehacker.com/hey-asshole-the-flu-shot-isnt-about-you-1829720979) 。
