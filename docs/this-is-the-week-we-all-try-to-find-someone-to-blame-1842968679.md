# 这一周，我们都想找个人来责怪

> 原文:[https://life hacker . com/this-is-the-week-that-we ' s-all-that-the-week-to-find someone-to-kill-1842968679](https://lifehacker.com/this-is-the-week-we-all-try-to-find-someone-to-blame-1842968679)

这场危机中的每一周似乎都有一个主题。三月初的那一周，我们中的许多人第一次经历了禁闭，这是一个在家锻炼和吃酸面团面包的天真的一周，希望这一切很快就会过去。就在不久前，有一周我们都在争论口罩的问题，包括口罩是否有用，以及谁应该戴口罩。本周，在我看来，一种即将爆发的怨恨已经冒泡，本周的主题是:责备。

Watch

没有人对这种情况感到高兴，对吗？我们要么害怕病毒，对限制感到恼火，不顾一切地想回去工作，要么是以上几种情况的结合。在本周的圆桌会议中，我认为只要稍微考虑一下，我们就可以对谁在指责谁，以及为什么进行元讨论。我看到的几件事情:

### 地方政府的抗议

这些都是极少数人，所以我们要么谈谈为什么人们觉得不得不 [出来抗议](https://nypost.com/2020/04/20/hundreds-of-protesters-defy-lockdown-orders-in-pennsylvania/) ，要么首先谈谈 [组织和煽动抗议](https://www.buzzfeednews.com/article/salvadorhernandez/coronavirus-quarantine-protests-facebook-groups) 背后的动机。这里的指责对象:主要是地方政府，因为他们发布了居家命令，阻止人们工作。

当然，聚集人群提出这些要求是具有讽刺意味的。我听说过抗议者争论说新冠肺炎不是真的，或者 [说它只是流感](https://lifehacker.com/sorry-we-did-not-all-get-and-recover-from-covid-19-thi-1842798877) ，或者其他反科学的谎言。一些对事实的否认是必要的，以证明是时候取消限制了。

### 对世界卫生组织的怀疑

这一个对我来说总是感觉有点奇怪，因为世卫组织是国际性的，在大多数时候给我的印象是中立和真实的。但每当我从 Twitter 上收到世卫组织简报开始的提醒时，对简报推特的最初几条回复是人们抱怨世卫组织被某个国家或领导人或其他人控制了。“这是一场大规模的有计划的学术骗局”，今天的回复中有一条这样写道。(Lifehacker 的文章也得到了这个待遇；请看一下 [这个帖子从今天起](https://vitals.lifehacker.com/what-it-will-take-to-reopen-everything-according-to-th-1842962652) 关于世卫组织的建议重新开放。)

关于世卫组织的评论通常带有反华的意味。 [《华盛顿邮报》这篇关于白宫、疾病预防控制中心和世卫组织之间政治的文章](https://www.washingtonpost.com/world/national-security/americans-at-world-health-organization-transmitted-real-time-information-about-coronavirus-to-trump-administration/2020/04/19/951c77fa-818c-11ea-9040-68981f488eed_story.html) 推测，总统想指责中国，但出于政治原因不得不改变指责方向，并最终抨击世界卫生组织隐瞒信息。

### 无视规则的个人

并非所有人都参与了一场明显的政治指责游戏。有时，我们不是针对某个政府或政策，而是彼此反目。因此，我看到人们大声争辩说，个人应该为未来的爆发浪潮负责。这些人可能包括不戴口罩走在人行道上的人，或者去佛罗里达重新开放的海滩的人。

尽管这种指责没有错，但(对我来说)令人愤怒地放错了地方。我们最大的问题包括政府的失误，导致我们在没有测试或任何方法来处理接触追踪的情况下感染了猖獗的病毒。(我估计这就是我在怪谁。)我也害怕看到人们在拥挤的公园里无面具慢跑，或者在复活节接待游客。

但我也理解走出家门的渴望，我认为我们的烦恼可以更建设性地集中在那些有能力做出更彻底改变的人身上，比如财政援助，这样人们就不会感到如此绝望。

反正就是我。你在哪里看到被指责的人？什么问题或错误信息不断出现？你希望我们用愤怒/紧张的精力做些什么？
