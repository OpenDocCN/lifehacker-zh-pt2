# 如何在 Windows 启动时阻止讨厌的应用程序加载

> 原文：<https://lifehacker.com/how-to-stop-annoying-apps-from-loading-when-windows-boo-1829810394>

你最后一次坐下来思考每当你在台式机或笔记本电脑上启动 Windows 10 时会自动启动多少应用是什么时候？你可能不记得了，因为这不是大多数人会做的事情。但是你应该这样做，因为你不需要一堆不必要的应用程序毫无理由地消耗你的系统资源。如果你有一台动力不足的电脑，你只会让你的情况变得更糟。而且，至少，拥有一堆后台应用程序会让 Windows 10 需要更长的加载时间。

### 如何查看(和更改)从您的电脑启动的应用

点击开始按钮，输入“任务”，选择任务管理器选项，打开 Windows 10 的任务管理器。简单。从那里，点击启动标签，你会看到一个漂亮的，当你的系统加载时加载的每个小应用程序的长列表:

尽管开始大规模禁用项目可能很诱人，但你会希望对你停止的项目进行战略规划。这不是一门精确的科学，但是这里有一些指导方针可以帮助你开始学习:

*   如果该应用程序看起来像一个微型应用程序，可以帮助你启动一个你并不经常使用的更大的应用程序(例如 iTunes Helper 或 Spotify)，那么禁用它可能是安全的。
*   如果应用程序需要一直运行，以便您从中获得最大利益(Dropbox、Razer Synapse 等。)，那么你大概应该让它在启动时启动。
*   如果 app 通过在后台加载(Steam、Battle.net 等)让你的生活稍微方便一点。)，具体问题具体分析。例如，你真的需要后台下载游戏更新吗？或者你可以让一个应用程序只在你启动的时候下载这些更新吗？
*   如果你不确定——因为一个应用程序有一个非常奇怪的名字，或者你不知道它是什么或它在做什么——使用你最喜欢的搜索引擎来获得更多关于它是什么、它有什么作用以及它是否有必要的信息。Windows 10 让这变得更加简单，因为你可以在任务管理器中右键单击任何应用程序，然后选择“在线搜索”来完成这一操作。
*   如果需要一点帮助，你也可以安装一个类似 [的应用程序，我应该删除它](https://www.shouldiremoveit.com/index.aspx) ，它使用“群众的智慧(当然还有我们的技术专家)”来帮助你找出什么可能是安全的，什么可能是你想要保留的。

### 我的启动列表

为了让你开始，下面是我在电脑上启用和禁用的各种应用程序的快速分类(以及原因)。

#### **使能**

*   Battle.net——只是因为我不喜欢在玩游戏前等待游戏更新。我可以让这个*而不是*在启动时加载，几乎不会错过任何东西。

*   bzbui.exe——这听起来很奇怪，但实际上这只是 Backblaze，我的不间断备份工具。如果启动时不运行，我就真的卡住了。
*   我用这个代替 Windows 10 的内置音量控制，所以它也必须在 Windows 启动时运行
*   **Dropbox** —与 Backblaze 的交易相同；如果它不运行，它就不会与云同步文件，所以还不如让它在我的电脑启动时加载。
*   **蒸汽**——和 Battle.net 一样
*   我在启动时运行这个程序，这样我就可以捕捉到我的*魔兽世界*插件的任何更新。我这么做只是为了提醒自己*要*更新我的插件，否则，只要你在电脑上启动应用程序，就不会花很长时间
*   这样我就可以永远和团队生活黑客聊天了

#### **禁用**

*   **任何与 Adobe 相关的东西** —我真的不需要任何帮助来启动 Acrobat、PhotoShop 或任何相关更新。
*   **任何与 iTunes 相关的东西**——同样的事情——在我需要的时候加载 iTunes 从来都不是问题，所以为什么要在后台运行一些奇怪的“助手”应用程序呢？
*   Spotify——类似于 iTunes，我可以在需要的时候启动它。
*   **Macrium Reflect** —我喜欢这个应用程序，但我不经常需要访问这个磁盘映像工具，所以没有必要在 Windows 启动时加载它。
