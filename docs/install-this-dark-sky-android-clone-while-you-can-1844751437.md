# 安装这个“黑暗天空”机器人克隆，而你可以

> 原文:[https://life hacker . com/install-this-dark-sky-Android-clone-while-you-can-1844751437](https://lifehacker.com/install-this-dark-sky-android-clone-while-you-can-1844751437)

“黑暗天空”是许多安卓用户的首选天气应用。也就是说，直到它被苹果公司收购并成为第一方产品后，才从谷歌商店下架。几个安卓应用程序 [提供类似的超本地化天气预报，](https://lifehacker.com/these-weather-apps-are-great-alternatives-to-the-apple-1842591377) 但我们中的一些人对黑暗天空的消失并不满意。幸运的是，一位匿名开发人员拉了一个弗兰肯斯坦，并在 Android 上复活了“黑暗天空”。

黑暗天空是最初的黑暗天空 Android 应用程序，作为一个单独的应用程序重新创建。除了一些关键的例外，它的外观和工作方式几乎与逝去的原作一模一样。

你将无法使用 Dark Sky 的付费功能，如恶劣天气警报通知或提交自己的天气报告，即使它们列在 Dark Sky 应用程序中。匿名开发者告诉 [XDA 开发者](https://www.xda-developers.com/dark-sky-android-bring-back/) 他们不打算让这些工作，所以你将被免费的预测功能所困——公平地说，这是周围最好的一些。该应用程序显示温度，风力和降水报告，加上其他有用的工具，如雷达视图和“时间机器”工具。黑暗主题也可以。

另一个快速警告:逆向工程和重建一个应用程序是一个大胆的举动，但对苹果公司拥有的应用程序这样做是公开招致报复，因为大多数试图扮演上帝的人经常面临这种情况。《黑暗天空》的寿命可能会很短，所以下面是如何下载并安装它的方法。

《黑暗的天空》必须被下载到你的设备上——因为显而易见的原因，你在 Google Play 上找不到它。侧装很容易，但安装非官方的。APK 文件会让你的安卓手机感染恶意软件。也就是说，我相当有信心推荐《黑暗的天空》;我已经下载并测试了该应用程序，没有任何问题，XDA 开发人员报告说，类似的顺利，无恶意软件的操作。知道潜在的风险是值得的，但这次你可能没事。

### 如何在 Android 上安装 Darker Sky

1.  安装 [电报 app](https://play.google.com/store/apps/details?id=org.telegram.messenger&hl=en_US) 如果你还没有的话(你需要它来下载 app 文件)。
2.  [在手机上打开此链接](https://t.me/darkerskyandroid) 查看更暗的天空 app 官方电报组。
3.  点击聊天记录中的**“下载更暗的天空”**链接。(不愿意可以不入群。)
4.  轻按下载横幅或在设备的“下载”文件夹中找到并轻按“更暗的天空 APK”进行安装。
5.  如果出现提示，允许 Telegram 安装应用程序。
6.  按照屏幕上的说明完成安装。
7.  从你的设备的应用程序启动屏幕运行黑暗天空。

未来的任何更新都会发布在该应用的官方 Telegram 服务器上，只要它可用。如果你想使用《黑暗天空》开发者添加的任何新功能，你需要安装这些新版本。

如果《黑暗天空》收到撤下通知，它可能会停止工作。如果发生这种情况，你必须寻找一个替代的天气应用程序来获得你的气象定位。谢天谢地， [你有足够的](https://lifehacker.com/these-weather-apps-are-great-alternatives-to-the-apple-1842591377) 可供选择。
