# 如果你想提高你的信用评分，定期检查

> 原文：<https://lifehacker.com/if-you-want-to-improve-your-credit-score-check-it-regu-1841608464>

想知道如何增加你的信用评分？做出改进的第一步可能是简单的意识。



来自 Javelin Strategy &研究和信贷局 TransUnion 的一项新调查发现，超过一半的美国消费者至少每月回顾一次他们的信用评分。如果这还不能打动你，考虑一下这个金块:很多被调查的人有次级(低)信用评分，并在一年的时间里监控他们，他们能够提高他们的分数。来自《今日美国》的独家报道:

> 超过三分之一的次级消费者(34%)在 2018 年 3 月至 2019 年 3 月期间监测了他们的信用，将他们的信用评分提高到接近优质或以上的信用风险等级。同期，那些没有监控自己信用的消费者的这一比例下降了近一半，至 18%。

当然，检查你的信用评分实际上并没有什么作用:它只是给你一个分数，除非你申请新的信用，否则这个分数实际上并没有什么用处。

但是免费的信用监测网站和金融机构提供的免费评分是最近才兴起的，所以仍然有一种新奇的感觉。每个人都想知道自己的状态。这是基本的心理学。

还记得你什么时候可以找到你的优步客户评级，但前提是你知道该去哪里找吗？每个人都想知道他们的分数。同样，有整个论坛专门讨论如何获得完美的信用评分。

但是，除了新奇之外，免费分数来源的优势似乎正在帮助人们。信用机构益百利最近关于信用状况的年度报告发现，2020 年千禧一代的信用分数比 2012 年高出 25 分。美国人达到 FICO 得分 700 的平均年龄是自 Experian 2012 年开始记录以来的最低水平——现在是 54 岁，而不是 62 岁。

你的信用评分并不能证明你的信用报告中的一切都很好，但它为熟悉你的分数正常范围的人提供了一个快速浏览和一般健康检查。

这也有助于现在有很多免费的方法来得到你的分数。 A 近 40%的 TransUnion 受访者表示，他们从不止一个提供商那里获得信用信息，银行和信用合作社是最主要的来源。尽管有许多公式可用于计算信用评分，能够看到你的分数从几个来源给你一个坚实的想法是什么贷款人会看到当他们通过自己的选择方法访问你的分数。

你当然不需要每天检查你的信用评分——那只是多余的——但是定期检查你的分数可以帮助你保持在正轨上，如果你正在齐心协力改善它的话。当然，如果有什么不对劲，这是一个快速指示器。

你分数的组成部分比总分更重要吗？是的，因为它们能更好地反映你的财务状况。但是，如果你在更好地管理你的钱或减少债务方面取得了进展，获得这个分数可以增强你的信心，给你一点动力。
