# 用味精做一杯绝对肮脏的马提尼酒

> 原文:[https://life hacker . com/make-an-absolute-dirty-martini-with-msg-1836892332](https://lifehacker.com/make-an-absolutely-filthy-martini-with-msg-1836892332)

肮脏的马提尼酒让我成为今天的杜松子酒饮用者，为此我欠他们一份情。尽管我现在更喜欢简单的五比一比例的杜松子酒和苦艾酒(加一颗橄榄)，但我仍然时不时地渴望一杯肮脏的 boi。如果你想要超级时髦，你可以用橄榄汁，或者你可以保持你的液体比例不变，并补充一点结晶鲜味——又名“味精”——这是一种绝对肮脏的马提尼酒，没有用盐水稀释。
