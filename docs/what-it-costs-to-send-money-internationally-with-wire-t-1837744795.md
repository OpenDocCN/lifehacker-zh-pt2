# 使用电汇应用程序进行国际汇款的费用

> 原文：<https://lifehacker.com/what-it-costs-to-send-money-internationally-with-wire-t-1837744795>

长久以来，国际汇款似乎有两种选择:在西联汇款柜台排队，或者在速汇金柜台排队。但是 P2P 支付应用的兴起意味着有更多的选择可以直接从你的手机向世界各地汇款。



由于几个原因，弄清楚这些服务的费用结构可能会令人困惑。首先，每个国家都有不同的用户收钱方式，每个国家都有不同的费用。费用也会根据你汇款的多少而有所不同。然后是汇率，这可能会影响你的收件人实际带回家的钱数。

为了尽可能简化事情，我们对五个主要参与者进行了测试:MoneyGram、TransferWise、Xoom、Western Union 和 WorldRemit。我们找到了从美国向加拿大、墨西哥、印度、菲律宾和英国汇款的费用。

**结论:**在开始流程之前，先尝试 MoneyGram 的简单、低廉且易于估算的费用。

让我们来看看通过五个转账网络中的每一个将 500 美元汇至这些地点的成本。

### [速汇金](https://www.moneygram.com)
