# 没有上身力量怎么开始攀岩

> 原文：<https://lifehacker.com/how-to-start-rock-climbing-when-you-have-no-upper-body-1833498240>

像任何其他看似硬核的锻炼场所一样，室内攀岩馆对第一次攀岩的人来说可能是令人生畏的地方——但也不尽然。事实上，对攀岩的一个普遍误解是，要开始攀岩，你必须有一吨的上肢力量。

“那不是真的，”真正的攀岩者苔丝·约翰逊说，她是纽约布鲁克林巨石城 的攀岩教练。当然，上半身的力量在攀岩中不会有伤害，但实际上是技术和坚持才能真正帮助你到达顶峰。

“我的学生中有 70%不会做引体向上，不会做俯卧撑，或者很难做，”苔丝说。缺乏上肢力量实际上可能是一个秘密武器，迫使你专注于适当的技术，而不是使用蛮力来攀登岩石表面。

“学习技术比锻炼肌肉更难，”苔丝说，她发现自己经常分享一些技巧，帮助初学者在墙上感觉更自在。

由于越来越多的室内攀岩场出现在城市地区，攀岩运动曾经是登山者和户外冒险家的领地，现在正迅速成为主流。这些健身房通常提供几种不同类型的攀岩运动。有些，像顶绳和铅攀登，利用绳索和其他设备。但许多新人从抱石开始，这是一种不需要任何花哨装备的攀岩方式，因为你很少能爬到离地 15 英尺以上。开始抱石唯一需要的装备是一双特别设计的鞋子，在大多数室内健身房都可以租到。

无论一个人的健康水平、体型或性别如何，每个人都可以通过遵循以下六条简单的攀岩规则得到很好的全身锻炼，并获得很多乐趣:

### **想猴子杠，而不是霸王龙的胳膊**

**新手错误:**悬空在离地面几英尺的地方，我们很多人会本能地用二头肌把脸尽量贴近墙壁。但是用泰丝所说的“霸王龙的胳膊”抓着墙，意味着你会消耗大量的能量，很快就会疲惫不堪。

相反，试着伸直手臂晃来晃去，就好像你是一个在金钱棒上荡来荡去的小孩。总的来说，你会有更好的耐力(而且你不会马上被认为是第一次攀岩)。

### **扭扭肩膀，把手伸得更高**

新手的错误:第一次攀岩者的另一个自然本能是倾向于保持你的肩膀完全平行于墙面并垂直于地面。

但是由于腋窝和臀部牢牢地粘在墙上，你的活动范围将受到限制，使得远距离的抓地力似乎不可能达到。

想象把你的肩膀扭成一个角度，类似于自由泳运动员把一个肩膀向后转，另一个肩膀向前转，以延长他们的行程。

一旦你开始尝试你肩膀的角度，你的臀部也会跟着改变。很快，你会发现你的手伸得更长了，你能够抓住那些曾经感觉遥不可及的东西。

### **用你的脚趾，而不是你的脚**

新手错误:众所周知，攀岩鞋会紧贴脚趾，但这种紧密的合脚是有原因的——它们的设计是为了将你所有的重量都集中到橡胶尖头上。大多数初学者常犯的一个错误是通过踩足弓来爬墙，就像爬梯子一样。

**诀窍:**利用鞋子的设计，踮着脚爬墙。用你的脚趾而不是你的足弓保持平衡，你将能够轻松地转动，并且你将能够用你的脚趾站立以获得更高的高度。

### **用臀部引导，而不是用手**

新手错误:攀岩时，很容易想到用手抓住下一个点，很难想到用身体的其他部分。在不稳定的平衡状态下，想要伸手抓住的冲动通常意味着你的身体会下意识地通过向相反方向倾斜臀部来平衡。

**诀窍:**当你试图向身体右侧的扶手移动时，不要用手去抓。相反，用你的右臀向它倾斜。一旦你的重心进一步向右侧集中，右手握拍就容易了。

### **规划您的路线**

新手的错误:作为一个新攀岩者，没有仔细考虑路线就跳到墙上是很诱人的。但“先爬后问”的方法意味着你更有可能被困在墙的中间，在你试图想出下一步行动时消耗宝贵的能量。

去任何一家攀岩馆，你都可能会看到攀岩者专注地盯着墙壁，然后喃喃自语，并用手比划动作。这些攀登者在想象他们攀登岩壁时的动作。

在这个例子中，苔丝仔细研究了一条路线(用黄色把手标记)，然后在她跳到墙上之前想象每个动作。这是一项有益的练习，可以最大限度地减少她被困在墙中间的可能性，因为她不知道接下来会发生什么，所以会消耗宝贵的能量。

对于初学者来说，养成“排序”习惯或想象一次攀登的好方法实际上是观察其他攀登者，尤其是那些和你差不多大的人，看他们如何完成一条路线。

### **爬坡后拉伸**

攀岩不同于大多数其他运动，因为它几乎利用了你身体的每一块肌肉。对大多数人来说，最容易疲劳的是我们的手臂，尤其是手和前臂。

“几乎没有人从前臂力量开始，”苔丝说。这就是为什么知道一些简单的伸展运动是个好主意，可以帮助你重新开始并继续攀爬，或者只是在你完成一天的工作后帮助你的肌肉感觉更好。

第一个是前臂伸展，旨在针对手腕和前臂内侧。你可以这样做，把你的手掌压在墙上或地板上，手指向下指向脚趾。这将拉伸你前臂的肌肉，这对新攀岩者来说是最容易疲劳的肌肉。

接下来是三头肌拉伸，这将有助于缓解肩膀和上臂的紧张。你可以通过将一只手肘弯曲到头顶上方，用另一只手将手肘靠近中线来做这个伸展动作。你应该感觉到上臂后部的拉伸。或者，你也可以用一根跳绳作为道具，像 Tess 上面演示的那样进行拉伸。

除了所有这些建议，攀岩最重要的事情就是走出去，认识到攀岩馆是一个友好的地方，欢迎初学者，更高级的攀岩者也很乐意分享技巧。
