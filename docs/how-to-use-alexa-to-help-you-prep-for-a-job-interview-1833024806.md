# 如何使用 Alexa 帮助你准备工作面试

> 原文:[https://life hacker . com/how-to-use-Alexa-to-help-you-prep-for-a-job-interview-1833024806](https://lifehacker.com/how-to-use-alexa-to-help-you-prep-for-a-job-interview-1833024806)

如果你已经很久没有参加过面试了，那么即使是最基本的面试问题也会让你陷入困境。如果你正在经历面试过程，那么一点点的准备可能意味着你是表现得像一个优雅的候选人还是一塌糊涂。

Watch

在我看来，最好的解决办法是让一个朋友和你一起经历一些潜在的面试问题，然后是另一个朋友，再是另一个。你练习得越多，你就会越好。并且坦率地说有人听你的回答会告诉你听起来像头蠢驴和/或你没有足够地自吹自擂，这很好。如果你没有朋友和你一起解决问题，那么 Alexa 也有一些技巧可以帮助你加快面试准备。

这种方法的想法是通过一系列不同的面试问题，然后思考你应该如何回答它们。每个人都可以回答这个老套的问题:“你认为自己五年后会怎么样？”询问他们是否事先考虑过自己的答案，但如果你没有，你可能会停顿一段不舒服的时间，而你想出一些好的东西，但并没有真正回答，或者回答，但没有尽全力。

[面试准备](https://www.amazon.com/Alexander-King-Interview-Prep/dp/B01LW7L2VD?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-use-alexa-to-help-you-prep-for-a-job-interview-1833024806&asc_source=&tag=kinjalifehackerlink-20) 是 Alexa 的一项技能，它为各种不同的工作类型和工作领域的面试提供了数百个不同的问题。例如，你可以问一些与工作表现或管理相关的问题，或者你可以问一些在像亚马逊和谷歌这样的大公司的面试中使用的问题。

Alexa 显然不会听你的答案并给你提供反馈，但它们可以作为很好的提示，让你开始思考你可能会被问到的问题。因为他们是随机向你走来的，所以也有一点令人惊讶的面试元素。

如果你成功地完成了面试准备中的所有事情，这里有一些其他类似的技能可以为你提供更多。先说几个好的: [面试我](https://www.amazon.com/nickviola-Interview-Me/dp/B01N6UNXQU?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-use-alexa-to-help-you-prep-for-a-job-interview-1833024806&asc_source=&tag=kinjalifehackerlink-20) 和 [求职面试问题](https://www.amazon.com/gp/product/B07DSDSJML?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-use-alexa-to-help-you-prep-for-a-job-interview-1833024806&asc_source=&ref=skillrw_dsk_pnps_dp_2&tag=kinjalifehackerlink-20) 。
