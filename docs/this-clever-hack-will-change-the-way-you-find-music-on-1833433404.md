# 这个聪明的方法将改变你在 Spotify 上寻找音乐的方式

> 原文：<https://gizmodo.com/this-clever-hack-will-change-the-way-you-find-music-on-1833433404>

我最喜欢的 Spotify 一直是音乐发现。因此，当该公司开始建立像《探索周刊》和《释放雷达》这样的个性化播放列表时，我就被迷住了。唯一的问题是听所有的新音乐要花很长时间。但是来自 Spotify 开发者的一个独立项目提供了一个惊人的探索数百万首歌曲的方法。他们称之为。



新的 Discover Quickly web 应用程序以专辑封面网格的形式显示您的 Discovery Weekly 播放列表。当鼠标悬停在某首歌曲上时，歌曲会在中途开始播放，您可以点按专辑封面来放大特定歌曲和艺术家。点击一首曲目也会将其保存为一个列表，您可以快速将其导出到 Spotify 作为新的播放列表。这有效地允许我在几分钟内整理我的每周发现播放列表，而不是必须听完整个。

这种新的浏览方式也不仅仅适用于个性化播放列表。你基本上可以在 Spotify 上调出任何播放列表，做同样的悬停和点击操作。一旦你放大任何给定的音轨，你也可以放大该艺术家的其他作品以及相关的专辑。我发现很容易就掉进一个又一个音乐的兔子洞，一路上发现各种各样的新事物。顾名思义，Discover Quickly 增强了最初让 Spotify 如此伟大的音乐发现元素。

Spotify 的设计师艾丽莎·奥弗里蒂格(Aliza Aufrichtig)在一封电子邮件中告诉 Gizmodo，“在快速发现中，你可以在普通的 Spotify 应用程序上做的事情很少，但我们选择以视觉化的方式快速浏览音乐。”“我们最初版本最大的增加之一是能够在许多不同的地方开始探索。”

再次值得注意的是，Discovery Quickly 不是 Spotify 的官方项目，尽管 Aufrichtig 和她的开发合作伙伴 Edward Lee 确实在 Spotify 的年度黑客周上开始了这项工作。Aufrichtig 解释说，她对日本音乐感到好奇，希望有一种方法可以轻松浏览一堆歌曲。使用 Spotify Web API，她从 1000 张日本专辑中提取了 30 秒的剪辑和专辑封面，在她看到将鼠标悬停在一张图片上并聆听一首歌曲的能力是多么巧妙之后，快速发现的概念诞生了。

“我把它给爱德华看了，”奥弗里蒂格说，“他说，‘哇，太酷了，但我还想要更多。如果你可以像迷失在维基百科中一样，无限地走进音乐的兔子洞，会怎么样？比如，如果你看到一个相关的艺术家，然后点击他们，可以进一步探索。"

维基百科的兔子洞类比其实很完美。实际上,“快速发现”可以吸引你，带你进入你从未意识到存在的流派，因为它们与你已经喜欢的音乐有某种联系。为了让事情变得更有趣，你可以浏览歌曲，就好像你在链接上徘徊，看看下一步去哪里。
