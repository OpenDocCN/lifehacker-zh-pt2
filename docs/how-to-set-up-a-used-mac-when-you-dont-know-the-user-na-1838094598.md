# 当您不知道用户名或密码时，如何设置二手 Mac

> 原文：<https://lifehacker.com/how-to-set-up-a-used-mac-when-you-dont-know-the-user-na-1838094598>

在本周的 [**科技 911**](https://lifehacker.com/c/tech-911) 中，我们正在解决一个有趣的问题。我希望我认识一个比我年长的人——一个父母、一个兄弟、一个很棒的邻居——他们有很多笔记本电脑、智能手机和小玩意，当他们厌倦使用它们时，他们可以从我身边经过。



不幸的是，我不知道，但是《生活黑客》的读者凯尔肯定知道。虽然免费的东西总是有趣和受欢迎的，但他最近的收购有一个怪癖，他需要帮助解决。他写道:

为了回答这个问题——至少是第一部分——我将假设您是在问在没有用户名和密码的情况下如何登录一台机器。不过，这个问题说明了一个更重要的问题:我如何*访问*一台被其他人锁定的机器？

幸运的是，我们不需要担心任何疯狂的恢复工具或密码破解建议，因为你的最终目标不是进入一个不属于你的账户，而是使用一个之前创建过该账户的系统。换句话说，我们可以将你的奶酪刨丝器重置为出厂默认设置，并将其设置为一个全新的系统，在此过程中消除登录问题。

做到这一点的简单方法是——假设你的 Mac 的前主人没有设置 [固件密码](https://support.apple.com/en-us/HT204455) ，这使得事情变得更加棘手——当你启动它时，只需启动 Mac 的 [恢复模式](https://support.apple.com/en-us/HT201314) 。打开电脑电源，立即开始按住 Command+R。继续这样做，直到你看到屏幕上发生了什么事情(通常是苹果标志或旋转的地球仪)，然后你可以释放按键。

如果您在恢复启动时没有看到 macOS“实用程序窗口”，而是看到类似“macOS 恢复”的内容，并提示您输入用户密码，请单击“恢复助理”并选择“抹掉 Mac”。点击蓝色的“擦除 Mac”超链接，让我们开始——这将擦除您的系统，并允许您从头开始设置新版本的 macOS(使用您想要的任何用户名和密码)。

否则，如果您进入主 macOS“实用工具”窗口，请启动“磁盘工具”。选择安装了 macOS 的驱动器，然后选择“抹掉”。完成后，退出“磁盘工具”,并通过“macOS 工具”主屏幕中的便捷链接开始重新安装 macOS。

我希望给你新 Mac 的人禁用了 iCloud(以及“查找我的…”特征) [之前咯](https://lifehacker.com/disable-icloud-before-you-get-rid-of-your-mac-1823425930) 。否则，您应该可以通过 [重置 Mac 的 NVRAM/PRAM](https://support.apple.com/en-us/HT204063) ，从而删除“查找我的”关联。然后，您可以登录您的 iCloud 帐户，您应该能够设置您的 Mac，就好像它永远是您的一样。(您也可以像正常的 一样登录 iCloud [，而无需重置您的 NVRAM/PRAM，以将已擦除的 Mac 与您自己的 iCloud 帐户相关联。我会尝试这两个步骤。)](https://medium.com/@mulligan/how-i-sold-an-old-mac-and-unknowingly-tracked-its-location-for-over-3-years-9a35cd3ca4cf)

这应该能让你开始使用你的新 Mac。如果没有，或者如果你仍然遇到任何问题或困惑，请回信！
