# 用一个破发夹打开拉链

> 原文：<https://lifehacker.com/open-zip-ties-with-a-broken-bobby-pin-1843882788>

如果你有一些不该绑在一起的东西——不管是什么，不管是什么原因——一个折断的发夹会是一个很好的释放工具。



是的，我知道在大多数情况下(例如，如果你正在打开一个包裹),你会想要用一把剪刀剪断拉链。但是在一些情况下，你手边可能没有剪刀，在一些情况下，你可能需要一个可以很容易藏在手中的工具。

这就是发夹的由来。导演兼电影制作人艾娃·德约列最近分享了一段来自抖音的视频，展示了这是如何做到的:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1268054695446118402&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1268054695446118402&autosize=1) 

以下是你自己的做法:

*   如果你不喜欢用发夹做发型，在头发上别一个发夹，或者在口袋里放一个。
*   当你发现某样东西的拉链拉得不合适时，把发夹从头发上取下，开始来回弯曲扁平的一端，直到它断裂。(这可能需要一段时间。坚持下去。)
*   一旦你有一个折断的发夹，将折断的扁平的一端插入拉链带，这样它会压下抓住带子的舌头。这就松开了拉链带的其余部分，然后可以随意拉开。
*   如果你看到其他人在处理不方便的拉链，把发夹传来传去和/或提供帮助。

如果你手边没有发夹，或者不想花时间弄断发夹，回形针也可以。Lifehacker 的 Beth Skwarecki 测试了这两种方法，你可以在下面的视频中看到:

Skwarecki 还说，虽然她没有足够大的拉链铁龙来系住直径超过几英寸的东西，但她可以用手在背后打开拉链。

以防你可能会发现自己处于类似的情况。

更新:虽然知道如何去除拉链是一项有用的技能，但在某些情况下，去除拉链可能比人们意识到的更危险。在选择解开拉链之前要小心，问问自己你可能经历的任何后果是否值得冒险。
