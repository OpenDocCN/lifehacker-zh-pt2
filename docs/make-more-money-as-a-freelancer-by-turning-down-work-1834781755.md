# 作为自由职业者，通过拒绝工作赚更多的钱

> 原文：<https://lifehacker.com/make-more-money-as-a-freelancer-by-turning-down-work-1834781755>

受欢迎的自由职业者:如果你让你的客户知道你已经预约满了，你可能最终会赚更多的钱。



正如卡罗尔·泰斯在 [中解释的那样，以写作](https://www.makealivingwriting.com/magical-words-freelance-writer-earn-more/) 为生，告诉客户你没有时间接手他们的项目，从长远来看会让你受益:

> 当你告诉一些潜在客户你因为太忙而不能接受他们时，他们会印象深刻。你一定是个好作家！
> 
> 他们甚至更想雇用你。有时，他们会给你更多的钱，希望让你把其他人从你的日程中踢出去，为他们的任务腾出空间。有时候，你会说是。

泰斯还指出，如果你因为没有时间而考虑拒绝一个新客户，你应该问问自己，拒绝一个现有客户是否更明智。也许是时候结束你最低薪酬的客户关系了，比如，给自己足够的时间和空间去接一个薪酬更高的。

简而言之:如果一个客户给了你一份工作，而你不确定你是否有时间来适应你的自由职业日程，不要承担额外的工作，把自己搞得筋疲力尽。告诉客户你愿意和他们一起工作，但是你现在已经预约满了，看看他们会有什么反应。

如果他们给你更多的钱，你就增加了你的潜在收入。

如果他们要求被安排在你日程表上的下一个空位，你就已经为自己预定了一些未来的工作——避免了潜在的自由职业者的干燥期。(你也可以在你的日程表中为客户提供下一个位置，而不是等他们来问。)

如果他们鬼鬼祟祟，表现出被侮辱的样子，或者不够灵活，不能根据你现有的工作量来安排他们的项目，那么你知道从长远来看，他们可能不会是你的好客户。

你也不用担心失去那个潜在客户。

毕竟，你已经预约满了。
