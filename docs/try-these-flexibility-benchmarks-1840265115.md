# 尝试这些灵活性基准

> 原文：<https://lifehacker.com/try-these-flexibility-benchmarks-1840265115>

在 2019 年的最后一个月，我们将做一些练习来衡量进展并设定目标。无论你今年已经锻炼了很多还是刚刚开始，现在设定一个具体的基准将有助于你欣赏自己目前和未来的进步。今天我们从灵活性开始。

不过，让我们明确一件事:关键是用自己来衡量自己，而不是达到某个武断的标准。(我们 [昨天谈过这个](https://vitals.lifehacker.com/measure-your-lifting-progress-with-symmetricstrength-1840234865) 关于实力标准，记得吗？)如果你以前做过这些伸展运动，你可以和你以前做过的做比较。本月晚些时候，我们的基准将涵盖体能的其他领域，如力量和耐力。

但是，制定基准的更大原因是作为给未来自己的礼物。如果你在接下来的一年里提高你的灵活性，你可以在 12 月再做一次测试(或者每月做一次，随便你怎么做)，看看你有什么进步。

那么，让我们试试这些。在你开始之前，记住热身的肌肉比你刚尝试冷拉伸时伸展得更远，所以考虑在锻炼结束时做这些。记下你是否热身，以及如何热身，这样你就可以在下次测试时做同样的事情。

### 坐伸式，或站着触摸脚趾

如果你曾经接受过健身评估，比如私人教练的评估，你可能做过坐伸测试。你坐在地板上，脚放在前面，然后触摸你的脚趾，同时脚边的一个盒子测量你能向前够多远。

你可以在家里通过把你的脚放在你前面的台阶、壁架或路边来重现这一场景。用一根卷尺向前伸，如果可以的话，把它贴在台阶上，然后测量。

站立脚趾触摸可能不太标准，但它以类似的方式测量腘绳肌的灵活性。站起来，在你试着触摸脚趾的时候，给自己拍一段视频(或者让朋友拍一张照片)。也许你只能够到你的小腿；也许你能到达你的脚趾；也许你可以把手掌平放在地板上。一段视频还将展示你是用圆背还是平背(后者更难)来完成这个动作的。这里没有判断，我们只是采取措施，继续前进。

### 衡量灵活性的其他方法

上面的拉伸实际上只是测量你后链肌肉的柔韧性，主要是你的腿筋。还有一个技巧的要素:如果你练习，你可以在那些特定的拉伸方面做得更好，即使没有增加你的柔韧性太多。但它们是测量柔韧性的流行方法，因为它们简单，而且在坐伸测试中，很适合特定的数值测量。

但也许你不太在乎你的腿筋。如果你对灵活性的另一个领域感兴趣，也把它记录下来。一些想法:

*   能不能把 [的手摸在背后](https://fitness.stackexchange.com/questions/14743/arm-cross-stretching-behind-the-back-how-to-improve-it) ，一个向下一个向上？
*   在蝶泳伸展动作中，你的膝盖能离地板多近？
*   脚跟不离地能蹲多低？

用手机拍摄视频是衡量这些或其他你想做的伸展运动进展的最简单的方法。保存视频——或者甚至只是拉伸最深部分的截图——然后你可以在下次尝试时将前后剪辑在一起。
