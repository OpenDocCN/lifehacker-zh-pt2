# 气候罢工运动已经正式走向全球

> 原文：<https://gizmodo.com/the-climate-strike-movement-has-officially-gone-global-1833305314>

周五，世界各地的学生醒来时知道，在数学、英语、生物或其他一些死记硬背的课程之间，他们会站起来走出去。从瓦努阿图到欧洲再到美国，气候罢工运动已经正式成为一种全球现象。预计周五将有 123 个国家的 100 万名学生进行罢工。



他们走出去的原因多种多样 ，但潜在的主题是:他们是成长过程中充分意识到他们将遭受气候变化影响的第一代人。他们拒绝接受无所作为。

“这场运动不得不发生，我们没有选择，”罢工的领导人 [在《卫报》](https://www.theguardian.com/commentisfree/2019/mar/15/school-climate-strike-greta-thunberg) 上写道。“这些袭击今天正在发生——从华盛顿 DC 到莫斯科，从特罗姆瑟到因弗卡吉尔，从贝鲁特到耶路撒冷，从上海到孟买——因为政客们辜负了我们。”

几个月来，青年气候罢工已经在欧洲和澳大利亚蔓延，学生们联合起来支持瑞典青少年格蕾塔·图恩伯格 ，她于 8 月开始在议会前罢工。但周五标志着一个巨大的升级，各大洲都计划大规模罢工，包括在南极洲的团结罢工。

太阳首先在太平洋升起，澳大利亚、新西兰和小岛国的学生走出学校。自那以后，大规模罢工已经从韩国蔓延到印度、尼日利亚和欧洲主要国家的首都。在斯德哥尔摩，估计有 10，000 名学生挤在像摇滚音乐会一样的路障前，去看图恩伯格，这与她七个月前开始的单独罢工相去甚远。在都灵，数千人唱着“我们将摇滚你”在达喀尔，学生们聚集在烈日下的阴凉处。在苏黎世，他们在雨中挤在雨伞下。
