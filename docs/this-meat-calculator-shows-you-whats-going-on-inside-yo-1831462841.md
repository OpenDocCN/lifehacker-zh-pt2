# 用这个肉计算器找出你的牛排里面发生了什么

> 原文：<https://lifehacker.com/this-meat-calculator-shows-you-whats-going-on-inside-yo-1831462841>

对一些人来说，肉是神秘的。如果你没有大量的烹饪实践，你可能很难准确地感觉到你应该烤多长时间，烤焦，腌制，或者用其他方式加热食物来使它变得安全和美味。得到一个数字温度计很有帮助，但是麻省理工学院的 [这个工具也可以给你一点指导。](http://up.csail.mit.edu/science-of-cooking/)



这个工具叫做“煮我的肉”，它的名字很有侵略性，但是玩起来很有趣。首先，你插入你的变量，比如厚度、肉的起始温度和锅的温度。(你需要一个红外线温度计来精确测量，但是 [这个指南](https://www.smartkitchen.com/resources/cooking-appendices/reference-materials/heat-temperature-charts/easy-heat-chart) 可以帮你估计。)然后你挑一个现成的程序，或者自己设置参数。例如，如果您计划将牛排的每一面都烹饪四分钟，那么您应该为第一面输入 302℉，并为第二面输入厨房的温度，因为一面将与热锅接触，而另一面将暴露在空气中。

一旦你输入了所有的信息，按下“烹饪”键，就会看到一个有趣的小图表出现。你可以看到热量扩散如何随着时间的推移影响肉类，包括蛋白质变性和褐变反应发生的时间。如果你也想看到这些信息，你也可以将图表切换到“温度”上。

当然，这个工具有一些限制。只有三种肉类可供选择——牛排、金枪鱼和火鸡——而且没有一种是鸡肉，这似乎很奇怪，而且它没有考虑到你会遇到的取决于锅材料的变量，但它确实让你对肉类在烹饪过程的各个阶段发生的变化有了有趣的见解，更多的知识几乎总是会带来更好的食物。(另外，摆弄它很有趣，尤其是如果你喜欢好的图形。)
