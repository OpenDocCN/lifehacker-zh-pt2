# 为什么你仍然需要一个锻炼笔记本

> 原文:[https://life hacker . com/why-you-still-need-a-workout-notebook-1834590216](https://lifehacker.com/why-you-still-need-a-workout-notebook-1834590216)

使用锻炼应用足够长的时间，你会意识到你不能相信任何一个。有时它们会失效，或者你在跑步的间隙忘记了密码，或者你的手机没电了，带走了你的数据。即使一切顺利，应用程序也不能总是跟踪你应该跟踪的指标。

Watch

这就是拥有你的训练数据的重要性。不要把它托付给应用程序或云；在纸上写下你的跑步里程数或举重动作，或者保留一份你能控制的电子文档(并备份)。原因如下。

### 你会忘记你在哪里记录了一切

当然，在你思考的时候，使用一个应用程序现在*有效*。但是我已经跑步很多年了，让我告诉你，我的锻炼历史是一条由废弃的网站和丢失的密码铺成的轨迹。我在哪里记录了我五年前参加的上一次半程马拉松的训练数据？不知道。或者是我在 2012 年的第一次马拉松？我找到了我认为记录里程的网站，但它不识别我的密码或我当前的任何电子邮件地址。

所以，我们经常专注于当前的日常锻炼，但为未来的自己留下数据也很有价值。五年或十年后，你可能会问“我在 2019 年是如何训练的？”你需要给自己留下数据来回答这个问题。

幸运的是，我有一些旧数据，那是因为我把它们写在了纸上。2012 年那次马拉松训练的时候，我把训练日历打印出来，挂在墙上。那天锻炼时，我把方格涂成绿色，没锻炼时涂成红色，休息日涂成灰色。我仍然把那本日历放在我办公室的一个文件夹里。我也有一两年的关于 [这种纸质日志](https://vitals.lifehacker.com/this-paper-template-helps-you-stick-to-your-running-reg-1782351372) 的锻炼数据，在这一点上我唯一的遗憾是我曾经懈怠了低技术含量的日志记录。

### 应用程序只测量他们想要测量的东西

对于一个跑步应用程序来说，最容易、最简单的方法就是测量你的总配速:锻炼时间除以总距离。这是你可能得到的最没有帮助的数字之一。

毕竟，不同的锻炼应该以不同的速度进行。如果我跑很长一段距离，我会期待一个缓慢的平均配速。但是如果我做一英里的慢速热身，然后三英里的快速跑，和一个慢速的冷却，总配速告诉我关于跑步没有任何有用的东西。如果我跑跑道间歇跑，谁会关心总速度是多少？没人，就是他。

当我跑步或者在健身房举重的时候，我仍然使用应用程序，但是现在我回到家，把重要的数字转移到一个笔记本上。对于每种类型的锻炼，我都会记下所用的时间和感觉——要么用笑脸系统，要么用数字表示强度。(10 是致命的努力，6 是公园里的慢跑。)

想一想:你为什么要跟踪你的数据？你要么想要在那一刻看到并思考它——在这种情况下，你应该写下你想要反思的指标和感受——要么你希望能够在以后回顾它。可能两者都有。所以，记住什么是重要的。

### 两个都用没问题

我不是说你需要*停止*使用方便的应用程序。我 [喜欢强大的举重应用](https://vitals.lifehacker.com/use-these-apps-for-tracking-strength-workouts-1830938052) ，因为它为我做了大量的数学计算，让我记录下当下最重要的事情(比如重量、重复次数和 rpe ),并提醒我上次举起了什么。但是我一回家，就打开我的训练笔记本，记下我做了什么练习，做了多少组，我的工作重量是多少。

跑步也是如此:我让耐克测量我的里程数，并制作印有我的数据的不可编程的小照片，但在一天结束时，我也会在笔记本上写下当天的里程数，以及关于配速、感觉如何以及其他可能会在以后派上用场的细节的笔记。

如果你想跳过应用程序，把你的笔记本带到健身房，这也很酷。(我甚至有点嫉妒你在锻炼时没有让任何公司跟踪你的位置和个人数据。)无论哪种方式，确保你有一个适合你的系统，并且没有丢失的密码可以拿走。
