# 如果无法从 Windows 7 升级到 Windows 10，该怎么办？

> 原文:[https://life hacker . com/if-you-cant-upgrade-from-windows-7-to-windows-10-what-1835696084](https://lifehacker.com/if-you-cant-upgrade-from-windows-7-to-windows-10-what-1835696084)

虽然我总是建议升级到你正在使用的任何软件的最新和最好的版本，但如果你运行的是已经很难运行的旧设备，这可能不是最好的选择。对于操作系统来说尤其如此——尽管有时甚至你的旧设备 [也能从操作系统升级中获益](http://www.iphonehacks.com/2018/09/ios-12-performance-improvements-tested-iphone-5s-6-ipad-mini-2.html) 。

Watch

如果你在实现这一升级时遇到了麻烦，你可能会感到有点卡住——就像 Lifehacker 的读者 **Daniela** ，她在本周的 [Tech 911](https://lifehacker.com/c/tech-911) 问题中写道:

> 鉴于微软将于 2020 年 1 月终止对 Win 7 的支持，你对那些因为笔记本电脑与 Windows 10 不兼容而运行 Windows 7 的人有什么建议？
> 
> 我和我丈夫在这里有两台这样的机器，不想买新的，因为这两台机器没有任何问题，虽然对 Linux 毫无经验，但对技术略懂。你认为 Linux 是一个好方法吗？如果没有，请问有其他选择吗？
> 
> *顺便说一下，我确定这两台相同的笔记本电脑与 Win 10 不兼容，因为几年前我试图手动升级，结果无法启动我的笔记本电脑，必须从旧备份中恢复一切。然后我在厂商网站上了解到，这些机器与 Win 10 不兼容。(我从来没有想到这可能是一件事，因为 Windows Update 不断向我提供升级。)*

首先，我想再深入调查一下，看看你的笔记本电脑是否真的不支持 Windows 10。有可能你最初的升级过程因为某种未知的原因而失败了。你的笔记本电脑制造商可能是正确的，但我想在注销最新版本的微软操作系统之前，至少再确认一次你的设备不合格。

以前有一个“升级兼容性检查器”工具，你可以在 Windows 7 中运行它，看看你的系统在升级 Windows 10 时是否会有任何问题。我相信微软早就把它从操作系统中移除了，但万一我错了，你会发现它是“获取 Windows 10”应用程序中的一个选项，位于系统任务栏的右下角。

否则，我相信你可以下载并运行微软的 [Windows 10 更新助手](https://www.microsoft.com/en-in/software-download/windows10) ，它应该会运行一次检查，以验证你的系统*能否*处理 Windows 10(它还会告诉你应该升级到什么版本)。

我手头没有 Windows 7 的拷贝，所以我不能直接带你完成这个过程。如果你的电脑可以运行最新的操作系统——或者如果你想尝试一下——我相信更新助手会指示你通过微软的 [媒体创建工具](https://go.microsoft.com/fwlink/?LinkId=691209) 下载 Windows 10。即使它没有，或者如果它启动了升级过程本身，我建议取消并使用该工具在单独的介质上创建 Windows 10 安装映像。闪存盘最好，但也可以刻录 DVD。

你要确保你正在执行一个“ [干净的](https://www.microsoft.com/en-us/software-download/windows10startfresh)”Windows 10 的安装，而不是任何种类的“ [就地升级](https://answers.microsoft.com/en-us/windows/forum/windows_10-windows_install/how-to-perform-an-in-place-upgrade-with-windows-10/7f0b236e-3c6c-4913-9691-c4baa2a6190a) ”这将给你在安装过程中避免任何问题的最好机会。这并不是说你不会遇到问题——因为你已经提到你上次尝试升级时遇到了一个问题——但是我想再试一次，看看这是否有帮助。

在开始升级之前，有必要对您的系统做一个完整的备份。我建议制作一个映像，如果升级过程不顺利，您可以用它来恢复您的笔记本电脑。你可以使用 [Windows 的内置工具](https://support.microsoft.com/en-us/help/17127/windows-back-up-restore) ，但是如果他们不允许你在 Windows 7 上创建你的系统的映像，像[**Macrium Reflect**](https://lifehacker.com/back-up-and-clone-your-hard-drive-with-macrium-reflect-1825289970)(免费)这样的第三方应用程序是另一个很好的替代选择。

此外，在开始升级之前，请确保您使用的是笔记本电脑制造商提供的最新 BIOS。幸运的话，这可能有助于解决你的系统在安装 Windows 10 时遇到的任何问题。(更新 BIOS 后，考虑将其重置为默认设置，以防某些奇怪的设置也影响 Windows 安装程序。)

### 一些 Windows 10 替代品

如果升级过程*仍然*不起作用，或者如果你在 Windows 10 上挥舞白旗，那完全没问题。至少，你可以考虑升级到 Windows 8.1——这不是 Windows 7 的巨大进步，但可以让你从微软获得 [三年以上的扩展支持](https://support.microsoft.com/en-us/help/13853/windows-lifecycle-fact-sheet) 。你可以免费安装，如果你不确定你的笔记本电脑是否能运行它，这很有用，如果一切顺利，可以事后激活。

至于 Linux，这当然是一个选择。我会先尝试一个类似于[**Linux Mint**](https://linuxmint.com/)的现场会话。换句话说，你将从光盘或闪存驱动器运行操作系统，而不是直接安装在你的系统上，所以如果感觉太难，你可以随时退回到 Windows 7。

你也可以试试[**cloud ready**](https://www.neverware.com/freedownload)，它是基于 Chromium OS 的——这个项目也是谷歌 Chrome OS 的基础。如果你曾经接触过 Chromebook，你应该会感觉很熟悉。即使你没有，这也是一个很好的、更简单的选择，不应该吓走任何你形容自己“稍微懂点技术”的人。
