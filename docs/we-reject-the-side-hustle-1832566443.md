# 我们拒绝旁敲侧击

> 原文:[https://lifehacker.com/we-reject-the-side-hustle-1832566443](https://lifehacker.com/we-reject-the-side-hustle-1832566443)

读了足够多关于职业和金钱的书，你会开始看到很多相同的建议被一遍又一遍地重复。大多数是明智的和无害的。 [有些简直烂透了。【T2](https://twocents.lifehacker.com/the-worst-stuff-written-about-money-in-2018-1831360785)

然而，我最厌倦听到的是“总是有一份兼职工作”的忠告，或者更直白地说，在你已经有全职工作的基础上强制性地堆积额外的兼职工作。为了纪念 [拒绝周](https://lifehacker.com/c/rejection-week) ，我在此拒绝“副业”——或者说“第二份工作”，一旦你剥去它在经济衰退后的重新命名。

在我职业生涯的最初几年，我经常在全职工作之外做兼职，要么在做其他事情来支付账单的同时保持稳定的写作剪辑流，要么在全职工作但工资很低的编辑工作时支付账单。从本质上来说，我并不后悔——它让我在经济上和事业上都达到了我需要的水平。额外的工作是不可避免的。

问题是，在我可以不这样做的很长一段时间后，我一直让自己处于不断自由职业的仓鼠轮上，在一个完整的工作日之前早早醒来，更好地赚钱，这有助于我更快地积累一些急需的应急储蓄，但我可以轻松地度过没有这些储蓄的日子。我告诉自己，这不是额外的努力，我只是“努力得很好”，在这样一个不稳定的职业和行业中，我聪明地用额外的选择来支撑自己。

也许这都是真的，但同样真实的是，工作之外的工作悄悄地成为我能量水平的主要消耗，对我的精神、情感甚至身体健康产生了负面影响。我也开始更愚蠢地花钱(有些人可能会说是强迫性的),作为对压力上升的回应，这产生了相反的效果，让我觉得我更需要额外的现金流。我的外部收入并没有起到财务安全毯的作用，而是成为了我生活中的一个幽灵，主要是出于恐惧。当我最终咬紧牙关，专注于一次只做一份工作时，不出所料，所有额外的压力和焦虑几乎立刻烟消云散。我睡得更多了，坐的高价出租车更少了，我的日常工作做得更好了。

就像尼克以前写的那样，把你热爱的爱好变成赚钱的骗局 最终会毁掉你和你曾经热爱的事物的关系。类似地，继续你的副业，超过经济上的必要性，会很快变成更接近“心理陷阱”的东西，而不是“精明的财务技巧，会让你提前退休。”当你不上班的时候，把这些时间花在锻炼、睡觉、见朋友、做非赚钱的兼职项目上——实际上是除了利用你宝贵的非工作时间做更多工作之外的任何事情，只是因为一些金钱博客告诉你这样做。
