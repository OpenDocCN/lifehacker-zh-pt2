# 问咸味女服务员:我怎么纠正一个发错音的服务员？

> 原文:[https://the take out . com/how-can-I-correct-a-server-who-mispronounctions-a-dish-1830446565](https://thetakeout.com/how-can-i-correct-a-server-who-mispronounces-a-dish-1830446565)

> 亲爱的 Salty:上周我和我的家人第一次尝试了一家新的意大利餐厅。一位年轻的女士为我们服务，她什么都做得很好——对她的服务没有任何抱怨。但当她在背诵特色菜时，她在谈论意大利面食时念错了“guanciale”这个词。她说的好像是“格温-切尔-李”？这与正确的发音相差甚远，但我没有说什么，因为这似乎是居高临下。但是那天晚上晚些时候，我们一回到家，我就和我的孩子们争论我是否应该纠正她，以免她将来说错了而尴尬(我们对半分)。这是一个意大利的地方，所以我认为 guanciale 可能出现在未来的菜单上并不牵强。你怎么说？
> 
> 谢谢，
> Guanciale Guy

亲爱的盖伊，

你真的让一位女士想到了这个，不仅仅是因为我偏离了对意大利肉的梦想。我做了很多“一方面”和“另一方面”才最终下定决心。(我告诉你这些只是为了让你欣赏我这个天才哲学家。)

Watch

你得好好想想你的意图。你只是想表现得成熟一点，炫耀你在罗马留学了半个学期吗？如果是这样的话，去和那些把羊角面包读作“kwah-sonn”而不是“cruh-saaaandt”的人一起站在耻辱的角落里吧。如果你的意图是真心诚意的，那么是的，如果你非常微妙地纠正她，可能会让她免于尴尬。

但所有这一切都取决于你如何进行纠正。小心翼翼地踩*。我们都是人，会发生发音错误。服务员可能会因为你纠正了她而感到有点尴尬，所以你不想让她生气或者感觉被人居高临下地对待。*

*如果我是你，我会装聋作哑:“哦，等等，这和 guanciale 是一样的吗？我是不是说错了？”*

*希望这能让她一会儿跑回厨房去问经理或厨师。(记住，她可能一开始就从其中一个人那里听错了。)*

*这确实是你应该采取的最大限度的修正。如果她将来继续把事情搞砸，那不是你要解决的问题，如果你一直喋喋不休，你可能会听起来像一个浮夸的气囊。*

*长话短说:以一个温和的问题的形式提及更正，放她一马，就此打住。*
