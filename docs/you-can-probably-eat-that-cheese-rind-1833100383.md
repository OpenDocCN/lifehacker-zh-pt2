# 你也许可以吃奶酪皮

> 原文:[https://life hacker . com/you-can-probably-eat-the-cheese-rind-1833100383](https://lifehacker.com/you-can-probably-eat-that-cheese-rind-1833100383)

我注意到不是每个人都在吃布里干酪皮 。这很傻，因为布里干酪皮很好，而且——如果你在公共奶酪盘上从楔子里刮出黏糊糊的中心——这非常粗鲁。但是，除了布里干酪，许多人都回避完全可食用的，有时甚至是美味的皮，而且根本没有理由。

### 不要吃蜡

我所说的“外皮”是指在奶酪制作过程中成熟阶段形成的坚硬的奶酪外层。我不是指你的干酪周围的红蜡，或者你的切达干酪周围的布。这几块无机物不会毒害你，但是不能食用，味道也不会好。(这对 Babybel 来说是双；我用那种蜡做我的*艺术品*。)

### 尝尝白色的东西(很好吃！)

据 [食&酒](https://www.foodandwine.com/news/cheese-rinds-facts) 说，果皮主要有三种:熟的、洗过的、天然的。开花的外皮是你在软奶酪(如布里干酪)外面看到的苍白、白色或灰白色、看起来有点模糊的外皮，它是奶酪的重要组成部分。(是的，这都是由于霉菌，但霉菌是给我们奶酪和许多奇妙的抗生素的东西。)这种类型的外皮不仅很难优雅地从其柔软的中心移除，而且通常具有令人愉快的、泥土般的浓郁风味，这是奶酪食用体验的一部分。除了布里干酪，你应该吃卡门贝干酪的外皮，美国珍宝塔姆山 和洪堡干酪，以及任何有法国名字的看起来超级软和奶油的东西。( [正经吃](https://www.seriouseats.com/2014/05/cheese-101-all-about-bloomy-cheese-aka-brie-and-brothers.html) 有好看的单子。)

然而，如果外皮与奶酪分离或者上面有黑色霉菌斑点，你应该跳过它，因为事情不应该是这样的。

### 橘子你很高兴你吃了它？

洗过的鸡皮看起来是橙色的，摸起来很粘，有时闻起来很怪。它们的味道各不相同，我不能告诉你你会喜欢你品尝的每一种洗过的奶酪的外皮，但是你很可能会喜欢，特别是当它是软的时候。一旦奶酪凝固，就用盐水处理，并伴有苹果酒、葡萄酒、白兰地或啤酒等美食；有时它们还会涂上药草、花朵或某种灰烬。这些咸的、厚的、略带肉味的奶酪皮充满了奶酪制造商精心挑选的风味，它们讲述着奶酪的故事。刮掉它们意味着你去掉了很多味道，而且(有点)不尊重人。Taleggio 可能是最具代表性的水洗皮，虽然气味可能会带来最初的挑战，但至少有一次值得一试——连皮一起。

### 留着炒股吧

然后是强硬、自然的男人。这是你的帕玛强尼瑞吉阿诺奶酪和任何其他经过老化和风干的奶酪，让外皮自然生长。尝一尝肯定没什么坏处，但它们可能对牙齿有点硬(或者只是有点苦)，较硬的样品最好留作调味原料。还应该注意的是，类别之间可能会有一些重叠——尽管 [Gruyere *是*一种洗过的外皮奶酪](https://www.seriouseats.com/2014/06/cheese-101-washed-rind.html) ，那种外皮非常坚硬，咀嚼起来没有的乐趣。

### 然后是你

你可能不会喜欢你放进嘴里的每一个果皮。见鬼，有些人觉得布里干酪皮——最容易吃的皮之一——的质地令人不快。你是否吃奶酪皮取决于个人喜好，但是请在切掉它之前试一试。像任何由劳动密集型、几乎艺术化的过程创造的食物一样，奶酪有一个故事要讲，它的外皮包含了很多情节。基本上，没有果皮会伤害你，所以尝一尝，做出你的选择，享受你的奶酪。
