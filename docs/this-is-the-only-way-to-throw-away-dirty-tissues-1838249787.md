# 这是扔掉脏纸巾的唯一方法

> 原文:[https://life hacker . com/这是扔掉脏纸巾的唯一方法-1838249787](https://lifehacker.com/this-is-the-only-way-to-throw-away-dirty-tissues-1838249787)

坏消息是，在我儿子的整个前半生，我一直在错误地处理他的脏的、有细菌的纸巾。好消息是，这里的一些事情即将改变。

Watch

当我的孩子生病时，他会擤鼻涕。很多。这是我们沙发旁边的茶几通常的样子:

然后我必须决定:当我捡起那堆东西的时候，我是不是又浪费了一张干净的纸巾来保护我的手不被鼻涕弄脏？还是我直接进去，赤手空拳，当天第 28 次洗手？

啊哈，但没有了，感谢我们的 [后代脸书集团](https://www.facebook.com/groups/2018785615043946/) 中一位出色的父母，他给我们展示了一条更好的道路！这就是感冒从现在开始的样子:

这些盒子是用橡皮筋绑在一起的，但是如果你找不到足够大的带子把它们绑在一起，你也可以用胶带把它们绑在一起。

孩子们会把每一张脏纸巾都放进空盒子里吗？嗯，我的孩子会的，因为他在这些方面比较精确。你的可能仍然会在这里或那里留下一个混乱的细菌节，但仍然——这将是一个进步。

现在，我并不主张在他们都完成后扔掉那个盒子。那是可回收的(在你移除塑料衬垫之后)。但是一旦细菌不再控制你的家，你可以把它的一边打开，把所有的脏东西直接倒进垃圾桶。
