# 只要在土豆泥中加入商店买的法式洋葱酱

> 原文：<https://lifehacker.com/just-add-store-bought-french-onion-dip-to-mashed-potato-1839621708>

每年，在开发新的令人兴奋的感恩节内容时，我都试图在“哇，这是独一无二的美味”和“她是一个必须被阻止的疯女人”之间游走这导致了赞扬(来自那些已经同意我胡说八道的人)，但也带来了许多节日的欢呼。



大喊大叫让我知道我在做正确的事情。感恩节有这样一个标准的、锁定的菜单，你必须玩弄细节，以免我们都无聊而死。这就是为什么你应该在土豆泥中加入法式洋葱酱。

我说的不是花式 [自制洋葱蘸](https://skillet.lifehacker.com/make-a-super-bomb-onion-dip-without-a-mix-1795872077)；我甚至不是指你发现的那种用奶油奶酪和酸奶油冷藏的。我说的是装在玻璃罐里的米色菜籽油蘸酱——那种完全乳化且耐储存的蘸酱，从容器里出来时会轻微晃动。那种带点的， [麦当劳式的洋葱块](https://skillet.lifehacker.com/top-burgers-with-mcdonalds-style-diced-onions-like-a-ge-1834644886) 。带 [味精](https://skillet.lifehacker.com/put-msg-in-everything-you-cowards-1831721707) 的那种。(Lay’s 做的很好。)

憎恨我的人会说我在胡说八道，但是商店里买的法式洋葱蘸酱确实能做出真正疯狂的土豆泥。就像我之前的 [蛋黄酱土豆泥](https://skillet.lifehacker.com/how-to-making-smashing-mashed-potatoes-1820225954) 一样，乳化油产品有助于将亲水性和疏水性结合起来，创造出一碗天鹅绒般柔滑的马铃薯，具有令人惊讶的微妙洋葱味和足够的鲜味。

没人会咬一口就说“搞什么？？这些尝起来像 Lay 的洋葱蘸酱！”不，他们不会。相反，他们会吃一勺，停顿一下，再进去吃更多，然后问你你的秘方是什么。他们会问你是怎么把这些土豆泥做得如此美味、如此顺滑、如此美味的。但你永远不会说出来。(也可能你会。我个人不擅长保守秘密。)

你可以在任何土豆泥食谱中添加法式洋葱蘸酱，保持每磅土豆四分之一杯蘸酱的比例，但你真的不需要很多其他配料。土豆、黄油和蘸酱对于一批优秀的捣碎器来说足够了。要制作它们，您需要:

*   2 磅土豆，黄褐色、育空金黄色或混合色
*   1/2 杯(1 根)盐黄油，切成 8 块
*   1/2 杯法式洋葱蘸酱(最好是 Lay's)

削皮并把你的土豆切成一英寸的方块。将它们放在一大锅盐水中煮，直到一块可以很容易地用叉子捣碎。用漏勺把土豆沥干。将火调至中低，将土豆放回锅中，烹饪并搅拌几分钟，以去除多余的水分。用马铃薯捣碎器捣碎马铃薯(或者用碾米机捣碎蜡质马铃薯)，然后加入黄油。继续捣碎，直到黄油融化，然后加入蘸酱。再把所有的东西都捣碎，然后用木勺搅拌，直到土豆光滑，颜色均匀。
