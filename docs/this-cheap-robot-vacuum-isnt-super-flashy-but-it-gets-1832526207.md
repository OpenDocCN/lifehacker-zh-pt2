# 这种廉价的机器人吸尘器并不华丽，但它能完成工作

> 原文：<https://gizmodo.com/this-cheap-robot-vacuum-isnt-super-flashy-but-it-gets-1832526207>

尽管它们还是有点笨，但机器人吸尘器恰好是目前最成功的家用机器人。但是几年前，它们对普通人来说太贵了，预算选项往往是 [非常糟糕的](https://gizmodo.com/robovacs-are-terrible-and-a-bad-gift-idea-1820876350) 。他们会错过明显的灰尘兔子，撞上家具，绝望地迷失在你的床下寻找充电站。



但是机器人吸尘器在短时间内已经有了长足的进步。你现在可以在 300 美元及以下的机器人上找到语音助手兼容性、多天日程安排和应用程序控制——以前的高级功能。在机器人吸尘器制造商中，Eufy 以一些价格实惠但质量上乘的机器人而闻名。其最新产品 [RoboVac 35C](https://www.walmart.com/ip/Eufy-RoboVac-35C-Wi-Fi-Connected-Robot-Vacuum/298006486) 延续了这一传统，并增加了触摸控制功能。结果是一个有能力的小清洁器，虽然大多数的改进大多是渐进的。

首先，RoboVac 35C 显然是一个零售价为 300 美元的预算选项。因此，尽管你可以通过应用程序轻松安排清洁时间，或使用亚马逊 Alexa 或谷歌助手控制机器人，但你不会像使用高级机器人 [iRobot Roomba i7+](https://gizmodo.com/irobot-finally-makes-a-roomba-that-empties-itself-1828854463) 或 [戴森 360 Eye](https://gizmodo.com/dyson-s-first-robovac-sucks-up-all-your-dirt-and-cash-1784141541) 一样，获得清洁地图、超级花哨的基于摄像头的导航或自动清空垃圾箱。但从好的方面来看，你也不会花 1000 美元。

我测试过 20 多种机器人吸尘器，RoboVac 35C 是我试过的预算较好的选择之一。它配备了所有标准的机器人真空吸尘器模式——靠墙边缘清洁、特定脏乱的现场清洁、单间清洁，以及当你感到懒惰时的自动清洁。和大多数平价机器人清洁器一样，35C 选择随机清洁模式，利用位于其前保险杠的红外传感器。所以这意味着它在你家周围蜿蜒曲折，当它撞到家具时会改变方向。这看起来非常令人沮丧，因为它在第一次通过时会错过大量的灰尘和毛发。它也不记得已经清理过的地方，所以它可能会多次经过相同的几个区域，而不去碰那个角落。

但同样，这是更便宜的机器人清洁器的交易，并不是专门针对 35C 的。也就是说，与市场上一些更便宜的选择相比，35C 有几个优势。

首先，对于机器人吸尘器来说，它非常安静。如果你关上卧室的门，你可能听不到它在你的客厅里吸灰尘。即使你们在同一个房间里，带上一副像样的耳机也足以盖过噪音。对于一些高级机器人来说，情况就不一样了——Roomba i7+听起来就像有人在喘息，直接把肺插进你的耳朵。

它的电池寿命也很长。在我位于布鲁克林的两居室公寓里，我一周内打扫了四五次——我的老狗拖着它的食物到处跑*——平均每次打扫大约 60-70 分钟。*

*35C 除了内置的转向选项之外，还配有一个物理遥控器。这意味着，如果看着这个笨笨错过那堆明显的宠物皮毛让你发疯，你可以手动驾驶它。有些人更喜欢需要尽可能少干预的机器人，但我很高兴有机会介入并温和地指导我的机器人。*

*EufyHome 应用程序非常简洁，但拥有您需要的一切。你可以操纵机器人，安排清洁，告诉它返回码头，并在它丢失时发出钟声，这样你就可以找到它。*

*导航方面，35C 一般。对于一个随机的清洁工来说，这很不错，但对于一个有流苏的低绒地毯来说，这有点麻烦，公平地说，这对大多数机器人吸尘器来说是一个可怕的挑战。它很好地处理了我的瓷砖厨房和走廊之间的凸起边缘，而它被我的浴室门口挡住了。虽然它毫无困难地自动找到了充电座，但它却很难从我的餐厅椅腿下面挣脱出来。*

*在功能上，我对 35C 没有太多问题。我对这个设计有异议。以前的 Eufy 机器人配备了玻璃顶部，可以最大限度地减少清洁，污迹，划痕和顶部的灰尘收集。35C 将顶部改为哑光表面，这不仅在第二次清洁时被划伤，而且是灰尘和头发的磁铁。*

*那个和增加的触摸菜单有点没用。它做了它应该做的事情，但当你考虑到你有做完全相同事情的应用程序和物理遥控器时，它完全没有必要。*

*我也不喜欢机器人附带的磁条，它用来标记你不想让机器人进入的区域。其他机器人带有信标，或者像 Neato Botvac D7， [虚拟边界](https://gizmodo.com/the-best-thing-to-ever-happen-to-robot-vacuums-is-this-1797948181) 你可以在应用程序本身内绘制。*

*至少 35C 只有 2.85 英寸厚。大多数机器人吸尘器的厚度在 3-3.5 英寸范围内。虽然这对于大多数床下来说完全没问题，但如果你家里有很多低调的家具，这种额外的纤薄也是一种恩惠。*

*总而言之，RoboVac 35C 是一个不错的选择——只要你在寻找清洁保养，而不是一个完全取代常规立式吸尘器的小工具。它擅长吸走碎片、散落的食物碎片、灰尘和头发，但事实是，目前没有机器人吸尘器足以捕捉你房子里的每一点灰尘。多亏了 35C，我的地板干净多了，但我仍然偶尔会在这里和那里看到灰尘兔子，尤其是在难以到达的区域。*

*尽管如此，超高端选项的价格可能高达 700 美元，甚至 iRobot 的更多预算选项也更贵，Roomba E5T3 零售价为 450 美元，Roomba 690T7[零售价为 350 美元。(此外，690 掀翻了我的两张餐厅椅子。)但是，如果价格是你的第一要务，35C 的升级相当少，你可以在网上找到更便宜的旧 Eufy 型号。如果你不介意放弃应用程序连接和语音助手支持，RoboVac 11S 是一个不错的选择，价格约为 225 美元，功能稍强的 RoboVac 30 价格合理，为 270 美元。](https://store.irobot.com/default/roomba-vacuuming-robot-vacuum-irobot-roomba-690/R690020.html)*

*但是如果一个联网的机器人能以 300 美元的价格提供一份体面的清洁工作，那就非常划算了。当然，RoboVac 35C 并不华丽，也没有最新的功能，但它完成了工作，并且不需要花费太多。这是有道理的。*
