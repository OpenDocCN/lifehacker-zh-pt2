# 尝试在没有许可证的情况下在“免费钓鱼日”钓鱼

> 原文：<https://lifehacker.com/try-fishing-on-free-fishing-days-without-a-license-1835157680>

我清晰地记得童年时和家人在某个码头钓鱼的情景。我决定在某个时候尝试一些我在电视上看到的飞蝇钓鱼动作，结果是我哥哥的耳朵被钩了一下，我放弃了钓鱼。



我不确定我是否有钓鱼执照，但我推测我的父母为这种不幸的户外活动照顾了一个。

钓鱼执照并不昂贵，但它们是一种成本，如果你说，不确定你的女儿是否会用她的钓竿挖出一只眼睛，你可能不想承担这种成本。

如果你想尝试一下钓鱼，但不确定你是否准备好了，大多数州都提供“免费钓鱼日”,你可以在没有许可证的情况下下水。这也是一个带孩子或朋友出去尝试钓鱼的好机会，他们还不需要申请执照。

你可以在这个钓鱼网站 上按州 [查看完整名单。](https://www.takemefishing.org/how-to-fish/how-to-catch-fish/free-fishing-days-2019/) 

不同的地方会有不同的规定，所以在你用这些鱼钩做诱饵之前，你可能还是想联系当地的国家鱼类和野生动物机构。

即使你已经是一个狂热的钓鱼者，日期也可能在假期计划中派上用场。

捕鱼许可证由州政府发放，这意味着你在亚利桑那州的许可证在你去加州度假时无效，反之亦然。这可能是有意义的预订您的下一个捕鱼假期的时候，执照是免费的，所以你可以避免支付执照，你只需要一个很短的时间。
