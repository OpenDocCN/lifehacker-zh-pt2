# 问那个咸味的女服务员:我怎样才能确保我的服务员得到我的小费？

> 原文：<https://thetakeout.com/how-can-i-make-sure-my-server-gets-my-tip-1832413271>

> 亲爱的 Salty 女士:当我还是个孩子的时候，我有过这样的经历:看到食客在离开时从废弃的桌子上拿走服务员的小费，我总是坚持直接给我的服务员小费。然而，越来越多的时候，我发现自己在聚会的其他人离开后被吊在桌子旁边，或者在餐厅周围寻找服务器，这对每个人来说都很尴尬。
> 
> 我知道我可以把它加到我的卡里，但我更喜欢用现金支付餐费，而且我也不太能摆脱这样的疑虑，即除非我直接把现金放到他们手里，否则他们可能不会给我小费。那么，是我多疑了吗？想多了？给人一种自命不凡的印象？感谢您的建议。
> 
> *真诚的，奥斯汀食客*

好吧，祝福你的心。作为你的服务员，我非常感谢你在给我辛苦挣来的小费时对细节的关注。不过，我不确定奥斯汀发生了什么，因为从来没有人拿走我的小费。事实上，在我的一生中，我只听说过几个这样的例子。在这一点上，这就像一个都市传奇:一个偷偷摸摸、冷酷无情的小偷从桌子上拿走了现金，然后戴上了他们的太阳镜。虽然，最近我在新闻中看到了一些与餐饮相关的盗窃案:一名妇女在澳大利亚 从一对夫妇的桌子上拿走了他们的食物，北卡罗莱纳州一个无情的讨厌鬼 [拿走了一个餐馆的小费罐](https://www.foxnews.com/food-drink/north-carolina-restaurant-warns-other-businesses-after-woman-steals-tip-jar) 。我可能很咸，但我有点天真——也许你没有想都没想就把现金放在桌子上是对的。



所以，我会尝试一些方法。如果你喜欢用现金支付餐费，你能不能确保你存够了所有的账单和小费，然后交给服务员？(请注意，更换皮带可能会有些过头。)这样，你就可以把现金交给服务器，只说我们都爱听的话:“留着吧。”

在展位的情况下，您也可以将吸头托盘一直滑到墙上。偷窃通常是一种机会犯罪:一个五英镑的人在过道附近徘徊？也许是一时冲动。一个小心翼翼地放在胡椒瓶下面，一直放在窗户旁边的小贴士，这需要一定的技巧才能达到？不是一个有吸引力的选择。

不过，相信我的话，你的服务员非常想得到那笔钱，就像你想把钱给他们一样，所以他们——还有餐馆——很可能都在那笔钱上。即使您的服务器很忙，也有总线人员、主机和管理人员在密切关注。我记得另一个故事，一个家庭被抓到从上一桌拿小费，这在脸书视频 上被曝光。这几天大家都在看。

但是如果你觉得把小费放在服务员滚烫的小手里最安全呢？那就去做吧。那个人可能不会觉得你讨厌或自命不凡，只会感激你的小费。你有没有给过别人现金，却让他们非常高兴地迎接？我猜不是。结案了。
