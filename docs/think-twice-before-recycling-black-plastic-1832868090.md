# 回收黑色塑料前要三思

> 原文：<https://lifehacker.com/think-twice-before-recycling-black-plastic-1832868090>

把你的黑色外卖容器扔进回收站？不幸的是，尽管你做出了努力，它们还是有可能最终被扔进我们的垃圾填埋场。

Watch

我们经常认为我们的回收站是我们所有塑料材料的通用总括(有人会分类，对不对？).不幸的是，并非所有由黑色塑料制成的东西都是如此，可回收材料的简·布雷迪。

在回收行业，黑色塑料因其颜色而难以回收。 [回收设施](https://www.treehugger.com/corporate-responsibility/5-packaging-materials-you-didnt-know-are-difficult-recycle.html) 的扫描仪通过红外摄像机反射光线，以区分不同的塑料材料。因为黑色塑料吸收光而不是反射光，它没有被分类，意外地被送到了垃圾填埋场。

回收黑色塑料也可能成为一个极其昂贵的过程，因为它只能被回收成其他 [黑色塑料材料](https://www.popsci.com/e-waste-black-plastic) ，不像其他颜色的塑料，它并不总是证明分类的努力是必要的。

我们到底在说什么 [产品](https://www.youtube.com/watch?v=wQ7AcZVCZSc) ？嗯，几乎任何带有 [炭黑](https://motherboard.vice.com/en_us/article/zm8z5j/black-plastic-recycling-ewaste-problem-toxic-chemicals) 的东西，这种颜料用于黑色塑料以提高其耐用性。想想那些外卖寿司托盘、DVD 盒或咖啡杯盖。

美国许多当地的路边回收项目要么已经停止接受黑色塑料，要么干脆开始将其倒入垃圾桶，包括加州的伯班克 [三藩市](http://www.sfexaminer.com/the-recycling-myth/) (英国和加拿大的城市也面临着自己的材料问题)。

更糟糕的是，这种塑料污染带来了许多环境和可能的消费者健康问题。在英国，电子垃圾被用来制造黑色塑料(yep)。英国研究人员的一项 [2018 研究](https://motherboard.vice.com/en_us/article/zm8z5j/black-plastic-recycling-ewaste-problem-toxic-chemicals) 发现，铅等有毒化学物质可能会污染这种塑料，当回收到食品容器中时，会带来巨大的健康风险。

这是全美国的问题吗？嗯，这取决于你住在哪里(一些设施可能会手工分拣，这带来了时间和主要不便的另一个问题)。

我与纽约市卫生局的一名发言人进行了交谈，他指出纽约*通常接受硬质塑料，但没有明确指出黑色塑料(我们已经要求澄清，如果我们得到回应，将更新)。*

那么，你能做些什么来帮助遏制黑塑料的问题呢？不言而喻，但使用更少的黑色塑料材料(和一般的塑料)是做好你的部分的最好方法。联系你的路边回收计划，在网上搜索你所在地区的当地计划，并特别询问他们是否接受黑色塑料(以及实现这一点的过程)，这样你就可以帮助当地的设施，你肯定会把你的塑料送到正确的地方。当然，如果你想回收一个外卖容器，在把它交给你的路边项目之前，一定要用水充分冲洗干净。
