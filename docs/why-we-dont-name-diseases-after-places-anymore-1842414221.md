# 为什么我们不再用地点来命名疾病

> 原文：<https://lifehacker.com/why-we-dont-name-diseases-after-places-anymore-1842414221>

所有合法的健康和政策组织(包括世卫组织和疾病预防控制中心)都用约定的名字、新冠肺炎来称呼冠状病毒疾病。但是一些人固执地认为我们应该称它为“中国”病毒，或者把它的名字与一个地点联系起来。胡说八道，原因如下。



过去的许多疾病都是以它们的发源地命名的，出于多种原因，这被证明是一个坏主意。首先，它通常是不准确的。1918 年的疫情流感在许多国家被戏称为“西班牙流感”,不是因为它始于西班牙，而是因为政府不希望战争期间疾病爆发的消息传出去。西班牙的报纸未经审查，所以他们是第一个对此进行详细报道的。 [病毒实际上起源于其他地方](https://www.ncbi.nlm.nih.gov/pubmed/17330793) ，很可能是在堪萨斯州。因此，更准确的地理名称可能是“美国流感”或“堪萨斯流感”

但是没有人希望他们的祖国因为传播疾病而声名狼藉，不管准确与否。如果你住在埃博拉河或寨卡森林附近，你会有什么感受？将一种疾病与一个地方联系在一起也会妨碍对疾病爆发的正确处理。当 1900 年旧金山唐人街爆发 [瘟疫](https://www.nature.com/articles/d41586-019-01239-x) 时，排华种族主义和仇外心理助长了歧视、污名化和拙劣的卫生政策。

反华情绪 [是冠状病毒的一个问题](https://lifehacker.com/your-racist-coronavirus-jokes-arent-funny-1841449955) 也是。关注*谁*携带疾病永远不会像关注如何适当地遏制它那样有效。病毒不在乎你从哪里来。谁都可以承包。

出于所有这些原因，世界卫生组织几年前发布了关于疾病命名的指南，以准确描述的方式命名疾病，而不指责人或地方，也不煽动不必要的恐惧。疾病现在应该根据它们的症状、特征和已知的病因来命名。新冠肺炎是“2019 年发现的冠状病毒疾病”的缩写，是一个合适的名字。以下是他们*不*推荐的:

> 疾病名称中应避免使用的术语包括地理位置(如中东呼吸综合征、西班牙流感、裂谷热)、人名(如克雅氏病、南美锥虫病)、动物或食物种类(如猪流感、禽流感、猴痘)、文化、人口、行业或职业参考(如军团菌)，以及引发过度恐惧的术语(如未知、致命、流行病)。

所以，是的，过去有这样命名的疾病，但公共卫生界从他们的错误中吸取了教训，我们不再这样做了。

今天，在 2020 年，任何人在争论一种疾病的地理名称，要么是对这段历史的无知(把这篇文章发给他们！)或者是试图故意煽动排外情绪。世界领导人现在因为病毒而互相指责，这是愚蠢的。这只是一种病毒。所以我们还是认真对待，直呼其名吧。
