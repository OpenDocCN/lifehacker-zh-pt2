# 用这个聊天机器人练习和某人分手

> 原文：<https://lifehacker.com/practice-breaking-up-with-someone-using-this-chatbot-1837590722>

如果你曾经和某人分手——并且是一系列愤怒短信或电话的接收方——你可能会对结束一段没有结果的关系感到畏缩。为了让自己完全不去谈，你甚至可以用约会来代替。

如果你是一个在分手中挣扎于对抗或诚实的人，你可以使用 [Quartz 的新聊天机器人](https://qz.com/quartzy/1547637/how-to-break-up-with-someone-youre-casually-dating-a-chatbot-lesson/)—“分手训练器”来练习你的文字利用关系专家的建议，这个机器人模仿一个未来的约会对象(以文本形式)，没有真人的反响。这是“优素福”的一段文字:

四个小时？仅此一点就值得重影！但是出于善意，聊天机器人会提供不同的回答作为选项。

我们用选项 A 吧！但是啊哦:

事后看来，正确的方法应该是(非常明显的)选项 b。在经历了一些不同的场景后，是的，正确的答案通常很容易，但你应该从使用聊天机器人的“正确”分手线中获得一个关键的洞察力:善良，但不要伤害你。

在我的一次“聊天”中，我用了一条很好听的分手短信，结果却收到了一条我的另一半还很感兴趣的短信；显然，美好可能会误导人，石英的专家建议永远不要给怀疑留有余地。

“我鼓励每个人都尽可能友善，”人际关系教练雷切尔·迪尔托(Rachel DeAlto)在聊天中提出建议。“但这打开了希望之门，如果你对另一个人不感兴趣，这种希望最终会被现实击碎。”在这种情况下，最好是坦率，不要以居高临下的赞美为开场白。

所以，放弃这个幽灵，不要为一段潜在的新关系没有成功而感到难过。如果你们不匹配，那不是任何人的错。你越早面对，你们就能越早向前看。
