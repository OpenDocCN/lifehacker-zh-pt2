# 使用 GSMArena 的手机搜索器挑选您的下一部智能手机

> 原文：<https://lifehacker.com/use-gsmarenas-phone-finder-to-pick-your-next-smartphone-1829911674>

对大多数人来说，购买一部新的智能手机很容易——你只需要得到你一直在使用的手机的最新版本。如果你不够满意，或者你不想要最新的 Galaxy、Pixel 或 iPhone(仅举几例)，那么是时候去购物了，为了找到最适合你的智能手机，你将有一大堆智能手机要涉水而过。

Watch

鉴于有这么多可看的东西——谢谢，安卓——寻找一部新的智能手机可能会令人生畏。谢天谢地，GSMArena 的优秀人才有一个方便的工具，可以用来缩小范围。该网站的“ [**”手机搜索器**](https://www.gsmarena.com/search.php3?) ”是一种非常有用的方式，可以根据对你来说最重要的规格来搜索你的下一部智能手机。你可以从很多选项中挑选:价格、尺寸、厚度、显示器分辨率、相机分辨率等等。

如果你在寻找有史以来最好的智能手机，手机搜索器不会是最有用的——毕竟，它不提供*排名*，只是一个符合对你来说最重要的规格的手机列表。然而，如果你不想花高价购买功能最齐全的智能手机，Phone Finder 是开始进行令人痛苦的价格/功能权衡的好方法。或者，至少，它可以帮助你确认你是否真的可以花 200 美元买到一部 6.3 英寸的智能手机。(可以！)

一旦你选择了你的规格，点击红色的“显示”按钮，你会看到符合你的标准的智能手机列表。你可以点击它们中的任何一个来查看该设备的全面规格，但我发现使用该网站的“比较”工具更有用——就在“手机搜索结果”下面

点击它，然后点击最多三个你想了解更多信息的智能手机(瞄准盒子，而不是智能手机本身的图片)。一旦你做出选择，再次点击“比较”，你会得到一个巨大的表格，详细说明每部智能手机可以做什么。

如果海量数据让你感到不知所措，请点击“差异”选项，查看智能手机的共同点(灰色文本)和不同点(黑色文本)，这可能有助于你做出购买决定。
