# 使用 Firefox Send 一次共享多达 2.5GB 的文件(免费)

> 原文：<https://lifehacker.com/share-up-to-2-5gb-of-files-at-once-for-free-with-fire-1833232502>

有时，将一个文件(或一吨文件)转储到云存储服务中会容易得多，比如 [Box](https://www.box.com/) 或 [Dropbox](https://www.dropbox.com/) ，而不是试图将它们全部附在一封电子邮件中。或者你在网上搜寻任何数量的“免费文件上传”服务，这些服务承诺为你暂时保存数据。

Watch

本周，Mozilla 公司推出了一项名为 [的全新服务，Firefox Send](https://send.firefox.com/) ，这使得使用端到端加密免费发送更大的文件变得更加容易。“发送”允许您通过简单地将文件拖放到页面上，将高达 1GB 的加密数据直接发送给另一个人。如果你想，你也可以很容易地用密码保护传输。

使用 Firefox Send 时，你不必注册 Firefox 帐户，但如果你使用 Firefox Send 登录，传输限制将达到 2.5GB，你可以创建一个更持久的传输包，在一定时间后过期(最多一周)或下载(最多 100 次)。尽管有品牌效应，Firefox Send 可以在任何浏览器上运行。

Mozilla 的产品算不上创新，但它很容易使用，而且感觉比其他服务不那么麻烦。虽然其他服务如 [WeTransfer](https://wetransfer.com/) 无需任何注册就能给你更多空间(这里是 2GB)，但它们的许多功能通常都被锁定在付费服务之后。基于账户的 Firefox Send 版本功能更加丰富，而且仍然免费。
