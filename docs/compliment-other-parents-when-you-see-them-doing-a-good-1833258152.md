# 当你看到其他父母做得很好时，赞美他们

> 原文：<https://lifehacker.com/compliment-other-parents-when-you-see-them-doing-a-good-1833258152>

在第 [期的一个坏妈妈播客](https://www.maximumfun.org/shows/one-bad-mother) 中，主持人比兹·埃利斯和特里萨·索恩在每集结束时都会互相说，“你做得真的很好。”我一直觉得这是一种鼓励彼此的甜蜜方式，当你处于育儿的最艰难时期，一切似乎都是一场斗争，你觉得周围没有人看到你有多努力。



当我在 r/Parenting subreddit 上看到这篇关于一位母亲 u/jmbrinker 的帖子时，我想到了这句话，她带着自己蹒跚学步的孩子去杂货店购物时受到了称赞:

> 我们刚刚搬家，在我们的新店，他们有小孩购物车，所以我让她推她自己的购物车，我帮忙一起把食物放进去。
> 
> 为了让她忙起来，我让她挑选一些物品，让她拿起来，然后让她把它们放进购物车。
> 
> 附近的一位女士看到了我们，她说:“我喜欢你对你女儿的耐心。看着真甜蜜。”

她接着说，去杂货店购物常常是一种行为灾难，“我希望她知道这几句话意味着什么。如果说她的话让我变得更有耐心的话！”

正如 u/Talisker875 在该帖子的评论中所指出的:“对父母的赞美比‘你的孩子很可爱’的赞美要好得多。“非常满意。”

这让我渴望更多这种感觉良好的故事，所以我问脸书育儿集团 的 [后代，他们是否曾经从野外的陌生人那里得到育儿方面的称赞。](https://www.facebook.com/groups/2018785615043946/)

“我们当时正和我 16 个月大的孩子一起乘坐一次跨国飞行，”小组成员 Shruti 说。“她还是个襁褓中的婴儿，所以我们尽了最大努力来控制她。在飞行结束时，坐在过道对面的女士递给我一只折纸鹤，说我做得很好，顺其自然，她可以看到我的女儿受到了喜爱。”

另一位评论者 Lily 说，当她十几岁的时候，她经常看到她的妈妈在公共场合称赞小孩子的父母。她当时觉得这很尴尬(因为十几岁)，但现在她自己有了一个蹒跚学步的孩子，她看到了她妈妈给那些父母的礼物。

“当你的孩子在公共场合调皮捣蛋时，即使没有人大声说什么，你也能听到别人看你的眼神，”莉莉说。“然而，当你的孩子举止得体地离开公园时，你会觉得没人注意到这一点。！我的孩子今天杀了它！”

有时候，当你发现一个甜蜜的互动时，简单的评论会让某人的一天变得愉快。

“我和我的女孩在推车里，我在塔吉特百货，”杰斯说。“我们在聊天，接吻。一位年长的女士开心地笑着说，我们的联系让她想起了很久以前的她和她的女儿。”

餐馆也是表达赞美的好地方。杰瑞莎带着她蹒跚学步的孩子和一个朋友去一家出乎意料地不适合孩子的餐馆吃早午餐。但她的女儿处理得像冠军一样，在整个用餐过程中静静地涂着颜色。

“吃过早餐后，我女儿说，‘好了，妈妈，该走了’，于是我们收拾行李，”杰瑞莎说。“直到我们要起身离开时，其他顾客才意识到房间里还有一个蹒跚学步的孩子！有几个人评论我的女儿是多么安静和乖巧。”

当然，每一个坐在餐厅里的天使般的孩子，都有一个因为饥饿或被控制而融化。对于每一个在杂货店里合作的小孩，都有一个在他们到达收银台时大发脾气的小孩。在这种情况下？小组成员 Miki 说要提供帮助。

“当我女儿几个月大的时候，我在商店里，我完成了整个购物过程，她没有哭。嗯，你难道不知道，她开始在收银台尖叫，”Miki 说。“我保持冷静，但我努力打包好东西，付了钱，然后离开那里。两位女士过来帮我。他们在我两边的车道上。他们甚至不认识对方。但他们打包了我的东西，告诉我只要抱着我的孩子，他们会照顾它。我拼命忍住眼泪，他们真是太好了。”
