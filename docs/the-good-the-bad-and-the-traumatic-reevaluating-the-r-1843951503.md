# 好的、坏的和创伤的:重新评估在社交媒体上分享黑死病的角色

> 原文：<https://www.theroot.com/the-good-the-bad-and-the-traumatic-reevaluating-the-r-1843951503>

从美国奴隶制时代到现在，世界已经看到了大量描述黑人创伤的媒体——从死亡到暴力，以及其间的一切。

在阵亡将士纪念日，克里斯蒂安·库珀是白人愤怒的目标。

在一场 [现在病毒般蔓延的遭遇](https://twitter.com/melodyMcooper/status/1264965252866641920) 中，这位狂热的观鸟者在中央公园与一名拒绝拴住她的小狗的白色遛狗者发生了冲突。宠物主人艾米·库珀(Amy Cooper)报了警，并错误地指控非裔美国男子克里斯蒂安·库珀(没有亲属关系)“威胁”她的生命。克里斯蒂安在摄像机 上捕捉到了 [的全过程，这场争吵成为头条新闻。他决定拍摄这一事件是故意的。](https://twitter.com/melodyMcooper/status/1264965252866641920)

“当菲兰多·卡斯蒂利亚在明尼苏达州被枪杀时，他做了所有我们被告知的事情，”克里斯蒂安告诉《根报》。“你应该把自己扭曲成各种各样的椒盐卷饼，以确保一些白人不会觉得你是一个威胁。”他继续说，“他做了所有这些事情，但最终还是在他女朋友面前被枪杀，而孩子还在车后座。当这种情况发生时，我对自己说，“无论如何，他们都会向我们开枪。”在这种情况下，我要带着我的尊严完整地走下去。"

在他的病毒中央公园遭遇，基督教是一个黑人证人(拍摄事件)和暴力的目标。虽然在某些情况下，手机镜头是证明暴行发生的必要条件——在这个特定的案件中，克里斯蒂安·库珀是无辜的——但人们不禁想知道一次又一次目睹针对黑人的暴力会产生什么影响。

[艾丽莎诉理查森](https://annenberg.usc.edu/faculty/allissa-v-richardson) 是南加州大学新闻学助理教授，著有《见证黑人:智能手机与抗议新闻 】一书。在她的文本中，她讨论了黑人见证的概念，并确定了针对黑人的国内恐怖的三个历史时期:奴隶制、私刑和警察暴行。

在研究最近针对黑人生活的抗议活动时，教授说主流媒体在报道中也表现出了差异，与 [【开放的美国】](https://www.theroot.com/the-open-america-protests-are-a-classic-example-of-the-1843022481) 的抗议活动形成对比。选择抗议的非裔美国人没有得到类似的公正。

“一开始是一场有意义的和平示威，然后是你在电视上再次看到的火与硫磺的报道。对那些为他们关心的人、他们爱的人抗议的黑人进行再创伤，甚至制作讽刺漫画。”理查森继续详述了差异:“他们[“开放美国”抗议者]可以示威，他们的要求几乎立即得到满足；他们一持枪站在台阶上，各州就开始重新开放。”
