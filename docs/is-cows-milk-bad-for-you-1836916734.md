# 牛奶对你有害吗？

> 原文：<https://lifehacker.com/is-cows-milk-bad-for-you-1836916734>

燕麦牛奶、杏仁牛奶、豌豆牛奶和其他基于植物的饮料随处可见——这可能会让人们看起来好像突然开始回避牛奶了。但是不用担心:如果你喜欢的话，牛奶还是很好的！



### 牛奶并不适合每个人

几乎每个人出生时都会产生一种叫做乳糖酶的消化酶。它帮助人类和其他哺乳动物的婴儿消化母乳中的乳糖。

随着年龄的增长，世界上大多数人都停止产生这种酶。如果你“乳糖不耐受”，我说的是你。我们中的一些人可以像成年人一样喝牛奶，但随着年龄的增长或继发于其他消化问题，最终会出现乳糖不耐受症。

也有可能对牛奶过敏，或者只是不喜欢喝太多牛奶。底线是，如果你觉得牛奶不适合你，没有必要喝它，你可能会喜欢植物性替代品。但是如果你喝了牛奶感觉很好，继续坚持。

### 牛奶比植物奶含有更多的卡路里和蛋白质

一些人出于营养原因选择植物性牛奶。 [都是](https://lifehacker.com/here-s-how-soy-milk-stacks-up-against-other-plant-based-1822584428) 每杯热量比全脂牛奶低，脂肪更低。

与脱脂牛奶相比，营养更相似:脱脂牛奶每杯含 85 卡路里(相比之下，各种植物奶含 45-115 卡路里)，几乎不含脂肪。

不过，牛奶确实比植物奶含有更多的蛋白质。其他一些牛奶含糖量更高，尽管这取决于配方和植物奶是否加糖。普通脱脂牛奶不添加糖，尽管如果你正在看巧克力或调味牛奶——任何种类——检查标签。

牛奶含有饱和脂肪，而一些植物牛奶不含饱和脂肪，但这是否是个问题仍有争议。有心脏病风险的病人经常被建议避免饱和脂肪，但是 [吃全脂乳制品的人不一定不健康](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC5867544/) 。(很复杂。)如果你不想摄入饱和脂肪，脱脂或脱脂牛奶是不错的选择，但植物奶也是不错的选择。
