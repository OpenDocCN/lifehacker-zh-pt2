# im filmmaker kelsey taylor and this is how i work 1836425040

> 原文：<https://lifehacker.com/im-filmmaker-kelsey-taylor-and-this-is-how-i-work-1836425040>

为纪念第一部*外星人*电影四十周年，20 世纪福克斯(通过工作室平台 [【通加尔】](https://tongal.com/) ) 委托拍摄了六部以*外星人*宇宙为背景的新短片。最受欢迎的是凯尔西·泰勒导演和制作的《外星人:标本》。泰勒还参与了 CollegeHumor、1 分钟恐怖片和她自己的几部短片的制作。我们与泰勒谈论了她在片场、编剧室和后期制作中的过程，并看到了《标本》的首映时间表。在她的第一部故事片上映前好好了解一下她。

* * *

**地点:**洛杉机
**现任 gig:** 自由导演，前期制作/开发
**现任电脑:** MacBook Pro 13”(没有 TOUCHBAR，我还没准备好)；iMac 27”(Retina 5K)
**当前移动设备:** iPhone 7，128GB
**一个最能描述你工作方式的词:**要么全部要么什么都没有。

#### 首先，告诉我们一些你的背景，以及你是如何走到今天这一步的。

我在华盛顿东部的一个小镇(那里不下雨)长大，从 11 岁起就非常渴望拍电影。它始于我为五年级的一个项目改编《永远的塔克》这本书，从那以后就没有停止过。

我一直知道我想去电影学院，尽管所有的警告说，我应该得到一个真正的学位，并回到电影制作。我就读于罗耀拉·玛丽蒙特大学，毕业时获得了电影制作学位，辅修了电影研究和音乐。

<label class="bxm4mm-13 juykRM">Advertisement</label>

大学毕业后，我做摄影助理和电工来支付账单。导演的机会少之又少，但我抓住了每一个我能抓住的机会——不管有没有报酬。这一切都是为了建立我的投资组合和建立关系，即使这意味着制作和助理导演来证明我的价值。你永远不知道一个项目会如何引发下一个项目。

进入[Tongal](https://tongal.com/)——一个非常酷的精英平台，允许各地的电影制作人为知名品牌和娱乐公司推介项目。他们与 20 世纪福克斯共同主持的一项令人难以置信的倡议真正改变了我的生活，该倡议向六名电影制作人提供预算，以创作以*外星人*宇宙为背景的原创短片。在 [那个短篇](https://www.youtube.com/watch?v=dYAHOZ_91A0) 的帮助下，我见到了我的经理，他把我写的一个剧本推进了开发。我真的很兴奋能过渡到故事片，并为明年开发另一部故事片。
