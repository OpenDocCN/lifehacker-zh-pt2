# 使用 Goo Gone 清洁污渍、口香糖和几乎所有东西

> 原文:[https://life hacker . com/use-goo-go-to-clean-stains-gum-and-pretty-much-ever-1835006511](https://lifehacker.com/use-goo-gone-to-clean-stains-gum-and-pretty-much-ever-1835006511)

Goo Gone 是一种神奇的粘合剂去除剂。我们许多戏剧和视频从业者都知道这是去除舞台和道具上的胶带和价格标签残留物的最好方法。

如果你有一瓶油性柑橘清洁剂，你可能会用它来温和地溶解任何难以洗掉的东西。(我主要在家里用它来清理粘在空调上的胶带。父母也可以用它来清除蜡笔痕迹。)

但是 Goo Gone 能做的远不止消除胶水(似乎也没有一个类似的通用名称)。我们发现它在去除水杯和盘子上的标签、清除剪刀上的胶水以及溶解织物和鞋子上的口香糖方面非常有用。它甚至可以去除黄色的腋窝污渍。(我告诉你，这东西很神奇！)

使用 Goo Gone 的方法是一样的，不管你溶解的是什么或者它在什么表面上:在 Goo 上擦拭 Goo Gone，等待大约 5 分钟，擦洗掉，用肥皂水清洗，干燥。

虽然它似乎可以将*几乎*任何从*粘上的物品几乎*任何表面，但制造商建议不要在丝绸、皮革、麂皮、橡胶、人造不锈钢、干墙、未加工的木材表面或未密封的石头上使用它。在开始之前，最好在一个小范围内做一点测试。
