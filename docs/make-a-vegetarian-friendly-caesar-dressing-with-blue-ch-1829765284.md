# 用蓝奶酪做一道素食友好的凯撒沙拉酱

> 原文：<https://lifehacker.com/make-a-vegetarian-friendly-caesar-dressing-with-blue-ch-1829765284>

真正的凯撒沙拉酱从凤尾鱼中获得丰富的咸味，但不是每个人都吃鱼。吹毛求疵的人可能会坚持认为那些人根本不配得到凯撒沙拉，但我认为这有点苛刻，特别是当你可以用蓝奶酪做一个非常好的 T4 素食者。



用蓝纹奶酪做的凯撒调味汁尝起来和用凤尾鱼做的完全一样吗？当然不是——奶酪不是鱼。用蓝色制成的凯撒调味汁会令人惊讶地接近，将那种咸咸的、难闻的、大蒜味的瘙痒遗忘吗？是的。为了达到最佳效果，你要使用你能找到的最令人讨厌的成熟蓝纹奶酪，每半杯油要用大约三汤匙碎奶酪。它几乎适用于任何凯撒调味汁配方；省略凤尾鱼，加入蓝纹奶酪和帕尔马干酪。我喜欢《好胃口》 中的 [这个食谱，并以此为基础做了我的素食版。要自己制作不含鱼的调料，你需要:](https://www.bonappetit.com/recipe/classic-caesar-salad)

*   两瓣大蒜
*   盐
*   2 个蛋黄
*   1 茶匙第戎芥末
*   2 汤匙柠檬汁
*   1/2 杯植物油，加上 2 汤匙橄榄油(你也可以使用所有的橄榄油，但它最终会与许多其他强大的味道竞争，最终变得浑浊。)
*   3 汤匙帕尔马干酪
*   3 汤匙蓝纹奶酪
*   大量的新鲜胡椒

切碎你的大蒜，撒上大量的盐，然后用你的刀的平刃挤压成糊状，拖着它穿过蒜泥堆，尽可能多的挤出蒜汁。把它和蛋黄、芥末和柠檬汁一起放在一个足够宽的杯子里，以适合浸入式搅拌机的搅拌头。把你的浸入式搅拌机放在那里，搅拌混合，然后慢慢把油倒在其他配料上。将浸入式搅拌机的头直接推入杯底，然后在高处搅拌，慢慢上下移动，直到一切都变好并乳化。加入奶酪，加工至光滑，然后用新鲜胡椒调味，再次搅拌。
