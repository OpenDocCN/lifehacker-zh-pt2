# 推出“家长咨询”建议专栏

> 原文：<https://lifehacker.com/introducing-the-parental-advisory-advice-column-1838976475>

在养育孩子的过程中，我们都需要一些帮助。初为人父母的人，已经做了一段时间的父母——我们所有人都会在某个时候发现自己处于十字路口，不知道该如何前进。正是在这个十字路口，我们希望后代的新父母咨询专栏能有所帮助。



我想帮助你解决困扰谷歌——甚至是你最好的朋友——的问题。复杂的人际关系、人际关系和我们每个孩子的独特需求所产生的问题。

那么，为什么我是帮助你解决最棘手的育儿难题的合适人选呢？如果你在这里已经有一段时间了，你可能知道我有一个九岁的儿子，你也可能知道我以前是养父母。做养父母给了我培训、见识和经验，让我知道养育多个孩子是什么感觉，不同性情的孩子，经历过创伤的孩子，不同种族的孩子，比我儿子小的孩子和比我儿子大的孩子。

其次，我已经写了几年关于为人父母的文章，但我主要是一名记者。有一个观点是好的，但是当那些观点被事实和被证明的方法支持时，那就更好了。我也喜欢认为，当我*不*知道一些事情的时候，我知道的足够多了(你明白吗？)，所以我会为你面临的最敏感、最复杂的障碍寻求专家的意见。

最后，我喜欢帮忙。为人父母真的，说实话，非常困难。它开始*硬*并保持*硬*。一旦你越过一个障碍，另一个障碍就在前方等着你。我们在一起，当感觉我们的脚不可能再向前迈一步的时候，互相支持，拖着对方穿过终点线。

所以，把它放在我身上。你最迫切的育儿问题是什么？你和你的伴侣在什么问题上*不能*达成一致？我该如何帮助你摆脱与大家庭成员的不良关系，或者解决你的孩子在学校遇到的令人烦恼的问题？

把你的问题发邮件给 mwalbert@lifehacker.com，在主题行写上“家长建议”，我会在这里试着回答你的问题。
