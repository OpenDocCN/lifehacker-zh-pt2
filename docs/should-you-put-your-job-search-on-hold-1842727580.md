# 你应该暂停找工作吗？

> 原文：<https://lifehacker.com/should-you-put-your-job-search-on-hold-1842727580>

如果你正在求职，最近的国际事件可能会让你重新审视自己的职业规划。公司 [停止了](https://lifehacker.com/before-you-apply-for-a-job-during-the-pandemic-check-t-1842568418) 的招聘过程，失业申请暴涨，人们在他们的州 [失业](https://twocents.lifehacker.com/how-unemployment-benefits-are-changing-with-the-senate-1842494496) 系统中导航，他们的耐心值得嘉奖。

现在找工作值得吗？这不是一个失败的原因，但你应该在填写下一份申请之前记住一些事情。

### 如果你有一份工作，但想要一份更好的

不要放弃找工作，但是如果你现在有工作，尽你所能保住这份工作是个好主意。

“这当然是一个寻找新工作的竞争时刻，”职业趋势专家阿米莉亚·格林-瓦莫斯说，她在 glass door 工作，该公司发现超过一半的雇主在疫情期间暂停招聘。

“如果你在受新冠肺炎影响最大的地区工作...医疗教育公司的首席执行官米丝蒂·弗罗斯特说:“你有一份工作，坚持下去，直到我们知道更多可能是安全的赌注。

但享受目前工作的相对安全感并不意味着你应该自满。格林-瓦莫斯说:“了解你所在行业的需求以及对你的技能和专业知识的需求非常重要。”。"了解需求可以帮助你评估如何最好地前进."

弗罗斯特建议求职者关注疫情期间和之后有望增长的领域，尤其是教育和医疗保健。

“医疗保健预计到 2028 年将创造近 200 万个新工作岗位，这还是在我们了解冠状病毒之前，”弗罗斯特说，举了一个例子。“现在我们看到，不仅医生和护士的需求在增加，辅助人员的需求也在增加，比如病人护理技术员、医疗助理和药房技术员等等。”

劳工统计局在其月度 [当前就业统计概要](https://www.bls.gov/ces/) 报告( [这是 3 月的报告](https://www.bls.gov/web/empsit/ceshighlights.pdf) )中概述了特定行业的雇佣数据。 [Glassdoor](https://www.glassdoor.com/research/job-market-report/) 提供可以按地区和行业过滤的月度就业市场报告。

### **如果你刚刚失业**

“试着保持耐心，要知道虽然就业市场现在看起来不确定，但仍有公司在这个时候雇佣有才华的员工，”格林-瓦莫斯说。“设置工作提醒，更新你的简历，重新思考给你以前的工作带来价值的技能，以及如何将这些技能转化为新的角色。”

格林-瓦莫斯还说，考虑你认为自己有资格胜任的工作，因为你的许多技能不仅可以转移到其他职位，也可以转移到各种行业。

当你失业时，你可能想考虑学习新技能的方法。弗罗斯特指出，虽然有耗时不到一年的正式工作培训和再技能培训项目，但在大流行相关限制解除之前，你可能无法参加这些项目。与此同时， [在线课程](https://lifehacker.com/where-to-take-free-online-classes-while-you-re-stuck-at-1842496408) 可能会帮助你填补你的时间和你发现的任何技能缺口。

### **记住，这是暂时的**

尽管就业市场现在看起来很惨淡，但要相信目前的状况不会永远持续下去。“尽管我们在过去几周看到了裁员高峰，但有些领域现在仍然需要工人，在疫情之后甚至需要更多工人，”弗罗斯特说。

格林-瓦莫斯说，如果你失业了，却能在疫情期间找到一份兼职，不要担心简历上的潜在差异。她说:“与其在简历中隐藏不相关的临时工作或零工，重要的是对招聘人员和招聘经理保持透明，以帮助他们了解你的工作经历。”“同时，利用这个机会，清楚地强调你所做的工作如何增加了新的技能和经验，这些技能和经验可以转移到你申请的全职工作中。”

她说找到临时工作的勇气和足智多谋展示了你的职业道德，所以不要掩饰。

弗罗斯特说:“我一遍又一遍地从雇主那里听到的一件事是，他们试图填补的许多关键职位既需要良好的人际交往技能，也需要自我激励的能力。”“临时工作无疑是展示雇主所寻求的那种主动精神的一种方式。”
