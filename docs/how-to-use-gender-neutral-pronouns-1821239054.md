# 如何使用中性代词

> 原文：<https://lifehacker.com/how-to-use-gender-neutral-pronouns-1821239054>

韦氏词典今年正式将单数非二进制“他们” 加入他们的字典，标志着我们看待和接受中性代词的方式发生了重大转变，并禁止那些拒绝使用他们/他们的人，因为他们“语法不正确”如果你还不习惯这种包容的语言——拜托大家，都 2019 年了！—这里有一本方便的入门书。

## 为什么要用中性代词？

有很多很好的理由让你养成使用中性代词的习惯，但这里有两大理由:

*   你可以指个人或一群人，而不用假设(或猜测)他们的性别，也不用假设他们是男是女。(性别不是二元的。)
*   当你知道一个人使用中性代词时，你会更容易自然地使用这些代词，而不必在说话前在头脑中“翻译”。

中性代词很棒，因为它们允许你用*到*和*来谈论*个人，而不会对他们的性别做出不正确的假设。毕竟，仅仅因为某人看起来女性化或男性化并不意味着他们是男人或女人——他们可能是年长的或非二元的，或者只是与你对男人或女人的期望不同。使用中性代词可以让你在说话时包含所有人，并鼓励其他人也这样做。

## **有哪些常见的中性代词？**

“他们”是最常用的中性代词——事实上，你可能已经在日常语言中不假思索地使用了“他们”。

我和客服代表谈过，他们帮我解决了问题。

好像有人把夹克落在椅子上了。

我不知道谁是负责人，但我们应该找到他们。

即使“他们”可以用作复数代词(即指一群人)，但当人们不知道某人的性别时，他们往往会自动使用“他们”作为单数代词。“他们”也是长辈或非晚辈使用的比较流行的代词之一(当然，根据韦氏词典的说法，它现在是官方的。)

这意味着将“他们/他们/他们的”添加到你的词汇中应该相对容易。用“他们”来指代那些告诉你这是他们的代名词的人，但是不要忘记用“他们”来指代那些你不知道性别的人——正如我上面提到的，几乎是所有你不认识的人。

看起来那就是我们要见的人。看，他们在向我们招手。

(在你问之前:是的，在学术和新闻工作中使用“他们”作为单数代词是合适的。它甚至出现在 AP 样本手册中。)

以下是人们可能会要求你使用的其他一些中性代词:

*   Ey/em/eir/eirs
*   Ne/nem/nir/nirs
*   Xe/xem/xyr/xyrs
*   Ze/hir/hir/hirs
*   泽/齐尔/齐尔/齐尔斯

这些不是唯一的中性代词——由于语言在不断发展变化，如果我们在未来几年开发出新的中性代词，不要感到惊讶。

另外:如果你在寻找一个中性词来代替先生/夫人/小姐/女士，试试 *Mx* ，发音为“mix”或“mux”【T2

## **“他或她”不是中性的**

当我们指所有性别的人时，我们中的许多人习惯于说“他或她”——我们中的许多人都足够大，可以记得“他或她”被视为包容性语言选择的时候。(信不信由你，过去人们只说“他”，我们用阳性术语来指代这个世界:人类、推销员等等。)

然而，使用“他或她”排除了年长的或非双生子的人。这是一个很好的机会，让你用一个中性且包容的词语来代替二元排斥的词语:

*   用“他们”、“他们”、“这个人”等代替“他或她”。
*   不要用“女士们、先生们”来问候人群，而要用“先生们”、“你们好”或者中性短语，比如“大家好！”

此外——这是不言而喻的——尽量避免称一群人为“你们”虽然有些人认为“你们这些家伙”是中性词，但其他人认为它是专有名词，因为“家伙”在传统上是阳性的。(正如 [石板](http://www.slate.com/blogs/lexicon_valley/2016/02/10/the_gender_neutral_use_of_guys_is_on_the_rise_but_it_s_a_slow_rise.html) 提醒我们的那样，“盖伊”在单数中几乎总是指男人。)

## **不要忘记名词中的代词**

一些名词包括性别代词，而另一些名词即使不包括代词，也隐含着性别二元性。(想想“服务员/女服务员。”)注意你在说什么。我们大多数人会自动使用“人类”和“国会议员”这样的词，所以这对你来说不应该是一个巨大的转变——但是尽你所能选择中性的术语，比如“服务器”

你还应该注意避免性别二元的其他机会，例如使用短语“来月经的人”而不是“来月经的女人”

#### 如何学习别人的代词？

有时候学习一个人的代词很容易。他们会在自我介绍时告诉你，或者他们会在姓名标签上写下他们的代词，或者在社交媒体档案中包括他们。

然而，向陌生人询问他们的代词不一定是个好主意。尽管这样做感觉是正确的，但要知道你可能会要求某人公开自己是跨性别者、非双性恋者——根据具体情况，他们可能不太愿意这样做。(想象一个保守的职场。)

相反，你可以通过分享你自己的代词来开始对话:

我是妮可，我的代词是她/她。

分享你的代词让其他人可以选择分享他们的代词，而不用直接问他们。另外:注意上面的例子使用了短语“我的代词”，而不是“我的首选代词”虽然你偶尔会听到“首选代词”这个术语，但它并不总是最好的用法，因为它暗示了一个人的性别只是一种偏好。

一旦你学会了某人的代词，如果你不小心用错了代词，不要大惊小怪。说“对不起，我是说*他们*”就可以了；说“我不是故意的，我很抱歉，我真的在努力”是在创造关于你的时刻。通过在接下来的对话中使用正确的代词来表明你在努力。

一旦你养成了使用中性代词的习惯，你就会意识到不用对性别做任何假设来指代他人是多么容易。我在整篇文章中使用了中性代词，这并没有妨碍你理解我想表达的意思。你越多地使用中性代词，我们所有人就会越来越多地将中性词汇纳入我们的词汇中——我们的语言也会变得越来越包容。
