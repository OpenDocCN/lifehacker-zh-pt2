# 这位科幻作家预言了互联网并发明了水床

> 原文：<https://lifehacker.com/this-sci-fi-writer-predicted-the-internet-and-invented-1829840361>

在我们最新的视频系列*历史上的生活黑客*中，我们审视了过去名人的惊人成就。首先是科幻作家罗伯特·A·海因莱因。

作为一名海军工程师，海因莱茵利用他的技术知识在他的作品中做出预测，其中许多预测最终都成为了现实，比如州际公路和互联网。

他还提出了现代水床的想法，给了我们写作的指导方针，并建造了一个定制的技术梦想家园。在上面的视频中查看他一生的工作，并让我们知道是否有任何你认为应该作为未来历史生活黑客的历史人物。
