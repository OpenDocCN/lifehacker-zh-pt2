# 更新你的网络解决方案，Register.com。还是 Web.com 现在的密码

> 原文：<https://lifehacker.com/update-your-network-solutions-register-com-or-web-com-1839635464>

如果你曾经从 Network Solutions、Register.com 或 Web.com 购买过域名，你应该尽快修改你的密码。域名注册商 [宣布](https://notice.networksolutions.com/?siteid=100&clickid=1000000000&channelid=P99C100S500N0B11A1D38E0000V288&cid=NetSol%20Customer%20Notice%20-%2011/04/2019&bid=567782307) 于 2019 年 8 月遭遇数据泄露。



在给客户的电子邮件中，这些公司解释说，他们在 2019 年 10 月得知了这起违规事件。虽然没有访问信用卡数据，但帐户信息可能已经泄露。这些信息可能包括姓名、地址、电话号码、电子邮件地址和订单详情。

Web.com 拥有网络解决方案公司和 Register.com。来自 Web.com[的一名发言人本周告诉 Krebs on Security](https://krebsonsecurity.com/2019/10/breaches-at-networksolutions-register-com-and-web-com/) 公司，账户密码已经加密，不怀疑已经被泄露。

为了安全起见，所有客户都需要重新设置密码。来自 Network Solutions 的一封电子邮件解释说,“受影响的客户”正在通过电子邮件和网站得到通知。

不得不为一个你可能只是偶尔访问的网站更改你的帐户信息是一件痛苦的事情(比如当你有了一个网站的“好主意”,然后在接下来的五年里一直使用这个域名...没有名字，没有名字)。但是如果你还在用 Password123！对于您的任何在线帐户，我们都有一个指南 [来支持您的在线安全](https://lifehacker.com/when-a-company-asks-you-to-reset-your-password-should-1837516590) 。面对现实吧，你可能早就该更新了。
