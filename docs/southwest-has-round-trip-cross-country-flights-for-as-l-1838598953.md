# 今天，西南航空公司的全国往返机票价格低至 164 美元

> 原文：<https://lifehacker.com/southwest-has-round-trip-cross-country-flights-for-as-l-1838598953>

如果你计划明年年初进行一次越野旅行，西南航空公司正在进行一次促销活动，你可能想去看看。



[积分专家](https://thepointsguy.com/deals/deal-alert-southwest-cross-country-flights-for-under-200/) 强调了一些不同的路线，今天你可以用低于 200 美元的价格获得一张跨国往返机票。例如，从洛杉矶到新奥尔良的票价是 198 美元，从洛杉矶到奥兰多的票价是 190 美元，从西雅图到芝加哥的票价是 164 美元。

旅行日期需要在明年 1 月到 3 月之间。

与大多数其他票价交易不同的是，由于这些交易是在西南航空公司进行的，所以它们也是可变更和可退款的。这意味着如果有更好的交易，你可以换一张不同的票。或者，如果你的旅行计划改变了，你决定在不同的时间旅行，你可以让它发生。

需要注意的一点是，退款将以西南航空信用的形式出现，所以一旦你买了机票，现金就必须用于乘坐西南航空的航班，尽管这不一定发生在你预订这张原始机票时。

如果你正在考虑一次旅行，今天就有必要查看一下航空公司提供的服务，看看你是否能达成交易。和往常一样，确保你也在查看同一天其他航空公司的价格。虽然这些肯定是打折的西南航班，但这并不意味着另一家航空公司可能不会在同一条航线上为你提供更好的交易。
