# 这个强大的 iOS 快捷方式可以让你创建自定义的应用程序图标

> 原文:[https://life hacker . com/this-powerful-IOs-shortcut-let-create-you-custom-app-I-1830748858](https://lifehacker.com/this-powerful-ios-shortcut-lets-create-you-custom-app-i-1830748858)

[MacStories](https://www.macstories.net/ios/home-screen-icon-creator-a-shortcut-to-create-custom-icons-for-apps-contacts-solid-colors-and-more/) 博主 Frederico Vitticci 创建了一个非常强大的 iOS 快捷方式，名为 [主屏幕图标创建器](https://www.icloud.com/shortcuts/c205225621f14baca558f13ab2ae734a) ，它允许你创建看起来不同的自定义按钮，如果你想变得有趣，还可以微调当你打开应用程序时会发生什么。它还可以让您为 iOS 快捷方式创建可自定义的图标，以及从您的通讯录中呼叫联系人的图标。

虽然它有许多可能的用途，但这个快捷方式最吸引人(也最有趣)的地方是创建一个新的应用程序按钮，将默认图像替换为 app store 上的任何其他图标。我要说的是，它看起来比实际更复杂，但是一旦你掌握了它，你就可以很容易地为你的每一个应用程序设置新图标。

要做到这一点，点击快捷方式，并按下“启动带有 URL 方案的应用程序，”这将提示一个文本框，要求您输入，你猜对了，一个 URL 方案。在大多数网站上,“方案”是 URL 的一部分——特别是冒号前最开始的部分——HTTP://(或 HTTPs://)。不过，在这种情况下，我们不是在搜索互联网，而是在调用一个应用程序。因此，你不用写 HTTP://，而是输入你想让图标打开的应用程序的名称，后跟一个冒号和两个反斜杠。

然后应用程序会询问“命令标题”，这是一种询问你想给应用程序起什么名字的奇特方式。您可以给它取一个与您要替换的应用程序相同的名称，或者取一个更合适的名称...

然后你会得到一个提示，要求你在 app store 中搜索图标使用。键入您想要搜索的内容，然后按下搜索栏上方的完成按钮(不是键盘上的按钮)。在那里，您将看到一个显示名称和图标的登录页面。

若要将图标添加到主屏幕，请按下底部菜单栏上的“共享”按钮(带有向上箭头的框)，然后选择“添加到主屏幕”从那以后，你还有一次重新命名它的机会:当你准备好了，按“添加”并...

瞧啊。这是您改进后的新应用图标。

这个快捷方式不允许你制作自己的应用程序图标，也不允许你使用应用程序图标以外的图片。然而，Vittici 的 [对快捷方式](https://www.macstories.net/ios/home-screen-icon-creator-a-shortcut-to-create-custom-icons-for-apps-contacts-solid-colors-and-more/) 的分解解释了如果你稍微修改一下，你可以如何修改他的快捷方式，从照片应用程序而不是应用程序商店中选择图像。

说到这里，让我们再多谈谈你还可以做些什么来定制你的应用启动器。通过让用户输入一个网址，而不是简单地选择一个应用程序，Vittici 为你打开了定制当你点击一个应用程序图标时会发生什么的大门。通过添加到字符串，你可以让它按下一个应用程序图标打开一个特定的菜单，执行特定的搜索，或者以其他方式自动完成你第一次打开它时总是做的事情。

正如你所料，这个过程可能会非常复杂，代码量很大，这取决于应用程序和任务的性质，但通常并不像你想象的那么困难。如果你想实验一下，来自 app developer Contrast 的 [本入门](https://help.contrast.co/hc/en-us/articles/200865293-URL-Scheme-Intro-for-App-Developers) 第一节有几个例子，可以帮助你理解编写非 web URL 的基本步调。

该快捷方式还让您能够使用从 app store 中提取的图标来制作快捷方式的主屏幕图标，这些图标将呼叫、文本或 FaceTime 特定联系人的指定照片用作图标，或者创建一个空白的彩色编码图标，您可以使用该图标来分隔您的应用程序或自定您的主屏幕。与更改现有应用不同，这些都非常简单。所以，玩一玩，找一些很酷的图标，在你的 iPhone 和/或 iPad 上拥抱这种新的、令人兴奋的审美自由。
