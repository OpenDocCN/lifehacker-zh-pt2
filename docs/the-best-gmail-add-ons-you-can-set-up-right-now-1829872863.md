# 你现在可以设置的最好的 Gmail 插件

> 原文：<https://lifehacker.com/the-best-gmail-add-ons-you-can-set-up-right-now-1829872863>

没有人对电子邮件的工作方式感到满意。

2017 年，谷歌增加了使用“ [”插件](https://gsuite.google.com/marketplace/category/works-with-gmail) 定制 Gmail 的能力，这些插件与 Chrome 和 Firefox 中的浏览器扩展的工作原理相同，但都是为基于网络和安卓系统的 Gmail 版本而设计的。附加组件位于收件箱右侧的小侧边栏中，允许您以新的、令人兴奋的方式定制和体验——尤其是如果您已经使用了一系列受支持的第三方服务。

Watch

这些附加组件中的大多数都为你提供了工具，当电子邮件要求你采取行动时，你可以更容易地做出反应。然而，上周 ，谷歌加倍努力，让开发者能够将插件直接集成到 Gmail 的撰写窗口中，为开发者开发更全面的工具打开了大门。

在推出插件时，谷歌本身 [将](https://www.blog.google/products/g-suite/do-more-your-inbox-gmail-add-ons/) 电子邮件描述为你工作生活的“任务控制”，最好的插件让 Gmail 感觉更像是命令中心，而不是简单(但高耸的)待办事项列表。目前你还不能做太多的事情，但是我们已经编制了一个最佳加载项列表，并解释了它们如何能够帮助你。

### [Dropbox](https://gsuite.google.com/marketplace/app/dropbox_for_gmail/33761876029) 和 [Box](https://gsuite.google.com/marketplace/app/box_for_gmail/828801433755)

如果你使用 Dropbox 或 box 进行云存储，这些插件是最先在 Gmail 的撰写窗口中工作的。例如，如果您安装了 Dropbox 插件，在附件回形针所在的工具栏中会有一个 Dropbox 按钮。

您可以轻松地将 Dropbox 文件添加到您撰写的新电子邮件中，如果您收到了带有附件的电子邮件，只需点击两次就可以将它们转储到您的 Dropbox 中。如果出于某种原因，你需要访问你的 Dropbox，有一个快速打开选项可以让你快速到达那里。所有这些使得从 Gmail 附加存储在这些服务中的文件变得非常非常快。

这两个插件都没有强大到让我们建议你切换存储服务的功能，但如果你使用的是 Dropbox 或 box 和 Gmail，安装这个插件是显而易见的。

### [特雷罗](https://gsuite.google.com/marketplace/app/trello_for_gmail/710372432947)T3 和[阿萨纳](https://gsuite.google.com/marketplace/app/asana_for_gmail/923474483785)T7】

如果你不熟悉，Trello 和 Asana 是项目管理网络应用。它们允许您可视化和管理具有许多移动部件的业务和大型项目。如果你正在使用这些服务中的任何一种，你的电子邮件和你的管理板之间的相互作用实际上与你的电子邮件和你的日历之间的关系相同——你通过电子邮件找到重要的信息，这些信息需要作为一项新任务或现有任务的一部分进入管理板。

这些 Gmail 插件使这个过程变得非常简单。通过点击侧边栏中的 Trello 或 Asana 标志，您可以将任何电子邮件的内容导入到服务中，作为现有任务的新任务。你可以在导入前编辑邮件的正文，这样如果你的同事只需要一小段，你就不需要用大量的文本来扩充任务。

与 Dropbox 和 box 一样，这里没有任何东西会动摇你使用其中一种服务。如果您的团队正在使用这些服务中的任何一个，那么这些确实使它们更容易使用。

### [DocuSign](https://gsuite.google.com/marketplace/app/docusign_for_gmail/469176070494)T3】

DocuSign 是一项让你在具有法律约束力的文件上安全使用数字签名的服务，它有一个 Gmail 插件，让你只需点击两次，就可以将电子邮件中的附件发送到 DocuSign。然后，它也会在 DocuSign 中打开文档，这样您就可以立即处理业务。就这样，便捷的服务变得更加便捷。

### 变焦

使用 Gmail 插件，企业通信服务 Zoom 允许其用户直接通过 Gmail 建立通话或视频聊天，使用电子邮件中的联系信息或保存到谷歌。它还允许你安排以后的会议，并在不打开 Zoom 或日历的情况下检查任何预定的电话。

虽然很有用，但通过 Zoom 协调来电总是意味着必须通过电子邮件手动发送邀请或来电号码。这个插件似乎通过与您已经使用的实用程序更紧密地联系起来，为该服务的一个更大的物流问题提供了一些潜在的解决方案。

### [【冬季】](https://gsuite.google.com/marketplace/app/hiver_gmail_addon/27334010516)

你可能已经注意到了，这些附加产品越来越小众。Hiver 是 Gmail 面向企业的第三方服务，它增加了一层额外的物流功能，帮助员工管理和回复为团体而非个人创建的桶式电子邮件账户:例如，“支持”或“职业”地址。

Hiver 增加了一个“笔记”功能，让您可以分配电子邮件或讨论如何回复消息，突出显示同事何时开始回复，并允许您保存和传播示例或模板消息。基本上，它允许你直接在 Gmail 中完成所有关于你的群组账户的工作，而不是分散在 Slack 或 Dropbox 等应用程序中。

值得指出的是，Hiver 建议你使用它的浏览器扩展，比 Gmail 插件有更多的功能。不过，Gmail 插件仍然是安卓用户的福音，如果你经常切换机器或浏览器，它甚至可能更适合在网上使用。

### [**gfy cat**](https://gsuite.google.com/marketplace/app/gfycat_gifs_for_email/410858369375)T5】

我不得不把最重要的附加组件留到最后。

Gfycat 有一个 Gmail 插件，可以让你快速搜索其库中的完美反应 GIF，并在快速回复中发送它——因为有时 GIF 是你真正想要(或必须)说的所有内容。

相对于其他一些附加组件，Gfycat 感觉有点...有限。该插件只允许您将 GIF 作为回复发送到现有的电子邮件线程，因此您不能使用它来随机发送 GIF 格式的电子邮件。此外，你必须通过插件进行回复，所以你不能写电子邮件，然后选择一个合适的 GIF。

尽管它的有效使用有限，但我在心里为 Gfycat 保留了一席之地。在你能为 Gmail 获得的所有插件中，它是迄今为止最有趣的。
