# 为了拯救生命，学习什么是 AED 和如何使用它

> 原文：<https://lifehacker.com/to-save-lives-learn-what-an-aed-is-and-how-to-use-one-1838740113>

如果你在公共场所看到有人晕倒，你可能已经知道应该拨打 911[开始胸部按压](https://cpr.heart.org/en/cpr-courses-and-kits/hands-only-cpr/hands-only-cpr-resources) 。但是还有一个重要的行动可以帮助这个人活下来:派人去抓最近的 AED。

Watch

自动体外除颤器(AED)是一种可以电击心脏的设备，类似于你可能见过电视医生使用的电气化桨。小型的、用户友好的版本在许多公共场所都有，它们被设计成即使是很少或没有受过训练的人也可以使用。

它们之所以有效，是因为许多心脏停搏是由心室颤动造成的。这意味着心脏的细胞，而不是像它们应该的那样同时跳动，变得不同步。心脏颤抖或振动，而不是在强有力的泵中挤压，不能有效地将血液输送到全身。来自 AED 的强烈电击可以重新启动心脏，并且比单独的 CPR 提高存活率。

### 知道离你最近的 AED 在哪里

当我出门在外时，只要看到 AED，我就会记下来。例如，我知道在我孩子的幼儿园里就有一个，就在前门旁边。它们通常放在公共区域，容易拿到，并且安装在你会注意到它们的地方。 [Pulsepoint](https://www.pulsepoint.org/pulsepoint-aed/) app 有 aed 地图，你可以把找到的加进去。在你花费大量时间(工作、学校等)的地方的前台询问他们是否有 AED 以及在哪里是一个不错的主意。

当需要使用 AED 时，记得拨打 911[开始胸部按压](https://lifehacker.com/start-chest-compressions-immediately-dont-worry-about-5666656) 同时有人去拿设备。

### 遵循指示

如果你参加心肺复苏课程，你可能会有机会使用训练版的 AED。对我来说，令人惊讶的是，你不需要记住具体要做什么——机器会跟你说话，并指导你应该采取的步骤。

其中包括:在你将电极贴在人的身体上后，机器将“倾听”心脏的电信号，以判断此人是否似乎处于心室颤动状态。如果机器检测到正常的心跳，就不会电击这个人。所以你不需要确定*为什么*这个人倒下了；只要连接上机器，让它做它的事情。

这里是 [全套说明](https://www.redcross.org/take-a-class/aed/using-an-aed/aed-steps) ，来自红十字会。请注意，它说 AED 是为“体重超过 55 磅的 8 岁以上无呼吸儿童或成人”准备的如果机器确定这个人需要电击，你要确保每个人都站得很远——没有人碰这个人——然后按下按钮。电击后，或者如果机器不建议电击，像往常一样做 CPR。
