# 通过每次都计划同一个日期来拥有一个成功的第一次约会

> 原文:[https://life hacker . com/have-a-successful-a-first-date-by-planning-the-same-first-1832623751](https://lifehacker.com/have-a-successful-first-date-by-planning-the-same-first-1832623751)

当然，约会是一场微妙的舞蹈，任何时候都有许多未知因素。

Watch

这就是为什么如果你想消除风险，专注于你与初次见面的人的联系，你应该在*同一天*进行第一次约会。

通过总是去同一个地方(如果你想更进一步，点同样的东西)，你不需要依靠任何未知来影响你对新追求者的看法——也许我们是因为糟糕的服务而结合在一起的？是我恋爱了还是炸鸡真的很好吃？他们建议在他们的社区里找一个地方，我是不是心怀恶意？

用这种方法去也消除了“我们应该去哪里？”的来回文本“你喜欢什么？”以及“什么对我们俩都方便？”你所要做的就是主动说“我知道一个好地方。我们要去这里。”

当然，如果你想更科学(或者像一些同事说的，“反社会”)，你可以通过多日疗法*获得额外控制的样本。没错，我曾经*有一个专利的三-约会系统:1)同一个冰激凌店——美好而随意，低赌注，如果他们不喜欢这个冰激凌，那么他们就不值得第二次约会。2)同一个意大利餐馆——中心位置，你知道你愿意花至少一个小时和他们在一起，意大利面也足够令人愉快。3)在我家吃晚饭——如果你想知道，是的，我每次都做*一样的菜*。*

*所以你有它:没有思考，没有不确定性。只要看看不同的人在同一种情况下的反应，就能了解他们的真实性格。也许不要让他们知道你也在那里约会了几十次。*

**是的，事实上，我处于一段幸福的长期一夫一妻制关系中，这进一步证明了这个系统是有效的。*
