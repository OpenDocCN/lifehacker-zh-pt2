# 最佳密码管理器

上周，我们要求您在您最喜欢的密码管理器 上填写 [。在梳理了您的回答，测试了一些新的回答，并了解了其他流行的选择之后，我们列出了最佳密码管理器。](https://lifehacker.com/tell-us-about-your-favorite-password-manager-1835910472)

### [](https://bitwarden.com/)****(免费)****

**我们收到的大多数读者评论都称赞了[](https://bitwarden.com/)**，你们中的许多人说你们是从 LastPass 或其他应用程序迁移到这里的。在亲自检查了这项服务之后，我们不得不承认它是最好的密码管理器之一。这项服务不仅可以在许多设备和平台上使用，而且是完全免费和开源的。****

****该应用程序具有许多 2FA 安全方法，并允许您从其他密码管理器、web 浏览器和各种应用程序中导入您的密码数据。BitWarden 还支持所有受支持平台的云同步，包括 Windows、Mac、Linux、iOS、Android、几乎所有流行的 web 浏览器、其基于云的客户端等等。**** 

****最重要的是，BitWarden 的开源设置意味着该服务在如何存储你的密码数据方面是高度透明的，这些数据通过多种形式的端到端加密和 salted hashing 得到保护。****

****唯一的缺点是 BitWarden 没有完整的离线模式，但正如 life hacker reader**binary visions**指出的那样:“当你登录时，你的保险库会发生完整的同步，”所以从技术上来说，只要你同步了最近的密码数据，即使你离线，你也可以*使用 BitWarden。*****

### ****[**KeePass**](https://keepass.info/)**(免费)******

****第二个被提及最多的密码管理器是 [**KeePass**](https://keepass.info/) 。KeePass 是另一个免费的、经过认证的开源选项——你知道我们有多喜欢开源选项。**** 

****正如一些人指出的那样，KeePass 第一眼看上去可能不会给人留下深刻的印象，但不要因此而却步。评论员 **JoshMC** 总结的很好:****

> ****“我承认，与背后有企业支持的密码管理器相比，这不是最容易使用的，也不是最漂亮的，但一旦你克服了困难，就可以一帆风顺了。”****

****用付费产品的天赋换取开源程序的安全性和透明性对我们来说感觉很好——特别是当你得到一个有很多功能的应用程序时(包括一个可以在桌面或 u 盘上使用的完全离线模式)。虽然这款应用的正式版本只支持 Windows、Mac 和 Linux，但在 KeePass 网站 上发布了许多针对其他平台和设备的“非官方”版本 [。其中包括一款安卓应用，](https://keepass.info/download.html)[**keepass 2 Android**](https://play.google.com/store/apps/details?id=keepass2android.keepass2android&hl=en_US)，可以与桌面版同步。****

### ****[](https://www.lastpass.com/)****(免费基础版；$3/mo 溢价)********

****尽管有一些反对者，LastPass 仍然是 Lifehacker 读者和互联网上最受欢迎的解决方案之一。LastPass 的可靠性和用户友好的界面是人们称赞这款密码管理应用的最主要原因。正如 Lifehacker 读者**象形图**所说:****

> ****“我使用 Lastpass 管理 700 多个独特的密码已经快十年了。我从未遇到过任何重大问题。”****

****另一位读者， **Abhimanyu Ghoshal，**表示同意:****

> ****“LastPass 可以轻松地从一个设备迁移到另一个设备，并登录我在手机上使用的所有服务。此外，它可以在你喜欢的任何设备上免费使用，我已经没有这些高级功能好多年了。”****

****抱怨主要集中在 LastPass 的定价上(以及对 2015 年安全漏洞 的挥之不去的担忧)，但现在这已经不是问题了，因为你可以使用 LastPass 的免费版本在你的所有设备上同步你的密码——以前只有高级功能。免费用户还可以自动填充密码，访问密码生成器，并存储 2FA 认证的令牌。****

****每月 3 美元的高级版增加了 1GB 的加密文件存储空间，为可信的朋友提供紧急访问功能，以及额外的认证选项(如 [YubiKeys](https://lifehacker.com/secure-your-accounts-and-passwords-with-a-hardware-toke-1830063430) )。****
