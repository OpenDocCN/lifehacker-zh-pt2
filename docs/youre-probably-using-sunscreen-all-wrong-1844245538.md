# 你可能完全用错了防晒霜

> 原文:[https://life hacker . com/you-possible-use-sun-all-错了-1844245538](https://lifehacker.com/youre-probably-using-sunscreen-all-wrong-1844245538)

没有什么比出去晒太阳被严重晒伤更能说明夏天的了，对吗？不对！

你应该涂防晒霜，而且很有可能你一直都没有正确使用它。

只要你的防晒霜是广谱、防水、SPF30+的，你应该没问题。你可以忽略瓶子上所有其他花哨的字眼，比如“抗氧化剂”和“天然 T2”。只要确保你喜欢乳液的感觉和味道。防晒霜的有效期只有三年，所以如果你已经用了很长时间，为了安全起见，买一瓶新的吧。

推荐量是 1.5 盎司(大约一满杯)，这是一个很大的量，但这是针对你的整个身体——包括衣服下面——并且不要错过像眼睑、眉毛、脚和耳朵后面这样的公共区域。如果你的皮肤上有一层很好的涂层，并且还有一些残留在玻璃杯里，你可能就没事了。

喷雾防晒怎么样？为此，你要在你的皮肤上喷一层，确保它有明显的光泽和湿润，然后完全揉进去。仅仅在你的身体上喷一层薄薄的雾提供的保护很少。

无论何时外出，你都应该涂防晒霜。即使你会在一把雨伞下。即使你大部分时间都在车里。即使外面多云。即使你的肤色更黑。每个人在户外都应该涂防晒霜！

你也要在出门前 15 分钟*涂上乳液，这样一旦你暴露在阳光下，它就会被完全吸收。如果你一出门就戴上它，那么在最初的几分钟里，你就把自己暴露在危险之中。别忘了每两个小时重新涂抹一次，尤其是当你出汗、游泳或用毛巾擦干身体的时候。现在去享受你的夏天吧！*
