# 如何阻止 Gmail 图像自动加载到您的 iPhone 上

> 原文:[https://life hacker . com/how-to-stop-Gmail-images-from-loading-automatically-on-1837966865](https://lifehacker.com/how-to-stop-gmail-images-from-loading-automatically-on-1837966865)

任何有手机的人都可以告诉你，有时候你有很好的数据连接，有时候你不确定你是否能够加载谷歌搜索页面。

Watch

当你没有最强的连接或者当你试图保存数据时，在 Gmail 中加载大图片并不理想。现在，Gmail 提供了阻止图像自动加载的功能，可以保存您的数据，并帮助您的电子邮件在连接受限的情况下更有效地运行。本周，CNET 在一篇文章中解释了如何使用 功能。

要激活它，请启动 Gmail 应用程序，然后点击页面左上角的汉堡菜单(这三行堆叠在一起)。从那里选择设置，然后选择您的 Gmail 帐户。前往“图像”，然后选择“显示外部图像前询问”

现在，每当你查看电子邮件时，默认情况下不会加载图像。如果你愿意，你仍然可以选择单独加载它们，但你也可以跳过在手机上看到它们，这应该会使 Gmail 在某些情况下运行得更快。
