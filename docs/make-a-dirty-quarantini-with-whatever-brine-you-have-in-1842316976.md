# 用你冰箱里的盐水做一次“肮脏的隔离”

> 原文：<https://lifehacker.com/make-a-dirty-quarantini-with-whatever-brine-you-have-in-1842316976>

看起来我们中的很多人将会花更多的时间在家里，这可能会导致无聊、厌倦和大量饮酒。Twitter——这是一个非常有趣的地方——似乎对这些潜在的“隔离”感到兴奋，这些隔离有两种形式:“ [一种你在家里独自喝的普通马提尼](https://twitter.com/stonecold2050/status/1238249845770653696) ”和 [这种饮料](https://twitter.com/J4CKMULL/status/1238475320115724289) ，其中含有 Emergen-C



就我而言，我最有可能尝试各种肮脏的马提尼酒，用来自 [的盐水，我冰箱里挂着的任何泡菜和发酵物](https://skillet.lifehacker.com/now-is-the-time-to-buy-your-favorite-condiments-1842296285) 。这部分是因为我喜欢 [一个开胃酒的时刻](https://skillet.lifehacker.com/3-ingredient-happy-hour-the-polarizing-pickletini-1797947090) ，部分是因为我很容易厌倦。我今天从马槟榔盐水开始，因为我一直想尝试马槟榔马提尼，我刚买了一个全新的罐子。

盐水马丁尼是非常个人化的，你加入的盐水或腌制液体的量取决于液体的咸度(以及你喜欢盐的程度)。我通常从四分之一盎司开始，然后添加更多，因为我经常渴望盐水。(我可能电解质失衡了。)至于杜松子酒和苦艾酒，我认为这需要五份干杜松子酒和一份苦艾酒。很醉，但也不像你会开车去任何地方。要做到这一点，你需要:

*   两盎司半杜松子酒
*   半盎司干苦艾酒
*   1/4-1/2 盎司马槟榔盐水(或任何其他盐水或任何其他你想要的 [时髦的泡菜液体](https://skillet.lifehacker.com/turn-that-last-bit-of-marinated-salad-into-a-cocktail-1829278964)

把所有的东西都加入到一个装满冰的混合杯中。搅拌至极冷，滤入小酒杯。重复各种盐水，直到你的无聊感减轻。
