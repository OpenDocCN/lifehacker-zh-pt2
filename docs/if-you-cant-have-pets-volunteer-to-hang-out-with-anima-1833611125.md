# 如果你不能养宠物，可以自愿和动物一起出去玩

> 原文：<https://lifehacker.com/if-you-cant-have-pets-volunteer-to-hang-out-with-anima-1833611125>

也许你住在一个禁止动物的房子里，也许你有一个过敏的伴侣；这并不意味着猫和狗不能成为你日常生活的一部分。在动物收容所做志愿者对渴望宠物的人来说是一个绝佳的机会。



redditor[u/mollylejames](https://www.reddit.com/user/mollyleejames)在 r/LifeProTips 上兴高采烈地宣布，许多 [收容所需要志愿者](https://www.reddit.com/r/LifeProTips/comments/b5fn8x/lpt_its_free_to_go_to_the_humane_society_and_pet/) 帮助猫和狗进行社交，让它们有事可做，受人喜爱:

> 去动物保护协会和宠物猫是免费的——这可以帮助那些不能养宠物和/或长期孤独的人。
> 
> 我经常这样！我所做的就是走进去，问接待员我是否可以进去摸摸小猫。他们总是说是的！
> 
> 我通常会在猫咪房间里坐一个小时，和它们一起玩耍、抚摸、互动、交流。小猫喜欢它，因为他们需要关注，我喜欢它，因为我可以告诉他们有多喜欢它。

> 对我来说，去那里让我一整天都很开心！
> 
> 有时你也会看到他们被挑选出来收养，这总是让我感到温暖。(:

在你当地的收容所，你应该确认一下你是否有能力走进去抚摸动物——首先，时间可能会有所不同，而且在等候区有额外的人可能会让人分心。人道协会还有更多官方志愿者职位，可以按地区搜索。

大多数城市也有许多需要这种支持的小型收容所；例如，布鲁克林的 [BARC 收容所](https://www.facebook.com/pg/BarcShelter/about/?ref=page_internal) 经常寻找穿着“收养我”小背心带狗散步的志愿者。去当地可能需要更多帮助的组织看看。花时间和动物在一起会让它们成为更好的领养对象，这将有助于填补你作为一个无宠物爱好者的空白。只是要小心:如果不把一只新的猫带回家，我不可能度过这样的一天。
