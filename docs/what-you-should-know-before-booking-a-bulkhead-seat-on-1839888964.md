# 在航班上预订舱壁座位之前你应该知道什么

> 原文：<https://lifehacker.com/what-you-should-know-before-booking-a-bulkhead-seat-on-1839888964>

在埃利奥特倡导组织的用户论坛 上，一位评论者写道，他们坐在加拿大航空公司承诺“更大的腿部空间”和其他公园的高级优先座位区，每个座位的额外费用为 100 美元。



他们写道:“当我们在波尔图登机时，我发现 27H 和 27K 座位前面实际上有一堵墙，这意味着实际的腿部空间比它应该有的要小得多。

事实证明，评论者不知不觉地预订了所谓的“舱壁”座位，通常是位于将一个客舱与另一个客舱隔开的墙后面的一排座位。但是，尽管被认为是“首选”座位，它并不总是有额外的腿部空间——或真正的任何其他主要福利——尽管额外的费用。

相反，航空公司可能会向客户提供这种服务，作为首先登机和下机的一种方式，因为你通常会获得某种优先权，或者只是一个更舒适的座位。如果你正在考虑买一个舱壁座椅，这里有一些你绝对应该首先考虑的事情。

首先，如果你想在有额外伸腿空间的高级座位上挥霍，确保在浏览时阅读附属细则。例如，像美国航空公司这样的航空公司在舱壁排提供 [主舱额外的](https://upgradedpoints.com/american-airlines-main-cabin-extra-vs-preferred-seating/) 座位，并保证有额外三到六英寸的腿部空间。

然而，加拿大航空公司的政策措辞微妙；根据其 [网站](https://www.aircanada.com/us/en/aco/home/fly/onboard/preferred-seats.html) 的说法，该航空公司在舱壁座位上提供“额外的膝盖空间”——这是有道理的，因为有人不能斜倚在你面前，但要小心的是*而不是*提到腿部空间。(记者无法立即联系到加拿大航空公司，请其就伸腿空间政策置评。)

如果你有疑问，总是使用 SeatGuru 航空公司的座位地图来寻找你的座位，并寻找出口排后面或周围的排。(例如，联合航空公司的波音 757-200 的 [座位图显示，除了少数例外，大多数舱壁座位确实提供了额外的腿部空间。)如果你发现你的那一排没有额外的腿部空间，试着换一排。](https://www.seatguru.com/airlines/United_Airlines/United_Airlines_Boeing_757-200_Flat.php?flightno=1431&date=)

由于舱壁座位往往位于洗手间附近，如果你不想应对不断有乘客站在你旁边排队的情况，你也可以重新考虑预订一个。(再一次，检查你的座位图以确保正确。)

最后，由于你将位于墙后，你前面的座位下可能没有任何储物空间。如果这是一个问题，你需要你的随身行李随时可用，也许可以考虑一个不同的座位。正如 [Smarter Travel](https://www.smartertravel.com/the-one-seat-you-need-to-avoid-on-a-plane/) 所写的，你的座位也可能比你的普通座位略窄，特别是因为它必须在你的扶手中为你的小桌腾出空间，尽管这可能意味着最多只有一两英寸。

万一有人要求换座位，好吧， [这里有一个](https://lifehacker.com/you-never-have-to-swap-seats-on-a-flight-1838151446) 告诉他们“在你的梦里，伙计。”
