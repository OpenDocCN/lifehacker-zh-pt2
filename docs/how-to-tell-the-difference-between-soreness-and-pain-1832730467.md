# 如何区分酸痛和疼痛

> 原文：<https://lifehacker.com/how-to-tell-the-difference-between-soreness-and-pain-1832730467>

有没有在健身房锻炼后感到特别酸痛的经历？

当你被挥之不去的肩膀酸痛或下背部僵硬所困扰时，重要的是要注意一些警告信号，以确定这是常规的锻炼后疼痛还是更糟糕的事情。



我和布雷克·德克森谈了谈，他是一位在 [进行预约治疗](http://www.bespoketreatments.com/) 的理疗医生，也是 [睡前训练](http://www.nightcaptrainingcrew.com/) 的创始人，他谈到了疼痛的危险信号，为什么休息并不总是一件好事，以及为什么你可能最终需要预约一位已经推迟了几个月的理疗师。在你测试伤势，让自己根本无法去健身房之前，请留意这些迹象。

### 持续超过 72 小时的疼痛

是的，这听起来很明显，但是如果你三天前锻炼过，现在仍然感觉像根面条，那就真的要担心了。根据德克森的说法，如果没有超过 72 小时，你可能会没事——特别是如果你是健身房的新成员或尝试新的电梯。

“如果你第一次做一些新的事情，或者在一个你从未做过的运动范围内工作——比如说你蹲得低一点，或者你很长时间以来第一次(举)过头顶——那么你会感到有点酸痛，”他说。“它被称为 ddelayed onset mus cle sor ess，如果你真的超过了极限，它会持续 24 到 48 小时，有时会持续 72 小时。”

然而，一旦你习惯了一个动作或常规，DOMS 应该会随着时间而减少。他说，如果三天后你发现没有什么改善，你就应该去找理疗师或医生。

肿胀也是不好的征兆。任何肿胀都意味着你的身体正在向你的伤口发送炎症因子，所以要小心！

怎么办:如果你疼痛超过三天，就去看医生。如果做得不够，你可能只是遭受新健身程序带来的痛苦。

### 剧烈而突然的疼痛

假设你在健身房感觉很好，但一个错误的动作突然让你的手腕、下背部或肩膀马上感觉不舒服。德克森说，这是去看医生的另一个好理由，尤其是如果你拉伤了肌肉。

“你仍然可以休息一两天，”他说。“或者，如果它没有使人虚弱，就完成锻炼并修改它，这样就不会再刺激它了。你应该更加谨慎。”

根据 [自我](https://www.self.com/story/5-types-of-workout-pains-you-should-never-ignore) ，局部疼痛也是需要注意的。如果你注意到你身体某处非常特殊的疼痛，而不是肌肉普遍酸痛*，这也可能是谨慎行事的理由。*

***怎么办:**休息一天。如果你没有极度痛苦，你可能会没事。如果无法忍受锻炼，那么尽快去看医生。*

### *辐射出的疼痛*

*有没有经历过“疼痛”,似乎是辐射或悸动？这可能是一个不好的迹象。德克森建议，辐射到手臂或腿部的疼痛被称为“根性疼痛”。*

*“这意味着神经系统也参与其中，”他说。"所以沿着手臂传来的疼痛意味着神经受到了某种压迫."*

*例如，神经受到挤压会导致神经根疼痛。*

*怎么办: 德克森建议缩减你的锻炼(减少重量，减少重复次数，等等。).*

### *不要在健身房休息*

*与我们大多数人被引导去相信的(休息能治愈一切)相反，德克森经常建议呆在体育馆里，尽你所能像平常一样锻炼，只要不会引起疼痛。*

*他说:“问题是，如果你休息一整周不做任何体育活动，那一整周，你的(身体)状况都不好，现在你比开始时更虚弱。”“当你越虚弱，就越容易受到进一步的伤害。所以你已经进入了这种消极的不活跃循环。”*

*德克森说，相反，保持活跃，减少运动，必要时使用冰敷和热敷。当其他方法都失败时，就预约物理治疗吧——安全总比后悔好。*
