# 如果你的孩子吸毒，你该如何应对

> 原文:[https://life hacker . com/how-to-react-if-your-kid-is-use-drugs-1839858392](https://lifehacker.com/how-to-react-if-your-kid-is-using-drugs-1839858392)

当涉及到我们的孩子和毒品时，我们倾向于把重点放在预防上——这是理所当然的。当涉及到物质使用时，我们考虑如何成为好的榜样；我们很早就告诉他们，并且经常告诉他们毒品、同辈人的压力以及与不良人群交往的危险。但是，当我们做了所有这些，他们仍然使用？

Watch

即使他们掌握了所有正确的信息——即使他们知道得更多——青少年有时也会做出错误的选择。而当父母发现时，他们很可能会感到震惊、害怕、愤怒，或者以上所有。心理学家和教育家艾玛·梅纳德为《对话》[写道，这让我们在最需要的时候失去了冷静:](https://theconversation.com/how-to-talk-to-your-teenagers-about-drugs-124135)

> 青少年吸毒专家告诉我们，这是明智的选择。他们建议我们接受这样的事实:作为父母，我们不可能阻止我们十几岁的孩子做他们选择的事情，因此，我们最好的方法是确保他们获得正确的信息，并且他们可以公开地与我们讨论问题。通过这种方式，我们可以确保青少年意识到风险，以及在他们需要帮助时该怎么做，从而帮助减少伤害。
> 
> 虽然这确实是很好的建议，但许多父母很难遵循。我正在进行的研究着眼于孩子吸毒的父母的经历。他们重视从业者与青少年交谈的方式，并理解建议减少伤害方法的价值。
> 
> 尽管如此，我采访过的大多数父母都说，他们的本能反应是做出不同的反应:零容忍多于减少伤害。他们倾向于让孩子接地，并停止他们的零花钱。在恐慌和反叛的无休止循环中，关于争吵和不断升级的制裁的报道比比皆是。

换句话说，当你认为你的孩子可能在吸毒时，保持冷静和理性说起来容易做起来难。我们想把他们锁起来以保证他们的安全。但是保护你和他们的关系应该是重中之重，这样你才能以最有建设性的方式提供帮助。

当然，你需要 [让合适的专业人士](https://drugfree.org/article/get-one-on-one-help/) 在你身边帮助你驾驭这一切。但是，一旦你发现药物的使用，你就可以在与孩子的互动中开始做一些事情，以帮助保持牢固的联系和开放的交流。

### 不要马上做出反应

如果你知道(或怀疑)你的孩子在吸毒，首先要做的是深呼吸。然后又是一个。如果你有一个伙伴，在你接近你的孩子之前和他们好好谈谈。无毒品儿童伙伴组织强调父母达成一致并呈现统一战线的重要性，即使你不完全同意你将采取的立场。

收集你能收集到的任何证据(这里有一个 [常见藏身之处的列表](https://drugfree.org/article/prepare-to-take-action/?gclid=EAIaIQobChMIv6KGqZ7q5QIVlpOzCh0hpwJTEAAYASAAEgIG_fD_BwE) )，设定你的 [目标和目的](https://drugfree.org/article/set-limits-monitor/) ，计划好你的 [初始对话](https://drugfree.org/article/start-talking/) ，并为你自己可能出现的非常负面的反应做好准备。

### 冷静的时候说话

梅纳德说，期望自己一直保持冷静是你不需要的额外压力。“但是选择什么时候说话会有帮助，”她写道。“我交谈过的父母都说了同样的话:心平气和地谈，他们也心平气和。那你就好好说话，好好听着。”

如果事情开始升级并变得激烈，按下对话中的“暂停”,等所有人都冷静下来后再继续。试着总是来自爱的地方。或者，正如成瘾和物质滥用中心的研究员莫莉·博贝克所说，来自一个“非判断性好奇心”的地方，它强调在保持开放心态的同时提出问题:

> 当你担心药物使用时，与青少年的关系是最重要的事情，不能忽视。如果没有一个强大的、相互关爱的家庭关系，就不可能长期有效地预防或停止青少年使用药物。

### 认真听

如果有一个时候要少说多听，那就是现在，不管用充满信息的讲座轰炸你的孩子有多么诱人。为了找出吸毒的潜在原因——并改变模式——你必须倾听“为什么”他们可能屈服于社会压力，寻求突破界限或渴望身体上的感觉。

一旦你确定了使用背后的动机，你就可以开始寻找解决方案或替代方案。

### 努力建立你的联系

当像吸毒这样巨大而可怕的事情进入你的生活时，你很难去想或谈论其他事情。但这不是一个一夜之间就能解决的问题，梅纳德说 现在休息一下，然后重新联系是可以的——甚至受到鼓励:

> 玩得开心。如果这意味着暂时回避这个话题，那就做吧。做一些不同的轻松的事情。谈论除毒品和任何副作用之外的事情，比如不良行为或学校问题。一起找乐子是我们能做的增强弹性的最好的事情之一，尤其是当关系变得紧张的时候。这也是我们首先忽略的事情之一。

博贝克还建议父母通过问自己这个问题来考虑他们生活中什么是有效的或顺利的,“如果我们不在这里谈论小的物质使用，我们会谈论什么呢？"
