画中画(PIP)视频播放是 Android 的 Chrome 应用程序上用户最喜欢的功能，现在你可以在你的 Chrome 桌面浏览器上获得它——只要你已经更新到 [Chrome 70](https://lifehacker.com/how-to-disable-chromes-automatic-sign-ins-in-version-70-1829819727) ，也就是说。

### 如何在 Chrome 70 中观看视频 PIP

你可以通过打开 Chrome 浏览器窗口来检查以确保你运行的是最新版本的 Chrome。如果有更新，您会在右上角看到一个图标，绿色表示更新不到两天，橙色表示更新已有四天，红色表示更新已有七天或更长时间。如果你有可用的更新，点击图标，然后点击**更新谷歌浏览器**运行更新。完成后，重新启动 Chrome。您现在可以在 PIP 中观看视频了。

**Windows、Mac 和 Linux 中的 Chrome**

如果您使用的是 Windows 或 Mac，您可以通过右键单击视频并单击“画中画”来启用画中画如果你在 YouTube 上观看，你需要双击鼠标右键。一些开发者可能还会提供按钮来启用 PIP， [，如这里看到的](https://googlechrome.github.io/samples/picture-in-picture/) ，一些视频甚至可能会在 PIP 中自动启动 [。(有些视频播放器不支持此功能，但您可以尝试使用](https://developers.google.com/web/updates/2018/10/watch-video-using-picture-in-picture) [画中画扩展](https://chrome.google.com/webstore/detail/picture-in-picture/hkgfoiooedgoejojocmhlaklaeopbecg) 来解决这个问题。)

您可以点按并拖移 PIP 窗口的角或边来调整其大小。播放器将保持在你打开的任何 Chrome 窗口的顶部，视频来源的标签将有一个蓝色轮廓的星号图标突出显示在标签标题的左侧，这使得你很容易找到你是否在多个标签之间跳跃。

但是，如果你想继续观看，不要关闭标签或离开视频网站。如果该标签被关闭或域被更改，您将切断播放器与视频的连接，它将关闭。

**Chrome OS**

虽然在 Windows、Mac 和 Linux 上 Chrome 70 的 PIP 是默认打开的，但在 Chrome OS 上使用要稍微复杂一些。现在，你需要在你的 Chrome 地址栏中输入或复制/粘贴以下内容:`chrome://flags#enable-surfaces-for-videos`

然后，将功能“允许使用视频的表面层对象”设置为“已启用”你可能需要重启 Chrome，但是你可以右键点击视频(双击 YouTube)，然后点击“画中画”在弹出播放器中打开视频。
