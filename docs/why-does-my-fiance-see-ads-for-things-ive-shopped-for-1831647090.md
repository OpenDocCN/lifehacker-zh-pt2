# 为什么我的未婚妻会看到我购物的广告？

> 原文：<https://lifehacker.com/why-does-my-fiance-see-ads-for-things-ive-shopped-for-1831647090>

在 Lifehacker 本周的 [技术建议专栏](https://lifehacker.com/c/tech-911) 中，我正在解决互联网最大的烦恼。不是尖叫 YouTube 的影响者或在*堡垒之夜*比你强的人——我说的是定向广告。



一位 Lifehacker 的读者想知道他所做的每一件事是否都被跟踪并被广告回给他——或他所爱的人。我会让约翰解释:

> 最近，我注意到我的未婚夫在她的手机上收到了基于我的浏览习惯的定向广告。例如，几个月前，当我在笔记本电脑上看结婚戒指时，她开始在她的脸书和 Instagram 上看到我访问的网站的广告。最近我一直在研究汽车，因为我很快就需要一辆新车，毫无疑问，她现在已经开始在她的社交媒体上看到汽车广告。
> 
> 几年前，我删除了我的脸书和 Instagram 账户，并试图关闭所有形式的谷歌定向广告。这感觉像是对我隐私的大侵犯，真的让我很困扰。此外，我担心在某个时候，我可能会为她买礼物，而脸书会开始展示她的广告，破坏这个惊喜。有什么方法可以让我的浏览数据保密？如果我要获得定向广告，至少我希望它们指向我的笔记本电脑和手机，而不是使用我们 wifi 路由器的每一个设备。我曾经考虑过开始永久的匿名浏览，但这似乎是一个极端的解决方案。

这是一个有趣的问题。这个问题对你来说并不有趣，John，对任何一个不得不处理无时不在的网络广告和追踪问题的人来说也是如此。

首先，回答你的问题，我要从显而易见的评论开始。如果你用你未婚妻的手机或电脑浏览过任何东西，那么有可能会出现在她的网络广告中。我觉得这并不需要太多。我经常注意到，当我在 Lifehacker 上时，我前一天看的一个网站或一系列产品突然出现在所有该死的地方。(而且我经常上 Lifehacker。)

假设你已经将你的设备彼此分开，也有可能各种广告追踪器(或你使用的服务)正在记录你的位置和/或 IP 地址。例如，任何其他使用相同外部 IP 地址的设备(例如，像你路由器的 wifi 连接上的所有东西) [都将看起来来自同一个家庭](https://www.reddit.com/r/advertising/comments/6sm8em/how_come_i_see_my_boyfriends_ebay_searches_on_my/dle1yd2/) ，而正是这个家庭 [将成为广告商的目标](https://www.facebook.com/business/news/first-look-holiday-trends-and-tools-to-drive-discovery-)——在错误的假设下，你拥有一堆设备，而不是一群使用不同设备的人。

你可能不太关心*为什么*这种跟踪会发生在你身上，而更关心你如何能够阻止它。首先，我会加入一些浏览器扩展，当你浏览网页时，它们会自动禁用大多数广告跟踪(通过烦人的 cookies)。 [UBlock 起源](https://github.com/gorhill/uBlock) 浮现脑海， [隐私獾](https://www.eff.org/privacybadger) 。使用这两个，你应该会得到很好的保护。

第二，你可以尝试打开你的浏览器可能有的任何“”设置。网站和广告商可以不服从这个，但这并无伤大雅。你也可以访问 [数字广告联盟](http://optout.aboutads.info/?c=2&lang=EN) 和 [网络广告倡议](http://optout.networkadvertising.org/?c=1#!%2F) ，选择退出一堆追踪器。同样，我不确定这将有多全面——我不能保证你浏览的东西永远不会意外地被广告给你家里的人——但每一点都有帮助。

如果你决心不让你的秘密泄露给别人，你可以试着用匿名模式浏览，就像你注意到的那样。或者你可以使用一个单独的浏览器，关闭所有 cookies(或者每次关闭浏览器时删除它们)，或者一个更加以隐私为中心的浏览器，如 [火狐焦点](https://www.mozilla.org/en-US/firefox/mobile/) 或 [Epic 浏览器](https://www.epicbrowser.com/) 。你也可以尝试进入各种服务领域——谷歌[脸书](https://www.facebook.com/ads/preferences/) 等等。—关闭个性化、基于兴趣或基于位置的广告的任何设置。别忘了重置你的广告 ID——这个选项隐藏在安卓谷歌设置菜单的“广告”部分，或者 iOS 设置应用的隐私部分。

在你未婚妻的设备上也执行同样的步骤——当然，要征得她的同意。或者你可以让她读这篇文章。你们两个一起，有望尽最大努力控制广告和跟踪，但正如你所看到的，要想控制这些烦恼，你还有很多事情要做。
