# 如果你今年不能使用你的航空公司精英会员身份，你可以暂时搁置

> 原文：<https://lifehacker.com/you-might-be-able-to-put-your-airline-elite-status-on-h-1841084329>

精英航空公司的地位可以是伟大的。拥有航空公司的地位通常会提供额外的好处，比如升级座位、免费托运行李和更早登机。对于一年前在一家航空公司坐过很多次飞机的人来说，这是一个巨大的福利。但是，如果你今年飞了很多次，但是像家里有了新生儿这样的事情会阻止你在接下来的一年里到达友好的天空呢？事实证明，一些航空公司会让你暂时保留精英会员的身份。



积分专家 [最近贴出了一份指南，指导你如何暂停](https://thepointsguy.com/guide/airlines-elite-status-hold-maternity-paternity-leave/) 的状态。

不是每个航空公司都提供这个选项，但是很多都提供。你也不能因为任何原因而搁置你的精英身份，你需要一个相当大的身份。

正如 Points Guy 解释的那样，生孩子、领养孩子或患重病等情况往往符合条件。也就是说，如果你只是预计在接下来的一年里没有时间去旅行，那么你很可能是 SOL。

至于航空公司提供什么样的选择，这因航空公司而异，许多航空公司没有正式提供政策，但会根据具体情况提供精英扩展服务。在积分佬的帖子 里可以看到各大航空公司的政策是什么 [。](https://thepointsguy.com/guide/airlines-elite-status-hold-maternity-paternity-leave/) 

在某些情况下，您可以很容易地将该状态推迟一年，而其他情况下，您可能需要满足某些“挑战”要求，才能再次获得该状态并运行。

如果你计划要求航空公司推迟你的精英会员身份，确保你也准备好文件来支持你为什么需要它。这可能是你的医生给你的，或者是你女儿的出生证明。
