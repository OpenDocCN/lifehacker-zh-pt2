# 玩这个免费游戏可能有助于找到冠状病毒疫苗

想帮助研究人员了解并找到冠状病毒的疫苗吗？通过玩 [这个视频游戏](https://fold.it/portal/node/2008926) 来帮助他们。

Foldit 是一款免费的益智游戏，让玩家通过折叠蛋白质链来改变他们的行为方式。它最初是由华盛顿大学的开发人员在 2008 年创建的，旨在众包阿尔茨海默氏症和艾滋病毒/艾滋病等疾病的潜在疗法，但本周增加了一个新的冠状病毒谜题。玩家需要创造或修改蛋白质，这些蛋白质可以与冠状病毒的一种“刺突蛋白”结合，从而防止它感染人类细胞和复制。

尽管 Foldit 的起源更侧重于技术和研究，但它玩起来相当简单——你甚至可以忽略所有的医学和科学术语，只需专注于移动结构和赚取积分。你所要做的就是点击蛋白质的一部分并拖动它来改变它的形状，尝试并匹配拼图的目标。Foldit 根据效率和效果给玩家打分，并通过游戏中的记分牌跟踪他们的结果。

华盛顿大学表示，他们将测试出解决冠状病毒难题的最有效方法，作为可能用于新冠肺炎的真实世界疫苗。希望渺茫，但也不是前无古人。PCGamer 指出， [Foldit 玩家在 2011 年](https://www.pcgamer.com/this-game-might-help-create-a-vaccine-for-coronavirus/) 帮助研究人员解码了艾滋病病毒——这是困扰科学家多年的事情。
