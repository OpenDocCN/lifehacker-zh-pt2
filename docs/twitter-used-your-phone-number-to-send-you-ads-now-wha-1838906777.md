# Twitter 用你的电话号码给你发广告。现在怎么办？

> 原文：<https://lifehacker.com/twitter-used-your-phone-number-to-send-you-ads-now-wha-1838906777>

谈谈被夹在推特和困境之间。拥有无限智慧的 Twitter 最近承认，出于广告目的，它篡改并使用了用户提供的电子邮件地址和电话号码——作为其账户安全措施的一部分。



正如推特 [描述的](https://help.twitter.com/en/information-and-ads#10-08-2019) :

> 定制受众是行业标准产品的一个版本，它允许广告商根据自己的营销列表(例如，他们编辑的电子邮件地址或电话号码)向客户投放广告。合作伙伴受众允许广告商使用相同的定制受众功能将广告定向到第三方合作伙伴提供的受众。当广告商上传他们的营销列表时，我们可能会根据 Twitter 账户持有人出于安全目的提供的电子邮件或电话号码，将 Twitter 上的人与他们的列表进行匹配。这是一个错误，我们道歉。

如果你对此感到恼火，你可以向 Twitter 提供一个不同于你经常使用的电子邮件地址——在 Gmail 中使用类似 ol' [的加号或句号](https://lifehacker.com/make-a-joint-email-address-with-a-gmail-filter-1836702340)——或者一个不同的电话号码，比如谷歌语音号码。你仍然可以利用像 [登录验证](https://twitter.com/settings/account/login_verification) 这样的功能，我建议每个人都打开它，如果你从你的备用号码转发你的文本或者有其他方法接收它们的话。(你也可以一直使用“移动安全应用”选项。)

为你的浏览器安装一个伟大的广告拦截扩展，或者在你的手机应用程序上综合利用 [Twitter 列表](https://lifehacker.com/private-lists-are-the-best-way-to-stay-organized-on-twi-1838834186) ，也可以帮助你逃避一些与服务的广告相关的烦恼。如果你想减少任何基于你的身份而不是你的原始 Twitter 活动来针对你的企图，你也可以编辑 Twitter 的 [个性化和数据](https://twitter.com/settings/account/personalization) 部分中的一些设置，并在你的设备 上禁用各种广告跟踪功能 [。](https://help.twitter.com/en/safety-and-security/privacy-controls-for-tailored-ads)
