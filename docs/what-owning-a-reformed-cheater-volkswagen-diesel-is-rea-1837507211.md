# 拥有一辆改造过的作弊大众柴油到底是什么样的

"你什么时候会再有新闻车？"我妈问。我低头看着我的右脚，在脚踝折断两根骨头一个月后，我的右脚仍然穿着支撑靴。她并不关心，或者只是漫不经心地询问我的驾驶时间表。在我受伤前几个月，当我骑自行车通过新闻汽车贷款时，我让她借了我的 [2015 大众 TDI SportWagen](https://jalopnik.com/i-bought-a-volkswagen-diesel-that-cheated-and-maybe-you-1795920529) ，她想要回来。

“可能暂时不会，考虑到我不会开车，”我告诉她。

“我只是想让它在我的吉普车上节省里程！”她赶紧解释。

好的，妈妈。她有一个严重的品牌忠诚度案例。她为克莱斯勒的一家供应商工作了多年，我们家族中有几个直接的克莱斯勒员工。她拥有一辆克莱斯勒 300，一辆克莱斯勒 200，现在正在租赁一辆蓝色吉普指南者——都是朋友和家人的折扣，就像你在密歇根做的那样——尽管她一直对这个品牌失望。

但是我的小柴油大众开始让她。它很快，轰鸣的引擎和惊人的里程数向她展示了真正享受驾驶的感觉。甚至有传言说要试驾斯巴鲁。最后，她终于明白了。

她不是唯一一个。现在，这些不诚实的柴油车型 [正成为备受追捧的商品](https://jalopnik.com/cheating-volkswagen-diesels-have-become-a-hot-commodity-1837505697) ，我自己的大众，昵称为 Jolene，本周行驶了 30，000 英里，我想我应该向大家更新一下我的 TDI 之旅的进展情况。

大部分时间都是阳光和彩虹。大部分是。柴油和欧洲车，如果你想买其中一辆，有些事情要记住。

### 零件

没多久，看起来没问题的轮胎就在我身上爆炸了。在我的车主之旅进行了大约六个月后，我的备用轮胎做了很多工作。

看，我的车停在一个经销商的停车场上，挡风玻璃上贴着一个大而亮的非卖品标签已经两年了 [等待它的修理](https://jalopnik.com/heres-the-first-approved-fix-for-some-of-the-cheating-d-1790930479) 以满足排放要求。我在停止销售结束两天后买了我的，但仍然需要坐很长时间。

虽然随车配备的全天候轮胎看起来没有任何腐烂，但在所有这些不作为期间，它们肯定会变弱。这是一个你可能会遇到的问题，尽管经销商似乎正在给回购的固定认证二手车安装新轮胎和刹车。当然，这对你来说是新的，但是要确保经销商给你换上新轮胎，否则很快就会换掉它们。

在坐了这么长时间后，换档拉索和刹车线也可能需要更换。我买了新的冬季轮胎和夏季轮胎，它们的标价都是 1100 美元。

## 维护

这辆大众不仅是我的第一辆柴油车，也是我的第一辆非美车。我从理智上理解欧洲车的维护费用会更高，但是柴油车的维护费用会更高。

首先，那些便宜的换油点通常没有适合我库存柴油的滤油器，至少在我所在的地区没有。如果我想打一个叔叔埃德的，我必须先打电话，看看我附近是否有一个部分，这是一个痛苦的屁股，否定了整个点快速午休换油。(当然，你可以自己动手，但是时间就是金钱。)

对于更复杂的 30，000 英里维护，我把它带回经销商处，以确保一切都做得正确。这些汽车以其长寿而闻名，但任何汽车的生命周期都将依赖于定期维护。冲洗并加满所有的液体，旋转我的轮胎，清洗喷油器(这是保持传说中的最高效率柴油里程的要求)和内部半体面的细节花了我整整 850 美元。完成基本工作比我预期的要多一点。

## 燃料

是的，由于直喷式柴油发动机的出色表现，你的柴油大众汽车最终会经过一个又一个加油站。我的汽车每加仑能跑 41 至 43 英里。但是一旦你需要加油，它偶尔会变成一个恐慌的寻找加油站。

我发现在我的家乡密歇根州，大约 60%的加油站都有柴油，高速公路边的加油站比市区的加油站多。但是柴油泵似乎也是最有可能发生故障的，可能是因为没有真正的燃料，所以泵可能会在一段时间内出现故障。

此外，在加油站停车场的谈判可能会变得令人担忧，因为你在等待一个柴油泵，而其他人都在使用每个备用的汽油泵，并想知道为什么你只是坐在那里。加油站的服务员会奇怪地看着你，仔细检查你的订单，甚至会从柜台后面冲出来阻止你给你的大众汽车加满可能会带来灾难的燃料。我经常遇到这种情况。

我遇到的另一个加油问题是柴油泵的喷嘴实际上不适合我的油箱颈部。尤其是老式泵(柴油更可能来自老式泵——同样，没有足够的需求来花钱买新的好泵)有时会在喷嘴周围有一个环，这使得它们仍然大到足以装在卡车上，但太大而不能装在客车上。这样做也是为了防止一些不知情的汽油车车主用砖头砸他们的车。

我非常清楚最终找到一个柴油泵，付了油钱，发现油嘴连油箱都装不进去时的心碎。如今，当离开我家的加油站时，我会在油箱快满四分之一的时候加油，以避免完全的恐慌和绝望的搜索。

柴油本身并不便宜。根据美国能源信息署的数据，目前，撰写本文时汽油的平均价格为每加仑 2.68 美元，而柴油为 3.03 美元。所以你加油少了，但当你这样做的时候，这是一个现实的检查。

### 好东西

一旦你花光了你为这辆车花的所有钱，它就是一台出色的机器，我还在努力省钱。柴油不便宜，但米其林是如此之好，你几乎注意不到差异。

大众汽车销售这些汽车的年利率为零的融资和令人敬畏的四年/48，000 英里保修，其中包括一些最昂贵的零件维修或更换。

驾驶绝对是一种享受，一辆车如此有用又如此有趣是非常罕见的。

当我最终需要在离开六个月后从我妈妈那里拿回我的大众汽车时，我所能想到的就是“啊，是的，这是一个正确的决定”，并给过去的自己一点鼓励。两年多过去了，30，000 英里过去了，我仍然对这辆车激动不已。它已经经历了无数次露营和背包旅行，并经常轻松地拖走两个完全长大的坑混合物。

在柴油发动机问世之前的几个月里，80%的大众跑车都是用柴油发动机销售的。丑闻发生后，当 TDIs 无人问津地坐在停车场等待修理时，许多车主错过了他们的车。原因很明显:它们舒适、安全、可靠、高效，是路上的最佳选择。

如果你能设法抢到一辆——大众经销商告诉《纽约时报》这一批最新的二手车卖得很快——并且你愿意处理一些小的不便，那就去做吧。睁大你的双眼，看看怎样才能留在方向盘后面。
