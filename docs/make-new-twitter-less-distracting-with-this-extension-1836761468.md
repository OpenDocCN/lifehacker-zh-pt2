# 使用这个扩展使“新推特”不那么令人分心

> 原文：<https://lifehacker.com/make-new-twitter-less-distracting-with-this-extension-1836761468>

如果你发现“新推特”令人分心，现在有一个扩展可以帮助你——简化的推特。

Watch

就像它听起来的那样，[Chrome](https://chrome.google.com/webstore/detail/simplified-twitter/kfopmjhmejbgomgeajemgpgpbckpoopg?ref=producthunt)扩展和 [Firefox 插件](https://addons.mozilla.org/en-US/firefox/addon/simplified-twitter/?ref=producthunt) 采用 Twitter 的传统桌面界面并将其简化，使其更容易专注于实际的推文，而不是其他任何事情。

本质上，它将大量的侧边栏菜单缩减为更小的图标，并在右侧抛弃了大趋势和“关注谁”菜单。

它使用起来超级简单，而且像所有的扩展一样，可以根据你的需要或需要来切换打开或关闭。

作为一个发现很容易分心的人(老实说，当我来到 Twitter 的时候，我可能首先会分心，我认为这是一个很好的、简单的方式来降低噪音，这样你就可以专注于推文。

如果你想让 Twitter 保持变化前的样子，也有一个扩展。本周早些时候 [我们写了关于好 Twitter](https://lifehacker.com/how-to-get-old-twitter-back-1836704742) 的文章，这将从本质上让这个网站回到它最初的辉煌。
