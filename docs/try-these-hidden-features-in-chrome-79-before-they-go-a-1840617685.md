[随着 Android Police](https://www.androidpolice.com/2019/12/20/chrome-beta-80-removes-experimental-reader-mode-and-gesture-navigation-deprecates-ftp-support-and-more/)的报道，Chrome 80 beta 测试人员已经注意到，Chrome 79 中仍然存在的几个功能已经丢失，包括几个期待已久的功能，这些功能将在 Chrome 80 上市后被放弃。

下面列出了 Chrome 79 中仍然存在但在 80 版本中将被取消的即将被删除的功能。大多数都可以在 Chrome 中启用，方法是进入 chrome://flags/ 并使用搜索栏找到下面列出的标志，然后将下拉框设置为**“启用”**

### 阅读器模式

尽管 [已经对该功能进行了几个月的测试](https://lifehacker.com/how-to-install-google-chrome-s-experimental-reader-mode-1834171339) ，谷歌甚至会在推出之前放弃其计划中的阅读器模式。

你仍然可以在 Chrome 79 中通过将 **#reader-mode-heuristics** 和**# enable-reader-mode-in-CCT**标志设置为“enabled”来启用它，但它将在 Chrome 80 中消失。

### 隐藏建议的文章

如果你不想让你的新标签窗口塞满浏览建议，我有一些坏消息:通过禁用**# enable-NTP-remote-suggestions**标志，将不再可能隐藏新 Chrome 标签窗口的建议文章。然而，你仍然可以通过点击空白标签页的“为你写的文章”部分旁边的三个点“更多”图标并点击“隐藏”来永久隐藏它们

### 过度滚动和水平标签切换

在 Chrome 80 测试版中，两个导航选项已经从 Chrome Mobile 中删除，预计一旦公开发布，这两个选项也将消失。第一个是 overscroll，这是一个基于手势的控件，通过向左或向右滑动，你可以在标签内前后移动。另一个是水平选项卡切换器，它将打开的选项卡托盘中的选项卡布局从默认的垂直方向更改为水平方向。

如果你从未尝试过这些，你可以在 Chrome 79 的 Chrome 标志设置中启用其中一个功能。搜索并启用**# overscroll-history-navigation flag**进行 over scroll，或**# enable-horizontal-tab-switcher**在水平标签切换器还在的时候使用它。

### FTP 支持

虽然许多文件传输协议(FTP)工具仍然存在，但谷歌表示，Chrome 用户对 FTP 服务的使用已经下降到不再值得支持该功能的程度。从 Chrome 80 开始，Chrome 的 FTP 支持将在接下来的几个版本中慢慢淘汰。你仍然可以通过打开 **#enable-ftp** 标志在 Chrome 80 中重新启用它，但随着 Chrome 82 的推出，这将被完全删除，所以尽可能地进行最后的传输。

如果你想看看去掉这些功能的 Chrome 会是什么样子——以及即将推出的一些新功能——你可以在这里 下载 [Chrome 80 beta。](https://www.google.com/chrome/beta/)
