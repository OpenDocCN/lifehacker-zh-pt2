# 在航班上靠近出口排的位置订票前，先检查一下你的座位

> 原文：<https://lifehacker.com/check-your-seat-before-booking-near-an-exit-row-on-a-fl-1833665018>

有什么比没有腿部空间的中间座位更糟糕的？发现你的座位不能倾斜——你被困在一次跨国飞行中，当你坐直时，紧张地拉紧两个扶手。

Watch

如果你的背部有问题，或者只是讨厌坐飞机，一位用户在最近的 [Reddit 帖子](https://www.reddit.com/r/LifeProTips/comments/b6kk55/lpt_when_choosing_a_seat_on_an_airplane_do_not/) 中提出了一个有趣的建议，让你的旅行体验不那么令人沮丧:不要预订前排的座位。

为什么？根据 [u/snoof123](https://www.reddit.com/user/snoof123) 判断，是因为您的座椅很可能不会放倒。在很大程度上，这是一个非常可靠的建议，尽管它有一个基本的警告——如果你已经预订了一个座位，你应该总是仔细检查你是否坐在靠近紧急出口的座位，因为你可能会发现出口后面的座位也会遭受同样的命运。

根据[*Smarter Travel*](https://www.smartertravel.com/the-one-seat-you-need-to-avoid-on-a-plane/)的说法，这样做是为了在紧急情况下出口排保持畅通，并防止斜倚座椅挡道。

使用 [SeatGuru](https://www.seatguru.com) ，你可能会发现自己乘坐的 [【西南】](https://seatguru.com/airlines/Southwest_Airlines/Southwest_Airlines_Boeing_737-700_new.php?flightno=1916&date=)[【联合】](https://www.seatguru.com/airlines/United_Airlines/United_Airlines_Boeing_767-300_V3.php)[阿拉斯加航空](https://www.seatguru.com/airlines/Alaska_Airlines/Alaska_Airlines_Airbus_A320_V1.php) [s](https://www.seatguru.com/airlines/Alaska_Airlines/Alaska_Airlines_Airbus_A320_V1.php) 航班上的座位不会因为出口排而倾斜。在一些 [捷蓝航空](https://www.seatguru.com/airlines/JetBlue_Airways/JetBlue_Airways_Airbus_A320.php) 航班上，两个出口排在一起，你甚至会发现出口排座位本身并不倾斜(所以不要被额外腿部空间的诱惑所迷惑)。

当然，这取决于你乘坐的飞机和航空公司本身。出于这个原因，你最好在 [SeatGuru](https://www.seatguru.com) 上搜索你的航班，并查找你的特定座位，这样你就不会再被困在航班的最差排了。
