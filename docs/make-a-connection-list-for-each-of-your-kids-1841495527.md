# 为你的每个孩子列一份“联系清单”

> 原文：<https://lifehacker.com/make-a-connection-list-for-each-of-your-kids-1841495527>

父母一天要做的事情很多。有地方要去，有饭要准备，有工作要完成，有盘子要洗。在这一切当中，似乎不可能优先考虑与孩子的实际联系和联系时间。我们想和他们联系，我们当然想！但是我们还得做意大利面，在去练曲棍球之前把饭端上桌。



瑞秋·加林豪斯为《恐怖妈咪》 写道，她正在为她的四个孩子中的每一个孩子列一份她称之为“联系清单”的东西来解决这个难题:

> 我有个主意。如果我列一个清单，列出我可以花时间和我的每个孩子在一起的具体有效的方式，他们喜欢和回应的方式，会怎么样？与其总是试图根据需要创造肤浅的时刻，不如我积极主动，有所准备？会有用吗？
> 
> 我拿出一个笔记本，在上面写下每个孩子的名字，然后开始填写他们喜欢和父母一起做的事情。

我喜欢这样，因为通常，当我们在一天中最忙碌的时候，我们可以看到我们的孩子渴望与我们建立一点联系。但是，当你已经忙于多项任务时，想办法让它发生，只是你的大脑没有时间做的又一件事。但是如果你的冰箱上已经挂了一张事先准备好的想法清单，你可以浏览一下，然后想，“啊，我会让她上楼去拿一本最喜欢的板书，一起读，同时我会等水烧开。”

这个清单可以——或许应该——包括一系列的 [、瞬间灵感](https://lifehacker.com/quick-ways-to-connect-with-your-kids-when-youre-busy-1838048560) 和更大承诺的想法，当你有一个意想不到的时间窗口或一个特殊的一对一“约会”时，你可以留着这些想法

快速联系的想法可能包括阅读一本书，在 Spotify 上播放他们最喜欢的歌曲，举办一个四分钟的舞会，玩一轮 Go Fish，或者独自带他们去遛狗。

手头上更大的承诺活动想法可能包括一起烘焙他们最喜欢的纸杯蛋糕，玩更深入的棋盘游戏，处理艺术项目或在公园里骑自行车。

根据每个孩子最喜欢的东西来制作清单。如果你需要更多的灵感，就去问他们。可能有些事情他们喜欢和你一起做，而你甚至没有意识到这些事情对他们来说是特别的。
