# 你应该考虑一个高收益回报的支票账户吗？

> 原文：<https://lifehacker.com/should-you-consider-a-high-yield-rewards-checking-accou-1844915211>

随着美联储 [保持低利率](https://twocents.lifehacker.com/how-this-week-s-fed-meetings-may-impact-your-wallet-1844561078) ，许多储户都 [感受到他们储蓄账户](https://twocents.lifehacker.com/where-to-put-your-money-when-interest-rates-are-falling-1844380477) 里的刺痛。 [美国消费者新闻与商业频道报道](https://www.cnbc.com/2020/08/05/high-yield-savings-accounts-slash-interest-rates-to-all-time-lows.html) 高收益储蓄账户已经将利率削减至创纪录低点——这让储户们开始寻找替代方案。一个已经引起注意的选择是:高 y 奖励检查账户。以下是如何知道这些账户中是否有一个适合你。

### **奖励支票账户的好处**

“这是目前唯一一个利率超过 1%的银行账户，”存款账户 的创始人肯·杜明说。根据存款账户的 [奖励支票页面](https://www.depositaccounts.com/ratetable/checking/reward-checking-accounts.html?adobe_mc_sdid=SDID%3D682FB81304DE1C5D-1F88178E2232470A%7CMCORGID%3D2E96401053DB518E0A490D4C%40AdobeOrg%7CTS%3D1598969148) ，您可以找到目前提供 2%以上利率奖励支票账户的银行。

Tumin 表示，大多数奖励支票账户都是免费的，没有每月维护费。另一个额外的好处是，大多数支票账户会报销你的自动取款机费用。

### **奖励支票账户的弊端**

虽然回报支票账户的利率很难被打破，但你必须经历一些额外的考验才能获得更高的收益(T2)。

“如果你不喜欢使用借记卡，奖励可能不值得，”Tumin 说，并指出大多数奖励支票账户要求你每月使用借记卡的最低次数。对于那些喜欢用现金或信用卡进行日常消费的人来说，这可能是一个不利因素。

另一个潜在的缺点是:许多奖励支票账户要求你使用网上银行进行月结单和其他交易。如果你更喜欢纸质报表——或者不喜欢网上银行——奖励支票账户可能不太合适。

最后，奖励支票账户可能对那些有大量余额的人不起作用。Tumin 说，这些账户通常会将你的余额限制在 10，000 到 50，000 美元之间。他说:“如果你在储蓄账户或定期存款账户中持有数十万英镑，奖励支票将不是一个好的选择。”

### **报名前阅读细则**

杜明说，小字中有一些东西值得期待。虽然大多数奖励支票账户都是免费的，但也有一些银行会收取月维护费。“你也许可以通过满足活动要求来避免费用，”他说。

此外，许多银行和信用合作社希望你使用他们的奖励支票作为你的主要账户。他警告说:“如果他们看到你只进行小额购物就满足了借记卡的使用要求，他们可能会威胁关闭你的账户。”

其他鬼鬼祟祟的举动，寻找的是平衡资本减少和利率降低。“一个奖励支票账户的余额上限最近从 35，000 美元降至 10，000 美元，”杜明警告说。

### **奖励支票账户的替代方案**

Tumin 说，如果你不喜欢借记卡使用要求、网上银行或较低的余额上限，你可能会更喜欢提供免费支票和高收益网上储蓄账户的网上银行。“几家网上银行还提供无罚息 CD 和标准 CD，可以帮助你提高收益率，”他补充道。
