# 用粉笔举起更多的重量

> 原文:[https://life hacker . com/use-chalk-to-lift-more-weight-1835493061](https://lifehacker.com/use-chalk-to-lift-more-weight-1835493061)

几个月来，我在举的时候都在和抓地力做斗争；在硬拉过程中，我会发现杠铃从我手中滑落，尽管我经常*觉得自己能举起更多。当我开始使用粉笔，健身房工具的救星时，很多事情都改变了。*

Watch

粉笔的工作原理是吸收你手上的汗水，并在它们和杠铃之间产生摩擦，有利于你的抓握。最终的结果是，你将经常能够更舒适地举起重物，正如 Greg Nuckols 为写的那样，允许你举更多次或更多重量或两者兼而有之。(这也是攀岩者和体操运动员为了更好地抓握而常用的方法。)

虽然粉笔通常被认为是力量举重者或交叉训练者的必需品，但它绝不是一个强制性的工具——尽管它确实对你的硬拉、农夫 T2 的搬运、引体向上或其他任何双手出汗的举重有很大帮助。除了粉笔，你可能还会发现握力强化练习对这些举重也有好处。

也没有一个精确的“正确”时间来使用它——只是尝试一下，找到最适合你的方法。就我个人而言，我不会在热身或一天较轻的重量时使用它，但绝对会在一天较重的硬拉时使用它。

你还可以在亚马逊或任何其他体育商店找到一些不同类型的粉笔。传统的干粉笔，通常是一个大球或大块，往往有点杂乱；你可能还需要在两次测试之间重新申请，但我也发现这是最便宜的解决方案。

液体粉笔本质上是和干粉笔完全一样的产品，但是添加了酒精添加剂。当它干燥时，它往往很粘，残余物可以在你接触的任何表面上，这是它最大的缺点。尽管如此，除了你可能在杠铃上留下的印记之外，我并不认为这是一个大问题。一般来说，你可以持续几套，也不需要重新申请。
