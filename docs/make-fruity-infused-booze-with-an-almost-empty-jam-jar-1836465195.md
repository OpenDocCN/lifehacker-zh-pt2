# 用几乎空了的果酱瓶制作果味酒

> 原文：<https://lifehacker.com/make-fruity-infused-booze-with-an-almost-empty-jam-jar-1836465195>

[果酱是一种美味的鸡尾酒配料](https://skillet.lifehacker.com/use-jam-to-make-any-booze-morning-appropriate-1832060017) ，即使是最微小的一点也能让酒类饮料变得更好地着色和调味——甚至是粘在罐子上的残余部分，无视任何刀子，拒绝取代烤面包。幸运的是，乙醇有办法溶解这种含糖物质，这意味着你可以用一个几乎空的果酱罐在几秒钟内给任何酒注入一些味道。把它一饮而尽，或者加一点柠檬和一大块巧克力作为快速鸡尾酒。
