# 如果你硬拉，买硬拉袜

> 原文:[https://life hacker . com/if-you-dy lift-get-dy lift-socks-1839329323](https://lifehacker.com/if-you-deadlift-get-deadlift-socks-1839329323)

你不需要特殊的袜子就能 [拿起一个杠铃然后再放下](https://lifehacker.com/why-deadlifts-are-worth-the-hype-and-how-to-do-them-sa-1756526319) 。它们无论如何都不是必需的。但是我发现在我开始穿齐膝袜后，我的硬拉训练进行得更好了，我想如果你还没有试过这种袜子，也许你应该试试。

Watch

关键是:当你把杠铃放在离身体很近的地方时，硬拉效果最好。当你站起来的时候，它应该看起来会沿着你的腿的前表面向上滑动。通过这样做，你可以保持平衡，并有效地将你的努力转移到重量上。

但是如果你擅长这份工作，这一条可能会擦伤你的小腿。如果你的酒吧有特别激进的滚花——哎哟。如果你参加比赛，你可能会被要求穿袜子，因为刮伤小腿是很常见的，没人想清理吧台上的血迹。

我总是想在我第一次比赛之前买双硬拉袜(事实上，下个月就要开始了)。但是有一天，我在清仓大甩卖时偶然发现了一些上面有可爱小猫的短袜，我想...为什么不呢？

朋友，他们有所作为。我没有意识到我害怕刮伤我的小腿，直到突然，我不必担心它。现在当我举起杠铃时，我感觉杠铃与袜子接触，让杠铃向上滑动。我给自己录像来检查我的状态，我可以告诉自己，我把杠铃放得更紧了。多亏了一双 2 美元的袜子，我现在举得更好了。

所以，试试硬拉袜吧。如果你想比清仓甩卖的廉价袜子更上一层楼，试试真正的硬拉袜([【SBD】](https://www.sbd-usa.com/product/sbd-deadlift-socks/)和 [Moxy](https://www.moxysocks.com/products/moxy-socks-premium-deadlift-cushion-knee-high-fitness-socks-3-pack) 是两个受欢迎的品牌)，它们往往由光滑的材料制成，以最大限度地减少阻碍，有时在小腿上有一点点缓冲。
