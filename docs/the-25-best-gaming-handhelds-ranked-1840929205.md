# 25 款最佳游戏手持设备，排名

> 原文：<https://kotaku.com/the-25-best-gaming-handhelds-ranked-1840929205>

我最近得到了一个任天堂 Switch 建兴，我喜欢它。我想写一写那种即将消亡的设备——专用便携式游戏机——的乐趣。我对自己说，我怎样才能用一种让大多数人讨厌我的方式来写这篇文章呢？而我意识到:一个排名榜！

所以，经过一番辩论和与自己的激烈争论，这里列出了 25 款最好的专用便携式游戏机，我认为这类设备在这个世界上并不长久。智能手机现在是手持游戏最受欢迎的地方，游戏设计正在围绕手机的功能进行弯曲，以至于带有操纵杆和按钮的独立设备几乎已经过时。你可以辩称，专用便携式设备之所以仍然存在，是因为任天堂已经以这样一种方式建立了 Switch 生态系统，即它可以在第一天发布一款包含 2000 多种游戏的便携式设备，而不必担心从头开始建立一个独家游戏库。

然而，几十年来，每个人都需要一个便携式系统，而且有很多选择，从 Atari Lynx 到 DS Lite。如何给他们排名？这是我的标准。实际上，我不会把每件事都分解得这么细，因为我们没有一整天的时间。这正是我所想的。

**便携性。**如果你的“便携式”游戏系统的大小和重量与一块砖头差不多，而且每小时消耗 6 节 AA 电池，那将是对你不利的。

**图书馆。**最后有几场？他们好吗？他们是唯一的吗？向后兼容以前版本的硬件是一个优势。(为了使这里的东西更加内部一致，我只是在看这些机器的北美版本，这意味着没有什么好奇怪的。抱歉，神奇天鹅。)

**特色。**它有一个漂亮的背光屏幕吗，或者你需要把它放在灯下吗？除了 D-pad 和一些按钮之外，还有其他很酷的铃铛和口哨吗？可充电电池与必须不断购买 AAs？

**外形尺寸。**它看起来有多棒？可用的配色方案如何？

我今天感觉如何。今天早上我是在床的哪一边醒来的？我有足够的快速眼动睡眠吗？我喝过咖啡了吗？让我们承认这项工作的最终任意性。

好了，我们走吧。

### 25.雅达利山猫(1989 年)

为什么会在这里？Atari Lynx 在美国仅比最初的 Game Boy 晚一个月发布，它呈现了一个完全不同的便携式游戏机的愿景。任天堂采用廉价、低功耗的部件来最大限度地延长电池寿命和降低成本，而雅达利则采用背光彩色显示器，价格翻了一番，可以在四小时内消耗六个 AAs。不过，当你有电池的时候，你的朋友会对它印象深刻。

**为什么不是更高？**我来数一下方式。它的两个型号都非常大，电池寿命非常差，除此之外，游戏库从来没有那么伟大或独特，没有真正的杀手级应用程序。不过，对于 1989 年来说，这是一个巨大的转变。你应该给他们信任。

### 24.TurboExpress (1990 年)

为什么会在这里？Game Boy 上有许多家庭任天堂游戏的可用版本，但 TurboExpress 做得更好，它实际上玩的是与家庭 TurboGrafx-16 相同的游戏盒。当然，游戏在低分辨率的彩色屏幕上看起来没有那么好，但这是意料之中的。

**为什么不是更高？TurboExpress 售价 250 美元，几乎是 Game Boy 的三倍，电池寿命甚至比 Lynx 还要差。虽然美国发行了许多优秀的 TurboGrafx 游戏，但它只是一个相当小的图书馆。**

### 23.创世纪游牧者(1995)

为什么会在这里？就像 TurboExpress，不过是给世嘉创世纪游戏用的！发行于 1995 年，当时创世纪的生命周期即将结束，Nomad 让你可以随时携带(几乎)整个 16 位家庭图书馆。

**为什么不是更高？**原因和其他飞得离太阳太近的高端便携式电脑一样:先进的技术意味着高昂的价格、过大的外形尺寸以及对电池的巨大需求。Nomad 甚至将电池卸载到一个独立的塑料外壳中，该外壳必须夹在主机上——尽管这可能更多地是考虑到这些“便携式”系统的大部分游戏通常是在家中进行的，同时插入电源插座。

### 22.新地理口袋颜色(1999)

为什么会在这里？鉴于 TurboExpress 和 Nomad 发生的事情，我很高兴 SNK 没有试图将整个 Neo Geo 家庭控制台缩小到一个便携式的外形，只有一个小小的传送带伸出来，你必须不断地将永无止境的 AA 电池放在上面进行祭祀。相反，它创造了一种更像高档游戏机的颜色，用一个整洁的小点击操纵杆代替了 D-pad。图书馆很小，但质量很好，有许多 SNK 格斗游戏的缩影。

**为什么不是更高？**好吧，如果你不喜欢迷你版的 SNK 格斗游戏，那么在 Neo Geo Pocket Color 上就没有更多的*整体*让你去做了。两年后，SNK 基本上切断了对该系统的支持。任何没有任天堂名字的便携设备都很难买到。

### 21.游戏装备(1991)

为什么会在这里？世嘉的第一款掌上电脑与 20 世纪 90 年代初的其他彩色便携式电脑一样存在电池寿命问题，但世嘉真的努力用端口和独家配件来填充游戏设备库。它甚至发布了一个官方转换器，让你可以在上面玩主系统游戏。

**为什么不是更高？**笨重的体积和电池寿命，当然，以及相对缺乏真正持久的内容。该图书馆主要是由港口或世嘉专营权的缩小版，没有太多的伟大的游戏称为自己的。

### 20.任天堂 2DS (2013)

为什么会在这里？考虑到通货膨胀因素，这是任天堂有史以来最便宜的便携式机器之一，是任天堂 3DS 生态系统的低端切入点，非常适合有多个孩子的家庭批量购买。它可能很丑，但它可以播放 3DS 的整个大型、优秀的库——如果只是在 2D 的话——以及所有的 DS 游戏。

**为什么不是更高？**它是 3DS 生态系统中最便宜的入口，但也是最不令人满意的。不可折叠的形状比 3DS 系列其余产品的蛤壳式设计有所下降。进入 3DS 有很多更好的选择。

### 19.PlayStation Vita (2012 年)

为什么会在这里？它可能已经在市场上露脸了，但不可否认 Vita 的吸引力。有机发光二极管屏幕上的高质量图形、双操纵杆、轻巧舒适的外形——Vita 酷毙了。它甚至播放了你在 PlayStation 3 上数字化购买的所有 PlayStation 1 游戏！

**为什么不是更高？**首先，有两个大的设计缺陷阻碍了它的发展——索尼决定使用极其昂贵的专有存储卡，加上无用的“背面触摸板”,这主要是因为人们不小心碰到了它。但真正摧毁 Vita 在这里的地位的是，在两年内， [索尼几乎像一个烫手山芋](https://kotaku.com/mobile-gaming-didnt-kill-the-vita-sony-did-1733350950) 一样放弃了 Vita，并放弃为它开发任何主要的 3a 游戏。它一直是 JRPGs 和独立宝石的天堂，直到 2017 年左右，Switch 正式将其扼杀，但这要归功于外部出版商和开发商，而不是索尼。

### 18.游戏男孩(1989)

为什么会在这里？ Game Boy 绝不是第一款便携式游戏系统，但它让便携式游戏走出了“好奇”的领域，成为“游戏文化的重要组成部分”Game Boy 是设计师 Gunpei Yokoi 的信条“用枯萎的技术进行横向思维”的一个极好的例子，它不是该块中最强大的系统，但它非常便宜，使用四节 AA 电池可以运行相当长一段时间。这在很大程度上归功于套装游戏*俄罗斯方块*，它如此成功，以至于任天堂在推出后的近十年里都不用担心添加彩色图形。

**为什么不是更高？比起使用旧的“砖块”模式，如今有太多更好的游戏方式。其屏幕笨重的尺寸和病态的绿色色调可能在我们的怀旧意识中根深蒂固，但至少你可以升级到…**

### 17.游戏男孩口袋(1996)

为什么会在这里？Game Boy Pocket 是单色便携式电脑之王，它的尺寸小得多，黑白屏幕更清晰，仅用两节 AAA 电池就能玩很长时间。

**为什么不是更高？**如果 Game Boy Pocket 有背光的话，它今天会更酷(这是由日本专用的 Game Boy Light 型号纠正的)。如果你讨厌颜色，这是最好的便携设备。但你可能不知道。

### 16.游戏王进阶(2001)

为什么会在这里？在一个游戏主机开发已经稳步进入 3D 领域的时代，Game Boy Advance 让 2D 游戏保持了活力。这款令人惊讶的小掌上电脑最终拥有一个庞大的任天堂和第三方优秀游戏库，无论是 99%准确的超级 NES 港口还是完全原创的游戏。

**为什么不是更高？**任天堂不愿在其便携式系统中包含耗电背光，这一点随着最初的 Game Boy Advance 达到了临界点，如果你想看到任何东西，基本上必须在直射光下玩。它也仍然依赖于 AA 电池，这在 2001 年变得有点旧。

### 15.任天堂 DSi (2009)

为什么会在这里？任天堂广受欢迎的原始 DS 硬件的第三次迭代试图使它更像一个现代设备。最大的变化是 DSiWare 商店，它为平台带来了可下载的游戏。它还可以拍照和播放 SD 卡上的音乐。你可以上传你的照片到脸书！

**为什么不是更高？**嗯，部分原因是没人真的想要这些。我们有 iPhones、iPods 和 Zunes 等产品，不需要把这些功能卸载到任天堂掌上电脑上。此外，任天堂删除了一些真正有用的东西 Game Boy Advance 游戏槽——给了我们这些不太有用的功能。

### 14.游戏男孩颜色(1998)

为什么会在这里？任天堂的第一款彩色便携式电脑姗姗来迟(在第一款 Game Boy 推出 9 年后)，但当它真的到来时，它发扬了第一款的设计理念:小巧、便宜、电池使用量轻。它不仅玩自己的游戏，还用自定义调色板增强了你的黑白 Game Boy 游戏库，使它们看起来比以前更好。

**为什么不是更高？没有背光，加上它似乎是任天堂在真正的继任者 Game Boy Advance 上工作时的一个权宜之计。这是大量的翻拍和移植，颜色的生命周期对它来说有点太早了，不能真正建立一个伟大的独家阵容。**

### 13.任天堂 3DS (2011)

为什么会在这里？虽然开始不顺利，但任天堂的裸眼 3D 掌机最终相当成功，拥有一个强大的独家图书馆。最初的过于昂贵的型号有很多后来版本没有的神韵，有着闪亮的光泽，引人注目的棱角分明的外形，就像抛光宝石的刻面一样，谁能忘记那支可怕的伸缩手写笔？

**为什么不是更高？**从 DSi XL 的超大显示屏回到 3DS 相对较小的屏幕上，感觉是一种降级。无眼镜 3D 的视角远不如后来的版本。

### 12.任天堂 DS (2004)

为什么会在这里？当任天堂当时的总裁山内浩史(Hiroshi Yamauchi)告诉他的工程师设计一款双屏掌上电脑时，他们采用了这个奇怪的方向，并将其推出，增加了触摸灵敏度，永远改变了我们玩便携式游戏的方式。

**为什么不是更高？**我的意思是，你*看过*了吗？任天堂已经生产了一些看起来很棒的掌机，最早的 ds 肯定不是，长官。现在有很多玩 DS 游戏的方法，但是我想不出一个理由推荐任何人使用 DS 的原始模型来玩。还好任天堂替换的快。

### 11.PSP (2005)

为什么会在这里？它并没有像所有人预测的那样彻底击败任天堂 DS，但索尼的第一款便携式电脑表现相当不错，积累了一个相当大的优秀软件库。硬件方面，索尼包括模拟滑动垫是正确的，任天堂直到 3DS 才有。

**为什么不是更高？**索尼试图通过 UMD 向世界推出一种全新的基于光盘的格式，这值得称赞，但事实证明，长加载时间和便携式游戏就像花生酱和被卡车撞一样不可避免。你不可能在公交车站打开 PSP 玩几分钟游戏；这是漫长的等待时间。此外，如果几天不插上电源，电池会自动耗尽。(此外，所有这些电池现在都爆炸了。PSP 有很棒的游戏，但 21 世纪初的索尼体验很糟糕，再也没有变得更好。

### 10.游戏男孩微(2005)

为什么会在这里？继最初的 DS 之后,发布了*,这款微型游戏是 Game Boy Advance 阵容中令人惊讶的最后一员。它将外形缩小到基本上可以放在钥匙链上的东西，看起来绝对超级酷，同时玩起来还算舒适。它甚至有可更换的面板。*

**为什么不是更高？**然而并没有*那么舒服，尤其是对于长时间的游戏。Game Boy Micro 与其说是便携式游戏的主力产品，不如说是一种新奇产品。但这仍然是一个非常迷人的新奇事物。*

### 9.任天堂 DSi XL (2010)

为什么会在这里？任天堂又一次在别人曲折的地方曲折前进。当其他人试图把他们的硬件做得越来越小的时候，任天堂把 DSi 做得更大。游戏在更大的屏幕上看起来令人惊叹，它甚至配有一个大而粗的手写笔，以增加舒适度！此外，它是葡萄酒色的，适合成年人。

**为什么不是更高？**DSi XL 几乎被后来的 XL 手持设备所囊括；他们做不到的事情并不多。但如果你只是想找个东西玩原版 DS 游戏，这绝对是最好的选择。

### 8.任天堂 3DS XL (2012)

为什么会在这里？在让所有人都从 DSi XL 缩小到 3DS 之后，任天堂“只是在开玩笑”，然后发布了这个更大尺寸的版本，我们都松了一口气，不再眯着眼睛。

**为什么不是更高？**嗯，再次，这是因为一个新的任天堂 3DS XL 是这台机器的一切，再加上一点点。此外，他们这次没有包括额外的手写笔。

### 7.游戏男孩高级 SP (AGS-001) (2003)

为什么会在这里？Game Boy Advance 的第一版感觉就像一个玩具，但 SP 感觉就像来自未来的东西。独特的翻盖设计保护了屏幕，使设备更加便携和舒适。使用前灯意味着不再需要将系统置于灯下。原始的 Game Boy 游戏看起来也从来没有这么好。

**为什么不是更高？**没有专用的耳机插孔可能是 SP 最令人恼火的地方。你必须使用一个插入电源插座的转换器，这意味着你不能同时充电和使用耳机。这不是一个交易破坏者或任何东西，只是讨厌。此外，前光不是背光，它往往会淡化图形，而不是增强图形。

### 6.任天堂 Switch 建兴(2019)

为什么会在这里？虽然最初的交换机可以随身携带，但它并不是 3DS 的完美替代品，因为它相当大，而且不是为最大便携性而专门设计的。Switch Lite 解决了所有这一切，它是一个单件、更轻、更小的单元，可以玩除 Labo 之外的所有 Switch 游戏。

**为什么不是更高？**Switch Lite 作为 Switch 家族中真正便携的首次尝试，完成了任务，但它是一种相当熟练的方法，没有任何令人惊讶的重新设计元素，比如 Game Boy Advance SP 或 DS Lite。它没有任天堂最好的设计那样强烈的吸引力。此外，由于 Switch 仍处于其生命周期的早期，我不愿意给它更高的评级，因为我们仍然没有看到它的库将走向何方。

### 5.新任天堂 2DS XL (2017)

为什么会在这里？这款任天堂 3DS 家族的最后一款作品可以说是最好的一款，如果你不在乎 3D 图形的话。它比新的 3DS XL 更轻，但仍然有“新”的附加功能，如额外的拇指棒，与《T2》等游戏的兼容性，以及可下载的 SNES 游戏。

**为什么不是更高？嗯，我*确实*关心 3D 图形，即使任天堂在 3DS 生命周期的中途停止了这种做法。此外，尽管这台机器的设计可以说比 3DS XL 更时尚，但它的触控笔绝对是任何 DS 或 3DS 型号中最差、最短的。**

### 4.第三。(并列)新任天堂 3DS/新任天堂 3DS XL (2015)

他们为什么会在这里？任天堂在同一天发布了新款任天堂 3DS 及其 XL 版本(反正是在日本)，很难挑出一个明确的赢家。新的 3DS 更小，但它的屏幕仍然相当大，而不是原来 3DS 上的邮票。它还有那些很棒的可更换面板。同时，加大码是加大码。两者都有“超级稳定的 3D”，使用红外摄像机跟踪你的眼球，给 3D 效果带来更宽的视角。

为什么他们没有更高？坦白地说，我仍然对 3DS 不是无地区的感到恼火。

### 2.游戏男孩高级 SP AGS-101 (2005)

为什么会在这里？在这个列表中，我还没有太多的增量升级，主要是因为它们没有从根本上改变设备的体验。不过，Game Boy Advance SP 的最终型号是一个主要的例外。任天堂将前光屏幕换成了明亮、美丽的背光屏幕，使这款完全向后兼容的机器成为了 Game Boys 的 *ne plus ultra* 。当我想玩游戏机的时候，这就是我伸手要拿的东西。

**为什么不是更高？**我的意思是，这是吹毛求疵，但我们已经谈论过耳机插孔...

### 1.2006 年世界杯足球赛

为什么会在这里？几乎很难相信，设计出原始、怪异、丑陋的任天堂 DS 设计的同一批设计师随后又推出了这件该死的艺术品。任天堂肯定抄袭了苹果 DS Lite 的作业——说真的，它看起来就像当年的 iPod 和 MacBook——但结果如此令人印象深刻，以至于每个人都不得不拥有一台，以免我们忘记，它们在全球范围内受到供应限制*年*。哦，和原版相比，屏幕看起来*很棒*。

**为什么不低一点？**是的，我确实说过，如果我今天想玩 DS 游戏，我会选择 DSi XL。但 DS Lite 的外观仍然是最好的，它在较低的屏幕上仍然可以很好地玩 Game Boy Advance 游戏。考虑到我为这个列表制定的所有标准，我认为 DS Lite 是赢家。
