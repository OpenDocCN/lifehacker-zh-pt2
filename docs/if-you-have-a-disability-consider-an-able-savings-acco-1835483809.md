# 如果你有残疾，考虑一个有能力的储蓄账户

> 原文:[https://life hacker . com/if-you-hacken-a-able-savings-acco-1835483809](https://lifehacker.com/if-you-have-a-disability-consider-an-able-savings-acco-1835483809)

如果你有残疾，你可能会错过省钱的机会。能力储蓄账户允许残障人士为他们的财务需求储蓄— *而*不会因为这些储蓄而取消他们享受政府福利如医疗补助的资格。

Watch

ABLE accounts(实现更好的生活体验法案)于 2014 年创建，并于 2016 年首次推出。现在， [纽约时报报道](https://www.nytimes.com/2019/06/07/your-money/able-accounts-savings-disabled.html) ，他们在 41 个州和华盛顿特区可用，迄今为止注册人数很低——只有大约 4 万人——尽管国家残疾研究所表示多达 800 万人可能有资格获得 able 账户。

《纽约时报》解释说，到目前为止，拥有超过 2000 美元的存款或其他资产可能会使你失去接受医疗补助或补充安全收入的资格。《泰晤士报》的安·卡拉斯报道:

> 存在银行账户上的钱...不算在总数里。因此，残疾人可以为未来的需要而储蓄，或者将这笔钱用于当前与残疾相关的费用，如医疗保健、住房、交通、教育和培训，而不会失去政府的帮助。

4 月份，住房和城市发展部发布了一份 [通知](https://www.hud.gov/sites/dfiles/OCHCO/documents/2019-09pihn.pdf) ，阐明有能力的资金不能取消你获得住房援助的资格。

## 谁能开一个有效的储蓄账户？

每个州都有一套不同的规则，但总的来说，你必须在 26 岁之前成为残疾人，才有资格申请这些储蓄账户中的一个。国会正在考虑将年龄限制提高到 46 岁。

您每年最多可以节省 15，000 美元。如果你工作，州法律可能允许你每年节省高达 27，000 美元。如果你有一个 529 大学储蓄账户，这些资金可以转入一个有能力的账户，只要金额低于每年的捐款上限。

你可以把你的钱投入到共同基金或其他投资中，存在传统的储蓄账户中，或者持有一个附带借记卡的支票账户。一些州允许非本地居民注册，尽管你的家乡州可能会对有能力的账户提供税收减免。

需要注意的一点是:如果你死了，州政府可以申请你的全部或部分积蓄来支付你的医疗补助。

国家残疾研究所维护着 [ABLE 国家资源中心](http://ablenrc.org/)，在这里你可以比较各州的项目，了解如何报名。
