# 如果您错过了转机，请立即联系航空公司

> 原文：<https://lifehacker.com/if-you-miss-a-connection-contact-the-airline-immediate-1839635585>

早在 8 月份，我们就写过所谓的“”的风险——当一个旅行者因为比直飞更便宜而故意错过一段连接时。航空公司并不这么认为，这就是为什么它会带来取消整个行程的真实风险(这在几份航空公司的运输合同中是不可避免的)。但是，如果您*不小心*错过了一段连接，并且面临同样的风险，您该怎么办？

Watch

在埃利奥特倡导组织的用户论坛 上，一位评论者提到了最近的一次经历，他们的儿子在美国航空公司的旅程中意外错过了转机。“当他办理从 PHX 到伦敦的返程航班登机手续时，他发现美国航空公司取消了他的行程，”他们写道。“他们没有通知他。他们不会恢复他的航班，除非他支付 150 美元的改签费，并支付票价差额(超过 600 美元)，即使该航班有许多座位，他也购买了机票。”

如果你最终错过了转机——并且仍然有返程计划——这里有一条重要的建议:立即联系航空公司，通知他们你可能会或已经错过了你的航班。有可能他们会让你付费改签航班*并且*确保不要取消你剩余的旅程。(查看 [我们的向导](https://lifehacker.com/heres-what-youll-pay-to-change-your-flight-1836410208) 关于改签费用；费用最终取决于你是乘坐国际航班还是国内航班以及时间安排。)

另一方面，如果连接是航空公司的错，或者有与天气有关的延误，那么你不应该担心；一般来说，航空公司会自动为你重新预订航班，你的返程不会被取消。
