# 你也许不应该吃豆芽

> 原文:[https://life hacker . com/you-should-possible-never-eat-sprouts-1841960543](https://lifehacker.com/you-should-probably-never-eat-sprouts-1841960543)

苜蓿芽、三叶草芽和豆芽很好吃，以我(也可能是唯一的)拙见。但是， [和生菜](https://vitals.lifehacker.com/lettuce-is-bullshit-1830778975) 一样，它们容易被污染而生吃。现在，我们正处于一场多州爆发的大肠杆菌*与吉米·约翰餐厅供应的三叶草芽有关的事件中。*

*Watch*

*吉米·约翰已经停止供应豆芽了，但是如果你还有剩豆芽，疾控中心建议你扔掉。然而，更重要的是，豆芽被污染是相当普遍的，许多传染病或公共卫生专家干脆 [不吃它们](https://twitter.com/InfectiousDz/status/1232811348322668545) 。*

 *[https://lifehacker.com/embed/inset/iframe?id=twitter-1232811348322668545&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1232811348322668545&autosize=1)* 

*不幸的是，这个问题是豆芽生长方式所固有的。不管是你买的还是自己发芽的，它们都被保持潮湿和温暖几天，为细菌的生长创造了完美的条件。*

*这意味着如果豆芽的种子被沙门氏菌或 T2 大肠杆菌污染，这些细菌会随着豆芽的生长而生长，并使你生病。 [洗涤并不能除掉细菌](https://www.nhs.uk/live-well/eat-well/sprouted-seeds-safety-advice/) 。烹饪可以，但它破坏了新鲜松脆的质地，这可能是你首先想要豆芽的全部原因。知道了这一点，你可能会决定不吃豆芽更安全。*
