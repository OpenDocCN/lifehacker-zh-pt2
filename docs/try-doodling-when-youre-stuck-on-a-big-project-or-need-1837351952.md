# 当你陷入一个大项目或需要放松时，尝试涂鸦

> 原文:[https://life hacker . com/try-doodle-when-you-stick-on-a-big-project-or-need-1837351952](https://lifehacker.com/try-doodling-when-youre-stuck-on-a-big-project-or-need-1837351952)

快速浏览一下我高中的笔记本，就能证明我是一个糟糕的艺术家，但这并不是因为我没有尝试过。

不管主题是什么，涂鸦是我在听学校讲座时经常做的事情。即使在今天，我的桌子上仍然堆满了业余爱好者画的花、狗和树的待办事项清单。事实证明，所有这些涂鸦实际上可能让我更有效率。

研究表明，涂鸦，就像漫不经心地画一幅图像而不是试图创作一件艺术作品，可以帮助增强你的记忆和认知，让你更有创造力并帮助你放松， [快公司报道](https://www.fastcompany.com/www.fastcompany.com/) 。

研究表明，当我们涂鸦时，我们实际上注意力更集中，并能记住更多呈现的信息。这对老年人尤其有帮助。虽然这显然会因情况而异，但这是很有希望的信息。

研究人员表示，涂鸦的效果相当于写笔记，而不是打字。你在运用多种感官，这让你总体上更加清醒。

那些涂鸦不一定是什么特别的东西。即使只是在一个单词周围画一个形状也可以帮助巩固你的记忆，在纸上画一条曲线一两分钟就足以帮助你放松。

如果你开始在工作中不知所措，或者被一个项目的解决方案困住，试一试吧，这可能正是你所需要的。
