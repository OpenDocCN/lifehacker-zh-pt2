# 只有一种方法可以分离鸡蛋，这就是

> 原文：<https://lifehacker.com/there-is-only-one-way-to-separate-an-egg-and-this-is-it-1837029623>

这在 Lifehacker 的办公室引发了一场激烈的辩论。

我们都同意购买一个鸡蛋分离器是没有必要的，用空瓶子吸出蛋黄这种做作的方法并不是真正的日常解决方案。一些员工认为，你应该把蛋壳掰成两半，把蛋黄从蛋壳中间穿过。

经过多次测试和多次打碎蛋黄后，我可以正式宣布，分离鸡蛋最简单、最快的方法就是用你的双手舀起蛋黄*。查看上面的视频，了解正确的方法。*
