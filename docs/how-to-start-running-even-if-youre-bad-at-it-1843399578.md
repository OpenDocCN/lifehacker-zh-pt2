# 如何开始跑步，即使你不擅长跑步

> 原文：<https://lifehacker.com/how-to-start-running-even-if-youre-bad-at-it-1843399578>

如果你想开始一个健身计划，跑步是你能做的最简单也是最便宜的事情之一。但是你如何开始跑步，尤其是如果你真的非常非常不擅长跑步的话？(我说的是“绕着街区跑让我忍俊不禁，怀疑自己是不是就要死了”的糟糕程度？)

幸运的是，跑步只会变得越来越容易。作为一名长期跑得很慢的人，我花了几个月的时间完成了 3 英里的例行训练，并获得了许多比赛的最后一名，以下是我在这一路上学到的一些来之不易的经验。

### 好鞋是必不可少的

一双好的、合脚的跑鞋是你需要的一件装备。其他的一切——高科技的跑步服、健身追踪器、各种各样的跑步装备——真的只是锦上添花。

说到买合适的跑鞋， [贵并不总是意味着更好的](https://lifehacker.com/you-dont-need-an-expensive-pair-of-running-shoes-1733660762) 。很多公司会试图让你相信了解你的步态和脚弓类型的重要性。虽然这些话听起来非常科学和精确，但它们实际上与预测哪种鞋最适合你没有多大关系。最好的鞋子是你穿起来舒服的，即使它不适合你的脚型。

在前疫情时代，找到合适的鞋子的最好方法是去专门的跑步用品店试穿各种不同的鞋子，看看哪一种感觉舒适又有支撑力。不幸的是，在这个时代，这不再是明智的，所以如果你有一双舒适的鞋子，最好的办法就是继续穿。

如果你*没有*一双舒适的鞋子，你可以做一双 [虚拟试衣](https://vitals.lifehacker.com/how-to-get-fitted-for-running-shoes-without-leaving-you-1843110073) 。这不是一个完美的答案，但它仍然比随意订购一双鞋并抱最好的希望要好。只要确保询问交换政策，以防你得到的那双不合适。

市面上有很多花哨的跑步服装，但严格来说，你并不需要其中任何一件。穿你能穿的舒服的衣服就行了。你的跑步服可以是折扣店的入门级装备。它们也可以是你从大学开始就有的 t 恤和短裤。他们不需要特别，他们只需要舒适。

### 一个好的运动胸罩会创造奇迹

对许多人来说，一个好的运动胸罩是必不可少的。即使你头重脚轻，即使你已经花了几年时间告诉自己因为你的体型你不能跑步，只要你有合适的运动胸罩，你仍然完全有可能进行高强度的锻炼，比如舒适地跑步。

作为一个已经与这个问题斗争多年的人，我发现尽管这需要一些时间、努力和反复试验，但还是有合适的选择。就我个人而言，我相信 [减震器的 Active D+运动胸罩](https://www.shockabsorber.co.uk/active-d-plus-max-support-sports-bra-n109) ，它的尺寸可达 40H，已经让我经历了我数不清的更多长跑和 5 公里/10 公里。比起其他受欢迎的选择， [Enell](https://enell.com/products/enell-sport) ，我更喜欢减震器，只是因为减震器提供了更广泛的尺寸和更个性化的贴合。也就是说，Enell 已经为许多人创造了奇迹，正如许多其他为更大尺寸量身定制的高影响力运动胸罩一样。

无论你的体质如何，总会有一种舒适、舒适的选择。诀窍是非常仔细地衡量，做你的研究和实验，直到你找到适合你的。如果你有疑问，subred dit[A Bra fit](https://www.reddit.com/r/ABraThatFits/)是一个非常有用的资源，可以帮助你计算尺寸，找到适合你特殊需求的建议。

### 一致性是关键

当谈到建立一个跑步习惯时，一致性是关键。跑步是一项高冲击力的运动；你的身体需要适应它。这就是为什么，即使你身材很好，刚开始也可能会有一段时间很糟糕。重要的是在跑步中保持一致，并在里程中放松。

如果你是一个完全的初学者，一个 [沙发到 5 公里的计划](https://vitals.lifehacker.com/all-the-questions-youll-have-when-you-start-couch-to-5k-1830857969) 是一个很好的开始，因为它会让你交替步行和慢跑，随着时间的推移慢慢建立你的跑步。到最后——通常是九周后——你应该能跑 3 英里了。

作为一个一直跑得很慢的人，在开始时有很多困难，我建议你慢慢来，集中精力保持一致。当你可以轻松跑完 3 英里的时候，跑步会变得更容易，你会开始真正感受到你的跑步伙伴们所称赞的一些好的身体效果。说到培养跑步习惯，固执一点，有条不紊一点就好。

### 知道受伤的迹象

一次不错的长跑会让你感到酸痛，但知道普通的 [疼痛和更严重的](https://vitals.lifehacker.com/how-to-avoid-injuries-as-a-beginning-runner-1833906254) 疼痛之间的区别很重要。肌肉酸痛，不舒服但不痛的疼痛，以及跑步时消失的不适通常是意料之中的。如果你感到剧烈的刺痛；当你跑步时，影响你步态的受伤或疼痛加剧，这通常是你需要停止跑步并咨询医生的信号。

“”一书的作者克里斯蒂·阿施万登在给 Lifehacker 的一封电子邮件中写道:“初学者可以做的避免受伤的最重要的事情之一就是慢慢来。“避免过快增加里程。一个非常粗略的经验法则是避免每周增加超过 10%的里程数。”

为了避免受伤，保持一致和避免过度训练是很重要的。 [跑得太多太快](https://lifehacker.com/five-running-mistakes-that-keep-you-locked-in-the-injur-1672356068) 是伤害自己的好方法。

“也一定要给自己时间休息，”Aschwanden 说。“定期跑步很好，但每周至少给自己一天休息日，这样你的身体才有机会恢复和再生。”

如果你开始受伤，给自己足够的时间恢复是很重要的。虽然被排除在外可能会令人沮丧，但这总比花几个月时间从因过早恢复训练而恶化的伤病中恢复要好。
