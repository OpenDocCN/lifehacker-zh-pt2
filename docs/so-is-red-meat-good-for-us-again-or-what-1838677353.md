# 那么红肉对我们又有好处了吗？

> 原文:[https://life hacker . com/so-is-red-meat-good for us-again-or-what-1838677353](https://lifehacker.com/so-is-red-meat-good-for-us-again-or-what-1838677353)

根据 [五项新的荟萃分析](https://annals.org/aim/fullarticle/2752328/unprocessed-red-meat-processed-meat-consumption-dietary-guideline-recommendations-from) ，没有足够的证据告诉人们要避免吃红肉。这是不是意味着我们被骗了，牛排一直都很好？或者这只是头条 的又一个反转，最好忽略？

### 你得到的答案取决于你问的问题

几十年来，有大量关于红肉和健康结果的研究。一些人发现了看起来像是一种联系:例如，在一些研究中，记得一生中吃更多红肉的人被证明更有可能患有某些癌症或心脏病。

但是离我们不应该吃红肉的结论还有很长的路要走。研究实际上不能问类似“红肉好吗？”相反，科学家收集非常具体的事实，并以他们希望能给我们更多信息的方式进行分析。

现实生活很乱。这里有几个例子可以说明这个问题有多混乱:如果在你的研究中吃红肉的人吃了很多快餐汉堡，是肉的问题还是他们吃的其他快餐的问题？在你的研究中，吃红肉的人比其他人更富有还是更贫穷？年长还是年轻？那些少吃红肉的人呢——他们在吃什么呢？

这些都是很难回答的问题，没有办法设计一项研究来明确回答这个问题。但我们可以做的一件事是仔细观察一大组研究，看看是否有任何模式突出。

### 解释的问题

新的分析没有发现红肉对我们有好处，只是没有足够有力的证据告诉人们停止食用红肉。下面是他们的 [总体结论](https://annals.org/aim/fullarticle/2752328/unprocessed-red-meat-processed-meat-consumption-dietary-guideline-recommendations-from) :

> **建议:**专家组建议成年人继续目前未加工的红肉消费(弱建议，低确定性证据)。同样，专家小组建议成年人继续目前的加工肉类消费(弱建议，低确定性证据)。

它说，想吃什么就吃什么，但即使这也是基于“低确定性证据”的“微弱建议”不需要吃更多或更少的红肉，他们甚至不会强烈地感觉到你是否继续吃任何数量的红肉。这是一种安慰。

但是事情是这样的:这个建议是基于与你读过的所有关于不吃红肉的建议完全相同的科学。流行病学家 [Gid M-K 写道](https://medium.com/@gidmk) :

> 因此，主要的差异来自解释，而不是证据本身。新的研究认为，由于我们拥有的证据相对较少，我们不能根据研究告诉人们该做什么。...争论实际上可以归结为当我们说红肉对健康有害时，我们能有多自信。

与此同时，主张避免吃红肉的研究人员和组织仍然可以依赖他们最初拥有的相同证据。例如，营养研究员 Christopher Gardner[在 Twitter](https://twitter.com/GardnerPhD/status/1178781311495835648) 上发表了一些对新分析的批评。例如，他指出一些减少肉类摄入的研究正在消失。正如研究可能会有使它们变得复杂或令人困惑的问题一样，荟萃分析也是如此。

### 那么我应该吃什么呢？

关于新的分析，一个有趣的事情是，他们包括了一项研究，调查人们是否真的在被告知改变饮食时改变了饮食。如果没人听推荐，那么也许不值得花力气大声推荐。

不过，这是另一个不同的观点。如果吃红肉有一点点可能的风险，难道我们不想知道吗？以防万一，我们中的一些人可能会选择不同的食物。你仍然可以做到这一点。

虽然科学家们在争论从所有关于红肉的研究中得出的最佳结论，但许多专家认为人们不要太担心这项研究，而是应该关注我们所知道的健康饮食模式和健康的行为。加德纳指出 更多的蔬菜、豆类、水果和纤维是健康的食物，不管你是否需要从饮食中去除肉类来适应它们。肥胖专家约尼·弗里德霍夫 [分享了他简短的健康行为清单](https://twitter.com/YoniFreedhoff/status/1178983307079323649) ，其中包括“接种疫苗”和“用全原料烹饪”

所以，如果你需要遵循一个规则，试着做所有你已经知道是健康的事情:首先，多吃蔬菜，少加糖。新的分析也没有考虑避免食用肉类的环境相关或伦理原因，这也是值得关注的问题。
