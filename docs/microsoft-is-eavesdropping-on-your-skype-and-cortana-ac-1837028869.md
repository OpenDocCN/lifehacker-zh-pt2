# 微软正在窃听你的 Skype 和 Cortana 活动

> 原文:[https://life hacker . com/Microsoft-is-tapping-on-your-Skype-and-cortana-AC-1837028869](https://lifehacker.com/microsoft-is-eavesdropping-on-your-skype-and-cortana-ac-1837028869)

许多公司开始对使用真人来“改善”他们的数字助理持保留态度，方法是审查你对智能扬声器或手机说过的话。我敢打赌，微软也将很快改变这种做法，但现在，承包商可能会听你告诉 Skype Translator 和 Cortana。

Watch

根据 Vice 的 [主板](https://www.vice.com/en_us/article/xweqbq/microsoft-contractors-listen-to-skype-calls) 的说法，一位未透露姓名的微软承包商能够提供使用 [Skype 翻译](https://www.skype.com/en/features/skype-translator/) 功能的人的录音——录音长度往往从 5-10 秒不等，但不限于此。为了帮助微软提高该功能的能力，这些承包商会听取用户的意见，并从可能的翻译列表中进行选择，或者在某些情况下，提供他们自己的翻译。

当被问及这一设置时，微软代表告诉 Motherboard，该公司通过一个安全的在线门户网站提供这些记录，并采取措施(未描述)删除任何可能用于事后识别用户身份的相关信息。然而，这并不能阻止人们在与 Cortana 这样的数字助理交谈时透露自己的信息(如地址)，而且似乎没有任何设置来阻止微软的承包商分析这种口头数据。

根据微软提供给主板的声明:

> *“微软收集语音数据是为了提供和改进支持语音的服务，如搜索、语音命令、听写或翻译服务。我们努力保持语音数据收集和使用的透明度，以确保客户能够就其语音数据的使用时间和方式做出明智的选择。微软在收集和使用客户的语音数据之前会获得客户的许可。"*

> *“在与我们的供应商共享这些数据之前，我们还实施了几项旨在优先保护用户隐私的程序，包括消除数据身份、要求与供应商及其员工签署保密协议，以及要求供应商满足欧洲法律规定的高隐私标准。我们将继续审查我们处理语音数据的方式，以确保我们向客户尽可能清晰地做出选择，并提供强有力的隐私保护。"*

### 你能阻止 Skype 把你说的话发给微软吗？

总之，没有。至少，当我们发表这篇文章时，我没有在微软的 Skype Translator 的 [隐私常见问题解答上看到任何指示，表明你可以限制该公司收集语音数据。这种做法说得有些清楚:](https://support.skype.com/en/faq/FA34583/skype-translator-privacy-faq)

> *“当您使用 Skype 的翻译功能时，Skype 会收集并使用您的对话来帮助改进微软的产品和服务。为了帮助翻译和语音识别技术学习和发展，我们会分析句子和自动抄本，并将任何更正输入我们的系统，以构建更高性能的服务。为了帮助保护您的隐私，用于产品改进的对话会用字母数字标识符进行索引，这些标识符不会识别对话的参与者。”*

我说有点，因为微软没有在它的 FAQ 中指出你的演讲是由真人来分析的。事实上，这种描述几乎意味着这是一个完全机械的过程，但事实并非如此——也不可能如此，因为机器无法选择正确的翻译。关键是人类必须训练系统变得更好。

我也没有看到 iOS Skype 应用程序中的任何设置会让你选择退出这一“改进”过程，但微软可能会改变这一方法。如果有一个选择退出开关，或者更好的是，在开关中有一个 opt- *来允许分析语音数据，那就太好了。*

### Cortana 呢？

正如 Vice 的报告所指出的，Cortana 命令也是承包商可以倾听的公平游戏。然而，你*可以*选择退出这种做法。为此:

1.  在 Windows 10 中调出设置应用
2.  点击隐私
3.  点击左侧边栏上的语音
4.  禁用“在线语音识别”功能

问题？禁用此功能*也会*束缚 Cortana。您仍然可以使用“数码助理”来访问信息，但是您将不能与它对话，也不能让它响应您的命令。

你最好提醒自己定期查看微软存储的 Cortana 语音数据。为此，请访问您的 [微软账户](https://account.microsoft.com/) 页面，点击顶部的 [隐私](https://account.microsoft.com/privacy/) 标签。向下滚动到“语音活动”并单击“查看和清除语音活动”按钮。在数据列表的右上角找到“清除活动”链接，然后单击它。删除所有的东西。

当然，我无法清除我的数据，但我希望你有更好的运气。

还要注意，这可能仍然不会阻止微软承包商监听你告诉 Cortana 的内容——这完全取决于你是否在之前删除了这些数据*,这些数据被用来“改进微软的功能”我们不知道在 Microsoft 将您的录音用于其他用途之前，您需要删除多少时间，甚至不知道此过程是否会删除该录音的唯一实例。很有可能微软只是复制了你所说的话，“匿名化”了它，然后用它来代替。*

最终，不使用在公司服务器上处理你声音的服务是确保其他人听不到你说话的最佳方式，但这是我们在当今数字世界为了方便而做出的权衡。如果你想让一个数字助理或应用程序理解你在说什么，并根据这些信息采取行动，你将不得不放弃一点隐私来从中受益。至少，这是一种设置，直到更多的公司认识到，让客户选择是否希望他们的语音由另一个人处理是很重要的。
