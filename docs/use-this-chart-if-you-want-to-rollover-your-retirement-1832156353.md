# 如果你想将你的退休金账户展期，可以使用这个图表

> 原文：<https://lifehacker.com/use-this-chart-if-you-want-to-rollover-your-retirement-1832156353>

记录退休账户对于普通美国人来说可能很棘手——根据美国政府问责局的数据，2004 年至 2013 年间，超过 2500 万工人在更换雇主时至少留下了一个账户，这意味着数千亿美元的福利损失 。



不仅如此，你还可能要支付额外的费用，或者损害你的投资业绩。

这就是整合发挥作用的地方。根据帐户的类型，您还有一些其他因素，合并帐户可能是有意义的。这样，你就知道你的钱在哪里了。

但不是所有的退休账户都可以转到另一个账户。虽然规则可能很复杂，但美国国税局有一个简化的图表来帮助你开始:

正如[Kiplinger](https://www.kiplinger.com/article/retirement/T047-C000-S004-consolidate-your-retirement-accounts-carefully.html)指出的，将雇主 401(k)计划转入单一个人退休账户是保留“资产的税收优势地位”并允许投资选择的一种方式。你还可以减少文书工作，并且更容易计算出退休年龄所需的最低分配额。

但有理由对整合保持警惕。“你可能希望保留一个 401(k)计划，该计划拥有共同基金的较低成本机构股和免佣金交易，而不是将其转入另一个不包括这些功能的账户，”Kiplinger 写道。如果你想早点退休，你会想要一个账户，让你存取你的钱免于罚款。
