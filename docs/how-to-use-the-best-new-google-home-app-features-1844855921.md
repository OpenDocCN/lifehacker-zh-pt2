# 如何使用最好的新 Google Home 应用功能

> 原文：<https://lifehacker.com/how-to-use-the-best-new-google-home-app-features-1844855921>

Google Home 版本为该应用增加了一系列新功能，包括在 [iOS](https://apps.apple.com/us/app/google-home/id680819774) 和 [Android](https://play.google.com/store/apps/details?id=com.google.android.apps.chromecast.app) 上对 Nest home 安全产品的新应用内控制，以及对 Android 11 的电源菜单快捷方式的扩展设备支持。哦，安卓终于有了黑暗模式——尽管可能很难打开。以下是使用所有新功能的方法。

### 新的 Nest 摄像头和智能锁功能(iOS 和 Android)

已经切换到 Google Home 作为他们主要控制应用的 Nest 用户将会很高兴地得知应用正在获得一些专门针对 Nest 产品的便利功能。您将能够使用它来:

*   **从摄像头页面查看所有嵌套摄像头。**如果你在家里使用多个 Nest 摄像头，你现在可以在 Google Home 应用程序中同时查看所有直播，类似于 Nest 应用程序过去显示的内容。要一次查看你所有的 Nest 相机，打开 Google Home 应用，点击你家名字下的**【相机】**选项。
*   **解锁智能锁(即将推出)。** Nest 智能锁用户将很快能够在谷歌 Home 应用内从远程解锁他们的设备。谷歌用弗莱德·阿米森 主演的一个短剧 [戏弄了即将到来的功能，并在 iOS](https://www.youtube.com/watch?v=AAMcFZwv-2k) 上 [Google Home 的 2.27 版本补丁说明中提到了它，但它尚未在安卓应用](https://apps.apple.com/us/app/google-home/id680819774) 中 [可用。很有可能谷歌只需要扳动一下开关，这有望很快实现。](https://play.google.com/store/apps/details?id=com.google.android.apps.chromecast.app)

### Android 11 的电源菜单支持更多设备

[Android 11](https://lifehacker.com/the-13-best-new-features-of-android-11-1844019029) 让你从 Google Home 电源菜单控制联网的智能家居设备。2.27 版本将电源菜单支持扩展到所有 Nest 和 Chromecast 设备的智能，包括内置 chrome cast 的产品。

要将设备添加到电源菜单:

1.  打开 Google Home。
2.  转到电源菜单。
3.  轻按您住宅名称旁边的菜单图标
4.  轻按**“添加控件”**
5.  选择要由电源菜单快捷方式控制的设备。(如果设备是通过手机上的另一个应用程序控制的，请点击**“查看其他应用程序。”**)
6.  点击**“保存”**
7.  返回电源菜单，短按设备/组的磁贴将其打开/关闭,或点击菜单顶部的快捷方式之一，对所有适用的设备执行该操作。

### Android 上的黑暗主题

据 Android PoliceT3[报道，Google Home 黑暗主题也是 2.27 版本更新的一部分。似乎你只能通过打开你的 Android 设备的系统级黑暗主题来启用它——没有应用内切换(至少现在没有)。](https://www.androidpolice.com/2020/08/25/google-home-app-v2-27-finally-gets-a-dark-mode-apk-download)

Android 的黑暗模式设置可以通过进入**设置>显示>黑暗主题**来更改，或者通过在您设备的设置应用程序中搜索“黑暗主题”(它的位置可能因手机和 Android 版本而异)。
