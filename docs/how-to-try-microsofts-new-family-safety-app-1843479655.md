# 如何试用微软新的家庭安全应用程序

> 原文:[https://life hacker . com/how-to-try-Microsoft-new-family-safety-app-1843479655](https://lifehacker.com/how-to-try-microsofts-new-family-safety-app-1843479655)

微软在三月份发布了新的家庭安全应用程序，你现在可以在 iOS 或 Android 上注册该应用程序 的 [有限预览版。该应用背后的想法是让父母们一瞥他们的孩子是如何在他们的设备上花费时间的，这样他们就可以根据需要做出调整。应用程序还允许家庭成员更容易地分享他们的位置并保持联系。](https://www.microsoft.com/en-us/microsoft-365/blog/2020/05/11/new-microsoft-family-safety-app-now-available-preview-ios-android/)

Watch

如果你想出去兜风，你可以现在就从 Google Play 或 App Store 下载应用程序。你还必须 [在微软网站](https://aka.ms/FamilyForm2) 上注册有限预览，并等待批准。如果你进入官方预览版，你将可以做很多事情:

### 监控您孩子的智能手机活动

家庭安全的一个很酷的地方是它可以监控多种设备上的活动。这意味着你不仅可以看到你的孩子如何使用他们的智能手机或平板电脑，还可以看到他们在电脑(或 Xbox)上做了什么。

作为家长，您可以看到孩子的总屏幕时间，以及他们访问的热门网站和他们在网上搜索的内容。你可以在应用程序中查看他们的活动，或者注册一个每周电子邮件总结你的孩子一直在做什么——可能是《我的世界》。总是《我的世界》。

### 为沉迷于设备的孩子设定屏幕时间限制

通过 Microsoft 家庭安全应用程序中的屏幕限制功能，您可以对特定类型的活动设置限制。例如，您可以限制孩子在他们喜欢的游戏网站上花费的时间，但允许他们无限制地访问他们用于学校作业的门户网站。(他们会喜欢那个的。)

如果您的孩子用完了您分配的时间，他们可以向您申请更多时间。你是否同意这个请求取决于你——以及他们那天修剪草坪的情况。

### 控制你的孩子对应用和网站的访问

该应用程序还允许您阻止您的孩子能够访问的应用程序、游戏和网站(如果他们使用的是微软的 Edge 浏览器，则为后者)。如果您的孩子试图下载比您允许他们访问的应用程序更成熟的应用程序，您也会收到通知。对不起，*末日永恒*的七岁粉丝。

### 找出你的家人在哪里

与其他位置共享应用 的 [类似，家庭安全也允许你知道你的家庭成员在任何给定的时间在哪里——或者至少知道他们的设备在哪里。](https://lifehacker.com/how-to-share-your-location-in-whatsapp-and-other-apps-1819658511)

你还可以在地图上保存经常去过的地方，比如工作和学校，这样你就可以快速确定某人在哪里。
