# 现在不是买曼陀林切片机的时候

> 原文：<https://lifehacker.com/now-is-not-the-time-to-buy-a-mandoline-slicer-1843132799>

如果你在 Twitter 上搜索“mandoline slicer”这个词，你会看到大量的广告，一些称赞它是“游戏改变者”的推文，以及许多悲伤和肢解的故事。曼陀林不是厨房玩具，你看，它们是厨房武器。



是的，他们可以很快将几乎任何东西切成漂亮的、像纸一样薄的、均匀的薄片。但是除非你已经用了好几年了——并且使用了防护装置或防割手套——否则现在就是使用的时候了。你会被它的高效、优雅的本质所吸引、抚慰和满足，而且，就在你觉得自己“掌握了窍门”的时候，你要么卸下防备，要么放松警惕，灾难就会降临。这不是*如果*你会砍自己的问题，这是*什么时候的问题。*精明的人最终总是会赢，可预防的厨房事故并不是这个国家的医院此刻所需要的。

你问为什么会突然爆发？我有两个原因，一个是公共原因，一个是个人原因。几天前，食品推特在下面的推特上爆发了，许多美食作家和厨师谴责任何现在甚至敢看这样一个锋利、危险的工具的人。
