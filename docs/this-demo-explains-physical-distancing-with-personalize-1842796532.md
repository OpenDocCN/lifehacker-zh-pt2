# 这个演示解释了个性化化身的物理距离

> 原文:[https://life hacker . com/this-demo-explains-physical-distanding-with-personalize-1842796532](https://lifehacker.com/this-demo-explains-physical-distancing-with-personalize-1842796532)

我们希望你呆在家里。但是，当你花这么多时间看着你自己的四面墙时，很难接受这样一个事实:你没有做的事情——把孩子送到日托所，去健身房锻炼——才是真正有所作为的。魁北克拉瓦尔大学的 [这个小演示](http://wlab.fmed.ulaval.ca/projects/socdist/#/) 可能会有所帮助:它说明了物理距离是如何工作的，并在这个过程中添加了一张脸(实际上是九张)。

它基本上只是两种不同场景的动画:一种疾病在有和没有隔离措施的情况下传播。在观看之前，您必须自定义一个头像来代表您自己。然后你被要求为你认识的人，或者你在社区中与之互动的人创建头像。我添加了一些我很少见到的家庭成员，托儿所的人，健身房的人。

当旁白解释正在发生的事情时，标记您创建的每个人的六边形会变成红色(对于严重的情况)或粉红色(对于较轻的情况)。演示没有明确地杀死任何人，但其含义足够了:我们可以看到当每个人都在同一个网络中时，疾病是如何传播的，然后当人们被分成小岛，相互之间的互动最小化时，会发生什么。

作为奖励，还有一个对 [群体免疫](https://offspring.lifehacker.com/a-visualization-of-how-herd-immunity-works-and-why-vacc-1797021621) 的直观解释，这是我们希望未来疫苗实现的目标(以及我们已经拥有的其他疫苗可预防的疾病，如麻疹和水痘)。只需几分钟就可以完成演示，所以请花时间去探索它——或者将它展示给最近的喜欢创建 Mii 头像但不明白为什么他们现在不能看到他们的朋友的孩子。
