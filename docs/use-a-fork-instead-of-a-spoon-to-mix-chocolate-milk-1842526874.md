# 用叉子而不是勺子搅拌巧克力牛奶

> 原文：<https://lifehacker.com/use-a-fork-instead-of-a-spoon-to-mix-chocolate-milk-1842526874>

一杯又高又冷的巧克力牛奶对年轻人和老年人来说都是很好的款待，但是大多数人用勺子搅拌它，我很遗憾地说，勺子是一种劣质的搅拌工具。你会问，正确的工具是什么？为什么，一叉当然！

Watch

我的意思是，当然，勺子会让你到达那里，但是叉子会让你更快到达那里。这样想:你试图让巧克力——无论是粉末还是糖浆——尽快形成一种溶液。一把有许多尖齿的叉子，通过在牛奶中制造几个(很可能是四个)小漩涡，几乎像一个微型的搅拌器。另一方面，勺子创造了一个单一的，缓慢的，懒散的漩涡。选择是明确的。

除了更快地将巧克力融入牛奶中，叉子还能给饮料充气，让它有一种很好的轻盈质感。所以，省下喝汤的勺子，享受更均匀、更有效的巧克力牛奶吧。
