# 阅读维基百科中关于“红衣女子”的新条目

> 原文：<https://lifehacker.com/read-new-wikipedia-entries-about-women-with-women-in-re-1832329713>

维基百科，一切的百科全书，大部分是关于男人的。根据维基百科内部改进项目 [红衣女子](https://en.wikipedia.org/wiki/Wikipedia:WikiProject_Women_in_Red) 的数据，该网站 150 万个传记页面中只有不到 18%是关于女性的。这个数字实际上代表了一种进步:在 2014 年末，这个数字是 15%。这种进步很大程度上要归功于《红衣女郎》本身，该网站成立于 2015 年，旨在增加和改善网站对女性的覆盖。



当然，大多数有文字记载的历史都与男性有关，因为有记载的历史中，大多数人都生活在父权制社会下。所以他们有更多的原始资料。但这并不能完全解释维基百科的偏见。 [正如《新政治家报》](https://www.newstatesman.com/politics/feminism/2019/01/chinese-spies-award-winning-geologists-we-re-making-women-visible) 所报道的，主要文章都遗漏了杰出的女性。例如，《化学史》提名了 200 名男性和 4 名女性，忽略了诺贝尔奖得主代谢研究人员 [格特里·柯里](https://en.wikipedia.org/wiki/Gerty_Cori) 和 CRISPR 研究人员詹妮弗·杜德纳(Jennifer Doudna)，根据她自己的维基百科页面， [“在生物化学和遗传学方面做出了根本性的贡献。”](https://en.wikipedia.org/wiki/Jennifer_Doudna) 并不是没有更多值得报道的女性；而是他们被忽视了。

红衣女子(以红色文本命名，表示链接到一个不存在的维基百科条目)正在与这种情况作斗争。该项目组织人们致力于维基百科上关于女性的条目。当维基百科贡献者创建或显著改进一个条目时，他们可以将其添加到项目列表中。一月份的名单包括 182 个条目，其中大部分是新的，涵盖了现在和过去的女性。大部分还是空头。但是如果你看看他们的页面历史，你会看到一些令人鼓舞的东西。

一旦有人创建了一个条目的存根，通常会看到其他人进来添加更多的支持材料，或者将条目放入现有的维基百科类别，或者清理一些格式。这些新条目不仅仅是被抛弃和忽视；他们正在进入生态系统，在那里更多的维基百科编辑可以改进他们。这一点很重要，因为绝大多数维基百科的编辑都是男性 。

《红衣女子》鼓励编辑宣传他们的贡献，如果他们在社交媒体上分享了新的贡献，就在项目页面上做标记。该项目的首页列出了被提名为特别功能的文章，如维基百科的“你知道吗”部分，以及被提名删除的文章。这样，项目成员可以捍卫，促进和改善彼此的作品，以确保他们在网站上赢得一个真实和永久的位置。

作为读者，你可以给《红衣女子》做书签，找到有趣的新词条。你会注意到世界上最大的百科全书直到最近都有哪些令人惊讶的遗漏。当你准备好了，你可以添加你自己的贡献。红衣女子有 [一个分类列表，列出了成千上万还没有维基百科条目](https://en.wikipedia.org/wiki/Wikipedia:WikiProject_Women_in_Red/Redlist_index) 的女性——尽管在网站的其他地方提到过。
