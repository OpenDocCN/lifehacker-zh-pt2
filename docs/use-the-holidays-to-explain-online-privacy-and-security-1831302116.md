# 利用假期向家人解释在线隐私和安全设置

> 原文：<https://lifehacker.com/use-the-holidays-to-explain-online-privacy-and-security-1831302116>

本周末*华尔街日报* [刊登了一篇报道](https://www.wsj.com/articles/33-mostly-free-ways-to-fix-your-familys-tech-problems-11545573601?mod=e2tw) 列举了许多不同的方法，在这个假期你可以免费解决你家的技术问题。列表中有很多好的想法，但是引起我注意的是关于隐私和安全的部分。我的家人很擅长打电话给我，问一些关于如何操作智能手机或“修理”网飞的基本问题，但他们从来不会问关于隐私和安全的问题——他们应该问。

Watch

利用假期查看家人在脸书和其他应用程序上的隐私设置，并确保他们了解他们选择的设置的含义。取决于你的家人，他们可能没有意识到有隐私设置到开始。如果奶奶想与全世界分享她的帖子，尽一切办法，但要确保她知道她已经设置为与所有人分享，而不仅仅是她的朋友。

现在和你的家人讨论密码也是一个好主意。确保他们使用的每个网站和服务都使用不同和独特的东西，如果有意义，看看是否可以将它们转移到密码管理器以确保安全。

虽然我们都容易受到黑客和在线安全问题的影响，但老年人往往不太了解如何解决这些问题。利用假期来讨论这个话题，可以让你免去事后处理事情的头疼，让每个人都放心，他们在安全可靠地操作网络。

《华尔街日报》*文章 的其余部分也提供了一些其他的好建议，比如向你的家人展示如何让他们的手机字体变大，让每个人的手机和电脑都安装最新版本的操作系统，为他们经常使用的东西设置快捷方式。* 

*想想看:你现在做得越多，你在 2019 年需要的技术支持就越少。*
