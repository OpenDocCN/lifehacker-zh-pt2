# 如何使用 Pinterest 虚拟试戴口红

> 原文:[https://life hacker . com/how-to-virtual-try-on-lipstick-using-Pinterest-1841418855](https://lifehacker.com/how-to-virtually-try-on-lipstick-using-pinterest-1841418855)

如果你想买一款新的口红，现在你可以在 Pinterest 上试试。Interest 本周推出了一款新的试妆工具，可以让你足不出户就可以带着不同颜色的口红出去转一圈。

Watch

该功能类似于欧莱雅、丝芙兰和 T2 莎莉美容等品牌的应用程序已经提供的功能，但有一个关键区别:当它添加口红时，它不会在你的脸上应用滤镜。

如果你尝试过其他选项，你可能会注意到，除了不同颜色的嘴唇，它们都在你的脸上放了一个滤镜，消除不完美，让你看起来像是化了妆，而不仅仅是不同颜色的嘴唇。如果你正在自拍，这可能是你想要的，如果你真的想看看你的唇色会是什么样子，那么这个功能可能不会很棒。

Pinterest 的试穿工具只能通过 iOs 和 Android 应用程序获得。目前包括雅诗兰黛、丝芙兰、bareMinerals、露得清、欧莱雅、NYX Professional、YSL、兰蔻和 Urban Decay 的口红，据报道还有更多合作正在进行中。

要使用它，打开 Pinterest 的智能相机，然后点击“尝试”来浏览当前可用的所有口红选项。你也可以通过搜索“红色口红”来找到口红选项当你找到你喜欢的东西时，你可以把它保存到你的板上以便以后访问，或者直接从应用程序中的立即购买链接购买。

这当然不是一个万无一失的解决方案来找出口红颜色在你身上的样子，但它可以是一个很好的方法来了解你正在考虑购买的消防车红色是否有可能在你穿上它后获得成功或失败。
