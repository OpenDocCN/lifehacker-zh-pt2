# 如何在使用优步时保持安全

> 原文：<https://lifehacker.com/how-to-stay-safe-while-using-uber-1840037128>

尽管方便且界面友好，但像优步这样的拼车应用仍然存在一定的安全风险。根据其有史以来第一份公共安全报告 ，优步表示，去年美国共报告了 3045 起性侵事件，比 2017 年的 2936 起略有上升。



虽然这些数字可能只代表优步整体骑行的一小部分——该公司报告称，2017 年至 2018 年期间，美国发生了 23 亿次骑行——但近 6000 起性侵报告的总数令人震惊；这个统计数据也没有考虑到美国的优步骑行

“这些数字令人震惊，难以消化，”优步首席法律顾问托尼·韦斯特上周告诉《纽约时报》 的记者。"它说的是优步是它所服务的社会的反映。"

在最近的过去，优步启动了几项努力来改善其安全实践，包括对司机 进行更多的 [彻底的背景调查，以及在紧急情况下你应该知道的几项安全功能。](https://money.cnn.com/2018/04/12/technology/uber-safety-update/index.html)

### 如何使用优步的安全功能

首先，如果你的车突然停了一段时间，优步会对司机和乘客进行“检查”。优步的安全报告写道:“这项技术主动检查骑手和司机，看看一切是否正常，如果需要，应用程序提供了他们可以用来获得帮助的工具。”。

对于乘客来说，如果你将优步通知设置为出现在应用程序的屏幕上，这可能会显示为一个提示；如果没有，提示将出现在您乘坐的“共享行程状态”选项下方。从这里，你可以选择拨打 911 应用程序(无论是通过电话还是短信，将你的位置发送给 911 调度员)，联系优步的安全热线，报告事故或与朋友分享你的乘车状态。

如果您没有看到此提示，或者需要立即联系某人，这些功能中的大多数也可以在您的骑行过程中通过“安全工具包”访问，该工具包在您的骑行地图上显示为蓝色盾牌。在这里，你会发现一组更有限的选项，包括优步在线“安全中心”的链接(在那里你可以了解更多关于其政策的信息)，与朋友或家人分享你的位置和旅行状态的选项，或在应用程序中拨打 911。

### 保持安全的其他策略

然而，除了优步的工具之外，在谈到拼车应用时，实践其他一般安全提示也很重要(对司机和乘客双方而言)。这是不言而喻的，但总是，总是通过交叉检查车辆的牌照号码，品牌，型号和颜色来验证出现在你面前的汽车。你也可以在上车前询问司机的名字，他们要接谁，以及你计划的目的地。如果你在优步的游泳池里，你也可以在上车前验证车里的其他乘客；在应用程序上，你可以通过滚动到你计划乘坐的底部来查看可能与你同行的乘客的姓名。

始终通过优步的信息平台保持联系，该平台使用匿名电话号码来保护您和您的司机的隐私。如果你想更加安全，你也可以使用优步的安全工具包，从一开始就分享你的位置。在这里，您可以将人添加到您的“可信联系人”中，这允许您向最多五个您选择的人发送您的乘车状态。
