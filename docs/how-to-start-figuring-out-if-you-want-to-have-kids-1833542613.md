# 如何开始决定你是否想要孩子

### 打破循环

根据大卫曼的说法，许多在要不要孩子的问题上犹豫不决的人陷入了一个反馈循环。所有的问题和焦虑不停地盘旋，从担心一个婴儿会如何影响他们的睡眠到他们的关系，到更大的对气候变化和这个可怕的世界的恐惧。大卫曼做的第一件事是让人们试着停止这种内心独白:

> 首先要做的是后退一步，选择不知道，让这一切变得无所谓。从优柔寡断的循环中喘口气，这种循环会让你一事无成。

她告诉《华盛顿邮报》,她让病人写下所有这些产生焦虑的问题，然后藏起来一段时间:

> “你把它们都放在一个信封里，”大卫曼说。“这些确实是重要的问题，但我们现在不想谈论它们。当你过早地考虑所有这些外部因素，而不知道你想要什么以及你为什么想要它时，它们只会碍事。”

恐惧和焦虑不是做出改变人生决定的地方。

### 将欲望与行动分开

决定要一个孩子有许多不同的因素，但人们通常试图同时解决两个最大的问题:你要不要孩子，以及你如何让他们发生。后一个问题是关于物流的；前者是关于欲望的。大卫曼说你需要分别看待它们:

> 最大的问题是试图弄清楚你想要什么，同时又想做什么。这两者必须分开。当你花时间去发现你的渴望时，这个决定必须被搁置。因此，首先要做的是后退一步，写一些东西来确定你可能需要暂时搁置的东西，以便弄清楚自己想要什么。

这在某种程度上也是一个循环。如果你没有一段坚定的感情，或者你在担心钱，或者你有一个五年计划，你可能会陷入思考生孩子的困难，然后再问自己是否想要一个孩子。这只会加剧大卫曼警告的焦虑。

### 如何进行区分

大卫曼带领人们进行的一个练习是为了发现哪些担忧真正与生孩子的计划有关，所以你可以在探索你的愿望时试着把它们放在一边一段时间。

> 1)我一直认为现在我的生活应该是…
> 
> 然后读你写的东西，写下看你写的东西的感受。
> 
> 2)我在做决定时最害怕的是……
> 
> 3)然后列出你生活中不断重复的外在事物。
> 
> 4)然后把这些都拿走，放在一边。
> 
> 5)然后花时间让自己不知道，不做判断，不试图让事情发生，不试图弄清楚事情。

在她网站的一篇博客文章中，大卫曼写道，“”是可怕的；不去纠结生孩子的实际意义，感觉很不自然。更重要的是，它迫使我们从内部审视自己，这是很多人不愿意做的事情。

做出坚定的选择不会让所有这些问题神奇地消失，但它会帮助你理解哪些关于孩子的焦虑来自一个可以解决的问题，哪些来自真正不愿意要孩子。清晰总是有助于照亮道路。
