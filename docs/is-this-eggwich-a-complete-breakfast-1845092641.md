# 这是全套早餐吗？

> 原文：<https://lifehacker.com/is-this-eggwich-a-complete-breakfast-1845092641>

我是一个充满激情的人，我的众多激情之一就是早餐三明治。不管你的价位如何，它们都是美味的稀有食物之一。在白吐司上涂上最薄、最薄的培根和美国奶酪的廉价 BEC，就像在面包店的羊角面包上涂上一个完美的荷包蛋、熏肉和精装切达干酪一样超凡脱俗。面包、鸡蛋、肉和奶酪的组合是一个完美的配方，永远不会让你失望，当这些关键元素中的一个被删除时，我会有点不安。

我们今天尝试的小吃(我们第一次看到是在 [一条推文](https://twitter.com/fillegrossiere/status/1298640757205721093) 中)取消了*面包*，取而代之的是对半煮熟的鸡蛋，形成了无碳水化合物早餐，中间是奶酪和培根。我并不反对整个“用蛋白质代替面包”的概念，但我通常更喜欢蛋白质像裹着面包屑的油炸鸡排，而不是滑滑的鸡蛋。

在这种情况下，去掉面包会让三明治更难拿，产生一个咸咸的、高脂肪的、高密度的小蛋白质炸弹。虽然我自己是一个高盐、高脂肪、高密度的小蛋白质炸弹，但我不喜欢无缘无故地拿走面包——这很棒。除非你是某种 keto caveman(或没有面包)，我真的没有看到这种特殊的 BEC 配置的需要。

话虽如此，这个“三明治”还是让我大吃一惊。我做了一些修改——因为我要做我自己！——而我们的资深视频制作人乔尔·卡恩(Joel Kahn)尝试了最初的推文版本。我跟你说实话——这个鸡蛋的事情带我踏上了*旅程*。观看视频，了解我们的想法、感受和面部表情，以及我所做的(我认为)非常需要的修改。如果你真的想在家里尝试这种方法，确保你带了一杯水。
