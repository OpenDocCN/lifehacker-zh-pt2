# Vivaldi 的新 Android 浏览器如何与竞争对手相抗衡

> 原文:[https://life hacker . com/how-vivaldis-new-Android-browser-stacks-up-against-the-1837991187](https://lifehacker.com/how-vivaldis-new-android-browser-stacks-up-against-the-1837991187)

高度可定制的网络浏览器 Vivaldi 现已在安卓设备 上推出测试版。我们已经测试了 Android 应用程序，发现它是一个非常强大的移动浏览器，具有许多独特的功能，特别是对于 Vivaldi 桌面用户来说，但是在移动设备上拥有另一个伟大的浏览器应用程序会带来一些重要的问题。

Watch

在这一点上，Android 有几乎和大多数桌面操作系统一样多的可行的网络浏览器选择——那么你应该选择哪一个呢？最终的答案是个人喜好的问题，所有的浏览器都有自己的使用案例，但是选择一个移动浏览器并不像在 PC 上那么简单。

在桌面上安装多种浏览器不仅是完全可行的，而且是聪明的； [Chrome](https://lifehacker.com/tag/chrome) 与谷歌的所有其他服务同步，Edge 的[Chrome 版本对微软的产品做了很多相同的事情。](https://lifehacker.com/nows-the-best-time-to-try-microsofts-chromium-based-edg-1837449442) [Firefox](https://lifehacker.com/tag/firefox) 已经迅速成为隐私、定制和速度之间的平衡选择，而 [则勇敢尝试创造一种全新的无广告体验](https://lifehacker.com/get-paid-to-watch-ads-in-the-brave-web-browser-1834332279)[Opera 已经尝试 VPN](https://lifehacker.com/operas-free-vpn-built-right-into-the-browser-rolls-ou-1786935648)等服务和其他安全功能很长时间了。

你可以很容易地在你的电脑上安装并使用所有这些浏览器( [和更多的](https://lifehacker.com/the-best-browsers-for-ad-blocking-that-arent-chrome-1835130559) )，但是在移动设备上要复杂一些。由于智能手机更有限的控制和简化的功能，许多浏览器应用程序都试图成为你的一体化解决方案，大多数用户每次只需要安装一个——也许两个。但无论你是一个手机浏览器一夫一妻制者还是喜欢玩游戏，Vivaldi 已经是一个具有独特功能集的强有力竞争者。

尽管还处于测试阶段，这款手机应用已经包含了许多桌面版的最佳功能，还为现有 Vivaldi 用户提供了一些便捷的快捷方式。以下是 Vivaldi 的 Android 应用程序中的一些最佳功能，包括一些你在任何其他移动浏览器中都找不到的功能。

### 快速拨号

Vivaldi 中的快速拨号很像手机联系人中的快速拨号:它们是您最喜欢和最常用的网站的快捷方式，您可以从主页或新标签中轻松访问它们。它们基本上是书签网站的网格，但是它们可以被管理和组织以便快速访问。

### 搜索引擎昵称和自动填充

最酷的功能之一是搜索引擎昵称。Vivaldi 为各种搜索引擎指定了一个默认的单字母“昵称”，比如“b”代表 Bing(这是默认的搜索引擎)或者“g”代表 Google。在地址栏中输入搜索查询前的昵称(例如“g Lifehacker”)，无需更改首选引擎或转到其主页即可快速搜索。

要找到这些快捷方式，打开 Vivaldi 应用程序，点击右上角的“V”图标，然后进入**设置>通用>搜索引擎。**

### 笔记

当你浏览时，你可以打开“笔记”标签来保存想法或记下想法，保存链接，或复制/粘贴文本，所有这些都可以在 Vivaldi 应用程序中完成。最好的是，你所有的维瓦尔第笔记将在你所有的设备之间同步。虽然你不会发现与 Evernote 或 OneNote 等应用程序完全相同的功能，但能够在单个应用程序中浏览时做笔记是非常方便的。

点击左下角的“面板”图标，然后点击记事本图标，即可找到笔记。

### 加密交叉同步

如果您已经是 Vivaldi desktop 用户，您可以在手机上登录您的帐户，并同步您的数据和帐户偏好设置，包括书签、密码、保存的支付信息、备忘录、快速拨号、设置等。—您的设备之间。您甚至可以访问您当前登录的其他 Vivaldi 浏览器中打开的标签。您的所有账户数据也使用端到端加密进行保护——您知道我们有多喜欢这里的数据安全选项，所以我们很高兴在 Vivaldi 的 android 应用程序中看到这一点。

### 安卓上该不该用维瓦尔第？

我们绝对鼓励 Android 用户给 Vivaldi 一个机会，如果他们很好奇，想亲自看看上面的一些功能。我发现它反应灵敏，能够在我使用该应用的几个小时内快速加载页面，但快速拨号等定制功能最有可能让 Vivaldi 从竞争对手中脱颖而出。如果你想要最好的维瓦尔第体验，我绝对推荐将它与网页版配对。

然而，正如您所料，移动和桌面版本之间并不是一对一的过渡。一些桌面功能是 Android 版本所没有的——比如调整 UI 元素的大小或标签堆叠——但同样，该软件仍处于测试阶段，因此未来可能会添加更多功能(尽管我们不能肯定)。说到测试版，用户在 Android 上使用 Vivaldi 的早期版本时可能会注意到稳定性问题。维瓦尔第要求用户报告任何重大错误，并在谷歌商店页面 的 [评论中留下其他反馈。](https://play.google.com/store/apps/details?id=com.vivaldi.browser)
