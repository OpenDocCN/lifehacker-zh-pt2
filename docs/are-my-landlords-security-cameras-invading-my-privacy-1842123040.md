# 我房东的监控摄像头侵犯了我的隐私吗？

> 原文：<https://lifehacker.com/are-my-landlords-security-cameras-invading-my-privacy-1842123040>

家庭安全？超级重要。隐私？也很重要。当两者发生冲突，有人开始使用他们的家庭安全摄像头侵犯他人的隐私*和*同时保护他们的包裹免受小偷的侵犯，会发生什么？



而且，不，我说的不是 [环](https://lifehacker.com/how-to-get-the-most-out-of-rings-new-privacy-features-1840832191) 。我说的是和室友一起生活，或者从房东那里租房子的棘手问题，房东在你家或公寓周围安装了安全摄像头。如果你允许在本周的 Tech 911 专栏中有一点分歧，我通常在那里回答*你的*问题，我想分享一些从我自己生活环境中最近发生的一件事中学到的东西。(如果除了搬出去之外，你还有什么建议给我——我会洗耳恭听！)

事情是这样的:我住在硅谷一个相当安全的角落，从我的一些朋友那里租了一间房子。他们在房子的四周安装了摄像头——主要是外部——以增加额外的保护，防止潜在的小偷、肇事逃逸(前室友的车就发生过这种情况),谁知道还会发生什么。

我通常觉得除了门廊摄像头之外的任何东西都有点过分，我的房子前面有摄像头，后面有摄像头，车库里有摄像头——你能想到的，我们可能都在记录。这些录音都被我的室友/房东，房子的主人和常驻技术专家保留了一段未知的时间。

这就是我感到有点困扰的地方。我不喜欢被跟踪的感觉，我的朋友或未婚夫也不喜欢这些摄像头让人们知道他们什么时候过来，呆多久，甚至更糟的是，记录他们在摄像头附近说的任何话。这感觉是侵犯性的，这种设置的复杂性是我搬出去的部分原因。

除了走一个没有任何实际意义的烦人的法律程序(而且感觉毫无意义，因为我要离开)，我绝对没有办法改善这种情况。然而，我确实从这个有用的设置变成跟踪者的系统中得到了一些东西，我认为在你下次搬到一个新的地方或与一个新的室友配对时值得考虑。

### 要求共享家庭安全摄像头

这是一个显而易见的问题，但是在发生之后，这可能是一个很难的话题。如果你要搬进某人的房子，他们使用安全摄像头、网络摄像头、门铃摄像头或其他疯狂的监控解决方案来提高该位置的安全性，那么你应该要求(如果不是礼貌地要求作为租赁条件)查看摄像头看到和记录的内容，这才是公平的。虽然你可能永远不需要或不想看这些视频，但知道你可以和你的室友或房东找到相同的视频或录像，这将使你们两人处于平等的地位。那不会*阻止*他们监视你或者跟踪谁来谁往，但是如果你也能确切地看到他们和摄像机能看到的东西，那就不会那么令人毛骨悚然了。

### 让你的室友或房东带你去看相机

不管你是否能从你家周围的安全摄像头获取视频(或视频),你至少应该知道这些设备在哪里。虽然怀疑，也许你的室友或房东甚至会让你张贴一个小标志，提醒客人注意这些录音设备。显然，门廊摄像头很容易被发现，但你和你的客人应该知道摄像头是否部署在房子的其他地方——车库、走廊、后院活动区、厕所上方等等。

### 如果你的室友或房东控制了家庭网络设置...

这是最难的部分。你的室友或房东可以向你展示你房子或合租公寓周围显而易见的摄像头，甚至允许你查看这些镜头，但没有什么可以阻止他们在其他地方安装摄像头，并且*不会*告诉你。我知道，我们现在进入了令人毛骨悚然的詹姆斯·邦德的领域。如果你不信任你的室友或房东，*你为什么要和他们住在一起？*

也就是说，我非常支持生活中的平等。不应该让一个人主宰家庭网络配置。请求访问路由器(或者任何你更复杂的设置)，你可以自己检查它是否给你*看不到*的房子周围的摄像头分配了 IP 地址。(如果这不起作用，或者相机不只是在你的操作系统的网络窗口弹出，总是有 [愤怒的 IP 扫描器](https://angryip.org/) ...)

是的，你的室友或房东可以设置一些封闭的网络解决方案来记录你在家庭网络之外做了什么。如果是这样的话，你可能已经有其他迹象表明他们是精神病患者。然后你应该搬出去。就现在。

### 假装你在 Airbnb

当你搬进去的时候，你有了自己的空间，考虑一下切换到“[”Airbnb 模式](https://lifehacker.com/how-to-find-hidden-cameras-in-your-airbnb-1829779293) ”，像夏洛克那样四处看看，看看你的房子或公寓周围有没有多余的摄像头。很有可能你不会找到任何东西，如果你找到了，希望它只是在公共休息室里记录你的猫或狗白天做了什么。(剧透:他们睡觉。)

### 做一点法律研究

法律因地点而异，但一般来说，你的室友或房东不应该把摄像头放在你有合理隐私预期的地方——比如你的房间或浴室。房东*可以*安装户外监控摄像头，但如果他们在捕捉音频，这在技术上是否违反了你们州的 [窃听法](https://thewirecutter.com/blog/security-cameras-ethics-and-the-law/) ？我会检查。如果房东不是用这段视频来检查丢失的包裹，而是用它来追踪你的下落，这可能属于租户骚扰。

如果你真的担心你的房东或室友的做法，我建议你记录下你与他们在房子周围安装的摄像头的任何互动。如果你能获得任何证据，证明他们可能以不完全光明正大的方式使用这些摄像头，那就更好了。否则，我不会倾家荡产，打电话给你能找到的最近的当地律师，但也许有一家专门从事租户权利的法律公司愿意为你提供快速、免费的咨询——可能会告诉你，你没有多少法律依据，但你永远不知道。

如果你是房客，而不是房客，情况会有所不同——当你的房东也住在某个地方时，你可以从这个地方的房子或公寓里租一个房间，而当房东不住在这个地方时，你就可以租一个房间。如果这是你的交易，那么被定义为“合理的隐私期望”的可能不是你的厨房，而只是你租的单间。如果你真的被生活环境周围的安全摄像头所困扰，仍然值得检查一下，但你能够通过法院(或措辞严厉的信件)解决这一问题的几率会降低一些。

### 试着和你麻烦的室友或房东讨论一下

我把这一步留到了最后，其实它应该是第一步，但我发现，如果你是那种在家里对着摄像头焦虑不安的人，你很可能根本没有机会和你的房东/室友/任何人谈论这个问题。我并不是说这会导致你想要的结果，但也许以冷静、理性的方式表达你的担忧会化解间谍、跟踪和网络摄像头概念可能带来的紧张局势。我从来没有做到这一点，但我希望你能做到，如果你像我一样处于棘手的监视情况下。
