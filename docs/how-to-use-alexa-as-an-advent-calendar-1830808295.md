# 如何使用 Alexa 作为降临节日历

> 原文：<https://lifehacker.com/how-to-use-alexa-as-an-advent-calendar-1830808295>

如果你是一个庆祝圣诞节的人，那么你可能对降临节日历很熟悉。他们背后的想法是，你每天打开一份小礼物，作为真正圣诞节倒计时的一部分。

Watch

当我还是个孩子的时候，我的意思是我父母会买的最好的一个“顶层”版本，包括每天晚上得到一小块巧克力。现在，你可以用威士忌、啤酒、袜子和几乎任何你能想到的当天的“礼物”来买它们。还有一个 Alexa 版本可以在你倒计时时为你提供音频“礼物”。

要访问它，只需每天说“Alexa，打开我的礼物”。Alexa 每天都会有不同的回应，从笑话和有趣的事实到音乐插曲。

也有一些降临节日历 Alexa 技能你可以使用，尽管你必须先在 Alexa 应用程序中启用它们。

为了更有宗教感， [降临节日历](https://www.amazon.com/Andy-Roberts-Advent-Calendar/dp/B01N00XKGK?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-use-alexa-as-an-advent-calendar-1830808295&asc_source=&tag=kinjalifehackerlink-20) 会在降临节的每一天朗读当天的经文，并告诉你这是哪一天降临节。

如果你一直想要所有的笑话， [趣味降临日历](https://www.amazon.com/BJAB-Fun-Advent-Calendar/dp/B077PP3Y4R/ref=sr_1_2?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-use-alexa-as-an-advent-calendar-1830808295&asc_source=&ie=UTF8&keywords=advent calendar&qid=1543786932&s=digital-skills&sr=1-2&tag=kinjalifehackerlink-20) 每天都会给你讲一个圣诞主题的笑话。你也可以指定前一天再听一次你喜欢的笑话。
