# 如果你是单身，情人节去哪里度假

> 原文：<https://lifehacker.com/where-to-vacation-on-valentines-day-if-youre-single-1832588301>

即使你声称不在乎情人节，也很难避免去想它。无论你走到哪里，爱都会扑面而来——所以，也许是时候独自在沙漠中漫步了？

Watch

死亡谷国家公园管理局的柯蒂斯提出了这个建议，他在 Instagram 上写道，这是一年中独自穿越灼热地球的最佳时间:因为情人节。虽然我个人觉得这很好笑，但作为一个单身人士，这也不是一个可怕的想法。如果你已经是一个人了，为什么不去某个地方和大自然交流呢？它让你感受到真正的亨利·大卫·梭罗的光芒。
