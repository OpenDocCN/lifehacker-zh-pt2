# 如果大家都没事，先说那个

> 原文：<https://lifehacker.com/if-everyone-is-okay-say-that-first-1833638145>

如果你曾经接到过这样的电话，开头是“某某人(你爱的人)出事了”，然后是一个曲折的故事，最后以“但他们没事”结束，你就会知道以好消息开头的重要性。

Watch

[Reddit 用户 u/lukeholigan 建议](https://www.reddit.com/r/LifeProTips/comments/b6d337/lpt_if_you_have_to_call_another_person_to_tell/) 如果你知道一切正常，那是开始对话的最佳地点:

> 一开始只告诉他们自己出了事故，会让他们有几秒钟怀疑自己可能已经死了。

如果你打电话告诉我，我的儿子在课间休息时从操场的设备上掉了下来，先告诉我*他没事然后*第二*他摔倒了。*

*也许你有一个健康恐慌，但最近了解到，事实上，这只是一个恐慌，你很好。当你稍后向你母亲讲述这个故事时，以“非常好”为开头。*

*发生了火灾？不太好。但是每个人都安全离开了？先说那个。*

*因为虽然你可能知道一切进展顺利，但新闻另一端的人很可能在想象每一个可能的最坏情况。*

*当然，有时候结果并不好，而 [我们已经得到了如何处理](https://lifehacker.com/how-to-give-someone-really-really-bad-news-1832908009#_ga=2.137518768.1837296093.1553520176-1723114163.1524514905) 的建议。但是如果消息并不都是坏消息呢？从最重要细节开始，从那里开始。*
