# 如何清洁啤酒咆哮者

> 原文：<https://lifehacker.com/how-to-clean-a-beer-growler-1833033148>

我喜欢啤酒咆哮者。我丈夫和我有一些来自我们最喜欢的啤酒厂的啤酒，当我们出去吃饭时，他们轮流跟着我们，这样我们就可以带一些新鲜的生啤酒回家。

我不喜欢 beer growlers 的地方是它们的细脖子和细小的开口，这让它们很难清洗。但是如果你正确地(并且立即)处理它们，它们比你想象的更容易保持干净。

### 马上打扫干净

当你倒完最后一盎司啤酒后，用热水(但不是沸水)冲洗咆哮者几次。如果你的水槽上有一个喷头附件，它可以很好地进入咆哮者的所有缝隙，但不管怎样，立即将热水倒入并四处搅动应该可以达到目的。

### 避免肥皂

肥皂会在玻璃上留下残留物，影响你下一杯啤酒的味道，所以大多数啤酒鉴赏家建议不要使用洗碗液。(这包括把咆哮者放进洗碗机，你无论如何都不应该这么做，因为它可能会被其他盘子碰碎或损坏。)

对于特别脏的咆哮者来说，其他更有效的选择是用水和一点点醋装满它们，或者用水和一点点无味的氧化氯浸泡它们。如果你是一个经常使用咆哮者的人，也许值得投资一瓶 [Star San](https://www.amazon.com/dp/B01467UGN8/ref=psdc_2265897011_t1_B00E5MXGKK?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-clean-a-beer-growler-1833033148&asc_source=&tag=kinjalifehackerlink-20) ，这是一种起泡的免冲洗消毒剂。

如果你有一把软毛刷或者奶瓶刷，也可以帮助清除角落和缝隙中的残留物；但是要避开会损坏玻璃的钢丝刷。

### 让它风干

冲洗后，把你的咆哮者倒过来，支撑起来，这样它就有一些呼吸空间风干。让它完全风干(潮湿的咆哮者是细菌的滋生地)。然后——这很重要——***不要给**戴上帽子。*不要带着盖子存放你的咆哮者。空气需要在咆哮者体内循环，以避免细菌滋生，特别是如果你不经常使用它。把咆哮者藏在一个灰尘或碎片找不到的地方。

### 如果你不马上清理干净

我们都应该在喝完最后一滴酒后，立即正确地清理我们的肚子，但有时我们不这样做。如果你已经喝完了你的啤酒，你没有心情清理(或者你不在水槽附近)，在 [玻璃罐](https://www.glass-jug.com/growlers/caring-for-your-growler) 的人建议把它储存在冷却器或冰箱里。寒冷的温度将有助于限制任何细菌的生长，直到你准备好清洗。

### 如果有疑问，试一试

在你清洗完它之后——在你要重新装满它之前——给它做一次老的嗅觉测试。如果它闻起来很臭，那就是很臭，你需要再次冲洗。如果它是无味的，你应该可以去。
