# 当你目睹了一起肇事逃逸事故时该怎么办

> 原文:[https://life hacker . com/1833038734](https://lifehacker.com/what-to-do-when-youve-witnessed-a-hit-and-run-accident-1833038734)

你在人行道上，突然，一个开特斯拉的司机撞上了一辆停着的车。汽车撞碎了对方的后视镜，留下了碎片。在你有时间处理发生的事情之前，司机已经开走了。回来，你这个肮脏的流氓！你的电荷撑不了多久！

Watch

虽然你不需要追着司机跑(祝你好运),但你可以帮助其他落在后面的司机。以下是当你作为目击者遇到事故时应该做的事情，以及为什么记录所有事情是至关重要的。

### 尽可能多地写下信息

汽车正在疾驰而去！假设你是步行，拿出手机拍一张司机的车或车牌的照片。如果不能，记录下汽车的品牌、型号和颜色(任何关于汽车的细节都很重要)。如果可以的话，也记录下关于司机本人的任何信息，以及事故发生的日期和时间。如果你在开车，在做这些之前，一定要把车停在安全的地方。

在一个 [Reddit](https://www.reddit.com/r/todayilearned/comments/4j48bm/til_hit_and_run_accident_solve_rates_are/) 的帖子里，u/ [Revenant10-15](https://www.reddit.com/user/Revenant10-15) 一位警官，解释了为什么证人是必不可少的。“在我的州，仅仅因为你是一辆车的注册车主，并不意味着你要对这辆车发生的事情负责，”他们写道。有证人证实是谁开的车会对案件有很大影响。

虽然这可能会超出范围，但你也可以获得事故其他目击者的姓名和电话号码。如果这是一起危及他人安全的严重碰撞——比如说，如果被撞的汽车漏油——请拨打 911 处理。

如果司机受伤，*不要*试图移动他们，除非他们有汽车着火的危险，正如建议的那样(移动他们会使伤势恶化)。拨打 911 并听取他们的指示。如果司机已经下车，让他们移到路边坐下或躺下，这样他们就不会受到过往车辆的威胁。

### 留下便条

如果司机不在，写下你能记得的所有信息，包括你的名字和联系方式。

正如杰瑞 建议的那样，如果你赶时间，就写下你的联系方式，并提及你是证人(你可以稍后讨论细节，但仍要记得记下细节，以免忘记！)

你可以把纸条放在雨刷下；如果下雨，把它放在某种自封袋或其他可密封的塑料袋里是个好主意。

如果受损汽车*的司机*在那里，交换联系方式，让他们知道到时候你可以作为证人提供细节。

### 向驾驶员的保险公司提供详细信息

有时，问题司机会要求你向警方提供一份证人报告。理论上你也可以自己写一份报告，但是有时候，最好让司机来处理这方面的事情。

然而，你*将*可能会在几个月后被司机的保险公司联系上(或者如果找到了另一名司机，两家保险公司都会联系上)。很简单:你提供一份关于发生了什么的声明。正如 [Geico](https://www.geico.com/more/driving/auto/car-safety-insurance/what-to-do-if-you-witness-a-car-accident/) 所建议的，只要与你的陈述事实相符即可。

### 发布到本地网站

如果你没有司机的消息(也许纸条丢失了，或者车已经停在那里很久了)，可以在社区网站上发帖，比如[【Craigslist】](http://craigslist.org)或者 [NextDoor](https://nextdoor.com/) ，解释你是事故的目击者。也要提供汽车的品牌和型号，以防邻居认出这辆车。

最终，作为证人，你的工作是尽可能提供更多的信息——有一天你可能会处于车主的位置，你肯定会感谢旁观者为你做了这些事情。
