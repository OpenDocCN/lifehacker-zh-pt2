# 使用这款 Chrome 扩展，快速锻炼休息

> 原文：<https://lifehacker.com/take-quick-exercise-breaks-with-this-chrome-extension-1845096107>

尽管我喜欢从办公桌前站起来做点运动的想法，但我不擅长真正贯彻这些意图。我最近安装了一个名为 [WFH 唤醒](https://www.wfhwakeup.com) 的 Chrome 扩展(也可以作为 Mac 应用程序使用)，让它变得简单。



默认情况下，它每 55 分钟打断你一次(尽管你可以改变间隔)，并提示你做一组三个练习。这款应用售价 4 美元，但有一天的免费试用期，所以你可以使用一天，看看它是否会让你发疯，或者是否值得承诺。

我试过其他应用程序，也试图让你离开你的椅子，但 WFH 唤醒做得特别好。休息前 5 分钟你会收到提醒，然后是 30 秒前。(如果你需要一分钟来完成一项任务，当你收到这些提醒时，你可以推迟休息时间。)还有一个巨大的暂停按钮，如果你要参加一个演示或缩放会议，你可以按下它。

当时间到了，你会看到一个类似于这篇文章顶部的屏幕。没有计时器阻止你回去工作，只是全屏提醒你也许你应该做一些练习——他们给了你三条建议。有时是上半身或下半身的锻炼，有时是伸展，有时是核心动作。你可以把这个机会变成真正锻炼的 [小版本](https://lifehacker.com/if-you-dont-have-time-for-a-workout-try-exercise-snack-1839892614) ，目的是在一天中完成完整的例行程序——或者你可以只做列出的三件事，每件做几次，然后回去工作。
