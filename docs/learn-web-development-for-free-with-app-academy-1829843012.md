# 通过 App Academy 免费学习 Web 开发

> 原文：<https://lifehacker.com/learn-web-development-for-free-with-app-academy-1829843012>

编码学校 App Academy 开设了 [免费在线互动版的 12 周课程](https://www.aaonline.io/) 。这是一笔非常划算的交易，因为该学院在旧金山和纽约的面授课程费用可能高达大学一个学期的费用。在线版本不太涉及直接的人际互动，但它包括在线导师和社区聊天。



乍一看，App Academy Open 有点像流行的免费编码课程 [Codecademy](https://www.codecademy.com/learn) 。入门课程在分屏指令界面和沙盒编码环境中教授基本的编码概念。当学生在他们自己的开发环境中工作时，后面的课程被组织成带有文本说明的自由形式的作业。

与代码学院制的主要区别在于课程设置。App Academy 创始人 Kush Patel 告诉 Lifehacker，虽然 Codecademy 是各种编程的良好资源，但 App Academy Open 专注于“教人们 web 开发技能，让他们获得软件工程工作。”(在面对面的学校，学费与学生毕业后的工资挂钩。)

在介绍了编程之后,“alpha”课程会教授数组和调试等概念。课程的其余部分讲授“全栈”:Ruby、SQL、Rails、Javascript 和 React 各一门课程。每门课程需要一周或两周的全职工作。

该课程还涵盖了科技领域的广泛社会问题:强化人类偏见和歧视的算法等外向问题，以及工作场所中少数民族、女性和 LGBTQ 人群的代表性和剥削等科技社区问题。App Academy 没有将这些课程分流到学生可能会忽略的单独模块，而是将它们贯穿于整个编码课程，将它们作为基本的职业课程。

如果你能在没有学费和面授课程压力的情况下投入 12 周的挑战性工作，那么你将获得价值数千美元的实用编码教育，这是专门为使你有资格从事开发工作而免费构建的。
