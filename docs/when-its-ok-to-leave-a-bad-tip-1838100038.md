# 什么时候可以给小费

> 原文：<https://lifehacker.com/when-its-ok-to-leave-a-bad-tip-1838100038>

虽然世界各地的风俗各不相同，但在美国，给小费是常态。我们知道这一点。我们接受这一点。我们已经看到小费率随着时间的推移而攀升。曾经被认为是慷慨的，现在 15%被认为是最低限度，而 20%或 25%的小费正在成为标准。



存在代际差异——根据 CreditCards.com 调查的数据，千禧一代倾向于在餐馆给 22%的小费，婴儿潮一代给 17%。但是就像每片雪花都不一样，每个人都有自己独特的小费哲学。

不管怎样，服务人员值得你的小费！他们为了低于最低工资的时薪而努力工作，你不愿意自己做的任务他们都做。他们经常要给其他服务人员小费，这些人不与顾客互动，但对你获得产品或服务至关重要。给别人一点小费或不给小费应该很少，只有在极端的情况下。

但是这个星期我们不是在这里友好相处，不是吗？ 虽然我们可能都喜欢从用自己的血汗钱光顾的商家那里获得一流的服务，但我们也都有过糟糕的经历(我相信你不需要任何鼓励就可以在评论中提交给小费很少的理由，所以我很快就会在那里见到你们每一个人。)

这里有一些情况，你的错误小费——或者完全没有小费——是绝对合理的。

### 你点的咖啡很简单或者是预先包装好的

这是一个简单的方法，但是如果你在咖啡店或柜台服务的餐馆使用信用卡支付，你可能会因给——或更多——提示小费百分比的数字登记系统而感到内疚。

但是如果你所做的只是快速买了东西，然后在柜台付款，你就不需要给小费。

很多时候，这是有保证的。你整个下午都在咖啡馆里用笔记本电脑工作，或者工作人员把你点的菜送到你的桌子上，并在你吃完后收拾你的桌子。

但是，如果你只是从冰箱里选择一份沙拉，或者在路上喝一杯滴滤咖啡，小费是不需要的。

### 你不得不等了很长时间

您不记得上次看到您的服务器是什么时候了。你的水杯已经干了半个小时了。你的饭菜不见了，而在你之后进来的每一桌客人都已经吃完离开了餐厅。

错误时有发生，订单也会被搞糟，但是如果你不得不为你的订单等很长时间，你不应该为留下比平时少的小费而感到难过。如果你点的菜终于来了，而且是冷的，不要因为你脾气暴躁，差点想吃餐巾来维持生计而保持沉默。送回去。要求一些热食。

### 服务器忽略了你的过敏问题

根据我和地球上每一个稍微有点同情心的人的说法，对某人的食物过敏指手画脚 是一种大罪。

一位同事给我讲了一个噩梦般的故事，说的是一家餐馆在你生日那天提供免费甜点。她的男朋友吃了蛋糕，发现里面有花生酱，两人去了医院，因为他有非常常见的花生过敏。下一次他们去餐馆庆祝另一个生日时，她问，“这里面有花生吗？”蛋糕送来的时候。服务器告诉她没有，但是她在男朋友之前试咬了一口。又是花生酱。

是的，说出你对过敏或成分的担忧是你的责任，是的，错误确实会发生。但是，当你的过敏请求被忽视或质疑，或者工作人员不会提供你做出安全选择所需的信息时，这就是一个糟糕的提示的理由。

### 是一个男人递给你一件 t 恤

我最近去了一个中等规模的音乐会，当我经过商品柜台时，我注意到 t 恤架旁边贴着一张手写的便条。“我就像一个酒保，”上面写道。"除了我没有给你做饮料，而是给了你衬衫."

### 员工直接侮辱了你

世界上有太多的人在磨你的齿轮。这些人可能会笨嘴拙舌，摸不清你的盘子，或者摸不清你的交易，但你可以看出他们是好意。

但是带有种族主义、性别歧视、同性恋或排外性质的评论是不允许的。如果你怀疑自己被孤立了，被定型了，那就由你来决定如何处理支票上的小费。

### 如果你正在考虑留下一个蹩脚的小费，先说出来

如果服务太差，以至于你正在考虑少给小费或者干脆不给小费，那就值得和你打交道的人或者他们的主管说出来。有时糟糕的服务是由服务器无法控制的情况造成的，或者他们只是过了糟糕的一天。你应该在留下一笔糟糕的小费之前，给公司一个改正的机会。

服务越差，就越有理由告诉经理你的经历。如果你被当成废物，应该有人知道。当你处于紧张的时刻时，很难用语言来表达你的失望。深呼吸，召唤出你心中的愤怒。明智地使用它。
