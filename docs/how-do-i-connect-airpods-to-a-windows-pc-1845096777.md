# 如何将 AirPods 连接到 Windows PC？

> 原文：<https://lifehacker.com/how-do-i-connect-airpods-to-a-windows-pc-1845096777>

我要为本周 Lifehacker 的 Tech 911 问答稍微作弊一下。我没有像往常一样通过电子邮件收到这个问题，但我*是*一个朋友问的，因为 AirPods Pro 是本周的热门话题，这要归功于苹果的 iOS 14 大更新 / iPadOS 14。



我来解释一下我的朋友，他基本上问了一个简单的问题，这可能是很多人的想法:*我到底如何在 Windows 上使用 AirPods？*

好消息是:这很容易。这是非传统的——或者，至少，它可能会有这种感觉，因为 AirPods 通常是苹果生态系统中最容易配置和使用的。不过这是有可能的，只是不要告诉蒂姆你正在 T2 的另一个平台上试用它们。

首先，请确保您的 AirPods 挂在充电盒中。在你的 Windows PC 上，点击**开始菜单**，调出**设置**应用。点击**“设备”**，然后确保您的 Windows 桌面或笔记本电脑启用了蓝牙。

接下来，点击**“添加蓝牙或其他设备”**，并在出现的屏幕中选择**“蓝牙”**。当你的 Windows PC 搜索新的蓝牙设备时，抓住你的 AirPods 充电盒，打开它，按住背面铰链下方的按钮，直到你看到一个白光开始闪烁。(一旦做了，就放手。)

很快，你会看到 AirPods 出现在可用设备列表中。当它们出现时，点击它们。

他们几乎应该立即连接:

点击“**完成”**退出窗口，然后把你的 AirPods 放在耳朵里转一转。一旦你完成了摇摆，并假设你想再次在你的 iPad、iPhone 或 Mac 上使用它们，你需要进入该设备的蓝牙菜单来重新建立连接。

(如果你有 iOS 14、iPadOS 14 或 macOS Big Sur 和 [最新的 AirPods Pro 固件](https://lifehacker.com/which-apple-devices-and-video-services-support-spatial-1845077927) ，你将能够在你的苹果设备之间“快速切换”，如果你在每台设备上使用相同的 Apple ID。)

要将 AirPods 与 Windows PC 重新连接，请在任务栏中调出蓝牙菜单，点击“显示蓝牙设备”，再次选择 AirPods，然后点击“连接”

当您将 AirPods 连接到 Windows PC 时，您通过 Apple 设备在 AirPods 上启用的任何设置(如噪音消除、透明模式或空间音频)都应该保持不变。你将无法更改这些设置，因为在 Windows 中没有这样做的选项，所以你必须在 Apple 设备上重新连接它们才能进行任何调整。

就是这样！是的，你的 AirPods 是跨平台的，如果你已经在苹果的所有产品上使用它们，那么在 Windows 上使用它们会非常方便。
