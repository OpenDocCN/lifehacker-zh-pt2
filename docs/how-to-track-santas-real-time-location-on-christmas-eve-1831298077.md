# 如何在平安夜追踪圣诞老人的实时位置

> 原文:[https://life hacker . com/how-to-track-Santas-real-time-location-on-Christmas-eve-1831298077](https://lifehacker.com/how-to-track-santas-real-time-location-on-christmas-eve-1831298077)

尽管政府停摆，北美防空司令部今年仍忙于追踪圣诞老人。这项服务从 50 年代就开始追踪圣诞老人在平安夜的行踪，当时它的号码被误认为是圣诞老人热线。

Watch

幸运的是，北美防空联合司令部有技术随时找出雪橇的位置。

虽然以前是镇上唯一的游戏，但现在你也可以获得这些信息和一些其他地方，包括亚马逊的 Alexa 和圣诞老人自己的应用程序。

### 北美防空联合司令部

圣诞老人信息的长“来源”，当谈到使用北美防空司令部跟踪圣诞老人时，你有许多选择。北美防空司令部是你当地新闻站使用的，所以如果你收听广播或看电视，你会得到它的最新更新。

北美防空司令部还有一个脸书主页 ，可以追踪圣诞老人的行动，一条热线，1-877-HI-NORAD (1-877-446-6723)， [和一个网站](https://www.noradsanta.org) ，可以追踪圣诞老人的位置并玩一些游戏。

### 亚马孙

如果你问 Alexa“圣诞老人在哪里”,她会告诉你他现在在哪里，并在之后提供一个额外的有趣的“圣诞老人事实”。

### 谷歌

谷歌的圣诞老人追踪器 让你知道这个大家伙现在在哪里，以及他预计什么时候到达你家。在北卡罗莱纳州我父母的房子里，我们期待着晚上 11 点的到来。

谷歌的追踪器还提供实时数据，比如圣诞老人现在离你有多远，他走了多远，他送了多少礼物。

如果你家里有人对跟踪那辆雪橇超级感兴趣，还有一个专门用于跟踪 的移动应用程序，附带一些跑步游戏。
