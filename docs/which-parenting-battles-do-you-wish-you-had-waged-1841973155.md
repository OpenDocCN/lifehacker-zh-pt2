# 你希望自己发起过哪些育儿战？

> 原文：<https://lifehacker.com/which-parenting-battles-do-you-wish-you-had-waged-1841973155>

有人可能会说，养育子女是一场长达数年的“选择你的战斗”的大型游戏有时候，你真的很清楚规则，并与后果保持一致。有时候，你会说，“去他妈的，我只想活过这一天。”



在我们的 [后代脸书群体](https://www.facebook.com/groups/2018785615043946/) 中，有一位父母正步入蹒跚学步的最艰难时期，他不知道哪些战斗实际上值得进行。凯利问道:

> 有哪些你希望防患于未然的行为？我们正走向三三两两，我担心我会这样，“嗯，那不是很可爱/有趣吗？”然后像这样，“哎呀，也许我不应该鼓励这一点？”

所以，让我们集思广益来帮助父母。你希望你从来没有让他们睡在你的床上吗？或者一周有五天(或更多)晚上不让他们吃鸡块？你认为你应该等到他们稍微大一点再交出 iPad 吗？

告诉我们:哪些育儿战真正值得进行，哪些不值得花费精力？
