# 如果你看到一个孩子做一些好事，告诉他们的父母

> 原文：<https://lifehacker.com/if-you-see-a-kid-do-something-nice-tell-their-parent-1831580706>

你知道当老师挥手让你去学校接机时你会有什么感觉吗？或者当另一位家长在你去取车的路上拦住你说，“嘿，我想你应该知道……”或者，当你抱着一大堆食品走到前门时，一个邻居喊着你的名字说:“我必须告诉你(你的孩子)前几天做的一件事。”



你振作起来了，对吗？你想直接融化在地下，食品杂货和所有的东西，或者用一个响亮的“啦啦啦啦啦”堵住你的耳朵因为接下来发生的肯定不是好事。

但如果是呢？

Reddit 上的一位家长讲述了一个 10 岁邻居小孩的故事，她在儿子放学回家的路上帮助他度过了焦虑。男孩无意中听到她和她儿子谈论他的感受，停下来说:“嘿，我知道你很难过，你的胃也疼。我有同样的事情，我真的希望你感觉好些；我知道有多糟糕。”

几天后，这位母亲遇到了另一个男孩的母亲，并告诉她他是如何帮忙的，事情是这样的:

> 出于某种原因，她似乎不同寻常地受到了这一点的影响，并竭力忍住眼泪。
> 
> 她接着告诉我，Seth 在家里有严重的情绪稳定性问题。他的日常行为是尖叫、抨击、崩溃等等。基本上他在家里是一个完全的恐怖，他们刚刚开始对他进行治疗，她不知道如何处理。然后她说了一件事，这让我想到这里来和大家分享这个故事。
> 
> “我以为我可爱的小男孩永远离开了。知道他仍然在某个地方，即使我永远看不到它，对我来说比你可能理解的更重要。”

评论者用他们自己感觉良好的故事来回应；当他们看到一个孩子做一些特别好的事情并告诉孩子的父母时，几乎总是让父母措手不及。

在一次疯狂的寻找复活节彩蛋活动中，一个孩子和一位患有自闭症的母亲的儿子分享了他的塑料彩蛋。滑板少年帮助一名妇女将包裹搬进 UPS 商店。另一个有焦虑倾向的男孩帮助一个夏令营同伴处理思乡之情。这个孩子在一个小女孩上学的第一天安慰她——然后在那一周每天早上去看望她，看她感觉如何。在操场上帮助小孩子或在飞溅垫上和他们一起玩的大孩子。

这些时刻经常发生在孩子们甚至没有意识到有大人在看的时候。很多时候，善良来自一个孩子，他通常因为行为不端而受到更多的关注，而不是同情。

自从几天前我第一次读到这个帖子，我就一直在绞尽脑汁，有一段时间，另一位父母或一个陌生人这样称赞我的儿子；我想不出一个。

我也绞尽脑汁了一段时间，我特意用这种方式称赞了另一个孩子；我也想不出其中的一个。

识别和关注推搡、辱骂和不听是很容易的。不尊重的语气，插队和不愿意分享。但是我们的孩子也做了很多不为人知或不为人知的好事。

正如 Reddit 用户 TrinhamTales 所说:“作为父母，我相信我们都同意，我们都知道孩子最坏的一面，虽然我们可能想知道他们的不端行为，以便我们可以处理它，但我们真正想听到的是对他们有多好的出乎意料的真诚评论。不是为了让我们在其他父母面前感到特别和优越。而是因为它告诉我们内心深处的一些东西，告诉我们无论事情有多艰难，我们仍然在做正确的事情。”

下一次我看到另一个孩子成为世界上善良的人时，我一定要感谢他们——然后告诉他们的父母。
