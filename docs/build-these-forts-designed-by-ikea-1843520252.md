# 建造这些由宜家设计的堡垒

> 原文:[https://life hacker . com/build-these-forts-designed-by-IKEA-1843520252](https://lifehacker.com/build-these-forts-designed-by-ikea-1843520252)

这些时代是建造堡垒的时代。这是蹲下，蜷缩，让我一个人呆着的时代。但是也许到现在，在疫情的第 10 周，你已经没有新的方法把毯子盖在几把椅子上，称之为堡垒。我当然有。但是宜家俄罗斯公司在这里帮助激发我们的堡垒工程想象力。

Watch

[Ikea Russia 和创意机构 Instinct](https://www.adweek.com/brand-marketing/ikeas-quarantine-campaign-offers-6-ways-to-make-furniture-forts/) 创造了六个堡垒设计，父母和孩子可以在家中重现。这些说明就像你对宜家的期望一样简单，基本上只有最基本的图纸来帮助指导你。但这真的是你所需要的。每个设计至少基于一件宜家家具，但你可以使用任何你已经拥有的类似家具。

我九岁的儿子和我决定试一试。我们仔细检查了每一个设计，把范围缩小到两三个，然后他选择了一个他认为最能保护他隐私的设计。(我发现，拥有堡垒的至少一半乐趣在于建造堡垒。)我们决定采用“洞穴”设计，现在对我来说似乎有点显而易见，但这是我们以前从未尝试过的方法。
