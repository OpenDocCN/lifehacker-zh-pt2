# 发现新音乐的最佳播客

> 原文：<https://lifehacker.com/the-best-podcasts-for-discovering-new-music-1839063483>

除了收音机，我喜欢在任何地方发现我的音乐:播放列表、电视原声音乐、最佳歌曲列表、地铁乐队、TikToks、在酒吧、商店和咖啡店偶然听到的歌曲...和播客。多棒的尝试新音乐的媒介啊。音乐播客就像你最喜欢的广播节目点播——当然，许多最好的只是实际广播节目的重播，有更安静的商业广告，没有古怪的早间 DJ。这些是我最喜欢的节目，用来发现新的和对我来说新的音乐，或者重新发现旧的最爱。



每个节目标题都链接到节目网站和苹果播客页面。

### [水族酒鬼:传输](https://aquariumdrunkard.com/?s=podcast) ( [苹果](https://podcasts.apple.com/us/podcast/aquarium-drunkard-sidecar-transmissions-podcast/id142891498) )

音乐家、喜剧演员和其他艺术家谈论他们喜欢的音乐，演奏一些样本和完整的歌曲。

德文德拉·班哈特最近 [拿出了一些深度剪辑](https://aquariumdrunkard.com/2019/09/20/transmissions-podcast-devendra-banhart-kristin-hersh-bill-orcutt/) ，其中一些你在 Spotify 甚至 YouTube 上都找不到。我已经订购了黑胶 上的《蓝眼威廉 [的 Slapp 欢乐版原版。](https://www.discogs.com/release/1601997)

### [所有歌曲都认为是](https://www.npr.org/sections/allsongs/) ( [苹果](https://podcasts.apple.com/us/podcast/all-songs-considered/id79687345) )

新音乐的采访、讨论和播放，倾向于仍能吸引势利者的流行音乐。主持人有那种典型的 NPR 风格，聪明和蔼，眼光敏锐但品味广泛。

在这个网站上，你可以播放 YouTube 上该节目的歌曲。你还会在“维京人的选择”播放列表中看到更多的推荐，如 [歌剧日本摇滚](https://www.npr.org/sections/allsongs/2019/09/24/763365668/vikings-choice-japanese-prog-rock-heavy-shoegaze-demented-dancehall) 。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-acRDJTYHWIo&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-acRDJTYHWIo&start=0) 

### [在沟里，爵士又超越了](https://jazzandbeyond.podbean.com) ( [苹果](https://podcasts.apple.com/us/podcast/in-the-groove-jazz-and-beyond/id185803769) )

爵士乐的解说很少，每周都有一个不同的主题，并经常向最近去世的爵士乐大师致敬。涵盖了广泛的爵士乐风格，子流派和时代。

剧集主题包括蓝草/爵士乐与贝拉·弗莱克的交叉，13 号星期五的“迷信”歌曲，Bitches Brew 的周年庆典，以及阿波罗 11 号 50 周年的“月亮”歌曲。主持人肯·林泓喜欢寻找爵士乐标准的特别好的版本。参见 [博客上的轨迹列表](https://jazzandbeyond.podbean.com) 。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-wn62VKlw6b4&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-wn62VKlw6b4&start=0) 

### [安妮·麦克的迷你混音](https://www.bbc.co.uk/programmes/p02nrtyg/episodes/downloads) ( [苹果](https://podcasts.apple.com/us/podcast/annie-macs-mini-mix/id268491175) )

DJ Annie Mac 的 [BBC Radio 1 show](https://www.bbc.co.uk/programmes/b006wkh9) 节选，每集由不同的嘉宾 DJ 进行五分钟的 DJ 混音。这些高度设计的混合揭示了旧的舞蹈俱乐部最爱的新细节；我特别喜欢 [上俏皮的引用《谁抽样谁》混音](https://www.bbc.co.uk/programmes/p07d6k33) 。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-wqtYXuNSZJQ&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-wqtYXuNSZJQ&start=0) 

### [宋爆炸器](http://songexploder.net/) ( [苹果](https://podcasts.apple.com/us/podcast/song-exploder/id788236947) )

每一集都是对一首歌曲创作的深入探究，由艺术家讲述。

像安妮麦克，歌曲爆炸是一个伟大的方式来重新发现一首熟悉的歌曲，或欣赏最喜爱的艺术家的新作品。每位艺术家都讨论了他们的歌词、旋律、各种曲目、效果，以及导致这张唱片的每个创作阶段的决定——以及他们本可以采取的其他方向。

当完整的音轨在这一集的结尾播放时，你会挑出细节，你甚至可能会听到那首没有出现的歌曲。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-CfizQsGWOxI&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-CfizQsGWOxI&start=0) 

### 嘉年华 Mix ( [苹果](https://podcasts.apple.com/us/podcast/carnival-mix/id1105616486) )

到处都是欢快的加勒比歌曲。很适合做派对背景。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-RoMMQEOxrNM&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-RoMMQEOxrNM&start=0) 

### [Bamf Radio - Lofi 和 Chill](https://anchor.fm/itsbamf) ( [苹果](https://podcasts.apple.com/us/podcast/bamf-radio-lofi-and-chill/id1290171642) )

没有人声或干扰的墙到墙的凉亭。混音通常会持续 45 分钟左右，有时会更长，并且是非常安静的背景音乐。

这是一种商品化的类型，这是一种被称为高保真嘻哈音乐的冷音乐，是工作/放松/学习的节奏。它在 YouTube 流和 Spotify 播放列表上很受欢迎，播放的都是一首歌的艺术家，一些评论家怀疑这是 [委托的匿名无版权作品](https://www.musicbusinessworldwide.com/spotify-denies-its-playlisting-fake-artists-so-why-are-all-these-fake-artists-on-its-playlists/) 校准以最小化版税。它之所以有效，是因为这种类型的歌曲没有太多的区别:它们都有相同的声音。但那声音很好听。

一些剧集以节目创作者 Bamf 的一首曲目开始或结束，如果你不阅读节目笔记，你永远不会注意到这一点，因为它们与其余的混音配合得很好。Bamf 的专辑 *Dreaming* 出现在所有常见的流媒体上，包括一个完整专辑的 YouTube 视频，其中包含必要的和平动漫艺术。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-VhfYJN6JvOQ&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-VhfYJN6JvOQ&start=0) 

### [索维尔市中心有美酒先生](https://wfmu.org/playlists/SV) ( [苹果](https://podcasts.apple.com/us/podcast/downtown-soulville-with-mr-fine-wine-wfmu/id266091433) )

复古蓝调和灵魂，一小时一集。一些著名的艺术家，如诱惑，雪莉和杰基威尔逊出现在不太明显的行为，如小鲍勃，泰特斯特纳和大奖赛的。

音频质量不可靠，但恰如其分，就像一个老家伙——老到他的软呢帽和墨镜看起来不可笑——让你坐在他的八楼无电梯里听一些唱片。他时不时会递给你一件 1962 年音乐会上的随身物品，并分享一些关于萨克斯手的八卦。

这真的是一个老人在放唱片。该网站有每集的完整播放列表，包括每首歌曲的播放格式——大部分是 7 英寸 45 英寸的。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-MqsJMaE9-jo&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-MqsJMaE9-jo&start=0) 

### [KEXP 呈现重要的音乐](https://kexp.org/podcasts/music-matters/) ( [苹果](https://podcasts.apple.com/us/podcast/kexp-presents-music-that-matters/id76069540) )

西雅图公共广播电台 KEXP 的独立、摇滚和各种其他流派的混合。如果说*所有被认为是*的歌曲是最好的流行音乐的一个很好的调查课程，那么 KEXP 就是建立在这个知识之上，花更多的时间和你还没有听说过的艺术家在一起。

如果你在过去的二十年里一直关注公共电台，你会注意到节目越来越少以西方为中心。之前，“世界音乐”往往有自己的节目区块；现在，它更有可能在西方音乐中受到欢迎，两者更有可能分享(并认可)彼此的声音。最近的*Music the Matters*插曲“土著民族日”呈现了由土著艺术家创作(并受其启发)的摇滚、流行、蓝调和舞曲，每首曲目都令人拍案叫绝。(试举个例子， [【噩】被一个叫做红色的部落](https://www.youtube.com/watch?v=zgsuWSxs1fw) )。)

### [打卡](https://punchupthejam.com)(T5】苹果 )

每一集的大部分时间都致力于一首非常著名的歌曲的吹毛求疵，一秒一秒地，然后表演一场戏仿来“修复”它。这是了解一首老歌的好方法。

但是在这一集的最后，主持人和嘉宾们给他们的“不可 punched jams”命名:已经很完美的歌曲。他们保存了一个运行中的 Spotify 播放列表 ，其中有 255 首来自各种流派和时代的无可挑剔的歌曲。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-qmOLtTGvsbM&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-qmOLtTGvsbM&start=0) 

### [混响电台](https://reverberationradio.com) ( [苹果](https://podcasts.apple.com/us/podcast/reverberation-radio/id520739212) )

墙到墙的播放列表大多背景友好，但仍有特色和丰富多彩的音乐。该网站包括这些深度剪辑和被遗忘的曲目的播放列表。每当你听到两首歌天衣无缝地融合在一起，一首可能是 20 世纪 70 年代的印度单曲 ，下一首是来自荷兰的新歌曲。

没有评论、介绍或打断，这意味着你很容易错过，比如说，这一集“蒂姆·希尔- *帕雅多尔*”是一个独家的完整播放的新专辑，仅在黑胶 上发布 [。每条赛道闻起来都像路边酒吧，有台球桌，Michelob 标志，地上还有花生。你知道是我买的。](https://shop.tikirocket.com/collections/calico-discos/products/tim-hill-payador-lp-first-edition-limited-to-500-copies)

## **优秀奖**

这两个节目已经结束了，它们不完全是关于音乐的，但我已经从每个节目中挑选了一些最喜欢的曲目，你也一样。

### [一个暮色世界的极致平滑](http://twilightsmoothness.com/) ( [苹果](https://podcasts.apple.com/us/podcast/twilight-world-ultimate-smoothness/id991218638) )

音乐频出的六集音频剧迷你剧。

《暮光之城》发生在一个虚构的电台广播中，一个性感的深夜节目，主持人滑稽可笑却不性感。在他与观众、客人和电台管理人员笨拙且越来越敌对的互动中，主持人播放了一些真正的顶级音乐。

### [蓝色果酱](https://fourble.co.uk/podcast/bluejam)

90 年代的音乐穿插着离奇、令人不安的喜剧小品。

英国讽刺作家克里斯·莫里斯创作了《黄铜眼》和《T2》中的《四头狮子》，他在 90 年代末为 BBC 广播一台制作了这个深夜小品和音乐节目。它在午夜播出，尽管莫里斯真的希望它在凌晨 3 点播出

虽然草图有时令人毛骨悚然或令人不快——有一集的 [在播出中被取消](https://www.digitalspy.com/showbiz/comedy/a559343/blue-jam-an-ethereal-mix-of-ambient-music-and-detached-reasoning/)——但音乐却没有。听这个盗版饲料记住或发现一些伟大的 90 年代的替代品。
