# 如何在不允许使用粉笔的健身房使用粉笔

> 原文：<https://lifehacker.com/how-to-use-chalk-at-a-gym-that-doesnt-allow-chalk-1839480265>

粉笔对你的握力帮助很大。举重运动员知道这一点，体操运动员知道这一点，奥林匹克举重运动员知道这一点。但是如果你在商业健身房举重，你可能永远也不会知道这种奇妙的物质；它几乎从来没有出现过，而且经常被禁止。好吧，这就是变通规则的方法。



首先，如果你从未见过，这里是如何使用粉笔。它是块状的，有时是小块的，健身房通常把它放在桶或碗里。你伸手进去，可以选择在皮肤上擦一块，或者只是把手伸进容器底部堆积的粉末中。

粉笔的问题是它会很脏。如果你正确地使用它，你不应该在地板上留下痕迹，或者把它拍向空中。但是粉笔确实有无处不在的趋势，一些健身房不喜欢这样。如果你到处留下白手印，他们会知道是你。因此，请尝试以下选项之一:

### 选项 1:偷偷放入一小块

与其他粉状白色物质不同，粉笔实际上并不违法。你可以在任何地方买到，包括体育用品店和网上。

因为一点点可以走很长的路，你可以只带一小块。我在健身房认识的一个家伙总是在口袋里放一小截粉笔，只够让他抓地力，少到他离开后在吧台上几乎看不到。直到有一天他给了我一些，我都不知道他会用它。

### 选项 2:使用液体粉笔

只要你不把它洒出来，“液体粉笔”往往不会造成太大的混乱。(许多不允许使用粉笔的健身房会为液体材料破例。)这也可以在体育用品店或网上买到。

使用时，在手上喷几滴。它看起来像掺水的牛奶，你会想“这对我有什么帮助？”但是把它擦遍你的整个手掌，然后挥动你的手直到它们变干，在 30 秒左右的时间里，你的手会被涂上一层白垩状的白色。它仍然可以在你的黑色运动裤上留下手印，但至少没有松散的粉笔灰在你周围漂浮。为此，你的健身房老板会心存感激。
