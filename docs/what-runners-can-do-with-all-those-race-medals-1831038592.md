# 跑步者可以用这些奖牌做什么

> 原文：<https://lifehacker.com/what-runners-can-do-with-all-those-race-medals-1831038592>

也许你赢了很多。或者你只是跑了很多比赛，这些比赛会给你带来奖牌——马拉松，半程马拉松， [新奇 5k](https://runningintheusa.com/variety/list/Novelty-Run/upcoming)。不管怎样，第一枚珍贵的奖牌最终会变成压在你门把手上或塞满你梳妆台抽屉的几磅重的硬件。这里有一些更有趣的方式来欣赏或处理这些奖牌。



### 把它们送人

[奖牌 4 枚](https://medals4mettle.org/) 接受马拉松、半程马拉松、铁人三项奖牌捐赠。他们用自己的丝带代替丝带，送给那些正在忍受癌症、慢性病、创伤或其他生活挑战的人们，其中大多数是儿童。“这些奖牌被授予那些必须参加比奖牌捐赠者的比赛更艰难的比赛的人；一场他们没有选择参加的比赛:一场终点线可能不为人知的比赛。"

### 弄一个合适的衣架

如果你还想看到那些奖牌一个挨着一个挂起来，试试 [其中一个展示架](https://www.medalhangers.com/) 下面有挂奖牌的钩子，上面写着“生来就是为了跑步”之类的俗气的话如果你喜欢保存你在比赛中穿的号码，有些还可以挂比赛围兜。 [这是一个巧妙的组合](https://yorksignshop.com/products/pr-chalkboard-race-bib-and-medal-holder-5k-10k-half-full) :黑板下的挂钩。你可以把围嘴夹在黑板上，也可以写下你个人记录的次数。

做个架子也不难。你可以重新利用挂外套或钥匙的架子，或者挂手巾的杆子。(把奖牌的带子绕在杆子上就行了。)如果你方便的话，可以在木板上挂上一些钩子或钉子，让它变得随心所欲。

如果你想纪念一场特殊的比赛——也许是你的第一次马拉松——你可以放上围兜、奖牌和任何你喜欢的纪念品(赛道地图？)在一个 [阴影框](https://www.amazon.com/shadow-boxes/b?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/what-runners-can-do-with-all-those-race-medals-1831038592&asc_source=&ie=UTF8&node=3019281&tag=kinjalifehackerlink-20) 里，基本上就是一个立体的画框。

如果你有一个家庭健身房，那是悬挂你的展示品的完美地方。用它来激励自己打破那些公关记录。

### 把它们变成某种东西

把磁铁粘在你的奖牌背面，它们就变成了冰箱磁铁:总是在展示，但仍然有用。(当然，你必须去掉丝带。)Redditor ChefAaronFitz 写道，它使用钕磁铁和大猩猩胶水。 [这是完成的结果](https://imgur.com/gallery/VzM0rnF) 。

或者 [买一个杯垫套装](https://www.kudos2u.com/) ，把勋章插在里面。这甚至适用于形状不规则的奖牌，只要它们能放在杯垫里，你的奖牌又一次被展示出来，而不占用任何墙壁空间。

如果你有一棵圣诞树，如果你的奖牌不是很大，把丝带换成挂钩，你就有了一套圣诞树饰品。
