# Zoom 上玩的最好的桌游有哪些？

> 原文：<https://lifehacker.com/what-are-the-best-board-games-to-play-over-zoom-1842564172>

屏幕，它们现在是不可避免的。在疫情席卷全球之前，我们的孩子可能已经有足够的时间看屏幕了，但是现在，他们不仅仅是用屏幕来娱乐。他们用手机与老师联系，完成作业，与家人和朋友交谈。

Watch

我九岁的儿子和他的几个朋友定期在 Zoom 上玩。他们大部分时间都在玩《《我的世界》》或一款名为 [侏罗纪怪物世界](https://apps.apple.com/us/app/jurassic-monster-world-3d-fps/id1417141579) 的多人恐龙战争游戏。这很好，他可以看到他的朋友，我现在赞成任何能给他提供一些联系的事情，即使他们花更多的时间盯着他们的平板电脑。

但是这种“就地避难”的事情持续的时间越长，找到一种将更多的传统游戏融入日常生活的方法就越好。

我意识到许多经典的桌游和卡牌游戏都有网络版的，人们现在可以在物理距离的快乐时光中加以利用。但是对于孩子们来说，在现实生活的棋盘上晃动手中的骰子和不必要的用力敲击游戏棋子是有特殊意义的。一个回到简单时代的活动。

因为我的儿子，你知道，和我一起在家，我想我也可以告诉他他想和他的朋友在 Zoom、Google Hangouts、FaceTime 或 Duo 上玩什么游戏。

“绝对不是纸牌游戏，”他说，“也许是抱歉？是的，对不起是个好主意。”

他提出了一个很好的观点——我无法想象像 Uno 或 Go Fish 这样的东西如何通过视频聊天进行实际工作。但是经典的桌游比如抱歉，麻烦，大富翁，或者滑梯？那些应该管用，只要 1。双方博弈相同且 2。孩子们已经长大，可以跟随其他玩家移动对手的棋子或 3。附近有家长帮忙那部分。

我的一个朋友最近买了一张 [的票，可以乘坐](https://www.daysofwonder.com/tickettoride/en/usa/) 和她六岁的儿子以及他们在全国各地的伙伴一起玩(见上图)。其他想法可能包括糖果乐园，线索，连接四，战舰，你甚至可以很容易地在电话上玩。但是扭扭乐只有在所有人身体都缠在一起的时候才有意思吗？

我们在评论里列个清单吧。您的孩子尝试过通过视频聊天玩什么游戏？有什么提高游戏流畅度的小技巧吗？所有人都过得很愉快吗，或者说它带来的麻烦比它值得的要多吗？
