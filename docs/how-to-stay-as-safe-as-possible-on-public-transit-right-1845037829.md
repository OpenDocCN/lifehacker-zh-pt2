# 如何在公共交通上尽可能保持安全

> 原文:[https://life hacker . com/how-to-stay-as-safe-only-on-public transit-right-1845037829](https://lifehacker.com/how-to-stay-as-safe-as-possible-on-public-transit-right-1845037829)

对于那些幸运地生活在拥有良好公共交通的地方并依靠它出行的人来说，过去的六个月可能是一个挑战。尽管我们生活中的许多人已经搬进并留在家中，但并不是每个人都可以选择远程工作或学习。当然，自从疫情开始以来，数以千计的重要工人一直在为我们提供我们需要的商品和服务，情况就是如此。(谢谢，必不可少的工作者！)

无论如何，随着学校和办公室开始重新开放，这意味着更多的人将再次使用公共交通。鉴于疫情仍在继续，是时候看看我们如何在乘坐地铁、公交车和火车时更加安全了。以下是最近纽约时报的一篇文章中关于如何尽可能安全地回到公共交通的一些建议。

### 先考虑清楚

在跳上公交车或进入地铁站之前，考虑一下哪种交通方式 [对自己和他人构成的](https://www.nytimes.com/interactive/2020/09/09/at-home/coronavirus-mass-transit-safety.html) 风险最小。这意味着要考虑到一些事情，比如你是在室内还是室外等待，以及旅行时间(即你必须和其他乘客在一个封闭空间内度过的时间)。

纽约大学流行病学教授罗宾·格申告诉《纽约时报》 “你在户外呆得越多，离其他人越远，其他人不戴口罩的可能性就越小，这是最安全的方式。”。

如果可能的话，最好不要在高峰时间出行。越空越好。

### 找到(最新鲜的)空气

这在地铁上不太可能，但是如果你在坐公交车，试着找一个靠窗的座位，如果可以的话，打开窗户。运用同样的策略来导航地铁站和公共汽车终点站。如果你可以选择走楼梯或自动扶梯而不是电梯(这是你身体条件允许的)，那么一定要走通风最好的路。

### 远离高触摸表面

在这一点上，我们应该已经习惯于尽量减少我们与高触摸表面的任何接触，这肯定适用于公共交通。例如，尽可能使用非接触式支付方式。当然，在乘坐任何形式的公共交通工具之前和之后都要洗手，一路上使用大量的洗手液。

另外，《纽约时报》的文章 [建议](https://www.nytimes.com/interactive/2020/09/09/at-home/coronavirus-mass-transit-safety.html) 远离地铁、火车、公共汽车和渡轮上的电线杆和栏杆，以及任何触摸屏、键盘和电梯按钮。此外，只要穿过十字转门——没有理由把手放在上面。

### 注意并遵守规则

不用说，所有通常的规则和 COVID 公共卫生措施仍然适用于公共交通。但是最重要的是，在车站、公交车站或车内看看有没有关于当前最佳安全措施的新标志或海报。别忘了往下看:可能会有社交距离贴花，或者关于走出车站的最佳方式的信息。

### 跳过零食(和聊天)

如果你属于“在公共交通工具上吃东西总是很恶心，不管是不是疫情”这一类，那么这不应该是个问题。但是如果你习惯在火车上吃早餐，你可能要考虑调整你的早晨时间表，这样你就可以在自己家里安全舒适地吃早餐了。

这有几个原因。首先，现在在公共汽车和地铁上应该避免任何接触/食物/嘴巴的情况。此外，要吃东西，这意味着你必须摘下你的面具(即使只是一小段时间)，这也不是一个好主意。

如果你有一张脸，出于某种原因，促使陌生人闲聊或告诉你他们的生活故事，你现在有一个很好的借口慢慢后退:正如 [《泰晤士报》的文章指出的](https://www.nytimes.com/interactive/2020/09/09/at-home/coronavirus-mass-transit-safety.html) ，说话“喷洒雾状飞沫，可以携带病毒粒子。”不客气，内向的人(以及任何喜欢独自乘坐公共交通工具的人)。
