# 当你怀孕时，要你真正需要的礼物

> 原文：<https://lifehacker.com/ask-for-the-gifts-you-really-need-when-youre-expecting-1840833437>

一个婴儿真的只需要这么多连体衣。有几个不同的接收毯挺好的；但是 12 条毯子太多了。婴儿在穿够或用够之前，就已经不再需要买给他们的东西了，甚至让它看起来像二手货。但你知道什么永远不会浪费吗？时间。方便。回忆。



我们以前写过关于你应该如何 [给你的婴儿登记处](https://lifehacker.com/put-services-on-your-baby-registry-1827846200) 添加服务——送餐、房屋清洁服务或产后导乐，仅举几例。尽管我们的文化在接受无物品登记方面进展缓慢，但我个人一直不明白，为什么在蜜月旅行时要求远足会显得很俗气，而不是，比如说，只使用一次的冰淇淋机。

然而，我们变得越来越开明，因为像 GoFundMe 这样的网站让人们在危机中寻求经济支持变得更加容易。此外，我们中的更多人正在接受极简主义的生活方式——或者至少试着不要过度浪费，不小心将更多我们的孩子几乎不用的塑料垃圾填满我们的垃圾填埋场。

公平地说，当你怀孕的时候，尤其是你第一次怀孕的时候，你确实有一些合理的“装备”需求。婴儿床。婴儿车。汽车座椅。等等。但是在“我们需要这些东西来为我们的生活准备一个宝宝”和“现在我只是添加一些东西，因为它很可爱，艾莉森阿姨一直在要求更多的想法”之间有一条细微的界限当你到达那一点时，问问你自己:我真正需要的是什么？当孩子出生时，什么能让我的生活更轻松或压力更小？

你当然可以从产后服务开始:膳食计划，每周一次的夜班护士，让你定期睡个好觉，或者哺乳顾问。但是，也许有其他你没有考虑过的非传统的东西会比一双你的宝宝永远也不会穿的可爱的婴儿鞋更有价值。而且你可以在 [Plumfund](https://www.plumfund.com/baby-fund/) 上为这些非物品欲望建立一个注册表，就像准父母的 GoFundMe。

### 儿童保育

对我们大多数人来说，指望我们的亲戚能够或愿意资助持续的儿童保育是不可行的。但是也许你真的希望在孩子出生的第一年有一个家庭保姆来照顾孩子；但是你负担不起额外的费用。或者有一个非常好的日托所，比你其他的选择每月花费多一点，但是钱已经够紧了。

或者也许你只是想建立一个“保姆基金”，这样你和你的伴侣就可以像你们承诺的那样，每月出去吃一次饭。如果一个朋友在怀孕期间提出要求，我个人会全力以赴。在最初的几个月里，为你想要或需要的儿童保育提供一点额外的资金可以减轻一些经济压力。

### 旅行

也许你知道你会希望你妈妈在孩子出生后经常来看你，但她离你有半个国家远，负担不起总是来回飞。或者你非常渴望你最好的朋友能来看你一两次，这样你就能在一段时间内感觉自己又是一个完整的人了。

当你刚为人父母，需要你所爱的人的支持时，短信和 FaceTime 是不够的。你可以申请资助机票，让你直接恢复理智——或者让你带着孩子去旅行，第一次炫耀你家庭的新成员。

### 为以后做准备

当你注册所有的东西时，你甚至还没有见过你的孩子。你不知道你的宝宝是喜欢秋千还是讨厌秋千。是洗澡时泼水还是洗澡时尖叫。会不会母乳喂养。因此，虽然你肯定需要一些基本的东西来开始，但一旦你弄清楚这笔费用对你的特定孩子来说是否值得，留出一点资金来购买你将使用的设备不是也很好吗？

你可以启动一些我刚刚虚构的叫做“为以后做准备基金”的东西，一旦你真正知道你的宝宝是否喜欢坐在婴儿车中愉快地散步，你就可以把每个人的钱投入到更好的使用中(为慢跑婴儿车而欢呼吧！)或者没有(还好你没有浪费别人的钱买慢跑推车)。

如果你能想象出它——这将对你有帮助——把它变成一个注册表，所有那些花在填充填充动物玩具上的钱将会以一种更加具体和个性化的方式帮助你。
