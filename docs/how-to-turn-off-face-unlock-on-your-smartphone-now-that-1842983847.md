# 既然你戴着面具，如何在智能手机上关闭面部解锁

> 原文:[https://life hacker . com/how-to-turn-off-face-unlock-on-your-smart phone-now-that-1842983847](https://lifehacker.com/how-to-turn-off-face-unlock-on-your-smartphone-now-that-1842983847)

大多数时候，使用面部解锁手机是一种无缝的访问设备的方式。然而，现在 [我们出门都戴着口罩，在](https://vitals.lifehacker.com/what-to-know-if-youre-wearing-a-cloth-mask-in-public-1842706701) 左右(对吧？)这个功能与其说是一个很好的免提使用方式，不如说是一个令人讨厌的麻烦。

如果你在使用 Face ID 或面部解锁时遇到了麻烦，你可以暂时禁用该功能，这样你就不必摸索你的面具或经历一系列输入密码的提示。

该功能在 iOS 和 Android 上很容易被禁用，也很容易重新打开，所以当你回到室内或前往不需要遮脸的地方时，你可以再次添加它。

如果你确实禁用了它，现在是一个好时机，确保你为你的设备默认一个安全的备用密码，如果你不经常戴手套，最好使用它的指纹传感器。如果你经常戴着手套外出，现在也是重新评估你正在使用的解锁代码或模式的好时机，并考虑将其改为更复杂的东西。

### 苹果手机

在你的 iPhone 上，你需要进入设置菜单，然后点击“Face ID 和密码”您的 iPhone 会要求您输入密码。一旦你这样做了，你就可以在下一页关闭“iPhone 解锁”了。

一旦你这样做了，你的 iPhone 将默认使用你的数字密码解锁。接下来，如果你想重新启用 Face ID，你可以按照这些指示再次打开 iPhone 解锁按钮。

如果你不想完全禁用它，当 Face ID 提示出现时，你也可以按住电源键和任何一个音量键。你的手机会显示断电、医疗 ID 和紧急求救菜单。当你退出时，你会被要求输入密码来解锁

### 机器人

在安卓系统上，你需要进入设置菜单，然后是“安全”，接着是“面部解锁”。和 iOS 一样，你需要在这里输入手机密码才能进入下一页。

切换“解锁你的手机”旁边的按钮来禁用面部解锁。

同样，和 iOS 一样，当你需要重新打开它的时候，只需再次按照这些指示，将“解锁你的手机”按钮重新打开。
