# 当在工作中变得脆弱真的值得时

> 原文：<https://lifehacker.com/when-it-actually-pays-off-to-be-vulnerable-at-work-1830683219>

不同领域的工作场所文化差异很大，但大多数人都会同意，在大多数办公室里，没有多少人性弱点的空间。然而，对于情商高的人来说，一点点脆弱实际上能给你惊人的力量。

作家格温·莫兰 [采访了《快公司》的领导力顾问彼得·布雷格曼和迈克·罗宾斯，询问他们为什么建议客户表露自己的情感。他们发现，更强的领导者更开放，最重要的是，他们知道何时寻求帮助。软弱的领导者不会分享对项目或其他不确定性的“相关担忧”，因为他们非常渴望保持他们所感知的权力。正如你可能想象的那样，这会引起问题。](https://www.fastcompany.com/90268440/how-being-vulnerable-can-make-you-a-better-boss)

然而，脆弱不应该包括分享“不必要的痛苦，假装谦虚，或者让问题都是关于你和你的担忧。”

“脆弱是风险、情感暴露和不确定性。罗宾斯解释说:“如果你思考这三件事，那么在我们的生活中，无论是个人生活还是职业生活，没有什么有意义或重要的事情是不需要这三件事中的一件、两件或全部来完成的。

这里有一些好处，可能会鼓励你在适当的时候敞开心扉。

### 你看起来更自信了

能够承认你没有完全理解某事需要勇气，尤其是如果你应该负责的话。你在展示你对自己的能力和身份有着坚定的把握，并且愿意成长。你似乎也相信没有人能在你脆弱的时候利用你——因为你非常坚强！

“那些没有太多自信的人，通常不会让自己变得脆弱。但是那些非常自信的人，实际上更愿意展示他们自己，对吗？他们愿意变得脆弱，因为他们不担心有人会趁虚而入，”布雷格曼说。

### 它帮助人们与你联系

分享一些诚实和开放的东西会鼓励你周围的人也这样做，从而建立联系。作为一名领导者，这是有益的，因为人们更有可能为他们感到有联系的人努力工作。布雷格曼说，这让同事觉得你是“和他们在一起的真正的人”

他说的是真实性。虽然大多数人可能希望老板表现出自信和控制力，但假装你的技能或知识没有弱点会让你看起来很假，因为没有人是没有弱点的。通过建立关系和团队合作，你也可以从这些弱点中获益。

### 它为每个人减轻压力

如果你感到紧张，办公室里的其他人可能也一样，不管他们是感受到了你的情绪，还是只是感受到了普遍的压力。罗宾斯分享了他打职业棒球时的一件轶事，承认他隐藏了自己对比赛的紧张，结果发现他的队友都在做同样的事情。

有时候大声说出你担心的事情可以减轻你的担心，罗宾斯说他们可以表现得更好。知道每个人都在同一页上，你会感到更安全，这意味着他们可以专注于自己的工作。比坐在座位上怀疑自己是不是唯一一个抓狂的人好多了。

### 创新时有发生

寻求指导、观点和答案为新思想的传播创造了更多的机会。如果你是负责人，表现得好像你把一切都锁在里面，就不会有成长的空间。

> 员工觉得自己受到重视，是这个过程的一部分。Bregman 说，因为你激发了信任，所以你的团队感觉对解决方案投入更多，也更愿意做出贡献。

脆弱是一个复杂的概念，需要强烈的自我意识来处理。如果是你从未尝试过的事情，那可能是因为你没有自己想象的那么强大。
