# 你应该增加你的退休金，比如现在

> 原文：<https://lifehacker.com/you-should-increase-your-retirement-contribution-like-1837382803>

为退休存钱对美国人来说是个好消息:根据 Bankrate.comT3 的一项最新调查，我们在实际做这件事上做得越来越好。自 2011 年以来，在过去一年中增加捐款的人的比例几乎翻了一番，其中 29%的人增加了捐款。46%的受访者表示，他们的储蓄与去年持平。

Watch

尤其是年龄较大的千禧一代(30-38 岁)，是最有可能提高储蓄率的人群。Bankrate.com 首席金融分析师格雷格·麦克布莱德说，“他们已经还清了学生贷款，或者买了第一套房子，或者获得了加薪，他们比职业生涯早期更多地考虑下游”。该调查有 2016 名受访者。

### 你真的应该“着手”调整你的贡献

这里有一个不那么鼓舞人心的消息:16%的人退休储蓄比去年少。许多受访者提到了其他财务目标，但 12%的人说他们只是没有“抽出时间”来调整他们的贡献。麦克布莱德说，年轻的千禧一代是受访者中最糟糕的罪犯，他们让任务滑到了待办事项清单的底部。

但他警告说，如果不关注他们的贡献，年轻人损失最大。“由于复利的力量，在为退休储蓄时，时间是你最大的盟友，”麦克布莱德说。"因此，尽快做出贡献是当务之急."

稳步前进赢得了这场马拉松，即使你今年能够 [增加 1%的贡献](https://lifehacker.com/you-just-need-to-increase-your-savings-by-1-a-year-1826984496) ，它也将有益于你的长期财务健康。“如果你在工作并领取薪水，你会希望这笔钱自动存入你的退休账户，”麦克布莱德说。

但是，嗯，关于经济预测呢？事情 [在那里](https://twocents.lifehacker.com/how-youll-know-a-recession-is-coming-1837273628) 看起来并不那么乐观，但是当你登录查看你的贡献水平时，这并不是开始扰乱你的分配的理由。“对退休储蓄者来说，最大的风险是市场波动带来的不安，以及改变退休储蓄的投资方式，”麦克布莱德说。"从长远来看，波动性实际上会对你有利."
