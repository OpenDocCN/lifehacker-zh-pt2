# 是的，健身室空着的时候你可以用

> 原文：<https://lifehacker.com/yes-you-can-use-the-fitness-studio-when-its-empty-1837940351>

有没有希望你的健身房有一个漂亮安静的空间，你可以在那里做一些伸展或体重练习，而不必把自己夹在机器和哑铃兄弟之间？这里有一个秘密:很可能是这样的。

Watch

如果你的健身房有一个工作室，他们在那里举办集体课程，很可能他们不会介意你在不上课的时候去那里闲逛。事实上，现在可能有人正在那里做放松瑜伽。

为了确保没问题，问健身房的前台。但是我去过的每个健身房都允许人们在空着的时候使用工作室。核实，但一般来说这些都是不成文的规定:

*   查看课程表，确保没有课要开始了。
*   如果你在房间里使用任何设备(如健身球或瑜伽垫)，确保将它放回原处，必要时擦拭干净。
*   如果有人已经在房间里，尊重他们先在那里。不要妨碍他们，如果你想开灯或做类似的事情，做一个正派的人，问问他们是否介意。
