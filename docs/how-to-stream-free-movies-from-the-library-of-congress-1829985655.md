# 如何从国会图书馆下载免费电影

> 原文：<https://lifehacker.com/how-to-stream-free-movies-from-the-library-of-congress-1829985655>

下次你发现自己在流媒体队列中滚动，认为没有什么值得看的，试试国会图书馆的国家放映室。

Watch

这个多媒体项目于上个月启动，将图书馆的视频收藏数字化，并向公众免费开放。可用的电影既有虚构的也有非虚构的，时间跨度从 1890 年到 1999 年。

国会图书馆不是网飞:你将无法观看当前的好莱坞大片和你最喜欢的电视节目的每一季。然而，你会发现历史电影和新闻短片，从大萧条时期的纪录片到家庭录像，再到关于心理健康的教育系列。

收藏中的所有电影，包括有版权的电影，都可以在 LOC 网站上观看。公共域中的文件(大部分是公共域)也可以下载成四种格式:GIF、JPEG、MP4 和 Quicktime。

为了观看， [参观国家放映室](https://www.loc.gov/collections/national-screening-room/) 。你可以按日期、地点、主题等分类，点击标题后，你可以了解每部电影的更多信息，包括下载的可用性。

在撰写本文时，国会图书馆的筛选收藏中有 298 项。该项目的计划包括每月增加新的内容。
