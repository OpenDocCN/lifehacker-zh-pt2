# 如何对着麦克风说话

> 原文:[https://life hacker . com/how-to-talk to-a-microphone-1836427140](https://lifehacker.com/how-to-talk-into-a-microphone-1836427140)

无论你是在婚礼或活动上讲话，在播客上客串，还是现场报道，对着麦克风说话并不完全像不带麦克风说话。如果你遵循一些简单的规则，你会听起来好很多。

### 靠近，但不要太近

为了获得良好的音质，你需要对着麦克风说话，但你不想把嘴唇放在麦克风上 [，就像你在 Twitter 笑话](https://twitter.com/search?q=%22lips%20right%20on%20mic%22&src=typd) 中一样。太近的话，每个人听起来都会觉得你是对的， [阴阳双胞胎风格](https://www.youtube.com/watch?v=yIb3GkuRHps) 。太远，你会听起来像是在走廊的另一端。

Buzzsprout 的播客麦克风技术指南 建议你的嘴与麦克风保持一只手宽的距离。这有助于你听起来最接近正常的人类对话距离——即使它比你通常接近某人的耳朵要近得多。

### 把你的嘴对着麦克风

当麦克风几乎没有接收到你的声音时，你很容易忘记自己，转过头，对着你旁边的人说话。如果你拿着一个免费的麦克风，只要把麦克风和你的头一起移动。如果你对着一个固定的麦克风说话，你需要不自然地保持指向麦克风。

转一点一般没事！但是当对着固定的麦克风说话时，将你的头移向远离你说话的一侧，这样你的声音仍然会穿过麦克风。

这实际上因麦克风的模式而异，因此如果您正在录制音频，请测试声音或询问您的工程师，当您通过麦克风而不是直接向下说话时，是否仍能获得良好的音频。

### 别放下那该死的麦克风

抱歉，我们真的被这件事激怒了。在现场活动中，不要问“不用这个你们都能听到我说话，对吗？”试着不用麦克风发出你的自然声音。你真的非常讨厌拿着麦克风，以至于你已经准备好让后排的观众完全走神了吗？

如果活动组织者给了你一个麦克风，他们这么做是有原因的。有时他们比你更了解这个空间的声学特性，或者他们知道在你的密室聚会进行到一半时，前面的酒吧会变得拥挤，开始淹没你。有时候，他们知道在没有麦克风的情况下讲半个小时会使你的声音变得紧张，到你演讲结束时，你听起来就像一个生病的孩子。麦克风是你的朋友。

如果一个事件正在被录制或广播，这一点尤其重要:在电话会议上，或者在录制的研讨会上，或者在直播的播客上。尊重每个可能听到你说话的人，不管他们现在是否在房间里。

### 保持音量稳定

每个人都这样做，但每次都很尴尬:后面的人说“大声点！”扬声器开大了一点，但几分钟后，它们的音量又降了下来。可能是耶稣干的。大概还有五个祝福，但是抄写员不太明白。

即使扬声器在麦克风上，也会发生这种情况。如果有人在积极操作音响设备，他们可以提高你的声音一点点，但只能在麦克风拾取其他声音之前。你的工作还是不要咕哝。

你有两种方法可以避免这种情况:一，选择坐在后排的人在你演讲的时候看着他们。我们喜欢这样做，因为这样比坐在前排不那么吓人。如果你的注意力在远处的人身上，你会为他们提高音量。

第二——这需要一点计划——请一个朋友坐在后排，给你信号。他们可以提醒你大声说话，还可以提醒你放慢速度，加快速度，不要无精打采，或者停下来。他们会帮助你展现最好的自己。
