# 在不知道作者或书名的情况下阅读书籍的第一页

> 原文：<https://lifehacker.com/read-the-first-pages-of-books-without-knowing-the-autho-1844887932>

如果你在家呆了几个月后，已经在你的书架上 [阅读了](https://lifehacker.com/how-to-fit-reading-into-your-stay-at-home-life-1844234242) ，你可能正在寻找一些新的东西。也许你通常喜欢历史小说，但不知道从另一种类型中挑选一些。有一些常见的方法，比如从朋友那里获得推荐，从 Goodreads 上的陌生人那里获得推荐，或者询问图书管理员或在书店工作的人，但是我们刚刚发现了一种新的(有趣的)方法，你可能想试试。



### 如何获得新书推荐

还记得小时候老师告诉我们不要以貌取人吗？这可能更多的是关于我们如何对待他人的评论(因为在那个时候，书籍确实有非常直观的描述性封面)，但是如果这个概念引起了你的兴趣(或者仍然如此)，有一个网站可能会感兴趣。它叫做 [“给我推荐一本书”](https://recommendmeabook.com) 这正是它的作用。

主页不包含任何华而不实的图形或任何其他视觉效果——只是一本书的第一页。不仅封面不在，而且你也不知道书名或作者。在页面底部，你可以选择显示作者和标题。如果你有兴趣阅读它，你可以把它添加到你的虚拟书架上。网站上只有书的第一页，但它确实直接链接到书的亚马逊页面，所以你可以很容易地订购它(或者使用这些信息并从你当地的独立书店或图书馆获得)。

当你准备好下一本书时，点击“下一本书”,看看会弹出什么。即使你目前对写一整本书不感兴趣，读一本书的第一页，找出是谁写的，然后读下一页也是一种乐趣。你可以玩其他有趣的游戏，比如根据一页纸猜测作者或流派。我不太擅长这一点，但在识别男性写的书方面却一帆风顺，仅仅基于他们如何描述女孩和/或女人的外表。
