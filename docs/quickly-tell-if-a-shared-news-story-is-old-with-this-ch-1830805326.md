# 使用这个 Chrome 扩展，快速判断一个共享的新闻故事是否过时

> 原文:[https://life hacker . com/quick-tell-if-a-shared-news-story-is-old-with this-ch-1830805326](https://lifehacker.com/quickly-tell-if-a-shared-news-story-is-old-with-this-ch-1830805326)

在过去的一年里，我们所有人(嗯，至少是大多数人)对我们在新闻中读到的东西，尤其是社交媒体上读到的东西，变得更加怀疑了。

Watch

在我自己的脸书朋友群体中，一个巨大的错误信息来源似乎是旧闻，而不是不准确的消息。

例如，上周我的一个朋友上传了一段视频，视频中有人唱“袖手旁观我”给特朗普 ，其中有一些考虑不周的歌词改写。不管你的政治倾向如何，这都很糟糕。

这位朋友将视频和一篇未注明日期的新闻文章一起上传到了网上，数十人在评论区讨论给显示屏加 X、Y 或 z 是多么不合适。

稍微挖掘一下，发现这个故事其实是两年前的事了，那个糟糕的音乐插曲发生在特朗普竞选总统期间，而不是在他获胜之后。

分享它的人对此一无所知。

[SureOrNot](https://chrome.google.com/webstore/detail/sure-or-not-news-date-val/hkiapjaidokhcajjnpldhioambcjbbhp?ref=producthunt) 是 Chrome 的一个扩展，可以帮助你指出那些类型的新闻错误。当它检测到一个故事在旧的一边，它就这样标记它。这样你就可以知道，有些事情可能并不像你想象的那样重要。

为了避免社交媒体上的“虚假”或不正确的新闻，这是一种快速简单的方法，可以让你在浏览新闻时获得更多信息。
