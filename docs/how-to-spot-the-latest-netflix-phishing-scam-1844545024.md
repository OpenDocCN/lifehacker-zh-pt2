# 如何识别最新的网飞网络钓鱼骗局

> 原文：<https://lifehacker.com/how-to-spot-the-latest-netflix-phishing-scam-1844545024>

每个人都喜欢网飞，尤其是当他们不用付钱的时候。这就是你分享别人账户的原因，对吧？嗯，骗子也喜欢免费的网飞，一种新的网络钓鱼攻击可能会让你放弃宝贵的用户名和密码，以及其他你不希望别人拥有的重要信息。

### 这一切都从一封电子邮件开始

正如 [Armorblox](https://www.armorblox.com/blog/blox-tales-netflix-credential-phishing) 所描述的，这种新的网飞网络钓鱼企图始于一封来自“网飞支持”的简单电子邮件它告诉用户，他们的账单细节有问题，他们需要确认这些，否则他们的网飞订阅第二天就会被关闭。

虽然我认为 [电子邮件*排序* *的*看起来是合法的](https://a.storyblok.com/f/52352/545x482/5f2f64ad57/netflix-phishing-email-body-final.png) ，而且一个人收到电子邮件也不是没有听说过，比如说，当信用卡因为任何原因不能被收取服务费用时，这种消息对任何人来说都应该是一个巨大的危险信号。首先，网飞不会在一天后就取消你的账户；他们会根据你最后一次付费的时间，在你的订阅到期后立即取消订阅——我猜可能是从你最后一次付费后的一个月。

第二，*不要点击类似这些*的邮件中的链接。尽管它们可能很方便，但你最好的做法是打开你的网络浏览器，输入上述网站或服务的网址。自己打开账户设置，看看是否一切正常；如果一家公司很好地向你发送了一封关于某个问题的电子邮件，并且这是合法的，那么他们无疑也会在你的帐户设置中标记出任何问题。(相信我，他们和你一样，也渴望你能理清自己的付款细节，以避免服务被取消。)

### 给我们你所有的信息

如果你点击钓鱼邮件中的一个诈骗链接，你首先会被带到一个验证码页面，它看起来像网飞网站上的东西:黑色背景，红色方框，平滑的白色文本，等等。这不仅为网络钓鱼企图增加了一点额外的合法性——尽管 scammy 网站可以像其他任何网站一样包含验证码——而且还有助于隐藏网络钓鱼攻击的最终页面，在该页面中，您可以输入您的帐户凭据，防止您的电子邮件提供商可能用来帮助保护您安全的各种反恶意软件服务。

当然，如果你查看你访问的网站的 URL，你会发现它并没有托管在任何网飞的域名上。尽管如此，还是有很多人在点击网页时不看网址，所以在这种情况下很难指望这种技术——尽管这是最容易让人相信你正在访问的网站可能是合法的。

正确解答验证码的用户会被带到一个看起来很像网飞的页面。他们说出自己的登录信息，然后被要求提供完整的账单地址、电话号码和支付细节(包括银行名称和账号)。这又是另一个危险信号——如果一个服务在你注册时没有要求你提供某些信息，你不应该在将来某个时候被要求提供这些信息。我是说，真的。为什么网飞会需要*你的银行账号*。

### 网络钓鱼诈骗通常有一些很好的“提示”

这个网络钓鱼骗局有很多漏洞，但事情就是这样；这些不是用来陷害像你这样聪明的人的。它们是为那些不太注意他们在网上所做事情的细节的人，或者那些非常害怕失去访问*机会的人制作的。*无论何时被问到，他们都愿意提供自己的账户信息。(我完全理解那种感觉。)

你对这些危险信号了解得越多，你就能更好地应对下一次网络钓鱼攻击——尤其是如果它比这次更聪明的话。它们包括:

*   一封电子邮件包含一种奇怪的紧迫感，因为你已经付费访问了一段时间的网站或服务/
*   一封邮件的发件人甚至不来自该公司的域名:在这种情况下，钓鱼邮件来自“网飞@csupport.co”，而不是像“netflix.com”这样的网站
*   你会收到一封关于某个账户问题的随机电子邮件，而在你使用某个网站或服务的整个过程中，这个账户问题从来都不是问题。
*   电子邮件中链接的任何网站的 URL 实际上都不存在于公司的域名中。
*   一个网站或服务要求你提供比以前要求的更多的关于你的金融服务或证券的信息。
