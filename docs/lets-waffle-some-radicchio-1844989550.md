# 我们来烤点菊苣吧

> 原文:[https://life hacker . com/let-waffle-some-radic hio-1844989550](https://lifehacker.com/lets-waffle-some-radicchio-1844989550)

许多人震惊地发现，我是一个多产的沙拉制造者和消费者。这是有道理的，因为我在这个世界上扮演的角色有点像浸了杜松子酒、涂了培根油的角色，而絮絮叨叨的和 [吃剩的披萨](https://skillet.lifehacker.com/you-should-waffle-two-pieces-of-pizza-together-1844905721) 并没有帮助。但我确实喜欢美味的沙拉，尤其是含有大量玉米、蓝奶酪或菊苣的沙拉。

[我对菊苣](https://skillet.lifehacker.com/radicchio-makes-a-superior-thanksgiving-salad-1839866691) 的爱乍一看毫无意义。这是一种坚硬的——几乎像橡胶一样——非常苦的叶子，但这种结构和苦味使得它在沙拉中如此神奇。不像嫩绿色蔬菜或温和的冰山，菊苣持续地坚持自己，不怕大量的盐、酸、脂肪或热。(此外，如果你把我对苦阿马里的痴迷归结为一个合理的结论，我对这片紫色叶子的喜爱是有道理的。)

Radicchio 的结实特性也使它成为烧烤或——在我们的情况下——闲聊的绝佳选择。烤沙拉受到的喜爱程度不一，很多人对热生菜的想法感到厌恶，但我发现一点点烧焦或焦糖化可以给沙拉增添一种美好的感觉，而且你不必趁热端上来。(我通常会让生菜冷却到室温，然后再调味。)羽扇豆没有完全烧焦，但它枯萎了，轻微地焦化了，而且比它的生品少了一点苦味。

但是，即使是非常热的华夫饼干烙铁也不能使菊苣变得柔顺和温和。它仍然能抵御其他强烈的味道。鹅肝酱是烤肉、红烧肉或烤肉的绝佳配菜，但搭配水煮鸡蛋或煎蛋也很不错。

要做华夫饼干，你所要做的就是把它切成薄的楔形(大约 3/4 英寸)，保持叶子和核心相连。在两面刷上一些橄榄油，然后用中高火将它放入华夫饼干烤模中烤至颜色变软变暗，并有斑点焦糖化。用一种超级有活力的调味汁，或者一些柠檬和蜂蜜(或者枫糖浆)来结束你的鹅肝酱，然后撒上一些奶酪和培根，如果你喜欢的话。真的堆上味；即使是在某种程度上被削弱的、有波纹的形式下，菊苣也能接受它。
