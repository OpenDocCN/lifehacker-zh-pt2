脸书最近 [发布了 CatchUp](https://npe.fb.com/2020/05/26/introducing-catchup-helping-you-stay-in-touch-with-the-people-you-care-about/) 的早期版本，这是一款纯音频聊天应用，有一个有用的功能:用户可以看到他们的联系人何时有空聊天，并且只需轻轻一点就可以开始对话(或群组通话)。

如果这听起来像没有文本和视频通话的 Facebook Messenger 或 WhatsApp，那是因为它确实如此。这也让我想起了在 Discord 服务器上闲逛，和在线的人聊天——除了 CatchUp 只是为你和保存在你手机上的联系人准备的。

想试试的话可以从 App Store 或者 [Google Play](https://play.google.com/store/apps/details?id=com.facebook.npe.vibe&referrer=utm_source%3Dandroidworld.nl%26utm_content%3Dbatch) 下载 app [。J 需要注意的是，它仍处于预览阶段，因此有些功能可能并不总是像预期的那样工作。](https://apps.apple.com/us/app/catchup-effortless-calling/id1480482152)

### 如何使用 CatchUp

1.  使用您设备的电话号码创建新的配置文件。你不需要脸书应用程序来使用 CatchUp

2.  轻按屏幕右上角的开关，将您自己设置为“有空”其他用户会看到您被列为“准备通话”，可以给您打电话或将您添加到群组。
3.  要开始您自己的通话，请点击“准备通话”列表中任何人旁边的**“加入”**。
4.  如果您想在通话中包括其他人，请切换到联系人列表，然后点击他们姓名旁边的**“添加”**。您还可以点击“隐私”图标来调整群组设置，让任何人都可以加入对话，或者保持对话的私密性。一次最多可以有八个用户进行群组通话。

除了通话功能，如果你想让离线用户知道你在线，你还可以向他们发送完全不绝望的“戳”。他们是否决定登录并开始聊天由他们决定。

“离线”部分下面是“联系人”列表。这些是您手机联系人中尚未安装 CatchUp 的用户，但您可以点击“邀请”来邀请他们安装。

### 你应该使用 CatchUp 吗？

在真空中，追赶是一个好主意。但是我不完全确定为什么不把它集成到主要的 Messenger 或 WhatsApp 应用程序中。

当然，不需要脸书账户是一个优势，但 Messenger 和 WhatsApp 也可以在没有脸书账户的情况下使用。我还怀疑邀请和戳通知是否会说服用户安装另一个应用程序，当核心的 Facebook Messenger 已经有了语音，视频和文本对话，现在支持多达 50 人的类似于 [缩放的会议](https://lifehacker.com/the-simplest-video-chat-apps-for-people-who-struggle-wi-1842780065) 。脸书，或者一个与脸书相关的应用程序，可能是我最不想信任的 [和我的联系人列表](https://lifehacker.com/how-to-delete-the-contacts-that-facebook-uploaded-witho-1834143845) 。

不过，如果你觉得有必要扩展你的聊天功能，这款应用还是有的。如果你决定试一试，它很容易使用。
