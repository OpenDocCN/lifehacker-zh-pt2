# 保存好你大学课本的收据

> 原文：<https://lifehacker.com/save-the-receipts-for-your-college-textbooks-1837313434>

今年秋天回去上课吗？虽然一些教科书正在将 [转移到云](https://www.wired.com/story/digital-textbooks-radical-transformation/) ，但很有可能你将不得不为你的大学课程购买或租赁至少几本教科书。总有一个教授让每个人都买最新版本的教科书，尽管它比去年的版本贵了 80 美元。但是你可以收回一些成本。



一旦你在校园书店 掏空了钱包，把你的收据藏在安全的地方，直到纳税季节到来。当你准备提交报税表时，你可能要参考这些文件。

因为国税局认为 [书籍、用品和设备](https://www.irs.gov/newsroom/tax-benefits-for-education-information-center#deductions) 对你的教育来说是必要的费用，所以它允许你在年度纳税申报表中扣除它们。

这项扣除是美国机会税收抵免 的一部分，它允许本科生从他们的税收中扣除教育成本，包括学费、食宿费、交通费和每学期你账单上的所有额外费用。每个纳税年度的税收抵免高达 2500 美元。

研究生可以通过 [终身学习信贷](https://twocents.lifehacker.com/all-the-tax-credits-you-can-take-for-2018-1831811601) 扣除课本费用，该信贷涵盖每个纳税年度高达 2000 美元的教育费用。(对于研究生来说，学费和费用可以通过单独的扣除项目扣除，不像 AOTC 那样将所有教育费用集中在一个学分下。)

当然，这不是现金回到你的口袋里，但任何能减轻你税务负担的东西都能帮助你在上学期间稳定你的财务状况。

国税局 [有一个在线工具](https://www.irs.gov/help/ita/am-i-eligible-to-claim-an-education-credit) 带你浏览你可能有资格获得的与教育相关的信贷或扣除。如果你正要去学校或在校外呆一段时间后再回来，你可能想浏览一下资格调查，以了解你在纳税时间能期待什么。
