# 每个 DIY 项目都需要去三趟五金店

> 原文：<https://lifehacker.com/every-diy-project-will-require-three-trips-to-the-hardw-1834245269>

你们中的一些人天生擅长自己动手做家居装修项目。你陶醉在混乱中，撕毁地毯、搭建棚子或推倒墙壁，以实现令人垂涎的“开放式概念”布局。

Watch

但是对于那些害怕周末充满油漆和事故的人来说，我们压力的答案可能是我们准备得太努力了。我们看了几个不同的教程——它们都声称这个项目(任何项目)非常简单，只要你遵循这 5 个简单的步骤*——*，我们已经列出了详细的供应清单。我们将收集我们需要的东西，然后我们将安装平板电视或安装新的水龙头，不会发生任何事故。

我们即将变得沮丧的原因是因为我们没有按照我父亲改善家居的黄金法则生活:

### 行程 1:做好准备

你带着你的基本清单进去。工具或油漆或硬件或任何你知道你需要开始的东西。也许在这一点上，你还在为完成而争论，或者你有一些技术问题要问专业人士。这是你敲定那些细节的旅行。

如果你忘了什么，没关系。事实上，如果你刚刚意识到你清单上的最后一件事是在商店的另一边，就把它留下。你很快就会回来的。

### 旅行 2:我没想到会这样

你的下一次旅行发生在项目进行到一半的时候，这时你遇到了一些意想不到的麻烦，这是你不可能预料到的。与你正在使用的孔相比,“标准”水龙头底座令人费解地巨大。哦，看，有隐藏的水渍！等等，你怎么会用完所有东西中的*垫圈、*？？

回到商店，你有点沮丧，汗流浃背，去拿你的新用品清单(加上你在第一次旅行中忘记买的东西，所以没关系，因为你无论如何都要回去)。

### 旅行 3:提醒我为什么我认为这是一个好主意？

到目前为止，第三次旅行是所有 DIY 五金店旅行中最令人讨厌的一次，因为这是你真的没有预料到的一次。你确信你在最后一次旅行中得到了你需要的一切——你甚至得到了多种品牌和尺寸的东西，以防万一。但是后来某个该死的小零件坏了，当然这是一个重要的小零件，没有这个小零件，整个事情就白费了。

你很想在喝啤酒和/或沉思的时候把你的伴侣送去商店，但是当你做出奇怪的手势，说“在洗手间左边过道的底部/中间附近的箱子里有一个非常宽的底座的螺丝”时，他们不知道你是什么意思

所以你回去拿最后一小片，现在，你的头发已经蓬乱了，你遇到了前两次来过的同一家商店的员工，他们看着你，眼中充满了爱，心中充满了怜悯。“还是有麻烦？”他们会问，你会对那该死的坏东西大喊一声，然后他们会阴沉地点点头。他们已经看过这个故事上演过一百万次了。

好消息是，你成功了！即使你的神智有点不正常，你的项目也是完整的。但是，如果你下次能记住这一点，如果你的新咒语是，“每个 DIY 项目都需要去五金店三趟”，你就会变得自信和坚韧地处理家庭装修项目。
