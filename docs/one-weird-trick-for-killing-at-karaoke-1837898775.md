# 在卡拉 ok 杀人的一个奇怪技巧

> 原文：<https://lifehacker.com/one-weird-trick-for-killing-at-karaoke-1837898775>

如果你不喜欢卡拉 ok，你很快就会。下次你被拉去参加成人合唱团时，你可以用一个技巧来博得满堂喝彩:改唱怪异的艾尔版本。



1.  选一首你非常熟悉的奇怪的艾尔的歌。
2.  (如果你不知道一首怪异的艾尔歌曲，哇。学一个。)
3.  把你手机上的歌词调出来，以防万一。
4.  播放原曲。
5.  改唱怪异的艾尔歌词。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-FklUAoZ6KxY&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-FklUAoZ6KxY&start=0) 

无论你选择什么怪异的模仿，你都会在卡拉 ok 菜单上找到原版。艾尔不会模仿深度切割。因为艾尔是一个如此细心的词曲作者，模仿的歌词将会非常好地映射到原作上。这就是为什么他的歌那么容易记住。

合适的朋友会和你一起唱。他们会为知道这些词而骄傲。你们将建立牢不可破的关系。

我更喜欢在没有警告的情况下玩这个把戏，它会带来惊喜，但如果人们在你透露之前就开始唱原来的歌词，这种效果就会被破坏。在歌曲开始前决定是否用含糊的“我要尝试一些不同的东西”来让他们安静下来

去把怪异的艾尔福音传播给全世界。愿你生活在阿米什人的天堂。
