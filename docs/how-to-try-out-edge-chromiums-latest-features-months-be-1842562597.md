# 如何在 Edge Chromium 的最新功能发布前几个月试用它们

> 原文：<https://lifehacker.com/how-to-try-out-edge-chromiums-latest-features-months-be-1842562597>

微软今天宣布了其 Edge Chromium 浏览器的一些有趣的新功能，这些功能将在未来几个月内陆续推出。我没有耐心，所以我看了看即将推出的东西——垂直标签式浏览、智能复制粘贴、密码检查和有组织的“收藏”——并找到了所有的扩展和变通办法，你现在就可以尝试这些功能。



### 垂直制表符

目前几乎所有主流浏览器都采用标签的问题是，你打开的标签越多，你的体验就越差。我并不是要责怪你，因为你的浏览器后台有太多令人不快的网站。这种情况很糟糕，但有时是不可避免的。然而，你塞进屏幕的标签越多，就越难看清每个标签到底是什么——这意味着导航到一个标签来处理它的内容，最好是关闭它，这一行为变成了一个试错过程，几乎肯定会鼓励你做其他任何事情。

输入垂直制表符。因为当你可以看到一个可爱的(虽然很长)垂直标签列表时，谁还需要小小的水平标签呢——每个标签都自豪地显示在你浏览器的一侧，以便更容易找到。垂直标签将很快登陆微软的 Edge Chromium 浏览器，但没有理由你现在就不能开始尝试。

由于 Edge Chromium 支持 Chrome 扩展(不管你的浏览器可能会对你大喊大叫)，我建议检查两个。首先，尝试 [**vTabs**](https://chrome.google.com/webstore/detail/vtabs/okpnlgbgcfchbicbhjmmhldhkbkfilce/related?hl=en) ，这是一种快速而冗长的方法来解锁你的标签页的垂直侧边栏。安装时有太多的选项让你摆弄，所以我建议除了你想要的基本选项——垂直标签——之外，不要勾选任何选项，坚持使用它。当你这样做的时候，你的浏览器看起来会像这样:

### 密码监视器

这个很简单。紧随其后，微软推出了 Edge Chromium 功能，如果或何时你的密码因某种数据泄露而被泄露，它会提醒你。更好的是，微软甚至会给你一个快速简单的链接，链接到你的各种服务的密码重置页面，让你可以轻松地更改密码。

有很多扩展可以模仿这种功能，或者，至少，给你足够的警告，然后你可以根据需要更改你的密码。想到的是谷歌自带的 [**密码检查扩展**](https://chrome.google.com/webstore/detail/password-checkup-extensio/pncabnpcffmalkkjpajodfhijclecjno?hl=en) 。免费的密码管理器 LastPass 有一个“ [安全挑战](https://helpdesk.lastpass.com/lastpass-security-challenge/) ”功能，可以让你在数据库中扫描泄露的密码；一直流行的 1Password 也有类似的“ [瞭望塔](https://support.1password.com/watchtower/) ”功能。

### 收集

无需寻找第三方扩展来体验微软 Edge Chromium 的“收藏”功能，这是一种从许多不同网站组织和导出保存信息的简单方法。多亏了一点小技巧，你现在可以在 Edge Chromium 的稳定版本中启用它。

若要开始，请找到用于启动 Edge 的快捷方式。右键单击它，选择“Properties”，并查找“Target”行。它应该以类似于`msedge.exe”`的东西结束，但是它也可能有一些额外的数据附加在后面，比如`--profile-directory=Default`

在最后出现的代码之后，您需要添加以下代码(在它和最后出现的代码之间留一个空格):

`--enable-features=msEdgeCollections`
