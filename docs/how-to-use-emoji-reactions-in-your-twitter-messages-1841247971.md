# 如何在你的推特信息中使用表情符号反应

> 原文：<https://lifehacker.com/how-to-use-emoji-reactions-in-your-twitter-messages-1841247971>

短信反应——这是 iPhone 和 iPad 用户应该熟悉的一种做法，你可以长按一条信息，在上面添加一个小心形或竖起大拇指/竖起大拇指——是传达反应的简单方式，而不必想出有趣的事情来说。我发现它们在我的 iPhone 上节省了大量时间，我很兴奋地看到它们现在也扩展到了 Twitter 这个伟大的社交自助餐。

是的，你现在可以对每个人在 Twitter 上的简洁评论做出类似表情符号的快速反应，但前提是他们已经通过直接信息联系上了你。不幸的是，你不能对该服务上的任何推文做出反应——只能对消息做出反应——但这项功能应该可以在你使用 Twitter 的任何平台上工作，无论是 Android、iOS 还是 web。

首先，打开直接消息对话，将鼠标悬停在帖子中的任何评论上，然后单击看起来像一个带加号的心形图标。(如果您使用的是手机，请双击该信息。)

Twitter 会在你的主要 Twitter 时间线上推出这个功能吗？我不会屏住呼吸，尤其是他们似乎热衷于 [而完全忽视了](https://www.tubefilter.com/2019/03/14/twitter-removing-likes-retweets-ratio/) 的点赞和转发。想象一下，如果新的“比率”对一个人的空洞帖子进行各种各样的否定(或抨击)，那会是怎样的丑闻。

严肃地说，我认为 Twitter 目前可能会在你的主要 Twitter 时间表中坚持简单、积极的喜欢，但至少当有人进入你的 DMs 时，你可以稍微更多地表达自己。Twitter 指出，如果有人使用的 Twitter 版本太旧，以至于他们无法看到这些反应，他们将获得“以文本信息显示的反应”。(然后他们也会知道作为一个安卓用户被困在与苹果粉丝的群聊中的恐怖。)
