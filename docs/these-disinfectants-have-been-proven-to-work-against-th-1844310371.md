# 这些消毒剂已被证明对冠状病毒有效

> 原文:[https://life hacker . com/这些消毒剂已被证明对 th-1844310371](https://lifehacker.com/these-disinfectants-have-been-proven-to-work-against-th-1844310371)

有 [431 消毒剂](https://www.epa.gov/pesticide-registration/list-n-disinfectants-use-against-sars-cov-2-covid-19) 美国环保局称将杀死新型冠状病毒病毒，这种病毒导致新冠肺炎。他们上榜是因为他们能杀死其他类似的病毒。但是现在两种消毒剂已经测试成功，对抗你所担心的真正的 T4 冠状病毒。

这些产品是来苏消毒剂喷雾(环保局登记编号 777-99)和来苏消毒剂最大罩雾(环保局登记编号 777-127)。他们已经被列入 T2 T3 名单，这是被批准用于对抗新冠肺炎的消毒剂名单。

另一个 418 呢？根据环保局关于“新兴”病原体 的规定，它们仍被视为获准用于对抗新型冠状病毒、 [，即尚未出现足够长时间的全新病原体，以供所有公司测试其产品对抗这些病原体。名单上的其他消毒剂仍有望与之对抗。如果你已经买了清单上的消毒剂，你就不必再去买新的了。](https://www.epa.gov/sites/production/files/2016-09/documents/emerging_viral_pathogen_program_guidance_final_8_19_16_001_0.pdf)

如果你想检查你的消毒剂是否在清单上，在标签上找一个 EPA 注册号。您可以在第 N 页 的 [列表中按编号查找产品。这一页会告诉你应该怎么做，因为消毒剂通常有多种用途。例如，如果你有擦洗泡沫消毒剂浴室尘垢战斗机，标签上会有使用它杀死轮状病毒的说明。这些是你应该用来消毒可能被导致新冠肺炎病毒污染的表面的说明。](https://www.epa.gov/pesticide-registration/list-n-disinfectants-use-against-sars-cov-2-covid-19)

为了确保您正确使用产品，无论是哪种产品，都要确保产品在建议的“接触时间”内保持湿润(例如，擦洗气泡需要 5 分钟，新批准的来苏消毒剂喷雾需要 2 分钟)。
