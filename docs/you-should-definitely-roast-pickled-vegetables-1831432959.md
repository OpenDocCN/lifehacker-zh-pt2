# 你绝对应该烤腌蔬菜

> 原文：<https://lifehacker.com/you-should-definitely-roast-pickled-vegetables-1831432959>

我喜欢烤蔬菜，也喜欢泡菜。但是拿个泡菜烤着吃？我的朋友们，那是一道美味的小吃，非常适合搭配奶酪、熟食或烈性鸡尾酒。

Watch

这种方法不适合那些上颚容易被压垮的人。与我们的 [盐焗菜](https://skillet.lifehacker.com/brine-a-cauliflower-just-as-you-would-a-turkey-1830285457) 相似，你会得到一种漂亮的褐色蔬菜，完全渗透了盐和调料。烘烤减少了水分，赶走了一些腌渍酸，强化了其他更辛辣的味道。大橄榄被转化成耐嚼、超级美味的小块食物(特别是如果它们塞满了羊乳酪)，而小橄榄则被呈现出令人愉快的清淡和松脆(在我看来，是沙拉的完美选择)。腌制的胡萝卜变得又甜又像肉干，朝鲜蓟的心变得非常脆，蘑菇(虽然严格来说不是蔬菜)变成了咸味小鲜味炸弹。

最棒的是，你不用调味。如果你的蔬菜是腌制过的，直接把它们从罐子里转移到有箔衬的盘子里。如果它们一直泡在水基盐水中，先用纸巾稍微吸一下。在 375 度的烤箱里烤 30 到 45 分钟，直到泡菜起泡变成褐色。把它们加入丰盛的沙拉中，放在奶酪盘里，或者用牙签戳戳，就成了非常精致的酒吧小吃。
