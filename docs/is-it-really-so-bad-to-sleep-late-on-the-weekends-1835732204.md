# 周末睡懒觉真的有那么不好吗？

> 原文：<https://lifehacker.com/is-it-really-so-bad-to-sleep-late-on-the-weekends-1835732204>

在一个完美的世界里，我们都会准时上床睡觉，享受七到九个小时的睡眠，并且每天早上在同一时间醒来，精神焕发。虽然这可能是我们的目标，但我猜我们大多数人都喜欢休息一天所带来的额外睡眠。那么周末睡懒觉有多糟糕呢？

### 这取决于你是否有足够的睡眠

睡眠科学家艾米·本德 说:“如果你周末不得不睡懒觉，比平时多睡一个多小时，那么这表明你一周睡眠不足。”。这比你的闹钟时间本身更可能是你的问题。

我们许多人的起床时间在休息日和工作日或学校日之间相差两个小时或更多，这表明我们的生物钟与我们被迫生活的时间表不同步。这种现象有时被称为 [社交时差](https://srbr.org/of-larks-and-owls-whats-in-a-chronotype/) ，对青少年和任何天生的夜猫子来说都是最严重的。(随着年龄的增长，我们中的许多人越来越擅长在早上醒来。)

一些研究表明，不一致的睡眠时间可能比睡眠不足但保持睡眠时间一致对我们的身体更有害。如果这是真的，那就支持我们每天早上把闹钟设在同一时间。然而，该研究“不一定是结论性的”，Bender 说，所以她不会建议以一致性的名义故意少睡。“我认为需要做更多的研究。”

如果你不能早点上床睡觉，午睡可以弥补失去的睡眠。Bender 推荐两种类型，取决于你的时间和你需要多少睡眠:

*   90 分钟的小睡会让你进入一个完整的睡眠周期，包括快速眼动睡眠。为了获得最佳效果，*不要*设置闹钟。
*   小睡 20 分钟可以消除疲劳。将闹钟设定为 30 分钟，这样在你闭上眼睛之前，可以给你一点时间静下心来放松一下。

### 你可以“储存”睡眠以备后用

如果你的日程表上有一些你知道会导致睡眠减少的事情，比如旅行，或者清晨的比赛，你可以提前储备睡眠。这可能意味着连续几个晚上都要早睡。“你有盈余，你处于睡眠的最大潜力，”本德说，“然后这有助于你度过睡眠剥夺。”她特别建议运动员在比赛前进行储备睡眠。

尽管如果可以的话，每天晚上睡个好觉可能会更好，但如果你知道自己在某一天很难获得足够的睡眠，那么银行业务仍然是一个有帮助的策略。考虑小睡和早睡，不仅仅是睡懒觉，周末的充足睡眠可能有助于你在一周的晚些时候不会感到太累。
