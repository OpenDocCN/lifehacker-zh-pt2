# 用芫荽茎做墨西哥风味米饭

> 原文:[https://life hacker . com/make-chipotle-style-rice-with-cilanto-stems-1832327493](https://lifehacker.com/make-chipotle-style-rice-with-cilantro-stems-1832327493)

当谈到芫荽叶时，叶子得到了大部分的关注，但聪明的芫荽叶食用者知道囤积茎。你可以把它们和你剩余的厨余 一起扔进你的原料袋里，或者你可以把它们切碎，用它们来调味一大堆松软的、 [墨西哥风味香菜石灰饭](https://skillet.lifehacker.com/make-chipotle-style-rice-with-cilantro-stems-1829820745) 放在你的电饭锅或速溶锅里，省得你跑一趟自己做卷饼店。
