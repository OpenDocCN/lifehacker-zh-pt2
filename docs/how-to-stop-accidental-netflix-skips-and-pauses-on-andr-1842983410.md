### 如何在 Android 上使用网飞的新锁屏功能

在应用程序中播放内容时，可以找到网飞的锁定按钮，它位于时间轴清除器的正下方。按下它会禁用播放 UI 和应用导航按钮，并用一个“锁定”按钮替换整个界面。下面是一个来自[9 到 5](https://9to5google.com/2020/04/20/netflix-android-screen-lock-accidental-touches/)的 GIF 图片示例:

我猜这在技术上更像是一个“隐藏”功能，而不是真正的“锁定”，但结果几乎是一样的:不再需要来回寻找你的位置。

H 下面是如何使用屏幕锁定功能 :

1.  打开网飞，开始一部电影或一集，并进入全屏模式
2.  点击屏幕，调出屏幕控制。
3.  点击**“屏幕锁定”**
4.  您应该会看到一个挂锁图标弹出消息“屏幕锁定。点击以解锁。这将在几分钟后消失，现在您可以观看您的流，而不用担心意外暂停或关闭窗口。
5.  如果你真的需要暂停、快进、调整音频或关闭应用，点击屏幕和然后点击挂锁图标解锁控件。
