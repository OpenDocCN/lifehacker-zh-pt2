# 为什么你应该参加首次购房者班

在网上浏览房屋清单或者看像《寻房者》这样的节目很有趣。但是如果你不是房主，你可能会对如何开始实现你的美国梦感到迷茫。

电视节目很少讨论花几十万美元买一个住处的细微差别，除此之外，还有大量关于买房的相互矛盾的信息。家人和朋友通常很乐意分享他们的经历，但我大胆猜测，最愿意提供建议的人是那些最近没有真正完成购房的人。房地产市场在短短几年内会发生很多变化，虽然你喜欢凯西阿姨，但你可能会喜欢另一种视角。

### 谁应该参加置业班？

这就是购房者教育课程的价值所在。我知道，不是每个人都会在作为成年人忙碌了一整天后，欣然接受坐在教室里的机会。但这是一个获得关于你可以从购房过程中期待什么的公正信息的机会，也是你最终一劳永逸地弄清楚什么是托管的机会。

此外，这些课程通常是参加首次购房者计划的先决条件，这些计划可以使房屋所有权在经济上更加可行。

在你注册之前，这里有一些你需要知道的。试想:你可以准备买一套房子*并且*能够在你的下一次 HGTV 马拉松中主宰你的沙发评论。

### **哪里找好的**

要找到提供购房者教育课程的当地机构，请访问美国住房和城市发展部的 [经批准的住房咨询机构目录](https://apps.hud.gov/offices/hsg/sfh/hcc/hcs.cfm) 。您可以通过在结果页面上输入您的城市或邮政编码来获得您所在州的结果并缩小列表范围。除了购房者教育课程，你还可以看到哪些中介机构提供其他房屋资源，如购房前咨询。

另一个寻找当地工作室的地方是国家房屋所有权教育和咨询行业标准 T2 T3。其目录中列出的机构符合 HUD 的要求，并且也采用了这种类型的咨询和教育的一套基线标准。

参加由 HUD 认证的课程是必不可少的，因为它通常是像首付援助或首选抵押贷款利率这样的买方援助计划所必需的。你需要提供成功完成课程的证明文件，比如证书。有些项目要求你在过去一两年内参加过这样的课程。

你也可以在网上找到课程。 [One Lifehacker 读者](https://twocents.lifehacker.com/1835589585) 推荐框架，符合 HUD 指南的在线课程。如果你的时间表排得满满的，或者你更喜欢按自己的节奏学习，那么在线课程可能更适合你。

无论你是亲自参加还是在线参加，你都可以期望一门课程持续四到八个小时。如果你在网上上课，你可能需要在一定的时间内完成课程，比如 30 天。

### **成本**

一门课程的费用会因主办者的不同而不同。一般来说，预计至少要支付 75 美元。但是你可以在你所在的地区和网上随意选择。例如，东北密歇根社区服务机构提供的一个在线课程允许你花 99 美元在线注册，但对在注册前联系他们的人提供折扣。

如果你打算和伴侣一起买房，你们可能都需要参加购房者教育课程。向你当地的住房组织了解他们对参加课程的夫妇的要求，以及每个人是否必须支付全部课程费用。

不要感到有压力去参加你在寻找中遇到的第一门课程。你可以货比三家，在时间、费用和课程方面找到适合你的课程。

### **你学到了什么**

这里有一个你可以期望了解的主题的总结，由国家房屋所有权和咨询行业标准概述:

*   **房屋所有权基础知识**，比如房屋所有权的利与弊以及房屋购买过程的步骤。
*   买房所需的预算和财务事项，比如计算你的购房预算，在申请抵押贷款前确保你的信用可靠。
*   **房屋融资**，从抵押贷款资格到抵押贷款类型，再到常见的贷款文件和结算流程。(Psst:这是关于代管的部分！)
*   **买房子**，知道你需要和谁一起工作，以及如何协商价格。
*   **维护你的家**，包括购买价格之外拥有一个家的成本。你还将了解掠夺性贷款风险，以及当你面临财务困难时如何避免丧失抵押品赎回权。

这些东西在纸上可能很无聊，但它会带走你在听到“”和“[【PMI](https://lifehacker.com/can-you-get-a-mortgage-with-2-to-3-down-1827110213)”等术语时可能会感到的一些即时焦虑

当你与专业人士如房地产经纪人或贷款人交谈时，这也会给你带来优势。在开始找房子之前，你获得的知识越多，你就越有信心离开那些太咄咄逼人或让你心惊肉跳的人。

### **接下来会发生什么**

当你完成一个购房者教育课程时，你可以决定做什么。你可能会马上加入竞争的要求，申请你所在地区的首次购房者计划，或者你可能会决定推迟你的购房计划。你可能在下课后意识到你根本不想拥有一个家。无论哪种方式，你都将拥有帮助你做出决定的信息。而且你一定会知道什么是“代管”。
