计算尺是一种多用途的计算器工具。在计算器和现在无处不在的电话和电脑出现之前，人们就是这样做数学的。它看起来像一把尺子，但中间有一个滑动部分。你可以用它来快速地乘和除大数，如果你是一个计算尺高手，你甚至可以做指数、根和三角学。

你不必去找老式的计算尺来玩:这里有一个虚拟的计算尺。计算尺可以变得非常花哨，但基本类型有一个*主体*，一个*滑道*(沿着中间运行的那个东西)，以及一个*光标*，它给你一条线，这样你就可以准确地将主体和滑道彼此对齐。

看到每个数字旁边的字母了吗？为了乘或除，w e 可以将 C 音阶(在幻灯片的底部)与 D 音阶(就在它旁边，在身体的底部)一起使用。准备好了吗？

### 如何乘法

假设你想将两个数相乘，也许是 32*45。看看你的计算尺。C 音阶在滑梯上，D 音阶在机身上。

现在看看每个刻度上的数字。有...很多数字。在这里的例子中，你可以看到 C 和 D 音阶都有数字 1，然后数字 1 *，*再到数字 9，最后是数字 2。所有这些数字实际上都代表 1.1、1.2、1.3 等等。所以当我们继续我们的例子时，确保如果你在寻找数字 3，你会找到实际的数字 3 而不是 1.3。

好吧。你准备好了吗？我们试试 32 × 45。每个步骤的照片都在上面的幻灯片中。

1.  将光标的发际线放在 D 刻度上要相乘的第一个数字上(让我们用 32)。要得到 32，你必须寻找 3，然后在它后面两个散列符号。(换句话说，现在你用的是 3.2 而不是 32。你很聪明，你会记得在你的最终答案中修正小数。)
2.  移动幻灯片，使索引(幻灯片上 C 刻度的最开始*或*端)与光标上的发际线对齐。如果我们在标尺的开始使用索引，我们的幻灯片将会滑动得很远，以至于我们不能移动光标来得到答案。所以我们在最后会用到索引。
3.  找到你的另一个数字，45(实际上是 4.5)，在 C 音阶上(在幻灯片上)。将发际线移动到这个数字。D 音阶上(身体上)的匹配数字就是答案。是 1.44，但是我们对小数很聪明，记得吗？所以我们的答案其实是:32 × 45 = 1440。

除法也差不多。假设你想做 16 ÷ 3。

1.  在 D(身体)尺度上找到 16。
2.  滑动标尺，使 D 标尺上的 16 与 C 标尺上的 3 对齐。
3.  C 音阶的索引会指向 D 音阶的答案。在这种情况下，5.3 和一点点。(现代计算器告诉我答案是 5.33333 重复。)
