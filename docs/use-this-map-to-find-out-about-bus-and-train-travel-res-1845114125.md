# 使用这张地图找出公共汽车和火车旅行的限制

> 原文：<https://lifehacker.com/use-this-map-to-find-out-about-bus-and-train-travel-res-1845114125>

在过去的六个月里，围绕 COVID 旅行选择的讨论主要集中在飞行的安全性和公路旅行的回程上。但是，这让那些主要通过火车或公共汽车出行的人怎么办呢？我们不是在谈论 [公共交通](https://lifehacker.com/how-to-stay-as-safe-as-possible-on-public-transit-right-1845037829) 的安全问题(尽管考虑这一点很重要)，而是美国国家铁路客运公司、灰狗、Megabus 和各种区域公交公司的长途旅行。



这带来了一堆问题:它安全吗？它如何与各州的旅行限制政策配合？公共汽车和火车恢复正常运行了吗？为了帮助我们应对这些新的旅行挑战， [地面旅行预订平台 Wanderu](https://www.wanderu.com/blog/covid19-travel-guide) 的团队整合了互动地图和其他资源，提供新冠肺炎期间火车和公共汽车旅行的最新信息。以下是需要了解的内容。

### 地图和其他资源

像其他旅游网站一样，Wanderu 已经创建了 [一个互动地图](https://www.wanderu.com/blog/covid19-travel-guide) 允许用户点击一个州来找出其特定的与 COVID 相关的旅行限制(如果有的话)。加拿大各省和欧洲国家的地图也是如此。地图上的所有信息也可以在下面的列表中找到。

但让这种特殊资源独一无二的是页面下方稍远一点的，在这里你会找到针对北美公司的 [变更和取消政策](https://www.wanderu.com/blog/travel-alert-covid-19)[口罩政策](https://www.wanderu.com/blog/face-mask-policies) 和 [清洁协议](https://www.wanderu.com/blog/coronavirus-bus-train-sanitize)[行程保护](https://www.wanderu.com/blog/travel-alert-covid-19/#protection) 以及一份 [承运商列表【相关信息】](https://www.wanderu.com/blog/resumed-bus-train-services)

那么，火车和公共汽车旅行恢复正常了吗？根据 [Wanderu](https://www.wanderu.com/blog/covid19-travel-guide) ，不确切地说:

> 虽然自 3 月中旬以来，一些公共汽车和火车运营商仍未上路，但大多数运营商已恢复服务。然而，由于旅行需求减少，目前运营的大多数公共汽车和火车运输公司都在有限的时间表上运营。

最后，还有一个页面提供了每个国家和地区的公共汽车和火车服务的 [联系方式](https://www.wanderu.com/en-us/carrier) ，以防您需要直接与他们取得联系。
