# 使用 DollarBird 进行简单的实际预算

> 原文：<https://lifehacker.com/use-dollarbird-for-easy-hands-on-budgeting-1830944057>

市场上不乏预算应用程序，但有时，与其用类别、图表、图形和每天提醒你超出预设预算或在某个类别(咳咳，薄荷)中有“不规则支出”，不如用一些简单的东西来跟踪你的支出。



[多拉比尔](https://dollarbird.co/) 填补了这一空白。用户手动在日历上填写他们的每月预算和支出，这样，这是一个很好的实际跟踪和记录你的支出的应用程序。你不用看应用程序为你提供的摘要，而是自己填写每一笔费用。

该应用程序使用起来简单直观。在免费版中，你可以输入每月预算，然后每天都可以将支出(或收入)添加到日历中，就像在 iCal 中添加事件一样。你还可以记下你把钱花在了什么地方，应用程序会为你分类，你可以按天、周、月或类别查看你的交易。

你可以升级到每月 4.99 美元或每年 39.99 美元的版本，这样你就可以访问更多的日历(这可以让你跟踪不同的帐户)，并让你与其他人合作。

作为一个很难坚持使用理财应用的人，我很喜欢使用它，因为它模仿了我已经在我的计划中做过的事情。不过，与我的计划器不同，我的手机总是带在身边，这意味着我可以抓住那些我可能会忘记的小开销。并非每个人都是这样，所以使用对你有用的东西，但我发现依赖 Mint 总是意味着我时不时地在逃避，避免打开应用程序。有了 DollarBird，输入我正在花的钱并知道我可以在哪里做出改变已经成为一种习惯。

最好的一点是，如果你担心将所有不同的金融账户连接到一个应用程序，你不必连接任何东西。你在手动输入信息。不仅事情不会被误解，而且你不需要担心应用程序会收集你所有的私人财务信息。还是那句话，用对你有用的东西；我喜欢 DollarBird 的简洁。

适用于 [iOS](https://itunes.apple.com/us/app/dollarbird/id1214299218?mt=8) 和 [Android](https://play.google.com/store/apps/details?id=co.dollarbird&hl=en_US) 。
