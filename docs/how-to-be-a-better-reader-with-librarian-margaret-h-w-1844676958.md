# 如何成为一个更好的读者，图书管理员玛格丽特·h·威廉森

> 原文：<https://lifehacker.com/how-to-be-a-better-reader-with-librarian-margaret-h-w-1844676958>

本周，在图书管理员兼播客玛格丽特·h·威廉森的帮助下，我们将学习如何成为更好的读者。请听 Margaret 分解我们应该如何思考我们对成为 【好】或“博览群书”的读者的定义是什么意思，以及我们可以用来改进我们自己的阅读游戏的策略——从利用有声读物到放弃此刻不吸引我们的书籍。

bOfvBY">我也鼓励人们将有声读物视为阅读。我想很多人都是这样的，哦，如果我听一本书，那我就没读过。它们是不同的处理经验，但没有一个比另一个更有效或更无效。我发现有声读物更容易融入我的时间表，因为我可以打扫房间，放上一本有声读物。我可以带着有声读物去散步。所以尤其是现在，当你很难为自己开拓时间的时候，这是寻找新作家的好方法。</p></blockquote><p class="sc-1bwb26k-1 kpbNNd" id="h1603"><strong>来自南希·珀尔的采访</strong></h4><p class="sc-77igqf-0 bOfvBY">关于快速找到一部快节奏、情节驱动的小说的最佳方法:</p><blockquote data-type="BlockQuote" class="sc-8hxd3p-0 ehyFRB"><p class="sc-77igqf-0 bOfvBY">找到一本快速流动的书的方法是去书店或图书馆的普通小说部，或者去推理小说部打开一本书，看看书页上有多少空白。如果页面上有大量的空白，这意味着这本书将会进展得很快，因为页面上大量的空白意味着大量的对话。当有很多对话的时候，所有的事情都以极快的速度进行。</p></blockquote><p class="sc-77igqf-0 bOfvBY">关于如何轻松找到角色驱动的书:</p><blockquote data-type="BlockQuote" class="sc-8hxd3p-0 ehyFRB"><p class="sc-77igqf-0
