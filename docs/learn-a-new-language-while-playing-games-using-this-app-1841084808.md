# 使用此应用程序玩游戏时学习一门新语言

> 原文：<https://lifehacker.com/learn-a-new-language-while-playing-games-using-this-app-1841084808>

说到学习一门新语言，多林哥语通常是我的首选。我已经到了一个点，我已经确定我可能永远也不会完全流利地使用另一种语言，但是每当我旅行到一个我通常不会说母语的地方，我都会在到达那里之前努力把基础知识记下来。这通常意味着，在任何大型旅行前的几周，我通常会每天花 10-15 分钟在网站上进行培训，这样我就不会完全迷路。

Watch

Duolingo 很棒，但有时你可能想尝试一些不同的东西。 [Vocly 是一款应用](https://simyasolutions.com/vocly-app/) ，可以通过玩游戏帮助你建立那些语言技能。

像 Duolingo 一样，Vocly 建议每天只花几分钟来提高你的语言技能。它的建议只有 7 分钟，这意味着你可以在上班的火车上或在杂货店排队时进行训练。

你会希望能够听到应用程序，所以如果你在外面走动，耳机会很有用。

一旦你开始上课，屏幕上方的一个计数器会告诉你还剩多少时间。他们对这七分钟是认真的，所以除非你想，否则你不会被卷入更长的时间。

通过一些训练，一切都足够简单，你不会被难住，但足够重复，你实际上在一路上学到了新的词汇，当你完成时，感觉你学到了一些东西。

如果你已经在做一个语言项目，那么它可能是一个很好的补充，可以帮助你加快学习进度。
