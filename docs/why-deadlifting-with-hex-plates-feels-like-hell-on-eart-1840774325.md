# 为什么用六角板硬拉感觉像在人间地狱

> 原文：<https://lifehacker.com/why-deadlifting-with-hex-plates-feels-like-hell-on-eart-1840774325>

如果你回家度假，去当地的健身房锻炼，你现在可能知道，不是所有的健身房都是平等的。有些本质上很差，堆满了老旧生锈的重物和不平的地板。但是，即使是在一个全新的、设备齐全的健身房，事情有时也会让人感觉不一样，因此，难度会成倍增加。杠铃可以感觉掉了。长凳会感觉不舒服。见鬼，即使是跑步机模型也可能不是你已经习惯的那种，跑步会感觉有点奇怪。



更糟糕的是，这个健身房有六边形板——这是最简单的增加你的硬拉的方法。这就是为什么六边形的盘子，比圆形的盘子要差。如果你从未见过六角形盘子，它有平的边；圆板，顾名思义，没有平边。当你用圆盘子硬拉时，盘子碰到地面并保持静止，就像它们应该的那样。当你用六角板硬拉的时候，嗯，很混乱。

你看，当你放下杠铃时，六边形板倾向于*滚动*。“它们基本上仍然会动，当你把它放下时，它要么会向你扑来并抓伤你，要么会滚开，导致有人被它绊倒，”Luis Cornier， [Precor](https://www.precor.com/en-us/resources/meet-precor-master-coaches-luis-cornier) 的私人教练和主教练通过电子邮件告诉我；这里有一个 [视频](https://www.youtube.com/watch?v=X3apGob1t40) 演示了这个问题。接下来的问题是，你必须不断地确保杠铃与你的身体保持适当的对齐，否则在举杠铃时你或其他人可能会受伤。(这里是 [另一个视频](https://www.youtube.com/watch?v=_Y1qIas11NU) 展示了提升六角板和圆形板之间的区别。)

过去一周，我在一个只有六角板和硬拉的健身房，这是我最喜欢的举重，变成了一个彻头彻尾的讨厌鬼。在代表之间完全重置没有太大帮助；有时，当我在地上抓住杠铃时，杠铃会前后滚动，不管我多么集中精力确保杠铃的一侧不会比另一侧更近或更远。

那么，为什么健身房选择可怕的六边形板呢？嗯，你 [可能会在网上读到](https://forum.bodybuilding.com/showthread.php?t=174796141&page=1) 一些商业健身房可能会专门使用它们来阻止严重的硬拉(更少的噪音，更少的人使用杠铃，等等)。)不管是什么原因，如果你发现自己处于这种情况，首先，很抱歉这种情况发生在你身上。第二，你无法改变它。你可以在盘子的两边放一个小重物来减少滚动，但这只能达到这么多。不管怎样，它可能会反弹或者滚过重物。

如果你愿意稍微挥霍一下，甚至有一种产品可以把六边形盘子变成圆形的 plates⁠—a“[【hex bumper](https://www.youtube.com/watch?v=NhECwIhJoq8)”⁠—but 也许值得去找一个新的健身房，仅此而已。 [下面是如何开始那个过程](https://vitals.lifehacker.com/how-to-find-the-right-gym-for-you-1709198069) 。
