# 这些天然驱蚊剂实际上会让你不被吞噬

> 原文：<https://lifehacker.com/which-natural-mosquito-repellents-actually-work-1833916649>

当我在外面享受大自然时，在全身喷洒合成化学物质似乎有点奇怪。所以当我买杀虫剂时，我总是看一看“天然”驱虫剂。有些是合法的，含有美国环保署注册的成分 ，可以有效驱除细菌，同时对皮肤安全。其他的是由芳香的药草和一厢情愿的想法组成的。

## 天然驱蚊剂中最重要的成分

购买天然驱蚊剂时，1 升柠檬桉树油 就是你要找的成分。(它也可以被标记为对薄荷烷-3，8-二醇或 PMD。)它存在于许多“天然”杀虫剂中，它已经获得了美国环保局的注册，T4 的研究表明它提供了与普通避蚊胺喷雾剂相似的保护水平。唯一的缺点是不建议 3 岁以下的儿童使用。

但要确保是真货。“柠檬桉树精油”可能听起来很相似，但它不是你想要的东西。如果你不确定，可以在标签的小字上找一个 EPA 注册号。

避免只含有植物成分的喷雾，如混合精油、香叶醇和其他成分。它们可能闻起来不错，但没有任何确凿的证据表明它们真的有效。

如果你找不到天然的驱虫剂(或者如果你需要在小孩身上使用)，要知道合成的驱虫剂实际上对你没有坏处。如果你按照指示使用避蚊胺是没问题的(*把它喷在你的皮肤和衣服上，*不要喝它)。Picaridin 可能不会出现在“天然”标签下，但它类似于黑胡椒中发现的一种化学物质，而且它在驱蚊方面也是安全有效的。在《消费者报告》的测试中, 中，这两种驱虫剂与柠檬桉树油一起，一直是 [胜过其他类型的驱虫剂。就我个人而言，我选择这三种中的任何一种，不用担心标签上是否写着“天然”。](https://vitals.lifehacker.com/consumer-reports-mosquito-repellent-rankings-are-now-a-1779140735)**
