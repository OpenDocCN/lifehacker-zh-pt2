# 用第三人称对自己说话，以提高清晰度

> 原文：<https://lifehacker.com/talk-to-yourself-in-the-third-person-for-improved-clari-1837084747>

唐纳德·特朗普、Flav 和 Elmo 有什么共同点？这个问题可能有多种可接受的答案，但我们今天寻找的答案是，他们都以第三人称谈论自己。现在，这并不完全是赞成以这种方式交流的响亮投票，但听我们说完:有研究表明，以第三人称 说话的 [实际上可以帮助我们思考各种情况，并有助于情绪调节。](https://aeon.co/ideas/why-speaking-to-yourself-in-the-third-person-makes-you-wiser)

加拿大滑铁卢大学 的一项研究声称是“第一个证据，证明与智慧相关的认知和情感过程可以在日常生活中训练，以及如何训练。”基本上，它发现非法行为——用第三人称称呼自己的专业术语——可以帮助我们稍微改变感知事物的方式，让我们与自己保持一定的距离。

不仅如此，与我们对以第三人称谈论自己的人的看法相反，研究表明，这样做可能会让我们变得更谦虚，更愿意考虑其他观点(尽管上面的例子表明了这一点——一分钟后会有更多的讨论)。

“第三人称自言自语在这里很有帮助，当它被用来*反映事件的*。也就是说，在反思中，使用第三人称视角会与眼前的自己产生距离，“加拿大滑铁卢大学心理学副教授、 [Igor Grossmann、](https://uwaterloo.ca/psychology/people-profiles/igor-grossmann)T5】博士、该研究的主要作者告诉 Lifehacker。

格罗斯曼说，但是*不*起作用的是出于自私目的使用第三人称自言自语，比如谈论自己，好像自己是一个品牌，或者捍卫自己的名字。“在这种情况下，甚至可能适得其反，”他补充道。

所以，不，你不需要整天都用第三人称来称呼自己，但是这可能是你下一次需要清晰的时候想要尝试的。
