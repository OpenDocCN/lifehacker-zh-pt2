# 间歇性禁食的最佳应用

> 原文:[https://life hacker . com/the-best-apps-for-intermittent-fasting-1836824408](https://lifehacker.com/the-best-apps-for-intermittent-fasting-1836824408)

间歇性禁食并不神奇，但对于那些设法坚持的人来说，它 [是一种很好的减肥工具](https://vitals.lifehacker.com/we-finally-have-long-term-data-on-an-intermittent-fasti-1794877196) 。如果是你，考虑使用这些禁食追踪器来提醒你开始和结束时间，并记录你过去坚持计划的情况。

Watch

所有的禁食应用程序都是美化的计时器。你告诉他们你什么时候开始禁食，他们会倒计时，直到你可以再次进食。当你在喂食窗口时，他们会告诉你还剩多长时间。除此之外，他们可以像你喜欢的那样花哨或简单。

### 最佳全能:零分

[零点](https://apps.apple.com/us/app/zero-fasting-tracker/id1168348542)(iOS 上免费)是很多间歇性禁食者的最爱。它有一个简单的开始屏幕，你可以选择一个预设的禁食类型(16 小时禁食，8 小时进食是一个受欢迎的类型)或设置自己的。该应用程序会跟踪你在当前窗口的位置，并在结束时向你发送通知。有一个仪表板可以记录你的平均禁食时间，你是否已经连续多天保持，还有一个地方可以查看你的体重趋势(如果你已经连接到健康应用程序)。设计简单，应用程序易于使用。~~是 iOS only~~ ( *更正:不是！我们莫名其妙地错过了* [*【安卓清单】*](https://play.google.com/store/apps/details?id=com.zerofasting.zero&hl=en_US) *！我们仍然想指出的是*[*Vora*](https://play.google.com/store/apps/details?id=com.fastapp&hl=en_US)*是一款同样广受欢迎的 Android 应用。*

### 更简单的应用程序:快速客户端、快速习惯

如果你不需要提前安排斋戒， [Fastient](https://fastient.com/) 是一个非常简单的应用程序，让你开始斋戒，选择性地设定目标，并添加关于你感觉如何的笔记。

FastHabit 甚至更简单:不需要创建一个帐户，你就可以得到一个当前禁食的计时器和一个你以前禁食的日志。你需要升级到 2.99 美元的版本来获得每天斋戒的提醒。
