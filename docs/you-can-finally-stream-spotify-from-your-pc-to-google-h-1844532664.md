# 你终于可以将 Spotify 从你的电脑上传到谷歌家庭音箱了

> 原文：<https://lifehacker.com/you-can-finally-stream-spotify-from-your-pc-to-google-h-1844532664>

Spotify 几乎可以在我们能想到的所有流行设备上使用。无论你是在智能手机、电脑上还是通过智能电视应用程序听音乐，每个应用程序都做着几乎相同的事情，但对桌面用户来说有一个奇怪的限制:你不能从电脑 将 Spotify 转换到“谷歌播放”设备，如 Chomecast 或谷歌智能扬声器[——至少，如果你是 macOS 或 Windows 系统，就不能。](https://community.spotify.com/t5/Desktop-Windows/Spotify-Connect-amp-Google-Home/td-p/4642461) 



最新版本的 Spotify 桌面播放器(终于)修复了这个缺点。Mac 和 Windows 用户现在可以通过 Spotify 桌面播放器的“已连接设备”选项卡连接到您的谷歌设备。这与将您的 Spotify 音乐从 PC 传输到外部扬声器的过程完全相同。

### 如何将 Spotify 投射到谷歌音箱和流媒体设备上

1.  打开你的电视。如果它是一台内置 Chromecast 的智能电视，请确保它与你的 PC 或笔记本电脑连接到同一个 wifi 网络。如果您使用外部 Chromecast 或带有内置 Chromecast 的设备，如 Nvidia Shield，请确保它已插入电视、打开并连接到网络。
2.  打开电脑上的桌面 Spotify Player 应用程序，确保根据提示安装任何更新。如果你还没有安装它，你也可以从[【spotify.com/download】](http://spotify.com/download)下载。该应用程序在 Mac 和 Windows 上都可用。
3.  点击播放器左下方的**“设备可用”**图标(就是队列和音量控件之间的图标，如上图截图所示)。
4.  从可用设备列表中选择您的 Google speaker 或 Chromecast。
5.  等待设备连接，然后就可以开始了。选择要听的东西，它会流过你选择的任何东西。

这一修复已经进行了多年，但我们不能再兴奋了，它在这里。你不再需要在手机上启动 Spotify，将它传输到你的谷歌扬声器，然后跳到你的电脑上继续互动；现在，您可以在办公桌上完成所有工作。
