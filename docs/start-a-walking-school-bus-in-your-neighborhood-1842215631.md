# 在你家附近开一辆“步行校车”

> 原文：<https://lifehacker.com/start-a-walking-school-bus-in-your-neighborhood-1842215631>

九岁时，我的儿子和他的朋友最渴望两件事:更多的独立和更多的时间出去玩。我们也正好在从我们家到他学校的中间距离——我们住得太近，没有资格用校车接送，但又太远，让他一个人走这条路让我感到不舒服。因此，我和其他几位家长想出了一个解决方案:我们正在启动一个社区“步行校车”



事情是这样的:一位家长(不住在我们附近)将把他们的两个孩子送到上班路上最远的房子。孩子们现在总共有四个，他们将离开，向东走几个街区到我家，在那里他们将“接”我的儿子；然后他们会继续沿着预定的路线去学校。人数上会有安全感，他们会得到一点额外的体力活动——以及独立的满足感和额外的联系时间。

我们认为我们在这里是完全创新的，但当我开始搜索这个主题时，我发现 [整个指南都致力于这个想法](http://guide.saferoutesinfo.org/walking_school_bus/) 。一些社区正在组织更大规模的步行校车，每天都有几十名学生和几名家长参加模拟乘车活动。

在它的 [可下载的 PDF](http://www.walkingschoolbus.org/WalkingSchoolBus_pdf.pdf) 中， [国家安全上学路线中心](http://www.saferoutesinfo.org/) 说当你为步行校车选择路线时，问自己这些问题:

*   他们有空间走路吗？有人行道和小路吗？交通太拥挤吗？
*   过马路容易吗？
*   司机表现好吗？他们会屈服于行尸吗？他们超速了吗？
*   环境感觉安全吗？有没有散养狗？有犯罪活动吗？

你的步行校车可以像我们的一样简单，从两个或三个家庭的孩子开始，然后发一条短信让下一个家长知道孩子已经在路上了。或者，它可以更有组织性，有固定的时间表、公共汽车站的位置和一组自愿的成年步行者。

如果孩子很小，或者无法独自行走，或者你面对的是一大群孩子，家长监护可能是成功推出步行校车的关键。但是对于像我儿子这样的小团体中的大孩子来说，让他们独自行走的合适年龄更多的是一个成熟的问题。

在不列颠哥伦比亚大学制作的纪录片 [*奔跑自由:儿童的独立流动性*](https://www.youtube.com/watch?v=OKOXfa8NfTY&fbclid=IwAR3OvbV0IBItRw7XwwJg_TYu1LX4YOLBSBOUCchIok4l2KBdDeNTozSjDxY) 中，一位专家谈到何时应该给予孩子独立步行去学校和其他地方的自由。

“例如，一个孩子什么时候大到可以理解在过马路时向左、向右、再向左看？”加拿大卫生研究院-加拿大公共卫生局(CIHR-PHAC)应用公共卫生主席 Guy Faulkner 问道。

“我不认为需要有一个固定的年龄，独立的行动能力会自动转移给孩子，”他在纪录片中继续说道。“我认为，对于孩子来说，什么时候能够在自己的社区和城市里自由自在地旅行和活动，这在很大程度上取决于他们自己。”
