# Nvidia 和 AMD 之间最无聊的斗争才是最重要的

> 原文：<https://gizmodo.com/nvidia-and-amds-nerdiest-fight-is-the-one-that-matters-1831784022>

在本月早些时候的 CES 上，Nvidia 和 AMD [交换了话语](https://gizmodo.com/nvidia-ceo-trashes-amds-new-gpu-the-performance-is-lou-1831621038#_ga=2.8601077.1721906064.1547472999-1595880060.1539636540) ，Nvidia 首席执行官黄仁勋表示，他的 GPU 将“碾压”AMD 新宣布的镭龙 VII。AMD 首席执行官苏姿丰在她的回应中显得更加，但她没有错过挖苦英伟达的机会，并指出当她的公司最终采用 [【光线追踪](https://gizmodo.com/why-ray-tracing-on-nvidias-new-gpus-is-so-exciting-1828964418) 时，人们实际上就会知道它是什么了。但是，让我们抛开他们在光线跟踪和 GPU 上的争论，专注于两家公司之间的另一场争吵——这一场是关于显示器的，如果你在未来几个月内购买一台游戏显示器，你需要注意。



两家公司都在支持可变刷新率的显示器上小打出手。什么是 VRR？无论是用于游戏、办公还是仅仅是电视，显示器上的显示总是会刷新。60Hz 的显示器每秒刷新图像 60 次，而 144Hz 的显示器每秒刷新 144 次。当显示器刷新时，它从上到下刷新— [如果你把图像放慢速度就能看到一些东西](https://www.youtube.com/watch?v=3BJU2drrtCM) 。

不过，你的电脑并不受显示器刷新率的影响。因此，当您的显示器仅处理 60 帧时，它可能每秒会吐出 120 帧。这就导致了屏幕撕裂。这时，你的显示器开始绘制一幅图像，并意识到计算机就像前面有 60 幅图像，所以它跳过前面来赶上，导致屏幕上出现看起来像 [的奇怪撕裂](https://www.youtube.com/watch?v=jVAFuUAKPMc) 。不是世界末日，但不吸引人！

具有可变刷新率的显示器更加灵活。它知道 GPU 每秒钟是如何绘制这些图像的，并以正确的速度刷新以减少撕裂，使事情变得更顺利。这项技术在实践中确实很漂亮，但是 AMD 和 Nvidia 使用它的方式不同。英伟达的 G-Sync 需要专有技术，在这种情况下，英伟达创建的显示器缩放器内置于显示器中，以与 GPU 快速通信，提供理论上应该更流畅的体验。它有两个版本，G-Sync 将为您提供良好的峰值亮度、体面的背光系统和纯色，以及 [G-Sync 终极 HDR](https://www.nvidia.com/en-us/geforce/news/g-sync-ces-2019-announcements/) ，它要求显示器具有更高的峰值亮度(1000 尼特)、更好的背光系统(全阵列)，并显示更多颜色(DCI-P3 色域)。

AMD 的 FreeSync 基于 VESA 的自适应同步技术，这是一种不同的可变刷新率技术，应该可以与更多的显示器和 GPU 一起工作，因为不需要定制的缩放器。FreeSync 是 Adaptive Sync 的一个分支，很像英特尔承诺的 Adaptive Sync 软件。所以所有的 FreeSync 显示器都是自适应同步显示器，但并不是所有的自适应同步显示器都支持 FreeSync。因为它是开源的，除了 DisplayPort 1.2a 端口之外不需要特殊的硬件，所以 FreeSync 显示器比 G-Sync 更容易制造，也更便宜。

这就是为什么，尽管 74.18%的 Steam 用户 使用英伟达图形处理器来玩游戏，大多数正在制造和销售的显示器仍然是 FreeSync，而不是 G-Sync。2019 年 1 月 24 日，当我在亚马逊 上查看时，第二畅销的显示器有 FreeSync，而最畅销的 G-Sync 排名第 34 位，NPD 硬件分析师 Stephen Baker 告诉我，FreeSync 显示器的销量是 G-Sync 的“3 至 4 倍”。

但据英伟达称，这些销售是以质量为代价的。Nvidia 非常确定 FreeSync 很烂，以至于在 CES 2019 新闻发布会上花了很大一部分时间嘲讽 FreeSync。首席执行官黄仁勋声称，在测试的 400 个自适应同步监视器模型中，只有 12 个实际工作，尽管他没有引用具体细节，但这些自适应同步监视器中有很大一部分是免费同步的。后来，在我参加的一个小型新闻发布会上，他说，“大多数免费同步显示器都不工作。”他继续声称，“他们甚至不能与 AMD 的显卡一起工作，因为没有人测试过它。我们认为，让顾客相信产品的承诺，然后让它失效，这是一个糟糕的想法。”

在紧随 之后的 [圆桌会议上，Nvidia 的新闻发布会上，AMD 首席执行官苏姿丰否认了这一说法。“我不相信我们已经看到了，”她说。第二天，AMD 产品营销总监马赛丽·马林科维奇更进一步，质疑 Nvidia 没有测试它声称已经测试过的每一款 AMD 显示器。“证明给我看，”他在一次采访中告诉我。](https://gizmodo.com/amd-ceo-talks-ray-tracing-development-and-future-window-1831634118#_ga=2.243940485.1077701367.1547493370-2135808592.1525885870)

感觉好像每个人都在互相指责。那么你相信谁呢？对两家公司最大胆的主张提出质疑可能是安全的。记住，大型盈利公司不是你的朋友。当被问及 Nvidia 支持自适应同步的决定时，黄指出了目前市场上自适应同步显示器的质量控制问题，而的 Stephen Baker 则表示，真正的原因是 Nvidia 在显示器领域单独使用 G-Sync 的竞争力不够。

英伟达正在被 AMD 击败，AMD 卖出了更多的显示器，并与更多的公司合作生产显示器。英伟达需要在对手的水平上竞争。“这真的是成本问题，”贝克告诉我，“人们可能会争论技术或其他什么，但只要 G-Sync 的成本大幅增加，这将是一个艰难的销售。”

因此，Nvidia 达到了 AMD 的水平，其新的 G-Sync 兼容显示器抛弃了很酷的 Nvidia scaler，并借鉴了 AMD 的做法，通过软件独家支持自适应同步。Nvidia 产品经理兼 G-Sync 负责人 Vijay Sharma 告诉我，可以把所有不同的标准想象成一棵家谱。

顶层是可变刷新率的概念。由此衍生出三种使用 VRR 的技术。G-Sync 及其自定义缩放器 Adaptive Sync，它依赖于软件和 DisplayPort 1.2a 标准，以及 HDMI VRR，这是一种可变刷新率的 HDMI 版本，正在慢慢获得三星等制造商对电视的支持。

这些技术随后扩展开来。从同步弹簧普通同步和同步终极 HDR。来自 Adaptive Sync 的 springs FreeSync、FreeSync 2(稍后将详细介绍)、英特尔未来的 Adaptive Sync 支持以及新的 G-Sync 兼容标准。从 HDMI VRR 泉，嗯，不是很多— [只是在一组精选的三星电视上免费同步](https://gizmodo.com/one-of-the-coolest-gaming-monitor-features-is-finally-c-1826592901) ，只适用于 Xbox One X 或 Xbox One S

回到这些公司的说法，特别是 Nvidia 反复暗示 AMD 只是在不工作的显示器上打自己的名字。这是真的吗？

不。AMD 正在做的是说，如果你有一个显示器，它说它可以与 FreeSync 一起工作，那么它将与你的 AMD GPU 一起工作，并给你某种可变的刷新率。哪种可以变化很大。当要求显示器每秒刷新 60 至 120 帧时，它可能只会进行自由同步。它可能在 60 岁以下不起作用——这是许多拥有廉价卡的游戏玩家最想要 VRR 的时候。

最初的开源免费同步标准留下了很大的回旋余地。更糟糕的是，很多 FreeSync 显示器出厂时都是关闭的。包括我桌子上的那个！直到我开始写这篇文章，我才意识到它已经默认禁用了 FreeSync。

如果浏览评论网站和生活在显示器的规格表中对你没有吸引力，AMD 指出了更新的 [FreeSync 2 HDR](https://www.techspot.com/article/1630-freesync-2-explained/) ，它给出了一套非常具体的指导方针，显示器必须满足才能被标记为 FreeSync 2。与 G-Sync Ultimate HDR 一样，这些准则包括支持 HDR 和更宽的色域。它还包括所谓的低帧率补偿，或 LFC。这是 AMD 保证游戏玩家即使在性能下降时也能拥有流畅游戏体验的方式。上面我提到过，并不是所有的 VRR 显示器都是一样的，在某些情况下，VRR 只能在有限的刷新率范围内工作。

对于符合 FreeSync 2 HDR 的显示器，它需要足够高的刷新率范围。根据 AMD 的说法，当您将显示器的最高刷新率除以其中点刷新率时，它必须等于 2.5 或更高。因此，144Hz 的显示器和 48Hz 的 mid 可以通过测试。中间为 84Hz(因此最低为 60Hz)的 144Hz 显示器将会失败。Nvidia 新的 G-Sync 兼容显示器也需要一个特定的数字 2.4 才能通过。

不幸的是，显示器制造商不只是在规格表上降低 VRR 运行的刷新率范围。那太容易了！因此，如果您想知道您的显示器或未来的显示器的功能，您必须查看支持显示器的图表。Nvidia 有一款 G-Sync 兼容显示器。 [AMD 的 FreeSync 显示器超级可搜索列表，包括 VRR 范围，可以在这里找到](https://www.amd.com/en/products/freesync-monitors) 。通过使用它，我了解到我的 4K FreeSync 显示器如此便宜有一个很好的原因——它只支持 40 赫兹到 60 赫兹的 VRR。

如果你打算买一台显示器，不想发现自己拿着的东西不能很好地实现承诺，那么你最安全的选择是寻找标有 FreeSync 2 HDR 或 G-Sync 兼容的显示器。否则，您应该在上面的链接中仔细检查显示器的范围。请记住，您希望显示器不仅能以最高刷新率支持 VRR，还能以最低刷新率支持它。

并且希望 Nvidia 和 AMD 的斗气继续让消费者受益。现在，它凸显了许多正在出售的 VRR 显示器的缺陷。更多的讨论可能会导致显示器制造商更容易注意到支持的可变刷新率范围。或者它可以让显示器更便宜。当被问及英伟达采用自适应同步时，苏姿丰暗示了一些事情。她不担心比赛。“我们认为这只是意味着这对游戏玩家来说更好。”
