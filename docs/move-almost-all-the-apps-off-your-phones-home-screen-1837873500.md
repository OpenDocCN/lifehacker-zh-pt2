# 将几乎所有的应用程序移出手机主屏幕

> 原文:[https://life hacker . com/move-almost-all-the-apps-off-your phones-home-screen-1837873500](https://lifehacker.com/move-almost-all-the-apps-off-your-phones-home-screen-1837873500)

我打手机打得太多了。我知道，这是一个独特的问题，但我已经受够了无意识地浏览社交媒体。每当我不得不等待一两分钟的时候，我就会拿出手机，出于习惯，我会点击-点击-点击，现在正在看每个人 Instagram 帖子的第二张照片，因为我已经滚动过一次了。

Watch

另一方面，我还没有*到*厌倦到准备删除我所有的社交媒体和其他浪费时间的应用程序；我只是需要一种方法来让自己在任何时候都怀疑自己是否真的想使用它们。

我在媒体 上看到了 [这篇文章，其中有一些减少手机使用的建议，其中一些我们之前已经写过，比如](https://medium.com/@neilpasricha/7-ways-to-reduce-cell-phone-addiction-with-step-by-step-photos-and-instructions-3e7a77cb22f5) [使用森林应用](https://lifehacker.com/change-your-screen-to-grayscale-to-combat-phone-addicti-1795821843) 将屏幕改为灰度 或 [。](https://lifehacker.com/the-best-app-to-help-you-stay-off-of-your-phone-1823474344)

但是有一个我以前从未听说过或考虑过的建议引起了我的注意:清空你的托盘。

托盘是主屏幕的最后一行，可能包含一些你最常用的应用程序。除掉他们，作者 [尼尔·帕斯查写道](https://medium.com/@neilpasricha/7-ways-to-reduce-cell-phone-addiction-with-step-by-step-photos-and-instructions-3e7a77cb22f5) :

> 在你手机屏幕的底部，有一个图标托盘，一直在那里。对大多数人来说，这可能是他们的电话、电子邮件、浏览器和信息。但是当这些图标在任何时候、任何屏幕上都可见时，你更倾向于点击它们。当然，电子邮件和短信会用小数字旗向你招手，吸引你的注意力。所以不要让他们得逞！

我考虑了一会儿，但我开始思考我托盘中的东西——我的电话、我的电子邮件、我的播客应用程序和我的天气应用程序——实际上是非常功能性的应用程序，我去那里是有目的的，是为了完成一项非常具体的任务。开始播客，打电话，检查今天下午是否会下雨。

无意识的浪费时间的东西是主屏幕上的其他东西。我去查看天气，在我知道之前，我再次滚动 insta gram*，看到甚至*更多*我朋友的孩子返校的照片。*

*所以我没有清空我的托盘，而是决定清空除了托盘之外的所有东西。我会把自己限制在四个最重要的应用上，我会让自己刷其他任何东西。*

*好吧，我试过了，但我不能把范围缩小到四个。我不想每次需要访问我的短信、相机、照片或 Slack 时都刷卡。但同样，这些应用程序是我为特定目的而使用的——拍照、阅读文本、回复工作信息。不是浪费时间。*

*刷卡很重要吗？不，当然不是。但这让我停下来思考了一会儿，“我刷卡是为了什么？”如果我确切知道我想访问什么，我就刷卡。如果我只是觉得无聊或者觉得我应该“打电话”，我会把这个愚蠢的东西放回去。(嘿，也许是时候开始为这些时刻随身携带一本书了。)*

*作为额外的奖励，我的主屏幕现在看起来不那么忙乱和混乱了。这是一个平静灵魂的主屏幕:*

*我甚至可能更进一步，把我所有的社交媒体应用组合在一起，把我所有的新闻应用组合在一起，等等。所有这些额外的滑动和点击会让使用我的手机更加麻烦。这正是我想要的。*
