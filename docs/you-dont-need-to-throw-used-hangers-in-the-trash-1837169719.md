# 你不需要把用过的衣架扔进垃圾桶

> 原文:[https://life hacker . com/you-dont-need-to-throw-used-hangers-in-trash-1837169719](https://lifehacker.com/you-dont-need-to-throw-used-hangers-in-the-trash-1837169719)

有了所有的塑料薄膜和有毒化学物质，干洗衣服可以成为一个的浪费过程。你很可能会在一天的使用后将剩余的铁丝衣架扔进垃圾桶。

Watch

一般来说，衣架由于其笨拙的形状，回收起来很棘手；它们很容易在回收设施中缠住机器，导致工人不得不用手移除每个障碍物而延误工作。出于这个原因，你可能*不会*将它们作为你路边计划的一部分扔进你的回收箱——但这并不意味着你现在就必须扔掉它们。根据具体的类型，有一些备用衣架问题的替代解决方案。

### 你也许可以回收钢丝衣架

首先，让我们简单了解一下每一种常见的衣架，包括塑料衣架、木质衣架和金属丝衣架。正如 [回收站](https://livegreen.recyclebank.com/column/because-you-asked/what-to-do-with-unwanted-coat-hangers) 所写的，大多数塑料衣架是由#6 塑料( [聚苯乙烯](https://earth911.com/earth-watch/360-recycling-plastic-6/) ，也可以在 CD 盒或杯子中找到)或#7 塑料(聚碳酸酯，也可以在水冷却器壶中找到)制成的。由于它们的塑料类型、形状和尺寸，你根本不可能回收它们。你也不太可能回收木制衣架，因为它们通常经过 [清漆](https://recyclenation.com/2014/06/recycle-hangers/) 或其他类型的处理，很难再利用。

钢丝衣架一般也不能回收，而且会对回收机械构成特别严重的挑战。也就是说，有*有*例外。在纽约市，作为该市路边项目的一部分，你可以把铁丝衣架丢进回收箱。你也可以把大量的钢丝衣架带到你附近的金属废料回收厂，因为它们是由贵重的钢材制成的。

### 重复使用旧衣架或把它们送到干洗店

对于任何衣架类型，首先考虑重复使用它们；如果你对一个简单的 DIY 项目感兴趣，你可以轻松地将放入毛巾或碗架。

然而，如果你想一劳永逸地摆脱它们，你应该在网上搜索当地路边回收项目的规则，看看他们是否接受金属丝、木头或金属衣架，以确保万无一失。(对于铁丝衣架，使用 [地球 911 的定位器](https://search.earth911.com/) 寻找可能接受它们的当地废金属回收商；使用术语“金属衣架”和您的邮政编码进行搜索。)

如果搜索结果是干的，那就退而求其次，问问你当地的干洗店是否接受用过的衣架(带一个 [服装袋](https://lifehacker.com/reduce-plastic-waste-by-bringing-a-garment-bag-to-the-d-1834424692) 以避免塑料薄膜的浪费，如果你还要送衣服的话 ) 。你也应该在网上搜索像收容所这样可能接受衣架捐赠的组织。根据 [让它远离垃圾填埋场](https://keepitoutofthelandfill.com/2019/03/24/about-those-target-hangers/) ，塔吉特也接受塑料和金属衣架进行再利用，但只有他们的*拥有*商店衣架。

展望未来，重新考虑购买那些轻薄的塑料衣架。木质衣架通常更耐用，可能比你通常选择的塑料衣架更耐用。拒绝商店里提供的衣架——你可以省去回收它们的麻烦。
