# 如何在 Windows 10 中解锁微软的免费图形计算器

> 原文：<https://lifehacker.com/how-to-unlock-microsofts-free-graphing-calculator-in-wi-1841064614>

虽然你的数学老师(或你孩子的数学老师)可能不会让他们带着笔记本电脑去上微积分课，但我仍然认为微软终于给 Windows 计算器增加了一点极客的味道，这太棒了。如果你认为“科学”模式很有趣，你还没有看到任何东西——即将推出的 Windows 10 是一个成熟的图形计算器。



我开玩笑，但只是一点点，因为它将最终有一个图形计算器烘焙到操作系统中。你再也不需要 [求助于第三方应用来模仿一个](https://lifehacker.com/how-to-emulate-a-graphing-calculator-on-your-phone-or-p-1835456320) 了，除非你渴望那种甜蜜的、德州仪器的感觉。

如果你不耐烦，你甚至可以今天就开始玩这个新的图形计算器。你必须牺牲一点潜在的稳定性来计算正弦曲线，因为你目前只能通过微软的 Windows 10 预览版来访问该计算器。

如果这还不能阻止你——你可以随时退出 Windows Insider 计划，一旦你玩够了，或者万一你遇到 Windows 10 预览版的问题——下面是如何开始的。

如果我是你，我会通过**控制面板>系统>系统保护**开始制作一个系统还原点*以防万一*。之后，点击**设置应用>更新&安全**，向下滚动到“Windows Insider 程序”，开始注册过程。确保你已经注册了快速通道。一旦你的电脑重新启动，再次访问相同的部分，坚持使用“Windows 更新”屏幕，而不是点击“Windows 内部程序”检查更新并安装提供的任何东西，这无疑将需要重新启动电脑。

在那之后，你应该有一个全新的 Windows 10 测试版可以玩——应该会在你的桌面右下角看到类似这样的东西，以确认:

你现在可以打开计算器，但是你可能看不到任何新的东西——至少，我第一次加载它的时候没有。要解决这个问题，请访问微软商店 的 [中的 Windows 计算器列表。点击“获取”重新安装，你应该会看到一个小通知，Windows 计算器已经更新。现在，当你打开桌面上的计算器时，你会通过左上角的“汉堡图标”菜单看到新奇的“图形”选项:](https://www.microsoft.com/en-us/p/windows-calculator/9wzdncrfhvn5)

你准备好了吗？点击它，然后输入你喜欢的方程式。按回车键，你会在左边看到结果。是的，你可以一次输入多个方程——颜色编码！—您可以使用追踪功能查看任何点的精确 x 和 y 坐标。啊，好像又回到了 AP 微积分。

如果您输入两个相互关联的方程，例如，在一个方程中使用“x ”,在另一个方程中定义“x ”,您会看到一个小调整条，显示不同的值如何影响主方程:

您还可以将鼠标悬停在您的方程上，如果支持的话，计算器甚至会为您提供快速分析，显示关键信息:域和范围、x 截距、y 截距、拐点等。

虽然计算器不会帮你做微积分作业，但它至少比你的手持图形计算器更容易操作——更漂亮，也更便宜。
