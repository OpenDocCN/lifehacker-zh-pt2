# 你 2020 年纳税申报表中的这一新扣除额奖励了慈善捐款

> 原文：<https://lifehacker.com/this-new-deduction-on-your-2020-tax-return-rewards-char-1842625403>

虽然冠状病毒救济法案中的许多项目都是暂时的，但这里有一项可以让你在未来几年的纳税申报中受益。



从 2020 纳税年度开始，您可以在所得税申报表上从调整后的总收入中扣除总额高达 300 美元的慈善捐款。这项扣除是为那些在准备报税时采用标准扣除而不是分项扣除的人准备的。

任何合格的慈善机构都有价值，从你最喜欢的动物收容所到你当地的食物银行。礼拜场所？检查。研究项目？没错。(实际上，你可以在国税局网站 上搜索申请免税的慈善机构的记录。)

它对任何接受标准减税的人开放，不管你的收入水平如何。2020 年，标准扣除额为个人 12，400 美元，夫妇共同申报 24，800 美元，户主 18，850 美元。

当 2017 年的减税和就业法案获得通过时，它提供的更高标准的扣除大大增加了人们的比率——[约 90%](https://twocents.lifehacker.com/everything-you-need-to-know-to-file-your-2019-taxes-1840839478)——他们转向这种方法，而不是逐项列出。但是非营利组织经常通过提醒支持者他们的 [现金捐赠](https://lifehacker.com/donate-money-not-food-to-your-local-food-bank-1830561819) 是可以减税的来募集捐款。

因此，最新的调整再次为纳税人创造了一个更容易实现的有形利益，让他们为慈善事业做出贡献，即使他们没有达到值得逐项扣除的水平。(当然，向你最喜欢的组织捐款所带来的温暖感觉永远不会消失。)

如果你觉得有必要向从事冠状病毒救援或研究工作的组织捐款， [慈善导航](https://www.charitynavigator.org/index.cfm?bay=content.view&cpid=7779#group-470) 已经收集了几个类别中排名靠前的慈善机构。
