# 如何写草书

> 原文：<https://lifehacker.com/how-to-write-in-cursive-1835450619>

我整天打字。我在手机上滑动和点击。我在纸上潦草地给自己写笔记。我对我的数字化生活很满意。但时不时地，我会收到一封手写的短信——通常是来自一位老太太——我发现自己希望自己能更经常地练习草书。那些老太太会*写*。

Watch

如果你想提高你的草书水平，或者从头开始学习，有两个主要的技巧需要提高:

1.  了解每个字母是如何形成的，以及它最终应该是什么样子
2.  在字母的大小、间距、倾斜和流动方面保持一致

首先是一点点学习，一点点实践；第二是所有的练习。所以让我们开始吧！

### 设置舒适

当你擅长草书时，你可以在任何地方写，但是如果你能坐在一张真正的桌子(或餐桌)上，坐在一把真正的椅子上，你将会有最轻松的时间来学习和练习。 [倾斜你的纸](https://www.teachhandwriting.co.uk/paper-position-for-comfortable-handwriting.html) 使底部朝向你写字的手。任何纸张都可以，但要确保表面光滑——在剪贴板上工作，或者如果你的桌子表面粗糙，可以叠几层纸。

当你的手处于舒适的书写位置时，你会想要一个移动顺畅的书写工具。关于什么是最好的笔 ，我们有很多观点，但有些人只有当他们垂直握着时才能写得好；其他人写斜体字更好。帮你自己一个忙，不要想太多:为这些早期的课程拿一支铅笔。

我要在这里说一件很有争议的事情:你的手的位置完全取决于你。我记得有人教过我一种 [动态三脚架握把](https://www.ot-mom-learning-activities.com/correct-pencil-grasp.html) ，这种握把非常适合铅笔和自来水笔，但其他握把也可能同样有效，这取决于你的手力、你喜欢的钢笔类型以及其他因素。你在网上学习草书，这意味着没人会给你打分。实验。

### 找到一个字母表模型

在你能写草书字母之前，你必须知道它们是什么样子的。搜索显示形状和笔画顺序的“字母参考线”。 [这是一个很好的基本的](https://www.k5learning.com/sites/all/files/Cursive-letter-guide.pdf) 显示整个字母表。你也可以下载并打印练习单 [，就像这些](https://www.k5learning.com/cursive-writing-worksheets) 一样，用浅色字母描摹而成。

在你选择引导符之前，先看看它们是如何构成字母的。传统上，大写的 Q 和 Z 很奇怪，但是你可以选择用你自己的方式来写。(还是那句话，没人给你打分，你是你自己的人)。字母表在其他方面也不同:例如，它们的大写字母 A 是否看起来像一个巨大的小写字母 A。选择(或者发明！)一个自己喜欢的风格，并坚持下去。

### 实践

你擅长这个的唯一方法就是练习。从直线向上的字母开始，如 l、b、f、h 和 u。然后转到曲线开头的字母，如 a、c 和 o。一遍又一遍地练习书写每个字母，直到你能始终如一地做出相同的形状。

有横线的纸有助于你保持信件各部分之间的比例。这种书法纸甚至有一个斜面，所以你可以确保你所有的垂直笔画(像 h 的高的部分)都整齐地平行。

当你沿着页面往下看的时候，保持纸张在一个舒适的位置。用你不写字的手向上移动纸张，而不是向下移动你写字的手。如果你有困难，记得用 [减慢](https://lifehacker.com/improve-your-handwriting-by-slowing-down-1822340446) 的速度。
