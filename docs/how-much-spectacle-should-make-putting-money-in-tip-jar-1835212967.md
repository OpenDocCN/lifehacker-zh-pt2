# 问那个刻薄的女服务员:把钱放在小费罐里该有多壮观？

> 原文：<https://thetakeout.com/how-much-spectacle-should-make-putting-money-in-tip-jar-1835212967>

> 嘿咸咸的，我有一个小费的问题。当我在柜台上有小费罐的餐厅点餐时，我会在服务员面对我的时候给小费，而不是在他们背对着我的时候(例如，当他们倒饮料或从箱子里拿东西的时候)。我认为给小费是一种表达感谢的方式，一种感激的姿态，如果我在别人看不见的时候给小费，并不能真正达到这个目的。从自我服务的角度来看，如果我是一个回头客，我想建立一种关系，我是一个感恩和礼貌的顾客，希望他们记住我。
> 
> 但是，我有点担心这看起来像是要求关注或感谢提示。有没有什么方法可以在小费罐里给小费，既表示我的感谢，又不会给人留下坚持要引起注意的印象？
> 
> 谢谢，来自芝加哥的本

回到现实世界。虽然在柜台服务场所给小费是礼貌的、正确的和恰当的——即使是在 [点](https://thetakeout.com/ask-the-salty-waitress-should-i-tip-on-takeout-orders-1822422611) ，甚至是在 [喝杯咖啡](https://thetakeout.com/should-i-tip-my-barista-for-a-cup-of-coffee-1827212450)——你真的觉得这会让你在未来获得更好的服务吗？我是说，这些地方大多都很热闹。你想让人记住你是个给小费大方的人，这是为了什么？一个漂亮的微笑？给你的鸡蛋沙拉三明治多拿几张餐巾纸？给小费是好事，但我只是说，我不知道从长远来看，它能给你带来多少实实在在的好处。

但是我们渴望自己的善良得到认可，对吗？仅仅做正确的事情是不够的，我们希望*被看到*在做正确的事情。你的信中表达了“我只是想表示感谢”，但你真正想要的是让服务员*看到你*表示感谢，对吗？这并不会让你成为坏人，而是让你成为人类。一个需要被认可的人。

事情是这样的，我击败了所有的评论者:这并不难。如果你愿意的话，当服务员面对你的时候，把小费放在罐子里，就这样吧。你不必费力地进行眼神交流(不必要的)，或者花言巧语地把钱放进去，并说一句过分真诚的谢谢(更没必要，简直是疯了)。你可以在里面塞一两块钱，如果你愿意，可以说“谢谢”，然后就放在那里。如果服务器错过了，真的，他们不会因为你缺乏感激之情而失眠。如果服务员没有说谢谢或感谢小费，也不要因此而责怪他们。
