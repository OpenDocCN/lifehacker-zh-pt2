# 在一个几乎空了的蜂蜜瓶子里做一份调味汁

> 原文：<https://lifehacker.com/make-a-vinaigrette-inside-an-almost-empty-honey-bottle-1834980660>

很难从瓶子里取出最后一点蜂蜜，尤其是如果瓶子的形状像熊(有很多裂缝)的话。幸运的是，挂在瓶子里的又甜又粘的东西是简单的调味汁的完美部分。

Watch

就像我们用酒精清洗几乎空了的果酱罐 的四周一样，我们可以用调味汁成分来溶解粘在塑料熊体内的蜂蜜。只需将四分之一杯油、两汤匙醋或柠檬汁、一汤匙第戎芥末酱，以及你想要的任何香草或调料倒入熊中。合上它，摇晃它直到蜂蜜溶解，然后用小熊的方便的喷口将调料撒在一堆绿色蔬菜、一些烤蔬菜或一个素食为主的饭碗上。
