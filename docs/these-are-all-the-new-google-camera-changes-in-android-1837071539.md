# 这些都是 Android 10 中谷歌摄像头的新变化

> 原文：<https://lifehacker.com/these-are-all-the-new-google-camera-changes-in-android-1837071539>



最终的 Android 10Beta—[Android 10Beta 6](https://android-developers.googleblog.com/2019/08/final-beta-update-official-android-q.html)—现在可供开发者和发烧友们玩。这一版本向我们展示了一幅近乎完整的图片，展示了 Android 10 的公开版本将会是什么样子，其中包括谷歌正在对其相机应用进行的一些调整。



Android 10 迄今为止最大的变化涉及新的手势控制和一般的 UI 调整，但许多其他核心 Android 应用程序和功能也在升级。以下是我们在谷歌相机应用中注意到的一些关键差异:

*   **夜间模式现在可以直接从主菜单**进入，取代了之前全景模式的位置。如果图像太暗，应用程序仍会显示烦人的药丸通知，建议你打开夜间模式，尽管现在首先使用它会容易得多。

*   **白平衡和色温控制**已经从应用中完全移除。如果你需要这些控制，是时候切换到 [一个更高性能的应用](https://lifehacker.com/the-best-camera-apps-1829781750) 了。
*   **前置闪光灯控制已经被简化为一个开/关开关，该功能已经被重命名为“照明”,因为它实际上并不使用闪光灯，而是照亮手机屏幕，充当替代闪光灯。**

可以肯定的是，这些变化将延续到完整版本，预计将在未来几周内发布。任何 Pixel 手机的所有者都可以通过 [选择加入 Android Beta 计划](https://www.google.com/android/beta) 来提前一瞥新的相机 UI 和所有其他 Android 10 功能。
