# 如何更快通过机场安检

> 原文：<https://lifehacker.com/how-to-get-through-airport-security-faster-1836385746>

当你赶不上 flight⁠—or 的时候，尽快通过机场安检，这是关键。除了参加美国运输安全管理局的预检、全球入境或清关，您还可以通过在正确的时间预订您的航班并组织您的随身行李来更快地通过机场安检线。

### 提前研究等待时间

在你去机场之前，试着研究一下安检等待时间，以避免排长队。像 [MyTSA](https://lifehacker.com/avoid-long-security-lines-with-the-tsas-app-1831848521) 这样的应用程序拥有大多数机场等待时间的众包报告；然而，因为它们是众包的，它们可能并不总是可靠的。尽管如此，它可能会帮助你更好地评估何时抵达机场——它还使用历史数据来估计等待时间，包括美国联邦航空局报告的机场延误信息。

一些机场，如 JFK 和拉瓜迪亚机场，也在他们的网站上有实时估计，尽管你不太可能从较小的机场找到这些统计数据。您还可以使用[【iFly.com】](https://www.ifly.com/)(使用航班量数据)和“”在线搜索您的机场的估计等待时间，以做出最佳猜测(尽管它不考虑实时报告)；只需搜索您的机场和“安全线 iFly”和相关的时间应该会出现在结果的第一页。

正如 [【今日美国】](https://traveltips.usatoday.com/busiest-time-air-travel-106177.html) 所写的，你可能想安排一个清晨的航班，那时机场可能不那么繁忙，也更容易避开长长的安检线。正如我们把 [写在](https://lifehacker.com/why-early-morning-flights-are-the-best-flights-1820947429) 之前，早间航班*也*不太可能被延误。或者你可以选择飞离比主要机场更安静的机场；在加州，你可能会选择像约翰韦恩机场或长滩这样的小机场，以避免在洛杉矶国际机场长时间的等待(和交通)。

如果你不熟悉机场地图，也可以查看一下地图，找到安全线本身，这对短暂停留特别有用。

一名飞行员在最近的 Reddit 帖子中写道:“我的基地很松散，整个机场被分成单独的航站楼，”[u/planeshangans](https://www.reddit.com/r/AskReddit/comments/7mfk4y/frequent_flyers_of_reddit_what_are_your_airport/)。"...令人惊讶的是，许多乘坐洛杉矶国际机场航班的乘客在不得不改变航站楼的时候，却留出 30 分钟来转机，这基本上保证了你会错过航班。只要提前做一点规划，就能防止这样的事情发生。”

在最近一次穿越东京成田机场的旅行中，我还发现许多不同的安检线遍布整个航站楼，并且能够找到一条特别短的安检线。

### 将电子产品和液体分开，并穿着合适的衣服

也许加快安检过程最简单的方法就是将你必须从随身行李中取出的电子产品和液体进行适当的分类。我总是把我的笔记本电脑放在一个可拆卸的套子里，所有液体(当然少于 3.5 盎司)都装在一个透明的塑料夸脱大小的袋子里，放在我随身行李的顶部，这样我就不用在安检时仔细检查我的物品了。正如 [SmarterTravel](https://www.smartertravel.com/airport-security-tips/) 所写的那样，将任何液体或凝胶深藏在你的随身行李中可能只会延迟安检过程。

此外，一定要拿走任何大于 3.4 盎司的液体，以及你可能在行李箱等随身行李中找到的可充电锂电池；你也可以把它们放在垃圾桶里，这样你就不会被安检耽搁了。

[分家伙](https://thepointsguy.com/2017/12/10-ways-through-airport-security-faster/) 还有一个不错的推荐:穿着得体。穿一件有口袋的夹克，里面可以放你的手机、护照、身份证或钱包，这样你就可以很快地把它们取出来，放在同一个箱子里。此外，当你在排队时，穿容易脱下的鞋子，摘掉任何皮带；换句话说，在到达安全线的尽头之前，尽你所能。

如果你愿意，你可以检查你的包，以避免安检的额外麻烦，但这只会意味着在你旅行的另一端又要在行李认领处等待一次。

就我个人而言，我还发现全身扫描仪的排队时间一般比步行金属探测器的排队时间长得多；如果你有选择的余地，我会选择有金属探测器的那条线。

### 向有美国运输安全管理局预检和下载手机护照的人预订旅行

虽然不是官方规定，但在很大程度上，你可以使用其他人的 TSA 预检状态(以及随之而来的特权)；为了实现这一点，预先检查的乘客必须在同一旅程下预订两个航班。当您办理登机手续时，两张机票都应显示预检状态。我们应该重申一下:这不是万无一失的，但过去它对我来说已经成功过几次了。一些信用卡，如大通蓝宝石储备，也可以免费提供 TSA 预检查注册作为额外津贴，所以一定要查找自己的卡的好处。

最后，如果你在海外旅行，想避开长长的海关队伍，别忘了下载 [手机护照](https://lifehacker.com/use-the-mobile-passport-app-to-breeze-through-customs-w-1833203914)；您可以通过应用程序提前提交您的信息，并在许多美国机场轻松通过海关。
