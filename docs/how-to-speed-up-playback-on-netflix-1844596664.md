# 如何在网飞上加快播放速度

> 原文：<https://lifehacker.com/how-to-speed-up-playback-on-netflix-1844596664>

一些人试图告诉其他人不要以 1.5 倍的速度 收听他们的播客，网飞最近宣布 [现在将允许人们控制其内容的播放速度，这一举动注定会引发一场自](https://gizmodo.com/netflix-adds-playback-speed-settings-so-now-i-can-burn-1844583307) [以来我们从未见过的互联网微争议。streamer](https://www.theverge.com/2015/2/17/8043077/stop-listening-to-podcasts-fast-speed) [去年开始测试的新功能](https://www.theverge.com/2019/10/29/20937546/netflix-playback-speed-testing-android-mobile-tv-movies-hollywood-filmmakers) 目前正在向其 Android 应用程序的用户推出，但很快也将适用于 iOS 应用程序和网络浏览器。
网飞声称已经限制了用户可以使用的控件，以安抚像贾德·阿帕图这样的好莱坞导演，他们在去年年底这部电影的策划阶段就表示强烈抗议；他宁愿你不要像一次往嘴里塞五块奥利奥一样快速浏览他倾注了心血的电影。就我个人而言，我尊重艺术意图，但只要观众没有被*强迫观看加速的电影，我不知道导演在这件事上应该有多少发言权。他们可能更希望我不要暂停他们的电影，走开去做 15 分钟的玉米片。如果我想让劳勃·狄·尼诺和阿尔·帕西诺在《爱尔兰人》[](https://www.netflix.com/title/80175798)*中更有活力一点，并在马丁·斯科塞斯(Martin Scorsese)的直接流媒体史诗电影中删减 70 分钟，那就由我决定。

### ***如何在 Android 上加快网飞播放速度***

*网飞让控制回放速度变得非常容易，提醒你每次开始一部新电影或电视节目时，你必须选择是手动慢放还是快放；与许多有声读物或 podcast 应用程序不同，默认情况下，您不能启用修改的回放速度。调整速度:

1。打开网飞应用程序，选择要观看的内容。*

*2.回放开始后，轻按屏幕调出回放控件，并在左下角查找看起来像速度计的图标。轻点它。

3。从弹出式菜单中，选取您偏好的回放速度。
4
。要进一步调整速度或恢复正常播放，请重复该过程。* 

### ***如何使用浏览器扩展调整播放速度***

*如果你通过网络访问网飞，不需要等待浏览器扩展来控制你的播放速度——你可以通过浏览器插件来完成。我个人喜欢视频速度控制器，Chrome 和 Firefox 上都有。下面是它的使用方法:*

*1.启动你选择的浏览器，导航到相应链接的扩展: [Chrome](https://chrome.google.com/webstore/detail/video-speed-controller/nffaoalbilbmmfgbnbgppjihopabppdk) 或 [Firefox](https://addons.mozilla.org/en-US/firefox/addon/videospeed/) 。在你的浏览器上安装它。*

*2.点击浏览器窗口中的扩展图标，会出现一个下拉菜单。选择“设置”*

*3.在这里，您可以调整哪些键盘输入将加快或减慢回放速度，或者预置您可以通过单次按键启用的首选速度。*

*4.通过相同的浏览器访问网飞，选择要观看的节目，并通过点击上一步中映射到各种功能的必要按键来控制播放速度。默认情况下，你可以通过按“r .”

将速度重置为正常播放。请注意，视频速度控制器实际上可以处理任何 HTML5 视频，而不仅仅是网飞——这意味着你也可以加速 YouTube、Hulu、HBO Max 等视频。它还允许你让视频播放*方式*比网飞官方实现的功能慢或者*方式*快；你可以根据你心中对艺术完整性的仇恨程度来做出自己的调整。*
