# 反对黑暗模式的理由

> 原文：<https://lifehacker.com/the-case-against-dark-mode-1834453136>

采用黑暗模式的应用程序的主要优势是，它们在深夜不会在我们眼中发出如此明亮的光。但也有一个缺点:黑暗模式实际上更难阅读，并可能导致眼睛疲劳。



在切达 T3 的这段视频中，美国验光协会的山姆·皮尔斯博士展示了反对黑暗模式的理由。

当白色文本在深色背景上时，它周围也可以有光环。事实上，黑色背景上的白色文字可能会*对比度太高*，导致文字更难阅读。

但是不管对比度如何，任何黑暗模式都会让你的瞳孔扩大。这正是我们的眼睛在昏暗的光线下所做的，这本身没有任何问题。但这让我们的眼睛更难聚焦(问任何摄影师景深的问题)。因此，我们眼睛中的睫状肌不得不更加努力地将眼睛的晶状体塑形。换句话说，你更有可能患上 [眼疲劳](https://universityhealthnews.com/daily/eyes-ears-nose-throat/eye-strain-and-ways-to-relieve-it/) 。

为了防止数字视疲劳，即在任何颜色的屏幕上阅读都会导致的头痛和眼睛酸痛，[AOA 建议](https://www.aoa.org/patients-and-public/caring-for-your-vision/protecting-your-vision/computer-vision-syndrome) 每隔 20 分钟看 20 英尺远的东西 20 秒钟。换句话说，有时候放下手机休息一下。

一直使用黑暗模式不会对你的眼睛造成永久性伤害，但是最好还是少用。发短信或偶尔使用手机的黑暗模式是没问题的，当你看视频或玩游戏时，黑暗模式不需要你阅读或观察细微的细节。但是，如果你要花时间在文字墙上滚动，你的眼睛可能会更好地适应浅色背景下的深色文字。
