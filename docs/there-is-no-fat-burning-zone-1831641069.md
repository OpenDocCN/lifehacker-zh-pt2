# 没有“脂肪燃烧区”

> 原文：<https://lifehacker.com/there-is-no-fat-burning-zone-1831641069>

所有的有氧健身器都在骗你。可能会有一些椭圆运动承诺让你保持在“脂肪燃烧区”，跑步机上的图表会告诉你，当你的心脏以一定的速率跳动时，这个区域存在。你做梦去吧。脂肪燃烧区是一个神话。



那些机器会告诉你这个区域存在于你最大心率的 50%到 75%的范围内(反正可能是)。在这个区域锻炼没有什么特别的，当然也不会比其他任何方式锻炼让你的身体燃烧更多的脂肪。

### 减脂不取决于你运动时的心率

如果你想减肥，首先要考虑的因素是 [你吃了多少](https://vitals.lifehacker.com/exercise-vs-diet-which-is-more-important-for-weight-l-1677532039) 。它应该少于你一天消耗的总热量。

但是运动不能燃烧更多的热量吗？当然可以。运动时间越长，消耗越多；你越努力锻炼，消耗越多。如何计划你的锻炼取决于你自己。一小时的步行和半小时的跑步消耗的卡路里差不多。

### 我们一直在燃烧脂肪

你的身体不会以备不时之需。它无时无刻不在燃烧脂肪，包括现在*你在马桶上读这篇文章的时候*而你根本不运动。

脂肪燃烧是有效的，但它提供缓慢而稳定的能量。如果你需要非常努力、非常快速地工作(想想:一只老虎冲进浴室，你必须跑)，你会使用其他能源，包括一些你的肌肉为此储存的糖。脂肪仍然在后台燃烧，但它不是主要的能量来源，比如说，60 秒的老虎尖叫奔跑。

如果你在健身房的有氧健身器上，你的身体正在使用这两种燃料的混合物:脂肪燃烧的基线量可能有所增加，但你的能量很大一部分来自燃烧其他燃料，包括储存的称为糖原的糖。(我在简化很多；如果你想出去玩，这里有一个稍微详细一点的版本 。)

总而言之，我们总是在燃烧脂肪，有时我们也会燃烧其他燃料。

### 那么为什么叫燃脂区呢？？

这个假定的区域存在于一个感觉轻松的水平上——快步走，慢跑，在椭圆车上或自行车上的配速，你觉得你可以无限期地保持下去。如果你刚开始锻炼，这是为了让你放心。你可以轻松愉快地进行锻炼，而且还能燃烧卡路里。

事实是:你越努力锻炼，你的身体就越需要其他燃料，比如糖原。所以*百分比*——明智地说，你在脂肪燃烧区燃烧了更高比例的脂肪。但那只是因为你燃烧的卡路里总量减少了。

真正的脂肪燃烧区，我们从脂肪中燃烧掉大部分热量的地方，是我们根本不工作的地方。所以，在沙发上闲逛。或者睡觉。

### 什么是适合锻炼的区域？

老实说。根据你的目标和喜好，找到快乐和工作之间的平衡点。把标有“脂肪燃烧”的区域想象成容易运动的区域，上面的区域代表中等强度和高强度运动。

在“脂肪燃烧区”锻炼，如果:

*   你想要轻度锻炼
*   你是锻炼的新手，艰苦锻炼的想法听起来很可怕
*   你在调整自己的节奏，这样你就可以进行长时间的锻炼

在较高的区域锻炼，如果:

*   你想努力工作
*   你想尽快结束你的锻炼
*   你想在同样的时间内燃烧更多的卡路里

如果你想改善你的整体健康状况，花时间在所有的区域:一些简单的有氧运动，一些中等强度的，一些最高强度的间歇运动。没有唯一的最佳锻炼方式。
