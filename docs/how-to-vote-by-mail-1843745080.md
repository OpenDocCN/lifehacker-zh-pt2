# 如何通过邮件投票

> 原文：<https://lifehacker.com/how-to-vote-by-mail-1843745080>

现在是 2020 年，所以自然而然地，邮件就从 [变成了政治化的](https://www.nbcnews.com/politics/2020-election/trump-rages-state-officials-quietly-press-forward-vote-mail-n1215781) 。具体来说， [总统坚称](https://twitter.com/realDonaldTrump/status/1265255835124539392?s=20) 推动在疫情期间通过邮件投票变得更容易是企图操纵选举，因为这样做容易出现欺诈——这种说法没有实际证据 。好消息是，各州基本上忽略了这一点:据研究选举技术的非营利组织开源选举技术研究所称，46 个州现在为每个合格的选民提供邮寄选票的选项。以下是你需要知道的关于邮件投票的事情，在这一年里，你既要面对一场有争议的总统选举，又要面对一场全球性的疫情。

### 基础知识

在进入细节之前，首先你要确定你已经登记投票了。确认这一点不会超过几分钟，而 [下面是怎么做的](https://lifehacker.com/how-to-check-if-youre-registered-to-vote-1787624689) 。如果你发现你*没有*注册投票， [以下是你需要做的](https://lifehacker.com/how-to-register-to-vote-1843476040) 来解决这个问题——包括在线、通过邮件和亲自注册的选项。

接下来， [查看你所在州的缺席投票法](https://lifehacker.com/check-on-your-states-absentee-voting-rules-right-now-1842378031) 。由于新的立法、法院案例和新冠肺炎，州一级的政策在不断发展。让自己跟上进度的最简单的方法是谷歌“[你的州]缺席投票。”例如， [这是我搜索纽约州时出现的](https://www.elections.ny.gov/votingabsentee.html) 。各州选举网站上的信息往往是最新的，并附有具体说明，告诉你需要做些什么来申请通过邮件投票。

通过邮件投票时要记住另外两件事， [瑞安·德托](https://twitter.com/RyanDeto/status/1263828354760933378) ， [匹兹堡城市报纸](https://www.pghcitypaper.com/) :

*   尽快收到您的选票，以确保/确认计票成功，并给县选举委员会充足的时间准备计票。
*   大选之夜的结果要有耐心。例如，在宾夕法尼亚州，对邮寄选票的统计可以在选举日的早上 7 点开始。虽然技术上有可能在当天晚上得到结果，但不要指望结果会提前出来。

### 邮寄投票:各州法规

鉴于最近 [州关于邮件投票的规定有所更新，](https://www.ncsl.org/research/elections-and-campaigns/absentee-and-early-voting.aspx) 你的第一站应该还是你所在州的选举委员会网站。但是为了快速概述，这里有 [开源选举技术研究所](https://trustthevote.org/wp-content/uploads/2020/05/27May20_BipartisanTruthAboutByMailVoting_v3.pdf) 的最新发现:

*   虽然从技术上来说，所有 50 个州和哥伦比亚特区都提供邮件投票的选项，但有四个州( [**【密苏里】**](https://www.kansascity.com/news/coronavirus/article243066146.html)**[**密西西比州**](https://www.wlbt.com/2020/05/18/mississippi-working-covid-contingency-plans-november-election/)**[**田纳西州**](https://www.wate.com/news/tennessee-vote-by-mail-case-ruling-expected-next-week/) **和** [**得克萨斯州**](https://www.npr.org/2020/05/29/864143739/texas-voters-are-caught-in-the-middle-of-a-battle-over-mail-in-voting) )要求选民有一个“借口”其他 46 个州向所有选民提供了这一选择。****
*   ****如果你在密苏里州、密西西比州、田纳西州或得克萨斯州居住或投票， [这张来自全国州议会会议](https://www.ncsl.org/research/elections-and-campaigns/vopp-table-2-excuses-to-vote-absentee.aspx) 的图表列出了请求缺席投票的所有可接受的“借口”。他们每个人都包括在选举日不在本县，以及生病或残疾。得克萨斯州最高法院最近的一项决定确定 [缺乏对新冠肺炎的豁免权是*而不是*要求通过邮件投票的有效借口](https://www.dallasnews.com/news/public-health/2020/05/27/texas-supreme-court-lack-of-immunity-to-covid-19-not-enough-to-vote-by-mail/) 。****
*   ****此前，其他 12 个州要求通过邮件投票的“借口”，但由于新冠肺炎的存在，已经放宽了限制。这些州是:**阿拉巴马州、阿肯色州、康涅狄格州、特拉华州、印第安纳州、肯塔基州、路易斯安那州、马萨诸塞州、新罕布什尔州、纽约州、南卡罗来纳州和西弗吉尼亚州**。****
*   ****目前，有五个州(**科罗拉多州、夏威夷州、俄勒冈州、华盛顿州和犹他州** ) [完全通过邮件进行选举](https://www.ncsl.org/research/elections-and-campaigns/all-mail-elections.aspx) 。****
