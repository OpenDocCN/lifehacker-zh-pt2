# 为了更有创意，设立一个“模拟桌”

> 原文：<https://lifehacker.com/to-be-more-creative-set-up-an-analog-desk-1835372964>

当我写作困难的时候，当我感觉被卡住或分心的时候，或者好像我是世界上最大的骗子的时候，我会强迫自己离开屏幕。我在家工作，所以我会拿一个迷你笔记本和一支笔，重新定位我的身体——我可能会在地板上伸展四肢，或者出去坐在我的摇椅上，或者绕着房子转圈。我以前写过运动 [如何打开我的大脑](https://lifehacker.com/im-michelle-woo-lifehacker-parenting-editor-and-this-1829059861) 。

对我来说，最终效仿作家奥斯汀·克莱恩(Austin Kleon)的做法，设立两张办公桌，一张是“模拟”的，一张是“数字”的，可能是有意义的。在他的《偷像艺术家[](https://www.amazon.com/Steal-Like-Artist-Things-Creative-ebook/dp/B0074QGGK6?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/to-be-more-creative-set-up-an-analog-desk-1835372964&asc_source=&tag=kinjalifehackerlink-20)*一书中，克莱恩解释道:*

> *我的办公室里有两张桌子——一张是“模拟”的，一张是“数字”的模拟桌除了马克笔、钢笔、铅笔、纸张、索引卡和报纸之外什么都没有。那张桌子上不允许有任何电子产品。这是我大部分作品诞生的地方，桌子上到处都是我创作过程中的物理痕迹、碎片和残留物。(与硬盘不同，纸张不会崩溃。)数字桌上有我的笔记本电脑、显示器、扫描仪和绘图板。这是我编辑和发布作品的地方。*

*他写道，“整天坐在电脑前会杀死你，杀死你的工作。”我相信。对我来说，让互联网触手可及不仅是一种干扰(我有一个 [tab 问题](https://adequateman.deadspin.com/close-your-tabs-1811909090) )，还是一种拐杖。当我第一次开始用纸和笔写作时，我会感到紧张不安，好像我需要再阅读 10 篇关于这个主题的文章，或者寻找关于 Thesaurus.com 的更好的同义词。我真正需要做的是在纸上写一些该死的字。远离数字教我如何更好地适应自己的大脑，让它做它已经知道如何做的事情。*

*如果你觉得自己被困在创造性工作中，拥有一个专用的模拟工作场所可以帮助你在自己的工作中再次找到乐趣，无论你是制作电影、书籍还是 PowerPoint 演示文稿。以下是一些入门建议:*

*   *以一种你不会看到你的数字工具的方式设置你的模拟桌。有证据表明，仅仅是智能手机 的 [存在，就会对你思考和解决问题的能力产生负面影响，即使是在关机的情况下。](https://hbr.org/2018/03/having-your-smartphone-nearby-takes-a-toll-on-your-thinking)*
*   *带一些需要你动手的模拟工具。当我写长篇故事时，我喜欢把主要观点写在索引卡上，然后在桌子上随意摆放，想象不同的叙事结构。*
*   *在你的模拟桌前，让你自己玩。这是你应该尝试疯狂想法的地方，即使它们很糟糕。当你在工作中感觉到一些完整性时，那就是你可以回到你的数字办公桌去执行和提炼你刚刚创造的东西的时候。如果你又卡住了，回到模拟。然后重复。你的身体会一整天都在运动——而想法会一直流动。*
