# 如何和清醒的人约会

> 原文：<https://lifehacker.com/how-to-date-someone-whos-sober-1833044831>

约会本来就很难。对于那些正在从药物滥用中恢复的人，或者干脆不喝酒的人来说，可能会更难。毕竟，在第一次约会时进行这样的谈话并不容易，而且当不可能选择在酒吧约会时，这可能会使最初的几次外出变得复杂。



与清醒的人约会，并寻找一些关于如何围绕酒精进行讨论的技巧？我和两位专家讨论了如何开始对话，以及为什么换位思考至关重要。

### 如果他们已经提到了，谈论他们的清醒是可以的

假设你正在约会，你的伴侣透露了他们不喝酒的趣闻。咨询服务机构“蔻驰康复”的主席辛迪·范伯格 说，询问某人关于他们的清醒程度是可以的——只要他们已经提出来了。

“如果有人把它放在那里，从一开始就让你知道他们正在康复，我不认为对某人说‘如果你愿意，我想听听你的故事’有什么错，”她说。“当然，你不必一直钻研下去，但问几个问题并不是不可以的。”

当有人告诉你他们清醒时，通过倾听来表达同情是最好的方法。但是，对于那些公开谈论自己戒酒的人，最好的简单回应方式是什么呢？根据范伯格的说法，你可以祝贺他们，并问后续的问题，“我可以喝酒吗？”

约会教练康奈尔·巴雷特 补充道, [经常会引发更大的谈话。“那时，我和前女友有了第一次约会，”她说。“我们在酒吧相遇。我在喝酒，她没有。我们谈论了大约 20 分钟，关于她在哪里以及如何到达那个点。这让我着迷，它将我们联系在一起。她愿意公开谈论这件事。”](https://datingtransformation.com/) 

如果你怀疑一次约会可能是清醒的，但你还没有讨论过，然而，这可能是一个最好推迟到另一天的谈话

“他们可能会对此感到尴尬，”范伯格说。“他们可能不想告诉别人，因为害怕被评判。我知道很多人可能会编造一些别的东西。一个标准的故事是，“我在服用抗生素，所以我现在不喝酒。””

相反，享受他们的陪伴，让他们自己决定这个话题。

### 你的约会对象对酒精的舒适度通常与他们的恢复程度一致

知道你的伴侣可能是清醒的，你如何计划约会？很明显，酒吧是不太可能的场所，但这并不意味着你不能喝酒。根据范伯格的说法，接触酒精确实取决于清醒者的恢复情况(对于不喝酒的人来说，酒吧并不总是问题)。

“在早期恢复阶段，我总是告诉人们，‘尽最大努力不要让自己处于危险之中，这意味着如果酒精是你的主要物质，试着从酒吧开始，试着把你的社交活动限制在不喝酒的人身上，’”她说。“但这在很大程度上取决于患者的康复情况。对于恢复得更久的人来说，如果他们的伴侣想喝一杯，问题会更小，但我(仍然)不想和一个正在约会的喝醉了的人在一起。”

当你和你清醒的伴侣约会时，你如何开始这种涉及喝酒的谈话？

“它在说，‘听着，我真的很想和你展开对话，我想喝杯酒，你觉得怎么样？“你同意吗，”并开始讨论，让每个人都真正感觉到他们的需求得到了照顾，”范伯格说。

巴雷特认为，仅仅因为一个人是清醒的，并不意味着一段长期的关系就行不通。“如果两个人需要一起喝酒来建立良好的关系——就像他们都需要保持清醒或者他们都需要两个人一起喝酒——那么你可能会看到一种相互依赖的情况。(但)我认为酒精与两个人长期在一起完全无关。这更多的是关于联系、同理心，以及让他们的人生大目标井然有序。”

因此，如果最初的几次约会进展顺利，不要把他们的清醒(以及你不能和他们一起喝酒)理解为事情可能会失败的迹象。最重要的是你的人脉和公开交流的能力。
