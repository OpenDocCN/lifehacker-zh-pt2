# 如果你在社交媒体上被拒绝该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-youre-rejected-on-social-media-1832560593>

在社交媒体上管理关系的细微差别会让任何人头疼。虽然我讨厌引用 [KonMari](https://lifehacker.com/konmari-your-twitter-feed-with-this-tool-1832407341) 这个短语，但你不应该忍受那些在你的数字聚会场所里不会引发欢乐的事物(或人)。我们已经给了你很多建议，告诉你如何 [从你的网络生活中清除那些愚蠢、烦人的人...但是如果你是那个让别人悲伤的人呢？](https://lifehacker.com/how-to-reject-someone-on-social-media-1832464867) 



我不是说你是巨魔。也许你有一些经过深思熟虑的观点，触动了你几个朋友的痛处。也许人们不喜欢你发布的 gif。你甚至可能正在经历分手，这往往会打破你以前与前任分享的朋友圈子。诸如此类的事情。

我们的生活充满了拒绝，而且并不总是向外。当网络世界中有人拒绝你时，这里有一些安全合理的方法可以让你处理。

### 与他们交谈，但不要太多

每次争论都有一个关键点——尤其是每次互联网上的争论——当对话开始消失的时候。如果你继续对话，你甚至可能会有一个小小的声音在你脑海中警告你，你可能会*推它——当有机会说最后一句话时，许多人似乎会忽略这个声音。*

处理别人给你的软拒绝的最简单的方法是——少回复你的信息，给你更短的回答，或者表现得像大多数人一样，当他们想继续他们的生活的时候——就是认识到这些时刻，并同意缓和关系的心照不宣的条款。当一个人想结束一次谈话时，注意到这一点并不困难，这并不意味着你需要对他们的 20 分钟贡献 80%。就此打住，或者自己包起来。不要添油加醋。

或者，更好的是，一旦你意识到你陷入了一场有争议的谈话，给自己一个回复的限制。同意在离开并且不再回来之前，你不会留下超过[x]条评论或回复。(如果可能的话，不要忘记取消关注或静音，这样当别人回复你之前说过的话时，你就不会再回来了。)

离开与格雷斯的垂死对话，在你们偶尔健康的争执后，你更有可能保留你的网友。

### 尊重他人对空间的需求

如果你对一个朋友或随机的网络陌生人做了什么，而他们把你静音了，甚至切断了他们与你的联系，你可能永远不会注意到。除非你经常和这个人交流，并对为什么你没有看到他们发帖或搭讪感到好奇，否则你可能有太多的其他联系人填充了你的信息，以至于失去一个“半经常”很容易被忽视。

你永远不会知道一个人是否在社交网络上屏蔽了你，除非你执行一些类似于夏洛克的仪式，检查每个喜欢你的内容或与你的内容互动的人，并与之前在某个时间与你互动的人进行交叉引用。那是许多工作。更重要的是，这让你看起来像个神经病。不要那样做。

很容易看出某人是否不再是你的朋友。大多数服务会让你知道你和一个不是你朋友的人是不是朋友——当你访问后者的个人资料时。有大量的第三方 [应用](http://www.twitonomy.com/) 和 [扩展](https://socialfixer.com/) 使得找到这一点更加容易。

无论你如何拼凑，失去网络联系后的第一步是花点时间想想这段友谊对你意味着什么。你想念那个人吗？*真的吗？*也许切断联系，即使损失是他们自己造成的，这正是你*需要*做的。你只是因为某种原因一直在拖延。

不要对地位的改变有下意识的反应。搁置几天，或者一周，重新评估你是否真的需要这个人回到你的生活中。我敢打赌，在你发现这个可怕的事实后的几天(或几个小时)里，它就不会那么痛了。在大多数情况下，你可能会继续你的生活，而不会为失去而悲伤——如果你可以这么说的话。

### 如果你不得不伸出手，小心行事

当有人表示不想再和你说话时——通常是通过切断你的友谊或直接阻止你——你应该冷静下来。不要成为这样的人，因为你试图强迫一段本该消失的友谊，而变得咄咄逼人，在一群共同的朋友中发现自己处于一个新的、升级的局面。

这并不是说，如果在我之前谈到的自我反省期间，你意识到自己是个混蛋，你就不应该试图去弥补。然而，道歉是有艺术的。保持谦逊、亲切，不要期望你说的话会改变你和某人的关系。事实上，不要假设对方会看到你的道歉，更不要说回应了。

根据你对其他人的厌恶程度，或者在解除好友关系或屏蔽之前的对话性质，发送道歉可能都不合适。当然，这是正确的做法，但是如果你刚刚和一个人进行了一场 30 条评论的战争，他们最不想看到的就是你的另一篇帖子，尽管看起来可能很善意。

只有你自己才能知道，在社交媒体上被人拒绝后，伸出援手的感觉是否正确。在大多数情况下，我宁愿选择“让它慢慢发酵”一个道歉或者一个月后，或者几个月后重燃爱火的尝试，给大家一个冷静的机会。你的话更有可能看起来真诚，而不是试图“结束”一场对话，重新获得一些权力的表象。你甚至可以更清楚地了解自己的某些方面，或者你做了什么而被拒绝，然后你可以用一封简短而简洁的便条来说明。

你也可以试着向一个共同的朋友表达你的遗憾或悲伤，他会在适当的时候代表你说出你的感受。这实现了两个目标:你得到了内心的满足感，因为你知道自己承认了自己做过的任何蠢事，你还可以让一个中立的人来决定何时(如果有的话)你的悔悟是否值得向被你冤枉的一方提及。(取决于你和你前朋友的关系，如果他们屏蔽了你，这也可能是和他们互动的唯一方式。)

我再强调一次，不要强求。没有人欠一个混蛋任何东西，也不是所有的友谊都意味着持久。从网上拒绝中你可以学到的最重要的事情是更好地理解你做了什么和你的社会行为，然后你可以根据需要修改，以避免再次走上这条路。
