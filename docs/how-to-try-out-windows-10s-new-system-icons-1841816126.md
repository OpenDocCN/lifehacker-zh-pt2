# 如何试用 Windows 10 的新系统图标

> 原文：<https://lifehacker.com/how-to-try-out-windows-10s-new-system-icons-1841816126>

Windows 10 将很快获得大量新的系统图标，给操作系统的外观带来急需的更新(先于 Windows 10X)。这些图标慢慢出现在 Windows 10 的测试版本中，如果你是 Windows 内部人员[，你可以提前查看它们。如果你不想处理测试版，你能做的最好的事情就是远远地呆呆地看着](https://lifehacker.com/get-the-newest-version-of-windows-forever-with-windows-1775733312) [整套新设计](https://medium.com/microsoft-design/iconic-icons-designing-the-world-of-windows-5e70e25e5416)——直到它们最终发布 Windows 10，这应该很快了。



我认为新设计看起来很棒；2015 年推出的单色轮廓已被色彩丰富、富有表现力的造型所取代。不幸的是，并不是每个人都对焕然一新的外观感到兴奋。但是最大的问题不是新图标，而是你的开始菜单的单色背景。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1230473719576846336&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1230473719576846336&autosize=1) 

“开始”菜单的背景色通常由 Windows 主题的强调色设置，并且它会在您调整设置或更新桌面图像时动态变化。不管你的设置如何，这些新图标都使用相同的蓝色背景(因为*当然*[Windows 更新会让事情变得更糟](https://lifehacker.com/how-to-fix-the-user-profile-bug-in-windows-10s-latest-p-1841699789) )。

像这样的特质在 Windows Insider 版本中很常见，我怀疑在新图标公开推出时它们会被修复。(微软计划在今年晚些时候将它们包含在 Windows 10X 的大更新中。)

如果你已经收到了更新的图标，它们让窗口看起来很糟糕，以至于你愿意放弃当前的桌面主题，请尝试更改你的背景图像和/或调整你的 Windows 主题的强调颜色，方法是右击你的桌面并选择**个性化>颜色**。这应该有助于新图标更好地适应——要么让你尽可能地匹配一成不变的蓝色，要么帮助你找到一个像样的 [补色](https://www.sessions.edu/color-calculator/) 。
