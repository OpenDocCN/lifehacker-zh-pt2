# 如何判断新的新冠肺炎故事是否值得兴奋

> 原文:[https://life hacker . com/how-to-tell-if-the-new-新冠肺炎-story-is-worth-get-1844052925](https://lifehacker.com/how-to-tell-if-that-new-covid-19-story-is-worth-getting-1844052925)

也许有一天会有疫苗或奇迹治愈新冠肺炎。但如果这种情况发生，它不会立即从你的电视或社交媒体上大肆宣扬的新闻故事中显而易见。看起来很重要的大型研究一直都有，一旦你了解了整个故事，大多数研究结果都令人失望。

如果你认为你在看一件大事，这里有几个问题要问:

### 实际的发现是什么？

标题会让事情听起来很有戏剧性，因为这就是标题的作用。但是请继续阅读这篇文章，并问这样一个问题:他们*实际上*发现了什么？

例如，我今天早上在 Twitter 上看到的第一个标题是“救命的冠状病毒药物已被发现。”( [此](https://www.bbc.co.uk/news/amp/health-53061281?__twitter_impression=true) 为文章；标题后来被改了。)这个标题暗示有一种特效药，我们现在已经找到了，冠状病毒死亡已经成为过去。这些都不是真的。

相反，该报道解释说，该药物将呼吸机患者的死亡率从 41%降至 28%，将未使用呼吸机但病情严重需要氧气的患者的死亡率从 25%降至 20%。

这个标题是准确的，因为生命似乎被拯救了，但它不是完全治愈，它不是对每个人都有效，它只在已经重病的人身上显示了这些结果。你第一次看到标题的时候是这样想的吗？

### 其他科学家怎么看？

如今科学发展很快，并不是每项研究都经得起推敲。一些新冠肺炎研究被证明是粗制滥造的，或者是基于最小的结果夸大了他们的结论，或者是基于那些 [看似粗略且无法验证](https://retractionwatch.com/2020/06/04/lancet-retracts-controversial-hydroxychloroquine-study/) 的数据。

我们并不倾向于马上发现研究中的问题。在问题暴露之前，科学家们通常需要至少几天的时间来讨论和检查一篇论文。(如果你听说过同行评议，即少数科学家能够在论文发表在学术期刊上之前对其进行审查的过程，这意味着在这场讨论中有了一个良好的开端。但这不能保证质量。)

我开始谨慎对待任何新的研究或公告。作为一名读者，你可以寻找的一件事是，一篇新闻报道的记者是否与读过这项研究、但自己没有参与研究的科学家交谈过。如果新闻报道中的每一句引语都来自进行这项研究的人，那么在你听到更多之前，请保持怀疑。

### **研究全文已经发布了吗？**

任何人都可以打电话给记者，说他们发现了一些看起来很酷的结果。但是，如果有科学支持，他们应该能够向每个人展示完整的研究。在新冠肺炎时代，我们已经看到许多结果是通过新闻发布会或新闻发布会宣布的，而科学家们并没有真正公布完整的方法或基础数据。

从伦理上讲，如果研究人员发现某种治疗方法特别危险，或者特别能救命，他们有义务迅速采取行动。这种想法是，尽早结束一项试验，或者公布重要的结果，可以拯救生命，不应该被推迟。

但是这种困境还有另一面:如果研究是错误的，或者结果是不完整的呢？请记住，你希望其他科学家能够看到这项研究，以帮助确定它是否真的有用，以及可能的局限性。在不公布数据的情况下宣布一个结果使得科学界的其他人几乎不可能完成他们的工作。

有时，甚至非医学研究也是在没有论文的情况下公布的，比如有报道称，讨论重新开放的 Twitter 账户中有一半实际上是“机器人”。结果由新闻稿 宣布 [，让其他研究人员想知道“机器人”是如何定义的，从而研究是否真的发现了研究人员所说的事情。](https://www.scs.cmu.edu/news/nearly-half-twitter-accounts-discussing-%E2%80%98reopening-america%E2%80%99-may-be-bots)

### 接下来是什么？

如果我们真的找到了治疗冠状病毒的特效药，这将不会是一个再也见不到的一次性新闻故事。这些结果将通过更多的试验得到证实，其他科学家和医生将对其工作原理进行评论，随着这一过程的继续，这个故事将一次又一次地出现在新闻中。

记住， [有几十种药物和疫苗正在研发中](https://vitals.lifehacker.com/track-all-the-covid-19-drug-and-vaccine-candidates-with-1843131548) ，我们要花很长时间才能确定它们中的任何一种是否成功。更有可能的是，我们会得到一些药物，每种药物都有一点点帮助，而不是找到一种神奇的治疗方法。所以不要只看一个故事；考虑一下这个新消息与已知消息的关系。等着看接下来会发生什么。
