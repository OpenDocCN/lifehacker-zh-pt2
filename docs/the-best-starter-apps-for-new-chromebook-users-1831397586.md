# Chromebook 新用户的最佳入门应用

> 原文：<https://lifehacker.com/the-best-starter-apps-for-new-chromebook-users-1831397586>

尽管 Chrome 操作系统是浏览器的雏形，但它是一个真正的桌面平台，拥有快照窗口、文件系统和键盘快捷键。作为 Chromebook 的新用户，你也可以利用绝大多数的 Chrome 扩展、Android 应用和 Linux 应用。(稍加欺骗，你甚至可以运行一些 32 位的 Windows 应用程序。)

Watch

然而，在你成为 Chromebook 超级用户之前，你需要开始熟悉一些基础知识，包括哪些应用和服务值得一开始就使用。Chrome OS 的功能可能像 Mac 或 PC，但它并不使用所有相同的软件。虽然它提供了与 Android 应用程序的兼容性，但它们并不都适合在 Chrome 硬件上正常工作。

我们四处搜寻，为 Chromebook 新用户寻找我们最喜欢的应用，下面是我们必备应用的快速列表:

### 对于文件管理: [**文件由谷歌**](https://play.google.com/store/apps/details?id=com.google.android.apps.nbu.files) (免费)

计划与附近的 Android 和 Chromebook 用户共享文件？或者您可能需要一些帮助来组织您的数据？

谷歌 的 [**文件是一款安卓应用，但在 Chrome OS 上也是一样的。它让你即使在离线时也可以共享文件(前提是你的共享对象安装了该应用)，随时记录你的存储空间，并在需要清理垃圾时提醒你。如果你有一个低存储容量的 Chromebook，或者如果你是一个迷妹的大收藏家，并且不太擅长在需要时清理空间，这款应用程序尤其有用。只需启动应用程序，选择干净标签，然后查看谷歌认为你可以不需要的东西。**](https://play.google.com/store/apps/details?id=com.google.android.apps.nbu.files) 

谷歌文件会定期提醒您将照片备份到谷歌照片，它甚至会为您获取重复的文件，这样您就不用担心自己去跟踪它们。您也可以使用谷歌的文件作为标准文件浏览器，访问任何您可能插入 Chromebook 的外部驱动器和设备。

### 消耗内容:[](https://play.google.com/store/apps/details?id=org.videolan.vlc)****(f**REE)**

**如果你打算在你的 Chromebook 上观看任何视频，请确保你已经安装了[](https://play.google.com/store/apps/details?id=org.videolan.vlc)**。这个开源媒体播放器支持各种文件类型，包括 MKV、MP4、AVI、MOV、Ogg 和 FLAC。这款强大的播放器还支持 URL 流和网络共享，以及 DVD ISOs。对于严肃的音乐爱好者来说，VLC 还提供了一个可调均衡器和滤波器，以及手势控制，这对触摸屏 Chromebooks 来说很方便。****

### ****为了获得更多的“桌面感觉”: [**添加我的文档**](https://chrome.google.com/webstore/detail/add-my-documents/plmanjiaoflhcilcfdnjeffklbgejmje) **(f** ree)****

****那些熟悉 Windows 操作方式的人可能会喜欢像 [添加我的文档](https://chrome.google.com/webstore/detail/add-my-documents/plmanjiaoflhcilcfdnjeffklbgejmje) 这样简单的应用程序，它会将一个标有“我的文档”的文件夹放入你的文件浏览器。除此之外，它不会做任何事情，但如果你仍然试图了解一个新的环境，尤其是如果你是 Windows 用户，它可以帮助你对 Chrome OS 感觉更好。当你整理文件和文件夹时,“我的文档”文件夹也是一个很好的垃圾场。****

****或者，如果你有 Dropbox，并希望在 Chrome OS 的原生文件浏览器中保持文件同步，你也可以为 Dropbox Chrome 扩展添加 [文件系统。虽然这对于从云中拖放单个文件到 Chromebook 很有效，但对于更重要的迁移项目，最好使用浏览器版本的 Dropbox，因为当处理多个文件时，扩展可能会很慢。](https://chrome.google.com/webstore/detail/file-system-for-dropbox/hlffpaajmfllggclnjppbblobdhokjhe?hl=en)****

### ****用于编辑照片:[**Adobe Photoshop Lightroom CC**](http:// https://play.google.com/store/apps/details?id=com.adobe.lrmobile)**(免费，带可选订阅功能)******

****无论你是用 Chromebook 编辑照片用于工作还是娱乐，[**Adobe Photoshop Lightroom CC**](http:// https://play.google.com/store/apps/details?id=com.adobe.lrmobile)是 Chrome OS 上处理这类工作的最佳安卓应用之一。Lightroom CC 支持所有 DNG raw 格式，允许您有选择地调整照片的任何部分，并提供许多 Lightroom 的基本功能，如斑点愈合。如果你已经拥有一个有效的 Creative Cloud 会员资格(从每月 10 美元开始)，这款应用效果最好，因为它让你可以访问高级功能，如前面提到的选择性调整功能，以及将原始照片文件上传到云的能力。**** 

****如果你不愿意为所有的花里胡哨买单，谷歌的安卓 [**Snapseed**](https://play.google.com/store/apps/details?id=com.niksoftware.snapseed) 应用也很方便。它提供了许多与此版本 Lightroom 相同的功能，包括原始图像编辑和选择调整。然而，与 Lightroom 相比，对照片进行增量更改通常会感觉有点乏味，Lightroom 更像是一个桌面应用程序，而不是移动应用程序。****

### ****对于您的手写笔记: [**鱿鱼**](https://play.google.com/store/apps/details?id=com.steadfastinnovation.android.projectpapyrus) **(f** ree，可选内购)****

****如果你是一个涂鸦者或者更喜欢长格式书写，并且你有一个带压力感应手写笔的 Chromebook， [**乌贼**](https://play.google.com/store/apps/details?id=com.steadfastinnovation.android.projectpapyrus) 是一个值得考虑的应用。Squid 可以让你做基于矢量的笔记，感觉就像你在纸上写一样真实。你也可以用这个应用来标记文件，比如 pdf。如果你想用它来做演示，Squid 可以与 Chromecast 和 HDMI 加密狗配合使用——它甚至可以让你使用手写笔作为指针。****

### ****用于编辑 pdf:[**卡米**](https://chrome.google.com/webstore/detail/kami-pdf-and-document-ann/iljojpiodmlhoehoecppliohmplbgeij?hl=en)**(**$ 2/月)****

****谷歌正在为 Chrome 操作系统开发自己的 原生 PDF 编辑解决方案，因为觉得在 Chromebook 上这样做对*来说很有挑战性。迄今为止，Kami 是我发现的唯一物有所值的服务，因为它非常适合在 Chrome 浏览器中管理 pdf 文件。该应用程序连接到 Google Drive 和 Google Classroom，可以让你随心所欲地给 pdf 文件添加注释。它的一些功能是免费的，或者你可以以每年 50 美元的价格订阅专业版，以启用存储数字签名等功能，这使得签署文档变得更容易。***** 

### ****对于屏蔽广告:[**【uBlock】来源**](https://chrome.google.com/webstore/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm) **(免费)******

****如果你想在阻止弹出广告和猖獗的互联网广告方面获得帮助，那么[**uBlock Origin**](https://chrome.google.com/webstore/detail/ublock-origin/cjpalhdlnbpafiamejdnhcphjbkeiagm)是一个免费的 Chrome 扩展，具有强大的控制功能。默认情况下让它开着，以获得无垃圾的浏览体验，或者如果你发现它干扰了网站的运行，你可以点击一下就关闭它。还有很多选项可以进一步定制，包括广告白名单和阻止网站个别元素的支持，如自动播放视频。这个扩展只在 Chrome 浏览器中有效，不会扩展到操作系统的其他部分。**** 

### ****编辑微软文档:[**office suite Pro**](https://play.google.com/store/apps/details?id=com.mobisystems.editor.office_registered)**(f**免费，但我们推荐几个内购)****

****微软在 Chrome 网络商店和 Play 商店中提供了许多官方应用，但无论是外观还是功能，都不完全像你在 Mac 或 PC 上用过的那些应用。如果你不介意支付 30 美元，[**mobi systems Office suite**](https://play.google.com/store/apps/details?id=com.mobisystems.editor.office_registered)将让你完全兼容 Office 的专有文件类型。它还兼容苹果 iWork 文件，以及 PDF，TXT，RTF，ZIP 和 OCR 文件。**** 

****这个特殊应用程序唯一的主要警告是弹出广告配套应用程序和字体包，即使尤尔已经支付了全套费用。只要你不介意不方便，OfficeSuite 是微软 Office 的一个有价值的替代品，特别是如果你打算在 Word 文档和 Excel 电子表格总是被传来传去的办公环境中使用 Chromebook。****

### ****用于管理密码:[**LastPass**](https://chrome.google.com/webstore/detail/lastpass-free-password-ma/hdokiejnpimakedhajhdlcegeplioahd)**(免费，可选 2 美元/月的高级计划)******

****如果你想让你的 Chromebook 感觉像是一台强大的电脑，请务必仔细锁定你的账户[**【LastPass】**](https://chrome.google.com/webstore/detail/lastpass-free-password-ma/hdokiejnpimakedhajhdlcegeplioahd)。就像它的移动和桌面版本一样，LastPass 会记住你所有的密码，并把它们藏在一个“保险库”里，只需一个主密码就可以进入。它还可以让你存储在网上使用的支付信息，以及 Chromebook 中可能需要保护以免被窥探的单个文件。****
