# 使用物理令牌来计算器械包

> 原文：<https://lifehacker.com/use-physical-tokens-to-count-sets-1845028198>

运动让你的大脑有点混乱。我知道我不是唯一一个在大约 30 分钟后不能准确计数的人...第二组。因此，如果你有多组练习，或者多圈循环，试着给自己一些物理标记来帮助计数。

Watch

例如，如果我在儿童秋千上做六组引体向上(我在一个秋千通常会去的地方安装了环)，每组之后我会在野餐桌上放一块石头。当我连续得到六块石头时，我就完成了。

我不能声称这个想法是我的功劳。我第一次了解它是在我有时去健身班的时候。一天，我们必须做十轮左右的三种不同的练习和何教练打开了房间角落里的一个小行李箱，露出一排排五颜六色的扑克筹码。我们每人拿了一叠，用它们来数子弹。

如果你的目标是做大量的练习——比如数百个俯卧撑——你可以更进一步。有一段时间，我每天做 300 个壶铃摆动。我一直记不清了，直到我记起我收到的一套宣传用的扑克筹码，并在下次我荡秋千的时候把它们带到了我的车库。十次摇摆是一个筹码，六个筹码让我在墙上得到一个散列标记，五个散列标记意味着我完蛋了。

即使你通常很擅长数你的盘数，也可以试一试。不知道自己是在第四盘还是第五盘是一种可怕的感觉，所以如果你完全明白我的意思，给自己拿一些扑克筹码或鹅卵石。
