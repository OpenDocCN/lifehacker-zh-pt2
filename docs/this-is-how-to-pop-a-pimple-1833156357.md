# 如何戳破痘痘

> 原文：<https://lifehacker.com/this-is-how-to-pop-a-pimple-1833156357>

拜托，我们都做过。

当你对着镜子看着自己下巴上感觉像连体双胞胎一样的青春痘时，很容易想掐掉它，直到它只剩下一个模糊的战争伤疤——但这从来都不是那么容易的。想想那些血！脓汁。最糟糕的是，另一个粉刺来拯救它，现在，你又回到了可怕的循环中。

Watch

有没有安全的方法可以戳破痘痘？虽然生活中没有任何保证，但有一些方法可以帮助你尽可能地处理好这个过程，尤其是在去除青春痘的时候。以下是你最终摆脱镜子里盯着你的粉刺所需的所有建议。

### 弄清楚它是否真的值得一试

首先，你应该知道，皮肤科医生很少推荐你去挤痘痘。

皮肤科医生拉内拉·赫希 博士在电话中说:“实际上，我们从来没有真的希望你长痘痘，从皮肤科医生的角度来看，这没什么好处。”感染的风险变得很大，这通常是不这样做的最大原因。"

她补充道，除此之外，你还会面临许多其他不良副作用的风险，比如加剧炎症或给你的皮肤留下永久性疤痕。

但是如果你一定要这么做(如果你无意中发现了这篇文章，很明显你很感兴趣)，你也应该知道不是所有的痤疮都是一样的。

一些粉刺长到了头上，形成了熟悉的白色或黄色肿块。这是一种更容易破裂的粉刺，因为它位于皮肤表面。然而，囊肿性痤疮是你永远也不想出现的那种，因为它在你皮肤的更深处。它通常更大，更痛，也更容易结疤——戳破它也没用。

“很多时候什么都不会出来，人们只会不停地挤压，试图得到一些东西。你真的不想那样做，”赫希说。

她补充说，如果你的皮肤上有开放性伤口或感染，最好不要戳破它，因为这通常表明你应该事先解决更大的健康问题(去看医生！).

### 准备青春痘

所以你正在做——祝你成功！你需要一些工具。首先，美国好施集团建议用外用酒精(就像急救箱里的酒精垫)对粉刺及其周围区域进行消毒。她说:“你真的会想要通过戳破粉刺来真正清除你将要引入的任何细菌。”

接下来，决定你的攻击计划。虽然你可以用你的指甲，但赫希建议用纱布盖住它们，这样你就不会传播细菌(根据经验，一些网站可能也会推荐棉签，但那些太乱了，你对它们的控制力度更小)。

“你也可以在像 [、丝芙兰](https://www.sephora.com/product/3-in-1-extractor-P231607) 或[【Ulta】](https://www.ulta.com/skin-care-tools-acne-removal?N=27hy)这样的地方买到提取工具，它们更好，因为你在试图提取的东西周围会受到更大的压力，”她说。

这些工具看起来更像针，比你的手指更精确一点(你也希望 T1 更精确)。

### 弹出它，并使用局部皮质类固醇

天啊，你做到了。

既然你已经用了一些外用酒精，准备好。根据 Hirsch 的说法，不管你使用什么工具，你都要从大约 90 度的角度按压粉刺(就好像你从侧面攻击粉刺一样)。也许它不会看见？)

继续推，直到你已经提取了所有的白头物质，一旦它消失了，再次用外用酒精消毒该区域——你不想冒险再长出一个粉刺。轻拍局部皮质类固醇或粉刺治疗剂，并继续保持该区域尽可能干净(这意味着在你完成后不要大惊小怪)。

如果付出足够的努力后还是没有结果，就停下来！你太过分了！你的粉刺可能是一个囊肿，除了使用治疗痤疮的药物或看医生之外，做任何事情都没有意义。戳破一个粉刺*应该很容易，如果需要更多的努力，你可能会造成更大的伤害——谁想这样呢？*

### 去看医生

所以你决定去看医生。根据美国好施集团的说法，这是更安全的选择。“如果它真的发炎、发热或困扰你，我们更希望你来看我们，这样我们可以在无菌条件下做，”她说。

在皮肤科医生的办公室，他们经常会给你注射皮质类固醇(这也有助于解决更深囊肿的问题)。这些只需要几秒钟的时间，在 24 小时内，粉刺就会明显地消退。

这需要注意的是，注射往往比通常的痤疮治疗昂贵得多，使用太多的皮质类固醇注射会导致“凹陷”或逐渐凹陷。但是如果你很绝望，而且明天是大婚日或其他重要事件，尝试一次也无妨(只是不要养成习惯)。
