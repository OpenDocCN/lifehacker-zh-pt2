# 你可以在线观看《罗杰斯先生的邻居》剧集

> 原文：<https://lifehacker.com/you-can-watch-mister-rogers-neighborhood-episodes-onlin-1833439180>

今天是弗雷德·罗杰斯的生日，这个教会我们如何从 [寻找帮手](https://www.youtube.com/watch?v=-LGHtc_D328) 的人，这个在 [成为国家药方](https://pediatrics.aappublications.org/content/142/3/e20182058) 之前就懂得玩耍的力量的人，这个能轻而易举地把他母亲 做的一件拉链毛衣 [摇得滚瓜烂熟的人。我想念他，也想念他的节目。](https://www.rewire.org/pbs/sweaters-sneakers-rogers-neighborhood/) 

Watch

虽然你可以在 PBS Kids、亚马逊 Prime Video 和 iTunes 上找到《罗杰斯先生的邻居》的剧集，但你现在也可以在 MisterRogers.org 弗雷德·罗杰斯制作公司的新网站上在线观看这些节目。新(旧)集在每个月的第一个和第三个星期一发布。

一次只有很少几集，但我相信罗杰斯先生无论如何也不会是狂热观看文化的粉丝。这个节目应该被慢慢地吸收，慢慢地品味。几周前，该网站报道了罗杰斯先生参观蜡笔工厂，这是我和女儿一起欣赏了多次的一集(“现在看蜡笔进入那些蜡笔收藏家手中”——多么令人兴奋)。本周在这个网站上，罗杰斯先生踩着高跷，谈论他花了多长时间来完成这项任务。

今天是罗杰斯先生的生日，可以带你的孩子去看看他的邻居。它仍然是 [最佳类型的屏幕时间](https://offspring.lifehacker.com/why-mister-rogers-neighborhood-is-the-best-type-of-scre-1823334997#_ga=2.122677064.2102045493.1552880043-1940201396.1552611778) 。
