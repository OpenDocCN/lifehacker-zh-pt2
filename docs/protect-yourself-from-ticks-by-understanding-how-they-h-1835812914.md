# 了解扁虱如何捕食，保护自己免受扁虱侵害

即使只是阅读有关蜱的文章，也会让我立即检查我的四肢，并开始感到发痒。(提前为这个故事会给你带来的影响道歉。)但与此同时，我对扁虱如此偏执，以至于我忍不住点击任何可能提高我避开那些小吸盘的机会的东西，哪怕只是一点点。

我们 [了解一下](https://vitals.lifehacker.com/youre-probably-going-to-get-a-tick-this-summer-good-lu-1793895235) (对吗？) : C 在户外呆了一段时间后(在树林、你的后院、公园等)，检查你的身体，使用 [适当的去除技术](https://lifehacker.com/properly-remove-a-tick-with-this-illustrated-guide-1590681271) 如果你找到一个，当你在大自然中时尽可能多地覆盖你的身体。但是，如果我们能进入一只扁虱的大脑，找出它究竟在人类身上寻找什么，难道不是更有帮助吗？是什么导致它在一种哺乳动物身上向右滑动而不在另一种上呢？因为蜱虫会主动寻找下一个目标，所以 [了解它们如何选择猎物](https://www.popsci.com/how-to-avoid-ticks/) 有助于避免被叮咬。

### 扁虱如何选择它们的目标

之前，我假设蜱虫会随机跳到它们接触到的第一个温暖的身体上 — ，不管是人、狗、鹿还是其他动物 — ，但事实远比这复杂。根据一篇 [科普文章](https://www.popsci.com/how-to-avoid-ticks/) :

> 没错，你经常会接触到蜱虫。但是当你碰巧经过的时候，它们并不只是挂在一片树叶上享受阳光。他们在埋伏。

那好吧。所以基本上，你在公园里天真的散步是在邀请壁虱进食。

但是他们怎么知道往哪里跳呢？原来，它们的前腿末端有微小的感觉装置，可以帮助它们察觉气味。可悲的是，再多的除臭剂或杀虫剂也救不了你，因为虱子可以根据你的呼吸——告诉你你在哪里[——具体来说，就是你呼气时释放的二氧化碳。扁虱也喜欢氨的味道，所以在树林里撒尿会给它们一个关于你的行踪的主要线索。](https://www.popsci.com/how-to-avoid-ticks/)

### 虱子是如何逃脱惩罚的

对我们来说不幸的是，很难马上判断一只扁虱是否落在了你身上并开始吸食你的血。据罗德岛大学资源中心的 [蜱虫称，我们注意不到被叮咬的是因为蜱虫通过它们的唾液分泌出一种叫做](https://tickencounter.org/faq/tick_bites#tickbites_question_01) [激肽酶](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3747359/) 的特殊止痛药。这也让他们可以在你不知情的情况下为你停留几天。

此外，这篇科普文章指出，并不是所有的壁虱都想在它们停在你身上的地方进食:一些壁虱喜欢在选择完美的晚餐地点之前探索你身体的不同区域。根据 [疾病控制中心的](https://www.cdc.gov/ticks/avoid/on_people.html) ，它们最喜欢的进食地点包括:

*   胳膊下面
*   耳朵里和周围
*   肚脐内侧
*   膝盖后部
*   头发内部和周围
*   两腿之间
*   腰部周围

所以，在户外活动后检查这些地方时，一定要格外仔细。

### 位置，位置，位置

如果你认为你可以搬到美国的另一个地方，不会被蜱虫叮咬，那就再想想吧。根据疾病控制中心 的说法，蜱虫不是 T4 本土物种的唯一一个州是阿拉斯加州。这意味着它们可以在其他任何一个州找到——包括你所在的州。CDC 指出，好消息是，并不是每一种蜱都会叮咬并传播疾病给人类。他们有一张方便的地图，上面显示了在美国的每一个地方都发现了哪些种类的蜱。

### 如何至少试着防止蜱虫叮咬

可悲的是，我可以根据经验说，无论你准备得多么充分，蜱虫仍然会咬你。来自 TickEncounter 资源中心的传统建议曾经是 使用避蚊胺驱虫剂，把你的裤子塞进袜子里，走在小径中央，回家后做一次彻底的蜱检查。但这并不总能解决问题。

现在，他们指导人们在户外活动的前一天用氯菊酯驱蚊剂处理他们的鞋子、袜子、短裤/裤子和衬衫。如果你花时间用氯菊酯处理你的衣服，它会持续一段时间:在家处理的衣服最多可以洗六次并保持有效，而商店买的处理过的衣服可以在 70 次后有效。

此外， [疾控中心建议](https://www.cdc.gov/ticks/avoid/on_people.html) 在户外两小时内洗个热水澡，因为这已被证明可以降低你患莱姆病的风险，并可能有效降低其他蜱传疾病的风险。此外，淋浴可以帮助洗掉没有完全附着在你身体上的虱子。

### 如果被蜱虫叮咬该怎么办

基本上，你 [想要确保](https://tickencounter.org/faq/tick_removal) 你以正确的方式去除了虱子，然后用温肥皂水清洗该区域。你最大的努力就是 [仔细阅读现在](https://lifehacker.com/properly-remove-a-tick-with-this-illustrated-guide-1590681271) 该怎么做，这样你就能为以后的做好准备。
