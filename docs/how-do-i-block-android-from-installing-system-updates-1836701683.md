# 我如何阻止 Android 安装系统更新？

> 原文：<https://lifehacker.com/how-do-i-block-android-from-installing-system-updates-1836701683>

我以前说过，而且我会继续说:让您的设备保持更新很重要。尽管我承认这条规则很少有例外。如果微软刚刚发布了一个巨大的 Windows 更新，你可能想给它几天时间*，以防* [发生灾难性的错误](https://lifehacker.com/back-up-windows-10-before-installing-the-october-update-1829556930) 。如果你有一部超旧的智能手机，并被要求升级到其操作系统的最新版本，考虑做一些研究，以确保它不会削弱你的设备的性能。



但是推迟所有的设备更新，因为你担心它们会对你的硬件造成什么影响？我不确定我是不是这个想法的粉丝，但这正是 Lifehacker 读者 **Valerie** 发给我的本周[**Tech 911**](https://lifehacker.com/c/tech-911)**建议专栏:**

> ***“我有一部三星 galaxy，买了不到 1 年。***
> 
> **我鄙视所有在手机上自动执行的更新。**
> 
> ***我的问题...1.有没有办法逆转已经执行的更新和 2。有没有办法阻止将来自动更新的执行？***

> **我打电话给威瑞森，他帮我接通了三星的电话。我与之交谈的三星代表相当傲慢，告诉我没有办法停止更新。他告诉我‘试试看，你会喜欢的。’**
> 
> **在先进的科技世界里....我在想，如果他们能把更新放在手机上，肯定有办法逆转或阻止他们。"**

**我知道你从哪里来。收到设备更新时，你可能会感到不舒服，感觉自己被抛到了一个陌生的领域，尤其是如果你的设备制造商对其操作系统进行了一些彻底的更改。每当苹果每年推出新版本的 iOS 我用 iPhone 作为我的主要随身携带设备——我总是觉得我必须努力理解所有新的东西:有趣的功能、微妙的变化、调整的手势等等。**

**这能阻止我*升级到新的 iOS 吗？从来没有。这只是我开始接受的一个过程，一个学习曲线，它的长度因人而异，取决于你使用设备的次数以及你理解新概念和工作流程的速度。感到不知所措，甚至完全敌视变化并不可耻。是人。***

**我还没有回答你的实际问题，因为我想强调一个事实:设备更新不仅仅是新功能。它们也是为了修复以前损坏的设置中的错误，或者更重要的是，修补漏洞以使您和您的数据更加安全。出于这个原因，最重要的是，我要提醒你继续接收你的智能手机的更新——不管花费多少。**

**对你来说，具体来说，考虑一下 [Android Q](https://lifehacker.com/the-best-new-features-in-android-q-1834620582) 接下来会发生什么。正如佛罗伦萨·伊恩在五月为我们写的:**

***“谷歌声称 Android Q 即将推出近 50 项与安全和隐私相关的功能。在 Android Q 对所有人开放之前，我们不会知道这些变化的程度。然而，谷歌暗示了一些你将能够在 Android 设置面板中访问的选项，包括一个新的前端和中心隐私部分，以及一系列用于控制哪些应用程序和服务可以访问你的数据的设置。还将有一个新的权限选项，让你选择如何以及何时与谷歌和其他第三方应用程序共享数据和其他元素。”***

**除此之外，Android Q 将使制造商更容易部署空中更新，以修复他们在 Android 操作系统中发现的 14 个不同(关键)模块中发现的任何问题。你甚至不需要重启你的设备就能收到它们；他们只会更新，仅此而已。然而，你需要使用 Android Q 来利用这个特性。谷歌不打算在老版本的操作系统中推广这一功能。**

### **阻止 Android 系统更新**

**希望我已经让你相信了更新你的 Android 设备的重要性。如果没有，我至少可以回答你的问题——尽管这让我很痛苦。**

**要阻止 Android 更新，我建议探索三个选项。首先，通过**设置>软件更新**调出你的软件更新屏幕，看看是否有 [任何一种关于软件更新自动下载的设置](http://downloadcenter.samsung.com/content/PM/201906/20190611065651730/EB/GEN_G970U_G973U_G975U_EN_FINAL_TEST2/troubleshooting_d1e17922.html#d1e17928) 你可以取消选中。(我面前没有 Galaxy 设备，否则我会给你截图。)**

**你可能需要多一点，以确保你的设备没有下载和安装更新，没有你的明确许可。还有另一个设置，你应该能够切换到禁用自动系统更新，但它有点埋没。再次拉起**设置****，向下滚动直到看到“**关于电话**部分。轻点那个。然后，点击“**软件信息**部分，向下滚动直到看到“**内部版本号**”开始点击*部分，直到你的设备显示你已经启用了“开发者模式”***** 

****然后，回到主设置屏幕，向下滚动。你应该会在“关于手机”旁边看到一个新的“开发者选项”部分点击它，然后寻找“**自动系统更新**选项。确保它没有启用。****

****那*应该*阻止你的手机接收自动更新。不过为了超级超级确定，你也可以购买并侧装 [**包禁用者 Pro**](https://www.packagedisabler.com/) app。然后，您将使用它来禁用设备的软件更新功能，这将使它永远保持在当前版本的操作系统上。****

****至于你的问题中关于回到旧版本的智能手机操作系统的部分，这当然是可能的，但这需要大量的努力和技术知识，我不建议普通用户这么做。这包括降级你的设备的引导加载程序，如果可能的话，并安装你的设备的操作系统的旧版本(如果你的制造商甚至提供这些/你可以找到一个)。正如 Android Central 的 Jerry Hildenbrand 所说:****

****有时，将软件更新到你的安卓系统上的方法很简单，也不难做到。但情况并非总是如此，你通常会看到一些人解释当情况变得更糟时，他们是如何毁掉自己的手机的。****

****我并不想劝你放弃，但你需要知道，把你的手机恢复到旧版本需要五分钟，而这需要几个小时的阅读时间。"****

****我的建议？掌握您现在拥有的设备，并在必要时禁用未来的设备更新。为了你的安全，我仍然不会走这条路，但我理解如果你想及时冻结你的设备的功能。****
