# 是的，家庭护理有可能获得报酬

> 原文：<https://lifehacker.com/yes-it-s-possible-to-get-paid-for-family-caregiving-1844883156>

在正常情况下，护理是一种负担——对许多家庭来说，疫情只会让事情变得更加困难。美国退休人员协会最近的一项研究发现，几乎五分之一的美国人无偿照顾另一个成年人。同样比例的人报告说，由于护理，他们的经济压力很大，原因很容易理解: [美国退休人员协会报道](https://www.aarp.org/caregiving/financial-legal/info-2019/out-of-pocket-costs.html#:~:text=The%20study%20found%20that%20family,more%20from%20the%20care%20recipient.) 你每年可能要花 7000 美元到 12700 美元在护理上。

### **医疗补助受益人计划**

[据美国退休人员协会报道](https://www.aarp.org/caregiving/financial-legal/info-2017/you-can-get-paid-as-a-family-caregiver.html) ，有可能通过 [医疗补助](https://longtermcare.acl.gov/medicare-medicaid-more/medicaid/medicaid-eligibility/#:~:text=It%20pays%20for%20health%20care,covers%20under%20its%20Medicaid%20program.)——一项为低收入美国人支付长期护理费用的政府医疗保健计划——获得支付。根据你居住的地方，你可以跳过中介机构的家庭护理，雇佣一名家庭成员。尽管每个州都有不同的福利和资格——比如配偶或法定监护人除外——所以你应该 [联系你所在的州](https://www.medicaid.gov/about-us/contact-us/contact-state-page.html) 了解规则。

### **退伍军人项目**

如果您的家庭成员是退伍军人，您可能有另外四个选项可供选择—[援助和出勤(A & A)福利](https://www.va.gov/pension/aid-attendance-housebound)[居家福利](https://www.benefits.va.gov/pension/aid_attendance_housebound.asp) 或 [家庭照顾者综合援助计划](https://www.caregiver.va.gov/support/support_benefits.asp) 。更多详情，请致电(855) 260-3274 联系 [退伍军人事务护理员支持热线](https://www.caregiver.va.gov/help_landing.asp) 。

### **准备为家人工作**

有偿照顾家庭成员似乎是一个理想的场景——但是在你开始之前有一些事情需要考虑。美国退休人员协会建议与老年人护理律师一起起草一份个人护理协议。这份合同应该概述你的职责、报酬和雇佣期限的细节。如果可能的话，试着让其他家庭成员也参与进来，比如你的兄弟姐妹。你还应该谈谈临时护理，它可以让你暂时摆脱护理的责任，并为你的病假制定一个计划。家庭护理可能比你想象的要花费更多的时间和精力，所以优先考虑你自己的自我护理是很重要的。正如美国退休人员协会的研究报告所述，23%的美国人表示，护理使他们的健康状况恶化。
