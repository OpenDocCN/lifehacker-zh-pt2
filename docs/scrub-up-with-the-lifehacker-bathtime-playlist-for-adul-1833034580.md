# 用 Lifehacker 成人洗澡时间播放列表擦洗

> 原文：<https://lifehacker.com/scrub-up-with-the-lifehacker-bathtime-playlist-for-adul-1833034580>

当我的宝贝女儿在浴缸里变得暴躁时，我把 TLC 的“无手术服”放在 Spotify 上，我知道我有一些特别的东西要与世界分享。有大量的儿童洗澡歌曲，但大多数都很乏味。所以我为成年人(以及不介意一些脏话和性暗示的任何年龄的人)做了一个洗澡时间播放列表。



这是一个不拘一格的组合，专注于巴斯特有的内容，以谦逊耗子乐团、奥特卡斯特、詹姆斯·泰勒、陶罐和 N.E.R.D .为特色。当然它包括几首经典的儿童歌曲，因为用漫画作家兰道尔·门罗的话说， [我们现在是成年人了，该轮到我们来决定什么是](https://xkcd.com/150/) 了。但没有大幅削减基兹伊茨第 26 卷。

Spotify 和 YouTube 播放列表在下面； [Apple Music 播放列表在这里](https://itunes.apple.com/us/playlist/lifehackers-bathtime-songs/pl.u-Ra15Sd9pGv) 。如果您将此播放列表导出到另一个流媒体服务，请在评论中分享它！
