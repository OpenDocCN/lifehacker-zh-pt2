# 抓取廉价的家庭健身器材现在

> 原文：<https://lifehacker.com/snag-cheap-home-gym-equipment-right-now-1844236576>

我想今年我对时机的直觉很好。三月份，我订购了一个杠铃和一些缓冲板，就在我的县宣布健身房将关闭的前一天。我在三月底告诉过你 [这整件事不会很快结束](https://vitals.lifehacker.com/this-isnt-going-to-end-soon-1842585548) 。两周前，就在病例开始真正激增之前，我指出 [会变得更糟](https://lifehacker.com/its-gonna-get-worse-1844002403) 。



作为一个彻头彻尾的傻瓜，我有一个重要的建议，我的跟踪记录表明你应该相信它:现在(取决于你的位置)是购买家庭健身器材的时候了。

还记得三月初，当成千上万的人意识到在可预见的未来他们将在家里锻炼时，所有的壶铃都消失了 吗？如果你所在地区的健身房已经开始营业，其中一些人可能已经准备好放弃他们的设备。而你，如果你聪明的话，会让 [在家工作](https://lifehacker.com/how-are-you-improvising-a-home-gym-1842867200) 的时间长一点。

时机提醒:如果你所在城市的健身房还没有开放，你的机会之窗即将到来。如果你所在地区的健身房已经开业，然后又关门了，那可能就太晚了。(以防万一，我还是会继续查看 Craigslist。)

一些供应商的新设备正在重新进货，但我一直喜欢购买二手设备。对于哑铃、杠铃和铁板，在以前，一个好的预算是每磅 50 美分，1 美元/磅仍然是一个合理的购买价格。(专业或者高质量的物品，像保险杠板或者竞赛式壶铃，总是比较多。)在检疫期间，如果你能买到的话，我所在地区的重量通常是 2-3 美元/磅；截至今天，他们回到了 1 美元。

如果你所在地区的 COVID 病例上升，健身房再次关闭，你会很高兴在价格再次飙升之前买到一些东西。如果我错了，健身房还开着，你的购物仍然可以让你在家锻炼，如果你生活中有人生病或需要隔离，这很有帮助。我们还有很多疫情要看。
