# 往烤架上扔些垃圾

> 原文：<https://lifehacker.com/throw-some-spam-on-the-grill-1843887252>

烧烤是一种相当简单的烹饪方法——你把食物放在热煤或真火上，然后把它留在那里烹饪——然而人们仍然会想方设法把它变得非常复杂(尽管，不可否认，很好吃)。但是你知道什么不复杂还很好吃吗？垃圾邮件。

Watch

这些天我都很轻松— [甚至比平时更轻松](https://skillet.lifehacker.com/how-to-feed-yourself-when-you-have-more-important-shit-1843859563) — ，垃圾邮件很容易就溜进了 这种生活方式。它不需要调味而且用高热做得真的很好。即使是烧焦的或烧焦的垃圾邮件也很美味，所以没必要小心翼翼或仔细观察。你可以烤厚木板做三明治或者把它切成方块，然后和蔬菜或水果一起插在串肉扦上。我最喜欢的搭配是烤牛排和烤菠萝，但是墨西哥胡椒和洋葱也是烤牛排的绝佳搭档。仔细想想，真的没有一种蔬菜不会因为一些咸味加工肉的存在而得到改善。

烧烤垃圾食品的热量会使一些脂肪散发出来，产生金黄色的松脆点。这种情况发生得相当快(尤其是如果你习惯于熏肉时间线)，所以最好将你打算用烤肉叉串起来的任何植物部分切成薄片。(你也可以使用腌制或卤制的蔬菜，它们不需要太多的热量来软化。)将串肉扦扔在热烤架上，每隔几分钟旋转串肉扦，直到你的肉变成漂亮的金黄色，还有一些烧焦的斑点。在享受之前让它冷却一分钟，然后想想为什么你不在每个夏天都这样做。
