# 父母需要了解什么样的蒸发性疾病

> 原文：<https://lifehacker.com/what-parents-need-to-know-about-the-vaping-illness-1838007921>

你可能一直怀疑抽烟对任何人来说都不是最健康的习惯，更不用说青少年了。现在，我们似乎正在收集这方面的证据，因为全国各地有数百人患有与 vaping 相关的严重肺部疾病，其中许多人是 20 岁出头的青少年或年轻人。



vaping——也称为 JUULing ，以流行品牌命名——是通过电子烟设备吸入和呼出气溶胶或“蒸汽”的行为。这种气溶胶通常含有尼古丁或 THC(大麻)，以及新英格兰医学杂志所描述的“超细颗粒、重金属、挥发性有机化合物和其他有害成分”

Vaping 对青少年来说特别有吸引力，因为这种液体有各种各样的甜味或水果味:芒果味、薄荷巧克力味、蛋奶沙司味、肉桂卷味、草莓和奶油味等等。根据 [《新英格兰医学杂志》](https://www.nejm.org/doi/full/10.1056/NEJMoa1911614) 的报道，自 2014 年以来，电子烟一直是美国年轻人最常用的烟草产品。去年，高中生的使用率远远高于成年人(20.8%比 3.2%)。

费城问询者报 报道称，这可能解释了为什么许多被诊断患有与 vaping 相关的严重肺部疾病的患者都很年轻，而且在其他方面都很健康，没有可检测到的细菌或病毒感染:

> 医生一个接一个地排除了感染、病毒、自身免疫性疾病和所有可能导致这些症状的原因。他们只剩下一个令人信服的罪魁祸首:蒸发。

> “我见过的病人都很健康，年轻。所有人都是男性，除此之外没有真正的医学问题，”托马斯·杰斐逊大学医院肺部和重症监护医学助理教授高塔姆·乔治说。"我们还不完全明白是什么导致了这一系列症状."

### 我们知道的(和不知道的)

这种疾病的早期症状包括咳嗽、气短、胸痛、恶心、呕吐、腹痛和发烧。它可能看起来像一种快速发展的肺炎，可能需要人工呼吸。

疾控中心和食品药品管理局仍在调查 T2 T3，并分析患者提供的样本，以确定所有接受治疗的患者之间的特殊联系。虽然他们指出，由于几乎所有的大麻样本都含有这种化学物质，因此正在对大麻样本中的维生素 E 乙酸酯进行专门调查，但并不是所有的患者都报告使用过大麻产品。

而且因为 vaping 仍然是一个相对较新的趋势，所以 vaping 对健康的任何其他短期或长期影响在很大程度上仍然是未知的。

### 你能做什么

像任何其他危险的行为一样，和你的孩子建立一个关于蒸发的持续对话是很重要的。即使你不怀疑他们尝试过，他们也很可能有朋友使用 vape。美国肺脏协会创建了一个名为 [、Vape Talk](https://www.lung.org/stop-smoking/vape-talk/) 的网站，为父母提供各种有用的资源，包括:

*   关于 [不同类型电子烟的信息](https://www.lung.org/stop-smoking/smoking-facts/e-cigarettes-parents.html)
*   [电子烟的基础知识及其健康后果](https://www.lung.org/stop-smoking/smoking-facts/e-cigarettes-and-lung-health.html)
*   [可下载的对话指南](https://www.lung.org/assets/documents/stop-smoking/vape-conversation-guide.pdf)

新泽西卫生部门也为青少年和家长准备了一份有用的指南，其中有关于戒烟的信息和电话号码 [，青少年可以发短信求助戒烟](https://www.thetruth.com/articles/hot-topic/quit-vaping) 。

最后，利用新闻。分享几天前我在当地报纸上报道的故事——比如这个——关于一个年轻人因与呼吸相关的疾病住院的故事。青少年经常认为他们是不可战胜的，提醒他们他们不是也无妨。
