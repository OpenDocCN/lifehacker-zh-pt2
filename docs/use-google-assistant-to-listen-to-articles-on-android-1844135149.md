# 用谷歌助手听安卓上的文章

> 原文：<https://lifehacker.com/use-google-assistant-to-listen-to-articles-on-android-1844135149>

这些天来，我的大部分阅读都转向了有声读物——它更容易适应我的时间表，有时也更有趣。我经常发现自己希望能像听有声读物一样听文章。事实证明，通过谷歌助手，Android 可以做这件事。以下是如何做到这一点:

1.  使用 Chrome 或另一个基于 Chrome 的浏览应用程序打开你想大声朗读的文章或网站。
2.  说“嘿，谷歌，读一下。”当然，这看起来有点简洁，但谷歌助手很乐意帮忙:它会迅速在一个新窗口中打开文章并大声朗读出来。(谷歌也会回应类似措辞的命令，但“读出来”效率更高)。
3.  如果谷歌助手说它无法读取屏幕，因为“屏幕上下文”被关闭了，点击助手屏幕打开一个对话框，然后确认启用“屏幕上下文”，而不必浏览手机的设置。

这个功能很容易打开和使用，但我们也将检查控件。“Read it”命令会在单独的 Read It 浏览器中打开网页，其底部有一个回放控制栏，包括播放/暂停、倒回和快进按钮。还有一个“速度”按钮，你可以点击它将阅读速度从 0.5 倍提高到 3 倍。

当 Google Assistant 阅读页面时，页面会自动滚动，但您也可以使用倒回/快进按钮或轻按/拖移媒体时间线来手动移动。

点击右上角的三点菜单图标，调整更多设置。最值得的两个是“朗读语音”和“翻译”选项。“朗读语音”允许你更改谷歌助手用于朗读的语音，而“翻译”则使用谷歌翻译从 42 种支持的语言中自动(并令人信服地)进行翻译。

正如我们上面提到的，Read It 只在谷歌 Chrome 等 Chromium 浏览器上运行，并且只适用于某些网站。如果有一个付费墙或者你必须登录才能访问该网站，谷歌助理可能无法大声阅读它，因为 read it 浏览器窗口不能传输你的 cookies 或登录信息。Android Police 还发现， [广告拦截应用导致 Read It 失灵](https://www.androidpolice.com/2020/06/21/how-to-make-google-read-any-text-out-loud-to-you-on-your-android-phone/)——可能是因为 Read It 浏览器内置了广告。

抛开这些特质不谈,《阅读》效果很好。我肯定会用它来了解我每天的新闻。
