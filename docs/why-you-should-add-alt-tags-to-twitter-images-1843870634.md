# 为什么应该在 Twitter 图片中添加替代文本

> 原文：<https://lifehacker.com/why-you-should-add-alt-tags-to-twitter-images-1843870634>

我是一个懒惰的推特用户，这是真的。我不怎么使用这项服务，当我使用的时候，我会偶尔上传一张图片和一些精辟的评论，然后就到此为止。但我认为这是错误的。我和你一样，也应该花点时间在我的推文图片上添加替代文字。



为什么要采取这一额外的步骤呢？因为你是个好人，这就是原因。这很容易做到，几乎不需要任何额外的时间，并且可以帮助那些有视觉障碍的人更好地理解你的推文的上下文。

Alt text， [为那些使用屏幕阅读设备或在电脑或智能手机上使用任何类型的 voiceover 功能的人提供图像的音频描述](https://webaim.org/techniques/alttext/) 。您可以通过图像替换文本来设置描述，这有助于为人们正在阅读/收听的内容添加有用的上下文。如果没有 alt 文本，屏幕阅读程序或 voiceover 功能将直接跳过图像。

对于一条推文，尤其是一条基本上是某种但不是真正的标题或对没有视觉背景就没有意义的照片的尖刻反应的推文，替代文本是必不可少的。你可以很容易地在网络上或使用 Twitter 的 iOS 和 Android 应用程序添加它们，所以你真的没有借口不使用它们。

在**网站**上，上传一张图片到你的 tweet，点击右上角的编辑按钮。当你这样做的时候，你会看到一个新的列，上面写着“ALT”点击它，你会首先看到这个屏幕:

单击“确定”，然后会提示您添加替代文本。为您的推文保存 Twitter snark 替代文本应该是对你的图像的清晰描述。保持简短，但不要太短。

这个过程在 iOS 系统和安卓系统上同样简单。以前者为例，开始撰写新的推文，并在帖子中添加图片。点击不容错过的“+ALT”按钮，输入你的描述。

替代文本不会计入你的推文字符限制——但是，同样，你不需要在这里写文章。简单的一句话，清楚地描述照片中的内容就足够了。请记住，有人——或某些人——终究要读这些。
