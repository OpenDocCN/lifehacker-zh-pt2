# 用蛋糕制作机一次做一堆沙拉三明治

> 原文：<https://lifehacker.com/make-a-bunch-of-falafel-at-once-using-a-cake-pop-maker-1835377135>

蛋糕爆米花机是你用来制作蛋糕爆米花的最糟糕的工具。如你所见，蛋糕爆米花是*而不是*一种已经成型并烘焙成球状的蛋糕；蛋糕已经被弄碎，混合了糖霜，然后*用手将*卷成一个球，粘在一根棍子上。但是仅仅因为看似单一用途的设备在制作蛋糕爆米花时是垃圾，这并不意味着你不能将它用于其他更美味的应用。



正如 Reddit 用户[zainedcrasvindow](https://www.reddit.com/user/zainedcrassvindow/)所发现的，这款设备非常擅长制作 [一大堆沙拉三明治](https://www.reddit.com/r/MealPrepSunday/comments/bxy0ez/was_a_about_to_donate_my_sisters_cake_pop_maker/?utm_source=share&utm_medium=ios_app) 而不需要使用一大堆油，因为大多数制作者都是不粘的。(外观不会像油炸法拉 fel 一样*脆，但你不得不佩服这种效率。)正如一些评论者指出的，没有必要停留在沙拉三明治上。这个制作器可以用来制作 [三文鱼肉丸](https://skillet.lifehacker.com/how-to-turn-a-can-of-salmon-into-fancy-canapes-1826967564) 、takoyaki(关于如何制作的视频 [这里](https://www.youtube.com/watch?v=TyadJhV8vHU) )，以及——尽管我那生于密西西比州的心很痛苦地说出来——“更健康”的嘘嘘小狗。别用它来做爆米花蛋糕。它能做出非常难吃的蛋糕。*
