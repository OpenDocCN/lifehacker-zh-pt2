# 这些伪造的冠状病毒追踪器可能会感染你的电脑

> 原文：<https://lifehacker.com/these-bogus-coronavirus-trackers-could-infect-your-comp-1842293731>

新冠肺炎的信息铺天盖地而来，我们都想保持更新，但你应该避免下载冠状病毒跟踪器仪表板。根据 Reason Security 的调查，这些 [冠状病毒地图中有一些包含了被称为 AZORult 的严重恶意软件](https://blog.reasonsecurity.com/2020/03/09/covid-19-info-stealer-the-map-of-threats-threat-analysis-report/) 。利用 AZORult，黑客可以窃取你的浏览器历史和社交媒体登录信息；访问您的银行账户或加密货币金库，甚至获得对受感染设备的完全远程访问权限。

Reason Labs 特别推出了“Corona Virus Map”Windows 软件，它甚至使用了与我们上周报道的约翰霍普金斯大学的追踪器相同的界面，看起来更合法。如果您安装了“电晕病毒地图”在任何时候，卸载它，并立即执行病毒扫描。如果你最近安装了任何其他冠状病毒相关的应用程序或软件，也值得运行一次扫描；如果不是投机取巧，黑客什么都不是，很可能“冠状病毒地图”只是 [众多冠状病毒相关骗局](https://lifehacker.com/how-to-avoid-coronavirus-scams-1842268661) 中的一个。如果你坚持要不断更新病毒的传播，我建议 [用约翰霍普金斯大学的官方追踪器](https://gisanddata.maps.arcgis.com/apps/opsdashboard/index.html#/bda7594740fd40299423467b48e9ecf6) 和*做书签，只使用那个*，并坚持使用 [可信的在线来源](https://vitals.lifehacker.com/follow-these-twitter-accounts-for-accurate-information-1842215514) 。

也就是说，这里有另一个建议:也许根本不用追踪器。需要说明的是，我并不是在鼓吹无知或者说这没什么好担心的，但是压力和焦虑会严重破坏我们的身心健康，削弱免疫系统。 [不断分析新冠肺炎的蔓延是一个巨大的压力源](https://lifehacker.com/how-to-deal-with-coronavirus-anxiety-1842119738) 。试着注销并寻找宁静的时刻——如果你以后再回来查看，突发新闻和最新统计数据仍然会在那里。见鬼，你甚至可以利用这段时间通过玩这些电子游戏来帮助找到疫苗，而不是整天盯着最新的统计数据。
