# 为你的孩子建立一个虚拟的展示和讲述

> 原文:[https://life hacker . com/set-up-a-virtual-show-and-tell-for-your-kids-1842447805](https://lifehacker.com/set-up-a-virtual-show-and-tell-for-your-kids-1842447805)

一直被关在家里对任何人来说都不容易。父母 [不得不继续工作](https://offspring.lifehacker.com/how-to-get-work-done-when-your-kids-are-home-1839415435) 同时兼任家庭学校的老师。孩子们没有机会和他们的朋友在一起，在这一点上可能会很烦躁。这就是为什么你应该使用 FaceTime、Skype、Google Hangouts 或 Zoom 为他们建立一个虚拟的展示和讲述。

孩子们喜欢展示和讲述:他们可以告诉别人一件心爱的物品，并有一个平台和观众谈论他们此刻所感兴趣的事情。(上周他们痴迷的商店 slinky 可能不再降价了。)他们可以展示他们在家时制作的艺术品，唱首歌，或者只是向你展示他们的房子是什么样子，以及他们在那个特殊的日子里的感受。

作为父母，你可能已经习惯了你孩子的唠叨，考虑到在可预见的未来你们将会在一起度过多少时间，T1 甚至想要和 T2 分开。这就是虚拟展示和讲述派上用场的地方:随机的父母、阿姨和叔叔会狼吞虎咽地吃那些东西。只要问他们什么时候有空进行 FaceTime 通话，打电话，把电话递给孩子，花一分钟时间深呼吸。你也可以把它作为他们在家学习时的“课程”之一。他们可以每天轮流给不同的朋友或家人打电话，给每个人一些变化。

或者，如果你的孩子的祖父母希望尽可能多地在 FaceTime 上看到他们，但 [的孩子没有什么可说的](https://offspring.lifehacker.com/get-your-kid-talking-about-their-day-with-this-question-1841043577) ，虚拟展示可以成为救星。而不是他们只是茫然地盯着屏幕上的自己或者用“我不知道”来回答每一个问题，这就给了你一个现成的谈话话题。

如果你想变得有趣，你甚至可以为每个展示和讲述环节选择主题。例如，有一天你可以让他们选择红色的东西，另一天让他们选择他们认为“最喜欢”的东西(比如最喜欢的书、最喜欢的帽子、最喜欢的袜子等等)。当然，如果电话另一端的人想展示和讲述他们自己的物品，他们也可以参与进来(尽管孩子们可能不太喜欢这一部分，会抗议/逃跑)。

在过去的几周里，我一直在和我朋友的孩子(两岁和五岁)进行 FaceTime 展示和交流。在那段时间里，我看到了国际象棋奖杯、霹雳舞动作(特别是咖啡研磨机)、艺术品、跳房子游戏、他们的脚趾、一些非常有趣的睡衣和他们小脸上的许多特写。这些电话不一定需要很长时间——只是从我们新的、隔离惯例中快速休息一下。
