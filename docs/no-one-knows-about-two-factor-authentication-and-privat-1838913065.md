# 没有人知道双因素身份认证，这将他们的安全置于风险之中

> 原文：<https://lifehacker.com/no-one-knows-about-two-factor-authentication-and-privat-1838913065>

听起来很奇怪，不是吗？尽管我们试图宣传这种安全措施的重要性——这是识别试图登录你账户的人实际上是你的第二种手段——但来自 [皮尤研究中心](https://www.pewinternet.org/2019/10/09/americans-and-digital-knowledge/) 的一项最新调查表明，超过一半的受访者无法说出一个双重身份认证的例子。

就账户安全而言，这并不好。

这里有一个快速复习。**双因素认证**非常重要，因为它为登录尝试增加了第二个挑战。例如，如果一个人有你的用户名和密码，但不能访问你手机上的 authenticator 应用程序，他们将无法实际登录你的帐户。您可能会收到一个通知，通知您有人尝试登录，这样您就可以将密码更改为更安全的密码，但您的帐户将保持安全。

正如 [Authy](https://authy.com/what-is-2fa/) 所描述的，您的第二个认证因素可能包括:

*   *你知道的事情:这可能是个人识别码(PIN)、密码、“秘密问题”的答案或特定的击键模式*
*   你拥有的东西:通常，用户会拥有一些东西，比如信用卡、智能手机或小的硬件令牌
*   *你是什么样的人:这个类别更高级一点，可能包括指纹、虹膜扫描或声纹的生物特征模式*

*“使用 2FA，这些因素中的任何一个潜在的危害都无法解锁帐户。因此，即使你的密码被盗或手机丢失，其他人拥有你的第二要素信息的可能性极小。从另一个角度来看，如果消费者正确使用 2FA，网站和 app 可以对用户的身份更加自信，并解锁帐户。”*

但这还不是全部。皮尤研究中心问了一些其他与网络安全和隐私相关的问题，以下是受访者的回答:

*   只有 25%的人知道“隐私浏览模式”只能防止使用同一台电脑的人看到他们的活动——你的 ISP 或雇主仍然可以看到你在做什么，网站仍然可以记录你的信息。
*   只有 48%的人知道什么是“隐私政策”。
*   只有 30%的人知道“https://”URL 表示你正在加密连接一个网站。
*   大约三分之一的人不确定他们可能会在哪里遭遇网络钓鱼诈骗，例如，通过电子邮件、短信或“我的网站”。

如果你想知道你的网络安全知识与其他人相比如何，你应该试着参加皮尤研究中心的测试[](https://www.pewresearch.org/quiz/digital-knowledge-quiz/)。美国国家标准与技术研究所也在 2017 年发布了一个值得探索的小测验——它有一只猫！—如果你想在证明你的网络安全能力的同时获得一些数字分数，棕榈滩州立学院有一个更具竞争力的测验。
