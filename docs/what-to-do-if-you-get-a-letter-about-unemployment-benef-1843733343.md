# 如果你收到一封关于你从未申请过的失业救济金的信，该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-you-get-a-letter-about-unemployment-benef-1843733343>

这种情况正在马萨诸塞州、华盛顿州发生，也许就在你家附近:骗子正试图利用你的个人信息，趁虚而入，利用疫情失业援助计划。



这个骗局最近在波士顿曝光，那里的居民收到了关于他们最近失业申请的电子邮件和信件。问题？那些居民没有申请福利。

在马萨诸塞州，拥有有效失业索赔的人可能需要提供额外的信息来证明他们的身份——这可能会延迟他们的付款。如果你听说过人们在试图获得合法的 T2 失业救济时遇到的麻烦，你就能理解这些诈骗企图对相关各方来说是多么令人沮丧。

如果你收到一份关于你没有申请的失业救济金的通知，尽快通知你所在州的失业办公室是很重要的。马萨诸塞州 [建立了一个在线表格](https://www.mass.gov/info-details/report-unemployment-benefits-fraud) 来帮助人们在线报告失业救济欺诈，但如果你的州没有出现大量的虚假索赔，它可能不会为这一特定骗局的受害者建立一个特殊的系统。你可能需要忍受电话或实时聊天的长时间等待。

但是不要停留在报告你所在州的诈骗企图:如果有人有足够多的关于你的信息以你的名义申请失业，他们就有足够多的信息对你以后的财务生活造成大量的损害。

根据马萨诸塞州劳工和劳动力发展办公室 的说法，用于提交这些欺诈性申请的个人数据来自以前的国家数据泄露。据《波士顿环球报》报道，一份的欺诈申请掌握了准确的信息，包括受害者的全名、地址、手机号码、社会保险号、出生日期，甚至他们的大概工资。

你会想向 [州检察长](https://www.usa.gov/state-attorney-general) 和投诉。在你这样做之后，这两个机构中的任何一个都不太可能会跟进，但没关系—公众能够提供的信息越多，联邦贸易委员会和国家机构就越容易追踪欺诈者并对他们采取法律行动。

但是等等，你还没说完。您还需要检查您的信用报告，以确定您的财务记录是否已经超出了这一虚假的福利申请。您可以访问 [您的免费信用报告](https://lifehacker.com/how-to-get-your-free-credit-report-once-a-week-1842984075) 每周一次，每个信用局在疫情期间。当你在那里时， [冻结你的信用点数](https://twocents.lifehacker.com/what-to-know-about-freezing-and-unfreezing-your-credit-1829474191) 以确保没有人能以你的名义开立新的信用帐户。这是免费的，但你必须单独向 Experian、Equifax 和 TransUnion 申请一个。
