# 使用此 Windows 应用程序安排您的电脑在特定时间关机

> 原文:[https://life hacker . com/schedule-your-PC-to-shut-down-at-a-specific-time with-t-1840929603](https://lifehacker.com/schedule-your-pc-to-shut-down-at-a-specific-time-with-t-1840929603)

不时地关闭或重启你的电脑是一件好事。它甚至可以通过清除临时文件和释放内存来解决小的性能问题，这是一种经典的故障排除技术，但我们许多人都忽略了这一简单的技术维护行为。将台式机或笔记本电脑置于睡眠(或休眠)状态要比完全关机容易得多。

如果 [关闭 Windows PC](https://lifehacker.com/why-does-my-laptop-shut-down-instead-of-going-to-sleep-1840067515) 已经成为你待办事项清单上又一个被忽视的琐事，那么看看[Adios](https://haendel.jimdofree.com/shutdowntimer/)——开发者 Bernd Haendel 博士开发的一款简单的 Windows 应用。它可以让你设置和忘记你的电脑自动关机定时器，这可以让你的生活更轻松。

### 如何使用 Adios

1.  使用此链接 下载并安装 [Adios。](https://haendel.jimdofree.com/shutdowntimer/)
2.  打开程序(它会在安装后自动运行)。
3.  点击**【关机】**标签。
4.  选择要启用的操作，如关机、重启、注销、睡眠、休眠等。—从左侧的选项中，然后使用箭头按钮设置您的计时器。您可以设置计时器，使其在特定时间后、一天中的指定时间或电脑闲置一定时间后触发。
5.  您的电脑将在指定时间自动关机或进入选定模式。

除了**【关机】**标签下的选项，Adios 还有一个**【特价】**标签，包括一些其他有趣的功能:

*   设定每天使用电脑的时间限制。
*   为应用程序设置“等待”例外，以防止在某些应用程序打开时触发自动定时器，从而防止丢失您的工作。
*   设置您的计时器，等待特定的下载链接完成。

您还可以在**“Specials”**选项卡中找到电脑启动文件夹的快捷方式，这是您的电脑存储每次启动时打开的应用程序的位置。最后，还有一个**“设置”**选项卡，用于定制应用程序的语言和各种显示设置。

当你的系统的关机和重启按钮可以从 Windows 开始菜单轻松访问时，你可能会质疑你是否真的需要这样的应用程序，但请尝试一下——你可能会发现当你不用思考它时，定期关闭你的电脑会容易得多。
