# 使用这个搜索引擎快速找到难以找到的古董衣服

> 原文：<https://lifehacker.com/quickly-locate-hard-to-find-items-with-this-search-engi-1840298629>

如果你正在购买新衣服，那么找到你想要的东西是一件相对容易的事情。当你寻找复古服装时；然而，找到一件完美的夹克、包包或牛仔裤变得有点复杂。 [Gem 是一个可以帮忙的网站和 app](https://gem.app/) 。

Watch

该网站类似于一个搜索引擎，但专门搜索古董。本周早些时候，29 号炼油厂写了一篇关于 T3 的文章。

这个想法是，你可以在 Gem 上一次搜索成千上万家商店，而不是在一堆不同的网站上搜索该商品，或者在谷歌上搜索并得到许多不相关的结果。

该网站抓取在线商店，建立一个销售商品的数据库。当你搜索时，它会将你的搜索词与数据库进行比较，并显示你找到了什么。

搜索结果以 Pinterest 页面的形式出现，有问题商品的图片以及价格图标，让你知道卖家的要价。

当你点击一件商品时，该网站会将你带到一个新的页面，在那里你可以看到该商品的任何其他照片，以及一个链接到该网站，在那里你可以继续潜在的购买它。

因为它爬了这么多网站，你有更好的机会找到那个复古蔻驰包或完美的运动夹克。很有可能你甚至会在一家你甚至不知道存在的零售商那里找到它。

下次你在寻找一件古董的时候，绝对值得去兜一圈，看看你能找到什么。
