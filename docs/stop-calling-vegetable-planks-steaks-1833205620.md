# 不要再把蔬菜板叫做“牛排”

> 原文：<https://lifehacker.com/stop-calling-vegetable-planks-steaks-1833205620>

我没有最喜欢的食物。事实上，我甚至不确定如何对这种东西进行分类，因为我想吃的东西会根据我所处的情况而有很大的不同。但是有一种食物，从我长牙齿的那一刻起直到现在，我一直都充满热情地吃着，那就是牛排。

Watch

现在，当你读到“牛排”这个词时，你马上就知道我在说什么了。你知道我指的是一块厚厚的红色肉，切成板状。你也可能认为肉来自于奶牛。也许，如果你是一个喜欢打猎的人，你会想我是不是指鹿肉。你不会认为我指的是花椰菜吧。

词语是有意义的，不管一种蔬菜有多丰盛、多饱满、多鲜味，它都不是牛排。这样想:你见过菜单上的“鸡排”吗？不，因为即使是多汁的、经过专业腌制和烹饪的、表皮呈漂亮棕色的鸡胸肉也不同于一块红色的肉。(这里有金枪鱼排，不过奇怪的是，我对它很满意。)但是如果我给你看一块鸡肉和一块花椰菜，然后问你“哪一块更像一块牛？”你可能会说“鸡”，因为在细胞水平上这是真的。然而，在许多食谱和菜单上，与牛排最不相似的东西却被称为牛排。

这太疯狂了。

这并不是说蔬菜不应该成为主食。我喜欢植物性食物，我不是那种要求知道牛肉位置的人。最无聊的家伙是那种在一篇关于美味素食的文章上评论“如果你加点熏肉也许会更好”的家伙。(而且永远是纨绔子弟。)使蔬菜好吃的东西和使肉好吃的东西是不同的，而且不得不以一种“肉食者”可以理解的方式来框定一种蔬菜是对蔬菜和喜欢蔬菜的人的侮辱。

一块烤茄子尝起来永远不会像动物的肉。这很好。事实上，如果你不喜欢吃动物肉，这很好。做一道美味蔬菜的关键在于理解并尊重蔬菜，顺其自然，而不是强迫它去扮演它本不该扮演的角色。再多的腌制、烧烤或品牌重塑都无法让肉食者相信一片厚厚的卷心菜可以取代他们的肋眼牛排。事实上，使用“牛排”这个词，你是在邀请一个比较，这个比较会让食谱失败。(我想，素食者不希望他们的蔬菜让他们想起肉，就像肉食者永远不会接受用菜花板代替真正的牛排一样。)当然，我犯了这种试图让人们相信蔬菜食谱值得一试的错误，但我正在努力变得更好。我确定我用过“你不会错过肉的”这句话。(我敢肯定没有人被愚弄，事实上，有可能肉被错过了)。

如果你真的想吃更多的蔬菜，并鼓励人们吃更多的蔬菜，你必须学会欣赏它们的本来面目，而不是拿它们和其他东西做比较。
