# 如果你戴隐形眼镜，现在考虑换成眼镜

美国眼科学会(AAO)表示，如果你戴隐形眼镜，你可能会想 [考虑换成眼镜](https://www.aao.org/eye-health/tips-prevention/coronavirus-covid19-eye-infection-pinkeye) 。

AAO 发言人、MetroHealth 医学中心的眼科教授托马斯·斯坦内曼(Thomas Steinemann)医学博士说:“我们告诉人们现在戴隐形眼镜要放松。

### 隐形眼镜使用不当会导致眼睛问题

AAO 的建议在一定程度上是基于现实世界的使用，因为许多隐形眼镜的佩戴者在佩戴时可能会松懈，这可能会导致眼睛问题。

“很容易忘记，我们都很忙，我们都想节省时间，我们偷工减料，有时这些决定对你的眼睛健康不利，”Steinemann 说，并补充说，在他的实践中，他一直看到由于不正确的隐形眼镜使用而出现问题的患者。

我们都犯有戴着隐形眼镜睡觉、没有定期给镜片消毒、在放入和取出隐形眼镜之前忘记洗手的错误。考虑到我们承受的压力有多大，正确使用隐形眼镜是又一件需要记住的事情，又一件容易出错的事情，并且伴随着发展成医疗问题的风险。

眼镜提供了一个更简单的选择，减轻了已经不堪重负的医疗系统的负担。

### 新冠肺炎患者的眼部问题

还有证据表明，很小比例的新冠肺炎患者，在 1-3%之间，会患上结膜炎，通常称为红眼病，并且在这些患者的眼睛分泌物中可以发现这种病毒。正如 Steinemann 指出的，这仍然是一个正在进行的关于的研究领域，他说“我们对新冠肺炎眼中的行为知之甚少。"

科学家们仍然不知道新冠肺炎相关结膜炎何时与其他症状相关，病毒何时开始出现在眼睛分泌物中，或者这在传播方面会造成多大的风险，但这仍然是一个需要考虑的因素。虽然隐形眼镜本身不会给你新冠肺炎，但它们可以在传播中发挥作用。

“隐形眼镜可以作为携带病毒的载体，”Steinemann 说。放置隐形眼镜和取出隐形眼镜都需要用手指接触眼睛，有传播病毒的风险。

### 如果你戴隐形眼镜，请正确使用

如果你选择戴隐形眼镜，更重要的是 [遵循](https://www.aoa.org/contact-lens-wear-during-covid-19) 的所有准则。这包括在接触隐形眼镜前后洗手，定期消毒隐形眼镜，生病时停止使用，扔掉过期的隐形眼镜。

“保护你的眼睛，保护你的健康，保护你周围人的健康，”Steinemann 说。
