# 如何打造终极“小鬼当家”

> 原文：<https://lifehacker.com/how-to-create-the-ultimate-kid-cave-1832334325>

我住在宾夕法尼亚州东部的一栋建于 1925 年的双胞胎住宅里。它的魅力很高，但在其他方面却很低，比如居住空间、储物空间和浴室数量。随着我儿子年龄的增长，玩耍的日子也从许多必要的徘徊中演变而来(“让我们记住分享吧！))为了更少干涉，我们一直在努力为他找一个和朋友们一起玩的地方。



他们会在他的卧室里玩一会儿，但我们还不允许他在他的房间里有电视，所以一旦他们想玩视频游戏或吃零食和看电影放松一下，他们就会回到我们的客厅，这既小又紧挨着我们同样小的餐厅/厨房，这意味着我们都在彼此的上面。

一个多雨的夏天和秋天给我 93 年未完工的地下室造成了一些水渍，之后，我面临着一个大的(意思是:昂贵的)“防水”项目，包括一个排水泵装置和一些更好看的(但仍未完工的)墙壁。我想到，有了更亮的墙壁、一点家具、一台旧电视，或许再加上一张毛绒绒的地毯，我就可以把地下室的一部分变成我儿子和他朋友的小游戏/闲逛空间。我把我的新视野命名为“小孩洞穴”

到了秋天，我开始清理并出售地下室里任何不能给我带来“快乐”的东西(或者说我已经五年没用过了，这种情况经常发生)。

接下来，四个男人来了，制造了我有史以来清理过的最大的混乱:

然后，我开始寻找一种方式来装饰这个空间，使它看起来古怪，在冬天感觉温暖。我还需要一种方法将“洞穴”与地下室的另一半隔开，那是一个储藏和洗衣的地方。哦，我不想花太多钱。他收到了几张海报和一个“儿童洞穴”的圣诞标志，所以我知道这些会挂在墙上。我们一直计划升级我们的主电视，所以旧电视将是他的。但除此之外，我几乎是从零开始。

我们在木梁上穿了一些旧电缆，挂了两个浴帘来隔开地下室的另一部分。我最大的挥霍是这张粗毛地毯 ，它是如此的蓬松和美妙，以至于物有所值。地下室的主地毯很薄，铺在混凝土地板上，所以更豪华的东西是关键。

但我最喜欢的东西——也是最让我儿子钦佩的东西——是这个巨大的、栩栩如生的两个冲锋队的纸板剪影:

它们很有用(它们守护着污水泵)，它们令人印象深刻地高，而且在我买下它们的头一两周，每当我抱着一堆要洗的衣服转向它们时，我都会喘一点气。

然后，我们把旧电视放在不再使用的梳妆台上，这也是存放他的视频游戏的最佳位置:

点睛之笔？几条毯子和表情符号枕头:

特别是便便表情符号枕头，每一分钱都是值得的，因为每次我儿子有新朋友来玩并带他们下楼时，他都会说，“只是警告你:我在沙发上便便了。两次。”

我儿子现在为他的朋友们提供了一个有趣的闲逛空间，他们离我们足够近，我们可以听到他们(大部分)的表现，我们可以收回客厅，看新电视。
