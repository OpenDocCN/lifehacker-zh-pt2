# 如何更新谷歌刚刚禁止的付费 Chrome 扩展

> 原文:[https://life hacker . com/how-to-update-payed-chrome-extensions-Google-just-banned-1841311124](https://lifehacker.com/how-to-update-paid-chrome-extensions-google-just-banned-1841311124)

我承认，我已经有一段时间没有碰到付费的谷歌扩展了。 [他们](https://chrome.google.com/webstore/detail/open-seo-statsformerly-pa/hbdkkfheckcdppiaiabobmennhijkknn?hl=en)[do](https://chrome.google.com/webstore/detail/timed-test-division-facts/mimjmfnekgfaepenmgcfdekeenpocmcg)[存在](https://chrome.google.com/webstore/detail/amazon-relay-auto-refresh/gooaddljkpdcjbdigogmajlcgifjjhgp) ，然而——还有一些其实是有用的，比如[stop themaness](https://chrome.google.com/webstore/detail/stopthemadness/gfdahjcdgfioalhekjflcpijhngbjgdo)(一个让网站无法禁用键盘快捷键或上下文菜单的扩展)。 如果你碰巧发现(并购买)了付费的 Chrome 扩展，你需要知道他们可能会在一段未知的时间内收不到更新。

Watch

这一点很重要，因为如果你已经付费，你可能想确保你的扩展的体验没有错误，并且尽可能的功能齐全。但是谷歌暂时禁止发布任何使用 Chrome 网上商店支付的 T2 Chrome 扩展。根据开发商倡导者 [Simeon Vincent](https://groups.google.com/a/chromium.org/forum/#!topic/chromium-extensions/EW0VuDjZSO4) 的帖子:

> 本月早些时候，Chrome 网上商店团队发现涉及付费 Chrome 扩展的欺诈交易数量显著增加，目的是剥削用户。由于这种滥用的规模，我们已经暂时禁止发布付费项目。这是一项临时措施，旨在阻止这种流入，因为我们正在寻找长期解决方案，以解决更广泛的虐待模式。
> 
> *[...]*
> 
> 我们正在努力尽快解决这个问题，但目前我们还没有解决时间表。抱歉给您带来不便。"

如果你已经购买了 Chrome 扩展，我建议你在他们的网站和社交媒体渠道上联系他们的开发者，看看他们有什么计划来更新他们的扩展。虽然他们可以在提交新版本的扩展时对谷歌的拒绝邮件提出上诉，但不能保证他们会得到更新。正如一位开发者在回复文森特的帖子时所写的:

> *“我已经多次写信回复拒绝信，内容是关于我的两个在商店里存在了一年多的付费延期。我没有收到任何回复，延期仍处于待审查状态。”*

很有可能你必须在你的浏览器中 [下载一个新版本的](https://lifehacker.com/how-you-can-still-download-chrome-extensions-without-us-1826796797) 扩展，这不是我通常建议做的事情——浏览器安全和所有的。然而，只要你直接从开发者那里下载一个合法版本的扩展，风险就很小。一旦你的扩展的开发者发现更多成功地将更新推送到 Chrome 网络商店，或者谷歌放松对付费 Chrome 扩展的临时禁令，你就可以切换回使用该扩展的常规自动更新版本。

一般来说，你可能不需要为 Chrome 浏览器扩展付费，除非你有直接的业务需求。可能性很大，有很多其他的扩展和服务可以实现你想做的任何事情，而且价格很低。

例如，考虑一下 [定时测试](https://chrome.google.com/webstore/detail/timed-test-division-facts/mimjmfnekgfaepenmgcfdekeenpocmcg) 或 [闪存卡](https://chrome.google.com/webstore/detail/flash-cards-division-fact/ehdcnknapbpfnbacjkdcfjnaadicclml?hl=en-US) 扩展。快速搜索可以找到大量的[备选方案](https://chrome.google.com/webstore/detail/addiemath-math/hmkomnmahappbhodlbgopniilemgiadf?hl=en) ，我敢肯定，如果你花 20 秒以上的时间进行搜索，你会找到更好的东西。这并不是说所有付费的扩展都是垃圾，但是在你付费购买其他地方可能以更好的形式存在的功能之前，请确保你带着一点怀疑的态度对待它们——免费。
