# 你可能不想要 Sinemia 的新借记卡

> 原文：<https://lifehacker.com/you-probably-dont-want-sinemias-new-debit-card-1830728397>

MoviePass 的竞争对手 Sinemia 最近因 [在顾客提前购买电影票时收取 1.80 美元的“手续费”](https://www.doctorofcredit.com/sinemia-adds-new-1-8-processing-fee-for-pre-paid-movies/) 而受到抨击(它已经 [对预售](https://www.reddit.com/r/Sinemia/comments/8d1b53/convenience_fees/) 收取了另一笔费用)。如今，随着实体借记卡的重新引入，它提供了一种“免除所有费用”的新选择。有了新卡，你可以在上映当天的票房或者在影院网站上购买电影。



如果你已经被通过该服务购买门票的相关费用所困扰，那么实体卡听起来是个不错的主意。除了这里有几个(相当大的)警告需要考虑

### 这张卡不是免费的

是的，使用 Sinemia 借记卡，您可以在订票时免除费用。也就是说，你不会完全避免费用。这张卡本身有一笔相当可观的费用:14.99 美元。对于那些给自己的 Sinemia 账户增加一点锻炼的人来说，14.99 美元的费用可能是一笔储蓄，但对于一个月一部电影的计划或类似计划的人来说，这笔储蓄要少得多。

### 它不收早票的费用

虽然你可以使用实体借记卡在票房和提前购买门票，但你实际规避费用的唯一方式是在你计划参加的放映的票房日或放映时使用它购买门票。

Sinemia 相对于竞争对手的最大优势之一是你可以提前购买门票。再说一次，如果你总是在走进电影院之前在售票处买票，借记卡是有意义的，但对其他人来说，这根本不是一笔交易。你用实体卡支付的费用和用数字卡支付的费用完全一样，除了现在你还为这张借记卡支付了 14.99 美元。

所有这些都是说，虽然实体卡对一些用户来说可能是一笔好交易，但对你来说更有可能是一笔坏交易。像所有的事情一样，在你注册之前，确保你理解所有的细则。如果你想注册，你可以从今天开始在 Sinemia 网站上的个人资料页面申请一张实体卡。
