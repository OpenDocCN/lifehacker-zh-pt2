# 制作这些经典电脑的纸艺模型

> 原文:[https://life hacker . com/build-these-paper craft-models-of-classic-computers-1843155398](https://lifehacker.com/build-these-papercraft-models-of-classic-computers-1843155398)

现在有很多打发时间的方法。你可以读更多的书。多看电视。晚饭后进行更长时间的家庭散步。或者— *或者—* 你可以制作和你一起长大的电脑的纸质模型，这样你的孩子就可以看到(并且欣赏，*该死的*！)科技已经走了多远，他们有多幸运，他们的 Chromebooks、ipad 和 Nintendo switch 是什么样的。

Watch

平面设计师 Rocky Bergen 致力于帮助我们通过 [将我们最喜爱的经典电脑和游戏系统(某种程度上)带入生活，这是一组免费可下载的 papercraft 模型](https://rockybergen.com/papercraft) ，其中包括 [Commodore 64](http://rockybergen.com/whatsnew/2018/6/28/commodore-64-papercraft-v3) 、 [Apple II](https://rockybergen.com/whatsnew/2018/11/27/apple-ii-papercraft) 、 [任天堂 GameCube](http://rockybergen.com/whatsnew/2018/7/21/nintendo-gamecube-papercraft) 、Atari 520 ST、T21

他甚至为许多型号提供了各种屏幕视图，这样你就可以根据自己使用它们的美好回忆对它们进行个性化设置(不过，Apple II 的俄勒冈小道选项真的是唯一的选择)。甚至他讲述的每个模特的故事都很可爱。以下是他对打造 [Amiga 500 车型](https://rockybergen.com/whatsnew/2019/1/15/amiga-500-papercraft-design) 的看法:

> 几年前，我买了一个 Amiga 和那个笨拙的 A-520 视频连接器。我花了大半个星期的时间试图让一幅图像出现在我的电视上。几个星期后，我点了一份比萨饼，当送比萨饼的人把头探进我的公寓时，他说:“亲爱的阿米加！你需要一个显示器和一些免费的游戏？”当然，我答应了。问题是，我从来没有成功地让阿米加做任何事情。
> 
> 现在，我有了一个树莓 Pi，它模拟了 Amiga 体验的最佳部分，并且只花了几分钟就完成了设置。我想我花了这么多时间做这些图案是为了弥补所有这些亵渎。

只需下载、打印、剪切、折叠和粘合你的模型；在其中一个帖子的评论中，Bergen 说他更喜欢这个 80 磅的 T2。白皮书创造他的模型。最终结果有利于家庭办公室装饰。

当你这么做的时候，让孩子们参与到你的项目中来，把你的怀旧之旅变成一次回到过去的家庭度假—为什么不告诉他们所有关于拨号上网的事情，然后给他们听一段的录音，那段 [不会弄错的来自你童年的声音](https://www.youtube.com/watch?v=gsNaR6FRuO0) 。
