# 哪里可以修复超出保修期的 iOS 设备

> 原文：<https://lifehacker.com/where-to-get-your-out-of-warranty-ios-device-fixed-1837845689>

拥有过保修期的 iPhones 或 iPads 需要维修的苹果用户现在可以通过该公司最近扩展的独立维修计划获得更多选择。



以前，苹果用户必须通过当地的苹果商店或苹果认可的服务提供商(AASP)计划中的公司来维修他们的设备。虽然苹果在最近几个月扩大了许多这样的地点——包括 [所有美国百思买商店](https://lifehacker.com/you-can-now-get-your-broken-apple-device-repaired-at-be-1835658223)——第三方公司可以申请成为独立维修提供商(IRP)计划的一部分，该计划允许他们访问苹果的技术培训材料、诊断工具，以及直接从苹果购买原装替换零件的特殊运输。同样，独立服务商可以申请自助服务账户(SSA)计划，该计划提供与独立维修提供商计划相同的好处，但只允许公司维修他们直接拥有或租赁的 iOS 设备。公司和独立技术人员可以在线申请[【IRP】](https://support.apple.com/irp-program)或 [SSA 项目](https://support.apple.com/self-servicing-account-program) 。

虽然客户不能自己申请进行维修，但这两个项目都让企业更容易获得苹果官方维修认证，这意味着更多公司将能够修理我们损坏的 MAC 电脑、iPhones 和其他苹果设备。以下是如何找到你附近的可用选项:

1.  前往苹果的支持页面 [找到离你最近的苹果认证维修点](https://getsupport.apple.com/) 。
2.  选择您的设备，然后选择问题。
3.  选择“送修”
4.  使用您的 Apple ID 登录
5.  使用搜索栏搜索附近的维修地点。所有认证的维修选项都会显示在地图上。
