# 为什么旅行时应该总是用当地货币支付

> 原文：<https://lifehacker.com/why-you-should-always-pay-in-local-currency-when-travel-1838844467>

当你考虑申请哪张信用卡时，你可能首先会注意到一张卡是否有签约奖金或提供高回报率。但如果你计划出国旅行，你应该忽略奖励，确保你得到了越来越普遍的信用卡好处:零国外交易费。

当您在另一个国家使用借记卡或信用卡时，您的银行必须将该费用兑换成美元，以便从您的账户中提款。这方面的费用可能在百分之一到百分之三之间。

这可能看起来不多，但是使用不收取这笔小额费用的信用卡还有一个额外的好处:你还可以避免最近的旅行者诈骗。

Vindu Goel 在《纽约时报》 上讲述了他的故事:在印度一个受欢迎的旅游景点吃饭时，服务员在账单上加了 5%,然后用美元而不是印度卢比开具收据。服务器没有征得他的同意就这么做了，Goel 指出这是 Visa 和 Mastercard 所要求的。这一举措被称为动态货币兑换。“外国商人在你的账单上加了很高的价格，将其兑换成你的本币，然后声称这对你有利，”Goel 解释道。

不只是旅游景点拉这个机动。经常在世界各地旅行的 Goel 说，从附近的药店到大型连锁酒店，他到处都能看到这种情况。有时你可以在刷卡机上选择用当地货币还是本国货币支付。

这就是零交易费卡派上用场的地方。如果你有的话，你可以选择——或者要求，如果需要的话——用当地货币支付。如果你有美国运通卡，你就不用担心这个——它不允许动态货币兑换。

戈埃尔说，然后，你应该仔细检查你的收据，确保商家没有在未经你同意的情况下兑换货币。他写道，“当收费单上用极小的字体列出你用本币表示的金额，声称你同意时，你就暴露了。”。
