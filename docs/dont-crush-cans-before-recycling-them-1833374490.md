# 回收罐头之前不要压碎它们

> 原文：<https://lifehacker.com/dont-crush-cans-before-recycling-them-1833374490>

你有没有感觉到在回收之前需要粉碎东西？好吧，这通常是正确的本能，因为它节省空间，并且对于回收者来说更有效。例如，塑料瓶在扔进回收站时通常最好被压碎(并在 上留下 [盖子)。](https://lifehacker.com/stop-recycling-plastic-bottles-without-caps-on-1832992981) 

Watch

然而，铝罐是一个罕见的例外，如 [科普](https://www.popsci.com/how-to-recycle) 上周提到的。事实证明，如果易拉罐被压碎或压扁，你通常会让回收设施的工作变得更加困难。

据美国铝业协会[的公共事务高级主管马特·米南(Matt Meenan)称，当被压碎的易拉罐进入回收流程时，它们变得更加难以分类，并可能污染其他可回收材料。一个扁平的汽水罐可以被分类为“纸”，例如，从而污染了可回收的纸。“压碎的铝罐可能会通过分拣设备的空间掉落，要么完全丢失，要么分拣不当，”他补充说。](https://www.aluminum.org/)

然而，这确实伴随着一个警告，那就是它可能依赖于你的回收基础设施。所有美国的回收项目都基于两种方法中的一种:单流回收(将所有可回收物品扔进路边的一个箱子里的那种)和多流或双流回收(在路边将它们分成不同的箱子)。

如果你在一个多溪流循环利用的城市，你很幸运。无论是否被压碎，你都可以用任何你想要的方式回收它们。

然而，对于许多美国城市来说(根据 2015 年的一份报告，美国十大城市中有九个)，单流回收是首选方法。这意味着你在一个尽可能完好无损地保存罐头的城镇里。

那么，你应该如何处理你的问题罐头呢？首先，你可以确定什么样的路边回收项目在你的社区运作。这很简单。如果你把你分开的玻璃、纸和塑料一起扔在路边的一个垃圾箱里，你就参加了一个单流项目。

如果这还不明显，那么很简单，当你把易拉罐扔进回收箱时，确保你不会试图压碎或弄平它们。同样，如果你是单身的 T2 流或者只是不确定，小心行事，不要让他们拥挤。凹痕可能是好的(它们不需要保持原样)，但要忍住不要挤压。
