# 这些是最好的笔记本

> 原文：<https://lifehacker.com/these-are-the-best-notebooks-1835480916>

拥有一个最喜欢的笔记本比拥有一支最喜欢的笔 还要傻。那是一堆你用来写字的纸。但是细微之处和特殊功能确实会影响您的工作流程。一本精装的 Moleskine 给你的笔记增添了几分档次，但是如果你想在看完一页纸后把它撕掉，这是一个糟糕的选择。这里是 Lifehacker 员工最喜欢的笔记本，以及我们为什么喜欢它们。

Watch

### 弗吉尼亚·k·史密斯，执行主编:

我有一个 [统治者，红色经典鼹鼠](https://us.moleskine.com/en/classic-notebook-red/p0456?ftm=9788862930048&gclid=Cj0KCQjwxYLoBRCxARIsAEf16-sqYiedTwCl1chgfsYMChTSbCvC_iE_aZSiGhAl5Lxuv7xJgFo8_0kaAvzrEALw_wcB) ，我很喜欢。它令人愉快，引人注目，让我比用黑色笔记本时更倾向于使用它。结实，适合我的包，完成任务。

### **米歇尔·吴，育儿编辑:**

笔记本应该 1)足够小，可以放进我的腰包，这样我就可以在散步的时候随身携带；2)灵活但不轻薄；3)网格设计；4)价格便宜。无印良品品牌的[5mm 网格 A5 尺寸笔记本](https://www.muji.us/store/4548076398349.html) 符合这个标准。

### 尼克·道格拉斯，特约撰稿人:

穿孔，穿孔，穿孔！一旦我数字化了一页文字，我就想扔掉它。顶端螺旋统治的罗地亚 大约有半个字母大小，大到足以扩展写作，但小到足以整天携带。我可以一页一页翻过去，所以笔记本可以平放，只占一页的空间。

### 健康编辑贝丝·斯克沃斯基:

Leuchtturm1917 是最好的精装笔记本。页面已预先编号，并且有一个空白目录。它们配有两个带状书签、一个橡皮圈封口、一个后口袋和一些粘性标签，当你将它们归档到书架时，可以将它们贴在书脊和封面上。我用 [中号【A5】](https://www.leuchtturm1917.us/notebook-medium-a5-hardcover-251-numbered-pages-5-3-4-x-8-1-4-in.html)，如果我觉得心情不好的话就用点画纸，否则就用线条或网格。

选择一种颜色很难，但我建议你在书店挑选每一种颜色的时候，考虑一下你的个性、希望和梦想。你不能求助，你要用心做这个决定。到目前为止，我的颜色是浆果色、天蓝色、冰蓝色和柠檬色。

### **乔尔·卡恩，高级视频制作人:**

我真的不知道我的笔记本的名字(因为它是我几年前在一次工作活动中得到的赃物)，但它的主要特点是:封面是非常硬的纸板，这使它几乎像一个剪贴板。易于在任何地方书写！基本上是内置表！此外，螺旋与纸张的比例较小，因此您可以轻松地在反面书写，而不会损失一半的页面空间来为您的手腾出空间。 [这个](https://www.amazon.com/dp/B07CWBNQCF/ref=twister_B07FBCSXWL?_encoding=UTF8&asc_campaign=InlineText&asc_refurl=https://lifehacker.com/these-are-the-best-notebooks-1835480916&asc_source=&psc=1&tag=kinjalifehackerlink-20) 似乎是最接近的近似值。

### 特约撰稿人梅根·沃尔伯特:

我用一个 [标准子弹日志](https://www.amazon.com/Notebook-Fineliner-Reusable-Stencils-Schedule/dp/B07L4CB6WS/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/these-are-the-best-notebooks-1835480916&asc_source=&tag=kinjalifehackerlink-20) ，连同各种各样的笔、贴纸和胶带来想象它。我用它来做每周/每月的计划，以及保存待办事项、长期目标跟踪和提醒的地方。橙色封面，因为橙色是欢快的。

### 科技编辑大卫·墨菲:

简单的[Moleskine](https://www.amazon.com/Moleskine-Classic-Cover-Notebook-Ruled/dp/8883701127?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/these-are-the-best-notebooks-1835480916&asc_source=&tag=kinjalifehackerlink-20)——划线，这样我写的东西看起来就不像是在试图写一封勒索信什么的。

### 创意制作人希瑟·哈斯:

我会用任何东西来写单词。可能是因为我喜欢写作(美学上的)，所以我的大部分笔记都有点...没用但是好看！对于绘画，我有点挑剔:比起任何素描本，我更喜欢鼹鼠皮的牙齿。

### 主编梅丽莎·基尔希:

过去，你必须去欧洲才能买到一本好的笔记本，我说的“好笔记本”是指那种没有广泛规则的，白纸上的蓝线，哼哼。我的巅峰笔记本体验包括我在罗马买到的一个鞍钉号码，它有一个棕色硬纸板封面(早于)和里面的绘图纸，它的网格是*棕色*，每页上都有这种微妙的棕色梯度。相信我，这太棒了。

从那以后，我到处寻找能与那个怪异的罗马笔记本媲美的东西，我找到的最好的是普林斯顿建筑出版社 的 [网格&指南笔记本。封面是一个坚实的布面纸板，带有压花网格图案，手感很好。在里面，每一页都有不同的令人愉快的网格或圆点或图案布局。散落在随意页面上的小图表充当了味觉清洁剂:*在你记笔记之前，为什么不看看这些可爱的不同种类的结的小图呢？*是个不错的笔记本！](https://www.amazon.com/Grids-Guides-Red-Notebook-Thinkers/dp/1616894229?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/these-are-the-best-notebooks-1835480916&asc_source=&tag=kinjalifehackerlink-20)

### 美食编辑克莱尔·洛儿:

我目前使用四个不同的笔记本，但我最喜欢的是 [这个剪贴板的情况从纸老虎](https://milligram.com/papier-tigre-clipboard-suits-a4-notepads)T3】。我在这些纸上做了大部分的食谱草图和开发，然后我可以根据需要撕下并归档。此外，将剪贴板放在潮湿的地方比将笔记本放在潮湿的地方要少得多。
