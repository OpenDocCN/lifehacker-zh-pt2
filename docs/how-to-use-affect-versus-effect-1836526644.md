# 如何使用“影响”和“效果”

> 原文：<https://lifehacker.com/how-to-use-affect-versus-effect-1836526644>

每次我写“影响”或“效果”时，我都要停下来仔细检查一下自己。他们的意思是相关的，他们的拼写是相似的；甚至他们的发音也几乎一样。但是其中一个*导致*某事，另一个是*结果*。知道区别是一件好事。

Watch

### 影响

[影响](https://www.dictionary.com/browse/affect) 是(通常)动词。是行动，是事情让一切变得不同。某事*影响*某人或某事。

剧院明亮的灯光影响了演员的表演。更严格的饮食和锻炼习惯应该会对 T2 的健康产生积极的影响。噩梦很可能会*影响我一整天。*

### 影响

[效果](https://www.dictionary.com/browse/effect?s=t) (通常)是名词。这是结果，行动的后果。

新法律的影响还不完全清楚。如果你戒烟，它将对你的健康产生积极的 T2 效应。我们都会感受到气候变化的*效应*，这只是时间问题。

Effect 在短语中的使用方式也更加灵活。新政策可以*生效*(不生效)，一旦生效，就正式*生效。*它也可以变成一个形容词，这是一种有效的*方式来表达你的观点。*

### 一些可能有帮助的技巧

如果你需要一点记忆装置来帮助你记住哪个是哪个，我会和你分享我的。我对自己说,“A”代表“行动”和“影响”所以如果是动词，那就是情感。

Lifehacker 的执行编辑 Virginia Smith 与我分享了另一个很好的例子:她想到了“特殊 ef-x”这个短语，因为特殊效果是对象。所以如果她写的单词是一个东西(名词)，它以“e”开头

你也可以使用来自 [作家文摘](https://www.writersdigest.com/online-editor/affect-vs-effect-2) 的这个建议短语:“**动作**是**影响**，**结束**结果是**效果**”

### 是时候让它变得更难了

现在你知道了规则，你知道接下来会发生什么:例外！这就是英语如此有趣的原因。

情感几乎总是一个动词，除了在极少数情况下，它是一个名词，用来描述一个人的情感或欲望。 [《读者文摘》作家梅根·琼斯](https://www.rd.com/culture/affect-versus-effect-difference/) ，他有一个可爱的名字，给我们举了一个很好的例子:

> “Affect”作为一个名词，描述一种感觉、情感或举止，通常会在它前面加上一个形容词:“我注意到我朋友悲伤的 *affect* 并问他是否还好。”

effect 是名词，*除了*偶尔是动词，意思是“导致某事发生”琼斯也举了一个例子:

> “效果”作为动词的意思是“带来”，你会看到它几乎只用于“改变”之前，比如“年轻人有很大的力量去*影响*改变。”

不过，这些情况不太常见，所以如果你能记住，影响是行动，结果是结果，那么大多数时候你都是对的。这会影响你的写作，效果是你会看起来更聪明。
