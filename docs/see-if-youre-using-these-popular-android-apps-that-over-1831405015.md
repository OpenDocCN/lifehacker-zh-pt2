# 看看你是否在使用这些向脸书过度分享信息的流行安卓应用

> 原文：<https://lifehacker.com/see-if-youre-using-these-popular-android-apps-that-over-1831405015>

最近的一项隐私国际研究 发现，Google Play 中 42.55%的免费应用程序可以与脸书共享数据，许多热门应用程序在打开的一秒钟就与脸书共享数据。

Watch

根据这项研究，该小组测试的应用程序中有 61%会在用户打开应用程序时自动将数据传输到脸书，不管这个人是否有脸书账户，也不管他们是否在设备上登录了该账户。

其中一些应用向社交网络发送可能被视为敏感的详细信息。例如，Kayak 向脸书发送人们的航班搜索信息，包括他们的出发城市、日期和机场，以及他们计划去哪里，他们想呆多久，以及他们有多少人同行。它也分享关于某人正在搜索的票的类型的信息。

该研究确定向脸书发送数据的其他流行应用包括 MyFitnessPal、Duolingo、Skyscanner、猫途鹰、Spotify、Yelp、Shazam 和 Indeed。

据报道，脸书已经通过其开发者工具包的新版本纠正了这个问题，但这些应用程序尚未实现它，或者至少没有正确实现它， [*Engadget* 报道](https://www.engadget.com/2018/12/30/more-popular-apps-are-sending-data-to-facebook-without-asking/) 。

这个问题似乎不会影响 iOS 版本的应用程序。

你可以在这里免费查看的完整报告。此外， [还为所有打开后立即向脸书传输数据的应用](https://www.privacyinternational.org/campaigns/investigating-apps-interactions-facebook-android) 创建了详细报告。

如果你使用它们中的任何一个，当你这样做的时候，知道你正在分享什么信息是值得的。
