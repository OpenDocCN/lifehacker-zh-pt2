# 是的，你应该停止使用谷歌浏览器

> 原文：<https://lifehacker.com/yes-you-should-quit-using-google-chrome-1844408232>

是时候 [停止使用 Chrome](https://www.wsj.com/articles/quit-chrome-safari-and-edge-are-just-better-browsers-for-you-and-your-computer-11594558801) 了吗？我们当然已经准备好放弃占用资源的浏览器，特别是考虑到最近有消息称，微软对 Chrome 占用内存问题的有望修复已经被谷歌自己挫败。



其他浏览器在速度、性能、隐私和/或安全性方面早就盖过了 Chrome，如果你不需要，就没有必要坚持使用如此臃肿和缓慢的东西——你也没有必要，因为已经有了更好的 Chrome 版本、，但它、不是由谷歌开发的。

### Chrome 是一个系统猪

Chrome 占用了你设备的内存，消耗了你设备的电池寿命。谷歌正在测试 [的实验性功能，以此来打压](https://lifehacker.com/how-to-try-out-chrome-86s-big-battery-saving-fix-today-1844283047) ，但他们只到此为止。更糟糕的是， [谷歌最近禁用了微软的一项节省内存的功能](https://www.techradar.com/news/google-kills-off-microsofts-ram-saving-feature-for-chrome-on-windows-10) ，该功能让 Windows 10 用户以性能问题为由遏制 Chrome 的系统停滞。

Chrome 占用资源的行为背后有几个因素，但它们大多是 Chrome 独有的；其他基于 Chromium 的浏览器如 Edge Chromium 和 Brave 没有 Chrome 的胃口。Safari 和 [Firefox](https://lifehacker.com/why-you-should-switch-from-google-chrome-to-firefox-1821879163) 这样的非 Chromium 选项也不行。

### Chrome 的内置广告拦截功能有限

谷歌的商业模式以广告为主，你在网络上看到的许多定向广告都来自谷歌(或者根据谷歌的数据提供信息)。为了确保用户仍能看到这些广告并与之互动， [谷歌限制了广告的类型](https://hub.packtpub.com/google-starts-experimenting-with-manifest-v3-extension-in-chrome-80-canary-build/) ，像 AdBlock 这样的内容屏蔽扩展可以禁用这些广告，这使得真正无广告的 Chrome 体验成为不可能。是的， [可以屏蔽部分广告，](https://lifehacker.com/block-bloated-web-ads-with-these-hidden-chrome-80-flags-1841470223) 却全部屏蔽。

然而，像 Brave(或其无广告软件替代产品 [Bold](https://github.com/BoldBrowser) )、 [Opera](https://lifehacker.com/protect-yourself-from-cryptojacking-with-opera-s-ad-blo-1822304630) 和 [Vivaldi](https://lifehacker.com/block-ads-and-tracking-with-vivaldis-new-built-in-brows-1843006826?rev=1587585580265) 这样的浏览器提供了类似 Chrome 的无广告浏览。它们还会屏蔽网站追踪器，并且通常会提供更好的隐私和安全特性。见鬼，Brave 对广告屏蔽如此重视，如果你决定*而不是*屏蔽广告 ，它就会*付钱给你*——即使已经因为 [用自己的标签](https://gizmodo.com/brave-blows-up-its-whole-reason-for-existing-1843951126) 替换附属链接而被指责，这是粗略的 AF。

### Chrome 的隐私控制没有那么全面

Chrome 在防止你的浏览活动和个人数据落入黑客和可疑的外部公司之手方面做得不错，但谷歌自己却做不到。

公平地说，谷歌的隐私控制比以前更好。用户可以 [控制谷歌收集的部分数据](https://lifehacker.com/how-to-automatically-delete-your-google-data-and-why-y-1839614022) ， [谷歌账户现在默认自动删除保存的数据](https://lifehacker.com/check-the-auto-delete-defaults-on-your-new-google-accou-1844153391) 。尽管如此，还是有很多技术 [谷歌使用](https://gizmodo.com/how-to-track-the-tech-thats-tracking-you-every-day-1843908029) 来收集和分享你的信息。

Chrome 会追踪你的数据，并尽可能合法地将数据传输到谷歌的服务器上——尤其是当你经常使用谷歌账户登录的时候。谷歌跟踪你不同于黑客或跟踪者收集你的浏览活动信息，但它仍然带来重大的隐私风险，特别是如果一个配置错误的服务器最终泄露了一堆个人信息 。

### 不再是最好的 Chromium 浏览器

Chromium 浏览器时尚、快速且易于使用。多年来，Chrome 一直是最好的 Chrome 浏览器，正因为如此，用户忍受了它的怪癖和缺点——无论是谷歌搜索偏向某些页面的方式，还是它如何削弱浏览器插件，从而破坏其基于广告的商业模式。

谷歌优先考虑其第一方产品是有道理的。本质上，我甚至不认为这样做是错的。Chrome 与谷歌其他应用和服务的无缝集成*很方便*。 B 但是并不是每个人都想仅仅因为喜欢基于 Chromium 的浏览体验而与谷歌的生态系统紧密相连。

幸运的是，现在有许多基于 Chrome 的替代品，它们都提供了几乎相同的类似 Chrome 的浏览器，没有谷歌的产品集成。

哦，如果你想的话，你仍然可以在几乎所有其他浏览器中使用 Gmail、Google Drive、YouTube 和其他谷歌网络应用程序——T2 T3，尽管也有很多很好的选择来替代 T4 T5。
