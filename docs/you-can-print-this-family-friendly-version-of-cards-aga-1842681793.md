# 你可以在家免费打印这种家庭友好版的反人类卡片

> 原文：<https://lifehacker.com/you-can-print-this-family-friendly-version-of-cards-aga-1842681793>

如果你们都待在家里，想找些有趣的事情和家人一起做，反人类的 [卡有一个新的适合家庭的游戏版本](https://www.cardsagainsthumanityfamilyedition.com/) ，你可以在家里下载并打印。



反人类卡片:家庭版是为 8 岁及以上的儿童设计的。该网站指出，“我们从未见过你的孩子，但我们为 8 岁及以上的人设计了这款游戏。我们鼓励父母在你和家人一起玩之前仔细看看这些卡片，去掉任何你不喜欢的东西，因为不同的孩子有不同的敏感度。”

这是一个全新版本的游戏，不仅仅是你一直在玩的没有脏卡的反人类的相同的卡。

它已经在咨询儿童发展专家和心理学家的基础上发展了一年多。最初它将于秋季发布，但考虑到疫情，该公司决定提前发布。

可打印版本是一个“测试版”,虽然它已经在儿童和成人中进行了测试，但还没有完全完成。最终，希望能以盒子的形式发布给大众。

可打印版本有小卡片和大卡片 PDF 两种形式。小卡片版本需要 21 张纸，大卡片版本需要 47 张纸。这款游戏目前总共有大约 420 张卡片。

如果你家里有无聊的孩子，那么打印后把这些卡片剪下来可能也是一件有趣的事(也很有用！)下午的活动将会以一个新游戏结束，一旦游戏结束，你全家都可以玩这个游戏。
