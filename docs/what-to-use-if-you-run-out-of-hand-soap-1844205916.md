# 如果洗手液用完了该用什么

> 原文：<https://lifehacker.com/what-to-use-if-you-run-out-of-hand-soap-1844205916>

因此，在全球疫情期间，你的洗手液已经快用完了，而且由于供应链问题和持续的抢购，很难再补充洗手液。或者也许你不能冒险去商店买更多，直到你用完了。还是要勤洗手——怎么办？



原来你可以用家里其他类型的肥皂洗手，包括洗碗精、沐浴露和肥皂。正如医学专家 [告诉战略家](https://nymag.com/strategist/article/can-you-use-dish-soap-to-wash-your-hands.html) 的那样，其他类型的肥皂可以成为洗手液的良好替代品，尽管有些是比其他更好的替代品。

这里有一些你可以使用的:

1.  **条形肥皂**。条形肥皂很便宜，而且可以用很长时间——只要确保不用时存放在干燥的地方。
2.  **沐浴露**。专家说，沐浴露通常与洗手液最相似，因为它是为你的皮肤准备的。
3.  **洗发水**。有些洗发水会在你的皮肤上留下奇怪的残留物，但它们可以很好地洗掉细菌。
4.  **洗碗皂**。如果你有其他选择，这款应该是你的最后选择，因为如果每天整天使用，它可能会对你的皮肤更刺激。但是你可以在紧要关头用它。

根据疾病预防控制中心的说法，你的肥皂是否被宣传为抗菌的并不重要——它的指导方针说 “对消费者来说没有额外的健康益处...与使用普通肥皂相比，使用含有抗菌成分的肥皂。”虽然洗手液是一种选择(尽管在许多地方可能比实际的肥皂更难获得)，但如果可能的话，肥皂和水一般被推荐用于清洗 [。](https://www.cdc.gov/handwashing/when-how-handwashing.html)

因此，如果你需要使用洗手液替代品，你可以。重要的是，你要继续定期洗手，并继续遵循我们谈论了几个月的最佳实践 :用肥皂洗手至少 20 秒，并用温暖、干净、流动的水清洗。而且做完之后涂上乳液大概也无妨。
