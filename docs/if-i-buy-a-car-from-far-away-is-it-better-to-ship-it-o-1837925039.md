# 如果我从很远的地方买车，是发货好还是开回家好？

> 原文：<https://jalopnik.com/if-i-buy-a-car-from-far-away-is-it-better-to-ship-it-o-1837925039>

作为 Jalopnik 的常驻购车专家和专业购车者，我会收到电子邮件。很多邮件。我决定挑几个问题，试着帮帮大家。本周我们将讨论从远方购买汽车，日本豪华品牌的定制订单，以及如何合乎道德地卖掉一个柠檬。

首先，如果你是从远处购买一辆汽车，是坐飞机回家还是用船运？

> “我正在挑选一辆低里程二手雷克萨斯(可靠性可靠性可靠性！)，但它离我舒适的家有 1900 英里。
> 
> 我是跳上一架飞机，几天后把车开回来，还是想办法把它运走？
> 
> 搜索网页，似乎有一个航运经纪人/公司的恐怖故事甚至混合。我问经销商，他们是否可以帮助安排或有一个首选供应商。另一方面，如果我把它开回来，它必须是及时的，因为我目前没有足够的灵活性来延长休假，并进行一次驾驶的乡村探索。"

这是一个非常好的问题。我要说的是，在我每月处理的汽车交易中，有一半以上是我客户的长途交易。通常，客户希望将运输成本降至最低，因此他们会设法搭乘航班，长途驾车回家。事情是这样的，一旦你开始计算长途驾驶的费用，包括机票，食物，燃料，事情开始增加。

但是你也不想忘记这个等式中最重要的部分，你的时间。每个人的时间都有货币价值，当你把你要花费的所有时间加上其他成本，运费通常不会那么糟糕。几乎在我为长途汽车工作的每一个案例中，交通费通常比个人旅行费用便宜或相等。我要说的是，如果你想开着你的新车长途旅行，看看风景，这是值得的。

然而，当涉及到船舶与驱动场景时，成本和汽车的类型很重要。在决定购买之前，你可能会想亲自评估一辆便宜的或项目车，但你可能不想依赖项目车带你一路回家。

接下来，你能订购你想要的讴歌跨界车吗？

> “我最近试驾了一辆讴歌 RDX，真的很喜欢它，我的心脏是蓝色的外观(fathom blue pearl ),内饰是棕褐色或深棕色，令人震惊的是，他们只提供了我讨厌的灰色内饰。我读了你的旧文章，本田一般不做定制，但你听说过这是否延伸到讴歌？我直接给该公司发了电子邮件，但没有收到回复。”

当谈到像本田/讴歌和丰田/雷克萨斯这样的大型日本品牌时，他们并不真的像欧洲汽车制造商那样做“订单”。这些汽车制造商中的大多数都试图尽可能提高生产效率，而完成工厂订单会让这一过程稍微慢一些。

除了 NSX，这是在自己的特殊设施，主流讴歌车型不能特别订购。所以如果你想要蓝色的外表，你就只能选择灰色的内饰。然而，我要说的是，即使你要订购一辆欧洲车，你通常也会受限于配置程序所允许的颜色组合。当然，如果你花重金购买保时捷 911 Turbo S，那就另当别论了。

最后，从道德角度销售有问题的汽车的最佳方式是什么？

> “我们希望用一款类似的 SUV 取代我们的 2013 款斯巴鲁傲虎。问题是这是控油环不好的燃油炉之一。随着里程的增加，它逐渐消耗了更多的石油。现在它已经跑了大约 20 万英里，在油灯亮之前我们还有大约 300 英里，我们必须加满油。外观和内部都非常好，所以我们希望它能找到一个好的归宿。唯一的难题是如何出售它，并确保最终用户知道石油的消耗量。
> 
> 如果我们以旧换新，或者在 craigslist 上出售，我们不能保证第一个买家不会回头再多出价两三千美元(并且不透露石油问题)。起初，我想在仪表板上刻上“汽车烧机油——需要新发动机！”因此，任何买家都会确切地知道他们得到了什么。现在我正在考虑往其中一个圆筒里扔一个螺栓，然后转动钥匙。我们希望为车辆的下一个用途保留最大价值，但也希望确信任何购买它的人都不会被卖给一个柠檬。你建议做什么？"

虽然你试图阻止其他人在不知情的情况下购买这辆问题车是高尚的，但你唯一真正的义务是向第一个买家披露这些问题。他们在那里做什么与你无关。对你的销售对象要超级坦诚。

此外，一个经销商将会拍卖这辆车，所以谁知道在它最终被另一个买家买走之前会有多少人会碰它。
