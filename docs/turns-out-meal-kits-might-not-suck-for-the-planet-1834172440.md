# 事实证明，餐盒对地球来说可能并不糟糕

> 原文：<https://gizmodo.com/turns-out-meal-kits-might-not-suck-for-the-planet-1834172440>

在过去的几年里，我看到蓝围裙和 HelloFresh 等公司生产的餐包越来越受欢迎，对此我越来越怀疑。塞满了冰箱贴和独立包装的食材，挂满了从农场到餐桌营销的和 [回收标志](https://www.hellofresh.ca/green/?redirectedFromAccountArea=true) ，这些包装便利的晚餐似乎是企业绿色环保的缩影。每次朋友通过 kit 告诉我他们最近的烹饪冒险时，我都会想，这对地球来说不可能是好事。

事实证明，我对这一趋势的判断可能有点快。

一项关于餐盒环境足迹的首次研究发现，餐盒实际上比在当地杂货店为同一餐购买同样的食材更环保。这里有一些重要的警告要记住，但它基本上归结为食物浪费:因为配料是预先分配的，更直接地提供给消费者，平均而言，餐包似乎导致更少的食物被扔掉。从碳排放的角度来看，这比令人震惊的包装更重要。

餐包正在起飞，仅在美国，这个大约有十年历史的行业现在就价值约 15 亿美元，并且以每年 25%的速度增长。密歇根大学的研究作者在他们的论文中指出，这种激增的受欢迎程度意味着餐包有可能对食品行业产生“变革”。然而，人们对它们的环境足迹知之甚少。

为了填补这一空白，作者从蓝色围裙中选择了五种两人餐——三文鱼、芝士汉堡、鸡肉、意大利面和沙拉——并用工具包和从杂货店购买的相同食材进行准备。对于每种准备，研究人员试图估计这顿饭的总碳足迹，考虑从农业生产到包装到运输到产生多少废物的一切。

总而言之，餐盒是环保的赢家。蓝色围裙一顿饭的平均碳排放量为 6.1 千克，比杂货店版本的 8.1 千克二氧化碳排放量低约 33%。除了芝士汉堡之外，在每一种情况下，套餐都比杂货店的套餐好。(在这种情况下，博士生和主要研究作者布伦特·赫德告诉 Earther，一些试剂盒成分明显比杂货店的成分重，导致更高的排放量。)

根据作者的分析，餐盒表现更好的最大原因是，它们通过预先分配配料平均减少了食物浪费。取消杂货店也有助于废物处理部门，因为，事实证明，杂货店也会把很多食物扔进垃圾桶。

“有一件事真的让我吃惊:我还没有完全意识到超市零售过程对环境的影响，”赫德告诉《地球》杂志。“这包括因库存过多或商店剔除不合格产品而造成的食品损失，也包括超市的运营。”

这项发表在《资源、保护和回收》杂志上的研究还发现，餐盒降低了所谓的“最后一英里”的排放，即从配送中心到厨房的最后一站。因为餐盒是通过多次送货的卡车运送的，每个单独的盒子都体现了最后一英里运输排放的一小部分——这与自己开车去商店为自己的晚餐而不是别人的晚餐挑选食材截然不同。

总的来说，减少浪费、减少最后一英里的排放以及餐盒中的冷藏包(事实证明，这比杂货店中的巨型冷藏系统更环保)抵消了餐盒包装带来的 0.17 千克额外二氧化碳。虽然作者的论文侧重于温室气体，但他们对土地和水的使用以及养分污染进行了类似的分析，赫德说，结果“基本相似”

现在，这里有很多假设——关于你的行为和你从哪里得到食物——这使得研究结果很难缩小到个体水平。根据赫德的说法，作者包括一项“敏感性分析”，其中关键变量，如消费者每周去杂货店多少次或他们扔掉多少食物，在两个极端之间变化，“我们没有发现整体结果有任何真正显著的变化。”尽管如此，主要作者谢莉·米勒告诉《地球》杂志，这些结果“确实趋向于平均水平。”

我问，作为一个走到杂货店，将我的食物残渣堆肥，并试图吃掉我所有剩菜的人，我是否会成为为了地球的利益而改用餐包的候选人。“把你的数据发给我们，”米勒开玩笑说，然后补充说，这项研究与其说是关于个人选择，不如说是关于“意识消费”。

“这项研究的一个关键亮点是有多少家庭食物垃圾，以及这会产生多大的影响，”她说。

事实上，美国农业部 [估计](https://www.usda.gov/oce/foodwaste/faqs.htm) 美国在 2010 年扔掉了大约 1330 亿磅食物——价值 1610 亿美元。另一项研究估计，美国人平均每天浪费大约一磅食物。种植这些被浪费的食物不仅需要能源和资源，而且当它们被填埋时，会释放出甲烷，一种强有力的温室气体。无论你是在农贸市场买食物还是每天晚上叫外卖，我们都有责任。

如果我们消费者能够更加小心，那么我们购买的公司也可以。杂货店的故事可以减少他们的食物损失，转而使用更环保的冷藏方式。

餐盒公司可以为他们的份额控制给自己一点鼓励。但他们还是需要想办法减少那该死的包装。
