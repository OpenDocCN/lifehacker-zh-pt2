# 本周的怪异举是单手举哑铃

> 原文：<https://lifehacker.com/this-weeks-weird-lift-is-the-one-hand-dumbbell-snatch-1839893931>

我很高兴上周你们中有超过 0 人和我一起参加了 [单手硬拉](https://vitals.lifehacker.com/lets-get-weird-and-try-a-one-handed-deadlift-1839717740) 。奇怪但很酷，对吧？这个星期我们又有了一个单手举，这个举过头顶，这让它变得特别的糟糕。这是单手举哑铃。



基本的想法是这样的:你抓住重物，用力把它从地面上扯下来，使它飞到腰部以上，也许更高(当然，你的手还在上面)，然后你俯下身体，用直臂把它接住。

特别感谢评论员 Ash78 为 [指出](https://vitals.lifehacker.com/1839721128) 这个举的哑铃版本比我更喜欢的杠铃版本对很多人来说更容易。再说一次，我只是希望每个人都能体验到新电梯的乐趣，所以电梯越无障碍越好。

(如果你用壶铃健身，或者如果你做 Crossfit，你可能见过壶铃版本的这个升降机。对于壶铃，你必须在半空中翻转铃铛。我总是把它敲进我的前臂，因为我在这个动作的壶铃版本上很烂，尽管我听说如果你用正确的形式做，你就不会有这个问题。哑铃可以解决这个问题，但是如果你喜欢的话，请随意使用壶铃。)
