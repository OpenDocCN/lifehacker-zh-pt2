# 使用这个 Chrome 标志在 Android 上安排下载

> 原文:[https://life hacker . com/schedule-downloads-on-Android-with-this-chrome-flag-184442037](https://lifehacker.com/schedule-downloads-on-android-with-this-chrome-flag-1844442037)

每当我不得不使用我的手机下载一些重要的东西的时候——或者，真的，下载任何东西到我的手机——我总是会考虑这样一个问题:这会不会烧光我的数据计划？“反对”我现在有多需要这个 app(或者游戏，或者视频) ？”

Watch

虽然游戏和应用程序给你选项来延迟你的下载直到你有 wifi 而不是使用你的数据计划通过你的网络浏览器对下载启动做同样的事情更棘手；一般来说，只要你点击一个链接，它们就会自动启动。或者他们会，除非你在 Chrome Android 中激活一个新的设置，允许你也延迟他们。

你现在就可以使用这个设置。Y 你可以在[**Chrome Beta**](https://play.google.com/store/apps/details?id=com.chrome.beta&hl=en_US)或者稍不稳定的 [**Chrome 金丝雀**](https://play.google.com/store/apps/details?id=com.chrome.canary&hl=en_US) **中找到。从下载任一浏览器到你的 Android 设备开始。完成之后，在 Chrome 的地址栏中输入下面的 URL:`chrome://flags/#download-later`**

如果输入太多，你也可以用`chrome://flags`搜索“下载”这个词无论如何，您都需要来找到并启用这个标志:

重启你的浏览器。现在，当你在 Chrome 中下载一个文件(到你的手机存储器)时，你*应该*看到一个提示，询问你是否想现在下载文件，一旦连接到 wifi 网络就开始自动下载，或者安排在未来的某个特定日期和时间下载。 P 阿尔来自 [Techdows](https://techdows.com/2020/07/chrome-android-download-later-feature.html) 的报道，该功能现在有点挑剔——他们报道他们只能让它工作一次。

如果你不想麻烦这个设置或者不能让它工作，有一些应用可以用来复制这个功能，包括 [**高级下载管理器**](https://play.google.com/store/apps/details?id=com.dv.adm&hl=en_US)**和[**IDM**](https://play.google.com/store/apps/details?id=idm.internet.download.manager)。你挑吧！**
