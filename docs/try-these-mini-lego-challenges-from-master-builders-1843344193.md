# 试试这些来自建筑大师的迷你乐高挑战

> 原文：<https://lifehacker.com/try-these-mini-lego-challenges-from-master-builders-1843344193>

你最近的乐高情况如何？在我家里，t he LEGO 在疫情期间增长了如此之多,以至于我不得不在地下室创建额外的展示架，因为我儿子的 LEGO 桌子、梳妆台、书架和窗台不再足以展示他所有的最爱。(我还计划在我的家庭办公室里放一些架子来展示他的霍格沃茨魔法学校和其他哈利波特作品，但我对此不感兴趣。)

Watch

我不是唯一一个现在有很多乐高玩具的无聊孩子的父母，所以加州乐高乐园度假村[的模型建造大师们正在给我们的孩子每周一次的建造挑战。到目前为止，挑战包括:](https://www.legoland.com/llcbuildingchallenge/)

*   [纸杯蛋糕](https://www.legoland.com/california/legoland-california/llcbuildingchallenge/legolands-21st-birthday/)
*   [球体](https://www.legoland.com/california/legoland-california/llcbuildingchallenge/build-a-lego-sphere/)
*   [明星](https://www.legoland.com/california/legoland-california/llcbuildingchallenge/lego-star/)
*   [狗](https://www.legoland.com/california/legoland-california/llcbuildingchallenge/lego-dog/)
*   [海盗](https://www.legoland.com/california/legoland-california/llcbuildingchallenge/lego-dog2/)
*   [滑板](https://www.legoland.com/california/legoland-california/llcbuildingchallenge/legolands-21st-birthday22/)
*   [羊](https://www.legoland.com/california/legoland-california/llcbuildingchallenge/legolands-21st-birthday2/)
*   [花](https://www.legoland.com/california/legoland-california/llcbuildingchallenge/mothers-day/)

( [海盗](https://www.legoland.com/california/legoland-california/llcbuildingchallenge/lego-dog2/) 最牛逼。)

在每一个一到三分钟的视频中，一位建筑大师在开始建造之前向观众展示他们将使用的部件，但这个想法不仅仅是从你的仓库中挖掘那些精确的部件，并按照步骤。相反，孩子们可以找到灵感和技巧，用他们自己的乐高发挥创意，构建他们自己的独特版本。

如果没有别的，看着大师们如此快速自信地建造东西是令人着迷的——没有指导！当你在“建筑挑战”网站上时，请查看乐高发布的其他每周活动，包括[迷宫挑战](https://www.legoland.com/california/legoland-california/llcbuildingchallenge/weeklu-maze-challenge/) 、疯狂的 Libs 风格的 [傻故事](https://www.legoland.com/california/legoland-california/llcbuildingchallenge/weekly-silly-story-challenge/)[更多](https://www.legoland.com/llcbuildingchallenge/) 。
