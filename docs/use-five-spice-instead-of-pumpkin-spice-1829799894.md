# 用五种香料代替南瓜香料

> 原文：<https://lifehacker.com/use-five-spice-instead-of-pumpkin-spice-1829799894>

虽然我并不完全反对南瓜香料，但我承认我对它有点厌倦。我并不反对肉桂、丁香、姜和其他任何 PSL 拿铁咖啡的配料，但我也想活得开心一点，我说的“开心一点”是指“用五香粉给我的馅饼调味”



这种主要用于中国和台湾菜肴的混合香料通常由——惊喜！—五香:八角、丁香、桂皮、花椒、茴香。它很温暖，有一点咸味，还有一点臭味，它让你洒在上面的所有东西都变得更有趣。虽然它是肉类摩擦和腌泡汁中的天然成分——尤其是鸭肉和火鸡——但在饮料和甜点中却令人惊喜。

对于鸡尾酒来说，它特别适用于以秋季水果 为特色的 [饮料，如苹果和梨，以及以](https://www.foodandwine.com/recipes/palisades-pear-fizz-cocktails-2011) [波旁](https://www.seriouseats.com/recipes/2015/06/five-spice-bourbon-punch-summer-pitcher-drink-whiskey.html) 和 [朗姆酒为基础的](https://www.liquor.com/recipes/chinese-5-spiced-dark-n-stormy/) 混合物。你可以在 1:1 的 Demerara 简单糖浆中加入一汤匙来制作糖浆，或者你可以直接将它注入烈酒中，就像波特兰的杰弗里·摩根塔勒 [在这里为他的黑暗风暴](https://www.liquor.com/recipes/chinese-5-spiced-dark-n-stormy/) 所做的那样。(只需将一茶匙香料加入一瓶 750 毫升的酒里，摇匀，静置一夜，然后第二天用咖啡过滤器过滤。)

对于馅饼，使用它就像你会任何秋季香料混合。如果你的南瓜派需要一茶匙半的南瓜派香料，试试一茶匙半的中国五香粉。(如果不想完全投入，可以尝试一半一半。)在 [苹果](https://www.epicurious.com/recipes/food/views/five-spice-apple-pie-102113)[木瓜](https://www.epicurious.com/recipes/food/views/five-spice-apple-quince-pie-109093)[梨](https://www.seriouseats.com/recipes/2017/10/pear-galette-recipe.html) 派里也特别好看。如果你想以一种更渐进的方式将这种混合物引入烘焙，你可以通过将半茶匙粉末和一汤匙蜂蜜搅拌在半杯奶油中来给浇头调味，而不是给馅料调味。
