# 不要在人行道上用这么多盐

> 原文：<https://lifehacker.com/stop-using-so-much-salt-on-sidewalks-1832588841>

冬天已经全面来临，随之而来的是，盐的问题充斥着我们的每一条人行道和街道，当然，除非你生活在一个季节差异几乎不存在的州(我很嫉妒)。



虽然盐通过融化雪帮助我们避免堆积，但盐经常污染我们的水源，破坏我们的环境，并影响我们自己的健康。人行道上的盐是如何进入我们身体的？

首先，你通常在路上看到的盐和你在人行道和街角看到的盐是有区别的，尽管它们可以互换使用。在道路上，一般使用 [岩盐](https://dengarden.com/misc/What-is-the-Difference-Between-Rock-Salt-and-Ice-Melt) 。它很便宜，成分和食盐一样，就像你晚餐用的那种。它有助于在道路上提供牵引力，但被认为对植物和动物(包括狗)更有害！).

然而，融化的冰是人行道上常见的颗粒状盐。它被认为对宠物更友好(虽然仍然有害，但能在低于 5 度的温度下工作，不像 T2 岩盐，所以它能更好地抵御冬天最糟糕的天气。它还被认为比岩盐腐蚀性小，所以它更经常在家庭附近使用。

但它们都是盐的形式，当它们渗入人行道或街道或被径流带走时，它们就会进入我们的地下水或河流，许多美国社区从那里获得饮用水。最近的一项 [研究](https://www.pnas.org/content/115/4/E574.full) 发现，美国 37%的排水区域，即雨水和融雪与溪流和河流交汇的区域，正在变得更咸。在东北地区，道路盐是罪魁祸首。

最坏的情况会是什么？嗯，当你喝更咸的水时，它有可能会使你的血压升高。对于已经患有高血压的数百万美国人来说，这是一个大问题。

2008 年的一项研究发现，在纽约州公爵夫人县 [、T3](https://blogs.ei.columbia.edu/2018/12/11/road-salt-harms-environment/)测试的 48%的水井超过了环保局对水中钠含量的限制，20%的水井含盐量足以危及当地高血压居民的健康。

盐还会威胁水生生物和整个水体，因为含盐太多会耗尽氧气。更糟糕的是，盐并没有完全消失，而是在水体中积累。也没有一种经济有效的方法来去除它(它溶解在水中，所以不再可见，加剧了问题)。

那么你能做些什么来帮忙呢？

首先，如果你要用盐，不要过量。将它均匀地铺在一个表面上，而不是堆成一堆，这样你需要的就少了(这样也更有效)。更经常地铲也将有助于抑制你对盐的需求。暴风雨过后，把多余的盐扫掉，这样它就不会再进入你的饮用水中了。
