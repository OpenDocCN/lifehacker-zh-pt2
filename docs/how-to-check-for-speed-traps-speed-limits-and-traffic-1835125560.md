# 如何在谷歌地图中检查速度陷阱、限速和交通事故

> 原文：<https://lifehacker.com/how-to-check-for-speed-traps-speed-limits-and-traffic-1835125560>

从今天开始，谷歌地图正在将其交通功能扩展到更多地区。该应用程序现在将显示您当前所在道路的速度限制，以及 40 多个国家的各种关键交通信息的位置，如事故、移动速度跟踪器和交通摄像头。

Watch

在导航模式下，速度限制显示在谷歌地图的左下角，如果前方有报告的事故、交通减速或速度陷阱，应用程序将口头提醒你。以下是用户如何在应用程序中手动添加限速标志的位置或其他交通信息:

1.  在谷歌地图中，通过输入地址并点击方向，或者点击谷歌地图主屏幕上的蓝色“Go”按钮并设置目的地，然后点击“Start”来开始导航模式。
2.  在导航模式下，点击“+”语音气泡图标(右侧第三个图标)。
3.  选择您希望报告的类型(事故、超速或减速)。
4.  应用程序将开始向地图添加报告，但如果您想取消，会有一个短暂的倒计时窗口来撤消报告。

这些功能在其他导航服务中已经存在很长时间了，如谷歌拥有的 Waze 应用程序、，但在谷歌地图中仅限于某些地区。此新更新将所有功能扩展到美国和以下国家的所有地区:安道尔、澳大利亚、巴西、波斯尼亚和黑塞哥维那、保加利亚、加拿大、克罗地亚、捷克、爱沙尼亚、芬兰、希腊、匈牙利、冰岛、印度、以色列、意大利、日本、约旦、科威特、拉脱维亚、立陶宛、马耳他、墨西哥、摩洛哥、纳米比亚、荷兰、挪威、阿曼、波兰、葡萄牙、卡塔尔、罗马尼亚、俄罗斯、沙特阿拉伯、塞尔维亚、斯洛伐克、南非、西班牙、瑞典、突尼斯、英国和津巴布韦。

这些新功能都是在服务器端添加的，这意味着用户不需要更新应用程序或更改设置就可以启用它们——它们将直接出现在谷歌地图中，随时可用。也就是说，推广可能不会在所有地方都完成，所以如果你还不能使用新的交通功能，请继续检查应用程序。当然，如果你想在路上查看地图或添加信息，不要在开车时尝试这样做——将这项任务委托给乘客座位上的任何人。
