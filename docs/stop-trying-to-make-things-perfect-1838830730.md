# 不要试图让事情变得“完美”

> 原文：<https://lifehacker.com/stop-trying-to-make-things-perfect-1838830730>

当我第一次开始写作时，像这样一篇 300-500 字的短篇小说要花我几天时间才能写完。我出版了一些东西给人们阅读，我想确保每一个字都是完美的。



说实话，如果我把我在写故事的第一个小时写的东西和经过一天修改的成品交给某人，他们可能不会注意到巨大的差异。事实上，在很多情况下，1 小时的版本可能比 8 小时的版本更好。

虽然有些时候“完美”是你应该努力争取的，但本周*的一篇文章强调了这样一个事实，有些事情最好努力争取“大部分是好的决定”，也就是 MFD 的。*

*这个概念很简单:不断追求完美会让你慢下来。用“好”来代替，你降低了你的期望，帮助你前进，更有效率。*

*同样，这不是你应该在生活的所有方面都应用的东西，但是如果有一些事情你经常陷入困境或者因为害怕失败而一直避免去做，追求好而不是完美可以帮助你克服它。*

*这里的想法是做一个有助于推动你前进的决定。对于那些花了我一天时间的故事，我应该看着它们，思考它们需要什么才能达到“完美”的状态。如果没有好的答案，那么是时候提交了。*

*同样的道理也适用于预订机票、整理房间或你能想到的几乎任何其他任务。如果你不能给出一个具体的答案，为什么有些东西现在还不完美，那么考虑满足于接近完美，继续前进。*

*追求“好”可以成为一个可靠的拖延杀手，让你更有效率，也许还能消除一点压力。*
