# 是的，你可以在 Android 设备上使用 AirPods(但你会失去一些功能)

> 原文：<https://lifehacker.com/yes-you-can-use-airpods-on-android-devices-but-youll-1841493140>

仅仅因为你能，并不意味着你应该。在上面的视频中，我讲述了如何将 AirPods 与 Android 设备配对，以及为什么这可能不是最好的主意。

当你将苹果 AirPods 与安卓设备配对时，它们就像任何其他蓝牙耳机一样工作。就连配对过程都是一样的。但是你最终会失去一些依赖于 iOS 的功能，这可能是一个障碍。

开箱后，你不能仅仅通过打开盖子并将它们放在 iPhone 附近来立即配对 AirPods。你必须经历通常的过程，按住外壳上的按钮，直到 Airpods 处于配对模式(灯将开始闪烁)，然后从 Android 设备上的蓝牙设置连接到它们。没什么大不了的，因为你可能只会做一次。

你将失去的另一个功能是轻松召唤语音助手的能力。AirPods 不支持开箱即用的谷歌助手，而且由于 Android 设备上没有 Siri，你将无法说“嘿 Siri”来使用语音命令。

AirPods 最好的功能之一就是自动耳朵检测。当你把一个耳机从耳朵里拿出来时，你正在听的任何东西都会自动暂停。一旦你把气垫放回耳朵，它就会恢复。这项功能在 Android 上无法使用。

另一个重要的功能是检查 AirPods 和外壳的电池电量。 [有一些变通办法](https://www.cnet.com/how-to/how-to-check-airpod-battery-stats-on-android/) ，但它们并不完美。从好的方面来看，如果你有 AirPods Pro，噪音消除和触摸控制将像在 iPhone 上一样工作。

最终，在 Android 设备上使用 AirPods 的决定取决于这些功能对你有多重要。AirPods 显然是为苹果自己的设备设计的。像让你快速打开和关闭噪音消除的用户界面，或者在多个苹果设备之间轻松切换而不必重新配对的能力这样的小事情，是增加整体体验的生活质量功能。

除非你真的喜欢 AirPods，否则作为安卓用户，你最好给自己买一副更好的蓝牙耳机。 [有很多绝佳的选择。](https://gizmodo.com/the-best-wireless-headphones-you-can-buy-right-now-1830336665)
