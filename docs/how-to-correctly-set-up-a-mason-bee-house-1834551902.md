# 如何正确设置泥瓦匠蜂房

> 原文：<https://lifehacker.com/how-to-correctly-set-up-a-mason-bee-house-1834551902>

梅森蜜蜂非常棒:它们温顺，容易饲养，是春天开花的水果和坚果树的神奇授粉者。这些蜜蜂不像蜜蜂那样使用蜂巢 ，而是喜欢把它们的卵放在狭窄的洞里，用泥堵住(因此得名“泥瓦匠”)。

它们温和的天性和喜欢独居的习性使梅森蜜蜂成为一个很好的“养”在你院子里的物种。也就是说，如果你做对了。如果你做得不对，你对他们的伤害可能会超过你对他们的帮助。

前进化生物学教授科林·普林顿几天前在推特 上告诉我们，我们的良好愿望是如何偏离的。

### 每年秋天打扫他们的房子

普林顿指出，一些商店购买的梅森蜜蜂屋的最大问题是，砖块和芦苇被粘在房子的后面。这意味着你不能在每年春天添加新的筑巢材料，这大大增加了寄生虫和真菌的风险。

你会在这里找到一些关于梅森蜜蜂 全年护理的 [大信息，包括储存巢管和巢块以及收获茧。](https://onegreenworld.com/mason-bee-care/)

### **位置，位置，位置**

梅森蜜蜂屋应该放置在一个平坦的表面上，远离大风，离地面大约 6 英尺，如果可能的话，朝南。不要用树枝上的细绳悬挂石匠的房子；允许鸡蛋在每一次经过的微风中被撞来撞去是没有帮助的。

它们还需要离产花粉的植物足够近(它们不会移动超过 300 英尺)，以及充足的粘土状泥土来掩盖它们的巢穴。

### 给他们一点空间

一个好的梅森蜂房的屋顶会有一点突出，以保护洞免受雨水的影响，并降低幼虫和蛹在巢内腐烂的风险。
