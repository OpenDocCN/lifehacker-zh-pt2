# 只在工作日看电影可以节省订购 Sinemia 的费用

> 原文：<https://lifehacker.com/save-money-on-a-sinemia-subscription-by-only-seeing-mov-1830270204>

MoviePass 的竞争对手 [Sinemia](https://www.sinemia.com/) 本周宣布了新的折扣订阅计划，面向那些只愿意在工作日使用该服务看电影的人。工作日套餐比该公司的标准价格低 20%左右，起价为 3.99 美元。他们只能在星期一到星期四使用。



如果你不熟悉 Sinemia，这项服务的运作方式与 MoviePass 类似，你每月支付固定的费用，以便观看预定数量的电影。与 MoviePass 不同的是，使用 Sinemia，你可以通过任何你喜欢的售票网站提前购买这些门票并选择座位。也有家庭计划可用。和许多电影院一样，当你预订票的时候，你还必须支付大约 1 美元的信用卡手续费。

我订阅了一项服务，那是我通过一个单独的推广活动 n 购买的——每月花费 9 美元，我和我的男朋友可以一起去看一场电影，我们每个月可以得到一张价值 25 美元的[REStaurants.com](http://rstaurants.com)礼券。我专门使用我的 Sinemia 订阅在我们当地的阿拉莫选秀屋购买门票，我也在那里获得奖励，所以虽然我还没有使用那些[restaurants.com](http://restaurants.com)证书中的任何一个，但我*已经在路上获得了*一两个免费爆米花。

以下是新的仅工作日订阅计划:

每月 3.99 美元(而不是 4.99 美元的标准价格)——每月一张电影票。

每月 8.99 美元(而不是 10.99 美元的标准价格)——每月三张电影票。

每月 9.99 美元(而不是 12.99 美元的标准价格)——每月两张电影票，包括 3D、4D 和 IMAX 格式。

每月 14.99 美元(而不是 17.99 美元的标准价格)——每月三张任何电影的门票，包括 3D、4D、IMAX 格式。

每月 23.99 美元(而不是 29.99 美元的标准价格)——每天一张 2D 电影票

这些促销通常不会持续太久，所以如果 T2 觉得这是个好主意，不要等太久。
