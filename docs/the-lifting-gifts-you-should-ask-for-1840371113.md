# 你应该要求的提升礼物

> 原文：<https://lifehacker.com/the-lifting-gifts-you-should-ask-for-1840371113>

读者可能已经注意到，在过去的一两年里，我真的迷上了举重。一路上，我买了一些有点贵的东西。如果我聪明的话，我会要求把它们作为生日或节日礼物，但是去年的这个时候，我不太确定我最终会需要什么。这是给过去的我的礼物指南，他可能也是你。



记住， [健身礼物可能会被视为侮辱](https://lifehacker.com/how-to-give-a-fitness-gift-thats-not-insulting-1840318595) ，但如果你要求一件对你真正有用的特定物品，你的送礼者就可以免费获得。因此，如果你有资格提出一个具体的要求，或者如果你希望得到一些可以凑齐的现金礼物，以下是你可能想要的。

### 一条腰带

如果你喜欢举重，或者任何最终需要做大量举重运动(包括深蹲和硬拉)的力量运动，你最终会 [想买一条腰带](https://lifehacker.com/how-to-choose-your-first-weightlifting-belt-1774624026) 。腰带并不能保护你的背部，但是它 [确实有助于稳定你的躯干](https://lifehacker.com/when-weightlifting-belts-are-necessary-and-when-theyre-1766557417) 这样你就可以举得更高，让你的腿得到更好的锻炼。

在举重比赛中，每个人通常都系一条皮革或人造革腰带，四周宽度相同，前面有一个带扣或杠杆。在奥运会举重项目中，你更可能看到带有 velcro 或衬垫的腰带。如果你打算参加比赛，检查你的联合会的要求，以确保你选择的腰带是合法的。例如，它的大小可能会有限制。

对于力量举重，你必须决定:

*   厚度，以毫米为单位。10 毫米对于几乎每个人来说都足够了，但是如果你想要非常结实的东西，也有 13 毫米的腰带。
*   宽度，以英寸为单位。很多皮带都是 4 英寸宽，但是除非你很高，否则你会发现 3 英寸的皮带更适合硬拉。
*   它关闭的方式。单叉带扣易于佩戴和调节。杠杆式安全带不能随时调整，但是一旦你有了你想要的，就很容易穿上。据我所知，双叉带扣有点让人头疼。

一些受欢迎的皮带品牌需要定制订单，所以要提前考虑。如果你能在订购一条腰带之前先试一试，那是最好的，所以问问你的健身房，看看人们会推荐什么，以及你是否能试一试他们的，看看感觉如何。

我最终得到了 [这条腰带](https://generalleathercraft.com/product/pioneer-cut-10mm-thick-4-power-lifting-belt/)10 毫米厚，3 英寸宽，单尖头，栗色带白色缝线。我喜欢它每半英寸有一个洞，让它更容易调节。(每英寸都有洞更常见。)

### 提升鞋

也许你仍然穿着跑鞋或多功能运动鞋去健身房。如果是这样，现在是升级的好时机。

我们有一个 [详尽的指南](https://lifehacker.com/what-shoes-to-wear-when-you-lift-weights-1837994384) 来介绍主要类型的登山鞋，但是如果可以的话，你还是会想试穿一些鞋子，并且想清楚你到底想从你的新鞋中得到什么。具体来说:

*   你想要高跟的吗？如果你参加奥运会举重比赛，或者如果你努力深蹲，那就是肯定的。买合适的举重鞋。如果不是，选择高跟还是平跟由你决定。

*   你想要薄而平的东西吗？卡盘、货车和其他平底鞋非常适合硬拉。也有人喜欢平底鞋蹲着，有人不喜欢。
*   你打算做的不仅仅是穿上新鞋抬举吗？有几个型号，像 [Metcons](https://www.roguefitness.com/shoes/training-shoes/nike) ，是全功能训练器和增高鞋的混合体。例如，如果你进行交叉训练，你可能需要一点鞋跟来帮助深蹲和推进器，但不需要在需要你跑步或爬绳的 WOD 中换下你的提升鞋。

如果你只做奥运会举重，举重运动员是你所需要的。如果你做交叉配合，像 Metcon 这样的东西会覆盖你的大部分基地。如果你是一名强力举重运动员，你可能会想从一双吸盘开始，并考虑把提升鞋作为你的下一次升级。

我自己呢，一开始是一双全黑的 hi-top 卡盘，然后加了一双 Nike Romaleo 举重鞋。

### 完美的运动包

这里有一个礼物，仍然是个人的，但不需要你的运动的具体知识。想象出你完美的运动包，然后开始给给你买礼物的人暗示。几乎每种尺寸、拉链和皮带的配置都在某个地方。需要考虑几个问题:

*   你会带鞋子吗？湿衣服还是脏衣服？有些包有口袋，可以和你的其他装备分开。
*   你打算带着你的运动包去旅行吗？如果是这样的话，你可能会想把它放进随身行李或者座位下的私人物品空间。
*   哪些口袋必不可少？我喜欢有一个专门放我的 [训练日志](https://vitals.lifehacker.com/why-you-still-need-a-workout-notebook-1834590216) 和其他论文的口袋。也许你想要多个小拉链口袋来装钥匙和其他小物品。
*   什么包会放在你的包里？如果你没有鞋柜，你可能需要一个鞋包。如果你在健身房淋浴，你可能需要一套洗漱用品。如果你的包感觉太大，可以考虑用 [包装立方体](https://lifehacker.com/how-packing-cubes-can-change-your-life-1833717212) 来存放物品。

我有 [阿迪达斯后卫三](https://www.amazon.com/adidas-Defender-III-Duffel-Large/dp/B07P7S5NT7?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-lifting-gifts-you-should-ask-for-1840371113&asc_source=&tag=kinjalifehackerlink-20) 。我用其中一个口袋放鞋子，另一个放笔记本。我还准备了打包盒和鞋袋，在旅行时用来装多余的鞋子，尽管它们不是我日常装备的一部分。

### 好健身房的会员资格

作为一个初学者，我猜你是从离家或工作地点最近的健身房开始锻炼的。但是随着时间的推移，你的兴趣变得更加专业，你可能会发现自己想要更多。

四处问问，试着找出你最有家的感觉的地方:一个有很多平台的举重场所？一个举重馆，在那里你可以尽情地使用粉笔，尽情地举重？一个有好教练的健身箱？一般来说，每个健身房都会很乐意给你免费的第一次锻炼——或者至少让你买一张一天的通行证——这样你就可以在那里锻炼，看看你有什么想法。

别忘了问问朋友们他们在哪里健身，他们希望在哪里健身，以及他们认为哪些教练和培训师很棒。

就我个人而言，我牢记我们的经典建议——你的健身房应该要么超级近，要么超级好——我属于这两种情况。我家附近有一家商业健身房，价格便宜，不会超出预算太多，这让我也可以加入镇上另一个地方的举重健身房。

或者再举一个例子:我自己并不在举重馆锻炼，但我认识一些这样做的人，我接受了他们的建议，在我丈夫的热情允许下，在那里给了他一套教练课程。记住，送礼的人应该确切知道受礼人想要什么。如果你是礼物的接受者，不要害怕要求你所需要的。
