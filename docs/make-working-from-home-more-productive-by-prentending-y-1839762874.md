# 假装你在办公室，让在家工作更有效率

> 原文：<https://lifehacker.com/make-working-from-home-more-productive-by-prentending-y-1839762874>

在过去十年的大部分时间里，我都是在家工作。在家工作很棒，但也会让人分心。一件让事情变得简单的事？假装你去了办公室。



通过让自己处于办公室的心态，即使你在家里，你也可以比在家里工作时更有效率。

要实现这一点，你可以做很多不同的事情。

## 交换

虽然一整天都穿着睡衣很诱人，但是每一篇关于在家工作时保持高效的文章都建议无论如何都要穿好去办公室的衣服。

我并不总是这样做，但我确实会脱掉睡衣。我还真的建议早上第一件事就是去散步或跑步，这能让你在工作开始前走出家门 15-30 分钟。对我来说，典型的是遛狗或沿街散步吃早餐。当你把睡眠时间和工作时间分开，并且能够“去工作”时，你会以更好的心态开始做事。

## 有专门的办公空间

你不需要一整间办公室(尽管那会很棒)，但是给自己一个每天都努力工作的专用空间，最好远离你下班后与家人共度时光的地方。

我在我们公寓的一间备用卧室里工作，但你的空间可以是厨房里的一个早餐角，也可以是客厅的一个角落。想一个你家里没有很多人流量的地方，看看你是否可以把它改造成你的办公空间。

## 假装你不在家

一旦你在早上进入办公室，最重要的是假装你在一个真实的办公室里。如果你在工作，你就不会洗衣服，洗碗，或者为营销人员开门，你会吗？总的来说，表现得像在办公室一样会对你的工作效率产生很大的影响。

如果你和别人住在一起，让他们知道你的工作空间，并要求当你在里面时不要被打扰。你也想试着避免和没有工作的朋友一起吃午餐之类的。你的工作时间就是工作时间，不要因为你在家就让别人占用你的时间。

当我第一次开始在家工作时，我被朋友们要求去机场接机、照看孩子以及几乎任何你能想象到的事情轰炸，他们以为因为我在家，我就可以“自由”做事。虽然在家工作可以很好地安排医生的预约和偶尔的午餐，但不要让你的朋友把你的工作日当成只是在家的时间。

虽然小要求看起来很小，甚至是可能的，但它们会迅速滚雪球般地占据你一天的工作，让你的工作效率大大降低。

想想:如果我在办公室工作，我会这样做吗？如果答案是否定的，那么你也许应该跳过它。
