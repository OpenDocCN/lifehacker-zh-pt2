# 浸泡并冷冻竹串，用于夏季烧烤

> 原文：<https://lifehacker.com/soak-and-freeze-bamboo-skewers-for-summer-grilling-1843755279>

在夏天的几个月里，吃棍子上的食物特别有趣，在烤架上扔一串烤肉串是实现这一点的简单方法。但是竹子做的串肉扦会裂开或者烧焦，烧焦的碎片也不开胃。稍微浸泡一下会有帮助，但是你必须记住浸泡它们。为了防止尴尬的骨头- 烤串时刻，只需提前浸泡一串并冷冻即可。



将竹签浸泡在温水中有两个好处:使它们更柔韧，有助于减少烧焦。我说“切下来”，因为即使浸泡过的串肉扦也会在超级热的烤架上烤焦，但是在顶端放一块肉或一点 T2 蔬菜可以防止至少一面烤焦。(你也可以在“手柄”端裹上一层箔片，不过如果裹得不够紧，箔片可能会脱落。)但是潮湿的串肉扦肯定更柔韧，在温水中浸泡 30 分钟可以防止你的棍子裂开。

你能买金属烤串跳过这一切吗？是的，但是我发现很难买到足够的金属串，特别是如果我要为一群人做很多串。(还记得聚会吗？)

准备一串肉串以备后用，在一个浅锅或盘子里装满温水，让这些婴儿浸泡半个小时。把它们从锅里捞出来，抖掉多余的水，放入一个一加仑大小的冷冻袋里。把袋子放在冰箱里，在整个烧烤季节需要的时候抓住串肉扦。
