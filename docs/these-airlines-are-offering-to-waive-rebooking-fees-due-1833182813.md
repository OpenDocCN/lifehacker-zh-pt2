# 由于冬季风暴泰勒，这些航空公司提供免除重新预订费用

> 原文：<https://lifehacker.com/these-airlines-are-offering-to-waive-rebooking-fees-due-1833182813>

一场冬季风暴将于本周末席卷中西部。如果你有一架飞越美国中部的飞机，你很可能是安全的，但如果你的旅行计划涉及离开或前往一些中西部目的地或在这些城市中的任何一个停留，你可能会经历一段相当艰难的时间。

Watch

鉴于这一问题，一些主要的航空公司提供免除机票改签费。也就是说，你可以将你的机票换成以后的机票，或者换成途经另一个城镇的机票，并且只需支付你当前机票和你选择的新机票之间的差价。

目前，美国航空公司、达美航空公司、西南航空公司、联合航空公司、捷蓝航空公司和太阳县航空公司都因风暴而提供重新预订的优惠。如果你有其中一个航班，并愿意改变你的旅行计划，这可能不是最糟糕的主意。

大多数航空公司对往返或途经中西部大部分地区的航班免收重新预订费，这些地区包括威斯康星州、北达科他州、南达科他州、明尼苏达州、密歇根州和爱荷华州。
