# 什么时候“现在”不是投资的最佳时机

> 原文：<https://lifehacker.com/when-now-isnt-the-best-time-to-invest-1833092120>

问一个金融专家投资的“最佳”时间是什么时候，他们会告诉你:昨天。假设你昨天没有开始，下一个最佳答案是*今天*。



这是因为复利和市场中的时间(随着时间的推移，股票市场一直呈上升趋势，即使偶尔有挫折)是积累财富的一些最重要的因素，机器人管理员 [Ellevest](https://www.ellevest.com/magazine/investing/when-invest) 写道。你在市场上有越多的时间，它就有越多的时间增长(并从偶尔的挫折中恢复)。

也就是说，在一些情况下，把钱投到其他地方可能更为明智:“当你仍在努力首先把你的财务基础准备好的时候，”Ellevest 写道。

### 当你需要建立应急基金时

如果你没有一些现金以备不时之需，这应该是你的首要任务。“这种安全性很重要，因为金融突发事件有时几乎肯定会发生，”Ellevest 写道。而且你的投资是不流动的，这意味着你不能在不招致某种惩罚或税收的情况下利用它们来支付紧急费用(除非你取出 [*供款*给罗斯](https://twocents.lifehacker.com/why-you-need-a-roth-ira-1823805955) )。

我要补充的是，如果你担心自己会失业，或者另一个重大的、昂贵的生活变化即将到来，那就值得考虑稍微降低你的自动缴款比例，以建立你的储蓄账户(如果你还没有这样做的话)。如果你丢了工作，又没有马上的替代者，你不想成为没有存款的。

### 当你不得不偿还债务时

偿还债务永远是与投资相匹配的最棘手的财务目标之一。例如，如果你有很多校外学生贷款债务，你可能会觉得这是你最紧迫的优先事项，投资应该(也必须)让位于偿还这些债务。这是可以理解的，但这也意味着你可能会错过多年的增长。

通常的建议是 [还债*同时投资*](https://twocents.lifehacker.com/how-to-pay-off-student-loan-debt-while-still-saving-and-1821805416)*，没有更多的指导。Ellevest 对此的看法略有不同。“还清利率高于 5%的债务，”报告建议，“从历史上看，这可能会让你付出比投资所得更多的代价。”*

*所以，举例来说，如果你有信用卡债务，优先考虑把更多的钱用于偿还，而不是全部投资。这会有更高的回报(因为你的债务也是复合的)。*
