# 尝试“何时/然后”技巧，让孩子遵循指示

> 原文：<https://lifehacker.com/try-the-when-then-trick-to-get-kids-to-follow-direction-1841599620>

有时候，偶尔，每隔一段时间，一个孩子就是不想顺从。他们需要去拿鞋吗？ *Nah* 。洗澡时间到了？*谢谢但不用谢*。你想让他们把玩具收起来？*不感兴趣*。这就是你使用“何时/然后”技巧的时候。



安吉拉·伊利亚为 [PopSugar](https://www.popsugar.com/family/avoid-power-struggles-with-toddlers-using-this-expert-hack-47161730) 写道，她从艾米·麦克里迪的 [积极育儿解决方案](https://www.positiveparentingsolutions.com/parenting/simple-words-to-avoid-power-struggles) 课程中学到了这个方法，内容如下:

> 比方说，我让我的儿子捡起他的玩具或洗手，他无视我的要求，为他不能做这件事提供借口，或者断然拒绝。我不再反复要求他做这件事，并最终以“你最好做，否则”的威胁让我失去冷静，现在我平静地解释说，当他做了不受欢迎的任务，*然后*他可以做他期待的事情。然后我走开。
> 
> 例如，我可能会说*当*你拿起你的玩具，*然后*你可以看你的表演。*当*你洗手的时候，*然后*你就可以吃零食了。

要做到这一点，“然后”必须是他们想要的东西。*当*你穿上睡衣，*然后*你就可以睡觉了，“不会让你有结果的。但是，“*当*你穿上外套，*然后*我们可以去游乐场，”应该会激发一些兴趣。

Elias 报告说，她第一次对她的儿子使用这种方法时，他花了一段时间才明白，他是负责得到他想要的东西(零食)的人——他只是必须遵守她为他规定的条件(洗手)。但她保持冷静，避免被卷入权力斗争，最终，他洗了手。

when/then 短语听起来也没有 if/then 那么贿赂。*如果*你把玩具收起来，*那么*你可以看一场听起来像是你在购买他们遵守屏幕时间的表演。When/then 更多的是关于用你允许的特权来控制事件的顺序和流程。
