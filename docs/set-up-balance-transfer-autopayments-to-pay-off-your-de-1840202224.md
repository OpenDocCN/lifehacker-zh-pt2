# 设置余额转账自动付款，在 0%的介绍年利率到期前还清债务

> 原文：<https://lifehacker.com/set-up-balance-transfer-autopayments-to-pay-off-your-de-1840202224>

如果你有很多信用卡债务，并且每月的利息费用在增加，余额转移信用卡可以是一个不用支付利息就能还清债务的好方法。这些卡给你一个免息宽限期，在此期间你可以全额付清你的余额，以此换取一笔小额的余额转账费(通常是你转账余额的 3%到 5%)。



然而，告诉自己“最终”会还清欠款是很容易的如果你想确保在 0%的介绍期结束前还清欠款，是时候做些计算了——并设置一些信用卡自动付款。

基本上，你想弄清楚在 0%的介绍期结束之前，你需要支付多少钱来清理你的余额。然后，你要设置每月自动付款来支付这笔金额。正如 [美国消费者新闻与商业频道](https://www.cnbc.com/select/what-to-do-once-your-balance-transfer-is-complete/) 所解释的:

> 例如，如果您将 3000 美元的余额转到一张卡上，15 个月不计息，只需将您的 3000 美元余额除以您的介绍期长度(15 个月)。你会发现你需要每月支付 200 美元，以确保在介绍期结束前你的余额为 0 美元，假设你没有在卡上进行任何新的收费。

如果你计算了一下数字，发现你无法支付每月 200 美元(或者不管你的计算结果是多少)，在 0%的介绍期到期前尽可能多的付清你的余额仍然是一个好主意。毕竟，当免息宽限期结束时，你的余额越低，你需要为卡上的余额支付的利息就越少。

关于“卡上还剩下什么”的问题:“确保你在偿还旧的余额时不要积累一堆新的信用卡债务，因为那样你就会回到你开始的地方。一些余额转移信用卡让 0%的介绍期延长到你转移的余额和你在卡上的任何新的购买——这可以鼓励消费者进行一系列额外的收费，因为，嘿，他们是无息的！

如果是你，确保调整那些仔细计算过的自动付款，清除你的旧余额和新余额。
