# Windows 10 版本 1909 中的最佳功能

> 原文：<https://lifehacker.com/the-best-features-coming-in-windows-10-version-1909-1838851768>

与 Windows 10 全年较小的例行更新不同，微软在添加新功能时，会对该操作系统进行两次重大年度检修——一次在春季，另一次在秋季。在测试阶段度过了最后几周之后，2019 年秋季更新，又称 Windows 10 1909(之前是 Windows Insiders 的 19H2)，终于向公众开放。



然而，Windows 10 版本 1909 在范围上比以前的同类更新更小。Windows 10 1909 没有对用户界面或核心功能进行巨大修改，而是用新功能增强了几个预先存在的应用程序和工具，例如管理应用程序通知的附加选项，更简单的日历编辑快捷方式，并为 Windows 10 上更好的第三方人工智能助手集成铺平了道路——但这只是用户可以预期的各种更新的一小部分。

以下是 Windows 10 版本 1909 的所有主要变化。

### **新的通知设置和快捷方式**

Windows 1909 的大部分变化是关于操作系统如何让用户管理他们的系统和应用程序通知，包括:

*   操作中心中新的“管理通知”快捷方式(通过单击 Windows 任务栏右下角的小对话框图标打开)。
*   单个系统通知现在将包括一个可点击的链接，直接带您到特定应用程序的通知设置。

*   通知设置页面中的新选项，用于单独或同时静音应用程序。
*   在“设置”菜单中也有针对特定通知类型的新图标。

### **Windows 10 日历的新事件快速创建工具**

Windows 10 1909 的另一个重大变化是对 Windows 日历应用的改变。现在，您可以直接将事件添加到快速访问日历，而无需打开主日历应用程序。方法如下:

1.  单击任务栏中的日期/时间，打开快速访问日历视图。
2.  单击日期开始添加活动。快速创建工具可让您命名和标记事件，并添加时间和位置。

快速访问日历中还添加了打开主日历应用程序的新快捷方式。

### **Windows 10 版本 1909 的其他变化**

除了方便的通知和日历快捷方式，用户在安装 1909 版更新后还会注意到一些其他有用的更改:

*   文件资源管理器中的搜索工具已经更新，现在是基于在线的，这意味着它现在将在搜索结果中包括 OneDrive 文件。它还会执行快速搜索，在搜索栏下方的下拉菜单中显示潜在的匹配项(不过你仍然可以通过简单地按下**“Enter”来执行经典的完整搜索)**)
*   将鼠标悬停在开始菜单的边栏快捷方式(如设置图标)上会显示更多有用的工具提示文本。
*   你现在可以从锁定屏幕上语音控制第三方人工智能助手——或者至少一旦第三方助手启用这一功能，这将成为可能，尽管有传言称 Alexa 很快会添加支持。
*   助手和讲述人辅助功能现在可以判断在具有功能键的设备上是否按下了功能键(“FN”)。
*   提高了某些设备的电池寿命和电源效率。
*   其他几个 [更多的技术变化针对企业用户](https://blogs.windows.com/windowsexperience/2019/08/26/testing-the-throttled-delivery-approach-for-19h2/#QZbBBpYwTtjvvqZG.97) 。

### **如何安装 Windows 10 版本 1909**

由于最近几次 Windows 10 主要更新的错误启动，操作系统不再像这样自动安装新功能更新，所以用户需要手动开始安装——尽管我们 [建议先花几分钟时间准备你的 PC](https://lifehacker.com/how-to-prepare-your-pc-for-windows-10s-november-update-1839138469) ，以防在更新期间或之后出现问题。准备好更新后，打开电脑上的“开始”菜单，然后单击齿轮状图标打开“设置”菜单。在设置菜单中，进入**更新&安全>窗口更新**，更新上线后点击**“下载”**。

这个故事最初是在一个月前发表的，现在 Windows 10 1909 已经公开发布，因此在 11/13/19 进行了更新。
