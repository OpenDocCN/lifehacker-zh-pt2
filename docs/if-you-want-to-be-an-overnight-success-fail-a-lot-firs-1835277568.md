# 如果你想一夜成名，先经历许多失败

> 原文：<https://lifehacker.com/if-you-want-to-be-an-overnight-success-fail-a-lot-firs-1835277568>

詹姆斯·霍尔茨豪尔的成名吸引了《危险边缘》!粉丝这两个月来。但是也许更吸引人的是一个 27 岁的图书管理员看似 [不知从哪里冒出来打败他](http://money.com/money/5646440/who-is-jeopardy-champ-emma-boettcher/) 。新科冠军艾玛·波特彻可以成为我们所有人的一个教训:如果你想成为一个一夜成功的故事，先计划失败一大堆。



当然，Boettcher 有很多优势，首先是她的职业(图书馆员超级聪明，知道很多)。另外，她去了普林斯顿。哦，她写了一篇关于*危险的硕士论文！*提问。可是...她前三次试镜都没有被选中。在第四次尝试时，她将被选中与那个已经连续赢了 32 场比赛，并且差一点就要打破肯·詹宁斯在节目中的最高记录的家伙对抗。

然而，她打败了他。

如果你没有通过危险测试！试镜，你会一遍又一遍的回想*和*吗？我知道我不会。(虽然平心而论，我本来是不会去的；我不擅长琐事。)

但是，你看，Boettcher 一直回去——在大约十年的过程中四次——这就是为什么她是一夜之间家喻户晓的名字，而我们不是。“当她想要什么东西时，她会非常专注，”她的父亲告诉《纽约时报》[。](https://www.nytimes.com/2019/06/03/arts/television/emma-boettcher-jeopardy.html)

失败很糟糕，但它是学习过程中非常重要的一部分几乎对我们做的任何事情都是如此。这就是为什么我们让孩子犯错——我们知道他们会从中学习，即使这意味着一些挫折或挫折的眼泪。

每当我想到失败以及一次又一次重新振作起来所需要的毅力，我就会想起斯蒂芬·金(也许你听说过他)。他在他的著作《论写作》 中写道，作为一名年轻的作家，他收到了如此多的退稿信，以至于他用来钉它们的钉子在如此多的纸张的重压下变形了，他不得不用一个长钉来代替它。有无数这样的故事，来自作家、运动员和各行各业的领导者，他们都是失败者——直到他们不是。

因此，如果你刚刚在某件事情上再次失败，从艾玛·波特切尔身上找到一点灵感，她现在已经三次获胜并正在计数(你可以观看她的突破集，该集于周一播出，并在 YouTube 上 [流传到周五)。](https://www.youtube.com/watch?v=CwgdVoDLHpY)

也许你离一夜成名只有一天多一次尝试。
