# 我什么时候才能在一辆被取消的通用汽车上得到可笑的折扣？

> 原文：<https://jalopnik.com/when-will-i-be-able-to-get-a-ridiculous-discount-on-a-c-1830765919>

作为 Jalopnik 的常驻购车专家和专业购车者，我会收到电子邮件。很多邮件。我决定挑几个问题，试着帮帮大家。本周，我们将讨论通用汽车公司取消的汽车 的潜在大甩卖，为朋友验车，以及买断租约。



首先，现在通用汽车将停止生产几款汽车，它们会变得多便宜？

> “现在是购买 Volt、Impala 或 CT6 等被取消的汽车的好时机吗？我应该等吗？我认为价格会进一步下跌。”

人们会认为通用汽车对他们计划取消的汽车采取了减价销售的态度。问题是，因为没有人购买它们，许多像 CT6、英帕拉、XTS 和君越这样的车型已经清仓销售了一段时间。

快速浏览一些主要的上市网站，你会看到一些汽车的折扣在 10，000 美元左右或更多。这里的例外是雪佛兰伏特混合动力车。在一些地区，Volt 相当受欢迎，因此很难找到一辆引擎盖上有很多现金的车。

购买者需要意识到的是，制造商和经销商不能在汽车上投入更多的钱。我会收到一些人的电子邮件，他们认为自己得到了 15，000 美元的折扣，而实际上他们想要 20，000 美元。抱歉，世界不是这样运转的。

当你购买最好的交易或最低的价格时，这个数字是通过比较一个市场区域内类似汽车的报价来确定的，而不是买方认为一辆汽车应该卖多少钱。

有些人不明白的是，经销商能够以明显低于发票的价格向汽车砸钱，通常是因为所谓的来自制造商的经销商现金。这种“隐性回扣”是提供给经销商，以清除库存，但在某个时间点，它不再可用。这限制了经销商在价格上的深度。因此，如果买家坚持太久，希望以 50%的价格买到 CT6，他们可能会失去获得最大折扣的机会。

如果你想买一辆大型通用轿车，顺便说一句， [很不错，](https://jalopnik.com/making-good-cars-isnt-enough-1830678090) 仔细购物，你应该能在新年前买到一笔好交易。

下一个问题:当你是“车友”，被要求检查一辆你并不真正了解的车时，你会怎么做？

> 我一个住在北卡罗来纳州的朋友的父亲让我在我附近的一家经销店帮他看一辆二手的 2010 款保时捷 911 Carrera S。他让我检查一下可能存在的问题。我的问题是我不是保时捷专家，我更喜欢生锈的吉普牧马人。他真正要求我的是确保没有任何凹痕/划痕/石屑，看看底盘，并确保内部是可以接受的。
> 
> 我不想让他失望，我想多做一点，但是我又不是专家。我能找到的所有汽车都有的明显的物品，但我问你是否有我应该寻找的某些东西，这样当它到达北卡罗莱纳州的卡车后面时，他不会不高兴。

虽然你帮助朋友买车很好，但你承认这不是你的专业领域也很好。因为你只是看油漆，车身，也许开车兜风并不能真正给他所需要的信息。你应该告诉你的朋友，保时捷不是你的菜，他的父亲需要让当地的保时捷专家检查这辆车。

一个好的购买前检查费用在 200 到 400 美元之间。他希望商店在电脑上运行代码，以发现任何不会立即触发发动机检查灯的问题。如果是手动的，他会想要一份超速报告来知道那辆车开得有多猛。你希望有人看看发动机安装和悬挂组件，以及刹车只是其中的几个关键点

与任何购买二手保时捷或任何专业汽车的人交谈，他们会说彻底和独立的检查是避免问题汽车的最佳方式。【T2

最后，有没有一种方法可以买断租约而不被交易商干掉？

> 我有一辆 Rav4 即将到期，决定买下它。我给经销商打了几次电话，都没有回音。我去拜访时被告知，除了支付总额加上销售税之外，我还得另外支付 995 美元的安全检查费和 449 美元的“文件费”。我有一个朋友在另一家经销店，他建议直接打电话给丰田，他们同意只支付销售税。通过邮件处理需要一段时间，但可以节省大约 1400 美元。这在经销商那里不常见吗？

[我不久前报道过这个话题](https://jalopnik.com/can-i-buy-out-my-lease-without-going-through-the-dealer-1765454741) 你绝对可以买断大多数汽车的租约，而不需要通过经销商。一些品牌不会让你这样做，你必须使用经销商作为中介。

我没有通过经销商就买断了我的马自达 3 租约。我承认，虽然我省了几块钱的文书费，但这是一件痛苦的事情，因为我想付汽车的首付，但贷款人就是想不出如何解决这个问题。我最终找到了一家帮助我摆脱困境的当地银行。

对于这家商店来说，449 美元的文件费有点高，但对于推动文书工作来说并不算太离谱。然而，995 美元的安全检查似乎过高。我肯定会直接打电话给丰田，看看你是否能与他们和贷款人合作，或者打电话给其他经销商，看看他们是否能提供一种更便宜的方式来促进这一过程。
