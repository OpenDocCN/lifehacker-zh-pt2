# 知道“海滨”房和“海景”房的区别吗

> 原文：<https://lifehacker.com/know-the-difference-between-an-ocean-front-and-ocean-vi-1835359202>

当你预订去海滩度假的房间时，一定要注意你预订的是什么类型的房间。



本周, [《商业内幕》采访了商业娱乐顾问](https://www.businessinsider.com/hotel-scams-2019-5?r=US&IR=T#if-the-price-seems-too-good-to-be-true-it-might-be-a-scam-4) ，杰夫·拉塞尔，谈论了一些旅游骗局。最基本的是，要留意度假村费用和其他强制性额外收费，或者警惕那些看起来太过好得难以置信的交易。他还建议了一件事，我知道我很少注意:确保你理解描述你房间的语言，在海滩度假时，确保你理解你报名参加的“观点”。

拉塞尔对《商业内幕》说:“‘海景’只是意味着你至少能从房间的某个地方看到一点点风景。一个*海滨*房间实际上面对着大海。”

作为一个最近在夏威夷适度拥有一个“海景”房间的人，这个房间实际上是一个“停车场景观”，回想起来似乎是一个显而易见的事情。

如果你预订的不清楚，不要害怕打电话给酒店，要求对你考虑的房间进行解释。

登记入住时，在接待处再次询问也是值得的。当我打电话到停车场房间询问时，我立即被转移到一个视野更好的房间。在我的情况下，他们有可用性，我所要做的就是问。
