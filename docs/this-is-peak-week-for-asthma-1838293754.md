# 这是哮喘的“高峰周”

> 原文：<https://lifehacker.com/this-is-peak-week-for-asthma-1838293754>

如果你患有哮喘，并且最近感觉病情恶化，这不是你的想象:9 月 1 日被非正式地称为 [【高峰周】](https://community.aafa.org/blog/asthma-peak-week-is-here-here-are-some-tips-to-help-you-avoid-asthma-episodes) ，因为几个加重哮喘的因素汇集在一起。

Watch

首先，有豚草花粉。这是到处都是*的季节*。豚草是风媒传粉的，所以每株植物都会从它看起来不太绚丽的花朵上释放出数十亿粒花粉。(它与美丽的黄花同时绽放，但 [它们不是同一株](https://www.mnn.com/health/allergies/stories/dear-allergy-sufferers-dont-blame-goldenrod) 。)

然后，孩子们回到了学校。无论你是孩子还是父母，细菌交换往往会回家，感冒、流感和其他呼吸系统疾病的季节开始了。

由于秋天我们都在室内度过了更多的时间，室内过敏也可能爆发。也许你最近没有用吸尘器清除沙发垫上的尘螨；也许你的朋友正在他们盖着猫毛的房子里举行聚会，而不是在公园里烧烤。

你是如何应对高峰周的？就像你在一年中的任何时候应对过敏、感冒和其他哮喘一样。但现在你知道这是一个特别糟糕的时期，你可以加快行动:

*   洗手，远离生病的人
*   清洗毯子和枕头，如果你过敏，用吸尘器打扫房间以减少灰尘
*   留意花粉数量，考虑在外出后更换鞋子或衣服，这样就不会追踪花粉进入你的房子

在我们的 [指南中，我们有更多的建议让你的房子](https://lifehacker.com/how-to-allergy-proof-your-home-this-spring-1823970075) 防过敏。现在也是一个好时机，确保你储备了足够的抗过敏和哮喘药物，并且你的救援吸入器还有足够的剂量。
