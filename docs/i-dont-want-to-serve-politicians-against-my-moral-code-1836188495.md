# 问那个咸咸的女服务员:我不想违背我的道德准则为政客服务

> 原文:[https://the take out . com/I-don-to-want-to-serve-politics-against-my-moral-code-1836188495](https://thetakeout.com/i-dont-want-to-serve-politicians-against-my-moral-code-1836188495)

> 亲爱的 Salty:我在你的网站上读到过一个芝加哥鸡尾酒吧的服务员向埃里克·川普 吐口水，这让我陷入了思考。我在华盛顿特区杜邦环岛附近的一家餐馆当了几个月的服务员。政界人士经常光顾我们的餐馆(这几乎是不可避免的)，对我来说，谢天谢地，他们大多数都是匿名人士。但我担心，如果我认识的人，对我的事业造成了伤害，走进我们的餐馆，我会有什么反应。
> 
> 我知道有些人会说:“做好你的工作就好”，但这不是我会永远做下去的工作，如果我为一个破坏我所代表的一切的人服务，我会恨自己。你会怎么处理这件事？
> 
> K

亲爱的 K，

这是一个棘手的问题，不是吗？

这不是你一句话就能回答的，因为这很大程度上是关于你，你和那家餐馆的关系，以及你内心的声音有多大。

Watch

有些是世代相传的事情。可怜的 Salty 已经不再是一只小鸡了，在他长大的时候，你会微笑着为每个人服务，只要他们穿着鞋子和衬衫，并且不想打架。也就是说，从我在餐桌上无意中听到的丑陋对话来看，我招待过很多混蛋。他们是来吃饭的，不是来改变我的观点的，他们的钱和圣人的一样好花。

但这是 2019 年，似乎有一大批政客被 [赶出](https://thetakeout.com/people-keep-targeting-the-wrong-red-hen-restaurant-1827241197)[在餐馆与](https://thetakeout.com/mitch-mcconnell-joins-club-of-politicians-being-angrily-1827456399) 对峙。时代在变，可接受的行为也在变。但是我是一个非常守旧的人，以我浅显的观点，这可能会对我们的行业产生不良影响。毕竟，好客意味着热情好客，我对我的工作很认真，我想确保我所有的客人都有我能提供的最好的体验。当然，你可能会有不同的感受。

与其让事态升级，我可以提几个建议吗:如果这个人是你真正讨厌的人(不一定是政客...如果马丁·什克里里走进来怎么办？)，只要记住你是在为别人工作，所以我会把你的经理拉到一边，解释你的情况。也许看看经理会不会调换你们的桌子。我总是认为:“眼不见，心不烦。”

假设你是店主，参议员 Reinhold McPatriarchy 来吃午饭。你必须权衡不为这个人服务的满足感，以及你的企业在社交媒体上被半个国家的人讨厌的满足感。要知道，你的 Yelp 页面将会被那些从未涉足其中的人的一星评论淹没。如果你的道德准则能经受住这种扯淡，那就继续前进，做你该做的事。你可能会意识到你的骄傲比一堆尖叫的推特蛋更有价值。
