# 现在是尝试微软基于 Chromium 的 Edge 浏览器的最佳时机

> 原文：<https://lifehacker.com/nows-the-best-time-to-try-microsofts-chromium-based-edg-1837449442>

微软基于 Chromium 的快速 Edge 浏览器 [已经有了](https://lifehacker.com/how-to-beta-test-microsoft-s-new-chromium-based-edge-br-1833889177) 的开发者版本和金丝雀版本(更不用说泄露的版本)好几个月了。现在，该公司已经正式推出了一个更稳定的浏览器“测试”频道，你可能想最终冒险尝试一下 Edge。

Watch

测试版预计将持续到该浏览器在 2020 年初全面发布。在此期间，微软应该每六周推出一次测试版的主要修订更新，并根据需要推出较小的补丁。这不同于 canary 和 dev 构建，它们分别接收每日和每周更新。虽然新的测试版将会有更少的实验功能，但使用起来应该会更稳定。

以下是如何加入测试计划并试用新的 Edge 浏览器:

1.  进入 [Edge 内幕网站。](https://www.microsoftedgeinsider.com/en-us/)T3】
2.  点击**“下载 Windows 10 测试版。”**对于 Windows 7、8、8.1 和 macOS 用户，点击**“其他平台和渠道”**，然后向下滚动并下载适合您操作系统的正确版本。(您也可以从该屏幕选择金丝雀和开发构建。)
3.  点击**“接受并下载”**
4.  下载完成后启动 MicrosoftEdgeSetupBeta.exe。
5.  等待安装程序下载并安装必要的文件。完成后，新的 Edge 浏览器将自动启动。
6.  在这里，按照屏幕上的说明从其他浏览器移植书签和其他数据，并自定义用户界面。
7.  如果您跳过设置，但决定稍后导入数据，请单击**“...”右上角的**图标，然后进入**设置>配置文件>导入浏览器数据**并按照说明进行操作。类似地，要改变 Edge 的外观——比如字体选择、主题和菜单布局——进入**设置>外观**。

与之前泄露的一些版本不同，Edge 的测试版不会取代安装在你电脑上的非 Chromium 版本，所以一些 Windows 用户会同时安装两个版本的 Edge。由于 Edge 快捷图标上的大“beta”文本，您将知道您正在启动测试版。

由于你使用的是 Edge 的早期版本，你肯定会遇到一些稳定性问题和 bug。如果你想做一点黑客工作，并在这个过程中增加你的银行账户，微软还为 Chromium Edge 的公共测试版推出了一个新的漏洞奖励计划 ，该计划可以为你报告的漏洞提供高达 15，000 美元的奖励(取决于其严重性)。还有一个 [官方微软 Edge 内部社区论坛](https://techcommunity.microsoft.com/t5/Articles/We-re-Listening/m-p/407743#M7) 用于发布更小的错误和一般反馈或故障排除。
