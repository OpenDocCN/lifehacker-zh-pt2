# 用这个扩展把 YouTube 变成一个不受干扰的音频播放器

> 原文：<https://lifehacker.com/transform-youtube-into-a-distraction-free-audio-player-1840433095>

YouTube 是一个寻找音乐的好地方。既然是视频网站；然而，当你真的只是在寻找一些曲调时，音乐往往会伴随着视频，这会分散你的注意力。 [音乐模式是 Chrome 的扩展](https://chrome.google.com/webstore/detail/music-mode-for-youtube/jeieekjcmkhfffleemgbiooooockfmha?ref=producthunt) 可以帮忙。



一旦启用，扩展会将你重定向到一个页面，该页面只播放视频中的音频，而不播放伴随的视频。该页面也从原始页面中剥离出来，没有广告、视频建议或评论。本质上，它让视频看起来就像是一个音频文件开始。

一旦你安装了扩展，它直接从谷歌搜索重定向。例如，当我从一些谷歌搜索结果中点击一首 Avett Brothers 的歌曲时，我被带到了这里，而不是标准的视频页面。

如果你真的想看附带的视频，你可以关闭扩展，像平常一样观看。

不幸的是，它不会自动播放下一首歌曲，所以一旦这首曲子结束，你必须手动选择另一首继续听。如果你只是想听几分钟“真相伤害”，而不是掉进兔子洞，这可能很好，但如果你正在为你的工作日寻找配乐，这就不太好了。

这是你肯定不会想一直使用的东西之一，但在那些时候，你会想要一个真正精简版的 YouTube 到只需听一两首歌就可以了。
