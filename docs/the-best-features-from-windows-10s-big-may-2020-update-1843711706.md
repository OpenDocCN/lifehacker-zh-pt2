# Windows 10 2020 年 5 月大更新的最佳功能

> 原文：<https://lifehacker.com/the-best-features-from-windows-10s-big-may-2020-update-1843711706>

微软将于 2020 年 5 月对 Windows 10 进行重大更新，其中包含一系列新功能和变化。这不是那些微妙的“安全更新”中的一种，它修补了 Windows 中的一些错误*，但并没有改变你的用户界面或你与操作系统的交互方式。这是值得关注的，所以我现在突出了新的最好的部分。*



在我们开始有趣的事情之前，让我们先来看一下微软更新誓言:

*   我承诺在安装这次重大的 Windows 10 更新之前备份我的关键文件。
*   *我会* [*延迟更新*](https://lifehacker.com/how-to-undo-and-prevent-windows-updates-1836420965) *如果我在做一些相当敏感的事情，以防更新出错*
*   如果我豁出去了，我不会抱怨，现在就安装更新，更新会破坏我的系统
*   *我将阅读* [*这本重新安装 Windows 10*](https://lifehacker.com/the-ultimate-guide-to-reinstalling-windows-from-scratch-1832897572) *的便捷指南，它将让我在灾难来袭时恢复速度*

作为 Windows 内部人员，我已经运行 Windows 10 2020 年 5 月更新的测试版有一段时间了。虽然我没有经历过任何灾难性的错误，但这并不是说它们不会发生。一如既往，更新风险自担。你*应该*安装 Windows 更新，但是在你这么做之前，给他们几天时间去破坏其他人的系统也没什么不对。

也就是说，这里有一些更新后你可以使用的更值得注意的功能:

### 在 Windows 10 上安装速度更快的 Linux

微软为 Linux 推出了全新版本的 Windows 子系统 WSL 2，它的性能大幅提升，并能够通过常规的 Windows Update 应用程序直接更新微软的 Linux 内核。让自己在 Windows 10 上玩 Linux 很容易，微软已经发布了 [一个方便的演练](https://docs.microsoft.com/en-us/windows/wsl/install-win10) 让你开始。

您应该知道，通过 WSL 2 玩 Linux 可能会搞乱您正在使用的其他一些虚拟化应用程序。在微软的综合 [WSL 2 FAQ](https://docs.microsoft.com/en-us/windows/wsl/wsl2-faq) 中阅读这些有趣的细节，以及更多。(此外，您还不能访问 GPU 或 USB 设备，但该功能应该会在未来的 WSL 2 更新中出现。)

* * *

### 告别微软画板和写字板

我不知道你，但我甚至不记得上一次打开写字板(而不是 [记事本++](https://notepad-plus-plus.org/downloads/) )或画图(而不是 [惊鸿一瞥](https://glimpse-editor.org/) )是什么时候了。有很多免费的替代品可以替代这些传统的 Windows 应用程序，微软现在已经将它们都变成了可选的，允许你现在就从 Windows 中卸载它们。只需点击设置>应用>可选功能(在主窗口)并点击微软画图和写字板来删除它们。

* * *

### 强制 pin、面部和指纹解锁

这听起来可能很奇怪，但使用 PIN 码访问 Windows 10 机器实际上比密码更安全。正如微软 [对](http://a) 的描述，一个 PIN 被绑定到设置它的特定机器上；如果有人可以访问您的 Microsoft 帐户密码，如果您用 PIN 锁定了密码，他们将无法登录到您的实际系统。

不过现在，你可以用 PIN 码*或微软账户密码*登录。访问**设置>账户>登录选项**，你将能够强制登录者使用 PIN、指纹或面部识别来登录你的台式机或笔记本电脑。密码，消失。

* * *

### FPS 游戏吧

您可以通过检查 Xbox 游戏栏内置的新每秒帧数计数器来监控您最喜爱的游戏是否正在将您的系统推向崩溃的边缘。当你在玩游戏时，按下 **Windows 键+ G** 弹出游戏条，寻找(或启用)性能小工具。

你必须“请求访问”并重启你的电脑才能看到你的 FPS，所以在启用之前不要太深入你的游戏会话。在你做了之后。你可以看到过去 60 秒内你的 FPS 的小图表。

说到游戏，这里有一个漂亮的额外功能:打开任务管理器，然后点击性能选项卡，你应该可以看到你当前的 GPU 温度。

* * *

### 重置电脑时下载 Windows 10 的最新版本

如果您出于任何原因需要重置您的电脑，无论是保留还是擦除您的个人文件并重新安装 Windows，您现在都可以选择让该工具在启动该过程之前从网站下载最新更新的 Windows 版本。如果你有足够的带宽，这是一个比从硬盘上的旧版本 Windows 10 重新安装更好的选择。以后你会节省很多时间，因为你不必重新下载和安装所有的更新。

* * *

### 召唤 Cortana，不要和她说话

如果你感到害羞，或者不想整晚和 Cortana 聊天吵醒你的室友，你现在可以简单地向微软的数字助理输入你的请求。更好的是，Cortana 的问答窗口不再与你的 Windows 任务栏相连。你可以移动它，调整它的大小，就像任何常规的窗口一样。虽然这些改进可能不会让我更多地使用 Cortana(这就是我一直使用它的方式)，但知道如果我不喜欢它，我不会*变得*喋喋不休，这是一种解脱。

也就是说，Cortana 已经从以前版本的 Windows 中剥离了许多功能。你已经失去了通过 Cortana 控制大多数智能家居设备的能力，更不用说流媒体音乐服务和其他第三方技能了，你以前可能已经集成到数字助理中。真扫兴。

* * *

### 重启时自动启动通用 Windows 平台应用

这个很小，但是很方便:访问**设置>账户>登录选项**，向下滚动一点直到你看到**“重启应用”**启用该设置，当您关闭电脑时运行的任何 UWP 应用(Windows 应用商店应用)应该会在电脑再次开机时自动启动。
