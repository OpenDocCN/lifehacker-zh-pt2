# 不，颈部绑腿并没有被证明比没有面罩更糟糕

> 原文：<https://lifehacker.com/no-neck-gaiters-werent-proven-to-be-worse-than-no-face-1844689356>

有一项新的研究声称 [像 buff](https://buffusa.com/how-to-wear)一样的颈部绑腿比根本不戴面具更糟糕。我不认为这项研究应该从表面价值来看，但也有理由怀疑 Buffs、bandanas 和其他不是设计来做面具工作的面部覆盖物。

我们早就知道 N95 口罩在保护佩戴者和其他人方面做得最好。外科“手术”口罩——那些一次性医用口罩——是仅次于口罩的最佳选择，它可以将大部分(但不是全部)呼吸飞沫留给自己，同时还可能保护自己免受他人伤害。

由于这些口罩并不总是可以买到，所以我们大多数人最终都戴上了布口罩。布口罩 [以类似于一次性手术口罩的方式提供保护](https://lifehacker.com/how-masks-protect-us-from-the-coronavirus-1844220877) ，尽管它们可能没有那么有效。

也就是说，并不是所有的面具都是平等的。当我们争先恐后地寻找布口罩时，各公司开始宣传不同类型的口罩。任何卖绑腿的人都在为他们的绑腿做广告；生产带阀门口罩的公司正在销售带阀门的口罩。当然，这并不意味着这些是最好的选择。

### 这项研究发现

到目前为止，还没有一个很好的方法来测试口罩，看看它们是否“足够好”来抵御冠状病毒。医用口罩经过一定的质量控制检查，但时尚口罩没有这样的程序。你在家缝的或即兴创作的也不行。

因此，来自北卡罗莱纳州的一组研究人员发表了一篇论文 [描述了一种廉价的工具](https://advances.sciencemag.org/content/early/2020/08/07/sciadv.abd3083) 可以用来测试口罩。这个设备基本上是一个盒子，里面有一个激光器和一个手机摄像头；你把你戴着面具的脸贴在盒子上，对着它说话。激光器发出的光反射到从你嘴里喷出的液滴上(或者没有，取决于面膜的效果)，然后被相机记录下来。

该设备不测试人们是否得病，也不测试是否传递了传染性剂量的病毒；它只是寻找空气中的水滴。这项测试中没有涉及冠状病毒。

因此，为了测试该设备，研究人员戴着各种面具对着盒子说话。正如预期的那样，人们戴 N95s 时产生的水滴很少，外科口罩表现更好，布口罩根据其材料和结构而有所不同。

不过，这里有一个很大的警告:这是一个概念验证，看看这个盒子是否可以测量水滴，以及不同面膜之间的水滴看起来是否不同。这不是对口罩本身的决定性测试。事实上，研究人员在他们的结论中写道:

> 同样，我们要注意的是，此处执行的屏蔽测试(所有屏蔽使用一个扬声器，所选屏蔽使用四个扬声器)仅作为演示。受试者之间的差异是可以预料的，例如由于生理、面罩佩戴、头部位置、说话方式等的差异。

### 关于不同类型的口罩，我们应该知道些什么？

首先，如果某样东西不是真正的面具，我们不应该期望它的表现像面具。疾病预防控制中心和世卫组织 [都建议](https://www.cdc.gov/coronavirus/2019-ncov/prevent-getting-sick/diy-cloth-face-coverings.html) 戴一个紧贴嘴巴和鼻子的口罩，这排除了追星族，以及绑着银行抢劫犯风格的头巾。他们还 [警告不要戴带有呼气阀](https://lifehacker.com/if-your-mask-has-a-valve-its-half-useless-1843182019) 的面具，因为那会把你的细菌空气排放到世界上。

这些都不是新的。这项研究的新发现是，buff *可能*比没有更糟糕，因为它的开放编织*似乎*能将较大的液滴分裂成较小的液滴。那个*可能*是个问题。看到那些狡猾的话了吗？这是我能准确描述结果的最佳方式。很有可能 buffss 比没有 buff 更糟糕，但是这个问题需要更多的研究。没有人研究过魔法是否真的会增加人们生病的几率。

我认为值得重新考虑你的 buff，如果你穿的话。我知道很多跑步者喜欢在脖子上挂一个，这样他们超过别人时就可以把它拉起来。我们现在有一些数据表明这可能不是一个好主意，但是我们已经知道 buff 并不是真正的面具——所以也许我们一开始就不应该依赖他们。
