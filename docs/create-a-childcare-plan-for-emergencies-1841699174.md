# 为紧急情况创建一个儿童保育计划

> 原文：<https://lifehacker.com/create-a-childcare-plan-for-emergencies-1841699174>

快速:你带着两个孩子，一个伤得很重，不得不去医院，但另一个不允许乘坐救护车。你是做什么的？或者！你在准备晚餐时切伤了手指，你知道缝针是必要的，但孩子们在跑来跑去，而你的伴侣还没有回家(或者你是单亲)。现在怎么办？



大多数时候，我们可以提前安排好托儿服务。即使这种关心由于某种原因落空了，现在我们开会要迟到了，它仍然不能真正归类为紧急情况。我们可以打几个电话，发几条短信，或者带着孩子一起去，就这一次。但是，当严重的疾病或伤害发生时，现在你面临着试图让生病或受伤的人得到他们需要的帮助*加上*不抛弃你的孩子，事情就变得危险了。如果你在旅行，那就更是如此了。

詹茜·邓恩为《纽约时报》 撰写文章称，这些情况就是为什么所有父母都应该有一个紧急育儿计划。在与丈夫、女儿和侄子去缅因州旅行后，她发现这是一个好主意；他们还在度假时，她的丈夫开始出现严重的胃痛:

> 我们很快得知，汤姆患了阑尾炎。由于远离我们在布鲁克林的家，我没有地方让孩子们停车——所以在汤姆接受紧急手术前后的 36 个小时里，我在脸上贴了一个南瓜灯笑容，试图让我的侄子和女儿振作起来。
> 
> 事实证明，这个任务比照顾我的丈夫更难。

### 创建联系人列表

从本地考虑开始你的紧急儿童保育计划。联系你所在地区的四五个朋友或家人，问问他们是否会在你的紧急联系人名单上。(你也应该主动提出加入他们的行列；这才公平。)

如果他们都住得很近，最好试着从你孩子的学校里挑选至少一个家长同伴(以防你需要接送)和一两个邻居(因为有时候你真的只有一点时间去敲最近的门)。

把它们储存在你的手机里，以一种容易获取的方式，比如“收藏夹”或者紧急联系人。

### 分享重要信息

接下来，创建一个可共享的文档——谷歌文档是一个很好的选择，这样你就可以很容易地更新它——来存储紧急护理人员可能需要的重要信息，例如他们的学校安排或过敏信息。也包括你的孩子自己的病史，儿科医生的联系方式和健康保险信息，以防万一，正如 [邓恩指出](https://parenting.nytimes.com/preschooler/family-emergency?module=editors-picks&action=click&region=1) ，他们在人的照顾下也需要医疗照顾。

你甚至可能想更进一步:

> 妮可·德拉霍萨伊(Nicole Delahoussaye)是得克萨斯州德洛特斯的一名律师，也是播客“海拉·斯马特·穆瓦斯”的主持人，她建议起草紧急临时监护令，指定当地人，在学校存档，在家里随时可以发现。她说，当这样的文件存档时，“如果孩子不在家或由未成年人照看，如果没有直系亲属，他们将不会被纳入儿童保护范围。”

### 如果你不在城里(或者没有人可以帮忙)

当然，当你远离家，远离你的支持系统时，最疯狂的护理紧急情况可能会发生。在出发去旅行之前，做一些关于该地区的调查，看看当地是否有紧急托儿服务。

还有国家紧急护理选项，如 [保姆 Poppinz](https://www.nannypoppinz.com/services/emergency-childcare-nanny-services.html) 。根据该服务的网站，每个保姆都“至少有一次与我们的保姆招聘者面对面的面试，至少 18 岁，受过 CPR/急救培训，并有至少三年的经验证明与儿童一起工作。”

医院通常没有 24-7 全天候的紧急儿童护理服务，所以你要准备好另一个选择。但在必要时，你也可以向医院的现场社会工作者寻求建议，因为这是他们以前可能遇到过的问题。

一旦你知道你的应急计划是什么样的，就和你的孩子交流。提前告诉他们谁在你的紧急联系人名单上，你相信他们会好好照顾他们，直到事情平息。
