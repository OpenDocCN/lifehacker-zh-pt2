# 今年夏天你应该做一个康普茶花车

> 原文：<https://lifehacker.com/you-should-make-a-kombucha-float-this-summer-1835844010>

尽管在俄勒冈州的波特兰，所有的嬉皮士宣传都被强加给我，我认为康普茶对于经常做瑜伽的人来说是昂贵的苏打水。然而，我确实喜欢很多布丁的味道，当我把冰淇淋放进去的时候，我就更喜欢了。



就夏季甜点而言，我一直非常推崇花车。这很容易做，又冷又提神，而且对大众来说是小菜一碟(就好像人们曾经聚集在我周围一样)。冰冷的小苏打晶体围绕着奶油状的冰淇淋，碳酸化的气泡在你的舌头上跳动——这是完美的炎热天气的款待，我不会接受任何批评。(想成为你参加的任何夏季晚会的英雄吗？接受吴宇森父亲的建议， [带着彩车用品](https://skillet.lifehacker.com/the-best-things-to-bring-to-a-party-that-arent-booze-1834954458) 现身。)

在你得到所有的感谢之前，我会坚持我的根啤酒！下面，我想温和地建议你拓宽你的味觉和你的*调色板*。康普茶不仅味道鲜美，而且非常漂亮。通过混合时髦的 buch 和 custardy 香草冰淇淋，你可以得到一个柔和的浮动，带有一点令人愉快的酸味，平衡了冷冻乳制品的甜味和丰富性。

就口味而言，所有水果味的宝贝都不错，但任何有姜的都是明显的赢家。对于冰淇淋，我坚信香草的力量——越丰富越好。你可以选择不含乳制品的路线，尝试水果冰糕，但是好的漂浮是关于平衡和对比的，将酸和酸搭配可以创造出一维的漂浮。就食谱而言，我想你已经掌握了:将一份冰淇淋倒入一个高脚杯(如果你有磨砂的话)，用康普茶填满剩余的空间，用一个细长的勺子和吸管食用。
