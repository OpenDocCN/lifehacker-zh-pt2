# 如何使用社会工程——但不是以邪恶的方式

> 原文：<https://lifehacker.com/how-to-use-social-engineering-but-not-in-an-evil-way-1838776443>

如果你想赢得朋友并影响他人，那么，有一整本书教你如何去做。但是，你如何以正确的方式进行社会工程——让每个人的生活变得更好，而不仅仅是操纵人们做你想做的事情？



由于搭讪艺术家、咄咄逼人的销售人员和在线骗子，社交工程已经形成了一个坏名声，但有一种不太为人所知的技术可以帮助你们建立联系，并让你知道什么时候该停止尝试:*镜像。**T3***T5】

如果你了解让人们喜欢你的科学，你可能对镜像很熟悉。这是我们开始将目光、手势、身体姿势和说话方式与周围的人相匹配的时候。我们中的许多人都不假思索地照镜子(例如，这就是为什么你经常看到人们开始将口音与他们周围的人相匹配)。然而，也有刻意模仿的方法——这是人们用来快速建立关系或建立信任的一种常见的社会工程技巧。

好吧。你正站在某人旁边，你想从这种互动中策划出某种社会结果，无论你是希望结交新朋友，建立职业关系，还是产生浪漫的兴趣。

你开始镜像。也许你倾斜你的头来配合他们，或者把你的身体重心转移到同一个方向。也许你会附和他们的某个手势。如果你感觉到额外的社会功能，你可能会加入另一种常见的策略，比如“在对话中使用这个人的名字”或者“请别人帮个小忙”

然后你仔细地观察对方的反应。

如果他们喜欢你(社交上、职业上或浪漫上)，他们会开始模仿你的肢体语言和说话方式。如果不是，他们会有意或无意地打破镜子。

你可以等待你的谈话伙伴有机地打破镜子(在许多情况下，他们只是把身体转向远离你的方向)，或者你可以在你的谈话中插入一个模式或位置，看看他们是镜像还是打破。例如，如果你与某人面对面，你将目光转向房间的另一个地方，他们会跟随吗？还是会以此为借口查看手机、喝鸡尾酒或耐心地盯着相反的方向？

因此，如果你说*土豆*，他们说*土豆*(正如歌曲所唱)，或者如果你开始向上屈折你的句子，他们开始向下屈折，从社会工程学的角度来看，你能做的最好的事情就是*取消整个事情*。

不要试图重新镜像破碎的镜子。不要用他们的名字超过 17 次。在任何情况下，都不要去刷手臂或拍肩膀或任何其他你可能读过的“升级基诺”技巧。

相反，在他们仍然对你有好感的时候，结束互动。

因为那是你的最终目标，对吗？在大多数时候，被大多数人看好。你不能通过社交工程让每个人都成为你最好的朋友或业务联系人，但你可以使用基本的社交工程让更多的人在你身边感到舒适——对他们必须说的话表现出兴趣，通过模仿或请求帮助建立联系点，然后如果他们不同意你的联系愿望，就优雅地结束互动。

记住，即使是最无聊的泡妞文学也建议你先做一些社会工程的事情，然后*等着别人来找你*。既然术语“社会工程”现在已经扩展到包括像网络钓鱼诈骗这样的东西，我不妨提醒你它们是在同样的原理下运作的。(骗子发起，等标记回应。)

因此，如果你打算善用社会工程，注意你交往的人对日常社会工程策略的反应。例如，在共同的目标上建立融洽的关系是和某人建立关系的好方法——但它也可能变成一种被称为“强迫团队”的操纵工具你如何让自己站在正确的一边？不要勉强。让人们选择他们是否想加入你的团队，你会开始和那些想和你在一起的人建立联系，而不会疏远那些不想和你在一起的人。
