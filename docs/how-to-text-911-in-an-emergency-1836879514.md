# 如何在紧急情况下拨打 911

> 原文:[https://life hacker . com/how-to-text-911-in-a-emergency-1836879514](https://lifehacker.com/how-to-text-911-in-an-emergency-1836879514)

在某些时候，你可能会发现自己处于紧急情况中，尽管你带着手机，但你可能无法拨打电话或与 911 接线员交谈。你可能想保持安静，因为你在躲藏；你可能会因受伤而无法说话；或者您的接收效果不佳，无法进行语音通话。在这种情况下，你可以发短信给 911。

Watch

2014 年，美国最大的手机运营商推出了 [短信转 911](https://lifehacker.com/you-can-text-911-in-an-emergency-starting-tomorrow-1576457059) 计划，发送到 911 的短信将被重新路由到当地警察局。然而，这取决于当地警察部门和呼叫中心实际执行他们的计划。在该计划推出后的五年里，许多司法管辖区增加了这一功能，使更多人可以选择紧急短信。

### 找出它在你的地区是否有效

联邦通信委员会跟踪哪些管辖区已经实施了文本到 911 程序。你可以 [在这里](https://www.fcc.gov/files/text-911-master-psap-registryxlsx) 下载列表，搜索你所在州、市或县。如果你还不在名单上，请定期回来查看；FCC 每月更新一次注册表。

如果这个项目在你所在的地区很活跃，你可以像给其他人发短信一样给 911 发短信。只需在手机的信息应用程序中打开一条新的短信，输入 911 作为收件人，输入你的信息，然后点击“发送”

如果您所在的地区没有该功能，并且您试图发送文本，您将会收到一条退回信息，通知您您的信息没有通过。

### 你应该发送什么，不应该发送什么

发短信是一种比较慢的交流方式，接线员也不能快速地问很多后续问题，所以 CNET 建议你在发送的第一条信息中包含尽可能多的信息。一定要马上写上你的地址或位置；他们可能会也可能不会自动收到这些信息，这取决于他们使用的设备。

> 最好把你的位置和附近的十字路口以及你所在的建筑、公园或企业的名称都包括在内。另外，说明情况，是否有人受伤，是否有武器，等等。
> 
> 尝试在一条信息中包含尽可能多的信息，以节省时间，避免在你试图隐藏的情况下不得不反复点亮手机屏幕。

如果你不能打电话，也要在信息中注明，因为接线员可能会建议你尝试打电话。不要向 911 发送群发短信、照片或视频——接线员无法在他们的系统上访问这些内容。

此外，尽量避免使用俚语、缩写或表情符号。CNET 提供了一个有用的紧急短信的例子:“救命，有人闯入了我在主街 123 号的房子。我躲在楼上的壁橱里，不能讲电话。”

然而，只要有可能， [FCC 建议](https://www.fcc.gov/consumers/guides/what-you-need-know-about-text-911) 拨打 911 而不是发短信。这仍然是最快最可靠的求助方式。
