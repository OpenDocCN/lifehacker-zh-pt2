# 如何跟踪当地学校的新冠肺炎疫情

> 原文:[https://life hacker . com/how-to-track-新冠肺炎疫情在当地学校-1844917259](https://lifehacker.com/how-to-track-covid-19-outbreaks-in-local-schools-1844917259)

学校正在开学，随着新冠肺炎疫情在学生和员工中被发现， [不可避免地至少一些学校会再次关闭](https://offspring.lifehacker.com/schools-will-close-again-1844604561) 。没有联邦系统来追踪学校疫情，但是 [国家教育协会一直在编辑新闻故事](https://app.smartsheet.com/b/publish?EQBCT=00a2d3fbe4184e75b06f392fc66dca13) 并按学区列出。

这项工作是由堪萨斯州的一名教师艾丽夏·莫瑞斯发起的，NEA 接管了这项工作，试图将它作为一个大型的、经过验证的国家数据库来维护。但它仍然依靠教育工作者和其他志愿者来报告 COVID 病例以及美国各地小学的安全问题

通过检查媒体报道来核实病例和疫情。这里有一个关于提交案例报告的链接，还有一个关于健康和安全问题的链接。问题被视为机密，您可以匿名提交。莫里斯建议，在缺乏信息的地方，人们可以向当地新闻媒体提交一个提示，他们可能会做进一步的调查。

点击州报告,你可以看到按学区列出的疫情，并有更多信息的媒体报道链接。这不是一个功能齐全的追踪器，其组织者强调它是不完整的，但它代表了惊人的工作量，如果你试图弄清楚你的学校是否有疫情爆发，那么会很有用。

遗憾的是，我们没有一个全国性的、有组织的、资金充足的努力来准确跟踪学校案例；由于联邦政府对新冠肺炎的反应迟缓，T2 各州正争先恐后地做他们力所能及的事情，以及他们有政治支持的事情，而这些政治支持对 T4 来说并不多。因此，我们赞扬 NEA 和他们的志愿者收拾残局，做下一件最好的事情。 [在此](https://app.smartsheet.com/b/publish?EQBCT=00a2d3fbe4184e75b06f392fc66dca13) 查找您所在州的报告。
