# 使用这些方便但礼貌的邮件回复来节省时间

> 原文：<https://lifehacker.com/use-these-convenient-but-polite-canned-email-responses-1840829836>

不管你认为自己有多忙，总有几封(或很多)邮件在那里，等着回复。你知道你应该回复，否则你会收到可怕的后续邮件，这只会进一步堵塞你的收件箱。即使是发送最基本的、重复的电子邮件也会耗尽宝贵的脑力。要是有办法让这个过程变得简单一点就好了。

Watch

幸运的是，有。通过[推荐做简讯](https://www.getrevue.co/profile/Recomendo) ，我们了解到一个名为 [的网站做的正是你想的那样:为你在许多职业和个人场合提供预先写好的、礼貌的回复。只需点击邮件主题，从列表中选择，包括“需要建议”、“为问题/bug 道歉”、“跟进”、“不感兴趣、抱歉”和“收到任务，以后再做”等。](http://www.cannedemails.com/)

例如，如果您点击“为问题/错误道歉”，网站将生成以下电子邮件:

> 我刚看到你的邮件。呀！
> 
> 很抱歉你一直有问题。

> 我要去调查发生了什么事，然后给你答复。
> 
> 我们将确保尽快解决此问题。
> 
> 与此同时，如果我能做些什么来确保你有一个很棒的经历，请告诉我。

甚至还有“和某人分手”的选项:

> 我知道通过电子邮件这样做很不体贴，但我需要告诉你一些重要的事情，我很难当面说。
> 
> 我想我们应该见见其他人。
> 
> 这是我需要为自己做出的人生决定。请不要把这放在你自己身上，想知道你本可以做得不同，或者认为你因此而不是一个人。
> 
> 你真的是一个伟大的人，我真诚地享受我们一起度过的时光。我一点也不后悔我们在一起。这是我必须做的决定，如果伤害了你，我很抱歉。这是我必须做的事情。
> 
> 我希望这次分手对我们双方都是友好的。如果你不开心，我能理解。我希望你找到幸福。

(把那封罐装分手邮件放在手边可能也是个好主意，这样你就能知道是否有人用它来对付你。)
