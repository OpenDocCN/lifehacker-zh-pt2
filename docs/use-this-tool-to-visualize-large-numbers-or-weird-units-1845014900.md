# 使用这个工具来可视化大的数字或奇怪的单位

> 原文:[https://life hacker . com/use-this-tool-visualize-large-numbers-or-verged-units-1845014900](https://lifehacker.com/use-this-tool-to-visualize-large-numbers-or-weird-units-1845014900)

你有没有遇到过一个非常大的数字或者一个模糊单位的度量，然后想，我不知道它到底有多大或者有多少？如果你能和你熟悉的东西比较，比如一场篮球赛的长度或者一个浴缸的大小，不是更容易吗？

Watch

将你奇怪的量输入，你会得到一个用你可以概念化的单位(至少是一般的)来衡量的比较和相对数字的列表，比如足球的体积或猫的重量。

即使你不能实际描绘出从纽约到洛杉矶的准确距离，你大概也能想象出它的十分之一(554 公里)是相当长的。或者，如果你住在西雅图，你可能会认为 554 公里也相当于 3000 个互相堆叠的太空针。

你也可以用事物的尺度来理解不熟悉的单位，比如用更标准的术语来理解“深、摇、石”。根据记录，一英寻大约相当于一张全尺寸床的长度，一个 shake 相当于一个 IndyCar 中转站的长度(所以，非常快)，一块石头大约相当于两块砖的重量。该数据库有重量，长度，速度，时间，高度，面积，体积和计算机数据的测量比较。

当您搜索您的测量时，您会得到从几页到几十页的结果，通常从最接近一对一的比较开始(尽管您也可以按最高/最大和最低/最小排序)。这意味着你有很多选择来找到一个你个人相关的衡量标准。
