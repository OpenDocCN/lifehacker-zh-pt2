# 把车从一个停车位开到另一个空车位可以吗？

> 原文：<https://jalopnik.com/is-it-okay-to-pull-through-one-parking-spot-into-anothe-1841005241>

我不确定我是否完全意识到这个概念有多么有争议，或者人们对这个相对简单的概念有多么分歧:当你在一个停车场，有背靠背的停车位，开车穿过一个空位，直接进入与它的尽头相连的空位可以吗？一个孩子可能会问的问题，但不是幼稚的问题。还有，你为什么让那个孩子开车？这是严肃的事情。

Watch

就我个人而言，我认为我赞成把车开到空位上。当我遇到一个空位正盯着我的情况时，这就像是一个小奖励:你可以把你的车停在机头朝外更容易出去的位置，真的不需要额外的努力。那么为什么*不*去做呢？

有很多很好的理由可以考虑这样做:从统计数据来看， [离开停车位往前走](https://www.nsc.org/road-safety/safety-topics/distracted-driving/parking-lot-safety) 要安全得多，因为即使有备用摄像头和小玩意儿在你要撂倒一只狗的时候向你发出嘟嘟声，透过前挡风玻璃，你从车里看到的景色仍然要好得多。

美国汽车协会也站出来反对退出停车点，尽管他们似乎反对退出的基本想法，并且对“渡过难关”策略也不清楚。

当然，这也有不好的一面:如果停车场非常拥挤，你可能会在另一个司机进入停车场时驶入一个车位，造成的短暂混乱，如果两个司机都很糟糕，就会发生低速事故。

还有其他一些情况，司机可能会被骗以为停车位是开放的，而实际上却不是——例如摩托车，或者如果你在两辆笨重的 SUV 之间有一辆小车，那么停车位被占用的事实也可能是不清楚的，但任何合理细心的司机都可以处理这种情况。

此外，一些地方 [已经将倒车入停车位定为非法，](https://www.khon2.com/news/drivers-surprised-frustrated-with-tickets-for-reversing-into-stalls/) 虽然这与停车无关，但更多是在只有后牌照的地方，因为警察希望能够方便地检查停车场中汽车的牌照。

总的来说，我仍然支持度过难关；如果你的情况是有可能做到的，而且事情不是那么繁忙，以至于你担心其他司机可能会猛扑过来与你锁定目光/前灯，那么我说犒劳一下自己。开始吧。
