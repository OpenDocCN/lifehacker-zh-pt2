# 少关注任务，多关注“基本技能”

> 原文：<https://lifehacker.com/focus-less-on-assignments-and-more-on-essential-skills-1843462793>

随着学年即将结束，漫长的夏季即将来临，我经常被两个问题所困扰:我的儿子是否已经学会了他应该在三年级学的所有东西？在本该是他快乐的暑假期间，如果不制定一个持续的学习计划，我该如何阻止他进一步落后呢？

Watch

他得到的一些任务看起来很有价值，而一些看起来很忙。我想学前班和小学的作业一直都是两者的混合体，但现在尤其如此，很难区分两者。这就是为什么凯瑟琳·希尔，一位学习专家和家长教育家，告诉纽约时报，家长可以向老师询问孩子应该学习的基本技能。

> 学龄前和小学年龄的学生需要父母的大量支持来完成他们的远程学习义务。但是现在在许多家庭中，这种疏忽是不可能的。希尔说，确定哪些作业最重要的一个方法是询问你的老师你的孩子在学年结束时应该掌握哪些技能。然后尝试将这些技能嵌入到孩子的自然活动中。

由于本学年的进展方式，他们很有可能在暑假到来之前掌握关键技能。但是如果你现在弄清楚这些技能是什么，你就可以在剩下的 [远程学习](https://offspring.lifehacker.com/dont-call-this-homeschooling-1842791078) 的情况下把它们作为重点。然后，随着春天变成夏天，你可以想办法将这些技能的强化融入到有趣的日常活动中。

举例来说，我注意到在我儿子的课堂上非常注重的一项技能是写两个连续的五句话段落的能力...随便啦。他们读过的一个故事，他们喜欢的一种食物，或者他们有一天想去的一个地方。话题本身几乎无关紧要；重点是段落写作的练习。

所以，整个夏天，我可以向我儿子建议，我们把我们在疫情的所有冒险都记下来。或者我可以鼓励他开始给一个要搬走的朋友写信。或者我们可以拿出人行道上的粉笔，开车去他祖父母家，在他们的车道上给他们写信。有很多方法可以让写作不再是一项“任务”,而是一件有趣的事情，让我们与所爱的人联系起来。

如果加减法是他们需要练习的技能，让他们去厨房，选择一个食谱，然后把它减半或加倍。没有比转换成分数量更好的数学练习了。

你可以练习看时间，方法是决定你们一起做一个拼图要花多长时间，然后盯着时钟。一起阅读棋盘游戏说明，通读路边的菜单选项，或者简单地通过 [打开电视字幕](https://lifehacker.com/help-your-learning-reader-by-turning-on-tv-captions-1832322982) 来练习阅读。

现在就和你孩子的老师谈谈他们应该关注哪些顶级技能，然后在我们从学年结束过渡到夏季时，寻找简单、压力小的方法来融入这些技能的练习。
