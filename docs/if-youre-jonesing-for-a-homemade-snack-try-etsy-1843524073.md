# 如果你想吃自制小吃，试试 Etsy

> 原文:[https://life hacker . com/if-you-jonesing-for-a-自制零食-try-etsy-1843524073](https://lifehacker.com/if-youre-jonesing-for-a-homemade-snack-try-etsy-1843524073)

感觉饿吗？也许是时候跳过 Doordash 和 Grubhub，尝试一种新的在线服务来满足你所有的零食需求了: [Etsy](https://www.etsy.com/) 。

是的 Etsy。这个迷人的小地方是大多数人购买小饰品和自制的版权作品仿制品的地方——为什么 [迪士尼没有起诉 Etsy 遗忘](https://news.avclub.com/disney-is-going-after-etsy-shops-selling-unlicensed-bab-1841095995) ，我永远不会知道——显然也是业余面包师典当他们美味商品的地方。

我承认，我不太喜欢 Etsy，但为了确保我不是一个完全的新手，我问了一些 Lifehacker 的员工，他们是否知道“Etsy 上的食物”这种东西。他们没有，所以我大胆猜测，你可能也不知道。

Etsy 没有“烘焙”这个类别。事实上，该网站的主页上没有任何东西给人留下你可以在上面买到食物的丝毫印象。食物的图片？当然可以。钩编的玉米棒子？绝对的。一些好吃的东西，你可以放在嘴里吞下去，给你一天所需的能量？号码

做一个快速的 [类别搜索](https://www.etsy.com/search?q=homemade+food&ref=guided_search_1_1&explicit=1&guided_search=1) ，然而，自助餐打开了:

你可能不会在 Etsy 上看到任何超级容易腐烂的东西，或者需要一些奇特的冷却装置来运送。换句话说，没有自制的冰淇淋、鸡块或披萨——任何强力隔离饮食的主食。美味的自制面包似乎是 Etsy 的必经之路，但如果你不想买别人创造的，你也可以自己做。给自己 [弄点酵母](https://www.etsy.com/search?q=yeast)[让‘er rip](https://skillet.lifehacker.com/how-to-make-a-single-packet-of-yeast-last-1842923653)，或者买 [别人的](https://www.etsy.com/search?q=bread%20starter)[弄去](https://skillet.lifehacker.com/heres-a-helpful-timer-for-all-that-sourdough-youre-maki-1842713080) 。你的电话。

或者你可以直接说见鬼去吧。工作太多了，我宁愿埋头做 [一个巨大的自制棉花糖](https://www.etsy.com/search?q=marshmallows) 或者合适的 [零食升级](https://www.etsy.com/search?q=rice%20krispie%20treat) 。还有我最喜欢的: [无所不包的 gummi](https://www.etsy.com/search?q=gummy%20candy) 。不过，如果你更喜欢野外生活，你也可以在 Etsy 上找到大量美味的辣酱。(我就过了。)

### 自制商品可能很美味，但不安全

关于 Etsy 购买食物有一点值得关注。截至目前，Etsy 似乎没有使用任何验证方法——如果有的话——让卖家证明他们遵守了他们所在州关于食品生产和销售的法律。例如，加州人 [可以在自己的房子里随意制作和出售食品](https://forrager.com/law/california/) 而无需检查，甚至可以在网上进行。但是，买家要直接从你这里取；你不能把它运出去。如果你是俄克拉荷马州的面包师，想在 Etsy 上交易，那么 [祝你好运](https://forrager.com/law/oklahoma/) 。

正如 Etsy 的一位发言人告诉我的那样:“Etsy 不是一个策划或裁决的市场，这意味着任何人都可以在任何时候在网站上列出任何东西。但是，当卖家在 Etsy 上开店或列出一个 n 商品时，他们同意遵守我们网站的 [政策](https://www.etsy.com/legal/sellers/?ref=list) 和 [条款](https://www.etsy.com/legal/terms-of-use) ，以及国际、联邦、地方和国家法律(包括关于食品和可食用商品销售的地方法律)。我们会移除不符合我们网站政策的商品和店铺，因为我们每天都会意识到并执行这一政策。”

坦白地说，我不太关心人们在家里卖他们想要的东西。但是你无法知道一个人的家庭烹饪环境是什么样的:有多干净卫生，食物是否是围绕着你可能会过敏的其他食物准备的，无论什么东西在卖给你之前已经放置了多长时间，等等。虽然你在网上购买的任何食品都是如此，但我觉得从受监管和检查的企业购买核仁巧克力饼比从某个随机的人的厨房购买要好一些。

考虑到我们现在生活在一个疯狂的疫情时代，这种情况只会被放大。因此，虽然你现在可能有点反对在 Etsy 上购买“妈妈美味的巧克力曲奇”，但一旦新冠肺炎稍微平息下来——无论是疫苗还是其他，你可能会想在心里将它标记为书签并重新访问。但是如果对自制食物的担忧没有困扰到你，并且你感觉很快，不要忽视艺术和手工艺网站的秘密菜单。
