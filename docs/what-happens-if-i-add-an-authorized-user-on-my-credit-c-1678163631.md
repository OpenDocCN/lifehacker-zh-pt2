# 如果我在信用卡上添加授权用户会发生什么？

> 原文：<https://lifehacker.com/what-happens-if-i-add-an-authorized-user-on-my-credit-c-1678163631>

将某人作为授权用户添加到您的信用卡帐户可以帮助他们建立或重建信用。但你也需要一路保护好自己的信用。



在进行这种安排之前，你需要知道以下几点。

### **主卡持卡人承担所有责任**

当您将某人添加为授权用户时，他们可以拥有自己的名片，上面有他们的名字。他们可以购物，但你要确保他们支付这些费用。

你可以让他们直接付钱给你。您还可以让他们在线访问帐户，这样他们就可以从自己的银行帐户直接向发卡机构付款。

但从法律上来说，授权用户不承担任何费用或余额。最终，账户余额和卡的全部责任都在于你，主卡持有者。

显然，你需要能够信任任何你考虑加入你的卡的人。你还应该有一个计划，告诉他们如何使用这个账户，以及如何付款。

### **授权用户的信用会怎么样？**

虽然授权用户不负责购买，但帐户会显示在他们的信用记录中。该卡的信用额度、卡内余额和支付历史都会显示在他们的名下，就像你自己的信用报告一样。

虽然主持卡人——在这种情况下是你——对账户负责，但你犯的任何损害自己信用的错误也会影响授权用户的信用，反之亦然。

如果主要用户最终错过付款或积累了太多债务，授权用户可以 [从账户中删除他们的名字](https://lifehacker.com/how-to-close-your-authorized-user-credit-card-without-h-1841158752) 。

### **授权用户的信用记录不会影响您的信用记录**

虽然您对您帐户上授权用户的购买和活动负责，但简单地将他们添加到您的帐户不会以某种方式影响您的信用。

他们的名字，以及您将他们添加为授权用户的事实，根本不会显示在您的报告中。但是他们的行为会影响你的信用。例如，如果你的授权用户有过去的债务与你的信用账户无关，这不会影响你自己的信用报告或分数。但是如果你的授权用户的消费习惯在你共享的账户上累积了相当多的余额，你可能会看到你的分数下降。

总的来说，给你的信用卡账户添加一个授权用户是有一些风险的，但是这来自于他们的财务行为。在做决定时，认真考虑他们的财务习惯，并为他们使用信用卡和用信用卡付款提供指导。
