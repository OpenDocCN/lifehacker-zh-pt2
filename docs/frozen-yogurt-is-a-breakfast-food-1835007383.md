# 冷冻酸奶是一种早餐食品

> 原文：<https://lifehacker.com/frozen-yogurt-is-a-breakfast-food-1835007383>

早餐吃冷冻甜点似乎是早晨的终极叛逆。这不仅是一天中的“错误”时间，而且用所有的糖开始你的一天是一个大胆的选择，尽管如果你仔细想想，并不比用肉桂卷开始你的早晨大胆多少。

但是以冻酸奶开始你的一天实际上是很有意义的，尤其是在夏季，如果你没有空调的话(尽管我真的希望你有空调)就更有意义了。

我并不是建议你把早晨的星巴克咖啡换成 Pinkberry，但这主要是因为我认为他们不会那么早开门。我建议你做的是用希腊酸奶制作你自己的(适度甜化的)蜂蜜香草酸奶，这样你可以对所有的蛋白质感到道德，但仍然可以在午餐前给你的一天带来一些奇思妙想和乐趣。有很多冷冻酸奶配方(也有一些不错的商店购买的选择(如)但我喜欢烹饪光的 [这种配方](https://www.cookinglight.com/eating-smart/nutrition-101/no-cook-frozen-yogurt-recipe) 的修改版(我使用更多蜂蜜和更高脂肪的酸奶)。要自己制作，您需要:

*   两杯希腊酸奶(我要全脂的)
*   3 汤匙蜂蜜
*   一个香草荚或一茶匙香草精的含量
*   2 汤匙重奶油

将除重奶油以外的所有材料放入碗中，搅拌至完全混合。倒入冷冻袋中，轻轻地将混合物推到袋的边缘，在密封之前尽可能多地挤出空气。把它平放在冰箱里， [冷冻，直到冻结](https://theconcourse.deadspin.com/rainbow-cake-recipe-inspires-comment-apocalypse-1592575661) (至少三个小时)。

将混合物打碎，和浓奶油一起放入食品加工机，让它旋转，直到混合物完全变得光滑。立即将酸奶面糊刮入面包盘中，盖上保鲜膜，将保鲜膜压在酸奶表面。把它放回冰箱再冷藏几个小时，然后舀起来，和含糖的谷类食物(如果你觉得淘气)或水果(如果你觉得善良)一起食用。
