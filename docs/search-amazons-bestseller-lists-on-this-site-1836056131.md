# 在这个网站上搜索亚马逊的畅销书排行榜

> 原文：<https://lifehacker.com/search-amazons-bestseller-lists-on-this-site-1836056131>

[【搜索畅销书】](https://www.searchbestsellers.com/) 是一个购物网站，帮助你在亚马逊上找到每个产品类别的畅销书排行榜。正如该网站解释的那样，虽然亚马逊保留了数百个类别中最受欢迎产品的列表，但它不允许你搜索这些列表。搜索畅销书可以。



与大多数搜索结果和亚马逊页面不同，畅销书排行榜不包含广告或付费广告。它们是基于销售额，而不是收视率(收视率更容易造假)。它们是从一个类别中挑选产品的最简单的方法，尤其是如果你没有时间去阅读像 Wirecutter 这样的推荐网站。

如果你不确定你要找的具体类别，这个网站会很有帮助。因为亚马逊将他们的产品细分为许多嵌套的类别，搜索一个类别可以帮助你找到另一个更精确的类别。搜索“徒步旅行背包”，你会找到最畅销的徒步旅行背包的链接，还会找到最佳徒步旅行背包、最佳内部框架背包和最佳外部框架背包的链接。搜索“蓝牙扬声器”，你会看到室内、室外和便携式扬声器的列表。

如果你在谷歌上搜索“亚马逊畅销书[产品类别]”，你会得到类似的结果，但搜索畅销书会在每个列出的类别中包括畅销产品的小图片，而不会包括任何可能让你犯错的类似名称的亚马逊页面。

有两个缺陷需要注意。第一，你不能轻易为这个网站做一个搜索关键词。第二，似乎有一个限速器，让你在搜索几次后停几分钟。让我们期待这个仍处于测试阶段的网站尽快解决这两个问题。由于广告、虚假评级、自动“亚马逊的选择”标签以及其他充斥搜索结果的废话，在亚马逊上找到最好的亚马逊产品变得越来越难。
