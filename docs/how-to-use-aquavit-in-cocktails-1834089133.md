# 如何在鸡尾酒中使用 Aquavit

> 原文：<https://lifehacker.com/how-to-use-aquavit-in-cocktails-1834089133>

如果我的丹麦之旅教会了我任何关于酒精的事情，那就是我没有喝足够的杜松子酒。如果说它教会了我第二件事，那就是我绝对没有喝足够的水。如果它(不知何故)设法教会了我另一件事，那就是我早就应该把这两种饮料混合在一起了。



Aqua vit，斯堪的纳维亚的“生命之水，”是杜松子酒更具侵略性、更美味的亲戚，我应该一直邀请他们参加聚会。就像杜松子对于杜松子酒，葛缕子和/或莳萝对于阿夸维特一样。但是与杜松子酒的杜松子酒不同，这些味道一点也不害羞。杜松子酒混合的地方，阿夸维特战斗。与其说是补充，不如说是削减。这使得它对很多事情都很重要，我现在会告诉你。

### 运送零食

你知道泡菜是如何用它们的对比味道来平衡油腻和咸味的吗？Aquavit 有点像泡菜，会让你喝醉。冰镇后饮用——将瓶子放在冰箱里——搭配烤肉(如猪肉)、奶酪和饼干，或任何其他咸味和奶油味的组合，或者完全北欧风味，搭配腌鲱鱼或其他罐头鱼享用。

### 冲洗你的杯子

如果你想进入莳萝和/或葛缕子风味的酒的世界，稍微清洗一下你的鸡尾酒杯。它和马提尼一起喝效果很好——尤其是如果你喜欢脏马提尼的话——但是 aquavit 和柑橘，尤其是葡萄柚一起喝效果惊人的好。只需倒入四分之一盎司，在玻璃杯周围旋转，倒出多余的部分(也许倒进你的嘴里)。

### 顶部加滋补品

我在哥本哈根喝的最喜欢的饮料无疑是我在喝的 aquavit and tonic，里面有冰块中的康尼康宁片。我啜饮着，心想"*这种*才是美味的鸡尾酒。"这种饮料让我想起了咸的食物，而不是用熏肉脂肪洗涤剂或其他一些废话打我的头。泡菜片——没有添加盐水——只提供了一点盐和酸。这是我喝过的最平衡的两种成分的饮料之一，你可以把 1.5 盎司的 aquavit 和 3 盎司的 tonic (Fever Tree 是不错的)混合在一个加冰的杯子里。C 欧尼康可选。

### 就当是“愤怒的杜松子酒”

杜松子酒能做的任何事情，aquavit 都可以用更多的态度来做(还有香菜和莳萝)。除了滋补品和马提尼酒，在骡子(只需加入姜汁啤酒)、简单的里基酒(T2)、甚至果味的提基酒(T6、T7)中饮用也很棒。你的味觉可能需要一段时间来适应更向前的味道，但这是一条值得学习的曲线。
