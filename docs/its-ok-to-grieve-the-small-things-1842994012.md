# 为小事悲伤是可以的

> 原文：<https://lifehacker.com/its-ok-to-grieve-the-small-things-1842994012>

Watch

在过去的几个月里，我们都经历了一个快速变化的局面，有很多大大小小的珍贵的东西，我们不得不放弃。无论是取消生日派对、舞会、婚礼、千载难逢的旅行、失去生计还是处理对亲人的担忧，过去几个月都有巨大的压力、担忧和焦虑。大大小小的牺牲不计其数。

伴随这一切而来的是一种悲伤感。正如悲伤专家大卫·凯斯勒(David Kessler)在接受《哈佛商业评论》 采访时说的那样，“常态的丧失；对经济损失的恐惧；失去联系。这对我们打击很大，我们很悲痛。集体。我们不习惯这种空气中的集体悲痛。”

### 为小损失悲伤

心理学家罗宾·古尔维奇博士说，即使是一些小事情——取消的舞会，错过的告别——悲伤也是正常的。杜克大学医学中心的精神病学和行为科学教授。

“我们仍然可以为我们的损失感到悲伤，并对生病的人感到同情和同情，”古尔维奇说。

Gurwitch 指出，除了对我们所爱的人生病的主要担忧之外，我们还担心一系列次要的不利因素。我们中的一些人失业了。一些人失去了他们花了多年时间建立起来的小企业。其他人的工作使他们处于危险之中，无论是在医院还是在杂货店工作。

我们所有人都在努力探索一个新的世界，一个看起来与几个月前大不相同的世界。这包括大的变化，也包括小的变化。

“有这么多层，”古尔维奇说。

### 有哪些东西你仍然可以珍藏？

在所有这些混乱中，为小事悲伤是可以的，在小事中寻找快乐也是可以的。

作为一个例子，Gurwitch 在爆炸发生时在俄克拉荷马城开始了她的心理学家生涯，她指出，鉴于所发生的一切，父母们觉得为他们的孩子举办派对是不合适的。

但是，正如她指出的，生日聚会对一个小孩来说可能意味着一切，即使是在难以想象的损失的背景下。同样的道理也适用于此刻发生在疫情中的小快乐。

"在最糟糕的时候，你还有什么可以珍惜的东西吗？"古尔维奇说。

### 如果你在挣扎，寻求帮助

Gurwitch 小心翼翼地指出，与普遍的看法相反，没有杀死我们的东西并不能保证让我们变得更强大。对于那些在这一切发生之前就在挣扎的人来说，这种情况的压力很可能会加剧任何已经存在的问题。对于幼儿来说尤其如此。如果你在挣扎，如果你对正在发生的事情感到不知所措，那么寻求帮助尤为重要。

“没有人应该独自完成这项工作，”古尔维奇说。
