# were the sklar brothers and this is how we work 1835075256

Sklar 兄弟是当今喜剧界最著名的兄弟表演，他们以在各种媒体上的工作而闻名:单口喜剧(包括六张专辑)；像*廉价座位*这样的电视节目以及在*体育中心*和*的亮相最好给索尔*打电话；还有他们的播客 [*Sklarbro 乡村*](https://www.earwolf.com/show/sklarbro-country/)[*观自便宜席*](https://starburns.audio/podcasts/view-from-the-cheap-seats/) ，还有 [*哑民镇*](https://starburns.audio/podcasts/dumb-people-town/) 。我们和他们谈论了他们如何创作一集，他们如何写作和磨练他们的单口相声，以及他们在现场表演中打出其他喜剧演员的笑话。

#### 首先，告诉我们一些你的背景，以及你是如何走到今天这一步的。

我们在密苏里州的圣路易斯长大。小时候，我们是喜剧的超级粉丝:飞机、T2、燃烧的马鞍、莱特曼，然后是像宋飞、山德林、SNL 这样的漫画。我们像海绵一样吸收一切，记住晦涩的喜剧演员的表演，当我们父母的朋友在我们家的地下室玩桥牌时，我们会向他们滔滔不绝地讲述。作为双胞胎，我们得到了很多关注，我们想，如果我们利用这种关注让人们发笑，那该有多好。

我们从高中开始表演单口相声，并在密歇根大学的大学里继续进行。我们都是英语专业的学生，在大学里学会了如何写作，这在我们为电视写作时帮助了我们。

我们在 1994 年搬到纽约，并一直做单口相声。三年后，我们在 MTV 上做我们自己的节目。2F ，一部融合了小品和单口相声的混合型情景喜剧。这很棒，它把我们和喜剧界的许多伟大的人联系在了一起，比如扎克·加利费安纳基斯、史蒂芬·科拜尔、艾米·波勒、杰夫·罗斯、帕顿·奥斯瓦尔特、大卫·韦恩、迈克尔·肖沃特以及其他许多了不起的人。

1999 年，我们搬到了洛杉矶，几年后，我们在 ESPN 经典频道(new York Classic)上看到了《廉价座位》,这部剧一共播放了四季 77 集。从那时起，直到今天，我们一直在电影、电视、单口相声以及现在的播客领域坚持不懈地努力。

#### **带我们度过最近的一个工作日。**

起床，送孩子上学，然后我们开始新的一天。我们聚在一起写一些东西。我们通常在 9:30 左右在兰迪家聚会，一直工作到 3 点，也就是去学校接孩子的时候。然后我们和他们在一起直到晚上，那时我们会让他们上床睡觉，然后出去表演单口喜剧直到凌晨。

我们会为即将到来的播客写提纲，或者写一些我们正在播放的单口相声。我们现在有两个处于不同开发阶段的电视项目，所以我们将在白天与我们合作的作家会面，以完成这些项目。如果我们有播客要录制，我们会去 Starburns Industries，在那里我们录制，并录制许多播客中的任何一个:要么是我们的体育/喜剧播客的*视图，要么是我们的每周探索和庆祝席卷我们社会的无声浪潮的*哑巴镇*。如果我们在制作一部电视剧或电影，我们的工作时间当然会很长很长，我们会一整天都在片场。*

#### 你练习单口相声的过程是怎样的？

我们写单口相声就像乐队写歌一样。通常，我们中的一个人有一个想法的核心，一个概念，我们会把它带给另一个人，看看它有什么好笑的，我们在前提方面挖掘了什么重要的真理。然后我们写了大约 40%。

然后我们把我们所拥有的带到洛杉矶的舞台上，展示出来，看看什么可行，什么不可行。也许我们会和有趣的朋友一起帮助即兴创作，并进一步冲洗出来，但是我们越是把它放上去，我们就越能看到我们在哪里得到了最大的笑声，并努力把这些写进去。

然后，最后，经过多次表演，我们真的觉得我们有所收获。当它在每一个环境中都能得到笑声时，它就成为新系列中的一个永久的主题，当我们继续这样做时，我们可能会添加一两个标签。

#### 你的每一期播客都经过了什么过程？

我们了不起的粉丝们给丹·范·柯克发送疯狂的故事，他是我们在哑吧镇的共同主持人。他们在推特上给他贴标签，他选择第一个发给他的人。丹把故事分解开来。我们通过联系我们出色的喜剧朋友来预订客人，我们的工作室制作人 Noah Eberhart 帮助我们组织。

当我们走进演播室时，丹是我们中唯一一个通读故事的人，所以我们的反应和与客人的即兴表演是真实的，现场的，未经排练的。我们喜欢这个想法，事实上，我们对这个愚蠢的世界的反应被这个播客捕捉到了。这部喜剧速度快，激情四射，非常有趣。

从便宜的座位上观看比赛，我们试图把本周最大的体育新闻带给我们。无论是美国职业棒球大联盟上座率的下降，还是将勒布朗在湖人队的任期与塞缪尔·L·杰克逊在首都一号的商业广告进行比较，我们都试图以一种热情，(滑稽的)承担一个只有我们才能承担的体育世界的大话题。

<label class="bxm4mm-13 juykRM">Advertisement</label>

然后我们预定一位体育嘉宾，像布雷克·格里芬这样的运动员，或者像鲍勃·科斯塔斯这样的麦克风背后的人，或者像*汤*和*勇敢向前冲*的约翰·汉森(尼克斯队的超级球迷)这样对体育非常了解的喜剧演员，我们采访他们或者和他们谈论体育。然后，我们在体育世界中找到三个更小、更古怪的有趣故事，并把它们分解开来，与顾客进行讨论。最后，我们请一位出色的配音演员留下语音邮件，或者我们采访他们，如克里斯托弗·沃尔兹、奥齐·吉兰、泰格·伍兹、比尔·沃顿或尼克·凯奇，让我们有机会通过伟大的性格即兴演员的思维来拍摄当前的体育景观。

#### 与你的兄弟合作对你的工作流程有什么影响？

我们真的很擅长为对方收拾残局，给对方我们需要的休息时间，和家人在一起，或者照顾个人义务。成为团队的一员意味着永远不要记录你和其他人做了什么。只是“为了二人组好，完成它。”

我们总是说作为一个团队最好的部分是我们可以分享钱。

我们彼此都很了解对方，在代表球队的问题上，我们彼此都很信任。

#### 帮助你完成工作的人是谁，你如何依赖他们？

诺亚·埃伯哈特是我们播客的制作人，他帮助我们协调嘉宾并处理我们非常分散的播客时间表。Starburns 的詹森·史密斯帮助我们塑造了我们的播客，建立了一个强大的粉丝社区，并为我们创造了新的机会。我们依靠我们的共同主持人丹·范·柯克来帮助组织节目和粉丝互动，最重要的是成为每场节目创作和执行的合作伙伴。我们依靠我们的经纪人 TJ Markwalter 为我们争取现场表演的机会。

#### 你最喜欢的副业是什么？

我们开始做一个名为 Tag It 的脱口秀节目。我们预定我们的滑稽朋友，他们是漫画家，当他们在做他们的布景时，我们在后台狂热地为他们的笑话写标签。我们让他们在布景后留在舞台上，给他们推销我们为他们的素材制作的所有标签，并与他们一起即兴创作，经常带着这些素材来到新的地方。对我们来说，这是一个超级有挑战性的节目，因为从来没有停顿过，但当一些东西在音高和重复片段上发出咔嚓声时，这是如此有趣和有益。

#### 你收到过的最好的建议是什么？

几年前，在我们的 MTV 秀之前，乔恩·斯图尔特和我们一起去打保龄球，他对我们说:永远要确保你犯的错误是你自己的，你不会因为别人的错误而倒下。你可以处理好自己的缺点，但为别人的错误判断承担责任是很难接受的。
