# 你可以带一只小狗乘坐这些航班

> 原文：<https://lifehacker.com/you-can-bring-a-puppy-on-these-airlines-1831553124>

如果你有一只新的小狗，正在考虑乘飞机旅行，你可能也想开始寻找宠物保姆。航空公司已经开始禁止幼犬登机。最新做出改变的是:联合航空。从周一开始，美联航将不再允许小于四个月大的宠物进入机舱。



政策的变化影响了情感支持动物、舱内宠物和服务动物，因为“四个月以下的动物通常没有接受必要的疫苗接种”，以确保乘客和机组人员的安全。

达美航空上个月也做出了类似的政策改变。

虽然最好的策略是在最初的几个月里把你的宠物留在家里(说真的，这对宠物来说是一个压力很大的情况，而且 [甚至可能是危险的](https://www.avma.org/public/PetCare/Pages/Short-nosed-Dogs-and-Air-Travel-FAQs.aspx) )，如果你绝对需要和你的新毛茸茸的朋友一起飞行，你仍然有几个选择。以下是仍然允许携带小狗的航空公司:

### 美国航空公司

美国仍然允许小狗进入机舱。也就是说，小狗需要八周大才能乘坐美国航空在美国和波多黎各旅行，在美国旅行时需要 16 周大。

该航空公司也不接受“任何‘组合’的短头或扁鼻狗作为托运宠物。”你可以在这里查看 [限制品种的完整列表](https://www.aa.com/i18n/travel-info/special-assistance/pets.jsp) ，但这些品种包括牛头犬、拳师犬、波士顿梗、北京狗和狮子狗。

### 西南航空公司

西南航空公司目前 [允许任何年龄的宠物](https://www.southwest.com/pets/) 乘坐其航班，但它们必须被放在一个能放在你面前座位下面的载体里。

根据先到先得的原则，允许携带宠物登机；然而，任何一次航班只允许携带六只宠物，并且每人只允许携带一只宠物。

### 阿拉斯加航空/维珍美国航空

阿拉斯加州允许小狗以及猫、狗、兔子和家禽进入客舱，但狗和猫[**必须至少八周大**](https://www.alaskaair.com/content/travel-info/policies/pets-traveling-with-pets) 并且已经“完全断奶”

阿拉斯加每次航班只允许五只宠物进入客舱，所以早点预订还是划算的。您的宠物也必须在一个载体中，并在飞行期间呆在载体中。

### 边界

Frontier 允许任何年龄的宠物在机舱内飞行，只要它们适合放在载体中，并放在您前面的座位下。

宠物 [需要在](https://www.flyfrontier.com/travel/travel-info/family-pets/?according=traveling%20with%20pets)号航母上停留飞行的时间。就像其他航空公司一样，你必须提前为你的狗狗预定一个位置，以确保它的位置。
