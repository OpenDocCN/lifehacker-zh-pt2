# 一起跑腿，花更多的时间和朋友在一起

> 原文：<https://lifehacker.com/spend-more-time-with-friends-by-combining-errands-1838909277>

如果你到了一定的年龄，你可能还记得和朋友逛商场是多么有趣。今天，你的购物之旅可能更多的是食品杂货、家居用品和特价冬装——但这并不意味着你不能邀请你的朋友一起去。

Watch

有几个原因可以解释为什么出差和朋友约会结合得如此之好。首先，它让你有机会和你关心的人在一起，同时也把一些事情从你的任务清单上划掉。(理想情况下，你的朋友也能从他们的任务清单上划掉一些差事。)

其次，任何曾带伴侣或孩子去 Safeway 或 Home Depot 旅行的人可能都知道，你可以在车里或第三通道进行意想不到的深入交谈。出差让你接触到很多极限空间——高速公路，长得离谱的结账队伍——给你时间思考，也给你时间分享你的想法。

与你典型的“让我们一起喝杯咖啡”的朋友聚会相比，你被置于一种迫使你盯着对方并试图想出一些话来说的情况下。(“所以...我们打篮球的天气怎么样？”)我们中的一些人非常擅长这种互动；其他人觉得它们很无聊。但是，如果你站在超市里，试图在三种不同的下水道清洁剂中做出选择，告诉你的朋友你需要下水道清洁剂，因为你的长发青春期前开始洗澡更频繁了，这可以引发一场关于育儿、衰老和青春期导航的谈话。

此外，和朋友一起跑腿可以缓解和你的伴侣、孩子或自己(取决于你目前的情况)一起跑腿的压力。没有对新玩具或含糖谷物的抱怨，没有关于预算的讨论，没有在长得离谱的结账队伍中盯着你的手机，因为你没有人可以交谈。相反，你将有机会与能够提供外部视角的人分享对你来说重要的东西——你将有机会了解他们的生活，并提供你的支持作为回报。

所以为什么不发短信给你的闺蜜，问问她们想不想去购物呢？毕竟，这在我们中学时代是有效的——在我们成年后也是有效的。我们 Lifehacker 的一位编辑最近评论说，与朋友一起去杂货店购物比独自购物时与他们来回发短信更能提升她的情绪，我还记得我们过去住在同一个镇上时与我姐姐在塔吉特百货闲逛的美好回忆。(上个月我拜访了我姐姐和她的家人，一次目标旅行是经历的一部分。)

此外，你的朋友可能是说服你进行冲动购物的最佳人选。
