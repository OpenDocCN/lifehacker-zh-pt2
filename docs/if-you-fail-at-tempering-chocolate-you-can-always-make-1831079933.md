# 如果你不能调和巧克力，你总是可以做松露

> 原文：<https://lifehacker.com/if-you-fail-at-tempering-chocolate-you-can-always-make-1831079933>

上周末，我在做薄荷树皮，这个网站称之为“难以置信的简单”， 但是我搞砸了。我在羊皮纸烤盘上倒了一层调和得很好的黑巧克力，然后在白巧克力中加入薄荷油。正当我要把白巧克力倒在黑暗中的时候，我注意到了一些东西:液体。



当水进入巧克力时，巧克力将不再一样。(是的，就我们这里的目的而言，白巧克力也算巧克力。)我搅拌着混合物，希望我的眼睛是在欺骗我，但它卡住了。白巧克力不是光滑的、融化的美味，而是曲奇面团的质地。它变成薄荷树皮的梦想破灭了。

被查封的巧克力永远不会倒出来，也不会正常变硬，但吃起来还是很惊艳。所以我把这些东西放进搅拌碗，加入一点点重奶油。(对于 20 盎司的巧克力，我目测了大约四分之一杯的奶油。)这就形成了 ganache，这种奶油质地的东西是块菌的绝佳馅料。如果你喜欢的话，可以加点料，让它更有活力:我在薄荷树皮上加入了一些糖果。

将混合物冷藏以帮助其变硬，然后将混合物舀成小球。你可以将松露撒在碎糖果棒、磨碎的坚果或可可粉上。或者，如果你够勇敢，调和另一批巧克力，把它们浸在里面涂上一层。这次祝你好运！
