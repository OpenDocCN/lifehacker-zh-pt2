# 试试这些稳定性核心球练习

> 原文：<https://lifehacker.com/try-these-stability-ball-core-exercises-1832654182>

是时候表达一点感激了。每当我做本周的核心练习时，我都非常感激我没有做厚木板 。即使动作很难，也没关系，因为至少它们不是无聊的*。*



所以今天我们给你带来一些你可以用健身球(稳定性球，瑞士球，随便你怎么称呼它)做的动作。大多数健身房都有一个或几个球，如果没有，这是一个便宜的投资和方便的东西。如果你有小孩，他们会试图用它把对方撞倒。这是正还是负由你决定。

所以这里有一些你可以尝试的事情，从最难的开始:

### 稳定性球棍

这从看起来像平板的姿势开始(手放在地上，身体平放)，但是你的脚掌在球上。然后你把屁股翘到空中，保持双腿伸直，脚放在球上。这很难。很疼，你可能会摔倒，而且基本上也是倒立练习。(还记得我们做倒立时的 [吗？](https://vitals.lifehacker.com/let-s-get-upside-down-for-the-august-fitness-challenge-1828029407) )所以自然这一个是我的最爱，虽然我勉强能做到。

### 稳定性球膝盖抱膝

这是上面的更简单的版本，如果有一天你想做屈体，你应该先掌握这个。从同样的位置开始，但是要一直保持背部平坦。弯曲你的膝盖在你下面滚动球，然后伸展你的腿回到起始位置。

### 传球

你可以不用球来做这个，但是用球来做更有趣。仰卧时，将球夹在两腿之间。保持双腿伸直，将球举到空中，用手抓住。再次躺下。然后把球从手上传回到脚上，重复。

### 侧球升降机

这是你可以用两腿之间的球做的另一个动作。侧躺，挤压斜肌将球(和你的腿)抬离地面。

### 仰卧起坐

哦，是的，如果你只是想做仰卧起坐，你可以在球上做。只要把你的上半身盖在球上，你会发现仰卧起坐现在更难了。这意味着你可以用一套，比如说，10 个而不是 50 个来达到燃烧腹部的感觉。
