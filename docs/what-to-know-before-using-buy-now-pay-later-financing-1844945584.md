# 用先买后付融资前知道什么

> 原文:[https://life hacker . com/what-to-know-before-use-buy-pay-la ter-financing-1844945584](https://lifehacker.com/what-to-know-before-using-buy-now-pay-later-financing-1844945584)

自大萧条以来，先买后付的概念就一直存在。分期付款购物在 20 世纪 80 年代基本消失了，它包括在你把东西带回家之前支付少量的定金和分期付款。尽管在大萧条期间，分期付款 [短暂地卷土重来](https://www.bankrate.com/finance/personal-finance/layaway-makes-a-comeback-1.aspx) ，但它已经被更高成本的融资选择所取代，比如信用卡。

现在，购物者有了另一个选择:现在购买，以后支付服务费用。与分期付款最大的不同是你不必等着把你的商品带回家，这可能是这些服务越来越受欢迎的原因。在店内收银台或网上购物时，你可能会看到现在购买，以后支付的公司，如 Affirm，AfterPay，Klarna，QuadPay— [和现在的 PayPal](https://techcrunch.com/2020/08/31/paypal-joins-the-buy-now-pay-later-race-with-new-pay-in-4-installment-program) 。

根据一份的新报告显示，超过三分之一的美国人习惯现在购买，以后付款。在疫情 期间，这些服务已经成为网上购物 [的特别需求。问题是，大多数购物者不了解这些服务是如何运作的。在使用“现在购买，以后付款”之前，您应该了解以下内容。](https://fortune.com/2020/08/31/paypal-buy-now-pay-later-financing-covid-19)

### **先买后付的好处**

现在购买，以后支付服务越来越受欢迎有几个原因。Ascent 的信用卡专家和行业分析师内森·汉密尔顿(Nathan Hamilton)表示:“现在购买，以后支付服务的最大吸引力在于，你可以获得无息支付计划，而且你不需要良好的信用来获得批准。”“它们也非常方便——在结账过程中，你可以在几分钟内设置好一切，”他说。

### **现在购买，以后付款**可以让您花费更多

虽然现在购买，以后支付可能会提供一些额外津贴，但也有一些缺点。最大的问题可能是花费超出你的预期。根据一份 [2019 Klarna 报告](https://www.klarna.com/knowledge/articles/how-klarna-helped-gymshark-to-increase-basket-size-by-33) 显示，这些服务使 Gymshark 的购物量增加了 33%。

“尽量不要养成使用这些服务的习惯。汉密尔顿建议:“如果你真的需要为购物融资，它们会很有帮助，但只要有可能，最好避免为日常购物负债。”

### **留意昂贵的费用和利息**

另一个不利因素是:潜在的费用和利息。根据“先买后付”公司和你的信誉，你可能会支付费用、利息或两者兼而有之，以方便融资。

例如， [根据你的信用评分向](https://www.affirm.com/business/apr-calculator) 收取 10%至 30%的年利率，而 [Klarna 对每月最低 2 美元的标准购买收取 19.99%的年费。

虽然有些公司可能会在特定时间内提供 0%的年利率，但一旦](https://www.klarna.com/us/customer-service/what-is-financing-and-how-does-it-work) [促销期结束](https://twocents.lifehacker.com/beware-same-as-cash-payment-offers-1836048918) ，你可能会支付更多。汉密尔顿警告说:“如果你坚持按时付款，现在购买，以后支付效果很好，但如果你不这样做，你可能会被收取费用和利息。”

为了避免麻烦，汉密尔顿建议审查付款时间表——包括确切的付款额、到期时间以及你必须在多长时间内付清购物款。“检查滞纳金和利息有多少也是值得的，因为这可能是不错过付款的巨大动力，”他补充道。
