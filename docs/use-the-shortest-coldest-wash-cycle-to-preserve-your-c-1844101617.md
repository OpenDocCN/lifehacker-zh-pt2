# 使用最短、最冷的洗涤周期来保存您的衣物

> 原文：<https://lifehacker.com/use-the-shortest-coldest-wash-cycle-to-preserve-your-c-1844101617>

当你买一件衣服时，通常的想法是尽可能让它看起来新。(这显然不适用于运动衫和睡眠 t 恤之类的东西，它们只会随着年龄和多次使用洗衣机而变得更好。)但是洗衣服是必要的，我们大多数人不像 90 年代情景喜剧中的人那样经常把衣服送到干洗店。我们知道反复洗涤不利于长寿，那么你如何保持你的衣服干净并且看起来相对较新呢？根据的一项新研究，这归结于你洗衣机上的时间和温度设置。



### 如何让衣服看起来很新

这项发表在《染料与颜料》杂志上的研究是由英国利兹大学和宝洁公司(宝洁公司 [拥有主要洗衣液品牌](https://us.pg.com/brands/#Fabric-Care) 如汰渍、增益、柔和和欢呼)的科学家完成的。现在，任何时候一个公司或专业组织资助和/或参与一项研究，都有理由持怀疑态度。在这种情况下，重点是洗衣机如何使织物喷洒微纤维，从而导致你的衣服看起来有点不适合穿。目的是找出哪个洗涤周期造成的损害最小。

事实证明，这是最温和，最短和最冷的洗涤周期。(这个你大概自己也猜到了。)你可能不知道，自 2005 年推出第一款冷水洗涤剂 以来，P&G 一直是该领域的领导者。但是没有什么能阻止你利用这些知识去购买另一个品牌。

### 环境冲击

微纤维难题的另一个方面是，在每次洗衣过程中都会释放出成千上万的微纤维，然后被排入下水道。在那之后，许多这些微小的纤维进入海滩和海洋，在那里它们会对当地的生态系统造成一些破坏，特别是当它们被海洋生物和其他海洋生物吞食时。此外，能够保留你的衣服更长时间意味着你不需要扔掉旧衣服(它们最终会被扔进垃圾填埋场)并购买更多的东西作为替代。

最终，研究人员发现，使用更快、更冷的周期 [洗衣服可以减少释放到环境中的微纤维](https://www.leeds.ac.uk/news/article/4524/quicker_and_cooler_is_best_for_clothes) 的数量高达 52%，减少释放的染料数量高达 74%。
