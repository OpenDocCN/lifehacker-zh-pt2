# 你可以选择不使用面部识别登机

> 原文:[https://life hacker . com/you-can-opt-out-using-face-recognition-to-board-a-fli-1836139521](https://lifehacker.com/you-can-opt-out-using-facial-recognition-to-board-a-fli-1836139521)

几个月前，在亚特兰大转机时，我看到了不寻常的一幕:面部识别被用于登机程序。

Watch

这项技术目前正在对离开机场的国际达美航班进行测试，它可以取代登机口工作人员在登机前对你的护照进行物理检查。当你要上飞机的时候，你会站成一圈，看着相机，然后等待绿灯，表示技术人员已经确定你是你所说的那个人，应该在飞机上。登机口工作人员会对你竖起大拇指，然后你会从机器上拿到收据，然后登机。

这意味着在这种情况下，你的脸就是你的登机牌。登机不需要再掏出机票或者护照，只需要看镜头。

除了实际的登机过程之外，如果你乘坐的是经过测试的航班，扫描仪还会在你办理登机手续以及接受美国运输安全管理局安检时发挥作用。去年年底，积分专家 [在第一次运行中尝试了一下](https://thepointsguy.com/news/i-just-experienced-the-future-of-boarding-a-plane-heres-what-i-learned/) 。

最终，这一想法是面部识别可能会完全取代与 TSA 代理的互动。

虽然这项技术可能会让事情变得更快，但它也涉及到让政府机构扫描你的脸，这可能会让你觉得不舒服。如果你不能接受，知道你有选择是很重要的。

我看到的大门旁边的技术广告标志指出，该技术的使用是可选的，并建议如果您对该过程有疑问或担忧，请咨询大门代理。事实证明，关于它是自愿的信息对其他人来说可能并不明显。

本周, *Wired* [发布了一个来自记者](https://www.wired.com/story/opt-out-of-facial-recognition-at-the-airport/) 的故事，该记者选择退出底特律的服务。就她的情况而言，她是在登机口而不是在办理登机手续时发现这项功能的，因为她是在另一个机场开始旅程的。虽然这个标志对我来说是最重要的，但对她来说却不是。

这使得这成为一个了解你的权利和知道你有选择的简单案例。当你办理国际航班登机手续时，要注意确保你的航班不是使用生物特征扫描的航班(在办理登机手续时，你会被要求选择加入)，如果你确实在登机口看到这种情况，而你不感兴趣，告诉登机口代理人你宁愿用传统的方式处理事情。
