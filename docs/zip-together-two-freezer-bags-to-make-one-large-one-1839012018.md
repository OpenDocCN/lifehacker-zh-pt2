# 把两个冷冻袋拉上拉链，做成一个大的

> 原文：<https://lifehacker.com/zip-together-two-freezer-bags-to-make-one-large-one-1839012018>

冷冻袋非常适合储存各种东西。这周我发现了一些关于它们的新东西: ***你可以把它们两个连在一起*** 。



Buzzfeed 在本周 的一篇帖子中指出了这个想法，以及许多其他伟大的想法，其中许多我们已经报道了多年。

它的工作方式非常简单:只需将一个较大的物品放入一个袋子中，然后将第二个袋子翻过来。这样做，你就能把两个包用拉链拉在一起，基本上就成了一个巨大的自封袋。

显然，你的 frankenbag 在结构上要比一个单独的自封袋差一点，而且根据你放进去的东西，它可能在密封和打开上都有点问题。

也就是说，如果你需要一点额外的空间在那个冷冻袋里放一大块肉或另一件东西( [)最初的 Reddit 海报密封了一个电脑键盘](https://www.reddit.com/r/lifehacks/comments/94qhd4/you_can_connect_two_ziplock_bags_by_flipping_one/) )，那么知道在一个选项中将两个密封在一起会很棒。
