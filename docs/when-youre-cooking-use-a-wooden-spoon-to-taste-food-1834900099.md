# 当你做饭时，用木勺品尝食物

> 原文：<https://lifehacker.com/when-youre-cooking-use-a-wooden-spoon-to-taste-food-1834900099>

即使你在遵循一个经过检验的真正的食谱，你仍然应该在烹饪时品尝你的食物。在烹饪过程中会出现一系列的其他变量，你需要知道锅里发生了什么。你的嘴会告诉你什么是什么。然而，金属勺子是一个潜在的危险。



这不是因为金属勺子不能把食物送到你的嘴里——它们可以——但它们比木头(最好的勺子材料)制成的勺子更容易烧焦食物。木勺不会以同样的方式保持热量，也不会像金属勺那样快速升温，因此是一个更安全的选择。

“硅胶之类的呢？“你无疑是一个 sking 。我发现硅树脂的滑溜溜的特性使得它不善于抓住食物，当你试图把所说的食物送到嘴里时，这很烦人。(但是如果你唯一的选择是塑料或金属，那就用塑料吧。)

虽然用木勺品尝会大大降低烫伤嘴的几率，但并不能完全消除危险。不管你用什么舀，沸腾的液体都是沸腾的液体，所以运用常识，也许等一两秒钟再把它倒在舌头上。舌头烧伤极其不方便。
