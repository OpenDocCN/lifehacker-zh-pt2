# 使用防水布和烹饪喷雾来处理多余的雪

> 原文:[https://life hacker . com/use-tarps-and-cooking-spray-to-deal-over-snow-1841575488](https://lifehacker.com/use-tarps-and-cooking-spray-to-deal-with-excess-snow-1841575488)

一年中的这个时候，假日冬季仙境已经被可怕的冰冻泥浆所取代，所以这里有一些今年冬天清除积雪的最佳技巧。

让我们从雪铲开始。虽然它在铲掉前台阶和人行道上的积雪方面做得很好，但它往往会粘上大量的雪，使它无法被铲掉。为了防止这种情况，在出门前用烹饪喷雾喷洒铲子。它会让雪不粘，给你更多的铲子。

当我们在门廊上时，试着在下雪前铺上防水布。用砖或石头固定住它，让它留到暴风雨来临的时候。一旦完成，只要把防水布滑开，抖掉所有的雪——不需要铲或刮！

当你出去铲雪时，在你的普通手套上戴上一次性乙烯手套，制作你自己的防水手套。用橡皮筋把它们绑在你的手腕上，让你的手保持干燥和温暖。

然后走向你的车。用长筒袜保护你的挡风玻璃雨刷不被冻住。然后把你的后视镜包在塑料袋里，用橡皮筋固定以防结霜。当你在雪后回来时，简单地脱下袜子和包就上路了。

最后，如果你有通风的窗户(特别是在刮风的暴风雪期间)，切下小块的搁板衬垫，塞进缝隙里。这将防止融雪的气流和水滴进入你的窗户。
