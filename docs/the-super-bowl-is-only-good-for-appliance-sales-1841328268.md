# 超级碗只对家电销售有好处

> 原文：<https://lifehacker.com/the-super-bowl-is-only-good-for-appliance-sales-1841328268>

如果你像我一样是一个“体育迷”，那么你会很乐意在周日做除了看球以外的任何事情(或者我假设欧洲人称之为 *soccer* )。

然而，我对超级碗的彻底鄙视不仅仅源于它的运动性——而是整个体验。这个大游戏试图用其华丽(浪费)的广告和华而不实(过度)的中场表演来吸引整个家庭。(我们真的需要聚在一起看的广告，听录制的音乐吗？这就是为什么我们有 Hulu 和 Spotify。)即使你的朋友坚持说“没人对游戏感兴趣！来就是了！”总会有一个*人让你嘘，这样他们就能专心地看。* 

因此，我建议在寒冷的周日晚上，不要参加超级碗，而是参加几项其他活动来占据你的时间。出去看一部当前奥斯卡提名的电影怎么样？或者去吃一顿美味的晚餐——任何非体育主题的场所都会空无一人。你真的会因为错过一场你在赛季剩余时间里都没有关注的比赛而感到难过吗？

好吧，你可能会认为我在没有好理由的情况下欺负体育。因此，出于对体育精神的兴趣，我将提供一些关于超级碗的好消息。首先，电视和烤架上经常有大减价，他们会让你购买与足球无关的东西。当然，这也是罗马数字唯一一次重新进入集体意识。所以某个地方的某个人可能会说“等等， *L* 是几号？”，然后他们会查找并学习*的一些东西*。

我不太清楚“49 岁”是什么意思，但对某些人来说，这听起来肯定是一种冒犯。
