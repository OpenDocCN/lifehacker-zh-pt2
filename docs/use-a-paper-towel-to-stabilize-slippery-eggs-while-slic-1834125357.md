# 切片时，用纸巾来稳定光滑的鸡蛋

> 原文：<https://lifehacker.com/use-a-paper-towel-to-stabilize-slippery-eggs-while-slic-1834125357>

煮熟的鸡蛋是一种近乎完美的食物，但它们很滑，而且取决于你的砧板有多光滑，当你试图用锋利的物体切片时，它们真的有滑动的危险。



如果你只切一个鸡蛋，受伤的风险非常低，但如果你切了十几个鸡蛋——比如做芥末鸡蛋——它们的滑溜会变得非常烦人，你滑倒割伤自己的可能性也会随着这种烦人的感觉而增加。

解决方法很简单:拿一张纸巾。就像纸巾可以防止你的砧板滑动，T2 可以防止你的鸡蛋滑动一样。在你剥了蛋并冲洗干净之后——我总是冲洗掉任何细小的蛋壳——把它们一个一个地放在干纸巾上。鸡蛋的水分会帮助它粘在毛巾上，稳定鸡蛋，防止它跑掉。安全地切片，在纸巾边缘擦刀以去除多余的蛋黄(这确保你的下一次切割是干净的)，重复直到你所有的鸡蛋减半。
