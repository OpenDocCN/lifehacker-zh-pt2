# 使用这个 Chrome 扩展向谷歌报告可疑网站

> 原文:[https://life hacker . com/report-suspective-sites-to-Google-using-this-chrome-ext-1835650376](https://lifehacker.com/report-suspicious-sites-to-google-using-this-chrome-ext-1835650376)

如果你使用 Chrome 有一段时间了，你可能会偶然发现它可怕的红色间隙——或者被称为谷歌安全浏览警告。如果你试图打开一个看起来有问题的网站， [就像一个恶意软件主机](https://transparencyreport.google.com/safe-browsing/overview) ，Chrome 会阻止你这样做，除非你稍微挖掘一下 [绕过它的警告](https://support.google.com/chrome/answer/99020?co=GENIE.Platform%3DDesktop&hl=en-GB) 。

Watch

虽然那个不能错过的红色拦截器应该可以帮助你阻止你访问你不想上的网站，但谷歌*也*提供了一些额外的工具，你可以用来检查一个网站的“安全浏览状态”，包括一个全新的 [可疑网站记者](https://chrome.google.com/webstore/detail/suspicious-site-reporter/jknemblkbdhdcpllfgbfekkdciegfboi) Chrome 扩展。

安装它，浏览器中的一个小图标会告诉你当前访问的网站是否看起来可疑。如果你同意——它的展示感觉有些不对劲，或者它试图在你的浏览体验中推送一些欺诈性的下载——你也可以使用该扩展为谷歌标记该网站。有了足够多的报告(我假设)，谷歌会更仔细地看看这个网站，如果它真的有问题，会考虑给它一个红色的警告。

换句话说，通过报告您访问的潜在不安全网站，您将帮助您的网民朋友在浏览时保持安全。请不要做一个混蛋，报告你不同意的网站，因为这很烦人，谷歌可能不会认为某人的脸书页面(或其他)是一个不安全的网站。不错的尝试。

除此之外，谷歌还提供了一个基于网络的小工具 ，你可以用它来检查你想要的任何网站的“安全浏览状态”。虽然大多数人可能只会等待他们的浏览器说“不要访问那个网站”——或者这个新的扩展在事情*出现*奇怪时给出建议——如果你想用另一种方法来验证谷歌是否对一个网站进行了安全检查，那么把这个工具放在你的后兜里也无妨。
