# 你真的需要预算

> 原文：<https://lifehacker.com/you-really-do-need-a-budget-1843440246>

预算是基础。这看起来像是个人理财 101 课，让我们面对现实吧，预算指南的书籍封面和营销似乎更针对中学生，而不是你，一个有能力的成年人，不知何故通常设法有足够的钱来买东西。

你可能觉得自己太超前了，不需要预算。你有退休账户！你自动转账你的账单！你有这个，对吗？

我知道你不喜欢这么做。但我是来告诉你无论如何要坐下来做。你需要一个预算。

*(注:我说的不是同名的书和 app，虽然有些 Lifehacker 的作者喜欢并活在* [*《你需要一个预算系统*](https://lifehacker.com/these-are-the-only-three-money-apps-you-need-1834720491) *》中。)*

如果你在钱的问题上是凭直觉行事，那么让我们来看看不做预算是如何伤害你的。

### 你会把你的健康减半吗？

假设你去医生那里做年度体检。一路上，你都在抱怨。“我不知道我为什么要烦恼，”你想。“我很健康！”

你来到医生的办公室，站在体重秤上，量体温和血压。护士对这些初始指标没有表现出任何关心，所以你跳下桌子。“我可以走了，”你告诉他们。“我很好。”

护士不能让你留下来，但是他们也不能告诉你当医生检查你的鼻子和喉咙，或者听你的心脏，或者摸你的肚子来感觉他们正在感觉的任何器官时，他们可能会注意到什么。

如果你没有做全面检查就离开，你就不能确定你是否真的健康。

你的钱也一样。仅仅因为你在下一个发薪日的支票账户上有正结余，并不意味着你在尽你所能管理你的钱。你需要对你的钱有一个更全面的看法，以确定什么是对的，什么是错的，什么可以稍微调整一下，让生活变得更好。

### 你需要积极对待你的钱

当你不确定你真的想知道你一个月在餐馆或啤酒厂花了多少钱时，很难把你的花费分类。也许看着你的儿童保育费用会让你反胃。也许看到你每月的学生贷款债务支付让你难过。

但是是时候坚持下去，组织起来了。如果你不这样做，你缺乏意识可能会回来咬你。

就在本周，一位读者评论说，他们煞费苦心地制定了一份预算，却在几天后发现，在疫情期间，他们的收入会下降。这是一个足以令人沮丧的理由，但这个人似乎也很沮丧，因为他们不得不通过调整预算来弥补收入损失。

听着，调整你的预算，找出你可以在困境中削减成本的地方，这不是一个超级有趣的夜晚。但是这位读者是幸运的:他们不是从零开始。这位读者没有盯着一张空白的电子表格，对自己的钱焦虑不安，而是有了一个基线，一个起点。这是一个调整预算的问题，而不是从头开始建立一个。

这就是你真正需要预算的时候:当事情变糟，你需要尽快制定计划。当你失业、生病或有意外开支时，不是坐下来说“今天是时候了，我要做个预算”的时候因为到那时，已经太晚了。到那时，无论外界压力如何支配你的花钱方式，你都会做出反应。

如果你在情况好的时候(或者至少是好的时候)做了预算，当意外发生的时候，你就采取了主动保护自己的措施。你正在制定一个计划，当最坏的情况降临在你身上时，这个计划会让你更容易调整。

### 如何做一个自己不讨厌的预算

制定一个不会让你感到恶心的预算的诀窍是，在脑海中想象一下你的预算是什么样子，然后把它扔掉。没有一个单一的电子表格来管理它们，也没有一个单独的方法比其他方法更好。

最好的预算是适合你的预算。这可能需要一些尝试和错误来找到一个对你个人情况有意义的。

我建议尝试一种预算方法不少于三个月，以获得对它是否适合你的生活方式的可靠感觉。

#### 让您开始的几个选项

[**50/20/30 法**](https://lifehacker.com/use-the-50-20-30-rule-to-outline-your-budget-for-every-1760532138) **:** 你把你的开销分成三大类:必需品(50%)、财务目标如储蓄或债务偿还(20%)和可自由支配的开销(30%)。

60/40 方法:你将 60%的钱用于必需品，40%用于其他东西，包括储蓄目标。

[](https://twocents.lifehacker.com/simplify-your-budget-with-the-kakeibo-method-1836252520)****:**你把团体支出分成四类:生存、额外、可选和文化。什么属于哪一类？用这种方法，自己做决定是一半的乐趣。**

**[**零基预算**](https://lifehacker.com/zero-out-your-accounts-every-month-to-enforce-a-zero-su-1591562485) **:** 你给每一块钱一份工作，争取月底有 0 块剩余/去向不明的艺术品。**

**如果你想尝试其中一种预算方式，你可以在网上搜索一些模板，让计算变得简单一些。**

**如果你用应用程序做预算，你会更少地关注预算方法，而更多地关注日常支出的。(应用程序爱好者，在评论中分享你的最爱，供寻找第一个预算应用程序的读者使用！)**

### **忍耐**

**关于做预算，你还需要记住另一件事:你会搞砸的。**

**我保证。你会忘记你一年只付两次汽车保险，直到那笔大的付款来临，或者你会忘记给孩子们的钢琴课分类。**

**那很好。因为预算不是你定了就能忘记的东西。这是你日积月累培养出来的。你会做出调整，你会做一份当前预算的副本来运行几个场景， [你会得到加薪，](https://twocents.lifehacker.com/what-to-do-first-when-you-get-a-raise-1839637903) 并且需要明智地分配这些资金。**

**你不会只想埋头苦干到下个月——聪明的预算者会回顾上个月，看看哪些可行，哪些不可行，同时展望下个月。**

**这都是关于积极主动的，如果你至少每个月检查你的钱，你的财务健康就很好。**
