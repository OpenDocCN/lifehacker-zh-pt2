# 我的设备应该使用哪个 Wifi 频段？

无线网络有点像你的汽车(或地震多发的加州住宅)的应急包。当它在那里的时候，你真的不会想太多，但是当你需要它的时候，你会注意到它不见了。此外，你要确保它的设置能给你最好的体验。

这正是我在本周的 [**Tech 911**](https://lifehacker.com/c/tech-911) 建议专栏中要解决的问题:找出一个简单但重要的家庭 wifi 设置，让你的设备能够以最佳速度连接。

我会让读者丹尼解释:

> *“有没有一种最佳的方法来将大量设备分配给家中的 2.4G 或 5G WiFi 频段？我理解范围/速度方面的考虑，但假设所有设备都是双频兼容的，在 5G 上路由器附近有 10 个固定设备和在 2.4G 上有 2 个移动设备是更好还是应该更均匀地分配它们(即每个上 6 个)或基于带宽需求(即 2.4G 上的回声点和恒温器，5G 上的嵌套摄像头/流媒体设备)，而不是通过固定设备和移动设备进行解析？”*

对于那些不太熟悉 wifi 的人，请允许我花点时间解释一下这是怎么回事。大多数现代路由器——如果它们不可怕的话——工作在两个频段:2.4GHz 和 5GHz。实际上，这意味着您的路由器可能会在您最初设置时要求您创建 SSID 和密码两个不同的无线网络。(有些路由器允许您对两者使用一个 SSID/密码，然后由您的设备决定它们是连接到 5GHz 还是 2.4GHz。)

我建议您为每个频段配备一个单独的 SSID，这样您就可以根据自己的需求分配设备。原因如下:

连接到 2.4GHz 频段的 wifi 网络，你可能会有更好的覆盖范围——你可以在尽可能远的地方观看视频，浏览网站，以及在你的设备上做任何其他事情。连接到 5GHz 频段的 wifi 网络，你会为了速度而放弃一些这个范围。这是您为您拥有的无线交流设备获得最佳可能速度的唯一方法(当与类似功能的 wifi 路由器配对时)，并且当您连接 wireless-n 设备时，该频段通常不太拥挤(wireless-n 设备也可以比 2.4GHz 连接更快的速度运行，这取决于它们的功能和它们与您的路由器的距离)。

明白了吗？换句话说。2.4GHz 代表范围，5.0GHz 代表速度— [a *非常*简化方式](https://gizmodo.com/why-your-router-has-two-wifi-channels-and-how-they-work-1828650288) 来看，这是一个不错的经验法则。

现在，为了解决你的问题的根源，我认为你的设备的带宽需求高于一切。例如，你可能可以将你的**智能恒温器和智能扬声器**连接到你的 2.4GHz 频段，因为我怀疑它们需要来回传递大量数据。**你的智能电视**？同样的道理，除非你做了很多流媒体(如网飞)。然后，我可能会选择 5GHz，以避免干扰并最大限度地提高速度——假设你的电视离你的路由器足够近。

你把一台**平板电脑**放在离你的路由器相当远的房间里？你可能需要 2.4GHz 才能获得最好的连接。(当我打这个的时候，我用我的无线交流笔记本电脑坐在一个 2.4GHz 的网络上，这让我很恼火，但是如果我连接到这个房子的 5GHz 网络上，我绝对没有任何东西可以工作。我离路由器太远了，帮不上忙。)

你的**窝摄像头**？你也许可以用 2.4GHz，因为它只是一个 1080p 的视频流。(这可能会消耗大约 5Mbps，这几乎可以忽略不计。)你的**笔记本电脑**？一路 5GHz，除非你发现你无法在远处获得良好的连接。你的无线交流智能手机也一样(如果你有的话)。

虽然这个建议听起来可能有点过时，但我会把 2.4GHz 用于那些无法从超高速连接中获益的设备，而把 5GHz 用于那些速度优先的设备。

有两个原因可以解释为什么你不会把所有东西都放在 5GHz 上，这*感觉*像是合乎逻辑的——这是一个允许你进入更快速度的频段。首先，有前面提到的范围问题。其次，连接到路由器 wifi 网络的设备越多，速度可能越慢。这并不是因为你只是简单地用*连接了*一堆设备，因为如果它们不主动传输大量数据，它们不会产生任何影响。然而，如果你有 20 个连接到 5GHz 的设备，并且其中一些正在积极地吞噬你的连接，这可能会导致其他一切的速度变慢。

或者，正如 [谷歌所说的](https://support.google.com/wifi/answer/6246489?hl=en) :

> *“每当一个设备连接到你的 Wi-Fi 网络，它都要和其他设备争夺带宽(网速)。电视流媒体网飞、笔记本电脑下载、Xbox Live 游戏会话，他们都希望获得最快的连接速度。但是带宽有限。您的带宽变得很窄，导致连接速度变慢。”*

而这里的 [是网件的](https://kb.netgear.com/24043/How-many-clients-can-you-connect-wirelessly-to-a-NETGEAR-router) 拍的:

> *"但是，由于路由器的无线信道由所有无线客户端共享，因此添加客户端将不可避免地导致所有客户端的网络访问速度变慢。如果一些客户端正在使用大量的无线带宽，例如观看视频或进行 torrent 下载，这将特别明显。因此，在连接到同一台路由器时，能够令人满意地运行的无线客户端的最大数量会因设备的用途而异。它还会根据路由器安装位置的无线拥塞或干扰程度而有所不同。”*

我怀疑当你使用无线 ac 路由器时，这不是一个大问题，因为你在典型的 AC1200+路由器上有比典型的 N300 或 N600 wireless-n 路由器多得多的带宽(至少 1.3Gbps)(在每个频带上最大 300Mbps，但在 2.4GHz 频带上通常只有 150Mbps)。许多联网的、数据饥渴的无线交流设备会降低所有设备的服务速度，但是对速度慢得多的 wireless-n 设备的影响不应该那么明显——因为它们本来就已经很慢了。
