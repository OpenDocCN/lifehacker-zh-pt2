# 如何观看苹果周一的 WWDC 2019 主题演讲

> 原文：<https://lifehacker.com/how-to-watch-apples-wwdc-2019-keynote-monday-1835165714>

苹果全球开发者大会于周一在加州圣何塞正式开幕。如果你没能抢到今年会议的门票(或者你只是对软件发布感兴趣)，你可以在网上看到很多活动。



主题演讲将于美国东部时间周一下午 1 点/太平洋时间上午 10 点开始，可能是每个人都会从会议中得到的一个事件。

WWDC 的主题演讲总是很长，所以如果你决定观看这个，请准备好至少两个小时。

观看节目最简单的方法是去苹果的活动页面。在那里，您可以使用 Safari、Chrome 或 Firefox 流化整个 keynote。

如果你有一台苹果电视，你也可以从活动应用程序中播放主题演讲。iOS 设备的官方 WWDC 应用程序也将允许您现场直播主题演讲。

至于会发生什么，WWDC 是面向开发者的，所以大部分的公告可能都与软件相关。这并不意味着我们不会看到任何新的硬件，但如果我们看到了，很可能不会像新的 iPhone 一样。
