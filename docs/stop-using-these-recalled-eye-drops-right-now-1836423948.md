# 立即停止使用这些召回的眼药水

> 原文:[https://life hacker . com/stop-use-these-recall-eye-drops-right-now-1836423948](https://lifehacker.com/stop-using-these-recalled-eye-drops-right-now-1836423948)

几种非处方药眼药水，包括沃尔玛、沃尔格林和 CVS 品牌下专门包装的眼药水，由于担心它们可能不是无菌的，已经被召回。

Watch

就你的眼睛而言，无菌是不可忽视的。正如发布召回的 [阿尔泰制药](https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts/altaire-pharmaceuticals-inc-issues-voluntary-recall-multiple-ophthalmic-products-sold-cvs) 在其公告中所说，“使用旨在无菌的非无菌产品可能导致严重的、潜在的威胁生命的感染或死亡。”

几十种不同类型的非处方药眼药水，以及几种处方眼膏都受到了影响。最好对照沃尔玛[沃尔格林](https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts/altaire-pharmaceuticals-inc-issues-voluntary-recall-multiple-ophthalmic-products-sold-walgreens) 和 [CVS](https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts/altaire-pharmaceuticals-inc-issues-voluntary-recall-multiple-ophthalmic-products-sold-cvs) 的全部召回清单，检查你的任何眼药水或软膏，以及任何品牌名称为 [Perrigo](https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts/altaire-pharmaceuticals-inc-issues-voluntary-recall-multiple-ophthalmic-products) 的处方软膏。

召回通知称，对此次召回有疑问的客户可以在美国东部时间周一至周五上午 8:30 至下午 5:00 通过拨打 1-800-258-2471 或发送电子邮件至 otcdruggist@aol.com 联系 Altaire Pharmaceuticals，Inc .

如果您遇到了任何可能与服用或使用这些产品有关的问题，请联系您的医生或医疗保健提供者。
