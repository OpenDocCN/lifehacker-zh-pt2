# 你应该总是买带骨的鸡腿

> 原文：<https://lifehacker.com/you-should-always-buy-bone-in-chicken-thighs-1840457445>

每个吃鸡肉的人都喜欢无骨无皮的大腿。多汁，鲜嫩，非常适合从油炸到慢炖的任何食物，几乎是任何食谱的完美选择。但所有这些现在都是常识，这意味着无骨大腿不再像几年前那样便宜。

Watch

这就是为什么我是带骨带皮鸡腿的忠实粉丝，这种鸡腿稍微不太方便，但更实惠。它们像 [一只巨大的鸡翅](https://lifehacker.com/instant-pot-buffalo-wings-are-fast-easy-and-very-good-1822565911) 一样又肥又大，但它们本身的肉又多又足可以作为一餐。大腿骨恰好也是自己分解肉的理想入门，这是一项甚至没有足够多的人认为需要发展的技能。只需一点点努力，选择带骨的而不是去骨的可以让你花一半的钱买到两倍的鸡肉。

我再怎么强调去鸡腿骨也不为过。正好有一块骨头。要取出它，用你的刀尖沿着它的边缘把它拔出来。就是这样。

如果你喜欢，你也可以去掉鸡皮(它会马上脱落)并留作他用，比如 schmaltz——或者像 YouTube 视频中的厨师建议的那样，用海盐烤鸡皮串。(是的，*请*。)只需几分钟，你就能得到一堆去骨的大腿肉，加上所有你需要用来做 [和](https://skillet.lifehacker.com/how-to-make-stock-that-isnt-bland-nonsense-1820626531) 的美味炖肉。

也许你宁愿避免任何数量的业余屠宰，这是一个有效的偏好。大腿骨仍然是运动的方向，尤其是你为即将到来的冬天准备的所有舒适的炖肉。将皮朝下放入冷锅中，转中火，继续煮 15-20 分钟，直到皮变脆并呈深棕色。然后，即使食谱上说不是这样，在继续食谱的其余部分之前，剥掉并保留鱼皮。

除了显而易见的金棕色酥脆美味，这种技术还有一些好处。尽早去皮可以限制成品菜肴中鸡肉脂肪的含量*和*可以让你避免稍后从滚烫的肉片上剥下光滑的鸡皮。但更重要的是，预酥皮可以让你毫不费力地品尝到最丰富、最深的原汁。这对我来说是足够的理由——尤其是在喝汤的季节。
