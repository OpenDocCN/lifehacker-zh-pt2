# 创建一个虚拟食谱俱乐部

> 原文：<https://lifehacker.com/start-a-virtual-recipe-club-1843005461>

我最近一直在忙着打电话，我发现有一个活动、一个主题或一些焦点来引导电话会很有帮助。不仅仅是简单的分享低调——但仍然令人心动！不管怎样，存在恐惧的感觉。虚拟的快乐时光很好，但不是每个人都喝酒，也不是每个人都想像这些电话发生的那样频繁地喝酒。如果这一切听起来对你来说很熟悉，那么通过虚拟食谱俱乐部的方式将焦点从饮料转移到食物上可能是正确的举措。



虚拟食谱俱乐部的前提很简单:你们都选择一个食谱，通过 Zoom(或者你选择的视频聊天服务[)一起制作，然后你们一边吃一边享受彼此的陪伴和比较笔记。如果有人在这道菜上有问题，俱乐部的其他成员可以帮助他们找出问题出在哪里。如果有人有提示、诀窍或特别成功的修改，他们可以与全班分享。你甚至可以烹饪同一个食谱的不同版本——考虑一个经典的葡萄酒炖鸡——并讨论什么有效，什么无效。](https://lifehacker.com/find-the-best-video-chat-app-with-this-mega-guide-1842732323)

除了社交元素之外，食谱俱乐部还能为你提供一点额外的动力，让你自己做一顿美餐，如果你对压力的反应是在床上吃几碗麦片，这可能是一个挑战。如果吃的乐趣不能吸引你，也许有意义的社会交往的承诺会。
