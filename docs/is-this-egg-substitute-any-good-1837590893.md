# 绿豆“鸡蛋”素食替代品好吗？

> 原文：<https://lifehacker.com/is-this-egg-substitute-any-good-1837590893>

寻求复制真实事物的味道和质地的纯素食食品正在经历一个时刻，看起来这个时刻可能会延长到一分钟，甚至一个小时。(这是好事，我相信。)虽然有很多方法可以模仿鸡蛋在烹饪和烘焙中的结构作用，但 [Just Egg](https://www.ju.st/en-us/products/consumer/egg/scramble) 是第一款为素食者提供早餐吃炒鸡蛋这种简单快乐的产品。



当然也有豆腐炒货，但那些尝起来像豆腐，而不是鸡蛋。它们看起来像豆腐，闻起来像豆腐，吃在嘴里感觉像豆腐。Just Egg 旨在让你相信你吃的是鸡蛋，差不多就到了。淡黄色的液体产品从一个不透明的瓶子中倒出(你必须先摇晃瓶子以避免分离)，然后煮成一堆蛋白质，看起来就像炒鸡蛋一样。

Just Egg 也像炒鸡蛋一样移动，无论是在你的叉子上还是在你的嘴里(至少在它热的时候)。然而，它尝起来不像炒鸡蛋。老实说，我并不期望鸡蛋尝起来就像鸡蛋一样，但是我很惊讶它尝起来有多像鸡蛋以外的东西。最成功的人造肉类和奶酪是那些味道根本不怎么样的。如果他们能得到正确的质地，而不是强烈地品尝其他东西，这通常足以让你的大脑相信你正在吃一个 [汉堡](https://skillet.lifehacker.com/are-fake-meat-burgers-finally-as-good-as-beef-1836815995) ， [香肠](https://lifehacker.com/the-best-fake-sausages-for-grilling-1835616180) 或 [烤奶酪](https://skillet.lifehacker.com/two-grilled-cheese-alternatives-for-people-who-cant-eat-1795388350) ，特别是如果有很多配料和酱料，或者如果你喝醉了。

但是鸡蛋有一种强烈的味道，这种味道我不会联想到鸡蛋。我以为我尝到了豌豆蛋白，这在非乳奶酪替代品中很常见，但它实际上是绿豆蛋白，天哪，我很尴尬，我犯了那个错误。它尝起来并不难吃，只是蔬菜味和咸味，但它没有蛋黄糊的那种奶油味，老实说，这整件事把我的脑子都搞乱了。我的感官处于战争状态。我的眼睛*看到了*鸡蛋。我的牙齿摸到了*的*蛋。但是我的舌头*尝到了*绿豆。在恐怖谷餐厅，我坐在一张单人餐桌旁，我不喜欢它。

另外，如果你想在恐怖谷餐厅点煎蛋卷，恐怕你来错地方了。Just Egg 被设计成炒鸡蛋，这意味着它需要在烹饪时移动。当我离开平底锅时，没有搅拌，它开始在两边凝固，但随后在中间像地狱一样冒泡，在没有煎蛋卷的底部产生斑驳的纹理，疯狂地粘在我的(抹油的，不粘的)平底锅上。(虽然也许我是不耐烦的；Just Egg 网站有 [一个煎蛋卷](https://www.ju.st/en-us/recipes/veggie-omelet) 的食谱，他们的看起来挺好看的。)

这并不是说它是不好的素食产品。如果你吃素已经有一段时间了，并且已经忘记了鸡蛋的味道——或者根本就不喜欢鸡蛋——你会对鸡蛋非常满意，尤其是如果你想要一个质地可信的戴亚肉丝和假培根的载体。我对完美炒鸡蛋的热爱使我明显不是鸡蛋的目标人群，但我的素食朋友们尝过之后真的很喜欢，我为他们感到高兴。也许有一天会有一种鸡蛋替代品，所有饮食哲学的炒鸡蛋爱好者都可以享用。那一天不是今天，但仅仅是鸡蛋就让它看起来不远了。
