# 这个奇形怪状的玩具是如何成为我烦躁的孩子最大的工具的

> 原文：<https://lifehacker.com/how-this-odd-shaped-toy-became-the-greatest-tool-for-my-1833809918>

我儿子 4 岁时，像大多数 4 岁的孩子一样，他需要搬家。很多。此外，他是那种即使在休息时间也能获得额外感官输入的孩子。



当他在学步/学龄前的时候，我们在家里的不同地方都有一个滑梯和一个迷你蹦床，这样他就可以释放能量，尤其是在早上的第一件事。

随着他年龄的增长(我们开始致力于收回主要的生活空间)，我给他买了这个 Bilibo 座椅 。我不知道有谁已经购买了一个；我在网上假日购物时偶然发现了它。30 美元的价格对于一个学步儿童座椅来说似乎有点贵，但评论向我保证，这将是他在观看节目时坐立不安/摇摆/旋转的完美选择。

但 Bilibo 给了我一个更好的东西:它成了他童年最持久、最常用的物品之一。

起初，他只是喜欢坐在里面摇来摇去(或者让我们旋转他)，但很快，它就成了他所有游戏的主要内容。他背上是一个龟壳，这是他的塑料家伙的一个巨大的游泳池。那是一颗流星，一个外星人，一座供他的恐龙攀爬的小山。

他和他的朋友会在我从来没有完全理解的游戏或场景中使用它。

他也以功能性的方式使用它。当他有各种各样喜欢玩的玩具，并想在第二天放在一起时，他会在晚上把它们都扔进 Bilibo。它非常结实，所以我经常走进客厅，发现他在看电视*和*，增强他的核心力量和平衡。

如果你需要一个新的玩具让你的小孩坐立不安，或者激发他们的想象力，Bilibo 将是一个不错的选择。
