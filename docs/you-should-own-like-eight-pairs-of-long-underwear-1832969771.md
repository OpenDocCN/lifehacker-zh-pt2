# 你应该拥有八条长内裤

> 原文：<https://lifehacker.com/you-should-own-like-eight-pairs-of-long-underwear-1832969771>

Lifehacker 的员工在 [一间冰冷的办公室](https://lifehacker.com/what-to-do-when-your-office-is-too-cold-1821872684) 里工作。有时我们戴帽子，总是穿毛衣，有几次甚至戴无指手套。今年冬天，每天气温都在 45 度以下，我在裤子里穿上了舒适的秋裤。 [我是一个大大的烤肉桂面包。](https://www.youtube.com/watch?v=GGfz7ptRjgM) 你应该也是。



你看，大冷天，你上半部穿几层？通常我有一件汗衫、一件长袖衬衫和一件毛衣。偶尔我会穿一件轻薄的羊毛衫和一件连帽衫。也许你会戴一条围巾或一顶帽子。如果你只穿了一件上衣和一条裤子，那么你也应该穿秋衣秋裤。如果你习惯穿打底裤，你知道会发生什么。如果没有，准备好感受被支持和被愉快地包围。你会发现你可以穿更薄的裤子，甚至短袜。你的腰部以下将是一个全新的世界。

囤货。我一直凑合着四双，但是最近又订了四双，所以我总是可以在洗前一周去。如果你在短内裤外面穿上秋衣秋裤，这样他们就不会整天熟悉你的肮脏部位，你可以只穿两三双。但在这一点上，你每次去小便都要处理很多层，所以我建议只使用你的长内裤作为内衣，并得到一大叠。

现在，在你弄清楚你喜欢哪种之前，不要一下子全买了。我是一个身材矮小的人，我的品味是量身定制的，所以我更喜欢日本服装品牌优衣库(Uniqlo)的紧身 Heattech 秋裤。它们有“ [超暖](https://www.uniqlo.com/us/en/men-heattech-extra-warm-long-johns-408119.html) ”和“ [超暖](https://www.uniqlo.com/us/en/men-heattech-ultra-warm-long-johns-408121.html) ”它们又软又薄又有弹性。他们有一只苍蝇，他们不会聚在一起。你可能更喜欢厚一点或者松一点的；老海军偶尔有一些库存。

它们并不总是很容易被发现，因为秋衣秋裤有点像旧时的煤矿工人伐木工 *Looney Tunes* 牛仔流浪汉*将会有血腥*的名声，当他们应该有一个圆滑的战术轮廓的表演服装的名声时。(我老婆说我的看起来很性感，她为什么要骗我？所以，一旦你找到一双你喜欢的，就囤起来。不要被一条停产的秋衣秋裤困住。

你不需要全身的约翰或者“上半身”;这就是你汗衫的用途。你可以买一件长袖汗衫，但坦白说，那是你的事，不是我的。

你有时会太热吗？是的。你会脱下你的毛衣。也许你需要时不时地挪到办公室里凉快一点的座位上。但是如果你更喜欢热跑而不是冷跑，或者你长时间处于次优温度下，你应该每天出门都穿上新鲜的秋衣秋裤。
