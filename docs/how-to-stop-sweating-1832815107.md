# 如何停止出汗

> 原文：<https://lifehacker.com/how-to-stop-sweating-1832815107>

我出汗很多。

我边走边流汗。我坐着的时候会流汗。我在谈话的时候会流汗。在任何工作面试中，我都会汗流浃背地离开房间，就像我不由自主地参加了一场湿 t 恤比赛。写这个的时候我都在流汗。

Watch

我的出汗问题长期困扰着我。这是我输了的基因彩票吗？

在与皮肤科医生米歇尔·格林(Michele Green)交谈后，结果表明的回答可能是肯定的，我可能要感谢我的遗传基因造成了这种负担。幸运的是，并不是所有的希望都破灭了，当生活感觉像桑拿浴时，有一些方法可以让你停止出汗。

我和格林讨论了我们为什么会出汗，为什么肉毒杆菌不仅仅是除皱剂，以及如何用一些快速的方法暂时解决你的出汗问题:

### 汗水到底是什么，我们为什么要流汗？

根据格林的说法，我们流汗是因为我们需要降温(这并不奇怪)。汗液本身主要是水，还添加了氯化物、蛋白质、糖类、氨、尿素和微量金属，当被我们大脑中的下丘脑区域触发时，这些物质就会到达我们的皮肤表面。

你身体上也有两种不同的汗腺。遍布皮肤的外分泌腺对运动和高温有反应。流行力学 写道， [对兴奋、紧张和情绪做出反应的大部分位于腋窝和腹股沟的大汗腺，所以你要感谢那些求职面试时留下的汗渍。](https://www.popularmechanics.com/science/health/a23922/the-science-of-sweat/) 

出汗只有在你让它停留在你的皮肤上并且蒸发的情况下才能有效调节你的体温，然而(换句话说，擦拭它或者让水滴落到地板上是浪费你出汗的努力！)

### 遗传学发挥了多大的作用？

很可能是一个大问题，尽管对这个问题的研究非常有限。一项 2002 年跟踪多汗症(或多汗)患者的研究表明，患有这种疾病的人有遗传因素。换句话说，你可能要为出汗的问题感谢你的父母。

根据 [切](https://www.thecut.com/2016/06/if-youre-way-too-sweaty-blame-your-early-childhood.html) ，有证据表明，你长大的地方(及其气候)也可能起作用。我们生来都有大致相同数量的汗腺。然而，在你作为婴儿的头两年，一些腺体从未变得活跃，这是基于需要。

如果你在炎热的地方长大，你会激活更多的腺体，因此，现在会出汗更多。

### 怎样才能不出汗？

嗯，如果你想长期停止流汗，那会很难，但是你可以做一些事情来控制这个问题。

如果你经常穿毛衣，食品和药物管理局最近批准了首个解决你出汗的处方方法。这是一种日常使用的 [药物布湿巾](https://www.qbrexza.com/how-qbrexza-works) ，通过阻断汗腺中的神经递质来减少腋下出汗问题。

如果你也在寻找美容改变，试试肉毒杆菌。根据格林的说法，在前额或腋窝注射肉毒杆菌素有助于显著减少出汗(不过，根据的说法，这并不总是能抑制气味的问题，所以如果气味也是一个问题，你可能仍然需要除臭剂)。

是的，喝冰水肯定会有帮助(如果你真的想补充能量，格林推荐佳得乐)，洗冷水澡也一样。

如果你想快速停止流汗，格林还推荐冰袋技巧——但是用在脖子后面，而不是额头或手上。“这就是热量传感器的位置，”她说。“这实际上比拿着它更有帮助。”

从经验来看，任何情况下都不要穿羊毛。就像穿了一件内置散热器的衣服。
