# 如果你不烤胡萝卜，你就是个傻瓜

> 原文:[https://life hacker . com/you-a-fool-if-you-don-grill-胡萝卜-1835812940](https://lifehacker.com/youre-a-fool-if-you-dont-grill-carrots-1835812940)

我不知道胡萝卜(作为一个整体)是被低估了还是被高估了，但是我知道*烤*胡萝卜几乎被忽视了。是的，很难与烤玉米竞争，但烤胡萝卜是一个烟熏，甜蜜，烧焦的奇迹，我在这里要求你注意它们。

胡萝卜里面有很多糖，烧焦的，几乎烧焦的糖味道非常好。烤架(或烤盘)散发出的黑色烟熏味让所有的甜味都有了发挥的空间，创造出一种完美的平衡，非常夏日，适合各种饮食的配菜。除了胡萝卜，你需要的只是橄榄油和盐。你可以做蘸酱，或者淋上一些墨西哥辣酱，但是我不知道你为什么想要为自己创造更多的工作。为了做出这些咸、甜、烟熏的美食，你需要:

*   一束胡萝卜，彩虹，如果你能找到的话
*   1 汤匙橄榄油
*   盐

去掉任何顶端，然后擦洗或去皮胡萝卜，纵向切成两半，除非它们是非常小的婴儿，在这种情况下，你可以不动它们。把它们放在一个大碗里，淋上油，加入两把盐，搅拌在一起。用中高温加热你的烤架(或烤盘)。将胡萝卜放在炉栅上，煮 15-20 分钟，每五分钟左右翻一次，直到胡萝卜变得又香又焦，可以用叉子轻松刺穿。作为简单的配菜，制作比你认为你需要的更多，因为任何剩菜都可以成为凉拌菜的绝佳佐料。(但是，不要把它们当成——那是对胡萝卜和热狗的不尊重。)
